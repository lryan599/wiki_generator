布氏硬度计是用于测量金属布氏硬度的精密计量仪器，广泛应用于铸铁、钢材、有色金属及软合金等材料的硬度测定，属于工业检测类仪器设备范畴[[S40,S7]]。布氏硬度试验在所有硬度试验中压痕最大，能够反映材料的综合性能，受显微偏析及成分不均匀的影响较小，是一种精度较高的硬度试验方法[[S40]]。

## 定义与基本原理

布氏硬度试验按照现行有效国标GB/T231系列（等同采用ISO 6506）执行，其核心工作机制为：采用规定直径的硬质合金球压头，以指定大小的试验力垂直压入试样表面，保持规定时长后卸载，测量试样表面残留的球形压痕直径d[[S19,S41,S8]]。布氏硬度的物理定义为施加的试验力除以压痕的球形表面积。当试验力单位为kgf时

\[
\text{HBW} = \frac{2F}{\pi D(D - \sqrt{D^2 - d^2})}
\]

当试验力单位为牛顿(N)时

\[
\text{HBW} = 0.102 \times \frac{2F}{\pi D(D - \sqrt{D^2 - d^2})}
\]

其中D为压头球体直径[[S19,S41,S32]]。实际测试中硬度值一般不需人工计算，可根据测得的d值与已知F、D值直接查表获得[[S8]]。布氏硬度值习惯上不标出单位，仅表示材料在规定条件下的相对数值大小[[S19]]。

现行标准规定布氏硬度测试统一使用硬质合金球压头，硬度符号为HBW，不再推荐旧标准中仅使用淬火钢球的HBS标识[[S33,S34]]。

| 硬度符号 | 压头材质 | 适用硬度范围 | 依据标准 |
|---------|---------|------------|---------|
| HBS（旧标识） | 淬火钢球 | ≤450 | GB/T 231 系列旧版及早期 ISO/ASTM 标准 |
| HBW（现行标识） | 硬质合金球 | ≤650 | GB/T 231 系列、ISO 6506、ASTM E10[[S34,S21,S6]] |

标准规定压痕直径d必须在0.24D~0.6D之间，否则测试结果无效[[S17]]。

## 设备分类与结构

布氏硬度计按使用场景可分为标准台式、便携式和全自动/数显式三类。标准台式布氏硬度计（如HB-3000B机型）多采用杠杆加荷与电子换向技术，具备试验力自动加载、计时保荷、自动卸载功能，测试精度符合GB/T231.2、ISO 6506-2及ASTM E10标准，适用于实验室固定场景下的试样硬度检测[[S7]]。

便携式布氏硬度计可对不便移动的大型铸件进行原位硬度检测，无需拆卸搬运至实验室，检测过程不会对工件造成破坏性损伤，仅要求被测铸件厚度不低于6mm即可获得稳定可靠的检测结果，适合生产现场大型构件的硬度快速抽检[[S39,S34]]。

布氏硬度计的核心组成部件及对应功能如下[[S34,S14,S22]]：

| 部件 | 功能 |
|------|------|
| 机架 | 为整机提供刚性支承，保证加载过程中无形变偏移 |
| 加载系统 | 平稳输出符合精度要求的试验力，无冲击振动 |
| 压头（含硬质合金球或淬火钢球） | 对试样表面施加压入作用 |
| 压痕测量装置（刻度读数显微镜） | 在两个相互垂直方向测量压痕直径，读数精度可达0.01 mm |

## 核心参数与标准规范

### 试验力与压头直径组合

三项核心标准（GB/T 231、ISO 6506、ASTM E10）规定的常用压头直径包含10 mm、5 mm、2.5 mm、1 mm四种，统一规定F/D²比值可选30、15、10、5、2.5、1，适配不同硬度范围材料的测试需求，以保证不同参数组合下同一材料测得的布氏硬度值一致[[S17,S6]]。当试样尺寸允许时，应优先选用直径10 mm的球压头进行试验[[S17]]。

### 布氏硬度值表示方法

布氏硬度值依次标出硬度数值、符号HBW和试验规范（压头直径/试验力/保持时间）。例如：
- 350HBW5/750：表示用直径5mm硬质合金球，在7.355kN试验力下保持10~15s测定的布氏硬度值为350[[S35,S6]]。
- 600HBW1/30/20：表示用直径1mm硬质合金球，在294.2N试验力下保持20s测定的布氏硬度值为600[[S6,S17]]。
- 若试验力保持时间为10~15s，保持时间可不标注，如229HBW10/30[[S17]]。

### 硬度计的检定与校准

GB/T 231.2—2002第6章（对应ISO 6506-2等效条款）规定：直接检验需在硬度计安装、移位后或间接检验不合格时执行，直接检验时间间隔不得超过12个月；间接检验周期在任何情况下不应超过12个月[[S12,S34]]。直接检验宜在(23±5)℃温度范围内进行，检测用仪器需溯源到国家基准[[S34]]。

ASTM E10标准16.2条款规定了更具体的计量要求[[S25,S31]]：

| 计量项目 | 精度要求 |
|---------|---------|
| 载荷测量精度 | 标称值±1%以内 |
| 标准球压头（10 mm）任意方向直径偏差 | 不超过0.005 mm |
| 压头在3000 kgf载荷下作用后直径永久变形量 | 不超过0.01 mm |

每日试验前，建议在标准硬度块上至少压出一个压痕进行日常检查。若硬度读数的平均值与标准块硬度值之差超出GB/T 231.2表2规定的允许范围，则应进行直接检验[[S42]]。

### 试样要求

标准对试样提出了明确的强制性要求[[S42,S18,S10]]：

| 项目 | 要求 |
|------|------|
| 试验面粗糙度 | 不粗于1.25 μm（部分早期文献要求Ra≤0.8 μm）[[S19]] |
| 表面状态 | 需去除表面脱碳层、氧化皮等缺陷 |
| 最小厚度 | 大于等于压痕深度的7倍（部分标准手册建议10倍）[[S18,S10]] |
| 试验后背面 | 不得出现可见凸起痕迹 |
| 压痕距边缘距离 | 至少为压痕平均直径的2.5倍 |
| 相邻压痕中心间距 | 至少为压痕平均直径的3倍 |

### 压痕测量与读数

压痕读数强制要求：需在两个相互垂直方向测量压痕直径，取两个读数的平均值计算布氏硬度。若两个垂直方向测得的压痕直径差值超过0.1mm（ASTM E10要求），则该压痕读数作废，需重新测试[[S42,S31]]。

### 关于“5点平均值”的说明

需特别指出：**GB/T 231、ISO 6506、ASTM E10三项核心标准均未强制要求常规试样测试时选取5个不同位置取平均值**，仅在标准硬度块校准时要求在工作面均匀测定5点并取算术平均值作为标准值[[S12]]。在压铸件实践中，针对铸态组织不均匀的铸件等材料，选取5个分散的测试位置取平均值是行业推荐的常见做法，旨在提升测量结果的代表性[[S40,S38]]。

## 在压铸领域的应用与工艺角色

### 铝合金压铸件硬度检验通用规范

压铸件通常不强制要求硬度检验，仅在产品有特殊需要时，需在产品图纸上明确标注检验部位和检验比例，测试方法需符合HB5168—81《金属布氏硬度试验方法》的相关规定[[S37]]。

### 时效处理后的硬度检测

铝合金压铸件一般不进行固溶处理，仅为消除内应力、稳定尺寸可采用人工时效工艺。时效后可对试样布氏硬度进行检测，测试结果可作为组织稳定性的评估依据[[S37,S28]]。时效后试样需对检测面进行打磨处理去除氧化皮及外来污物[[S40,S38]]。

![图 四种铸造工艺（砂型重力铸造、金属型重力铸造、砂型低压铸造、金属型低压铸造）铝合金试样的布氏硬度对比柱状图，显示金属型低压铸造工艺得到的铸件布氏硬度最高。](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/b8510c18-dc9a-4ead-8cbc-4ad71ed4ee09/resource)  
图 四种铸造工艺铝合金试样布氏硬度对比，直观显示了不同铸造工艺对铸件最终硬度的影响规律[[S27]]。

### 铝铜合金附铸试棒的布氏硬度测试

以铝铜合金典型牌号为例，ZL205A类经T5人工时效热处理后的附铸试棒，平均布氏硬度可达HB107，对应抗拉强度可达483MPa，屈服强度达317MPa，满足金具类压铸件的力学性能要求[[S16]]。

当铸件本体直接进行布氏硬度检测有困难时，可由供需双方商定采用附铸在铸件上的专用硬度试块开展布氏硬度测试。附铸试块需和其所代表的铸件采用完全相同的工艺进行热处理，测试位置由供需双方约定且不得破坏铸件使用性能[[S36,S30]]。

### 开裂部位与失效分析中的硬度检测

布氏硬度计可直接对压铸件开裂区域开展硬度检测。通过对比开裂区域与正常基体区域的布氏硬度差值，可评估开裂位置的材料性能劣化程度，为压铸件失效原因分析提供量化性能数据支撑[[S40]]。布氏硬度试验的另一传统应用场景即为检测断裂区域，在拉伸试验无法实施的情况下尤为适用[[S10]]。

### 锌合金及镁合金压铸件的适用性

**锌合金**：压铸锌合金的布氏硬度检测可完全采用GB/T 231系列金属布氏硬度试验方法开展，试样制备可参照GB/T 13822《压铸有色合金试样》的相关规定执行，适用于常规锌合金压铸件的硬度验收[[S29]]。

**镁合金**：ZM6类镁合金压铸件经T6热处理后布氏硬度可显著提升（实测可达约64HB）。测试时推荐采用10 mm钢压头、试验力3000 kg、加载时间30 s的参数设置，试样厚度需大于压痕深度的10倍以避免测试值失真[[S9]]。

## 影响因素与局限性

### 影响测量结果的主要因素

| 影响因素 | 影响规律 |
|---------|---------|
| 试样表面制备质量 | 残留氧化皮、粗糙度超标会导致读数偏差，测得硬度结果不可靠[[S3,S35,S34]] |
| 加载时间 | 加载时长不足标准要求的10~15 s时，材料塑性变形不充分，测得硬度值偏高[[S34]] |
| 压头变形 | 硬质合金压头超出许用载荷后发生永久变形，会导致压痕偏大，测得硬度值偏低[[S35,S34]] |
| 测试环境温度 | 超出23±5℃范围会引发材料和设备形变，引入额外测量误差[[S34]] |
| 压痕直径范围 | 压痕直径必须在0.24D~0.6D之间，否则测量结果无效[[S17]] |
| 试样厚度 | 厚度不足7倍压痕深度时背面会出现凸起痕迹，不符合标准要求[[S31]] |

### 不适用范围

布氏硬度计在以下情形不适用[[S35,S31,S4]]：
- **材料硬度过高**：硬度超过650HBW的硬质材料，测试过程中会造成硬质合金球压头永久变形，数据失真，且ASTM E10标准明确不推荐对硬度超过650HBW的材料使用布氏硬度试验[[S31]]
- **薄壁工件**：厚度不足7倍压痕深度的薄壁工件，测试背面会出现凸起痕迹
- **极小尺寸精密零件**：过大的压痕会破坏成品结构，不满足无损检测要求
- **高温材料**：布氏硬度测试不宜用于高温条件下的材料[[S3]]

布氏硬度试验的优点是压痕面积大，能较好地反映材料的平均硬度，数据稳定、重复性好；缺点是测试过程相对繁琐，压痕较大，不适宜测量成品件及薄件[[S8,S35]]。
