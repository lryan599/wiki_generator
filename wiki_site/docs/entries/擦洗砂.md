## 概述与定义

擦洗砂（英文术语：*scrubbed sand*）是铸造用天然硅砂中经过擦洗工序加工得到的一种细分品类。根据我国现行国家标准，擦洗砂的含泥量（粒径 < 0.020 mm 的颗粒质量分数）**不大于 0.3%**，清洁度显著优于水洗砂（含泥量 ≤ 1.0%），但低于经过浮选的精选砂（含泥量 ≤ 0.2%，SiO₂ ≥ 96%）[[S4,S11,S29]]。在铸造行业术语体系中，天然硅砂包括水洗砂、擦洗砂和精选砂（浮选砂），擦洗砂的加工目的是通过机械擦洗去除原砂中的泥分及表面附着物，而不是通过化学或浮选方式提纯 SiO₂ 含量 [[S1,S14]]。

在压铸后清理及表面处理领域，擦洗砂作为天然石英基磨料，主要被用作**干喷砂用清理介质**，属于“喷砂砂”“清理砂”大类下的高洁净度磨料 [[S26,S15]]。

## 分类与牌号

按照 GB/T 9442《铸造用硅砂》的命名规则，擦洗砂的牌号由前缀 `ZGS`（铸造用硅砂）加上二氧化硅等级、主要粒度组成和平均细度值构成。一个典型的擦洗砂牌号为 **ZGS 90-50/100（52A）**，代表二氧化硅含量为 90 级，主要粒度组成为四筛，首筛 50 目、尾筛 100 目，其 AFS 平均细度为 52 [[S4,S14]]。

擦洗砂本身属于天然硅砂，其颗粒形状通过角形因数（angularity）进行五级分类 [[S29]]：

| 角形因数范围 | 颗粒形状 |
| ------------ | -------- |
| ≤ 1.15        | 圆形     |
| >1.15 ~ 1.30  | 椭圆形   |
| >1.30 ~ 1.45  | 钝角形   |
| >1.45 ~ 1.63  | 方角形   |
| > 1.63         | 尖角形   |

## 化学成分与基本物理性能

以牌号 **ZGS 90-50/100（52A）** 的典型擦洗砂为例，其化学成分的质量分数如下 [[S10]]：

| 成分 | 含量（%） |
|------|-----------|
| SiO₂ | 92        |
| Al₂O₃ | 4         |
| Fe₂O₃ | 0.5       |
| CaO + MgO | 0.5   |
| K₂O + Na₂O | 1.5  |
| TiO₂ | 0.1       |

擦洗砂的主体矿物为石英，其固有物理属性为 [[S9,S31]]：
- **莫氏硬度**：7 级
- **真密度**：约 2.65 g/cm³

擦洗砂用作喷砂磨料时，其粒度集中度较高。实际检测数据显示，某典型擦洗砂的粒度分布为：50 目含量 30.2%、70 目含量 38.6%，即 50/70 目两筛占比超过 60%，颗粒高度集中在中粗砂范围 [[S5,S6]]。

> **待核实项**：当前公开资料中未明确给出擦洗砂作为清理介质时的松散堆积密度和可循环使用次数，这些参数需进一步实测或查询供应商技术资料 [[S31]]。

## 工艺作用与在压铸链中的应用

在压铸工艺链中，擦洗砂喷砂主要用于铸件的后清理与表面预处理，其典型功能包括 [[S15,S16,S30]]：
- 去除铸件表面的残留污物、微小毛刺和粗糙痕迹；
- 清除氧化皮、残留型砂及熔渣；
- 对铸件表面进行粗化，以增大后续涂装、镀层或粘接工序的附着力；
- 获得均匀一致的消光砂面，达到装饰效果。

对于不同压铸合金，磨料选择有所区别：**铝合金**等非铁合金铸件可选用硅砂（包括擦洗砂）作为喷料；**锌合金、镁合金**等轻质压铸件，推荐使用的擦洗砂粒度应小于普通铸钢件所采用的磨料粒度，以避免表面损伤 [[S33,S18,S22]]。

## 主要设备与工艺参数

擦洗砂喷砂以压缩空气为动力，常用的干喷砂装置有**吸入式、压入式和重力式**三类；也可采用**液体喷砂机**进行湿式作业，后者无需配套通风除尘系统，且磨液可循环使用 [[S17,S33]]。

对于非铁合金压铸件，压缩空气喷砂模式下使用硅砂的推荐工艺参数为 [[S18]]：
- 磨料：40/70 目硅砂（擦洗砂属于该范围）
- 压缩空气压力：0.1 ~ 0.15 MPa
（作为对比，若使用 0.5 ~ 1.5 mm 铁砂，压力可提高至 0.3 ~ 0.4 MPa）

此外，喷砂质量还受**喷射入射角、喷嘴至工件距离、磨料粒度**等参数的综合影响 [[S12,S15]]。通用喷砂规范中建议的较合理操作条件为：喷射角度 30° ~ 60°，喷嘴距加工面 120 ~ 150 mm，具体应视工件材质与结构进行调整 [[S2]]。

> **待核实项**：针对锌合金、镁合金压铸件的专属擦洗砂工艺参数，以及单批次典型处理时长，目前公开手册中尚未给出明确数值，需行业验证数据补充 [[S22]]。

## 相关标准与规范

擦洗砂及其在压铸清理中的应用涉及以下主要标准：

**国内标准** [[S25,S19,S28,S8]]：
- **GB/T 9442—2010**《铸造用硅砂》：规定了擦洗砂的定义、分级、牌号和含泥量要求；
- **GB 6485—86**《铸砂》：规定了铸造用擦洗砂的外形规格、技术要求和验收规则；
- **GB/T 2478—2008**、**GB/T 2479—2008**、**GB/T 2480—2008**：分别对应棕刚玉、白刚玉和碳化硅磨料的技术条件，可作为其他喷砂介质的对比依据；
- **GB 8923**：规定了喷砂表面清洁度评定方法，用于判定擦洗砂喷砂后达到的 Sa 级清洁度；
- **GB/T 25370—2020**《铸造机械 术语》：明确了抛喷丸清理等工艺定义；
- **航空制造领域要求**：若采用高纯度擦洗石英砂，SiO₂ 含量须大于 99.6%，Fe₂O₃ 含量小于 0.02%。

**国际标准** [[S20]]：
- **SSPC-SP5/SP6/SP7/SP10**：规定了不同清洁度等级的喷砂清理要求；
- **ASTM D4417**：提供了喷砂后表面轮廓粗糙度的测试方法。

> **待核实项**：经检索，尚未发现专门针对压铸清理用擦洗砂产品的 ISO 或 ASTM 专属产品标准 [[S20]]。

## 潜在缺陷与注意事项

使用擦洗砂喷砂处理压铸件时，可能引发以下工艺缺陷与安全问题，需在工艺设计中加以控制 [[S28,S2]]：

1. **砂粒嵌入**：铝合金等软质压铸件表面硬度远低于石英颗粒，喷砂压力过大或砂粒过粗时，砂粒容易嵌入基体表层。该缺陷可通过喷砂后增加短时碱蚀工序予以去除。
2. **过度粗化与尺寸变化**：喷射角过小、喷嘴距工件过近或连续喷射时间过长，可导致表面粗糙度超出工艺上限，甚至局部的薄壁区发生减薄变形。推荐的控制参数为：喷射角 30° ~ 60°，喷嘴距离 120 ~ 150 mm。
3. **粉尘与职业健康风险**：干法擦洗砂喷砂过程中，石英砂破碎会产生大量游离二氧化硅粉尘。作业现场必须配备密封除尘系统，并为操作人员提供带正压送风的喷砂防护装备，否则长期暴露有诱发矽肺病的风险。
4. **表面后处理要求**：铝合金压铸件经擦洗砂喷砂后，不可直接进行阳极氧化或化学转化处理，必须先经过碱蚀；且喷砂后应在 **12 小时内**完成后续表面处理，以避免二次氧化。
