## 概述

虚拟正交试验（亦常称为仿真正交试验、CAE正交优化）是压铸领域将正交试验设计方法与铸造CAE数值仿真技术深度结合的工艺优化手段。其核心思想是通过有限元、有限差分或有限体积等数值算法替代实体物理试模，在虚拟环境中完成多因素、多水平的方案遍历与结果采集，从而快速获得压铸工艺的最优参数组合[[S14,S23]]。

传统物理正交试验需投入实体铸件与模具，结果易受操作误差、设备波动的干扰；虚拟正交试验完全在数字化环境下运行，不消耗实体物料，试验结果不受人工操作的随机误差影响，采集到的充型流场、温度场、缺陷分布等信息一致性更高[[S14,S18,S4]]。

## 分类与类型

虚拟正交试验可从多个维度进行分类，常见分类方式包括按仿真算法、按正交表类型、按优化目标维度划分，如下表所示。

| 分类依据 | 类型 | 说明 | 来源 |
|---------|------|------|------|
| 仿真数值算法 | 基于有限元法（FEM）的虚拟正交试验 | 对应ProCAST等软件，几何边界适应性强 | [[S12,S26]] |
|  | 基于有限差分法（FDM）的虚拟正交试验 | 算法简单，适用于规则域；对应部分铸造CAE工具 | [[S12,S26]] |
|  | 基于有限体积法（FVM）的虚拟正交试验 | 对应Anycasting等软件，兼备FDM的简单性与FEM的边界适应性 | [[S12,S26]] |
| 正交表类型 | 等水平正交表类 | 各因素水平数相同，如L9(3⁴)、L16(4⁵) | [[S23,S32,S5]] |
|  | 混合水平正交表类 | 直接适配不同因素水平数不统一的场景 | [[S23,S32,S5]] |
| 优化目标维度 | 单目标压铸虚拟正交试验 | 仅以缩孔率、充型时间等单一指标为优化目标 | [[S11,S22]] |
|  | 多目标压铸虚拟正交试验 | 同时兼顾气孔率、力学性能、模具寿命等多维度 | [[S11,S22]] |

## 核心流程

压铸领域虚拟正交试验的标准实施流程通常包含以下几个步骤[[S14,S18,S35]]：

1. **明确优化目标与评价指标**：确定待优化的压铸工艺响应，常见的为缩孔缩松体积、气孔率、充型时间、模具热应力等。
2. **筛选因素与确定水平**：从浇注温度、压射速度、模具预热温度、压射比压等关键工艺参数中选取核心因子，并为每个因子设定若干试验水平。
3. **选择正交表并生成试验方案**：根据因素数和水平数选取适配的正交表，生成全部虚拟试验组合。
4. **执行仿真试验**：将每组试验的参数逐一导入铸造CAE仿真软件（如Anycasting、ProCAST等），完成数值计算并采集各响应变量的输出结果。
5. **数据分析**：通过极差分析、方差分析、信噪比分析等手段处理仿真结果，确定各因素对优化目标的影响主次顺序，计算理论最优参数组合。
6. **验证与确认**：对最优参数组合进行补充仿真验证，必要时开展少量实体试模与无损探伤验证，最终确认可投入量产的工艺方案。

## 在压铸中的应用场景

虚拟正交试验在压铸领域主要应用于以下三类场景[[S14,S23,S34]]：

- **工艺参数优化**：针对铝合金、镁合金等材质压铸件，通过虚拟仿真获取不同参数组合下的充型、凝固结果，降低缩孔缩松、卷气等缺陷占比。
- **模具结构优化**：通过虚拟仿真获取不同模具冷却水道结构参数下的型腔热应力与形变量数据，提升模具服役寿命。
- **缺陷预测与控制**：基于虚拟仿真得到的缺陷分布结果，量化不同工艺参数对缺陷形成的影响权重，为工艺设计提供依据。

## 关键参数与正交表

### 常用因子（自变量）

压铸虚拟正交试验中常选取的工艺因子包括：合金液浇注温度、压射速度（慢压射速度、快压射速度）、模具预热温度、压射比压、真空度、保压参数、冷却水流量等[[S17,S23,S24,S33]]。多数研究选择3～5个核心因子开展试验；部分面向模具结构优化的试验还会纳入模具水道直径、水道距壁距离等几何参数[[S14,S23,S34,S28]]。

### 常用响应变量（评价指标）

常用的仿真输出响应有：铸件典型截面缺陷面积总得分（缺陷概率与面积的加权值）、充型时间、凝固时间、孔隙率/缩松率、卷气/氧化夹杂占比、铸件重量、硬度、尺寸偏差等[[S19,S24,S16,S33,S29]]。多数研究将同时关注的评价指标控制在3项以内，以控制试验规模[[S19,S23,S13,S7]]。

### 常用仿真软件

压铸领域可搭配虚拟正交试验的主流铸造CAE软件包括：Anycasting、MAGMASOFT、ProCAST、Flow-3D及华铸CAE/IntelCAST，这些软件均支持压铸充型、凝固过程模拟与缺陷预判输出[[S25,S21,S10]]。

## 优势与局限性

### 优势

- **成本与效率**：无需消耗金属液、模具等实体物料，开发周期可从数周缩短至数天；采用该方法优化后铸件综合质量可提升15%[[S18,S22,S9]]。
- **数据一致性**：试验结果不受实体操作偶然误差与系统误差干扰，可精确输出任意位置的流场、温度场、应力场数据，便于直观分析参数对缺陷的作用规律[[S14,S18]]。
- **信息丰富度**：可覆盖传统物理试验中难以直接测量的参数维度，实现对全变量的精细化分析[[S18,S22]]。

### 局限性

- **模型精度依赖**：最终优化结果完全依托仿真模型的准确性，若材料热物性参数、边界条件与实际工况存在偏差，所得最优参数将偏离实际需求[[S2,S27]]。
- **复杂现象复现困难**：部分压铸中的复杂物理现象（如金属液对砂芯的冲蚀破碎、模具表面镀层失效脱落等）难以通过现有通用仿真模型完全复现，此类场景的虚拟试验结果无法等效物理试验[[S2,S27]]。
- **前处理敏感性**：网格划分精度与求解器设置不合理会直接引入计算误差，进而影响正交分析的可靠性[[S27]]。

## 与其他设计方法的关系

压铸领域常用的优化设计方法包括全因子设计、均匀设计、正交试验、响应面法及田口方法（Taguchi），其核心差异如下表所示[[S15,S13]]。

| 方法 | 特点 | 试验规模 | 适用场景 |
|------|------|----------|----------|
| 全因子设计 | 覆盖所有因子水平组合 | 极大 | 因子数极少时的验证场景 |
| 均匀设计 | 试验点在参数空间均匀分布 | 极少 | 样本量受限、统计建模要求高 |
| 正交试验 | 分数因子设计，代表性选点 | 小 | 压铸行业应用最广的基础试验框架 |
| 响应面法 | 在初优区间拟合非线性回归模型 | 中等 | 小范围精准寻优 |
| 田口方法 | 结合正交表，引入信噪比以降低噪声敏感性 | 小 | 强调鲁棒性的工艺优化 |

工程实践中，通常采用“虚拟正交试验获取初优结果，再结合响应面法或神经网络等智能算法在初优区间开展精准全局寻优”的组合流程[[S3,S7]]。部分面向质量稳定的研究则直接采用“田口法＋压铸CAE仿真”路径，输出抗干扰能力更强的量产工艺参数[[S7]]。
