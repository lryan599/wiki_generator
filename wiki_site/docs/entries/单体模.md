## 概述

**单体模**（整体式压铸模，英文对应术语为 **integral die**，型腔一体化加工的标准描述为 **cavity machined from/in solid**[[S9,S40]]）是压铸模按构造分类下的一个核心子类[[S4,S22]]。其本质特征是：构成压铸件几何形状的型腔（有时连同浇注系统、溢流排气系统）直接在一整块模块（通常为热作模具钢）上加工成型，模块本体与型腔构成不可拆分的单一整体[[S34,S35,S37,S46]]。在行业俗称中，也被称为“直雕模”[[S5,S23]]。

“单体模”强调模具成形零件的整体性，与 **单穴模（单腔模）** 属于完全不同的分类维度。单穴模仅指模具只含一个型腔，而该型腔完全可以采用镶拼结构；单体模则专指型腔无镶拼、由整块材料直接加工的结构形式[[S5,S22]]。

## 定义与核心特征

在压铸模结构分类中，整体式压铸模（单体模）定义为：**成形部分的型腔直接在一个整体模块上加工成形的压铸模**[[S34,S35,S37,S30]]。其核心技术特征包括[[S4,S38,S35,S39,S46]]：

- **高强刚度**：模具整体强度高、刚性好，装配工作量小，外形尺寸紧凑。
- **无镶拼接缝**：型腔件为一体，压铸件表面无拼缝痕迹，完整无毛刺，尺寸精度更高，可大幅减少后续去毛刺工作量。
- **冷却水道易于布置**：无镶块缝隙干扰，在模块内部开设冷却水道更灵活，散热条件优于镶拼式结构。
- **高熔点合金寿命优势**：压铸高熔点金属时，整体式模具的使用寿命显著高于同规格镶拼式模具。

## 分类归属

根据《压铸模设计手册》等多本权威著作，压铸模按构造（结构）分为 **整体式压铸模（单体模）、镶拼式压铸模、共用模套式压铸模、组合式压铸模** 四大类[[S4,S22]]。单体模是该分类体系中独立的一级子类，与镶拼式结构构成最基本的成型零件结构大类[[S42]]。其内部可进一步细分为：整个模具为整体、套板与镶块为整体、镶块为整体（装入套板）三种形式，但核心均要求成型型腔由单一基材整体加工而成[[S45]]。

## 结构与组成

整体式压铸模的典型结构是：在动、定模板（或专门的成型模块）上直接铣削或电蚀出型腔、型芯、浇道和排气槽，不存在分体镶件[[S15,S20]]。动、定模合模后，这些一体化加工的特征形成一个完整的压铸型腔。其结构示意如图：

## 制造工艺

单体模的常规制造工艺流程（含CNC与电火花工序）为[[S31,S24,S13]]：  
毛坯下料 → 六面粗铣留余量 → 型腔粗加工 → 热处理（淬火+回火）→ 精磨六面 → 型腔半精铣、精铣（高速铣削可直接加工大部分浅型腔成型面）→ 剩余刀具难以触及的窄槽/复杂特征用电火花成型补充加工 → 抛光。

其中，高速铣削加工淬硬模具钢时采用TiAlN涂层整体硬质合金刀具，可大幅缩短电火花加工耗时，减少钳工修配工作量[[S31]]。

## 材料与热处理

整体式压铸模成型模块的材料选择取决于适配合金：  
- 低熔点合金（锌、锡、铅）：可选用普通合金结构钢或45钢，生产批量极小时可用；批量大、形状复杂时可选用热作模具钢（如5CrNiMo、4Cr5MoSiV等）[[S8]]。  
- 中高熔点合金（铝、镁、铜）：优先选用H13（4Cr5MoSiV1）等热作模具钢[[S8,S41]]。

对应推荐热处理硬度[[S21,S41,S32]]：  
- 适配锌合金的整体模：46～50 HRC  
- 适配铝、镁合金的整体模：43～48 HRC（最适宜）  
- 适配铜合金的整体模：40～44 HRC（一般不超过44 HRC）  
热处理后可按需进行软氮化处理，氮化层深度0.08～0.15 mm，硬度≥600 HV[[S32,S41]]。

## 优势与局限性

与镶拼式压铸模相比，整体式压铸模的优势与局限性可系统对比如下表（综合[[S4,S38,S14,S35,S40,S20]]）：

| 对比维度 | 整体式压铸模（单体模） | 镶拼式压铸模 |
|----------|------------------------|--------------|
| **强度与刚性** | 高，无拼接弱面 | 较低，存在拼合面 |
| **铸件表面质量** | 无拼缝痕迹，尺寸精度高，毛刺少 | 易产生镶拼飞边，去毛刺工作量大 |
| **模具装配** | 装配量小，外形尺寸紧凑 | 装配复杂，外形尺寸较大 |
| **冷却系统设计** | 冷却水道易于布置，温控均匀 | 冷却水道开设困难，热扩散条件较差 |
| **高熔点合金寿命** | 显著更高 | 相对较低 |
| **制造难度** | 形状复杂时整体加工困难，材料利用率低，整块模具钢消耗大 | 可分块加工，降低加工难度，可合理选用不同档次材料 |
| **局部维修** | 局部易损无法单独更换，局部损坏后模具基本整体报废 | 易损件可单独更换，维修成本低 |
| **热处理风险** | 整体模块变形、开裂风险高 | 镶块可分别热处理，变形小 |
| **适用规模** | 适合小批量、试制、简单形状 | 适合大批量、复杂形状、大尺寸 |

## 典型应用场景

整体式压铸模适用于以下场合[[S4,S38,S35,S45,S20]]：  
1. 型腔较浅的小型单腔模具，或型腔加工较为简单的模具。  
2. 压铸件形状简单、精度要求不高、合金熔点较低（如锌、锡、铅合金）的铸件。  
3. 新品试制、小批量生产、老产品改型或替换件等不需要严格热处理或追求短周期的场合。  
4. 受压铸机拉杆空间尺寸限制，模具外形尺寸必须缩小，无法采用镶拼结构的工况。  
在铝合金、镁合金小批量试制中也可采用，但通常不用于十万模次以上的大规模量产[[S38,S14,S27]]。

## 与其他模具术语的区别

为避免概念混淆，需厘清以下术语边界：

- **单穴模（单腔模）**：属于按型腔数量维度的分类，与型腔是否整体加工无关。单穴模完全可以采用镶拼结构，与“单体模”属于不同分类体系[[S5,S22]]。  
- **单元模（Unit Die）**：属于标准化快换模具体系，指自带导柱、推出机构等功能的可快速更换的型腔单元，装入标准保持座使用。其分类维度是模块化快换，而非型腔是否整体加工[[S5,S23]]。  
- **复合模（Combination Die）**：在同一模具上设置不同铸件型腔，实现不同零件同步压铸，或冲压领域内同一工位完成多道工序的模具，均不属于成型零件是否整体加工的范畴[[S10,S1]]。

## 工艺参数与操作条件

- **冷却系统**：冷却水管道需远离型腔壁10～15 mm，无镶块缝隙干扰，温控稳定性优于镶拼模[[S7,S20]]。  
- **脱模斜度**：型芯脱模斜度推荐范围1°～1°30´，铸件壁越厚、合金收缩率越大，所需脱模斜度相应增大[[S7,S12]]。  
- **模具寿命**：主要受热处理质量、预热工艺、冷却水温控制及表面强化状态影响。正确预热（铝、镁合金模具预热温度130～180℃）并定期重新氮化可使铝合金压铸模寿命达20万模次以上[[S33,S18]]。  
- **温控特性**：仿真云图显示整体式模具工作时型腔区域温度分布约为200～312℃，温度均匀性优于同工况镶拼式模具[[S47]]。
