## 概述

触变性（Thixotropy）是一种依时性非牛顿流变特性，指材料的表观粘度在恒定剪切条件下随作用时间推移逐步降低，剪切停止后又随静置时间延长逐步恢复至初始值[[S11,S12,S27]]。在半固态金属成形领域，固液两相区间内的半固态浆料普遍具备该特性，其表现直接支配着半固态压铸过程中浆料的充型能力与最终铸件质量[[S17,S34]]。

## 宏观特征

半固态金属的触变行为在宏观上表现为类固态与类液态之间的可逆转变[[S4,S23]]：

- **低剪切或静止状态**：浆料内部固相网络结构完整，整体表观粘度极高，宏观呈现类固态特征，可保持形状稳定并直接夹持搬运。
- **剪切作用阶段**：施加持续剪切后，固相网络逐步解聚破碎，表观粘度随剪切时间延长不断下降，材料流动性显著提升。
- **剪切撤除后**：分散的固相颗粒重新搭建三维网络，表观粘度随静置时间逐步回升，最终恢复至剪切前的高粘度类固态特征。

## 微观机制

半固态浆料触变行为的微观本质是固相颗粒网络的破坏与重建过程[[S21,S22,S45]]：

1. **静置网络形成**：近球形固相颗粒通过固液界面结合作用相互搭接，形成贯穿整个浆料体系的三维固相网络，赋予浆料高粘度和类固态特征。
2. **剪切解聚**：施加持续剪切时，固相网络的解聚破碎速率远大于颗粒重新聚集速率，原有连续网络被逐步拆分为分散悬浮于液相基体中的固相颗粒，流动阻力随时间推移不断降低。
3. **静置重构**：剪切停止后，分散的固相颗粒通过布朗运动和碰撞聚集重新搭接，三维网络逐步重构，表观粘度同步回升至初始水平。重新聚集（团聚）速率远小于解聚速率，是触变具备依时性的根源[[S22]]。

## 与剪切稀化（假塑性）的区别

触变性与纯剪切稀化（假塑性）具有本质区别[[S11,S12,S15,S47]]：

| 特性 | 触变性流体 | 假塑性流体（剪切稀化） |
|------|-----------|---------------------|
| 时间依赖性 | 有，粘度随剪切时间逐步变化 | 无，粘度仅与瞬时剪切速率相关 |
| 结构恢复 | 剪切停止后逐步恢复，存在滞后 | 剪切停止后瞬时恢复至初始值 |
| 滞后环 | 剪切速率升降过程产生明显闭合滞后环 | 升/降曲线重合，无滞后 |
| 相互关系 | 凡有触变性的材料必有假塑性 | 有假塑性的材料未必有触变性 |

触变性的时间依赖特征源自固相网络破碎与重建均需一定时间完成，而假塑性流体的内部结构调整在剪切速率变化时瞬时完成[[S11]]。

## 影响因素

半固态合金触变性能受以下关键参数影响[[S33,S22,S32,S19]]：

| 影响因素 | 影响规律 |
|---------|---------|
| 固相体积分数 | 固相分数越高，滞后环面积越大，触变性越强；高固相率下时间依赖性更显著 |
| 固相颗粒形貌 | 球形非枝晶颗粒比枝晶结构流动阻力低约2个数量级，触变成形必需的理想形态 |
| 冷却速率 | 冷却速率升高可细化初生晶粒，但相同固相分数下表观粘度增大 |
| 温度 | 温度降低→固相分数升高→表观粘度显著上升，触变效应同步增强 |
| 剪切历史 | 剪切速率升高后粘度降低速率显著快于降低剪切后的恢复速率，历史依赖效应在高固相分数下更突出 |

对Sn-15%Pb半固态合金的定量研究进一步表明：初始剪切速率越高触变性越弱，剪切速率上升时间t<sub>u</sub>越长触变性越弱，剪切速率下降时间t<sub>d</sub>越长/最大剪切速率越高则触变性越强[[S33]]。

## 测量与表征

半固态金属触变性能的主要量化评价方法分为两类[[S17,S20]]：

### 滞后环法（触变环线法）

标准操作流程[[S33,S20]]：
1. 半固态材料在预设初始剪切速率下剪切至初始稳态；
2. 将材料静置指定时长；
3. 在设定的上升时间t<sub>u</sub>内将剪切速率由零提升至最大设定值；
4. 在最大剪切速率下保持剪切至材料达到新稳态；
5. 在设定的下降时间t<sub>d</sub>内将剪切速率从最大值降低至零，全程记录剪切应力-剪切速率曲线形成闭合滞后环。

滞后环的积分面积直接表征触变强度，面积越大触变性越强[[S11,S33]]。

![高触变性能材料的剪切应力-剪切速率滞后环实测曲线](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/ae498c4b-1c9b-4ad9-840b-54192016f03e/resource)

该实测曲线展示了剪切速率上升与下降过程中剪切应力不重合所形成的闭合触变环，环面积为8.683 Pa/s，属于高触变水平的工程样本[[S38]]。

### 静置恢复法

先在固定剪切速率下将浆料剪切至表观粘度稳定，停止剪切并保持静置指定时长，随后重新以恒定剪切速率施加作用，连续记录表观粘度随剪切时间的变化趋势。静置时间越长，重新剪切后粘度下降至稳态值所需的时间越长，代表材料触变程度越高[[S20,S36]]。

该曲线展示了触变流体在施加恒定剪切后表观粘度随时间逐步下降、停止剪切后粘度随静置时间逐步回升的完整依时变化过程[[S40]]。

### 触变指数

行业常用触变指数简化评价触变程度[[S13]]：

- **时间对比法**：B = (η<sub>30</sub> − η<sub>600</sub>) × 100% / η<sub>30</sub>，其中η<sub>30</sub>和η<sub>600</sub>分别为测试开始30 s和600 s时的表观粘度值，B=0时材料无触变性。
- **转速对比法**：取低转速5.6 r/min下粘度η<sub>1</sub>与高转速65 r/min下粘度η<sub>2</sub>的比值A = η<sub>1</sub>/η<sub>2</sub>作为评价指标。

### 测试标准与设备

半固态金属浆料触变性能测试常用的五类方法为同轴双筒剪切流变法、平行板旋转剪切流变法、平行板压缩变形法、反向挤压法和改性毛细管流变法，适配剪切速率区间覆盖10⁻⁴~10³ s⁻¹，适配固相分数覆盖5%~65%[[S7]]。

- **ASTM D2196**：指定旋转粘度计法为非牛顿流体触变性能的标准测试方法，广泛应用于半固态金属浆料的触变性能标定[[S39,S42]]。
- **ISO 3219**：旋转流变仪测量非牛顿流体流变性能的通用国际标准，为触变性能测试结果的跨实验室比对提供统一基准[[S30]]。
- **GB/T 40809-2021**《铸造铝合金-半固态流变压铸成形工艺规范》：通过差热分析采集合金加热与冷却过程的焓值数据，计算得到半固态浆料的实时固相率，为触变性能测试时的状态标定提供统一依据[[S1]]。

## 在压铸工艺中的角色

触变性能是半固态压铸技术的核心物理基础，直接决定浆料的充型能力和铸件质量。

### 流变铸造

低剪切状态下半固态浆料具备足够粘度实现可控输送，受剪切时粘度快速降低获得理想流动性，在合理压力与速度匹配下实现平稳层流充型[[S24,S5,S48]]。

### 触变铸造

该工艺完全依托半固态合金的触变特性：室温下制备的非枝晶铸坯重新加热到固液两相区后，低剪切力下表观粘度极高呈现为软固体状态，可直接机械夹持转运；注入压室后受高剪切作用粘度骤降，平稳充入型腔实现成形[[S16,S41]]。

### 工艺优势

依托触变剪切变稀特性，半固态压铸获得了传统液态压铸难以实现的工艺优势：

- **层流充型**：充型过程实现全壁厚层流流动，无喷溅，工程测试得到的临界极限速度为4 m/s，低于该速度可完全避免紊流[[S37,S3,S28]]。
- **低卷气**：半固态高粘度浆料的层流充型特性完全避免常规液态压铸的卷气现象，触变成形Cp/AZ91D镁合金孔隙率仅为1.2%，约为常规重力铸造同材料孔隙率的10%[[S2,S28]]。
- **低热裂倾向**：凝固过程潜热释放量远低于常规液态压铸，固相颗粒呈球形均匀分散，补缩通道通畅，凝固收缩量显著降低[[S31]]。
- **力学性能提升**：组织均匀、无粗大枝晶缺陷，触变铸态A357铝合金力学性能比常规同成分高压铸件高10%以上，铸件可实现T6热处理强化[[S25,S43]]。
- **近净成形**：可实现复杂薄壁件近净成形，脱模斜度可降至零，铸件公差精度高、表面质量优异[[S28,S25]]。
- **高致密性**：无明显卷气孔洞和缩松缺陷，压力密封性优异，可直接用于承压类高端构件制造[[S25]]。

低固相半固态压铸的标准推荐工艺参数如下[[S44]]：

| 工艺参数 | 推荐值 |
|---------|-------|
| 模具温度 | 120–160 ℃ |
| 压射低速 | 0.3 m/s |
| 压射高速 | 3 m/s |

## 应用场合与局限性

### 应用优势

触变成形技术依托半固态合金的触变特性，实现了近净成形与低缺陷制造，在复杂薄壁件、承压类高端构件的生产上展现出明显优势[[S25,S28]]。

### 工程限制

当前触变性能的可控性不足带来多项制约[[S29,S6]]：

- 专用非枝晶预制坯成本比普通铸坯高约40%；
- 二次加热过程中合金氧化损耗占坯料总质量的5%~12%；
- 适配合金种类有限；
- 常规生产条件下废料无法直接在压铸车间内循环回收；
- 整体工艺流程长，综合制造成本显著高于常规液态压铸。

## 相关关系

- **与固相分数的关系**：随固相分数升高，半固态合金的剪切-粘度滞回圈面积逐步增大，触变效应增强，流动阻力显著提升[[S22]]。
- **与颗粒形貌的关系**：在相同固相分数条件下，枝晶结构的半固态合金流动阻力比球形非枝晶浆料高约2个数量级[[S22]]。
- **与冷却速率的关系**：在可获得合格球形非枝晶组织的工艺窗口内，冷却速率提升可细化初生晶粒，但相同固相体积分数下浆料的表观粘度随冷却速率升高而增大[[S22,S24]]。
