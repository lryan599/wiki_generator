## 抽芯距

### 定义与范畴

抽芯距（core pulling distance / core withdrawal distance / core travel distance / core stroke / core-pulling stroke）是指型芯从成形位置抽至完全不妨碍压铸件脱模位置时，型芯或滑块在抽芯方向所移动的距离[[S7,S8,S16,S42]]。这一参数决定了侧抽芯机构能否使型芯完全脱离铸件侧孔或侧凹区域，是侧向分型/抽芯脱模动作的核心控制参数[[S7,S8,S15,S19]]。

在工程术语体系中，与之密切关联但有所区别的两个概念是**抽芯行程**与**滑块行程**：
- **抽芯距**——满足压铸件顺利脱模所需的理论最小必要移动距离[[S25,S29,S37]]；
- **抽芯行程**——抽芯驱动机构实际可输出的有效运动距离，其数值须≥抽芯距以预留安全冗余[[S25,S29,S37,S30]]；
- **滑块行程**——抽芯机构滑块整体的全移动行程，须≥抽芯行程，以保证滑块运动过程中始终处于导滑槽的有效导向范围内[[S25,S29,S37,S30]]。

### 工艺作用

抽芯距直接决定侧向分型/抽芯动作的完整性[[S7,S16]]：
- **过小的抽芯距**将使侧型芯无法完全脱离压铸件侧孔/侧凹，导致取件卡滞甚至铸件无法脱模，同时引发侧孔、侧凹成形尺寸精度不合格及表面划伤等缺陷[[S26,S10,S42]]；
- **过大的抽芯距**会不合理地增大模具整体尺寸，提升制造成本，同时可能导致滑块在导向槽内的有效导向长度不足[[S26,S10,S42]]。

### 基础计算方法

常规侧抽芯场景下抽芯距的通用公式为[[S26,S10,S16,S43]]：

**S = h + K**

式中：
- S —— 抽芯距（mm）
- h —— 压铸件侧孔或侧凹的成形深度（mm），当同一滑块上抽出多个型芯时应取其中最大成形深度[[S6,S19]]
- K —— 安全余量，常规压铸场景下取3～5 mm[[S42,S19,S15]]

抽芯动作结束时，所抽取的型芯端面与压铸件上对应孔端面的距离不应小于3 mm，即安全冗余量至少为3 mm以上[[S11]]。

### 安全值K的选取

安全值K应根据抽芯距离长短及抽芯机构类型分级选取，常用的抽芯距安全值K见表[[S6,S15,S19]]。

| 抽芯距离范围 / mm | K值 / mm |
|:---:|:---:|
| ＜30 | 3～5 |
| 30～80 | 5～8 |
| 80～180 | 8～12 |
| 180～360 | 12～18 |

注：① 斜销、弯销、斜滑块所取安全值应满足角度误差要求；② 所抽型芯直径大于成形深度时，K值按直径尺寸查取；③ 同一滑块上有多个型芯时，K值按型芯最大间距查取[[S6,S19]]。

### 特殊结构铸件的抽芯距计算

当铸件外形全部在滑块内成形时，需根据滑块等分方式采用差异化计算公式[[S10,S26,S16]]。

**二等分斜滑块抽芯（圆形铸件）**

对于圆形骨架压铸件采用二等分斜滑块合模结构[[S26,S10,S16]]：

**S = √(R² − r²) + K**

式中：R 为压铸件最大外形半径（mm），r 为阻碍铸件推出的外形最小半径（mm）。

**多等分斜滑块抽芯（圆形铸件）**

采用n等分斜滑块结构时[[S26,S10,S16,S2]]：

**S = (R·sinθ)/sinβ + K**

其中：
- θ = 180° − β − α
- α = arcsin((r·sinβ)/R)
- β 为滑块夹角，通式 β = 180°(1 − 1/n)，n 为等分数

典型β取值：三等分 120°；四等分 135°；五等分 144°；六等分 150°[[S10,S26,S16]]。

### 按驱动形式的分类及其抽芯距特性

抽芯距的数值范围与抽芯机构的驱动形式紧密相关。

| 驱动形式 | 抽芯距特性 | 典型范围 |
|:---|:---|:---:|
| 斜销驱动 | 抽芯距 S = H·tanα，受开模行程H及斜销倾角α限制；倾角越大抽芯距越大，但斜销弯曲应力也越大，最大倾角≤25° | ≤40 mm |
| 油缸驱动 | 抽芯距按需定制，不受开模行程限制，仅需液压缸额定行程 Sₕ ≥ 目标抽芯距即可 | 可达数百毫米 |
| 弹簧驱动 | 仅适配短抽距、小抽芯力的简易场景，抽芯距受弹簧有效变形长度限制 | 短行程 |

**斜销抽芯机构**：常规斜销倾角α推荐范围为10°～25°[[S42,S3,S28]]。一般适用于中小行程、抽芯力不大的场合，是应用最广泛的侧抽芯机构[[S29,S33,S28,S9]]。

**油缸驱动抽芯机构**：抽芯动作与开模动作相互独立，抽芯距离、抽芯速度均可灵活设定，不受斜导柱类机动抽芯机构的行程限制[[S41,S30,S36,S24]]。适用于中大型压铸模具的大行程侧抽芯需求[[S24,S14]]。

![液压抽芯机构结构图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/4df23948-d773-467f-8337-27ada78739a2/resource)

图：液压抽芯机构结构示意，展示大抽距定制化侧抽芯的典型部件组成[[S14]]。

**弹簧驱动抽芯机构**：完全依赖弹簧弹力提供抽芯动力，抽芯完成后依靠合模时的模具外力完成滑块复位，仅适用于极短抽距场景[[S4,S17,S18]]。

### 影响因素与设计约束

**合金种类对安全余量的影响**

不同压铸合金的单位包紧力不同，影响抽芯距安全值K的选取倾向[[S7,S16]]：
- 锌合金：p = 6～8 MPa
- 铝合金：p = 10～12 MPa
- 铜合金：p = 12～16 MPa

包紧力越大，抽芯行程需完全释放铸件对型芯包紧区域的长度要求越高，对应安全值宜适当放大[[S7,S16]]。

**与压铸机及周边设备的干涉约束**

抽芯距设计必须校核压铸机最大开模距离：压铸机最大开模距需大于模具总闭合厚度与抽芯行程之和，并预留压铸件顶出安全余量[[S6]]。同时还需避免以下干涉[[S11]]：
- 抽芯在操作者侧时：避免压铸机安全门与抽芯装置、限位装置干涉；
- 抽芯在模具上方时：避免自动喷涂装置与限位装置干涉；
- 抽芯在操作者对侧时：避免自动取件装置与限位装置干涉。

**与导向及锁紧结构的匹配**

抽芯距需小于滑块在导向滑槽内的有效导向行程，确保抽芯完成后滑块仍不脱离导滑槽[[S26,S8,S42]]。合模状态下锁紧块须完全贴合滑块承压面，抵消高压充填产生的侧向胀型力，防止滑块后退影响侧孔/侧凹成形精度[[S26,S8,S42]]。

### 与相邻参数的耦合关系

| 关联参数 | 耦合说明 |
|:---|:---|
| 抽芯力 | 所选抽芯距须保证抽芯行程完全脱离包紧区域，完全释放合金冷凝收缩产生的包紧力，避免残余行程摩擦卡滞[[S21,S27]] |
| 开模行程 | 斜销类机动抽芯机构中，抽芯距 S = H·tanα，受开模行程严格约束；油缸类则相互独立[[S39,S31,S41]] |
| 斜导柱倾角 | 常规α = 10°～25°，有效工作行程须根据抽芯距与倾角的三角函数关系匹配[[S42,S3]] |
| 顶出距离 | 抽芯完成后须为顶出动作让出完整脱模空间，避免型芯端面与铸件碰撞干涉[[S11,S6]] |

### 工程实例

在某铝合金箱体压铸模设计中，1号液压抽芯型芯位移 h = 260 mm，安全值 K = 20 mm，考虑组合芯序次抽芯的预留安全距离156 mm，最终抽芯距达 S₁ = 436 mm，为典型的油缸驱动大行程抽芯案例[[S44,S21]]。
