## 概述

在压铸语境下，“清理”是一个涵盖两大独立子领域的广义术语。依据国家标准GB/T 25370-2020的定义，铸造领域的“清理”是指铸件落砂或出型后，去除铸件表面粘砂、残留砂芯、浇冒口、飞边、毛刺、氧化皮等多余材料的过程总称[[S15]]。在压铸生产中，这一概念可明确划分为两大子范畴：**压铸模具清理**与**压铸件后清理**[[S22,S36]]。两者在对象、时机、目的和方法上存在显著差异，但均直接关系到最终产品质量、模具寿命及生产效率。

压铸全流程中，清理工序的劳动占比极高——铸件清理的工作量往往可达压铸本体工序的10~15倍[[S36,S31]]，是压铸生产中实现机械化、自动化以降本增效的关键环节。

## 定义与分类

### 压铸模具清理

指在单模或多批次生产间隔中，对型腔、分型面、浇口套、排气槽、活动型芯、流道等模具部件的清洁作业。目的是避免因残留金属屑、脱模剂积垢等导致的合模不严、通道堵塞等问题。按作业深度分为三级[[S22]]：

- **日常清吹**：每次开模后，利用喷涂装置的压缩空气喷吹型腔及分型面，必要时辅以手工清理。
- **常规维护清理**：单批次生产结束后，拆卸模板和顶出板，清除型腔、分型面附着的金属屑与脱模剂污物。
- **定期拆模深度清洁**：数个批次或预定生产数量完成后，对浇道、浇口、溢流槽、排气道损伤部位进行全面检修与深度清洁。

压铸工艺循环流程图明确将“清理模具”列为正式生产工序的第一个前置步骤[[S19]]。

### 压铸件后清理

指铸件从模具取出后，去除附带的多余部分，包括浇注系统、排溢系统、分型面飞边、型芯及顶杆等处的飞边、毛刺及印痕等的作业[[S36]]。具体可分为[[S15,S41]]：

- **去浇冒口**：去除浇注系统的料饼、横浇道、浇口等。
- **去飞边毛刺**：清除分型面飞边、型芯痕迹、顶杆痕迹等。
- **去氧化皮**：清除铸件表面氧化皮及锈蚀。
- **精整**：以改善铸件表面状况为主要目的的精细清理，包括打磨、修整等。

## 工艺作用与目的

规范执行清理作业对压铸全流程具有多重关键作用：

- **保障产品质量**：彻底去除铸件表面飞边、油污、氧化皮，可提升后续电镀、涂装、浸渗处理的界面结合力，保障铸件外观精度、装配适配性与气密性要求，避免残留毛刺引发装配划伤、应力集中开裂等缺陷[[S31]]。
- **延长模具寿命**：及时清除分型面残留金属、型腔表面脱模剂积垢，可避免合模应力不均、粘模拉伤、表面热疲劳裂纹萌生。NADCA技术资料明确指出，符合规范的定期模具清洁可将H13类铝合金压铸模具的服役寿命提升15%~30%[[S14]]。
- **提升生产效率**：采用机械化自动化清理设备替代人工粗放清理，可将单工位压铸单元的有效生产率提升20%以上，大幅降低后续返工返修的时间与人力成本[[S30]]。

## 清理方法分类

### 压铸件机械清理

#### 滚磨清理
将小型压铸件与磨料共同装入滚筒，通过铸件之间、铸件与磨料之间的相互碰撞摩擦去除表面毛刺与残留浮渣。适配小批量、结构简单的小型非铁合金压铸件。滚筒转速一般控制在20~30 r/min[[S38,S1]]。优点是成本低、操作简单；缺点是无法处理复杂内腔零件。

#### 抛喷丸清理
通过叶轮离心力抛射（抛丸）或压缩空气喷射（喷丸）钢丸（或磨料）高速冲击铸件表面，实现除氧化皮、去飞边与表面强化三重效果[[S15,S41]]。可分为抛丸、喷丸、抛喷丸联合清理三种模式。联合清理模式适合大型复杂压铸件，先由固定抛头完成大面积表面清理，再以手持喷枪补充处理内腔、死角[[S15,S3]]。

### 压铸模具清理

#### 干冰清理
一种无磨料新型机械清理工艺，以低温固态干冰颗粒为清理介质。高速喷射模具表面后，干冰瞬时升华，无二次残留[[S22]]。主要特点包括[[S21,S20]]：

- 可在模具无需完全冷却、不拆卸的状态下进行在线清理。
- 无磨料嵌入型腔风险，不损伤高精度模具工作面，基体无磨损。
- 可穿透清理直径0.7 mm以上、深度50 mm以上的模具排气孔。
- 干冰完全升华，无需二次清理，无化学污染，对环境友好。

### 压铸件化学清理

#### 溶剂化学清理
分为冷溶剂刷洗与溶剂气相脱脂两类。ASTM B252标准明确其适配抛光后锌合金压铸件的预清洁场景，可高效溶解残留抛光膏、油污。三氯乙烯、全氯乙烯类含氯溶剂的气相脱脂工艺清洁效率高，适配大批量电镀前预处理场景[[S26]]。

#### 碱性清洗
分为常温浸泡、热喷淋、电化学阳极（或阴极）清洗三级工艺。低浓度弱碱性配方可高效去除压铸件表面油脂残留[[S17]]。阳极电清洗工序的电流密度控制在1.6~3.2 A/dm²，处理时长25~45 s即可完全去除表面顽固附着物，适配非铁合金压铸件电镀前最终清洁工序[[S17]]。

#### 电化学清理（电解清理）
通过向碱性清理溶液中通入直流电，利用阴极析出的初生态钠还原铸件表面氧化层[[S3]]。适配结构复杂、深孔与细小内腔较多的中大压铸件的深度清理，常规工艺无法到达的内腔死角位置均可获得均匀清洁效果。

### 超声波清洗

依托高频声波在液体介质中产生空化效应，形成微小冲击作用，可在不损伤精密零件表面的前提下深入清理细小缝隙、盲孔位置的残留杂质[[S17]]。适配高精度小型压铸件、装配精密组件的最终清洁场景，可大幅缩短人工精细清理工时。

### 各类清理方法对比

下表汇总了典型清理方法的适用对象与主要特点：

| 清理方法 | 适用对象 | 主要优点 | 主要局限 |
|---------|---------|---------|---------|
| 滚磨清理 | 小型非铁合金压铸件 | 成本低，操作简单 | 无法处理复杂内腔零件 |
| 抛喷丸清理 | 大中型压铸件（通用） | 效率高，兼具表面强化功能 | 可能引起起皮、弹丸嵌入，需防死角 |
| 干冰清理 | 模具在线日常清洁 | 无磨损、无残留、在线作业 | 需专用设备与CO₂气源 |
| 溶剂化学清理 | 抛光后锌合金件（电镀前） | 除油效率高 | 需配套废气防护，有职业健康风险 |
| 碱性清洗+阳极电清洗 | 非铁合金件电镀前最终清洁 | 彻底去除顽垢 | 需严格控制电流与时间防止过蚀 |
| 电化学清理 | 复杂内腔中大压铸件 | 内腔清洁均匀 | 需导电碱性介质，设备投资较大 |
| 超声波清洗 | 高精度小型件/精密组件 | 无损精细清洁 | 设备成本较高，批量受限 |

## 工艺参数与操作条件

### 干冰清理核心参数

干冰清理模具的适配颗粒规格及喷射参数如下：

| 参数项目 | 规格/数值 |
|---------|----------|
| 干冰颗粒直径 | 2~3 mm（柱状颗粒）[[S20,S2]] |
| 干冰颗粒长度 | 2.5~10 mm[[S2,S9]] |
| 颗粒冲击速度 | 约300 m/s[[S20,S9]] |
| 所需空气流速 | 180~300 m/s[[S20]] |
| 作业温度（常压） | -78°C[[S20]] |

### 喷砂磨料规格

压铸件喷砂作业可选用磨料粒度覆盖24~300目区间。非铁合金压铸件优先选用玻璃砂、硅砂类磨料以避免铁质污染[[S18]]。

### 碱性电清洗参数

阳极电清洗工序参数如下[[S17]]：

| 参数 | 范围 |
|-----|------|
| 电流密度 | 1.6~3.2 A/dm² |
| 处理时间 | 25~45 s |
| 典型液温 | 70~82°C |

## 设备与工具

### 典型清理设备

- **干冰喷射机**：分为单喉管和双喉管两种类型。单喉管机型喷射速度可达音速的1.7~3.0倍，冲击力大、清理效果好，适合模具顽固污垢清洗[[S2]]。
- **抛丸/喷砂机**：形式包括转台式、滚筒式、履带式、通过式、回转式等，需根据铸件形状与产量选型。便携压送式自循环喷丸器适用于模具清理[[S43]]。
- **压力切边机**：分机械型与液压型两种。机械型动作快、效率高；液压型行程与压力可调，适合大型铸件，动作平稳无冲击[[S36]]。
- **超声波清洗槽**：适合油污较重的精密压铸件后处理[[S17,S7]]。

### 设备选型原则

压铸件清理设备选型需匹配生产场景需求[[S7,S31,S43]]：

| 生产场景 | 推荐设备类型 |
|---------|------------|
| 小批量多品种 | 手动/半自动化喷砂打磨设备 |
| 大批量标准件 | 自动化流水线抛丸机、切边压力机 |
| 模具日常维护 | 便携移动型干冰清洗机 |
| 精密件油污后处理 | 超声波清洗槽 |

## 清理效果评价

### 表面可清洁性指数（CI）

ASTM C756标准定义了表面可清洁性指数（Cleanability Index, CI）的测定方法。其原理是对测试表面施加定量荧光水溶性污染剂，经标准化擦拭后，以溶剂萃取残留污染物并测量荧光强度，再与标准参考表面进行对比。CI值计算如下[[S11,S10]]：

- **CI > 1.0**：被测表面比标准参考表面更难清洁。
- **CI < 1.0**：被测表面比标准参考表面更易清洁。

### 油污残留检测

压铸件清理后表面油污残留的验证可采用[[S42]]：

- **称重法**：测定洗油率，计算清洗前后油污重量变化百分比。
- **荧光分光光度法**：以环乙烷萃取表面微量残留油污，用荧光光度计或荧光分光光度计进行定量测定，适用于涂装、粘接等工序前的洁净度验证。

## 常见清理缺陷与影响

### 清理不当引发的缺陷

实际生产中，清理不当造成的主要质量缺陷包括缺肉、磕碰伤、变形、裂纹以及漏工序等。成因多为手工操作失误或清理工装状态不良。这类缺陷通常为易于检出的外观缺陷，可通过操作者100%自检配合首检、巡检、尾检的全流程检验制度管控[[S28,S33]]。

### 具体缺陷示例

- **浇口清除时机不当**：可能引发铸件变形、翘曲等缺陷[[S28]]。
- **清理过度**：使用砂带机、砂轮打磨时进给过猛会直接划伤铸件表面导致尺寸超差；抛丸清理工艺参数控制不当易出现铸件表面起皮、弹丸嵌入残留等缺陷[[S13,S41]]。
- **浇口清理操作不当**：可引发由封闭气体膨胀残留形成的铸件表面凸起缺陷，可通过优化浇口清理工艺、缩短内浇口予以消除[[S4]]。

## 安全与环境保护

### 核心安全风险

清理工部主要安全风险包括[[S13,S6,S29]]：

| 风险类别 | 来源 |
|---------|------|
| 粉尘吸入 | 砂轮机、砂带打磨产生的金属粉尘 |
| 爆炸风险 | 镁合金粉尘聚集遇明火 |
| 高压伤害 | 高压介质喷射清理 |
| 机械划伤 | 手工清理尖锐部位 |
| 高温烫伤 | 接触未完全冷却铸件 |

### 镁合金清理专项安全规范

GB 26488-2025《镁合金压铸安全生产规范》7.4清理条款明确要求[[S29]]：

- 镁合金铸件清理必须使用气动工具。
- 湿式除尘设备未开启时，禁止开展清理作业。
- 清扫积尘需采用无扬尘清扫方式、使用不产生火花的清扫工具。
- 收集的镁合金粉尘须做泡水氧化处理。
- 打磨区域动火作业前，要完全清理所有区域内的积尘，配备合规消防设施，办理动火作业票后方可挂牌作业。

### 个体防护要求

- 砂轮机、砂带机操作需佩戴合规防尘口罩[[S13]]。
- 抛丸清理作业必须佩戴有机玻璃防护面罩、防尘口罩、防滑工作鞋；非作业人员禁止靠近抛丸作业区域[[S32,S5]]。

### 环保要求

清理工部核心环保管控点包括：打磨抛丸粉尘达标排放、脱模剂残液合规回收处理、含金属渣屑废水经净化后循环利用[[S13,S12]]。压铸企业应等同遵循ISO14000系列标准，建立合规的环境管理体系[[S12]]。含脱模剂、液压油残留的清洗废水需先经污水井沉淀、粗过滤净化后进入冷却池循环使用，禁止未处理直接排入市政下水道[[S39]]。
