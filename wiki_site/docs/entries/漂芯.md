## 概述

漂芯（core floatation / core raised）是压铸及铸造过程中型芯在金属液热作用、充型压力及浮力共同影响下发生非预期上抬、位移、漂浮甚至断裂，致使铸件内孔位置偏错、形状与尺寸不符合图样要求的形状尺寸不合格类缺陷[[S21,S31]]。该缺陷在国家标准GB/T 5611-2017《铸造术语》中归入9.7“形状及重量差错类缺陷”，与偏芯（core lift）同属芯子位置偏移缺陷范畴，核心特征为型芯在竖直方向因浮力主导而漂浮移动[[S31,S15]]。

漂芯发生后典型外观表现为铸件一侧壁厚减薄甚至穿透，另一侧对应位置壁厚异常增厚；严重失效时型芯断裂上浮，铸件下部形成不规则帽状金属凸起，还可能伴随脱落芯块形成的砂眼[[S21]]。图GB/T41972-2022收录的漂芯示意可直接说明这一特征[[S4,S31]]。

## 定义与术语边界

### 标准定义

| 依据 | 术语 | 定义要点 |
|---|---|---|
| GB/T 5611-2017 9.7.8 | 偏芯 / 漂芯 (core lift / core raised) | 型芯在金属液作用下漂浮移动，使铸件内孔位置、形状和尺寸发生偏错，不符合铸件图要求[[S31]] |
| 英汉铸造词汇 | core floatation | 对应中文译名“漂芯 / 拖芯”[[S29,S15]] |
| 英汉铸造词汇 | core shift | 对应中文译名“偏芯”，指砂芯在分芯面处的错位[[S15,S11]] |
| 英汉铸造词汇 | core lift / core raise | 对应中文译名“抬芯 / 浮芯”[[S15,S16]] |
| 行业通用 | 型芯偏移 / 芯子偏移 (core displacement) | 泛指充型阶段型芯在非平衡力作用下发生的任意方向位置偏移，导致壁厚不均和尺寸超差[[S2,S13,S18]] |

### 术语区分

在铸造和压铸领域内，漂芯及相关术语存在明确的概念包含层级关系[[S24]]：

- **芯子偏移 (core shift / core displacement)**——广义大类，涵盖型芯所有水平及任意方向的位置偏移[[S11,S16]]；
- **偏芯 (core shift)**——特指砂芯在分芯面处相互错开的水平方向位移缺陷，区别于整芯漂浮[[S31,S11]]；
- **漂芯 (core floatation / core raised)**——金属液浮力主导、伴型芯沿竖直方向上浮运动的型芯失效类缺陷，属于偏芯的细分场景[[S31,S24,S15]]；
- **浮芯 / 抬芯 (core lift / core raise)**——仅描述型芯竖直抬起位移的狭义表述[[S15,S16,S24]]。

鉴别漂芯与错箱、错芯类缺陷的关键依据为：漂芯系由整芯偏移或漂浮运动导致，在砂芯与铸件的分芯接触面上不存在相互错开痕迹[[S21]]。

## 成因及力学机理

### 载荷分解

压铸充型及增压阶段作用于金属型芯的载荷包含三个主要分量[[S32,S22]]：

1. **冲击动压力**——高速金属液流直接冲击型芯工作面产生的瞬时动载荷[[S2,S30]]；
2. **静浮力**——熔融金属对型芯的持续上浮推力[[S2,S21]]；
3. **侧胀推力**——增压阶段作用于型芯成形端面投影面积上的分胀型力，沿楔紧角方向分解产生侧向推力[[S22,S6]]。

当三力合力超过型芯定位锁紧机构的承载阈值时，型芯即发生位移，形成漂芯。压力传递与胀型力生成的原理图示如下[[S6]]：

### 不同抽芯机构的漂芯成因差异

- **斜导柱抽芯机构**——侧向锁紧完全依赖固定在定模的楔紧块，分胀型力计算式为 \(F_1 = \sum [(A_1 p/10) \tan \alpha]\)。当金属液侧胀推力超过楔紧块锁紧极限时，型芯易发生退让位移引发漂芯，该机构无额外主动补偿力，漂芯风险相对更高[[S22,S32]]；
- **液压抽芯机构**——除楔紧块外，可借助液压抽芯器插芯力抵消部分侧胀推力，分胀型力计算式为 \(F_1 = \sum [(A_1 p/10) \tan \alpha - F_2]\)，相同工况下漂芯发生概率低于斜导柱机构[[S22,S32]]。

### 浇口位置与充型顺序

浇口直接正对型芯工作面时，高速金属液流持续冲击型芯单侧，造成型芯两侧受力严重不对称，显著提升漂芯概率；充型流态不均匀、远端区域金属液充填时差过大，会在不同时刻对型芯不同位置施加非同步侧向载荷，进一步加剧偏移风险[[S2,S30]]。

## 影响因素

### 工艺参数维度

| 影响因素 | 作用机制 | 来源 |
|---|---|---|
| 快压射速度过高 | 增大金属液对型芯的冲击动载荷 | [[S13,S20,S14]] |
| 增压压力过大 | 增大侧向分胀型力，可能突破抽芯机构锁紧阈值 | [[S13,S20,S14]] |
| 浇注温度过高 | 降低型芯局部刚度，同时增大金属液浮力 | [[S13,S20,S14]] |
| 模具温度分布不均 | 型芯热变形不对称，引发初始预偏移 | [[S13,S20,S14]] |
| 锁模力不足 | 无法抵消总胀型力，带动动定模局部相对位移间接引发漂芯 | [[S13]] |

### 型芯设计维度

| 影响因素 | 作用机制 | 来源 |
|---|---|---|
| 型芯长径比 > 5 | 型芯刚性急剧下降，挠曲变形量显著增大 | [[S8,S25,S17]] |
| 导向段长度不足 | 支撑刚度降低，抗侧向载荷能力大幅下降 | [[S8,S25,S17]] |
| 配合间隙过大 | 预留侧向位移活动空间，为漂芯直接诱因 | [[S8,S25,S17]] |
| 材料选型/热处理不当 | H13 钢未氮化处理提升表面硬度与整体强度，易弯曲磨损引发位置偏移 | [[S8,S17]] |

### 设备状态维度

- 抽芯器设定锁紧力不足，无法抵消侧向胀型推力，直接导致抽芯滑块退让[[S13]]；
- 压铸机合模机构刚性不足、四根大杠受力不均，动定模板出现平行度偏差，带动模具内型芯位置整体偏移，诱发大面积漂芯问题[[S13]]。

## 后果与表征

漂芯引发的主要质量后果包括[[S2,S9,S27]]：

- 铸件壁厚分布严重不均，局部区域壁厚低于图纸最小壁厚阈值；
- 内孔位置偏移、孔位错位、同轴度丧失、整体尺寸超差；
- 诱发孔偏次生缺陷，导致后续机加工时光孔、螺纹孔出现露黑皮、装配干涉等问题。

可检出特征为：目视检测可观察到铸件开放内壁轮廓错位、非加工面局部壁厚异常[[S19,S21]]；通过三坐标测量仪、壁厚仪等可定量测得内孔偏移量、同轴度偏差及壁厚实际值与设计值之差，精准判定漂芯严重程度[[S19,S21]]。

## 缺陷鉴别

漂芯的鉴别方法为肉眼外观直接检查，核心区分特征是缺陷由整芯偏移或漂浮运动导致，在砂芯与铸件的分芯接触面上不存在相互错开痕迹，可据此与错箱、模具分型偏移类类似缺陷直接区分[[S21]]。

漂芯的完整形成原因包括9类：起模操作不当导致芯座尺寸过大或芯头修削过量致配合间隙超标、下芯过程型芯初始定位偏移、细长型芯的芯骨刚度不足与芯头支撑面积过小、芯撑选型或安放不当、芯砂耐火度/高温强度不足及烘干工艺不当、高吊砂/高砂台结构设计不合理未加固、起模失误导致高凸起砂结构断裂、浇注系统设计不合理（直浇道过高致静压力过大、内浇道直冲型芯）、浇注温度/速度/高度过高导致芯撑过早熔化[[S21]]。

## 防治措施

### 模具设计层面

- 增设型芯双侧定位结构，消除自由端悬臂效应[[S1,S17]]；
- 优化楔紧块结构，增大接触承压面积提升锁紧载荷上限[[S1,S17]]；
- 型芯导向段长度加长至型芯直径的2~3倍，提升侧向支撑刚度[[S1,S17]]；
- 型芯易受冲刷部位采用高硬度氮化H13钢镶件替换常规材料，降低挠曲变形概率[[S17]]。

### 工艺层面

- 优化压射曲线调整快压射切换位置，避免金属液在充型前段达到过高流速直接冲击型芯[[S20,S13]]；
- 匹配铸件投影面积校核填充压力，将侧向推力控制在锁紧承载阈值以内[[S20,S3,S13]]；
- 严格按照总胀型力的1.2~1.25倍安全系数核算锁模力，避免锁模不足引发模体位移[[S3]]。

### 运维层面

建立定期点检机制，每生产预设模数后检查型芯工作面磨损量、导向部位配合间隙变化值、抽芯滑块锁紧面磨损情况，对磨损超差零件及时更换维护[[S13,S7]]。

### 综合防治清单

针对漂芯缺陷，业界已总结系统性的12项防治措施[[S5]]：

1. 正确设计芯头尺寸，修削芯头适度，保证芯头与芯座间有适当的间隙和足够的支撑面积
2. 加大芯骨尺寸，提高芯砂的耐火度和高温强度
3. 适当增加湿型砂黏结剂含量，提高型砂湿强度，芯座处保证足够紧实度
4. 芯盒磨损后及时修复，防止芯头尺寸变小
5. 下芯位置正确，防止偏斜
6. 正确选择芯撑材料、类型、尺寸和数量，安放位置适当；位于砂芯上表面的芯撑要压紧，避免过早接触充型金属液
7. 合理设计浇注系统，适当降低直浇道高度，防止内浇道直冲砂芯、砂台和芯撑
8. 起模操作正确，不过分敲击模样，以免芯座尺寸变大或砂型产生裂纹
9. 砂型和砂芯烘干工艺正确，防止加热过快、温度过高，防止烘干不足、过烘或返潮
10. 高砂台和深吊砂使用砂钩和铁钉加固
11. 合型前修补砂型和砂芯裂纹，并插钉加固
12. 适当降低浇注温度、浇注速度和浇注高度

### 缺陷补救规则

- 非加工面的轻微漂芯可直接打磨或焊补后打磨平整[[S5]]；
- 加工面的漂芯形变量未超出预留加工余量的可直接机加工去除；超出余量允许范围的可焊补后再机加工[[S5]]；
- 不可焊补或无法通过机加工消除的漂芯缺陷需直接报废铸件[[S5]]。

## 相关关系

漂芯与以下压铸工艺要素存在直接逻辑关联：

- **模具刚性**——模具刚性不足时，金属液填充过程中模板发生抖动，显著增大漂芯发生概率[[S12,S28]]；
- **冲头前进稳定性**——压室冲头与压室配合异常、压射中前进速度不平稳，导致充型冲击力出现非预期波动，引发型芯偏移[[S12,S28]]；
- **脱模斜度**——过小的型芯侧脱模斜度导致脱模阻力过大，开模抽芯过程中受拉力作用易发生位置偏移，保证标准要求的脱模斜度可有效降低漂芯风险[[S27]]。
