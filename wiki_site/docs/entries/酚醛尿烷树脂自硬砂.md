## 概述

酚醛尿烷树脂自硬砂是一种有机树脂自硬砂，由双组分苯基醚酚醛树脂和聚异氰酸酯添加胺类催化剂混制而成，室温下无需加热即可自行硬化[[S8,S1]]。该工艺于1968年由美国Ashland化学公司开发，因粘结剂系统中采用专利的聚苄醚酚醛树脂（Poly-benzylic Ether Phenolic，简称PEP），国际上通称为Pep Set法[[S19,S17]]。

标准英文术语为 **pep-set no-bake sand** 或 **phenolic urethane no-bake sand**，通用缩写为 **PUNB**[[S8,S40]]。中文别名包括**酚醛脲烷自硬砂**、**酚醛聚氨酯自硬砂**，亦可简称为**酚脲烷树脂砂**[[S8,S11]]。

## 分类定位

根据GB/T 5611—2017《铸造术语》，酚醛尿烷树脂自硬砂属于有机化学硬化砂中的**自硬砂**（no-bake sand）类别[[S8]]。在树脂自硬砂三大分类体系中，它与酸固化（呋喃或酚醛）树脂自硬砂、酯固化碱性酚醛树脂自硬砂并列[[S13,S38]]。

与酚醛尿烷冷芯盒砂（phenolic urethane cold box，PUCB）的定义边界明确：

| 对比维度 | 酚醛尿烷树脂自硬砂（PUNB） | 酚醛尿烷冷芯盒砂（PUCB） |
|----------|---------------------------|--------------------------|
| 硬化方式 | 室温自硬，预先混入液态胺类催化剂 | 气硬，需向砂型/芯中吹入汽化叔胺触发硬化 |
| 催化剂 | 液体叔胺（混砂时加入） | 汽化叔胺（三乙胺、二甲基乙胺等，造型后吹入） |
| 后处理 | 无需后处理 | 需通入压缩空气扫除残留胺蒸气 |
| 树脂加入量 | 总粘结剂0.8%~1.8%（砂重） | 铸铁件约1.5%，铝合金约0.8%（砂重） |

[[S22,S31,S42,S8]]

## 组成与成分

### 粘结剂体系

采用三组分粘结剂体系[[S19,S16]]：

**组分I — 聚苄醚（苯基醚）型含羟基酚醛树脂**：由苯酚和甲醛在特定条件下缩合得到，可溶解于高沸点芳香烃类溶剂[[S21,S17]]。

**组分II — 聚异氰酸酯**：异氰酸酯基团数目为2~5个。国内常用多亚甲基多苯基异氰酸酯（PAPI），国外常选用二苯基甲烷二异氰酸酯（MDI），可使用20%~50%的非极性溶剂稀释以降低粘度[[S21,S18]]。

**组分III — 液体叔胺催化剂**：催化组分I与组分II之间的加成聚合反应。反应体系适配pH范围为4~11[[S17,S21]]。国内常用品种包括苯丙吡啶、N,N-二甲基乙醇胺等[[S21,S17]]。

### 原砂

优先选用圆形颗粒的硅砂，要求含泥量低、角形系数小于1.3[[S20,S40]]。原砂粒度选用规则如下[[S19,S26]]：

| 铸造合金类型 | 推荐筛号 |
|-------------|---------|
| 铸钢件 | 40/70筛号 或 40/70与50/100筛号各半 |
| 铸铁件 | 50/100筛号 或 70/140筛号 |
| 非铁合金铸件 | 70/140筛号 或 140/70筛号 |

**原砂含水量控制**为关键要求：最佳含水量质量分数&lt;0.1%，允许范围≤0.2%。水分会与异氰酸酯发生副反应，降低型砂流动性、终强度以及可使用时间与起模时间的比值[[S19,S26,S20]]。

### 辅助材料

可添加**有机硅烷偶联剂**提升粘结剂与砂粒表面的界面结合强度，使树脂砂强度提升1~2倍[[S16]]。实际生产中为改善高温强度和减少表面缺陷，可加入氧化铁粉（铸铁件砂重的0.25%~2.0%，铸钢件2%~3%）[[S26]]。为改善退让性，可加入砂重2%~3%的木粉等溃散剂[[S29]]。

## 硬化机理

在叔胺催化剂作用下，酚醛树脂结构上的活性羟基（-OH）与聚异氰酸酯的异氰酸酯基团（-NCO）发生**加成聚合反应**，生成氨基甲酸乙酯（聚氨酯）结构，最终形成三维交联网络实现砂粒粘结[[S18,S17]]。

该反应的硬化特性[[S17,S40]]：
- 硬化全过程**无小分子副产物**生成；
- 砂型内外**同步硬化**（整体硬化），不存在软区，硬透性优异；
- 几乎不释放反应热量，硬化曲线在敞开和封闭环境下几乎完全重合。

| 影响因素 | 推荐/最佳值 | 影响规律 |
|----------|------------|---------|

### 热解与溃散特性

树脂组分在400℃左右即可完全分解，500℃条件下残留抗压强度几乎降至零，溃散性优异，清砂难度低[[S25,S35]]。

### 再生回用

- 热法再生工艺：旧砂回用率可达90%[[S10,S25]]；
- 机械强力摩擦再生：西欧传统工艺回用率为57%~87%[[S10]]；
- 再生砂控制指标：微粉质量分数≤0.8%，灼烧减量≤3.0%，水分≤0.2%，砂温20~30℃[[S25,S26,S10]]。

## 应用领域

### 适配铸造合金

酚醛尿烷树脂自硬砂适用于铸钢、灰铸铁、球墨铸铁、铝合金、镁合金、铜合金等各类黑色及有色铸造金属[[S19,S25,S6]]。

### 典型应用场景

特别适合批量较大的中小尺寸复杂铸件生产，代表铸件类型包括[[S13,S25,S19]]：
- 汽车、农机发动机的**缸体、缸盖**；
- 各类**液压阀体**等精密砂型铸件。

该工艺在北美应用普遍，中国使用范围正在扩大[[S13,S38]]。

### 与压铸工艺的关系

目前**未发现**酚醛尿烷树脂自硬砂在工业量产压铸场景中的主流直接应用记录[[S7]]。其与压铸工艺的间接关联体现为：
- 可用于低压铸造、压铸件试制阶段的砂型/砂芯制作；
- 替代其他砂型工艺完成复杂结构压铸件的小批量验证[[S7]]。

## 标准与规范

### 术语与定义标准

- **GB/T 5611—2017《铸造术语》**：给出了酚醛尿烷树脂自硬砂（条目4.5.24）的标准定义、英文名称及特性注记[[S8]]。

### 试验方法标准

- **JB/T系列铸造行业标准**：规定了树脂自硬砂的常规性能检验方法，采用标准“8”字型试样检测不同硬化时间的小时强度与24h终抗拉强度，测试流程与普通自硬砂检测规范一致[[S39]]。

### 产品标准

- 2009年制定的铸造用酚脲烷树脂国家标准，将铸造用酚脲烷自硬树脂按使用条件分为**普通型**和**高强度型**两类（PUZ-P型和PUZ-G型）[[S17]]。

### 安全规范

- 美国AFS铸造协会发布的《铸造粘结剂安全操作规范》对酚醛尿烷类自硬砂的操作安全提出要求：明确聚异氰酸酯为呼吸道致敏物质，作业场所需配备局部排风设施，需使用符合NIOSH标准的呼吸器、防护手套与护目装备[[S33,S27,S15]]。

## 常见缺陷与对策

| 缺陷类型 | 产生机理 | 防治措施 |
|----------|---------|---------|
| **氮气孔/针孔** | 组分II含氮6.0%~7.6%，浇注时氮化合物分解，氮、氢溶入液态金属，冷却时析出形成气孔[[S26,S29]] | 降低树脂加入量；下调组分II比例（52.5:47.5至60:40）；添加氧化铁粉（铸铁件0.25%~2.0%，铸钢件2%~3%）；强化排气[[S26]] |
| **表面光亮碳/皱纹** | 浇注时粘结剂热解产生碳氢化合物，碳沉积在铸件表面[[S26]] | 提高浇注温度；缩短浇注时间；加强型腔排气；添加氧化铁粉（砂重1.5%~2%）[[S26,S40]] |
| **机械粘砂** | 树脂在500℃左右完全热解，砂粒间粘结点烧蚀形成间隙，金属液渗入[[S32,S29]] | 提高型砂热强度；选用铬铁矿砂等高耐火骨料；涂刷高耐火度涂料；适当降低浇注温度[[S32,S29]] |
| **热裂** | 树脂膜焦化形成刚性碳化骨架，型砂高温退让性不足，阻碍铸件收缩[[S29]] | 添加木粉等溃散剂（砂重2%~3%）；在满足强度要求下尽量降低树脂加入量[[S29]] |
| **型（芯）砂硬化不良** | 各组分定量不正确；催化剂加入量偏低；原砂含水量过高；砂温/芯盒温度偏低[[S26]] | 校准各组分配比；严格控制原砂含水量和砂温[[S26]] |
| **可使用时间过短** | 砂温偏高；催化剂加入量过多；混砂时间过长[[S26]] | 控制砂温21~27℃；选用活性适当的催化剂牌号；优化混砂工艺[[S26,S40]] |

## 安全与环保

### 健康危害

- 组分II所用的聚异氰酸酯（PAPI/MDI）属于**刺激性物质**，接触皮肤或吸入蒸汽可引发皮肤刺激、呼吸道刺激，长期反复接触可诱发**过敏性哮喘**反应[[S21,S27,S15]]。
- MDI为已知的**呼吸道致敏物质**[[S15,S27]]。
- 混砂、造型、浇注过程中会产生含游离酚和异氰酸酯的烟气[[S25]]。

### 防护措施

| 防护类型 | 要求 |
|----------|------|
| 工程控制 | 混砂、浇注、落砂工位设置**局部排风系统**[[S19,S33,S15]] |
| 个人防护 | 使用符合NIOSH标准的呼吸器、防护手套、护目装备[[S33,S15]] |
| CO控制 | 合理工况下工位CO排放浓度可控制在50×10⁻⁶以下[[S19,S37]] |
| 粉尘控制 | 配套除尘系统，将粉尘排放控制在国家职业卫生标准允许范围内[[S19]] |
| 噪声控制 | 车间整体噪声可降至85dB以下[[S19]] |

### 废砂处理

废砂再生主要采用**机械强力摩擦再生**和**热法再生**两类技术，旧砂回用率可达57%~90%[[S10,S25]]。再生砂需控制灼烧减量&lt;3.0%、微粉含量≤0.8%，方可满足生产复用要求[[S25,S26]]。
