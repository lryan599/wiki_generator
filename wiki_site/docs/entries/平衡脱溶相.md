## 定义

平衡脱溶相是铝合金过饱和固溶体脱溶序列的最终热力学稳定产物，其成分与晶体结构完全符合对应合金系平衡相图中的稳定相特征，在所有时效析出相中自由能最低，与铝基体一般为完全非共格界面，形核位置多为晶界、晶格缺陷等高界面能处，界面能远高于GP区和亚稳过渡相[[S57,S13,S14]]。当合金进入过时效阶段后，平衡脱溶相的析出与粗化导致共格应变完全消失，合金强度与硬度持续降低[[S23,S43]]。

## 脱溶序列中的位置

在压铸可热处理铝合金的常规时效脱溶序列中，平衡脱溶相位于GP区、亚稳过渡相之后。完整的通用脱溶序列为：

> 过饱和固溶体 → GP区 → 亚稳过渡相 → 平衡脱溶相

不同合金系的平衡相具有明确的对应化学式[[S10,S42,S17]]：

| 合金系 | 脱溶序列 | 平衡脱溶相 |
|--------|----------|------------|
| Al-Cu | GP区 → θ'' → θ' → θ(CuAl₂) | θ (CuAl₂) |
| Al-Mg-Si | GP区 → β'' → β' → β(Mg₂Si) | β (Mg₂Si) |
| Al-Cu-Mg | GP区 → S' → S(Al₂CuMg) | S (Al₂CuMg) |
| Al-Zn-Mg | GP区 → η' → η(MgZn₂) | η (MgZn₂) |

## 与GP区和亚稳过渡相的边界

平衡脱溶相与脱溶序列中其他产物的核心差异如下[[S58,S51,S19,S13,S8,S12]]：

| 对比维度 | GP区 | 亚稳过渡相 | 平衡脱溶相 |
|----------|------|-----------|-----------|
| 晶体结构 | 与基体完全一致 | 介于基体与平衡相之间 | 完全独立的结构 |
| 与基体界面关系 | 完全共格 | 半共格 | 完全非共格 |
| 热力学稳定性 | 最低 | 中等 | 最高（自由能最低） |
| 形核方式 | 均匀形核 | 均匀或不均匀形核 | 非均匀形核（晶界/缺陷处） |
| 尺寸 | 1～5 nm | 细小弥散 | 粗大，10³～10⁴ Å |
| 形成温度区间 | 室温～150°C | 150～250°C | 通常＞200°C或更长时间 |
| 强化贡献 | 共格应变强化 | 主要强化相，峰值强度来源 | 无共格强化效应，导致过时效软化 |

## 晶体结构与形貌特征

### Al-Cu系（θ-Al₂Cu）

Al-Si-Cu压铸合金的平衡脱溶相为θ-Al₂Cu，属于四方晶系，理论晶格常数a=6.07 Å、c=4.87 Å，典型微观形貌为块状、共晶状、短棒状或层片状，与α-Al基体完全非共格[[S35,S15,S37,S9]]。θ相在高于300°C的温度区间析出，高于过渡相θ'的稳定区间[[S57,S53,S33]]。

### Al-Mg-Si系（β-Mg₂Si）

Al-Si-Mg压铸合金的平衡脱溶相为β-Mg₂Si，属于面心立方晶体结构，平衡态下典型微观形貌为角状蛛网状、鱼骨状或粗大的块状，与α-Al基体完全非共格[[S35,S20,S55]]。在高于β'过渡相稳定温度的区间析出，时效温度升高至接近共晶温度区间时由过渡相β'无扩散转变得到[[S57,S41]]。

### Al-Cu-Mg系（S-Al₂CuMg）

Al-Cu-Mg系平衡脱溶相S相，化学式为Al₂CuMg，晶体结构为斜方晶系，析出起始温度为370°C，完全析出温度为140°C，平衡相S形成后与铝基体完全失去共格性[[S57,S27,S28]]。

### Al-Si-Cu-Mg系（Q相及多相共存）

Al-Si-Cu-Mg压铸合金的平衡脱溶相包括β-Mg₂Si、θ-Al₂Cu以及Q相（Q-Al₅Cu₂Mg₈Si₆）。其中Q相属于密排六方晶系p62m空间群，理论晶格常数a=10.32 Å、c=4.05 Å，析出温度约为505°C左右，通常分布在θ-Al₂Cu相附近[[S11,S41,S56]]。随Cu含量从0.15 wt.%提升至0.45 wt.%，Q相形貌从细小分散的颗粒状逐步转变为块状、短棒状，高Cu含量下可形成片状、树枝状形态，与α-Al基体完全非共格[[S35,S44,S36,S32,S24,S9]]。

上述三类压铸Al-Si合金的脱溶序列均为过饱和固溶体→GP区→亚稳过渡相→平衡脱溶相，平衡相阶段所有脱溶产物均完全失去与铝基体的共格关系[[S35,S44,S57,S38,S63]]。

## 形成条件：由亚稳相到平衡相的转变

平衡脱溶相的形成是过饱和固溶体自发追求最低自由能状态的热力学必然结果，但在动力学上受温度和时间条件的控制。

### 热力学基础

脱溶时不直接析出平衡相的原因，是平衡相一般与基体形成新的非共格界面，界面能大；亚稳定的脱溶产物往往与基体完全或部分共格，界面能小。相变初期新相比表面积大，界面能起决定作用，界面能小的相形核功小，容易形成，因此首先形成形核功最小的过渡结构，再演变成平衡稳定相[[S8,S54]]。Al-Cu合金中平衡相θ的自由能在所有析出相中最低，是其作为最终稳定产物的热力学根本原因[[S13]]。

### 过时效触发条件

过时效过程中平衡脱溶相析出、粗化的触发条件包括[[S23,S52,S26]]：

1. **时效温度过高**：时效温度显著高于合金设计的人工时效上限，超过亚稳相的稳定区间。一般铝合金时效温度超过200°C后亚稳相快速溶解，平衡相优先形核长大。
2. **保温时间过长**：在常规人工时效温度区间保温时间远超峰值时效所需时长，亚稳相不断溶解，作为溶质供给平衡相持续长大粗化，析出相颗粒间距增大。
3. **时效温度与序列关系**：同一成分合金，时效温度越高越容易析出平衡相。高温时效时，预脱溶阶段或过渡相可能不出现或出现的过渡结构较少；低温时效时，则可能停留在GP区或过渡相阶段[[S18]]。

## 对性能的影响

### 强度与硬度：过时效软化

随着平衡脱溶相的析出和粗化，合金强度与硬度先升高至峰值后逐步降低，延伸率则随过时效的推进逐步回升，整体变化趋势与硬度曲线呈反向对应关系[[S62,S22]]。

过时效阶段的核心软化机制为**Orowan绕过机制**：完全非共格的平衡脱溶相不断粗化，相间距显著增大，位错不再需要剪切析出相，仅通过绕流即可运动，所需切应力大幅下降；同时共格应变场完全消失，合金强度与硬度持续降低[[S43,S59]]。位错移动所需的切应力与析出相颗粒间距成反比，颗粒间距越大，强度越低[[S23]]。

### 延伸率

当析出相细小共格时，位错可轻松穿过，塑性变形分散均匀，延伸率较高；峰值时效时亚稳相呈半共格，位错运动受阻严重，延伸率最低；进入过时效后非共格平衡相使位错可绕过，部分延伸率得以恢复[[S62]]。

### 耐蚀性

合金进入过时效阶段后耐蚀性得到提升：峰值时效下合金因亚稳相与基体电位差大，晶间腐蚀、应力腐蚀敏感性最高；平衡脱溶相充分粗化后，晶界与晶内的电位差显著减小，腐蚀速率降低，抗应力腐蚀性能明显改善[[S47]]。

### 尺寸稳定性

平衡脱溶相的完全析出与粗化过程可消除压铸合金中残留的亚稳相和淬火应力，抑制后续服役过程中的相变与组织演变，显著提升合金的尺寸稳定性[[S3]]。

## 压铸工艺控制要点

### 常规工业操作窗口

压铸常用A356类Al-Si系可热处理合金的固溶+人工时效常规工业操作窗口为：固溶处理温度区间510～550°C，保温时长1～8 h，淬火转移时间控制在8～30 s；后续人工时效温度控制在150～190°C区间，保温时长3～12 h。该参数范围内析出相以GP区和亚稳过渡相为主，可获得峰值附近的强度与塑韧性匹配的综合力学性能[[S40,S21,S49,S4]]。

### 避免过时效的工艺控制

针对真空压铸、低压铸造工艺制备的低气孔率可热处理铝合金铸件，避免过早析出平衡脱溶相导致性能不合格的核心控制要点包括[[S39,S16,S34]]：

1. **双级时效方案**：采用低温短时双级时效，如170°C/2 h + 185°C/1 h工艺，既保证亚稳强化相充分析出，又避免长时间高温保温触发平衡相形核。
2. **短固溶+短时效路线**：真空压铸专用热处理可采用476°C/22 min短固溶+159°C短时效的工艺路线，抑制平衡相析出的同时避免铸件起泡、变形。
3. **炉温均匀性控制**：严格控制时效炉温均匀性，杜绝局部超温区域导致的晶界平衡相优先析出。

### 国家标准中的热处理状态划分

GB/T 1173系列铸造铝合金国家标准将人工时效分为：不完全人工时效（T5）、完全人工时效（T6）、稳定化处理（T7）、软化处理（T8）。其中T8状态下允许平衡脱溶相部分析出聚集以提升铸件塑性，其余常规强化状态需严格控制时效参数，避免平衡相大量析出导致力学性能达不到标准要求[[S4,S61,S49]]。

## 相关术语关系

| 术语 | 与平衡脱溶相的关系 |
|------|-------------------|
| 过饱和固溶体 | 脱溶序列的初始状态，经固溶处理+淬火获得，是平衡脱溶相的母相[[S60,S6]] |
| 固溶处理 | 将合金加热至接近固相线温度后快速淬火，获得过饱和固溶体，为后续析出提供热力学驱动力[[S60,S64]] |
| 自然时效 | 室温下时效，优先析出GP区，不会产生大量亚稳过渡相与平衡脱溶相[[S17]] |
| 人工时效 | 高温时效，随温度升高和时间延长依次析出亚稳过渡相及平衡脱溶相[[S60,S64]] |
| 析出硬化（沉淀硬化） | 由GP区和亚稳过渡相的共格/半共格应变场实现的强化机制，平衡脱溶相不产生此效应[[S13,S14]] |
| 过时效 | 平衡脱溶相大量析出、粗化导致的强度与硬度下降阶段，标志共格强化效应完全消失[[S23,S62]] |
