## 概述

高磷铸铁是一类以磷为主要合金元素的特种耐磨灰铸铁，通过在珠光体灰铸铁基体中添加磷，使其在组织中形成坚硬的磷共晶骨架，从而显著提升材料的减摩耐磨性能[[S8,S27]]。高磷铸铁分为普通高磷铸铁和合金高磷铸铁两类，后者额外添加Cu、Ti、Cr、Mo等元素以进一步优化力学与耐磨性能[[S8,S17]]。

## 定义与分类

高磷铸铁的标准定义为含磷质量分数0.35%~0.65%的灰铸铁[[S8,S17]]。另一种常见定义将其磷含量区间表述为0.4%~0.75%，国内的MTP系列高磷耐磨铸铁明确将磷含量范围设定为0.4%~0.65%[[S27,S24,S13]]。磷元素在铸铁中以磷共晶形式存在，呈断续网状分布，形成坚硬骨架，使铸铁具有良好的耐磨性[[S8]]。

### 与其他铸铁的区别

高磷铸铁与其他常用铸铁的核心差异如下[[S14,S16,S22,S19]]：

| 铸铁类型 | 磷含量典型值 | 抗拉强度范围（MPa） | 耐磨性特征 |
|----------|--------------|---------------------|------------|
| 普通灰铸铁 | ≤0.3% | 100~350 | 耐磨性远低于高磷铸铁 |
| 孕育铸铁 | ≤0.2% | 较高，经孕育处理 | 耐磨性低于高磷铸铁 |
| 蠕墨铸铁 | ≤0.1% | 360~440 | 耐磨性优于孕育铸铁与高磷耐磨铸铁，导热性和抗氧化性更优 |
| 高磷铸铁 | 0.35%~0.65% | 150~300 | 耐磨性显著优于普通灰铸铁 |
| 合金铸铁 | 不固定 | 按目标性能调整 | 磷仅为可选添加组分之一 |

### 牌号与磷含量

国内GB/T 9439《灰铸铁件》标准未单独设置高磷铸铁专属牌号，高磷铸铁作为特种耐磨灰铸铁，一般在HT200~HT300常规灰铸铁牌号基础上按需将磷含量提升至0.35%~0.65%，供需双方可在订货协议中约定化学成分验收要求[[S12,S25]]。

国外部分连续铸造用高磷铸铁的磷含量上限可根据供需协议放宽至1.5%，在快速冷却的连续铸造工艺下，该成分铸铁可获得244~276 MPa的抗拉强度，同时兼具优异的致密性和液压气动承压密封性能[[S26,S30]]。

## 显微组织

### 磷共晶的类型

磷共晶按其组成分为二元磷共晶、三元磷共晶、二元磷共晶-碳化物复合物及三元磷共晶-碳化物复合物四类[[S6,S23]]。二元磷共晶由Fe₃P与α-Fe组成，硬度适中；三元磷共晶硬度过高，易碎裂脱落成为磨料，会加剧工件磨损。高磷铸铁生产中一般优先控制得到二元磷共晶，避免生成三元磷共晶[[S11,S18]]。

### 磷共晶的分布形态

磷共晶在铸铁中的分布形态按GB/T 7216分为四类[[S6,S23]]：
- 孤立块状
- 均匀分布
- 断续网状
- 连续网状（需严格避免的有害组织）

高磷铸铁理想的磷共晶组织形态为断续碎网状，细小且均匀分布，在实现高耐磨属性的同时尽可能降低材料脆性[[S11,S18]]。

### 磷共晶数量与磷含量的关系

磷含量与磷共晶数量的大致对应关系及组织特征[[S12,S15]]：

| 磷含量范围 | 磷共晶分布形态 |
|------------|-----------------|
| ＜0.05% | 仍可能出现孤立块状（因磷偏析） |
| ＜0.3% | 孤立块状分布 |
| 0.3%~0.45% | 大小不等的块状分布 |
| ＞0.45% | 断续网状分布 |
| 更高 | 连续网状分布 |

## 性能特性

### 耐磨性

高磷铸铁中呈网状分布在珠光体基体上的硬质点磷共晶形成典型的软基体镶嵌硬耐磨相的多相结构，耐磨性显著优于普通灰铸铁，可在润滑工况下大幅降低摩擦副的磨损速率[[S27,S8]]。合金高磷铸铁通过添加铬、钼、铜、钛等元素进一步细化组织，同步提升强韧性和耐磨性能[[S27]]。

### 导热性

磷元素对灰铸铁导热性能存在小幅提升作用，常规灰铸铁基准导热系数约为46.05 W/(m·K)，磷元素提升不会明显降低铸铁导热能力。在100℃~400℃常规服役区间内，铸铁导热系数随温度升高整体呈下降趋势，可适配连续服役的散热需求[[S29]]。

## 工艺适配

### 在压铸领域的潜在应用

高磷铸铁的高耐磨属性与压铸设备动摩擦部件需求匹配，可用于制造压铸机滑动导轨、模具定位块等低冲击高频滑动的耐磨构件，同时适配压射冲头、压射室等需要对抗低熔点有色合金金属液冲刷与反复滑动磨损的工况，可作为低成本耐磨铸铁替代材料应用于常规铝合金、锌合金压铸生产线中[[S21,S5]]。

### 传统应用场景

高磷铸铁长期广泛用于普通机床领域制造床身、滑动导轨、工作台等需要耐磨滑动的构件，同时是内燃机行业制造缸套、活塞环等高负载耐磨部件的经典用材，行业应用成熟度高[[S24,S13]]。

### 热处理

高磷铸铁可参考通用灰铸铁时效工艺开展去应力退火，通过可控升温、保温、缓冷流程消除铸造残余内应力，避免后续加工和服役过程中出现变形开裂。此外，可通过石墨化退火工艺调整基体组织，消除可能析出的白口硬脆相，优化磷共晶的分布状态[[S10,S18]]。

## 缺陷与限制

高磷铸铁存在以下固有缺陷和使用限制[[S18,S4]]：

1. **白口倾向**：高磷含量会提升铸铁白口生成倾向，在碳硅含量偏低或薄壁工况下容易析出硬脆碳化物，提升机加工难度。
2. **缩松缺陷**：磷元素存在严重的晶界偏析特性，高磷相富集在奥氏体枝晶交界位置，大幅提升晶界缩松、夹杂缺陷的发生概率。
3. **冷脆风险**：分布在晶界的硬脆磷共晶会破坏金属基体连续性，显著降低材料韧性，提升低温脆性。硅磷总含量超标时，厚大断面构件服役过程中冷脆裂风险远高于普通灰铸铁，必须严格控制磷含量以防止冷脆性[[S4]]。
4. **三元磷共晶危害**：三元磷共晶极硬而脆，易碎裂脱落形成磨料，加剧工件磨损，生产中应优先保证获得二元磷共晶[[S11,S18]]。

高磷铸铁理想的磷共晶结构应是断续碎网状、细小而均匀，以在实现高耐磨属性的同时尽可能降低材料脆性[[S18]]。

## 相关关系

高磷铸铁在铸铁分类体系中属于特种性能铸铁中的耐磨铸铁系列，与普通灰铸铁、孕育铸铁相比以磷共晶耐磨相为核心特征，与蠕墨铸铁相比导热性、韧性和耐热疲劳性能处于劣势，但成本较低且工艺成熟。在通用机床和内燃机耐磨构件领域，高磷铸铁可与其他耐磨铸铁（如磷铜钛铸铁、铬钼铜铸铁、钒钛铸铁等）通过力学性能和应用场景进行对比选材[[S24,S22,S16]]。
