## 概述

脱磷（dephosphorization，phosphorus removal）在金属冶炼领域是一个涉及面较广的工艺概念。在钢铁冶金中，脱磷是炼钢阶段的核心主工序之一，通过高碱度氧化渣将钢液中质量分数为0.02%~0.1%的磷脱除至目标范围[[S12,S2]]。而在压铸轻合金（铝/锌/镁合金）熔炼场景下，脱磷属于熔体净化的辅助工序，目标是将熔体中ppm级的微量磷杂质脱除，以保障合金的力学性能与铸造工艺性能[[S22]]。两者在工艺定位、反应体系与操作条件上存在根本性差异[[S22,S12]]。

## 定义

**钢铁脱磷**：在碱性电弧炉或转炉的氧化期中，利用炉渣中的FeO将钢液中的磷氧化生成P₂O₅，进而与渣中CaO结合形成稳定的磷酸钙（Ca₄P₂O₉）进入炉渣，通过扒渣操作实现磷的脱除[[S12,S32]]。脱磷是炼钢"四脱（C、O、S、P）二去（去气、去夹杂）"基本任务的重要组成部分[[S26]]。

**压铸轻合金脱磷**：在铝合金、锌合金等低熔点合金熔炼过程中，通过熔剂处理或吹气搅拌等方式，将熔体中微量磷杂质（通常为痕量级）降低至压铸合金标准允许限值以下，属于浇注前的事前净化工序[[S22,S37]]。其与铸件凝固完成后针对磷偏析缺陷的铸后校正处理（如磷共晶超标的热处理或机加工去除）在工艺目的和时序上完全不同[[S13,S24]]。

## 工艺目的与磷的危害

### 钢铁中的磷危害

磷是钢中最有害的杂质之一[[S12]]。磷含量过高会导致钢的冷脆性显著增加，降低冲击韧性。因此在炼钢过程中必须将磷含量控制在严格的规格范围内，例如成品规格要求P的质量分数≤0.040%时，扒渣前钢液中P的质量分数需控制在≤0.020%[[S36]]。

### 压铸铝合金中的磷危害

在压铸Al-Si系铝合金中，磷作为杂质元素会直接与常用的Na、Sr共晶变质剂发生反应，大幅降低共晶硅的变质效果，导致以下问题[[S22,S37]]：

- 亚共晶Al-Si合金中初晶硅异常析出、晶粒尺寸粗大
- 合金热裂倾向显著上升
- 塑性延伸率明显下降
- 合金的耐腐蚀性能劣化

磷还会消耗Na、Sr等共晶变质剂，降低变质处理的有效性[[S37]]。

值得注意的是，在过共晶Al-Si合金（如A390）中，适量磷元素反而具有细化初生硅片尺寸的正面作用[[S4,S22]]。这种双重效应决定了实际工艺中需根据具体合金类别决定是否控磷。

## 工艺机理与化学反应

### 钢铁脱磷反应机理

脱磷反应在钢-渣界面进行，其核心反应历程如下[[S6,S32]]：

**第一步：磷的氧化**
$$2[\mathrm{P}] + 5(\mathrm{FeO}) = (\mathrm{P}_2\mathrm{O}_5) + 5[\mathrm{Fe}]$$

**第二步：生成稳定磷酸盐进入炉渣**
$$(\mathrm{P}_2\mathrm{O}_5) + 3(\mathrm{FeO}) = (3\mathrm{FeO} \cdot \mathrm{P}_2\mathrm{O}_5)$$

**合并反应式**：
$$2[\mathrm{P}] + 8(\mathrm{FeO}) = (3\mathrm{FeO} \cdot \mathrm{P}_2\mathrm{O}_5) + 5[\mathrm{Fe}]$$

**碱性造渣脱磷总反应式**：
$$2[\mathrm{P}] + 5(\mathrm{FeO}) + 4(\mathrm{CaO}) = (\mathrm{Ca}_4\mathrm{P}_2\mathrm{O}_9) + 5[\mathrm{Fe}]$$
该反应为放热反应，反应热ΔH = -544310 J/mol [[S32,S21]]。

**喷粉脱磷反应式**[[S21]]：
- 钢液与石灰粉界面：$2[\mathrm{P}] + 5[\mathrm{O}] + 4\mathrm{CaO}(\text{固}) = (\mathrm{Ca}_4\mathrm{P}_2\mathrm{O}_9)$
- 钢液与矿石粉界面：$2[\mathrm{P}] + 5[\mathrm{O}] + \mathrm{Fe}_2\mathrm{O}_3 + \mathrm{Fe}(\text{液}) = 3\mathrm{FeO} \cdot \mathrm{P}_2\mathrm{O}_5(\text{液})$

### 钠基熔剂脱磷反应

对于采用Na₂CO₃熔剂体系的脱磷处理，其反应式与平衡常数经验公式如下[[S20,S9]]：
$$2[\mathrm{P}] + 5[\mathrm{O}] + 3(\mathrm{Na}_2\mathrm{O}) = (3\mathrm{Na}_2\mathrm{O} \cdot \mathrm{P}_2\mathrm{O}_5)$$

$$\log K' = 10.81 \times 10^4 / T - 46.92$$

### 脱磷反应动力学

脱磷反应速率受熔渣侧磷的传质过程控制[[S8]]。Mori和Kawai（1977）的研究指出，脱磷速率由磷在液态渣中向界面的传质速率控制，而返磷速率则由磷从界面向液态金属的传质速率控制，且脱磷速率慢于返磷速率[[S8]]。

## 热力学条件与工艺参数

脱磷反应的有利热力学条件可归纳为以下四点[[S12,S35,S23]]：

| 条件要素 | 推荐范围/要求 | 机理说明 |
|---------|-------------|---------|
| 炉渣碱度（CaO/SiO₂） | 3~3.5 | 提供充足的CaO与P₂O₅结合成稳定磷酸盐[[S12,S23]] |
| 渣中FeO含量 | ≥16%（质量分数） | 提供氧化性环境将[P]氧化为P₂O₅[[S12]] |
| 熔体温度 | 较低温度有利 | 脱磷为放热反应，低温利于正向进行[[S12,S21,S32]] |
| 渣量 | 约3%（熔体质量分数） | 增大渣量有利于吸收更多磷，但过大增加能耗[[S21]] |
| 操作方式 | 多次扒除含磷渣、更换新渣 | 防止磷酸钙浓度过高阻碍脱磷正向进行[[S12]] |

过量CaO会增大炉渣黏度，降低脱磷效率[[S12,S35]]。

### 典型工艺参数

**炼钢熔化期**：可去除熔体中50%~70%的磷，分批加入铁矿石总加入量为炉料总质量的1%~2%[[S1,S36]]。

**酸性感应电炉脱磷**（见表）[[S16,S30]]：

| 参数 | 数值范围 |
|------|---------|
| 处理温度 | 1480~1620 ℃ |
| 脱磷熔剂 | 石灰 + 氧化铁皮 + 萤石 |
| 熔剂加入量 | 钢液质量的3%~5% |
| 处理时长 | 10~20 min |
| 脱磷效率 | 15%~40% |

**Na₂CO₃熔剂脱磷**推荐条件[[S5]]：
- 精炼温度 < 1400℃
- 熔剂加入量：30 g/kg·HM
- 铁水初始[Si]含量 < 0.25%
- 当渣中Na₂O/SiO₂比值达到2.5时，磷在渣金间分配比(P₂O₅)/[P]可高于900

**含氟盐类熔剂处理铝熔体**参数[[S7]]：
- 处理温度：800~825℃
- 叶轮叶片角度：30°~40°
- 叶轮旋转速度：100~300 r/min
- 处理时长：数分钟
- 后续熔体静置2h以上，磷含量可降至(0.1~10)×10⁻⁴%量级

## 压铸合金中的磷含量标准

对于ADC12（中国牌号YL113）、A380（中国牌号YL102）等主流压铸铝合金，JIS H5302、ASTM B85、ISO 3522、中国GB/T 15115—2024四大主流标准体系中，均未设置磷的专属单独最高允许含量条款。磷元素归入通用"其他杂质元素"管控范畴，统一要求单项杂质的质量分数最高不超过0.05%，各标准之间该限值完全一致无差异[[S31,S10]]。

ADC12、A380等主流压铸铝合金原生配料的磷元素本底含量普遍低于0.05%，常规压铸生产线几乎不会设置主动脱磷工序，脱磷工艺仅在高比例再生铝回用的特种杂质管控场景下小范围应用，行业整体普及度极低[[S14,S25]]。

## 效果评价与检测方法

### 评价指标

压铸行业脱磷效果的核心评价指标为[[S15]]：

**脱磷率**：
$$\eta = \frac{w_{[P]0} - w_{[P]1}}{w_{[P]0}} \times 100\%$$

式中：
- $w_{[P]0}$：脱磷前熔体磷质量分数
- $w_{[P]1}$：脱磷后熔体磷质量分数

**残留磷含量**：脱磷完成后熔体中剩余的磷质量分数，是判定是否达到合金成分标准的直接指标。

### 检测方法

压铸生产中磷元素的主流检测方法包含两类[[S19]]：

| 方法 | 适用场景 | 检测精度 | 原理简述 |
|------|---------|---------|---------|
| 直读光谱分析 | 炉前快速检测 | 0.001% | 批量快速筛查 |
| 化学分析法（碳酸钠-硼酸熔融、萃取-分光光度法） | 实验室精准定量 | 0.0005% | 用异丁醇萃取磷钼黄络合物，氯化亚锡还原为磷钼蓝，730nm处测吸光度 |

## 副作用与局限性

脱磷工序在实施过程中存在以下负面效应[[S3,S35]]：

1. **优先氧化烧损**：脱磷过程会优先氧化熔体中硅、锰等比磷对氧亲和力更强的合金元素，提升对应元素的烧损率[[S3,S12]]。

2. **危废处置成本**：脱磷工序需额外排出质量分数约3%的高磷危险废渣，显著提升危废处置成本[[S21]]。

3. **返磷现象**：若高温阶段未完全排走含磷渣，炉渣中已固定的磷酸钙在还原性条件下会发生逆反应，已进入渣中的磷重新返回金属熔体，导致磷含量超标[[S21,S35]]。

4. **工序复杂度增加**：脱磷作为独立工序增加熔炼流程的复杂度和作业时间。

## 与其他工序的关系

脱磷属于针对单一杂质磷的专属去除工序，独立于除铁、除钙等其他单杂质去除工序，也区别于面向除气、除杂的整体熔体精炼流程[[S28]]。

工业生产的工序排布逻辑为[[S29,S38]]：

```
脱磷 → 彻底排渣 → 常规除气除渣 → 除铁除钙 → 变质处理
```

各工序依次衔接，脱磷作为氧化型渣处理工序必须设置在常规除气、除渣工序之前执行。脱磷完成后需彻底全部排走炉内含磷炉渣，再转入后续的除气、除渣工序，避免后续还原氛围下发生返磷现象[[S38]]。
