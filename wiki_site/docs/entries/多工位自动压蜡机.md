## 定义

**多工位自动压蜡机**（又称多工位射蜡机、全自动模蜡机）是特种铸造/精密熔模铸造中用于自动化批量生产蜡模（熔模）的核心制模设备[[S13,S2]]。国家标准 GB/T 25370—2020《铸造机械 术语》将其正式收录，标准英文术语为 **Multi-station automatic wax injection machine**[[S13]]。在目前检索到的行业标准、专著与手册中，未发现“MAWI”作为该设备通用行业缩写的公开记录[[S13]]。

该设备通过多工位转盘结构实现注蜡、冷却、开模、取件、喷涂分型剂、合模等工序的连续自动化循环，是熔模铸造全流程中“制模环节”的核心自动化单元[[S5]]。

## 分类

多工位自动压蜡机的行业通用分类维度如下[[S2,S17]]：

| 分类依据 | 具体类型 |
|---------|---------|
| 动力源 | 气动多工位压蜡机、液压多工位压蜡机 |
| 工位布局 | 卧式转盘式、立式转盘式、多圆盘工位型 |
| 自动化程度 | 半自动多工位压蜡机、全自动多工位压蜡机 |

按动力源区分，气动压蜡机以压缩空气直接或通过气缸活塞驱动，结构简单、价格低廉；液压压蜡机通过液压油驱动活塞，压射压力大、结构紧凑，适用于树脂基中温模料且参数控制精度更高[[S2]]。

## 核心结构与工作原理

### 主体结构

以十二工位卧式自动压蜡机为例，其核心组成包括：多槽轮间歇转位转盘工作台、变速箱传动机构、射蜡系统、模具锁紧气缸机构、止退气缸、刮型气缸、开合型曲线导轨机构、喷油气缸、恒温输蜡管路系统、循环水冷却系统及自动取件潜水胶带输送装置等[[S5,S4]]。

### 工位分配与工作循环

十二工位机型通过槽轮机构实现每次30°间歇转位，具体工作循环如下[[S5,S4]]：

| 工位 | 工序内容 |
|------|---------|
| 工位1 | 注蜡：锁紧气缸顶紧压型，止退气缸顶紧型芯轴，压蜡气缸推动压蜡嘴贴合注蜡口完成注蜡 |
| 工位2～6 | 自然冷却：压型随工作台转位，进行自然冷却 |
| 工位7 | 刮余蜡及涂分型剂：刮型气缸上刮蜡弹簧片刮除注蜡口表面余蜡并涂油 |
| 工位8～9 | 自动开模：压型滑轮借固定斜面装置打开压型 |
| 工位9～10 | 抽芯：固定曲线导轨抽动型芯轴向后滑动 |
| 工位10 | 自动取件：熔模自动脱落掉入冷却水槽，由潜水胶带压入水中冷却约12分钟 |
| 工位11 | 喷涂分型剂：喷油气缸带动喷油嘴伸入型腔喷涂脱模油 |
| 工位12 | 自动合模：固定合型曲线板驱动完成压型合拢，进入下一轮循环 |

主流工位数包含2工位、6工位、10工位、12工位等配置[[S5,S13]]。

## 关键工艺参数

### 注射压力

多工位自动压蜡机的注射压力因机型而异[[S2,S8,S16]]：

| 机型类别 | 注射压力范围 |
|---------|------------|
| 气动机型 | 0.4～5 MPa |
| 液压机型（常规） | 3.5～18 MPa |
| 高压液压机型（如进口高端设备） | 可达18 MPa |

部分高端机型配置流动压力补偿控制阀，可严格控制模料充填型腔的流动状态[[S8]]。

### 注射温度

可支持的温度范围[[S8,S7]]：

| 蜡料类型 | 注射温度 |
|---------|---------|
| 低温蜡料（石蜡-硬脂酸基） | 48～80 ℃ |
| 中温模料（树脂基） | 最高可达100±10 ℃ |

设备控温精度可达±1 ℃[[S8]]。

### 典型工业运行参数

以150T多工位液压压蜡机搭配中温蜡料为例，标准工艺参数如下[[S10]]：

| 参数项 | 设定值 |
|-------|--------|
| 压型温度 | 35 ℃ |
| 射蜡温度 | 65 ℃ |
| 高压注射压力 | 3.0 MPa |
| 低压注射压力 | 1.5 MPa |
| 高低压保压时间 | 120 s |

## 适用蜡料

多工位自动压蜡机可兼容的蜡料种类包括[[S2,S15]]：

- **石蜡-硬脂酸基低温蜡料**：适用于普通精密铸件；
- **树脂基中温模料**：适用于尺寸精度要求高的铸件；
- **聚乙二醇基可溶性蜡料**：填充60%体积比固体添加剂，可用于批量压铸生产所需的复杂内腔可溶蜡芯制备。

## 工艺角色与流程衔接

多工位自动压蜡机处于熔模铸造全流程的制模环节，前置对接化蜡槽、贮蜡槽、搅蜡机、蜡膏制备机组成的模料供给工序；输出合格蜡模后，直接衔接后续修模、组树、浸涂制壳、脱蜡、焙烧、金属浇注等工序[[S19,S1]]。

## 应用领域与典型场景

多工位自动压蜡机的主要应用领域包括[[S2,S4,S11]]：

- **航空航天**：航空精密结构件、燃气轮机叶片蜡芯的批量生产；
- **汽车制造**：大批量中小精密铸件的前置蜡模生产；
- **通用精密铸造**：标准化蜡模的连续大批量生产场景。

### 与压铸工艺的结合点

在压铸领域，多工位自动压蜡机可用于批量制备**可溶性蜡芯**。典型应用为燃气轮机静叶片等复杂压铸零件——其异形冷却通道无法采用传统抽芯机构直接成型，通过多工位自动压蜡机压制可溶性蜡芯预置入压铸模具，压铸完成后蜡芯在水溶液（必要时可加酸加速溶出）中完全溶出，从而形成复杂内流道结构[[S15]]。

## 优缺点与限制

| 对比维度 | 多工位自动压蜡机 | 单工位/简易压蜡机 |
|---------|----------------|-----------------|
| 生产效率 | 产能远高于单工位机型，适合大批量 | 效率低，适合小批量 |
| 人力需求 | 单台仅需1名操作人员 | 需较多人工操作 |
| 蜡模质量 | 尺寸一致性好，废品率低 | 手动控制多，质量波动大 |
| 设备投资 | 一次投资大，结构复杂 | 结构简单，投资低 |
| 换模灵活性 | 差，不适合多品种小批量 | 灵活，适合多品种 |
| 维修难度 | 维修工作量大 | 维修简便 |
| 尺寸适配 | 仅适合中小尺寸标准化蜡件 | 悬臂式单工位机型可适配超大异形件 |

综上，多工位自动压蜡机适配**少品种、大批量**的标准化蜡模生产[[S5,S2,S8,S12]]。

## 相关标准

目前公开标准体系中，尚无针对多工位自动压蜡机整机参数或验收条件的独立GB/T、JB/T专属专用标准[[S6]]。现有关联标准包括：

- **GB/T 25370—2020**《铸造机械 术语》：明确定义了多工位自动压蜡机的标准术语[[S13]]；
- **GB/T系列熔模铸造模料性能测试标准**（如流动性、黏度、热稳定性测定方法等）：提供模料性能检测支撑[[S6]]；
- **JB/T系列铸造机械通用安全要求**：提供设备安全技术规范[[S6]]。

ISO、ASTM标准体系中也未发现该设备的专属整机验收规范[[S6]]。
