## 概述

**偏芯**（core shift / core lift），在国家标准 GB/T 5611-2017《铸造术语》中收录为标准铸造缺陷术语，归属于**形状及重量差错类缺陷**，定义为“由于型芯在金属液作用下漂浮移动，使铸件内孔位置、形状和尺寸发生偏错，不符合铸件图的要求”[[S14,S5]]。压铸领域工程中亦俗称**型芯偏位**、**单边**、**孔偏**等[[S8,S5]]。

偏芯与错型（错缝）、错芯、型芯下沉等同属尺寸偏差类缺陷体系，是导致铸件形位公差超差、壁厚不均乃至报废的典型缺陷之一[[S3,S2]]。

**术语辨析**：压铸领域中“偏心轴（压铸机运动部件）”与“偏心型芯（模具异形型芯结构）”均不属于“偏芯”的标准正式术语范畴，属于工程口语化的简略称呼，行业正式场合不以此指代上述两类对象[[S4,S6]]。本词条的释义以缺陷定义为准。

## 定义与分类

**偏芯**属于铸造缺陷中的“形状及重量差错类缺陷”，其父类为**尺寸不合格**缺陷，与之并列的缺陷包括错型（shift）、错芯（core shift）、型芯下沉（core sag）、拉长、挠曲、串皮等[[S14,S3]]。

在压铸领域，偏芯常被标记为**型芯偏位**或**单边**，其核心判定特征是：**由型芯成型的铸件孔位、腔体位置与图纸要求不符**，表现为一侧壁厚减薄甚至穿透，另一侧壁厚增厚[[S8,S7]]。

**偏芯与错芯的区别**：根据 GB/T 5611-2017，错芯（core shift）指砂芯在分芯面处错开导致孔腔尺寸不符；偏芯（core lift）特指型芯在金属液作用下发生整体漂浮移动。两者均属形状及重量差错类缺陷，但产生机理有所不同[[S14]]。

## 产生机理

偏芯的核心机理是**型芯在金属液充型和凝固过程中失去定位稳定，发生非预期的上抬、位移、弯曲或断裂**。压铸条件下，这一过程的主要力学诱因包括[[S7,S1,S9]]：

1. **金属液冲击力与射流冲击**：充型时，自内浇口喷出的金属液形成激流，直接冲击型芯头部或侧面，导致型芯瞬时偏斜、弯曲变形。这是压铸偏芯最典型的致因路径。
2. **浮力与充型压力作用**：型芯浸没于高密度液态金属中，受浮力及充型静压作用，若型芯定位刚度不足，将发生漂移。
3. **型芯刚度不足与热变形**：细长型芯在反复热循环下刚度退化，受金属液持续冲刷产生蠕变弯曲。
4. **型芯定位结构失效**：型腔尺寸磨损、型芯定位尺寸超差、型腔内配合间隙过大或芯座被挤砂压坏，均会导致下芯时即存在初始偏位，充型时偏位进一步放大[[S13,S2]]。
5. **锁模力不足与胀型效应**：当胀型力超过压铸机锁模力时，模具开缝导致型芯随型腔一起偏移[[S2,S13]]。

## 对铸件质量的影响

偏芯缺陷直接影响铸件的几何精度与装配适用性，具体表现为[[S11,S8]]：

- 铸件对应孔位或型腔位置整体偏移，导致**局部壁厚严重不均匀**（一侧过薄、一侧过厚），极端情况下薄壁处甚至被穿透；
- 后续机械加工时，加工余量不足以补偿偏移量时出现**孔位露黑皮**（即加工后仍留有毛坯面）；
- 装配过程中发生**孔位干涉、零件无法安装**，或强制装配后产生附加应力。

压铸工程实践中，孔位偏斜与尺寸超差常被记录为“孔偏”缺陷。据某铝合金变速箱侧盖压铸案例，因油道型芯针弯曲导致的孔偏缺陷不良率曾高达 **20%** [[S1]]。

## 主要影响因素

压铸偏芯的发生是多因素耦合的结果。以下为经案例验证的主要影响因素[[S2,S13,S1,S9]]：

| 影响类别 | 具体因素 | 作用方式 |
|---|---|---|
| 型芯设计 | 型芯截面过细、长度/直径比过大 | 刚度不足，受冲击易弯曲变形 |
| 型芯定位 | 芯头尺寸偏小、定位面缺失、配合间隙过大 | 下芯时初始偏心，充型时进一步偏移 |
| 内浇口设计 | 内浇口位置使射流直冲型芯 | 型芯承受喷射冲击载荷，发生瞬时弯曲 |
| 热负荷 | 模具温度波动大、型芯冷却不足 | 型芯产生热蠕变弯曲，累积塑性变形 |
| 模具状态 | 型腔磨损、型腔与模套配合不良 | 定位基准失效，型芯重复定位精度下降 |
| 压铸工艺 | 压射力过高、压射速度过快、锁模力不足 | 胀型或冲击力超限导致型芯移位 |

实际生产中还应注意：模具装配不良（导柱/导套间隙过大）、分型面杂物未清理干净等也会间接诱发或加剧偏芯[[S2,S13]]。

## 检测与判定方法

偏芯的检测需聚焦于铸件内部型腔位置与壁厚的比对，常用手段包括：

- **截面解剖测量**：沿垂直于型芯轴线的方向剖切铸件，使用游标卡尺或千分尺测量两侧壁厚，判断型芯偏移量[[S1,S9]]；
- **三坐标测量（CMM）**：测量铸件孔位空间坐标并与 CAD 名义值比对，量化位置偏差[[S11]]；
- **X 射线探伤（RT）**：适用于不可剖切的中大型压铸件，通过影像观察型芯成型孔相对于外轮廓的偏移趋势；
- **功能量规 / 检具判型**：使用专用通过-不通过量规进行装配匹配性快速判检，判定孔位是否超出装配公差[[S11]]。

## 防止与纠正措施

根据压铸偏芯的产生机理，工程实践中已形成多层次的防控措施体系[[S7,S1,S2,S13]]：

### 1. 优化型芯定位设计
- **加大芯头尺寸**，增大芯头支撑面，增强型芯稳定性[[S7]]；
- **设计下芯基准面**，确保型芯装配时可目视或量具确认定位精度。对于无自然基准面的长型芯，可设置专用下芯定位样板[[S7]]；
- 检验型腔/型芯定位尺寸的配合间隙，确保不超差、不磨损[[S13]]。

### 2. 改进型芯结构与冷却
- 对细长型芯**增设内部冷却水道**（如 φ6 mm 通水孔），降低工作温度，减少热变形[[S1]]；
- 采用**型芯随动间隙设计**：在型芯固定端设置适当间隙（如单边 0.5 mm），在受力变形时允许型芯微量随动，防止卡死断裂[[S1]]；
- 对受冲击严重的型芯，**缩短受力悬臂段**或采用**分段式组合型芯**，避免单支长悬臂承受全部射流冲击[[S9]]。

### 3. 优化内浇口与充型条件
- **避免内浇口正对型芯头部**，调整浇口位置和流向，使金属液沿型腔壁面填充，减小对型芯的直接冲击[[S9,S1]]；
- 在型芯下游腔内适当设置溢流槽，均衡型芯两侧压力[[S8]]。

### 4. 调整压铸工艺参数
- 当胀型力超过锁模力导致模具开缝时，应**降低压射力**或**降低压射速度**，减小冲击峰值；必要时调整增压机构降低增压峰值[[S2,S13]]；
- 确保模具闭合前**清理分型面杂物**，保证合模严密[[S2]]。

### 5. 定期维护与检查
- 定期检查型芯是否存在弯曲、变形、磨损，发现异常及时更换[[S13,S2]]；
- 检查导柱、导套间隙及滑块配合面，保障模具顶出和运动精度[[S2]]。

**典型效果案例**：某铸铁轮毂（250 型）在湿型铸造条件下，偏芯率长期高达 **31.3%**；在增设专用下芯定位样板后，偏芯率骤降至 **3.5%** [[S7]]。某铝合金变速箱侧盖压铸中，型芯针经增设内部冷却及随动间隙设计后，因弯曲变形导致的废品率从 **20% 降低至 0** [[S1]]。

## 标准化术语与双语对照

| 语言 | 标准术语 | 别名 / 俗称 |
|---|---|---|
| 中文（GB/T 5611-2017） | **偏芯** | 漂芯、型芯偏位、单边、孔偏 |
| 英文（GB/T 5611-2017 对应术语） | **core lift** / **core shift** | core misalignment, core displacement |
| 英文（压铸语境常用） | **core shift** | core movement |

注：“core lift”侧重型芯在浮力作用下上抬偏移，“core shift”侧重型芯整体相对位置的横向偏移，两者在压铸工程中常混用以指代型芯偏位类缺陷[[S14,S5]]。该术语无通用缩写形式。
