**术语溯源**

“纸浆残液”是铸造行业已收录的标准术语条目，在《中国机械工业标准汇编 铸造卷》中与“纸浆废液”“亚硫酸盐纸浆废液”列为同一条目进行统一定义 [[S2]]。需要特别指出的是，该术语**并非压铸领域的专用术语**。在压铸技术文献中，没有将其作为脱模剂、冷却液或冲头润滑剂的正式品类收录；该词条的全部公开应用场景均集中于普通砂型铸造、消失模铸造的涂料粘结剂环节 [[S5]]。当在压铸语境下讨论该物料时，通常指其作为一种低成本辅助有机组分被少量引入某些通用水性涂料的可能，而非常规压铸脱模剂的主体材料。

**概述**

纸浆残液是亚硫酸盐法造纸过程的副产品。用亚硫酸盐法蒸煮木材或芦苇等原料、提取木质纤维素后所剩余的液体，经浓缩处理即得纸浆残液 [[S5,S7]]。若将该废液进一步发酵提取酒精，所得的浓缩废液称为亚硫酸盐酒精废液，二者铸造性能相近，生产上常统称为纸浆废液 [[S6]]。其主成分包括木质素磺酸盐、树脂、糖分及亚硫酸盐等 [[S5,S7]]。以木材为原料制成的纸浆残液，其干强度通常优于以芦苇为原料者 [[S5,S7]]。

在铸造涂料体系中，纸浆残液属于水溶性有机粘结剂 [[S6]]。其本质是一种低附加值的工业废液再利用产品，粘结强度低于糖浆，吸湿性较强，烘干硬化过程可逆 [[S5,S7]]。在压铸领域，主流水基脱模剂以蜡系材料、含硅有机物或其复合体系为主体 [[S4]]，纸浆残液并不属于上述任何一类压铸专用脱模剂的常规原料。

**工艺作用**

在包含纸浆残液组分的砂型及消失模铸造涂料中，其核心作用是作为辅助粘结剂，用以提升涂料层的干强度和附着性 [[S5]]。纸浆残液与粘土共用可获得较好的涂层强度，但因其呈弱酸性，当以碱性膨润土作悬浮剂时，其添加量需严格控制，一般不超过涂料总量的1% [[S5,S7]]。

典型的应用方式为将配制好的水基涂料以雾化喷涂方式施加到模型或型腔表面 [[S9]]。适量添加可提高涂层附着强度，减少涂层脱落，但过量喷涂会引入过多水分，在金属液充型时汽化，反而劣化铸件表面质量 [[S9]]。

**压铸语境下的角色界定**

现有证据不支持将“纸浆残液”视为压铸专用脱模剂、润滑剂或冷却液。

- **分类归属**：纸浆残液是水溶性有机粘结剂，仅在极少数廉价通用铸造水基涂料配方中作为辅助成膜组分被提及 [[S6]]。它不属于压铸行业定义的蜡系、硅系或复合系水基脱模剂中的任何一类 [[S4]]。
- **应用缺失**：目前无公开记录的、将纸浆残液作为主组分直接用于压铸量产的应用案例 [[S5]]。压铸工厂普遍选用的脱模剂已是由专业化工厂生产的定型产品，其主体成分和性能体系与纸浆残液存在本质差异 [[S4]]。
- **可能的误用语境**：在压铸行业口语交流或小型作坊中，可能存在将含有纸浆残液组分的廉价通用涂料称为“纸浆残液脱模剂”的情形，这与术语的准确技术定义存在偏差。

**缺陷与局限性**

若将含纸浆残液的涂料应用于压铸或类似的高温瞬时充型场景，可能导致以下缺陷：

- **气孔缺陷**：纸浆残液吸湿性强且硬化可逆，干燥涂层可重新吸水。在高温金属液充型时，过量未挥发水分瞬间汽化，卷入铸件形成气孔 [[S6,S9]]。
- **夹杂与积碳**：高温下纸浆残液中的有机物不完全热解，生成残留碳化物，无法随涂层完全排出型腔，在铸件内表面形成夹杂缺陷；长期积累则导致模具表面积碳 [[S6]]。
- **表面质量劣化**：过量喷涂导致脱模剂积聚，会引发铸件疏松、夹渣、气泡等缺陷 [[S9]]。

与其他压铸辅料相比，纸浆残液作为低成本工业废液，存在一系列显著劣势 [[S1]]：

| 对比维度 | 纸浆残液/廉价通用涂料 | 主流压铸专用水基脱模剂 |
|---|---|---|
| 润滑脱模性 | 低，非设计功能 | 高，核心功能 |
| 模具积碳倾向 | 严重 | 低（经配方优化） |
| 废水COD/BOD | 显著偏高 | 相对较低 |
| 烟气与气味 | 热解产生刺激性气味 | 低烟、低味设计 |
| 废液处理难度 | 大 | 部分可回收再生 |
| 成本 | 低廉 | 较高 |

**关联术语**

以下术语与“纸浆残液”在铸造语境下存在同义、近义或相关关系：

- **亚硫酸盐纸浆废液 / 纸浆废液**：标准中收录的同义别名 [[S2]]。
- **亚硫酸盐酒精废液**：纸浆残液经发酵提取酒精后的浓缩液，铸造性能与之相近 [[S6]]。
- **木质素磺酸盐**：纸浆残液的主要功能组分，行业内常将二者归为同源的低成本工业基料 [[S5,S7]]。
- **黑液**：碱法制浆产生的废液，与酸法（亚硫酸盐法）产生的纸浆残液在化学成分上有明确差异 [[S5]]。部分地区可能将其统称为“黑液”类材料，但非精确术语。
