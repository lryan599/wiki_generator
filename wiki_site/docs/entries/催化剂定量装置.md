## 概述与工艺范畴

催化剂定量装置（catalyst dosing device）在铸造领域是专门用于冷芯盒制芯与树脂砂造型工序的精密计量供给设备，属于砂处理系统的辅助配套装置[[S4]]。该装置并不直接应用于压铸过程，而是服务于铸造前端的砂芯制造环节。在行业文献与工程实践中，该装置也被称为催化剂供给装置、胺定量装置，或具体称作冷芯盒气体发生器的核心计量单元（胺定量单元、催化剂量化供给单元），英文对应术语包括 catalyst dosing device、catalyst metering unit、amine catalyst dosing unit 等[[S4,S5]]。

须特别注意，GB/T 25370-2020 明确规定压铸机用 **脱模剂定量装置** 的标准术语为 releasing agent dilutor，与本文所述催化剂定量装置所属的设备范畴完全不同[[S7]]。

## 定义

依据 GB/T 25370-2020《铸造机械 术语》，催化剂定量装置被定义为 **具有自动调节定量功能的催化剂吸入装置**，属于铸造机械砂处理辅助设备类，用于树脂砂造型制芯工序中催化剂的精准投加[[S4]]。

## 工艺部署与应用场景

催化剂定量装置的部署位置取决于所服务的冷芯盒工艺类型：

- **气雾冷芯盒制芯（如三乙胺法）**：该装置位于冷芯盒气体发生器内部，处于液态胺储料罐与雾化混合室之间。装置定量输出的液态三乙胺送入雾化器，经干燥压缩空气或氮气稀释为浓度约 5% 的催化气雾后，吹入芯盒完成砂芯硬化[[S12,S8]]。此时，催化剂定量装置正是制芯工艺链“吹气工段”中负责胺类催化剂定量供给的核心硬件，直接对接催化剂载体气管路系统[[S8]]。配套的完整上下游链路为：催化剂定量装置 → 胺气发生器 → 冷芯盒制芯机/射芯机 → 吹气固化系统 → 尾气洗涤处理设备[[S8]]。
- **自硬冷芯盒制芯**：催化剂定量装置部署于混砂工段上游，与树脂定量器并列，将固化用催化剂精准计量后送入高速混砂机，混合料经射砂筒射入芯盒后完成硬化[[S9]]。

## 核心功能与精度要求

装置的核心功能是针对不同重量的砂芯，精确定量对应体积的液态催化剂，保证催化剂量与砂芯重量成正比例匹配。定量偏差可能带来以下双重后果[[S14]]：

- 催化剂过量 → 易导致砂芯中多余胺类残留，后续浇注时生成氮气孔缺陷，并延长后序通风除味工时；
- 催化剂不足 → 砂芯硬化反应不充分，出现局部未硬化缺陷。

定量精度的最低要求为 ±0.5 cc[[S2,S14]]。GCB4 型冷芯盒胺发生器配套的高精度定量装置，体积调节精度可达 0.1 cm³[[S12]]。

## 关键工艺参数

气态催化剂混合气流在冷芯盒吹气固化阶段的典型工艺参数如下[[S1,S8]]：

| 参数 | 典型数值/条件 |
|---|---|
| 催化气雾工作压力 | 0.14～0.2 MPa |
| 雾化用压缩空气压力（气雾冷芯盒法） | 约 2.5 大气压 |
| 吹扫置换用洁净空气压力 | 约 3.5 大气压 |
| 催化剂稀释浓度（三乙胺/载体气） | 约 5% |
| 载体气种类 | 干燥压缩空气、CO₂ 或高纯氮气（推荐氮气） |

液态胺被定量送入 35～50 ℃ 的预热腔后，在干燥低压载体气流作用下气化后进入芯盒。应避免过高压力下的过量供给，以防止芯盒入口处胺液冷凝并导致近端砂芯饱和、远端砂芯硬化不足的工艺缺陷[[S12,S14]]。

## 分类方式

### 按适配催化剂状态分类

催化剂定量装置可按照所处理催化剂的相态及其适配的铸造工艺进行分类[[S13,S8]]：

| 分类 | 适配工艺 | 代表性催化剂 |
|---|---|---|
| 液体催化剂定量 | 酚脲烷自硬树脂砂体系 | 液态叔胺类物质 |
| 液体催化剂定量 | 酯硬化酚醛树脂砂体系 | 液态有机酯类物质 |
| 气体/气雾催化剂定量 | 气雾冷芯盒制芯体系 | 三乙胺(TEA)、二甲基乙胺(DMEA)、异丙基乙胺、三甲胺(TMA)等叔胺类物质 |
| 气体/气雾催化剂定量 | SO₂ 法冷芯盒制芯 | SO₂ 气态酸类催化剂 |

### 按定量原理分类

铸造砂处理场景中，常用催化剂定量器按结构原理可分为以下类型[[S15]]：

| 类型 | 特点 |
|---|---|
| 星形定量器 | 结构简单，适用于连续定量 |
| 容积式定量器 | 以固定容积周期投加 |
| 杠杆式定量器 | 可按需调整定量参数 |
| 电子秤式定量器 | 称量精度相对更高，多用于混砂机配套的精准投加工位 |

## 结构与组成

催化剂定量装置的典型核心部件包括[[S6,S10,S8]]：

- **计量泵**（容积式/柱塞式），用于液态催化剂的高精度输送；
- **专用流量计**，适配所输送介质的物化特性；
- **流量/压力控制阀**，调节输出参数；
- **密闭催化剂储存储罐**，保证介质不受污染；
- **浓度/压力/流量实时检测传感器**；
- **可编程逻辑控制器(PLC)** 作为核心控制系统，实现精确程序控制。

## 装置材料与耐腐蚀要求

胺类催化剂的化学腐蚀性对装置过流部件的材料选型提出严格要求。无水叔胺类催化剂常温下即可腐蚀纯铝材质；低浓度含水胺（甲胺质量浓度 < 30%、丁胺质量浓度 < 70%）对普通铝材无明显腐蚀作用。长期接触高浓度胺类介质的过流部件，应选用 304/316 不锈钢或内衬聚四氟乙烯（PTFE）的防腐结构，以延长设备服役寿命[[S3]]。

## 维护与典型故障

催化剂输送管路的定期维护是保证定量精度与设备正常运行的关键。硅胶 + 石英棉过滤组件每 6 个月应定期更换；输送泵配套管路每 90 天应检查更换；入口筛网过滤器出现流量不足时应立即拆解清洁。最常见的故障“定量偏差超出允许范围”的核心诱因多为管路内胺盐结晶堵塞，拆解管路后使用无水乙醇冲洗即可恢复正常工作状态[[S11]]。
