## 概述

缺陷铸件（压铸缺陷件）是指在压铸生产过程中产生的、不满足验收标准要求的不可接受铸件。压铸件由于在高压高速条件下成形，其缺陷类型繁多、成因复杂，且具有大批量生产中缺陷快速复制的特点[[S2]]。缺陷铸件属于压铸件质量管控体系的子集，与普通可接受缺欠（imperfection）存在明确边界：缺欠是铸件中存在的不连续区域或与预设几何形态/材料规格不符的偏差，而缺陷（defect）则是不满足验收标准的不可接受缺欠[[S11,S50]]。

在质量管理体系中，缺陷铸件是不满足预期使用要求或存在安全风险的铸件，按照质量缺陷分级可划分为A级（严重缺陷）、B级（一般缺陷）和C级（轻微缺陷）[[S27]]。

## 定义与范畴

依据ISO 8402质量术语定义，压铸缺陷件属于压铸件质量管控体系的子集，其与普通可接受缺欠存在明确边界。缺陷铸件是不满足预期使用要求、或存在安全风险的铸件，不同于仅尺寸/性能未达规范的普通不合格品，对应质量分级中A（严重）、B（一般）、C（轻微）级的不可接受缺陷产品[[S27]]。

ISO/TR 16078:2013对铸造缺欠与缺陷给出了明确定义[[S11]]：

- **imperfection（缺欠）**：铸件中存在的不连续区域，或与预设几何形态/材料规格不符的偏差。
- **defect（缺陷）**：不满足验收标准的不可接受缺欠。

该分类体系已被GB/T 41972-2022等同采纳，并与GB/T 5611—2017《铸造术语》相协调[[S50]]。我国GB/T 29092-2012《镁及镁合金压铸缺陷术语》为压铸专项缺陷提供了官方统一定义，明确了飞边、顶杆痕迹、气孔、缩孔、疏松、非金属夹杂物、热裂纹、冷裂纹等压铸典型缺陷的规范命名与特征描述[[S21]]。

## 缺陷分类体系

### 国际通用分类

国际铸造技术协会（ICFTA）《国际铸件缺陷图谱》建立了一套基于物理特征的缺陷分类体系，无需依托成因经验即可通过外观、位置、尺寸对缺陷进行分类编码，采用1位字母+3位数字的编码规则[[S42,S8]]。该体系将铸件缺陷划分为7大类[[S5]]：

| 大类编码 | 类别名称 | 典型缺陷举例 |
|----------|----------|-------------|
| A | 多肉类 | 飞边、毛刺、多肉 |
| B | 孔洞类 | 气孔、缩孔、缩松、针孔 |
| C | 裂纹、冷隔类 | 热裂纹、冷裂纹、冷隔 |
| D | 表面缺陷 | 表面花纹、流痕、网状毛翅 |
| E | 铸件残缺 | 欠铸、未浇满、掉肉 |
| F | 尺寸或形状差错 | 变形、错型、尺寸超差 |
| G | 夹杂物和金相组织不合格 | 非金属夹杂、硬质点、偏析 |

### 国内标准分类

我国GB/T 5611-1998《铸造术语》将铸造缺陷统一划分为8大类，共包含100余种具体缺陷，该标准分类与ISO/TR 16078国际体系兼容[[S42]]。

### 压铸专项分类

针对压铸生产场景，通用的专项缺陷分类共分为6大类[[S14]]：

| 类别 | 涵盖缺陷 |
|------|---------|
| 形状尺寸类 | 变形、尺寸超差、错型、型芯移位、欠铸等 |
| 表面类 | 冷隔、花纹、飞边、缩陷、拉伤、网状毛翅、粘附物痕迹等 |
| 孔洞类 | 气孔、缩孔、缩松、针孔、渣孔、气泡等 |
| 裂纹类 | 热裂纹、冷裂纹 |
| 夹杂类 | 非金属夹杂、硬质点等 |
| 材质性能不合格类 | 化学成分不达标、力学性能不符合要求、偏析等 |

## 典型缺陷及形成机理

### 孔洞类缺陷

**气孔**的核心形成机理是压铸过程中气体卷入金属液。卷入方式分为两种：流体前沿表面湍流卷气和铸件表面补型卷气，卷入的气体在凝固过程中无法通过浮力逸出，最终在铸件内部或近表面形成表面光滑的孔洞[[S26,S45,S36,S43]]。

工艺参数对气孔形成的影响：
- 半固态压铸工况下，初始固相分数固定时，内浇口速度从0.8m/s提升至4m/s，芯部气孔缺陷形成频率从0上升至1；内浇口速度固定时，初始固相分数从0.3提升至0.5，芯部气孔缺陷形成频率相应降低[[S26]]。
- 气孔率主要受压射速率和增压压力影响，随着增压压力增大，气孔率明显下降[[S23,S35]]。
- 卷气率受参数影响排序为：压射速率 > 模具温度 > 浇注温度[[S23]]。

模具排气布局不足、内浇口位置不合理、冷却回路设计不当都会导致表层附近残留气体形成鼓泡类气孔缺陷[[S38]]。铝熔体中的氧化铝夹杂、溶解氢含量过高会显著提升气孔缺陷的产生概率，熔体净化不到位时铝与水蒸气反应生成的氢分子会在凝固过程中析出形成孔隙[[S22]]。

**缩孔/缩松**的核心形成机理是合金凝固过程发生体收缩，若凝固后期金属液无法有效补充收缩产生的空隙，就会形成表面粗糙、形态不规则的孔隙。细小网状分布的孔隙为缩松，集中的大尺寸孔隙为缩孔[[S43,S15]]。

缩孔率受参数影响排序为：浇注温度 > 模具温度 > 压射速率。过高的浇注温度和模具温度会导致合金体收缩量增大[[S23,S10]]。通过正交极差分析，压射速度对变速箱壳体类铸件缩松缩孔缺陷的影响权重最高，其次是浇注温度，模具预热温度影响相对较小[[S29]]。

### 夹杂缺陷

夹杂缺陷核心形成机理为熔体熔炼过程中产生的高硬度氧化铝等非金属氧化物颗粒，密度接近金属液难以浮升至液面，最终随金属液充型进入型腔残留在铸件内部。其熔点远高于合金正常压铸温度（约2000°C），始终以固态形态存在于基体中[[S22]]。氧化的炉料表面含30%~60%的氢氧化铝成分，常规熔炼温度下难以完全分解去除，会大量生成氧化铝夹杂[[S22]]。

### 冷隔缺陷

冷隔缺陷的核心形成机理是两股温度较低、表层已提前发生凝壳的金属流在充型过程中发生碰撞对接，未能完全熔合在一起，最终在对接位置形成边缘圆滑、结合力薄弱的凹陷状表面缺陷[[S47,S38]]。浇注温度、模具温度偏低，压射速度不足，压射比压过低时，金属液流动前沿冷却速度过快，极易产生冷隔缺陷[[S47,S23]]。

### 裂纹类缺陷

**热裂纹**产生于合金凝固的固液共存温度区间，凝固收缩受到机械阻力或热应力约束，半固态糊状区的晶间连接发生断裂形成裂纹，断口表面通常存在轻微氧化特征[[S38]]。不合理的参数设置导致铸件凝固过程温度梯度过大，半固态区域剪切变形应力过高时，会大幅提升热裂纹的产生概率[[S31]]。

**冷裂纹**为铸件温度降至固态区间后，残余应力超过材料强度极限引发的开裂，断口表面洁净、具有金属光泽[[S21]]。

### 变形与表面缺陷

**变形缺陷**的核心形成机理是铸件在铸造残余应力作用下发生翘曲、扭转等形态偏差，也可能是开模顶出过程受力不均、铸件未完全凝固就提前脱模所引发的机械形变[[S38,S16]]。模具脱模斜度过小、顶出杆分布不均匀顶出力不对称是造成铸件变形的常见模具设计诱因[[S38]]。

**网状毛翅**是压铸件特有的表面缺陷，其形成机理为：压铸模型腔表面长期承受冷热交变载荷引发热疲劳，产生网状龟裂裂纹，后续充型过程中金属液填充进入裂纹，在铸件表面形成随压铸次数增加不断扩展延伸的网状发丝状凸起痕迹[[S47]]。

### 常用检测方法

| 检测方法 | 适用缺陷类型 | 检测特点 | 来源 |
|----------|-------------|---------|------|
| 目视检测 | 花纹、流痕、缩凹、变形、冷隔、缺肉、变色、斑点等表面缺陷 | 最通用的表面初检方法，可在普通日光下进行，必要时搭配5~10倍放大镜 | [[S7,S33,S48]] |
| 荧光渗透检测 | 宽度低至0.01mm的微裂纹、表面冷隔、连通表层的孔隙 | 表面开口微小缺陷无损检测，操作流程：清洁→渗透→去除多余渗透剂→显像→紫外灯下观察荧光 | [[S6,S20,S28]] |
| X射线探伤 | 内部缩孔、缩松、夹杂物、气孔、内部裂纹 | 利用射线穿透差异在胶片形成影像，可判定缺陷位置、形状、尺寸与分布；近年自动化系统可通过数字图像处理自动识别缺陷 | [[S7,S49,S30]] |
| 工业CT | 内部任意深度微小缺陷 | 可实现三维重建，精准定位空间分布，多用于航空航天、高端汽车零部件等高附加值铸件检测 | [[S12]] |
| 超声波探伤 | 内部缩孔、气孔、裂纹 | 利用超声波在金属-空气界面的反射判定缺陷位置，灵敏度高，对人体无害；难以判定缺陷具体类型与精确尺寸，复杂异形铸件适配性差 | [[S7,S6]] |
| 金相分析 | 孔洞类缺陷定性（缩孔vs气孔）、夹杂、硬点、裂纹 | 破坏性检测方法。缩孔呈现不规则边缘和暗色内腔，气孔呈现光滑边缘和光亮内腔，是缺陷属性定性判定的核心手段 | [[S9,S34]] |

### 缺陷分级与验收标准

**ISO 9915标准**将铝合金铸件射线检测缺陷验收等级划分为4个层级[[S51]]：

| 验收等级 | 适用范围 |
|---------|---------|
| A级 | 最严格，高应力重载铸件 |
| B级 | 中等应力铸件 |
| C级 | 一般应力铸件 |
| D级 | 普通铸件 |

ISO 9915同时规定，同一5cm×5cm检测区域内同时存在4种及以上不同缺陷时直接判定不合格，单区域仅允许最多3种缺陷同时存在[[S51]]。

## 缺陷影响

### 对力学性能的影响

压铸件内部的连通性孔隙类缺陷会直接破坏基体连续性，大幅降低铸件拉伸强度、冲击韧性等力学性能，在交变载荷服役环境下极易成为疲劳裂纹的萌生点，大幅缩短产品服役寿命[[S39,S52]]。

### 对密封性的影响

当压铸件内部的微孔类缺陷在壁厚方向贯通时，会直接导致承压部件出现漏气、漏水、漏油等密封失效问题。常规情况下可通过浸渗工艺对孔径0.5mm以下的微孔进行封堵，一次浸渗合格率可达95%以上[[S52]]。

### 对后加工的影响

压铸件表层的冷隔、凹陷、流痕等缺陷会直接降低表面精度，在后续机加工工序中，若缺陷分布位置恰好处于加工余量范围内，加工完成后缺陷会直接暴露，造成机加工件报废。同时表面存在硬点类缺陷会大幅提升机加工刀具的损耗率[[S15]]。

## 常见缺陷成因与对策

压铸行业通用的缺陷全流程防控方法核心覆盖五大维度：优化铸件结构避免局部厚大热节和不合理壁厚突变、优化模具排气系统与溢流槽设计降低卷气概率、匹配合理的压射参数控制充型速度与充型时间、管控合金熔炼质量与回炉料占比降低杂质含量、采用机器人自动化喷涂替代人工喷涂保证工艺稳定性[[S17,S4]]。

行业内已形成覆盖典型缺陷的成因-对策对应体系[[S53,S1]]：

| 缺陷类型 | 常见成因 | 对策措施 |
|----------|---------|---------|
| 冷隔 | 排气不畅、浇注温度偏低、充型速度不足 | 优先检查排气系统通畅性、适当提升浇注温度与充型速度 |
| 局部缩孔/缩松 | 厚壁区域热节、补缩不足 | 优先在对应热节位置设置局部挤压销补缩 |
| 卷气类气孔 | 型腔内空气残留过多 | 优先在模具侧增加真空排气系统降低型腔内空气残留量 |
| 夹杂 | 熔体净化不到位、炉料氧化严重 | 加强熔体精炼处理、控制回炉料比例与质量 |
