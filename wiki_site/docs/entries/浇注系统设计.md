## 定义与功能

浇注系统是铸型中引导液态金属流入型腔的一系列通道的总称，属于铸造工艺设计核心组成部分 [[S10,S53]]。根据中国机械工业标准汇编中的定义，浇注系统设计是指根据铸件材质、重量、形状和尺寸以及铸造设备条件，确定铸件浇注位置和浇注系统形式，计算浇注时间和浇道最小横截面积，确定各浇道的截面积比、浇注系统结构及各组元位置的过程 [[S27]]。

合理的浇注系统设计可避免铸件产生冷料、裂纹、缩松等常见缺陷 [[S4,S10]]。在实际生产中，约有20%~30%需要后期修复的铸件由浇注系统设计不合理所导致 [[S33]]。

浇注系统在铸造全流程中的核心功能包含以下五点 [[S6,S33]]：

- **控制充型速度与时间**：保证在规定的浇注时间内充满型腔，保证铸件轮廓、棱角清晰
- **平稳流动**：使金属液均匀平稳流动，避免紊流、卷气和冲刷铸型
- **挡渣排气**：有效阻挡熔渣与夹杂物进入型腔，不卷入多余气体
- **调节温度场**：在充型金属中造成理想的温度分布，引导铸件按预设顺序完成凝固
- **压力传递**：在部分工艺场景下（如压铸）实现压力传递，辅助后续补缩压实

## 分类

浇注系统可按两种主流方式分类：按内浇道注入铸件型腔的相对位置分类，以及按各组元截面积比例分类。

### 按注入位置分类

浇注系统按内浇道位置可分为顶注式、底注式、中注式、侧注式、阶梯式、缝隙式六大主流结构形式 [[S36,S54,S55,S50]]。

| 类型 | 特征 | 优劣势 | 适用场景 |
|------|------|--------|----------|
| 顶注式 | 金属液从铸件顶部注入 | 上高下低的温度梯度适配顺序凝固；充型流量大；但易产生喷溅、冲刷铸型 | 高度较低、补缩要求高的厚壁小型铸件 |
| 底注式 | 金属液从铸件底部平稳上升充型 | 排气、挡渣效果优异；但形成上低下高的反向温度场，削弱顶部冒口补缩能力 | 易氧化的铝镁合金铸件、大型机床床身、复杂套筒类铸件 |
| 中注式 | 内浇道开设在分型面、铸件高度中部 | 兼具顶注、底注优缺点；造型操作简便 | 壁厚均匀、高度小于100mm的中小型铸件 |
| 侧注式 | 从铸件侧壁位置注入 | 可开设多组大截面内浇道实现大流量快速充型；分型清理简便 | 薄壁、小型复杂铸件，尤其是管类铸件 |
| 阶梯式 | 金属液分层自下而上依次注入 | 充型冲击力小，兼有平稳充型与顺序凝固补缩需求；但造型复杂 | 气缸体、大型床身等高要求大尺寸铸件 |
| 缝隙式 | 金属液沿立缝注入型腔 | 充型平稳，排气良好，可防止铝液二次氧化，型腔热分布较合理 [[S23]] | 铝、镁合金薄壁件 |

### 按截面比分类

根据中国机械工业标准汇编的定义，浇注系统按浇道比（直浇道截面积A₁、横浇道截面积A₂、内浇道截面积A₃之比）分为三类 [[S27,S30,S46]]：

| 类型 | 截面比关系 | 特征 | 典型应用 |
|------|-----------|------|----------|
| 封闭式 | A₁ > A₂ > A₃ | 内浇道为阻流截面，充型时系统快速充满，挡渣能力优异 [[S55,S57]] | 铸铁件 |
| 开放式 | A₁ < A₂ < A₃ | 直浇道为阻流截面，金属液流动平稳，对铸型冲刷力小 [[S55,S57]] | 铸钢件、非铁合金铸件 |
| 半封闭式 | A₁ < A₂ 且 A₂ > A₃ | 介于开放式与封闭式之间，兼顾挡渣与低冲刷 [[S57]] | 铸钢、铸铁件 |

## 基本组成与典型构件

浇注系统的典型基础结构包含以下四个核心部分 [[S10,S4,S50]]：

| 构件 | 作用 | 典型几何特征 |
|------|------|------------|
| 浇口杯/浇口盆 | 承接浇包倾倒的金属液，缓冲冲击 | 小型铸件用漏斗形，上口直径≥2倍直浇道直径；中大型铸件用盆形，可增设集渣装置 |
| 直浇道 | 连接浇口杯与横浇道，将金属液从外部导入铸型内部 | 通常采用垂直圆柱形 |
| 横浇道 | 分配液流、阻隔杂质、稳流 | 高度通常大于内浇道高度5~6倍以提升挡渣效果 [[S50]] |
| 内浇道 | 将金属液直接导入型腔的最后一段通道 | 截面尺寸最小，位置、形状根据铸件结构设计 |

不同铸造工艺还配置专属构件 [[S7,S4,S11,S20]]：

- **低压铸造升液管**：连接低压保温炉液面与铸型直浇道，引导金属液在气压作用下上升。某型燃油调节器升液管直径设计为55mm，外围包覆硅酸铝类陶瓷纤维保温套以减少温降 [[S4]]。
- **高压压铸浇口套/分流锥**：浇口套连接压室与横浇道，分流锥布置在直浇道入口处引导金属液均匀分流，减少涡流产生 [[S7]]。

## 设计核心参数与原则

### 截面比设计

浇注系统截面比（A_直：A_横：A_内）是控制充型行为的关键参数，不同材质和工艺类型的推荐值差异显著。

**灰铸铁**：
- 封闭式系统推荐截面比 A_直:A_横:A_内 = 1.2:1.4:1 [[S13]]

**球墨铸铁** [[S29,S9,S13]]：
- 普通件封闭式：A_内:A_横:A_直 = 1:(1.2~1.3):(1.4~1.9)
- 厚壁件开放式：A_内:A_横:A_直 = (1.5~4):(2~4):1
- 薄壁小型件半封闭式：A_内:A_横:A_直 = 0.8:(1.2~1.5):1

**铝合金（重力砂型铸造）** [[S18]]：
- 大中件：A_直:A_横:A_内 = 1:(2~3):(2~4)
- 小件：A_直:A_横:A_内 = 1:(1.5~3):(1~1.5)

**轻合金（铝、镁）非加压开放式系统典型比值**：
- 直:横:内 = 1:4:4 [[S8,S44]]

**黑色金属加压封闭式系统典型比值**：
- 直:横:内 = 1:2:1 [[S8,S44]]

### 流动参数与雷诺数控制

金属液在浇注系统中的流动状态由雷诺数（Re）判定 [[S17,S21]]：

Re = vD / ν

式中：v为金属液在管路中的流速，D为管路特征直径，ν为流体运动黏度。临界雷诺数2300为层流-紊流分界点。实际常规浇注系统中金属液流动基本处于紊流状态，设计应尽可能降低紊流程度以减少卷气和夹渣缺陷 [[S17]]。

铝合金浇注温度下运动黏度推荐取值为6×10⁻⁷ m²/s，镁合金为7×10⁻⁷ m²/s [[S47,S26]]。轻合金各流动区需设定最大允许雷诺数阈值，避免氧化膜破裂卷入金属液形成夹渣缺陷。

### 充型时间与速度

铸钢件充型总时长经验计算公式 [[S48]]：

t = (1.5~2.3)·G⁰·⁵

式中t为充型时间（s），G为包含浇冒口在内的总金属质量（kg）。

高压压铸内浇道出口金属液推荐流速范围 [[S5,S41]]：

| 铸件类型 | 推荐流速 (m/s) |
|----------|---------------|
| 常规工程类铝合金铸件 | 20~60 |
| 表面装饰类铸件 | 10~30 |
| 耐压致密件 | 40~70 |

生产中需控制流速不超过40 m/s以降低模具冲蚀速率 [[S5]]。

### 挡渣与排气设计

**集渣包（溢流槽）参数** [[S59,S34]]：
- 横浇道末端集渣包推荐尺寸：最小长度 l_III ≥ 3.5h_K，最小高度 h_III ≥ 2h_K，过渡圆角 R ≥ 2h_K（h_K为横浇道高度）
- 集渣包优先布置在直浇道出口后第一个内浇道之前

**集渣包布置原则** [[S19,S52]]：
- 金属液最后到达的部位
- 两股金属流汇合处及易产生涡流区域
- 型芯背面低压区
- 排气不良深腔处和局部模温过低部位

**高压压铸排气槽参数** [[S14,S2]]：
- 深度：0.1~0.2mm
- 宽度：10~20mm
- 总截面积为内浇道截面积的1.5~2.5倍

## 特殊工艺与材料下的设计要点

### 不同铸造工艺的设计差异

不同铸造工艺类型的浇注系统设计存在显著通用差异 [[S7,S4,S12,S3,S15]]：

| 工艺类型 | 设计要点 |
|----------|----------|
| 重力砂型铸造 | 以浇口杯为入口，通用截面比适配铸件材质，优先保障挡渣、平稳充型 |
| 高压压铸 | 核心功能含高压传递，截面从直浇道到内浇口逐步收窄减少卷气，内浇口布置在铸件厚壁区域，主流形式含扇形、锥形切线、缝隙、环形、点浇口等 [[S7]] |
| 低压铸造 | 配置升液管为核心部件，金属液在低压作用下平稳上升充型 [[S4]] |
| 消失模铸造 | 优先采用封闭式浇注系统，截面积比常规砂型铸造大30%~50%以抵消泡沫气化吸热反压，常搭配底注式 [[S3,S15]] |
| 熔模铸造 | 浇注系统同时承担模料排出流道作用，需保证模料先于铸件型腔内的结构流出 [[S12]] |

### 大型薄壁件设计实例

大型薄壁铁路摇枕铸件采用多内浇道大流量快速充型设计：设置7个内浇道搭配1个直浇道的布局，浇注时间控制在25~35s，在铸件厚壁配套激冷措施，可有效降低热裂缺陷生成概率 [[S4]]。

### 镁合金防氧化设计

镁合金壳体缝隙式浇注系统布局设计：采用外轮廓为圆形的横浇道，内部设置6条分支，配套立筒缝隙浇道沿铸件外圆周面一周均匀分布，可均匀分配镁合金液流量，降低局部过热和充型紊流度，减少合金液与空气界面扰动，抑制氧化燃烧倾向 [[S4]]。

其设计依据为：采用非加压开放式浇注系统结构，降低金属液充型雷诺数，实现近似层流的平稳充型，避免紊流卷吸空气导致的镁合金液表面氧化膜破裂，减少氧化夹杂缺陷生成 [[S23,S8]]。

### 低压铸造升液系统

复杂燃油调节器低压铸造升液系统设计：采用单一升液管搭配单一直浇道的布局，升液管直径设计为55mm，外部包覆硅酸铝类陶瓷纤维保温套，可降低升液段金属液热损耗，避免升液通道内金属提前凝固，保障充型过程连续 [[S4,S11,S20]]。

## 常见缺陷关联与修正方向

浇注系统设计不当与多种铸造缺陷直接关联 [[S43,S37,S16,S38,S39,S31,S22]]：

| 缺陷类型 | 关联机制 | 修正方向 |
|----------|----------|----------|
| 浇不足 | 内浇道截面积偏小、充型流程过长导致薄壁区域提前凝固 [[S43,S37,S16]] | 增加内浇道总截面积，调整多内浇道布局缩短填充距离 |
| 冷隔 | 多股液流充型温差过大、浇注总时间过长 [[S43,S39,S31]] | 缩短浇注时间，优化内浇道排布使各区域金属液流同步充型 |
| 卷气 | 无直浇道窝座、流道截面突变、充型速度过高 [[S16,S38]] | 直浇道底部设窝座（直径约为横浇道2倍），优化圆角过渡 |
| 缩孔缩松 | 内浇道提前凝固阻断补缩通道，内浇道位置未匹配热节 [[S38,S22]] | 采用顺序凝固布局，将内浇道设置在厚壁热节附近 |
| 氧化夹杂 | 无稳流设计，充型扰动剧烈导致氧化膜破损卷入熔体 [[S22]] | 采用底注式平稳布局，设置过滤结构，降低紊流度 |
| 冲砂 | 内浇道正对型芯/型壁开设、出口流速超限 [[S38]] | 增设直浇道窝座缓冲，调整内浇道位置，加大截面积降低出口流速 |

## 设计方法演变与工具

浇注系统设计方法经历了从传统经验公式法到数值模拟仿真的发展路径 [[S51,S38]]。

传统方法采用水力学公式计算浇道截面积比，参考经验图表选取参数。现代设计依托专业铸造流体模拟软件开展多物理场耦合仿真优化，大幅降低了试模成本和试制周期。

主流铸造数值模拟软件 [[S51,S42]]：

| 软件 | 方法/特点 | 适应场景 |
|------|----------|----------|
| Magma（德国） | 内置材料数据库或多套专用工艺模块，同步计算充型流场、温度场、应力场和微观组织演化 | 全品类铸造工艺 |
| ProCAST（美国） | 基于有限元方法实现流固热多场耦合，可预判缩孔、卷气、冲砂、冷隔等缺陷位置 | 砂型、金属型、熔模、压铸等 |
| AnyCasting（韩国） | 采用简化高效的求解框架 | 快速完成初步浇注系统迭代优化 |

三类软件均已成为当代铸造厂浇注系统正式投产前的标准验证工具 [[S51]]。
