## 概述

熔盐法在铝合金领域通常又称为混合盐反应法，属于颗粒增强铝基复合材料的原位自生（原位合成）制备工艺[[S1,S5,S17]]。其基本路线是将按比例配置的含目标元素的无机盐加入铝熔体中，盐类在高温下被铝还原，直接在熔体中原位生成陶瓷增强颗粒[[S1,S17]]。该工艺与接触反应法、放热弥散法、自蔓延高温合成法等并列，构成原位合成法的主流实现路径[[S5]]。

已确认通过该路线可生成的典型增强相包括 TiB₂、ZrB₂、TiC、Al₂O₃ 等陶瓷颗粒[[S5,S1,S10]]。

## 工艺原理

熔盐法原位生成增强颗粒依赖一系列热力学驱动的化学反应。以工业上应用最广泛的 K₂TiF₆−KBF₄−Al 体系为例，液态铝与氟钛酸钾、氟硼酸钾发生如下反应[[S29]]：

- 液态铝与 K₂TiF₆ 反应，可生成单质钛或金属间化合物 TiAl₃：
  - 3K₂TiF₆ + 4Al = 3Ti + 4AlF₃ + 6KF
  - 3K₂TiF₆ + 13Al = 3TiAl₃ + 4AlF₃ + 6KF[[S29]]
- 液态铝与 KBF₄ 反应生成 AlB₂：
  - 2KBF₄ + 3Al = AlB₂ + 2AlF₃ + 2KF[[S29]]
- 两种氟盐共同存在时，可直接生成高熔点 TiB₂ 颗粒，总反应为：
  - 6KBF₄ + 3K₂TiF₆ + 10Al = 3TiB₂ + 10AlF₃ + 12KF[[S29,S25]]
  - 在 800°C 下该反应吉布斯自由能 ΔG < 0，可自发进行[[S25]]

上述反应均为强放热反应。KBF₄ 与铝反应的焓变 ΔH₁ = −688.8 kJ/mol，K₂TiF₆ 与铝反应生成 TiAl₃ 的焓变 ΔH₂ = −575 kJ/mol，混合生成 TiB₂ 的总反应焓变 ΔH₃ = −1407 kJ/mol[[S18]]。

反应在铝熔体与盐的界面处发生。体系可大致分为三个区：底部为 750~850°C 的铝液区，中部为 900~1200°C 的高温反应层，顶部为 800~1000°C 的上浮盐渣区。反应放热释放的大量热量使得反应层温度远高于铝熔体初始温度[[S18]]。生成的高熔点 TiAl₃、TiB₂、AlB₂ 等颗粒密度大于铝液，向下沉降进入铝基体；密度较小的副产物盐渣上浮至熔体表层，可隔绝空气防止二次氧化[[S18]]。

## 典型工艺步骤

以工业级熔盐法制备 Al−Ti−B 中间合金为例，常规流程如下[[S24]]：

1. **熔化**：将纯铝锭在石墨坩埚中熔化并升温至 750~900°C。
2. **配盐**：将烘干的 K₂TiF₆ 与 KBF₄ 粉体按比例预混。制备 Al−5Ti−1B 目标产物时，氟盐添加占比约为 30% K₂TiF₆ 和 15% KBF₄[[S18]]。
3. **加盐反应**：将混合盐分批或以控速方式连续加入铝熔体，同时配合机械搅拌或感应电磁搅拌，使氟盐与铝液充分接触反应[[S24]]。
4. **扒渣与除气**：反应完成后，将上浮的液态盐渣倾出分离，对铝熔体进行除气处理，最后浇铸为锭料[[S24]]。

该工艺流程也可直接制备复合材料熔体，除渣后即可对接后续压铸、挤压铸造等工序生产零件，无需单独制备中间合金[[S1]]。

## 关键工艺参数与控制

熔盐法的产品质量受多项工艺参数协同影响，主要控制要点如下：

| 控制项 | 技术要求 | 来源 |
|-------|---------|------|
| 铝熔体初始温度 | 通常设定于 750~900°C 范围 | [[S24]] |
| 盐类型与配比 | 常规为 K₂TiF₆ + KBF₄，优化配方为 30%K₂TiF₆ + 29%KBF₄ + 27%KCl + 14%C₂Cl₆ | [[S30,S6]] |
| 加盐方式 | 分批或控速连续加入，避免一次加入引发剧烈反应、熔盐溢出 | [[S13,S11,S18]] |
| 搅拌方式 | 机械搅拌或感应电磁搅拌 | [[S24]] |
| 反应速率调控 | 通过铝熔体温度、加盐速率、搅拌强度共同控制，防止局部温度过高导致颗粒异常长大 | [[S18]] |
| 添加剂控制 | 还原剂与添加剂质量分数控制在 5%~10%，过量会导致表面粗糙度增加及非预期共渗反应 | [[S13,S11]] |

配方优化的核心目标之一是控制副产物黏度。原始的 K₂TiF₆、KBF₄ 与铝反应会生成低熔点粘稠的 KF、AlF₃，减慢孕育反应进程。改良配方中加入的 KCl 可与 KF 生成熔点仅 605°C 的低共晶体，C₂Cl₆ 分解产生的 AlCl₃、Cl₂ 等气体可促进熔体搅拌，大幅加快反应速率[[S30,S6]]。

此外，在熔盐反应阶段及合金凝固阶段引入超声（空化）辅助处理，可加速盐与铝熔体的反应进程，促进生成的 TiB₂ 颗粒从反应区逸出，将 TiB₂ 颗粒细化至约 1 μm，大幅优化颗粒尺寸分布均匀性、减少团聚，晶粒细化效率可使平均晶粒尺寸从 226 μm 最低降至 157 μm[[S21,S9]]。

## 微观组织特征

采用 K₂ZrF₆ 与 KBF₄ 混合盐与铝熔体反应，可原位生成纳米级、亚微米级及微米级 ZrB₂ 颗粒，颗粒形貌涵盖球形、圆柱形、六边形等多种形态[[S1,S17]]。氟盐以原子态高度弥散分布于熔体中，生成的成核质点数量远多于以中间合金形式加入的体系，所得增强相颗粒尺寸显著更小，细化效果更优[[S30,S6]]。原位反应原位生成的增强颗粒全程未暴露于外界环境，表面无外来污染，与铝基体界面结合良好[[S1]]。

## 力学性能

熔盐法原位生成的陶瓷增强相可显著提升铝基体的力学性能：

- 引入 ZrB₂ 增强颗粒可细化铝基体晶粒，提高显微硬度与抗拉强度，但会降低延伸率[[S1,S17]]。
- 通过 K₂TiF₆ 与 KBF₄ 在 6061 铝合金中制备 5 wt% TiB₂/6061Al 复合材料，与 6061 铝合金相比，抗拉强度可提高约 31.9%[[S5]]。

## 与其他颗粒制备工艺对比

熔盐法、搅拌铸造法、粉末冶金法是三类主流的颗粒增强铝基复合材料制备工艺，其核心维度对比如下表所示。

| 对比维度 | 熔盐法 | 搅拌铸造法 | 粉末冶金法 |
|---------|--------|------------|-----------|
| 制备成本 | 工艺简单，设备投入低，成本低 | 工艺简单，设备投入低，成本低 | 工艺繁琐，成本高 |
| 颗粒尺寸 | 纳米至微米级，原位生成 | 外加 5~100 μm 陶瓷颗粒 | 可低至 200 nm，分布均匀 |
| 界面质量 | 原位生成，界面洁净，结合强度最高 | 易存在界面污染、润湿性差 | 次之 |
| 量产适应性 | 适合大规模量产，可直接浇铸大尺寸构件 | 适合大规模量产，可直接浇铸大尺寸构件 | 工艺繁琐，尺寸和形状受限，量产难度高 |

## 工艺优势与局限

### 优势

- 制备成本低，工艺流程简单，可直接浇铸成型复杂形状铸件[[S1,S17,S5]]
- 增强颗粒在铝熔体中原位生成，全程未暴露于外界，界面清洁无污染，与铝基体结合强度高[[S1,S5]]
- 所得增强颗粒尺寸细小，可显著细化基体晶粒[[S1,S17]]

### 局限

- 难以制备高体积分数颗粒增强复合材料，增强相含量过高时易出现严重颗粒团聚劣化材料性能[[S5,S1]]
- 反应过程中产生大量含氟有毒烟气，需配套烟气回收处理装置，避免污染环境、腐蚀设备、危害操作人员健康[[S18,S5]]
- 残余盐渣不易完全分离，残留于熔体中会形成非金属夹杂缺陷[[S13,S27]]
- 传统硼砂基熔盐流动性较差，工件或铸锭表面残盐清洗困难[[S13,S27]]
- 反应过程中产生大量逸出气体，易导致熔体气孔缺陷[[S13,S27]]
- 氟盐组分化学活性强，对坩埚及熔炼设备存在腐蚀作用[[S13,S27]]

## 相关标准

经查现行 GB、ISO、ASTM 中铝合金铸件及复合材料类标准，未发现专门针对熔盐法制备颗粒增强铝合金的工艺或对应产品的专项标准条文。现有公开标准均为通用铝合金铸件/复合材料的常规通用要求[[S15,S16,S2]]。

## 应用场景

在铝合金铸造领域，熔盐法可直接制备得到复合材料熔体，除渣后即可对接后续压铸、挤压铸造等工序生产零件，无需单独制备中间合金[[S1]]。该路线适合对增强颗粒体积分数要求不极端、对界面结合质量要求高、且希望成本可控的轻量化结构件生产。
