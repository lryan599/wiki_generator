## 概述

气压浇注炉（press pouring furnace）是一种利用压缩空气或惰性气体压力将金属液从密封炉体内定量压出的浇注设备[[S36]]。该术语由国家标准 GB/T 25370-2020《铸造机械 术语》正式定义，归入“浇注设备”类别，条目编号 9.4.5，英文对应名为 press pouring furnace，亦称气压浇注机（pressure-pouring unit）[[S36]]。

其定义原文为：“在密闭的浇注炉顶部，通入压缩空气或惰性气体，通过调控炉内气体压力，将金属液从浇注口压出的浇注机。”[[S36]]《英汉铸造词汇》中同样收录 press pouring furnace 为“气压浇注炉”[[S15]]。

需要特别指出，气压浇注炉与常见的**低压铸造机**（low pressure casting machine）有本质区别：低压铸造机将铸型安放在密封保温炉上方，依靠气体压力使熔融金属沿升液管向上充填铸型，属于反重力充型设备[[S17]]；而气压浇注炉是独立的密闭承压浇注装置，不需要配套固定安装在炉体上方的铸型，依靠气压直接将金属液从炉内浇注口压出，实现定量浇注[[S10]]。

气压浇注炉亦区别于普通保温炉。普通保温炉无密闭承压结构，仅用于维持金属液温度，不具备气压驱动的定量浇注功能[[S44]]。

## 分类

气压浇注炉可按以下三种体系进行分类[[S4,S14,S2]]：

| 分类维度 | 类型 | 说明 |
|---------|------|------|
| 按加热方式 | 电阻加热式 | 运维难度低，广泛用于铝镁合金保温 |
| | 感应加热式（工频/中频） | 熔化效率高、能耗低，适配铜合金、钢铁等高熔点金属 |
| | 燃料（燃气/燃油）加热式 | 适用特定工况 |
| 按炉体结构 | 单室式 | 单一密闭压力腔 |
| | 双室式 | 双腔结构 |
| | 一体式（带自动补料结构） | 集成补料功能 |
| 按压力介质 | 压缩空气驱动型 | 常规应用 |
| | 惰性气体（氮气/氩气）驱动型 | 同步实现金属液防氧化保护，铸造黄铜保温尤为适用 |

GB/T 25370-2020 进一步定义了“气压式保温浇注炉”（automatic press pouring furnace），指带有感应加热装置的一体化气压浇注机[[S36]]。

## 结构与组成

气压浇注炉的核心结构为密闭承压炉体，通过顶部通入压缩气体驱动金属液从浇注口定量压出[[S36]]。主要组成部件分述如下：

**炉体**：整体为密闭压力容器结构。不同工况适配不同材质方案——铸造黄铜用气压浇注炉的工作压力通常为 20~75 kPa[[S46]]；钢水浇注等高压工况需采用带钢制耐压外壳的专用感应炉，以避免大容积承压引发的控制滞后和安全风险[[S25,S35]]。

**炉盖**：需具备足够的强度和刚度，通常采用球墨铸铁制造，配套密封结构保证炉腔承压时无泄漏，预留接口用于接入压力管路、升液管、测温元件等部件[[S18]]。

**坩埚**：盛放金属液的容器。铸铁材质为最通用选型，壁厚要求 30~35 mm，表面需涂刷防护涂料避免铁元素渗入铝合金造成成分污染。对铁杂质含量要求严格的工况可选用石墨坩埚。不同容量型号覆盖 70 kg~800 kg 铝合金装料需求[[S18]]。

**升液管**：导流和补缩的专用通道，与坩埚盖采用可拆卸密封连接。铝合金场景可选用石墨或碳化硅材质，铸钢等高熔点场景需采用特种耐火材质。工作时下端完全浸入金属液面以下，卸压后未凝固金属可依靠自重回流坩埚[[S3,S5]]。

**加热系统**：常见有电阻加热和工频/中频感应加热两类。硅碳棒电阻炉运维难度低，广泛用于铝镁合金保温；工频/中频感应炉熔化效率高、能耗低，适配铜合金、钢铁等高熔点金属的保温浇注工况，配套热电偶测温元件可实时监测炉内温度[[S18,S46]]。

**加压排气系统**：包含空压机、减压阀、调节阀、进气管路、排气阀、压力传感器等部件。需配置两级减压阀降低气源压力波动，压力调节范围通常为 0.16~0.27 kg/cm²[[S28]]。高精度系统通过数字组合阀结合 PLC 实现压力曲线的精确复现，不受气源波动、炉体泄漏和金属液面高度变化影响[[S31]]。

**保温层**：位于炉体外壳和内部坩埚之间的绝热结构，通常采用耐火保温材料填充，减少炉体散热，保证金属液温度稳定，降低加热系统能耗[[S18]]。

**控制系统**：核心为 PLC 或专用电控单元，可实现浇注全流程的压力、温度参数自动调控。部分先进型号配置数据存储功能防止参数掉电丢失，可实现计量浇注周期控制、故障告警、炉内金属液余量提示、与压铸机联动等功能。量产工况下浇注量精度可达 ±1%，温度控制精度可达 ±2%[[S3]]。

## 工作原理

### 基本工艺逻辑

气压浇注炉通过在密闭炉顶部通入压缩空气或惰性气体，调控炉内气体压力，将金属液从浇注口压出实现定量浇注[[S36]]。金属液出液重量/体积由施加压力的大小、保压时长、浴面工作液位和出液口孔径共同决定；成熟商用 Ajax 气压定量浇注装置通过气压和时间双维度控制金属液输出量，可实现 ±2% 的定量精度[[S22,S2]]。

### 浇注全流程的压力阶段

气压浇注完整工艺过程可划分为五个连续阶段[[S26,S9,S6]]：

| 阶段 | 动作逻辑 | 金属液状态 | 控制要点 |
|------|---------|-----------|---------|
| 1. 加压/升液 | 炉内压力按设定速率上升 | 金属液沿升液管平稳上升至浇注口处，无紊流无飞溅 | 控制压力上升速率恒定 |
| 2. 充型 | 继续控制压力上升速率 | 金属液从浇注口自下而上平稳充满整个型腔 | 控制充型速度避免卷气 |
| 3. 保压 | 充型完成后提升至设定保压值并保持预设时长 | 在压力作用下完成金属液补缩和结晶，获得致密组织 | 保压压力与时间决定补缩效果 |
| 4. 卸压 | 开启炉体排气通道释放压力至常压 | 升液管内未完全凝固的剩余金属液依靠自重回流至坩埚 | 防止升液管内金属提前凝固 |

### 压力控制方式

气压浇注系统的压力控制可分为两类[[S26]]：

1. **定流量控制**：浇注阶段固定通入坩埚内的气体体积流量 Q 为常量，直接调控金属液上升速度；
2. **定压力上升速度控制**：浇注阶段固定炉内压力随时间的变化速率 dP/dt 为常量，不受炉内金属液面高度变化影响，可获得稳定的充型过程。

后者是现代气压浇注炉的主流控制策略，有利于铸件质量的批量一致性[[S26]]。

### 定量浇注的两种方式

气压浇注炉的定量实现主要有两种途径[[S22,S2,S33]]：

- **气压-时间控制定量**：通过控制施加气压的大小和保压时长实现重量/体积定量，商用装置定量精度可达 ±2%[[S2]]；
- **浇包容积定量**：通过气压将保温炉坩埚内液态金属压入可调节倾角的定量保温浇包，实现容积式定量[[S33]]。

## 工艺参数与操作条件

### 工作压力范围

气压浇注炉常规工作压力区间为 0.0098~0.05 MPa（约 0.1~0.5 kg/cm²）[[S45]]。部分工况下结晶阶段加压后压力可达 0.5~0.7 kg/cm²（约 0.05~0.07 MPa）[[S42]]。铸造黄铜用气压浇注炉的工作压力通常为 20~75 kPa[[S46]]。

压力过低无法完成正常充型，过高易导致胀砂、跑火等缺陷[[S32]]。

### 浇注温度

不同合金在气压浇注炉工艺中的允许浇注温度范围[[S43,S32]]：

| 合金种类 | 浇注温度范围 |
|---------|------------|
| 铝合金 | 620~730 ℃ |
| 镁合金 | 610~680 ℃ |
| 锌合金 | 410~450 ℃ |
| 铜合金 | 940~980 ℃ |

浇注温度过高会导致晶粒粗大、缩孔倾向提升、粘模风险增加[[S43]]；温度过低则易出现冷隔、浇不足缺陷[[S23]]。实际低压铸造条件下，浇注温度一般比重力浇注低 10~20 ℃[[S32]]。

### 充型速度

气压浇注炉工艺中的充型速度常规参数范围[[S23,S49]]：

| 工艺阶段 | 推荐速度范围 | 说明 |
|---------|------------|------|
| 升液阶段 | 10~15 cm/s | 保证金属液上升时不产生飞溅 |
| 充型阶段 | 2~10 cm/s | 典型范围 5~10 cm/s，保证平稳充型 |

充型速度过快易产生紊流、卷气、氧化夹杂缺陷[[S23]]；充型速度过慢会提升金属液散热速率，大型复杂薄壁铸件易出现冷隔、浇不足问题[[S49]]。

### 保压时间

保压时间指充型结束后保持压力直至浇口部凝固的时间[[S49]]。典型范围：小型薄壁铸件为数十秒，大型厚壁铸件可达数百秒[[S42,S32]]。

保压时间过短会导致补缩不足产生缩孔、金属液回流造成铸件中空报废[[S49]]；保压时间过长会导致升液管内金属提前凝固、铸件难以脱模，降低生产效率[[S32]]。

## 应用范围

### 适配合金

气压浇注炉可覆盖铝、镁、锌、铜等几乎所有常规有色合金[[S40,S12]]，同时也可用于铸铁、铸钢等黑色合金的反重力浇注[[S47,S40]]。对易氧化合金的浇注适配性尤为突出，可有效减少浇注过程中的氧化夹杂[[S40]]。

### 典型产品

气压浇注炉（及其关联的低压铸造工艺）的行业典型铸件产品包括[[S19,S21,S9]]：

- 铝合金轮毂
- 发动机缸盖、气缸体
- 增压器叶轮
- 液力变矩器构件
- 飞行器舱体结构件
- 气动元件阀体

可实现单炉最大 3000 kg 以上铝合金铸件的浇注[[S19]]。

## 优缺点与工艺对比

### 核心工艺指标对比

气压浇注炉配套工艺（低压铸造）与其他主流铸造方法在关键性能指标上的对比：

**材料利用率**[[S11,S29]]：

| 铸造方法 | 材料利用率 |
|---------|-----------|
| 重力铸造 | 45~55% |
| 高压铸造 | 50~60% |
| 气压浇注炉/低压铸造 | 85~95% |
| 差压铸造 | 90~98% |

**铸件致密度**（由低到高排序）[[S30]]：
砂型重力铸造 < 金属型重力铸造 < 砂型低压（气压浇注工艺）< 金属型低压（气压浇注工艺）< 差压铸造

差压铸造致密度最高，铝合金 80 mm 壁厚内可实现无针孔，强度较常规铸造提升约 25%[[S1,S13]]。

**生产效率**（由高到低排序）[[S40,S41]]：
高压铸造（小时产量可达数百件）> 重力铸造 > 气压浇注炉配套工艺（约为高压铸造的 1/4~1/3）> 差压铸造（工艺流程最长，生产效率最低）

**能耗水平**[[S39,S48]]：
铝制品常规形状铸造单位产品能耗约为 2.56 kWh/kg。气压浇注炉配套工艺因未凝固金属可回流至坩埚减少熔化损耗，能耗低于常规重力铸造与高压铸造[[S48]]。差压铸造需对上下双腔同步增压，能耗略高于普通气压浇注炉工艺。

### 主要优势

1. **材料利用率高**：浇注系统简单，剩余金属液可回流，工艺成品率可达 95%[[S19,S29]]；
2. **铸件组织致密**：金属液在压力下充型和结晶凝固，补缩效果好[[S19,S40]]；
3. **充型平稳可控**：充型速度可调，液流平稳，减少卷气和氧化夹杂[[S19,S40]]；
4. **合金适用范围广**：不仅适用于有色合金，也可用于铸铁、铸钢[[S40]]；
5. **金属液品质有保障**：密闭系统减少氧化，惰性气体保护时效果更佳[[S2,S46]]。

### 主要局限

1. **生产效率较低**：约为高压铸造的 1/4~1/3[[S40]]；
2. **对加压控制系统精度要求高**：压力曲线需精确复现，泄漏、液面变化等均需自动补偿[[S31]]；
3. **黑色金属浇注时安全设计要求高**：高密度合金需采用缩小加压容积的承压炉体结构，带耐压钢壳的专用感应加压炉安全性能最优[[S25,S35]]。

## 相关术语与标准

### GB/T 25370-2020 规定的配套术语[[S36]]

| 术语 | 定义 |
|------|------|
| 浇注气压（pouring pressure） | 气压浇注炉在浇注过程中通入炉内的压缩空气或惰性气体的压力（条目 9.4.7） |
| 气压控制装置（pressure control device） | 气压浇注炉控制和调节炉内金属液面上压缩空气（或惰性气体）压力的装置（条目 9.4.8） |
| 气压式保温浇注炉（automatic press pouring furnace） | 带有感应加热装置的气压浇注机（条目 9.4.6） |

### 与邻近概念的边界澄清

- **气压浇注炉 与 低压铸造机**：低压铸造机是将铸型安放在密封保温炉上方，依靠气压使金属液沿升液管向上充填铸型[[S17]]；气压浇注炉为独立密闭承压浇注装置，不需要配套固定安装在炉体上方的铸型，直接通过浇注口压出金属液实现定量浇注[[S10]]。
- **气压浇注炉 与 普通保温炉**：普通保温炉无密闭承压结构，不具备气压驱动的定量浇注功能[[S44]]。
