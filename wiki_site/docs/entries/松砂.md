## 概述

在铸造及压铸相关技术语境中，“松砂”一词具有多义性，其含义需根据具体应用场景进行区分。当前工程实践中主要存在三类标准定义：其一指砂处理工艺流程中，将混制后结块的型砂进行破碎松散的操作工序；其二指铸件上因残留松散型砂未清理干净或砂芯表面强度不足形成局部疏松孔洞的缺陷形态；其三指无粘结或粘结强度极低、呈自由松散状态的铸造用砂材料 [[S9,S14,S18,S12]]。在带砂芯压铸生产场景下，松砂特指合模前压铸模具型腔与砂芯间隙内残留的未压实松散型砂，属于需清理除去的异常物料 [[S12]]。

## 定义与分类

根据工程语境，松砂的标准定义分属以下三类场景 [[S9,S14,S18,S12]]：

1. **砂处理工艺场景**：指混砂后将结块型砂打散，使其松散均匀的砂处理操作，归属于砂处理工艺流程的后混砂环节。该操作旨在消除型砂团块，使粘结剂分布均匀，提升型砂的综合工艺性能。
2. **铸件缺陷场景**：指铸件上因残留松散型砂未清理干净，或砂型/砂芯表面强度不足掉落散砂混入金属液，在铸件内部或表面形成的局部疏松孔洞缺陷。此类缺陷在传统砂型铸造和带砂芯压铸工艺中均有出现。
3. **材料场景**：指无粘结或粘结强度极低、呈自由松散状态的铸造用砂。此类松砂常出现于3D打印砂型未固化区域、砂芯清砂后余料中，亦可用作消失模铸造砂箱的填充支撑材料。

## 工艺应用与设备

### 工艺作用

作为砂处理工艺的松砂操作，其正向作用在于：打散混制后形成的紧实砂团，使型砂颗粒之间的粘结剂分布更趋均匀，有效提升型砂的透气性、流动性与成型性，避免后续造型工序出现砂型紧实度不均的问题 [[S9,S18]]。

### 操作步骤

常规松砂操作的完整步骤依次为 [[S18,S12]]：

1. 混砂卸料后将带有结块的型砂送入松砂设备进料口；
2. 通过高速旋转叶轮或梳齿的击打、抛射作用破碎紧实砂团；
3. 松散后的型砂经由出料口排出，进入后续筛分工序，最终得到无结块的均匀松散型砂。

### 设备类型与技术参数

根据设备结构与适用场景，松砂机可划分为多种类型，其工作参数与应用范围各异。

**表1：松砂机类型与应用**
| 类型 | 工作原理 | 特点 | 适用场景 |
|------|----------|------|----------|
| 轮式松砂机 | 松砂轮将胶带上的型砂扬起抛出，经链条疏松 | 生产率高，松砂可靠，可直接安装于输送带上，不占车间面积 | 大、中型自动化砂处理系统 |
| 叶片式松砂机 | 叶轮将型砂抛射至链条上进行松砂 | 生产率高，固定密封，松砂可靠；需另占用车间面积 | 中、大型砂处理系统 |
| 带式移动松砂机 | 带横条的胶带将型砂抛射至筛网上松砂 | 可随地移动，生产率较小 | 小型铸造车间，就地处理背砂 |
| 梳式松砂机 | 安装于型砂输送机上直接进行松散 | 结构紧凑，不额外占用车间面积 | 型砂输送线 |

轮式松砂机的典型工作参数 [[S20]]：生产率最高可达 **200 m³/h**；松砂叶轮直径为 **260~500 mm**；叶轮外沿线速度约 **12 m/s**，对应转速为 **500~820 r/min**；允许通过砂层厚度为 **25~100 mm**；叶轮与输送带间隙控制为 **5~8 mm**；叶轮工作缘镶有硬质合金。

### 工艺优化要点

为保证松砂效果并维持设备稳定运行，需关注以下操作要点 [[S20,S24,S12]]：

- 定期校准松砂机叶轮与输送带的间隙，保证松散效果一致性；
- 及时更换磨损的硬质合金叶轮镶边，避免松散效率下降；
- 在砂处理流水线中，合理布置双轮松砂机于混砂机卸料口下游，直接对接后续筛分工序，避免松散后的型砂在输送过程中二次结团。

## 材料属性

### 性能指标

松散无粘结或低粘结强度的松砂材料，其核心性能指标范围如下 [[S10,S3,S19,S21]]：

| 性能指标 | 典型范围/数值 |
|----------|---------------|
| 粒度分布集中区间 | 50/75/100 目筛网 |
| AFS平均细度 | 53~56 |
| 湿压强度 | ≤ 0.02 MPa |
| 透气性 | ≥ 100 |
| 松散密度 | 1.4~1.5 g/cm³ |
| 含泥量 | ≤ 0.3% |
| 发气量 | < 5 cm³/g |

注：常规造型用型砂的湿压强度一般要求 0.08~0.22 MPa，松砂材料的湿压强度远低于此区间 [[S3]]。

### 典型应用场景

材料类松砂的核心应用场景包括 [[S14,S23,S4]]：

- **3D打印砂型后处理**：PCM等工艺制造的铸型，型腔内部在造型结束后充满未固化的松砂，需通过倒置倾倒或真空抽吸方式清理取出。
- **消失模铸造填砂**：消失模铸造砂箱采用低粘结松散砂作为填充支撑材料，便于后续落砂工序快速溃散取出。

## 缺陷形态与机理

### 形貌特征

松砂缺陷的典型形貌表现为铸件局部出现充填散砂的不规则孔洞，孔洞内壁粗糙，无金属连续结晶层。此类缺陷常出现在以下部位 [[S15,S6]]：

- 金属液流直接冲刷砂芯的部位；
- 压铸模具分型面缝隙周边；
- 铸件厚大拐角区域。

### 形成机理

松砂缺陷的形成过程为：松散未粘结的砂粒未被完全排出型腔，在高速充型过程中被金属液裹挟进入铸件内部，冷却后在对应位置形成内嵌散砂的孔洞缺陷。可诱发该缺陷的典型工艺参数范围包括 [[S22,S8,S6]]：

- 砂芯固化时间不足设计值的 **70%**，导致砂芯表面强度过低；
- 合模前型腔清砂吹扫时间不足 **3 s**，残留游离散砂；
- 压铸充型速度超过工艺推荐上限 **30%** 以上，将松散砂粒冲入型腔。

### 危害

松砂缺陷对铸件质量的危害包括 [[S6,S11]]：直接破坏铸件内部组织连续性，降低铸件有效承载截面积；导致耐压铸件出现渗漏；降低铸件力学性能；增加后续清砂、打磨工作量；严重时直接导致铸件报废。

### 预防控制措施

完整的松砂缺陷预防控制方案包括 [[S15,S7,S22]]：

1. 严格管控砂芯固化保温时间，确保砂芯终强度达标；
2. 合模前增加高压空气吹扫步骤，完全清理型腔与砂芯缝隙内的残留游离散砂；
3. 调整压铸充型速度至工艺推荐区间，避免过高流速冲刷砂芯表面造成掉砂；
4. 定期检查砂芯脱模后转运过程是否存在磕碰掉砂问题。

## 相关关系与对比

松砂与夹砂、落砂、砂芯溃散、砂眼等概念在铸造工程中频繁交叉出现，需进行明确区分 [[S5,S8,S6,S26,S22]]：

| 概念 | 定义要点 | 与松砂的关系 |
|------|----------|--------------|
| 松砂（工艺/材料） | 属于砂处理工序正常操作或正常物料 | — |
| 夹砂 | 高温金属液热膨胀将砂型表层拱起脱落，在铸件表面形成带凸起疤痕的缺陷 | 均与型砂松散脱落有关，但夹砂发生于充型阶段且由热膨胀驱动，属缺陷；松砂（工艺）属正常操作 |
| 落砂 | 浇注冷却后强制分离铸件与型砂的后处理操作 | 属后续工序，与松砂（工艺）分处铸造流程的不同阶段 |
| 砂芯溃散 | 浇注后砂芯受热强度大幅降低而自动解体的材料性能表现 | 属有利特性；过度的表面松散则可能转化为松砂缺陷 |
| 砂眼 | 铸件内部或表面充塞型砂的孔洞类缺陷 | 松砂清理不到位是诱发砂眼的典型原因之一 |
