## 概述

重力因数在铸造技术中具有两类明确的物理定义与工程用途。第一类属于重力浇注系统范畴，是用于修正重力驱动下金属液实际流速与理论流速差异的流量系数；第二类专用于离心铸造，表征离心力场对金属液产生的加重倍数。中国国家标准GB/T 5611-1998《铸造术语》以“gravity factor”条目正式收录该术语，对应中文命名为重力因数/重力系数 [[S3]]。

## 定义与物理意义

### 重力浇注系统中的重力因数

在重力铸造充型过程中，熔融金属由浇口杯经直浇道、横浇道、内浇道流入型腔。金属液在流动中因黏性摩擦、流道截面变化、方向改变等因素产生压头损失，使实际流速低于理想流速。重力因数μ定义为实际金属液流量与理想流体流量之比值，物理实质是表示有黏度的实际金属液与无黏度理想流体流量的比值 [[S7,S6]]。

数学表达式为：

\[
\mu = \sqrt{\frac{1}{1 + \sum\xi_{i}\left(\frac{F_{i}}{F_{\text{阻}}}\right)^{2}}}
\]

式中：ξ<sub>i</sub>为浇注系统中局部阻力系数；F<sub>i</sub>为对应某局部的截面积；F<sub>阻</sub>为阻流截面积 [[S7]]。

由伯努利方程推导得出的实际流速公式为 v = μ√(2gH<sub>0</sub>)，流量公式为 Q = μA√(2gH<sub>0</sub>) [[S9,S6]]。

### 离心铸造中的重力系数

离心铸造条件下，旋转铸型中的金属液除受普通重力外还受离心力作用。单位体积金属液在离心力场中受力大小为q = k·γ，其中k = ω²r/g，称为重力系数。其物理意义是在离心力作用下物体被加重的程度，k值越大表示金属液在充型时所受离心力相对于重力的放大倍数越高 [[S12]]。

重力系数G的记法为：

\[
G = \frac{\omega^{2}r}{g} = \frac{n^{2} \cdot r}{980}
\]

式中：ω为旋转角速度（rad/s）；r为旋转半径（cm）；n为转速（r/min）；g为重力加速度 [[S13]]。

常规生产中重力系数取值范围为30~100，远大于1，因而金属液在离心方向上的流动性显著高于普通重力铸造 [[S12,S13]]。

## 表达式体系

重力因数相关表达式基于伯努利定律推导而来。以浇口杯液面和阻流截面建立能量平衡方程，经简化整理后得到：

**流速方程：**  
v<sub>阻</sub> = μ√(2gH<sub>0</sub>) [[S9,S7]]

**流量方程：**  
m = t·ρ·F<sub>阻</sub>·μ·√(2gH<sub>0</sub>) [[S7,S9]]

**阻流截面积计算公式：**  
F<sub>阻</sub> = m / (ρ·μ·t·√(2gH<sub>0</sub>)) [[S7,S6,S9]]

在压铸领域，同类参数称为流量系数C<sub>n</sub>，取值范围为0.4~0.8。压铸模需求压力线方程为：

\[
P = \frac{\rho}{2gC_{n}^{2}S_{n}^{2}} \cdot Q^{2}
\]

式中：P为冲头压射压力；S<sub>n</sub>为内浇口截面积；Q为填充流量；C<sub>n</sub>为浇注系统流量系数 [[S1]]。

## 典型数值范围

重力浇注系统的效率系数C（即重力因数μ的等效表达）因浇注系统类型不同而有不同的经验取值。英文专业教材给出了各种典型浇注系统的效率系数推荐值 [[S10,S4,S8]]。

**各类浇注系统效率系数C值 [[S8]]：**

| 浇注系统类型 | 系统特征 | C值 |
|---|---|---|
| 直浇道带锥形浇口塞 | 单浇道 | 0.85 |
| 直浇道带直浇道塞 | 单浇道 | 0.80 |
| 直浇道带锥形浇口塞 | 双浇道，带90°弯管 | 0.76 |
| 直浇道带直浇道塞 | 双浇道，带90°弯管 | 0.73 |

带直浇道塞的封闭式浇注系统C值整体高于带90°弯管的双浇道开放式浇注系统的对应数值 [[S8,S10]]。

对于铝合金铸件，在非增压式浇注系统中，直浇道面积每1 mm²对应的金属流量推荐为4.044 g/min [[S10]]。

压铸流量系数C<sub>n</sub>的经典取值范围为0.4~0.8 [[S1]]。

## 影响因素

重力因数的数值受浇注系统各组成元件的局部阻力系数K<sub>i</sub>及对应截面积变化的综合影响，其严格计算公式为：

\[
C = \frac{1}{\sqrt{1 + K_{1}\frac{A^{2}}{A_{1}^{2}} + K_{2}\frac{A^{2}}{A_{2}^{2}} + \dots}}
\]

式中：K<sub>1</sub>、K<sub>2</sub>……为方向变化或截面积变化处的损失系数；A<sub>1</sub>、A<sub>2</sub>……为变化位置下游的截面积；A为阻流截面积 [[S10,S4]]。

从表达式中可归纳出主要影响因素：
- 流道截面形状与面积变化梯度（A/A<sub>i</sub>项）
- 流道方向改变次数与幅度（影响K<sub>i</sub>值）
- 流道交汇与分流结构（影响∑项累积）
- 浇注系统元件表面粗糙度与雷诺数（隐含在局部阻力系数中）[[S10,S7,S6]]

浇注系统元件横截面应为圆形，因为圆形截面的表面积与体积之比最小，有利于减少热损失和摩擦损失。此外，对流道各元件进行流线型化处理可大幅提高浇注系统的容积效率 [[S4]]。

离心铸造重力系数受转速n与旋转半径r的直接控制。生产中一般通过调节转速来改变G值，G值在30~100范围内取值常见 [[S12,S13]]。

## 工程应用

### 阻流截面积计算

在浇注系统设计中，重力因数是计算阻流截面积的关键参数。对于给定铸件，先由铸件重量、浇注温度和碳当量确定浇注时间，然后选取匹配浇注系统类型的效率系数C值，最终计算阻流面积。计算实例表明，当选用封闭式浇注系统、双横浇道配四个内浇口时，C值取0.73，可据此求得各内浇口的具体尺寸 [[S10]]。

### 直浇道有效高度校验

阻流截面流速由有效压头H<sub>0</sub>与重力因数共同决定。直浇道设计为锥形，锥度依据连续性方程确定：A<sub>t</sub>=A<sub>c</sub>√(h<sub>c</sub>/h<sub>t</sub>)。这种设计可补偿金属液下落过程中速度增加所引起的截面收缩，避免吸气卷气 [[S4]]。

### 充型平稳性控制

合理选取重力因数可降低充型过程中的紊流度，减少卷气、夹渣类缺陷的产生 [[S10,S4]]。封闭式浇注系统的效率系数高于开放式系统，在铸件质量要求较高的场合优先选用 [[S8]]。

离心铸造通过设定重力系数G来提高金属液的充型能力。由于G值通常为30~100，液态金属具有比重力作用下更高的流动性，能够充填更为细薄的型腔，这是熔模精密铸件在离心力作用下浇注的主要优势 [[S12]]。离心铸造工艺还可显著细化铸件晶粒，各层的平均晶粒尺寸均小于同条件下普通重力铸造的对应数值 [[S5]]。

## 压铸领域的对比

压铸工艺中虽不直接使用“重力因数”一词，但其浇注系统流量系数C<sub>n</sub>具有相同的物理本质——表征实际流动与理想流动之间的效率损耗。压铸P-Q²图中需求压力线的斜率k=ρ/(2gC<sub>n</sub>²S<sub>n</sub>²)，明确了浇注系统流动效率与流量系数C<sub>n</sub>的直接关系 [[S1]]。

压铸数值模拟软件（如FLOW-3D）在物理选项设置中需同时勾选重力模块与粘度、紊流模块，使流动求解器正确计入重力与能量损失的综合作用 [[S11]]。重力铸造仿真软件ProCAST同样需要设置重力加速度（默认9.81 m/sec²）与重力方向（通常为-Y轴），为精确模拟充型流动提供基础物理场参数 [[S2]]。

## 与相关概念的关系

| 概念 | 关系 | 说明 |
|---|---|---|
| 流速系数 | 等价/同类 | 对应表达式v=μ√(2gH<sub>0</sub>)中的μ，表征实际流速与理论流速之比 [[S9,S7]] |
| 流量系数 | 等价/同类 | μ=实际流量/理想流量，与流速系数数值相同 [[S7,S6]] |
| 效率系数C | 同义 | 英文教材中使用的命名，与重力因数µ物理含义一致 [[S10,S4,S8]] |
| 阻力系数ξ | 构成因子 | 各流道元件的局部阻力系数，求和折算后构成重力因数分母项 [[S7,S10]] |
| 收缩系数 | 关联但区别 | 收缩系数仅考虑截面收缩带来的流量损失，重力因数覆盖全部压头损失构成 [[S7]] |
| 压铸流量系数C<sub>n</sub> | 同族参数 | 压铸领域对应名称，取值范围0.4~0.8，物理来源一致 [[S1]] |
