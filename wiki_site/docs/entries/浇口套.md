## 概述

**浇口套**（英文：sprue bushing）是压铸模具浇注系统中的核心组件，属于定模侧嵌入的工具钢衬套，内部设有直浇道通道，负责引导高压、高温液态金属从压室平稳流入模具型腔[[S12,S13]]。该零件持续承受高温高压和高速金属液的反复冲蚀，其设计、制造与维护状态直接影响压射性能、铸件质量及模具整体寿命[[S10,S16]]。

在北美NADCA行业标准中，通用术语为 **sprue bushing**；国内压铸领域常用别名包括 **唧嘴** 和 **浇口衬套**。此外，**sprue bush** 与 **gate bushing** 亦为压铸行业通用英文叫法；**sprue insert** 特指嵌入型浇口套结构[[S12,S13]]。

需要区分的是：注塑模具中的“唧嘴”功能为连接注塑机喷嘴与塑料模流道，其内锥度更小、承压要求远低于压铸浇口套，两者不可等同[[S13]]。


## 结构组成与分类

### 典型结构要素

压铸浇口套的典型结构要素包括[[S8,S17,S13]]：
- 带脱模斜度的锥度内孔（直浇道通道）
- 定位法兰（用于与定模安装板或压铸机压室法兰配合）
- 可供选配的内部冷却水道（环绕内孔，用于调节热平衡）
- 定位止口或定位缺口（防止转动或保证安装方向）

### 分类方式

浇口套可按多种维度进行分类[[S8,S17,S18]]。

**按适配压铸机类型：**

| 类型 | 说明 |
|------|------|
| 立式冷室压铸机用浇口套 | 与喷嘴球面或圆锥面对接，常配合分流锥使用 |
| 卧式冷室压铸机用浇口套 | 与压室对接，位于定模侧，应用最广 |
| 热室压铸机用浇口套 | 内孔锥度约5°（单边），小端与鹅颈喷嘴球面配合 |

**按装配形式：**

| 形式 | 特征 |
|------|------|
| 整体式 | 直浇道由浇口套一体加工而成，金属液流动顺畅，装拆方便；其中带压铸机定位止口的整体式浇口套可完全消除安装装配误差，保证直浇道与压室内孔的同轴度 |
| 分体式（套接式） | 直浇道由浇口套与定模镶块共同构成，增加接合面，易产生横向飞边 |

**按是否带冷却结构：**

| 类型 | 特征 |
|------|------|
| 普通型 | 无内部冷却水道 |
| 水冷型 | 环绕浇口套外径或内孔设有冷却水通道，可改善模具热平衡条件，有利于提高压铸生产率 |

### 常用材料

浇口套常用热作模具钢制造，以保证在高温、高压及反复冲蚀工况下的服役寿命[[S3,S4,S11]]：

### 表面处理与制造工艺

- **表面氮化处理**：行业通用强化工艺，可显著提升内孔抗高温金属冲蚀的性能[[S11]]。
- **内孔加工**：内孔在热处理和精磨后，应沿脱模方向进行镜面研磨，表面粗糙度要求不大于Ra0.2μm[[S7]]。
- **圆弧配合面**：浇口套与压室筒内表面的圆弧配合面须严格保证公差；制造最后一道工序为圆弧磨削[[S4]]。


## 关键尺寸参数与配合精度

### 内孔几何参数

| 参数 | 推荐值/要求 | 说明 |
|------|-------------|------|
| 内孔单边脱模斜度 | 2°~6°（立式/卧式）；约5°（热室） | 保证直浇道余料顺利脱模，不应出现倒拔角 |
| 内孔表面粗糙度 | ≤ Ra0.2μm | 经精磨后沿脱模方向镜面研磨 |
| 浇口套内径与压室内径关系 | 浇口套内径比压室内径大0.04~0.06mm | 弥补不同轴度偏差，防止冲头卡死 |

以上参数依据文献[[S7,S16,S17,S13]]。

### 配合公差

浇口套与各相关部件的推荐配合公差[[S7,S8]]：

| 配合部位 | 配合精度 |
|----------|----------|
| 浇口套外径与定模安装孔 | H7/h6 |
| 浇口套内孔与压射冲头 | F7/e8 |
| 定位孔与压铸机压室定位法兰 | E8 |
| 压室内径与压射冲头 | H7/e8 |

### 喷嘴对接参数（热室/立式）

- 球面对接时，浇口套凹球面半径应略大于喷嘴端部球半径，即 SR = Sr + 0.4mm，以利于球面中心部位的紧密对接[[S17]]。
- 直浇道入口处孔径D应大于喷嘴出口孔直径d，即 D = d + 1mm[[S17]]。


## 工艺角色

浇口套在压铸全流程中的核心功能是连接压室与横浇道，引导熔融金属平稳充填型腔[[S10]]。其工艺角色主要体现在以下方面：

- **连接与导向**：位于压室末端与模具浇注系统之间，构成高压金属液进入模具的第一段通道。
- **承受极端工况**：持续承受高压、高速液态金属的反复冲蚀及热冲击[[S14]]。
- **影响压射性能**：若尺寸精度或装配出现偏差，将导致压射压力和速度显著损失，直接影响充型稳定性与铸件内部质量[[S16]]。
- **决定模具寿命**：作为高频率接触高温金属的易损件，其状态直接关联整套模具的使用寿命[[S10]]。

### 接口关系

浇口套与以下关键部件存在严格的配合关系[[S8,S9,S16]]：
- **压射冲头（锤头）**：间隙配合，内孔配合精度F7/e8
- **压室（料筒/熔杯）**：端面密封对接或定位法兰配合
- **浇道镶块**：形成完整的直浇道通道
- **分流锥/分流器**（立式/热室用）：调整直浇道截面，改变金属流向


## 常见失效模式与相关缺陷

### 浇口套自身失效模式

| 失效模式 | 机理 | 导致的产品/过程缺陷 |
|----------|------|---------------------|
| 内孔磨损 | 高速金属液反复冲蚀 | 配合间隙增大，冲头卡死，压射缺料→铸件冷隔 |
| 热疲劳裂纹（热裂） | 表面与内部温差引起的反复热应力 | 碎屑混入金属液→铸件夹杂；裂纹与冷却水路贯通→漏水 |
| 装配位移/脱落 | 固定不牢或受热松动 | 直浇道错位→金属液充型偏斜→铸件流痕 |
| 内部堵塞 | 残留金属屑积累 | 通道截面减小→充型不足→铸件欠铸 |
| 冷却水通道开裂漏水 | 热应力或加工缺陷 | 水渗入型腔→铸件内部气孔超标 |

以上信息依据文献[[S14,S9,S2]]。

### 生产过程中的漏水隐患点

浇口套漏水的主要原因包括[[S14]]：
- 浇口套与压室的法兰结合面渗水
- 冷却水管安装不当导致水从安装孔流入压室
- 浇口套出现裂纹，冷却水从裂缝渗入

一滴水接触高温铝液后瞬间蒸发为气体，体积膨胀1000倍以上，是铸件气孔的重要成因[[S14]]。


## 安装、维护与更换

### 安装配合要求

- 安装前须检查浇口套内孔的尺寸和表面粗糙度，孔口（含肩台孔口）应无磕碰和磨损[[S16]]。
- 安装后用冲头从分型面处放入内孔，试探在浇口套与压室的交接处是否有阻塞感[[S16]]。
- 浇口套、压室和压射冲头三套配件建议选择同一家供应商，以保证三者最佳的配合效果[[S9]]。

### 日常维护

| 维护事项 | 要求 |
|----------|------|
| 档案管理 | 建立生产模次档案，记录编号、模次、尺寸测量数据，追踪磨损状态 |
| 预加热 | 生产前与模具整体同步预热 |
| 吹气清理 | 每批次生产完毕后对内孔进行吹气，清除残留金属屑 |
| 冷却水检测 | 每批次生产完毕后进行试水作业，确保水路畅通及流量 |
| 磨损监控 | 定期检测内孔磨损量（如3-9点、6-12点位置），判断是否在可接受范围内 |

以上维护规范依据文献[[S2,S4,S16]]。

### 更换标准

- 当内孔磨损量超过设计允许偏差范围时，须强制更换[[S2]]。
- 当热裂纹深度超过0.2mm时，须强制更换[[S2]]。
- 发现冷却水通道裂纹漏水时，须立即更换[[S14]]。
