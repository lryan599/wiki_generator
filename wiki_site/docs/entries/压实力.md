## 概述

压实力是铸造机械领域经国家标准正式定义的术语[[S10]]，指压实工序中作用于被压实材料（如型砂、金属液）的力。在压力铸造与挤压铸造工艺中，压实力泛指在金属液充型结束后，于凝固阶段通过压射/挤压系统施加于金属液使之致密化的力，是实现铸件补缩、消除内部缩孔缩松缺陷的核心过程参数。该术语在GB/T 25370-2020《铸造机械 术语》中编号为4.1.55，与压实比压（4.1.54）、压实行程（4.1.52）、压实（2.22）共同构成压实工序的标准术语体系[[S10]]。

压实力与压射力（充型阶段推动金属液填充型腔的力）和锁模力（合模机构抵抗胀型力的力）在概念上存在本质区别：压实力作用的时间窗口处于金属液充型完成后的凝固-补缩阶段，其直接目的是通过高压迫使半固态枝晶间的液态金属充填收缩空隙，而非单纯推动金属液流动或保持模具闭合。

## 计算公式

对于配置增压机构的常规冷室压铸机，压实力（或增压力）的大小取决于液压系统工作压力及各缸径的几何放大关系[[S6,S9]]：

```
F = p_g × (πD²/4) × (D₁/d₁)²
```

式中：
- F——压实力/增压力，N
- p_g——液压系统工作压力（或蓄压器工作压力），MPa
- D——压射缸直径，mm
- D₁——增压缸直径，mm
- d₁——增压活塞杆（小端）直径，mm

该公式表明，压实力与压射缸直径平方及增压缸直径平方成正比，与增压活塞杆直径平方成反比[[S6,S9]]。作用于压室内金属液上的实际增压比压则进一步需要考虑压射冲头直径的影响，其与压射缸及增压缸直径的平方成正比，与压射冲头及增压活塞杆直径的平方成反比[[S6]]。

## 工艺作用与机理

压实力的核心作用机制在于：在合金凝固阶段，当铸件内部已形成枝晶骨架但仍存在未完全凝固的液态金属时，通过外力迫使这部分液态金属穿过枝晶间隙，充填因凝固收缩而产生的空隙，从而消除缩孔缩松缺陷[[S3,S11]]。同时，高机械压力下的凝固过程有利于细化晶粒组织，提升铸件致密度与力学性能[[S3,S11]]。

实验研究表明，压实力存在一个“临界压力”阈值效应[[S3]]：当压实力低于该临界值时，铸件内部缩孔体积随压实力降低而显著升高，补缩效果不充分；一旦达到临界压力以上，可完全消除宏观缩松缺陷[[S3]]。临界压力值与铸件形状、尺寸、挤压部位及合金材料等多种因素相关。一般情况下，直接冲头挤压的临界压力大于60 MPa，柱塞挤压的临界压力大于80 MPa[[S3]]。

## 典型工艺参数范围

### 不同合金挤压铸造比压推荐值

挤压铸造工艺中，比压（单位面积上的压实力）是决定铸件质量的首要参数[[S8]]。不同合金种类与挤压方式对应不同的比压推荐范围[[S8]]：

| 工艺类别 | 合金类型 | 推荐比压范围（MPa） |
|---------|---------|-------------------|
| 直接冲头挤压铸造 | 非铁合金 | 25~50 |
| 柱塞/间接冲头挤压铸造 | 非铁合金 | 60~100 |
| 挤压铸造 | 黑色金属 | 约为非铁合金的2倍 |

### 不同压铸合金比压推荐范围

常规压铸工艺下，各合金种类的压实比压典型范围如下[[S8]]：

| 合金种类 | 推荐比压范围（MPa） |
|---------|-------------------|
| 铝合金 | 20~100 |
| 锌合金 | 10~70 |
| 镁合金 | 25~100 |

## 直接挤压与间接挤压铸造中压实力的差异

压实力在直接挤压铸造与间接挤压铸造两种工艺中的作用特征存在显著差异[[S2,S11]]：

| 对比维度 | 直接挤压铸造 | 间接挤压铸造 |
|---------|------------|------------|
| 施力方式 | 冲头直接作用于铸件本体 | 压力通过料饼、内浇道传递至铸件 |
| 传力路径 | 极短，压力损耗极小 | 较长，存在明显的压力损失 |
| 所需压实力阈值 | 相对较低（比压25~50 MPa） | 相对较高（比压60~100 MPa） |
| 金属液定量要求 | 必须精确定量 | 无需精准控制浇注量 |
| 适用铸件 | 形状简单的对称结构件 | 形状较复杂的铸件 |

直接挤压铸造中，冲头将压力直接施加于铸件本体，传力路径短，压力损耗极小，因而所需压实力阈值更低，无需额外压力补偿[[S2,S11]]。间接挤压铸造中，压实力需通过料饼和内浇道传递，路径长，压力损失明显，因此要求的压实力阈值更高，但其优势在于无需精确控制金属液浇注量，且适配形状更复杂的铸件[[S2,S11]]。

## 压实力不足与过高的缺陷影响

压实力不在合理区间时将引发对应类型的铸造缺陷[[S1,S3]]：

**压实力不足时：**
- 铸件热节区域出现缩松、缩孔，表面对应位置可能出现凹陷[[S3]]
- 内部组织致密度不足，引发耐压渗漏失效[[S1]]
- 力学性能（抗拉强度、硬度等）大幅下降[[S1]]

**压实力过高时：**
- 超出模具锁模负载能力，引发胀型飞边[[S3]]
- 加剧模具磨损，缩短模具使用寿命[[S3]]
- 部分薄壁铸件可能出现铸造裂纹[[S3]]
