液压传动系统是压铸机（含液锻机/液态模锻机）的核心动力与动作控制系统，主要由动力元件、执行元件、控制元件、辅助元件和工作介质五部分组成，通过各类液压元件和回路传输动力，驱动锁模、压射、顶出、抽芯等机构完成预设动作程序[[S20,S47,S61]]。在压铸设备中，液压系统决定压射速度、生产率和运行可靠性，通常被称为压铸机的“动脉”[[S38,S24]]。对于液锻机，液压传动系统同样是其核心组成部分，负责为锁模、充型、挤压保压等工序提供动力[[S51,S32]]。

## 分类
压铸机液压传动系统可按传动形式和元件安装位置进行分类[[S38,S44]]。

### 按传动形式分类
- **分组集中传动**：多台压铸机共享公用泵-蓄能器站供给高压工作液，适合大规模连续生产的专业压铸厂，具有设备投资低、动力利用率高的优点。
- **单机传动**：单台压铸机配备独立的高压工作液源，运行灵活不受公共泵站限制，是当前主流配置[[S38]]。

### 按元件安装位置分类
- **内装式**：液压元件与压铸机刚性集成连接，整机外形整齐美观，但元件维护与管路连接难度大。
- **外装式**：液压元件仅通过电气线路、液压管路与主机相连，检修更换方便，是现有压铸机的主流布局形式[[S38,S44]]。

## 系统组成
液压传动系统由五类基本元件组成[[S20,S4]]。

| 元件类别 | 包含项目 | 功能 |
|---------|---------|------|
| 动力元件 | 液压油泵 | 将电动机的机械能转换为液压油的压力能 |
| 执行元件 | 液压缸、液压马达 | 将液压油压力能转换为驱动工作部件的机械能 |
| 控制元件 | 方向控制阀、压力控制阀、流量控制阀 | 调节系统内油液的流向、压力与流量 |
| 辅助元件 | 油箱、过滤器、蓄能器、冷却器、管件、密封装置等 | 保障系统长期稳定运行 |
| 工作介质 | 液压油（阻燃型液压介质） | 作为能量传递与控制信号传输的载体[[S20,S4,S21]] |

> 注：压铸机液压系统强制要求使用阻燃型液压介质（如符合ISO 7745标准的水乙二醇），蓄能器只允许充入氮气[[S21,S22]]。

## 工作原理与关键回路
压铸与液锻设备液压系统采用多种专用回路，以满足高压大流量、快速响应及保压补缩等工艺需求。

### 高压大流量供给回路
常用高低压组合泵配置实现高压大流量供给。典型方案为1台高压柱塞泵配合2台低压大流量齿轮泵：空程/回程阶段3台泵同时供油实现快速动作；工作加压阶段系统压力升高后，低压大流量泵自动卸荷，仅高压泵单独供油。配合多组大容积蓄能器可瞬时释放数百升/分钟的大流量，满足快压射与液锻高速充型需求，系统工作压力最高可达32MPa[[S26,S60]]。

### 保压回路
普遍采用液控单向阀与蓄能器联合保压回路。压射充型完成后液压泵可切换至卸荷状态，由密闭回路内的液控单向阀锁止油液、蓄能器补充微小泄漏，长时间维持型腔内高压状态，满足合金凝固过程中的补缩需求。保压10分钟的系统压力降符合压铸工艺对铸件致密性的要求，可有效减少缩孔疏松缺陷[[S27,S37]]。

### 卸荷回路
在执行元件待机或保压阶段，卸荷回路将油泵负载降至几乎为零，减少功率消耗、降低油温升高速度。常用三种结构[[S54,S3]]：
- **M型或H型中位机能三位四通电液换向阀卸荷**：中位直接连通进回油路实现卸荷。
- **电磁溢流阀远程控制式卸荷**：通过二位二通电磁阀打开溢流阀远程控制口实现零压回油。
- **二位二通大流量阀直接卸荷**：将油泵出口直接连通油箱。

## 压铸/液锻工艺对液压系统的特殊要求
与通用工业液压系统相比，压铸与液锻专用液压系统具有以下特殊配置与要求[[S44,S28,S41]]：

| 类别 | 通用工业液压系统 | 压铸/液锻专用液压系统 |
|------|-----------------|---------------------|
| 蓄能器配置 | 常规蓄能器 | 独立的快压射蓄能器 + 增压蓄能器双蓄能器组 |
| 控制阀 | 普通阀组 | 大流量高速响应阀组、比例插装阀、伺服阀 |
| 工作介质 | 一般液压油 | 强制使用阻燃液压介质（水乙二醇等） |
| 清洁度要求 | 一般要求 | 指定等级的高清洁度控制（代码22/19） |
| 温度监控 | 常规监控 | 连续工作油温≤55℃，超温自动报警 |
| 工艺适配 | 通用动作控制 | 适配慢压射-快压射-增压三段式压射流程，或液锻高压长时间保压 |

### 快速建压要求
建压时间是指压射冲头运动停止点至增压压力达到其稳态值90%的时间间隔[[S58]]。压铸工艺要求增压压力在金属液凝固之前迅速建立，以减少铸件内部缩松缺陷[[S13,S50]]。

冷室压铸机建压时间标准[[S16,S50]]：

| 机型 | 锁模力范围 | 最大空压射速度 | 建压时间要求 |
|------|----------|-------------|------------|
| 卧式 | 6300kN～60000kN | ≥8m/s（≥50000kN时为10m/s） | 20～35ms |
| 立式 | ≤4000kN | ≥3m/s | ≤30ms |
| 立式 | >4000kN | ≥3.5m/s | ≤30ms |

> 当前行业先进压铸机实测建压时间已可达10ms以下[[S13]]；头部厂商的液锻机通过高性能比例阀与高能蓄能器组合，可实现压射力在50μs内达最大值[[S14,S55]]。

### 多级压射速度控制
冷室压铸机普遍支持三级及以上多级压射速度调节，部分先进机型可实现多达20个速度档位自由设定，或采用十段无级调速、负载口独立控制比例插装阀方案[[S19,S10,S53,S23]]。典型两级压射配置中：第一级低速压射约0.2m/s，用于将金属液平稳推送至浇口位置并排出压室内部气体；第二级高速压射可达1.4m/s以上，用于在数毫秒内完成型腔充型[[S10,S53]]。

### 液压冲击抑制
现代压铸机配置反压缓冲机构，在充型结束前瞬间降低压射冲头运行速度，减轻充型突然停止时产生的水锤效应与瞬时压力峰值，避免压力振荡导致的胀模、铸件飞边等缺陷。消除压力峰值已成为现代压铸机液压系统优化的核心性能目标[[S13]]。

## 关键运行参数
压铸机液压系统在运行中需严格控制以下参数[[S2,S15,S22,S17]]：

| 参数项目 | 要求/范围 |
|---------|---------|
| 最高工作压力 | 普遍在10MPa以上，最高可达30MPa |
| 连续工作油液温度 | 不应超过55℃，超温需自动报警 |
| 长期最高允许油温 | 不超过60℃ |
| 推荐正常工作油温 | 15～55℃ |
| 系统清洁度 | 应符合GB/T 31562规定的清洁度代码22/19 |
| 蓄能器压力降 | 每压射一次压力下降不应超过工作压力的10% |
| 蓄能器充气介质 | 只允许充氮气 |
| 承压通道零件耐压试验 | 1.5倍额定压力，保压3min无渗漏 |
| 工作介质 | 应使用阻燃液压介质 |

## 常见故障与维护

### 常见故障
液压系统的主要故障类型包括[[S1,S12,S9]]：

- **泄漏**：分为外泄漏（缸筒与缸盖、活塞杆等外部位置，可直接观测）和内泄漏（高压腔向低压腔的内部渗漏，需单向通压力油后观测另一油口是否出油判定）。主要成因是密封失效、连接面配合不良，或缸筒受压膨胀、焊缝缺陷引发[[S1,S12]]。
- **油温过高**：需依次排查冷却水进水量、冷却器积垢、油箱油位、系统内泄、冷却器进出水口连接方向等[[S1]]。
- **油液污染**：导致节流小孔、阀台缝隙堵塞，元件配合面异常磨损，可通过定期过滤、全周期换油避免[[S9]]。
- **气穴（气蚀）**：多由吸油管路堵塞、泵入口进油阻力过大吸入空气、系统存在负压区域引发，伴随泵及管路异常噪声。双泵合流系统中，双泵出口汇合位置距油泵出油口需大于200mm可降低该故障概率[[S1]]。

> 内部泄漏与油温升高会形成恶性循环：内部故障引发油温上升、油液粘度下降，进而内漏范围扩大，油温进一步升高，最终加速密封件老化、造成泵阀元件冲刷磨损[[S31]]。

### 维护周期
- 首次运行500小时后首次换油并全量清洗油箱，后续每3000小时换油清洗一次[[S48]]。
- 运行1500小时后拆卸清洗水冷却器去除水垢[[S48]]。
- 每半年清洗检查液压吸油、回油过滤器[[S48]]。

## 典型应用实例

### 液锻机液压系统
专用液锻机采用插装式集成阀门组，主缸与顶出缸压力可在设计范围内任意调整。滑块与锁模补缩缸快下动作配置充液阀，具备低控制压力、高充液流速、优异保压性能。通过伺服电机驱动可实现1～5mm/s的微进速度，满足合金液强制补缩要求，可在触摸屏界面实时完成压力参数设定与数据显示[[S32,S51]]。

### 布勒Carat系列大型压铸机
瑞士布勒集团Carat系列5600t～9200t大吨位压铸机采用两板式合模液压系统替代传统三板式结构，输出更均匀的合模力，模具受力更平衡、变形控制精度更高，适配超大型薄壁一体化压铸产品。其自研Parashot压射液压系统采用伺服调节阀实现连续加速式压射运动，降低充型过程金属液面紊流，提升铸件致密度与表面质量[[S6,S53]]。

### 伊之密LEAP系列超大型压铸机
伊之密LEAP系列7000t级超大型压铸机适配大流量高压液压系统，支持稳定输出大吨位锁模力，配套的伺服驱动系统可实现高精度压力位移控制，适配大尺寸一体化压铸结构件生产场景[[S29,S6]]。
