## 概述

惯性振动落砂机是砂型铸造清理工序中的核心落砂设备，依靠惯性激振器（由旋转不平衡质量产生的离心惯性力作为激振源）完成振击作业，实现型砂与铸件的分离。国家标准GB/T 25370-2020《铸造机械 术语》给出的标准中文名称为“惯性振动落砂机”，标准英文术语为 *inertial vibrating shakeout* [[S36,S9]]。该设备属于铸造机械术语体系中“落砂除芯设备 > 落砂设备”分支下的独立设备类型 [[S16,S22,S32]]。

**适用工艺说明**：惯性振动落砂机几乎不用于常规压铸生产。压铸工艺采用金属型高压充型，不存在整体砂型落砂工序，因此该设备不在典型压铸生产线上通用应用。仅在极少数特殊带预制砂型型芯的压铸场景中，会使用参数适配的振击式除芯设备完成铸件内腔残留砂芯的去除，该类设备属于专门的除芯单元，与通用型惯性振动落砂机在激振参数、作业对象、结构配置上有显著差异 [[S16,S27,S31,S18]]。

## 分类与判定

### 在落砂机大类中的位置

在落砂机一级分类体系中，落砂机可分为机械振动、滚筒式、其他（电磁振动等）三个一级分支。惯性振动落砂机属于机械振动分支下的独立子类，可进一步分为普通惯性式、冲击式惯性两类，每类又可按激振轴数细分为单轴、双轴亚型 [[S5]]。

在按驱动形式、工作特点、激振轴数、参振质体数量、调谐值等多维度进行的全维度细分中，惯性振动落砂机属于“远共振”调谐子类，其下进一步分为惯性振动落砂机和惯性振动输送落砂机两类 [[S7]]。

### 主要子类划分

| 分类依据 | 子类 | 说明 |
|---------|------|------|
| 工作模式 | 固定式惯性振动落砂机 | 振动方向角90°，落砂过程中铸型无水平方向运动 |
| 工作模式 | 输送式惯性振动落砂机 | 振动方向角60°～70°，实现边落砂边输送 |
| 激振轴数 | 单轴惯性振动落砂机 | 单一主轴带偏心块，栅床做圆周方向振动 |
| 激振轴数 | 双轴惯性振动落砂机 | 两轴对称反向同步旋转，产生定向直线振动 |
| 参振质体 | 单质体惯性振动落砂机 | 参振机体为整体结构，激振力直接传递到栅格 |
| 参振质体 | 双质体惯性振动落砂机 | 工作质体与激振质体分开，中间由共振弹簧连接 |
| 特殊类型 | 冲击式惯性振动落砂机 | 落砂机顶部与固定支架间留有间隙，靠撞击落砂 |

其中，双轴惯性振动输送落砂机是应用最广泛的品类 [[S24,S36,S38]]。单质体与双质体的区别在于：单质体的参振机体为一整体结构，激振力可直接传递到栅格；双质体的参振机体分为工作质体和激振质体两部分，中间由共振弹簧连接，传递到基础的动载荷更低 [[S36]]。

## 工作原理与结构组成

### 激振原理

惯性激振器的核心原理是利用旋转不平衡质量（偏心块）产生的离心惯性力作为设备激振源。惯性激振器主要分为三类 [[S36,S27]]：

| 类型 | 组成与驱动方式 | 特点 |
|------|---------------|------|
| 箱式激振器 | 由箱体、轴承、齿轮、主动轴、从动轴、偏心块组成；单台电机驱动，两轴通过齿轮啮合等速反向旋转 | 激振力大，但油温升高、润滑油易泄漏 |
| 双轴激振器 | 由轴承座、轴承、传动轴、偏心块组成；两台电机分别反向驱动两轴等速反向旋转 | 激振力可调，应用广泛 |
| 电动激振器（振动电机） | 偏心块与特殊电机集成一体 | 结构简单，维修方便，无需额外传动部件 |

### 单轴与双轴振动方向差异

**单轴惯性振动落砂机**：激振器为单根带偏重的主轴，主轴旋转产生旋转方向的惯性激振力，弹簧支撑的栅床做圆周方向振动。栅架平面设置水平倾角后，可使铸件边振动落砂边向下输送，倾角大小直接决定铸件在栅架上的停留时间和输送速度 [[S24,S37]]。

**双轴惯性振动落砂机**：两个对称布置的、质量尺寸完全相同的偏心块做反向同步旋转，两偏心块的离心惯性力在轴心连线方向的分力完全抵消，仅垂直于两轴心连线方向的分力相互叠加，使栅床仅产生定向直线振动。将双轴激振器整体倾斜安装后，定向激振力可分解为垂直和水平两个分量，实现边落砂边输送的功能。这类机型的激振方向角（激振力与落砂机床面的夹角）通常取60°～70° [[S37,S38]]。

### 远共振工作模式

惯性振动落砂机普遍采用远共振工作模式，其强迫振动核心公式为：

$$X = \frac{P}{k} \cdot \frac{1}{1-\left(\frac{\omega}{\omega_n}\right)^2} \cdot \sin\omega t$$

其中 $P = mR\omega^2$ 为偏心块旋转产生的激振力、$k$ 为弹簧刚度、$\omega$ 为激振器驱动频率、$\omega_n$ 为振动系统固有频率。远共振区间要求驱动频率 $\omega$ 远大于系统固有频率 $\omega_n$，此时振幅稳定，激振力输出大，振动对基础的作用力小，设备运行平稳、噪声低，停车过程平稳，对基础设计要求低 [[S28,S19]]。

### 典型结构部件

惯性振动落砂机的典型结构组成包括 [[S23,S29,S17]]：

**栅床与栅格**：栅床作为核心参振部件，需要具备足够的抗振刚度。栅格板有焊接条缝式、百叶窗式、铸钢圆孔/长孔等多种可选形式，栅格形状、板孔尺寸、布置形式需要根据实际落砂的铸件和砂团尺寸匹配，避免浇冒口、小铸件卡堵栅格 [[S17,S29]]。

**弹簧系统**：分为两类——（1）隔振弹簧，安装在参振机体和机座/基础之间，通常采用圆柱形螺旋弹簧、钢橡胶混合弹簧、橡胶弹簧等类型；（2）共振弹簧，仅应用于双质体落砂机，安装在工作质体和激振质体之间，可大幅降低设备传递到基础的动载荷 [[S29,S27,S38]]。

**驱动传动结构**：分为三类——（1）V带传动结构，单/双电机通过V带直接驱动激振器主轴；（2）可挠弹性联轴器传动轴传动结构，单电机通过传动轴、联轴器同时驱动两台箱式激振器强制同步旋转；（3）振动电机直接集成结构，无需额外传动部件，结构简单维修方便 [[S17,S29]]。

## 工艺作用与应用场景

### 选型依据

惯性振动落砂机的选型取决于铸件类型和砂型特性 [[S26,S8,S2,S10]]：

| 砂型/铸件类型 | 推荐选型 |
|-------------|---------|
| 低强度粘土砂湿型砂 | 低振幅惯性振动落砂机 |
| 高强度水玻璃砂、干型砂、树脂砂 | 高振幅高激振力惯性振动落砂机 |
| 铸钢件 | 高振幅低频率落砂机（振击力大） |
| 铸铁和有色金属铸件 | 低振幅落砂机（避免铸件损伤） |
| 消失模铸造砂处理线 | 可配套用于开箱后型砂与铸件的分离作业 |

### 使用模式

**单机使用模式**：适用于单件小批量、无连续造型线的铸造车间。中小件落砂时间1～5min，大件5～10min，特大件10～20min，落砂过程无连续输送功能 [[S26,S8]]。

**自动化砂处理线模式**：需选用带输送功能的惯性振动输送落砂机，可采用1～3级串联布置，适配高压造型、无箱射压造型等高节拍连续生产场景，实现旧砂和铸件同步输送分离 [[S26,S30,S38,S25]]。

### 砂处理线协同工艺

惯性振动落砂机落砂后，筛分出的铸件直接输送至浇冒口分离设备进行切除作业，筛分落下的旧砂进入后端的振动输送机，依次输送至砂块破碎机、磁选设备、振动筛、冷却设备，最终接入砂再生系统完成旧砂再生循环。部分场景中惯性振动落砂机可与滚筒落砂机串联布置，对高硬度砂型进行两级落砂 [[S30,S25,S24,S10,S27]]。

## 优缺点与局限性

### 核心优势

相较于电磁振动落砂机，惯性振动落砂机激振力更大、振幅稳定，可承载处理更大重量的砂型，对高强度干型砂的落砂破碎效果更优。相较于滚筒落砂机，落砂过程对铸件的磕碰磨损更小，无需等待滚筒转动的长周期，生产效率更高，可直接适配铸造流水线连续生产作业 [[S29,S24]]。

### 典型局限性

| 局限类型 | 具体表现 |
|---------|---------|
| 输送堵卡 | 激振角60°～70°的输送型机型运行时容易发生浇冒口、小型铸件堵卡栅格，栅格形状、板孔尺寸及布置必须与所处理铸件完全适配 |
| 润滑与维护 | 配置箱式激振器的机型运动部件发热导致油温升高，润滑油易泄漏，未及时补充将直接导致轴承损坏，日常维护工作量大 |
| 安装空间 | 双质体机型结构复杂、参振元件多、整机庞大高度高，对安装空间要求显著高于普通固定式落砂机 |

| 故障现象 | 产生原因 | 排除方法 |

| 运转噪声明显增大，栅床跳动异常 | 栅床振断或振裂；缓冲弹簧断裂或联轴器中橡胶件断裂；连接螺栓松动 | 焊补裂纹处；更换缓冲弹簧或联轴器缓冲橡胶件；更换螺栓并重新紧固 [[S33]] |
| 单轴机型传动带经常脱落 | 栅床床身平衡失调；两个传动带轮轴线不平行或不在同一平面内 | 加平衡块调整栅床平衡；调整电机座和栅床相对位置，保持轴线平行度 [[S33]] |
| 双轴机型联轴器橡胶件经常断裂 | 栅床负荷过重；电机轴和偏重轴轴线不同心；两偏重块调整不当；两台电机性能差异过大 | 减轻栅床负荷；调整电机座位置使轴线同心；调整偏重块位置并紧固；更换性能一致的电机 [[S33]] |
| 经常断弹簧 | 弹簧刚度太大（与实际工况过载状态不匹配） | 适当拆去1～2组弹簧，使剩余弹簧总刚度与实际载荷匹配，降低交变应力幅值 [[S33]] |
| 轴承过热 | 偏心轴轴心线偏差过大；润滑油不足或密封件损坏后异物进入；传动部件被积砂埋入 | 将正常工况下的轴承温升严格控制在25℃以下，定期为偏心轴承补充润滑脂，及时更换损坏的轴承密封件，定期清理传动部件周边堆积旧砂 [[S4,S33]] |
| 偏心块松动 | 长期交变负载下键销磨损松动；螺纹未做有效防松处理；安装后未按规定力矩紧固 | 采用厌氧胶对振动部位螺纹连接进行防松固化；定期检查紧固力矩；磨损的键和键槽采用堆焊法修复后重新校准安装 [[S15,S33]] |
| 栅格磨损 | 长期承受铸型、铸件反复撞击和砂粒高速冲刷 | 根据落砂工况选用高耐磨栅格结构；定期检查磨损余量；磨损超限时堆焊耐磨层修补或直接更换栅格组件 [[S17]] |
| 双轴机型激振不同步 | 两台异步电动机性能差异过大；接线转向设置错误；两偏重轴偏心块调整位置不一致；弹性联轴器橡胶缓冲件老化断裂 | 选用空载电流与转速参数完全匹配的两台电机；确认两台电机转向严格相反；调整两偏重轴偏心块夹角位置完全一致；定期更换联轴器内老化橡胶件 [[S4,S33]] |

## 相关设备与关系

### 与同类落砂设备的对比关系

惯性振动落砂机与偏心振动落砂机、电磁振动落砂机、滚筒落砂机等同属落砂设备大类。偏心振动落砂机通过偏心轴转动产生振动，振幅不随载荷变化，但主轴承受全部撞击力，工作寿命短。电磁振动落砂机通过电磁振动器经“电—磁—力”转换产生激振力，结构简单、无机械传动，但激振力相对有限 [[S31,S24]]。

### 落砂机分类体系

落砂机按驱动方式可分为机械振动、滚筒式、其他（电磁振动等）三个一级分支。机械振动类下细分为偏心振动、惯性式（单轴、双轴）、冲击式惯性（单轴、双轴）等子类 [[S5]]。按调谐值可将振动落砂机分为近共振和远共振两类，惯性振动落砂机属于远共振子类 [[S7]]。

### 与砂处理系统的工艺衔接

惯性振动落砂机是连接造型线、砂处理系统和铸件清理的枢纽环节，其落砂后的旧砂依次经过磁分离、砂块破碎、振动筛分、冷却等工序后进入砂再生系统，实现旧砂循环使用 [[S30,S25]]。

> ⚠️ **工艺适用范围说明**：惯性振动落砂机是砂型铸造领域的典型设备，在常规压铸生产中无通用应用案例。压铸件表面残留物（飞边、毛刺等）的清理通常采用振动光饰（振磨）设备，其工作原理和激振参数与惯性振动落砂机有本质差异。如需了解压铸件清理工艺，请参阅压铸件后处理相关词条。
