## 概述

辗轮混砂机（别名辗轮式混砂机、碾轮混砂机；英文对应 *sand muller*、*wheel-type sand mixer*、*roller sand mixer*、*muller mixer*、*pan mixer*）是粘土砂混砂机中一种以碾压和搓揉作用为主的间歇或连续式混砂设备[[S15,S1,S9]]。其核心工作方式为：辗轮随主轴公转的同时，受砂层摩擦力驱动绕自身水平轴自转碾压砂层，配合刮板将压实砂层翻起松散，反复循环完成型砂各组分的均匀混合，使粘结剂在砂粒表面形成均匀包覆膜[[S15,S1,S19]]。

在设备分类体系中，辗轮混砂机隶属于粘土砂混砂机大类，与摆轮式混砂机、辗轮转子式混砂机、转子式混砂机等机型并列，是砂处理工序中最基础的混砂设备之一[[S18]]。

## 定义与术语

依据 GB/T 25370-2020《铸造机械 术语》，辗轮混砂机是由垂直的主轴带动辗轮和刮板在辗盘上旋转作业的混砂机。其关键部件"辗轮"定义为随主轴公转并绕水平轴自转、对砂粒起辗压和搓揉作用的轮形部件[[S9,S27]]。整体结构简单，维护便捷，多为间歇式作业机型[[S9]]。

## 与其他类型混砂机的区别

辗轮混砂机与转子式、摆轮式混砂机在结构和适用场景上存在本质差异，三者对比如下[[S28,S15,S22,S1,S8,S25]]：

| 对比维度 | 辗轮混砂机 | 转子式混砂机 | 摆轮式混砂机 |
|---|---|---|---|
| **核心工作部件** | 至少1对绕水平轴自转的重型辗轮 + 配套刮板 | 2台独立固定混砂转子 + 3块仰角30°的刮板 | 2~3个绕垂直短轴安装的摆轮，靠离心力贴向碾盘壁 |
| **有无高速转子** | 无独立高速混砂转子 | 有高速混砂转子 | 无 |
| **混砂机理** | 高碾压力 + 低搓揉速度，碾压与翻松交替 | 高速转子抛击作用，快速完成砂粒覆膜 | 离心力驱动摆轮碾过抛起的砂流 |
| **型砂松散度** | 易生成块状，卸砂后需加松砂装置 | 混出的型砂松散无结块 | 松散性好（可配套鼓风系统） |
| **生产率** | 偏低，混砂周期较长 | 高，混砂周期短 | 较高 |
| **适用批量** | 中小批量 | 大批量高速自动化造型线 | 大批量单一砂和背砂 |
| **结构复杂度** | 简单，维修方便 | 结构相对复杂 | 主轴转速高，电机功率较大 |

## 工作原理

以应用最广的单碾盘间歇式辗轮混砂机为例，其工作机理如下[[S19,S1,S17]]：

1. **碾压阶段**：加料后在碾盘上形成一定厚度的砂层，辗轮随十字头绕主轴公转，同时受砂层摩擦力驱动绕自身水平轴自转，对砂层施加高碾压力进行挤压和搓揉。
2. **翻松阶段**：随十字头同步转动的刮板（内刮板和外刮板）随即将被压实碾压的砂层翻起、松散，使砂料重新暴露。
3. **循环混合**：辗轮碾压与刮板翻料两道工序反复循环交替，逐步完成型砂各组分的均匀混合，使粘结剂在砂粒表面形成均匀包覆膜。

## 主要结构组成

辗轮混砂机主要由以下核心部件构成[[S10,S14,S17,S23,S28,S1,S8]]：

| 部件名称 | 功能说明 | 结构特点 |
|---|---|---|
| **辗轮** | 碾压搓揉砂层，是混砂作用的核心执行件 | 普遍配套弹簧加压机构；新型设备采用加宽凹面辗轮提升混碾效率 |
| **刮板** | 翻起压实砂层、搅拌砂料、导向砂流 | 分内刮板、外刮板；工作面可选配硬质合金镶条；设仰角实现砂料高效翻动抛起 |
| **碾盘（混砂盘）** | 承载砂料，构成混砂腔体 | 由外圈、内圈、底板三层结构组成；盘面可铺设辉绿岩铸石衬板或锰钢板衬里；配套围圈防外溢 |
| **传动机构** | 驱动辗轮和刮板运动 | 由驱动电机、减速器、斜齿轮副、主轴、十字头组成；可配套液力联轴器实现带载启动 |
| **卸砂装置** | 卸出已混制完成的型砂 | 通常为气缸驱动的抽屉式或摆动式卸料门，配套卸料溜槽；可增设自动控制模块 |
| **弹簧加压机构** | 调整辗轮碾压力，适应不同型砂混制要求 | 通过调节弹簧预紧力改变碾压力，防止混砂过程产生砂饼；克服辗轮离心惯性力 |

## 分类

辗轮混砂机可按照以下维度分类[[S28,S8,S23,S14,S10]]：

| 分类维度 | 类型 | 说明 |
|---|---|---|
| **按工作模式** | 间歇式 | 加料、混制、卸砂分批次进行，S11系列单碾盘双辗轮机为代表 |
| | 连续式 | 加料、混制、卸砂同时连续进行 |
| **按碾盘数量** | 单碾盘 | 常规间歇式机型 |
| | 双碾盘 | 两台单碾盘机构组合，物料沿"8"字形轨迹运动，生产率提升50% |
| **按加压形式** | 普通自重式 | 依靠辗轮自身重量施加碾压力 |
| | 弹簧加压式 | 可动态调整碾压力，适配不同硬度型砂 |

双碾盘连续混砂机（又称"8"字形双辗盘连续混砂机）将两台规格相同的单碾盘机构并排布置，两盘中心距约为盘径的0.9倍，两盘主轴反向同速旋转。刮板驱动物料在两盘间沿"8"字形轨迹反复交换位置，通过控制卸料门开启度调节物料停留时间，实现连续进料连续出料，相比同级间歇式机型生产率提升50%[[S8,S14,S28]]。

## 加料顺序与混碾时间

辗轮混砂机混制粘土砂时，正确的加料顺序对型砂质量至关重要[[S7]]：

1. **先加干料预混**：先加入旧砂、新砂、煤粉等干料完成预混。
2. **再加湿态粘结剂**：预混均匀后再加入水和膨润土等湿态粘结剂进行混碾。

不应将膨润土和水直接加入预混工序中，否则粘土易结块附着在设备腔壁，影响混砂均匀性[[S7]]。

混碾时间的确定取决于混砂机类型、型砂中新砂与旧砂比例及粘土含量。粘土含量高，混砂时间需相应延长。各种混砂机混砂时间可参考相应工艺手册[[S7]]。

## 常规应用

辗轮混砂机在砂型铸造中主要适用于以下型砂的混制[[S8,S20,S6,S19]]：

- **黏土湿型砂**：面砂、芯砂、单一砂和背砂
- **普通水玻璃砂**：面砂、芯砂
- **树脂自硬砂**（如呋喃树脂砂）：批量间歇混合
- **覆膜砂制备阶段**：树脂包覆原砂的混合作业
- **铝合金金属型铸造芯砂**：油砂、合脂砂等[[S6]]

## 与压铸工艺的关系

辗轮混砂机**不属于高压压铸的核心工艺设备**，属于压铸配套上游砂处理设备[[S6]]。高压压铸（die casting）以金属型腔直接成形，通常不使用砂芯。

但在采用砂芯的近压铸工艺中——如铝合金金属型铸造（重力浇注、低压铸造）——辗轮混砂机是混制砂芯用油砂、合脂砂等芯砂的常用设备[[S6]]。其碾压与搓揉混合作用能满足中小批量砂芯对芯砂均匀度的混制要求，混制时间一般为15~20分钟[[S6]]。因此，辗轮混砂机与压铸行业的关系定位于铸造上下游的芯砂制备环节，而非压铸工艺的核心工位。

## 局限性与改进方向

辗轮混砂机的主要局限性在于[[S19,S1,S8]]：

1. **生产率偏低**：碾轮对型砂产生的速度差小，搓揉作用弱，主要依靠碾压力混制型砂，批料量受限，生产率低于转子和摆轮混砂机。
2. **型砂质量一般**：碾压易使型砂生成块状，用于高密度砂型时需保证足够混碾时间，且卸砂后需增加松砂装置。
3. **能耗较大**：高碾压力和低搓揉速度的混制模式使单位能耗偏高。

针对上述问题，现代辗轮混砂机普遍采取了以下改进措施[[S1,S10,S5]]：弹簧加减压机构、压力喷雾加水、矮刮板设计并提升主轴转速、加宽凹面碾轮等，以提升型砂质量和设备生产率。
