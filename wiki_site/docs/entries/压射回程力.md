## 概述

**压射回程力**（英文：*injection return force*）是中国压铸行业标准GB/T 25370-2020《铸造机械 术语》中明确定义的术语，对应国际英文术语为 *injection return force*，压铸行业通用同义表述还包括 *plunger return force*、*shot return force* 及 *Return force of injection punch* [[S7,S25,S21]]。

值得注意的是，行业内存在两类对“压射回程力”的不同理解：一类为上述国家标准定义的油缸输出作用力；另一类为部分非标准化文献和现场俗称所指的压射冲头回程过程中需克服的总负载力（包括冲头与压室间的摩擦力、残留凝固余料的粘滞阻力等） [[S16,S8]]。本文以国家标准术语定义为准。

## 定义

依据GB/T 25370-2020《铸造机械 术语》，压射回程力定义为：**作用在压射缸有杆腔，使压射活塞带动压射冲头退回到原位的力** [[S7]]。该定义从标准层面明确压射回程力是压铸机压射回程油缸输出的作用力，而非冲头回退所需克服的外部负载力。

## 工艺作用

压射回程力驱动的压射回程动作是压铸机完整生产循环的必要环节 [[S1,S35,S15,S39]]，其核心工艺作用包括：

1. **冲头复位**：将压射冲头退回到初始位置，为下一次压射的金属液加料（浇料）动作预留空间。
2. **压室余料清理**：配合开模后冲头前伸动作，将压室内的残留凝固余料推出压室，避免余料卡滞影响后续生产循环 [[S1,S35,S15,S39]]。在冷室压铸机的常规回程动作结束后，开模阶段可同步驱动冲头向前伸出压室分型面，将余料（料柄）完全推出压室外 [[S38,S42]]。
3. **供料辅助（热室压铸机）**：在热室压铸机中，压射冲头退回时可在鹅颈通道内形成局部真空，辅助熔融金属自动回流填充鹅颈与喷嘴区域，为下一次压射自动完成供料准备 [[S5,S43]]。

## 工作液压原理

在三级压射机构的卧式冷室压铸机中，压射活塞的回程动作由两路压力油同时进油实现 [[S29,S40,S34,S11,S4,S6,S9,S18,S48,S52,S41,S50]]：

1. 一路压力油进入**回程腔C2**；
2. 另一路压力油进入**尾腔C3**，推动回程活塞顶开单向阀，将U形腔与压射腔C1导通，建立完整回油通路。

压射缸内部的运动时序为：回程阶段，增压活塞率先回缩使单向阀开启，随后压射活塞完成回缩复位 [[S26]]。

## 参数的确定与设定

### 回程力计算依据

压射回程力的设定由三部分阻力计算确定 [[S31]]：

1. 压射冲头、活塞杆等运动部件的摩擦力；
2. 压射腔内残留金属液/余料的回程阻力；
3. 回程动作的液压背压与各密封件的摩擦阻力。

工程上回程力的计算可采用公式（引自铝挤压机回程力计算，在压铸领域具有通用参考价值） [[S55]]：

$$P_{回} = P_{运} + P_{背} + P_{密} + \Delta$$

其中：
- *P*<sub>运</sub>：运动部分重量产生的摩擦力（钢对铜摩擦系数取0.1~0.15）；
- *P*<sub>背</sub>：回程时液压背压作用力（常规背压压强取3~5 kgf/cm²）；
- *P*<sub>密</sub>：各密封处的摩擦阻力（密封件摩擦系数取0.1~0.13）；
- Δ：预留额外拔出力，取前三项之和的5%~10%。

### 与压射力的近似比例关系

压射活塞杆直径与压射缸直径的设计比例约为 *d*<sub>g</sub> ≈ 0.63 *d*<sub>p</sub>，压射回程力有效作用面积约为压射活塞总面积的40% [[S34]]。因此常规压射回程力的额定值约为同机型额定压射力的 **1/3~1/2**，与锁模吨位的近似比例关系为：回程力 ≈ (0.01~0.02) × 锁模吨位，或回程力 ≈ (0.3~0.5) × 额定压射力 [[S34]]。经验取值通常为设备吨位的6%~12%，大吨位机型取大值；若配置低压充液罐，回程力取值可提高到设备吨位的8%~14% [[S55]]。

### 典型范围

| 机型类别（按锁模力） | 锁模力范围 | 压射回程力典型范围 |
| :--- | :--- | :--- |
| 小型压铸机 | 630~1600 kN | 15~60 kN |
| 中型压铸机 | 2000~8000 kN | 60~250 kN |
| 大型压铸机 | 10000~25000 kN | 250~1200 kN |

> 注：同锁模吨位的冷室压铸机回程力略高于热室压铸机。[[S46,S27]]

## 调节与控制

压射回程力支持全范围调节，常规操作方式包括 [[S53,S24]]：

1. **压力调节**：通过回程油路的减压阀/溢流阀调节回程腔输入油压；
2. **速度调节**：通过回程支路的节流流量控制阀调节液压油流量，同步改变压射冲头返回速度。

压射冲头的回程时序有多种联动模式。以J1163型卧式冷室压铸机为例，回程动作由开模行程触发的限位开关控制，支持两种逻辑 [[S42]]：

- 开模前直接触发冲头回程；
- 开模后先驱动冲头前伸将余料推出压室再执行回程复位。

## 冷室与热室压铸机回程机构差异

| 对比维度 | 冷室压铸机 | 热室压铸机 |
| :--- | :--- | :--- |
| 回程运动方向 | 水平直线运动 | 垂直向上提拉动作 |
| 压射系统布置 | 压室与保温炉完全分离 | 压射系统与鹅颈、坩埚集成一体 |
| 整机倾角 | 水平/垂直 | 约15°倾角 |
| 压射压力范围 | 40~60 MPa | 7~14 MPa |
| 最大锁模力 | 突破45000 kN | 普遍不超过4000 kN（约400吨） |
| 冲头冷却 | 内置冷却水路 | 无冷却装置（浸入熔液中） |

## 对生产与质量的影响

### 回程力设定不当的影响

| 现象 | 可能原因 | 后果 |
| :--- | :--- | :--- |
| 回程力不足 | 输出压力或流量偏低 | 冲头返回速度下降，拉长空循环时间，降低生产效率 |
| 回程力过大 | 压力设定过高 | 回程动作冲击增大，易造成冲头与压室碰撞磨损 |
| 回程时序不当（热室） | “先回冲头后开模” | 熔融合金滴落至模具分型面，冷却形成硬壳阻碍合模 |
| 回程时序不当（热室） | “开模与回程同步” | 鹅颈卷入空气，铸件内部孔洞缺陷，产生氧化浮渣 |

[[S1,S15,S5,S43]]

### 对循环时间的影响

在冷室压铸机中，一次空循环时间定义为合模、压射、开模、锤头推出、压射回程、顶出、顶出返回诸动作时间的总和 [[S1,S35,S44,S28]]。回程力不足会直接导致冲头返回速度下降，拉长空循环时间，降低设备产能。

## 标准与规范现状

- **GB/T 25370-2020《铸造机械 术语》**：首次在国家标准层面给出了压射回程力的精确定义 [[S7]]。
- **GB/T 21269-2023《冷室压铸机》**：将压射回程纳入空循环时间的定义范畴，作为压铸自动循环的必选动作，但未对回程力数值给出独立强制规定 [[S44,S28]]。
- **JB/T 8083系列标准**：以合模力、压射力为核心规定参数，未将压射回程力列为公开的强制性能指标，仅通过设备配套技术文档明确回程动作由压射缸回程腔提供动力实现 [[S16,S36]]。
- **国际标准（NADCA/VDG/ISO）**：目前未检索到针对压射回程力的独立量化强制要求；行业普遍将回程力作为设备制造商按需适配的非公开配套参数，无统一公开标准约束 [[S20,S17]]。欧美主流压铸机厂商将回程力列为标准可配置工艺参数，可通过设备液压系统独立校准标定 [[S54]]。

## 关联术语

- **压射力（injection force）**：充型过程中压射机构传递到压射冲头上的最大力 [[S7]]，与回程力存在约 (0.3~0.5) 的近似比例关系 [[S34]]。
- **空循环时间（dry cycle time）**：包含压射回程动作时长的压铸机完整空循环时间 [[S44]]。
- **增压（intensification）**：利用活塞面积差或更高压力蓄能器使压射缸压力高于系统压力的过程 [[S44]]，回程动作发生在增压保压完成之后。
- **压射比压（injection pressure）**：充型结束时压射冲头作用于熔融金属单位面积上的压力 [[S7]]。
