## 概述

在压铸行业语境下，“主轴”指压铸机压射系统内部的传动杆件，其标准中文术语依据 GB/T 25370-2020《铸造机械 术语》定名为**压射杆**（plunger rod），标准定义为“冲头与压射驱动装置之间的连接杆”。[[S1,S20]]该部件用于传递液压缸输出的压射力，驱动压射冲头沿压室完成一系列压射动作，属于压铸机注射系统的核心传动部件；其不指代机床主轴或液压泵主轴。压铸行业内同样存在“冲头杆”“活塞杆”“主活塞杆”“注射主轴”“增压主轴”等同类型部件的习惯叫法。[[S12,S20,S24]]

欧美术语体系中，shot rod、plunger rod、injection piston rod 均对应压铸行业的“压铸主轴/压射杆”部件。[[S9,S18]]通用英文短语 “die casting machine main shaft” 未在主流压铸标准中作为该部件的官方命名使用。

## 分类

### 按机器类型分类

- **冷室压铸机主轴**：卧式水平布置，杆体多设置内置冷却水路以承受高温工况；配置独立增压功能的冷室压铸机可设置独立的增压主轴段，完成充型后的瞬间增压动作。[[S5,S21]]
- **热室压铸机主轴**：竖直布置，直接驱动浸入熔融金属的热室压射冲头；常规通用型热室压铸机无独立增压结构，一般不设置独立的增压主轴部件。[[S5,S21,S28]]

### 按功能分类

| 分类 | 功能描述 |
|------|----------|
| 注射类主轴 | 直接与压射活塞刚性连接，全程带动压射冲头完成低速排气、高速充型的全压射行程。[[S24]] |
| 增压类主轴 | 独立设置的增压缸活塞杆，仅在金属液充型填满型腔后的增压阶段动作，向凝固中的金属传递高比压。[[S22,S24]] |

## 结构与材料

### 主体结构

压射杆常规结构为带内置冷却水路的中空圆杆。两端分别通过螺纹与冲头和压射活塞/联轴节连接，杆体上设置导向部分用于保证运动直线度。[[S14,S16]]

### 配合件

压射杆在装配中与压射缸活塞杆、压射冲头、压室（压射室）、导向套、密封件等协同工作。冷室压铸机压射机构主要包含蓄能器、压射缸、增压缸、压室、压射杆及冲头等构成。[[S5]]

### 材质与热处理

SDCE（美国压铸工程师协会）标准及行业手册推荐主体材质选用 H13 热作模具钢，热处理后硬度达到 HRC 46~48。[[S14,S16]]

### 形位公差要求

SDCE 冷室压射杆标准规定以下关键形位公差：[[S14,S16]]

| 公差项目 | 要求值 |
|----------|--------|
| 杆两端凸肩平面相对杆轴线的垂直度偏摆 | ≤ 0.001 英寸 |
| 杆前端螺纹段及导向部分相对杆轴线的同轴度偏摆 | ≤ 0.002 英寸 |
| 连接螺纹精度等级 | 二级统一螺纹配合标准 |
| 杆螺纹导向直径（精磨）相对螺纹外径 | +0.030 / +0.000 / -0.001 英寸 |
| 冷却水回流截面积 | 不小于进水管截面积 |

## 工艺作用与动作时序

压射杆在完整压铸循环中随压射活塞依次完成以下五个阶段动作：[[S10,S11,S26,S2]]

| 阶段 | 动作描述 | 工艺目的 |
|------|----------|----------|
| 1. 低速封口阶段 | 带动冲头以 0.2~0.3 m/s 低速移动 | 平稳排出压室内部气体，避免金属液从浇料口溅出、防止卷气 |
| 2. 堆聚阶段 | 以略高于低速的速度推送金属液至内浇口前端 | 突破内浇口阻力前完成金属液堆聚 |
| 3. 高速充型阶段 | 以 1.2~9 m/s 的最大速度推动金属液快速填充型腔 | 在极短充型时间内完成型腔填充 |
| 4. 增压保压阶段 | 以极小位移传递高压 | 完成凝固补缩，获得组织致密铸件 |
| 5. 开模回程阶段 | 带动冲头将余料推出压室并复位 | 完成脱模并准备下一循环 |

压射杆运动状态波动会直接影响铸件质量：高速启动过早会导致金属液提前紊流卷气造成铸件气孔缺陷；高速启动过迟会导致金属液温度下降产生冷隔、缺料缺陷；压射杆同轴度偏差会直接引发冲头偏磨，导致压室内部运动阻力不均，造成充型速度不稳定，最终降低铸件尺寸精度、增大废品率。[[S23]]

## 工艺参数与操作条件

### 速度参数

通用压铸机压射杆的慢压射运行速度常规范围为 0.2~0.3 m/s，快压射运行速度主流范围为 4~5 m/s，高端机型最大运行速度可达 9 m/s。[[S2]]

### 同轴度要求

压射杆装配完成后，与压室的运动同轴度公差要求为 100 mm 长度上偏差不超过 0.04 mm。[[S3]]

### 承载压力

压铸工况下压射杆可传递的最大铸造压力最高可达 43 MPa，对应 4.0 m/s 级别的常规高速压射工况。[[S13]]

## 常见缺陷与失效形式

| 失效形式 | 产生原因 | 失效后果 |
|----------|----------|----------|
| 配合面磨损 | 长期运动摩擦、同轴度偏差引发 | 运动阻力增大、密封泄漏 [[S25,S27]] |
| 杆体弯曲变形 | 过载冲击、热应力叠加 | 破坏运动同轴度，加剧偏磨 [[S25,S27]] |
| 工作面刮伤 | 金属碎屑侵入配合间隙 | 同步划伤导向套内壁，加剧磨损 [[S25,S27]] |

上述失效会导致压射压力传递不稳定，压铸件出现气孔、缩松、轮廓不全缺陷，还会引发液压系统内泄漏增大、油温异常升高的故障。[[S25,S27]]

## 维护与寿命

- **日常检查**：每日检查所有管接头密封状态排除漏液；定期检测压射杆与压室的同轴度。[[S3,S15]]
- **润滑**：运行过程中需保证压射导向部位持续润滑。[[S15]]
- **更换标准**：当压射杆工作面磨损量超过设计允许阈值、弯曲变形量超出同轴度公差范围时需强制报废更换。[[S15]]
- **使用寿命**：常规工况下压射杆的平均使用寿命可达数万模次，其寿命长短与压射杆和导向套之间的配合间隙精度直接相关。[[S15]]

## 相关术语

- **压射冲头（plunger）**：压射杆前端直接与熔融金属接触的柱塞部件，两者通过螺纹刚性连接。[[S1]]
- **压射室（shot sleeve/压室）**：容纳待压射金属液并供压射冲头及压射杆在其中运动的圆筒形容器。[[S1]]
- **压射缸（injection cylinder）**：为压射杆及压射活塞提供液压动力的驱动缸。[[S1]]
- **压射驱动装置（injection drive）**：驱动冲头及压射杆，并对其施加作用力的装置。[[S1]]
- **增压器（intensifier）**：独立设置的增压缸组件，用于在充型结束后通过增压主轴段（增压类主轴）向金属液传递高比压。[[S24,S22]]
