## 概述

机器能力指数（Machine Capability Index）是用以评价压铸单元自身设备稳定性与精度，衡量其满足压铸件生产要求并实现稳定生产能力的量化指标[[S12,S19]]。该指数仅考虑设备本身的固有影响，排除外部材料、环境、人员等非设备因素的干扰[[S19]]。在压铸生产中，机器能力指数通常以CMK（Capability Machine Index）作为核心代号，通用工业场景下也采用Cm/Cmk体系[[S5,S20]]。

## 定义与核心概念

依据GB/T 37371-2019《压铸单元 术语》，机器能力指数（CMK）被定义为“分布的平均值与规范中心值的偏移”，其测量前提是仅考虑设备本身的固有影响[[S19]]。GB/T 37365-2019《压铸单元 性能检测方法》进一步将CMK定义为评价压铸单元满足压铸件生产要求、实现稳定生产能力的量化评价指标[[S12]]。

机器能力指数的核心区分对象是过程能力指数（Cpk/Ppk）。机器能力测试属于设备级验证，在严格限定条件下——短时间连续采集样本、排除外部干扰因素——仅反映设备自身的固有重复精度；过程能力测试则覆盖完整生产全周期，纳入所有长时间维度的波动影响[[S20,S1]]。两者的关键区别如下：

| 对比维度 | 机器能力指数（Cm/Cmk） | 过程能力指数（Cp/Cpk/Ppk） |
|---|---|---|
| 测试对象 | 设备自身固有精度 | 全生产链路综合能力 |
| 外部影响 | 排除时间和环境等外部干扰 | 纳入所有长周期波动因素 |
| 样本量 | 通常50个连续样本 | 一般不少于125个样本 |
| 合格判定要求 | ≥99.994%的样本落在公差范围内 | ≥99.73%的样本落在公差范围内 |

[[S20,S1]]

## 符号、参数与计算公式

### 通用符号体系

机器能力指数计算涉及以下核心参数[[S17,S6]]：

| 符号 | 含义 |
|---|---|
| USL | 公差上控制限（Upper Specification Limit） |
| LSL | 公差下控制限（Lower Specification Limit） |
| x | 目标参数的名义值 |
| X̄（或x̄） | 全部样本数据的算术平均值 |
| S（或σₘ） | 样本标准差（无偏估计） |

样本标准差S的计算公式为[[S17]]：

**S = √[Σ(xᵢ - x̄)² / (n - 1)]**

其中xᵢ为单次观测值，n为有效样本总数。

### Cm计算公式

通用工业手册给出的Cm基础计算公式为[[S5]]：

**Cm = P / (8σₘ)**

其中P为产品公差带宽度，σₘ为机械设备自身运行输出的标准偏差。

### Cmk计算公式

Cmk取CMU与CML两者中的较小值[[S17]]：

- 上侧机器能力指数：**CMU = (USL - x̄) / (3S)**
- 下侧机器能力指数：**CML = (x̄ - LSL) / (3S)**
- 最终结果：**Cmk = min(CMU, CML)**

### 多场景计算规则

根据《航空制造工程手册》，机械能力指数Cm/Cmk的计算涵盖四类场景[[S10,S3]]：

| 能力分布类型 | 计算公式 | 特殊规定 |
|---|---|---|
| 双侧标准，分布中心无偏移 | Cm = (Tu - Tl) / (8σₘ) | — |
| 双侧标准，分布中心有偏移 | Cmk = (1 - k)·(Tu - Tl) / (8σₘ)，其中k = |M - μₘ| / [(Tu + Tl)/2] | 当k ≥ 1时，规定Cm = 0 |
| 仅设上限的单侧标准 | Cmu = (Tu - μₘ) / (4σₘ) | 当Tu ≤ μₘ时，规定Cm = 0 |
| 仅设下限的单侧标准 | Cml = (μₘ - Tl) / (4σₘ) | 当Tl ≥ μₘ时，规定Cm = 0 |

[[S10]]

## 评价准则与判定标准

### 通用合格判定阈值

在通用工业场景下，机器能力指数的合格判定标准如下[[S20,S5]]：

- **Cm ≥ 1.67**：表示设备固有精度完全匹配对应公差要求[[S5]]
- **Cmk ≥ 1.33**：代表设备临界能力合格，该等级下设备可保证至少99.994%的输出样本落在允许公差范围内[[S20]]
- 常规通用加工设备合格要求Cmk ≥ 1.33；高精度应用场景的验收门槛要求Cmk ≥ 1.67，属于行业内较高的设备稳定性要求[[S8]]

### 工序能力与机械能力的综合评价

《航空制造工程手册》给出了基于Cp（工序能力指数）和Cm（机械能力指数）联合评判的结构化判定表[[S11]]：

| 工序能力等级 | Cp条件 | Cm条件 | 处置措施 |
|---|---|---|---|
| 工序能力充足 | Cp ≥ 1.33 | Cm ≥ 1.00 | 正常工序管理；若Cm过剩，可放宽设备要求或修订标准 |
| 有工序能力 | 1.00 ≤ Cp < 1.33 | Cm ≥ 1.00 | 设备标准化；排查非设备因素，提高工序能力 |
| 有工序能力 | 1.00 ≤ Cp < 1.33 | Cm < 1.00 | 重点改善机械设备，确认设备能力充足后过渡到工序管理 |
| 工序能力不足 | Cp < 1.00 | Cm ≥ 1.00 | 分层排查外部因素，采取改善措施；采用控制图监视工序能力 |
| 工序能力不足 | Cp < 1.00 | Cm < 1.00 | 对设备进行维修或更换；用控制图验证改善效果后提升工序能力 |

[[S11]]

### 压铸场景下的公差设定规则

在压铸单元CMK测试中，GB/T 37365-2019规定了简化的默认公差设定规则[[S17]]：

- **USL = x × (1 + 3%)**
- **LSL = x × (1 - 3%)**

因此公差带宽度为名义值的±3%。

## 压铸场景下的应用要求

### 测试前提条件

依据GB/T 37365-2019，压铸单元CMK测试需满足以下前提条件[[S22,S20]]：

1. 压铸单元经调试后处于全自动模式连续运行的无故障平稳状态
2. 液压系统油温稳定在30 ℃～55 ℃区间
3. 关闭压射冲头跟踪功能
4. 测试流程需覆盖关门、合模、浇注、压射、开模、开门、无负载顶出、顶回、压回、无负载取件、无吹气无涂料喷涂等全核心驱动动作
5. 连续采集50次无异常停机的独立样本
6. 测试过程中不存在刀具磨损、外部环境扰动引入的系统性偏差

### 检测项目与仪器要求

GB/T 37365-2019的附录表A.4规定了压铸单元CMK测定的全部项目，覆盖压铸机侧参数与熔体保温浇注设备侧参数[[S7]]：

| 区域 | 检测参数 | 测定仪器、量具 |
|---|---|---|
| 压铸机 | 锁模力 | 无纸记录仪和应变传感器（精度0.5%）或专用锁模力检测系统 |
| 压铸机 | 慢压射速度1、慢压射速度2、快压射速度 | 位移传感器（量程不低于压铸机压射行程，分辨率≥0.1 mm，精度±0.5%）和无纸记录仪 |
| 压铸机 | 快速起始点 | 位移传感器和无纸记录仪 |
| 压铸机 | 快速建速时间 | 位移传感器和无纸记录仪 |
| 压铸机 | 增压稳态压力 | 压力传感器（量程60 MPa，精度±0.1%）和无纸记录仪 |
| 压铸机 | 增压建压时间 | 压力传感器和无纸记录仪 |
| 熔体保温与浇注设备 | 熔体温度 | K型热电偶、变送器和无纸记录仪 |
| 熔体保温与浇注设备 | 浇注量 | 电子秤（精度±0.01 kg） |

注：无纸记录仪的运行不依托于压铸单元的电控系统程序，可独立进行数据和曲线的采集、显示、存储和输出，记录时标0.001 s，单次连续记录时间不低于20 s[[S7]]。

## 应用场景

在压铸行业实践中，机器能力指数（CMK）应用于以下关键环节[[S16]]：

- **新机出厂验收**：作为核心交付指标之一，验证压铸单元在出厂时满足合同约定的精度等级要求
- **日常定期精度校验**：定期评估设备精度衰减状态，作为预防性维护的决策依据
- **更换模具前精度验证**：在安装新模具前，提前验证压铸机当前精度状态是否匹配该模具的精度等级要求
- **压铸工艺调试前验证**：确保设备基础稳定性满足高精度压铸件的量产前提条件

## 与过程能力指数的关系

在压铸批量生产启动之前，必须先验证压铸设备的Cm/Cmk达标。只有确认设备本身的固有能力满足要求后，后续开展的Cpk/Ppk过程能力验证才具有实际意义；否则，即便依靠优化工艺参数也难以获得长期稳定的高过程能力表现[[S20,S11]]。

值得注意的是，经参数优化后的冷室压铸过程可以使关键尺寸的Cpk达到1.5以上，高于通用工业1.33的基准要求，处于±6σ能力区间，大批量生产条件下的不合格品率极低[[S2]]。

## 影响因素与局限性

### 核心受控参数

压铸场景下直接影响Cm/Cmk计算结果的核心参数包括[[S7,S16]]：

- **压铸机侧**：锁模力、多段慢压射速度、快压射速度、快速起始点、快速建速时间、增压稳态压力、增压建压时间
- **熔体侧**：熔体温度、浇注量

上述参数的测量需使用指定精度等级的检测工具以保障数据准确性[[S7]]。

### 局限性

Cm/Cmk测试属于严格受控的短周期设备能力验证，存在以下局限[[S20,S4]]：

1. 完全无法覆盖压铸生产全流程中存在的外部波动因素，包括：
   - 原材料批次成分波动
   - 不同操作人员的操作差异
   - 环境温度随季节变化的长期漂移
   - 设备/模具的缓慢磨损老化
2. 仅靠Cm/Cmk达标不能直接证明全生产过程的长期稳定性
3. 测试时未纳入刀具磨损、冷却润滑液耗用等随时间累积的外部影响[[S20]]
