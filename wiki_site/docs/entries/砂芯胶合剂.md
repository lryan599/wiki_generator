## 概述
砂芯胶合剂（core adhesive）是铸造生产中用于粘接两半砂芯、修补砂芯表面缺陷及密封砂芯组合接缝的专用胶合剂。其核心功能与制芯阶段混入原砂的主粘结剂（如树脂、水玻璃等）有明确分工边界——后者在混砂成形时赋予砂芯整体强度，而砂芯胶合剂则是在砂芯成形烘干后，单独施加于接合面或缺陷部位，属于砂芯后处理工序用的粘结材料[[S1,S2]]。

## 定义
依据国家标准GB/T 5611-2017《铸造术语》，砂芯胶合剂的正式技术定义为：“用来胶合砂芯或修补砂芯的粘结剂。要求干拉强度大于0.7MPa” [[S6,S9]]。该定义确立了砂芯胶合剂的两个关键属性：一是用途限定在胶合（组装）与修补两个场景；二是以干态抗拉强度不低于0.7MPa作为基本验收门槛。

> **术语边界**：砂芯胶合剂不同于制芯过程中预先混配在芯砂中的主粘结剂（如呋喃树脂、水玻璃、硅溶胶等），后者用于整体制造砂芯并决定砂芯的常温及高温强度；而砂芯胶合剂仅在砂芯单独成形并烘干后使用，施加于局部接合面或缺陷位置，属于后续组装与修复环节的辅助材料[[S1,S2]]。

## 分类
砂芯胶合剂可按多种维度进行分类，工程上最常见的有以下三种方式[[S8]]：

**（1）按化学属性分类**
- **无机系**：以水玻璃、膨润土等为基体的胶合剂
- **有机系**：以合成树脂、糊精、动物胶等为基体的胶合剂
- **复合系**：为改善单一体系性能而设计的无机‑有机复合胶合剂

**（2）按固化方式分类**
- **室温自硬型**：涂敷后依靠空气或组分间化学反应在常温下自行固化
- **热硬/烘干型**：需经加热烘烤才能完成固化并获得足够强度，典型固化条件为180～250℃保温约30分钟[[S5]]
- **气硬型**：通过通入气体（如CO₂）触发硬化反应，常见于改性水玻璃体系

**（3）按使用用途分类**
- **组装胶**：用于两半或多块砂芯的粘接组合
- **修补胶**：用于填补砂芯烘干后出现的掉角、缺肉及工艺孔堵塞
- **接缝密封胶**：用于密封砂芯组合后产生的合缝间隙

## 典型配方与组成
不同应用场景和效率需求下的砂芯胶合剂，在配方方向上差异显著。

**传统胶合剂**：主要由膨润土、纸浆废料、糊精、糖浆等配制而成，价格低廉、制备简单，但粘合的砂芯需自然干燥4～8小时方可投入使用，适用于生产节拍较慢的场合[[S1]]。

**壳型用胶合剂**：专用于壳型铸造中的半型粘接。可利用混合有硬化剂的粉状热塑性酚醛树脂，在半型加热过程中熔化并在冷却后建立粘结强度；亦可采用添加促凝剂、表面活性剂等辅助成分的改性水玻璃[[S1]]。

**快速胶合剂**：匹配热芯盒、冷芯盒等高效制芯生产线的即时组芯需求，主要分为两类：
- **快干胶**：由树脂、溶剂、填料三部分构成，树脂种类与加入量决定强度，溶剂决定干燥速度
- **热熔胶**：聚乙烯类、聚酯类、聚酰胺类等，加热熔融后涂敷，冷却即快速固化，已实现商品化供应[[S1]]

## 工艺角色与使用方法
砂芯胶合剂的使用需嵌入整个砂芯组合工艺流中，与芯头、芯座设计及涂料的配合密切相关。

**粘合前准备**：为保证粘合后的砂芯尺寸精度，在芯子粘合面上须预先留出0.5～2 mm的砂芯磨量（core joint finishing allowance），经打磨或互研平整后方可涂胶[[S4,S5]]。

**涂敷操作**：常用方式包括刷涂、浸涂、喷涂。涂胶时应保持涂层薄而均匀[[S5]]。

**固化工艺**：
- 对于烘干型胶合剂，涂胶压合后送入180～250℃的烘炉中保温约30分钟完成固化；固化后须清除合缝处溢出的多余胶料[[S5]]
- 自硬型或快干型胶合剂则可在常温下自然固化或通过接触压合快速凝结

**与芯头、芯座、涂料的配合关系**：胶合剂粘接后的大砂芯，其最终在型腔中的定位是通过芯头与芯座的准确配合以及必要时的芯撑（chaplet）支承共同完成的[[S6]]。对于铸件表面质量有要求的部位，粘合后的砂芯工作表面通常还需在接合缝及局部损伤处填补修补膏，随后再整体涂刷涂料层并二次烘干[[S5]]。

**砂芯预装**：在机械化砂型铸造生产线上，为平衡节拍，常在专用组装台上预先使用胶合剂将若干小砂芯组装为大砂芯组件，并用砂芯量具检测相对位置[[S5]]。

## 关键性能参数与质量管控
砂芯胶合剂的核心验收指标以干态抗拉强度最为基础。GB/T 5611-2017明确要求其干拉强度大于0.7MPa[[S6]]。在工程实践中，完整的性能检测维度还应覆盖以下各项参数[[S5]]：

| 检测项目 | 技术意义 |
|---|---|
| 干态抗拉强度 | 衡量胶合接头抵抗外力分离的基本能力，最低要求＞0.7 MPa |
| 粘度/流动性 | 影响涂敷工艺性和浸渗深度 |
| 适用期 | 胶合剂配制后保持可使用状态的有效时间窗口 |
| 固化时间 | 在规定温度或气氛条件下达到使用强度的所需时长 |
| 高温粘结强度 | 在浇注温度下维持接头完整性的能力 |
| 发气量与发气速度 | 与铸件气孔缺陷密切相关，需控制在允许范围内 |
| 落砂溃散性 | 浇注后胶合接头在铸件清理阶段的破碎脱落难易程度 |

**质量控制关键点**[[S4]]：
- 配制过程中需不断搅拌，防止结块
- 配制出的胶合剂必须经型砂实验室测试合格后，方可贮存备用
- 必须明确规定产品的有效期，超期胶合剂应作报废处理
- 在不改变配合比的原则下，每次配制数量可适当增减

## 典型缺陷与局限性
砂芯胶合剂若选择或使用不当，会直接诱发多种铸件缺陷[[S4]]：

- **气孔缺陷**：胶合剂中残留的大量挥发性溶剂未完全烘干，浇注时骤热析出过量气体侵入金属液
- **尺寸偏差/型芯移位**：胶合剂粘接强度不足，浇注过程中金属液对砂芯的浮力和冲击力导致接头松脱或砂芯整体漂移
- **表面粘砂**：胶合剂耐火度不足，在高温下提前软化并与金属液发生反应，造成铸件表面局部粘砂

此外，胶合剂自身在贮存过程中存在老化风险。行业规程明确要求胶合剂必须规定有效贮存期，逾期者严禁使用[[S4]]。

## 相关标准与规范
- **GB/T 5611-2017《铸造术语》**：给出砂芯胶合剂的法定定义及最低干拉强度要求（＞0.7 MPa）[[S6]]
- **JB/Z 234.5-85《手工造型工艺规程》**：单列“砂芯胶合剂配制工艺”一节，系统规定了配料表、持续搅拌、试验室合格检验及有效期报废等质量管控要点[[S4]]
- **印度标准IS 8779-1978**：Core gums (sodium silicate based) for use in foundries——水玻璃基芯胶的技术条件[[S7]]
- **印度标准IS 9008-1978**：Core repairing paste for use in foundries——砂芯修补膏的技术条件[[S7]]
