## 概述

附铸试块（英语：cast-on test block / attached test coupon / integral test block）是将试块直接附连在铸件重要部位、与铸件在同一铸型内一体浇注成型的试样坯料[[S25,S32,S43]]。其核心作用是为后续力学性能检测提供与铸件本体冷却凝固条件更为接近的代表性试样[[S32]]。附铸试块须在铸件全部热处理工序完成后方可从铸件本体上切割取下，切割后不损害铸件本体的使用功能[[S47,S14,S24]]。

作为通用铸造领域的重要质量检验手段，附铸试块区别于单独造型浇注的单铸试块，其检测得到的材质性能更加贴近铸件本体的实际性能[[S25,S32,S43]]。但该概念设计之初针对的是厚大砂型铸件，并不针对高压压铸的薄壁快速凝固工艺场景[[S23,S7,S47,S1,S32]]。

## 适用范围

附铸试块的常规适用对象为大型厚壁铸件[[S23,S7,S47,S1]]：

- 当铸件质量 ≥2000 kg、壁厚在 30~200 mm 时，**优先采用**附铸试块[[S32,S43,S12,S26,S30]]。
- 当铸件质量 >2000 kg 且壁厚 >200 mm 时，**必须采用**附铸试块[[S32,S43,S12]]。

## 分类

### 按形状分类

根据 ISO 4990 标准规定[[S25]]：

| 铸件主要壁厚 t (mm) | 试块形状 | 说明 |
| :--- | :--- | :--- |
| 28 < t ≤ 56 | **t × t 方柱形**附铸试块 | 试样中心线距铸造表面距离不少于 14 mm |
| t > 56 | **t × t 方柱形** 或 **t × 3t × 3t 板状**附铸试块 | 板状试块最大尺寸不超过 500 mm |

### 按检测用途分类

附铸试块可加工用于多种检测[[S32,S35]]：

- 截取**拉伸试样**，测定抗拉强度、屈服强度、断后伸长率等；
- 加工为 **V 型缺口冲击试样**，测定冲击吸收功；
- 制备**金相试样**检验显微组织；
- 制备**光谱试样**检验化学成分；
- 直接用于**布氏硬度**检测。

## 与本体铸件的关系

附铸试块与铸件之间存在严格的工艺关联要求，以保证试块的代表性：

- **同炉同包**：必须使用浇注对应铸件的同一炉、同一包的金属液浇注，不允许使用单独批次金属液[[S42,S7]]。
- **冷却条件一致**：附铸试块在铸件上的安装位置应保证其冷却条件与所代表铸件部位大致相仿，放置在铸件有代表性的部位[[S42,S43]]。
- **位置要求**：具体放置位置由供需双方协商确定，不得影响铸件的使用性能、外观质量，同时须保证试块自身组织致密无缺陷[[S42,S43,S26,S30]]。
- **热处理同步**：附铸试块须与对应铸件同步同炉进行全部热处理工序，**只有在热处理全部完成后才可从铸件上切割取下**；仅针对消除应力的时效处理工序，附铸试块可随铸件完成而无需单独处理[[S47,S7,S42,S14]]。

## 工艺设计与缺陷预防

### 模数匹配

附铸试块设计的关键原则是使试块与铸件对应取样部位的**模数比值接近 1**，以保证二者凝固冷却条件基本一致。ProCAST 模拟结果显示，当内浇道与热节的模数比为 1.17 时，试块及对应区域无缩松缺陷[[S44]]。

### 连接颈部设计

附铸试块的连接颈部需经供需双方商定。标准规定，当采用比 A 型更小尺寸的附铸试块时，须满足 b = 0.75a、c = a/2 的尺寸约束[[S26,S30,S32]]。连接部位应设计适当的圆角过渡，避免应力集中。

### 常见缺陷及预防

| 缺陷类型 | 主要成因 | 预防措施 |
| :--- | :--- | :--- |
| **缩松** | 补缩通道设计不合理、补缩压力不足 | 优化冒口与冷铁布局、匹配模数参数[[S24,S39,S28]] |
| **气孔** | 充型过程中卷入气体、型砂发气量过大 | 优化浇注系统排气设计、严控原材料除气工艺[[S24,S39,S28]] |
| **裂纹** | 连接颈部应力集中、铸件与试块冷却收缩不同步 | 优化连接部位圆角过渡设计、控制冷却速度差值[[S24,S39,S28]] |

## 与单铸试棒、本体取样试块的对比

| 对比维度 | 单铸试棒 | 附铸试块 | 本体直接取样 |
| :--- | :--- | :--- | :--- |
| **力学性能代表性** | 仅反映同批次铁水整体冶金质量，与铸件本体差异较大[[S36,S15]] | 贴近铸件本体实际性能，但仍不能完全等同[[S32,S15,S46]] | 完全对应铸件关键受力部位真实性能[[S36,S15]] |
| **检测结果准确性** | **最低**[[S36,S15]] | **次之**，偏差小于单铸试棒[[S36,S15]] | **最高**[[S36,S15]] |
| **生产成本** | **最低**，可独立批量浇注加工[[S15]] | **次之**，仅需预留连接部位余量[[S15]] | **最高**，会破坏完整铸件，需后续修复[[S15]] |
| **适用场景** | 批量小、常规薄壁小件的日常批量验收[[S12,S34]] | 单重 ≥2000 kg 的大型厚壁铸件，需贴近性能又不能破坏本体[[S12,S34]] | 批次抽样检验、供需双方有特殊验证要求的场景[[S12,S34]] |

TC4 合金受油管的实测数据也证明了附铸试样的优势：附铸试样 UTS 平均 991 MPa、YS 886.5 MPa、EL 9.5%，各项性能均优于铸件本体，性能均匀性明显提升[[S46]]。

## 应用场景

- **大型球墨铸铁/灰铸铁件力学性能验收**：对应 GB/T 1348、GB/T 9439 等标准。牌号后标注“A”即代表性能由附铸试块测定[[S11,S12,S13]]。
- **铝合金低压铸件力学性能验收**：如 ZL114A 导弹舱体低压铸造工艺中，于缝隙浇道底部设置附铸试样，辅以 45# 钢激冷冷铁保证组织致密[[S3,S4]]。
- **铸造工艺验证**：在新工艺开发过程中，附铸试块可同步获取与铸件同充型、同凝固、同热处理条件下的力学性能数据，用于验证工艺参数调整效果[[S40,S22]]。

## 在高压压铸中的适用性与局限性

### 基本定位

附铸试块原属砂型重力铸造领域的概念，设计目标是保证试块与铸件的冷却凝固条件大致相仿，以更准确地反映厚大铸件本体力学性能。该概念从设计之初并未针对高压压铸的薄壁快速凝固工艺场景[[S23,S7,S47,S1,S32]]。

当前高压压铸行业**不存在**附铸试块的大规模常规使用情况，仅在极个别大尺寸厚壁压铸件（如平均壁厚 ≥15 mm 的大排量铝合金缸体）生产场景中有零星尝试[[S5,S48]]。

### 不适配原因

高压压铸与常规重力铸造在铸件壁厚和冷却速度上存在根本差异，导致传统附铸试块无法适用：

- 常规压铸件壁厚大多在 1~6 mm 区间，铝合金压铸件最小壁厚可低至 0.5 mm，金属液在钢制压铸模腔内冷却速度极快，表层可形成 0.3~1 mm 左右的细晶粒致密硬化层[[S31,S38,S19,S18,S9,S45,S20]]。
- 传统附铸试块截面厚度通常 ≥30 mm，是为砂型厚大铸件设计的[[S21,S26]]。若在薄壁压铸件上强行设置厚大附铸试块，试块因截面热容量远大于压铸件本体，凝固缓慢，内部易生成缩孔、缩松等缺陷，且结晶晶粒远大于薄壁处的激冷细晶组织，最终力学性能检测结果完全失真[[S5,S48]]。

高压压铸件壁厚对力学性能影响显著——厚壁处硬度显著低于薄壁处，这进一步印证了厚大附铸试块在薄壁压铸场景下的不适用性[[S10]]。

### 压铸行业替代检测方案

| 替代方案 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **同工艺单铸试棒/试块** | 在相同压铸工艺条件下独立成型，符合 GB/T 13822《压铸有色合金试样》标准 | 行业批量生产中最常用的力学性能检测手段[[S29,S5,S27]] |
| **压铸件本体取样** | 从压铸件指定关键受力部位直接切取试样 | 试制或关键受力件批量稳定生产场景，检测结果最真实[[S29,S5]] |
| **硬度测试 + 硬度-强度换算** | 结合已建立的硬度-抗拉强度换算关系快速校验材质 | 配合 X 射线/超声无损探伤、铸件解剖抽检完成综合验证[[S32,S43,S29]] |
