浇注速度，又称浇注速率，是铸造工艺的核心参数之一，指单位时间内流过浇注系统特定截面的熔融金属的量。该参数存在质量流速（法定单位 kg/s）与线速度（法定单位 m/s）两种表达形式 [[S17,S24,S31,S36]]。

## 定义与量纲
在铸造工程中，浇注速度主要有以下两类物理表达：

*   **质量流速 (Pouring rate)**：单位时间内通过浇注系统某截面的金属液质量，多用于重力铸造的整体浇注能力与充型耗时估算 [[S17,S24]]。
*   **线速度 (Linear velocity)**：金属液流动的线速度，多用于校核充型平稳性、防止紊流卷气及冲刷，例如型腔内的液面上升速度 [[S99,S61]]。

在压铸领域，有三个核心术语需严格区分 [[S51,S60,S19]]：
*   **压射速度 (Injection speed)**：压铸机压射冲头前进的线速度，单位为 m/s。
*   **充填速度/内浇口速度 (Filling/Gate velocity)**：熔融金属在压力作用下通过内浇口进入型腔的平均线速度，单位为 m/s。二者完全等价 [[S51,S62,S76]]。
*   **速度关联**：根据等流量连续流动原理，三者满足体积守恒关系：压射室截面积 × 压射速度 = 内浇口截面积 × 充填速度 [[S62,S76,S15,S116]]。

## 对充型行为与缺陷的工艺作用
浇注速度直接决定金属液在型腔内的流动状态（层流或紊流），进而影响铸件的最终质量。

### 流动状态判定
流体状态由雷诺数（Re）判定。液态金属运动黏度约为1×10⁻⁶ m²/s，当 Re < 2320 时为层流，流动阻力与流速一次方成正比；当 Re > 13800 时进入紊流，阻力与流速平方成正比 [[S38,S72,S74]]。
不同工艺的层流临界速度差异显著：
*   普通砂型重力铸造，实现层流充填的最大速度需低于 0.3 m/s [[S58]]。
*   镁合金低压铸造，充型速度 ≤ 42 mm/s (≤ 0.7 kPa/s) 时可实现层流平稳充型，超过 50 cm/s 则转为紊流 [[S20,S85]]。
*   AZ91D 镁合金挤压铸造，在 0.91 m/s 时保持层流，充型性好；在 2.10 m/s 时转为紊流，易产生夹杂和气孔 [[S120]]。

### 速度失当引发的铸造缺陷
浇注速度过快或过慢，均会通过不同的物理机制引发多种铸造缺陷，其因果关系归纳如下：

| 缺陷类型 | 浇注速度过快（机制与后果） | 浇注速度过慢（机制与后果） |
| :--- | :--- | :--- |
| **气孔/裹气** | 金属液紊流严重，裹挟大量气体；型腔内气体难以及时排出 [[S54,S8]]。 | - |
| **氧化夹杂** | 流股高速冲刷型壁、产生飞溅，造成金属液二次氧化 [[S54,S75]]。 | - |
| **冲砂** | 金属液冲击压力超过型砂承压阈值，冲毁型壁或砂芯 [[S54,S78]]。 | - |
| **缩孔/缩松** | 补缩通道被过早凝固的金属封堵 [[S54,S75]]。 | 补缩不足，导致内部出现收缩孔洞 [[S37]] |
| **热裂** | 铸件不同部位温差增大，热应力加剧 [[S54,S75]]。 | 铸件各区域温差大增，热应力不均 [[S37]]。 |
| **浇不足/冷隔** | - | 沿程温降过大，远端金属液降至液相线以下，流股前沿无法熔合 [[S5,S37]]。 |
| **夹砂** | - | 砂型受高温金属液长时间烘烤，表层脱落 [[S37]]。 |

## 影响浇注速度选择的关键因素
浇注速度的优选需综合考虑合金特性、铸件结构与铸造工艺类型。

**合金特性**
*   **粘度**：粘度随温度降低而升高，流动性下降，需更高浇注速度补偿以避免浇不足 [[S82]]。
*   **密度**：在相同压射比压下，合金密度越低，可实现的浇注速度越高 (v = √(2P/ρ)) [[S52]]。
*   **凝固区间**：宽凝固区间的糊状凝固合金（如锡青铜），推荐采用略高速度以保证在大量固相析出前完成充型；窄区间合金则优先匹配低中速，配合顺序凝固 [[S71,S82]]。
*   **氧化倾向**：含铝、锌等易氧化元素的合金，需降低浇注速度实现平稳充型，防止卷入二次氧化夹杂 [[S71,S28]]。

**铸件结构**
*   **壁厚**：薄壁铸件（尤其是钢件），为避免提前凝固产生冷隔、浇不足，必须采用较高浇注速度 [[S37,S13]]。其最小浇注速度可由经验公式 v_J = K·L/δ_min 估算 [[S13]]。
*   **表面积与体积比**：该比值越高（如板状件），散热越快，所需推荐最小浇注速度越高 [[S23,S25]]。
*   **特殊结构**：复杂叶轮类铸件需匹配充型节奏，如铝合金消失模叶轮推荐的稳定浇注速度约为 1.2 kg/s [[S73]]。

**工艺类型**
*   **重力铸造 (砂型/金属型/熔模)**：铸铁、铸钢及非铁合金的推荐浇注速度随铸件重量、壁厚不同而变化，范围可从 0.5 kg/s 至 60 kg/s [[S63,S45]]。金属型铸造常采用“先慢后快”的策略 [[S26]]。
*   **压铸 (冷室/热室)**：充填速度（内浇口速度）极高，通常为 30~100 m/s，以保证在极短时间内完成充型 [[S62,S76]]。薄壁件、表面质量要求高时取高值；厚壁件、内部致密度要求高时取低值 [[S69]]。
*   **低压铸造**：充型速度以压力速率（dP/dt）表征，如薄壁金属型铸件为 0.002~0.005 MPa/s [[S113]]。
*   **消失模铸造**：浇注速度需与泡沫模样的气化速度匹配，浇注温度越低、模样密度越高，所需最小浇注速度越高 [[S23]]。

## 典型推荐值与数值模拟优化
不同铸造工艺条件下，浇注速度存在明确的工程推荐范围。

### 压铸合金的充填速度推荐值
各合金均存在一个较为公认的充填速度适用范围，下表对此进行了汇总 [[S62,S69,S87,S96]]。

| 合金类型 | 推荐充填速度范围 (m/s) | 备注 |
| :--- | :--- | :--- |
| 铝合金 | 25 ~ 40 | 薄壁、复杂件取上限；厚壁、致密件取下限。 |
| 镁合金 | 40 ~ 100 | 镁合金流动性好，可选用高充填速度。 |
| 锌合金 | 25 ~ 50 | - |
| 铜合金 (如 YZCuZn40Pb) | 20 ~ 50 | - |

### 数值模拟软件优化案例
现代铸造工艺常借助 ProCAST、MAGMA、AnyCasting 等软件，通过模拟充型、凝固过程来优化浇注速度，预测并消除缺陷 [[S80,S50,S59]]。
*   **ProCAST 优化案例**：针对 ZG12CrMoV 转子盘熔模铸造，通过正交试验优化得到浇注速度为 309 mm/s 时，铸件无缩孔缺陷 [[S105]]。对大马力发动机排气弯头铸钢件，优化浇注速度 1.5 kg/s 可使铸件无铸造缺陷 [[S11]]。GH4738 合金真空感应熔炼铸锭的仿真表明，采用变速浇注（12-6 kg/s）可有效控制缩孔 [[S91]]。
*   **AnyCasting 优化案例**：对 A356 铝合金倾转半固态制浆，模拟表明临界浇注速度需控制在 ≤ 0.5 m/s 以避免卷气 [[S11]]。AnyCasting 通过残余熔体模数（M = V/A，即孤立液相区体积与表面积之比）模块识别收缩缺陷高风险区 [[S107,S27,S104]]。研究发现，在铝合金挤压铸造中，通过优化浇注速度等参数可将残余熔体模数控制在低水平（如 < 1），有效降低缩孔缩松缺陷率 [[S39,S102]]。

## 工程计算与控制
### 核心计算公式
在重力铸造中，金属液的质量流速是浇注系统设计的核心，其与浇注系统阻流断面截面积的关系由奥赞公式（流体力学推导）给出 [[S68,S86]]。对于压铸工艺，浇注速度的计算与控制则基于完全不同的原理 [[S47,S52,S84,S83,S56,S10,S6]]。

*   **质量流速与断面关系 (重力铸造)**：`F_min = G / (0.31 × 10⁵ μ t √H_p)`，式中 `G` 为金属液总质量(kg)，`t` 为浇注时间(s)，`μ` 为流量因数，`H_p` 为平均静压头(m) [[S86]]。
*   **内浇口速度与压射速度的关系 (压铸)**：`v_n = (π d² v_y) / (4 A_n)`，式中 `v_n` 为内浇口速度(m/s)，`d` 为压射冲头直径(m)，`v_y` 为压射速度(m/s)，`A_n` 为内浇口总截面积(m²)。这说明内浇口速度与压射冲头直径的平方、压射速度成正比，与内浇口截面积成反比 [[S47]]。
*   **内浇口速度与压射比压的关系 (压铸)**：`v_n = η √(2 p_b / ρ)`，式中 `η` 为阻力系数（通常取 0.3~0.6），`p_b` 为压射比压(Pa)，`ρ` 为合金液态密度(kg/m³)。这表明提高压射比压或降低合金密度均可获得更高的充填速度 [[S84,S83]]。

### 浇注时间反推法
对于铸铁件，常先根据铸件重量和壁厚由经验公式确定浇注时间，再反推所需的浇注速度：
*   **中小型铸铁件 (质量 < 450 kg)**：`t = S √m` (S 为与壁厚相关的经验系数) [[S95,S67]]。
*   **中大型铸铁件 (质量 100~10000 kg)**：`t = S₁ ∛(δ m)`，(常规 S₁=2.0，快浇 S₁=1.7~1.9) [[S67,S77]]。

### 浇注速度的模块化设定与先进控制
为保证浇注速度稳定，横浇道截面积需沿程均匀或逐渐缩小，避免截面积突变导致涡流和速度波动 [[S81,S48]]。不同类型的浇注系统（加压/非加压）也直接影响流动状态 [[S100,S21]]。

在压铸中，压射速度被进行精细分段控制，以实现最佳的排气与充型效果 [[S19,S101,S64,S103]]。
*   **慢压射阶段**：速度通常 < 0.3 m/s，目标是平稳推动金属液充满压室，并将室内空气充分排出而不产生卷气 [[S19,S101]]。
*   **快压射阶段**：金属液被高速推入型腔，速度可高达 4~10 m/s。

在现代自动化铸造中，先进的实时控制系统能实现对浇注速度的超高精度控制。例如，带伺服阀的高精度自动浇注系统可将浇注速度控制误差降低至 0.1~0.4 kg/s 量级 [[S29,S33]]。东芝机械开发的音圈伺服阀压铸控制系统，能在 19ms 内实现压射速度从 0 到 10 m/s 的无级切换，有效减少铸件内部气孔 [[S30,S9]]。

## 相关标准与规范
涉及浇注速度/充填速度或相关术语的标准包括：
*   **GB/T 21269-2007** 明确规定了国产压铸机的最高空压射速度应 ≥ 6~7 m/s [[S101]]。
*   **GB/T 25370-2020《铸造机械 术语》** 对压铸领域的快压射速度、快压射行程等术语进行了规范 [[S79]]。
*   **JB/T 8084.2-2000** 规定了卧式冷室压铸机压射性能的参数分类标准，包含最大空压射速度等相关指标 [[S62]]。
*   公开文献提及，德国 **VDG 压铸行业技术规范** 给出了不同吨位压铸机对应的最大推荐压射速度区间 [[S111]]。
