拉杆（塑料模）是塑料注射模具中设置在多分型面模具（如三板模）内的核心定距部件，其本质功能是在开模时限制指定模板仅在设定的限定距离内完成拉开和停止动作，从而实现顺序分型，保证浇注系统凝料顺利取出和各分型面的开模动作有序完成[[S18,S37]]。在专业文献与教材语境中，拉杆也被称为“定距拉杆”或“小拉杆”[[S18,S19]]。在标准体系中，兼具导向与定距双重功能的零件被明确定义为“拉杆导柱”[[S10]]。

## 术语边界辨析

在模具行业口语及部分非规范文献中，“拉杆”一词存在多义性，需严格区分以下易混淆概念：

*   **注塑机合模机构拉杆 (Tie Bar)**：连接注塑机固定模板与移动模板的承载部件，属于注塑机本体结构，**不属于**塑料注射模的内部零件，与本词条讨论的拉杆在功能与归属上完全不同[[S2,S33]]。
*   **拉料杆 (Sprue puller)**：依据GB/T 8845-2017《模具术语》，其定义为开模分型时拉住浇注系统凝料或制件、头部带有侧凹形状的杆状零件[[S30]]。其核心功能是开模时将主流道凝料留在动模侧，与以定距为核心功能的拉杆边界完全不同。在部分口语场景中，拉料杆可能被误称为“拉杆”，但在技术定义上二者相互独立[[S9,S34]]。
*   **复位杆 (Return pin)**：部分场景被俗称为“复位拉杆”，但依据标准定义，其功能为借助模具闭合动作推动推出机构复位的杆状零件[[S30]]。其结构、安装位置与以定距为核心功能的拉杆完全不同，不属于标准定义的拉杆范畴[[S16,S39]]。

## 分类与形式

根据功能与安装应用场景，塑料注射模中的拉杆类零件主要可分为以下几类：

*   **定距小拉杆**：三板式点浇口注塑模中最典型的定距部件。用于限制脱料板（流道推板）与定模板（A板）的开模行程。它通常为一体式或分体式结构，其设计核心是通过杆身上的限位台阶实现行程控制[[S19]]。此类拉杆的安装必须考虑足够空间，以确保机械手能够顺利取出浇注系统凝料[[S6,S7,S31]]。
*   **拉杆导柱**：依据GB/T 4169.20—2006《塑料注射模零件 第20部分：拉杆导柱》规定的标准零件。此类零件兼具导向与定距双重功能，可同时为动定模开合模提供导向，并限制特定分型面的开模行程[[S10,S15]]。
*   **标准模架配套拉杆**：在点浇口标准模架（如富得巴 D&E 系列）中，拉杆按安装位置与附件组合可分为四类[[S22,S25]]：

| 拉杆配置类型 | 说明 |
| :--- | :--- |
| OH 型 | 拉杆安装在模架外侧，附导套 |
| IH 型 | 拉杆安装在模架内侧，附导套 |
| ON 型 | 拉杆安装在模架外侧，无导套 |
| IN 型 | 拉杆安装在模架内侧，无导套 |

该系列模架的拉杆长度覆盖90mm至320mm的全系列标准规格[[S22,S25]]。与之功能类似的还有安装在模具外侧的拉板，在空间受限时作为替代方案[[S4,S7]]。

## 结构与尺寸设计

定距拉杆的典型结构要素包括头部的限位台肩、杆身的光滑段、以及末端的连接螺纹。其关键尺寸设计需根据模具的开模行程、模板厚度及凝料取出空间等因素确定[[S19]]。

### 安装与配合

定距拉杆的常规安装数量为4支，要求沿模具分型面四周均匀对称布置，以确保各拉杆受力均衡，防止偏载[[S7]]。其典型的连接方式为一端依靠头部限位台肩定位，另一端通过杆身末端的螺纹段配合螺钉锁紧，实现与对应模板的轴向刚性固定[[S4]]。

对于拉杆导柱，其配合公差有严格要求：
*   **固定段**：杆身固定安装段采用 **H7/m6** 过渡或小过盈配合，实现压装固定[[S20,S24]]。
*   **滑动段**：滑动导向段与对应模板孔采用 **H7/f7** 间隙配合，以保证运动顺畅[[S20,S24]]。

## 标准零件规范

GB/T 4169.20—2006规定了塑料注射模用拉杆导柱的尺寸规格与公差。其主要技术条件包括[[S13,S15,S36]]：
*   **未注表面粗糙度**：Ra = 6.3 μm。
*   **未注倒角**：C1。
*   **形位公差**：同轴度、直线度等应符合GB/T 1184—1996的规定，精度等级为6级[[S13]]。
*   **其他工艺要求**：可选设砂轮越程槽或R0.5~R1 mm圆角；允许开设油槽；允许保留两端中心孔[[S13,S35,S36]]。

其标准直径系列最小为8mm，常规规格包括8mm、10mm、12mm、14mm、16mm等，最大可至63mm；标准长度覆盖150mm至800mm全区间[[S12,S41]]。

## 关键材料与热处理

根据工况与标准要求，拉杆导柱及定距拉杆的材料选择与热处理规范如下：

| 适用工况 | 推荐材料 | 热处理与硬度要求 |
| :--- | :--- | :--- |
| 普通规格（局部受冲击） | 45钢 | 球面等局部位置热处理至 **38~45 HRC** [[S13,S35]] |
| 轻载定距拉杆 | 45钢 | 整体热处理至 **34~38 HRC** [[S4]] |
| 重载或耐磨性要求高 | T10A, GCr15 | 整体淬火回火至 **56~60 HRC** [[S13,S36]] |
| 重载或耐磨性要求高 | 20Cr | 渗碳淬火，渗碳层深度 **0.5~0.8 mm**，硬度 **56~60 HRC** [[S13,S36]] |
| 部分定距拉杆 | T8A | 整体热处理至 **53~57 HRC** [[S38]] |

## 工作原理与动作时序

定距拉杆是实现多分型面模具顺序开合模的关键元件。以典型的三板式点浇口模具为例，其工作时序如下[[S1,S27,S29,S37]]：

1.  **第一分型面打开（A分型面）**：在弹簧或尼龙锁模器作用下，脱料板与定模板首先分离。此过程中，点浇口被拉断，浇注系统凝料在拉料杆作用下留在定模侧脱料板上。
2.  **第二分型面打开（B分型面）**：开模到达设定距离后，定距小拉杆头部的限位台肩与定模板接触。拉杆通过刚性连接，带动流道推板（脱料板）运动，使其与定模座板分离。此动作将浇注系统凝料从拉料杆及浇口套中强行脱出，实现自动落料。
3.  **主分型面打开（C分型面）**：模具继续开模，在另一组拉杆或拉钩机构作用下，定模板与动模板最终分离。塑件包紧在动模型芯上随动模移动，进入后续顶出脱模工序。
4.  **合模复位**：合模过程中，各模板依次复位。拉杆的杆身在模板导向孔内空程滑动，直至限位台肩完全贴合对应模板端面，为下一次开模的定距动作做好准备[[S8]]。

## 常见故障与预防措施

| 故障模式 | 常见成因 | 预防与处理措施 |
| :--- | :--- | :--- |
| **断裂** | 设计应力或强度不足、材料硬度过高导致韧性下降、安装不同心引发偏心交变载荷、运行中出现模板卡滞导致瞬时过载[[S32,S40]]。 | 合理校核许用强度，选用韧性匹配材料，保证同轴度安装精度，定期检查运动间隙以防卡滞[[S32,S40]]。 |
| **弯曲变形** | 材料硬度或屈服强度不足、杆件长径比过大、承受侧向偏载力矩[[S26,S40]]。 | 优化杆件径长比，选用高强度材料并保证热处理硬度达标，增设辅助导向结构以抵消侧向载荷[[S26,S40]]。 |
| **拉毛与磨损** | 滑动副零件硬度差过小、配合间隙侵入硬质异物、润滑不足导致干摩擦[[S17,S23]]。 | 保证导套硬度低于拉杆5 HRC左右，每班次对滑动部位加注润滑油脂，定期拆解清理并检查，磨损轻微时可研磨修复，超差后更换新件[[S17,S23]]。 |
| **卡死** | 拉杆与配合孔垂直度超差、热膨胀导致间隙不足、模板变形引发轴线偏移[[S3,S23]]。 | 加工时采用合模后一次镗孔工艺保证同轴度，设计时预留合理运动间隙，增设侧向定位块以抵抗注射侧向力[[S3,S23]]。 |
