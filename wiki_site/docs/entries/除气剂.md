## 定义与概念辨析
除气剂是压铸及金属熔炼场景下，常温下呈固态的除气耗材。使用时通常借助钟罩压入铝液，依靠自身热分解或与铝液的化学反应生成弥散气泡，脱除铝液中溶解的氢元素，同时附带去除部分悬浮非金属夹杂[[S15,S11]]。该类材料与通入氮气、氩气等惰性气体的旋转除气工艺有明确区分，前者以固体化学产气为核心机制，后者以物理剪切分散气泡为原理。业界对“除气剂”这一术语无明显定义分歧，但在实际应用中常与“精炼剂”“除渣剂”等术语存在功能交叉，后文将予以辨析。

## 分类与成分
固体除气剂按其活性成分与功能复合程度，可分为以下主要类型。

### 六氯乙烷基除气剂
以六氯乙烷（C₂Cl₆）为有效活性成分，部分复配产品添加氟硅酸钠（Na₂SiF₆）等组分以缓和反应速度、延长作用时间[[S12,S18]]。C₂Cl₆密度约2.097 g/cm³，本身不吸湿，使用和保存方便[[S34,S18]]。常规产品形态为压制而成的直径φ30~50 mm、厚度10~20 mm的圆饼或片状[[S15]]，采用密封干燥容器包装。

### 复盐型混合除气覆盖剂
典型成分为氯化钠、氯化钾、氟硅酸钠、碳酸钠、氟化钙构成的复配体系[[S21,S31]]。活性成分主要为可热分解产气的Na₂SiF₆与Na₂CO₃，同时兼具除气、除渣、液面覆盖多重作用。产品通常为颗粒状或粉状，以防水编织袋或塑料桶密封包装[[S20]]。

### 无毒硝酸盐/碳酸盐类除气剂
以硝酸钠和碳为核心组分，典型反应生成N₂和CO₂气泡实现除气，部分产品添加冰晶石粉、耐火砖粉等调节反应速率[[S11,S34]]。此类除气剂同时兼具一定的铝硅合金变质作用，但发烟量大、金属损耗率相对偏高[[S11]]。常用精炼剂配方见下表[[S37]]。

| 项目 | 内容 |
|------|------|
| 1号配方 | 硝酸钠34% + 石墨粉6% + 冰晶石粉20% + 食盐10% + 耐火砖粉30% |
| 2号配方 | 硝酸钾40% + 石墨粉6% + 冰晶石粉24% + 食盐10% + 耐火砖粉20% |
| 3号配方 | 硝酸钠40% + 石墨粉6% + 氟硅酸钠24% + 食盐10% + 耐火砖粉20% |

### 含稀土/铝基中间合金的复合净化剂
在基础除气组分之外额外添加Al-Ti-B、Al-Sr或稀土元素类铝基中间合金，可在除气作业的同时完成晶粒细化与变质处理，无需再单独添加细化剂。产品形态多为块状或蜂窝状结构，采用铝箔封装后装入铁桶密封保存[[S4,S33]]。

### 压铸专用除气剂
针对压铸场景小容量保温炉短流程精炼需求开发，添加量更低，产气速度快，可在2~5 min内完成全部反应实现除气，适配压铸不间断连续生产节奏。产品形态多为小尺寸独立封装片，采用铝箔袋分装或桶装，单份质量适配单炉保温炉铝液标准处理量，无需现场额外称量[[S14,S23]]。

## 工艺作用与反应机理
固体除气剂的核心工艺作用是通过化学反应生成大量不溶于铝液的气泡，利用分压差原理脱除铝液中溶解的氢，同时气泡在上浮过程中吸附氧化铝等非金属夹杂物，实现除气与除渣双重效果。

**六氯乙烷类反应路径**：C₂Cl₆受热分解生成C₂Cl₄和Cl₂，Cl₂进一步与铝反应生成气态AlCl₃。AlCl₃（沸点183℃）与C₂Cl₄（沸点121℃）均为不溶于铝液的气态物质，形成大量气泡上浮，铝液中溶解的氢在分压差作用下扩散进入气泡并随之逸出，同时气泡吸附携带夹杂物至液面[[S43,S34]]。

![活性氯气体除气分压差原理示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/e831e817-fd33-4ffc-aa45-58e21d3d140f/resource)

**氟盐类反应机理**：以Na₂SiF₆为代表，受热分解为NaF和气态SiF₄，SiF₄气泡上浮实现除氢。同时生成的NaF与铝液中Al₂O₃反应生成冰晶石Na₃AlF₆，可溶解润湿氧化铝夹杂物，实现高效除渣[[S21,S31]]。

**无毒硝酸盐类反应机理**：以硝酸钠与碳反应生成Na₂CO₃、N₂和CO₂为代表，N₂和CO₂均为不溶于铝液的气体，以气泡形式实现氢的捕获和夹杂物浮选携带[[S34,S11]]。

## 操作工艺与参数
### 温度与添加量
固体除气剂通用使用温度范围为670~750℃，常规工业作业控制区间为690~720℃[[S38,S3]]。常用固体除气剂的推荐添加量见下表[[S6,S15,S40]]。

| 除气剂类型 | 推荐添加量 |
|-----------|-----------|
| 六氯乙烷（C₂Cl₆） | 炉料总重0.2%（约2000 g/吨） |
| 氯化锌（ZnCl₂） | 炉料总重0.15%~0.2%（约1500~2000 g/吨） |
| 氯化锰（MnCl₂） | 炉料总重0.1%~0.2%（约1000~2000 g/吨） |
| 常规复合型除气剂 | 1500~2500 g/吨 |

### 加入方式
目前主流工业加入方式有三种[[S38,S46,S1]]：

1. **钟罩压入法**：将除气剂块用铝箔包裹后装入带孔的压入钟罩，浸入铝液底部缓慢匀速移动，分3~4批次逐步加入，待铝液“翻腾”静止后再加入下一批次[[S38]]。
2. **氮气携带喷吹法**：以0.1~0.35 MPa的高纯氮气作为载气，将粉末状除气剂通过不锈钢喷管送入铝液底部，全程移动喷管避免出现除气死角[[S46,S1]]。
3. **炉前直接投入法**：除气剂预造粒后直接均匀撒入铝液表面，依靠铝液高温自然分解反应。

### 静置与效果检验
除气作业完成后，铝液最小静置时间应不低于10 min，以保证未完全上浮的小气泡充分逸出，浇注前需完成除气效果复检[[S41,S43]]。常见检验方法包括目测法（观察液面是否有气泡逸出）和真空凝固法[[S38]]。

## 效能表现与影响
### 除气效率
工业实测中，常规固体除气剂的氢去除效率为50%~70%，处理后铝液氢含量可降至0.15 mL/100g Al以下[[S19,S45]]。压铸铝液氢当量（DI值）可降至2%以下，铸件针孔缺陷等级可降低1~2级[[S45]]。

不同除气工艺的效果对比表明，六氯乙烷除气效果最优，氩气旋转除气效果随静置时间延长逐步提升，无毒除气剂效果次之[[S2]]。

### 对铸件质量的影响
规范使用除气剂可使压铸铝合金铸件的抗拉强度提升10%~15%，气密性测试合格率较未除气铸件提升30%以上[[S25,S13]]。除气不足将直接导致铸件内部产生气孔、针孔缺陷，严重降低力学性能与气密性。

### 不当使用的副作用
除气剂使用不当可能引发三类问题[[S29,S43,S30]]：

1. **氯残留**：过量添加氯盐类除气剂会导致合金中氯元素含量超标，降低铸件耐腐蚀性能。
2. **腐蚀性烟气**：氯盐除气剂反应生成的Cl₂、AlCl₃、HCl均为刺激性强腐蚀性烟气，无完备抽风系统会危害操作人员健康并腐蚀车间设备。
3. **炉渣增加**：过量添加使浮渣生成量提升20%~30%，增加金属烧损率，降低坩埚耐火材料寿命。

添加量不足则除气反应不完全，铸件产生针孔、气孔缺陷的概率大幅提升[[S15,S10]]。对于ZnCl₂和MnCl₂类除气剂，还需注意多余金属元素（Zn、Mn）进入合金后可能造成化学成分超标[[S15]]。

## 局限性与环保趋势
氯盐除气过程中生成的AlCl₃、HCl等气态产物属于有害污染物。国内现行环保法规要求氯系除气剂作业必须配套完备的抽风排气系统，近年行业政策正逐步限制纯氯气、高氯含量除气工艺的使用，推广低氯/无氯替代方案[[S8,S26,S38]]。

当前主流无氯环保替代方案采用MgCl₂基低氯熔剂搭配Ar气作为载气的旋转喷射工艺，碱金属杂质脱除效果与除气效率不弱于传统通Cl₂工艺，夹杂物去除效果甚至更优，全程无游离Cl₂废气排放，可满足现代压铸行业环保合规要求[[S44]]。

## 与相关工艺的对比
### 与旋转除气对比
旋转除气通过高速转动的转子将通入的惰性气体（Ar或N₂）剪切为微米级细小气泡，大幅提升气泡总表面积与在铝液内的停留时间，除气效率远高于传统固体除气剂。主流工业级SNIF、ALPUR旋转除气设备转子转速可达300~900 rpm，可将铝液氢含量稳定控制在0.08 cc/100g以下，适配连续在线除气的大规模生产场景[[S44,S36,S47]]。

固体除气剂的优势在于操作简便，无需配套复杂设备，适配小规模间歇式压铸熔炼场景；旋转除气则设备投入及运维成本较高[[S9,S22]]。

### 与覆盖剂、变质剂的区分
四种材料功能差异明确[[S21,S22,S32,S9]]：

| 材料类型 | 核心功能 | 除气能力 |
|---------|---------|---------|
| 固体除气剂 | 以化学反应产气泡实现除气 | 强 |
| 覆盖剂 | 隔绝铝液与大气接触，防氧化烧损 | 微弱附带 |
| 变质剂 | 改变铝硅等合金晶体生长方式细化组织 | 无 |
| 旋转除气（惰性气体） | 物理剪切分散气泡除气 | 强（无杂质引入） |

在实际生产中，除渣剂和除气剂都兼有除渣和除气作用，只是各有所侧重。当炉料污染严重、熔体氢含量高时，应以除气剂为主；当非金属夹渣物含量高时，应以除渣剂为主；两者缺陷均严重时，可搭配使用[[S21,S31]]。
