## 概述

起模斜度（亦称拔模斜度、脱模斜度，英文对应 draft angle）是在压铸模具型壁出模或抽拔方向上设置的倾斜角度，目的为使压铸件顺利脱离模具型腔以及从型芯上脱出，减少脱模过程产生的摩擦阻力，保护模具成型表面，延长模具使用寿命。斜度不计入铸件尺寸公差，当图样未指定端点尺寸时，未指定端尺寸按斜度值独立核算[[S28,S42,S13,S4]]。

起模斜度应按压铸件脱模时金属收缩的方向来确定：熔融金属冷却凝固后体积收缩，铸件会紧紧包住型芯及型腔凸出的成型部分（内表面），而趋向远离型腔壁面（外表面）。因此必须就外表面、内表面、铸孔以及非装配短壁等不同区域合理配置斜度，以保证压铸件顺利顶出，避免拉伤、粘模及变形等缺陷[[S13,S23,S29]]。

## 工艺作用

合理设置起模斜度是保证压铸生产稳定与铸件质量的基础，其核心工艺作用体现在：

- **降低脱模阻力**：斜度可直接减小铸件与型壁、型芯之间的摩擦阻力，降低脱模时所需的顶出力，避免抽芯、顶出过程中出现卡滞[[S13,S21,S2]]。
- **延长模具寿命**：降低摩擦同时减少了成型零件表面的磨损速率，有效保护型腔、型芯等关键成型表面，延长模具服役周期[[S13,S21,S2]]。
- **减少粘模拉伤**：当起模斜度设置过小甚至为负斜度时，会引起高温熔融合金与模壁的粘附作用加剧，导致铸件表面被撕破而出现多肉或缺肉的粘模拉伤痕迹[[S8,S35,S25]]。适度增大的斜度可显著缓解这种倾向。
- **防止机械拉伤与变形**：无斜度或斜度过小将引发沿开模方向具有一定深度的机械拉伤，严重时可伴随掉肉、局部撕裂[[S8,S11,S25]]；同时过大的脱模阻力还会造成铸件顶出时受力不均，导致翘曲、扭曲等整体或局部变形[[S20]]。

## 内外拔模斜度分类与参数配比

根据压铸件收缩行为，起模斜度分为外拔模斜度（型腔侧，对应铸件外表面）与内拔模斜度（型芯侧，对应铸件内表面）。因铸件冷却收缩后紧紧包住型芯，内表面脱模难度远大于外表面，故内表面斜度需取较大值[[S13,S10,S28,S6,S4]]。

常规配比规则为：

- 外表面斜度 θw 为基准值；
- 内表面（普通）斜度 θn 为 θw 的 **2 倍**，即 θn = 2θw[[S13,S23]]；
- 铸出孔等深腔型芯对应的内表面斜度 θg 需取 θn 的 **3 倍及以上**，即 θg ≥ 3θn[[S13,S7,S23]]；
- 非装配短壁的斜度 θc 需取 θn 的 **2 倍及以上**，即 θc ≥ 2θn[[S13,S7,S23]]。

上述关系反映了不同区域金属收缩时包紧力的差异。随着铸件壁厚增大，合金对型芯的包紧力进一步提升，所需的起模斜度也需相应增大[[S13,S23]]。

## 典型参数与推荐值

### 常规场景推荐值（基于 GB/T 8844 系列）

根据国家标准体系，成型部位内侧壁的未注拔模斜度应按合金种类和脱模方向高度分档取值，外侧壁斜度取对应内侧壁数值的 1/2[[S10,S28]]。以下为压铸模具标准中给出的内侧壁拔模斜度常用推荐值[[S17]]：

| 拔模高度/mm | 锌合金 | 镁合金 | 铝合金 |
|---|---|---|---|
| ≤3 | 2°30' | 3°30' | 4° |
| >3～6 | 2° | 3° | 3°30' |
| >6～10 | 1°30' | 2° | 2°30' |
| >10～18 | 1°15' | 1°45' | 2° |
| >18～30 | 1° | 1°30' | 1°45' |
| >30～50 | 0°45' | 1° | 1°15' |
| >50～80 | 0°30' | 0°45' | 0° |
| >80～120 | 0°30' | 0°45' | 0°45' |
| >120～180 | 0°20' | 0°30' | 0°30' |
| >180～250 | 0°15' | 0°30' | 0°30' |

注：上表数据引自行业手册中与 GB/T 8844 体系配套的常规压铸场景推荐值[[S17]]。

### 高精度场景推荐值

在高精度精密公差条件下，压铸模具设计师手册中建议的拔模斜度取值如下[[S16]]：

| 拔模高度/mm | 锌合金 | 镁合金 | 铝合金 |
|---|---|---|---|
| ≤3 | 3° | 4° | 5°30' |
| >3～6 | 2°30' | 3°30' | 4°30' |
| >6～10 | 2° | 3° | 3°30' |
| >10～18 | 1°30' | 2°15' | 2°30' |
| >18～30 | 1°15' | 1°30' | 1°45' |
| >30～50 | 1° | 1°15' | 1°30' |
| >50～80 | 0°45' | 1° | 1°15' |
| >80～120 | 0°30' | 0°45' | 1° |
| >120～180 | 0°30' | 0°30' | 0°45' |
| >180～250 | 0°15' | 0°30' | 0°30' |

### 最小斜度参考值

基于 NADCA 北美压铸协会及行业资料，不同压铸合金的最小起模斜度基准值如下[[S32,S4,S29,S3]]：

| 合金 | 最小起模斜度 | 斜度比 |
|---|---|---|
| 锌合金 | 0.3° | 1:200 |
| 铝合金 | 0.6° | 1:100 |
| 镁合金 | 0.6° | 1:100 |
| 黄铜/青铜 | 0.7° | 1:80 |

常规生产中外壁起模斜度不小于 1°，内壁起模斜度不小于 2°[[S29]]。

### 特殊结构与其他约束

- **表面纹理处理**：铸件表面带有细纹纹面时斜度不小于 1.5°，粗纹纹面不小于 2°，皮纹处理需额外加大斜度以避免拉伤粘模[[S38,S42,S26]]。
- **高光洁度表面**：模具成型表面抛光至 Ra0.1 µm 及以下时，起模斜度可适当减小[[S26]]。
- **文字符号**：铸件表面文字、符号的起模斜度推荐取 10°～15°[[S10,S28,S33,S9]]。

## 设计标注与图纸表示

起模斜度在压铸图纸中有两种常规标注方式：一是直接标注倾斜角度 α，二是标注斜度对应的单侧偏移尺寸 a[[S31]]。

当斜度方向与尺寸公差相关时，需注意以下原则：

- 起模斜度不计入 GB/T 6414 等标准规定的铸件尺寸公差范围内[[S42,S13,S15,S38]]。
- 当尺寸指定在斜度方向上的某一个端点后，公差仅对该端点生效，另一端点尺寸由斜度值独立计算[[S42,S13,S15]]。
- 若两处端点均需公差限制，则应在图样上分别标注各自的公差值[[S23]]。

## 与分型面、顶出机构及抽芯的耦合关系

起模斜度的设置与分型面位置、顶出系统及侧向抽芯机构密切关联，需在模具设计阶段统筹优化。

- **分型面确定时的同步设计**：分型面位置一旦确定，应立即明确对应的拔模方向与斜度方案[[S12,S14]]。若选定分型面后铸件对定模侧的包紧力大于动模侧，可通过加大定模侧铸件的起模斜度进行平衡，防止开模时铸件滞留在定模[[S12,S14,S38]]。
- **与顶出机构的匹配**：适当增大起模斜度可降低脱模阻力，减少脱模时推杆的受力，从而避免推杆损伤或顶出印痕超标，优化顶出系统的可靠性[[S29]]。
- **与侧向抽芯的协同设计**：侧向活动型芯对应成型面的斜度 α 直接影响抽芯力——斜度越大，压铸件对侧型芯的正压力和摩擦阻力越小，所需起始抽芯力越低[[S40]]。此外，通过合理增大侧向孔或凹坑对应壁面的起模斜度，保证侧向特征大端尺寸 B ≥ 小端尺寸 A + (0.1~0.2) mm，有时可将侧孔改由动定模直接成型，完全省去侧向抽芯机构[[S27,S40,S38]]。

## 极限与局限

起模斜度取值须在脱模可行性与铸件质量、经济性之间取得平衡，过小与过大均会带来负面问题。

- **斜度过小或无斜度**：会导致脱模摩擦阻力大幅提升，引发表面机械拉伤、粘模拉伤缺陷，同时模具成型面磨损速率显著加快，缩短模具服役寿命[[S29,S13,S11,S25]]。顶出力不均时还易引发铸件整体或局部变形（翘曲、扭曲）[[S20]]。
- **斜度过大**：会造成不必要的金属材料浪费，使需机加工的表面加工余量超标，加工时容易穿透压铸件表层的致密激冷层而暴露内部粗晶组织[[S29,S4]]。过大的斜度还会直接导致铸件不同区域壁厚分布不均匀，增大内部产生缩孔、变形等缺陷的风险[[S29]]。

因此，设计时应遵循“在技术条件容许范围内尽可能取较大斜度”的基本策略，同时兼顾壁厚均匀性、后加工余量以及表面质量要求，不得盲目取上限。
