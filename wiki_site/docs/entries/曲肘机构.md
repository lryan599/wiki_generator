## 概述

曲肘机构（常称曲肘锁模机构、肘杆合模机构，英文常用 Toggle Mechanism / Toggle Clamping Mechanism）属于液压机械式合模机构，由液压缸与曲肘连杆系统组成，是冷室、热室压铸机通用的核心锁模部件，两类压铸机仅压射系统存在差异，锁模部分可通用[[S28,S18,S7,S10]]。其核心功能是完成开模、合模动作，并在合模到位后利用曲肘机构的死点位置实现自锁锁模，保障金属液高速注入型腔时模具不被胀开溢料。

压铸机的合模机构主要有两种形式：液压合模机构（全液压直压式）和机械合模机构。曲肘合模机构是机械合模机构中应用最广泛的一种，已成为国产压铸机的主流选择[[S6,S2,S33]]。

## 分类与应用场景

### 按结构形式分类

| 分类 | 增力倍数 | 承载特性 | 适用机型 | 说明 |
|------|----------|----------|----------|------|
| 单曲肘合模机构 | 约10多倍 | 单臂受力，易出现模板偏载 | 小型压铸机 | 结构紧凑、制造容易，但承受偏载能力差，仅适配模板面积较小的小型机[[S8,S27,S21]] |
| 双曲肘合模机构 | 20～40倍 | 对称布置，承受偏载能力强 | 中型至大型压铸机 | 增力作用和合模力较大，得益于对称受力[[S8,S27]] |

其中，双曲肘内翻式锁模机构在开模时曲肘相对于轴线向内收拢，结构紧凑，广泛应用于中小吨位压铸机，能有效减小设备体积、降低生产成本[[S24,S14]]。

## 典型结构与部件功能

典型双曲肘五铰点连杆机构可视为两套对称布置的单曲肘连杆机构组合而成，每套单曲肘又可视为由两组曲柄连杆滑块机构组成：一组由大曲肘杆、大连杆与动模板构成；另一组由小曲肘、小连杆与十字头、合模活塞杆构成，大曲肘与小曲肘刚性连接，以同一角速度绕尾板上的后支铰旋转[[S9,S24,S37]]。

核心部件及功能如下：

- **大曲肘杆＋大连杆**：作为第一组曲柄连杆构件，将曲肘的转动转化为动模板的直线进给运动[[S9,S24]]。
- **小曲肘＋小连杆**：与十字头、合模活塞杆配合构成第二组曲柄连杆机构，传递液压动力带动整套曲肘系统运动[[S9,S24]]。
- **十字头**：将合模液压缸的直线推力同步传递至双侧对称的小连杆，保证两侧曲肘的运动同步性，避免单侧受力不均导致的卡死故障[[S9]]。
- **调模装置**：通常由调模液压马达、调模大齿轮、拉杆螺母组成，通过同步旋动四根拉杆端部的螺母，调整尾板与定模板之间的间距，适配不同厚度的压铸模具，同时完成锁模力的校准。部分机型带自动调模功能，可快速完成参数设置[[S25,S42]]。
- **动模板**：与连杆铰接，作为模具动模的安装载体，在曲肘驱动下沿拉杆做往复直线运动，配套滑脚结构降低摩擦阻力[[S42]]。

## 工作原理与自锁机制

曲肘合模机构由三块座板与四根导柱串联组成，中间为动模座板。合模时，液压合模缸推动合模活塞带动连杆及三角形铰链摆动，驱动动模前移。当模具完全闭合后，曲肘连杆系统中a、b、c三点恰好共处于同一条直线上，此位置称为"死点"。机构到达死点位置后，继续受驱动使合模架发生弹性变形，四根拉杆受拉伸长、肘杆受压，该弹性变形对应的弹性反力形成稳定锁模力。即使切断合模端液压动力，受铰链间摩擦力及过点后机构反力特性约束，机构仍不会发生解锁，实现无动力自锁[[S24,S40,S33,S7]]。

从机构受力角度分析，这是一个力的封闭系统：液压系统通过锁模活塞杆将力传递给十字头，再经过机铰对力的放大传递给动模板及尾板，最后由头板与格林柱将力返回给尾板，形成封闭力链[[S9]]。

开模时动作反向执行：压力油改变方向，活塞回退，曲肘被拉动"缩回"，动模板后移，模具打开[[S32]]。

## 运动学特性：变速开合模

曲肘锁模机构自带动模板变速特性，开合模过程中实现"慢→快→慢"的速度变化[[S28,S14,S24,S33]]：

- **合模阶段**：动模板速度从零快速升至峰值，随后逐步降低，接近死点位置时速度降至极低至零，实现两半模具低速平稳贴合，避免撞击。
- **开模阶段**：初段速度缓慢（利于抽芯和预顶出），随后升至峰值，临近完全开模时再次降速，避免机械冲击。

这一特性天然适配压铸工艺中对抽芯、铸件平稳顶出的需求，且整个生产周期更短，易于实现高速循环[[S28,S14,S24,S40]]。

## 核心参数与力放大特性

### 标准设计参数

曲肘系统设计参数直接影响锁模力、开模力和机构外形尺寸。现代压铸机的肘杆布置通常与水平面呈4°～6°倾斜角，以在不明显增大合模缸座尺寸的前提下获得最大动模座板行程。各核心参数取值区间如下[[S5]]：

| 参数 | 常规取值区间 | 说明 |
|------|-------------|------|
| 肘杆倾斜角 | 4°～6° | 与水平面的倾斜角，兼顾行程与外形尺寸 |
| 支撑肘杆总长度与动模板行程比 | 1:(0.65～0.70) | 决定行程与机构尺寸关系 |
| 肘杆长度a/b比值 | 0.75～0.85 | 两段肘杆长度比例 |
| 中间杆与垂直面夹角α（带防过转挡块） | 15°～20° | 保证转轴耳结构强度 |
| 中间杆与垂直面夹角α（无挡块无合模缸限位） | 0°～5° | 用于无限制构型 |
| 夹角β | 接近90° | 合理参数 |

设计规律：减小肘杆长度a和α角，以及增大β角，均会增大锁模力；增大肘杆长度a和减小α角，则会加大开模行程[[S5,S26]]。

### 力放大比

曲肘连杆系统可将合模缸推力放大16～26倍。增力倍数的大小与肘杆机构的形式、各肘杆的尺寸及相对位置有关[[S40,S32,S42,S24]]。按约束构型进一步细分[[S5,S33]]：

| 机构构型 | 力放大系数（推力系数） |
|----------|----------------------|
| 配置防过转挡块的肘杆系统 | 16～20 |
| 配置合模缸活塞杆限位块的系统 | 22～26 |

不计铰链摩擦时，理想力放大比 η₀ 的表达式为[[S16]]：

η₀ = (L₂,₃ / L₃,₄) · [sin(α+φ+β) / sin(β+γ)] · [cosγ / cosα]

计入铰链摩擦后，实际力放大比 η 需包含摩擦修正项，涉及铰链摩擦系数f、各铰链轴半径r和r₁等参数[[S29,S23,S16]]：

η = (L₂,₃/L₃,₄) · { sin(α+φ+β) + f·r₁/L₂,₃·[1-2·(L₂,₃/L₁,₂)·cos(α+φ+β)] } / { sin(β+γ) + f·r/L₃,₄·[1+(1+2·L₃,₄/L₄,₅)·cos(β+γ)] } · cosγ/cosα

在极限位置（β=0、γ=0，即死点位置）时，无摩擦理想放大比 η₀ 趋近于无穷大，但计入铰链摩擦后放大比为有限值[[S16]]。

### 开模力特性

最小开模力为合模缸开模推力的0.65～0.8倍，该最小值出现在开模总行程的30%～50%区间内[[S5]]。

### 锁模力计算

压铸机所需锁模力的通用工程计算为[[S24,S15]]：

Fₛᵤₒ ≥ K · p · A     （安全系数K一般取1.2～1.3）

其中p为金属液最大胀型比压，A为铸件与浇注系统的总投影面积。

从弹性系统角度，曲肘机构产生的实际锁模力由合模架所有弹性元件的总变形决定，总变形包含拉杆伸长、型板弯曲、肘杆压缩、模具压缩等分量，符合虎克定律[[S20,S11]]：

Pₕₑ = Cₜₒₜ · yₕₑ

其中Cₜₒₜ为弹性系统总刚度，yₕₑ为系统总弹性变形量。

## 与液压直压锁模机构的对比

| 对比维度 | 曲肘锁模机构 | 液压直压锁模机构 |
|----------|-------------|-----------------|
| 增力方式 | 机械连杆放大16～26倍 | 液压缸直接施力，无放大 |
| 液压缸尺寸 | 直径小，所需油压低 | 直径大 |
| 自锁能力 | 死点自锁，切断动力仍保持锁模 | 需持续供油维持锁模力 |
| 刚性 | 高，过载条件下形变量≤直压式的1/3 | 较低，易退让 |
| 开合模速度 | 自带动模板"慢→快→慢"变速特性 | 速度由液压泵流量/压力决定 |
| 模具厚度适配 | 需调模装置调整行程，操作较繁琐 | 无需调整合模缸座位置，直接适配 |
| 控制系统 | 简单 | 较简单 |
| 可靠性 | 刚性大、锁模可靠 | 刚性差，胀模时可能喷料 |
| 维护 | 维修方便，但对肘杆、销轴精度要求高 | 结构简单 |

**曲肘机构的优势**：力放大显著（可使合模缸推力的16～26倍转化为锁模力）、具备自锁能力、刚性高（相同过载下总形变仅为直压式的1/3以内）、自带动模板变速特性适配压铸工艺要求、控制系统简单、维护难度较低[[S28,S6,S2,S12,S36,S14]]。

**曲肘机构的局限**：不同厚度模具需要调整行程较为麻烦；运行过程中受模具热膨胀影响，合模框架的预应力会变化，易引发拉杆过载；肘杆、销轴等核心传动部件加工精度要求极高[[S28,S6,S2]]。

液压直压式无需调模行程，生产过程中可直接稳定维持设定锁模力，结构原理简单操作便捷[[S6,S2]]。

## 常见问题与缺陷关联

### 机构磨损与失效

曲肘锁模机构核心部件销轴和衬套的常见失效原因包括：润滑不充分、摩擦副材料选型不合理、长期高载荷运行磨损。严重磨损后会直接导致卡滞，设备无法正常锁模[[S25,S17]]。

### 同步性偏差与模板平行度

曲肘两侧传动部件磨损量不一致、不同步运行时会出现合模同步性偏差，直接导致动定模板平行度超标，锁模力分布不均[[S25,S34]]。现行GB/T 21269-2023中明确规定动模安装板与定模安装板工作面的平行度需符合对应精度要求[[S4]]。

### 缺陷关联

销轴衬套磨损、同步性偏差、锁模不平行等故障会直接关联铸件质量缺陷[[S13,S41]]：
- 锁模力局部不足 → 分型面溢出金属形成飞边/毛刺
- 模板错位 → 铸件错型
- 严重时 → 铸件尺寸超差报废

### 调模精度保障

现代压铸机采用伺服调模，通过伺服电机驱动同步调节四根导柱螺母位置，调整过程中不会破坏模座板之间的平行度。高精度机型的点动调模精度可达0.04～0.06 mm，锁模力控制误差可低于5%～10%[[S25]]。
