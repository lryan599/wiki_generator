## 概述
压射压力/位移/速度-时间曲线是反映压铸过程中压射压力、压射冲头位移、压射冲头速度与时间关系的一组核心过程监控曲线[[S32]]。在国内压铸实践中，通常将压射工艺曲线直观描述为三条主线：蓝色位置线记录冲头全行程运行距离（mm），红色速度线记录各阶段速度（m/s），绿色压力线记录各阶段压力（bar）[[S30]]。该曲线通过实时复现单次压射循环中各工艺参数的实际达成情况，支撑工艺一致性校验与压铸缺陷溯源，在质量控制中用于稳定压铸件批量生产的工艺一致性[[S32]]。

在英文压铸文献中，shot profile 是工艺监控存档场景下的通用表述[[S5]]；injection pressure curve 多指单独分析压射全流程的压力特征；displacement-time curve 则用于表征压射冲头位移随时间的运动规律[[S5]]。

## 曲线构成与典型阶段
### 冷室压铸压射阶段划分
国内长期压铸实践表明，冷室压铸压射过程主流采用三级（三阶段）划分模式。部分应用在高速压射阶段后额外增加增压压实阶段，形成四阶段划分体系[[S19,S11]]。标准级冷室压铸四阶段压射进程如表所示。

| 阶段 | 进程描述 |
|------|----------|
| 第Ⅰ阶段 | 冲头从起始位置运行至越过浇料口，低压低速（v₁），防止金属液从浇料口溢出，有利于气体排出。压力主要用于克服系统摩擦阻力 [[S2]] |
| 第Ⅱ阶段 | 冲头越过浇料口至金属液充满至内浇口处，压力增大，速度加快（v₂），避免金属液提前进入型腔减少卷气 [[S2]] |
| 第Ⅲ阶段 | 金属液抵达内浇口至型腔完全充满，压力进一步升高，速度达最大值（v₃），用于克服内浇口流动阻力完成高速充型 [[S2]] |
| 第Ⅳ阶段 | 充型结束后增压压实，冲头速度迅速降至零，压力升至全过程最高值，对凝固中金属液加压补缩 [[S2]] |

### 热室压铸压射阶段划分
热室压铸普遍采用两阶段压射模式：阶段Ⅰ为升液阶段，阶段Ⅱ为填充阶段，图示直观呈现了两阶段的时间特征和压力速度对应变化[[S19,S35]]。热室压铸通常不使用增压阶段，充型结束后压力达到系统最大值[[S34]]。

### 现代多点压射模式
现代全电动压铸机支持自定义多点压射参数设置，速度设定点可达10个以上，压力设定点也可达10个或更多，可根据铸件需求灵活调整压射曲线形态以减少卷气[[S19,S11]]。某些压铸机还具备压射冲头运动优化程序，根据压室参数、金属液充满度等计算最佳压射模式[[S19]]。

### 位移曲线上的关键节点

| 特征点 | 定义与说明 |
|--------|------------|
| 速度切换点（高速切换点） | 慢压射结束、快压射启动的冲头位置节点。计算公式：S₁ = 空打行程 - 高速行程 - 料柄厚度 - (10~15) mm [[S30]] |
| 增压起点 | 快压射充型后期触发增压启动的冲头位置节点，典型工况标准增压启动位置为 850 mm [[S30]] |
| 位移曲线停止点 | 冲头运动完全终止的位置节点，对应 GB/T 21269-2023 建压时间计算的起点 [[S32]] |

### 压力曲线上的关键节点

| 特征点 | 定义与说明 |
|--------|------------|
| 压力开始上升时刻 | 压射缸压力从慢压射阶段低摩擦阻力水平开始抬升、向充型压力过渡的起始节点，对应压射第二阶段的启动点 [[S15]] |
| 峰值压力 | 增压阶段压力曲线达到的最高稳态压力值，冷室压铸典型增压比压可达 945 bar [[S30]]，正常增压比压取值范围 50~300 MPa [[S29]] |
| 建压时间 | 冲头运动停止点至以增压压力稳态值 90% 作水平线与压力曲线交点的时间间隔 [[S32]]，先进压铸机可达 10 ms 以下 [[S34]] |

### 异常特征点与关联缺陷
各段异常可能关联的铸件缺陷如下表所示[[S22,S1,S30]]。

| 异常工况 | 曲线特征 | 关联缺陷 |
|----------|----------|----------|
| 速度切换点过早 | 快压射在冲头过早位置启动 | 内浇口前后区域乱流卷气，模具冲蚀加剧寿命缩短，最终产生气孔 [[S22]] |
| 速度切换点过晚 | 快压射在冲头过晚位置启动 | 合金液温度损失过大导致流动性下降，产生冷隔、流痕、气孔、抛丸起皮 [[S22]] |
| 慢压射速度过慢 | 速度曲线远低于设定下限 | 容杯后部气体无法顺利排出，合金液温度损失快，产生欠铸、冷隔、气孔 [[S30]] |
| 冲头卡滞 | 慢压射启动阶段速度曲线出现非预期波动 | 铝液激荡浪卷裹入大量气体，铸件内部气孔率显著上升 [[S22]] |
| 增压起点过早 | 充型未完全结束时增压提前达到峰值 | 增压压力与涨型压力叠加，存在卡冲头或飞料安全风险，在分型面产生大量飞边 [[S22]] |
| 增压起点延迟 | 冲头停止运动后延迟一段时间才完成建压 | 型腔内大部分合金已开始凝固，增压压力无法传递到凝固前沿，产生严重缩松、缩孔缺陷 [[S1]] |

## 与工艺参数的关系
### 压力-速度对应关系
压射压力与压射速度在理论上呈平方正比关系，相同工艺条件下特定速度对应特定压力[[S34]]。

### 压射系统有效压力线
压铸机压射系统有效压力线满足经验公式[[S4]]：
\[ p_s = p_A\left(1 - \frac{v_p^2}{v_d^2}\right) \]
式中，p_s 为压射缸内有效压力，p_A 为储能器压力，v_p 为实测冲头速度，v_d 为最大空压射速度。

### 内浇口速度与冲头速度关系
内浇口速度 v_gate 与压射冲头速度 v_plunger 满足关系式[[S31]]：
\[ v_{gate} = v_{plunger} \times \frac{A_{plunger}}{A_{gate}} \]
其中 A_plunger 为冲头截面积，A_gate 为内浇口截面积。

### 高速行程与产品设计的关系
高速行程 S₂ 的计算公式为[[S30]]：
\[ S_2 = \frac{\text{渣包重量} + \text{产品重量}}{\text{合金比重} \times \text{压射头截面积}} \]

### 建压时间对增压补缩效果的影响
建压时间必须短于型腔内合金液从充型结束到开始凝固的时间间隔，以保证增压补缩有效[[S34]]。先进冷室压铸机的建压时间可低至 10 ms 以下[[S34]]。

## 测量方法与数据采集
### 传感器选型与安装要求
压射压力测量常用传感器类型包括贴应变片的薄膜液压传感器、半导体式、电导式、电感式压力传感器[[S24,S33]]。针对液压压铸机，常用量程范围为 0~5000 lb/in²，可完整捕捉压射过程的冲击波峰值信号[[S33]]。

压射位移测量最通用的选型为挠性线-鼓轮型位移传感器，也可选用光栅式、磁阻式、光电式位移传感器[[S24,S12,S23]]。该类传感器通过复位弹簧将拉线与冲头刚性杆相连，输出电压信号与柱塞位置线性对应[[S24]]。

### 国家标准精度要求
GB/T 37365-2019 明确规定了压射曲线独立测量系统的硬件参数要求[[S14]]。

| 项目 | 要求 |
|------|------|
| 位移传感器量程 | 不低于压铸机压射行程 |
| 位移传感器分辨率 | ≥ 0.1 mm |
| 位移传感器精度 | ±0.5% |
| 压力传感器量程 | 60 MPa |
| 压力传感器精度 | ±0.1% |
| 无纸记录仪记录时标 | 0.001 s |
| 单次连续记录时间 | ≥ 20 s |

测量系统可完全不依托压铸机原有电控系统独立完成压射全流程曲线的采集、存储和输出[[S14]]。

### 传感器安装规范
压力传感器应尽可能靠近压射缸的取压口/溢流塞孔，使用快换接头安装到液压回路的测量孔中，以减少长油路带来的压力信号滞后误差[[S33,S12]]。位移传感器需固定在与压射缸外壳相连的支撑板上，远离高温区域，耐受温度不超过 200°C，避免直接接触压射室高温区域引发测量漂移[[S12]]。

### 采样频率要求
压射曲线采集系统采样率最低要求不低于 200 Hz[[S20]]；针对动态参数高精度分析的专业级系统，采样率可达 2000 Hz（每 0.5 ms 采集一个数据点），可完整捕捉增压建压阶段的毫秒级快速变化过程[[S25]]。

### 曲线记录设备功能
主流压铸机标配的压射曲线记录系统功能覆盖：压射全流程曲线单屏/多段显示、与基准优良品曲线重叠对比、存储 50~140 个压射循环的完整参数数据、超限自动报警、故障事件带时间戳记录存储、支持参数历史回溯与导出[[S10,S21]]。

## 实际应用与典型值
### 工艺调试场景
压射曲线在工艺调试中可直接确认增压起点位置。研究表明，当铝液填充型腔的 55%~58%（含溢流槽重量）时启动增压，可大幅改善铸件缩松缩孔缺陷[[S6]]。若增压启动过早会导致模具跑水、增压行程耗尽无法有效补缩；增压启动过晚则导致远端铸件补缩不足，残留气孔缺陷[[S6]]。

### 批量生产质量监控场景
系统实时对比当前压射曲线与预设合格基准曲线的偏差。当压射速度、增压压力、建压时间任意参数超出允许公差范围时自动分级报警，统计每批次生产中超限产品占比，辅助追溯质量异常批次[[S13,S17]]。

### 设备故障诊断场景
通过曲线特征可快速定位故障点[[S37,S1]]：
- 压力曲线偏高而速度曲线偏低 → 冲头卡滞、浇道堵塞或液压管路阻尼异常；
- 压力曲线偏低速度无法上升 → 系统泄压或压力设置不足；
- 速度曲线低速段出现不规则波动 → 冲头卡滞，易引发容杯内金属液卷裹气体产生铸件气孔。

### 典型合格曲线参数参考值
典型合格冷室铝合金压铸压射曲线参数参考值范围如下[[S30,S27]]：
- 慢压射阶段速度：0.15~0.35 m/s，曲线平滑无卡滞波动；
- 高速填充阶段速度：约 4~4.5 m/s；
- 高速切换点位置偏差：控制在 -10~-15 mm 以内；
- 先进压铸机增压建压时间：≤ 10 ms；
- 铝合金常规工艺增压比压：约 945 bar。

## 限制与注意事项
压射曲线在解读和实际应用中存在以下局限性：
1. **传感器信号滞后**：间接采集的液压压力信号受油路阻尼影响存在时间滞后，无法完全等同于直接作用在金属液上的实时压射比压[[S12]]。
2. **位移传感器振动干扰**：位移传感器安装刚性不足时，会引入额外振动干扰，造成位移曲线出现伪波动，导致误判[[S12]]。
3. **增压曲线解读偏差**：将增压启动后压力急剧快速上升的异常特征直接判定为合格曲线，忽略了增压位置不合理导致的建压过快缺陷，误将增压过早/过晚的异常工艺判定为合格[[S6,S1]]。判断增压效果时，若增压过程中冲头存在一定移动，压力上升应较缓慢；若压力急剧上升并马上下降，往往意味着补缩不足[[S6]]。
