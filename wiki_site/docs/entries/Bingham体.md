## 概述

Bingham体（宾汉体）是流变学中一种经典的理想力学元件模型，其核心特征是带有明确的屈服点。该模型由美国流变学家宾汉（Bingham E.C.）早年研究涂料流变性能时发现并命名，因此得名[[S15]]。与其他流变基础元件相比，Bingham体能够同时表现弹性、塑性与黏性三种效应：当施加应力低于屈服值时，材料表现为完全弹性（类固体）；一旦应力超过屈服值，即开始发生不可逆的黏塑性流动，撤去外力后会留下永久变形[[S12,S20]]。Bingham体在铸造工程领域具有特殊地位——它不仅是半固态合金在高固相分数阶段的典型流变描述模型，也是铸造合金准固相区五元件流变模型[H]‑[H|N]‑[N|S]中负责刻画黏塑性变形的核心子元件，对热裂预测有着关键意义[[S7,S13]]。

## 定义与流变学行为

### 元件结构与机械模型

Bingham体的机械等效模型由三个基础力学元件组合而成[[S5]]：

- **弹簧（Hooke体）**：表征屈服前的弹性响应；
- **阻尼器（Newton体）**：表征屈服后的黏性流动行为；
- **圣维南摩擦滑块（St.Venant体）**：模拟临界屈服应力阈值，仅当应力超过屈服值时元件才开始滑移。

三者的标准组合方式是一根弹簧与“阻尼器串联圣维南滑块”的并联结构再串联而成[[S12,S5]]。

![宾汉体的标准流变机械模型](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/2fff2460-2416-4456-b378-71f0f8e5e597/resource)

### 屈服前后的力学行为分界

Bingham体的流动行为以屈服应力τₛ为明确界限，呈现两阶段截然不同的特征：

| 应力状态 | 行为特征 | 变形性质 |
|-----------|----------|----------|
| τ ≤ τₛ | 不发生宏观流动，变形规律完全等同于Hooke弹性体，材料表现出类固体刚性/弹性行为，屈服前的半固态合金浆料甚至可直接夹持搬运 | 可逆弹性变形 |
| τ > τₛ | 材料开始发生黏塑性流动，流动行为与牛顿流体相似，撤去外力后留下不可逆残余变形 | 不可逆黏塑性流动 |

屈服前的弹性行为使Bingham体在应力低于阈值时保持形状不变，而屈服后的黏塑性流动则使其能够在外力下产生永久塑性应变[[S12,S21]]。这双重特性使Bingham体成为连接固体力学与流体力学的重要桥接模型。

### 稳态本构方程

当作用应力超过屈服值后，Bingham体在不变应力下的稳态流动满足如下线性方程[[S12,S21]]：

**τ = τₛ + η · γ̇**

其中：
- τ：剪切应力（Pa）
- τₛ：屈服应力（Pa），流动曲线在τ轴上的截距
- η：塑性黏度（Pa·s），流动曲线的斜率
- γ̇：剪切速率（s⁻¹）

值得注意的是，Bingham体的塑性黏度η在其屈服后的整个流动范围内保持恒定，不随剪切速率变化——这一点使其区别于伪塑性体（表观黏度随剪切速率增大而下降）和膨胀体（表观黏度随剪切速率增大而上升）[[S20]]。

### 表观黏度与剪切变稀

Bingham体在τ>τₛ区间流动时，若采用常规牛顿体黏度计测量，所得到的表观黏度ηₐ并非恒定值，而是满足[[S20]]：

**ηₐ = η + τₛ / γ̇**

在低剪切速率时，τₛ/γ̇项占主导，表观黏度很大；随着剪切速率增大，该项贡献逐渐减小，表观黏度趋近于材料固有的塑性黏度η。这一特性表明Bingham体的表观黏度随剪切速率增大而降低，属于剪切变稀（shear-thinning）行为[[S20]]。

### 蠕变与应力松弛

Bingham体同时具备两种依时性黏弹性行为[[S9,S12]]：

1. **蠕变**：在恒定应力τc>τₛ作用下，Bingham体能够像牛顿体一样持续发生黏塑性流动变形，随时间的推移应变不断累积。
2. **应力松弛**：在变形量保持不变（γ₀恒定）的条件下，Bingham体内部的应力会随时间按照指数衰减，其松弛规律表达为：
   τ = (τ₀ − τₛ) · exp(−t / θₙ) + τₛ

其中θₙ=η/G为其松弛时间参数，G为屈服前弹性体的弹性模量，η为塑性黏度[[S12]]。这表明Bingham体的应力松弛以屈服应力τₛ为极限值，应力不会衰减至零——固体骨架的屈服强度为材料保留了底限承载能力。

## 半固态合金中的宾汉特征

在一般非牛顿流体中，Bingham模型描述的是屈服后黏度恒定的理想宾汉流体，半固态金属合金在高固相率下的行为更加复杂：其屈服应力和塑性黏度并非恒定常数，而是随固相分数、温度和剪切速率动态变化[[S18,S14]]。

以半固态Sn‑15%Pb合金的变温流变试验结果为例[[S14,S11]]：

| 固相分数 fₛ 范围 | 流变类型 | 主要特征 |
|------------------|----------|----------|
| fₛ < 0.2 | 牛顿流体 | 无屈服应力，τ与γ̇正比，流变线过原点 |
| 0.25 < fₛ < 0.4 | 伪塑性体（假塑性体） | 剪切变稀，无显著屈服应力 |
| fₛ ≥ 0.4 | **Bingham体** | 出现不连续的固相网状骨架结构，屈服应力显著，流变曲线呈线性但不过原点 |

随着固相分数从0.4继续升高，合金内部的固体晶粒逐渐连成网络骨架，屈服应力随之进一步增大，流动行为越来越接近理想宾汉体[[S14]]。压铸工艺中常见的半固态区间的固相率通常在0.4以上，因此该阶段的材料行为很大程度上决定了充型能力和缺陷倾向[[S2,S18]]。

## 五元件流变模型中的角色

### 模型总览

试验和系统辨识研究证实，铝合金、624低碳合金钢、ZG270‑500、ZCuSn5Pb5Zn5铜合金及ZG310‑570等多种铸造合金在准固相区（温度处于固液相线之间的糊状区）的流变性能，均可以用统一的**H‑[H|N]‑[N|S]五元件流变模型**描述[[S7]]。该模型由三个子元件串联构成，Bingham体（代号[N|S]）即为其中的第三元件，负责表征材料的黏塑性变形：

![铸造合金准固相区五元件流变模型，其中Bingham体作为黏塑性子元件](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/c0cae7ab-89a7-406e-bb2c-49c92c080e0d/resource)

模型三个子元件的分工如下[[S7]]：

- **Hooke体 [H]**：独立的串联弹簧元件，模拟准固相区金属的瞬时弹性响应；
- **Kelvin体 [H|N]**：弹簧与阻尼器并联而成的子模型，模拟黏弹性蠕变与延迟弹性效应；
- **Bingham体 [N|S]**：阻尼器与圣维南滑块串联再与弹簧并联构造的黏塑性子模型，负责屈服前弹性储能与屈服后黏塑性流动，特别是塑性应变累积与应力松弛。

三个子元件串联意味着在外部约束下宏观总应变为三者应变之和，而各元件承受的内应力相等。凝固进程中温度逐步降低、固相骨架逐渐建立，Bingham体的硬化趋势与塑性应变集中区域的演变直接影响铸件的热裂萌芽[[S13]]。

### 塑性应变累积与热裂判据

当铸件中存在受阻热节（即局部被迫滞后凝固且收缩受约束的区域），该区域在准固相区基本处于Bingham体状态。此时，Bingham体的应力松弛特性发挥双重作用[[S13]]：

- 凝固时间越长，应力松弛越充分，最终残存的等效应力越低；但同一过程中Bingham体的黏塑性应变却在不断累积增大。
- 当黏塑性应变ε_b超过该温度条件下Bingham体的**临界塑性储备值ε_bcri**时，材料无法继续容纳变形，发生热裂。

据此建立的热裂判据式为[[S13]]：

- ε_b / ε_bcri > 1 → **产生热裂**
- ε_b / ε_bcri < 1 → **不产生热裂**

这一判据涵盖了热裂发生的两个必要条件：一是Bingham体已屈服（应力超过屈服值），二是黏塑性应变量超过该温度下的极限塑性储备值[[S13]]。该模型将热裂由定性经验判别推进至本构应变判据层面，为基于准固相区热应力有限元分析的热裂预测提供了理论基础。

## 与其他流变学元件的对比

Bingham体是流变学最复杂的单一组合元件之一，与其他常用流变学元件和复合模型的属性对比见下表[[S10,S7,S12,S4]]：

| 元件/模型 | 组成方式 | 弹性 | 黏性 | 屈服应力 | 典型行为 |
|-----------|----------|------|------|----------|----------|
| **Hooke体** | 纯弹簧 | 有 | 无 | 无 | 纯弹性，τ=Gγ |
| **Newton体** | 纯阻尼器 | 无 | 有，黏度恒定 | 无 | 纯黏性，τ=η·γ̇ |
| **St.Venant体** | 纯摩擦滑块 | 无 | 无 | 有，τₛ | 应力达屈服后无限塑性流动 |
| **Kelvin‑Voigt体** | 弹簧与阻尼器并联 | 有，有延迟 | 有 | 无 | 蠕变，弹性后效 |
| **Maxwell体** | 弹簧与阻尼器串联 | 有 | 有 | 无 | 应力松弛，无屈服 |
| **Bingham体** | 弹簧+（阻尼器∥滑块） | 屈服前有 | 屈服后有，黏度恒定 | 有，τₛ | 屈服前弹性，屈服后黏塑性蠕变+应力松弛 |

Bingham体与Maxwell体相比，最显著的差别在于**屈服门槛**的存在：Maxwell体在任何应力水平下都可以发生流动并伴随应力松弛，而Bingham体只有超过屈服应力后才进入黏弹性松弛区间，其松弛后的残余应力不会低于屈服值τₛ[[S12]]。这一底限承载特征使得Bingham体在描述存在物理接触骨架的半固态材料时更为贴切。

从广义流变学角度看，Bingham体也可以视为**施韦道夫体（Schwedoff body）**的一种特殊简化形式，二者同时具备蠕变和应力松弛双重流变行为[[S9]]。但由于Bingham体构型更简明、工程参数提取更直接，在实际铸造准固相区建模中被优先采用[[S7]]。

## 工程应用与局限

基于[H]‑[H|N]‑[N|S]五元件模型的应力分析算法已在铸件热裂数值模拟中得到一维、三维轴对称以及三维有限元模拟的应用[[S13]]。该模型在受阻热节处实现的黏塑性应变集中计算，为评估大型复杂铸件（如应力框结构件）的热裂风险提供了一种超越传统补缩准则的力学判据途径[[S13]]。

与此同时，五元件模型中Bingham体的参数（屈服应力τₛ、塑性黏度η、塑性储备临界应变ε_bcri）需要通过对应合金材料在两相区范围内的力学性能试验测定。由于准固相区力学性能的直接测量难度极高，模型参数的准确获取仍是限制该本构方法在商用软件（如ProCAST、MAGMA、ABAQUS等）中标准集成的主要瓶颈[[S7]]。尽管如此，流变学模型将热裂研究与材料变形物理紧密连接，已被公认为凝固应力分析与缺陷预测中的重要研究方向[[S7,S13]]。
