## 概述

脱蜡是熔模铸造（又称失蜡铸造、精密铸造）工艺链中的核心必选工序，指将包覆在耐火型壳内的一次性蜡模通过加热完全熔出、从而在型壳内部获得与蜡模形状对应的中空型腔的过程[[S7,S16]]。

需注意区分两个易混淆的场景：

- **熔模铸造脱蜡（本词条主体）**：属于熔模铸造（失蜡铸造）工艺流程的必须环节，是获得铸型空腔的关键步骤。
- **压铸模具除蜡（辅助维护操作）**：常规压铸工艺不存在蜡模制备与脱出的工序，仅在压铸模具长期使用后因蜡基脱模剂残留积垢，需进行化学溶剂浸泡、喷砂清理、超声波清洗等辅助除蜡清理。后者并非压铸主流程环节，属于模具维护范畴[[S1,S10,S11]]。

此外，石油化工领域的溶剂脱蜡（dewaxing of oils）与本词条无关，不在讨论范围内。

## 工艺位置

脱蜡在熔模铸造全流程中处于承上启下的关键节点[[S9]]：

- **上游工序**：蜡模制作 → 蜡模组树（组装）→ 多层浸涂挂砂 → 硬化制壳（反复涂挂4~8层至型壳厚度5~10mm）[[S13]]
- **本工序**：脱蜡（熔失蜡模获得中空型壳）
- **下游工序**：型壳焙烧（850~950℃）→ 浇注金属液 → 落砂清理[[S2]]

型壳制好后通常需停放12~24h待完全硬化后方可进入脱蜡环节，脱蜡后的型壳可直接送入高温炉焙烧[[S17]]。

## 脱蜡方法分类

工业常用脱蜡方法包括以下六大类[[S17,S14]]：

| 脱蜡方法 | 适用场景 | 主要特点 |
|----------|----------|----------|
| 热水脱蜡 | 低熔点石蜡硬脂酸模料、水玻璃型壳 | 设备简单，蜡回收率≥90%，但型壳湿强度有所降低 |
| 高压蒸汽脱蜡 | 中温模料、高精度复杂铸件 | 脱蜡速度快，型壳不易开裂，蜡回收率可达95% |
| 溶剂脱蜡 | 采用三氯乙烯蒸气渗透溶解 | 蜡可回收，但需强通风，速度较慢，大模需半小时以上 |
| 远红外脱蜡 | 需内外同步升温场景 | 辐射加热使蜡模内外同步升温，蜡回收率高 |
| 微波脱蜡 | 特殊加热需求 | 加热效率高，需限制微波泄漏≤10毫瓦/厘米² |
| 闪烧脱蜡 | 小批量、低价值、无回收需求铸件 | 无需专用介质，蜡完全燃烧不可回收 |

## 主要脱蜡工艺参数

### 热水脱蜡

热水脱蜡的水温宜控制在90~98℃之间，应避免剧烈沸腾以免将槽底砂粒及脏物翻起进入型腔。型壳脱蜡时间控制在15~20min，以不超过30min为宜[[S17]]。

针对水玻璃型壳，可在脱蜡水中加入3%~5%的氯化铵或1%硼酸等酸性添加剂，起到补充硬化作用，同时降低皂化现象，脱蜡后水玻璃型壳中Na₂O含量可降至0.5%~1.0%以下[[S17,S13]]。蜡料回收率一般可达90%以上[[S17]]。

### 高压蒸汽脱蜡

高压蒸汽脱蜡是适用于中温模料的主流工艺[[S8,S17]]：

| 参数项 | 数值/要求 |
|--------|-----------|
| 蒸汽压力 | 0.4~0.8MPa，最低不小于0.5MPa |
| 升压速度 | 14s内压力应快速升至0.6MPa |
| 蒸汽温度 | 约150~160℃ |
| 脱蜡总时长 | 6~20min（中温模料15~30min） |
| 蜡回收率 | 正常工况下可达95% |

快速升压的目的在于使型壳外表面同步承受压力，以抵消蜡模受热膨胀时产生的膨胀力，从而避免型壳开裂[[S17,S5]]。在2~3min短时间内即可使模料熔化流出[[S17]]。

## 材料特性与裂壳机理

铸造蜡料的物理特性是脱蜡工艺设计的核心约束因素[[S6]]：

- **低温蜡**（如50%石蜡+50%硬脂酸）：熔点54~57℃[[S2]]
- **中温蜡**：熔点60~90℃[[S8]]

蜡料在接近熔点区间的线膨胀系数可达10⁻³K⁻¹量级，在熔点附近的膨胀量是常温下的6~7倍。而常规陶瓷型壳（主要成分为SiO₂和Al₂O₃）的线膨胀系数仅为(8~15)×10⁻⁶K⁻¹，二者膨胀率的巨大差异是脱蜡过程中型壳被胀裂的核心诱因[[S6]]。

## 缺陷与防治

脱蜡工序的主要质量风险及对应防治逻辑如下：

1. **型壳开裂**：由于蜡料热膨胀远大于型壳，升温过程中蜡模膨胀力可胀裂强度尚不高的型壳。防治方法是采用快速高压蒸汽脱蜡，14s内使釜内压力升至0.6MPa，以外部气压抵消蜡的膨胀力；或采用热水脱蜡时控制每框装入量、快速升温、缩短脱蜡时间[[S17,S5,S6]]。

2. **脱蜡不完全（夹蜡）**：型壳内残留蜡料将导致后续焙烧后型腔内积碳或浇注时产生气孔。需严格控制脱蜡温度和保温时间，残留蜡料在焙烧阶段（850~950℃）可进一步烧除[[S2]]。

3. **型壳煮烂/强度下降**：热水脱蜡时间过长（超过30min）或水温过低导致加热缓慢，会损害型壳湿强度，型壳易胀裂或煮烂[[S17]]。

## 脱蜡设备

### 全自动蒸汽脱蜡釜

全自动蒸汽脱蜡釜是熔模铸造的主流脱蜡核心设备，配备密封釜体、操作控制面板、压力显示仪表及快开锁紧门结构，可自动完成快速升压、保温脱蜡、缓慢释压全流程操作，减少人工操作失误，提升脱蜡安全性与成品率[[S12,S5]]。

## 工艺选型适配

脱蜡工艺的选择需依据模料类型、型壳材质及铸件品质要求进行适配[[S17,S4,S14]]：

- 低附加值普通铸件优先选择热水脱蜡以降低成本；
- 复杂高精度中温模料铸件强制采用快速高压蒸汽脱蜡以降低裂壳率；
- 水玻璃型壳不可直接用溶剂脱蜡，以免溶解水玻璃粘结剂；
- 低强度生坯陶瓷模（如粉末冶金陶瓷模工艺中的陶瓷模）需采用定向导能快速脱蜡新工艺，避免传统脱蜡方法造成模坯散架；
- 闪烧脱蜡仅适合小批量、低价值无回收需求的铸件场景。

## 安全与环保

脱蜡工序的安全环保控制要点包括[[S14]]：

- **蜡烟废气处理**：脱蜡产生的蜡烟需配备通风集气+活性炭吸附/燃烧净化装置，满足GB 39726-2020《铸造工业大气污染物排放标准》中非甲烷总烃排放要求；
- **溶剂脱蜡防护**：使用三氯乙烯等溶剂时需设置全封闭通风系统，避免有毒溶剂挥发造成人员中毒；
- **微波设备安全**：微波脱蜡设备泄漏辐射需控制在10毫瓦/厘米²以下；
- **设备安全**：脱蜡釜安全联锁门需定期校验泄压阀，防止超压爆炸。

## 局限与替代

对于粉末冶金陶瓷模工艺等特殊场景，传统热水法、蒸汽法和闪烧法均不可直接套用：热水法和蒸汽法因陶瓷模使用水溶性粘结剂而不具备抗水性，闪烧法因粘结剂在高温下快速挥发会导致模坯散架。此时需采用定向导热快速脱蜡新工艺（如管径集中热量+定向导能），在陶瓷模和大部分蜡料处于较低温度时使蜡料直接蒸发[[S4]]。
