## 概述

喂丝机（亦称喂线机）是铸造领域的一种专用熔体处理设备，编号归属GB/T 25370-2020《铸造机械 术语》9.5.10类“其他熔炼和浇注配套设备” [[S2,S9]]。其核心功能为将预制成线材或铠装线材的金属液处理剂，通过送进机构连续送入熔池或金属液流中，对金属液完成精炼、变质、孕育或合金化处理 [[S3]]。该设备配套的标准工艺含**喂丝孕育**（inoculation through wire feeding）与**喂丝球化**（spheroidization through wire feeding），对应英文术语分别为wire feeding process与wire injection process [[S3,S13,S16]]。

喂丝机在压铸与铸造应用中，主要面向铝液、铁液等金属熔体，与焊接领域用于焊丝输送的普通送丝机在工作环境、送丝对象及应用场景上完全不同，二者不属于同类设备 [[S3,S16]]。同时，喂丝机与“喷丝机”存在明确概念边界：喷丝机多用于纺织、高分子挤出或粉末喷射场景，不属于金属液处理的铸造配套设备范畴 [[S3]]。

## 定义

根据中国机械工业标准汇编铸造卷的定义，**喂丝法（喂线法）**为：

> 用专用设备将预制成线材或铠装线材的金属液处理剂送入熔池或金属液流中，对金属液进行精炼、变质、孕育或合金化处理的工艺方法。英文名称为wire feeding process、wire injection process。 [[S3,S13]]

**喂丝机**即为执行喂丝法的专用机械设备。国家标准进一步明确：喂丝孕育是指将孕育剂制成包芯线，通过喂丝机连续送入浇包内铁液中进行的孕育处理工艺；喂丝球化是将球化剂制成包芯线，通过喂丝机连续送入球化包铁液中进行的球化处理工艺。两项工艺均可在铁液自动转运过程中完成 [[S16]]。

## 分类

铸造领域喂丝机可按多种维度进行分类，不同参数维度适配不同的金属液处理场景需求 [[S8,S1]]。

| 分类维度 | 类型 | 说明 |
|----------|------|------|
| 丝材类型 | 药芯丝喂丝机 | 适配铠装粉剂包芯丝（球化剂、脱硫剂、变质剂包芯线），用于球化、脱硫、变质处理 [[S8,S1]] |
| 丝材类型 | 纯金属（实芯）丝喂丝机 | 适配纯金属实芯功能线材（Al-Ti-B丝、纯合金金属丝等），用于晶粒细化、合金添加 [[S8,S1]] |
| 安装形式 | 固定式喂丝机 | 固定安装在流槽或铁液转运线旁工作 |
| 安装形式 | 移动式喂丝机 | 可根据处理位置灵活移动 |
| 出丝数量 | 单丝喂丝机 | 一次送进一股丝材 |
| 出丝数量 | 多丝喂丝机 | 可并列同时送进多股丝材（如2根杆状中间合金丝）[[S1]] |

面向变形铝合金炉外处理的辊式喂料机属于喂丝机的常见子类，支持无级调速，可并列传送最多2根杆状中间合金丝，将丝材送入流槽铝液液面以下精确控制添加量 [[S1]]。

## 结构与组成

铝合金流槽用喂丝机（辊式喂丝送进装置）主要由以下核心部件构成：

- **放丝盘（丝卷/丝盘）**：承载盘卷状的合金丝材（如Al-Ti-B丝卷），供连续放丝使用 [[S19,S1]]；
- **送进机主体（送丝驱动机构）**：包含对压送进轮组（送丝轮组/辊轮），通过电机驱动实现丝材的定量输送 [[S19,S1]]；
- **矫直机构**：保证丝材从盘卷状态释放后平直送入铝液；
- **驱动电机及控制系统**：支持无级调速，实现送丝速度的灵活调控 [[S1]]；
- **导管与送丝嘴**：引导丝材进入并固定送进角度，保证丝材浸入铝液液面以下 [[S1]]；
- **流槽（launder）**：铝液从静置炉流向铸造单元的流动通道，丝材在其中与铝液完成熔化和反应 [[S19,S1]]。

## 工作原理

喂丝机的基本工作原理为：将盘卷装的功能丝材（如φ6~φ10 mm的Al-Ti-B丝或铁液处理用药芯包芯线）从放丝盘连续放送，经矫直后通过送丝轮组定量推入导管并到达送丝嘴，最终以设定角度送入流动金属熔体中 [[S20,S1,S19]]。

丝材在熔体中受热熔化，释放有效处理成分（形核颗粒如TiB₂、球化元素、脱硫剂粉体等），在熔体流动的搅拌混合作用下均匀分布，完成晶粒细化、球化、孕育或脱硫脱气等冶金反应 [[S20,S12]]。以铝合金晶粒细化为例，Al-Ti-B丝熔化后生成异质结晶核心，促进大量均匀细小等轴晶的形成 [[S20]]。

送进方向推荐朝向铝液 incoming 流动方向，以保证丝材充分溶解并均匀分布于整个熔体 [[S6,S10]]。根据铝液质量流量和所需添加的活性元素量，通过控制系统无级调节送丝速度，精确控制细化剂或处理剂的添加量 [[S1,S10]]。

## 工艺参数与操作条件

### 丝径
铝钛硼丝变质法使用的Al-Ti-B丝典型丝径为**φ6~φ10 mm** [[S20]]。国际常用AlTiB中间合金线材典型直径为**9.5 mm（3/8英寸）**，多以盘卷形式供货 [[S10]]。

### 送丝速度
送丝速度取决于铝液温度与铝液质量流量（熔体总流量），不同流量下存在不同的最大允许添加速率，需避免过快的送丝导致丝材未完全溶解 [[S10]]。辊式喂料机支持无级调速，可根据流槽中铝液流量和所需添加的Ti量灵活调节传送速度 [[S1]]。

### 喂入位置
喂入位置可选择在**流槽内**或**除气室**中 [[S6,S10]]。针对陶瓷过滤板的上下游位置存在以下考量 [[S6,S10]]：
- 若在过滤板上游添加，TiB₂颗粒可能被过滤板截留，降低晶粒细化效率，且TiB₂颗粒的聚集可能导致过滤板堵塞；
- 若在过滤板下游添加，需保证有足够的时间使丝材完全溶解，避免未溶解颗粒进入后续铸造工序。

在压铸/连铸生产线中，喂丝机通常布置于静置炉与铸造机之间的铝液在线精炼工段，实现在线连续处理 [[S4,S15]]。

### 喂丝角度与浸没要求
丝材必须以平稳姿态浸入铝液液面以下，不可飘忽不定，以防添加量波动过大 [[S1]]。推荐送进方向朝向铝液 incoming 流动方向 [[S10]]。

### 接触时间
从晶粒细化剂加入到铸造机的铝液流经时间应保持在有效接触时间之内。接触时间少于10 min时通常不会出现细化衰退，而接触时间过长（如25~30 min）可能严重影响细化效果 [[S1]]。

## 过程角色与应用

### 铝液在线细化与变质
喂丝机作为铝液在线调质的核心设备，匹配压铸领域连续生产流程。铝液从静置炉流出后，在流槽环节通过喂丝机连续定量送入细丝（Al-Ti-B丝）或变质丝（Sr丝），无需在炉内长时间静置处理，即可完成调质工序 [[S4,S15]]。添加形式以盘卷装 Al-Ti-B 杆料为主，通过自动或半自动送丝机（automatic or semi-automatic rod feeders）实现定量连续添加 [[S10]]。炉外铝钛硼丝变质法所采用的典型丝径为 φ6~φ10 mm，丝材在流动铝液中熔化后生成外来异质结晶核心，实现晶粒细化 [[S20]]。

### 变形铝合金连续铸造/铸轧
在铝板带连铸连轧生产中，辊式喂料机将杆状Al-Ti-B中间合金晶粒细化剂从保温炉顶部插入流槽铝液液面以下，与铝液流逆向加入，使细化剂充分溶解并均匀分布 [[S1]]。

### 铁液球化与孕育
喂丝法可应用于蠕墨铸铁球化、高牌号灰铸铁瞬时孕育、球墨铸铁球化处理等工序。在高炉铁液-中频炉双联短流程工艺中，采用精准喂丝法可有效消除短流程铁液的过冷倾向差异，稳定铸件凝固组织与力学性能 [[S18]]。喂丝球化工艺将球化剂制成包芯线，通过喂丝机连续送入球化包铁液中完成处理；喂丝孕育则将孕育剂包芯线通过喂丝机连续送入浇包内铁液中完成瞬时孕育 [[S16]]。

|----------|---------------|
| 合金丝（Al-Ti-B丝、Sr丝、Na丝等） | 喂丝机主要送进处理对象，预制成型的金属基功能线材，用于铝液细化或变质 [[S3]] |
| 药芯丝（包芯线） | 喂丝机另一类适配处理对象，外层为金属壳，内部包裹粉剂态脱硫剂、球化剂、变质剂 [[S3]] |
| 喂丝孕育 / 喂丝球化 | 喂丝机执行的核心标准工艺名称 [[S2,S16]] |
| 喂线法 / 喂丝法（wire feeding process） | 喂丝机执行的工艺方法的标准术语 [[S3,S13]] |
| 精炼处理 | 喂丝机核心应用场景之一，通过喂入功能丝材实现脱氧、脱硫、除杂净化 [[S3]] |
| 在线处理 | 喂丝机典型运行场景，在连续流动熔体中实时添加处理丝材，无需占用炉内静置时间 [[S3,S15]] |
| 喷丝机 | 概念无关，喷丝机用于纺织、高分子挤出或粉末喷射，不属于金属液处理铸造配套设备 [[S3]] |
| 送丝机（焊接用） | 应用场景不同，焊接送丝机用于电弧焊焊丝输送，与铸造喂丝机非同类型设备 [[S3]] |
