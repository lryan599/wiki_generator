## 概述

铸件图，又称毛坯图，是反映铸件实际形状、尺寸和技术要求的工程图样，属于铸造工艺设计中的核心文件。依据GB/T 5611-2017《铸造术语》5.2.13条目的定义，铸件图是“铸造生产、铸件检验与验收的主要依据”，根据已确定的铸造工艺方案，用图形、工艺符号和文字标注[[S14,S23]]。

铸件图不同于产品零件图（成品图），也不同于铸造工艺图。产品零件图仅表达零件经过全部机加工之后的最终使用尺寸、公差与装配要求，不包含铸造工艺附加余量、拔模斜度等铸态要素[[S15]]。铸件图则专门表达铸态毛坯的完整几何信息和技术要求，是连接铸造生产与后续冷加工的关键技术文件。

## 分类

### 按生产阶段分类

1. **试制阶段临时铸件图**：适配小批量试生产需求，精度要求较低，可随工艺迭代快速修改[[S13]]。
2. **定型生产正式铸件图**：适配大批量量产场景，需经冷加工部门和热加工部门及设计科室共同会签，具备正式生产效力[[S7,S13]]。一般仅在大批、大量生产的工厂才绘制铸件图[[S20]]。

### 按尺寸标注方法分类

1. **以成品零件尺寸为基础标注**：标注零件尺寸，加工余量和拔模斜度的尺寸界限在零件尺寸线向外标注，无需换算即可核对成品要素[[S22,S31]]。
2. **以铸件毛坯尺寸为基础标注**：直接标注完整的铸件毛坯尺寸，加工余量由铸件外廓尺寸线向内标注，核对铸态要素更为直观[[S22,S31]]。

大多数工厂采用第一种标注方法[[S22,S31]]。

### 按铸造工艺特性分类

针对压铸工艺，存在专属的**压铸件铸件图**类型，其内容适配压铸工艺特性，包含压铸收缩率取值、分型面标注、压铸件特有的出模斜度、浇口残留允许值等专项要素[[S12]]。

## 与相邻图样的边界与信息流转

### 铸件图 vs 铸造工艺图

铸造工艺图用于试制和小批量生产，是在成品零件图上用规定的工艺符号标注工艺参数的参考性文件。铸件图仅在成批及大批量定型生产阶段正式生成，直接标注铸件毛坯的完整参数，可作为独立的生产交付依据[[S13,S26,S35]]。

### 铸件图 vs 产品零件图（成品图）

产品零件图仅标注铸件经过所有机加工之后的最终使用尺寸、公差与装配要求，不包含铸造工艺附加余量、拔模斜度等铸态要素，仅用于最终零件检验，不能作为铸造生产的依据[[S15,S8,S13]]。

### 四类图样的信息流转逻辑

在完整的技术流程中，图样信息的流转顺序为：以**机械零件图**为起点 → 确定铸造工艺方案生成**铸造工艺图** → 工艺定型后生成正式**铸件图** → 最终以铸件图为依据完成**压铸模具图**设计[[S40]]。

| 图样类型 | 核心用途 | 包含的铸造工艺要素 | 适用阶段 |
|:---|:---|:---|:---|
| 机械零件图 | 最终成品检验与装配 | 无 | 全程 |
| 铸造工艺图 | 试制工艺指导 | 分型面、浇冒系统、型芯、冷铁等 | 试制/小批 |
| 铸件图（毛坯图） | 验收依据与加工工装设计 | 加工余量、拔模斜度、分型面、公差等 | 批量生产 |
| 压铸模具图 | 模具制造 | 型腔尺寸、抽芯机构等 | 模具制造 |

## 在压铸全流程中的工艺角色

### 压铸模具设计的法定输入依据

压铸模具设计环节必须以经正式会签的铸件图为基础，计算型腔尺寸、校核分型面位置、设定出模斜度参数，确保最终铸出的毛坯完全符合铸件图要求[[S7,S18,S11]]。

### 压铸件交付验收的直接依据

铸态毛坯出厂检验、入厂复检环节均需直接对照铸件图标注的尺寸精度、表面质量要求、缺陷允许范围开展检测，判定铸件是否合格[[S14,S7]]。

### 后续机加工工装设计的基础

冷加工工序的夹具、刀具、定位基准设计必须严格按照铸件图中标注的铸态毛坯形状、预留加工余量、指定加工基准开展，避免机加工过程出现余量不足、定位偏移等问题[[S7,S18]]。

### 压铸工艺编制的基础依据

用于确定浇注参数、浇口设置、工艺补正量取值，支撑完整压铸工艺卡的编制工作[[S16,S46]]。

## 标注内容与绘制规则

### 强制标注要素

铸件图必须包含以下核心技术要素[[S22,S31,S14]]：

- 铸件毛面上的加工定位点（面）和夹紧点（面），常用符号为◆、♠、♣（定位点）和↓（夹紧点）
- 切削余量（加工余量）
- 工艺余量（工艺凸台、补缩余量等）
- 拔模斜度（仅标注特殊斜度，通用斜度在技术条件中统一说明）
- 分型面，以及“上”“下”字样以说明浇注位置
- 内浇口和冒口残余限值
- 铸件全部形状和尺寸（含未注明的圆角、壁厚）
- 不铸出的孔槽
- 铸件尺寸公差
- 加工基准
- 铸件金属等级
- 热处理规范
- 涂漆种类
- 特殊部位允许的缺陷限制

### 线型标注规则

| 要素 | 线型规则 |
|:---|:---|
| 加工面表示 | 一般视图上以细双点划线表示加工面[[S22,S31]] |
| 铸件轮廓 | 以粗实线表示铸件轮廓形状[[S22,S31]] |
| 加工余量范围（视图） | 双点划线和实线之间标注加工余量尺寸[[S22,S31]] |
| 加工余量范围（剖面图） | 在原有剖面线上附加一层垂直剖面线，形成正方形网格线，表示切削去除部分，覆盖加工余量和不铸孔及沟槽区域[[S22,S31]] |
| 分型面 | 用细实线画出分型面在铸件上的痕迹，注明“上”“下”[[S22,S31]] |
| 浇冒口残余 | 用细双点划线画出内浇道、冒口根的位置和形状，以引出线加文字说明[[S22,S31]] |

### 简化标注规则

- 只标明特殊的铸造圆角尺寸，相同的铸造圆角在技术条件中统一说明[[S22,S31]]。
- 只标出特殊的拔模斜度，相同角度的拔模斜度统一在技术条件中说明[[S22,S31]]。

## 压铸件脱模斜度与尺寸公差的适配规则

在压铸件铸件图绘制中，脱模斜度的设置需与加工余量方案配合使用，遵循以下规则[[S2]]：

| 加工余量情况 | 型腔尺寸 | 型芯尺寸 |
|:---|:---|:---|
| 无加工余量 | 以大端为基准，另一端按脱模斜度相应减小 | 以小端为基准，另一端按脱模斜度相应增大 |
| 两面留有加工余量 | 以小端为基准 | 以大端为基准 |
| 单面留有加工余量 | 以非加工面大端为基准，加上斜度值及加工余量，另一端相应减小 | 以非加工面小端为基准，减去斜度值及加工余量，另一端相应增大 |

## 相关标准体系

现行国内铸件相关国家/行业标准体系条目包括[[S45,S19]]：

| 标准编号 | 标准名称 | 主要适用内容 |
|:---|:---|:---|
| GB/T 5611-2017 | 铸造术语 | 铸件图、铸造工艺图等术语定义（5.2.13/5.2.6条） |
| GB/T 6414-1999 | 铸件尺寸公差与机械加工余量 | 等效ISO 8062:1994；CT1~CT16共16级公差体系 |
| GB/T 11350 | 铸件机械加工余量 | 加工余量分级与选择 |
| GB/T 11351 | 铸件重量公差 | 铸件重量偏差验收 |
| JB/T 2435-2013 | 铸造工艺符号及表示方法 | 工艺图与铸件图标注符号规范 |
| GB/T 6060.1 | 铸造表面粗糙度比较样块 | 铸造表面粗糙度评定 |
| ISO 8062：1994 | 铸件尺寸公差体系 | 国际标准，GB/T 6414等效采用 |
| ISO/CD 8062-2 | 铸件形位公差体系 | CTG2~CTG8共8个公差等级，覆盖直线度、圆度、平行度、垂直度、同轴度等类型[[S34,S9]] |

GB/T 6414-1999定义“铸件基本尺寸”为机械加工前的毛坯铸件尺寸，包含必要的机械加工余量RMA；加工余量等级由精到粗分为A至K共10个级别；尺寸公差等级CT1至CT16共16级，并补充了压铸件、熔模铸件和金属型铸件的典型可达公差等级[[S41,S17]]。

## 常见标注错误

铸件图标注中的典型错误包括[[S22,S42,S4,S27]]：

1. **加工余量取值不当**：未按GB/T 11350、GB/T 6414标准区分顶面、侧面、底面的余量等级，未根据铸件尺寸大小和浇注位置适配对应余量值。
2. **基准选择缺失**：遗漏加工定位点和夹紧点的专属符号标注，导致后续机加工工装定位基准不统一。
3. **拔模斜度标注不合理**：未仅标注特殊斜度，将所有通用角度逐一在视图中标注造成图面冗余，或未将通用拔模斜度统一在技术条件中说明。
4. **圆角标注不明确**：未仅标注特殊圆角尺寸，所有相同通用圆角未在技术条件中统一说明，遗漏转角过渡要求。
5. **未标注分型面“上/下”标识**：无法追溯浇注位置。

## 铸造缺陷的标注与验收规则

### 缺陷标注核心规则

- 不同部位的允许缺陷限制需直接在对应图形位置标注并补充文字说明[[S22,S31]]。
- 铸件需分类为I、II、III类进行验收，铸件类别、合金牌号、铸造方法、供应状态、执行标准需统一在图面指定位置按照规范格式标注[[S43]]。
- **铸件图样未明确标注缺陷验收级别时**：加工表面默认为c级，非加工表面默认为d级[[S10,S37]]。
- 螺纹孔内前4个旋入牙距区域严禁存在任何缺陷[[S10,S37]]。

### 常见缺陷类型标注与验收阈值

| 缺陷类型 | 标注与验收要求 |
|:---|:---|
| 裂纹 | 各类铸件内部不允许存在裂纹；作为加工基准的表面不允许存在裂纹；穿透性裂纹属于直接报废项[[S10]] |
| 气孔/孔洞（单个） | 最大直径不大于3 mm，深度不超过对应位置壁厚的1/3，安装边区域不超过壁厚1/4且最大深度不大于1.5 mm；同一截面反面对称位置不能出现同类缺陷；直径小于1 mm（非加工面）或0.5 mm（加工面）的微小孔洞不计入统计[[S10,S25,S6]] |
| 成组孔洞 | I、II类铸件最大直径不大于1.5 mm，深度不超过壁厚1/3且不大于1 mm；III类铸件最大直径不大于2 mm，深度不超过壁厚1/3且不大于1.5 mm[[S10,S37]] |
| 缩松/针孔 | I类铸件或液压气压类铸件加工表面允许2级针孔，局部3级针孔占受检面积不得超过25%；II类铸件按低一级针孔度验收[[S10,S6]] |

## 压铸工程师读图要点

压铸工程师正确解读铸件图的标准化步骤[[S22,S31,S1]]：

1. **第一步**：核对铸件类别、合金牌号、铸造工艺、执行标准的标注完整性。
2. **第二步**：识别加工定位点（◆/♠/♣）、夹紧点（↓）符号，明确机加工基准。
3. **第三步**：逐项核对加工余量、特殊拔模斜度、特殊圆角的参数取值，结合对应标准判断合规性。
4. **第四步**：读取分型面“上/下”标识明确浇注位置，确认浇冒口残余的允许最大尺寸。
5. **第五步**：核对指定部位缺陷限制标注，明确气孔、缩松等缺陷的验收阈值。
6. **第六步**：交叉校验所有统一在技术条件中说明的通用参数（通用圆角、通用拔模斜度）是否符合压铸工艺要求。

### 常见易错点

- 将铸件图的铸造轮廓直接等同于产品零件轮廓[[S1]]。
- 误读加工余量网格区域为实体铸件部分[[S1,S22]]。
- 忽略非加工面的缺陷限制要求[[S1]]。
