## 概述

精整工作台（finishing table）是压铸后处理中用于人工完成铸件精整作业的专用工位设备，核心功能包含承载工件、夹紧工件，并可根据作业需求额外配置工件升降、翻转功能[[S19,S12]]。该术语在GB/T 25370-2020《铸造机械 术语》6.4.8条目中有明确定义，对应的英文标准术语为“finishing table”[[S19]]。

在国际铸造行业中，与精整工作台对应的英文术语还包括trimming workbench、deburring bench、fettling bench、casting finishing station等。其中“fettling”为英式铸造领域对铸件切割、打磨、去除飞边毛刺工艺的通用称谓，“fettling bench”即传统人工精整工作台的典型表述[[S22,S16]]。

## 工艺作用

在压铸后处理全流程中，精整工作台属于清理整修装置范畴[[S6]]。其标准部署位置为冲切机、切边模工序的下游，振动研磨机、抛丸机工序的上游：承接冲切浇口后仍残留多余毛刺的铸件，完成人工精细修整作业后，工件再转入后续批量光整加工环节[[S5,S2]]。

精整工作台承载的典型作业内容包括[[S17]]：
- 去除浇注系统、排溢系统残留的飞边与毛刺
- 对变形工件实施矫形修整
- 完成铸件外观质量目视检验

压铸件清理工作的总工作量可达压铸本体工序工作量的数倍乃至十几倍，精整作业占铸件总劳动量的比例可达1/5~1/3，精整工作台作为该工序的核心载体，其结构设计的合理性直接影响作业效率与操作人员劳动强度[[S10,S17]]。

## 结构与组成

典型的压铸精整工作台由以下核心组件构成：

**承载与夹紧系统**：工作台台面用于承载铸件，配置夹紧装置以固定工件，适应手持工具进行铲削、打磨等操作[[S19]]。

**升降与翻转机构**：根据需要可配置工件升降和翻转功能，便于处理不同高度和方位的铸件。面向大型铸件的精整工作台可配置垂直高度调节机构[[S19,S9]]。配套浇冒口切除作业的工作台可搭载2.5kW驱动电机，带动台面与铸件实现水平往复移动，配合固定圆盘锯完成浇道、冒口切割作业[[S9]]。

**除尘与排屑系统**：面向中小型、中型压铸件人工精整的精整工作台，标配悬挂式吸尘罩、废料传送带与集料斗结构，可大幅降低打磨修整过程中金属粉尘向车间环境扩散的程度[[S20]]。GB/T 37371-2019《压铸单元 术语》明确要求清理整修装置必须配套粉尘收集过滤系统，处理精整作业产生的打磨碎屑和金属粉尘[[S6]]。

**辅助装置**：包括工具挂架（用于放置铲刀、气动打磨头等）、局部照明设施，以及气动/电动接口供动力工具使用[[S1]]。

精整工作台通常作为封闭或半封闭精整室的标配组成部件，该精整室具备隔声和防止粉尘外逸的防护设计[[S19]]。

## 分类

压铸行业对精整工作台的主流分类包含三类维度[[S1,S3]]：

| 分类维度 | 类型 | 特征说明 |
|---------|------|----------|
| 按安装形式 | 固定式精整工作台 | 固定安装在精整室内，连接固定除尘管路 |
| | 移动式精整工作台 | 配套移动吸尘系统，可在不同清理工位间灵活流转 |
| 按集成属性 | 独立式精整工作台 | 单独设置的作业工位 |
| | 产线集成式精整工作台 | 嵌入铸件清理流水线，作为人工补修工位 |
| 按配置等级 | 纯手动式精整工作台 | 仅提供工件承载支撑，全部使用手持工具操作 |
| | 带动力辅具复合式精整工作台 | 集成工件夹紧、液压升降翻转、局部除尘、悬挂式动力打磨工具 |

## 与同类设备的区别

精整工作台与同类自动化设备在功能定位上存在明确差异[[S11,S14]]：

| 对比项 | 精整工作台 | 全自动切边机 | 固定主轴打磨机 |
|-------|-----------|-------------|--------------|
| 操作方式 | 人工操作 | 全自动冲切 | 自动化磨削 |
| 模具需求 | 无需定制模具 | 需搭配定制切边模具 | 无需模具 |
| 适用场景 | 多品种小批量、复杂异形铸件 | 单一品种大批量 | 规则开放表面 |
| 换型成本 | 无额外成本 | 需更换切边模 | 受限于固定主轴 |

全自动切边机通过定制切边模具仅能完成固定轮廓浇口、溢流边的强制切除；专用固定主轴打磨机仅能对规则开放表面执行自动化磨削作业；而精整工作台以人工操作为核心，无需定制模具，可任意适配不同形状铸件的夹紧、翻转，灵活搭配手持铲刀、气动打磨头等辅具，柔性远高于前两类设备[[S11,S14]]。

## 安全与维护

精整工作台的作业区域需满足以下安全要求：

- **防尘通风**：必须配套完善的除尘和通风设施，控制金属粉尘浓度[[S6,S20]]。
- **镁合金特殊要求**：镁合金压铸精整工作台的作业区域严禁布置明火，避免镁合金粉尘与空气混合后遇火源发生爆炸事故[[S7]]。
- **照明要求**：工作地需有良好照明环境，以分辨铸件表面缺陷，混合照明最低照度采用白炽灯时应为300 lx，采用荧光灯时应为500 lx[[S1]]。

## 应用场景

精整工作台主要适用于中小型压铸件的人工批量精整作业，常见于铝合金、锌合金压铸车间的清理工部。在压铸件出模冷却后、自动化大批量切边/抛丸工序之间，精整工作台作为柔性人工处理工位，主要承接小批量异形铸件、不规则残留飞边、局部缺陷修整的非标准化清理任务[[S15]]。
