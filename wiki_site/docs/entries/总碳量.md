## 概述

在压铸领域，“总碳量”通常指材料体系中全部碳元素的质量分数，包含以化合态（如碳化物）存在的碳与游离态（如石墨）存在的碳两部分，是压铸材料成分控制的核心基础指标[[S6]]。实际应用中，总碳量指向两种截然不同的场景：一是压铸模具材料（以热作模具钢为主）中的碳含量，直接影响模具的淬硬性、韧性及热疲劳寿命；二是压铸轻合金（铝合金、锌合金、镁合金）中的碳杂质含量，关系铸件的夹杂缺陷与气孔倾向。两者在管控目标、典型数值范围及影响机理上差异显著，应在使用中明确区分。

## 检测方法与标准

总碳量的主流检测手段以燃烧-红外吸收法为核心，当前压铸行业广泛应用高频感应炉燃烧后红外吸收法（常规方法）[[S7]]。主要检测标准覆盖：

| 标准编号 | 名称 | 适用范围 |
|---------|------|---------|
| GB/T 223.69 | 钢铁及合金 碳含量的测定 管式炉内燃烧后气体容量法 | 常规钢铁材料碳元素定量检测 [[S7]] |
| GB/T 20123 | 钢铁 总碳硫含量的测定 高频感应炉燃烧后红外吸收法（常规方法） | 压铸行业主流检测手段，效率与精度俱佳 [[S7]] |
| ASTM E1019 | 钢、铁、镍合金和钴合金中碳、硫、磷的测定方法标准 | 国际通用的多元素同步检测要求 [[S7]] |

上述方法的基本原理为：样品在纯氧气氛中高温燃烧，碳元素氧化生成CO及CO₂（以CO₂为主），经催化转化后，由红外检测单元定量测定CO₂浓度，换算为总碳量[[S7]]。

### 压铸轻合金

压铸铝合金（如ADC12）、镁合金（如AZ91D）、锌合金（如ZZnAl4系列）等轻合金中，碳被视为杂质元素，相关标准限定总碳量最高占比一般不超过0.1%[[S3,S4]]。

## 影响机理

### 对模具钢性能的影响

碳是热作模具钢中最关键的合金元素之一，其含量变化直接影响模具的综合力学性能：

- **正效应**：提高碳含量可显著提升钢的高温强度、热态硬度及耐磨损性[[S4]]。
- **阈值效应**：当碳含量超过约0.4%时，钢的塑性和韧度明显降低，抗热疲劳性能恶化，易引发碳化物带状偏析，加速热疲劳裂纹扩展[[S4]]。
- **设计原则**：在钢的抗拉强度达1550 MPa以上时，含碳量以0.3%～0.4%为宜[[S4]]。

H13钢中铬一部分溶入基体起固溶强化作用，另一部分与碳结合以(FeCr)₃C、(FeCr)₇C₃及M₂₃C₆等形式存在，共同影响钢的性能[[S4]]。当钢中含Cr、Mo、V时，Cr＞3%会阻止V₄C₃生成并推迟Mo₂C共格析出，这种交互作用可提高钢的耐热变形性能[[S4]]。

### 对压铸合金铸件质量的影响

对于压铸铝合金等轻合金，总碳量超标将产生两方面负面效应[[S4,S2]]：
1. 熔体中生成初生硬质碳化物夹杂，降低铸件力学性能；
2. 增加熔炼过程氢气析出量，显著提升铸件气孔缺陷概率。

## 工艺控制

### 模具钢生产端

采用真空冶炼、电渣重熔等工艺可精确控制总碳量的均匀性，有效抑制碳化物偏析，保证材质的冶金质量[[S2,S8]]。

### 压铸合金熔炼端

通过精炼工序精准调控增碳与脱碳过程，将总碳量稳定控制在标准限定区间内。管控到位后，可将铝合金压铸用H13模具的常规寿命从3～5万模次提升至10万模次以上，同时压铸件碳化物夹杂与气孔缺陷占比降低，良品率可提升15%～30%[[S8]]。

## 相关关系

### 与碳当量的区别和关联

总碳量与碳当量（Carbon Equivalent, CE）是两个物理意义完全不同的概念[[S6,S5]]：

- **总碳量**：实测得到的单一碳元素质量分数，是材料成分分析的直接结果。
- **碳当量**：经验换算指标，将硅、磷等其他元素的石墨化（或共晶化）作用折算为等效碳含量。铸铁中典型的碳当量公式为 CE = C + (Si + P)/3，或将磷纳入时扩展为 CE = C + 0.3×(Si + P)[[S5]]。

总碳量是碳当量计算的基础输入参数。碳当量常用于快速评估材料的共晶凝固倾向与白口抗力，间接关联铸件热裂敏感性。下图展示了不同碳当量合金的热裂敏感指数（CSC\*）变化规律，碳当量4.21%时CSC\*最高，4.67%时CSC\*最低，清晰呈现碳当量对铸件热裂缺陷的影响传导路径[[S11]]。
