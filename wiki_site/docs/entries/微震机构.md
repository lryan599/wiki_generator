## 概述

**微震机构**是铸造机械中用于产生微幅高频振动的核心执行部件，在国家标准 GB/T 25370-2020《铸造机械 术语》中被列为正式术语（条目编号 4.1.35），归属于震击压实造型机的配套部件范畴 [[S3]]。该机构通过刚性碰撞产生微幅震击，将振动能量传递至工作台面的砂型或熔体，实现型砂紧实或凝固组织调控。

在压铸与特种铸造的延伸应用中，微震机构（亦称振动辅助压铸微振动装置）被用于在充型和凝固阶段向金属熔体输入微幅振动，以细化凝固组织、改善熔体润湿性并提升铸件力学性能 [[S4]]。

需要指出的是，在现行压铸专用标准（如 GB/T 21269-2023《冷室压铸机》、GB/T 25716-2022《镁合金冷室压铸机》）中未见"微震机构"作为压铸机本体的独立部件条目，其定义主要源于砂型铸造设备领域 [[S19,S1]]。压铸语境下的微震功能通常以附加功能部件形式集成于工艺系统中。

## 分类

微震机构按缓冲结构可划分为**弹簧式**与**气垫式**两大类，每类下含若干细分类型 [[S17,S20]]。

| 大类 | 细分类型 | 核心特征 |
|------|----------|----------|
| 弹簧式 | 弹簧缓冲式 | 以弹簧为缓冲储能元件，结构相对简单 |
| 弹簧式 | 弹簧与气垫综合缓冲式 | 震击前气垫缸进气托起震击部分，震击后排气返回，缓冲效果优于纯弹簧结构，但复杂度更高 |
| 弹簧式 | 消音式 | 撞击面设置在机体内部，压缩空气从机座空腔排出，消音效果好，工作台依托压实缸导向提升稳定性，缺点是结构复杂、安装便利性较低 |
| 气垫式 | 单柱塞式 | 配置控载阀以保障工作台浮起高度不受载荷变化影响 |
| 气垫式 | 双柱塞式 | 无独立气垫缸，借用接砂缸内气体作为预震气垫、压实缸内气体作为压震气垫；若接砂缸不进气预震则变为砧座式震击，对基础刚度要求更高 |
| 气垫式 | 环形气垫式 | 气垫为封闭环形腔，充入压缩空气后工作台与砧铁贴紧不抬升；震铁剖面呈 H 形，震击效果优于柱塞式结构 |

不同类型在结构复杂度、缓冲效果、噪声水平、耗气量及基础承载要求等维度存在显著差异 [[S17,S20,S6,S10]]。《机械工程手册 第57篇 铸造机械化与自动化》收录了弹簧式与气垫式微震机构的详细对比表，从结构性能、造型效果、司气参数、缓冲措施、耗气量等多维度进行了系统对比 [[S23]]，以及不同类型气垫微震机构的对比表，从气源类型、耗气量、工作台浮动特性、结构复杂度、基础承载要求等维度进行了差异分析 [[S12]]。

## 结构与组成

微震机构的核心组成零件包括 [[S9,S17,S6]]：

| 部件 | 功能角色 |
|------|----------|
| 震击缸 | 激振腔，压缩空气充入后驱动震击动作 |
| 砧铁 | 震击碰撞部件，与工作台发生刚性碰撞产生震击 |
| 震击活塞 | 运动执行部件，受气压推动实现往复运动 |
| 工作台 | 承载部件，直接承载砂型或模具，传递震动 |
| 缓冲弹簧 / 气垫腔 | 减震储能部件，吸收震击反力并提供回弹力 |
| 震击进气阀 / 控载阀 | 运动逻辑控制部件，控制压缩空气的进气、排气时序 |

## 工作原理

经典气动微震机构的工作循环如下 [[S17,S20]]：
1. 压缩空气经震击进气孔进入震击缸空腔；
2. 缸内气压推动工作台向上运动，同时砧铁向下运动；
3. 当进气环形槽与排气环形槽连通时，震击缸内压缩空气从排气孔排出；
4. 工作台快速下落，与砧铁发生刚性碰撞，产生微幅震击；
5. 震动传导至工作台面，实现型砂紧实。

该循环可与造型机的后续压实工序协同联动，完成砂型制备。

弹簧与气垫综合缓冲式微震机构在震击前先由气垫缸进气将震击部分托起，震击后气垫缸排气使震击部分返回，以此获得更好的缓冲效果 [[S17]]。单柱塞气垫式微震机构则通过控载阀调节气垫腔压力，使工作台浮起高度不受载荷变化影响 [[S6,S10]]。

## 关键参数

### 砂型铸造用气动微震压实造型机振动参数

气动微震压实造型机在不同工作阶段具有不同的振动参数范围 [[S11]]：

| 工作阶段 | 频率范围 | 振幅范围 |
|----------|----------|----------|
| 预震阶段 | 150～500 次/min | 25～80 mm |
| 压震阶段 | 700～1000 次/min | 5～10 mm |

### 压铸 / 特种铸造用振动辅助参数

在振动辅助消失模固-液复合铸造中，典型的振动参数范围为 [[S5]]：振动频率 20～50 Hz，振幅 0.2～0.6 mm。拉铸铁管振动装置的低频参数范围为频率 140～180 次/分钟（约 2.3～3 Hz），振幅 4～10 mm，电机功率 2.8～4.5 kW [[S21]]。

### 设计计算公式

弹簧式微震机构的震击活塞直径计算式为 [[S6,S10]]：

$$d_{z} = \sqrt{\frac{4K(Q_1 + Q_2 + R)}{\pi p}}$$

式中，举升力储备系数 _K_ 的取值范围为 1.3～2.2，小规格造型机取上限，大规格造型机取下限。

## 工艺作用

在压铸及特种铸造中，微震机构向金属熔体输入机械振动后，产生以下工艺效应 [[S4]]：

- **改变流动模式**：将金属液的自然对流转变为强制对流，显著提升充型流速；
- **改善润湿性**：改善熔体与嵌体/型壁的润湿性能；
- **均匀温度场**：使铸件内部温度场更趋均匀；
- **细化凝固组织**：细化界面金属间化合物（IMCs）层中的凝固组织，减小 IMCs 层厚度；
- **提升界面结合性能**：经实测，施加适配参数的微震辅助压铸工艺后，双金属界面剪切强度最高可达 46.83 MPa，相比无振动对照组提升幅度达 47%。

## 生产应用效果

在压铸模维护方面，配合振动时效处理（压铸模生产 ≤5000 模次时进行一次去应力处理），可大幅减少压铸模热变形与型腔表面龟裂故障。某厂的实际应用数据表明：采用振动时效技术后，同等条件下压铸生产线日产量可从处理前的初期 2～3 t 稳定提升至 11～12 t，大量减少因卡模、粘模和表面拉伤等缺陷导致的停机修模 [[S13]]。

在压铸件后处理领域，振动清理技术被用于去除压铸件飞边残余的虚尖毛刺，振磨后压铸件在所有尖角虚尖处产生均匀圆角 R0.1～0.2 mm，可供直接装配使用 [[S15]]。

## 局限与注意事项

- **结构复杂度**：消音式及弹簧-气垫复合式微震机构结构较复杂，安装便利性较低 [[S17]]；
- **对基础的要求**：双柱塞式气垫微震机构在无独立气垫缸或接砂缸未进气时，预震变为砧座式震击，对设备基础刚度要求更高 [[S6]]；
- **振动清理设备局限**：振动清理设备存在易损坏、维修费用大、噪声较大、对磨料要求严格、占用辅助面积较大等缺点 [[S15]]；
- **振动时效设备成本**：传统热时效去应力工艺使用费用高、辅助设备多、耗能高，但振动时效技术在操作不当时可能效果不及预期 [[S13]]。
