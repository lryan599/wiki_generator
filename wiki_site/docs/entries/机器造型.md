## 词条说明

**“机器造型”在压铸领域并非独立的标准工艺术语。**

经查现行国家标准 GB/T 5611—2017《铸造术语》，“机器造型”是**砂型铸造领域的专属术语**，其定义为利用专用造型机械完成填砂、紧实、起模等核心工序来制造砂型的过程[[S40]]。而在压铸领域的国家标准（如 GB/T 25370—2020《铸造机械 术语》、GB 20906—2007《压铸单元安全技术要求》）及权威压铸手册中，均未出现“机器造型”作为独立术语的定义条目[[S4,S27]]。

在压铸行业非正式场合，偶有从业者以“机器造型”泛称“压铸成型/压力铸造”的全流程生产工序，但此表述属于不规范口语化泛称，不具有标准术语地位。

**本词条因此需要做出重要界定**：下文所述内容属于**砂型铸造“机器造型”**，并特设一节说明其与压铸工艺的核心区别，以澄清概念混淆。

## 概述

机器造型是砂型铸造生产中实现机械化制芯与造型的关键工艺方法。其定义为：通过专用造型机械完成填砂、紧实、起模、合箱等主要工序，以制造砂质铸型的过程[[S40]]。该工艺取代传统手工造型，显著提高了铸造生产效率和铸型质量的一致性，是现代砂型铸造车间实现批量化和自动化生产的基础。

与此相对，压铸（或压力铸造）是将熔融金属液在高压、高速条件下直接压入永久性钢制模具型腔，并在高压下凝固成型的特种铸造方法，完全不涉及型砂和砂质铸型的制作[[S4,S17]]。二者工艺原理、作业对象和所属铸造类别均根本不同。

## 分类体系

砂型铸造机器造型按紧实原理和设备形式可分为以下类别[[S26]]：

| 主类 | 子类 |
|------|------|
| 压实造型 | 低压压实造型、中压压实造型、高压压实造型（含多触头高压造型） |
| 震实造型 | 震击造型 |
| 震压造型 | 震击+压实复合造型 |
| 抛砂造型 | 固定式抛砂、移动式抛砂 |
| 射砂造型 | 射压造型、水平分型脱箱射压造型 |
| 气压造型 | 气冲造型、静压造型 |
| 其他 | 微振压实造型、多工位造型等 |

该分类严格归属于砂型铸造领域，与压铸设备或压铸工艺无涉[[S26]]。

## 工艺特征

与手工造型相比，机器造型对分型面选择和工艺方案设计有明确影响。以同一铸件（机后盖）为例[[S7]]：

- **机器造型**方案：大平面朝下布置，下型带出内孔，便于机械化起模与合箱操作；
- **手工造型**方案：大平面朝上布置，上型带出内孔，操作灵活性高但一致性差。

这一对比表明，机器造型的工艺设计必须适配造型机械的结构约束与动作逻辑[[S7]]。

## 与压铸的关键区别

| 对比维度 | 砂型铸造“机器造型” | 压铸（压力铸造） |
|----------|---------------------|-------------------|
| 术语定位 | GB/T 5611 定义的标准术语[[S40]] | GB 20906、GB/T 25370 定义的标准术语“压铸”[[S4,S27]] |
| 成型对象 | 一次性使用的砂质铸型 | 永久性钢制模具型腔 |
| 成型材料 | 型砂（含粘结剂） | 液态非铁金属（铝、锌、镁、铜等）[[S17,S34]] |
| 工艺原理 | 机械力紧实型砂成型铸型 | 高压高速将金属液射入模具型腔并凝固[[S4,S34]] |
| 最终产物 | 用于浇注的砂型 | 直接获得金属铸件 |
| 对应英文术语 | machine molding | die casting（北美）/ pressure die casting（英国及欧洲）[[S8,S33,S38]] |

“机器造型”在压铸语境下的不规范泛称，实际指代的正式工艺术语是**压铸（die casting / pressure die casting）**[[S8,S33,S38]]，其核心定义已由国家标准明确：将熔融金属射入模具中，并在高压状态下保持到完全凝固为止[[S4]]。
