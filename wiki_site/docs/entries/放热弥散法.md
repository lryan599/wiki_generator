## 放热弥散法（XD法）概述

放热弥散法（Exothermic Dispersion Method），又称XD技术或XD™工艺，是1983年由美国Martin Marietta实验室在自蔓延高温合成法（SHS）的基础上改进发明的一种金属基复合材料原位反应合成技术[[S6,S35]]。该技术的核心特征是通过将含有反应剂元素的混合粉末体系整体加热至基体熔点以上，使组元间在熔融介质中发生放热化学反应，从而原位生成尺寸细小、弥散分布的亚微米级增强相颗粒[[S35]]。由于增强颗粒是通过整体加热引发的同步放热反应生成并实现弥散分布的特点，该方法被定义为放热弥散法[[S35]]。

在工艺分类上，放热弥散法属于反应加工技术大类，与自蔓延高温合成（SHS）、DIMOX法、液相-气相反应法等同属于原位反应合成类材料制备工艺分支[[S38]]，同时也是原位纳米颗粒增强铝基复合材料的主流制备方法之一[[S2,S23]]。

## 工艺原理与反应机制

放热弥散法的基本原理是将可以反应生成增强相的反应物粉末与金属基体粉末按化学反应计量比均匀混合，经冷压或热等静压成型为预制体，然后以可控加热速率将预制体整体加热至基体熔点以上、增强相熔点以下的温度区间，使组元间在熔融金属介质中发生原位放热反应，生成亚微米级陶瓷或金属间化合物增强颗粒[[S4,S35]]。

与SHS技术依靠单点引燃后通过燃烧波自蔓延的方式完成全域反应不同，放热弥散法采用整体均匀加热，使反应在整个混合物体系中处处同时发生，无需引燃装置，反应过程可控性更强，产物致密度更高[[S35,S36]]。该工艺的实质是以熔融金属基体为反应介质，通过组元间的扩散反应生成陶瓷增强粒子[[S36]]。

放热反应可涵盖铝热反应等各类原位放热反应类型，典型体系如Al-Ti-B₄C、Al-TiO₂-C体系等[[S5,S36]]。XD反应是一个短暂的过程，基体与生成的强化相之间界面反应极弱，界面干净、结合良好[[S35]]。

## 工艺流程与关键参数

### 典型工艺步骤

放热弥散法的通用全流程步骤分为[[S35,S8,S23]]：

1. **粉末混合**：将生成增强相的反应元素粉料与金属基体粉料按反应配比均匀混合，可搭配高能球磨辅助分散；
2. **压制成型**：将混合粉料通过冷压或热等静压成型为预制体；
3. **加热反应**：以特定加热速率将预制体升温至基体熔点以上、增强相熔点以下的温度区间，在惰性气氛保护下引发各组元间的原位放热反应；
4. **母合金制备**：通常先制备高体积分数增强相的母合金；
5. **稀释调配**：将母合金与常规合金基体在后续熔炼、铸造环节按所需比例稀释，得到指定增强相体积分数的复合材料。

### 关键工艺参数

基于Al-TiO₂-煅烧高岭土体系制备铝基复合材料的实测典型工艺参数如下[[S9]]：

| 工艺环节 | 参数描述 | 参考范围 |
|---------|---------|---------|
| 粉末干燥 | 真空干燥预处理 | 120℃, 2h |
| 冷压成型 | 压制成型压力 | 30MPa |
| 反应气氛 | 保护气氛类型 | 氩气（惰性气氛） |
| 反应温度 | 加热反应温区 | 800~1000℃ |
| 保温时间 | 高温段保温时长范围 | 0min~12h |

加热速率是放热弥散法的重要工艺控制参数。研究表明，采用的加热速率越高，体系的燃烧温度Tc提升越显著，较高的燃烧温度会促使大量液相形成，从而提升产物的致密度，并改变产物的显微组织[[S35,S17]]。增强相颗粒的尺寸可在0.1~10μm之间调节，形状均匀无尖角，且可通过调整保温参数实现有效控制[[S29]]。

增强相的尺寸、形状及体积分数构成该工艺的核心管控要素[[S6,S4]]。

## 材料体系与化学反应

### 基体合金类型

已公开的放热弥散法适配铝合金基体牌号包括A356、6061、ZL109、205A、Al-Cu-Mg系列等[[S9,S8,S28]]。

### 前驱体体系与增强相

| 前驱体体系 | 生成增强相 | 特征说明 |
|-----------|-----------|---------|
| Al-纳米SiO₂ | 纳米α-Al₂O₃ | 采用高能球磨辅助分散[[S23]] |
| Al-TiO₂-C | Al₂O₃+TiC | 同时生成两种增强相[[S4]] |
| 铝-煅烧高岭土 | 纳米Al₂O₃ | 以廉价工业原料为前驱体[[S9]] |
| Ti-B-Al元素粉 | TiB₂ | 典型XD增强相体系[[S21]] |
| K₂ZrF₆+KBF₄+Al熔体 | ZrB₂ | 混合盐法衍生体系[[S2]] |

### 关键反应方程式

对于Al-TiO₂-C反应体系，已公开的完整化学反应过程包括[[S4,S36]]：

- 主反应：4Al + 3TiO₂ + 3C = 2Al₂O₃ + 3TiC
- 铝热反应阶段：4Al + 3TiO₂ → 2Al₂O₃ + 3Ti
- 碳化反应阶段：Ti + C → TiC

当C与TiO₂摩尔比为1时，组织中无中间产物残留，所制得的复合材料室温抗拉强度达350.7MPa、延伸率达6%，相比对应基体合金分别提升27.9%和100%[[S32]]。

### 增强相体积分数范围

美国Martin Marietta公司采用XD工艺已实现0~15vol.% TiB₂颗粒增强Al-Cu-Mg合金复合材料的制备[[S28]]，Gotman等人进一步制备出30vol.% TiB₂/Al和30vol.% TiC/Al复合材料[[S29]]。XD法制备的复合材料中，增强体与基体界面干净、结合强度高，无外加增强体的预处理工序，整体制备流程简单，易于规模化生产[[S8]]。

## 与压铸的关联及应用

### 母合金重熔工艺路线

XD法直接制备的产物通常为高孔隙率的毛坯，在实际工业应用中不会直接作为压铸生坯投入压铸流程[[S29]]。常规工艺路线为先通过XD法制备含有0~30vol.%增强相的高体积分数颗粒增强铝基母合金，再将该母合金与常规铝合金基体按照目标比例混合重熔，得到预设增强相含量的复合材料熔体，可直接接入高压压铸、半固态压铸的生产流程[[S29,S35,S20]]。

### 充型能力影响

XD法制备的原位颗粒增强铝基复合材料的充型流动性随增强相体积分数上升而降低。根据螺旋型试样测试结果，当SiC类增强颗粒体积分数达到15%时，合金的流动距离仅为无增强相基体铝合金的50%左右[[S22]]。高压压铸对应的高冷却速率可大幅减少增强颗粒的偏析沉降，半固态压铸的层流充型特性可降低湍流卷气概率，有利于改善该类复合材料的充型稳定性[[S22,S30]]。

### 铸件力学性能

XD法原位生成的纳米颗粒增强铝基复合材料经压铸/挤压铸造成形后，力学性能显著提升：

- 0.6vol.%纳米Al₂O₃颗粒增强A356铝合金经挤压铸造成形后，晶粒细化率可达30.55%[[S33]]；
- 对比普通铸态同成分基体合金，极限抗拉强度提升29.81%，屈服强度提升43.18%，硬度和耐磨性能显著优于普通铝合金[[S33]]；
- XD工艺制备的0~15vol.% TiB₂颗粒增强Al-Cu-Mg复合材料经压铸成形后，高温强度较未增强基体合金有大幅提升[[S29]]。

### 缺陷控制优势

XD法原位生成的增强相与铝基体界面洁净无外来杂质污染，界面结合强度高[[S36]]。配合压铸过程的高压快速凝固作用，可有效降低铸件内部孔隙率，减少气孔、夹杂类铸造缺陷，最终获得的复合材料铸件致密度远高于普通重力铸造的同材质铸件[[S10,S25]]。

### 商业化应用

当前已有商业化的XD法衍生原位颗粒增强铝基高压压铸材料体系，典型代表为Duralcan系列20vol.% SiC增强AA360铝合金，专为高压压铸工况开发，用于制备汽车制动部件、高负荷结构件，可实现批量化工业生产[[S37]]。Duralcan等行业厂商量产的SiC颗粒增强铝基复合材料已广泛应用于汽车制动盘、自行车传动构件等工业场景[[S27]]。

## 优势与局限性

### 核心优势

1. **增强颗粒原位生成**：增强颗粒通过放热反应在基体内部原位生成，表面无氧化污染，与基体界面洁净、结合强度高[[S35,S36]]；
2. **颗粒弥散均匀**：增强相颗粒分布均匀性优于常规铸造工艺[[S17]]；
3. **设备简单、成本可控**：无需引燃装置，设备简单，且可通过母合金稀释策略大幅降低规模化制备成本[[S35]]；
4. **增强相含量灵活调控**：增强体的体积分数可在大范围内灵活调控[[S17]]；
5. **界面反应弱**：XD反应过程短暂，基体与强化相之间的界面反应大大减弱[[S35]]。

### 局限性

1. **纳米颗粒分散成本高**：若要实现纳米陶瓷增强相的均匀分散需搭配高能球磨工序，整个分散过程耗时较长，能耗和物料成本相对较高[[S23]]；
2. **产物致密度不足**：XD工艺制成的产品存在较大孔隙度问题，产物为疏松多孔材料，需通过后续再熔铸或二次热加工技术实现致密化[[S29,S4]]；
3. **工艺技巧性强**：该方法工艺技巧性强、难度高，不易掌握[[S29]]。

## 与其他制备方法的对比

### 与SHS法的核心区别

| 对比维度 | 放热弥散法（XD） | 自蔓延高温合成法（SHS） |
|---------|-----------------|----------------------|
| 引燃方式 | 整体均匀加热，无需引燃装置 | 单点局部引燃，需专用引燃装置 |
| 传质传热模式 | 全体系同步发生反应 | 燃烧波自蔓延模式 |
| 反应可控性 | 可控性强 | 可控性差 |
| 产物致密度 | 较高 | 较低 |
| 设备复杂度 | 简单 | 需引燃装置 |
| 适用体系类型 | 金属间低放热量体系均可 | 需达到较高绝热温度 |

XD法是为针对SHS法处理金属间低放热量体系无法达到高绝热温度的不足而专门开发的改进技术[[S6,S35]]。

### 与机械搅拌法的对比

| 对比维度 | 放热弥散法（XD） | 机械搅拌法 |
|---------|-----------------|-----------|
| 增强相来源 | 原位反应生成 | 外加颗粒混入 |
| 界面结合 | 界面洁净、结合强度高 | 润湿性差、易产生界面不良反应 |
| 颗粒分散均匀性 | 弥散分布均匀 | 均匀性控制难度大 |
| 典型缺陷 | 反应孔隙度较高 | 孔隙、颗粒团聚、氧化夹杂 |
| 设备成本 | 较低（但球磨耗能） | 成本极低、平台搭建简单 |
| 量产成熟度 | 母合金+稀释路线 | 成熟、具备批量生产能力 |
| 增强相体积分数上限 | ~30vol.% | ~30vol.% |

机械搅拌法可将5~100μm规格的陶瓷颗粒最高混入30%体积分数，可近净成形制备零件，但普遍存在增强相分散均匀性控制难度大、增强相与铝熔体润湿性差、易产生孔隙、颗粒团聚、氧化夹杂、额外界面不良反应等缺陷[[S15,S26]]。

### 与普通粉末冶金法的对比

普通粉末冶金工艺可在低于液相加工的温度下完成粉末混合、压制和烧结，基体与增强相间的反应程度更低，可稳定实现最高约50%体积分数的高含量增强相添加，但对混粉工序的控制精度要求极高[[S39,S37]]。

## 工程标准与工业化现状

### 配套标准

国内颗粒增强铝基复合材料相关通用基础标准包括GB/T 3880.2、GB/T 3190，覆盖6061等常用基体铝牌号的成分和性能要求[[S18]]。配套适用于XD工艺制粉环节的标准包括GB/T 5150-2004《铝镁合金粉》[[S31]]。

国外颗粒增强铝基复合材料常用通用配套ASTM标准包括ASTM B209（铝及铝合金板材和薄板规范）、ASTM B211（铝及铝合金棒、条、线规范）、ASTM B221（铝及铝合金挤压棒、条、线、型材规范）等，覆盖颗粒增强铝基复合材料不同形态制品的技术要求[[S18]]。

### 工业化应用

已公开的XD工艺工业化应用案例显示[[S27,S35]]：

- Duralcan等行业厂商量产的SiC颗粒增强铝基复合材料已广泛应用于汽车制动盘、自行车传动构件、高尔夫球头、网球拍等工业与体育用品领域；
- 美国相关研究机构采用XD工艺开发的TiB₂增强TiAl、TiB₂/NiAl金属间化合物基复合材料具备显著提升的高温压缩性能，已在航空航天高温结构件领域开展验证应用；
- Martin Marietta公司利用XD工艺结合熔模铸造工艺制成了TiB₂颗粒增强的NiAl基板材，日本也利用XD工艺制备出多种形状和尺寸的TiNi记忆合金及其他金属间化合物[[S17]]。

## 术语辨析

**XD™工艺**：属于专利限定的特定放热弥散类原位反应技术，区别于接触反应法、混合盐法等常规原位合成工艺，其核心特征为反应全程处于液态金属熔体介质中，依靠组元扩散完成全域同步放热反应，可先制备高增强相体积分数母合金再与常规熔铸工艺结合稀释至目标体积分数[[S35,S2]]。

**自蔓延高温合成（SHS）**：是放热弥散法的前身技术，依靠燃烧波自蔓延模式完成反应，需要专用引燃装置，适用于放热量较高的反应体系[[S6,S36]]。

**接触反应法**：是在综合SHS和XD法优点后发展起来的制备方法，将预制体直接置于高温金属熔体中，依靠熔体自身热量诱发反应[[S8]]。
