## 概述

合金线收缩率（alloy linear contraction coefficient / free linear contraction coefficient）是铸造合金在固态冷却阶段产生的线性尺寸缩减比例，属于铸造合金的核心工艺性能参数之一[[S63]]。根据 GB/T 5611—2017《铸造术语》定义，合金线收缩率（别名为自由线收缩率）是指合金在自由收缩条件下，从线收缩起始温度冷却至室温时，线尺寸的相对收缩量[[S58,S56,S11]]。其常用计算公式为[[S9]]：

\[
E_{\text{线}} = \frac{L_0 - L}{L_0} \times 100\%
\]

式中 \(L_0\) 为线收缩起始温度时试样的长度（mm），\(L\) 为冷却至室温后的长度（mm）。

## 收缩阶段与相关概念区分

合金从浇注温度冷却至室温的收缩过程分为三个连续的阶段[[S28,S48]]：

| 收缩阶段 | 温度区间 | 表现形式 | 与缺陷的关系 |
|---------|---------|---------|-------------|
| 液态收缩 | 浇注温度 → 液相线温度 | 体积缩减，液面下降 | 缩孔、缩松的形成根源之一 |
| 凝固收缩 | 液相线温度 → 固相线温度 | 体积缩减，与状态改变及温度下降相关 | 缩孔、缩松的直接成因 |
| 固态收缩 | 固相线温度 → 室温 | 三维线尺寸缩减 | 铸造应力、热裂、冷裂、变形的直接诱因 |

液态收缩和凝固收缩仅表现为体积缩小，不会引起宏观线尺寸变化；线收缩仅属于固态收缩阶段的产物，其物理起点并非固相线温度，而是枝晶搭接形成连续固相骨架的温度，即线收缩起始温度[[S58,S56,S33]]。固态体收缩率约为同等条件下合金线收缩率的 3 倍[[S29,S48]]。线收缩是铸件应力、裂纹和尺寸偏差的主要成因，体收缩则是缩孔、缩松的主要成因[[S28,S29]]。

## 分类

### 按约束状态分类

| 类型 | 定义 | 特点 |
|------|------|------|
| 自由线收缩 | 铸件冷却过程中不受铸型、型芯或结构约束，可完全自由进行的线收缩[[S58,S56,S49]] | 理论上的最大线收缩值，热裂风险低 |
| 受阻线收缩 | 铸件受到型芯、型腔结构、摩擦阻力或铸件自身温差引起的热阻力的机械阻碍，不能自由进行的收缩[[S58,S11,S49]] | 受阻线收缩值通常小于自由线收缩值，易引发铸造应力和热裂 |

实际压铸条件下的收缩状态可进一步细分为三类[[S13,S57]]：

- **自由收缩**：如无型芯阻碍方向上的尺寸收缩，收缩过程从压射完成持续到铸件冷却至常温。
- **阻碍收缩**：如包紧固定型芯方向的径向尺寸，在推出前基本不发生收缩，推出后才开始收缩。
- **混合收缩**：同时受到自由收缩和阻碍收缩综合影响的收缩状态，是压铸生产中最为常见的情况。

## 典型数值范围

不同合金在不同铸造条件下表现出显著差异的线收缩率[[S17,S14,S2,S43]]：

| 铸造方法 | 合金类型 | 线收缩率范围（%） |
|---------|---------|-----------------|
| 砂型铸造 | 铝合金（自由收缩） | 1.2～1.3 |
| 金属型铸造（砂芯） | 铝合金 | 0.8～1.0 |
| 金属型铸造（金属芯） | 铝合金 | 0.6～0.8 |
| 重力/低压铸造 | Al-Si 合金 | 0.5～0.75（共晶合金约 0.5） |
| 压铸 | 铝合金 | 约 0.5～0.6 |
| 压铸 | 镁合金 | 约 0.7 |

压铸条件下铝合金综合线收缩率显著低于砂型铸造，反映了高压约束对收缩的抑制作用[[S14,S2]]。

针对铝硅系压铸合金，不同收缩状态下的线收缩率参考值[[S44,S36,S24]]：

| 收缩状态 | 铝硅合金线收缩率（%） |
|---------|-------------------|
| 约束收缩（阻碍收缩） | 0.3～0.5 |
| 自由收缩 | 0.7～0.9 |
| 混合收缩 | 0.5～0.7 |

不同压铸合金的线收缩率从大到小依次为：铜合金 ＞ 镁合金 ＞ 铝合金 ＞ 锌合金[[S16]]。

## 影响因素

### 合金成分

合金成分是线收缩率的决定性内因。在二元合金系中，规律为[[S46,S53,S4]]：

- **纯金属和近共晶合金**：线收缩率最大。
- **共晶型合金**：线收缩率随第二组元浓度增加在共晶点前基本保持不变，到达共晶成分后开始降低。
- **固溶体型合金**：线收缩率随成分变化呈连续曲线形式变化。
- **结晶温度范围越宽的合金**：线收缩率越小。

线收缩率随合金成分的变化趋势可由平衡状态图与线收缩率的关系图直观说明：

![平衡状态下合金线收缩率与成分、状态图的关系](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/2a759465-b7b0-4a8d-bfd0-3d778d3f04af/resource)

该图标注了线收缩开始线 γ，清晰呈现共晶系、固溶体系等不同合金类型的线收缩率随成分的变化规律[[S10]]。

在压铸铝合金中，Si 含量对线收缩率的负相关效应极为显著：Si 含量越高，热膨胀系数和线收缩率越低[[S35,S6,S60]]。当 Si 质量分数达到 18%～20% 过共晶区间时，线收缩率达最低值[[S35]]；当 Si 质量分数达到 25% 以上时，线收缩率近似为 0%[[S6,S60]]。ADC1、ADC12 等共晶铝硅合金的线收缩率低于 ADC5 等亚共晶铝合金[[S36]]。

### 铸件结构与壁厚

- 壁厚越大，线收缩率越高；薄壁铸件的实测线收缩率低于厚大铸件[[S46,S35]]。
- 铸件结构越复杂、型芯越多，收缩所受的阻碍越大，实际线收缩率越小[[S16,S20]]。

### 铸造工艺条件

- **冷却速率**：任何金属在砂型中浇注的线收缩率最小，在急冷模（金属型）中线收缩率较大[[S46]]。压铸快冷条件下，凝固阶段仍可通过浇道持续补缩，大部分液态体收缩无法转化为宏观外部线收缩，内部体积收缩随冷却速率提升占比升高[[S15,S32]]。
- **浇注温度**：浇注温度愈高，线收缩反而减小[[S46]]。
- **模具温度**：模具温度越高，铸件与室温的温差越大，线收缩率越大[[S57,S16,S20]]。
- **压射比压与保压时间**：压射比压和保压时间的提升可促进液态金属补缩，在一定范围内降低铸件表观线收缩率，减少内部缩松缺陷[[S70]]。
- **模具热平衡**：压铸充型后型腔内部温度分布不均，靠近内浇口位置模温高，线收缩率较大；远离浇口位置模温低，线收缩率较小[[S57,S20]]。

## 标准测定方法

### 国内行业标准 JB/T 4022.1—1999

JB/T 4022.1—1999《合金铸造性能测定方法 自由线收缩测定方法》适用于测定铸钢、铸铁、铸造有色合金的自由线收缩率，核心技术要求如下[[S7,S30,S51,S61]]：

| 项目 | 要求 |
|------|------|
| 测试系统综合精度 | 不低于 1.5% |
| 空载最大机械静摩擦阻力 | ≤ 0.588 N |
| 铸型类型 | 湿型，水平整体型 |
| 造型材料 | 石英砂 + 膨润土粘结剂（不同合金对应不同砂粒度） |
| 铸型径向吃砂量 | 40～50 mm |
| 砂型紧实度 | 1.50～1.65 g/cm³ |
| 测试环境温度 | 10～30 ℃ |
| 相对湿度 | 30%～85% |
| 浇注温度 | 合金液相线以上 50～80 ℃ |
| 标距长度 | 200 mm |
| 计算公式 | εₗ = ΔL / 200 × 100% |

### 国际标准

ASTM B208 标准配套的铸造合金试样尺寸规范，广泛用于铜合金、铝合金等有色铸造合金拉伸性能及线收缩率的同步测定，采用金属型标准圆柱试样[[S21]]。

## 在压铸工艺中的作用

### 模具型腔尺寸计算

合金线收缩率是压铸模型腔尺寸设计的核心输入参数[[S57,S39,S66,S65]]。压铸模型腔尺寸计算公式为：

\[
D'_0^{+\Delta'} = (D + D\varphi - 0.7\Delta)_0^{+\Delta'}
\]

型芯尺寸计算公式为：

\[
d'_{-\Delta'}^0 = (d + d\varphi + 0.7\Delta)_{-\Delta'}^0
\]

式中 \(\varphi\) 即为选取的压铸件计算收缩率[[S39]]。压铸场景下，计算收缩率应采用综合线收缩率：

\[
K = K_1 - K_2
\]

其中 \(K_1\) 为铸件的线收缩率，\(K_2\) 为压铸模成形零件在工作温度下的热膨胀率，不能直接采用常规重力铸造的自由收缩测试数据[[S65]]。

### 尺寸精度控制

线收缩率选取的合理性直接决定压铸件的最终尺寸精度，是将铸件公差控制在国标 CT 等级要求范围内的核心影响因素[[S25,S68]]。设计时可根据铸件不同部位的收缩受阻情况差异化选取局部线收缩率[[S1,S31,S40]]。

### 热裂敏感性预测

合金在凝固脆弱温度区间内累积的线收缩变形量与热裂倾向性直接正相关[[S37,S45]]。热裂形成于凝固末期固液两相区，当凝固过程中收缩受阻产生的拉应力超过该温度下合金的高温强度极限，且残余液相无法补缩裂纹时，即产生热裂[[S8,S47,S23]]。合金的自由线收缩量越大，在脆弱温度区间内累积的变形量越大，对应的热裂倾向性越高。自由收缩无额外拉应力产生，热裂风险低；受阻收缩则在对应位置累积拉应力，是压铸场景下诱发热裂的核心诱因[[S57,S34]]。

### 铸造模拟输入

合金线收缩率是压铸铸造模拟软件材料数据库的必填输入参数，用于精准计算凝固过程中的尺寸变形、应力分布与缺陷预测[[S50,S55]]。

## 与缺陷的关联

| 缺陷类型 | 与线收缩的关系 |
|---------|--------------|
| 热裂 | 线收缩受阻产生的拉应力超过合金高温强度极限时发生；线收缩率越大，热裂倾向越高[[S8,S47]] |
| 缩孔、缩松 | 收缩阶段液态金属补缩路径被阻断时，无法通过外部尺寸收缩抵消的体积收缩在内部形成空穴[[S34,S54]] |
| 尺寸偏差 | 线收缩率选取不准确直接导致压铸件尺寸超出公差范围[[S44]] |
| 铸造应力与变形 | 固态阶段不同部位的线收缩差异引起热应力、收缩应力，可能导致变形或冷裂[[S29]] |

## 局限性

1. **实测数据的局限性**：通用实测线收缩率通常采用自由收缩的无约束试样，无法体现实际生产中普遍存在的受阻收缩工况[[S16,S26]]。实测数据受试样形状、冷却条件和约束状态影响，不能直接等同于铸件的实际尺寸收缩[[S17,S43]]。

2. **温度场不均的影响**：通用测试数据未考虑实际压铸型腔内部的温度分布不均特点——靠近内浇口位置模温高，远离浇口位置模温低，不同部位对应的线收缩率存在明显差异，通用单一数值无法匹配同一铸件不同位置的实际收缩特性[[S57,S20]]。

3. **壁厚差异的影响**：通用测试数据未覆盖不同壁厚铸件的冷却速率差异所对应的线收缩率变化规律，无法适配实际复杂结构压铸件不同壁厚特征的差异化收缩表现[[S16,S62]]。

4. **工艺方法的差异**：压铸条件下快冷和高压注射会使金属凝固阶段仍可通过浇道持续补缩，外部宏观线收缩的表现形式与重力铸造存在显著差异[[S15,S32]]。

## 应用场景

- **模具尺寸补偿设定**：设计人员可按统一线收缩率对铸件三维模型进行整体放大，也可针对不同收缩受阻条件的特征选取差异化线收缩率分别补偿[[S1,S31,S40]]。
- **浇冒口系统设计**：线收缩率是确定补缩通道尺寸、溢流槽体积与布置位置的重要参考依据[[S34,S71]]。
- **铸造模拟**：作为材料数据库的必填参数，用于凝固变形、应力分布和缺陷预测[[S50,S55]]。
- **工艺方案优化**：作为核心参考指标，辅助优化压射比压、保压时间、模具温度等工艺参数，实现线收缩行为的可控化[[S22,S50]]。
