## 定义

未浇满（poured short）是铸件残缺类缺陷之一，指铸件上部产生缺肉，其边角略呈圆形，且浇冒口顶面与铸件平齐[[S6,S10,S23,S18]]。该缺陷归属于GB/T 5611-2017《铸造术语》9.6节“残缺类缺陷”大类下的独立子类，与浇不到（浇不足）、型漏、跑火、损伤并列，不属于孔洞类、表面类或裂纹冷隔类缺陷[[S17,S13,S22]]。在国际WFO/ISO TR 16078缺陷分类体系中，未浇满对应残缺类E分类下的“Unvollständige Formfüllung”（缺陷代码E122）[[S21,S11]]。

该缺陷属于不可逆缺陷，一旦出现铸件无法补救，必须报废处理[[S24,S2]]。

## 宏观外观特征

未浇满缺陷具有下列典型宏观特征[[S24,S10]]:

- 缺肉部位仅出现在铸件上部轮廓，而非远端或薄壁处；
- 缺失部位边缘边角略呈圆形；
- 浇冒口顶面与铸件缺失面平齐，不存在对接熔合缝隙或远端薄壁处轮廓缺失特征；
- 缺肉表面通常呈现自然液面状凝固形态。

## 形成机理

压铸过程中未浇满的形成核心机理为：金属液浇注总重量不足，或浇注系统/内浇口部位率先发生凝固，导致压射冲头压力无法将压室内剩余金属液补充输送到铸件上部型腔，最终上部型腔未被金属液完全填充，压块处金属无法完成对铸件上部的补料[[S8]]。

## 主要影响因素

### 工艺参数

| 影响因素 | 具体要求/风险 |
| --- | --- |
| 压室充满度 | 低于70%~80%的推荐范围，压室上部空间过大导致金属液包卷气体严重，金属液在压室内被激冷，对充型不利[[S20,S7]] |
| 浇注温度 | 浇注温度比合金液相线高出不足10~20°C时，金属液流动性降低[[S7]] |
| 模具温度 | 模具预热温度过低，金属液流动前沿过早凝固[[S7]] |
| 压射速度/比压 | 压射速度低于工艺阈值，或压射比压不符合工艺要求[[S7,S20]] |
| 浇料定量 | 每一模次浇注的金属液重量不准确，总浇入量不足[[S20]] |

### 模具设计

- 内浇口截面积不足或位置设置不合理，形成大的流动阻力[[S1,S7]]；
- 排溢/排气系统设计不当，型腔内气体无法及时排出[[S1]]；
- 深腔部位未设置对应排气塞、溢流槽容积不足[[S1]]。

### 合金特性

- 合金熔体流动性差[[S14]]；
- 合金成分不符合标准导致液相线温度过高[[S14]]；
- 熔炼过程中金属液氧化严重[[S14]]。

## 与相似缺陷的区分

### 未浇满 vs. 浇不足（浇不到） vs. 冷隔

| 鉴别项目 | 未浇满 | 浇不足（浇不到） | 冷隔 |
| --- | --- | --- | --- |
| 缺陷位置 | 铸件上部轮廓缺肉 | 远离浇口的部位及薄壁处 | 远离流道的铸件表面和薄壁处 |
| 外观形貌 | 缺肉边缘边角略圆形，浇冒口顶面与铸件平齐 | 铸件残缺或轮廓不完整，或轮廓完整但边角圆且光亮 | 两股金属流对接形成穿透或不穿透的线性缝隙，边缘圆角 |
| 浇注系统状态 | 浇冒口未充满 | 浇注系统完全充满 | 与浇注系统充满状态无直接关联 |
| 形成原因 | 总浇入量不足或内浇口过早凝固，补料无法到达上部 | 金属液流动性不足、压力不足、远端过早凝固 | 两股金属流温度过低，对接时未完全熔合 |
| 可补救性 | 不可补救，必须报废[[S24]] | 不可补救 | 视严重程度可焊补或报废 |
| 标准编号 | GB/T 5611-9.6.3 | GB/T 5611-9.6.2 | GB/T 5611 裂纹冷隔类 |

### 未浇满 vs. 缩孔

缩孔属于孔洞类缺陷，出现在铸件厚断面内部或表面收缩凹陷处，铸件外部轮廓完整，不存在缺肉缺失，与未浇满成因及外观完全不同。

### 大面积薄壁压铸件缺陷对照

| 缺陷名称 | 缺陷描述 |
| --- | --- |
| 浇不足 | 铸件成形过程中填充不完整，表面形状为自然液面状[[S5]] |
| 翘曲变形 | 铸件因铸造应力或残余应力产生的形变 |
| 冷隔 | 温度较低金属流对接未熔合而产生的缝隙 |
| 裂纹 | 铸件本体出现的连续性断裂 |

## 预防对策

1. **定量浇注控制**：严格执行压室70%~80%的充满度要求，精确控制每一模次浇入的金属液重量，确保包括铸件、浇道系统、排溢系统和余料饼在内的总金属液量充足[[S20,S1]]。
2. **温度参数优化**：适当提升浇注温度（高于液相线10~20°C）与模具预热温度，保证合金流动性[[S7,S1]]。
3. **模具设计改进**：优化内浇口截面积和位置，保证充填流道通畅；加强排溢系统设计，深腔部位设置排气塞、增大溢流槽容积[[S1,S7]]。
4. **设备状态维护**：定期校准压铸机压射压力与压射速度，检查冲头密封状态[[S1]]。
5. **合金质量控制**：严格控制合金成分，防止熔炼过程氧化[[S14]]。
