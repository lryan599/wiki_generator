## 概述

在压铸领域，“硬点”（hard spot）也称为“硬质点”，是指压铸件中硬度显著高于正常基体的局部异常区域[[S2,S11,S10]]。该术语最早由日本学者大日方在1934年前后提出，用于描述铝合金压铸件在切削加工中导致刀具异常快速磨损的硬质夹杂物[[S2]]。硬点是压铸件典型的材质缺陷之一，主要形成于熔炼、保温及充型过程中[[S13]]。在压铸场景下，“硬点”术语默认指向硬点缺陷，现行通用资料中不存在与之对应的非缺陷类结构术语定义[[S2,S13]]。

## 硬点分类体系

根据压铸行业主流权威分类，硬点共分为四大类[[S2,S9,S5,S14]]：

| 类别 | 子类/成因 | 核心特征与成分 |
| :--- | :--- | :--- |
| **非金属性硬点** | 非金属夹杂物、合金液表面氧化物、合金与炉衬/耐火材料反应物、砂/耐火砖粉等混入异物 | 以Al₂O₃、SiC、硅酸盐炉渣等高硬度非金属相为主[[S9]] |
| **金属性硬点** | 未熔解硅元素、初晶硅、Fe/Mn系金属间化合物 | 以游离硅、初晶硅、Fe-Mn-Si金属间化合物为主[[S2]] |
| **复合性硬点** | 非金属基体中复合包裹大量金属相大结晶 | 兼具非金属与金属性硬点特征，如筑炉材料与铝液反应生成的结晶状物[[S1,S2]] |
| **偏析性硬点** | 充型中飞散的细小合金液滴急冷凝固 | 组分偏析，形成急冷（冷硬）组织，合金成分偏析率显著高于基体[[S7,S15]] |

## 各类硬点特征与辨识

### 非金属性硬点
非金属性硬点主要由金属氧化物、碳化物、硫化物等非金属化合物构成[[S9]]。最典型的是铝氧化物（α-Al₂O₃），其在熔融状态下形成，密度与铝液差异小，易悬浮并随熔体进入型腔，最终成为硬点[[S11,S9]]。其显微维氏硬度范围为 200–1000 HV[[S9]]。

### 金属性硬点
金属性硬点以未熔或析出的高硬度金属相为核心。最早观测到的典型金属硬点呈球状/粒状时尺寸约0.1 mm，板状/片状时可达0.3 mm以上，莫氏硬度达6–7级，显微维氏硬度在 300–500 HV 以上[[S2]]。

### 复合性硬点
复合硬点是在非金属性基体中同时包含大量金属性大结晶的混合型硬点[[S2]]。其形貌与成因复杂，常涉及熔体与筑炉材料的化学反应产物。

![复合硬点微观形貌实拍图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/01f6070e-cce4-4290-9f72-fa2ec2a600c4/resource)

### 偏析性硬点
偏析性硬点，又称急冷性硬点，是压铸过程特有的缺陷[[S15]]。其成因是充型时高速喷射的合金液雾化为细小液滴，与型腔壁撞击或急速飞散后快速凝固，导致局部成分偏析，形成晶粒极细的急冷组织[[S15,S7]]。

- **硬度范围**：显微维氏硬度为 110–200 HV，硬度仅为正常基体的1.1–1.7倍，属于“较软”的硬点，一般不会严重损坏刀具[[S15,S7]]。
- **分布与外观**：常分布在合金液流前端最先到达的型腔区域，加工面上呈现与正常基体光泽不同的亮斑或暗斑，宏观上易被误判为金属异物[[S15,S4]]。
- **微观组织**：金相组织与正常铝合金基体相同，但组织更细小致密。硬点与正常组织之间的边界多样，可呈清晰边界、绳状边界、细线状边界，甚至没有明显边界[[S7]]。硬点的致密度越高，其硬度也越高，合金成分（如Si、Cu）的偏析率也越大[[S7]]。

## 检测与判定方法

硬点的主要检测与判定手段如下[[S4,S8]]：

1. **加工后外观检查**：这是最直观的初步判别方法。硬点所在的加工面会显现出与正常金属基体光泽和亮度不同的异色区域（亮斑/暗斑）[[S4,S10]]。
2. **显微硬度检测**：这是判定硬点的关键定量方法。通过对比疑似硬点与周围正常基体的维氏硬度，可确认是否为硬点缺陷。正常铝合金压铸基体的显微维氏硬度约为 80–100 HV[[S4]]。
3. **金相观察**：对截面进行抛光、腐蚀后，通过显微镜观察微观组织差异，可区分各类硬点（如急冷组织、氧化物夹杂、未溶初晶硅等），同时便于识别其形态和边界特征[[S7,S8]]。
4. **无损检测局限**：常规X射线与超声波探伤仅能识别尺寸较大、致密度较高或与基体存在明显密度差的硬点，对于尺寸细小、致密度低的多数硬点难以有效识别[[S4,S8]]。

**标准与分级**：
目前，国内及国际压铸通用标准中尚无统一的硬点分级条文。硬点的允许数量、尺寸及合格判定标准，均由供需双方根据铸件的使用要求、加工精度等协商制定，属于定制化验收范畴[[S3]]。

## 危害与后果

硬点的危害主要集中在铸造后的机械加工环节[[S11,S6]]：

- **加剧刀具磨损**：硬点的高硬度物质在切削过程中，如同磨料一样，会显著加速刀具刃口的磨损速率。
- **导致刀具崩刃破损**：对于硬度极高或尺寸较大的硬点，在切削力冲击下可能导致刀具刃口发生崩碎、破损，造成加工过程失稳[[S6,S11]]。
- **影响加工表面质量**：硬点在加工过程中可能被刀具从基体中拖拽剥离，形成不规则的微小凹坑，或者在表面留下划痕，严重劣化加工表面的质量和完整性[[S11,S4,S10]]。
- **劣化局部力学性能**：高硬度、大尺寸且与基体结合不良的硬点，会成为应力集中点，对铸件的局部力学性能造成潜在不利影响[[S6]]。

## 预防与控制对策

针对不同成因的硬点，需采取系统性的预防与控制措施[[S12,S9,S10]]。

### 1. 熔体净化管控
- 熔炼与保温过程中充分进行除气、除渣精炼处理[[S9,S10]]。
- 浇注及舀取合金液时，严格避免卷入液面氧化皮[[S12,S11]]。
- 定期清理保温炉（坩埚）底部沉渣和炉壁残渣[[S12,S4]]。
- 选用与铝合金液不发生反应或反应性极低的高质量炉衬、耐火材料[[S12,S9,S6]]。

### 2. 合金成分与原料管控
- **硅元素添加**：配制Al-Si合金时，严禁直接使用纯硅粉或单质硅块，必须使用铝-硅中间合金进行成分调整[[S12,S9]]。
- **硅含量控制**：控制Al-Si系合金的硅含量远离共晶点（建议 ≤ 10.5 wt%），以抑制初晶硅的过度析出[[S12,S10]]。
- **杂质元素控制**：严格控制Fe、Mn等容易形成高硬度金属间化合物的杂质元素含量，防止其在熔体中偏析[[S12,S10]]。
- 确保所有入炉原材料（包括新料、回炉料）的洁净度，回炉料应去除油污，避免混入异物[[S4,S6]]。

### 3. 熔炼与保温工艺管控
- 适当提高熔化温度与浇注温度，保证硅元素及其他合金化元素充分溶解[[S12,S10]]。
- 避免合金液在保温炉内长时间静置，以防成分偏析和炉底沉淀物的生成[[S4]]。
- 熔炼操作应保持稳定，确保每批次合金液的成分均匀、洁净[[S13]]。

### 4. 浇注与充型过程管控
- 优化浇注系统与充型流道，避免充型过程中金属液产生过度湍流、飞溅和雾化，减少急冷孤立颗粒的生成[[S10]]。
- 选用设计合理的压射工艺参数，确保充型过程平稳有序[[S10]]。
