## 概述

固定式落砂机，全称固定式惯性振动落砂机，是砂型铸造后处理环节中的关键落砂设备。依据国家标准GB/T 25370-2020《铸造机械 术语》，其标准定义为振动方向角为90°的惯性振动落砂机，亦称为振动栅格（knock-out grid；vertical vibratory shakeout）[[S4]]。该设备通过纯垂直方向的振击作用，使铸件、铸型和/或砂箱在栅格上不发生水平运动，完成型砂与铸件的分离作业[[S4]]。

需要明确指出的是，在压铸领域，固定式落砂机并非用于浇口或渣包去除的专用设备。压铸场景下的浇口去除属于 degating 工序，对应设备为液压浇口切割机、浇口液压钳等专用切边设备，与固定式落砂机属于完全不同的设备分类[[S1,S9,S12]]。

## 工艺作用

在砂型铸造流程中，固定式落砂机是连接造型、砂处理和铸件清理三大环节的枢纽设备[[S18]]。其工艺位置处于浇注冷却工序之后，前道对接捅箱机或翻箱出箱工位，将铸型整体置于落砂栅台上；后道工位则分别输出裸铸件进入清砂、抛丸、打磨工序，同时将破碎下落的型砂送入旧砂回收处理系统[[S18,S3]]。

落砂作业的主要功能包括[[S3]]：
- 使铸件从型砂中分离出来；
- 破碎结块型砂；
- 去除砂箱内壁和铸件表面的粘砂；
- 尽可能地去除残留砂芯。

标准作业时间根据铸件规格有所差异：中小型铸件落砂时间取1～5 min，大吨位铸件取5～10 min，特大重型铸件取10～20 min[[S13]]。

## 设备结构

固定式落砂机的整机基座固定安装于地面，运行时整机无整体位移，激振力仅垂直作用于工作台面，不会推动铸型沿台面产生水平输送运动[[S4,S18]]。其核心组成部件包括[[S18,S13,S17]]：

| 部件名称 | 功能说明 |
|---------|---------|
| 振动框架/机架 | 具备足够抗振刚度，承载全部激振部件与工作台面载荷，为整机结构主体 |
| 激振单元 | 核心振源，分为双轴激振器和振动电机两种主流形式，依靠反向同步旋转的偏心块产生纯垂直方向激振力，支持按需调节 |
| 落砂栅格/工作台面 | 带格栅的工作面，承载砂箱与铸型，允许破碎型砂通过栅格下落，同时阻拦铸件留在台面上 |
| 弹性支承/减振弹簧 | 采用圆柱螺旋弹簧、橡胶弹簧等材质，用于吸收振动能量，降低传递至设备基础的动载荷 |
| 基座/座架 | 整机固定安装的承重底座，下方可集成旧砂漏斗直接收集下落的型砂 |

## 驱动形式与技术参数

国内主流产品为L12系列固定式双轴惯性振动落砂机，共有四类驱动形式[[S13,S16]]：
1. 单台电动机驱动单根激振轴；
2. 两台电动机独立驱动双激振轴（无强迫联系）；
3. 单台电动机经V带及齿轮同步驱动双激振轴（强迫联系）；
4. 振动电动机直接驱动。

无锡锡南铸造的XNL系列与济南铸锻所的L12D系列均等效采用L12系列参数基准[[S16]]。设备允许2台或4台并联组合使用，适配大吨位铸型落砂[[S13]]。

通用参数匹配规则方面，对于远共振工况的固定式落砂机，机械指数取值范围为1.5～3[[S17]]。频率及振幅的选型应根据铸件材质调整[[S13,S17]]：

| 铸件材质 | 参数配置方向 | 选型依据 |
|---------|------------|---------|
| 铸钢件 | 高振幅、低频率 | 砂型紧实度高，落砂难度大 |
| 铸铁件 | 低振幅 | 铸件易受损坏，需避免高冲击 |
| 非铁合金（铝合金、铜合金等） | 低振幅 | 薄壁铸件需防磕碰变形 |

## 与其他设备的区别

固定式落砂机的核心特征是振动方向角为90°，即仅产生垂直方向振动，无铸型输送能力[[S4]]。与其他同类或相关设备的区分如下[[S4,S25,S18]]：

| 设备类型 | 振动输送功能 | 落砂功能 | 主要应用场景 |
|---------|------------|---------|------------|
| 固定式落砂机 | 无水平输送，仅垂直振击 | 有，专门落砂 | 砂型铸造型砂分离 |
| 输送式落砂机 | 有，振动方向角＜90°，边落砂边输送 | 有 | 连续化造型线 |
| 振动输送机 | 有，仅水平输送 | 无，不承担落砂功能 | 铸型、铸件、旧砂转运 |
| 落砂滚筒 | 有，筒内旋转输送 | 有，兼有冷却功能 | 无箱射压造型线 |
| 移动式落砂机 | 底部配行走轮组，可移动至不同工位 | 有 | 需要灵活调度的工位 |
| 悬挂式落砂机 | 沿悬挂轨道移动 | 有 | 就地清砂、特殊大件落砂 |
| 压铸专用切边机（trim press） | 无 | 无（仅去除浇口、飞边） | 压铸件浇口切除 |

## 产线布局与协同

在砂型铸造自动化产线中，固定式落砂机的典型布局可分为三类[[S11,S10]]：
1. **非机械化/小批量车间**：单机独立布置，直接对接地面冷却后的人工转运铸型；
2. **半自动造型线**：串联布置在铸型输送机末端，出箱工位输出的砂型直接输送至落砂工位；
3. **全自动高压造型线**：常配置多台落砂机并联组成落砂机组，对接鳞板式铸件输送工位，适配大产能连续生产。

产线协同逻辑为[[S18]]：固定式落砂机通常布置在捅箱机下游，承接脱箱后的高温铸型完成集中振击落砂；分离出的旧砂向下游输送至带式磁选机、砂块破碎机、振动筛等砂处理单元，分离出的铸件则转运至鳞板输送机进入清理工序。

## 优缺点分析

**核心优势**[[S13,S18]]：
- 设备整机刚性固定于基座，振击载荷可被基础有效分散，运行稳定性显著高于移动式设备和悬挂式设备；
- 可承受大吨位重型铸型的高载荷振击需求，适配铸钢件、大型铸铁件等高难度落砂工况；
- 多台并联可组成落砂机组，实现特大铸件的落砂作业，广泛应用于工程机械、矿山设备、电站装备等行业的重型铸件生产。

**局限性**[[S4]]：
- 无物料连续输送功能，作业过程中铸型需由起重设备或转运工装在固定工位上下料；
- 无法适配连续流转的全自动化造型线；
- 场景适配灵活性弱于可自由移动的移动式落砂设备，设备迁移改造成本远高于悬挂式落砂装置。

## 安装与粉尘控制

**基础安装要求**[[S18,S8]]：
- 整机支承采用减振弹簧（圆柱形螺旋弹簧/橡胶弹簧组合）实现隔振，全部振动力经弹簧隔离，避免振动向厂房地基扩散；
- 落砂机栅台与配套固定撞击架的间隙控制在6～8 mm范围内，适配冲击式落砂工艺需求。

**粉尘收集控制方案**[[S8,S20]]：
- 落砂工位须设置密闭防尘罩配合侧吸式或上吸式吸尘罩；
- 对接干式布袋除尘系统，适配落砂高扬尘工位的废气处理需求，防止粉尘外溢；
- 大于7.5 t的落砂机宜配置移动式密闭罩，实现降噪抑尘[[S14]]。

## 安全与技术规范

### 行业标准与安全要求

固定式落砂机需遵循以下现行标准：
- **JB/T 6575-93《落砂机技术条件》**：规定了固定式落砂机的核心技术要求，包括落砂效率不低于90%、空载噪声声压级不得高于93 dB(A)、首次大修期不低于6000 h、减振弹簧寿命不低于2000 h等强制指标[[S22]]。
- **GB/T 25370-2020《铸造机械 术语》**：明确界定了固定式惯性振动落砂机及相关落砂设备的标准术语与分类边界[[S4]]。
- **GB 20905-2007《铸造机械 安全要求》**：为整机安全设计及防护配置提供通用强制约束[[S7]]。

### 运行安全规定

根据JB/T 6575-93标准要求[[S22]]：
- 过共振区运行的固定式落砂机必须配备可靠的制动或限幅装置，保证停机过程快速平稳；
- 振动电机轴承工作过程中温升不得超过35℃，最高工作温度不得高于70℃；
- 减振弹簧需符合GB 1239.4标准的2级精度要求，表面须进行强化处理。

### 日常点检核心项目

固定式落砂机的日常点检应重点关注以下内容[[S22,S24]]：
1. 检查所有紧固连接点位的防松状态，避免运行过程中松动断裂；
2. 检测设备两侧对称点的垂直振幅偏差，其差值不得超过两点平均双振幅的20%，侧向偏摆量不得超过2 mm；
3. 检查同一基准高度所有减振弹簧的水平度，偏差不得超过1 mm；
4. 检查落砂栅格板的磨损与变形情况，及时更换断裂或过度磨损的栅条。
