## 氧化夹杂的术语边界

氧化夹杂与以下相关概念需明确区分：

| 术语 | 关系 | 说明 |
|------|------|------|
| **夹渣（slag inclusion）** | 上下位关系 | 夹渣涵盖氧化物、熔剂渣、耐火材料残屑等多类组分；由纯氧化物构成的夹渣属于氧化夹杂的子类；普通熔剂夹渣、砂眼等不属于氧化夹杂 [[S23,S79]] |
| **氧化膜（oxide film）** | 位置关系 | 金属液表面未被卷入熔体的完整固态氧化层，不属于内部氧化夹杂；仅当其被紊流裹挟进入熔体内部残留于铸件中时，才成为氧化夹杂的组成部分 [[S24,S65]] |
| **双膜缺陷（bifilm）** | 特殊亚类 | 由浇注紊流作用下金属液表面的固体氧化膜干面与干面接触折叠卷入熔体内部形成，两层氧化膜之间无冶金结合，属于未被完全压实的特殊双层氧化夹杂 [[S70,S15,S75]] |
| **浮渣（dross）** | 部分交叠 | 金属液表面富集的氧化产物裹挟大量金属微滴形成的渣层；未被卷入铸件的部分属于液面渣层，随紊流卷入铸件的dross内部的氧化组分属于氧化夹杂的组成部分 [[S3,S23]] |

## 分类体系

### 按生成时序分类

氧化夹杂按生成时序分为两大类 [[S65,S32]]：

| 类别 | 定义 | 特征 |
|------|------|------|
| **初次夹杂（一次氧化夹杂）** | 熔炼过程、浇注前已生成的氧化夹杂，包括炉料带入的氧化物、脱氧反应产物等 | 分布无明显规律 |
| **二次夹杂（二次氧化夹杂）** | 浇注、充型过程中，金属液表面氧化膜被紊流卷入熔体残留形成的氧化夹杂 | 多分布在铸件上表面、型芯下表面或铸型死角位置；据统计，钢铸件中二次氧化夹杂物占总夹杂量的40%~70% |

镁球墨铸铁一次与二次氧化夹杂物的组成存在显著差异：一次夹杂中MgO占比更高，二次夹杂中SiO₂占比相对更高，该特征可作为识别氧化夹杂生成时序的重要依据 [[S52,S61]]。

### 按形态分类

氧化夹杂按形态可分为薄膜状（如铝合金中的Al₂O₃氧化膜、镁合金的MgO皮膜）、点状/球状、多面体状、不规则多角形、链状、板条状等多种类型 [[S71,S30,S24]]。

### 按成分分类

按成分维度，氧化夹杂可分为三类 [[S10,S30,S76]]：

| 类别 | 典型物相 | 特点 |
|------|----------|------|
| **简单氧化物夹杂** | FeO、Al₂O₃、SiO₂、MgO、MnO、Cr₂O₃、TiO₂等 | 单一氧化物组分 |
| **复杂尖晶石类氧化物夹杂** | MgO·Al₂O₃、FeO·Al₂O₃等 | 通式AO·B₂O₃，A为二价金属，B为三价金属 |
| **硅酸盐类复合氧化物夹杂** | lFeO·mMnO·nAl₂O₃·pSiO₂等 | 熔点较低，常以球形液滴状或玻璃态存在 |

### 按尺寸分类

以铝合金氧化夹杂为例，按尺寸可分为三类 [[S24,S72,S22]]：

| 尺寸范围 | 特征 | 可去除性 |
|----------|------|----------|
| >20 μm 大块氧化夹杂 | 分布不均匀，危害性高 | 可通过常规熔体净化去除 |
| 10~20 μm 细片状氧化夹杂 | 分布相对均匀 | 可通过熔体净化去除 |
| <10 μm 弥散微片状氧化夹杂 | 大部分悬浮于熔体中 | 难以通过常规净化手段完全去除，是影响合金冶金质量的核心因素 |

## 形成机理

### 热力学条件

氧化夹杂生成的通用热力学判据：对于反应 m[Me] + n[C] ⇌ MeₘCₙ，标准吉布斯自由能 ΔG⁰ = A + BT，ΔG⁰ < 0 时反应可自发进行，ΔG⁰ 负值越大则对应氧化物稳定性越高。非标准状态下需用 ΔG_T = ΔG⁰ + RT·ln(a_C^c·a_D^d/(a_A^a·a_B^b)) 判断，ΔG_T < 0 时可自发生成氧化夹杂 [[S51]]。

### 动力学条件

氧化夹杂生成需同时满足热力学条件（温度降低使反应平衡向生成夹杂方向移动）以及元素在金属基体中的扩散条件。冷却速率越慢，金属与夹杂反应时间越充分，生成的夹杂成分越接近平衡态。当金属液与夹杂之间界面的 ΔF < 0（即 σ_IC < σ_LI + σ_LC）时，夹杂可优先附着于晶体表面，反之则被晶体排斥，最终偏聚于晶界 [[S44]]。

### 二次氧化夹杂的生成机制

金属液浇注充型过程中，表面生成的氧化膜在湍流作用下被卷入金属液内部，若无法上浮逸出则残留在金属中成为二次氧化夹杂 [[S65]]。其生成受三个核心因素影响：

- **化学成分**：合金中元素的氧化活性和含量决定优先形成的氧化物类型；
- **液面条件**：铸型中水分蒸发形成的氧化性气氛加速二次氧化；
- **液流特性**：平稳层流时氧化膜卷入概率小，紊流、涡流或飞溅时卷入概率显著增大 [[S32]]。

### 压铸工艺的特殊影响

压铸充型过程中，当金属液韦伯数 We > 1 时，表面张力无法约束金属流动，易出现破波、飞溅与卷吸行为，将表面氧化膜卷入液体内生成双层氧化膜（bifilm）类夹杂。充型入口处流速超过 0.5 m/s 时出现喷射与液面失稳，显著提升氧化夹杂生成概率 [[S15,S7,S85]]。

不合理的浇注系统布局会加剧液流紊流程度，无法实现有效撇渣；设计中若采用高于 100 mm 的大落差顶注工艺，会加剧金属液飞溅、卷气，最终生成大量氧化夹杂 [[S88]]。

真空度不足时，型腔内部残留的空气无法在充型过程中被完全排出，与高温流动的金属液充分接触发生氧化反应，紊流卷入会将生成的氧化产物裹挟至铸件内部，最终导致氧化夹杂与气孔共生缺陷 [[S33,S87]]。

### SLM工艺的特殊生成机制

即使成形舱内填充惰性保护气，若舱内残留氧含量约为 0.1%，仍会在高温熔池作用下生成 Al₂O₃ 等多边形氧化颗粒。氧在液态金属中的溶解度随温度升高而提升，溶解度计算公式为 lg[O]_max = -6320/T + 2.734。生成的氧化夹杂会阻碍金属原子扩散，劣化后续成形性，引发球化缺陷，降低层间结合强度与致密度，诱发裂纹 [[S2,S41,S48]]。

### 铝合金中Al₂O₃的相变特性

Al₂O₃在铝合金熔炼过程中的相变序列：室温下为非晶态；450℃以上转变为 γ-Al₂O₃；750℃开始向 α-Al₂O₃ 转变；900~1100℃经过过渡相；1200℃完全转变为致密的刚玉型 α-Al₂O₃ [[S82]]。

铝镁合金中，Mg的化学活性高于Al，优先被氧化生成MgO。当铝基体中Mg质量分数 > 1.0% 时，合金表面氧化膜完全由MgO构成，组织疏松无保护作用。MgO与Al₂O₃在500℃以上可进一步反应生成MgAl₂O₄尖晶石相 [[S56,S81,S40]]。

## 形态与分布特征

### 典型分布

二次氧化夹渣常分布在铸件上表面、型芯下表面、铸型死角区域。大尺寸氧化夹杂易上浮至铸锭头部聚集，细小夹杂若来不及上浮会在凝固过程中嵌入晶内或偏聚于晶界。厚大截面铸件、最后凝固区域中易聚集大量未排出的氧化夹杂 [[S65]]。

### 宏微观形貌

**宏观特征**：多为无固定形状的黑色凹坑，与基体无清晰分界，断口中呈现黑色条状、块状、片状 [[S66,S13]]。

**微观特征**：显微组织中多为絮状的黑色紊乱组织，由黑色线条构成，与银白色基体色差显著。微观下可分为薄膜状、点状、不规则孔洞状、条状等多种形态。铝熔体中由回炉料表面氧化膜分解生成的氧化夹杂常呈现絮状聚集分布特征 [[S66,S13]]。

### 与气孔、缩松的共生机制

氧化夹杂具备吸附氢与水分子的能力，可作为氢气泡非均质形核的核心，大幅降低氢析出所需的形核功。当金属液凝固降温过程中氢处于过饱和状态时，氢优先在氧化夹杂表面析出形成气泡。若气泡在枝晶间无法完全长大上浮，最终会共同形成氧化夹杂、气孔、显微缩松三者共生的缺陷。氧化夹杂含量越高，对应生成的显微缩松数量越多，合金中氢含量也随之升高 [[S20]]。

## 对材料性能的影响

### 力学性能劣化

铸件中尺寸达到 20~30 μm 以上的大块氧化夹杂，会直接割裂金属基体连续性，是铸件发生渗漏的核心起源点，同时作为应力集中源大幅降低铸件的伸长率、冲击韧度和疲劳性能 [[S47,S59]]。

5A03和3A21铝合金铸锭的实测数据表明，随着氧化夹杂面积占比从 0 逐步提升至 4.5%，合金的抗拉强度、屈服强度均出现不同程度下降，伸长率随夹杂占比升高发生显著降低 [[S12,S14]]。

含氧化夹杂和孔隙的普通压铸Al-Mg-Si-Mn合金试样抗拉强度为 321 MPa、伸长率为 3.9%，同成分真空压铸试样因氧化夹杂更少，抗拉强度可达 331 MPa、伸长率达 4.4%，且真空压铸试样的疲劳寿命离散度明显更低 [[S38]]。

### 疲劳裂纹源

工程统计显示，90% 的汽车零件疲劳断裂案例中，裂纹萌生源都可追溯到非金属氧化夹杂。氧化夹杂作为高应力集中的核心起源点，会大幅加速微裂纹扩展进程 [[S10]]。

弥散夹杂物（平均颗粒尺寸在 10~20 μm 以下）使合金液的粘度增加，降低合金液的流动性（特别是凝固时的补缩能力），阻滞气体的扩散和析出，促使铸件形成气孔。皮膜状氧化铝若混入合金液中，会在铸件凝固中成为气泡的成核中心，是产生气孔的最危险的夹杂物 [[S11]]。

## 检测与识别

| 方法 | 原理 | 适用性 | 局限性 |
|------|------|--------|--------|
| **金相显微镜观测** | 试样经粗磨、精磨、抛光、腐蚀后在 100~500 倍下观测，通过暗黑色、不规则形貌特征识别氧化物夹杂 | 检测成本低、操作便捷 | 破坏性抽检，无法实现全零件无损检测 [[S19]] |
| **断口分析+EDS能谱** | 在拉伸或疲劳断口的裂纹起源位置观测典型氧化夹杂形貌，结合EDS能谱通过氧元素特征峰判定缺陷为氧化物 | 可追溯铸件断裂根因 | 破坏性检测 [[S39,S89]] |
| **X射线探伤** | 利用不同密度物质对X射线的差异化衰减特性，在底片上呈现与基体密度存在差异的氧化夹杂的投影图像 | 适用范围广，可得到位置、形状、大小信息 | 对与基体密度接近的薄层氧化膜识别灵敏度较低 [[S74]] |
| **工业CT** | 通过三维断层扫描重建氧化夹杂的完整三维形貌，最高分辨率可达 0.5 μm，可精准测量夹杂的空间位置、体积、占比 | 相比二维X射线探伤识别准确率大幅提升 | 检测耗时久、设备成本高 [[S39,S77,S69]] |
| **超声检测** | 依据氧化夹杂与基体之间的声阻抗差异产生的超声反射波判定缺陷位置 | 检测效率高、对人体无害 | 要求检测面平整光滑，无法精准判定夹杂具体类型，结果依赖操作人员经验 [[S19,S35]] |
| **熔体含氧量分析** | 采用压力过滤法将熔体中夹杂物截留在滤网表面进行定量计数，或采用缓慢减压法使铝液中氧化膜展开直接观测统计 | 浇注前预判熔体洁净度，炉前在线质量控制 | 离线采样分析 [[S54]] |

## 防治措施

### 熔体净化

熔体净化控制氧化夹杂的三类核心措施 [[S42,S84]]：

| 措施类别 | 具体方法 | 工艺要点 |
|----------|----------|----------|
| **吸附净化** | 采用适配牌号的精炼剂通过吸附作用脱除悬浮的氧化夹杂，搭配惰性气体吹洗让夹杂上浮进入熔剂层 | 精炼温度控制在 730~745℃，静置时间 15~30 min |
| **过滤净化** | 采用泡沫陶瓷过滤网拦截熔体中残留的氧化夹渣 | 过滤网规格一般为 15~25 目 |
| **静置处理** | 将熔体在高温下静置，让密度小的氧化夹杂充分上浮后扒除 | 严格控制静置温度和时间 |

### 压铸浇注系统优化

控制充型速度低于 0.5 m/s 避免紊流卷吸氧化膜进入型腔；在横浇道末端设置大容量集渣包截留最先进入型腔的冷料和氧化渣；采用封闭式流道设计避免铝液流股与大气大面积接触发生二次氧化 [[S5]]。

控制铝液过热度不宜过高——铝液温度每提升 20℉（约 11℃）氧化速率翻倍，高温下生成的氧化夹杂会大量悬浮在熔体内难以去除；避免过多使用含油、受潮的废铸件料重熔 [[S1]]。

### 真空压铸

在抽真空阶段将型腔内大部分空气提前排出，大幅降低铝液充型过程中与氧气的接触概率，可显著减少充型过程中紊流卷吸生成的新氧化膜，相比普通压铸可将铸件伸长率、疲劳寿命的稳定性大幅提升 [[S38,S45]]。

### SLM工艺氧化夹杂控制

将成形舱内充入高纯氩气，控制舱内氧含量在 100 ppm 以下，采用循环气流系统将激光作用下的烧蚀氧化粉尘及时带出熔池区域。在首层成型完成后对已凝固层追加二次激光重熔，可预先将层表面的薄层氧化膜熔解消除，避免后续铺粉熔化过程中氧化物混入熔池形成夹杂 [[S2,S41,S8]]。

## 材料特异性

### 铸造铝合金（ADC12、A356）

| 特性 | 内容 |
|------|------|
| **典型主成分** | 以 Al₂O₃（γ-Al₂O₃、α-Al₂O₃）为主，含 MgO、MgAl₂O₄ 尖晶石等伴生相。Mg含量较高的 A356 合金中尖晶石相占比显著高于 ADC12；ADC12 合金常规 Mg 添加量限定在 0.3% 以下以抑制 Mg 系氧化物大量生成 [[S11,S22,S31,S36]] |
| **氧化倾向** | 铝熔体和氧亲和力高，熔炼温度超过 750℃ 时 γ-Al₂O₃ 会向高温稳定的 α-Al₂O₃ 转变；铝合金中氧的摩尔平衡浓度约 4.5×10⁻⁷，熔体紊流卷气会大幅提升氧化夹杂生成概率 [[S46,S29]] |
| **典型形貌** | 除颗粒状夹杂外，普遍存在 Campbell 双膜结构，两层氧化铝薄膜外侧与铝基体相接、中部空腔非紧密接触。充型过程卷入氧化膜厚度仅数微米，大块膜状夹杂尺寸可超 100 μm [[S29,S24]] |
| **突出危害** | 严重割裂金属基体连续性，显著降低延伸率、冲击韧性及疲劳性能；同时作为氢析出形核基底大幅提升铸件气孔生成概率；弥散分布的细小夹杂会提升铝液粘度降低补缩能力，造成铸件渗漏 [[S11]] |
| **防治措施** | 采用 KCl/NaCl 混合盐熔剂覆盖隔绝熔体与空气接触；浇注系统设置陶瓷过滤网拦截大尺寸氧化膜；严格控制熔炼温度不超过 800℃ 减少高温氧化 [[S17]] |

### 铸造镁合金（AZ91D）

| 特性 | 内容 |
|------|------|
| **典型主成分** | 以 MgO 为主，占总氧化夹杂质量占比超 80%。镁屑燃烧产生的氧化夹杂粉末中 Mg₃N₂、AlN 等氮化物相占比约 20% [[S55,S4]] |
| **氧化倾向** | 镁熔体亲氧能力远强于铝合金，高温下镁与空气中氧、氮可自发发生氧化氮化反应生成 MgO、Mg₃N₂。熔体紊流剪切破碎表面氧化膜可将熔体内 MgO 颗粒数密度从 2.1×10⁵ mm⁻³ 提升至 3.2×10⁸ mm⁻³，夹杂生成效率陡增 [[S58,S80]] |
| **典型形貌** | 多为黑色分散簇状、皮膜状分布，典型颗粒尺寸 5 μm 左右，厚度 1~8 μm。特定尺寸的 MgO 夹杂可作为 α-Mg 的有效异质形核基底实现晶粒细化，属于罕见的"有害夹杂可资源化利用"体系 [[S24,S9]] |
| **突出危害** | 未受控的大块簇状 MgO 夹杂会割裂基体连续性，显著降低铸件延伸率与冲击韧性，同时易成为腐蚀萌生源提升镁合金的点蚀概率 [[S53]] |
| **防治措施** | 熔炼过程全程采用覆盖熔剂隔绝镁熔体与空气接触；浇注过程中在液流表面喷洒硫磺与硼砂混合物形成防氧化气氛，避免液流直接接触空气发生二次氧化 [[S28,S53]] |

### 铸造铜合金

| 特性 | 内容 |
|------|------|
| **典型主成分** | 普通纯铜和黄铜中以 Cu₂O 为主要氧化夹杂；铝青铜、含铝黄铜中 Al₂O₃ 为最主要氧化夹杂；硅黄铜中少量 Al 杂质生成的 Al₂O₃ 夹杂是铸件渗漏的核心诱因 [[S28]] |
| **氧化倾向** | 铜合金氧化倾向随合金中 Al、Si 等易氧化元素含量提升显著升高。普通黄铜中 Zn 元素本身可作为脱氧剂抑制 Cu₂O 大量生成；铝青铜熔炼过程中铝元素优先氧化生成 Al₂O₃，热力学生成优先级远高于铜的氧化物 [[S49,S28]] |
| **典型形貌** | 多呈暗灰色沿晶界分布的不规则颗粒状；铝青铜中高熔点 Al₂O₃ 夹杂难以上浮排除，大量悬浮在铜液中不易通过常规扒渣去除 [[S27]] |
| **防治措施** | 熔炼阶段加入少量磷铜合金进行脱氧处理；铝青铜等高氧化倾向铜合金采用平稳底注式浇注系统 + 集渣包 + 陶瓷过滤网的组合结构拦截二次氧化夹杂 [[S28,S49]] |

### 灰铸铁/铸钢

| 特性 | 内容 |
|------|------|
| **典型主成分** | 灰铸铁中以细小硅酸盐、少量 FeO 夹杂为主；铸钢中氧化夹杂分为两类——原生细小夹杂以 Al₂O₃、复合铝硅酸盐、MnO·SiO₂ 等为主，外来大尺寸夹杂以刚玉相、莫来石相、大颗粒硅酸盐为主 [[S62,S60]] |
| **氧化倾向** | 铸钢熔点远高于铸铁，熔炼过程中氧溶解度更高。铝终脱氧钢中最终残留 Al₂O₃ 含量稳定在 0.015~0.020%，生成后可聚合成大颗粒快速上浮排除 [[S63]] |
| **典型形貌** | 铸钢中原生氧化夹杂多沿一次晶界呈网状分布，大尺寸外来夹杂长度可超 3 mm；灰铸铁中氧化夹杂多为细小弥散的点状硅酸盐颗粒，无明显连续网络状分布特征 [[S16,S26]] |
| **突出危害** | 铸钢中沿晶分布的 FeO 氧化夹杂会大幅提升铸件热脆性；大尺寸外来氧化物夹杂在动态载荷下成为应力集中点，诱发疲劳裂纹扩展，降低铸钢件的横向力学性能与冲击韧性 [[S62,S64]] |
| **防治措施** | 采用分步脱氧工艺——先加入锰铁、硅铁预脱氧，最后用铝终脱氧；浇注系统设置多级挡渣、集渣结构；严格控制出钢与浇注过程避免二次氧化生成大尺寸外来夹杂 [[S60,S26]] |
