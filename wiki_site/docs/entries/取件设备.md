## 定义

依据国家标准GB/T 37371-2019《压铸单元 术语》与GB/T 20906-2007《压铸单元安全技术要求》，取件设备（或称取件装置）属于压铸单元核心辅助设备，定义为压铸生产流程中从打开的压铸模内取出刚成型的高温压铸件并转运至指定工位的自动化装置[[S10,S15,S26]]。在国标术语体系中，取件设备归入“取件与传送装置”大类，与之并列的配套子项包括夹持装置与传送装置[[S27]]。

压铸取件设备需适配200 °C以上刚脱模的高温铸件取放场景，且须与压铸机开合模工序严格联锁；普遍搭载铸件在位检测传感器，以避免铸件残留导致合模损伤模具。该应用特征使其区别于半导体晶圆取件设备（超高洁净环境、纳米级定位精度要求）和物流分拣取件设备（常温普通物料、大批次无序作业场景）[[S13,S8]]。

## 核心性能参数

GB/T 37371-2019 明确规定了压铸取件设备的三项核心性能参数[[S26,S27]]：

| 参数 | 定义 |
|------|------|
| 取件装置最大负载 | 能可靠取出压铸件的最大质量 |
| 取件装置的定位精度 | 取件设定位置与实际位置的偏差 |
| 单次最小取件周期 | 从取件位启动，取出铸件，转送至取出位的最短时间 |

## 分类

### 按取件原理分类

国外压铸领域早期分类体系将取件装置划分为三类，覆盖25~630吨全规格压铸机[[S8,S24,S9]]：

| 类别 | 说明 | 适配场景 |
|------|------|----------|
| 降落式 | 通过输送器或传送板承接顶出落下的铸件，结构简单，周期快，占用空间小 | 适合结构简单、易自动脱模的锌合金小型压铸件，适配小型压铸机 |
| 夹钳式 | 通过机械夹持结构主动抓取铸件，含夹钳、轨道等变体 | 适配绝大多数铝、镁合金铸件工况，以及中大型压铸机 |
| 喷涂取件一体式 | 取件装置同时集成模具辅助喷涂功能 | 在平行臂式装置中体现 |

另一基于实际取料原理的分类将取件方式分为两大类型[[S7,S23]]：

- **机械手取件**：适合铝合金、镁合金以及大型或深腔结构的锌合金压铸件取出。机械手在两半模之间作水平移动，夹住铸件后反向运动，脱离模具后放下铸件；为便于夹持，可在铸件上铸出专门凸块（后续加工切除）或利用直浇道作为夹持部分。
- **两次顶出落件式**：主要适配锌合金热室压铸机生产的结构简单小型压铸件。该方法通过再顶一次将铸件从推杆端部推落，经传送带运至冷却槽；铸件下落触发下方启动盘限位开关（或红外线探测器、光束探测器）方可开始下一压铸循环，以确保无铸件黏附模具。

### 按机构运动方式分类

专用取件机械手按机构运动方式可分为两大类[[S1,S18]]：

- **直移坐标式（桁架式）**：以直角坐标系三轴运动为核心结构，适配合模力大于1 600 kN的大型压铸机，负载能力强，运动路径刚性好，可兼容大质量复杂结构压铸件的取放要求。
- **回转加平移式**：适配中小型压铸机，运动范围灵活，在较小开距下亦可完成取件动作。

### 按驱动类型分类

压铸取件设备主流驱动类型包含三类[[S3,S14,S5]]：

| 驱动类型 | 特点 | 适用场景 |
|----------|------|----------|
| 气动驱动 | 响应快、成本低、结构简单、维护方便；伸入压铸模和拔出铸件的动作为直线运动，可在横向水平及垂直方向无级调节 | 中小型压铸机、中小负载高速工况 |
| 液压驱动 | 负载能力大、运行平稳 | 大吨位压铸机、重型铸件取件场景 |
| 伺服电机驱动 | 定位精度高、路径可编程、运动可控性显著优于气动取件机 | 高端压铸单元的主流配置，适配高节拍高精度生产线场景 |

> 【存疑】伺服驱动压铸取件机的独立分类条目目前缺少NADCA等海外压铸标准的独立权威定义支撑，现有技术资料仅作通用性描述[[S17]]。

### 典型机型

- **四连杆式取件机械手**：可直接固定在压铸机定模板侧，无需额外地基，不占用额外地面空间，沿预设“S”轨迹进出模具型腔。结构简单，动力消耗少，夹取铸件准确，运动平稳，更换夹钳夹板即可适应各种浇口直径的铸件[[S12]]。
- **平行臂式取件装置**：安装在压铸机侧方地面基座上，沿球面轨迹运动，可同时集成取件与模具辅助喷涂功能[[S5]]。
- **通用型六轴机器人**：具备6自由度运动能力，可实现复杂轨迹避障；除取件外，可兼容喷涂模具、嵌件放置、铸件转运至切边工位等复合工序；通过示教可快速修改运动路径[[S17]]。

## 结构与核心组件

典型压铸取件设备由夹持器（夹爪/夹钳）、机械臂部、驱动系统、控制系统、传感器与安全限位组件构成。

### 夹持器

夹持器与压铸机压室中心处于同一高度，支持两种取料模式[[S21,S30]]：
1. 先夹住料饼，然后随铸件顶出动作同步跟进夹持；
2. 待铸件完全顶出后再夹持料饼。

可根据铸件特点选择其中一种方式。通过更换夹钳夹板，可适配各种浇口直径的铸件[[S8]]。

### 机械臂部与驱动系统

气动驱动取件机采用气动-机械传动式结构，伸入压铸模和拔出铸件的动作为直线运动，可在横向水平及垂直方向无级调节，以适应不同压铸模厚度和浇口位置的要求[[S3]]。

液压驱动型取件机以主油缸、副油缸配合电磁阀、蓄压器实现动作控制，其油路原理图清晰呈现了液压驱动取件机的动作控制逻辑。

### 传感器与安全限位

取件机标配铸件取出确认传感器，用于确认铸件是否被完整取出，以保护模具和确保下一压铸循环的正常进行。传感器数量最多可设5个（含光电传感器与接触传感器），具体数量根据铸件形状结构确定[[S20,S21]]。

## 工作原理与流程时序

压铸取件机与压铸机开模信号联动的标准全流程包含以下时序节点[[S19,S5]]：

| 步骤 | 动作描述 |
|------|----------|
| 1 | 压铸机完成冷却计时，执行三段式开模动作到位，输出开模完成信号 |
| 2 | 取件机接收到开模完成信号，手臂前进（或横行前进）进入分型面区域；压铸机同步触发顶出动作 |
| 3 | 夹爪前进至夹持工位 |
| 4 | 夹爪闭合夹紧铸件 |
| 5 | 沿脱模方向拉拔（横行后退），使铸件脱离顶针 |
| 6 | 臂部完全退出模具工作区域，向压铸机返回取件完成互锁信号 |
| 7 | 旋转至下料工位，夹爪打开，将铸件放置到指定位置（冷却槽、输送带或切边工位） |
| 8 | 旋转退回，回归初始待命位置，等待下一次触发 |

## 安装、调试与安全规范

### 安装精度要求

取件机安装时须满足以下精度条件[[S31]]：
- 地脚螺丝紧固，整机水平安装；
- 横行装置与压铸机平行；
- 夹手中心与铸件料柄对中精度达标；
- 取件后铸件掉落高度控制在较低区间，避免磕碰变形；
- 运动路径与输送带不干涉。

### 调试要求

调试时需根据料头直径调整夹钳，调整夹钳与压铸机动模顶出动作的同步性，并设置好取件机动作时序与压铸机信号交互逻辑[[S16]]。

### 安全规范

取件机防护罩须与设备动作互锁——防护罩开启时取件机无法启动运行，避免人员误入动作区域引发安全事故[[S17]]。

## 与压铸岛其他设备的协作关系

### 取件机—喷雾机联锁

压铸机输出开模到位信号后，取件机进入分型面区域取件；取件完成返回原点后，输出就绪信号给自动喷雾机；喷雾机方可在接收到取件机原点位置信号后进入分型面执行喷涂作业。该互锁逻辑可完全避免取件机与喷雾机动作干涉碰撞[[S31]]。

### 铸件后续流转

取件机取出铸件后可直接将铸件转运至冷却工位水槽，再输送到切边压力机料位，完成去飞边后可将浇口料饼直接送回熔化保温炉，构成闭环、无废料落地的全自动压铸单元流水线[[S5,S11]]。

### 多机共享

通用型六轴工业机器人可同时为2台运行周期不同的压铸机服务，实际取件效率由两台压铸机中更长的循环节拍决定，适用于柔性多品种生产场景[[S25]]。

## 维护要点与常见故障

### 日常点检与定期维护

依据《压铸生产培训教程》中自动取出机检修周期表[[S28]]，主要维保项目如下：

| 检查项目 | 检查内容 | 日 | 周 | 月 |
|----------|----------|:--:|:--:|:--:|
| 压力表 | 压力是否在4~4.5 kg | ○ |   |   |
| 缓冲器 | 缓冲效果是否良好 | ○ |   |   |
| 直线运动轴承 | 是否异常磨损，是否加油并保持干净 | ○ | ○ | ○ |
| 运动部件 | 是否有磨损，线性轴承是否加黄油 | ○ | ○ | ○ |
| 空压管 | 是否断裂 |   | ○ | ○ |
| 空压接头 | 是否漏气 |   | ○ | ○ |
| 螺丝 | 各部螺丝是否松动 |   |   | ○ |
| 近接开关 | 是否正常 |   |   | ○ |
| 连接线 | 接头是否松动 |   |   | ○ |

### 高频故障与标准化排除方案

| 常见故障 | 产生原因与排除方法 |
|----------|--------------------|
| 未检出 | 排查铸件是否完整触发鼠尾开关，确认鼠尾开关接线与触发行程 |
| 动作到位信号冲突（进退限同时到位） | 排查对应磁性开关、微动开关是否损坏或位置偏移，校验电气线路绝缘 |
| 动作不到位 | 确认气源压力是否达标（气压应保持在约5 kg），清理调速阀内铝屑杂物，校验传感器安装位置 |
| 设备通讯中断（无压机自动/无开模限） | 确认压铸机自动运行状态信号、开模到位信号是否正常接入取件机PLC |
| 鼠尾开关短路 | 检查开关是否损坏不复位，检查配线是否短路 |

以上故障排查方案依据自动取出机常见故障及排除表[[S32]]。
