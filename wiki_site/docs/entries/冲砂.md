冲砂是砂型或砂芯表面局部砂层被高速充型金属液流冲刷脱落，在铸件对应位置形成不规则金属凸起，同时脱落的砂粒随金属液流动在其他部位造成砂眼的铸造缺陷。该缺陷归类于多肉类缺陷，国际分类体系中将砂型侵蚀（Core erosion / Sand wash）编码为 A212，属于砂类相关表面缺陷 [[S3,S17,S24,S27]]。

## 定义与同类缺陷区分

根据 GB/T 5611—1998 及行业标准，冲砂的定义为：金属液流冲蚀砂型或砂芯表面后，在铸件上形成的带点、块状凸起的粗糙表面，常伴随砂眼缺陷 [[S17,S3]]。冲砂属于“多肉类缺陷”，其本质是型砂材料被机械冲刷剥落而非型砂热膨胀起层，因此与夹砂、冲蚀结疤等缺陷成因截然不同 [[S17,S14]]。

冲砂与外观相近的含砂类缺陷可依据以下特征严格区分 [[S16,S21]]：

| 缺陷类型 | 宏观特征 | 形成机制 |
|----------|----------|----------|
| 冲砂 | 铸件表面局部不规则金属瘤状凸起，表面粗糙，位于内浇口直对区域，常伴砂眼 | 金属液高速冲刷剥落砂粒，原位留凸起 |
| 砂眼 | 铸件内部或表面的含砂粒孔洞，无对应表面多肉 | 脱落的砂粒被裹入金属液形成孔洞类夹杂 |
| 掉砂 | 块状金属凸起，外形与脱落的砂块完全吻合 | 砂块受机械力整体脱落 |
| 夹砂 | 片状薄金属层，其下方存在对应凹陷空腔 | 砂层受热膨胀起层后被金属液覆盖 |

## 形成机理

冲砂的形成是流体冲刷、高温热劣化与砂粒脱落三个环节依次作用的结果。

- **流体剪切触发**：金属液在型腔内流速超过安全临界值（一般不能大于 1 m/s，超过 3 m/s 时发生显著冲蚀）时，液流对砂型/砂芯表面施加的切向摩擦力超过砂粒间在浇注温度下的粘结强度，砂粒开始被剥离 [[S8,S10,S5]]。
- **高温弱化砂层**：持续流动的高温金属液将冲刷部位砂层急剧加热，使砂型内部粘结剂失效——粘土质砂的粘结网络失水脆化，树脂/水玻璃砂的粘结剂组分分解烧蚀，导致砂层高温抗剪强度断崖式下降 [[S26,S17]]。
- **砂粒脱落与缺陷形成**：紧实度不足的砂层在热‑力双重作用下，表层砂粒逐层剥离，随液流移动。冲刷原位的空缺被金属液填充，形成表面金属瘤凸起；脱落的砂粒随金属液流动至铸件其他部位，留存为砂眼伴随缺陷 [[S3,S28,S10]]。

若内浇口正对砂芯且无缓冲液池，金属液会持续冲蚀同一部位，甚至将初期形成的保护性金属硬壳重新熔化，使失去保护的砂面直接暴露，冲蚀程度进一步加剧 [[S10,S2]]。

## 形貌与检测特征

- **肉眼外观**：冲砂部位通常位于内浇口正对的型、芯表面，呈现无规则、凹凸不平的点块状金属瘤凸起，沿金属液流动方向有延伸趋势，凸起周边常散布砂眼孔洞 [[S3,S29]]。
- **X 射线检测**：X 光胶片上，冲砂部位的金属瘤呈现边缘不规则的亮斑；亮斑周边可见离散的暗点（低密度夹杂影），为裹入的砂粒，可与不含夹杂的普通多肉缺陷区分 [[S15]]。
- **金相与能谱**：冲砂位置取样抛光后，在金相显微镜下可见非晶态 SiO₂ 类非金属颗粒，与金属基体有明显界面。EDS 能谱分析可检测到异于基体的高硅含量峰值，证实砂粒残留 [[S34]]。

## 主要成因

冲砂的成因贯穿工艺设计、砂芯质量、造型操作及浇注工艺多个维度 [[S3,S2,S12,S1,S26]]。

**工艺设计维度**
1. 内浇口设计不当，金属液未分散，以高速集中冲击局部砂型/砂芯。
2. 内浇口无法形成缓冲液池，冲击后金属流反复冲刷同部位，无法形成保护硬壳。
3. 流经砂型表面的金属液总量过大，重熔浇注初期形成的保护硬壳。
4. 各内浇口流量分配不均，局部流速仍超出安全阈值 [[S2,S12]]。

**砂芯质量维度**
1. 砂芯表面松软，紧实度或硬化程度不足，高温强度剧降。
2. 砂芯表面加固插钉不足，修芯、装芯过程中表面损伤。
3. 涂料使用不当：粘度不匹配、水基涂料错用于呋喃树脂砂芯致终止固化、涂料烘干过烧。
4. 出气眼或芯骨距表面过近，形成低强度区，受热冲击易剥落。
5. 砂芯修补、砂孔填补不当，局部强度不足 [[S26,S1]]。

**造型与浇注工艺维度**
1. 型砂春砂过软或不均匀，导致局部高温强度不足。
2. 浇注温度过高（超过造型材料允许范围，如过热 60 ℃ 以上），使原本合格型砂失效。
3. 浇注速度过快，液流直接冲刷砂型表面。
4. 过量使用脱模剂（如煤油溶剂），降低局部型砂干强度和高温强度 [[S1,S26]]。

## 影响因素与临界参数

冲砂倾向受多种工艺参数和材料性能影响，部分定量阈值如下。

| 影响参数 | 安全阈值/范围 | 作用说明 |
|-----------|---------------|----------|
| 内浇口金属液流速（铸铁件） | ≤ 1 m/s；超过 3 m/s 显著冲砂 | 流速越高，剪切冲刷力越大 [[S10,S5]] |
| 差速器壳体类铸件底注内浇口流速 | ≤ 0.6 m/s | 该流速下冲砂隐患可降至最低 [[S19]] |
| 横浇道流速控制 | 靠近直浇道段 ≤ 0.3 m/s，其余 ≤ 0.6 m/s | 确保挡渣同时避免冲刷 [[S5]] |
| 型砂活性粘土含量 | 不超过 9%，低于阈值冲砂废品率上升 | 过高对抑制冲砂不再提升 [[S13]] |
| 型砂水分 | 3% 对应膨润土中水分 25% 时，抗压强度达峰值 | 水分偏离会降低强度 [[S13]] |
| 型砂压缩比 | 适宜值 40% 左右，需超过 35% | 压缩比不足混砂不匀易冲砂 [[S13]] |
| 模具/模温（压铸涂料成膜） | 240 ℃~320 ℃ 生成厚度均匀、高抗冲刷皮膜；160 ℃~200 ℃ 无法连续成膜；360 ℃ 皮膜过薄不耐冲刷 | 涂料成膜质量直接影响砂芯表面保护效果 [[S30]] |

## 预防与解决措施

综合权威著作与工程案例，防冲砂措施覆盖工艺设计、砂芯/砂型强化、涂料优化及浇注参数控制。

**工艺设计优化**
- 分散布置内浇道，避免金属液直冲型壁、芯子和型腔转角；采用底注为主、顶注为辅的阶梯式浇注系统，通过多次搭接控制流速和流量 [[S3,S19]]。
- 内浇口末端设置缓冲窝座，降低直冲动能 [[S19]]。
- 保证各内浇口流量均衡，流速不超过安全阈值。1017 型差速器壳体采用阶梯缓流式浇注，上层与下层内浇口流量比 3:7，底注内浇口流速 ≤ 0.6 m/s，铸件表面平整无砂孔，冲砂完全消除 [[S19]]。

**砂型/砂芯强化**
- 均匀混砂和舂实，提高砂型、砂芯的紧实度和强度；在凸出部位插钉加固 [[S3]]。
- 采用自硬树脂砂工艺可完全消除冲砂、胀砂类缺陷 [[S14]]。
- 控制砂芯烘干规范，避免过烧或硬化不足，防止存放返潮 [[S3,S26]]。

**涂料与表面保护**
- 在易冲砂部位及浇道系统采用耐火管、耐火砖及抗冲刷涂料 [[S3]]。
- 日本三菱重工高压造型线通过两阶段涂料优化，先由石墨系改为焦炭系涂料，再进一步调整粘结剂配比，量产验证后冲砂废品几乎完全消除 [[S13]]。
- 压铸涂料需满足耐冲刷、耐高温等核心性能，模具温度在 240 ~ 320 ℃ 区间喷涂水基涂料可获得厚度均匀的高抗冲刷皮膜 [[S30,S20]]。

**浇注参数控制**
- 适当降低浇注高度和浇注温度，浇注速度保持在不引发冲刷的正常范围 [[S3,S1]]。
- 避免过量使用脱模剂，防止型砂局部强度下降 [[S1]]。

## 衍生缺陷与鉴别

冲砂过程中脱落的砂粒随金属液流动，会在铸件非冲刷区域直接衍生为砂眼缺陷；严重冲砂时，碎砂裹挟分解的粘结剂组分还会同步形成分散性氧化夹杂缺陷，降低铸件的力学性能与耐压密封性能 [[S10,S18]]。因此，发现冲砂缺陷后，除打磨去除表面多余金属瘤（严重时铸件应报废）外，还需对铸件整体进行砂眼和夹杂检查 [[S3]]。

鉴别冲砂时应特别注意与 **冲蚀结疤** 的区分。二者外观相似，但成因完全相反：冲蚀结疤由型砂膨胀引发，对应春砂过硬的工况；而真正的冲砂源于春砂过软或高温强度不足。误判将导致错误的工艺调整方向，使缺陷加剧 [[S17]]。

## 英文术语与标准

- 英文 “sand wash” 在铸造术语中同时指冲砂缺陷和砂型表面防冲蚀涂料两层含义 [[S23]]。
- 国际铸造缺陷分类体系（ISO 铸铁缺陷分类标准、WFO、AFS《国际铸造缺陷图谱》）将砂型侵蚀/冲砂统一编码为 **A212**，列于砂类相关表面缺陷大类之下 [[S27,S7,S24]]。
