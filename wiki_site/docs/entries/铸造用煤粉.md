## 概述与定义

铸造用煤粉（英文常用名：seacoal for green sand mold、foundry coal dust）是铸铁用湿型粘土砂造型工艺中的一种关键附加物。根据现行机械行业标准 JB/T 9222—2008 的定义，**湿型铸造用煤粉** 是指“在铸铁用湿型砂中加入的以煤为原料经粉碎制成的附加物，其作用是防止铸铁件产生粘砂和夹砂结疤，提高型砂溃散性”[[S15,S4]]。产品外观为黑色或黑褐色细粉，多以烟煤为原料经粉碎制得[[S11,S24]]。

铸造用煤粉与普通锅炉煤粉、民用煤粉有本质区别：后者的质量控制以完全燃烧和热值为核心，而铸造用煤粉需优先保障 **光亮碳析出能力**、**焦渣特性** 等满足铸件表面质量要求的指标，并不单纯追求燃烧发热[[S2,S10]]。

## 分类与牌号

我国铸造用煤粉的牌号体系历经多次修订：

- **ZB J31 002—88 / JB/T 9222—1999**：按挥发物含量划分为 **SMF‑35**、**SMF‑30**、**SMF‑25** 三个牌号，其中 SMF 为“湿型用铸造煤粉”汉语拼音字头，后缀数字代表挥发物含量的下限[[S5,S3,S23]]。
- **JB/T 9222—2008（现行）**：改为按光亮炭等综合性能指标分为 **SMF‑I**、**SMF‑II**、**SMF‑III** 三个牌号[[S15,S4]]。
- 另有 **JB 2950—81《化铁炉用煤粉》**，将煤粉划分为一类、二类、三类，对应煤粉在化铁炉中的使用，其挥发分、灰分、全硫及粒度要求与湿型砂用煤粉有所不同[[S23]]。

部分技术手册中还出现过按多维度参数组合的命名方式（如“1M16‑95D”），依次对应煤粉类别、灰分上限、粒度通过率及含硫量等级，在企业采购选型时仍具参考价值[[S7]]。

## 组成与理化性能要求

铸造用煤粉的关键化学、物理指标由行业标准统一规定，核心参数见表。

| 项目 | 指标要求 | 检测标准 |
|------|----------|----------|
| 挥发分 | 按牌号分类控制，一般要求挥发分充足以保证还原气氛和光亮碳析出 | GB/T 212《煤的工业分析方法》 |
| 灰分含量 | ≤ 10% | GB/T 212 |
| 全硫含量 | ≤ 2% | GB/T 214《煤中全硫的测定方法》 |
| 水分含量 | ≤ 4% | GB/T 212 |
| 粒度 | 95% 以上颗粒通过 0.106 mm 筛孔 | GB/T 2684《铸造用原砂及混合料试验方法》 |
| 焦渣特性 | 反映煤在干馏过程中软化、熔融、固化成焦的特性，按 GB/T 212 分级 | GB/T 212 |
| 光亮碳析出量 | 按 JB/T 9222—2008 附录 A 规定的石英管‑石英棉法，在 900℃下测定 | JB/T 9222 附录 A |

以上参数中，挥发分、灰分、水分、硫分及粒度是出厂必检项目。对于光亮碳和焦渣特性，旧版标准仅作供需双方协商项目，现行 JB/T 9222—2008 已将其纳入规范性控制指标[[S15,S5,S3,S4]]。挥发分的来源和煤种选择至关重要：长焰煤、气煤虽然挥发分高，但形成的胶质体不稳定，不利于生成光亮碳层，因而不宜用作湿型砂的抗粘砂材料[[S2]]。

## 工艺作用机理

在湿型粘土砂中注入铁液的高温作用下，煤粉通过多路径协同发挥抗粘砂、防夹砂及提高溃散性的综合作用：

1. **形成还原性气氛**  
   煤粉热解释放大量还原性气体（包括挥发分及碳与水反应生成的 H₂），防止铁液氧化、还原液面氧化物，减少金属氧化物与型砂的化学反应[[S29,S24]]。

2. **缓冲石英砂热膨胀**  
   煤粉受热后软化形成具有可塑性的胶质体，可在较宽温度区间内缓冲型砂中石英颗粒的膨胀应力，从而减轻因砂型热膨胀引起的夹砂缺陷[[S29,S24]]。

3. **堵塞砂型表面孔隙**  
   胶质体产生气、液、固三相膨胀，部分填充堵塞砂型表面砂粒间隙，增加金属液渗入的阻力[[S29,S24]]。

4. **析出光亮碳膜**  
   煤粉热解生成的芳烃类挥发分在 650～1000 ℃ 的还原性气氛中发生气相热解，在金属‑铸型界面上沉积出一层平滑光亮的光亮碳膜。该碳膜不被铁液润湿，能有效阻止机械粘砂，显著提升铸件表面光洁度[[S29,S24]]。

除上述直接作用外，煤粉还可改善型砂浇注后的溃散性，有利于降低清砂难度[[S11,S24]]。但煤粉的加入同时伴随负面影响：型砂灰分和含泥量上升，透气性、流动性和韧性下降；过量加入时发气量过大，易引发气孔或浇不足缺陷[[S20,S29]]。

## 使用工艺与配比控制

### 加入量范围

煤粉在湿型砂中的推荐有效含量因造型方法和铸件特征而异[[S35,S16,S19]]：

| 应用场景 | 型砂中有效煤粉含量（质量分数） |
|----------|-------------------------------|
| 高紧实度造型单一砂 | 4%～5% |
| 普通机器造型灰铸铁件 | 5%～8% |
| 薄壁小件 | 3%～4% |
| 小型家用铸造（经验配比） | 约 2%（占砂总重） |

铸件表面出现蓝色光泽往往表明煤粉已过量，而表面粘砂则可能反映煤粉含量不足或型砂透气性过高[[S16]]。

### 加料顺序与协同关系

湿型砂混制时，标准的加料顺序为：**新砂/旧砂 → 加水混匀 → 加入膨润土和煤粉 → 补充水分调节至目标含水率**[[S31]]。先加水形成砂粒水膜，有利于膨润土快速均匀包覆，避免煤粉与膨润土结团，同时可减少混砂扬尘。

煤粉、膨润土与水分三者之间存在紧密的匹配关系[[S8,S20]]：
- 煤粉添加量升高会使型砂灰分与含泥量增加，此时需同步提高膨润土添加量以维持粘结强度，并适当提高含水量以补偿煤粉的吸水作用。
- 煤粉过量将导致型砂总发气量急剧上升，增大气孔缺陷风险，此时必须同步降低总含水量以控制发气水平。

### 与铸件材质的适配性

- **灰铸铁**：可直接选用常规牌号煤粉[[S9]]。
- **球墨铸铁**：煤粉释放的还原性气氛有利于抑制型砂表面氧化，减少皮下气孔缺陷[[S9]]。
- **铸钢**：传统煤粉的硫含量较高（通常＞1%），易引发铸钢件热脆，因此铸钢湿型砂一般避免使用煤粉，或仅允许在低硫特种煤粉且限制添加量的情况下使用[[S9]]。

## 质量标准与检验

**国内主要标准**：JB/T 9222—2008《湿型铸造用煤粉》是现行湿型砂用煤粉的产品标准。该标准相比 1999 版的主要变化包括：牌号表示方法由挥发物改为光亮炭等性能分级，新增光亮炭和焦渣特性检测条款，修订了灰分与硫含量的控制指标，并更新了光亮炭测量方法[[S15]]。其中，工业分析方法引用 GB/T 212（等效采用 ISO 11722:1999），全硫测定引用 GB/T 214（非等效采用 ISO 334:1992、ISO 351:1996）[[S15,S1]]。

**国际标准**：目前不存在专门针对铸造用煤粉产品的 ISO 或 ASTM 标准，仅通用煤质分析的相关国际测试方法可作为煤粉性能的检测依据[[S1]]。

## 适用范围与局限性

**适用工艺**  
铸造用煤粉适用于以膨润土为粘结剂的湿型粘土砂工艺，可用于手工造型、震击造型及高压造型等各类常规黑砂造型生产线，是中小型灰铸铁件大批量生产的主流抗粘砂附加物[[S11,S27]]。

**不能使用的工艺**  
煤粉不适用于树脂砂、水玻璃砂等非粘土粘结剂砂工艺。主要原因在于：煤粉是高灰分细颗粒，会大量消耗树脂或水玻璃粘结剂，导致需用量上升且砂型固化后强度下降；其高含碳组分还干扰树脂或水玻璃的交联固化反应，影响砂型硬化效率[[S33,S34]]。此外，这几种砂工艺通常采用在砂型（芯）表面涂刷专用抗粘砂涂料，无需在砂料内掺入煤粉。

## 环保、安全与储运

煤粉在混制、输送过程中易产生扬尘，长期无防护吸入可造成尘肺类职业病；浇注阶段煤粉燃烧和热解会释放大量有机烟气及挥发性污染物，恶化作业环境[[S20,S18]]。

安全包装与储运要求[[S25,S5]]：
- 采用 2～3 层包装，内层为塑料袋，外层为编织袋；
- 每袋标称重量 25 kg；
- 运输过程须防雨淋并完全隔绝明火等热源；
- 储存应置于阴凉、干燥、通风良好的库房，分行叠放，严格控制堆高以防积热自燃。

将煤粉预先与膨润土等惰性物料混合制成复合添加剂，可有效降低煤粉单独存储和转运过程中的自燃风险[[S18,S8]]。

## 替代与复合添加剂

为克服天然煤粉灰分高、烟气大、降低型砂性能等不足，工业上已开发多种替代或复合方案。

### 主要替代品对比

| 类别 | 光亮碳能力 | 典型加入量 | 主要优点 | 主要缺点 |
|------|------------|------------|----------|----------|
| 沥青类增效煤粉 | 12%～20%（天然煤粉的约 2 倍） | 约为天然煤粉的一半 | 灰分、硫分较低；软化区间宽；烟气较少[[S18]] | 需专门配制，成本高于天然煤粉 |
| 重油/渣油 | 光亮碳析出量为煤粉的 1～6 倍 | 与煤粉配合使用，不能全替换 | 灰分＜0.1%，延缓旧砂灰分积累；改善韧性、起模性[[S20,S26]] | 过量加入产生大量烟气，可能造成薄壁件冷隔、浇不足[[S16]] |
| 聚苯乙烯 | 80%～85%，挥发分约 100% | 仅为煤粉质量的 1/6～1/9 | 型砂需水量降低 20%；透气性、流动性提升；CO 和粉尘大幅降低[[S26]] | 价格最贵 |
| 淀粉类 | 不析出光亮碳，但有效防粘砂 | 可部分或全量替代（工业上多复配使用） | 降低型砂泥分与含水量，提高透气性、改善起模性，环境污染小[[S18]] | 生产成本明显升高 |

GB/T 5611—2017《铸造术语》将煤粉代用品分为两大类：第一类为重油、渣油、炭油等油类产物；第二类为聚苯乙烯、无规聚丙烯等合成聚合物。代用品普遍具有光亮碳含量更高，而硫、灰分更低的优势[[S30]]。

### 预混复合添加剂

欧美铸造行业广泛采用 **膨润土‑有机物复合添加剂**，即将煤粉、膨润土、淀粉等按比例预混制成单一商品，混砂时一次性加入。这不仅能简化物料管理、防止煤粉自燃，还可提升生产控制的稳定性，美国该类添加剂的年用量可达 80 万吨[[S18,S8]]。

### 选型与验证

市场上存在多种未标明成分的“抗粘砂添加剂”或“光亮剂”。建议企业选用前先进行实验室对比验证（例如采用同炉铁液一次浇注四块标准阶梯试块对比表面质量），然后进行不少于一年的小批量工业试用，确认型砂性能与铸件质量稳定后再扩大应用[[S18,S26]]。阶梯试块的结构尺寸见图。
