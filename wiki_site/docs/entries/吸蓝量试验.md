## 概述

吸蓝量试验（methylene blue value test）是利用膨润土中的蒙脱石强烈吸附亚甲基蓝的特性，用滴定法测定膨润土或粘土砂吸附亚甲基蓝数量，用以检验膨润土纯净度或粘土砂中活膨润土含量的试验方法[[S12,S28]]。该试验属于铸造原辅材料检验中的一项基础性检测方法，在铸造行业标准中也被称为亚甲基蓝吸附量测定[[S16,S25]]。

膨润土中的蒙脱石具有极强的阳离子交换能力，可吸附亚甲基蓝阳离子染料的量远高于其他黏土矿物：每100g干蒙脱石矿物约可吸附亚甲基蓝44g，而高岭石类黏土矿物每100g的吸附量一般小于10g[[S5,S33]]。基于这一显著差异，吸蓝量试验可以快速区分活性膨润土与低吸附性杂质及非活性矿物[[S2]]。

在铸造用膨润土质量管理中，吸蓝量是判定产品纯度的重要指标。我国行业标准明确规定，各种牌号铸造用膨润土的吸蓝量不得低于20g/100g土[[S25,S29]]。

## 试验原理

吸蓝量试验的原理基础是膨润土中蒙脱石矿物的强阳离子交换特性。蒙脱石的阳离子交换容量可达80~150mg当量/100g土，而高岭石仅为3~15mg当量/100g土，二者相差悬殊[[S3,S21]]。亚甲基蓝（C₁₆H₁₈N₃SCl·3H₂O，相对分子质量373.9）在水溶液中以阳离子形式存在，与蒙脱石颗粒表面的可交换阳离子发生定量置换吸附[[S2,S5]]。

当向膨润土悬浮液中逐滴加入亚甲基蓝标准溶液时，染料阳离子首先被蒙脱石的活性交换位点全部吸附，溶液中无游离染料存在。当活性位点被完全饱和后，游离的亚甲基蓝开始出现在液相中，此时利用滤纸晕圈法即可判别滴定终点[[S13,S7]]。吸蓝量高低与膨润土阳离子交换能力呈明显正相关[[S21]]。

**需特别注意**：不同产地膨润土的蒙脱石含量与吸蓝量的换算系数并不统一，不存在适用于所有产地的通用固定换算系数。因此，国内现行铸造用膨润土标准直接以吸蓝量作为判定产品纯度的直接指标，禁止使用固定系数进行统一换算[[S29,S24]]。

## 仪器与试剂

### 主要仪器

| 仪器 | 规格要求 |
|------|----------|
| 滴定管 | A级，符合GB/T12805规定[[S13,S8]] |
| 分析天平 | 感量0.001g[[S13]] |
| 磁力搅拌器 | 转速可达1550r/min[[S19]] |
| 锥形瓶 | 250mL[[S13]] |
| 电炉 | 用于加热煮沸[[S13]] |
| 中速定量滤纸 | 用于终点判定[[S13]] |

此外，还可选用专用SND型亚甲基蓝黏土测定仪进行检测[[S19]]。

### 试剂配制

**焦磷酸钠溶液（1%，质量分数）**：使用分析纯焦磷酸钠溶解于蒸馏水定容得到[[S13]]。

**亚甲基蓝标准溶液（0.002g/mL）**：先将三水亚甲基蓝在93±3℃烘干至恒重，准确称取2.000g烘干后的亚甲基蓝完全溶解于蒸馏水，在1000mL容量瓶中用蒸馏水稀释至刻度并摇匀。配制好的溶液应倒入棕色玻璃瓶中避光密封贮存，使用前可通过已知吸蓝量的标准膨润土样品进行标定校准[[S13,S16,S22]]。

> **注**：亚甲基蓝试剂使用前应一直在干燥器中密封避光储存[[S13]]。

## 操作步骤

### 标准滴定法（据JB/T 9227）

**1. 样品前处理**

称取在105~110℃烘干至恒重的膨润土试料0.200g±0.001g，置于已加入50mL蒸馏水的250mL锥形瓶内，使其预先润湿[[S13,S7]]。

**2. 分散处理**

加入1%焦磷酸钠溶液20mL，摇匀后在电炉上加热煮沸5min，然后在空气中冷却至室温[[S13,S26]]。焦磷酸钠作为分散剂，可促使膨润土颗粒充分分散，使吸附点位完全暴露[[S34]]。

**3. 滴定与终点判断**

- 首次滴加预计亚甲基蓝总消耗量的2/3，摇晃1min使充分反应[[S13]]；
- 以后每次滴加1~2mL，摇晃30s后用玻璃棒蘸取1滴试液点在中速定量滤纸上，观察中央深蓝色斑点周围是否出现淡蓝色晕环[[S13,S7]]；
- 当开始出现淡蓝色晕环时，将试液继续摇晃2min后再复测[[S13]]；
- 若2min后淡蓝色晕环消失，说明未到终点，继续每次滴加0.5~1mL亚甲基蓝溶液，反复测试[[S13]]；
- 若摇晃2min后仍保持宽度约0.5~1.0mm的明显淡蓝色晕环，即为滴定终点[[S13,S26]]。

### 分光光度比色法（变体方法）

除滴定法外，还存在分光光度比色法：称取定量试样与过量亚甲基蓝溶液充分混合吸附后，使用比色计测量残余上清液中的剩余亚甲基蓝浓度，换算得到试样吸蓝量。该方法操作复杂度更高，需要配备分光光度计，铸造厂常规检测场景中普及率低于滴定法[[S2,S10]]。

## 结果计算与精度

### 计算公式

吸蓝量的标准计算公式为[[S13,S30]]：

$$MB = \frac{cV}{m} \times 100$$

式中：
- MB——吸蓝量，g/100g试料；
- c——亚甲基蓝标准溶液浓度，g/mL；
- V——亚甲基蓝溶液滴定体积，mL；
- m——试料质量，g；
- 100——每克吸蓝量换算为100g试料吸蓝量的系数。

### 精度要求

取两次平行测定结果的算术平均值作为最终检测结果，两次平行测定的相对偏差不得大于2%[[S13,S30]]。不同的操作者进行试验，亚甲基蓝滴定体积的偏差不得大于1mL[[S19]]。

## 影响因素与操作注意事项

| 影响因素 | 具体说明 |
|----------|----------|
| pH值 | 中性条件下亚甲基蓝吸附率最高，是试验推荐控制条件[[S34]] |
| 样品分散程度 | 必须加入焦磷酸钠分散剂并配合加热煮沸，保证膨润土颗粒充分分散，否则吸附点位无法完全暴露，导致检测结果偏低[[S34]] |
| 滴定速度 | 接近终点时需严格控制每次滴加量（0.5~1mL），每次滴加后充分搅拌2min[[S34]] |
| 滤纸选择 | 应使用中速定量滤纸进行点滴试验[[S34]] |
| 亚甲基蓝溶液贮存 | 必须存放于深棕色玻璃瓶中，避免光解[[S34]] |
| 杂质干扰 | 死粘土、惰性硅质粉尘不会被计入吸蓝量检测结果，不干扰活性膨润土定量[[S14,S20]] |
| 天平分度 | 称量天平灵敏度需达到±0.01g[[S19]] |

## 在铸造生产中的实际应用

### 进厂膨润土品质检验

吸蓝量试验是铸造厂入库膨润土来料品质校验的快速方法，可直接反映膨润土的纯净度（蒙脱石含量水平）。普通粘土与膨润土的吸蓝量存在本质差异[[S9]]：

| 指标 | 普通粘土 | 膨润土 |
|------|----------|--------|
| 吸蓝量 (g/100g) | 5~15 | >20 |
| 吸水率 (%) | 60~80 | >100 |

膨润土纯度越高（吸蓝量越高），其湿态黏结力越大，型砂热湿拉强度越高，抗夹砂缺陷能力越强[[S11,S2]]。

### 旧砂有效膨润土含量监控

在铸造循环系统砂管理中，吸蓝量试验可以测定旧砂中的有效活性膨润土含量，相比传统AFS水洗沉降法的全黏土总量检测更为精准，因为它仅检测仍具有阳离子交换活性的蒙脱石，无法检出已热失效的"死黏土"、惰性细粉和泥分[[S14,S20,S32]]。

具体操作方法是预先标定亚甲基蓝滴定量-有效膨润土含量标准曲线，通过实测旧砂试样的亚甲基蓝滴定体积，从标准曲线上直接查取有效膨润土质量分数[[S7,S27]]。下图即为该标准曲线的典型形式：

![旧砂有效膨润土含量测定标准曲线——亚甲基蓝滴定量与膨润土加入量呈线性正相关](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/01829da7-4803-4f71-bac5-37512fc50bc6/resource)

国外铸造行业规范要求每2个砂循环需同步对比吸蓝量结果与湿压强度实测值，动态调整新膨润土补加量[[S14,S20]]。

### 膨润土热复用性评定

将不同温度（200℃、300℃、400℃……）下焙烧后的膨润土试样冷却后测定吸蓝量，绘制吸蓝量随焙烧温度的变化曲线。吸蓝量显著下降所对应的焙烧温度越高，表明膨润土的耐热复用性越好。通常钠基膨润土的吸蓝量随温度下降的阈值显著高于钙基膨润土[[S23,S31]]。

## 相关指标对比

吸蓝量、胶质价、膨胀指数均与膨润土中蒙脱石含量呈正相关，但指向不同的性能维度[[S12,S26,S28]]：

| 指标 | 定义 | 单位 |
|------|------|------|
| 吸蓝量 | 100g试料饱和吸附的亚甲基蓝质量 | g/100g |
| 胶质价 | 粘土与水搅匀后静置24h，凝胶层占全部混合物体积的百分比 | % |
| 膨胀指数 | 粘土水浊液静置24h后全部保持溶胶状态的蒸馏水体积与粘土质量的比值 | mL/g |
| 膨润值 | 3g膨润土+5mL 1mol/L氯化铵溶液定容至100mL静置24h后的沉淀体积 | mL |

- **GB/T 5611-2017《铸造术语》**：定义了吸蓝量试验与活膨润土量[[S12,S28]]
- **JB/T 9227-2013**（替代原JB/T 9225及ZB J31 009系列）：规定了铸造用膨润土的吸蓝量测定方法、试剂配比、操作步骤及合格要求[[S13,S25,S30]]
