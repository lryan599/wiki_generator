超声分散是以功率超声为核心手段，向高温金属熔体（或半固态浆料）中引入可控超声能量，利用瞬态空化、声流及声辐射力等效应，破碎纳米/微米级增强相、初生相或夹杂物团聚体，并改善颗粒与熔体润湿性，最终实现弥散相在熔体中均匀分布的外场辅助工艺。该技术属于功率超声辅助铸造技术范畴，在金属基纳米复合材料制备、半固态浆料调控及熔体净化中起关键作用[[S8,S24,S3,S41]]。与常规常温液相体系的超声分散不同，金属熔体场景面临更高的工作温度（铝合金常达630~720℃）、更高的空化阈值，超声作用同时兼具分散颗粒、脱气、细化晶粒等多重冶金效果[[S8,S26]]。

## 工艺原理

### 声空化效应
频率高于20 kHz的超声波在熔体中传播时，正负压交替的声场引发周期性应力变化。负压期熔体被拉裂形成空化泡，正压期空化泡以极高速率瞬间溃灭。溃灭过程在微米级局部区域产生高达约1.6×10⁴ K的瞬时温度、约1.013 GPa（约1000 atm）的冲击高压，以及10⁹~10¹⁰ K/s的升/冷却速率，同时向外辐射强冲击波和高速微射流[[S21,S24]]。这些极端条件可直接击碎纳米颗粒团聚体。

### 声流效应
超声波在熔体中因衰减形成声压梯度，驱动熔体产生定向宏观环流与局部微涡流，声流速度可达常规自然对流的5~1000倍。声流将被空化作用破碎的分散相颗粒输送到熔体全域，实现温度场、溶质场的均匀化，防止局部颗粒重新团聚[[S31]]。

### 声辐射力
连续声场中，作用于微小颗粒的非周期性平均作用力（主辐射力和次辐射力）可驱动弥散相、夹杂物沿声压梯度定向移动。其大小与声强、颗粒半径正相关，经验公式为F=2π²IR₀/(δ_tot ω₀)。适当的声场参数可辅助分散，但参数失当可能导致颗粒在驻波节点聚集[[S39,S44]]。

### 团聚体破碎与分散流程
初始团聚 → 空化泡生成 → 空化泡溃灭产生微射流/冲击波击碎团聚体 → 局部高温高压脱除颗粒表面吸附气体、降低接触角、提升润湿性 → 声流驱动单个颗粒均匀扩散至整个熔体[[S24,S4,S47]]。其过程可参照下图。

## 设备与系统构成

## 实施分类与压铸集成方式

### 直接浸入式
将超声探头直接插入高温金属熔体内部，能量利用率高，空化区域主要集中在探头附近，分散效率高。探头材料需具备优良的耐高温熔体腐蚀性能，常用钛合金、铌合金或陶瓷。该方式是实验室和中试线最常用的方案，例如2000 W功率搭配直径40~60 mm的高振幅陶瓷探头，可实现20 kg/min以上的连续处理[[S7,S42]]。

### 间接振动式
超声振子不直接接触熔体，而是将振动施加在坩埚壁、模具壁或盛料容器外部，通过固体壁面传导入熔体。此方案完全避免了探头腐蚀与熔体污染，适合密闭加压压铸和大容积熔体批量处理，但能量经容器传导时衰减较大，远离施振端的区域分散效果下降，需要匹配更高的超声功率[[S7,S42]]。

### 压铸产线集成方案
在压铸工业应用中，主要采用三种部署形式：
- **浇包批次处理**：将超声探头插入转运浇包对整包熔体进行分散处理，适用于中小批量柔性生产；
- **在线流道连续处理**：将超声模组布置在从保温炉至压室的浇注流道内，实现流动熔体的连续分散，适用于大规模自动化压铸线；
- **半固态浆料预处理**：在制浆工位完成超声处理，直接制备低固相率的半固态压铸浆料[[S1,S45]]。

## 典型应用
超声分散在铝合金、镁合金的熔体处理中具有广泛的应用价值，主要体现在以下几个方面。

- **纳米/微米增强相分散**：将SiC、Al₂O₃、CNTs、TiB₂等陶瓷或碳质增强相均匀引入铝/镁合金熔体，制备金属基纳米复合材料。例如，采用预氧化处理的40 nm β-SiCp结合高能球磨、机械搅拌与超声分散，制备的2wt%纳米SiCp增强铝基复合材料拉伸强度提高20%，伸长率提高44%[[S35]]；超声辅助半固态搅拌铸造可使CNTs均匀分布于AZ91D镁合金的晶界与β-Mg₁₇Al₁₂相周围[[S40]]。
- **初生相及第二相细化**：对过共晶Al-Si合金进行超声处理可将初生Si细化至约20 μm，且保温后圆整度进一步提升；间歇超声（空振比0.5）可获得更优细化效果[[S35]]。在Al-Si活塞合金中，液相线上施振可实现初晶Si细化并显著提高硬度[[S35]]。
- **熔体净化与除杂**：超声处理可有效脱除熔体中气体，并使夹杂物聚合去除。例如，2A14铝合金超声处理后夹杂物中有害元素Ca、S、P消失，净化效果显著[[S30]]；同时力学性能得到提升[[S9]]。
- **半固态浆料制备**：在液相线以上对铝合金施振可获得细小圆整的半固态组织，经后续压铸成形，铸件抗拉强度较液态压铸明显提高[[S1,S5]]。

## 与其他搅拌方法的对比
超声分散与机械搅拌、电磁搅拌的优劣势对比见表。

| 对比维度 | 超声分散 | 机械搅拌 | 电磁搅拌 |
| :--- | :--- | :--- | :--- |
| 分散效果 | 可有效破碎微米/纳米级增强相团聚，实现均匀分散 | 纳米级颗粒团聚破碎效果有限，搅拌速度达临界值后效果不再改善，甚至恶化 | 存在集肤效应，磁场随深度衰减，边部作用强而中心弱，整体分散不均匀 |
| 附加冶金作用 | 兼具熔体脱气、净化颗粒表面、改善润湿性、细化初生/第二相 | 无 | 无 |
| 对熔体污染 | 直接浸入式有探头腐蚀污染风险 | 搅拌桨腐蚀易污染熔体，易卷入气体形成气孔 | 无接触，不污染 |
| 处理规模适应性 | 超声衰减导致有效作用半径有限，大体积熔体中心区域效果较差 | 存在搅拌死区，影响均匀性 | 适合大体积连续处理，但搅拌力分布不均 |
| 设备复杂度与成本 | 中等 | 简单、成本低 | 较高，参数可控性好 |

综合对比，超声分散在颗粒破碎均匀化和组织细化方面具有突出优势[[S36,S15,S48]][[S17,S3,S10]]，而机械搅拌虽简单经济，但对纳米颗粒的分散效果有限[[S38,S2,S18]]，电磁搅拌则因非接触特性适用于大体积无污染处理，但分散均匀性受限于物理场衰减[[S18,S29]]。

## 局限性与优化策略

### 主要局限性
- **声场衰减与作用范围**：超声波在高密度金属熔体中衰减显著，有效作用半径有限，超大吨位熔体中心区难以获得良好分散效果[[S17,S3]]。
- **探头腐蚀与熔体污染**：长期直接浸入高温熔体的探头易发生腐蚀磨损，腐蚀产物可能成为熔体夹杂物来源[[S3,S10]]。
- **大体积连续处理困难**：工业规模的大体积熔体连续超声处理配套装备尚不完善，多探头协同和声场设计仍是技术难题[[S10]]。

### 优化策略
- **脉冲间歇超声模式**：控制空振比（如0.5）可提升初生相细化效果[[S35]]。
- **多探头阵列布置**：在大体积熔体或多位置同时施振，抵消衰减影响，扩大有效处理区域[[S22]]。
- **增强颗粒表面预处理**：如对SiC颗粒进行Cu/Ni化学镀包覆，以提升其与铝熔体的润湿性，制备的SiCp/Al复合材料抗拉强度可达389 MPa，断裂应变6.3%[[S35,S22]]。
- **变幅杆与施振位置优化**：根据处理对象和熔体容积选择合适变幅杆类型与探头位置，提升声能传递效率[[S6]]。

## 相关概念与关联词条
- **超声处理（Ultrasonic Treatment, UT）**：超声分散是其核心功能之一，共同构成超声熔体处理技术。
- **熔体超声处理 / 超声铸造**：以超声场干预金属凝固全过程的工艺，与超声分散共享空化与声流原理。
- **超声脱气**：利用超声空化效应去除熔体中溶解气体的工艺，常与分散同时发生。
- **半固态超声搅拌**：针对半固态浆料的超声处理，综合实现颗粒分散与组织球化。
