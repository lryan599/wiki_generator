## 定义

**比浇注速度**（又称**比流量**、**比填充率**，英文：**specific pouring rate**）是压铸工艺中表征金属液充填型腔能力的质量流量型核心参数，其物理定义为**单位时间内通过内浇道（内浇口）单位截面积的熔融金属液流量**[[S11]]。该参数存在两种维度表述方式：

- **质量流量维度**：行业常用单位为 kg/(s·mm²)
- **体积流量维度**：行业常用单位为 L/(s·cm²) 或 cm³/s

两种单位间的换算可通过对应压铸合金的标准密度推导：铝合金约 2700 kg/m³、锌合金 6500~6600 kg/m³、镁合金 1700 kg/m³[[S11]]。

### 与浇注速度的区别

在普通铸造领域，**浇注速度**（pouring rate）定义为铸件总质量与实际浇注时间之比，单位为 g/s 或 kg/s，属于总质量流量类参数[[S6]]。而**比浇注速度**在浇注速度的基础上进一步除以内浇口截面积，实现了单位面积上的流量归一化，使其成为能够脱离具体模具尺寸进行比较和选型的标准化工艺参数，这也是“比”（specific）一词的核心内涵。

### 与内浇口速度（充填速度）的严格区分

压铸领域标准术语[[S22,S23,S5]]明确规定：

- **充填速度**（filling velocity，又称**内浇口速度**、gate velocity）为熔融金属在压力作用下通过内浇口进入型腔的**线速度**，单位为 m/s，属于线速度类参数
- **比浇注速度**为质量/体积流量型参数，二者物理量维度完全不同，**不存在概念混淆**[[S35]]

> 北美 NADCA 压铸标准术语体系中，gate velocity 被明确定义为流经内浇口的金属液线速度，pouring rate 作为独立的质量/体积流量类参数存在，与中文术语“比浇注速度”的核心属性完全对应[[S35,S1]]。

### 英文术语

国际压铸行业通用官方英文译名为 **specific pouring rate**，国内压铸领域习惯使用的等价英文表述包括 **specific pouring speed** 和 **specific filling rate**，三者均明确区别于指代内浇口线速度的 gate speed[[S3,S14]]。

## 计算公式

### 基本公式

比浇注速度的核心量化定义公式为[[S20]]：

```
Q_s = Q_total / A_gate
```

其中：
- **Q_s**——比浇注速度（质量流量维度或体积流量维度）
- **Q_total**——总充型流量，由待充型型腔（含溢流槽）内金属液总体积 V 除以设计充填时间 t 计算得出：
  ```
  Q_total = V / t
  ```
  其中 V 为铸件本体体积与对应溢流槽体积的总和[[S20]]
- **A_gate**——所有内浇口的总截面积

该公式是压铸 PQ² 工艺计算体系的核心参数之一[[S20,S11,S18]]。

### 质量流量与体积流量的换算

质量流量维度的比浇注速度 G（单位 kg/s）与体积流量维度 Q（单位 L/s 或 cm³/s）之间的换算关系为：

```
G = Q × ρ
```

其中 ρ 为对应压铸合金的密度[[S11]]。

## 工艺设计中的作用

### 浇注系统设计的核心指导参数

比浇注速度是压铸浇注系统设计的核心指导参数，用于校核浇注系统内浇口截面积、预判充型平稳性及潜在缺陷风险，是浇注系统参数匹配的核心计算基准[[S20]]。

### 与内浇口速度及充填时间的联动计算

在压铸模具设计环节，比浇注速度与内浇口线速度（gate velocity，单位 m/s）的量化换算关系为[[S20]]：

```
A_g = Q / (V_g × 100)
```

其中：
- A_g——内浇口截面积（cm²）
- V_g——内浇口线速度（m/s）

三者联动调整可快速完成浇注系统的参数匹配计算。

### 对铸件力学性能的显著影响

正交实验分析表明，比浇注速度（单位 kg/s）对铸件等效应力的影响极差 R 值远高于浇注温度、充型压力两项因子，是对铸件等效应力影响最显著的工艺参数[[S2]]，可见该参数在压铸工艺设计中具有核心地位。

## 取值与缺陷关系

### 取值过高的影响

比浇注速度取值过高时，金属液流动紊流程度显著提升，易产生喷溅、雾化翻卷现象，造成型腔气体被液体卷入形成卷气缺陷，最终引发铸件内部气孔；同时高速流动的裸露金属液与空气接触氧化，易形成氧化夹杂缺陷[[S17,S28,S16]]。

### 取值过低的影响

比浇注速度取值过低时，金属液充型过程中温度下降速率过快，流动性提前劣化，易产生冷隔、浇不足、流痕等铸件表面及轮廓成形类缺陷[[S27,S29,S13]]。

---

## 影响因素

影响比浇注速度选择的核心因素包括[[S9,S24]]：

| 影响因素 | 影响规律 |
|---------|---------|
| **铸件壁厚** | 薄壁件需更高比填充率保证充型无冷隔、浇不足缺陷 |
| **合金种类** | 低熔点、流动性好的锌合金可采用更高体积流量比速 |
| **铸件复杂度** | 复杂深腔、薄壁特征占比高的铸件需提升比速保证充型流畅 |
| **表面质量要求** | 电镀件、高光装饰级铸件需较高比速保证轮廓充型完整性和表面光洁度 |

## 合金类型与推荐取值

### 充填速度基准范围

国内压铸行业公认的不同主流合金充填速度（内浇口线速度）基准取值范围为[[S15,S24,S32,S34]]：

| 合金类型 | 充填速度范围（m/s） |
|---------|-------------------|
| 铝合金 | 20～60 |
| 锌合金 | 30～50 |
| 镁合金 | 20～55 |

结合对应铸件不同壁厚匹配的推荐充填时间区间，可反推得到各工况下对应的比浇注速度推荐值[[S15,S24,S32,S34]]。

### 按铸件类型细分（冷室压铸）

不同压铸合金按压铸件类型可进一步细分选取参照值[[S21]]：

| 铸件类型 | 锌合金 | 铝合金 | 镁合金 |
|---------|--------|--------|--------|
| 一般件 | 较低区间 | 中区间 | 中低区间 |
| 耐压件 | 低区间 | 低区间 | 低区间 |
| 大平面薄壁件 / 高表面要求件 | 较高区间 | 较高区间 | 较高区间 |
| 受力件 | 中等 | 中等 | 中等 |

## 与其他工艺参数的关系

### 与内浇口速度及充填时间的联动

前文公式 `A_g = Q / (V_g × 100)` 已给出比浇注速度、内浇口截面积、内浇口线速度三者的直接换算关系[[S20]]。

### 与压射比压的联动

比浇注速度与压射比压的联动关系通过伯努利衍生公式关联[[S25,S26]]：

```
v_充 = μ × √(2 × g × p_b / γ)
```

其中：
- v_充——与比浇注速度直接相关的金属液流速
- μ——阻力系数，取值通常为 0.358
- g——重力加速度
- p_b——压射比压
- γ——金属液重度

结合压射冲头流量连续性公式 `A₁ × v₁ = Q`，可进一步建立压射比压与比浇注速度的完整联动匹配规则[[S25,S26]]。

## 按工艺类型分类

不同压铸工艺类型对应比浇注速度适配规则存在明确差异[[S30]]：

| 工艺类型 | 适配特点 |
|---------|---------|
| **常规高压压铸（HPDC）** | 基准比浇注速度，按合金类型及铸件特征选取 |
| **半固态压铸** | 比浇注速度取值显著低于常规高压压铸——因金属浆料含一定比例固相颗粒，避免固相颗粒卡滞或紊流卷渣缺陷 |
| **真空压铸** | 可适配更高的比浇注速度——因型腔排气条件大幅优化，可减少冷隔缺陷风险 |
