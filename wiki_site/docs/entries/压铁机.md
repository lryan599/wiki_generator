## 概述

在铸造工业领域中，“压铁机”在现行中国国家标准体系内具有明确且单一的术语指代，特指砂型铸造造型生产线上的配套辅助设备，其标准术语编号为 GB/T 25370-2020 中的 4.2.17 项[[S14]]。该设备的核心功能是在自动化或半自动化砂型造型线上完成压铁的循环使用：合箱后向待浇注砂型上方放置压铁以防止浇注过程中抬箱跑水，待铸型冷却后取回压铁并输送回前端工位重复使用[[S9,S19]]。

需要说明的是，国内部分铸造行业场景中，也存在将用于金属切屑压块回收的“金属屑压块液压机”（如 Y83 系列设备）俗称为“压铁机”的非标准指代。此类设备在标准体系中正式归属于锻压机械大类下的打包、压块液压机类别[[S5,S20]]。

以下按该术语在标准体系下的正式指代分述砂型铸造压铁机，并单列一节说明俗称指代的金属屑压块设备，以确保技术信息的准确性与完整性。

## 砂型铸造压铁机

### 定义与术语归属

根据 GB/T 25370-2020《铸造机械 术语》，“压铁机”正式收录为铸造机械类设备，条目编号 4.2.17，归类于砂型铸造造型辅机范畴。同系列关联设备“压铁-套箱机”条目编号为 4.2.18[[S14,S17]]。该标准体系内未将该术语定义为铁合金压铸专用压铸机或挤压铸造设备。

压铁机整机由三个核心功能部分构成：取压铁机构、放压铁机构以及压铁输送系统[[S9,S4]]。

### 分类与机型

#### 升降式压铁机

升降式压铁机为砂铸压铁机的主流机型之一，依靠底部液压顶杆完成压铁的取放动作。滚道架通过销子铰接在机架上，可采用机械手、连杆等不同机构实现开合。该机型结构简单，放压铁动作平稳，但驱动液压缸需安装在地面以下，检修难度大，对压铁定位精度要求较高，适配中小批量半自动砂型造型线场景[[S9,S15]]。

#### 机械手压铁机

机械手压铁机为适配全自动造型线的压铁机机型。整机刚性强，动作定位准确、运行效率高，适配大生产节奏，但结构复杂，制造和维护成本较高。其标准工作流程为：放压铁工位的机械手将压铁移送至铸型输送机上方放置到待浇注砂型表面，浇注并冷却后，取压铁工位的机械手将压铁取回放置到输送滚道上，由输送缸驱动将压铁送回循环工位重复使用[[S9]]。

机械手压铁机由缓冲缸、放压铁机构、取压铁机构、输送滚道、压铁输送缸、机械手开合缸、机架、多组独立驱动油缸等构件组成[[S11,S9]]。

### 工艺角色与上下游关系

在砂型铸造自动化造型生产线上，压铁机典型部署于合箱机工位与铸型输送机的衔接区域。其上游对接合箱机设备，下游对接自动浇注段；浇注后的压铁回收段则对接铸型输送机冷却工位以及捅箱机前置工位。压铁机是高压自动砂型造型线的标准配置辅机[[S16]]。

对于大型砂型，行业趋向于通过加重上砂箱自身重量来取代独立的压铁[[S9,S19]]。

### 安全操作与维护

砂型压铁机常规安全操作流程包括：启动前确认设备动作行程范围内无人员滞留，液压/气动管路无泄漏；压铁取放作业区域需配套安全光电保护装置；调试阶段须切换至单步操作模式，禁止在机械手或升降臂行程范围内放置无关杂物[[S12]]。

日常维护要点包括：定期清理地下升降缸周边积砂，检查液压密封件老化情况，定期校准压铁定位行程开关，按时润滑机械手铰接运动副[[S3]]。

## 金属屑压块机（俗称“压铁机”）

### 术语界定与归类

在行业非正规用语中，将铁屑、废料压实成块的金属屑压块液压机常被俗称为“压铁机”。该设备在标准体系中属于锻压机械大类下的“打包、压块液压机”组别，正式名称为“金属屑压块液压机”（英文对应 Metal scrap briquette hydraulic press），型号代号为 Y83 系列，归类于 JB/T 9965-1999 标准中的 Y 系列液压机[[S5,S20]]。

### 工作原理与应用场景

该类设备可将经破碎、干燥处理后的铸铁、钢、铝、铜等切削加工切屑直接压制成圆柱状块料，以便于运输、存储和回炉熔炼。在压铸车间废料回收环节中，其完整上下游衔接关系为：上游对接压铸件机加工工序产生金属屑的收集输送装置及预处理离心除油/回转式干燥设备，下游对接压块成品输送装置及回炉熔炼工段[[S8]]。经过压块处理后，松散金属屑的体积可压缩至原体积的约 1/4，熔炼烧损可减少 5%～8%，并可显著缩短熔炼作业时间[[S8,S2]]。

### 分类体系

按 JB/T 9965-1999 型号编制规则及行业产品型谱，金属屑压块机的分类如下[[S5,S20,S6]]：

| 分类维度 | 类别 |
|---|---|
| 动力源 | 液压驱动型（当前主流产品）、机械驱动肘杆型、气动驱动小型 |
| 自动化程度 | 手动/半自动机型、全自动连续送料机型 |
| 适配材料 | 铁/钢屑专用、铝/铜等有色金属屑专用、复合混杂金属屑通用机型 |
| 标准系列代号 | Y83 系列（金属屑压块液压机） |

### 核心性能参数（以 Y83-400 型为例）

Y83-400 型金属屑压块液压机为典型代表产品，其核心技术参数如下[[S1]]：

| 参数项 | 参数值 |
|---|---|
| 公称压力 | 400 吨 |
| 冲头行程 | 450 mm |
| 工作频次 | 6 次/分钟 |
| 铝/钢屑压块尺寸 | φ140 × (60~90) mm |
| 铸铁/铜屑压块尺寸 | φ120 × (60~90) mm |
| 生产率 | 1.5～2.1 吨/小时 |
| 系统最高油压 | 200 公斤/厘米² |
| 总装机功率 | 81.5 千瓦 |
| 设备整机重量 | 24 吨 |

### 待压材料的技术限制要求

待压金属屑须满足以下技术条件[[S18,S13]]：

- 含油/含乳化液总占比需 ≤ 10%；
- 非金属杂质（沙粒、纤维碎屑等）占比需 ≤ 5%；
- 标准机型默认适配常温冷态金属屑；
- 配置特殊预热腔的升级机型可适配 200℃ 以内的热态余温金属屑；
- 镁合金等活性金属屑压块需额外配套惰性气体保护装置，以防止燃爆事故。

### 适用标准与安全规范

金属屑压块液压机须遵循以下国家/行业安全与性能规范[[S6]]：

|---|---|
| JB 1829 | 锻压机械通用技术条件 |
| GB 5083 | 生产设备安全卫生设计总则 |
| GB/T 3766 | 液压传动系统通用安全要求 |
| GB 26488 | 镁合金压铸安全生产规范（镁合金场景下配套适用，含粉尘防爆要求） |
