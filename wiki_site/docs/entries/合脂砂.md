## 概述

合脂砂（英文：synthetic fatty acid bonded sand）是指以合脂粘结剂为主要粘结剂配制的芯砂[[S18]]。合脂的全称为合成脂肪酸蒸馏残渣，是石蜡氧化制取合成脂肪酸过程中产生的工业副产品[[S3,S11]]。合脂砂属于油类有机粘结芯砂中的一类，是我国于1962年研制成功的干性植物油（桐油、亚麻油）低成本代用芯砂，从20世纪60年代初期应用至今，在我国铸造生产中曾获得广泛应用[[S3,S17]]。

## 合脂的来源与化学本质

合脂是石蜡氧化制取合成脂肪酸过程中的蒸馏残渣[[S17]]。C₁₀~C₂₀的合成脂肪酸用于制造肥皂，蒸馏后剩余的高分子量残渣经溶剂稀释后用作铸造粘结剂[[S17]]。合脂的组成复杂，难以精确分离，可粗略认为包含三类主要成分[[S3]]：

| 组分类型 | 说明 |
|---------|------|
| 稀碱液可溶成分 | 高碳脂肪酸（饱和与不饱和）、羟基酸 |
| 稀碱液不溶但可皂化成分 | 羟基酸的内脂和交脂，是起粘结硬化作用的主要组分 |
| 稀碱液不溶又不可皂化成分 | 中性氧化物和未氧化的蜡 |

根据所用石蜡原料的不同，合脂又分为**软蜡合脂**和**硬蜡合脂**[[S1,S11]]：

| 类型 | 石蜡熔点 | 外观 | 特性 |
|-----|---------|------|------|
| 软蜡合脂 | 30~44 ℃ | 黑褐色 | 羟基酸含量较多，黏度大，硬化强度高 |
| 硬蜡合脂 | >52 ℃ | 浅褐色 | 羟基酸含量较少，黏度不大，硬化强度低 |

铸造用粘结剂应尽量采用软蜡合脂[[S11]]。

## 分类地位

在铸造材料分类体系中，合脂砂归属于**油类有机粘结芯砂**大类[[S18]]。与其他芯砂的比较如下[[S22,S23]]：

- **与油砂（植物油砂）相比**：合脂砂的干强度、退让性、出砂性等工艺性能与油砂相近，但合脂来源于制皂工业副产品，原料成本远低于干性植物油、来源更广[[S8,S17]]。
- **与合成树脂砂相比**：合脂砂的硬化过程无需化学硬化辅助环节，烘干工艺宽容度更高[[S8]]。
- **与渣油砂相比**：合脂粘结剂具有更好的烘干性能，且烘干温度范围更宽[[S17,S19]]。

## 合脂粘结剂的配制

合脂在常温下为膏状或半固体，温度低时结成块状，使用时必须加热稀释[[S11,S14]]。

**稀释工艺**：将合脂原料加热至80~100 ℃熔融，加入稀释剂并充分搅拌至无沉淀、不分层为止[[S1,S14]]。常用稀释剂为煤油，也可使用200号溶剂油[[S17]]。煤油加入量一般为合脂粘结剂质量的44%~50%，夏季或对流动性要求不高时可降为33%~42%[[S1,S11]]。稀释比（合脂残渣与溶剂质量比）一般控制在10:6~10:10范围内[[S14,S16]]。

## 合脂砂的混制工艺

合脂砂的混制工艺与植物油砂基本相同，但因合脂黏度较大，总混砂时间较长（12~20 min）[[S16,S24]]。典型工艺如下[[S4,S16,S20]]：

1. **干混**：原砂与膨润土、糊精等粉状附加物干混2~3 min；
2. **湿混**：加入水、纸浆废液再混2~4 min；
3. **混合**：加入稀释后的合脂再混碾10~15 min后卸砂。

| 工艺参数 | 典型值 |
|---------|--------|
| 合脂加入量（占原砂质量） | 2.5%~4.5% |
| 膨润土加入量（占原砂质量） | 1%~2% |
| 糊精加入量（占原砂质量） | 1%~1.5% |
| 纸浆废液加入量（占原砂质量） | 约3% |
| 水分加入量（占原砂质量） | 2%~3% |
| 总混砂时间 | 12~20 min |

混制好的合脂砂需保存于密闭容器或用湿麻袋覆盖，避免失水干硬[[S1,S20]]。

## 工艺性能参数

### 关键性能指标

| 性能参数 | 典型数值范围 |
|---------|-------------|
| 湿强度 | 2.0~4.0 kPa |
| 干拉强度 | >1.5 MPa |
| 透气性 | 良好 |
| 发气性 | 随合脂加入量及烘干程度变化 |
| 溃散性/出砂性 | 优良（300 ℃以上快速分解，500 ℃以上碳化烧毁） |

合脂砂的性能特点[[S8,S16]]：

- **干强度高**：与油砂相近，能满足Ⅱ、Ⅲ级及部分I级砂芯要求；
- **退让性和出砂性优良**：合脂在300 ℃以上开始快速分解，500 ℃以上碳化烧毁，浇注冷却后残留强度低，出砂性好[[S16]]；
- **吸湿性低**：合脂本身为憎水材料，不加附加物的纯合脂砂芯存放后强度下降幅度小[[S8]]；
- **发气量可控**：充分烘干后发气量显著降低。发气量随合脂加入量增加呈线性上升[[S8]]。

### 合脂加入量与性能的关系

合脂加入量是影响砂芯强度和发气量的核心参数。在加入量达到约5%时，砂粒表面已完全被粘结剂膜覆盖，继续增加合脂仅会增大油膜厚度，使流动性降低、发气量显著增大、粘模加重[[S8,S19]]。

## 制芯与烘烤规范

### 制芯方法
合脂砂适用于手工制芯和机器制芯，以机器制芯为主[[S4]]。因合脂芯砂流动性较差，制芯时砂芯易出现填不满或拐角疏松，在后续烘烤中可能导致蠕变变形[[S4,S20]]。

### 烘干硬化
合脂砂的烘干硬化过程主要为不饱和脂肪酸的氧化聚合反应和羟基酸的脱水缩聚反应，**缩聚硬化反应不需要氧气**，烘干时不必考虑供氧问题，但需及时排出溶剂挥发物防止炉内爆炸[[S4,S20]]。

| 烘烤工艺参数 | 推荐值 |
|------------|--------|
| 烘干温度范围 | 180~240 ℃ |
| 最适宜烘干温度 | 200~240 ℃ |
| 烘干时间 | 2~3 h |

烘干适当的砂芯表面呈棕黄色或棕色，带有光泽；呈黄色或淡黄色表示烘烤不足，呈暗棕色或暗黑色且表面酥松表示烘烤过度[[S3]]。

## 应用场景与局限性

### 适用范围

合脂砂可替代桐油制作部分I级砂芯，绝大部分用于Ⅱ级和Ⅲ级砂芯[[S17,S21]]，适配铸铁、铸钢及非铁合金（如铝合金）铸件的生产[[S5,S21]]，尤其适用于对出砂性要求高的复杂小型芯场景[[S12]]。

### 优势

- 原料来源为石蜡制皂工业副产品，成本远低于干性植物油[[S8,S17]]；
- 干强度、退让性、出砂性与油砂接近[[S8]]；
- 吸湿性低，无附加物的纯合脂砂芯存放稳定性好[[S8]]；
- 与酸性的有机水溶性粘结剂配合使用时，干拉强度大幅提高，这是合脂粘结剂的一个主要特点[[S14,S17]]。

### 局限性与应对措施

| 局限性 | 表现 | 应对措施 |
|--------|------|---------|
| 湿强度低 | 远低于油砂 | 加入膨润土、糊精或纸浆废液提高湿强度[[S11,S16]] |
| 流动性差 | 常温黏度大，制芯易出现拐角疏松 | 降低合脂黏度、减少加入量、加入少量植物油（≤0.5%）、选用粒形较圆的原砂[[S16,S20]] |
| 蠕变变形 | 湿态及烘烤阶段砂芯逐渐变形 | 加入糊精、增加芯骨、采用成形烘板、高温入炉快速硬化[[S6,S20]] |
| 粘模倾向高 | 比桐油砂更易粘附芯盒 | 降低合脂黏度、减少水分、使用脱模剂、采用金属芯盒[[S6]] |
| 发气量较大 | 合脂添加过量后铸件气孔风险上升 | 严格控制合脂加入量在保证强度的最下限[[S11,S19]] |

## 相关标准

合脂砂所使用的合脂粘结剂执行国家标准 **GB/T 12216-1990《铸造用合脂粘结剂》**，该标准是国内合脂生产和使用单位统一执行的技术规范[[S15,S17]]。

标准主要分级与牌号规则[[S15,S17]]：

- **按粘度值分为3级**，以（N-6粘度计，30 ℃）秒数区分；
- **按工艺试样干拉强度值分为2级**；
- **牌号命名规则**：`HZ-粘度等级-强度等级`，如 HZ-40-14 表示粘度值为40 s、干拉强度为1.4 MPa的合脂粘结剂。

主要技术指标[[S10,S17]]：

| 指标 | 要求 |
|------|------|
| 酸值 | 15~70 mgKOH/g |
| 稀释溶剂 | 符合GB/T 253的煤油或符合GB/T 1922的200号溶剂油 |
