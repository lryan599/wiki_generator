## 概述

松砂机是铸造砂处理系统中的关键设备，用于对混制完成的型砂或芯砂进行松散处理，降低其容积密度，进而提升型砂的综合工艺性能[[S17,S23]]。在压铸带砂芯工艺场景中，松砂机对混制后的芯砂进行松散破团，避免芯砂结块，保障射砂充型的均匀性[[S8]]。国家标准GB/T 25370-2020《铸造机械 术语》将松砂机（sand aerator）定义为“使混制的型砂松散的机器”[[S13,S4]]。

## 定义与工艺作用

**松砂**（aeration/sand-cutting）指使型砂松散并降低其容积密度的工艺过程[[S17,S23]]。**松砂机**（aerator/sand cutter）则是执行该过程的机械设备，核心功能是松散型砂、降低容积密度，从而提高型砂的综合工艺性能[[S17,S23,S8]]。

松砂机与混砂机（sand muller）存在明确的功能边界：混砂机的核心功能是完成砂、粘结剂、附加物的混合，实现粘结剂在砂粒表面的均匀包覆；松砂机不参与组分混合与粘结成膜工序，仅承担对已混制完成的型砂/芯砂的松散破团作业[[S17,S23,S13,S11]]。

经过松砂机处理后，型砂/芯砂内部的紧实团块被完全打散，砂粒间粘结剂分布均匀，流动性和透气性显著提升，可直接提高后续砂型或砂芯的紧实度[[S8,S18]]。

## 工作原理

松砂机的松散机理主要依靠以下三类作用方式[[S21,S13,S8]]：

- **冲击打散**：高速回转的叶片或松砂轮产生的冲击力打碎砂团。
- **剪切切割**：高速回转部件产生的剪切力切割分散砂料。
- **抛甩分散**：高速旋转部件将砂料向筛网或挡链抛甩，进一步分散砂粒，使空气混入砂料中实现容积密度的显著降低。

部分带式机型还依靠带齿胶带的梳动作用完成梳松[[S8]]。

## 分类及结构组成

根据国家标准GB/T 25370-2020，松砂机分为以下四种型式[[S13]]：

| 型式 | 松砂机型式与特点对比 |
|------|----------------------|
| **梳式松砂机（comb-type aerator）** | 利用高速旋转转盘上的梳子状撞齿梳松型砂，配套弹性挡砂条，结构紧凑，通常安装在型砂输送机上[[S13,S9]] |
| **叶片式松砂机（blade aerator）** | 利用高速旋转的带辐射叶片轮打松型砂，并采用弹性挡砂条[[S13]] |
| **带式松砂机（belt aerator）** | 利用高速运行的梳齿皮带梳松型砂，并采用挡帘[[S13]] |
| **轮式松砂机（wheel aerator）** | 利用高速旋转的松砂轮切割、破碎型砂，并采用弹性挡砂条[[S13]] |

### 各型式详细特点

**叶片式松砂机**由驱动电机、进料口、高速旋转带辐射叶片的叶轮、弹性挡砂链、出料口及底座组成。工作原理为高速旋转的叶片将型砂抛射至挡链上实现撞击松散，特点是固定密封、生产率高、松砂可靠性好，但需要占用独立车间面积，适配中、大型砂处理系统[[S8,S13,S9]]。

![叶片式松砂机原理示意图，展示加料口、叶片、链条及出料口的完整工作流程](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/99d31ddc-fb7f-4dda-87bc-385b2c9b13b9/resource)

**带式松砂机**由移动轮、带横条/梳齿的环形胶带、筛网、驱动装置与装料槽组成。工作原理为高速运行的梳齿胶带将型砂抛射至筛网完成松散，特点是整体可移动、结构紧凑，但生产率较低，适配小型铸造车间背砂就地松散处理场景[[S8,S13,S9]]。

![典型移动型松砂机构造示意图，标注环状胶皮带、出口导板、调节弹簧、钢丝等核心部件](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/9b34d5cb-e14f-4a30-a1e4-60e0419a24ab/resource)

**轮式松砂机**（含双轮及三轮机型）由驱动电机、松砂轮、弹性挡砂条/挡链、托板及配套安装支架组成。工作原理为高速旋转的松砂轮对输送带上的型砂进行切割、破碎，将砂料抛向链条实现疏松。其突出特点是可直接安装在带式输送机上方，不额外占用车间空间，生产率高，松砂效果稳定，适配大、中型自动化砂处理系统的在线型砂松散及热旧砂破碎搅拌工序[[S8,S13,S9,S21]]。

**梳式松砂机**通常安装在型砂输送机上，依靠高速旋转转盘上的梳子状撞齿梳松型砂，配套弹性挡砂条，结构紧凑、松砂效果好，适配混碾后型砂的在线松散场景[[S13,S9]]。

此外，**片式松砂器**体积小巧，专门用于混砂机型砂在线检测流程中对取样型砂进行松散处理[[S9]]。

## 关键工艺参数

轮式/双轮松砂机的通用规范参数如下[[S19,S25,S10]]：

| 参数项 | 数值或范围 |
|--------|------------|
| 最大松砂生产率 | 200 m³/h |
| 松砂叶轮直径 | 260～500 mm |
| 叶轮外沿线速度 | 约12 m/s |
| 对应转子转速 | 500～820 r/min |
| 可通过砂层厚度 | 25～100 mm |
| 叶轮与输送带间隙 | 5～8 mm |
| 安装倾角 | ≤15° |
| 叶轮工作缘材质 | 堆焊硬质合金 |

试验研究表明，当再生机转子转速达到约800 r/min后，砂粒表面残留粘结剂的剥离效果趋于最优，砂芯强度达到峰值，为松砂机最优工作转速区间的选择提供了参考依据[[S7]]。

## 在压铸制芯系统中的角色

在压铸带砂芯工艺的制芯系统中，松砂机布置在混砂机出料口下游、射芯机上方砂斗的上游位置，属于芯砂制备的中间预处理工序。其作用是将混制后含有团块的芯砂打散，再供给射芯机，避免硬砂团直接进入射芯系统影响射砂填充效果[[S8,S20]]。

松砂机对砂芯及压铸铸件质量的影响机制为：
- 经过松砂处理的芯砂流动性提升，射砂填充过程中砂粒分布更均匀，可获得更高且均一的砂芯紧实度，砂芯表面轮廓清晰度与光洁度同步提升[[S8,S22]]。
- 芯砂透气性提升后，砂芯内部残留气体在压铸充型阶段可顺利排出，减少铸件内部气孔缺陷的产生概率，同时降低金属液渗入砂粒间隙导致的机械粘砂风险[[S8,S3]]。

## 与替代方式的对比

| 松散方式 | 特点与局限性 |
|----------|-------------|
| 人工松散 | 仅适合小批量极小规模生产，效率极低，均匀度完全依赖操作人员经验[[S8,S1]] |
| 振动松砂 | 依靠激振力打散砂团，松散效果差，无法完全破碎硬砂团，仅适配初步松散需求[[S8]] |
| 机械松砂（松砂机） | 松散均匀度远高于前两种方式，连续作业能力强，适配规模化压铸砂芯生产线[[S8]] |

## 安装与维护

不同类型松砂机的安装特性差异明显：轮式松砂机直接集成安装在输送带上，无需额外预留独立安装空间，占地面积最小；带式移动松砂机占地面积小，可灵活移动部署；叶片式松砂机需要独立基础放置，额外占用车间面积[[S10,S8]]。

松砂机整体结构简单，维护工作量低。主要维护要点包括[[S21]]：
- 定期检查松轮叶片磨损情况，磨损后可堆焊硬质合金修复，完成动平衡校验后即可继续使用；
- 日常调整V带张紧度；
- 控制松砂轮与输送带间隙在6～10 mm范围内；
- 设备出厂需完成不少于1 h连续空运转试验，核心轴承温升不得超过35℃；负载运行试验需连续运行不少于2 h，最大工作电流不超过电机额定电流。

## 型号命名与相关术语

松砂机属于铸造机械S类砂处理设备下的独立列，主流国产量产系列包含[[S9,S2]]：
- S32系列叶片式松砂机
- S34系列梳式松砂机
- S35系列双轮松砂机
- S388型带式移动松砂机

常用中文别名包括松砂设备、砂处理松砂机[[S2]]。标准英文名称为sand aerator，常用别名包括sand cutter、sand loosener，部分行业场景下亦称为sand fluffer[[S13,S5,S3]]。
