## 灼减

在铸造及压铸工程语境中，“灼减”并非单一术语，而是存在两类完全独立的应用场景，二者在概念上无重叠[[S37,S9]]。第一类是**造型材料测试领域的标准术语“灼烧减量”（loss on ignition，LOI）**，指型砂、涂料、覆膜砂等非金属材料在高温灼烧后的质量损失率[[S37,S14]]。第二类是**压铸行业车间口语中对金属熔炼损耗（烧损）的简称**，指合金在熔炼过程中因氧化、蒸发等造成的不可回收质量损失[[S39,S9]]。本词条以压铸熔炼场景为核心，同时依据国家标准明确定义灼烧减量，并对二者进行辨析。

### 标准定义

**灼烧减量（灼减）** 在国家标准 GB/T 5611-2017《铸造术语》（条文 4.6.46）中有明确定义：将砂样经 105 °C～110 °C 烘干排除游离水后，在 950 °C～1000 °C 高温下灼烧至恒重，其失重占烘干砂样总重量的百分率[[S37]]。

这一概念适用于铸造造型材料（型砂、覆膜砂等），其测量方法与铝冶炼行业的灼减测定在原理上相近——有色金属冶炼通用流程同样采用将 1.0000 g 试样在 950～1000 °C 灼烧 1.5 h 后称重，并按灼烧前后质量差除以灼烧前试样质量计算[[S27]]。

**压铸场景下的熔炼损耗（俗称“灼减”）** 则完全属于另一范畴。国家标准《铸造术语》将其定义为：熔炼过程中，由金属氧化、蒸发等原因造成的金属炉料损耗，即投炉金属总重量与最终收得金属总重量之差[[S39]]。压铸行业通用手册将此标准称为“熔炼损耗”或“熔损”“烧损”，其对应的英文标准术语为“total melting loss”或“melting loss of alloying element”[[S39,S48]]。

### 压铸场景的使用习惯与等效概念

在国内压铸车间及生产管理实践中，工程师常将金属熔炼过程中发生的氧化烧损口头称作“灼减”，但该用法不属于国家标准定义的正式术语范畴[[S9,S37]]。行业手册在涉及配料计算时，通常使用“烧损”“烧损率”“熔损”“总熔炼损耗”等标准表述[[S39,S12]]。

英文术语使用习惯同样明确区分两类场景:
- **元素烧损**对应标准术语为“melting loss of alloying element”[[S48]]
- **熔炼损耗（综合）** 对应“total melting loss”或“burning loss”“metal loss”“oxidation loss”[[S39,S48]]
- **灼烧减量**对应“loss on ignition (LOI)”或“ignition loss”[[S37,S48]]

在压铸生产全周期场景中，“总熔损”包括了合金熔炼过程中不可回收的氧化烧损以及浮渣中因扒渣作业被连带带出的可用金属[[S12,S11]]。压铸行业常规工况下，通过小规模称重法测得的典型烧损区间为：铝合金 2%～3%，锌合金 1%～2%，铜合金与镁合金的烧损值更高[[S41,S11]]。

### 影响因素

#### 合金元素成分

合金元素对氧化烧损的影响由两方面共同决定：该元素氧化物生成后体积与消耗金属体积之比（α 值），以及该元素对氧的亲和力（由氧化物生成热与分解压反映）[[S45]]。α > 1 时氧化膜致密，能阻止内部金属继续氧化；α < 1 时氧化膜疏松，氧可穿透至金属内部[[S45]]。

铝合金中常见元素的烧损率从大到小排序为：Mg > Al > Si > Mn > Zn > Cu[[S4]]。镁、锂等活泼元素会显著抬高合金整体的灼减水平，在高镁铝合金中加入 0.001%～0.005% 的铍可以大幅降低易氧化元素的烧损率[[S4,S25]]。

#### 熔炼温度与保温时间

铝合金的氧化行为在温度转折点上存在明确规律：400 °C 以下氧化遵循抛物线规律，表面生成的致密 Al₂O₃ 氧化膜可有效阻止内部金属进一步氧化；500 °C 以上氧化转为直线规律，750 °C 时表面氧化膜易发生破裂，原子扩散速度大幅提升，氧化烧损速率随之显著加快[[S36,S8,S30]]。常规压铸铝合金熔炼温度需控制在 750 °C 以下，并在实际工艺中尽可能将熔化温度控制在 780 °C 以下，以抑制 Sr、Mg 等易氧化活性元素的过量烧损[[S4,S29]]。

保温时长与合金整体灼减水平呈正相关。高镁铝合金在常规熔剂覆盖下，随保温时间延长，镁元素含量大幅下降——初始镁含量 7.0% 的 AMr7 合金熔体保温 14 h 后镁含量可降至约 5.4%[[S7]]。工业生产中要求尽可能缩短熔体高温停留时间[[S36]]。

#### 炉气氛围与炉型

铝、镁等压铸合金对氧亲和力极强，炉气中的 O₂、CO₂ 和 H₂O(g) 均属于氧化性组分，会加剧合金的氧化烧损[[S28]]。不同炉型因加热方式和熔池表面积不同，其典型灼减区间存在明显差异[[S2,S28]]：

| 炉型 | 铝合金灼减率参考值 |
|---|---|
| 低频感应炉 | 0.4%～0.6% |
| 电阻反射炉 | 1.0%～1.5% |
| 火焰反射炉 | 1.5%～3.0% |

![金属氧化核心影响因素示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/08758961-9ee9-4af8-85d0-3e61b3902020/resource)
*金属氧化速率直接由炉内空气含量、炉内熔炼温度、熔炼保温总时长和加热方式四类核心要素决定，此关系同样适用于压铸合金灼减分析* [[S1]]

#### 炉料形态与加料工艺

相同熔炼条件下，不同形态炉料的灼减率差异极大[[S45,S18,S28]]：

| 炉料形态 | 灼减率参考值 |
|---|---|
| 重熔铝锭 | 0.8%～2.0% |
| 大块回收废料 | ≈1.5% |
| 打捆薄片废料 | 3%～10% |
| 碎屑类炉料 | 13%～30% |

工业生产中采用“大块规整炉料先装、细碎废料后入炉、易氧化金属最后加入”的加料顺序，并在条件允许时将铝屑等细碎料提前压块处理，可大幅降低熔体比表面积，减少与氧化性炉气的接触时长[[S8,S4,S25]]。

### 典型合金数值范围

综合不同炉型和保护条件，主流压铸合金在常规工业生产工况下的典型灼减参考区间如下[[S9,S2,S21]]：

| 合金牌号 | 工况条件 | 灼减率参考值 |
|---|---|---|
| ADC12（压铸铝合金） | 中频感应炉 | 0.4%～1.0% |
| | 电阻反射炉 | 1.0%～1.8% |
| | 火焰反射炉 | 1.5%～2.5% |
| A380（压铸铝合金） | 中频感应炉 | 0.5%～1.2% |
| | 电阻反射炉 | 1.1%～2.0% |
| | 火焰反射炉 | 1.6%～3.0% |
| AZ91D（压铸镁合金） | 惰性保护气氛 | 1.2%～2.5% |
| | 普通空气熔炼 | 3%～6% |
| ZZnAl4Y（压铸锌合金） | 常规热室压铸熔炼 | 0.3%～0.8% |

锌合金因熔化温度低且一般不进行精炼处理，烧损总量仅为 1%～2%，其整体灼减水平显著低于铝合金和镁合金[[S9,S41]]。铝合金行业全流程总灼减率（含多次废料复熔）通常波动在 2.5%～5%[[S12]]。

### 对生产的影响

在铝合金压铸生产中，金属铝原材料是占比最高的耗材成本项，常规铝锭成本约占产品总生产成本的 45%～50%。若实际灼减率较行业基准值 1.5% 高出 2 个百分点，原材料成本上升比例将超过 4%[[S32]]。

过高的灼减率会直接导致熔体中氧化物夹杂含量同步抬升——灼减率每升高 1 个百分点，Al₂O₃ 等夹杂物含量可升高 0.3～0.5 个百分点[[S31]]。这些高浓度非金属夹杂物进入压铸熔体后，成为铸件产生内部气孔、针孔、硬质点及分层缺陷的核心诱因，显著提高力学性能不合格和渗漏不良的报废率[[S47,S44]]。

### 减控措施

**覆盖剂与惰性气体保护**——在铝合金熔体熔化初期添加专用压铸级覆盖剂，在熔池表面形成连续隔离防护层，可降低灼减率 0.5～0.8 个百分点并同步减少熔体吸气量[[S34]]。采用氮气/氩气喷射精炼系统搭配旋转除气装置对熔炉气氛进行保护，可将最终合格熔体的灼减率稳定控制在 0.8%～1.0% 的低水平区间[[S34]]。

**炉体优化与快速熔化**——采用小型熔池工频感应快速熔化炉替代传统大熔池反射炉，可将铝合金基础灼减率从 2%～3% 降至 0.5%～0.8%[[S16]]。

**低温短时间熔炼**——将铝合金熔化温度控制在 780 °C 以下，尽可能缩短合金液高温停留时间，可避免 Sr、Mg 等易氧化活性元素的过量烧损，在降低灼减的同时保证合金成分符合设计要求[[S29]]。

**浮渣管控**——铝液保温炉炉面的灰白色浮渣层是氧化产物的主要可视化聚集区域。规范打渣操作、充分搅拌浮渣中的熔剂并保证充足反应时间，可有效降低浮渣中的可用金属含量，减少因扒渣作业导致的附加金属损失[[S3,S11]]。

*压铸铝合金保温炉打渣前的炉面状态，灰白色浮渣层覆盖整个液面，此部分氧化产物是金属灼减的主要来源组成之一* [[S3]]

### 测量方法与计算

#### 金属熔炼损耗的称重法

压铸行业通用的单炉熔炼损耗率标准定义式为[[S46]]：

> 熔铸损耗率 α = Ws / W × 100%

其中 Ws 为熔铸过程中损耗的金属质量，W 为投入炉内的金属总质量（即“灼减率 = 熔炼前后金属质量差 / 初始投入金属总质量 × 100%”）。

对于更精确的全周期金属衡算（亦称“金属会计”），需统计期初全部库存金属总重量（含库存金属锭、炉内留存金属、在制铸件、成品待发件、浇道及废件、外送待处理废料、客户拒收待检铸件等），加上统计周期内全部金属入库重量，再减去统计周期内全部金属流出量（含发运铸件净重、外发协作件、退回供方料件等）和期末库存金属总重量，二者差值即为周期内的总损耗金属量[[S11,S6,S20]]。最终测得的损耗率允许误差约为 ±½%[[S42,S6]]。

*压铸全流程中金属从入炉到铸件产出的物流走向及各节点损耗分布示意，全周期衡算方法覆盖熔炼、浇注、后处理等全部工序* [[S13]]

#### 灼烧减量的标准测量方法（造型材料）

依据 GB/T 7143-1986 标准，灼烧减量试验操作步骤为[[S14]]：称取 1 g ± 0.0001 g 试样置于已恒重的铂坩埚中，放入高温加热炉内逐渐升温至 1000～1050 °C，保温 30 min 后取出置于干燥器中冷却至室温称量，如此反复灼烧（每次 15 min）直至恒重。灼烧减量按下式计算：

> w(G) = (m₁ − m₂) / m × 100%

其中 m₁ 为灼烧前试样和坩埚的质量，m₂ 为灼烧后试样和坩埚的质量，m 为试样质量[[S14]]。

典型应用场景中，铸造用覆膜砂按其灼烧减量进行分级，不同代号的灼减上限不同[[S15,S22]]：

| 覆膜砂代号 | 灼烧减量上限（质量分数） |
|---|---|
| 15 | ≤ 1.5% |
| 20 | ≤ 2.0% |
| 25 | ≤ 2.5% |
| 30 | ≤ 3.0% |
| 35 | ≤ 3.5% |
| 40 | ≤ 4.0% |
| 45 | ≤ 4.5% |

### 应用场景与质量管控

灼烧减量（LOI）的典型应用场景为**造型材料测试和质量管控**。在呋喃树脂砂铸造生产中，再生砂的灼烧减量与残留氮含量、残留硫含量呈线性正相关[[S35]]。行业通用控制要求将再生砂的灼减值控制在 3% 以下，部分先进工厂则将其控制在 1%～2.5% 区间[[S33,S35]]。灼减过高会显著增加型砂发气量，升高铸件产生气孔、球化衰退等缺陷的风险[[S35,S33]]。

![旧砂灼烧减量与残留氮含量关系图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/29a87d68-e6a2-47a0-922f-fc117831d705/resource)
*再生砂（旧砂）的灼烧减量（质量分数）与残留含氮量呈线性正相关，灼减值越高则砂中有机物残留越多，发气量随之增大* [[S5]]

在金属熔炼场景中，灼减/烧损率的直接工程意义在于：
- **配料计算**：必须计入各元素的预期烧损率（尤其是镁、锌等易烧损元素），以保证最终合金化学成分符合设计要求[[S9,S10]]
- **成本核算**：通过建立金属平衡表，精准衡量年度金属总损耗与铸件发运量之间的关系[[S11,S20]]
- **炉况监控**：持续追踪熔炼损耗率的波动可作为判断炉况异常、覆盖剂失效或操作不规范的重要指标[[S11]]

### 概念辨析

**压铸行业俗称的“灼减”** 与国家标准中正式定义的 **“灼烧减量”** 是两个完全独立的技术概念，使用者需根据上下文区分[[S37,S9]]：

- 若讨论对象为**金属熔炼的烧损率**（投炉金属量与收得金属量的差值），标准正式名称应为“熔炼损耗”“熔损”或“元素烧损”，英文对应 melting loss / burning loss / metal loss[[S39,S24]]
- 若讨论对象为**造型材料的有机物和可挥发物含量**（按 GB/T 5611-2017 条文 4.6.46 执行试验），标准正式名称应为“灼烧减量/灼减”，英文对应 loss on ignition (LOI)[[S37,S14]]

二者在测试方法、适用材料、统计单位、工程意义上均无任何重叠。压铸工厂在实际车间管理中常混用“灼减”一词指代烧损，但在撰写工艺文件、物料衡算报告和学术文献时，建议严格使用国家标准定义的规范术语，以避免歧义。
