## 概述

分流锥（sprue spreader）是压铸及挤压铸造模具浇注系统的核心组成部件，国家标准 GB/T 8845-2017《模具 术语》已正式收录该术语，条目编号为 2.1.1.66 [[S40]]。在压铸行业内，“分流器”是分流锥的通用等效别名，行业广泛通用；而“分流块”仅在部分简化设计、非锥形小型分流结构的应用场景下使用，属于局部约定俗成称谓 [[S57,S31,S25]]。英文通用标准术语为 **sprue spreader**，北美行业内亦常用 spreader、sprue pin 等简称 [[S40,S52,S46]]。

分流锥安装在直浇道的末端位置，分别与浇口套、横浇道区域配合，在立式冷室压铸机、卧式冷室压铸机、热室压铸机以及多腔型挤压铸造模具中均有应用 [[S57,S60,S62]]。

## 工艺作用

分流锥在浇注系统中的核心功能包括 [[S31,S45,S62,S60]]：

- **引导合金液平稳转向**：使金属液完成约90°的流向改变，避免直冲型壁。
- **均匀分配金属流**：将单股金属流均匀分配至各条横浇道，实现多腔模具的充填同步。
- **降低紊流与卷气**：将直浇道原本的圆形流动截面转换为环形截面后导入横浇道，有效降低紊流程度，减少充填过程的卷渣卷气倾向。
- **辅助脱模**：金属液凝固后包紧在分流锥锥面上，开模时将直浇道凝料从定模侧带出。

## 分类与结构特征

### 常见结构形式

| 结构类型 | 特征 | 适用场景 |
|---|---|---|
| 圆锥形分流锥 | 导向效果好，结构简单，应用最广泛 | 将金属液向四方分流的常规场景 [[S61,S17,S2]] |
| 中心设置推杆式 | 推杆与分流锥配合间隙实现辅助排气，利于推出直浇道 | 大尺寸直浇道的压铸场景 [[S61,S10]] |
| 圆锥面带圆环浅槽式 | 增大金属液冷凝收缩时的包紧力，便于从定模带出直浇道；但对金属流导向有不利影响 | 仅在特定工况下使用 [[S61,S2]] |
| 偏心圆锥形 | 将液态金属引向单一特定方向 | 单型腔侧向分流、侧浇口的压铸模具场景 [[S61,S10]] |
| 台阶式组合分流锥 | 结构简单、导向效果好、使用寿命长、易于拆装和更换，采用台阶式压板配螺钉固定 | 多用于间接式挤压铸造模具 [[S36]] |

### 固定方式

分流锥在模具中的固定方式主要有以下几种 [[S9,S36,S19]]：

- **台阶式压板配螺钉固定**：台阶式组合分流锥普遍采用此方式，在压铸与挤压铸造模具中均有应用。
- **单板螺接**：使用长螺栓和销钉将分流锥固定在单块分流支架板两侧，需要型芯截面上有足够螺栓安装空间，属于早期常用的固定方式。
- **双板螺接**：采用两块分流支架板分别用螺栓和销钉固定型芯和分流锥，可避免固定螺栓位置干涉，便于设置内筋内流道，是目前主流采用的固定结构。
- **锥芯一体固定**：将型芯和分流锥作为整体零件加工，通过分流筋和镶键嵌接到分流支架板上，适合不需要复杂内流道的小型场景。

### 横向浇道结构

面向双向或多向流动的间接式挤压铸造模具，可在分流锥本体上直接开设横向浇道，使合金液平稳换向，顺畅经过内浇道进入多个型腔，实现多向分流的充型效果 [[S36]]。

### 材料与表面处理

分流锥常用制造材料为 **3Cr2W8V** 或 **H13** 热作模具钢。热处理与表面处理工艺路径如下 [[S62,S24]]：

1. 半精加工后进行调质处理；
2. 精加工后进行真空热处理，使硬度达 **50～55 HRC**；
3. 或采用较低硬度（35～40 HRC）的真空热处理后，进行表面氮化处理：氮化层深度 **0.2～0.4 mm**，表面硬度达 **900～1200 HV**，显著提升表面耐磨性能。

与合金液接触的所有表面不允许存在裂纹、发裂及各种孔洞等缺陷 [[S62]]。部分量产模具的分流锥内部会设置冷却水道以延长使用寿命 [[S60]]。

## 设计与工艺要点

### 安装位置与尺寸参数

分流锥常规安装在主流道末端或分支位置。核心设计参数推荐如下 [[S43,S18,S47,S62]]：

| 参数 | 推荐值 |
|---|---|
| 单边锥角（一般） | ≤ 60° |
| 卧式冷室压铸机圆锥形分流锥锥角 | 30°～60° |
| 锥尖球面圆角 R | R2～R3 |
| 径向间隙 | ≥ 3 mm |
| 导流面高度 | 可匹配压头直径按经验公式确定 |

北美压铸协会关联教学资料定义 sprue spreader 为布置在动模侧的锥形销，单边斜度约 **5°**，合模时伸入定模侧浇口套的锥形孔内，与浇口套配合形成供合金液流通的环形通道 [[S52]]。

### 热室压铸模具中的设计

热室压铸模具中分流锥设置在定模侧动模镶块位置，需保证直浇道从定模侧顺利脱出，分流锥旁需配套推杆同步推出直浇道 [[S14,S55,S16]]。分流锥与浇口套间可预留 **1 mm** 左右间隙，以容纳冷金属杂物 [[S55]]。

### 冷室压铸模具中的设计

立式冷压室场景下，分流锥需满足流道截面积校验公式 [[S19,S20]]：

$$d_1^2 - d_2^2 = (1.1 \sim 1.3)d^2$$

$$\frac{d_1 - d_2}{2} \geq 3\text{ mm}$$

式中 $d_1$ 为直浇道底部与分流锥所形成环形的外径，$d_2$ 为内径，$d$ 为喷嘴导入口小端直径。部分带推杆的分流锥结构可兼具排气、辅助推出直浇道的功能 [[S19]]。

### 挤压铸造模具中的设计

挤压铸造模具中分流锥多用于一模多件的间接挤压铸造场景。当需要实现双向或多向合金液流动时，分流锥上需开设对应横向浇道，引导合金液平稳转向分流进入多个型腔，避免直接冲击型腔壁形成卷气缺陷 [[S62,S36]]。

### 更换与维护

组合式结构分流锥采用螺纹或台阶压板螺钉的分体固定方式，可实现快速拆装更换，无需将整个分流锥从模具整体拆除，维护更换工时大幅缩短，适配分流锥易损、需定期更换的使用需求 [[S36,S19]]。

## 对铸件质量的影响

分流锥表面必须保持光滑无烧损，方可避免产生紊流裹气现象 [[S42]]。经数值模拟验证，优化设计的分流锥可引导合金液均匀流向型腔外围，液面波动小，充型过程平稳无明显卷气，铸件整体缺陷孔隙率低于 1%，组织致密度显著提升 [[S7]]。

在升液盆顶端增设分流锥后，可在充型拐点位置抑制充型速度持续上升，消除金属液前沿回落卷气现象，大幅降低铸件气孔缺陷产生概率 [[S56]]。

## 应用案例：18″管钳柄挤压铸造模具分流锥

18″管钳柄间接挤压铸造模具采用**组合式分流锥**结构，具体特征如下 [[S36,S29]]：

- **固定方式**：台阶式压板加螺钉固定。
- **分流设计**：分流锥上开设横向分流浇道，实现合金液双向分流。
- **工艺参数**：比压 60 MPa，充填速度 0.8 m/s，充型时间 0.9 s，持压时间 10～18 s，生产节拍 50～70 s。
- **生产效果**：模具成品率 95% 以上，所生产的铝合金管钳柄抗弯扭强度符合产品要求。

## 缺陷与失效模式

### 使用寿命

分流锥属于压铸/挤压铸造模具中的易损零件。正常使用寿命短的数个班、数周不等，长的可达数月至半年 [[S54,S35]]。失效后经多次修复及上机使用，修复成本过高或无法再修复至合格时即予报废 [[S35]]。

### 主要失效形式

分流锥服役过程中的核心失效形式涵盖以下五大类 [[S49,S1,S26]]：

| 失效形式 | 产生机理 | 对铸件质量的影响 |
|---|---|---|
| 冲刷磨损 | 高速熔融金属（gate 流速可达 30～60 m/s）反复冲击分流锥表面，冲应力超过材料屈服强度，逐步造成表层材料损耗，形成犁沟、凹坑 [[S11,S63,S27]] | 表面凹凸不平，改变导流路径，导致铸件表面纹路不均、进料流痕、夹渣缺陷；剥落碎屑进入型腔造成内部夹杂超标 [[S49,S13]] |
| 熔蚀（冲蚀） | 分流锥表面长期与 600℃ 以上熔融铝合金接触，模具钢中铁元素与铝元素发生扩散反应生成铁铝金属间化合物，随高速金属流剥离脱落，常伴随粘模（soldering）现象 [[S6,S33,S39]] | 表面材料逐步损耗，改变分流锥外形轮廓，影响金属液流态 |
| 热疲劳开裂 | 每个压铸循环中交替接触高温金属液和冷却介质，表层反复承受压缩/拉伸应力，在数万次热交变循环后萌生网状微裂纹（热龟裂），占铝材压铸模具总失效的 **60%～70%** [[S6,S49,S23,S1]] | 熔融金属液挤入裂纹加速扩展，渗出的金属液形成飞边/批锋缺陷，破坏金属液流态导致铸件内部气孔率超标 [[S1,S22,S41]] |
| 高温塑性变形 | 分流锥局部长期受高压金属液冲击，工作温度超过模具钢高温屈服强度时，发生不可逆塑性流变，出现局部塌陷、锥面轮廓变形 [[S49,S58]] | 导流轮廓精度下降，改变充型速度和流道截面积，导致充型不均、局部冷隔、缩松缺陷，严重时造成浇口余料脱模困难 [[S21,S50]] |
| 断裂 | 分型面位置不当形成的横向毛刺卡滞在分流锥与浇口套间隙中，造成额外机械应力集中，最终引发整体断裂；分流锥头部残留未熔合金属、顶出阻力过大也会加速断裂 [[S34]] | 浇注系统完全失效，导致铸件大面积缺肉、浇不足，金属液窜流甚至出现分型面跑火缺陷 [[S34,S53]] |

### 延缓失效的优化措施

1. **材料选型与热处理优化**：常规选用 3Cr2W8V 或 H13 钢，真空热处理后硬度 50～55 HRC，再进行表面氮化处理（深度 0.2～0.4 mm，硬度 900～1200 HV）[[S62]]。
2. **梯度复合结构**：分流锥头部镶嵌/堆焊钨钢材质，兼顾基体强韧性和工作面耐熔蚀性能 [[S37]]。
3. **表面强化工艺**：采用等离子体源离子注入工艺在分流锥工作面沉积钨金属涂层，可显著提升抗冲蚀磨损性能 [[S51]]。
4. **结构设计优化**：分流锥在垂直分型面方向需低入分型面 **3～5 mm**，避免横向毛刺卡滞引发早期断裂；锥面不应设计尖角过渡，锥面与底面之间预留 **2～3 mm** 垂直面避免流道死角滞留金属 [[S34,S12,S50]]。
5. **冷却系统优化**：分流锥内部内置带导流结构的水冷/风冷通道，强制带走工作面热量，降低温度梯度，减少热交变应力，抑制热疲劳裂纹萌生 [[S52,S54]]。
