## 概述
石墨钢（graphite steel/graphitic steel，德语为 Graphitstahl）属于钢范畴的特种自润滑材料，我国典型商用牌号为 SiMnMo，又称石墨模具钢，为国内自主研发的冷作模具钢。成分上为高碳高硅设计，需经过专门的石墨化退火工艺才可析出一定量的游离石墨，以此获得天然自润滑特性，整体归类不属于铸铁大类 [[S28,S44]]。根据 GB/T 5611-2017《铸造术语》，石墨钢是通过热处理使钢中碳化物部分分解、析出游离石墨的特殊钢种，基体保留钢的基本力学特征，内部弥散分布游离石墨相 [[S40]]。

## 概念边界

### 与灰铸铁的区别
石墨钢与灰铸铁的概念边界清晰：灰铸铁碳质量分数通常为 2.7%～4.0%，属于铸铁大类，石墨是液态金属凝固过程中直接析出的连续片状石墨，对金属基体割裂作用强烈，抗拉强度远低于同基体钢材；而石墨钢属于钢范畴，游离石墨是固态热处理过程中析出的弥散细小石墨，基体为典型钢的马氏体/回火马氏体组织，力学强度远高于灰铸铁 [[S23]]。

### 与石墨化钢的区别
石墨钢与石墨化钢（graphitized steel）的概念边界清晰：石墨化钢是泛指在长期高温服役下或经特定热处理发生渗碳体分解、析出游离石墨的各类钢材的泛称，并非特指人为可控添加特定成分、定量析出游离石墨的专用石墨钢；其中 Graphitic Nitralloy（氮化石墨钢）属于特种氮化钢品类，碳质量分数为 1.2%～1.3%，析出的石墨为回火碳形态，不属于商用石墨模具钢的品类 [[S6,S32]]。

## 典型牌号与标准状况
经对中国 GB、美国 ASTM、日本 JIS、德国 DIN 公开可查的铸钢、工具钢、模具钢相关标准检索，未发现上述四大标准体系中明确标注为石墨钢的统一通用收录牌号；仅国内行业应用领域将 SiMnMo 石墨模具钢作为典型商用石墨钢使用，该牌号未被收录进 GB/T 1299 及公开可查的 JB/T 工具钢标准条目，无明确可溯源的国际对应标准牌号，该牌号的标准身份存在不确定性 [[S35]]。

## 化学成分与石墨化元素作用

### 基础化学成分
石墨钢典型基础化学成分范围（质量分数）：C 为 0.4%～1.2%，Si 为 0.6%～2.0%，Mn 为 0.3%～1.5% [[S33,S5]]。

### 元素石墨化作用梯度
各元素对石墨化的作用呈梯度分布：C、Si、Al、Ni、Cu、Zr 为强烈/中等促进石墨化的元素，Nb 为石墨化中性元素，V、W、Mn、Mo、S、Cr 为阻碍石墨化的元素 [[S5]]。其中硅的促进作用最为显著 [[S29,S41,S37]]。

### 石墨化因子
石墨化因子 K 是评价合金凝固时按稳定系转变析出石墨倾向的标准参数，计算公式为 K=4Si[1 - 5/(31C + Si)]/3，K 值越大石墨化倾向越高 [[S24]]。

## 组织特征

### 石墨形态
经石墨化退火析出的石墨多为细小点状、团絮状，尺寸覆盖 1 μm 至 20 μm，按 GB/T 7216 标准可分为 8 个长度等级，均匀弥散分布于钢基体中，无明显枝晶偏聚 [[S34,S30]]。石墨面积率是评定单个石墨形状的标准指标：石墨面积与其外接圆面积之比 0.81～0.98 为球状石墨，0.61～0.80 为团状石墨，0.21～0.40 为蠕虫状石墨 [[S24]]。

### 基体组织
石墨钢基体组织参数可通过热处理调控：完全石墨化退火状态下，基体中铁素体占比高于 90%，几乎无游离渗碳体；正火/调质状态下，基体中珠光体占比可达 70%～95%，可匹配不同强度与自润滑性能需求 [[S20,S1]]。

## 生产与制造工艺

### 冶炼与脱氧
石墨钢冶炼过程中需调控合金元素组分。采用 Al 作为脱氧剂时，氧化铝微粒可作为石墨非均质形核核心，大幅缩短石墨化的孕育期，加速石墨形核过程 [[S37]]。硫的质量分数需低于 0.15%，避免抑制渗碳体分解 [[S29]]。

### 锻造规范
以 SiMnMo 为代表的石墨钢锻造规范为：始锻温度 1040～1100 ℃，终锻温度不低于 850 ℃，锻后采用砂冷；锻后基体无游离石墨 [[S44]]。

### 石墨化退火
石墨化退火是石墨钢的专属必要热处理工序，作用是在锻后原本无游离石墨的钢基体中，可控析出定量的弥散游离石墨，使石墨钢获得天然自润滑特性，该工艺不属于普通商用钢材的常规热处理流程 [[S44]]。

依据 GB/T 5611-2017 铸造术语标准，石墨化退火分为两类：低温石墨化退火加热温度区间为 720 ℃～750 ℃，作用是降低石墨钢硬度、使部分共析渗碳体分解；高温石墨化退火温度区间为 900 ℃～980 ℃，用于获得铁素体基体或完成第一阶段石墨化反应 [[S24,S40]]。

石墨化退火全流程关键控制参数如下 [[S31,S21]]：

| 阶段 | 温度/参数 | 工艺要点 |
|------|-----------|----------|
| 升温 | ≤100 ℃/h | 避免白口基体因升温过快产生开裂 |
| 第一阶段高温石墨化 | 940～970 ℃ | 壁厚≤25 mm 工件保温 3～4 h，完成共晶渗碳体全部分解；不超过 980 ℃ |
| 中间冷却 | 冷速可至 200 ℃/h | 不会析出二次渗碳体 |
| 第二阶段石墨化（方案一） | 720～760 ℃ | 以 2.5～3 ℃/h 速度连续缓冷，完成奥氏体全部分解 |
| 第二阶段石墨化（方案二） | 680～700 ℃ | 保温 14～20 h，实现共析渗碳体完全分解 |
| 最终冷却 | 650 ℃左右出炉空冷 | 消除内应力，避免缓冷脆性 |

退火温度过高会导致碳原子扩散速度过快，石墨分枝程度大幅提升，形成鸡爪状劣质石墨，直接降低材料强度与韧性；退火温度不得超过 980 ℃。搭配合理的保温时长，使石墨晶核各个生长方向的碳原子扩散速率趋于均等，最终获得团絮状均匀分布的石墨组织 [[S22,S12]]。

### 基体组织调控
通过调整最终冷却速度可获得不同基体组织：高温石墨化完成后快速通过共析转变温度区间，可获得珠光体基体+团絮状石墨的组合，提升材料高温强度与耐磨性；缓慢冷却完成第二阶段石墨化，可获得铁素体基体+团絮状石墨的组合，提升材料抗冲击性能与导热系数 [[S12]]。

## 核心性能参数

### SiMnMo 工艺性能
以 SiMnMo 为典型代表，经石墨化退火后硬度 ≤269 HB，需严格控制游离石墨含量在 0.2%～0.4% 区间以获得最优自润滑效果。790～820 ℃ 油冷淬火后硬度可达 HRC 62～64，250 ℃ 以下回火后硬度仍保持 HRC 60～63，抗拉强度为 140～150 kg/mm²，抗弯强度为 180～200 kg/mm²，热处理后相对尺寸变化量仅为 +0.02%～+0.06%，尺寸稳定性极佳 [[S44]]。

### 压铸工装用石墨钢实测性能
压铸工装用石墨钢的公开实测核心性能参数如下 [[S26,S42,S9]]：

| 性能指标 | 典型值/范围 |
|----------|-------------|
| 室温导热系数 | 40～70 W/(m·K)（显著高于常规 H13） |
| 常温抗拉强度 | ≥450 MPa（部分高牌号可达 700 MPa 以上） |
| 硬度范围 | 180～350 HB（淬火后最高可达 55 HRC） |
| 耐热疲劳性（Coffin 常数） | 石墨球数从 55 粒/mm² 提升至 75 粒/mm² 可提升 25%；片状石墨使常数下降约 50% |
| 摩擦系数 | 比常规不含石墨模具钢降低约 30% |
| 磨损失重 | 同等工况下降低 40% 以上 |

## 在压铸中的应用

### 适用性限制
石墨钢（SiMnMo）作为典型自润滑模具钢，适用场景为冷作模具工况：可用于制造尺寸精度要求高、润滑条件较差的拔丝模、拉伸模、冷冲模，因导热系数较低，不适用于高温压铸的服役环境 [[S44]]。

### 服役部件（广义石墨钢/高石墨铸铁在近似工况的应用）
以下为石墨钢或石墨分散合金在压铸近似工况的使用场景，具体温度和热负荷环境需视牌号而定：

| 部件 | 服役特点 | 选择理由 |
|------|----------|----------|
| 冲头 | 长期承受高压推挤熔融金属、往复运动摩擦 | 自润滑减少与压室内壁磨损，降低卡滞概率，相比常规铸铁冲头寿命显著提升 [[S10]] |
| 压射嘴、压室 | 直接接触高温金属液 | 高导热加速金属液冷却，减少局部热量积聚，抑制粘模现象，提升温度均匀性 [[S10,S36]] |
| 顶杆、局部镶块 | 往复运动需低摩擦 | 无额外频繁润滑状态下保持低摩擦运动特性，降低顶出过程压铸件拉伤概率 [[S7,S17]] |

石墨钢基体中弥散分布的石墨相可在接触摩擦表面形成固体润滑膜，具备固有自润滑特性，相比常规热作模具钢可大幅降低压铸过程中铝合金等熔融金属的焊合倾向，减少额外石墨润滑剂的使用频次 [[S18,S39]]。石墨钢的热导率显著高于常规 H13 类热作模具钢，部分牌号热导率可达碳钢的 2 倍，可强化压铸部件的热导出效率，降低工件表面热应力集中风险，延长服役寿命 [[S13]]。

## 与 H13 压铸模具钢的对比
H13（4Cr5MoSiV1）为热作压铸模具钢，国内标准收录于 GB/T 1299，对应美国 ASTM A681 Type H13、日本 JIS SKD61、德国 DIN 1.2344，核心适用场景为模腔温度低于 600 ℃ 的铝、锌、铜合金压铸模，具备高韧性、优异的抗冷热疲劳性能，基体无内置游离石墨自润滑相，需额外做表面渗氮处理才可获得减摩特性，常规热处理后硬度为 HRC 47～49，与石墨钢的冷作自润滑属性完全不同 [[S43,S8]]。

## 局限性与改善方向

### 固有局限性

| 问题 | 机理 |
|------|------|
| 冲击韧性不足 | 石墨相割裂金属基体，大幅降低材料冲击韧性与塑性，常规牌号抗冲击性能远低于同碳量热作模具钢，承受大冲击载荷时易发生脆裂剥落 [[S15,S4]] |
| 大截面石墨偏析 | 石墨颗粒聚集形成尺寸超标的石墨团块，恶化基体连续性，导致工件不同位置性能不均，服役时局部提前开裂失效 [[S39]] |
| 工艺窗口窄 | 退火温度过高、保温时间过长或冷却速度过慢均易诱发石墨相异常析出，导致工件出现黑脆缺陷、淬火硬度不足 [[S4,S15]] |

### 优化改进方向

| 方向 | 方法 |
|------|------|
| 石墨形态球化 | 通过合金成分调控、球化处理将弥散分布的片状石墨改性为球状石墨，降低对基体的割裂作用，提升材料整体韧性 [[S38]] |
| 抑制偏析 | 采用随流瞬时孕育工艺、控制凝固冷速，抑制大截面石墨钢中的石墨偏析与漂浮现象，保证石墨颗粒均匀分布在 100～1000 μm 区间，实现工件大截面位置性能均匀 [[S14]] |
| 分级热处理 | 先通过高温石墨化退火消除大截面成分偏析，再进行调质处理获得索氏体基体+球状石墨的复相组织，在保留自润滑与高导热优势的同时兼顾材料强度与韧性 [[S16]] |
