## 1 定义与概述

**压铸单元**（die casting unit）是由**压铸机**与**辅助设备**共同组成的**自动化生产成套装备**，适用于高压铸造场景[[S18,S20]]。该术语由国家标准 *GB/T 37371-2019《压铸单元 术语》* 正式定义，标准中明确定义其为“压铸机与辅助设备形成的自动化生产成套装备”，并配以标准图示，标示了12个核心组成部分：压铸机、集成系统装置、模温控制装置、真空装置、保温与浇注装置、检测装置、清理整修装置、冷却装置、安全系统装置、取件与传送装置、喷涂装置、收集排放除尘装置[[S20]]。

压铸单元是压铸车间生产体系的核心独立生产单元。单套标准压铸单元可独立完成单个批次铸件的全部压铸生产流程，保证生产周期一致性与铸件精度、质量的稳定性，是现代压铸大规模生产的基础配置[[S21,S4,S43]]。当厂房内同时安装有若干个压铸单元时，可组成一条压铸自动化生产线[[S37]]。

## 2 分类

### 2.1 按锁模力划分

压铸单元可按锁模力大小划分为三个等级[[S38]]：

| 等级 | 锁模力范围 | 说明 |
|------|-----------|------|
| 小型压铸单元 | ≤4000 kN | 适用于小件压铸 |
| 中型压铸单元 | 4000~10000 kN | 中等吨位机种 |
| 大型压铸单元 | ≥10000 kN | 大吨位，适用于大型结构件 |

当前工业界大吨位规格压铸单元常见有6000t、7000t、7200t、8800t、9000t、12000t、13000t等，一体化压铸场景一般需要≥60 MN锁模力的大吨位压铸单元[[S31]]。国内现行压铸机规格标准 *JB/T 8083-2000* 规定冷室压铸机锁模力覆盖630~25000 kN，并允许扩展至31500 kN、40000 kN规格；当前商用量产大吨位压铸单元锁模力已突破160000 kN[[S33,S16]]。

### 2.2 按自动化程度划分

压铸单元按自动化程度主要分为**半自动**和**全自动**两类[[S25]]：
- **半自动压铸单元**：仅实现单工序或部分动作自动化，大吨位压铸机早期仅实现压铸循环单步动作机械化，下一循环需人工触发；
- **全自动压铸单元**：可实现合模、浇注、压射、开模、取件、喷涂全流程无人自动运行，部分小型热室、冷室压铸单元最早实现全自动化改造[[S5]]。

## 3 核心设备组成

### 3.1 标准设备配置

根据国家标准 *GB/T 37371-2019* 的定义与分类，压铸单元的标准功能装置包括以下12类[[S18,S20]]：

| 序号 | 功能装置 | 功能说明 |
|------|---------|---------|
| 1 | 压铸机 | 将熔融金属通过高压、高速压射入可重复使用的模具中完成铸造成型，是单元的核心动力与执行主体[[S20,S31]] |
| 2 | 集成系统装置 | 实现压铸单元工艺流程所需的全部控制装置及程序[[S15]] |
| 3 | 模温控制装置（模温机） | 通过介质循环按设定自行调节，有效控制模具运行温度，保障压铸凝固温度场稳定，减少热裂、粘模等缺陷[[S20,S15,S35]] |
| 4 | 真空装置（真空机） | 将型腔中气体快速抽至预设真空度，降低铸件内部气孔缺陷概率，提升致密度[[S20,S15,S35]] |
| 5 | 保温与浇注装置 | 保温炉存放金属熔体并保持预设温度，给汤机构定量取熔融金属并输送到压射室，保障每模次供料量精度[[S20,S35]] |
| 6 | 检测装置（检测台） | 对铸件外观、完整性、尺寸等特征进行检测，出现缺损时实时反馈，筛选不合格品[[S20,S15]] |
| 7 | 清理整修装置（切边机） | 去除压铸件浇注系统、溢流系统、毛刺并完成整形操作，得到符合要求的毛坯件[[S20,S15]] |
| 8 | 铸件冷却装置 | 将铸件温度降至工艺要求[[S12]] |
| 9 | 系统安全装置 | 保护压铸单元生产过程中设备、人员、工艺安全，含报警单元与联锁装置[[S15]] |
| 10 | 取件与传送装置 | 从模具取出铸件并运送到指定位置，实现工序间转运[[S20]] |
| 11 | 喷涂装置 | 将脱模剂配比、雾化，定点定量喷涂在模具型腔表面[[S20]] |
| 12 | 收集排放除尘装置 | 收集和处理压铸单元产生的油雾、油烟、废水及粉尘颗粒[[S15]] |

### 3.2 辅助与配套设备明细

除上述标准配置外，一套完整的压铸单元通常还需包含以下设备[[S8,S21]]：

| 设备 | 功能 |
|------|------|
| 给汤机构（自动给料机/浇注装置） | 从保温炉定量提取熔融金属并输送到压射室[[S35]] |
| 自动取件机械手 | 从模具取出铸件并运至指定位置后释放[[S20]] |
| 自动喷雾机械手 | 利用压缩空气将脱模剂雾化后定量均匀喷涂于模具表面[[S35]] |
| 保温浇注炉（定量炉） | 同时具备保温、定量、浇注三项功能，适配高自动化连续生产[[S20]] |
| 点冷机 | 对模具局部过热区域强制通水冷却，解决局部温度过高导致的粘模、铸件缩松问题[[S8]] |
| 模具自动交换装置 | 实现模具快速切换[[S21]] |
| 镶件预热装置 | 对铸入镶件进行预热[[S37]] |

对于大型压铸件（如汽车发动机缸体），还需额外配套铸入镶件预热装置、铸件冷却槽、密封性检测装置等专用设备[[S37]]。压铸单元的配置齐全程度与产品压铸工艺要求及压铸机型大小有关[[S8]]。

集成压铸单元将喷涂、浇注、取件等周边设备控制系统统一集成到压铸机主控制系统中，消除多设备独立控制的分散信号交互壁垒，减少硬件故障点，提升设备联动响应效率[[S46]]。

## 4 典型工艺参数与性能特征

### 4.1 压铸机端关键性能参数

国家标准 *GB/T 37371-2019* 对压铸机端的核心性能参数做出明确界定[[S12,S28,S42]]：

| 参数 | 定义 | 单位/说明 |
|------|------|----------|
| 最大金属浇注量 | 一次允许浇入压射室的最大金属液质量 | kg |
| 慢压射速度 | 慢压射过程中压射冲头的平均速度 | m/s |
| 快压射速度 | 快压射过程中压射冲头的平均速度 | m/s |
| 最大空压射速度 | 空压射情况下的最大快压射速度 | m/s |
| 建压时间 | 压射冲头运动停止点至增压压力稳态值90%交点的时间间隔，T = t₃ + t₄ + t₅（系统升压时间+系统延压时间+增压时间） | ms |
| 增压压力稳态值 | 压射活塞停止运动后200ms时的压射压力 | MPa |
| 压射压力、位移-时间曲线 | 反映压射压力、压射冲头位移与时间的关系曲线 | 二维曲线 |

### 4.2 辅机端关键性能参数

| 参数 | 定义 |
|------|------|
| 浇注精度 | 每次浇注量相对于标准浇注量的误差值[[S12]] |
| 取件装置最大负载 | 能可靠取出压铸件的最大质量[[S28]] |
| 取件装置定位精度 | 取件设定位置与实际位置的偏差[[S28]] |
| 控温精度 | 模温控制装置稳定工作后，设定温度与实际温度的偏差值[[S28]] |
| 喷涂定位精度 | 喷涂设定位置与实际位置的偏差[[S28]] |
| 真空泵抽气速率 | 真空泵在工作压力下单位时间内抽排气体的能力[[S28]] |

### 4.3 整机层面性能参数

| 参数 | 定义 |
|------|------|
| 压铸单元空循环时间 | 合模、浇注、空压射、开模、顶出、取件、喷涂全流程动作总耗时[[S42,S12]] |
| 平均无故障时间（MTBF） | 相邻两次故障之间的平均工作时间[[S42]] |
| 机器能力指数（CMK） | 仅考虑设备本身影响的分布平均值与规范中心值的偏移[[S42]] |
| 压铸单元综合能耗 | 规定耗能体系在一段时间内实际消耗的各种能源实物量折算为一次能源后的总和[[S42]] |

### 4.4 典型工艺参数范围

**压射速度与压力：**
- 慢压射速度通常为0.15~0.3 m/s[[S3,S29]]
- 常规快压射速度为4~5 m/s[[S29]]
- 现代高性能机型最大空压射速度可达9 m/s[[S29]]，大吨位大型薄壁件压铸单元快压射速度最高可达10 m/s[[S3]]
- 常规铝合金压铸填充过程总时长区间为0.01~0.2 s[[S3]]

**真空度：**
- 常规真空压铸单元的型腔内真空度典型区间为60~140 mmHg（约8~18.7 kPa）[[S14]]
- 高端高等级真空压铸系统可将填充全过程中型腔内真空度稳定控制在3~5 kPa，该水平下铝合金铸件热处理后体积膨胀率可低于1%[[S9]]

**生产节拍（不同吨位）：**

| 锁模力等级 | 典型空循环时间/生产节拍 |
|-----------|---------------------|
| ≤2000 kN 中小吨位 | 约60~90 s[[S24]] |
| 2000~4000 kN 中型 | 约90~120 s[[S24]] |
| 21500 kN（约2000吨）大型 | 实测约120 s[[S24]] |
| ≥6000吨 超大吨位一体化 | 约100~180 s[[S16]] |

## 5 布局与物料流

### 5.1 车间级布局

压铸车间布局主要有两种形式[[S36,S11]]：
- **中小吨位压铸机（≤800t）**：车间中间设置主物流通道，通道两侧排布压铸机，中间通道输送铝合金液；
- **大吨位压铸机**：采用压铸机居中、两侧分别设置合金液输送通道和铸件/回炉料输送通道的双通路互不干扰布局。压铸机距墙体最小距离不小于1.5m，具有大杠抽出功能的压铸机需预留对应抽出空间。

### 5.2 单元级平面布局

压铸单元可选用的典型平面布局包括[[S7,S34]]：
- **U形布局**：适配手工物料传输的多机成组生产单元，支持多技能操作工跨工位操作；
- **直线布局、循环布局、矩形布局**：后三者适配机械化传送带自动物料传输的流水成组生产单元。

### 5.3 与机加工单元的衔接

压铸单元与后续机加工单元之间的缓存面积可参考日常**2~6天生产量**的物料存储需求进行设计[[S44]]。自配套机加工的压铸车间在均衡日作业计划执行条件下可实现低库存甚至无独立成品库的“无库管理”。

### 5.4 控制系统架构

压铸柔性生产单元采用**PLC作下位机、工控机作上位机**的两级控制架构，通过开放现场总线网络连接压铸机本体、周边辅机、分布式I/O及后处理输送线，可对接上层生产监控站实现全单元集中管控，支持远程设备诊断[[S37]]。

## 6 应用场景

### 6.1 新能源汽车一体化压铸

大型一体化压铸单元的最主流应用场景为新能源汽车车身轻量化制造。面向大型薄壁一体化压铸件的典型配置包括[[S17,S30]]：
- 锁模力≥60000 kN的超大型冷室压铸机为核心；
- 免热处理高强韧铝合金熔炼系统；
- 三段抽真空系统，确保真空度小于50 mbar；
- 大型喷涂取件一体化机械手；
- 快速水冷站；
- 自动化渣包/料饼去除工装；
- 在线X-Ray探伤设备；
- AGV无人小车完成铝锭投料及大尺寸铸件转运，全程取消人工搬运。

当前主流商用超大型一体化压铸单元机型包括[[S22,S16]]：
- **力劲**：6800t / 12000t / 16000t系列
- **布勒**：CARAT 840（84000 kN）/ 930 / 950系列
- **海天金属**：HDC8800t / 12000t系列
- **伊之密**：60000~90000 kN系列

已落地量产的典型产品包括特斯拉Model Y一体式后底板、蔚来ET5后底板、高合HiPhi Z超大尺寸后舱、极氪009一体式后端铝车身、奔驰VISION EQXX仿生结构后舱等，可将传统数十至数百个冲压焊接零件集成为单个铸件，实现整车减重15%~30%，开发周期缩短1/3[[S27,S39]]。

### 6.2 其他应用

一体化压铸单元的延伸应用场景涵盖5G/6G通信基站超大尺寸壳体、散热腔体、滤波器等薄壁高导热压铸件的批量生产[[S27]]。此外，传统压铸单元广泛应用于摩托车、家电、通信仪表等零部件的各类规格生产[[S21,S43]]。

## 7 相关概念辨析

**压铸单元、压铸岛、压铸生产线**是三个不同层次的生产组织概念，其核心区别如下[[S18,S21,S1,S11]]：

| 概念 | 配置边界 | 覆盖范围 | 使用场景 | 工程实例 |
|------|---------|---------|---------|---------|
| **压铸单元** | 以单台压铸机为核心 | 仅覆盖压铸成型核心环节及周边配套辅机 | 多品种小批量的柔性压铸生产 | 单台压铸机+喷涂机+取件机+切边机+模温机等辅机 |
| **压铸岛** | 以多台同规格压铸单元为核心 | 延伸集成就近的切边、冷却、清理后续工序，是封闭性局部生产集群 | 单一高产量零部件的专一化量产 | 3台2250t压铸机+3台切边机+输送装置+抛丸机，生产率达45s/件（3机协同） |
| **压铸生产线** | 串联熔炼、多组不同规格压铸单元集群 | 覆盖全流程：集中清理、后处理、质量检测、物流配送，属全工厂级完整产线体系 | 全品类压铸件的工厂级规模化连续生产 | 熔炼工部+多台多规格压铸单元+清理工部+模具库+检验站+成品库 |
