## 术语界定与歧义说明

在铸造领域，"打箱"一词存在两种不同的含义，需要严格区分：

1.  **砂型铸造领域的"打箱"（主要含义）**：国内铸造行业通用术语"打箱"的主流使用场景隶属于砂型铸造工序范畴，是**落砂工序的行业俗称**。其核心操作是将浇注冷却后的砂箱放置在振动落砂筛上，通过振动使型砂从砂箱中散出并落入收集装置，完成铸件与外层型砂的初步分离 [[S39,S15]]。该操作并非压铸工序中将铸件直接从金属模具中取出的操作。
2.  **压铸领域的"打箱"（shakeout）**：在高压压铸的国际标准术语中，最后一步工序称为"shakeout"，其作用是将成型的铸件与浇口、横浇道、直浇口、飞边等废料分离 [[S4]]。

国家标准 GB/T 25370-2020《铸造机械 术语》和 GB/T 5611-2017《铸造术语》对相关工序有明确定义，术语边界清晰，互不重叠 [[S33,S19,S12]]。

## 定义

**打箱**，即落砂（shakeout/knockout），是砂型铸造中将浇注并冷却后的砂箱放置在振动落砂设备上，通过机械振动使型砂从砂箱中散出，实现铸件与外层型砂初步分离的工序 [[S33,S39]]。

打箱工序位于铸件浇注冷却之后，是铸造后处理的第一道工序，与除芯（decoring/core knockout，清除铸件内腔砂芯）、捅箱（punch-out，将浇注并冷却后的铸型和铸件从砂箱中分离出来）等工序有明确边界 [[S19,S33]]。

**英文术语对应关系**：
- 打箱/落砂 → shakeout / knockout [[S10,S26,S33]]
- 除芯/清芯 → decoring / core knockout [[S26,S33]]
- 压铸顶出铸件 → die casting knockout / ejection [[S4]]

## 工艺作用

打箱工序在铸造工艺链中的核心作用是完成铸件与型砂的初步分离。全自动化砂型铸造生产线上，打箱工位布置在铸件浇注冷却工序之后：完成冷却的砂型被输送至打箱工位，打箱设备将铸件与型砂分离，分离出的铸件直接由专用输送带输送至后续清理工段，脱出的型砂则被运送至砂再生工段循环回用 [[S32]]。

打箱属于铸造后处理工序，可与清理、打磨、喷丸等并列 [[S20]]。

## 标准工艺流程

打箱工序的完整标准工艺流程如下 [[S39,S15]]：

1.  **转运**：将浇注后的砂箱转运至打箱专用工位；
2.  **冷却等待**：等待铸件冷却至指定温度。从浇注到开箱的时间视铸件复杂程度与合金性质而定，铸件愈复杂，在铸型中需要停留的时间愈长 [[S39]]；
3.  **振动落砂**：将砂箱放置到振动落砂筛上，启动振动使型砂全部散出并收集至配砂工段；
4.  **输送除芯**：将脱除外层型砂的铸件输送至型芯清理工段，后续独立开展除芯操作；
5.  **后续加工**：之后进行浇冒口切割、表面清理等后续工序。

对于基于砂型模具的打箱操作，普遍采用摇晃或滚筒翻滚的作业方式，使凝固铸件从砂型中完全脱出。该操作完成后铸件表面仍附带附着的金属流道、浇口结构，需输送至后续专门的去浇口工位完成最终分离 [[S34]]。

## 操作条件与参数

### 铸件重量与冷却时间

常规100kg以内的小型及中型铸件可将带砂型整体推送至振动落砂栅格或输送式落砂机上直接振动破碎脱砂 [[S13,S22]]。重量约100kg的铸件约需要2小时的冷却时间后，方可进入打箱工序 [[S13]]。

### 砂型粘结体系的影响

不同砂型粘结体系的打箱特性差异显著 [[S13]]：

| 粘结体系 | 打箱特性 |
| :--- | :--- |
| 普通水玻璃砂、绿色砂 | 适配常规冲击式落砂设备即可获得90%以上的落砂效率 |
| 树脂粘结砂 | 砂块常温保留强度高，常规落砂栅格难以通过，需优先选用输送式落砂设备或配置更高振动加速度的专用除芯设备 |

对于树脂砂，模具中远离热影响区的大块角部砂块往往保持很高的硬度，因此普通落砂栅格通常效果不佳，输送式落砂机是更好的选择，但设备成本较高 [[S22]]。

### 合金种类的影响

不同压铸合金对应的打箱适配特性存在差异 [[S4]]：
- **锌合金铸件**：可采用相对较低的振动强度即可完成除芯；
- **铝合金、镁合金铸件**：需要中等振动强度；
- **高熔点铜合金铸件**：需要更高的冲击能量实现残砂脱除，同时需避免铸件磕碰变形。

## 相关因素

### 砂芯设计

根据ADCI铸造设计规范，对于直径大于1英寸（约25.4mm）的砂芯，其直径与深度的推荐比例为1:6。在该比例下配合适配的振动参数可实现完整脱芯，避免断芯残留于铸件内腔 [[S41]]。

### 铸件冷却与打箱时机

铸件在铸型内需要有足够的冷却时间。打箱过早会使热态铸件刚性不足、铸造残余应力未完全释放，容易引发变形缺陷。但对一些形状复杂的铸件，为减小铸型和型芯的阻力，又不能打箱过迟 [[S31]]。

## 常见缺陷与对策

### 铸件变形（扭曲、翘曲）

**产生机理** [[S16,S31]]：
- 铸件结构设计不良导致冷却收缩不均匀；
- 打箱开模过早，热态铸件刚性不足；
- 打箱冲击力超过铸件表面许用冲击载荷；
- 铸件在铸型内冷却时间不足，未完全释放铸造残余应力。

**防控措施**：
- 严格控制打箱温度阈值，保证铸件充分冷却以消除残余应力；
- 采用柔性可控的分级振击打箱工艺，避免集中冲击铸件薄弱部位。

### 局部开裂、边角缺损、表面磕伤

**产生机理** [[S8,S3]]：
- 振击锤击点设置不合理，直接接触铸件本体；
- 打箱操作时铸件无支撑固定，导致自由掉落碰撞。

**防控措施**：
- 优化振击点布置，仅将振击力作用于浇注余料或砂型部位；
- 设置铸件承接缓冲装置，避免铸件与刚性结构直接碰撞。

## 安全与环保

### 粉尘控制

打箱工序作业现场粉尘中游离SiO₂含量通常大于10%，属于矽尘（石英矽尘）。依据GBZ2.2-2007标准要求，清理作业现场矽尘的PC-TWA（时间加权平均容许浓度）不得超过2mg/m³ [[S30,S29]]。

防尘措施包括 [[S23,S30]]：
- 打箱设备必须配置密闭吸尘罩；
- 优先采用湿法作业、设备密闭等源头抑尘措施；
- 配置通风除尘系统，常用袋式除尘器、旋风除尘器等。

### 噪声控制

打箱工序噪声职业接触限值需符合GBZ2.2-2007标准 [[S29,S28]]：
- 作业场所连续稳态噪声8小时等效A计权声压级不得超过85dB(A)；
- 脉冲噪声声压级峰值不得超过对应规定限值。

降噪措施包括：
- 对振动落砂设备加装隔音罩；
- 气动工具排气口安装消声器；
- 高噪声作业区域设置隔音隔离操作间。

### 个体防护

打箱、清砂作业个体防护需符合GB39800.3《个体防护装备配备规范 第3部分：冶金、有色》要求。作业人员必须配套防尘口罩、防冲击护目镜、隔音耳罩/耳塞、防砸防烫伤劳保鞋，避免粉尘、噪声、高温铸件飞溅造成职业伤害 [[S25,S30]]。

## 替代工艺

主流脱芯清砂替代工艺及其特征对比 [[S37,S11,S1,S20]]：

| 工艺类别 | 原理 | 特点与适用性 |
| :--- | :--- | :--- |
| **热法脱芯** | 通过高温加热使砂芯内部粘结剂氧化分解溃散 | 适合复杂内腔中小尺寸砂芯脱除；无污水排放；能耗较高 |
| **超声波清芯** | 利用频率高于16kHz的超声空化振动效应在液体介质中冲击剥离残留砂芯 | 加工作用力小、热影响低；不会造成薄壁铸件变形损伤；适合高精度精密铸件内腔清砂 |
| **湿法清砂（水爆清砂）** | 利用红热铸件遇水汽化爆炸冲击落砂 | 因后续污泥污水处理成本高，且红热铸件遇水激冷易引发铸件开裂问题，已基本被干法振动清砂技术替代 |
| **湿法清砂（水力清砂）** | 高压水射流冲蚀除砂 | 同上，仅在液压件小壳体类复杂铸件场景少量保留 |
| **湿法清砂（电液压清砂）** | 水中高压放电产生冲击波清砂 | 同上，少量应用于特定场景 |

总体而言，干法清砂及除芯工艺（振动清砂除芯及抛喷丸清砂除芯）是当前技术发展的主流方向 [[S37]]。
