## 概述
砂芯铸造模上模（上芯盒）在GB/T 8845-2017中的标准术语为“upper core box”，定义为安装在成型设备上工作台面的砂芯铸造模部分，其配对部件砂芯铸造模下模（下芯盒）安装在成型设备下工作台面，二者合模后共同组成完整砂芯成型型腔[[S41,S2,S9]]。上芯盒是水平对开式芯盒的上部组成部分，核心结构为带凸缘和加强筋的薄壁箱形结构，设置有对应下半芯盒的定位销/套安装位、密封结构、排气布置以及部分区域的镶块设计，用于成型砂芯的对应型腔部分[[S7,S11,S10]]。

## 工艺作用
上芯盒通常直接与制芯设备的上工作台紧固连接，射砂机、壳芯机等制芯设备的射砂板、吹气板常集成或对接在上芯盒的上方接口位置，实现射砂、吹入硬化气体的功能导入[[S2,S39,S7]]。在完整制芯流程中，合模阶段上芯盒沿设备导向下行，通过导向销与下芯盒精准贴合完成锁模；射砂阶段上芯盒顶部的射砂板将砂料高速射入合模后的封闭型腔；硬化阶段热芯盒上模维持设定加热温度、冷芯盒上模通入对应硬化气体完成砂芯固化；开模阶段上芯盒沿设备导轨上行脱离砂芯；取芯阶段下模携带成型砂芯移出工位完成取芯[[S14,S39,S32]]。

## 分类
### 按制芯方法分类

| 制芯工艺 | 上芯盒结构与特点 |
|-----------|------------------|
| 热芯盒法 | 内置集成加热结构，可长期保持200℃以上工作温度，配置射砂孔、排气系统，材料多采用铸铁制造，保证加热均匀性与结构刚性[[S1,S16,S32]] |
| 冷芯盒法 | 设置密封环、排气塞结构，配置对接吹气板的密封装置防止三乙胺硬化气体泄漏，单独设置排气腔用于集中导出硬化后的废气，无需内置加热装置[[S1,S7]] |
| 壳芯法 | 顶射式壳芯盒上模通常对接水冷射砂板，射砂后通过加热使覆膜砂结壳，结构预留对应排砂、倒砂操作空间，适配壳芯机的翻转倒砂动作[[S32,S39,S4]] |
| 手工制芯 | 多为敞口或对开分体结构，无复杂加热、气路接口，配置蝶形螺母快速锁紧结构和手把结构，适配人工填砂、春砂操作，可采用木质、塑料或轻质铝合金材质，降低人工操作劳动强度[[S13,S22,S1]] |

### 按模具结构分类

- **二开模（水平/垂直对开分盒）**：对应的上芯盒结构简单，仅需配置定位销套结构与下芯盒的定位销配合完成导向合模，结构件少，加工维护成本低，广泛适用于形状规则的普通砂芯成型[[S28,S29,S13]]。
- **多开模（多向拆卸组合结构）**：对应的上芯盒由多个分块模块组合而成，适配具有多方向倒扣、复杂异形曲面的砂芯成型，上芯盒的各分块模块都配置独立的定位锁紧结构，保证组合精度[[S22,S23]]。

## 结构与组成

### 主体结构
普通金属芯盒一般采用带有凸缘和加强筋的薄壁箱形结构[[S7]]。芯盒主体一般都设计成带有边缘并设有加强筋的薄壁盒体结构，加强筋厚度取芯盒壁厚的80%～100%，一般根部取100%，下部取80%，使筋带斜度[[S28]]。

### 定位销/套
上芯盒的定位销/套推荐采用T8、T10工具钢或45钢制造，工作部分淬火硬度40~45HRC，常用销子直径范围为8mm、10mm、12mm，适配不同尺寸的芯盒，首套芯盒定位销中心距偏差不超过±0.05mm，普通使用场景下定位销位置偏差不超过±0.25mm[[S11,S20]]。定位销套设在芯盒两端或对角位置，销子装在下芯盒上，销套装在上芯盒上[[S28]]。

### 分盒面
上芯盒的分盒面与下半芯盒贴合，对开式上芯盒的分合面间隙要求≤0.1mm[[S38]]。热芯盒工况下，芯盒公称尺寸≤500mm时分盒面间隙不大于0.05mm，公称尺寸>500mm时允许最大间隙0.1mm[[S38]]。

### 排气系统
上芯盒的表面排气系统包括排气塞和排气槽两类布置，排气塞多设置在砂芯充型死角区域，配合排气槽将制芯过程中产生的气体顺利导出型腔[[S7,S1]]。冷芯盒上芯盒还需配套密封环结构，防止制芯过程中三乙胺等硬化气体泄漏[[S7]]。

### 射砂口与填充通道
射砂口优先设置在上芯盒对应砂芯的芯头区域，保证射砂时砂流充型顺畅，减少窝气死角，上芯盒的射砂孔与射砂设备侧的水冷射砂板配合，上芯盒射砂口尺寸比水冷射砂板侧射口尺寸大1~2mm[[S39,S18]]。

### 活块与镶件
上芯盒配置活块的典型应用场景为砂芯在上芯盒侧壁存在侧凹结构无法直接随上芯盒开模取出的情况，活块可随砂芯一同取出后再单独分离；镶件则应用于上芯盒局部易磨损、复杂异形型腔部分，通过单独加工后镶装在主体盒体上，降低整体加工难度，方便磨损后局部更换[[S28,S11,S29]]。

## 关键设计参数

### 尺寸精度与表面质量
| 项目 | 要求 |
|------|------|
| 制造尺寸精度等级 | IT11级[[S20,S1]] |
| 型腔允许偏差 | 取负值[[S20]] |
| 芯盒整体尺寸公差 | ≤0.2mm[[S20,S1]] |
| 型腔工作面表面粗糙度 | Ra≤1.6μm[[S20,S1]] |

### 材质选用
| 材质 | 适用场景 |
|------|----------|
| 铝合金（ZL101/ZL102/ZL103/ZL104等） | 适配中、小批量制芯场景，芯体质量轻易加工[[S21,S1]] |
| HT150/HT200铸铁 | 适配大批量制芯场景，热芯盒工况下普遍选用，保证200℃以上工作温度下的结构稳定性[[S21,S28,S39]] |
| 预硬钢、热作工具钢（如H13、P20、718等） | 多用于上芯盒的易磨损镶块、定位组件部位，提升使用寿命[[S21,S25]] |

### 表面处理要求
铝合金上芯盒的分盒面、刮砂面需加装厚度3mm的Q235钢耐磨护板，防止反复操作过程中芯盒工作面磨损，延长使用寿命[[S11]]。为了防止铝芯盒磨损，在凸缘面上加设耐磨护板，耐磨护板常采用Q235-A钢制造，厚度多为3mm，每块耐磨护板用沉头螺钉紧固在芯盒上[[S11]]。

### 加热管布置（热芯盒）
单块上芯盒的加热管布置数量不少于3根，排布位置需避开上芯盒上的射砂口、定位销套、导杆、排气塞等组件，保证上芯盒整体温度均匀[[S8,S37]]。上芯盒加热体到型腔内壁的最小壁厚需大于10mm，防止局部温度过高将砂芯烤焦[[S8,S37]]。

## 常见缺陷与故障

### 上芯盒关联的砂芯缺陷
| 缺陷类型 | 成因 |
|----------|------|
| 砂芯披缝过大 | 上芯盒与下芯盒的合模面配合间隙超差，上芯盒定位结构磨损导致两半芯盒错位[[S28,S38]] |
| 砂芯尺寸超差 | 上芯盒工作温度升高产生的热膨胀量无法被砂芯冷却收缩量完全抵消；上芯盒的型腔加工精度未达标[[S3,S1]] |
| 错型偏芯 | 上芯盒的定位销/销套长期受开合模冲击磨损，定位配合间隙超出允许范围，导致上下芯盒合模时型腔错位[[S28,S38]] |
| 砂芯疏松 | 上芯盒的排气孔、排气塞位置布局不当，制芯过程中型腔内的气体无法顺利排出，形成窝气区域[[S12,S3]] |
| 砂芯粘模 | 上芯盒型腔表面脱模剂涂刷不足、表面粗糙度不符合要求，或者型腔表面出现局部腐蚀麻点[[S6]] |

### 上芯盒自身失效形式
| 失效形式 | 产生机理 |
|----------|----------|
| 热疲劳裂纹 | 热芯盒工况下长期处于200℃以上的周期性加热、冷却交变循环中，型腔表面反复受到热应力冲击，当应力累积超过材料疲劳强度阈值时萌生裂纹[[S40]] |
| 磨损 | 合模面长期受开合模反复挤压摩擦，射砂孔位置长期受高速流动的芯砂冲蚀，定位销孔位置长期受定位销往复滑动磨损[[S3]] |
| 变形 | 设计阶段壁厚过薄且未合理布置加强筋，在长期高温载荷作用下出现整体挠曲变形，型腔尺寸出现不规则偏差[[S40]] |
| 腐蚀 | 长期接触覆膜砂中的树脂固化剂、脱模剂等化学介质，在高温环境下介质与上芯盒型腔表面金属发生缓慢腐蚀反应[[S27]] |

## 标准与规范

### 芯盒精度等级
我国铸造行业将芯盒划分为三级精度：一级芯盒适用于对铸件尺寸精度、表面质量要求高且长期使用的场景，二级适用于中等精度、多次使用场景，三级适用于低精度、单次或少量使用场景，不同等级的上芯盒对应不同的公差控制要求[[S30]]。

### 核心公差要求
- 对开式上芯盒的分盒面间隙应小于0.1mm[[S38]]
- 公称尺寸小于500mm的热芯盒合模间隙不大于0.05mm，公称尺寸大于500mm的热芯盒合模间隙不大于0.1mm[[S38]]
- 上芯盒定位销位置偏差符合指定公差要求时，需保证上下芯盒型腔完全对正[[S38]]

### 相关标准
GB/T 6414-1999标准等效采用国际标准ISO 8062:1994，明确了与芯盒成型砂芯对应的铸件尺寸公差体系，公差等级分为CT1至CT16，同时对由芯盒错位引发的错型偏差给出了允许范围要求[[S36,S34]]。
