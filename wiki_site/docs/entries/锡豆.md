## 概述

锡豆（tin sweat）是一种收录于国家标准的铸造缺陷。根据GB/T 5611-2017《铸造术语》定义，锡豆是锡青铜铸件的表面或内部孔洞中渗析出来的高锡低熔点相豆粒状或汗珠状金属物，其成分与铸件基体本体不一致，接近低熔点共晶成分[[S26,S11,S39]]。在我国铸造缺陷分类体系中，锡豆被归入9.8类夹杂类缺陷，属于内渗物（内渗豆）的子类，同时也符合铸造硬点（硬质点）的定义范畴，属于典型的偏析析出型金属硬质点[[S39,S10,S34]]。英文标准术语为tin sweat，部分专业词典标注其等价术语为tin exudation[[S26,S13,S28]]。

## 定义与归属

依据GB/T 5611-2017，锡豆的严格定义及分类归属如下[[S26,S33]]：

| 维度 | 内容 |
|------|------|
| 标准名称 | 锡豆（tin sweat） |
| 缺陷类别 | 9.8 夹杂类缺陷 → 9.8.12 |
| 母类缺陷 | 内渗物（内渗豆，9.8.10） |
| 相关类别 | 硬点（9.8.13），锡豆属于偏析析出型硬质点 |
| 标准定义 | 锡青铜铸件的表面或内部孔洞中渗析出来的高锡低熔点相豆粒状或汗珠状金属物 |

内渗物（内渗豆）的通用定义为存在于铸件孔洞缺陷内部的带光泽豆粒状金属渗出物，其化学成分与铸件本体不一致，接近共晶成分[[S21]]。锡豆作为内渗物的一种，其析出的高锡低熔点相属于偏离基体成分的共晶类富锡相[[S26]]。

## 特征与形貌

### 宏观特征

锡豆在铸件表面呈现灰白色豆粒状或汗珠状形态，具有类似“出汗”的成群分布特征[[S9]]。外露于铸件表面的锡豆常集中在冒口底部、铸件敞露缩孔或气孔位置[[S25]]。对于铅锡青铜铸件，有实测成分记录表明：当基体成分为85% Cu、13.79% Sn、0.17% Pb、1.04% Zn时，析出的锡豆成分实测为74.2% Cu、22.85% Sn、0.34% Pb、1.2% Zn，锡豆的生成伴随铸件微孔缺陷同步存在[[S1]]。

### 显微与界面特征

在微观层面，锡青铜凝固时以枝晶模式生长，富锡低熔点液相被排挤到枝晶间区域，最终在收缩应力和气体压力作用下渗出至表面或孔洞中[[S12]]。锡豆区域与基体之间存在明显成分差异，可通过金相检验和SEM/EDS成分分析加以确认[[S19,S24]]。

## 形成机理

锡豆的形成是多因素耦合作用的结果，其热力学与动力学基础如下：

### 热力学驱动力：溶解度陡降

Pb-Sn二元共晶体系的共晶点位于183℃、61.9wt%Sn处，Sn在Pb中的固溶度在183℃时为19%，温度冷却至室温时固溶度仅为1.9%[[S42]]。温度降低导致溶质溶解度大幅下降，为低熔点富锡相从合金基体中析出提供热力学驱动力[[S16,S8,S29]]。

### 动力学过程：枝晶偏析与抽吸渗出

锡豆的微观生成遵循以下经典偏析机理[[S6,S12]]：

| 阶段 | 过程描述 |
|------|----------|
| 枝晶生长 | 锡青铜凝固时以枝晶模式生长，先结晶固相将低熔点Sn、Pb溶质排出到剩余液相中 |
| 溶质富集 | 枝晶间区域溶质浓度持续升高，形成富锡低熔点液相 |
| 外力作用 | 溶解氢析出产生的气体压力叠加铸件固态收缩力 |
| 液相迁移 | 富锡液相沿枝晶间孔隙通道被强制迁移 |
| 最终析出 | 低熔点液相渗至铸件表面或内部孔洞中凝固，形成锡豆 |

通用的锡豆、锡汗、铅豆形成机制为：合金凝固过程中溶解氢析出的压力叠加铸件固态收缩力，迫使低熔点富锡/富铅液相沿晶间孔隙通道迁移至铸件表面或孔洞内最终析出[[S1]]。

## 影响因素

锡豆的生成受多维度因素控制，可归纳为以下内外因[[S4,S25,S32]]：

| 因素类别 | 具体条件 | 影响机制 |
|----------|----------|----------|
| 熔炼温度 | 熔炼温度过高、保温时间过长 | 低熔点Sn、Pb杂质有充足时间在液相中迁移富集 |
| 冷却速度 | 铸件冷却凝固速度过慢 | 低熔点液相上浮偏聚的时间窗口更长，加剧富锡相聚集析出 |
| 模具温度 | 模具温度过高 | 延长铸件整体冷却时间，放大低熔点相偏析概率 |
| 合金特性 | 锡青铜结晶温度间隔宽（平衡150℃，不平衡达250℃） | 凝固区域大，枝晶间缩孔多，为液相渗出提供通道 |
| 含气量 | 铜液含气较多，尤其是溶解氢 | 凝固时气体析出压力是液相迁移的核心驱动力之一 |
| 浇注温度 | 浇注温度过高 | 同时加剧含气量与冷却延缓，协同促进锡豆生成 |
| 杂质元素 | 原料中Sn、Pb杂质超标 | 为低熔点相提供物质基础 |

在压铸合金领域，过量Sn、Pb杂质的主要来源为再生铝原料、多次循环使用的回炉料，其中锌合金电镀废品、反复回收的水口料会持续累积Sn、Pb杂质[[S40,S18]]。

### 成分控制阈值

DIN 1725.5压铸铝合金成分标准中明确规定Sn、Pb等低熔点杂质单种质量分数最大值不超过0.05%，杂质总质量分数不超过0.15%，该阈值是压铸行业预防锡豆析出的常规参考控制指标[[S37]]。

## 危害与影响

锡豆对铸件质量产生多方面危害：

### 力学性能劣化

锡豆作为局部成分异常的硬质点，造成铸件整体化学成分分布不均，直接降低铸件强度、塑性、疲劳寿命、冲击韧度等力学性能[[S38,S17]]。

### 加工性能恶化

富锡质点在机械加工过程中导致切削刀具剧烈磨损，加工后工件表面易出现局部划痕和硬点凸起，显著提升后续抛光修整的工作量[[S7,S15]]。

### 致密性与密封性下降

锡豆伴随形成的孔洞类缺陷降低铸件整体致密性，导致承压类部件在耐压测试过程中出现介质渗漏问题，无法满足密封工况的长期使用要求[[S25]]。

### 表面处理质量缺陷

锡豆区域的富锡相与青铜基体之间电化学电位、氧化特性存在明显差异[[S38]]：
- 涂装工序出现局部异色、涂层附着力不良缺陷
- 阳极氧化工序出现局部无法正常成膜的漏底、色差问题
- 直接报废表面处理后的成品件

## 检测与识别

锡豆的检测方法涵盖从宏观到微观的多层次技术手段：

| 检测方法 | 适用范围 | 判定依据 |
|----------|----------|----------|
| 低倍目视检测 | 外露于铸件表面的锡豆初筛 | 灰白色豆粒状“出汗”群集特征，集中于冒口底部、敞露缩孔位置[[S22,S25]] |
| 金相检验 | 观测锡豆微观分布 | 光学显微镜下富锡低熔点相的分布、尺寸、形貌确认[[S19]] |
| SEM/EDS成分分析 | 准确定性区分 | 检测到局部锡元素含量远高于基体的富锡区域即可明确判定[[S24,S31]] |
| X射线探伤 | 内部隐藏未外露锡豆定位 | 可集成在压铸岛内实现产线级快速质量筛查[[S3,S25]] |
| 超声波检测 | 大尺寸厚壁锡青铜压铸件筛查 | 可同步检出内部大颗粒锡豆伴随的孔洞、疏松类缺陷[[S35,S25]] |

## 预防与控制措施

### 原材料管控

严格控制入炉原材料质量[[S15,S25]]：
- 炉料严格保证干燥、无锈蚀、无油污
- 废铸件、机加工切屑类回收料需预先压块或重熔制备为二次锭
- 回料占比不超过3:1与新料混配
- 从进料环节避免外来杂质带入

### 熔体精炼除杂

严格执行熔炼工艺规范[[S15,S25]]：
- 避免熔炼温度过高、熔炼时间过长
- 熔清后造渣覆盖熔池表面，减少吸气与氧化
- 采用惰性气体（氩气、氮气）转子搅拌除气
- 采用可控气氛或真空熔炼进一步降低气体和夹杂物含量
- 保温阶段预留足够静置时间，让杂质充分上浮沉降

### 合金成分优化

从合金设计层面抑制锡豆倾向[[S38,S25]]：
- 适当提高锡青铜中的锌元素含量，缩小合金的凝固温度区间，从根源抑制反偏析行为
- 严格管控合金中磷、硫等易生成低熔点相杂质的占比，减少富锡共晶相的析出倾向

### 压铸工艺参数调整

通过优化工艺条件减少锡豆生成[[S38,S25]]：

| 工艺措施 | 作用机制 |
|----------|----------|
| 适当降低浇注温度 | 加快铸件冷却凝固速度，减少低熔点富锡相上浮聚集 |
| 采用自然振动结晶器 | 显著降低锡青铜压铸件的反偏析倾向 |
| 严格控制热处理温度 | 禁止压铸件在过高温度下长时间保温，避免晶界低熔点相重熔析出 |
| 优先采用真空压铸、吹氧压铸等无气孔压铸工艺 | 消除内部孔洞，从根源消除锡豆的析出空间 |

### 缺陷补救

根据铸件技术要求判定处置路径[[S25]]：
- 轻微小尺寸锡豆可通过打磨去除后补焊
- 重要承压构件可采用热等静压工艺消除内部锡豆及伴随的孔洞缺陷
- 带残余内应力的铸件可通过时效或扩散退火降低应力水平，避免后续使用过程中发生变形开裂

## 关联与区分

### 锡豆与锡汗的关系

中英文术语体系存在指代差异。英文铸造标准中tin sweat同时覆盖锡汗、锡珠（锡豆）两类现象[[S13,S28]]：
- **锡汗**：是锡青铜发生反偏析产生的宏观缺陷，铸件表层含锡量显著高于内部基体，严重时表面析出灰白色富锡δ相，反偏析层最大深度可达25~30mm，属于宏观成分偏析类缺陷[[S32,S9]]
- **锡豆**：是局部析出的豆粒状或汗珠状金属物，属于内渗物类夹杂缺陷[[S26]]

国内铸造标准体系将二者明确区分为不同维度的缺陷类型。

### 锡豆与铅豆（铅汗）的区分

铅豆（铅汗）仅在高铅锡青铜（铅含量10%~25%、锡占比低于5%）铸件中出现，表现为局部析出的厚层低熔点铅相或最大尺寸接近豌豆的珠状金属颗粒[[S1]]。

### 锡豆与普通硬质点（硬点）的区分

| 缺陷类型 | 定义 | 形成机制 |
|----------|------|----------|
| 锡豆 | 高锡低熔点相渗析形成的豆粒状金属物 | 低熔点相偏析析出，伴随反偏析和气体压力 |
| 普通硬点（硬质点） | 铸件断面中分散或大尺寸硬质夹杂物 | 非金属夹杂（氧化物、耐火材料）、金属间化合物等外界混入或凝固析出[[S39,S11,S33]] |

普通硬点多在机械加工或表面处理工序中被发现，不属于低熔点相偏析析出形成的缺陷[[S39]]。

### 锡豆与不同压铸合金体系的关联

三大主流压铸合金的锡豆缺陷分布规律存在显著差异[[S41,S23,S5]]：

| 合金体系 | 锡豆关联性 | 说明 |
|----------|------------|------|
| 铜基合金（锡青铜、含铅锡青铜） | **典型高发基体** | 锡为主合金元素或高含量添加元素，是锡豆唯一典型高发体系 |
| 铝基压铸合金 | 无典型锡豆缺陷 | 不以锡为主合金元素，无官方标准收录的典型锡豆缺陷；富锡相仅作为夹杂类硬质点的潜在组成类型[[S43,S18,S27]] |
| 镁基压铸合金 | 无典型锡豆缺陷 | 未检索到公开的规模化生产锡豆析出案例[[S5]] |
| 锌基压铸合金 | 无常规锡豆析出 | 锡仅为杂质组元，超标时典型缺陷为晶间腐蚀引发的铸件老化，无常规锡豆析出明确记录[[S23]] |

### 其他元素对偏析的抑制作用

含少量锡的镍黄铜合金可抑制铅汗（铅豆）缺陷的生成，降低低熔点铅相偏析析出风险[[S14]]。
