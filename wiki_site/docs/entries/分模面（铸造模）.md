## 概述
分模面（又称分型面）是压铸模具中动模与定模部分相互贴合的可分离接触表面[[S40,S6,S37]]。其作用是将模具分成两个可相对开合的部分：合模时两部分贴合封闭构成完整的成型型腔；开模后动模与定模分离，以便顺利取出压铸件及浇注系统凝料[[S40,S3]]。分模面是决定压铸模整体结构形式的核心设计要素，直接影响金属液充型特性、铸件脱模效果与最终成型质量[[S40,S6]]。

分模面（parting surface/plane）与成品铸件表面的分模线（parting line）紧密关联：分模面是动、定模贴合的连续面域，而分模线是分模面与压铸件外表面相交得到的轮廓边界线，在铸件上表现为合模痕迹或飞边痕迹[[S2,S47,S1]]。分模面的位置与形态直接决定了分模线的分布走向。

## 分模面的分类
压铸模的分模面可分为**单分型面**与**多分型面**两大类[[S37,S15,S45]]。单分型面通过一次开模即可完成铸件与浇注余料的脱模；多分型面则由一个主分型面与一个或多个辅助分型面组合而成，用于适配复杂结构的铸件[[S37,S14,S21]]。常见的分型面形式及其特点见下表。

| 类别 | 子类型 | 结构特征 | 适用场景 | 参考 |
|------|--------|----------|----------|------|
| **单分型面** | 平直分型面 | 分型面平行于压铸机动、定模固定板平面 | 形状规整、分型轮廓为平面的盘类、盖类铸件；加工难度低 | [[S37,S23,S46,S15]] |
| | 倾斜分型面 | 分型面与动、定模固定板呈指定夹角 | 存在倾斜外壁或锥面的铸件，可避免复杂侧抽芯结构 | [[S37,S23,S46,S15]] |
| | 阶梯（折线）分型面 | 由多个错位的平面组合构成 | 具有多段不同高度台阶轮廓的复杂铸件，可缩短抽芯行程 | [[S37,S23,S46,S15]] |
| | 曲面分型面 | 分型面匹配铸件外轮廓曲面形态 | 非规则圆弧面外壳类铸件 | [[S37,S23,S46,S15]] |
| | 垂直分型面 | 分型面与开模方向平行 | 特殊结构需要，常配合侧向抽芯机构 | [[S45]] |
| **多分型面** | 双分型面 | 由一个主分型面与一个辅助分型面构成 | 铸件结构特殊，单分型面无法满足脱模或工艺要求时采用 | [[S37,S15,S23]] |
| | 三分型面 | 一个主分型面配合两个辅助分型面 | 更为复杂的铸件结构 | [[S37,S15]] |

## 工艺作用
### 模具开合与导向
分模面的贴合面导向结构与导柱、导套共同保证动定模合模对位精度，防止压铸过程中的动定模错位，避免铸件出现错边缺陷[[S43,S4]]。合模时分模面的有效接触面积与贴合精度直接决定了压铸机锁模力的承载能力，防止胀型金属液喷射飞溅[[S4]]。

### 铸件脱模
合理的分模面设计应确保开模后铸件包紧力较大的一侧位于动模侧，使铸件随动模移动脱离定模，继而由内置推出机构完成脱模，简化模具脱模系统结构[[S1]]。分模面的位置选取直接决定了脱模方向与脱模斜度的倾向[[S32]]。

### 排气与溢流
设置在分型面上的薄型排气槽是型腔排气路径的核心载体，可将型腔内部积存的气体顺利导出，防止铸件卷气缺陷[[S32,S3]]。分型面的走向直接影响排气路径的长度与排气效率。同时，常规工艺将储存冷金属和涂料残渣的溢流槽布置在分型面靠近金属流末端的位置，利用分模面的加工基准特性简化设计与加工[[S6,S37]]。

### 浇注系统的基准
分模面位置一旦确定，即直接决定了浇注系统的布置形式、内浇口的导流方向以及金属液的充型路径，是浇注系统设计的首要前置约束[[S6,S12]]。

## 分模面选取原则
分模面的位置选择需综合权衡铸件形状、精度、模具制造成本及压铸工艺要求。以下原则是设计中的主要考量依据[[S14,S25,S45,S18,S30]]：

| 原则 | 具体要求与说明 | 参考 |
|------|----------------|------|
| **最大截面原则** | 分型面必须设置在压铸件外形轮廓尺寸最大的截面处，保证开模时铸件可顺利脱出；若仍无法完全脱模，需增设辅助分型面配合侧向抽芯机构 | [[S14,S25,S45]] |
| **动模包紧原则** | 保证开模后铸件对动模型芯的包紧力大于定模，使铸件留于动模侧；必要时可借助动模侧抽芯机构强制实现 | [[S25,S45,S46,S12]] |
| **侧抽芯避让原则** | 优先将抽芯距离短、投影面积小的特征设为侧向抽芯，避免复杂的长距侧抽结构，简化模具整体结构 | [[S18,S30,S12]] |
| **浇注与排溢配合原则** | 分型面应设置在金属液流动方向的末端，保证最后充填分型面区域，为溢流槽、排气槽提供合理布置空间，减少卷气与冷隔缺陷 | [[S25,S45,S30,S46]] |
| **尺寸精度控制原则** | 同轴度要求高的多个圆柱面应置于模具同一半侧内加工；关键尺寸避免跨分型面设置，以减少合模错位引起的偏差 | [[S25,S45,S30,S12]] |
| **飞边控制原则** | 尽可能减小铸件在分型面上的投影总面积，避免投影面积接近压铸机最大允许面积，以降低溢料飞边概率 | [[S18,S30]] |
| **模具简化原则** | 优先采用平直分型面，避免不必要的曲面、阶梯分型或多分型面，以简化加工并降低制造成本 | [[S32,S30]] |

合理的分模面设计可最大程度减少侧抽芯结构数量，降低模具制造成本15%~30%，减少飞边后续修整工时20%以上，并延长模具服役寿命[[S18,S32]]。

## 常见缺陷与分模面的关联
### 飞边
飞边是压铸件在分型面边缘或活动侧抽部位出现的突出金属薄片[[S39,S44,S31]]。其主要诱因包括：分型面残留杂物未清理干净、合模研合精度不足、模具镶块或滑块磨损导致与分型面不平齐[[S39,S31]]。预防措施为：生产前彻底清理分型面残留物，定期检测合模间隙，对磨损部位进行焊补打磨修复[[S39,S44,S31]]。

### 错边（错模）
错边表现为铸件在分型面位置发生相对错开位移，通常由分模面导向装置加工安装精度不足、合模动作偏移引起[[S9,S16]]，最终导致跨分型面尺寸超差。

### 粘模拉伤
分模面附近成型部位的脱模斜度匹配不当或分型面边缘圆角过渡不合理，会使开模时铸件与模具型腔发生啮合粘结，在脱模方向产生擦伤痕迹[[S16,S39]]。

### 毛刺
分型面排气槽深度超出工艺允许范围（常规压铸排气槽深度为0.005~0.010英寸）或分模面研合间隙超差，会导致金属液渗入缝隙快速冷却形成不规则毛刺，增加后续修整工作量[[S5,S27]]。

## 相关标准与规范要求
### 中国国家标准（GB/T 系列）
- **平面度与间隙**：GB/T 8844-2003《压铸模技术条件》及 GB/T 4679-2003 规定，动、定模镶块平面应分别与动、定模板齐平，允许高出量≤0.05mm；合模后分型面应紧密贴合，除排气槽区域外最大允许配合间隙≤0.05mm[[S24,S35]]。
- **平行度**：模具分型面对定、动模座板安装平面的平行度公差按被测面最大直线长度分区间设定，需在设计总图中明确标注[[S24,S42]]。
- **分型面影响附加公差**：GB/T 25747（镁合金压铸件）明确，跨动、定模分型面的线型尺寸需在基础公差上叠加指定的附加公差，量化了分型面对尺寸精度的影响[[S34]]。其他合金压铸件（如铝合金）标准中也有类似条款[[S28]]。
- **分型面位置推荐**：标准推荐分型面优先选在压铸件外形轮廓最大截面处，保证铸件包紧动模脱出定模，并便于浇注与排溢系统布置[[S14,S19]]。

### 国际标准
- **ISO 8062**：受压铸模分型面影响的线性尺寸，需在基础尺寸公差上按铸件在分型面上的投影面积额外附加补偿量[[S28,S29]]。
- **NADCA 技术规范**：
  - 平分型面为基准构型；跨分型线的尺寸因合模错移和胀型位移，偏差显著高于单一模半内的特征[[S13,S22]]。
  - 普通商用级跨分型面公差基准范围为0.001~0.010 in，精密级公差较商用级收紧约50%[[S17]]。
  - 阶梯分型面构型会额外增加约17%的模具制造成本[[S13]]。
- **JIS 相关要求**：对于ADC12等压铸合金，分型面应沿铸件最长外轮廓线所在平面选取，保证脱模并匹配锁模力核算需求[[S7]]。
