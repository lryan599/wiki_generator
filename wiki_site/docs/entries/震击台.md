震击台（振动除芯机）是压铸后清理工序中用于去除铸件内部砂芯、盐芯及表面粘附型砂的专用振动设备，亦可用于辅助完成铸件与浇注系统流道余料的震击分离。[[S23]]与普通砂型铸造落砂设备不同，震击台面向压铸件薄壁易磕碰特性，无需破碎大型砂型，通过高频震击使内嵌芯体松散破碎并从预留工艺孔排出。[[S23]]国家标准 GB/T 25370-2020《铸造机械 术语》将其归类于除芯机（decoring machine），定义为采用气动或液压振击锤清除铸件砂芯的机器，并区分了辊道通过式、过渡车式、转台式等输送形式。[[S23]]

## 别名与分类

震击台常见中文别名包括振芯台、振动除芯机、落砂台；对应英文术语为 decoring machine、vibratory decoring machine、knockout table、shakeout table、impact table for core removal 等。[[S23,S1]]

按驱动形式可分为三类：[[S23,S1]]
- **气动式**：以压缩空气驱动振击锤产生周期性冲击力；
- **电动式**：由振动电机（电动激振器）作为激振源，利用偏心块旋转产生激振力；
- **电磁式**：依靠电磁系统周期性吸合实现台面高频振动。

## 结构与组成

震击台的主要组成部分包括：[[S23,S20,S12]]
- 承载压铸件的**台面**与工装夹具；
- **激振器**或振动电机（部分机型配置双轴激振器，内置旋转不平衡质量偏心块）；
- **隔振弹性元件**（减振弹簧），常见形式为橡胶弹簧或空气弹簧；
- **PLC控制系统**，实现自动、半自动或手动操作；
- **封闭除尘室**，内壁可敷设玻纤吸声材料，出入口设进、出料门，与车间除尘系统对接；
- **工件输送机构**，可采用机动辊道、移动小车、旋转工作台或上下料机械手等方式，适应不同生产线布局。

## 工作原理

不同驱动形式的震击台产生震击的机制如下：[[S14]]
- 电动式：振动电机内部的偏心块随转轴高速旋转，产生交变离心力，带动台面及铸件振动；
- 气动式：压缩空气往复驱动振击锤，高频冲击安装有铸件的工装；
- 电磁式：电磁系统周期性通断吸合台面，产生高频往复振动。

震击能量通过专用工装夹具刚性传递到压铸件整体，使铸件内部砂芯或盐芯在交变冲击作用下松散、破碎，碎屑从铸件预留的工艺孔排出，达到清砂除芯目的。[[S14]]

## 关键工艺参数

震击台的工作参数直接决定除芯效果与铸件损伤风险。典型参数范围如下：

| 参数类别 | 参数名称 | 典型值/范围 | 说明 | 来源 |
|---------|---------|------------|------|------|
| 振动基础参数 | 频率 | 15～50 Hz，压铸常用 20～30 Hz | 振动频率影响作用强度 | [[S9]] |
| | 振幅 | 2～20 mm，压铸常用 3～6 mm | 振幅直接影响冲击能量 | [[S9]] |
| 气动行程参数 | 进气行程、膨胀行程、排气行程 | — | 可调整排气行程大小来适配不同震击力需求 | [[S26,S4]] |
| 循环时间 | 单批次除芯时间 | 约 25 s（主流多振击器机型） | 可匹配大批量生产线节拍 | [[S2]] |
| 负载能力 | 额定承载能力 | ≥1.2×(铸件+芯体总重) | 选型时的安全余量要求 | [[S4]] |
| 安全与环保限值 | 空运转噪声 | ≤95 dB(A) | 参照铸造清理设备通用安全规范 | [[S8,S17]] |
| | 粉尘排放浓度 | ≤150 mg/m³ | 运行时机内须保持负压 | [[S8,S17]] |
| 激振器规范 | 额定频率 | 50 Hz | JB/T 5772-91 规定 | [[S16]] |
| | 额定电压 | 660 V, 380 V, 220 V | — | [[S16]] |
| | 外壳防护等级 | ≥IP55（接线盒 IP56） | — | [[S16]] |

此外，台面的安装精度直接影响震击平稳性：台面与砂箱滚道面（或类似基准）的平行度公差为 0.08 + 0.15L/1000 mm，震击工作台升降运动中心线与该基准的垂直度公差为 0.05 + 0.02L/100 mm（L 为测量长度，单位 mm）。[[S13]]

## 在压铸流程中的位置

震击台通常安放在压铸机与切边机之间，或布置在去浇口工序之后、热处理炉之前。设备通过自动输送线（如机动辊道）与上下料机构实现铸件自动流转，配套的除尘管网接口直接连接车间除尘系统，除芯过程在全封闭状态下进行，粉尘集中收集不外溢。[[S2]]

## 典型机型及应用场合

压铸行业常用的震击台（振动除芯机）有多种定型和专用型号，适应不同铸件尺寸、内腔复杂度和产量需求。

| 型号 | 振动形式 | 适用对象 | 特点 | 来源 |
|------|----------|----------|------|------|
| L145型 | 单气动振动器 | 溃散性较好的中小型压铸件 | 结构简单，入门级，但噪声大、劳动条件欠佳 | [[S25]] |
| QZX型旋转式 | 气动振击 + 卧式旋转 | 发动机缸体类大中型铸件 | 除芯时铸件旋转，便于倒出内腔浮砂；PLC 控制，可自动/半自动/手动操作；配全封闭除尘罩 | [[S25,S2]] |
| ZYSB06型 | 多振击器（气动） | 发动机缸盖等复杂内腔铝合金压铸件 | 空气弹簧支撑工作台，全封闭隔音除尘罩，内衬玻纤吸声材料；单次除芯循环约 25 s | [[S6,S2]] |
| ZL2型 | 振动（未详） | 汽车、摩托车、航空发动机中小型复杂内腔铸件 | 结构紧凑，占地面积小 | [[S6]] |

上述机型可适配普通树脂砂芯，部分高激振力多振击器型号还可辅助清理残留的盐芯或石膏芯。[[S2]]

## 常见问题与缺陷

震击台在使用中若参数控制不当，可能引发多种质量问题：
1. **铸件变形与裂纹**：震击力过大或工装夹持不合理，会使压铸件（尤其是铝合金薄壁件）产生变形、表面裂纹甚至局部缺损。[[S3]] 现场可能出现如图所示的敲打缺损缺陷，其原因多归于震击力异常或夹持不当。[[S10]]
2. **除芯残留**：对于内腔极端复杂、存在深窄盲孔的铸件，若震击时间不足或角度单一，可导致芯砂残留，后续形成砂眼类缺陷。[[S6]]
3. **台面磨损**：长期高负载下，金属工作台面受铸件与夹具反复冲击产生快速磨损；未按要求在震击接合面设置减震缓冲垫会加剧磨损。[[S27]]
4. **噪声与粉尘污染**：无全封闭隔音结构的常规震击台运行时噪声远超 95 dB(A)，无配套除尘罩时车间扬尘严重。[[S27]]

## 选型依据与使用限制

### 选型考量
- **铸件重量与尺寸**：设备额定承载能力应大于含芯体铸件总重的 1.2 倍以上。[[S4]]
- **材质抗冲击性**：铝合金薄壁压铸件宜选择低振幅、高频参数，避免变形；铸铁及厚壁高强度铝合金件可采用高振幅、大激振力配置以确保芯体充分溃散。[[S9]]
- **生产线节拍**：单班次产量低于 200 件的中小批量，可选用人工上下料的单振击器机型；年产十万件以上的大批量生产，应选用多振击器全自动机型，并配置辊道、机器人等组成除芯单元。[[S23,S25]]
- **环保合规**：必须配置全封闭除尘罩及隔音内衬，满足铸造车间粉尘排放与职业噪声限值。[[S2]]

### 使用限制
- **水溶盐芯**：采用水溶性盐芯生产的压铸件，禁止使用震击台除芯，应通过温水浸泡或化学溶出方式清芯，以避免冲击破坏铸件密封性。[[S5]]
- **高精度薄壁嵌件/精密微孔件**：电子类压铸件或带有精密嵌件的铸件，震击冲击易导致嵌件移位或微孔变形失效，不推荐使用震击台清砂。[[S7]]
