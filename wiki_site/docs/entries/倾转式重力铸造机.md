## 概述

倾转式重力铸造机（tilting type gravity die casting machine）是金属型重力铸造设备中一种以合模系统整体倾转为核心动作特征的机型，归属于重力铸造（Gravity casting）大类，完全依靠金属液自身重力作为充型动力[[S32,S16]]。该设备通过将模具连同浇口杯从近水平状态按预设速度逐步翻转至竖直位置，使金属液以低流速、层流状态平稳填充型腔，显著降低飞溅、氧化膜卷入与卷气缺陷，是常规固定式直浇金属型重力铸造的重要升级构型[[S3,S17,S23,S31]]。

## 定义与分类

### 国家标准定义
依据GB/T 25370-2020《铸造机械 术语》的规定，倾转式重力铸造机定义为“合模系统可在一定角度范围内倾转的金属型重力铸造机”[[S32]]。该标准明确其按模具分型方式通常分为垂直分型和水平分型两种结构型式[[S32,S37]]。

### 设备分类
倾转式重力铸造机按分型结构可分为垂直分型、水平分型两大基础类别；按驱动方式可分为手动驱动、液压驱动、电动驱动三类配置[[S1,S26]]。其中液压驱动机型具备大负载、运行平稳的特点[[S42]]。典型机架结构分为固定式、移动式（配置滚轮沿地轨移动，炉体与机架独立设置）和摇臂式（绕固定回转立柱旋转）三类[[S13]]。

## 工作过程与充型原理

### 核心充型原理
倾转式重力铸造的核心充型方式为：预先将模具连同浇口杯倾斜至近水平状态并导入金属液，随后按设定的倾转速度将模具逐步翻转至竖直位置。金属液在整个充型过程中保持低流速平稳填充型腔，相比常规固定式顶部浇注，金属液下落高度大幅降低，可显著减少金属液飞溅、氧化膜卷入与卷气缺陷[[S3,S17,S23,S31]]。

### 分段变速控制逻辑
主流倾转浇注采用三段式运动轨迹控制逻辑[[S14]]：

| 倾转区段 | 倾转角度范围 | 倾转速度要求 | 工艺功能说明 |
|----------|--------------|--------------|--------------|
| 进料段 | 70°～90° | 极低倾转速度 | 避免金属液从料斗进入型腔时产生紊流，防止卷气 |
| 快速充填段 | 55°～70° | 较高倾转速度 | 快速完成约85%型腔填充，减少氧化、冷隔缺陷 |
| 终填段 | 35°～55° | 低速平缓停止 | 避免急停产生机械振动导致砂芯掉砂、夹渣缺陷 |

行业研究明确，倾转铸造中金属液产生紊流的临界流速为0.5 m/s。控制型腔内部流速低于该阈值即可实现无卷气层流充型[[S14]]。

## 关键结构与组件

### 核心组成
依据GB/T 25370-2020的规定，倾转式重力铸造机核心标配部件包括开合型机构（合模系统）、抽芯机构、顶出机构及倾转机构，可按需选配模具冷却系统[[S32,S37]]。

### 倾转机构
倾转机构是倾转式重力铸造机驱动合模系统完成倾转动作的专用组件。该标准明确支持将整个倾转角度范围划分为多个独立倾转区段，每个区段可单独设置不同的倾转速度，实现分段变速控制[[S37]]。

### 驱动与安装方式
液压式倾转机构以油缸作为动力输出部件，传动特性为运行平稳、支持液压自锁功能，倾转力矩大，倾转位置控制精度高，适配大吨位铸件生产需求[[S42]]。金属型（模具）标准安装固定方案采用斜销连接搭配压板锁紧结构，可快速完成模具拆装定位，合型位置精度高[[S42]]。

### 配套浇注系统
倾转配套浇注系统设计核心要求为采用底注式布局，将直浇道设置在动模侧，横浇道与内浇口设置在定模侧，保证倾转过程中金属液以层流状态平稳填充型腔，同时预留充足排气通道避免卷气缺陷[[S23]]。

## 典型工艺参数

### 倾转速度参数
倾转式重力铸造机常规工作倾转速度区间为3°/s～12°/s，行业通用合理控制区间为5°/s～9°/s[[S24,S35]]。不同倾转速度对铸件质量的影响规律明确[[S10,S24,S35,S37]]：

| 倾转速度条件 | 对充型过程的影响 | 可能诱发的缺陷 |
|--------------|------------------|-----------------|
| 低于7°/s | 金属液流动受表面张力阻滞，流动显著受限 | 冷隔、浇不足 |
| 5°/s～9°/s（合理区间） | 充型平稳，金属液以层流状态流动 | 缺陷风险最低 |
| 高于15°/s | 熔体前端流速易超过0.5 m/s临界值 | 湍流卷气、氧化夹渣 |

GB/T 25370-2020允许将整个倾转行程划分为多个独立倾转区段（倾转区段），不同区段可单独设置倾转速度，实现变速率充型控制[[S37]]。

### 倾转角度参数
常规工作倾转角度范围为0°～90°，可根据铸件实际需求自定义起始倾转角度[[S40,S35]]。推荐选用正向大于20°的起始角度实现金属液上坡填充，避免金属液在重力作用下自由落体造成扰动。若初始倾转起始位置低于水平方向10°，将无法获得无湍流的稳定充型效果[[S35]]。

### 倾转时间-角度关系
典型倾转时间与倾转角度的对应关系为近似线性：倾转时间1s对应倾转角度5°，10s对应完成90°全行程倾转，是铝合金缸盖等典型零件倾转工艺参数设定的核心参考依据[[S22]]。

### 模具温度控制
金属型模具预热阶段温度推荐控制在150℃～300℃，连续生产工作状态下的模具温度控制区间为240℃～400℃。当模具温度超过400℃后，会大幅延长铸件凝固时间，同时降低金属型使用寿命，提升铸件热裂、粘模缺陷风险[[S9]]。

### 涂料选型要求
金属型铸造涂料以耐火骨料作为基底，添加石墨等润滑剂和粘接剂，兼具隔热、防金属液粘模、调节铸件凝固速率的功能。针对不同合金需适配不同耐火度的涂料，高温合金浇注场景下应选用高耐火度厚涂层以降低金属液对模具的热冲击[[S9,S38]]。

### 铝合金铸件典型工艺参数组合
以某铝合金缸盖铸件为例，优化后的典型工艺参数组合为[[S40]]：

| 参数项 | 设定值 |
|--------|--------|
| 浇注温度 | 约720℃ |
| 浇口杯预热温度 | 320℃ |
| 上模预热温度 | 400℃ |
| 下模预热温度 | 350℃ |
| 砂芯预热温度 | 室温（约25℃） |
| 总倾转时间 | 约12 s |
| 最大倾转角度 | 90° |

该参数组合可保障形成自下而上的顺序凝固路径，降低缩松缺陷概率[[S40]]。

## 适配材质与典型应用

### 适配材质范围
倾转式重力铸造机适配的主流材质范围覆盖[[S5,S12,S20,S43]]：

- **铝合金**：应用最广泛，如AC4B（近YL112）等铸造铝合金
- **镁合金**：进气歧管等发动机部件
- **铜合金**：黄铜、铝青铜等易氧化铜合金的适配性尤为突出
- **锌合金**：标准力学性能试块
- **特殊材料**：可兼容钛合金测试棒/叶片、半固态浆料、大块非晶等

ISO 1338-1977《铸造铜合金 成分和力学性能》标准中明确将倾转浇注作为铜基合金永久模标准试块的规范浇注方式；ASTM B791等相关铸造合金标准亦将倾转浇注列为永久模重力铸造的通用合规工艺[[S8,S4]]。

### 典型适配铸件特征
适合生产的铸件具备以下特征：最小壁厚普遍≥3 mm，中等复杂程度，允许内置砂芯形成内腔结构的非回转类零件。不适合壁厚小于2 mm的超轻薄大批量压铸件类型[[S40,S33]]。

### 典型工业应用案例
| 应用领域 | 典型零件 |
|----------|----------|
| 汽车工业 | 铝合金缸盖、进气歧管、铝合金活塞、轮毂、排气管、铝合金底盘结构件 |
| 五金水暖 | 易氧化黄铜阀体、大型铜合金导体铸件 |
| 航空航天 | 钛铝合金叶片、2L99系列高性能铝合金精密铸件 |

## 优点与局限性

### 与关联工艺的对比
倾转式重力铸造机与各类关联铸造工艺的应用边界差异如以下两表所示[[S2,S6,S15,S18,S19,S33]]：

| 对比维度 | 倾转式重力铸造 | 普通固定式重力铸造 |
|----------|----------------|---------------------|
| 充型平稳性 | 层流充型，金属液下落高度大幅降低 | 顶部浇注，下落高度大，易飞溅 |
| 排气效果 | 倾转过程利于气体排出 | 排气条件差 |
| 铸件质量 | 气密性高、缩松缺陷率低 | 受人为因素干扰大，质量不稳定 |
| 自动化程度 | 可编程控制，参数精准可控 | 人工浇注为主 |

| 对比维度 | 倾转式重力铸造 | 低压铸造 |
|----------|----------------|----------|
| 充型动力 | 金属液自身重力 | 压缩空气反重力驱动（0.06～0.15 MPa） |
| 设备投资 | 仅为同规格低压铸造机的1/6 | 投资较高 |
| 模具设计 | 模具尺寸更小、设计简单、寿命更长 | 模具设计要求配合密封炉体 |
| 极致致密度 | 无持续反压力补缩功能，略低于低压铸件 | 可持续补缩，组织更致密 |
| 推荐应用 | 中等批量、多品种生产 | 对耐压性要求极高的薄壁件 |

| 对比维度 | 倾转式重力铸造 | 高压压铸 |
|----------|----------------|----------|
| 充型方式 | 重力低压速充型 | 高速高压注射充型 |
| 卷气问题 | 可控层流，卷气风险低 | 高速充型易卷气 |
| 可热处理性 | 可进行完整热处理强化 | 内部气孔限制热处理 |
| 生产节拍 | 低于高压压铸 | 适合十万件级超大规模量产 |

| 对比维度 | 倾转式重力铸造 | 离心铸造 |
|----------|----------------|----------|
| 成形原理 | 重力倾转移液 | 回转离心力成形 |
| 适配零件形状 | 可适配非回转复杂内腔结构零件 | 仅适合筒状、套类等回转铸件 |

### 存在的主要局限
倾转式重力铸造的典型局限性包括：结构简单但难以实现全自动精准控速，属于非铁合金铸件批量重力铸造的主流选型[[S36,S39]]；生产节拍低于高压压铸，不适合十万件以上超大规模简单小零件的生产；极致致密度略低于低压铸件，不推荐对壁面耐压性要求极高的薄壁轮毂类零件使用[[S33]]。

## 行业标准

GB/T 25370-2020《铸造机械 术语》是倾转式重力铸造机的核心国家标准，其中明确定义了以下关键技术术语[[S32,S37]]：

| 术语 | 定义 |
|------|------|
| 倾转式重力铸造机 | 合模系统可在一定角度范围内倾转的金属型重力铸造机 |
| 倾转机构 | 倾转式重力铸造机驱动合模系统进行倾转的机构 |
| 倾转力 | 倾转机构驱动合模系统倾转的力 |
| 倾转速度 | 倾转式重力铸造机合模系统倾转时的运动速度 |
| 倾转区段 | 将合模系统整个倾转角度范围分为若干个区段的其中一个区段，每个区段可单独设置不同的倾转速度 |

## 关联概念辨析

### 与倾转式低压铸造的本质区别
必须严格区分倾转式重力铸造与倾转式低压铸造[[S16]]：

- **倾转式重力铸造**：完全依靠金属液自身重力作为充型动力，仅通过设备倾转调整进液轨迹，属于重力铸造大类。
- **倾转式低压铸造**：依靠密封坩埚内0.06～0.15 MPa的压缩空气作为反重力驱动力，驱动金属液沿升液管自下而上填充型腔，属于反重力/低压铸造范畴，二者分属完全不同的铸造工艺体系。

### 与永久模铸造的关系
倾转式重力铸造是永久模铸造（Permanent mold casting）中重力工艺（Gravity process）的核心分支。在重力永久模铸造中，存在两种基本工艺形式：静态浇注（static pouring，类似砂型铸造的顶部直浇）和倾转浇注（tilt pouring，模具在近水平位置导入金属液后逐步倾转至竖直位置）[[S23,S33]]。倾转式重力铸造机正是实现后者工艺路径的专用装备。
