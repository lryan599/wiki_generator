## 概述

冷等静压成型（Cold Isostatic Pressing，CIP）是粉末冶金与先进陶瓷领域的一种室温粉末致密化成型工艺，其核心特征为通过液体传压介质对密封于柔性包套内的粉末施加各向均匀的等静压力，实现粉末体的致密化与近净成型。[[S3,S22]]该工艺属于等静压成型大类下的室温分支，与高温作业的热等静压（HIP）、温等静压工艺形成明确区分。[[S1,S22]]

冷等静压成型在国内是应用最广泛的先进陶瓷成型工艺之一，同时在难熔金属、粉末冶金特殊合金、金属基复合材料等领域发挥着不可替代的作用。[[S3,S15,S24]]

## 分类

依据弹性模具与高压容器间的装配关系，冷等静压成型分为两类基本工艺形态：[[S3,S12]]

| 分类 | 操作特征 | 生产适应性 |
|------|----------|------------|
| **湿袋式** | 包套模具在压力容器外完成装粉密封，整体浸入传压介质施压，卸压后取出模具，模具可清洗干燥重复使用 | 形状灵活性强，适合多品种、形状复杂、小批量或大尺寸制品 |
| **干袋式** | 柔性模具永久固定在压力容器内部，装粉与脱模工序在容器外完成，加压时仅径向传压作用为主 | 生产效率高，适合大批量连续自动化生产形状相对简单的制品 |

## 工艺原理与过程

### 压力传递与致密化机制

冷等静压的致密化经历多阶段微观演变：包套内粉末在均匀等静压力作用下，首先发生颗粒相对位移、增大接触面积；随压力升高，颗粒接触区域发生弹性变形；当接触区应力超过材料屈服极限或强度极限后，发生塑性变形或脆性破碎，最终形成颗粒间冷焊接与机械啮合，赋予压坯足够的机械强度。[[S22]]

### 常规工艺流程

冷等静压通用工艺流程依次为：[[S22,S16]]

```
粉末原料准备 → 装入包套 → 包套密封 → 抽真空 → 等静压压制 → 包套剥离 → 压坯检验
```

包套密封性是该工艺成败的关键环节。若密封失效，传压介质将渗入粉体，导致压制失败。[[S22]]

### 湿袋式与干袋式操作特征

**湿袋式工艺**在高压缸外部完成装粉、密封后，将整套模具完全浸入高压缸内的传压介质中施压，卸压后取出整套模具经清洗干燥即可重复使用，因其模具在成形前后不固定于高压缸内，亦称“自由模具”。该工艺可同时放置多个不同形状模具，对多品种、形状复杂的制品适应性优异，但操作耗时，人工干预较多。[[S29,S36,S21]]

**干袋式工艺**将柔性模永久固定在高压缸内部，无需每次作业取出整套模具，可在缸体外部直接完成装粉和脱模工序。该工艺生产效率显著高于湿袋式，适合大批量标准化制品的自动化生产，但其形状设计灵活性弱于湿袋工艺，且在实际应用中往往仅径向压力分量作用于高径比大的压坯，整体密度均匀性较湿袋式有所不足。[[S29,S11,S21]]

| 工作温度 | 室温（无额外加热） | 属常温成型工艺 [[S3,S25]] |
| 常见压坯致密度 | 理论密度的60%~80% | [[S12,S22]] |

### 传压介质

常见的传压工作介质分为两类：添加了防锈剂与润滑助剂的水（成本低、流动性好，适配普通工况）；专用液压油（润滑性与抗腐蚀性优异，适配高压工况）。部分工业场景也可选用甘油或刹车油。介质选型需满足与包套、容器、加压系统的化学兼容性要求，同时兼顾低压缩率与低成本特性。[[S22,S12,S11]]

### 包套材料选型

塑性包套的常用材料包括天然橡胶、合成橡胶、硅橡胶、PVC、聚氨酯。其中PVC和普通橡胶多用于单次使用场景，聚氨酯因耐磨性和抗撕裂性优异，是多次重复使用包套的主流材料。[[S22,S21]]

包套硬度应根据工艺类型与粉体材质匹配设计：[[S31]]
- 湿袋法塑性包套肖氏硬度约**45 HSA**
- 干袋法固定模包套肖氏硬度区间**65~95 HSA**
- 成形普通金属/陶瓷粉末推荐硬度低于**65 HSA**
- 成形金属纤维管类压坯推荐橡胶包套硬度**60~85 HSA**

### 包套尺寸设计核心准则

冷等静压包套模具设计需依据以下关键原则：[[S20,S19]]
- 须先通过小样试验测定粉体的充填密度与线性压缩比，再按模腔截面尺寸 = 压坯成品截面尺寸 / (1−线性压缩率) 进行尺寸校核；
- 大高径比棒坯的径向压缩比通常比轴向压缩比高约10%；
- 刚性模件所有棱角须磨圆至半径至少1 mm，以防撕裂塑性包套；
- 在191~294 MPa压力下成形内径100~200 mm、壁厚50~100 mm的管坯时，金属模芯壁厚宜大于20 mm，以防模芯翘曲导致内孔尺寸失控。

## 设备组成

冷等静压设备系统主体由高压泵、高压工作室（高压容器）、压力表、密封盖及卸压阀构成，通过外部高压泵向密闭容器内注入高压传压介质实现加压。[[S34]]

### 高压容器结构

高压容器主流结构形式包括：整体锻造单层厚壁容器、多层嵌套式组合容器，以及预应力钢丝缠绕式安全结构容器。端部密封形式分螺纹锁紧密封和销轴+轭架无螺纹密封两类。由于高压容器承受周期性加卸压载荷，需定期开展无损检测，防止疲劳裂纹引发安全事故。[[S21]]

### 包套模具构件

冷等静压包套模具由四大基本构件组成：[[S36,S5]]
- **塑性模件**：弹性橡胶/塑料材质，兼具型模与压力传递双重功能；
- **刚性模件**：金属材质，用于成型高精度内孔或复杂外轮廓；
- **端口密封装置**：需兼具塑性（适配包套变形避免撕裂）和刚性（捆扎密封）双重特性；
- **支撑装置**：分为固定型带孔支撑筒和随机型弹性夹板两类；成形高径比铍粉棒坯时采用随机支撑装置可将弯曲量控制在5 mm/m以内。[[S10]]

## 优势与局限性

### 对比单向钢模压制

| 维度 | 冷等静压优势 | 冷等静压劣势 |
|------|-------------|-------------|
| 密度均匀性 | 无模壁摩擦，压坯截面任意位置密度均匀；同压力下生坯密度和强度更高 [[S28,S18,S27]] | — |
| 形状能力 | 可成型大高径比、薄壁、带倒扣/交叉孔结构的复杂形状零件；最大可制备长度5 m的陶瓷管件 [[S28,S33]] | 柔性包套直接成型的部位尺寸公差大、表面质量差 [[S28,S18]] |
| 润滑需求 | 无需添加润滑剂 [[S28]] | — |
| 生产效率 | — | 湿袋式工艺需反复开闭容器，人工干预多，生产效率较低 [[S28,S18,S33]] |
| 成本 | 低批量零件模具成本更低 [[S28]] | 高压设备初始投资大，操作安全管理要求严格 [[S3]] |

### 对比热等静压

| 维度 | 冷等静压特点 |
|------|-------------|
| 设备成本 | 无需配套高温加热系统，设备投资更低 [[S15]] |
| 操作复杂度 | 室温作业，工艺更简单 [[S3]] |
| 工序衔接 | CIP+烧结后的坯件可直接进行无包套热等静压处理 [[S15]] |
| 致密化程度 | 仅能实现室温预致密化到60%~80%理论密度，无法直接获得全致密零件，后续需配套烧结或热等静压 [[S8,S32]] |

## 典型缺陷及机理

| 缺陷类型 | 成因 | 防控思路 |
|----------|------|----------|
| 包套破裂/密封失效 | 包套材质选型不当、焊接封装工艺缺陷 | 提前开展密封性检漏；依传压介质类型匹配耐油/耐水柔性包套材质 [[S27]] |
| 素坯局部密度不均 | 装粉松装密度差异过大、未充分抽除孔隙残留空气 | 优化装料均匀性，压制前充分抽真空 [[S27,S28]] |
| 素坯开裂 | 泄压速率过快导致残余应力瞬间释放 | 设定多级梯度泄压制度，缓慢释放压力 [[S28,S18]] |
| 素坯分层 | 粉体颗粒级配不合理、混料均匀性不足 | 优化颗粒级配，提升混料均匀度，必要时添加粘结剂 [[S6]] |

## 应用领域

冷等静压在以下领域发挥关键作用：

1. **粉末冶金零部件**：成型常规铸锭冶金工艺无法制备的特殊合金与复杂形面零件，配套烧结工序可生产无油轴承、结构件等。[[S15,S24]]

2. **先进陶瓷制品**：国内应用最广泛的先进陶瓷成型工艺之一，批量生产氧化铝陶瓷真空开关管壳、陶瓷轴套、火花塞、陶瓷磨球等产品。其中配套自动干袋系统生产陶瓷件产能最高可达10,000件/小时，8腔干袋式火花塞绝缘子量产产能可达2,800件/小时。[[S3,S28,S18]]

3. **难熔金属**：适配钨、钼、钽等常规熔铸工艺难以成型的高熔点材料。例如在W‑La₂O₃‑CeO₂弥散强化钨合金制备中，可采用220 MPa保压3 min的冷等静压工艺完成预成型。[[S15,S3]]

4. **金属基复合材料**：作为粉末冶金法制备金属基复合材料的前置成型工序，完成混合粉体的均匀预压，配合后续热等静压、烧结等工序获得高性能复合材料制品。[[S4]]

5. **航空航天铝合金预成型坯**：用于快速凝固铝合金粉末的工业级压实工序。[[S17]]

### 与压铸行业的工艺互补关系

压铸工艺仅适配铝、镁等低熔点有色合金，无法直接成型高熔点难熔金属及高性能陶瓷类零件；冷等静压可制备上述压铸工艺无法覆盖的材料预成型坯，经后续致密化工序得到高性能零件，二者在大尺寸复杂构件制备领域形成工艺互补。[[S15,S24]]

## 相关标准与规范

GB/T 8845-2017《模具术语》国家标准明确了冷等静压模的分类定义：[[S26]]
- **湿袋冷等静压模**：常温下利用液体介质直接通过包套对粉末加压成形的等静压模。
- **干袋冷等静压模**：常温下利用液体介质通过可变形软模对粉末加压成形的等静压模。
- **包套**：容纳粉末并传递压力的柔性腔体零件。
