## 定义与术语指称

“压缩区（塑料模）”是一个多义术语，在我国国家标准 GB/T 8845-2017《模具术语》中作为正式词条被收录[[S2,S17]]。在塑料成型领域内，该术语主要存在三类通用指称[[S19,S31]]：

1. **注塑机/挤出机塑化螺杆的压缩段**（行业使用约占65%），亦称塑化段，是螺杆上完成物料压实、排气与熔融过渡的功能区。
2. **挤出类模具内部的流道压缩区**（约占25%），指从分流支架出口到口模入口之间流道截面积逐步收敛缩小的区域。
3. **压缩成型工艺的压缩阶段**（约占10%），即合模施压后型腔内物料完成压实、充模、固化定型的工艺阶段。

## 螺杆压缩段

### 核心作用

螺杆压缩段（压缩区）位于加料段与计量段（均化段）之间，承担四项核心功能[[S28,S43]]：

- **塑化压实**：通过螺槽深度逐渐变浅造成容积缩小，将松散固态物料压实，补偿熔融后物料密度的增大，使熔体获得足够致密度。
- **排气脱挥**：将颗粒间裹挟的空气、水分及成型过程中释放的挥发性物质挤出，避免带入后续模腔。
- **熔融均化**：通过剪切生热配合外加热，使物料从固态连续转变为黏流态，并提升熔体温度与黏度的一致性。
- **建压输送**：为后续计量段生成必要的熔体压力，保障定压、定量、定温地向机头或喷嘴输送熔体[[S34]]。

### 结构分类

按压缩段长度特征，塑化螺杆压缩区分为三类[[S19,S1,S32]]：

| 类型 | 压缩段长度特征 | 塑化能量转换特征 | 适配塑料类型 | 典型品种 |
|------|----------------|------------------|--------------|----------|
| 渐变型 | 占螺杆全长的55%～65% | 缓和 | 宽软化温度范围、高黏度非结晶型塑料 | 硬质聚氯乙烯、聚碳酸酯、ABS、聚苯醚 |
| 突变型 | 约1～4倍螺杆直径D | 剧烈 | 低黏度、有明确熔点的结晶型塑料 | 聚乙烯、聚丙烯、聚甲醛、尼龙 |
| 通用型 | 约3～5个螺距（3～4倍D） | 适中 | 兼顾结晶与非结晶塑料 | 两类塑料均可加工，但塑化质量与能耗不如专用螺杆 |

### 几何参数与设计公式

**螺槽深度渐变度**：  
压缩段螺槽深度 h₂ 为连续渐变变量，从加料段定值深螺槽 h₁ 逐步变为计量段定值浅螺槽 h₃[[S35,S6]]。渐变度 B 按下式计算：

\[
B = \frac{h_1 - h_3}{L_2}
\]

式中 L₂ 为压缩段轴向长度[[S6]]。

**几何压缩比 ε**：  
定义为螺杆加料段第一个螺槽容积与计量段最后一个螺槽容积之比，亦可由螺槽深度直接计算[[S25,S34]]：

\[
\varepsilon = \frac{(D - h_1)h_1}{(D - h_3)h_3}
\]

式中 D 为螺杆外径，h₁ 为加料段螺槽深度，h₃ 为均化段螺槽深度。

**不同类型螺杆的压缩比取值区间**[[S40,S18,S12]]：

| 螺杆类型/机种 | 压缩比范围 |
|--------------|-----------|
| 渐变型热塑性注射螺杆 | 2～3 |
| 突变型热塑性注射螺杆 | 2.5～3.5 |
| 通用型热塑性注射螺杆 | 2～2.5 |
| 挤出成型螺杆（通用） | 2～5 |
| 热固性注射螺杆（压缩型） | 1.05～1.2 |
| 热固性注射螺杆（无压缩型） | 1～1.05 |
| 热固性注射螺杆（变深型） | 0.8～1 |

**不同塑料品种推荐压缩比**[[S14]]：

| 塑料品种 | 推荐压缩比 |
|----------|-----------|
| 聚苯乙烯 | 2～4 |
| 尼龙 | 2～4 |
| 聚碳酸酯 | 2.5～3 |
| 聚氯乙烯（粒料） | 3～4 |
| 聚氯乙烯（粉料） | 3～5 |
| 聚乙烯 | 3～4 |
| 聚丙烯 | 3～5 |

**压缩段长度参数速查**[[S15,S29]]：

| 螺杆类型 | 压缩段长度取值 |
|----------|---------------|
| 渐变型螺杆（非结晶型塑料） | 占螺杆全长55%～65% |
| 突变型螺杆（结晶型塑料） | 1～4倍螺杆直径D |
| 通用型螺杆 | 3～5个螺距（约3～4倍D） |

### 工艺参数条件

- 螺杆圆周速度：常规工况取0.4～1.5 m/s；有一般混合要求时取0.1～0.3 m/s[[S11]]。
- 常规注射机塑化阶段螺杆转速无级可调范围：0～90 r/min[[S13]]。
- 热固性塑料注射压力：通常处于120～170 MPa区间[[S13]]。

注射过程中螺杆速度并非恒定，典型工艺曲线显示，随着填充时间增加螺杆速度逐步降低（例如2 s时速度降至最大速度的22%，5 s时仅为6%），以此避免压缩区熔体受到过度剪切而产生过热[[S20]]。

### 与上下游区段的关系

压缩段前端衔接加料段，后端衔接计量段[[S15,S34]]：
- 加料段负责将固态物料稳定输送至压缩段入口，其输送速率须与压缩段的熔融速率相匹配，否则会造成堵料或熔融不充分。
- 计量段要求压缩段出口物料已100%完成熔融，以便实现定压、定量、定温的熔体输送。

## 模具流道压缩区

挤出类机头内部的流道压缩区位于分流支架出口与口模入口之间[[S31,S43]]。其作用为：对经过分流锥分流的多股熔体施加剪切作用以进一步提升塑化程度，同时提升熔体压力以消除分流支架造成的熔接痕，保障挤出制品密实度[[S43]]。

**关键结构参数**[[S3,S9]]：

| 参数 | 取值规则 |
|------|---------|
| 压缩角 β | 高黏性胶料取10°～30°；低黏性胶料取20°～40°；设计要求 β < 扩张角 α |
| 压缩区轴向长度 L₂ | L₂ = (1.5～2.5) d₀，d₀ 为栅板出口直径 |

## 压缩成型工艺阶段的压缩区

在压缩模塑工艺中，压缩区指合模施压后型腔内物料在压力与温度共同作用下完成压实、充模及固化定型的阶段。典型参数为[[S44]]：

- 压力范围：15～55 MPa
- 温度范围：120～260 ℃
- 压缩时长：根据制品壁厚差异，从数分钟至十余分钟不等

## 常见缺陷与机理

与压缩区直接相关的制品缺陷主要包含以下四类：

1. **塑化不均/白点（鱼眼）**：压缩比过小或压缩段长度与物料软化特性不匹配时，未完全熔化的固态颗粒残留在熔体中，在制品表面形成白点或鱼眼缺陷，同时降低外观均匀性和局部力学性能一致性[[S36]]。

2. **银纹/气泡**：压缩区排气能力不足，颗粒间裹挟的空气无法完全排出，残留气体随熔体进入模腔，最终在制品内部形成气泡、外部形成银纹气痕[[S26,S39]]。

3. **焦化/变色（热降解）**：压缩区剪切速率过高、局部剪切作用过强或物料停留时间过长，导致热敏性塑料温度超过热稳定阈值，引发分子链断链降解、变色及力学性能显著下降[[S38,S41]]。

4. **剪切过热**：压缩区螺槽过浅或螺杆转速设置不当，使熔体受到过度剪切摩擦，局部温度异常升高，进一步加剧热降解风险[[S10]]。

## 设计局限

当前通用压缩区结构存在两方面的固有限制[[S36]]：
- 固定压缩比的单螺杆压缩区无法完全适配全品类塑料加工需求，对共混改性体系或高填充特种塑料，通用结构容易出现塑化不足与过度剪切降解的两难矛盾。
- 现有压缩区的脱气能力存在上限，无法完全排除物料中高含量的挥发分，须搭配额外的排气辅助结构（如排气式螺杆）。

## 与压铸领域压缩区域的异同

塑料领域的压缩区与半固态压铸/挤压铸造领域的压缩区域虽同属成型工艺中的压力传递核心区段，但存在显著差异[[S4,S22]]：

| 对比维度 | 塑料成型压缩区 | 压铸/挤压铸造压缩区域 |
|---------|---------------|---------------------|
| 加工对象 | 可压缩非牛顿流体塑料熔体 | 近似不可压缩的半固态金属浆料/熔体 |
| 工作温度 | 通常200～300 ℃ | 可达600 ℃以上 |
| 核心功能侧重 | 固态颗粒熔融、塑化均混 | 半固态金属补缩、消除内部气孔实现致密化 |
| 共性特征 | 均通过缩小流通截面积提升介质所受压力与剪切作用 | 同上 |
