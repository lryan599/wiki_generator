## 概述

输送式落砂机（英文标准术语：conveyer shakeout machine）在国家标准GB/T 25370-2020《铸造机械 术语》中定义为“振动方向角小于90°的惯性振动落砂机”，属于兼具铸件落砂与输送功能的专用振动输送机，落砂过程中铸件和铸型在槽体上可产生水平方向上的连续运动[[S28,S13]]。该设备是自动化连续造型生产线配套的核心落砂设备，也是铸造全流程中连接造型工部、砂处理工部与铸件清理工部的关键枢纽[[S25,S20]]。

输送式落砂机利用激振力的垂直分力实现撞击破型与摩擦脱砂，同时依靠激振力的水平分力驱动铸型呈跳跃式前进，在连续作业过程中同步完成大砂团破碎、铸件与型砂分离的落砂功能[[S31,S19]]。其激振角（激振力方向与落砂机床面的夹角）通常设置为60°~70°，设备工作在远共振区，具有激振力大、振幅稳定的特点[[S31,S19]]。

## 分类与在落砂设备体系中的位置

### 在铸造落砂设备体系中的位置

输送式落砂机属于铸造落砂设备大类下的机械振动落砂机分支，与固定式惯性振动落砂机、滚筒落砂机、电磁振动落砂机等并列[[S28]]。在行业术语体系中，通常将可实现全流程不间断砂型进给到铸件输出的输送式落砂设备统称为连续式落砂机；通过式落砂机是连续式落砂机的特殊子类，特指可匹配自动化造型线节拍、砂箱或无箱铸型完全沿水平方向贯通穿过设备全程完成落砂的结构类型[[S21,S25]]。

两台或多台串联运行的输送式落砂机可构成落砂机组（shakeout set），用于造型生产线两级分段落砂作业[[S28,S25]]。

### 与同类设备的主要区别

**与固定式惯性振动落砂机对比**：固定式惯性振动落砂机无主动连续输送功能，仅依靠栅格振动完成落砂，作业为间歇式，适合非生产线、单件小批量生产场景，铸件停留落砂时间可设置1~20min不等，可适配重量差异较大的重型铸件[[S21,S10]]。输送式落砂机自带连续输送功能，为连续作业模式，直接匹配自动化造型线节拍，仅适配生产线连续产出的固定规格铸型[[S21]]。

**与滚筒落砂机对比**：滚筒落砂机依靠筒体内螺旋叶片将铸件和型砂带到一定高度后自重坠落，依靠往复撞击和摩擦完成落砂，铸件磕碰磨损风险高，仅适配不怕磕碰的简单铸件，设备结构复杂、维修不便，且对进入滚筒的铸件温度及砂铁比有严格限制[[S17,S19,S27]]。输送式落砂机为敞开栅格振动结构，铸件输送过程磕碰损伤风险远低于滚筒落砂机，适配高精度复杂铸件落砂，可直接对接自动化造型线，无滚筒落砂机的温度和砂铁比限制[[S17]]。

### 子类划分

输送式落砂机可分为以下三大子类[[S19]]：

| 子类 | 特点 |
|------|------|
| 单轴式惯性振动输送落砂机 | 栅架平面与水平设置倾角，依靠单轴激振器驱动，通过倾角调整控制铸件停留时间与输送速度 |
| 双轴式惯性振动输送落砂机 | 依靠反向旋转的双轴激振器生成定向激振力，是行业应用最广泛的主流类型 |
| 惯性横振动输送落砂机 | 振动电机安装在双层圆筒一侧，驱动筒体内部型砂和铸件产生规律公转自转向前运动，自带定量喷水与除尘装置，可同步实现型砂冷却与低粉尘排放 |

按驱动方式可分为三类[[S15,S17,S31]]：

- **一台电动机驱动的箱式激振器型**：由一台电动机经联轴器、传动轴驱动两台箱式激振器强制同步旋转产生直线运动
- **两台电动机无强迫联系驱动的双轴激振器型**：由两台电动机分别驱动没有强迫联系的激振轴，激振力及振幅可通过调整偏心块位置灵活调节
- **振动电动机直接驱动型**：激振源与动力源合为一体，结构简单、维修方便，是目前主流配置

振动电动机的安装有三种布置形式：侧置（适合高度受限场合）、上置（适合落砂机下方布置输送设备的造型线）、下置（重心低但维修难度大）[[S15,S17]]。

## 工作原理与工艺作用

### 核心工作机理

输送式落砂机通过激振装置产生定向激振力，激振角通常设置为60°~70°，使激振力分解为垂直分力与水平分力[[S31,S19]]：

- **垂直分力**：产生撞击和振动作用，使砂型破碎，实现铸件与型砂的分离
- **水平分力**：驱动砂型在槽体上呈跳跃式前进，实现边落砂边输送

整个落砂过程同时包含振动输送、撞击破型、摩擦脱砂三种作用，在连续作业过程中同步完成大砂团破碎、铸件与型砂分离[[S31,S19]]。设备工作在远共振区，激振力大、振幅稳定[[S31]]。

### 在铸造生产线中的工艺角色

输送式落砂机是连接造型、砂处理和铸件清理的枢纽设备[[S25,S20]]，可直接布置在静压自动化造型线、FBO高压自动化造型线的捅箱工序下游[[S25]]。针对高比压、高紧实度的铸型，可采用1~3级串联配置实现分级落砂：第一级落砂段砂、件不筛分，水平输送自击落砂；第二级及之后完成筛分落砂，整体可实现旧砂和铸件平行输送，布局紧凑[[S25,S20]]。

## 结构组成与关键部件

### 典型整机结构

输送式落砂机主要由输送槽（包括鳞板或栅格）、激振装置、弹性支撑系统、机架及驱动系统等组成。机架和栅格作为直接承载铸型的核心参振部件，需满足足够的抗振刚度要求，避免长期交变振动作用下出现结构开裂、变形故障[[S24,S20]]。

### 输送槽类型

**栅格型输送槽**：常用栅格可设置为焊接条缝式、百叶窗式、铸钢圆孔/长孔等多种形式[[S15,S31]]。栅格尺寸需与待落砂铸件尺寸匹配，避免浇冒口、小铸件卡塞[[S31]]。

**鳞板型输送槽**：依靠环锁链条连接的连续鳞板承载铸件，牵引力大、运载量高，同时具备耐高温、耐冲击的特性，多用于短距离输送热铸件和热砂场景[[S9,S2]]。

### 激振装置

激振装置是输送式落砂机的核心驱动源，激振力应可调，用于振动输送落砂机时还需能调节激振角[[S24,S6]]。国家标准定义以下主要类型[[S28,S23]]：

| 激振器类型 | 标准定义与特点 |
|-----------|--------------|
| 箱式激振器（directional force exciter） | 采用单台电动机驱动，激振力来自双轴旋转不平衡质量所产生的离心惯性力合力。主要由箱体、轴承、齿轮、主动轴、从动轴和旋转不平衡质量（偏心块）等组成。运行中运动部件发热导致油温升高，润滑油泄漏问题难以解决，维护工作量大[[S28,S31]] |
| 双轴激振器（two-shaft exciter） | 采用两台电动机反向驱动，主要由轴承座、轴承、传动轴和旋转不平衡质量（偏心块）等组成。激振力及振幅可通过调整偏心块位置灵活调节，适配高生产率的自动造型线[[S28,S15]] |
| 电动激振器（electric vibrator，即振动电机） | 由旋转不平衡质量（偏心块）与特殊电动机组合为一体，激振源与动力源合为一体。结构简单、维修方便，是目前主流应用的激振配置[[S28,S15]] |

### 弹性支撑

弹性支撑的选型直接决定落砂机整体运行性能[[S24,S6]]。国家标准定义两类弹性元件：布置在参振机体和机座/基础之间的为隔振弹簧（isolation spring），布置在双质体落砂机的工作质体和激振质体之间的为共振弹簧（resonant spring）[[S23]]。

常规选型方案[[S24,S6]]：
- 圆柱形螺旋弹簧：多用于减振或谐振场景
- 钢与橡胶组合弹簧：多用于减振或共振场景
- 板弹簧或纯橡胶弹簧：多用于支撑或减振场景

### 驱动系统

驱动系统可分为单电机驱动配置和双电机驱动配置两类。单电机驱动方案依靠传动轴、联轴器同步驱动两台激振器，结构紧凑但传动链故障点多；双电机独立驱动方案无额外传动轴组件，整体可靠性更高，激振参数调节灵活性更强[[S15,S31]]。

### 适用条件

输送式落砂机可适配的铸件材质覆盖灰铁、球铁、铸钢、有色合金等所有常规砂型铸造材质，无特殊材质排异限制[[S17]]。可适配潮模砂、呋喃树脂砂等主流砂型的落砂作业[[S25,S12]]。适用砂箱尺寸需与设备栅床尺寸匹配，L254B型可适配1.5m×1.2m×0.95m的最大砂型[[S15]]。

## 应用场景

输送式落砂机是自动化连续造型生产线的核心配套落砂设备[[S28,S10]]，主要应用场景包括：

- **高压造型生产线**：适配高比压、高紧实度铸型的1~3级串联落砂配置[[S25,S20]]
- **静压自动化造型线**：可直接布置在捅箱工序下游[[S25]]
- **FBO高压自动化造型线**：匹配生产线连续生产节拍[[S25]]
- **引进高端造型线**：国内主流铸机厂商可提供适配德国KW、丹麦GF、意大利IMF等进口高端自动化静压造型线的定制型输送式落砂机，满足高节拍连续生产要求[[S17]]

## 优缺点与限制

### 主要优势

相比于固定式惯性振动落砂机，输送式落砂机具有连续化作业、直接匹配自动化造型线节拍的核心优势，生产过程中无需额外翻转或搬运动作[[S21]]。相比于滚筒落砂机，其敞开栅格振动结构使铸件输送过程中的磕碰损伤风险大幅降低，适配高精度复杂铸件落砂，且无铸件温度和砂铁比的严格限制[[S17,S19]]。

### 局限性与注意事项

1. **栅格堵卡问题**：输送过程中浇冒口系统及小铸件容易堵卡栅格，栅格形状、板孔尺寸及布置必须与对应落砂铸件的规格匹配[[S31]]

2. **箱式激振器机型维护负担**：配置单台电动机驱动箱式激振器的机型存在运动部件发热导致油温升高、润滑油泄漏的问题，易引发轴承损坏，维护工作量较大[[S31]]

3. **超大铸件不适用**：常规单台机型无法承载远超额定负载的大质量铸型，无法直接适配特大吨位、超大尺寸铸件的落砂需求[[S10]]

4. **高溃散性砂型适配问题**：针对高溃散性特殊砂型，若输送速度参数设置不当，易出现砂型停留时间不足导致的落砂不彻底问题[[S10]]

## 相关标准与规范

### 术语定义

- **GB/T 25370-2020《铸造机械 术语》**：将输送式落砂机定义为振动方向角小于90°的惯性振动落砂机（类目编号5.1.7），属于兼具落砂功能的振动输送机[[S28,S13]]。该标准同时定义了相关核心术语，包括箱式激振器（5.1.19）、双轴激振器（5.1.20）、电动激振器（5.1.21）、隔振弹簧（5.1.22）、共振弹簧（5.1.23）、振动方向角（5.1.24）、落砂机额定负荷（5.1.27）等[[S28,S23]]。

### 技术条件

**JB/T 6575-1993《落砂机 技术条件》**对输送式落砂机的主要规定[[S26]]：

| 项目类别 | 具体要求 |
|----------|---------|
| 零部件要求 | 激振器零件不得存在裂纹，回转件禁止焊补修整 |
| 轴承温升 | 滚动轴承温升不得超过35°C，最高工作温度不超过70°C |
| 振幅偏差 | 机身两侧板对称点垂直方向的振幅差值不得大于两点所测双振幅平均值的20% |
| 侧向偏摆 | 侧向偏摆量不得大于2mm |
| 落砂效率 | 额定有效负荷工况下不应低于98% |
| 噪声限值 | 空载工况下运行的噪声声压级不得大于93dB(A)；无法空载启动的机型可在模拟负载下测试，噪声限值同为93dB(A) |
| 制动要求 | 过共振区运行的机型必须配置可靠的制动或限幅装置，保证停车过程快速平稳 |
| 首次大修期 | 正常使用条件下首次大修期（不含更换弹簧）工作时长不得低于6000h |
| 弹簧寿命 | 正常工况下弹簧使用寿命不低于2000h，弹簧性能符合GB 1239.4对应要求 |

### 国际噪声标准

ISO相关铸造工作场所噪声防护推荐标准中明确铸造作业岗位长期暴露噪声不应超过90dB(A)，热作业岗位限值为85dB(A)，输送式落砂机的车间噪声防护设计需满足该类国际通用职业健康要求[[S18]]。

## 操作与维护要点

### 日常点检

开机前应清理设备周边砂块、铸件、浇冒口等杂物；检查所有紧固件无松动；检查激振器、传动轴承、联轴器运行过程无异常温升、异响；检查栅格衬板无过度磨损、断裂卡料；检查各类弹性元件无开裂、脱胶、异常形变；检查润滑油油位、油质符合要求；点检发现异常立即停机处置[[S29]]。

### 弹性元件维护

禁止任意旋动弹簧刚度调节螺母、增减或替换非原厂匹配的弹簧[[S5,S23]]。运行中若出现以下情况需立即更换[[S5]]：
- 金属弹簧表面裂纹、中心线弯曲超过长度1%、残余变形大于最大变形量5%
- 橡胶弹簧接合面脱胶或老化

不允许在弹簧断裂、撕裂的状态下带病运行。以橡胶弹簧作为弹性元件的设备进行焊接作业时，必须单独接好接地线，禁止将电机驱动线路作为接地回路，防止烧毁电气部件[[S5]]。

### 润滑维护

采用箱式激振器的机型应重点关注润滑油管理，定期补充对应牌号润滑脂，避免运动部件发热导致油温升高、润滑油泄漏而造成的轴承干磨损坏。联轴节需选用耐振型特殊材质产品，按设备说明周期加注润滑脂防止频繁断裂[[S31]]。
