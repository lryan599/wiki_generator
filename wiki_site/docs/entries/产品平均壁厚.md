## 概述

产品平均壁厚是压铸件几何设计中一个特殊而核心的参数。它与铸件的填充条件、凝固行为、缺陷倾向以及最终的力学性能紧密相关，几乎所有压铸工艺的核心参数（如填充时间、内浇口速度、浇注系统尺寸等）都直接将其作为计算依据[[S25,S2]]。在压铸工程实践中，该术语并不总是严格指向数学上的算术平均值，而更多地对应于铸件上占主导地位、代表绝大多数主体结构特征的壁厚数值[[S15,S23]]。

## 定义与分类

在压铸技术文献中，“产品平均壁厚”同时存在两种通用定义，均在行业实践中流通，不存在单一的强制命名边界[[S2,S15,S23,S36]]：

- **主导壁厚（名义/代表性壁厚）**：铸件上分布占比最高、出现频率最多的那种壁厚数值，直接反映零件主体的壁厚特征。这是压铸工艺设计中最常采用的“平均壁厚”含义，尤其用于快速查表选取填充时间和内浇口速度[[S23,S26]]。
- **面积加权算术平均壁厚**：将铸件划分为不同壁厚区域，按各区域表面积加权计算得到的壁厚平均值。该定义主要用于需要精确计算的场合，或当铸件壁厚分布很不均匀时[[S2,S23]]。

从参数属性上看，产品平均壁厚兼具双重角色：一方面它是压铸件结构设计固有的几何特征参数；另一方面它又是压铸工艺规划阶段的核心输入参数，是计算填充时间、选择内浇口速度和设计浇排系统的基础依据。北美压铸协会（NADCA）发布的压铸充型时间公式中，直接以平均壁厚 $S_m$ 作为核心变量[[S30,S15,S33]]。

## 计算方法

根据应用场景与所需精度，产品平均壁厚的获取方法主要有以下几种。

### 众数壁厚（主导壁厚）法

直接取铸件上相同壁厚最多、占表面积比例最大的那一档壁厚作为平均壁厚。这种方法无需复杂计算，尤其适用于压铸现场快速估算工艺参数[[S23,S26]]。在英文文献中，这往往对应“dominant wall thickness”或“representative wall thickness”的概念[[S36]]。

### 面积加权算术平均法

当铸件壁厚差异较大、需要更精确的代表值以代入公式计算时，采用面积加权平均：

$$b = \frac{b_1 S_1 + b_2 S_2 + \dots + b_n S_n}{S_1 + S_2 + \dots + S_n}$$

式中 $b_1 \dots b_n$ 为各部位壁厚（mm），$S_1 \dots S_n$ 为对应部位的表面积（mm²）[[S2,S23]]。该方法适用于绝大多数常规结构的压铸件。

### 体积加权平均（凝固模数关联法）

通过铸件的凝固模数 $M = V / F$（体积/散热表面积）来反算平均壁厚。对于均匀薄壁铸件，存在近似关系 $s_m \approx 2M$，即平均壁厚约为凝固模数的2倍[[S36,S10,S31]]。该方法直接基于热学原理，计算精度最高，尤其适用于需要进行详细热分析的精密压铸件[[S3,S36]]。

### 壁厚不均匀时的简化加权法

当铸件壁厚分布极不均匀，且缺乏完整的三维模型时，可采用如下简化公式：

$$s_m = \frac{s_{\max} \cdot P_1 + s_{\min} \cdot P_2 + \frac{s_{\max}+s_{\min}}{2} \cdot P_f}{P_1+P_2+P_f}$$

其中 $P_1$、$P_2$ 分别为最大、最小壁厚区域所占百分比，$P_f$ 为剩余未单独统计区域的百分比。由此计算得到的结果与精确的体积/表面积法非常接近[[S36]]。

现代压铸件设计广泛采用三维CAD软件（如UG NX），其内置的壁厚检测工具可基于滚球法或射线法自动生成全域壁厚分布云图，并直接输出计算得到的平均壁厚值，如某汽车壳体压铸件经软件分析显示平均壁厚为5.31 mm[[S8,S4,S37]]。这类方法能够直观分辨厚壁与薄壁区域的占比，为工艺设计提供可靠的几何输入。

## 工艺作用

### 填充时间

填充时间是压铸工艺中受平均壁厚影响最直接的参数。两者大体呈正相关关系：薄壁铸件需要极短的填充时间和较高的内浇口速度以防止提前凝固，厚壁铸件则可选用较长的填充时间以保证排气充分[[S20,S2]]。

国际上已积累了大量基于平均壁厚计算填充时间的经验公式：

- NADCA通用公式（考虑热条件）：  
  $$t = K \left( \frac{T_i - T_f + S Z}{T_f - T_d} \right) T$$
  式中 $T$ 直接代入铸件平均壁厚（mm），$K$ 为模具钢经验常数，$T_i$ 为内浇口金属温度，$T_f$ 为最低流动温度，$T_d$ 为模具表面温度，$S$ 为充型结束时允许的固相分数，$Z$ 为单位转换系数[[S33,S1]]。

- 带温度修正的通用式（形式多见）：  
  $$t = 0.034 \, \frac{T_n - T_y + 64}{T_n - T_m} \, b \quad \text{(t: s, b: mm)}$$
  式中 $T_n$ 为内浇口金属温度，$T_y$ 为液相线温度，$T_m$ 为模具温度[[S2,S23,S26]]。

- 针对壁厚1.5～6.4 mm铝合金的简化估算式：  
  $$t = 0.0125 \, b \quad \text{(t: s, b: mm)}$$ [[S2,S38]]

- 瑞士布勒公司集成在压铸计算尺中的早期D.C.R.F.公式：  
  $$t = 35 (T_a - 1) \quad \text{(t: ms, } T_a \text{ 为平均壁厚 mm)}$$
  其充型时间允许公差为计算值的±40%[[S15,S20]]。

- 英国BNF统计公式（$H$ 为平均壁厚mm，$t$ 单位ms）：  
  铝合金：$t = 40H$  
  镀铬要求高的锌合金：$t = 6H + 7$  
  镀铬要求低或无镀铬要求的锌合金：$t = 7H + 13$ [[S15,S9]]

实际应用中，常直接查表选取平均壁厚对应的推荐填充时间范围。以下为压铸行业常用的对照数据（以铝合金为典型，也适用于其他合金）[[S35,S7,S19,S23]]：

| 平均壁厚（mm） | 充填时间推荐范围（s） |
|:-----------:|:----------------:|
|     1.0     |    0.010～0.014    |
|     1.5     |    0.014～0.020    |
|     2.0     |    0.018～0.026    |
|     2.5     |    0.022～0.032    |
|     3.0     |    0.028～0.040    |
|     4.0     |    0.040～0.060    |

|     6.0     |    0.056～0.084    |

|    10.0     |    0.100～0.130    |

### 内浇口厚度与浇注系统设计

平均壁厚通过凝固模数间接决定了内浇口的合理厚度。行业经验公式为：

- 铝合金：$h_{\text{Al}} = 3.7M + 0.5$
- 锌合金：$h_{\text{Zn}} = 3.3M + 0.4$
- 镁合金：$h_{\text{Mg}} = 2.3M + 0.4$

其中凝固模数 $M$ 可取为平均壁厚的二分之一（单位：cm）[[S27,S40]]。因此，平均壁厚越大，所需内浇口越厚，以保证补缩通道在压力下保持畅通。

### 冷却速率与凝固

铸件冷却速率随平均壁厚增大而显著下降。以AC46000合金的Rheo-HPDC（半固态流变压铸）试验为例：壁厚3 mm时平均冷却速率可达256 °C/s，5 mm时为211 °C/s，8 mm时降至147 °C/s，10 mm时仅为95 °C/s[[S32]]。冷却速率的降低会导致晶粒粗化和偏析程度变化，进而影响铸件力学性能。

### 缩孔倾向

壁厚超过约6 mm的压铸件，中心区域极易形成孤立液相区，补缩通道提前凝固而失效，导致内部缩孔、缩松缺陷显著增加[[S43,S44]]。因此，压铸件设计的基本原则是利用均匀薄壁和加强筋来替代局部厚大截面，避免因壁厚过大而无法通过压力补缩消除内部孔隙[[S22,S21]]。

### 模具热平衡

平均壁厚直接决定单次压铸带入模具的总热量。铸件壁厚越大，所需匹配的冷却系统功率越高（包括点冷、循环冷却水回路等），否则模具温度将持续上升，引发局部热节、粘模等缺陷，无法达到稳定生产的热平衡状态[[S12]]。

## 设计准则与典型取值范围

### 推荐壁厚与极限

在满足强度与刚度的前提下，压铸件应尽可能减小壁厚。通常建议的壁厚上限为4.5 mm，一般工况下最大壁厚不宜超出6 mm[[S21,S25]]。特殊需求场景下壁厚可放宽至12 mm[[S42]]。主流压铸合金的合理壁厚区间大致为[[S43,S42]]：

- 锌合金：1～3 mm
- 铝合金、镁合金：1.5～4 mm
- 铜合金：2～4 mm

三类合金在优良工艺条件下可达到的极限最小壁厚如下[[S42,S22]]：

| 合金种类 | 极限最小壁厚（mm） |
|:------:|:------------:|
| 锌合金   |      0.5      |
| 铝合金   |      0.7      |
| 镁合金   |      0.8      |

铸件外形尺寸与体积也会影响壁厚选取。体积大于200 in³的压铸件，建议壁厚不低于3/16 in；体积小于5 in³的小型铸件，壁厚可在1/16 in区间内设置[[S14]]。

### 壁厚均匀性与过渡

壁厚均匀是压铸件设计的第一准则。局部壁厚突变应尽量避免：常规设计要求局部最大壁厚不宜超过相邻最小壁厚的2倍，且需要通过圆滑过渡或增设加强筋等方式减薄厚区，实现壁厚均匀分布[[S21,S22]]。不等壁厚区域之间的平滑过渡可参考专用圆角设计原则[[S6,S18]]。

### 壁厚对力学性能的影响

压铸件壁厚增加会导致力学性能明显下降。以铝合金为例，当壁厚从约1 mm增加至10 mm，抗拉强度从接近300 MPa逐步降低至约120 MPa，密度也从约2.85 g/cm³降至2.7 g/cm³左右[[S28]]。在壁厚从2 mm增加到6.5 mm的区间内，锌合金强度最大降幅约为20%，而铝合金和镁合金的降幅可达30%[[S45,S17,S29]]。因此，单纯依靠增加壁厚来提高铸件强度的设计理念是错误的。

## 与其他核心参数的关系

### 凝固模数

对于空间对角线超过50 mm、整体壁厚均小于0.5 cm的均匀薄壁压铸件，可近似取凝固模数 $M \approx s/2$（$s$ 为平均壁厚，单位cm）[[S10,S16,S31]]。此关系极大简化了浇注系统和补缩系统的计算。但当铸件存在大量厚壁区或厚度差异显著时，必须用 $M = V/F$ 的精确公式计算。

### 模流分析中的输入角色

主流压铸模流分析软件（如MAGMAsoft、Flow-3D、Moldflow等）将铸件全域的实测壁厚作为基础几何参数导入前处理模块。软件支持滚球法、射线法等自动生成壁厚分布云图，区分不同壁厚等级，并以此为基础开展填充与凝固模拟[[S24,S4,S37]]。仅在壁厚极均匀的小型铸件场景下，才可输入单一平均壁厚实现简化计算。

## 常见误区与注意事项

压铸实践中一个典型而危险的错误是：不加区分地使用全局简单算术平均得到的壁厚值，代替铸件的代表性特征壁厚来进行工艺设计。这种做法会掩盖局部远高于名义值的厚壁区域，导致：

- 局部厚区在工艺设计阶段未被识别，实际生产中产生未预判的缩孔、缩松缺陷；
- 基于错误壁厚值计算的内浇口厚度和填充时间严重偏离实际需求，引发充型不良与压力补缩失效；
- 壁厚差异大的铸件，不同区域冷却速率相差悬殊，导致不均匀收缩和翘曲变形，产品力学性能大幅低于理论预期[[S14,S22,S13]]。

因此，在确定用于工艺计算的“平均壁厚”时，必须明确其物理意义，优先采用基于主导壁厚或面积加权的计算方法，并通过三维壁厚分析软件对厚壁区域的分布进行充分识别与评估。
