## 定义

石墨面积率在铸铁金相分析中是一个有严格标准定义的技术术语，特指**单颗石墨的实际面积与该石墨最小外接圆面积的比值**，是评定单个石墨形状的指标[[S18,S37]]。其计算公式为：

\[
单颗石墨的面积率 = \frac{石墨的实际面积}{石墨的最小外接圆面积}
\]

确定外接圆时，一般以石墨的最大投影长为直径，无需对石墨三维轮廓进行额外测算[[S7,S37]]。石墨面积率可用图像分析仪直接测定，也可采用其他近似方法测定[[S26]]。

> **术语辨析**：在铸铁领域的日常交流与部分文献中，也会出现以“石墨体积分数”或“石墨面积百分比”指代石墨相总面积占测量区域总面积之比的用法（例如灰铸铁中石墨体积占比可达6%~10%[[S29]]）。严格区分二者的依据在于：GB/T 9441标准中“石墨面积率”专指单颗石墨的**形状规整度参数**，属于二维剖面度量；而“石墨体积分数/面积分数”描述的是材料中石墨相的整体**含量参数**，属于体视学参数。本词条以GB/T 9441标准定义为准。

## 分类与判定

### 按石墨形态划分

GB/T 9441标准以石墨面积率为核心判据，将石墨形态划分为五类[[S3,S8,S14]]：

| 石墨形态 | 面积率区间 |
|---------|-----------|
| 球状石墨 | ≥0.81 |
| 团状石墨 | 0.61～0.80 |
| 团絮状石墨 | 0.41～0.60 |
| 蠕虫状石墨 | 0.10～0.40 |
| 片状石墨 | <0.10 |

标准还给出了GB/T 9441各种形状石墨面积率的详细对照表[[S17]]，进一步细化了每种石墨形态下面积率的具体数值分布：

| 石墨形态 | 典型面积率数值（由高到低） |
|---------|------------------------|
| 球状石墨 | 0.98、0.96、0.94、0.90、0.88、0.86、0.84、0.83、0.82、0.81 |
| 团状石墨 | 0.80、0.78、0.75、0.73、0.71、0.68、0.67、0.65、0.63、0.61 |
| 团絮状石墨 | 0.60、0.58、0.55、0.53、0.50、0.48、0.45、0.43、0.42、0.41 |
| 蠕虫状石墨 | 0.40、0.38、0.37、0.35、0.33、0.32、0.28、0.26、0.23、0.21 |
| 蠕虫状及片状石墨 | 0.20、0.19、0.18、0.17、0.16、0.14、0.10、0.08、0.07、0.05 |

### 按铸铁类型的特点

不同类型的铸铁因其石墨形态不同，面积率分布具有明显特征[[S17,S29,S33,S16]]：

| 铸铁类型 | 典型石墨形态 | 典型面积率范围 | 性能特点 |
|---------|-------------|--------------|---------|
| 灰铸铁 | 片状（ASTM Type VII） | 0.05～0.20 | 抗拉强度最低，消振性最优 |
| 可锻铸铁 | 团絮状（ASTM Type III） | 0.41～0.60 | 力学性能显著优于灰铸铁，消振性介于灰铸铁与球墨铸铁之间 |
| 蠕墨铸铁 | 蠕虫状（ASTM Type IV） | 0.21～0.40 | 导热性显著优于球墨铸铁，抗拉强度、延伸率优于灰铸铁但低于球墨铸铁 |
| 球墨铸铁 | 球状（ASTM Type I） | 0.81～0.98 | 对基体的割裂作用最小，基体强度利用率可达70%～90%，力学性能最优 |

蠕墨铸铁在二维抛光面上要求至少80%的石墨为蠕虫状[[S33]]，其余应为球团状石墨，不允许出现片状石墨。

## 与相关参数的异同

### 石墨面积率与球化率

石墨面积率是表征**单颗石墨个体形状**的独立参数，而球化率是**统计级综合指标**。GB/T 9441标准中球化率的计算流程为：先查出视场中每颗石墨的面积率，对照球状修正系数表换算每颗石墨的加权系数，再经加权求和计算该视场的球化率[[S26,S40]]。

球化率的计算公式为：

\[
球化率 = \frac{1 \times n_{1.0} + 0.8 \times n_{0.8} + 0.6 \times n_{0.6} + 0.3 \times n_{0.3} + 0 \times n_{0}}{n_{1.0} + n_{0.8} + n_{0.6} + n_{0.3} + n_{0}} \times 100\%
\]

其中系数1.0、0.8、0.6、0.3、0为根据石墨面积率区间确定的球状修正系数，n为对应各修正系数的石墨颗数[[S36]]。

2021年修订的GB/T 9441—2021标准将球化率评定方法从传统计数法改为面积法，定义球化率为圆整度≥0.6（对应石墨面积率≥0.6）的石墨颗粒面积占石墨总面积的百分比[[S4]]。

不同标准体系中球化率的计算方法存在差异：
- **GB/T 9441**：采用基于面积率转换修正系数的颗粒计数加权法[[S36]]
- **ASTM E2567**：采用面积加权法，公式为 Nodularity = (ΣA_Nodules + 0.5×ΣA_Intermediates) / ΣA_Particles≥10μm × 100%，仅统计等效直径≥10μm的石墨颗粒[[S20]]
- **ISO 945-4**：定义石墨圆度 Roundness = (4/π) × A / l_max²（与GB/T 9441的石墨面积率物理含义等价），用于自动化图像分析定量评定石墨形态[[S20]]

### 石墨面积率与石墨大小等级、石墨数量密度

石墨面积率、石墨大小等级和石墨数量密度是三个**完全独立的评价维度**，三者共同作为GB/T 9441标准规定的球墨铸铁金相检验核心参数，从形状、尺寸、分布数量三个独立维度表征显微组织特征[[S3,S23]]：
- **石墨面积率**：衡量单颗石墨的**形状规整度**
- **石墨大小等级**：衡量石墨最大投影尺寸的**分级评定参数**，高面积率的球状石墨可对应任意大小等级[[S3]]
- **石墨数量密度**：单位面积视场内的石墨颗粒**计数参数**，与单颗石墨的面积率无直接关联[[S8]]

## 测量方法与技术规范

### 制样要求

金相试样应在与铸件同时浇注、同炉热处理的单铸试块、附铸试块或铸件有代表性的部位上截取[[S14,S37]]。试样制备的规范为：
- 截取和制备过程中应防止组织发生变化，最终抛光试样表面不允许存在粗大划痕[[S14]]
- 石墨相需完整保留在抛光表面，不被撕裂或拖拽脱出；推荐在最终抛光阶段使用金刚石抛光粉以提升石墨留存率[[S14,S21]]
- **石墨观测步骤仅需抛光完成后直接进行**，无需进行硝酸酒精浸蚀；基体组织的检验则在经2%～5%硝酸酒精溶液浸蚀后进行[[S14,S37]]

### 图像采集规范

石墨面积率测量对图像采集有明确规范[[S7,S26]]：
- 默认观测放大倍数为100倍，对应视场直径为70mm
- 被视场周界切割的石墨不计入统计
- 100倍放大下尺寸小于2mm的石墨可忽略不计
- 若石墨大多数小于2mm或大于12mm，可适当放大或缩小倍数，但视场内石墨数一般不少于20颗
- 正式采集图像前需完成系统像素尺寸标定
- 可配置景深扩展功能，采集多层不同对焦的图像，拟合得到全视场清晰的全景深图像[[S1]]

### 阈值分割与二值化

图像分析的核心步骤为基于灰度阈值的二值化处理[[S1,S19]]：
- 将灰度大于等于设定阈值的像素标记为石墨目标区域，其余标记为金属基体背景
- 可采用大津法（OTSU）或迭代法自动计算最优分割阈值
- 必要时通过设置双阈值精准分离石墨相与背景相
- 完成二值化后可搭配中值滤波、数学形态学运算去除孤立噪声点，得到边缘清晰的石墨二值图像

## 与铸铁性能的关联

石墨面积率与铸铁力学性能、物理性能之间存在明确的关联规律[[S27,S35]]：
- **石墨面积率越高，石墨形态越接近球形**，对基体的割裂作用越小，铸铁的抗拉强度、延伸率、断裂韧性越高；球墨铸铁因球状石墨面积率最高，力学性能为四类铸铁中最佳[[S16,S6]]
- **石墨面积率越低，石墨形态越接近片状**，铸铁的热导率、消振减震性能越优异；灰铸铁因片状石墨面积率最低，消振性为所有铸铁中最优[[S31,S32]]
- 蠕墨铸铁的导热性显著优于球墨铸铁，介于灰铸铁与球墨铸铁之间；抗拉强度、延伸率同样介于二者之间[[S32]]

## 局限性

### 二维修正本质的局限

石墨面积率仅能反映**二维投影下**单颗石墨的形状规整度，无法表征石墨的三维立体形貌特征[[S7]]。标准规定的外接圆计算以石墨最大投影长为直径，无法覆盖部分异形石墨的真实外接轮廓，会引入固有计算偏差[[S7]]。

### 主要误差来源

石墨面积率测量的误差来源可分为三类[[S25,S24]]：

| 误差类别 | 具体来源 |
|---------|---------|
| 仪器误差 | 灰度阈值设置不当、图像采集系统空间分辨率不足、像素尺寸标定偏差、视场边缘切割特征物计数偏差、显微镜光源不均匀造成的阴影误差、电子系统引入的噪声误差、透镜扫描系统几何畸变误差 |
| 制样误差 | 试样残留磨痕、抛光粉嵌入石墨脱落、石墨与基体界面轮廓模糊、试样表面平整度不足 |
| 人为误差 | 阈值人工设置偏差、标定操作失误、放大倍数选择不当、取样代表性不足、统计视场数量不达标 |

其中，金相试样制备不良引入的测量误差（如石墨相与基体界面轮廓模糊甚至石墨相脱落）经常成为图像分析仪测量误差中最重要的来源之一[[S24]]。
