## 概述

砂型铸造模是用于成型砂型的铸造工装[[S51]]，属于铸造模具的一种。在铸造术语标准中存在明确概念边界：砂型是以型砂为主体制成的铸型，当砂型采用砂箱支撑时砂箱属于砂型的组成部分[[S13,S32]]；而“砂型铸造模”指成型砂型的工装，禁止将砂质铸型直接称为“砂模”，也禁止将用于成型砂型的铸造工装直接称为“砂型”，二者指代对象不同[[S13]]。

砂型铸造是当前应用最广泛的铸造基础工艺，国内砂型铸造产出的铸件占铸件总质量的90%以上[[S32,S39]]，全球范围内占比约60%～80%[[S52]]。

## 分类

砂型铸造模可按**型砂粘结剂类型**和**造型方式**两个维度进行分类[[S21]]。

### 按粘结剂分类

| 大类 | 代表子类 | 主要特点 |
|---|---|---|
| 粘土砂型 | 湿砂型、干砂型、表面干砂型 | 原料易得、成本低、旧砂回用性好，应用占比最高 [[S22]] |
| 水玻璃砂型 | CO₂硬化型、有机酯自硬型 | 发气量低、不含N/S/P等有害元素，主要用于铸钢件；溃散性差、落砂清理困难 [[S7]] |
| 树脂砂型 | 热芯盒、冷芯盒、室温自硬型 | 强度高、透气性好、尺寸精度高，适用于薄壁复杂铸件；700℃以上树脂烧失，退让性好 [[S7,S25]] |

湿砂型（潮模）以硅砂+粘土+水为主要原料，浇注前无需烘干，生产周期短、成本低，适用于中小型普通铸件，但含水量高，浇注过程中易产生气孔、砂眼类缺陷[[S38,S27,S29]]。干砂型浇注前完成烘干工序以除去多余水分，砂型强度和透气性显著提升，发气量大幅降低，可用于质量要求高、中大型黑色金属铸件生产，但需要烘干设备、生产周期长、铸件尺寸精度低于自硬砂型[[S38,S12]]。

### 按造型方式分类

- **手工造型**：全部由人工完成填砂、紧实操作，工装投入低、工艺灵活性极强，适用于单件、小批量、大型特殊定制铸件的生产[[S21]]。
- **机器造型**：由专用造型设备自动完成紧砂操作，可配套自动化流水线生产，砂体紧实均匀、铸件精度一致性高，适用于大批量定型铸件量产；高压造型工艺是当前主流的机器造型工艺[[S21]]。

## 结构与组成

典型砂型铸造模的结构包括以下核心组成部分：

**砂箱**：分为上砂箱（上型）和下砂箱（下型），是容纳和支撑砂型的刚性框，起到固定砂体结构、方便转运合箱的作用[[S46,S24]]。

**型腔**：砂型中与铸件外形对应、浇注后成型铸件主体的空腔部分，不包含由芯头形成的辅助定位空腔[[S24]]。

**浇注系统**：由浇口杯、直浇道、横浇道、内浇道四个核心部分组成，作用是平稳引导液态金属流入型腔，阻挡熔渣、砂粒杂质进入型内，调节铸件不同部位的温度分布、控制凝固顺序[[S53,S24]]。

**冒口**：砂型中专门设置的补缩结构，用于存储额外的液态金属，补充铸件冷凝收缩所需的金属液，避免铸件内部产生缩孔、缩松缺陷[[S1,S53]]。

**通气孔**：开设在砂型上部的排气通道，用于快速排出浇注过程中砂型受热析出的气体，避免气体滞留型腔内导致铸件产生气孔缺陷[[S19]]。

**砂芯**：由芯砂预制的砂体，用于成型铸件的内部空腔、侧凹等无法由外部砂型直接成型的结构，配套的芯头用于在砂型内定位支撑砂芯、导出砂芯内部产生的气体[[S19]]。

## 工艺作用

砂型铸造模在工艺流程中承担以下核心功能[[S17,S49]]：

1. **支撑型腔**：作为一次性铸型，在金属液充型阶段支撑型腔结构、承接注入的高温熔融金属液。
2. **排气通道**：砂型的多孔透气结构可将金属液受热后砂料、粘结剂分解产生的大量蒸汽、氢气、二氧化碳等气体及时排出型腔，避免铸件产生气孔缺陷。
3. **控制冷却**：砂型整体导热系数较低，相比金属型可延缓铸件冷却速度，避免铸件晶粒过于细化，获得更均匀的凝固组织；砂型内部还可通过设置冷铁等局部激冷构件调控铸件局部冷却速率，实现顺序凝固、补偿铸件收缩。

| 强度类型 | 数值范围 |

| 粘土砂湿压强度 | 30～160 kPa |

### 耐火度

砂型耐火度需匹配铸造合金类型[[S17,S16]]：

| 铸造合金 | 适配砂型耐火度/纯度要求 |
|---|---|
| 铝合金铸件 | ≥650℃ |
| 普通铸铁件 | 硅砂纯度94%～98%，耐受约1300～1400℃ |
| 铸钢件 | 硅砂纯度≥98%，耐受≥1500℃ |
| 特种工况（刚玉砂） | 耐火度高于1700℃ |
| 特种工况（铬铁矿砂） | 耐火度高于1790℃ |

### 溃散性

溃散性定义为浇注后型砂容易解体脱离铸件表面的性能，行业内通过浇注过程一次或多次加热－冷却循环后测定的残留强度指标进行衡量，残留强度越低则溃散性越好[[S48]]。

## 应用领域

砂型铸造模的应用范围极为广泛[[S23,S52,S36,S30]]：

- **合金适配性**：可适配铸钢、铸铁、铝/镁/铜等绝大多数有色合金，砂型铸造是唯一对各种合金几乎无限制的铸造工艺。
- **铸件重量与尺寸**：覆盖从几克到数百吨量级，尺寸跨度从几毫米到数十米，复杂带内腔铸件可通过内置砂芯完成生产。
- **生产批量**：可支持单件试制、小批量手工生产到大批量自动化造型的全场景需求。
- **典型产品**：机床床身、大型水电机组转轮、内燃机缸体缸盖、铁路车辆铸件、各类通用机械壳体构件等。

## 局限性

砂型铸造模的典型工艺局限性包括[[S49,S40,S26]]：

1. **精度短板**：普通砂型铸件尺寸公差最高仅能到CT7～CT11级，相较于金属型、压铸等工艺精度低，铸件加工余量大。
2. **表面质量**：砂型铸造是所有主流铸造工艺中表面粗糙度较差的类型，普通砂型铸件表面带有明显砂粒纹理，部分铸件表面易产生粘砂、砂眼、夹砂等缺陷。
3. **生产效率**：砂型为一次性铸型，每生产一件铸件必须重新制备砂型，相比可重复使用的金属铸型单位人力投入下的产出效率较低，属于相对劳动密集型的铸造工艺。

## 与压铸模的对比

在压铸技术百科语境下，砂型铸造模与压铸模构成铸造模具的两个重要类型，二者存在根本性差异。

| 对比维度 | 砂型铸造模 | 压铸模 |
|---|---|---|
| **模具主体材料** | 基础硅砂+粘结剂（膨润土/树脂/水玻璃等）+功能性添加剂+分型剂 [[S5,S28]] | 热作模具钢（如H13、8407等高温合金模具钢），低熔点合金可采用普通碳钢 [[S5,S28]] |
| **模具寿命** | 一次性使用，每件铸件需重新制备砂型 [[S11,S34]] | 铝合金压铸模可达数十万至百万模次；铜合金压铸模10,000～50,000模次 [[S18,S3]] |
| **铸件尺寸精度** | IT12～IT16级（GB/T 1800系列），常规公差约±1/16英寸/英寸 [[S4,S43]] | IT11～IT14级，最优可达IT9级，公差可达±0.001英寸/英寸 [[S18,S35]] |
| **表面粗糙度** | Ra 6.3～50 μm [[S6,S18]] | Ra 0.2～3.2 μm，最优可低至0.2 μm [[S4,S18]] |
| **工作温度** | 无强制预热要求，常规室温至300℃以内 [[S9]] | 常规预热及工作温度150～350℃，铜合金工况约550℃ [[S37]] |
| **充型压力** | 重力充型，接近常压 [[S3]] | 热室压铸0.5～50 MPa，冷室压铸30～150 MPa [[S43]] |
| **冷却速率** | 低，砂质型壳热传导系数低，凝固缓慢 [[S15,S9]] | 高，金属模具大多内置水冷通道，铸件晶粒更致密 [[S9]] |
| **适用合金** | 几乎所有常用铸造合金，覆盖铸铁、铸钢及全部有色合金 [[S36,S31]] | 常规适配熔点低于700℃的合金（锌、铝、镁、铜等），钢铁类高熔点合金工业应用极少 [[S3,S31]] |
| **生产效率** | 25～600件/小时 [[S43]] | 50～150次/小时，热室机可达300～7,000次/小时 [[S18,S4,S45]] |
| **最大铸件重量** | 无明确上限，可生产单重数十吨级大型铸件 [[S30]] | 铝合金铸件上限约50 kg，锌合金铸件上限约200 kg [[S43]] |
| **模具初始投入** | 极低，仅需简易木模/金属模即可批量制备砂型 [[S35,S31]] | 较高，通常数十万至数百万元，仅在数千件以上大批量生产时单位成本可低于砂型铸造 [[S35,S31,S43]] |

**核心差异总结**：砂型铸造模以廉价砂质材料一次性成型、靠重力充型、适用合金范围广但精度和表面质量较低，适合小批量到大批量的各类铸件；压铸模以高合金模具钢制造、可重复使用、靠高压高速充型、精度和表面质量优异但仅适配低熔点有色合金，适合大批量高精度生产[[S34,S43]]。
