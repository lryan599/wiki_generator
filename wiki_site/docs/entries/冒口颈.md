## 概述

冒口颈，别名补缩颈 [[S35,S19]]，是浇冒口系统中连接冒口与铸件的关键功能部件，英文对应术语为 riser neck 或 feeder neck [[S34,S39]]。其核心作用是作为铸件与冒口之间的补缩通道，在铸件液态收缩阶段保证金属液从冒口顺畅补入铸件，并在补缩完成后及时凝固封闭，阻止铸件内部金属液逆流回冒口，从而避免缩松、缩孔等缺陷 [[S19,S35]]。

冒口颈的设计直接影响补缩效果与铸件内部质量。通用设计要求满足定向凝固原则，即凝固时间顺序为：冒口 > 冒口颈 > 铸件被补缩部位，对应的模数关系为 Mr > Mn > Mc [[S7]]。经典的铸件-冒口颈-冒口模数比为 1.0 : 1.1 : 1.2 [[S39,S9]]。

## 核心设计准则与模数关系

冒口颈模数 Mn 是决定其凝固时间和补缩通道关闭时机的核心参数。根据不同铸造工艺条件和合金类型，模数的选取遵循不同的规则：

| 适用场景 | 模数计算公式/系数 | 说明 | 来源 |
|---------|------------------|------|------|
| 通用铸钢件模数法 | Mn = 1.1 Mc | 扩大系数 fn = 1.1，保证顺序凝固 | [[S7,S38]] |
| 均衡凝固——球铁石墨自补缩 | Mn = (0.5\~0.7) Mc | 冒口颈比例系数 K 取 0.5\~0.7 | [[S6,S26]] |
| 均衡凝固通用原则 | Mn = (0.8\~1.0) Mc | 保证定向凝固，凝固时间不短于铸件 | [[S35]] |
| 减压冒口/控制压力冒口 | Mn = 0.67 Mr | 短冒口颈，冒口模数取 Mr | [[S10,S32,S45]] |
| 控制压力冒口（按铸件模数） | Mn = (0.35\~0.6) MT | 冶金质量好时取小系数，MT 为输送模数 | [[S46]] |
| 铸铁直接实用冒口理论 | Mn = Ms × (tp−1150)÷(tp−1150+L/c) | 设计目标：液态收缩结束时冒口颈刚好冻结 | [[S12,S15,S30,S45]] |
| 球铁侧冒口推荐 | Mn = (0.95\~1.0) Mc | 保证充足补缩时间 | [[S26,S37]] |

基于铸件-冒口-冒口颈模数比 1.0:1.1:1.2 的基准配比，由于相邻铸件与冒口的热传导效应，实际冒口颈的凝固时间会显著长于其几何模数独立计算得到的理论凝固时间 [[S20,S39]]。采用短、薄、宽结构的冒口颈，其凝固时间可延长为几何模数对应理论凝固时间的 2\~4 倍，既保障充分补缩，又避免生成接触热节 [[S14,S21]]。

## 关键几何参数

### 冒口颈厚度（截面尺寸）

冒口颈截面厚度直接控制补缩通道的关闭时机。常规砂型铸造中，冒口颈厚度推荐取铸件被补缩部位厚度的 0.4\~0.6 倍，该取值可在保证补缩通流能力的前提下避免过早凝固或延迟关闭 [[S35]]。

### 冒口颈长度

冒口颈长度决定热流传导路径。过长会增大散热面积，加速冒口颈冷却而提前关闭补缩通道；过短则受铸件和冒口两侧热干扰严重，会显著延迟冒口颈凝固时间，导致石墨化膨胀阶段发生铁液回流失效 [[S14]]。通用经验取值为 2.4 倍铸件被补缩部位模数 Mc [[S38]]。侧冒口或短颈冒口的最大长度限制通常为冒口直径 D 的 1/3 [[S41]]。暗冒口场景下，冒口颈长度宜控制在 0.1 倍冒口直径以内 [[S39]]。

### 通流截面积

冒口颈靠近铸件一侧的截面积通常取冒口底部截面积的 0.3\~0.4 倍，靠近冒口一侧的截面积比前者大 10%\~15%，截面积过小易导致补缩通道过早凝固阻断 [[S8]]。

## 截面形状与模数计算

冒口颈的截面形状直接影响其模数和补缩特性。以下是主要截面形状及其计算方法和特点：

### 矩形截面

矩形冒口颈在砂型铸造、熔模铸造中应用广泛，截面长宽比 a:b 通常取 1:2 [[S6]]。等效模数计算公式为：

> Mn = (a × b) / (2(a + b))

其中 a、b 为矩形截面两边长，公式适用于 a ≤ b < 5a 的情况 [[S31]]。

### 圆形截面

圆形冒口颈的等效模数计算公式为：

> Mn = d / 4

其中 d 为圆形截面直径 [[S38]]。在集渣包兼作冒口的多铸件布置中，圆形冒口颈直径 d 推荐取铸件热节圆直径 T 的 1/3\~1/2，长度 f 取对应冒口直径 Dm 的 1/3\~1/4 [[S11]]。

### 梯形/楔形截面

梯形截面冒口颈可近似换算为等效矩形截面计算模数，其典型扁梯形结构接触面高度 3\~4 mm、底边宽度 5\~7 mm，具备自动脱落特性，铸件冷却后冒口可在外力作用下直接断裂，减少打磨清理工作量 20%\~30% [[S35]]。

## 分类

### 按位置分类

- **顶冒口颈**：配套上部设置的明/暗顶冒口，直接在铸件顶部开设补缩通道，适配铸件最高处或顶层热节的补缩需求，部分结构可集成易割片降低后续清理工作量 [[S24,S28]]。
- **边冒口颈**：适配侧置的边冒口，作为铸件侧向的补缩通道，通常设计为长条形结构，可在铸件侧面热节位置实现定向补缩，多用于无法在顶面布置冒口的复杂铸件 [[S37]]。

### 按形状与结构分类

- **矩形冒口颈**：最常见的形式，截面长宽比 a:b 通常取 1:2，模数计算简便，适用于各类常规铸造场景 [[S31,S6]]。
- **梯形/楔形冒口颈**：采用扁梯形截面（高度 3\~4 mm、底边 5\~7 mm），具有自动脱落特性，减少了后续切割打磨工作量 20%\~30%，适配球墨铸铁等高石墨化膨胀合金 [[S35,S21]]。
- **圆形冒口颈**：适用于集渣包兼作冒口或内浇道兼作冒口颈的场景，直径 d 通常取热节圆直径 T 的 1/3\~1/2 [[S11]]。
- **易割片式冒口颈**：采用薄片式易割砂芯结构，砂芯推荐厚度为 0.1 倍冒口直径 D，中心补缩通孔直径为 0.4D，铸件成型后敲击即可断裂，大幅降低切割工作量 [[S39]]。
- **斜向冒口颈**：轴线夹角推荐取 10°\~30°，可减小接触热区面积，避免接触部位形成滞留热节 [[S35]]。
- **双缩颈冒口颈**：单顶冒口通过两个独立短冒口颈分别对接铸件两个独立热节，适用于多热节共用单冒口的场景 [[S28,S40]]。

## 缺陷与失效机理

冒口颈参数设计不当会导致多种铸造缺陷，以下为主要失效模式及其机理：

### 冒口颈尺寸过大

冒口颈尺寸偏大时无法在铸件液态收缩完成后及时闭合，导致冒口颈部位及相连铸件内部产生连通的缩松缩孔缺陷 [[S19]]。典型案例如 QT450-10 轮毂铸件因原冒口颈尺寸偏大导致冒口颈缩松深入铸件内部，采用模数周界商法重新设计缩小冒口颈尺寸后缺陷消除 [[S17]]。

### 冒口颈尺寸过小

冒口颈厚度偏小时会提前凝固封闭，阻断补缩通道，导致铸件补缩不足而产生缩松缺陷。硅钼球墨铸铁排气接管铸件中厚度仅 8 mm 的过薄冒口颈，除造成法兰部位缩松外，还因铁液流速过高而冲蚀周围型砂，引发砂眼缺陷 [[S27]]。

### 接触热节缩孔

当冒口直接正对铸件热节布置时，冒口颈形成的接触热节会扩大原有热节尺寸，即使冒口颈模数远小于铸件被补缩部位模数，冒口颈区域仍可能最后凝固而产生集中缩孔。QT700-5 高牌号球墨支架铸件原工艺冒口正对热节，冒口颈区域缩孔缺陷发生率超过 80%，调整冒口位置避开热节后缺陷完全消除 [[S5,S22]]。

### 铁液回流缺陷

当冒口颈凝固时间晚于铸件共晶石墨化膨胀启动时间时，铸件内部石墨化膨胀产生的高压铁液会反向流入冒口，导致铸件内部补缩压力不足，引发内部缩松缺陷 [[S24,S37]]。

## 设计优化与工艺适配

### 短、薄、宽结构优化

均衡凝固理论推荐采用短、薄、宽的冒口颈形式（含耳冒口、飞边冒口、压边冒口、热侧冒口等）。依托浇注流动过程的更新作用和流动热效应，这类结构可将冒口颈凝固时间延长为其几何模数对应凝固时间的 2\~4 倍，既充分补缩，又避免接触热节处产生缩松缺陷 [[S14,S21,S40]]。铜合金铸件采用此类结构可将工艺出品率提升至 65.7% [[S35]]。

### 不同材质适配

不同合金材质对冒口颈尺寸有显著不同的敏感性：

| 材质 | 推荐冒口颈参数 | 特点与说明 | 来源 |
|------|--------------|-----------|------|
| 铸钢件 | Mn = 1.1 Mc；颈直径可缩至 0.25D，长度 ≤ 0.03D | 须满足顺序凝固，导热系数较低，颈长受限严格 | [[S38,S39,S36]] |
| 普通球墨铸铁 | Mn = (0.95\~1.0) Mc | 侧冒口保证充足补缩时间 | [[S26,S37]] |
| 球铁——石墨自补缩 | Mn = (0.5\~0.7) Mc | 利用石墨化膨胀实现自补缩 | [[S6]] |
| 球铁——控制压力冒口 | Mn = 0.67 Mr | 短冒口颈，平衡内压 | [[S10,S32,S45]] |
| 灰铸铁（直接实用冒口） | Mn 按理论公式计算 | 液态收缩结束时刻冒口颈同步关闭 | [[S12,S45]] |
| 铝合金/铜合金 | 颈直径可缩至 0.25D，长度 ≤ 0.06D | 导热系数高，颈长可放宽至钢的两倍 | [[S39,S36]] |

### 补缩梯度要求

为保障补缩通道通畅，不同合金须满足对应的最小轴向温度梯度：普通碳钢铸件朝向冒口的纵向最小温度梯度不低于 0.5 °C/cm；Al-7%Mg 合金为 1\~2 °C/cm；Al-4.5%Cu 合金为 4\~8 °C/cm；高硅铝铸造合金梯度为 1\~2 °C/cm [[S44,S2]]。

### 明冒口与暗冒口的冒口颈适配

- **明冒口**：配套易割片冒口颈时开孔直径推荐为 1.13\~1.45 倍冒口模数；砂型明冒口开孔直径为 1.72\~2.2 倍冒口模数 [[S24]]。
- **暗冒口**：侧冒口配套的冒口颈可借助两侧铸件与冒口的蓄热效应，较理论计算值进一步缩小尺寸，长度宜控制在 0.1 倍冒口直径以内 [[S39]]。
