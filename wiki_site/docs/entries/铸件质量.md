## 概述

铸件质量是指铸件的形状、尺寸、成分、组织、性能等对使用要求的符合程度[[S3,S49]]。它综合反映了铸件满足设计图纸、技术协议及使用工况的能力，是铸造生产全流程管理的核心目标。依据GB/T 5611-2017《铸造术语》的界定，铸件质量涵盖**外观质量**、**内在质量**和**使用质量**三个维度[[S3,S49]]。

## 质量范畴

### 外观质量

铸件外观质量是指铸件表面状况和达到用户要求的程度[[S3,S64]]。其检验内容一般通过目视观察、量具测量等非破坏性方式完成[[S39]]，具体包含：

- 表面粗糙度
- 表面缺陷
- 尺寸公差
- 形状偏差
- 重量偏差

### 内在质量

铸件内在质量是指一般不能用肉眼检查出来的铸件内部状况和达到用户要求的程度[[S3,S49]]。其检测通常依赖化学分析、材料试验、金相检查、无损探伤等方法[[S14]]，具体涵盖：

- 化学成分
- 物理和力学性能
- 金相组织
- 内部孔洞、裂纹、夹杂物等缺陷

### 使用质量

铸件使用质量是指铸件能满足各类使用要求的性能[[S3,S64]]，如在强力、高速、磨耗、腐蚀、高温等不同条件下的工作性能，以及切削性能、焊接性能、运转性能等[[S3,S14]]。

## 质量分级与评价指标

### 质量分等

依据JB/JQ 82001-90《铸件质量分等通则》，在所有质量项合格的基础上，成品铸件可分为三个等级[[S17,S70]]：

| 质量等级 | 要求 |
|---------|------|
| 合格品 | 外观质量、内在质量符合现行国标/行标；生产过程质量稳定；能满足使用性能 |
| 一等品 | 达到工业发达国家70年代末、80年代初同类水平，国内领先；生产过程质量稳定 |
| 优等品 | 达到国际同类铸件的当代先进水平；生产过程质量很稳定；在国际市场上有竞争力 |

### 尺寸公差

依据GB/T 6414-1999（等效ISO8062:1994）《铸件 尺寸公差与机械加工余量》，铸件尺寸公差从精到粗分为16个等级（CT1~CT16），其中CT3~CT13公差比例系数取√2，CT13~CT16取∛2[[S65,S61]]。壁厚尺寸公差可比普通尺寸降一级，例如普通尺寸公差为CT10时，对应壁厚公差为CT11[[S32]]。

### 表面粗糙度

我国现行铸造表面粗糙度评价体系以GB/T 6060.1《表面粗糙度比较样块 铸造表面》（等效ISO 2632/Ⅲ）和GB/T 15056—1994《铸造表面粗糙度评定方法》为核心依据[[S59,S76]]。常规评定采用视觉+触觉方式与同材质同工艺的标准样块进行比对，存在争议时采用轮廓仪仲裁检测[[S31,S76]]。

### 力学性能指标

铸件常规力学性能评价指标包含六大核心项[[S8,S60]]：

| 指标 | 说明 |
|------|------|
| 抗拉强度 | 材料断裂前能达到的最大应力 |
| 屈服强度 | 材料开始产生永久变形时的应力 |
| 延伸率 | 塑性变形能力 |
| 断面收缩率 | 塑性指标 |
| 冲击韧性 | 冲击破坏时单位面积消耗的功 |
| 硬度 | 常用布氏硬度（HB）或洛氏硬度（HR） |

不同材质铸件的指标要求对应对应材质的专属国家标准执行。一般而言，铸件本体取样测得的力学性能普遍低于单铸试棒的测试结果[[S3]]。

## 影响铸件质量的主要工艺因素

影响压铸件质量的核心因素可归纳为五大类[[S7,S74]]：

- **压铸机性能稳定性**：类型和技术参数应符合生产要求
- **压铸工艺参数选取**：包括压力、速度、时间、温度等
- **合金成分与熔炼处理**：熔融合金在保温炉内静置时间不宜超过4小时，超时须重新精炼[[S7]]
- **模具结构合理性与制造质量**：成形零件的精度、表面质量和强度/刚度，以及浇注系统、排溢系统设计
- **压铸件自身结构设计合理性**：壁厚、弯角位、拔模斜度等

### 浇注系统

浇注系统是引导金属液进入型腔的通道，对金属液的流动方向、溢流排气条件、压力传递、充填速度起重要控制与调节作用[[S38]]。浇注系统设计不合理会直接引发欠铸、冷隔、紊流裹气等缺陷[[S38,S7]]。

### 排溢系统

排溢系统（溢流槽）的核心作用是容纳最先进入型腔的冷污金属液，带出混入其中的气体与氧化夹杂，是预防铸件产生冷隔、气孔、夹渣类缺陷的关键结构。溢流槽设计位置不当、体积偏小将无法有效排出冷污金属[[S38]]。

### 压室充满度

压室充满度即浇入压室的金属液质量占压室完全充满时总金属液质量的百分比。常规压室充满度最佳区间为**70%~80%**。充满度过小将导致压室上部空间过大，金属液大量包卷气体，气孔率显著升高，同时金属液在压室内提前激冷降低充型能力[[S74]]。

## 典型铸件缺陷与质量的关系

### 缺陷分类体系

国际铸造技术协会（WFO/ICFTA）发布的通用《国际铸件缺陷图谱》将所有铸造缺陷基于物理外观特征归为7大类[[S69,S12]]：

| 代码 | 类别 |
|------|------|
| A | 多肉类 |
| B | 孔洞类 |
| C | 裂纹、冷隔类 |
| D | 表面缺陷类 |
| E | 残缺类 |
| F | 尺寸或形状差错类 |
| G | 夹杂物和金相组织不合格类 |

我国现行GB/T 41972-2022《铸铁件铸造缺陷分类及命名》修改采用ISO/TR 16078:2013，采用类-组-子组-具体缺陷四级分层分类方法，与GB/T 5611-2017《铸造术语》中相关缺陷术语完全协调统一[[S75]]。

### 主要缺陷类型

**气孔**：分布在铸件内部，形状多为圆形或椭圆形，孔洞内壁光滑，呈暗灰色。形成机制为充型过程中金属液紊流包卷气体或金属液自身含气量过高。气孔显著降低铸件的韧性、气密性，是削弱疲劳性能和抗断裂性能的典型有害缺陷[[S7,S68,S11]]。

**缩孔缩松**：孔洞形状不规则、内壁粗糙、表面呈暗色。形成机制为铸件冷凝过程中内部补缩压力不足。合金凝固温度范围小更易形成集中的缩孔，凝固温度范围大更易形成分散的缩松[[S7,S73]]。缩孔缩松会破坏内部结构连续性，降低力学承载能力，易在缺陷位置引发应力集中[[S11]]。

**冷隔**：呈不规则线形缝隙，分为穿透型和非穿透型，在外力作用下存在持续扩展趋势。主要诱因是浇注温度、模具温度过低，金属液充型流动性不足[[S7,S68]]。

**裂纹**：分为热裂纹和冷裂纹。热裂纹形成于准固相凝固阶段，沿晶撕裂，断口因氧化呈暗灰色无金属光泽；冷裂纹形成于弹性状态的较低温度阶段，为穿晶断裂，断口带有金属光泽。裂纹极大损害铸件整体强度，属于高安全风险缺陷[[S11,S73]]。

**夹渣**：熔炼过程中除渣不彻底、金属液运输/浇注过程中混入氧化皮、炉渣、炉衬碎屑等外来杂物，残留在铸件内部或表面。夹渣属于典型的应力集中源，大幅降低动态力学性能，同时增加机械加工的刀具磨损与崩刀风险[[S68,S22]]。

### 缺陷-影响因素关联

常用压铸缺陷与各工艺因素的对应关联关系如下[[S46]]：

| 缺陷类型 | 主要关联因素 |
|---------|-------------|
| 欠铸 | 合金杂质含量、浇注/排溢系统设计、温度参数 |
| 变形 | 模具温度不均、脱模力不平衡 |
| 孔穴（气孔） | 压室充满度不足、充型速度过高 |
| 疏松（缩松） | 浇注温度、模具温度、保压压力 |
| 裂纹 | 合金成分、模具温度梯度 |
| 冷隔 | 浇注温度过低、模具温度过低 |
| 粘模 | 模具温度过高 |
| 飞边 | 锁模力不足、模具精度 |

## 质量检测与控制

### 检验方式

铸件质量检验分为全检和抽检两种模式[[S39]]：

| 方式 | 适用场景 |
|------|---------|
| 全检 | 单件、小批量生产、试生产或特殊场合重要铸件的关键质量检验项目 |
| 抽检 | 生产工艺稳定成熟前提下的成批、大量生产铸件的质量判定 |

### 无损检测方法

现有铸件无损检测技术可分为五大类[[S30,S42]]：

| 方法类别 | 具体技术 | 检测深度 | 适用材质限制 |
|---------|---------|---------|-------------|
| 光学检查法 | 目测、渗透探伤 | 表面/表层 | 无 |
| 放射线检查法 | X射线透射、工业CT | 深部/三维 | 无 |
| 电磁检查法 | 磁粉探伤、涡流探伤 | 表面/近表面 | 磁粉探伤仅适用于铁磁性材料 |
| 声波检查法 | 超声波探伤、声发射法 | 深部 | 无 |
| 热传导检查法 | 表面温度分布图像显示 | 表面 | 无 |

**荧光渗透检测**可捕捉铸件表面微米级的开口缺陷（如微裂纹、针孔），灵敏度高于着色渗透检测，尤其适配非铁磁性材料（铝、铜合金）以及复杂几何表面铸件[[S23,S16]]。

**X射线无损检测**可直观呈现铸件内部气孔、缩孔、夹渣、疏松、冷隔、裂纹等缺陷的形态、尺寸与分布，影像可长期保存用于复验分析，已实现自动化缺陷识别[[S23,S20]]。

**工业射线CT检测**可对铸件内部三维缺陷进行定量分析，适用高精度关键铸件的质量核验[[S72]]。

### 金相分析

金相分析的常规试样制备流程为[[S23,S72]]：
线切割取样 → 500~1200目金相砂纸分级粗磨精磨（每更换砂纸打磨方向旋转90°消除前道划痕）→ 机械抛光至镜面光洁度 → 对应腐蚀液刻蚀（常用盐酸-氯化铁混合腐蚀液，部分材质腐蚀时间3~5s）→ 乙醇清洗吹干 → 显微组织观测。

### 过程监控

铸造过程可通过部署温度、压力、位移等高频采样传感器，采集压铸/低压铸造全周期的工艺数据，建立铸造过程参数与最终铸件质量的关联模型，实现量产过程的缺陷提前预判[[S66]]。

## 提升铸件质量的措施

### 浇道设计优化

环形梯形浇道设计采用截面积逐渐缩小的结构，可有效减缓金属液流动速度，抑制飞溅、卷气和夹杂缺陷，保障充型过程平稳，尤其适配精密铸造和高力学性能要求的铸件[[S71,S9]]。

双列环形对称浇道设计的分流平衡效应可使铸件充型平稳性提升**23%**，缩松缺陷体积降低**56%**，还可将缺陷定向迁移至浇注系统内，实现铸件本体缺陷完全消除，复杂薄壁不锈钢铸件的小批量试制成品率可达**98%**[[S15,S26]]。

### 冷却系统控制

在铸件热节部位增设水冷冷却系统进行冷却控制，可优化凝固温度梯度、缩短凝固时间，使铸件二次枝晶臂间距最大值降低**13.35%**，在细化晶粒提升力学性能的同时基本消除铸件内缩松缩孔缺陷[[S10]]。点冷冷却方案相比传统线性冷却方案，可使压铸模具高温区域冷却效率更高，冷却效果更均匀[[S27]]。

### 数值模拟辅助优化

主流商用铸造数值模拟软件特性[[S54,S62,S57,S21]]：

| 软件 | 技术框架 | 主要功能 |
|------|---------|---------|
| ProCAST | 有限元法 | 充型/温度/流动/微观组织演化多模块，适配高压铸造、离心铸造等 |
| MAGMASOFT | 有限差分法 | 热过程分析、充型分析、结晶分析、缺陷预测 |
| AutoCAST | 几何推理+快速仿真 | 自动浇道/冒口设计、近实时凝固仿真，适配中小铸造企业 |
| AnyCasting | 有限差分法快速网格 | 每5秒生成2000万网格，高级网格质量检测 |

采用铸造数值模拟辅助工艺优化，可在实际投产前完成浇道结构、冷却系统、浇注参数的迭代验证，大幅减少试制次数，降低材料能耗成本[[S33,S50]]。
