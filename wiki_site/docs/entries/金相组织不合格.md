## 定义与分类

“金相组织不合格”在国家标准GB/T 5611-2017《铸造术语》中被明确定义为“铸件的金相组织不符合技术条件的规定”，英文对应名称为 **metallograph abnormal** [[S48,S57]]。该缺陷在国际铸造缺陷分类体系中归属于G类（夹杂物和金相组织不合格类）缺陷范畴 [[S60]]。

从检验尺度上，金相组织不合格可分为两个层次 [[S48]]：
- **宏观级组织异常**：肉眼或低倍放大镜可识别，特征尺寸通常≥100 μm，如低倍针孔度超标、宏观偏析带等；
- **微观级组织异常**：需借助光学显微镜观测，特征尺寸处于微米级，如共晶硅形态不合格、过烧组织、针状铁相等。

金相组织不合格与宏观尺寸不合格（如尺寸超差、缺肉、变形等）的本质区别在于：前者反映的是材料内部微观结构偏离技术条件，直接影响铸件的力学性能、耐压性、气密性及耐腐蚀性等内在质量，而非铸件的外形几何精度 [[S11]]。

## 典型不合格组织形态

压铸件常见的金相组织不合格类型及其微观形貌特征归纳如下表。

| 不合格类型 | 典型微观特征 | 来源 |
|---|---|---|
| 粗大晶粒 | α-Al枝晶尺寸达100~300 μm，晶粒未在压铸压力下充分破碎，枝晶形态粗大 | [[S66,S17]] |
| 过烧组织 | 出现过烧三角、晶界熔化、复熔共晶球，枝晶间低熔点相熔化，组织性能显著恶化且无法挽救 | [[S57,S22]] |
| 共晶硅变质不良 | 未变质时共晶硅呈长针状/粗大片状；变质不足时分布不均匀；过变质时出现过变质带、块状初晶硅残留，共晶硅未实现粒状/短杆状的合格形态 | [[S25,S49,S6]] |
| 针状β-AlFeSi铁相 | 呈黑色或亮白色长针状形态，直接割裂α-Al基体，属Fe含量过高且未通过Mn、Cr等元素转化为汉字状/多边形状无害相的异常组织 | [[S25,S16]] |
| 微观孔洞/缩松 | 视场内分布圆形或短棒形微小孔隙，部分区域形成连续连通的疏松带，成为应力集中的裂纹萌生源 | [[S53,S40]] |
| 组织不均匀/偏析 | 共晶组织局部富集，形成微观偏析带，宽度约200~500 μm，沿铸件轮廓线分布 | [[S34,S37]] |
| 氧化夹杂/脱模剂残留 | 黑色块状氧化物紧贴晶界分布，或脱模剂残留物进入铸件内部形成显微非金属夹杂相 | [[S37,S61]] |
| 冷隔 | 两股提前部分凝固的金属流前沿未完全熔合，金相下表现为沿熔合线分布的氧化膜层及元素富集异常带 | [[S55,S35]] |

## 判据与标准

压铸件金相组织合格与否的判定主要依据以下标准体系：

**1. 铸造铝硅合金钠变质等级判定（JB/T 7946.1）**

该标准将钠变质效果分为6级，试样经0.5% HF水溶液浸蚀5~10 s后在200倍显微镜下观测判定 [[S28,S24]]。其中：
- **合格范畴**：变质正常（共晶硅呈细小粒状或短杆状，分布均匀）
- **不合格范畴**：未变质、变质不足、变质衰退、轻度过变质、严重过变质

| 级别 | 共晶硅形态特征 | 判定 |
|---|---|---|
| 未变质 | 长针状、粗大片状 | 不合格 |
| 变质不足 | 部分条片状未完全破断 | 不合格 |
| 变质正常 | 粒状、短杆状，均匀分布 | 合格 |
| 变质衰退 | 共晶硅重新出现粗化倾向 | 不合格 |
| 轻度过变质 | 出现过变质带 | 不合格 |
| 严重过变质 | 大面积过变质带、块状初晶硅 | 不合格 |

**2. 铸造铝硅合金过烧组织等级判定（JB/T 7946.2）**

过烧组织分为5级：正常、过热、轻微过烧、过烧、严重过烧。判定依据共晶硅形态（圆滑→聚集长大→多边化）、是否出现过烧三角、复熔共晶球等特征 [[S22,S46,S52]]。其中过热及以上级别均判定为不合格 [[S22]]。

**3. 铸造铝合金针孔度评级（JB/T 7946.3）**

针孔度共分5级，试样取自铸件凝固较慢的厚大部位，经10%~15% NaOH水溶液在25±5℃下浸蚀6~12 min后目视或低倍观测。超过供需双方商定允许级别的针孔度判定为组织不合格 [[S7]]。

**4. 国际标准与通用规范**

ASTM E3规定了金相试样制备的通用规范，ASTM E407规定了金属及合金的显微浸蚀剂选择与操作指南，构成压铸金相组织合格判定的统一制样与观测基础 [[S5,S44]]。NADCA 207标准规定了压铸工具钢H13的热处理显微组织验收要求，不合格组织表现为碳化物聚集、分布不均 [[S27,S3]]。

## 产生原因

金相组织不合格的形成涉及从合金熔炼到凝固成形的全过程，主要成因按工艺维度归纳如下：

| 成因维度 | 具体机理 | 来源 |
|---|---|---|
| 合金成分控制不当 | Fe等有害元素超标生成粗大硬脆金属间化合物相；熔体搅拌不均匀引发局部共晶相偏析；微量元素添加量偏离配方要求致初晶相形态异常 | [[S8,S37]] |
| 熔化与浇注温度 | 浇注温度过高→晶粒异常长大、形成粗大枝晶；浇注温度过低→生成未熔相、冷隔类异常组织 | [[S45,S51]] |
| 冷却速度不适配 | 冷却过慢→晶粒不受抑制生长致组织粗大；冷却过快→内应力过高引发微裂纹、析出相分布不均匀 | [[S59,S34]] |
| 模具温度偏离最优区间 | 以ADC12铝合金超低速压铸为例，模具温度150℃时可获得细小枝晶+人字形态共晶Si的合格组织；偏离该区间（如100℃或200℃）则引发晶粒粗大、共晶相形态异常 | [[S41,S38]] |
| 热处理工艺参数失控 | 保温时间/温度不符合要求时，生成异常粗大析出相、过烧组织、残留过量未溶脆性相 | [[S10,S29]] |
| 脱模剂残留 | 充型阶段残留脱模剂被金属液流冲刷进入铸件内部，形成晶界或基体中的显微非金属夹杂相 | [[S61,S55]] |
| 金属液气体卷入 | 金属液以湍流状态高速充型时，型腔中气体被卷入熔体内部，形成直径20~100 μm的显微气孔缺陷 | [[S4,S35]] |
| 增压补缩不足 | 增压不及时或保压时间不足，导致厚大截面最后凝固区域补缩不充分，产生缩孔/缩松 | [[S53,S37]] |

## 对性能的影响

金相组织不合格对压铸件各项性能产生显著恶化作用：

**力学性能**：高压压铸AlSi9Cu3(Fe)合金中，当断裂面缺陷面积占比提升至约23%时，合金真实抗拉强度从290 MPa线性下降至160 MPa，真实断裂延伸率从约2.1%骤降至0.3%；而弹性模量、初始屈服应力受缺陷影响幅度较小 [[S43,S58]]。压铸Mg-Al合金中，硬脆相β-Mg17Al12以连续网状析出分布在晶界时，会导致合金抗拉强度、延伸率显著下降，断口呈解理断裂特征 [[S15]]。

**疲劳寿命**：压铸合金微观组织中存在的氧化薄膜褶皱缺陷附近是显微孔洞的优先形核位置，会大幅降低材料的疲劳寿命，缺陷尺寸越大、距离构件自由表面越近，疲劳寿命越低 [[S4,S42]]。

**耐腐蚀性**：压铸AlSi10MnMg合金中局部共晶组织过度偏析富集的区域，会导致局部电位差增大，耐腐蚀性显著降低，成为优先腐蚀的敏感位置 [[S37]]。

**承压密封性**：微观孔洞/缩松形成的连续疏松带破坏了组织致密性，直接导致铸件气密性和承压能力下降 [[S53]]。

## 与其他缺陷的关系

金相组织不合格在显微尺度上与多种铸造缺陷存在内在关联：

- **氢脆**：压铸合金中溶解的氢在凝固过程中析出形成显微气孔/氢致微孔，氢在应力作用下向拉应力区富集，与显微孔洞耦合共同诱发氢脆断裂，显微尺度下氢原子优先在孔隙内壁偏聚 [[S33,S35]]
- **热裂纹**：铸件中分布的显微夹杂、脆性网状第二相成为应力集中优先位置，在凝固收缩拉应力作用下直接诱发热裂纹形核，是热裂纹起源的常见微观组织诱因 [[S37,S55]]
- **冷隔**：两股提前部分凝固的金属流前沿未完全熔合，在金相尺度下表现为沿熔合线分布的氧化膜层和元素富集异常带，属于界面类金相组织不合格的典型形态 [[S55,S35]]
- **贯穿性气孔**：压铸过程中卷入的大体积气体在铸件壁厚方向连通形成贯穿性气孔，金相截面上表现为多个连通的圆滑孔洞簇，是气孔类微观组织缺陷发展到宏观尺度的极端形态 [[S11,S61]]

**微观鉴别要点**：缩孔缺陷的金相特征为边缘曲折不平、内腔呈暗哑无金属光泽状态；气孔缺陷的金相特征为边缘圆滑规则、内腔呈光亮金属反射状态，二者可通过显微观察明确区分 [[S11,S2]]。

## 检测方法

### 取样位置

金相检验取样应选取铸件受力最大的区域、厚大截面热节区域、CAE仿真预测的缩松/卷气高风险区域，取样需避开边口受激冷严重的表层组织 [[S54,S2]]。

### 试样制备流程

压铸件金相试样标准制备流程为：线切割在目标取样区域切取小块试样→镶嵌固定→依次使用400目、1200目金相砂纸研磨至无明显划痕→帆布粗抛光+绒布细抛光得到镜面→化学腐蚀后酒精清洗吹干，即可上机观察 [[S65,S20,S50]]。

### 腐蚀剂推荐

铝合金压铸件金相检验常用腐蚀剂：

| 腐蚀剂 | 配比 | 腐蚀时间 | 适用范围 |
|---|---|---|---|
| HF水溶液 | 体积分数0.5%~4%氢氟酸水溶液 | 5~20 s | 清晰显现α-Al基体与共晶相边界 [[S63,S47,S30]] |
| Keller试剂 | 10 mL HF + 5 mL HCl + 5 mL HNO₃ + 380 mL H₂O | 8~30 s | 适配2系、6系及铸造铝硅合金等绝大多数铝合金，可清晰显现不同合金相衬度差异 [[S19,S64,S56]] |

### 仪器配置

压铸件金相检验通常使用专业金相显微镜及配套显示电脑进行显微组织观察与图像采集记录 [[S18]]。

## 预防与纠正措施

针对各类金相组织不合格的产生机理，可采取以下预防与纠正措施：

| 不合格类型 | 预防与纠正措施 | 来源 |
|---|---|---|
| 氧化夹杂类 | 铝液精炼阶段通氮气除气；搭配1.5%~2.5%质量分数Al-10Sr杆变质处理，控制Sr含量0.03%~0.04%；采用多层高目数泡沫陶瓷过滤网进行熔体过滤 | [[S1,S37]] |
| 缩孔/缩松类 | 延长压铸增压补压保压时间；在铸件厚大热节区域优化溢流槽与冷却管道布局；采用高真空度压铸工艺降低内部孔洞体积分数 | [[S37,S13]] |
| 共晶偏析/晶粒粗大类 | 优化模具温度分区控制系统，普通铝合金压铸件模具温度控制区间120~280℃，150℃左右可获得最致密的凝固组织；在熔体保温阶段延长电磁搅拌时间确保合金元素均匀混合；必要时配套均匀化固溶+时效热处理消除枝晶偏析 | [[S38,S23,S14]] |
| 过烧类 | 严格控温热处理的加热温度与保温时间，防止因温度过高或时间过久导致低熔点共晶相熔化，过烧组织一旦形成无法挽救 | [[S22,S57]] |

对于已确认金相组织不合格的铸件，需根据不合格类型和严重程度进行处置：变质不足可在熔炼阶段补充变质剂重新处理；固溶不充分可通过再次热处理（确保充分固溶+快速淬火）予以纠正；而过烧组织则属不可逆缺陷，对应铸件应予以判废 [[S25,S57]]。
