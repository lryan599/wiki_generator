## 概述

粘土在压铸领域是一种天然层状硅酸盐无机粘结剂，主要成分为水化硅酸铝（mAl₂O₃·nSiO₂·xH₂O），颗粒尺寸小于2μm，分为高岭土和膨润土两大类[[S18,S19]]。在压铸工艺中，粘土的角色与其在普通砂型铸造中作为型砂核心粘结剂的功能截然不同：压铸场景下粘土绝大多数仅作为压铸模涂料的悬浮剂或辅助粘结剂组分使用，不作为型砂主粘结剂[[S15]]。粘土从未作为压铸生产的核心主功能材料，完全区别于其在高岭土型砂铸造中的主导地位[[S5,S38]]。

## 定义与分类

### 标准定义

国家标准GB/T 5611-2017《铸造术语》将粘土明确定义为：主要成分为二维层状构造水化硅酸铝（mAl₂O₃·nSiO₂·xH₂O）的无机粘结剂，颗粒尺寸小于2μm，分为高岭土和膨润土两大类[[S18]]。

### 分类体系

压铸行业常用粘土分为两大主类，并可进一步细分为若干子类：

| 大类 | 主要矿物 | 子类 | 特征说明 |
|------|----------|------|----------|
| 高岭土（耐火粘土） | 高岭石 | — | 耐火度较高，吸水率与膨胀性较小[[S18]] |
| 膨润土 | 蒙脱石 | 钠基膨润土 | 表面和内部晶层吸附物主要为钠离子，膨胀倍率可达10~20倍，热湿拉强度高[[S18,S3]] |
| | | 钙基膨润土 | 吸附物主要为钙离子，湿强度高但干强度偏低[[S18,S3]] |
| | | 活化膨润土 | 钙基膨润土经离子交换处理后部分钙离子被钠离子取代[[S18]] |
| | | 有机膨润土 | 用季胺盐或季胺碱置换阳离子后获得，可被醇或油类良好润湿[[S18,S1]] |

在国际铸造领域，粘土同样被明确分为kaolin（高岭土）和bentonite（膨润土）两大类别，其中膨润土进一步区分为钠基（western bentonite）和钙基（southern bentonite）[[S3]]。

### 耐火度分级

铸造用普通粘土按耐火度分为两级[[S7]]：

| 等级代号 | 名称 | 耐火度要求 |
|----------|------|------------|
| G | 高耐火度粘土 | >1580℃ |
| D | 低耐火度粘土 | 1350~1580℃ |

钠基膨润土的熔点范围为1250~1300℃[[S3]]。高岭土的Al₂O₃含量越高、Fe₂O₃及碱金属/碱土金属氧化物杂质含量越低，耐火度越高[[S22]]。

## 工艺作用与在压铸中的角色

### 压铸与砂型铸造的核心差异

在砂型铸造中，粘土是型砂的核心粘结剂，直接混配进原砂中制备粘土砂型/砂芯，承担决定型砂强度的主导功能[[S15]]。压铸工艺中粘土完全不参与此类砂型制备流程，其作用角色发生根本性转变[[S15]]。

### 压铸中的主要应用场景

压铸领域可确认的粘土主流工业应用集中在以下方面：

**第一类：涂料/脱模剂中的悬浮稳定组分。** 在早期压铸水基涂料/脱模剂配方中，主要以膨润土类粘土作为辅助悬浮稳定剂添加，利用其高比表面积防止涂料内固体耐火粉料沉淀[[S23,S32]]。GB/T 5611-2017《铸造术语》明确将膨润土列为涂料悬浮剂的典型示例[[S32]]。

**第二类：压铸配套砂芯的辅助添加组分。** 仅在压铸配套砂芯制造场景中，粘土以极低比例作为油砂/合成树脂砂芯的辅助添加，作用是提升砂芯常温湿强度，不作为砂芯主粘结剂[[S13]]。

**第三类：金属型涂层组分。** 永久型及压铸模具所用涂层常规配方包含高岭土（kaolin clay）、硅酸钠、滑石粉和滑石，涂层厚度一般控制在0.8mm以内，起到防止金属粘模、降低热冲击对模具损伤、调控铸件凝固速率的作用[[S8,S9]]。

### 已排除的应用假设

在已检索的公开标准、专业手册、核心期刊文献中，未发现粘土作为半固态压铸粘结剂、独立高温润滑介质、金属液净化介质或浇口密封功能性材料的大规模主流工业应用的明确记录[[S26,S16]]。当前绝大多数现代主流商业压铸脱模剂、压铸专用砂芯配方已不再将粘土作为必要组分[[S4,S12]]。

## 关键属性与性能参数

### 化学组成要求

压铸用耐火粘土典型化学成分要求[[S22]]：

| 指标 | 要求 |
|------|------|
| Al₂O₃ | ≥28% |
| Fe₂O₃ | <2.5% |
| Na₂O+K₂O | <3% |
| CaO+MgO | <2% |

高品级铸造膨润土[[S35]]：

| 指标 | 要求 |
|------|------|
| 蒙脱石含量 | ≥70% |
| 胶体分散粒径<0.001mm颗粒占比 | >70% |
| 总交换容量 | ≥70 mg eq/100g |

### 吸水与含水量控制

膨润土干基含水量要求不大于12.0%（冬季允许上限15.0%）[[S41]]；普通粘土含水量要求不大于10.0%[[S14]]。

### 粒度要求

| 粘土类型 | 筛网要求 | 筛孔孔径 |
|----------|----------|----------|
| 膨润土 | 95%以上通过10号铸造试验筛 | 0.075mm |
| 普通粘土 | 95%以上通过9号铸造试验筛 | 0.106mm |

资料来源：[[S14,S41]]

### 粘结与力学性能

膨润土粘结性能显著优于普通粘土：钠基膨润土湿态粘结力比普通粘土高2~3倍，干强度更高，对水分含量波动的耐受能力更强[[S39]]。钙基膨润土湿强度更高但干强度偏低；钠基膨润土的热湿拉强度远高于钙基膨润土，抗夹砂缺陷能力更优异[[S39,S3]]。

### 膨胀性能指标

粘土的膨胀性能可通过三项行业指标量化[[S11]]：

| 指标 | 定义 |
|------|------|
| 胶质价 | 粘土与水搅匀静置后凝胶层占混合体系的体积百分数 |
| 膨润值 | 3g膨润土加氯化铵溶液和水静置24h后的沉淀物体积 |
| 膨胀指数 | 静置24h后可保持全溶胶状态的蒸馏水体积与粘土质量的比值，单位mL/g |

钠基膨润土的膨胀倍率可达10~20倍[[S3]]。

### 热稳定性

粘土的热稳定性直接影响其在压铸循环中的耐用性[[S39]]：

| 矿物类型 | 失去结构水温度区间 | 热耐久性评价 |
|----------|---------------------|-------------|
| 蒙脱石（膨润土主矿物） | 500~800℃ | 较优 |
| 高岭石（普通粘土主矿物） | 400~600℃ | 较差 |

膨润土失去结构水的温度高于普通粘土，因此在相同压铸浇注循环下粘土失效比例更低，耐用性更好[[S39]]。

### 悬浮增稠特性

钠基膨润土在水基压铸涂料中用作悬浮剂的性能优于钙基膨润土；有机膨润土可直接适配醇基压铸涂料作为悬浮组分配方，无需提前水溶胀处理即可获得良好的触变性能[[S1]]。

粘土类辅料悬浮增稠性能优异且成本极低，但其本身几乎无高温自润滑效果，高温下易脱水烧结，无法单独作为功能性润滑/脱模组分使用[[S29]]。

## 结构与组成

### 晶层结构

粘土矿物均属于层状硅酸盐。高岭石为两层型结构，由一层Si-O四面体和一层Al-(O·OH)八面体构成一个晶体单元，相邻晶层间由氢键连接[[S38]]。蒙脱石为三层型结构，晶层间可吸水膨胀，层间距随含水量变化，这是膨润土粘结力和膨胀性能的微观基础[[S38]]。

### 改性方式

有机膨润土通过季胺盐或季胺碱置换钠、钙离子获得，能被醇或油类良好润湿，是醇基涂料的优良悬浮剂[[S18,S1]]。钙基膨润土可通过加碳酸钠进行活化处理，部分钙离子被钠离子取代，提升性能[[S10]]。

## 替代材料对比

压铸行业中存在多种可替代粘土功能的其他辅料，各有优劣[[S29,S24,S27]]：

| 材料类型 | 优点 | 缺点 |
|----------|------|------|
| 粘土/膨润土 | 悬浮增稠优异，矿藏丰富成本极低 | 无高温自润滑，高温易脱水烧结，无法单独作润滑/脱模组分 |
| 石墨 | 层状结构自润滑好，高温还原环境下可耐3000℃ | 空气中400℃即开始氧化，易造成铸件表面发黑污染，成本中等 |
| 六方氮化硼 | 层状结构润滑性接近石墨，空气中可耐900~1000℃无氧化，不污染铸件表面 | 材料成本极高 |
| 纯合成无填料水基涂料 | 无固体颗粒残留，铸件表面质量高，发气量低不易产生气孔，环保性好 | 成本显著高于含固体填料体系 |

## 缺陷与质量影响

### 夹砂结疤

粘土使用不当可直接引发夹砂结疤缺陷。当型砂pH值<7（偏酸性）且型砂缺陷倾向参数F（热压应力/热湿拉强度）远大于455~600的常规控制区间时，铸件表面极易出现夹砂结疤。预防措施为选用热湿拉强度更高的钠基膨润土，调整型砂pH至中性区间[[S11]]。

### 气孔与表面气泡

粘土烘干不充分或含水量超标时，压铸浇注过程中水分急剧汽化产生大量气体，极易引发铸件内部气孔和表面气泡缺陷。预防措施是严格控制粘土辅料及混配后型砂的含水量，使用前按规范完成烘干工序[[S31,S21]]。

### 夹杂与表面粗糙

粘土粒度不达标、粗颗粒占比过高会直接导致压铸涂料涂覆均匀性下降，铸件表面粗糙度上升并出现粗糙麻点；脱落的粘土粗颗粒易混入金属液形成非金属夹杂缺陷。进场粘土需严格按对应标准筛检验通过率，不合格粘土必须重新研磨处理至符合粒度要求[[S25,S17]]。

### 粘模与表面拉伤

粘土添加比例过高时，涂料的发气量显著提升，且粘土砂型/涂料层的退让性下降，易导致铸件粘模和表面拉伤。应严格按工艺配比控制粘土添加量，确保其不超过配方规定的合理阈值[[S25,S40]]。

压铸件常见缺陷（表面凹瘪、粘模等）与粘土使用之间的具体关联及排除措施，在行业手册中已有系统收录[[S36]]。

## 证据局限

需要明确的是，在已检索范围内，粘土在压铸中的角色受限于上述辅助功能层面。目前未检索到压铸行业将粘土作为核心主功能材料广泛大规模应用的公开权威记录，绝大多数现代主流商业压铸脱模剂、压铸专用砂芯配方已不再将粘土作为必要组分使用[[S4,S12]]。压铸领域粘土的“辅助悬浮”定位是其最主要的行业共识[[S5,S38]]。
