## 概述

铁液净化在铸造标准中被明确定义为**去除铁液中气体、杂质元素和夹杂物的处理方法**，常用方法包括气体氧化法、渣‑金属反应法及过滤法三大类[[S29]]。其处理对象限定为铸铁、铸钢等铁基熔体，与非铁合金（铝、镁等）熔体净化在工艺语境上存在明确区分，现有压铸领域公开资料中未发现将“铁液”指代有色金属熔体的特殊行业用法记录[[S26,S5,S36]]。

## 工艺作用与意义

铁液净化对铸件质量的提升体现在多个层面：

- **提升铁液纯净度**：高温铁液过热至约1475 ℃时，发生 SiO₂ + 2C = Si + 2CO↑ 反应，氧以CO形式逸出，铁液沸腾促进夹渣上浮、溶氧下降，显著提升纯净度[[S22,S37]]。
- **减少铸造缺陷**：球墨铸铁生产中，合格的净化处理可将铁水原始0.1%的硫含量降至0.01%甚至更低，大幅减少铸件中针孔、渣孔、缩松等缺陷[[S20,S16,S23]]。
- **改善力学性能与气密性**：净化后的铁液可得到组织更致密、力学性能更高的铸件，同时提升铸件气密性和表面质量[[S22,S20]]。

铁液的纯净度、熔炼温度、化学成分是铁液冶金质量的三项核心指标[[S11,S22]]。

## 分类与方法

### 按处理位置分类

- **炉内净化**：在熔炼炉内通过控制炉渣碱度、添加净化剂等方式进行。
- **炉外净化**：铁液出炉后在铁水包等容器内进行脱硫、脱氧、除气等处理，是球墨铸铁等高品质铸铁生产的主流工艺路线。

### 主要净化方法

#### 脱硫

硫是铸铁中的有害元素，会提升铁液黏度并生成低熔点 FeS 诱发铸造缺陷[[S7,S9]]。脱硫是铁液净化的核心单元操作之一，常用脱硫剂及工艺见下表。

| 脱硫剂 | 典型添加量 | 脱硫效果 | 备注 |
|--------|------------|----------|------|
| 纯碱 (Na₂CO₃) | 2%~3% | 含硫量从0.1%降至0.02% | 向渣中添加石墨可提升效率；产生有害烟雾，对耐火材料侵蚀严重[[S10,S2]] |
| 碳化钙 (CaC₂) | 0.4%~1.0% | 含硫量从0.1%稳定降至0.01%以下 | 铸造行业应用最广泛的脱硫剂；粒度8~60目(2.36~0.25 mm)；需搅拌混合；残渣遇水生成乙炔有安全风险[[S2]] |
| CaO-CaF₂ 复合脱硫剂 | 1.3%~2.0% | 适配对低硫等级要求不高的工况 | 添加5%石墨可显著提升脱硫效率[[S28]] |

#### 脱氧与净化

向铁液中加入0.30%的包钢1号稀土硅铁，可同时实现脱氧和铁液净化，有效降低铁液氧化引发的针孔缺陷发生率[[S23]]。向铁液中添加碳化硅(0.7%~1.0%)也有助于提高铁液纯净度[[S11,S22]]。

#### 锰‑硫反应净化

硫会生成低熔点FeS，使铁液黏滞并产生缺陷。添加锰可与硫生成密度远低于铁液的MnS上浮进入渣层，实现脱硫净化。锰的添加量计算公式为：1.7 × 硫含量 + 0.3%[[S7,S9]]。

## 关键设备与工艺参数

球墨铸铁生产对原铁水硫含量有严格要求：原铁水含硫量应控制在0.07%以下，球化处理后铁液硫含量需降至0.03%以下[[S4]]。

### 主要炉外脱硫工艺

#### 摇包脱硫

铁水包承受偏心旋转运动产生搅拌，使脱硫剂与铁液充分混合。工艺参数如下[[S16,S20]]：

| 参数项 | 数值 |
|--------|------|
| 包包衬耐火材料 | 70%氧化铝 |
| 旋转转速 | 50~55 rpm |
| 处理时长 | 8~10 min |
| 碳化钙添加量 | 0.5%~1.0% |
| 单次处理量 | 5~10 t |
| 温降 | 28~44 ℃ |

#### Rheinstahl 机械搅拌脱硫

采用倒T型耐火搅拌桨进行机械搅拌混合。工艺参数如下[[S16,S19]]：

| 参数项 | 数值 |
|--------|------|
| 搅拌桨形式 | 倒T型耐火搅拌桨 |
| 搅拌转速 | 30~100 rpm |
| 搅拌时长 | 6~10 min |
| 碳化钙添加量 | 0.7%~1.0% |
| 单次处理量 | 5~20 t |
| 脱硫后硫含量 | 0.009%~0.014% |

#### 多孔塞吹气脱硫

在铁水包底部放置多孔耐火塞，通入氮气等惰性气体产生弥散气泡搅动铁液，同时投加脱硫剂。工艺参数如下[[S16,S20,S14]]：

| 参数项 | 数值 |
|--------|------|
| 碳化钙添加量 | 1.0%~1.5%（间歇式） |
| 脱硫后硫含量 | 可达0.005% |
| 温降 | 30~40 ℃ |
| 特点 | 投资与维护成本低，适合中小吨位处理 |

连续式多孔塞脱硫工艺在8~15 t/h冲天炉出铁场景下，碳化钙添加量1.5%，可将初始硫含量0.1%的铁液稳定脱至0.008%~0.011%，处理后铁液温度不低于1485 ℃[[S20]]。

### 铁液温度控制

球墨铸铁用原铁水出铁温度需控制在1454~1566 ℃区间，球化处理前铁液温度维持在1360~1380 ℃，可降低镁元素汽化反应剧烈程度，提升镁的吸收率，减少球化后氧化夹杂生成[[S6,S12]]。

球化处理完成后的铁液需在铁水包内充分静置，让裹挟的镁‑氧化膜类夹杂物充分上浮去除，避免后续铸件出现夹渣缺陷[[S33]]。

## 效果评价与检测

### 铁液质量评价指标

灰铸铁冶金质量可采用以下三项量化指标综合评价[[S11,S22]]：

- **成熟度** = 实测抗拉强度 / (1000~800 Sc)
- **硬化度** = 实测硬度 / (530~344 Sc)
- **品质系数** = 成熟度 / 硬化度

品质系数越高，代表铁液纯净度与铸件力学性能的匹配性越好[[S11,S22]]。

### 化学成分快速检测

热分析法是铁液质量炉前快速检测的行业通用标准方法，可在约3.5分钟内准确测得碳、硅含量[[S21,S3]]：

| 检测项目 | 检测误差 |
|----------|----------|
| 碳当量 | ≤ ±0.05% |
| 碳含量 | ≤ ±0.05% |
| 硅含量 | ≤ ±0.1% |

### 含气量检测

铁液含气量检测的行业通用方法包括常压凝固试验法、减压凝固法、气相色谱法等。其中减压凝固法通过测定试样密度反推含气量，操作简便、灵敏度高，是炉前快速检测的首选方法[[S32,S8]]。

## 与压铸工艺的关联

压铸场景下铁基合金熔体的液相线温度约为2200 ℉（约1204 ℃），远高于锌合金（700 ℉/371 ℃）、铝合金（1100 ℉/593 ℃）的液相线温度[[S25]]。因此铁液净化需配套耐超高温的熔体处理设备，工艺难度显著高于有色金属熔体净化。

## 常见问题与缺陷

### 净化不足导致的缺陷

铁液净化不足引发的析出气孔缺陷机理为：熔炼过程中大量气体被铁液吸收，冷却凝固时气体溶解度急剧下降，过饱和析出的气泡无法完全上浮逸出，最终在铸件内部形成弥散分布的针孔类缺陷[[S24]]。

### 二次污染

铁液净化操作不当可引发二次污染，核心来源包括[[S15,S1]]：

- 未充分烘干的浇包等工具带入水汽；
- 转包过程中铁液紊流裹挟空气发生二次氧化；
- 未选用合格的耐材造成炉衬侵蚀剥落进入铁液。

**防控要点**：所有接触铁液的工装充分预热烘干；转包过程采用密闭流槽减少裸露液面；选用高温下化学稳定性良好的铝镁尖晶石类耐火材料[[S15,S1]]。

## 术语边界

| 术语 | 与“铁液净化”的关系 |
|------|---------------------|
| 熔体净化 | 上位概念，涵盖所有金属熔体去除杂质的操作；“铁液净化”是其铸铁/铸钢场景下的子类应用[[S29,S35]] |
| 铁水处理 | 比“铁液净化”范围更大的上位概念，包含球化、孕育、温度调整等非净化类操作[[S10]] |
| 脱硫 | 铁液净化的下属单元操作，不能等价于铁液净化[[S10,S40]] |
| 脱氧 | 铁液净化的下属单元操作，不能等价于铁液净化[[S40]] |
