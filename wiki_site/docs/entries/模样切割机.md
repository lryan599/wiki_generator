## 概述

“模样切割机”是一个在铸造行业内流传的区域俗语或厂内惯用语，并未被现行国家标准 GB/T 25370-2020《铸造机械 术语》收录为规范术语[[S17,S37]]。在工程实践中，该词条根据使用场景可能指称两类完全不同的设备：一是在消失模铸造（EPC）及实型铸造领域，指用于加工泡沫塑料模样的电热丝切割机或 CNC 泡沫模样加工机；二是在压铸领域，作为压铸件切边机（俗称去浇口机）的现场代称[[S34,S36]]。以下按照这两种主流技术语境分别阐述。

## 释义一：消失模用发泡模样切割机

### 定义

在消失模铸造与实型铸造体系中，“模样切割机”的规范名称为电热丝切割机或泡沫切割机，在英文技术文献中对应 pattern cutting machine[[S33,S18]]。设备通过电热丝的热辐射作用使泡沫塑料（EPS、STMMA、EPMMA 等）局部熔化或汽化，实现板材下料、模样分块加工及浇冒口成型，是消失模白区工序中的专用加工设备[[S33]]。

### 工艺作用

发泡模样切割机属于消失模铸造白区核心工序设备，在全流程中的位置紧随预发泡成型/泡沫板材制备工序之后，在泡沫模样分块粘接、表面修整、浸涂烘干涂料工序之前，是单件或小批量生产中制造大、中型铸件泡沫模样的首选加工手段[[S33,S18,S14]]。

在大、中型铸件实型铸造生产线中，常规工艺路径为：大尺寸 EPS 泡沫板材先经门式电热丝切割机完成下料，再由数控加工设备加工分块模片，浇冒口则由台式电热丝切割机加工成型，模片最终通过热熔胶或冷胶粘接为完整模样[[S18]]。

### 类别与工作原理

泡沫模样的加工方法主要可分为两类：电热丝切割与数控铣削加工[[S33,S7]]。

- **电热丝切割**：利用镍铬电热丝通电发热，使接触区域周围的泡沫塑料材料熔化或汽化，随电热丝与泡沫坯的相对运动完成切分。切割过程中，部分熔融液态树脂会在切割表面冷凝，形成致密玻璃状纤维层，获得平整光洁的加工面[[S33,S22,S35]]。
- **CNC 泡沫加工**：采用专用高速旋转的中空锉铣式特殊刀具，按照从 STL 三维模型生成的数控刀路，以披削原理高速切削泡沫坯，通过锋利刀刃快速切断泡孔组织，完成三维复杂模样的精加工[[S7,S16]]。

### 结构与核心组件

电热丝型切割机的核心结构包括：刚性焊接机架、可张紧的镍铬电热丝加热系统、工作台进给运动轴、调压温控电控单元；部分机型配置可旋转回转盘，用于加工圆柱形回转体模样，重锤恒张力结构可保证电热丝加热状态下张力稳定[[S35,S10]]。

CNC 泡沫模样加工机则包含：整体龙门机架、带负压吸尘通道的专用高速铣削主轴、多轴联动数控运动系统及 CAM 数控编程控制单元，其内置负压排屑吸尘装置可将泡沫碎屑直接吸入除尘器，实现连续无人化清洁生产[[S4,S7]]。

国内还出现了便携式快速切割设备，结构简化，可快速拆卸收纳、靠墙放置，采购与使用成本约为同性能常规机型的 50%，适应中小铸造企业空间有限的使用场景[[S2]]。

### 工艺参数与要求

#### 电热丝切割参数

| 参数项 | 推荐值/范围 | 说明 |
|---|---|---|
| 工作温度 | 250~500 °C | 需保持在熔化-气化区间，避免温度过低产生熔化粘丝，或过高引发燃烧裂解 [[S33,S32]] |
| 粗加工电热丝直径 | φ1.0~1.2 mm | |
| 精加工电热丝直径 | φ0.2~0.5 mm | 直径越小，切割表面越光洁 [[S33,S32]] |
| 切割速度 | 均匀缓慢 | 切割长度增加时需相应降低速度以保证表面质量 [[S24,S22]] |

#### CNC 泡沫铣削参数

| 参数项 | 推荐值/范围 | 说明 |
|---|---|---|
| 加工板材密度 | 16~18 kg/m³ | 板材压缩强度 70~80 kPa，吸水率 <3%（体积分数） [[S7]] |
| 主轴转速（加工平面） | 1400~2700 r/min | |
| 主轴转速（加工内圆/内圆角） | 2700~4200 r/min | 低速易导致泡沫珠粒被刮落，表面拉毛 [[S28]] |
| 国外专用铣床主轴转速 | 6000~12000 r/min | 远高于常规金属加工铣床，有效避免凹坑拉毛缺陷 [[S7]] |
| 叶片式专用泡沫铣刀切削深度 | 3~5 mm，最大 10 mm | |

#### 可加工材料

发泡模样切割机可加工的消失模模样材料涵盖 EPS（聚苯乙烯）、STMMA（苯乙烯-甲基丙烯酸甲酯共聚树脂）和 EPMMA（聚甲基丙烯酸甲酯）三类主流泡沫板材，以满足不同铸件材质对模样热解碳残留和发气量的差异化要求[[S18,S33]]。

### 缺陷与维护

- 电热丝切割时，随切割长度增加，泡沫与电热丝的热接触面积增大，表面质量随之下降，需对应降低切割速度；长切割段宜选用稍粗电热丝以保持刚性，短段则选用细丝以获得更高光洁度[[S24,S22]]。
- 铣削泡沫模样若使用常规金属加工麻花铣刀，极易刮落泡沫珠粒，导致加工表面出现凹坑拉毛缺陷，必须更换为专用的锉铣式中空泡沫加工刀具[[S7]]。
- 日常维护要点：电热丝切割设备需定期校验温控单元输出精度，防止温度过低造成泡沫熔化粘丝或温度过高引发泡沫裂解冒烟[[S22,S35]]；CNC 加工设备需定期清理负压除尘通道内堆积的泡沫碎屑，防止风道堵塞降低吸尘效果[[S35]]。

### 应用场景

发泡模样切割机广泛应用于工程机械大型铸件模样制作、汽车模具实型铸造、大型艺术铸造（如 70 米观音像的 1:1 巨型青铜像脸部整面 EPS 泡沫模，采用三轴加五轴 CNC 加工中心分块铣削制作）以及单件定制重工机床底座实型铸造生产[[S25,S38]]。

## 释义二：压铸切边机（压铸清理设备）

### 定义

在压铸生产现场，“模样切割机”也可能是操作人员对压铸件切边机（trim press）的俗称。在英文压铸权威专著《Die Cast Engineering》中，trim press 的定义为：配合专用切边模将压铸件飞边、溢流口及浇注系统流道等多余材料切除的机械式或液压式压力机[[S8,S40]]。其对应的国内压铸领域标准名称为“压铸件切边机”，是压铸生产线上标配的铸件后处理设备[[S34,S36]]。

### 工艺作用

压铸切边机在压铸全工艺链中的工位位于：压铸件从压铸机取出、经过水冷淬冷工序之后，后续对接手工自检、后处理打磨与抛丸清理工序，是构成自动化压铸岛的标准配置单元[[S3,S9]]。单台偏心式切边机的产能通常可匹配 6 台压铸机同时产出的铸件清理需求[[S9]]。

完整的切边工艺流程为：将压铸件定位放置在切边模下工位，动力系统驱动工作滑块下行，通过切边模阴阳模的剪切作用一次性切断浇口、分型面飞边与溢流槽余料；完成后滑块上行，自动分选出合格铸件与金属余料，实现连续作业[[S26,S9,S3]]。

### 驱动类型与比较

| 驱动类型 | 优点 | 缺点 | 适用场景 |
|---|---|---|---|
| 机械型（曲轴/偏心传动）| 动作快，切边效率高，结构简单故障少 [[S34]] | 加压压力固定为瞬间冲击模式，定位偏差时易损伤切边模具 [[S34,S36]] | 中小型压铸件小批量生产 |
| 液压型 | 工作台面积、行程、压力、速度均可无级调整，运行平稳无冲击，极少过切 [[S34,S36]] | 冲压速度慢，设备购置成本高，运行能耗大 | 大型压铸件切边 [[S34,S36]] |
| 气动型 | 结构简单，无液压泄漏风险 [[S9]] | 生产效率低于机械型与液压型 | 小型锌合金压铸件的简易浇口切除 [[S9]] |

### 切边力计算

切边机所需的总切边力可由公式 Ft = L × T × S 计算，其中 L 为飞边/浇口总切边长度，T 为飞边/浇口厚度，S 为铸件材料切断强度。大多数常用压铸合金的 S 值取 0.214 kN/mm²（31000 lb/in²）[[S26]]。

压铸金属切边作业的阴阳模剪切间隙推荐取 0.075 mm（0.03 英寸）；若需冲剪完整铸件壁厚的预开孔，冲套孔径应比冲头直径大 0.15 mm（0.06 英寸），以避免卡料和铸件变形[[S26]]。

### 结构与组成

压铸切边机的典型结构包含以下功能部分[[S27,S19]]：

| 组成部分 | 功能 |
|---|---|
| 机架 | 整机承重基座，一般采用厚钢板焊接整体结构，保证运行刚性 |
| 动力系统 | 液压/气动/机械驱动单元，提供剪切所需额定压力 |
| 工作滑块 | 带动上切边模上下往复动作，完成剪切行程 |
| 工作台面 | 为下切边模提供安装定位基准面，可根据铸件需求扩展尺寸 |
| 切边模定位机构 | 以压板与螺栓组合固定模具，保证模具安装位置精度 |
| 安全光栅 | 安全防护组件，探测到人员肢体误入作业区时立即触发急停 |
| 控制单元 | PLC 程序控制完整切边循环，可适配不同产品参数调用 |
| 自动上下料（选配）| 搭配工业机械手实现无人化全自动切边作业 |

切边模分为下落料式和复合式两种主流类型。下落料式模具结构简单、制造成本低，切边后工件直接下落至操作工位，实现铸件与飞边余料自动分离，是切边机最常用的适配模具类型；复合式切边模可在切边前将脆弱铸件牢牢夹紧，避免切边变形，适用于易变形薄壁铸件或带有内孔切除需求的特殊压铸件[[S26]]。

### 典型应用与产能

压铸切边机适配的加工材质覆盖铝合金、镁合金、锌合金等绝大多数常用压铸有色合金，典型应用产品包括汽车变速箱壳体、发动机结构件、3C 电子金属外壳等批量化压铸零件[[S3,S39]]。

以 50 吨级倾斜滑块式自动化切边机为例，其滑块可在机床工作平台内前后移动，适配机器人取放料动作；单次循环完成切边后，滑块可 180° 翻转，通过吹扫装置自动排出飞边余料，全程无需人工干预，量产件不良率可控制在 2% 以下[[S27]]。

### 常见故障与维护

- 切边模定位偏差或设计不当会导致飞边残留、铸件切口变形或工件表面碰伤缺陷；切边模刃口硬度不足或过载剪切会引发崩刃，需及时更换合金工具钢制成的合格刃口备件[[S36,S27]]。
- 液压机型长期运行后密封件老化，可能出现液压油泄漏，需定期检查更换 O 型圈、密封垫、液压阀等易损损耗件[[S27]]。
- 设备启动前需按规定手动加注润滑油脂，定期检查飞边刮料板是否贴合工作台面，建立设备保养台账以跟踪易损备件库存状态[[S11]]。

## 小结

“模样切割机”并非铸造行业的规范术语，其实际指代需根据应用场景区分：在消失模铸造和实型铸造中，分别指以电热丝切割或 CNC 铣削方式加工泡沫塑料模样的专用设备；在压铸领域，则是压铸件切边机（trim press）这一后处理清理设备的现场俗称。两种设备处于完全不同的铸造工艺链位置，分别服务于模样制备与铸件后处理工序，其设备结构、工作原理与工艺参数均截然不同。
