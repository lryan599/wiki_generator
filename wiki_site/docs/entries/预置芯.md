## 概述

在高压压铸（HPDC）工艺中，预置芯（Embedded core / Ram-up core）是指合模前预先放置在压铸模型腔内、与铸件同步从模内取出，后续通过特定方式移除以获得铸件复杂内腔或异形外轮廓的可除型芯[[S10,S21]]。该术语在GB/T 5611-2017《铸造术语》标准条款6.3.20中正式定义，属标准型芯分类项[[S20,S10]]。其全部作业流程涵盖预置芯制造、存储、预热、模具内定位固定、压铸填充保压、开模取件、后处理型芯去除共7个核心工序[[S5]]。

预置芯技术的核心价值在于能够成形传统金属抽芯机构无法实现的复杂封闭异形内腔，如发动机缸体水套、交叉油路等，从而大幅简化模具结构、降低模具制造成本[[S2]]。

## 术语辨析

在铸造术语体系中，“预置芯”与若干相关术语存在明确的语义边界：

| 术语 | 与“预置芯”的关系 | 说明 |
|------|-------------------|------|
| 预置型芯 | **完全同义** | GB/T标准下二者为同一概念的不同表述[[S3,S18]] |
| 预制芯 | 上位概念 | 泛指所有预先制备完成的型芯，包含可通过压铸机机动抽芯直接取出的常规金属活动型芯，覆盖范围远大于仅支持后续非机动抽芯的HPDC预置芯[[S3]] |
| 可溶芯 | 子类 | HPDC预置芯中可通过水溶液或化学试剂溶解去除的一类，不等同于全部预置芯[[S18]] |
| 砂芯 / 盐芯 / 陶芯 | 子类 | 按材料属性划分的预置芯子类，并非独立于预置芯的平行概念[[S3]] |

HPDC可除预置芯的工程定义更为具体：合模前放入压铸模的独立零件，射入金属在它四周凝固后与铸件同时从模中取出，后续通过粉碎、溶解、熔化等方式从铸件内部去除，由型砂、低熔点金属、可溶性盐、陶瓷等材料制成[[S18]]。

## 分类体系

高压压铸场景下的预置芯可依据材料类型和去除方式两个维度建立完整的分类体系[[S17,S26,S11]]。

### 按材料类型分类

| 类别 | 典型材料 | 关键特征 |
|------|----------|----------|
| 金属预置芯 | 锌、铝、低熔点合金 | 后续加热熔化去除，适用于黄铜等高温合金压铸[[S17]] |
| 树脂砂预置芯 | 覆膜砂（呋喃/酚醛树脂粘结剂） | 配备标准芯头稳固安放在型腔中，可通过振动方法去除[[S17,S26]] |
| 盐/水溶性预置芯 | 海盐/岩盐+铝矾土、KCl、Na₂CO₃等复合盐 | 浸水溶解去除，表面精度优异，为当前主流路线[[S25]] |
| 陶瓷预置芯 | 磷酸盐基陶瓷、磷酸盐-硅酸盐系陶瓷 | 一次性使用，无需脱模斜度，氢氧化钠溶液浸蚀去除[[S26]] |

### 按去除方式分类

| 去除方式 | 适用芯材 | 典型脱芯参数 / 介质 |
|----------|----------|---------------------|
| 机械振动破碎 | 树脂砂芯 | 高压水配合振动清砂[[S17]] |
| 水溶液溶解 | 盐芯、水玻璃粘结砂芯（含NaOH）、KCl烧结芯 | 70～90℃热水5～10 min；高压水射流1～2 min[[S25,S17]] |
| 化学试剂溶解 | 陶瓷芯（NaOH溶液）、磷酸盐基陶瓷（稀醋酸） | 浸入指定化学溶液溶解[[S26,S30]] |
| 低熔点合金热熔 | 锌、铝制金属芯 | 加热熔化清除[[S17]] |

## 核心技术要求

可溶类预置芯需满足10项通用基础技术指标，方能适配高压压铸工况[[S31,S2]]：

1. **熔点高、溶失点低**：防止接触高温金属液时提前熔化变形，同时保证快速脱芯缩短生产周期。
2. **复制性良好**：尺寸精度高、表面光洁，确保内腔轮廓无误地反映到铸件上。
3. **无多孔性、不吸潮**：具备长期存储能力。
4. **无毒性**：工作温度下不析出有害气体。
5. **抵抗热冲击**：在高温金属液冲刷下不碎裂、不变形。
6. **足够的抗压抗弯强度**：承受金属液高速充型的冲击力。
7. **低热膨胀系数与凝固收缩率**：保障尺寸精度。
8. **成本低廉、易清除、可回用**。
9. **制造简便**：无需特殊设备。
10. **便于修补**。

### 核心性能基准数据

以无机粘结剂系水溶性型芯为参考基准，其核心性能参数如下[[S3]]：

| 性能指标 | 数值 |
|----------|------|
| 湿压强度 | 0.022 MPa |
| 常温干抗压强度 | 12.5 MPa |
| 常温干抗弯强度 | 3.25 MPa |
| 高温强度（600～1200℃） | ≥1.5 MPa |
| 型芯发气量 | 13 ml/g |
| 透气性 | ≥65 |
| 水溶性 | 良好 |

### 热冲击抗力

盐芯的热冲击抗力由以下公式表征[[S30]]：

$$
K = \frac{\sigma_b \cdot \lambda}{E \cdot \delta}
$$

其中：*K*——热冲击抗力系数；*σ_b*——抗拉强度；*λ*——热传导系数；*E*——弹性模数；*δ*——热膨胀系数。提高热冲击抗力的途径是提高σ_b和λ，降低E和δ。加入20%铝矾土后，盐芯的热膨胀系数被有效抵消，尺寸精度得到保障[[S30,S25]]。

## 工艺流程

预置芯在高压压铸中的完整作业流程可分为7个核心工序[[S5,S20]]：

### 1. 预置芯制造

以经典盐质可溶预置芯配方为例：70%质量分数的海盐/岩盐（纯度≥95%）搭配30%质量分数的铝矾土，经950～1000℃熔融后浇入200℃预热的金属芯盒成型，或采用压铸法直接压制以提高表面质量和尺寸精度[[S25]]。制芯过程中需严格控制盐液温度和持压时间，二者是保证型芯尺寸精度的核心要素[[S30]]。

对于无水CaCl₂基水溶芯，制芯过程必须完全去除结合水，完成后需在200℃无水环境中保温贮存，否则残留水分受热释放将导致铝压铸过程产生气孔和氢致缺陷[[S6]]。

### 2. 存储

可溶芯制芯后需在干燥环境中密封储存。水溶性型芯相比传统油粘结剂砂芯无吸潮性，对储存条件要求相对宽松[[S13]]。

### 3. 预热

压铸前，预置芯需进行预热。盐芯推荐预热温度为200℃左右，避免压铸后与高温铝合金接触产生热应力开裂[[S25,S30]]。对于合成树脂砂基溃散芯和陶瓷基可溶芯，预热温度区间通常在150～250℃范围内调整[[S29]]。

### 4. 模具内定位与固定

预置芯必须采用带刚性防冲击的方台定位结构，定位支撑点需避开主浇口金属液直射区域，保证70 MPa以上高压充型过程中芯体不发生移位、变形或断裂[[S12]]。由动型镶块上可溶或可熔型芯的固定结构可见，刚性多点支撑是实现可靠定位的关键。

### 5. 压铸填充保压

带预置芯高压压铸需要采用特殊的浇注系统和压铸参数。与常规HPDC不同，内浇口速度必须限制在15～20 m/s（常规为30～60 m/s），优先采用切线侧进浇设计[[S12]]。压射工艺须采用匀加速多段低速压射，低速速度区间控制在0.05～0.54 m/s，最高增压压力相比常规无预置芯工艺调低10%～15%[[S12,S32]]。整个充型过程中预置芯表面受力波动应小于0.5 MPa，受力均匀无局部集中载荷[[S32]]。

### 6. 开模取件

铸件冷却后开模，预置芯与铸件一同取出。对于一次性使用的型芯，取件时需注意避免因顶出偏斜造成断芯[[S24]]。

### 7. 后处理除芯

不同材质的预置芯需匹配不同的除芯工艺[[S17,S4,S25]]：

| 芯材类型 | 除芯方式 | 典型参数 |
|----------|----------|----------|
| 普通盐芯 | 70～90℃高压水冲洗/浸泡 | 常温水4～6 h；80～90℃热水5～10 min；高压水射流1～2 min |
| 水玻璃粘结砂芯 | 氢氧化钠溶液浸泡溶解粘结剂 | — |
| 磷酸盐基陶瓷可溶芯 | 稀醋酸溶解 | — |
| 覆膜砂芯 | 高压水配合振动清砂 | — |
| KCl烧结可溶芯 | 常温水中直接溶解 | — |

以难熔耐火材料为主体、可溶性盐作粘结剂的可溶芯，清理过程无粉尘噪音污染，旧砂和粘结剂水溶液均可回收[[S16]]。

## 常见缺陷与防治

### 偏芯（芯子移位）

**现象**：铸件对应成型区域壁厚分布不均匀[[S15,S24]]。

**成因**：
- 型芯制造或组配时尺寸偏差[[S15]]
- 芯座配合间隙过大[[S15]]
- 金属液充型时的浮力和冲击推力大于芯体支撑锁紧力[[S15]]

**防控措施**：
- 加大支撑芯头尺寸，设置导向定位基准[[S15]]
- 使用专用下芯工装校验芯体位置[[S15]]
- 采用底部方台定位，抵抗浇注和增压冲击力[[S12]]

### 断芯

**现象**：铸件内部芯体断裂，堵塞成型通道[[S24]]。

**成因**：
- 型芯强度不足（未做氮化等强化处理）[[S24]]
- 内浇口位置不当，金属液直接冲击芯体薄弱位置[[S24,S22]]
- 铸件顶出偏斜别断芯体[[S24]]
- 芯座装配过盈造成芯体应力集中断裂[[S22]]

**防控措施**：
- 调整内浇口流向，避免金属液正面直冲型芯[[S24,S12]]
- 优化型芯壁厚和支撑结构提升强度[[S22]]
- 采用匀加速料筒孕育半固态压铸，使充型过程芯面受力均匀[[S32]]

### 冲砂/盐污染与表面质量

**现象**：砂芯受金属液反复高速冲刷造成粘砂、表面粗糙，盐芯表面热裂纹导致金属渗透[[S32,S30]]。

**成因**：常规HPDC充型过程中金属液压力波动剧烈、速度高[[S32]]。

**防控措施**：采用匀加速多段低速压射工艺，保证整个充型过程中预置芯表面受力波动小于0.5 MPa[[S32,S12]]。验证结果表明，合格盐芯压铸件的表面光洁度可完全达到常规压铸件同等水平[[S28]]。

### 气孔

**现象**：铸件内部出现气孔。

**特殊成因（预置芯相关）**：含水预置芯在压铸高温下释放水汽，与铝液反应产生氢气[[S6]]。

**防控措施**：制芯过程完全去除结合水，在200℃无水环境中保温贮存[[S6]]。

## 优势与局限性

### 工艺优势

1. **模具结构大幅简化**：替代多组复杂斜向抽芯机构，盐芯专用制造模具成本仅为铝合金压铸件主成型模具的1/5～1/4[[S19]]。
2. **改型开发成本低**：仅需修改盐芯模具即可调整铸件内部流道结构[[S19]]。
3. **内腔表面精度优异**：盐质预置芯成型后铸件内腔表面精度达到常规压铸件同等水平，几乎不存在热冲击开裂故障[[S19]]。
4. **制造成本降低**：水溶类预置芯相比传统油粘结剂砂芯综合制造成本低30%，其中后续除芯工序单位成本仅为油砂工艺的10.7%[[S13]]。
5. **环保优势**：清理过程无粉尘噪音污染，材料可回收[[S16]]。

### 核心局限

| 局限项 | 具体描述 |
|--------|----------|
| 几何极限 | 盐芯可成型封闭内腔通道最小直径不小于10 mm；覆膜砂预置芯可实现长径比≥5.5:1的细长异形通道[[S34,S33]] |
| 重量限制 | 量产场景下单颗预置芯实用重量上限约1 kg[[S33]] |
| 脱芯效率 | 盐芯常温下水溶需4～6 h，80℃热水浸泡仍需5～10 min[[S25,S34]] |
| 材料回收 | 盐芯材料回收利用率低[[S34]] |
| 流动性限制 | 盐料流动性差，难以成型直径小于10 mm的精细复杂内腔通道[[S34]] |
| 生产节拍 | 带预置芯生产节拍相比同规格无内腔压铸件延长15%～30%[[S12]] |
| 工艺兼容性 | 传统无特殊工艺优化的HPDC很难直接适配预置芯量产，需专用参数优化[[S32]] |

## 相关技术关联

### 模具设计

预置芯的模具芯座必须采用带刚性防冲击的方台定位结构，定位支撑点避开主浇口金属液直射区域[[S12]]。预置芯在镶块上的标准固定结构通过多点刚性支撑实现可靠定位，可防止高速充型下芯体移位[S8]。

### 浇注系统设计

内浇口速度须限制在15～20 m/s，优先采用切线侧进浇设计，避免金属液直接正面冲刷预置芯体[[S12]]。

### 压铸参数优化联动

必须采用匀加速多段低速压射工艺，保证整个充型过程中预置芯表面受力波动小于0.5 MPa，受力均匀无局部集中载荷。最高增压压力相比常规无预置芯工艺调低10%～15%[[S12,S32]]。

### 后处理工艺选型

不同材质预置芯需严格匹配对应的除芯方式：盐芯选用70～90℃高压水冲洗/浸泡；水玻璃粘结砂芯需NaOH溶液浸泡；磷酸盐基陶瓷可溶芯选用稀醋酸；覆膜砂预置芯采用高压水配合振动清砂[[S17,S4]]。
