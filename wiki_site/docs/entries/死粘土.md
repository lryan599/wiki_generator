## 概述
在铸造及相关耐火材料领域，“死粘土”一词对应两种不同的工艺概念，须结合语境加以区分：

- **砂型铸造语境**：指国家标准 GB/T 5611—2017 定义的“枯粘土[死粘土]”，即湿型砂在循环使用中因受高温金属液热作用而永久丧失粘结能力的失效组分 [[S9,S1,S3]]。
- **耐火材料/压铸熔炼语境**：常狭义指向工业上经高温充分烧结的硅酸铝质耐火熟料（亦称死烧粘土、chamotte、煅烧耐火粘土），广泛用作压铸车间熔铝炉、保温炉、转注系统等接触铝液的关键内衬材料 [[S2,S17,S13,S12,S8]]。

本词条围绕上述两种含义展开，重点关注与压铸质量直接相关的死烧粘土（耐火熟料）的属性、应用及失效机理。

## 定义与标准（砂型铸造中的死粘土）
### 定义
国家标准 GB/T 5611—2017《铸造术语》明确将“死粘土”（亦称“枯粘土”）列为砂型铸造术语（编号 4.3.12）[[S9]]：

> 浇注后型砂中部分粘土因受高温金属热作用而丧失化合水，矿物结构被破坏，永远丧失粘结作用的该部分粘土生成物。

该定义在《中国机械工业标准汇编 铸造卷》及《机械基础制造工艺标准汇编 铸造》中均出现为“枯粘土[死粘土]” [[S1,S3]]。

### 成因与影响
湿型砂在反复循环使用过程中，靠近铸件表面的一层型砂中的粘土受热脱水，生成以烧结膜形式包裹砂粒的死粘土。死粘土膜具有多孔性、低熔点（约1080 ℃）、高吸水率等特征 [[S14]]。死粘土过多会导致：

- 型砂有效粘土含量下降，塑性、韧性劣化 [[S14]]；
- 铸件出现粘砂、夹砂结疤、冲砂等缺陷 [[S14,S11]]；
- 型砂三锤法试样质量低于148 g（反映SiO₂降低、死粘土增高），常作为现场快速判据 [[S14]]。

因此，在砂型铸造生产中须通过检测有效粘土含量、死粘土含量以及型砂的塑性/韧性间接判断死粘土积累，并及时补充新砂与膨润土 [[S14,S11]]。

## 耐火材料中的死烧粘土（熟料）
在压铸与铝合金熔炼领域，“死粘土”更多指代死烧粘土基耐火材料，其工艺定义为硬质耐火粘土经 1250~1410 ℃ 高温氧化气氛煅烧至充分烧结、体积稳定后破碎筛分得到的颗粒状熟料 [[S17,S18,S13]]。国际通用名称为 chamotte、calcined fireclay 或 dead‑burned clay [[S2,S12,S8]]。

### 成分与矿物组成
死烧粘土属于黏土质硅酸铝耐火材料，典型化学成分为（质量分数）[[S17,S18,S8]]：

| 组分 | 含量范围 / % |
|------|--------------|
| Al₂O₃ | 30 ~ 45 |
| SiO₂  | 50 ~ 65 |
| Fe₂O₃ | ≤ 2.5 |
| 其它（TiO₂、CaO、MgO 等） | 少量 |

煅烧后主要矿物相为**莫来石**（3Al₂O₃·2SiO₂）和玻璃相，伴有少量游离方石英（SiO₂）[[S17,S13,S4]]。

### 制造工艺
常规工艺流程如下 [[S17,S18,S7]]：
1. 原料：选用硬质高岭土（如焦宝石、上店土等），Al₂O₃ 含量较高的生粘土。
2. 煅烧：在隧道窑或间歇窑中以 1250~1410 ℃ 氧化气氛充分烧结，使结合水和结晶水完全脱除，体积趋于稳定，烧失量 ≤ 0.5 %。
3. 冷却与破碎：熟料块冷却后经颚破、对辊等破碎，筛分成所需颗粒级配。
4. 混配：可作为不定形耐火材料骨料，搭配软质粘土、水玻璃等结合剂，制备浇注料、捣打料或涂料。

### 核心性能参数
死烧粘土及以其为基的耐火制品的主要性能范围如下 [[S10,S16,S15,S4]]：

| 性能指标 | 典型值/范围 | 说明 |
|---------|-------------|------|
| 耐火度 | ≥ 1580 ℃ | 三角锥测温达 27 号以上 |
| 荷重软化温度 (T₀.₆) | 1250 ~ 1350 ℃ | 0.2 MPa 荷重下开始软化温度 |
| 热震稳定性 (水冷) | 15 ~ 25 次 | 1100 ℃ ↔ 水冷循环次数 |
| 加热永久线变化率 | ≤ 0.5 % | 重烧 10 h 线收缩或膨胀 |
| 显气孔率 | 18 ~ 28 % | 反映烧结致密程度 |
| 热导率 | 0.8 ~ 1.2 W/(m·K) | 较低，有利于保温 |
| 最高使用温度（铝液工况） | ≤ 1200 ℃ | 适配常规铝合金、铜合金熔炼温度 |

抗铝合金熔体侵蚀能力优于普通酸性硅质耐火材料，在不同炉部位可依据工作条件选配合适的颗粒级配与结合剂 [[S15,S4]]。

### 在压铸中的应用
死烧粘土基耐火材料在压铸车间铝/铜合金熔炼与转注系统中承担关键内衬功能 [[S5,S4,S15]]：

- **熔铝炉/保温炉内衬**：炉底、炉壁、熔解室的核心工作层，采用抗铝渗透粘土砖或捣打料，隔离铝液与炉壳；
- **转移浇包耐火衬里**：用作铝液转运、除气处理时接触金属的衬层；
- **定量炉隔层与流槽衬里**：保证铝液平稳流动，防止冲刷侵蚀；
- **除气转子涂层**：以涂料形式涂覆在旋转除气装置表面，增强抗蚀寿命。

内部资料《压铸技术手册》中给出了铝合金熔炼炉分层耐火材料结构示意，其中**抗铝渗透粘土砖层**即为死烧粘土基耐火材料在压铸中的典型应用 [[S6]]。

## 死烧粘土基炉衬失效与压铸件缺陷
### 铝液‑耐火材料反应
死烧粘土中的主要物相莫来石和游离方石英会与熔融铝发生热力学自发反应 [[S4,S15]]：

\[
4\mathrm{Al} + 3\mathrm{SiO_2} = 2\mathrm{Al_2O_3} + 3\mathrm{Si}
\]

\[
3(3\mathrm{Al_2O_3} \cdot 2\mathrm{SiO_2}) + 8\mathrm{Al} = 13\mathrm{Al_2O_3} + 6\mathrm{Si}
\]

反应生成刚玉（Al₂O₃）和游离硅（Si）。刚玉沉积层可暂时阻止反应深入，但炉衬的周期性热冲击和机械刮擦会使表面层剥离，导致反应反复进行 [[S15]]。

### 对压铸件的直接影响
炉衬长期服役后产生的剥落碎屑进入铝液，将导致：

- **硬质点缺陷**：脱落的死烧粘土颗粒作为高硬度夹杂被裹入铸件，加工时崩刃或表面划伤；
- **硅酸盐夹杂缺陷**：熟料颗粒与铝液反应生成硅酸盐类夹杂，降低铸件延展性；
- **合金成分超标**：反应生成游离 Si 溶入铝液，可能使铝合金中硅含量超出规格上 [[S4,S15]]。

上述缺陷与炉衬材料中死烧粘土的含量、烧结质量以及游离 SiO₂ 比例直接相关，炉衬选材应优先采用低游离 SiO₂、高莫来石含量、显气孔率适宜的致密死烧粘土制品 [[S15]]。

## 与其它耐火材料的对比
与高铝砖（Al₂O₃ ≥ 48 %）、碳化硅、莫来石质浇注料等相比，死烧粘土质耐火材料的特点如下：

- **成本**：死烧粘土原材料丰富，制备工艺成熟，单位成本显著低于白刚玉、碳化硅和纯莫来石材质；
- **抗铝侵蚀**：优于普通酸性硅砖，但不及高铝砖（>60 % Al₂O₃）或碳化硅复合料；在 1200 ℃ 以下工况，死烧粘土可满足大部分压铸熔铝需求 [[S15,S4]]；
- **易产生游离硅**：死烧粘土中不可避免存在少量游离 SiO₂，长时间接触铝液会生成 Si，而高 Al₂O₃ 系或碳化硅系对铝液更惰性 [[S15]]；
- **应用定位**：适合用作中小型压铸车间的熔炼与保温炉工作层，以及浇包、流槽等辅件内衬，在重载、高通量或高锌镁铝合金场景则建议升级为更高铝含量或特种耐火材料 [[S15]]。

## 相关关系
- **枯砂[焦砂]**：与死粘土同为旧砂循环体系中的失效组分，枯砂指失去烧结能力的砂粒 [[S1,S3]]。
- **耐火粘土/高岭土**：死烧粘土的原料，经高温煅烧后转化为熟料 [[S9,S17]]。
- **高铝砖/莫来石浇注料**：在铝合金熔炼炉中属死烧粘土的升级替代品，适用于更高使用温度和更苛刻的抗侵蚀要求 [[S15]]。
