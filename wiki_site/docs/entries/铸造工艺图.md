## 定义与标准术语

**铸造工艺图**（foundry molding drawing）是铸造行业特有的指导性技术文件，用于表示铸型分型面、浇冒口系统、浇注位置、型芯结构尺寸、控制凝固措施（冷铁、保温衬板）等的图样 [[S19,S14]]。该图规定了铸件的最终毛坯形状尺寸、基础生产方法和核心工艺过程，是后续设计和制造全部铸造工艺装备的首要输入依据 [[S4,S8]]。

铸造工艺图可按规定的工艺符号或文字标注在铸件图上，也可独立绘制为单独的工艺图 [[S19]]。在英文铸造工程领域，与"铸造工艺图"完全匹配的标准术语为 **foundry molding drawing**；**casting process drawing** 和 **foundry process drawing** 为行业内通用的等效常用表达 [[S19]]。需注意，**casting layout drawing** 在英文工业场景中通常指冲压或模具设计领域的排样图，不属于铸造工艺图的对等术语 [[S1]]。

## 分类

铸造工艺图可从以下三个维度进行分类 [[S26,S33,S39,S16]]：

| 分类维度 | 类别举例 |
|---------|---------|
| 按铸造方法 | 砂型铸造工艺图、重力铸造工艺图、压力铸造工艺图、低压铸造工艺图等 |
| 按合金类别 | 铝合金铸造工艺图、锌合金铸造工艺图、镁合金铸造工艺图、铸铁/铸钢铸造工艺图等 |
| 按图纸用途 | 工艺方案图、浇注系统图、工序过程图、工装指导图等 |

不同类别铸造工艺图存在明确的内容侧重差异 [[S11,S12,S4,S16]]：

- **砂型铸造工艺图**：侧重完整标注砂芯数量、芯盒参数、砂箱规格等砂型工装信息；
- **压铸工艺图**：侧重标注压射比压、内浇口截面积、溢流槽排布、抽芯机构位置等压铸专属参数；
- **重力铸造工艺图**：侧重冒口补缩系统、金属型温度分区控制参数标注；
- **工艺方案图**：仅标注核心宏观工艺决策，无需细化全部尺寸；
- **浇注系统图**：单独放大标注浇道各段的精确尺寸与角度。

## 绘制形式与符号

JB/T 2435 标准规定了铸造工艺图的两类标准绘制形式 [[S7,S4,S29]]：

| 形式 | 绘制方式 | 颜色规定 | 适用场景 |
|------|---------|---------|---------|
| **甲类** | 在产品蓝图上叠加标注 | 红、蓝两色 | 单件小批生产，直接指导制造 |
| **乙类** | 纯墨线单独绘制 | 全墨线符号 | 大批生产，便于长期保存和复制交流 |

标准共收录 **24 种常用工艺符号**及对应表示方法，包括分型面、分模面、机械加工余量、不铸出孔和槽、工艺补正量、冒口、出气孔、砂芯边界、冷铁等。不常用的工艺符号可由各生产企业自行补充规定 [[S29,S7]]。

## 核心构成要素

铸造工艺图可标注的核心要素及其作用如下表所示 [[S4,S8,S37]]：

| 要素 | 作用 |
|------|------|
| 浇注位置 | 明确金属液充型时铸件在铸型内的摆放方位 |
| 分型面/分模面 | 确定铸型开合拆分界面 |
| 加工余量 | 预留后续机械切削去除的材料厚度 |
| 拔模斜度/起模斜度 | 降低起模/脱模过程铸件与模具的刮蹭损伤 |
| 砂芯/型芯结构参数 | 定义型腔内部不可直接出模的复杂空腔成型方案 |
| 浇口/冒口/溢流槽结构尺寸 | 控制金属液充型路径与补缩排渣效果 |
| 冷铁布置 | 加速铸件局部热节区域冷却避免缩松 |
| 铸造收缩率 | 补偿金属液凝固冷却过程的尺寸收缩量 |
| 芯头形式与间隙 | 保障砂芯定位稳固且下芯顺畅 |
| 工艺补正量/反变形量 | 抵消铸造过程铸件翘曲变形导致的尺寸偏差 |
| 技术说明 | 统一标注全图通用的圆角、斜度、工艺规范 |

并非上述所有内容都必须在每张铸造工艺图上完整标注，具体内容的取舍取决于铸件的生产批量、产品性质、造型制芯方法、铸件材质和结构尺寸等因素 [[S4,S37]]。

## 与相邻工艺文件的边界

### 铸件图（毛坯图）

铸件图是专门反映铸件实际毛坯形状、尺寸和验收技术要求的独立图样，是铸件成品验收及后续机械加工工装设计的法定依据 [[S19,S14]]。其专属标注内容包含切削余量、工艺余量、铸件尺寸公差、加工基准、热处理规范、铸件全项验收技术条件等 [[S19,S3,S16]]。

铸造工艺图聚焦铸型制备相关的工艺定义要素，不直接输出铸件完整的最终毛坯尺寸集 [[S19]]。单件小批生产时，铸造工艺图可直接用于指导生产，无需额外绘制铸件图；大批大量定型产品生产中，需单独绘制铸件图 [[S31,S3]]。

### 铸造工艺卡（foundry process card）

铸造工艺卡是配套合型装配简图的**文字类工艺管理文件**，内容涵盖铸件材料与重量、造型制芯设备参数、合型规范、浇注规范、清理规范、工时定额等全流程生产管理要素，用于指导现场操作人员全工序生产作业 [[S19,S14]]。铸造工艺卡是**管理性文件**，与铸造工艺图这种**图样类文件**面向不同使用场景。

### 模具图（模样图、芯盒图、压铸模图等）

模具图是可直接投入机械加工生产的正式工程图样，所有加工尺寸、配合公差、装配关系均为精确可直接加工的数值，是铸造工艺图下游产出的工艺装备类工程成果 [[S6,S31]]。铸造工艺图属于顶层工艺方案定义文件，不直接输出模具零件的加工尺寸和形位公差要求。

### 铸造工艺流程图

铸造工艺流程图属于面向整个铸造车间的通用宏观流程文件，作用是梳理从金属熔炼到铸件清理的全生产工序流转顺序，完全不涉及单个铸件的具体专属工艺参数标注 [[S21]]。铸造工艺图则是单款铸件专属的工艺图样。

## 在工艺链中的角色

铸造工艺图通常由具备产品工艺匹配能力的铸造/压铸工艺工程师牵头绘制，绘制环节处于产品开发早期的工艺定型阶段 [[S31,S16,S8]]。其在生产流程中的角色因生产批量而异 [[S31]]：

- **单件、小批量手工造型场景**：铸造工艺图直接叠加绘制在原产品零件图上，直接用于指导木模制造、造型生产和铸件检验，无需额外绘制铸件图；
- **大批量定型产品生产场景**：试制阶段完成的铸造工艺图经过多轮工艺参数调整后，将作为金属模具设计的法定输入依据；正式量产阶段往往以最终模具加工图样替代原始铸造工艺图直接指导生产 [[S31]]。

在压铸全工艺链中，铸造工艺图起到核心枢纽传导作用 [[S22,S30,S24]]：

1. 作为上游铸件设计端输出的产品尺寸公差与性能要求的承接载体，将产品设计约束转化为可落地的工艺边界条件；
2. 直接作为下游压铸模具结构设计的输入文件，为分型面确定、浇排系统排布、抽芯机构设计提供明确依据；
3. 工艺图中预设的浇注位置、内浇口速度、充型路径等参数，可直接导入 AnyCasting、ProCAST 等 CAE 仿真软件开展流动-温度场耦合模拟验证，提前预判卷气、缩松等缺陷；
4. 经 CAE 仿真迭代优化后的最终工艺参数，可直接映射到压铸车间现场的压射比压、模具温度、保压时间等生产参数设定环节，大幅减少试模迭代次数。

## 标注规范与标准依据

### 尺寸公差标注

GB/T 6414《铸件 尺寸公差》规定了铸造公差在图样上的三种标注方式 [[S34,S35]]：

| 标注方式 | 示例格式 |
|---------|---------|
| 统一公差代号标注 | "一般公差GB/T6414—CT12" |
| 限制错型时补充最大错型值 | "一般公差GB/T6414—CT12—最大错型1.5" |
| 特定尺寸单独标注个别公差 | "95±3"或"200+5/-3" |

机械加工余量（RMA）标注分为两种形式 [[S34,S43]]：

- **统一标注**：如轮廓最大尺寸 400~630 mm、余量等级 H、余量值 6 mm、公差等级 CT12 的铸件标注为 "GB/T6414-CT12-RMA6(H)"；
- **特定表面单独标注**：在图样特定表面上标注非标准加工余量。

### 通用绘制规范

铸造工艺图绘制须符合 JB/T 2435 规范 [[S37,S31]]：

- 相同尺寸的铸造圆角、相同角度的拔模斜度无需在单张视图重复标注，统一写入技术条件即可；
- 工艺尺寸集中标注避免分散重复，不得遮挡原零件图上的产品基准尺寸；
- 砂芯边界线与零件轮廓、加工余量线重合时可省略砂芯边界重合段，确保图面清晰。

### 国际标准参考

ISO 10135:1994《技术制图 模压件、铸件和锻件的简化表示》作为国内铸造尺寸公差体系等效参考的国际标准，被收录进 GB/T 6414 标准的参考文献体系 [[S38]]。ASTM 相关规范规定压铸件未明确标注公差的尺寸需符合 ADCI Product Standard 系列要求，图纸标注的所有特殊检测要求需在采购订单或零件图样中明确说明 [[S25]]。

## 数字化工具关联

当前压铸行业 CAD/CAE 协同的主流工作流为 [[S13,S23,S27]]：工艺人员先在 NX、SolidWorks 等三维 CAD 软件完成铸件结构分析、型腔布局、浇注排溢系统设计，输出三维工艺模型后导出 STL/IGES/STEP 等中性文件导入 AnyCasting、Magmasoft、FLOW-3D CAST 等 CAE 软件完成充型、凝固仿真，缺陷预测结果反馈回 CAD 端优化工艺参数，最终输出标注完善的三维或二维铸造工艺图。

传统独立的铸件设计、凝固模拟、模样加工系统正在逐步向三维模型全流程贯通方向发展，部分头部压铸企业已实现工艺设计数据全三维化，但全行业完全用三维标注替代传统二维工艺图仍处于推广过渡阶段 [[S13,S2]]。
