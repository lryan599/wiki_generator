铸造用原砂是砂型（芯）的骨架材料，其颗粒尺寸直接影响砂芯的强度、透气性、表面质量以及铸件最终缺陷的产生。

## 概述
在铸造和压铸工艺中，**原砂颗粒尺寸**特指未经涂覆粘结剂的原砂（基砂）颗粒的平均和可控线性尺寸，通常通过筛分分析确定[[S20]]。它是一个核心的参数，与型砂（已混合粘结剂的砂）的颗粒尺寸不同：后者因砂粒表面包裹了粘结剂膜，其等效粒径会大于对应的原砂粒径[[S42]]。原砂颗粒尺寸的精确控制是获得高质量铸件的先决条件。

## 定义与相关概念辨析
原砂颗粒尺寸领域涉及多个易混淆的术语，其精确定义如下：

| 术语 | 定义 | 备注 |
| :--- | :--- | :--- |
| **原砂粒度 (原砂颗粒尺寸)** | 由筛析或其他方法确定的原砂颗粒的平均和控制线尺寸[[S20]]。 | 反映颗粒的实际物理尺寸。 |
| **原砂细度 (AFS平均细度)** | 一个等效理论值，假设所有砂粒尺寸相同，使其总表面积与实际砂粒总表面积相等。砂粒越细，AFS值越大[[S20,S3]]。 | 无量纲数值，综合反映比表面积，不直接代表具体线性尺寸或分布[[S27,S42]]。 |
| **型砂粒度** | 混合粘结剂、添加剂后，表面包裹着粘结剂膜的复合砂粒的等效粒径[[S42]]。 | 其尺寸大于配制它的原砂。 |

## 分类与表征方法
原砂颗粒尺寸的核心表征方式包括粒度分布表示和平均细度计算。

### 筛分分析
标准的筛分流程为：取去除泥分并烘干的50g±0.01g原砂试样，放入按筛孔尺寸从大到小叠放的标准筛组顶部，在筛砂机上振动15分钟（电磁微振式筛砂机振幅调至3mm），然后分别称量各筛及底盘的残留砂质量，换算为质量百分比[[S27,S10,S49]]。

### 三筛集中度与粒度表示法
中国标准体系使用相邻筛号残留砂百分比来命名粒度。当残留量最大的相邻三筛砂粒质量百分比总和**不小于75%**时，即以此三筛的首尾筛号表示该原砂的主要粒度组成，如 **50/100** 或 **40/70** [[S3,S27,S47]]。若相邻四筛残留量总和不小于85%，则为四筛集中度[[S3,S4]]。此方法直观地反映了颗粒的集中程度。例如，代号 **50/70** 表示该砂的颗粒主要集中在50目、70目、100目这三个筛网上，且相邻三筛的残留量总和不小于75%[[S26]]。

### AFS平均细度 (GFN)
这是国际上广泛采用的综合性指标。计算公式为： **AFS细度 = Σ(各筛残留砂质量百分比 × 相应的细度因数) / Σ(各筛残留砂质量百分比)**，结果经修约取整[[S3,S4]]。该值反映单位质量原砂的总表面积，AFS值越大，代表砂粒总体越细[[S17]]。大部分工业铸造用砂的AFS细度在40~220之间[[S55]]。然而，单独使用平均细度可能掩盖粒度分布的差异，因此中国标准通常在规定主要粒度组成的同时，也要求提供平均细度值[[S27,S47]]。

### 不同标准体系的核心差异

| 标准体系 | 核心表征参数 | 关键规定 | 特点 |
| :--- | :--- | :--- | :--- |
| **中国 GB/T 系列** | 主要粒度组成（三筛/四筛首尾筛号）+ 平均细度 (AFS) [[S27,S47,S26]] | 三筛≥75%，四筛≥85%；牌号中可带Q（前筛>后筛）或H（后筛>前筛）标识[[S27,S10]]。 | 兼顾粒度分布集中度和平均粗细。 |
| **美国 AFS 标准** | AFS平均细度 (GFN) [[S55,S17]] | 无强制三筛占比阈值要求，不对筛分布规则做强制限定[[S55]]。 | 侧重于整体比表面积的衡量，简单但可能忽略分布。 |
| **ASTM C778** | 指定牌号标准砂的精确粒度区间 [[S62]] | AFS 50/70测试砂颗粒尺寸范围为-50/+70目，即230~270μm[[S62]]。 | 仅用于特定测试场合的砂样规定，不作为全系列原砂分级标准。 |

不同标准体系的测试用筛规格基本一致或历史上有对应关系。例如，GB/T9442与AFS规定的筛孔尺寸基本相同，早期中国JB435-63的筛号规格也与AFS标准完全一致[[S11,S15]]。ASTM E11标准的筛系为AFS检测指定用筛，其相邻筛号筛孔尺寸比例为1.414[[S38]]。

## 工艺作用
原砂颗粒尺寸是决定砂芯综合性能及最终铸件质量的关键因素。

*   **透气性与气孔缺陷**：原砂粒度越粗，砂粒间孔隙体积越大，透气性越高[[S44,S53]]。相反，微粒（细粉）含量过高会导致砂型/砂芯的透气性急剧下降，气体排出不畅，易引发铸件内部的气孔缺陷[[S56,S16,S30]]。
*   **砂芯强度**：在固定树脂加入量下，中等偏细的合理粒度能使砂芯强度最优。粒度过粗，砂粒间接触点少、间隙大，粘结强度降低；粒度过细，砂粒总表面积过大，会导致树脂膜厚度不足，同样使强度下降[[S21,S51,S9]]。
*   **表面质量**：较细的颗粒能更好地复制模具内腔的精细特征，获得更低的铸件表面粗糙度。但这需在保证透气性的前提下进行，否则可能引发其他缺陷[[S39,S6,S16]]。
*   **尺寸精度**：粒度分布均匀、级配合理的原砂可确保砂芯的尺寸稳定性，避免射不实或局部疏松，从而保证铸件内腔的尺寸精度[[S41,S60]]。
*   **溃散性**：粒度过细的砂芯溃散性更差，铸件凝固后不易清理；粒度过粗则砂芯内部孔隙大，高温下金属液易渗入，并可能因退让性不足阻碍铸件收缩，导致热裂[[S16]]。

## 与其他参数的关系
原砂颗粒尺寸与造型材料的多种性能存在复杂的关联。

*   **透气性**：颗粒尺寸是决定透气性的首要因素。粒度越大（筛号越小），透气性越高；反之，透气性迅速下降[[S44,S53]]。铸钢件倾向于使用较粗的砂以获得≥150的透气性，而铸铁件使用中等粒度砂，透气性通常保持在80~110[[S58,S50]]。
*   **强度与粘结剂消耗**：粒度越细，比表面积越大，达到同等粘结强度所需的粘结剂（如树脂）用量越高[[S29,S22]]。在粘结剂用量固定时，粗细适中的颗粒能获得最高的砂芯强度[[S21,S51]]。
*   **耐火度**：粗粒度原砂的耐火度高于同种矿物成分的细砂和尖角形砂。杂质含量低的粗砂，其GFN为30~45时，耐火度可达约1650℃[[S53,S50]]。
*   **角形系数**：角形系数（表征颗粒形状偏离球形的程度）与粒度共同作用。在同等粒度下，角形系数越大（颗粒越尖角），砂粒比表面积越高，透气性越低，所需的粘结剂也越多[[S29,S22]]。采用粒度分布集中、角形系数小的原砂有助于提升型砂流动性，获得均匀的紧实效果[[S58]]。

## 缺陷与影响
原砂颗粒尺寸选择不当是导致多种铸造缺陷的直接原因。

| 缺陷类型 | 成因机制 | 与颗粒尺寸的关系 |
| :--- | :--- | :--- |
| **机械粘砂** | 金属液在静压力（特别是压头超过500mm时）下，通过毛细管渗透作用渗入砂粒间隙，凝固后形成金属与砂粒的机械混合物，呈海绵状[[S35,S18,S30]]。 | **粒度过粗**导致砂粒间孔隙过大，加剧金属液渗透。 |
| **化学粘砂** | 金属液表面氧化物（如FeO）与砂中的SiO₂反应，生成低熔点硅酸盐化合物（如铁橄榄石），牢固粘附在铸件表面，硬度极高，断口呈连续蜂窝状[[S18,S43,S57]]。 | **粒度过粗**，同时存在大量细小粉尘、铸件局部过热时，会促进该化学反应。 |
| **气孔** | 砂芯受热后产生的大量气体无法通过透气性不足的型芯砂及时排出，侵入金属液并残留其中形成气孔[[S56,S30,S7]]。 | **粒度过细、细粉含量过高**会严重降低透气性，阻碍排气。 |
| **表面粗糙** | 粗大的砂粒在铸件表面直接留下相应的凹凸痕迹[[S6,S46]]。 | **粒度过粗**直接导致复制精度差，表面粗糙。 |
| **冲砂** | 高速流动的高温金属液冲刷剥离砂型/芯表面强度较低的砂粒，在铸件对应位置形成缺陷[[S8,S61]]。 | **粒度过粗**时，砂芯表面的局部抗冲刷能力不足。 |

## 典型范围与应用
压铸及砂铸中，原砂的颗粒尺寸选择需根据合金类型和工艺要求进行匹配。

**常用压铸砂芯原砂的推荐粒度范围**：油砂芯常用40/70目、50/100目、70/140目；铬铁矿砂、锆砂常用粒度范围为**0.15~0.30mm**，对应约**50~100目**；消失模工艺用宝珠砂的粒径覆盖**10~300目**等较宽范围[[S39,S19,S33]]。

**不同合金类型的差异化选择**：铸钢件推荐用粒度集中在**40/70目**、细粉含量≤1%的硅砂；铸铁件推荐用**40/70目**左右的天然硅砂；铸铝、镁等非铁合金件在保证透气性的前提下可选用粒度偏细的砂料，以获得更高的表面光洁度，铸铝件常用**40~70目**[[S28,S54]]。

**特种工艺要求**：低压及差压铸造用面层砂推荐采用均匀一致的**45~75目**，以避免砂粒动能过大或过小导致的涂层穿透或附着不牢问题，从而影响铸件表面粗糙度[[S59]]。
