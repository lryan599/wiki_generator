## 定义

电铸是利用金属电沉积原理，在母模（芯模）表面沉积金属，之后将沉积层与母模分离，从而制造或复制构件的精密电化学成形工艺[[S7,S38]]。在直流电场作用下，电铸液中的金属阳离子向作为阴极的母模迁移并还原为金属原子，逐步堆叠为所需厚度的沉积层，整个过程金属离子与母模表面无间隙接触且不对母模造成损伤[[S20,S8]]。

电铸与常规电镀在核心目的、沉积层厚度及结合要求上存在根本差异[[S8,S10,S37]]。

| 对比项目 | 电铸 | 常规电镀 |
|---|---|---|
| 工艺目的 | 复制型面，获得可独立使用的金属制品 | 表面装饰、防腐蚀、耐磨改性 |
| 典型层厚 | >0.05 mm，常用0.05～数毫米，可达厘米级 | 1～50 μm |
| 结合要求 | 最终沉积层须与母模脱离，成为独立制品 | 镀层与基体牢固结合，不可分离 |

## 工作原理

电铸的电沉积机制包含四个连续步骤[[S20]]：

1. 电铸液中的水化金属离子向阴极母模表面迁移；
2. 水化膜变形，离子被吸附到母模表面活化位点；
3. 金属离子获得电子，还原为金属原子；
4. 金属原子排列形成晶体晶格，完成沉积。

在电解槽中，以导电母模为阴极、待沉积金属为阳极，接通直流电源后，阳极金属不断以离子形式溶入电铸液补充消耗，阴极母模表面沉积层逐渐增厚至预定厚度后取出，将沉积层与母模分离，即得与母模型面凹凸相反的电铸制品[[S20,S32]]。

## 工艺分类

### 按沉积金属种类

| 类型 | 特性与适用场景 |
|---|---|
| 镍电铸 [[S18,S14]] | 成形清晰，复制性好，硬度35～50 HRC，抗拉强度1400～1600 MPa；适用于塑料模型腔、小型拉深模 |
| 铜电铸 [[S18,S14,S35]] | 导电性能优异，成本较低；适用于EDM铜电极、电铸镍壳加固层、非贵金属模具型腔；缺点是机械强度与耐磨性较低，易氧化不耐酸 |
| 铁电铸 [[S18,S47]] | 成本低，多用于电铸镍壳加固、磨损机械零件修复；质地偏软易腐蚀，操作有刺激性气味 |
| 镍钴合金等特种合金电铸 [[S18,S47]] | 通过合金沉积调控力学性能与耐磨性能，适配特殊工况 |

### 按供电模式

| 类型 | 特征 |
|---|---|
| 直流电铸 [[S5,S34]] | 连续直流供电，电压3～20 V可调，常用电流密度15～30 A/dm²，应用最广泛 |
| 脉冲电铸 [[S5,S23]] | 可调节脉冲宽度、间隔、峰值电流参数；相比直流电铸可获得晶粒更细小、致密度更高、针孔与内应力更低的沉积层，同时提高分散能力与深镀能力 |
| 周期换向电铸 [[S5,S23]] | 正反向交替供电，可抑制尖端树枝状毛刺，优化复杂型面铸层均匀性 |

## 工艺流程

电铸完整工艺流程主线为[[S24,S44,S49,S48]]：

**产品图样分析 → 母模设计与制造 → 母模前处理（金属母模镀脱模层 / 非金属母模导电化处理、防水、绝缘包扎） → 电沉积至预定厚度 → 衬背加固 → 脱模 → 后续精加工 → 清洗干燥 → 成品**

对于大批量生产或EDM电极制造需求，可采用二次制模路径：先制作标准母模 → 反制阴模 → 二次制模后进入后续电铸流程[[S24,S49,S41]]。

### 母模制作

母模的表面粗糙度直接决定电铸复制件的表面质量，母模表面的划痕与缺陷会被完整复现到电铸件表面[[S24,S6]]。母模设计阶段需注意[[S24,S6,S48,S2]]：
- 预留15′～30′脱模斜度，降低脱模时电铸层受力变形概率；
- 边缘采用大圆角过渡，避免尖锐尖角，抑制电流集中效应导致尖端沉积增厚、凹槽处偏薄的不均匀缺陷；
- 母模应比制品长3～5 mm，以备切除电铸后两端粗糙沉积物。

母模材料可选用金属（钢、铝、低熔点合金等）或非金属（环氧树脂、石膏、石蜡、有机玻璃等）[[S24,S48]]。

### 导电化处理与脱模层

非金属母模需进行导电化处理，常用方法包括[[S24,S35,S14,S15]]：
- 涂刷混有导电金属粉（铜、银）的导电漆；
- 真空蒸镀或阴极溅射沉积极薄金属膜；
- 化学镀银或化学镀铜生成连续导电层。

导电层的连续性直接影响电沉积初期的均匀形核质量，局部缺失会导致对应位置无法正常沉积形成缺陷[[S24]]。

金属母模则需镀脱模层（一般为8～10 μm厚硬铬层或钝化处理），以便电铸后顺利分离[[S6,S24]]。

### 电沉积

沉积过程的电铸液组分、温度、pH值、电流密度和搅拌强度协同影响电结晶质量。合理控制参数可将残余应力控制在较低水平，避免电铸件变形、开裂；简单工件1 mm厚度沉积耗时3～4 h，复杂工件可达数十小时[[S21,S19,S47,S32]]。

### 脱模与后处理

脱模方式需根据母模材料适配：机械脱模、热胀冷差分相脱模、低熔点母模熔化脱模等，不合适的脱模外力会导致薄型、大尺寸电铸件塑性变形，破坏型面精度[[S46,S21,S23]]。

后续精加工仅限于电铸件的非成形面尺寸修整，不可对已复制的高精度型面进行过量加工[[S44,S49,S48]]。

## 关键工艺参数

电铸过程的参数控制直接影响沉积层质量与内应力水平[[S21,S19,S47]]。

| 参数 | 影响规律 |
|---|---|
| 镀液温度 | 温度升高可提高沉积速率，但过高会使结晶粗大、抗拉强度下降[[S3,S45]]；镍电铸：金属母模75～80 ℃，非金属母模45～55 ℃[[S14]] |
| 电流密度 | 过高导致结晶粗糙、强度降低；直流电铸常用15～30 A/dm²[[S5,S39]]；起始阶段宜采用较低密度（0.5～1 A/dm²），逐渐升至正常值[[S14,S23]] |
| pH值 | 镍电铸液宜控制5～5.6[[S14]]；铁电铸液1.4～1.7[[S49]]；偏离范围影响沉积层应力与硬度 |
| 搅拌方式 | 降低浓差极化，允许使用更高电流密度，提升沉积速率与均匀性；常用桨叶搅拌或循环过滤系统[[S35,S5]] |
| 添加剂 | 含硫有机添加剂可提升硬度，但超过200 ℃将导致沉积层脆化[[S13]] |

## 在压铸领域中的应用

电铸在压铸领域主要应用于制造锌合金压铸的型腔镶件及EDM铜电极[[S13,S30,S17]]。此类应用充分利用了电铸工艺极高的复制精度与低表面粗糙度优势[[S11]]。

### 电铸型嵌块制造

典型工艺流程为[[S13,S30]]：
1. 按目标压铸件形状制作环氧树脂母模；
2. 母模表面喷涂约1 μm厚银层实现导电化；
3. 在添加氨基磺酸钴的氨基磺酸镍镀液中沉积至3～5 mm厚度；
4. 切除边缘多余沉积层后与母模分离；
5. 背面以约50 mm厚电沉积铜层或金属喷涂铝-硅合金、铜、青铜进行背衬加固；
6. 嵌入模套内完成装配。

针对压铸工况的加固装配方案更为严谨：将电铸镍质模腔置入预制加热管，外部加装铝或钢支撑结构，模腔背板与支撑之间填充特殊混凝土、陶瓷或矿石填充树脂；这类填充材料热膨胀系数与镍、钢几乎一致，可承受压铸高温高压工况[[S51]]。

### 常规未掺杂电铸镍层硬度约30 HRC；在氨基磺酸镍镀液中添加15%氨基磺酸钴可获得镍钴合金沉积层，硬度提升至44 HRC，无需含硫添加剂，避免超过200 ℃服役时脆化[[S13]]。

### 表面质量与脱模性

电铸制得的压铸模具零件表面粗糙度Ra最低可达0.05 μm，无需额外抛光即可满足高光面压铸成型要求[[S11]]。极低的表面粗糙度使锌合金低压压铸脱模阻力更小，可有效降低粘模概率，减少脱模剂喷涂频次，提升生产节拍稳定性[[S11]]。

## 优势与局限性

### 核心优势

- **复制精度极高**：可精准复现母模表面微米级细节，表面粗糙度Ra≤0.1 μm[[S11,S6]]；
- **经济效益显著**：同一母模可重复使用制备多组一致性极高的镶件；与传统机械加工+EDM制模相比，制模周期可缩短30%～60%，制造成本可降低40%～80%[[S11]]；
- **材料利用率高**：电铸通过阴极沉积获得型腔，几乎无切削浪费[[S11]]。

### 固有局限性

| 局限性 | 具体表现 |
|---|---|
| 沉积效率低 | 常规镍层沉积速率每日0.2～1.0 mm，制备6～8 mm工作层需约8天[[S11,S42]] |
| 厚度分布不均 | 尖角、深窄狭槽处沉积不足，硬镍材质存在“角脆弱”特性，尖内角沿平分线方向易产生微裂纹甚至断裂[[S9]] |
| 内应力控制难 | 不合适的工艺参数直接导致电铸件变形报废[[S11,S9]] |
| 工作层厚度有限 | 常规有效工作层厚度3～8 mm，难以制备大尺寸整体型腔零件[[S11,S18]] |
| 材料种类有限 | 实用可沉积金属仅镍、铜、镍钴合金等少数几种，无法量产难熔金属或特种热作钢[[S47,S11]] |
| 工作温度受限 | 常规硬镍电铸层最高允许工作温度仅约300～400 ℃，远低于铝合金压铸600 ℃以上型腔表面温度要求；含硫添加剂改性沉积层在超过200 ℃工况下快速脆化出现热疲劳裂纹，仅适配锌合金等低熔点合金压铸[[S9,S1]] |

## 与常规制模方式的对比

| 对比维度 | 电铸制模 | 机械加工+EDM制模 |
|---|---|---|
| 表面加工效果 | Ra≤0.1 μm镜面级，几乎无需抛光 | EDM加工面残留放电纹，需长时间人工抛磨 |
| 复杂微细特征 | 完整复刻微细纹路、窄槽等超精密特征 | 需额外制作复杂电极，难以成型某些特征 |
| 制造周期 | 制备6～8 mm镍层需约8天 | 需额外耗时制作对应铜电极，多道工序总周期较长 |
| 材料利用率 | 阴极沉积获得型腔，近乎无浪费 | 机械切削去除大量坯料，材料利用率低 |
| 受载性能 | 存在内应力，不能承受强冲击载荷 | H13钢经热处理后抗冲击、耐热疲劳性能远优于电铸层 |

[[S11,S39,S36]]

耐热疲劳性能是电铸模具在压铸应用中的核心短板：电铸硬镍类模零件耐热疲劳性能明显低于经热处理的H13等热作模具钢，仅适配锌合金等压铸温度不超过400 ℃的低熔点合金压铸生产，无法胜任铝合金、镁合金等更高温的量产工况[[S9,S1]]。
