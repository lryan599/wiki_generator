## 概述

“过烧”是铝合金压铸及后续热处理过程中一种不可逆的冶金缺陷。当加热温度超过合金内低熔点共晶相的熔点，导致低熔点共晶与晶界发生复熔，即形成过烧。过烧一旦发生，组织损伤无法通过后续热处理或塑性加工消除，铸件即判定为废品 [[S3,S5,S15,S16,S18]]。

压铸场景下，过烧与熔炼温度控制直接相关。以ADC12铝合金为例，当熔炼温度高于750℃时，铁在铝液中的溶解速度急剧升高，该工况被判定为“铝合金过烧” [[S27]]。

过烧与“过热”具有本质区别——过热仅是高温下保温导致的晶粒粗化，不存在低熔点相熔化；而过烧的核心判据是晶界及晶粒交汇处发生复熔，形成复熔共晶球、复熔三角形等特征组织 [[S4]]。

## 分类与分级

### 按发生位置分类

过烧按发生位置及形成特征可分为三类 [[S14,S25]]：

| 类别 | 特征 |
|------|------|
| 表面过烧 | 过烧特征仅发生在铸件表层，多由合金液充型时表层接触温度异常型腔或表面局部过热导致 |
| 内部过烧 | 过烧特征分布在铸件内部厚大热节区域，由充型后内部凝固阶段局部温度超过低熔点共晶熔点导致 |
| 晶界过烧 | 仅在合金晶粒边界位置出现低熔点相复熔、氧化，是压铸过烧最普遍的微观表现形式 |

### 按严重程度分级

国内行业标准 **JB/T7946.2—1999《铸造铝硅合金过烧》** 将过烧组织划分为五个等级 [[S14,S24,S25,S31]]：

| 级别 | 显微组织特征 |
|------|------------|
| 正常组织 | 组织均匀，无过烧特征 |
| 过热组织 | 共晶硅边角圆滑、粗化且聚集长大，是过烧的前奏和征兆 |
| 轻微过烧组织 | 出现少量复熔共晶球或轻微晶界复熔特征 |
| 过烧组织 | 过烧特征明显，晶界复熔加宽，出现三角形复熔物 |
| 严重过烧组织 | 过烧特征大量出现，晶界严重复熔粗化和平直，低熔点共晶大量熔化和聚集 |

## 形成机理

压铸铝合金中除α-Al基体外存在多种不同熔点的低熔点共晶相。当合金熔炼或充型过程中温度持续升高时，熔点最低的共晶相率先熔化；温度进一步升高，其他次低熔点共晶相继复熔 [[S1,S2,S8,S15]]。

以2A12变形铝合金为例，其含有两种主要共晶：
- α(Al) + CuAl₂（熔点548℃）
- α(Al) + CuAl₂ + Al₂CuMg(S相)（熔点507℃）

三元共晶熔点显著低于二元共晶，因此当合金在较高温度热处理时，三元共晶首先熔化，其熔化温度（507℃）即为2A12合金的过烧触发温度 [[S3,S5,S15]]。

ADC12压铸Al-Si合金的固液区间为539~590℃，其过烧触发温度为固相线温度539℃，即当熔炼或充型温度超过此温度时，合金内的低熔点多元共晶相开始复熔 [[S21]]。

熔化的低熔点相沿晶界分布过程中，Mg、Al等活性合金元素与气氛中的氧发生反应生成氧化物夹杂，后续凝固后即形成晶界复熔、氧化物夹杂分布的典型过烧组织 [[S2,S8]]。

## 外观与微观特征

### 宏观特征

过烧程度较轻时，铸件表面色泽变暗、失去金属光泽，出现暗斑；过烧严重时，铸件表面发黑，呈现深灰至黑色，伴随局部无规律起泡、结瘤甚至挠曲变形和开裂 [[S10,S14,S16,S22]]。部分压铸合金元素（如Mg、Zn）会发生烧损，严重过烧时表面甚至析出低熔点共晶液形成的“汗珠”状凸起 [[S10,S14]]。

不同系列铝合金的过烧断口存在特征差异 [[S22]]：

| 合金系 | 过烧断口特征 |
|--------|------------|
| Al-Si系 | 断口黑色，晶粒粗大 |
| Al-Cu系 | 断口可见白色亮斑 |
| Al-Mg系 | 断口可见黄色亮斑 |

### 微观金相判定特征

判定过烧须以显微组织特征为依据。依据相关标准，存在以下特征之一即可判定为过烧 [[S3,S5,S8,S15,S16]]：

1. **复熔共晶球**——独立的球状复熔共晶体；
2. **晶界局部复熔加宽**——晶界局部熔化后出现晶界加宽、晶界“发毛”现象；
3. **复熔三角形**——三个晶粒交叉的三晶交界处出现的复熔三角形区域；
4. **梯田花样**——电子显微镜下，与复熔产物相接触的α-Al基体表面呈现“梯田花样”枝晶露头结晶台阶，表明该区域已发生过复熔。

## 主要影响因素

过烧的形成受多因素耦合影响，可归纳为以下维度 [[S9,S11,S20,S23]]：

| 影响类别 | 关键因素 | 作用机制 |
|---------|---------|---------|
| 合金成分 | 低熔点共晶相含量、杂质元素（如Fe、Si）占比 | 低熔点共晶相含量越高，过烧温度越低；Mg、Si含量波动影响三元共晶反应温度和过烧敏感性 |
| 浇注温度与保温时间 | 熔炼温度、保温时长 | 温度超过合金固相线即触发低熔点相熔化；保温时间过长导致低熔点相聚集 |
| 模具温度场 | 模具预热温度、连续工作温度、模温均匀性 | 模温过高使型腔局部温度异常，加速表面过烧；温差过大致使局部过热 |
| 脱模剂 | 类型、兑水倍率、喷涂量及均匀性 | 过量或不均匀喷涂影响模温及冷却，间接引发局部热聚集 |
| 压射速度与压力 | 压射速度、充填速度 | 过高充填速度使金属液与型腔壁摩擦升温，增大表面过烧风险 |

## 对铸件性能的影响

### 力学性能

过烧对力学性能的影响随严重程度变化 [[S15,S22]]：

- **轻微过烧阶段**：部分合金因第二相固溶更加充分，抗拉强度和伸长率可能短暂小幅升高（如2A12合金），但疲劳性能和应力腐蚀性能已开始下降；
- **中等及严重过烧阶段**：力学性能出现断崖式下降。以ZAlSi7Mg合金为例，558℃出现明显过烧特征后抗拉强度和伸长率同步下降；ZAlCu5Mn合金在550℃低熔点共晶体开始复熔后力学性能急剧恶化。

7A04和6063铝合金出现过烧后，塑性指标的下降幅度远大于强度指标 [[S15]]。

### 服役性能

过烧组织存在的零件承受动载荷时，低熔点相复熔的晶界位置极易成为疲劳萌生源，大幅缩短零件整体疲劳寿命；晶界连续性被破坏后，零件的气密性和耐腐蚀性能也会同步劣化 [[S15,S22]]。

过烧组织不可逆，无法通过后续热处理或塑性加工消除，任何出现过烧的铸件均判定为绝对废品 [[S5,S15,S16,S18]]。

## 检测与识别方法

过烧检测可采用三种方法，各有适用场景 [[S7,S13,S15]]：

| 检测方法 | 方法说明 | 适用场景 |
|---------|---------|---------|
| 目视断口检查 | 依据不同合金过烧断口特征颜色（Al-Si系黑色、Al-Cu系白色亮斑、Al-Mg系黄色亮斑）快速排查 | 生产线大批次快速抽检，仅作初筛 |
| 金相显微检测 | 试样抛光后用Keller试剂或对应比例混合酸浸蚀，观察是否存在复熔共晶球、晶界复熔等特征。可依据JB/T7946.2标准精准区分过烧等级 | 过烧判定的金标准，适用于实验室全检和异议仲裁 |
| 二次重溶复检 | 将疑似过烧试样在略低于预设过烧温度下保温后快速冷却，排除常规固溶慢冷导致的晶界粗化假象，避免误判 | 区分过烧与淬火慢冷晶界析出假象 |

## 防止与改进措施

过烧的防控应覆盖工艺全链路，重点措施如下 [[S15,S17,S20,S22,S34]]：

| 措施类别 | 具体内容 |
|---------|---------|
| 浇注温度控制 | 严格控制合金浇注温度低于对应牌号过烧触发温度20~30℃ |
| 升温速率管理 | 采用梯度升温工艺（200~250℃/h升温速率）或分级加热，避免低熔点相提前熔化 |
| 仪表校准 | 定期校准控温仪表，确保炉内温度均匀 |
| 模具温度管理 | 使用模温机将模具温度稳定在对应合金推荐区间，保持模腔处于适宜温度区间是抑制过烧的核心手段 |
| 脱模剂规范 | 规范脱模剂喷涂工艺，保证膜层均匀性，避免局部冷却异常 |
| 全链路风险排查 | 合金成分管控、熔炼温度控制、模温管理三个环节为过烧风险高发节点，可通过全流程节点反向排查定位根因 |

## 与其他缺陷的关联

### 过烧与过热

过热仅是合金在高温下保温导致晶粒粗化，不存在低熔点相熔化；过烧是加热温度超过低熔点共晶相的固相线熔点后发生复熔，金相特征完全不同 [[S4]]。

### 过烧与热裂

热裂缺陷的特征是沿铸件热节区、厚薄交接处分布的不规则曲折狭缝，断面被严重氧化呈深灰色或黑色、无金属光泽，属沿晶开裂 [[S12]]。过烧则是组织熔化类缺陷，基体局部复熔后重新凝固，与热裂的撕裂类特征属性不同。

## 标准与规范

| 标准/规范 | 适用内容 |
|----------|---------|
| JB/T7946.2—1999《铸造铝硅合金过烧》 | 国内行业标准，将过烧分为正常、过热、轻微过烧、过烧、严重过烧五级，明确各级显微组织特征，为过烧程度判定的官方依据 [[S14,S24,S25,S31]] |
| NADCA《压铸缺陷排查指南》、ASTM B86 | 国际规范中将过烧归类为冶金类不合格项，最终判定需结合金相检测结果确认 [[S30]] |

过烧是压铸铝合金的致命冶金缺陷，会破坏金属组织的完整性，导致铸件力学性能显著下降，常规条件下判定为不合格废品，不允许用于服役工况 [[S18]]。
