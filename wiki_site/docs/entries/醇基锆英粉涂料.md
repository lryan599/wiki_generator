## 定义与别名

醇基锆英粉涂料（Alcohol-based zircon mould coating）属于砂型铸造用有机溶剂系快干型涂料，是以锆英粉为耐火骨料、醇类（乙醇/异丙醇）为载体，辅以粘结剂、悬浮剂等配制而成的浆状或膏状涂覆材料，涂敷后通过点燃醇类载体即可实现快速干燥固化 [[S40,S33]]。行业通用别名包括**锆英粉醇基快干涂料**、**锆英粉醇基涂料**、**酒精基锆英粉涂料**，海外行业常称其为 *Zircon alcohol quick drying foundry coating* [[S33]]。

需注意区分三个相关术语 [[S12,S34,S5,S20]]：

- **锆英石**：天然存在的硅酸锆（ZrSiO₄）矿物本体；
- **锆砂**：锆英石经选矿得到的粗颗粒砂状产品，多用于铸型面砂；
- **锆英粉**：锆砂经进一步研磨得到的细粉状产品（95%以上颗粒过270目筛），是醇基锆英粉涂料的核心耐火骨料 [[S5]]。

## 快干机理

醇基锆英粉涂料的"快干"核心在于热化学联动：以乙醇或异丙醇为主体的醇类载体在点燃后快速燃烧挥发，同时涂层内的有机粘结剂（如液体酚醛树脂）受热交联固化，无需送入窑炉烘干即可在短时间内获得具有足够强度的干燥涂层，可显著缩短生产周期、降低烘干环节能耗 [[S4,S40]]。

## 结构与组成

醇基锆英粉涂料为多组分悬浮体系，核心组分为四大类 [[S4,S5,S18,S44]]：

**1. 载体（溶剂）**
国外优先选用异丙醇（燃烧特性优良），国内多采用低成本工业乙醇，性能可满足使用要求 [[S4]]。

**2. 耐火填料**
符合JB/T 9223要求的锆英粉，要求ZrO₂质量分数＞65%，95%以上颗粒过270目筛 [[S5,S12]]。

**3. 粘结剂**
常用液体酚醛树脂和硅酸乙酯，其中液体酚醛树脂应用最广 [[S4]]。

**4. 悬浮剂**
包括有机改性膨润土、水-酒精膨润土膏、改性锂膨润土、聚乙烯醇缩丁醛（PVB）等 [[S4,S18]]。

### 典型成分配比

工业经验证配方以锆英粉为100份（质量基准），复合悬浮剂（固体）2份，PVB 0.4份，特种粘结剂3.0～4.0份，混合醇类溶剂25～28份，附加剂0.05～0.1份 [[S44]]。

标准基础配方（质量份）进一步归纳为 [[S18,S3,S44]]：

| 组分 | 质量份/比例 | 说明 |
|------|-------------|------|
| 锆英粉 | 100 | 耐火骨料，ZrO₂＞65% |
| 有机膨润土或锂膨润土 | 1～4 | 悬浮剂 |
| PVB | 0～1 | 悬浮剂/辅助粘结剂 |
| 树脂粘结剂 | 3～5 | 液体酚醛树脂为主 |
| 异丙醇或工业乙醇 | 适量 | 载体溶剂 |

## 关键理化参数

### 耐火骨料参数

锆英粉典型理化指标 [[S5,S20,S15]]：

| 性能指标 | 数值 |
|----------|------|
| 密度 | 4.6～4.7 g/cm³ |
| 莫氏硬度 | 7～8级 |
| 耐火度 | ≥1700℃ |
| 熔点（纯） | 2430℃（1540℃开始分解为ZrO₂和SiO₂） |
| 理论成分 | ZrO₂ 67.2%，SiO₂ 32.8% |
| 行业合格标准 | ZrO₂质量分数＞65% |
| 总放射性比活度 | ≤7×10⁴ Bq/kg |
| 热膨胀系数 | 仅为石英的1/6～1/3 |
| 热导率 | 石英的2倍以上 |
| 高温化学特性 | 弱酸性/中性 |
| 粒度要求 | 95%以上过270目筛，部分产品可达320目 |

### 涂料性能参数

醇基锆英粉涂料密度行业通用范围为 **1.7～1.9 g/cm³**，远高于醇基石墨涂料的1.2～1.4 g/cm³，该参数是现场配制与稀释作业的核心质控指标 [[S35]]。

优化配方实测典型性能 [[S45]]：

| 性能指标 | 典型值 |
|----------|--------|
| 24h悬浮率 | ≥95% |
| 发气量 | ≈12.6 mL/g |
| 刷涂等级 | Ⅰ级 |
| 渗透深度 | 0.8～1.5 mm |

在生产验证中应用于30吨机床床身铸件时，铸件表面粗糙度可控制在Ra 12.5 μm，粘砂面积占比低于2%，无明显大于2 mm的气孔缺陷 [[S45]]。

## 标准与牌号

JB/T 9226-2008《砂型铸造用涂料》规定了醇基锆英粉涂料的牌号编码规则，采用三段式结构 [[S37,S38,S25,S27,S23,S14]]：

- **首段（载体）**：Y——有机溶剂（醇基）
- **中段（物理状态）**：G——膏状；F——粉状
- **末段（耐火粉料）**：GY——锆英粉

典型标准牌号：
- **YF-GY**：有机溶剂基粉状锆英粉涂料
- **YG-GY**：有机溶剂基膏状锆英粉涂料

涂料按稀释至使用状态后的悬浮性和涂层耐磨性分为两级，性能检测按JB/T 5107《砂型铸造用涂料试验方法》执行，测试项目覆盖密度、条件粘度、悬浮性、发气量、涂层耐磨性等 [[S38,S25]]。

目前无针对醇基锆英粉涂料的独立ISO/ASTM专项标准，相关涂层通用检测方法已纳入现有ISO/ASTM测试标准体系 [[S22,S28]]。

## 工艺操作

### 涂敷方式

常用涂敷方式包含刷涂、喷涂、流涂三种 [[S39,S21,S40,S4,S41,S33]]：

| 涂敷方式 | 适用场景 | 特点 |
|----------|----------|------|
| 刷涂 | 复杂内腔、小批量生产 | 砂型渗透深度大，操作灵活 |
| 喷涂 | 大面积型面、高效批量生产 | 效率高，需压缩空气及喷枪 |
| 流涂 | 自动化树脂砂生产线 | 涂料可自动回收利用，涂层无刷痕 |

三种方式均适配自硬树脂砂、水玻璃砂等不同型砂体系，涂覆完成后可直接点燃自干，无需额外进窑烘干 [[S40,S4]]。

### 涂覆规范

行业实测主流涂覆规范：刷涂一般作业1次，喷涂一般作业2次，涂层厚度实测范围通常为 **0.3～1.5 mm** [[S21,S43]]。线索中"厚度3～4 mm"的表述未得到专业文献验证，不符合生产实际 [[S43]]。

树脂砂砂型脱模后应至少放置3小时（24小时为佳）方可涂覆涂料，以防砂型开裂 [[S17]]。必要时可涂刷两遍不同密度的涂料：第一遍使涂料渗入砂型足够深度，第二遍增加涂层厚度 [[S17]]。

### 涂料密度与配比控制

涂料密度直接影响粘度、悬浮性、涂层厚度及涂刷性等使用性能 [[S35]]。200目锆英粉与醇基载体适配时，涂料密度随锆英粉添加量增加而升高，可参照密度-添加量关系曲线进行配比校正，辅助工程人员快速完成配方标定 [[S11]]。

## 应用范围

### 适配合金

醇基锆英粉涂料适用于以下合金类型 [[S9,S32,S5,S18]]：

| 合金类型 | 适用性 |
|----------|--------|
| 普通碳钢 | ✅ 常用 |
| 大部分合金钢 | ✅ 常用 |
| 高熔点有色合金 | ✅ 适用 |
| 大型铸铁件 | ✅ 常用 |
| 高锰钢等碱性钢种 | ❌ 不适用（锆英粉属弱酸性） |

高压蒸汽室类厚大高要求铸钢件属于该涂料的典型高附加值应用场景 [[S9,S32]]。

### 商用牌号适配差异

不同配方的商用醇基锆英粉涂料适配场景存在差异 [[S16,S18]]：
- **高纯度单一锆英粉配方**：适配大型碳钢、合金钢厚壁铸件的高耐火要求防粘砂场景；
- **锆英粉与莫来石复配牌号**：适配普通中小型铸钢件降本场景；
- **专用牌号**：对应刷涂、浸涂、流涂不同工艺需求，适配不同生产批量与砂型复杂程度。

## 防粘砂机理与缺陷防护

### 防粘砂作用机制

醇基锆英粉涂料的防粘砂作用通过三个层面协同实现 [[S36,S29,S19,S31,S8]]：

**① 物理隔离作用**：致密涂层完全分隔砂型与熔融金属的直接接触，封闭砂粒表面孔隙，阻止金属液机械渗入砂型；

**② 界面润湿性调控**：锆英粉高温化学惰性强，不与金属及其氧化物发生反应，大幅降低金属液在涂层表面的润湿性，切断化学粘砂的反应基础；

**③ 高温抗冲刷性能**：锆英粉高温下形成低熔点相填充的致密烧结壳，抗钢液长期高速冲刷，避免涂层剥落破损。

### 典型缺陷与预防

使用醇基锆英粉涂料时仍可能出现以下缺陷 [[S13,S26,S33]]：

| 缺陷类型 | 成因 | 预防措施 |
|----------|------|----------|
| 涂层开裂剥落→机械粘砂 | 骨料粒级级配不良，烧结收缩过大 | 优化骨料粒级级配，降低烧结收缩率 |
| 界面反应→化学粘砂 | 涂料与碱性钢种（高锰钢）不匹配 | 严格控制适配合金范围 |
| 铸件气孔 | 残留溶剂未完全点燃挥发 | 点燃后预留足够溶剂挥发时间再合箱浇注 |

### 与同类涂料对比

| 涂料类型 | 干燥方式 | 优点 | 缺点 | 典型适应场景 |
|----------|----------|------|------|-------------|
| 醇基锆英粉涂料 | 点燃自干 | 无需烘干、作业效率高、耐火度高、防粘砂优异 | 成本偏高、VOC排放 | 大型碳钢/合金钢铸件 |
| 醇基刚玉涂料 | 点燃自干 | 中性耐火、抗酸碱侵蚀优于锆英粉、适配高锰钢 | 热导率低于锆英粉涂层 | 高锰钢铸件 |
| 水基锆英粉涂料 | 需进窑烘干 | VOC排放低、成本较低 | 生产周期长、不适合快干流程 | 非紧急生产流程 |

## 限制与安全

### 储存与运输要求

依据铸造行业标准 [[S30]]：
- 存放温度**不得超过45℃**，包装全程密封，严禁靠近明火与高温热源；
- 膏状产品质保期 **6个月**，粉状产品质保期 **12个月**；
- 运输过程需按照易燃危险化学品相关规范执行。

### 施工安全

施工过程安全要求 [[S10,S17]]：
- 作业区域必须设置强制排风设施，确保空气中有机溶剂浓度低于安全阈值；
- 作业点周边10米范围内严禁任何明火；
- 作业人员需佩戴有机溶剂防护过滤面罩，穿戴防静电工作服，避免静电火花引发爆燃风险。

### 环保问题（VOC排放）

醇基锆英粉涂料以乙醇、异丙醇为载体，在施工作业及点火干燥过程中直接向大气排放大量挥发性有机化合物（VOC）。国内现行铸造行业VOC管控规定将该类高溶剂含量涂料纳入重点监管范畴，生产企业必须配套完整有机废气收集治理装置才能合规排放。相比水基锆英粉涂料，其VOC排放负荷显著更高，环保合规成本也更高，属于行业政策逐步限制推广的溶剂型涂料品类 [[S33,S6,S42]]。
