术语“漏模”在现行压铸国家标准、行业标准及权威教材中均未作为正式术语收录[[S20,S17,S27,S35,S6]]，其原始归属领域为砂型铸造，指造型起模环节的专用工艺与装置[[S25]]。在压铸工程实践中，“漏模”属于典型的跨领域非标准俗称，根据使用语境存在两种完全不同的指代：其一，被普遍用作压铸件缺陷“飞边”的工厂口语化表述[[S15,S31,S1]]；其二，被部分现场技术人员用来指代符合GB/T 8847-2003定义的压铸模成型零件——“活动镶件”[[S22,S3,S2,S30,S26]]。此外，极少数场合下该表述还可能被用于描述模具冷却水路的泄漏故障[[S11]]。由于该术语缺乏统一的官方定义，其在技术沟通中易产生歧义，建议在正式技术文件中优先使用“飞边”或“活动镶件”等标准术语。

## 术语来源与概念澄清
“漏模”一词源于砂型铸造，原指为便于起模而设计的局部可分离式模具结构。在砂型铸造中，当模样（模型）上存在阻碍起模的局部结构时，可将该部分做成活动件（活块），起模时活块先退入模板内（即局部漏模），或留在砂型中随后取出[[S3,S25]]。“漏模”在此语境下描述的是起模工艺而非缺陷，其对应的标准术语为“活块”或“局部漏模机构”。该概念与压铸领域的用法存在根本性区别。

## 压铸领域含义一：“漏模”指代缺陷（飞边）
### 定义与现象
压铸生产现场普遍将金属液从模具分型面、滑块间隙、镶块缝隙泄漏溢出，在铸件边缘部位形成的薄片状多余金属层俗称为“漏模”[[S15,S31]]。该缺陷的行业正式标准术语为**飞边**，英文对应 `mold flashing` / `flash` / `burr`，部分工厂亦称为“压射跑水”或“模具溢料缺陷”[[S15,S31,S1]]。飞边不仅损害铸件外观与尺寸精度，还会大幅增加后续人工去毛刺或机加工的工序成本，残留于模具间隙的飞边在合模时可能压伤型面，导致后续铸件出现缺肉缺陷，甚至卡住滑动抽芯机构造成运动失效[[S15,S13,S5]]。

### 形成机理
| 机理类别 | 具体机制 | 关键影响参数 |
|---|---|---|
| 高压渗透 | 金属液在压射阶段的高压作用下渗入模具微间隙 | 常规压铸压力：20–50 MPa；高压压铸：60–150 MPa；渗入所需间隙：≥0.02 mm |
| 动态锁模力波动 | 压铸机液压系统响应延迟导致瞬时锁模力不足，模具微开 | 形成飞边的最小开模量：≥0.05 mm |
| 热力耦合 | 分型面局部温度不均导致钢材膨胀差异，形成局部溢料间隙 | 模温达250 ℃时钢材线膨胀量约0.12 mm/m；局部温差>30 ℃即可能形成间隙 |
| 合金流动性增强 | 高温下高硅铝合金黏度极低，渗透能力显著增强 | ADC12在680 ℃时黏度低至0.15 Pa·s |

### 常见直接诱因
- 锁模力不足[[S14,S31,S13]]
- 压射速度过高，形成过大的压力冲击峰[[S14,S31,S13]]
- 分型面上杂物未清理干净[[S14,S31]]
- 模具强度不足，发生变形[[S14,S31,S13]]
- 镶块、滑块磨损后与分型面不平齐[[S14,S31]]
- 模具制造精度不足，活动配合间隙过大[[S14,S13]]

### 控制与解决措施
**模具端管控要求：**
- 分型面平面度：普通压铸≤0.05 mm/m，高压压铸≤0.02 mm/m[[S15,S28]]
- 可制备DLC类金刚石涂层，涂层硬度≥HV 2000，摩擦系数≤0.1，以减少间隙磨损[[S15,S28]]
- 采用分段式排气槽设计：前段深度0.05–0.08 mm（阻流），后段深度0.1–0.15 mm（排气）[[S15,S28]]

**工艺端管控要求：**
- 采用伺服液压系统实现锁模力动态补偿，锁模力波动控制在±3%以内[[S15]]
- 锁模力计算公式：(产品投影面积 + 进浇料道口面积) × 0.8 × 1.2 安全系数[[S15]]
- 多段压射工艺：低速填充阶段速度控制在0.1–0.3 m/s，避免卷气冲击分型面；高速增压阶段延迟至型腔填充完成95%后启动[[S15]]

**故障排查指导：**

| 诱因类别 | 具体原因 | 排除措施 |
|---|---|---|
| 模具镶块位移 | 模具镶块在压铸过程中发生松动或移位 | 调整并紧固镶块 |
| 导向件磨损 | 导柱、导套等导向元件磨损导致合模精度下降 | 更换导柱、导套 |
| 镶块制造误差 | 镶块加工尺寸偏差造成配合间隙过大 | 修整并消除误差 |

## 压铸领域含义二：“漏模”指代活动镶件（活动镶块）
### 概述与标准术语对应
工程实践中，“漏模”亦被部分技术人员用来指代压铸模中符合GB/T 8847-2003标准定义的**活动镶件**（`movable insert`），即“根据工艺与结构要求，随铸件一起出模后，方能从铸件中取出的块状成型零件”[[S22,S26]]。该俗称源于砂型铸造领域“局部漏模机构”概念的移植，指代一种模外取出的独立成型元件[[S2,S30]]。在英文压铸技术文献中，该概念对应 `loose core` 或 `knockout`，指用于成型内部倒扣结构的可拆卸型芯[[S30,S26]]。

### 结构与材料
| 特征项 | 要求或参数 |
|---|---|
| 配合结构 | 常采用燕尾槽与动模型芯实现导向配合[[S7,S34]] |
| 端面高度 | 镶块成型端面需低于动模型芯对应表面0.1 mm[[S7,S34]] |
| 配合面粗糙度 | ▽6~▽7[[S29]] |
| 配合间隙 | 0.05–0.15 mm[[S29]] |
| 材料 | 主流采用H13等热作模具钢[[S22,S24,S21]] |
| 加工前硬度 | 调质至26–32 HRC[[S22]] |
| 完工硬度 | 淬硬至44–48 HRC[[S22]] |
| 表面处理 | 工作面可按需做氮化处理提升耐磨性能[[S22,S21]] |
| 内部结构 | 尺寸较大的漏模可设计为空心结构，便于内部通入冷却介质[[S29]] |

### 功能与应用场景
漏模（活动镶块）的核心功能在于成型常规机动/液压抽芯机构无法实现的复杂内倒扣、异形内槽结构，可大幅降低特殊复杂结构铸件的模具整体设计难度与制造成本，替代部分复杂抽芯结构以满足小批量特殊铸件的成型需求[[S23,S8,S30]]。

典型使用场景包括[[S23,S7]]：
- 轴向叶片存在部分重叠的螺旋叶片类压铸件成型（如风机叶轮）
- 无法布置侧向抽芯的深腔异形内槽铸件成型
- 批量较小的特殊异形内孔压铸件成型

配套模具可设置中停工位，用于人工取出漏模的操作[[S23]]。

### 与常规抽芯机构的核心区别
| 对比维度 | 常规滑块/抽芯机构 | 漏模（活动镶块） |
|---|---|---|
| 集成方式 | 为模内集成结构，由模具机构驱动 | 独立可拆装离散零件 |
| 动作方式 | 开模动力或液压缸驱动自动完成抽芯 | 每模次人工装入模具定位，压铸后随铸件一同顶出，再人工取出 |
| 生产效率 | 高，适配大批量连续自动化生产 | 低，需人工干预，无法适配全自动化生产线 |
| 适配场景 | 大批量标准化铸件 | 小批量、特殊复杂倒扣结构铸件 |

### 设计与维护注意事项
- 设计阶段需设置可靠的定位结构，保证每一模次安装位置精度，避免压铸过程中发生位移引发飞边[[S4]]
- 生产操作时漏模必须提前预热至合适温度再装入模具，防止冷热冲击导致开裂或铸件尺寸偏移[[S4]]
- 日常维护需定期检查漏模定位接触面是否存在磕碰损伤或毛刺，及时修磨以避免合模错位引发压铸件尺寸超差[[S4]]

## 术语辨析提示
鉴于“漏模”在压铸领域缺乏标准定义，且同时可能指代缺陷（飞边）或模具零件（活动镶件）两类截然不同的概念，建议在技术文件、质量报告及跨部门沟通中避免使用该口语化表述，优先采用国家标准规定的正式术语“飞边”或“活动镶件”，以消除歧义、确保技术交流的准确性[[S15,S22,S20]]。在工程现场听到“漏模”一词时，应结合上下文判断其所指，并可向说话人确认其确切含义。
