## 概述

孕育铸铁（Inoculated Cast Iron）是指向熔融铸铁中加入一种或多种孕育合金后所得到的铸铁（无论处于液态还是固态），其核心目的在于通过形核剂作用促进石墨等初生相析出，避免初生碳化物生成[[S20]]。在国内铸造领域，孕育铸铁通常特指将低碳低硅原铁液经炉前孕育处理获得的高强度灰铸铁，其显微组织为珠光体基体上均匀分布细小的片状石墨，对应HT250及以上的高牌号灰铸铁[[S27]]。孕育铸铁的力学性能、断面均匀性远优于未经处理的普通灰铸铁，是灰铸铁品类中的高端主力产品。

## 在铸铁分类中的位置

在国内主流铸造合金分类体系中，孕育铸铁被列为铸铁大类下的一个独立子类，与蠕墨铸铁、合金铸铁、球墨铸铁、麻口铸铁、白口铸铁、可锻铸铁、灰口铸铁等并列，属于高牌号高强度灰铸铁的典型代表[[S24]]。从1912年经典铸铁冶金分类示意图可知，铸铁大类源生铁，经不同工艺衍生出各类铸铁，孕育铸铁即处于该演化路径中[[S47]]。

在工业实践中，孕育处理的适用范围远不止灰铸铁，球墨铸铁、蠕墨铸铁、可锻铸铁甚至特种铸铁的生产过程均必须进行孕育处理[[S1]]。但狭义的“孕育铸铁”通常指通过孕育处理获得的高强度灰铸铁，它与普通灰铸铁的核心差异在于：
- 普通灰铸铁（多为HT225及以下）未经专门孕育处理，石墨片粗大，断面壁厚敏感性高，壁厚增加时强度急剧下降；
- 孕育铸铁（HT250及以上）经孕育后共晶团数量大幅增加，组织均匀，壁厚增加5倍时抗拉强度仅下降约10%[[S1]]。

2023年国内铸件产量统计显示，灰铸铁占比40.3%，球墨铸铁占比29.0%，孕育铸铁作为高牌号灰铸铁，广泛应用于机床床身、发动机缸体等对强度和组织均匀性要求较高的场景[[S58]]。

## 孕育处理的目的

孕育处理的核心目的可归纳为四个维度[[S1,S39]]：
1. **促进石墨化**：避免铸件薄壁部位出现白口组织，降低整体白口倾向；
2. **细化石墨与增加共晶团**：生成大量均匀分布的结晶核心，使石墨形态细化，大幅增加共晶团数量；
3. **强化基体**：细化基体珠光体组织，提升铸铁的强度、硬度等综合力学性能；
4. **降低断面敏感性**：减弱铸铁对冷却速度的敏感性，减小不同壁厚部位的性能差异，提升全断面组织均匀性。

## 常用孕育剂

孕育剂是为铁液提供形核核心的添加物。目前工业上使用最广泛的孕育剂以硅铁为基，并根据需要添加钙、锶、钡、稀土等元素以改善抗衰退能力和特定孕育效果。

### 硅铁类孕育剂

普通75%硅铁（FeSi75）是应用最多的孕育剂，其标准成分及特点如下[[S59]]：

| 牌号 | Si（%） | Al（%） | Ca（%） | 其他 |
|------|---------|---------|---------|------|
| 75SiFeAl0.5 | 74~80 | ≤0.5 | ≤1.0 | Mn、Cr、P、S、C少量，余量为Fe |
| 75SiFeAl1.0 | 74~80 | ≤1.0 | ≤1.0 | 同上 |
| 75SiFeAl1.5 | 74~80 | ≤1.5 | ≤1.0 | 同上 |
| FeSi45 | 40~47 | ≤2.0 | ≤1.0 | 同上 |

75%硅铁的常规推荐加入量：厚壁铸件为铁液质量的0.2%~0.4%，薄壁铸件为0.3%~0.5%；瞬时孕育时可低至0.1%~0.3%[[S9,S3]]。推荐使用粒度1~3mm，以保证快速溶解，避免粉状硅铁的氧化烧损[[S9,S46]]。其作用机理是FeSi溶解后形成富硅微区，造成局部成分过冷，提高碳活度促进石墨析出；同时所含微量Ca、Al等元素可生成CaS等硫化物作为石墨形核核心，增加共晶团数量[[S9,S31]]。

### 硅钙类孕育剂

硅钙合金执行YB/T 5051-2007标准，主流牌号成分如下[[S57]]：

| 牌号 | Ca（%） | Si（%） | Al（%） |
|------|---------|---------|---------|
| Ca31Si60 | ≥31 | 50~65 | ≤2.4 |
| Ca28Si60 | ≥28 | 50~65 | ≤2.4 |
| Ca16Si55 | ≥16 | 50~60 | ≤2.5 |

硅钙孕育剂的常规加入量一般为0.2%~0.6%，除石墨化作用外兼具强脱氧、脱硫效果，防白口能力优于普通硅铁，特别适合薄壁灰铸铁件[[S57,S37]]。标准要求硅钙合金以块状供货，最大块重不超过10kg，小于15mm×15mm的碎块质量占比不超过10%[[S54]]。

### 含锶硅铁类孕育剂

含锶硅铁孕育剂典型牌号的成分要求[[S49]]：

| 牌号 | Si（%） | Sr（%） | Al（%） | Ca（%） |
|------|---------|---------|---------|---------|
| Si75Sr1 | 73~78 | 0.6~1.0 | <0.5 | ≤0.1 |
| Si50Sr1 | 46~50 | 0.6~1.0 | <0.5 | ≤0.1 |
| Si75Sr2 | 73~78 | 1.0~2.0 | <0.5 | ≤0.5 |

其常规加入量仅需0.1%~0.2%，即可达到普通硅铁两倍的石墨化效果，抗衰退能力强，并可减少铸件缩松渗漏倾向，尤其适合高牌号薄壁灰铸铁件[[S49]]。在低硫铸铁中，浇包内加入含锶硅铁往往无显著孕育效果，但采用随流瞬时加入或浇注系统内加入则能获得长效孕育效果[[S37,S18]]。

### 硅钡类孕育剂

硅钡合金执行YB/T 5358-2008标准，7个牌号的Ba含量覆盖≥2%至≥30%，对应Si含量从≥35%至≥65%，Al≤3.0%，Mn≤0.4%，C≤0.2%~0.3%，P、S均≤0.04%，同一批钡含量波动不得超过2%[[S52]]。标准规定硅钡合金的常规交付粒度为10~200mm，小于10mm的部分不超过总量的5%[[S54]]。

### 稀土硅铁类孕育剂

稀土镁硅铁合金执行GB/T 4138-2004标准，其RE含量范围为0.5%~20%，常规交付粒度为5~30mm，小于5mm和大于30mm的部分各不超过总质量的5%[[S54,S5]]。稀土元素（如Ce）可降低孕育衰退速率，提升形核核心的留存时间，其在铁液中的最优有效添加范围为0.0045%~0.008%[[S32,S37]]。

## 孕育工艺方法

孕育处理工艺根据加入位置和时间可分为包内孕育和瞬时后孕育两大类。各方法的操作要点及特点对比如下[[S34]]：

| 孕育方法 | 细分方法 | 操作要点 | 评价 |
|----------|----------|----------|------|
| 包内孕育 | 出铁槽孕育 | 出铁时将孕育剂均匀撒在出铁槽铁流上，于出铁时间的3/4内撒完 | 方法简便，但易衰退，用量多，效果不稳定；要求5~6min内浇完，最长不超过15min |
| | 包底冲入法 | 将孕育剂预置于包底，冲入铁液 | 同上 |
| | 倒包孕育 | 将孕育剂放在分包底部，通过倒包冲入铁液 | 同上 |
| 瞬时后孕育 | 随流孕育（流股孕育） | 在浇注过程中向铁液流中连续定量加入粒状或丝状孕育剂，可借助重力、气力或喂丝机 | 无衰退，加入量可降至0.10%~0.15%，粒度0.2~0.7mm，适合自动化流水线；灰铸铁最大浇注流量4.5kg/s |
| | 浇口杯孕育 | 将粒状或块状孕育剂预置浇口杯底部，铁液冲入熔解后进入型腔 | 基本无衰退，加入量0.1%~0.2%，需控制拔塞时机，浇口杯尺寸不能过小 |
| | 型内孕育（型腔孕育） | 将孕育块或粒状孕育剂置于直浇道底部或横浇道反应室，铁液充型时瞬时完成孕育 | 无衰退，加入量仅0.05%~0.1%，粒度1~4mm，孕育均匀，明显降低壁厚敏感性 |
| | 孕育丝孕育 | 用喂丝机将包芯孕育丝送入浇口盆或直浇道铁流中 | 无衰退，加入量可低至0.08%，但费用较高，适用于自动造型线 |

出铁槽孕育作为最简便的包内孕育方法，操作门槛极低，但孕育效果稳定性差，衰退快，要求处理后5~6min内完成浇注[[S34,S6]]。随流孕育的粒状孕育剂一般通过重力给料或气力喷射加入铁液流，其粒度要求为0.2~0.7mm，加入量仅0.1%~0.15%，完全不存在孕育衰退问题[[S4,S36,S34]]。型内孕育则利用预制成型的孕育块或合适粒度的粒状孕育剂，最优粒度为1~4mm，可有效抑制薄壁铸件白口缺陷[[S4,S35,S34,S11]]。浇口杯孕育适用于大型球墨铸铁件，需严格控制拔塞时机，采用粒状孕育剂时要防止浮起导致夹渣[[S4,S34]]。

图1展示了浇口杯孕育法的完整流程，包含浇包、塞杆、浇口杯、孕育剂等操作部件，可直观反映从孕育剂加入到铁液冲入型腔的全过程[[S10]]。

## 微观组织与力学性能

经孕育处理的亚共晶灰铸铁通常可获得均匀分布、无方向性的A型石墨，以4、5、6级为佳，这是砂型铸造条件下性能最优的灰铸铁石墨形态[[S22]]。与普通灰铸铁中粗大片状石墨相比，孕育铸铁的石墨呈细小片状，均匀分布于珠光体基体上，显著降低了石墨对基体的割裂作用[[S45,S19]]。图2展示灰铸铁中常见片状石墨的金相分类形态，包括A、B、C、D、E等类型，可辅助辨认孕育处理后的合格石墨形态[[S53]]。

![灰铸铁中常见片状石墨的金相分类形态图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/ddb19a80-7021-4d73-9d24-390d3dfbebd2/resource)

高强度孕育铸铁（HT250、HT300、HT350）的抗拉强度范围为250~350MPa，较同成分未经孕育处理的灰铸铁提升幅度接近100%，硬度和韧性同步提高，但伸长率极低，仍属于脆性材料[[S50,S14]]。孕育处理不仅提高强度，还大幅降低铸件对壁厚的敏感性，使不同壁厚部位的组织和性能趋于均匀[[S1,S22]]。

## 孕育衰退

孕育衰退是指铁液经孕育处理后，随液态停留时间延长，孕育效果逐步降低直至完全消失的现象。孕育效果在孕育剂加入瞬间达到峰值，随后立即衰退，不存在衰退酝酿期[[S38,S55,S51]]。

衰退的根本机制在于异质形核核心的减少与失效，具体包含三类理论[[S38]]：
- 细小的1~4μm形核夹杂物粒子随停留时间延长不断碰撞、扩散、合并而粗化，丧失形核能力；
- 孕育处理形成的高硅微区、碳过饱和显微集团等成分/温度起伏随时间消散，形核质点数量大幅减少；
- 铁液发生再氧化，脱氧产生的SiO₂等核心被氧化污染，失去石墨形核作用。

对于灰铸铁，常规FeSi75孕育剂的作用在加入后约30s达到峰值，1min时已衰退近半，3min后几乎完全消失[[S55]]。球墨铸铁的孕育有效作用时间约为12~15min[[S60,S11]]。加剧衰退的因素包括：铁液保温时间过长、处理温度过高、搅拌剧烈、孕育剂粒度过细、白口炉料占比高、原铁液硫氧含量偏高等[[S40]]。

工业上防衰退措施主要有[[S40,S55,S38]]：
1. 选用含Sr、Ba、Zr、Ce等元素的长效抗衰退孕育剂，其生成的抗铁液氧化碳化物稳定性远高于普通Al₄C₃、CaC₂；
2. 采用随流孕育、浇口杯孕育、型内孕育等瞬时迟后孕育工艺，最大限度缩短孕育至凝固的时间间隔；
3. 对铁液实施二次或多次孕育，补充形核质点；
4. 适当提高铸件冷却速度，提升形核有效性。

## 典型应用及标准牌号

GB/T 9439、ISO 185、ASTM A48等标准中，要求抗拉强度≥250MPa的HT250及以上牌号灰铸铁，均须经过孕育处理才能满足力学性能要求，涵盖HT250、HT300、HT350等典型孕育铸铁牌号[[S25,S26,S14]]。

孕育铸铁的典型工业应用包括机床床身、发动机气缸体、气缸盖、大型设备基座等对强度、组织均匀性和减震性能要求较高的铸件[[S33,S22]]。

## 与其他铁基材料及工艺的关系

**与球墨铸铁**：球墨铸铁除必须进行孕育处理外，还需额外添加球化剂，使石墨呈球状，基体强度利用率可达70%~90%，抗拉强度远高于孕育铸铁，且具有一定的塑性和伸长率[[S13,S28]]。孕育铸铁的石墨仍为片状，强度较低，但减震性、铸造流动性优于球墨铸铁。

**与蠕墨铸铁**：蠕墨铸铁需在孕育处理基础上额外添加蠕化剂，获得蠕虫状石墨，其力学性能高于孕育铸铁，导热性和抗热疲劳性能也更优，部分工况下可替代孕育铸铁以实现减重或提高成品率[[S13,S41]]。

**球化/蠕化处理与孕育处理的配合**：球化或蠕化处理后必须配套进行后续孕育处理，通常采用FeSi75孕育剂，冲入法加入量为铁液总质量的0.1%~0.3%，目的是消除处理带来的白口倾向，增加石墨核心，细化晶粒，并延缓球化/蠕化衰退[[S12,S46,S16,S8]]。

**与压铸工艺的适用性**：当前主流压铸工艺以锌、铝、镁、铜等非铁合金压铸件为主，黑色金属铸铁压铸工艺尚未实现大规模工业应用，标准压铸生产场景中很少对铸铁材质执行前置孕育处理操作[[S2]]。
