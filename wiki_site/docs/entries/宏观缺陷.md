## 定义与范畴

在压铸领域，**宏观缺陷**（Macro Defect）是指铸件中无需借助金相显微镜即可识别的不连续性缺陷。其判别标准为：通过裸眼或10倍以下低倍放大镜可直接观测，缺陷的辨识尺度阈值为≥100 μm。与之对立的**微观缺陷**（如显微缩松、晶界微裂纹等）尺寸低于该阈值，必须借助50倍以上金相显微镜才能观测，二者判别边界清晰可量化 [[S15,S27]]。

宏观缺陷的检测覆盖压铸件的表面及内部区域：表面宏观缺陷可通过目视直接发现，内部宏观缺陷则需借助X射线、超声等无损检测手段或剖切等破坏性方法暴露 [[S16]]。在中文铸造标准语境（如GB/T 29092-2012《镁及镁合金压铸缺陷术语》）中，所列的冷隔、气孔、缩孔、凹陷、裂纹等缺陷类型即属于典型的宏观缺陷范畴 [[S32]]。

## 分类体系

压铸宏观缺陷可依据缺陷在铸件上的空间位置与形貌特征划分为三大类：表面宏观缺陷、内部宏观缺陷及裂纹类宏观缺陷 [[S25,S58,S20]]。

| 类别 | 缺陷名称 | 英文名/别名 |
|:---|:---|:---|
| **表面宏观缺陷** | 冷隔/冷纹 | Cold shut / Cold lap |
| | 缩水/凹陷 | Shrinkage sink mark / Shrinkage depression |
| | 流痕 | Flow mark / Flow line |
| | 拉伤 | Scratch / Die sticking mark / Caught casting |
| | 起泡 | Blister / Surface blister |
| **内部宏观缺陷** | 宏观缩孔 | Shrinkage cavity |

## 各缺陷特征与形成机理

### 表面宏观缺陷

**冷隔（冷纹）** 是充型过程中两股不同步的低温金属液流相遇后，边界发生预凝固而未能充分熔合形成的线性断续缝隙，边缘呈圆角状 [[S32,S29]]。其出现位置多在远端薄壁充型区域或多股流交汇处，当锌合金模温低于150℃、铝合金模温低于180℃时，冷隔发生概率显著上升 [[S29,S31,S64]]。相关工艺影响因素包括浇注温度不足、模具温度过低、压射填充速度偏低及排气不畅。

**缩水（凹陷）** 是铸件厚壁区域或断面交接部位，表层金属优先凝固后内部液相体积收缩无后续金属补缩，导致对应表面向内塌陷的平滑凹痕，凹陷下方常伴有隐蔽缩孔 [[S25,S13,S47]]。相关因素包括铸件壁厚差过大、增压压力不足、留模时间过短及冷却不均匀。

**流痕** 是与金属液流动方向一致的微凸/微凹连续条纹状缺陷，无纵深扩展趋势，由两股不同步填充的金属液面先后在型壁凝固留下的流动痕迹所致 [[S25,S64]]。相关因素包括填充速度匹配不合理、模具温度偏低及涂料涂覆过量。

**拉伤** 是铸件脱模过程中，铸件表层与型腔壁发生粘结啮合，沿开模方向被刮擦形成的连续擦伤痕迹，分为机械拉伤和粘模拉伤两类 [[S61]]。相关因素包括脱模斜度不足、模具表面粗糙度过高、浇注温度过高及填充速度过快。

### 内部宏观缺陷

**宏观缩孔** 是凝固过程中体积收缩得不到液态金属补缩，在铸件最后凝固的热节位置形成的形状不规则、内壁粗糙的集中大孔洞，属典型宏观可识别缺陷 [[S4]]。

**宏观缩松** 是金属液凝固收缩得不到补缩，在晶界、枝晶网络间形成的尺寸≥100 μm的分散细小微孔聚集区，裸眼在剖切面上可直接观测 [[S27,S11,S26]]。相关因素包括浇注温度过高、增压压力不足及持压留模时间偏短。

**卷气气孔** 是充型过程中型腔内残留气体未完全排出，被高速金属液包裹，凝固后保留在铸件内部形成的表面光滑圆形/近圆形孔洞 [[S32,S7,S40,S30]]。相关因素包括压射填充速度过高、模具排气通道设计不良及浇注温度匹配不合理。

### 裂纹类宏观缺陷

**热裂（热裂纹）** 是铸件在高温半固态状态下，收缩应力超过晶间结合力，沿晶界开裂形成的曲折长缝隙，裂纹断面严重氧化呈深灰色/黑色，无金属光泽 [[S57,S32]]。相关因素包括厚壁尖角部位设计不合理、合金浇注温度过高及留模时间不足。

**冷裂（冷裂纹）** 是铸件冷却到低温固态后，残余铸造应力超过金属本身抗拉强度，穿晶开裂形成的平直折线状缝隙，裂纹断面洁净具有金属光泽，无明显氧化痕迹 [[S57,S32]]。相关因素包括脱模冲击载荷过大、冷却不均匀及残余应力未充分释放。

## 对压铸件质量的影响

宏观缺陷直接削弱压铸件的使用性能与安全性，主要影响体现在以下方面：

- **力学性能劣化**：缩孔、缩松等孔洞类缺陷减少有效承载截面积，并引起应力集中，导致强度与韧性下降。脆性缺陷会使压铸件韧性显著恶化，该类缺陷件一般需直接判废 [[S59,S4]]。
- **气密性丧失**：连通型的气孔、缩孔、缩松、裂纹、冷隔、夹杂等缺陷破坏基体致密度，导致压铸件在密封工况下出现漏水、漏气，无法满足气密性要求 [[S59]]。
- **疲劳寿命降低**：宏观不连续性缺陷作为疲劳裂纹的萌生源，加速裂纹扩展，大幅降低制件疲劳寿命，对于交变载荷工况下的汽车结构件等关键部件尤需严格管控 [[S45]]。

## 检测与评估

压铸宏观缺陷的检测方法覆盖破坏性与无损检测两大类，依据缺陷所处位置（表面/内部）和验收精度要求选用 [[S16,S14]]。

| 检测方法 | 适用范围 | 检出能力与技术要点 |
|:---|:---|:---|
| 目视比对检测 | 表面宏观缺陷（花纹、流痕、缩凹、冷隔、裂纹、缺肉等） | 普通照明下操作，可搭配5倍放大镜；宜结合标准样件对比以降低人为偏差 [[S16,S60,S52]] |
| 荧光渗透检测 | 表面开口微小不连续性缺陷（细微裂纹、冷隔等） | 检出下限0.02~0.05 mm，紫外线下观察发光迹痕 [[S14,S50]] |
| X射线检测 | 内部宏观缺陷（气孔、缩孔、缩松、裂纹等） | 可获取缺陷位置、形状、尺寸、分布信息；工业场景常与AI算法结合 [[S16,S14]] |
| 超声检测 | 厚壁压铸件内部宏观缺陷 | 利用异质界面反射特性，可同步测壁厚、分析材料 [[S16,S14]] |
| 剖切检查 | 内部缺陷形貌直接观测（抽样验证） | 破坏性手段，暴露断口处气孔、缩孔、夹杂的实际形貌与分布 [[S22]] |

### 相关标准中的验收准则

- **ISO 9915**：规定了多缺陷共存场景下的射线检测验收规则——若同一5 cm×5 cm单元面积内同时出现至少2种缺陷且均达到最大验收极限时，判定不合格；同一单元内出现4种及以上不同类型缺陷直接判废。验收严格度分为A（高应力关键铸件）至D（普通铸件）4级 [[S63]]。
- **ISO 9916:1991**：铝合金/镁合金铸件渗透检测标准，将缺陷迹痕分为非线型、点线型、线型三类，采用105 mm×148 mm标准评定框选取最严重区域，分别按4级（非线型）和6级（线型）进行严重度判定 [[S10]]。
- **ISO 10049:1992**：铝合金铸件机加工表面针孔度目测评定法，规定了表面预处理（NaOH腐蚀生成黑膜、硝酸去膜）后目视分级判定的流程 [[S8]]。
- **GB/T 13819-1992**：铜合金铸件标准，规定了图样标识规则及加工表面浇冒口残留允许限值 [[S24,S53,S65]]。
- **NADCA产品规范表**：北美压铸协会发布的检查表与表面质量规范，明确不同功能等级压铸件的缺陷允许阈值 [[S9,S6]]。

## 双控成形对宏观缺陷的抑制

双控成形（全称液态铸锻双控成形）是由国内学者于2005年左右提出的铸锻复合工艺，属于挤压铸造（液态模锻）的发展分支，并非双冲头控制或普通半固态成形 [[S36,S42,S2]]。该工艺的核心定义为：在液态金属成形过程中，同时实现对铸件外形尺寸精度和内部组织性能的双重精准控制，结合普通压铸的复杂薄壁高精度充型优势与液态模锻的高压压实消除内部缺陷优势 [[S18,S44,S36]]。

双控成形的标准技术路径为：合模浇注→低速平稳充型（充型速率远低于常规高压压铸，保证型腔气体充分排出）→在金属液尚未完全凝固时施加足够锻造压力完成补锻→开模取件 [[S12,S56,S17]]。

**对宏观缺陷的抑制效果**：

- 充型阶段卷入的气体可在后续锻造工序中被排出或压实焊合，显著抑制气孔、缩松、缩孔类宏观缺陷，铸件致密度大幅提升 [[S34]]。
- AZ91D镁合金实验数据显示，与相同工艺条件下的普通高压压铸件对比，铸态下双控成形件抗拉强度提升84.39%，延伸率提升94.67%；经热处理后，抗拉强度可提升101.76%，延伸率提升121.62%，常规压铸件存在的大量宏观气孔、疏松缺陷基本消除 [[S36,S42,S21]]。

需注意，“双控成形”为国内该铸锻复合工艺的专属定名，国际主流压铸学术资料与行业手册中未收录对应的通用标准化术语（如“dual control forming die casting”等） [[S46,S5]]。
