## 概述

型内球化（in‑mold nodularization / in‑mold spheroidization）是将球化剂预置于铸型浇注系统内部的特制反应室中，铁液在充型流经反应室时与球化剂发生反应，在型腔内完成石墨球化的一类球墨铸铁球化处理工艺[[S36,S12]]。该术语已被中国铸造标准术语体系正式收录，其标准定义为：“在铸型内对铁液进行球化处理的工艺。该法通常在浇注系统中设有放置球化剂的反应室，铁液在反应室内接受球化处理后进入型腔。要求原铁液高温、低硫、质量稳定，并严格控制球化剂的成分和粒度。该法操作简便、球化剂回收率高、无污染、球化效果稳定。”[[S27,S37,S28]]

在球墨铸铁球化处理工艺体系中，型内球化是与冲入法、盖包法、G.F.转包法、喂线法并列的成熟工业应用工艺，属于镁系球化添加方法下的原位反应类工艺范畴[[S35,S23]]。美国铸造协会发布的“Ductile Iron Handbook”亦将“In‑the‑Mold Treatment”正式列为球墨铸铁镁处理方法下的独立分支工艺[[S4]]。

## 工艺原理与分类

型内球化与冲入法、喂线法等传统包内球化工艺的核心差异在于：球化反应不在浇注前的铁液处理包内完成，而是随铁液充型流动过程同步发生，从源头上大幅缩短球化反应结束到铸件凝固的间隔时间，可有效抑制球化衰退现象[[S21,S16]]。浇注过程中液态金属流经预埋球化剂的反应室，在型腔内部完成球化处理，球化反应几乎与外界空气完全隔绝，避免了普通包内球化的镁氧化烧损与烟尘溢出问题[[S36]]。

型内球化与型内孕育是两类完全独立的铸型内原位处理工艺：型内球化向反应室添加的是镁/稀土镁类球化剂，作用是促使铁液中石墨以球状形态析出；型内孕育向浇注系统添加的是75硅铁类孕育剂，作用是消除铁液的白口倾向、细化石墨晶粒。二者工艺目的、添加材料完全不同[[S6,S21]]。

### 反应室设计参数

反应室的设计是型内球化浇注系统的核心环节，其关键设计参数如下[[S7]]：

| 设计参数 | 计算公式或推荐值 | 说明 |
|----------|------------------|------|
| 反应室横截面积 \(F_{\text{室}}\) | \(F_{\text{室}} = v/f\) | \(v\) 为浇注速率；\(f\) 为球化剂熔解系数，推荐取值 0.05~0.07 kg/(s·cm²) |
| 球化剂堆积高度 \(h_{\text{堆}}\) | \(h_{\text{堆}} = V/F_{\text{室}}\) | \(V\) 为球化剂体积 |
| 反应室总高度 | 球化剂堆积高度 + 1.27 cm（预留反应空间）+ 2.54 cm（铁液进口高度） | 反应室水平截面在全高范围内保持一致 |
| 进口落差 | ≤ 2 cm | 避免液流过载冲击 |
| 出口截面比 | 出口截面比进口小约10% | 利用阻流作用促进球化剂充分溶解 |
| 进出口布置 | 进出口水平方向错位 | 使液流在反应室内形成环流，强化与球化剂的接触 |

### 浇注系统截面比

型内球化配套的浇注系统普遍采用半封闭式设计，兼顾平稳充型与阻渣能力。针对中小型球铁铸件，推荐截面比为 \(F_{\text{内}} : F_{\text{横}} : F_{\text{直}} = 0.8 : (1.2 \sim 1.5) : 1\) [[S9]]。

## 优点与局限

### 优点

- **球化元素吸收率高**：镁吸收率可达80%以上，远高于冲入法的30%~40%、盖包法的约60%和喂丝法的40%~50% [[S17]]。
- **球化衰退显著抑制**：球化反应至铁液凝固的间隔时间极短，在残留镁量低至0.02%、含硅量低于2%的条件下仍可获得合格球墨铸铁组织，有效消除铸态渗碳体[[S21,S16,S10]]。
- **石墨球细小、数量多**：铸件壁厚3~10 mm时，单位面积石墨球数可达400~500个/mm²，石墨球圆整度高[[S21]]。
- **力学性能优异**：相同材质下，型内球化处理的球墨铸铁抗拉强度可由普通球化法的520 MPa提升至720 MPa，断后伸长率由2%提升至4%，硬度由210 HBW提升至220 HBW[[S2]]。珠光体球铁曲轴经该工艺处理，最低抗拉强度可达900 MPa，伸长率保持5%[[S21,S23]]。
- **促进薄壁件铁素体化**：在壁厚3~50 mm阶梯试块测试中，型内球化铸件延伸率可达包内球化铸件的约2倍，更适于直接生产铸态高韧性延性球墨铸铁[[S16]]。
- **环保效果好**：球化反应在铸型内部完成，几乎无镁氧化烧损烟尘和光污染[[S36]]。
- **允许较低浇注温度**：型内球化所需铁液浇注温度比包内球化工艺低60~90°C，降低能耗[[S26]]。
- **可与过滤工艺联用**：配合泡沫陶瓷过滤网可有效拦截II、III类夹渣，获得更均匀的等轴晶组织[[S23]]。

### 局限

- **对原铁液要求高**：要求原铁液高温、低硫、成分稳定，冲天炉铁液因含硫量较高一般不适用此法[[S13,S25,S32]]。
- **浇注系统设计复杂**：需专门设置球化剂反应室和高效挡渣结构，设计调试难度大[[S32,S10]]。
- **工艺成品率可能较低**：部分批量生产案例中，某些铸件品类的工艺成品率仅为30%左右，远低于常规球化工艺[[S10,S43]]。
- **技术门槛高**：美、英、加等国已布局大量相关专利，国内企业量产需完成充足技术储备[[S35,S32]]。

## 常见缺陷与防控

### 球化不良与球化衰退

形成机制包括：原铁液硫含量过高消耗球化元素；反球化元素含量超标；残留镁和残留稀土不足；铸型表层硫元素向铸件表层扩散消耗表层球化元素；铁液过度氧化；球化后静置时间过长导致球化元素烧损等[[S8,S15,S14]]。

**防控措施**：① 严格控制原铁液含硫量，使用低硫纯净炉料，限制钛、铅等反球化元素含量；② 采用低硫造型材料，在铸型表面涂刷阻硫涂料；③ 保证球化剂成分与粒度符合工艺要求，通过反应室结构设计控制球化剂平稳释放；④ 选用粒度约2.63 mm的颗粒状Fe‑Si孕育剂，大幅延缓石墨球数随保持时间的衰退速度[[S8,S14,S31]]。

### 夹渣

球化处理过程中生成的MgO、MgS等氧化物与硫化物熔渣，若未完全截留在反应室和集渣结构中并随铁液进入型腔，即形成夹渣缺陷。分为浇注前未清除的一次夹渣和浇注凝固过程中二次氧化生成的细小二次夹渣，严重时导致铸件力学性能下降及耐压铸件渗漏[[S30]]。

**防控措施**：① 尽可能降低原铁液含硫量；② 在保证球化合格的前提下降低球化剂添加量；③ 设置高效挡渣集渣结构，搭配泡沫陶瓷过滤网阻挡II、III类渣进入型腔；④ 浇注温度不低于1350°C，必要时浇注前向铁液表面添加占铁液重量0.1%~0.3%的冰晶石进行除渣覆盖[[S30,S23]]。

### 局部白口与反白口

局部区域冷却速度过高致石墨形核不足，形成游离渗碳体；或凝固后期氧、氢等在铸件心部偏析抑制石墨析出，形成反白口碳化物组织，恶化切削加工性能与韧性[[S15]]。

**防控措施**：① 选用含Ca、Sr、Ce等元素的高效孕育剂，采用型内孕育复合型内球化工艺，增加石墨形核核心、降低过冷度；② 严格控制炉料中Cr、V、Mo、Te等强碳化物形成元素含量；③ 针对薄壁铸件，选用Ce‑Ca‑Al‑S‑O‑FeSi类孕育剂，可有效降低过冷现象、减少白口倾向[[S22,S25]]。

### 球化不均匀

球化剂在反应室内溶解速度不均，导致不同时刻流经反应室的铁液获得的球化剂量差异过大；或不同壁厚位置冷却速度差异过大，导致球化率和石墨球数分布不均[[S29,S19]]。

**防控措施**：① 合理设计反应室结构，控制球化剂匀速释放，保证全程铁液获得的球化剂量均匀稳定；② 调整铸型冷却条件，增大冷却强度（如提升冷却水流量），可显著提升铸件不同深度位置的球化率均匀性，扩大高球化率层的厚度[[S29,S19,S40]]。

### 型内球化与包内球化的球化效果对比

在球化剂添加量同为0.15%的条件下，型内球化工艺在3~50 mm全壁厚区间内可稳定保持85%以上的高球化率，而包内球化工艺的球化率随壁厚增加明显下降，薄壁区间仅约75%[[S29]]。相应地，型内球化工艺在6~50 mm全壁厚区间内石墨球数均显著高于同添加量的包内球化工艺，在壁厚小于10 mm的薄壁区石墨球数可达400个/mm²以上[[S19]]。这从原理上解释了型内球化工艺在力学性能和壁厚适应性上的明显优势。

## 应用场景

型内球化工艺适配电离批量流水线、机械化自动化生产的球墨铸铁铸造场景，当前在工业发达国家应用较多[[S23]]。典型适用铸件品类包括[[S10,S25]]：

- **汽车零部件**：球墨铸铁曲轴、十字叉轮毂、轿车制动器卡钳、铸态高强韧性板簧座等。
- **薄壁球墨铸铁件**（壁厚3~10 mm）：型内球化的高石墨球数特性可降低薄壁件高过冷下的白口倾向，提升薄壁件强韧性。
- **中小规格流体承压铸件**：球墨铸铁液压阀体、管件等，稳定的球化效果可降低渗漏缺陷概率。

## 金相组织特征

型内球化处理得到的球墨铸铁，其典型金相特征为石墨球细小、圆整度高。在铸件壁厚3~10 mm范围内，单位面积石墨球数量可达400~500个/mm²，可稳定获得铸态无自由渗碳体的组织[[S21]]。下图展示了球墨铸铁壳体球化正常区的典型金相形貌，呈均匀分布的球状石墨组织[[S39]]。

## 标准相关规定

国家标准GB/T 1348《球墨铸铁件》明确规定，采用型内球化工艺生产铸件时，配套的力学性能检测试块需与铸件共用同一浇注系统成型，或采用同条件的型内球化工艺单独制备，以确保试块与铸件的球化效果一致[[S11]]。

国家标准GB/T 5611《铸造术语》对型内球化给出了权威定义，明确指出该法操作简便、球化剂回收率高、无污染、球化效果稳定[[S27,S37]]。
