## 概述

压铸浇注系统（英文通用术语 *gating system*，也称 *runner system*）是将金属液从压室引入模具型腔的进料通道总称，其范围自压室末端开始，至内浇口为止[[S41,S53,S39]]。该系统对金属液在模内的流动方向与流态、型腔排气条件、模具整体热量分布、压力传递效率、填充时间长短以及金属液通过内浇口处的速度等均起关键的控制和调节作用，是决定充填形态与压铸件内部质量的核心因素之一，同时对生产效率、模具寿命及后续清理工序也有显著影响[[S41,S39]]。广义上，浇注系统通常与排溢系统（溢流槽、集渣包、排气槽）协同设计，二者共同构成控制充型过程的不可分割的整体[[S39,S54]]。

## 定义

根据国家标准 GB/T 8847-2003《压铸模术语》的明确定义，浇注系统是“熔融合金在压力作用下充填模具型腔的通道，包括直浇道、横浇道和内浇口”[[S34]]。该定义已作为国内压铸模具技术文件的标准术语使用，并在 GB/T 8844-2003《压铸模技术条件》的模具设计规范框架下被进一步约束[[S8]]。

工程实践中，浇注系统亦常被表述为从压室到内浇口的金属液输送通路，其主要功能包括导入金属液、控制充型流态及传递压射压力[[S10,S26,S54]]。

## 工艺作用

浇注系统在压铸成形全过程中的工艺作用覆盖以下核心维度[[S41,S39]]：

- **导引金属液流动方向与流态**：决定金属液进入型腔的部位、流向，形成顺序填充模式。
- **控制充填速度与填充时间**：通过内浇口截面积的设计，调控金属液通过内浇口的速度及整体型腔填充时间。
- **配合排溢系统辅助排气**：将充型前沿裹挟气体、氧化夹杂的金属液引导至溢流槽和排气槽。
- **调节模具热量分布**：合理的流道布局可调节型腔各部位的热输入，维持模具热平衡。
- **保障压力传递实现补缩**：充型完成后通过仍保持开放的内浇口传递压射压力，对铸件进行补缩压实。
- **避免压铸缺陷**：防止涡流裹气、氧化夹杂、冷隔、充不满等常见压铸件缺陷。

## 组成结构

### 标准组成部分

浇注系统核心由三个单元构成[[S10,S26,S54]]：

1. **直浇道（Sprue）**：传递压力的首要部位，用于将压室或喷嘴内的金属液平稳引入横浇道，其结构参数直接影响金属液流速、压力损失及充填条件[[S41,S37]]。
2. **横浇道（Runner）**：从直浇道末端到内浇口的连接通道，将金属液由直浇道导入内浇口，并要求其截面积从直浇道末端向内浇口方向逐步缩小或保持不变[[S10,S26,S3]]。
3. **内浇口（Gate/Ingate）**：连接横浇道与型腔的狭小通道，其作用是将横浇道输送的低速金属液加速，形成理想流态顺序填充型腔，直接决定金属液的充填形式与铸件质量[[S41,S37,S39]]。

配套的排溢系统由**溢流槽（Overflow）** 和**排气槽（Vent）** 组成：溢流槽是设置在型腔末端的槽穴，用于储存混有气体、涂料残渣及氧化夹杂的冷污金属液；排气槽设置在溢流槽后方，用于从型腔内排出空气及分型剂挥发产生的气体[[S39,S41,S56]]。

### 不同压铸机类型的结构差异

不同类型的压铸机，其浇注系统的结构形式存在差异[[S41,S39,S60]]：

| 压铸机类型 | 浇注系统组成 | 结构特点 |
| :--- | :--- | :--- |
| 热室压铸机 | 直浇道、横浇道、内浇口 | 压室与坩埚直接连通，无余料（料柄） |
| 立式冷室压铸机 | 直浇道、横浇道、内浇口、余料 | 开模前余料由反料冲头切断并顶出 |
| 卧式冷室压铸机 | 直浇道、横浇道、内浇口 | 余料与直浇道合为一体，开模时随动模脱离定模 |
| 全立式冷室压铸机 | 直浇道、横浇道、内浇口 | 余料与直浇道合为一体，余料轴线垂直于水平方向 |

## 分类

浇注系统的分类可依据内浇口的位置与形式、金属液导入方向及横浇道过渡区形式等多个维度进行划分[[S41,S39,S30]]。

### 按内浇口位置与形式分类

| 类型 | 特点 | 适用铸件 |
| :--- | :--- | :--- |
| **侧浇口** | 行业应用占比约70%，开设在模具分型面上，可从侧面或端面搭接进料，设计与制造简单，浇口去除难度低，适应性极强[[S39,S57]] | 各类大小压铸件，尤其板类、浅型腔盘盖类、壳类，单型腔及多型腔模具均可使用 |
| **中心浇口** | 金属液从铸件中心位置导入，有利于均匀充型、缩短流程 | 具有中心开口的大尺寸压铸件 |
| **顶浇口** | 浇口位于型腔顶部，利用重力方向进料 | 结构适宜的特定形状铸件 |
| **点浇口** | 小截面圆形浇口，充填速度高 | 外观要求高、需自动断浇口的薄壁件 |
| **环形浇口** | 在管状铸件端部沿圆周均匀进料，流程最短，排气条件好 | 中空管状、环形压铸件 |
| **缝隙浇口** | 沿型腔壁长条缝隙进料 | 大型薄壁平板类铸件 |

- **切向浇口**：金属液沿型腔壁切线方向导入，可减少对型芯的直接冲击。切向浇口的典型结构如图示[[S5]]。

## 关键设计参数

### 内浇口截面积

内浇口截面积是浇注系统设计的核心参数，常用经验计算公式为[[S52,S18,S27]]：

$$A_g = \frac{G}{\rho v_g t}$$

式中：
- $A_g$ ——内浇口截面积（mm²）
- $G$ ——通过内浇口的金属液总质量（g）
- $\rho$ ——液态金属密度（g/cm³）
- $v_g$ ——内浇口处金属液流速（m/s）
- $t$ ——型腔填充时间（s）

### 内浇口速度

不同压铸合金的推荐内浇口速度范围如下[[S27,S35,S24]]：

| 合金类型 | 推荐内浇口速度（m/s） |
| :--- | :--- |
| 铝合金 | 17～40 |
| 锌合金 | 30～60 |
| 镁合金 | 40～50 |

> **注意**：内浇口速度超过40 m/s时，模具冲蚀风险会显著上升[[S24]]。

### 内浇口厚度

内浇口厚度的设计需根据铸件凝固模数确定，分合金经验公式如下[[S27,S18]]：

| 合金 | 经验公式 | 备注 |
| :--- | :--- | :--- |
| 铝合金 | $d_{Al} = 3.7M + 0.5$ | $d$ 单位 mm；$M = V/A$，单位 cm |
| 镁合金 | $d_{Mg} = 2.3M + 0.4$ | $V$ 为压铸件体积（cm³） |
| 锌合金 | $d_{Zn} = 3.3M + 0.4$ | $A$ 为压铸件表面积（cm²） |

常规设计中，内浇口厚度取相连铸件壁厚的20%～40%即可满足要求[[S27,S18]]。

NADCA 规范进一步要求内浇口厚度不超过铸件壁厚的75%，且内浇口宽高比应大于10[[S46]]。

### 填充时间

理想充填时间的经验计算公式为[[S35]]：

$$t_{fl} = k \times \frac{T_g - T_f + S \times Z}{T_f - T_d} \times T_{cw}$$

式中：
- $t_{fl}$ ——理想充填时间（s）
- $k$ ——经验常数，取34.09 s/m
- $T_g$ ——浇注温度（℃）
- $T_f$ ——最低金属流动温度（℃）
- $T_d$ ——模温（℃）
- $S$ ——固相分数（%）
- $Z$ ——变换因子
- $T_{cw}$ ——铸件特征壁厚（m）

### 横浇道截面积比

横浇道截面积与内浇口截面积的推荐比值如下[[S15,S27,S25]]：

| 压铸机类型 | 横浇道截面积与内浇口截面积之比 |
| :--- | :--- |
| 冷室压铸机 | 3～4 |
| 热室压铸机 | 2～3 |

### 直浇道脱模角

分流锥脱模斜度的推荐范围为3°～15°，通常取5°左右即可保证直浇道余料顺利从浇口套中脱出[[S3,S45,S27]]。

### PQ² 图与工艺窗口判定

PQ² 图基于流体力学伯努利定律构建，是验证浇注系统设计及优化压铸系统匹配的核心工具[[S59,S32,S38]]。

- **压铸机特性线（ML）**：金属压力随流量增大而线性降低，由最大金属静压与空压射最大流量两个极限点绘制。
- **模具压力线（DL）**：浇注系统阻力随流量平方线性上升。
- **工作点（E）**：两条线的交点，反映模具与压铸机的实际匹配工况[[S59,S32,S11]]。

工艺窗口判定规则：由最大/最小内浇口速度对应的两条压力水平线和最大允许填充时间对应的最小流量垂直线共同围成的阴影区域即为合格工艺窗口，实际工作点必须完全落在该区域内，才能保证铸件充型合格并具有充足的工艺调整柔性[[S35,S19,S11,S4]]。

## 设计准则

以下12项设计准则沿用多年工程经验[[S10,S26,S17,S46,S47]]：

| 序号 | 设计准则 |
| :---: | :--- |
| 1 | 浇注系统截面积从直浇道到内浇口逐步缩小，减少卷气与吸气 |
| 2 | 尽量减少浇注系统转折，避免卷气及压力损失过大 |
| 3 | 保证金属液充型流程最短，内浇口到型腔各部位距离尽量均等 |
| 4 | 内浇口应避开液流障碍位置，禁止金属液正面冲击型芯或型壁，避免冲蚀、粘模和流态混乱 |
| 5 | 内浇口优先设置在铸件厚壁区域，防止内浇口金属过早凝固，保障压力传递完成补缩 |
| 6 | 金属液进入型腔后的流向尽量顺沿型腔壁及加强肋的方向 |
| 7 | 优先采用单个内浇口；多内浇口设计时必须避免多路金属液进入型腔后相互冲击产生涡流、卷气和氧化 |
| 8 | 优先形成顺序填充模式，保障排气顺畅 |
| 9 | 金属液充型过程中不得提前封闭分型面、溢流槽和排气口 |
| 10 | 禁止在浇口区域形成额外热节，避免缩松缺陷 |
| 11 | 内浇口不得设置在无加工余量的高精度、高表面质量要求部位 |
| 12 | 浇注系统结构需便于后续从铸件上切除 |

NADCA 设计规范进一步细化了对浇口位置选择的工程原则：高表面质量、低孔隙率要求的区域应优先获得直接金属流供给；对称型腔布局的模具，所有型腔浇口应采用完全一致的设计方案[[S46]]。横浇道截面积变化规则要求主横浇道截面积大于所有分支横浇道截面积之和，流道在弯位处递增3%～5%，在Y型接头处递增3%～10%[[S46]]。

## 缺陷关联与优化

浇注系统的设计不当与常见压铸缺陷之间存在直接的因果关系[[S10,S20,S29,S36,S51]]：

| 缺陷类型 | 主要诱因 | 优化方案 |
| :--- | :--- | :--- |
| **卷气/气孔** | 多内浇口液流对冲；流道截面积突变；充型提前封闭排气通道 | 合并多余横浇道降低液流交汇冲击；调整流道保持收敛截面；在液流最终汇聚位置增设溢流槽和排气通道 |
| **热节过大/缩松缩孔** | 内浇口布置在薄壁区域导致压力无法传递；浇口局部形成额外热节 | 将内浇口移至铸件厚壁处；减小浇口局部金属聚集体积并适配冷却系统布局 |
| **冷隔/流痕** | 充型流程过长；充型速度过低导致金属液前沿提前冷却；多股冷金属液汇流未完全熔合 | 缩短充型流程；适当增加内浇口截面积降低温降；远端增设大容量溢流槽容纳低温前沿金属 |
| **型芯冲蚀/粘模** | 金属液正面高速冲击型芯或型壁 | 调整内浇口角度改变液流流向；适当降低内浇口流速 |

充型流态方面，符合收敛截面积设计规则的浇注系统可引导金属液以平缓层流状态完成充型，大幅减少紊流产生概率。若存在截面积突变、多浇口对冲、流道转折过多等不当设计，会直接诱发大量紊流，导致金属液卷吸型腔气体形成分散性气孔[[S10,S28,S20]]。

从量化优化效果来看，铝合金散热器浇注系统经设计优化后，铸件裹气率从原方案的19.29%降至3.12%；铝合金凸轮轴盖浇注系统通过将6道横浇道改为5道，显著降低多股金属流汇合卷气风险，经X射线探伤无宏观缩孔缺陷[[S28,S48]]。

两种不同浇注系统设计的金属液流速分布对比热力图直观展示了结构差异对充型流态的影响效果[[S7]]。

## 与排溢系统及温控系统的关系

### 与排溢系统的协同设计

浇注系统与排溢系统的配合规则如下[[S54,S46,S36,S44,S1]]：

- 溢流槽应设置在液流最后到达的位置或两股液流的交汇位置。
- 溢流入口总截面积约为所有内浇口总截面积的1/2。
- 排气槽总截面积约为内浇口总截面积的20%，深度不超过1 mm。
- 浇注系统的顺序填充流向需引导型腔气体向排气槽/溢流槽侧流动，将裹挟气体、氧化夹杂的低温前沿金属完全推入溢流槽。

### 与冷却/加热系统的协同设计

冷却水道布置在型腔内热量集中区域，水道直径通常取φ8～φ18 mm，水道孔壁距型腔壁不小于10 mm[[S54]]。针对浇注系统区域的冷却通道，需布置在横浇道附近且与内浇口保持足够间距，不得靠近内浇口，以防止内浇口位置金属过早凝固导致压力传递中断影响补缩[[S54,S20]]。受热最剧烈的浇口套区域需单独设置冷却结构，可采用螺纹冷却水槽或环形水套保证换热效率[[S54,S20,S33]]。压铸模具预热阶段通过加热系统将模温提升至150～180℃，以匹配浇注系统的充型温度需求并避免冷隔缺陷[[S33,S29]]。

### 三大子系统协同运行逻辑

浇注-排溢-冷却三大子系统协同运行：浇注系统负责控制金属液的充型流态和压力传递路径；排溢系统同步排出浇注系统引导过来的型腔气体和低温夹杂金属；冷却系统根据浇注系统带来的型腔温度分布精准控制各区域凝固速率。三者配合实现顺序充型、顺序凝固的目标，最终大幅降低压铸件气孔、缩松、冷隔缺陷的发生率[[S29,S22,S28]]。

## 标准与规范

涉及压铸浇注系统设计与计算的国内外主要标准与规范如下：

- **GB/T 8847-2003《压铸模术语》**：明确定义浇注系统的标准组成“直浇道、横浇道和内浇口”[[S34]]。
- **GB/T 8844-2003《压铸模技术条件》**：为国内压铸模具的零件与总装配提供通用技术约束，国内工业场景下的浇注系统工程化设计需满足该标准的技术条件[[S8]]。
- **NADCA Gating Die Casting Die 设计规范**：北美压铸协会发布的系列设计手册，提出内浇口厚度不超过铸件壁厚75%、宽高比大于10、横浇道截面积变化规则、溢流槽入口总截面积比等一系列工程化设计准则[[S46,S23,S40]]。

## 应用案例

### 薄壁件

铝合金下箱体（平均壁厚3 mm，材质EN AC 43400，外观要求高）通过MAGMASOFT仿真调整浇口布局，成功解决充型末端两股金属流汇合卷气缺陷，铸件外观质量完全符合后续喷涂工艺要求[[S12]]。

### 复杂结构件

复杂盖形ZL102铝合金压铸件（外形尺寸450 mm × 120 mm × 45 mm，最大壁厚16 mm，最小壁厚3 mm，要求100%无渗漏）沿铸件长度方向均匀布置6道内浇道，大幅缩短金属液充型流程，保障铸件平稳充型并满足零渗漏要求[[S2]]。

### 大尺寸件

大尺寸铝合金汽车发动机盖压铸件按等体积原则将铸件划分为5个区域，每个区域配置1道扇形内浇道，配合常规梯形横浇道布局，溢流槽统一设置在金属液流动末端，实现全型腔均衡充型[[S58]]。

### 对称结构件

铝合金摩托车曲轴箱盖沿分型线对称布置2道扇形内浇道，除流动末端外额外在侧边增设独立溢流槽以提前捕获初始冷污金属液，有效降低铸件缺陷率[[S58]]。
