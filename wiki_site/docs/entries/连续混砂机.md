## 概述

连续混砂机是专门用于混拌无粘土型砂的砂处理核心主机，配置一个或两个用于混拌和推送型砂的螺旋及泵系统，可实现连续进料、连续出料，多用于混制自硬砂、树脂砂等化学粘结剂砂[[S43,S38]]。该术语已被 GB/T 25370-2020《铸造机械 术语》正式纳入铸造机械术语体系，英文对应名为 continuous mixer[[S17,S23]]。

在铸造砂处理系统中，连续混砂机是自硬砂工艺的最适配混砂设备，可实现边混砂、边填砂、边紧实的作业模式，按需产出混制好的型砂，避免浪费。配套计算机自动控制原辅料定量，操作便捷，在大型铸件生产和大批量连续造型场景中优势突出[[S25]]。

## 分类

### 按混砂原理分类

连续混砂机按混砂原理可分为三类[[S30,S39,S34]]：

| 类型 | 工作原理 | 典型特点 |
|------|----------|----------|
| 双碾盘连续混砂机 | 物料在两个碾盘中走“8”字形运动，完成碾压搓揉混合 | 生产率比间歇碾轮式混砂机高约50% |
| 搅笼/螺旋式连续混砂机 | 依靠内置螺旋叶片边推进边搅拌完成混制 | 结构简洁，应用最广 |
| 高速单槽连续混砂机 | 单段混砂槽内一次完成混合，主轴转速可达1000 r/min | 物料停留时间仅3~5 s，停运后可完全排空 |

### 按技术代际分类

现代化学粘结剂砂混砂机的发展可分为三代[[S11,S46]]：

| 代际 | 类型 | 主轴转速 | 物料停留时间 | 适用粘结剂体系 |
|------|------|----------|-------------|----------------|
| 第一代 | 低速单槽混砂机 | 约85 r/min | 约30 s | 固化时间30 min以上的慢固化体系 |
| 第二代 | 双初级预混+高速次级混砂机 | — | 次级约5 s | 固化时间最短5 min的快固化体系 |
| 第三代 | 单槽高速混砂机 | 约1000 r/min | ≤5 s | 快固化体系，可配气相抑制剂自清洁系统 |

第三代混砂机将所有组分一次完成混合，总混砂时间控制在5 s或更短，粘结剂消耗量相比前两代可显著降低，允许多种型砂配方的即时切换[[S11,S34]]。

### 按适配工艺分类

连续混砂机可分别适配树脂自硬砂（呋喃、酚脲烷等体系）、水玻璃自硬砂、冷芯盒砂、快固化冷芯粘结剂砂等工艺，部分机型也可适配对型砂强度要求不高的粘土背砂连续混制[[S22,S26,S9]]。

## 结构与组成

搅笼式连续混砂机的核心结构由五大模块组成：储砂单元、混碾执行机构（搅笼，含传动装置、进料口、推进叶片、搅拌叶片、出料口、反向叶片）、臂架输送装置、液料添加单元（含树脂/固化剂泵、管路、定量装置）、电控控制系统[[S39,S4]]。主流双臂机型的一级输送搅笼采用低速输送模式，二级混砂搅笼采用高速混制模式[[S2]]。

## 混制工艺

### 单砂双混法（主流工艺）

目前行业应用最广的混制工艺为单砂双混法：原砂先加入固化剂充分混匀，之后在搅笼中部特定位置加入树脂再次混匀后直接卸砂输出。试验表明，该工艺得到的自硬砂终强度高于先加树脂后加固化剂的工艺，且随着存放时间的延长，终强度下降速度更慢[[S2]]。

### 其他混制工艺

树脂自硬砂的混制工艺总体上可分为三种[[S2]]：

- **单砂单混法**：各种物料同时定量加入混砂机一次性混拌，高速单搅笼连续混砂机常用。
- **双砂三混法**：将砂分为两份分别与固化剂和树脂预混，再在二级混砂器中混合。此方法结构复杂、能耗大，混砂质量和效率较低。
- **单砂双混法**：先加固化剂混匀后再加树脂混匀，效果最优。

## 工艺参数与控制要求

### 定量精度

连续混砂机的核心定量精度行业要求[[S6,S13,S31]]：

| 控制对象 | 允许误差范围 |
|----------|-------------|
| 砂流量 | ≤ ±5% |
| 液体粘结剂（树脂）流量 | ≤ ±3%（部分标准要求 ±1%~±2%） |
| 固化剂流量 | ≤ ±3%（部分标准要求 ±0.8%~±1.5%） |
| 混砂均匀性 | ≤ ±10% |

### 砂温控制

连续混砂机配套型砂的合理砂温控制区间为20~35 ℃，生产过程中砂温波动值需控制在 ±3 ℃ 以内。砂温过高将导致树脂砂预固化失效，砂温过低则延缓固化速度，影响生产节拍[[S45,S24,S33]]。

### 物料停留时间

不同类型连续混砂机的物料停留时间范围[[S34,S46,S37]]：

| 机型 | 停留时间 |
|------|----------|
| 单槽高速型连续混砂机 | 3~5 s |
| 低速单槽型连续混砂机 | 约30 s |
| 冷芯盒配套连续混砂机总混碾时长 | < 1 min |

## 与压铸及制芯工艺的关联

连续混砂机适配压铸、低压铸造、重力铸造全工艺链的大批量连续生产场景。专门为压铸件砂芯生产配套的连续混砂机可直接安装在冷芯盒机上方，以送砂小车或输送带向一台或多台冷芯盒射芯机连续稳定供砂，实现现混现用，完全匹配冷芯盒砂存放期短、不宜长距离运输的特性[[S37,S18,S15]]。

同时，连续混砂机可与热芯盒、温芯盒、无机砂制芯工艺兼容，适配温芯盒20 ℃以下24 h长存放期的芯砂制备需求，也可满足无机水玻璃砂的连续均匀混制要求，实现砂芯性能的稳定输出[[S47,S12]]。

## 与间歇式混砂机的对比

连续式与间歇式混砂机的主要优劣势对比[[S8,S20,S35]]：

| 对比维度 | 连续混砂机 | 间歇式混砂机 |
|----------|-----------|-------------|
| 生产效率 | 双碾盘连续混砂机比同规格间歇碾轮式高约50%；第三代高速机型产能可达第一代的2~3倍 | 存在加料、卸料辅助时间，效率较低 |
| 能耗 | Omega自清洁机型闲置功耗仅为传统高速混砂机的1/3 | 能耗较高 |
| 砂质稳定性 | 通过高速剪切作用均匀分散粘结剂，头尾砂性能无明显波动 | 可能存在批次间波动 |
| 粘结剂消耗 | 总消耗量更低 | 消耗量较高 |
| 占地面积 | 无需多台串联布局，整体紧凑 | 多台串联时占用面积大 |
| 操作灵活性 | 切换配方相对不便 | 可快速调整不同批次砂的配方，适配小批量多品种生产 |

## 标准与规范

GB/T 25370-2020《铸造机械 术语》已将连续混砂机（continuous mixer）纳入正式术语体系[[S23,S17]]。GB/T 5611-2017《铸造术语》给出了连续混砂机的基本定义[[S38]]。

现行混砂机 JB/T 系列通用技术要求在以下方面适用于连续混砂机的管控[[S14,S42]]：

| 技术要求项 | 限值/标准 |
|-----------|----------|
| 滑动轴承温升 | ≤ 25 ℃，最高工作温度 ≤ 60 ℃ |
| 滚动轴承温升 | ≤ 35 ℃，最高工作温度 ≤ 70 ℃ |
| 传动系统清洁度 | ≤ 润滑油油标下限的 0.03% |
| 空运转噪声（盘径 < 2000 mm） | ≤ 80 dB(A) |
| 空运转噪声（盘径 ≥ 2000 mm） | ≤ 85 dB(A) |
| 刮砂板使用寿命 | ≥ 500 h |
| 首次大修期工作时长 | ≥ 12000 h |

## 安全与维护

### 安全操作规范

连续混砂机的核心安全操作要求[[S29,S41]]：

- 操作人员进入混砂腔体内部作业前，必须先切断主机总电源并悬挂醒目安全警告牌，防止误启动引发人身伤害。
- 检修作业完成后需开机核验十字头主轴旋转方向，确认主轴按逆时针方向正常运转。
- 新安装或大修后的设备需完成不少于2 h的带额定负载试运转，混碾过程中卸砂门无颤抖现象，立轴、十字头无上下窜动；允许带载启动的机型不得出现电机过热或跳闸故障。

### 维护保养要点

分级维护保养周期与要点[[S29,S34,S44]]：

- 电动丝杠内部部件、电动机轴承需每年至少检修一次。
- 日常作业中需定期检查转子、刮板、搅拌叶片的磨损状态，磨损超限后需立即采用铬基焊条堆焊修补或直接更换新件。
- 单槽高速型连续混砂机混砂叶片需每班完工彻底清洁，出砂口每间隔2 h清理一次，避免固化砂块堆积。
- 单槽慢速型连续混砂机每次停机超过1~2 min时需清空混砂槽内余料，防止砂料硬化结块卡死部件。

### 核心配合间隙控制

双层转子式连续混砂机的核心配合间隙要求[[S29,S10]]：

| 间隙控制部位 | 控制范围 |
|-------------|---------|
| 上下转子间距 | 10 ± 3 mm |
| 下层转子与底板、底刮板间隙 | 3~5 mm |
| 刮片到底板间隙 | 5~8 mm |
| 前刮片与可换摩擦板径向间隙 | 3~5 mm |

### 常见故障及排除

树脂砂连续混砂机高频常见故障覆盖混砂机构转不动、液料/砂子定量不准、卸砂门漏砂、混砂均匀度达不到工艺要求等典型问题，固化剂泵等易堵液料泵在每次作业结束后需将吸入端放入温水中清洗，可有效避免泵体堵塞故障[[S5,S3]]。
