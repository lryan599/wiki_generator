## 概述

在压铸工程领域，有限元分析（Finite Element Analysis, FEA）又称有限元法（Finite Element Method, FEM），是压铸模CAE（计算机辅助工程）的核心数值模拟技术[[S100]]。它通过将连续体离散化为有限数目单元的组合体来获得工程近似解，用于求解复杂压铸件与模具的结构强度、刚度、热传导、弹塑性变形及多物理场耦合问题[[S53]]。在NADCA（北美压铸协会）的相关规范中，FEA被列为压铸模温度场热设计的四种主流技术手段之一，可用于精准定位冷却水路位置并确定热控热电偶的设定点温度[[S33]]。

## 数学基础与基本原理

有限元法的核心思想是连续体离散化。将原本连续的求解域假想分解为有限数目的单元组合体，通过分片插值的方式用离散节点上的未知量近似表示全域的待求场量，可灵活适应复杂曲面边界，有效避免有限差分法在曲面离散时产生的阶梯误差[[S14,S53,S87]]。

### 单元类型

压铸场景常用的有限元单元分为二维与三维两类：二维问题采用四结点四边形等参元或三角形单元，三维问题采用四结点四面体单元或八结点六面体等参元。工业实践中一般优先使用低阶线性单元以避免不可压缩约束条件不满足的问题；高阶二次单元因迭代计算量大、稳定性差，在压铸塑性大变形场景中较少使用[[S15,S20,S43]]。

### 控制方程

压铸有限元分析遵循三类核心平衡方程：充型过程不可压缩流体满足连续性方程与Navier-Stokes动量方程；结构分析场景基于虚功平衡方程；热传导场景基于能量守恒方程。所有控制方程通过变分原理或加权余量法转化为单元矩阵后进行总装求解[[S53,S77,S87]]。

### 材料本构关系

压铸工艺场景适配的有限元材料本构关系分为三类：模具结构分析场景采用弹塑性本构关系；高温大变形场景采用刚塑性或刚粘塑性本构关系；凝固过程需专门二次开发适配准固相区的应力应变本构模型，以满足固液两相区的特殊力学行为求解需求[[S26,S87,S93]]。

## 与有限差分法、有限体积法的区别

在压铸仿真场景下，三种主流数值方法的核心差异如下[[S8,S75,S89]]：

| 比较维度 | 有限元法 (FEM) | 有限差分法 (FDM) | 有限体积法 (FVM) |
|---------|--------------|-----------------|-----------------|
| 数学基础 | 积分方程与变分原理 | 微分方程泰勒展开近似 | 控制体积积分 |
| 网格类型 | 任意非规则单元 | 正交规则网格 | 互不重叠控制体积 |
| 边界拟合 | 精度高，适应复杂几何 | 复杂边界拟合差 | 兼顾边界适应性 |
| 算法复杂度 | 高，对硬件资源要求高 | 简单，计算成本低 | 中等 |
| 守恒特性 | 非严格全局守恒 | 非严格全局守恒 | 严格满足全局积分守恒 |
| 压铸主要应用 | 模具力学分析、结构设计 | 充型、温度场快速模拟 | 充型流动求解 |
| 典型软件 | ProCAST | MAGMASOFT | AnyCasting, FLOW-3D Cast |

## 分类

### 按物理场分类

按求解物理场的维度，压铸有限元分析可分为三类[[S64,S100,S86]]：

1. **结构力学场有限元分析**：求解模具和铸件的应力、应变、变形分布，用于预测热裂、热疲劳、模具强度失效等缺陷。
2. **温度/热传导场有限元分析**：求解压铸全周期模腔及铸件的温度分布变化，用于优化冷却水路布局。
3. **流体流动场有限元分析**：求解高压高速充型过程金属液的流动速度与压力分布，预测卷气、浇不足等充型缺陷。

### 按求解类型分类

按求解类型维度，可分为[[S20,S64,S86]]：

1. **静态求解**：稳态下模具结构的静强度与刚度校核。
2. **动态求解**：压铸冲击、振动、疲劳循环加载过程的动态响应分析。
3. **稳态求解**：压铸生产达到热平衡状态下的模腔温度场分布计算。
4. **瞬态求解**：高压充型、非稳态凝固及多生产循环下温度场/应力场的动态演化过程分析。

## 在压铸过程中的核心应用

FEA在压铸全流程中包含五类核心应用[[S53,S22,S64]]：

1. **充型过程流动与卷气模拟**：通过求解Navier-Stokes流体控制方程、质量守恒方程及能量守恒方程，实现金属液流动速度场、充型顺序及卷气区域的动态计算。
2. **凝固过程缩松缩孔预测**：通过非稳态热传导方程求解不同时刻的温度场分布，识别孤立液相区与热节位置。
3. **模具温度场与应力场分析**：模拟多批次循环生产下模具的热平衡状态，定位热应力集中区域。
4. **铸件变形与残余应力预测**：基于热弹塑性或热粘塑性本构模型，计算凝固冷却阶段的应力应变演化。
5. **开模与顶出分析**：分析顶出力分布、铸件与模具界面的摩擦接触状态，预判顶出变形风险。

## 核心流程与参数

### 前处理

前处理占整个有限元分析流程约40%~45%的工作量，是决定模拟精度的关键环节[[S13]]。

#### 网格划分类型

压铸FEA前处理的网格划分为三类主流类型[[S60,S91,S99]]：

- **结构化六面体网格**：网格排列规则、计算速度快，适用于轮类等结构规整的铸件。
- **非结构化四面体/三棱柱网格**：边界拟合能力强，适用于结构复杂的压铸件。
- **自适应网格**：可根据局部流场/温度场梯度自动动态调整网格疏密，在保证关键区域计算精度的同时降低整体计算量。

#### 网格质量控制

压铸场景下有限元网格质量需满足三项核心控制标准[[S57,S70,S6]]：

| 质量判据 | 控制要求 |
|---------|---------|
| 单元长宽比（三角形最长边与最短边之比） | 推荐控制于5以内 |
| 单元半径比（内切圆与外接圆之比） | 不低于0.2 |
| 二面角 | 无接近0°或180°的严重畸变单元 |

#### 网格疏密配置

遵循"重点区域加密、非重点区域粗化"的原则：薄壁处、浇道转弯处和热节预测区域网格尺寸需小于铸件最小壁厚，保证关键区域至少3层以上网格；模具等非铸件区域可适当放大网格尺寸以降低总计算量。主流工程案例中铸件常用网格尺寸为1~4 mm，模具厚大区域网格尺寸可放宽至3~4 mm[[S34,S70,S84]]。

#### 典型边界条件设定

压铸场景典型边界条件的工业常规取值区间如下[[S12,S30,S84,S96]]：

| 参数 | 常规取值范围 |
|-----|------------|
| 快压射阶段活塞速度 | 0.7~4 m/s |
| 内浇道局部充型速度 | 可达30 m/s级 |
| 铝合金模具初始温度 | 150~220℃ |
| 镁合金模具初始温度 | 120~200℃ |
| ADC12类铝合金浇注温度 | 620~680℃ |
| 锌合金浇注温度 | 390~420℃ |
| 镁合金AM50/AZ91D浇注温度 | 640~710℃ |

#### 界面热交换系数

不同实体间界面换热系数的典型工业取值[[S21,S24,S57,S84]]：

| 界面类型 | 常规换热系数 (W/(m²·K)) |
|---------|------------------------|
| 压铸件与H13模具接触面 | 1000~2000 |
| 高压接触峰值工况 | 可达42000 |
| 模具各分体模块间 | ~1000 |
| 模具外表面与空气对流辐射 | 10~20 |

### 求解器设置

压铸FEA专用求解器需根据模拟目标选择计算类型，分为独立温度场求解、独立流场求解及流热耦合求解三类。迭代算法可选择SOR逐次超松弛迭代法或GMRES广义最小残差法。显式动力学求解需控制沙漏总能量占比低于模型总内能的10%以保证结果有效性；时间步长可由软件根据最小单元尺寸自动计算[[S23,S27,S49,S58]]。

压铸场景隐式求解的收敛控制常规要求[[S23,S27,S72]]：

| 控制参数 | 常规取值范围 |
|---------|------------|
| 流动场迭代收敛公差 | ≤1×10⁻³ |
| 温度场迭代收敛公差 | ≤0.1℃ |
| 最大迭代步数 | 50~200 |

### 后处理

压铸FEA后处理的标准化输出内容包括三类物理场的可视化[[S58,S61,S63]]：

- **温度场输出**：充型/凝固过程温度分布云图、温度梯度云图、任意点温度-时间曲线、孤立液相区自动识别统计。
- **流场输出**：金属液流动速度矢量图、充型动态动画、卷气区域预测图、最后填充位置标识。
- **应力应变场输出**：模具热应力分布云图、铸件残余应力云图、变形位移分布结果，支持任意方向三维剖视分析生成CT切片视图。

### 通用有限元平台

- **ANSYS**：通用多物理场有限元仿真平台。在压铸场景下主要用于模具温度场循环仿真及铸件服役状态的结构力学分析，已被验证用于薄壁开关盒类压铸件的模具温度场预测，仿真与实测误差可控制在10%以内[[S56,S62,S76]]。

- **Abaqus**：通用非线性有限元仿真平台。擅长处理压铸模具钢的高温非线性力学行为计算及铸件大变形服役行为仿真，常与HyperWorks前处理工具配合使用完成压铸件轻量化结构的性能校核[[S19,S47,S59]]。

### 专业前处理与辅助工具

- **ANSA**：作为专业前处理工具，承担复杂曲面压铸件模型的高质量几何清理与快速面/体网格生成工作。尤其擅长处理汽车大型一体化压铸结构件等几何特征复杂的模型，可输出兼容所有主流铸造仿真求解器的网格文件[[S78,S81]]。

- **HyperWorks (HyperMesh/OptiStruct)**：HyperMesh专注于高效高质量网格划分，适配绝大多数压铸仿真软件的输入格式，可完成复杂压铸件的几何简化和网格质量控制；OptiStruct负责压铸结构件的拓扑优化与尺寸优化，支撑汽车领域压铸件轻量化设计[[S35,S51,S59]]。

- **Deform**：专业金属塑性成形有限元软件。在压铸FEA全流程中不覆盖常规充型凝固仿真场景，专注于压铸后液态模锻/挤压压铸复合成形阶段的仿真分析，可精准模拟压力下补缩过程中金属的三维流动、等效应力场及温度场分布，有效预测复合成形的缩孔缺陷改善效果[[S41,S90,S97]]。

## 与缺陷机制的关联

FEA识别典型压铸缺陷采用标准化数值判据体系[[S4,S37,S48,S67]]：

| 缺陷类型 | 常用判据 | 判定原理 |
|---------|---------|---------|
| 缩松缩孔 | Niyama判据 (G/√R)、残余熔体模数、概率缺陷参数 | 温度梯度G与冷却速率√R之比低于临界阈值时，判定易生成孔洞缺陷 |
| 热裂 | RGD判据 | 准固相区拉应力超过材料临界断裂阈值且区域横截面积较大时，判定易产生热裂 |
| 冷隔 | 汇合区温度低于合金固相线 | 充型结束时两股金属液汇合区域温度判断未完全熔合 |
| 卷气 | 空气体积分数、流前交汇与排气判断 | 两股流前交汇区域或排气不畅的最后充填区域的空气体积分数判定 |
| 变形 | 残余应力超过屈服强度的位移偏移量 | 全域残余应力超过铸件屈服强度区域的位移偏移量判定 |

NADCA公开的典型案例验证了FEA在缺陷诊断中的工程有效性：针对AL380铝合金汽车阀件出现的肋根位置开裂问题，通过25000单元、13000节点的FEA凝固仿真（浇注温度650℃、模温250℃），识别肋根位置热节在约6秒的凝固周期中长时间保持半固态，形成恶劣热应力梯度导致开裂[[S73]]。

## 工程应用实例

NADCA公开的压铸工程案例中，锌合金Zamak3按钮锁铸件通过45000单元、23000节点的FEA充型仿真，识别出原浇注设计下远离内浇口区域的最后充型缺陷区，优化内浇道布局后大幅降低了表面缺陷率[[S11]]。

国际头部车企的FEA落地案例[[S40]]：
- 德国大众使用MAGMASOFT完成镁合金变速箱体的多物理场FEA仿真优化，支撑该零件批量应用于Passat及奥迪A4/A6车型。
- 美国福特汽车使用ProCAST完成5.4L Triton发动机壳体的流场-温度场耦合FEA分析，大幅缩短了零件开发周期。

## 局限性

压铸有限元分析在实际应用中存在以下主要局限：

1. **计算成本高**：有限元法离散算法复杂度高，对计算机硬件配置要求较高，一定程度上限制了在中小压铸企业的普及应用。常规流程中前处理占40%~45%工作量，后处理占50%~55%，实际数值计算仅占总工作量的约5%[[S13,S54]]。

2. **大规模仿真算力瓶颈**：以新能源汽车大型一体化压铸件为代表的超大规模仿真任务，当总网格量达到1.22亿级别时，使用96核心计算资源仍需约13.7小时才能完成全部充型过程计算[[S39]]。

3. **网格畸变误差**：单元形状出现严重畸变会直接降低求解精度，甚至引发矩阵缺秩问题导致仿真过程无法收敛[[S13]]。

4. **材料数据库依赖性**：主流商用压铸CAE软件自带的通用材料数据库默认界面换热系数HTC数据集与工业实际工况偏差较大，需工程师手工迭代十余次进行参数修正才能匹配实测温度数据，仿真校准门槛很高[[S52]]。

5. **流固耦合处理局限**：常规压铸有限元仿真大多对流场、温度场、应力场采用解耦分步计算模式，很难完整覆盖压铸特有的5~50 m/s超高速湍流充型过程中气-液-固三相耦合的复杂交互行为，且目前通用仿真体系未建立专门适配压铸非平衡凝固特性的缩孔缩松缺陷预测判据[[S39,S54]]。

## 相关标准与指南

1. **NADCA标准与指南**：NADCA官方出版物《Die Casting Dies: Designing》明确规定压铸模具热分析场景下有限元分析的通用使用规范，要求将FEA热分析结果与基础热平衡方法结合使用以优化冷却水道布置设计[[S33]]。NADCA #207-2003 H-13钢显微组织验收标准及其配套参考图谱，是压铸FEA仿真中模具钢热疲劳寿命预测的重要参考对标依据[[S28,S55]]。此外，行业通用的P-Q²仿真校核最佳实践准则要求将仿真得到的压射工作点与压铸机实际功率边界线进行对比验证[[S32]]。

2. **ISO标准**：ISO 12165国际标准规定了压铸模具零部件的统一术语和符号体系，是压铸FEA仿真前处理阶段几何建模与模型命名标准化的通用参考规范[[S101]]。ISO 8062国际铸件公差标准可用于校验压铸FEA仿真预测的铸件变形量结果是否符合工业产品公差允许范围[[S38,S83]]。

3. **ASTM标准**：ASTM B系列压铸合金化学成分与力学性能标准及ASTM A609/A609M铸件超声波检测标准，分别作为压铸FEA仿真材料参数输入与仿真预测缺陷结果进行实验对标校验的通用行业依据[[S2,S94]]。

4. **国内行业标准**：国内已发布铸造工艺仿真行业标准，明确规定了压铸仿真工作的统一流程规范，覆盖仿真环境配置要求、核心工作内容定义、标准执行步骤及仿真结果输出与工程应用方法，有效解决了此前压铸仿真操作随意性高、结果无法横向对比的行业问题[[S79]]。
