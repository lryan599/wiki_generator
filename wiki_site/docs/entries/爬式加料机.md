## 概述

爬式加料机（inclined skip charger）是采用卷扬机构牵引沿向上倾斜轨道运行的加料车，将固体炉料提升后装入炉内的专用加料设备，属于铸造领域冲天炉加料设备类别[[S18]]。该设备服务于熔化系统的前端固态物料输送环节，负责将金属炉料（如铝锭、回炉废料）、焦炭、石灰石等从地面输送至炉顶加料口，不属于压铸领域面向液态金属自动舀取与给汤的系统范畴[[S5,S21]]。

> **术语说明**：在现行国家标准 GB/T 25370-2020 中，“爬式加料机”的英文对应名称为 **inclined skip charger**[[S18]]。需注意该中文设备名称在不同工业领域可能指代截然不同的设备。例如，冶金行业中存在采用橡胶履带行走底盘的履带式炉料输送设备，塑料加工行业中存在用于颗粒原料斜坡提升的加料装置，均与铸造行业《铸造机械 术语》标准所定义的爬式加料机在应用场景、结构形式和设计规范上有明确的专业边界差异[[S18,S22]]。

## 分类

爬式加料机依结构特性形成多维度的分类体系，常见分类维度如下[[S5,S7,S14,S19]]：

| 分类维度 | 型式 | 特征说明 |
|---------|------|----------|
| 整机运动特性 | 固定式 | 导轨及整机固定安装，制造成本较低，适配单台熔炉 |
| 整机运动特性 | 回转式 | 以回转盘为中心沿加料平台弧形轨道回转，可支持两台熔炉轮替作业，钢材用量较固定式高出约50%，结构更复杂 |
| 尾架结构 | 长尾型 | 可实现跨车间跨度物料输送，无需额外配置过渡料车 |
| 尾架结构 | 短尾型 | 仅适用于配料系统与加料工位布置在同一车间跨距内的场景 |

此外，依导轨截面形式可将导轨系统分为四型：**I型**（槽钢导轨，轮翼内行走）、**II型**（工字钢导轨，轮翼外行走）、**III型**（工字钢/工型导轨，轮翼内行走）、**IV型**（板梁导轨，轮翼内行走）。选型需依据料桶小车载重等级和厂房布置条件确定[[S19]]。

翻斗加料机是爬式加料机的简易变体，采用倾翻卸料代替料桶开底卸料，结构简化、制造成本低，但卸料过程易造成炉料向一侧倾斜堆积，仅适配小型熔炉或对熔化控制要求较低的生产场景[[S7,S12]]。

## 结构与组成

爬式加料机主要由导轨系统、驱动装置（卷扬机）、料桶小车、尾架支撑、上台架、受料漏斗等核心部件组成。导轨的上、下圆弧段分别由上台架和尾架支撑，中间的直线段以桁架或花架结构保障整体刚性[[S7,S17]]。部分机型配置批料计数装置与牵引钢丝绳保险装置，用于作业状态监测与安全防护[[S14]]。

### 料桶小车

料桶小车是爬式加料机的核心执行部件，按卸料机构原理分为后轮翘起式、转杆式、钢丝绳式三类，分别适配不同熔炉吨位和工况要求[[S11,S19]]：

| 卸料型式 | 动作原理 | 优点 | 局限 | 适用场景 |
|---------|---------|------|------|----------|
| 后轮翘起式 | 卸料时后轮进入导轨上岔道，车架以前轮为中心转动压下吊杆使桶底张开；回程时后轮进入下岔道桶底自动闭合 | 结构简单，动作可靠 | 对上下圆弧段岔道制造安装精度要求高 | 中小型熔炉 |
| 转杆式 | 卸料时转杆触碰炉壁碰块打开桶底；回程时转杆触碰尾架挡板闭合桶底，依靠转杆自重弯矩自锁 | 结构简单，安装调试较容易 | 桶底开启时冲击较大，关闭情况不便检查 | 中小机型 |
| 钢丝绳式 | 卸料时导轨侧挡块推开手柄放松钢丝绳打开桶底；回程时闭合杆触碰上台架复位拉紧钢丝绳闭合桶底，配置液压缓冲器 | 动作平稳可靠 | 结构复杂，维护工作量较大 | 大中型熔炉 |

### 导轨系统结构设计要求

导轨系统的设计需满足纵向刚性、横向稳定性及桁架自身稳定条件，导轨圆弧段曲率半径按小车轮距的关联比例确定。导轨本身强度一般足够，可不必单独验算[[S14,S19,S11]]：

| 设计条件 | 满足条件 | 说明 |
|---------|---------|------|
| 纵向刚性 | *h* / *L* ≥ 1/15 ~ 1/20 | *h*——导轨断面垂直高度；*L*——导轨直线段跨度 |
| 横向稳定性 | *b* / *L* ≥ 1/40 ~ 1/50 | *b*——导轨断面横向宽度 |
| 桁架纵向稳定性 | λ ≤ 120 | λ——桁架总高与纵向长度之比 |
| 上下圆弧段曲率半径 | R₁, R₂ = (1.5 ~ 3) × 小车轮距 | — |

## 工作原理与工作循环

爬式加料机的工作循环逻辑如下所述[[S14,S21,S5]]：

1. **底部装料**：受料漏斗承接来自配料系统的金属炉料（铝锭、回炉废料等）及辅助炉料，料桶小车行进至底部工位承接物料；
2. **倾斜爬升**：卷扬机牵引料桶小车沿倾斜导轨向上爬行，全程沿导轨上下圆弧段及中间直线桁架段运行；
3. **炉口卸料**：料桶小车行至导轨上部卸料段时，卸料机构（后轮翘起、转杆或钢丝绳式）动作，物料从料桶底部卸入熔炉加料口；
4. **回程复位**：卸料完成后料桶小车在卷扬机控制下沿导轨下行，返回底部初始位置，准备下一次装料。

爬式加料机加料动作数目少、单循环周期短，控制逻辑简洁，易于实现自动化。配合电磁配铁秤、称量车、料位控制装置等设备联动，可实现定量配料、输送、加料全流程自动化作业[[S21]]。

## 应用场景

在典型的压铸或铸造车间中，爬式加料机适配以下核心场景[[S20,S9,S15]]：

- **固定短尾爬式加料机配置**（适配3 t/h~5 t/h级集中熔化炉）：每台熔炉各配一台设备，两炉合用同一套配料系统，两炉可同时熔化，实现产能倍增；
- **固定长尾爬式加料机配置**（适配7 t/h~10 t/h级集中熔化炉）：利用长尾机型的过跨能力直接完成两个车间跨距间的物料转运，省去辅助运料设备；
- **回转长尾爬式加料机配置**（低产能场景）：当两炉不同时熔化，采用一台回转长尾爬式加料机和一套配料设备，减少总设备投入。

爬式加料机的服务对象是上游集中熔化炉或保温炉的固态补料环节，与面向压铸主机的自动给汤机（automatic filler / ladle）有明确的功能分工差异：自动给汤机负责从保温炉舀取熔融金属液并定量送入压铸机压射室，单次加料精度可达±2%，服务对象为压铸主机；爬式加料机则解决大吨位批量固体炉料向熔化炉的输送，服务对象为熔化炉系统[[S8,S23]]。

## 与单轨式加料机的比较

爬式加料机与单轨式加料机同为铸造冲天炉领域的主流加料设备，二者各具优劣[[S3,S5]]：

| 对比项目 | 爬式加料机 | 单轨式加料机 |
|---------|----------|------------|
| 加料速度 | 可较高 | 稍低 |
| 载重能力 | 大 | 较小 |
| 适用熔炉吨位 | 可达10 t/h以上 | 一般小于10 t/h |
| 能否过跨 | 能（长尾机型） | 一般不能 |
| 动作程序 | 少 | 多 |
| 自动化实现 | 较易 | 较难 |
| 结构复杂程度 | 复杂 | 简单 |
| 制造安装要求 | 较高 | 一般 |
| 维修工作量 | 较大 | 较小 |
| 设备投资 | 高 | 较低 |

## 优势与局限性

爬式加料机的主要优势为[[S5,S14]]：
- 加料动作少，控制逻辑简单，单循环周期短；
- 易于与电磁配铁秤、称量车和料位控制装置联动，实现全自动加料作业；
- 长尾机型具备跨车间跨度物料直接输送能力，减少过渡转运设备。

其主要局限性为[[S5,S14]]：
- 整机结构复杂，制造精度和安装调试要求高，设备造价偏高；
- 生产现场需配套较深的地坑基础以容纳底部导轨段及料桶受料区域，占用炉后面积较大；
- 相较单轨式加料机，日常维护工作量更大。
