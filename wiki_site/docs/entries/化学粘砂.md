## 概述

化学粘砂是铸造生产中一类典型的表面缺陷，指铸件部分或整个表面上牢固粘附一层由金属氧化物、型砂及粘结剂相互作用生成的低熔点化合物。该粘砂层硬度极高，与铸件表面结合牢固，无法通过常规喷丸或抛丸清理去除，必须使用砂轮打磨[[S42,S46]]。在国内铸造标准GB/T 5611—2017中，化学粘砂的英文标准术语为**burn-on**，其标准别名标注为**烧结粘砂**；行业通用的英文别名还包括**sand burn-on**、**sand burn-in**，以及衍生的**chemical burn-on**[[S46,S51]]。

化学粘砂与机械粘砂（即渗透粘砂，metal penetration）共同构成粘砂缺陷的两大基本类型，但二者的形成本质截然不同：机械粘砂是金属液或金属氧化物在压力下纯物理渗入砂型孔隙的过程，属于物理作用主导的缺陷[[S25]]；而化学粘砂则是高温下金属液与铸型材料发生化学反应生成低熔点化合物，属于化学作用主导的缺陷[[S16,S19]]。行业中还曾存在“热粘砂”的分类概念，但因其本质是金属氧化物与型砂在高温下发生化学反应，现已归入化学粘砂范畴，其别名“热-化学粘砂”仍常见于部分文献[[S3,S32]]。

## 定义与分类体系

### 标准定义

根据GB/T 5611—2017《铸造术语》标准，化学粘砂的定义为：**铸件的部分或整个表面上牢固地粘附一层由金属氧化物、砂子和粘结剂相互作用而生成的低熔点化合物**。该缺陷属于表面缺陷大类下的粘砂子类，硬度极高，只能用砂轮磨去[[S42,S46]]。

### 在铸造缺陷分类中的位置

在中国机械工业标准体系及GB/T缺陷分类框架中，化学粘砂的层级归类如下[[S46]]：

- 表面缺陷（surface defect）
  - 粘砂（burning-on）
    - **化学粘砂（burn-on）**
    - 机械粘砂 / 渗透粘砂（metal penetration）

## 形成机理

化学粘砂的形成是一个多阶段的高温物理化学反应过程，其核心是金属液表面氧化生成的FeO与型砂中的SiO₂反应生成低熔点铁橄榄石，随后该低熔点相渗入砂粒间隙，冷却后将砂粒与铸件牢固粘结为一体。

### 完整的反应流程

浇注过程中，钢液或铁液与铸型内的氧、二氧化碳、水蒸气等氧化性组分接触，首先发生起始氧化反应[[S34]]：

**2Fe + O₂ = 2FeO**

生成的FeO熔点为1370~1380°C，在铸钢浇注温度下呈液态，其对SiO₂的润湿角仅为21°，因此极易润湿石英砂，在毛细作用下渗入砂粒间隙[[S19]]。

渗入砂粒间隙的FeO接着与SiO₂发生核心成相反应[[S19,S34]]：

**2FeO + SiO₂ = Fe₂SiO₄（铁橄榄石，fayalite）**

铁橄榄石的熔点为1205°C[[S6,S10,S34]]。在FeO-SiO₂体系中，当w(SiO₂)=22%处的共晶点熔点更是低至1220°C，该低熔点相的流动性极强，即使在铸件表层已经完全凝固后仍可保持液态，持续向砂型内部渗透，将砂粒粘结为一体[[S10]]。

除上述主反应路径外，还存在以下副反应路径[[S10]]：

**Al₂O₃·2SiO₂ + 2FeO → 2FeO·SiO₂ + Al₂O₃**

该路径解释了含粘土组分的砂型中无需游离SiO₂即可生成铁橄榄石的现象——粘土矿物偏高岭石在高温下分解后也可与FeO反应。

### 高锰钢场景下的特殊反应

在高锰钢浇注条件下，锰元素的氧化产物MnO同样会与型壳中的SiO₂反应生成系列低熔点化合物[[S34]]：

| 产物 | 化学式 | 熔点 |
| :--- | :--- | :--- |
| 正硅酸锰 | MnO·SiO₂ | 1270°C |
| 橄榄石型硅酸锰 | 2MnO·SiO₂ | 1320°C |
| 蔷薇辉石型硅酸锰 | 3MnO·SiO₂ | 1200°C |

这些低熔点锰硅酸盐的大幅生成是高锰钢铸件使用硅质型壳时极易发生化学粘砂的核心原因[[S34]]。

### 化学粘砂的层状结构

化学粘砂层在最终凝固状态下呈现合金、型壁、易熔物、砂粒的清晰分层结构。低熔点易熔物填充于砂粒间隙并粘接铸件，是化学粘砂区别于机械粘砂的典型微观特征之一[[S30]]。

### 粘砂层的结合与剥离特性

化学粘砂层与铸件基体的结合强度主要由界面处铁的氧化膜厚度决定。当FeO氧化膜厚度足够大时，粘砂层可以沿着该氧化膜整体从铸件表面剥离，获得表面光洁的铸件；若氧化膜较薄或生成的物相特性不佳，即便粘砂层厚度不大也难以通过喷丸清理去除，必须采用砂轮打磨[[S19,S40]]。

温度条件对氧化过程的影响也十分关键：低于843K（570°C）时，Fe₃O₄是热力学最稳定、反应活性最高的铁氧化物；高于843K时，FeO的反应活性占主导地位。因此，化学粘砂过程中起主导作用的氧化物组分随温度条件不同而不同[[S40]]。

## 化学粘砂与机械粘砂的鉴别

正确区分化学粘砂与机械粘砂对制定防治策略至关重要。以下四类鉴别方法均经过同行验证[[S16,S32]]：

| 鉴别方法 | 化学粘砂特征 | 机械粘砂特征 |
| :--- | :--- | :--- |
| **宏观观察（肉眼+敲击）** | 断口呈连续蜂窝状物质，看不到单个分散砂粒，无金属光泽 | 断口可见完整单个砂粒夹在金属毛刺间，清铲可见金属光泽 |
| **显微观察（金相）** | 反应生成的新相将铸件与砂粒粘连，分不清砂粒与金属边界 | 砂粒与金属毛刺分界线清晰，无化学反应产物 |
| **电测法（万用表）** | 粘砂层为不导电硅酸盐，电阻接近无穷大 | 粘砂层砂粒间为金属连接，1cm间距电阻仅约0.4Ω |
| **40%盐酸浸泡法** | 反应平缓、气泡少，酸液颜色无明显变化，残留多孔团絮状物质 | 反应剧烈有气泡，液体颜色由无色→黄色→棕红，残留单个砂粒 |

## 影响因素与防控措施

化学粘砂的发生受型砂/涂料成分、金属液特性、浇注工艺及冷却条件等多方面因素的综合影响。下表系统汇总了各级影响因素及其对应的工程防控措施。

### 影响因素与对应防控措施汇总表

| 影响维度 | 影响规律 | 防控措施 | 来源 |
| :--- | :--- | :--- | :--- |
| **型砂SiO₂含量** | 低纯度硅砂烧结点<1200°C时易促进化学粘砂；高纯度硅砂烧结点>1450°C可显著降低发生率 | 采用高品位硅砂；大型厚壁及高合金钢铸件选用锆砂、铬铁矿砂、橄榄石砂、棕刚玉砂等非硅质砂 | [[S24,S17,S12]] |
| **型砂杂质含量** | 死烧黏土、灰分、金属氧化物、废金属、铁包砂等杂质含量过高会显著降低型砂耐火度，加剧化学粘砂 | 再生砂使用前去除过细砂粒、粉尘及有害杂质；生产周期中定期补充10%~15%新砂（薄壁件取下限、厚壁件取上限） | [[S18,S27,S24]] |
| **粘结剂含量** | 粘结剂含量过高会降低型砂耐火度；水玻璃砂中硅酸钠过多显著降低烧结点 | 粘结剂含量控制在合理区间，保证砂粒均匀裹覆并保留适度透气性；水玻璃砂混砂时加入1%~2%煤粉辅助防粘砂，配套使用防粘砂涂料 | [[S18,S24,S27]] |
| **浇注温度** | 浇注温度越高，金属液过热程度越大，界面反应越剧烈，铸件处于液态时间越长，粘砂风险越高 | 满足成型要求下适当降低浇注温度；铸铁件可控制在1380~1420°C区间 | [[S38,S19,S11,S18]] |
| **金属液Mn含量** | 高锰钢浇注时Mn氧化生成MnO，与SiO₂反应生成低熔点锰硅酸盐系列（熔点1200~1320°C），大幅提升粘砂风险 | 高锰钢铸件选用中性耐火材料（如电熔刚玉）作为面层制壳/造型材料，避免使用酸性硅质材料 | [[S53,S34,S38]] |
| **型腔气氛** | 氧化性气氛越强，FeO等金属氧化物生成量越大，化学粘砂越易发生 | 湿型砂严格控制水分，加入适量煤粉、沥青等含碳材料形成还原性气氛；高压造型时适当降低含碳材料加入量以控制发气量 | [[S19,S18,S34]] |
| **冷却速度** | 厚大铸件开箱落砂过晚，长时间高温下与型砂持续反应，易形成固态热粘砂 | 大型厚壁铸件适当提早开箱，加快冷却速度，避免长时间停留高温环境 | [[S31,S18]] |
| **涂料因素** | 涂料耐火度不足、涂层过薄或不均、涂层剥落均会增加化学粘砂风险 | 采用防粘砂涂料并均匀涂覆，易粘砂部位适当增加涂层厚度 | [[S18,S47]] |

此外，浇注系统设计不当（如内浇道直冲型壁、冒口颈尺寸过大等）会造成型砂局部过热，也会显著增加化学粘砂的发生概率[[S18,S27]]。

## 不同铸造工艺下的化学粘砂

### 砂型铸造

砂型铸造是化学粘砂最常发生的工艺类型。铸钢件因浇注温度高、钢液极易氧化，其发生率显著高于铸铁件；湿砂型铸造又比干型铸造的化学粘砂出现概率更高[[S16]]。化学粘砂多集中在铸件厚壁处、浇冒口附近及型芯过热部位，其反应路径完全在由硅质砂粒与粘结剂组成的多孔砂型体系中完成，低碳钢及合金钢铸件是主要风险对象[[S16,S19]]。黄铜和轻合金中一般不会发生化学粘砂，部分青铜和磷青铜铸件则有可能出现[[S19]]。

### 熔模精密铸造

在熔模精密铸造中，机械粘砂的发生概率极低，最常见的表面粘砂缺陷即为化学粘砂。当采用硅质型壳浇注钢铸件时，金属氧化物与型壳中SiO₂发生化学反应，生成低熔点硅酸盐化合物直接牢固粘附于铸件表面[[S34]]。高锰钢、高合金钢铸件使用硅质型壳时风险尤为突出，建议面层制壳采用电熔刚玉等中性耐火材料[[S53]]。

### 消失模铸造

消失模铸造中化学粘砂的发生条件与涂料选型密切相关：当涂料与金属液不匹配且干砂中混入细小砂粒灰尘时易产生化学粘砂[[S47,S2]]。无负压浇注时，化学粘砂多出现在铸件底部、侧面、热节区及型砂不易紧实区域；施加负压浇注后，铸件所有表面均可能出现化学粘砂，尤其集中在转角位置和组串浇注的过热区域，且真空度越高、浇注温度越高，概率越高[[S2,S47]]。

### 覆砂金属型重力铸造

覆砂金属型铸造中的化学粘砂成因与砂型铸造存在明显差异：多由浇注温度过低、铁型连续使用后温度升高导致覆砂受热焦化、强度下降引发，且多出现于连续浇注的最后几箱铸件，与砂型铸造粘砂多出现于首批次铸件的特征完全相反[[S37]]。

### 压铸场景——粘模（die soldering）

在高压压铸中，化学粘砂的概念和表现形式与砂型铸造有本质区别，工业界普遍称其为**die soldering**（粘模）。其反应机制不同于砂型体系的硅酸盐类粘砂[[S13,S49,S50]]：

- **反应的物质基础**：在压铸中，该反应发生在液态合金（如铝/镁合金）与H13热作模具钢的界面处。当金属液高速冲刷破坏模具表面的保护涂层或润滑剂膜后，模具中的铁元素向铸造合金中溶解，合金中的铝元素向模具基体扩散，在界面处生成系列富铝的Fe-Al金属间化合物相[[S13,S22]]。
- **反应的温度条件**：当界面温度高于该体系的焊合临界温度T<sub>c</sub>时，界面处生成富铝的液态相，冷却后将压铸合金与模具牢固粘结[[S13,S49]]。此类反应可在等于甚至低于压铸合金液相线温度的条件下发生，局部凝固时间是决定粘模严重程度的核心参数[[S1,S48]]。
- **反应的两类路径**：面向内浇口的模具表面，金属液冲击能量高，金属间化合物相优先在接触初期快速生成；背对内浇口的模具表面，合金沉积优先发生，随后才逐步在沉积层下生成金属间化合物相[[S1]]。
- **与常规化学粘砂的差异**：压铸粘模不涉及砂粒组分，生成的是金属间化合物型粘附层，其损伤机制不仅表现为粘附，还与模具冲蚀、涂层失效耦合关联——已生成的粘铝层在脱模过程中被牵拉，进一步损伤模具表面，加速后续涂层失效和冲蚀进程[[S22,S20]]。

抑制压铸粘模的核心措施包括：使用含Ti、Cr等元素的硬质涂层（如TiN、CrN、TiAlN等）将金属液与模具基体物理隔离，提升反应临界温度T<sub>c</sub>；或通过模具水冷系统将工作温度控制在临界焊合温度以下，阻断界面液相传质反应[[S49,S41,S14,S56]]。

GB/T 5611—2017中“粘型（soldering, stick-on）”的术语独立于化学粘砂进行定义，专指熔融金属粘附在金属型型腔表面的现象，表明了标准层面对两类缺陷的概念区分[[S42]]。

## 检测与分级

### 标准化的鉴别检测方法

化学粘砂与机械粘砂的鉴别检测已有成熟的方法体系，涵盖宏观、微观、电测、化学溶解四个维度[[S16,S32]]（详见前述“鉴别”章节）。

### 分级评定标准

对于铸造表面缺陷的严重程度分级，ISO 9916：1991提供了液体渗透检测的定量评定框架，适用于所有铸造工艺生产的铝合金及镁合金铸件[[S4]]。该标准将表面迹痕分为非线型（SR）、聚集型（AMR）、点线型（AR）、线型（LR）四类，在105mm×148mm评定框内分别规定了不同严重程度等级（非线型/聚集型为01~3级，点线型/线型为001~5级），要求受检表面无油污及残留造型材料等干扰物，在不超过3倍放大倍数的肉眼观测条件下完成定级[[S4]]。

对于铸钢件，ASTM体系提供了涵盖不同壁厚的射线照相检测参考底片标准[[S35]]，国内GB/T 5677—2007铸钢件射线照相检测标准等同采用ISO 4993，评级方法参照ASTM参考底片执行[[S21,S35]]。

消失模铸造领域则提供了化学粘砂的目视分级参考图谱，在100mm×60mm标准评定视野下展示从1级（轻微粘砂）至5级（严重粘砂、铁包砂）的全梯度视觉特征，配套标尺可直接定量测量粘砂覆盖面积[[S44]]。

## 补救措施

对于已形成的化学粘砂，工程中常用的补救处理方法包括[[S18]]：

1. **喷丸/抛丸清砂**——适用于粘砂层结合较弱的轻中度化学粘砂。
2. **砂轮打磨**——适用于硬度高、结合牢固的化学粘砂层，是化学粘砂清理中不可避免的手段。
3. **电化学清砂**——尤其适用于深腔部位及精密铸件的严重粘砂清理。
