## 定义与基本概念
离心铸造转速是离心铸造时铸型的转速，单位为转每分（r/min），是决定离心力大小与铸件成形质量的核心工艺参数[[S16,S34]]。  
转速过低，金属液无法有效附着型壁，出现淋落、不致密、夹杂、气孔等缺陷；转速过高则易引发偏析和表面裂纹[[S16,S34]]。因此在保证铸件质量的前提下，应选取最低的可用转速[[S11,S39]]。

## 核心参数与换算关系
离心铸造转速体系由三个互算指标构成[[S46,S32,S15]]：  

| 参数 | 符号 | 单位 | 定义与换算 |
|------|------|------|------------|
| 铸型转速 | \(n\) | r/min | 铸型单位时间内的转数。 |
| 圆周线速度 | \(v\) | m/s | 铸件内表面质点的线速度，\(v = \pi R n / 30000\)，\(R\) 为铸件内半径（mm）。 |
| 重力系数（G值） | \(G\) | 无因次 | 离心加速度与重力加速度之比，\(G = \omega^2 R / g = 0.112\,(n/100)^2 R\)，是表征离心力强度的核心参数。 |

重力系数是选型与控制的基础，工程中通常推荐 \(G\) 在 40 ~ 100 之间，超过 100 G 后铸件力学性能无明显提升而缺陷风险显著增大[[S8,S13,S32]]。

## 转速选择与计算公式
### 临界转速
卧式离心铸造时，为避免金属液滞留型底或出现雨淋现象，必须满足铸件内表面最高点的离心力大于其重力（\(m \omega^2 r_0 \ge m g\)）[[S31]]。实际铸型的临界转速还受铸件直径（直径越小临界转速越高）、壁厚（壁厚越大临界转速越高）以及铸型内壁粗糙度的影响，通常需通过生产试验确定[[S31]]。

### 常用经验公式
生产实践中主要采用以下四类公式进行转速计算，最终值还需结合试产调整[[S1,S21]]。

1. **康斯坦丁诺夫（Konstantinov）公式**  
   \[
   n = \beta \frac{55200}{\sqrt{\gamma R}}
   \]
   式中 \(n\) 为转速（r/min），\(\gamma\) 为合金重度（N/m³），\(R\) 为铸件内半径（m），\(\beta\) 为工况调整系数。该公式基于铸件内表面有效重度达到 \(3.4\times10^6\,\text{N/m}^3\) 的假设，适用于外内半径比 ≤ 1.5 的薄壁件[[S11,S1,S21,S29]]。

2. **基于重力系数 \(G\) 的公式**（最广泛）  
   \[
   n = 299 \sqrt{\frac{G}{R}} \quad (R\text{ 单位 cm}) \quad\text{或}\quad n = 29.9 \sqrt{\frac{G}{R}} \quad (R\text{ 单位 m})
   \]
   该方法不随合金密度变化，计算便捷。常用 \(G\) 值按合金种类选取[[S11,S17,S8,S13]]。

3. **凯门（Cammen）公式**（欧美常用）  
   \[
   n = \frac{C}{\sqrt{R}}
   \]
   \(C\) 为综合系数，由合金、铸型及浇注工艺共同决定。该式适用于外内半径比 ≤ 1.15 的薄壁件[[S11,S1,S35]]。

4. **基于线速度的公式**  
   \[
   n = 9.55 \frac{v}{R}
   \]
   推荐铸件内表面线速度 \(v\) 取 250 ~ 450 cm/s，也可用于临界转速的核算[[S15,S9]]。

5. **立式离心铸造壁厚差控制公式**  
   \[
   n = 422 \sqrt{\frac{h}{K(2R-K)}}
   \]
   式中 \(h\) 为铸件高度（cm），\(R\) 为上口内半径（cm），\(K\) 为允许的轴向壁厚差（cm）。计算后还需用 \(G = R (n/299)^2\) 校验重力系数是否合理[[S13,S40]]。

## 典型工艺参数与推荐范围
### 不同合金的重力系数 \(G\) 推荐值
合金种类不同，为控制偏析和保证补缩，要求的重力系数差异明显[[S5,S42,S29,S13]]。

| 合金类别 | 重力系数 \(G\) 范围 | 备注 |
|----------|---------------------|------|
| 铸钢 | 30 ~ 50 | |
| 灰铸铁 | 50 ~ 100 | |
| 铝硅合金（含 Si 8 %） | 80 ~ 110 | |
| 共晶铝硅合金（Si 11 ~ 12 %） | 30 ~ 50 | |
| 铝合金（整体） | 80 ~ 120 | 高强铝合金提 G 可细化晶粒，提高强度和延伸率[[S29]] |
| 锌合金 | 40 ~ 62 | |
| 铜合金套类 | 35 ~ 80 | |
| 锡青铜等高偏析倾向合金 | 应适当降低 | 减少成分偏析 |
| 宽结晶温度区间合金 | 可适当提高 | 强化补缩效果 |

### 卧式与立式铸造的转速特点
- **卧式离心铸造**：主要用于长度大于直径的管、套类件，常规采用 50 G ~ 100 G 对应的转速。大直径铸件转速显著降低，例如内径 550 mm 的铸铁管转速约为 450 r/min，管径越大转速越低[[S10,S23]]。
- **立式离心铸造**：用于高度小于直径的环、盘类件。提高转速可使内壁抛物线更接近竖直，减小上下壁厚差[[S33,S23]]。
- **半离心/离心浇注**（非真离心铸造）：仅需提供约 10 G 的低离心力实现充型和补缩，其转速远低于常规真离心铸造[[S20,S37,S33]]。

工业中通用转速区间大致在 300 ~ 3000 r/min，经验偏差在 ±15 % 以内通常不会对成形质量产生显著影响[[S38,S26]]。

## 转速对铸件质量的影响
### 转速过低引发的缺陷
在卧式离心铸造中，转速不足时金属液无法获得足够的离心力，滞留在铸型底部滚动，随后出现雨淋（散落）现象，直接导致冷隔、充型不足、铸件内表面凹凸不平等缺陷[[S7,S36,S39]]。此外还会造成铸件疏松和夹渣[[S12]]。

### 转速过高引发的缺陷
转速过高导致离心力过大，会加剧不同密度组分的相对运动，增大宏观偏析倾向；过高的离心应力也容易在铸件外表面产生纵向裂纹；使用砂型时还可能造成胀砂和粘砂，并显著增加设备能耗与结构负荷[[S11,S18,S45]]。

### 转速优化对组织的改善
在合理区间内，提高转速可增强金属液充型能力与定向凝固效果，细化晶粒，提高铸件致密度，减少缩孔缩松[[S19]]。  
例如，在 ZL205A 铝合金薄壁件的立式离心铸造中，转速由 150 r/min 升至 450 r/min 时，晶粒尺寸从 410 μm 逐步降至 385 μm[[S19]]。  
另一方面，当转速过高时，缩孔缩松体积占比反而可能上升——如铝合金在 800 rpm 时缺陷占比为 1.103 %，而 1600 rpm 时升至约 1.362 %[[S6,S30]]，说明需要合理选取转速。

## 工艺控制与检测
主流离心铸造设备采用伺服电机驱动的半闭环控制方式：在电机轴端集成光电编码器或测速发电机，实时采集铸型转速信号并反馈至数控系统，与预设转速指令比较后动态调节电机输出扭矩和转速[[S44,S24]]。配合交流无级变速驱动，可在整个浇注过程中实现转速精确管控，稳态误差通常控制在 ±5 r/min 以内[[S44,S4]]。

## 相关术语辨析
- **离心力**：\(F = m R \omega^2\)，与转速的平方成正比，是离心铸造的基本动力[[S25]]。  
- **重力系数（G 因子）**：离心力与重力的比值，是决定铸型转速的核心无量纲参数[[S34]]。  
- **临界转速**：保证金属液能正常附着型壁成形的最小铸型转速，受铸件直径、壁厚及型壁粗糙度影响[[S31]]。  
- **铸造速度（浇注速度）**：指单位时间浇入铸型的金属质量或体积，与铸型转速是独立参数，共同影响充型与缺陷[[S22]]。  
- **压铸中的压射速度**：属于高压压铸的快压射或慢压射参数，与本词条所指的离心铸型转速分属不同工艺体系，除非证据明确指向压铸中的离心力辅助应用，否则不应混淆。

## 分类与适用场景
### 按旋转轴方位
- **卧式离心铸造**：用于长管、套类件，内径越大转速越低，常采用 50 G ~ 100 G。管径 550 mm 时转速约 450 r/min[[S10,S23]]。  
- **立式离心铸造**：用于高度小于直径的环、盘类件，转速选取需兼顾轴向壁厚差，高转速可减小内壁倾斜[[S33,S23]]。  
- **倾斜轴离心铸造**：介于上述两者之间，兼具二者特点[[S33]]。

### 按工艺类型
- **真离心铸造**：无型芯，靠离心力形成中空内腔，所需转速较高，G 值通常大于 50。  
- **半离心铸造/离心浇注（centrifuging）**：采用中心浇道和型腔，离心力主要用于充型和补缩，所需 G 值较低（约 10 G），模具转速显著低于真离心铸造[[S20,S37,S33]]。
