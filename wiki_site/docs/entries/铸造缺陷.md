## 概述

铸造缺陷（别名铸件缺陷，英文对应 Casting Defect / Foundry Defect）是指在金属铸造过程中产生的、不符合预期要求的不规则特性。其中，轻微缺陷可能处于允许范围之内，无法接受的缺陷则需通过再加工或判废处理。该术语覆盖砂型铸造、压力铸造、熔模铸造等全部铸造工艺，是一个广义的上位通用术语 [[S25,S49,S70,S11,S13]]。

## 术语与层级关系

在国内标准体系中，GB/T 5611-2017《铸造术语》为铸造缺陷相关术语的基础参照标准，而 GB/T 41972-2022《铸铁件铸造缺陷分类及命名》则进一步建立起系统的缺陷分类与编码规则，后者主要适用于砂型生产的铸铁件，其他铸造方法生产的铸铁件也可参照使用 [[S70,S75]]。

压铸缺陷是铸造缺陷概念下的一个重要下位分支：它专指高压压铸工艺中出现的缺陷，既包含与通用铸造共有的缩孔、气孔、裂纹等缺陷类型，也涵盖花纹、流痕、顶杆痕迹、网状毛刺等压铸工艺特有的缺陷品类。国内针对压铸缺陷的专项标准为 GB/T 29092—2012《镁及镁合金压铸缺陷术语》，其适用范围限于镁及镁合金压铸件的缺陷分析与判定 [[S41,S38,S11]]。

## 缺陷分类体系

### 国内通用分类框架

基于 GB/T 5611 系列标准，铸造缺陷被划分为 **8 大类**，共 100 余种，具体分类如下 [[S22,S63,S75]]：

| 类别 | 主要内容 |
|------|----------|
| 多肉类缺陷 | 飞边、毛刺、多肉等 |
| 孔洞类缺陷 | 缩孔、缩松、气孔、针孔等 |
| 裂纹、冷隔类缺陷 | 热裂、冷裂、冷隔、热处理裂纹等 |
| 表面类缺陷 | 粘砂、表面粗糙、皱皮、缩陷等 |
| 残缺类缺陷 | 浇不足、损伤等 |
| 尺寸或形状差错类缺陷 | 变形、尺寸超差等 |
| 夹杂类缺陷 | 夹杂物、冷豆、渣气孔等 |
| 性能、成分、组织不合格类缺陷 | 硬度不合格、金相组织异常等 |

### 国际分类体系（AFS 标准）

美国铸造协会（AFS）发布的《国际铸件缺陷图谱》采用基于缺陷视觉外观、特性、尺寸和位置的分类方法，将缺陷分为 **7 大类**，并以 1 位大写字母加 3 位数字进行编码，形成国际通用的标识体系 [[S27,S20,S17]]：

| 类别代号 | 类别名称 | 说明 |
|----------|----------|------|
| A | 金属突出物 | 飞边、毛刺、脉纹等 |
| B | 孔洞类缺陷 | 气孔、缩孔、缩松等 |
| C | 不连续缺陷 | 裂纹、冷隔、热裂等 |
| D | 表面缺陷 | 粘砂、表面折叠、氧化皮等 |
| E | 铸件残缺缺陷 | 浇不足、跑火等 |
| F | 尺寸或形状差错缺陷 | 尺寸偏差、变形等 |
| G | 夹杂物/组织异常缺陷 | 夹杂、偏析等 |

### 按成因分类

还可将铸造缺陷分为以下两种成因大类，以直接关联产生机理并指导工艺优化 [[S63]]：

- **物理‑力学性能变化引发的缺陷**：如收缩引起的缩孔、缩松，应力引起的裂纹、变形等。
- **化学反应过程异常引发的缺陷**：如气孔、氧化夹杂、反应性粘砂等。

## 典型缺陷形成机理与影响因素

### 缩孔与缩松

缩孔（shrinkage cavity）是铸件凝固过程中，因补缩通道被截断，无法补偿凝固收缩而在最后凝固区域形成的大而集中的不规则孔洞，孔壁粗糙，多位于热节、厚大截面处 [[S73]]。缩松（shrinkage porosity）则是尺寸细小、分布分散的微小孔洞，系凝固后期枝晶骨架封闭后，枝晶间残余液相无法补缩所致 [[S67,S30,S32]]。

金属从浇注温度冷却到室温经历三个阶段：液态收缩、凝固收缩和固态收缩，其中前两者是缩孔与缩松的核心诱因 [[S57]]。铸件的补缩行为按顺序可分为液态补缩、浆态补缩、枝晶间补缩、爆发式补缩和固态补缩五类，生产中应尽可能使补缩维持在开放的流动态阶段，避免补缩中断 [[S67]]。

### 气孔

气孔可分为析出型和卷气型两类。

- **析出型气孔**：高温金属液中溶解大量气体，凝固时固态合金的气体溶解度急剧下降，过饱和气体以气泡形式析出并被凝固前沿包裹，形成多呈规则球形的内部气孔 [[S74,S69]]。
- **卷气型气孔**：充型过程中金属液雷诺数超过 4000 进入湍流状态，型腔内气体被卷入流动金属液中而无法完全排出，残留为不规则气孔 [[S67]]。

### 裂纹

- **热裂**：铸件处于准固相区时，枝晶间残余液膜在凝固收缩拉应力作用下被撕裂，且无充足液相弥合，形成沿晶开裂的缺陷；开裂表面因高温氧化而无金属光泽 [[S65,S68]]。
- **冷裂**：铸件完全进入弹性态后，内部残余铸造应力超过该温度下合金的强度极限而产生的开裂，属于穿晶断裂，断面带有金属光泽或轻微氧化色。当合金延伸率低于 1.5% 时，冷裂风险显著上升 [[S65,S46,S31]]。

### 冷隔与浇不足

冷隔是两股独立液流在型腔内汇合时，因前沿金属已提前凝固或存在氧化膜，无法完全熔合而形成的不连续界面缺陷，严重削弱局部力学性能 [[S65,S8]]。浇不足则是金属液在充型过程中，流动前沿在填满型腔前即达到临界固相率（通常 0.1～0.4），丧失流动能力，造成铸件局部缺失的不完整成形缺陷 [[S46,S8]]。

### 夹杂

夹杂缺陷是指铸件内部与基体金属成分不同的外来非均质质点。其主要来源包括熔炼中生成的氧化物、硫化物，未清除的熔渣，浇注过程中卷入的型砂及涂料脱落碎屑等；夹杂物作为应力集中点，会大幅降低铸件的力学性能 [[S44,S6,S48]]。

### 关键工艺参数的影响

以下表格整理了主要工艺参数对铸造缺陷的作用规律：

| 工艺参数 | 正面影响 | 负面影响（参数过高/过大的风险） |
|----------|----------|--------------------------------|
| 浇注温度 ↑ | 增强流动性，降低冷隔、浇不足概率 | 延长凝固时间，增大吸气量，增加缩孔缩松、氧化夹杂和晶粒粗化风险 [[S51,S4,S35]] |
| 模具/砂型预热温度 ↑ | 降低表面冷却速率，减少冷隔、浇不足 | 延缓整体凝固，导致晶粒粗大、缩孔缩松加重，并加剧模具热疲劳 [[S45,S35,S66]] |
| 充型速度 ↑ | 缩短充型时间，防止液流前沿过早凝固，抑制冷隔、浇不足 | 诱发湍流，大幅提升卷气概率，导致大量内部气孔 [[S45,S67,S56]] |
| 冷却速度/冷铁设置 | 增大温度梯度，实现顺序凝固，降低缩孔缩松倾向 | 温差过大时显著提高热裂、冷裂发生概率 [[S31,S51,S9]] |
| 排气系统完善程度 ↑ | 降低型腔反压，排出卷入气体，改善远端薄壁填充，同时减少气孔、冷隔、浇不足 | — |
| 涂料控制 | 调整局部换热速率，避免脱落形成夹杂 | 过厚会堵塞排气通道，增大型腔发气量，提升气孔风险 [[S45,S68]] |

## 检测与诊断

### X 射线探伤

X 射线探伤利用射线穿透被测材料时缺陷与基体的射线衰减系数差异，在胶片或数字成像器件上形成黑度差影像，可检出铸件内部的缩孔、缩松、气孔、裂纹等体积型缺陷，适用铸件厚度一般在 150 mm 以下 [[S28,S14,S7]]。

![X射线探伤法示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/04824d14-e543-443c-b76d-4b7d13016cc3/resource)

X 射线探伤原理示意如上，射线穿过含缺陷铸件后，在感光片上形成缺陷影像 [[S3]]。

### 工业 CT

工业 CT 可实现铸件内部缺陷的三维成像，分辨率高于常规 X 射线探伤，能够检测复杂结构中的微小缺陷并规避检测盲区，实测初检准确率可达 99.4%，复核后可达 100%，并能对缺陷的体积、空间位置进行精确定量 [[S23,S77,S80]]。

### 超声波检测

超声检测采用 1～5 MHz 的高频声波，利用缺陷处的声阻抗突变产生反射回波，以判定缺陷位置和尺寸，对面积型缺陷（如裂纹、分层）灵敏度高。该方法探测深度大、成本低且无辐射防护需求，但仅适用于形状简单、表面平整度高的铸件，难以直接判定缺陷类型 [[S14,S80,S7]]。

### 渗透检测

渗透检测只适用于铸件表面开口型缺陷，可检出宽度小于 0.01 mm 的细微裂纹，分为着色显示和荧光显示两类，设备简单、操作便捷，但无法检测铸件内部及闭合的近表面缺陷 [[S10,S7]]。

### 缺陷影像识别特征

| 缺陷类型 | X 射线影像典型特征 |
|----------|-------------------|
| 缩孔 | 轮廓较清晰，呈管状、喇叭状或团簇状暗色斑，中心黑度突变 [[S78,S23,S21]] |
| 缩松 | 纤维状（不规则连续黑色显影）、树枝状（枝晶状高黑度区）或海绵状（云雾状弥散分布）[[S78,S23]] |
| 气孔 | 边缘锐利的类圆形暗斑，内部黑度均匀 [[S78,S21]] |
| 裂纹 | 两端尖中间粗的分叉或锯齿状暗色条纹，有时呈龟裂状 [[S78,S21]] |

超声检测则通过回波信号的幅值时域特征判定缺陷的埋深与等效尺寸，但无法直观呈现缺陷形貌 [[S7,S80]]。

### 在线集成与智能检测

国内新能源汽车头部企业在大型一体化压铸岛中已集成在线 X-Ray 探伤检测工位，所有铸件后处理工序均在岛内完成，减少了铸件转运环节，实现了大型结构件的即时内部质量筛查 [[S34]]。面向铝合金压铸件大批量检测设计的 XR160INLINE 系统，搭载 160 kV 高频恒压 X 射线机与高分辨率数字平板探测器，支持半自动多工位成像，结合 AI 缺陷识别模块可自动输出 OK/NG 判定并实现自动分拣，适配压铸量产线的快速质量反馈需求 [[S40]]。

## 预防与控制

铸造缺陷的全流程预防与控制覆盖 **模具/砂型结构优化、浇注系统设计、工艺参数调控、合金熔体净化处理** 以及 **铸造模拟仿真辅助优化** 五大核心技术维度 [[S59,S24,S55]]。

### 结构优化与冷铁布置案例

铁路摇枕（材质 ZG230-450，尺寸 2815 mm×700 mm×500 mm，山字形结构）在原有工艺下 A～H 共 8 个位置热裂纹发生率高达 80%～90%，单件裂纹长度 30～220 mm。通过 **在 A、B、C、D 立面处放置 R51×80×12×55 mm 冷铁并取消 E、F、G、H 处原有冷铁**，热裂纹缺陷得到完全消除 [[S43]]。

类似地，CO₂ 水玻璃砂生产的铁路摇枕出现严重夹砂缺陷，报废达上百件。通过 **将原砂由交城 50/100 目更换为膨胀系数更低的海城 40/70 目原砂，并将浇注速度从 28～38 s 调整为 25～35 s**，夹砂缺陷得到根治 [[S29]]。

### 压铸工艺参数与浇注系统优化案例

大面积薄壁铝合金压铸件初始生产合格率仅 20%。通过以下措施使合格率提升至 95% [[S26]]：

- 浇注温度控制为 680±15 ℃；
- 压铸机由 5000 kN 更换为 6500 kN；
- 模具预热温度控制在 230±20 ℃；
- 压射速度 5 m/s，充填时间 0.012 s，保压时间 1.5 s，压射力 550 kN；
- 将扇形浇道修改为 T 形浇道，增加内浇口数量并同步增加排气孔与溢流槽。

其他典型压铸案例包括：在汽车发动机油底壳传感器位置增设集渣包、采用机器人喷涂及增加排气块，消除密封面气孔；在节温器壳体上加装真空排气装置并在厚壁处设置局部挤压销，解决缩孔缩松与拉裂漏气问题 [[S33]]；5G 通讯壳体通过分型面激光熔覆、梯度增压及真空排气，使飞边发生率从 18% 降至 0.5%，模具寿命也从 8 万模次提升至 12 万模次 [[S12]]。

### 浇注系统设计与熔体净化

合理设计浇注系统可有效降低金属液充型湍流程度，减少裹气与夹渣缺陷，**控制内浇口处金属液流速不超过 0.5 m/s 可显著抑制氧化膜卷入** [[S58]]。铝合金压铸前的熔体净化常采用螺旋喷吹除气工艺，高纯氩气（或氮气）通过转子喷头分散为细小气泡，同步吸附氢和非金属夹渣，有效减少铸件内部气孔与缩松 [[S24]]。

### 铸造仿真

常用铸造仿真工具包括 MAGMA、ProCAST、AnyCasting 等，可实现充型速度场、温度场及凝固过程模拟，通过残余熔体模数、Niyama 判据等方法精准预测缩松缩孔位置，支持工艺迭代优化，大幅减少实际试模次数以降低研发成本 [[S15,S61]]。
