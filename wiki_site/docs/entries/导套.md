## 概述
导套是压铸模具中的核心导向零件，英文名称为 guide bushing，与导柱配对使用，二者共同构成模具导向系统。其主要功能是确定动模与定模的合模相对位置，保障模具开合模运动过程中的定位精度与运动稳定性，常规配合间隙最高控制在0.05mm以内[[S24]]。导套属于压铸模标准导向零部件，对应GB/T 4679—2003等国内压铸模零件标准规定的系列规格，在主流压铸模架体系中可通用互换[[S33]]。

## 分类与结构特征
导套按结构形式主要分为直导套和带头导套（带肩导套），并可按润滑方式与运动形式进一步细分[[S24,S35,S25,S16]]。

| 分类维度 | 类型 | 主要特征与适用场景 |
|---|---|---|
| 结构形式 | 直导套 | 无侧向法兰台阶，多用于较薄模板或小型简单压铸模具[[S35,S25]] |
| | 带头导套（带肩导套） | 一端设有径向法兰限位台阶，用于轴向限位，多用于较厚模板或精度要求更高的中大型压铸模具[[S35,S25]] |
| | 推板导套 | 专用于推出机构，外圆与推杆固定板/推板安装孔过盈配合，内孔与推板导柱间隙配合[[S24,S5]] |
| 润滑形式 | 油槽型 | 内壁加工有螺旋形或点状油槽，用于存储润滑油，需定期加注耐高温润滑脂[[S25,S16]] |
| | 自润滑型 | 内嵌石墨或二硫化钼（MoS₂）等固体润滑剂，常规工作温度上限为80℃或150℃，无需额外注油维护[[S16,S24,S35]] |
| 运动形式 | 普通滑动导套 | 导套与导柱为滑动摩擦，常规采用H7/f7或H8/f8间隙配合[[S16]] |
| | 直线轴承导套（滚珠导套） | 内部集成滚珠滚动结构，无滑动摩擦，可实现零间隙预紧导向，属高精度导向配件[[S16]] |

### 结构设计要点
- 导套导向工作段内孔及外圆柱面均需磨削精加工，必要时做研磨处理以保障同轴度与表面质量[[S24,S35]]。
- 带头导套的径向法兰台阶用于轴向限位，防止开模时导套从模板中脱出；直导套通常在侧壁开设防脱槽，通过压入内六角螺塞或止动螺栓实现防转防脱落固定[[S24,S35,S33]]。
- 导套端面向内部倒圆角，便于导柱顺利导入[[S26]]。装配后导套端面应低于分型面3~5mm，既保障分型面贴合，又作为动定模分离时的撬模操作间隙[[S33]]。

## 材料与热处理
导套推荐制造材料主要为GCr15轴承钢、20Cr低碳钢和T10A碳素工具钢，也可采用FC250灰铸铁嵌石墨或CAC304高强度铜合金等自润滑材料[[S35,S33]]。

| 材料 | 热处理工艺 | 成品硬度要求 |
|---|---|---|
| GCr15 | 整体淬火 | 58~62 HRC[[S22,S28]] |
| 20Cr | 渗碳淬火 | 58~62 HRC[[S22,S28,S32]] |
| T10A | 整体淬火 | 58~62 HRC[[S22,S28,S32]] |

导套导向段硬度应略低于配对导柱的硬度，优先磨损导套以便于维护更换[[S33]]。

## 技术参数与配合
### 国家标准规定
国内现行压铸模导套标准主要由GB/T 4678系列规定，关键参数汇总如下：

| 标准编号 | 导套类型 | 导向段内径D公差 | 固定段外径D₁公差 | 硬度要求 | 推荐材料 |
|---|---|---|---|---|---|
| GB/T 4678.6—2017 | 带头导套 | H8[[S22]] | k6[[S22]] | 58~62 HRC[[S22]] | GCr15、20Cr、T10A[[S22,S28]] |
| GB/T 4678.7—2017 | 直导套 | H8[[S32]] | r6[[S32]] | 58~62 HRC[[S32]] | GCr15、20Cr、T10A[[S32]] |
| GB/T 4678.10—2017 | 推板导套 | H9[[S5,S29]] | k6[[S5]] | 58~62 HRC[[S5]] | GCr15、20Cr、T10A[[S5,S29]] |

未注公差尺寸的极限偏差应符合GB/T 1804—2000中f级的规定[[S22,S5]]。

### 配合关系
- **固定配合**：导套外圆与模架套板安装孔采用过盈配合，常用配合等级为H7/k6[[S14,S19,S33]]。直导套需额外加装止动螺钉防转防脱出[[S24,S26]]。DIN（HASCO）标准中紧配部位选用H7/k6或H7/m6[[S20]]；FUTABA标准中无定位导套紧配部位采用H6/m5[[S20]]。
- **滑动配合**：导套内孔与导柱采用间隙配合。若内孔取H7，对应导柱轴取f7；若内孔取H8，对应导柱轴取e7[[S14,S19]]。在压铸锌、铝合金场景下常用H7/e8配合，压铸铜合金场景下常用H7/d8配合[[S33]]。

## 模具布局与工作原理
### 布局规则
导套与导柱在模架中的相对布置遵循以下原则[[S21,S27,S1,S4]]：
- 绝大多数常规压铸结构中，导套安装在动模侧套板上，导柱布置在定模侧。
- 采用推板卸料/顶出机构的模具中，导柱布置在动模侧，对应导套安装在定模侧。
- 卧式压铸机采用中心浇口的模具，导柱设置在定模座板上，配套导套对应布置。
- 推出系统的推板导套安装在动模侧的推板/推杆固定板对应导向位。
- 导套一般沿模架套板四角位置对称布置。

### 工作原理
合模过程中，导柱与导套导向副先于型腔、型芯接触，实现预导向，约束动模运动路径，保证动定模同轴度与相对位置精度，避免成型零部件碰撞干涉[[S24,S33]]。

### 精度失效与铸件缺陷关联
导套长期使用后内孔磨损量超出阈值，导柱导套配合间隙超过允许范围，动定模将发生无约束侧向偏移，分型面出现配合错位间隙，封料面密封性下降。金属液从分型面间隙溢出形成飞边，同时成型型腔相对位置偏移直接导致压铸件孔位与轮廓尺寸超差[[S10]]。

## 常见失效与维护
### 失效模式
| 失效类型 | 诱因与机制 |
|---|---|
| 磨损失效 | 导套与导柱长期往复相对运动、合模侧向载荷持续作用下，内孔工作面逐步磨损，配合间隙持续超标[[S2,S8]] |
| 拉伤、咬合卡滞 | 模具长期处于100~350℃工作温度区间，不同部位热膨胀量差异导致配合间隙大幅缩小，局部发生金属粘连，运动时将导套内表面拉伤，严重时出现卡滞无法正常开合模[[S2,S8]] |
| 受载变形 | 合模瞬间侧向冲击载荷超过导套结构承载阈值，导套发生塑性变形，同轴度超差[[S8]] |

### 润滑维护要求
- 油槽型普通导套：需定期在导套内孔油槽加注耐高温润滑脂，保证开合模运动顺畅[[S24]]。
- 自润滑导套：在80℃或150℃以下工作区间无需额外频繁补油，仅需在模具定期保养时清理导向副工作面残留的金属碎屑及脱模剂杂质[[S24]]。

### 更换判定标准
符合以下任何一项即建议更换[[S13]]：
1. 导套内孔磨损后，导柱导套实际配合间隙超出设计允许最大值，合模检测动定模偏移量超出压铸件允许形位公差。
2. 导套内表面出现明显大面积拉痕、局部咬合粘连痕迹，运动时出现卡滞异响，无法顺畅完成开合模动作。
3. 导套内孔表面磨损量超过0.03~0.05mm，经检测内孔同轴度超差，且通过重新研磨无法恢复原设计精度。

## 标准体系归属
导套作为标准件，由中国国家标准GB/T 4678系列（压铸模零件）中的三个独立部分分别规范，同时配合GB/T 4679《压铸模零件技术条件》对未注明技术要求的通用规定[[S22,S33]]。此外，GB/T 4169系列模具标准件的导套参数（如直导套表面粗糙度Ra=3.2μm，带头导套形位公差6级精度）也可作为通用设计参考[[S12]]。国际标准体系如FUTABA公制模具标准、DIN（HASCO）标准等对导套紧配与滑动面公差均有相应规范[[S20]]。
