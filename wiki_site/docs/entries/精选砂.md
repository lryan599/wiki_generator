## 概述

精选砂在现行铸造标准中的正式名称为浮选砂（floated sand），属于天然硅砂中加工等级最高的品类[[S5,S12,S31]]。它是以石英（SiO₂）为主要矿物成分、粒径范围为 0.020 mm ~ 3.350 mm 的耐火颗粒物，其生产核心是在常规擦洗基础上增加浮选工序，从而大幅降低杂质含量[[S17,S31,S6,S7]]。与仅经水洗或擦洗处理的普通铸造原砂相比，精选砂在化学成分、含泥量、粒形和粒度集中度等方面均达到现行生产工艺所能实现的最高纯度水平[[S12,S4]]。

## 定义与基本属性

根据 GB/T 9442《铸造用硅砂》与 GB/T 5611《铸造术语》的规定，精选砂是“经擦洗和浮选的天然硅砂”[[S5,S12,S17]]。其法定核心品质指标为：

- SiO₂ 质量分数 ≥ 96 %[[S12,S31,S17]]；
- 含泥量（质量分数） ≤ 0.2 %[[S12,S31,S18,S7]]。

这一纯度远高于水洗砂（含泥量 ≤ 1.0 %）和擦洗砂（含泥量 ≤ 0.3 %），使其成为天然硅砂产品中粘结剂消耗最低、型砂性能最稳定的品种[[S31,S12,S7]]。

表征精选砂颗粒特性的两个标准参数是**角形因数**和**平均细度（AFS 平均细度）**。角形因数定义为实际比表面积与同粒径理想球形砂理论比表面积的比值，用以定量反映颗粒的圆整度[[S17,S4,S7]]；AFS 平均细度则由筛分后各筛号停留砂粒质量百分数与对应乘数乘积之和除以总质量分数得出，代表砂样的整体粒度水平[[S32,S12]]。

## 分类与定位

### 石英系与非石英系分类

在整个铸造用造型材料体系中，造型用砂首先分为**石英系砂**与**非石英系砂**两大部类[[S26,S1]]。精选砂（浮选砂）在分类上属于石英系砂中的天然硅砂子类，是天然硅砂经浮选提纯后的最高等级产品[[S11,S12]]。机械工程手册中的“造型用砂分类示意图”直观地展示了这一层级关系，可清晰界定精选砂在整个铸造用砂体系中的位置[[S1,S26]]。

### 非石英系精选品种

在行业实践中，“精选砂”一词亦常被泛指经过浮选、磁选等提纯工艺的高端特种原砂。常见的非石英系精选品种包括[[S11,S22]]：

- **精选锆英砂**：主成分为硅酸锆（ZrO₂·SiO₂），粒度细，热膨胀系数低，导热率高[[S11]]。
- **精选铬矿砂**：以 FeO·Cr₂O₃ 为主要矿物，耐火度高于 1790 ℃，固相烧结性好[[S11]]。
- **精选球形陶粒砂（宝珠砂）**：以 Al₂O₃‑SiO₂ 为主要成分，呈球状，20–1000 ℃ 膨胀系数仅约 6×10⁻⁶/℃，耐火度可超过 1800 ℃[[S22,S11]]。

需要指出的是，国家标准层面并未用“精选砂”单独命名这些特种砂；上述命名多为商业或应用领域的通俗称谓。

## 工艺作用与应用场景

精选砂因其高纯度和低含泥量，在各类铸造工艺中广泛用作制芯、造型的基础骨架材料：

- **砂型与砂芯制备**：适用于重力铸造、低压铸造等工艺的砂型及砂芯生产，可满足铝合金、镁合金、铸铁、铸钢等全系列合金的浇注要求[[S6,S15]]。
- **复杂内腔压铸件**：在压铸生产中，对于带有复杂流道、无法用金属型芯抽芯的铸件，可采用精选砂制备的特种砂芯代替传统金属型芯，解决工艺难题[[S10,S30]]。
- **典型工程案例**：铁路货车车钩 CO₂ 硬化水玻璃砂工艺中，精选砂的技术要求为 SiO₂ ≥ 95 %，40/70 三筛粒度占比 ≥ 90 %，含泥量 ≤ 0.2 %，水分 ≤ 0.2 %[[S20]]。

## 关键性能参数与检测标准

### 核心指标

精选砂在供货和应用中需满足以下关键技术要求（表 1）。

| 项目 | 指标要求 | 依据标准 / 说明 |
|:---|:---|:---|
| SiO₂ 质量分数 | ≥ 96 % | GB/T 9442[[S7,S12]] |
| 含泥量 | ≤ 0.2 % | GB/T 9442[[S7,S18]] |
| 含水量（袋装烘干态） | ≤ 0.3 % | GB/T 9442[[S7]] |
| 酸耗值（使用化学粘结剂时） | ≤ 5.0 mL | GB/T 9442[[S7]] |
| 主要粒度组成（三筛） | ≥ 75 % | GB/T 9442[[S7]] |
| 主要粒度组成（四筛） | ≥ 85 % | GB/T 9442[[S7]] |
| 主要粒度组成（五筛） | ≥ 95 % | GB/T 9442[[S7]] |
| 角形因数 | 按标准分级（圆形 ≤ 1.15，椭圆形 1.15–1.30，钝角形 1.30–1.45 等） | GB/T 9442[[S7,S31]] |

### 检测方法

精选砂的性能检测需遵循下列标准[[S4,S6,S14]]：

- **GB/T 9442**《铸造用硅砂》：规定含泥量、粒度、角形因数、化学成分等分级与方法。
- **GB/T 2684**《铸造用砂及混合料试验方法》：通用检测流程。
- **JB/T 9224**《检定铸造粘结剂用标准砂》：对高纯低泥铸造用砂的性能约束。

AFS 平均细度的具体计算规则为：将筛后各筛号上停留砂粒质量的百分数乘以对应乘数，求和后除以各筛上停留砂粒质量分数的总和[[S32]]。

## 与相近材料的对比

### 水洗砂、擦洗砂与精选砂

这三种天然硅砂的主要差异体现在含泥量与 SiO₂ 纯度上（表 2）。

| 砂类型 | 加工方式 | 含泥量（质量分数） | SiO₂ 含量要求 |
|:---|:---|:---|:---|
| 水洗砂 | 仅水洗去泥 | ≤ 1.0 % | 无明确规定（级外） |
| 擦洗砂 | 强力擦洗去除表面附着泥层 | ≤ 0.3 % | 通常低于精选砂 |
| 精选砂 | 擦洗 + 浮选双重提纯 | ≤ 0.2 % | ≥ 96 % |

表中数据来自标准文本[[S12,S7,S31]]。

### 与焙烧类原砂的比较

熟料砂、煤矸石砂等焙烧类铸造型砂系由高温煅烧后的硬质铝矾土、高岭土或废旧耐火材料破碎筛分制成，具有热膨胀系数小、耐火度高的优点，多用于大型铸钢件的面砂和涂料。但它们不属于天然硅砂序列，其化学成分、颗粒形貌和粘结剂适应性均与精选砂存在本质区别，价格亦更高[[S12,S8,S5]]。

### 与覆膜砂的关系

精选砂是高端覆膜砂的理想基材。以精选球形陶粒砂制备的宝珠砂覆膜砂，其透气性检测值可显著优于普通覆膜砂。有实验数据显示，普通覆膜砂透气性约为 102 Pa，满足 ≥ 90 Pa 的合格指标[[S19]]；以高圆整度精选砂制备的覆膜砂则可获得更优越的透气性能。

## 回收与复用性能

精选砂经热法再生处理后，表面残留的粘结剂膜可被有效脱除，硅氧含量回升，回收率可达 85 % 以上，综合砂料循环利用率最高可达 96 %[[S33,S13]]。再生后的砂粒可重新匹配粘结剂制备合格砂芯，实现高质量循环利用[[S35]]。这一点在成本与环保方面进一步放大了精选砂的工艺优势。
