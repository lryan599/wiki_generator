## 概述

压铸单元空循环时间（dry running cycle time / dry cycle time）是表征压铸单元固有运行速度的核心设备性能参数，指压铸单元在没有金属注入、无铸件凝固等待的空运转条件下，按预定动作顺序完成一个完整工作循环所需要的最短时间，单位为秒（s）[[S24,S44]]。该参数与模具结构、合金工艺条件完全独立，仅取决于压铸主机及配套周边自动化设备的自身动力、传动与控制响应特性，因此是设备选型、出厂验收、定期维保中衡量设备理论生产效率的基准指标 [[S38,S28]]。

## 定义与术语

### 标准中文定义

国标 **GB/T 37371-2019《压铸单元 术语》** 对该术语的定义为：

> 压铸单元按机动顺序所作的一次空循环所需的时间。一次空循环时间指：合模、浇注、空压射、开模、顶出、取件、喷涂诸动作时间的总和。[[S40,S33]]

**GB/T 37365-2019《压铸单元 性能检测方法》** 进一步将其补充为：

> 压铸单元在进行各项动作空运转的情况下，实现约定所有工序所需的最短时间。[[S24,S44]]

该标准同时给出的英文对应术语为 **dry running cycle time** [[S24]]。

### 压铸机单机定义

与上述“压铸单元”（含周边辅助设备）的定义相对，**GB/T 21269-2023《冷室压铸机》** 定义了仅针对压铸主机的空循环时间（dry cycle time）：

- **卧式冷室压铸机**：合模、压射、开模、压射回程、顶出、顶出返回各动作时间的总和 [[S32]]。
- **立式冷室压铸机**：合模、压射、压射回程、反料、反料返回、开模、顶出和顶出返回各动作时间的总和 [[S32]]。

热室压铸机的单机空循环时间则定义为合型、压射、压射回程、开型、顶出、顶出返回诸动作时间的总和 [[S31,S20,S11,S13,S29]]。

上述两类定义的核心区别在于：压铸单元空循环时间涵盖了浇注、取件、喷涂等全部周边设备动作，而单机空循环时间仅计主机的合模-压射-开模-顶出动作链。

### 中文别称

在不同应用场景中，该参数存在多种中文别名，使用时需注意其范畴差异 [[S22,S1,S29]]：

| 别称 | 常见使用场景 | 所指范畴 |
|---|---|---|
| 干循环时间 / 空车循环时间 | 压铸机单机产品样本、早期压铸手册 | 仅压铸主机，不含外围设备 [[S22]] |
| 空循环周期 | 设备参数对比文档 | 不同吨位压铸机的公开标称值 [[S37,S9]] |
| 机器空运行时间 | 点检、空运转验收 | 压铸机整机连续空载运行的周期时长 [[S36]] |

### 国际术语对应

国际压铸行业的通用标准英文术语为 **dry cycle time**，在 DIN 类压铸机设备规范中被列为通用性能参数 [[S7,S21,S45]]。empty cycle time、idle cycle time 仅为非通用行业俗称，无统一标准定义指向；dry run time 仅指代单次无负载试运行动作，不属于周期类参数 [[S45]]。

## 工艺地位与用途

空循环时间在压铸设备全生命周期中发挥三个核心作用 [[S28]]：

1. **设备选型评估**：作为衡量设备理论最高运行速度的核心参数，用于推算最大理论生产率（每小时最大理论出件数 = 3600 ÷ 空循环时间，单位为 s）[[S28,S27]]。
2. **设备到货验收**：作为必检指标，判定实际设备性能是否符合出厂标称参数。国标 GB/T 39957-2021《压铸单元 技术条件》明确规定压铸单元一次空循环时间不应大于设备规定的最短循环时间之和 [[S5]]。
3. **定期维保校验**：以出厂或历史空循环时间作为基准，通过对比长期运行后的偏差来判断液压、电气及传动机构的性能衰减程度 [[S28]]。

## 测量方法

### 标准检测仪器

按照 GB/T 37365-2019 的规定，空循环时间的检测工具为**秒表**（也可使用无纸记录仪）[[S44,S17]]。GB/T 21269-2023 允许使用精度不低于秒表的电子计时器，计时起止点为被测运动机构开始移动的瞬间到该机构完成全动作停止的时刻 [[S6,S23]]。

### 测试前置条件

按照 GB/T 37365-2019 的规定，检测需满足以下条件 [[S44]]：

- 压铸单元安全性能符合 **GB 20906** 要求，压铸机符合 **GB/T 21269** 要求且经调试后处于稳定工作状态；
- 在定模板中心安装尺寸 ≥ 0.8b × 0.8b（b 为拉杆间内尺寸）的专用模垫，模垫厚度取 GB/T 21269 规定的压铸模厚度最小值或设备样本厚度均值；
- 模垫两安装平面平行度小于 0.05 mm，锁模工况下模垫应变小于 30 μm/m；
- 待测压铸单元除主机外必须包含配套自动化辅助设备——浇注装置、取件装置、喷涂装置 [[S44]]。

### 测试动作序列

空循环时间的标准测试动作全流程为：关门 → 合模 → 无料空压射 → 开模 → 开门 → 顶出 → 顶回 → 压回 → 取件 → 喷涂，全程无金属填充和铸件冷却等待步骤 [[S44,S40]]。

### 测试规则

在全自动连续运行模式下开展测试 [[S44]]：

| 锁模力 | 累计测试次数 |
|---|---|
| < 10,000 kN | 50 个连续空循环 |
| ≥ 10,000 kN | 30 个连续空循环 |

测试过程中若出现设备报警停机，则本次测试无效，需重新开展。最终取全部有效测试数据的**算术平均值**作为被测压铸单元的空循环时间 [[S44]]。

### 空运转验收要求

GB/T 21269-2023 规定，被测压铸机需完成累计 **8 小时或 5000 次**的连续空运转测试，过程中锁模力 ≥ 80% 额定值、空压射速度 ≥ 5 m/s、压射压力 ≥ 80% 额定值，同时开/合模与压射动作响应灵敏、整机无异常松动、电气系统运行可靠，验收通过后方可判定空循环时间指标合格 [[S6]]。

## 影响因素

空循环时间的数值由多方面因素共同决定：

| 影响维度 | 具体作用机制 |
|---|---|
| 液压系统响应 | 液压系统流量、压力配置及蓄能器充液效率直接决定各动作执行速度，系统动力匹配度越高，空循环时间越短 [[S46,S15,S26]] |
| 电气控制逻辑 | 采用多动作并行时序控制的 PLC/工业电脑控制系统可显著压缩总时长；老式继电器控制因严格串行执行，空循环时间普遍更长 [[S46,S4]] |
| 运动部件结构惯量 | 动模板、曲肘连杆等运动部件质量随合型吨位增大而增大，大吨位机型空循环时间显著高于小吨位机型；同吨位下轻量化设计可降低数值 [[S35,S7]] |
| 周边设备同步策略 | 周边设备（自动浇注、取件、喷涂）与主机并行联动时可压缩单元总空循环时间，串行运行则显著拉长总周期 [[S8,S2,S26]] |

## 与相关参数的关系

空循环时间与多个常用生产指标存在密切的数值关系，但不属于同一概念范畴：

| 参数 | 概念界定 | 与空循环时间的关系 |
|---|---|---|
| 理论循环时间 | 在空循环时间基础上叠加预设的固定工艺延时得到的理想生产周期 | 空循环时间 + 工艺延时 = 理论循环时间 [[S22]] |
| 实际生产循环时间 | 包含金属浇注、充型、凝固冷却的真实生产单周期时长 | 约等于开型时间的 2.5~3.3 倍（开型时间占完整生产周期的 0.3~0.4）[[S16,S27]] |
| 最大理论生产率 | 以空循环时间为基准计算的理论小时出件数 | 小时理论出件数 = 3600 ÷ 空循环时间 (s) [[S28,S27]] |
| 设备综合效率 (OEE) | 综合衡量设备有效利用率的管理指标 | 以空循环时间作为理论基准值，与实际生产循环时间对比得到运行效率维度的 OEE 分项值 [[S28]] |
| 空循环运行频率 | 单位时间内完成的空循环次数 | 空循环频率 = 3600 ÷ 空循环时间 (次/h) [[S27]] |
