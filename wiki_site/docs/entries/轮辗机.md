在铝合金车轮制造领域，“轮辗机”在专业文献中等同于**辗环机**（亦称扩孔机），属于特种锻压设备。其核心功能是对压铸或锻造后形成的带孔铝合金环坯进行局部渐进施压，使环坯截面面积减小、直径扩大，最终成形为无缝环形轮辋工件 [[S11]]。该设备通过径向－轴向联合辗压改善金属流线，提升力学性能与尺寸精度，是铝合金轮毂锻造－旋压复合产线中的关键制坯单元。

## 分类
根据机架安装形式和辗压辊配置，轮辗机主要分为以下几类 [[S11]]：

| 分类依据 | 类型 | 特征与适用 |
|----------|------|-------------|
| 机架安装形式 | 立式 | 适用于辗环外径≤400 mm的中小型轮辋；操作便利，占地面积小 |
| | 卧式 | 适用于辗环外径>400 mm的大尺寸轮辋；刚性更优，可承载更大辗压力 |
| 辗压辊形式 | 径向辗环机 | 仅配置径向辗压辊与芯辊，结构简单，多为立式，适合大批量小型环形件生产 |
| | 轴向‑径向辗环机 | 同时配置径向和轴向辗压辊，多为卧式；可同时控制环坯直径扩展与端面垂直度，适合高精度大尺寸轮辋成形 |

## 结构与组成
轮辗机（辗环机）的关键部件包括 [[S11]]：
- **径向辗压辊**：由主电动机驱动旋转，并在液压系统驱动下向环坯水平移动施加径向辗压力，逐步减薄环坯截面。
- **芯辊**：套入环坯内孔，与径向辗压辊对辊实现径向辗压支撑。
- **轴向辗压辊**：用于支撑变形中的环坯，控制环件宽度及端面对轴线的垂直度，防止轮辋端面翘曲。
- **驱动系统**：为辗压辊提供旋转动力和进给驱动力。
- **机架**：为所有受力部件提供刚性支撑，保证形变可控。

## 工作原理
轮辗机的工作流程为 [[S11]]：
1. 将冲有孔的环坯套入直径略小的芯辊。
2. 主电动机带动径向辗压辊旋转，液压机构驱动径向辗压辊水平移动并对环坯局部施压。
3. 环坯在旋转过程中截面面积逐渐减小，直径逐渐扩大，直至获得目标截面形状与尺寸。
4. 轴向辗压辊同时支撑环坯，控制其宽度及端面与轴线的垂直度；径向辗压辊与芯辊的外形直接决定成形件的截面轮廓。

## 材料与适用范围
- **适用材料**：主要为铸造铝合金A356（AlSi7Mg，ZL101A），用于乘用车轮毂 [[S28]]；6061系列变形铝合金用于商用车轮毂 [[S14]]。  
- **工件范围**：原则上所有直径40～10000 mm的轴对称无缝环形件均可由辗环机加工，小型立式辗环机适合摩托车轮毂等小尺寸环件，中大型轴向‑径向辗环机可加工乘用车、商用车及大客车铝合金轮辋 [[S11]]。

## 在轮毂生产中的工序位置
在铝合金车轮的复合锻造旋压工艺流程中，轮辗（碾锻）工序处于棒料加热之后、初锻之前，起制坯作用 [[S4,S13]]。典型流程为：  
**下料 → 棒料加热 → 碾锻（轮辗工序）→ 初锻 → 终锻 → 切边扩孔 → 预机加工 → 旋压 → 热处理 → 机铣加工 → 表面涂装** [[S4]]

通过引入辗压工序，可显著提高轮辋部位微观组织致密度，获得更均匀的等壁厚结构，成品力学性能优于纯铸造轮毂，并减少后续加工余量 [[S10]]。

## 缺陷与质量控制
轮辗及后续锻造成形中易出现的典型缺陷及其控制措施：

| 缺陷 | 产生机理 | 控制措施 | 引用 |
|------|----------|----------|------|
| 轮辋充型不足（未充满） | 摆辗毛坯与初锻模具定位不匹配，金属流动受阻 | 毛坯随形修改以实现自适应定位，锻造型面配合公差 ≤0.8 mm | [[S18]] |
| 折叠、金属流线紊乱 | 闭式模锻窗口R圆角过小（3～5 mm），金属回流速度过大形成紊流 | 将窗口R圆角增至R20 mm，避免高速回流 | [[S18]] |
| 坯料开裂 | 模具预热温度低，坯料温降过快 | 模具预热温度由150～200 ℃提升至250～350 ℃ | [[S21]] |
| 环件几何精度下降 | 信号辊刚性支承接触环件产生误动作和撞击 | 拆除信号辊，改为目测控制外径 | [[S5]] |

## 与相关设备的区别
轮辗机（辗环机）常与旋压机、轮辋滚型机、辊锻机等设备混淆，主要区别如下：

| 对比项 | 轮辗机/辗环机 | 旋压机 | 轮辋滚型机 | 辊锻机 |
|--------|---------------|--------|------------|--------|
| 功能本质 | 带孔环坯渐进扩径，截面减薄 | 板坯套芯模，旋轮走轨迹整体减薄，仿车削运动 | 预成形钢质轮辋的辊轮整型，弯曲校正 | 相向槽辊对棒料连续局部轧制，实现截面变形 |
| 工作对象 | 压铸/锻造的带孔环坯 | 板坯或管材，套在芯模上 | 钢质轮辋焊接件 | 棒料 |
| 径扩功能 | 有（环坯直径随辗压扩大） | 无 | 无 | 无 |
| 变形性质 | 局部渐进径向‑轴向塑性变形 | 整体或局部减薄旋压 | 弯曲校形，塑性变形量极小 | 沿棒料长度方向的轧制变形 |
| 设备公称力参考 | 最高可达40000 kN | 数千千牛级 | 公称力远低于轮辗机 | 数万千牛级，无法加工环形件 |

- 旋压机结构与仿形车床相似，不具备环坯径向扩径功能，与轮辗机的成形逻辑完全不同 [[S26]]。
- 轮辋滚型机属于弯曲校正设备，无材料流动扩径的塑性变形功能，公称力远低于轮辗机 [[S25]]。
- 辊锻机通过带型槽轧辊对棒料进行连续局部轧制，无法完成环形工件的整周连续渐进辗扩 [[S20]]。

商业宣传中，部分设备厂商会将摆辗、辗环、旋压类轮毂加工设备统称为“轮辗机”，但国际学术术语中，“flow forming”对应强力旋压，“roll forging”对应辊锻，均不属于轮辗范畴 [[S12,S26]]。行业内英文标准称谓为 **ring rolling machine**（辗环机），设备厂商也常使用 **wheel rim rolling mill** 指代铝合金轮毂轮辋专用轮辗机 [[S11,S3]]。

## 标准与规范
当前尚未检索到以“轮辗机”或“辗环机”命名的独立国家标准（GB）或机械行业标准（JB/T），该类设备通常归入锻压机械大类，执行锻压机械通用技术条件系列标准 [[S8,S11]]。在国际标准体系中，也未发现ISO、JIS、ASTM、SAE针对铝合金车轮轮辗工艺的专项标准，相关环件轧制工艺要求多纳入各国锻压成形通用规范范畴 [[S22]]。

## 发展趋势
从工艺与装备发展角度看，轮辗机（辗环机）正朝着以下方向演进 [[S9,S24]]：
- **大型化**：国外最大辗环机加工直径已达10 m，可满足海上采油、核电等特大型无缝环形件的需求。
- **自动化与数控化**：引入CNC技术，减少调整时间，提高制件精度，适应多品种小批量生产。
- **冷辗精密化**：发展中、小型冷辗环机，提升加工精度，实现少、无切削加工。
