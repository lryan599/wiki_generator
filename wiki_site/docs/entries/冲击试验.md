## 概述

冲击试验是通过对金属试样施加单次冲击载荷，测定其断裂时吸收能量的动态力学测试方法，用于评价材料抗冲击性能、识别内部微缺陷、判断材料变脆倾向[[S76,S64]]。在压铸领域，冲击试验是材料韧性评价、热处理效果验证以及工艺质量控制的重要手段。

工业主流冲击测试类型包括夏比（Charpy）冲击试验（简支梁式）、艾佐德（Izod）冲击试验（悬臂梁式）、拉伸冲击试验、无缺口冲击试验和扭转冲击试验[[S54]]。其中，夏比冲击试验是压铸行业评价铝合金、锌合金、镁合金韧性的主流标准方法[[S43,S10,S66]]。

## 试验原理

### 夏比冲击试验（简支梁式）

夏比冲击试验采用简支梁式结构：将带缺口的试样放置于两个固定支座上，摆锤从指定高度下落冲击试样跨中位置，通过冲击前后的势能差计算试样的冲击吸收功[[S43,S10,S66]]。

![夏比冲击试验原理示意图，标注摆锤、固定支座、试样及冲击方向、摆锤升高高度h₁和剩余高度h₂](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/959e6917-5a01-46c1-8a9a-a7506a1ac349/resource)

冲击吸收功计算公式为：AK = G(H₁−H₂)，其中G为摆锤重量，H₁为摆锤初始高度，H₂为冲断试样后摆锤回升高度[[S66]]。冲击韧度值αK由冲击吸收功除以试样缺口处横截面积获得，单位为J/cm²[[S66,S71]]。

### 摆锤式冲击试验机

摆锤式冲击试验机由摆锤、机架、试样支座、指针和度盘等核心部件构成，将一定质量的摆锤从规定高度释放下落冲击放置在支座上的待测试样[[S28,S69,S76]]。

![摆锤式冲击试验装置示意图，展示摆锤式冲击锤、带缺口试样，标注冲击前后能量关系](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f2f4b5f7-394e-43a4-81dc-75ac47920816/resource)

落锤式冲击试验机依靠可更换配重的落锤从设定高度自由下落冲击待测试样，通过调整落锤质量和下落高度控制冲击能量，适配板状、片状试样的动态冲击性能测试[[S21,S60]]。

## 核心参数与试样类型

### 标准夏比冲击试样

标准夏比冲击试样分为V型缺口和U型缺口两类，标准试样外形尺寸为10mm×10mm×55mm，缺口深度2mm[[S58,S27]]。

| 参数 | V型缺口（KV） | U型缺口（KU） |
|------|---------------|---------------|
| 缺口特征 | 较尖锐，应力集中较大 | 较钝，加工方便 |
| 冲击吸收功符号 | AKV | AKU |
| 冲击韧度符号 | αKV | αKU |
| 特点 | 对低温脆性转变响应更灵敏 | 对大尺寸材质缺陷检出能力更强 |
| 主要应用 | 低温韧性评定 | 材质缺陷检验 |

V型缺口试样对低温脆性转变响应更灵敏，U型缺口试样加工难度更低，对大尺寸材质缺陷的检出能力更强[[S27]]。

![符合GB/T 229标准要求的夏比U形缺口冲击试样结构示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/94b1aee4-79e1-47f3-b89d-87910439fedc/resource)

### 压铸专用冲击试样

按照GB/T 13822-92/2017《压铸有色合金试样》标准，压铸无缺口冲击试样需在专用压铸型中直接压铸成型，试样可截为两段，试验时摆锤冲击在试样最窄面[[S5,S47,S53,S19]]。该标准规定的冲击试样尺寸与ASTM、JIS、BS相关标准中的对应试样尺寸一致，不同地区检测结果具备可比性[[S38,S31]]。

### 试验温度

常规室温冲击试验的标准温度区间为20±10℃，同时支持低温冲击测试和高温冲击测试，不同温度下的冲击结果可用于绘制材料韧脆转变曲线[[S26,S70]]。

## 试验标准与规范

### 中国标准

- **GB/T 229系列**（含GB 2106、GB 229-1994、GB/T 229-2007等）：金属夏比摆锤冲击试验通用规范，明确V型、U型缺口标准试样尺寸要求，规定试验机校验要求和试验条件[[S58,S50,S52]]
- **GB/T 13822-92/2017**：压铸有色合金试样标准，规定压铸件专用冲击韧性试样的尺寸和要求[[S38,S31,S19]]
- **GB/T 3808-1995**：摆锤式冲击试验机技术校验要求，需定期经国家计量部门检定[[S58]]

### 国际标准

- **ASTM E23**：金属材料缺口试样冲击试验通用标准，规定Type A、B、C三类夏比冲击试样，是国际压铸行业通用参考标准[[S41,S51,S14,S62,S20,S3,S45,S7]]
- **ISO 148-1**：金属夏比冲击试验国际标准
- **JIS Z 2242**：日本工业标准金属材料冲击试验通用规范，是压铸件出口日本市场需遵循的核心标准[[S55]]

### 标准在压铸件检测中的引用情况

在ASTM压铸件规范中明确：除非在采购合同中特殊约定由制造商担保，否则压铸件合格验收不强制以冲击试验结果为判定依据。当合同或订单明确要求时，铝合金压铸需采用ASTM E8图18的拉伸试样和ASTM E23图6的冲击试样[[S41,S14,S62]]；锌合金压铸冲击测试要求在ASTM B791中引用[[S7,S51]]；镁合金压铸冲击测试要求在ASTM B94中引用[[S45,S20]]。

## 在压铸工艺中的角色

### 合金韧性评价

**压铸铝合金**：对于Al-Si-Cu系压铸铝合金（如ADC12），冲击值随Mg含量降低而升高，随Zn含量上升呈小幅下降趋势。由于面心立方结构特性，常规压铸铝合金无明显低温韧脆转变现象，部分牌号V型缺口冲击试验因结果分散度高不作为常规验收指标[[S35,S22]]。

**压铸锌合金**：在温度低于-10℃时冲击韧度会显著下降；低Al含量的压铸锌合金抗冲击性能优异，适用于承受动载荷的结构类压铸件[[S68,S11]]。压铸锌合金缺口冲击测试可参考DIN 50116标准执行[[S11]]。

**压铸镁合金**：可吸收冲击能，冲击韧性随Gd含量增加单调递增。AE系列压铸镁合金的冲击韧性高于AZ、AM、AS系列，且压铸镁合金的抗冲击性能显著优于同成分砂型铸造镁合金[[S64,S67,S2]]。

### 热处理效果验证

验证热处理对压铸合金韧性的提升效果，标准流程为采用GB/T 229规定的夏比V型缺口冲击试验获取不同热处理工艺下的冲击功数据，同步结合冲击断口SEM形貌观测，对比韧窝占比、解理面占比变化，定量表征韧性提升幅度[[S39,S18]]。

### 缺陷对冲击性能的影响

压铸过程中产生的气孔（等效直径20~100μm）、缩松、冷隔三类典型缺陷会显著降低材料动态冲击性能[[S34,S1,S12]]。缺陷减少有效承载面积并形成局部应力集中作为裂纹萌生源，大幅降低冲击功值。不同批次带相同名义工艺参数的压铸试样因缺陷分布差异，动态冲击性能表现出显著离散性[[S1]]。

压铸铝合金减震塔不同位置的冲击性能与区域缺陷含量直接相关：远离内浇口的填充末端位置因卷入的气孔、夹渣多，冲击功相比近浇口无缺陷区域可降低30%以上[[S12]]。

### 结构设计与选材依据

汽车铝合金轮毂开发中采用13°动态冲击试验验证轮毂耐冲击性能，测试条件为规范载荷492kg，落下高度230mm，分别在0°、90°、180°、270°四个方位施加工况冲击，作为最终合金选型的核心判定依据[[S23,S25]]。

一体化压铸前舱结构开发阶段，动态台架冲击试验数据与仿真标定结果结合，可校准压铸铝合金材料的失效模型，保证整车碰撞仿真中开裂模式和承载能力的预测精度[[S36]]。

## 设备校准要求

摆锤式冲击试验机定期强制检定要求[[S63,S44,S74]]：

| 校准项目 | 要求 |
|----------|------|
| 机座不水平度 | ≤2/1000 |
| 检定环境温度 | 15~25℃ |
| 摆锤自由摆动总能量损失 | ≤最大冲击能量的2.5%（小能量规格可适当放宽） |
| 仅转轴摩擦和风阻能量损失 | ≤最大冲击能量的1% |
| 摆锤等效质量偏差 | ≤±0.4% |
| 冲击中心到转轴距离偏差 | ≤±3mm |
| 悬臂梁机型额定冲击速度 | 3.35m/s |
| 冲击刃位置公差 | ≤±0.2mm |
| 度盘显示精度 | 不低于最大量程的±0.5% |
| 维修或更换核心部件后 | 必须重新完成全项检定 |

常规周期强制检定应符合GB/T 3808相关要求[[S73]]。

## 试验操作步骤

标准冲击试验通用操作步骤[[S69,S65,S46,S16]]：

1. **试验前检查**：检查整机运行状态，完成空击校验确认能量损失符合要求
2. **量程选择**：匹配试样预估断裂能量选择对应量程的摆锤，确保试样断裂消耗能量处于摆锤总能量的10%~85%区间内
3. **试样安装**：按对应机型标准安装要求固定试样，调整支座跨距使试样缺口正对冲击刃中心
4. **冲击与读数**：释放摆锤完成一次冲击，读取冲击示值并完成风阻、摩擦修正；低能量场景下需额外补充抛掷修正
5. **结果处理**：每组至少测试3~5个有效试样，排除未在缺口位置断裂的无效数据，取平均值作为最终结果

## 压铸试样特殊性要求

### 试样制备要求

压铸冲击试样应清理毛刺，不得弯曲和矫直，非夹持部分不得有顶杆印痕、擦伤、麻面、冷隔、裂纹、夹杂和孔穴等缺陷[[S5,S53,S19]]。

### 小尺寸试样适应性

符合GB/T 13822标准的压铸小尺寸冲击试样，单支标准长度试样可在试验前截为两段独立开展测试[[S5,S19]]。非标小尺寸试样可换算冲击模数MC（单位体积吸收能量）进行性能对标，需满足试样长径比l=6d、支点间距为5d等匹配条件[[S26]]。

### 板状试样处理

厚度低于3mm的压铸薄板试样不开展冲击测试，厚度超过10mm的板状试样需单侧机加工至10±0.2mm厚度，冲击缺口优先开在未加工的压铸原始表面[[S32,S16]]。若为各向异性压铸板材，需沿板材充模流动的纵横两个方向分别取样测试[[S32]]。

## 局限性与替代方法

### 尺寸效应与局限性

- 压铸铝合金因面心立方结构特性，无明显低温韧脆转变，部分牌号V型缺口冲击试验结果分散度高，不作为常规验收指标[[S22,S35]]
- 压铸薄壁件由于壁厚限制，难以制备标准尺寸冲击试样
- 小尺寸试样的冲击试验结果与标准试样间存在可比性限制

### 替代检测方法

针对压铸薄壁件传统冲击试验的局限性，更适配的替代检测方法为断裂韧性测试：通过在紧凑拉伸试样或三点弯曲预制尖锐预裂纹试样上施加载荷，测算应力强度因子K₁C或J积分值[[S18,S4]]。该方法无需依赖大尺寸标准试样，可直接量化带微裂纹压铸零件的临界裂纹扩展阻力，预测不同工况下的失稳风险，精度优于小尺寸冲击试验[[S18]]。

## 验收判定规则

压铸合金冲击韧性检测标准流程中，每组测试试样数量不少于3根，若超过2根不合格可加倍重试。重试后仅允许1根不合格且整体平均性能达标，即可判定该批次材料韧性合格[[S8]]。压铸锌合金冲击韧性测定应符合GB 229的规定，但不作为验收依据[[S8]]。
