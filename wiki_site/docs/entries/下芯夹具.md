## 定义与术语

下芯夹具（又称下芯工装、芯子安放夹具，英文对应术语包括 core setting fixture、core placement jig、core jig、core positioner，配套的自动化设备则称 core setter/下芯机）是铸造领域用于将砂芯/型芯精准定位并夹持后，安放到铸型内指定位置的专用工艺装备，属于铸造核心辅助工装之一[[S9,S10,S30]]。在国标 GB/T 25370—2020 所定义的铸造机械术语体系中，下芯夹具的作业工位与 core setting station（下芯工位）相对应，是组成铸造生产线的重要功能单元[[S6]]。

## 功能与工艺角色

下芯夹具在保证铸件尺寸精度和实现机械化下芯作业中承担着核心作用[[S11,S15]]：

- **提高下芯定位精度**：通过夹具上的定位元件与夹紧机构，将砂芯在夹具内部完全约束六个自由度，消除人工下芯时难以避免的定位偏差，最终提升铸件内腔的尺寸精度；
- **降低操作者劳动强度**：在手工造型或流水线造型中，借助夹具的吊运与机械/气动夹紧功能，可将多枚砂芯在组芯工位预装后一次吊装就位于砂型中，减少了操作工人的手工搬运与位置调整负担；
- **实现多砂芯同步就位**：部分带负压吸附结构的下芯夹具可通过真空吸附方式抓取砂芯，移至铸型上方后撤销负压、通入正压，实现砂芯与夹具的快速脱离，适配多砂芯一次性精准落位的情形[[S11]]。

在常规砂型铸造工艺流程中，下芯夹具的作业位置处于制芯、修芯等工序之后，铸型造型完成而合箱操作之前——夹具在组芯工位完成砂芯夹持与定位检验后，移送至上砂箱或下砂箱上方完成下芯动作，后续再开展砂芯位置复核、合箱密封等操作[[S6,S21]]。

### 发动机缸体类复杂铸件的典型工作流程

以发动机汽缸体为例，下芯夹具的常规工作流程如下[[S21]]：砂芯先在组芯模具中完成定位预装，下芯夹具通过定位销与组芯模具配合就位；利用夹具上配套的位置检具（如测高度检具）核验各砂芯的相对位置精度，确认合格后通过手轮或气缸驱动夹紧机构（如吊砂钩）夹持砂芯；整体吊运至下砂型上方，通过合箱销完成夹具与砂箱的相互定位，待夹具平稳坐落于砂箱后释放夹紧机构，整套砂芯一次性落入铸型预定位置。下芯夹具全程须保持水平状态，常借助调节螺栓进行平衡校准[[S21]]。

## 类别与选型

### 按驱动结构分类

下芯夹具按驱动方式及结构特征可分为七种主流类型[[S11]]：

| 类型 | 特征及适用场景 |
|---|---|
| 简易下芯抓爪 | 结构简单，用于形状简单砂芯的手动下芯作业 |
| 电磁吸盘式下芯夹具 | 利用电磁力吸附砂芯，适用于特定材质砂芯的抓取与释放 |
| 齿轮齿条自动下芯夹具 | 由气缸、齿条、齿轮及夹板等构成，可实现自动夹持与释放 |
| 气囊式下芯夹具 | 利用气囊膨胀夹持砂芯，接触柔和，适合易损砂芯 |
| 螺旋手动下芯夹具 | 人工旋转螺杆实现夹紧，适配中、小批量复杂铸件（如缸体）生产 |
| 手柄杠杆下芯夹具 | 以手柄-连杆机构驱动夹钩同步动作，用于夹持较重砂芯 |
| 气动下芯夹具 | 定位精度高、动作速度快，适于缸体等复杂铸件的大批量流水线生产 |

其中，气动下芯夹具构造较为复杂，但兼具定位严密与操作方便的优点，是批量生产线中最常见的自动下芯方案之一[[S11]]。

### 按专业化程度分类

从工装设计与适用范围角度，下芯夹具可分为以下两类[[S3]]：

| 类别 | 特点 | 优劣 |
|---|---|---|
| 专用定制下芯夹具 | 针对单一特定砂芯设计制造，定位精度高 | 通用性差，砂芯改型后工装基本报废，制造成本高；大批量稳定生产场景下优势明显 |
| 模块化下芯夹具 | 采用标准化、可更换的定位元件与夹紧元件拼装而成 | 调整便捷、研发周期短，适配多品种中小批量砂芯生产；但整体定位精度低于专用夹具，长期使用后拼接面易出现磨损间隙 |

### 不同下芯方式与机构的适配场景

下芯夹具与铸造设备的下芯机构（起芯机构）配合使用，不同机构的工况适用性有所差异[[S22]]：

| 机构型式 | 适用砂芯类型 | 优势 | 局限 |
|---|---|---|---|
| 漏出式 | 形状简单、重量轻的小型砂芯 | 操作难度低 | 下芯同轴度精度有限 |
| 移出式 | 中型箱体类砂芯 | 运动平稳 | 作业占地空间大 |
| 转出式 | 多工位联动下芯生产线 | 作业效率高 | 设备成本高，维护复杂 |

### 在高压压铸中的适用性边界

常规砂质下芯夹具的设计基础依赖于砂芯具有易溃散、可机械夹持或在低压铸造、重力铸造中承受较低冲刷力的特点。在普通高压压铸场景中，高速高压金属液对型芯产生极大的不平衡冲击力，常规砂芯的力学强度不足以承受此类载荷。因此，绝大多数常规金属型高压压铸场景采用模具内置的液压缸驱动芯缸完成抽插芯操作，不依赖外置下芯夹具进行砂芯下入作业。只有在使用高力学强度的可溶盐芯等特殊型芯的高压压铸工艺中，才可能涉及外置下芯装置的介入[[S25,S23]]。

## 典型结构与组成

下芯夹具作为一种专用夹具，其基本架构与通用机床夹具的构成原则一致，但在功能元件配置上具有铸造工装的典型特征。下芯夹具的通用核心组成包括以下四大功能模块[[S16,S28,S14]]：

- **定位元件**：直接与砂芯的定位面相接触或配合，用于限定砂芯在夹具中的六个自由度，保证砂芯安放位置精度。常见形式包括支承钉、支承板、定位销、定位心轴、定位套、V形块等[[S16,S28]]。
- **夹紧机构**：在砂芯定位后施加夹紧力，防止下芯过程中因自重或冲击力造成砂芯移位。常见类型涵盖手动螺旋夹紧、气动夹紧与液压夹紧。大批量自动化生产场景中优先采用联动多点夹紧结构，可同步完成多砂芯夹持，以缩短辅助作业时间[[S28]]。
- **导向元件**：引导砂芯沿预定路径准确下入铸型型腔，避免下芯过程中发生磕碰、刮蹭。典型导向件包括导柱、导套与导向槽，其尺寸精度与配合要求均须符合 GB/T 8845—2017《模具 术语》等标准对导向类元件的相关规定[[S30]]。
- **夹具体**：作为承托所有定位元件、夹紧机构及导向元件的基础骨架，常见结构形式有铸造结构、焊接结构及组装结构。夹具体须具备足够的刚度与强度，避免在夹持与吊运砂芯过程中发生变形而导致砂芯位置偏移[[S16,S28]]。

在复杂铸件专用下芯夹具（如缸体下芯夹具）中，夹紧动作常以气缸为驱动源，由夹板、气缸及夹具本体等构件协同工作，实现多砂芯同步快速夹紧与释放[[S29]]。

## 设计要点

下芯夹具须满足高刚度与高精度要求[[S21]]，设计时需重点关注以下方面：

- **夹具体刚度**：夹具体是整套工装的精度保持基础，其结构设计应避免因受力变形引发的砂芯定位偏移现象[[S28]]。
- **定位元件精度与耐磨性**：芯头定位部分不应设置拔模斜度；为便于加工与保证精度，侧壁与定位块可采用装配式结构[[S21]]。定位元件工作表面推荐采用 20Cr 或 20 号碳素钢制造，渗碳层深度控制在 0.8～1.2 mm，淬火后硬度达到 HRC 58～62，以保证工作面长期耐磨和精度稳定[[S1]]。
- **适应设备要求的工装尺寸校核**：下芯夹具设计需与造型/造芯设备的工艺参数匹配，如 Z2310 型翻台式造型机适配的下芯夹具设计最大外廓尺寸为 1440 mm×1000 mm，设计时须同步校核起模行程、翻台高度与震击力等参数，以避免下芯作业时发生设备干涉[[S12,S27]]。同理，763 型壳芯机配套下芯夹具的顶芯杆直径范围为 φ20～φ30 mm，固定板间距须大于 20 mm，宽度限值不超过 350 mm[[S7]]。
- **经济性与维护性**：下芯夹具制造成本较高，日常维护工作量亦较大，同一车间中不宜过度配置数量[[S21]]。在大批量复杂铸件生产的工装设计流程中，下芯夹具须与组芯模具、砂箱和砂芯定位系统协同设计，图纸文件是铸造工艺装备设计文件的必要组成部分[[S13]]。

## 标准化

在现有公开检索范围内，尚未发现单独以“下芯夹具”命名的独立国家推荐标准或专项铸造行业标准。下芯夹具的设计与制造可参照通用夹具类标准作为合规依据，已公开的现行有效标准包括：

- HB 4534 系列-1991《航空行业组合夹具定位夹紧元件》；
- HB 4535 系列-1991《航空行业孔系组合夹具基础件》；
- JB/T 系列机械工装夹具通用零件标准。

上述标准覆盖了下芯夹具设计所涉及的标准紧固元件、通用定位与夹紧元件的尺寸、材料、热处理以及公差配合等技术要求，可作为设计输入与质量控制的参考基准[[S4,S19]]。
