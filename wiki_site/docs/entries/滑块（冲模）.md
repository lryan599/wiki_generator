## 概述

滑块（冲模）在国家标准 **GB/T 8845-2017《模具 术语》** 中定义为“在斜楔作用下沿变换后的运动方向作往复滑动的零件”（条目编号 2.1.1.98），是冲压模具中将压力机垂直运动转换为水平或倾斜运动的核心功能组件 [[S17,S25]]。当冲压件存在侧向倒扣、侧孔等特征，无法沿直向脱模时，必须采用滑块结构实现侧向成型与脱模 [[S3]]。

## 分类

冲模滑块按驱动方式和运动方向可分为以下主要类型：

| 分类 | 运动特征 | 典型应用 |
|------|----------|----------|
| 斜楔滑块（水平动作） | 斜楔驱动滑块沿水平方向运动 | 侧向冲孔、侧向翻边 [[S10]] |
| 斜楔滑块（向上倾斜动作） | 滑块沿向上倾斜10°以内方向运动 | 特殊角度侧成型 [[S10,S51]] |
| 斜楔滑块（向下倾斜动作） | 滑块沿向下倾斜10°以内方向运动 | 向下方向的侧向成型 [[S10]] |
| 空间机构斜楔（吊楔） | 滑块悬挂在上模，由下模斜楔传动 | 大型覆盖件修边冲孔 [[S11]] |
| 多滑块组合 | 多组独立斜楔-滑块协同工作 | 复杂覆盖件多侧同步加工 [[S29]] |

斜楔滑块是应用最广泛的类型，通过斜楔将压力机上模的垂直向下运动转换为滑块的水平侧向运动 [[S50]]。

## 结构与组成

典型冲模斜楔滑块机构由五大核心部分组成 [[S3,S6,S43]]：

- **滑块本体**：直接安装侧向成型/加工凸模的承载体，需具备足够刚性与耐磨性 [[S44]]
- **斜楔（斜导柱）**：安装在上模，随上模下行通过斜面接触驱动滑块沿侧向运动，是动力转换的核心部件 [[S18,S50]]
- **耐磨板（防磨板）**：安装在滑块与导向面的接触位置，当滑动面接触压应力超过50MPa时必须设置，常用淬火钢材质（硬度52~56HRC），可有效降低滑动磨损 [[S18,S44]]
- **导向压条/导板**：为滑块提供运动导向约束，保证滑块沿预设轨迹平稳运动 [[S18,S14]]
- **复位机构**：分为弹簧弹性复位和强制刚性复位两类，弹性复位依靠弹簧力带动滑块回到初始位置，刚性复位依靠斜楔回程带动滑块抽离工件 [[S22,S35,S50]]

滑块的导向精度要求：滑块与导轨的常规配合精度为H7/f6，高精度工况下可采用H7/h6或H8/h7配合 [[S18,S14]]。滑块侧向移动的允许偏摆量为100mm行程内不超过0.03mm [[S1]]。

## 工作原理

斜楔滑块的核心运动转换逻辑是将压力机输出的垂直往复运动转换为滑块的水平或倾斜运动 [[S51]]。其运动关系满足：

$$S_1 = \frac{S}{\tan\alpha}$$

式中 $S_1$ 为斜楔垂直行程，$S$ 为滑块侧向行程，$\alpha$ 为斜楔斜面与垂直方向的夹角（斜楔角）[[S10,S51]]。

工作流程：压力机滑块下行→斜楔随上模下降→斜楔斜面与滑块接触→滑块沿导向槽侧向进给→完成侧向冲孔/翻边/成型→压力机滑块回程→复位机构带动滑块退回→工件取出 [[S19,S10]]。

复位结构分为两类：弹性复位依靠模座内部弹簧提供复位动力，结构简单维护方便；刚性复位依靠斜楔自身结构特征驱动滑块反向退回，适用于侧向力大、弹簧无法可靠复位的工况 [[S50,S22,S35]]。

## 设计参数

### 斜楔倾角与驱动角

| 参数 | 推荐范围 | 说明 |
|------|---------|------|
| 常规水平运动斜楔角 $\alpha$ | 30°~50° | 常用40°；$\alpha$ 越小侧向力越大但行程越短 [[S50]] |
| 大行程工况 $\alpha$ | 可提升至50° | 增大滑块移动距离 [[S35]] |
| 空间吊楔中小模具倾角 $\beta$ | ≤30° | [[S11]] |
| 空间吊楔大模具倾角 $\beta$ | ≤22° | [[S11]] |

### 导向与配合参数

| 参数 | 推荐值 | 来源 |
|------|--------|------|
| 滑块导向长度与宽度比 $L:B$ | (2~2.5):1 | [[S18]] |
| 常规配合精度 | H7/f6 | [[S18,S14,S5]] |
| 高精度配合 | H7/h6 或 H8/h7 | [[S18]] |
| 导滑槽表面粗糙度 $R_a$ | 0.63~1.25μm | [[S5]] |
| 抽芯后滑块保留在导滑槽内的长度 | ≥总导滑长度的2/3 | [[S18]] |

### 锁紧角与行程安全

| 参数 | 设计准则 | 来源 |
|------|---------|------|
| 锁紧角 $\alpha'$ | 比斜楔驱动角大2°~3° | [[S49]] |
| 滑块总行程 | 工作行程 + 工件取放安全余量 | [[S14]] |
| 手工作业安全余量 | 50mm | [[S14]] |
| 自动化作业安全余量 | 30mm | [[S14]] |
| 斜楔与滑块初始接触面长度 | ≥50mm，且≥斜面总长的2/3 | [[S11,S14]] |

斜楔的有效行程必须大于滑块的全行程；滑块高度 $H_2$ 应小于其长度 $L_2$，一般取 $L_2:H_2 = (2\sim1):1$ [[S14,S35]]。

## 材料与热处理

冲模滑块材料的选择随载荷等级和工况要求而异：

| 材料 | 热处理硬度 (HRC) | 适用工况 | 来源 |
|------|------------------|----------|------|
| 45钢 | 40~45 | 常规低载荷侧向抽芯、侧向压制成型 | [[S16,S5]] |
| T8A/T10A | 54~58 | 中等载荷、较高耐磨要求 | [[S31,S5]] |
| CrWMn | 58~62 | 高耐磨、承受较大侧向锁紧力 | [[S26]] |
| Cr12MoV (SKD11) | 58~62（常规）；56~60（二次硬化） | 长寿命级、厚板冲裁、复杂弯曲 | [[S7,S8,S28]] |
| H13 (4Cr5MoSiV1) | 45~52 | 热冲压、热弯曲等连续高温工况 | [[S47,S28]] |

导滑槽（滑块配套导向部件）常用45钢、T8、T10等材料制作，推荐热处理硬度52~56HRC，滑块与导滑槽的硬度差建议控制在5~8HRC，避免同硬度钢材直接配对滑动导致拉伤 [[S5,S48,S27]]。

成型滑块作为直接参与摩擦磨损的工作零件，需整体淬火至60~64HRC [[S21]]。

## 典型失效与预防

冲模滑块常见失效形式及防控措施如下 [[S46,S38]]：

| 失效形式 | 形成机理 | 预防措施 | 来源 |
|----------|----------|----------|------|
| 磨损 | 配合面硬度过低、润滑不足、异物进入间隙；分初期塑变磨损→正常摩擦磨损→后期疲劳损坏三阶段 | 合理匹配滑块与导滑槽硬度，表面强化处理，定期清理润滑 | [[S46,S38]] |
| 拉伤 | 滑块与导滑槽硬度差过小、表面粗糙度不达标，发生粘着磨损 | 硬度差控制在5~8HRC，镜面抛光配合面 | [[S48,S27]] |
| 卡死 | 配合间隙过小、热处理变形未做配磨补偿、偏载导致导向面咬粘 | 预留0.02~0.05mm合理配合间隙，热处理后研磨配刮 | [[S4,S5]] |
| 回位不良 | 回位弹簧弹力不足、斜导柱回程干涉、导向面磨损导致阻力剧增 | 弹簧预压量取工作行程的1.2~1.5倍，运动路径干涉校验 | [[S33,S19]] |
| 锁紧让位变形 | 材料屈服强度不足、热处理硬度偏低，侧向锁紧高压造成塑性变形 | 选用强度足够的合金工具钢，保证热处理硬度，避免薄壁应力集中 | [[S34,S38]] |

## 与压铸滑块的关联与对比

冲模滑块与压铸侧抽芯滑块在核心结构上具有共性：二者均属于沿导向结构往复滑动的模具组件，均采用斜楔/斜销驱动并配楔紧块锁紧定位，均在脱模方向与开模方向存在夹角时完成侧向成型/抽芯动作 [[S25]]。

但压铸工况的特殊性使二者存在显著差异：

| 对比维度 | 冲模滑块 | 压铸侧抽芯滑块 |
|----------|----------|----------------|
| 工作介质 | 常温板料 | 高压高温液态金属 |
| 密封要求 | 无特殊要求 | 需配合面密封结构，防止熔融金属窜入导滑槽导致卡死 [[S9]] |
| 冷却需求 | 无内置冷却 | 需内置独立冷却回路维持工作温度 [[S20,S13]] |
| 材料选用 | 多为冷作模具钢（Cr12MoV、CrWMn等） | 多为热作工具钢（4Cr5MoSiV1），成型部分真空淬火至50~55HRC，表面可做耐冲蚀涂层 [[S52]] |
| 导滑槽结构 | 多为开式矩形导滑槽 | 锌铝合金常用外露式矩形导滑槽；铜合金优先采用封闭偏心套圆滑块结构，防止杂物进入和高温变形 [[S15]] |
| 寿命要求 | 按冲次和维护周期设计 | NADCA规范要求正常工况10万~15万次以上，锁止楔块需有足够承压面积抵抗金属液反推力 [[S45,S23]] |

## 应用场景

**冲压领域**：冲模滑块广泛应用于汽车覆盖件冲压工艺（通过斜楔驱动完成侧翻边、侧向冲孔等工位），以及精密电子元件高速级进冲压中微米级精度的侧向切边、局部成型 [[S24]]。

**压铸领域**：压铸侧抽芯滑块已在铝合金压铸件（减振器接头、带轮、轮毂、气缸体、机车门把手等）、锌合金压铸件（轿车阀体、化油器主体等）量产模具中成熟应用，适配斜销抽芯、弯销抽芯、齿轮齿条抽芯、液压抽芯等多种驱动形式 [[S42,S32,S41,S2]]。
