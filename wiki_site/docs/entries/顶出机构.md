## 定义与基本功能

顶出机构（又称顶出系统、脱模机构，英文对应 ejection mechanism、ejector system、ejector assembly，旧称 knockout mechanism）是压铸模中将已冷凝成形的压铸件从型腔内顶推脱离，使压铸件与型腔分离的所有相关零件的总称，绝大多数布置在压铸模的动模半侧 [[S49,S22,S59]]。它是压铸循环中承接开模与取件工序的关键子系统。

顶出机构的动力来源分为两类：
- **机械式动力**：依靠压铸机的开模动作提供推力，顶出行程固定；开模时动模后移过程中顶出板接触压铸机固定挡块，将铸件顶出 [[S58,S61]]。
- **液压式动力**：依靠压铸机内置的液压油缸直接驱动顶出板，顶出速度、时间、行程均可自由调节，可实现多次顶出动作，为现代压铸机主流配置 [[S58,S1]]。

在压铸全循环中，顶出机构的标准动作时序为：保压结束、金属液完全凝固冷却之后，先完成侧向抽芯动作，随后压铸机执行开模；开模行程到达设计指定位置后顶出机构启动，将压铸件完全顶出动模型腔，之后取件、喷涂、合模复位，进入下一循环 [[S11,S31,S69]]。

## 分类

顶出机构可按动力来源、动作层级和顶出元件形式进行分类 [[S44,S13,S53]]。

| 分类维度 | 类别 | 特征 |
|----------|------|------|
| 动力来源 | 机动（机械式） | 利用开模动作驱动，行程固定，有一定冲击 [[S58,S28]] |
|          | 液压式 | 独立液压缸驱动，行程、速度可调，动作平稳，支持多次顶出 [[S42,S27,S59]] |
|          | 手动式 | 人工操作，多用于特殊或定模侧滞留场景 [[S13]] |
| 动作层级 | 一次顶出 | 单次顶出动作即完成全部脱模 [[S44,S4]] |
|          | 二次（多级）顶出 | 分步顶出，先松动铸件再完全顶出，避免应力集中 [[S46,S33,S68]] |
| 顶出元件 | 顶针（推杆） | 圆杆件，应用最广，结构简单可靠 [[S44,S42,S24]] |
|          | 顶管（司筒） | 管状件，套在固定型芯外侧，适用于通孔/BOSS柱 [[S32,S4,S20]] |
|          | 扁顶针 | 截面为矩形，用于深筋位、狭小空间 [[S32,S3,S55]] |
|          | 推板（脱料板） | 沿铸件周边接触，顶出力均匀，不留顶痕 [[S4,S48,S57]] |

**顶针顶出**是最普遍的型式，其制造、修配方便，更换容易，但顶出接触面积小，不适宜直接用于脱模阻力大、薄壁易变形的铸件部位 [[S44,S42,S24]]。

**扁顶针**适用于筋条深窄、无法布置常规圆顶针的部位，其截面形状适配筋条轮廓，接触面积更大，可有效避免深筋处顶出断裂变形 [[S32,S3,S55]]。

**顶管（司筒）**套在固定型芯外侧，将铸件整体从型芯上剥离，顶出力均匀，能较好地保证铸件内孔与外圆的同轴度 [[S32,S4,S20]]。

**推板**从铸件整个外周同步顶出，适用于大筒形、薄壁容器、罩壳类压铸件，顶出平稳，不易变形，表面几乎不留痕迹；非圆形轮廓推板加工难度高，大型深腔铸件需配套引气装置防止真空吸附导致变形 [[S4,S48,S57]]。

**二次顶出机构**通过两级分步动作完成脱模，常用于包紧力大、深腔或一次顶出易变形断裂的铸件。常见结构型式有双顶针板二次顶出、单顶针板二次顶出、三角块超前二级顶出等；先使铸件整体松动脱离型芯，再完全顶出，可避免转接部位应力集中断裂，多用于小件大批量自动化生产 [[S46,S33,S68,S23]]。

**液压顶出**由压铸机内置液压缸驱动顶出板，支持多次分级顶出，行程精确可调，是现代大中型自动化生产线的标配顶出驱动型式 [[S42,S27,S59]]。

## 典型结构组成与装配

一套完整的压铸模具顶出机构通常由以下零部件构成 [[S58,S50,S41,S45,S5]]：

| 功能组 | 零件 | 作用 |
|--------|------|------|
| 顶出元件 | 顶针、顶管、扁顶针、推板 | 直接与铸件接触，施加顶出力 [[S58]] |
| 固定件 | 顶针固定板 | 固定各类顶出元件尾部，均匀传递顶出力 [[S50]] |
| 受力件 | 顶针板（推板） | 承接压铸机顶出动力，与固定板紧固为一体 [[S50]] |
| 复位元件 | 回位销（复位杆、回程柱） | 合模时首先接触定模分型面，强制顶出机构完全复位 [[S41,S45]] |
| 限位元件 | 限位块（止动梢） | 限定最大顶出行程，保护冷却水管等组件，保证行程精度 [[S5]] |
| 导向元件 | 顶出导柱、顶出导套 | 引导顶出板平稳直线运动，防止卡死和弯曲 [[S41,S45]] |

装配层级（从型腔侧向压铸机侧）为 [[S16,S50,S41]]：顶出元件穿过动模型腔镶块、动模套板的对应孔位，尾部固定在顶针固定板上；顶针固定板与顶针板通过紧固件锁合为整体；顶出导柱、导套安装在顶针板与动模支承板之间实现导向；回位销尾部固定在顶针固定板上，头部朝向定模分型面；限位块安装在顶针板与动模底板之间控制最大行程；整套组件布置在动模侧的模脚空间内部。

## 关键设计参数与要求

### 顶出力与顶出行程

顶出力设计推荐取值为压铸件脱模总阻力的 **1.2 倍**（总阻力包含铸件收缩包紧型芯的脱模阻力及运动摩擦力），实际选型需匹配压铸机标称顶出能力 [[S58]]。

顶出行程最低要求保证压铸件**完全脱离动模型腔**，并满足人工取件空间；若配套自动化旋转取件结构，需额外预留行程余量 [[S58,S70]]。

### 顶杆尺寸与布局

顶杆滑动导向段的有效配合长度一般取顶杆直径的 **2～4 倍**，大直径顶杆取偏小倍数，小直径顶杆取偏大倍数，避免导向段过长增大滑动卡滞风险、过短导致运行晃动 [[S70]]。

顶杆直径应比成型镶件的对应内孔直径**小 0.4～0.6 mm**，顶杆边缘与型芯侧壁预留最小 **3 mm** 安全距离，以保证型芯边缘强度，防止高压金属液窜入顶杆配合间隙引起卡滞 [[S70,S60]]。

顶针布局应**均匀对称布置**，顶出合力作用点对准铸件重心，优先将顶出点设置在脱模阻力最大、铸件刚性高的部位；禁止直接作用在薄壁平面上，避免顶穿或变形；允许将顶出点布置在浇道、溢流槽或额外增设的工艺凸台上，以避免在铸件外观面留下可见顶痕 [[S58,S70,S67,S24]]。

小规格有托顶针的工作直径覆盖 φ1.0～φ2.5 mm 区间，当工作直径小于 φ1.5 mm 时，加强段直径常规取 φ2.0～3.0 mm，以保证细小顶针刚性，防止弯折断裂 [[S10]]。

### 回位确认

常规回位确认采用**复位杆**，合模时复位杆首先接触定模分型面，将整个顶出机构推回初始位置。高自动化量产模具配套顶出行程限位开关，输出全行程回位校验信号，避免合模时伸出的顶针与机构发生干涉碰撞 [[S67,S56]]。

## 顶出缺陷及其工艺关联

### 常见缺陷机理

- **顶白**：顶出局部压力超过高温下铸件表层的许用应力，在表面形成发白应力痕迹，多发生在顶出速度过快、铸件未完全冷却时强行顶出的工况 [[S6,S70]]。
- **顶裂**：顶出局部受力超过铸件高温拉伸断裂强度，导致开裂，多由受力不均或过大的顶出压力超出对应合金最大许用受顶压力引发 [[S58,S60]]。
- **顶凸**：顶出速度过快、顶出力远大于包紧力时，铸件非顶出侧被反向顶起形成凸起变形；模温过低、表层脆化时发生概率显著升高 [[S2,S12]]。
- **粘模顶出困难**：铸件收缩后对型芯产生的包紧力过大，超出额定顶出力，当不同部位包紧力差值达到约 3 倍以上时，会出现单侧粘模顶不出的现象 [[S18]]。
- **顶出变形**：顶出机构出力不平衡、各部位位移不同步，强制顶出薄壳类铸件会引发永久翘曲变形 [[S31,S2]]。
- **顶针痕**：顶针作用于冷却不久的铸件表面形成的典型圆形痕迹。可通过在溢流槽或工艺凸台处设置顶针并在后续工序去除凸台来避免外观面痕迹 [[S31,S69,S24]]。

### 工艺条件对缺陷的影响

| 工艺因素 | 影响机理 |
|----------|----------|
| 模温过高 | 延长冷却时间，顶出时铸件强度不足，增加顶凸、顶裂概率 [[S47,S71]] |
| 模温过低 | 加剧合金收缩，型芯包紧力显著升高，易导致粘模顶出困难 [[S47,S71]] |
| 脱模剂喷涂不均 | 摩擦系数大幅升高，脱模阻力增大，引发粘模、顶白、顶裂 [[S36]] |
| 压射压力过大 | 填充致密、收缩量增大，型芯包紧力升高，同时顶出时局部压力超标，诱发顶凸、顶白 [[S60]] |

适当降低顶出速度、采用推板与推杆组合顶出，可有效降低顶出变形发生率 [[S2]]。

## 操作与维护

顶出机构中滑动配合部位需要定期润滑。标准工况下，每完成 50 000 次生产周期或每 8 周更换一次润滑脂；高负荷连续生产的模具需每日对顶出板导向副、顶针孔滑动副补充润滑，并合理设计润滑槽，防止润滑剂溢出污染型腔 [[S38]]。

顶针与导向孔的配合间隙应控制在 **0.005～0.01 mm** 范围内：间隙过大会导致金属液窜入配合面引发卡死，间隙过小则受热膨胀后运动卡滞 [[S21,S60]]。

顶出元件的早期裂纹检查方法包括：日常目视检查顶针工作端面有无网状热疲劳裂纹；批量生产每完成 20 万模次后可采用渗透探伤检测顶针根部、变截面过渡位置的隐性微裂纹，发现裂纹须立即更换，避免顶针断裂掉落型腔 [[S29]]。

顶出元件推荐采用 H13（5% Cr）热作钢制造，热处理硬度控制在 38～42 HRC，表面可进行氮化处理提高耐磨性。当顶针工作端面磨损量超过 0.02 mm、弯曲变形量超过 0.03 mm 时，必须立即更换；常规工况下顶针使用寿命约为 50 万～80 万模次 [[S31]]。

## 与模具其他部件的关联

### 型芯
推杆边缘距离型芯立壁的边距须大于 **3 mm**，以保证型芯边缘强度，防止顶出时崩裂；推杆滑动配合长度取顶杆直径的 2～4 倍 [[S70,S60]]。

### 型腔
顶出元件不允许突出型腔装饰面；作为基准面的顶出元件可略凸出成型表面，使铸件上顶痕为凹入状态，不影响后续基准面加工精度 [[S39]]。

### 滑块（活动型芯）
顶出元件布置须避开滑块活动投影区。若投影区域存在重叠，合模时会发生干涉，此时必须设置**先回程机构**（如三角块先回程机构、弹簧复位机构或电气联锁装置），保证合模时顶出机构先于滑块抽芯机构完成复位 [[S39,S51,S17]]。

### 模架
模架作为安装平台，必须配套推板导柱和推板导套完成顶出板的导向，并通过限位块严格限制最大行程，防止超程损坏部件 [[S41,S14]]。

### 回程机构
回程杆与顶出板固定为一体，合模过程中定模板先接触回程杆，带动整个顶出板完全复位。弹簧仅作为辅助复位元件，不可单独作为唯一复位动力 [[S41,S40]]。

## 标准与规范

- **GB/T 8845‑2017《模具术语》** 明确定义了顶出机构全系列标准术语，包括顶出力、顶出行程、推杆、推管、推板、顶出固定板、顶出板等 [[S53]]。
- **GB/T 4678 系列**（如第8、9、10、11 部分）分别规定了推板、推板导柱、推板导套、推杆等顶出系统零件的结构型式、尺寸、材料与硬度要求。其中第11部分推荐推杆材料为 4Cr5MoSiV1，硬度 45～50 HRC [[S37]]。
- **NADCA 标准** 明确两类通用顶出结构：机械式顶出由开模动作驱动，液压式顶出由压铸机液压油缸驱动；顶出系统通用组成元件包括顶出元件、复位杆、导柱导套、限位钉/圈以及推板、固定板等 [[S20,S58]]。
- 按照我国现行压铸模具系列国家标准，压铸模具**总装图必须标注**顶出机构的顶出形式（中心顶出/顶杆顶出）、顶杆位置示意图及顶出行程等核心参数 [[S8,S19]]。
