关于“定芯”的最核心共识已在行业手册和国家标准中形成：它是一套保证活动型芯在压铸充型过程中保持预设位置的综合性技术体系，并非一个孤立的零件或单一工艺动作。以下内容均基于双语证据展开，不添加未经证实的信息。

## 定义与范畴
在压铸领域，“定芯”（亦称活动型芯定位，英文对应术语为 core locating / core register / core setting）是指将活动型芯在浇注铸件过程中加以固定定位的综合性通用技术体系[[S3,S12,S28]]。该体系同时涵盖工艺操作层面与配套的定位结构/定位元件，行业内将模具中用于固定活动型芯的接触点统称为定位点[[S3]]。定芯属于压铸工艺与模具结构交叉的综合性术词语，既包括“如何做”的操作要求，也包括“靠什么”实现定位的元件与配合关系。

## 工艺功能
定芯的核心功能是在压铸充型阶段提供可靠的型芯位置锁定，主要解决两类力学干扰问题：抵消高速高压金属液对型芯的冲击力，以及型芯所受的金属液浮力[[S11,S25]]。若定芯失效，型芯会产生偏移，即出现“偏芯”（core shift）缺陷[[S12]]，进而导致铸件内孔或内型腔的位置度、同轴度超出图纸公差，出现壁厚不均、尺寸超差甚至报废[[S11,S25]]。

## 活动型芯定位形式
定芯并不限定单一结构，行业实际采用多种安装和定位形式以适应不同重量、不同安装方向的活动型芯[[S11]]。常见形式可归纳如下（来源依据单一出处整合，不额外分类编号）：

- **重力定位**：型芯安装方向与重力一致时，仅依靠自重实现初步固定，见图3‑55(a) [[S11]]
- **弹力定位**：通过开口槽自身的弹张力（图3‑55(b)）、加设弹性圈（图3‑55(c)）、弹性套（图3‑55(d)）或弹力簧（图3‑55(e)）提供定位力[[S11]]
- **弹顶销定位**：在活动型芯内或在模体上设置弹簧驱动的弹顶销，用于质量较大的活动型芯（图3‑55(f)、(g)）[[S11]]
- **定位销定位**：以圆柱销（H7/f7）或圆锥销作为精确定位元件[[S23]]
- **锥面对接定位**：型芯端部采用锥面相互对接，消除圆柱配合间隙，实现超高精度定位[[S23,S25]]

## 配合精度与设计参数
### 定芯配合基本原则
由于压铸模在高温工况下工作，定芯配合必须满足“热态不松、不卡、不进料”的要求：固定型芯受到合金液冲击时不得发生位置偏移；活动型芯热膨胀后仍应保持间隙配合，保证运动自如且金属液不窜入配合缝隙[[S18,S26]]。

### 推荐配合精度
- **固定型芯与安装孔**：整体式优先选用 H7/h6 或 H8/h7[[S18]]；为保证密封，嵌件定位段应具备不小于5 mm的配合长度，精度 H7/h8[[S11]]
- **活动型芯定位孔与型芯轴**：孔取 H7，轴取 e7、e8 或 d8[[S18,S26]]
- **常规定位销**：直径8 mm，配合 H7/f7，间隙范围 0.015～0.043 mm，预定位精度可控制在 0.03 mm以内[[S23]]
- **锥面对接定位**：完全消除圆柱配合间隙，可满足孔距公差 ±0.02 mm 的超高精度定位要求[[S23]]
- **压铸合金配合表**：锌合金、铝合金、铜合金三类压铸合金分别对应不同的型芯与导向孔配合推荐间隙，已收录于《压铸模具简明设计手册》[[S1]]

## 定芯不良的典型失效
### 型芯偏位（孔偏/偏芯）
型芯成型位置与预设位置不符，直接导致孔位偏差、后续机加工后孔壁出现未加工黑皮、装配干涉[[S21,S22]]。
- **常见原因**：型芯定位尺寸不合格、定芯装配间隙超差、压铸机锁模力不足[[S24]]
- **判定标准**：型芯成型位置偏差超出零件图纸规定的位置度公差[[S24]]

### 披锋（多肉）
定位间隙过大使高压金属液窜入配合缝隙，在铸件孔边缘形成多余金属凸起[[S24]]。
- **判定标准**：铸件表面多余金属厚度超过 0.05 mm，超出正常打磨工艺容差范围[[S8,S24]]

### 粘模
定芯失准导致型芯偏移后，高温合金液对型芯局部表面的持续冲刷加剧，大幅提升粘模缺陷发生概率[[S22]]。在镁合金压铸场景下，定芯失准引发的型芯偏移冲刷粘模是造成尺寸公差不合格的主要诱因[[S30]]。

### 型芯断裂
定芯支撑失效使型芯在充型受力状态下偏斜，局部应力集中，在冷热循环作用下快速出现裂纹乃至断裂[[S27]]。

## 相关术语辨析
### 定芯与抽芯
二者术语边界明确：定芯是对型芯进行位置锁定的**静态**定位工艺，在合模及充型全过程中保证型芯无位移[[S7]]；抽芯（Core Pull）特指非固定的活动型芯或滑动型芯**主动位移**、脱离铸件以实现脱模的**动态**过程，两者功能完全独立[[S7,S13,S16]]。

### 定芯与相关组件的配合逻辑
定芯是型芯定位的核心执行环节，并非独立完成，而与芯座、滑块、型芯定位等组件形成协同：芯座提供型芯的预定位固定；滑块由抽芯机构驱动至预设工位后，由定芯结构完成最终位置锁止，保证充型过程中全部型芯的可靠锁定[[S2,S5,S14]]。
