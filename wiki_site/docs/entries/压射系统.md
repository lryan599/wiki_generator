## 概述

压射系统是压铸机中将金属从压射室压入模具，并使金属在固化过程中持续受压的核心功能部件 [[S11]]。作为对压铸件成形质量影响最直接的子系统，其性能直接决定铸造压力、压射速度和增压压力等关键工艺参数 [[S10,S7]]。在结构上，压射系统主体可分为压射室部分（压射室与冲头）、液压缸部分（液压缸、液压缸活塞及活塞杆）和增压缸部分（增压缸、增压缸活塞及活塞杆）三大模块 [[S7,S8]]。

## 定义与分类

依据国家标准 GB/T 20906-2007，压射系统是压铸机内将金属从压射室压入模具并使金属固化期间受压的部件 [[S11]]。其分类可从压铸机类型、控制方式与设备吨位三个维度划分：

| 分类依据 | 类型 | 特征说明 |
| :--- | :--- | :--- |
| 适配压铸机类型 | 热室压铸机压射系统 | 压射室与冲头直接浸入保温炉熔融金属液面以下，金属可自动充料，生产效率高，多用于锌、锡、铅等低熔点合金 [[S11,S5]]。 |
| | 卧式冷室压铸机压射系统 | 压射室水平布置，独立于保温炉，需定量舀入金属液。充型压力损失小，操作便利性好，是目前工业中应用最广泛的类型，适配铝、镁、铜等高熔点合金 [[S11,S3,S51]]。 |
| | 立式冷室压铸机压射系统 | 压射室垂直布置，配有底部反冲头，压射前封闭浇道入口，压射后将余料顶出压射室 [[S11,S19]]。 |
| 控制方式 | 开环控制 | 仅依靠预设时序与阀位执行动作，无实时反馈校正，控制精度较低但设备成本低 [[S14,S58]]。 |
| | 闭环控制 | 配置位移、压力传感器，实时采集参数并动态调整阀口开度，使压射速度、建压时间等精准可控，现代智能压铸机普遍采用 [[S14,S58]]。 |
| 适配设备吨位 | 常规中小吨位 | 适配锁模力 60~4000 t 的压铸机，多用于中小尺寸普通铸件 [[S49]]。 |
| | 大吨位/超大吨位 | 适配锁模力 ≥6000 t 的压铸机，为一体化压铸专用配置，可实现超大型薄壁结构件的高速高压充型 [[S49,S58]]。 |

## 组成结构

压射系统由多个核心组成部分协同工作，实现金属液的注入与加压 [[S19,S55]]。

- **压射室（料筒）**：容纳待压射熔融金属并对其施加压力的圆筒形部件，内壁通常作氮化处理以提高耐磨性能 [[S19,S21]]。
- **压射冲头（柱塞头）**：将熔融金属从压射室射入模腔，并承受金属凝固过程中所产生压力的柱塞，是推送金属液的直接执行部件 [[S19]]。
- **压射液压缸**：为压射机构提供动力的液压缸，内部含独立活塞与活塞杆，将液压能转化为推动冲头运动的机械能 [[S19,S6]]。
- **增压缸**：充型结束后提升压射系统输出压力的专用部件，通过活塞面积差使压射腔内压力短时显著高于系统常规压力，实现高压补缩 [[S19,S8]]。
- **蓄能器**：储能部件，多数压铸机分别配置独立的快压射蓄能器与增压蓄能器，可在极短时间内释放大流量高压液压油，为高速充型和快速增压提供动力 [[S22,S42]]。常用活塞式或气囊式蓄能器，内部仅允许填充氮气 [[S42]]。
- **控制阀组**：包含插装阀、电磁换向阀、比例阀等，集成于油路板，可独立调节慢压射速度、快压射速度、增压建压速度等全部过程参数 [[S55]]。

## 工作过程与阶段

压射系统在一个工作周期内典型经历慢压射、快压射、减速与增压四个过程，在实际压铸机中多整合为三级压射（两级速度加一级增压）[[S8,S30,S46]]。

![压射系统压力-时间特性曲线](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/8e417d83-8dcb-4cd5-8326-ea1b01847654/resource)

图 压射压力‑时间特性曲线，标注慢压射压力 p₁、快压射压力 p₂、系统压力 p₃ 及增压压力稳态值 p₂₀₀ 等时序参数 [[S43]]。

### 慢压射（低速排气）
压射冲头从初始位置运动至熔融金属到达内浇口的动作过程 [[S19]]。此阶段压力较低，工艺目标是避免金属液溅出浇口，并将压室内 45%~70% 的空气有序排出 [[S39,S56]]。

### 快压射（高速充型）
慢压射结束后，快速将熔融金属充填模腔的过程 [[S19]]。压力主要用于克服内浇道阻力，确保在型腔凝固前充满 [[S19]]。

### 减速
设置在快压射接近完成的节点，通过降低冲头运动速度减小型腔充满瞬间的金属液冲击效应，降低水锤造成的飞边缺陷与模具损耗 [[S8]]。

### 增压（保压）
充型结束后，利用增压缸或更高压力的蓄能器瞬间提升压射缸内压力，使之高于系统压力 [[S19]]。增压阶段使金属在高压下凝固，实现压实效果，常规要求建压时间不大于 0.04 s，先进机型可压缩至 7~20 ms [[S12,S19,S17,S39]]。

**三级压射**的主流定义为：第一级将慢压射与封孔合并为慢压射阶段，第二级为快压射充型阶段，第三级为增压阶段。其中前两级为速度控制，第三级为压力控制 [[S30,S46]]。各阶段边界为：慢压射结束于金属液抵达内浇口，快压射结束于型腔完全充满，增压阶段自冲头停止时刻起持续至铸件凝固 [[S46,S17]]。

## 关键参数

压射系统关键参数直接影响铸件成形质量，其典型范围及影响如下表所示。

| 参数 | 含义/定义 | 典型范围或要求 | 对质量的影响 |
| :--- | :--- | :--- | :--- |
| 铸造压力（压射比压） | 压室内熔融金属单位面积承受的压力 | 常规 30~500 MPa [[S30]] | 压力过小导致轮廓不清、致密性不足；压力过大易胀模产生飞边 [[S30]]。 |
| 慢压射速度 | 慢压射阶段冲头移动速度 | 0.05~0.5 m/s（国标 [[S39]]）；薄壁装饰件 0.25~0.35 m/s，耐压件 0.15~0.25 m/s [[S56]] | 过快导致卷气形成气孔，过慢导致冷隔 [[S56]]。 |
| 快压射速度 | 快压射阶段冲头移动速度 | 常规 0.1~7 m/s，部分机型可达 10 m/s [[S56,S19]] | 过慢引起充型不足，过快加剧紊流卷气、冲蚀模具 [[S56]]。 |
| 增压压力 | 增压器提供给冲头的高于系统常压的压力 | 常规 40~200 MPa，部分机型可达 200 MPa 以上 [[S19,S17,S35]] | 不足则无法有效消除缩松缩孔，影响力学性能和气密性 [[S17,S35]]。 |
| 建压时间 | 冲头运动停止点到压力达增压压力稳态值 90% 的时间间隔 | ≤0.04 s，先进机型 7~20 ms [[S39,S29,S12]] | 建压过长则在内浇道凝固前无法传递压力，丧失增压效果 [[S29]]。 |
| 压射行程 | 冲头从回程极限位置运动到压射极限位置的最大位移 | 依据铸件所需浇注重量调整 [[S19]] | 设置不当导致排气不充分或快压射启动位置错误 [[S19,S2]]。 |

## 控制原理

卧式冷室压铸机的压射系统是典型的阀控液压缸系统，由蓄能器提供主要动力，通过电磁换向阀和比例/插装阀调节液压油流量，推动活塞杆实现压射 [[S27]]。

### 完整液压回路工作逻辑
一个标准的压射工作循环包含五个阶段（以某 25 MN 卧式冷室压铸机为例）[[S53,S20]]：
1. **储能阶段**：二通插装阀得电导通，泵站高压油向压射蓄能器和增压蓄能器充油，达设定压力后阀关闭。
2. **慢速压射阶段**：进油侧比例插装阀开至最大，压射蓄能器油液进入压射缸无杆腔；通过出口侧比例插装阀的开度调节有杆腔回油流量，实现低速控制。
3. **快速压射阶段**：到达快压射位置后，出口侧比例插装阀开度迅速增大，调节快压射速度，并在预设位置进行减速。
4. **增压阶段**：充型完成后二通插装阀打开，增压蓄能器油液进入增压缸右腔，推动增压活塞左移，建立起高压。
5. **回锤阶段**：进油侧比例插装阀关闭，电磁换向阀得电，高压油分别进入压射活塞左腔和增压活塞左腔，使各活塞归位。

在整个压射过程中，90% 以上的流量由蓄能器提供，压射蓄能器负责快速充型，增压蓄能器负责高压压实；非压射时段蓄能器完成充液蓄压，大幅降低了液压系统的装机功率需求 [[S24]]。

### 开环与闭环速度控制
- **开环出口节流控制**：进油侧阀口全开，仅通过调节有杆腔出口节流阀开度控制回油流量。无反馈回路，存在启动冲击（建压过程中因油液压缩可产生最高 0.4 m/s 的瞬时前冲）、低速段速度波动大、快速段约 3% 超调等问题 [[S27,S26]]。
- **出口比例插装阀闭环控制**：蓄能器充液完成后，先打开进油侧阀口建立两腔压力，待稳定后仅通过出口比例插装阀调节速度，并搭配前馈加反馈的复合控制逻辑。具备速度反馈校正能力，但受出口阀大压差工况影响，低速调节稳定性仍存在不足 [[S27,S26]]。

## 系统特性

压射系统作为阀控液压缸系统，具有强非线性和时变性，主要来源于：压射活塞与缸壁之间的摩擦随速度和时间变化；冲头与压射室、模具之间的接触摩擦随负载和温度动态改变；液压油的黏滞系数随油温及工况变化；以及整个机构存在较大运动惯性等多变量耦合影响 [[S34]]。这些特性导致常规机理建模难以覆盖全工况动态响应，难以建立完全匹配实际运行状态的高精度数学模型 [[S34,S31]]，因此基于精确模型的控制算法在实际中控制精度较低 [[S31]]。

## 应用与选型

### 与模具的匹配校核
压射系统与压铸模具的匹配通用方法采用 **P‑Q² 特性图**。以压射系统可输出的压力‑流量特性线（ML 线）与模具浇注系统所要求的压力‑流量特性线（DL 线）的交点作为理论工作点，要求该点落在预设工艺窗口内，以确保压射能力与内浇口面积、充型速度要求完全匹配 [[S38,S9]]。

### 与合金的匹配
不同铸造合金对压射系统的性能要求差异显著 [[S57]]：
- **铝合金**：常规压射系统空压射速度配置区间为 3~7 m/s。
- **镁合金**：结晶潜热小、凝固速度快，要求空压射速度达到 6~10 m/s，增压建压时间需控制在 20 ms 以内。
- **高致密度铸件**：需选用最大输出比压高于 100 MPa 的压射系统配置 [[S57]]。

此外，压射系统的选型须校核最大压射力 P₀ ≥ p × A₀（p 为工作压射比压，A₀ 为冲头有效工作面积），并保证压室容积满足单次浇注重量要求，压室填充率推荐控制在 60%~80% [[S45]]。

### 超大吨位压射系统特征
锁模力 ≥6000 t 的超大吨位压铸机常配置专门的压射系统 [[S49]]。部分机型采用双压射结构，可在单次锁模周期内完成两次压射，突破传统单压射系统的填充距离限制，适配新能源汽车大型一体化车身等超大容积铸件 [[S4]]。这些系统通常配备千升级大容积独立蓄能器组，铝合金最大可浇注重量可达 80 kg 以上，高速射流出口速度可超过 10 m/s，并集成全过程排气真空模块，以降低充型卷气缺陷率 [[S4,S33]]。

## 与其它子系统的关系

压铸机通常包含合模系统、压射系统、顶出装置和主液压站等核心模块 [[S37]]。压射系统与这些子系统之间存在严格的时序互锁与协作：主液压站提供动力源，部分机型为避免动作干扰，为压射系统配置独立的液压源；整机控制系统通过总线实现互锁，仅当合模完全锁紧后才会允许启动压射动作，压射完成且增压延时结束后方可执行开模，从而防止胀型溢料等故障 [[S37,S48]]。
