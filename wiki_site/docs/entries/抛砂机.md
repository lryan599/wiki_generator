## 概述

在压铸行业的日常语境中，“抛砂机”是一个广泛使用但容易产生歧义的口语统称。实际生产中，该词通常泛指所有利用磨料冲击完成压铸件表面清理或强化的后处理设备[[S1,S4,S9]]。根据国家标准 GB/T 25370—2020《铸造机械 术语》的正式定义，此类设备应称为“抛喷丸设备”（abrasive blasting equipment），其核心工艺依据动力方式明确区分为**抛丸清理**（利用高速旋转叶轮的离心力抛射磨料）与**喷丸清理**（利用压缩空气喷射磨料）[[S4,S9]]。

> **⚠ 重要概念区分**  
> 压铸行业口语中的“抛砂机”与砂型铸造领域中用于向砂箱抛射型砂以达到紧实目的的 **sandslinger（造型用抛砂机）** 存在本质区别。后者属于造型设备，其功能为填砂紧实，与压铸件表面清理完全无关，不在本文讨论范围内 [[S5,S20]]。

在英文术语体系中，对应关系亦存在混用现象，需根据上下文语境判断 [[S4,S17,S27,S28]]：

| 英文术语 | 动力方式 | 典型磨料 | 应用场景 |
|----------|----------|----------|----------|
| shot blasting machine / wheel blasting machine | 离心叶轮抛射 | 金属丸（钢丸、不锈钢丸等） | 大批量高效清理、表面强化 |
| sand blasting machine | 压缩空气喷射 | 石英砂等非金属磨料 | 表面粗化清理（因粉尘危害已逐步停用） |
| airless blasting machine | 离心叶轮抛射（不依赖压缩空气） | 金属丸 | 等同于抛丸清理机（国内行业使用频次低） |
| abrasive blasting equipment | 离心力或压缩空气均可 | 各类磨料 | 通用术语，涵盖所有抛喷丸设备 |

## 定义

根据 GB/T 25370—2020 的术语定义 [[S4,S9]]：

- **抛喷丸清理**（abrasive blasting / shot blasting）：通过磨料对工件表面强力持续冲击以达到清理或强化预期的工艺。
- **抛丸清理机**（wheel blasting machine）：以高速旋转叶轮所产生的离心力作为加速磨料方法清理工件的抛喷丸设备。
- **喷丸清理机**（air blasting machine）：采用压缩空气作为加速磨料方法清理工件的抛喷丸设备。
- **抛丸器**（wheelblaster）：利用带叶片的旋转叶轮加速抛丸介质的装置，俗称**抛头**。
- **喷丸器**（airblaster）：由喷嘴和连接软管或管子组成，通过压缩空气加速喷丸介质的总成。

| 分类 | 介质类型举例 | 典型用途 |
|------|-------------|----------|
| 干式抛磨设备 | 钢丸、不锈钢丸、铝丸、铸铁丸 | 表面清理、强化 |
| 湿式抛磨设备 | 水砂混合介质 | 特殊清洁要求 |
| 普通砂类磨料 | 石英砂、河沙（已被逐步淘汰） | 表面粗化 |
| 金属丸类磨料 | 不锈钢丸、锌丸、钢丸、铝丝丸 | 高效清理、表面强化 |
| 非金属特种磨料 | 刚玉、玻璃珠、金刚砂粒 | 精细清理、特殊表面处理 |

## 结构与工作原理

### 核心组成部件

根据 GB/T 25370—2020 及相关技术文献 [[S2,S4,S19]]，压铸用抛砂机主要由以下系统组成：

1. **抛头（抛丸器）**：由叶轮、分丸轮、定向套构成，用于将磨料加速抛出。叶轮体上装有叶片，与分丸轮共同装在主轴上旋转，弹丸经进丸管流入，由分丸轮赋予初速度，再经定向套窗口飞出至叶片，被进一步加速后抛射至工件表面。
2. **喷枪（喷丸器）**：由喷嘴、喷丸管组成，在压缩空气驱动下将磨料喷射到工件指定区域。喷嘴孔径一般不小于弹丸直径的 3.5～4 倍。
3. **磨料循环系统**：由提升机、螺旋输送器、丸砂分离器、储料斗等组成，实现磨料的回收、筛分、循环利用。
4. **除尘系统**：多采用干法两级除尘结构，将作业过程中产生的粉尘过滤后达标排放。
5. **工件承载装置**：根据结构形式不同，分别采用滚筒、履带、转台、吊钩等结构，带动工件运动以实现全表面均匀清理。

### 两类动力模式的工作差异

| 对比项 | 离心抛射模式（抛丸） | 压缩空气喷射模式（喷丸） |
|--------|---------------------|--------------------------|
| 驱动方式 | 高速旋转叶轮 | 压缩空气 |
| 磨料速度 | 60～80 m/s | 50～70 m/s |
| 典型工作压力 | — | 4.90～5.88 MPa（压缩空气） |
| 清理效率 | 高 | 相对较低 |
| 单位能耗 | 低 | 较高 |
| 适用场景 | 大批量工件外表面快速清理 | 工件内腔、死角补充清理，灵活性更强 |

以上数据依据 [[S14,S17,S18]]。

> **抛喷丸联合清理**：采用抛丸和喷丸相结合的方式，即先进行抛丸清理，而后通过喷丸方式对工件内腔或部分表面再进行补充清理。该方式兼具抛丸的高效率和喷丸的灵活性优势 [[S9,S14]]。

## 在压铸工艺中的角色

### 核心应用场景

抛砂机（抛丸清理机）在压铸全流程中主要承担以下功能 [[S1,S3,S8,S11]]：

1. **去除飞边毛刺**：清除压铸件分型面位置的细小飞翅与残留毛刺；
2. **表面氧化皮清理**：去除铸件表面附着的氧化皮及压铸脱模剂残留杂质；
3. **涂装前表面预处理**：为后续涂装/电镀工序提供可控粗糙度的锚定表面，提升涂层附着力；
4. **模具维护**：清除模具型腔表面附着的残留铝积瘤。

### 在压铸后处理流程中的衔接关系

压铸用抛砂工序设置在以下流程节点 [[S6,S8,S11]]：

```
切边/去浇口（粗去除大余量飞边）
    ↓
抛砂/抛丸清理（去除微量残留飞边及表面杂质）
    ↓
可选后续工序：
    ├── 抛丸强化（进一步提升疲劳性能）
    ├── 干冰清洗（清理盲孔内残留磨料）
    ├── 振动研磨/抛光（表面终处理）
    └── 涂装/电镀（需前处理锚定表面）
```

对于存在高精度封闭盲孔清理需求的工件，可在抛丸工序之后配套**干冰清洗**完成盲孔内部的磨料残留清理，相比单一的振动研磨工序效率显著提升 [[S6]]。

## 核心工艺参数

根据 GB/T 25370—2020 及相关文献，压铸用抛砂处理需控制的工艺参数包括 [[S2,S4,S17,S18,S21,S25]]：

| 参数名称 | 定义 | 说明 |
|----------|------|------|
| 抛射速度 | 弹丸脱离抛丸器叶片时的初始速度 | 通常 60～80 m/s（抛丸） |
| 喷射速度 | 磨料从喷嘴喷出时的速度 | 通常 50～70 m/s（喷丸） |
| 抛丸量 / 喷丸量 | 单位时间内抛射/喷射出的磨料量 | 通常以 kg/min 为单位计量 |
| 工作压力 | 喷丸时送入喷嘴的压缩空气压力 | 根据铸件材质和弹丸种类选择 |
| 喷射距离 | 喷嘴与工件被喷表面的距离 | 气压 350～550 kPa 时取 350～500 mm 为宜 |
| 喷射角度 | 磨料束中心线与被喷工件表面的夹角 | 常取 75°～90°，难清理表面取较大角度 |
| 覆盖率 | 受到磨料冲击的表面占要求处理表面积的百分比 | 100% 覆盖为疲劳性能优化的临界值 |
| 投射密度 | 受到磨料冲击的表面内，单位面积所投射的磨料量 | — |
| 循环时间 | 根据工件复杂度和质量要求设定 | 需避免过抛或欠抛 |
| 介质种类与粒度 | 弹丸材料、形状、尺寸规格 | 详见下文“选型与应用场景” |

> **注意**：由于不同压铸合金、铸件尺寸及质量要求差异较大，具体工艺参数数值需结合实际验证后确定。上表中的速度和工作压力范围为通用铸造数据，针对薄壁压铸件需降低冲击力以避免变形风险。

## 对压铸件质量的影响

### 正向影响

1. **引入残余压应力，提升疲劳性能**：100% 完全覆盖的抛丸处理可在压铸件表层引入均匀分布的残余压应力，抑制表面裂纹扩展，有效提升 Al-7Si 等铝合金压铸件的疲劳性能 [[S21,S27]]。
2. **改善涂装/电镀结合力**：抛丸作用下可将压铸件表面调整为连续可控的微观粗糙形貌，大幅提升后续涂装层、电镀层与基体的结合强度 [[S27]]。

### 负面风险

1. **过度抛丸导致表面损伤**：当表面抛射覆盖度达到 1000%（即超量重复打击）时，工件表面会出现层状剥落损伤，反而降低材料疲劳性能 [[S21]]。
2. **薄壁件变形风险**：薄壁结构压铸件在高冲击力抛丸作用下存在形变失稳的风险，需严格控制抛丸线速度与总处理时长 [[S21]]。

## 局限性及缺陷风险

| 风险类型 | 说明 | 应对措施 |
|----------|------|----------|
| 粉尘污染 | 抛丸处理过程产生大量金属粉尘 | 设备须配套高性能除尘系统 |
| 镁合金爆炸风险 | 镁合金粉尘在抛丸作业中存在燃爆危险 | 镁合金抛丸工位须额外配套防爆防护系统，满足 GB 20906 安全规范要求 [[S7]] |
| 磨料嵌入表面 | 硬质磨料在高冲击力作用下有概率嵌入较软的铝、锌压铸件表层 | 后续机械精加工时会加剧刀具非正常磨损 [[S21,S27]] |
| 微小孔洞堵塞 | 工件表面微小盲孔、高精度计量孔在抛丸时容易被磨料完全堵塞 | 抛丸前需提前做好遮蔽防护 [[S23]] |
| 起皮缺陷 | 抛丸处理可能引起压铸件表面起皮缺陷 | 需合理控制工艺参数 [[S23]] |

## 选型与应用场景

### 基于工件特征的机型选择

| 工件特征 | 推荐机型 | 原因 |
|----------|----------|------|
| 中小型锌、铝合金压铸件，大批量 | 履带式（滚筒式）抛砂机 | 单次装料即可完成全表面覆盖清理，属于压铸行业通用主流机型 [[S25]] |
| 大尺寸、易磕碰异形结构压铸件 | 吊钩式抛砂机 | 单独吊挂旋转抛射，避免工件间相互碰撞造成外观损伤 [[S25]] |

### 基于合金类型的磨料选型

| 压铸合金 | 推荐磨料 | 规格/配比 | 效果与注意事项 |
|----------|----------|-----------|---------------|
| 铝合金 | 不锈钢丸 | φ0.14～0.3 mm（常规）；φ0.8 (50%) + φ0.6 (20%) + φ0.4 (20%)（高涂装要求） | 表面光亮、纹理细致、抗腐蚀能力强、涂装附着力好 [[S1,S25]] |
| 锌合金 | 锌丸 | — | 安全可靠、经济合理；抛后配合钝化处理可获得优异防锈性能 [[S10,S25]] |
| 镁合金 | 不锈钢丸 + 铝丝丸（3:7 混合） | — | 抛丸完成后需尽快开展表面涂装，避免表面氧化腐蚀 [[S10]] |

以上选型依据 [[S1,S10,S25]]。

## 与其他工艺/设备的关系

| 对比工艺 | 与抛丸/抛砂的关系 | 说明 |
|----------|-------------------|------|
| 切边/去浇口 | 前后衔接 | 抛砂设置于切边工序之后，去除微量残留飞边；大余量飞边需先经冲压切边粗去除 [[S6]] |
| 振动研磨 | 可配合或替代 | 振动研磨利用研磨石摩擦去毛刺；抛丸效率更高且兼具强化功能，但可能需后续干冰清洗清理盲孔残留磨料 [[S8]] |
| 干冰清洗 | 补充配合 | 针对封闭盲孔，在抛丸后配套干冰清洗清除磨料残留 [[S6]] |
| 喷砂清理 | 功能相近但动力不同 | 喷砂以压缩空气驱动石英砂，主要用于表面粗化，因粉尘危害已逐步被抛丸替代 [[S1,S27,S28]] |
| 模具保养 | 同一设备可拓展 | 抛砂设备亦可用于模具型腔表面残铝积瘤清理 [[S11]] |

## 与造型用抛砂机的明确区分

为澄清行业术语的常见混淆点，下表对比两种名称相似但功能完全不同的设备 [[S5,S20]]：

| 对比维度 | **压铸用“抛砂机”（口语）** | **砂型铸造造型用抛砂机（sandslinger）** |
|----------|---------------------------|---------------------------------------|
| 规范术语 | 抛丸清理机 / 抛喷丸设备 | 抛砂机（sandslinger） |
| 所属领域 | 压铸件后处理 | 砂型铸造造型 |
| 核心功能 | 利用磨料冲击清理和强化铸件表面 | 向砂箱内抛射型砂实现砂型紧实 |
| 工作介质 | 金属丸、非金属磨料（钢丸、砂粒等） | 型砂 |
| 动力方式 | 离心叶轮或压缩空气 | 高速旋转叶片离心抛射 |
| 目标对象 | 已成形压铸件 | 砂箱/芯盒内的型砂 |
