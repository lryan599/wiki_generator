## 概述

金属型重力和低压铸造模具的热平衡系统，是指在重力铸造与低压铸造连续生产条件下，使金属型模具从注入的高温合金熔液中吸收的总热量与其向外界环境释放的总热量达到动态相等的一整套温度调控体系[[S26]]。当进入模具的循环热量 $\\Sigma Q_{\\text{得}}$（包括金属液过热显热、结晶潜热和固态显热）与模具向环境散失的热量 $\\Sigma Q_{\\text{失}}$ 趋于一致时，系统即处于**热平衡状态**[[S68]]。该系统的设计目标是：在连续生产过程中，充型前维持规定的型腔工作温度，以避免充不足、冷隔等充型缺陷；充型后则在凝固阶段维持合理的模具温度梯度，支撑铸件顺序凝固，消除缩松类缺陷；在保证铸件质量的前提下压缩生产节拍、提高生产效率[[S59]]。

## 工作原理与分区逻辑

热平衡系统的核心是精确调控模具的温度场，使其既满足充型对模温的需求，又满足凝固对温度梯度的要求。系统采用**两级分区控制**思路[[S65]]：

- **第一级——模架低温区**：在模块与模架之间预留1～2 mm隔热间隙，并尽可能减小接触面积以阻断热量向设备传递。对模架上的关键部位（如油缸支架）设置常开冷却水路，防止模架高温膨胀变形。
- **第二级——型腔模块工艺温度区**：根据铸件不同部位的凝固需求，差异化分配温度梯度，形成从铸件远端向浇口方向（或升液口）逐步递增的温度分布，保障顺序凝固[[S65]]。

在顺序凝固原则下，铸件薄壁远端温度最高，铸件本体次之，内浇口或冒口对应区域温度最低，而冒口部位应是全模工作温度最高的区域[[S18]]。理想状态下，模具型腔最远端应比浇口附近更热[[S48]]。

## 系统构成与结构

热平衡系统由**加热系统、冷却系统、散热与保温系统**共同组成[[S59]]。

### 冷却回路类型
冷却介质通常设计为以下三种回路形式[[S27,S37]]：

| 回路类型 | 介质 | 工作压力/参数 | 特点 |
|---------|------|-------------|------|
| 风冷开放回路 | 压缩空气 | 0.21～0.28 kg/mm² | 结构简单、无泄漏风险，效率较低 |
| 水冷闭合回路 | 冷却水 | 由冷却塔、模具水道组成闭环 | 冷却强度为风冷的几十至上百倍，效率最高 |
| 水气复合冷回路 | 水雾+压缩空气 | 混合介质 | 冷却效率较纯风冷提升约30% |

低压铸造模具中需要强制冷却的典型部位包括：浇口衬套、厚大热节、细长型芯以及流经大量金属液的位置[[S27]]。

### 高密度冷却通道（随形通道）
借助选区激光熔化（SLM）等增材制造技术，可加工出完全贴合型腔轮廓的随形冷却通道。典型构型包括并行多通道型、单通道迂回型、单通道螺旋型和横纹型变截面通道[[S38]]。此类通道能使型腔表面温度分布更均匀，例如并行多通道型随形水路的凸模温差可控制在0.76 ℃以内[[S61]]，单通道螺旋型水路也可实现极小的温度梯度[[S43]]，从而大幅降低热应力。

### 传感器与控制单元
模具关键位置（型腔近壁、热节区、型芯、模架等）布置多路热电偶，配合流量、压力传感器对介质状态进行监测[[S49]]。控制单元可采用PLC或数字温控器，对各冷却回路进行分区独立控制（介质通入时间、循环参数等）[[S49]]；部分先进系统进一步集成CAE-AI预测闭环控制模块，实现模具温度场的实时在线优化[[S14]]。

## 工艺参数与操作条件

### 模具预热温度
金属型在浇注前必须充分预热，以减少热冲击、避免表面微裂纹[[S30]]。不同合金的金属型常规预热温度范围如下[[S35,S53]]：

| 铸造合金 | 预热温度（℃） |
|---------|--------------|
| 铸铁 | 250～300 |
| 铸钢、铝合金 | 150～300 |
| 镁合金 | 150～300 |
| 铜合金 | 120～200 |

对铝合金薄壁复杂铸件，实际生产中常将预热温度选取在180～220 ℃区间，以降低型腔表面温度梯度[[S69]]。某城轨车辆铝合金牵引梁的金属型重力铸造工艺中，实测金属型预热温度取250 ℃，浇铸温度730 ℃，换热系数2000 W/(m²·K)[[S3]]。

### 冷却介质运行参数
不同冷却介质推荐的运行参数范围如下[[S52]]：

| 介质类型 | 工作压力 | 推荐流速 / 流量 | 入口温度 | 说明 |
|---------|---------|---------------|---------|------|
| 常压冷却水 | 0.2～0.6 MPa | 2～3 m/s（φ14 mm水路最大流量60 L/min） | 20～35 ℃ | 常规冷却通道 |
| 高压点冷水 | 约1 MPa（通径0.85～2.5 mm点冷管）<br>约0.6 MPa（3 mm以上常规通道） | ─ | ─ | 用于型芯、局部热节快速冷却 |
| 热油（加热/冷却） | 0.1～0.3 MPa | ─ | 80～200 ℃可调 | 兼具预热与恒温功能 |
| 强制风冷 | 0.5～0.7 MPa（工业气源） | ─ | ─ | 常作开模后辅助冷却 |

### 模具工作温度窗口
模具温度过低会导致冷隔、流纹、欠铸；温度过高则易产生粘模、表面气泡、缩松和收缩变形。稳定的合格品生产通常要求模具温度控制在**200～300 ℃**的窄区间内[[S36]]。通过在模具内设置热电偶及联动的风冷/水冷回路，可将开模前后模具温差控制在50 ℃以内，模具全域工作温差控制在100 ℃以内[[S45]]。

## 设计方法与分区管理

模具热平衡系统的冷却水路设计，通常采用以下四类方法[[S21]]：

1. **经验设计法**：依托工程师经验排布水道，常见于早期或简单模具，但易导致冷却不均匀。
2. **基础热平衡计算法**：根据铸件单次循环释放的总热量，减去模具自然散热量，反推出需要由冷却系统带走的热量及所需通道长度与通流面积[[S46,S16]]。
3. **有限元分析法（FEA）**：利用ProCAST、ANSYS、MAGMA等软件对模具温度场进行多循环迭代模拟，精确定位冷却通道位置及热电偶安装点位[[S33,S21]]。
4. 无论是哪种方法，设计时均需同步考虑**生产节拍、铸件热节体积、模具材料热物性、冷却介质换热系数**等关键边界条件，以使仿真结果与实际工况匹配[[S21,S46]]。

### 冷却通道几何设计准则
冷却通道的几何参数需遵循以下安全与工艺准则[[S54,S58]]：

- **通道直径**：常规取值 **6～12 mm**，最小通径不低于6 mm；使用热油冷却时直径需额外增加1/16英寸以补偿粘度影响[[S54]]。
- **型腔壁距**：冷却通道外壁与型腔表面的最小距离不得小于 **1.5～2倍** 通道直径[[S58]]；国内行业规范进一步要求所有水冷通道距型腔表面至少 **19 mm**（约3/4英寸），以防止热冲击开裂及水渗风险[[S54]]。
- **通道间距**：相邻冷却通道的中心距常规取 **3～5倍** 通道直径，保证冷却覆盖均匀性[[S54]]。
- **水路流量**：每条冷却回路的推荐流量为 **5～10加仑/分钟**，以避免结垢导致换热效率下降超过40%[[S54]]。

对于局部型芯或滑块等高热负荷区域，常采用**高压点冷**方式，通过通径0.85～2.5 mm的点冷管在约1 MPa压力下进行定点冷却，迅速消除局部过热[[S49]]。

## 工艺作用与效果

通过合理的热平衡系统设计与控制，能够从多个维度提升铸造生产质量与效率：

- **降低缺陷率**：某直列四缸发动机缸体量产案例显示，热平衡优化后孔洞缺陷发生率从13.5%降至3.5%，冷隔缺陷从2.7%降至0.3%，整体良品率由82.3%提升至95%以上[[S45]]。低压铝合金缸体的机加一次泄漏检测合格率则从不足80%提升至约97%[[S45]]。
- **提高生产效率**：某低压铸件在热平衡系统升级后，生产节拍从508 s缩短至365 s，整体效率提升约28%[[S5]]。
- **延长模具寿命**：抑制周期性冷热交变引起的热应力波动，可延缓热疲劳裂纹的产生。合理的热平衡控制条件下，复杂壳体模具的模拟服役寿命可达10万模次[[S34]]。
- **改善微观组织**：在受控热平衡条件下凝固的金属型铸件，可获得细化的枝晶组织，进一步提升力学性能[[S25]]。

## 应用对象

金属型重力/低压铸造热平衡系统已在以下典型铸件的大批量生产线上实现规模化应用[[S41,S40,S62,S17]]：

- 铝合金车轮（汽车与摩托车轮毂）
- 发动机铝合金缸盖、缸体
- 铝合金换热片（散热器类薄壁复杂件）
- 各类承压、薄壁复杂铝合金铸件

其中，采用热平衡系统的铝合金缸盖低压铸造工艺出品率可达85%以上，远高于普通重力金属型铸造约50%的水平[[S62]]。对于局部深腔、细小型芯或孤立热节，系统常配置扩散器、热管、导热销等热交换组件进行精准热量导出[[S9,S29]]。

## 与其他系统的协同

热平衡系统与周边工艺控制系统紧密配合[[S40,S11]]：

- **温度监测与加热系统集成**：通过模具内布设的热电偶实时采集温度信号，反馈至PLC，同步控制模具加热回路启停功率及冷却回路通断时长/介质流量。
- **与低压铸造压力控制系统联动**：在低压铸造中，冷却通道的通断时序须与充型加压曲线相匹配，确保凝固阶段铸件从远端向升液口建立定向凝固温度梯度，压力释放后升液管内未凝金属回落，避免补缩中断。

## 局限性及典型问题

### 冷却通道结垢
冷却水温度超过50 ℃时，水中溶解的碳酸氢钙会分解并析出不溶性碳酸钙沉淀，附着于通道内壁形成结垢[[S15]]。仅0.005英寸（约0.13 mm）厚的水垢层即可使系统换热效率降低40%[[S54]]。生产实践表明，未定期清理的螺旋水冷型芯，其铸件缩松缺陷率会从0.2%逐渐回升至2%，采用专用模具水路清洗机每半个月清洗一次，可将缩松率稳定维持在0.2%的水平[[S24]]。

### 热应力与疲劳裂纹
模具在每次循环中承受剧烈的温度交变：充型时型腔表面产生压应力，开模冷却后转变为拉应力。交变应力反复累积，超过材料疲劳极限后即会萌生热疲劳裂纹[[S34,S60]]。裂纹表现为表面网状龟裂，或在结构突变部位（如中毂、辐条根部）出现独立的大深度裂纹[[S6]]。硬度42 HRC的模具在10000次循环后便可观测到深度0.37 mm、宽度0.76 mm的典型微裂纹[[S31]]。铝合金重力铸造轮毂模具若未进行消应力处理，约5000件后即出现疲劳开裂；在3000～4000件时安排一次去应力回火可显著延长模具寿命[[S39]]。

### 温度控制延迟
低压铸造实验表明，冷却水到达冷却点的最大延迟可达30 s；水接触高温模具时瞬时沸腾气化会使换热系数短时间内大幅上升，破坏预先设计的热平衡时序[[S57]]。模具温度偏离设定区间（过低或过高），将直接引发冷隔、流纹、欠铸或粘模、缩松等一系列缺陷[[S36]]。

### 维护成本
铸造模具的全生命周期成本中，模具材料购置仅占约10%，其余90%均用于使用过程中的定期维护、冷却通道清理及裂纹修复等环节，热平衡相关维护是模具运维成本的最大组成部分之一[[S60]]。

## 分类与变体

该热平衡系统可根据**加热/冷却方式**、**模具适配类型**及**控制策略**进行分类[[S26,S27,S13,S65,S49]]：

| 分类依据 | 变体类型 | 特征 |
|---------|---------|------|
| **加热/冷却方式** | 纯自然散热+保温结构型 | 依靠模具自然对流辐射散热，配合实体保温镶件、空气隔热层实现温控，结构简单，适合小批量简单铸件 |
| | 风冷主动温控型 | 适配低散热需求，广泛应用于部分欧美低压铸造机，无漏液风险 |
| | 水冷主动温控型 | 冷却效率高、换热均匀，适配大产能复杂铸件生产线 |
| | 复合加热/冷却型 | 集成电加热/火焰加热与多组独立冷回路，满足宽工况高精度铸件生产 |
| **模具适配类型** | 金属型重力铸造款 | 侧重整体均温控制，适配重力充型温度分布 |
| | 低压铸造款 | 针对升液管、浇口衬套等易积热位置设置专用回路，保证顺序凝固方向 |
| | 带复杂型芯/多滑块款 | 额外配置针对细长型芯、活动滑块的定点冷却（点冷）通道 |
| **控制策略** | 被动非时间控制型 | 保温/散热结构固化于模具本体，全浇注周期持续生效，无需外部干预，稳定性最高 |
| | 时间程序控制型 | 预先设定各回路介质通断时序，匹配充型、凝固、开模等工序 |
| | 全闭环实时智能控制型 | 基于实时测温数据动态调节各回路介质流量与通断状态，实现自适应热平衡 |
