## 概述与定义

型芯下沉（英文对应 core sag，别名芯子下沉）是铸造生产中形状及重量差错类缺陷的一种。依据 GB/T 5611—2017《铸造术语》的定义：由于芯砂强度低或芯骨软，不足以支撑自重，使型芯高度降低、下部变大或下弯变形而造成的铸件变形缺陷[[S16,S13]]。

在压铸场景下，该缺陷表现为型芯在金属液充型及凝固过程中沿重力方向向下偏移、下坠，最终导致铸件对应位置的尺寸超差[[S6,S17]]。国际标准中将其列为 9.7.9 类形状尺寸偏差缺陷[[S6]]。

型芯下沉主要发生于型芯长径比大、支撑点位少的压铸件结构，其失效方向明确为沿型芯轴向向下，与侧向偏移为主的型芯偏移属于不同的独立缺陷类型[[S32,S13]]。

## 形成机制与原因

### 机械原因

1. **定位与锁紧结构失效**：定位销与销套配合松动，组合型芯安装偏移，无法约束型芯垂直方向位移；型芯与芯座配合间隙过大，缺少有效轴向限位结构[[S32,S1]]。
2. **芯骨刚性不足**：型芯芯骨刚性不足、支撑数量不够，无法抵消型芯自身重力作用而发生弯曲下沉[[S10]]。
3. **芯撑配置不当**：芯撑配置数量不足、支撑点位未精准定位，型芯受力点不足，难以承担载荷[[S10]]。

### 工艺原因

1. **冲击载荷过大**：压射比压过高、金属液冲击速度过大，或单侧浇口设计导致金属液持续对型芯底部施加载荷，推动型芯向下位移[[S32,S2]]。
2. **浇注温度过高**：金属液温度过高使型芯在高温下出现热软化、热变形，刚性大幅下降，不足以抵抗自身重力与金属液冲击[[S32]]。
3. **浇注过猛**：瞬间大流量金属液的冲击载荷超出型芯固定结构的承载上限，引发下沉移位[[S32]]。

### 模具设计原因

1. **长径比过大**：细长型芯悬臂段过长，刚性不足，在金属液冲击下易发生下弯下沉[[S25,S7]]。
2. **轴向支撑结构缺失**：未设置台阶定位面、支撑垫板、限位嵌件等结构，无法抵消下沉趋势[[S7,S19]]。
3. **热膨胀差异**：型芯与模具钢热膨胀系数差值过大，高温充型过程中型芯受热伸长与芯座发生干涉，额外应力引发下沉[[S25]]。

## 对铸件的影响

型芯下沉对压铸件的直接危害包括[[S32,S11,S2]]：

| 影响类别 | 具体表现 |
|----------|----------|
| 尺寸超差 | 铸件对应内孔深度、壁厚尺寸偏差超出公差范围，导致尺寸不合格报废 |
| 形位公差失效 | 内孔位置偏移，同轴度、平行度、对称度等精度参数不满足装配要求 |
| 断芯堵孔 | 极端情况下型芯根部断裂残留铸件内部，形成堵孔、断芯缺陷 |
| 轮廓精度失效 | 铸件对应部位出现不规则缺肉或多肉，表面轮廓精度丧失 |

## 高发场景

型芯下沉广泛出现于铝合金、镁合金、锌合金等主流压铸合金的生产过程中，高发于带有**深孔、细长孔、悬臂式异形弯孔**结构的压铸件。例如：带内置长油道的发动机部件、小型薄壁弯管类压铸件等。这类结构的型芯普遍具有长径比大、支撑点位少的特点，自身抗变形能力弱[[S35,S21,S23]]。

## 工艺参数与型芯下沉风险

高压压铸（HPDC）场景下，以下参数区间易触发型芯下沉[[S12,S14,S30]]：

| 工艺参数 | 危险阈值范围 | 说明 |
|----------|-------------|------|
| 内浇口填充速度 | 高于 60 m/s | 金属流对型芯的瞬时冲击载荷大幅提升，远超常规固定结构承载阈值 |
| 压铸比压 | 超过 60 MPa | 型芯承受的轴向压力显著增大 |
| 铝合金浇注温度 | 高于 720°C | 型芯长时间受高温熔融金属作用，易产生软化蠕变 |
| 模具预热温度 | 低于 150°C | 型芯受冷热循环交变应力作用，定位结构松动概率显著提升 |

## 预防与控制措施

### 模具结构优化

| 优化方案 | 作用原理 |
|----------|----------|
| 型芯根部设置轴向限位圆柱台阶[[S7,S27,S33]] | 承担承压作用，抵消金属液轴向向下的压力 |
| 无台阶型芯增加底部支撑垫板[[S7,S10]] | 分散受力，防止下沉 |
| 型芯安装端嵌入模板内部增加嵌合深度[[S7]] | 增强稳定性 |
| 长悬臂型芯受力下方增设辅助支撑芯撑[[S7]] | 分担载荷，降低单点受力 |
| 铆死锁止结构固定型芯顶部[[S8]] | 完全限制型芯向下的位移自由度 |

### 侧向锁紧机构优化

采用整体式锁紧块结构替代分散锁紧螺钉，锁紧块斜角设置为比对应斜销斜角大 3°~5°，确保压射过程中型芯滑块完全锁死无下沉位移；增加侧部辅助楔紧块，提升型芯抗冲击刚性[[S5]]。

### 压铸工艺调整

| 调整措施 | 目标区间 |
|----------|----------|
| 控制快压射开启时机，避免金属液直冲型芯底部[[S15,S30]] | — |
| 降低填充速度[[S15]] | 30～50 m/s |
| 降低铝合金浇注温度[[S15,S30]] | 650～700°C |
| 配置模具温控系统，维持模温波动[[S30]] | 小于 ±30°C |

## 检测与识别

生产现场识别型芯下沉的实用检测方案包括[[S3,S24,S9,S20]]：

| 检测方法 | 说明 |
|----------|------|
| **离线尺寸检测** | 使用游标卡尺、三坐标测量仪对铸件型芯成型区域的壁厚、深度尺寸进行抽检，精度可达 0.01 mm，适用于批量终检环节 |
| **无损检测（X光）** | 工业 X 光断层扫描识别铸件内部隐蔽型芯偏移导致的壁厚不均，可检出肉眼不可见的隐蔽下沉缺陷 |
| **CAE 预模拟** | 采用 Moldflow 等软件开展型芯偏移预计算，提前预判型芯下沉量，避免试模后才发现缺陷 |
| **在线过程监控** | 在压铸机压射端配置位移传感器，实时监测型芯受力后的微位移数据，预警异常下沉风险 |

数值模拟表明，典型工况下型芯最大下沉变形可达 0.73 mm，此类热图可直观展示不同区域的下沉量梯度分布规律[[S26]]。

## 与相关缺陷的区分

型芯下沉属于型芯位移类缺陷的一种，与其他相关缺陷的区分如下[[S22,S13]]：

| 缺陷类型 | 英文术语 | 失效方向 | 典型特征 |
|----------|----------|----------|----------|
| **型芯下沉** | core sag | 沿重力方向轴向向下 | 型芯高度降低、下部变大或下弯，铸件对应部位尺寸偏差 |
| **型芯偏移** | core shift | 水平侧向偏心移动 | 受侧向不对称充型压力作用，铸件两侧壁厚不均 |
| **型芯弯曲** | core bending | 本体发生弹性/塑性弯折 | 型芯本体变形导致位置偏差 |
| **型芯后退** | core retraction | 向安装固定侧轴向回缩 | 型芯沿轴向向后退缩 |
| **型芯断裂** | core breakage | 本体完全断裂失效 | 型芯折断，残留在铸件内部 |

GB/T 29092—2012《镁及镁合金压铸缺陷术语》明确定义了型芯移位术语，为型芯位移类缺陷的判定提供了官方依据[[S18]]。

## 验收标准依据

国内现行压铸相关标准中，GB/T 15114 对应系列标准、GB/T 6414 铸件尺寸公差标准、GB/T 13821 压铸件表面质量标准中规定的压铸件同轴度、对称度、位置公差体系，是型芯下沉缺陷所致铸件尺寸偏差的验收判定依据[[S28,S34]]。
