## 定义与矿物学特征

钙基膨润土是指表面和内部晶层可交换吸附物以钙离子（Ca²⁺）为主的膨润土，属于以蒙脱石为主要矿物组成的层状硅酸盐粘土类矿物[[S49,S11]]。在铸造行业分类标准中，当钙离子占膨润土总阳离子交换容量的50%或以上时，即可判定为钙基膨润土，行业代号为**PCa**[[S9,S26]]。

钙基膨润土是铸造无机粘结剂体系中膨润土的重要子类。根据国家标准GB/T 5611—2017《铸造术语》的定义，钙基膨润土为“土粒表面和内部晶层吸附物主要是钙离子的膨润土”[[S28,S37]]。在国内标准体系中，膨润土作为粘结剂的分类如下图所示：

![铸造用粘结剂分类图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/cf968c36-43de-46d7-a556-01f64e445306/resource)

蒙脱石晶体结构为2:1型三层结构，由两层硅氧四面体夹一层铝氧八面体构成。相邻单位晶层之间依靠范德华引力连接，水分子及其他极性分子易进入晶层之间，引起晶格沿c轴方向膨胀[[S42,S26]]。钙基膨润土原矿在透射电镜下可清晰观测到典型的六方堆叠晶片结构，如图：

## 化学组成与基本性质

### 核心化学与物理参数

钙基膨润土的核心粘结成分是蒙脱石，其典型化学通式为Al₂O₃·4SiO₂·nH₂O[[S42,S4]]。商用铸造级钙基膨润土的关键组成参数如表中所示：

| 指标 | 数值/描述 | 数据来源 |
|------|-----------|----------|
| 蒙脱石含量（优质粘结级） | ≥70% | [[S38]] |
| 蒙脱石含量（高等级商用产品） | 可超过80% | [[S43]] |
| 总阳离子交换容量（CEC） | 80~150 mg当量/100g土 | [[S38]] |
| 层间主要可交换阳离子 | Ca²⁺ | [[S30,S26]] |

作为对比，普通高岭土的阳离子交换容量仅为3~15 mg当量/100g土，远低于膨润土[[S38]]。

### 热稳定性

钙基膨润土的热行为具有阶段性特征[[S42,S4]]：

| 温度区间 | 变化过程 |
|----------|----------|
| 100~200℃ | 失去大部分层间吸附水 |
| 500~700℃ | 完全失去晶格结构水 |
| 800℃以上 | 蒙脱石晶格彻底破坏，转变为无定形物质 |
| 1100~1200℃ | 开始析出高温新相 |
| 1250~1300℃ | 熔融温度区间 |

### 吸水膨胀与粘结特性

钙基膨润土属于**非膨胀型膨润土**，遇水体积无明显膨胀，吸水与脱水速率快[[S48,S43]]。其与钠基膨润土在水分吸附行为上的关键差异在于：蒙脱石颗粒表面的非液态水向普通液态水的过渡层厚度，钙蒙脱石仅为钠蒙脱石的1/7~1/6[[S42,S17]]。

钙基膨润土粘结性能的工艺特征表现为[[S43,S5,S31]]：
- **湿态粘结强度高**——可提供优良的湿压强度
- **干强度、热湿拉强度低**——型砂在浇注前干燥后强度下降明显
- **溃散性好**——高温残留强度低，落砂时不易形成热结块

## 钙基与钠基膨润土的性能对比

钙基膨润土与钠基膨润土在铸造工艺性能上存在显著差异，适用场景各有侧重[[S43,S21,S17]]：

| 对比项目 | 钙基膨润土 | 钠基膨润土 |
|----------|-----------|------------|
| 膨胀类型 | 非膨胀型，吸水膨胀小 | 膨胀型，体积膨胀10~20倍 |
| 湿压强度 | **高** | 中等至低 |
| 干压强度 | **低** | 高 |
| 热湿拉强度 | **低** | 高 |
| 水分适应性 | 差，水分波动时强度下降明显 | 好，水分较高时仍能保持强度 |
| 抗夹砂能力 | **弱** | 强 |
| 溃散性 | **好** | 差，需控制加入量 |
| 耐热复用性 | 差，反复焙烧后粘结性衰减快 | 好 |
| 造型后表面稳定性 | 易掉砂，需尽快浇注 | 表面稳定性好 |

两种膨润土型砂在不同水分及粘土含量下的干压强度对比曲线也证实了上述差异：相同水分条件下，钙基膨润土砂的干压强度显著低于钠基膨润土砂[[S1]]。

## 人工钠化活化改性

### 活化原理

利用蒙脱石层间阳离子可交换的特性，通过加入碳酸钠（Na₂CO₃）将钙基膨润土中的Ca²⁺置换为Na⁺，可显著改善其综合性能[[S26,S15]]。反应方程式为：

**Ca²⁺-蒙脱石 + Na₂CO₃ → Na⁺-蒙脱石 + CaCO₃↓**

### 工艺参数

人工钠化主流工艺为干法，关键参数如下[[S22,S27]]：

| 工艺参数 | 推荐值 |
|----------|--------|
| 碳酸钠添加量 | 膨润土总质量的3%~5% |
| 典型工业化用量 | 40 kg Na₂CO₃ / t膨润土 |
| 混合物料含水率 | 约33.5% |

干法工艺流程为[[S23,S22]]：钙基膨润土破碎 → 添加Na₂CO₃水溶液混合挤压 → 静置老化完成离子交换 → 低温干燥 → 碾磨分级 → 成品钠化膨润土。其流程如图所示：

![钙基膨润土干法人工钠化典型生产工艺流程图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/5f56dc46-43e1-4b58-b31d-945d05d29034/resource)

### 活化效果

钠化处理后，膨润土膨胀倍数可提升至**15~20倍**，粘结性能、耐火性、抗夹砂能力均显著提升[[S27,S22]]。

## 在铸造领域中的应用

### 砂型铸造——核心应用

钙基膨润土是湿型砂、表面烘干型砂和干型砂的常用无机粘结剂[[S31,S5]]。在砂型砂芯分类体系中，钙基膨润土作为粘土类无机粘结剂，覆盖湿型、表面干型和干型三种砂型类别[[S10]]：

![砂型砂芯按粘结剂类型分类示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/2954160f-ba15-424e-b23b-297e10dbf75e/resource)

**典型混砂配比与工艺条件**：

| 参数 | 典型范围 | 数据来源 |
|------|----------|----------|
| 湿型砂有效膨润土添加量 | 6%~10% | [[S41,S15]] |
| 标准测试配方 | 95%标准砂 + 5%膨润土 | [[S35,S15]] |
| 型砂紧实率控制 | 43%~47% | [[S17,S32]] |
| 表干型砂有效膨润土含量 | 5%~10% | [[S2]] |

钙基膨润土用于表干型砂时，常需加入占膨润土量4%左右的碳酸钠进行活化处理，以提升湿压强度和热湿拉强度[[S2]]。钙基膨润土型砂的湿压强度随水分含量及附加物添加量的变化规律已有系统研究：

![钙基膨润土型砂湿压强度与水分含量关系曲线](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/b07bc079-9ecd-4ba6-8d37-ed587d9f5544/resource)

### 铸造涂料——悬浮剂与辅助粘结剂

经钠活化后的钙基膨润土可作为水基铸造涂料的悬浮剂和辅助粘结剂。其作用机理为：膨润土在水中膨胀形成水化膜，搭接成立体网状结构，赋予涂料屈服值，阻止耐火填料颗粒下沉[[S44,S7,S14]]。

涂料中钙基膨润土（活化后）的典型添加量不超过总质量的**4%**[[S44]]。在金属型离心铸造中，也推荐使用钠活化的钙基膨润土作为涂料的悬浮剂兼粘结剂[[S14]]。

### 关于压铸应用——重要说明

经对全部权威压铸工艺手册、压铸水基涂料/脱模剂相关学术文献、行业标准及专利公开记录的检索，**未发现存在直接将钙基膨润土用作压铸脱模剂、润滑剂载体或压铸模具涂料常规组分的公开可靠应用记录**[[S24,S29,S6]]。

现有主流压铸水基/油基脱模剂和模具涂料的通用组分体系均不含钙基膨润土。已涉及的膨润土相关润滑材料记载，仅为表面活化改性膨润土作为稠化剂的普通工业润滑脂场景，与压铸工艺脱模/模具涂料无直接关联[[S3,S29]]。

## 技术标准与规格

### 国内标准体系

中国铸造用膨润土的标准化经历了逐步完善的过程[[S40,S9,S32]]：

| 标准版本 | 发布状态 | 关键内容 |
|----------|----------|----------|
| ZB J31 009—90 | 已废止 | 早期行业标准 |
| JB/T 9227—1999 | 已废止 | 行业标准升级版 |
| JB/T 9227—2013 | **现行有效** | 最新版本 |

### 分类与牌号

根据JB/T 9227—2013，铸造用膨润土依据阳离子交换容量和交换性阳离子含量进行分类，钙基膨润土的判定依据为可交换钙离子占主导地位[[S40]]。旧标准（JB/T 9227—1999）的牌号格式为**PCaS-3-5**（酸性钙膨润土，湿压强度3级，热湿拉强度5级），现行标准的牌号格式已更新为**ZP-Ca-\***类型[[S40,S9]]。

按pH值可将膨润土分为酸性（S）和碱性（J）两类，按湿压强度和热湿拉强度各分为四级[[S8,S32]]。

### 强制技术要求

铸造用钙基膨润土需满足以下通用技术指标[[S8,S13]]：

| 技术要求 | 指标 |
|----------|------|
| 吸蓝量（亚甲基蓝吸附量） | ≥20 g/100g膨润土 |
| 含水量 | ≤12.0%（冬季允许≤15.0%） |
| 粒度（通过0.075mm筛） | ≥95% |

### 国际标准参照

美国铸造协会（AFS）标准规定铸造用膨润土出厂含水量范围为6%~12%。在美国行业体系中，钙基膨润土被归类为“南部膨润土”（Southern bentonite），其特征为低干强度、高湿压强度，采用纯碱（碳酸钠）改性可有效提升其综合使用性能[[S9,S21,S4]]。

## 典型优势与局限性

### 优势

- **湿压强度高**：可比肩或超过钠基膨润土，满足湿型砂造型对湿态强度的要求[[S43,S5]]
- **溃散性优良**：高温残留强度低，落砂时不易形成热结块，无需额外添加大量溃散改良附加物[[S5,S31]]
- **成本低**：国内资源丰富，开采供应方便[[S22,S15]]
- **混砂能耗低**：吸水速度快，混碾所需能量较低[[S43]]

### 局限性

钙基膨润土的工艺局限性较为突出[[S17,S43,S21,S36,S33]]：

1. **膨胀性差**：原矿膨胀倍数仅为钠化后产品的1/5~1/10，膨润值低。
2. **热湿拉强度低**：抗夹砂缺陷能力弱，用于大平面铸件或浇注前长时间暴露的砂型时夹砂缺陷发生率较高。
3. **水分适应性差**：水分波动时型砂强度下降明显，高水分条件下强度保持能力远低于钠基膨润土。
4. **耐热复用性差**：反复高温焙烧后粘结性能衰减速度快，烧损后形成的“死粘土”在旧砂系统中持续积累。
5. **表面稳定性不足**：造型完成后砂型表面易掉砂，需尽快完成浇注工序。
6. **大吨位铸件适用性受限**：用于大吨位铸件浇注时易出现冲蚀砂型表面的问题，可能产生砂眼夹杂缺陷。

## 应用实例

工业生产实践表明，采用高等级钙基膨润土搭配合适的造型工艺，可在无额外表面涂料的条件下生产制动卡钳类球铁铸件，获得较高的铸件表面质量，从而简化后处理工序[[S25]]。

## 与相关材料的对比关系

### 与钠基膨润土

钙基膨润土适合小型薄壁铸件、造型后短时间内浇注的批量湿型砂生产；钠基膨润土适合大平面厚壁铸件、对抗夹砂能力要求高的生产场景[[S43,S19]]。

### 与有机膨润土

钙基膨润土仅可在水相中溶胀分散，作为水基体系的粘结剂/悬浮剂使用；有机膨润土经过季铵盐改性，可在醇类、油类有机介质中溶胀，仅用作醇基/有机溶剂型铸造涂料的悬浮剂，无法直接作为水基湿型砂粘结剂使用[[S28,S12,S44]]。

### 与水玻璃粘结剂

钙基膨润土属于粘土类湿型粘结剂，无需额外化学硬化即可获得湿态强度，旧砂回用性好、溃散性佳；水玻璃粘结剂为化学硬化无机粘结剂，可实现无烘干砂型快速硬化，但旧砂再生难度大、残留强度高、溃散性差[[S19,S28]]。

### 与树脂粘结剂

钙基膨润土成本极低，湿型造型生产效率高，但铸件表面精度有限；树脂粘结剂可获得较高的砂型表面强度和尺寸精度，适合高精度铸件生产，但材料成本高，部分含氮树脂易导致铸件针孔缺陷[[S28,S33]]。
