## 概述

压铸合金相组成是指在高压高速充型、极快冷却凝固的特殊工艺条件下，压铸合金内部所包含的全部固相（含基体相、初生相、共晶相、第二相、金属间化合物相等）的类型、含量、尺寸、分布与形貌的总和。和通用材料学基于平衡凝固或近平衡慢冷条件定义的相组成不同，压铸工艺的急冷特征会生成常规重力铸造不存在的特殊亚稳相、压室预结晶相，且相尺寸远小于普通砂型或金属型铸件，无需额外变质处理即可获得细晶相结构 [[S41,S3,S53]]。

相组成对压铸合金的力学性能、铸造性能和使用性能具有根本性的决定作用，因而是压铸合金研究与质量控制的核心理念。

## 压铸条件下相组成的特殊性

压铸合金的凝固过程包含两个阶段：部分熔体在压室中先行凝固形成预结晶组织（ESCs），随后含ESCs的熔体被高速压射入型腔完成充型与快速凝固 [[S43]]。型腔内模具与熔体之间的界面换热系数高达10000~12000 W·m⁻²·K⁻¹，冷却速率远超常规铸造，使得压铸合金的相组成具有以下显著特征：

- **非平衡凝固效应**：工业常规0.1~100℃/s冷却条件下，原本平衡凝固可得到成分均匀的单相固溶体，非平衡状态下溶质扩散完全受阻，最终得到枝晶状初生固溶体+非平衡共晶的混合组织，非平衡共晶中β相以不典型的网状分布在枝晶网胞周围，无标准平衡共晶形貌 [[S37,S59]]。
- **梯度异构组织**：铸件表层分布细小等轴晶粒和高占比细第二相，心部分布粗大等轴晶、第二相及ESCs，从表层到心部第二相面积占比逐步降低、固溶元素含量逐步减小。该异构组织是压铸工艺的普遍现象，不受合金体系、零件形状和工艺的显著影响 [[S43]]。

当冷却速率从1.3℃/s提升至6.4℃/s时，Al-Cu合金中Al₂Cu共晶相的平均宽度从10.40μm减小到5.43μm，共晶相平均面积分数从6.8%降低至3.5%；溶质原子被直接"冻结"在α-Al枝晶干内形成过饱和固溶体 [[S30,S56]]。高冷速下液固界面移动速度远超溶质原子的体扩散速率，溶质截留系数大幅提升，生成的θ-Al₂Cu相实际Cu含量仅为21.62~23.88at%，远低于理论值33.33at% [[S30,S56]]。

压铸合金因上述非平衡凝固特征，其相组成属于亚稳状态，存在自发向平衡组织转变的趋势 [[S37,S59]]。

## 压铸铝合金的典型相组成

### Al-Si系压铸合金

压铸Al-Si合金的典型相组成包含α-Al初生枝晶基体相、共晶硅相、纳米级Mg₂Si析出相，部分牌号还包含富铁金属间化合物相、Al₂Cu相等强化相。快速凝固条件下生成的直径大于10μm的枝晶状压室预结晶组织（ESCs）属于压铸合金独有的相组分 [[S55,S23,S41]]。

ADC12（Al-Si-Cu系）常规铸态相组成为初生α(Al)基体 + (α+Si)共晶组织 + 少量Al₂Cu相 + 微量Mg₂Si相，同时含杂质相AlFeMnSi。XRD实测验证其主物相为Al、Si、Al₂Cu三类 [[S12,S47]]。ADC12压铸件表层的急冷层区域α-Al相体积分数低于60%，靠近铸件心部区域的初生α-Al相占比从距表面0.5mm位置起保持稳定 [[S5]]。

A380与ADC12同属Al-Si-Cu系，相组成体系同源 [[S60]]。

### 合金元素对相组成的调控

**表1：ADC12铝合金主要元素含量范围** [[S39]]

| 元素 | 质量分数（wt.%） |
|------|-----------------|
| Si | 9.6～12.0 |
| Cu | 1.5～3.5 |
| Fe | <1.3 |
| Mg | <0.3 |
| Mn | <0.5 |
| Zn | <1.0 |
| Ni | <0.5 |
| Al | 余量 |

不同含量的合金元素对压铸合金的相组成存在明确调控规律。对于Al-xZn-8Si-2Cu压铸合金，Zn元素含量在指定范围内提升不会改变合金整体相组成（仍为α-Al、共晶Si、富Zn相、Al₂Cu相四类），但会提升基体中固溶的Zn原子占比，引发晶格畸变实现固溶强化，同时合金中富Zn相的占比随Zn含量升高而同步提升 [[S9,S6]]。

针对Fe含量不超过0.15wt%的压铸AlSi10MgMn合金，在常规非平衡凝固条件下，随Mn含量从低于0.6wt%提升至高于0.6wt%，合金的初生相从原本的α-Al相转变为α-Fe相，常规快冷条件下甚至可让α-Fe相与α-Al相同步析出，改变合金中富Fe相的形成路径 [[S18]]。

## 压铸镁合金的典型相组成

AZ91D（Mg-Al-Zn系）铸态微观组织由α-Mg晶粒和晶界相组成，晶界主要由不连续分布的Mg₁₇Al₁₂（β相）和富Al的过饱和α-Mg固溶体构成 [[S60]]。高压压铸AZ91D的表层冷硬区相比铸件心部含有更高体积分数的Mg₁₇Al₁₂金属间相，薄壁铸件的细晶表层占比更高，对应更高的β相整体占比 [[S28]]。

## 压铸锌合金的典型相组成

ZZnAl4Y（中国GB/T 13818-1992标准）铸态相组成为η-Zn固溶体、富Al的初生α相以及微量MgZn₂析出相，具备良好的尺寸稳定性与压铸工艺性 [[S17]]。

根据GB/T 46786-2025《锌合金压铸件金相检验》，压铸锌合金的相组成术语定义为：初生η相是少量铝溶于锌基体优先析出的固相；初生α相是锌溶于铝基体优先析出的固相；共晶组织是由共晶转变形成的两相或多相组织；共析组织是由共析转变形成的两相或多相组织 [[S49]]。

**表2：压铸锌合金金相组织类型及特征**（依据GB/T 46786-2025） [[S25]]

| 合金牌号 | 组织类型 | 特征 |
|----------|----------|------|
| YZZnAl4A, YZZnAl4B, YZZnAl4Cu1, YZZnAl4Cu3 | 初生η相+共晶组织 | 初生η相呈胞状，共晶组织呈不连续层片状 |
| YZZnAl8Cu1, YZZnAl11Cu1 | 初生α相+共晶组织 | 初生α相呈胞状，共晶组织呈不连续层片状 |
| YZZnAl27Cu2 | 初生α相+共析组织 | 初生α相呈梅花状或短棒状，共析组织呈不连续网状 |

压铸锌合金相组成中初生相的体积分数可通过网格数点法、网格截线法、图像分析仪测定法三种标准方法定量测量，图像分析软件自动提取初生相的面积百分数即可等效为初生相的体积分数 [[S49,S24]]。

## 相组成对性能的影响

### 对力学性能的影响

非平衡快冷条件下获得的细小球形初生晶粒、细小均匀分布的第二相可大幅提升合金的强度和延伸率；粗大针状或片状富Fe相、未被充分弥散的大尺寸块状金属间化合物会割裂基体，直接成为裂纹萌生的起点，大幅降低合金的塑性和冲击韧性 [[S13]]。

A380在150℃和175℃下的蠕变强度为同工况AZ91镁合金的6~8倍，150℃下35~50MPa区间的蠕变应变比AZ91低1~2个数量级，高温相热稳定性远优于AZ91镁合金 [[S60]]。

### 对铸造性能的影响

高占比共晶相可大幅提升合金的充型流动性，同时降低合金的固液区间宽度，减少枝晶间的液相补缩阻力，降低缩孔、缩松的生成概率，减小热裂倾向性；粗大枝晶结构或异常分布的非平衡相则会增大流动阻力，提升热裂生成风险 [[S8,S44]]。

压铸Al-Si合金因结晶温度间隔小、硅相凝固结晶潜热和比热容大、线收缩率小，且具有良好的流动性能和较小的热裂疏松倾向，因此应用最为广泛 [[S44]]。

### 对使用性能的影响

高占比单一α固溶体相的Al-Mg压铸合金内部无多相电位差，具备优异的耐蚀性能；压铸过程中快冷导致的高占比过饱和固溶原子会大幅提升自由电子的散射概率，使得压铸合金的热导率普遍显著低于重力铸造同成分合金 [[S13,S26]]。当黄铜压铸合金组织中同时存在α+β双相结构时，两相电位差较大，会显著加速脱锌腐蚀的发生，降低合金耐蚀性 [[S2]]。

## 相组成的分析手段

压铸合金相组成的完整表征需联合使用金相、XRD和SEM/EDS三种手段：优先采用XRD完成全局物相的初步定性，结合金相观测宏观相分布特征，最后通过SEM/EDS对XRD无法识别的微量/微区相进行精准元素组成分析，三者组合可覆盖压铸合金全尺度相组成的完整表征，避免单一检测手段的漏检问题 [[S58,S10]]。

### 金相（光学显微组织分析）

**适用场景**：观测合金基体、共晶组织、第二相的形貌与分布，结合图像统计晶粒尺寸、第二相面积/体积分数，识别α相、共晶Si、析出相等典型相的宏观分布特征。定量统计第二相体积分数的典型误差小于5%，晶粒尺寸测量分辨率可达微米级 [[S36,S16]]。

**操作要点**：试样切割为合适尺寸后依次用不同目数砂纸逐级打磨（每道工序旋转90°消除划痕，持续冷却避免过热），随后抛光至表面无划痕，根据合金类型选择对应腐蚀液腐蚀数秒后清洗吹干。在优化物镜、光源、柯勒照明的光学显微镜下采用明场/暗场模式观测，搭配ImageJ或Image Pro Plus软件统计定量参数，统计时需至少选取5张以上不同视场的金相图取平均值提升可靠性 [[S36,S16]]。

**适用标准**：GB/T 13298-1991《金属显微组织检验方法》、GB/T 46786-2025《锌合金压铸件金相检验》、JB/T 7946.1-1999《铸造铝合金金相 铸造铝硅合金变质》、JB/T 7946.2-1999《铸造铝合金金相 铸造铝硅合金过烧》、JB/T 7946.4-1999《铸造铝合金金相 铸造铝铜合金晶粒度》 [[S1,S20,S46,S32]]。

### X射线衍射（XRD）

**适用场景**：对物相进行全局定性识别，检测晶格参数、晶体畸变、残余应力，定量计算各晶相的体积/质量分数，但无法识别含量过低、衍射强度被基体覆盖的微量相。定量相分析的常规相对误差在3%~10%区间，对含量大于5%的主相检测准确性高，对含量低于2%的微量相容易出现漏检 [[S50,S22]]。

**操作要点**：试样测试面打磨至平整光滑，清洗烘干后采用Cu靶X射线，常规扫描参数设置为2θ范围10°~90°，扫描速度1°~4°/min。测试数据通过Jade等软件与PDF标准衍射卡片匹配完成物相鉴定，采用Ritveld全谱拟合等方法完成定量相分析。若需提升低含量金属间化合物的检测精度，可对试样进行深腐蚀以降低基体衍射强度 [[S36,S40,S58,S16,S35,S38]]。

**适用标准**：测试操作符合ASTM E505、ASTM E915相关XRD检测通用规范，国际标准层面符合ISO 17025关于材料晶体衍射分析的精度要求 [[S57,S27,S50,S22]]。

### SEM/能谱（EDS）

**适用场景**：观测微纳尺度第二相的精细形貌，对选定微区进行点、线、面扫描得到元素分布数据，精准测定单个第二相粒子的元素组成，结合全局XRD结果完成微区相的精准判定，可有效识别XRD漏检的低含量微区相。微区元素定量的相对误差在1%~5%区间，元素含量检测下限约0.1wt.%，但无法区分原子序数小于4的轻元素，受电子束激发体积影响，检测分辨率为微米级 [[S42]]。

**操作要点**：试样制备流程同金相试样，无需喷金的导电金属试样直接放置在SEM真空仓内观测，常规工作加速电压设置为15~30kV，对选定微区进行足够时长的信号采集以提升计数统计精度，测试结果结合元素原子百分比推导对应物相分子式 [[S36,S40,S29,S42]]。

**适用标准**：测试操作符合ASTM E1508能谱分析的通用规范，国际标准层面符合ISO 17025微区成分分析的精度要求 [[S27,S33]]。
