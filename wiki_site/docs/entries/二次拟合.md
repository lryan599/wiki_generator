## 概述

二次拟合（Quadratic Fitting），又称二次曲线拟合或二阶多项式回归（Second-Order Polynomial Regression），属于多项式回归的二阶特例。该方法基于最小二乘原理，通过最小化所有实测点与二次拟合曲线之间的残差平方和，拟合出能反映数据整体非线性趋势的抛物线关系。与插值不同，二次拟合无需曲线严格经过每一个离散数据点，使其特别适合处理含测量噪声的工程试验数据[[S3,S23,S34]]。

在材料加工工程领域，二阶多项式的拟合精度通常满足绝大多数场景需求，而高于二阶的多项式极易出现过拟合问题[[S39,S40]]。二次拟合能够同时捕捉单自变量或多自变量的线性效应、二次弯曲效应以及变量间的交互作用，是压铸工艺建模中应用最广泛的低阶多项式模型之一[[S10,S11,S29]]。

## 数学形式

### 一元二次拟合

对于单个自变量 \(x\) 与因变量 \(y\) ，一元二次拟合的标准表达式为：

\[
y = a_0 + a_1 x + a_2 x^2
\]

其中 \(a_0\) 为常数项，\(a_1\) 为一次项系数，\(a_2\) 为二次项系数。采用最小二乘法，通过求解三个线性方程来确定三个未知系数[[S1,S41]]：

\[
\begin{aligned}
a \sum x^4 + b \sum x^3 + c \sum x^2 &= \sum x^2 y \\
a \sum x^3 + b \sum x^2 + c \sum x &= \sum x y \\
a \sum x^2 + b \sum x + cn &= \sum y
\end{aligned}
\]

### 多元二次拟合

当涉及多个自变量时，二次多项式回归的通用形式为[[S10,S11,S26]]：

\[
y = \beta_0 + \sum_{i=1}^{k} \beta_i x_i + \sum_{i=1}^{k} \beta_{ii} x_i^2 + \sum_{1 \leq i \leq j \leq k} \beta_{ij} x_i x_j
\]

该模型包含所有自变量的一次项、平方项与两两交互项，能够完整描述线性主效应、非线性弯曲效应以及变量间的耦合作用。

## 拟合效果评价

### 决定系数 R²

决定系数 \(R^2\) 是二次拟合最常用的拟合效果评价指标，其计算公式为：

\[
R^2 = 1 - \frac{SSE}{SST}
\]

其中 \(SSE\) 为残差平方和，\(SST\) 为因变量的总偏差平方和。\(R^2\) 取值范围在 0 到 1 之间，越接近 1 代表拟合曲线对数据趋势的解释能力越强[[S26,S32]]。

### 调整 R²（校正决定系数）

调整 \(R^2\) 是对 \(R^2\) 的修正指标，会惩罚模型中不必要的冗余自变量项，避免随模型项数增加而虚增的拟合优度判定，更适合多元二次拟合场景下的拟合效果公平对比[[S25,S30]]。

### 残差分析

残差分析是检验二次拟合质量的重要手段，常用方法包括[[S24,S26,S37]]：

- **残差正态概率图**：验证残差是否符合正态分布；
- **预测值与实际值散点图**：检验数据点是否均匀分布在参考直线周围，避免系统性偏差；
- **剩余标准离差 \(S\)**：评估拟合的绝对误差水平；
- **\(F\) 检验**：判定整体回归方程的统计显著性。

在铸造领域应用中，二次拟合模型常要求 \(P < 0.05\) 以证明模型显著性[[S21]]。例如，大尺寸铝合金轮毂低压铸造的二次枝晶间距模型中，\(R^2 = 0.9898\)，校正 \(R^2 = 0.9556\)，标准差 0.5731，表明拟合效果优良[[S30]]。

## 压铸工艺中的应用

### 铸件缺陷率建模

针对高压压铸工艺，可采用三水平中心复合试验设计（CCD）构建包含快压射速度、慢快压切换点、增压压力、保压时间等关键参数的二阶非线性回归模型，用于预测铸件孔隙率。该模型经方差分析（ANOVA）验证显著，12 组随机验证样本的预测误差处于工程允许范围内，可在无额外试铸的情况下得到不同参数组合对应的铸件质量结果[[S16,S22,S36]]。

### 缩松缩孔体积预测

铝合金水泵座压铸研究中，以模具预热温度、铝液温度、活塞压射速度为自变量，建立了缩松缩孔体积的二次回归模型。该模型 \(F\) 值为 17.27，\(P = 0.0005 < 0.05\)，说明模型非常显著，拟合精度较高，可覆盖实测缩孔体积 0.451～0.721 cm³ 的范围[[S21]]。

### 工艺优化降低缺陷

多筋薄壁铝合金减震塔压铸工艺优化中，基于响应面中心复合设计方法，以模具预热温度、浇注温度、压射速度为自变量的二阶多项式模型可准确映射三类参数与铸件缺陷率的非线性关系。基于该模型寻优得到的工艺参数组合可将铸件缺陷率降低 40% 以上[[S43]]。

H3NG 气缸盖压铸工艺优化中，经二次回归得到的漏水率预测模型的预测 \(R^2\) 达 60%，显著性 \(P = 0\)，基于该模型优化后可将铸件漏水不良率降低 27%[[S33]]。

## 数据要求与局限性

### 数据要求

二次拟合对数据量和质量有明确要求[[S18,S35,S44]]：

- **样本量**：观测样本总数必须严格大于 3（待估计系数数目），工程应用场景下通常要求样本数远大于 3 以避免过拟合；
- **异常值处理**：拟合前需预先筛查并剔除明显偏离整体趋势的测量异常"坏点"，降低异常值对拟合结果的干扰；
- **数据分布**：数据点建议在自变量的目标取值区间内尽可能均匀分布，避免观测点过度集中在局部自变量区间导致拟合系数估计失真。

### 局限性

二次多项式拟合存在以下限制[[S5,S19,S31]]：

- **单极值限制**：仅能捕获单一的全局凸或凹的非线性趋势，无法描述含有两个及以上拐点的复杂非线性数据关系；
- **外推风险**：外推预测能力极差，在样本观测区间之外的区域拟合曲线会出现无依据的快速抬升或下降的振荡失准现象，外推预测误差通常远高于区间内插值；
- **阶次限制**：与三阶及以上多项式相比，无法拟合更高阶的非线性特征，但高于二阶的多项式又极易出现过拟合，丢失参数对响应的全局作用规律。

## 与相关方法的对比

### 一元/多元线性拟合

一元线性拟合（\(y = \beta_0 + \beta_1 x\)）仅能描述单自变量与响应值的一阶线性关系，适配自变量与响应近似呈完全线性单调趋势的场景[[S15,S17]]。多元线性拟合（\(y = \beta_0 + \sum\beta_i x_i\)）用于描述多个自变量与响应值的线性关联，当压铸工艺参数与铸件质量近似为线性作用关系、不存在明显极值拐点时可直接选用[[S8,S15]]。

### 三次及以上高阶拟合

三次及以上高阶多项式拟合存在显著缺陷：待求参数数量随阶数快速增长，极易出现过拟合，丢失参数对响应的全局作用规律。在压铸低维建模场景中，无特殊需求时不推荐使用，仅适配数据量极大的超高维复杂映射场景[[S5,S28,S31]]。

### 响应曲面方法（RSM）

响应曲面方法是 1951 年由 Box 和 Wilson 提出的试验统计方法，其核心是基于系统化试验设计（中心复合设计 CCD、Box-Behnken 设计 BBD 等）采集数据，通过二次多项式回归拟合得到全参数的响应映射[[S12,S20,S36]]。与普通多项式事后拟合相比，RSM 具备试验点正交性、统计显著性可控、优化导向性强等特征，是当前压铸工艺参数多目标优化的主流方法。

### 通用非线性曲线拟合

通用非线性曲线拟合需预先指定指数、幂函数、S型曲线等特定非线性函数形式，通过变量代换转化为线性拟合或直接迭代求解。在压铸建模中仅用于已知明确物理机理的特定映射关系拟合（如冷却速率与二次枝晶臂间距的幂律关系），不适合无明确机理的多参数耦合优化场景[[S26]]。

### 方法选用原则

在压铸工艺建模中，方法选用的通用原则如下[[S13,S24,S27]]：

| 场景 | 推荐方法 |
|------|---------|
| 快速标定单参数线性关系 | 一元线性拟合 |
| 少参数且作用近似线性 | 多元线性拟合 |
| 存在单极值非线性关联且样本量有限 | 二次拟合 |
| 需要定位多参数全局最优工艺窗口 | 响应曲面方法（RSM） |
| 无明确物理依据 | **禁止**选用三次及以上高阶多项式拟合，避免过拟合导致优化结果失真 |
