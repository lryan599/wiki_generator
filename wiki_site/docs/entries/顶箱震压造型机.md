## 概述

顶箱震压造型机是震压式造型机的一个子类，采用震击后补充压实的联合紧砂工艺，并以顶杆或顶框将砂箱向上顶升的方式实现起模。按 GB/T 25370—2020《铸造机械 术语》的规定，其标准中文名称为“顶箱震压造型机”，英文对应术语为 **jolt‑squeeze stripper molding machine** [[S7,S9]]。在铸造专业文献中，jolt squeeze stripper、jolt‑squeeze pattern‑draw molding machine 等亦作为通用英文表述变体使用 [[S15,S20]]。

顶箱震压造型机广泛应用于国内中小铸造厂的湿型砂造型作业，配套的国产主流系列为 Z14 系列，典型型号包括 Z145、Z148、Z1410 等，由苏州铸造机械有限公司、保定维尔铸造机械有限公司、浙江临海市铸造机械厂等企业生产 [[S18,S16,S2,S3]]。

## 定义与类别归属

在 GB/T 25370—2020 的造型设备分类体系中，震压造型机（jolt‑squeeze molding machine）被定义为“采用震实和压实方法（顺序进行或同时进行）紧实型砂的造型机”，其下存在两个并列子类：脱箱震压造型机与顶箱震压造型机。后者的专属特征在于配备独立的顶杆或顶框起模机构 [[S18,S11]]。

顶箱震压造型机的核心定义可概括为：以气动微震震实型砂底部，再以压实动作强化砂型上部型砂，紧实完成后通过顶箱机构将砂箱整体抬升，使固定于工作台上的模板与砂型分离，完成起模 [[S21,S12,S16]]。

## 工作原理

顶箱震压造型机的紧砂过程分为两个顺序进行的阶段 [[S18,S12,S21]]：

1. **震击（震实）阶段**：压缩空气进入震击缸，驱动震击活塞带动工作台及砂箱反复上升后依靠自重跌落，以惯性作用完成砂箱底部及模样周围型砂的初步紧实。
2. **压实阶段**：震实结束后，压缩空气通入压实缸底部，压实活塞上升，将砂箱举升至压头位置，使砂箱上部型砂受到进一步压实。

紧实完成后，由专用顶箱机构驱动顶杆（布置于砂箱四角对应位置），将砂箱整体向上顶起，型板固定在工作台面保持静止，从而完成起模 [[S18,S12,S16,S21]]。部分机型采用紧实后砂箱不动而模板下降的回程起模方式，亦属于顶箱起模范畴 [[S21]]。

## 核心结构

顶箱震压造型机主要由以下功能组件构成 [[S18,S12,S21,S3]]：

| 结构组件 | 功能 |
|---------|------|
| 震击机构 | 以气动方式驱动工作台往复跌落，完成型砂震实 |
| 压实机构 | 通过压实缸将砂箱举升并压向压头，实现上部型砂补充压实 |
| 顶箱起模机构 | 由顶箱活塞/气缸及顶杆组成，驱动砂箱抬升至脱离模板 |
| 模板系统 | 固定在工作台上，承载铸件模样 |
| 压头（回转式或移动式） | 提供压实工序中的上压紧面 |
| 底座及机身 | 承载全部机构与砂箱工作载荷 |
| 管路控制系统 | 实现震击、压实、起模等各工序的气动/液压逻辑控制 |
| 震击缓冲机构 | 降低运行冲击与噪声，所有量产机型均配置 |

## 与其他造型方法的差异对比

### 与单一紧实方法的区别

| 对比对象 | 核心差异 | 来源 |
|---------|---------|------|
| 震实台（振实台） | 震实台无专用起模机构，仅靠振动紧实型砂；顶箱震压造型机同时具备震实、压实两套紧实机构与独立顶箱起模结构 | [[S23,S17]] |
| 压实造型机 | 压实造型机仅依靠单一压实动作紧砂，无震击环节，砂箱高度越大，上下紧实度偏差越大；顶箱震压造型机通过震击+压实联合作用显著改善紧实均匀性 | [[S23,S3,S17]] |
| 微震压实造型机 | 微震压实造型机的震击与压实同步进行，且本身不配置顶箱起模机构，需搭配外置起模装置；顶箱震压造型机的震击和压实为先后顺序独立完成 | [[S17,S19]] |
| 气流冲击造型机 | 气流冲击造型机依靠压缩空气或燃气的瞬间压力波冲击紧砂，无机械震击与压实动作，也不具备顶箱自动起模结构 | [[S23]] |

### 与其他造型方式的工艺经济性对比

| 对比对象 | 优势 | 局限性 | 来源 |
|---------|------|--------|------|
| 高压造型机（压实比压 >0.7 MPa） | 设备采购成本低、维护难度小 | 砂型紧实度上限较低，自动化程度普遍偏低 | [[S23,S19]] |
| 无箱造型机 | 适配砂箱范围更大，可生产铸件尺寸上限更高 | 需配套砂箱工装，砂箱成本更高 | [[S23,S19]] |
| 射压造型机 | 结构更简单，模板使用寿命更长 | 运行噪声更大，砂型顶部紧实度不如射压造型机，造型周期更长 | [[S23,S19]] |

### 顶箱起模与其他出箱方式的区别

顶箱起模的核心特征在于：砂型紧实后，布置在砂箱四角的顶杆直接将砂箱向上顶起，型板保持固定在工作台面不动，砂型与模样分离。[[S18,S16]] 与翻转式出箱（需整台砂箱连同模板翻转 180°）相比，顶箱起模的起模流程更短、结构更简单，生产率更高，适用于造上型或无大块吊砂的普通中小型铸件；与移箱式出箱（需接箱台水平移入托住砂型）相比，顶箱起模占用横向空间更小，结构复杂度更低 [[S18,S14]]。

## 关键参数与工艺条件

顶箱震压造型机以压缩空气为动力，微震结构型号的震击频率可达 600~800 次/分，振幅 2~6 mm，压实比压范围为 0.3~0.6 MPa（属于中低压造型范畴，压实比压 0.7 MPa 以上界定为高压造型）[[S22,S10,S3]]。

## 变体与分类

### 按自动化程度

顶箱震压造型机可分为半自动与全自动两类 [[S23]]。

### 按改进结构形式

- **微震机型**：在压实的同时施加高频（600~800 次/分）、小振幅（2~6 mm）微震，进一步提升砂型紧实度均匀性 [[S19,S10]]。
- **气垫震击型**：采用气垫复位式微震机构替代传统弹簧复位，对地基的减震要求显著降低，但微震部位的磨损相较于普通纯震击结构更快 [[S19]]。

在行业标准的产品分类中，气动微震压实造型机进一步按起模方式区分为 A 型（顶箱起模）与 B 型（翻转起模），顶箱震压造型机则明确归入 A 型范畴 [[S22]]。

## 在铸造流程中的角色

顶箱震压造型机是湿型砂制备环节的核心设备之一，适用于中小型砂箱造型，广泛部署于国内中小铸造厂 [[S11,S19]]。其通过震击与压实组合作用得到的砂型，相比单纯的压实造型机，砂箱上部砂层的紧实度显著提升；相比纯震击造型机，整体紧实均匀性更优，最终可有效降低铸件掉砂、冲砂类缺陷的发生概率 [[S11,S19]]。

适应的典型铸件类型包括 [[S11,S19]]：
- 小型机床件
- 通用箱体类零件
- 中小规格轮毂铸件
