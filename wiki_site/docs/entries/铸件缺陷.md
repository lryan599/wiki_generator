## 概述
在压铸生产中，铸件缺陷是指压铸件在尺寸、形状、致密度、外观、宏观或微观组织以及力学性能等方面，出现不符合技术标准、图纸规定或供需双方商定条件的不允许偏差[[S49,S20,S26]]。同一类缺陷在不同验收规则下，可被定义为可接受偏差、不可接受缺陷、可返修缺陷或必须报废的缺陷[[S49]]。

铸件缺陷对产品质量和生产成本产生重大影响。它会损害力学性能，降低承载能力和疲劳强度，局部孔洞或裂纹会成为应力集中源，直接引发零件失效；穿透性缺陷会破坏气密性或水密性，表层气泡则会导致后续涂装或电镀起泡脱落。此外，由于压铸是典型的大批量生产工艺，某类缺陷的批量爆发会造成整批报废，显著提升废品率及检测、返修环节的工序成本[[S11]]。

## 缺陷分类体系
### 国内标准分类
国内现行标准对压铸缺陷采用系统分类。依据GB/T 5611-1998《铸造术语》和GB/T 29092-2012《镁及镁合金压铸缺陷术语》，可将压铸缺陷划分为六大类[[S29,S19]]：

*   **形状尺寸类缺陷**：如变形、尺寸超差、错型、欠铸等。
*   **表面类缺陷**：如冷隔、花纹、流痕、飞边、缩陷、粘附物痕迹等。
*   **孔洞类缺陷**：如气孔、缩孔、缩松、内部针孔及气泡等。
*   **裂纹类缺陷**：分为热裂纹和冷裂纹两种。
*   **材质类缺陷**：如材质不良、金相不合格、化学成分不均、脆性过大等。
*   **后处理类缺陷**：如渗漏、烤漆起泡等。

压铸领域存在一些高频专属缺陷，如硬点、卷气类气孔、冷隔、粘模拉伤和飞边等，这使其与砂型铸造的缺陷特征（如砂眼、鼠尾）有明显区别[[S19]]。

### 国际标准分类
国际铸造技术协会（ICFTA）出版的《国际铸件缺陷图谱》确立了一套通行的分类系统，并被世界铸造组织（WFO）采用。该体系将所有铸造缺陷划分为7大类，并以一位大写字母加三位数字的编码进行唯一标识，便于跨语言和跨区域交流。[[S51,S16,S52]]。

| 类别代码 | 缺陷类别 |
| :--- | :--- |
| A | 多肉类 (Projections) |
| B | 孔洞类 (Cavities) |
| C | 裂纹、冷隔类 (Discontinuities) |
| D | 表面缺陷类 (Defective surface) |
| E | 铸件残缺类 (Incomplete casting) |
| F | 尺寸或形状差错类 (Incorrect dimensions or shape) |
| G | 夹杂物和金相组织不合格类 (Inclusions & metallurgical defects) |

### 其他分类维度
除了基于形貌的分类方法外，在实际工程应用中也常按形成机理或产生环节对缺陷进行分类，以支撑根因溯源和面向质量管控的前端设计。[[S39,S65]]

## 常见缺陷类型与形成机理
铝合金压铸件常规含气量可达15至45 ppm，远高于金属型铸造的约2 ppm，使得孔洞类缺陷在压铸中尤为突出[[S63]]。以下列出主要的缺陷类型及其特征。

### 孔洞类缺陷
#### 气孔
*   **特征**：孔洞表面光滑，形状多为规则的圆形或椭圆形，可从表面鼓出形成可见气泡[[S1]]。
*   **机理**：其成因主要包含三种[[S63,S27]]：
    1.  **卷入性气孔**：金属液在型腔中高速紊流充填时卷入空气，这是最主要原因。
    2.  **析出性气孔**：合金液在凝固过程中，因溶解度骤降而析出气体（主要为氢气）。
    3.  **反应性气孔**：脱模剂、涂料等受热分解产生的气体未能及时排出而被包裹。
*   **合金特性**：镁合金压铸件中的气孔等效直径通常在20至100 μm之间，其含量与压射过程的紊流程度呈正相关。[[S30]]

#### 缩孔与缩松
*   **特征**：解剖或探伤检查可见，孔洞表面呈暗灰色、不光滑，形状极不规则。大而集中的不规则孔洞为缩孔，而分散分布的细小、不连续网状孔洞为缩松，多集中于铸件的厚大热节区域。[[S1]]
*   **机理**：压铸件在凝固阶段发生体积收缩，但内浇口先于铸件主体凝固，切断了高压补缩的液相通道，导致厚大部位的收缩得不到金属液补充，最终在凝固末端形成孔洞。镁合金中的预结晶组织（ESCs）相互聚集，会额外增加缩松的生成概率。[[S30,S50]]
*   **合金特性**：在铝合金中，Al-Si系合金的线收缩系数较小，其缩松倾向远低于Al-Mg和Al-Zn系合金。[[S27]]

#### 孔洞微观形貌
通过三维重构技术，可区分压铸件内部的四种微观孔洞形态：气孔、气缩孔、岛状缩松和网状缩松[[S70,S50]]。其形态分布主要受充填过程中的卷气现象和凝固阶段的补缩能力控制[[S50]]。

### 表面缺陷
#### 流痕
*   **特征**：铸件表面可见与金属液流动方向一致的条纹，颜色与基体存在差异，手触有轻微感觉，无扩展趋势，可通过抛光去除。[[S28,S1]]
*   **机理**：首先进入型腔的金属液接触低温模壁后，快速冷却形成一层极薄的不完全凝固层，后续的高温金属液无法将其完全熔合覆盖而留下的痕迹。[[S1,S28]]

#### 冷隔
*   **特征**：铸件表面出现不规则的、细长下陷线状纹路，分为穿透型和非穿透型。其界面结合强度极低，外力作用下会沿纹路扩展开裂，多发于远离内浇口的充填末端。[[S28,S1]]
*   **机理**：两股温度较低的金属液流前端在型腔中相遇，但接触时表层金属已提前凝固，导致两股流体未能实现完全熔合。[[S57,S1]]

#### 鼓泡
*   **特征**：铸件表面出现鼓包。
*   **机理**：位于铸件近表层的封闭气孔，在后续加热过程中（如烤漆工序），因内部气体受热膨胀使表层金属塑性变形而向外隆起所形成。[[S49]]

### 形状尺寸缺陷
#### 欠铸
*   **特征**：型腔未被完全充满，铸件出现边缘、棱角缺损，或局部呈现自然流动终止的液面轮廓，多发生在薄壁、深腔区域。[[S1,S49]]
*   **机理**：熔融金属在充型完成前提前凝固，或充型压力不足以将其推送至远端薄壁区。[[S57,S1]]

#### 变形
*   **特征**：铸件整体或局部发生翘曲、弯扭，超出图纸规定的公差范围，可通过三坐标测量等方法检测。[[S69,S1]]
*   **机理**：铸件各部位冷却收缩速率不一，导致内部留存残余应力。过早开模顶出、顶出受力不均，或后续机加工破坏了原有应力平衡，都可能引发变形。[[S18,S1]]

### 裂纹类缺陷
#### 裂纹
*   **特征**：金属基体断开，形成细长缝隙。浸入碱性溶液后，裂纹处会呈暗灰色。热裂纹的断面被氧化，而冷裂纹断面光亮洁净，这是区分两者的关键特征。[[S28,S1]]
*   **热裂机理**：铸件在固相线附近的半固态区间，收缩受到模具阻碍产生拉伸应力，导致晶界处的低熔点液相膜被撕裂，形成沿晶界分布的不规则裂纹。[[S18,S1]]
*   **合金敏感性差异**[[S1]]：
    *   **锌合金**：裂纹主要源于铅、镉、铁等有害杂质含量超标，且过早开模会大幅增加风险。
    *   **铝合金**：Al-Mg系、Al-Zn系合金裂纹倾向更高，硅含量偏低时热裂风险上升。
    *   **镁合金**：裂纹风险随合金中铝、硅含量升高而增大。

### 夹杂与附着缺陷
#### 夹杂
*   **特征**：分布在金属基体中的硬质点或暗黑色不规则聚集相，硬度远高于基体。机加工时会显著加剧刀具磨损，加工后表面显现出不同亮度的斑点。[[S1,S32]]
*   **机理**：熔炼过程中产生的Al₂O₃等氧化膜以及熔渣，在充型过程中被卷入铸件内部并未能上浮排出。[[S32,S1]]

## 检测与评级方法
对压铸缺陷的检验和分析，需结合多种无损和有损检测手段。

### 无损检测方法

| 检测方法 | 适用场景与能力 | 局限性与备注 |
| :--- | :--- | :--- |
| **目视检查** | 检查所有宏观表面缺陷，如花纹、流痕、缩凹、变形、冷隔、欠铸、变色、斑点等。可借助5倍以上放大镜辅助提升精度。[[S43,S15]] | 仅限表面可见缺陷。 |
| **渗透检测** | 检测所有非多孔性铸件（铝、镁等非铁合金）的表面开口缺陷，如裂纹和开口孔隙。可检出宽度低至0.01mm的缺陷。操作流程包括清洗、渗透、显像等环节。[[S36,S31,S9]] | 对铸件内部的非开放性缺陷无能为力。 |
| **超声检测(UT)** | 检测内部体积型缺陷（缩孔、气孔等）。定位精度高、穿透能力强、成本低且无辐射危害。对面积型缺陷（裂纹）检出率高。[[S44,S23,S55]] | 对表面粗糙、形状不规则的薄壁复杂铸件适应性较差，且单独用难以精准判定缺陷具体类别。 |
| **X射线检测(RT)** | 检测内部缺陷，提供缺陷平面投影的位置、形状、大小和分布。底片可长期保存。[[S15,S55]] | 三维物体二维成像，存在缺陷影像重叠失真问题，效率相对较低。 |
| **工业CT检测** | 可获取内部缺陷的三维形貌与空间分布，彻底消除二维影像的重叠失真问题。能精准量化缺陷的体积和位置。检测精度可达体素尺寸0.127mm级别。在国内执行标准为GB/T 29070-2012。[[S13,S54,S2]] | 成本较高，检测效率相对较低。 |

### 缺陷评级标准
*   **ASTM E155系列**：该标准是全球铝合金铸件射线检测的通用核心参考，它将气孔、圆形针孔、长形针孔、疏松、缩孔等7类典型缺陷依据标准图谱划分为8个等级，适用于最大厚度不超过51mm的铸件。[[S58,S48]]
*   **GB 5677和GB/T 11346系列**：国内标准GB 5677规定了气孔、夹渣、缩孔类缺陷在射线底片上的点数换算规则和多类缺陷共存时的综合降级原则[[S45]]。GB/T 11346则针对铝合金铸件的圆形针孔缺陷，提供了8个级别的评级图谱，适用最大厚度为50mm。[[S58]]
*   **ISO 10049**：该标准专门针对铝合金铸件的宏观针孔度，提供了0~5共6个等级的目视评定参考图谱，明确了各等级所对应的单位面积内孔洞数量与尺寸阈值。[[S17]]
*  **ISO 8062系列**：不属于缺陷评级标准，而是规定了铸件的尺寸、形位公差的分级与取值范围，为缺陷判定提供尺寸公差基准。[[S33]]

## 预防与工艺对策
缺陷控制贯穿压铸件的设计、模具制造、合金熔炼和工艺参数设置全过程。

### 模具与浇排系统设计
*   **浇注系统**：由直浇道、横浇道、内浇口和余料等组成，负责引导金属液充填型腔。其设计直接控制了金属液的流动方向、充填速度、压力传递和排气效果，是决定内部质量的关键。[[S5,S60,S12]]
*   **排溢系统**：由溢流槽和排气槽组成，用于储存混有气体和涂料残渣的冷污金属液，并将型腔内气体迅速导出，对减少卷气和冷隔至关重要。二者需与浇注系统作为整体协同设计。[[S22,S53]]
*   **减缺陷设计准则**：在不增加内浇口厚度前提下，通过提升总内浇口截面积（如增加约50%）、调整各分支的不对称轮廓和周向角度，可使铸件表面冷流缺陷几乎完全消除[[S46]]。此外，降低浇道内熔体流动速度，能有效减少紊流卷裹的氧化双层膜，从而降低形成孔隙的核心数量[[S59]]。
*   **冷却系统**：其设计原则是需避开铸件厚壁热节区域，以避免局部过热导致缩松缩孔，同时要控制模具表面温度交变温差，以降低热疲劳开裂概率。[[S41]]

### 合金熔体管控与关键工艺参数
*   **合金熔炼**：炉料需化验合格并按制度配比使用；熔炼温度须严格控制，接触金属液的工装应提前预热；除气剂使用前必须经干燥处理，并完整执行除气工序，以将熔体含气量控制在最低。[[S38,S35]]
*   **量化工艺影响**：增压压力是影响铸件致密度的最关键参数，其对孔隙率的贡献占比可达68%，远高于熔体温度和模具温度约10%的贡献率[[S30]]。模具温度控制同样关键，例如，锌合金模温低于150℃或铝合金模温低于180℃时，极易产生流痕和冷隔缺陷。[[S1]]
*   **工艺参数优化**：在满足质量要求的前提下，应尽可能选用最低的压射速度、压射压力和浇注温度，以降低设备和模具负荷、减少故障概率并延长模具寿命。[[S41]]
*   **喷涂规范**：使用水基脱模剂时，必须制定包含配比、喷涂量、喷涂时间、喷涂顺序、喷枪距离与角度、压缩空气压力等全部要素在内的标准化喷涂工艺，确保涂层均匀无过量堆积。[[S41]]

### 典型缺陷的针对性预防策略
| 缺陷类型 | 针对性预防对策 |
| :--- | :--- |
| **气孔** | 改进内浇道导入位置与面积，合理增设溢流槽与排气槽以增大排气量；采用慢速压铸工艺减少压室气体卷入；使用干燥洁净炉料，充分除气；选用发气量小的涂料并控制用量。[[S61,S38]] |
| **缩孔、缩松** | 优化铸件结构以消除热节，使壁厚均匀；条件允许时降低浇注温度；适当提高压射比压；改善浇注系统，确保压力能有效传递至铸件最后凝固的区域。[[S61]] |
| **冷隔** | 适当提高模具温度与合金液浇注温度；合理延长留模时间；成型部位均匀喷涂涂料；提高压射比压和充填速度；强化熔炼的精炼与撇渣；改善型腔排气条件。[[S56]] |
| **裂纹** | 严格管控合金成分，杂质含量不可超标（如铝合金中铁不宜过高、硅不宜过低）；优化铸件结构，保证壁厚均匀过渡，避免尖、凹角；控制恰当的留模时间；确保顶出时铸件各处受力均匀。[[S61]] |
| **夹杂** | 熔炼与浇注前彻底清除金属液表面熔渣；手工舀取金属液时严格遵循规范，防止舀入熔渣；使用石墨坩埚时边缘加装铁环；控制涂料喷涂量，保证涂层均匀。[[S35]] |
