## 定义

随机森林是在Bagging框架下，将自助采样策略与随机子空间方法相结合的集成学习算法。它以单棵决策树为基础弱学习器，通过降低集成模型的方差来抑制单决策树的过拟合问题，适用于工业数据中复杂的非线性映射场景[[S17,S8,S16]]。

## 类别与变体

随机森林包含两类核心变体[[S32,S18]]：

- **Random Forest Classifier**：面向分类任务，默认以基尼系数为节点划分准则，集成阶段通过多数投票方式输出最终类别。
- **Random Forest Regressor**：面向回归任务，默认以均方误差为节点划分准则，集成阶段以所有基决策树输出结果的算术平均值作为最终连续预测值。

## 核心原理与机制

### 随机性来源

随机森林的两类核心随机性分别为[[S8,S16]]：

- **样本层**：有放回自助采样，每棵基决策树的训练集从原始训练集中随机有放回抽取。
- **节点分裂层**：特征随机选择，在节点分裂阶段仅从全部特征的随机子集中筛选最优划分特征，以降低基学习器间相关性、提升泛化能力。

### OOB误差估计

OOB（袋外）样本为每轮自助采样中未被抽取的训练样本子集。无需额外划分验证集，即可通过OOB样本计算袋外误差，作为无偏估计量评估随机森林的泛化性能，并可用于辅助确定最优决策树数量等超参数[[S13]]。

### 特征重要性评估

随机森林特征重要性评估有两类主流方法[[S9]]：

| 方法 | 原理 |
|------|------|
| 基尼指数法 | 统计所有特征在各节点分裂时带来的基尼不纯度下降总和，以此衡量特征贡献度 |
| 袋外误差变化量法 | 依次随机打乱单个特征取值，统计OOB误差的增幅，增幅越大则该特征对预测结果的贡献越高 |

## 关键超参数

以下为小批量压铸工业过程数据分析场景下随机森林核心超参数的典型配置范围[[S18,S1]]：

| 超参数 | 推荐范围/取值 | 说明 |
|--------|---------------|------|
| n_estimators | 100～500 | 决策树数量，超过该范围后精度不再显著提升，仅增加计算开销 |
| max_depth | 6～17 | 最大深度，避免小样本过拟合 |
| min_samples_split | 2～10 | 最小分裂样本数，适配小数据集 |
| max_features | 特征数平方根 | 通常设为全部特征数的平方根 |
| bootstrap | True | 启用自助采样 |
| oob_score | True（小样本推荐） | 以袋外误差完成模型无偏验证 |

在压铸实测案例中，决策树数量超过150后训练损失即趋于收敛[[S25,S31]]。

## 技术优势与局限性

### 优势

- 对工业场景含噪声、缺失值的非规整表格数据鲁棒性强[[S17,S2]]；
- 可并行训练，计算效率高[[S17]]；
- 无需特征预处理即可输出特征重要性结果[[S17]]；
- 不易过拟合，泛化能力显著优于单棵决策树[[S17,S2]]；
- 将工业噪声分散到各棵决策树中，降低噪声对最终预测结果的干扰[[S17,S19]]。

### 局限性

- 模型可解释性弱于单棵决策树[[S18]]；
- 对极端不平衡数据敏感，少数类检出率可能严重偏低[[S25,S7]]；
- 当训练数据量足够大时，预测精度上限低于人工神经网络[[S2,S30]]；
- 训练过程仍需一定计算资源[[S25]]。

## 在压铸与铸造领域的应用

### 缺陷预测

**镁合金压铸缺陷分级预测**：基于48000+样本、71个原始压铸工艺参数，经皮尔逊相关系数（\|ρ\|>0.6判定强相关）降维得到23个弱相关输入特征，预测标签为0～3共4级铸件质量等级。采用Scikit-Learn的RandomForestClassifier，决策树数量500。原始不平衡数据集下总体准确率93.67%，AUC为0.9708，但少数类检出率低于20%；经降采样与SMOTE过采样处理后，准确率降至89.54%，AUC升至0.9838，平均真正率ATPR达87.65%，可稳定检出占总样本仅0.1%的最高等级缺陷件[[S31,S25,S7,S15]]。

**砂铸多缺陷分类**：基于18维铸造工艺特征的随机森林模型，总体召回率94.08%，平均分类精度90.72%，所有缺陷类别召回率均高于90%[[S2,S9,S26,S30]]。

### 工艺-性能关系建模

**铝合金压铸成分-力学性能预测**：数据集按4:1划分训练集与测试集，训练集决定系数R=0.989，测试集R=0.92，远优于线性回归的测试集R²=0.79。模型预测值与实测值偏差始终处于±5%误差带内[[S19,S4]]。

### 残余应力预测

随机森林回归器是电弧增材制造构件残余应力分布预测的主流机器学习方案之一。相比传统经验公式最高7.1%的预测误差，随机森林类回归模型预测精度显著更高；相比耗时数小时的全尺寸有限元残余应力仿真，推理效率可提升数个量级，能够支撑WAAM构件残余应力的快速在线评估与工艺参数动态优化[[S32,S14,S29]]。

## 可视化与可解释性工具

在压铸场景下，以下两类工具最为常用：

- **基尼特征重要性排序图**：直接输出各压铸工艺参数（如比压、填充时间、模温机温度等）对铸件质量的影响权重，快速定位影响冷隔、气孔、缩松等缺陷的核心工艺变量[[S15,S9]]。
- **SHAP值分析**：从全局和单样本两个维度解析工艺参数对铸件性能的正负向影响，标识贡献方向，验证工艺优化方向的合理性[[S20,S12]]。

## 与相关算法的对比

在铸造/压铸数据分析场景下，随机森林与其他算法的选用对比如下[[S2,S17,S19,S25,S31,S30]]：

| 对比对象 | 随机森林的相对表现 |
|----------|-------------------|
| 单一决策树 | 准确率显著更高，泛化能力更强 |
| 支持向量机 | 分类精度优于SVM，调参复杂度更低 |
| K近邻 | 分类精度优于KNN |
| 人工神经网络 | 对一维表格类数据适配性远优于NN；大数据量下NN可达更高精度，但超参数配置复杂、可解释性差 |
| GBDT/XGBoost | RF对噪声数据鲁棒性优于部分Boosting算法 |
| 线性回归 | 回归精度远优于线性模型 |

## 标准与规范

目前在压铸领域，GB/T 6414-1986《铸件尺寸公差》等效采用ISO 8062-1984铸件尺寸公差体系，尚未发布专门针对随机森林等机器学习数据分析应用的ISO或国内行业标准，所有压铸大数据分析应用均在现有铸造质量管控标准框架下开展[[S28,S3]]。
