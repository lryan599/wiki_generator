## 概述

非小平面型生长（non‑faceted growth），亦称粗糙界面生长（rough interface growth），是指在原子尺度上，原子在粗糙固‑液界面上随机堆砌的晶体生长方式。其枝晶前端生长面呈圆滑锥面，生长方向为锥尖指向。该生长模式的热力学条件为熔化熵 ΔSₘ ≤ 2R（R 为气体常数），大多数金属晶体的结晶属于此类型 [[S19]]。非小平面型生长所需动力学过冷度极低，仅为 10⁻²~10⁻⁴ K，生长速度可达 1~100 cm/(s·K)，是压铸合金凝固中最常见的基本生长模式 [[S20,S4]]。

## 定义与核心区分标准

### 原子尺度特征
非小平面型界面（粗糙界面）在原子尺度下，固相一侧仅约 50% 的点阵位置被固相原子随机占据，剩余位置为空位，形成厚度为数个原子层的固‑液过渡区 [[S32,S10]]。该界面无明确清晰的两相分界，液相侧与固相侧之间不存在原子级锐利边界，属于典型的多层弥散结构（弥散性界面）[[S10,S23]]。与之对应，小平面型界面（光滑界面）在原子尺度为基本完整的密排晶面，两相边界截然分开 [[S21,S27]]。

### 宏观形貌的反向特征
需特别注意，两类界面的微观特征与宏观形貌呈现反向关系：原子尺度粗糙的非小平面界面在显微尺度下反而平直光滑；而原子尺度平整的小平面界面由一系列轮廓清晰的小晶面构成，宏观呈现锯齿状参差不齐的形貌 [[S32,S27,S21]]。

**表1：非小平面型与小平面型生长对比**

| 特征 | 非小平面型生长 | 小平面型生长 |
|---|---|---|
| 界面原子尺度形态 | 粗糙，约 50% 位置被占据 [[S32]] | 平整，完整的密排晶面 [[S21]] |
| 界面过渡区 | 多层、弥散性结构 [[S10,S23]] | 锐利边界，两层结构 [[S21]] |

| 原子占据模式 | 随机等效占据任意位置 [[S20,S37]] | 台阶处逐层沉积 [[S16,S37]] |

## 分类与判定

### 凝固生长动力学分类
在凝固科学的生长动力学分类体系中，非小平面生长与连续生长机理完全对应，属于两类经典的固‑液界面生长模式之一 [[S4,S7]]。该模式下的生长速率 R 与动力学过冷度 ΔTₖ 呈线性正比关系：

```
R = μ₁ ΔTₖ    （连续生长动力学方程）
```

其中 μ₁ 为连续生长动力学常数，取值约 1~100 cm/(s·K) [[S4,S7]]。实际铸态凝固条件下，对应的动力学过冷度仅为 10⁻⁴~10⁻² K，几乎无法直接测得 [[S4,S20]]。

### 共晶体系分类对应关系
基于金属凝固过程中熔化熵 ΔSₘ 的数值，可将共晶凝固划分为三类，非小平面型生长占据其中一类 [[S15]]：
- **非小平面–非小平面共晶**（规则共晶）：两相熔化熵均 < 23 J/(mol·K)
- **不规则/复杂规则共晶**（小平面–非小平面）：一相熔化熵 > 23 J/(mol·K)，另一相 < 23 J/(mol·K)
- **双小平面共晶**：两相熔化熵均 ≥ 23 J/(mol·K)

典型的非小平面–非小平面共晶具有对称的共晶共生区，如金属‑金属共晶、金属‑简单金属间化合物共晶体系 [[S12,S15]]。

### Jackson 因子的判定机制

![不同 α 值下相对界面自由能变化量与覆盖率的关系曲线](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f5f19c11-5ff3-4d8e-8840-1307f6a4770e/resource)

Jackson 因子 α 是判定固‑液界面生长模式的核心热力学判据，其原始的表达式为 [[S16,S37]]：

```
α = (e / kTₘ) × (N₁ / N)
```

其中 e 为固相内部原子结合键能，k 为玻尔兹曼常数，Tₘ 为平衡凝固温度，N 为固相内部原子配位数，N₁ 为界面层内原子配位数。

**等效形式 I**（焓–熵形式）[[S2,S22]]：
```
α = (ΔS_f / R_g) × (η / v)
```

**等效形式 II**（熔化熵简化形式，忽略取向因子时）[[S13]]：
```
α = ΔS_t / R
```

其中 ΔS_f 为摩尔熔化熵，R_g 为气体常数，η 为界面晶面内原子近邻数，v 为固相内部原子近邻总数 [[S2,S16,S22]]。

**判据阈值** [[S37,S25,S13]]：
- α < 2 → 非小平面粗糙界面
- 2 < α < 5 → 两类界面的过渡区间
- α > 5 → 完全小平面光滑界面

**实例验证**：AlP 晶体的 {100} 晶面 α = 1.89（< 2，非小平面生长）；{110} 晶面 α = 0.95（< 2，非小平面生长）；而 {111} 晶面 α = 2.84（> 2，小平面生长）[[S25]]。

### 粗糙化转变与中间过渡状态
Jackson 原始模型基于双层界面结构的假设，但后续研究表明界面粗糙程度并非完全由 α 值唯一决定 [[S21,S10]]。多原子层界面模型指出，当过冷度升高、凝固速率加快时，界面过渡层厚度增加，粗糙度呈连续提升趋势，原本 α > 2 的小平面型材料在足够大的过冷度下也可发生粗糙化转变，实现非小平面模式的连续生长 [[S21,S30]]。

## 核心属性

### 界面结构与生长机制
非小平面粗糙界面的核心结构特征为多层弥散过渡：液相原子在该多原子层厚度的过渡区内逐步完成从完全无序（液态）到完全有序（固态）的有序度过渡 [[S10,S23]]。

在生长动力学上，界面上存在大量随机分布的现成台阶位置（约 50% 的点阵空位），液相原子可等效随机地占据界面任意空位，无需额外能量触发形核即可实现连续的均匀垂直推进 [[S4,S20,S7]]。持续维持的高效生长速度使得该类模式在所有生长机理中具有最快的动力学响应 [[S7]]。

### 生长速率的各向同性
由于粗糙界面上原子沉积位置完全随机，界面能的各向异性极弱，晶体不同晶向的生长阻力几乎没有差异，宏观生长行为缺乏强的择优取向，呈现各向同性的生长特征 [[S37]]。

这与小平面生长形成鲜明对比：小平面的二维形核生长速率随过冷度呈指数衰减，螺型位错台阶生长速率仅与过冷度的平方成正比，两类小平面生长方式所需的最小过冷度均远高于非小平面连续生长所需的极小动力学过冷度 [[S7,S28]]。

## 压铸工艺中的角色与应用

### 商用压铸合金为何表现为非小平面型生长
绝大多数商用压铸合金（铝基、锌基、镁基）因其金属键合特征具有较小的熔化熵（ΔSₘ < 2R，在共晶分类中熔化熵 < 23 J/(mol·K)），在 Jackson 判据下 α < 2，天然以非小平面粗糙界面模式进行生长 [[S19,S15,S37]]。

在高压压铸（HPDC）的极高冷却速率条件下，早期已形成的初生 α‑Al 或 α‑Mg 非小平面枝晶在料筒和充型阶段承受强烈剪切流动 [[S3]]。由于高冷速对应的大过冷度增加了界面过渡层厚度和粗糙度，合金的非小平面生长倾向进一步加强，界面粗糙化转变提前完成甚至原需较高过冷度才可发生的转变也被促进 [[S21,S30]]。

### 晶粒细化与组织演变
高冷速高压压铸条件下，非小平面生长模式可大幅提升形核速率，将常规铸造晶粒尺寸（0.5~1.0 mm 以上）显著细化为压铸典型晶粒尺寸（约 0.01 mm），无需额外添加晶粒细化剂即可获得极细晶组织 [[S35]]。

非小平面–非小平面共晶合金在正温度梯度条件下，随晶体生长速度/过冷度提升，组织可从平界面向胞状、树枝状形态依次转变，最终可形成粒状（等轴晶）共晶组织 [[S33]]。

当非小平面生长伴随晶粒细化时，柱状晶向等轴晶转变（CET）得到显著促进 [[S6,S24]]。晶粒形态从树枝晶向胞状晶乃至无枝晶形态转变的同时，决定糊状区渗透性的主导参数也由二次枝晶臂间距切换为晶粒平均尺寸 [[S6,S24]]。

### 糊状区特性与缺陷影响

#### 渗透率与补缩能力
对于非小平面枝晶组织，糊状区渗透率可在液相分数 0.1~0.4 范围内由 Kozeny‑Carman 经验公式准确描述；当液相占比进一步降低时，需通过数字重构真实凝固结构的数值模拟方法进行精准预测 [[S9]]。

非小平面生长形成的较粗大树枝晶结构可提供更宽的液相流动通道，提高糊状区整体渗透率，在相同凝固条件下有利于后续液相补入潜在裂纹区域，降低热裂倾向 [[S6,S24]]。

#### 晶粒尺寸的阈值效应
晶粒细化的影响并非单向线性：若非小平面生长伴随晶粒细化至尺寸小于 50 μm 的细小胞状晶或无枝晶形态，糊状区渗透率反而明显下降，可能提升合金的热裂敏感性；当晶粒尺寸大于 100 μm 时，无论枝晶或球状晶形态，细晶组织的热裂倾向普遍低于粗晶组织 [[S6,S24]]。

#### 压铸特有缺陷
高压压铸工艺的高冷速显著压缩传统非小平面凝固合金的糊状区宽度，最终凝固阶段施加的增压压力可驱动液相穿过糊状区完成补缩，降低宽凝固区间非小平面合金中常见的分散缩松比例 [[S35]]。

但在高剪切充型条件下，压铸镁合金等非小平面合金中易形成平行于铸件表皮分布的偏析带、孔隙带缺陷。这些带状缺陷的形成与糊状区流变特性、充型阶段非小平面初生相的剪切流动行为直接相关 [[S3,S17]]。

## 关键影响因素

影响非小平面型生长模式的关键参数包括：
1. **合金元素的本质特性**：熔化熵（ΔSₘ）是决定生长模式的最基础热力学参数 [[S19,S15]]
2. **界面取向因子**：不同晶面的 η/v 比值导致同一材料的不同晶面可呈现不同生长模式 [[S25,S22,S16]]
3. **过冷度与冷却速率**：高过冷度可增加界面过渡层厚度，促进粗糙化转变 [[S21,S30,S35]]
4. **生长方向与温度梯度**：温度梯度方向影响枝晶形态从平界面→胞状→树枝状→粒状转变的路径 [[S33]]
5. **溶质浓度分布**：合金元素在界面前沿的溶质再分配影响局部过冷度，间接参与粗糙化行为 [[S7,S30]]

## 微观组织表现

非小平面生长模式下可呈现以下典型凝固组织形态：
- **树枝晶**（dendritic）：等轴树枝晶或柱状树枝晶，尖端圆滑无棱角 [[S34,S37]]
- **胞状晶**（cellular）：在较高温度梯度与中等生长速度下形成 [[S33,S6]]
- **球状晶/粒状晶**（granular/globular）：在高生长速度、高形核率下最终形成的等轴晶形态 [[S33,S6]]

非小平面–非小平面共晶在凝固初期，领先 α 相以球形核心析出，后经 β 相在其侧面依附形核，两相通过交替搭桥的方式实现片层状或棒状的耦合共生生长 [[S33]]。

## 相关概念的关联与区别

- **与小平面型生长**：核心区分在于熔化熵阈值（ΔSₘ 与 2R 的关系）、Jackson 因子 α、原子占据模式、宏观形貌上的反向特征。两者均为结晶学的经典两极分类，但存在 α 值 2~5 的过渡区间以及过冷度驱动的粗糙化转变 [[S37,S21,S10]]。
- **与连续生长**：非小平面生长在动力学机制上完全对应连续生长模式 [[S4,S7,S13]]。
- **与伪共晶生长**：非小平面–非小平面共晶属对称型共晶共生区，两相生长动力学匹配良好；而小平面–非小平面共晶属非对称型共生区，易出现伪共晶组织偏离 [[S12]]。
- **与快速凝固中的无扩散生长**：在极高界面速度（> 1 m/s）条件下，固‑液界面局部热力学平衡失效，Jackson 模型的适用范围受限，需引入无扩散凝固理论进行独立处理 [[S13]]。
