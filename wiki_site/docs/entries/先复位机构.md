## 概述

先复位机构（又称预复位机构、提前复位机构）是压铸模具中一类关键的顺序动作组件，其核心定义为：在动、定模合模之前，驱动推出机构（顶出系统）预先退回到初始成形位置的专用机构，以实现提前消除干涉风险的工艺目标[[S6,S2]]。该机构在压铸领域对应的通用英文术语为 **pre-reset mechanism**（置信度极高）与 **early return mechanism (die casting mold)**（置信度高）；当侧重强制驱动方式时可称为 **forced return mechanism**（置信度高），侧重弹簧式组件时对应 **early return spring / return pin assembly**（置信度高），侧抽芯配套场景的专用细分术语为 **core pull return mechanism**（置信度中等）[[S3,S12]]。

其核心工艺作用为：在推出机构完成铸件推出后，确保顶杆/推杆等在合模动作启动前先行退回，避免推出元件与侧向抽芯的活动型芯发生运动轨迹交叉碰撞，或为嵌件安放操作留出无障碍空间[[S6]]。

## 分类与典型结构

先复位机构按动力源总体分为**液压先复位机构**与**机械先复位机构**两大类：前者依托压铸机自带的液压推出器，将模具推杆板与液压缸直连，通过电气液压控制系统按预设时序完成先复位；后者则通过纯机械联动驱动推出机构预回退，主流结构共 7 种[[S6,S5]]。

各类机械先复位机构的结构组成与适用场景见表 1。

## 工作原理

### 弹簧式先复位机构的工作过程

弹簧先复位机构利用弹簧的弹力使推出机构在合模之前进行预先复位。弹簧被压缩安装在推杆固定板与动模支承板之间，最常用形式是将 4 根弹簧套装在 4 根复位杆上，均匀分布于推杆固定板四周[[S6,S8]]。

开模阶段：压铸件随动模后退，斜销驱动侧抽芯完成后，动模继续后退至开模行程结束，压铸机顶杆顶动推板使推出机构执行推出动作，弹簧随之进一步压缩。合模阶段：压铸机顶杆脱离推板接触的瞬间，弹簧回复力驱动推杆迅速退回初始位置，在斜销尚未驱动侧型芯滑块复位之前即完成先复位，由此避免推杆与侧型芯的干涉碰撞[[S8]]。

需要特别指出，弹簧先复位机构中复位杆必须设置，用作推杆的最终精确复位定位元件[[S8]]。图 1 为该机构的典型结构示意。

### 摆杆式先复位机构的工作过程

合模过程中，安装在定模侧的预复位杆（或复位杆）推动摆杆端部滚轮，使摆杆绕其固定在支承板上的转轴做逆时针方向旋转，摆杆另一端压迫推板或推杆固定板，带动整个推出机构在合模前先行退回初始位置。为防止滚轮与推杆固定板接触面磨损，该处设有淬硬垫块[[S6,S8]]。

## 设计计算与判定

### 干涉判定

当推出元件与侧向抽芯机构的活动型芯在合模插芯过程中存在运动轨迹交叉时，需要进行系统性的干涉判定。判定公式为[[S9]]：

\[
e = (h - S) \cdot \tan\alpha
\]

其中：
- \(e\) ——推杆尚未复位时斜导柱提前驱动侧型芯插入的干涉距离（mm）
- \(h\) ——斜拉杆在开模方向上的伸出距离（mm）
- \(S\) ——推杆推出行程（mm）
- \(\alpha\) ——斜导柱斜角（°）

当 \(e > 0\) 时，表明必然发生插芯与推杆的碰撞干涉，必须配置先复位机构[[S9]]。

### 先复位行程计算

对于摆板式与三角滑块式先复位机构，所需先复位距离按以下公式计算[[S4]]：

\[
s_{\text{先}} = s' \cdot \cot\alpha + K
\]

其中：
- \(s_{\text{先}}\) ——先复位距离（mm）
- \(s'\) ——活动型芯起始干扰位置与插芯终止位置的距离（mm）
- \(\alpha\) ——斜销斜角（°）
- \(K\) ——安全余量，常规推荐取值范围 5～10 mm[[S4]]

## 常见故障与安全设计

### 弹簧式机构的典型故障

弹簧长期承受交变载荷，易发生疲劳断裂，导致复位动力不足、出现复位不到位的问题；此外，弹簧式先复位机构无法单独依靠弹簧实现绝对可靠复位，必须配套常规复位杆完成最终精确定位，生产中需定期检查更换弹簧[[S6]]。

### 行程开关安全联锁

为防止先复位失效后误操作强行合模导致推杆与侧抽芯机构碰撞损坏，可在模具上加装行程开关，将顶针板的复位到位信号与压铸机合模回路串联。仅当推出机构完全复位并触发行程开关后，压铸机才允许执行合模动作[[S7]]。

## 应用场景

先复位机构主要适用于以下两类核心场景[[S6,S9]]：

1. **侧向抽芯干涉场景**：推出元件与活动型芯的合模运动轨迹相交，若按常规顺序合模，侧型芯将在推杆尚未退回时插入，导致插芯碰撞干涉。
2. **嵌件安装场景**：推杆完成铸件推出后停留在嵌件安放区域，需要先复位动作将推杆退回，留出嵌件安放空间，避免人工放置嵌件时与推杆发生干涉。

## 与标准复位机构的关系

先复位机构并非取代常规复位机构，而是在其基础上增设的预动作环节。采用先复位的推出机构仍需配置复位元件（复位杆）和限位元件（限位钉等），以保证合模状态下的最终精确复位位置[[S2,S9]]。复位杆的结构型式、尺寸系列、材料要求与公差精度执行现行国家标准 GB/T 4678.12—2018《压铸模 零件 第12部分：复位杆》，该标准替代旧版 GB/T 4678.12—2003，推荐材料由 T8A、T10A 调整为 GCr15、20Cr、T10A[[S10]]。
