## 概述

高强度灰铸铁是灰铸铁材料中的一个高性能分支，属于通过充分孕育处理获得的孕育铸铁[[S12,S19]]。在GB/T 9439标准体系中，通常将HT250、HT275、HT300及HT350四个牌号划归为高强度灰铸铁[[S17]]。其基体组织以珠光体为主体，石墨呈细小均匀的A型片状分布，与低牌号铁素体或铁素体-珠光体灰铸铁形成鲜明区分—后者抗拉强度低于200 MPa，石墨形态较为粗大[[S12,S19]]。

灰铸铁在中国乃至全球铸件产量中占据主导地位。2023年中国各类材质铸件总产量中，灰铸铁占比达40.3%，为产量占比最高的铸造合金品类，高强度灰铸铁则是该品类中面向高载荷工况的核心材料分支[[S21]]。

## 牌号与标准体系

### 国内标准（GB/T 9439）

GB/T 9439—2009规定了灰铸铁的八个牌号，高强度灰铸铁涵盖其中四个牌号[[S17]]：

| 牌号 | 说明 |
|------|------|
| HT250 | 最小抗拉强度 250 MPa |
| HT275 | 最小抗拉强度 275 MPa |
| HT300 | 最小抗拉强度 300 MPa |
| HT350 | 最小抗拉强度 350 MPa |

其中HT225和HT275是相对1988版标准新增的牌号[[S17]]。

### 国际标准对应关系

以下为高强度灰铸铁在主要国际标准中的牌号对应关系[[S5,S11]]：

| GB/T 9439 | ISO 185 | ASTM A48 |
|-----------|---------|----------|
| HT250 | 250 | Class 40 |
| HT275 | 275 | Class 45 |
| HT300 | 300 | Class 50 |
| HT350 | 350 | Class 55 |

## 化学成分与合金化

高强度灰铸铁典型的化学成分范围为：w(C)=2.7%～3.6%，w(Si)=1.2%～1.8%，w(Mn)=0.6%～1.4%，w(P)≤0.3%，w(S)≤0.15%[[S19,S8,S20]]。

为获得高强度所需的100%细小珠光体基体，常需添加合金元素。生产实践中常采用Cr+Mo+Cu、Cu+Mo+Ni、Cu+Mo、Cu+Cr、Cu+Sn、Cu+V等组合[[S8]]。低熔点金属Sn（ω(Sn)=0.03%～0.1%）或Sb（ω(Sb)=0.006%～0.01%）可在石墨/金属界面形成薄层，阻碍碳向石墨扩散，从而阻止铁素体生成，有利于获得全珠光体基体，其中Sn添加量以0.02%～0.04%为较优范围[[S8]]。

## 金相组织特征

高强度灰铸铁的典型金相组织为细小均匀分布的A型片状石墨，基体中珠光体体积分数达80%～100%，无粗大的D/E型过冷石墨，符合GB/T 7216相关分类要求[[S18,S7]]。

## 制造工艺特点

### 熔炼控制

铁液过热温度一般控制在1500～1550℃，遵循“高温出炉、低温浇注”的原则[[S18]]。适当提高过热温度可细化石墨及基体组织，提高铁液纯净度；但超过约1500℃后，石墨数量和化合碳的变化趋势将发生逆转[[S18]]。化学成分控制精度要求严格，例如Δω(C)≤±0.05%，Δω(Si)≤±0.10%[[S8]]。

### 孕育处理

孕育处理是高强度灰铸铁制备的核心环节，旨在降低铁液过冷倾向，促使石墨按A型分布析出，同时细化共晶团[[S18]]。典型工艺要点包括[[S7]]：

- 孕育剂选择：硅钡、硅锶等复合孕育剂效果较理想，钡（Ba）具有抗衰退能力并提高A型石墨占有率，锶（Sr）具有强消除白口能力
- 孕育方式：多次随流复合孕育可明显改善石墨分布均匀程度
- 时限控制：孕育处理后的铁液应在8 min内浇注完毕，包内二次孕育后3～5 min孕育效果最佳

### 合金化策略

通过合金化可获得100%细小珠光体基体，同时可适当提高碳当量而不致降低强度，有利于减小白口倾向、改善铸造性能[[S8]]。

## 在压铸中的潜在关联

高强度灰铸铁在压铸工艺中的适用场景受到其热物理性能的显著制约，具体表现为以下两方面[[S9,S15]]：

**可应用场景**：
- 压铸机机身、重型机架
- 模架承重模板
- 压铸辅助工装底座

上述应用主要利用高强度灰铸铁的高刚度与高阻尼减振特性，可有效吸收压铸过程的冲击振动。

**不可应用场景**：
- 压铸成形工件的型腔材料
- 铝合金高压压铸直接生产的形态件本体材料

限制原因在于高强度灰铸铁的热疲劳性能较差，抗极端循环热冲击能力不足，频繁温度交变工况下易产生热疲劳裂纹[[S15]]。

## 典型应用领域

高强度灰铸铁广泛应用于对静刚度、减振性能有较高要求的结构件，包括[[S17,S14]]：

- 发动机缸体、缸盖
- 液压阀体
- 机床床身、工作台、卡盘
- 内燃机气缸体、泵体、壳体

## 局限性

高强度灰铸铁的主要使用限制包括[[S6,S20,S4]]：

- **塑性与冲击韧性极低**：延伸率普遍低于1%，基本不具备抗冲击和抗爆性能
- **焊接性差**：焊接热影响区易产生白口组织，导致裂纹缺陷；适当焊后热处理可在一定程度上减轻此效应
- **抗热疲劳能力不足**：在频繁温度交变工况下易产生热疲劳裂纹
- **加工难度随强度升高而增加**：ASTM A48中Class 40及以上牌号（对应高强度灰铸铁）通常加工难度更大，阻尼能力相对较低
- **工艺控制精度要求高**：对化学成分和孕育工艺的控制要求严格，生产过程容差窗口较窄
