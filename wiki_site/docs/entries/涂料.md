## 概述

在压铸技术体系中，“涂料”一词通常指 **压铸涂料**，即压铸过程中对模具型腔、型芯表面、滑块、推出元件、压铸机的冲头和压室等部位喷涂的润滑材料与稀释剂的混合物。其核心作用是为压铸合金与模具之间提供有效的隔离保护层，辅助压铸件顺利从模具中取出，保证产品完整性和后续加工性 [[S45,S51,S33]]。

## 定义与别名

### 中文常用名
- **压铸涂料**（标准术语）
- **脱模剂**（压铸用脱模剂）
- **模具涂料** [[S9,S45]]

以上名称均属于压铸工艺体系下的专用辅助材料范畴，在行业文献与技术手册中通用。

### 英文对应术语

目前压铸行业内上述术语均被使用，尚未形成广泛通用的统一英文缩写 [[S21,S56,S2]]。

### 广义与狭义区分

在铸造领域，“涂料”一词存在广义与狭义之分：
- **狭义（本词条核心）**：指压铸模具涂料，涂覆对象为金属模具型腔，核心组分为润滑/脱模成分，功能为提供隔离与润滑。
- **广义**：涵盖传统砂型铸造涂料，涂覆对象为砂型/砂芯表面，核心组分为耐火骨料（如石墨、锆英粉），功能以防止粘砂和改善铸件表面光洁度为主 [[S17,S5,S55]]。

本词条以压铸涂料为核心进行阐述。

## 分类体系

### 按载体类型分类

| 类型 | 特征 | 现状 |
|---|---|---|
| 油基/溶剂基涂料 | 以石油类矿物油、动植物油脂、油基石墨、蜂蜡等为代表 | 成本高、污染大、易燃，使用占比逐渐降低 [[S33]] |
| 水基涂料 | 以水为载体，矿物油/石蜡等乳化制成 | 当前行业主流应用类型 [[S33,S29]] |
| 粉末涂料 | 以空气为分散载体 | 环保性好、使用效率高，近年来市场需求年增速达10% [[S33]] |

### 按功能类型分类

| 功能类型 | 核心作用 |
|---|---|
| 润滑型 | 降低金属液与模具界面的摩擦系数 |
| 隔热型 | 调节模具界面热传导速率，平衡温度场 |
| 保护型 | 隔离金属液对模具的直接冲蚀、腐蚀 |
| 促进充型型 | 保持金属液流动性，提升薄壁复杂件的成型能力 |

来源：[[S51,S53]]

| 成分类型 | 典型组分 | 特点 |
|---|---|---|
| 石墨基 | 鳞片石墨、胶体石墨 | 脱模性优异，但易造成铸件表面发黑 [[S33,S24]] |

| 蜡系 | 合成蜡、天然蜡 | 脱模性好，化学稳定性持续改进中 [[S56]] |

## 组成与配方

### 典型有效组分

| 组分类型 | 代表物质 | 功能 |
|---|---|---|
| 固体润滑剂 | 鳞片石墨、胶体石墨 | 层状滑移结构提供界面润滑 [[S33,S24]] |
| 高温脱模填料 | 六方氮化硼（h-BN） | 耐高温惰性，防粘，900°C下铝液接触角达160° [[S32,S47,S46]] |
| 耐火填料 | 氧化铝（Al₂O₃） | 提高涂层强度与耐温性 |
| 粘结剂 | 硅酸盐 | 保证涂层附着与成膜完整性 |
| 表面活性剂 | — | 改善润湿性，促进均匀铺展 |
| 溶剂/稀释剂 | 水（水基）、有机溶剂（油基） | 承载有效成分，调节涂敷性 |

### 水基涂料基础配方

常规水基压铸涂料的基础组分为矿物油、石蜡，辅以十多种添加剂经乳化制成。配方敏感性极高：任意一种原料配方变更1%就会导致成品性能出现明显差异。面向需要后续表面涂装处理的压铸件，可选用不含硅、蜡的专用组分体系 [[S29,S26]]。

### 组分演变趋势

近20年压铸水基润滑剂的组分呈现显著演变趋势：从早期含约10%油分的溶剂型体系，逐步发展为当前固含量低于0.2%的水基体系 [[S56,S41]]。

## 核心功能

压铸涂料的核心功能可归纳为以下五项：

### 1. 脱模润滑

极性化学键与模具表面结合形成再生吸附薄膜，聚硅氧烷分子在模具表面取向排列，表面烷基密集覆盖形成低摩擦界面，减少压铸件与模具成型部分（尤其是型芯）之间的摩擦阻力，保障顺利顶出脱模 [[S1,S45]]。

### 2. 隔热控温

在金属液与模具之间形成低导热系数的隔离层，降低模具界面导热率，平衡模具各部位温度场分布，降低模具整体热冲击幅度 [[S51,S54]]。

### 3. 防止粘模/焊合

依靠形成的连续隔离膜层完全阻隔高温液态合金与模具钢表面的直接接触，避免铝、镁等活性合金在高温高压下与H13等模具钢发生润湿、熔焊粘黏，可大幅降低铝合金压铸的粘模故障概率 [[S24,S43]]。

### 4. 提高铸件表面质量

避免金属液直接冲刷型腔表面带来的划痕、麻点，减少铸件表面粥状花纹、发黑等缺陷；同时可细化铸件表层晶粒，增加细晶层厚度，降低微气孔缺陷概率，提升压铸件气密性 [[S43,S29]]。

### 5. 延长模具寿命

避免高温金属液直接冲刷型腔和型芯表面，减少热疲劳裂纹生成的诱因，隔离金属液对模具的热腐蚀作用，正常工况下可使压铸模使用寿命得到显著提升 [[S51,S49]]。

## 关键性能参数

### 稀释比例

| 合金/铸件类型 | 推荐稀释比（水:浓缩液） | 来源 |
|---|---|---|
| 铝合金铸件（常规） | 约30:1 | [[S8]] |
| 小型锌合金零件 | 最高可达100:1 | [[S8]] |
| 行业平均 | 约40:1 | [[S8]] |
| 国内水基涂料生产推荐 | 兑水100～150倍 | [[S3,S10]] |
| 西欧二氧化硅类水基涂料 | 20～30倍水稀释 | [[S49]] |
| 美国苯基甲基硅酮类乳化液 | 20～30倍水稀释 | [[S49]] |

稀释倍数选择原则：大件、薄壁、复杂件选择更低稀释倍数（浓度较高），小件、简单件选择更高稀释倍数 [[S10]]。

### 模具温度窗口

| 温度区间 | 现象 | 影响 |
|---|---|---|
| 低于150°C | 涂料无法附着，从表面流淌 | 无法形成有效涂层 [[S10]] |
| 180～200°C | 常规适配喷涂温度区间 | 正常成膜 [[S10]] |
| 低于270°C | 喷涂1s内即可形成均匀皮膜 | 理想成膜条件 [[S10]] |
| 超过398°C | 触发莱顿弗罗斯特效应，液滴汽化反弹 | 无法润湿成膜 [[S10,S20]] |

### 喷涂工艺参数

| 参数 | 推荐值/范围 |
|---|---|
| 喷射距离 | 100～200mm [[S10]] |
| 喷涂时长 | 0.1～2.0s [[S10]] |
| 涂层干膜厚度 | 100～300μm（常规型腔），200μm为常用规格 [[S18,S44]] |

### 热安全性能

水基涂料载体蒸发后残留的润滑组分闪点普遍超过200°C，相比传统溶剂型涂料（基础载体闪点约36°C），无明火风险，适配全封闭自动压铸机生产场景 [[S3,S41]]。

### 不同压铸合金模具预热温度

| 合金类型 | 模具预热温度区间 |
|---|---|
| 锌合金 | 150～200°C |
| 铝合金 | 200～300°C |
| 铜合金 | 250～300°C |
| 镁合金 | 需高于200°C |

来源：[[S13]]

### 涂料核心技术要求

压铸涂料需满足以下7项核心技术要求 [[S52,S51,S34]]：

1. 300～400°C高温下保持良好润滑性
2. 稀释剂在100～150°C可快速挥发
3. 涂覆性能好
4. 对模具和压铸件无腐蚀
5. 存放过程中性能稳定，不稠化
6. 高温下不分解有害气体
7. 不会在型腔表面生成积垢

## 涂敷工艺

### 涂覆方法分类

压铸涂料的涂覆方法涵盖流体涂料与粉末涂料两大范畴。流体涂料可采用手工刷涂、手动喷涂、自动喷涂等方式；粉末涂料则通过静电喷涂或流化床等工艺实现 [[S35]]。

### 各类涂覆工艺特征

| 工艺类型 | 特征 | 优缺点 |
|---|---|---|
| 手工刷涂 | 适配糊状或膏状高粘度涂料，刷后用压缩空气吹匀 | 操作灵活、用量低；涂层均匀性差 [[S30]] |
| 手动空气喷涂 | 靠压缩空气使涂料雾化后喷射到型腔表面 | 操作简便、成本低；涂层质量受操作经验影响大，易局部不均 [[S10,S30]] |
| 全自动喷涂系统 | 通过可编程机械手或往复机构执行固定喷涂路径，可针对深肋、长型芯等区域精准分配喷涂量 | 涂层均匀度大幅提升，适配大批量自动化生产；微量喷涂系统可实现无载体纯浓缩脱模剂的精准定体积喷涂，减少废水量与涂料消耗 [[S8]] |

### 静电喷涂的局限性

静电喷涂在压铸领域未实现大规模推广，原因如下：
- 水基压铸涂料本身具备导电性，静电喷涂需额外配置昂贵适配设备
- 压铸高温区域表面易形成蒸汽层，排斥带电涂料液滴，导致涂料大量沉积在低温非工作区域，无法实现均匀成膜
- 压铸工艺要求的极薄涂层无法实现钢模表面的电绝缘，易出现涂料异常堆积 [[S27,S42]]

### 涂层表面质量控制要求

- 涂层需均匀，无局部过厚堆积
- 型腔转折、凹角位置避免涂料沉积，防止铸件轮廓不清
- 喷涂完成后必须清理压铸模排气通道，避免涂料堵塞排气道造成铸件卷气缺陷 [[S11,S49]]

## 常见缺陷与失效机制

### 涂料相关铸件缺陷

| 缺陷类型 | 成因 | 后果 |
|---|---|---|
| 涂层堆积致气孔 | 浓度过高，涂层脱落进入金属液 | 铸件内部气孔 [[S29]] |
| 涂层厚度不足致粘模 | 稀释过度，涂层不足以抵抗热应力冲刷 | 脱模不顺、粘膜 [[S29]] |
| 脱模性能不足致铸件变形开裂 | 铸件与型腔粘连，顶出阻力大 | 顶杆顶穿铸件、动模损坏 [[S31]] |
| 铸件表面斑纹 | 涂料使用不当或用量过多 | 外观不符合验收要求，导致产品报废 [[S38]] |
| 排气道堵塞 | 多余涂料堵塞排气通道 | 卷气、气孔缺陷 [[S29,S11]] |
| 型腔表面积聚坚硬沉积物 | 涂料再生沉积，仅能用化学溶剂或打磨清除 | 铸件尺寸超差 [[S31]] |

### 涂层工作寿命

压铸单批次涂层的有效工作寿命有限：
- 冲头和压室部位：每刷涂/喷涂1次可连续压铸3～5次
- 型腔、浇道部位：每刷涂/喷涂1次可连续压铸3～8次
- 大型压铸件：通常每压铸1次就需重新喷涂一次涂料 [[S19,S16]]

## 局限性

### 环境与健康问题

传统油基/溶剂基涂料存在显著的VOCs排放、有机溶剂危害和易燃风险。尽管水基涂料的推广已大幅减少火灾危险和污染，但喷涂过程中产生的雾气仍需有效收集与处理 [[S41]]。废弃涂料产品的排放需符合当地环保规定 [[S41]]。

### 涂层均匀性控制

压铸涂料喷涂均匀性控制难度较高，实际生产中需通过控制稀释比例、喷涂距离、喷涂角度等参数，避免涂层在模具转折、凹角位置沉积 [[S51,S34]]。

### 模具维护要求

涂料在型腔表面积聚形成的坚硬再生沉积物难以清除，仅能通过化学溶剂溶解或人工打磨的方式彻底清理 [[S31]]。

## 与砂型铸造涂料、消失模涂料的区别

| 对比维度 | 压铸涂料 | 砂型铸造涂料 | 消失模铸造涂料 |
|---|---|---|---|
| 涂覆对象 | 金属模具型腔表面 | 砂型/砂芯表面 | EPS泡沫塑料模样表面 |
| 核心组分 | 润滑/脱模成分（矿物油、蜡、石墨、氮化硼等） | 耐火骨料（石墨、锆英粉等）+粘结剂 | 高耐火度填料+高透气性粘结剂 |
| 核心功能 | 脱模润滑、隔热隔离、防粘模焊合 | 防粘砂、改善铸件表面光洁度 | 防止金属液渗入干砂、提高模样刚度、导出热解气体 |
| 涂层性质 | 临时消耗性涂层，每次/数次压铸后重涂 | 占位涂层，一次性使用 | 不占位涂层，随模样气化 |
