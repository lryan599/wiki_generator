## 概述

脱模剂配送装置是压铸单元喷涂系统的核心组成单元，在国家标准中对应英文术语为 **mixing carrier for parting agent**，其核心功能为将脱模剂按特定比例与水进行混合，并压送到喷涂机的装置 [[S11]]。

在行业实践中，该设备有一系列通用的中文别名，包括脱模剂自动稀释压送机（自动压送机）、脱模剂自动喷涂系统、模具喷雾机及脱模剂喷淋装置等；在北美压铸行业，对应的通用英文称谓包括 die spray system、die lube spray system 以及 automatic die sprayer [[S7]]。需要指出，上述部分别名的外延常包含下游的雾化喷涂执行机构，但在本文按国家标准规范的含义范围内，**脱模剂配送装置**特指完成脱模剂定比混合与压力输送的功能单元，与下游的自动喷雾机（雾化喷涂执行单元）共同构成完整的压铸喷涂子系统 [[S23]]。

在压铸单元的生产时序与功能边界方面，脱模剂配送装置与浇注装置（给汤机）及取件传送装置（取件机）完全独立。给汤机的核心功能是将熔融金属从保温炉定量转运至压射室，面向高温液态金属；取件机的核心功能是将凝固后的铸件从模具型腔中抓取并转运至指定位置，面向固态铸件；而脱模剂配送装置不接触实体工件与高温金属液，仅完成常温液态润滑介质的配比与输送功能，三者在自动化压铸单元中属于功能互不重叠的独立工位 [[S11]]。

## 定义

依据国家标准 **GB/T 37371—2019《压铸单元 术语》**，脱模剂配送装置的定义为：

> 将脱模剂按特定比例与水进行混合，并压送到喷涂机的装置。 [[S11]]

该标准同时定义压铸喷涂系统为将压铸脱模剂配比、雾化，定点定量喷涂在模具型腔表面的功能系统，其中脱模剂配送装置是承担配比与压送功能的配套单元 [[S11]]。

另一项国家标准 **GB/T 25370—2020《铸造机械 术语》** 中，定义了功能等价的术语——脱模剂自动配比压送机，其英文对应术语为 releasing agent mixer 或 releasing agent dilutor，功能描述为：将脱模剂按设定比例与水进行混合并压送到自动喷雾机的装置 [[S23]]。该标准同时定义了自动喷雾机为利用压缩空气将脱模剂雾化后定量均匀地喷涂在模具各指定点和表面的装置 [[S23]]。以上两项标准清晰界定了脱模剂配送装置在喷涂子系统中的配比输送角色，与末端雾化喷涂单元的职能分工关系。

## 结构与组成

一套完整的脱模剂配送系统通常包含以下核心功能单元：

- **储液单元**：存储高浓度脱模剂原液、工艺用水及预混合后的稀释工作液。配置液位检测组件，可在稀释液液面低于下限时自动触发补液流程，无需停止喷涂作业即可完成补充 [[S24]]。
- **动力泵**：提供稳定的输送动力，将混合后的脱模剂工作液以指定压力输送至喷涂支路。行业常见气动泵配置的输出压力可在 0.2 MPa ~ 0.5 MPa 区间自由调节，达到设定压力后可自动停机以降低能耗 [[S20]]。
- **混合配比阀**：通过高精度计量组件精准控制脱模剂原液与稀释水的流量比例，实现预设倍率的自动配比。部分高端机型配比控制精度可达 ±1%，以保障不同批次喷涂效果的一致性 [[S12]]。
- **输送管路**：连通混合单元与末端喷涂装置，输送脱模剂混合液与雾化压缩空气。配套过滤器可滤除管路内杂质以避免喷嘴堵塞，全管路配置压力检测组件实时监控工作状态 [[S2]]。
- **雾化喷嘴/喷枪**：分为扇形喷嘴与通用雾化喷嘴两类——扇形喷嘴保证大面积型腔喷涂均匀性，通用雾化喷嘴适配深腔复杂结构区域。喷嘴结构分为吹气支路与喷涂支路，可分别完成型腔清渣与脱模剂覆盖作业 [[S6]]。
- **PLC 控制模块**：整套喷涂系统的核心控制单元，支持参数自定义编辑，可接入压铸机主机信号实现喷涂工序与压铸全流程联动控制，内置反馈程序可根据模温传感器实测值动态调整喷涂时长 [[S10]]。
- **执行支架（喷涂机械臂/运动模组）**：搭载喷枪组件完成预设路径运动。部分机型支持喷枪 90° 自动旋转动作，可实现多角度伸入动定模间隙完成双侧型腔同步覆盖，高精度机型重复定位精度可达 ±0.1 mm [[S24]][[S6]]。

## 工作原理

脱模剂配送系统在压铸循环中的完整时序流程如下 [[S19]][[S24]][[S2]]：

1. 原液加注完成后接入水源与气源。
2. 在控制系统界面设定预设稀释倍率，启动自动配比程序。
3. 当稀释液罐液位低于下限时自动完成补液动作，无需停止喷涂作业。
4. PLC 接收压铸机开模到位信号后触发喷涂动作，执行支架带动喷枪进入动定模之间。
5. 依次执行：下降 → 吹气清理型腔残留铝屑 → 喷涂定模 → 喷涂动模 → 二次吹气吹走多余积液 → 抬升复位 → 等待下一周期指令。

控制逻辑支持自定义喷涂模式，可分别设置喷涂 1、喷涂 2 的独立运行时间与运行路径。配置模具温度传感器后，系统可根据实测模温自动修正喷涂时长，实现差异化冷却效果 [[S19]][[S11]]。

## 分类

### 按自动化程度

| 类型 | 特征 | 适用场景 |
|------|------|----------|
| **手动喷涂** | 依靠人工操作完成脱模剂配比和喷涂作业，灵活性高、设备投入成本低，但喷涂均匀性高度依赖操作人员技能，易出现喷涂位置偏差、局部堆积或覆盖不全的问题。 | 小批量多品种、结构简单的低吨位压铸机生产。[[S15]] |
| **固定式自动喷涂** | 喷涂头固定在压铸机定模板/动模板旁，由压铸机控制系统触发固定路径的喷涂动作，结构紧凑、动作稳定无干涉风险，仅适配单一特定模具，无需额外占用压铸机周边空间。 | 无深腔的简单模具场景。[[S9]][[S2]] |
| **机器人自动喷涂** | 基于多轴机械臂（六轴重复定位精度可达 ±0.1 mm），柔性高，可依据不同模具结构灵活编辑喷涂路径与姿态。 | 复杂型腔、多品种生产场景。[[S6]] |

### 按雾化方式

| 类型 | 特征 |
|------|------|
| **气动雾化** | 利用压缩空气将水基脱模剂雾化，为行业最常规的雾化方式，工作压力推荐区间为 0.3 ~ 0.6 MPa。[[S6]] |
| **微量喷涂** | 仅输出极少量雾化脱模剂，适配高真空压铸、薄壁精密件生产，可显著降低脱模剂引发的铸件气孔缺陷。验证数据显示匹配优化的喷涂工艺可将真空阀故障发生率降低约 5.9%。[[S18]] |
| **无气/粉末喷涂** | 采用真空吸引法将粉末脱模剂吸附在模具表面，无废液排放，可大幅降低脱模剂消耗量，适配镁合金等特殊压铸场景，系统包含封闭腔室、排气结构及粉末回收模块。[[S4]][[S26]] |

### 按脱模剂供应方式

| 类型 | 特征 |
|------|------|
| **单点配比** | 单台压铸机单独配置一台稀释压送装置，稀释倍率 40 ~ 300 倍，独立设置配比参数，适配不同产品差异化脱模剂使用要求，单台设备投入成本低。[[S22]] |
| **集中供液系统** | 单台大容量供液主机可同时为多台压铸机喷涂单元输送配比后的脱模剂，配比一致性高，人工干预少，适合大批量连续化生产的大型压铸车间使用。[[S20]][[S5]] |

## 关键过程参数

以下为脱模剂喷涂工艺中的核心参数及其推荐范围。

**稀释倍率**：水基脱模剂标准稀释倍率范围为 40 ~ 300 倍，铝合金压铸常规使用工况推荐体积配比为 1:80 ~ 1:100。薄壁复杂件稀释倍率可适当降低，简单小零件可采用更高稀释倍率 [[S22]][[S20]]。

**喷涂压力与流量**：行业推荐喷涂工作压力控制区间为 0.3 ~ 0.6 MPa，该区间下脱模剂雾化均匀度最优，对应流量控制精度可达到 ±5%，可保障模具表面成膜均匀性 [[S6]]。

**喷涂时间**：推荐单模单次喷涂时长范围为 0.1 ~ 0.2 s，常规生产过程中可根据模具表面积与型腔复杂程度做对应调整 [[S3]]。

**喷涂距离**：行业通用合理喷涂距离区间为 100 ~ 250 mm。距离过小将导致脱模剂反弹流失，距离过大会使雾滴融合为大液滴破坏表面成膜均匀性 [[S9]][[S6]][[S3]]。

**模具温度匹配**：适配水基脱模剂的合理模具工作温度区间为 150 ~ 300 ℃。模温过低将导致脱模剂无法附着形成有效薄膜，还会残留大量水汽使铸件产生气孔；模温过高将导致脱模剂接触表面后快速汽化，无法完成有效润湿成膜 [[S6]][[S17]][[S3]]。

**喷涂角度**：喷嘴喷射方向与模具型腔表面垂直偏差建议 ≤ ±10°，以避免飞溅和盲区 [[S6]]。

## 选型要点

脱模剂配送装置的选型应着重考虑以下性能指标。

**喷涂均匀性**：喷涂距离应控制在 100 ~ 250 mm 区间，喷涂角度与模具型腔表面垂直偏差 ≤ ±10°，覆盖无盲区。局部喷涂差异导致的模具表面温度梯度应 ≤ 50 ℃，以避免局部过热引发的模具龟裂 [[S15]]。

**雾化效果**：雾滴粒径应均匀细腻，可快速在模具表面成膜，避免出现大液滴堆积。工作气压和脱模剂压力需稳定维持在 5 ~ 6 kg/cm²，适配各类脱模剂的雾化要求 [[S21]][[S9]]。

**与压铸机联动兼容性**：喷涂单元需可与压铸机信号端对接，实现「开模到位 → 取件完成 → 喷涂启动 → 喷涂结束 → 发送合模信号」的联锁逻辑，且须具备喷涂头运动路径安全互锁功能，以避免与取件臂、压铸机模板发生碰撞干涉 [[S21]][[S8]]。

**维护便利性**：优先选用模块化结构、易损件通用化程度高的机型。气动元件故障率低且更换便捷，设备应自带故障代码显示功能，以降低停机维修时长 [[S27]]。

## 常见问题与维护

| 故障现象 | 诱因 | 排查与应对 |
|----------|------|-------------|
| **喷嘴堵塞** | 脱模剂原液析出杂质、管路残留铝屑、长期使用形成的沉积垢层。 | 拆解喷嘴检查滤网状态，定期使用通针清理；每批次生产结束后使用清水循环冲洗管路，避免残留液干结堵孔。[[S15]] |
| **混合比例不均** | 配比计量泵精度漂移、原液液位检测失效、管路混入空气。 | 定期校准配比参数，确认原液输入管路无泄漏；每生产班次抽检稀释后脱模剂浓度，保证配比偏差控制在要求范围内。[[S20]] |
| **管路泄漏** | 管接头松动、密封圈老化。 | 定期目视检查所有管路连接部位，每半年紧固一次全部接头；承压管路选用适配工况的氟橡胶密封圈，避免液体泄漏影响喷涂压力稳定性。[[S16]] |
| **喷涂动作时序异常** | PLC 控制程序错乱、传感器信号丢失、安全联锁条件不满足。 | 依次校验喷涂单元与压铸机、取件机的交互信号，确认喷涂启动的前置安全条件全部到位；定期测试互锁逻辑有效性。[[S21]][[S8]] |

### 喷涂效果与铸件质量的关联

喷涂不足区域润滑膜缺失，直接导致铸件粘模、表面拉伤；脱模剂过量喷涂或局部堆积，会大幅提升型腔内发气量，引发铸件内部气孔、表面流痕缺陷。实测数据显示，喷涂均匀性提升 20% 可直接将铸件表面不良率从 3.2% 降至 0.8% [[S15]]。

### 定期维护规范

应按照设备手册明确的周期，对气动元件、运动导轨、计量泵、管路滤网进行分级点检更换，建立预防性维护台账，对关键易损件明确使用寿命，实现故障前更换，以大幅降低非计划停机时长 [[S21]][[S13]]。

## 相关关系

脱模剂配送装置在压铸单元内与其他周边设备的联动关系如下：

- **与给汤机（浇注装置）**：功能边界完全独立。给汤机负责将保温炉内熔融金属定量转运至压射室，脱模剂配送装置负责脱模剂配比与输送，二者在压铸生产时序中属于完全不重叠的独立工位 [[S11]]。
- **与取件机（取件传送装置）**：功能边界完全独立。取件机负责将已凝固的压铸件从模具型腔中抓取并转运，脱模剂配送装置不接触实体工件，仅完成润滑介质的配比输送功能 [[S11]]。
- **与自动喷雾机**：脱模剂配送装置与自动喷雾机构成上下游功能链。前者完成脱模剂的定比混合与压力输送，后者负责将混合后的介质雾化，定点、定量均匀喷涂到模具指定表面 [[S23]][[S11]]。
