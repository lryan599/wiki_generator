## 概述

在中文铸造术语体系中，“沸腾期”一词在不同工艺背景下具有两种截然不同的含义。**铸钢熔炼领域**的“沸腾期”是国家标准中规定的正式工艺阶段名称，特指氧化熔炼法中的氧化期，属于正常且必要的工序[[S16,S18]]。而在**有色合金铸造（含压铸）领域**，“沸腾期”并非标准化生产工序，而是指由于水分、气体剧烈汽化爆发，导致铸件内部形成“沸腾孔（气穿孔）”等缺陷的异常过程，本质上是缺陷诱发的气体爆发阶段[[S11,S25]]。两种用法所指的物理本质、发生环境及工艺后果完全不同，需要在具体语境中加以辨别。

## 标准术语：铸钢熔炼中的沸腾期（氧化期）

### 定义
根据《中国机械工业标准汇编 铸造卷 上》及《机械基础制造工艺标准汇编 铸造 上》，“沸腾期”即“氧化期”，是在铸钢氧化熔炼法中，向熔池吹氧或加入铁矿石，使硅、锰、磷、碳等元素发生氧化反应的阶段，用于脱除钢液中的气体、碳磷杂质和非金属夹杂物，并提高钢液温度[[S18,S4,S16]]。该术语对应的英文为 **oxidizing stage** 或 **boil stage**[[S18]]。

### 工艺作用
沸腾期对钢液质量具有决定性影响，主要体现在以下方面[[S21]]：
- **脱气**：剧烈沸腾促使溶解于钢液中的氢、氮等气体排出；
- **去除杂质**：非金属夹杂物及氧化产物被排入炉渣；
- **均匀化**：熔池受到强烈搅拌，使钢水成分和温度趋于均匀。

钢液氢含量与沸腾期脱碳量的关系曲线直观地印证了其脱气效果[[S2]]。

### 工序位置与操作条件
在氧化熔炼法中，沸腾期发生在炉料完全熔清之后[[S9]]。操作上通常分两个阶段进行：
- **矿石沸腾期**：加入铁矿石，引发熔池激烈沸腾，强力脱碳去气；
- **纯沸腾期**：停止加矿后的一段持续沸腾阶段，继续净化钢液[[S9]]。

脱碳速度是沸腾期的重要控制参数，一般维持在 **0.25 %/h ~ 0.5 %/h** 的水平，部分工艺在不同阶段采用不同的降碳速率（如 0.9%/h、0.72%/h、0.36%/h）[[S24]]。提高钢水温度或向熔池吹氧均能有效加快脱碳速度[[S21]]。

需要指出，采用**不氧化熔炼法**时不经历沸腾期，因此对炉料清洁度（含水、锈、硫、磷等）要求更为严格[[S21]]。

## 工艺缺陷：有色合金与压铸中的沸腾期

### 术语界定
在铝合金、镁合金等有色合金的砂型铸造及压铸领域中，“沸腾期”不是公认的标准生产阶段，而是被用来描述**沸腾孔（气穿孔）缺陷的形成过程**。相关文献在分析缺陷时，将气体爆发穿透半凝固金属的阶段称为“形成缺陷的条件”，即在薄壁部位已凝固结束，而厚壁部位仍处于半固体‑半液体状态时发生的气体剧烈穿透阶段[[S11]]。部分中文表格文件以“铸件沸腾”或“沸腾孔”命名此类缺陷，描述的即是这一异常过程[[S19,S25]]。

### 发生机理
压铸及有色合金铸造中，沸腾期的核心诱因是**水分与高温金属液接触发生瞬时汽化**。一滴水接触到高温铝液后瞬间蒸发，体积膨胀约 1000 倍，产生大量高压蒸汽[[S20]]。其他气体来源还包括：已卷入熔体的气体受热膨胀、熔体中析出的气体聚集等。当这些气体压力超过半凝固金属的强度极限时，气体便会穿透合金，形成连通或不连通的孔洞结构[[S11]]。

具体触发条件包括：
- 型腔、压室或冷铁表面附着的水分（如脱模剂残留水、冷却系统渗水、型芯未干透等）[[S11,S20]]；
- 水基脱模剂喷涂过量，未完全挥发即合模浇注[[S6,S20]]；
- 模具龟裂处渗入的冷却水[[S20]]。

沸腾期一般发生在**充型完成后**，此时薄壁区域已经凝固，厚壁区域仍处于半固态半液态，气体压力只能在局部半凝固区域打开通道，从而形成特征性的内部空洞[[S11]]。

### 典型缺陷表现
沸腾期导致的缺陷最典型的是**沸腾孔（气穿孔）**，其外观为形状不规则的空洞，孔壁布满带有不同色泽的氧化膜，严重时气体爆发会形成贯穿孔[[S19]]。根据气体来源，可分为来自铸型、来自型芯及来自冷铁三类[[S19]]。

此外，沸腾期还可引发：
- **气孔**：高压蒸汽被裹挟在金属液内部，形成表面光滑的圆形气孔，若铸件后续进行热处理，内部气体受热膨胀可导致表面鼓泡变形报废[[S20,S7]]。
- **冷隔**：高压蒸汽提前推动部分前沿金属液冷却凝固，不同流股前沿无法熔合，形成带圆角凹陷的表面缺陷[[S5]]。
- **表面起泡与粘砂**：强烈的水汽爆发甚至可造成铸件表面金属氧化皮鼓起或机械粘砂现象[[S8]]。

从金相鉴别角度，水分沸腾引起的气孔缺陷在 100～500 倍显微观察下表现出**边缘圆滑、内腔光亮**的特征，可与边缘曲折、内腔暗色的缩孔缺陷明确区分[[S1]]。

## 影响因素与工艺控制（压铸/有色铸造场景）

为减少沸腾期及其相关缺陷，须从水源控制、喷涂工艺、模具温度管理及排气设计等多方面入手。

### 模具温度
模具表面温度直接影响水基脱模剂的成膜与水分挥发：
- 模温 **低于 150 ℃** 时，喷涂的脱模剂无法正常汽化，会形成液滴流淌，残留大量液态水，大幅增加沸腾风险；
- 模温 **超过 398 ℃** 时，脱模剂液滴在接触模具瞬间即被蒸发反弹，无法润湿模具表面，丧失保护作用；
- 推荐的正常工作区间为 **180～280 ℃**，此温度下喷涂 1 s 内即可形成均匀隔离膜[[S6,S14]]。

### 脱模剂喷涂参数
水基脱模剂的使用要点包括稀释比例、雾化效果及喷涂时间：
- **稀释比例**：常规商用高稀释比水基脱模剂为 **1:100～1:300**，高性能产品可达 **1:150～1:200**；稀释比过高则有效成膜物质不足，过低则引入过多水分残留[[S17,S14,S3]]。
- **雾化与喷射距离**：理想喷射距离为 **100～200 mm**，保证脱模剂雾化成“微液滴”均匀铺展，避免大液滴残留[[S6,S3]]。
- **喷涂时间**：单次喷涂 **0.1～2.0 s** 已足形成所需隔离膜[[S6]]。

### 干燥与吹除
喷涂作业结束后，必须留足水分挥发时间。性能良好的脱模剂在开模状态下，典型挥发静置时间为 **17～20 s**[[S6]]。同时应配合**定向吹气**操作，将型腔、滑块槽及压室内的残留水分彻底排出[[S20,S6]]。

### 水分来源管理
除脱模剂自身水分外，还须防范额外的水分侵入：
- 生产前对模具进行充分预热干燥，避免积存冷凝水；
- 定期检查冷却水管路、密封圈及浇口套，防止冷却水沿裂纹或装配缝隙渗入型腔；
- 冷铁使用前须确保干燥、清洁，并规范涂覆涂料[[S20,S19]]。

### 排气系统
型腔、型芯应设置充足且畅通的排气通道。若排气不足，高压蒸汽无法及时排出，形成背压，会显著加剧沸腾期间的穿透破坏[[S19]]。

下表汇总了防止有色合金铸造沸腾缺陷的关键工艺措施：

| 控制要素 | 推荐参数/措施 | 目的 |
| :--- | :--- | :--- |
| 模具温度 | 180～280 ℃ | 确保脱模剂正常成膜，避免水分残留[[S6,S14]] |
| 喷涂距离 | 100～200 mm | 实现均匀雾化，减少大液滴[[S6]] |
| 喷涂时间 | 0.1～2.0 s | 形成充分隔离膜，不过量带入水分[[S6]] |
| 挥发静置时间 | 17～20 s | 让水分充分汽化逸出[[S6]] |
| 吹气操作 | 定向吹净型腔、滑块槽等 | 清除残留液态水[[S20]] |
| 模具预热与检漏 | 预热排除冷凝水；定期排查冷却系统渗漏 | 杜绝额外水源[[S20]] |

## 与其他概念的关系

在有色合金铸造和压铸缺陷分析中，沸腾期及其导致的沸腾孔须与气孔、卷气、水蒸气爆炸等邻近概念加以区分[[S11,S10,S20,S1]]。

| 对比概念 | 与沸腾期的关系与区别 |
| :--- | :--- |
| **普通气孔** | 普通气孔多由卷气直接滞留形成，孔壁光滑但无气体剧烈穿透半凝固金属的行为；沸腾孔则是在半固态半液态时由气体爆发贯穿而成，孔壁常带氧化膜且形状不规则[[S11,S10]]。 |
| **卷气** | 卷气是气体进入熔体的前置过程；沸腾期是卷入气体或新生蒸汽压力超过半凝固金属强度后的最终爆发阶段，二者存在时间先后和程度差异[[S11]]。 |
| **水蒸气爆炸** | 水蒸气爆炸是熔体与大量水接触时外部发生的瞬时剧烈能量释放，常伴随响声与飞溅；沸腾期是铸件内部相对持续的气体迁移和穿透过程，无剧烈外部爆炸表现[[S11]]。 |
| **缩孔** | 缩孔边缘曲折、内腔暗色，是凝固收缩补缩不足所致；沸腾气孔边缘圆滑、内腔光亮，可通过金相区分[[S1]]。 |
| **初期溃散** | 初期溃散指砂型/砂芯在高温下的强度失效，属于造型材料侧行为；沸腾期完全属于金属熔体侧的气体动力学行为，二者载体和机理不同[[S11]]。 |

通过综合管理上述工艺因素，可以显著抑制压铸及有色合金铸造中由沸腾期引发的各类缺陷，提高铸件致密度与表面质量。
