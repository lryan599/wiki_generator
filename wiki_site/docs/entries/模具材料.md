## 概述

压铸模具材料是指用于制造压力铸造模具中直接与高温、高压、高速流动的液态压铸合金相接触的核心工作零件的特种热作模具材料，其主体类别为热作模具钢[[S27,S17]]。在模具总成中，型腔镶块、型芯、浇道镶块、浇口套、分流锥以及滑块、顶杆等不同功能零件依据其服役条件差异，可选用不同类别的材料[[S24]]。压铸模具在服役过程中长期承受周期性急冷急热的温度循环、金属液冲蚀与高温氧化腐蚀的复合作用，模具材料的性能直接决定模具的使用寿命[[S47,S35]]。

在概念上，压铸模具材料须与模具表面镀层或涂层类强化材料严格区分——前者指构成模具本体的钢种或合金，后者指通过氮化、PVD等方法施加于模具表面的功能性薄层[[S47]]。

## 分类

### 按合金元素体系分类

压铸用热作模具钢按合金元素体系可分为以下三大类别[[S51]]：

**① W系（钨系）热作模具钢**  
典型牌号为3Cr2W8V，钨元素可减小钢的热膨胀系数，其碳化物能显著提高钢的耐磨性。该类钢种使用温度可达650°C，但因导热性较差、冷热疲劳性能不足，在国外已被铬系和钼系热作钢逐步替代，目前多用于铜合金压铸等高温工况[[S51]]。

**② Cr-Mo系（铬-钼系）热作模具钢**  
典型牌号为4Cr5MoSiV1（即H13/SKD61），在使用温度不超过600°C时可替代3Cr2W8V，抗冷热疲劳性能更优，是铝、镁合金压铸的主流选用材料[[S51]]。

**③ Ni-Co系（镍-钴系）马氏体时效钢**  
含碳量低于0.01%，通过Ni₃Mo、Ni₃Ti等金属间化合物的弥散析出实现强化，其热强性和韧性可达普通H13钢的2倍以上，热处理变形量极低，适用于高精度复杂压铸模[[S51]]。

### 通用热作钢与压铸专用钢的区别

通用型热作模具钢（如5CrNiMo、5CrMnMo）以热锻为核心应用场景，侧重大截面淬透性和高冲击韧性以适配锤锻冲击载荷，在工作温度、抗液态金属冲蚀抗力、连续冷热循环疲劳寿命等方面均低于压铸专用热作模具钢[[S9,S47]]。压铸专用热作模具钢则针对性优化了合金成分配比，专门应对长期接触高温液态金属、承受20～120 MPa高压作用的服役条件[[S47,S35]]。

## 核心性能要求

压铸模具材料的核心性能指标体系包含以下关键维度[[S27,S34,S17]]：

| 性能指标 | 技术要求 | 工程意义 |
|---------|---------|---------|
| **高温强度/热屈服强度** | 在工作温度下保持足够的高温强度 | 直接决定高温下抵抗塑性变形的能力，热屈服强度越高，塑性变形越小，热疲劳破坏程度越低 |
| **红硬性（热硬性）** | 长时间接触高温液态金属后仍保持高硬度 | 避免型腔塌陷与磨损 |
| **抗热疲劳性** | 抵抗周期性冷热循环产生热裂纹的能力 | 决定模具寿命的核心指标，多数模具失效由此引发 |
| **抗热裂性** | 依托高延伸率，在不开裂条件下承受塑性变形 | 影响热裂纹萌生的临界循环次数 |
| **耐冲蚀性** | 抵抗高速流动液态金属冲刷腐蚀 | 减轻型腔表面机械磨蚀与化学腐蚀 |
| **高导热性** | 降低模具表面温度梯度 | 削减热应力水平，抑制热裂纹萌生 |
| **高韧性** | 承受冲击载荷时避免脆性断裂 | 与材料纯净度、碳化物分布均匀性密切相关 |
| **抗回火稳定性** | 长期高温服役下抵抗组织软化 | 防止服役中因硬度持续下降导致塑性变形幅度扩大 |
| **淬透性与热处理变形** | 淬火后截面硬度均匀，变形量小 | 保证大型复杂模具热处理后尺寸精度 |
| **高温抗氧化性** | 型腔表面在高温下不发生显著氧化 | 氧化形成的腐蚀沟是热疲劳裂纹的起源之一 |

## 典型材料牌号与成分

### H13类（4Cr5MoSiV1）

4Cr5MoSiV1是中国GB/T 1299标准下的牌号，对应美国AISI H13、日本JIS SKD61、德国DIN 1.2344，是目前应用最广泛的通用型压铸模具钢[[S13,S26]]。其核心合金元素含量范围如下[[S12]]：

| 元素 | 质量分数 w/% |
|------|-------------|
| C | 0.32～0.45 |
| Si | 0.80～1.20 |
| Cr | 4.75～5.50 |
| Mo | 1.10～1.75 |
| V | 0.80～1.20 |
| Mn | 0.20～0.50 |
| P | ≤0.030 |
| S | ≤0.030 |

北美压铸协会NADCA 207系列标准对H13钢的含碳量限定为0.37%～0.42%，相比国标区间更窄，旨在保证强度的前提下尽可能降低碳含量，提升材料韧性、减少热疲劳裂纹扩展风险[[S20]]。

H13钢经标准淬火回火处理后硬度可达HRC 47～49：在46 HRC时抗拉强度约为1503 MPa，在51 HRC时约为1937.5 MPa，密度为7.8 g/cm³[[S13,S20,S4]]。

多标准体系下H13类钢的等效牌号映射关系为[[S13,S26]]：

| 标准体系 | 牌号 |
|---------|------|
| 中国 GB/T 1299 | 4Cr5MoSiV1 |
| 美国 AISI/ASTM A681 | H13 |
| 日本 JIS | SKD61 |
| 德国 DIN/EN 10027 | 1.2344 |

### H11类（4Cr5MoSiV）

4Cr5MoSiV即美国牌号H11，属于空冷淬火硬化型热作模具钢。其V元素质量分数范围为0.30%～0.60%，低于H13钢，具有优异的热稳定性、淬透性和热疲劳性能，常规工况下最高工作温度不超过600°C，可用于铝铸件压铸模等部件制造[[S25]]。

### H10类（4Cr3Mo3SiV）

4Cr3Mo3SiV对应美国AISI H10牌号，核心成分为C 0.35%～0.45%、Cr 3.00%～3.75%、Mo 2.00%～3.00%、V 0.25%～0.75%，退火状态交货硬度≤229 HBW，淬透性和高温强度优于普通H13钢，适用于高负荷压铸模制造[[S15]]。

### 时效硬化类与马氏体时效钢

**沉淀硬化型（时效硬化型）热作模具钢**的含碳量通常在0.2%左右，添加V、Nb、Ni、Al、Mo等沉淀强化合金元素。淬火后经400°C左右低温回火后硬度约40 HRC，此时机加工性能优异。模具服役过程中，型腔表面受工件加热至500～600°C时，因合金碳化物和金属间化合物的析出，表面硬度可上升至45～48 HRC，而心部仍保留高韧性。代表牌号包括日本日立金属YHD3、YHD26、YHD28，日本大同特殊钢DH76（2Cr3Ni3V），以及中国国产牌号2Cr3Mo2NiVSi（PH）等[[S31,S40,S38]]。

**马氏体时效钢**（以18Ni系列为代表）为超低碳高合金超高强度钢，C含量低于0.01%，典型成分为Ni约18%、Mo约5%、Co约11%、Ti约0.3%。固溶状态下屈服强度800～900 MPa，断后伸长率约20%，断面收缩率70%～80%，热处理变形极小。此类钢的模具服役寿命可达普通H13模具的3～4倍，多用于高精度铜、铝合金压铸模制造[[S7,S16]]。

### 铜合金与钨基合金模具材料

铜合金压铸专用模具材料主要包括高导热铜基合金（铬锆钒铜、铬锆镁铜等）和高熔点钨基合金（如Anvilloy 1150）。在600°C下，铜合金的力学性能显著高于1000°C下的3Cr2W8V钢，模具使用寿命可比3Cr2W8V钢提高1.5～2倍[[S7,S49]]。

### 压铸模不同零件选材

压铸模具不同功能零件需适配不同类别材料[[S24]]：

| 零件类别 | 推荐材料 | 热处理硬度要求 |
|---------|---------|--------------|
| 型腔镶块、型芯（锌合金） | 3Cr2W8、5CrNiMo、4CrW2Si | 46～50 HRC |
| 型腔镶块、型芯（铝镁合金） | 3Cr2W8、4Cr5MoSiV | 48～52 HRC |
| 型腔镶块、型芯（铜合金） | 同类型热作钢 | 40～44 HRC |
| 浇道镶块、浇口套、分流锥 | 3Cr2W8、5CrMnMo、5CrNiMo | 44～48 HRC |
| 导柱、导套、滑块、推杆 | T8A、T10A、9Mn2V | 50～55 HRC |
| 动定模套板、支承板 | 45钢 | 调质220～250 HBW |
| 模座、模脚 | 30～45钢或Q235 | 回火处理 |

### 德国标准体系

德国压铸模具钢体系中，不同牌号适配不同压铸合金种类和零件类型。例如2343（4Cr5MoVSi）和2567（3Cr2W4V）钢用于压铸锌合金镶块及细型芯，2343和2606（4Cr5MoWVSi）等用于铝镁合金镶块，直径小于15 mm的微细型芯则需选用2365（3Cr3Mo3V）或2581（3Cr2W9V）等更高强度钢种[[S36]]。

## 热处理与表面处理

### 全流程热处理规范

压铸模具的全流程热处理包含多道关键工序[[S19,S18]]：

1. **去应力退火**：粗加工后在650～680°C保温3～5 h。型腔简单的模具出炉空冷；形状复杂模具随炉冷却至400°C后出炉空冷。
2. **预处理**：采用球化退火或调质处理，获得均匀组织和弥散分布的碳化物，改善钢的强韧性。
3. **分级预热**：400～650°C低温预热在空气炉中进行，更高温度预热采用盐浴炉，保温时间均按1 min/mm计算。
4. **淬火加热**：盐浴炉中保温系数取0.8～1.0 min/mm。要求高韧性的模具选用较低淬火温度；要求高温强度高的模具选用较高淬火温度以提升热稳定性。
5. **淬火冷却**：形状简单、变形要求低的模具采用油冷；形状复杂、变形要求高的模具采用分级淬火。冷却至150～180°C均热后立即回火，均热时间按0.6 min/mm计算。
6. **回火**：总回火次数不少于3次。第一次回火落在二次硬化温度区间；第二次使模具达到目标硬度；第三次回火温度比第二次低10～20°C。回火保温时间不少于2 h，回火后采用油冷或空冷。

### 不同合金种类的使用硬度

不同压铸合金所对应的模具淬火回火后常规使用硬度范围[[S33]]：

| 模具类型 | 常用材料 | 热处理 | 硬度（HRC） |

### 表面强化处理

针对锌、镁、铝合金压铸成型零件，氮化/软氮化处理规范为氮化层深度0.08～0.15 mm，表面硬度不低于600 HV。该类表面强化工艺可大幅提升模具成型面的耐磨性与抗铝合金焊合能力，降低冲蚀损伤，延长模具服役寿命[[S28,S10]]。

## 失效形式与机理

### 失效定义与分类

压铸模具失效的广义定义为模具完全无法服役，生产场景下通常指模具核心工作零件丧失功能、无法通过修复继续使用。失效过程可分为初期失效、随机失效和耗损失效三个阶段，分为达到预设寿命的正常失效和远低于预期寿命的早期失效两类[[S39]]。

铝、铜合金压铸模不同类型失效的占比统计[[S37]]：

| 失效类型 | 铝合金压铸模占比 | 铜合金压铸模占比 |
|---------|:-------------:|:-------------:|
| 热疲劳 | 60%～70% | >95% |
| 开裂或崩块 | 12%～25% | <5% |
| 变形 | 5%～15% | <5% |
| 热熔损或其他 | <5% | <5% |

### 热疲劳龟裂

热疲劳龟裂是压铸模具最常见的失效形式，占整体压铸模具失效的60%～70%。其生成机理为：模具型腔表面在压铸生产过程中反复承受激冷激热的交替作用，型腔表层和内部因热变形相互约束而产生循环交变热应力，引发材料组织结构损伤与韧性下降，逐步萌生微裂纹。随压铸循环次数增加，裂纹持续扩展，熔融金属液侵入裂纹内部叠加机械应力，进一步加速裂纹扩展，最终导致模具报废[[S32,S8]]。

热疲劳裂纹往往在晶界碳化物和夹杂物集中区萌生，因此钢质洁净、显微组织均匀的优质热作模具钢具有更高的热疲劳抗力[[S32]]。氧化会在模具表面产生腐蚀沟，成为热疲劳裂纹的起源，加剧裂纹萌生与扩展[[S35]]。

### 整体脆性开裂

整体脆性开裂是由机械过载或热过载引发的模具灾难性断裂。材料的塑韧性与热稳定性不足，或钢材内部存在严重杂质、冶金缺陷，外加压铸操作不当、模具结构设计存在明显应力集中位置，均会诱发该类失效。整体脆性开裂在实际生产中发生的占比较低[[S23,S32]]。

### 冲蚀与熔蚀

金属液冲蚀失效是机械冲刷与化学腐蚀共同作用的结果：高压高速射入型腔的熔融金属液对模面形成机械磨蚀，同时金属液中的合金元素与模具钢表面发生化学反应生成脆性化合物，这些脆性相成为热裂纹新的扩展源头。后续充填的金属液渗入裂纹内部，叠加热应力会进一步放大裂纹尖端的拉应力，加速裂纹扩展[[S23,S5]]。

### 焊合（粘模）

铝合金压铸过程中的合金焊合（soldering/粘模）现象，是高温金属液与钢材表面发生冶金反应的结果：熔融铝合金破坏模具表面的保护涂层或润滑膜后，模具中的铁元素溶解进入铝熔体，铝中的元素向钢基体扩散，在界面处逐步生成Al-Fe系、Al-Fe-Si系金属间化合物层。凝固后合金铸件与模具表面发生粘连，导致脱模时铸件表面划伤甚至无法取出，直接影响生产效率与铸件品质。该现象在内浇口附近高温、高液流速度位置发生概率显著更高[[S14,S46,S1]]。

## 选材准则

### 按压铸合金种类选材

压铸模具选材需匹配对应铸造合金的热物理特性和工艺温度[[S33,S2]]：

- **锌合金压铸**：熔点低，可选用成本较低的低合金工具钢（如CrWMn、30CrMnSi）以及5CrNiMo、4Cr5MoSiV1、3Cr2W8V等，硬度范围47～55 HRC。
- **铝、镁合金压铸**：优先选用4Cr5MoSiV1（H13）类Cr-Mo系热作模具钢，兼顾热强性与韧性，硬度范围45～55 HRC。
- **铜合金压铸**：铜液浇注温度高、热载荷大，需选用含W、Co等元素的高热强性热作模具钢（如3Cr2W8V、3Cr3Mo3V、3Cr2W9Co5V等）以提升抗高温软化能力，硬度范围31～41 HRC。此外可选用高导热铜基合金（铬锆钒铜等）或钨基合金（Anvilloy 1150）作为镶块材料，使用寿命可比传统3Cr2W8V钢提升1.5～2倍[[S7,S49]]。

### 按铸件批量与壁厚选材

当铸件年生产批量大（超过10万模次）或铸件壁厚较厚、金属液充型热量大、冲蚀载荷高时，优先选用高纯度电渣重熔冶炼的高级别热作模具钢，并搭配配套的氮化/PVD表面处理，以降低单位零件分摊模具成本。对于小批量试制、低载荷的压铸产品，可选用普通级别热作模具钢以控制模具初始投入总成本[[S8,S29]]。

## 与模具设计及工艺参数的交互关系

压铸模具材料的服役表现与模具设计、压铸工艺参数之间存在密切的交互作用。模具中冷却通道的合理布置可降低型腔表面温度梯度，减小热应力，延缓热疲劳裂纹萌生[[S27]]。浇注温度、模具预热温度、内浇口速度、压射比压等工艺参数对模具材料的热载荷和机械载荷有直接影响：过高的浇注温度和充填速度显著加剧冲蚀和焊合倾向[[S14,S46]]；模具预热不充分或冷却不均匀则增大热冲击效应，加速热疲劳裂纹扩展[[S8]]。

引起压铸模具失效的因素既包括外因（浇注温度、模具预热、涂料喷涂量、压铸机吨位匹配、压铸压力、内浇口速度、冷却水开启同步性、铸件材料种类与含Fe量、铸件壁厚与形状等），也包括内因（模具材质的冶金质量、坯料锻制工艺、模具结构设计合理性、浇注系统设计合理性、加工内应力、热处理工艺以及配合精度与表面粗糙度等）。合理的选材与正确的热处理是延缓模具失效、保证模具寿命的基础[[S8,S39]]。
