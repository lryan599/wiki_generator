## 概述

渗流铸造（英文：infiltration casting）是一种特种铸造工艺，其核心是将液态金属在重力或外加压力作用下渗入预先制备的多孔预制体或填料床的孔隙中，经凝固后获得多孔金属（如泡沫铝）或金属基复合材料的成形方法[[S3][S14]]。在中国机械工业标准汇编的分类体系中，渗流铸造归属于特种铸造大类[[S12]]。

该工艺与用于铸件微孔缺陷封堵的后处理工艺“浸渗（Porosity sealing / Impregnation）”有本质区别，后者不属于铸造成形类工艺[[S6][S19]]。

## 定义

国家标准GB/T 5611-2017《铸造术语》对渗流铸造的正式定义如下：

> 渗流铸造（别称渗铸，英文对应infiltration casting）是使金属液在重力或压力作用下，渗入颗粒、纤维材料或二者预制件的特种铸造方法，工艺产出物包含复合铸件、多孔（泡沫）金属铸件[[S3][S14]]。

渗流铸造的浸渗过程与液态金属在多孔介质中的毛细管压力降直接相关。预制体内孔隙狭窄且弯曲，阻力大，流体流动状态属于层流。当液态金属对增强相不润湿时，必须引入额外的外加压力克服毛细管阻力，才能实现完整的充填[[S23]]。

## 工艺变体分类

渗流铸造按驱动外力类型可划分为四大类[[S4][S11]]：

| 分类 | 驱动力 | 典型压力范围 | 特点 |
|------|--------|-------------|------|
| 机械压力（挤压）渗流 | 液压压头直接加压 | 70~100 MPa | 制品致密度高、工艺效率高，可批量生产金属基复合材料零部件，但需厚壁压室，不适合大尺寸构件[[S4]] |
| 气体压力渗流 | 压缩气体 | 通常在十几兆帕以下 | 成本较低、操作灵活，可对预制体和金属液独立调控温度与压力参数[[S4][S11]] |
| 真空/负压渗流 | 型腔抽负压 | — | 降低渗流阻力、减少氧化缺陷[[S6]] |
| 无压渗流（自润湿渗流） | 毛细管力（无外加压力） | — | 通过改善润湿性实现自发渗透，可制备大尺寸复杂构件，但适用体系有限，局部不完全渗透和残留气孔风险较高[[S6]] |

## 典型工艺流程

渗流铸造的完整操作流程一般包含六个核心阶段[[S6][S17][S15]]：

1. **预制体制备**——以纤维（如氧化铝短纤维、碳纤维）或颗粒增强相与少量黏结剂混合，成形为所需形状的预制体，经脱脂去除挥发物；或用可溶性填料（如食盐）颗粒压实获得多孔预制体。
2. **预热**——将预制体随铸型一同加热至规定温度，避免低温金属液前沿过早凝固阻断渗流通道。
3. **熔体注入**——将熔融金属浇注至装有预热预制体的型腔内。
4. **压力施加与保持**——通过机械压力或气压等外力克服毛细管阻力，推动金属液渗入预制体全部孔隙，按工艺要求保持规定时间。
5. **压力下凝固**——在外力约束条件下完成凝固，避免缩松和孔隙析出。
6. **脱模与后处理**——取出成型件。若采用可溶性填料预制体，通过浸泡溶解去除填料获得具有连通孔隙的多孔金属构件。

以“上压渗流铸造法”制备泡沫铝为例，其步骤为：将渗流颗粒装入嵌套内压实，整体置入加热炉完成预热后加装钢套，浇注铝熔体后立即加盖施压，冷却后取出浸入水中溶解去除填料颗粒，即可得到目标泡沫铝制品[[S18][S31]]。

## 关键工艺参数

渗流铸造的工艺参数对成品质量具有决定性的影响，其权威取值范围可整理如下[[S17][S7][S30][S10][S8][S5]]：

| 参数 | 作者推荐范围 | 条件或场景 |
|------|------------|------------|
| 基体（铝基）熔体温度 | 700~800 ℃ | 预制增强压铸渗流，一般场景 |
| | 740~760 ℃ | 以食盐为填料制备泡沫铝 |
| 预制体/铸型预热温度 | 500~800 ℃ | 纤维/陶瓷增强相预制体对铝基合金 |
| | 约450~500 ℃ | 食盐填料预制体 |
| 渗透压力 | 3~100 MPa | 通用压力渗流 |
| | ≥50 MPa | 要求无孔隙复合材料 |
| | 5~15 MPa | 挤压渗流制备泡沫铝 |
| 加压速率 | 1~3 cm/s | 避免预制体变形 |
| 预制体孔隙率 | 最高可达75% | 金属基复合材料 |
| | 平均约70% | 加压渗流制备泡沫铝 |
| 金属液过热度 | 80~140 ℃ | 针对1.5~2 mm粒径盐粒填料[[S8]] |

*注：食盐作为可溶性填料时，须预先在600 ℃左右保温0.5~0.6小时去除结晶水，以避免与高温金属液接触时爆裂产气[[S30]]。*

## 主要优缺点

### 优点

- 工艺原理简单、设备投资低，可实现机械化批量生产，制备泡沫铝成品率可高于90%[[S22]]。
- 可制备增强相体积分数高达70%以上的金属基复合材料[[S22]]。
- 金属熔体与增强相在高温下接触停留时间短，可有效抑制严重界面反应和脆性相生成[[S4]]。
- 可实现近净成形，减少后续机加工，降低生产成本[[S22]]。
- 可实现局部选择性增强，仅在构件指定耐磨或受力区域布置渗流预制体[[S22]]。

### 缺点

- 若基体合金与增强相润湿性差，必须施加较高外力并配备高压设备才能完成渗流[[S6]]。
- 高渗流压力下，若预制体力学强度不足，易发生变形和增强相分布失稳[[S23][S19]]。
- 未优化润湿性的体系中，预制体尖角等细微区域易出现渗流不足，残留微观孔隙[[S6][S23]]。
- 温度控制不当会导致铝、镁等活性金属与增强相表面发生有害界面反应，生成脆性界面相，劣化材料力学性能[[S4][S6]]。

## 适用材料与典型应用

### 适用材料

渗流铸造适用的基体以轻质有色金属为主，涵盖铝基合金和镁基合金[[S6]]。近年来，生物医用可降解镁钙合金也已用于渗流复合构件的制备[[S22]]。常用增强相包括氧化铝纤维、碳纤维、SiC颗粒和B₄C颗粒等陶瓷增强材料。制备泡沫金属时，可采用食盐等可溶颗粒作为占位填料[[S20][S30]]。

### 典型应用场景

1. **金属基复合材料**：批量生产陶瓷短纤维、颗粒、晶须增强的铝/镁基零部件，典型产品为内燃机活塞和重载连杆。活塞生产已形成年产数百万件的规模[[S17][S15]]。
2. **多孔/泡沫金属**：制备高孔隙率连通孔泡沫铝构件，用作建筑吸音降噪材料、电磁屏蔽构件和交通防撞吸能部件[[S15]]。
3. **压铸领域渗透充填**：将局部耐磨预制体预先布置在压铸模型腔指定位置，通过压铸高速注入金属熔体加压渗流，一次成型获得局部增强的高耐磨压铸构件[[S15]]。

## 常见缺陷及控制

| 缺陷类型 | 形成机理 | 控制要点 |
|----------|----------|----------|
| 不完全渗透 | 渗流压力不足以克服毛细管阻力；预热温度或浇注温度过低使金属液前沿过早凝固、流动性不足[[S8]] | 针对填料粒径匹配合适的充型压力；优化填料预热温度与金属液过热度 |
| 气孔 | 填料含结晶水或吸附水汽；预制体内气体无法排出而被金属液包裹卷入[[S30][S6]] | 填料高温充分预热除水；增设排气通道；采用分级加压引导气体有序排出 |
| 纤维冲蚀 | 充型速度过快，高速金属液冲刷、移位、冲断短纤维预制体，破坏结构均匀性[[S29]] | 在保证渗流长度的前提下降低充型升压速率，避免形成湍流冲刷预制体 |
| 界面反应 | 高温下金属液与增强体接触时间过长，生成脆性金属间化合物界面层[[S4][S6]] | 在保证渗流充分前提下缩短高温接触时间；对增强体进行表面改性调控润湿性和界面反应 |
| 宏观偏析 | 高渗流压力下低熔点共晶相被强制挤出渗流通道，在铸件表层形成共晶相富集偏析层[[S21]] | 合理控制渗流压力上限，金属液完全填充孔隙后立即停止施压 |

## 与挤压铸造的核心差异

渗流铸造与挤压铸造（squeeze casting / squeeze die casting）在国家标准定义层面存在本质区别，二者不可混淆。GB/T 25370-2020定义：

- **挤压铸造**：在高挤压压力作用下将熔融（或半固态）金属缓慢、平稳注入铸型，并使其在高压力下凝固成形，获得致密的整体铸件，工艺目标是消除常规铸件的缩松缩孔缺陷[[S13][S16]]。
- **渗流铸造**：使金属液在重力或压力作用下渗入颗粒、纤维材料或二者预制件的特种铸造方法，产物为复合铸件或多孔（泡沫）金属铸件[[S3][S14]]。

二者的充型对象和产物目标存在本质差异：挤压铸造充填的是空型腔，追求高致密度；渗流铸造充填的是多孔预制体的孔隙网络，目标是获得复合材料或多孔结构[[S15]]。
