## 概述

限位块是压铸模具侧向抽芯机构中的核心定位零件，常被称为“限位挡块”，部分生产场景下也俗称“停止块”[[S30,S27]]。国际压铸行业中其通用英文名称为 **die casting stop block** 或 **die casting limit block**[[S24,S34]]。限位块的基本功能是限制侧滑块抽芯后的极限位置，保证开模后滑块稳固停留在指定抽芯终止位置，避免合模时斜销无法准确插入滑块导向孔，防止合模错位引发型腔碰损；同时避免滑块过度运动破坏弹簧等辅助复位零件[[S31,S4,S28]]。

## 功能与工艺作用

在斜销侧向抽芯机构中，侧向抽芯动作完成后，侧滑块的端面直接抵靠在限位块的定位面上，配合辅助复位弹簧使滑块始终保持与限位块贴合的状态，保证合模前滑块的位置精度，后续合模时铲基（楔紧块）可顺利对滑块进行锁紧定位[[S28,S23,S37]]。

压铸全循环过程中限位块的受力状态与作用逻辑如下[[S32,S33,S5]]：

| 循环阶段 | 受力/作用状态 |
|---------|-------------|
| 开模阶段 | 承受抽芯动作完成后滑块随弹簧惯性撞击的冲击载荷 |
| 抽芯结束后 | 持续接触滑块实现精准定位 |
| 合模阶段 | 承受斜销驱动滑块移动前的预定位支撑载荷 |
| 压射阶段 | 配合楔紧块辅助分担侧向反作用力，避免滑块微量移位 |
| 冷却阶段 | 模具热胀冷缩微量位移不会导致限位块承载过载 |

## 结构与分类

压铸模限位块常见的标准结构分为**矩形**和**圆形**两类[[S1,S21]]。一般使用M12规格内六角螺钉进行固定，限位块安装对应的基座位置需设计有凸台或加强筋以保证刚性[[S1,S21]]。行业通用CAD标准件库中包含A型、B型、C型三种规格选项，对应不同的直径、厚度及轴孔紧配参数[[S29,S9]]。

限位块存在不同应用场景下的结构变体：推板限位块用于推出机构的行程限位[[S7]]；下滑块结构中常区分主限位与副限位，其中主限位面（合模时与动模镶块贴合）与副限位面（合模时与左右镶块组合形成密闭腔体）的配合质量直接影响铸件成型精度[[S37]]。

## 材料与性能

压铸模具限位块常用材料及性能对比如下表所示：

| 材料牌号 | 热处理硬度 | 适用场景 | 来源 |
|---------|-----------|---------|------|
| 45钢（常规） | 28~32 HRC | 普通常规使用场景 | [[S21,S36]] |
| 45钢（高频淬火） | 40~48 HRC，表面可做四氧化三铁处理 | 高承载耐磨场景 | [[S21,S22]] |
| 45钢/SKS93 | 54~58 HRC | 重载、频繁承压工况 | [[S1,S8,S20]] |
| T8A/T10A | 40~50 HRC | 标准工况通用选材 | [[S18]] |
| Cr12MoV | 60~62 HRC | 大型精密压铸模具，长期服役 | [[S12]] |

45钢是压铸模具限位块最常用的基础选材之一[[S21,S36]]。合金工具钢SKS93（或45钢经相应热处理）制成的限位块，刚性及耐磨性能优于普通碳钢材质[[S20,S8]]。高铬合金工具钢Cr12MoV为高端选材，耐磨性能和抗塑性变形能力远优于普通碳钢[[S12]]。

## 关键参数与安装要求

压铸模具限位块加工形位公差通用要求：基准面平行度精度等级应符合GB/T 1184—1996规定的5~6级精度要求；未注表面粗糙度Ra=6.3μm；所有棱边预留2×45°倒角[[S36]]。

安装布置与使用要求[[S21,S8]]：

| 要求项 | 具体要求 |
|-------|---------|
| 接触面积 | 接触面需保证至少80%的面积相接触 |
| 垫片数量 | 下方叠加垫片总数不超过2片，垫片尺寸略小于安装面外轮廓、边缘不得超出 |
| 支撑区域 | 下方必须为实体结构或设置加强筋 |
| 相邻间距 | 常规取值300~450 mm |
| 布置位置 | 模具四个角落，总数不少于4个 |
| 编号 | 限位块需编号防错 |

与国家标准相关的限位构件标准：压铸模圆柱类限位构件（限位钉）现行国标为 GB/T 4678.14-2023，新增B型限位钉规格并调整了A型尺寸系列[[S6]]。冲压类限位柱行业标准为JB/T7652.2—2008，材料推荐45钢，硬度控制在43~48 HRC[[S16]]。

## 常见缺陷与失效模式

限位块精度不足引发的典型质量问题是铸件飞边、毛刺缺陷。当限位块因磨损、松动等导致定位尺寸偏移时，抽芯后滑块停留位置出现偏差，合模时斜销无法带动滑块完全回到预设位置，滑块成型面与定模镶块之间形成超出公差的间隙，高压液态金属渗入该间隙，最终在铸件侧孔/侧凸台边缘形成飞边、毛刺[[S25,S10]]。

常见失效模式及原因[[S26,S38,S3]]：

| 失效模式 | 产生原因 |
|---------|---------|
| 磨损失效 | 滑块反复撞击、相对滑动产生磨粒磨损；接触表面硬度低于磨粒硬度时磨损加剧 |
| 压塌失效 | 表面长期承受高频冲击载荷，表面淬硬层厚度不足或硬度未达标，发生局部塑性变形凹陷 |
| 断裂失效 | 材料内部冶金缺陷、冲击过载引起应力集中，或长期交变载荷下疲劳累积导致脆性断裂 |
| 松动失效 | 紧固螺钉未达到规定预紧力矩，长期受振动冲击发生退丝，导致限位块整体位置偏移 |

### 预防措施与日常维护

**预防措施**[[S21,S13]]：

1. **材料与热处理**：选用T8A、T10A等高碳工具钢，热处理硬度控制在40~50 HRC，保证表面硬度大于50 HRC以降低磨损概率。
2. **表面强化**：可选用PVD沉积类金刚石涂层、钼基润滑涂层等提升表面耐磨性能。
3. **结构设计**：保证接触面积占比达到80%以上，减少局部应力集中；调整垫片不超过2片，且尺寸略小于安装面边缘。
4. **安装校验**：定期校验固定螺钉预紧力矩，不同型号模具的限位块单独编号防错。

**日常维护方法**[[S14,S35]]：

| 维护项目 | 具体要求 |
|---------|---------|
| 磨损检查 | 磨损量超过0.03mm时可研磨修复，超过0.1mm直接更换新件 |
| 润滑 | 定期在接触表面加注高温润滑剂，形成稳定吸附膜减少黏着磨损 |
| 清洁 | 定期清理安装面与接触面上残留的金属碎屑、涂料残渣等污染物 |
| 可更换设计 | 重载工况可采用可更换镶块结构，无需整体拆除即可快速更换损耗部件 |

## 关联组件与配合

限位块在侧向抽芯机构中与以下组件紧密配合[[S28,S18,S37]]：

- **滑块（侧滑块）**：抽芯后滑块端面抵靠限位块实现定位，合模时由斜销驱动复位后楔紧块锁紧。
- **铲基（楔紧块）**：在压射阶段锁紧滑块，防止滑块后退，限位块配合楔紧块辅助分担侧向反作用力。
- **弹簧/拉杆**：提供张力使滑块紧贴在限位块侧面，弹簧张力需大于滑块自重以保证定位准确。
- **耐磨板**：设置于滑块活动面上，减少滑动摩擦，与限位块共同保证滑块运动精度和定位可靠性。

下滑块结构中常存在主限位与副限位的过定位问题，优化方向为取消主限位以消除其对副限位贴合的影响，保证合模密封能力[[S37]]。
