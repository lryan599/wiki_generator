## 概述

“串皮”在我国通用铸造标准中被正式收录为铸造缺陷术语，标准英文对应术语为 **open core** [[S46]]。该术语在熔模铸造领域指型芯外露导致的铸件缺肉缺陷，而在压铸生产现场，“串皮”属行业俗称，国内现行压铸类专用标准（如 GB/T 29092-2012《镁及镁合金压铸缺陷术语》）并未将其列为官方标准压铸缺陷术语 [[S19,S46]]。

## 定义与别名

根据 GB/T 5611-2017《铸造术语》，**串皮（open core）** 定义为：熔模铸件内腔中的型芯露在铸件表面，使铸件缺肉。标准注明其主要成因为型芯在型壳焙烧、浇注过程中移位、变形或折断，浇注后型芯偏离正常位置露出铸件表面；当浇注温度和型壳温度偏低、空心铸件壁厚过薄导致金属液充填不足时也会引发该缺陷 [[S46]]。别名可称为 **露芯** [[S46]]。

## 缺陷分类

在国家标准 GB/T 5611-2017 和国际铸造缺陷分类体系中，串皮被归类为 **形状及重量差错类缺陷** [[S46]]，与铸件变形、尺寸不合格、错型、错芯、型芯下沉等缺陷同属一个大类 [[S12,S26]]。

从形成本质来看，串皮属于 **缺肉类缺陷**，本质是铸件局部金属未填充完全，外露内部型芯；这一特征与飞边（溢料类缺陷，金属液溢出形成额外突出金属薄片）在形成机理上完全不同 [[S19,S46]]。

## 形成机理（压铸场景）

虽然串皮在标准层面主要定义于熔模铸造，但在压铸行业现场，该术语常用于描述分型面处金属液渗透形成的薄层状分层缺陷。其形成可从以下四个维度理解 [[S17,S37,S49]]：

| 形成维度 | 机制描述 |
|---------|---------|
| 金属液充型特性 | 高压下高速流动的合金液处于紊流状态，可侵入模具微小间隙，先于主充型流在间隙内形成薄层半凝固金属 |
| 模具配合间隙 | 分型面精度不足、磨损后形成 ≥0.02 mm 微间隙；锁模力瞬时波动导致模具微开 ≥0.05 mm 时，金属液沿间隙渗透形成镶嵌在铸件表层的薄金属皮 |
| 热应力作用 | 型腔温度梯度超过 30℃ 时模具膨胀不均，进一步加大分型面间隙，金属液渗透形成分层结合不良的薄皮结构 |
| 脱模剂堆积渗透 | 过量喷涂的脱模剂在分型面缝隙内堆积，被高速充型的金属液裹挟后嵌入铸件表层，冷却后形成融合强度极低的表层缺陷 |

## 影响因素

### 模具因素

| 因素 | 具体要求/影响 |
|------|-------------|
| 分型面精度 | 普通压铸分型面平面度要求 ≤0.05 mm/m，高压压铸要求 ≤0.02 mm/m；精度超标会大幅提升金属液渗透概率 [[S17,S23]] |
| 模具硬度 | 硬度不足会导致分型面在周期性高压冲击下快速出现塑性变形，形成不均匀间隙 [[S17]] |
| 分型面磨损 | 长期受高温高压金属液冲刷后出现局部磨损凹陷，间隙超过临界渗透阈值时缺陷发生率显著上升 [[S17,S23]] |

### 工艺参数

| 参数 | 影响机制 |
|------|---------|
| 注射速度 | 压射速度过快时金属液动压过大，渗透进入分型面微间隙的概率显著提升，是生成串皮的关键诱因 [[S17,S49]] |
| 压射压力 | 常规压铸 20–50 MPa，高压压铸 60–150 MPa，压力超出设计范围会大幅提升串皮出现率 [[S17]] |
| 金属液温度 | 温度过高降低合金粘度；例如 ADC12 铝合金在 680℃ 时粘度低至 0.15 Pa·s，渗透能力显著增强 [[S17]] |
| 锁模力 | 锁模力计算未计入所有投影面积、动态补偿不足，导致模具出现微开缝隙后直接引发串皮 [[S17,S49]] |

### 合金种类

| 合金 | 串皮发生特性 |
|------|-------------|
| 铝合金 | 以 ADC12 为代表的高硅铝合金粘度低，680℃ 下流动性极好，金属液渗透能力强，串皮发生概率高于锌合金 [[S38,S17]] |
| 锌合金 | 结晶温度范围小、凝固速度快，在常规压铸条件下金属液还未完成渗透就已凝固，串皮发生概率低于铝合金；本身电镀性能好，串皮对后续电镀工序的负面影响更直接 [[S10]] |
| 镁合金 | 液态流动性相对较差，但凝固范围宽，合金液充型后维持液态的窗口更长，渗透进入分型面间隙的时间更充足；易氧化产生熔渣夹杂，嵌入间隙后更易与串皮缺陷结合出现复合型缺陷 [[S4]] |

### 熔炼质量

- 铝合金中 Mg 含量 ≥0.3%、Mn ≥0.5%、Zn ≥1.2% 时，铸件表层易形成硬脆结构，渗透生成的薄皮与基体结合强度极低，抛丸或后处理时极易剥离露出串皮缺陷 [[S37]]。
- 回炉料使用比例 ≥50% 时，除渣除气难度大幅提升，杂质和氧化膜大量混入金属液，容易在分型面缝隙处形成夹杂核心，加速金属液渗透形成串皮 [[S37]]。

## 对铸件质量的影响

| 影响维度 | 具体表现 |
|---------|---------|
| 尺寸精度 | 串皮沿分型面附着在铸件表层，导致局部尺寸超出公差范围，破坏近净成形的高精度特性；精密配合类零件出现串皮后直接导致装配尺寸不合格 [[S28]] |
| 气密性 | 串皮形成的薄层与铸件基体之间存在大量微观缝隙，直接破坏铸件致密性，承压类铸件出现串皮后极易泄漏，无法满足耐压、密封类工况要求 [[S14]] |
| 力学性能 | 串皮属于两层金属融合不良的薄弱界面，降低铸件分型面位置的有效承载截面积，界面处应力集中，导致铸件整体强度和冲击韧性显著下降 [[S37]] |
| 表面处理 | 电镀时串皮部位薄皮剥落导致漏镀、镀层起皮、结合力不合格；喷涂时缝隙内残留脱模剂、油污导致涂层附着力不足，出现局部鼓泡、涂层脱落 [[S34]] |

压铸行业通用检验规范中，串皮属于 **必须完全清除的不合格缺陷**，承压、外观、高精度类铸件出现串皮即判定为报废或需要全量打磨修整至完全去除 [[S34]]。

## 检验与评判方法

### 检验方法

| 检验方法 | 适用性与特点 |
|---------|-------------|
| 目视外观检验 | 最基础方式，搭配 5 倍以内放大镜在常规照明下快速识别分型面、活动镶块位置的串皮凸起或偏差；建议以标准样片为基准避免人为判定偏差 [[S33,S29,S41]] |
| 荧光渗透探伤 | 检出精度可达 0.02–0.05 mm，适用于表面密封性要求高的铸件串皮缺陷精准检测；操作流程为清洗 → 渗透剂浸润 5–10 min → 擦拭多余渗透剂 → 喷涂显像剂后观察判定 [[S33,S30]] |
| 着色渗透探伤 | 简单快速，将着色剂渗入缺陷后喷涂显像剂，从颜色改变判定缺陷位置和大小 [[S33]] |
| 尺寸测量 | 游标卡尺检测局部壁厚偏差，三坐标测量仪判定整体形位偏移，标准检测棒/塞规检测孔径偏移，适合高精度装配场景的量化判定 [[S33,S25]] |
| 超声波检测 | 可探测串皮引发的铸件内部结合不密实处的分层/位移缺陷，同时辅助测量异常壁厚，适用于内部隐伏类串皮缺陷的无损检测 [[S30,S25]] |

### 评判标准

- GB/T 15114《铝合金压铸件》标准未单独设置“串皮”术语对应的专门评级条目，仅明确铸件允许存在隔皮类缺陷，缺陷程度、位置、允许限度需由供需双方协商确定；标准要求铸件浇口、飞边、溢流口、**隔皮**等残留部分原则上需清理干净，仅允许留下符合要求的轻微痕迹 [[S35,S15]]。
- GB/T 13821 标准配套的《各类压铸件表面缺陷不同级别的要求》表格依据 1–3 级表面质量等级明确了包含串皮/隔皮类凸起缺陷的面积百分比、深度限制规则，可直接作为串皮缺陷评级的通用参考依据 [[S43]]。
- 公开可查的 NADCA 产品规范标准、ISO 16232 相关标准文本中，未检索到与中文“串皮”完全对应的专属缺陷名称及对应评级规则，此类模具错位/分型面挤入金属产生的凸起/位移类缺陷，在国际压铸行业标准中通常归类于 **mold joint misalignment**、**flash caused by die shift**、**die blow** 维度下管控 [[S13,S16,S32]]。

## 预防与去除措施

### 预防措施

**模具设计与维护**：定期检查清理分型面杂物，修整磨损的滑块、镶块、闭锁元件，校验导柱导套磨损量保证定位精度，校核模具整体刚度避免高压射料下模壁位移，定期对服役到规定模次的模具做去应力退火维护 [[S23,S41,S11]]。

**工艺参数优化**：准确计算铸件投影面积与所需锁模力的匹配值，定期校准压铸机锁模力和增压曲线参数，调整压射比压峰值避免过大压力冲击模壁，适配合理的浇注温度、模具预热温度、留模时间参数 [[S23,S45,S42]]。

**涂料管理**：选择适配对应压铸合金的合格脱模剂，控制喷涂量避免涂料堆积在分型面造成合模间隙，喷涂后保证稀释剂完全挥发再合模，同时定期清理排气道避免堵塞错位 [[S7,S6,S44]]。

### 去除措施

| 去除方法 | 特点与适用场景 |
|---------|--------------|
| 手工打磨 | 使用锉刀、手持式砂轮打磨串皮凸起部位；操作灵活，适用于小批量复杂铸件的局部修复；加工效率低，容易过度打磨损伤铸件基体，需操作人员熟练控制打磨量 [[S7,S45]] |
| 冲模冲切 | 采用专用冲模搭配压力机冲除分型面大面积串皮连带飞边；加工效率高，切口规整一致性好；适合大批量生产的中大型铸件清理，但需定制专用冲模，前期成本较高 [[S7,S6]] |
| 振动滚磨 | 将铸件放入螺壳式振动清理机，搭配对应规格磨料进行批量滚磨；自动化程度高，可同时处理小批量小型铸件的多位置串皮及毛刺，清理过程不易产生碰撞损伤；不适合高精度薄壁精密铸件 [[S6,S45]] |
| 喷丸清理 | 采用高速喷射的钢丸或石英砂冲击串皮凸起部位；清理后同步可强化铸件表层、去除氧化皮，清理效率高；需控制喷丸压力避免过度冲击造成薄壁铸件变形；适合外观要求不高的非精密结构件串皮处理 [[S7,S45]] |

## 相关术语对比

| 术语 | 定义 | 与串皮的核心差异 |
|------|------|----------------|
| 飞边（joint flash） | 压铸件在分型面边缘上突出的金属薄片，属于溢料类缺陷 [[S19]] | 串皮属缺肉类缺陷，飞边属溢料类缺陷，形成机理完全不同 [[S19,S46]] |
| 冷隔（cold shut） | 两股或多股半凝固金属液汇合时未完全熔合形成的不连续缝隙/对接痕，常出现在薄壁远离浇口的位置 [[S39,S27]] | 冷隔属充填连续性缺陷，串皮属型芯外露/分层类缺陷，特征和成因完全不同 |
| 隔皮 | 国内标准（如 GB/T 15114）中使用的术语，指铸件分型面或活动部位残留的薄层金属，与压铸现场的串皮概念部分重叠 [[S35]] | 隔皮常与串皮混用，在标准语境下属于清理残留类要求管控的项目 |

## 术语说明

“串皮”在国内压铸行业的实际使用具有明显的**行业俗称性质**。GB/T 5611-2017 中的标准定义主要适用于熔模铸造领域；在压铸生产现场，该词常被用来泛指分型面处因金属液渗透形成的分层、起皮、薄层状缺陷，与“隔皮”“分层夹皮”等术语含义相近 [[S46,S35,S26]]。

公开检索到的英文压铸标准和技术资料中，不存在 “skin defect die casting”“metal penetration skin die casting”“thin web defect die casting”“crost feather” 等假设词汇与中文压铸“串皮”的对应定义；仅 “open core” 是通用铸造标准中收录的与“串皮”对应的标准英文术语 [[S32,S36,S46]]。
