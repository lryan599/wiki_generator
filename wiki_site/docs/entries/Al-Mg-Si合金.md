## 概述
Al-Mg-Si（Al-6Mg-2Si）压铸铝合金是指名义镁含量约6 wt%、硅含量约2 wt%的铝-镁-硅系铸造合金。该合金目前**不属于**中国GB/T 15115系列、日本JIS ADC系列、美国ASTM系列、欧洲EN系列等公开压铸铝合金标准牌号中的任何一个通用牌号，其总Mg+Si质量占比通常超过8 %，远高于常规标准化Al-Mg-Si类压铸合金通常低于2 %的水平[[S25,S23]]。因此，该成分范围属于企业或实验室研发的非标准高镁压铸Al-Mg-Si合金，旨在通过高镁固溶强化与高体积分数Mg₂Si沉淀强化，在免热处理或轻热处理状态下兼顾高强度与塑性[[S1,S10]]。

## 名义化学成分
基于同体系合金的研究，Al-6Mg-2Si型压铸合金的典型成分控制区间如下[[S27,S18]]：

| 元素 | 含量范围（wt %） | 说明 |
|------|----------------|------|
| Mg | 5.5 ~ 6.5 | 主合金元素，提供固溶与沉淀强化 |
| Si | 1.8 ~ 2.2 | 主合金元素，与Mg形成Mg₂Si相 |
| Fe | ≤ 0.8 | 杂质，控制粘模倾向 |
| Cu | ≤ 0.2 | 杂质 |
| Mn | ≤ 0.5 | 微合金化/杂质 |
| Zn | ≤ 0.2 | 杂质 |
| Al | 余量 | 基体 |

该合金通过**镁显著过量**设计，实际Mg/Si质量比约**3.0**，远高于生成Mg₂Si相的理论质量比**1.73:1**[[S13,S27,S11]]。镁过量的主要作用是降低Mg₂Si在铝基体中的固溶度，优化时效析出动力学，并避免常规过剩硅型Al-Mg-Si合金易发生的晶间腐蚀倾向，从而提升耐蚀性能[[S13,S23]]。

## 微观组织与相组成
### 铸态组织特征
高Mg含量Al-Mg-Si压铸合金的铸态组织主要由**α-Al基体**、**骨骼状Mg₂Si相**、**点状或网络状Al₆Mg₄Cu相**以及少量**Al₆Mn相**组成[[S9]]。在压铸快速凝固条件下，Mg₂Si相形核析出速度快，其尺寸较常规铸造显著细化，通常呈骨骼状形态[[S9,S20]]。当合金Mg含量接近6 wt%时，Al₆Mg₄Cu相多为点状非连续分布；随着Mg/Si比进一步增大，该相逐渐转变为连续网络状或条块状[[S9]]。

### 析出行为与强化机制
Al-Mg-Si合金中Mg₂Si相的完整析出序列为：**GP区（针状，直径约6 nm，长200~1000 nm）→ 亚稳过渡相β'（高度有序的Mg₂Si）→ 平衡相β(Mg₂Si)** [[S28]]。在该合金中，过饱和固溶的Mg和Si原子在时效过程中脱溶形成弥散分布的β'和β相，起到显著的沉淀强化作用；同时，基体中固溶的Mg元素本身可带来约**34 MPa/wp(Mg)%**的抗拉强度增量[[S17,S22]]。

![图2：Al-Si-Mg三元系富铝角相图，标注α(Al)、Mg₂Si、Si等相区及577℃、555℃、449℃等特征温度](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/53c3d146-eaa9-422f-a29e-86d04fe5b714/resource)  
*图2：Al-Si-Mg三元系富铝角相图，示出各相区及关键三相平衡温度[[S15]]*

Al-Mg₂Si伪二元系共晶温度约为**594 ℃**，Mg₂Si在铝中的最大固溶度在583.5 ℃时约为**1.9 wt%**，而室温固溶度仅为约**0.08 wt%**，这一宽幅固溶度变化是固溶与时效强化的热力学基础[[S4,S13]]。采用镁过量设计可进一步降低Mg₂Si在基体中的固溶度，强化析出效果[[S13,S23]]。

## 物理与力学性能
目前尚无该精确成分合金的标准化性能数据。作为参考，同体系**Al-6Mg-1Si-0.4Mn-0.5Cu-0.15Zr**压铸铝合金在铸态下可获得的典型力学性能如下[[S2]]：

| 性能指标 | 数值 |
|----------|------|
| 屈服强度 (YS) | 170.0 MPa |
| 抗拉强度 (UTS) | 268.7 MPa |
| 断后伸长率 (EL) | 5.8 % |

商业化的高Mg压铸Al-Mg-Si合金**Magsimal‑59**（名义成分AlMg5Si2Mn）在免热处理铸态下实测抗拉强度约**300 MPa**，屈服强度约**150 MPa**，延伸率可达**25 %**，通过Mg₂Si第二相强化实现了优异的强塑匹配[[S1,S10]]。上述数据表明，合理设计的Al-6Mg-2Si型合金有望在铸态下同时获得较高的强度和可接受的塑性。

## 压铸工艺特性
### 推荐工艺窗口
经工业验证，Al-6Mg-2Si型压铸Al-Mg-Si合金的推荐工艺参数区间如下[[S26]]：

| 工艺参数 | 推荐值或范围 | 单位 |
|---------|-------------|------|
| 浇注温度 | 660 ~ 710 | ℃ |
| 模具预热温度 | 180 ~ 260 | ℃ |
| 压射比压 | 50 ~ 100 | MPa |

### 工艺行为对比
与常规通用压铸铝合金（如ADC12、A380）相比，该高镁Al-Mg-Si合金表现出以下工艺特性差异[[S16,S14]]：

| 特性 | 相对于ADC12/A380 |
|------|----------------|
| 充型流动性 | 更低 |
| 热裂倾向 | 更高 |
| 粘模倾向 | 更弱 |

其较宽的凝固温度区间是导致流动性偏低和热裂倾向增高的主要原因；而较高Mg含量下形成的表面氧化膜以及适量Fe元素的存在，则有助于减轻压铸过程中的粘模问题[[S16,S14]]。作为参考，常用压铸合金ADC12的实测液相线温度约**580 ℃**，固相线温度约**515 ℃**[[S21]]。

## 典型应用
该型合金已在实际零部件中得到商业化验证，典型应用场景包括[[S5,S31]]：
- **汽车后副车架**等结构件
- **5G通信基站散热壳体**
- **新能源电动机壳体**
- **高比强度电动工具承重部件**

这些应用充分利用了该合金无需完全固溶处理即可获得的较高强度、良好塑性及耐腐蚀性能。

## 缺陷与限制
### 常见压铸缺陷
- **缩松**：当Mg/Si配比不当时，或在压铸过程中增压不及时、补缩不充分，易在凝固末期产生缩松缺陷[[S7,S16]]。
- **热裂**：较宽的凝固温度区间使该合金具有较高的热裂敏感性[[S16,S12]]。
- **氧化夹杂**：高Mg含量使熔体表面易形成氧化膜，若熔炼及充型过程控制不当，可能卷入形成氧化夹杂[[S7]]。

### 使用局限性
- **原料成本**：较常规ADC12压铸合金高出约**20 %~30 %**[[S30]]。
- **焊接性能**：合金焊接性较差，焊接过程中易出现热裂纹，需要采取特殊焊接工艺[[S30,S29]]。

## 标准与牌号映射
截至目前，**Al-6Mg-2Si**这一成分范围尚未被GB/T 15115—2024、ISO、ASTM、JIS ADC/AD系列、EN等主流压铸铝合金标准收录为独立牌号[[S25,S23]]。在实际设计与选材时，多将其视为与商业高Mg压铸Al-Mg-Si合金（如Magsimal‑59、Castaduct‑42等）同体系但不同成分的研发合金[[S1,S14]]。使用者应依据具体化学成分和供应商提供的技术数据单进行工程应用，不宜直接套用某一标准牌号的性能保证值。

> 注：本文所述“Al-Mg-Si合金”在语境中特指约含6 wt% Mg、2 wt% Si的压铸铝合金，与标准中总合金元素含量通常低于2 %的常规Al-Mg-Si变形或铸造合金有显著成分差异。对该成分范围的公开标准化信息目前尚不充分，部分性能数据取自已发表文献中的相近成分实验合金，实际应用时应以材料实测数据为准。
