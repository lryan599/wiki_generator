## 概述

定模是压铸模具中固定在压铸机定模安装板上的模具部分，在压铸生产循环中保持不动，与随压铸机动模板移动的动模配合闭合，共同形成铸件的型腔与浇注系统通道。根据GB/T 8847—2003《压铸模术语》，定模的标准英文术语包括fixed die、stationary die half以及cover die half [[S36,S19,S42]]。

在不同工程语境中，定模存在多个别名。“固定模”在压铸领域广泛使用。在注塑模具行业，通常称为“前模”或“母模”，部分场景下也用于指代压铸模具中朝向定模安装板的半模。尽管称呼不同，其核心均指向不随机型开合动作位移的固定半模。压铸领域的定模直接连接压射系统，与部分可配置推出机构的注塑定模在使用上存在差异 [[S32,S13,S34]]。

## 典型结构与组成

定模由多个功能部件共同构成，以完成承接金属液、成型铸件对应部分型腔、实现均匀温控、对接压射系统以及导向定位等任务。其常规组成包括 [[S42,S22,S5]]：

| 部件 | 说明 |
|------|------|
| 定模座板 | 安装定模于压铸机定模安装板上的板件 |
| 定模套板 | 固定定模镶块、型芯、导柱或导套等零件的板件，起承托与增强刚性作用 |
| 定模镶块 | 型腔成型的主体零件，安装在定模套板内 |
| 浇口套 | 构成直浇道，贯穿定模座板与镶块对接，承接压室或喷嘴流出的金属液 |
| 冷却水道 | 布置在型腔附近，实现模具温度调节 |
| 导柱/导套 | 导向定位零件，通常导柱安装于定模，导套安装于动模 |
| 分流锥 | 用于中心进料结构，对金属液实施分流导向 |

**定模型腔的成型部分可分为整体式与镶拼式两类** [[S14]]：整体式结构的型腔直接在整块模块上加工成型，强度高、刚性好、装配量少且易于设置冷却水道，适用于型腔较浅的小型单腔模具、短期小批量不需热处理的模具，以及受拉杆位置限制无法采用镶拼结构的场景。镶拼式结构则由多块独立镶块组合装入定模套板固定，可简化复杂成型面加工、节省耐热钢材、易更换易损件，利于型腔排气并减小热处理变形，广泛用于型腔较深或大型模具、多型腔模具及成型表面复杂的模具。

## 工艺角色与工作原理

定模在压铸循环中保持完全固定。合模时，定模与动模闭合构成完整的型腔与浇注系统通道，高压金属液经定模侧浇注系统充填型腔并凝固；开模时，动模随压铸机动模板移动分离，铸件通常随动模侧结构脱离定模型腔，由动模侧的推出机构顶出取件 [[S28,S37,S51]]。

**安装方式**：定模通过定模座板上的定位结构与压铸机定模安装板定位对接，采用压板配合紧固螺栓实现全周多点紧固，保证安装后定模端面与压射喷嘴的同轴度满足工艺要求 [[S23,S50]]。

**浇注系统的作用**：定模侧承接压射系统，内置直浇道以及与分型面对接的横浇道，共同构成金属液流入型腔的前置通道，保证金属液由压室或喷嘴平稳导入型腔入口，减少流态紊乱和卷气缺陷。对于中心进料结构，定模侧配置的分流锥可对金属流实现分流导向，优化充填路径并提升充型均匀性 [[S4,S2]]。

## 与动模的配合关系

### 合模导向定位

合模导向通过导柱与导套的配合实现定、动模的精准对位。通常情况下导柱布置在定模侧，导套对应安装在动模侧，导柱伸出分型面的高度须高于型腔内最高型芯，以避免合模或搬运中型芯碰撞受损 [[S53,S55,S40,S16]]。

导柱与导套的标准配合为间隙配合，常规选用H7/f7；导柱与定模固定板的固定配合选用H7/k6；配合面需预留环形油槽存储润滑剂，降低磨损并防止卡滞 [[S7]]。导柱直径可采用经验公式 \\(d = K \\sqrt{A}\\) 估算，其中 \\(A\\) 为模具分型面总表面积（mm²），系数 \\(K\\) 取值范围为 0.07~0.09，依分型面面积大小确定 [[S53,S55,S16]]。

对于大型压铸模，动、定模受热不均可能导致导柱与导套卡死。此时可采取加大配合间隙并增设圆柱辅助定位装置或矩形侧边定位结构的措施，以保证合模精度 [[S40]]。

### 铸件留置与分型面密封

开模时铸件留置在动模侧是压铸模具的基本设计要求。其原理是将动模侧型芯的包紧力设计得大于定模侧，铸件在冷却收缩过程中抱紧动模型芯，随动模从定模侧脱离，从而适配压铸机动模侧的推出机构完成取件，无须在定模侧额外增设推出结构 [[S11,S21]]。

分型面密封方面，定、动模合模后主分型面贴合间隙通常需控制在0.05 mm以内（排气槽区域除外）。非主分型面的预留间隙因压铸合金种类而异：铝合金宜控制在0.04~0.08 mm，锌合金控制在0.02~0.06 mm，以防止压铸过程金属液溢出形成飞边 [[S45]]。

## 设计参数与材料

### 材料与硬度

定模型腔及浇注系统常用热作模具钢包括H13、8407、SKD61、4Cr5MoV1Si等牌号。定模成型镶块的常规热处理硬度范围为43~47 HRC [[S9]]。根据GB/T 8844—2003，锌、铝、镁合金压铸成型零件经淬火后，成型面可进行表面渗氮处理，渗氮层深度 0.08~0.15 mm，硬度 ≥600 HV，以提升表面耐磨性和抗熔蚀性能 [[S48]]。NADCA #207标准对H13钢的杂质元素控制亦提出了严格要求：优级H13硫含量低于0.005%，超级H13硫含量低于0.003%、磷含量低于0.015% [[S3,S18]]。

### 定模配合公差

定模各配合面的典型公差范围如下 [[S45]]：

| 配合类型 | 配合代号 | 说明 |
|----------|----------|------|
| 固定类（套板与镶块、套板与浇口套等） | H8/h7 或 H7/h6 | 受热后不产生相对位移 |
| 滑动类（定模侧抽芯组件） | H7/f7 | 保证运动顺畅且无金属液窜入 |

导柱、导套对定模/动模安装基准面的垂直度公差，需依据导柱导套有效长度选取；模具分型面对安装基准面的平行度公差，则依据分型面最大轮廓长度选取，二者均需符合GB/T 4678系列压铸模零件标准的要求 [[S30]]。

### 冷却与加热系统

定模冷却系统的设计原则包括 [[S26,S43]]：
- 冷却水道优先布置在浇口附近等热量集中区域，水道壁距型腔表面 10~15 mm；
- 常规冷却水道直径取 8~16 mm，大尺寸模具按模坯宽度分级选取；
- 并联水道须保证各支路流程长度相等，避免冷却不均导致铸件变形；
- 水路需避开抽芯、推杆等活动构件，并做好连接处密封防止漏水。

定模的加热系统通常用于模具预热，电热元件安装孔可布置在定模套板或镶块内，孔位须避开活动构件；电热元件与安装孔的配合间隙 ≤0.8 mm 以保证传热效率，同时可在定模侧布置测温孔安装热电偶以实现模温的精准控制 [[S26]]。

## 常见缺陷与定模关联

### 定模与动模错位（错型）

错型是指型腔或型芯错位导致铸件形状偏移、尺寸失准的缺陷。其成因包括导柱松动、滑块与导轨配合不良、模具装配调整不良，以及成形镶块与模套配合不良 [[S10]]。

### 定模粘模（烧付/粘铝）

铝合金压铸时，若定模表面涂层脱落，且金属液温度高于655 ℃，熔融铝与模具表面充分接触生成 FeAl₃ 金属间化合物，逐步形成粘铝层。脱模时该层会拉伤铸件表面，反复循环还会损伤型腔面 [[S31,S17]]。

定模侧的改进措施包括 [[S8]]：
- 降低粘模区域定模温度（增设极小直径喷水冷却、强化局部喷雾）；
- 选用高导热系数定模材料（TZM、Anvilloy、Mitech）；
- 适当增大粘模位置的拔模角；
- 消除型腔凹槽并降低表面粗糙度；
- 控制铝合金Fe元素含量高于0.75%。

### 定模侧金属液冲蚀

高速流动的高温金属液直接冲刷定模型腔面（尤其在内浇口正对区域），先破坏表面保护氧化膜，随后发生热、力、化学反应耦合作用，导致模具材料逐步脱落形成冲蚀坑。可分为浇口冲蚀、主流冲刷冲蚀以及空泡溃灭引发的气蚀三类 [[S31,S39,S56]]。

定模侧改进措施包括：选用耐液态金属熔损性能好的模具钢材、提升表面涂层与基体的结合强度、适当降低冲蚀区域型腔表面温度，以及在满足充型要求的前提下降低金属液进入型腔的入口速度 [[S31]]。

### 定模侧表面热龟裂

定模在反复急冷急热循环中，表面承受循环热应力作用而产生疲劳裂纹（通常深度不超过1 mm）。定模型腔钢材的抗拉强度与热稳定性越高，其抗热龟裂能力越强 [[S31]]。

### 定模侧碳化物聚积

压铸润滑剂残留或水-润滑剂混合沉积物在定模型腔表面不断累积形成碳化物堆积层，会导致铸件表面粗糙。改进措施包括将定模侧局部区域润滑剂喷涂量控制在最低限度、避免定模局部过冷，以及优先选用与定模工作温度匹配的脱模剂 [[S8]]。

## 与其他模具系统的接口关系

**浇注系统**：定模是浇注系统的安装载体，主流道浇口套直接嵌入并固定在定模座板与定模套板之间，其端面与压铸机射嘴贴合，金属液经定模内部直浇道进入型腔前侧的横浇道和内浇口 [[S46]]。

**定模抽芯**：对于带有侧向凹凸结构的铸件，抽芯滑块整体安装固定在定模套板的槽位内，其动力源（液压油缸）可直接安装在定模外侧。开模时滑块沿定模导向槽滑动完成侧向抽芯，合模时由定模侧锁紧块提供锁模力以防止高压金属液将滑块顶开 [[S1]]。

**定模顶出**：对于因包紧力过大可能留在定模型腔内的铸件，可在定模侧集成顶出机构，由定模侧驱动的推杆在开模时同步动作，将铸件从定模型腔中推出以避免粘定模 [[S52]]。
