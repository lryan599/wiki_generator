## 概述

离心铸造模是安装在离心铸造机旋转主轴上、可随主轴同步高速旋转的特种铸型[[S28,S9]]。在浇注和金属凝固过程中，该模具同时承受熔融金属施加的离心载荷与循环热冲击，是离心铸造工艺中直接决定铸件成形质量的核心成型部件[[S28]]。

离心铸造模的核心功能包括四个方面：其一，驱动注入的熔融金属在离心力作用下完成充型，无需额外型芯即可成形回转体类零件的内孔；其二，促进金属液内部裹挟的密度较低的夹渣、气体向铸件内表面定向迁移，实现熔体净化；其三，控制铸件由外到内的定向凝固过程；其四，有效提升铸件致密度与综合力学性能[[S47,S17]]。

## 分类

离心铸造模可从四个独立维度进行完整分类[[S7,S33,S10,S29,S12]]：

| 分类维度 | 类型 | 简要说明 |
|---------|------|---------|
| 旋转轴空间方位 | 卧式离心铸造模 | 铸型绕水平轴旋转，主要用于管、套类长铸件 |
| | 立式离心铸造模 | 铸型绕垂直轴旋转，主要用于环、轮、辊类短铸件 |
| | 倾斜轴离心铸造模 | 铸型绕倾斜轴旋转，用于双金属轧辊、锥形套等 |
| 离心力作用模式 | 真离心铸造用模 | 仅靠离心力成形内孔，铸件轴线与旋转轴重合 |
| | 半真离心铸造用模 | 中心孔可由型芯形成，离心力辅助充型与凝固 |
| | 非真离心铸造用模 | 铸件轴线与旋转轴不重合，离心力仅增加凝固压力 |
| 铸型基体材料 | 金属型离心铸造模 | 用于大批量长周期生产 |
| | 砂型离心铸造模 | 用于大尺寸单次使用场景 |
| | 带耐火层的衬里金属型离心铸造模 | 金属型内衬砂或涂料 |
| | 石墨/石膏等特种材料离心铸造模 | 用于中小批量或特种场景 |
| 铸型工作温度控制模式 | 冷模（水冷）离心铸造模 | 铸型本体温度在40~250℃之间波动 |
| | 热模离心铸造模 | 铸型温度在350~450℃之间波动 |

立式离心铸造与卧式离心铸造的旋转轴线差异直接决定了适用铸件形态。立式离心铸造模绕垂直轴旋转，自由表面呈抛物面，适宜生产高度小于直径的环类、轮类铸件；卧式离心铸造模绕水平轴旋转，自由表面呈圆柱面，适宜生产长度较大的管、筒、套类铸件[[S1]]。

## 结构与组成

真离心铸造用金属型模具呈整体回转筒型结构，型腔直接形成铸件外表面[[S45,S24]]。典型结构包括型体、端盖、浇注系统、排气装置等组成部分[[S3]]。由于依靠离心力自然形成中空内孔，该类模具无需配置复杂的抽芯机构、溢流系统和专用顶出机构[[S45,S17,S12]]。

## 材料选型

离心铸造模基体材料的选型规则与生产批量、合金种类密切相关[[S3,S17,S27,S8,S11]]：

| 适用场景 | 推荐材料 | 选型理由 |
|---------|---------|---------|
| 大批量长周期生产 | 21CrMo10合金钢 | 抗热裂纹性能优异，为首选材料 |
| 一般金属型铸模 | 20CrMo、30CrMo、34CrMo等低合金钢 | 常规工况适用 |
| 中小批量短周期生产 | 石墨 | 机加工难度低，成本更低 |
| 低熔点轻合金浇注 | 铜合金 | 适配轻合金工况 |
| 钛合金等高活性合金 | 4Cr5MoV1Si模具钢 | 满足高活性合金浇注要求 |
| 大尺寸单次使用 | 干砂型 | 一次性使用，成本低 |

## 涂层与技术要求

离心铸造模内壁涂层具有多重作用：降低金属液与模具的摩擦以提升脱模成功率，避免铸件表层过快冷却产生白口缺陷，削弱金属液对模具的热冲击以延长使用寿命，提升铸件表面光洁度，并通过增加金属液与模壁摩擦力来加快金属跟随模体同步旋转的速率[[S25,S16]]。

不同场景下的涂层选型规则[[S30,S14,S27,S3]]：

| 应用场景 | 涂层方案 | 技术要求 |
|---------|---------|---------|
| 常规铸铁管类铸件 | 硅石粉（SiO₂质量分数≥98%）+钠基膨润土或活化钙基膨润土粘结，载体为水 | 禁用石墨以避免高导热性导致激冷缺陷；环保禁硅场景替换为刚玉粉 |
| 铸铁水管类铸件 | 上述涂层中添加硅藻土 | 外添2%~3%硅溶胶可提升粘结强度 |
| 铝合金2xxx系真空离心铸造 | 0.5mm厚纳米氮化硼复合水基涂层 | 可大幅提升铸件延伸率，减少表面气孔、热裂纹 |
| 钛合金浇注 | Y₂O₃基陶瓷涂层 | 极低的熔融钛合金反应活性 |
| 普通离心铸铁管水冷金属型 | 无需喷涂常规耐火涂层，直接采用石墨粉 | 作为防护层使用 |

## 工艺参数与控制要求

离心铸造模的运行状态由多个关键工艺参数决定，各参数的选择直接影响铸件质量[[S47,S42,S36,S40]]：

**（1）模具转速（离心转速）**

转速过低会出现“雨淋”现象，导致铸件产生缩松、夹杂缺陷；转速过高会引发熔体充型紊乱、飞溅断流，造成铸件偏析、冷隔、裂纹缺陷[[S42,S36,S47]]。

铝合金离心铸造适配重力系数范围为80~120，转速计算修正系数β取值范围0.9~1.1[[S47,S40]]。对高稀土镁合金，经实验验证的优化工艺参数为：离心转速400~1092 r/min，浇注时间15 s，浇注温度730 ℃，铸型预热温度150 ℃[[S20]]。

**（2）模具预热温度**

预热温度过低会使熔体冷却速度过快，晶粒形态不易控制，容易引发冷隔缺陷，并阻碍气体和夹杂物排出；预热温度过高会延长熔体凝固时间，造成铸件晶粒粗大、出现偏析，且批量生产时过高的预热温度会引发模具热疲劳，导致内壁龟裂，缩短模具寿命[[S42,S36,S40]]。

**（3）浇注温度**

离心铸造因离心力提升了充型能力，相较传统重力铸造的同合金浇注温度可降低5~10K[[S40]]。浇注温度过高会造成晶粒粗大、偏析，同时缩短金属模具寿命；浇注温度过低会降低熔体流动性，充型不足，引发冷隔缺陷[[S40,S36]]。

**（4）涂层条件**

涂层的种类、厚度、喷涂方式改变铸件与模具间的换热系数，直接影响铸件的冷却速度和温度梯度分布，进而影响疏松倾向和缺陷形成概率[[S40]]。

## 缺陷与模具关联因素

离心铸造常见的缺陷类型及其与模具有关的成因[[S47,S35,S37,S40,S12,S44]]：

| 缺陷类型 | 核心成因 | 模具相关诱因 |
|---------|---------|-------------|
| 偏析（成分/密度偏析） | 密度差大的两相合金中，不同密度的相在离心力作用下沿径向向外/向内聚集 | 模具冷却速率过低；转速选择不当；模具预热温度过高延长凝固时间 |
| 纵向裂纹 | 旋转系统不平衡，模具动平衡精度不足，运行过程中产生周期性交变离心力 | 旋转系统不平衡；铸型平衡精度不足 |
| 横向裂纹 | 铸件轴向收缩受阻，热应力集中 | 模具表面存在缺陷；模具内壁阻力过大阻碍铸件轴向收缩；模具预热温度过低 |
| 粘模/表面拉伤 | 熔体与型壁焊合 | 模具表面未喷涂足量防粘涂料；模具表面粗糙度超标；模具预热温度过低 |
| 淋落（坍流） | 合金液逆旋转方向向下淋落，在铸件内表面形成凹陷、凸起或金属瘤 | 铸型转速过低；过早停机；砂型离心铸造时模具局部过热 |

## 应用与典型产品

离心铸造模适配的铸件形状以旋转对称类铸件为主，包括管类（输水/输气球墨铸铁管）、套筒类、环类、轧辊、气缸套、轴套，同时适配半真/非真离心铸造的异形件如叶轮、金属假牙、小型阀门、电机转子等[[S1,S13]]。

适配的合金范围几乎覆盖所有常用铸造合金，包括各类铸铁、铸钢、铝合金、铜合金（铝青铜、锡青铜等）、镁合金、高温钛合金。可生产铸件的最小内径为8 mm，最大直径达3 m，最大长度为8 m，铸件重量覆盖数克到数十吨[[S13,S1,S41]]。

## 与压铸模的异同对比

离心铸造模与压铸模同属金属型模具范畴，但两者的设计理念、结构特征、服役条件和生产场景存在系统性差异[[S45,S43,S23,S18,S48,S31,S2,S24,S17,S12,S22,S4,S46]]。

| 对比维度 | 离心铸造模 | 压铸模 |
|---------|-----------|-------|
| 核心驱动力 | 铸型以250~1500 r/min旋转，依靠离心力驱动金属液向模壁外侧充填、定向凝固 | 金属液以5~50 m/s速度在高压下充型，承受数十至数百大气压的压射力 |
| 典型结构 | 回转筒型结构，无需抽芯、溢流和专用顶出机构，依靠金属液充填后自然形成中空内孔 | 动模+定模对开结构，内置抽芯机构、顶出系统、溢流排气槽系统、加热/冷却循环流道，须设置刚性锁模框架 |
| 结构复杂程度 | 结构简单，加工难度低，初始制造成本低 | 结构复杂，设计制造加工精度要求极高，整体造价昂贵 |
| 常用模具材料 | 可选范围广：21CrMo10等合金钢、铸铁、石墨、砂型；铜合金和模具钢用于特种合金 | 3Cr2W8V、H13等热作耐热合金钢 |
| 主要服役载荷 | 旋转离心载荷，热交变冲击强度远低于压铸模 | 5000~20000 psi级别的压射锁模力，同时承受高频冷热交变冲击 |
| 典型模具寿命 | 金属型可完成数千至上万次浇注；一次性砂型仅可单次使用 | 锌合金铸件约30万模次，铝合金铸件约15万模次 |
| 生产场景适配性 | 适配中空回转类铸件，可生产最大直径2000 mm、最大长度8000 mm的大型筒类铸件；铸件内孔表面通常需后续机械加工 | 适配大批量生产薄壁复杂形状的小型非铁金属铸件；尺寸精度可达IT11~IT13，表面粗糙度可达Ra0.8~Ra3.2；可直接铸出小孔、螺纹、精细花纹 |
| 经济性 | 小批量甚至单件生产场景下经济性更优 | 需要通过数十万件的大批量生产才能摊薄模具成本 |
