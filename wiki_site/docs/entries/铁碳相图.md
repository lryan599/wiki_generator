## 概述

铁碳相图（Iron‑carbon phase diagram）是描述铁‑碳二元合金在缓慢冷却（或缓慢加热）平衡条件下相组成与温度、碳含量关系的图解，是选材及制定铸造、锻造、焊接、热处理工艺的核心理论基础[[S42,S48]]。由于铁碳合金中碳可以渗碳体（Fe₃C）或石墨（G）两种形式出现，实际存在两套相图[[S44]]：

* **亚稳系 Fe–Fe₃C 相图**（实线表示）：适用于常规冷却速度，此时碳以渗碳体形式存在。渗碳体为亚稳定相，在适当条件（高温长时间保温）下可分解为铁和石墨[[S52]]。
* **稳定系 Fe–G（石墨）相图**（虚线表示）：仅在极缓慢冷却或合金中含有促进石墨化的元素（如Si）时成立，石墨为热力学更稳定的平衡相。稳定系中所有相变线均位于亚稳系对应相变线的上方或左上方[[S52,S44,S32]]。

亚稳系Fe–Fe₃C相图研究的碳质量分数范围为0～6.69%，其中w(C)＞5%的铁碳合金脆性过高而无实用价值[[S20,S24]]；实用铸铁的碳含量通常处于2.4%～4.0%区间[[S24]]。图1为铁‑碳合金双重相图，同时标注了稳定系与亚稳系以及各特征点的温度和成分[[S36]]。

## 核心相与组织

铁碳合金在固态下存在五种基本相和组织：铁素体（F）、奥氏体（A）、渗碳体（Fe₃C）、珠光体（P）和莱氏体（Ld / Ld′）。其中铁素体、奥氏体和渗碳体为单相，珠光体和莱氏体为机械混合物[[S17]]。各相/组织的特征与力学性能见表 1。

| 相/组织 | 晶体结构 | 碳含量 / 最大溶解度 (质量分数) | 典型力学性能 | 备注 |
|----------|----------|-----------------------------|--------------|------|
| 铁素体 (F, α) | 体心立方 | ≤0.0218% (727 ℃)，室温约0.0008%[[S53,S13]] | 硬度50～80 HBW，抗拉强度180～280 MPa，伸长率30%～50%，冲击吸收功128～160 J[[S53]] | 770 ℃以下具有铁磁性[[S53]] |
| 奥氏体 (A, γ) | 面心立方 | 1148 ℃时2.11%，727 ℃时0.77%[[S6,S53]] | 硬度160～220 HBW，抗拉强度≈400 MPa，伸长率40%～50%[[S53]] | 塑性优异的非铁磁性高温相，仅在727 ℃以上稳定存在[[S53,S6]] |
| 渗碳体 (Fe₃C, Cm) | 复杂斜方 | 6.69%（固定）[[S15,S6]] | 硬度≈800 HBW，伸长率与冲击韧度几乎为零[[S15,S6]] | 硬而脆，高温长时间保温可分解为铁和石墨[[S15]] |
| 珠光体 (P) | 层片状铁素体＋渗碳体 | 约0.77%[[S27,S8]] | 抗拉强度≈770 MPa，硬度180 HBW，伸长率20%～30%，冲击韧度30～40 J/cm²[[S27]] | 性能介于铁素体与渗碳体之间 |
| 莱氏体 (Ld / Ld′) | 高温：奥氏体＋渗碳体<br>低温：珠光体＋渗碳体 | 平均4.3%[[S17,S8]] | 硬度＞700 HBS，塑性极差[[S17,S8]] | 白口铸铁的基本组织 |

## 关键转变点与特征线

亚稳系Fe–Fe₃C相图内包含包晶、共晶、共析三个恒温转变，以及表征溶解度变化的特征线。

### 恒温转变

| 转变类型 | 温度 (℃) | 反应式 | 产物 |
|----------|----------|--------|------|
| 包晶转变 | 1495 | L₀.₅₃₀% C + δ₀.₀₉% C → A₀.₁₇% C[[S33,S35]] | 奥氏体 |
| 共晶转变 (亚稳系) | 1148 | L₄.₃₀% C → A₂.₁₁% C + Fe₃C[[S22,S30]] | 高温莱氏体 (Ld) |
| 共析转变 (亚稳系) | 727 | A₀.₇₇% C → F₀.₀₂₁₈% C + Fe₃C[[S1,S38]] | 珠光体 (P) |

稳定系中对应转变温度更高、平衡碳含量更低（共晶点1153 ℃ / 4.26% C，共析点736 ℃ / 0.69% C）[[S28,S41]]。

### 重要特征线

* **ECF 共晶线**（1148 ℃）：碳含量介于2.11%～6.69%的合金均发生共晶转变[[S30]]。
* **PSK 共析线（A₁ 线）**（727 ℃）：碳含量大于0.0218%的合金中的奥氏体均发生共析转变[[S35,S38]]。
* **ES 线（Acm 线）**：碳在奥氏体中的溶解度线，冷却时从中析出二次渗碳体 Fe₃CⅡ[[S35]]。
* **GS 线（A₃ 线）**：冷却时奥氏体开始析出铁素体的线，或在加热时铁素体完全溶入奥氏体的终了线[[S35]]。
* **PQ 线**：碳在铁素体中的溶解度线，冷却时可能析出三次渗碳体 Fe₃CⅢ[[S35]]。

## 杠杆定律在相图计算中的应用

在铁碳相图的任意两相区，可通过杠杆定律计算两平衡相的相对质量[[S34,S7]]。设合金的整体碳质量分数为 C₀，在两相区作水平连接线（结线），读取两端点对应的平衡相碳质量分数 Cα 和 Cβ（Cα＜Cβ），则两相的质量分数 Wα 和 Wβ 满足[[S34]]：

Wα + Wβ = 1，  Wα·Cα + Wβ·Cβ = C₀

导出：
Wα = (Cβ − C₀) / (Cβ − Cα)， Wβ = (C₀ − Cα) / (Cβ − Cα)

该公式仅在碳含量以质量百分比表示时直接适用；也可通过直接测量结线各段线段长度按比例计算[[S34]]。

## 铁碳相图在压铸领域中的应用

### H13 压铸模具钢的热处理窗口

H13钢（4Cr5MoV1Si）的临界点 Ac₁ = 850 ℃，Ac₃ = 910 ℃，经合金化后奥氏体区发生偏移[[S4,S5]]。根据铁碳相图的指导，淬火加热温度应定在单相奥氏体区的 1020～1080 ℃[[S4,S26,S51]]：
* 以韧性为主的压铸模具：1020～1050 ℃ 淬火；
* 以热硬性为主的压铸模具：1050～1080 ℃ 淬火。

回火工艺窗口划定为 530～620 ℃，必须避开 500 ℃ 附近的回火二次硬化脆性峰，否则模具韧性大幅劣化[[S51,S5,S40]]。推荐进行两次回火；铝合金压铸模具回火后硬度控制在 42～46 HRC，可适配常规压铸工况[[S51]]。

### 铸铁坩埚与压射料缸的服役温度限

铸铁坩埚长期服役温度需严格控制在 Fe–Fe₃C 相图的共析温度 (727 ℃) 以下，以免珠光体转变为奥氏体，导致蠕变抗力骤降、寿命锐减[[S31,S12]]。具体温度管理要求：
* 镁合金 AZ91 压铸用铸铁坩埚长期保温温度：650～680 ℃[[S46,S10]]；
* 普通铝/锌合金用铸铁坩埚长期保温温度不得超过 730 ℃——该温度为铸铁居里点，磁性消失会导致工频感应炉加热效率骤降[[S12]]。

### 黑色金属压铸件的浇注温度与工艺设计

根据铁碳相图，浇注温度一般设定在液相线以上 50～100 ℃[[S14,S37]]。共晶或近共晶成分的铸铁凝固温度区间最小，流动性好，分散缩孔倾向低，广泛用于压铸生产[[S37,S18]]。含碳量 0.2%～0.6% 的铸钢因结晶温度范围较大，需匹配更快的压射冷却速率，防止粗大枝晶和缩松[[S14,S50]]。高碳钢类黑色金属压铸件可采用“低温压铸、半固态充型”的工艺方案，以补偿大凝固区间带来的缺陷倾向[[S18]]。

### 铝合金固溶时效及冲头的材料选择

铝合金固溶处理温度通常为 500～550 ℃，在该温度区间内，可依据铁碳相图选用不发生非预期相变的钢材制作热处理工装，避免工装变形影响热处理精度[[S11,S47]]。同时，通过控制压射冲头的回火温度，可使其在连续服役过程中始终保持回火索氏体稳定组织，防止相变软化导致快速磨损失效[[S47,S40]]。
