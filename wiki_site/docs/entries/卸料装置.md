## 概述

在压铸领域，国家标准GB 20906-2007《压铸单元安全技术要求》将“顶出装置”明确定义为“从模腔中顶出压铸件的装置”，对应英文术语为 **ejector system** [[S2,S6,S25]]。行业中也常称其为**卸料装置**、脱模机构、出件装置或卸料机构，均为指向同一功能的通用别名。该装置是压铸机与模具系统的核心功能组件之一，其基本任务是在金属液在型腔内完全凝固、模具打开后，将留滞在动模侧的压铸件及浇注系统余料可靠地推出型腔，实现取件操作 [[S3,S23]]。

## 分类

压铸顶出装置可从驱动方式、安装位置及功能形式三个维度进行分类，不同类别适用于不同的工艺场景 [[S7,S23,S29,S30]]。

| 分类维度 | 类别 | 主要特征及适用场景 |
|---|---|---|
| **驱动方式** | 机械式顶出 | 依靠开模动作触发固定挡块推动顶出板，行程固定、不可调速，顶出瞬间存在一定冲击 [[S23,S29]] |
| | 液压式顶出 | 由独立顶出油缸直接驱动顶出板，顶出行程与速度均可调节，支持多次往复顶出动作，运行平稳，是现代主流压铸机的标准配置 [[S23,S29]] |
| **安装位置** | 动模侧顶出 | 压铸行业主流结构，适配大部分铸件随开模动作留滞在动模侧的工况 [[S23]] |
| | 定模侧顶出 | 用于铸件因结构或工艺原因粘附定模的特殊脱模需求 [[S23]] |
| **功能形式** | 顶杆顶出 | 通用形式，利用圆形截面顶杆在局部位置施力顶出铸件，适配常规铸件 [[S7,S30]] |
| | 推管顶出 | 采用管状顶出元件环绕固定型芯，适用于筒形、薄壁铸件 [[S7,S30]] |
| | 推板/卸料板顶出 | 利用大面积板状元件均匀施力，顶出过程在铸件表面不留顶痕，适用于大面积薄壁易变形铸件 [[S7,S30]] |

## 结构与组成

典型的顶出装置由若干功能明确的零件与组件构成，共同完成顶出、导向、限位及复位等动作序列 [[S1,S3,S4,S14,S15,S23,S30]]。

| 组成部件 | 功能说明 |
|---|---|
| 顶杆 (ejector pin) | 固定在推杆板上，直接接触铸件表面施加顶出力，将铸件从模腔中脱出 [[S14,S23]] |
| 推杆板/顶出板 (ejector plate) | 平行于机器动模板的钢板，用于安装固定所有顶出执行元件，传递压铸机输出的顶出力 [[S14,S23]] |
| 推杆固定板 (ejector retaining plate) | 将顶杆尾部固定于推板上，防止顶杆在往复运动中松脱移位 [[S14]] |
| 回程杆/复位杆 (return pin) | 装在推出板上的大直径杆件，合模过程中定模分型面首先接触回程杆，推动整个顶出机构回到初始浇铸位置完成复位 [[S14,S4,S15]] |
| 导向元件（导柱、导套） | 引导顶出板平稳往复运动，保证顶出动作同轴无卡滞 [[S4,S15,S23]] |
| 限位装置（限位钉、限位块） | 限定顶出板的最大顶出行程，保证顶出距离恰好满足铸件完全脱出的要求 [[S4,S15,S23]] |
| 拉料杆/浇道推杆 | 专门用于顶出浇注系统中的冷凝余料，将直浇道、横浇道、溢流槽内的全部固态金属料同步推出型腔 [[S15,S17]] |

顶杆常用材料为H13热作模具钢（5%铬型），热处理硬度控制在38～42 HRC。部分顶杆经氮化处理以进一步提升表面耐磨性能，防止卡滞故障 [[S11,S28]]。顶杆通常已高度标准化，可从专业供应商直接采购成品 [[S11,S28]]。

## 工艺作用

在压铸全循环中，顶出装置按严格的时序逻辑完成脱模动作 [[S3,S15,S17,S23,S24]]：

1. 压铸件在模腔内完全冷却凝固后，模具首先完成必要的侧抽芯动作。
2. 动、定模开模分离，铸件及浇注系统余料附着于动模侧。
3. 顶出装置在压铸机控制系统指令下启动，顶出位置均匀分布在铸件包紧力最大区域和铸件结构强度较高的部位，以保证铸件各处受力均衡，避免因局部应力集中导致的铸件变形或顶穿缺陷。
4. 同时，专门设置的浇道推杆将直浇道、横浇道及溢流槽内的全部冷凝余料同步顶出动模型腔，无需人工二次清理 [[S15,S17]]。
5. 顶出完成后，由回程杆或复位弹簧使顶出机构退回原位，为下一压铸循环做好准备 [[S4,S15]]。

## 关键设计参数

顶出装置的设计须综合考虑顶出力、行程、速度及复位精度等多项性能指标 [[S21,S22,S23,S26,S30]]。

| 参数 | 推荐取值范围或设计规则 |
|---|---|
| 顶出力 | 取铸件冷却收缩产生的包紧力的1.2倍；需根据不同压铸合金的许用顶出压强进行校核，避免顶出部位出现顶凹缺陷 [[S23]] |
| 顶出行程 | 保证铸件完全脱离动模型腔，通常需超出型芯深度3～5 mm，并预留取件操作空间 [[S23,S30]] |
| 顶出速度 | 常规范围0.1～0.5 m/s，可根据铸件结构复杂程度适当调整 [[S23]] |
| 多次顶出 | 支持1～3次顶出模式，适配复杂深腔铸件的脱模需求 [[S23]] |
| 多顶杆同步性偏差 | ≤0.1 mm，确保铸件各处顶出受力均匀 [[S21]] |
| 复位精度 | ≤0.05 mm，避免后续合模动作与侧抽芯机构发生干涉 [[S21]] |

国家标准GB/T 21269-2023《冷室压铸机》亦正式定义了**顶出力**为“压铸机顶出铸件时，压铸机顶出机构所产生的最大推力”，**顶出行程**为“压铸机顶出机构的最大运动距离”，将其列为冷室压铸机的基础性能参数 [[S22]]。

## 常见故障及缺陷

顶出装置在高温、高应力工况下可能出现多种故障模式 [[S11,S20,S28]]。

| 故障模式 | 成因及影响 |
|---|---|
| 顶出不平衡 | 多顶杆分布不对称或顶出板平行度偏差超限，导致薄壁铸件受偏载发生变形或局部顶穿；处置措施为重新校核顶杆分布并调整顶出板平行度 [[S20]] |
| 顶杆卡滞、磨损 | 配合间隙不当或表面硬度不足；采用氮化处理或优化配合精度可降低故障率 [[S11,S28]] |
| 复位不良 | 回程弹簧失效或回程杆磨损、定模面接触不良，可能导致合模时与侧抽芯机构干涉 [[S4,S15]] |

## 应用适配

冷室压铸机通常需配置更大顶出力的卸料装置，以适应铝、镁、铜等高熔点、高收缩率合金，顶出机构的结构刚度设计余量一般比热室压铸机高出30%以上 [[S10]]。热室压铸机配套的卸料装置行程较小，多用于锌合金等低熔点小型铸件 [[S10]]。

不同吨位压铸机对应着标准化的顶出力规格与安装接口尺寸，以下为行业常用参照数据 [[S12]]。

| 压铸机吨位 | 顶出力 (吨) |
|---|---|
| 200吨以下 | 6 |
| 201～400吨 | 10 |
| 401～600吨 | 15 |
| 601～800吨 | 20 |
| 801～1000吨 | 28 |
| 1001～1200吨 | 38 |
