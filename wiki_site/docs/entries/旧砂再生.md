## 概述

**旧砂再生**是通过焚烧、水洗、气流撞击或机械摩擦等物理或化学方式，去除旧砂颗粒表面包覆的惰性膜（残留粘结剂和其他杂质），使旧砂恢复其使用性能以代替新砂的工艺过程[[S40,S30]]。其核心工艺目的为实现铸造旧砂循环回用，减少新砂资源消耗，降低废砂排放污染，并降低铸造生产用砂成本[[S40]]。

旧砂再生是砂型铸造生产的重要组成部分，尤其对于采用化学粘结剂砂（如树脂砂、水玻璃砂）的造型制芯工艺而言，旧砂能否高效再生直接关系到系统的经济性与环保性[[S37]]。

---

## 再生工艺分类

旧砂再生主流工艺按原理可划分为四大类[[S37]]：

| 类别 | 英文名称 | 核心原理 | 适用范围 |
|------|----------|----------|----------|
| **机械再生（干法再生）** | Dry/Mechanical Reclamation | 依靠机械动力或空气动力，在干态下实现砂粒间、砂粒与设备间的碰撞摩擦去除惰性膜，无需额外加热或加水[[S12,S37]] | 各类旧砂通用，但单独使用对水玻璃旧砂效果有限[[S37]] |
| **热法再生** | Thermal Reclamation | 通过高温焙烧（800~900℃）使有机物燃烧分解，彻底去除砂粒表面残留粘结剂[[S12,S37]] | 有机粘结剂旧砂（如树脂砂）的理想工艺，也可用于水玻璃旧砂的辅助处理[[S37]] |
| **湿法再生** | Wet Reclamation | 以水为介质，通过溶解、擦洗、旋流去除惰性膜，以液相处理为核心特征[[S12,S37]] | 水玻璃旧砂效果最佳，因为残留粘结剂可溶于水[[S32]] |
| **化学再生** | Chemical Reclamation | 通过加入指定化学试剂（如氯化铝、海泡石、水滑石、CaCl₂、NaOH等）发生化学反应，溶解或中和残留有害成分[[S37,S18]] | 通常与热法或干法联合使用，单独大规模应用较少[[S37]] |

## 各再生工艺详解

### 1. 机械再生（干法再生）

#### 工艺原理

机械再生利用空气动力或机械动力将旧砂颗粒加速至一定速度，靠旧砂颗粒之间或与设备之间发生碰撞和摩擦，使砂粒表面的粘结剂膜产生破裂、剥离以实现再生[[S37,S24]]。常见形式包括机械离心撞击式、气流撞击式、高压擦磨式等，其再生机理均为“碰撞-摩擦”[[S24]]。

#### 典型设备与关键参数

干法再生典型工业设备包括机械离心撞击式再生机、两级串联干法再生机（如日本太洋铸机产品）、高压擦磨式再生机（如日本新东公司产品）、气流撞击式再生机等[[S39,S24,S25]]。常规操作参数如下[[S39,S24,S34]]：

| 参数 | 典型数值/要求 |
|------|---------------|
| 旧砂预处理含水量 | ≤ 3%（超过此值需预烘干至约1%） |
| 常规机械再生停留时间 | 5~10 min |
| 常规脱膜率 | 5%~25% |
| Na₂O去除率 | 最高可达40%以上 |
| 再生砂回收率 | ≥ 90% |

#### 工艺流程

![机械法旧砂再生工艺流程图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/4ade765c-0528-4bc5-b30e-42297be66919/resource)

机械再生的典型工艺流程为：旧砂→过滤杂质→遴选→废砂库→机械再生→除尘→筛分→再生砂[[S9]]。

干法再生系统的优点是设备结构和系统布置较简单，投资较少，二次污染较易解决；缺点是对水玻璃旧砂的Na₂O去除率低，一般不超过30%，单独机械再生后砂粒表面残留Na₂O质量占比可达8.18%，制备砂芯室温强度仅为0.46 MPa，远低于新砂水平[[S32,S37,S38]]。

### 2. 热法再生

#### 工艺原理

热法再生通过焙烧炉将旧砂加热到800~900℃，去除旧砂中可燃残留物。针对有机粘结剂旧砂，该工艺可将粘结剂膜完全燃烧分解，使旧砂恢复到接近新砂的状态，属于完全再生方法[[S12,S37]]。

#### 典型设备与关键参数

热法再生设备一般采用回转窑、沸腾炉或燃气加热焙烧炉[[S25]]。典型参数如下：

| 参数 | 典型数值 |
|------|----------|
| 完全再生焙烧温度 | 800~900℃[[S12,S37]] |
| 旧砂回收率 | 96%~98%[[S12]] |
| 热预处理温度（用于脆化水玻璃粘结膜） | 300~350℃、1 h[[S15]] |

#### 再生效果

经过焙烧处理的再生砂相比原砂具有更优的热物理性能——热膨胀率可从原砂的1.4%左右降至0.5%，砂粒圆整度和流动性均优于普通原砂[[S14]]。使用再生砂制备覆膜砂时，树脂添加量可降低10%~15%，显著减少砂芯断裂和型面裂纹缺陷的发生[[S14]]。100~200目覆膜砂旧砂经热法再生1 h+机械法再生10 min处理后，制备砂型的抗弯强度可达6.08 MPa，性能与同规格新砂一致[[S14]]。

热法再生无需配套污水处置设施，仅需针对焙烧烟气配套除尘与VOCs处理即可满足排放要求[[S34]]。水玻璃旧砂热法再生线的回收率可达85%以上[[S36]]。

### 3. 湿法再生

#### 工艺原理

湿法再生利用水的溶解、擦洗作用及机械搅拌，去除旧砂粒上残留的粘结剂膜。由于旧砂表面残留的高模数水玻璃可溶于水，湿法对水玻璃旧砂效果尤为显著[[S37,S32]]。

#### 典型设备与关键参数

湿法再生典型工业设备包括滚筒式湿法再生机、搅拌筒式湿法再生机组等[[S10,S8]]。关键操作参数如下：

| 参数 | 推荐范围/典型值 |
|------|-----------------|
| 再生机转速 | 优选800 r/min[[S10]] |
| 砂水体积比 | 1:0.6~2:1（即加水量600~1000 mL/kg旧砂）[[S10,S2]] |
| 再生停留时间 | 20~30 min[[S2]] |
| Na₂O去除率 | 80%~98%[[S10,S32]] |
| 再生砂回收率 | ≥ 95%[[S32]] |

#### 工艺流程

![湿法旧砂再生工艺流程图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/dbd34f97-80d7-4b53-a61d-1347725911d6/resource)

湿法再生工艺流程为：旧砂→筛分破碎→水擦洗→烘干→除尘→筛分→再生砂[[S31]]。其中水擦洗环节同步分离出含悬浮物污水，需后续处理[[S31]]。

#### 工艺局限性

湿法再生效果显著，可将水玻璃旧砂残留Na₂O含量控制在0.2%左右，制芯室温抗拉强度达到2.19 MPa，接近新砂水平[[S10]]。但其缺陷同样明显：系统庞大、需烘干湿砂、耗水量大（吨旧砂耗水0.6~0.9 t）[[S4,S37]]。污水处理方面，达标排放的处理成本为0.51元/t，若需循环回用则处理成本增至1.62元/t[[S4]]。此外，北方冬季低温环境下操作受限[[S10]]。

### 4. 化学再生

#### 工艺原理

化学再生是向旧砂中加入指定化学试剂（或溶剂），通过化学反应去除旧砂中的残留粘结剂及有害成分，使旧砂恢复或接近新砂的物理化学性能[[S37,S32]]。

#### 典型工艺参数与效果

以水玻璃旧砂的热法+化学法联合再生为例，优化配比为：每1 kg旧砂添加氯化铝3.5 g、海泡石5 g、水滑石5 g，在200℃下加热1 h[[S18]]。处理后的再生砂关键指标为：

| 指标 | 数值 |
|------|------|
| 残留Na₂O含量 | 0.21%[[S18]] |
| 室温抗拉强度 | 1.71 MPa[[S18]] |

经制芯工艺优化后（粘结剂加入量2.0%、促硬剂0.3%、水0.15%），化学法+热法联合再生砂制备的砂芯即时热强度可达0.57 MPa、室温强度可达2.17 MPa，性能优于机械法+热法联合再生的效果（即时热强度0.45 MPa、室温强度1.85 MPa）[[S20]]。100%使用该再生砂制备的砂芯可浇注得到内腔光洁、无缺陷的铝合金铸件[[S18]]。

化学再生因药剂成本较高，工业化大规模应用场景较少[[S37]]。

## 再生砂质量控制指标

### 国家标准规定

中华人民共和国国家标准GB/T 26655—2011《铸造用再生硅砂》规定了再生硅砂的技术要求，标准于2012年3月1日起实施，适用范围排除水玻璃旧砂[[S33,S23]]。再生硅砂按以下五个维度分级[[S23]]：

| 指标 | 检测方法 |
|------|----------|
| 二氧化硅（SiO₂）含量 | GB/T 7143[[S23]] |
| 含泥量 | GB/T 2684[[S23]] |
| 酸耗值 | GB/T 2684[[S23]] |
| 颗粒角形因数 | GB/T 9442[[S23]] |
| 灼烧减量 | GB/T 2684[[S23]] |

牌号表示规则为：**ZZGS + SiO₂含量 - 粒度范围(平均细度) - 灼烧减量最大值 - 酸耗值最大值**[[S23]]。统一要求再生砂含水量不得超过0.3%，每批次供货重量不超过70 t，不同牌号产品需分类存放运输并配置防潮措施[[S23,S13]]。

### 行业通用控制要求

铸造行业通用的再生砂质量控制指标如下[[S29,S34]]：

| 控制项目 | 合格范围/要求 |
|----------|---------------|
| 140筛号以下微粉质量分数 | < 0.8%（底盘粉尘< 0.2%）|
| 含水量 | < 0.2% |
| 铸钢件灼烧减量 | < 1.5% |
| 铸铁件灼烧减量 | < 3.0% |
| 铸铝件灼烧减量 | < 4.0% |
| 酸耗值 | < 2 mL（使自硬呋喃树脂砂pH值稳定在3~6） |
| 含泥量（树脂砂/冷芯盒砂用） | < 0.1%~0.3% |
| 再生砂粒度与新砂偏差 | ≤ 1个筛号 |
| 有机粘结剂热法装置脱膜率 | ≥ 50% |
| 整体旧砂回用率 | > 90% |

再生砂的灼烧减量与发气量几乎呈直线关系，超标将直接引发铸件气孔缺陷；微粉含量每增加0.5%，树脂砂强度下降约20%[[S29]]。

### 现场快速检验方法

手捏法可快速判断型砂性能适配性，操作依次为：捏成砂团 → 手放开后观察清晰手纹 → 折断砂团检查无碎裂且强度达标[[S11]]。此方法可用于现场快速筛除不合格再生砂。

## 与造型/制芯工艺的关联

**全流程闭环再生链路**为：上游从落砂工序产出旧砂 → 依次经过初破碎、除杂、脱膜处理、冷却分级 → 得到合格再生砂 → 输送至下游混砂-制芯工部循环利用[[S22,S26]]。系统运行需维持热稳定状态，通过按比例补充少量新砂稳定再生砂核心指标，避免性能随循环次数大幅漂移[[S26]]。

以化学粘结砂系统为例，典型回收流程涵盖砂存储、混砂、浇注、铸件处理、旧砂的多工序回收净化及再生砂回用等环节[[S22]]。

## 在重力铸造与低压铸造中的应用

### 重力铸造应用

采用复合热硬水玻璃改性粘结剂配合热法+化学法联合再生砂工艺，制备的缸盖浇冒口砂芯面向重力铸造铸铝件场景[[S5,S20]]。生产验证表明：混砂工艺中粘结剂加入量占砂重2.0%，促硬剂0.3%，水0.15%，制得的砂芯表面质量好、强度较高，两次合模无断裂失效，浇注后溃散性良好，铸件表面无铸造缺陷[[S5,S20]]。

### 低压铸造应用

砂型低压铸造的生产工况下，可采用90%回用再生砂+5%配套红砂的型砂配比，水分控制在5%~6%区间，湿压强度控制在5.5~6.5，可满足低压充型的工艺要求[[S7]]。

### 通用使用限制

在重力铸造、低压铸造的砂芯/砂型场景下，再生砂的微粉超标将直接导致树脂耗量上升、灼减量超标引发铸件气孔缺陷；残留Na₂O超标将直接降低砂芯可使用时间，引发混砂阶段提前硬化失效[[S29,S28]]。

## 再生砂性能特性

合格再生砂的典型实测性能为：粒度AFS可达56.12（优于高硅砂的53.14），流动性14 s（与陶瓷砂持平、优于高硅砂的15 s），粒度三筛集中度可达80.65%[[S21,S6]]。

经化学法+热法联合再生后，旧砂表面能谱分析显示：O元素质量占比27.44%、Na元素2.30%、Si元素70.26%，相比未处理旧砂钠残留大幅降低，硅占比显著提升，砂粒表面光洁度高，残留粘结剂膜少，形貌接近新砂[[S41,S19]]。

热硬水玻璃旧砂直接回用制备砂芯时，室温抗拉强度和热抗拉强度仅为0.03 MPa，几乎无可用时间，说明未经再生的旧砂无法直接投入使用[[S28]]。合格再生砂制备的砂芯，其即时热强度和室温抗拉强度均需严格控制以满足铸造工艺要求[[S28]]。
