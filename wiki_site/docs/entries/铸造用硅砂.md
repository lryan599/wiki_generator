## 概述

铸造用硅砂（foundry silica sand）是以石英（SiO₂）为主要矿物成分，粒径区间为 0.020mm ~ 3.350mm 的耐火颗粒物，按开采和加工方法的不同，可分为水洗砂、擦洗砂、精选砂等天然硅砂和人工硅砂 [[S6,S27]]。在砂型铸造中，硅砂是造型、制芯工序最基础的核心原辅材料 [[S27]]。在压铸场景下，硅砂主要作为预制砂芯的骨料用于成型复杂内部腔体，亦可在半永久型铸造中作为金属型表面的面层砂或辅助砂芯使用 [[S26,S21]]。

> **注**：常规压力金属型压铸不存在直接使用硅砂作为铸型本体的公开应用。硅砂仅通过砂芯形式间接服务于压铸工艺，属于压铸技术维基中的“边缘辅助材料” [[S26]]。

## 化学与矿物组成

### 主要成分与杂质影响

铸造用硅砂的主要化学成分为 SiO₂。天然硅砂中 SiO₂ 含量区间多为 80%~97%，精制人工硅砂的 SiO₂ 含量可 ≥99% [[S7,S16]]。硅砂中的主要杂质矿物为长石、云母、铁的氧化物及碳酸盐等 [[S31]]。

杂质对硅砂性能的影响如下 [[S31,S23,S24]]：

- **Al₂O₃**：是硅砂中最主要的杂质成分，含量约 5% 时可将硅砂耐火度由纯石英的 1710℃ 降至 1550℃。
- **Fe₂O₃**：铁的氧化物易与其他组分形成低熔点共熔物，直接降低硅砂耐火度，同时引发铸件表面化学粘砂。
- **K₂O、Na₂O**：碱金属氧化物会显著降低硅砂高温熔点，恶化热化学稳定性，并增大硅砂的酸耗值。

### SiO₂ 含量分级

GB/T 9442 将铸造用硅砂按 SiO₂ 质量分数分为多个等级，各级代号对应 SiO₂ 最低含量要求。不同铸造合金对硅砂的 SiO₂ 含量要求差异显著 [[S7,S23,S35]]。

| 分级代号 | SiO₂ 最低含量（质量分数 %） | 适用合金举例 |
|:--------:|:--------------------------:|:------------|
| 98 | 98 | 大型铸钢件 |
| 96 | 96 | 铸钢件 |
| 93 | 93 | 铸钢件、大型铸铁件 |
| 90 | 90 | 铸铁件 |
| 85 | 85 | 铸铁件、非铁合金件 |
| 80 | 80 | 非铁合金件 |
| 75 | 75 | 非铁合金件（有色合金） |

> 来源：[[S7,S23,S35,S45]]

## 物理工艺性能

### 颗粒形貌与角形系数

角形因数（角形系数）定义为硅砂实际比表面积与理论比表面积的比值，用于精确表征颗粒形貌。颗粒形貌直接影响型砂的粘结强度、透气性及粘结剂用量：圆形高圆整度硅砂适合低树脂用量的覆膜砂、树脂自硬砂工艺 [[S40,S7,S10]]。

| 颗粒形状 | 角形因数范围 |
|:--------:|:------------:|
| 圆形 | < 1.15 |
| 椭圆形 | 1.15 ~ 1.30 |
| 钝角形 | 1.30 ~ 1.45 |
| 方角形 | 1.45 ~ 1.63 |
| 尖角形 | > 1.63 |

> 来源：[[S40]]

### 粒度分布与 AFS 细度

**AFS 细度**（AFS grain fineness number，平均细度）是国际通用的硅砂平均颗粒尺寸表征参数，由各筛上残留砂粒质量百分数乘以对应细度因数后求和取整得到 [[S29,S38,S44]]。常规铸造用硅砂 AFS 细度覆盖 20~70 区间 [[S44]]。

GB/T 9442 对粒度分布的要求为：相邻三筛残留质量和 ≥75%，相邻四筛残留质量和 ≥85%，五筛 ≥95% [[S7,S44]]。

粒度分布的集中度对型砂性能和铸件质量影响显著 [[S29,S11,S41]]：
- 细度过高：透气性下降，易导致铸件气孔。
- 细度过低：铸件表面粗糙度恶化。
- 分布过宽：型砂性能波动增大。

### 含泥量

含泥量定义为硅砂中粒径小于 0.020mm 的颗粒质量占砂样总质量的百分比。GB/T 9442 对不同加工品类的含泥量分级要求如下 [[S7,S10,S3]]：

| 加工品类 | 含泥量上限（质量分数 %） |
|:--------:|:------------------------:|
| 水洗砂 | ≤ 1.0 |
| 擦洗砂 | ≤ 0.3 |
| 精选砂 | ≤ 0.2 |

> 来源：[[S7,S36]]

含泥量过高会显著降低型芯砂的透气性和干强度，增大粘结剂消耗量，容易导致铸件气孔和粘砂缺陷 [[S7,S3,S10]]。

### 水分

GB/T 9442 要求袋装烘干硅砂水分质量分数 ≤ 0.3%。水分过高会在金属液浇注时瞬间汽化产生大量水蒸气，大幅提升铸件气孔缺陷的发生概率 [[S7,S36]]。

### 耐火度与烧结点

纯硅砂理论耐火度可达 1760℃，但受杂质含量影响，实际工业级硅砂耐火度范围一般为 1500~1700℃ [[S14,S43,S25]]。耐火度不足会导致高温金属液浇注时砂粒熔化粘结在铸件表面，造成严重粘砂缺陷 [[S14,S25]]。

- 铸钢件要求硅砂耐火度 ≥ 1550℃ [[S25]]
- 铝合金、镁合金压铸配套砂型可选用较低耐火度硅砂以降低成本 [[S25]]

烧结点指硅砂颗粒在高温下开始发生表面熔融粘结的最低温度，普通铸造用硅砂烧结点常规区间为 1000~1350℃ [[S36,S43]]。烧结点过低会导致浇注时砂粒表面过早烧结，降低型芯溃散性，增加铸件落砂难度 [[S36]]。

### 酸耗值

酸耗值定义为中和 50g±0.01g 硅砂中碱性物质所消耗的 0.1mol/L 盐酸溶液毫升数 [[S40]]。GB/T 9442 明确要求使用化学粘结剂的硅砂酸耗值 ≤ 5.0mL [[S7,S46]]。酸耗值过高会消耗酸性固化剂，延长树脂砂、水玻璃砂固化时间，降低型芯强度 [[S40,S46]]。

### 热膨胀性能

硅砂在 573℃ 发生 β-石英向 α-石英的快速相变，体积膨胀率约 0.82%，室温至 1000℃ 的总热膨胀率范围为 1.0%~1.8% [[S23,S31,S29]]。热膨胀量过大且不均匀会直接导致铸件产生夹砂结疤、鼠尾等膨胀类缺陷 [[S23,S29]]。

## 分类与牌号

### 按来源分类

铸造用硅砂按来源分为两大体系 [[S7,S16]]：

| 大类 | 细分品类 | 典型特征 |
|:---:|:--------|:--------|
| 天然硅砂 | 水洗砂 | 含泥量 ≤ 1.0% |
| 天然硅砂 | 擦洗砂 | 含泥量 ≤ 0.3% |
| 天然硅砂 | 精选砂（浮选砂） | SiO₂ ≥ 96%，含泥量 ≤ 0.2% |
| 人工硅砂 | 普通人工硅砂 | — |
| 人工硅砂 | 精制人工硅砂 | SiO₂ ≥ 99%，经酸洗和水洗 |

> 来源：[[S7,S16]]

### 按标准牌号表示（GB/T 9442）

铸造用硅砂的牌号以 ZGS（铸造用硅砂拼音首字母）开头，后依次为 SiO₂ 分级代号、主要粒度组成首筛筛号、主要粒度组成尾筛筛号，括号内标注平均细度值。2010 版标准还补充了平均细度偏差分级代号 [[S7,S17,S8]]。

示例：**ZGS 90-50/100（54A）** 表示该牌号硅砂的最小 SiO₂ 含量为 90%，主要粒度组成为三筛（首筛 50、尾筛 100），平均细度为 54，平均细度偏差值为 ±2 [[S7]]。

## 工艺角色与选材原则

### 在造型与制芯中的应用

在砂型铸造中，硅砂是冷芯盒、热芯盒、壳芯、树脂自硬砂、覆膜砂等各类造型制芯工艺的基础骨料 [[S22,S42]]。制芯用硅砂的通用要求为：含泥量 ≤ 0.3%，优先选用粒形圆整的砂料以减少粘结剂加入量，平均细度应匹配对应铸件的表面精度要求 [[S22,S42]]。

### 不同合金用砂选用原则

不同铸造合金的浇注温度和对铸件表面质量的要求差异决定了硅砂的选材标准 [[S22,S42,S23]]：

| 合金类型 | SiO₂ 最低要求 | 典型用途说明 |
|:--------:|:------------:|:------------|
| 铸钢件 | ≥ 95%（一般），大型件 ≥ 98% | 高耐火度需求 |
| 铸铁件 | ≥ 85%~90% | 可选用内蒙古通辽、江西鄱阳湖等地天然硅砂 |
| 有色合金件 | ≥ 75%~85% | 浇注温度低，可选较低品位硅砂 |

> 来源：[[S22,S42,S23]]

### 在压铸工艺中的关联

硅砂在压铸中的直接应用极为有限。仅在特殊压铸工艺（如硅青铜泵叶轮的真空压铸）中使用由硅砂制备的预制砂芯来成型复杂内部腔体 [[S26]]。在半永久型铸造工艺中，硅砂可作为金属型表面的面层或辅助砂芯使用 [[S26]]。

## 缺陷关联与控制措施

### 脉纹缺陷

**产生机制**：硅砂中的石英组分在 573℃ 发生 α→β 晶型相变，体积急剧膨胀，使砂芯表层热应力超过高温粘结强度后开裂。金属液渗入裂缝后凝固形成条状或网状金属凸起，即为脉纹 [[S19,S12,S33]]。

**常用控制措施** [[S19,S33,S47]]：
1. 在硅砂中添加 0.5%~0.7% 挥发性纤维素，高温燃烧灰化后为砂粒膨胀预留空间。
2. 添加锂辉石等抗脉纹添加剂或混合铬铁矿砂等低膨胀特种砂。
3. 优化砂芯粒度分布，使膨胀发生的时间分散。
4. 在砂芯表面涂刷高耐火度防脉纹涂料。

### 粘砂缺陷

**产生机制** [[S28,S1,S23]]：
- **机械粘砂**：硅砂耐火度不足或粒度级配偏粗，金属液渗入砂粒间隙。
- **化学粘砂**：SiO₂ 在高温下与 FeO 反应生成低熔点铁橄榄石（2FeO·SiO₂），使砂粒熔结于铸件表面。

**常用控制措施** [[S28,S1]]：
1. 提高硅砂 SiO₂ 含量，保证耐火度。
2. 调整粒度级配，降低砂粒间隙。
3. 在铸型表面刷涂高耐火度防护涂料。

### 气孔缺陷

**产生机制**：硅砂细粉含量超标、透气性能不足，或配套粘结剂发气量过高时，气体无法及时排出，侵入金属液形成气孔 [[S11,S18]]。

**常用控制措施** [[S11,S18]]：
1. 严格控制硅砂细粉含量不超过标准限值。
2. 在砂芯中预留充分的排气通道。

## 局限性及替代材料

### 普通硅砂的核心劣势

硅砂尽管价格低廉且来源广泛，但存在以下不可忽视的局限 [[S28,S4,S34]]：
1. **热膨胀大**：573℃ 晶型相变引发体积突变膨胀，是铸件产生夹砂、脉纹等膨胀缺陷的根源。
2. **热化学稳定性差**：高温下易与 FeO 反应生成低熔点化合物，引发化学粘砂。
3. **职业健康风险**：生产过程中产生的粉尘（游离 SiO₂）易使作业人员罹患矽肺病。

### 主流替代特种砂比较

| 特种砂种类 | 关键性能优点 | 主要局限 | 典型适用场景 |
|:--------:|:-----------|:--------|:-----------|
| 锆砂 | 热膨胀系数仅为硅砂的 1/6~1/3，熔点约 2430℃ | 价格昂贵，储量有限 | 大型铸钢件厚壁面砂、熔模铸造 |
| 铬铁矿砂 | 耐火度 >1900℃，热导率高，高温固相烧结可阻挡金属渗透 | 价格高于硅砂 | 合金钢铸件面砂 |
| 镁橄榄石砂 | 不含游离 SiO₂，无 573℃ 突变膨胀特性 | 粒形较差，价格高于硅砂 | 高锰钢铸件 |

> 来源：[[S34,S28]]

## 标准与试验方法

| 标准编号 | 覆盖的检测项目 |
|:--------|:--------------|
| GB/T 9442 | SiO₂ 含量、含泥量、角形因数、粒度分布（平均细度）、含水量、酸耗值、细粉含量 |
| GB/T 2684 | 铸造用原砂及混合料试验方法（含泥量、含水量、粒度分析） |
| GB/T 7143 | 铸造用硅砂化学分析方法 |
| ASTM E11、AFS 5110 | 砂粒筛析方法、AFS 细度计算规则 |

> 来源：[[S36,S20,S7,S13,S29]]

AFS 平均细度的计算式为：平均细度 = Σ(Pₙ × Xₙ) / ΣPₙ，其中 Pₙ 为任一筛号上停留砂粒质量占总质量的百分数，Xₙ 为对应细度因数，GB/T 9442 附录中提供了全系列筛号对应的细度因数参考表 [[S29,S38,S44]]。
