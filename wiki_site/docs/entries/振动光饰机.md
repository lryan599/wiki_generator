## 概述

振动光饰机是中国国家标准GB/T 25370《铸造机械 术语》正式定义的铸造机械，对应英文名为vibrating finisher，别名振动研磨机、振动抛光机、振动去毛刺机[[S2,S15]]。该设备属于通用光饰类设备，在滚筒滚光基础上发展而来，是压铸生产流程中承担批量去毛刺与表面精整的核心后处理工艺设备[[S15]]。

在压铸件后处理环节，振动光饰机主要用于大批量去除冲切大飞边后残留的分型线毛刺、对尖角进行倒角（可形成均匀 R0.1～0.2mm 的圆角）、整平铸件表面缺陷，处理效率远高于手工打磨[[S16]]。部分类型的振动光饰机由于容器敞开，可在加工过程中随时抽检零件表面质量[[S15]]。

## 结构与组成

振动光饰机由以下关键组件构成[[S15,S24,S12]]：

- **振动电机/激振器**：自带可调偏心重块，通过调整偏心块角度改变振幅，为设备提供工作所需的激振力。
- **支撑弹簧**：作为弹性连接部件，既传递振动保证容器正常运动，又隔绝振动向底座与地面传递，以降低设备运行噪音与基础震动。
- **槽体/容器**：加工物料承载部件，内壁普遍衬有聚氨酯耐磨层，可避免金属直接磨损并降低零件磕碰损伤[[S12]]。
- **控制面板**：集成振动频率、运行时长、出料模式等参数的调节与显示模块。
- **介质回收分离系统**：通过筛网实现加工完成后压铸件与研磨介质的自动筛分分离，研磨介质可循环回用。

## 分类

振动光饰机可分为以下主要类型[[S15,S4]]：

| 设备类型 | 结构特征 | 物料运动方式 | 加工特点 | 适用场景 |
|---------|---------|------------|---------|---------|
| 圆形振动槽式（碗形振动机） | 碗形或轮胎形容器，安装在弹簧上，由中心垂直轴的偏置重块驱动；多数带有挡料圈与过滤筛网，可实现自动筛分 | 沿固定方向做螺旋运动 | 磨削作用较柔和，表面粗糙度低于筒形机，但处理效率相对较低 | 中小精密压铸件批量加工 |
| 直线式（筒形振动机） | U形、球状锁眼形、直下锁眼形三类容器；U形原始款存在物料回流及研磨不均匀缺陷，锁眼形结构可消除回流 | 沿筒体轴线方向循环运动，锁眼形无回流 | 物料运动均匀，可加工大型压铸件 | 可加工长12m、宽2m的大型压铸件，适配大产能生产[[S15]] |
| 倾斜式振动槽 | 通过调整槽体安装倾角控制物料流动速度与研磨时长 | 受倾角控制的定向流动 | 工艺适配性更强 | 需要精确控制加工时长的应用场景 |

## 工作原理

振动光饰机的工作原理属于阻尼强迫振动范畴[[S16]]。设备通过高速旋转的可调偏心重块产生离心激振力，驱动安装在弹簧上的容器产生三维方向振动，带动容器内的压铸件、研磨介质、水及化学助剂持续做相对滑动摩擦，如同锉刀切削一般将毛刺与尖角逐步磨削掉[[S15,S17,S22]]。

振源可采用两种机构：一是由可调的一对振动子高速旋转产生的离心力；二是以50～60Hz频率工作的电磁系统[[S15,S4]]。

## 工艺参数

### 通用参数范围

| 参数 | 通用范围 | 压铸工况常用值 |
|-----|---------|-------------|
| 振动频率 | 15～50 Hz | 20～30 Hz |
| 振幅 | 2～20 mm | 3～6 mm |

上述参数适用于压铸件为主的振动光饰工艺[[S15,S6]]。

### 锌合金压铸件专用参数

振动光饰用于锌合金压铸件时，美国材料与试验协会标准给出了系统化的参数规范[[S8]]：

| 参数 | 推荐值/范围 |
|-----|-----------|
| 振动频率 | 700～2100 cpm（约11.7～35 Hz） |
| 振幅 | 0.8～6.4 mm |
| 0.5 m³振动槽最大磨料装载量 | 约900 kg |
| 0.5 m³振动槽最大锌合金件装载量 | 约180 kg |
| 精加工频率 | 1700～2100 cpm |
| 精加工振幅 | 3.2～6.4 mm |
| 精加工时长 | 2～4 h |
| 精加工可达表面粗糙度 | Ra 0.15～0.25 μm |
| 材料去除速率 | 约 5 μm/h |

锌合金压铸件专用实验研究得出的参数为[[S14,S17]]：

| 参数 | 推荐值 |
|-----|-------|
| 零件与磨料重量比 | 1～1.5 |
| 振幅 | 4～6 mm |
| 单批次加工时长 | 30～40 min |
| 尖边倒圆半径 | R0.1～0.3 mm |
| 平面磨损量 | 0.01～0.02 mm |

### 装载比例

磨料与压铸件的合理装载比例直接影响加工质量与效率[[S11,S8]]：
- 体积比一般推荐范围为 2:1～6:1。
- 锌合金压铸件质量比推荐为 5:1～6:1；当质量比低于 5:1 时，极易出现铸件表面刮伤或凹陷缺陷[[S8]]。

### 液体参数

当采用湿式加工时，加水量控制为卧式槽体内半圆弧直径的 1/3～1/4；锌合金件应采用重铬酸钾溶液作为工作液，铝、铜压铸件可直接采用清水[[S14]]。湿式加工的磨削效率较干式提升一倍，同时液体膜的存在使得在平面处的磨损量极小（实测约 0.01～0.02 mm），而尖角毛刺处则被有效去除（倒圆 R0.1～0.3 mm）[[S14]]。

### 振幅与加工效率的关系

在固定振动频率20Hz条件下，振幅与磨料颗粒运移速度呈近线性正相关：当振幅从2.5mm增加至4mm时，颗粒运移速度从0.1 m/s上升至0.18 m/s[[S3]]。通用试验也表明，在振动加工场景下，加工速度随工具振幅增大而同步提升，且相同振幅下振动频率越高、磨料粒度越粗，加工效率越高[[S10]]。

### 可加工毛刺尺寸

振动光饰可直接有效去除的压铸件毛刺最大厚度为 1～1.5 mm；大于此尺寸的大飞边须先通过冲切模具预先去除后，再转入振磨工序[[S17,S16]]。

## 工艺定位

在压铸全流程中，振动光饰机的上游紧邻切边压力机工序，待加工件为已完成大浇口和厚飞边冲切的压铸件毛坯；下游为零件清洗工序，此后可衔接电镀、喷涂等表面精加工[[S5,S16]]。

## 适用合金与磨料适配性

| 压铸合金 | 磨料适配要求 |
|---------|------------|
| 锌合金 | 可直接适配常规陶瓷、石英、塑料基磨料。石英石磨料兼具高效磨削和较好表面光洁度[[S17]]。 |
| 铝合金 | 可直接适配常规陶瓷、石英、塑料基磨料，湿式加工用清水即可[[S14]]。 |
| 镁合金 | **严禁使用铁系磨料**，否则铁颗粒嵌入表面会引发局部腐蚀。推荐选用粘结陶瓷磨料、铝质磨料或玻璃珠[[S1,S18]]。 |

## 局限性

振动光饰机的主要不足之处包括[[S15,S6,S16]]：
- 不适用于精密零件与脆性压铸件的加工；
- 不能获得极低粗糙度的镜面级表面；
- 设备长期运行噪音较大，维护成本偏高；
- 对盲孔、深窄内腔区域的毛刺去除效果差，因磨料不易进入该区域。

## 与其他精整工艺的对比

| 对比工艺 | 与振动光饰的关系与差异 |
|---------|---------------------|
| 滚筒光饰 | 振动光饰效率约为普通滚筒光饰的 6～10 倍[[S9]]；加工周期较长，但噪音更低[[S11]]。 |
| 离心光饰 | 离心光饰加工时长仅为振动光饰的 1/50[[S19,S20]]；且零件间碰撞更小，尺寸精度保持性更好[[S19]]。 |
| 喷砂/喷丸 | 喷砂/喷丸仅适用于外露表面处理，薄壁件易变形，加工一致性低于振动光饰[[S23]]。 |
| 人工打磨 | 人工打磨劳动强度大，加工一致性差，尺寸精度远低于振动光饰，综合加工成本约为传统人工工艺的 1/3[[S9]]。 |
