## 概述

裂纹是压铸及金属成形领域最常见、危害最大的线性不连续缺陷之一，属于铸件连续性中断类缺陷 [[S79]]。按形成温度与机理不同，裂纹主要分为**热裂纹（热裂）**与**冷裂纹**两大类，两者在形貌特征、显微组织、断口色泽、形成温度区间及控制策略上存在本质差异 [[S7,S39,S48]]。在压铸领域，应力腐蚀裂纹、疲劳裂纹等不纳入典型的铸造原生裂纹缺陷范畴，它们多属于构件服役阶段产生的失效类缺陷 [[S40,S3]]。

## 定义与标准化

GB/T 5611—2017《铸造术语》将裂纹类缺陷分为若干子类，其中与压铸最直接相关的是热裂和冷裂 [[S7]]：

- **热裂（Hot tearing）**：铸件在凝固后期或凝固后在较高温度下形成的裂纹。其断面严重氧化，无金属光泽，裂口沿晶粒边界产生和发展，外形曲折而不规则 [[S7]]。
- **冷裂（Cold cracking）**：铸件凝固后在较低温度下由铸造应力导致的裂纹。裂口常穿过晶粒延伸到整个断面 [[S7]]。

在压铸领域的专用术语标准 GB/T 29092—2012《镁及镁合金压铸缺陷术语》中，对两种裂纹给出了更具压铸针对性的定义 [[S43,S39]]：

- **热裂纹**：压铸件上不规则的曲折隙缝，纹路狭小而长。多发生在压铸件尖角处的内侧、厚薄断面交接处及浇冒口与压铸件连接的热节区。裂纹处的断面被严重氧化呈深灰色或黑色，无金属光泽。其显微组织特征为沿晶开裂 [[S39]]。
- **冷裂纹**：外观呈平直折线状的裂纹。裂纹处的断面洁净，具有金属光泽，裂纹处的金属表面没有氧化或轻微氧化。其显微组织特征为穿晶开裂 [[S39]]。

国际铸造技术协会 ICFTA 将铸件裂纹归为 C 类（裂纹、冷隔类）缺陷 [[S79]]。GB/T 41972—2022《铸铁件铸造缺陷分类及命名》（修改采用 ISO/TR 16078:2013）将裂纹类缺陷作为单独的缺陷大类进行分级编码 [[S79]]。

## 分类体系与核心特征

### 热裂纹（热裂）

热裂纹形成于合金线收缩开始温度至非平衡固相线温度的有效结晶区间内 [[S2,S48]]。其核心特征可概括如下 [[S80,S39,S33,S29,S76]]：

- **开裂路径**：沿晶撕裂，裂纹沿枝晶晶界萌生并扩展。
- **形貌**：宏观上裂纹曲折而不规则，呈撕裂状，常有分叉；外热裂从铸件表面应力集中处向内延伸，表面宽内部窄；内热裂分布在铸件最后凝固的区域，常在缩孔附近，分叉多，极少延伸至铸件表面 [[S2,S67]]。
- **断口**：表面被强烈氧化，呈黄褐色或深灰黑色，无金属光泽 [[S80,S33]]。
- **晶界填充物**：裂纹处常存在低熔点共晶相填充物，其尺寸小而分布致密 [[S29,S76]]。

### 冷裂纹

冷裂纹形成于铸件完全凝固后的低温弹性状态，当铸造残余应力超过该温度下合金的强度极限时萌生 [[S2,S48,S71]]。工业生产中大量冷裂纹是由已存在的微热裂纹在冷却过程中进一步扩展生成的 [[S2,S48]]。

核心特征 [[S39,S29,S30,S5,S24,S11,S60,S46]]：

- **开裂路径**：穿晶开裂，裂纹横穿基体和枝晶网络。
- **形貌**：宏观形态平直、细且清晰，呈直线或折线状，无明显分叉交错。
- **断口**：洁净，具有明显金属光泽，仅可能存在极轻微氧化色；断口比较整齐，颜色新鲜呈亮灰色或浅灰色 [[S48]]。
- **应力集中源**：常起源于铸件内部的夹杂物、缩孔、疏松等应力集中位置 [[S5]]。

### 冷裂纹与热裂纹特征对比

| 对比维度 | 热裂纹（热裂） | 冷裂纹 |
|---|---|---|
| 形成温度区间 | 线收缩开始温度至非平衡固相线<br>（凝固末期高温阶段）[[S2,S48]] | 完全凝固后的低温弹性状态 [[S2,S48]] |
| 开裂路径 | 沿晶（沿枝晶晶界扩展）[[S39,S80]] | 穿晶（横穿基体和枝晶）[[S39,S29,S30]] |
| 宏观形貌 | 曲折不规则，常有分叉，呈撕裂状 [[S7,S39,S2]] | 平直、细且清晰，呈直线或折线状 [[S39,S30,S46]] |
| 断口色泽 | 严重氧化，黄褐色或深灰黑色，无金属光泽 [[S7,S39,S80,S33]] | 洁净，具有金属光泽或极轻微氧化色 [[S39,S30,S5]] |
| 晶界特征 | 晶界处常见低熔点共晶填充物 [[S29,S76]] | 无明显的晶界共晶填充物 [[S60]] |
| X射线影像 | 各向曲折的分枝形态，两端尖中间粗，呈分叉交错的黑色条纹；电视屏幕上对应亮线状 [[S82,S76]] | 边缘清晰锐利的平直细线状影像，无分叉交错 [[S60,S46]] |

## 形成机制与诱因

### 热裂纹形成机理

热裂纹的形成机理主要包括以下四种理论 [[S44,S48,S77,S63]]：

1. **液膜理论**：凝固末期，晶间/枝晶间残留有薄的低熔点液态液膜。当铸件凝固收缩受阻时，拉应力集中在低强度的液膜上，将液膜拉伸撕裂形成晶间热裂纹。液膜的表面张力越小、液膜越薄，热裂的临界应力阈值越低 [[S44,S77,S63,S54,S20]]。
2. **强度理论**：在凝固末期固相线附近的“热脆区”（脆性温度区间）内，合金的强度和塑性极低。当凝固收缩受阻产生的铸造应力超过该温度下合金的强度极限，或变形量超过其塑性极限时，即引发热裂纹。有效结晶温度区间越宽，合金处于低塑性状态的时间越长，热裂倾向越高 [[S44,S48,S77,S81,S21]]。
3. **晶间搭桥理论**：凝固后期枝晶间存在晶间搭桥，可提升晶间结合力以抵抗收缩应力。一旦搭桥因收缩应力作用发生断裂，热裂纹即萌生 [[S77,S17]]。
4. **凝固收缩补偿理论**：凝固过程分为准液相阶段、可补缩阶段、不可补缩阶段和晶间搭桥阶段。在不可补缩阶段，若液相无法及时填充收缩间隙，微裂纹将在凝固收缩应力作用下扩展为宏观热裂纹 [[S77,S17,S72,S75]]。

### 冷裂纹形成机理

冷裂纹在铸件完全凝固后的低温弹性状态下产生，当铸件内外温差过大，残余铸造应力超过该温度下合金的强度极限时萌生 [[S2,S48]]。裂纹常从铸件内部的夹杂物、缩孔等应力集中位置起源并扩展 [[S5,S71,S62]]。实际生产中，大量冷裂纹是由已存在的微热裂纹在冷却过程中进一步扩展生成 [[S2,S48]]。

### 裂纹产生的主要诱因

| 诱因维度 | 具体因素 | 说明 |
|---|---|---|
| 合金成分 | 铁含量过高或硅含量过低/过高；Pb、Sn 等低熔点杂质含量超标；非金属夹杂物超标 | 降低合金塑性，扩大有效结晶温度区间，使晶间分布大量低熔点相；增加热裂与冷裂倾向 [[S51,S18,S19,S67,S5,S42]] |
| 模具温度场 | 模腔整体预热温度过低；局部温差过大、温度梯度过高 | 不同区域冷却速率差异过大，不均匀收缩产生的拉应力集中，在薄壁及厚薄过渡区引发裂纹 [[S51,S18,S26,S40]] |
| 工艺参数 | 浇铸温度过高/过低；铸轧速度/压射速度异常；压射延时不合理；热处理冷却速度过快 | 浇铸温度过高延长凝固时间、增大总收缩量；热处理冷速过快形成过高残余热应力 [[S19,S51,S77,S28,S36]] |
| 零件结构设计 | 壁厚突变、内尖角半径过小、厚薄断面过渡不合理 | 在局部位置形成严重应力集中，使凝固收缩受阻，是热裂和冷裂高发的典型结构因素 [[S5,S51,S18,S61,S67]] |

## 检测与表征

### 无损检测方法

**X射线探伤**是检测铸件内部裂纹的重要手段，可确定缺陷的平面投影位置、大小及种类 [[S10,S73]]。

- **热裂纹影像特征**：在 X 射线底片上呈现各向曲折的分枝形态，两端尖中间粗、多头发散的黑色条纹，部分呈龟裂状；在射线实时成像的电视屏幕上对应亮线状 [[S82,S76]]。
- **冷裂纹影像特征**：呈现边缘清晰锐利的平直细线状影像，无明显分叉交错的曲折形态，断口未氧化、穿晶扩展的特征在射线影像中表现为边界锐利的规整线条 [[S60,S46]]。

需要指出，常规工业 X 射线照相法对完全闭合、未与射线入射方向平行的裂纹检出率较低，通常仅当裂纹朝向与射线投射方向基本一致时才能被有效识别 [[S45,S53]]。

**渗透检测**适用于检测与铸件表面开口连通的裂纹，可检测出宽度小于 0.01 mm 的表面开口裂纹 [[S10,S37]]。着色渗透检测下裂纹会析出鲜艳的红色线状痕迹，荧光渗透检测下裂纹在紫外光下发出高亮度的黄绿色荧光痕迹，相关检测要求符合 EN 1371-1、EN 1371-2 标准 [[S47]]。

**磁粉检测**仅适用于铸钢、铸铁等铁磁性压铸件。交流电磁化可检出表面下 2 mm 以内的近表面裂纹，直流电磁化可检出表面下 4 mm 以内的近表面裂纹，对裂纹类缺陷灵敏度极高 [[S12,S73,S56]]。

**超声检测**可检测形状简单、表面平整的压铸件内部裂纹。裂纹与基体界面的声阻抗突变会产生特征性的尖锐反射回波，可准确定位裂纹的深度与尺寸，但难以直接判定裂纹的具体类型，且铸件晶粒粗大、表面粗糙时容易产生杂波干扰 [[S10,S73,S49]]。

### X射线CT三维表征

现代 X 射线 CT 技术可实现铸件内部裂纹的三维立体分布定量表征，对于铝合金压铸样品可完整呈现内部裂纹的体积占比与三维空间分布，为裂纹缺陷的精确定性与定量分析提供了高分辨率手段 [[S69]]。

### 金相显微观察

- **热裂纹**：金相显微镜下呈现典型的沿晶分布特征，裂纹沿枝晶间或晶界扩展，形态曲折分叉。裂纹内部常填充低熔点共晶物或氧化夹杂物，断口呈现黄褐色氧化色 [[S76,S58,S6]]。
- **冷裂纹**：呈现典型的穿晶分布特征，裂纹形态平直，断口无明显氧化痕迹，色泽新鲜，无明显的晶界共晶填充物 [[S60,S15]]。

## 标准验收准则

ASTM E505 标准作为铝镁合金压铸件射线探伤的参考标准，明确规定：在所有 A~D 级验收等级中，裂纹类缺陷均为不可接受项，无允许的裂纹尺寸或数量限值 [[S59,S4]]。这充分说明裂纹在压铸件质量控制中的严重性。

## 预防与处理措施

### 热裂纹预防

| 预防方向 | 具体措施 |
|---|---|
| 零件结构优化 | 内圆角半径推荐不小于 3 mm；相邻两处壁厚交接的过渡圆角半径按公式 R = (A+B)/6 计算（A、B 为相邻两处壁厚，单位 mm）；避免尖锐转角与壁厚突变 [[S70,S78,S50]] |
| 合金成分控制 | 严格控制低熔点共晶相占比与有害杂质含量，采用细化剂细化晶粒提升凝固过程塑性，减少热裂敏感性 [[S78,S57,S22]] |
| 模具温度管理 | 铝合金压铸模具生产过程中模温需维持在 300℃ 以上，减小铸件凝固阶段的激冷效应与收缩应力；避免局部温度梯度过大 [[S27,S57]] |
| 工艺参数优化 | 在满足充型要求的前提下降低浇注温度（铝合金铸件推荐 710±5℃），优化内浇道布局实现弱顺序凝固；保证铸件完全凝固后控制合理留模时间，提前抽芯开模 [[S57,S66]] |

### 冷裂纹预防

| 预防方向 | 具体措施 |
|---|---|
| 零件结构优化 | 避免铸件局部截面刚度差异过大，保证铸件各区域冷却收缩过程中受力均匀，减少脱模后残余应力集中点 [[S78]] |
| 合金成分控制 | 严格控制铝合金中铁元素含量上限、铜/锌等元素占比，避免合金出现冷脆性倾向 [[S78]] |
| 工艺参数优化 | 合理匹配开模时机，推荐按铸件壁厚 3 s/mm 设定基础留模时间，保证铸件具备足够冷态强度后均匀推出，避免局部受力过载开裂 [[S66]] |
| 淬火水温控制 | 对于形状复杂、尺寸较大的铝合金铸件，推荐淬火水温控制在 60~100℃ 区间，80±10℃ 的参数处于国内外公开工艺标准的推荐范围内，可有效降低淬火过程热应力，避免淬火冷裂纹与变形 [[S55,S38,S31,S35,S85]] |

### 表面微裂纹处理

压铸行业针对深度不超过 0.1 mm 的模具热疲劳衍生铸件表面微裂纹缺陷，常规处理方案包含补焊对应模具型腔位置，同时匹配合理的模温区间与压射压力参数，消除表面裂纹复现倾向 [[S83]]。

## 相关材料与工艺关联

### 铝合金压铸件

典型裂纹形态多为直线或波浪形、贯穿或不连续的局部开裂。高发区域为尖角过渡处、型芯周围、浇口附近，由压铸过程收缩应力、脱模机械过载共同诱发。防控要点为优化结构圆角、控制推出受力均匀性、匹配合理开模时间 [[S78,S25]]。

### 铝合金铸轧板带

典型裂纹形态为月牙形（马蹄裂）、边部开裂、通条沿轧制方向分布的裂纹带。高发区域为板边及结晶前沿位置，由凝固速率不均、铸轧区内部剪切应力、液穴过深共同诱发。防控要点为合理分布供料嘴垫片、匹配适宜的铸轧速度与温度，控制液穴深度，提升熔体过滤除渣精度 [[S68,S34,S22,S13]]。
