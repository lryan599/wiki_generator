树脂砂是指以人工合成树脂为粘结剂，通过固化剂引发树脂交联反应而成型的铸造用型砂或芯砂，属于有机化学硬化砂 [[S33,S29,S27,S23]]。作为铸造生产三大砂型体系之一，树脂砂在造型、制芯过程中无需烘干即可获得高强度，制备的铸件尺寸精度高、表面光洁，退让性和出砂性优异，旧砂可实现高比例再生回用，广泛应用于汽车、重工装备等行业的单件小批量至大批量铸件生产 [[S27,S18]]。

## 分类

### 按粘结剂体系分类
树脂砂按粘结剂化学体系可划分为四类 [[S6,S1,S20,S28]]：
- **呋喃树脂砂**：以含呋喃环的糠醛改性树脂为粘结剂，多采用酸固化（如磺酸），是应用最广、成本较低的品种。
- **酚醛树脂砂**：包括酸固化酚醛树脂砂和酯固化碱性酚醛树脂砂。碱性酚醛树脂砂 pH 为 12～14，以有机酯为固化剂，特适用于对氮、硫敏感的铸钢及合金钢 [[S24,S28]]。
- **脲烷（酚醛尿烷）树脂砂**：由双组分苯基醚酚醛树脂和聚异氰酸酯加胺类催化剂混制，可使用时间长、硬化速度快，但成本较高、含氮量较高，用于铸钢件易产生气孔 [[S6,S1]]。
- **酯硬化碱性酚醛树脂砂**：以碱性酚醛树脂和液态有机酯为固化剂，无刺激性气味，高温强度好，落砂性能优良 [[S5,S28]]。

### 按硬化方式分类
依据固化触发条件，树脂砂可分为三类 [[S6,S24,S27]]：
- **自硬砂（no-bake sand）**：室温下树脂与固化剂发生化学反应自行硬化，主要包含酸固化呋喃/酚醛自硬砂、酯固化碱性酚醛自硬砂和酚醛尿烷自硬砂。
- **热硬树脂砂（hot hardening resin sand）**：通过外部加热触发交联反应硬化，代表类型为热芯盒砂和覆膜砂（壳型砂）。
- **气硬砂（gas hardening sand / cold box sand）**：向砂型（芯）通入气态固化剂（如三乙胺、甲酸甲酯）实现硬化，固化耗时短至数秒，尺寸精度高。

### 覆膜砂与普通混拌树脂砂
覆膜砂（precoated sand / resin coated sand）属于热硬树脂砂子类，砂粒在造型前已预先包覆一层固态酚醛树脂涂层，加热时涂层熔融交联固化；而普通现场混拌树脂砂是将原砂、液态树脂、液态固化剂在造型前临时混合，通过室温化学反应完成硬化。二者在制备工艺、树脂存在形态及固化触发条件上截然不同 [[S6,S1,S7]]。

## 组成与制备

### 原砂
树脂砂所用原砂一般为硅砂，对品质要求严格。原砂含泥量升高会显著降低树脂砂的粘结强度 [[S10]]。特殊要求下也选用铬铁矿砂、锆砂、橄榄石砂等，以适应不同合金及热物性需求。

### 树脂与固化剂体系
- 呋喃自硬砂常用糠醇呋喃树脂，配套酸固化剂（如磺酸、磷酸），树脂加入量约为砂重的 1% 左右 [[S13,S28]]。
- 碱性酚醛树脂自硬砂树脂添加量约 1.3%，含水率约 50%，以液态有机酯为固化剂 [[S13]]。
- 酚醛尿烷自硬砂采用双组分树脂（苯基醚酚醛树脂 + 聚异氰酸酯）和胺类催化剂 [[S1,S28]]。

工业上不同树脂砂的造型方法和配套牌号实例见下表 [[S11]]：

| 树脂砂类型 | 造型方法 | 树脂牌号 | 固化剂/催化剂 |
| :--- | :--- | :--- | :--- |
| 呋喃树脂砂 | 喷墨打印造型 | 呋喃树脂 | 磷酸 |
| PEP-SET 树脂砂 | 手工造型 | I - 5160 / II - 5240（双组分） | 胺类催化剂 |
| 三乙胺冷芯盒树脂砂 | 手工造型 | I - 395 / II - 695（双组分） | 三乙胺气体 |

### 混砂工艺
树脂砂的混制通常采用碗形树脂砂混砂机等专用设备，将原砂、树脂粘结剂、固化剂按一定顺序和时间混合均匀 [[S17]]。为获得稳定性能，需严格控制混砂时间、温度和加料顺序，避免型砂结球 [[S1]]。

## 核心物性与工艺参数

树脂砂的物性参数因粘结剂类型、原砂种类、紧实度及温度不同而异，不存在单一固定值。铸造凝固模拟软件（如 ProCAST）公共材质库中预定义的通用石英基树脂砂默认常温参数为：密度 1520 kg/m³，比热容 1.22 kJ/(kg·K)，热导率 0.53 W/(m·K)，透气性 1×10⁻⁷ m²。该组参数适用于无特殊树脂类型说明的普通石英树脂砂常温铸型模拟场景 [[S27]]。

典型实际测量的物性数据如下：
- **呋喃树脂砂**：经称量与 ProCAST 密度对比，实测密度约为 1.59 g/cm³（1590 kg/m³）[[S2]]。
- **碱性酚醛树脂自硬砂**（70～140 目石英砂 + HA 碱性酚醛树脂）：实测密度 1.49 g/cm³，比热容 1013 J/(kg·K)，热导率 0.795 W/(m·K)，常温透气性良好，可直接应用于铝合金低压砂型铸造 [[S18]]。

3D 打印密实树脂砂的导热系数约为 0.50 W/(m·K)；若在镂空砂型中填充铬铁矿散砂，导热系数最高可升至 0.66 W/(m·K)，而清散砂后的镂空砂型导热系数可降至 0.25 W/(m·K)，镂空率与导热系数呈线性相关 [[S32,S14]]。

树脂砂的热物性随温度变化的趋势已有实验数据 [[S30,S31]]：
- **热导率**：0 ～ 600 ℃ 区间内先下降，约 600 ℃ 达到谷值，600 ～ 1000 ℃ 回升，700 ℃ 时传热能力处于较高水平。
- **比热容**：50 ～ 200 ℃ 区间上升，250 ℃ 略降后回升，400 ℃ 后趋于稳定。

不同自硬树脂砂的高温抗变形性能可通过热变形曲线对比，呋喃、酸固化酚醛、酯固化碱性酚醛、酚醛脲烷四类树脂砂的热变形随时间变化趋势各有特点 [[S9]]。

### 常用树脂砂性能对比

下表列出三类主流自硬树脂砂的优缺点对比 [[S5]]：

| 树脂砂类型 | 优点 | 缺点 |
| :--- | :--- | :--- |
| 呋喃树脂砂 | 粘结剂消耗量低，结合力强，热稳定性好，铸件表面光洁度高 | 气味强烈，退让性差，发气量较大 |
| PEP SET 自硬砂（酚醛尿烷系） | 硬化速度快，落砂性能好，旧砂再生较容易 | 树脂加入量高，含 N 量高，易产生气孔类缺陷 |
| 碱性酚醛树脂自硬砂 | 无刺激性气味，高温强度高，落砂性能好 | 型砂强度较低，树脂加入量较高 |

## 工艺优势与传统型砂对比

与粘土砂、水玻璃砂相比，树脂砂在多个维度具有显著优势 [[S29,S8,S15,S27,S22,S13]]：
- **尺寸精度**：可达 CT8～CT11，远优于普通粘土湿型砂。
- **表面质量**：铸铁件表面粗糙度达 Ra 25～50 μm，铸钢件 Ra 25～100 μm，明显优于粘土砂。
- **溃散性与清砂**：树脂膜高温分解后残留强度低，铸件落砂清砂远易于水玻璃砂。
- **旧砂再生**：树脂砂旧砂再生利用率高，呋喃砂可达 75%～80%，碱性酚醛砂约为 70%～75%，而水玻璃砂常规再生率仅约 70%～75% 且砂粒表面残留严重。
- **环保与适应性**：无 N、S、P 等有害元素释放的碱性酚醛树脂砂对铸钢件气孔和裂纹防控有利；水玻璃砂虽无有机挥发物，但溃散性和再生性差，限制了其在高品质铸件上的应用。

## 应用场景

树脂砂适用于制造复杂、高精度铸型和砂芯，典型应用涵盖汽车发动机缸体、缸盖、变速器壳等铸件，亦可用于镁合金、铝合金等薄壁件的低压或重力铸造 [[S3,S18,S11]]。在光固化 3D 打印快速铸造技术中，树脂砂可直接制备大尺寸复杂砂型，体现出良好的成型能力和尺寸精度 [[S3]]。

## 缺陷与防治

树脂砂使用不当可能导致以下典型铸造缺陷 [[S1,S5,S13]]：
- **皮下气孔**：与树脂含氮量或固化剂分解产物有关，碱性酚醛砂因不含 N、S 而较少出现，呋喃砂和酚醛尿烷砂则需严格控制氮含量并加强排气。
- **硫污染**：使用磺酸固化剂时，回用砂中硫累积可破坏球墨铸铁的近表层石墨球化，需通过控制回用率或改用非硫固化剂缓解。
- **脉纹与机械粘砂**：与高温下树脂砂热膨胀和型砂紧实度相关，可通过调整原砂粒度分布、使用涂料或添加抗脉纹剂改善。

## 旧砂再生

树脂砂旧砂再生主要采用机械摩擦再生或热法再生。呋喃树脂砂的再生回收率较高，一般可达 75%～80%，碱性酚醛砂约为 70%～75% [[S13]]。再生砂的粒度（AFS 平均细度）和流动性是评价再生效果的重要指标，例如某实验再生砂 AFS 达 56.12，流动性 14 s，与新砂性能接近 [[S19]]。铸造时砂铁比越小，粘结剂燃烧分解率越低，对再生砂性能有一定影响 [[S25]]。

## 相关标准

国内树脂砂标准体系以 GB/T 5611-2017《铸造术语》为基础，对各类树脂砂及其粘结剂进行定义和术语规范，配套行业标准主要包括 [[S26,S16,S21]]：
- JB/T 7526-2008《铸造用自硬呋喃树脂》
- JB/T 8583-2008《铸造用覆膜砂》
- JB/T 11739-2013《铸造用自硬碱性酚醛树脂》
- JB/T 3828-2013《铸造用热芯盒树脂》
- JB/T 11738-2013《铸造用三乙胺冷芯盒法树脂》
- JB/T 8834-2013《铸造覆膜砂用酚醛树脂》
