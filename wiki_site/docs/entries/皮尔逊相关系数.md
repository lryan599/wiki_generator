## 定义

皮尔逊相关系数（Pearson product‑moment correlation coefficient，简称Pearson's r）是统计学中用于度量两个变量之间线性相关强度和方向的核心统计量[[S21,S3,S24]]，定义为两个变量的协方差除以二者标准差的乘积[[S21,S3,S17]]。

其总体形式数学表达式为[[S3,S17,S14]]：

\[
r = \frac{\mathrm{cov}(X, Y)}{\sigma_X \cdot \sigma_Y}
\]

式中，cov(X,Y)为变量X与Y的协方差，σ_X、σ_Y分别为X和Y的总体标准差。

样本形式的皮尔逊相关系数完整展开式为[[S3,S17,S14]]：

\[
\rho(x,y) = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2 \sum_{i=1}^{n}(y_i - \bar{y})^2}}
\]

式中，x_i为样本x的第i个元素，x̄为样本x的均值；y_i为样本y的第i个元素，ȳ为样本y的均值；n为样本总量。

该系数在中文领域存在四个行业通用别名：**皮尔逊线性分析**、**皮尔逊相关系数公式**、**皮尔逊系数**、**皮尔逊相关系数法**[[S21]]。

## 数值含义与判定

皮尔逊相关系数的取值范围严格限定在[-1, 1]区间，其数值含义如下[[S21,S27,S7]]：

| 取值 | 含义 |
|---|---|
| r > 0 | 正相关，两变量变化方向相同 |
| r < 0 | 负相关，两变量变化方向相反 |
| r = 1 | 完全正线性相关，所有数据点落在一条斜率为正的直线上 |
| r = -1 | 完全负线性相关，所有数据点落在一条斜率为负的直线上 |
| r = 0 | 无线性相关（可能两变量完全无关，或存在非线性的函数关系） |

绝对值大小体现线性相关强度：|r|越接近1，线性相关性越强；接近0则线性相关性越弱[[S21,S7]]。常用的相关程度划分标准为[[S6]]：

| |r| 区间 | 相关程度 |
|---|---|
| 0 ~ <0.3 | 弱相关 |
| 0.3 ~ <0.5 | 中相关 |
| 0.5 ~ <0.8 | 强相关 |
| 0.8 ~ 1 | 极强相关 |

进行相关性判定时，需结合**显著性检验**：计算得到的r值应大于对应自由度、指定显著性水平下的临界值，方可判定线性关联具有统计意义[[S11,S13]]。

## 适用条件与局限性

### 适用条件

皮尔逊相关系数的应用前提包括[[S22,S10,S9,S13]]：

1. 变量总体近似服从正态分布；
2. 两变量之间存在线性相关趋势；
3. 数据集中无极端离群点；
4. 样本量足够大，避免小样本下r值虚高导致的假阳性关联。

### 典型局限性

1. **仅能检测线性关系**：无法度量变量间存在的非线性单调或非单调关联，即使两变量呈严格的函数型非线性相关，也可能得到r≈0的结果[[S22,S10,S1,S13]]；
2. **对离群点高度敏感**：少量远离数据主体的异常值可大幅偏移r的取值，完全改变关联判定结果[[S22,S10]]；
3. **小样本虚高**：样本量较小时|r|容易偏近1，易将随机波动误判为显著线性相关[[S13]]。

### 常见误用情形

典型的误用包括：直接用皮尔逊系数判定非线性关联强弱、未清洗离群点即计算关联结果、样本量极小时不进行显著性检验直接下结论、在变量明显不符合正态分布时仍强制使用皮尔逊系数等[[S26,S22,S10]]。

## 在压铸领域的应用

皮尔逊相关系数在压铸工程领域主要用于两方面：分析工艺参数与质量指标的线性相关性，以及识别并过滤强线性相关的冗余特征以降低模型过拟合风险与计算成本[[S21]]。

### 工艺参数与质量指标的相关性分析

针对铝合金汽车离合器壳体真空压铸工艺的研究表明，5项核心工艺参数与残余熔体模数的皮尔逊相关系数实测值如下[[S27]]：

| 工艺参数 | 皮尔逊相关系数 |
|---|---|
| 模具温度 | 0.29 |
| 浇注温度 | 0.03 |
| 慢压射速度 | 0.03 |
| 真空度 | 0.03 |
| 快压射速度 | -0.06 |

其中模具温度是与残余熔体模数线性相关性最高的参数，快压射速度与之线性相关性极弱[[S27]]。

在铝合金曲轴箱压铸产线中，压射压力设定值（MPa）与模具跑铝水故障率之间的皮尔逊相关系数达到0.921（P=0.000），呈现显著强正相关，表明压射压力异常升高会直接提升跑铝水故障风险[[S18]]。

### 特征筛选与冗余过滤

在压铸数据驱动的建模与质量预测中，皮尔逊相关系数被广泛用于特征降维：

- 某3200t压铸机采集的54931条镁合金仪表盘支架工业生产数据集中，对原始71个压铸工艺参数计算两两皮尔逊相关系数，以|r|>0.6为强线性相关阈值，每组强相关特征仅保留一个，最终筛选得到23个线性弱相关/不相关的独立特征用于后续铸件质量预测模型[[S23]]；
- 数据驱动调压铸造工艺智能设计案例中，对模数、各类壁厚、对称度、长宽比等9项工艺设计特征计算皮尔逊相关系数，以0.7为阈值，识别到平均壁厚与模数强相关、长宽比与长高比/宽高比强相关，剔除平均厚度和长宽比后剩余特征间无明显强线性相关性[[S16]]；
- 铝车轮压铸模具温度场预测研究中，用皮尔逊相关系数分析实际采集压铸工艺特征与各热电偶监测温度的相关性，判定铝液剩余量、工艺号两项与各监测温度普遍弱相关后直接删除；对铝液温度、熔炼炉温度虽相关性低但经核实系采集周期内波动不足所致，予以保留[[S17]]；
- 面向中小压铸企业的大数据分析平台中，对27项预处理后的传感器参数计算皮尔逊相关系数，保留|r|>0.7的13个关键核心参数，异常检测时可自动提示关联参数的同步检查[[S15,S2]]。

## 与Spearman秩相关系数的区别

皮尔逊相关系数与Spearman秩相关系数是压铸数据分析中最常用的两种相关度量，其核心差异如下[[S9,S12,S22,S10]]：

| 比较维度 | 皮尔逊相关系数 | Spearman秩相关系数 |
|---|---|---|
| 统计类型 | 参数方法 | 非参数秩方法 |
| 计算基础 | 原始数值 | 变量的排序值（秩） |
| 检测的关联类型 | 仅线性关系 | 单调线性及单调非线性关系 |
| 对数据分布的兼容性 | 要求近似正态分布 | 无严格分布要求 |
| 对离群点的敏感性 | 高度敏感 | 不敏感 |
| 鲁棒性 | 低 | 高 |

在压铸特征工程实践中，当数据集不满足正态分布假设或存在较多离群点时，应优先选用Spearman秩相关系数进行特征筛选[[S9]]。
