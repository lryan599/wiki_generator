## 概述

模芯（又称模仁；英文：mold core、die core、die insert）是压铸模具中心部位直接与熔融金属液接触的关键精密成形零件，其结构通常较为复杂，加工难度大，造价高，制造成本中人工支出往往高于材料成本[[S51,S24,S48]]。模芯作为压铸模具成形部分的核心组成部件，直接决定压铸件对应成形区域的尺寸精度与表面质量[[S38,S30]]。

按照安装位置，模芯可划分为**动模模芯**与**定模模芯**：定模模芯固定在压铸机定侧安装板对应位置，属于定模部分的核心成形结构[[S16]]；动模模芯则安装在压铸机动侧安装板上，是根据模具主零件尺寸计算生成的核心成形结构[[S10,S38]]。图1为压铸模具定模模芯的3D结构示意图。

## 与其它模件的关系

- **型腔（cavity insert）**：是用于形成压铸件外轮廓的镶块零件。模芯与型腔共同构成压铸模具的完整成形腔体，二者配合边界对应压铸件分型面区域[[S59,S6,S49]]。
- **型芯（core）**：是用于形成压铸件高精度内部形状的分立零件，可分为与开模方向平行的固定型芯、与开模方向不平行的活动型芯两类。模芯范畴通常包含型芯及部分独立的小成形镶件，二者为整体与下属组成部件的包含关系[[S19,S8,S49]]。
- **滑块（slide）**：是可沿垂直或成角度于开合模方向滑动的运动组件，滑块头部可集成侧型芯，用于成型压铸件的外部倒扣或侧向孔位结构。滑块不直接归属于模芯本体，二者为运动载体与安装在载体上的侧成形零件的装配关系[[S7,S52,S6]]。

## 材料性能要求

压铸模芯直接承受高温、高压、高速填充金属液的反复冲刷，对材料性能要求严苛，核心要求包括：抗热疲劳性能、高冲击韧度、高硬度、抗铝液熔损/焊合性、合适的导热系数及高抗回火稳定性。大流量一体化压铸的严苛工况对上述性能要求进一步提升[[S15]]。

杂质元素控制对模芯冲击韧度影响显著。国家标准规定：特级优质模具钢S含量≤0.003%、P含量≤0.015%；高级优质模具钢S含量≤0.003%、P含量≤0.020%[[S15]]。

## 典型材料及牌号

### H13钢（4Cr5MoSiV1）

H13（牌号4Cr5MoSiV1）是C-Cr-Mo-Si-V型热作模具钢，为压铸模芯最广泛应用的基础材料。其与JIS标准SKD61、德国1.2344属于同系列等效钢种，具备空冷淬硬特性，广泛适用于铝、锌、镁合金压铸[[S2,S23,S34]]。

H13钢的主要技术参数如下表所示（执行标准GB/T1299-2000）[[S2,S23]]：

| 项目 | 参数 |
|---|---|
| C含量 | 0.32%~0.45% |
| Mn含量 | 0.20%~0.50% |
| Si含量 | 0.80%~1.20% |
| Cr含量 | 4.75%~5.50% |
| Mo含量 | 1.10%~1.75% |
| V含量 | 0.80%~1.20% |
| S、P含量 | 均＜0.030% |
| 密度 | 7.8 g/cm³ |
| 热处理后硬度 | HRC47~49 |
| 800℃热导率 | 28.4 W/(m·K) |
| 22℃~500℃线胀系数 | 11.5×10⁻⁶/℃ |
| 适用工况 | 模腔温升＜600℃ |

H13钢冷热疲劳抗力显著高于传统3Cr2W8V钢，模具寿命可提升1~10倍[[S2,S5]]。该钢有良好的渗氮工艺性能，渗氮后可进一步提高耐磨性和耐蚀性[[S67]]。

### 高品质热作模具钢

瑞典一胜百生产的8407热作模具钢采用特殊精炼工艺，纯度和内部组织均匀性优于常规H13，等向性更优异，抗机械疲劳及热应力疲劳性能显著提升。在完全不损失韧性的前提下，硬度可比普通H13提升1~2倍，可大幅降低模芯服役过程中的开裂倾向[[S40]]。

在一体化压铸中，服役工况较好的部位可采用H13、SKD61、1.2344等热作模具钢；在深筋、近浇口等严酷工况部位，可选用进口Dievar、W350等高品质钢[[S15]]。此外，内嵌随形控温流道的增材制造模芯镶件正逐步推广，可大幅提高控温效率与精度[[S36]]。

## 表面处理技术

### 氮化处理

氮化（含软氮化/气体氮化）是压铸模芯广泛应用的表面处理工艺。氮化处理温度通常为480~600℃，工件处理后变形量极小。生成的氮化层显微硬度可达950~1200HV，表层化合物层厚度约10μm，下部氮扩散硬化层厚度不超过0.2mm。氮化可显著提升模芯表面耐磨性、热硬性、抗咬合性与抗铝液熔损能力[[S11,S32,S53]]。

### PVD涂层

PVD（物理气相沉积）工艺处理温度控制在200~550℃，低于热作模具钢的常规回火温度，处理完成后工件变形极小，无需额外热处理。压铸场景典型PVD涂层（TiN、TiAlN、CrN）厚度为2~4μm，其中TiAlN类涂层硬度可达3300HV，可大幅提升模芯抗铝液/镁液焊合、抗高速冲蚀性能。PVD工艺的局限性在于对深孔、窄沟槽部位的绕镀性较差，需配合专用工装夹具或多源靶布局保障涂层均匀性[[S21,S11,S60]]。

TiN、TiAlN、CrN类PVD涂层均可显著提升模芯抵抗730℃高温熔融铝液腐蚀冲蚀的能力，但TiN涂层抗氧化温度较低，长期高温服役下寿命较短[[S13]]。TiAlN基PVD涂层应用于镁合金压铸型芯销，服役寿命可达普通氮化处理型芯的2倍[[S66]]。TiAlN/a-C纳米复合PVD涂层对镁合金熔体润湿附着性极低，可支撑超过27.6万次镁合金压铸注射[[S1]]。

## 工作参数与寿命

铝合金压铸场景下，模芯常规循环使用寿命可达100,000模次左右；锌合金压铸用模芯因接触熔融金属温度更低，其使用寿命约为铝合金压铸模芯的10倍[[S56,S44]]。压铸模芯典型工作条件包括：铝合金液压入型腔速度45~180m/s，压力2000~12000N/cm²，保压时间5~20s，每次压射间隔20~75s；模芯表面承受高温金属液反复冲刷，脱模后进行喷水冷却[[S20]]。

在铝合金压铸充型过程中，充填速度可达40~60m/s（典型内浇口速度），金属液对模芯表面产生强烈的力、热、化学耦合作用[[S41,S63]]。充型追踪粒子分布图可直观展示金属液冲击定模型芯的过程[[S65]]。

![压铸模芯六大典型失效形式形貌分类示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/fcb6d541-6654-4fed-b574-b3138e7af622/resource)

### 1. 热龟裂（热疲劳裂纹）

热龟裂是铝合金压铸模芯最常见的正常失效形式，约占模具报废总量的80%[[S18,S69,S45]]。其机理为：模芯在反复激冷激热工况下，成型表层急剧升温膨胀受内部约束产生压应力，冷却时（尤其脱模剂喷淋快速降温）产生拉应力；循环交变应力超过材料屈服强度后萌生微裂纹，逐步扩展形成网状裂纹。热龟裂纹深度通常不超过1mm[[S28,S3,S36]]。铝原子扩散形成的脆性铁铝化合物也会成为裂纹萌生源[[S28,S17]]。

热疲劳寿命数值模拟的低疲劳寿命区域与实际生产中热裂纹萌生位置高度吻合[[S55]]。

**预防对策**：压铸前将模芯预热至300℃，生产过程中维持模温250~300℃，冷却水预先加热至30~40℃以减小温差；生产过程中定期进行中间去应力回火，对成型表面进行喷丸处理引入表层压应力延缓裂纹扩展；采用冷热疲劳抗力更优的H13钢替代3Cr2W8V钢，模具寿命可提升1~10倍[[S3,S33]]。

### 2. 冲蚀

冲蚀多发生在近浇口位置，是高速流动（典型内浇口速度40~60m/s）高温金属液对模芯表面的力、热、化学耦合作用结果：先冲蚀破坏表面防护涂层，裸露基材直接与高温金属反应生成脆性金属间化合物，该化合物持续被高速液流冲刷剥落形成表面损伤[[S41,S50,S63]]。实验表明，短周期（100次压射内）的高速液流直接冲击不会先形成显著冲蚀，往往在焊合层生成后，后续液流剥落脆性焊合层才逐步形成可见冲蚀形貌；长期服役下气泡溃灭引发的空化效应也会辅助加剧表面材料流失[[S64,S27,S43]]。

**预防对策**：选用耐液态金属熔损性能更优的热作模具钢；提升表面涂层与基材的结合强度；在满足充型要求的前提下降低铝液温度、模芯工作温度及内浇口压射速度[[S41]]。

### 3. 合金焊合（粘铝）

当模芯表面涂层脱落后，裸露钢铁基材直接与655℃以上铝液接触，铝原子向铁基材扩散生成FeAl₃等富铝脆性金属间化合物。该化合物的固液相区处于压铸工况温度区间时会形成可流动液相，冷却后将铸件与模芯表面焊合在一起，在浇口区域发生概率最高[[S61,S25,S54,S41]]。

**预防对策**：提升涂层与基体结合力；降低铝液和模具温度至铝铁反应临界温度以下；在易发生粘铝位置将脱模斜度从2°提升至5°，无需额外投入成本即可大幅降低粘铝扣模概率；选用Mo合金替代H13钢制作局部易受冲蚀的芯针，降低铝铁亲和性[[S4,S46]]。

### 4. 整体开裂

整体开裂属于非正常灾难性失效，是模芯在拉应力状态下的脆性裂纹失稳扩展结果，通常由偶然机械过载或热过载触发，常起源于尖角、控温流道接刀痕等应力集中位置[[S28,S17,S36]]。

**预防对策**：模芯所有尖角、过渡位置设置足够大的R角——将控温流道过渡位置R角半径从0.25mm提升至0.75mm，可使材料V型缺口夏比冲击功从7J提升至24J；选用洁净度高、S/P杂质含量低的电渣重熔热作模具钢；机加工后的刀痕、电火花加工痕迹须完全打磨抛光消除[[S36,S33]]。

### 5. 塑性变形

模芯工作温度长期超过材料回火温度时，基体马氏体持续分解发生软化，在压铸高压载荷作用下发生不可逆塑性流动，表现为模芯棱边堆塌、型腔面塌陷，导致铸件尺寸超差[[S28,S20]]。

**预防对策**：适当提高回火温度、延长回火保温时间，采用充分的两段回火工艺（例如200℃×1.5h + 260℃×1.5h），避免生产过程中模芯继续发生回火转变；在需要保证韧性的重载工况下，避免为提升韧性刻意降低模芯硬度，确保高温屈服强度处于安全阈值以上[[S20]]。

### 6. 热磨损

由反复热循环与铸件脱模时的机械摩擦共同造成：模芯表层循环软化后，表面氧化层失去支撑，在铸件脱模反复刮擦下逐步发生材料流失，造成表面粗糙度升高、尺寸精度下降[[S28,S17]]。

**预防对策**：对模芯成型表面进行氮化等表面强化处理，提升表层硬度和耐磨性；保证生产过程模温均匀稳定，避免局部过热[[S26,S9]]。

## 标准与规范

- **GB/T 24594《热作模具钢》**：规定了压铸模芯常用热作模具钢的牌号、各元素质量分数、淬火温度及冷却方式、用途举例等技术要求，是国内模芯材料选型的核心参考规范[[S62]]。
- **NADCA#207-2003《H-13钢显微组织验收标准》**：北美压铸协会（NADCA）发布的H13钢专用显微组织质量验收规范，对模具钢带状组织、显微偏析等缺陷的控制要求作出明确规定[[S35,S22]]。
- **NADCA压铸产品技术要求检验核对表**：将不同产品技术要求按对成本的影响分级，明确了不同精度等级模芯对应的验收成本影响，可用于模芯技术要求设定的参考[[S14,S39]]。

## 设计要点

模芯设计应特别关注：型芯零件脱模斜度的设置（例如在易发生粘铝位置可采用5°脱模斜度替代2°）[[S4]]；冷却通道布局应精准导出铸件凝固释放的结晶潜热，降低模具过热部位温度[[S36]]；在满足结构强度和充型要求的前提下，壁厚应尽量保持均匀，避免产生热节导致应力集中[[S17]]。
