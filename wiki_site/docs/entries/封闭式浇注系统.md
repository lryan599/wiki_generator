## 定义

封闭式浇注系统（choked runner system / pressurized gating system）是直浇道出口截面积大于各横浇道截面积总和、横浇道出口截面积总和大于各内浇道截面积总和的浇注系统[[S2,S19,S15]]。正常浇注条件下，该系统的全部组元（直浇道、横浇道、内浇道）均为液态金属所充满，因而也称为充满式浇注系统或有压/正压浇注系统[[S19,S16]]。其各组元截面积满足关系式：

> **F直 > F横 > F内** [[S2,S19,S15]]

国家标准GB/T 5611-2017给出了封闭式浇注系统的正式定义，规定标准英文译名为 choked runner system / pressurized gating system[[S2]]。中国机械工业标准汇编进一步将截面比关系 $A_1 > A_2 > A_3$ 作为封闭式浇注系统的分类判据[[S28,S41]]。

## 核心特征

封闭式浇注系统的阻流截面位于内浇道处，内浇道是整个浇注系统中截面积最小的部位[[S3,S19]]。由于阻流截面的存在，浇注过程中系统内维持背压（back pressure），所有截面上金属液的压力均高于型壁气体压力，从而保证浇道始终处于充满状态[[S9,S20,S16]]。

主要特征包括：
- **挡渣能力良好**：浇道始终充满，有利于熔渣和非金属夹杂物在横浇道中上浮并滞留，不进入型腔 [[S19,S45]]。
- **金属消耗少**：浇道体积相对较小，铸件出品率高，清理方便 [[S19,S45,S9]]。
- **内浇道出口流速高**：随着截面积逐步缩小，金属液在内浇道出口处流速显著升高，易产生喷射，对铸型及砂芯形成冲刷，型内金属液紊流程度较高，易发生氧化和卷渣 [[S19,S45,S24]]。

## 分类与对比

### 按截面比的浇注系统分类

浇注系统按各组元截面积比例关系可分为四种基本类型[[S3,S42]]：

| 类型 | 截面比关系 | 阻流截面位置 | 核心特征 |
|------|-----------|------------|----------|
| 封闭式 | A直 > A横 > A内 | 内浇道 | 充满流动，挡渣好，内浇道出流速度快[[S42,S19]] |
| 开放式 | A直 < A横 < A内 | 直浇道底部 | 充型平稳，冲刷小，挡渣差[[S19,S39]] |
| 半封闭式 | A直 < A横 且 A横 > A内 | 内浇道 | 先开放后封闭，介于两者之间[[S28,S3]] |
| 封闭-开放式 | 阻流设在横浇道 | 横浇道/阻流装置 | 先封闭后开放[[S3]] |

### 封闭式与开放式系统的关键对比

| 对比维度 | 封闭式浇注系统 | 开放式浇注系统 |
|---------|-------------|-------------|
| 截面比 | F直 > F横 > F内 [[S19]] | F直 < F横 < F内 [[S19]] |
| 阻流截面位置 | 内浇道 [[S3,S19]] | 直浇道底部 [[S9,S39]] |
| 浇道充满状态 | 各组元均充满 [[S19,S15]] | 正常浇注时难以完全充满 [[S19,S16]] |
| 挡渣能力 | 良好 [[S19,S45]] | 较差，需配合过滤网使用 [[S19,S16]] |
| 内浇道出流速度 | 高，易喷溅冲刷 [[S19,S24]] | 低，充型平稳 [[S19,S16]] |
| 金属消耗 | 少，出品率高 [[S19,S9]] | 多，出品率低 [[S9,S39]] |
| 适用合金 | 不易氧化的铸铁类 [[S19,S45]] | 易氧化轻合金、球铁件、漏包浇注铸钢件 [[S19,S45,S39]] |

## 设计准则与推荐数据

### 截面比（浇道比）推荐

不同铸造材质与工艺条件下，封闭式浇注系统的截面比存在差异。以下汇总来自教材、手册和研究文献的推荐数据：

| 材质/工艺 | 截面比（ΣA直∶ΣA横∶ΣA内 或相应标注） | 来源 |
|----------|----------------------------------|------|
| 普通砂型灰铸铁件 | 1.2∶1.5∶1 [[S12]] | 《金属液态成形工艺设计》 |
| 重型机械类灰铸铁件 | 1.2∶1.4∶1 [[S12]] | 同上 |
| 一般球墨铸铁件 | 内∶横∶直 = 1∶(1.2~1.3)∶(1.4~1.9) [[S27,S8]] | 铸造工艺设计简明手册 |
| 薄壁小型球墨铸铁件 | 内∶横∶直 = 3∶8∶4（内浇道宽厚比4:1，横浇道高宽比2:1）[[S8,S44]] | 新编铸造技术数据手册 |
| 完全封闭式小型/中等铸铁件 | 直∶横∶内 = 1.0∶(1.05~1.2)∶(1.1~1.4) [[S40]] | 钢铸件的浇注系统 |
| 低压铸造铝合金铸件 | 内∶横∶直 = 1∶1.75∶2.5 [[S22]] | 封闭式浇注系统（百科词条） |
| 铁系铸件（通用参考） | 直∶横∶内 = 1∶2∶1（不可用于铝/镁轻合金）[[S9,S39]] | 制造技术（英文版） |
| 铸钢小型锌合金件（封闭-底注） | 直∶横∶内 = 2.6∶1.6∶1 [[S7]] | 低熔点合金工艺研究 |

### 阻流截面积计算——奥赞（Ossan）公式

封闭式浇注系统的阻流截面积通常采用奥赞公式计算。该公式基于伯努利流体力学原理，将充满流态下的浇注系统等效为管道流动模型[[S25,S30]]：

$$S_{阻} = \frac{m}{\rho \mu t \sqrt{2gH_p}}$$

式中：
- $S_{阻}$ — 阻流截面积（内浇道总截面积）
- $m$ — 流经阻流截面的金属总质量
- $\rho$ — 金属液密度
- $\mu$ — 流量系数
- $t$ — 总充型（浇注）时间
- $H_p$ — 平均计算压力头
- $g$ — 重力加速度

该公式的适用场景限定为转包浇注的封闭式浇注系统[[S25,S30]]。

### 流量系数 μ 的取值

不同浇注系统组元配置下的流量系数 μ[[S14]]：

| 系统配置 | 流量系数 μ |
|---------|-----------|
| 浇口杯 + 直浇道（两组元） | 0.76 |
| 浇口杯 + 直浇道 + 横浇道（三组元） | 0.58 |
| 浇口杯 + 直浇道 + 横浇道 + 内浇道（四组元） | 0.48 |
| 湿型中、小型球墨铸铁件 | 0.35～0.50 [[S27]] |

### 平均压头 $H_p$ 的确定

根据内浇道在铸件上的注入位置，平均计算压头有不同算法[[S18,S35]]：

| 注入方式 | $H_p$ 计算公式 | 符号说明 |
|---------|---------------|---------|
| 顶注式 | $H_p = h$ | h——直浇道总高度 |
| 底注式 | $H_p = h - c/2$ | c——铸件总高度 |
| 分型面中注式 | $H_p = h - p^2/(2c)$ | p——分型面以上铸件型腔高度 |

### 浇注时间估算

球墨铸铁件封闭式浇注系统的经典浇注时间经验公式[[S27,S18]]：

$$t = (1.17 \sim 1.63) \cdot \sqrt[3]{G}$$

式中 $t$ 为浇注时间（s），$G$ 为流经浇道的金属液总质量（kg）。

### 内浇道流速限制

- 铸铁类铸件内浇道允许最大流速通常不超过 **1 m/s**[[S43]]。
- 铝合金砂型铸造的内浇道允许最大流速推荐控制在 **≤ 0.5 m/s**[[S33]]，以避免高速出流裹挟破碎氧化膜诱发夹杂缺陷。

## 结构与组成

封闭式浇注系统的基本组元包括浇口杯、直浇道、直浇道窝、横浇道、内浇道以及横浇道末端延长段[[S15,S46]]。其中横浇道末端延长段具有重要功能：容纳低温含渣的前端金属液，使干净的后续金属液通过内浇道进入型腔，起到辅助挡渣和稳定液流的作用。

## 应用场景

封闭式浇注系统主要适用于**不易氧化的各类铸铁件**，包括灰铸铁、球墨铸铁、可锻铸铁等[[S19,S45]]。铸造工艺设计通用原则指出，铸铁类铸件优先适配封闭式或半封闭式浇注系统，以便熔渣和非金属夹杂物上浮滞留在横浇道内实现有效挡渣[[S29]]。

不推荐用于以下场景[[S19,S45,S9]]：
- 易氧化的铝、镁等轻合金铸件（因内浇道流速高、紊流剧烈，加剧氧化）
- 漏包浇注的铸钢件（铸钢件通常优先选择开放式或半封闭式系统[[S29]]）
- 高大铸铁件

### 工程应用案例

- **大型灰铸铁件**：573 kg 灰铸铁前轴承座上半铸件采用封闭式浇注系统，截面比 ΣA直∶ΣA横∶ΣA内 = 1.4∶1.2∶1，阻流总面积为 26.4 cm²，设置 8 个梯形截面内浇道，实际充型时间约 35 s[[S34]]。
- **铜合金套类铸件**：采用半圆形蛇形直浇口及两道内浇口封闭式浇注系统的铜合金套类铸件，加工后合格率达95%以上[[S36]]。

## 设计不当所致的典型缺陷

封闭式浇注系统设计或选用不当时，可能引发的典型缺陷及其机理[[S19,S45,S31,S46]]：

| 缺陷类型 | 产生机理 |
|---------|---------|
| 喷射卷气（气孔） | 内浇道出口流速过高，金属液以喷射状进入型腔，卷裹型腔气体 |
| 冲砂、粘砂 | 高速金属液冲击铸型和型芯表面，破坏砂型表层完整性 |
| 夹渣（氧化夹渣） | 紊流剧烈导致金属氧化，氧化渣随液流卷入型腔 |
| 金属液扰动过大 | 充型不平稳，加剧各类铸件缺陷生成概率 |

## 与凝固补缩的关系

封闭式浇注系统可通过调整内浇道引入位置（如配合顶注或合适的底注布局），形成自上而下的温度梯度，满足铸件顺序凝固的要求；配合冒口设计可充分发挥冒口的补缩作用，减少铸件缩孔、缩松缺陷[[S17,S31]]。内浇道经过冒口引入铸件的封闭式浇注系统（常见于可锻铸铁件），既有利于挡渣，又能实现较快且平稳的充型[[S12]]。

## 相关标准

- **GB/T 5611-2017《铸造术语》**：规定了封闭式浇注系统的术语定义与分类，标准英文译名为 choked runner system / pressurized gating system[[S2]]。
- **JB/T 7162《凿岩机械与气动工具用结构钢熔模铸件》**：包含对应的试样浇注系统设计要求[[S32]]。
- **ASTM B208《Standard Practice for Preparing Tension Test Specimens for Copper Alloy Sand, Permanent Mold, Centrifugal and Continuous Casting》**：明确了铜合金浇注试样的浇注系统设计规范[[S13]]。
- ISO 国际标准体系中针对封闭式浇注系统的专属术语定义及专项设计导则：**未找到可靠来源**。

## 术语注记

- 国标 GB/T 5611-2017 将封闭式浇注系统与阻流浇注系统（chocked runner system）定义为两个独立概念：封闭式浇注系统的阻流截面设在内浇道，而阻流浇注系统的阻流段常设在直浇道出口和横浇道前端[[S2]]，两者存在明确区分。
- 行业通用中文同义名包括：收缩式浇注系统、充满式浇注系统、正压浇注系统[[S19,S16,S10]]。
- 英文术语 pressurized gating system 与 choked gating system 为通用同义译名[[S2,S22]]。
