精速密压铸是一种通过同轴双冲头进行二次挤压补缩的特种压铸工艺，其标准英文对应术语为 Acurad die casting，在国标铸造术语标准中正式定名为“精速密压铸[双冲头压铸]”[[S20,S36,S9]]。该工艺别名“套筒双冲头压铸法”，得名于其压室套筒内同轴嵌套大、小双冲头的独特压射结构，区别于采用外部独立增压缸的普通二次加压压铸[[S18,S29,S10,S12]]。

关于“Acurad”一词的构成，权威文献存在两套合理的解释：1975年引进的《ACURAD铸造法》技术译著记载，该词由 accurate（精确）、rapid（迅速）、dense（致密）三个形容词的词冠合成，对应中文“精、速、密”的命名逻辑，常用工业缩写为ARD [[S8,S40]]；而NADCA关联的《Die Casting Engineering》文献则记录其为 accurate（精确）、reliable（可靠）、dense（致密）的合成词[[S26,S1]]。

### 历史起源
精速密压铸由美国通用汽车公司（GM）联合雷诺兹金属公司、赛拉丘萨大学共同研发，1959年完成核心技术发明，1966年正式对外公开并将其应用于公司内部的45台冷室压铸机，从而实现了规模化生产[[S8,S35]]。1967年，日本推出首台800吨级商业化精速密压铸机；随后美国、日本、西德等国先后将该工艺纳入工业压铸装备序列[[S8,S33]]。

### 工艺分类与定位
精速密压铸归属于高致密、高完整度压铸范畴，是铝合金特种压铸的经典改进子类，也是间接挤压铸造技术的前驱，不属于半固态压铸[[S26,S1,S17]]。与常规冷室压铸及热室压铸相比，该工艺在普通冷室压铸机的基础上增设了套筒内置的双冲头二次补压机构，整体压射速度远低于常规压铸，内浇口厚度显著增大；与普通外部独立增压的二次加压压铸相比，其补压动作由嵌套在大冲头内部的中心小冲头在压室套筒内直接完成，工艺边界清晰[[S33,S11,S40]]。

### 工作原理与双冲头动作逻辑
精速密压铸的核心结构为同轴嵌套的大、小双冲头，两者组成一体但各自独立地连接专属液压缸，由独立液压驱动力分别控制运动[[S18,S39,S25]]。其完整工作时序如下：

1. **同步充型阶段：** 压射初始时，大冲头（外压射冲头）和小冲头（内压射冲头）同步向前推进，共同完成金属液自下而上的平稳充型。
2. **二次补压阶段：** 待铸件外壳形成凝固层、大冲头因受到凝固前沿阻力无法继续前行时，小冲头在独立液压缸驱动下单独继续向前行进50–150 mm，将压室中尚未完全凝固的剩余金属液压入型腔，实现补缩压实[[S39,S25,S5,S7]]。

### 关键工艺参数
精速密压铸通过慢速充型与厚大内浇口保证金属液充填平稳，并借助受控的顺序凝固方向使二次补压能够有效传递到整个铸件。其关键工艺控制参数如下表所示。

| 工艺参数项 | 数值或范围 | 说明 |
|---|---|---|
| 慢压射速度 | 0.1–0.5 m/s | 远低于常规压铸，整体压射速度仅为普通压铸的10%左右[[S39,S25,S33]] |
| 内浇口速度 | 4–6 m/s | 约为普通压铸的20%，大幅降低涡流与卷气倾向[[S39,S25,S5]] |
| 内浇口厚度 | 5–15 mm | 内浇口截面积约为普通压铸的10倍，设计原则要求补压启动时内浇口区域尚未完全凝固[[S39,S25,S5]] |
| 二次补压比压 | 3.5–100 MPa | 部分工程手册标注为350~1000公斤/厘米²，低于常规压铸的峰值增压压力[[S39,S25,S5,S33]] |
| 小冲头补压行程 | 50–150 mm | 小冲头独立推进行程[[S39,S25,S5,S7]] |
| 补压延时窗口 | 50–200 ms（可调，±10–20 ms） | 精准匹配铸件外层凝固、内层金属液仍可流动的时间点[[S19]] |

精速密压铸的典型压室参数可参见一例实测设计数据：压室长度374 mm，冲头直径70 mm，一次快压射长度264 mm，二次快压射长度110 mm[[S34]]。

### 装备结构与系统改造
精速密压铸专用设备在常规冷室压铸机基础上至少新增三大核心系统[[S24,S19,S33]]：
- **嵌套式双压射缸总成**：可独立驱动大冲头（外压射冲头）和小冲头（内压射冲头），实现各自的差异化运动控制。
- **独立的延时补压蓄能器液压系统**：支持独立于充型系统的延迟增压控制，补压动作不受充型阶段液压回路波动的影响。
- **压铸模恒温精准控制系统**：通过使铸件从远离内浇口的远端向浇口方向逐层顺序凝固，延长高压补压的有效传递距离，彻底消除缩孔缩松[[S39,S25,S7]]。

双冲头部分还配备有专属的冷却与润滑系统：大冲头通过外冲头内部的螺旋式水道实现循环冷却，小冲头通过中心独立内水管完成冷却；润滑方面采用自动定量装置，可分别通过外压射杆油路为大冲头配合面供油、通过外压射杆内部细长油路为小冲头配合面供油，单次润滑量在0.2–2 ml范围内可调节[[S24]]。

常规精速密压铸机的最小适配锁模力通常在4000 kN以上，应用机型吨位范围可达300吨至2500吨[[S5,S33,S6]]。

### 铸件性能优势与材料行为
精速密压铸件内部孔隙率低、组织均匀，其在力学性能与致密性方面的量化改善如下：

| 性能指标 | 改善效果 |
|---|---|
| 致密度提升 | 总体提高3%–5%（日本实测场景下平均值提高1%–2%）[[S5,S2,S33]] |
| 抗拉强度提升 | 约提高20%以上，部分实测平均值可达17.7公斤/毫米²（普通压铸件为12.9公斤/毫米²）[[S11,S4,S13]] |
| 强度提升（综合） | 约提高30%[[S15,S4]] |
| 疲劳强度提升 | 提高15%–20%[[S15,S4]] |
| 铸件内部硬度提升 | 提高20%[[S15,S4]] |
| 可热处理性 | 530°C保温4小时条件下无鼓泡，可正常实施T6等热处理并大幅提升屈服强度[[S11,S38,S23]] |
| 可焊性与高温喷涂 | 精速密压铸件可进行焊接及高温喷涂，彻底突破普通压铸件的性能限制[[S11,S38,S29]] |

### 微观组织与内部质量对比
与普通压铸件相比，精速密压铸件的铸态组织极细小，呈现典型的金属型铸造凝固特征：表面下约0.5 mm处为致密冷硬层，再向内依次为柱状晶和细小的等轴晶，枝晶边缘无明显的微缩孔[[S13]]。下方X光透视对比图直观展示了精速密压铸件（上图）与普通压铸件（下图）在内部气孔数量上的显著差异[[S30]]：

### 工艺局限性与弊端
精速密压铸在带来显著致密性优势的同时，也存在明确的工程限制[[S15,S31,S5]]：
- **壁厚限制：** 无法稳定成型5 mm以下的薄壁铸件。
- **内浇口与余料处理：** 内浇口厚度远大于普通压铸，大件内浇口厚度可达15 mm，无法敲断去除，需要锯除；余料和浇道重量更大，材料利用率低于普通压铸。
- **模具冷却加工难度：** 完全按照热分析结果加工模具冷却水孔道，在实际工业场景下往往难以实现。
- **设备复杂性与适用范围：** 设备结构相对复杂，工艺参数控制难度较高，不适合小型压铸机，通常仅用于锁模力4000 kN以上的大型设备。

### 适用的合金类型与典型应用
精速密压铸是首个可成功压铸A356、A357等含少量铁的铝合金的工艺，这些合金在传统压铸中容易与模具发生焊合[[S26]]。该工艺同样兼容ADC12、ZL101、ZL104等铝硅系压铸铝合金，并支持锌合金、镁合金以及MIL-A-21180美军标特种铝合金等有色合金的致密成型[[S26,S37]]。

该工艺特别适用于需要耐压密封、可热处理强化或后续焊接的高致密性零件。自20世纪60年代末商业化以来，日本在70年代部署了超过100台300吨至2000吨级的精速密压铸机，批量生产阀体、耐压壳体等车用高性能铸件；通用汽车同期以精速密压铸机替换其500吨以上普通压铸机，开展耐压类零件的大规模量产[[S6]]。
