## 概述

压铸法兰盘是采用压力铸造工艺成型的金属法兰构件，区别于锻制法兰，通常以铝基、锌基、镁基等低熔点合金为主要材料，具备近净成形、加工余量小、批量制造成本低的特点[[S37,S4]]。作为特种压铸件，其核心功能是通过法兰盘本体、螺栓孔、密封面及颈部等结构实现管路或设备的可拆卸密封连接[[S8,S57]]。

## 分类与压铸适配形式

依据ASME BPVC规范中法兰与管路的连接类型，压铸法兰可适配为两种行业特有形式[[S46]]：
- **整体法兰**（Integral Type Flange）：法兰与管件一体压铸成型，形成刚性整体结构。
- **分离式松套法兰**（Loose Type Flange）：法兰本体与管件无直接刚性连接，各自成型后装配。

在上述基础上，压铸可实现的主流基础法兰适配形式包括压铸成型板式平焊法兰、压铸带颈对焊法兰、压铸整体法兰、压铸盲法兰等，均通过压铸直接成型基础轮廓，再辅以局部机加工达到装配精度要求[[S3]]。此外，金属成型领域按结构变形特征可划分为直边法兰、拉伸法兰、收缩法兰三大基础类，并可衍生出卷边法兰、反向法兰、孔法兰等复合结构，这些分类可为压铸法兰异形结构的设计提供参考[[S23]]。

## 结构与组成

压铸法兰的典型结构由四个核心部分组成[[S8,S57]]：
- **法兰盘本体**：承受连接载荷的主要盘状结构。
- **螺栓安装孔**：分布在法兰圆周上，用于穿入螺栓锁紧。
- **密封工作面**：提供接触密封的配合面，其平面度和粗糙度直接影响密封性能。
- **颈部结构**：与管路或设备连接，起到过渡导流作用。

一个量产铝合金压铸法兰的产品实例尺寸为φ80mm圆盘形，带有轴向抽芯成型的中心通孔和周向分布多个固定连接孔，整体高度18mm，最小壁厚1.8mm，材料为YL102铝合金，不允许存在冷隔、夹渣等缺陷[[S7]]。

## 材料

压铸法兰主流应用合金包含Al-Si-Cu系ADC12、A380压铸铝合金，AZ91D/AM50压铸镁合金以及高流动性压铸锌合金[[S47,S16,S50]]。其中：
- **ADC12** 执行日本JIS H5302-2006标准，硅含量9.6~12%，典型抗拉强度约240MPa，硬度≥85HV[[S16,S47]]。
- **A380** 符合美国ASTM B85系列标准，多用于轻量化法兰壳体场景[[S47]]。
- **AZ91D** 压铸镁合金室温抗拉强度约193~207MPa，延伸率约3%，硬度典型值63HB[[S50]]。

铝基压铸法兰的材料合规要求需符合现行GB/T 15115-2024《压铸铝合金》标准，该标准规定了压铸铝合金的牌号、成分及性能检验方法[[S41,S35]]。

## 工艺参数与控制要求

以Al-Si系A356材质铝合金压铸法兰为例，其典型压铸工艺参数为：浇注温度680℃，浇注压力85MPa，快压射速度1.5m/s[[S15]]。当法兰作为嵌件嵌入复杂壳体压铸件时，工艺设计要点包括：嵌入部位需增设滚花、切槽、铣扁等防滑移结构；嵌件周围金属层最小厚度需符合规范要求；嵌件边缘避免尖角；嵌件定位必须稳固；且需避免嵌件材料与压铸合金产生电化学腐蚀[[S13,S1]]。

铝合金压铸嵌入法兰壳体的工艺参数推荐区间如下[[S29,S54,S36]]：

| 工艺参数         | 推荐范围            | 典型生产取值 / 备注            |
|------------------|---------------------|--------------------------------|
| 浇注温度         | 640~720℃           | 常规约670℃                    |
| 压射比压         | 30~80MPa           | 50MPa（壳体类常见）            |
| 充型速度（厚壁） | 25~35m/s           | 内浇口处金属液速度            |
| 充型速度（薄壁） | 45~60m/s           | 薄壁复杂壳体部位               |
| 模具预热温度     | 150~300℃           | 稳定生产控制200~240℃          |
| 模具冷却水温差   | 进出口温差≤20℃     | 水路控温要求                   |

压铸法兰后续加工精度要求：密封面端面车削后表面粗糙度推荐Ra0.8~3.2μm，密封面装配间隙控制在0.1mm左右；法兰螺栓孔采用钻孔‑扩孔‑铰孔工艺，孔径大于18mm时加工余量推荐为孔径的0.4%~0.6%且总余量小于0.3mm；法兰孔口及边缘锐边需倒圆0.1~0.15mm，不允许残留飞边毛刺[[S14,S25,S30]]。

## 常见缺陷与机理

压铸件常见的10类典型缺陷包含气孔、缩孔、疏松、流痕、冷隔、气泡、欠铸、花纹、硬质点、裂纹，均可出现在压铸法兰上，直接影响产品质量[[S52]]。以下为法兰密封性能相关性较高的缺陷机理：

- **气孔与缩松**：高压高速充填过程的卷气现象是气孔的主要诱因，气体多呈变形球形、表面带氧化亮斑；缩孔缩松则因凝固过程中孤立液相区得不到充分补缩而产生。二者是导致法兰密封面承压泄漏的核心内部缺陷[[S26,S53]]。
- **冷隔**：金属液流股温度偏低，相互对接时未能完全熔合形成不规则线形缝隙，部分呈穿透状态，若出现在密封面位置将直接破坏密封面的连续致密性[[S58]]。
- **热裂纹**：多形成于铸件凝固阶段，出现在固相线温度附近的低塑性温度区间，常起源于气孔、缩孔等已有缺陷位置，裂纹周围伴随大量缩松组织，大幅降低法兰力学性能和密封可靠性[[S24]]。
- **变形（翘曲/扭曲）**：结构设计不合理导致不均匀收缩、顶出受力偏斜、开模过早等诱因可造成法兰面平面度超差，影响后续密封装配[[S32,S58]]。

当压铸法兰密封面区域存在超过0.01mm的开口裂纹、连通性气孔或缩松缺陷时，介质会沿缺陷形成的连通通道穿透法兰壁厚，直接导致流体泄漏[[S28]]。高压工况下密封面承受过大比压超出材料屈服强度时，会发生大于0.01mm的塑性变形，引发平面度偏差或局部塌陷，同样破坏密封精度[[S33]]。对于微小孔隙缺陷，可通过真空浸渗工艺进行封堵补救，大幅降低产品报废率[[S45,S28]]。

## 质量标准与检测

压铸法兰的耐压密封性检测包含水压试验与气压试验两类主流方法：水压试验需将试验压力提升至铸件工作压力的1.25~1.5倍，保压10~20分钟，无水滴渗漏为合格；气压试验通入1.05MPa（或超过工作压力30%~50%）的压缩空气，将工件浸没水中5秒以上无连续气泡溢出即为合格[[S17,S31]]。ASME锅炉及压力容器规范第VIII卷要求承压铸件按ANSI B16.5标准对应等级的壳体试验压力进行水压试验，保压期间无任何泄漏即为合格[[S44,S39]]。法兰外形连接尺寸可参照ASME B16.5标准中规定的螺栓孔分布及密封面尺寸进行匹配，但工况压力需控制在材料允许的安全阈值内[[S49,S34]]。

尺寸检测方面，基础尺寸可用卡尺、千分尺等常规量具核验，精密公差要求则使用三坐标测量机进行全尺寸扫描，表面粗糙度可采用比较样块比对，符合GB/T 6414等相关标准[[S17,S12]]。表面开口缺陷可采用渗透探伤方法检测，可识别宽度小于0.01mm的表面微裂纹及连通孔洞；内部缩孔、缩松、裂纹、气孔等缺陷可采用X射线探伤或工业CT断层扫描，承压类产品关键区域孔隙率需控制在0.3%以下[[S5,S18]]。镁合金压铸法兰还应符合GB/T 25747-2022的相关检验规则[[S11]]。

## 应用领域

压铸法兰在汽车制造业中广泛应用于各类管路连接场景，涵盖汽车空调压缩机壳、冷却系统管路、燃油管路等部位的密封连接，该类铝合金压铸法兰可满足≥2.9MPa的氦气检漏要求，是汽车轻量化方案的重要组成零件[[S42,S19,S45]]。在液压系统中，压铸法兰被大量用作高压管路的连接部件，可适配不同压力等级的液压油管路、液压阀组接口、液压缸端盖等部位，法兰盘式管接头的喇叭口锥度要求为45°，椭圆度不超过0.05mm[[S2,S51]]。此外，在气动元件、通用机械连接场景中，压铸法兰主要用于低压/中压压缩空气管路接口、通用机械机架管路连接及流体输送系统配套部位，依托压铸工艺高精度、少后续加工的优势实现大批量低成本生产[[S27,S10]]。
