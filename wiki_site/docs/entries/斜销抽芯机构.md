## 概述

斜销抽芯机构（angled pin core pulling mechanism）是压铸模侧向抽芯机构中应用最广泛的类型之一，属于利用压铸机开模力实现侧向抽芯的典型机动抽芯结构[[S2,S7,S29,S46]]。该机构通过将开模方向的运动转化为滑块侧向运动，完成成型侧孔、侧凹、倒扣等特征的活动型芯抽出，是实现压铸件顺利脱模的关键功能组件。中文公认别名包括斜销-滑块侧抽芯机构、斜导柱侧抽芯机构，标准英文术语为 angle pin / angled pin core pulling mechanism[[S25,S13]]。

斜销抽芯机构具有结构简单、制造方便、可直接利用压铸机开合模动力完成抽芯与复位的特点，在中小型压铸模中应用广泛[[S2,S7]]。其抽芯距离受斜销倾角与开模行程约束，一般适用于抽芯距离较短、抽芯力不大的场景[[S14,S43]]。

## 工艺作用

斜销抽芯机构的核心工艺作用是在压铸件冷却凝固后，将成型侧向特征（如侧孔、侧凹、U形薄板长圆孔等）的活动型芯从铸件中抽出，解除侧向包紧力对脱模的阻碍[[S14,S43]]。开模时，压铸机的开模力通过斜销作用于滑块，分解为沿滑块侧向的抽芯分力与沿斜销轴向的分力，克服压铸件冷却收缩对侧型芯的包紧力，驱动滑块沿导滑槽完成抽芯动作[[S4,S46]]。合模时，斜销重新进入滑块斜孔，带动滑块及活动型芯复位至成型位置，楔紧块随后锁紧滑块，承受压铸充型时金属液的高压侧推力[[S12,S30]]。

## 结构组成

斜销抽芯机构的通用典型结构由五大功能部分组成[[S37,S34]]：

- **成型元件**：活动型芯、侧型芯，用于成型压铸件的侧孔、侧凹等特征。
- **运动元件**：滑块（slide block），带动侧型芯在动模套板的导滑槽内做定向滑动。
- **传动元件**：斜销（angle pin），通过与开模方向成一定倾斜角的结构，在开合模过程中驱动滑块完成抽芯与复位动作[[S30,S12]]。
- **锁紧元件**：楔紧块（wedge block/locking wedge），合模后压紧滑块，承受压铸时金属液的高压侧推力，避免滑块后退[[S30,S12]]。
- **限位元件**：弹簧、拉杆组合及限位块（或挡块），开模抽芯完成后驱动滑块保持在设定限位位置，确保合模时斜销可准确插入滑块斜孔[[S30,S36]]。

关于各核心零件的通用功能说明：

① **斜销**：为机构传动核心，其与开模方向成固定倾角，开模过程中与滑块斜孔产生相对位移，驱动滑块完成侧向抽芯及复位动作[[S30,S12]]。

② **滑块**：安装在动模套板的导滑槽内，带动侧型芯沿导滑方向做往复直线运动。

③ **楔紧块**：合模后压在滑块尾部，斜面研合接触，承受压铸时金属液对侧型芯的反压力，防止滑块后退出位[[S30,S12]]。

④ **弹簧与拉杆组合**：在抽芯结束后拉紧滑块，使其准确停靠在限位块上，保证合模时斜销顺利插入滑块斜孔[[S30,S12]]。

⑤ **止动调整元件（止钉）**：仅在部分定制化案例中出现，可通过螺纹旋入深度微调滑块的最终停止位置，实现抽芯距离的精确调整，不属于行业通用设计[[S18,S36]]。

## 工作原理与运动过程

斜销抽芯机构的动作过程可分为三个阶段[[S30,S12,S36]]，如图3所示[[S17]]：

![斜销抽芯机构合模锁紧状态](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/47969ac0-5bea-4e79-9600-028bf24cb76a/resource)

> 图注说明：图3出自《压铸模具设计及CAD》，展示合模锁紧状态下斜销、滑块、楔紧块的位置关系，可用于说明合模阶段的机构状态[[S17]]。

**第一阶段——合模锁紧阶段**：斜销完全进入滑块斜孔中，滑块由楔紧块完全压紧锁牢，侧型芯处于成型工作位置，模具处于待压铸状态[[S30,S36]]。

**第二阶段——开模抽芯阶段**：压铸机开模力驱动动模与定模分离。斜销固定于定模套板，滑块随动模运动，斜销与滑块斜孔产生相对位移。在斜销斜面的作用下，滑块沿动模套板导滑槽向外侧移动，带动侧型芯从压铸件侧孔或侧凹中脱出，完成抽芯动作[[S30,S4,S12,S36]]。

**第三阶段——抽芯结束阶段**：斜销完全脱离滑块斜孔后，滑块在弹簧作用下被拉紧在限位块上，保持在抽芯最终位置。此过程保障后续合模时斜销可准确插入滑块斜孔，驱动滑块完成复位[[S30,S12,S36]]。

**复位过程**（合模时）：斜销重新进入滑块斜孔，沿斜孔推动滑块向内侧移动，将侧型芯复位至成型位置。随后楔紧块与滑块尾斜面再次研合锁紧，为下一压铸循环做好准备。

**斜销抽芯的力学逻辑**：压铸机的开模力为抽芯主动力，作用于斜销后分解为沿滑块侧向的抽芯分力与沿斜销轴向的分力，克服压铸件冷却收缩对侧型芯的包紧力，实现侧向抽芯[[S4,S46]]。

## 关键设计参数与计算

斜销抽芯机构的核心设计参数包括斜销倾角、斜销直径、抽拔距（抽芯距）、斜销总长度、楔紧块楔角、滑块导滑长度等。以下关键参数取值根据设计手册及标准推荐[[S2,S7,S22,S39,S44,S45,S3,S15,S10,S20,S35]]。

### 斜销主要参数

| 符号 | 作用 | 选用尺寸/取值范围 |
|------|------|-------------------|
| α | 强制滑块作抽芯运动 | 10°~25°，常用15°~22° |
| d | 承受抽芯力 | φ10~40mm |
| L1 | 固定于模套内，使斜销工作稳固可靠 | 按计算确定 |
| L2 | 完成抽芯所需工作段尺寸 | 按计算确定 |
| L3 | 引导部分，插芯时保持斜销准确插入滑块斜孔 | β=α+(2°~3°)或β=30° |
| B | 工作段两侧铣平面宽，减小摩擦阻力 | B=0.8d |

来源[[S39]]。

**斜销倾角α**：推荐取值范围为10°~25°，工程常用区间为15°~22°，最大不得超过25°[[S2,S7,S22,S39,S33]]。倾角过大会导致斜销所受弯曲力 P=F/cosα 显著增大，引发斜销变形断裂，同时增大滑块与导槽间的摩擦阻力导致滑动卡滞[[S1,S33]]。当抽芯距离短而抽芯力大时，倾角宜取小值；当抽芯距离长、抽芯力小时，倾角可取较大值[[S43]]。

**斜销直径d**：通用选取范围为φ10~40mm[[S39]]。直径受抗弯强度限制，计算公式为[[S2,S7,S22]]：

$$d \\geq \\sqrt[3]{\\frac{10Fh}{\\cos^2\\alpha[\\sigma]_w}}$$

其中：F为抽芯力（N），h为A、B点在开模方向上的距离（mm），α为斜销倾角，[σ]ₓ为斜销材料许用抗弯强度，通用取300MPa[[S2,S7,S22]]。

**抽拔距（抽芯距）S**：通用计算公式为[[S44,S45,S3,S15]]：

$$S = h + K$$

其中h为侧凹/侧孔的最大成型深度，K为预留安全值，通用推荐K取3~5mm。对于延时抽芯场景，延时距离S推荐取值2.5~5mm，延时滑块斜孔的延时余量δ计算公式为 δ = S × sinα[[S3,S15]]。

**斜销总长度L**：通用计算公式为[[S22]]：

$$L = \\frac{1}{2}D\\tan\\alpha + \\frac{H}{\\cos\\alpha} + \\frac{1}{2}d\\tan\\alpha + \\frac{s}{\\sin\\alpha} + (5\\sim10)$$

其中：D为斜销固定台阶直径，H为斜销固定部分套板厚度，d为斜销直径，s为抽芯距，α为斜销倾角。

### 楔紧块与导滑参数

| 参数项 | 要求/推荐值 | 说明 |
|--------|------------|------|
| 楔紧块楔角α' | α'=α+(2°~3°)，也可直接取30° | 必须大于斜销倾角，确保合模时将滑块完全压牢 |
| 滑块导滑长度 | ≥滑块宽度的1.5倍 | 保证滑块运动平稳 |
| 抽芯后滑块留槽长度 | ≥滑块总长的2/3 | 避免复位时滑块偏斜 |
| 滑块定位弹簧力 | ≥滑块自重的1.5~2倍 | 保证抽芯完成后滑块可靠定位 |
| 滑块斜孔与斜销间隙 | 滑块斜孔直径比斜销大0.5~1mm | 避免侧向压力直接作用在斜销上 |

来源[[S10,S20,S35,S44,S28]]。

楔紧块的楔角α'需大于斜销倾角α，通用推荐差值为2°~3°，即α'=α+(2°~3°)，也可直接取30°。该差值可保证合模时楔紧块完全压牢滑块，压铸时合金液侧向压力不会传导至斜销导致其弯曲变形[[S39,S10]]。

### 斜销固定端与配合要求

| 参数项 | 要求 |
|--------|------|
| 固定端配合长度l | l≥1.5d |
| 固定端台阶外径D | D=d+(6~8)mm |
| 固定端台阶高度h | h≥5mm |
| 斜销与安装孔配合 | H7/m6 |
| 斜销与滑块斜孔配合 | H11/h11（可留有0.5~1mm间隙） |
| 工作段两侧铣平面宽B | B=0.8d |

来源[[S39,S30,S19]]。

## 与注塑模斜导柱机构的差异

压铸用斜销抽芯机构与注塑用斜导柱抽芯机构在配合间隙与承载方式上存在核心差异，主要表现在以下方面[[S9,S28,S13]]：

| 对比维度 | 压铸斜销抽芯机构 | 注塑斜导柱抽芯机构 |
|----------|------------------|---------------------|
| 滑块斜孔与斜销（导柱）间隙 | 滑块斜孔直径比斜销大0.5~1mm | 通常为小间隙配合 |
| 斜销（导柱）是否承受侧推力 | 不承受，侧压力完全由楔紧块承担 | 斜导柱可辅助承担部分熔体侧推力 |
| 楔紧块斜角与斜销倾角关系 | 楔紧块斜角比斜销倾角大3°~5° | 通常差值2°~3° |
| 斜销（导柱）材料工艺要求 | 45钢、T8、T10、渗碳钢，热处理硬度50~55HRC以上 | 材料要求相对较低 |
| 适用倾角范围 | 10°~25° | 10°~25° |

来源[[S9,S28,S13,S39,S33]]。

## 与其他侧抽芯机构的比较

斜销抽芯机构与其他常用侧抽芯机构的特点对比[[S14,S43]]：

| 抽芯方式 | 驱动原理 | 优势 | 局限 | 适用场景 |
|----------|----------|------|------|----------|
| **斜销抽芯** | 开模力通过斜销驱动滑块 | 结构简单，制造方便，成本低 | 抽拔距受限制，需一定开模行程 | 中小型芯、抽芯力不大、抽芯距较短 |
| **弯销抽芯** | 开模力通过弯销驱动滑块 | 可设计多段斜角，承受较大弯曲应力，弯销斜角最大可取30° | 制造加工复杂、费时 | 抽芯距较长、需要变速度抽芯的场景 |
| **液压缸抽芯** | 独立液压动力 | 传动平稳，抽芯力大，抽芯距长，运动独立于开合模 | 需专门液压管路，操作程序增加 | 大中型模具、抽芯角度特殊的场合 |
| **斜滑块抽芯** | 开模时斜滑块沿斜面滑动分型抽芯 | 结构紧凑 | 适用于较浅、面积较大的侧向分型 | 侧凹深度较浅的铸件 |

来源[[S14,S43,S29]]。

斜销抽芯机构作为机械式抽芯机构中最普遍的形式，最大优势在于结构简单、工作可靠、无需额外动力。其抽芯距离等于抽芯行程乘tanα，抽芯所需开模距离较大是其主要限制[[S14]]。

## 常见缺陷、故障及对策

斜销抽芯机构在服役过程中易出现以下典型故障[[S1,S24,S27,S31,S43,S14,S33,S47]]：

| 故障模式 | 根本原因 | 整改对策 |
|----------|----------|----------|
| **斜销变形断裂** | ①斜销倾角超过25°；②斜销材料选用不当、刚度不足；③斜销与滑块斜孔配合同轴度差产生蹩劲 | ①斜销倾角控制在25°以内；②选用韧性氮化合金钢制作斜销；③保证斜销安装孔与滑块斜孔垂直度公差符合设计要求 |
| **模具滑块卡死** | ①滑块抽芯后露出导滑槽长度超过1/3导致合模复位时歪斜；②多斜销驱动滑块时各斜销动作不同步产生互相牵制；③导槽加工精度不足配合间隙过小 | ①保证滑块抽芯后留在导槽内的长度不小于导槽全长的2/3；②多斜销布置时保证各斜销与对应斜孔配合精度一致；③导滑槽配合精度选用H7/f6 |
| **滑块烧付拉毛** | ①滑块成型面硬度不足，铝合金熔液高温冲刷粘接；②滑块导滑面未开设规范油槽，润滑不足 | ①滑块成型面镀铬或选用铬钒钢做氮化处理；②导滑面采用硬度高于滑块本体1~2HRC的耐磨块并加工规范油槽 |
| **抽芯不到位** | ①抽芯计算时未预留3~5mm以上的安全附加距离；②滑块抽芯完成后无可靠限位装置，受自重或振动移位 | ①抽芯总距离=侧孔深度+3~5mm安全值；②在滑块末端配置弹簧+限位杆组合的限位机构保证停位准确 |
| **与推杆干涉** | ①合模时推杆复位先于滑块复位；②无预复位机构 | 增设液压先复位机构，保证推杆在斜销驱动滑块复位之前完全退至型腔投影面之外 |
| **楔紧块磨损** | ①压射时滑块受金属液反压力向后退让反复冲击楔紧面；②楔紧块斜角小于斜销倾角导致面接触应力过大 | ①楔紧块斜角比斜销倾角大2°~3°以提升承载能力；②楔紧块选用淬硬到45~50HRC的模具钢并做可拆卸设计便于磨损后更换 |

来源[[S1,S24,S27,S31,S43,S14,S33,S47]]。

此外，二次分型斜销抽芯复杂薄壁压铸模通常还需注意以下设计要点：需设置定距拉紧机构保证两次分型动作顺序同步，斜销安装孔与滑块斜孔必须保证与导滑面垂直以避免蹩劲卡滞[[S1,S24]]。

## 应用实例

斜销抽芯机构可适配压铸件侧部常规侧孔、异形侧凹、U形薄板长圆孔等各类侧向特征的成型抽芯需求，适用于抽芯距离较短、抽芯力不大的中小型抽芯场景[[S14,S43]]。

**实例一：三斜销侧向分型抽芯不推件压铸模**[[S21,S16]]

该模具用于卧式冷室压铸机生产异形弯头类薄壁压铸件（材料ZL7）。通过两个对称布置的斜销驱动对合滑块完成侧分型，另一组斜销驱动侧型芯抽芯。铸件和余料依靠横浇道梯形槽约束留在动模侧，抽芯完成后直接取出铸件，无需推出机构，避免了推出动作损伤薄壁铸件，生产运行稳定[[S21]]。

**实例二：ZL104望远镜柄压铸件斜销延时抽芯压铸模**[[S3]]

该铸件内外形均为锥形，偏心12°位置设有侧孔。冷却后定模型芯包紧力远大于动模型芯，若采用常规斜销抽芯，铸件将滞留定模。设计采用延时斜销抽芯机构，将滑块斜孔延时间隙δ按公式 δ=S×sinα 计算，设定延时距离S=4mm。开模后先保证铸件脱离定模留在动模侧，之后斜销才驱动滑块完成侧抽芯，解决了铸件留定模的生产难题[[S3]]。

**实例三：汽车油泵支架压铸件斜销侧抽芯压铸模**[[S45]]

抽芯距离设定为27mm（侧孔深度23.5mm+安全值3~5mm），斜销倾角选用15°，楔紧块采用高可靠易维护的安装结构。通过UG软件完成运动仿真验证动作全程无干涉，模具投产后抽芯动作运行稳定，铸件侧孔成型精度满足IT12级公差要求[[S45]]。

此外，斜销抽芯机构在二次分型复杂薄壁压铸模中已有成熟的落地应用。针对开模时固定模型芯包紧力大于动模的特殊薄壁铸件（如电刷端盖类零件），可通过设置滑块斜孔延时间隙实现延时抽芯，保证开模后铸件留在动模侧，配合中心浇口成型复杂薄壁环形端盖类零件[[S21,S3]]。
