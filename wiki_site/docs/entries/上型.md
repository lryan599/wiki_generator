## 定义与概述

在压铸技术标准体系中，“上型”一词对应模具总成中固定的那一半压铸模。根据 GB/T 8847-2003《压铸模术语》及 GB/T 8845-2017《模具 术语》，该部件官方定义为**定模**，即“固定在压铸机定模安装板上的模具部分”[[S31,S33]]。与之对应，随压铸机动模安装板作开合移动的另一半模称为**动模**。

> **术语两个维度**
>
> GB/T 8845-2017 区隔了两套术语维度：“上模/下模”与“定模/动模”。当压铸设备工作台面为水平布置，且固定侧位于上方时，这两个维度所述的部件完全一致。鉴于行业交流惯例，上型在日常语境下与定模互为通用[[S33]]。

在不同压铸机构型下，“上型”的具体指代关系如下表所示[[S55,S46,S20]]：

| 压铸机类型 | 开合模方向 | 上型指代对象 | 备注 |
| :--- | :--- | :--- | :--- |
| 全立式冷室压铸机 | 垂直方向 | 固定侧半模（上方） | 模具水平安装，直接沿用上/下概念 |
| 立式冷室压铸机 | 垂直方向 | 固定侧半模（定模） | |
| 卧式冷室/热室压铸机 | 水平方向 | 固定侧半模（定模） | 行业惯例，将“上型”延伸代指“定模” |

国际压铸行业该部件的通用英文术语包括：**cover die half**、**fixed die half**、**stationary die half**，行业中同样使用 **upper die half** 和 **top mold** 等称呼[[S18,S49]]。

需要注意的是，砂型铸造体系中具有历史渊源的“上箱”（cope）一词，专指砂铸造型工序中处于上层的砂型半型；压铸标准体系未将该词列为正式术语，仅在部分跨领域交流场合出现借用现象[[S44,S8,S10]]。

## 结构组成

上型位于压铸模的定模侧，是整个模具总成中核心的“静止”半模，其结构组成涵盖型腔成型、导向定位、浇注衔接、冷却温控及排溢等多个功能系统[[S45,S24]]。

![图：压铸模结构示意图，标注了上型（定模侧）各组成部件在模具总成中的定位关系](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/bf4de291-fac8-4390-a48e-5cb4f565eb14/resource)

上型系统包含以下典型内置组件[[S2,S7,S24]]：

- **定模座板**：与压铸机定模板固定的基础板件；
- **定模套板**：固定定模镶块并保证其强度与刚度的框式板件；
- **定模镶块**：直接构成压铸件型腔表面的核心成型零件；
- **浇口套与直浇道段**：衔接压射系统并将金属液导入模具的关键通道；
- **横浇道（对应部分）**：在定模分型面上延长金属流路的通道结构；
- **导柱/导套组件**：配合动模侧的对应组件，保证开合模动作精确导向；
- **冷却/加热循环管道**：控制上型热平衡的温控管道系统；
- **排气槽与溢流槽（分型面侧）**：排出型腔气体、容纳冷却金属的排溢结构；
- **分流锥**：部分机型配置，正对浇口套中心，调整金属流束方向（立式和部分热室配置）。

## 材料与热处理要求

上型成型零件（定模镶块、型芯及浇注系统零件）工作于高温、高压、高速液流冲刷的严苛环境，标准材料基本以热作模具钢为主线[[S19]]。

### 常用材料牌号

当前压铸模具行业采用的常用热作模具钢牌号如下表所示[[S19,S36,S54]]：

| 钢种/来源 | 牌号示例 |
| :--- | :--- |
| 国产系列 | 4Cr5MoSiV1（对应H13）、4Cr5MoSiV |
| 日本标准（JIS） | SKD61 |
| 日本厂商牌号 | DAC、DH31-S |
| 瑞典厂商牌号 | 8407 |
| 美国牌号 | H13 |

### 不同合金对应的硬度要求

针对不同压铸合金的工况差异，上型成型零件的推荐硬度亦不相同[[S19,S36,S17,S13]]：

| 适用压铸合金 | 推荐硬度（HRC） | 说明 |
| :--- | :--- | :--- |
| 锌合金 | 48 ~ 50 | 浇注温度低，模具可取较高硬度 |
| 铝、镁合金 | 43 ~ 47 | 兼顾热疲劳抗力与韧性 |
| 铜合金 | 不超过 42 | 工作温度极高，需降低硬度以防止开裂 |

### 热处理规范

上型成型零件必须采用合理热处理以保障服役性能：淬火后须进行 2~3 次回火以避免后期开裂[[S36,S19]]。对于锌、铝、镁合金压铸用上型，若实施表面软氮化处理，渗氮层深度应达到 **0.08~0.15 mm**，表面硬度 **≥600 HV**，该要求符合 GB/T 4679—2003 的规定[[S41,S7,S19]]。

## 设计要点

### 装配精度

上型的总装精度须符合 GB/T 8844—2003 的要求[[S18,S51,S26]]：

- 合模后分型面应紧密贴合，允许局部间隙**不大于 0.05 mm**（排气槽区域除外）；
- 定模镶块平面与定模模板齐平，允许高出量**不大于 0.05 mm**；
- 导柱/导套对上型安装基准面的垂直度需依标准公差表控制，以确保与下型（动模）的合模导向精度。

### 脱模斜度

上型成型壁面须设置合理脱模斜度[[S41,S7]]：

- 形成铸件内侧壁的脱模斜度不大于国家标准表列值；
- 构成铸件外侧壁的脱模斜度取内侧壁取值的 **1/2**；
- 圆形型芯脱模斜度依标准规定取值；
- 文字符号的脱模斜度取 **10°~15°**；
- 未注明方向时，按减小铸件壁厚方向制造。

### 冷却水道布局

上型冷却水道布局对于模具热平衡与铸件质量至关重要。合理排布的冷却水道能够降低上型局部温差，避免陡峭热梯度引发的热疲劳开裂；同时促进铸件实现顺序凝固，有效减少缩松缩孔缺陷[[S35,S42,S25]]。

## 工艺角色

### 工作温度与热场特征

行业依据经验公式计算上型工作温度[[S48]]：

$$
T_m=\frac{T_j}{3}\pm25\,^\circ\mathrm{C}
$$

其中，\(T_m\) 为上型工作温度， \(T_j\) 为液态金属的浇注温度。

相较于动模，上型由于型腔结构更复杂、与高温金属液接触面积更大，整体温度梯度显著偏高。数值模拟显示，充型保压阶段上型局部最高温度可达 **550 ℃**；凸台、热节等厚壁区域冷却速率远低于周边，极易形成局部高应力集中区[[S22]]。

### 充型压力承载

上型在压射阶段直面高速、高压冲入的液流，是承受金属液冲蚀载荷的主要半模。其所分配的单位承压力须与压铸机总锁模力匹配，有效抵消充型过程中产生的胀型力[[S23,S27]]。

### 热平衡管理

上型须配置冷却-加热复合温控系统，通过调节冷却介质的流量、压力和温度来匹配压铸循环各阶段的散热需求，将上型整体温度波动控制在合理区间，避免局部过热或过冷造成的质量与寿命问题[[S5]]。

## 常见缺陷关联

上型侧产生并反映于铸件的典型缺陷可归纳如下[[S47,S32,S53,S30]]：

| 缺陷类型 | 国标定义简述 | 易发区域（上型侧） | 主要原因 | 优化方向 |
| :--- | :--- | :--- | :--- | :--- |
| **冷隔** | 铸件上穿透或不穿透的边缘圆角状缝隙 | 远离浇口的上型宽大表面、多股汇流处 | 上型温度过低；浇排系统设计不良 | 提高上型预热温度；优化溢流与排气结构 |
| **缩松/缩孔** | 缓慢凝固区产生的细小弥散孔洞 | 上型厚壁热节区域 | 冷却水道排布不合理，局部无法补缩 | 增设局部点冷结构 |
| **粘模** | 金属液粘附上型表面，开模时无法脱离 | 高速金属液正面冲刷区域 | 局部温度过高；脱模剂层被破坏 | 降低局部充型流速；增强该区冷却强度 |
| **表面龟裂（龟纹）** | 型腔表面热疲劳产生的微细网状裂纹，复制于铸件 | 上型型腔表面 | 反复急冷急热交变应力超过材限 | 定期打磨去除微裂纹层；控制上型预热均匀性 |

## 维护与寿命

### 典型失效形式

铝合金、铜合金压铸用上型的主要失效模式为**热疲劳龟裂、金属冲蚀/冲刷、整体开裂及烧付（粘模）**四类；锌合金压铸用上型则以**侵蚀与磨损**为主[[S3,S13]]。

热疲劳龟裂的形成机理概为：上型型腔表面接触高温金属液时急速升温产生压应力，开模后受脱模剂和空气激冷产生拉应力，长期往复的交变热-机械作用导致微裂纹萌生并逐步扩展[[S34,S56]]。

### 日常维护

- 分别在压射 **500 次、5000 次、25000 次**三个节点对上型进行去应力退火处理[[S39]]；
- 型腔表面一旦出现微裂纹，应及时打磨去除全部裂纹层[[S32]]；
- 保证上型整体预热均匀，避免局部温差过大加重热冲击[[S39,S32]]。

### 寿命提升

- 在受高速金属液正面冲刷的易损区域采用**局部镶拼结构**，损坏后单独更换而不致整体报废[[S3,S21]]；
- 选用经合理锻造与热处理的 H13 等优质热作模具钢，并辅以离子氮化等表面强化工艺，可大幅提升上型服役性能[[S21,S3]]。

行业经验表明，常规铝合金压铸用上型平均寿命约 **5 万次**，锌合金用上型可达 **百万次**以上[[S21]]。

## 与其它部件的配合关系

### 与浇口套的配合

浇口套以**过盈配合**方式嵌入上型的对应安装孔；径向配合公差选用 **H7/h6** 系列以控制同轴度精度；直浇道段拔模斜度取 **1.5°~2°**，保障金属液流动顺畅且余料易于脱出[[S50,S52]]。

### 与分流锥的配合

分流锥正对直浇道中心，在上型侧定位安装。分流锥凹腔深度与浇道厚度相等，二者同轴度误差须控制在 **0.05 mm 以内**。热室压铸机配套的上型浇口套和分流锥内部需预留冷却水道[[S29,S37]]。

### 与侧向抽芯机构的配合

安装于上型侧的侧向抽芯机构（如液压抽芯、齿轴齿条抽芯组件）须保证合模状态下楔紧块对滑块的锁紧刚度足以承受充型胀型力，配合间隙每边控制在 **0.25~0.3 mm**，防止抽芯动作发生干涉[[S15,S4]]。

### 与溢流排气系统的配合

在上型分型面布置的排气槽深度取 **0.05~0.2 mm**，总截面积须达内浇口截面积的 **20%~50%**。溢流槽布置于金属流末端汇合到达的上型远端区域，与排气槽串联排布，兼具集渣与排气双重功能[[S29,S50]]。
