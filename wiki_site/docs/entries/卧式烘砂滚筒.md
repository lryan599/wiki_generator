## 概述

卧式烘砂滚筒（英文标准术语为 *rotary sand drier*）是铸造砂处理系统中用于新砂及再生砂烘干的核心设备，在国家标准 GB/T 25370-2020《铸造机械 术语》中归入编号 3.1.2，属于新砂烘干设备，官方英文对应词还有 *revolving drier* [[S13,S18,S28]]。该设备滚筒轴线水平布置，整体沿砂料前进方向向下倾斜安装，主要用于对含水率较高的原砂和循环回用旧砂进行连续烘干处理，为后续混砂、制芯工序提供水分合格的干砂 [[S4,S7,S28]]。

## 工艺作用

在压铸配套砂处理流程中，卧式烘砂滚筒位于混砂工序之前。用水玻璃砂、树脂砂、油砂等工艺制芯时，为保证砂芯质量，新砂必须经过烘干 [[S7]]。对于湿法清理铸件后含水量大幅升高的回用旧砂，同样需要先经该设备烘干才可重新投入使用 [[S5,S7,S17]]。

型砂水分控制不达标对铸件质量有直接影响。型砂水分过多时，浇注过程中水分受热大量汽化析出，容易导致铸件产生气孔缺陷；同时砂粒间粘结力下降，金属液充型时冲刷脱落的砂粒进入型腔，会直接造成砂眼缺陷。湿型砂水分质量分数超过 4.5%~5% 时，此类缺陷发生率显著上升 [[S8,S2,S6]]。

## 结构与组成

卧式烘砂滚筒主要由以下核心部件组成：滚筒体、传动机构、燃烧炉/加热系统、进砂口、出砂口和除尘口 [[S10]]。

### 滚筒体

主流三回程机型在结构上具有显著特点，其滚筒分为内、中、外三层，通过 20 块肋板相互隔开，形成蜂窝状分布的狭长换热通道。这种多层嵌套结构大幅增加了热交换接触面积，使设备整体结构紧凑、占地面积小 [[S10]]。

筒体内部设置有扬料板（举升板），其作用是在筒体转动时反复抄起并抛落砂料，使砂料在筒体横截面上均匀分布，从而大幅提升砂料与热气流的接触换热效率 [[S15,S27,S23]]。

### 传动系统

传动系统包含托轮支撑部件、传动齿轮、减速箱和驱动电机 [[S1,S19]]。运行时要求传动齿轮啮合侧隙符合 GB 10095 的 8 级精度要求；同一滚轮轴上两滚轮的直径差不得大于 0.1 mm；运行过程中滚筒不得与炉体、进出料漏斗发生碰擦 [[S1,S19]]。

![三回程卧式烘砂滚筒横截面结构剖视图，展示三层嵌套筒体、内部扬料板结构以及底部托轮支撑的径向布置](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f0392e5b-20f3-4675-a478-e9efcf199b97/resource)

### 加热系统

卧式烘砂滚筒可适配多种加热方式，包括燃煤、燃气、燃油直接燃烧产生高温热气流的直接加热模式，以及电加热元件间接加热热风的模式，可根据铸造车间燃料供应条件灵活选配。烘干过程热质传递以强制对流换热为核心 [[S27,S16,S29,S24]]。

## 工作原理

卧式烘砂滚筒整体沿砂料前进方向下倾 5°~8° 安装，筒体转速为 0.5~8 r/min。砂料依靠重力沿轴向从进料端缓慢向出料端移动，物料填充量通常不超过滚筒总容积的 20%。通过调整倾斜角度可控制砂料的移动速度 [[S17,S5,S15]]。

工作时湿砂从进料端加入，随滚筒转动，内部扬料板将砂料反复抄起抛落，同时高温热气流进入筒体与砂料进行强制对流换热。三回程机型中，热气流与湿砂在滚筒中完成三次折返顺流热交换，砂料充分换热后从出料口排出，携带水汽的废气经除尘系统过滤后排出车间。三回程机型热效率较高，但烘干后的新砂温度较高，需搁置降温后方可使用 [[S7,S21,S14]]。

| 安装倾角（下倾） | 5°~8° |
| 热风进口温度 | 800~850 ℃ |
| 可烘干湿砂最高含水率 | 20% |
| 入料含水率 6% 时生产率 | 7~26 t/h |
| 典型烘干效果 | 含水率从 15% 降至 1% |
| 物料填充率 | 不超过滚筒总容积的 20% |
| 工作转速 | 0.5~8 r/min（通用取值范围，暂无行业统一标定数值）[[S17,S5,S19,S1]] |

## 限制与常见问题

卧式烘砂滚筒在运行中可能遇到以下问题 [[S14,S1,S10,S19]]：

1. **出料砂温偏高**：热效率虽然较高，但三回程机型烘干后的新砂温度偏高，必须额外设置冷却工序或静置降温后才可进入下一工序。
2. **传动系统精度要求高**：托轮与滚筒圈之间的同轴度、接触间隙有严格公差要求——两滚筒圈同轴度不得大于 1.2 mm，滚轮与滚筒圈最大接触间隙不得大于 0.2 mm，否则会引发传动异响和加速磨损。
3. **扬料板磨损**：扬料板长期受砂粒冲刷，磨损量大，定期更换维护工作量大。
4. **砂粒过破碎风险**：温度控制系统精度不足、热风温度过高或砂料在筒内停留时间过长时，会导致砂粒过破碎，破坏原砂粒度级配。
5. **进料管烧损**：进料管长期处于高温受热工况，易烧损，需定期检查修补或更换。
6. **筒体粘砂结壁**：筒体内壁局部温度过低或砂料初始含水率过高时，易出现粘砂结壁现象，降低热交换效率，增加设备能耗。
7. **设备投资较高**：投资约为热气流烘砂装置的 6~7 倍 [[S4,S5]]。

## 标准与规范

国内针对卧式烘砂滚筒的主要标准为行业标准 **JB 5551—91《三回程滚筒烘砂装置》**，该标准对设备的技术要求和质量检验做出了以下明确规定 [[S1,S19]]：

| 标准要求项目 | 具体指标 |
|---|---|
| 传动齿轮啮合侧隙 | GB 10095 附录 C 中表 C2 的 8 级精度 |
| 减速箱接合面接触面积 | ≥70% |
| 同一滚轮轴两滚轮直径差 | ≤0.1 mm |
| 两滚筒圈同轴度 | ≤1.2 mm |
| 两滚筒圈直径差 | ≤0.8 mm |
| 滚轮与滚筒圈最大接触间隙 | ≤0.2 mm |
| 空运转噪声声压级 | ≤82 dB(A) |
| 减速箱内部清洁度 | ≤5000 mg |
| 轴承温升 | ≤35 ℃ |
| 轴承最高工作温度 | ≤70 ℃ |
| 安全防护 | 符合 JB 5545 规定 |
| 首次大修期工作时间 | ≥12000 h |

在术语标准层面，GB/T 25370-2020《铸造机械 术语》将回转式烘砂设备纳入铸造机械通用术语体系，卧式烘砂滚筒为该术语对应的典型砂处理烘干设备 [[S18,S3]]。

国内铸造车间通风设计规范要求，卧式烘砂滚筒等原砂烘干类设备必须配套设置除尘通风系统，同时需采取针对性措施防止除尘管路结露、滤袋粘堵 [[S25]]。

经检索，目前国际标准化组织（ISO）尚未发布专门针对卧式烘砂滚筒/回转式铸造砂烘干设备的独立正式标准，通用工业回转式干燥器的性能准则不对铸造砂烘干专属场景做出专项限定 [[S23,S27]]。
