## 定义

清理整修装置是压铸单元中去除铸件浇注系统、溢流系统、毛刺并对铸件进行整形等工艺的辅助设备[[S14,S27]]。它属于压铸自动化生产成套装备的组成部分，在行业实践中也被称为压铸件清理设备、去飞边装置、清整机、铸件整修机，对应的英文术语包括 *cleaning and finishing machine*、*fettling machine*、*casting trimming and cleaning machine*、*deburring machine for die castings* 等[[S14]]。

## 工艺作用

脱模后的压铸件必须经过清理整修才能进入后续加工或入库，其主要作业内容涵盖以下方面[[S6,S13,S12]]：

- 去除残留的脱模剂及表面流痕；
- 切除浇注系统（浇口）和排溢系统（溢流槽、排气道）的余料；
- 去除分型面及顶杆部位产生的飞边、毛刺；
- 对铸件表面进行初步修整，为后续加工提供基准；
- 对变形铸件进行整形校正。

## 分类与典型设备

清理整修装置可按工作原理和专用设备类型进行划分。

### 按工作原理分类

依据作业机理，清理整修装置一般可分为五大类[[S48,S55,S15,S20,S21]]：

| 类别 | 工作原理 | 典型适用场景 |
|------|----------|--------------|
| 机械式滚磨/振动式 | 通过滚筒旋转或振动使铸件与磨料相互摩擦，去除毛刺并光饰表面 | 中小批量结构简单的小件，中等尺寸压铸件的批量去毛刺与光整 |
| 喷丸/喷砂式 | 利用压缩空气或离心力将弹丸/磨料高速喷射到铸件表面，进行清理和强化 | 压铸件整体表面强化、去除氧化皮与残留杂质，可获得光亮细致的表面 |
| 化学清洗式 | 在碱性或酸性溶液中通过化学反应去除表面残留杂质 | 精度要求高、几何结构复杂的压铸件，难以用机械方式清理时 |
| 高压水射流式 | 利用高压水流的冲刷和切割作用清除表面附着物和深腔内飞边 | 铝合金等有色金属压铸件清理，无粉尘污染，环保性好 |
| 超声波式 | 通过高频振动产生的空化效应剥离油污和细微毛刺 | 薄壁精密压铸件的高精度清理，无外力冲击本体 |

### 按专用设备形式分类

在压铸生产中，还常按具体设备形式选用清理装置。表 5-161 列举了传统压铸件清理设备及其应用[[S25,S49]]：

| 设备名称 | 使用方法要点 | 应用说明 |
|----------|--------------|----------|
| 压床或冲床 | 利用与铸件外形相似的模具，一次清除浇口、溢流槽、排气道及飞边、毛刺 | 大批生产，效率高 |
| 锯床（带锯）、车床、铣床 | 切除中心浇口或内浇口较厚的浇注系统 | 成批或大批，效率较高 |
| 清理或抛光滚筒 | 内衬橡皮或硬木板，加入钢球或皮块、木屑等磨料，干磨或湿磨；转速10~30 r/min | 形状简单的小型铸件，对质量要求不高时使用 |
| 砂轮、砂带 | 利用油或油水混合液润滑，主要打磨浇口毛刺 | 应用广、效率高，但工作条件较差 |
| 钻床 | 清理深腔、孔部位的毛刺和披缝，夹具与钻头须同心，转速1130~1890 r/min | 针对特定部位 |
| 手工 | 使用锉刀等简单工具 | 小型或浇口较薄的铸件，劳动强度大，仅用于单件小批生产 |

在现代化大批量生产中，更多采用振动埋入式清理装置、螺壳式振动清理机、抛丸机、高压水射流装置等高效设备[[S36,S48,S13,S12]]。

## 结构与组成

不同类型的清理整修装置在结构上差异明显，以下择要说明。

### 机械振动与滚磨类

常用的螺旋振动研磨机由电动机、弹簧、螺壳、栅格、橡胶衬等组成，利用螺旋流三元振动原理实现零件与磨料的相对运动[[S13]]。振动光饰机的磨料循环装置包括沉淀箱、水箱、泵、管道、软管和阀门，可及时排除金属和磨料粉末，保持磨料锋利和加工液清洁[[S41,S38]]。

普通滚筒清理机的滚筒转速为10~65 r/min，内部装有螺旋导筋或橡胶衬层，带动铸件翻转进行光整加工；离心滚光机转塔可产生0.98 N的离心力，加工效率可达普通振动光饰的1/50，并能在零件表层产生残余压应力[[S43,S39]]。

### 喷丸/喷砂类

喷砂机分为吸入式干喷砂机、压入式干喷砂机和液体喷砂机。吸入式干喷砂机由结构系统、介质动力系统、管路系统、除尘系统、控制系统和辅助系统六部分组成；压入式干喷砂机由压力罐、介质动力系统、管路系统和控制系统四部分组成；液体喷砂机由结构系统、介质动力系统、管路系统、控制系统和辅助系统五部分组成，可有效抑制粉尘[[S16,S57]]。环保自动循环回收式喷砂机进一步配置了砂尘分离器和滤桶除尘器，实现磨料循环利用[[S46]]。

抛丸机核心结构分为四大模块：①抛丸器（抛头），由双盘铆接叶轮、8片辐射状叶片、分丸轮、定向套和进丸管构成，叶轮常规转速2250~2700 r/min，弹丸抛射速度可达70~171 m/s；②弹丸收集、分离和运输循环系统；③工件承载体，根据铸件特点可选择滚筒式、履带式、吊钩式或悬链步进式等结构；④除尘系统[[S35,S52,S9]]。

### 高压水射流类

高压水射流清洗装置主要由高压柱塞泵、动力单元（柴油机或电动机）、高压软管、喷嘴及工作附件、密闭清洗室、工件夹持旋转机构和水循环沉淀过滤系统组成[[S5,S2,S42]]。水经高压柱塞泵增压后通过管路送至喷嘴，形成高速射流，实现对铸件表面附着物的切割和冲刷。

## 关键工艺参数

各类清理整修装置的主要工艺参数汇总如下。

### 机械振动与滚磨

| 参数 | 常规范围 | 说明 |
|------|----------|------|
| 振动频率 | 15~50 Hz，常用20~30 Hz | 影响切削效率与表面质量[[S17,S29,S22]] |
| 振动振幅 | 2~20 mm，常用3~6 mm | 过大会损伤零件，过小则效率低[[S17,S29,S22]] |
| 磨料与零件体积比 | 3:1~6:1 | 粗加工用较小比，精饰用较大比[[S22,S39]] |
| 磨料粒度 | 小于零件内孔直径的1/3 | 防止磨料卡入孔槽[[S39]] |
| 振动研磨机工作频率 | 1700~2100 次/分钟 | 对应振动式清理的典型参数[[S21]] |
| 振幅 | 0.8~6.4 mm（对应1700~2100 cpm） | [[S21]] |
| 处理时间 | 1~4 h（振动光饰） | 视毛刺量和表面要求而定[[S21]] |
| 可达表面粗糙度 | 0.15~0.25 μm（振动光饰，塑料磨料） | 更细磨料可获得0.075~0.125 μm[[S21]] |
| 滚筒转速 | 10~65 r/min | 滚光用20~30 r/min，清理表面时10~15 r/min[[S25,S56]] |

|------|---------------|------|
| 工作压力（干喷砂） | 0.4~0.7 MPa，最高限值不超过0.8 MPa | 压力过低不适宜喷砂除锈[[S32,S51]] |

### 化学清洗

- 弱碱性清洗液 pH 值应控制在8~9，铝合金清洗时需添加防锈成膜添加剂，防止磨料微粒嵌入表面引起腐蚀[[S41]]。

## 在压铸工艺流程中的位置

清理整修装置是压铸件脱模冷却后的**首道后处理工序**设备。其上游衔接压铸机取件与铸件冷却工序；经清理整修去除浇口余料和飞边后，铸件依次进入整形校正、浸渗处理、必要时的时效/负温时效热处理、最终表面涂装和精密机加工等工序[[S28,S30,S4]]。在压铸车间布局中，清理整修装置与清理工部紧密结合，典型自动化压铸单元平面布置见图示[[S27,S53]]。

## 适用合金与零件范围

清理整修装置可覆盖主流的压铸非铁合金，包括铝合金、锌合金、镁合金、铜合金等[[S10,S48]]。在零件尺寸方面，可处理的压铸件外部尺寸从几毫米到1~2 m，重量从几克到数十千克，国外已能支持直径达2 m、质量为50 kg的铝合金铸件的清理[[S55]]。

## 技术标准与安全环保要求

### 设备通用技术要求

依据 GB/T 39957-2021《压铸单元 技术条件》，清理整修装置须满足以下四项强制性要求[[S47,S54]]：

- 不应对压铸件本体造成损伤；
- 装置的循环周期应小于压铸机的循环周期；
- 应具有碎屑回收功能；
- 装置下方应配有废料框。

### 安全操作规范

- 手工清理时，锉刀必须装有牢固的手柄，禁止将锉刀作为榔头、撬棒或冲头使用；使用砂带时应使铸件逐步接触，不可猛撞，避免砂带断裂[[S1,S50]]。
- 抛丸清理操作者须佩戴有机玻璃面罩、口罩及防滑工作鞋；非工作人员不得靠近抛丸室[[S1,S50]]。
- 抛丸设备在防护帘破坏、外壳被打穿、供丸系统堵塞、运转不平稳或通风除尘装置失效时禁止运行[[S50]]。

### 环保与职业健康

清理工部中砂轮机、砂带机、抛丸机等设备作业时产生大量粉尘，必须配备正常运行并确保车间粉尘含量符合国家标准的吸尘除尘装置[[S44,S11]]。镁合金清理区域需额外强化通风，严禁使用明火，以防镁合金粉尘引发爆炸和火灾[[S3,S44]]。我国已等同采用ISO14000系列环境管理标准，压铸企业的清理工序应纳入体系管控，建立相关制度文件并定期评审[[S3,S18]]。
