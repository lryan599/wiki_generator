## 定义

滚筒筛（rotary screen / drum screen），在铸造领域亦称转筒筛、回转筛、滚筒筛分机，属于筛砂机品类[[S30]]。GB/T 25370-2020《铸造机械 术语》将其定义为“筛体采用由筛网构成的滚筒，通过旋转方式工作的筛砂机”，标准编码3.2.9，归类于旧砂处理设备大类[[S30,S13]]。兼有砂块破碎功能的衍生型号被定义为滚筒破碎筛（drum breaker screen）[[S30]]。

## 分类

铸造领域滚筒筛可按多个维度划分类型[[S29,S14]]：

- **按截面形状**：圆柱形滚筒筛、多角形（多为六角形）滚筒筛；
- **按轴向轮廓**：等径圆柱形滚筒筛、大端进料小端出料的角锥形滚筒筛；
- **按支撑结构**：托轮支撑式滚筒筛（筛体通过滚道与托轮承载）、中心贯穿轴带辐条支撑式滚筒筛；
- **按功能类型**：普通筛分型滚筒筛、滚筒破碎筛（兼有砂团破碎功能）；
- **按筛面层数**：单层单级筛分滚筒筛、双层双级分级滚筒筛（设内筛筒与外筛筒）。

国产标准化系列包含S41、S42、S42C、S43精细六角筛、S44破碎筛以及120t/h双层筛砂机等机型[[S3,S1]]。

## 结构与组成

典型铸造用滚筒筛由以下核心部件构成[[S1,S3,S11]]：

**滚筒体与筛网**
- S42C系列采用特种橡胶筛板，不粘砂、使用寿命长[[S3]]；
- S43系列六角筛采用长孔冷拔不锈钢编织筛网，网面六角相互拉紧固定，防折损且便于更换[[S6]]；
- 双层筛分机型设内筛筒（筛孔通常为φ30 mm）和外筛筒（网孔尺寸12.5 mm×12.5 mm）[[S1,S11]]。

**驱动装置**
- 多采用轴装式减速器加电机直联驱动；部分机型采用齿轮传动[[S3,S6]]。

**支撑结构**
- 托轮支撑式机型设置滚道与托轮承载筛体重量[[S3]]；
- 中心轴式机型通过贯穿轴加辐条支撑筛筒[[S14]]。

**进出料与除尘系统**
- 配置进料口、合格砂出料口、筛余杂物/废砂出口，两端设标准吸尘口直接对接除尘管路[[S1,S6]]。

**内部功能结构**
- 筛筒内设置螺旋输送叶片和提升叶片，实现物料推进与提升抛落[[S1,S11]]；
- 部分带风选功能的机型设吹清装置，用于清理筛面黏附的湿砂[[S6]]。

## 工作原理

待筛分的落砂后旧砂从进料口进入倾斜布置（轴线与水平面夹角约5°）的筛筒内部[[S14]]。筛筒低速旋转带动内置轴向螺旋将物料向出料端推送，同时轴向分布的提升板将物料抬升至一定高度后自由跌落，该过程同步完成砂块破碎与物料松散[[S1,S11]]。小于筛孔的合格砂粒穿过筛网完成筛分，大颗粒砂团、未破碎杂物向出料端移动并由废砂口排出[[S1,S14]]。双层机型则先由内筛筒筛除超大尺寸杂物，再由外筛筒完成精细筛分，合格砂粒由外层筛网导出[[S1]]。

滚筒筛除基础筛分功能外，还附带砂块破碎、通风冷却旧砂的附加作用[[S3]]。配套通风系统的抽风气流穿过旋转筛筒，可同时完成除尘和旧砂预冷却，降低后续砂冷却设备的热负荷[[S1,S6]]。

## 与振动筛/直线筛的对比

铸造车间常用筛分设备分为滚筒筛与振动筛两大类[[S3]]。两者核心性能差异如下[[S1,S19,S7,S3]]：

| 对比项 | 滚筒筛 | 振动筛/直线筛 |
|---|---|---|
| 运行平稳性 | 优良，低速旋转 | 需激振器高频振动 |
| 密封性 | 好，可全封闭罩壳 | 较差，粉尘溢出风险高 |
| 过载能力 | 强 | 一般 |
| 附加功能 | 可同步完成砂团破碎、旧砂冷却、降低含泥量 | 无附加功能 |
| 过筛效率 | 低于振动筛 | 高，筛孔自清洁能力强 |
| 结构体积 | 体积庞大，占地大 | 结构紧凑，占地小 |
| 安装检修 | 难度较高 | 较便捷 |
| 高水分/高黏附物料适配性 | 较好 | 差，易堵塞甚至停机 |
| 运行噪声 | 较低 | 较高 |

由于滚筒筛工作时比较平稳，密封性好，过载能力大，且具有破碎和冷却功能，因此大量流水生产的铸造车间多数采用滚筒筛[[S3]]。

## 应用场景

滚筒筛在压铸/铸造砂处理及铸件清理环节的典型应用包括四类[[S22,S2,S16,S1]]：

1. **落砂后分离**：布置于振动落砂机下游，分离落砂后成型铸件与残留型砂；
2. **抛丸丸砂分离**：在抛丸机丸砂循环系统中筛分抛丸后混合物料，将铸件与弹丸、残留砂分离，同时向磁分离滚筒均匀布料；
3. **旧砂再生前预处理**：破碎筛分磁选后未完全溃散的砂团，筛除大尺寸芯头与杂物，得到粒度均匀的合格待再生旧砂；
4. **砂芯去除配套**：滚筒滚动作业的摩擦作用可辅助震落铸件内腔残留砂芯，同时筛除脱落的砂芯碎块。

## 上下游设备关系

**上游配置**：滚筒筛直接衔接落砂机、斗式提升机下游，前端常配套永磁磁选机预先筛除混杂在旧砂中的铁质杂物，避免大块金属进入筛筒造成筛网破损[[S16,S8,S15]]。

**下游配置**：筛分后的合格物料直接输送至砂冷却器、旧砂再生设备前端；粗杂物从废料口排出后送入废料收集系统[[S15]]。

**标准配套联动设备**：振动给料机、螺旋输送机、斗式提升机、永磁滚筒磁选机、除尘系统[[S1,S6,S28]]。

## 局限性与常见缺陷

**过筛效率**：被筛物料沿筛网轴向平行运动，未沿垂直方向穿透筛网的最优路径运动，导致整体过筛效率低于同工况配置的振动筛[[S3]]。

**筛网堵塞与磨损**：高湿度、高黏性旧砂工况下筛孔易被黏附砂团堵塞，其中双层滚筒筛的内外筛筒夹层区域堵塞风险显著高于普通单层滚筒筛；金属编织筛网长期受砂粒摩擦磨损严重[[S1]]。

**体积与安装**：设备整机体积偏大，进出料口高度差大，现场安装与检修难度高于常规筛分设备[[S3]]。

**筛分精度下降**：当工况超出额定砂料含水率阈值、喂料量过载时，大量符合粒度要求的合格砂料随筛余物排出[[S4,S9]]。

**噪音与粉尘**：滚筒筛为低速旋转结构，运行噪音显著低于共振式工作的振动筛，但仍需配置全封闭罩壳并接入两级除尘系统，配套除尘管道需做保温防结露处理，避免高湿含尘气流导致粉尘结块堵塞管路[[S12,S23]]。

## 改善措施

面向铸造场景的滚筒筛行业通用改善措施[[S10,S1,S6]]：

- 筛网改用长孔不锈钢编织网或特种橡胶筛板，提升抗磨损、防黏附性能；
- 配置强力抽风系统，同步实现砂料冷却、降低含水率，减少筛孔堵塞；
- 建立例行维护制度，定期停机清理筛面附着杂物，及时更换磨损筛板；
- 选用大端进料小端出料的倾斜主轴布置结构，提升物料行进速度适配高喂料量生产场景。

## 相关标准与检验要求

**ZB J61011-89 滚筒式筛砂机技术条件**[[S4]]：

- 空载运行：滚筒旋转需灵活平稳，无冲击碰擦；圆筒筛轴心游动偏差不得大于5 mm；
- 滚轮与滚圈接触面积：涂色法检验不得低于70%；
- 负载试验：必须检测轴承温升、整机噪声、电机功率、过筛效率、生产率五项核心指标；
- 轴承最高允许工作温度：不超过75℃；滑动轴承温升不超过25℃；滚动轴承温升不超过35℃；
- 额定过筛效率：不低于95%。

**JB/T 9156-1999《铸造用试验筛》**：为滚筒筛筛分性能标定的配套检测标准，要求筛面有效直径不低于185 mm，筛孔尺寸偏差及金属丝直径偏差需符合GB 6004相关规定[[S18]]。
