## 定义

铸型（mold）是用型砂、金属或其他耐火材料制成，包括形成铸件形状的空腔、型芯和浇冒口系统的组合整体。砂型用砂箱支撑时，砂箱也是铸型的组成部分。[[S28]] 铸型是铸造生产中用于形成铸件形状、尺寸和性能的核心工装，通过浇入熔融金属凝固后得到金属毛坯或零件。[[S22]]

在铸造技术语境中，铸型是广义上位概念，包含所有用于容纳金属液成型得到铸件的工装类型，覆盖砂型、金属型、陶瓷型、石膏型等所有材质类别的成型腔体。[[S18,S17]] 在压铸领域，铸型特指由淬硬耐热钢加工而成的可重复使用的永久金属型腔工装（即压铸模），可承受高压高温金属的高速冲刷，能连续生产数千至数十万件合格压铸件。[[S7,S13]]

## 分类

### 按成型材料分类

按铸型的成型材料，可分为四大核心类别：[[S18]]

| 类别 | 特征 |
|------|------|
| 砂型 | 以砂为主体成型材料，适用于最广泛的铸造场景 |
| 金属型 / 硬模 | 铸铁、铸钢等金属材质，可重复使用 |
| 陶瓷型 | 耐高温、高精度 |
| 石膏型 | 适用于低熔点合金精密铸造 |

### 按使用寿命分类

按使用寿命可分为三大类别：[[S15]]

| 类别 | 说明 |
|------|------|
| 一次型 | 浇注完成后即刻损毁，仅可单次使用（如砂型） |
| 半永久型 | 可多次使用但寿命有限 |
| 永久型 | 可重复浇注数千次以上，长期循环使用（如金属型、压铸型） |

在英文体系中，铸型通常被分为一次性使用的可消失型（Expendable Mold）和可重复使用的非消失型（Non-expendable Mold）两大体系。[[S5]] 国内经典铸造分类体系则将铸型归入砂型铸造、精密铸造、金属型铸造和特种铸造四大类下，形成完整的树状层级结构。[[S25]]

### 按制型方法分类（砂型）

对于砂型铸型，按造型方法可分为12种细分类型：湿型、表面干型、干型、焙烧型、油砂型、空气硬化型、冷硬型、壳型、水泥型、气体硬化型、自硬型、流态自硬型，各类型适配不同铸件批量、精度和生产效率要求。[[S18,S14]]

## 结构与组成

### 典型砂型结构

典型砂型的完整结构包含以下组成部分：[[S9]]

| 组成部分 | 功能 |
|----------|------|
| 上砂型、下砂型 | 构成铸型主体，形成型腔外围 |
| 分型面 | 上型与下型的接触面，用于模具定位分型 |
| 型腔 | 形成铸件外形轮廓的空间 |
| 型芯 | 形成铸件内部的空腔、孔洞或侧面凹凸部分 |
| 芯头、芯座 | 芯头为型芯的延伸部分，插入芯座内，用于定位、支撑和排气 |
| 浇注系统 | 含外浇口、直浇道、横浇道、内浇道，引导金属液充型 |
| 排气系统 | 排除型腔内气体 |

### 压铸模（金属型）结构

压铸模整体分为定模和动模两大模块，并可细分为八大功能单元：[[S10]]

| 功能单元 | 作用 |
|----------|------|
| 浇注系统 | 熔融金属进入型腔的通道 |
| 溢流与排气系统 | 排除气体，储存冷金属和涂料余烬 |
| 成形工作零件 | 定、动模镶块及型芯，形成铸件内外表面 |
| 抽芯机构 | 侧向抽芯脱模 |
| 推出、复位机构 | 铸件顶出及复位 |
| 导向机构 | 引导动、定模开合模运动 |
| 模具支撑与固定零件 | 模体安装固定 |
| 模具加热与冷却系统 | 控温控热平衡 |

金属型（metal mold）是用金属材料制成的铸型，禁止将其误称为"金属模"，属于永久型铸型的核心类别，可多次重复使用。[[S1]]

## 工艺参数与操作条件

### 砂型铸造典型工艺参数

砂型低压铸造的推荐工艺参数如下：[[S11]]

| 工艺参数 | 推荐值 |
|----------|--------|
| 浇铸温度 | 730 ℃ |
| 升液压力 | 0.008 MPa |
| 保压压力 | 0.012 MPa |
| 换热系数 | 500 W/m²·K |
| 砂型温度 | 室温烘干 |

在砂型工艺中，型砂的紧实度和湿压强度受混砂机规格（如直径30 cm与100 cm）、水分含量和捣实次数等参数影响，需根据具体工况调控。[[S19]]

### 压铸模操作条件

压铸过程中，模具需承受高温、高压、高速金属熔体的冲刷及急冷急热转换。压力铸造使模具内的液体金属受到7.0–140 MPa（1000–20000 psi）的压力，模具吸收铸件的凝固潜热，需在下一循环前通过加热与冷却系统从型腔部分散发热量，热流的设计和控制是压铸过程中最为关键的环节之一。[[S7]]

## 相关术语

下列与铸型直接相关的术语均属铸造标准体系范围：[[S28,S29,S1]]

| 术语 | 定义 | 关系 |
|------|------|------|
| 型砂 | 由铸造用砂、粘结剂及附加物按一定配比混制而成、符合造型要求的混合料，按粘结剂类型分为黏土砂和非黏土砂两类 | 制作砂型（铸型）的核心原材料 |
| 金属型 | 用金属材料制成的铸型，禁止称为"金属模" | 铸型的子类，属永久型 |
| 型芯 | 用芯砂或金属制成，在铸型中形成铸件内部空腔或凹凸部分的组件 | 铸型的组成部分 |
| 芯头 | 型芯的延伸部分，不形成铸件轮廓，插入芯座内 | 用于型芯在铸型中的定位、支撑和排气 |

## 应用与影响

砂型铸造采用砂型作为铸型，其铸件总产量占所有铸件总重量的90%以上。[[S22,S4]] 不同类型铸型直接影响铸件的尺寸精度、表面质量和凝固质量。金属型因冷却速率快，其铸件拉伸性能通常高于砂型铸件和熔模铸件，接近同合金的锻件水平。[[S6]] 在离心铸造等特种铸造中，铸型转速不足会导致内部熔融金属无法完全贴附铸型内壁而出现滞留滚动，影响充型质量。[[S12]]
