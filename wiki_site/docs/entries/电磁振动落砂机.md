## 概述

电磁振动落砂机是由电磁振动器产生振动完成铸件落砂作业的专用落砂设备[[S12,S14]]。其标准英文名称为 **electromagnetic vibrating shakeout**[[S14]]。该设备依靠“电—磁—力”的能量转换产生激振力，驱动振动台上下跳动，通过周期性的撞击使铸件与型砂（或砂芯）分离[[S28]]。电磁振动落砂机无传统机械传动结构，振动参数可调，运行可靠性高，功率消耗低[[S28]]。

该设备传统上属于砂型铸造辅助设备，主流应用场景集中在大、中型铸型的落砂工序[[S9]]。在压铸领域，仅适配采用RIC压铸专用砂芯制造的特殊复杂压铸件（如汽车大型复杂压铸发动机缸体），用于清除水冷套深腔、异形内通道部位的残留砂芯[[S13]]，不适用于常规不带砂芯的压铸件后处理。

## 分类位置

在落砂设备的分类体系中，电磁振动落砂机归属于**振动落砂机**大类，与偏心振动落砂机、惯性振动落砂机、气动振动落砂机并列，采用电磁驱动方式，无传统机械传动结构[[S6,S3,S8]]。按调谐属性归类为**近共振类落砂机**[[S6]]。

## 结构与组成

电磁振动落砂机的标准核心结构由五大部件组成[[S5,S25]]：

1. **固定撞击架** — 与落砂框架配合产生撞击作用
2. **双冲程电磁激振器** — 核心驱动元件，完成电-磁-力转换
3. **落砂栅格（振动台）** — 承载砂箱和铸件，传递振动
4. **落砂框架** — 振动参振主体结构
5. **差动式弹簧系统** — 实现谐振匹配与减振隔振

此外，设备配套专用**降频电路**，将三相交流电源转换为低频脉冲方波供电[[S5,S27]]。

弹簧系统的选型直接影响设备性能[[S5,S25]]：
- **圆柱形螺旋弹簧** — 用于减振或谐振工况
- **钢与橡胶混合弹簧** — 用于减振或共振场景
- **板弹簧或橡胶弹簧** — 用于支撑减振场景

按 GB/T 25370-2020 术语定义[[S24]]：
- **隔振弹簧**：安装于参振机体和机座/基础之间的弹性元件
- **共振弹簧**：双质体落砂机中安装在工作质体和激振质体之间的弹性元件

## 工作原理

电磁振动落砂机的工作过程包含三个关键环节[[S27,S21,S28,S5]]：

**（1）电磁激振环节**
三相交流电源经降频电路转换为 **10～12.5 Hz** 的低频脉冲方波电压，交替向双冲程电磁激振器供电实现激振。电磁激振器衔铁安装气隙推荐取值范围为 **0.5～1.5 mm**[[S27,S21]]。电磁振动器完成“电—磁—力”的能量转换，输出的激振力直接驱动振动台上下高频跳动[[S28]]。

**（2）共振调节环节**
设备依靠配套的差动式弹簧系统完成谐振匹配，使整机振动系统运行在**近共振状态**，可在较低输入功率条件下获得足够大的激振输出振幅[[S25,S5]]。

**（3）振动传递与落砂动作**
放置在振动台（落砂栅格）上的砂箱被周期性抬升后自由下落，通过反复撞击固定撞击架实现型砂与铸件的分离[[S28]]。该设备无传统机械传动部件，振动参数可灵活调节[[S28]]。

## 关键参数

电磁振动落砂机的主要额定参数如下[[S27,S21]]：

| 参数项 | 规格/范围 |
|:---|:---|
| 额定工作振动频率 | 10～12.5 Hz |
| 供电方式 | 三相交流电源经降频电路转换为低频脉冲方波 |
| 电磁激振器衔铁安装气隙 | 0.5～1.5 mm |
| 工作状态 | 近共振 |

## 技术特点与对比

电磁振动落砂机与其他常见落砂方案的对比如下[[S28,S9]]：

| 类型 | 基本原理 | 技术特点 | 适用场景 |
|:---|:---|:---|:---|
| **电磁振动落砂机** | 降频供电驱动双冲程电磁激振器，框架谐振，固定撞击架撞击落砂 | 无机械传动，结构简单，工作可靠，功率消耗低；振动强度可调，并联组合同步控制容易；但谐振状态下对基础振动冲击强、地基要求高，电器设备复杂、成本较高 | 大、中型铸型落砂 |
| 偏心振动落砂机 | 偏心轴带动框架强迫振动，铸型抛起后与栅格撞击落砂 | 振幅恒定不受载荷变化影响，落砂效率稳定；但主轴和基础承受全部撞击力，使用寿命短，维修工作量大 | 3吨以下中、小型铸型落砂 |
| 惯性振动落砂机 | 激振器轴旋转时偏心块产生惯性力驱动框架振动，远共振工作状态 | 结构简单，经减振弹簧传递后对基础受力小、地基要求低；可覆盖重型至小型全规格铸型；但落砂效果随载荷变化存在波动 | 重型及大、中、小型铸型落砂 |
| 气动落砂（除芯） | 压缩空气驱动振击锤作业 | 专供清除铸件内腔型芯砂，对粘结力较弱的油类、纸浆类粘结剂砂芯适配性更好 | 铸件内腔砂芯清除 |
| 高压水力清砂 | 高压水射流对砂芯产生剪切、渗透和冲刷作用 | 无粉尘污染，劳动条件好，芯骨可直接回收；但需配套大容量沉淀池和湿砂干燥设备，占地面积大 | 复杂内腔砂芯清除 |

电磁振动落砂机的核心优势特征[[S28]]：
- 无机械传动结构，运行可靠
- 振动参数可按需调节，适配不同落砂工艺需求
- 整机功率消耗远低于惯性类落砂机

主要局限[[S9]]：
- 谐振状态下对基础振动冲击强，对地基要求高
- 配套电器设备复杂，设备购置成本较高

## 在压铸中的应用

电磁振动落砂除芯方案在压铸领域的适配边界明确[[S13]]：
- **不适用于**绝大多数不带砂芯的常规压铸件后处理工序
- **仅适配**采用 RIC 压铸专用砂芯制造的特殊复杂压铸件后处理场景
- **典型适用件**：汽车大型复杂压铸发动机缸体，用于清除常规金属型芯无法脱模的水冷套深腔、异形内通道部位的残留砂芯[[S13]]

> **说明**：按照 GB/T 25370-2020 的定义，落砂（knock-out / shakeout）是指*砂型铸造*中铸件与铸型/砂箱、砂芯与铸件分离的过程[[S20,S7]]；除芯（decororing / core knock-out）是指采用锤击或其他方式清除铸件内腔砂芯的过程[[S7]]。电磁振动落砂机在压铸中的应用本质上属于**除芯**操作，适用范围限于压铸工艺中配合砂芯使用的特定场景。

## 选型依据

电磁振动落砂机的核心选型判断维度[[S23]]：

| 选型维度 | 判别依据 |
|:---|:---|
| **铸件材质** | 铸钢件需选用大激振力高振幅低频率落砂机；铸铁、有色金属铸件选用低振幅落砂机避免铸件磕碰损伤 |
| **铸型强度** | 低强度湿型砂适配低振幅落砂机；高强度干型砂适配高振幅落砂机 |
| **生产纲领（非生产线）** | 按最大批量大件砂箱底面尺寸及额定重量选型；单件落砂时间：中小件1～5 min，大件5～10 min，特大件10～20 min |
| **生产纲领（造型线）** | 需匹配主机生产率；复杂大尺寸砂型可采用两台串联配置实现完全落砂 |

## 安全与环保

电磁振动落砂机属于铸造高噪声、高粉尘作业环节的设备[[S18]]：
- 运行需配套**隔振降噪措施**（如采用橡胶/钢-橡胶复合弹簧实现减振）
- 需配套**密闭除尘装置**
- 运行产生的噪声需符合《工业企业噪声卫生标准》相关限值要求
- 应避免长期振动、噪声暴露损伤操作人员健康[[S18]]

## 日常维护

电磁振动类振动设备的日常维护强制要求[[S4]]：

| 维护要求 | 说明 |
|:---|:---|
| **禁止随意调整** | 禁止旋动弹簧刚度调节螺母、增减替换弹簧、改动槽体长度，避免破坏出厂预调匹配的振动参数 |
| **定期检查更换** | 及时更换老化橡胶元件和损坏零件 |
| **禁止带病运行** | 不允许在机体开裂、弹簧折断撕裂的状态下运行，不允许超载或承受超过设计允许的大冲击 |
| **焊接安全** | 焊接机件时确保接地线连接合规，防止电磁驱动装置导线作为接地线烧毁 |

## 试车与验收

**空载试车要求**[[S2]]：
- 观察栅床无异常跳动
- 异常跳动常见成因：栅床平衡失调、重心偏移、缓冲弹簧刚度不一致、传动带张力调整不当
- 排除方法：加装平衡块调整栅床平衡、调整各缓冲弹簧刚度一致、保证传动带长度弹性均匀

**负载试车与整机验收要求**[[S2]]：
- 负载试车需调整激振力达到最佳落砂效果
- 整机验收时，额定负载下功率消耗需控制在公称额定功率范围内
- 正常工作状态下轴承温升应 ≤ 25 ℃
