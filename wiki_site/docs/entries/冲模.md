## 定义与术语辨析

在通用模具术语标准中，“冲模”是指使金属或非金属板料或型材在压力作用下分离、成形或接合为制品的模具，包含冲裁模、拉深模、弯曲模等多个子类[[S22]]。

在压铸行业语境下，“冲模”主要指压铸后处理工序中用于去除压铸件飞边、料柄、溢流槽等多余附属材料的专用冲切工艺装备，英文对应 **trim die / trimming die**，业内亦常直接称为**切边模**或**修边模**[[S23,S4,S31]]。该后处理工序是压铸全产业链中除成形外使用最广泛的“冲模”场景，北美压铸协会技术资料及国内行业手册对这一术语的默认指代均为此类除边工装[[S23,S4]]。

在模具术语层级中，通用“冲模”是最高大类；冲裁模是冲模的核心子类，实现材料分离；而压铸后处理用冲模（切边模）属于冲裁模下的细分专用子类，功能为切除成型后制件的多余边缘材料[[S42,S44]]。

**歧义说明**：在压铸机主机内部，“冲头”或“压射冲头”有时也被非正式地称为“冲模”，该部件属于压铸成形设备的核心零件，承担将金属液注入型腔的功能，与本词条所定义的压铸后处理工装“冲模”无关联[[S21,S22]]。

## 分类

### 按动作方式
- **单动冲模**：仅配置一组动力驱动结构，单次行程完成全部剪切动作，结构简单、成本低，适用于中小型普通压铸件的单工位除边[[S37]]。
- **双动冲模**：配置两组独立驱动滑块，可先完成预压紧再冲切，或同时完成多方向同步冲切，冲切后工件变形更小，适用于精度要求高、结构复杂的大型压铸件批量生产[[S37]]。

### 按排料方式
- **落料类冲模**：沿封闭轮廓将飞边、料柄等废料整体从铸件上分离，适用于外形规则的压铸件[[S24,S14]]。
- **冲孔类冲模**：专门针对预留孔位中的多余披锋、盲孔残留料进行冲切分离，多用于带内腔、暗孔类复杂压铸件[[S14]]。
- **复合类冲模**：在单次冲压行程内同步完成落料与冲孔等多道工序，生产效率高，位置精度一致性好，适用于多特征压铸件的大批量连续生产[[S24,S14]]。

### 按自动化程度
- **手动操作冲模**：完全依靠人工完成上下料、定位及取件，结构简单，制造成本低，适用于新品试制或极小批量非标准件[[S24,S46]]。
- **半自动冲模**：人工仅需上下料，冲切由设备自动执行，生产效率和安全系数优于手动模，适合中小批量常规生产[[S24,S46]]。
- **全自动冲模**：配备自动上料、定位、冲切、排废及下料联动机构，可接入压铸自动化生产线连续运行，适用于大批量标准化压铸件的量产[[S24,S46]]。

## 典型结构与组成

压铸切边模（冲模）的核心零部件包括[[S49,S34,S6,S7,S27,S29]]：

| 零件 | 功能 | 
|------|------|
| **凸模（冲头）** | 主动剪切刃口，将铸件飞边从本体分离，大型复杂轮廓常采用镶块拼接结构，以螺钉、销钉定位固定，材质选用高耐磨合金工具钢 |
| **凹模（冲套）** | 从动剪切刃口，刃口轮廓完全贴合铸件分型面投影，通过斜楔、螺钉压块或螺钉紧固，复杂轮廓可做成分块组合式，便于局部更换 |
| **卸料板** | 配合弹性元件将切边后的工件推出，防止卡滞在刃口上 |
| **导柱导套** | 为上下模合模提供精准导向，保证凸凹模间隙均匀，JIS标准规定有详细的尺寸公差和硬度要求 |
| **模架** | 由上模板、下模板、支承板等组成，固定所有功能零件并与切边压机可靠安装，焊接结构模架需退火消除应力以保证刚性 |
| **定位机构** | 定位销、定位块等，确保铸件在模内位置与刃口轮廓完全匹配，防止切边偏移产生毛刺 |

行业通用的**推通式压铸冲模**结构要求：剪切刃口完全贴合铸件分型线轮廓，剪切动作优先朝向铸件本体，避免撕裂和残留毛刺；距离刃口3/16英寸的竖直段为“die life”刃磨余量段，下方设1/32英寸台阶让位实现自动卸料，无需额外卸料机构[[S23]]。

## 工作原理

### 冲切变形过程
正常间隙下的冲切过程可分为三个阶段[[S43,S48,S30]]：

1. **弹性变形阶段**：凸模接触待冲切部位，材料产生弹性压缩和轻微弯曲，内应力未超过弹性极限。
2. **塑性变形阶段**：应力达到屈服极限，刃口处发生塑性剪切变形，边缘形成塌角，剪切面产生光亮直边。
3. **断裂分离阶段**：拉应力超过抗拉强度，刃口周边出现微裂纹，裂纹贯通后材料完全分离，冲切断面形成断裂带与少量毛刺。

### 配套压力设备
- **机械冲床（曲柄压力机）**：冲切速率快，效率高，适配大多数中小型压铸件切边，优先选用开式冲床[[S28,S1,S40]]。
- **油压（液压）冲床**：输出压力恒定，行程可调，过载保护能力强，适配大吨位、复杂大型压铸件的高精度切边[[S28,S1]]。
- **气压驱动设备**：动作速度快，清洁无泄漏，适用于小型薄壁压铸件的轻量化切边[[S28]]。

## 工艺参数与选型

### 冲裁力计算
冲裁力通用计算公式[[S20,S3]]：
\[
P = L \times t \times \sigma_b
\]
其中 \(L\) 为飞边总剪切周长（mm），\(t\) 为飞边平均厚度（mm），\(\sigma_b\) 为对应压铸合金的抗剪强度。采用锌基合金等软模时，冲裁力约比普通钢模大50%，需预留余量[[S20]]。

### 压机吨位选择
切边压机吨位可参照以下方法计算[[S17,S15]]：
\[
P_z = \frac{K_3 \times p_j \times \sum F_i}{1000}
\]
- \(P_z\) ——所需压机吨位（t）
- \(K_3\) ——投影面积比例系数
- \(p_j\) ——不同壁厚对应合金的计算比压（kgf/cm²）
- \(\sum F_i\) ——所有压铸件分型面上的总投影面积之和（cm²）

安全系数 \(K\) 一般取1～1.3，并可按选定压机的最大比压或安全工作比压进行校核[[S15]]。

### 冲裁间隙选择
冲裁间隙是凸、凹模刃口尺寸的差值（通常指双边间隙），直接影响冲件断面质量与模具寿命[[S3]]。压铸切边模可参照以下推荐值（按GB/T16743“按质定隙”原则结合经验数据）[[S36,S41,S20,S47]]：

| 压铸合金 | 推荐冲裁间隙（占飞边厚度百分比） | 备注 |
|----------|-------------------------------|------|
| 锌合金 | 5%～8% | 厚度≤1.0 mm时，最小双边间隙约0.05 mm；锌合金软模初始间隙可取零，动态平衡间隙自动适配[[S20]] |
| 铝合金（软态） | 6%～10% | 厚度0.15～2.5 mm时，双边间隙约0.012～0.25 mm [[S41]] |
| 铝合金（硬态） | 8%～12% | 材料硬度越大，间隙系数取大值[[S41]] |
| 镁合金 | 参照铝合金上限，略大于铝合金 | 避免崩刃[[S47]] |

## 材料与热处理

### 常用工具钢牌号
| 牌号 | 适用场景 |
|------|----------|
| Cr12MoV（国产） | 高耐磨高淬透性，适用于铝合金、镁合金中批量切边刃口[[S25,S45]] |
| SKD11（进口） | 与Cr12MoV同属冷作模具钢，用于高精度大批量压铸件切边[[S39]] |
| SKD12 | 韧性更高，抗崩刃，适用锌合金复杂轮廓切边模[[S16]] |
| 8Cr3、5CrNiMo等热作钢 | 用于热切边工况，如高合金铝合金锻件热切除飞边[[S25]] |

常规刃口材料也可选用O-1油淬工具钢，便于焊接维修，大尺寸复杂冲模常采用分块式结构以便局部磨削更换[[S35]]。

### 典型热处理硬度
| 工况 | 凹模硬度（HRC） | 凸模硬度（HRC） |
|------|----------------|----------------|
| 冷切边 | 58～62 | 56～60 |
| 热切边 | 43～50 | 35～45 |

被冲切材料强度较高时，硬度上限不超过50 HRC以避免崩刃[[S25,S49,S12,S18]]。

### 表面强化工艺
- **氮化处理**：适用于压铸锌、镁、铝合金零件刃口，氮化层深度0.08～0.15 mm，表面硬度≥600 HV，显著提高耐磨性[[S16,S19]]。
- **TD处理**：形成高硬度碳化钒覆层，大幅提升抗磨损性能，适配大批量铝合金切边长期运行[[S16]]。
- **PVD涂层**：获得纳米级超硬表面涂层，降低摩擦阻力，避免粘铝，适用于高精度外观压铸件的无毛刺切边[[S16,S19]]。

## 与压铸工艺的联动

压铸冲模通常布置在压铸机取件机械手侧，与压铸机、取件机器人共同构成压铸后处理核心工位，用于自动切除浇道、溢流口及飞边[[S26]]。

典型全自动压铸切边流水线运行流程[[S52]]：
1. 压铸机开模后，自动取件机械手将铸件取出送入冷却工位淬冷；
2. 冷却后的铸件定位放置在切边冲模工位，上模下行自动剪切浇注系统及飞边；
3. 冲切分离后的成品铸件经输送机构转运至后续清理工位，废料由下方收集装置统一回收回熔。

## 常见缺陷与维护

### 典型故障及对策
| 故障 | 原因及对策 |
|------|------------|
| 刃口磨损 | 可用油石现场研磨恢复锋利度；无法恢复时卸模进行平面磨削[[S9,S32]] |
| 崩刃 | 损伤部位局部补焊后重新磨削刃口[[S32]] |
| 残留毛刺过大 | 重新调整凸、凹模剪切间隙，将单边间隙控制在0.025～0.05 mm范围内[[S9,S32]] |
| 卡料 | 清理模腔内残留铝屑，疏通下方排料通道[[S32]] |
| 定位偏移 | 重新校准定位销位置，调整压料块压力均匀度[[S32]] |

### 寿命参考
| 压铸合金 | 冲模参考寿命（冲次） |
|----------|---------------------|
| 锌合金 | 约1,000,000 |
| 铝合金 / 镁合金 | 约100,000 |
| 黄铜 | 约10,000 |

上述数据来源于行业通用经验[[S2]]。影响寿命的主要因素包括模具材料、冲裁间隙、维护质量及生产频次。

### 日常维护要点
- 每班生产结束后清理模腔表面残留金属屑，刃口和导向部位涂防锈油[[S38]]；
- 定期检查导向衬套磨损及紧固件松动情况，防止运行偏移[[S38]]；
- 建立预防性维护计划，在非生产时间完成刃口刃磨和易损件更换，减少突发停机[[S38]]。

## 应用实例

- **铝合金电机壳体类深腔铸件**：采用整体型刃口配随形压料块结构，贴合外轮廓实现无变形切边，避免深腔壳体在剪切过程中塌边变形[[S8]]。
- **大批量精密锌合金装饰件**：可配套液氮低温切边工艺，降低切边毛刺生成概率，提升边缘光洁度，免去后续人工打磨[[S51]]。
- **滤网类薄板铝合金压铸件**：采用密集多小冲头排列的冲孔类冲模，完成多孔位同步冲切[[S10]]。
- **大型曲轴箱压铸产线**：在线布置可往复运动的切边模，配合自动取件实现流水线作业[[S11]]。
