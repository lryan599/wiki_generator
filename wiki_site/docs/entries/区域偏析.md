## 概述

区域偏析（zone segregation / regional segregation），在铸造学科中亦称为宏观偏析（macrosegregation）或长程偏析，是指合金凝固过程中，在铸件或铸锭的不同宏观区域内出现有规律的化学成分不均匀现象，其成分差异发生在与铸件整体尺度相当的大范围内，属于宏观尺度的偏析缺陷[[S34,S58,S71]]。区域偏析一旦形成，由于溶质元素所需扩散距离过大，无法通过常规均匀化扩散退火予以消除，只能通过改进铸造工艺进行预防和减轻[[S2,S40]]。

偏析作为铸件的典型凝固缺陷，整体上分为宏观偏析和微观偏析两大类别，其中宏观偏析的别称即为区域偏析，是铸件全断面大尺度范围内的成分不均匀；微观偏析（别称显微偏析/短程偏析）则局限于单个晶粒尺寸范围，包含晶内偏析（枝晶偏析）、晶界偏析等类型，可通过扩散退火缓解或消除[[S58,S59,S41]]。两者在偏析尺度、产生机理和消除手段上存在本质区别。

根据铸件断面上易熔组分富集的部位不同，区域偏析可进一步分为正偏析（铸件心部富集低熔点溶质）、反偏析/逆偏析（铸件表层富集溶质、心部溶质贫乏）、中间偏析（溶质集中于表层与心部之间的环形中间区域）以及带状偏析等类型[[S34,S15,S55,S70]]。

## 分类与层级关系

在铸造学科中，偏析类别的完整层级结构如下[[S58,S19,S72]]：

| 一级 | 二级 | 三级（下属类型） | 主要特征 |
|------|------|------------------|----------|
| 偏析 | 宏观偏析（区域偏析/长程偏析） | 正偏析 | 低熔点成分从外部到中心逐步富集 |
| | | 反偏析（逆偏析） | 表层溶质富集、中心贫乏 |
| | | V型偏析 | 铸锭中心呈V形分布的偏析带 |
| | | 逆V型偏析 | 与V型偏析方向相反的偏析带 |
| | | 带状偏析 | 平行于固液界面周期性分布的偏析带 |
| | | 重力/比重偏析 | 因密度差沿高度方向产生的成分分层 |
| | 微观偏析（显微偏析/短程偏析） | 胞状偏析 | 胞状晶间的成分不均匀 |
| | | 枝晶偏析（晶内偏析） | 枝晶干与枝晶间成分差异 |
| | | 晶界偏析 | 晶界处低熔点物质的富集 |

### 正偏析

正偏析（normal segregation）是最常见的区域偏析形式。对溶质平衡分配系数K<1的合金，在凝固过程中，凝固界面不断向液相中排出溶质，导致液相中溶质浓度持续升高，愈是后结晶的固相溶质含量愈高，最终造成低熔点成分和易熔杂质从铸件外部到中心逐渐增多，偏析核心位于铸件心部或最后凝固的热节位置[[S70,S55,S19]]。

### 反偏析（逆偏析）

反偏析（inverse segregation / abnormal segregation）与正偏析的溶质分布方向相反。对K<1且凝固区间宽的合金，在缓慢凝固条件下形成粗大枝晶，枝晶间富集溶质的剩余金属液在凝固收缩力、大气压力、液柱静压力以及析出气体压力的共同作用下，沿枝晶间毛细通道向外迁移至先凝固的表层区域，导致铸件表层溶质含量高于合金平均成分、心部溶质匮乏的现象[[S70,S55,S43,S19]]。严重时，富溶质液相会突破凝壳薄弱处渗出铸件表面，形成偏析瘤[[S11]]。

大野笃美提出了反偏析形成的另一种解释：凝固初期模壁上形成根部呈“颈缩”形态的初生晶体，当相邻晶粒接触后，溶质富集的熔体被封闭在晶粒根部，形成表层溶质富集带。该反偏析形成示意图直观描绘了这一过程[[S69]]：

![铸锭反偏析形成示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f3119561-4c2f-47f6-8ef0-ad2444a4ed0d/resource)

反偏析是铝合金连续铸造中的典型偏析形式[[S8]]，也是镁铝系压铸合金中最常见的偏析类型，合金中铝含量增高、晶粒粗大、铸件冷却速度过快且温度梯度较大时，反偏析程度显著增大[[S32,S45]]。

### 带状偏析

带状偏析出现在定向凝固的铸锭或铸件中，偏析带平行于固/液界面，沿凝固方向周期性分布[[S49,S65,S47]]。其形成与固/液界面前沿溶质偏析引发的成分过冷以及凝固速率的周期性波动有关——当凝固速率突然增大时形成负偏析带，速率突然减小时形成正偏析带[[S47]]。

### 中间偏析

中间偏析是区域偏析的一种特殊类型，易熔组分既不富集于铸件心部也不富集于表层，仅分布在心部与表层之间的环形中间区域[[S70,S8]]。

### 比重偏析（重力偏析）

比重偏析属于宏观偏析（区域偏析）的下属类型，是因固相与液相的密度差异，在重力或离心力作用下，使先结晶的固相上浮或下沉，导致沿铸件高度方向出现成分分层现象[[S15,S25,S19]]。比重偏析常见于铅青铜、锡锑合金等组元密度悬殊的合金体系[[S40]]。

## 形成机理

### 溶质再分配基础

合金凝固过程中，溶质再分配是区域偏析形成的热力学基础。其核心参数为平衡分配系数k₀，定义为同一温度下固相溶质平衡浓度C_S与液相溶质平衡浓度C_L的比值，即k₀ = C_S / C_L。绝大多数常见合金的k₀<1，凝固时固相优先将溶质排入前沿液相，随着凝固界面向前推进，液相中溶质浓度持续升高[[S39,S18]]。

在非平衡凝固条件下，固相中溶质扩散可忽略不计时，Scheil方程是描述非平衡凝固微观溶质再分配的经典基础模型，也是现代区域偏析理论定量分析的基础[[S14,S31,S22]]。

### 溶质迁移驱动机制

区域偏析的形成，本质上需要固液两相区内的溶质富集液相发生长距离宏观迁移。主要的溶质迁移驱动机制包括以下三种[[S14,S3,S52,S4]]：

1. **热溶质对流**：由温度梯度和溶质浓度梯度共同造成熔体密度差异，驱动轻重不同的液相上浮或下沉，产生宏观尺度的溶质对流迁移。
2. **凝固收缩诱导流动**：凝固过程中因固液相密度变化产生体积收缩，引发枝晶间补缩流动，带动富溶质液相沿枝晶间隙定向迁移。
3. **外部强制对流**：在电磁力、机械力等外力场驱动下，熔体产生强制运动，带动溶质重新分布。

### 压铸区域的特殊偏析机制

压铸条件下的区域偏析形成机制与重力铸造存在显著差异，主要体现在以下两方面[[S9,S42,S68,S61]]：

**（1）充型阶段——外生晶迁移与缺陷带（偏析带）形成**

高压压铸熔体在压室中即开始预结晶，形成外生固相晶体（externally solidified crystals, ESCs）。在高速充型过程中，这些ESCs随半固态熔体冲入型腔，并向铸件厚度中心区域聚集。与此同时，存在两个相反的凝固前沿：从铸件表层向中心推进的外部冷却凝固前沿，以及从中心ESCs聚集区向表层推进的内部凝固前沿。两个凝固前沿交汇的位置因凝固收缩未能得到有效补缩，会形成沿铸件轮廓分布的带状偏析（缺陷带），这是重力铸造几乎不会出现的偏析形态。该偏析带中常富集孔洞与偏析元素，厚度约为7~18个晶粒宽度[[S42]]。

**（2）高压凝固阶段——压力驱动挤出与通道偏析**

在挤压铸造或高压压铸的高压凝固阶段，施加于糊状区的压力会将正在生长的枝晶间富集溶质的液相强行挤出，使其向最后凝固的热节区域聚集，形成挤压铸造特有的异常偏析（通道偏析/热节正偏析）。这种偏析在重力铸造条件下通常不明显，但在压力作用下会异常加重[[S61]]。

## 工艺影响因素

区域偏析的程度和分布形态受合金特性与铸造工艺参数的综合影响。压铸条件下，主要影响规律如下[[S29,S50,S42,S45,S68]]：

| 影响因素 | 影响规律 |
|----------|----------|
| 合金性质 | 结晶温度范围愈宽、溶质分配系数|1-k₀|愈大，偏析倾向愈大；合金的体收缩系数愈大，枝晶间毛细孔道愈多，反偏析愈严重[[S43]] |
| 铸件壁厚 | 壁厚越大，不同位置冷却速率差异越大，横截面偏析和中心偏析的面积越大，伴生裂纹尺寸也越大 |
| 压射低速速率 | 低速速率提高，ESCs在铸件中心的偏析带逐渐向心部移动、宽度减小 |
| 压射高速速率 | 高速速率越大，ESCs分布愈弥散，偏析带愈向心部靠拢；无高速压射时会出现双条偏析带 |
| 模具温度 | 适当提高金属型工作温度，可减少冷却速度与温度梯度，降低反偏析生成倾向[[S32]] |
| 浇注温度 | 适当降低浇注温度，有利于缩小枝晶生成区域，减轻偏析 |
| 增压压力 | 增压压力提升可增强高压补缩能力，降低枝晶间富溶质液相的迁移聚集程度 |
| 真空辅助 | 真空辅助压铸可减小ESCs的尺寸和数量，降低区域偏析程度 |

## 缺陷与危害

区域偏析作为典型的铸造凝固缺陷，对铸件质量的影响体现在多个维度[[S34]]：

**（1）力学性能不均**

偏析区域通常伴随缩松缺陷，力学性能显著低于健全区域。典型案例为HPb59-1铅黄铜出现区域偏析后，局部抗拉强度仅为174.5~207.8MPa、伸长率仅2%，而无偏析正常铸件抗拉强度可达392MPa、伸长率达30%，性能差异超过1倍[[S20]]。偏析导致的强度不均匀还会使压力加工后易出现局部热脆、轧制板材夹层失效[[S27]]。

**（2）耐腐蚀性能下降**

偏析区域富集的低熔点共晶或溶质元素会成为腐蚀介质的优先侵蚀位点，暴露在腐蚀环境下偏析区域将率先发生电化学腐蚀，大幅缩短铸件服役寿命[[S27]]。

**（3）加工硬化不一致**

偏析区易出现大块状异常相（如粗大α相、富集的金属间化合物），导致加工过程中不同区域硬度波动大。局部硬点会显著加剧刀具磨损；软点区域则易产生粘刀、加工表面白斑等缺陷。例如，ZL108铝合金中硅元素区域偏析时，共晶硅聚集区硬度高、脆性大，而硅元素缺失区出现软点，加工一致性完全失控[[S44]]。

**（4）气密性失效**

区域偏析往往伴随连续分布的显微缩松和枝晶间隙，这些缺陷连通后会形成贯穿性渗漏通道，在承压工况下直接出现液体或气体泄漏。典型案例为60%Pb-38%Cu-2%Ni合金机车垫圈铸件，未采用加压铸造前因严重区域偏析导致垫封性能失效、铸件漏耗蒸汽，该类零件报废率高达60%[[S21,S32]]。

## 消除与减轻措施

### 双控成形技术

双控成形，全称为“液态铸锻双控成形”，是2005年由李远发等人提出、哈尔滨工业大学罗守靖学者团队联合研制出国内首台专用铸锻双控成形机的特种压铸衍生工艺，已获得公开发明专利保护[[S38,S35]]。其定义为：在液态金属成形过程中同时精确控制零件形状精度和内部力学性能，集合传统压铸复杂充型优势与锻造细晶强韧化优势的一体化成形工艺[[S35]]。

双控成形的核心工艺原理为：液态金属低速平稳充型进入模具型腔，在金属液尚未完全凝固的极短时间内（通常为充型结束后0.02~0.07s），启动专用锻压冲头施加远高于常规压铸补压力的锻造载荷（锻造压力8~12MPa），破碎正在生长的粗大枝晶，强制变形促进形核，同时将残余枝晶间液相均匀分散，直接阻止大尺度范围内的溶质元素聚集，避免区域偏析形成[[S13,S30]]。

现有公开学术文献验证表明，双控成形可获得晶粒细小、组织致密、无区域偏析的压铸衍生制件。针对AZ91D镁合金的对比试验显示，双控成形制件相比普通压铸件，抗拉强度提升84.39%，延伸率提升94.67%，热处理后性能进一步提升，未检测到宏观尺度的成分区域偏析现象[[S38,S35]]。

### 其他消除与减轻措施

| 措施类别 | 具体方法 | 适用场景 | 原理说明 |
|----------|----------|----------|----------|
| 电磁搅拌 | 施加脉冲/螺旋电磁场搅动凝固前沿 | 半固态压铸、大规格连续铸锭 | 打乱溶质定向富集流动，细化枝晶组织，降低元素偏析波动幅度[[S37]] |
| 超声波熔体处理 | 超声波振动作用于熔体 | 半固态压铸 | 通过空化效应打乱固液两相区溶质分布[[S37]] |
| 梯度控温优化凝固 | 合理布置冷铁、分区控制模具冷却水路 | 壁厚差异大的复杂结构铸件 | 实现近似同时凝固，避免厚大热节部位溶质长时间聚集[[S40]] |
| 熔体变质处理 | 添加细化剂、稀土元素、硫/锑等添加剂 | 易发生比重偏析的合金体系（如铅青铜、锌铝合金） | 增加形核核心，限制粗大枝晶生长，阻碍异相质点上浮/下沉[[S10]] |
| 流变压铸 | 采用半固态浆料进行压铸成形 | 普通压铸件 | 相比普通压铸，可显著减轻不同位置Zn、Mg、Cu等元素的偏析波动[[S36]] |

值得特别指出的是，均匀化热处理仅能减轻微观尺度的枝晶偏析，由于区域偏析的溶质扩散距离过长，常规扩散退火完全无法消除宏观区域偏析，这是铸造领域的行业共识[[S2,S40]]。

## 检测与表征

### 金相低倍浸蚀法

沿铸坯横截面切取约10mm厚度薄片，采用盐酸和水体积比1:1的溶液，在60~80℃条件下浸蚀5~20min，浸蚀完成后用清水冲洗，通过高清拍摄直接记录宏观偏析分布[[S1]]。

### 直读光谱法

将待测区域表面用1000#砂纸打磨平整后直接测试，依靠电火花激发金属表层获取元素特征谱线，可快速得到被测点的多元素定量含量，适用于铸锭不同位置的批量成分检测，大幅提升偏析程度筛查效率[[S56,S46]]。

### 电子探针微区分析（EPMA）

可实现选定区域的元素分布面扫描，精准表征C、Mn、Si等典型偏析元素的二维分布特征，直接输出不同位置的元素偏析比，是区域偏析微观尺度定量表征的核心手段。例如，低合金钢中偏析区域的Mn元素偏析比最高可达1.56[[S1,S53]]。

### 偏析度定量计算

行业通用的偏析程度定量计算公式为[[S1]]：

> R_i = M_i / (∑M_i / n)

其中，R_i为i点的元素偏析度，M_i为i点的溶质元素实测含量，n为待测区域的总取样点数。R_i > 1为正偏析，R_i < 1为负偏析。

### 典型偏析形貌

## 标准与规范

目前国内外尚无直接以“区域偏析”为命名的独立评级标准，偏析类缺陷的评级主要通过各类无损检测标准中的偏析图谱进行。

### 国内标准体系

- **GB/T 5677-2007**《铸钢件射线照相检测》：通过底片判定偏析缺陷等级[[S17]]
- **GB 7233-1987**《铸钢件超声探伤及质量评级方法》：将偏析类非平面缺陷分为4个质量等级，与ISO 4992等效对应[[S33,S17]]
- 在连铸坯领域，国内常将中心偏析缺陷分为A、B、C三大类，从0.5级到3.0级不等。例如Q355钢铸坯在优化电磁搅拌工艺下可将中心偏析等级控制到C1.0级[[S54]]

### 国外标准体系

- **ASTM E446**：适用于<51mm薄壁铸钢件的射线参照图谱
- **ASTM E186**：适用于51~114mm中厚壁铸钢件的射线参照图谱
- **ASTM E280**：适用于114~305mm厚壁铸钢件的射线参照图谱

以上ASTM系列标准将偏析类缺陷从轻微到严重共分为8个等级[[S12,S33]]。

- **ISO 9915**：铝合金铸件射线检测标准，在ASTM图谱基础上补充了多类复合偏析缺陷的联合评定规则[[S28,S63]]

## 相关术语与混淆项

| 术语 | 定义与关系 |
|------|----------|
| 偏析（segregation） | 总称，铸件内化学成分不均匀的现象 |
| 宏观偏析（macrosegregation） | 别名区域偏析/长程偏析，大范围成分不均匀，无法通过热处理消除[[S19]] |
| 微观偏析（microsegregation） | 别名显微偏析/短程偏析，晶粒尺度内的成分不均匀，可通过扩散退火改善[[S19,S59]] |
| 正偏析（normal segregation） | 区域偏析的子类型，低熔点成分向铸件心部富集[[S19]] |
| 反偏析/逆偏析（inverse segregation） | 区域偏析的子类型，低熔点成分向铸件表层富集[[S19]] |
| 重力偏析/比重偏析（gravity segregation） | 因固液密度差引起的沿高度方向的成分分层，属于宏观偏析[[S19,S25]] |
| 枝晶偏析/晶内偏析（dendritic segregation） | 微观偏析的子类型，枝晶干与枝晶间成分差异[[S19]] |
| 晶界偏析（grain boundary segregation） | 微观偏析的子类型，低熔点物质富集于晶界[[S19]] |
