## 概述

发热剂是铸造工艺中应用广泛的一类放热辅助材料，常用于制造发热冒口（套）或直接撒敷于冒口顶面，通过自身的氧化还原放热反应向冒口内金属液提供附加热量，延长冒口内金属保持液态的时间，从而强化补缩、改善铸件内部致密性 [[S3,S6]]。

在压铸领域中，发热剂的典型应用场景集中在重力铸造、低压铸造等永久模铸造工艺；高压铸造中仅在特殊厚壁零件的局部热节补缩时才会选用低发气型发热产品 [[S10,S3]]。

## 定义与工作原理

发热剂的补缩作用建立在其可控的氧化还原放热反应之上。其核心反应以铝热反应为代表：

> 2Al + Fe₂O₃ = Al₂O₃ + 2Fe + Q，放热量 Q 约为 8.38×10³ J [[S5]]。

这一反应在熔融金属热作用下被触发，持续释放的热量可有效补偿冒口内金属液向型壁的热散失，显著延长冒口中金属的凝固时间，为铸件本体实现顺序凝固提供温度条件 [[S5,S3]]。

对于不同熔点的铸造合金，发热体系的触发条件有所区别：铸钢等高温合金可直接以铝粉‑氧化铁体系在 1250℃ 以上引发反应；铝合金、镁合金等低熔点有色合金则需额外添加镁作为点火剂，以及硝酸钠、硝酸钾等氧化剂以降低反应起始温度 [[S1,S3]]。

## 分类

发热剂可按形态、成分体系及用途三个维度进行分类 [[S3]]。

| 分类维度 | 类型 | 适用场景说明 |
|----------|------|--------------|
| 按形态 | 发热套 | 预制成型，直接镶嵌于型腔内冒口对应位置，使用前需烘干 |
| | 发热板 | 适用于特定平面或局部补热区域 |
| | 发热粉末（散状） | 浇注后直接撒入明冒口液面，适用于单件、大中型铸件 |
| | 发热带 | 用于特殊几何形状或局部窄缝区域的贴敷 |
| 按成分体系 | 铝基（铝热剂） | 以铝粉为核心还原剂，反应放热量高，广泛用于铸钢 |
| | 硅基（硅铝热剂） | 以硅铁粉等为核心还原剂，常与铝粉搭配使用 |
| 按用途 | 冒口发热剂 | 延长冒口凝固时间，提高补缩效率，减少缩孔缩松 |
| | 浇道发热剂 | 维持浇注系统温度，改善充型与补缩通道 |
| | 局部激热剂 | 对铸件局部热节进行定点补热，消除孤立热节缺陷 |

## 组成与配方

通用发热剂由五类组分构成 [[S5,S1]]：

1. **发热还原剂**：铝粉、硅铁粉等，提供放热反应的还原主体；
2. **氧化剂**：氧化铁（Fe₂O₃、Fe₃O₄）、硝酸钾、氯酸钾等，与还原剂发生氧化还原反应；
3. **耐火骨料**：石英砂、刚玉砂、镁砂等，提供高温结构骨架；
4. **延缓填充剂**：硅砂、耐火砖粉等，调节反应速度，延长发热持续时间；
5. **粘结剂**：膨润土、矾土水泥、水玻璃、酚醛树脂等，保证成型强度。

典型铸钢用散状发热剂的组分配比范围如下 [[S9,S8,S5]]：

| 组分 | 质量分数 (%) |
|------|-------------|
| 铝粉及氧化铁等发热还原剂 | 20~40 |
| 氧化剂（硝酸钾、氯酸盐等） | 10~15 |
| 硅砂、刚玉砂等耐火填充剂 | 30~50 |
| 助熔剂、粘结剂 | 余量 |

当制作直径小于 φ200 mm 暗冒口的发热套时，需额外添加水玻璃（质量分数 8%~9%）作为粘结剂，在 180~220℃ 下烘干 1~2 h 成型 [[S9,S8]]。发热套燃烧反应温度可达约 2300℃ [[S9]]。

## 工艺作用与效果

发热冒口（套）的核心工艺目标是通过延长冒口内金属液保持液态的时间，建立朝向冒口的顺序凝固温度梯度，提高补缩效率。

- 补缩效率：采用发热冒口后，冒口补缩效率可达 30%~50%，最高可达 67% [[S6,S3]]；
- 工艺出品率：相比普通砂型冒口，铸件工艺出品率可提高 10%~25% [[S6,S3]]；
- 缺陷控制：有效抑制缩孔、缩松缺陷，改善铸件内部致密性 [[S6,S3]]；
- 温度场调控：在定向凝固装置中，顶部发热剂可配合底部水冷结晶器建立自下而上的顺序凝固条件，获得定向组织铸件 [[S11]]。

## 使用方式与操作条件

两种主要的应用方式及其操作要点：

**散状发热剂（明冒口顶部撒敷）**：
- 金属液浇注完毕后，立即将发热剂匀撒于冒口液面；
- 铺层厚度应不低于 25 mm，或占冒口总高度的 10%；
- 用量约为冒口中钢液质量的 20% [[S9,S8]]。

**预制成型发热套**：
- 使用前须在 160~200℃ 条件下烘干，去除水分，避免返潮 [[S8]]；
- 安装时不宜与铸件直接接触，须预留 25~40 mm 隔离砂层，并设置排气孔，防止因气体析出导致喷溅 [[S8]]；
- 适用于重力永久模铸造时，金属型模具常规预热温度为 150~200℃ [[S2]]。

## 应用场景与工艺适配

发热剂在不同铸造工艺中的适用性差异显著 [[S3,S1,S10]]：

- **铸钢件**：应用最为成熟、广泛；大型铸钢冒口可直接采用以铝粉为核心的发热剂，反应起始温度高于 1250℃；
- **有色合金铸件（铝合金、镁合金、锌合金）**：需调配含镁点火剂及硝酸盐氧化剂的低触发温度配方；在金属型（永久模）重力铸造、低压铸造中，发热保温套可大幅延长冒口凝固时间；铝合金推荐采用明冒口以获得更好的补缩静压力；
- **高压铸造**：常规工况下应用有限，仅在厚壁压铸件的局部热节处，选用低发气型发热产品进行定点补缩；
- **熔模铸造**：可在浇口杯或冒口上加发热剂，改善补缩能力；
- **定向凝固**：顶部发热剂加热配合底部水冷结晶器，实现自下而上的顺序凝固。

## 标准与规范

与发热剂、发热保温冒口套直接相关的现行国家标准为 GB/T 3716-1995《保温冒口套》，规定了产品的尺寸公差、性能检测规范等通用技术要求 [[S4]]。

## 局限性与注意事项

- **发气风险**：发热套反应过程产生大量气体，须设置排气孔并留隔离砂层，避免铸件气孔缺陷 [[S8]]；
- **合金适配性**：低熔点有色合金必须使用专门配方的低触发温度发热剂，普通铸钢用发热剂在该温度下难以引发反应 [[S1,S3]]；
- **吸潮敏感**：预制成型发热套使用前必须充分烘干，返潮后应重新烘干，否则可能引发安全问题 [[S8]]；
- **工艺出品率与冒口模数设计关联**：发热冒口的模数约为同尺寸普通冒口的 1.43 倍，设计时需引入模数增大系数进行核算 [[S8,S9]]；
- **高压铸造适用性有限**：在高压铸造中并非通用方案，仅特定厚壁局部工程验证后使用 [[S10]]。

## 替代方案（关联概念）

在发热剂不适用或需组合使用的场景下，常见替代或协同措施包括 [[S10,S6]]：

- **冷铁**：内冷铁或外冷铁放置于热节部位，加速局部冷却，与发热冒口配合使用可强化顺序凝固条件；
- **保温冒口套**：不依赖放热反应，仅通过低导热的保温材料延长冒口凝固时间，补缩效率可达 25%~45%，模数增大系数 E 约为 1.3~1.4 [[S9]]；
- **补浇工艺**：大型铸件中采用冒口后期补浇补充金属液；
- **工艺参数调整**：适当提高浇注温度或改变浇注系统设计，优化补缩通道。
