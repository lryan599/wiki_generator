## 概述

点阵常数（lattice constant），亦称晶格常数、晶胞参数，是描述晶体晶胞大小的核心几何参数，2019年由《冶金学名词》第二版审定公布为正式冶金学名词[[S34,S22]]。该参数表征晶格基矢的物理长度，反映晶体内部原子排列的空间周期性特征[[S34]]。

![规则排列的晶体点阵示意图，展示原子排列的周期性特征，对应点阵常数的基础空间定义](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f5ecda0d-db09-4588-ab1f-0b6884154f43/resource)

点阵常数由晶胞的三个棱边长度a、b、c和三个棱间夹角α、β、γ共6个参数共同定义[[S35,S31]]。常用单位为埃（Å）和纳米（nm），换算关系为1 Å = 0.1 nm = 10⁻¹⁰ m[[S33,S31]]。

## 晶系依赖性

不同晶系的点阵常数具有不同的几何约束特征，7个晶系的点阵参数关系如下表所示[[S10,S3]]：

| 晶系 | 棱边长度关系 | 轴间夹角关系 |
|---|---|---|
| 立方 | a = b = c | α = β = γ = 90° |
| 四方 | a = b ≠ c | α = β = γ = 90° |
| 六方 | a = b ≠ c | α = β = 90°, γ = 120° |
| 正交 | a ≠ b ≠ c | α = β = γ = 90° |

立方晶系满足a = b = c，仅用单值a即可描述全部棱边参数[[S35]]。

## 与晶面间距及布拉格方程的关联

点阵常数与晶面间距d通过一系列定量关系联结，并经由布拉格方程与X射线衍射实验数据挂钩。布拉格方程的基础形式为[[S24,S18]]：

\[
n\lambda = 2d\sin\theta
\]

将不同晶系的d与点阵常数的对应公式联立，即可通过X射线衍射测得的θ值计算点阵常数。主要晶系的晶面间距与点阵常数关系为[[S20,S11]]：

- **立方晶系**：1/d² = (h² + k² + l²)/a²
- **四方晶系**：1/d² = (h² + k²)/a² + l²/c²
- **六方晶系**：1/d² = 4(h² + hk + k²)/(3a²) + l²/c²

对于立方晶系，联立后可得点阵常数的直接计算式[[S37,S21]]：

\[
a = \frac{\lambda}{2\sin\theta}\sqrt{h^2+k^2+l^2}
\]

## 测量与表征方法

### X射线衍射法（XRD）

XRD法是点阵常数测量的核心方法，通过测量高角度衍射峰的θ值可得到精度达1×10⁻⁵ nm的结果[[S21]]。高精度测定通常采用外推法消除系统误差：以cos²θ/sinθ为横坐标、实测点阵常数值为纵坐标做线性拟合，外推至cos²θ/sinθ趋近于0获得真实点阵常数a₀[[S14,S15]]。

![A356铝合金α-Al相点阵常数的Nelson-Riley外推拟合结果，以cos²θ/sinθ为横坐标，晶格常数为纵坐标](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/7aae5659-5c11-4f9f-adf8-96acf2a351bd/resource)

常用外推函数为Nelson-Riley函数：*f*(θ) = ½(cos²θ/sinθ + cos²θ/θ)，将多个高角度衍射峰计算的点阵常数对该外推变量作图，线性外推至θ = 90°处获得真实点阵常数，可消除相机偏心、样品吸收、发散等系统误差[[S29,S25]]。

内标法是将已知点阵常数的标样（常用高纯Si、石英、Au、Ag等）与待测粉末均匀混合，同步采集衍射图谱，通过标样已知衍射角校准仪器系统角度误差，修正待测样品各衍射峰位置后计算高精度点阵常数[[S28,S29]]。

### ICDD PDF卡片的参考使用

ICDD PDF卡片是点阵常数测定中的关键参考数据源，卡片编号为9位结构SS-VVV-PPPP，其中SS代表来源代码：00为实验花样、01为ICSD数据库条目、02为CSD数据库条目、03为ICDD-NIST合作金属与合金数据库条目、99为用户自建花样[[S13]]。

使用Jade等XRD分析软件时，需预先导入PDF卡片索引，选定待测物相对应的标准卡片后，通过晶胞精修（Cell Refinement）功能完成点阵常数的拟合计算，不可直接使用卡片标注的点阵常数作为未知畸变样品的实测结果[[S4,S13]]。

### 透射电子显微镜法（TEM）

TEM电子衍射法可实现微区局部的点阵参数表征，配合高分辨像可直接测量析出相与基体的共格应变，但仅能表征样品表层约50 nm厚度的信息，适合纳米尺度微区结构的高精度测量，无法获得宏观平均的整体点阵常数结果[[S23,S26]]。

### 中子衍射法

中子衍射法基于与XRD一致的布拉格衍射定律，中子的穿透深度远高于常规X射线，可实现大体积样品内部的三维点阵参数无损测量。但其点阵应变测量精度不高于200×10⁻⁶，设备造价高、运行通量低，难以在工程现场推广使用[[S2]]。

### 相关标准

点阵常数残余应力测定相关的ASTM标准包括：ASTM E915专门规定了残余应力测量用X射线衍射仪的校准方法，明确仪器角度标定精度要求以保证点阵参数测量的可靠性；ASTM E837标准通过钻孔法结合XRD点阵畸变关联残余应力[[S19,S1]]。

## 影响因素

### 合金元素固溶与Vegard定律

合金元素固溶于基体时，点阵常数发生规律性变化。对于置换固溶体，若溶质原子半径大于溶剂原子半径，晶格发生膨胀（正畸变），点阵常数变大；若溶质原子半径小于溶剂原子半径，则发生收缩（负畸变）[[S7]]。Vegard定律描述了固溶体点阵常数随组成线性变化的规律，实际体系常存在正偏差或负偏差[[S17]]。

![Vegard定律示意图：固溶体晶胞参数随组成的变化，实线为理想Vegard定律，虚线为实际偏差](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/841012b8-28f0-42aa-abf1-3aefff57b7c5/resource)

### 冷却速率的影响

以7A99铝合金为例，冷却速率对点阵常数有显著影响。冷却速率为86.0 °C/s的试样晶格常数约4.0524 Å，小于冷却速率14.5 °C/s试样的4.0559 Å。这是因为不同溶质原子对基体产生正畸变或负畸变，冷却速率改变溶质原子的固溶量，进而改变晶格畸变程度[[S7]]。

![射频溅射TiN涂层点阵常数随N/Ti原子比的变化曲线，在N/Ti=1时点阵常数达到极大值](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d4e63b66-1aa5-4ccd-a990-f405a8488478/resource)

## 压铸合金点阵常数实测数据

### Al-Si系压铸铝合金

ADC12/A380铝合金的面心立方α-Al基体，因Si、Cu溶质原子固溶产生晶格畸变，实测点阵常数范围为4.048 Å ~ 4.055 Å[[S7]]。A356铝合金α-Al相的点阵常数可通过XRD外推法精确测定[[S14]]。

### Mg-Al-Zn系压铸镁合金

AZ91D镁合金的密排六方α-Mg基体，高压压铸态实测a值约3.209 Å、c值约5.211 Å，略高于纯镁的标准值[[S12]]。

### Zn-Al系压铸锌合金

ZA27锌铝合金的富Al面心立方基体，实测点阵常数约4.07 Å，匹配ICDD标准卡片对应物相的参数范围[[S9]]。

## 残余应力测定应用

XRD sin²ψ法依赖于点阵常数的变化进行残余应力测定：当材料存在宏观残余应力时，晶格发生均匀畸变，导致点阵常数改变，衍射峰位置随之偏移，通过测量不同ψ角度下的点阵应变可计算应力状态[[S2,S30]]。
