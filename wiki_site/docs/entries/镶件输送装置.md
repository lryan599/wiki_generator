## 概述

镶件输送装置（inserts feeding）是压铸单元中镶件装置的核心组成设备，根据国家标准GB/T 37371-2019的定义，**镶件输送装置是"将镶件输送至特定位置的装置"**，标准英文术语为inserts feeding[[S8,S14]]。该装置属于压铸单元标准辅助设备的组成部分，与取件机器人、喷涂装置功能相互独立[[S10]]。

在工业应用中，镶件输送装置常见的别名包括：镶件供料系统、镶件上料机、镶件植入装置、嵌件输送机，对应英文术语有insert feeder、insert feeding device、insert loading system、die casting insert conveyor、automatic insert placement system等[[S17]]。

在压铸自动化流程中，镶件输送装置承担着将预制的镶件（嵌件）从散乱供料端稳定、准确地定向输送至取料工位，或经由抓取机构最终置于模具型腔内的功能，属于压铸镶件放置工序的前置专用设备[[S17]]。

## 工艺角色与工作原理

### 在全自动压铸单元中的位置

全自动压铸单元中各设备执行标准化协同工作时序，镶件输送装置的动作发生在以下时序节点中[[S21,S8]]：

| 工序序号 | 动作 | 执行设备 |
|:---:|------|----------|
| 1 | 压铸机开模 | 压铸主机 |
| 2 | 取出铸件 | 取件机器人 |
| 3 | 型腔吹气、喷涂作业 | 脱模剂喷涂装置 |
| 4 | **镶件送入模具指定位置** | **镶件输送装置/放置机械手** |
| 5 | 系统检测镶件放置到位 | 检测传感器 |
| 6 | 插芯、合模 | 压铸主机 |

### 标准工作流程

镶件输送装置的完整标准化工作流程分为以下步骤[[S17,S18,S13]]：

1. 散乱堆放的原始镶件通过振动料斗或其他定向结构自动完成排序，姿态错误的镶件被筛选并自动回流至料仓；
2. 定向完成的镶件通过输送轨道间歇输送至取料定位工位；
3. 到位传感器检测确认镶件到达定位工位后，抓取机构动作完成镶件夹持；
4. 抓取机构执行移动动作，将镶件运送至模具型腔对应放置位置；
5. 位置检测传感器触发，确认镶件准确放置到型腔内预设位置；
6. 抓取机构完全退出模具区域，向压铸主机发送联锁信号，主机收到信号后执行合模工序；若检测到镶件缺失或放置位置异常，则触发报警并终止合模动作。

### 安全互锁机制

镶件输送装置与压铸单元其余设备通过现场总线实现分布式信号交互，须与国家压铸单元安全控制标准相符[[S8,S20]]。镶件放置动作的触发需同时满足以下联锁条件[[S11]]：

- 压铸机开模到位；
- 取件设备完全退出型腔区域；
- 喷涂装置返回安全位。

镶件放置完成后，向主控制系统发送确认信号方可执行后续合模流程。当检测到镶件缺料、放置不到位等异常时，装置直接向主控制系统发送停机信号，禁止合模动作[[S8]]。

## 结构与组成

镶件输送装置的典型主流结构由以下核心部件组成[[S2,S24,S13,S17]]：

| 核心部件 | 功能说明 |
|----------|----------|
| 物料仓/振动送料盘 | 散乱存放批量镶件，依靠激振力完成镶件自动排序定向 |
| 输送轨道 | 将定向完成的镶件按节拍逐件输送至取料工位 |
| 定位夹具 | 在取料工位完成镶件的精确定位，保证抓取姿态一致性 |
| 抓取机构 | 采用气缸、电动夹爪或真空吸盘结构，完成镶件的夹持/吸取动作 |
| 检测传感器 | 包含光电传感器、红外传感器等，完成缺料检测、到位检测、放置确认检测 |
| 配套控制系统 | 与压铸单元主控制系统信号联锁，实现节拍同步与异常报警 |

镶件的自动进给结构形式主要分为**上顶式**和**皮带输送式**两类，更换模具时可通过程序编号直接调用预设输送参数，实现快速切换，无需重新调试[[S3,S7]]。部分应用场景中，镶件输送装置可与取件机械手共用同一台六轴机器人完成镶件放置动作，一台机器人可服务多台压铸机完成镶件装填[[S15]]。

## 分类与特征

### 按驱动方式分类

镶件输送装置的驱动方式可分为三类[[S16,S5,S9]]：

| 驱动方式 | 特点 | 适用场景 |
|----------|------|----------|
| **气动驱动** | 利用车间压缩空气为动力源，设备成本最低；末端定位精度一般；加/减速过程冲击大，运动不平稳 | 精度要求中等、成本敏感的应用 |
| **液压驱动** | 可输出大驱动力，运动平稳、负载能力强；需配套油箱、阀组等外围部件；易出现漏油污染问题 | 大负载、高功率需求场景 |
| **电动驱动**（步进电机/伺服电机） | 运动控制精度最高，可实现连续路径位置反馈；伺服电机可提供平稳、连续可控的运动 | 高精度镶件定位需求场景 |

### 按自动化程度分类

按照GB/T 39957-2021标准要求，镶件装置须支持手动、半自动、全自动三种供料模式[[S20]]。实际工业应用中常见两类配置：

| 类型 | 特征 | 适用场景 |
|------|------|----------|
| **半自动款** | 需人工完成镶件补料、启动触发动作，仅自动完成输送、定位工序 | 小批量、多品种试生产 |
| **全自动款** | 全程自动完成上料、排序、输送、抓取、放置全流程，无需人工介入，符合GB/T 39957-2021规定的全自动供料功能要求 | 大批量连续压铸生产 |

## 标准规范与性能要求

### GB/T 39957-2021 强制功能要求

根据国家标准GB/T 39957-2021，压铸单元中的镶件装置应满足以下技术要求[[S20]]：

| 项目 | 要求内容 |
|------|----------|
| 供料模式 | 应具备手动、半自动、自动供料功能 |
| 放置检测 | 应具有放置到位检测功能 |
| 缺料报警 | 应具有缺料报警功能 |
| 定点传送 | 完成镶件的定点传送，镶件放置重复定位精度、温度等应满足压铸工艺需求 |
| 放置数量 | 单次放置镶件数量应满足压铸工艺的需求 |

### 关键性能指标

在成熟量产场景下，镶件输送装置的关键性能表现如下：

- **镶件放置成功率**：可达98%以上[[S17]];
- **整线配套能力**：可适配常规25～630吨通用压铸机型，同时兼容2150吨级以上大吨位压铸生产单元[[S4,S17]];
- **负载能力**：搭载双钳爪的取放一体机器人式装置额定负载可达120 kg[[S17]];
- **生产节拍**：在2150吨压铸单元中，完整生产循环可适配120 s的压铸整线节拍，经工艺微调可进一步压缩总循环时长10～15 s[[S1]];
- **综合废品率**：配套2150吨压铸单元的自动生产整线综合废品率低于2%[[S17]]。

## 优势与局限

### 相比人工镶件放置的优势

自动镶件输送装置相比传统人工放置具有多方面显著优势[[S3,S7,S6,S1]]：

- **提高放置精度与一致性**：消除人工操作带来的节拍波动和定位一致性差的问题；
- **提升生产效率**：通过稳定的放置节拍显著提升整线生产效率，综合生产效率可提升约50%；
- **改善劳动安全**：将操作人员从高温、高噪音、高铝液飞溅风险的作业环境中脱离，降低体力劳动强度；
- **降低人力依赖**：操作人员占用率可从约96%下降至57%左右[[S6]]。

### 局限性与约束

自动镶件输送装置也存在以下局限[[S12,S22]]：

- **异形镶件适配性较差**：特殊形状、复杂姿态的非标准结构镶件需定制专属定位工装才能稳定供料；
- **初始投资高**：首次购置的投入成本远高于人工放置方案，仅适合大批量、连续化的稳定生产场景；
- **维护要求**：需定期对送料导轨、定位传感器、夹爪部件进行维护保养；
- **换产成本高**：小批量多品种换产场景下的调试成本较高。

## 选型原则

镶件输送装置的选型应依据生产规模和产品特性进行决策[[S12]]：

| 生产场景 | 推荐方案 |
|----------|----------|
| 小批量、多品种试生产 | 优先选择兼容多类镶件的柔性机器人式镶件输送单元 |
| 大批量、单一品种量产 | 可选择定制化振动盘+定向输送的专用镶件送料装置 |
| 现有自动化水平不足的压铸单元 | 可升级加装独立式镶件输送装置，无需对原有整线控制系统做大规模改造 |

## 应用案例

某汽车零部件厂配置的2150吨锁模力全自动压铸生产单元，搭载专用镶件送料装置（insert feeder），所用镶件为钢制，用于批量生产带大尺寸轴承孔的铝合金变速箱壳体。该单元包含以下核心配置[[S17]]：

- 2150吨合模力压铸机；
- 自动金属浇注装置；
- 保温炉；
- 取件与镶件加载双钳口机器人（额定负载120 kg）；
- 镶件输送装置（镶件经切边回收后通过装置输送至机器人取料位，路径上设有温控干燥区域以去除冷却后残留湿气）；
- 红外检测站、淬火槽、切边压力机。

该单元小时产量可达96件，综合生产效率相比纯人工放置方案提升约50%，操作人员占用率从96%下降至57%[[S6]]，废品率低于2%，完整循环时间为120 s[[S17,S1]]。
