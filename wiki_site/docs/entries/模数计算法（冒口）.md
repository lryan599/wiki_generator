## 概述

模数计算法（冒口）（modulicalculation method）是铸造工艺中一种经典的冒口设计计算方法，通过计算模数来确定冒口尺寸。模数（modulus，M）亦称换算厚度，定义为铸件或冒口的体积V与散热表面积A的比值（M = V/A），模数越大，对应的凝固时间越长[[S15,S31]]。该方法属于常规冒口设计计算方法序列，是铸件补缩系统核心设计方法之一，本质上是基于顺序凝固原则的凝固过程定量控制手段[[S46,S48]]。

中文别名包括模数法、冒口模数法、热模数法、凝固模数法、换算厚度法[[S40,S46]]。权威英文术语涵盖modulicalculation method（GB标准官方英译）、modulus method、modulus method of risering、modulus calculation method for risers、riser design by modulus、casting modulus、modulus of riser等[[S15,S22,S38]]。

## 核心原理

### Chvorinov凝固法则

模数计算法的理论基础是Chvorinov凝固法则（平方根定律）。该法则指出，铸件的凝固时间t与模数M的平方成正比，数学表达式为：

\[ t = \left(\frac{M}{k}\right)^2 \]

其中k为凝固系数，与铸件材质、铸型材料、浇注温度等因素有关[[S46,S5,S6]]。

### 补缩条件的模数判定

冒口实现有效补缩需满足两个基本条件：一是冒口的凝固时间必须大于或等于铸件被补缩部位的凝固时间；二是冒口必须有足够的金属液补充铸件的体积收缩[[S46,S28]]。

在相同浇注工况下，可以近似认为冒口、冒口颈及铸件被补缩部位的凝固系数相等（kᵣ = kₙ = kₒ）。因此，将补缩所需的顺序凝固条件“冒口凝固时间 ≥ 冒口颈凝固时间 ≥ 铸件被补缩部位凝固时间”，简化为直接对比模数的判定规则：

\[ M_r \geq M_n \geq M_c \]

其中Mᵣ为冒口模数，Mₙ为冒口颈模数，Mₒ为铸件被补缩部位模数[[S46,S9,S3]]。

### 模数扩大系数

在冒口补缩过程中，冒口内金属液逐渐减少，顶面形成缩孔使散热表面积增大，导致冒口模数实际呈逐渐减小的趋势；相反，铸件因获得炽热金属液补充，模数相对有所增大。因此需引入模数扩大系数以保证补缩充分[[S46,S5]]。根据试验，冒口模数相对减小值约为原始模数的17%[[S46,S5]]。对于碳钢、低合金钢常规普通冒口场景，工程上通常设置为：

- 冒口模数扩大系数：fᵣ = 1.2
- 冒口颈模数扩大系数：fₙ = 1.1

即Mᵣ = 1.2Mₒ，Mₙ = 1.1Mₒ[[S46,S9,S42]]。扩大系数过大将导致冒口尺寸增大，浪费金属，并加重铸件热裂和偏析倾向[[S5]]。

## 计算步骤

模数法设计冒口的完整标准流程包括以下5个步骤[[S45,S42,S46]]：

| 步骤 | 内容 |
|------|------|
| ① | 划分铸件各补缩区域，计算各区域的铸件热节模数Mₒ |
| ② | 选定冒口类型，按模数扩大系数计算冒口颈模数Mₙ及冒口模数Mᵣ |
| ③ | 根据冒口安装位置形状及砂箱结构确定冒口形状，对照标准冒口尺寸数据表查取具体参数 |
| ④ | 对照材质的冒口有效补缩距离范围，校核所需冒口数量 |
| ⑤ | 结合铸件体收缩率、冒口补缩效率校验冒口补缩能力 |

### 关键参数

冒口颈尺寸参数的经验取定[[S45,S46]]：

- 冒口颈长度：L = 2.4Mₒ
- 矩形截面冒口颈：满足公式 ah/[2(a+h)] = Mₙ（a为颈宽，h为颈高）
- 圆形截面冒口颈：直径 d = 4Mₙ

补缩能力校核公式[[S9,S42,S3]]：

\[ \varepsilon(V_c + V_r) + V_e \leq V_r \cdot \eta \]

其中ε为金属体收缩率，Vₒ为铸件被补缩部分体积，Vᵣ为冒口体积，Vₑ为型壁移动扩大的体积，η为冒口补缩效率。

### 热节模数计算方法

复杂铸件通常由简单几何体与相交节点构成，掌握基础几何体及其节点的模数计算方法是关键[[S28,S45]]。工程上常用两种方法计算热节模数[[S45,S8]]：

- **热节圆当量板（或杆）法**：将热节部位等效为以实测热节圆直径为厚度的板或杆件代入公式计算
- **一倍厚度法**：取热节相邻一倍壁厚范围的区域作为计算边界

两种方法所得模数值偏差在工程允许范围内，均可满足工艺设计精度要求[[S45,S8]]。

## 各类冒口的模数比

### 通用冒口

铸钢件重力铸造场景下，不同类型冒口的推荐模数比如下表所示[[S46,S28,S14,S9]]：

| 冒口类型 | 模数比关系 |
|----------|------------|
| 明顶冒口 | Mᵣ : Mₒ = (1~1.2) : 1 |
| 暗侧冒口 | Mₒ : Mₙ : Mᵣ = 1 : 1.1 : 1.2 |
| 钢液经侧冒口进入型腔 | Mₒ : Mₙ : Mᵣ = 1 : (1~1.03) : 1.2 |

### 发热冒口

发热冒口通过氧化还原反应释放热量延缓冒口凝固，等效于放大普通冒口模数。发热冒口模数Mᵣ₂与普通冒口模数Mᵣ的换算关系为[[S14,S13]]：

\[ M_{r2} = M_r / 1.43 \]

发热冒口补缩效率可达66.7%[[S14]]。

### 保温冒口

采用保温冒口时，由于冒口凝固系数低于铸件凝固系数，模数扩大系数f可降至0.8，从而缩小冒口体积，提升铸造工艺出品率[[S1,S46]]。保温冒口尺寸的确定方法为[[S11,S34]]：

1. 求保温冒口有效模数：M_R,e = 1.2Mₒ
2. 求模数增大系数：e = kₘ / k_R,ₘ（分别为铸型与保温材料的传热系数比值，常规e值在1.2~2.0区间）
3. 求保温冒口实际模数：M_R = M_R,e / e

### 铸铁件的特殊性

铸铁件（灰铸铁、球墨铸铁）具有石墨化膨胀带来的自补缩能力，相比铸钢件可选用更小的模数比，无需严格沿用1.2倍的模数基准放大系数。《中国材料工程大典》给出了球墨铸铁和灰铸铁收缩模数法的专门设计参数系列[[S41,S16,S51]]。

## 模数法变体

### 直接模数法

直接模数法是模数法的基础形式，仅通过计算铸件补缩部位模数，按对应冒口类型的模数比例直接确定冒口尺寸，不额外校核补缩液量。适用于简单几何体的中小型铸件快速冒口设计[[S30,S21]]。

### 模数-体积法

模数-体积法是直接模数法的延伸，在确定冒口模数后，额外引入冒口体积、铸件补缩体积、金属体收缩率、冒口补缩效率等参数，通过方程校验冒口补缩能力是否满足收缩需求。包含周界商法、三次方程法等子类，计算精度高于基础直接模数法，适合中大型重要铸钢件的冒口设计[[S35,S23]]。

### Niyama判据关联应用

将模数法计算得到的冒口初始布置方案，结合铸造数值模拟中的Niyama判据（G/√R ≤ M，其中G为局部温度梯度，R为凝固冷却速度）进行二次校验，判定局部区域是否存在缩松倾向，据此修正冒口位置和大小。适合航空、核电等要求较高的关键铸件的冒口优化设计[[S26,S24]]。

## 压铸场景下的特殊性

高压压铸与重力铸造在冒口应用上存在显著差异[[S49,S29]]：

| 项目 | 重力铸造（砂型/金属型） | 高压压铸 |
|------|--------------------------|----------|
| 补缩机制 | 顺序凝固下的自然补缩 | 7~140 MPa增压保压强制补缩 |
| 传统冒口 | 设置明/暗冒口，严格按模数法设计 | 几乎不设置传统大体积冒口 |
| 模数概念的应用 | 直接用于计算冒口尺寸 | 以“残余熔体模数”形式评估缩松缺陷风险 |
| 设计目标 | 确定冒口尺寸与数量 | 识别高缩孔风险区域，指导溢流槽/集渣包位置及尺寸优化 |

在高压压铸工艺中，极快的冷却速度和非均匀热交换条件不符合模数法原始假设，无法直接执行常规模数法的冒口尺寸计算流程[[S46,S48,S29]]。残余熔体模数越大，对应区域的缩孔缺陷风险越高[[S24,S29]]。

## 适用条件与局限性

模数计算法适配所有采用顺序凝固原则的铸造工艺，涵盖砂型铸造、金属型铸造、低压铸造、部分重力压铸场景；不适用于高压压铸[[S46,S48]]。

模数计算法的核心假设条件包括[[S3,S46,S28]]：

1. 同一浇注工况下，铸件被补缩部位、冒口、冒口颈的凝固系数K完全相等，三者与铸型之间的界面热交换特性一致
2. 基于Chvorinov平方根定律，凝固时间t与模数M的平方成正比

该方法在实际应用中存在以下局限性[[S30,S33]]：

- **复杂几何形状**：需将复杂铸件拆解为多个简单几何体分别计算模数，不同设计人员的拆解方法不统一，导致最终计算模数值存在差异，降低设计可重复性
- **非均匀热边界**：在设置冷铁、局部保温层等情况下，直接应用原始模数法未考虑局部热效应影响，计算得到的冒口尺寸可能偏小，需要额外修正
- **简化模数法精度不足**：基于经验的设计方法，计算结果的准确性有限，对于要求较高的铸件需要采用周界商法等更精确的方法
