## 概述

无焦冲天炉（cokeless cupola），又称无焦化冲天炉、天然气冲天炉或油气冲天炉，是指完全不以焦炭作为燃料的冲天炉[[S3,S15]]。该炉型保留冲天炉竖式连续加料与出铁的核心结构特征，属于传统焦炭冲天炉按燃料分类的一个改进子类，而非独立于坩埚炉或反射炉的全新熔化炉类别[[S2,S22]]。最早的工业化连续运行装置于1970年在英国投入生产[[S18,S19]]。

## 定义与标准称谓

国家标准 GB/T 25370-2020《铸造机械 术语》明确定义无焦冲天炉为“不以焦炭作为燃料的冲天炉，通常采用天然气等作为燃料”[[S3]]。行业标准进一步补充：无焦冲天炉不用焦炭而用石油、天然气、煤气作燃料；炉料加在由置于水冷钢管炉条上的多层耐热陶瓷球形成的、用以过热铁液的炉床上，炉条至炉底间设有增碳燃料喷嘴，常用工频前炉实现铁液过热与成分调整[[S15,S21]]。

中文别名包括无焦化冲天炉（通用口语变体），以及按燃料细分的天然气冲天炉、燃油冲天炉、油气冲天炉[[S23]]。英文标准术语为 cokeless cupola 或 cokeless cupola furnace，等价变体有 coke-less cupola furnace、coke-free cupola；燃料细分场景下 gas-fired cupola 对应燃气冲天炉，oil-fired cupola 对应燃油冲天炉[[S19,S9]]。

## 分类归属与核心区别

无焦冲天炉归属于冲天炉大类，按燃料分类为区别于焦炭冲天炉的独立分支，保留竖式竖炉、连续加料的核心特征[[S2,S3]]。与传统焦炭冲天炉的核心区别如下：

- **取消底焦层**：完全取消焦炭底焦，以多层耐热陶瓷球构成的耐火床替代，作为铁液过热的热交换介质[[S15,S18]]。
- **无硫增量路径**：取消焦炭可消除焦炭引入的增硫问题，若配合无硫气体燃料和高纯石墨增碳剂，铁液硫含量仅来自入炉金属料[[S25]]。
- **简化烟气净化**：仅需常规湿法捕集装置即可满足严苛的颗粒物排放标准，无需焦炭冲天炉常见的高成本复杂除尘系统[[S19]]。
- **炉衬磨损降低**：炉衬磨损速率仅为同吨位燃焦冲天炉的约三分之一[[S13]]。

## 结构与组成

### 整体结构

水冷长炉龄天然气无焦冲天炉是当前主流结构型式，包含加料口、预热段、水冷熔化段、耐火球垫层、水冷炉栅、天然气喷嘴、连续出铁槽、天然气燃烧室及前炉等核心部件。喷嘴布置在水冷炉栅下方的方案结构最简单、修炉最方便，应用最广泛；另有两种变体：在炉外设置独立燃烧室通过火道导入炉气，或配置卧式固定前炉[[S23]]。

此图为《新编铸造技术数据手册》所载水冷长炉龄天然气无焦冲天炉结构示意图，标注了加料口、预热段、水冷熔化段、耐火球垫层、水冷炉栅、连续出铁槽、天然气喷嘴、天然气燃烧室和卧式前炉（熔池）共九个核心部件[[S7,S23]]。

### 水冷炉栅

水冷炉栅是无焦冲天炉的关键承力与支撑部件，采用专利设计的鳍片水冷管结构以降低传导热损失。熔化率8 t/h以下的小型炉子推荐使用耐火涂层水冷管以避免过大热损失，耐火涂层平均使用寿命约60小时熔化时长；裸水冷管平均寿命可达数周[[S5]]。

### 燃烧器与喷嘴

燃烧器分为有焰喷嘴与无焰喷嘴两类，核心设计参数包含燃料流量、火焰长度、火焰角度。燃烧器在外部完成燃烧，炉栅下方区域仅作为高温烟气分布区，不进行燃烧反应[[S18,S23]]。

### 耐火球垫层

耐火球垫层材质为碳素与硅酸盐复合成分，耐火度高，承担均热、传热与支撑炉料作用。铺设初始深度为450~600 mm，耐火球消耗量为金属熔化量的1.5%~3%，具体消耗随目标铁液温度升高而增大[[S5,S10]]。

## 工作原理与热工特性

### 热能来源与热交换

无焦冲天炉的热量主要来源于甲烷等气体燃料的完全燃烧，燃料燃烧比≈100，远高于燃焦冲天炉的0.4~0.6。因不存在焦炭，炉内基本不存在CO₂还原为CO的反应，炉气中CO含量几乎为零，尾气仅含物理显热而不含化学潜热[[S16,S1]]。

铁液过热过程95%由炉气辐射完成，不存在传统焦炭层与铁液之间的对流换热[[S16]]。

### 燃烧学参数对比

天然气与焦炭冲天炉的燃烧学特性对比如下[[S1]]：

| 对比项目 | 天然气冲天炉 | 燃焦冲天炉 |
|----------|-------------|------------|
| 燃烧比 | ≈100 | 0.4~0.6 |
| 完全燃烧时炉气CO₂体积分数(%) | 9.5 | 21 |
| 完全燃烧时炉气H₂O(g)体积分数(%) | 19.0 | 0 |
| 尾气余热形式 | 主要物理显热，基本无化学潜热 | 主要化学潜热，兼有物理显热 |

### 冶金反应体系

无焦天然气冲天炉内的冶金反应与焦炭冲天炉显著不同，炉气中的CO₂与H₂O(g)与金属炉料和CO发生如下反应[[S12]]：

| 反应名称 | 反应式 | 热效应 | 说明 |
|----------|--------|--------|------|
| CO₂与金属的氧化反应 | CO₂+M→MO+CO | 吸热 | 炉气中CO₂使金属烧损 |
| H₂O(g)与CO反应 | H₂O(g)+CO=H₂+CO₂ | 弱放热 | 水煤气反应是炉气含H₂的来源 |
| H₂与MO的还原反应 | H₂+MO→M+H₂O | 放热 | 空气过量系数影响炉气氧化性 |

## 冶金反应与元素烧损

### 空气过量系数的影响

空气过量系数是控制元素烧损的关键参数[[S20,S16]]：

| 空气过量系数 | Si烧损率(%) | Mn烧损率(%) |
|-------------|-------------|-------------|
| 1 | 15~20 | 18~20 |
| 1.2 | ≈40 | ≈60 |

应在保证天然气完全燃烧的前提下，尽可能降低空气过量系数，同时缩短铁液滴与氧化性炉气的接触时间[[S16]]。

### 主要元素增减

天然气无焦冲天炉与焦炭冲天炉的元素烧损对比如下[[S6]]：

| 元素 | 燃焦冲天炉趋向 | 燃焦增减率(%) | 天然气冲天炉趋向 | 天然气增减率(%) |
|------|--------------|-------------|----------------|--------------|
| 铁 | - | 0.5~1.0 | - | 1~2 |
| 碳 | + | 3~8 | - | 3~10 |
| 硅 | - | 15~20 | - | 18~25 |
| 锰 | - | 20~25 | - | 20~25 |
| 硫 | + | 40~80 | ± | 0 |
| 磷 | ± | 0 | ± | 0 |

使用低硫燃油作为燃料时，铁液硫增量仅为0.02%~0.03%，球化处理后残余硫低于0.01%。采用无硫天然气/丙烷和高纯石墨增碳剂时，铁液不会额外增硫，无需炉外脱硫即可直接进行球化处理[[S25]]。

## 工艺操作参数

### 典型工艺参数

已验证的商用无焦冲天炉熔化率规格包括4 t/h、5 t/h、7 t/h、10 t/h。以英国Hayes Shell Cast的5 t/h炉为例，运行参数如下[[S10,S18]]：

| 参数 | 数值 |
|------|------|
| 吨铁能耗 | ≈3100 MJ |
| 丙烷输入 | 1477 MJ/t |
| 低硫燃油输入 | 1583 MJ/t |
| 增碳剂消耗 | 金属熔化量的1%~3% |
| 耐火球消耗 | 金属熔化量的1.5%~3% |

### 出铁温度与铁液特性

使用低硫燃油时出铁温度可稳定控制在1450~1480 ℃，氧富集工况可进一步提升。炉内全程保持弱还原性气氛，铁液流动性和温度稳定性优于传统焦炭冲天炉，可采用比常规浇注温度低约300 ℃的条件完成复杂薄壁球墨铸铁件浇注[[S17,S10]]。

### 操作流程要点

- 首次点火后约15分钟即可出铁；停炉时停止加料30分钟后调低烧嘴功率排空铁液[[S13]]。
- 炉内保持约1小时熔化量的存料即可维持连续稳定运行[[S13]]。
- 首次出渣在熔炼启动后2.5~3小时，后续出渣间隔约1.5小时，也支持连续排渣[[S13]]。
- 停炉后喷水冷却剩余耐火球以防过度氧化，后续开炉无需更换全部床层[[S13]]。

## 优势与环保特性

**冶金质量**：铁液硫含量极低，使用无硫燃料与高纯石墨增碳剂时无额外增硫，可直接生产球墨铸铁[[S19,S25]]。

**排放控制**：加料口上方始终有火焰形成“后燃室”效应，无可见烟气排放；仅需简单湿法捕集器即可将颗粒物排放降至全球最严苛标准以下，无需昂贵除尘设备[[S13,S19]]。

**炉衬寿命**：炉衬磨损速率为传统焦炭冲天炉的1/3，无焦炭灰分形成熔渣总量更少[[S13]]。

**双联熔炼**：无焦冲天炉作为初级熔化设备与中频感应电炉组成双联工艺，可同时利用冲天炉低能耗熔化优势与电炉高精度过热及成分微调优势，降低整体运行成本并提升电炉处理产能[[S10,S26]]。

## 局限与操作难点

- **高比例废钢熔炼受限**：冷鼓风工况下，炉料中废钢比例不宜超过50%。每额外添加1%增碳剂（占金属熔化量），出铁温度约下降50 ℃，高比例废钢熔炼需配套氧富集系统补偿温度损失[[S25]]。
- **元素烧损较高**：炉气氧化性强，硅、锰等元素烧损率高于焦炭冲天炉，需严格控制空气过量系数[[S16,S20]]。
- **耐火球消耗成本**：耐火球为消耗性材料，熔化每吨金属消耗1.5%~3%，直接影响运行成本；高质量耐火球早期主要依赖进口[[S10,S23]]。

## 应用场景

无焦冲天炉已在英国、伊朗、德国、埃及等多国铸造厂实现工业化应用，适用于灰铸铁、球墨铸铁、可锻铸铁的批量熔炼[[S19,S18]]。部分可锻铸铁生产线采用无焦冲天炉配合电热前炉的双联配置完成连续生产[[S18]]。在焦炭进口/高价地区或环保排放标准严苛的场合，无焦冲天炉具有显著的运行成本优势[[S10]]。
