## 概述

热传导（heat conduction，亦称导热）是热量传递的三种基本方式之一，指物体各部分之间不发生相对位移时，依靠分子、原子及自由电子等微观粒子的热运动发生碰撞来传递内能的传热方式[[S28,S23,S44]]。热平衡发生的条件是物体体系内部不存在温度梯度，各处温度均匀一致[[S28,S23]]。

在压铸、焊接、选区激光熔化（SLM）及铸造凝固等热加工过程中，热传导往往是凝固后期和冷却阶段的主导传热机制，直接影响温度场分布、冷却速率、微观组织形态及缺陷形成[[S19,S56,S3,S18]]。

## 定义与基本机理

### 核心定义

热传导本质上是依靠介质微观粒子（分子、原子、自由电子）的碰撞传递内能的传热方式，**无法在绝对真空无介质的环境中直接传递热量**，该场景下的跨空间热量传递仅能依靠热辐射实现[[S51,S43,S50]]。

对于金属及合金，热传导的物理机制具有明确的载流子贡献构成：热导率主要由**自由电子载体贡献主导**（κₑ），声子（晶格振动波）贡献占比较小（κ_g），光子对热导率的贡献仅在极高温度下才会显现。合金的热导率可由下式描述[[S11]]：

κ = κₑ + κ_g（式1-1）

式中 κₑ 为电子热导率贡献项，κ_g 为声子热导率贡献项，单位为 W·m⁻¹·K⁻¹[[S11]]。

### 流体中的特殊性

需注意，**不存在纯粹的流体热传导**：即使处于完全静止的流体介质中，温度梯度引发的密度差会自然驱动流体产生微流动，进而出现自然对流效应，实际流体场景中热传导始终与自然对流耦合发生[[S51]]。

## 核心数学描述

### 傅里叶导热定律

傅里叶定律是热传导的基本定量定律，由法国科学家傅里叶于1822年提出[[S7]]。

**一维标量形式**：稳态条件下，单位时间通过给定截面的热流量 φ 正比于垂直于该截面方向上的温度变化率和截面面积 A，热量传递方向与温度升高方向相反[[S7,S28,S23]]：

φ = −λA · (dT/dx)

式中 φ 为热流量（W），λ 为材料热导率（W/(m·K)），A 为导热截面积（m²），dT/dx 为温度梯度（K/m）[[S28]]。

**热流密度形式**：单位时间通过单位面积的热流量[[S22,S44]]：

q = −λ · (∂T/∂x)（单位：W/m²）

**全矢量形式**：适用于任意三维各向同性介质[[S22,S6]]：

**q** = −λ · grad T = −λ(∂T/∂n)**n**

式中 grad T 为温度梯度矢量，**n** 为等温面法线方向单位矢量。

### 三维非稳态导热微分方程

在均质各向同性材料、存在内热源 Q 的条件下，三维导热微分方程（即非稳态热传导控制方程）为[[S28,S31]]：

λ(∂²T/∂x² + ∂²T/∂y² + ∂²T/∂z²) + Q = ρC_p · (∂T/∂t)

式中 ρ 为材料密度（kg/m³），C_p 为定压比热容（J/(kg·K)），T 为温度（K），t 为时间（s）。

## 压铸材料热导率

### 压铸合金与模具钢典型值

热导率 λ 是衡量材料导热能力的核心参数，其数值取决于材料种类、成分及温度[[S22,S44]]。以下是压铸领域常用材料的热导率实测数据：

| 材料 | 热导率 | 单位 | 来源 |
|------|--------|------|------|
| BS1004B 锌合金 | 1088.6 | W/(m·K) | [[S9]] |
| Zn96Al4 压铸锌合金 | 约98.8 | W/(m·K) | [[S17]] |
| Al86Si10Cu4 压铸铝合金 | 约96.5 | W/(m·K) | [[S17]] |
| 5%Cr 压铸模具钢 | 约31.0 | W/(m·K) | [[S17]] |
| 铝基压铸模具材料 | 约146.5 | W/(m·K) | [[S17]] |
| H13 压铸模具钢 | λ = 24.5607 + 1.44×10⁻³ t | W/(m·K)（t 为摄氏温度） | [[S9]] |

**H13模具钢热导率随温度变化**的表达式 λ = 24.5607 + 1.44×10⁻³ t 可在压铸热模拟中直接赋值使用[[S9]]。

### 热导率测试方法

压铸合金的热导率测量可采用[[S20,S54,S34]]：
- **Hot Disk 方法**：直接测量热导率
- **激光闪射法**（Laser/Xenon Flash）：测量热扩散率后，结合比热和密度计算热导率
- **Wiedemann-Franz 定律换算**：通过电导率间接获取热导率，公式为 λ = LT/ρ，其中 L 为洛伦兹数[[S20]]

压铸合金的国际标准涵盖：ASTM B86（锌及锌铝合金压铸件）、ASTM B94（镁合金压铸件）、ASTM B85（铝合金压铸件）[[S20,S54]]。

## 压铸过程中的热传导

### 充型与凝固阶段的传热方式演变

高压压铸（HPDC）过程中，热量的传递方式随工艺阶段而变化[[S19,S56,S16]]：

- **充型阶段**（金属液高速流动填充型腔）：以**对流换热为主**
- **凝固阶段**：热传导占比持续升高；型腔完全充满进入保压阶段、铸件表层形成连续固态壳层后，**热传导成为绝对主导的传热方式**
- **凝固后期**：即使铸件收缩产生界面间隙，热传导仍占全部传热比例的**70%以上**[[S19,S56]]

压铸过程的温度场求解以非稳态热传导偏微分方程为核心理论基础，傅里叶定律广泛用于压铸CAE热分析模块的铸件凝固温度场仿真、模具热循环温度分布仿真、冷却水道热交换计算及铸件-模具界面热传导计算等场景[[S40,S29]]。

### 界面热传导与界面换热系数

#### 界面热阻构成

压铸金属液与模具之间的界面热传导核心阻力由两部分组成[[S8,S32,S55]]：
1. **模具涂层自身的导热热阻**
2. **粗糙铸件表面与模具涂层之间被困空气层的导热热阻**

其中，**空气层的热阻占总界面热阻的60%以上**，通过接触点的直接导热占比可忽略[[S8,S32]]。对于300μm厚的绝缘白涂层，仅约33%的隔热性能来自涂层本身，约67%来自界面空气层；对于100μm白涂层，涂层自身仅提供约15%的总热阻[[S55]]。

#### 界面换热系数（IHTC）典型值

| 接触界面 | IHTC 典型值（W/(m²·K)） | 来源 |
|----------|--------------------------|------|
| AlSi9Cu3Fe铝合金压铸峰值 | 42000 | [[S10]] |
| AZ91镁合金压铸平均值 | 约19624 | [[S10]] |
| 常规铝-模具接触区间 | 1000~79400 | [[S10]] |
| 铝液-压室 / 铝液-冲头 | 2000 | [[S13]] |
| 铝液-压铸模 | 1000 | [[S13]] |

> **表注**：IHTC数值受模具初始温度、涂层类型与厚度、接触压力、铸件厚度等工艺参数显著影响[[S15,S46]]。

#### IHTC随压铸进程的演化

界面换热系数随压铸进程分为四个典型阶段[[S46]]：
1. **初始上升阶段**：IHTC迅速增大至峰值，峰值主要受模具初始表面温度影响
2. **高值维持阶段**：IHTC保持较高水平
3. **快速下降阶段**：IHTC随铸件固相分数升高而降低
4. **低值稳定阶段**：IHTC趋于稳定

#### 固-固界面接触状态的影响

铸造过程中界面接触可分为三个阶段[[S38]]：
- **完全接触阶段**：固-固热传导占主导，接触面积大，热阻极小
- **部分接触阶段**：仅局部区域实现热传导，气隙区域以辐射、气体导热为主
- **完全分离阶段**：气隙占据全部界面，固-固热传导几乎消失

## 热传导对铸件质量的影响

### 冷却速率与晶粒尺寸

界面热传导效率越高，铸件冷却速率越快，晶粒尺寸随冷却速率升高而细化。经验拟合公式为[[S12,S35,S49]]：

d = 72.5 × V_c⁻⁰·¹¹ （R² = 0.99）

式中 d 为平均晶粒尺寸（μm），V_c 为冷却速率（℃/s）。

### 温度场不均与热节

若铸件不同区域热传导效率差异过大，会导致局部区域存在明显热集中，形成**热节**。热节位置冷却速率显著低于周边区域，成为最后凝固的位置[[S36,S14,S47]]。

### 缩松缺陷

当铸件薄壁位置热传导过快、先完全凝固时，会阻断后续增压系统对厚大热节位置的金属液补流路径。热节区域的凝固体积收缩无法被补充，最终在该位置形成**分散的缩松或缩孔缺陷**[[S36,S19]]。

### 热裂与残余应力

铸件不同区域热传导速率差异过大时，局部位置会出现超过合金固相线对应强度阈值的热应力集中，在合金的**凝固脆性区间**直接诱发热裂纹。不同部位不均匀冷却引发的非同步收缩会在铸件内部残留显著残余应力，大幅降低构件服役性能[[S12]]。

## 影响因素与局限性

### 模具涂层

不同模具涂层对界面换热系数的影响存在显著差异[[S39,S55]]：
- **石墨基涂层**的界面换热系数**显著高于**绝缘类涂层
- 同类型涂层厚度从100μm提升至300μm时，界面换热系数降低约**50%**[[S55]]
- 铝合金低压铸造中，涂层厚度从0增加至300μm时，界面换热系数呈**线性下降**趋势[[S37]]

### 接触压力

2. **呈幂指数正相关**关系[[S38]]：
- 铝合金铸造中压力由7MPa提升至14MPa时，热传导效率提升**20%**
- 压力提升至原有3倍时，热传导效率仅提升**40%**
- 压力远超过材料屈服强度后，实际接触面积占比趋于稳定，热传导效率不再随压力升高明显增加[[S38]]
- 对于**薄壁铸件**，压力对IHTC的提升作用几乎可忽略；仅在较厚铸件位置，接触压力对IHTC峰值的影响占主导[[S15]]

### 材料孔隙率

材料内部孔隙率越高，整体等效热传导效率越低[[S52,S1]]。Sih 模型可定量描述粉末态材料有效热导率与孔隙率的对应关系[[S52]]。实测表明，多孔介质中直径大于50μm的大孔隙虽然数量占比仅为17%，但体积贡献占比达78.64%，其对材料整体热传导效率的削弱作用远高于小孔隙[[S1]]。

### 数值模拟的局限

工程中大部分复杂非稳态热传导问题无法获得解析解，仅能通过数值离散得到近似解。数值模拟的精度受网格划分、边界条件简化程度的制约，存在不可避免的误差[[S21,S5]]。

## 关联工艺中的热传导

### 焊接

焊接工艺冷却阶段，热传导是**固体内部的主导传热方式**，直接决定焊接接头的温度分布和冷却速率，是后续焊接热应力、相变及残余变形数值模拟的核心计算基础[[S3,S51]]。

### 选区激光熔化（SLM）

在高能激光移除后的熔池冷却凝固阶段，热传导为**熔池热量向固态基板传递的主导传热机制**，直接决定冷却速率、微观组织形态、残余应力水平与零件成形质量。SLM过程的三维瞬态温度场控制方程以非稳态热传导为核心建立[[S18,S27]]。

### 燃油凝固

燃油凝固后期，当燃油温度降至冷凝点以下、内部自然对流完全消失后，热传导成为**唯一主导**的传热方式，凝固进程显著放缓。实测数据显示，该阶段剩余约10%的液态燃油完成凝固所需时间超过前期90%燃油凝固的总耗时[[S41]]。
