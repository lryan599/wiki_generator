## 概述
AlSi9Cu3合金是一种属于Al-Si-Cu系的压铸铝合金，在欧美汽车工业中占据主流地位，也是目前车用压铸发动机缸体、壳体等功能件最常用的材料之一。欧洲标准（EN 1706）规定其正式牌号为 **EN AC-46000**（数字牌号），符号命名为 **EN AC-AlSi9Cu3(Fe)**；在中国国家标准（GB/T 15115）中对应的压铸牌号为 **YL113**[[S29,S4,S37,S39]]。该合金通过较高的铜、硅含量保证了良好的铸造流动性与力学性能，同时允许一定量的铁以改善脱模性，广泛用于制造传动系统壳体、底盘重载部件及结构件[[S23,S41]]。

## 分类与牌号
AlSi9Cu3属于铸态Al-Si-Cu系压铸铝合金，在EN 1780-1分类体系中归属 **46xxx** 系列（AlSi9Cu）。全球主要标准中的对应牌号如下[[S39,S38,S16]]：

| 标准体系 | 牌号 / 代号 |
|----------|--------------|
| 欧洲 EN 1706 | EN AC-46000 / EN AC-AlSi9Cu3(Fe) |
| 中国 GB/T 15115 | YL113 |
| 美国 ASTM B85 | A380（成分极为相近，但控制标准有差异） |
| 日本 JIS H 5302 | ADC12（成分范围有重叠） |

工业生产中，AlSi9Cu3、A380、ADC12三种合金的铝液包常可相互临时调用，仅需挂牌标识区分[[S24]]。

## 化学成分
根据多个标准及研究文献，AlSi9Cu3(Fe)的典型主元素含量范围如下表所示[[S3,S10,S38]]。

| 元素 | 含量范围 (wt.%) | 备注 |
|------|-----------------|------|
| Si   | 8.0 – 11.0 | 保证流动性与强度 |
| Cu   | 2.0 – 4.0 | 形成Al₂Cu强化相 |
| Mg   | 0.10 – 0.55 | EN标准要求0.15~0.55，部分资料0.1~0.5 |
| Fe   | ≤1.30 (通常0.6~1.10) | 改善脱模，过高会形成脆性β-Fe相 |
| Mn   | ≤0.55 | 用于变质Fe相形态 |
| Zn   | ≤1.2 | 控制时效敏感性 |
| Cr   | ≤0.15 | 部分标准允许，细化富Fe相 |
| Al   | 余量 | — |

欧标EN AC-46000对Fe含量的要求更为严格，通常控制在0.6%~1.10%，以平衡脱模性与力学性能[[S38]]。美标A380则在Zn含量上放宽至≤2.9%，Mg上限压至≤0.01%，优先追求最佳流动性[[S38]]。日标ADC12的Si含量区间为9.6%~12%，Cu 1.5%~3.5%[[S9]]。

## 工艺性能与工艺参数
采用压铸行业通用的1～5级评分制（5为最优），AlSi9Cu3的综合工艺性能评级如下[[S15,S13,S28]]：

| 工艺性能 | 评级 | 说明 |
|----------|------|------|
| 流动性 | 4~5 | 优异，适合复杂腔体充填 |
| 热裂倾向 | 3~4 | 较低 |
| 气密性 | 3~4 | 良好，高气密件常需高真空 |
| 切削加工性 | 4 | 优良 |
| 抛光性能 | 2~3 | 中等 |
| 焊接性 | 2 | 较差，一般仅限电阻焊或特殊钎焊 |

推荐的压铸工艺参数范围[[S12,S5,S17,S14]]：

| 参数 | 推荐值 / 范围 |
|------|---------------|
| 铝液浇注/保温温度 | 620~720°C，工业量产常采用680±5°C[[S5,S17]] |
| 模具工作温度（预热/稳态） | 180~250°C，稳定区间200~230°C[[S14,S27]] |
| 压射比压 | 50~100 MPa |
| 第一阶段柱塞速度 | 0.2~0.8 m/s |
| 第二阶段柱塞速度 | 1.2~3.8 m/s |
| 内浇口充填速度 | 30~60 m/s |

优化工艺参数可显著提高铸件密度，降低内部孔隙率[[S12]]。对于壁厚变化较大的铸件，不同壁厚的推荐充填时间可参考同系列合金的充型曲线[[S18]]。

## 微观结构
典型的铸态组织为亚共晶Al-Si结构，特征如下[[S40,S36,S25]]：
- **α-Al基体**：呈树枝晶形态。在高压压铸快冷条件下晶粒显著细化；部分α-Al来自压室预结晶，尺寸较粗大，另一部分为型腔快冷形成的细小近球状晶。
- **共晶Si**：经Sr变质处理后呈细小纤维状分布，改善塑性。
- **富Fe金属间化合物**：主要包含针状/片状的β-Al₅FeSi相（脆性）和块状/汉字状的α-Al₁₅(Fe,Mn)₃Si₂相。加入适量Mn（m(Mn)/m(Fe)≈0.67~0.83）可将针状β-Fe转变为对基体割裂作用较小的α-Fe相[[S1,S22]]。
- **含Cu相**：θ-Al₂Cu以块状或共晶形态分布于晶界和共晶区，是主要强化相。当Mg、Cu共存时还可能出现Q-Al₅Cu₂Mg₈Si₆相和Mg₂Si相[[S40]]。

## 应用案例
AlSi9Cu3主要应用于汽车工业，典型零部件包括[[S23,S41]]：
- 发动机缸体（功能件）
- 自动变速箱壳体（如9AT壳体，轮廓尺寸560mm×440mm×400mm，重15.18kg）[[S8]]
- 离合器壳体、转向器壳体
- 底盘重载支架类零件
- 通讯电子壳体、通用机械壳体等非汽车领域

某款9AT变速箱壳体采用AlSi9Cu3(Fe)合金，通过37000kN冷室压铸机批量生产，年产5万件，内部气孔按ASTM E505标准控制为1级，密封面加工后允许气孔直径<0.4mm、间隔>8mm、数量<3个，在70kPa气压下泄漏量<3mL/min[[S8]]。

## 局限性及常见缺陷
1. **缩松与气孔**：常规高压压铸过程中极易因卷气而产生孔隙，需配合高真空压铸、仿形挤压销等辅助手段才能满足气密性要求[[S8,S31]]。铸态内部孔隙的存在导致合金无法进行常规的固溶+淬火类热处理，否则内部气体会膨胀起泡，仅可实施低温时效或退火[[S17,S33]]。
2. **Fe相脆化**：粗大片状β-AlFeSi相会严重割裂铝基体，降低塑性并成为裂纹扩展源。通过添加Mn元素转变为α-Al₁₅(Fe,Mn)₃Si₂相可削弱其危害[[S1,S22]]。
3. **耐蚀性**：合金含Cu量较高，富Cu的θ-Al₂Cu与α-Al基体间存在较大电位差，易发生电化学腐蚀，整体耐蚀性较差[[S26,S32]]。
4. **焊接性**：焊接性能较差，通常仅能采用电阻焊或特殊的软钎焊[[S33]]。

## 相关合金对比
AlSi9Cu3与相近的A380、ADC12虽同属Al-Si-Cu系，但成分控制和使用特性存在差异[[S38]]：

| 元素/特性 | AlSi9Cu3 (EN AC-46000) | A380 (ASTM) | ADC12 (JIS) |
|-----------|-------------------------|-------------|-------------|
| Fe (%) | 0.6~1.1（窄范围控制） | ≤1.0 | <1.3 |
| Zn (%) | ≤1.2 | ≤2.9 | <1.0 |
| Mg (%) | 0.15~0.55 | ≤0.01 | <0.3 |
| Cr (%) | ≤0.15 | 不主动添加 | — |
| 设计思想 | 通过Mg₂Si、θ相强化，均衡性能 | 低Mg、宽Zn限制，优先流动性与经济性 | 较高Si含量，利于充型 |

三种合金在实际生产中因成分重合度高，可在铝液包上相互调用[[S24]]。
