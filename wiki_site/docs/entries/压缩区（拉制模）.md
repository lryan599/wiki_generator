## 定义与位置

压缩区（又称变形区、工作区、压缩锥、减径区）是拉制模模芯的核心工作区域，位于润滑区之后、定径区之前。其核心功能是使金属坯料在模壁挤压下发生塑性变形，实现截面积缩减，从而得到目标直径的制品[[S30,S35]]。该区域的型面光洁度要求高于入口区、润滑区，以保证变形均匀并降低磨损[[S19]]。

在 GB/T 8845-2017《模具术语》中，压缩区被明确规定为拉制模的标准功能要素，压缩区长度定义为模孔压缩区沿轴线方向的尺寸，压缩角定义为压缩区工作面与模具轴线的夹角[[S1,S35]]。国家标准附图（图39）完整标注了压缩区与相邻区域的空间关系[[S6]]。

## 核心几何参数

### 压缩角（模角 α）

压缩区锥角定义为压缩区工作面与模具轴线的夹角[[S30]]。其合理取值与道次加工率、被拉制金属的抗拉强度及摩擦系数密切相关：合理模角随道次加工率增大而增大，随金属抗拉强度增加而减小[[S18]]。

不同拉制材料与场景的压缩角推荐值如表所示。

| 拉制场景 | 推荐模角 α | 来源 |
|---|---|---|
| 有色金属及合金棒线材 | 8°～13° | [[S18]] |
| 黑色金属及合金棒线材 | 3°～9° | [[S18]] |
| 金刚石模拉制铝及低强度合金 | 8°～12° | [[S18,S14]] |
| 金刚石模拉制紫铜、青铜 | 5°～6° | [[S18,S14]] |
| 金刚石模拉制钢材 | 3°～6° | [[S18,S14]] |
| 管材拉拔（常规） | 7°～15°，一般取 11° | [[S32]] |
| 小直径薄壁管 | 7°～8° | [[S32]] |
| 大规格空拉管 | 15° | [[S32]] |
| 工业常规拉拔（国际通用） | 半锥角取 8°，面减缩率 15%～20% | [[S14]] |

### 压缩区长度

线材模压缩区长度通常等于定径带直径，即 l<sub>压</sub> = d<sub>定</sub>；棒材模压缩区长度取 l<sub>压</sub> = (0.7～0.8)d<sub>定</sub>[[S18]]。也可通过几何关系精确计算[[S18,S20]]：

> **l<sub>压</sub> = (D<sub>0max</sub> - d<sub>定</sub>) / (2 tan α)**

式中，D<sub>0max</sub> 为坯料可能的最大直径，d<sub>定</sub> 为定径带直径，α 为压缩区锥角。

压缩区长度需适配拉拔工艺：湿式拉拔的压缩区长度短于干式拉拔，拉制硬度较高材料时需适当增加长度，以避免局部压力过大加剧模具磨损[[S19,S18]]。

### 道次压缩比（减面率）

道次减面率 r 表征单道次截面的缩减程度，是关联压缩区几何设计与拉拔工艺的核心指标[[S22,S10]]：

> **r = (A<sub>0</sub> - A<sub>1</sub>) / A<sub>0</sub> = 1 - (d<sub>1</sub>/d<sub>0</sub>)²**

常规工业拉拔单道次断面减缩率很少超过 30%～35%，常用值约 20.7%[[S22]]。

## 结构类型分类

压缩区的型面轮廓主要分为三类，分别适用于不同的加工率和使用场景[[S30,S5]]。

| 类型 | 特征 | 适用场景 |
|---|---|---|
| 圆锥形压缩区 | 直线锥面，结构简单，接触面积偏小 | 大加工率拉拔工况；直径 >1.0 mm 的棒材、线材通用结构 |
| 放射形压缩区（圆弧型/曲线型） | 弧线锥面，大小加工率均保持充足接触面积 | 直径 <1.0 mm 的细线材拉制模，可通过振动金属针磨光抛光加工 |
| 弧线形压缩区（抛物线凹型） | 非直线锥面，入口处变形速度最大、出口处最小 | 对变形均匀度要求高的高精度线材拉制 |

抛物线型压缩区的拉制力比锥形模高 10%～30%，其变形速度分布规律与锥形模相反，可实现压缩过程中单位变形量的均匀分布，降低瞬时应力波动[[S5]]。

## 工艺作用与协同关系

### 参数对拉拔的影响

压缩角取值对拉拔过程存在非线性影响[[S30,S18,S34]]：

- **压缩角过小**：坯料与模具工作面的接触面积增大，拉拔摩擦阻力上升，拉拔力增大。
- **压缩角过大**：金属流线急剧转弯，附加剪切变形增大，非接触变形增加；模壁单位正压力大幅升高，润滑剂易从缝隙中被挤出，润滑条件恶化，拉拔稳定性下降。
- **不当组合**：采用大于合理值的压缩角搭配小的道次压缩率时，金属易出现不连续流动，线材内部可能形成中心周期性裂纹。
- **优化组合**：合理压缩角搭配大的道次压缩率，可获得最均匀的材料流动状态[[S34]]。

### 与相邻功能区的协同

压缩区在模孔中并非独立工作，而是与其他功能区协同完成拉拔任务[[S19,S20,S32]]：

- **与入口区、润滑区协同**：入口区导入坯料并承载润滑剂，润滑区将预先涂布的润滑剂完全包覆在线材表面，确保润滑剂顺畅进入压缩区的变形接触界面，避免润滑中断导致的拉拔划伤[[S19,S32]]。
- **与定径区协同**：压缩区完成主塑性变形后，材料平滑过渡进入定径区，由定径区完成最终尺寸定型，保证制品的直径精度、圆度和表面光洁度[[S19,S20]]。
- **与出口区协同**：出口区在定径后提供支撑和导向，保证拉拔对中性，防止因弹性恢复刮伤制品表面，同时保护压缩区与定径区的交接边缘不发生崩裂[[S20]]。

国内行业规范要求拉制模所有功能区域衔接处需光滑过渡，不得存在明显台阶[[S29]]。

## 制造材料

压缩区模芯的主流制造材料按性能与成本差异分为四类[[S19,S1,S25]]。

| 材料类别 | 优势 | 劣势 |
|---|---|---|
| 硬质合金（WC-Co 烧结基体） | 易抛光、硬度高、红硬性好，是当前工业最常用材料 | 硬度上限有限，长期运行稳定性不足，整体寿命偏短 |
| 天然金刚石 | 拉制线材表面质量优异，断丝率低 | 各向异性导致磨损不均，加工难度大，采购成本高 |
| 聚晶金刚石（PCD） | 耐磨性远高于常规硬质合金 | 成本较高 |
| 工程陶瓷（碳化钛基、氧化铝基、四方氧化锆类） | 高硬度、高耐磨性，拉拔不锈钢等高硬度金属时性能优异 | 脆性高，烧结加工难度大，制造成本高 |

## 失效模式与影响

压缩区在服役中承受强烈的摩擦和径向应力，高速拉延时工作表面温度可达 400～500 °C[[S27]]。典型失效模式包括[[S27,S33]]：

- **磨粒磨损**：高速拉拔持续强摩擦作用下，压缩区工作面出现沟槽、模孔尺寸超差，直接导致线材表面划伤、尺寸精度不合格。
- **材料粘着**：接触面局部高温使拉拔金属发生微焊合，附着在压缩区表面生成粘附瘤，后续拉拔在制品表面形成连续划痕，严重时引发断丝。
- **开裂与崩刃**：脆性材料模芯长期承受较大径向拉应力，可能发生崩刃或开裂，导致拉制过程受力突变，线材内部残余应力超标或直接断裂。

## 标准依据

GB/T 8845-2017《模具术语》将压缩区列为拉制模的标准功能要素，统一了压缩区长度和压缩角的术语定义[[S1,S35]]。国内行业规范要求压缩区（工作区）角度公差控制在 ±2° 范围内[[S29,S3]]。

## 与压铸及挤压模具的差异

### 压铸收敛流道

压铸模具采用扇形收敛横浇道搭配窄截面内浇口，构成截面积沿流向逐步收敛的锥形结构，可在液态金属充填型腔时实现加速[[S26]]。然而，该结构与拉制模压缩区存在本质差异：

- 压铸内浇口长度通常仅 2～3 mm[[S7]]，远小于拉制模压缩区长度。
- 压铸收敛流道作用于液态合金的流动，不存在固态塑性变形；拉制模压缩区作用于室温固态金属，以塑性变形实现截面减缩[[S26,S1]]。
- 现有公开工业实践中不存在压铸模具直接集成拉制模压缩区的应用案例。常规压铸+拉制工艺流程为压铸生产粗坯后，单独送入独立拉制设备配套的拉制模中进行冷拉变形，两道工序完全独立[[S15,S8]]。

### 挤压模压缩区

挤压工艺同样存在压缩区（变形区）概念，但与拉制模压缩区有多项根本性区别：

- **坯料容置**：挤压模压缩区前方存在大容积挤压筒容纳坯料，压缩段之后存在明显的难变形“死区”；拉制模则为短锥面紧凑结构[[S2,S31]]。
- **驱动力方式**：拉制工艺依靠出口侧牵引力拉动坯料通过压缩区，以轴向拉应力主导实现小减面率连续变形；挤压工艺依靠入口侧压力推送坯料通过压缩区，坯料全程处于三向压应力状态，可实现大变形量成形[[S13,S24]]。

综上，拉制模压缩区与压铸收敛流道、挤压模压缩区虽然在“截面积收缩”这一外观特征上存在浅层类比，但在作用机制、力能状态和工程应用上属于相互独立的技术范畴。
