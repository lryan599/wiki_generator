## 概述

铸造不锈钢是指铬质量分数不低于12%，通过砂型铸造、精密铸造、压铸等铸造成形方法直接成型的不锈耐酸钢铸件[[S45,S50]]。与通过轧制或锻造成型的变形不锈钢不同，铸造不锈钢为适配铸造流动性需求而在成分上有所调整，铸态下通常保有5%~20%的δ铁素体以抑制热裂倾向，力学性能整体呈现各向同性[[S12,S41]]。铸造不锈钢广泛用于腐蚀环境下的泵壳、阀体、叶轮及海洋工程等部件生产[[S9]]。

按显微组织分类，铸造不锈钢可分为马氏体型、铁素体型、奥氏体型及奥氏体-铁素体双相型四大类[[S50]]。

## 定义与分类

### 铸造不锈钢与变形不锈钢的区别

| 对比维度 | 铸造不锈钢 | 变形不锈钢 |
|---------|-----------|-----------|
| 成型方式 | 铸造直接成型 | 轧制/锻造成型 |
| 显微组织 | 通常含有5%~20% δ铁素体 | 多为完全奥氏体组织 |
| 力学性能各向异性 | 各向同性 | 存在明显各向异性 |
| 强化方式 | 主要通过合金化和热处理 | 可通过冷加工进一步强化 |
| 加工硬化倾向 | 较低 | 较高，易发生加工硬化 |

来源：[[S12,S41]]

### 组织分类与牌号对照

铸造不锈钢按显微组织可分为以下主要类型：

- **奥氏体型**：以CF8、CF8M、CF3、CF3M（美标）及ZG1Cr18Ni9、ZG0Cr18Ni9（国标）为代表，具有优异的耐腐蚀性和延展性[[S10,S50]]。
- **马氏体型**：以CA15、CA40（美标）为代表，可经热处理硬化，适用于较高硬度和耐磨性要求的场合[[S7]]。
- **铁素体型**：以CB30、CC50（美标）为代表，耐蚀性高于马氏体钢但强度稍低[[S7]]。
- **双相型**：以CD4MCu为代表，兼具奥氏体和铁素体的性能优势[[S7]]。

**美标典型牌号与变形不锈钢的对应关系**[[S2,S4]]：

| 铸造牌号 (ASTM A351/A743) | 等效变形不锈钢牌号 | 特征 |
|--------------------------|-------------------|------|
| CF8 | 304 | 通用奥氏体不锈钢 |
| CF8M | 316 | 含钼，耐点蚀性更优 |
| CF3 | 304L | 超低碳，焊接态耐腐蚀 |
| CF3M | 316L | 超低碳含钼，综合耐蚀性优 |

**国标典型牌号**：ZG1Cr18Ni9对应变形牌号1Cr18Ni9，ZG0Cr18Ni9对应变形牌号0Cr18Ni9（即常规304牌号）[[S30]]。

## 牌号标准体系

### 国际标准体系

铸造不锈钢的主要国际标准包括：

- **ASTM A351/A351M**：高温承压用奥氏体不锈钢铸件标准，收录CF3、CF8、CF8M等牌号，限定高温工况使用[[S8,S21]]。
- **ASTM A743/A743M**：通用耐腐蚀工况用不锈钢铸件标准，收录34个牌号，覆盖马氏体、奥氏体、双相等全系列[[S19,S3]]。
- **ASTM A744/A744M**：严苛耐腐蚀工况专用标准，牌号已全部列入ASTM A743[[S19]]。
- **ISO 11972:1998(E)**：通用耐蚀铸钢国际标准，共收录17个铸造不锈钢牌号[[S10,S44]]。

### 中国国家标准体系

- **GB/T 2100-2002**《一般用途耐蚀钢铸件》：等效采用ISO 11972:1998，增补ZG20Cr13和ZG03Cr14Ni14Si4两个国内牌号，共收录19个牌号[[S10,S37,S45]]。
- **GB/T 6967-2009**《工程结构用中、高强度不锈钢铸件》：收录9种马氏体不锈钢，以力学性能为主要验收依据[[S45]]。
- **JB/T 6405-2006**《大型不锈钢铸件》：规定14种铸造不锈钢的化学成分与力学性能[[S45]]。

### ISO与国标牌号对照

ISO 11972:1998(E)与GB/T 2100-2002中铸造不锈钢牌号存在明确的对应关系，可直接用于标准牌号转换，具体对照关系详见相关标准表格[[S44]]。

中国铸造不锈钢牌号与美、英、德、日等国际标准亦存在对应关系，例如ZG1Cr18Ni9Ti可映射至CF20、SCS22等牌号[[S30]]。

## 物理性能与铸造工艺特性

### 热导率

CF8类牌号（1.4308）不锈钢热导率随温度变化特性：温度低于600°C时热导率先小幅波动后下降，600°C左右达到最低值，之后随温度升高持续增长，2100°C时约为31 W/(m·K)[[S16]]。

### 铸造工艺特性

18-8型奥氏体铸造不锈钢具有以下铸造工艺特性[[S23]]：

- **流动性**：含铬量高，流动性较差，易产生冷隔、浇不足、表面皱皮和夹杂缺陷。
- **推荐浇注温度**：1540~1580°C，应在较高温度和较大速度下进行浇注。
- **体收缩**：体收缩大，易产生缩孔缩松缺陷，冒口尺寸需比碳钢大20%~30%。
- **自由线收缩**：3.0%~3.2%，热裂倾向等级高。
- **氧化倾向**：浇注温度越低、浇注时间越长，氧化现象越严重。

### 合金元素对铸造性能的影响

合金元素对铸造不锈钢缩松和热裂倾向有显著影响：碳、铬、镍、锰等元素增大钢的结晶温度间隔并提高凝固收缩，从而增加缩松程度和热裂倾向；同时合金元素普遍降低导热性，进一步加剧收缩与热裂风险[[S29]]。

马氏体不锈钢由于高铬含量，易产生氧化铝膜、氮化物、碳化物等难熔质点，流动性差，体积收缩大于碳钢，热裂倾向显著更高[[S29]]。

## 耐腐蚀性能

### 晶间腐蚀

铸造奥氏体不锈钢（CF8、CF8M等）晶间腐蚀的产生机理：晶界处铬的碳化物M₂₃C₆析出，造成晶界附近局部贫铬，导致沿晶粒边界选择性溶解[[S10,S48]]。宏观上材料表面仍保留金属光泽，但晶间结合力完全丧失，力学性能大幅下降[[S48]]。

晶间腐蚀检测依据国标GB/T 2100，可选试验方法包括[[S50]]：
- GB/T 4334.1：10%草酸浸蚀试验
- GB/T 4334.3：65%硝酸腐蚀试验
- GB/T 4334.5：硫酸-硫酸铜腐蚀试验

### 点腐蚀

点腐蚀集中在金属表面微小区域，向内部快速扩展直至穿透基体[[S48]]。Cl⁻等卤素离子的存在会显著加速点腐蚀进程[[S48]]。含钼的CF8M、CF3M牌号比不含钼的CF8、CF3牌号具有更优的耐点腐蚀性能[[S4]]。

## 压铸工艺适用性

### 工艺现状与可行性

不锈钢压铸目前仍处于探索与发展阶段。当前行业量产级不锈钢铸件的主流生产工艺为熔模精密铸造和砂型铸造，压铸工艺在不锈钢材质的大规模商业化批量生产中尚未进入常规应用阶段[[S5,S42,S46]]。

据文献报道，黑色金属压铸（包括不锈钢）的研究工作近年来有所进展，部分国家已进行了试验性研究与小批量试生产，但公开报道的大规模量产级工业化应用极少[[S46,S39]]。

### 已公开的压铸工艺参数

不锈钢压铸已公开的试生产实测工艺参数区间为[[S39,S46]]：

| 工艺参数 | 范围 |
|---------|------|
| 浇注温度 | 1450~1600°C |
| 模具预热温度 | 350~600°C |
| 压射比压 | 100~250 MPa |

### 模具材料选型

由于不锈钢的高熔点及对模具材料的强烈熔蚀作用，常规热作模具钢难以满足压铸模具的服役寿命要求。适配方案包括[[S27,S17]]：
- 选用高热强度的铬锆铜类铜合金作为镶块
- 采用钨基硬质合金作为关键抗冲刷部位的镶块
- 上述特种材料可提升模具抗高温熔蚀与热疲劳性能

### 专用脱模剂

适配不锈钢压铸的专用耐高温脱模剂需添加硼化物、石墨、陶瓷类固体填料，可在350°C以上的高模具温度下稳定成膜，避免高温下脱模剂快速分解失效，以降低高温不锈钢液与模具发生粘模的风险[[S34,S22]]。

### 压铸特有缺陷

不锈钢压铸存在以下典型独有缺陷：

**1. 热粘着（Soldering）**

温度在1400°C以上的液态不锈钢与模具表面发生冶金亲和熔合，在型腔表面生成不锈钢-模具材料的合金化层，铸件取出时造成铸件表面撕裂、型腔表层材料附着剥落。该缺陷的发生概率远高于常规低熔点有色合金压铸场景[[S20,S31]]。

**2. 模具冲蚀（Erosion）**

在高压高速状态下流动的高温不锈钢液持续冲刷型腔局部区域，对模具表层材料产生溶解磨损作用，其冲蚀速率远高于铝合金、铜合金压铸工况，会大幅降低压铸模具的实际服役寿命[[S15,S6]]。

**3. 早期热裂（Heat Checking）**

1400°C以上的高温不锈钢液与300~600°C的模具型腔表面反复发生剧烈冷热交变，型腔表层持续承受远超常规热作钢耐受阈值的热应力冲击，导致模具在远低于常规有色合金压铸的循环次数下就提前出现龟裂开裂现象[[S38,S1]]。

### 不锈钢压铸相较于其余铸造方法的优劣

与传统精密铸造、砂型铸造相比[[S18,S49]]：

| 维度 | 不锈钢压铸 | 传统铸造方法 |
|------|-----------|-------------|
| 尺寸精度 | IT11~IT13级，部分可达IT9级 | 较低 |
| 表面光洁度 | 更优 | 一般 |
| 晶粒组织 | 表层晶粒更细 | 相对粗大 |
| 生产节拍 | 更快 | 较慢 |
| 近净成形能力 | 更强 | 一般 |
| 模具采购与制造成本 | 极高 | 较低 |
| 模具服役寿命 | 极低 | 较高 |
| 适用铸件尺寸 | 难以兼容大尺寸厚壁件 | 可生产大型件 |
| 热处理强化 | 常规压铸件不可直接热处理强化 | 可进行热处理 |

## 典型缺陷与对策

### 通用砂型/精密铸造缺陷

铸造不锈钢件的常见缺陷及成因[[S51,S53]]：

| 缺陷类型 | 成因 |
|---------|------|
| 缩孔缩松 | 体收缩大、补缩不足、铸件结构壁厚不均 |
| 裂纹（热裂） | 线收缩大、热裂倾向高、铸型结构不合理 |
| 气孔 | 型腔潮湿、金属液含气量高、排气不畅 |
| 氧化夹杂 | 含铬量高、浇注过程氧化、紊流裹挟氧化渣 |
| 浇不足/冷隔 | 流动性差、浇注温度偏低、充型速度不足 |

非金属夹杂物（如FeO、MnS）可作为形核核心诱导孔洞缺陷[[S43]]。

### 通用铸造缺陷防控工程措施

针对铸造不锈钢件的通用缺陷防控措施[[S51]]：

1. **优化浇注系统**：实现金属液平稳充型，避免紊流裹挟氧化渣。
2. **优化补缩系统**：保障铸件顺序凝固，为热节区域提供充足液态补缩。
3. **提升砂型紧实度**：避免型壁外胀引发收缩缺陷。
4. **充分烘烤型腔**：排除水分，降低气孔生成概率。
5. **合理提高浇注温度与速度**：增强流动性，减少氧化渣含量。

针对不锈钢压铸场景下气孔与缩松两类典型内部缺陷，已有明确成因分析与配套防控措施表格可供现场工艺参考[[S28]]。

## 典型应用与工艺选择

### 砂型铸造不锈钢应用

砂型铸造适用于大型不锈钢泵体、阀体、海洋工程重型构件等大尺寸铸件生产，可适配单件至大批量制造，成本较低、适配性广[[S33,S9]]。

### 精密铸造不锈钢应用

精密铸造（熔模铸造）适配形状复杂、尺寸精度要求高的不锈钢小零件生产，包括[[S35,S40,S24]]：
- 不锈钢叶轮
- 食品机械精密构件
- 核电用小通径阀件
- 井下密封件
- 各类高表面质量要求的小型零件

精密铸造可获得优异表面质量并减少后续机加工工序[[S40]]。

### 特种压力铸造应用案例

在海洋工程领域，采用差压工艺生产的07Cr18Mn10NCu高氮钢船用螺旋桨，具备优异的耐海水腐蚀性能与腐蚀疲劳力学性能，属于特种压力铸造的工业落地场景[[S26]]。

### 压铸领域应用

目前不锈钢压铸尚未在大规模商业化生产中确立明确的常态化应用领域，现有报道主要集中在试验性研究与小批量试制阶段[[S46,S42]]。
