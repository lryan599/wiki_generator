## 概述

成型推杆（又称成形推杆、型面推杆）是压铸模具推出机构中的一种特殊功能零件，区别于GB/T 4678.11规定的普通平端面圆柱推杆，其端部加工有与铸件局部轮廓完全匹配的型面，同时兼具成型指定特征结构和推出铸件的双重作用[[S16,S9]]。在全立式压铸机压铸模结构中，成型推杆作为独立功能件被明确标注[[S16]]。

行业内对成型推杆的英文表达主要有两种使用频率最高的译名：**contoured ejector pin** 与 **profile ejector pin**，其次为 forming ejector pin，其余备选术语在压铸领域文献中的使用占比普遍较低[[S8,S11]]。

## 分类与结构

### 按头部型面形状分类

成型推杆可按其与铸件接触的端部型面形状分为以下四类[[S2,S7]]：

- **平面成型推杆**：端部为平面，设置于铸件的端面、凸台、肋部及浇注系统部位；
- **球面成型推杆**：端部呈球面形状；
- **锥面成型推杆**：端部呈圆锥形，常用于分流锥中心处，兼具分流与推出作用；
- **随形成型面推杆**：端部形状与铸件局部轮廓完全匹配。

### 按尾部肩部形式分类

按推杆尾部结构形式可分为[[S7]]：

- **直身无肩型**：尾部无台阶，结构简单；
- **台阶带肩型**：尾部设有台阶，通过推杆固定板与推板夹持固定。

### 按固定方式分类

推杆在推出机构中的固定方式直接影响其工作可靠性，常用固定形式包括[[S7]]：

| 固定方式 | 特点 |
|----------|------|
| 台阶沉入固定式 | 推杆尾部台阶被推杆固定板与推板夹持，强度高，不易变形 |
| 夹紧固定式 | 通过垫块或垫圈保证推杆定位尺寸一致性 |
| 螺塞顶紧固定式 | 螺塞拧入推板顶住推杆轴肩，防止轴向移动 |

当推出位置有方位性要求时，需在推杆尾部增设止转结构（如在固定板上装平键，推杆台肩上设切口），防止推杆周向转动偏移[[S10]]。

## 工艺作用与工作循环

成型推杆安装于动模侧，在压铸完整循环中贯穿充型、冷却、推出、复位全流程[[S16]]：

1. **充型阶段**：成型推杆端部型面参与型腔充型，承受高压合金冲击，成型铸件局部结构[[S2]]；
2. **冷却阶段**：与铸件一同冷却，型面特征定型；
3. **推出阶段**：在压铸机推出装置驱动下，推板带动推杆将附着在动模型芯上的铸件平稳推出[[S16]]；
4. **复位阶段**：合模时复位杆带动推出机构同步复位至型面平齐位置[[S16]]。

成型推杆可参与成型的铸件局部结构包括：加强筋端面、小凹槽、防伪标识字符、特征定位凸台等，无需二次机加工即可保证成型特征精度[[S16]]。其与普通平直推杆的本质区别在于：普通推杆仅承担推出载荷，而成型推杆既需在充型阶段承受高压冲击完成型面充型，又需在推出阶段平稳推出铸件，避免铸件变形[[S5]]。

成型推杆穿装入动模镶块对应导向孔后，尾部台肩被夹持在推杆固定板与推板之间，由推板导柱、推板导套组成的导向副保证推出过程轴线同轴度[[S6,S10]]。可与浇道推杆配合完成铸件与浇注余料的同步推出[[S16]]。

## 材料与性能要求

### 常用材料及硬度

压铸成型推杆的常用材料及对应热处理要求如下表：

| 材料牌号 | 类别 | 推荐硬度（HRC） | 备注 |
|----------|------|----------------|------|
| 4Cr5MoSiV1 | 热作模具钢 | 45～50 | GB/T 4678.11-2017推荐材料[[S9,S18]] |
| H13（5%铬热作模具钢） | 热作模具钢 | 38～42 | 国际通用材料[[S8,S19]] |
| T8A | 碳素工具钢 | 50～55（工作段） | GB/T 4678.11-1984标准材料[[S17]] |
| 3Cr2W8V | 热作模具钢 | 45～50（工作段） | GB/T 4678.11-1984标准材料[[S17]] |

现代标准（GB/T 4678.11-2017）推荐采用4Cr5MoSiV1，硬度45 HRC～50 HRC[[S18]]。国际压铸行业也常用H13钢，常规热处理后硬度38 HRC～42 HRC[[S8]]。早期标准（GB/T 4678.11-1984）曾使用T8A和3Cr2W8V，其中T8A工作段淬火硬度50 HRC～55 HRC，3Cr2W8V工作段淬火硬度45 HRC～50 HRC[[S17]]。

### 表面处理

为提高耐磨性能并防止与推杆孔配合面发生咬合粘滞，成型推杆常采用以下表面处理工艺：

- **渗氮处理**：渗氮层深度0.08 mm～0.15 mm，基体硬度40 HRC～44 HRC，表面硬度不低于900 HV[[S18,S15]]；
- **TD处理（Tufftriding）**：表面渗层处理，提升耐磨性能，防止配合面咬合[[S8,S19]]。

美国NADCA压铸标准明确规定推杆表面必须做氮化处理，以保障自动压铸生产过程中推出机构无故障连续运行[[S5,S8]]。

## 配合精度与标准规范

### 配合公差

成型推杆与推杆孔的配合需精确控制，间隙过小会在模具预热后因金属热膨胀导致推杆卡死，间隙过大则铝液会窜入缝隙导致溢料[[S13]]。

- **推杆导向段与镶块孔的推荐配合间隙**：0.05 mm～0.10 mm[[S10]]；
- **配合精度等级**：推杆工作直径配合选用三级转合座精度[[S13]]；
- **工作直径D的极限偏差**：符合GB/T 1801-2009中e8的精度要求[[S18,S13]]。

### 国家标准

GB/T 4678.11-2017《压铸模 零件 第11部分：推杆》规定了压铸模用推杆的结构型式与尺寸、要求、材料和标记[[S18]]。标准包含A型和B型两种推杆，其尺寸系列规定了直径D、长度L、头部厚度h等参数。标准要求：成型推杆的成形端面不允许留有中心孔；未注表面粗糙度Ra=6.3 μm；端面棱边不允许倒钝[[S1]]。

小于φ5 mm规格的推杆非工作端应做加粗处理以提升抗断强度[[S13]]。

国际标准方面，日本JIS B 5103为压铸模用推顶杆对应标准，规定了压铸场景下推杆的材料、尺寸、形位公差通用要求[[S12,S3]]。

## 缺陷与设计准则

### 常见缺陷

成型推杆在压铸生产中可能引发以下缺陷：

- **推杆印痕**：推杆端面在铸件表面留下痕迹，即使端面与型腔完全平齐也无法完全避免[[S11]]；
- **铸件顶溃变形**：顶出力分布不均，单根推杆推力超过铸件被推部位许用应力时导致变形[[S20,S13]]；
- **推杆穿透铸件**：小规格推杆推出软质高温铸件时发生穿透失效[[S5]]；
- **推杆断裂**：细长推杆在推出过程中失稳断裂[[S20]]；
- **推杆卡死**：配合间隙过小或润滑不足导致运动卡滞[[S13]]。

### 设计准则

1. **直径限制**：直径小于3/16英寸（约4.8 mm）的小规格成型推杆在压铸生产中不推荐使用，此类小推杆容易发生穿透铸件、自身断裂失效的问题[[S5]]；
2. **稳定性校核**：成型推杆工作时的稳定安全系数推荐取值范围为1.5～3.0，需通过稳定性校核避免推出过程中推杆失稳断裂[[S20]]；
3. **布置原则**：推杆位置应均匀分布在铸件包紧力大的区域，增大推杆直径可有效降低单位面积压强，避免顶穿铸件[[S20]]；
4. **推杆直径与成型尺寸关系**：推杆直径d应比成型尺寸d₀小0.4 mm～0.6 mm，推杆边缘与成型立壁保持一个小距离δ，形成小台阶以防止金属窜入[[S2]]。

## 关系组件

成型推杆与压铸模推出机构中以下组件存在紧密的装配与配合关系[[S6,S16,S7]]：

| 关系组件 | 功能关系 |
|----------|----------|
| 推杆固定板 | 通过推杆尾部台阶将推杆固定在推出机构上，通过螺栓紧固在推板上 |
| 推板 | 与推杆固定板配合夹持推杆尾部，传递推出力 |
| 复位杆 | 合模时将推出机构同步推回至浇铸位置 |
| 推板导柱/导套 | 组成导向副，保证推出过程中推杆轴线的同轴度 |
| 浇道推杆 | 与成型推杆配合完成铸件与浇注余料的同步推出 |
| 动模镶块 | 提供成型推杆穿装的导向孔，保证推杆的推出方向精度 |
