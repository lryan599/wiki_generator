## 概述

残余熔体模数（Residual Melt Modulus, RMM），是铸造数值模拟领域中用于定量预测铸件凝固后期孤立液相区产生缩孔、缩松缺陷倾向的核心判据参数。该参数通过表征凝固末期被枝晶骨架封闭的残余液相熔池的几何特征，来判定补缩通道堵塞后局部区域形成孔洞类缺陷的概率，已广泛应用于铸钢、铝合金等合金体系的多种铸造工艺缺陷预测中，并集成于主流商业铸造模拟软件（如AnyCasting、Flow-3D Cast）的概率缺陷预测模块[[S12,S25,S20]]。

## 定义与表达式

残余熔体模数的基础计算逻辑为残余熔体体积与残余熔体表面积的比值，量纲为长度，常用单位为cm或m。行业内标准表达式统一为[[S12,S5,S25,S31,S15]]：

$$M_r = \frac{V_r}{A_r}$$

式中各符号含义如下[[S12,S5,S31,S26]]：

- **$M_r$**（或记为 $RM$、$M$）——残余熔体模数，单位为cm或m；
- **$V_r$**（或记为 $RV$、$V$）——残余熔体体积，即凝固后期形成的孤立液相区总体积，单位为cm³或m³；
- **$A_r$**（或记为 $RA$、$A$）——残余熔体表面积，即该孤立液相区的外表面散热总面积，单位为cm²或m²。

不同文献采用的符号命名存在微小差异（$M_r$、$RM$、$M$），但物理含义完全统一[[S31,S26,S12,S25,S27]]。需注意，某一面向离心泵铸造工艺的文献曾将残余熔体模数反向定义为"残余熔体表面积与残余熔体体积的比值"，该表述与行业通用的$V/A$标准定义完全矛盾，其余所有检索到的中文铸造核心期刊论文及高校研究资料均统一采用$V/A$的标准定义[[S13]]。

## 英文术语

目前文献中出现的英文表述共有三类[[S12,S25,S16]]：

- **Residual Melt Modulus (RMM)** ——标准通用表述，为学术论文中最为规范的英文全称；
- **Retained Melt Modulus** ——AnyCasting软件及部分工业文献中广泛使用的别称；
- **Residual Melt Volume/Surface Ratio** ——部分文献使用的功能描述式称呼，即残余熔体体积表面积比。

## 物理意义与缺陷预测原理

### 孤立熔池与补缩通道堵塞

残余熔体模数的缺陷预测原理建立在凝固后期孤立液相区形成机制之上。铸件凝固过程中，先期生长的树枝晶骨架相互连接，将剩余未凝固的金属液分割为若干封闭的孤立液相区。一旦孤立液相区被完全固相包裹，外部金属液无法进入该区域提供补缩，补缩通道即被完全堵塞。这些封闭区域在继续冷却凝固时无法获得液态金属的收缩补偿，最终形成缩孔或缩松等孔洞类缺陷[[S8,S25,S9,S16]]。

### RM值与缺陷概率的正相关关系

残余熔体模数值与缩孔缩松缺陷的产生概率呈直接正相关。$V_r$（残余熔体体积）越大，说明孤立熔池区域的体积越大；$A_r$（残余熔体表面积）越小，说明残余熔体越集中。RM值越高的区域，对应的孤立熔池体积越大、熔体集中程度越高，凝固过程中得不到补缩的液态收缩量越高，最终形成孔洞类缺陷的风险也越高。在软件应用中，通常认定概率缺陷参数大于0.9的区域将产生严重缩松缺陷[[S12,S5,S25,S15,S20]]。

### 高RM值区域与高风险热节点

铸件中残余熔体模数峰值的出现位置对应着典型的缩孔缺陷高风险热节点。以多轮辐铝合金轮毂低压铸造为例，轮毂中心与轮辐的连接处因冷却较慢、补缩通道易被先行凝固的轮辐薄壁部分截断，残余熔体模数出现峰值，该区域属于发生缩孔缺陷的高风险区[[S15,S2,S18]]。同理，铸件壁厚中心区域、截面突变过渡处等最后凝固部位，也常表现为高RM值区域[[S3]]。

## 计算方法与数据需求

残余熔体模数的标准计算流程如下[[S5,S25,S12,S27,S20]]：

1. **温度场与固相率场计算**：基于凝固数值模拟，获得铸件在凝固全过程各时间步的温度场和固相率分布；
2. **孤立液相区识别**：在凝固后期，按固相率阈值识别出被完全固相包裹、与外部金属液隔绝的孤立液相区（即残余熔体）；
3. **几何参数提取**：统计各孤立液相区的总体积$V_r$与外表面散热总面积$A_r$；
4. **模数计算**：通过$M_r = V_r / A_r$直接计算出对应的残余熔体模数。

该参数的计算完全依赖于凝固过程的动态模拟结果，而非铸件的静态几何尺寸[[S17,S30,S1]]。

## 与其他判据的比较

### 与传统几何模数（$M = V/A$）的区别

传统静态几何模数$M = V/A$为铸件区域整体体积除以其全部冷却散热表面积的静态几何计算结果，仅基于铸件几何尺寸得出，不涉及凝固过程的动态变化；而残余熔体模数专门统计凝固后期已形成的孤立液相区的体积和边界表面积，仅表征失去补缩通道的残余熔体的属性，属于凝固动态过程参数[[S17,S30,S1]]。

### 与Niyama判据的比较

| 对比维度 | 残余熔体模数法 | Niyama判据 |
|---|---|---|
| 核心机理 | 以孤立液相熔体模数为依据，直接判定补缩通道堵塞后的缩孔缩松风险 [[S12]] | 通过温度梯度$G$与冷却速度$R$的比值$G/\sqrt{R}$判定缺陷倾向 [[S17,S29]] |
| 适用缺陷类型 | 宏观缩孔、宏观缩松、集中孔洞类缺陷 [[S12,S17]] | 枝晶间显微缩松 [[S12,S17]] |
| 预测精度特征 | 对铸钢件、铝合金铸件的宏观孔洞缺陷预测精度更高 [[S12]] | 对宏观集中孔洞缺陷预测精度不足 [[S12,S27]] |
| 典型软件应用 | AnyCasting概率缺陷参数模块、Flow-3D Cast [[S5,S25,S20]] | ProCAST、MAGMA、FTSolver [[S17]] |
| 工业配合惯用方式 | 作为Niyama判据的补充，二者联合使用以提升整体缺陷预测准确率 [[S27,S5,S12]] |

### 灰铸铁件场景的特殊性

对于灰铸铁件的缩松缺陷预测，残余熔体模数法与Niyama判据的判定结果并不完全一致。残余熔体模数法在部分场景下可考虑石墨化膨胀的影响：若残余熔体模数法判定结果优于Niyama判据，说明灰铸铁自膨胀现象是缺陷的主要诱因；若判定结果差于Niyama判据，则说明缺陷主要由补缩不足引发。因此，在灰铸铁及球墨铸铁场景中需结合两种判据综合分析，或引入针对性的膨胀修正因子[[S29,S12,S17]]。

## AnyCasting软件中的应用

### 功能位置与计算逻辑

在AnyCasting软件中，残余熔体模数属于**概率缺陷参数（Probabilistic Defect Parameter）**模块功能，其计算逻辑定义为残余熔体体积与残余熔体表面积的比值，单位为cm。数值越高代表铸件凝固后期形成孤立熔池的概率越高，对应缩孔、缩松缺陷的产生风险越大[[S25,S5,S27]]。

### 标准操作流程

AnyCasting中残余熔体模数功能的标准操作流程为[[S27,S25]]：

1. **前处理（anyPRE）**：导入三维模型、划分网格、设置材料属性与工艺参数、配置边界条件和浇口条件；
2. **求解计算（anySOLVER）**：提交充型与凝固全流程计算；
3. **后处理（anyPOST）**：在概率缺陷参数选项中，选择**Retained Melt Modulus**即可直接生成可视化结果。

### 输出形式与解读

残余熔体模数的输出形式为anyPOST后处理模块中的彩色云图，配套数值色标将模数区间映射为不同颜色梯度。红色高亮区域为高缺陷概率位置，常规工况下40%~50%的缺陷概率即对应高亮区域[[S14]]。判读时关注模数峰值部位，这些区域即为最可能出现缩孔缩松缺陷的位置。

### 与其他判据的联合使用

受限于Niyama判据仅对枝晶间显微疏松预测效果较好的特性，工业应用中通常将残余熔体模数作为其补充判据联合使用。针对同时存在宏观缩孔和显微缩松的复杂缺陷场景，二者结合可提升整体缺陷预测准确率[[S27,S5]]。

## 工艺参数对残余熔体模数的影响

工艺参数可直接调控残余熔体模数以降低缩孔缩松风险。以铝合金真空压铸工艺的正交试验为例，各因素对残余熔体模数的影响程度从大到小排序为[[S10]]：

| 排序 | 工艺参数 |
|:---:|:---|
| 1 | 浇注温度 |
| 2 | 模具温度 |
| 3 | 快压射速度 |
| 4 | 慢压射速度 |
| 5 | 真空度 |

在铝合金发火座压铸工艺中，当浇注温度为630℃（接近液相线）、浇注速度为40mm/s、模具预热温度为220℃时，残余熔体模数可降至0.6456cm，获得最小的缩孔缩松生成倾向[[S3]]。在铝合金离合器壳体真空压铸的参数组合为浇注温度680℃、模具温度210℃、快压射速度5.5m/s、慢压射速度0.6m/s、真空度30KPa时，得到的残余熔体模数最低为1.23%，对应最低的缩孔缺陷概率[[S28]]。

## 典型工业应用案例

### 轮毂/轮辐连接处缺陷检测

多轮辐铝合金轮毂低压铸造的仿真与实测案例验证，轮毂中心与轮辐的连接位置会出现残余熔体模数峰值，该区域为典型的缩孔缺陷高风险点。通过适当提升保压压力可有效降低该位置的残余熔体模数，减少缩孔缺陷的产生[[S2,S18]]。薄壁轮辐铝合金轮毂的仿真结果同样表明，轮辐与轮毂的连接处以及轮毂中心和轮辐的连接处是残余熔体模数峰值的出现部位，属于发生缩孔缺陷的高风险区[[S15]]。

### 铝合金缸体/缸盖铸造工艺验证

在铝合金缸体压铸工艺优化中，可通过残余熔体模数分布云图直观对比不同工艺参数下的缺陷倾向差异，模数峰值从0.4410cm降至0.4256cm即对应着缩孔风险的降低[[S19,S23,S24]]。在发动机铝合金缸盖低压铸造中，通过在远离浇口且冷却缓慢的区域增加冷却水管并适当增压，可显著减少孤立熔池区域，仅靠近浇口位置的不可避免区域保留残余熔体[[S31,S26]]。

### 间接挤压铸造工艺优化验证

优化后的间接挤压铸造铸件采用AnyCasting概率缺陷参数模块生成残余熔体模数判据结果，整体缺陷概率仅0.02142，铸件几乎全部区域为低风险，仅边缘极小区域有低强度缺陷信号，验证了工艺优化的有效性[[S6]]。

## 适用范围与局限性

### 已覆盖的合金与工艺类型

残余熔体模数法已被工程验证覆盖的合金类型包括[[S12,S7,S20,S21]]：

- 铸钢：ZG07Cr19Ni10等
- 铸造铝合金：ADC12、A356、AC4B、ZL205A等
- 特别适配结晶温度范围较宽、易产生孤立液相区的合金

已被验证的铸造工艺场景包括[[S8,S31,S25,S20]]：
- 高压压铸、低压铸造、间接挤压铸造、砂型铸造、熔模精密铸造
- 主流商业铸造模拟软件AnyCasting、Flow-3D Cast均已集成该判据

### 固有局限性

1. **显微缩松预测能力弱于Niyama判据**：残余熔体模数法针对的是凝固后期形成的大尺寸孤立液相区，对完全由枝晶间微小间隙未补缩形成的显微缩松，预测灵敏度不及Niyama判据[[S12,S17]]；
2. **未考虑石墨化膨胀效应**：默认忽略了灰铸铁、球墨铸铁凝固过程中的石墨化膨胀自补缩效应，直接应用于此类铁合金时预测结果易出现偏差，需针对性引入膨胀修正因子或配合Niyama判据联合分析[[S29]]；
3. **不适用于补缩通道未完全堵塞的场景**：当凝固末期的孤立液相区尚未被完全封闭时，残余熔体模数法的预测准确性将有所下降[[S16]]。
