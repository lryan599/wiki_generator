## 定义

在压铸语境下，开合模动作与锁模功能的统一整体被称为**合型机构**（又称**合模机构**、**锁模机构**）[[S4,S6,S14,S13,S17]]。该机构负责带动压铸机动模部分完成与定模的开合动作，同时在金属液压射充填时提供足够的锁紧力以抵消模具被胀开的反作用力，从而保障压铸过程安全并确保压铸件满足尺寸公差要求[[S4,S6,S14,S13,S17]]。该定义同时适用于冷室压铸机与热室压铸机[[S4,S6,S14,S13,S17]]。

NADCA官方压铸教材进一步明确合型系统的核心功能为两部分：一是在熔融金属注入型腔时锁紧两半模具，二是执行模具开合动作完成压铸件取件[[S3]]。主流的合型框架结构分为拉杆-模板式和整体框架式两类，其中拉杆-模板式应用最为广泛，具备更灵活的调整能力[[S3]]。

## 分类体系

压铸机合型机构可划分为全液压式与液压驱动机械扩力式两大类别，各类型特征与适用机型如下表所示[[S7]]：

| 分类 | 特点 | 应用机型举例 |
|------|------|-------------|
| **全液压合型机构** | 安装不同厚度压铸型时无需调整开挡；可自动补偿压铸型生产过程中的热膨胀而不影响合型力大小；合型刚性和可靠性较差；生产率低；仅适配小型压铸机 | J113A、J113C |
| **液压-曲肘合型机构** | 更换不同厚度压铸型时需调整开挡；无法自动补偿模具热膨胀；通过曲肘连杆系统可实现数十倍锁型力放大；合型刚性高、开合型速度变化合理、生产率高；应用最为广泛，尤其适配中大型压铸机 | J116B至J11200全系列国产压铸机 |
| **液压-斜楔合型机构** | 合型可靠、刚性大；压铸型热变形对合型力无影响；结构复杂、生产率低、楔体抽出难度大；多用于大型压铸机 | FKM-600 |
| **液压-闸板合型机构** | 结构复杂但闸板插拔顺畅；合型可靠性高；生产率低于前三类结构；仅用于超大型压铸机 | Castmatic-3000 |

### 曲肘机构的细分类型

在曲肘合型机构中，按肘杆配置方式可分为单曲肘与双曲肘两种形式：

- **单曲肘合模机构**：采用单组曲肘连杆配置，液压缸行程短，开模行程受限，主要用于小型压铸机[[S11]]。
- **双曲肘合模机构**：配置两套对称的曲肘连杆组件，中心液压驱动，可实现更大开模行程，增力倍数可达数十倍，结构紧凑刚性好，广泛用于中小型至25000kN级的大中型压铸机[[S11,S2,S1]]。双曲肘内翻式结构在开模时曲肘相对于轴线向内翻，适用于中小型机，可节省设备体积并降低成本[[S10]]。

## 结构与组成

液压-曲肘合型机构（拉杆-模板式）是最典型的合型机构形式，其核心系统组成及部件功能如下[[S3,S14]]：

| 部件名称 | 功能说明 |
|---------|---------|
| 固定模板（定模座板） | 用于固定压铸机定模侧，与拉杆端部刚性连接承受合模反力 |
| 移动模板（动模座板） | 用于固定压铸机动模侧，沿拉杆轴向滑动实现开合模动作 |
| 拉杆（哥林柱） | 共4根对称布置，是合模力的主要承载部件；在锁模状态下受拉伸产生弹性变形提供锁紧预应力，同时作为移动模板的导向基准 |
| 曲肘连杆组件 | 通过连杆铰接死点位置实现力的放大和自锁，锁模后即使切断液压动力也不会松模 |
| 合模液压缸 | 提供初始驱动力推动曲肘机构动作，输出力经曲肘放大后转化为最终锁模力 |
| 顶出机构 | 集成在移动模板侧，开模后将压铸件从动模型腔中顶出脱模 |
| 调模机构 | 用于调整后模板位置，适配不同厚度的压铸模具，保证曲肘机构伸直后拉杆获得要求的预拉伸量以输出设定锁模力 |

此外，合型机构的标准组成还包括液压系统（压力控制阀、方向控制阀等）、控制系统（PLC或电脑控制方式）以及安全保护装置[[S12,S5]]。

## 工作原理与动作循环

以液压-曲肘合型机构为例，其标准工作循环分为四个阶段[[S5,S15,S10]]：

1. **合模阶段**：合模液压缸驱动曲肘机构展开，带动移动模板快速向定模侧移动。模具即将闭合时自动切换为低压低速模式，以保护模具并检测异物[[S5]]。
2. **锁模阶段**：曲肘继续伸直逐渐接近直线死点位置，移动模板带动拉杆产生弹性拉伸，曲肘力放大效应充分发挥，锁模力持续上升至设定额定值[[S15,S10]]。
3. **锁模保压阶段**：曲肘完全伸直处于死点自锁状态，拉杆弹性变形产生的预应力持续作用于模具分型面，无需持续提供大液压动力即可维持稳定锁模力，刚性好无退让性[[S10]]。在此阶段即使切断输入端的动力，由变形产生的弹性反力也无法打开模具[[S10]]。
4. **开模阶段**：合模液压缸反向动作，曲肘机构解锁向内折叠，带动移动模板平稳返回至最大开模位置，留出足够空间取出压铸件[[S5]]。

据NADCA教材描述，合模-锁模的顺序为：曲肘伸直时先闭合模具，随后接近最高机械优势位置时拉伸拉杆，产生作用于模具的锁紧力[[S3]]。

## 核心参数

合型机构的关键技术参数定义与作用如下[[S20,S9]]：

| 参数名称 | 定义与说明 |
|---------|----------|
| **锁模力**（合型力） | 压铸机模具闭合后，合模机构施加于模具分型面上的静压力[[S20]]。按GB/T 25370—2020定义，是通过拉杆弹性拉伸产生的预紧力总和[[S20]] |
| **拉杆间距** | 四根哥林柱内边缘之间的横向与纵向间距，决定可安装模具的最大外形尺寸 |
| **模板尺寸** | 固定模板与移动模板的平面外形尺寸，是模具安装定位的基准 |
| **最大开模行程** | 移动模板可滑动的最大移动距离，决定开模后分型面之间的最大间隙，需满足取出最高压铸件的取件空间要求[[S9]] |
| **最大允许模具厚度** | 合模机构可安装的最大压铸模具合模厚度，核算时需预留5\~10mm余量[[S9]] |
| **最小允许模具厚度** | 合模机构可安装的最小压铸模具合模厚度，核算时需预留5\~10mm余量[[S9]] |

通用压铸机产品的参数取值范围覆盖锁模力从250kN小型热室机到60000kN以上的大型一体化压铸机全系列[[S4,S6]]。国产压铸机代号中，锁模力以"合型力"为主参数，如J1110表示锁模力为1000kN的卧式冷室压铸机[[S4,S6]]。

## 锁模力计算与胀型力校核

压射时熔融金属在高压作用下充填型腔，会在分型面产生向外的胀型力。合型机构提供的锁模力必须大于该胀型力，以防止模具被胀开而产生飞边缺陷[[S15,S18,S3]]。

锁模力校核公式为[[S15]]：

F<sub>锁</sub> ≥ K × F<sub>胀</sub>

其中F<sub>胀</sub> = P × A，P为金属液作用在型腔内的最大压力，A为铸件在开模方向的总投影面积，K为安全系数（常规取1.2\~1.3）[[S15]]。

若合型机构参数配置不当，锁模力小于实际胀型力，将直接导致模具分型面被胀开，金属液从间隙中溢出产生飞边缺陷，严重时还会损坏模具分型面甚至发生安全事故[[S15,S18,S3]]。

## 在压铸工艺中的协同角色

在压铸生产全流程中，合型机构与其他核心系统按预设时序联动工作[[S16,S12]]：

- 合型机构完成合模锁模动作并确认到位后，压射机构方可启动压射程序将金属液注入型腔[[S16]]。
- 压铸模具完全安装定位在合型机构的两块模板之间，锁模力均匀作用于模具分型面，保证型腔的密封状态[[S16]]。
- 模具温度控制系统实时调节模具的工作温度，配合合型机构的生产节拍，保证铸件在锁模保压阶段的凝固质量[[S12]]。

## 常见故障与排除

合型机构的典型故障模式、产生原因及常规解决方案如下表所示[[S19]]：

| 故障现象 | 可能原因 | 排除方法 |
|---------|---------|---------|
| **锁模力不足**（飞边、尺寸超差） | 曲肘机构不到位、拉杆拉伸预紧量不足、液压系统压力未达设定值 | 重新调整调模机构位置，检测锁模终止开关状态，校准液压系统压力 |
| **曲肘机构磨损卡死** | 曲肘铰接轴套润滑不足、长期过载运行导致销轴磨损变形 | 定期加注润滑脂，更换磨损的销轴与轴套，校准各铰点同轴度 |
| **开合模冲击过大** | 运动速度曲线设置不合理、缓冲装置失效 | 优化开合模速度分段参数，检修液压缓冲阀件 |

锁模行程未终止（锁模终止开关未压到位或解码器参数变化）也是导致无法正常锁模的常见原因，需排查相应的限位开关和参数设置[[S19]]。

## 安全要求

合型机构的安全运行是压铸单元整体安全的重要组成部分。根据GB/T 20906—2007《压铸单元安全技术要求》的规定，合型机构应配备安全防护装置、限位开关、液压阀与监控电路的联动系统，以确保在安全门关闭等联锁条件满足后方可执行合模动作[[S8]]。下图为标准中给出的合型安全装置示意结构，展示了防护装置、限位开关与液压监控回路的连接关系[[S8]]：
