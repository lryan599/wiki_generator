## 概述

气冲造型机是利用压缩空气或燃气瞬间膨胀产生的压力波冲击紧实型砂的造型机，英文对应名称为 air impact molding machine 或 air impulse molding machine[[S25,S22]]。该设备属于铸造设备大类下的气力紧实造型机分支，基础功能是通过瞬时释放的高压冲击波快速紧实砂箱内的型砂，批量制备砂型用于铸件生产[[S22]]。

在铸造造型设备分类体系中，气冲造型机隶属于气力紧实造型机类别，同类别还包含静压造型机，区别于依靠机械震击叠加压实实现型砂紧实的震压造型机，以及依靠单纯高压缓慢施压实现紧实的普通高压造型机[[S6]]。

## 定义与分类

依据 GB/T 5611-2017《铸造术语》，气冲造型指用燃气或压缩空气瞬间膨胀所产生的压力波紧实型砂的造型方法，气冲造型机即实现该方法的设备[[S29]]。在 GB/T 25370-2020《铸造机械 术语》中同样收录了"air impact molding machine"对应术语[[S22]]。

气冲造型机的分类地位如下表所示：

| 分类层级 | 归属 | 相关概念 |
|---------|------|---------|
| 大类 | 铸造设备 → 造型机 | 震实造型机、压实造型机、射压造型机等[[S29]] |
| 紧实方式类别 | 气力紧实造型机 | 静压造型机（气流预紧实+压实）[[S6]] |
| 动力源细分 | 压缩空气气冲 / 燃气爆炸气冲 | 燃气爆炸式已被淘汰，现均指压缩空气式[[S6,S24]] |

**与相似设备的区分要点**：气冲造型机与静压造型机同属气力紧实大类，但气冲造型靠单次或多次气力冲击完成紧实，而静压造型则采用气流预紧实后再加压头压实的复合工艺[[S6]]。

## 工作原理与循环步骤

### 气冲紧实机理

气冲紧实的核心机理为快开冲击阀在极短时间内释放储气室内的压缩空气，形成的瞬时压力波直接作用在松散型砂表面，驱动型砂高速向下运动。型砂撞击下方模板与模样时，砂粒的动能在碰撞过程中转化为紧实功，完成型砂的快速紧实[[S8,S24]]。

紧实过程分为两个阶段[[S16]]：
1. **自上而下加速及初步紧实阶段**：砂箱顶部气压迅速提高，表面层型砂形成初实层，在气压推动下以高速向下移动，使下方砂层加速并初步紧实；
2. **自下而上冲击紧实阶段**：紧实前锋到达模板后发生冲击，靠近模板处紧实度剧烈提高，随后冲击向上发展，型砂由下而上逐层滞止直至砂层顶部。

### 标准工作循环

标准全流程包含五个核心步骤[[S8,S24]]：
1. **举升合腔**：主机举升缸将模板、加满型砂的砂箱和辅助框一起举升，与气冲头组成密闭造型腔；
2. **充气蓄压**：储气室（气包）内储存的高压压缩空气处于待释放状态；
3. **冲击紧实**：快开阀在几毫秒内完全开启，压力波垂直冲击砂型背面，砂粒高速运动撞击模板完成紧实；
4. **排气泄压**：快开阀关闭，排气阀打开，排出砂箱上方余气；
5. **下降起模**：工作台下降，完成回程起模，砂型进入后续工序。

## 整机结构组成

气冲造型机整机核心组成包括以下部分[[S6,S30]]：

| 组件 | 功能 |
|------|------|
| 主机压实缸（举升缸） | 驱动工作台升降，实现合腔与起模 |
| 大容量气包（储气室） | 储存高压冲击用压缩空气 |
| 快开冲击阀 | 几毫秒内完全开启，释放压力波 |
| 模板穿梭机构 | 单次循环内自动完成模板更换 |
| 起模机构 | 保证砂型脱模精度，避免砂型损伤 |
| 加砂系统（含砂斗） | 向砂箱内定量填充型砂 |
| 气冲头 | 作用面与砂型背面贴合，传导冲击波 |
| 辅助框（填砂框） | 补偿紧砂过程中砂柱高度的压缩量 |
| 液压气控系统 | 驱动各执行机构动作 |
| 电控系统 | 协调各工序时序与联锁控制 |

## 改进型气冲造型机

由于单一气冲造型存在砂型顶部紧实度不足、高模样深凹区"架桥效应"、对设备机架循环冲击导致疲劳开裂等问题，行业陆续开发了多种衍生改进构型[[S18,S19]]：

| 改进构型 | 工作原理 | 针对问题 |
|---------|---------|---------|
| 两级气冲造型机 | 第一次气冲预紧实难紧实部位，第二次加砂后全面冲击 | 高模样大吊砂构型紧实度不足[[S18]] |
| 增压气冲造型机 | 小冲击阀弱气冲预紧实 + 大冲击阀强气冲紧实 | 高模样间狭窄区域紧实[[S18]] |
| 流态化气冲造型机 | 辅助框引入横向气流使型砂局部流态化 | 避免"架桥现象"[[S18]] |
| 差动气冲造型机 | 栅格冲击阀下加装带孔挡板，冲击波优先沿砂箱四壁作用 | 改善方形砂箱四角紧实度[[S18]] |
| 气冲+压实造型机 | 气冲后再用弹性橡胶压头或多触头压实 | 提高砂型背面紧实度[[S5]] |

## 优缺点

### 相比于传统震压造型机的优势

气冲造型机无需震击工序，仅靠单次冲击即可完成型砂紧实，具有以下优点[[S24,S6,S11]]：

- **生产效率高**：紧实时间仅约0.3 s，造型周期短，实际生产线可达170整型/小时以上；
- **运行能耗低**：极短的冲击时间使压缩空气消耗量减少；
- **砂型整体紧实度均匀**：紧实度自分型面向上逐渐降低，但分布合理；
- **透气性优**：在三种高比压造型工艺中透气性最好，有利于减少铸件气孔缺陷；
- **可利用车间常规气源**：工作压力0.35～0.5 MPa，无需配置高压空压机；
- **设备结构简单**：初始投资与后期维护成本均低于高压造型与静压造型。

### 技术局限性

单一气冲造型机存在以下技术不足[[S19,S11]]：

- **噪声问题**：冲击过程峰值噪声可达95 dB（A），超过标准要求的85 dB（A）上限[[S12]]；
- **密封要求极高**：砂箱、辅助框与气冲头之间的密封失效会导致型砂喷出、砂箱周边硬度不足，引发铸件跑火缺陷；
- **"架桥效应"**：高模样深凹区域容易出现该现象，导致该区域紧实度不足，需加大吃砂量而降低模板利用率；
- **设备疲劳开裂**：强烈的压力波循环冲击会导致机架焊缝与板材出现疲劳开裂[[S11]]；
- **砂型顶部紧实度不足**：顶部约30 mm区域达不到紧实要求，不适用于高度低于150 mm的矮砂箱[[S16]]。

## 与高压造型、静压造型的对比

公开标准对比数据涵盖了三种工艺共25项核心指标差异[[S11]]：

| 对比维度 | 高压造型 | 气冲造型 | 静压造型 |
|---------|---------|---------|---------|
| 紧实原理 | 高压实力紧实 | 强烈冲击波紧实 | 气流预紧实+压实 |
| 砂型紧实度均匀性 | 紧实度高但均匀性差 | 整体一般，高模样间有架桥现象 | 紧实度高且均匀 |
| 砂型透气性 | 较差 | 最好 | 好 |
| 冲击阀开启时间 | — | 0.005～0.01 s | 约0.3 s |
| 砂箱上方升压速度 | — | ≥30 MPa/s | 约5 MPa/s |
| 紧实时间 | 较长 | 约0.3 s | 较长 |
| 噪声 | 微震加砂压实时噪声大 | 气流冲击时噪声大 | 噪声低（70～80 dB） |
| 设备结构 | 复杂 | 简单 | 复杂 |
| 一次投资 | 高 | 低 | 高 |
| 运行成本 | 高 | 低 | 高（约为气冲的3倍） |
| 生产率 | 低 | 高 | 一般 |
| 对设备机架影响 | 较大 | 最大（循环冲击疲劳开裂） | 一般 |
| 起模性 | 有残余应力，起模困难 | 起模好，但架桥下方有缺陷 | 最好 |

## 砂型紧实度分布特性

气冲造型的砂型紧实度分布具有典型的梯度特征[[S16]]：

- **靠近模底板处**：紧实度最高；
- **随距离模底板高度上升**：紧实度逐步降低；
- **砂型顶部**：紧实度最低。

该分布特性使砂型分型面及型腔部位保持高紧实度，保证铸件表面光洁与精度，同时砂型整体透气性更优，可有效降低铸件气孔缺陷风险。但与震压紧实方式的对比表明，单一气冲造型砂型最上部约30 mm区域紧实度不足，因而不适用于高度小于150 mm的矮砂箱造型[[S16,S19]]。

## 型砂性能要求

气冲造型属于高比压造型类设备，对型砂有专门的性能要求[[S17,S3,S10]]：

| 型砂指标 | 要求 |
|---------|------|
| 湿压强度 | 0.17～0.21 MPa |
| 含水量 | 2.8%～3.5% |
| 紧实率 | 36%～45% |
| 透气性 | ≥80～120 |
| 砂温 | ≤40℃（或比模板温度高5～10℃） |
| 型砂状态 | 松散，无结块、砂团及杂物 |

## 应用场景

### 传统砂型铸造应用

气冲造型机广泛应用于汽车、拖拉机（农机）、通用机械、轻纺等行业，是大批量砂型铸造流水生产线的核心设备之一[[S6,S19]]。国内外主要供应商包括原瑞士GF公司、原德国BMD公司、西班牙LORAMENDI公司、国内济南铸锻所、苏州铸造机械有限公司等[[S8,S2]]。设备砂箱适配范围为长500～2200 mm、宽400～1524 mm、高180～600 mm，生产率覆盖10～200型/小时[[S19]]。

GF公司和BMD公司在20世纪80—90年代曾大量生产气冲造型机，被国内外铸造厂大量采用。但到20世纪90年代末，单一气冲造型机已几乎不再生产，使用的气冲造型机也逐渐被改进型气冲造型机或静压造型机所代替[[S19]]。

### 与压铸领域的关系

经多关键词跨优先级检索，**未发现气冲造型机可直接生产压铸配套冷芯盒、热芯盒砂芯的权威公开文献或产业应用案例**。气冲造型机属于砂型制备专用设备，可作为砂型铸造配套单元，适配低压铸造、重力铸造等常与压铸配套的砂型浇注工艺场景，但不属于专用制芯设备[[S20,S4]]。

## 与射芯机的关系辨析

气冲造型机与射芯机（冷芯盒、热芯盒射芯机等）属于铸造装备体系中功能完全不同的两类设备[[S28,S9,S7,S1]]：

| 对比维度 | 气冲造型机 | 射芯机 |
|---------|-----------|--------|
| 核心功能 | 压缩空气瞬间释放压力波紧实整体砂型 | 压缩空气携带芯砂高速射入芯盒，同步完成填砂与紧实 |
| 产出物 | 砂型（用于形成铸件外形） | 砂芯（用于形成铸件内腔） |
| 设备分类归属 | 造型设备 | 制芯设备 |
| 硬化方式 | 不涉及硬化（湿型砂直接使用） | 热芯盒需加热硬化，冷芯盒需吹气硬化 |
| 适用场景 | 大批量流水线中大型粘土湿型砂砂型生产 | 中小规格树脂砂砂芯、壳芯生产，配套砂型铸造或低压/重力铸造 |
| 典型规格 | 砂箱长度500～2200 mm | 中小尺寸芯盒 |

## 相关标准

国内现行有效针对气冲造型机的行业标准包括[[S12]]：

**JB/T 5773-91《气流冲击造型机基本参数》**
- 规定气冲头气包内压缩空气的工作压力（气冲工作压力）等基础参数阈值。

**JB/T 5547-91《气流冲击造型机技术条件》**
- 电气系统需符合 GB 5226 要求；
- 液压系统需符合 GB 3766 要求；
- 顶升液压缸内油液清洁度不大于 10 mg/100 mL；
- 余隙内平均升压速率不低于 23 MPa/s；
- 工作台上平面 1000 mm 长度内平面度公差为 0.16 mm（只许中间凹）；
- 工作台上平面与冲击头底面的平行度公差在 1000 mm 长度上为 0.5 mm；
- 气冲噪声声压级不得超过 85 dB（A）；
- 首次大修前工作寿命不低于 12000 小时。

## 关联概念

依据相关术语标准[[S29,S26]]，与气冲造型机紧密关联的铸造术语定义如下：

- **气冲造型**（air impact molding）：用燃气或压缩空气瞬间膨胀所产生的压力波紧实型砂的造型方法；
- **紧实度**（degree of ramming）：型砂紧实后的压缩程度，可用密度或砂型硬度表示；
- **紧实率**（compactability）：由型砂紧实前后体积变化率表示，是衡量型砂适宜水分的指标；
- **起模性**（liftability）：型砂对起模过程的适应能力，通常用型砂韧性表征，含水分少或粘土多的型砂韧性差，起模时易损坏砂型；
- **砂型硬度**（mold hardness）：砂型抵抗压划或磨损的能力，与型砂强度和紧实度有关；
- **辅助框**（prefiller）：机器造型时放置在砂箱上的框，用以补偿紧砂过程中砂柱高度被压缩；
- **高密度紧实法**（high density compaction）：使砂型平均密度达1.5～1.7 g/cm³的紧实方法，气冲造型属于此类[[S25]]。
