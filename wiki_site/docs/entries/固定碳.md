## 概述

固定碳（Fixed carbon，FC）是碳质燃料工业分析（Proximate analysis）体系下的核心组分之一，属于非挥发性固体可燃碳组分。在压铸（die casting）工艺语境中，“固定碳”并非压铸辅料或产品的常规出厂检测指标，而是更多地作为理解特定含碳辅料（如早期石墨基模具润滑剂、传统铸造涂料用石墨粉）高温行为与残留特性的分析概念出现[[S11,S9]]。本章节将依据通用工业分析标准方法阐明其定义、计算逻辑，并结合现有压铸及相近铸造领域资料，说明其在该领域可迁移应用的可能性与明显的局限性。

## 定义与计算逻辑

固定碳并非通过元素分析直接测定的一种物质，而是一个在完成水分、灰分、挥发分三项指标独立测试后通过差减法计算得到的参数值。其核心通用计算逻辑为：在同一分析基准下，固定碳质量分数 = 100% − 水分质量分数 − 灰分质量分数 − 挥发分质量分数，工业分析四大组分总和恒为100%[[S16,S3,S15,S1]]。因此，固定碳表征的是样品中除去可蒸发水分、高温可氧化燃烧的挥发分以及完全燃烧后残余无机灰分之后的非挥发性可燃固体组分占比[[S10]]。

以GB/T 2001《焦炭工业分析测定方法》为例，焦炭的固定碳无需直接实测，需先依次完成全水分/空气干燥基水分测定、815℃±10℃马弗炉灼烧至恒重的灰分测定、隔绝空气条件下的挥发分测定，再通过差减法计算得到固定碳含量[[S20]]。该流程可视为碳质物料固定碳通用分析逻辑的典型标准化实施路径。

## 标准测试方法

固定碳并无可独立执行的标准，它的获得完全依赖于水分、挥发分、灰分的标准测定。以下标准构成了固定碳计算的基础：

-   **GB/T 2001**《焦炭工业分析测定方法》，明确规定了焦炭的灰分、挥发分测定条件及固定碳差减法计算程序[[S20]]
-   **ISO 562** 是国际通用的矿物燃料挥发分测定标准，其规定的隔绝空气、特定温度下加热测得的挥发分结果，是固定碳计算的核心输入参数之一[[S1]]
-   **ASTM D3172** 是美国材料与试验协会制定的煤和焦炭工业分析通用基础标准，固定碳属于该标准覆盖的四大工业分析组分之一，采用差减法完成测定[[S1]]

> ⚠️ 重要提示：上述标准均原生面向煤炭、焦炭等碳质燃料，并非压铸行业辅料分析标准。在压铸领域应用时，需将其作为物料基础性质分析的参考框架，而非工艺质量验收依据[[S11]]。

## 与压铸及铸造领域的关联

### 铸造涂料用石墨的固定碳指标

在消失模铸造和砂型铸造领域，石墨是铸铁涂料常用的耐火粉料，其固定碳含量具有明确的分级体系与质量要求[[S8,S11]]。

**表 1 石墨按固定碳含量分级**

| 类别 | 固定碳含量质量分数（%）[[S8,S11]] |
| :--- | :--- |
| 低碳石墨 | 50.0 ~ 79.0 |
| 中碳石墨 | 80.0 ~ 93.0 |
| 高碳石墨 | 94.0 ~ 99.0 |

**表 2 铸造涂料用石墨质量指标**

| 项目 | 指标要求 |
| :--- | :--- |
| 固定碳含量 | ＞ 75% [[S12,S17]] |
| 挥发分 | ≤ 6% [[S12,S17]] |
| 灰分 | ≤ 5% [[S12,S17]] |
| 水分 | ≤ 2% [[S12,S17]] |
| 硫分 | ≤ 2% [[S12,S17]] |
| 耐火度 | 1700 ~ 2000℃ [[S12,S17]] |

高固定碳含量的鳞片石墨耐火度超过2000℃，在大气中不易燃烧，能够提供优良的高温稳定性和化学惰性[[S8,S11]]。此特性也是早期压铸领域在其模具润滑剂中使用石墨的根本原因。

### 压铸领域中的早期应用与现代转型

早期压铸实践中，曾广泛使用含有大量石墨或铝颜料的糊状、膏状模具润滑剂。石墨凭借其高熔点（约1650℃）和良好的高温润滑、脱模性能，满足了压铸工艺对脱模剂“转移润滑”和隔离的基本需求[[S9,S19]]。然而，此类传统配方的固有缺陷在自动化、高洁净度要求的现代压铸生产中日趋突出——石墨等高碳颜料在模具表面积聚残留，导致铸件表面染色、污染，并可能恶化铸件气密性[[S9]]。

现代主流压铸脱模剂已基本完成从“含颜料配方”到“无颜料配方”的转型。当前广泛使用的水基脱模剂，通常利用油或蜡的稳定水乳液作为载体，固含量低于0.2%，以矿物油等添加剂实现脱模与润滑功能，明确要求在高温下尽可能完全气化或分解，避免形成固态碳残留物，以减少铸件孔隙率升高的风险[[S2,S9]]。

一个典型的物理机制对比是：在消失模铸造场景下，气化模（高分子碳氢化合物）高温热解生成的固定碳会部分沉积在涂料层表面。在高温缺氧条件下，固定碳进一步转化为CO，可导致铸件表面增碳；若碳质残留物大量局部聚集，则会直接引发铸件的夹渣（shrinkage porosity & slag inclusion）缺陷[[S18]]。该机理在逻辑上可迁移解释早期石墨基压铸脱模剂残留可能带来的类似表面缺陷风险，但其并非压铸工况下的专项实验验证结论[[S11,S18]]。

## 相关术语与辨析

固定碳与若干含碳的常见参数在概念上易被混淆，明确区分对正确理解其含义至关重要。

**表 3 固定碳与近似概念的区分**

| 术语 | 所属分析体系 | 定义核心 | 与固定碳的差异 |
| :--- | :--- | :--- | :--- |
| **固定碳**（Fixed carbon）[[S15,S10]] | 工业分析 | 差减法计算得到的非挥发性可燃碳组分 | —— |
| **碳含量**（Carbon content）[[S10,S7]] | 元素分析 | 样品中所有碳元素的总占比 | 包含固定碳、挥发分中碳、碳酸盐中碳等所有碳形态，通过直接测定元素得到，数值通常高于固定碳 |
| **残碳**（Carbon residue）[[S6,S10]] | 工艺/热解分析 | 油品、有机物料热裂解后得到的全部含碳固态残渣 | 残渣包含灰分等杂质，不是纯差减碳组分，与固定碳的定义体系与试验方法完全不同 |
| **焦炭残渣**（Coke residue）[[S15,S14]] | 工艺/产物描述 | 煤或碳质物料热解后得到的实体固体产物 | 是同时包含固定碳和全部灰分的完整固态产物，而非仅指其中差减计算得到的碳组分 |

## 总结

在压铸技术语境中，“固定碳”是一个来源于传统碳质燃料工业分析的概念，其自身并非压铸工艺的专有参数。现有证据表明[[S11,S9,S18]]：压铸行业现行标准中未将固定碳作为各类辅料的常规强制检测指标；现代主流无颜料压铸脱模剂的设计方向是力求零碳残留；早期石墨基润滑剂残留物的影响虽可借助消失模铸造领域“固定碳沉积→增碳/夹渣”的机理进行类比理解，但缺乏压铸条件下直接、完整的因果链证据。因此，对于当代压铸实践而言，理解固定碳概念的核心价值，在于为解析传统高碳辅料的残余物特性提供理论基础，而非作为脱模剂、涂料性能的直接评价指标。
