## 概述

预发泡机（英文：pre-foaming machine）是消失模铸造白区制模工部的核心专用设备，正式收录于GB/T 25370-2020《铸造机械 术语》。其基本功能是将含有指定发泡剂的可发性泡沫塑料原始珠粒（如EPS、EPMMA、STMMA）通过加热软化膨胀至目标直径或堆密度，且珠粒间保持独立而不互相融合，获得具有残余发泡剂的预发泡珠粒[[S1,S11]]。

## 分类

预发泡机可按以下维度进行分类[[S31,S24,S33]]：

| 分类维度 | 类别 |
|---------|------|
| 发泡驱动原理 | 真空预发泡机、蒸汽预发泡机 |
| 作业连续性 | 间歇式预发泡机、连续式预发泡机 |
| 自动化程度 | 半自动预发泡机、全自动预发泡机、底卸式预发泡机 |
| 特殊品类 | 全电热一体化预发泡机（面向无蒸汽源生产场景） |

消失模铸造所需珠粒量远小于泡沫板材，通常采用间歇式预发泡机[[S2]]。

## 结构与组成

各类预发泡机的通用核心组成包括[[S1]]：

- **发泡室**：采用不锈钢板制作，筒体可为单层或双层（带加热夹层）结构；
- **搅拌系统**：由搅拌器（含动杆与不动杆）、电动机及调速传动系统组成；
- **加料与出料机构**：间歇式为人工或自动定量加料，气缸开门机构出料；连续式为风动加料与连续自动出料；
- **加热介质及参数控制显示系统**：含蒸汽温度/压力显示、压缩空气压力显示、料位显示等；
- **设备机架**。

真空预发泡机的显著结构特征是筒体为带夹套的双层结构，夹套中通蒸汽或导热油加热，加热介质不直接接触珠粒[[S13,S3]]。

## 工艺作用与工作原理

预发泡机处于原始珠粒供给工序与熟化工序之间，是控制最终泡沫模样密度的关键装备[[S2,S1]]。其工艺原理是：将可发性珠粒加热至玻璃化转变温度以上，珠粒软化并具有塑性，内部发泡剂受热气化产生压力使珠粒膨胀形成互不连通的蜂窝状泡孔；蒸汽（或真空+加热）持续作用使泡孔进一步胀大，冷却后珠粒尺寸即被固定[[S17]]。

蒸汽是公认的实现低密度预发泡的理想介质。实验数据表明，蒸汽对聚苯乙烯薄膜的渗透速度是氮气的4000倍、二氧化碳的136倍、空气的120倍[[S19,S5]]。

### 蒸汽预发泡

蒸汽预发泡机将蒸汽直接通入发泡腔接触珠粒完成加热发泡。设备通过光电料位传感器进行容积定量控制：当预发泡珠粒在筒体内上升至光电管设定高度时，自动发出停止进汽并卸料的信号，实现预发泡密度定量控制[[S33,S19]]。

蒸汽预发泡的工艺特点是：珠粒直接与蒸汽接触，卸料后珠粒含水率约10%（质量分数），必须配套热风干燥工序将含水率降至2%以下方可进入熟化[[S33,S8]]。

### 真空预发泡

真空预发泡机的加热介质（夹套内蒸汽或导热油）不直接接触珠粒，珠粒发泡由加热与抽真空双重作用驱动发泡剂加速气化逸出实现。发泡完成后喷水雾冷却定型[[S13,S4]]。

真空预发泡可实现极低预发泡密度（EPS可达16 kg/m³，EPMMA可达20 kg/m³），珠粒含水率低，无需配套蒸汽源和干燥熟化设备，特别适合无蒸汽源的小型工厂及科研院所[[S31,S24,S13]]。

## 关键工艺参数

### 蒸汽预发泡参数

真空预发泡与蒸汽预发泡的核心工艺参数有明显差异[[S13,S3,S7,S4]]：

| 参数项 | 真空预发泡 | 蒸汽预发泡 |
|--------|-----------|-----------|
| 真空度 | -60~-80 kPa (0.06~0.08 MPa) | — |
| 抽真空时间 | 20~30 s | — |
| 加热方式 | 夹套间接加热（介质不接触珠粒） | 蒸汽直接接触珠粒 |
| 容积定量方式 | 无 | 光电料位传感器 |

国内中小工厂广泛采用的间隙式蒸汽预发泡机典型性能参数[[S19,S33]]：

| 参数项 | 数值/范围 |
|--------|----------|
| 预发桶尺寸 | φ500 mm × 600 mm |
| 一次投料量 | 1~2 kg |
| 搅拌机转速 | 30~250 r/min |
| 最大蒸汽压力 | 0.03 MPa |
| 最高工作温度 | 145 ℃ |
| 发泡倍率 | 40~60 倍 |
| 适应原始珠粒粒径 | 0.2~1.0 mm |
| 适用材料 | EPS、EPMMA、STMMA |

### 不同材料的预发温度

蒸汽预发泡机针对不同材料需设定不同预发温度[[S32,S26]]：

| 材料 | 预发温度范围 |
|------|-------------|
| EPS | 100~105 ℃ |
| STMMA（共聚树脂） | 105~115 ℃ |
| EPMMA | 115~120 ℃ |

### 预发泡密度范围

- 真空预发泡可实现的极限低密度：EPS为16 kg/m³，EPMMA为20 kg/m³[[S13,S7]]
- 国际EPS包装级预发泡珠粒的工业化量产密度范围一般为20~40 kg/m³，对应预发倍率约16~32倍[[S9]]

## 后处理衔接：熟化与干燥

### 蒸汽预发泡后的干燥

蒸汽预发泡后珠粒含水率约10%（质量分数），须经25~35 ℃热风干燥处理至含水率2%以下方可进入熟化工序[[S8,S32,S27]]。

### 熟化处理

预发泡珠粒出料后因骤冷，泡孔内发泡剂与水蒸气冷凝形成负压，须在通风专用熟化仓中静置使空气渗入泡孔，实现泡孔内外压力平衡，珠粒恢复弹性[[S5,S14]]。

熟化工艺核心参数[[S34,S17,S27,S3]]：

| 参数项 | 数值/要求 |
|--------|----------|
| 最佳熟化温度 | 20~25 ℃ |
| EPS珠粒熟化时间 | 4~12 h |
| STMMA/EPMMA珠粒熟化时间 | 8~24 h |
| 熟化仓容量 | 1~5 m³ |
| 防静电措施 | 仓体与管路接地，通风环境 |

熟化合格后的珠粒方可送入发泡成型机配合专用发泡模具完成二次发泡成型[[S30,S12]]。

## 缺陷与工艺限制

### 操作关键点

蒸汽预发泡设备的两个关键工艺要求[[S19,S33]]：
1. 蒸汽进入不宜过于集中，压力和流量不能过大，以免造成结块、发泡不均匀，甚至部分珠粒因过度预发而破裂；
2. 珠粒直接与蒸汽接触导致含水率偏高，卸料后必须经过干燥处理。

### 三类典型缺陷

| 缺陷类型 | 特征 | 后续影响 |
|---------|------|---------|
| 预发不足 | 珠粒未充分膨胀，表面凹凸不平，达不到目标低密度[[S13,S3]] | 泡沫模样密度偏高，充型时发气量过大→铸件产生皱皮、冷隔、浇不足、炭黑缺陷、浇注反喷断流[[S25,S22]] |
| 过发 | 珠壁被发泡剂冲破，后续存储中收缩瘪塌[[S13,S3]] | 泡沫模样强度不足，搬运与振实过程中易破损→铸件轮廓失真[[S25]] |
| 珠粒密度不均 | 不同批次或同批次内珠粒膨胀倍率差异大[[S13,S3]] | 模样不同部位热解速率不一致→铸件局部增碳、表面残留炭渣缺陷[[S25]] |

过高的预发泡温度会加快发泡速度，但易导致过度预发、结块等缺陷，通常不宜将预发温度提得过高[[S13,S25]]。

## 适用材料与兼容性

预发泡机兼容的主流珠粒材料包括[[S31,S33,S26]]：
- **EPS**（可发性聚苯乙烯）：适用于铝合金、铜合金、灰铸铁及中碳钢以上铸件，碳含量较高，用于低碳钢铸件时存在增碳风险；
- **STMMA**（苯乙烯-甲基丙烯酸甲酯共聚树脂）：在解决碳缺陷和发气量大两方面取得较好平衡，广泛用于铸钢和球墨铸铁件；
- **EPMMA**（可发性聚甲基丙烯酸甲酯）：专为消除低碳钢/球铁件增碳缺陷而开发，但发气量和发气速度较大，浇注时易产生反喷。

## 应用场景与压铸关联度

预发泡机是消失模铸造制模工部的专用设备，目前所有正式收录的预发泡机均面向EPS、EPMMA、STMMA等聚合物发泡珠粒的预加工，尚未发现面向压铸场景用于金属熔体泡沫化的专用预发泡设备的明确定义与标准化品类划分[[S23]]。

### 消失模-压铸复合应用案例

陕西法士特汽车传动集团铝合金变速器壳体生产线是预发泡机直接服务于压铸相关成形工艺的典型量产案例[[S21]]：该产线采用德国Teubert TDV-100LF全自动预发泡机完成EPS珠粒预发泡，整条线设计年产18万件合格铝合金变速器壳体，最大铸件尺寸600 mm×580 mm×400 mm，毛坯重约45 kg。
