## 概述

压铸工艺准备是压力铸造正式充型浇铸工序之前所有前置作业的统称，属于压铸生产全流程中压铸机、压铸模、压铸合金三大核心要素有机结合的前置实施环节[[S12,S8]]。其作业目标是保证后续压铸成形工序稳定产出符合质量要求的压铸件[[S12]]。压铸工艺准备覆盖模具准备、金属液准备和压铸机调试三大子过程[[S12]]。

### 模具准备

模具准备子过程涵盖压铸模具结构校验、模具与压铸机安装匹配性校核、模具预热、型腔脱模剂喷涂、模具滑动配合部件间隙调整等步骤[[S13,S20]]。其操作目的是保证模具与压铸机适配、模具温度处于合理区间、铸件可顺利脱模，避免粘模、卡模等异常状况[[S20]]。

模具预热的具体作用包括[[S4,S20]]：
- 避免金属液因激冷而使流动性急剧降低，防止铸件出现欠铸、冷隔或收缩裂纹；
- 减少模具热疲劳应力，防止模具因激热而胀裂；
- 调整模具滑动配合部分的热膨胀间隙，避免金属液窜入间隙；
- 降低型腔中空气的密度，有利于型腔气体的排出。

预热操作须在确认所有其他准备工作完成后进行，预热前应将模具清理干净，预热后须检查各活动部位确保无卡模现象[[S23]]。

### 金属液准备（熔体准备）

金属液准备子过程包含炉料分类清理与预热、熔炼炉与坩埚烘干预处理、合金熔炼、除气精炼、成分检测与温度保温等步骤[[S21,S6,S2]]。其操作目的是获得化学成分合格、气体与非金属夹杂物含量达标、温度符合工艺要求的熔融金属，避免后续铸件出现气孔与夹杂缺陷[[S22]]。

炉料的准备要求包括：全部炉料化学成分必须明确无误；全部炉料必须清洁并经仔细选择；清理好的炉料须存放在干燥通风处；入炉前必须预热以除去表面吸附水分[[S6,S21]]。不同系列牌号的合金不得在同一坩埚内熔炼，应各有专用坩埚[[S21]]。

### 压铸机调试

压铸机调试子过程包含压铸机日常点检维护、压射系统参数校准、合模力校验、配套氮气储能装置压力校准、压射冲头与压室预热、冷却水路通畅性检查等步骤[[S13,S23]]。其操作目的是保证压铸机运行状态符合工艺设定，避免设备带故障运行引发的压射异常、跑料等问题[[S13]]。

设备起动前必须依据点检表全面检查设备，确保设备处于正常状态；设备发生故障或报警时应立即查看原因并报修，严禁设备带病工作[[S13]]。氮气在压铸机中起重要辅助作用，合理充填定量氮气可提高压铸机生产效率并保证压射性能[[S23]]。

## 典型工艺参数

### 模具预热温度

不同合金类型对应的模具预热温度范围有所差异，典型数值如下[[S1,S15]]：

| 合金类型 | 模具预热温度 |
|:---|:---|
| 锌合金 | 218°C 左右 |
| 铝合金 | 288°C 左右 |
| 镁合金 | 260°C 左右 |
| 铜合金 | 500°C 左右 |

### 铝合金除气工艺参数

铝合金压铸常用旋转喷吹除气工艺参数[[S24,S11,S16]]：

| 参数项 | 典型范围 |
|:---|:---|
| 惰性气体 | 氩气（Ar） |
| 气体流量 | 3~10 L/min |
| 处理时间 | 10~30 min |
| 处理后氢含量目标 | < 0.15 cm³/100g Al（约0.15 mL/100g Al） |

合理的除气处理时间可有效降低密度指数（DI）并减小拉伸性能波动[[S24]]。

### 脱模剂喷涂参数

压铸脱模剂喷涂操作中，稀释后的水基脱模剂需匹配模具型腔结构设定差异化喷涂参数[[S19,S5,S25]]：

| 参数项 | 控制要求 |
|:---|:---|
| 压缩空气压力 | 根据模具结构匹配设定 |
| 不同部位喷涂时长 | 根据型腔复杂程度差异化设置 |
| 涂料稀释比例 | 根据涂料品牌与工艺要求确定 |
| 喷涂顺序 | 按工艺规范分部位依次执行 |

喷涂操作兼具模具冷却与润滑双重作用，保证开模后铸件顺利脱离型腔[[S5]]。

## 配套设备与工具

压铸工艺准备阶段涉及的核心设备及作用如下[[S22,S8,S9]]：

| 设备/工具 | 作用 |
|:---|:---|
| 熔炼炉 | 将炉料加热熔化，制备初始合金液 |
| 保温炉 | 将熔炼完成的金属液维持在稳定的浇注温度区间 |
| 除气装置 | 通入惰性气体或熔剂，去除合金液中的氢与悬浮氧化物 |
| 自动喷涂系统 | 向模具型腔均匀喷涂脱模剂，实现润滑与控温 |
| 测温仪 | 实时监测金属液与模具的温度 |
| 定量浇注装置 | 向压射室精确输送定量金属液 |

铝合金熔化设备主要采用坩埚炉或无坩埚连续熔化保持炉，后者具有熔化速度快、熔化能力大（可达100~4500 kg/h）、氧化烧损少等优点[[S22]]。

## 准备不当引发的缺陷

### 模具预热操作不当

模具预热不足时，金属液接触低温模具会发生激冷，流动性急剧下降，引发铸件欠铸、冷隔、收缩裂纹等缺陷；同时模具交变热应力增大，将大幅缩短模具使用寿命[[S4,S3,S18]]。若预热温度过高，则易导致铸件粘模及表面氧化缺陷[[S18]]。

### 金属液除气不充分

铝合金熔体中氢含量超过0.15 mL/100g Al阈值时，凝固过程中氢会从过饱和固溶体中析出，在铸件内部形成弥散分布的针孔类气孔，大幅降低铸件致密度与力学性能[[S24,S2]]。经后续高温热处理时，还会出现表面鼓泡缺陷[[S24]]。

### 脱模剂喷涂不均匀

型腔部分区域若未覆盖足够脱模剂，金属液充型后会与高温模具合金面直接发生粘结反应，引发粘模缺陷；开模时铸件表面被撕裂，产生拉伤、缺肉等表面缺陷[[S5,S7]]。此外，铸件脱模阻力过大还可能导致顶出机构过载变形[[S7]]。

## 相关关系

压铸工艺准备与压铸生产全流程的上下游工序紧密衔接。在完整的压力铸造工艺流程中，压铸工艺准备（含模具准备、合金液准备、设备准备）位于合金熔化与压铸成形工序之间，承接前段熔体及模具准备作业，为后续充型浇铸提供必要的设备、模具与金属液条件[[S12,S14]]。准备阶段各项操作的执行质量直接影响压铸件的尺寸精度、内部致密度、表面质量以及模具的使用寿命[[S4,S18,S24]]。
