## 概述

粘模（die sticking）是压铸过程中熔融铸造合金粘附于模具型腔或型芯表面的一种常见冶金缺陷，以铝合金压铸最为突出。该缺陷严重时不仅导致铸件在脱模过程中表面撕裂、缺肉或多肉，还会引发脱模困难与模具过早失效，显著降低生产效率与铸件品质 [[S50,S47]]。根据国标术语体系，压铸粘模在制品端表现为固定位置重复出现的凹痕（粘附物痕迹），或铸件与型腔壁粘连产生的沿脱模方向的拉伤痕迹（粘模拉伤） [[S23,S58,S25]]。

## 术语辨析

压铸领域涉及粘模现象的英文术语存在使用语境上的差异，需加以区分：

| 英文术语 | 使用语境 | 说明 |
|---|---|---|
| **die sticking** | 压铸粘模的通用标准术语 | 广义上指熔融合金与模具表面发生粘附的一类现象 |
| **die soldering** | 特指焊合型粘模 | 描述高温下铝合金与钢模发生金属间化合反应的冶金类粘附，属于粘模的核心子类 |
| **soldering** | 高压压铸行业中与模具/合金交互作用直接相关的术语 | 同等用于指代焊合类粘模及其导致的模具失效 |
| **mold sticking** | 泛指各类模具粘滞现象 | 并非压铸领域专属术语，使用范围更泛 |
| **metal adhesion** | 广义金属粘附概念 | 泛指一切金属与模面的接触附着，不构成压铸粘模的标准专名 |

## 亚类与分类

工业界将压铸粘模划分为两大类基础亚类 [[S16,S66]]：

1. **化学类粘模（die soldering）**：由铝合金与钢质模具表面发生冶金反应，生成Fe-Al系金属间化合物造成的粘附，是粘模最核心的机理类型。
2. **物理类粘模**：由金属液在高压下嵌入模具表面微凹坑、粗糙峰谷或热疲劳裂纹中，通过机械咬合作用形成粘附，不涉及界面化学反应 [[S57,S27,S9]]。

此外，生产现场还可按发生部位和工况，将粘模进一步场景化分类 [[S66]]：
- **冲蚀型粘模（impingement soldering）**：发生於内浇口附近受高速金属流股正面冲击的区域。
- **沉积型粘模（deposition soldering）**：发生在充型流速较低的模具表面区域，由局部高温长时间接触促成。

## 产生机理

### 氧化层/防护膜冲刷破裂（前置条件）

高速熔融合金液首先破坏模具表面的氧化层、氮化防护层或脱模剂润滑膜——即"washout"冲蚀失效，暴露出无保护层的新鲜钢基体后才能触发后续的界面反应与元素互扩散 [[S16,S54,S63]]。

### 元素互扩散

裸钢模具与熔融合金直接接触时，模具基体中的Fe元素向合金液侧溶解扩散，同时合金液中的Al元素向模具钢表层渗透。界面处的元素浓度梯度随接触时间延长持续升高，为后续金属间化合相的生成提供物质基础 [[S27,S16,S64]]。

### 临界温度与液相生成——焊合机制的核心

对于纯铝-铁体系，当模具表面局部温度超过 **655 °C**（Al-Fe共晶温度）时，界面区域会生成富铝液相以及Fe₂Al₅、FeAl₃等固态金属间化合物 [[S40,S30,S4]]。该液相在凝固后直接将铸件与钢模连接焊合，形成牢固的焊合粘模。

在含硅铝合金中，还会进一步生成Al₈Fe₂Si等三元金属间相 [[S19,S52]]。高压压铸工况下的粘模反应可以在等于或低于合金液相线温度的条件下发生，局部凝固时间是决定粘模严重程度的核心控制因素 [[S52,S2]]。

对ADC12铝合金，安全模具温度区间推荐为 **220 °C ~ 280 °C**，当模温超过280 °C时，厚大区域长时间处于高温状态极易出现严重粘模 [[S11]]。

### 低熔点机械锁合机制（物理粘模）

当模具型腔表面粗糙度偏高（存在微凹坑或热疲劳裂纹）时，高压下的熔融合金嵌入这些微观凹陷结构，凝固后通过机械咬合与模具牢固结合，显著提高粘模发生概率 [[S57,S27,S9]]。模具表面的凹槽、毛刺以及脱模斜度不足的部位尤其容易发生此类机械锁合型粘模。

## 合金亲和力与成分影响

不同压铸合金的粘模倾向存在根本性差异：

- **锌合金**：与铁的亲和力极低，常规压铸条件下几乎不出现粘模，仅当模具温度控制严重失当、缺乏表面防护时，偶有锌元素在模具表面累积形成锌基结合层 [[S42,S18,S33]]。
- **镁合金**：与铁的亲和力远低于铝合金，即使使用较小的脱模斜度也极少发生粘模，模具寿命较铝合金压铸高出4–5倍 [[S56,S42,S9]]。
- **铝合金**：与钢模在高温下极易发生冶金反应，粘模倾向最为突出，尤以低铁含量的铝合金（如铁含量低于0.1%）粘模风险最高。少量掺入0.05% – 0.1%的铁即可改善低铁铝合金的粘模问题 [[S15]]。

铝合金粘模倾向随合金系变化呈现明确排序—纯铝的粘模倾向最高，其次是铝镁合金、亚共晶铝硅合金、铝硅铜合金，共晶铝硅合金的粘模倾向最低 [[S64]]。硅含量越高，界面铁铝金属间化合物层的生长速率越低，粘模风险随之降低 [[S64]]。

铝合金中铁含量的调控作用尤为显著：当Fe质量分数接近其在铝液中的饱和溶解度（650 °C时约1.8%）时，界面两侧铁元素化学势梯度大幅降低，粘模倾向显著下降；实验证实Fe含量达1.1%的铝合金，其粘模倾向远低于Fe含量0.8%的同成分合金 [[S64,S47]]。

不同合金元素对粘模临界温度的影响如表所示 [[S30]]：

| 对临界温度的影响 | 合金元素 |
|---|---|
| 提升临界温度（降低粘模倾向） | 钛（Ti）、铬（Cr）、锰（Mn） |
| 降低临界温度（增大粘模倾向） | 镍（Ni）、锌（Zn）、镁（Mg）、硅（Si）、铜（Cu） |

其中铜元素对降低粘模临界温度的作用最为显著。

## 检测与评估

### 人工观察检测

- **铸件侧**：观察铸件表面是否沿开模方向出现拉伤痕迹、缺肉或多肉形貌 [[S41,S25,S59]]。
- **模具侧**：直接观测型腔工作面是否存在合金粘附形成的凸起区域，判断表面平滑度 [[S41,S25,S59]]。

### 粘模严重程度五级评价

行业内通用的粘模严重程度评级体系如下 [[S10]]：

| 级别 | 状态描述 |
|---|---|
| 0级 | 无合金粘附，铸件与模具表面均无异常 |
| 1级 | 仅模具表面点状微量合金残留，铸件表面无肉眼可见痕迹 |
| 2级 | 局部条状合金粘附，铸件表面存在沿开模方向的浅拉伤痕迹，脱模无卡顿 |
| 3级 | 大面积合金粘附，铸件存在较深拉伤缺肉，可正常脱模，缺陷区域可打磨修复 |
| 4级 | 合金大面积烧结，铸件无法正常脱出，强行取出即报废 |

### 数值模拟预判

压铸充型数值模拟可标记两类高风险区域以提前预警粘模 [[S13,S26]]：
1. 金属液流速比周边区域高1.4倍以上且持续冲击型芯的位置；
2. 铸件局部合金液温度保持在515 °C以上超过25 s的位置。

这些高风险区域的预判可用于提前调整工艺或模具设计方案，从而降低实际生产中的粘模发生概率。

## 预防与控制措施

### 合金成分调控

| 措施 | 效果 | 局限 |
|---|---|---|
| 控制铝合金Fe含量在0.6%–0.75%及以上 | 大幅降低铝液与模具铁元素发生反应倾向，从合金侧抑制化合物粘附 [[S8,S14]] | Fe含量过高会显著降低铸件力学性能，尤其是伸长率 |
| Fe含量控制在0.8%–0.9%范围 | 有效抑制粘模倾向 [[S68]] | 同上述力学性能局限 |
| 避免低铁特种合金盲目用于压铸 | 对于需低铁含量以保障导电性能的合金，掺入少量无碍的0.05%–0.1%铁即可改善粘模倾向 [[S15]] | 需结合性能要求衡 |

### 模具设计优化

- **脱模斜度**：粘模高发区域脱模斜度应提升至1.5°以上，完全避免负脱模斜度结构 [[S8,S41]]。GB/T 8844-2003推荐成型高度≤50 mm、表面粗糙度Ra≤0.1 μm时，外表面脱模斜度不小于0.5°，内表面脱模斜度不小于外表面2倍，过小或负向斜度会增大粘结力并诱发粘模 [[S60,S24,S53]]。
- **表面抛光**：粘模易发部位采用600目砂纸精细抛光，消除所有局部凹坑与毛刺 [[S8,S41]]。
- **浇注系统**：引导金属液平行于型腔表面流动，避免正面直冲型壁或型芯，以根本消除高速冲刷导致的冲蚀粘模 [[S58]]。内浇口应设置为切线方向进入型腔。
- **模具表面粗糙度**：成型面粗糙度控制在Ra≤0.1 μm，粗糙表面的局部凸起温升更高，铝液留存更多，显著提升粘模概率 [[S53]]。

### 模具表面处理

| 处理方案 | 效果 | 适用场景与局限 |
|---|---|---|
| **氮化**（离子氮化、软氮化） | 改变模具表层组织，提升表面硬度与抗熔蚀能力，可延长粘模发生周期数倍，成本低 [[S62]] | 长期生产后氮化层仍会被合金液逐渐熔蚀失效 |
| **PVD涂层：TiAlN** | 抗镁合金粘模能力为常规氮化处理的2倍，型芯总生产模次可达27.6万次 [[S65]] | 涂层厚度仅几微米，不能承受长时间高速金属流直接冲刷（如内浇口区域） [[S35,S7]] |
| **PVD涂层：TiN** | 显著提升模具抗铝合金熔蚀能力 [[S7]] | 氧化温度较低，长期高温工况下失效周期较短 |
| **PVD涂层：CrN** | 在720 °C高温下仍保持优异的抗氧化与抗腐蚀性能，适配高浇注温度压铸工况 [[S7]] | 同其他PVD涂层一样，存在冲蚀区域的耐久性约束 |
| **低成本方案：蓝氧化、镀锡、镀硬铬等** | 降低模具表面与合金液的亲和性，有效减少粘模倾向 [[S8,S44]] | 主要适配锌合金等低熔点合金压铸场景 |

### 脱模剂应用优化

| 工艺参数 | 推荐范围/效果 |
|---|---|
| 水基脱模剂常规稀释比 | 1:100，最高可至1:150–1:200 [[S29,S20]] |
| 喷涂压力 | 0.3–0.6 MPa |
| 喷涂距离 | 150–250 mm |
| 喷涂角度 | 垂直±10° |
| 模具表面工作温度 | 150–250 °C |
| 高性能脱模剂（如Safety‑Lube A）350 °C附着量 | 为常规脱模剂的2.15倍，高温点脱模效果更佳 [[S28]] |
| 自动化优化效果 | 六轴机器人+高精度计量泵可将喷涂均匀度提升20%，粘模不良率从3.2%降至0.8% [[S12]] |
| 脱模斜度不足部位 | 选用油基或蜡基脱模剂强化局部润滑效果 [[S29]] |

在保证不出现粘模的前提下，脱模剂用量应控制到最低，避免过量喷涂引起铸件气孔、夹渣等缺陷并堵塞排气通道 [[S29]]。水基脱模剂适配模具温度200–300 °C范围，残留碳化物少、发气量低 [[S34]]。

### 工艺窗口调整

- **模具温度**：浇注温度与模具温度应严格控制在工艺规程范围之内 [[S55,S9]]。模温推荐按经验公式 t<sub>m</sub> = (1/3)t<sub>j</sub> ± 25 °C 设定（薄壁复杂件取上限，厚壁简单件取下限），偏离该热平衡区间会增大异常粘模风险 [[S38,S43]]。
- **充型速度**：控制金属液速度在1000–1400 ips区间，略高于雾化速度即可，经PQ²计算验证工艺，降低冲蚀强度 [[S8]]。
- **冷却系统**：针对粘模高发的铸件厚大热节区域设置随形冷却结构，采用小孔径点冷却系统精准控制局部模温 [[S8,S45]]。硬水结垢会降低冷却管路换热效率，引发局部过热导致粘模 [[S66]]。
- **生产节拍**：热点区域可通过局部喷雾强化冷却并适当延长生产周期以降温，但过度降温将导致铸件出现冷隔等其他缺陷 [[S8]]。

## 相关缺陷辨异

粘模与以下压铸/铸造缺陷在表象上接近，但本质不同，需注意区辨 [[S59,S23]]：

- **与机械拉伤**：机械拉伤仅由模具脱模斜度不足、反向斜度、局部表面压伤或铸件顶出偏斜导致，不存在模面金属粘附层。
- **与冲蚀（die erosion）**：冲蚀是金属液长期高速冲刷使模具材料直接熔蚀或剥离的模具损耗行为，冲蚀区域常伴有粘模现象，但二者属不同类型缺陷。
- **与铸焊**：铸焊为异种金属熔接的制造工序，与粘模的化学焊合子类完全是不同概念。
- **与热裂纹**：热裂纹是铸件凝固过程中在拉应力下沿晶开裂的缺陷，与模具表面金属附着无关联。
- **与粘砂**：粘砂是砂型铸造中熔融金属渗入砂粒缝隙产生的表面附着砂层缺陷，压铸工艺中极少出现。
