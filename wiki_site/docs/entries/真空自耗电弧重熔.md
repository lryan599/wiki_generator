## 概述

真空自耗电弧重熔（英文全称 Vacuum Arc Remelting，缩写 VAR，又称真空电弧重熔、自耗电极真空电弧重熔）是一种在工业领域广泛应用的高端金属二次重熔精炼工艺。其基本原理为：在无渣低压环境或惰性气体保护氛围中，以低电压大电流的直流电弧作为热源，熔化由待精炼金属预先制备的自耗电极；熔化后的金属液滴穿过高温弧区进入水冷铜结晶器，逐步定向凝固成高纯度、组织均匀的铸锭，最终实现材料净化、结晶结构优化与力学性能大幅提升的目标 [[S24,S11]]。

## 工作原理

VAR工艺通常采用正极性熔炼接线方式，即自耗电极接直流电源负极，底部金属熔池接电源正极。工艺全过程在真空室内完成，工作真空度可低至10⁻³ Torr（约0.133 Pa）级别 [[S19]]。在直流电弧加热作用下，自耗电极端头表层首先熔化，熔化的金属在重力作用下沿端面流下，于阴极斑点区汇集成液滴，当液滴达到临界尺寸后穿过弧区落入熔池 [[S1]]。

熔化过程中金属熔滴穿过温度约4700℃~5000 K的弧区，在高真空环境下完成以下精炼反应 [[S16,S1]]：
- 溶解气体（H₂、N₂、O₂）的脱除
- 低熔点杂质的挥发
- 非金属夹杂物在低压高温下的分解与上浮

同时，水冷铜结晶器的强制冷却作用使熔池实现自下而上的顺序定向凝固，从根本上避免常规铸锭常见的缩孔和宏观偏析缺陷 [[S16,S1]]。

## 设备构成

现代VAR炉的核心设计遵循三大原则 [[S3]]：
1. **同轴性**：将阴、阳极电缆近距离平行布置，抵消大电流感应磁场对合金质量的负面影响并提升电效率
2. **再现性**：通过重量传感器与电控系统精准控制熔炼过程的电弧长度与熔化速率，保障批次间质量一致性
3. **灵活性**：支持同一设备生产不同规格的铸锭，适配多样化生产需求

VAR炉由以下核心部分构成 [[S14,S10]]：

| 组成部分 | 功能说明 |
|---------|---------|
| **真空室** | 为重熔全流程提供密闭低压操作环境，杜绝大气污染 |
| **直流电源系统** | 输出25~65 V低电压、数千至数万安培的大电流，采用直流供电（交流易引发电弧断续、劣化铸锭质量） |
| **自耗电极夹持与升降机构** | 固定待熔炼的自耗电极，根据熔速实时调整电极进给位置，维持电弧长度恒定 |
| **水冷铜结晶器** | 高导电导热纯铜材质，内部通2.5~3.5 kgf/cm²冷却水强制冷却；金属接触内壁后迅速形成凝壳，避免熔体与坩埚反应，支撑顺序凝固 |
| **真空系统** | 通常由机械增压泵+蒸气增压泵组合，可在10~20分钟内将炉腔抽至10⁻²~10⁻¹ Pa初始操作真空度，并能连续排出熔炼过程中释放的气体 |

## 工艺过程

标准VAR重熔全流程分为五个阶段 [[S28,S1]]：

1. **电极焊接**：在真空或保护气氛中将自耗电极同轴牢固焊接在电极杆的过渡电极上，保证焊接区域的导电性能满足大电流负载要求
2. **引弧阶段**：在结晶器底部铺垫同材质金属底垫与引弧剂，快速引燃电弧形成初始熔池，随后使用1.1~1.2倍额定熔炼电流稳熔一段时间以消除铸锭底部疏松与气孔
3. **正常熔炼阶段**：保持稳定熔炼电流与电弧长度，自耗电极匀速熔化，金属熔滴连续通过弧区进入熔池完成精炼，同步实现自下而上的连续定向凝固
4. **封顶阶段**：逐级降低输入电流，逐步减小熔化速率使熔池体积不断缩小，将最终缩孔集中在铸锭顶部区域，减少切头量提高成材率
5. **冷却阶段**：熔炼完成后铸锭在真空或惰性气体氛围中冷却至400℃以下出炉

## 工艺变体与联合冶炼

### 真空电弧双极重熔（VADER）

常规单电极VAR工艺的延伸变体——真空电弧双极重熔（VADER），采用两支水平对置的自耗电极作为直流电源两极，引弧后两电极端头熔化汇聚形成熔滴落入旋转的非水冷结晶器中。在熔滴半熔融状态与结晶旋转破碎的共同作用下，最终获得无粗大柱状晶的细等轴晶铸锭，可大幅降低后续热加工能耗 [[S28,S14]]。

该工艺的主要局限性在于 [[S28]]：
- 难以制备直径超过500 mm的大规格铸锭
- 离心力作用下合金元素因密度差异在径向上分布不均，易产生偏析

### VIM-VAR双联工艺

航空航天等高端应用场景广泛采用VIM-VAR双联冶炼工艺：首先通过真空感应熔炼（VIM）获得成分精准可控的初炼铸锭，加工为自耗电极后送入VAR炉进行二次重熔，进一步脱除残留的O、H、N等有害气体与非金属夹杂物，最终得到高纯净度、低偏析的高性能合金 [[S27,S11]]。

### 三联复合冶炼工艺

更高等级的航空高温合金产品可采用三联工艺，主流组合形式包括 [[S18,S15]]：
- VIM + VAR + ESR（真空感应熔炼+真空自耗电弧重熔+电渣重熔）
- VIM + ESR + VAR
- NAV + EBR + VAR（非自耗熔炼+电子束重熔+真空自耗电弧重熔）

通过三次不同原理的精炼叠加，将合金内部的气体含量与夹杂物等级控制在极致水平，满足极端工况下的材料服役需求。

## 面向压铸模具钢的工艺参数

以H13（4Cr5MoSiV1）压铸热作模具钢为例，VAR工艺的典型条件如下 [[S19,S8,S10]]：

| 参数项 | 典型值/范围 |
|-------|-----------|
| 工作真空度 | 10⁻³ Torr（约0.133 Pa） |
| 供电方式 | 直流（DC） |
| 工作电压 | 25~40 V |
| 供电电流（10 t级设备） | 约15 kA |
| 常规熔化速率 | 约1 kg/min |
| 电极与熔池间隙控制精度 | ±2 mm |
| 真空泵组配置 | 三级（机械活塞泵+机械增压泵+蒸气喷射泵） |

## 组织与质量效果

### 纯净度与致密度

VAR工艺可大幅降低H13钢中的O、H、N等溶解气体含量，同时去除大部分大型非金属夹杂物，几乎完全消除常规铸锭中的缩孔缺陷及宏观化学成分偏析问题，最终获得致密度更高、微观组织更均匀的细晶态钢锭，材料的韧性与塑性得到显著优化 [[S24]]。

### 压铸模具钢性能提升

经VAR重熔后的4Cr5MoSiV1（H13）压铸模具钢，与常规大气电弧炉熔炼的同牌号钢相比 [[S13,S21,S34]]：

| 性能指标 | 提升效果 |
|---------|---------|
| 纯净度与组织均匀性 | 显著提升 |
| 纵向/横向性能差异 | 大幅缩小（各向同性改善） |
| 疲劳强度 | 提升25%以上 |
| 可硬化性 | 同韧性水平下硬度可高出4~6 HRC，不影响耐磨性 |
| 冲击韧度 | 明显提升 |
| 导热性 | 正向改善，有助于模具快速散热 |
| 铝合金压铸模具整体服役寿命 | 可提升25%~50% |

VAR精炼的H13钢在多向同性与纯净度指标上优于同材料经电渣重熔（ESR）处理的试样，特别适合大截面H13压铸模具以及横向延伸率、冲击韧性、耐疲劳性能要求高的H13模具产品 [[S13]]。

## 与其他熔炼工艺的对比

在压铸模具钢领域，VAR与电渣重熔（ESR）、真空感应熔炼（VIM）各有侧重 [[S23,S33,S24]]：

| 对比维度 | VAR | ESR | VIM |
|---------|-----|-----|-----|
| 工作环境 | 真空 | 大气 | 真空 |
| 脱气能力 | 强，高效去除H/O/N | 弱，存在吸氢风险 | 有限，脱氮效果差 |
| 脱硫能力 | 无法脱硫 | 强，通过渣洗反应高效脱硫 | 不适用 |
| 去除非金属夹杂 | 有效去除氧化物类大型夹杂 | 深度吸附去除各类非金属夹杂 | 受坩埚耐火材料污染限制 |
| 去除高蒸气压有害杂质（Pb、Bi等） | 高效 | 有限 | 可部分去除 |
| 铸锭表面质量 | 较差，需扒皮 | 光洁，无需额外机加工 | 取决于后续处理 |
| 铸锭截面形状 | 通常为圆形 | 可制备矩形截面 | 取决于模具 |
| 大规格无偏析铸锭能力 | 优于ESR | 限于较小规格 | 受限于凝固条件 |
| 生产成本 | 高于普通大气熔炼 | 较低，无需真空设备 | 需真空感应炉投资 |

## 在压铸领域的应用

VIM-VAR双联或三联精炼的高温合金、高纯模具钢作为高端压铸领域的特种原材料，主要应用于 [[S18,S27,S2]]：
- 大吨位压铸机的大型H13模具
- 高熔点合金压铸模具（如压铸铜合金用3Cr2W8V等）
- 压射冲头、料筒（压射室）等压铸机核心高负荷耐磨耐热部件

在上述应用中，VAR精炼材料可显著提升部件的抗热疲劳与耐冲蚀服役性能。

## 标准与规范

### 中国标准

- **GB/T 34565.1《热作模具钢 第1部分：压铸模具用钢》**：明确将VAR钢列为四类压铸模具用热作模具钢质量等级之一，并规定了对应的取样规则、12项检验项目清单，以及气体含量检测、夹杂物评级、低倍组织检验的指定试验方法标准 [[S4]]
- **GB/T 1299《合金工具钢》**：收录4Cr5MoSiV1（H13）等压铸用热作模具钢牌号，允许高端品级采用VAR精炼工艺 [[S12,S2]]

### 美国标准

- **ASTM A597《铸造工具钢》**：明确允许采用真空重熔（含VAR）工艺生产工模具钢 [[S9]]
- **ASTM A681《合金工具钢》**：针对H系列热作模具钢的高端品级，指定VAR或VIM-VAR为强制精炼路径，以满足航空工业与高端压铸装备领域的高纯净度性能要求 [[S5,S26]]

## 优缺点总结

| 项目 | 内容 |
|-----|------|
| **优势** | 高效脱气（H/O/N）；去除高蒸气压有害杂质；无渣相引入，元素烧损可控；可获得大规格无宏观偏析圆截面铸锭；化学成分稳定性高；组织致密均匀 [[S23,S33]] |
| **局限** | 无法实现脱硫；对硫化物夹杂去除能力弱；铸锭表面质量差，需额外扒皮工序；必须预先制备自耗电极，增加工序和成本 [[S23,S14]] |
