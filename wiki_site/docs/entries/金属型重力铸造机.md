## 定义与概述

金属型重力铸造机（别称重力压铸机[重力铸造机]、永久模铸造机，英文称 Gravity Die Casting Machine / Permanent Mold Casting Machine）是用于金属型重力铸造方法生产铸件的专用设备。其核心特征为：依靠地球重力作为唯一驱动力将熔融金属注入可重复使用的金属铸型内部完成充型 [[S18, S28]]。

设备标配开合型、抽芯、顶出机构，可选配驱动系统、控制系统与模具冷却系统 [[S28]]。在国家标准 GB/T 25370-2020《铸造机械 术语》中，金属型重力铸造机归类为第 7.1.1 类设备，同时存在金属型重力铸造流水线对应分类 [[S18]]。

## 分类

依据 GB/T 25370-2020 的分类体系，金属型重力铸造机可按照以下维度归类 [[S28]]：

| 分类维度 | 类型 | 说明 |
|----------|------|------|
| 操作方式 | 手动金属型铸造机 | 以人力操纵机构完成开合型动作 |
| 操作方式 | 机动金属型铸造机 | 借助气动、电动或液压装置完成开合型、抽芯等动作 |
| 模具分型方式 | 垂直分型重力铸造机 | 模具采用垂直分模面 |
| 模具分型方式 | 水平分型重力铸造机 | 模具采用水平分模面 |
| 结构功能 | 倾转式重力铸造机 | 合模系统可在一定角度范围内倾转 |
| 产线形式 | 多工位金属型铸造机 | 由多台单机搭配输送转盘组成自动化生产线 |

其中，固定式重力铸造机（无倾转功能）是实际生产中的常见基础形式，合模系统保持固定姿态，金属液由上方浇口直接注入型腔，结构简单、造价低、操作维护便捷，适合形状简单、充型流动路径短的中小型铸件批量生产 [[S9, S29]]。

> **术语歧义说明**：在北美工业语境中，"die casting"通常特指高压压铸，而金属型重力铸造被称为 "permanent mold casting" 或 "gravity die casting"；英国语境中则用 "gravity die casting" 指重力金属型铸造 [[S12, S25]]。中文语境下 "重力铸造机" 若未特别指明，通常指金属型重力铸造机 [[S28]]。

## 结构与组成

金属型重力铸造机的主要功能机构构成如下 [[S28]]：

| 机构名称 | 功能 |
|----------|------|
| 开合型机构（合模系统） | 打开和闭合金属铸型，在金属液充型和铸件凝固全过程中提供足够合模锁紧力，避免胀型跑液 |
| 抽芯机构（边模机构） | 在开模前完成侧置活动型芯的抽出与复位，适配复杂内腔铸件生产 |
| 顶出机构（脱模机构） | 在开模后将完全凝固的铸件从型腔中平稳顶出，避免铸件变形损伤 |
| 倾转机构 | 倾转式机型专用，驱动整套合模系统按预设角度程序完成倾转 |
| 冷却系统 | 通过嵌入铸型内部的风冷/水冷流道控制铸件各区域冷却速度，实现顺序凝固，消除缩松缺陷 |
| 驱动系统 | 为开合型、抽芯、倾转等动作提供动力（气动/电动/液压） |
| 控制系统 | 对倾转速度、合模力、模具预热温度、冷却介质流量等参数实现可编程精确控制，保障不同批次铸件质量一致性 |

## 工艺作用

金属型重力铸造机在整套金属型重力铸造工艺中承担以下协同角色 [[S14, S33, S17, S9]]：

1. **合模锁紧**：设备锁紧金属铸型并提供合模力。
2. **温度管理**：作业前控制系统控制加热模块将金属型预热至指定温度（铝合金场景 200~300℃，灰铸铁场景 250~350℃）。
3. **浇注承载**：操作人员或机械手将符合温度要求的合金液注入浇注系统，依靠重力充型进入预热完成的铸型型腔。
4. **凝固控制**：在冷却系统温控作用下，铸件按预设顺序完成凝固。
5. **自动取件**：设备到达设定冷却延时后自动执行开模、抽芯、顶出动作取出铸件。
6. **循环准备**：触发清模、喷涂层流程，完成下一轮生产准备。

基础的作业循环流程为：安放型芯 → 合模锁紧 → 人工/机械手浇注金属液 → 保温 10~60 s 完成凝固 → 开模 → 顶出铸件 → 清理铸型并喷涂保温/润滑涂层 → 进入下一轮循环 [[S14, S33]]。

|----------|----------|
| 铝合金 | ≈ 760℃ |
| 镁合金 | ≈ 650℃ |

### 模具温度条件

- 铝合金铸件：模具预热温度 200~300℃ [[S14, S3]]
- 灰铸铁铸件：模具预热温度 250~350℃ [[S14, S3]]
- 黄铜浇注：模具常规工作温度 200~450℃ [[S7]]
- 铝青铜浇注：模具最高允许工作温度可达 650℃ [[S7]]

### 倾转工艺参数

在典型铝合金缸盖倾转铸造场景中，倾转工艺分为三个阶段，总倾转时长通常设置为 10 s [[S21, S5]]：

| 阶段 | 角度范围 | 说明 |
|------|----------|------|
| 起步加速 | 0~5° | 缓慢启动 |
| 匀速倾转 | 5~85° | 金属液平稳层流充型 |
| 减速停稳 | 85~90° | 平稳停止 |

倾转工艺可实现金属液平稳层流充型，有效减少卷气、氧化夹杂缺陷，大幅降低铸件气密性废品率 [[S5]]。

## 与其它铸造设备的差异对比

### 与砂型铸造设备

砂型铸造设备每生产一件铸件都需重新制备一次性砂型，而金属型重力铸造机配套的金属铸型可重复使用 10,000~25,000 次，生产效率显著更高，铸件尺寸精度更高、表面光洁度更好，晶粒更细小、力学性能更优，但前期模具投入成本更高，适合大批量铸件生产场景 [[S31, S15, S25]]。

### 与低压铸造设备

低压铸造依靠密封炉膛内的压缩空气驱动金属液沿升液管反重力向上充型，充型压力通常低于 0.5 MPa，铸件金属利用率可达 90%~95%；金属型重力铸造依靠重力充型，需额外设置浇道和冒口用于补缩，浇冒口金属占比最高可达 50%，铸件整体金属利用率仅为 50%~70%。设备和模具投入成本显著低于低压铸造设备，但充型可控性弱于低压铸造 [[S9, S8, S24, S16]]。

### 与高压压铸设备

高压压铸依靠压射系统在极高压力（最高可达 140 MPa）下以极快速度将金属液注入型腔，生产效率更高，可铸造 0.6 mm 级别的极薄壁厚铸件，但铸件内部易卷入大量气体产生气孔缺陷，多数无法进行热处理强化；金属型重力铸造无高压压射过程，金属液充型速度平缓，铸件内部致密度更高，允许进行完整热处理提升力学性能，但不适合超复杂薄壁铸件生产 [[S32, S6, S35, S12]]。

## 优势与局限性

### 优势

- 生产效率远高于砂型铸造，金属铸型可重复使用 10,000~25,000 次 [[S31, S15]]
- 铸件晶粒细小，力学性能更优，尺寸精度更高、表面光洁度更好 [[S31, S25]]
- 铸件疏松远小于砂型铸件，具有良好的气密性 [[S31]]
- 设备和模具投入成本显著低于低压铸造和高压压铸设备 [[S9, S8]]
- 铸件内部致密度高，允许进行完整热处理以提升力学性能 [[S32, S12]]

### 局限性

- 前期模具成本高，小批量生产不经济，适合大批量生产场景 [[S31, S15]]
- 铸件金属利用率仅为 50%~70%（需冒口补缩），低于低压铸造的 90%~95% [[S9, S16]]
- 充型可控性弱于低压铸造，重力浇注条件下更易裹入空气 [[S9]]
- 超复杂薄壁铸件生产受限 [[S32, S35]]
- 常规固定式浇注存在排气条件差、人为因素影响稳定性等工艺难点 [[S5]]

## 典型应用

金属型重力铸造机适配的典型合金范围包括铝合金、铜合金，同时可扩展适配镁合金、锌合金等非铁金属，也可生产灰铸铁类黑色金属铸件 [[S14, S27]]。

| 合金类别 | 典型零件/应用场景 |
|----------|-------------------|
| 铝合金 | 汽车发动机缸体、缸盖、活塞、转向节、支架、车轮、泵轮等 [[S14, S8]] |
| 铜合金 | 卫浴零件、阀门配件等民用产品 [[S14]] |
| 灰铸铁 | 小件铁素体灰铁/球铁铸件 [[S8]] |

应用领域覆盖汽车制造、航空航天、电力机械等行业 [[S14]]。
