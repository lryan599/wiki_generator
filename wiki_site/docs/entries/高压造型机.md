## 定义与术语定位

在铸造领域，“高压造型机”是一个术语界限清晰的概念。根据国家标准GB/T 25370-2020《铸造机械 术语》，高压造型机被明确定义为**采用压实比压大于700kPa的压实造型机**[[S37]]，其对应标准英文术语为**high pressure molding machine** [[S2,S24,S26]]。国家基础标准GB/T 5611-2017《铸造术语》进一步补充，高压造型工艺的压实比压范围为**700~1500kPa**，行业主流选型约为**1.3MPa** [[S38,S17]]。

必须明确指出：**高压造型机属于砂型铸造设备范畴**，其核心功能是对型砂施加高比压以制取高紧实度、高强度的砂型。压铸机/高压铸造机（high pressure die casting machine）属于金属特种铸造设备，功能是将熔融金属在高压下高速充填金属模具型腔。两者在应用工序、核心功能和设备原理上完全不同，不应混淆 [[S25,S22]]。

## 分类

高压造型机按多种维度可形成系统的分类体系 [[S36,S23,S16]]：

**按紧实方式：**
- 上压紧实法：压头与辅助框位于砂箱上方，向下施压将型砂压入砂箱
- 下压紧实法：辅助框设置在模板上，模板向上运动将型砂压入砂箱
- 差压/双向紧实法：砂箱上下方均设置辅助框，同时实施上下双向施压

**按主要结构特征：**
- 有箱高压造型机 / 无箱高压造型机
- 垂直分型高压造型机 / 水平分型高压造型机
- 平压头高压造型机 / 成形压头高压造型机 / 多触头高压造型机
- 单工位高压造型机 / 多工位高压造型机

上述分类体系中，多触头高压造型机是最核心的子类。该机型采用由若干独立小触头组成的压头进行紧实，触头能随模样外形自动调整压实行程，从而获得紧实度分布更均匀的砂型 [[S38,S37]]。

## 工艺作用与工作流程

高压造型机处于砂型铸造流程的核心紧实工位时段，布置于配砂工序之后、合箱工序之前 [[S23,S36]]。典型的单工位多触头高压造型机工作流程如下 [[S33]]：

| 步骤 | 工作内容 |
|------|----------|
| 1 | 空砂箱进入，工作台回升至接砂位置 |
| 2 | 定量加砂斗加砂，同时预震 |
| 3 | 多触头压入，压实加微震 |
| 4 | 保压一定时间 |
| 5 | 起模，砂型顶出 |
| 6 | 砂型推出，准备下一循环 |

现代机型常同时采用压实与微震复合紧实方式，可按工艺需要分别选择：纯压实、震击后压实、震击压实同步、或预震后震击压实同步等紧实模式 [[S8]]。

## 技术参数

多触头高压造型机的核心技术参数覆盖砂箱规格范围与紧实能力。根据行业资料与产品标准（JB2847-79），主要参数范围如下 [[S19,S27,S32,S34,S8]]：

| 参数 | 典型范围/指标 |
|------|--------------|
| 压实比压 | 0.7~1.5 MPa（常规选型约1.3 MPa） |
| 多触头机型比压调节范围 | 0.5~15 kgf/cm²（约0.05~1.5 MPa），无级可调 |
| 砂箱内框尺寸 | 最小数百mm级，最大可达1600×1600 mm |
| 砂箱高度 | 250~450 mm |
| 压实力 | 数十吨~数百吨 |
| 生产率 | 150~360 箱/h |
| 压实机构工作油压 | ≥5 MPa |
| 震击机构压缩空气压力 | ≥0.5 MPa |

国外主要生产商包括德国KW公司、BMD公司，瑞士GF公司，日本三菱公司、新东公司，美国CE公司等；国内生产商包括苏州铸造机械有限公司、济南铸锻所捷迈机械有限公司等 [[S31,S35]]。

## 结构与组成

现代多触头高压造型机的典型结构组成包括以下核心部件 [[S8,S19,S34]]：

- **定量加砂机构**：负责向砂箱精准供给定量型砂，常见重力加砂与射砂填砂两种方式
- **多触头压实机构**：分为浮动式（被动式）和主动式两类。浮动式各触头油缸相互连通形成封闭联通器，压实阻力自适应；主动式每只触头带有独立驱动油缸。部分机型触头尺寸可调，适配多品种生产需求
- **液压控制系统**：为压实、起模等核心动作提供高压动力，常规工作油压≥5MPa
- **气动微震机构**：压实过程中同步施加微震，提升型砂流动性，降低紧实阻力，缩短压实节拍
- **四立柱导向机架**：采用高强度铸造或焊接结构保证整机刚性。部分机型（如BMD公司FPE系列）采用封闭力系压实设计，压实力不传递至地基，机架受力条件改善
- **起模机构**：配置大行程导向柱保证起模精度，避免砂型起模回弹破损
- **模板快换装置**：适配多品种铸件生产快速换模需求

## 型砂与铸件材料

高压造型工艺对型砂性能有较高要求。型砂需具备**高流动性、高透气性、高强度**的特性，关键参数指标如下 [[S20,S23]]：

| 型砂参数 | 要求范围（高比压造型） |
|----------|----------------------|
| 含水量 | 2.8%~3.5% |
| 紧实率 | 36%~45% |
| 透气性 | ≥80~120 |
| 湿压强度 | ≥0.15 MPa |
| 砂温 | ≤40℃ |

型砂性能检测执行GB/T 2684《铸造用砂及混合料试验方法》，对含水量、含泥量、紧实率、透气性、强度等关键指标规定了标准的试样选取和试验方法 [[S18]]。高压造型机适配的铸件材质覆盖各类铸铁件、铸钢件及铸造非铁合金件，广泛适用于汽车、工程机械等领域的大批量砂型铸件生产 [[S23,S10]]。

## 缺陷与工艺影响

高压造型机的比压参数选择直接关联铸件质量与砂型性能：

- **比压过低**：铸型紧实度不足、均匀性差，引发铸型变形、铸件胀砂、尺寸超差。砂型局部紧实度不足会降低砂型抗金属液冲刷能力，金属液渗入砂型间隙，减缓铸件冷却速率，显著提升内浇口附近出现缩孔、缩松缺陷的概率 [[S23,S12,S29]]
- **比压过高**：砂型透气性显著降低，易引发铸件气孔、夹砂；同时加剧起模回弹导致砂型破损，增加废型率；还会提升对设备和工装的强度、刚度要求 [[S23,S31]]

## 应用场景与技术局限性

**主要应用领域** [[S23,S31,S5]]：
- 汽车行业：缸体、缸盖等大批量铸铁件
- 拖拉机和工程机械：壳体、桥壳等中小型铸钢及铸铁件
- 阀门、车辆等复杂铸件

高压造型线多组成自动化流水生产线使用，铸铁件生产节拍可达300箱/时以上，铸钢件砂箱尺寸约1000mm时机型生产率可达100箱/时以上。高压造型工艺生产的铸件表面粗糙度可达Ra12.5~Ra6.3，铸件尺寸精度和重量稳定性高于普通震压式造型工艺 [[S23,S5]]。

**技术局限性** [[S36,S31,S23]]：
- 设备结构庞大复杂，初期购置投资和维护成本高
- 对砂箱、模板、模框等工装的刚度和加工精度要求高，工装制备周期长、成本高
- 高比压下砂型透气性偏低，若参数控制不当易引发铸件气孔缺陷
- 搭载气动微震机构的机型作业噪声较大
- 与新兴气冲造型机、静压造型机相比，紧实度均匀性仍有差距，新建生产线已较少选用高压造型机

## 结论

综合以上国家及行业标准、铸造专业文献的系统证据，**高压造型机一词在铸造领域的标准用法明确指代砂型高比压压实造型设备**，定义边界清晰，标准英文对应术语为 high pressure molding machine。该设备以压实比压大于700kPa为核心技术特征，主要用于制取高紧实度、高尺寸精度的砂型，服务于大批量砂型铸件生产。该术语与压铸机/高压铸造机（high pressure die casting machine）属于不同工艺领域的铸造设备，不应混用或相互替代。若在压铸语境下需表述高压金属液充型设备，应使用“压铸机”或“高压铸造机”等专门术语。
