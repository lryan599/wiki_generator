## 概述

转运包（transfer ladle）属于浇包类专用子类型容器，其核心功能限定为跨工位转运金属液，而不直接执行向铸型或压射室的定量浇注[[S2,S8]]。在压铸生产中，转运包承担将熔炼炉或保温炉产出的合格金属液安全、保温地输送至压铸机前保温炉的关键角色[[S10]]。中文别名包含金属液转运包、铝合金转运包、保温转运包[[S8]]。

## 定义

国家标准GB/T 25370-2020《铸造机械 术语》将转运包定义为“在熔炼炉和/或保温炉和浇注工位之间转运金属液的浇包”，是浇包容器的功能细分子类[[S2]]。铸造行业标准汇编补充定义为“把金属液从熔炼炉输送到另一个熔炼炉、保温炉或浇包的容器”[[S8]]。英文正式术语为transfer ladle，等效行业术语含transport ladle、hot metal transfer ladle[[S8]]。

## 与相近设备的功能边界

转运包的定位与以下邻近设备间存在清晰界定[[S2,S6]]：

- **普通浇包**：兼有转运与直接浇注功能；转运包仅定位跨工位转运，不直接执行向压射室的定量浇注。
- **保温包**：优先实现长时间恒温存储，用于等待或暂存；转运包以移动转运为第一功能，适配厂区内短周期流转需求。
- **自动给汤机**：对接保温炉的末端小容量定量舀取装置，单次取料量对应单模浇注量，远低于转运包100 kg以上的承载能力。

## 结构与组成

压铸场景下常见转运包可按承载与操作方式分为三种结构形式：倾转式（自带倾转机构）、吊包式（配置标准吊耳适配起重机）、叉车配套式（底部预留叉装工位）[[S1,S2]]。

核心组成部件包括[[S2,S8]]：

- **包体**：由钢板焊接制成。
- **耐火内衬**：砌筑或捣实在包体内壁的耐火材料层。
- **包盖**：可开合的保温包盖，用于减少温降。
- **加热保温装置**：可选配，以维持转运过程中的金属液温度。
- **倾转驱动机构**：倾转式所配，用于绕水平轴转动包体倒出金属液。
- **吊耳**：吊包式配置，用于悬挂起吊。
- **适配组件**：叉装工位及其他转运载体配合结构。

## 工艺角色与使用流程

在压铸车间，转运包从集中熔炼炉或保温炉处接取已精炼完毕的合格金属液，随后经厂区内运输送至各台压铸机前保温炉完成交付[[S1,S10]]。

使用前置条件与操作要点[[S3,S10]]：

1. **烘包**：接取金属液前须完成烘包操作，整体预热温度应达到800～850℃，以完全排干内衬残余水分。
2. **接液**：由集中溶解炉放出的铝合金液温度控制在720～760℃区间，精炼除气温度为700～750℃，精炼过程中保持液面平稳，尽量缩短保温时间以控制氧化[[S10]]。
3. **精炼与镇静**：金属液须经过除气、除渣处理后，在700～720℃条件下静置镇静约20 min，方可装入已预热的转运包内输送[[S10]]。
4. **转运与交付**：转运中需测控温度，可采用叉车、起重机、轨道运输等载具完成配送[[S1]]；配置密封保温盖及加热保温装置可有效抑制温降[[S1]]。

## 关键参数与工作条件

压铸用转运包的典型参数如下表所示。

| 参数项 | 取值范围或说明 | 来源 |
|---|---|---|
| 额定容量 | 100～1000 kg；部分工业版可达3 t级 | [[S1,S5]] |
| 适用合金 | 铝合金、锌合金、镁合金等压铸常用有色金属 | [[S1]] |
| 接液温度（铝合金） | 720～760℃ | [[S3]] |
| 烘包预热温度 | 800～850℃ | [[S3]] |
| 自然温降速率（无辅助加热、无保温盖） | 约3～8℃/10 min | [[S1]] |
| 配置密封保温盖后温降速率 | 约2～5℃/10 min | [[S1]] |
| 推荐转运载体 | 叉车、起重机、轨道自动转运车 | [[S1,S4]] |

注：温降速率为常规工况下参考值，实际速率受包体保温状况、环境温度与转运距离等因素影响。

## 缺陷、局限与控制要求

转运包使用过程中的主要质量问题与控制要求如下：

- **温度损失**：金属液在输运过程中温度下降显著，需要实时测控温度；缺乏保温和包盖保护将加速温降[[S1,S10]]。
- **氧化与造渣**：转运前的精炼过程应保持液面平稳、合理控温、缩短保温时间以减少氧化；精炼后经镇静处理可促进夹杂物上浮清除[[S10]]。
- **内衬维护**：包衬为耐火材料，需通过烘包确保干燥，使用过程中应关注内衬使用寿命与定期维护[[S2,S3]]。
- **清洁要求**：集中溶解炉需至少每12 h清理一次[[S3]]；转运包内残留金属及熔渣需定期清理，以减少交叉污染风险。

## 相关关系

转运包在压铸生产线中处于上下衔接的关键位置，与以下设备和操作环节紧密关联[[S1,S2,S6,S10]]：

- **集中熔炼炉/保温炉**：接液输入端，需与转运包配合完成出液与转运衔接。
- **保温炉（压铸机前）**：接收端，转运包将金属液注入后，由给汤机自保温炉中定量舀取并供给压射室。
- **给汤机（自动浇注系统）**：末端定量浇注执行装置，与转运包功能区隔。
- **精炼与除气工序**：转运前必须完成的预处理工序，包括除气除渣和静置镇静。
- **烘包器及预热系统**：用于接液前的包衬预热干燥。
- **AGV及自动转运系统**：大型一体化压铸车间中，AGV（自动导引车）已用于熔融金属、渣包及料饼的自动化流转，可与转运包配合实现全流程无人转运[[S4]]。
- **管道输运系统**：与转运包形成替代关系的一种更先进的输运方式，采用管道密封与电加热保温，由液体泵驱动金属液在管道内流动，输送过程安全、自动且节能[[S10]]。

## 分类方式

按操作与配置方式可将压铸用转运包分类如下[[S1,S2]]：

| 分类依据 | 类型 | 特征说明 |
|---|---|---|
| 结构/操作方式 | 倾转式转运包 | 自带倾转机构，绕水平轴转动包体倾倒金属液 |
| | 吊包式转运包 | 配置标准吊耳，适配起重机吊运 |
| | 叉车配套式转运包 | 底部预留叉装工位，由叉车叉运 |
| 保温能力 | 有保温/加热型转运包 | 配置密封保温盖与/或加热保温装置，温降较慢 |
| | 无保温型转运包 | 无保温盖及加热装置，温降较快 |
| 转运载体 | 叉车型、起重机（吊车型）、轨道自动转运车型、AGV型 | 适配不同车间物流环境 |
