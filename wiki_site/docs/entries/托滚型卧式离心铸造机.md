## 概述

托滚型卧式离心铸造机是卧式离心铸造机的主要类型之一，通过托轮（又称托滚）系统支撑并驱动管状铸型（型筒）绕水平轴线旋转，利用离心力完成金属液充型、凝固的铸造设备。在国家标准GB/T 25370-2020《铸造机械 术语》中，其官方定义为：通过四个托轮支撑型筒，并由其中一个（或两个）托轮依靠摩擦力驱动型筒绕水平轴线旋转的离心铸造机，标准英文术语为wheel-support type horizontal centrifugal casting machine，行业内亦常称为滚轮式、滚筒式卧式离心铸造机[[S27]]。

该机型在离心铸造设备体系中属于卧式离心铸造机的三大核心子类之一，与悬臂型卧式离心铸造机和夹紧型卧式离心铸造机并列[[S27,S12]]。相较悬臂式机型，其无悬臂主轴结构，依靠两对托轮共同承载铸型重量，可适配更大规格的长轴类铸件生产，属于间接大载荷驱动类型[[S11,S7]]。

## 分类与定位

托滚型卧式离心铸造机在离心铸造设备分类体系中的位置如下：离心铸造机分为立式、卧式两大主类，卧式主类进一步细分为多工位、滚轮式（即托滚型）、悬臂式三个子类，悬臂式下又包含单头、双头两个亚型[[S8,S29,S17]]。

从多维度分类角度看，托滚型卧式离心铸造机可归入以下分类维度[[S1]]：

| 分类维度 | 归类 |
|----------|------|
| 按旋转轴 | 卧式（铸型绕水平轴线旋转） |
| 按铸型工艺性质 | 真离心铸造（铸件内腔不用型芯） |
| 按铸型材料 | 金属型为主 |
| 按铸型温度 | 冷模或热模离心铸造 |

## 结构与组成

### 托轮/托滚系统

托轮系统是该设备的核心支撑与传动部件。整机共设置四个托轮：两个为带突缘的主动托轮，突缘与铸型外侧的箱箍配合以约束铸型轴向窜动；另外两个为无突缘的被动托轮，随铸型同步从动[[S7,S28]]。

**支撑角参数：** 主动托轮与被动托轮的圆心与铸型圆心连线的夹角θ取值范围为95°~120°，工程常规设计取值为105°~115°[[S7,S16]]。部分工程指南推荐取120°，允许在110°~123°范围内取值[[S28]]。两托轮轴中心距l可通过以下公式调整以适配不同直径的铸型：

\[
l = 2\cos\alpha(R_{\phi} + r),\quad \alpha = 90^{\circ} - \frac{\theta}{2}
\]

其中，\(R_{\phi}\)为铸型与托轮接触处的外半径（型轨半径），\(r\)为托轮自身半径[[S2]]。

**托轮设计精度要求：**

| 设计参数 | 要求 |
|----------|------|
| 材质 | 45钢、40Cr等，经正火或淬火处理，表面硬度高于铸型外接触轨道 [[S26]] |
| 直径与型轨直径比 | 1:1~1:2.5 [[S26,S2]] |
| 托轮宽度 | 80~200mm [[S26,S2]] |
| 圆柱度、同轴度 | 按GB/T 1184-1996标准达到或优于8级 [[S26,S2]] |
| 两传动轴中心线平行度 | 需保证高平行度，避免铸型旋转偏摆 [[S2]] |

**托轮轴向间距可调的三类主流结构方案[[S9,S2]]：**

- **主动轴断开单轮驱动式**：电动机与一个托轮连接，另一组托轮可沿机座梯形槽移动，对铸型长度适应性强，但因单轮启动力矩不平衡，不宜用于极长铸件。
- **三段式活节传动轴式**：传动轴分三节，生产较短铸件时可去掉中间一节，适用于批量生产。
- **滑道式可移动托轮式**：一组托轮装在滑道上靠滑板移动调整托轮距，灵活性最高，多用于中小型机型。

### 铸型（型筒/管模）

该设备配套的管状铸型称为型筒或管模，常用材质为铸钢、球墨铸铁或低合金铸钢（如21CrMo10）。铸型外部设置与托轮配合的环形型轨，两端采用带销钉锁紧的端盖密封，可适配不同长度、直径的铸件生产[[S30,S20]]。

### 传动机构

传动形式包含三类[[S9,S28]]：

- **V带弹性传动**：电动机通过V带驱动主动轴，V带松紧可调，属弹性启动，但铸型转速稍有损失。
- **单电机直连主传动轴传动**：电动机轴直接与主传动轴连接，转速准确、结构简化，但属硬性启动，启动功率消耗大，电动机安装精度要求高。
- **双电机分别驱动两根主传动轴同步传动**：大型设备多采用此方案，部分机型设置通轴同步结构保证两主动托轮转速一致。

大型托滚型卧式离心铸造机多采用60kW级直流电机搭配晶闸管无级调速系统[[S28]]。托轮距与铸型总长度的比值取值范围为0.55~0.8[[S9]]。

### 浇注系统

设备多配置三向可调式移动浇注小车，浇注台沿轨道进给，可根据铸件内径调整浇口高度和横向位置，将浇口伸入旋转的铸型内部完成进给式浇注，浇注完成后拉出[[S7]]。部分小型机型配套固定式旋转浇注装置[[S26]]。

### 冷却与辅助系统

该设备可设置以下辅助系统[[S7,S30,S28]]：

- **外喷淋水冷却结构**：实现铸型强制冷却，加速金属液凝固。
- **压缩空气驱动涂料喷涂装置**：含带搅拌的密闭涂料罐，通过通入压缩空气使涂料喷入具有一定温度的铸型内，完成内壁涂挂。
- **可调压轮装置**：设置在铸型上方，用于抑制旋转振动、限制铸型径向跳动。
- **自动拔管机**：通过驱动装置自动从高温型筒内取出成形的铸件。

### 衍生机型——托滚型水冷金属型离心铸管机

该衍生整机包含浇注系统、移动离心主机、拔管机、液压站、轨道桥架、运管小车、控制系统共8个核心模块，可连续批量生产直径1000mm以下的球墨铸铁输水管道[[S15]]。

## 工作原理与流程

托滚型卧式离心铸造机依靠托轮与铸型外表面之间的摩擦力带动型筒旋转[[S7]]。其完整工作流程为[[S7,S26,S24]]：

1. **托轮间距调整**：根据铸件外径调整主动托轮与被动托轮的间距，达到预设支撑角。
2. **启动加速**：驱动系统启动，使铸型升至设定工作转速。
3. **浇注定型**：三向可调式浇注小车沿轨道将浇口伸入旋转铸型内部，完成定量金属液浇注；金属液在离心力作用下沿铸型内壁充型、凝固。
4. **制动停机**：制动机构使铸型减速停止。
5. **拔管取件**：拔管装置将成形的管状铸件从型筒内取出。

## 核心属性与特点

### 优势

- **大承载能力强**：无悬臂主轴结构，依靠两对托轮共同承载铸型重量，可承载远大于悬臂机型的铸件重量[[S12,S11,S7]]，最大承载铸件重量可达十几吨[[S6]]。
- **装卸便捷**：铸型装卸无需与主轴法兰对接，操作方便[[S9,S24]]。
- **适配性强**：可通过调整托轮间距适配不同长度、不同直径的铸件，设备改造成本低[[S9,S24,S28]]。
- **动平衡稳定性好**：沿长轴方向可布置多组托轮支撑长铸型，旋转动平衡稳定性优于悬臂机型[[S28,S25]]。
- **成型质量优良**：铸件绕水平轴线旋转，成型出的管状铸件内孔平直均匀，无立式离心铸造机因重力与离心力矢量叠加导致的铸件上薄下厚、内孔呈旋转抛物面的缺陷[[S18,S6]]。

### 局限性

- **摩擦传动风险**：极端工况下存在铸型与托轮相对打滑的风险，转速存在小幅损失[[S9,S24,S28]]。
- **占地面积大**：整体占地面积相比悬臂型机型更大[[S9]]。

## 关键工艺参数

### 转速与适用尺寸

| 参数 | 范围 |
|------|------|
| 铸型工作转速 | 150~800 r/min [[S33,S34]] |
| 铸件最大外径 | 1100 mm [[S33,S34]] |
| 铸件最大长度 | 标准型4000 mm，特殊定制机型最长可达8000 mm [[S33,S34]] |
| 最大承载铸件重量 | 可达十几吨 [[S6]] |

对比悬臂式机型（最大铸件尺寸约为φ350mm×340mm，转速840~1170r/min），托滚型机型的尺寸容量显著更大[[S23,S12]]。

### 浇注温度

不同材质合金的典型浇注温度区间如下[[S31]]：

| 合金类型 | 典型浇注温度区间 |
|----------|-------------------|
| 铸铁 | 高于液相线60~120℃，常规1250~1380℃ |
| 铸钢 | 1500~1650℃ |
| 铜合金 | 950~1200℃ |

### 铸型涂料与型温控制

铸型涂料需根据铸件材质选用对应配方[[S20,S5]]：

- 铸铁缸套：常用硅砂类水基涂料。
- 铸钢铸件：采用以锆英粉为骨料的高温涂料。
- 有色合金铸件：用石膏或滑石粉基涂料。

铸型需预热至150~350℃后完成喷涂操作，保证涂料均匀附着并完全干燥，以避免产生铸件气孔、白口等缺陷[[S20,S5]]。

## 应用领域

托滚型卧式离心铸造机主要用于工业领域长轴类、筒类对称回转铸件的规模化生产，适合生产长径比大于1的大尺寸管状铸件[[S13,S10]]。典型稳定产品包括[[S13,S10,S21]]：

- 内燃机缸套
- 铜/铸铁/钢质轴套、轴瓦
- 输水/化工用离心铸管
- 造纸机干燥滚筒
- 轧钢机用轧辊
- 冶金炉底辊、滚道辊
- 双金属复合件（如钢背铜合金轴瓦、复合辊套）

## 局限性、常见缺陷与设备诱因

托滚型卧式离心铸造机在运行中可能产生如下缺陷，其根源与设备设计的运行条件密切相关：

- **成分偏析**：托轮同轴度偏差、两侧传动轴平行度不足会引发铸型轴向窜动和周向转速波动，是导致铅青铜等易比重偏析合金出现径向成分不均的核心设备诱因[[S26,S6]]。
- **裂纹**：铸型旋转偏摆导致的非正常振动可引起铸件内部残余应力分布异常，进而诱发裂纹[[S26,S6]]。
- **气孔**：托轮磨损不均匀导致的铸型非正常振动会引发金属液裹气，最终形成铸件气孔缺陷[[S26,S6]]。

## 相关标准与规范

现行公开相关标准包括[[S27,S30,S19]]：

- **GB/T 25370-2020《铸造机械 术语》**：明确了托滚型卧式离心铸造机（7.5.5）、托轮（7.5.12）、型筒（7.5.11）、多工位托滚型卧式离心铸造机（7.5.6）、托滚型斜卧式离心铸造机（7.5.7）的术语定义规范。
- **ASTM A667/A667M《离心铸造双金属（灰口和白口铸铁）气缸》**：对托滚型机型生产的双金属离心铸件的力学性能、材料层占比、检测方法作出强制要求。

## 安全、安装与运维

安装与安全要求[[S16,S25]]：

- 大型托滚型卧式离心铸造机需全部安装在地下基坑内，中型机可半露出地面，所有机型必须加装防护安全罩。
- 操作前必须详细检查离心铸造机的每个部件紧固情况、安全罩防护状态、电器线路情况是否正常安全。
- 操作工具、模具、浇口和浇包等必须充分预热，上涂料的需充分干燥。
- 金属液必须严格定量，脱模必须在停机后进行。
- 采用冷却系统时须严格检查是否有损漏，保持地面干燥。
