## 概述

模流-结构联合仿真技术（又称多尺度联合仿真方法、基于统一网格的模流-结构联合仿真方法）是针对注塑/压铸成型场景开发的专项耦合数值模拟方法。该技术由华中科技大学材料成形与模具技术国家重点实验室开展核心研究，将模流分析得到的完整成型热机历史（温度场、应力场、纤维取向等）作为初始条件定向迁移至制品结构力学分析中，实现从模内流动到制品服役性能的全链路仿真[[S27,S33]]。

区别于通用多尺度仿真（常覆盖微观晶粒至宏观构件的跨尺度求解），该技术聚焦注塑/压铸场景下宏观模流场-热场向宏观结构力学的定向映射，仅在纤维增强材料场景下额外叠加介观纤维取向的跨尺度映射[[S24,S39]]。相比传统串行耦合仿真仅做通用数据格式传递、未针对成型场景开发专用本构模型，该技术实现了统一网格与自由网格两套映射方案，预测精度大幅提升[[S24,S39]]。

该技术重点解决三类工程痛点：1）全周期残余应力高精度计算；2）翘曲变形高精度预测（误差可从50%降至10.3%）；3）考虑成型诱导各向异性的制品尺寸精度与服役性能预测[[S16,S14,S25]]。

## 理论基础与核心模型

### 应力应变本构模型

面向注塑制品动态冷却过程，华中科技大学材料成形与模具技术国家重点实验室相关团队构建了考虑初始应变、热应变和蠕变的应力应变本构模型，适配聚合物的粘弹性应力松弛特性[[S2,S12]]。

对于任意时间步n+1，总应变满足[[S2,S12]]：

\[
\varepsilon^{n+1} = \varepsilon_0^{n} + \varepsilon_h^{n+1} - \varepsilon_p^{n+1}
\]

其中 \(\varepsilon_0^{n}\) 为前一时刻更新得到的残余应变，\(\varepsilon_h^{n+1}\) 为当前步的热应变，\(\varepsilon_p^{n+1}\) 为当前步的蠕变应变。

热应变迭代公式[[S2,S12]]：

\[
\varepsilon_h^{n+1} = \varepsilon_h^{n} + \alpha(T^{n+1})(T^{n+1} - T^{n})
\]

其中 \(\alpha\) 为随温度变化的聚合物热膨胀系数。

蠕变应变迭代规则[[S2,S12]]：

\[
\varepsilon_p^{n+1} =
\begin{cases}
\varepsilon_p^{n} + \dfrac{\sigma^{n} \Delta t}{\mu(T^{n})}, & \sigma^{n} > 0 \\[6pt]
\varepsilon_p^{n}, & \sigma^{n} \leq 0
\end{cases}
\]

其中 \(\mu\) 为随温度变化的粘壶系数。当聚合物受拉应力时产生新蠕变，受压应力时无新蠕变产生。

### 耦合机制

该联合仿真属于**单向耦合**类型，无结构分析结果反向回传修改模流场的求解流程。数据传递逻辑分为两类方案[[S44]]：

| 方案 | 网格关系 | 适用场景 | 数据映射方式 |
|------|----------|----------|------------|
| 统一网格方案 | 模流与结构网格完全一致，单元一一对应 | 单制件结构分析 | 直接映射，无需坐标变换 |
| 自由网格方案 | 两类网格独立，拓扑、数量、尺寸可不同 | 多制件装配体联合仿真 | 通过空间索引配对，经坐标变换后映射 |

## 关键技术

### 统一网格生成技术

统一网格生成有两类主流方案[[S9,S28,S23,S38]]：

**同源共享网格方案**：首先在结构分析软件（如Ansys、Hypermesh）中生成适配结构仿真质量要求的网格，再通过APDL参数化语言将其转换为Moldflow/Z-MOLD可识别的*.fem格式。网格的单元形状与空间位置完全一一对应，适用于单制件2D中面三角形单元场景。Moldflow对于三角形单元不做任何拓扑修改，输出单元信息与输入完全一致[[S9,S23]]。

**异构自由网格映射方案**：适配模流侧仅输出四面体/三角形单元、结构侧可使用四面体/六面体/四边形混合单元的差异，不强制对齐网格拓扑，通过空间立方体索引完成不同仿真域单元的配对映射，支持多制件装配体联合分析[[S38,S28,S21]]。

### 矢量场参数变换与匹配方法

模流结果向结构分析传递时，处理规则为[[S7,S9,S33,S45]]：
- **温度场**为标量场，无需坐标变换，直接进行节点插值传递；
- **压力场、纤维取向张量**为有向矢量场，需先基于方向余弦矩阵完成不同仿真软件单元坐标系之间的矢量分量转换，再采用三角形面积坐标插值算法完成场量从源模流单元到目标结构单元的映射。传递过程保证总能量、总质量守恒，无额外数值耗散[[S7,S9,S45]]。

### 模外多时间步瞬态温度场计算

脱模后制品在环境中自由冷却的计算规则为[[S41,S10,S11,S29]]：以开模瞬间的全域温度分布为初始条件，设置对流换热边界，默认时间步长取0.5s，采用非稳态热传导方程逐时间步迭代求解。每一步校验全域温度，当所有节点相邻两时间步温度相对变化量小于0.01时判定本步收敛，迭代直至制品全域温度降至环境室温。

以聚碳酸酯材质激光雷达罩为例，在对流换热系数5 W/(m²·K)、环境温度23℃的工况下，总时长约270s即可完成全冷却过程模拟[[S41]]。

### 多载荷步变形预测

变形预测按严格顺序分阶段施加载荷[[S12,S16,S37,S41]]：
1. 导入模流求解得到的充填结束残余应力作为初始条件；
2. 引入保压阶段压力诱导的应力场；
3. 叠加模内冷却阶段形成的梯度温度应力；
4. 释放模具约束，模拟脱模瞬间的应力释放；
5. 拆分多个子载荷步，按温度场的逐时间步下降结果依次叠加对应时刻的热应变与蠕变应变，完成后收缩阶段的变形迭代。

该多载荷步计算方法将传统单步线弹性计算的翘曲预测误差从50%降低至10.3%[[S16]]。

## 接口转换器 Interface for UM

Interface for UM（Interface for Unified Mesh）是面向注塑成型领域的自研模流-结构统一网格联合仿真接口转换器，可实现将模流分析得到的矢量场、标量场参数转换后导入结构分析软件，完成注塑制件从开模温度到室温的多时间步瞬态温度场计算以及多载荷步变形与应力分析[[S38,S23,S33]]。

研发主体为华中科技大学材料成形与模具技术国家重点实验室[[S43,S48,S5]]。核心开发成果首次公开于2022年5月的博士学位论文《注塑成型与制品结构行为联合仿真理论和实验验证》[[S37,S33,S1,S16]]。

接口支持模流软件输出的*.fem文件格式，可对接商用模流软件Moldflow（完成"Fill+Pack"分析后导出的*.pat单元信息文件、*.osp单元材料参数文件可直接经接口完成转换）以及国产自研模流软件HSCAE/Z-MOLD[[S28,S23]]。

接口适配的结构分析软件包括[[S26,S41]]：
- **Ansys**：可直接使用模流分析导入的单元材料数据，因两者单元坐标系定义方式相同，无需额外坐标转换；
- **Abaqus**：需通过Interface for UM完成矢量场参数坐标系变换后实现材料参数映射导入。

> **待考证**：当前公开文献未明确证实Interface for UM被集成在正式对外发布的华塑CAE商品化版本中[[S43,S5,S30]]。现有检索范围内未查询到该工具对应的公开软件著作权登记或发明专利信息，目前可证实该工具存在的唯一公开文献为2022年5月发表的工学博士学位论文[[S37,S1]]。

## 典型工作流程

基于统一网格的模流-结构联合仿真标准流程为[[S23,S38]]：
1. 针对分析对象，在Ansys/Hypermesh中划分网格，通过APDL命令流转换为*.fem格式的模流网格；
2. 设置成型参数，在模流软件（Moldflow或Z-MOLD）中开展充填、保压、冷却等分析；
3. 导出模流分析模型及开模时刻的温度、应力、单元刚度参数；
4. 经Interface for UM完成矢量场坐标系变换，将参数作为初始载荷导入结构分析软件（Abaqus/Ansys）；
5. 开展多时间步瞬态温度场计算（从开模温度至室温）；
6. 执行多载荷步变形与应力分析，得到室温自由状态下制品变形和应力分布；
7. 该最终应力状态作为制品服役期结构分析的初始载荷。

## 优势

1. **精度大幅提升**：完整传递注塑成型产生的纤维取向、残余应变、熔接线分布等非均匀场信息至结构分析模型，避免传统分析将材料简化为各向同性均匀体带来的预测偏差[[S31,S24]]。

2. **全流程闭环**：可实现从浇注系统设计、工艺参数优化、成型质量预测到服役力学性能验证的全流程闭环，减少人工重复建模和手动数据转换中的人为误差，缩短产品开发周期[[S17]]。

3. **定量化设计能力**：使纤维增强注塑制品结构设计从定性层面上升到定量层面，对类似汽车尾门等大尺寸高性能产品的开发具有不可替代的指导作用[[S27]]。

## 局限性

1. **统一网格方案适用受限**：仅适用于单制件分析，不支持多零件装配关系。模流与结构两类分析对网格质量要求不同，优化相互干扰可能降低各自计算精度[[S9]]。

2. **材料本构模型标定工作量大**：需提前完成多组试样力学测试以获取完整的非线性特性曲线，前置准备周期长[[S42]]。

3. **适用材料范围有限**：当前成熟方案主要集中在玻纤增强聚丙烯（PP+LGF）、PA66-GF30等常规热塑性注塑复合材料，对未建立完善纤维取向映射模型的新型特殊材料适配性较差[[S13]]。

4. **计算链路复杂，成本高**：需同时部署模流分析软件、中间映射平台（Helius PFA/Digimat/DCtool等）、结构分析软件三类工具，软件采购与人力培训成本远高于单一传统分离仿真流程[[S17]]。
