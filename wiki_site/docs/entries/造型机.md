在压铸专业语境中，行业俗称的“造型机”实指配套制芯设备，以射芯机、壳芯机为代表，功能是为压铸件成型内部复杂腔道制备一次性砂芯，而非通用铸造中直接生产完整浇注砂型的造型设备。[[S4,S43]] 以下基于现行国家标准与铸造工程文献，以制芯设备为中心阐述压铸配套“造型机”的定义、原理、类型、参数及应用。

## 术语界定与归类

国家标准《铸造机械 术语》（GB/T 25370-2020）将传统的“造型机”定义为造型类设备，核心功能是直接制备带型腔的完整砂型（条目编号4.1.1）。[[S49]] 然而，在高压压铸工艺中，型腔由永久金属模具构成，并不采用砂型直接浇注成型，因此现行国标体系中不存在面向压铸浇注的“压铸砂型造型机”正式条目。[[S49,S43]]

压铸领域实际使用的“造型机”在国标中对应的是制芯设备，其中**射芯机**被定义为“利用压缩空气骤然膨胀将芯砂射入芯盒完成砂芯制备的设备”（条目编号4.3.5），**壳芯机**则作为射芯机子类，专门用于制备树脂砂壳芯（条目编号4.3.11），分为顶吹与底吹两种结构型式。[[S61,S14,S26]] 这些制芯设备在国标系统中统一归入“压铸机辅助设备”大类（条目编号7.2.72），既不属于压铸主机，亦不属于压铸模具组件。[[S49]]

压铸行业内“压铸造型机”作为非正式称谓，尚未纳入现行行业标准（本说法标注**待核实**）。[[S49,S43]]

## 在压铸中的工艺作用

压铸配套制芯设备的核心功能是生产一次性砂芯，用于成型压铸件内部的异形空腔、弯曲流道、深腔等传统金属型芯无法脱出的复杂内腔结构。砂芯在铸件完成、开模取件后，通过振动处理或后处理工序即可从成品中移除，无需额外复杂机加工。[[S56]]

设备协作逻辑为：制芯设备完成合格砂芯预制→砂芯被转运并精准定位装入压铸模具的预留芯位→压铸主机执行高压压射，熔融金属注入模具型腔并包裹砂芯完成成型→开模取件后脱除砂芯，获得带有复杂内部孔洞的压铸件。[[S8]]

## 核心工作原理

射芯机以射砂紧实为核心成型方法，利用压缩空气骤然膨胀将芯砂以高速射入芯盒，在极短时间内同时完成填砂与紧实两道工序，依靠砂流动能与压力差双重作用实现砂芯紧实，生产效率极高。[[S58,S50,S29]]

此后根据粘结剂体系的不同，进入硬化阶段，主要分为热芯盒固化和冷芯盒气体固化两大路线：

- **热芯盒固化**：将添加潜伏型酸性固化剂的热固性呋喃树脂砂射入180~250℃的金属芯盒中，表层树脂快速缩聚硬化，形成5~10mm硬壳后即可脱模，砂芯内部未硬化部分依靠芯盒余热完成自硬化，硬化时间通常为30~120s，射砂压力0.3~0.6MPa。[[S21,S47,S53,S29]] 砂芯最大截面厚度不宜超过50mm。[[S21]]

- **冷芯盒气体固化**：射砂完成后向密闭芯盒通入气态催化剂（如三乙胺气雾、SO₂、CO₂等），在室温下快速引发树脂粘结剂交联而硬化，无需对芯盒整体加热。采用CO₂硬化的水玻璃砂，推荐工作压力20~60psi，固化完成后需通入洁净空气吹扫排出残留有害气体并配套废气净化装置。[[S9,S17,S11,S29]]

壳芯机制芯时，先将树脂包覆的芯砂射入（或吹入、重力加入）热的芯盒，砂粒表面树脂熔化后在盒壁附近聚合硬化形成壳层，达到设定结壳厚度后，中间多余芯砂被翻转倒出回用，即可从芯盒取出处于硬化状态的壳芯，取出冷却后可立即用于铸造。[[S8]]

## 主要类型与分类

压铸行业配套的制芯设备（俗称“造型机”）可按硬化方式和芯盒分型方式两个维度分类。

### 按硬化方式分类

| 类别 | 芯盒温度 | 硬化方式 | 主要用途 |
|------|----------|----------|----------|
| 热芯盒制芯机 | 200~250℃ | 砂芯整体受热，表层树脂缩聚硬化，内部余热自硬化 | 大批量中型砂芯，配合呋喃树脂砂 |
| 壳芯机 | 230~290℃ | 覆膜砂仅在芯盒壁附近受热形成硬壳，多余砂可倒出回用 | 树脂砂壳芯，分为顶吹式和底吹式 |
| 温芯盒制芯机 | <175℃ | 砂芯整体受热硬化（可与热芯盒制芯机通用） | 能耗低于热芯盒法，待核实详细工艺参数 |
| 冷芯盒制芯机 | 室温 | 通入气态催化剂（三乙胺、SO₂等）或吹CO₂，发生化学交联硬化 | 适配自动化制芯线大批量生产 |
| 多功能制芯机 | 可调 | 可用两种或两种以上不同硬化方式制芯 | 柔性生产，适应多品种砂芯需求 |

表据[[S66]]整理，温芯盒专项参数目前证据不完备，标注待核实。

### 按芯盒分型方式分类

- **水平分盒制芯机**：开合芯盒采用升降运动完成取芯与清模，适配中低高度芯盒。[[S66]]
- **垂直分盒制芯机**：适配较高芯盒，一般将一侧芯盒翻转90°后顶出取芯，多为上顶式。[[S66]]
- **水平加垂直复合分盒制芯机**：可实现部分侧抽活块动作，结构比纯水平/垂直分型复杂。[[S66]]
- **多面分盒制芯机**：支持多方向开模，适用于结构极为复杂的异形砂芯生产。[[S66]]

## 典型结构组成

典型热芯盒射芯机核心部件包括：[[S12,S54]]

- **砂斗**——存储并定量供给芯砂；
- **射砂筒**——为射砂过程提供稳压储气空间，配合环形进气阀实现均匀射砂；
- **工作台**——用于夹紧固定芯盒，提供射砂时的密封支撑；
- **模板**——安装固定模样的刚性基体，保证砂芯成形轮廓精度；
- **气动/液压系统**——为整机各动作提供动力并控制各工位运动逻辑。

冷芯盒射芯机在上述结构基础上，取消了芯盒加热部分，增设吹气机构及前后工序配套设备（如气体发生器、三乙胺雾化器、余气净化装置等）。[[S12]]

## 砂芯起模要点

砂芯顶出困难是常发问题，其诱因主要包括：

- 砂芯与芯盒接触部位附着力过大，尤其射孔区域附着力占总附着力2/3以上；
- 顶芯杆与射砂孔位置不对中，顶芯杆刚性不足易弯曲偏斜；
- 芯盒拖动缸缓冲失效引发撞击偏移。

优化措施：射孔锥度设置为1.5°~2°，射孔有效长度控制在40mm 以内；顶芯杆直径通常比射砂孔小约2mm，每班检查顶芯杆状态，每半年更换一次拖动缸密封件。[[S48]]

## 在压铸中的具体应用

### 不同压铸工艺对砂芯的要求

不同压铸工艺对配套砂芯的强度、耐冲刷性和溃散性要求差异显著：

- **高压压铸**：充型速度可达5~100m/s，普通砂芯受高速金属液冲刷和反复压力波动影响，极易出现断裂、错位和严重粘砂，需适配抗弯强度≥21MPa的高抗冲专用覆膜砂芯、可溶性盐芯或陶瓷芯。[[S62,S34,S57]]

- **挤压铸造**：液流速度低于高压压铸，采用专用砂芯（如HDSC01型）并配合涂料（如HDCoat03），在比压小于83MPa的分段阶梯加压条件下可完成带砂芯成型，所得铸件无粘砂、夹砂和气孔缺陷，组织细密且可热处理强化。[[S57]]

- **低压压铸**：充型压力通常低于0.5MPa，金属液流动平稳，常规覆膜砂芯即可适配。需针对砂芯热膨胀特性在冷态和350~450℃工况下进行配合间隙研配，小型件0.1~0.3mm、大型复杂件0.3~0.5mm，以避免砂芯挤压开裂或跑铝缺陷。[[S38,S27]]

- **匀加速料筒孕育半固态压铸**：采用普通覆膜砂芯即可生产可高温热处理、可氩弧焊的复杂空腔铸件，可部分替代工艺复杂、成本高的盐芯方案。[[S62]]

### 典型件例

- **柴油发动机汽缸体水冷套砂芯**——RYOBI公司采用专用压铸砂芯工艺制备，支撑复杂冷却流道的一体成型加工。[[S15]]

- **硅青铜D型泵叶轮**——采用真空压铸配套砂芯成型工艺，模具含独立砂芯组件与上/下模部件，实现泵体内复杂流道的一体成型。[[S39]]

- **铝合金发动机缸盖**——采用组合砂芯方案，包含进气管砂芯、排气管砂芯、上/下水套砂芯等多个独立组件，同时成型缸盖内部水道、油道、气道等多组复杂腔道。[[S44,S42]]

充型过程中，砂芯与金属液直接接触区域温度可达300~500℃，对应铸件孤立液相区，是缩孔缺陷高危位置，生产时需针对性调整砂芯排气结构。[[S46]]

## 相关设备与生产线

压铸配套制芯设备（俗称“造型机”）在生产线上的上下游协作逻辑为：上游衔接混砂机，完成原砂、粘结剂与固化剂的精准混配；核心射芯工位将芯砂射入芯盒完成成型；下游依次对接固化炉完成砂芯硬化、取芯机械手完成砂芯脱模转运、砂芯修整浸涂工位完成表面涂层处理，最终输送至压铸工位完成下芯。[[S2,S42]]

自动化制芯线通过自动控制系统覆盖砂料供给、射砂紧实、砂芯硬化、取芯转运、后处理全流程作业，其生产能力需配置为匹配压铸线单循环需求量的1.25~1.4 倍，并应配备自动故障监测与自动集中润滑系统。[[S20,S14]]

## 常见缺陷与维护要点

### 射砂成型不良

射砂不足导致砂芯短射（局部欠注）的常见诱因及对应排除方法：[[S45]]

| 故障现象 | 产生原因 | 排除方法 |
|----------|----------|----------|
| 砂芯局部成型缺陷 | 射孔堵塞 | 检查并疏通射孔 |
| | 芯盒排气不畅通（新盒设计问题/旧盒排气塞堵塞） | 修改新芯盒或疏通旧盒排气塞 |
| | 射头射嘴与芯盒未对准 | 调整射嘴对准位置 |
| | 射砂筒砂量不足 | 补充型砂 |
| | 射砂时间参数不当 | 调整射砂时间 |
| | 射砂压力不在合理区间 | 校准射砂压力 |
| 射砂过程喷砂 | 芯盒夹紧机构未锁止 | 检查举升/夹紧机构 |
| | 射嘴与芯盒未对准 | 调整射嘴对准位置 |
| | 密封圈损坏 | 更换密封件 |
| | 射砂筒砂量不足 | 补充型砂 |
| 不射砂 | 射砂控制阀卡滞/闸板未关闭无电气信号 | 检修控制阀及信号回路，关闭闸板 |
| | 储气室进气阀未打开 | 打开进气阀 |

### 压缩空气品质引发的系统故障

射芯机全部故障中约20%~25% 由含水量过高的压缩空气引起，水分导致管路、气阀、气缸内部锈蚀磨损，还会在射砂过程中进入芯砂，直接影响砂芯成品质量。[[S55]]

核心运维要求：
- 在制芯车间配置1~2级水分分离器，每日定期排放积水、油和杂质；
- 冬季当车间温度低于3℃时，在排气阀处加装电加热装置，确保阀体温度保持在0℃以上，避免排气降温导致结冰卡阀。[[S55]]

## 标准与规范

- **GB/T 25370-2020《铸造机械 术语》**——2020年11月19日发布，2021年6月1日实施，明确界定造型机、制芯机（含射芯机、壳芯机等）全品类铸造设备的术语定义，是铸造机械领域的通用基础标准。[[S4]]

- **EN 710**——欧盟针对砂型铸造造型和制芯机械及附属工厂设备的核心安全标准，需与电气安全标准 EN 60204-1、功能安全标准 EN ISO 13849-1/2 配合使用。[[S7]]
