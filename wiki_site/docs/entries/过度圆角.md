## 概述

在压铸生产中，“过度圆角”指铸件壁与壁连接处的圆角半径超出对应壁厚及合金种类所对应的合理推荐区间，引起局部截面异常增厚的结构或尺寸偏差现象[[S27,S18]]。该状态可源于设计阶段取值失当（几何设计缺陷），也可源于模具长期磨损或加工制造偏差导致型腔实际圆角尺寸超出设计阈值（模具状态缺陷）[[S11,S25]]。过度圆角会破坏铸件截面均匀性，在圆角对应部位形成局部金属富集，成为诱发缩松、缩孔、热裂等收缩类缺陷的关键起始条件[[S27,S33,S20]]。

## 定义与分类

在压铸语境下，“过度圆角”有明确的形态与成因定义：

- **形态定义**：壁与壁相交处的圆角半径超越遵循该壁厚、该合金的推荐取值范围，使实际截面厚度异常升高，不符合合理的铸造圆角设计准则[[S27,S18]]。  
- **成因分类**：
  - **几何设计缺陷**：设计阶段即赋予超出推荐区间的圆角尺寸，属于几何参数缺陷[[S11,S25]]。
  - **模具状态缺陷**：模具经长期服役磨损或制造偏差而导致型腔实际圆角尺寸偏大，超出原设计允许上限[[S11]]。

过度圆角并不属于单一独立的缺陷大类。在国际AFS铸造缺陷分类体系中，由尺寸偏差本身属于F类（错误的尺寸或形状），而其引发的缩松、缩孔缺陷归属于B类（孔洞）下的收缩缺陷子类，其中与fillet相关的条目代码为A 115[[S11,S29]]。在我国GB/T 5611体系中，过度圆角被视作诱发缩孔、缩松的前置设计或模具偏差诱因，未单独列为独立缺陷大类[[S11]]。

## 缺陷形成机制

过度圆角使壁连接处体积远大于相邻薄壁，形成**几何热节**，其体积与表面积之比较大，冷却凝固显著滞后于周围区域[[S26,S35]]。随后引发一连串失效过程：

1. **补缩通道阻断**：热节周边的薄壁区域率先完成凝固，封闭了通向热节的外部液相补缩路径，热节转变为孤立液相区[[S8]]。  
2. **收缩缺陷生成**：在孤立液相区内，液态收缩与凝固收缩总量无法得到外界液态金属补充。当枝晶搭桥后，残余液相不能实现枝晶间补缩，最终在热节中心形成不规则的连通缩松或集中缩孔[[S28,S14,S34]]。  
3. **热裂产生**：热节部位晶界处富集低熔点相，在脆性温度区间凝固收缩应力超过合金塑性极限时，沿晶界萌生、扩展形成热裂纹，断口呈氧化、无金属光泽特征[[S34,S2]]。  
4. **残余应力与变形**：不同冷却步序区域间的温度梯度导致残余应力集中，既在圆角周边形成高应力点，又增大了铸件整体变形与翘曲倾向[[S26]]。

## 关联缺陷与判断依据

过度圆角可同时诱发多种缺陷，其典型清单及判断特征如下[[S27,S21,S26]]：

| 缺陷类型 | 典型判断依据 |
|----------|--------------|
| 集中缩孔 | 位于两壁或多壁相交的圆角对应内部热节位置，孔洞形状不规则、孔壁粗糙并带有枝状晶，与气孔的规则球形相区别[[S28,S34]] |
| 分散缩松 | 分布在热节中心或周边，呈细小、连通的孔洞群[[S28]] |
| 晶间热裂纹 | 断口严重氧化无金属光泽，沿晶界曲折延伸，常与缩松并存[[S34,S2]] |
| 表面凹陷（缩陷） | 铸件外表面对应圆角热节处出现凹陷，多产生于厚薄差明显位置[[S21]] |
| 局部残余应力超标 | 圆角周围区域应力集中，易导致变形或提前疲劳失效[[S26,S28]] |
| 浇不足（局部） | 因充型时流态紊乱、裹气或局部提前凝固导致轮廓不清[[S21]] |
| 裹气 | 充型过程中气体积聚在热节附近，形成气孔[[S21]] |

圆角异常所致缺陷具有固定的空间分布特征：缺陷群严格匹配热节的几何轮廓，集中在圆角对应内部区域，非分散随机分布；且常呈缩松、热裂、应力集中等组合出现，与由工艺参数波动引起的全域随机分布缺陷明显有别[[S26,S4]]。

## 设计准则与参数范围

为防止过度圆角，压铸件铸造圆角设计应遵循壁厚关联的推荐取值公式。当圆角半径**超出**以下推荐值范围时，即属于过度圆角范畴[[S16,S12,S27,S18,S33]]。

### 等壁厚连接

| 合金类别 | 内圆角半径 r 计算式 | 备注 |
|----------|---------------------|------|
| 锌合金 | r = 1/4·b（b为壁厚） | [[S16]] |
| 铝合金、镁合金、铜合金 | r = 1/2·b | [[S16]] |

另有手册推荐内圆角允许范围 r₁ = (1～1.25)·b 或 r₁ = 0.5s 至 s 之间[[S12,S18,S33]]。

### 不等壁厚连接

不等壁厚直角连接内圆角通用计算式为：

r ≥ (b₁ + b₂)/3  [[S16]]  
或 r = 2/3·(b₁ + b₂)  [[S12]]  
或简化式 R = (A + B)/3  [[S27,S18]]

上述推荐主要针对铝合金和镁合金[[S18,S33]]。

### 最小圆角基础阈值

不同合金的最小圆角半径不得低于以下数值，否则会因圆角过小引发裂纹等缺陷[[S30,S22]]：

| 合金 | 最小圆角半径 |
|------|--------------|
| 通用压铸件 | ≥0.5 mm |
| 锌合金 | ≥0.5 mm |
| 铝合金 | ≥1.0 mm |
| 镁合金 | ≥1.5 mm |
| 铜合金 | ≥1 mm（且推荐不超过最薄壁厚的1/2） |

## 标准与规范

在现行公开标准中，过度圆角本身未被设立为独立验收限值条款，其合规性通常通过**合金‑壁厚匹配的圆角推荐区间**校核[[S19]]。  
- ASTM B系列压铸标准关于“Fillets, Ribs, and Corners”章节明确规定：压铸件相交面采用圆角过渡时应避免设计过大的圆角导致局部壁厚超出允许范围，深腔内部角落可选用合理的大圆角，但不可无限制放大圆角半径[[S15]]。  
- 在ISO 8062、GB/T 5611、GB/T 15114、NADCA公开文档中，未发现针对过度圆角的单独专属限值条款[[S19]]。  

尽管如此，圆角尺寸公差规则仍给出了偏差方向：凹圆角（内圆角）尺寸偏差取负值，凸圆角（外圆角）尺寸偏差取正值，普遍按压铸10级精度控制[[S7,S1,S6]]。

## 限制与后果

过度圆角对铸件质量、后续加工及模具寿命造成多重负面影响：

- **力学性能劣化**：内部缩松、微裂纹减少有效承载面积，降低拉伸强度、屈服强度，并形成高应力集中点，显著缩短疲劳寿命[[S28]]。  
- **气密性失效**：连通的缩松与微裂纹形成渗漏通道，导致铸件在保压工况下出现漏气、渗水、渗油[[S34,S32]]。  
- **机加工报废风险**：过度圆角额外增加的局部金属厚度会改变预设的加工余量分布，加工后易将表层致密层去除，暴露内部隐藏的缩松缺陷，形成针孔或孔洞直接报废[[S10]]。  
- **模具寿命缩减**：模具型腔上的大圆角棱边在交变冷热载荷下易成为热疲劳裂纹萌生点，加速模具表面龟裂，降低模具整体使用寿命[[S17,S15]]。

## 相关术语关系

- **铸造圆角**：压铸件壁连接处按规范建议设计的圆弧过渡。过度圆角是其异常形态，取值超出推荐计算区间，与正常铸造圆角的设计要求完全对立[[S33]]。  
- **内圆角/外圆角**：过度圆角涵盖所有内圆角、外圆角超出设计规范允许上限的异常情况[[S33]]。  
- **热节**：过度圆角产生的局部体积富集形成几何热节，是引发后续缺陷的核心物理条件[[S26,S35]]。  
- **应力集中**：合理圆角可降低拐角处应力集中，但过度圆角不仅不降反增应力，该区域因热节引起的变形和残余应力会进一步放大应力集中[[S15,S31,S35]]。
