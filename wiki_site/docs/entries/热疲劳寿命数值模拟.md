## 概述

在压铸过程中，模具型腔表面频繁接触高温金属液并受到脱模剂冷却，承受剧烈的周期性冷热交变载荷，由此产生交变热应力，最终导致热疲劳失效。铝合金压铸模具热疲劳失效占比约70%~80%，铜合金压铸模具则可达80%~90%[[S19,S22]]。

热疲劳寿命数值模拟是面向压铸模具服役场景，通过热-力耦合有限元分析，结合疲劳损伤累积理论，量化预测模具在冷热交变循环下失效总模次的计算技术。行业通用标准流程为：采用ProCAST软件计算压铸周期热平衡状态下的温度场，将温度场结果导入Abaqus完成热应力场求解，最终将热应力分析结果导入FE-Safe类专业疲劳软件计算热疲劳寿命[[S4,S34]]。与纯实验研究方法相比，数值模拟具有计算成本低、操作便捷、周期短的优势，其输出的最低寿命区域与实际生产中模具热疲劳裂纹萌生位置高度吻合[[S19,S8]]。

## 工业应用场景

压铸热疲劳寿命数值模拟在工程中的典型应用场景涵盖四大类[[S4,S29,S6,S32,S17]]：

1. **冷却系统优化设计**：通过仿真全模温度分布与梯度，调整冷却水路布局，降低型腔局部温度波动。
2. **浇注工艺参数匹配**：以热疲劳寿命为优化目标开展正交模拟，筛选最优的模具预热温度、合金浇注温度、脱模剂换热系数组合。
3. **模具提前开裂预警**：提前识别应力集中高风险区域，预判热疲劳裂纹萌生位置。
4. **全模具全域寿命评估**：得到不同区域的理论模次寿命分布，支撑模具材料选型与表面强化方案验证。

## 分类与常用方法

### 基于经验的应变-寿命模型（Manson-Coffin模型）

Manson-Coffin（Coffin-Manson）模型是压铸热疲劳领域应用最广泛的低周疲劳预测模型。其基础公式将塑性应变幅与疲劳断裂循环次数直接关联，可通过四点关联法估算疲劳性能指数完成工程化计算[[S24,S31,S41]]：

$$\frac{\Delta \varepsilon_p}{2} = \frac{\sigma_f'}{2}(2N)^b + \varepsilon_f'(2N)^c$$

该方法优势是形式简单、参数易获取，适合等温低周疲劳场景[[S10]]。但模型采用固定温度下的材料参数，直接用于温度动态变化的压铸热疲劳场景时预测误差显著增大，对高温蠕变、氧化耦合损伤的表征能力不足。例如，采用经典Manson-Coffin模型预测铝合金导轮压铸模寿命为184747模次，而实际该模具在41808模次时出现开裂[[S24,S31]]。

### 能量法

能量法以材料循环滞回过程中耗散的非弹性应变能作为损伤评价核心指标，属于典型的唯象模型。代表性模型包括Smith-Watson-Topper (SWT)模型、Ostergren拉伸迟滞能模型、三参数幂函数能量模型等[[S24]]。2016年发表在《International Journal of Fatigue》上的研究结果证实，热疲劳裂纹萌生与材料点的非弹性耗散能量直接相关[[S30]]。该类方法可规避单一固定温度参数表征动态热疲劳过程的不足，预测精度优于传统应变-寿命模型[[S24,S7]]。

### 基于断裂力学的裂纹扩展模拟

该方法通过裂纹尖端的应力应变场量化裂纹扩展速率，通过积分建立裂纹长度与总寿命的对应关系，适合预测热疲劳裂纹萌生之后的扩展阶段寿命。常结合扩展有限元技术、微观三维多晶模型实现裂纹扩展路径的可视化预测。针对H13钢、4CrMo4钢的热疲劳裂纹扩展数值模拟结果，与试验测得的不同循环次数下主裂纹长度吻合度较高[[S4,S22,S34]]。

### 基于连续损伤力学的寿命预测模型

连续损伤力学方法可耦合疲劳-蠕变-氧化多损伤机制，实现全损伤演化过程的定量计算。针对H13压铸模具钢开发的热机械疲劳归一化预测模型，其预测寿命与试验寿命的分散带可控制在1.5倍线以内，并可迁移至4Cr5Mo2V等其他压铸模具钢材料使用。该类方法严谨性与场景适用性更强，但计算复杂度高，需要大量工程简化处理[[S26,S7,S27]]。

### 多轴疲劳准则

压铸模具型腔尖角、浇口附近等应力集中区域存在多向复杂交变载荷。常用的多轴疲劳准则包括Sines准则、Fatemi-Socie临界面准则、Ellyin-Golos多轴损伤准则等，可修正单轴疲劳模型在多向应力状态下的预测偏差，适配压铸模具非对称非比例加载的复杂热机械疲劳工况[[S13,S38]]。

### 方法对比

| 方法类别 | 优势 | 劣势 |
|---------|------|------|
| Manson-Coffin应变-寿命模型 | 形式简单，参数易获取 | 固定温度参数，对动态热疲劳误差大 |
| 能量法（SWT、Ostergren等） | 规避单一温度参数不足，精度较高 | 模型参数标定需要准确的非弹性应变能数据 |
| 断裂力学裂纹扩展模拟 | 适合扩展阶段预测，可实现可视化 | 对萌生阶段不适用 |
| 连续损伤力学模型 | 耦合多机制，严谨性强 | 计算复杂度高，需大量简化 |

## 核心输入参数

### 模具材料属性

压铸模具基准材料普遍为H13钢（4Cr5MoSiV1）[[S16]]。H13钢的主要热物性参数[[S37,S18,S43]]：

| 参数 | 典型值/特性 |
|------|------------|
| 密度（室温） | 7.78 g/cm³ |
| 弹性模量（0℃） | 约220 GPa |
| 弹性模量（800℃） | 约120 GPa |
| 热导率变化 | 随温度升高逐步上升，1200℃后稳定在约34 W/(m·K) |
| 泊松比 | 0.3 |

20~700℃区间的热膨胀系数、比热、热扩散系数、热导率、杨氏模量等参数，可通过激光闪射法、差示扫描量热法、热膨胀仪等标准测试方法获得[[S43]]。

### 温度循环与工艺参数

铝合金高压压铸的典型工况参数范围[[S19,S32,S9]]：

| 参数 | 典型范围 | 优化推荐值 |
|------|---------|-----------|
| 金属液浇注温度 | 670~710℃ | 650℃（正交优化） |
| 充型压力 | 50~80 MPa | — |
| 充型速度 | 30~100 m/s | — |
| 模具预热温度 | 200~250℃ | 220℃ |
| 脱模剂换热系数 | — | 800 W/(m²·K) |
| 冷却水换热系数 | — | 5000 W/(m²·K) |

开模后经脱模剂冷却，模具表面温度骤降，开模温差普遍处于200~400℃区间[[S19]]。

## 典型模拟流程

压铸模具热疲劳寿命数值模拟的标准操作流程可划分为六个核心步骤[[S34,S21,S36]]：

1. **三维几何建模**：使用Creo、SolidWorks等三维建模软件建立与实际尺寸完全匹配的压铸模具、压铸件三维几何模型。
2. **网格划分**：将几何模型导入HyperMesh等网格划分工具，完成面网格与体网格的剖分。
3. **边界条件定义**：依次定义热边界条件（不同阶段对应换热系数、初始温度设置）和机械约束条件（通常将模具安装固定位置设为全约束）。
4. **温度场求解**：求解多压铸循环下的温度场，直至模具进入热平衡状态（约需15次连续循环）[[S33]]。
5. **热应力场求解**：将热平衡下的瞬态温度场作为热载荷导入结构场求解器，完成单个循环周期内的热应力、热应变场求解。当前主流采用热弹塑性应力力学模型，假设材料在屈服前为线弹性，屈服后转为塑性，弹性模量和屈服应力随温度变化动态调整，屈服服从Mises准则，模拟精度与稳定性表现优异[[S24,S31]]。
6. **疲劳寿命计算**：将应力应变结果导入疲劳分析模块，结合修正后的材料S-N曲线、疲劳累积损伤模型完成全寿命计算。

![压铸模具热疲劳数值模拟的典型二维有限元基准模型](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f9807ae3-cdfb-4ccd-985d-27fa3833986d/resource)

已有公开工程应用案例采用ProCAST计算温度场，导入ABAQUS求解瞬态热应力，最终在FE-Safe中完成热疲劳寿命预测。模拟得到的最低寿命区域与实际压铸模热疲劳裂纹萌生位置完全吻合，预测寿命与实际生产中20000~30000次循环的实测结果误差小于4.2%[[S8,S19]]。

## 局限性与不确定性

压铸热疲劳寿命数值模拟当前存在以下公认的局限性与不确定性[[S24,S26,S23,S42]]：

1. **模型简化假设**：普遍采用材料各向同性假设和热弹塑性本构模型，未完全计入高温氧化、铝液冲蚀腐蚀、蠕变耦合作用等实际服役过程中的失效加速因素，导致模拟得到的理论寿命普遍高于模具实际生产服役寿命。

2. **参数敏感度**：模拟输入参数如界面换热系数、材料高温力学性能的敏感度较高，微小的参数偏差可能导致最终寿命结果出现数量级差异。

3. **等温与热机械疲劳的差异**：采用固定温度下材料参数的等温疲劳模拟精度远低于考虑全周期温度动态变化的热机械疲劳模拟。以铝合金导轮压铸模为例，Coffin-Manson等温疲劳模型预测寿命为184747模次，而考虑温度波动的修正模型预测结果为23954模次，与实际41808模次出现裂纹的服役表现更为接近[[S24,S31]]。

## 相关标准与试验体系

压铸热疲劳相关的行业标准体系包含[[S15,S12,S40]]：

- **NADCA**：发布压铸行业相关研究报告、会议技术论文，定义压铸专用热疲劳试样推荐测试流程，给出不同H13类压铸模具钢的热疲劳寿命基准参考范围。
- **ISO 12107**：金属材料疲劳试验通用标准体系，为热疲劳试样加载、数据采集、寿命统计提供统一方法框架。
- **GB/T 15824-1995**：中国国家标准《热作模具钢热疲劳试验方法》。
- **JIS Z 2278**：日本金属热疲劳测试标准。
- **ASTM系列**：热疲劳试验STP标准，共同构成试样级热疲劳性能标准测试体系。

标准试样级热疲劳试验与全域数值模拟方法具备互补关系：标准试验可获精准的材料基础热疲劳性能参数为模拟提供输入边界，数值模拟则可突破标准试样尺寸限制将性能数据扩展到复杂三维全模具场景实现全域寿命预测，二者结合可大幅降低传统压铸模具设计过程中的试模成本[[S20]]。
