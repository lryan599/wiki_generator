## 消歧说明

在工程技术领域，“模具”是一个广义术语，泛指通过改变成形材料物理状态来获得特定形状制品的工具[[S89]]。由于不同成形工艺原理差异显著，对应的模具结构、材料与设计准则无法合并为统一描述。本条目以**压铸模具**（Die Casting Mold / Die）为核心进行阐述，以区别于轮毂铸造模具、复合材料热压罐成型模具、金属型重力铸造模具等其他类别的模具。

## 概述

压铸模具是压铸生产的核心工艺装备，属于压铸三大基本要素（压铸合金、压铸模具、压铸机）之一[[S37,S82]]。它直接决定了压铸件的形状、尺寸精度和表面质量，实际生产中大部分工艺问题和质量缺陷均与模具直接相关[[S37,S31]]。

压铸模具工作环境极为苛刻：需承受高温（300~1000 °C）金属熔体的高压（150~500 MPa）、高速（内浇口速度40~60 m/s，甚至更高）冲刷，并反复经历急冷急热的热循环。热疲劳失效是影响其使用寿命的核心因素[[S37,S82,S69]]。

压铸模具与其他铸造类模具存在核心差异：压铸模具适配10~50 MPa高压充型工艺，型腔内部开设专用溢流排气结构以应对高速充型工况；常规金属型（重力铸造）模具依靠重力充型，无高压充型对应的结构设计；低压铸造模具依靠低压升液充型，整体结构不匹配压铸机的高锁模力安装接口[[S4,S6]]。

## 结构与组成

压铸模具整体分为**定模**（固定在压铸机定模安装板上，与压射机构压室连接）和**动模**（安装在动模安装板上，随机器移动实现合模/开模）两大主体部分[[S39,S31,S74]]。二者合模后共同构成完整型腔与浇注系统，开模时动模侧的推出机构将成形后的压铸件脱出。

根据功能可将压铸模具划分为以下核心组件[[S44,S51,S112]]：

| 功能模块 | 主要构成零件 | 功能说明 |
|---------|-------------|---------|
| **成形零件** | 型腔、型芯、镶块 | 构成铸件几何形状的核心部分。型腔形成外表面，型芯形成内部结构（孔、槽等） |
| **模架（模框）** | 套板、座板、支承板、垫块 | 集成固定所有模具功能零件，保证模具可稳定安装在压铸机上，需满足足够刚性与强度[[S39]] |
| **浇注系统** | 直浇道、横浇道、内浇口 | 连接压铸机压室与模具型腔的金属液流通通道[[S39,S31]] |
| **排溢系统** | 排气槽、溢流槽 | 排出型腔内残留气体，储存最先进入型腔的冷金属和氧化夹杂[[S31]] |
| **推出机构** | 推杆、推板、推杆固定板、复位杆 | 克服铸件冷却收缩后的包紧力将铸件脱出，推出点需均匀分布避免局部变形[[S39,S31,S73]] |
| **温控系统** | 冷却水道、加热油道 | 调节模具温度维持热平衡，中小型模配置冷却水路，复杂模具可同时配置冷却和加热通道[[S28,S104]] |
| **导向定位组件** | 导柱、导套 | 引导动定模沿指定方向精准运动，保证合模精度[[S39,S10,S104]] |
| **抽芯机构** | 斜销、滑块、楔紧块等 | 活动型芯的插拔机构，用于成形与开合模方向不一致的侧凹特征[[S31,S39]] |

不同压铸机型适配的模具中，浇注系统结构随压铸机类型差异调整，其余核心结构逻辑通用[[S51]]。

### 成形零件结构形式

型腔和型芯分两类结构[[S44,S50]]：

- **整体式结构**：直接在模板上加工成形，结构紧凑刚性好不易变形，适用于高熔点合金压铸，但复杂结构加工难度高。
- **镶拼式结构**：由多块镶块拼接构成，加工难度低、便于更换易损件，适配复杂成形需求，在压铸模具中广泛应用。但装配较复杂，易产生飞边[[S20]]。

### 浇注系统设计要求

浇注系统由直浇道（适配压室出口）、横浇道（将金属液分配至各型腔入口）、内浇口（控制进入型腔的速度与方向）组成。关键设计准则[[S87]]：

- 内浇口高度不超过铸件对应位置壁厚的75%
- 内浇口宽高比大于10
- 横浇道截面积沿流动方向平缓缩小，保证流速提升避免紊流

### 排溢系统设计准则

溢流槽用于储存最先进入型腔的冷金属和氧化夹杂，同时帮助排出气体。排气槽设于溢流槽后方。设计准则[[S31,S72,S83]]：

- 溢流槽总入水口截面积约为所有内浇口总截面积的1/2
- 排气槽总截面积为内浇口总面积的20%
- 排气槽最大深度可达1 mm，避免金属进入后快速凝固堵塞

### 按模腔数量

按压铸模腔数量可分为[[S88]]：

- **单腔压铸模**：大、中型压铸件通常采用
- **同形多腔压铸模**：小型压铸件常采用
- **异形多腔压铸模**：应用较少，容易出现型腔之间充型不平衡的问题

- **平面分型压铸模**：加工简单，操作及维护方便
- **阶梯分型压铸模**
- **倾斜分型压铸模**

## 模具材料

### 成形零件材料性能要求

与金属液直接接触的成形零件（型腔、型芯、镶块）需满足以下性能要求[[S8]]：较高的热强性和回火稳定性；优良的耐热疲劳性能和高的导热性；良好的抗氧化性和耐蚀性；热处理变形小、淬透性高；良好的可锻性和切削加工性。

### 常用热作模具钢牌号

**H13钢（4Cr5MoSiV1）** 是国际上广泛应用的空冷硬化热作模具钢，对应国标GB/T1299-2000。化学成分：C 0.32%~0.45%，Cr 4.75%~5.50%，Mo 1.10%~1.75%，V 0.80%~1.20%，硫磷含量低于0.030%，密度7.8 g/cm³。热处理后硬度可达HRC 47~49，在型腔温升低于600 °C的工况下可用作压铸模材料。对应美国AISI H13、日本SKD61、德国1.2344[[S17,S3,S49]]。

| 牌号 | 产地/厂商 | 主要特性 |
|-----|----------|---------|
| H13 (4Cr5MoSiV1) | 中国/美国等 | 较高的韧性和耐冷热疲劳性能，强韧兼备[[S3]] |
| 8407 | 瑞典一胜百(Uddeholm) | 纯度高、组织微细，等向性优于普通H13，硬度可比普通H13高1~2倍而不牺牲韧性[[S60,S53]] |
| Dievar | 瑞典Uddeholm | 专为高温高压压铸设计，抗热疲劳性能约为普通H13的2倍，硬度覆盖40~52 HRC[[S79]] |
| DAC (SKD61) | 日本日立 | 高温强度良好、韧性佳、耐磨性较好，适用于铝/镁/锌/铜合金压铸[[S53]] |
| 1.2344 | 德国 | 材质均匀，切削加工性和抛光性优越，等向性优于传统钢材，高温强度及耐热疲劳性良好[[S58,S95]] |
| 3Cr2W8V | 中国 | 传统压铸模用钢，GB/T8844-2003规定使用于型腔型芯，锌合金压铸硬度40~50 HRC，铝镁合金48~52 HRC[[S110]] |

材料选型应优先选用高纯净度热作模具钢，将硫元素含量控制在0.003%以下可使模具寿命提升1.3倍；电渣重熔冶炼的钢组织更均匀，可避免提前龟裂失效[[S46,S8]]。

### 表面处理技术

- **气体氮化**：操作温度约525 °C，以氨气为渗剂，获得0.3 mm深度硬化层需数日，表层硬度可达950~1200 HV。适用于顶杆、冷室机压射套筒的硬化处理，处理后压射套筒表面硬度可达70 HRC，处理变形小[[S84,S85]]。
- **离子氮化**：在400~510 °C低温下通过等离子体作用渗氮，处理后工件变形量极小，且氮化层可在补焊修复时无需完全去除直接局部处理，特别适用于大型压铸模批量处理[[S2]]。离子氮化试样的抗铝液熔蚀深度比常规热处理试样低2.8倍[[S59]]。
- **PVD涂层**（TiN、TiC等）：处理温度204~482 °C，远低于回火温度，工件几乎无变形。涂层硬度极高、减摩性优异，可提升抗粘模、抗磨损性能。但属于视线沉积工艺，绕镀性差，涂层厚度仅2~4 μm[[S2,S15]]。

大型型腔整体采用表面强化处理的应用占比很低，复杂型腔表面的交变热应力容易破坏薄强化层导致提前失效，表面处理更多用于顶杆、小型芯、压射套筒等小型零件[[S81]]。

## 设计要点

### 分型面设计

分型面分为平直分型面、倾斜分型面、阶梯分型面、曲面分型面四类[[S42,S107,S14]]。设计优先遵循以下规则：

1. 分型面设置在铸件最大轮廓投影位置，保证顺利脱出
2. 优先选用平直分型面，降低加工难度
3. 分型面布置需方便浇注系统与排溢系统排布
4. 保证开模后铸件留在动模侧
5. 高精度配合面尽量布置在同一半模内，避免错模误差
6. 尽量以压铸件机械加工面作为分型面

### 脱模斜度

脱模斜度需根据合金类型和脱模高度选取对应数值，型腔侧和型芯侧分别设置对应斜度，保证铸件顺利脱出，避免拉花、粘模缺陷[[S56,S1]]。

### 锁模力校核

压铸过程中金属液高压填充型腔会产生胀型力。常规计算公式为[[S40,S21,S29]]：

$$F_{锁} \geq K(F_{主} + F_{分})$$

其中安全系数 $K$ 取1.25，$F_{主}$ 为主胀型力（由铸件、浇注系统、排溢系统在分型面的总投影面积乘以压射比压计算），$F_{分}$ 为侧抽芯产生的附加胀型力。锁模力不足会导致模具被胀开，铸件出现飞边、尺寸超差等缺陷。

### 热平衡与温控

热平衡定义为压铸生产循环中，模具从高温金属液中吸收的热量与向外界环境散出的热量达到动态平衡的状态[[S75]]。需匹配对应温控系统保证不同区域温差控制在合理范围，避免局部过热或过冷导致冷隔、缩松缺陷。

不同合金压铸的模具工作温度适配范围[[S106,S11,S111]]：

| 合金类型 | 模具工作温度范围 (°C) |
|---------|-------------------|
| 锌合金 | 150~200 |
| 铝合金 | 200~300 |
| 镁合金 | 220~300 |
| 铜合金 | 300~380 |

模具温度过低会导致金属液充型流动性不足，出现冷隔、浇不足缺陷；过高则冷却速度过慢，易产生缩松、粘模，同时降低模具寿命[[S106,S11,S111]]。

## 工艺参数关联

### 模具温度对充型与凝固的影响

压铸充型完成后，模具温度直接影响铸件的冷却凝固速率：温度越高凝固越慢，晶粒更细小但收缩率升高；温度越低凝固越快收缩率低。不合理的模温分布会导致铸件不同位置收缩不均，产生变形与残余应力[[S111,S41,S35]]。

### 胀型力与锁模力

金属液高压填充型腔时产生作用在分型面上的胀型力，压铸机锁模力必须大于总胀型力。锁模力不足将导致模具胀开，产生飞边和尺寸超差[[S40,S21,S29]]。

## 典型缺陷与对策

### 粘模（Die Soldering）

粘模是铝液与模具表面发生冶金反应的失效现象。当模具表面保护涂层被高速金属流冲刷失效、金属液温度高于655 °C时，铁铝元素发生反应生成FeAl₃金属间化合物层，铸件凝固后与型腔表面发生粘结，开模时造成铸件表面拉伤甚至难以取出[[S62,S102,S25]]。

**模具端成因**：脱模斜度过小甚至为负值、型腔表面粗糙度高、金属液局部流速过高冲刷型腔面、模具局部温度超过焊合临界温度、表面保护涂层脱落失效、狭窄凹槽结构设计不合理造成局部金属液滞留[[S108,S67,S54]]。

**预防与修复**：对模具型腔进行TiN、CrN等硬质涂层处理，优化内浇口尺寸降低冲击速度，优化冷却系统保持模温低于粘焊临界温度，拉伤部位沿脱模方向打磨抛光，定期清理型腔表面并重喷脱模剂[[S96,S108,S76]]。

### 冲蚀（Erosion）

多发生在内浇口附近，高速高温金属液长期冲刷下，表面保护涂层首先被冲刷脱落，随后金属液直接与模具基体发生熔蚀反应，造成型腔表面材料流失，出现不规则凹坑损伤[[S62,S96,S27]]。

**模具端成因**：模具钢抗高温软化能力不足，Mo、V等耐热合金元素含量偏低，钢材纯净度不达标（S/O含量高），内浇口设计不合理造成冲击速度过高[[S63,S113]]。

**预防与修复**：选用耐液态金属熔损性能好的高纯度热作模具钢，浇口区域做硬质涂层强化处理，适当降低压射速度，冲蚀损伤区域采用氩弧焊补焊修复后重新抛光[[S62,S86,S96]]。

### 热龟裂（Heat Checking / Thermal Cracking）

压铸模占比最高的失效模式，约70%的模具最终失效由热疲劳开裂引起。产生机理是模具型腔表面在高温金属液和低温冷却介质反复交替作用下，表层材料产生周期性热胀冷缩应变，当累积塑性应变超过材料承受极限时逐步生成微裂纹，最终连接为网状龟裂纹，裂纹深度通常不超过1 mm[[S69,S48,S71]]。

**模具端成因**：尖角过渡处未做大圆弧处理造成应力集中；模具预热不足导致极大内外温差热冲击；热处理工艺不当造成表面残留高残余应力或脱碳；开模后立即直接喷大量含水脱模剂造成模面骤冷加速裂纹扩展；长期使用未定期做去应力回火处理[[S57,S96,S16]]。

**预防与修复**：热处理阶段控制型腔硬度在合理区间，淬火后配合氮化等表面强化工艺；所有型腔拐角处设置足够大平滑圆角；新模投产前逐步预热到工作温度区间；开模后先用高压气吹扫模面缓慢降温再喷涂脱模剂；按铸件质量对应模次阈值执行去应力回火处理（质量 <0.1 kg：25000模次；0.1~1 kg：10000模次；>1 kg：5000模次）[[S19,S86,S12]]。

### 模具变形

**模具端成因**：整体刚度设计不足，在高压金属液作用下出现弹性或永久塑性变形；冷却系统设计不合理，型腔不同区域温差过大造成不均匀热膨胀变形；浇口垫渣/局部过热导致对应区域高温软化出现局部凹陷变形[[S30,S76,S65]]。

**预防与修复**：校核垫板厚度，必要时增加支撑柱提升整体刚性；优化冷却水路布局确保温度场均匀；对局部变形凹陷区域氩弧焊补焊后重新修配加工至要求尺寸[[S30,S77,S103]]。

### 型芯折断

**模具端成因**：细长型芯结构设计强度不足，在金属液高速冲击下发生反复弯曲疲劳断裂；型芯冷却水路布局不合理造成局部温度过高软化；型芯安装固定不到位出现松动窜动导致交变载荷下断裂[[S93,S26,S101]]。

**预防与修复**：校核型芯结构强度，加粗根部圆角过渡降低应力集中；优化型芯内冷却水路控制工作温度；定期点检活动型芯安装紧固状态，折断型芯取出后更换同规格备件[[S93,S96,S26]]。

### 披锋（飞边/Flash）

压铸件沿分型面位置出现的向外延伸的层状多余金属薄片[[S91,S97,S76]]。

**模具端成因**：分型面维护不到位长期残留铝屑杂物造成合模间隙过大；镶块、滑动抽芯块等活动部件长期磨损配合间隙超差；模具整体刚性不足在高压填充下出现胀型变形导致分型面贴合不严。

**预防与修复**：每班生产前清理分型面残留铝屑杂物；定期点检所有镶块、抽芯滑块配合间隙，磨损后立即配研修复；模具结构设计阶段校核整体刚性[[S100,S77,S91]]。

## 寿命与维护

### 影响寿命的主要因素

压铸模寿命受多重因素影响[[S22,S8]]：

- **材料因素**：钢材冶金质量（洁净度、有害元素含量）、锻造工艺、选用钢种
- **热处理因素**：真空热处理可减少脱碳、氧化、变形和开裂；成型零件淬火后必须进行2次及以上回火，仅调质后直接氮化会导致数千模次后即出现表面龟裂、开裂[[S94,S18]]
- **使用条件**：正式生产前必须对模具进行预热，不预热直接填充高温金属液会导致型腔内/外层产生极大温度梯度引发裂纹甚至开裂；配套温控系统使模具维持在合理工作温度区间可显著延长寿命[[S22]]
- **应力管理**：需定期进行去应力回火，回火温度取480~520 °C，真空回火可取上限[[S94,S90]]

| 合金类型 | 去应力回火周期（模次） |
|---------|-------------------|
| 锌合金 | 20000~50000 |
| 铝合金、镁合金 | 10000~20000 |
| 铜合金 | 500~1000 |

### 日常分级维护

- **小维护**（单批次生产结束后）：拆卸顶出部件清洗，清除分型面、型腔附着的金属残留和脱模剂污垢，检查滑动型芯润滑情况[[S23]]
- **中维护**（数个生产批次完成后）：修复浇道、溢流槽、排气道的损伤部位
- **大维护**（生产模次达到模具预期寿命1/3时）：修整模框平面度和直角度，消除成型件残留热应力，更换老化顶杆[[S23]]

### 焊接修复规范

焊前彻底清除表面缺陷区域的变质层，H13钢预热温度需达510 °C。焊接过程中工件温度低于260 °C时需重新加热，焊条成分须和模具母材成分一致，采用氩气保护焊。焊后立即升温至475 °C并以25 °C/h速率缓慢降温消应力，完全冷却后再进行型腔精加工[[S36,S98]]。

### 镶块更换

易受金属液直接冲蚀的内浇口区域和局部易损型芯设计为独立可更换镶块，采用过盈配合安装。当镶块接合面出现间隙导致压铸件产生飞边时，拆除旧镶块重新研配或更换新件，恢复原精度，无需整体报废型腔，大幅降低维修成本[[S7,S20]]。

## 标准与规范

### 中国国家标准

- **GB/T 8844**《压铸模 技术条件》：规定了压铸模零件技术要求、总装配技术要求等内容[[S68,S92]]
- **GB/T 4678**《压铸模 零件》系列标准：涵盖锁模力范围对应压铸机用模具零件标准
- **GB/T 4679**《压铸模 零件 技术条件》
- **GB/T 8847**《压力铸造模具 术语》：规定了压铸模基本结构中常用模具零件的术语及定义
- **GB/T 44530-2024**《压铸模 模架》：2024年首次发布，规定了圆导柱、方导柱两类压铸模模架的型式、尺寸与验收要求[[S32]]
- **GB/T 8845**《模具 术语》：2017版统一整合了压铸模相关术语定义

### 国际标准与行业规范

- **NADCA（北美压铸协会）产品标准系列**：包含模具设计准则、H13压铸模具用钢热处理规范、压铸模质量验收要求，是北美压铸行业通用技术规范[[S109,S24]]
- **ISO标准**：明确了压铸模具尺寸偏差随铸件尺寸增大的线性递增规则，是压铸模具制造公差设定的通用参考国际标准[[S78]]
