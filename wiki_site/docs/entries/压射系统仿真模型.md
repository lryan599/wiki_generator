## 建模方法

### 集总参数法（机理建模）

集总参数法是目前压射系统仿真领域应用最成熟的机理建模方法。该方法将液压系统拆解为集中参数的离散元件进行分层建模，可依托AMESim的HCD（Hydraulic Component Design）液压元件库完成比例插装阀、复合结构压射缸等核心部件的独立建模，再逐步整合形成完整系统仿真模型，提升建模层次性与后续调试维护效率[[S11,S46]]。

### 数据驱动建模

数据驱动建模方法可针对性解决传统纯机理模型难以精确表征压射系统强非线性、时变特性的问题。典型实现方案采用基于时间序列的LSTM神经网络，配合小波阈值去噪算法处理实测压射过程的位移-时间序列数据，可高精度预测压射速度与插装阀阀芯位移量之间的对应关系。研究表明，传统理论公式模型预测的阀口开度存在约-17.36%的平均误差，而压射系统机理模型预测的平均误差仅为0.11%，均方根误差RMSE为0.336%，相关系数R²为0.998[[S30,S19]]。

### 混合建模

混合建模方法融合机理建模与数据驱动建模的优势，以传统液压机械机理模型作为基础框架，引入大量现场实测生产数据对模型未知参数进行校正、对难以通过理论推导的非线性环节进行补偿，可在降低对精确物理参数依赖的同时，保证模型的泛化能力与物理可解释性[[S30]]。

### AMESim仿真平台

AMESim是压射系统液压仿真领域应用最广泛的商用仿真平台，支持大吨位卧式冷室压铸机的全压射液压系统建模，可对蓄能器充放油特性、比例插装阀动态响应特性、压射全周期5个工作阶段（储能、慢速压射、快速压射、增压、回锤）的运行特性进行完整仿真验证，是行业内主流的压射系统方案验证工具[[S11,S21]]。

## 关键参数及其影响规律

### 典型仿真参数配置

以25MN卧式冷室压铸机为典型案例，压射系统仿真模型的关键参数取值如下表所示[[S11,S42]]：

| 参数 | 数值 |
|------|------|
| 恒压源压力 | 16 MPa |
| 压射蓄能器初始气体压力 | 12 MPa |
| 压射蓄能器初始气体体积 | 600 L |
| 增压蓄能器初始气体压力 | 12 MPa |
| 增压蓄能器初始气体体积 | 360 L |
| 压射缸活塞直径 | 210 mm |
| 压射缸活塞杆直径 | 150 mm |
| 压射活塞行程 | 930 mm |
| 压射缸组件重量 | 210 kg |
| 增压缸活塞直径 | 330 mm |
| 增压活塞行程 | 145 mm |
| 增压缸组件重量 | 170 kg |
| 入口比例插装阀通径 | 100 mm |
| 出口比例插装阀通径 | 80 mm |

入口通径100mm的比例插装阀满幅阶跃流量响应时间为28ms，出口通径80mm的比例插装阀满幅阶跃流量响应时间为23ms[[S11]]。

### 蓄能器参数的影响

| 参数变化 | 对压射性能的影响 |
|----------|-------------------|
| 压射蓄能器设定压力增大 | 压射阶段可达到的最大压射速度增大；增压开启时间提前 |
| 压射蓄能器容积增大 | 压射阶段速度无显著影响；增压开启时间提前 |
| 压射蓄能器设定压力过低 | 无法实现有效增压 |
| 增压蓄能器设定压力变化 | 对慢速启动、高速填充阶段的动态响应无影响 |
| 增压蓄能器设定压力增大 | 增压切换阶段完成后最终可获得的增压压力越高 |

压射蓄能器容积对压射阶段速度无显著影响；增压力可在3~29.5MPa范围内连续调节[[S8,S20]]。

### 压射速度设定的影响

压射速度设定阈值取值不合理会直接影响铸件质量：压射速度取值过低会导致铸件轮廓不清、冷隔缺陷；压射速度过高会使充型紊流加剧，铸件内部气孔、氧化夹渣缺陷增多，同时严重冲刷模具型腔表面降低模具使用寿命[[S13,S2]]。

### 比例插装阀控制方式的影响

| 控制方式 | 特点 | 局限性 |
|----------|------|--------|
| 单独出口节流控制 | 结构简单，建压后通过出口阀调节速度 | 建压过程产生最高0.4m/s的瞬时前冲启动冲击；慢速阶段速度波动大、偏差大；高速填充阶段速度超调可达3%；有杆腔压力最高升至32MPa，低速段控制稳定性差 |
| 进、出口双侧独立负载口控制 | 同步调节进、出口双侧比例插装阀 | 完全消除启动速度前冲；慢速阶段速度波动大幅降低；高速阶段速度超调降至1%以下；全压射过程两腔总压力控制在14MPa以内 |

独立负载口控制方式通过取消建压环节，使两腔压力始终保持在14MPa以下，大幅提升了低速段控制稳定性[[S5]]。

## 模型构建范式

压射系统仿真模型的规范构建遵循分层建模范式——首先在AMESim等平台中利用HCD库独立建立比例插装阀和压射液压缸部件模型，再将上述部件模型封装后与其余液压元件（方向阀、蓄能器、管路等）从标准液压库中选型整合，最终形成全系统仿真模型。负载的设置分两段：低速和快速阶段将型腔内金属液负载等效为速度阻尼负载，增压阶段等效为大弹性负载[[S18]]。

## 仿真数据生成策略

### 采样方法

拉丁超立方采样（Latin Hypercube Sampling, LHS）作为分层随机采样方法，可在压射系统仿真的多维度参数空间内均匀分布样本点，每个参数维度的样本点也保持均匀分布，能够显著减少小数据集带来的模型泛化性偏差，是面向机器学习训练集生成的主流高效采样方案[[S47,S48,S51]]。

蒙特卡洛方法可用于基于AMESim搭建的全参数化压射液压仿真模型，对所有液压元件物理参数、工艺运行参数进行全工况遍历扫略，批量生成覆盖全运行区间的仿真样本数据[[S46,S11]]。

### 故障样本注入

在压射系统仿真平台中可通过直接修改元件参数实现典型故障批量注入：蓄能器预充压力衰减通过逐步下调仿真模型中蓄能器初始气体压力参数实现；阀芯卡滞通过调整比例插装阀最大行程、阀芯摩擦系数实现；液压缸内泄漏通过设置液压缸内泄漏系数实现；管路节流故障通过修改液压管路通径、附加节流系数实现。将所有正常运行工况生成的仿真样本作为机器学习训练正样本，所有故障工况生成的仿真样本作为负样本，可快速构建故障诊断任务所需的正负样本集[[S42,S18]]。

### 测量噪声添加

面向压射系统仿真生成的速度、压力等时序传感数据，可通过两种主流方式添加模拟测量噪声：其一为在仿真输出信号上叠加指定幅度系数乘以标准正态分布N(0,1)的随机分量；其二采用sym8小波对原始仿真信号做两层分解，通过调整高频细节系数的幅度重构出与工业现场采集噪声特性一致的带噪信号[[S47,S19]]。

## 应用场景

1. **故障诊断**：基于压射仿真生成的大量标注故障正负样本训练机器学习模型，可实现压射系统运行异常状态的高精度自动识别，无需依赖生产现场采集的稀缺故障真实数据[[S6]]。

2. **剩余寿命预测**：通过蒙特卡洛参数遍历生成压射系统各元件全生命周期退化过程的仿真时序数据，训练得到的寿命预测模型可有效提前预判关键液压元件的剩余服役时间[[S1]]。

3. **工艺参数优化**：基于仿真生成的全参数组合样本可快速遍历不同浇注温度、快慢压射速度、高低速切换点的匹配效果，大幅降低实际试生产的迭代成本[[S28]]。

4. **控制系统开发与调试**：基于AMESim搭建的高精度压射系统仿真模型可离线完成压射速度控制算法的全工况调试、调优，比如验证独立负载口双阀口控制方案对压射启动冲击的抑制效果，无需占用实际压铸机台调试时间[[S5,S10]]。

## 常用简化假设与局限性

### 常用简化假设

开发压射系统仿真模型时，为降低建模复杂度与计算成本，通常采用如下简化假设[[S37,S26,S18]]：

- 保压阶段将压射系统受力状态简化为静力学方程进行求解；
- 慢压射、快压射阶段将型腔内金属液负载等效为速度阻尼负载，增压阶段等效为大弹性负载；
- 压射系统各组成部件材料满足连续性、均匀性、各向同性、小变形、完全弹性假设；
- 液压元件特性直接采用AMESim内置HCD库的标准模型近似。

### 已知局限性

| 局限性 | 说明 |
|--------|------|
| 压射室充填忽略 | 传统"流量法"压铸仿真方案仅模拟金属液在型腔内部的流动状态，完全忽略金属液在压射室阶段的充填过程，导致仿真结果与实际生产过程存在明显偏差 |
| 超高速薄壁件预测不稳定 | 在超高速薄壁镁合金高压压铸场景下，数值仿真对充型过程的微小扰动敏感性极强，预测稳定性与精度大幅下降 |
| 次要物理效应省略 | 常规仿真模型普遍省略润滑液蒸发、金属液表面氧化燃烧等次要物理效应，与极端生产工况的真实物理过程存在偏差 |
| 极端故障样本不足 | 压射液压系统模型未覆盖所有极端磨损、罕见耦合故障的物理特征，部分极端故障样本生成精度不足 |

## 相关标准与规范

### GB/T 21269-2023《冷室压铸机》

该标准明确规定了压射相关核心术语定义，包含增压、压射压力/位移/速度-时间曲线、慢压射、快压射、快压射速度、最大空压射速度、建压时间、增压压力稳态值等，同时设置第5.4节为压射性能要求、第6.6节为压射性能试验方法，配套提供卧式/立式冷室压铸机两类机型的压射性能参数表，可作为压射系统仿真模型的标定基准[[S22,S49]]。

### 压射性能重复精度要求

GB/T 21269-2018引用对应的压射性能检测规则要求：压射系统仿真的验证试验可通过连续10次空压射，分别采集慢压射速度、快压射速度、建压时间、压射增压稳态压力数据计算重复精度。具体指标如下[[S27,S40]]：

| 项目 | 重复精度允许差 |
|------|----------------|
| 慢压射速度重复精度 | ≤ 0.05 m/s |
| 快压射速度重复精度 | ≤ 0.3 m/s |
| 建压时间重复精度 | ≤ 5 ms |
| 增压压力稳态值重复精度 | ≤ 1 MPa |

### NADCA标准

NADCA 2012版《压铸件产品规范》第7章质量保证部分，明确要求压射过程的统计过程控制（SPC）验证方法，可用于压射系统仿真输出的压射工艺参数分布置信度校验[[S31]]。此外，NADCA行业通用的PQ²压射性能校核规范，通过压射系统的最大可提供压力-流量平方曲线与模具需求曲线的交点计算有效压射工作点，是压射系统仿真模型必须输出的核心验证结果[[S43,S50]]。
