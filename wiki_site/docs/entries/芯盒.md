## 定义

芯盒（英文对应 core box、sand core box、core making tool）是制造砂芯或其他耐火材料芯的专用工艺装备，其内腔形状、尺寸与待成型的砂芯完全匹配，属于铸造成型工序中制芯环节的核心工装 [[S10,S11]]。芯盒在铸造全流程中处于制芯工序的上游，通过填入、紧实芯砂后固化脱模，得到合格砂芯；后续砂芯被置入铸型内部，用于成型铸件的内腔、复杂孔洞等外部铸型无法直接成型的结构 [[S1,S17]]。中文别名包括砂芯盒、制芯工装 [[S1]]。

## 分类

芯盒的分类涵盖三个维度，各类别之间存在交叉对应关系 [[S17,S16,S1]]。

**1. 按结构类型分类**

| 结构类型 | 英文 | 特征说明 |
|---|---|---|
| 单面敞开式 | — | 整体式芯盒，不可拆卸，填砂面与砂芯支撑面为同一平面，填砂紧实后翻转180°起芯，适用于形状简单、具有平整填砂面的砂芯 |
| 水平对开式 | horizontal split core box | 由上、下两半组成，分盒面与填砂面为同一平面 |
| 垂直对开式 | vertical split core box | 由左、右两半组成，分盒面垂直于填砂面 |
| 多向开模式 | — | 含多个分盒面，适配形状复杂、需多方向开模的砂芯 |
| 套框脱落式 | troughed core box | 形成砂芯轮廓的组块在有倾斜侧壁的斗形外框内组合，造好的砂芯与组块一起自框内倒出后分离 |
| 多向拆卸式 | — | 多个可拆卸组件组合成型 |

**2. 按操作方式分类**

- 手动开盒取芯芯盒
- 机械动作开盒取芯芯盒

**3. 按适配制芯工艺/型芯类型分类**

- 普通芯盒（采用盒外烘干方式，如粘土砂芯、油砂芯、合脂砂芯、水玻璃砂芯）
- 热芯盒（需加热至200~250℃，砂芯在盒内整体受热硬化 [[S28]]）
- 壳芯盒（需加热至230~290℃，芯砂外表层受热硬化 [[S28]]）
- 冷芯盒（常温下通过气雾催化剂（如三乙胺）实现砂芯整体硬化，无需加热 [[S28,S4]]）

## 典型结构设计

### 对开式芯盒结构与前后起料原理

对开式芯盒（half core box）是应用最广泛的一种芯盒结构，由两个独立半盒通过销轴/铰链连接为整体，仅设置单一分盒面 [[S10]]。其核心操作特征为**前后双向起料结构**：开模时两侧半盒可分别向前后两个方向分离移出，无需侧向大行程动作，取芯便捷，整体结构复杂度低，适配绝大多数截面规则的简单砂芯成型需求 [[S1,S10,S15]]。

### 分盒面设计

分盒面（parting of core box）是对开或多开芯盒相互间的接触面 [[S10]]。设计原则如下：

- 优先选择平面作为分盒面，通常取砂芯最大投影面，便于脱模 [[S10,S16]]
- 分盒面需保证有较大的敞开面，便于填砂、紧砂、放芯骨、开通气道及出芯操作 [[S18]]
- 若需控制开模后砂芯定向留在指定半盒中，可将分盒面向目标留芯半盒偏移0.6~1mm，使该侧半盒的包络面更大 [[S27]]
- 针对高精度大型芯，分盒面处需预留合模补正量，每边型腔名义尺寸缩减0.15~0.2mm，以抵消合模间隙带来的尺寸偏差 [[S27]]
- 若分盒面与铸件分型面不重合，需针对性计算标注芯腔斜度起点尺寸，保证最终铸件壁厚均匀 [[S2]]

### 开合定位机构

对开式芯盒采用以下定位方式：

- **定位销与定位套**：常规配置两个圆形定位销；大尺寸芯盒可采用一个圆形定位销+一个定向销的防卡滞配置 [[S27]]。定位销、套常用T8/T10工具钢制造，工作部分淬火至40~45HRC（定位销可达50~55HRC），销直径根据芯盒轮廓尺寸匹配8/10/12mm三档 [[S9,S13,S18]]
- **止口凸阶定位**：上半芯盒边缘设凸阶，下半芯盒边缘设凹阶相配合，适用于立放在平板上填砂的小尺寸芯盒 [[S18]]

### 退芯（取芯/顶芯）结构

不同类型芯盒适配对应的退芯机构：

- 热芯盒、冷芯盒、壳芯盒均配置独立顶芯杆组件，顶芯受力需均衡，避免砂芯变形 [[S4]]
- 常规退芯方式包括：机械顶杆退芯、气动/液压驱动退芯、人工倒出退芯三种 [[S27]]

### 排气系统设计

芯盒排气方式分为两类 [[S27,S12]]：

| 排气方式 | 设计要求 |
|---|---|
| 排气槽 | 开设在分型面处，深度0.2~0.3mm，宽度10~25mm（出口端可扩至2~3mm），排气方向与吹砂方向相反 |
| 排气塞 | 布置在型腔凹陷区域、砂流交汇处、砂流返向处，需定期清理避免堵塞 |

活动件配合间隙（0.2~0.3mm）也可利用作为间隙排气通道 [[S12]]。

### 拔模斜度

芯盒内壁拔模斜度取值范围为0.5°~2° [[S9]]。设计要求：

- 同一芯盒内的拔模斜度类型尽可能统一，减少数控加工中换刀次数，提升加工效率 [[S2]]
- 若需控制开模后砂芯的留向，可将目标留芯侧半盒的拔模斜度设置为更小值，确保砂芯贴合在指定半盒中 [[S22,S9]]

### 锁紧结构

常用锁紧方式包括蝶形螺母-活节螺栓、双螺母关节式锁紧、快速铰链螺杆锁紧等，需保证填砂、紧实过程中两半盒完全贴合，不出现错位漏砂 [[S20,S13]]。

## 材料选择

芯盒材料根据制芯方法、砂芯形状、大小、复杂程度以及生产批量、加工条件等因素选取 [[S16]]。常用材料性能对比如下：

| 材料类别 | 常用牌号 | 特点 | 适用场景 |
|---|---|---|---|
| 铝合金 | ZL101、ZL102、ZL104、ZL201 | 密度低、自重轻、易加工、表面光洁不生锈；强度硬度较低，不耐磨 | 中小尺寸芯盒的中批量生产；可在摩擦工作面镶装30钢耐磨片以延长使用寿命 [[S18,S25,S13]] |
| 铸铁 | HT150、HT200 | 强度硬度高、耐磨性好、生产成本低；自重大、易锈蚀 | 大尺寸芯盒及需要200℃以上工作温度的热芯盒场景 [[S18,S1,S25]] |
| 钢材/铜合金 | — | 高耐磨性 | 多用于芯盒的镶块、活块部位，延长易损部位使用寿命 [[S18,S16]] |
| 木材/塑料 | — | 加工成本低、修改便捷 | 小批量手工制芯场景，精度要求低 [[S1,S17]] |

不同材料芯盒的使用寿命随材料硬度提升逐步延长 [[S18,S13]]。

## 制造精度与加工要求

金属芯盒的制造精度需满足以下技术条件：

| 技术指标 | 要求 | 来源 |
|---|---|---|
| 芯腔工作面尺寸精度 | IT10级（凹形工作面偏差取负，凸形工作面偏差取正）；普通金属芯盒按IT11级 | [[S23,S15]] |
| 分盒面间隙 | ≤0.1mm | [[S23,S3]] |
| 活块与窝座配合间隙 | 0.2~0.3mm，整个深度上接触良好、下落自如 | [[S23,S3]] |
| 芯腔工作面表面粗糙度 | Ra1.6μm | [[S23,S1,S21]] |
| 定位销位置偏差 | ≤±0.25mm；高精度第一套基准芯盒≤±0.05mm | [[S23,S3,S15]] |
| 中心距偏差 | 正负对称分布，精度比其他尺寸高1~2级 | [[S23]] |

毛坯加工前需进行人工时效处理以消除内应力；大型复杂芯盒粗加工后需增加二次人工时效，保证长期使用过程中的尺寸稳定性 [[S23,S21]]。

## 与压铸/铸型的关系

芯盒制出的型芯通过芯头与压铸模具（或铸型）的芯座配合安装，其配合形式、芯头斜度、间隙、定位结构需符合JB/T 5106要求 [[S7]]。型芯的尺寸精度偏差会直接传导到压铸件的内腔尺寸——芯盒精度超差将直接导致型芯变形、偏移，最终引发压铸件壁厚不均、缺料等缺陷 [[S7]]。

## 使用与维护

### 易损部位

芯盒日常使用中易损部位主要包括 [[S18]]：

- 填砂面边缘的耐磨护板
- 活块配合窝座
- 定位销与销套
- 排气槽/排气塞

### 维护要点

- 定期检查定位销套磨损情况，磨损量超过0.2mm需及时更换 [[S18]]
- 出现磨损后可通过镶配、修磨的方式修复 [[S13,S12]]
- 排气塞需定期清理，避免堵塞 [[S27,S12]]

## 工艺参数——热芯盒温度与砂芯强度

热芯盒的芯盒温度对砂芯强度有显著影响。研究表明，芯盒温度160℃搭配热空气温度120℃时，砂芯室温抗拉强度可达较高水平；过高或过低的芯盒温度均可能导致强度下降 [[S24]]。

## 标准与规范

现行有效的芯盒相关行业标准主要包括 [[S14,S7,S19]]：

- **JB/T 7699-1995**《铸造用木制模样和芯盒技术条件》：将木制芯盒分为一级、二级、三级三个精度等级，规定了不同等级的尺寸公差、平面度公差、表面粗糙度（未涂漆前）、涂漆工艺要求；适用对象包括砂型铸造用木制模样和芯盒，也适用于木质与金属混合模样和芯盒
- **JB/T 5105**《铸件模样 起模斜度》：规定芯盒的起模斜度设计要求
- **JB/T 5106**《铸件模样型芯头 基本尺寸》：规定型芯头的基本尺寸、定位形式（水平型芯头与垂直型芯头）、配合间隙参数

术语标准方面：

- **GB/T 5611-2017**《铸造术语》给出芯盒、对开芯盒、脱落式芯盒、分盒面等术语定义 [[S10]]
- **GB/T 25370-2020**《铸造机械 术语》将芯盒归入铸造工装类别，定义其为“制造砂芯用的模具，其内腔具有砂芯所需要的形状和尺寸” [[S11]]
