直浇道是压铸浇注系统的核心组成部分，指金属液从压室进入型腔前首先经过的专用通道，连接压射单元与横浇道，承担引导金属液流动、传递压力的关键作用[[S1,S43,S39,S64]]。根据GB/T 8845—2017《模具术语》，在压铸模范畴内直浇道（sprue）被定义为“从浇注系统的入口至横浇道之间的通道”[[S49]]。其结构设计直接影响金属液的充型状态与铸件的最终质量。

## 分类与结构差异

直浇道的结构与所选用的压铸机类型密切相关，可分三类[[S22,S32,S41]]：

| 压铸机类型 | 组成构件 | 主要特征 |
| :--- | :--- | :--- |
| 卧式冷室 | 压室 + 浇口套 | 直浇道直径等于压室直径；压射结束后留下的金属段称为余料（料饼）[[S22,S44]] |
| 立式冷室 | 喷嘴 + 浇口套 + 分流锥 | 喷嘴导入口至最小环形截面（I–I截面）为整个直浇道，分流锥用于调整截面、引导流向[[S1,S5,S32]] |
| 热室 | 喷嘴 + 浇口套 + 分流锥 | 由压铸机喷嘴与模具浇口套、分流锥共同构成；分流锥可伸入浇口套，截面变化较复杂[[S2,S43,S61]] |

此外，按流道几何形状可分为直锥形、圆柱形、蛇形及多直浇道等类型。直锥形（上大下小）在铝、镁合金压铸中广泛采用，可保持直浇道内正压状态、抑制涡流；圆柱形制造简便，散热面积小，多用于小型薄壁件；蛇形直浇道路径长、阻力大，能显著降低流速，适用于易氧化的铝镁合金；多直浇道结构采用多个小截面直浇道并行供料，可改善大型铸件充型的均匀性[[S43,S9,S14]]。

## 结构与几何特征

### 卧式冷室压铸机直浇道

- 直浇道直径 **D** 由压铸机压室直径决定，其值根据所需压射比压及压室充满度选定[[S4,S7,S28]]。
- 余料饼厚度 **H** 一般取直径D的 **1/3 ~ 1/2**[[S4,S7,S32]]。
- 浇口套靠近分型面一端，在长度 **15 ~ 25 mm** 范围内内孔加工出 **1°30′ ~ 2°** 的脱模斜度[[S22,S28,S32,S50]]。
- 浇口套长度应小于压射冲头的跟踪距离，以保证余料能够被顺利推出[[S2,S44,S53]]。
- 横浇道入口应开设在浇口套（或压室）的上方，防止金属液在压射前因自重流入型腔提前凝固[[S1,S22,S32]]。
- 采用中心浇口时，直浇道的设计类同于立式冷室压铸机，同样要求流道位于上方[[S32]]。

### 立式冷室压铸机直浇道

- 喷嘴导入口小端截面积一般为内浇口截面积的 **1.2 ~ 1.4** 倍，小端直径 d₁ 的计算公式为：  
  **d₁ = 2 √[(1.2 ~ 1.4) A内 / π]** ，其中 A内 为内浇口截面积[[S32,S5]]。
- 直浇道各段连接处直径单边放大 **0.5 ~ 1.0 mm**，以避免截面突然扩张[[S32,S5]]。
- 喷嘴部分脱模斜度取 **1°30′**，浇口套脱模斜度取 **1°30′ ~ 3°**[[S1,S5]]。
- 分流锥环形通道截面积一般为喷嘴导入口截面积的 **1.2 倍**左右，分流锥直径 d₃ 按下式确定：  
  **d₃ = √[d₂² - (1.1 ~ 1.3) d₁²]** ，且应满足 **(d₂ − d₃)/2 ≥ 3 mm**（d₂ 为环形截面外径）[[S4,S7,S5]]。
- 分流锥锥角一般取 **30° ~ 60°**，底部圆柱段高度 **1 ~ 6 mm**[[S17]]。
- 直浇道与横浇道连接处圆角半径取 **R5 ~ R20 mm**[[S17,S5]]。

### 热室压铸机直浇道

- 直浇道单边斜度一般取 **2° ~ 6°**[[S2,S55]]。
- 喷嘴喷口截面积推荐为内浇口总截面积的 **1.1 ~ 1.2** 倍[[S46]]。
- 直浇道最小有效截面积 Az 可按以下公式校核（参见图2）：  
  **Az = π cosα [ ((d cosα/2) + L₁ sinα)² − R² ]**  
  其中 α 为单边锥度，d 为浇口套入口直径，R 为分流锥顶部圆弧半径，L₁ 为喷嘴端面至R中心的距离[[S60,S46]]。
- 为适应高效率生产，浇口套与分流锥内部通常设置独立的冷却系统[[S4,S61]]。

### 锥度限制

直浇道必须呈锥体，扩张方向与金属液流动方向一致。当锥形浇道的总锥度大于 **8°** 时，金属液难以充满整个截面，会引起局部压力损失、低压区、涡流和包气空穴缺陷。因此，压铸行业常规推荐直浇道总锥度控制在 **6°** 以下[[S17,S41,S47]]。立式冷室压铸机常用总锥度 **6°**（单边斜度3°）；热室直浇道单边锥度为 **2° ~ 6°**[[S41,S55]]。

### 直浇道窝（缓冲窝）

在直浇道底部设置凹井结构（直浇道窝），推荐窝座直径为直浇道下端直径的 **1.4 ~ 2** 倍，高度为横浇道高度的 **2** 倍，侧壁在保证起模的前提下尽量垂直，底部为平面，转角处避免尖角[[S15,S51,S21]]。该结构的主要功用见下一节。

## 作用机理

1. **压力传递**：压射阶段压射冲头输出的高压通过直浇道及其末端的余料（料饼）持续传递至型腔内部，为铸件凝固补缩提供压力支撑，避免缩孔、缩松[[S43,S9]]。
2. **缓冲与减紊流**：直浇道窝能吸收高速下落金属液的动能并转化为压力能，大幅缩短直-横浇道交接处的强烈紊流区，降低对浇道壁的冲刷，同时改善各内浇口的流量分配均匀性[[S8,S15,S21]]。
3. **排气**：采用上大下小的锥形直浇道，充型过程中直浇道内部可维持正压状态，避免形成负压区而卷入气体；配合直浇道窝可进一步排出裹挟的气泡[[S43,S14]]。
4. **撇渣**：平稳的流态与底部窝座的容渣空间可将充型初期的低温冷污金属、氧化夹杂滞留在窝座区域，防止其随主流进入型腔[[S43,S14]]。

## 关键设计参数与流道匹配

| 项目 | 推荐值/关系 | 说明 |
| :--- | :--- | :--- |
| 卧式冷室直浇道出口截面积 | = 所有分支横浇道入口截面积之和 | 保证流量匹配，避免局部壅塞[[S1,S2,S31]] |
| 立式冷室喷嘴导入口面积 | = (1.2 ~ 1.4) × 内浇口总截面积 | 控制初始流速与压力传递[[S32,S5]] |
| 热室喷嘴喷口面积 | = (1.1 ~ 1.2) × 内浇口总截面积 | 实际设计中有时取到2倍[[S46,S55]] |
| 分流锥环形通道截面积 | ≈ 1.2 × 喷嘴导入口截面积 | 适用于立式冷室[[S1,S7]] |
| 直浇道有效截面积（热室） | > 内浇口总截面积 | 用校核公式计算确保通流能力[[S31,S60]] |
| 卧式冷室余料饼厚度 H | = (1/3 ~ 1/2) × 直浇道直径 D | 为补缩提供充足未凝固金属[[S4,S7,S32]] |
| 总锥度上限 | ≤ 6° | 超过8°将导致未充满、涡流、包气[[S17,S47]] |
| 浇口套脱模斜度 | 1°30′ ~ 3°（立式、卧式） | 卧式冷室仅在出口段15~25 mm内设置[[S1,S22,S50]] |
| 直-横浇道连接圆角 | R5 ~ R20 mm | 保证流动顺畅，减小局部损失[[S17,S5]] |

对于铝合金轮毂类铸件，内浇道、横浇道、直浇道的断面积比例推荐为 **4:5:6**，即直浇道横断面积约为横浇道的1.2倍，以保证约15 s的标准充型时间[[S24,S38]]。

## 与浇注系统其他部件的协调

- **直浇道—横浇道界面**：交接位置应设有较大圆角（R6 ~ 20 mm），避免截面突变引发紊流；横浇道入口宽度需小于直浇道底端直径；冷室压铸机的横浇道入口必须设置在直浇道的上半部分区域，防止金属液提前流入[[S1,S22,S2]]。
- **横浇道—内浇口界面**：横浇道截面积自直浇道端向内浇口端应逐步减小，不允许出现扩大段，否则会形成负压区吸入空气、加剧紊流并削弱补缩压力[[S10]]。
- **与溢流槽、排气槽的配合**：从直浇道引出的横浇道可设置长度不小于其高度5倍的延长段，末端接入溢流槽，将首批冷污金属及裹挟氧化渣导入溢流槽；同时在溢流槽侧方配置排气槽，实现彻底除渣排气[[S14,S63,S57]]。

## 常见缺陷与设计准则

| 缺陷现象 | 机理 | 相关设计准则 |
| :--- | :--- | :--- |
| 吸气、卷气 | 直浇道非充满流动时，金属液流股表面裹挟气体向下冲入型腔；流量能力过剩（达到平均需求的4倍以上）时，过早开启上层内浇口，形成瀑布式充型[[S42,S26,S56]] | 直浇道入口采用圆角，半径 **r ≥ d/4**（d为上口直径）可确保充满流动[[S42,S26]]；直浇道下部节流面截面积应控制在理论计算值的 **1 ~ 2** 倍以内[[S54,S3]] |
| 冲砂（冲蚀）、砂眼 | 直浇道底部流速过高：超过1 m/s即有明显冲砂，超过3 m/s会显著冲刷砂型/涂层[[S30,S54]] | 设置直浇道窝，窝座直径1.4~2倍直浇道下端直径[[S15,S21]]；对下落高度超过1 m的大型铸件，采用耐冲刷陶瓷窝座替代砂型窝座[[S54,S30]] |
| 冷隔、填充不良 | 直浇道流量分配不合理，远端内浇口流量不足，金属液前端提前凝固碰撞；或总流量供给不足，充型时间超限[[S48]] | 合理匹配直浇道、横浇道、内浇口的截面积比，确保各分支流量均衡[[S48,S10]] |
| 缩孔、缩松 | 直浇道出口过早阻塞或压力传递不足，补缩压力无法有效作用[[S43]] | 保证余料饼足够厚度（D的1/3~1/2），维持压力传递通道[[S4]] |

## 典型应用案例

### 铝合金轮毂

- **正置中心直浇道**：对于φ508 mm×215 mm的AlSi7Mg0.3铝合金轮毂，原反置（轮毂反面中心）浇道方案在充型0.5 s内出现金属液骤降，形成明显紊流和卷气；优化为正置中心单管底注方案后，充型流态平稳，缩松缩孔体积显著降低，成品率大幅提高[[S23]]。该方案的内浇道、横浇道、直浇道断面积比推荐为4∶5∶6，直浇道面积约为横浇道的1.2倍，保证约15 s的充型时间[[S24,S38]]。
- **重载大型轮毂**：可采用对称布置的双直浇道设计，两个直浇道对称置于轮辋区域，上部设计半球形结构存储杂质，可解决单直浇道方案轮辐厚大部位补缩不足、产生细小缩松的问题[[S36]]。

### V型气缸体

- 12V气缸体原直浇道设计面积为8438 mm²，流量能力达到平均每秒铁液流量需求的4.2倍，流量严重过剩，直浇道窝座处流速高达5.02 m/s，引发严重冲砂，同时上排内浇口在浇注开始后15 s即提前进铁，形成瀑布式流态冲刷砂芯，造成爆砂和夹杂缺陷[[S56,S30]]。
- 优化对策：减少直浇道直径，下部设为节流面（面积大于理论值、小于2倍理论值），并采用陶瓷直浇道窝座[[S54,S3]]；调整浇注系统截面积比（如L12缸体取 **S直:ΣS横:ΣS内 = 1:2.7:10.5**），延长横浇道、增加竖浇道[[S6,S11]]。优化后充型全程以层流为主，卷气大幅减少，铸件加工后废品率由53%降至4%[[S11]]。
