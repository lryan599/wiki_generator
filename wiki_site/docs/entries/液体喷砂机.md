## 概述

液体喷砂机（wet blasting machine / slurry blasting machine / vapor blasting machine / liquid honing machine）是喷砂机整体分类中与干喷砂机并列的独立子类，属于喷砂清理类表面处理/铸件清理设备下的湿法喷砂分支[[S29,S30,S20]]。其工作介质为磨料与水混合形成的砂浆/磨液，最大特征是可完全控制喷砂加工过程中的粉尘污染，大幅改善作业环境[[S15,S31,S30]]。

## 工作原理

液体喷砂机以磨液泵作为磨料的供料动力，将预先搅拌均匀的磨料-水混合磨液输送至喷枪内；同时压缩空气作为磨液的加速动力通过输气管接入喷枪，在喷枪内对磨液完成加速后从喷嘴高速射出，冲击工件表面实现清理、粗化、光饰等加工目的。在该模式下，磨液泵负责供料，压缩空气负责射流加速[[S30,S23]]。

湿喷砂（液体喷砂）工艺按动力输送原理可细分为三类：低压文氏管输送的雾化喷砂、高压泥砂泵输送的水-气喷砂、纯水压输送的水喷砂。三类设备的粗糙度调节方式不同：雾化喷砂仅可通过改变磨料粒度调整粗糙度，水-气喷砂可通过调整压缩空气压力调节粗糙度，纯水压式水喷砂仅可通过调整水压调节粗糙度[[S31,S26]]。

## 分类与命名

液体喷砂机别名包括湿喷砂机、砂浆喷砂机、水汽喷砂机、液体珩磨机，行业内也常将其对应的工艺称为湿喷砂[[S15,S31,S30]]。在设备体系中，喷砂机整体分为干喷砂机和液体喷砂机两大类，干喷砂机又可细分为吸入式和压入式两类[[S29,S30]]。

## 核心结构与组成

常规液体喷砂机由五大核心系统构成：结构系统、介质动力系统、管路系统、控制系统、辅助系统[[S30]]。具体到各功能部件，主要包括以下关键模块：

### 喷枪

液体喷砂机的喷枪分为水环式和注射式两类。水环式喷枪的喷水头套在标准喷嘴外部，通过多个小孔喷射水雾在喷口处润湿磨料实现粉尘抑制；注射式喷枪的缓蚀剂水溶液在喷嘴入口前即与磨料、压缩空气充分混合，粉尘润湿效果更优，可通过节流阀精准调节液体流量，但该结构流体压力为常规压缩空气压力的10倍，需配套专用供水泵系统[[S32]]。

### 搅拌系统

液体喷砂机配置独立搅拌管组件，对储液槽内的磨料水混合液持续搅拌，避免磨料在槽底沉降分层，保证整机运行过程中磨液配比均匀稳定，避免因局部磨料浓度过高或过低导致的喷砂效果不一致[[S6]]。

### 砂浆泵与液体循环系统

液体循环系统由磨液泵、输水胶管、控制阀、分水过滤器组成，实现磨液在储液槽-喷枪-作业腔-分离单元之间的闭环循环输送，可大幅降低磨料与水资源消耗，无需频繁补充工作介质[[S6]]。

### 工作舱

工作舱为密闭作业腔，配套透明观察窗、气动刮水器、操作防护手套、内置工件放置台（SS-2机型配置圆盘回转工作台），舱体设置独立排气口，可避免处理过程中磨液飞溅外泄，为操作人员提供安全的作业观察与操作空间[[S6]]。

### 分离器

液体喷砂机配套独立砂尘/磨料分离器，可将循环回收的混合物中失效细粉、破碎过细的废磨料与合格可重复使用的磨料进行筛分分离，一方面降低新磨料的补充损耗，另一方面避免不合格细颗粒混入喷射体系影响表面处理质量[[S34]]。

## 典型设备型号与技术参数

国内SS系列液体喷砂机包含SS-1、SS-2、SS-5、SS-5A四款主流型号，相关技术参数涵盖电机功率、工作台直径、适用磨料粒度范围、喷嘴直径、喷枪配置数量、压缩空气消耗量及工作压力、整机外形尺寸、整机重量和典型应用范围[[S6,S12]]。

CIC500型液体喷砂机核心参数包括：磨液泵与分离器水泵电机功率适配，适用磨料粒度要求大于46号筛，喷嘴直径为10~12mm，压缩空气工作压力0.5~0.6MPa，同时标注了工作舱门、圆盘工作台尺寸、整机外形尺寸及整机重量[[S2]]。

## 关键工艺参数

### 磨料与水混合比

通用工况下磨液中磨料与水的重量比为1:5；针对铸件表面精细加工场景可调整为磨料:水=1:6~1:10[[S6]]。另有资料指出，湿喷砂的水与磨料比一般控制在7:3为宜[[S28,S18]]，砂浆状态下磨料体积占比常规范围为20%~35%[[S28,S31]]。使用320~400目细磨料时需额外添加10%膨润土作为悬浮剂，防止磨料在储箱底部沉降[[S28,S31]]。

### 喷射压力

液体喷砂机压缩空气工作压力常用范围为0.5~0.7MPa[[S13]]，专用湿喷砂设备最高工作压力可达0.56MPa（约5.6公斤/厘米²）[[S18]]。

### 喷嘴直径与喷射距离

以CIC500型为例，喷嘴直径为10~12mm[[S2]]。喷射距离覆盖10~120mm区间[[S13]]。

### 适用磨料种类

液体喷砂适用磨料以刚玉（Al₂O₃）、碳化硅（SiC）为主，精细加工场景可选用玻璃砂等非金属磨料[[S6,S28]]。铝及铝合金零件不宜采用碳化硅磨料[[S28]]。

不同用途对应的磨料粒度要求如下[[S18]]：

| 加工目的 | 推荐磨料粒度（目） |
|---------|----------------|
| 除氧化皮 | 30~40 |
| 粗糙表面清理 | 60~80 |
| 普通表面清理 | 120~220 |
| 高光洁度表面清理 | 320~400 |

### 磨液使用寿命

国产SS系列液体喷砂机的磨液常规使用寿命为40~70工作小时[[S6,S8]]。在喷砂过程中，磨料因高速撞击工件而破碎，当磨料破碎率超过10%~15%时需全部更换[[S6]]。

### 工艺参数分类选择

液体喷砂工艺参数可依据铸件光饰与精细加工两类场景进行区分：精细加工需设置更低压力与更小喷射角，喷射距离可根据铸钢/高温合金、有色金属的材质特性分别调整[[S24]]。

## 在压铸与金属加工中的应用

### 适配压铸行业的特点

面向压铸加工领域，液体喷砂机依靠水介质缓冲磨料的冲击力度，相比干式喷砂更适合高精度压铸铝合金、镁合金等非铁合金铸件的表面精饰，不会对薄壁精密压铸件造成表面过度划伤，可实现无粉尘作业，适配压铸行业连续化流水线布局[[S33]]。

### 典型应用场景

液体喷砂机在压铸行业的典型应用场景包含四类[[S29,S11,S14]]：

1. **精密压铸件微小毛刺清理**：可在去除边角毛刺的同时避免压铸件表面发生二次划伤。
2. **压铸件表面氧化层、浮锈去除**：有效清除轻微氧化皮与锈蚀。
3. **压铸件亚光消光处理**：可获得均匀的哑光表面效果，适配外观件的表面装饰需求。
4. **涂装、电镀前预处理**：可在压铸件表面构建均匀的微观粗糙界面，大幅提升镀层、涂层与压铸件基体的结合力，涂层服役寿命比普通简易前处理提升4~5倍。

此外，经认真清理的表面与一般简单清理的工件相比，涂层寿命可相差4~5倍[[S29]]。国产SSI型（SS系列）液体喷砂机占地面积仅约2㎡，无需专用喷砂作业房间，作业过程中液体自带缓蚀剂可保证喷砂后数天内压铸件不会返锈，已在钢制压铸件镀锌前处理工序中实现成熟工业应用[[S14]]。

## 与干喷砂及其他湿法喷砂的对比

干喷砂直接输出干磨料流，加工后的工件表面粗糙度更高，适合残砂、氧化皮等重清理作业；液体喷砂由于水的缓冲作用磨料冲击强度更低，加工后工件表面更细致均匀，适合高光饰度精密铸件的表面处理[[S16,S5,S21]]。

无循环简易非密封湿喷砂的清理效率约为3.5~4m²/h，低于循环式液体喷砂机，仅适合小批量低要求的除锈作业[[S4]]。

| 对比维度 | 干喷砂 | 液体喷砂（循环式） |
|---------|-------|-----------------|
| 粉尘产生 | 大量游离粉尘，需配套大功率通风除尘系统 | 全程无游离粉尘溢出[[S22,S8,S30]] |
| 表面效果 | 较粗糙，适合重清理 | 较细致均匀，适合精密光饰[[S16,S21]] |
| 磨料消耗 | 磨料损耗大，不可循环 | 磨液可循环使用，损耗少[[S8,S22]] |
| 适用工件 | 各类铸件清理 | 精密铸件、非铁合金、薄壁件[[S33]] |

## 操作条件与维护要点

### 工作压力范围

压缩空气工作压力常用范围0.5~0.7MPa[[S13]]，专用湿喷砂设备最高工作压力0.56MPa[[S18]]。

### 磨料更换

磨料破碎率超过10%~15%时需全部更换[[S6]]。SS系列配套磨液整体使用寿命40~70工作小时[[S6]]。

### 防锈措施

钢铁工件加工时需在水中加入亚硝酸钠、碳酸钠、重铬酸钠等缓蚀剂，浓度控制在0.3%~0.5%区间[[S19,S28,S18]]。湿喷砂完成后工件需经冷流水冲洗，之后用热压缩空气吹干，钢铁件可浸入含8.66g/L苯甲酸钠+4.33g/L亚硝酸钠的70℃以上热防锈溶液中处理，或用冷脱水油封存，防锈效果可保持10~12昼夜[[S4,S18]]。奥氏体不锈钢和耐热合金钢喷砂后还需在250~300g/L的硝酸室温溶液中钝化2小时以上[[S19,S28]]。

### 污染防控注意事项

铝合金、钛合金等轻合金零件不得与钢铁件共用同一液体喷砂机，避免不同金属磨料交叉污染，影响工件表面处理质量[[S28]]。带有双层壁的钣金焊接件不宜采用液体喷砂工艺，内部狭窄间隙的残留磨液难以完全排出，极易诱发缝隙锈蚀[[S28]]。

## 安全与环保特征

液体喷砂作业全程无游离粉尘溢出，无需配套专用大功率通风除尘系统，相比干喷砂完全消除硅尘吸入的职业健康风险，大幅改善现场作业环境[[S22,S8,S30]]。磨液可重复循环使用，相比干喷砂大幅降低磨料损耗，作业产生的废弃砂浆统一收集后按危废分类处置即可，无游离粉尘排放，环保属性更优[[S8,S22]]。
