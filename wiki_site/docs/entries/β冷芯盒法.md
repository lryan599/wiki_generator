## 概述

β冷芯盒法（Beta-set process），国家标准名称为酚醛-酯冷芯盒法（phenolic ester cold box process），是一种气硬冷芯盒制芯工艺。该工艺采用水溶性碱性甲阶酚醛树脂为粘结剂混制芯砂，射制砂芯后吹入甲酸甲酯气雾，使砂芯在室温下快速硬化[[S2,S11]]。该术语明确收录于我国铸造术语国家标准 GB/T 5611-2017[[S2,S11,S1]]。

作为冷芯盒制芯工艺中的低毒气体硬化分支，β冷芯盒法与以三乙胺为催化剂的α冷芯盒法（酚醛-异氰酸酯胺气固化工艺）并列，同属气硬冷芯盒工艺下的酚醛树脂系[[S11,S18,S14]]。其固化剂甲酸甲酯对人体危害极小，芯盒无需特殊密闭，残余硬化剂可直接排入大气，尾气处理工艺大幅简化，是一种环保友好型的精密制芯技术[[S16]]。

## 定义

根据 GB/T 5611-2017《铸造术语》国家标准：

> **酚醛-酯冷芯盒法［β冷芯盒法］**：用水溶性碱性甲阶酚醛树脂混制的芯砂射制砂芯，吹入甲酸甲酯气雾而使砂芯硬化的制芯法。[[S2,S11]]

英文对应术语为 **phenolic ester cold box process** 或 **β-set process**[[S2,S14]]。β冷芯盒法属于低毒气体硬化气硬冷芯盒工艺[[S11,S8]]，在冷芯盒分类体系中归属于气硬冷芯盒工艺下的酚醛树脂系分支[[S11,S18]]。

## 分类体系与工艺定位

### 冷芯盒法总体分类

冷芯盒制芯工艺按粘结剂与固化介质的不同，可划分为六大子类[[S17,S18,S2]]：

| 序号 | 工艺名称 | 粘结体系 | 固化介质 |
|------|----------|----------|----------|
| 1 | 水玻璃吹 CO₂固化法 | 水玻璃 | CO₂气体 |
| 2 | SO₂冷芯盒法 | 酚醛树脂/呋喃树脂+有机过氧化物 | SO₂气体 |
| 3 | 胺气雾固化α冷芯盒法（三乙胺法） | 酚醛树脂+聚异氰酸酯双组分 | 叔胺气雾（三乙胺/二甲基乙胺） |
| 4 | **酯固化β冷芯盒法（酚醛-酯法）** | **水溶性碱性甲阶酚醛树脂单组分** | **甲酸甲酯气雾** |
| 5 | 聚丙烯酸酯树脂 CO₂冷芯盒法 | 聚丙烯酸钠+钙离子硬化剂 | CO₂气体 |
| 6 | 氧负离子硬化冷芯盒法 | 甲阶酚醛树脂 | CO₂气体 |

### β与α冷芯盒法的分类差异

β冷芯盒法与α冷芯盒法虽同属气硬冷芯盒工艺，但在分类层面存在三项核心差异[[S2,S12,S18]]：

| 对比维度 | β冷芯盒法 | α冷芯盒法（三乙胺法） |
|----------|-----------|----------------------|
| 粘结体系 | 单组分水溶性碱性甲阶酚醛树脂 | 双组分：酚醛树脂（组分Ⅰ）+ 聚异氰酸酯（组分Ⅱ） |
| 固化介质 | 甲酸甲酯气雾 | 叔胺类气雾（三乙胺/二甲基乙胺） |
| 固化反应产物 | 直接生成体型酚醛树脂 | 生成氨基甲酸酯结构树脂 |

## 工艺过程与设备

### 完整工艺流程

β冷芯盒法所适配的气硬冷芯盒完整工艺流程依次为[[S22,S6,S24]]：

1. **配砂混制**——将原砂与水溶性碱性甲阶酚醛树脂按比例混合均匀；
2. **射砂充填芯盒**——混制好的芯砂通过射砂筒射入常温芯盒内紧实成形；
3. **吹入甲酸甲酯固化气雾**——以干燥气体为载体，向芯盒内吹入雾化的甲酸甲酯，使砂芯快速硬化；
4. **通入净化气吹扫**——吹入清洁干燥空气，清除砂芯内部残留的硬化介质；
5. **开盒取芯**——打开芯盒，取出已硬化砂芯，供后续合箱浇注使用。

### 核心配套设备

β冷芯盒法的核心配套设备包含以下类型[[S17,S21,S24]]：

- **混砂设备**：适配单砂单混、单砂双混、双砂三混三种混砂模式的混砂机系统；
- **冷芯盒射芯机**：分为水平分盒和垂直分盒两大类，主机结构与热芯盒射芯机相似，但增设吹气固化装置；
- **甲酸甲酯专用气体发生装置**：用于将液态甲酸甲酯雾化并与载气混合；
- **空气干燥与净化系统**：压缩空气干燥除湿系统，以及尾气净化装置；
- **冷芯盒工装**：满足密封要求、保证固化气体均匀分布的芯盒结构。

### 吹气固化系统原理

β冷芯盒固化工艺系统包括催化剂存储发生器、定量控制混合装置、压缩空气管路、密封式/开放式芯盒及尾气净化器等核心部件，通过精确控制甲酸甲酯气雾的生成、输送与排放实现砂芯的高效均匀固化[[S9]]。

### 冷芯盒工装结构

β冷芯盒工艺允许采用多种材料制作芯盒，包括木质、塑料、铝合金及铸铁等。对于小批量生产场景，可采用无加热要求的常温芯盒（如手动式导轮结构冷芯盒）来降低工装成本[[S21]]。

## 原材料与技术要求

### 粘结剂树脂

β冷芯盒法所用粘结剂为水溶性碱性甲阶酚醛树脂，其主要技术指标如下[[S16,S13]]：

| 技术指标 | 要求 |
|----------|------|
| 氮含量（质量分数） | ≤ 0.5% |
| 粘度 | ≤ 1000 mPa·s |
| 游离酚（质量分数） | ≤ 1% |
| 游离甲醛（质量分数） | ≤ 0.5% |
| 树脂添加量 | 原砂质量的 1.5% ~ 1.8% |
| 适用场景 | 大批量、中等复杂型芯 |

### 固化剂

β冷芯盒法所用有机酯固化剂主要成分为甲酸甲酯（methyl formate），其雾化工艺参数为[[S16]]：

| 参数 | 数值 |
|------|------|
| 雾化温度 | 约 60 ℃ |
| 雾化压强 | 0.2 MPa |

甲酸甲酯对人体危害极小，采用酯固化工艺时芯盒无需特殊密闭，设备工装要求低，残余硬化剂可直接排入大气[[S16]]。

### 原砂要求

β冷芯盒法可选用硅砂、锆砂、铬铁矿砂等作为原砂，其中应用最广泛的是硅砂。对原砂质量有严格要求[[S23,S5]]：

| 指标 | 要求 |
|------|------|
| 含水量（质量分数） | < 0.2% |
| 含泥量（质量分数） | < 0.3% |

不同类型冷芯盒树脂的主要性能指标及适用范围对比如下[[S13]]：

## 性能特点

### 核心优势

β冷芯盒法及其粘结剂体系具有以下突出性能特点[[S16,S12,S5]]：

1. **无硫无氮污染铸件**：粘结剂不含硫，仅含少量氮和碳（氮含量 ≤ 0.5%），铸件不会出现增硫、氮气孔、增碳类铸造缺陷，特别适合对氮、硫敏感的合金铸件。
2. **溃散性优良**：酯硬化法砂芯的落砂性能与呋喃树脂砂相近。
3. **旧砂可再生**：旧砂可通过热法或机械法实现再生。
4. **环保性优于α冷芯盒法**：硬化剂甲酸甲酯对人体危害极低，芯盒无需特殊密闭，设备工装要求更低，残余硬化剂可直接排入大气，尾气处理工艺大幅简化。
5. **常温制芯节能**：芯盒无需加热，节省能源消耗，且可采用成本较低的木质或塑料芯盒。
6. **砂芯尺寸精度高**：常温硬化消除了芯盒受热变形的问题。
7. **存放期较长**：硬化后砂芯可存放一定时间。

## 在压铸与铸造中的应用

β冷芯盒法适配低压铸造、高压压铸（高压铸造）等多种铸造工艺场景，尤其适合以下应用[[S12,S5]]：

- **对氮、硫敏感的合金铸件**：如铝合金、镁合金铸件的高精度复杂砂芯，β冷芯盒法不会额外引入氮、硫组分，可有效避免因氮引起的针孔缺陷风险及球墨铸铁件因硫导致的球化衰退。
- **球墨铸铁件**：无硫配方避免了传统磺酸固化呋喃树脂砂导致的铸件表面石墨球化衰退缺陷。
- **有色金属精密铸件**：因其尺寸精度高、无有害元素污染，在非铁合金铸造行业有广泛应用。

相比三乙胺法、SO₂法冷芯盒体系，β冷芯盒法在保证砂芯精度的同时，从根源上杜绝了硬化剂对铸件冶金质量的潜在干扰[[S12]]。

## 缺陷机理与质量影响

### 砂芯强度不足缺陷

产生机理为：芯砂含水量超标（原砂含水量 > 0.2%、净化空气受潮）会直接阻碍粘结体系交联反应进程，导致砂芯存放过程中强度快速衰减，无法满足转运和浇注受力要求[[S20]]。

### 铸件表面/皮下气孔

产生机理为：芯砂中总树脂添加量过高（超出原砂质量 1.8%）、树脂配比失衡，浇注时过量有机组分受热分解释放大量气体，超出铸型排气系统承载阈值，最终在铸件表层或皮下形成气孔[[S20]]。

### 铸件表面光亮碳/皱皮缺陷

产生机理为：砂芯受热后树脂释放的烃类物质在铸件表面沉积，若浇注过程中金属液紊流严重、铸型排气不畅，该类缺陷的发生概率会显著升高[[S20]]。

## 局限与改进方向

### 当前技术局限

根据行业实践文献，β冷芯盒法存在以下主要局限[[S12]]：

1. **固化剂气味问题**：传统甲酸甲酯固化剂挥发性气味刺激性较强，对车间通风净化系统配置提出更高要求。
2. **旧砂再生率偏低**：旧砂再生率仅为 70% ~ 75%，显著低于呋喃冷芯盒体系的 75% ~ 80%，单位铸件废砂排放量更高。
3. **砂芯抗吸湿性较差**：接触高湿度环境或涂刷水基涂料后，砂芯强度降幅可达 30% 以上。

### 改进方向

当前行业内已广泛推广的β冷芯盒法改进方向以**低气味改性酯类固化剂升级**为主，通过对原配方中高挥发性的甲酸甲酯组分进行改性替换，大幅降低车间作业环境的 VOC 排放，改善工人操作环境[[S12]]。
