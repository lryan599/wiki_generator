## 概述与定义

砂丸分离器，亦称丸砂分离器、钢丸分离器、丸料分离器，是抛/喷丸清理设备磨料循环系统的核心组成部件，在GB/T 25370-2020《铸造机械 术语》中收录为条目6.2.27[[S4,S5]]。该标准将其定义为“利用风力和/或磁力等方式从使用过的磨料中去除杂质，分离出可再用磨料的装置”，其中采用磁力方式工作的亚型也被称为磁力分离器[[S5]]。在压铸后处理尤其是采用砂芯的铝合金、锌合金、镁合金压铸件的抛丸/喷丸清砂除芯环节中，砂丸分离器负责将可复用的金属弹丸（钢丸、铁丸）从回收混合料中分离出来，同时将残留型砂、粉尘、失效碎丸、氧化皮等废料分级排出，实现弹丸的循环回用与砂料的回收[[S22,S21]]。

## 分类与选型

国内常用的砂丸分离器主要分为风选式（帘幕式/流幕式）和磁风选联合式两大类[[S6,S22]]。选型的核心依据是待处理丸砂混合料中的含砂量，具体选型规则如下表所示[[S6,S22]]：

| 含砂量范围 | 适用机型 | 典型应用场景 |
|---|---|---|
| ≤5% | 风选式（帘幕式）丸砂分离器 | 常规抛丸清砂 |
| 5%~20% | 两级风选或先风选后磁选结构 | 抛丸除芯清砂 |
| >20% | 磁选+风选复合式（磁风选联合式） | 抛丸落砂除芯清砂 |

风选式分离器利用密度差与粒度差实现分离，结构相对简洁；磁风选联合式在风选基础上引入磁选滚筒，对高含砂量工况具有更优的分离精度与弹丸回收率[[S13,S6]]。

## 结构组成

砂丸分离器的通用结构主要由以下部分组成（参照标准结构示意图）[[S6,S21,S13]]：

1. **进料斗/进料口**：接收斗式提升机输送上来的丸砂混合物料。
2. **螺旋给料器**：对来料进行均匀布料，保证分离区全宽度喂料均匀。
3. **滚筒筛/筛网**：筛除混合料中的大块芯骨、砂团等粗渣，粗渣从废料口直接排出。
4. **分离室**：风选式分离器采用风室，磁风选联合式则包含风室与磁选滚筒组合段，是不同物料轨迹分流的核心工作区域。
5. **多组调节板**：包括一级撇滤板、二级撇滤板、舌板、节流板（闭板）等，用于调整不同物料组分的分隔边界，优化分离精度[[S2,S6]]。
6. **多级出料口**：分别设置完全分离合格弹丸出口、未完全分离混合料返回口、再生砂出口、粗渣废料出口。
7. **吸风口**：对接除尘系统，抽走微细粉尘，维持设备整体负压运行[[S17,S21]]。

## 工作原理

### 风选式（帘幕式）分离原理

风选式砂丸分离器利用混合料中弹丸、砂粒、粉尘的密度差和粒度差进行分离[[S6]]。工作时，丸砂混合料受重力作用自由下落，经定量门控制形成均匀帘幕。在水平气流作用下，不同密度和粒度的颗粒所受气流曳力不同，产生差异化的偏斜轨迹[[S6,S8]]：

- 密度大的钢丸/铁丸几乎垂直下落，进入储丸斗供循环使用；
- 碎丸和大颗粒砂子落入一级撇滤板与二级撇滤板之间的中间暂存区，返回到清理室待下一次分离；
- 细砂落入再生砂出口；
- 微细粉尘被气流抽入除尘器净化。

### 磁风选复合式分离原理

磁风选复合式砂丸分离器由螺旋滚筒筛、分离壳体、一级磁选滚筒和二级磁选滚筒等部件组成[[S13]]。丸砂混合物先在螺旋滚筒筛中筛除大块杂质，随后均匀布流下落形成流幕。下落过程中风系统先吸走部分粉尘，物料进入一级磁选滚筒，金属弹丸被磁场吸附，随滚筒旋转至无磁区后下落，细小碎丸被气流抽走。未被磁选吸附的含砂混合物经舌板落入二级磁选滚筒再次分选，型砂自然落下排出，漏选的弹丸被二级滚筒吸附分离后回用[[S13]]。该方式分离效率可达99.5%，弹丸损失量极低，分离出的型砂洁净度高，可直接送入砂处理系统回用[[S13]]。

## 工艺参数与性能指标

风选式砂丸分离器的关键工艺参数与行业标准性能要求汇总如下：

| 参数/指标 | 数值/要求 | 来源 |
|---|---|---|
| 帘幕区域分离风速 | 4.5~5.0 m/s（推荐值），一般取4~5 m/s | [[S2,S21]] |
| 系统风压 | 100~150 mmH₂O | [[S21,S6]] |
| 分离区风量占比 | 占分离器总风量的60%~68% | [[S2,S6]] |
| 分离效率 | ≥99%（风选式），最高可达99.5%（平衡锤帘幕式、磁风选式） | [[S2,S13,S22]] |
| 合格弹丸中废料含量 | ≤1% | [[S2,S13,S17]] |
| 废料中合格弹丸含量 | ≤1% | [[S2,S13,S17]] |
| 大型风选式帘幕分离器最大处理能力 | 420 t/h | [[S10]] |
| 设备运行状态 | 负压运行 | [[S17]] |
| 配套除尘排放浓度 | ≤150 mg/m³ | [[S17]] |

可调节参数包括：定量门高度（控制物料流量）、舌板伸出位置、一级和二级撇滤板调节板位置、闭板重锤位置以及除尘管道蝶阀开度，这些参数的协同调节可适配不同工况下的分离精度要求[[S2,S6,S13]]。

## 在压铸后处理中的角色与配套关系

砂丸分离器是抛丸清理设备（包括抛丸落砂除芯设备）的七大核心组成部件之一[[S16]]。在压铸后处理清理流程中，其位于丸料循环输送链路的终端、抛丸器储丸斗的上游[[S16,S3]]。典型的丸料循环流程为：抛丸作业后掉落的弹丸、型芯砂、氧化皮等杂物经螺旋/振动输送机收集，送入斗式提升机，提升后物料先经滚筒筛筛除粗杂物，再进入砂丸分离器完成分级分选，分离后的合格弹丸直接进入抛丸器上方的储丸斗，等待下一轮循环使用[[S16,S18]]。

砂丸分离器与斗式提升机需协同工作：提升机的输送能力须与分离器的额定处理能力匹配，避免供料不足或物料溢出[[S15,S18]]。

砂丸分离器与除尘系统紧密联动：分离器的吸风口直接接入除尘系统管路，分离室内部风速需大于粉尘悬浮速度且小于弹丸悬浮速度，一般取3~5 m/s[[S15]]。除尘系统的总吸风量须匹配分离器工况，同时兼顾清理室的通风吸尘需求；喷抛联合清理场景下总吸风量需额外增大50%~80%[[S15,S18]]。

## 分离异常与影响

砂丸分离器运行过程中的典型异常及后果包括[[S7,S21]]：

1. **筛面杂物未及时清理**：导致内部通道堵塞，降低丸料循环输送效率，影响抛丸器供丸稳定性。
2. **风速、帘幕厚度等参数失准**：引发分离效果不达标，使回用弹丸中含砂量超过1%的行业标准限值。据文献记载，弹丸含砂量为2%时，抛丸器叶片的磨损速度将达到使用纯钢丸工况的5~10倍[[S22]]；将含砂量控制在1%以下可大幅降低易损件磨损速度、延长叶片使用寿命，同时稳定抛丸射流强度，保障铸件表面处理质量的一致性[[S22,S21]]。
3. **核心部件异常磨损**：风选壳体、磁选滚筒、配套抛丸器叶片等因长期未维护或在含砂量超标工况下运行，出现加速磨损。

日常运维要求主要包括：定期清理分离器筛面杂物保障通路顺畅，按工艺规范调整定量闸门开度、分离区风速、撇滤板与舌板位置，确保分离效果处于最优区间[[S7]]。

## 应用场景延伸

砂丸分离器普遍配置于带砂芯的铝、锌、镁合金压铸件后处理抛丸除芯生产线，作为抛丸落砂（芯）清理设备的核心部件实现丸砂分离与弹丸回用[[S1,S21]]。此外，砂丸分离器同样适用于砂型铸造、消失模铸造等其他铸造工艺的丸料回收与旧砂再生工序，用于分离弹丸中混入的型砂与粉尘，提升旧砂回用率[[S13,S9]]。
