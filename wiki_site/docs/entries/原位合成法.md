## 概述

原位合成法（in-situ synthesis），又称熔体内生法，是一种制备颗粒增强铝基复合材料的核心工艺，其基本原理是通过化学反应在铝合金基体内部原位生成增强相，以达到改善材料性能的目的[[S16,S24]]。该方法属于金属基复合材料制备体系中的内生复合法大类，于20世纪末期正式形成，后续衍生出多种分支技术[[S16,S1]]。

## 工艺原理与反应机制

### 热力学与动力学条件

原位合成反应的设计需满足三项核心原则：反应在热力学层面具备可行性、生成的增强相在铝合金基体中为热力学稳定相、整体反应进程具备动力学可控性[[S19]]。

以Al-煅烧高岭土体系原位生成纳米Al₂O₃颗粒为例，其热力学计算式为 ΔG_T⁰ = -2093230 + 166.41T，令 ΔG_T⁰ = 0 得反应温度上限为12578.75K。常温下该反应热力学趋势很大，但动力学条件不足，反应极难发生；提升烧结温度或延长保温时间可满足动力学要求，启动反应[[S49]]。动力学分析可通过不同升温速率下的DSC热分析曲线结合Kissinger方程计算反应活化能，升温速率升高会导致对应放热峰的特征温度向高温方向偏移[[S49]]。

对于燃烧合成类原位反应（如自蔓延高温合成法），采用绝热燃烧温度 T_ad 作为反应能否自我维持的核心热力学判据，该判据可根据产物298K生成焓、不同温度下产物热容及熔化热等参数通过分段公式计算得到[[S11,S37]]。

### 典型反应体系

**Al-Ti-B体系**：通过向铝熔体中加入KBF₄和K₂TiF₆混合盐，原位生成TiB₂增强颗粒。典型商业Al-Ti-B中间合金中含大尺寸Al₃Ti相和微米级TiB₂颗粒，TiB₂呈六边形盘状形貌[[S40,S14]]。

**Al-Ti-C体系**：采用熔体反应法合成TiC增强颗粒，微观组织中大尺寸Al₃Ti相表面附着有块状形貌的TiC颗粒。相比Al-Ti-B体系，TiC颗粒尺寸更细小，不易聚集成团，可避免Zr中毒、硬颗粒团聚堵塞过滤器等应用缺陷[[S35,S14,S33]]。

**Al-Ti-B₄C体系**：该体系可同时生成TiC和TiB₂两种增强相，反应路径分为三个阶段：第一阶段铝粉先熔化包裹Ti粉和B₄C粉，优先生成Al₃Ti中间相；第二阶段Al₃Ti转化为AlnTi并与B₄C反应生成中间产物，形成Al-Ti-B-C共熔液相；第三阶段B₄C分解释放的[C]、[B]与[Ti]达到临界反应浓度时，优先生成热力学更稳定的TiC，待C元素消耗完毕后B元素过饱和析出TiB₂，所有不稳定中间产物最终完全转化为稳定的TiC和TiB₂颗粒[[S21,S6]]。

**Al-Zr体系**：向铝熔体中加入K₂ZrF₆盐，原位合成Al₃Zr颗粒。Al₃Zr与α-Al基体的晶格匹配度最高可达99.2%，润湿性良好，可有效细化铝基体晶粒，显著提升复合材料力学强度[[S32]]。

**Al-O体系**：Al-TiO₂-C混合体系在1200℃以上烧结时发生分步原位反应——首先Al与TiO₂反应生成α-Al₂O₃颗粒和游离[Ti]原子，后续[Ti]与C反应生成TiC颗粒，最终得到纳米级Al₂O₃和亚微米级TiC混杂沿晶分布的增强相[[S42]]。

## 工艺类别与子方法

原位合成制备颗粒增强铝基复合材料可按反应物初始状态分为气-液、固-液、固-固三大反应模式[[S16,S38,S2,S10,S43]]：

| 反应模式 | 子方法 | 简要说明 |
|---------|--------|---------|
| 固-液 | 混合盐法（固体盐添加） | 将KBF₄、K₂TiF₆等无机盐混合后加入铝熔体，反应生成TiB₂、ZrB₂等陶瓷增强颗粒[[S12,S2]] |
| 固-液 | 接触反应法 | 将反应粉末混合压制成预制体，压入铝熔体利用熔体热量诱发反应[[S2]] |
| 固-液 | 熔体直接反应法 | 将反应物粉末直接加入铝熔体中反应生成颗粒，搅拌均匀后浇铸[[S2]] |
| 固-固 | 自蔓延高温合成法（SHS） | 1967年前苏联Merzhanov等人提出，依靠反应释放的自维持热量完成后续合成，节能、产物纯度高、反应速度快[[S27]] |
| 固-固 | 放热弥散法（XD技术） | 1983年美国Martin Marietta实验室在SHS基础上改进，无需引燃装置，反应过程更易控制，产物致密度更高[[S7]] |
| 气-液 | 气液反应合成法 | 将含C、N元素的反应性气体通入高温铝合金熔体，原位生成TiC、AlN等增强相[[S12,S43]] |

搅拌方式包括机械搅拌、超声振动搅拌、电磁搅拌三类，可进一步分为全液态搅拌与半固态搅拌两种模式。液态搅拌更利于增强体均匀分散但易卷入气体，半固态搅拌可减少缺陷但易出现颗粒分散不均、晶粒粗大问题[[S19,S25]]。常规搅拌转速区间为200~500rpm，最大适配增强相质量分数不超过30%[[S19,S25]]。

## 关键工艺参数

| 参数项 | 典型范围/要求 | 说明 |
|--------|-------------|------|
| 反应温度 | 720~900℃ | 为铝熔体提供足够的反应能量[[S45,S5]] |
| 保温/反应时间 | 10~60min | 保证反应充分进行[[S45,S5]] |
| 搅拌转速 | 200~500rpm | 促进反应物分散与增强体均匀分布[[S19,S25]] |
| 保护气氛 | 氩气 | 避免铝熔体高温氧化[[S45,S5]] |
| 增强相质量分数 | ≤30% | 过高将导致严重团聚[[S19,S25]] |

## 特征与优势（与外加颗粒法对比）

相较于外加颗粒法（ex-situ），原位合成法具有以下显著优势[[S22,S12,S38,S8]]：

| 对比维度 | 原位合成法（in-situ） | 外加颗粒法（ex-situ） |
|---------|---------------------|---------------------|
| 颗粒表面状态 | 无污染，原位形核长大 | 表面可能存在氧化、污染层 |
| 界面结合强度 | 高，润湿性良好 | 较低，存在润湿性差、相容性问题 |
| 热力学稳定性 | 增强相为热力学稳定相，后续加工不分解 | 可能出现有害界面反应 |
| 颗粒尺寸 | 可达亚微米至纳米级 | 通常较粗大，易团聚 |
| 分布均匀性 | 原位生成，分布较均匀 | 需额外分散工序，易偏聚 |
| 工艺复杂度 | 省去增强体预合成工序，工艺简化，成本低 | 需独立合成、处理增强体 |

典型性能对比数据：采用混合盐法制备的5wt%原位TiB₂/6061Al复合材料，抗拉强度较基体提升31.9%；260℃下原位8.3wt% TiB₂p/ZL109复合材料的抗拉强度较基体ZL109提升105MPa，远高于同体积分数外加颗粒增强铝基复合材料的高温性能[[S12]]。

## 缺陷与局限

原位合成法制备颗粒增强铝基复合材料存在以下典型缺陷[[S18,S30]]：

| 缺陷类型 | 形成机制 |
|---------|---------|
| 颗粒偏聚 | 凝固过程中未完全固定的增强颗粒被固液界面前沿推移至晶间区域；高体积分数下颗粒间范德华力引发团聚 |
| 反应不完全残留 | 反应参数控制不当导致部分反应物未完全转化为目标增强相，残留于基体中形成硬脆杂质相 |
| 副产物杂质 | 部分反应体系生成非目标有害相，劣化材料塑性 |
| 熔体脱气困难 | 原位放热反应过程中易卷入大量气体，常规除气工艺难以完全去除，残留形成孔隙缺陷 |

针对上述缺陷的主流控制方案包括：采用高能超声+机械搅拌复合作用分散增强颗粒，抑制颗粒团聚偏聚；通过热力学计算优化反应温度、保温时长参数以保证反应完全；采用挤压铸造、压力浸渗等高压成型工艺封闭微孔隙；配套热等静压处理进一步消除残余气孔[[S30,S17]]。

## 在铸造/压铸中的应用

### 典型产品案例

原位合成法制备的颗粒增强铝基复合材料已应用于汽车零部件制造：

- 以A356合金为基体，通过铝-煅烧高岭土体系放热弥散法制备原位纳米Al₂O₃p/A356复合材料，可配套挤压铸造工艺成型，通过调整不同压射比压参数优化显微组织与力学性能[[S8,S3]]。
- 采用原位混合盐反应法在BH135铝合金熔体中生成TiB₂颗粒增强相，制备出的原位铝基复合材料活塞在300℃条件下抗拉强度较基体合金提升30%，可达160MPa[[S44]]。

### 流动性与充型特性

颗粒增强铝基复合材料的流动性随增强颗粒体积分数上升而降低，成因包括两方面：一是等效凝固潜热随颗粒占比增加而减小，导致流动提前终止；二是熔体黏度随颗粒占比上升而增大。但复合材料充型过程中流动更平稳，射流与涡流边界模糊，无明显紊流，获得的温度场更有利于定向凝固[[S41,S31,S29]]。针对流动性偏低的问题，采用高斜率压力充型方案可快速充型，配合低压铸造工艺可大幅提升充型能力[[S41]]。

- **中国标准**：GB/T 15115（压铸铝合金）
- **国际标准**：ISO 3522（铸造铝合金-化学成分和力学性能）、ISO 2379（铝合金砂型铸件标准试样）、ISO 7722（铝合金铸件通用交货条件）、ISO 9915（铝合金铸件射线检测）
- **美国ASTM标准**：ASTM B85/B85M（铝合金压铸件）、ASTM B26/B26M（铝合金砂型铸件）
