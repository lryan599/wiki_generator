## 概述

相场法（Phase-field method，也称为Phase-field model、Phase-field theory、PF method），中文文献中亦常称为相场方法、相场理论或相场(PF)方法[[S44,S32,S36,S38]]，是一种基于金兹堡-朗道（Ginzburg-Landau）二级相变理论构建的求解界面演化问题的数学模型[[S44,S12]]。该方法也被称为直接微观组织模型，主要应用于凝固动力学模拟[[S44]]。相场法通过在固液界面处引入称为序参量的连续辅助场，替代传统模拟方法中显式追踪界面的操作，从而自动处理复杂界面拓扑演变问题[[S44,S25,S32,S6]]。在尖锐界面极限（界面厚度趋于无穷小时），相场模型可还原符合物理规律的真实界面动力学[[S32,S6]]。

相场法由Fix和Langer于20世纪80年代首次提出原型框架[[S32,S6]]。1994年，Kobayashi首次利用相场法实现了纯金属过冷熔体中二维枝晶生长的仿真模拟，成为该领域发展的标志性节点[[S33,S42,S25]]。此后，WBM模型、KKS模型和Steinbach模型等经典相场模型相继提出，分别适配不同合金体系的凝固模拟场景[[S38,S44]]。Karma和Rappel提出的薄界面极限方法进一步降低了相场模拟对超精细网格尺度的要求，推动相场法在凝固领域的定量应用[[S44,S32]]。

## 基本原理

### 序参量与相场变量

相场法的核心是引入空间/时间依赖的连续有序参量——相场变量φ，用以表征系统各时空位置的物理状态（液态或固态）[[S44,S25]]。在经典表述中，相场变量φ在纯液相、纯固相区域内各取一个恒定特征值，在固液两相区的有限宽度扩散界面区间内从一个特征值连续过渡到另一个特征值[[S44,S25,S32]]。

当前主流取值约定包含两类[[S44]]：
- **φ ∈ [-1, 1]**：φ = -1代表液相，φ = 1代表固相[[S8,S44,S34]]
- **φ ∈ [0, 1]**：φ = 0代表液相，φ = 1代表固相[[S25,S36]]

在固液两相区，φ的值在特征值之间连续变化，形成具有有限厚度的扩散界面[[S44,S25,S6]]。对于多相、多晶粒系统，需引入多个独立序参量，每个序参量对应一个特定相或单个晶粒，并满足序参量求和约束 sum(φ_i) = 1[[S34,S17]]。

### 自由能泛函

相场模型通过构造系统的全局自由能泛函，基于变分原理推导相场演化控制方程。基于自由能函数法的标准自由能泛函形式为[[S44,S7,S8]]：

$$F = \int \left[ f(\phi, U, C) + \frac{\varepsilon^2}{2} |\nabla\phi|^2 \right] dV$$

其中：
- **f(φ, U, C)** 为金兹堡-朗道型双阱自由能密度[[S44,S7]]
- **ε** 为梯度能修正系数，决定界面层厚度[[S44,S8]]
- **U** 为无量纲焓[[S44,S7]]
- **C** 为溶质浓度[[S44,S8]]

二元合金场景下，局部自由能密度可进一步展开为[[S36]]：

$$f(\phi, T) = h(\phi)f^S + (1-h(\phi))f^L + W g(\phi)$$

其中h(φ)为势插值函数（可取h(φ)=φ³(10-15φ+6φ²)或h(φ)=φ²(3-2φ)），g(φ)=φ²(1-φ)²为剩余自由能函数，W为双阱势高度，f^S、f^L分别为固相和液相自由能[[S36]]。

此外，基于熵函数框架的相场模型给出系统总熵的构建形式为 S = ∫[s(φ,e,C) − ε²/2|∇φ|²]dV，该框架引入独立的熵梯度函数和界面能梯度函数，可消除非平衡条件下的非物理异常项，为参数调整提供额外自由度[[S44,S7]]。

### 相场控制方程

基于热力学势能最小化原理变分推导，典型相场控制方程分为三类[[S44,S7]]：

| 方程类型 | 表达式 | 描述 |
|----------|--------|------|
| 相场演化方程（Allen-Cahn型）| ∂φ/∂t = −M_φ·δF/δφ | 描述有序参量的时间演化 |
| 溶质演化方程（Cahn-Hilliard型）| ∂C/∂t = ∇(M_c ∇ δF/δC) | 描述溶质浓度的扩散与再分配 |
| 焓演化方程 | ∂U/∂t = ∇(M_u ∇ δF/δU) | 描述温度场的演化 |

其中M_φ为相场迁移率，M_c为溶质迁移率，M_u为热迁移率，均为对应过程的动力学参数[[S44,S7]]。

### 界面能各向异性

枝晶沿特定晶体学择优取向生长是金属凝固的典型特征。在Karma-Rappel相场模型中，通过引入随界面法向矢量n变化的各向异性梯度系数实现对各向异性的表征[[S7,S17]]：

$$A(\vec{n}) = (1-3\varepsilon)\left[1 + \frac{4\varepsilon}{1-3\varepsilon} \cdot \frac{\phi_x^4 + \phi_y^4}{|\nabla\phi|^4}\right]$$

将该系数耦合进入相场控制方程后，即可模拟枝晶沿特定取向生长的各向异性行为[[S7]]。

### 多相场模型

针对多相、多晶粒系统，多相场模型定义n相通用自由能密度函数，满足序参量求和约束 ∑φ_i = 1[[S17]]。通过总系统自由能极小化推导每个相场变量的演化规则，适合描述多晶粒竞争生长、包晶反应等复杂凝固场景[[S17]]。常用多相场模型中，Steinbach模型可直接将纯物质相场模型扩展为多相场模型，从而实现多元多相合金微观结构的动态模拟[[S38]]。

## 相场-温度场-溶质场耦合

相场法的一个核心优势是可以与温度场、溶质场、流场等多物理场灵活耦合，实现复杂条件下凝固微观组织的综合仿真[[S44,S16,S20]]。

### 温度场与溶质场耦合方程

在多相耦合场景下，温度场传热守恒方程形式为[[S22]]：

$$\sum_{i=1}^n \phi_i (\rho c_p)_i \frac{d}{dt} T = \nabla(\lambda\nabla T) + \sum_{i,k=1}^n \frac{L_{ik} q_{ik}}{2}$$

其中(ρc_p)_i为i相的比热容，λ为有效热导率，L_ik为相变热。在能量守恒方程中引入符合高斯分布的热扰动项，可用于模拟二次枝晶臂的生成[[S22]]。

对应的溶质场传质守恒方程形式为[[S22]]：

$$\frac{d}{dt}c = \nabla D_\lambda \left[\nabla c - c \sum k_{i\lambda}\nabla\phi_i\right]$$

其中D_λ为有效溶质扩散系数，k_{iλ}为相i与参考相之间的分配系数[[S22]]。

## 历史发展与标志性文献

相场法的发展经历了若干重要里程碑[[S32,S12,S25,S38,S44,S42,S33]]：

| 时期 | 标志性事件 | 意义 |
|------|-----------|------|
| 20世纪80年代 | Collins、Caginalp最早提出相场模型框架；Fix和Langer首次提出耦合Cahn-Hilliard方程与Allen-Cahn方程求解凝固问题的相场原型 | 理论奠基 |
| 1994年 | Kobayashi首次构建纯物质凝固的相场数学模型，成功复现纯金属二维枝晶形貌 | 首次验证可行 |
| 此后 | WBM模型、KKS模型、Steinbach模型三类经典相场模型相继提出 | 适配不同合金模拟场景 |
| 此后 | Karma和Rappel提出薄界面极限方法 | 降低网格要求，推动定量应用 |

三类经典模型的特点[[S38]]：
- **WBM模型**：假定固液界面由浓度相同的固、液相混合构成，含额外双阱势假设
- **KKS模型**：假定平衡时界面由化学势相同的固、液相混合构成，适合稀二元合金凝固模拟
- **Steinbach模型**：通过纯物质与合金之间变量匹配，直接扩展为多相场模型，可实现多元多相合金微观结构动态模拟

## 数值求解方法

相场控制方程的数值求解主要采用有限差分法和有限元法两类方法[[S31,S43]]。

### 两种方法的比较

| 属性 | 有限差分法 | 有限元法 |
|------|-----------|---------|
| 近似本质 | 数学层面的近似，用差商替代微分方程中的导数项 | 物理层面的近似，通过离散为插值单元得到模型化结构的数值精确解 |
| 网格类型 | 结构化规则网格 | 非结构化单元，可灵活生成任意疏密、任意形状 |
| 实现特点 | 实现逻辑简单、计算效率高、内存占用低 | 边界拟合精度高，可对复杂几何铸件域进行高精度求解 |
| 适用场景 | 大规模二维、三维规则域相场快速模拟，尤其是流动场、温度场的高效求解 | 包含复杂曲面边界、薄壁结构的铸件全尺度相场耦合模拟 |
| 引用 | [[S2,S31,S43]] | [[S45,S4]] |

此外，相场模拟的空间步长选取至关重要。过大的步长会导致精度损失，过小的步长则显著增加计算负担；同时步长设置若不满足稳定性判据，会引发数值发散[[S11]]。

## 应用场景

### 枝晶生长模拟

相场法能够直接解析枝晶侧枝发育、二次/三次臂生长等微观细节，精准揭示固液界面迁移的动力学机制[[S23,S3,S25]]。Kobayashi最早通过相场法实现纯金属过冷熔体中枝晶生长的二维仿真[[S42,S33]]。此后研究者们实现了纯金属过冷熔体中枝晶生长的二维和三维模拟，得到了与真实凝固组织相似的模拟结果[[S36]]。

对于多组元复杂合金，将相场模型与热力学数据库、动力学参数耦合构建的多元相场模型，可显著提升枝晶生长模拟的精度，定量建立合金成分、凝固工艺参数与枝晶形貌之间的对应关联关系[[S13]]。

### 溶质再分配与偏析预测

相场法通过量化溶质再分配行为，能够动态追踪两相界面和溶质扩散[[S13]]。针对镍基高温合金定向凝固的相场模拟结果显示：晶粒倾角低于20°时一次枝晶臂间距与0°倾角的基准值无显著差异，当倾角超过20°后一次枝晶臂间距随倾角增大呈显著上升趋势[[S13]]。

康永生等人利用相场法模拟铁碳合金定向凝固的液相通道，指出适当降低提拉速度可减少液相通道引起的偏析[[S42]]。Hara等人采用相场法分析Cu-Fe合金凝固过程与微观组织，证实冷却速度、合金成分对铁相晶粒尺寸存在显著调控作用[[S42]]。

### 微观组织演化

相场法可灵活耦合温度场、溶质场、流场、应力场等多物理场，实现复杂条件下的凝固微观组织耦合仿真[[S16,S20]]。该方法已扩展至定向凝固、快速凝固、增材制造熔池微观演变等各类场景[[S20]]。

## 优点与局限性

### 优点

1. **物理基础严谨**：基于Ginzburg-Landau相变理论构建，具备严格的统计物理理论基础，通过构造全局自由能泛函、基于变分原理推导控制方程，直接体现体系自由能最小化的热力学本质[[S16,S12]]
2. **无需界面追踪**：将固液界面处理为具备有限厚度的连续过渡区，无需显式追踪界面位置与几何形态，自动处理复杂界面拓扑演变[[S6,S39,S25]]
3. **可解析微观细节**：可直接解析枝晶侧枝发育、二次/三次臂生长、溶质偏析、晶粒粗化、共晶相析出等微观细节[[S23,S3,S25]]
4. **多物理场耦合能力**：可灵活耦合温度场、溶质场、流场、应力场等，实现复杂条件下微观组织耦合仿真[[S16,S20]]

### 局限性

1. **计算量极大**：采用传统均匀网格有限差分解算时，计算量随计算域边长的三次方量级增长[[S9,S27]]。引入自适应网格技术后，可将计算量从与边长三次方成正比降至与边长二次方成正比[[S9]]
2. **模拟尺度受限**：常规仿真的典型空间尺度被限定在微米级范围，仅能完成单枝晶或少量晶粒的微观演变模拟，无法直接覆盖厘米级以上的宏观铸件全尺度仿真[[S30,S25,S27]]
3. **网格依赖性**：为保证界面模拟精度，空间步长Δx必须远小于界面厚度，步长选取过大会直接导致求解精度损失，同时不符合稳定性判据的设置会引发数值发散[[S11]]
4. **模型调试难度高**：传统相场模型针对二元合金推导，拓展至多元素多相合金时需结合热力学数据库完成参数校准，且多取向晶粒演化需要额外增加相场方程[[S39]]

## 与其他模拟方法的对比

金属凝固微观组织模拟领域还有多种其他方法，与相场法各有优缺点和适用场景。

### 方法对比表

| 方法 | 核心特点 | 优点 | 不足 | 引用 |
|------|---------|------|------|------|
| **确定性模型** | 基于凝固动力学 | 符合晶粒生长物理背景 | 无法处理随机过程，难以模拟枝晶生长形貌 | [[S27]] |
| **蒙特卡罗(MC)法** | 基于最小界面能和概率统计 | 可定性描述温度、浓度对组织的影响 | 缺乏严格物理基础，无定量物理关联性，模拟步长与实际时间不对应 | [[S27,S25,S28]] |
| **元胞自动机(CA)法** | 基于离散元胞与局部演化规则 | 计算效率高，可实现毫米级以上大尺度仿真，能定量描述过冷度及溶质影响 | 状态转移规则多为经验设定，缺乏严格的连续物理方程支撑，存在格点依赖性 | [[S28,S39]] |
| **相场法** | 基于Ginzburg-Landau理论的弥散界面模型 | 无需追踪相界面，物理逻辑清晰，可直接计算晶粒生长过程和枝晶内部亚结构 | 计算量大，可模拟尺度小，工业场景落地难度高 | [[S27,S9,S25]] |

### 相场晶体法（PFC）

相场晶体法可在原子尺度揭示晶体位错、缺陷演化等微观行为，仿真精度较常规相场法更高，但其计算量远高于常规相场法，可模拟的空间尺度通常仅为纳米级，不适合常规凝固过程中枝晶生长的介观尺度仿真[[S19]]。

### 水平集方法（Level Set）

水平集方法将自由界面描述为距离场函数的零值面，在铸造充型过程的流体流动自由面追踪领域应用成熟，可高效处理大变形界面的拓扑演化。但该方法仅面向界面几何追踪，未集成凝固相变的热力学动力学求解框架，无法独立完成完整的凝固微观组织演化仿真[[S35]]。

## 在压铸特殊条件下的适用性

压铸工艺具有高压（型腔压力通常可达15~500 MPa）、薄壁构件超快冷却等特殊工况[[S37]]。当前压铸领域的主流商用CAE软件如ProCAST已内置适配压铸工况的宏观充型、凝固模拟模块，同时也集成了可预测微观组织演化、成分偏析的相场类微观分析能力[[S37,S42]]。然而，面向压铸特有的高压环境、薄壁构件超快冷却工况的专项相场模拟公开研究成果相对较少[[S37]]，相场法在压铸微观组织模拟中的专项应用仍有待深入发展。
