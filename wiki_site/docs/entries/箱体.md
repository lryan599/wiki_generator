## 概述

在铸造技术语境下，“箱体”指一类用于支承齿轮、轴、轴承等传动部件，固定传动系统空间定位并传递载荷的机械设备基础核心结构件[[S49,S68]]。该类铸件需保证内部各运动件保持正确相对位置以实现协调运转，并承受内部零部件的重力、相互作用力及部分工况下的冲击载荷，因此应具备足够的抗压强度、刚度以及良好的耐磨性和减震性[[S68,S20]]。

铸造语境下的"箱体铸件"必须与日常通用中空存储容器严格区分：箱体铸件作为机械装备专用的功能结构件，须预留高精度轴承孔、定位销孔、润滑油路等复杂内腔和装配接口，满足传动系统指定的几何公差和力学性能要求[[S49,S68]]。其国际通用对应英文术语为 cast housing、cast casing，典型子类对应术语包括 transmission case、flywheel housing、crankcase、pump housing 等[[S9,S85,S80]]。

## 分类

### 按应用场景分类

箱体铸件按应用场景可分为曲轴箱、变速箱壳体、泵体、压缩机壳体、气缸体、机床主轴箱、进给箱、溜板箱等子类，广泛应用于汽车、通用机械、机床、航空航天等装备领域[[S68,S49]]。

**材质选型规则**：常规工况（以静压力为主、要求减震的箱体，如普通机床主轴箱）常选用灰铸铁（如HT200、HT250）；重载且受较大冲击载荷的箱体可选用铸钢；轻量化、自重要求低的工况多采用铸造铝合金[[S68,S64,S53]]。

### 按铸造工艺分类

箱体铸件按铸造工艺可分为砂型铸造箱体、金属型铸造箱体、压力铸造箱体、消失模/实型铸造箱体等主流子类，不同工艺适配不同生产批量与精度要求的箱体产品[[S40,S86]]。

## 结构与壁厚特征

### 典型结构要素

箱体类铸件的典型核心结构要素包含封闭/半封闭型腔、阵列式布置的加强筋、集成冷却油道/水道、轴承座安装凸台、法兰安装面五大类特征，多数复杂箱体类铸件会同时集成多类特征以满足装配、刚度、散热的综合需求[[S22,S87,S59]]。

### 壁厚设计准则

不同铸造工艺下箱体类铸件的通用壁厚范围如下[[S33]]：

| 铸造工艺与合金 | 常规推荐壁厚 |
|---|---|
| 砂型灰铸铁箱体 | 3~8 mm |
| 砂型铸铝箱体 | 3~6 mm |
| 金属型铝合金箱体 | 2~5 mm |
| 压铸铝合金箱体 | 1.5~4 mm |

**壁厚过渡设计**：当相邻位置壁厚比≤2时，直接采用适配尺寸的铸造圆角过渡即可；当相邻位置壁厚比>2时，必须设置渐变过渡段，避免局部金属集中形成热节[[S29,S36]]。

**加强筋设计要求**：外表面加强筋厚度取被加强壁厚度的0.6~0.8倍，内腔布置的加强筋厚度取对应连接壁厚度的0.6~0.7倍；大平面区域可采用直方格形或交错方格形布置，多条筋交汇位置可设置不通凹槽避免金属积聚形成热节[[S2,S30,S12,S52]]。

### 热节与缩孔倾向

箱体类铸件的典型热节集中分布区域为轴承座安装凸台根部、法兰与箱体壁衔接位置、多壁交叉交汇处[[S70,S14,S54]]。局部热节的内接圆直径如果超过相邻壁厚度的1.5倍，会显著提升缩孔、缩松的产生概率[[S14]]。

热节判断方法主要有三种[[S8,S23,S46]]：
1. **内接圆法**：通过测量局部交汇位置的最大内接圆直径快速预判热节大小
2. **模数法**：对比不同区域的模数数值判断冷却顺序
3. **数值模拟法**：采用POORS判据预判宏观缩孔分布，搭配Niyama判据预判显微缩松分布

## 铸造工艺设计

### 浇注系统设计

铝合金箱盒类铸件优先选用底注式浇注系统，金属液从型腔底部平稳上升充型，有效避免紊流、飞溅和卷气氧化，利于实现顺畅排气和浮渣上浮，适配易氧化铝合金的充型要求，可配合上部冒口实现自下而上的顺序凝固，降低薄壁箱体的缺陷率[[S63,S61,S6,S44,S38]]。

![铝合金箱体金属型铸造底注式浇注系统结构示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/4113166b-fbf7-498d-8726-45fb110a7647/resource)

底注式浇注系统在充型过程中存在流速波动，部分点位流速峰值可超过1.4 m/s，须严格控制浇注流量匹配以保证工艺稳定性[[S50]]。

### 分型面选择与抽芯机构

铝合金箱体金属型铸造的垂直分型选型需满足以下原则[[S61,S6,S44,S75]]：
- 分型面数量尽量少，优先选用平面分型保证两半模合模精度，降低错箱偏差
- 避开加工基准面设置分型线
- 减少活块使用量，适配侧向抽芯机构的布局
- 开模时保证铸件随带顶出机构的半模脱出

对于带侧向凹陷、侧开孔特征的铝合金箱体铸件（如齿轮箱两侧耳部空腔），需设置对称分布的侧抽芯机构，适配垂直分型结构，避免砂型铸造用活块易偏移翘曲的缺陷，满足复杂内腔的顺利脱模要求；大深度型芯可配套风冷结构控制模具热场均匀性[[S44,S75]]。

### 冒口设计方法

**缩管法**是铝合金金属型箱体冒口设计的有效方法，其核心计算逻辑为以铸件整体体积为基数、按设定收缩率（铝合金取3%）分配总补缩量，直接反算冒口内部缩管容纳空间。主要几何关系如下[[S75,S39,S44]]：

| 参数 | 计算公式/取值 |
|---|---|
| 补缩处热节模数 | _M_h（按冒口根部被补缩处热节计算） |
| 缩管体积 | _V_op = 平均分配到每个冒口的补缩量 |
| 缩管直径 | _D_p = (2/π·_V_op)^(1/3) |
| 缩管高度 | _H_p = 2·_D_p |
| 冒口保温壁厚 | _W_ = 2·_M_h |
| 冒口中径 | _D_m = 2·_W_ + _D_p |
| 冒口最小压力头 | _H_m = 20 mm |
| 冒口总高度 | _H_ = _H_p + _H_m |
| 冒口锥度 | ≈ 0.3° |

与模数法（以冒口与被补缩部位凝固时间匹配为核心，要求_M_r = _f_·_M_c，_f_取1.1~1.2）相比，缩管法从总补缩液量需求出发直接反算，无需优先计算冒口模数，更适配定收缩率的非铁合金铸件场景，可缩小冒口体积、提升铸件工艺出品率[[S81,S75,S55,S79,S44]]。

### 模具材料与表面处理

**5CrMnMo钢**为低耐热高韧性热作模具钢，适配工作温度低于500 ℃的铝合金箱体金属型模具。推荐热处理流程[[S48,S42,S4]]：

| 工序 | 工艺参数 | 硬度 |
|---|---|---|
| 锻后等温退火 | 840~860 ℃保温2~4 h，700~720 ℃等温4~8 h，炉冷至500 ℃出炉空冷 | ≤229 HBW |
| 淬火 | 850~880 ℃油冷 | 56~58 HRC |
| 回火 | 480~520 ℃ | 42~47 HRC |

> 提升淬火温度至900 ℃以上可获得板条状马氏体，显著提升模具服役寿命[[S42]]。

**H13（4Cr5MoSiV1）钢**为中耐热高热强性热作模具钢，执行GB/T1299-2000标准，适配铝合金压铸、金属型箱体模具。推荐热处理流程[[S76,S32]]：

| 工序 | 工艺参数 | 硬度 |
|---|---|---|
| 锻后退火 | 860~890 ℃保温3~4 h，炉冷至500 ℃出炉空冷 | ≤229 HBW |
| 淬火（两次预热后） | 600~650 ℃、800~850 ℃分段预热，1020~1050 ℃加热油冷/空冷 | 54~58 HRC |
| 回火（两次，第二次降低20~30 ℃） | 530~580 ℃ | 47~49 HRC |

> 500 ℃附近为回火二次硬化峰，须规避以避免韧性劣化[[S76]]。

**QPQ表面处理**：铝合金箱体金属型模具型腔工作面可采用碳氮氧复合QPQ盐浴处理（低温氮碳共渗后续氧化工序），处理后型腔表面耐磨性、抗铝液冲刷粘连性能显著提升，模具服役寿命可大幅延长。该工艺已在ZAlSi12铝合金齿轮箱全金属型量产模具上得到实际应用验证[[S44]]。

## 典型工程应用案例

### 全金属型齿轮箱铝铸件（ZAlSi12）

铸造手册公开的一项典型案例为：铸件材质ZAlSi12铝合金，箱体最大轮廓尺寸307 mm×236.8 mm×90.5 mm，平均壁厚4 mm，最厚位置27 mm（箱盖最厚处16 mm），厚大部位集中在中心孔周围凸台及安装螺钉位置。其工艺要点如下[[S44,S57]]：

1. 外型和型芯全部采用经QPQ表面处理的模具钢制作；为减少大平面变形和残余应力，非干涉区域结构做沉降处理以减小整体平面尺寸
2. 采用立式底注、左右分型的全金属型重力铸造工艺；对箱体对称分布的凸台位置设置抽芯机构，保证"耳朵"类结构充型完整无偏移翘曲
3. 厚大凸台热节位置采用紫铜棒设置冷却结构，主型腔芯部采用风冷，上部设置冒口保证自下而上的顺序凝固，依照缩管法理论计算设计冒口尺寸，完全消除凸台位置缩孔风险

### 压铸箱体工艺要点

压铸工艺适用于壁厚薄（1.5~2.0 mm）、腔体深度大、结构复杂的箱体件大批量生产，可实现高压高速充型，生产效率高，并避免充型过程中铝合金氧化夹渣问题[[S58,S72,S43,S25]]。模具配合面间隙需严格控制，合模后红丹检查红丹率要求超过90%，避免铝液挤入模具间隙形成夹铝、毛刺缺陷导致后续装配漏油问题[[S72]]。

## 缩孔缩松缺陷与防治

### 形成机理

箱体铸件厚大凸台、热节位置缩孔缩松的形成机理[[S51,S35,S69,S46,S74,S5]]：

1. 液态合金充满型腔后，紧靠型壁处首先结壳，宽液-固两相共存区逐步向铸件中心推进；发达的树枝晶将中心部分的合金液分隔成许多独立小液体区，这些区域同时凝固得不到液态金属补充，最终形成分散的缩松
2. 纯金属、共晶合金和结晶温度范围较窄的合金逐层凝固，倾向于形成集中缩孔，常分布在厚大凸台、最后凝固的热节区域
3. 铸件结构不合理（存在局部厚实部位形成孤立热节）、合金收缩率大、内浇口位置不当截面积过小、比压低、模具温度过高等因素均会进一步加剧缩孔缩松的形成风险

### 工艺防控措施

箱体铸件厚大凸台热节缩孔缩松防控的主流工艺措施[[S51,S35,S74,S69,S44]]：

| 措施类别 | 具体方法 | 典型参数示例 |
|---|---|---|
| **冒口补缩** | 依据顺序凝固原则设计冒口，厚大凸台对应冒口根部尺寸可根据仿真结果调整 | 电机传动箱体热节处顶部冒口根部尺寸从96 mm×40 mm增大至110 mm×45 mm |
| **冷铁布置** | 在底部厚大凸台、孤立热节对应型壁位置设置外冷铁加速冷却 | 厚大位置增设10 mm厚铝冷铁 |
| **浇注温度梯度控制** | 浇注温度和铸型温度适配箱体材料，结合底注式浇注系统构建自下而上递增的温度梯度 | 铝合金箱体浇注温度720 ℃、铸型温度45 ℃ |
| **压力补缩** | 压铸、低压铸造工艺中控制比压参数，保持足够压力作用于凝固阶段的合金 | — |

## 质量标准与验收依据

### GB/T 9438《铝合金铸件》

GB/T 9438针对箱体类铝铸件的质量验收要求[[S24,S62,S15,S3,S46]]：

- **铸件分类**：I类（关键部位，损坏危及整机安全）；II类（重要部位，损坏影响部件运行）；III类（一般部位）；未注明类别时默认按III类执行
- **力学性能**：采用附铸或本体切取试样检测时，三根试样的抗拉强度和伸长率平均值分别不低于GB/T 1173规定值的75%和50%
- **表面质量**：不允许存在冷隔、裂纹、缩孔、穿透性缺陷及严重浇不足、机械损伤缺陷；金属型铸件非加工表面分型、顶杆痕迹凸起不超过1 mm，凹下不超过0.5 mm
- **无损检测**：I类铸件X射线探伤需100%覆盖，II类铸件按比例抽查；有承压密封要求时需开展渗漏试验

### ISO 3522:2006 / ISO 7722:1985

ISO 3522规定18个铸造铝合金牌号，分为砂型和金属型铸造用、压铸用、高纯度铝合金三大类；压铸用铝合金在该标准中未强制要求力学性能，其余牌号的最低力学性能均做明确规定[[S19,S60]]。

ISO 7722将铸件控制等级分为3类：1类强制检验内部致密性和力学性能；2类按需检验；3类无特殊要求常规交付；若订货未指定等级默认按3类执行[[S45,S84,S13,S47]]。

### ADC12与A380

**ADC12**是日本JIS H 5302-2006标准规定的Al-Si-Cu系压铸铝合金，Si含量9.6~12%，Cu含量1.5~3.5%，Fe<1.3%（低铁版本可控制在<0.9%）。国内GB/T 15115-2024中YL113合金与ADC12成分配比对标，单铸试棒最低抗拉强度≥230 MPa，延伸率≥1%[[S34,S77]]。常规压力铸造铸态下实际制件抗拉强度约194 MPa，屈服强度约128 MPa，延伸率1.5%，平均显微硬度约97 HV[[S78,S21,S66]]。

**A380**属Al-Si-Cu系压铸铝合金，国内相近牌号为GB/T 15115体系下的YL112（代号ZL112Y），对应日本JIS标准ADC10合金，具有良好的流动性、气密性和中等水平的抗热裂倾向性，铁含量较高可降低压铸过程粘模风险[[S65]]。

## 结论

箱体铸件是铸造技术体系中的重要结构件类别，其设计制造需综合考量合金材质（灰铸铁、铸钢、铸造铝合金）、铸造工艺（砂型、金属型、压铸）、结构壁厚特征及补缩系统等多方面因素。对于薄壁大平面铝合金箱体，全金属型铸造配合底注式浇注系统、缩管法冒口设计、QPQ表面处理模具可有效解决厚大凸台部位的缩孔缩松问题；对于大批量薄壁箱体，压铸工艺在尺寸精度、表面质量和生产效率方面具有显著优势。相关技术内容符合GB/T 9438、ISO 3522等现行标准要求。
