## 概述

**压力球化包**（英文标准名：pressurized spheroidization ladle）是一种能承受球化剂蒸气压力的带有密封盖的球化包，用于球墨铸铁铁液的密闭球化处理，属于铸造机械大类下铁液浇包子类的专用设备[[S6]]。行业内更常用的通俗称谓为**压力加镁包**[[S1,S22]]。其核心特征在于利用镁蒸气在密闭空间自建压力，将镁的吸收率提升至50%~90%，远高于常压处理工艺[[S22,S14]]。

该设备专用于球墨铸铁生产中纯镁球化剂的高效加入，与当前主流的压铸工业（处理对象为锌、铝、镁等非铁有色合金）不存在常规工艺交集——压铸领域使用的铁液浇包仅承担金属液输送功能，不具备耐压密封和镁加入机构[[S27,S10]]。本章节内容归属于**砂型铸造及特种铸造的球墨铸铁熔炼处理工序**。

## 工作原理

压力球化包的核心原理是利用纯镁的低沸点特性（大气压下约1107℃）在密闭容器内实现受控气化反应[[S14,S21]]。将包盖密封后，压入铁水液面以下的镁块受热气化，包内镁蒸气分压迅速上升，使镁的沸点随之提高，减小了镁蒸发温度与铁水处理温度（1360~1400℃）之间的差距，从而抑制铁水的剧烈沸腾与飞溅[[S14]]。整个过程无需外接加压气源，完全依靠镁自身蒸发自建平衡压力[[S14]]。

这一压力封闭环境使镁元素在铁水中的溶解吸收率大幅提升，一般可达50%~90%[[S22,S14]]，显著优于冲入法的25%~45%[[S17,S8]]。

## 分类

压力球化包按工作原理和结构形式可分为三类主流结构[[S23,S21,S20,S14]]：

| 类别 | 工作原理 | 典型特征 |
|------|----------|----------|
| 钟罩式（插入式）压力球化包 | 通过带孔钟罩将纯镁块压入铁水液面以下 | 配置导向杆和重锤机构，依靠机械下压实现镁的受控加入[[S21,S22]] |
| 灭容式压力球化包 | 将铁水上方的预留空间缩至最小，限制镁蒸发 | 镁回收率可达约90%，加镁量可低至0.1%~0.2%[[S14]] |
| 摇包/转包式（GF转包法）压力球化包 | 利用包体偏心旋转或整体转动实现球化剂与铁水的充分接触反应 | 无需额外下压机构，代表产品为瑞士GF公司开发的鼓形回转包[[S20,S11,S16]] |

## 结构与组成

以下以钟罩式压力球化包为例说明典型结构[[S18,S1]]。

**包体及密封系统**：包壳采用8mm厚度钢板焊接制作，包体有效内径与高度的比值控制在1.2~1.5，设计承压≥10个大气压[[S14]]。内壁砌筑耐火黏土+石英砂复合耐火衬，防止铁水高温烧蚀壳体[[S14]]。包体与包盖配合面设置密封垫，通过多组紧固卡子机械锁合，保证反应过程中的气密性[[S12]]。

**加镁钟罩**：核心反应部件，用于将纯镁块整体压入铁水液面以下，避免镁直接暴露于铁水表面与空气接触发生剧烈氧化燃烧，延缓镁的蒸发速度，提升镁元素在铁水中的溶解吸收率[[S21,S22]]。

**导向及压入机构**：包含导向杆、导向架和重锤，用于控制钟罩的平稳下压行程[[S1,S18]]。

**安全防护罩**：设置于包体上方，防止高压球化反应过程中铁水喷出泄漏[[S12]]。

## 工艺参数

压力球化处理的关键工艺参数范围如下[[S12,S14,S21,S16]]：

| 参数项 | 数值/范围 |
|--------|-----------|
| 包壳设计承压 | ≥10 标准大气压 |
| 球化处理前铁水温度 | 1360~1400℃ |
| 纯镁加入量（单独使用） | 铁水总重量的0.25%~0.35% |
| 镁+稀土联合压入：镁量 | 铁水总重量的0.12%~0.2% |
| 镁+稀土联合压入：稀土合金 | 铁水总重量的0.3% |
| 完整处理操作周期 | 约5~6分钟（含补加铁水、随流孕育） |
| 镁综合吸收率 | 50%~90%（GF转包50%~60%，灭容型可达90%） |
| 球化率要求 | ≥85%~90% |

## 与冲入法工艺对比

压力加镁法与当前广泛应用的传统冲入法球化工艺存在显著差异[[S17,S8,S23]]：

| 对比项 | 压力加镁法 | 冲入法 |
|--------|-----------|--------|
| 球化剂类型 | 纯镁 | 稀土镁复合合金 |
| 球化剂加入量（占铁水重量） | 0.25%~0.35% | 1.0%~2.0% |
| 镁吸收率 | 50%~90% | 25%~45% |
| 球化处理铁水温降 | 30~60℃ | 50~100℃ |
| 球化反应时间 | 60~120s | 1~2 min |
| 处理包结构 | 密封承压包体 | 开放型堤坝/凹坑式包 |

## 工艺作用与过程关系

在球墨铸铁的完整生产流程中，压力球化包处于铁水脱硫预处理工序之后[[S12]]。具体操作流程为[[S12,S17,S22]]：

1. 向压力球化包内出铁水，处理量约占浇注总铁水量的1/3~1/2，铁水温度1360~1400℃；
2. 密封包盖，将纯镁钟罩压入，球化反应在密闭压力下持续60~120s；
3. 反应结束后泄压开盖，补加高温铁水至浇注总量；
4. 同步开展炉前/随流孕育处理；
5. 扒渣、覆盖保温剂后进入浇注环节，处理完毕的铁水建议在15min以内浇注完毕。

## 安全要求

压力球化包属于铸造行业特种承压类设备，须符合《固定式压力容器安全技术监察规程》TSG R0004的相关监管要求，结构设计阶段应按承压类容器标准完成强度校核[[S3,S17]]。

安全装置的配置要点[[S4,S7]]：

- 配套经校验合格的压力表、安全阀；
- 设置安全联锁系统：包内残留压力未完全释放时，禁止开盖及执行相关开型动作；
- 包体与包盖间须有可靠的密封结构，并设置防铁液喷溅的防护罩。

典型安全隐患包括：包盖密封失效引发铁液喷溅、密闭空间内镁蒸气集聚引发爆燃。防控措施要求配置足够排量的烟气除尘系统，所有操作人员须经专项安全培训考核合格后方可上岗[[S16,S17]]。

## 局限性与应用现状

**工艺局限性**[[S22,S17,S16]]：

- 仅适配以纯镁为球化剂的处理工艺；
- 安全管控难度大，对操作人员要求高；
- 经济性依赖一定产量规模，1t容量的GF转动包需每小时至少完成4~5次处理才能达到经济运行区间；
- 需配套专用的烟气除尘系统。

**应用现状**：随着稀土镁冲入法工艺因操作简便、安全性好而普及，国内新增压力球化包的应用占比持续降低。该工艺目前主要适配低硅要求的特种球墨铸铁牌号、大吨位关键铸件的稳定生产需求[[S22,S17]]。

## 应用场景

压力球化包生产的高稳定性低杂质铁水可适配高要求球墨铸铁件，典型应用包括[[S28,S29,S24,S12]]：

- 大型船用柴油机曲轴（如主轴颈φ220mm、总长4m、毛重约5t的大型球墨铸铁曲轴）；
- 风电行业的QT700-2高等级星形架大断面球墨铸铁件；
- 重载汽车领域的等温淬火球墨铸铁（ADI）后板簧支架、差速器、离合器压板等受力复杂核心结构件。

经典GF压力加镁转包的单包处理容量覆盖0.5~5t，截至1976年已在全球40余个国家工业应用，年处理合格球墨铸铁件超过40万吨[[S11,S16]]。
