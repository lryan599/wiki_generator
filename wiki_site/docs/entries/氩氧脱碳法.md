氩氧脱碳法（Argon Oxygen Decarburization，简称 AOD 法）是一种向钢液中吹入氩-氧混合气体进行氧化和搅拌熔池，以脱除钢液中的碳、气体和杂质的常压炉外精炼技术，主要用于生产含铬不锈钢、高强度钢等特种钢 [[S22,S27,S9]]。该工艺于1968年在美国开发，采用可倾转转炉形式的精炼容器，无需配套真空系统，设备容量覆盖1 t至60 t [[S21,S16]]。AOD法是目前全球铸钢和不锈钢生产领域应用最广泛的精炼工艺，约70%的不锈钢精炼炉采用AOD炉，美国不锈钢总产量的95%以上由AOD工艺生产 [[S21,S19]]。在铸钢行业，AOD也是应用最多的炉外精炼方法，其次是VOD法及其衍生工艺 [[S16,S2]]。

## 工艺原理

不锈钢冶炼的核心困难是“脱碳保铬”——将碳含量降至很低的水平同时尽量减少昂贵的铬氧化损失。在大气条件下，钢液中碳活度 $a_{\text{C}}$ 与铬活度 $a_{\text{Cr}}$ 及CO分压 $p_{\text{CO}}$ 之间存在近似比例关系 $a_{\text{C}} \propto a_{\text{Cr}} \cdot p_{\text{CO}}$ [[S6,S8,S35]]。常规电弧炉常压吹氧脱碳时 $p_{\text{CO}} \approx 1\,\text{atm}$，随着碳含量下降，铬的平衡含量急剧降低，大量铬被氧化烧损，通常可达2%～4%，需要大量补加昂贵的微碳铬铁 [[S1,S11]]。

AOD法通过吹入氩气稀释炉气中的CO，显著降低 $p_{\text{CO}}$，从而在热力学上使碳氧化反应 $\mathrm{[C]+[FeO] \to \{CO\}+[Fe]}$ 的吉布斯自由能比铬氧化反应 $2[\mathrm{Cr}]+3[\mathrm{FeO}] \to (\mathrm{Cr_2O_3})+3[\mathrm{Fe}]$ 更负，实现碳的优先氧化并最大限度抑制铬的氧化 [[S31,S6,S35]]。由于铬的氧化物为固相，$p_{\text{CO}}$ 的变化对铬氧化反应基本没有影响，吹氩只促进脱碳而不促进铬的氧化 [[S31]]。这一机理使得AOD法能够搭配廉价的高碳铬铁作为原料，在低含碳量下获得极高的铬收得率，大幅降低超低碳不锈钢的生产成本 [[S2,S11,S31]]。因无真空系统而靠氩气稀释降低CO分压，AOD法也被称为简化的真空精炼法 [[S16]]。经过AOD处理后，钢液中碳的质量分数可降至0.03%以下，氢和氮的体积分数可分别降至 $2\times10^{-4}\%$ 和 $80\times10^{-4}\%$ 以下 [[S4]]。

## 工艺流程

AOD属于典型的二次精炼（炉外精炼）工序，安装于电弧炉等初炼炉之后，是不锈钢二步法冶炼工艺中的核心脱碳保铬环节，全球约80%的不锈钢产能通过AOD工艺进行精炼 [[S19,S13,S20]]。典型的操作流程如下 [[S5,S4,S36,S32]]：

1. **电弧炉初炼**：将炉料熔化并使钢液温度达到不低于1560 ℃，然后倾注入AOD炉内。初炼钢液碳含量可达1%～3%，硅含量宜低于0.25%，以减轻对炉衬的侵蚀。
2. **分阶段吹气脱碳**：吹炼过程一般分为四个标准化阶段。
   - 第一阶段（高碳期）：氧氩比 $3\sim4:1$，以高强度吹氧脱碳为主。
   - 第二阶段：逐步降低氧氩比至约2，提高氩气比例以维持低CO分压，推动脱碳反应继续进行。
   - 第三阶段：进一步降低氧氩比，氩气流量超过氧气，将碳含量降至规格上限。
   - 第四阶段：停止供氧，只吹入纯氩进行高温碳脱氧，同时加入硅铁、铝等还原剂以及石灰造渣，将钢液升温至1700 ℃以上，使渣中的 $\mathrm{Cr_2O_3}$ 被还原回收铬。
3. **还原与脱硫**：在强碱性还原渣、高温及强烈氩气搅拌下，铬收得率可达99%，锰收得率约90%；硫的质量分数可降至0.01%以下，若要求更低可采用双渣法。
4. **出钢**：成分和温度合格后倾炉出钢。整个精炼作业时间通常在100 min左右，配置自动取样测温装置后可缩短至65 min左右。

## 设备结构

AOD精炼炉的炉体为类转炉结构，安装在支承托圈之上，由倾转机构实现前后各约160°的倾转，以满足装钢、吹炼、扒渣、取样、测温、出钢等操作要求 [[S26]]。炉体由三部分组成：

- **炉帽**：防止进钢时钢液流入氩氧吹入口，并阻挡吹炼过程中钢液与炉渣飞溅。
- **炉身**：熔炼主要空间。
- **炉底**：为倒锥形，与炉身夹角20°～25°，以避免气流直接冲刷侧壁炉衬。

熔池深度、熔池面直径、炉膛有效高度的比例大致为 $1:2:3$ [[S26]]。

![AOD精炼炉炉体剖面结构示意图，标注了炉帽、炉身、炉底、氩氧吹入口及关键尺寸比例](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/b0da15b4-d383-4f1a-b20d-c5957c3138ac/resource)

耐火内衬采用双层结构，靠近炉壳铺设约115 mm厚的隔热砖，内部砌筑300～400 mm厚的铬镁砖或镁白云石砖 [[S26]]。常规炉衬寿命为40～60炉次，优化后可提高至150～200炉次，通常需配备2～3个炉壳轮换使用以维持生产 [[S26,S36]]。

氩氧喷枪布置在炉底侧壁，采用套管结构：内管为纯铜管，通入氩氧混合气体；外管为不锈钢管，通入冷却用氩气 [[S26]]。喷枪数量随炉容变化：20 t以下配置2支，30～50 t配置3支，90 t以上配置5支。气体工作压力需控制在0.7845～1.373 MPa之间，以保证气流稳定喷入钢液深处并形成均匀搅拌流场 [[S26]]。

## 关键工艺参数

AOD精炼的核心操作参数如下表所示。

| 参数类别 | 控制要求或范围 | 说明 |
|---------|----------------|------|
| 阶段一氧氩比 | $3\sim4:1$ | 高碳期以脱碳为主 |
| 阶段二氧氩比 | 逐步降至约2 | 通过提高氩气比例降低CO分压 |
| 阶段三氧氩比 | Ar流量高于O₂ | 将碳降至规格上限 |
| 阶段四供气 | 纯Ar，停止供氧 | 高温碳脱氧、还原及合金化 |
| 可使用N₂替代Ar的比例 | 40%～70% （无低氮要求的钢种） | 不影响脱碳效果，降低运行成本 [[S26]] |
| 初炼入炉温度 | ≥1560 ℃ | 电弧炉出钢温度，吹炼过程中钢液温度不降乃至略升 [[S5]] |
| 还原期温度 | >1700 ℃ | 强化铬回收和脱硫 [[S36]] |
| 喷枪气体压力 | 0.7845～1.373 MPa | 保证稳定喷入和搅拌效果 [[S26]] |
| 炉渣碱度R（CaO/SiO₂） | 2～3.5 | 控制脱硫、脱磷能力，避免粘度过高 [[S14]] |
| 渣中FeO | ≤0.5 % | 保证还原效果 [[S14]] |
| 渣中SiO₂ | ≤2.0 % | 限制酸性氧化物 [[S14]] |
| 渣中S / P | S≤0.05 %，P≤0.02 % | 保证脱渣效果和钢液纯净度 [[S14]] |
| 完整吹炼周期 | $1\sim1.5$ h （约65～100 min） | 包含装钢、吹炼、还原、出钢 [[S32,S36]] |

## 应用领域

AOD法最初为生产超低碳不锈钢而开发，后来应用范围扩大到多种高合金钢和低合金钢 [[S4]]。在压铸行业，H13（4Cr5MoSiV1，相当于SKD61）等热作模具钢的生产中已引入AOD精炼：通过深度脱碳、控铬以及气体吹洗，可有效降低钢中的非金属夹杂物和气体含量，提升钢的纯净度，从而改善模具的热疲劳性能和高温强度 [[S7,S10]]。国内已有多家配备AOD炉外精炼设备的钢厂生产H13钢 [[S10]]。

除此之外，AOD还广泛用于生产高强度低合金铸钢，以及进行脱硫、脱氧、去除气体和杂质的综合精炼 [[S4]]。

## 优点与局限性

**主要优点：**

- 铬回收率高：在低含碳量下仍可保持高达99%左右的铬收得率 [[S11,S36]]。
- 设备简单、投资低：无需配置昂贵的真空系统，一次投资较少 [[S16,S11]]。
- 生产灵活：设备容量范围宽（1～60 t），可熔炼钢种多，适配铸钢件多品种、变批量的生产特点 [[S16,S11]]。
- 原料成本低：允许使用价格低廉的高碳铬铁和返回料，降低生产成本 [[S2,S11,S31]]。

**主要局限性：**

- 炉衬寿命短：高温吹炼对耐火材料侵蚀严重，常规炉衬仅40～60次，需频繁更换炉壳 [[S26,S36]]。
- 氩气消耗大：相同净化效果下，AOD的氩气用量约为VOD的10～80倍，运行成本高 [[S3]]。
- 常规冶炼周期较长，处理时间多在100 min左右，高于部分真空精炼工艺 [[S3,S36]]。

## 与其他脱碳工艺的对比

| 对比项目 | AOD （氩氧脱碳） | VOD （真空氧脱碳） | RH （真空循环脱气） |
|----------|-----------------|------------------|------------------|
| 脱碳极限 | 高于VOD，常规可至0.03%C以下 | 可降至0.01%C以下 | 略低于VOD |
| 气体去除能力 | 无真空支持，脱气能力弱于真空工艺 | 可将钢中氢降至 $1\times10^{-6}$ 以下 | 具有极强的循环脱气能力 |
| 氩气消耗 | 高，是VOD的10～80倍 | 低 | 远低于AOD |
| 设备成本 | 低，无真空系统 | 高，需配置真空罐和真空系统 | 高 |
| 生产灵活性 | 高，钢种和容量范围广 | 较低，受真空系统限制 | 较低 |
| 主要降CO分压方式 | 氩气稀释（常压） | 抽真空 | 抽真空 + 循环搅拌 |

（数据来源：[[S3,S13,S23]]）

## 工艺变体

- **AOD-L**：在传统侧吹AOD转炉基础上增设顶枪喷吹氧气或混合气体，可加快脱碳速度、缩短冶炼周期并提高生产能力 [[S19]]。
- **CLU法**：用氧气-蒸汽混合物替代部分氩气，同样利用稀释原理降低CO分压，达到 AOD 类似效果 [[S8,S35]]。
- **氮气替代**：对于无低氮含量要求的钢种，可用占总供气量40%～70%的氮气替换部分氩气，降低运行成本 [[S26]]。
- **AOD精炼过程的排气切换**：在吹炼间歇可自动将管路切换为压缩空气或氮气，以节省昂贵的氩气 [[S32]]。
