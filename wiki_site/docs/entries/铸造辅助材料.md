## 概述

铸造辅助材料是指在压铸生产全流程中，除主金属原料（铝、镁、锌、铜等压铸合金锭材）之外所消耗使用的各类非主原料类材料。这类材料不直接构成压铸件的基体主体，仅在工艺过程中发挥辅助作用，绝大多数品类在生产过程中随烟气、废渣带出压铸系统，不成为压铸件最终成分的主要部分[[S40,S50]]。辅助材料涵盖从金属熔炼处理到模具喷涂、压射润滑等各道工序，是保障生产过程顺利进行、提升产品质量与设备寿命的关键要素。压铸车间通常需单独设置辅助材料存放区域以保障供应[[S50]]。

国内铸造行业标准体系已将“压铸涂料”明确列为独立的专业术语，归入压铸工艺辅助材料类目[[S27]]。压铸生产中归类使用的辅助材料包括熔剂、覆盖剂、精炼剂、变质剂、除杂剂、脱模剂、冲头润滑剂、各类压铸涂料等，其来源分为专业厂商按标准合规供应和压铸厂自行配制两类[[S39,S25]]。

## 分类体系

压铸辅助材料按工艺环节与功能可分为以下三大主流类别：

| 大类 | 核心品类 | 主要功能 |
|------|---------|---------|
| 模具型腔与表面防护类 | 压铸脱模剂（离型剂）、模具涂料 | 形成隔离膜，防止粘模，降低脱模阻力 |
| 压射系统润滑类 | 冲头润滑剂 | 降低冲头-压室摩擦，防止卡死与热侵蚀 |
| 金属熔炼处理类 | 精炼除气剂、清渣剂、覆盖剂、变质剂 | 熔体净化除气、排渣隔绝氧化、晶粒细化 |

此外，还包括用于模具防锈的防护油、模板防护油脂、浇斗浇道用耐高温保护层材料以及液压工作液等辅助品类[[S17,S15]]。

### 模具型腔与表面防护类

**压铸脱模剂（离型剂）** 是压铸生产中最核心的辅助材料之一。其核心功能为喷涂后在模具表面形成均匀离型膜，使成型压铸件顺利脱模，同时可有效防止粘模，提升压铸件表面质量，细化表层晶粒，增加细晶层厚度，减少微气孔，提升压铸件气密性[[S42]]。

脱模剂可按活性成分与使用形态进行多维分类[[S33,S2]]：

| 分类维度 | 类型 |
|---------|------|
| 按活性成分 | 硅系列、蜡系列、氟系列、表面活性剂系列、无机粉末系列、聚醚系列 |
| 按使用形态 | 溶剂型、水性、无溶剂型、粉末型、膏状 |
| 按基体类型 | 油基脱模剂、粉基脱模剂、水基脱模剂 |

当前行业主流应用为水基脱模剂，其主要成分为硅油、合成蜡、添加剂、合成烃、乳化剂、水[[S4,S49]]。水基脱模剂按成分进一步分为三类：以蜡系材料为主体乳化制备、以含硅有机物为主体、同时包含蜡和含硅组分的复合类型[[S44]]。

**压铸模具表面涂料** 的核心功能为在高温金属液与模具钢（通常为H13）之间形成隔离保护层，降低金属液对型腔的直接冲刷热冲击，减少粘模概率，降低铸件脱模阻力，同时对滑块、顶杆等模具运动副起到减摩作用，延长模具整体使用寿命[[S44,S8]]。

压铸涂料行业通用要求为：在100~150℃工况下稀释剂可快速挥发不额外增加型腔内气体；涂覆性好不堆积；高温下可形成均匀薄膜层；无腐蚀性；低污染[[S33]]。

### 压射系统润滑类

**冲头润滑剂** 仅用于冷室压铸机场景——热室压铸机冲头完全浸没在低熔点金属液中，无需额外润滑。其核心功能是在压射冲头与压室接触表面形成耐高温抗腐蚀润滑薄膜，降低高压力、高冲速工况下的摩擦阻力，避免冲头卡死和过度磨损，保障压射系统的输出效率[[S53]]。

商用冲头润滑剂按类型可分为三类[[S53]]：

| 类型 | 特点 |
|------|------|
| 传统高石墨含量油脂类 | 石墨熔点约1650℃，成本低，润滑效果优异，但易造成发黑污染 |
| 无石墨型 | 添加氮化硼等白色固体润滑剂，清洁度更高 |
| 水基型 | 采用喷雾方式施涂，可降低铸件黑斑和气孔缺陷，减少现场烟气排放 |

此外，冷室压铸机配套冲头润滑剂还可分为液态油剂和固体润滑珠两类[[S24]]。

### 金属熔炼处理类

金属熔炼处理类辅助材料覆盖精炼除气剂、清渣剂、覆盖剂、变质剂等产品，分别承担金属熔体净化除气、排渣隔绝氧化、晶粒细化功能。压铸合金熔炼用熔剂涵盖覆盖剂、造渣剂、精炼熔剂、变质剂等不同类型，分别对应铝合金、镁合金、铜合金等不同压铸合金的熔炼场景[[S45,S7]]。

**覆盖剂** 的核心功能是在高温金属液表面形成连续隔离层，隔绝空气中的水和氧气，避免活泼合金元素氧化烧损，同时起到保温作用[[S12,S36]]。三类经典铝合金覆盖剂成分与适配场景[[S12]]：

| 类型 | 成分 | 适配场景 |
|------|------|---------|
| 三号覆盖剂 | 50%KCl+50%NaCl | 非高镁铝合金常规熔炼，成本低廉 |
| 一号覆盖剂 | KCl-NaCl体系添加冰晶石 | 绝大多数普通铝合金，兼具覆盖和轻度精炼功能 |
| 二号覆盖剂 | KCl、MgCl₂、BaCl₂复配 | 高镁铝合金熔炼，可将镁烧损率从0.1%降至0.05%以下 |

**精炼除气剂** 主流商用产品为氯盐类，常用类型包含氯化锌(ZnCl₂)、氯化锰(MnCl₂)、六氯乙烷(C₂Cl₆)，对应推荐添加量分别为炉料总重0.15%~0.2%、0.1%~0.2%、0.2%，使用前需经过充分烘干除水处理[[S10]]。经典六氯乙烷基精炼剂可通过添加不同辅料适配不同铝合金系列：添加氟硅酸钠适配Al-Si系合金精炼，添加二氧化钛适配Al-Cu系合金精炼，常规添加量为炉料总重量的0.2%~0.4%[[S19]]。

商用铝合金精炼剂常规基础成分为氯化钠(NaCl)、氯化钾(KCl)、冰晶石(Na₃AlF₆)等无机盐，部分配方还添加硝酸钠、碳、耐火材料组分，按形态可分为粉末状、颗粒状和压块状三类，常规使用温度区间为690~720℃[[S30]]。

## 性能要求与使用工艺

### 脱模剂/模具涂料

**推荐性能指标**[[S8,S33]]：
- 100~150℃区间稀释剂可快速完全挥发，不额外增加型腔内气体含量
- 成膜均匀无局部堆积积垢
- 高温工况下无有毒有害分解产物析出
- 对模具钢和压铸合金材料均无腐蚀作用
- 长时间储存不分层，性能稳定性好

**喷涂工艺参数**（水基脱模剂铝合金压铸场景）[[S5,S24,S16]]：

| 参数项 | 推荐范围 |
|--------|---------|
| 模具表面温度 | 150~210℃ |
| 喷涂距离 | 100~200mm |
| 单次喷涂时长 | 0.1~0.2s |
| 常规稀释倍率 | 40~300倍（优质脱模剂可达1:150~200） |
| 喷涂供气系统工作压力 | 最大至0.2MPa |
| 单批次成膜覆盖模次 | 8~10模 |

水基脱模剂推荐使用工况下模具温度区间为200~300℃，常规稀释倍率可达1:150~200[[S28,S11]]。模具表面涂料的推荐施涂温度为180~280℃：温度过高导致涂料瞬间汽化反弹无法有效附着，温度过低导致水分无法完全蒸发，易诱发铸件气孔缺陷[[S8]]。

**商业化产品案例**：耐高温型脱模剂Safe-Lube A在350℃条件下成膜附着量为Safe-Lube B的2.15倍，高模温工况下成膜能力优异，可有效降低粘模风险，同时具备优秀的模具冷却能力，可将喷涂时间从常规20s缩短至17s[[S21,S1]]。

### 冲头润滑剂

冲头润滑剂的推荐性能指标要求[[S24]]：
- 300~500℃高温工况下仍可保持稳定低摩擦润滑效果
- 与金属表面结合力强，不易脱落流失
- 对钢铁和压铸合金均无腐蚀作用
- 燃烧产物无有害烟气，不会造成铸件气孔缺陷
- 常温储存和使用过程中不分解变质，性能一致性好

使用方式上，推荐以压铸机动作信号触发的自动加注方式为主，需将润滑材料尽可能推送至压射套内部最高摩擦区域，严格控制最低加注量，避免多余润滑剂残留污染合金或影响成膜质量[[S24,S16,S23]]。

### 覆盖剂与精炼剂

铝合金覆盖剂的推荐性能指标[[S12,S51]]：
- 熔点低于铝合金常规熔炼温度
- 密度小于铝合金液密度，避免下沉混入熔体
- 覆盖铺展性好，可自动在液面形成完整连续保护层
- 分离性佳，扒渣时可与金属液完全分离
- 低吸湿性，长期存储不易吸水变质
- 低有害烟气排放

铝合金压铸用除气剂配套旋转叶轮除气工艺，需将铝液温度控制在适配区间，保证搅拌处理后合金液氢含量降至0.08mL/100g以下[[S29]]。

## 对压铸件质量与模具寿命的影响

### 正面影响

规范适配的压铸润滑剂（脱模剂+冲头润滑剂）可在模具型腔、压射部件表面形成稳定隔热润滑薄膜，既隔绝高温液态金属直接冲刷熔蚀模具钢基体，又可降低模具表面冷热交变的温度梯度。实测显示水基润滑剂带来的热冲击贡献占比仅不到压铸铝合金热冲击的2%，可有效延缓模具热疲劳裂纹萌生，大幅延长压铸模具服役寿命[[S20,S26,S14,S3,S22]]。

在铸件质量方面，合理使用脱模剂可实现：防止粘模、提高压铸件表面质量、细化晶粒与增加细晶层厚度、减少微气孔、提升压铸件气密性、提高生产效率[[S42]]。

### 使用不当引发的缺陷

脱模剂使用不当会诱发多类典型铸件缺陷[[S5,S16,S31,S46,S32]]：

| 不当操作 | 引发缺陷 |
|---------|---------|
| 浓度过低、喷涂量不足 | 无法形成完整隔离润滑膜，直接引发粘模，导致铸件表面拉伤 |
| 浓度过高、喷涂过量 | 膜层过厚堆积，残留有机物分解产气，引发气孔、表面斑纹、夹渣缺陷 |
| 过量水分残留 | 导致铸件欠铸、冷隔 |

冲头润滑剂加注过量会使大量可分解含碳物质进入压室，随合金液填充型腔后生成大量游离气体，大幅提升铸件内部卷气、气孔缺陷占比，同时污染合金基体性能[[S16,S31]]。冲头润滑剂选型不当或润滑缺失时，压射冲头与压射套之间干摩擦加剧，会快速引发部件表面磨损、拉伤，大幅降低冲头及压室的使用寿命[[S23,S41]]。

## 质量管理与标准规范

压铸生产的质量管控规范明确要求：所有压铸辅助材料（脱模剂、涂料、熔剂、添加剂等）的工艺参数必须纳入生产过程质量控制范围；辅助材料入厂需提供质量保证单，必要时需经过复验确认合格后方可投入使用；对应生产记录需留存可追溯[[S34]]。
