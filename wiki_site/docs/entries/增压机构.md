## 概述

增压机构（别称增压器、压力倍增器，对应英文术语 pressure intensifier、pressure multiplier）是冷室压铸机三级压射系统的核心功能单元，热室压铸机一般不配置该机构[[S6,S12,S10,S5]]。其作用是在金属液充满模具型腔后、铸件尚未完全凝固前，于极短时间内快速建立高压，对已填充完全的金属液施加压实压力[[S20,S37,S29]]。

## 定义

根据 GB/T 21269-2023《冷室压铸机》标准（条款 3.16），"增压"是指利用活塞面积差或更高压力的蓄能器，使产生的压射缸内流体压力高于系统压力[[S36]]。增压机构即为实现该增压功能的液压执行装置。

## 工艺作用

增压机构的核心功能在于充型结束后对半凝固状态的金属液施加高压，产生以下效果[[S20,S37,S29]]：

- **压缩内部气孔**：高压使铸件内部裹入的气体体积被压缩，降低孔隙率。
- **补缩消除缺陷**：金属液在凝固过程中产生体积收缩，增压压力推动压射冲头将余料中的金属液持续补入型腔，消除缩松、缩孔。
- **提升力学性能**：致密度提高带来更高的抗拉强度与延伸率，同时细化晶粒组织。

对厚壁复杂铸件，增压的压实与补缩效果尤为显著[[S20,S29]]。

## 分类

压铸机的压射机构分为**无增压器**和**有增压器**两大类，有增压器类型下可进一步按接通方式细分[[S25,S4]]。

| 分类维度 | 类别 | 说明 |
|--------|------|------|
| 有无增压功能 | 无增压器 | 普通压铸机，建压时间 200～300 ms，增压效果极弱[[S25]] |
|  | 有增压器 | 现代冷室压铸机主流配置[[S25]] |
| 按接通控制方式 | 按背压接通 | 结构简单，但可调性差，动作滞后时间较长[[S25,S3]] |
|  | 按行程接通 | 由行程开关发信，调节简便，但余料长度波动时触发稳定性受影响[[S25,S3]] |
|  | 按时间继电器接通 | 与行程接通方式无原则性区别[[S25]] |

从系统结构类型划分，主流的增压系统可分为**预充液式增压器**与**液压增压器两大类**。液压增压器的增压比（输出压力与输入系统压力的比值）常规范围为 1.6:1～4:1，部分设计可达 4:1～9:1，倍率由增压活塞两端的作用面积比决定[[S12]]。

从控制回路角度，国产卧式冷室压铸机多采用**双回路无背压增压器**，其动作滞后时间短、结构简单、调整方便[[S3]]。

## 结构与组成

带增压功能的压射系统相对于无增压系统，额外集成了独立的增压缸部件[[S28]]。增压机构的核心零部件包括[[S31,S26,S34]]：

| 核心零部件 | 功能 |
|-----------|------|
| 增压活塞 | 大小端直径差直接决定增压倍率，大端通低压油、小端作用于封闭油腔实现压力放大 |
| 增压缸缸体 | 为高压油腔提供承压载体 |
| 增压专用蓄能器 | 为增压动作提供独立的高压油源 |
| 液压控制元件组（换向阀、单向阀、节流阀） | 控制增压触发时机和升压速率 |
| 密封组件 | 保证高压油腔无泄漏，维持压力稳定 |

增压机构在压铸机上的安装布局有**独立式**与**内置式**两类：独立式增压器安装在压射缸旁侧，内置式（装入式）增压器集成嵌入压射缸内部，二者均与压射蓄能器液压管路连通，直接作用于压射活塞后端，最终驱动压射冲头完成增压补压动作[[S21,S12,S24]]。

## 工作原理

### 压力放大原理

液压增压机构基于帕斯卡原理工作：增压活塞大端通入较低压力的液压油，小端作用于封闭油腔，通过作用面积差将输入的液压能转换为高倍率的高压液压能，进而提升压射缸内的油液压力，增大压射冲头作用在金属液上的比压[[S12,S26]]。

增压作用力的理论计算公式为[[S20,S27,S31,S26]]：

$$P_{\text{增}} = P_{\text{蓄}} \cdot \frac{\pi D^2}{4} \cdot \left(\frac{D_1}{d_1}\right)^2$$

式中：\(P_{\text{增}}\) 为增压力；\(P_{\text{蓄}}\) 为蓄能器工作压力；\(D\) 为压射缸直径；\(D_1\) 为增压缸大端直径；\(d_1\) 为增压活塞小端直径。

作用在金属液上的增压比压（\(p\)）为[[S20,S26,S27]]：

$$p = P_{\text{蓄}} \cdot \left(\frac{D}{d}\right)^2 \cdot \left(\frac{D_1}{d_1}\right)^2$$

式中：\(d\) 为压射冲头直径。由此式可知，压射比压与压射缸直径及增压缸直径之比的平方成正比，与压射冲头直径及增压活塞小端直径之比的平方成反比[[S26,S27]]。

### 压铸循环中的动作时序

在全压射周期中，压射比压随时间变化可分为四个阶段[[S32]]：**Ⅰ 慢速封孔 → Ⅱ 充填 → Ⅲ 增压 → Ⅳ 保压**。

一个完整压铸循环中，增压机构与其他部件的联动动作逻辑如下[[S36,S16,S22]]：

1. **慢压射阶段**：压射冲头低速推进，排出压室空气。
2. **快压射阶段**：压射蓄能器释放压力油，驱动压射冲头高速将金属液填充型腔。
3. **充型结束与触发**：型腔充满后压射冲头运动停止，系统压力上升触发增压控制单元（行程开关或压力检测元件），关闭压射回路止回阀，将压射缸前腔封闭。
4. **增压建压**：增压蓄能器释放压力油推动增压活塞快速移动，将压射缸内油压放大至设定值，作用于尚未完全凝固的型腔金属液。
5. **保压与复位**：保压完成后增压回路泄压，增压器复位，压射冲头随开模动作退回原位。

增压触发方式主要有三种[[S25]]：
- 基于**压力反馈**触发——充型后压射系统背压上升到设定值自动启动增压；
- 基于**行程信号**触发——压射活塞运行到预设位置时由行程开关发信启动增压；
- 基于**时间继电器延时**触发——在快压射启动后经预设固定延时启动增压。

## 关键工艺参数

| 参数 | 定义/说明 | 典型范围/要求 |
|------|----------|-------------|
| 增压比 | 增压器输出压力与输入系统压力的比值，由活塞面积比决定，为固定结构参数，不可任意调整[[S12,S10]] | 1.6:1～4:1（常规），部分 4:1～9:1[[S12]] |
| 增压比压 | 增压时作用在金属液上的压强[[S31,S2]] | 30～200 MPa[[S2,S18,S37]]，低于 30 MPa 缩松缺陷率高，高于 200 MPa 模具磨损加剧、易溢料飞边[[S2,S18,S37]] |
| 建压时间 | 从增压触发到达到增压压力稳态值 90% 的时间间隔[[S36]] | 应 < 0.03 s[[S25,S31,S17]]；优质压铸机控制在 20～40 ms[[S30,S29]] |
| 增压延时 | 充型结束到增压开始的时间间隔，可调参数，需保证内浇口未完全凝固前建压[[S12,S25]] | 通常由现场试模确定|

无增压结构的压铸机建压时间普遍在 200～300 ms，此时铸件浇口及薄壁部位已完全凝固，增压压力无法传递至内部厚壁热节区域[[S25]]。

## 局限性及常见故障

### 固有局限性

带行程触发式增压的压铸机，若生产过程中余料长度出现波动，易出现增压触发时间提前或滞后，无法保证增压作用的稳定性[[S25]]。

### 常见故障及对策

| 常见故障 | 成因 | 解决对策 |
|---------|------|---------|
| 动作响应滞后 | 控制阀组卡滞、增压活塞运动阻力过大、预充压力不足 | 定期清洗增压控制阀组、校验蓄能器预充氮气压力、优化增压触发位置参数[[S25,S29,S35]] |
| 高压油腔泄漏 | 密封垫圈磨损、逆止阀卡滞闭合不严 | 定期检测密封件并更换、拆解研磨逆止阀密封面[[S15,S9]] |
| 液压油温升过快 | 溢流阀压力设置过高、节流部件长期高损耗 | 校准系统工作压力至额定值、优化增压流量控制阀开度[[S15,S11]] |
| 增压压力保持不稳定 | 蓄能器氮气泄漏预充压力不足、控制油路存在进气 | 定期检测蓄能器氮气压力并补充、对增压控制回路排气[[S7,S35]] |

## 应用场景

以下压铸件必须配置增压机构[[S1,S13,S18,S14]]：

| 铸件特征 | 典型产品示例 | 增压必要性 |
|---------|------------|-----------|
| 高气密性要求 | 液压阀体、汽车缸体缸盖、水冷壳体内腔类零件 | 无增压时内部孔隙率过高，承压工况下发生泄漏[[S1,S13]] |
| 厚壁/厚大节点结构 | 含热节集中区域的复杂压铸件 | 普通压铸无法补缩消除集中缩孔，增压压力将余料金属推入凝固收缩区[[S1,S2]] |
| 高力学性能承力结构件 | 汽车底盘结构件、航空航天承力压铸构件 | 增压使晶粒细化、致密度提升，获得更高抗拉强度与延伸率[[S18,S14]] |
