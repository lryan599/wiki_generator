## 术语来源与消歧

“铸坑”是铸造行业中的一个多义术语。根据《英汉铸造词汇》，“pit”在铸造领域对应三重中文释义：（1）地坑；（2）铸坑、锭坑；（3）铸件表面的凹坑[[S1,S8]]。这明确了该术语在铸造行业存在两大核心义项，分别指向**铸件表面凹坑类缺陷**和**铸造车间地坑设备**。

在压铸领域行业文献、标准与技术手册中，“铸坑”指代铸件表面凹坑类缺陷的出现频次占比约65%，指代铸造地坑设备的出现频次占比约35%，缺陷类义项是压铸领域的主流引用场景[[S1,S24,S11]]。相应的英文术语对应关系为：缺陷类“铸坑”等效英文术语 **casting pit / pit defect**；设备类“铸坑”等效英文术语 **foundry ground pit / casting pit equipment**[[S1,S16]]。

## 缺陷类铸坑

### 定义与特征

缺陷类铸坑是压铸及铸造铸件表面形成的局部开放点状或不规则凹坑类表面缺陷，属于表面缺陷大类[[S1,S9]]。在压铸领域中，铸坑指铸件成型后表面出现的局部非贯穿性凹陷类形貌，也称缩凹、缩陷、表面麻点[[S1]]。

压铸工艺手册中将该类缺陷记为“凹陷（收缩）”，描述为：在铸件厚度大部分的表面上有平滑的凹坑，形状如盘、碟一样[[S27]]。行业文献中亦明确其特征为“铸件平滑表面上出现凹陷部位”[[S14]]。

### 与近似缺陷的区分

铸坑与缩陷、缩孔、砂眼、气孔等缺陷存在明显差异，需要严格区分：

| 缺陷类型 | 定义特征 | 与铸坑的关键区别 |
|---------|---------|----------------|
| **铸坑** | 完全开放于铸件外表面的纯凹陷类缺陷，无深层连通的封闭孔洞结构 | —（参照基准） |
| **缩陷 (depression / sink mark)** | 铸件厚断面或断面交接处上平面的塌陷现象，其下方常伴随缩孔，有时也出现在内缩孔附近的表面[[S11,S13,S17]] | 缩陷是大面积局部塌陷，伴随内部缩孔；铸坑是独立表面小凹坑，无深层连接的缩孔结构 |
| **缩孔** | 铸件内部封闭的孔洞，不直接暴露在表面 | 缩孔封闭于铸件内部，铸坑完全开放于表面[[S24,S3]] |
| **砂眼** | 铸件表面/内部由砂粒嵌入脱落后形成的带渣层的孔洞 | 砂眼带渣层，铸坑为纯凹陷无渣层附着[[S24,S3]] |
| **气孔** | 金属液中气体析出形成的内壁光滑的孔洞 | 气孔内壁光滑且多为封闭孔，铸坑为开放表面凹陷[[S24,S3]] |
| **crater (弧坑/弹坑)** | 因表面张力差，液体从低表面张力点向外流动形成的边缘隆起、中心薄的环形凹坑[[S4,S5]] | crater具有边缘隆起的环形形貌，铸坑为无边缘隆起的常规凹坑 |

### 分类与典型参数

按成因，铸坑可分为三类[[S6,S22]]：

| 类别 | 形貌特征 | 典型参数 | 分布规律 |
|-----|---------|---------|---------|
| **微小型麻点类铸坑** | 分散分布的半球形凹坑 | 直径 0.3~0.8 mm，深度 0.3~0.5 mm | 大件及大平面铸件尤为严重，密集状分布[[S6]] |
| **厚大区域收缩型铸坑** | 凹坑形貌平滑，呈碟形 | 常规压铸铝合金件凹坑直径多在 2~5 mm 区间 | 分布于铸件热节、厚壁位置[[S6]] |
| **模具复制型铸坑** | 凹坑轮廓与型腔表面损伤完全重合 | 尺寸与模具表面凹坑一一对应 | 固定出现在铸件同一位置[[S22]] |

### 形成机理

**1. 收缩型铸坑**

收缩型铸坑由金属液凝固过程的体积收缩引发。当铸件存在局部厚大截面、热节区域补缩通道受阻时，表层金属在未完全凝固阶段随内部金属体积收缩向内部牵拉，最终在对应位置形成平滑凹陷[[S27,S2]]。

![压铸厚截面铸件缩孔形成过程示意图，展示从金属液充入到冲头压力下凝固的两个阶段](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/20780f24-1c64-42ab-90ac-a3e17a0790ef/resource)

> **图注**：压铸厚截面位置铸坑形成的两个阶段——(a)刚压入的液态金属状态；(b)冲头压力下凝固阶段，厚大区域因补缩不足形成表面凹陷。[[S2]]

压铸中凹陷产生的直接原因包括：合金本身收缩性大；铸件结构设计不当，壁厚差太大；内浇口位置不当；压射比压低，凝固补缩不足[[S27]]。

**2. 憋气型铸坑**

憋气型铸坑由充型过程型腔局部排气不畅引发。部分区域的残留气体无法通过排气系统排出，被高速流动的金属液包裹压缩在型壁与铸件表层之间，在该区域形成气垫阻碍金属液完全填充型壁表面，最终冷却后形成表面凹坑[[S14]]。根本原因是“由模具高温引起型腔气体排不出，被压缩在型腔表面与金属液界面之间”[[S14]]。

**3. 冲蚀型凹坑**

金属液高速冲刷模具表面可产生冲蚀型凹坑。研究表明，冲蚀坑深度随冲刷角度提升而增大，冲蚀坑长度随冲刷角度增大而减小，冲蚀坑宽度随角度先升后降[[S18]]。

> **图注**：冲蚀坑长度、宽度、深度随冲刷角度的变化规律对比（水模拟 vs. 数值模拟），为铸坑形成机理中外力冲刷作用研究提供定量参考。[[S18]]

**4. 服役环境的腐蚀扩展**

铸坑在服役环境下（尤其铝合金铸件）的扩展遵循电化学腐蚀自催化机理：氯离子渗透破坏表面钝化膜，裸露的铝基体区域形成阳极发生溶解，周围完好氧化膜区域作为阴极发生吸氧反应，蚀坑内部逐步浓缩生成酸性溶液，推动凹坑持续向基体纵深发展[[S20]]。

流体湍流亦可诱发铸坑：湍流冲击铸件表面时破坏原有氧化膜，裸露出的金属区域为阳极，周围未破损氧化膜覆盖区域为阴极，局部形成电偶电池引发选择性溶解，最终发展为深谷状蚀坑[[S25]]。

### 核心影响因素

铸坑的产生受多维度因素综合影响：

| 维度 | 影响因素 | 作用机制 |
|-----|---------|---------|
| **浇注温度** | 浇注温度过高 | 增大合金整体液态收缩率，提升铸坑生成概率[[S27,S14]] |
| | 浇注温度过低 | 金属液充型前沿提前凝固，局部补缩不足形成表层凹坑[[S27,S14]] |
| **模具温度** | 模具预热温度过高 | 延缓厚大区域冷却速度，加大体积收缩差值诱发表面凹陷[[S27,S14]] |
| | 模具局部过热 | 过热部分凝固慢，凹陷多产生在厚壁处[[S14]] |
| **压力参数** | 压射比压不足 | 无法在金属液完全凝固阶段提供足够补缩压力，厚大热节区域收缩无法得到补充，直接形成铸坑[[S14]] |
| | 增压触发位置滞后 | 补缩压力作用窗口错过最佳补缩区间，提升铸坑出现概率[[S14]] |
| **合金成分** | 合金凝固收缩率过高 | 铸坑生成倾向性越高[[S27]] |
| | 回炉料占比 ≥ 50% | 熔体除渣除气难度大幅提升，杂质与析出气体易在表层聚集形成麻点凹坑[[S27]] |
| | Cu元素含量过低 | 降低铸件表面硬度，受外力作用易出现压痕类凹坑[[S27]] |

### 检测方法

不同检测方法的适用范围与检出能力各异[[S19,S10]]：

| 检测方法 | 检出能力 | 适用场景 | 特点 |
|---------|---------|---------|------|
| **目视检测** | 可观察尺寸 ≥ 1 mm 的铸坑；配合5倍以上放大镜可识别更小凹坑 | 在线全检快速筛查 | 无额外检测成本，可直观看到缩凹等缺陷[[S19]] |
| **渗透探伤** | 荧光渗透与着色渗透两类，可检出宽度低至 0.01 mm 的表面开口型微小型铸坑 | 非多孔质铸件表面开口缺陷检测 | 成本低、操作便捷[[S19]] |
| **机器视觉影像检测** | 结合图像阈值分割算法，自动识别表面分散凹坑并完成尺寸、数量统计 | 批量生产场景下的智能在线检测 | 可清晰识别脆断凹坑、斑点等表面缺陷[[S10]] |

### 预防与纠正措施

**工艺调整类**[[S27,S14]]：

- 适当降低浇注温度与模具整体温度，针对模具局部过热区域增设点冷系统强制冷却；
- 提升压射比压，调整增压触发时机使其在充型完成后立即作用，保证补缩压力可传递到热节区域；
- 减少涂料喷涂量，喷涂完成后用压缩空气将型腔表面涂料吹匀，选用低发气量的压铸涂料；
- 优化充型速度避免金属液产生剧烈卷气湍流。

**模具改进类**[[S27,S14,S23]]：

- 优化铸件产品结构设计，消除局部孤立热节，保证壁厚均匀、厚薄截面过渡平缓；
- 合理增大内浇口截面积，优化浇口导入位置保证补缩通道畅通；
- 在充型末端、热节附近增设足够容量的溢流槽与排气通道，必要时加装通气塞提升排气能力；
- 定期对压铸模型腔表面进行打磨抛光，型腔出现局部凹坑损伤后及时采用氩弧焊补焊修复；
- 每约 30000 模次对型腔进行氮化处理恢复表面硬度[[S23]]。

## 设备类铸坑

### 定义与用途

设备类“铸坑”指铸造车间设置的地下型专用设施，可用于地坑造型、大型铸件浇注、立式半连续铸造设备安装、高温铸件安全防爆存放等场景，配套有通气、排水、防护结构[[S21,S26,S7]]。

立式半连续铸造中通常采用地坑式布置，铸锭以竖直方向拉出[[S7]]。大型实型铸造铸件多为单件小批量生产，铸造工艺多采用地坑造型，该方法省去制作砂箱的费用和周期，更便于工人操作[[S15]]。

### 地坑尺寸参数

《国外铸锻件制造新技术资料汇编》收录有铸坑尺寸参数表，包含号数、宽度（m）、长度（m）、深度（m）等数据[[S22]]。

消失模铸造地坑造型工艺中，不同重量等级铸件对应的吃砂量参数如下[[S21,S15]]：

| 铸件重量等级 | 侧部吃砂量（mm） | 底部吃砂量（mm） | 备注 |
|------------|---------------|---------------|------|
| 10~30 t | 150~200 | 150~200 | — |
| 30~50 t | 200~250 | 200~250 | 侧部需加铁板支撑加固[[S15]] |
| 50~80 t | 300~350 | 250~350 | 侧部需加铁板支撑加固[[S15]] |

地坑造型工艺底部通气排气问题至关重要，一般底部铺放干碎焦炭通气，通气管将浇注过程中产生的气体顺畅引出[[S21]]。
