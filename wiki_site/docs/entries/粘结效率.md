## 概述

「粘结效率」在国内现行压铸类国家标准GB/T 8847-2003《压铸模术语》、行业标准JB/T系列压铸相关规范中均未被作为正式标准化术语收录[[S39,S31]]。该术语属于压铸工程领域行业通用俗称，是衡量压铸过程中模具表面、脱模剂与熔融合金之间界面抗粘附能力的核心综合指标，直接表征发生粘模的难易程度[[S38]]。其本质作用机理在于：通过在合金液与模具之间形成稳定隔离保护层，避免熔融金属直接冲蚀模面，从而阻止合金与型腔发生化学/冶金粘结[[S26,S25]]。

## 表征方法与评判体系

粘结效率在行业中形成了三种主流量化表征路径：

| 表征方法 | 原理 | 评判准则 |
|---------|------|---------|
| 粘模面积比法 | 以指定压铸循环次数后，型腔表面合金粘附区域总面积占整个型腔接触表面积的百分比计算 | 比值越低，粘结效率越高 [[S41,S15]] |
| 脱模力比值法 | 在完全相同工艺参数与模具条件下，测得无粘模正常状态下的铸件脱模力与试样发生粘模时的瞬时脱模力的比值 | 比值越接近1，粘结效率越好 [[S35,S27]] |
| 首次粘模循环次数法 | 从第一次压铸循环起算，直到型腔表面首次出现可见合金粘附层时的累计生产循环数 | 循环次数越大，粘结效率越高 [[S46,S44]] |

此外，国际压铸领域ASTM压铸合金标准附录中提供了1~5级抗粘模性能（anti-soldering）评级体系，评级1代表最优，评级5代表最差。该评级判定基准基于合金名义铁元素含量约为1%的工况下，合金液无粘附在模具表面的能力来赋值[[S5]]。

## 应用场景与分类

### 脱模剂与涂层评估

粘结效率指标最核心的应用场景为压铸脱模剂性能评估：用于量化不同脱模剂在不同温度下在模具表面形成的隔离膜阻止合金与型腔发生冶金粘结的能力。同时，该指标也可用于表征模具表面改性涂层、模具表面强化工艺的抗粘模能力[[S16,S6]]。

### 冷室与热室压铸差异

冷室压铸场景下粘结效率指标的管控优先级显著高于热室压铸。冷室压铸主要面向铝合金、镁合金等高熔点、冶金活性更强的合金，合金液温度更高、压射速度更快，发生焊合粘模的风险远高于热室场景；热室压铸多用于锌合金等低熔点合金，常规工况下粘模概率较低，粘结效率指标通常仅用于高端半永久热室模具涂层的性能校验[[S30,S14]]。

## 评测方法

### 实验室成膜附着量法

国内压铸行业已推广的评测方法之一：在250℃与350℃两个典型模温点下，将浓度稀释至2%的脱模剂喷涂到200mm×200mm不锈钢测试板上，控制喷涂距离150mm、喷涂量3mL、喷涂时间1.0s，以电子秤称量成膜附着量。350℃下附着量高的试样，其粘结效率更优[[S21]]。

### 综合性能测试组合

铝合金压铸用水基脱模剂粘结效率实验室常规测试组合方法包括[[S32,S3]]：

| 测试项目 | 技术要求 | 与粘结效率的关联 |
|---------|---------|----------------|
| 黏度测试 | 合格范围 1×10⁻³~10×10⁻³ Pa·s | 影响成膜均匀性 |
| 粒径测试 | 平均粒径宜低于220nm，PDI小于0.15 | 影响涂层均匀性与稳定性 |
| 热重测试 | 220℃与570℃两温度点下残炭剩余量高者更优 | 直接反映高温成膜与隔离能力 |
| 接触角与表面张力测试 | — | 表征润湿铺展能力 |
| 稳定性测试 | 室温3个月静置无分层；2000r/min搅拌30min无油水分离 | 影响长期使用可靠性 |

### 抗焊合加速试验

国际公认的压铸抗焊合加速试验方法：将直径19mm的H13钢测试销（硬度HRC46~48，表面粗糙度45μm）以最高67Hz（4000r/min）的转速浸入恒温±5℃的熔融铝合金浴中，模拟实际压铸时型壁受到的高速紊流冲刷工况，统计不同循环次数后测试销表面的粘铝质量来量化粘结效率[[S38]]。

## 关键影响因素

粘结效率受多重因素耦合影响，主要因素如下表所示：

| 影响因素 | 影响规律 |
|---------|---------|
| 脱模剂种类与稀释比 | 优质耐高温水基脱模剂稀释比可达1:150~200；纳米改性脱模剂在350℃下的附着量可达常规产品的2.15倍 [[S28,S21,S1]] |
| 喷涂参数 | 喷涂距离常规控制在100~200mm；喷涂过量导致残留引发气孔，喷涂不足则隔离层不完整 [[S23]] |
| 模具温度 | 模温过高导致脱模剂有效成分快速过量挥发；模温过低则成膜粗糙且残留未挥发水分，两者均降低粘结效率 [[S30]] |
| 合金成分（以铝合金为例） | Fe元素质量分数提升至0.7%~1%区间时，可显著降低与铁基模具的粘结倾向；Fe含量过低导致粘结效率快速衰减 [[S34]] |
| 模具表面粗糙度与涂层 | 模面粗糙度越高，局部凸起区域实际接触温度与铝元素富集程度越高，降低有效粘结效率。TiN、CrN、CrC类硬质涂层可物理隔离熔融铝与钢模基底，有效提升粘结效率临界耐受温度 [[S43,S12]] |
| 冲头润滑条件 | 冲头用高温润滑剂需在300~500℃区间保持稳定成膜能力，若润滑成膜失效，间接降低整套生产循环的粘结效率 [[S23]] |

## 缺陷关联

### 焊合（Die Soldering）机理

当脱模剂隔离层被高速金属流冲蚀失效后，熔融铝直接接触钢模表面，Fe从模具中溶解进入铝液，Al、Si等元素向模面扩散，在界面生成Fe-Al-Si系金属间化合物。当模面局部温度高于焊合临界温度（纯铝-纯铁体系下为655℃）时，界面会生成富铝液相，该液相凝固后直接将铸件与模面粘结，形成永久焊合层，此处粘结效率被判定为完全失效[[S11,S2,S44]]。

### 粘模与拉伤

粘结效率低于正常阈值时，界面隔离层失效，直接诱发粘模、焊合缺陷。铸件脱模过程中，与模面粘结部位受剪切作用，沿出模方向撕裂形成表面拉伤。严重时会导致铸件缺肉、模面残留铝层，需停机清模维护，大幅降低生产节拍[[S9,S36]]。

## 提升措施

### 脱模剂优化

纳米改性耐高温脱模剂（如Safety-Lube A）在350℃高模温条件下的型腔附着量可达常规脱模剂的2.15倍，在保证铸件顺利脱模的前提下，可将喷涂时间从20s缩短至17s，大幅降低高温区域粘模风险[[S21,S1]]。

### PVD涂层

TiN、TiAlN、CrN类PVD涂层可显著提升压铸模具抵抗熔融铝合金腐蚀与冲蚀的能力，有效降低金属粘模倾向，提升粘结效率[[S4]]。新型TiAlN/a-C纳米复合涂层应用于镁合金压铸型芯，抗镁合金粘附性能优于常规TiAlN涂层，模具寿命可达到127.6万次注射，是普通渗氮型芯的2倍[[S47]]。

### 压铸工艺参数优化

采用PQ²匹配方法，将压铸机与压铸模的工作点调整到工艺窗口合理区间，降低内浇口金属液流速，可减少金属液对型芯的高速冲蚀效应，有效降低粘模概率[[S18,S19]]。

## 局限性

### 实验室与现场差异

压铸现场实际粘模故障是多因素耦合的结果，涵盖高速金属液流持续冲蚀型芯、高温区域长时间滞留515℃以上温度、铝铁元素界面扩散渗透、充型过程湍流效应等多重作用。单一实验室测得的粘结效率指标仅能反映静态界面的抗粘附性能，无法完全覆盖生产动态多场耦合工况，导致实验室测试结果与实际压铸生产的表现存在显著偏差[[S18,S10,S22]]。

### 数值模拟预测盲区

现有主流压铸数值模拟软件（如AnyCasting）没有针对粘模现象的直接分析模块，仅能通过观测充型流动状态与温度场变化间接预估粘模发生概率，依靠单一粘结效率指标无法完整预测复杂工况下的粘模故障[[S10]]。

## 相关概念关系

| 相关术语 | 与粘结效率的关系 |
|---------|----------------|
| 脱模性 | 比粘结效率覆盖范围更广的综合性能指标，除界面抗粘附能力外，还包含模具拔模斜度、顶出系统设计、铸件凝固收缩率等多重因素[[S42]] |
| 接触润湿角 | 表征液体与固体表面之间润湿特性的基础物理子指标，接触角数值越高，金属液或脱模剂越难在模具表面铺展，是计算粘结效率的基础输入参数之一，不等于粘结效率本身[[S17,S24,S45]] |
| 界面反应层厚度 | 熔融合金与模具钢表面发生元素扩散、金属间化合物生成的厚度参数，厚度越大则粘结效率越低，是粘结效率动态演化的核心微观影响因子[[S38]] |
| 脱模剂有效率 | 特指脱模剂在高温模具表面成膜的附着能力与留存性能，属于脱模剂专属子指标，仅部分贡献于整体粘结效率[[S21,S33]] |
