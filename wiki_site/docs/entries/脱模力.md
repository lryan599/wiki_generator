## 脱模力（Ejection Force）

### 概述

在压铸领域，主流的标准英文术语为 **Ejection Force**，是高压压铸（HPDC）行业通用的官方命名，对应压铸机顶出机构的输出顶出力，所有压铸设备铭牌标注的参数统一使用该表述[[S14]]。而“Demoulding Force”属于跨全模具行业的泛用术语，覆盖塑料成型、压铸等所有成型工艺的脱模场景，不特指HPDC工况下的主顶出力[[S12]]。中国国家通用模具术语标准GB/T 8845-2017已正式收录“脱模力（ejecting force）”及“ejection force”等术语条目，完成了该核心概念的标准化定义[[S33]]。

脱模力是指为了克服脱模阻力所需要的力，其核心来源是压铸件冷凝收缩时对模具包紧部位产生的包紧力。设计时需满足 \(P_t > P_{tz}\) 的判定关系，其中 \(P_t\) 为脱模力（单位kN），\(P_{tz}\) 为综合因素下的脱模阻力（单位kN）[[S36,S31]]。

### 力学组成与脱模过程

脱模力的核心组成包括：
- 克服包紧型芯的摩擦阻力；
- 封闭壳体产生的真空吸力；
- 铸件与模具的粘附力；
- 脱模机构自身的运动摩擦阻力[[S8,S1,S5,S3]]。

![脱模力受力分析示意图，标注了脱模力F_t、包紧力F_b、摩擦阻力F_m及脱模斜度α，用于理解脱模力的力学组成与受力平衡。](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/dba2f3af-0d1f-43be-abd8-e58661e9cc95/resource)

在脱模过程中，脱模力并非恒定不变。脱模动作初始瞬间的**初始脱模力（初始抽芯力）**数值最大，用于克服铸件冷凝收缩后对型芯的最大包紧力；而后续继续将型芯全部抽出所需的**相继脱模力**远小于初始值。因此，在模具设计中仅需重点校核初始脱模力[[S8,S1,S5]]。压铸件在抽拔移动后，对活动型芯的包紧力比在开始抽拔的瞬时小，故相继抽芯力总是小于初始抽芯力[[S1]]。

实测脱模力随顶出行程变化曲线表明：初始阶段达到峰值即最大初始脱模力，随顶出行程推进，脱模力快速降低[[S32]]。

![脱模力随顶出行程变化的实测曲线，横轴为脱模距离（单位mm）、纵轴为脱模力（单位N），分别展示上模、下模两组脱模力的变化规律，说明初始阶段脱模力最大，随后随行程推进快速降低。](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/ad600269-e12d-4921-97e0-a2380a703ddc/resource)

### 与相关术语的辨析

#### 包紧力与脱模力
包紧力是压铸件金属冷却凝固时对成型型芯的法向正压力，是脱模力的核心计算输入参数，属于铸件对型芯的作用力；脱模力是克服包紧力及其他脱模阻力的外部驱动力。两者存在明确的属性边界，不可混淆[[S11,S27]]。

#### 抽芯力与脱模力
抽芯力与主脱模力的工程边界明确：抽芯力作用对象是侧活动型芯，运动方向沿侧向抽拔轴线，与主顶出方向不重合，对应独立的侧向抽芯机构；脱模力作用在完整铸件上，沿主开模方向运动。二者物理属性不同，行业设计流程中两个参数须独立计算选型，必须作为两个独立技术条目分开表述[[S30]]。

#### Stripping Force
Stripping Force 主要用于冲压行业的卸料脱料场景，在压铸领域仅少量用于描述特殊嵌套薄片类铸件的强行拔出工况，不作为通用脱模力术语使用[[S44]]。

### 主要影响因素

影响脱模力大小的因素分为四大类[[S7,S41,S16,S17]]：

| 影响因素类别 | 具体因素及影响方向 |
|---|---|
| **铸件几何参数** | 包络型芯侧面积越大，脱模力越大；壁厚越大、形状越复杂，脱模力越大；圆形型芯包紧力最小，矩形或异型曲线轮廓型芯包紧力更大[[S7,S41]] |
| **模具相关参数** | 拔模斜度越大，脱模力越小；模具成型面粗糙度越低、表面越光洁，脱模力越小；加工纹向应与抽拔方向一致；模具刚性不足产生变形会额外增大脱模阻力[[S7,S16]] |
| **合金特性** | 合金线收缩率越大、弹性模量越高，对型芯的包紧力越大，脱模力越高[[S11,S17]] |
| **压铸工艺参数** | 压射比压、保压压力越高，脱模力越大；模具温度越高，脱模力越小；铸件在模内停留时间/冷却时间越长，收缩完成度越高，脱模力越大[[S7,S11]] |

此外，影响压铸脱模力大小的核心因素还包括：铸件包紧部位的截面形状（非圆形截面包紧力大于圆形截面）、收缩受阻部位数量与间距、铸件脱模温度、合金与模具温差、脱模机构配合阻力、粘模状态、以及脱模斜度过小等[[S36,S7]]。

### 计算方法

#### 基础经验估算公式

压铸领域通用脱模力基础估算经验公式为[[S7,S29]]：

\[
F_t \geqslant K p A
\]

式中：
- \(F_t\) ——所需脱模力（N）；
- \(p\) ——单位面积挤压应力（见下表）；
- \(A\) ——铸件包紧型芯的侧面积（m²）；
- \(K\) ——安全系数，一般取1.2。

#### 考虑拔模斜度的精确估算公式

\[
F = pA(f \cos\alpha - \sin\alpha)
\]

式中：
- \(F\) ——脱模/抽芯力（N）；
- \(p\) ——单位面积包紧挤压应力（MPa）；
- \(A\) ——铸件包紧型芯的侧面积；
- \(f\) ——铸件和型芯表面的摩擦系数，压铸场景下取0.2～0.25；
- \(\alpha\) ——型芯的拔模斜度[[S8,S24,S17]]。

#### 不同合金的单位面积挤压应力

| 压铸合金 | 挤压应力 \(p\)（MPa） | 摩擦系数 \(f\) |
|---|---|---|
| 锌合金 | 6～8 | 0.25 |
| 铝合金 | 10～12 | 0.25 |
| 铜合金 | 12～16 | 0.35 |
| 镁合金 | 参考铝合金偏低区间选取 | 0.25 |

数值选取方法：厚壁铸件、复杂异形型芯选取应力区间上限，薄壁简单圆形型芯选取区间下限[[S7,S8,S24,S17,S31]]。

#### 脱模阻力精确计算公式

当被包紧型芯长度 \(L \leqslant 0.05\text{m}\) 时[[S31]]：

\[
P_{tz} = \frac{L A p (\mu_m \cos\alpha - \sin\alpha) \times 10^6 + L K}{1000}
\]

当 \(L > 0.05\text{m}\) 时[[S31]]：

\[
P_{tz} = \frac{L A p (\mu_m \cos\alpha - \sin\alpha) \times 10^6 + \frac{1}{2} K (L + 0.05)}{1000}
\]

式中：
- \(P_{tz}\) ——脱模阻力（kN）；
- \(L\) ——铸件包紧模具型芯的长度（m）；
- \(A\) ——包紧部位周长（m）；
- \(p\) ——挤压应力（MPa）；
- \(\mu_m\) ——压铸合金对模具的摩擦系数，锌、铝、镁合金取0.25，铜合金取0.35；
- \(\alpha\) ——包紧部位的脱模斜度（°）；
- \(K\) ——考虑壁厚的单位长度附加脱模阻力（N/m）。

此公式可同时用于抽芯力计算和铸件是否滞留在定模内的校核[[S31]]。

#### 简化安全估算公式

对于周长不超过320mm、壁厚大于3mm的圆形或正方形型腔[[S24]]：

\[
P = L_z A_z p \mu_0
\]

式中：
- \(L_z\) ——内腔折算深度（cm）；
- \(A_z\) ——内腔折算周长（cm）。

对于壁厚<3mm的类长方形铸件，脱模力为标准计算值的70%～80%；对于三侧包围、长宽比≥2的开口类铸件，脱模力仅为标准计算值的30%～50%[[S24]]。

#### 有限元分析处理方式

通过热弹性应力公式计算铸件冷却过程中产生的接触包紧应力[[S23]]：

\[
p_c = \frac{E}{1 - \nu} \times \alpha \Delta T
\]

式中：
- \(E\) ——合金弹性模量；
- \(\nu\) ——泊松比；
- \(\alpha\) ——合金线膨胀系数；
- \(\Delta T\) ——铸件脱模前后的温度差。

所有热物性参数需按对应温度点取值，以提升计算精度[[S23]]。

### 模具设计中的角色

#### 顶出系统驱动方式

压铸顶出系统驱动方式分为两类[[S37,S42]]：

| 驱动方式 | 特点 |
|---|---|
| **机械式顶出** | 依靠开模动作驱动顶出，行程固定，存在瞬时冲击 |
| **液压式顶出** | 由顶出油缸直接驱动，顶出行程、速度可灵活调节，可实现多次顶出动作，现代压铸机普遍采用 |

#### 总顶出力取值

总顶出力取值应为铸件总包紧力的1.2倍，保证有足够推力克服脱模过程中的额外摩擦阻力，避免铸件滞留在模腔内[[S37]]。压铸机配置的顶出油缸额定推力必须大于模具设计所需的总计算顶出力[[S13,S22]]。

#### 顶杆布局设计规则

| 设计规则 | 具体要求 |
|---|---|
| **优先位置** | 顶杆需优先布置在铸件加强筋、厚壁等铸件强度较高的区域[[S25,S37,S40]] |
| **关键部位** | 包紧力最大的型腔部位必须布置顶杆，保证局部脱模力充足[[S37]] |
| **靠近薄壁** | 顶杆尽量靠近薄壁区域但不能直接作用于薄壁面上，避免顶穿[[S25,S37]] |
| **均匀对称** | 顶杆位置均匀对称分布，使铸件整体受力平衡，保证顶出变形量最小[[S37]] |
| **浇注系统** | 溢流槽、横浇道、内浇口位置也需设置辅助顶杆，保证整个浇注系统余料可顺利脱出[[S25]] |

北美NADCA压铸行业设计标准给出了顶出机构的三类标准化结构：刚性顶杆式、套筒推管式、弹簧回程式，同时规定锌合金铸型顶针最小直径≥3/16英寸（约4.76mm），避免顶穿高温软质铸件[[S9,S10]]。

#### 顶杆强度校核

单根顶杆的截面积需满足顶出压强低于对应合金的高温许用压力，避免顶出压强过高导致顶白、顶穿缺陷。校核不通过时需增大顶杆直径或增加同区域顶杆数量以降低局部压强[[S37]]。

#### 顶板刚度要求

推板、推杆固定板需具备足够结构刚性，顶出过程中推板最大变形量须控制在允许范围内，避免因刚性不足导致顶杆不同步、受力不均，引发铸件局部变形、顶裂问题[[S36]]。

#### 均匀顶出要求

所有顶杆的顶出动作须同步，顶出速度平稳均匀，铸件整体不同区域的顶出位移差须控制在允许范围内；顶出距离须保证铸件完全脱离动模型腔，留出机械手自动取件的充足操作空间[[S37]]。

### 脱模力过大导致的缺陷

| 缺陷类型 | 产生机理 |
|---|---|
| **变形/开裂** | 脱模力过大超过铸件脱模温度下的高温强度极限，顶出过程中铸件非均匀受力产生塑性变形甚至完全开裂。常见诱因包括脱模剂性能不足导致铸件与型腔粘连、脱模斜度不足导致包紧力骤增[[S34]] |
| **顶白/顶穿/顶杆印痕** | 顶杆截面面积不足，顶出压强超过对应合金的高温许用接触应力时，顶杆作用位置出现局部发白（顶白）、表面凹陷印痕甚至直接顶穿铸件薄壁部位[[S37]] |
| **表面裂纹** | 顶出过程中脱模力分布严重不均，铸件局部拉应力超过该脱模温度下的合金断裂强度，在非直接受力部位产生延伸至铸件内部的连续性表面裂纹[[S34]] |
| **粘模** | 脱模斜度过小、型腔表面粗糙度过高、合金与模具亲和力过强导致脱模阻力远超设计值，脱模力不足以推动铸件完全脱离型腔，铸件部分表面被强行撕裂留存在模具上[[S36]] |

### 脱模力匹配控制原则

| 控制措施 | 具体要求 |
|---|---|
| **安全余量** | 计算理论脱模力时预留≥1.2倍的安全余量[[S31,S21]] |
| **脱模斜度优化** | 合理增大脱模斜度以降低包紧力[[S31,S21]] |
| **表面质量** | 合理控制模具表面粗糙度，并保证加工纹路与出模方向一致[[S21]] |
| **脱模剂** | 使用适配的脱模剂降低合金与模具界面摩擦系数[[S31]] |
| **脱模温度** | 控制铸件在模内冷却到合适温度后再进行顶出操作，避免脱模温度过低导致收缩量过大、包紧力异常升高[[S31]] |
