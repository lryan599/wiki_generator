## 概述

落砂机是一种在铸造工艺中，用于将浇注并冷却后的铸件与包围其周围的型砂（或砂芯）分离的机械设备。落砂工序是连接砂处理、造型与铸件清理的枢纽环节[[S23,S27]]。在国家标准中，“落砂”被定义为用手工或机械方法使铸件与型（芯）砂分离的操作[[S2]]，而“落砂机”特指带有栅格、采用振动方法完成此作业的机器，其标准英文对应术语为 *shakeout* 或 *knock-out equipment*[[S14]]。

需要特别指出的是，目前公开可查的落砂机权威术语定义绝大部分来自通用铸造领域（砂型铸造）。未检索到针对压铸场景的专属落砂机定义[[S1,S9]]。然而，当压铸件采用砂芯形成内部复杂空腔时，浇注后同样需要去除砂芯，此时通用铸造领域的落砂、除芯设备技术特性可直接迁移应用，适配性可覆盖90%以上的常规除芯需求[[S1,S9]]。

## 标准定义与分类体系

国家标准 GB/T 25370-2020 对落砂机进行了明确定义，并将其与“除芯机”区分开来：落砂机（*shakeout / knock-out equipment*）是带有栅格、采用振动方法完成落砂作业的机器；除芯机（*decoring machine*）则是采用气动或液压振击锤清除铸件砂芯的机器[[S14,S25]]。这表明在标准分类中，去除外围型砂（落砂）与清除内腔砂芯（除芯）是两个有机关联但相对独立的工艺环节。

在实际工程应用中，落砂机的分类更为细致，形成了一个多级分类框架[[S3,S6]]：

![落砂机通用分类框架图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/252c31af-1254-4a88-ac67-789eae23ea43/resource)

常见的落砂机类型主要包括以下两大分支[[S3,S6,S14]]：
1.  **机械振动落砂机**：依靠激振器产生周期性的机械振动，是应用最广泛的类型。其下又可根据驱动方式细分为：
    *   **偏心振动落砂机**：通过偏心轴转动产生振动。
    *   **惯性振动落砂机**：由带有偏重块的主轴旋转产生离心激振力。
    *   **电磁振动落砂机**：由电磁振动器提供激振力。
    *   **冲击式惯性振动落砂机**：一种特殊的惯性式，铸型不直接放置在振动台上，而是通过下方落砂机撞击铸型支撑架来落砂[[S20]]。
2.  **滚筒式落砂机**：依靠滚筒旋转，将铸件和型砂提升后跌落、撞击，从而实现分离和冷却[[S5,S14]]。

此外，针对铸件内腔的砂芯，风动型芯落砂机（如 L415 型）是直接作用于铸件进行除芯的典型设备，在压铸领域具有直接应用价值[[S2,S28]]。

## 工作原理

不同类型落砂机的工作原理与技术边界清晰，其核心均是利用机械力使铸型破碎，达到铸件与型砂分离的目的。

| 设备类型 | 工作原理 | 关键特性 |
| :--- | :--- | :--- |
| **偏心振动落砂机** | 电机带动偏心轴转动，使装有砂箱的栅格产生周期性撞击振动[[S28]]。 | 落砂效率稳定，不受载荷变化影响；但主轴直接承受全部撞击力，使用寿命短，维修量大[[S28]]。 |
| **惯性振动落砂机** | 电机驱动带有不平衡偏重的主轴旋转，产生离心激振力，带动支承在弹簧上的栅架振动[[S20,S22]]。 | 主轴不直接承受撞击力，寿命长，维修量小；但落砂效果会随载重变化[[S20]]。其原理如下图所示：<br><br>![惯性振动落砂机原理示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/99249d9d-5123-4089-8709-994912ea140f/resource) |
| **冲击式惯性振动落砂机** | 铸型放在固定支架上，下方惯性振动落砂机顶面与支架底面留有5~8mm间隙，通过撞击铸型底部实现落砂[[S20,S22]]。 | 载荷不全程作用在落砂机上，对负载波动适应性强，额定能力更高[[S20]]。 |
| **电磁振动落砂机** | 经降频电路将三相交流电转换为低频脉冲电压，供给电磁激振器，使振动台上下跳动撞击砂箱[[S27,S28]]。 | 无机械传动结构，工作可靠，激振力可调，生产率高，功耗低；衔铁安装气隙推荐0.5～1.5mm[[S27,S28]]。 |
| **滚筒冷却落砂机** | 脱箱后的铸型进入倾斜（推荐角度4°～6°）旋转的滚筒内，被带到一定高度后跌落，通过反复撞击和摩擦实现落砂[[S5,S27]]。 | 适用于无箱射压造型线，作业环境粉尘少；但对铸件类型有局限，不适用于薄壁易损件，且设备制造复杂，维修不便[[S5,S22]]。进入滚筒的铸件温度不得高于700℃[[S19]]。 |
| **风动型芯落砂机** | 以压缩空气为动力，产生振动直接作用于铸件，将内腔的砂芯震落[[S2,S28]]。 | 专门用于清除溃散性好的油砂、纸浆砂等砂芯，是压铸行业带砂芯铸件除芯的典型设备[[S2,S28]]。 |

## 设备结构与组成

一套完整的落砂机设备由多个功能部件构成，主要包括：
*   **激振源（激振器/振动电机）**：是设备的核心，通过偏心块旋转产生可调离心激振力，是振动的来源[[S26,S31]]。
*   **落砂栅格/床体**：直接承载铸型和铸件，通过撞击使型砂破碎并通过栅格缝隙排出。栅格尺寸需适配铸件，防止铸件磕碰损伤或型砂堵塞，其形式有焊接条缝式、百叶窗式等[[S24,S19]]。
*   **减振/谐振弹簧**：隔离振动向基础的传递，保护厂房基础。常采用圆柱形螺旋弹簧、钢-橡胶混合弹簧等，其安装精度要求所有弹簧在自由状态下的顶面高度公差≤1mm[[S26,S30]]。
*   **铸件承载平台/振动框架**：作为参振基体，承载激振器、栅格和载荷，需具备足够的抗振刚度，防止在长期交变载荷下变形或开裂[[S26,S32]]。
*   **除尘接口/抽尘罩**：对接工厂集中除尘系统，捕集落砂过程产生的大量扬尘，改善作业环境[[S4,S5]]。

## 关键工艺参数与技术要求

### 选型参数与适配规则
落砂机的选型需综合考虑铸件材质、型砂强度和生产节拍[[S24,S8]]：
*   **铸件材质**：铸钢件难落砂，应选用高振幅、低频率的落砂机；铸铁及有色金属铸件（包括压铸件）易损伤，应选用低振幅落砂机[[S24,S8]]。
*   **型砂类型**：湿型砂强度低，适配低振幅落砂机；干型砂等高强度铸型需选用高振幅落砂机[[S24]]。
*   **落砂时间**：一般中小件为1～5分钟，大件5～10分钟，特大件10～20分钟[[S24]]。

### 性能、安全及寿命要求
根据行业标准 JB/T 6575-93，落砂机需满足以下强制性技术要求[[S10,S30]]：
*   **效率指标**：固定式落砂效率不低于90%；输送式在额定负荷下效率不低于98%。
*   **运行温度**：滚动轴承温升不超过35℃，最高温度不超过70℃。
*   **安全与噪声**：激振器零件不得有裂纹，回转件严禁焊补；空载噪声声压级不得大于93dB(A)。工作在过共振区的惯性落砂机必须配备可靠的制动或限幅装置，以保证启停平稳[[S10,S30]]。
*   **使用寿命**：正常工况下首次大修期不少于6000小时（不含弹簧更换）；减振/谐振弹簧寿命不低2000小时[[S30]]。
*   **操作规范**：不得将温度大于等于150℃的型砂带入落砂机，以免损伤后续输送设备[[S7]]。

## 应用实例与设备局限性

在铸造实践中，落砂机是自动化造型线的标准配置。例如，在大型超临界C12A阀体的酚醛树脂砂铸造中，冷却后的铸型通过落砂机完成型砂与铸件的分离[[S11]]。与此同时，落砂机产出的旧砂会经磁性分离、破碎、筛分、冷却等设备处理后进入再生系统，实现循环利用[[S23,S13]]。

尽管应用广泛，落砂机仍有其局限性：
*   **铸件损伤风险**：滚筒式落砂机不适用于薄壁轻型铸件，持续撞击可能造成磕碰损伤[[S22]]。
*   **维护与易损件**：核心易损件包括激振器、轴承、减振弹簧和栅格板。其中，同组弹簧要求顶面高差在1mm以内[[S32]]。
*   **压铸关联性**：在压铸领域，落砂（除芯）主要针对带有砂芯的压铸件。除使用通用振动落砂机外，L415型风动型芯落砂机是专门用于去除压铸件内腔砂芯的典型设备[[S28]]。
