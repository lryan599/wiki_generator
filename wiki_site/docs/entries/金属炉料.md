## 概述

金属炉料（metallic charge）在压铸熔炼工序中专指装入熔化炉内参与熔炼过程的全部金属类原材料。根据《机械基础制造工艺标准汇编 铸造 上》的术语定义，金属炉料包括金属锭、中间合金、废金属、回炉料等[[S49]]。在压铸生产中，合金熔炼是获得优质铸件的关键工序，金属炉料作为熔炼的初始投入物料，其品质直接决定合金液的化学成分、含气量及夹杂物水平，是能否获得符合压铸工艺要求的合格合金液的基础[[S38,S49]]。

## 分类与组成

### 按来源分类

压铸用金属炉料按来源可分为五大类[[S18,S39,S2]]：

| 类别 | 说明 | 典型物料 |
|------|------|----------|
| 原生锭 | 由专业冶金厂按压铸合金牌号的国家或国际标准生产供应的预制合金锭 | ADC12合金锭、A380合金锭、AZ91D合金锭 |
| 回炉料 | 压铸生产过程产生的废料 | 料饼、浇口、溢流块、废压铸件、毛刺、切屑 |
| 外购废料 | 来源复杂的厂外社会回收金属废料 | 社会回收铝合金废料、其他加工场景边角料 |
| 中间合金 | 将高熔点难熔合金元素预先与基体金属熔合得到的低熔点过渡合金 | Al-Si、Al-Mn、Al-Cu等中间合金 |
| 添加剂 | 用于调整合金成分、改善冶金性能的纯金属小料 | 纯镁锭、纯锌锭、变质剂等 |

回炉料进一步按品质分级[[S39]]：
- **一级回炉料**：化学成分合格的废料，包括报废铸件、浇注系统、冒口等，确认无混料可直接回用，回用量可达 **80%**。
- **二级回炉料**：含有较多杂质、气体但化学成分尚合格的废料，需经重熔、精炼、化验后方可酌情使用，回用量可达 **70%**。
- **三级回炉料（杂烩废料）**：来自机加工切屑、社会回收杂料、炉渣回收金属等，成分复杂、品位最低，搭配比例通常限制在 **25%～30%**。

### 按物理形态分类

炉料按物理形态可分为锭、块、屑、碎料四类[[S28,S18]]：锭类多为原生合金锭，块类多为大块回炉料，屑类为机加工切屑，碎料为破碎后的低等级回炉料或外购杂料，不同形态对应不同的预处理要求和允许掺用比例。

## 典型压铸合金牌号

### ADC12

ADC12 是日本 JIS H5302 标准下的 Al-Si-Cu 系压铸铝合金牌号，对应中国国标 GB/T 15115 中的 YL113。其典型化学成分范围如下[[S25,S50,S3]]：

| 元素 | 含量范围（质量分数 %） |
|------|------------------------|
| Si | 9.6～12 |
| Cu | 1.5～3.5 |
| Fe | ≤1.3 |
| Mg | ≤0.3 |
| Mn | ≤0.5 |
| Zn | ≤1.0 |
| Ni | ≤0.5 |
| 单个其他杂质 | ≤0.05 |
| 杂质总含量 | ≤0.2 |
| Al | 余量 |

### A380

A380 是美国 ASTM B85 标准下的 Al-Si-Cu 系压铸铝合金牌号，是全球应用最广泛的通用压铸铝合金之一。其典型化学成分要求如下[[S43,S17,S10]]：

| 元素 | 含量范围（质量分数 %） |
|------|------------------------|
| Si | 7.5～9.5 |
| Cu | 3.0～4.0 |
| Fe | ≤2.0 |
| Mg | ≤0.1 |
| Mn | ≤0.5 |
| Ni | ≤0.5 |
| Zn | ≤3.0 |
| Sn | ≤0.3 |
| Al | 余量 |

### AZ91D

AZ91D 是美国 ASTM B94 标准下的高纯度压铸镁合金牌号，命名规则为 A 代表 Al、Z 代表 Zn，数字 9 和 1 分别代表铝和锌的名义质量分数，D 代表高纯度级别。其化学成分要求如下[[S11,S53,S21]]：

| 元素 | 含量范围（质量分数 %） |
|------|------------------------|
| Al | 8.5～9.5 |
| Zn | 0.45～0.9 |
| Mn | ≥0.17 |
| Fe | ≤0.005 |
| Fe/Mn 比值 | ≤0.032 |

## 炉料品质要求

### 化学成分可追溯性

所有入炉压铸炉料的化学成分必须明确可追溯。外购合金锭需附带官方化学成分检验报告单（标明出产厂家、牌号、规格、重量、生产日期、炉号等内容），入炉前需复验确认符合对应合金牌号标准要求[[S14,S29]]。购入的纯金属锭、预制合金锭必须符合国家或部颁标准[[S38]]。

### 洁净度要求

入炉前必须完全清除炉料表面附着的油污、泥沙、铁锈、氧化皮及各类夹杂异物。不同牌号合金的回炉料需严格分类存放管理，不得发生混料[[S19,S29]]。各种牌号合金的化学成分与允许杂质含量各不相同，有些元素在某一合金中是主要成分，而在另一合金中却是杂质[[S18]]。

### 尺寸与形状合规要求

- **大块炉料**（料饼、浇口、完整废压铸件）：清理除油后可直接投入熔炼，无需重熔[[S19,S5]]。
- **细碎炉料**（飞边、毛刺、机加工切屑、炉渣回收金属）：必须破碎、重熔铸锭、完成成分检验合格后才能投入后续配料使用，其用量需严格管控[[S19,S5]]。

### 预热与干燥规范

所有入炉炉料必须经过预热以去除表面吸附水分，预热规范如下[[S31,S52]]：

| 炉料类型 | 预热温度 | 保温时间 |
|----------|----------|----------|
| 铝合金锭、硅铝明锭等常规铝基炉料 | 350～450℃ | 2～4 h |
| 镁锭 | 200～300℃ | 1～2 h |

入炉的铝锭和回炉料的表面应清洁干燥[[S28]]。充分预热可完全去除炉料表面吸附的水分，避免加料时发生金属液飞溅或水爆事故[[S31,S52]]。

压铸镁锭预热装置通过内置加热器均匀加热镁合金锭，完全去除表面吸附水汽，避免加料时引发水爆，属于镁合金压铸生产的配套选用设备[[S16]]。

## 装料工艺与参数

### 加料顺序

加料顺序因炉型不同有所差异[[S28,S36]]：

| 炉型 | 加料顺序 | 原因说明 |
|------|----------|----------|
| 快速熔炼炉、坩埚炉 | 先加回炉料垫底，再加合金锭 | 避免高空落下的合金锭损伤炉床和炉体 |
| 反射炉 | 先加原生合金锭，熔化形成金属液后再加回炉料 | 避免回炉料高温下直接接触炉内气氛发生过度氧化 |

对于易于损耗、熔点低的合金元素（如镁、锌、锡），应在全部主炉料完全熔化后、金属液温度达到 **690～700℃** 的熔化末期再加入，可大幅降低其蒸发烧损量[[S33,S15]]。

### 炉料配比

**压铸铝合金回炉料配比**[[S5,S28]]：
- 大块废铸件、粗大浇道类洁净度合格的回炉料，最大使用占比可达 **60%** 及以上。
- 飞边、切屑类细碎回炉料经重熔、除气除杂、成分核验合格后，最大使用占比不得超过 **30%**。
- 对力学性能、内部致密性要求严苛的高端压铸件，需进一步降低回炉料使用比例。

**压铸锌合金回炉料配比**[[S45]]：
- 新料与回炉料的推荐配比为 **2:1**，最大回炉料占比不得超过 **50%**。
- 回炉料必须严格剔除微量 Pb、Sn、Cd 等有害杂质，避免引发锌合金晶间腐蚀。

### 熔化速率控制

在保证炉料完全熔化的前提下，应尽可能缩短总熔炼时长，严格控制金属液过热温度。熔炼时间越长、过热温度越高，合金液吸气和氧化夹杂生成的概率就越高，会直接降低最终压铸件的力学性能和内部质量[[S33]]。

### 覆盖剂与精炼剂的使用

覆盖剂、精炼剂在加入炉内之前必须提前放置在 **110～120℃** 的烘箱中充分烘干去除内部吸附水分。禁止直接向未完成预热的高温金属液中投入未除水的熔剂，以防止发生喷溅、水爆安全事故[[S14,S29]]。

## 对熔体质量与压铸缺陷的影响

### 潮湿炉料导致氢致气孔

潮湿炉料表面吸附的水分和铝熔体反应生成氢，氢在固态铝中溶解度极低，凝固时析出形成内部气孔。炉料表面氧化层含 **30%～60%** 的 Al(OH)₃ 结合水，即使加热到 900℃ 也难以完全去除，持续向熔体中释放氢，加剧气孔缺陷产生[[S27,S37]]。

铝-硅压铸合金中因氢气引起的多孔性空洞（放大200倍）的显微图像可为该缺陷机制提供直观表征[[S35]]。

### 油污与有机物导致夹杂

炉料表面附着油污、未完全烧除的切削液等有机物，在熔化过程中分解生成气体和非金属残碳，既提升熔体含气量促进气孔形成，也会作为外来夹杂留存于熔体中，最终形成夹杂缺陷[[S32]]。

### 杂质元素超标导致硬质点

炉料中杂质元素含量超标会生成高硬度的金属间化合物或不熔的高熔点质点，随熔体进入铸件后形成硬质点缺陷。这类硬质点在加工过程中会大幅提升刀具磨损率，严重时直接造成铸件报废[[S7,S23,S30]]。

![带夹杂缺陷的压铸件](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/8ab4f1fe-1f63-4500-b0d4-5fe94a921227/resource)

### 氧化夹杂引发渣孔与缩松

氧化夹杂超标的炉料会直接向熔体中引入大量 Al₂O₃ 等高熔点氧化物质点。这类质点密度小、难以从熔体中上浮排除，最终随充型进入铸件内部形成肉眼可见或探伤可检出的渣孔、夹杂缺陷。同时，这类氧化物质点作为非自发形核核心，会提升合金熔体凝固过程中的缩孔、缩松倾向[[S35,S4,S41]]。铝-镁压铸合金中铝镁尖晶石氧化物（MgAl₂O₄）聚集物的显微图像（放大500倍）直观展示了该类氧化夹杂的微观形态[[S35]]。

### 对铸件力学性能稳定性的影响

炉料的成分纯净度、含气量、夹杂含量直接决定压铸合金的力学性能稳定性。不合格炉料会导致同一批次铸件的拉伸强度、延伸率等力学指标出现大幅离散，在相同压铸工艺参数下部分铸件无法满足强度设计要求，提升生产废件率[[S42,S12]]。

## 与熔炉及后续工序的关系

### 不同炉型的配套要求

**坩埚炉**[[S28,S46,S26]]：
- 适用场景：小批量生产
- 装料顺序：先以回炉料垫底，再加入合金锭
- 配套保温炉容量：**0.5～1 小时**内压铸生产的金属消耗量
- 要求：金属液表面积尽可能小，减少舀取作业过程中的金属氧化

**反射炉**[[S28]]：
- 局限：不适合高要求精密铸件生产
- 装料顺序：先加合金锭，熔化后再加回炉料
- 缺点：金属吸气多、氧化烧损大，仅能适配低附加值大体积铸件

**集中熔化炉**[[S20]]：
- 适用场景：大批量连续压铸生产
- 熔化速率：**100～4500 kg/h**
- 优势：可集成炉料预热功能，氧化烧损极低，合金成分均匀、温控精度高，可同步配套在线除气除渣装置

### 与熔体处理工序的衔接

炉料处理与后续工序的衔接流程为[[S28,S48]]：所有入炉炉料完成清理去除表面油污杂物后 → 先进行 **350～450℃** 保温 **2～4 小时**的预热除水处理 → 投入熔化炉升温至 **670～760℃** 完全熔化 → 调整温度至 **700～730℃** 区间依次开展除气、除渣作业 → 根据产品性能要求完成变质处理、晶粒细化 → 经炉前质量检测合格后转入保温炉供压铸机使用。

## 安全与存储管理

### 加料防爆防喷溅

- 所有投入熔化炉的炉料必须经过充分预热至 **150℃** 以上完全干燥，确认无可见水分后缓慢投入高温金属液中[[S22,S47,S6]]。
- 熔化炉周围及炉前坑区域不得存在任何积水，避免液态金属接触水发生爆炸喷溅事故[[S22,S47]]。
- 坩埚使用前必须预热到 **600℃** 以上充分干燥[[S47,S22]]。
- 工具使用前必须预热，精炼剂、变质剂必须烘干[[S47,S22]]。

### 人员防护

熔炼操作人员必须穿戴全套耐高温防护用品，包括劳保服、帆布防护手套、护目镜、防烫靴。作业时裤脚需套在防烫靴外侧，上衣不得塞入腰带中。开展变质、精炼作业时操作人员需佩戴防护口罩并站在上风位置，避免吸入有害烟气[[S6,S22,S47]]。

### 炉料存储管理

| 存储要求 | 规范 |
|----------|------|
| 金属料垛高度 | 不超过 **2 m** |
| 过道宽度（纵横主通道） | 不小于 **1 m** |
| 车辆通行通道宽度 | 不小于 **2 m**（仓库内电瓶车通道） |
| 配料场炉料储备量 | 满足车间 **24 小时**以上生产需求 |
| 镁合金炉料/碎屑 | 单独存放于干燥、远离明火和水源的区域，严禁在镁合金熔炼工位周边堆放过量镁合金料 |
| 不同牌号/成分炉料 | 分类存放，设置明确标识避免混料[[S13,S6]] |
