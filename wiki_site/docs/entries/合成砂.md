## 概述

合成砂（synthetic sand）是铸造行业中一类重要的造型材料，特指由原砂、粘结剂、附加物按一定配比人工混制而成的、具备所要求造型或造芯性能的型砂或芯砂[[S12,S8]]。其核心特征在于组分可完全定制，性能可实现精准调控，区别于天然存在的粘土砂[[S12,S15]]。

> **适用范围说明**：合成砂的主流应用场景为砂型铸造（含重力铸造、低压铸造）的铸型及砂芯制备。截至现有公开证据，尚无合成砂作为常规材料直接用于高压压铸（HPDC）大批量生产的量产级应用记录；在半永久模铸造（semi-permanent mold casting）中，合成砂仅作为可消耗砂芯使用[[S3,S11]]。

## 定义与基本概念

根据 GB/T 5611-2017《铸造术语》的定义：

> **合成砂**：由原砂、粘结剂、附加物按一定配比混制成的具有所要求造型（芯）性能的型（芯）砂。[[S12]]

合成砂与天然粘土砂的核心区别在于组分来源与可控性[[S12,S15]]：

| 对比维度 | 合成砂 | 天然粘土砂（天然型砂） |
|---------|--------|----------------------|
| **组分来源** | 人工选取原砂、粘结剂、附加物进行配比混制 | 天然存在的硅砂，粘土含量（质量分数）8%~20% |
| **制备方式** | 需经过配料、混砂等工序 | 仅添加适量水即可直接造型 |
| **组分可控性** | 所有组分完全可定制 | 组分天然分布，不可定制 |
| **性能调控** | 可精准调控各项性能参数 | 性能由天然组分决定，调控空间有限 |

## 分类体系

合成砂可按多种维度进行体系化分类[[S12,S4]]。

### 按粘结剂类型分类

| 大类 | 粘结剂类型 | 典型代表 |
|------|-----------|---------|
| **无机粘结剂合成砂** | 水玻璃 | 水玻璃砂（钠水玻璃应用最广）、酯硬水玻璃砂 |
| | 磷酸盐 | 磷酸盐自硬砂 |
| | 水泥 | 水泥砂（加水泥速凝剂、溃散剂等附加物） |
| | 粘土 | 粘土合成湿砂（以膨润土等为粘结剂） |
| **有机粘结剂合成砂** | 呋喃树脂 | 自硬呋喃树脂砂、热芯盒树脂砂 |
| | 酚醛树脂 | 壳型（芯）覆膜砂、三乙胺法冷芯盒砂 |
| | 油类 | 油砂 |
| | 合脂 | 合脂砂（合成脂肪酸残渣粘结剂） |

### 按用途分类

- **型砂**（molding sand）：用于形成铸型外轮廓型腔的混合料[[S8,S12]]
- **芯砂**（core sand）：用于形成铸件内部孔洞或复杂内腔的混合料[[S8,S12]]

### 按固化方式分类

化学硬化砂按硬化方法可分为三型[[S12,S15]]：

- **自硬砂**（self-hardening sand）：在室温下通过粘结剂与固化剂化学反应自行硬化，所造型、芯不需烘干[[S15]]
- **气硬砂**：通过吹入气体（如 CO₂、三乙胺等）引发硬化反应
- **热硬砂**：通过加热实现硬化（如壳法、热芯盒法）

## 组成与结构

### 原砂选型

合成砂可选用的铸造原砂品类丰富[[S5]]：

| 原砂类型 | 主要组分 | 特性与典型应用 |
|---------|---------|---------------|
| 硅砂（天然/人工） | SiO₂ | 最广泛应用的基础原砂 |
| 锆砂 | ZrO₂·SiO₂ | 粒度细、热膨胀系数低、导热率高；用于大型铸钢件面砂及涂料 |
| 铬铁矿砂 | FeO·Cr₂O₃ | 固相烧结性好、耐火度高于1790℃；用于合金钢铸件 |
| 铝矾土砂/球形陶瓷砂 | Al₂O₃-SiO₂ | 经电弧炉熔融喷制，球形颗粒，适用于造型制芯 |
| 刚玉砂 | Al₂O₃（≥90%） | 耐火度高于1700℃；用于大型铸钢件和合金钢件 |
| 镁砂 | MgO | 用于高锰钢铸件型砂及涂料 |
| 镁橄榄石砂 | 镁橄榄石为主 | 适用于高锰钢铸件型（芯）砂 |
| 熟料砂 | 煅烧硬质粘土 | 膨胀系数小、耐火度高；用于大型铸钢件面砂 |
| 炭粒砂 | 煅烧石油焦 | 球形砂 |

### 典型组分配比

合成砂的配比因粘结剂体系和用途而异[[S12,S14]]：

- **合脂砂**：合脂粘结剂添加量（质量分数）为 3%~4.5%，常额外添加粘土和纸浆废液或糊精以提升湿强度[[S12]]
- **呋喃树脂类合成砂**：
  - 铸铁、铸钢场景：树脂添加量（质量分数）为 1.5%~2.0%
  - 非铁合金场景：树脂添加量（质量分数）为 1%~1.5%
  - 需配套固化剂按比例调配[[S14]]

## 核心性能参数

合成砂的性能表征涉及多项指标，其中部分参数的实测数据如下。

### 粒度与流动性

三种铸造用砂的 AFS 平均细度及流动性检测结果[[S9]]：

| 砂种 | AFS 平均细度 | 流动性（s） |
|------|:----------:|:----------:|
| 高硅砂 | 53.14 | 15 |
| 陶瓷砂 | 52.94 | 14 |
| 再生砂 | 56.12 | 14 |

陶瓷砂与再生砂的流动性优于常规高硅砂[[S9]]。

### 透气性与发气性

合成砂的水分含量显著低于天然造型砂，可降低浇注时过量水汽产生的概率，从而减少铸件气孔缺陷[[S10]]。合成砂制备的砂芯需具备足够透气性，以确保金属液加热过程中砂芯内部产生的气体能被完全排出，避免铸件内部形成气孔类缺陷[[S10]]。

### 其他关键性能

根据合成砂的应用需求，还需关注以下性能指标（相关试验方法见本标准体系）：

- 湿压强度、干压强度、热强度
- 溃散性（浇注后砂芯的崩解能力）
- 耐火度
- 热膨胀率
- 可使用时间与可脱模时间

## 工艺角色与应用范围

合成砂在砂型铸造全流程中承担核心工艺功能[[S10]]：

- **形成铸型外轮廓**：作为型砂使用，形成铸件的外部形状
- **形成复杂内腔**：作为芯砂使用，制备常规造型手段难以成形的内部孔洞和复杂内腔[[S10]]

合成砂可适配多种铸造工艺及合金类型[[S3,S11,S14]]：

- **适用工艺**：砂型铸造（含重力铸造、低压铸造）、半永久模铸造（仅限可消耗砂芯）
- **适用合金**：铝合金、镁合金、锌合金、铸铁、铸钢等
- **不适用场景**：高压压铸（HPDC）的大批量生产场景暂无公开可查的量产应用证据[[S3,S11]]

## 有机与无机粘结剂体系对比

| 对比维度 | 有机粘结剂砂（树脂砂） | 无机粘结剂砂（水玻璃砂为主） |
|---------|---------------------|---------------------------|
| **粘结剂类型** | 酚醛树脂、呋喃树脂等合成树脂 | 水玻璃（硅酸钠）、磷酸盐等 |
| **硬化方式** | 加热硬化（壳法、热芯盒法）、气体硬化（三乙胺法、SO₂法）、固化剂自硬 | CO₂硬化、有机酯硬化、加热硬化 |
| **砂芯强度与表面质量** | 高，制芯效率高 | 适中 |
| **环境特性** | 制芯和浇注过程中产生有毒气体，环境污染较严重 | 无毒无害、无有毒气体排放，符合绿色铸造要求 |
| **经济性** | 适中 | 原材料来源广泛，具经济性优势 |

## 常见缺陷及成因

合成砂使用过程中可能产生以下典型缺陷[[S13]]：

| 缺陷类型 | 成因机制 |
|---------|---------|
| **金属渗透型粘砂** | 砂粒粒度偏粗、未涂刷防护涂料、浇注温度过高，导致液态金属渗入砂粒间隙 |
| **砂眼/表面剥落** | 合成砂整体强度不足，金属液冲刷铸型导致砂粒脱落并卷入铸件 |
| **气孔缺陷** | 砂芯排气通道设计不合理、水分或粘结剂添加过量，导致发气量过大，气体无法及时排出 |
