## 概述

压铸模（die casting die）是压力铸造生产中使用的核心工艺装备，与压铸机、压铸合金并称为压铸生产三大基本要素[[S36,S6,S17]]。三大要素通过压铸工艺协同作用，最终产出合格的压铸件，而压铸模的结构设计合理性直接决定铸件的成型质量、生产效率与制造成本[[S14]]。

![压铸生产三大要素协同关系图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/6c6fdfa5-d1a6-4b80-a846-37f8ad677b65/resource)

压铸模由定模和动模两个主要部分组成。定模固定在压铸机定模安装板上，与压铸机压射机构的压室连接；动模安装在压铸机动模安装板上，随动模安装板移动而与定模实现合模或开模[[S14,S18]]。

## 结构与系统组成

压铸模完整包含八类功能子系统：成形部分、模架部分、浇注系统、排溢系统、温控系统、抽芯机构、推出机构、导向部分[[S14,S18,S26]]。各子系统通过预设装配约束配合，共同完成金属液充型、铸件成型、排气排渣、模温调控及铸件完整脱出的全流程压铸动作[[S14]]。

### 各子系统功能与关键零部件

| 子系统 | 功能 | 关键零部件 |
|--------|------|------------|
| 成形部分 | 决定压铸件几何形状和尺寸精度，构成型腔和型芯 | 定模镶块、动模镶块、型芯、侧型芯 [[S18,S26]] |
| 模架部分 | 将模具各部分按规律组合固定，实现与压铸机安装 | 动模座板、定模座板、动模套板、定模套板、支承板、垫块 [[S18,S26]] |
| 浇注系统 | 沟通模具型腔与压铸机压室，为金属液进入型腔的通道 | 浇口套、分流锥、直浇道、横浇道、内浇口 [[S18,S26]] |
| 排溢系统 | 排除型腔气体，储存冷金属和涂料余烬 | 排气槽（排气块）、溢流槽 [[S14,S26]] |
| 温控系统 | 控制模具工作温度，满足铸件质量和工艺需求 | 冷却水路、加热通道 [[S14]] |
| 抽芯机构 | 抽动与开合模方向不一致的活动型芯 | 斜销、滑块、楔紧块、弯销、液压抽芯器 [[S14,S18]] |
| 推出机构 | 将压铸件从模具中脱出，并完成推出元件复位 | 推杆、推板、推杆固定板、复位杆、推板导柱、推板导套 [[S18,S26]] |
| 导向部分 | 引导动、定模开模和合模运动 | 导柱、导套 [[S18,S26]] |

## 分类维度

### 按压铸机类型分类

- **热室压铸机用压铸模**：适配锌、锡、铅等低熔点合金，主流采用中心浇口结构，浇口套端面突出定模座板与压铸机喷嘴球面贴合，压铸循环时间短，适合小型精密压铸件量产[[S45]]。
- **卧式冷室压铸机用压铸模**：当前应用最广泛的压铸模类型，适配铝合金、镁合金、铜合金等高熔点有色合金，也可用于低熔点合金压铸，通用性最强[[S8]]。
- **立式冷室压铸机用压铸模**：浇注系统布置形式与卧式冷室压铸模不同，压室垂直设置[[S8]]。

### 按型腔数目分类

| 类型 | 特征 | 适用场景 |
|------|------|----------|
| 单腔压铸模（一模一穴） | 结构简单，模具成本低 | 大型复杂压铸件中小批量生产 [[S2,S23]] |
| 一模两腔压铸模 | 生产效率比单腔提升约1倍，平衡充型流道设计难度低 | 中小批量中等尺寸压铸件的主流选型 [[S40,S2]] |
| 一模多腔压铸模 | 3个及以上型腔共用同一浇注系统，生产效率最高 | 小型压铸件大批量量产，对流道平衡设计精度要求高 [[S23,S38]] |

### 按分型面与型腔相对位置分类

分型面与铸件空腔的相对位置可分为三类：型腔全在定模内、型腔分处定动模中、型腔全在动模内，不同布局适配不同脱模需求和排气设计要求[[S7]]。

## 工作环境与失效形式

### 服役条件

压铸模属于所有热作模具中服役条件最苛刻的类别[[S1,S30]]：

| 参数 | 数值范围 |
|------|----------|
| 承受压力 | 20~150 MPa [[S1,S30]] |
| 接触金属熔体温度 | 锌合金约400℃ 至 黑色金属1200~1600℃ [[S1,S30]] |

不同合金压铸模型腔表面最高温度差异显著：锌合金压铸模不超过400℃，铝合金压铸模可达600℃以上，铜合金压铸模可达800℃以上，黑色金属压铸模超过1000℃[[S44,S33]]。服役过程中随压铸循环反复发生急冷急热的温度波动[[S44]]。

![压铸模工作周期温度变化曲线](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/e34ad70d-78f4-410a-928f-ad69a7f55a8c/resource)

### 主要失效模式

1. **热疲劳龟裂**：占比60%~70%的主流失效模式。铜合金压铸模的热疲劳失效占比进一步升至80%~90%[[S30,S51,S46]]。失效机制为反复急冷急热导致表层产生交变压应力和拉应力，在晶界碳化物、夹杂物集中区萌生微裂纹，后续熔融金属挤入裂纹中加速扩展[[S30,S46]]。

![压铸模具的热疲劳裂纹实物示例](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f0c776e4-13d2-4c05-bb13-6773bbbf043d/resource)

2. **粘模（焊合）**：主要发生在铝合金压铸场景中，高温铝液与钢铁质模具表面发生化学反应生成脆性化合物，伴随冲蚀作用加速表面损伤[[S1,S12]]。

3. **冲蚀与熔损**：高温、高压、高速金属液对型腔表面产生强烈碰撞和冲刷，造成模具表面软化和材料磨损[[S12]]。

模具失效的关联因素涉及压铸循环过程中合金液热传导、型腔表面压力、内浇口材料流动三类路径，分别引发热疲劳/热脆裂、机械应力主导的裂纹扩张/变形、冲蚀磨损/熔蚀三类失效模式[[S19]]。

## 关键性能影响因素

### 模具材料

国内牌号4Cr5MoSiV1（执行GB/T1299-2000标准）与美国AISI H13、德国DIN 1.2344均属于C-Cr-Mo-Si-V系中合金热作模具钢[[S9,S11,S25]]。其中DIN 1.2344的碳含量范围控制在0.37%~0.43%，相比国产普通4Cr5MoSiV1和AISI H13的碳含量区间更窄，材质均匀性、各向韧性塑性表现更优[[S11]]。

H13钢经规范热处理后硬度可达HRC47~49，淬透性优异（直径100mm棒材空冷即可完全淬硬），热疲劳抗力优异，产生的热疲劳裂纹细而短、扩展速率低，适用于铝合金、镁合金压铸模成形部件[[S53,S9]]。

替代热作模具钢牌号包括[[S16,S49,S29]]：

| 牌号 | 主要特点 |
|------|----------|
| 4Cr5MoSiV（H11） | Mo、V含量略低于H13，热强性稍弱但成本更低 [[S16]] |
| HM1（4Cr3Mo3W2V） | 冷热疲劳抗力高于3Cr2W8V [[S49]] |
| HM3（3Cr3Mo3VNb） | 600℃以上高温强度优于普通H13，适配铜合金等高熔点合金压铸 [[S49,S29]] |
| CG-2（60Cr4Mo3Ni2WV） | 高强韧性热作模具钢 [[S16]] |
| 012Al（5Cr4Mo3SiMnVAl） | 高强韧性热作模具钢 [[S16]] |

### 模具预热与温控

不同合金对应的模具预热温度规范[[S21]]：

| 合金种类 | 预热温度范围 |
|----------|--------------|
| 锌合金 | 150~200℃ |
| 铝合金 | 180~300℃ |
| 镁合金 | 200~250℃ |
| 铜合金 | 300~350℃ |

采用专用温控系统稳定模温可使压铸模整体寿命延长2~3倍[[S21]]。

### 去应力回火

压铸模中间去应力回火的推荐周期[[S21,S35]]：

| 压铸合金 | 回火周期（模次） |
|----------|-------------------|
| 锌合金 | 20,000~50,000 |
| 铝、镁合金 | 10,000~20,000 |
| 铜合金 | 500~1,000 |

回火温度取480~520℃，该措施可提升模具使用寿命1~2倍[[S21,S35]]。

## 设计参数与精度要求

### 尺寸公差

压铸件成型部位尺寸公差需符合GB/T 6414-1999、GB/T25747—2010、GB/T15114—2009相关标准。壁厚公差相比对应等级的普通线性尺寸公差粗一级；受分型面、抽芯活动部位影响的尺寸需在基础公差上叠加规定附加量[[S28]]。

### 壁厚控制

普通压铸场景下，推荐常规壁厚不宜超过4.5~6mm，同一压铸件内最大壁厚与最小壁厚之比不应大于3。过厚的壁面会导致铸件中心晶粒粗大、缩孔缩松缺陷概率上升，铸件力学性能随壁厚增加呈下降趋势[[S34,S22,S15]]。

### 脱模斜度

不同合金推荐脱模斜度取值要求为：锌合金最小脱模斜度低于镁合金，镁合金低于铝合金，铝合金低于铜合金。外壁脱模斜度取值为对应内孔脱模斜度的1/2，斜度不计入公差范围。配合面最小脱模斜度可压缩至外表面5'、内表面30'[[S5,S15,S50]]。

### 表面粗糙度

压铸件表面粗糙度可达到Ra 2.5~0.63μm，高要求场景可达Ra 0.32μm[[S34]]。

## 模具寿命

常规工业生产条件下各类合金压铸模寿命[[S27,S51,S31]]：

| 合金种类 | 典型模具寿命（模次） |
|----------|---------------------|
| 锌合金（H13钢） | 100万次以上 [[S27,S51]] |
| 铝合金 | 10~15万次 [[S27,S46]] |
| 镁合金 | 铝合金的3~5倍 [[S46]] |
| 铜合金 | 约1万次 [[S51]] |

行业实际案例中，优化后的铝合金风扇零件压铸模使用寿命可达100万模次，常规汽车零部件压铸模寿命可达40万模次以上[[S37]]。

## 应用实例

### 铝合金凸轮轴盖压铸模

采用一模两腔布局，设置4个侧置内浇口，单个铸件配置5个梯形截面溢流槽，采用水平侧抽芯结构，针对壁厚不均特征设置分区冷却水路，解决高壁厚部位的缩孔、气孔缺陷[[S40]]。

### 铝合金电机壳压铸模

适配中心浇口结构，采用二次分型设计，搭配斜销侧抽芯驱动壳体成型，依托推件板二次推出机构完成深腔薄壁筒体类零件顺利脱模[[S3]]。

### 铝合金减速器壳体压铸模

包含弯销抽芯、齿条齿轴联动抽芯等多向抽芯机构，配套防转压条、滚轮等辅助结构，用于多侧孔壳体类零件的复杂抽芯设计[[S32]]。

### 新能源车大型薄壁铝合金压铸模

采用梯度温控系统，匹配高填充速度的充型需求，模具排气系统开设密度高于常规压铸模，部分型腔配置真空排气机构，抽芯机构采用液压驱动长距离抽芯结构以适配大尺寸复杂外轮廓特征[[S47]]。

## 经济性

压铸模具材料成本仅占压铸模总制造成本的10%左右，选用优质模具钢可显著提升压铸模整体使用寿命，降低单件压铸件分摊的模具成本[[S37]]。

单位压铸件总成本中，压铸模相关费用占比为10%~30%，大批次量产场景下模具成本可得到充分分摊，综合生产成本远低于小批量生产场景[[S41]]。减少模具修模总时长、降低单次修模占用时间，可直接提升有效生产时长，优化修模周期带来的产能增益幅度显著[[S41]]。

## 权威标准与术语

### 国内标准

- **GB/T 8844-2003《压铸模技术条件》**：规定压铸模零件技术要求、总装配技术要求，适用于所有有色金属压铸模的设计、制造与验收[[S10]]。
- **GB/T 8845-2017《模具术语》**：整合了原GB/T 8847-2003《压力铸造模具术语》，完整覆盖压铸模全系列零部件的标准术语与定义[[S39]]。

### 国际规范

北美压铸协会（NADCA）发布的系列压铸模设计推荐规范为国际行业通用设计参考准则[[S13]]。
