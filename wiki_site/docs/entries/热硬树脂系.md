## 概述

“热硬树脂系”（hot hardening resin system）是我国国家标准GB/T 5611-2017《铸造术语》中明确规定的铸造领域规范术语，定义为“在高于室温的温度下发生交联反应使所造砂型（芯）硬化的各种树脂砂粘结剂”，其法定应用场景限定为热芯盒砂和覆膜砂（壳芯/壳型）[[S2,S30,S32]]。该类树脂体系在加热条件下发生不可逆的分子间交联反应而固化，冷却后无法恢复原有物理状态与性能，与热塑性树脂粘结剂具有本质差异[[S2,S30]]。

在压铸工艺语境中，热硬树脂系不直接用于整体金属型造型，而是作为配套砂芯的粘结剂，制备用于形成复杂内腔（如发动机缸体水道、油道芯）的高精度砂芯，同时亦可用作浸渗密封剂封堵铝、镁合金压铸件内部微孔缺陷[[S8,S16,S37,S20]]。

## 术语界定与英文对应

依据GB/T 5611-2017，中文规范术语为“热硬树脂系”，对应标准英文术语为 **hot hardening resin system** [[S2,S32]]。该词条在铸造领域的英文别名为 **thermosetting resin system**，在国内标准与教材中亦常称为“热固性树脂粘结剂”（加热硬化类）或“加热硬化树脂系统”[[S2,S9]]。

### 与热塑性树脂的核心差异

热硬树脂系与热塑性树脂粘结剂的区别在于其化学固化机制：

| 对比维度 | 热硬树脂系 | 热塑性树脂粘结剂 |
|---------|-----------|----------------|
| 分子结构 | 可发生交联的体型结构 | 线型分子结构 |
| 固化机理 | 高于室温下发生不可逆分子间交联反应 | 加热软化/熔化，冷却回复固态，不发生交联 |
| 热循环行为 | 冷却后无法恢复原有状态与性能 | 可在特定温度区间反复加热冷却，性能可逆 |
| 铸造中典型用途 | 热芯盒砂/覆膜砂粘结剂 | 覆膜砂预涂树脂、涂料、型芯胶粘剂 |

**来源：[[S2,S30,S9,S13]]**

## 分类与组成

### 标准分类定位

依据GB/T 5611-2017，热硬树脂系在铸造用树脂体系中与自硬树脂系（self-hardening resin system）、气硬树脂系（gas cured resin system）并列，属于按固化方式分类的三大树脂砂粘结剂体系之一[[S32]]。其法定应用工艺限定为**热芯盒砂**和**覆膜砂（壳型/壳芯）** [[S2,S30]]。

![树脂砂制壳型（芯）工艺分类，加热硬化类均属于热硬树脂系对应的典型应用场景](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/cdfa0999-b771-4b2c-b756-eafe289c76a4/resource)

该分类图来自《航空制造工程手册──特种铸造》，明确将树脂砂制壳型（芯）工艺分为加热硬化与室温硬化两大类，其中加热硬化分支所列的热芯盒法、温芯盒法等均属于热硬树脂系的典型应用工艺[[S33]]。

### 典型树脂类型

热硬树脂系包含以下主要子类：

**1. 酚醛树脂（Phenol-Formaldehyde Resin）**

由苯酚类和甲醛类缩聚合成。在酸性催化剂、苯酚与甲醛摩尔比大于1的条件下得到热塑性酚醛树脂（诺沃腊克型），需配合硬化剂（如六亚甲基四胺/乌洛托品）方可在加热条件下发生不可逆交联，形成不熔不溶的体型结构，是铸造用壳型砂、覆膜砂的主流粘结剂[[S30,S26,S32]]。

**2. 呋喃树脂（Furan Resin）**

分子结构中含呋喃环，以糠醇为主要原料，可通过醛类改性糠醇合成，具备不可逆加热交联硬化特征。常见子类包括糠醇呋喃树脂、尿醛-糠醇呋喃树脂（呋喃I型）、酚醛-糠醇呋喃树脂（呋喃II型）等，可用于热芯盒砂制备[[S30,S2,S4]]。

**3. 尿醛-糠醇改性树脂**

以甲醛、尿素、糠醇等为主要原料缩聚而成的热固性树脂，是热芯盒工艺的典型粘结剂。依据JB/T 3828-2013《铸造用热芯盒树脂》，该类树脂按含氮量分级，适用于热芯盒制芯工艺[[S36,S34]]。

### 组成-工艺对应关系

依据《金属材料液态成型工艺》收录的“铸造常用树脂类型、构成及对应工艺”表，热硬树脂各子类与其适配工艺的对应关系如下[[S21]]：

| 树脂类型 | 主要化学构成 | 适配工艺 |
|---------|-------------|---------|
| 酚醛树脂 | 苯酚+甲醛 | 壳型（芯）/覆膜砂、热芯盒 |
| 呋喃树脂 | 糠醇+醛类改性 | 热芯盒、自硬砂 |
| 尿醛-糠醇树脂（呋喃I型） | 尿素+甲醛+糠醇 | 热芯盒 |
| 酚醛-糠醇树脂（呋喃II型） | 苯酚+甲醛+糠醇 | 热芯盒 |

## 工艺作用

### 在压铸中的角色与边界

在压铸场景中，热硬树脂系的工艺作用集中于以下三个维度：

**1. 作为砂芯粘结剂（核心应用）**

热硬树脂系在压铸中主要用于制备配套金属型铸型使用的砂芯，而非用于整体铸型造型[[S30,S40]]。典型应用场景包括：
- 高压压铸复杂内腔砂芯（如发动机缸体水道、油道芯）——适配热芯盒工艺[[S16,S37]]
- 低压铸造、金属型重力铸造铝合金铸件精密内腔成型——适配壳芯（覆膜砂）工艺[[S29,S40]]

**2. 作为浸渗密封剂（压铸后处理）**

以环氧类、改性酚醛类热固性树脂为主要组分的真空高压浸渗剂，是封堵铝、镁合金压铸件内部微孔缺陷的主流材料，固化后可完全填充压铸件内部孔隙，提升铸件气密性[[S8,S20]]。

**3. 作为砂芯胶合剂**

用于两半砂芯粘合及砂芯修补的热硬树脂粘结剂，要求干拉强度大于0.7 MPa，可适配压铸工艺下砂芯组合定位、金属镶件粘接固定的使用需求[[S5,S38]]。

### 与普通砂型铸造的边界区分

压铸场景下热硬树脂仅用于制备配套金属型使用的砂芯，对单位质量发气量、高温溃散性指标要求显著高于普通砂型铸造；普通砂型铸造场景下热硬树脂可直接用于制备完整壳型铸型，适配单件、小批量砂型铸造的整体铸型成型需求[[S30,S40]]。

## 工艺参数与操作条件

### 热芯盒法典型参数

热芯盒法采用液态热固性树脂与潜伏型催化剂混合制成的芯砂，射入预热芯盒后完成固化。关键参数[[S16,S17,S37]]：

| 参数项 | 典型范围 |
|-------|---------|
| 金属芯盒预热温度 | 180～260°C |
| 表层固化时间 | 数十秒即可形成数毫米厚固化层 |
| 后固化方式 | 砂芯取出后依靠自身余热与固化反应放热完成内部后固化 |
| 树脂加入量 | 2%～3%（占砂重） |
| 催化剂用量 | 树脂量的15%～25%（氯化铵-尿素-水溶液体系） |
| 适用场景 | 大批量生产中、小尺寸高精度砂芯 |

### 壳芯法（覆膜砂）典型参数

壳芯法以酚醛热硬树脂为粘结剂，预制成覆膜砂后进行制芯。关键参数[[S29,S40,S23]]：

| 参数项 | 典型范围 |
|-------|---------|
| 覆膜温度 | 130～180°C（原砂预热后混入粉状酚醛树脂） |
| 芯盒预热温度 | 200～280°C |
| 保温结壳时间 | 20～30 s |
| 结壳厚度 | 5～12 mm |
| 砂芯抗拉强度 | 294～392 kPa |
| 树脂加入量 | 2%～4%（占砂重） |
| 固化剂 | 六亚甲基四胺（乌洛托品） |

## 缺陷分析与防控

### 典型缺陷与机理

热硬树脂作为砂芯粘结剂，在压铸充型过程中因高温金属液作用可诱发以下铸造缺陷：

**1. 卷气型气孔**

树脂在高温铝/镁合金液作用下快速析出大量气体。当发气速率与压铸充型速度不匹配、气体无法及时从型腔排气通道排出时，易被卷入高速流动的金属液内部，形成卷气型气孔缺陷[[S18,S7,S39]]。

**2. 气渣坑/气渣隔**

这是树脂砂特有的铸造缺陷。当铸件大平面向上浇注时，树脂高温分解的大量气体聚集在型腔顶部形成气袋，压力足够阻止金属液完全充填该区域，形成表面凹坑；随后气体压力降低，金属液中的熔渣流入凹坑附着在表面形成黄色渣层；后续充填的金属液如覆盖该渣层，会形成中间夹渣层的气渣隔缺陷[[S7]]。

**3. Burn-on粘砂**

热硬树脂受热分解产生的残炭与金属氧化物、硅砂杂质发生界面反应，生成低熔点共晶相，促使金属液渗透进入砂粒间隙，在铸件表面形成无法通过常规抛丸清理去除的牢固粘附砂层[[S42,S25]]。

**4. 溃散性不足与清砂残留**

酚醛甲醛类热硬树脂砂芯的高温热强度高但溃散性相对较差，铸件凝固收缩过程中砂芯无法及时破碎退让，易导致铸件内部残留砂粒；针对薄壁铝合金压铸件充型热量不足的工况，可能出现树脂分解不完全、出砂困难的问题[[S18,S24,S41]]。

**5. 浸渗层失效与表面变色（浸渗场景）**

热硬聚酯/厌氧树脂类有机浸渗剂长期工作耐温上限仅约200°C，若铸件服役温度超过该值，浸渗层会分解失效导致铸件重新出现渗漏缺陷；浸渗前若铸件清洗不彻底，树脂无法完全渗入孔洞会残留浸渗痕迹，引发铸件表面变色斑点缺陷[[S1,S10,S6]]。

### 缺陷防控措施

依据公开的呋喃树脂自硬砂铸造工艺规范，上述典型缺陷可通过以下组合方案实现针对性防控[[S35,S7,S15,S18]]：

**砂芯制备维度：**
- 降低树脂添加量——再生砂场景下树脂加入量不宜超过1.5%
- 提升旧砂再生处理脱膜率，控制砂灼减量不超过3.5%
- 使用较粗的原砂、严格控制粉尘含量，提升砂芯透气性
- 填砂后人工扎出直径φ3～φ5 mm的通气孔，孔距60～80 mm，加速砂芯内部气体排出

**压铸工艺维度：**
- 分型面设置出气溢流冒口，溢流部分截面积必须大于内浇口截面积
- 适当降低充型阶段的喷射速度，减少湍流卷气概率
- 通过数值模拟优化金属液流态，避免气体在型腔局部聚集形成气袋

**砂料配方维度：**
- 选用溃散性更优的脲醛甲醛热硬树脂作为铝/镁合金压铸件砂芯粘结剂，该类树脂在铸件浇注温度下可快速分解破碎，减少热裂和清砂困难问题[[S18]]

## 与自硬树脂系的对比

热硬树脂系与自硬（冷硬）树脂系在固化机理、生产效率、能耗及适配场景等方面存在系统性差异[[S3,S22,S12,S2]]：

| 对比维度 | 热硬树脂系 | 自硬树脂系（冷硬） |
|---------|-----------|-----------------|
| 固化方式 | 高于室温的芯盒温度下发生交联硬化 | 常温下依靠催化剂与树脂的化学反应自行硬化 |
| 起模/固化时间 | 数十秒完成固化出芯 | 数分钟至数十分钟 |
| 生产效率 | 高，适合大批量 | 较低，适合单件小批 |
| 环境适应性 | 受环境温度、湿度影响极小，工艺稳定性高 | 对环境温湿度变化敏感，需严格的工艺管控 |
| 能耗 | 需加热芯盒，能耗较高 | 无需加热芯盒，整体能耗低 |
| 适配场景 | 大批量中小精密压铸件自动化制芯 | 中大尺寸、小批量压铸件配套砂型/砂芯生产 |

## 不确定性说明

本次检索未发现“热硬树脂系”在压铸领域中作为铸型涂料组分、脱模剂成分的独立应用证据。术语在压铸中的定指范围以GB/T 5611-2017的法定定义为准，即限定为热芯盒砂和覆膜砂粘结剂体系，浸渗密封剂与砂芯胶合剂属于热硬树脂在压铸后处理及砂芯组装中的扩展应用[[S2,S30,S32]]。
