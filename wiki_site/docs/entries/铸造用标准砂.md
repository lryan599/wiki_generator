## 概述

铸造用标准砂（正式英文命名为 **foundry standard silica sand**）[[S1,S4]]是一种按照特定技术规格生产的硅砂基准物质。其在铸造行业中的核心定位是作为检定铸造用型（芯）粘结剂强度等性能的标准物质[[S1,S4]]。该材料经天然石英砂加工而成，通过严格的粒度分级、水洗、擦洗等工艺达到标准规定的化学成分与物理性能要求，是铸造实验室进行仪器校准、方法验证及实验室间比对的基础性基准材料。

中国铸造标准体系中“铸造用标准砂”的技术框架参考了美国铸造学会（AFS）早年发布的成熟规范；在国际铸造行业，与之对应的通用英文表述包括 **AFS standard sand**、**reference sand for foundry** 等[[S11]]。

## 牌号

### 现行标准牌号（GB/T 25138-2010）

根据现行国家标准 GB/T 25138-2010，铸造用标准砂的牌号表示为 **ZBS90-53A-1.20**[[S1]]。牌号中各字段的含义如以下表格所示。

| 字段 | 含义 | 说明 |
|---|---|---|
| ZBS | 铸造用标准砂（铸、标、砂汉语拼音首字母） | 产品类别标识 |
| 90 | 二氧化硅分级代号 | 表示SiO₂质量分数 ≥ 90% |
| 53 | AFS平均细度 | 平均细度标称值 |
| A | 平均细度允许偏差 | 允许偏差为 ±2 |
| 1.20 | 角形因数标称值 | 颗粒形貌指标 |

### 旧版标准牌号（JB/T 9224-1999）

在已被替代的机械行业标准 JB/T 9224-1999 中，铸造用标准砂的牌号表示为 **ZBS90-21-30**，其中 21 为粒度分组代号，30 为粒形代号[[S4,S6,S8]]。

## 标准演变

国内铸造用标准砂的标准迭代路径经历了从机械行业标准到国家标准的演进过程：

**JB 2487-78 → ZB J31006-89 → JB/T 9224-1999 → GB/T 25138-2010（现行有效）**[[S10,S7]]

其中，GB/T 25138-2010 全称为《检定铸造粘结剂用标准砂》，于2010年9月26日发布，2011年2月1日起实施，是当前唯一有效的铸造用标准砂产品标准[[S1,S10]]。

## 技术要求

### 化学成分

GB/T 25138-2010 规定的铸造用标准砂化学成分要求如下[[S12]]。

| 化学成分 | 质量分数 / % |
|---|---|
| SiO₂ | ≥ 90 |
| Al₂O₃ | < 4.5 |
| Fe₂O₃ | < 0.30 |
| CaO + MgO | < 0.40 |
| Na₂O + K₂O | < 3.00 |

旧版 JB/T 9224-1999 标准仅规定了 SiO₂ ≥ 90% 的单一化学成分指标[[S6,S9]]。

### 粒度组成

GB/T 25138-2010 规定的铸造用标准砂粒度分布要求如下，以各筛号上的余留量（质量分数）表示[[S2,S1]]。平均细度控制为 53 ± 2[[S1]]。

| 筛孔基本尺寸 / mm | 筛号 | 余留量（质量分数） / % |
|---|---|---|
| 3.350～0.600 | 6～30 | < 2 |
| 0.425 | 40 | < 13 |
| 0.300 | 50 | 18～23 |
| 0.212 | 70 | 40～46 |
| 0.150 | 100 | 13～17 |
| 0.106 | 140 | < 8 |
| 0.075 | 200 | < 1.5 |
| 0.053～底盘 | 270～底盘 | ≤ 0.3 |

### 通用性能要求

现行标准 GB/T 25138-2010 对铸造用标准砂的其他通用性能规定如下[[S1]]。

| 检测项目 | 指标要求 |
|---|---|
| 细粉含量（0.020～0.075 mm） | ≤ 0.3% |
| 含泥量（<0.020 mm） | ≤ 0.3% |
| 含水量 | ≤ 0.3% |
| 酸耗值 | ≤ 5.0 mL |
| 角形因数 | ≤ 1.20 |

JB/T 9224-1999 旧版标准中角形因数要求为 ≤ 1.30，且不作为出厂验收强制指标，仅在牌号中标注粒形代号[[S6,S9]]。

### 酸耗值

酸耗值定义为中和 50 g ± 0.01 g 砂中碱性物质所消耗的 0.1 mol/L 盐酸溶液的毫升数，单位为 mL[[S1]]。该指标用于表征标准砂中的碱性杂质含量，直接影响粘结剂的固化反应可重复性。

## 试验方法

GB/T 25138-2010 规定的各项性能检测方法及所引用的标准如下[[S1]]：

| 检测项目 | 引用标准 |
|---|---|
| SiO₂ 及杂质含量 | GB/T 7143《铸造用硅砂化学分析方法》 |
| 粒度组成、细粉含量、含泥量、含水量、酸耗值 | GB/T 2684《铸造用砂及混合料试验方法》 |
| 角形因数 | GB/T 9442《铸造用硅砂》 |
| 试验筛要求 | JB/T 9156《铸造用试验筛》 |

## 检验规则

铸造用标准砂以 **100袋** 为一个检验批次，若不足100袋也按一个批次计[[S1,S6]]。检验取样时，从同一批次的任意三个包装袋中各取 1 kg 砂样，混合均匀后以四分法缩分制备待测试样[[S1,S6]]。供需双方若对检测结果存在争议，可委托双方协商选定的第三方机构进行仲裁检测[[S1]]。

## 包装与储运

铸造用标准砂单袋标称净重为 **25 kg**，包装必须具备防潮、防漏性能[[S1,S4]]。包装外表面须标注产品名称、牌号、净重、生产单位全称及商标[[S1]]。每批次产品必须附带产品合格证明文件[[S1,S4]]。在运输和贮存过程中应保持防潮、防漏条件，以确保标准砂含水量及其他性能指标的稳定性。
