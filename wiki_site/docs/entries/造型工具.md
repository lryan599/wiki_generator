在压铸领域，“造型工具”的狭义定义等同于压铸模具（Die Casting Die），是使液态金属在压力作用下快速充填型腔，冷却凝固成型为制件的专用工艺装备[[S20,S29]]。其广义范畴则覆盖全部直接参与压铸件成型的压铸专用工艺装备，核心为压铸模具，也包含配套的切边模等，但不涵盖制模工具或成型后的修整工具[[S31,S26]]。该术语与砂型铸造中使用砂钩、镘刀等手工工具的“造型工具”是完全不同的概念[[S34,S35]]。在国际标准中，其英文译名通常对应 **die casting tooling**，狭义场景下则完全等同于**die casting die**[[S19,S36]]。

压铸模具是压铸生产三大核心要素（压铸合金、压铸模具、压铸机）之一，直接决定压铸件的尺寸精度、表面质量和生产效率[[S13,S5]]。一副典型的压铸模具主要由动模和定模两部分构成，合模后形成完整的型腔、浇注系统和排溢系统[[S11,S38]]。

## 分类
压铸模具可根据多种维度进行分类，以适应不同的生产需求和设备条件。

| 分类依据                     | 类别                                                                 | 说明                                                                                                                              |
| :--------------------------- | :------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------- |
| **模具构造**[[S9,S32,S24]]   | **整体式压铸模**                                                     | 型腔直接在整体模块上加工成型。强度高、刚性好，易于设置冷却水道，适用于型腔较浅的小型单腔模或小批量生产。                          |
|                              | **镶拼式压铸模**                                                     | 型腔和型芯由镶块组装而成。便于复杂型面加工、节省耐热钢材、易于更换易损件，广泛用于型腔较深、大型或多型腔的模具。                |
|                              | **共用模套式压铸模**                                                 | 镶拼式的变形，不同型腔镶块共用一个模套，可快速切换生产不同款式的压铸件。                                                          |
|                              | **组合式压铸模**                                                     | 配备标准模座、推出机构和冷却管，可快速更换不同模体，大幅缩短换模时间。                                                            |
| **型腔数量**[[S32,S24]]      | **单腔压铸模**                                                       | 多用于中大型压铸件。                                                                                                              |
|                              | **同形多腔压铸模**                                                   | 多用于小型同规格压铸件的大批量生产。                                                                                              |
|                              | **异形多腔压铸模**                                                   | 因不同铸件的出品率和生产节拍难以匹配，实际生产应用较少。                                                                          |
| **适配压铸机**[[S9]]         | 普通热室压铸机用、卧式冷室压铸机用、立式冷室压铸机用、全立式冷室压铸机用 | 其中，**卧式冷室压铸机**配套的压铸模应用范围最广。                                                                                |
| **适配压铸合金**[[S25]]      | 锌合金、铝合金、镁合金、铜合金、黑色金属压铸模                       | 不同合金的熔融温度差异巨大，对模具材料的耐热、抗热疲劳性能有截然不同的要求。                                                      |

## 结构组成
压铸模具是一个由多个功能模块组成的复杂系统，通常包含以下八大组成部分[[S13,S11,S38,S3]]。

| 组成部分         | 功能描述                                                                                                                                                                               | 关键组件举例                                                                 |
| :--------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| **成型部件**     | 合模后形成铸件轮廓的空腔，是直接决定压铸件形状、尺寸精度和表面质量的核心部件，与高温金属液直接接触，必须选用耐热钢材制作[[S11,S18]]。                                               | 动模镶块、定模镶块、活动/固定型芯                                           |
| **浇注系统**     | 作为熔融金属从压铸机压室进入模具型腔的通道，可调控金属液充型的方向、速度与流态，对铸件质量、生产效率和模具寿命有显著影响[[S27,S11]]。                                              | 直浇道、横浇道、内浇口、余料（视压铸机类型而定）                               |
| **排溢系统**     | 由溢流槽和排气槽组成，用于在充型过程中排出型腔内的气体、涂料挥发气，并储存混有氧化夹杂的冷污金属液，能有效降低压铸件的气孔缺陷[[S27,S3]]。                                       | 溢流槽、排气槽                                                              |
| **温控系统**     | 通过内置冷却/加热管路通入循环介质（如水或导热油），平衡模具温度分布，避免因局部温差过大导致热疲劳开裂，保证铸件均匀凝固和稳定生产[[S32,S11]]。                                    | 冷却/加热管路、模温机                                                        |
| **模架部分**     | 将模具各部分组合并固定，确保模具能够精确、可靠地安装到压铸机上[[S13,S11]]。                                                                                                            | 动/定模座板、动/定模套板、支承板、垫块                                      |
| **导向部分**     | 精确引导动模与定模的开合模运动，保证成型部件能准确对合[[S11,S3]]。                                                                                                                     | 导柱、导套                                                                  |
| **推出机构**     | 在开模后将压铸件从模具型腔中顶出脱模[[S11,S3]]。                                                                                                                                     | 推杆、复位杆、推板、推杆固定板                                              |
| **抽芯机构**     | 用于成型与开模方向不一致的侧孔或侧凹，在开模前或开模过程中将活动型芯从铸件中抽出[[S11,S3]]。                                                                                         | 型芯、滑块、斜销、楔紧块                                                    |

## 核心属性
### 模具材料与硬度
压铸模具材料的选择主要取决于所生产铸件的合金类型及其浇注温度。模具型腔镶块和型芯通常采用热作模具钢，其中**H13钢**应用最广[[S22,S25,S30]]。

| 压铸合金种类     | 模具部件       | 推荐材料（举例）                 | 推荐硬度 (HRC/HB) |
| :--------------- | :------------- | :----------------------------- | :---------------- |
| 锌/铅/锡合金     | 型腔镶块       | P20 (小批量), H11, H13         | 46~50 HRC (H11/H13) / 290~330 HB (P20) |
| 铝/镁合金        | 型腔镶块       | H13, H11                       | 42~48 HRC         |
|                  | 型芯           | H13, DIN 1.2367                | 44~48 HRC (H13) / 42~48 HRC (DIN 1.2367) |
| 铜/黄铜合金      | 型腔镶块       | DIN 1.2367, H20/H21/H22        | 38~44 HRC (DIN 1.2367) / 44~48 HRC (H20系列) |

### 模具寿命
模具的使用寿命受合金类型、模具材料、设计、使用和维护等多种因素影响，不同合金间的差异巨大[[S1,S30,S7]]。

| 压铸合金种类 | 模具寿命（工作循环次数）                                                                                                                             |
| :----------- | :-------------------------------------------------------------------------------------------------------------------------------------------------- |
| 锌合金       | 最高可达 **1,000,000** 次[[S1]]。                                                                                                                    |
| 铝/镁合金    | 常规H13模具为 **5,000 ~ 25,000** 次；改用马氏体时效钢后可达 **150,000** 次[[S30]]；最高约 **100,000** 次[[S1]]。                                      |
| 铜合金       | 国内常规为 **0.5 ~ 1** 万次；国外最高可达 **6.5** 万次，个别采用特殊材料可达 **8 ~ 10** 万次[[S7]]；最高约 **10,000** 次[[S1]]。                      |
| 黑色金属     | 常规3Cr2W8V钢不超过 **350** 次；采用TZM钼基合金材料可提升至 **15,000** 次[[S7]]。                                                                    |

### 工作温度
压铸模具需要通过预热和温控系统维持在一个适宜的工作温度范围内，以保证铸件质量和模具寿命[[S1]]。

| 压铸合金 | 推荐模具工作温度 (°C) |
| :------- | :------------------ |
| 锌合金   | 218                 |
| 铝合金   | 288                 |
| 镁合金   | 260                 |

## 工艺作用与参数匹配
压铸模具并非独立工作，它必须与**压铸机**和**压铸工艺参数**精确匹配，形成一个高效的生产系统。

- **三大要素的耦合**：压铸合金、压铸模具和压铸机是压铸生产的三大要素，压铸工艺将这些要素有机整合，以获得高品质的压铸件[[S13,S5,S3]]。
- **P-Q² 图的应用**：P-Q² 图是判断压铸机与压铸模具是否匹配的核心分析工具。图中，代表压铸机提供能量的**机器性能线（ML）** 与代表模具能量需求的**模具压力线（DL）** 的交点，若落在理想的“工艺窗口”内，则表明二者匹配良好，工艺参数的可调节范围（柔性）更大[[S14,S6]]。
- **优化匹配的案例**：例如，在铸铝转子压铸模的优化中，通过增大内浇口截面积（从275.86mm²增至307.29mm²），使模具压力线DL₂在工艺窗口内的截取段变长，优化后的冲头速度可调节范围从1.66-3.61m/s拓宽，显著提升了模具与压铸机的匹配度和工艺柔性[[S10,S17,S23]]。
- **精度要求**：模具与压铸机的安装配合需满足多项精度指标，如分型面与模板的平行度、导柱导套的垂直度、合模后镶块分型面局部间隙需小于0.05mm等[[S2]]。

## 温控系统设计要点
温控系统的设计对模具寿命和铸件质量至关重要[[S32]]。

- **安全距离**：冷却水道中心距模具成型面的最小距离建议**不小于25mm**，以防止因局部急冷急热产生过高热应力而引发裂纹[[S7]]。
- **随形冷却**：现代模具设计趋向于采用随形冷却水道。以某铝合金水泵座压铸模为例，优化的随形冷却推荐参数为：水道直径1.5mm，冷却水流量25L/min，水道距型腔壁距离1.5mm。该设计能使铸件深腔区域冷却更均匀，降低局部最大温差，从而减少缺陷[[S39]]。
