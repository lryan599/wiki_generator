## 概述

“多孔板”在压铸语境下是一个涵盖多个功能子类的统称性术语，并非特指单一零件。依据国家标准 GB/T 8845-2017《模具术语》体系，其概念边界可明确界定：区别于体积较小、仅局部带微孔的排气塞，亦不同于仅开设少量大通孔实现流道均分的普通分流板，压铸用多孔板泛指那些整板分布有排气槽、透气微孔或过滤孔洞的板状功能元件[[S31,S1,S22]]。

在压铸工艺中，多孔板主要承担**透气排气**、**金属液过滤**、**冷却液分流**及**均压导流**等核心功能，是连接模具型腔质量、充型过程控制与铸件最终致密性的关键辅助组件。

## 分类

依据功能与结构两个独立维度，可对压铸用多孔板进行系统分类。

### 按功能分类

| 功能子类 | 安装位置与作用 | 典型场景 |
|:---|:---|:---|
| 透气排气类 | 装配于分型面真空排气通道末端、集渣包排气口或型腔局部易困气区域（如熔接痕处），用于排出型腔内气体以减少气孔缺陷 | 冷室压铸大型一体化压铸件快速抽真空；型腔全域均匀排气 [[S15,S3,S17,S12]] |
| 金属液过滤类 | 放置于浇注系统横浇道内，过滤金属液中的氧化膜、熔渣等非金属夹杂物，减少铸件夹渣缺陷 | 铝合金、铜合金压铸件浇道过滤 [[S13]] |
| 冷却液分流类 | 置于模具冷却通道中，起分水板或导流板作用，使冷却水流均匀分布，控制模温场 | 需精确控温的模具冷却系统 |
| 均压导流类 | 在浇注系统或料筒内引导熔体流动，使料流由螺旋运动转变为平稳直线运动，或实现多束分流以保障充型均匀 | 热压铸工形磁芯分注板，保障坯体密度均匀 [[S28,S1]] |

### 按结构分类

| 结构子类 | 工艺特征 | 材质与应用 |
|:---|:---|:---|
| 机加工实板钻孔类 | 在实心板材上通过机械钻孔加工出规则排列的通孔，孔参数可控 | 均压导流板、冷却分流板；典型板厚 12~20 mm，开孔直径 3~7 mm，孔位沿同心圆周紧凑排列，可采用中心小孔、外圈孔径渐大的差异化设计 [[S8]] |
| 烧结多孔金属类 | 采用粉末冶金工艺，将金属粉末经成形、烧结制成具有贯通孔隙的板材，孔隙率与透气率可精确调控 | 透气排气板、过滤板；材质覆盖 H13 钢、烧结铜/青铜合金、烧结不锈钢等 [[S19,S10]] |
| 金属丝网复合类 | 以金属丝网为主体或复合于基板上，形成网状多孔结构 | 低阻力过滤或气流均布场景 [[S30]] |

在压铸领域内，应特别注意将金属液过滤类多孔板（以泡沫陶瓷过滤板为核心子类）与透气排气类烧结金属多孔板加以区分，二者在材质、孔结构与安装位置上有本质差异。

## 结构与材料

### 烧结多孔金属类材质选择

粉末冶金多孔材料分类体系（GB/T 4309—2009 中类 F2）覆盖了压铸多孔板常用的烧结金属材质[[S19]]：

- **H13（4Cr5MoSiV1）热作模具钢**：适用于高温度载荷的压铸模透气多孔板工况，利用其优异的高温强度与抗热疲劳性能[[S10]]。
- **烧结铜/青铜类**：适用于工作温度低于 500℃ 的气液过滤及均匀流场场景[[S19]]。
- **烧结不锈钢类**：适用于接触酸碱腐蚀性流体或要求高耐蚀性的工况[[S19]]。

### 机加工开孔式多孔板结构参数

根据塑料异型材挤出模具领域的设计规范（该结构设计原则对压铸均压导流类多孔板具有直接参考价值），常规参数范围如下[[S8]]：

| 参数 | 典型范围 |
|:---|:---|
| 板厚 | 12~20 mm |
| 开孔直径 | φ3~φ7 mm |
| 孔位排列 | 多个同心圆周上紧凑排列 |
| 特殊设计 | 中心部位取较小孔径，外圈逐步放大孔径，以适应流场分布需求 |
| 进料端要求 | 入口倒成锐角，避免滞留区与尖角挂料 |

### 均压导流类典型实例

《中国模具设计大典》收录的热压铸工形磁芯分注板，属于典型的均压导流类机加工实板钻孔型压铸多孔板。其分布通孔尺寸为 φ0.8~φ1.5 mm，板厚 6 mm，用于保障压铸坯体密度均匀与外观质量[[S28]]。

## 工艺作用与工作原理

### 透气排气功能

在高压压铸充型过程中，型腔内预先存在的空气以及润滑剂挥发气体若不能及时排出，将在铸件内形成气孔缺陷。采用多孔性金属嵌件（透气板）作为模具的局部透气结构，可在型腔整个表面上实现全域均匀排气，特别是在料流前锋会合的熔接痕等最易困气区域效果显著。相比单独设置排气槽的方案，透气多孔板可提升充型速度自由度，简化浇口布置，并防止铸件表面出现因排气不畅造成的纹理缺陷[[S12]]。

透气多孔板的排气机制可通过包含渗透系数 K 的 Ergun 方程进行数值仿真描述，其渗透系数 K 可由 Kozeny-Carman 公式推导得出，为透气性能的参数化设计提供了理论支撑[[S29]]。

### 金属液过滤功能

泡沫陶瓷过滤板（ceramic foam filter）的孔洞曲折，能有效阻止非金属夹杂物流过。其核心作用机制是通过曲折的三维孔隙网络截获金属液中的氧化膜、熔渣等杂质，同时保持足够的金属液流通能力[[S13,S14]]。

## 关键性能参数

### 泡沫陶瓷过滤类多孔板参数

根据《中国材料工程大典》及液态成型工艺文献，用于铝合金、铜合金压铸场景的氧化铝基泡沫陶瓷过滤板性能参数如下[[S13,S14,S9]]：

| 参数项 | 取值或范围 |
|:---|:---|
| 孔隙率 | 70%~95%（铝铜合金用板为 70%~90%） |
| 透气率 | (400~8000)×10⁻⁷ cm² |
| 常规厚度 | 10~20 mm（浇注系统用） |
| 孔径（细孔档） | φ0.5~φ0.8 mm |
| 孔径（粗孔档） | φ1.2~φ1.9 mm |
| 铝铸件过滤面积要求 | ≥ 4~8 倍横浇道截面积 |
| 铜铸件过滤面积要求 | ≥ 2~4 倍横浇道截面积 |

微孔透气隔板（原用于流化床涂敷工艺，其性能指标对理解压铸透气类多孔板选型具有参考价值）的性能参数范围如下[[S6]]：

| 性能指标 | 参考范围（涵盖 LS-1 至 LS-12 及 S-C1 至 S-C3 型号） |
|:---|:---|
| 最大孔径 | 微米级（具体数值因型号而异） |
| 透气度 | 型号间差异显著 |
| 耐压强度 | 型号间存在梯度分布 |
| 气孔率 | 与孔隙率等级对应 |
| 劈裂抗拉强度 | 反映多孔结构力学强度 |

## 常见缺陷与失效模式

压铸多孔板在服役过程中面临高温金属液、交变热应力、冲蚀及配合精度的多重考验，主要失效机制如下：

| 失效类型 | 成因机制 | 限制条件与后果 |
|:---|:---|:---|
| 铝屑/氧化物堵塞 | 飞溅的细小铝屑、合金熔体表面氧化薄膜碎渣进入孔隙内部，逐步填充闭塞透气通道[[S4,S27]] | 孔隙堵塞率超过 40% 后完全丧失排气功能，直接导致铸件出现卷气缺陷[[S4,S27]] |
| 热疲劳龟裂 | 长期处于压铸开合冷热交替循环工况，材料表面反复承受热胀冷缩产生交变热应力，逐步萌生细微裂纹并扩展为龟裂形貌[[S18,S2]] | 服役热循环次数超过材料热疲劳阈值后失效概率显著上升[[S18]] |
| 高温变形 | 长期承受高于材料许用温度的高温金属液热冲击，材料屈服强度随温度上升衰减，在正向压射压力下产生不可逆塑性变形[[S18,S11]] | 变形量超过 0.2 mm 后密封面出现间隙，发生金属液窜漏风险[[S18,S11]] |
| 金属液腐蚀 | 高温高速流动的铝合金、镁合金熔体持续冲刷多孔板表面，发生力-热-化学反应耦合作用，形成金属间化合物层并不断剥落[[S21]] | 孔隙表面逐步被侵蚀破损，降低有效透气面积 |
| 安装配合失效 | 安装面加工形位公差超标、装配预紧力不足或模具热膨胀预留间隙不合理，长期循环受力后出现多孔板移位、脱落[[S26,S25]] | 配合间隙超过 0.1 mm 时出现漏渣、窜铝故障[[S26,S25]] |

压铸多孔板堵塞失效后，将直接引发铸件多孔性缺陷，其成因与防控措施在压铸学文献中有系统梳理。缺陷的 4 类核心诱因包括：合金注入温度过高、注入合金量过少、脱模剂/润滑剂喷涂过量、氮气储能桶压力不足[[S24]]。

## 应用场景与安装位置

### 不同压铸机类型的安装位置

| 压铸机类型 | 多孔板典型安装位置 |
|:---|:---|
| 冷室压铸机 | 分型面的真空排气通道末端、集渣包排气口位置[[S15,S3,S17]] |
| 热室压铸机 | 鹅颈通道末端、模具溢流槽排气口位置[[S15,S3,S17]] |

### 不同合金压铸场景的适配要求

| 合金类型 | 多孔板选型特点 |
|:---|:---|
| 铝合金 | 适配高排气量的大孔径透气多孔板，满足大型一体化压铸件型腔快速抽真空需求[[S3,S16]] |
| 镁合金 | 适配耐镁液腐蚀的高致密多孔板，配合保护气密封要求[[S3,S16]] |
| 锌合金 | 适配高精度小孔隙多孔板，满足薄壁锌合金饰品类压铸件的微排气需求[[S3,S16]] |

## 相关标准与配合要求

压铸多孔板作为功能元件，其安装配合需满足 GB/T 4678 系列压铸模零件标准规定的公差等级要求。该系列标准涵盖模板、镶块、导柱导套等 19 部分内容，对尺寸公差、形位公差及材料均有明确技术规范[[S26,S25,S7]]。多孔板与安装槽之间的配合间隙控制是防止漏渣、窜铝等故障的关键，配合间隙超过 0.1 mm 即存在失效风险[[S26,S25]]。

ISO 标准体系下，有色金属铸件用撇渣多孔板的工程示意图为压铸领域同类产品的尺寸设计提供了国际标准参照基准[[S23]]。
