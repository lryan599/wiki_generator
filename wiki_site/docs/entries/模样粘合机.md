## 概述

模样粘合机是消失模（实型）铸造白区工序中用于将分块成形的独立泡沫模片高精度组装为核心完整泡沫模样的专用模样组装类装备，中文亦常称作消失模粘模机或泡沫模样粘合设备，英文对应术语包括 pattern bonding machine、pattern gluing machine、pattern assembly machine 及 lost foam pattern gluing equipment [[S2,S18,S3]]。该设备利用机械定位与自动施胶手段，保证多块模片在粘合方向上的尺寸精度与接缝质量，是消失模铸造中连接模样分片与后续涂料涂挂工序的关键节点。

自动模样粘合机按照粘合方式分为热粘合机和冷粘合机两大类，分别对应热熔胶和冷粘胶两种粘合体系。行业内的典型商用产品包括美国 Vulcan 公司热粘合机、德国 Common 公司冷粘合机以及德国 Teubert 公司冷粘合机等 [[S2,S18,S20,S12]]。图 1 展示了热粘合机与冷粘合机的典型外观，直观反映了两种机型在结构布局上的差异。

## 分类与特性

模样粘合机的两大技术路线在适用场景、粘合效率与灵活性等方面存在显著差异，具体对比见表 1。

| 对比维度 | 热粘合机 | 冷粘合机 |
|:---|:---|:---|
| 配套粘合剂 | 热熔胶 | 有机溶剂型冷粘胶 |
| 施胶方式 | 随形涂胶印刷板转印 | 六轴机械手注射涂胶 |
| 适用粘合面形态 | 起伏较小的平面粘合面 | 复杂曲面、高精度拼接面 [[S9,S13,S3]] |
| 涂胶工装更换 | 需更换涂胶印刷板，操作繁琐 [[S20,S10]] | 仅需切换涂胶轨迹程序 [[S9,S12]] |
| 多品种适应性 | 较差，单台不适应多品种小批量 [[S20,S10]] | 优良，适配多品种泡沫模片 [[S9,S3]] |
| 典型单周期耗时 | ≤60 s [[S12,S11]] | 较长，机械手涂胶与烘干硬化耗时较多 [[S9,S13,S3]] |
| 生产效率 | 60～100 操作循环/h [[S12,S11]] | 低于热粘合机 [[S9,S13,S3]] |
| 主要局限 | 不适用于大起伏折平面或曲面；耗电较高；烟气影响环境 [[S20,S10]] | 整体粘合效率较低 [[S9,S13,S3]] |

## 结构与组成

### 热粘合机

热粘合机的核心组成部分包括 [[S19,S10,S3]]：

- **机械主体**：由上/下随形胎模、可移动机架及升降驱动缸构成，上胎模设有负压管路用于固定上模片，切换后可通压缩空气辅助脱模；
- **胶液供给系统**：包括 100～120 ℃ 恒温热熔胶熔池和耐热模具钢制造的随形涂胶印刷板。印刷板凸台高度为 5～10 mm，窄粘合面（5～10 mm）采用单凸台，宽粘合面（>10 mm）采用双凸台，以保证胶层外凸披缝高度控制在 0.5～1.5 mm 范围内 [[S19]]；
- **定位夹紧机构**：胎模定位销实现模片初定位，粘合压力通过上胎模型腔高度尺寸低于泡沫模片 0.1～0.2 mm 实现，合模后对两块泡沫模片产生微量压缩，使粘合面贴紧 [[S10]]；
- **控制系统**：包含温控单元与移动行程控制器；
- **溢胶控制**：胎模型腔分型面周边设有 3 mm×3 mm 的凹槽，用于收集粘合面挤出的多余热熔胶，防止胶液粘连胎模 [[S10]]。

### 冷粘合机

冷粘合机的核心组成部分包括 [[S9,S13,S5]]：

- **机械主体**：双层密封结构的上下胎模（内胎模加外封框），上胎模同样设有负压吸嘴与压缩空气管路，下胎模设有定位销与热气管接口；
- **涂胶系统**：六轴涂胶机械手，通过压力式冷胶注射装置实施注射涂胶，涂胶量由注射压力精准控制，可通过编程自定义三维涂胶轨迹；
- **热风供给系统**：将 50～60 ℃ 定向热风导入下胎模，加速有机溶剂型冷胶中溶剂挥发，提高粘合强度与效率；
- **控制系统**：支持自定义三维涂胶轨迹编程，适配多品种复杂曲面模样粘合。

## 工作原理与工作循环

### 热粘合机

热粘合机的标准作业流程如图 3 所示，完整工作循环分为以下步骤 [[S12,S11,S16,S19]]：

1. **上料定位**：人工将两块待粘合泡沫模片分别置入上、下胎模，上胎模的负压管路通过吸嘴固定上模片；
2. **转印涂胶**：上胎模移动至热熔胶熔池工位，涂胶印刷板升起与上模片粘合面接触保持 1～2 s，完成热熔胶转印涂胶后回落至熔池，多余胶液直接回流熔池；
3. **合模保压**：上胎模移动复位，下胎模由升降缸抬升完成合模，两块模片精准对接并保压 5～15 s 实现热熔胶固化定型；
4. **脱模取件**：开模后释放负压，切换为压缩空气管路输出正压，将已粘合的成品泡沫模样从上胎模型腔中吹脱分离，人工取出成品。

整个操作单周期一般不超过 60 s，1 h 内可完成 60～100 个操作循环 [[S12,S11]]。

### 冷粘合机

冷粘合机的完整工作循环如下 [[S9,S13,S14]]：

1. **上料定位**：人工将两块待粘合泡沫模片分别置入带定位销的上、下双层胎模，上胎模通过负压吸嘴固定上模片；
2. **机械手涂胶**：六轴涂胶机械手沿预先编程的三维涂胶轨迹运行，通过注射器将冷胶逐点注射涂抹到两块模片对应的粘合面上；
3. **热风烘干**：将 50～60 ℃ 的热风从下胎模的热气管导入，加速冷胶中有机溶剂挥发；
4. **合模保压**：升降缸抬升下胎模完成精准合模，保持保压状态使胶层充分固化定型；
5. **脱模取件**：开模后关闭负压，切换为压缩空气管路输出正压，将粘合完成的完整泡沫模样从上胎模型腔中吹脱分离，人工取出成品。

冷粘合机无需更换涂胶工装，仅通过修改涂胶轨迹程序即可快速切换生产不同规格的泡沫模样，适配汽车发动机缸体缸盖等高端消失模铸件的复杂曲面模片粘合 [[S9,S3]]。

## 关键工艺参数

热粘合机与冷粘合机的主要工艺参数见表 2。

| 参数项目 | 热粘合机 | 冷粘合机 |
|:---|:---|:---|
| 粘合剂类型 | 热熔胶 | 有机溶剂型冷粘胶 |
| 工作温度 | 熔池 100～120 ℃ [[S10,S12]] | 热风 50～60 ℃ [[S12,S9]] |
| 粘合压力形成方式 | 上胎模型腔高度低于模片 0.1～0.2 mm [[S10]] | 通过胎模合模施加 |
| 涂胶接触时间 | 1～2 s [[S11,S12]] | 机械手按编程轨迹执行 |
| 保压时间 | 5～15 s [[S11,S12]] | 视粘合面复杂度与烘干效果而定 |
| 单周期耗时 | ≤60 s [[S12,S11]] | 高于热粘合机（机械手涂胶+烘干耗时较长） |
| 粘合负数 | 偏大（热熔胶粘度较高） [[S2,S18]] | 偏小（冷胶粘度相对较低） [[S2,S18]] |

**粘合负数**是为补偿粘合剂固化后的胶层厚度 δ 而在发泡成形模具上预设的尺寸减量。一般取值范围为 0.1～0.3 mm，采用热熔胶粘合时取值偏大，采用快干胶或冷胶粘合时取值偏小；机械粘合方式的粘合负数整体小于手工粘合 [[S2,S18]]。粘合负数可对称分摊于上、下模片的成形模具，也可仅在其中一侧考虑。

## 材料兼容性与精度控制

模样粘合机可兼容的待粘合对象为 EPS 泡沫模样。粘合剂的选型需满足对泡沫模样无腐蚀、软化点适中、快干性能好、粘合强度足够且具有适当柔性以及气化残留少的基本要求，可分为热熔胶、水溶胶、溶剂挥发胶和双组分胶四大类 [[S2]]。

粘合精度控制方面，厚实处建议采用凸销与凹孔配合定位，薄壁处采用凸缘与凹槽配合定位 [[S6]]。对于壁厚 5～6 mm 的薄壁模片，可在对接处增厚至 7～8 mm 以提高粘合强度，且优先采用外壁增厚方案以降低发泡模具的加工难度 [[S6]]。

## 应用场景与流程定位

模样粘合机仅应用于消失模（实型）铸造的白模制备工序，现有公开专业资料未记载其在压铸模具制造或压铸周边常规工序中的应用 [[S17,S4]]。其在消失模铸造全流程中的位置如下：

- **上游工序**：对接发泡成形机产出的独立分块泡沫模片；
- **配套辅件**：模片组装台、专用粘合胎模（夹具）、热粘合涂胶印刷板或冷粘合涂胶机械手等 [[S8,S4]]；
- **下游工序**：输出完成组装的整体泡沫模样，后续接入模样干燥工序与涂料涂挂工序 [[S4]]。

典型应用场景包括发动机缸体缸盖、复杂箱体铸件等需要分片发泡成形后粘合组装的消失模铸件。

## 常见缺陷与局限性

| 缺陷/限制 | 描述 | 解决方案 |
|:---|:---|:---|
| 粘接不牢/局部强度不足 | 热粘合机中热熔胶在印刷板斜面上流淌，影响相应部位粘合强度 [[S10]] | 采用冷粘合机处理起伏面；选用适当粘度的冷粘胶附着于倾斜面 [[S9]] |
| 粘合错位 | 模片对接精度不足 | 在粘合面上设计凸凹镶嵌结构，厚实处用凸销凹孔，薄壁处用凸缘凹槽 [[S6]] |
| 胶液溢出 | 溢出的热熔胶黏附胎模工作面上影响后续作业 | 胎模型腔分型面周边加工 3 mm×3 mm 溢胶凹槽收集多余胶液 [[S10]] |
| 热粘合机烟气 | 粘合过程中释放有害烟气 [[S20,S10]] | 改善车间通风；必要时选用冷粘合工艺 |
| 多品种小批量生产适配性差 | 热粘合机更换涂胶印刷板繁琐，灵活性低 [[S20,S10]] | 冷粘合机通过修改涂胶轨迹程序即可快速切换品种 [[S9,S12]] |
| 冷粘合机效率偏低 | 机械手涂胶与热风烘干耗时较长 [[S9,S13,S3]] | 合理安排流水作业；对快速粘接需求选用热粘合机 |
