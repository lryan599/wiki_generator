## 概述

压铸工艺参数（die casting process parameters），全称压力铸造工艺参数，是压铸生产中将压铸机、压铸模与压铸合金三大要素有机结合并加以综合运用的过程中，对压铸件质量、生产效率及模具寿命起关键作用的各类可控因素的集合[[S31,S44,S33]]。行业内有时也将相关术语统称为压射参数（shot parameter／injection parameter），但严格来讲，“压射参数”仅指压射冲头运动过程中的射料相关参数，是压铸工艺参数的子集；而压铸工艺参数还涵盖模具热平衡、留模时间等非压射阶段的参数[[S44,S13,S11]]。英文语境下，shot parameter 与 injection parameter 在压铸领域均常被译为“压射参数”，injection speed 则对应“压射速度”[[S10,S24]]。

压铸工艺参数的核心类别通常包括压力、速度、温度和時間四大类；不同参数之间的合理匹配是获得合格压铸件的前提[[S31,S33]]。

## 分类

### 按物理属性分类

根据参数的物理属性，可将压铸工艺参数划分为五类[[S31,S16,S48]]：

- **温度参数**：浇注温度、模具温度
- **压力参数**：压射压力、增压压力、压射比压、锁模力
- **速度参数**：慢压射速度、快压射速度（冲头速度）、内浇口充填速度（内浇口速度）
- **时间参数**：充填时间、持压（保压）时间、留模时间
- **位置参数**：压室行程、压射速度切换位置

### 按工艺阶段分类

按压铸充型与凝固阶段，可将工艺参数划分为三个阶段[[S32,S53,S55]]：

- **填充阶段参数**：慢压射速度、快压射速度、充填时间、压射压力（动态压射比压）、压室行程
- **补缩阶段参数**：增压压力（增压比压）、持压时间
- **热平衡参数**：浇注温度、模具温度、留模时间、浇注量

## 各工艺参数详细解析

### 压力参数

压力是压铸区别于其他铸造方法的主要特征，表现为压射力与比压两种形式[[S44]]。比压又分为填充阶段的压射比压与增压阶段的增压比压。选择比压需综合考虑铸件结构特征、合金特性、浇注系统及质量要求[[S22]]。

不同铸件类型与合金的压射比压推荐范围如下表所示[[S35,S14,S6]]：

| 合金 | 铸件类型 | 压射比压（MPa） |
|---|---|---|
| 锌合金 | 一般件 | 13～20 |
| 锌合金 | 承载件 | 20～30 |
| 锌合金 | 耐气密性件／大平面薄壁件 | 25～40 |
| 铝合金 | 一般件 | 30～50 |
| 铝合金 | 承载件 | 50～80 |
| 铝合金 | 耐气密性件／大平面薄壁件 | 80～120 |
| 镁合金 | 一般件 | 30～50 |
| 镁合金 | 承载件 | 50～80 |
| 镁合金 | 耐气密性件／大平面薄壁件 | 80～100 |

镁合金热室压铸的推荐压射比压略低：受力件20～30 MPa，耐压或大平面薄壁件15～25 MPa[[S14]]。冷室压铸机的增压压射比压整体高于热室压铸机[[S14]]。对于承载件（如汽车发动机缸体），压射比压一般取50～80 MPa，增压比压可达300 MPa甚至更高[[S22,S42]]。

**对质量与模具寿命的影响**：压射压力过低会导致铸件致密度不足，产生气孔和缩孔；压力过高虽能提高致密度，但会增加飞边毛刺、模具磨损以及在复杂区域形成内部缺陷带的风险[[S30,S58]]。实验表明，随压射比压增大，压铸模具的疲劳寿命明显下降，同时冲头与压室之间的磨粒磨损显著加重[[S25]]。

### 速度参数

压铸速度主要分为**压射速度**（冲头速度）和**充填速度**（内浇口速度）。压射速度又分为慢速和快速（快压射速度）两个阶段[[S31]]。

常用压铸合金的内浇口充填速度推荐范围如下[[S39,S5,S14]]：

| 合金 | 推荐内浇口速度（m/s） |
|---|---|
| 铝合金 | 17～40 |
| 锌合金 | 30～60 |
| 镁合金 | 40～50（冷室常规件40～90，薄壁件可达80～100） |

锌合金和铝合金的常规充填速度一般为40～60 m/s，过高容易导致模具烧蚀[[S14]]。镁合金因密度小、凝固快，冷室压铸充填速度通常为40～90 m/s，薄壁件（如笔记本电脑外壳）可达80～100 m/s，而热室压铸的高速压射速度可达6 m/s，冷室最高可达8 m/s[[S14,S57]]。

**对质量的影响**：充填速度过低会导致浇不足、冷隔；速度过高则金属液易发生喷溅、雾化、翻卷，卷入大量气体形成气孔，且会增加模具冲蚀[[S30,S51]]。

### 温度参数

温度参数包括**浇注温度**和**模具温度**。

镁合金的推荐浇注温度：热室压铸约640 ℃，冷室压铸约680 ℃[[S14]]。铝合金的浇注温度通常在650～690 ℃范围内选取，具体值取决于产品壁厚与复杂度[[S17,S56]]。

模具工作温度对铸件质量和模具寿命至关重要。镁合金的推荐模具温度为200～250 ℃，连续压铸40次后可获得稳定的热平衡温度[[S6]]。铝合金模具预热温度通常设定在浇注温度的1／3±25 ℃区间，如浇注温度650 ℃时，模温约191～242 ℃[[S56]]。

**对质量与生产效率的影响**：浇注温度适当升高可提高合金流动性、改善表面质量，但会增大凝固收缩率，增加缩孔、缩松倾向；温度过低则直接导致冷隔、欠铸[[S17,S30]]。模具温度过低时，极易产生欠铸、冷隔和流纹；模温过高会引起粘模、表面气泡及收缩缺陷，同时延长铸件凝固时间，降低生产节拍[[S27,S30]]。

**对模具寿命的影响**：模具型腔表面周期性接触高温金属液，若模温控制不当，冷热交变温差过大，将产生交变热应力，导致热疲劳微裂纹萌生并扩展，大幅缩短模具服役寿命[[S27,S50]]。

### 时间参数

时间参数主要包括**充填时间**、**持压（保压）时间**和**留模时间**。

**充填时间**（t_f）是金属液自内浇口开始进入型腔至充满型腔所经历的时间。锌合金的充填时间推荐为6～40 ms，其中机械受力件或需电镀处理的工件应小于20 ms，需喷漆表面处理的工件应小于40 ms[[S28]]。铝合金和镁合金的充填时间常根据壁厚由经验公式计算（见后续优化工具部分）。

**持压时间**（t_h）及**留模时间**（t_r）的推荐值如下表（锌合金）[[S28]]：

| 壁厚（mm） | 持压时间（s） | 留模时间（s） |
|---|---|---|
| ＜2.5 | 1～2 | — |
| 2.5～6 | 3～7 | — |
| ＜3 | — | 5～10 |
| 3～4 | — | 7～12 |
| ＞5 | — | 20～25 |

持压时间一般应大于0.5 s，确保金属在压力下完成凝固补缩[[S28]]；留模时间不足会在开模顶出时导致铸件变形或拉裂[[S28,S58]]。

### 其他参数

除上述四类核心参数外，压铸工艺参数还包含**浇注量**（压室充满度）、**压室行程**、**锁模力**等扩展要素[[S31,S52,S56,S26]]。

锁模力（F_锁）是选用压铸机的首要参数，其计算公式为[[S39,S40,S54]]：

F_锁 = K × (F_主 + F_分)

其中，K为安全系数，通常取1.25[[S39,S40]]；主胀型力 F_主 = A × p，A为铸件（含浇注、溢流系统）在分型面上的投影面积，p为压射比压[[S40,S54]]；分胀型力 F_分为各侧向抽芯机构产生的胀型力在锁模方向的投影之和，当侧向成型面积较小时可忽略[[S40]]。国标GB/T 10925规定了压铸机的锁模力额定规格：热室压铸机覆盖630～6300 kN，卧式冷室压铸机覆盖630～25000 kN[[S9,S37]]。

## 参数交互作用与优化工具

### 充填时间与壁厚关系

充填时间与平均壁厚之间存在经验关系，常用公式为[[S43,S8,S42]]：

t_fl = k × (T_g − T_f + S × Z)／(T_f − T_d) × T_cw

式中，k为经验常数（铝合金取0.0346 s／m），T_g为内浇口处合金入模温度，T_f为合金液相线温度，S为充填结束时允许的固相占比，Z为单位换算系数，T_d为入模前模具预热温度，T_cw为铸件平均特征壁厚。

### PQ²图与工艺窗口

**PQ²图**（PQ squared diagram）是基于伯努利定律构建的压铸系统分析线图，其横轴为流量Q的平方，纵轴为压力P[[S59,S1,S60,S45]]。图上包含**压铸机特性线**（ML，表示设备可输出的能量）和**模具特性线**（DL，表示浇排系统所需能量），两条线的交点即为实际生产工作点[[S1,S59,S60]]。预先设定的最小／最大充填时间与最小／最大内浇口速度四条边界线共同围合成**工艺窗口**；工作点处于窗口中部时，系统匹配性最优，工艺调整裕度充足[[S34]]。

![压铸PQ²特性分析图（含压铸机线、模具线及工艺窗口）](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/1c7655fe-9d62-4557-8ffe-ad150a5e5660/resource)

PQ²技术的应用存在一定局限：流量系数（开放系数Cd）受合金类型影响，铝合金、镁合金的Cd经验值约为0.5，锌合金约为0.6，实际生产中难以完全精确计算，需工艺人员结合经验预估[[S7,S1]]。

## 冷室压铸与热室压铸的工艺差异

冷室和热室压铸在压力、速度、温度及适用合金上存在显著差异[[S15,S19]]。

| 对比项目 | 冷室压铸 | 热室压铸 |
|---|---|---|
| 动态压射比压 | 30～90 MPa | 20～50 MPa |
| 增压比压 | 50～300 MPa，设有独立增压液压缸 | 一般无独立增压机构，压力损失大 |
| 常规冲头速度 | 最高可达11 m/s | 1.6～2.5 m/s |
| 压射阶段 | 三阶段或四阶段压射 | 主要为两阶段压射（一速升液、二速填充） |
| 浇注温度 | 镁合金约680 ℃，铝、铜等更高；温度波动大，氧化夹杂风险较高 | 镁合金约640 ℃，温度稳定，氧化夹杂风险低 |
| 适用合金 | 高熔点合金（铝、镁、铜等） | 低熔点低腐蚀性合金（锌、铅、部分镁合金） |
| 典型应用 | 大尺寸厚壁结构件 | 1 kg以下小尺寸薄壁件 |

热室压铸机的压室与冲头长期浸没在金属液中，金属液在封闭管路内流动，温度波动小，可使用较低的浇注温度，且卷气机会少；但受机件耐热和耐侵蚀能力限制，主要适用于锌、铅及部分低熔点镁合金的薄壁件生产[[S47,S15]]。冷室压铸通过独立舀料、冷压室供料，与金属液接触时间短，可适应高熔点合金，但每模温度波动较大、氧化夹杂比例更高[[S47,S15]]。

## 大型厚壁件与小型薄壁件的参数设置特点

**大型厚壁压铸件**（如发动机缸体）通常采取以下策略[[S23,S49]]：
- 适当降低浇注温度，使金属液呈近凝固点的“粥状”充型，减少体收缩；
- 适当延长充填时间，为型腔气体排放留出路径，降低内部气缩孔概率；
- 设置较高的增压比压（远高于常规件），保证压力通过仍未凝固的内浇口传递至厚壁区域以抵消缩松倾向；
- 适当延长持压时间，保证厚壁区域完全凝固后再开模。

**小型薄壁压铸件**（如镁合金笔记本电脑外壳）则采取相反措施[[S46,S29]]：
- 采用高内浇口速度（镁合金薄壁件充填速度可达80～100 m/s），确保在合金凝固前快速充满型腔，避免冷隔和浇不足；
- 适当提高压射比压，降低合金液流动阻力以提升充型能力；
- 采用宽而薄的扁形内浇口，使金属液呈薄层高速喷射状态，防止充填前沿提前凝固。

## 工艺参数选择与优化原则

压铸工艺参数的确定需遵循以下原则（综合自[[S22,S31,S44]]）：
1. 根据铸件的结构特征（壁厚、复杂程度）、合金的流动性与凝固特性初选压力、速度与温度的基础值；
2. 结合锁模力计算与PQ²图进行压铸机选型，优先选择工作点处于工艺窗口中部、调整裕度充足的机型（NADCA推荐方法）[[S2,S41]]；
3. 通过浇注系统与排气系统设计间接调整充填时间与内浇口速度，并通过数值模拟（如MAGMAsoft）验证流场、「卷气率」与「缩孔率」[[S30,S51,S8]]；
4. 在生产过程中监控压射曲线（压射速度-位置曲线、压力-時間曲线）的实际形态，及时判断参数偏离与潜在缺陷[[S58,S31]]。

研究表明，卷气率主要受压射速率和增压压力影响，缩孔率受浇注温度的影响最大，缺陷带的形成则受增压压力和模具温度的共同控制[[S30,S51]]。因此，任何单一参数的调整都须与其关联参数协同变动，以获得综合最优的铸件质量与生产效率。
