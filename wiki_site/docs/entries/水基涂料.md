## 概述

在压铸生产中，水基涂料是指以水为连续相（载体），由基础油、润滑剂、乳化剂及多种功能添加剂经乳化工艺制成的消耗型辅助材料[1]。行业通用定义为：以水为连续相，由硅油、合成蜡、合成烃、乳化剂、添加剂等组分乳化构成，喷涂到150~300℃高温压铸模具表面后水分蒸发形成微米级隔离薄膜，核心作用是在每次压射循环实现铸件顺利脱模。水基涂料自20世纪70年代后期逐步工业化普及，目前占据压铸涂料市场90%以上份额[[S21,S42,S18]]。

需注意，压铸语境下的“水基涂料”通常等同于**水基脱模剂（水基乳液涂料）**，但行业内还存在一种**水基半永久模具保护涂层**，二者有本质差异：

| 对比维度 | 常规水基脱模剂（消耗型） | 水基半永久模具保护涂层 |
|---------|----------------------|---------------------|
| 喷涂频率 | 每次压射循环前必须喷涂 | 喷涂一次可支撑数十至上百模次 |
| 核心功能 | 单次脱模润滑与隔离 | 长期强化模具表面抗焊合、抗热疲劳性能 |
| 膜层寿命 | 单次成膜仅支撑1~数个压射件 | 长效，无需每模次补喷 |
| 典型产品 | 市场主流通用水基乳液涂料 | 如美国Axel实验室Xtend-W7283A |

常规水基脱模剂属于每次压射循环前必须喷涂的消耗型润滑介质，而水基半永久模具保护涂层属于喷涂一次可支撑数十至上百个模次的长效特种功能涂层[[S1, S4, S17]]。本词条主要阐述前者。

## 组成与配方特征

水基涂料以水为连续相，起主要润滑隔离作用的油相以微小液滴形式分散于水中[[S3]]。其基本组成为矿物油、石蜡、加上添加剂等十几种原料经乳化制成[[S24]]。

**水基涂料按基本成分可分为三类**[S4,S26,S39]：

1. **蜡系材料为主体**：分别加以油性或其他润滑剂乳化而成，适配中小型铸件及常规吨位压铸机；
2. **含硅有机物为主体**：耐热脱模性能优异，适配大吨位压铸机及厚壁铸件；
3. **同时含上述两类成分**：综合性能均衡。

**典型配方示例**（Lub-2型水基乳液涂料，质量分数%）[S19]：

| 组分 | 含量 | 功能 |
|------|------|------|
| 改性有机聚硅氧烷 | 14.4 | 高温润滑、脱模 |
| 蜂蜡 | 4.3 | 成膜、润滑 |
| 氧化聚乙烯蜡 | 2.9 | 成膜、润滑 |
| Span80 | 1.3 | 乳化剂 |
| OP-10 | 4.1 | 乳化剂 |
| 增稠剂 | 0.43 | 调节粘度 |
| 2,6-二叔甲基对甲酚 | 0.14 | 抗氧化剂 |
| 杀菌防腐剂 | 0.14 | 防腐 |
| 消泡剂 | 0.29 | 消泡 |
| 水 | 72.0 | 载体 |

在上述配比及工艺条件下制得的水基乳液涂料高温润湿温度320℃且成膜性较好，容易在金属表面形成一层具有一定强度的均匀润滑膜，有效润滑成分在高温下发气平稳且持续[S19]。

**关键组分功能**[S26]：

- 配方中蜡类成分占比过高会导致难分解物在模具表面积碳残留；
- 生物油添加过量会导致模具表面发粘；
- 使用低质廉价硅油会导致喷涂成膜不良、铸件表面产生气泡缺陷。

## 工艺操作与控制参数

### 稀释要求

浓缩型商用压铸水基涂料使用前需加水稀释，行业常规稀释比例为40:1，高端产品最高稀释倍率可达1:(150~200)，部分特殊场景下最高可稀释至100倍，稀释后水相占比普遍高于90%[[S30,S2]]。

**推荐稀释比例范围**[S24,S7]：

- 一般工况：兑100~150倍水；
- 大件、薄壁、复杂铸件：选用更低稀释倍数；
- 小件、简单铸件：选用更高稀释倍数；
- 实验室验证最优稀释比例：特定配方为80:1[S9]；
- 特殊高负荷压铸场景：可低至30:1[S14]。

**稀释用水要求**：需严格管控稀释用水的纯净度、软硬度、pH值及稀释温度；水中适量钙离子可提升涂料粘附性能，钙离子浓度过低则粘附性差，过高则降低合金液流动性；优质涂料的稀释用水若钙离子浓度过低则涂料粘附性能较差，过高则影响合金液的流动性[S19]；开封兑水后需在一周内用完[[S13,S19]]。

### 喷涂工艺

水基涂料**禁止采用手动涂刷方式**操作，涂刷时涂料遇高温模具表面水分急剧汽化，无法有效粘附在型腔表面形成有效润滑膜，必须采用喷枪雾化喷涂工艺，可分为人工手动喷涂和往复式自动喷涂两类[[S13,S16,S7]]。

**推荐喷涂参数**[S7,S9]：

| 参数 | 推荐值 |
|------|--------|
| 喷涂压力 | 0.3~0.5 MPa（最佳0.5 MPa） |
| 喷涂距离 | 100~200 mm（特殊点喷可缩短至70 mm） |
| 喷涂时间 | 0.1~3 s |

喷涂压力过小则雾化不良，涂液成股流下；喷涂压力过大则卷入大量空气，降低成膜质量[S9]。

### 模具温度窗口

水基涂料适配的模具工作温度推荐区间为180~280℃[[S24,S7]]：

- 模温低于150℃时，涂料水分无法完全汽化，会发生流淌无法成膜，导致铸件产生气孔缺陷[S24]；
- 模温高于398℃时，涂料受Leidenfrost效应影响会直接蒸发反弹，无法润湿模具表面形成有效润滑膜[S24]；
- 一般水基涂料润湿温度区间为200~300℃[S18]；
- 部分高润湿性水基涂料的临界润湿温度可达320~325℃，适配更高模温的压铸工况[[S41,S19,S29]]。

当模温低于270℃时，喷涂后1s内即可完成全部水分汽化，形成完整的隔离润滑薄膜[[S24,S7,S16,S47]]。

## 优点与局限性

### 优点

- **环境友好**：无烟雾污染，一氧化碳、碳氢化合物等有害气体排放量大幅降低，作业环境更友好[[S21,S26,S42]]
- **铸件质量优异**：铸件表面光亮、无油基涂料燃烧残留的暗灰色斑痕与沟纹缺陷；内部致密度可提升0.1%~1.7%[[S21,S46]]
- **模具保护**：不腐蚀模具，冷却效果强，有利于延长模具寿命[[S21,S42]]
- **表面后处理兼容性**：残留物少，铸件表面质量好，对后续涂装、电镀等工艺影响小[S26]

### 局限性

- **脱模稳定性**：相比高端油基涂料，脱模稳定性略低；对于复杂、深腔压铸件的生产，还需配合使用油性的强力脱模剂或抗焊合蜡[S46]
- **模具温度敏感性**：对模具温度要求严格，过高或过低均无法有效成膜[[S24,S18]]
- **残留硅油的影响**：浇注后残留的硅油附着在模具表面，会大幅降低水的热态润湿性；当型腔表面残留质量分数仅0.2%的乳化硅油时，水的雷顿点会从250℃降至150℃，直接导致后续喷涂的水基涂料无法正常润湿成膜[S41]

## 应用领域

水基涂料可适配全品类主流压铸合金，典型牌号及应用范围如下[S44]：

| 型号 | 种类特征 | 稀释比 | 应用范围 |
|------|---------|--------|---------|
| W-13型 | 水基，白色，pH=7~9 | 1:50~100 | 中小型铝合金铸件 |
| W-14型 | 水基，白色，pH=7~9 | 1:50~100 | 锌合金压铸件 |
| W-15型 | 水基，白色，pH=7~9 | 1:50~100 | 中大型铝合金铸件 |
| YT-P | 水基，乳白色 | 1:10~80 | 型腔与浇注系统 |
| MSO-25 | 无蜡型水基涂料 | — | 铝、锌合金高温压铸成型[S17] |

**典型应用场景**：汽车发动机离合器壳体、变速箱结构件、底盘轻量化部件，家电洗衣机铝合金结构件、3C电子薄壁壳体等大批量量产场景，适配自动化压铸生产线的高效生产需求[[S16,S23]]。

### 特殊工艺要求

**高真空压铸**：普通水基涂料含有大量水分，汽化后会大幅提升型腔内水汽分压，破坏预设的高真空度。高真空压铸场景需要使用低发气量、低水分占比的特殊改性水基涂料，或配套使用固体粉状脱模剂替代常规水基涂料，以保证最终压铸件可焊性、可热处理性达标[[S38,S20,S12]]。

**半固态压铸**：半固态合金浆料高速充填型腔的剪切力远高于常规液态压铸，要求水基涂料形成的润滑膜具备更高的高温附着强度与抗冲蚀能力。半固态压铸场景优先选用高润湿温度、成膜强度高的含硅有机物类水基涂料[[S21,S10,S46]]。

## 常见缺陷与机理

### 气孔缺陷

稀释比例过大、涂层过厚时，压铸过程中涂料会大量挥发产气，多余涂料堆积脱落后进入液态金属，同时堵塞型腔排气通道，最终诱发铸件内部气孔缺陷[[S27,S13,S43]]。

### 粘模与焊合

稀释比例过小（加水过多）会导致热态润湿性变差，无法在高温模具型腔表面形成连续牢固的防护润滑膜，高温液态合金直接与H13钢模具基体接触发生铝-铁冶金反应，最终引发粘模、焊合缺陷[[S18,S13]]。

### 表面冷隔

水基涂料使用过量、后续清理不到位时，残留的多余油脂和涂料成分会在型腔内部阻碍金属液的正常充型流动，使金属液流前端提前冷却凝固，最终形成铸件表面冷隔缺陷[[S43,S34]]。

### 表面斑纹

对于脱模难度高需要多次喷涂补充涂料的复杂铸件，水基涂料在型腔表面局部不均匀堆积，高温烧结后残留的固体颗粒附着在铸件表面会形成大量外观斑纹缺陷；这类铸件力学性能通常不受影响，但外观不合格无法作为成品交付[[S34,S22]]。

### 成膜失败（Leidenfrost效应）

水基涂料正常成膜的适用模具温度范围通常为200~300℃，当模具温度超过该阈值时，水基涂料液滴会发生莱顿弗罗斯特（Leidenfrost）效应，液滴被蒸汽膜托举发生爆裂反弹，无法在型腔表面附着成膜；当模具温度低于140℃时，涂料中水分无法完全汽化，易发生流淌，也无法形成均匀连续的润滑膜[[S18,S16]]。

## 与其他涂料的对比

### 与油基涂料对比[S21,S26,S46]

| 对比维度 | 水基涂料 | 油基涂料 |
|---------|---------|---------|
| 冷却能力 | 强 | 较弱 |
| 烟雾污染 | 无 | 严重 |
| 铸件表面 | 光亮洁净 | 有燃烧残迹、暗灰色斑痕或沟纹 |
| 脱模稳定性 | 略低于高端油基 | 高 |
| 环境安全性 | 好 | 存在健康隐患和火灾风险 |
| 当前应用趋势 | 市场主流（>90%） | 逐渐被淘汰 |

### 与粉剂涂料脱模力对比

对于AZ91HP镁合金压铸（熔化温度700℃、模温240℃、冲头二级压射速度2 m/s），水基涂料（1:60稀释）工况下铸件平均推出力约17~19 kN，而粉剂工况下推出力仅为6~10 kN，表明水基涂料的脱模阻力相对较大[[S15]]。
