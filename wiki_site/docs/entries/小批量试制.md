## 概述

小批量试制（英文对应术语包括 pilot run、small batch trial production、pre-production trial 等）是压铸新产品开发流程中，在正式大规模量产前设置的预生产验证环节。其核心目的在于识别产线瓶颈、物料流转异常等未预见问题，规避量产后因工艺不稳定造成的批量报废损失。该阶段明确处于 T1 试模完成且工艺窗口充分固化之后、客户 PPAP/PSO 量产批准之前，是新产品导入（NPI）和先进产品质量策划（APQP）流程中的关键验证节点[[S47]]。

与样件试制阶段不同，小批量试制强制使用量产级压铸模具和匹配的量产压铸机台进行生产，重点验证全流程良率与工艺稳定性，不允许借助简易模具、3D 打印等非量产手段制造铸件[[S21,S27]]。

## 工艺作用

小批量试制在压铸产品开发中承担多重技术验证功能[[S47,S39,S28]]：

- **模具综合性能验证**：在接近量产工况的条件下，确认模具型腔充型、冷却、脱模等功能件动作的可靠性和模具寿命预估；
- **工艺参数固化**：在 T0/T1 试模初步建立的工艺窗口基础上，通过连续批量运行，寻找、固化并记录最优工艺参数组合；
- **铸件质量一致性评估**：对连续试制批次开展全维度的多件检验，判断铸件在尺寸、内部质量、力学性能和表面质量方面是否能持续稳定地满足产品设计要求；
- **后加工与表面处理路线验证**：验证后续机加工工序（如定位夹紧、切削性能、表面处理适配性等）与小批量试制件的实际匹配度；
- **量产节拍与良率预测**：以连续运行节拍为统计依据，预测正式大规模生产时的效率、废品率及模具维护周期。

## 在开发流程中的位置

高压压铸新产品导入遵循从模具设计完成到正式量产的递进路径[[S26,S39]]：

| 阶段 | 节点/活动 | 关键目标 |
|------|-----------|---------|
| 模具设计完成 | 模流分析、P-Q² 图校核[[S32]] | 确认浇排系统方案和压铸机吨位的理论可行性 |
| T0/T1 试模 | 多轮试模、正交试验调参[[S26,S33]] | 建立初步压铸工艺窗口，消除致命缺陷 |
| **小批量试制** | **量产模具+量产机台连续运行**[[S27]] | **全流程良率和稳定性验证** |
| 中试 | 产线节拍验证[[S27]] | 验证大批量适配性，暴露物流与节拍瓶颈 |
| PPAP/PSO 客户批准 | 提交全量批次试制样件、检验记录 | 确认量产级模具和整套生产工艺满足全部限值要求 |
| 批量生产 | 稳态自动化生产 | 进入正式供货 |

APQP 工作在压铸模具制造阶段即同步推进，PPAP 所辖的验证活动作用于小批量试制阶段，确认整套工艺和模具在规定的产品设计限值内稳定运行[[S8,S18]]。

## 各试制阶段边界区分

根据压铸行业实践，不同试制阶段在模具条件、生产数量、验证目标和判定准则上存在明确的层级差异[[S21,S27]]：

| 试制阶段 | 模具条件 | 典型数量 | 核心目标 |
|----------|---------|---------|---------|
| 样件试制 | 允许使用简易模具、3D 打印砂芯、快速软模等非量产手段[[S27,S46]] | 极少（≤ 几十件） | 验证产品结构/功能，评判基本成形性 |
| **小批量试制** | **必须使用量产级钢模、同量产压铸机台**[[S21]] | 以小批次为单位（常见 10–20 件级或客户准批所需件次） | 验证全流程良率和工艺稳定性 |
| 中试 | 量产模具和量产产线 | 规模大于小批量试制 | 验证产线节拍和大批量适配性 |
| 批量生产 | 量产模具，稳态运行 | 大规模 | 进入订单交付状态 |

## 工艺参数

### 典型试制参数范围

小批量试制阶段的工艺参数选择，通常以 T0/T1 多轮正交试验获得的最优组合为基础，涉及压射速度、压力、温度及真空度等核心参数[[S45,S22,S48,S38,S14]]：

| 合金/工艺类型 | 参数 | 推荐值/范围 | 来源 |
|---------------|------|-----------|------|
| AlSi10Mg 铝合金（试制） | 快压射速度 | 3.5 m/s | [[S45]] |
| | 增压压力 | 60 MPa | [[S45]] |
| | 浇注温度 | 645°C | [[S45]] |
| AZ91D 薄壁件（如手机支架） | 浇注温度 | 670°C | [[S22]] |
| | 快压射速度 | 2.3 m/s | [[S22]] |
| AZ91D 常规压铸件 | 浇注温度 | 650–675°C | [[S48]] |
| | 模具预热温度 | 200–250°C | [[S38]] |
| 普通真空压铸 | 型腔真空度 | 50–80 kPa | [[S14]] |
| 高真空压铸 | 型腔压力 | 50–100 mbar | [[S14]] |
| 超高真空压铸 | 型腔压力 | < 50 mbar | [[S14]] |

### 参数调整策略与边界

小批量试制过程中，关键参数的调试需遵循严格的边界管控[[S33]]：

- **慢压射速度调试范围**：0.25–0.4 m/s。在 0.25 m/s 时压室内金属液不形成波峰，到 0.40 m/s 时金属液出现包卷现象，最佳试验值为 0.35 m/s（波峰适中且无卷气）。
- **压射速度切换点**：内浇口快压射起点距冲头起始位置需预留至少 50 cm 安全行程（以直径 10 cm 冲头为例），否则易导致切换位置偏差引发内部气孔和表面品质问题。
- **模具温度边界**：抽芯长针等关键温控点必须达到预设温度后方可批量投料，否则铸件顶部易出现冷隔缺陷。浇注温度过低同样会导致充型不良[[S33]]。

## 装备与模具

### 立式/卧式压铸机的适用场景

小批量试制阶段压铸机的选用取决于合金类型和铸件结构[[S40]]：

- **卧式冷压室压铸机**为铝合金常规试制首选机型，同样适用于镁合金大批量生产场景；
- **立式冷压室压铸机**仅在生产带中心浇口的小型镁/铝铸件，或中心辐射状、筒形压铸件时优先选用；
- 多品种小批量试制场景下，应优先选用液压系统适应性强、可快速完成参数调整的压铸机；少品种大批量量产试制阶段再配置自动化辅助机构[[S40]]。

### 快速模具与量产模具

试制阶段可采用快速原型技术制造的软模以缩短开发周期和降低成本[[S10,S21,S46]]：

| 比较项目 | 快速模具（软模） | 量产模具（钢制硬模） |
|----------|-----------------|---------------------|
| 制模周期 | 传统钢模的 1/10–1/3 | 完整周期（数周至数月） |
| 生产成本 | 传统钢模的 1/5–1/3 | 高 |
| 寿命 | 几十件至 1 万件 | 铜合金模 1–5 万件；钢制热作模具可达数十万件[[S5]] |
| 适用场景 | 单件或极小批量试制、工艺验证[[S21]] | 小批量试制和正式量产 |

需要注意的是，小批量试制阶段本身要求使用量产级钢模[[S21]]；快速软模主要服务于更早的样件试制阶段，或与 SLS 快速砂芯配套用于复杂结构铸件的单件试制[[S46]]。

## 材料：AZ91D 镁合金在试制中的关键行为

### 热裂倾向控制

AZ91D 镁合金在压铸试制中的热裂倾向受合金成分和杂质含量共同控制[[S19,S23,S4]]：

- **Zn 含量**：应控制在 0.6% 左右，Zn 含量 > 1% 时合金热脆性显著增大；
- **Mn 与 Fe/Mn 比**：Mn 可细化晶粒、降低热裂倾向，但含量不宜过高以免引起锰偏析降低塑性。需控制 Fe/Mn 质量分数比 ≤ 0.015，且 Fe 的最大质量分数限值为 0.005%[[S19]]；
- **夹杂物管控**：Mg₃N₂、MgO、过量 MgCl₂ 等夹杂均会扩大合金凝固温度区间，显著提升热裂风险，试制前须对原材料杂质含量严格检验。若热裂纹缺陷在调整工艺无效时，应考虑更换原材料厂家[[S19,S23]]；
- **Al 含量**：Al 为引起热裂纹的主要合金元素，需在力学性能与热裂倾向之间合理权衡[[S19]]。

### 综合性能特征

AZ91D 为高耐蚀高纯压铸镁合金（标准成分 Mg-9Al-0.7Zn-0.2Mn[[S13]]），常规压铸态下力学性能满足 ASTM 标准要求，流动性良好、热裂倾向小，适用于 3C 产品薄壁外壳类小批量试制件[[S12]]。半固态触变注射成形可进一步减少该合金铸件的孔隙率，有效提升伸长率[[S37]]。

## 质量检验与评价

### 检验体系框架

小批量试制作业的质量检验体系由四大核心模块构成：检验项目、检验仪器/工具/方法、验收标准/合格/报废判定规则、检验执行人员[[S34]]。按照 GB/T 25747–2022《镁合金压铸件》标准，试制作业的验收覆盖化学成分、力学性能、尺寸、表面质量、内部质量五类核心指标，任一试样不合格即判定对应件失效，同批次其余产品需逐件排查[[S16]]。

### 外观检测

小批量试制阶段外观目视检测通常执行 100% 全检，检查项覆盖表面状态、填充不良、冷隔、裂纹、变形、推出销痕的凹凸、型芯破损等典型缺陷[[S9,S7,S6]]。对于装饰品或需电镀、喷漆的压铸件，需额外严格把控表面质量[[S9]]。

### 内部质量检测

| 检测方法 | 应用方式 | 判据与特点 | 来源 |
|----------|---------|-----------|------|
| X 射线探伤 | 试制阶段全检 | 以无宏观缩孔/缩松为合格判据；操作简便、设备成本低，适合作试制阶段缺陷初步筛查 | [[S28,S11,S17]] |
| 工业 CT | 精密补充手段 | 体素精度可达 0.127 mm，可识别微米级缩松、裂纹等 X 射线探伤无法检出的微小复杂内部缺陷 | [[S17]] |

20 件级的小批量试制中，所有工件经 X 射线检测无内部缺陷即可判定工艺初步可行[[S28,S11]]。

### 力学性能测试

试制阶段力学性能测试执行不合格复试规则：任一试样的力学性能不合格时，需从对应批次中额外抽取双倍数量试样重新测试；重测后若仍存在不合格试样，则整批试制件判为不合格，不允许流向后续验证环节[[S16]]。

### 尺寸检验

试制件尺寸可采用三坐标测量仪（CMM）配合游标卡尺完成全尺寸校核，实测尺寸须完全符合图纸设计公差要求。任一尺寸不合格即判定对应试制件报废，同批次其余工件需全检排查[[S16,S42]]。

### ASTM 标准检验要求

根据 ASTM 压铸标准，小批量预生产阶段详细尺寸符合性验证通常仅需对 1 件或多件首批试制样件完成测试；后续性能抽检可由供需双方根据试制规模协商约定[[S30,S31]]。

## 局限性与常见问题

1. **经济性限制**  
压铸模具制造和压铸设备采购的固定成本极高，当量产规模低于数千件时，小批量试制的单件分摊成本远高于批量生产场景，显著制约试制经济性[[S41,S43,S20]]。

2. **试制与量产条件差异的偏差风险**  
小批量期间参数调整边界（如模具温度、压射速度）必须被严格监控和记录，若关键温控点在未达到预设温度时批量投料，将导致冷隔缺陷，影响良率数据的代表性[[S33]]。

3. **小样本统计意义不足**  
小批量试制的样本数量有限，其合格率数据难以完全代表大规模量产条件下的统计波动水平，对良率预测需留有合理安全裕度。

4. **X 射线探伤的固有局限性**  
试制阶段普遍使用的 X 射线探伤分辨率低于工业 CT，无法精准识别微小内部缺陷和复杂结构部位的隐蔽缺陷；量产阶段如需更高检测精度，应切换为工业 CT 抽检方案[[S17]]。

## 应用案例与行业实践

### 大众汽车 AZ91 镁合金变速器壳体

大众汽车早在上世纪 70 年代已完成 AZ91 系列镁合金汽车变速器壳体（单体重量 6.93–11.70 kg）的小批量压铸试制验证，后续逐步实现规模化量产。该类镁合金壳体相比铝合金壳体具有明显的减重效果，同时具备更优的耐海水腐蚀性能，使用寿命得到延长[[S3,S1,S24]]。

### 东风汽车 AZ91D 半固态压铸试制

东风汽车联合北京有色金属研究总院开展 AZ91D 镁合金半固态压铸小批量试制项目：采用半连续铸造棒料经热挤压制备坯料，通过 6 工位中频感应加热完成二次加热，使用 2500 kN 冷室压铸机完成触变成形，成功试制出汽车空压机连杆、发动机支架等零件。该案例充分利用了镁合金的高阻尼特性以降低发动机向底盘的振动传递[[S25]]。

### AZ91D 镁合金摩托车发动机壳体铸锻双控成形试制

基于液态铸锻双控成形工艺（压铸充型与高压锻造工序集成于单台 2500 kN 级设备），采用 AZ91D 镁合金完成摩托车发动机壳体的小批量试制。试制样件组织相比普通压铸得到显著细化，力学性能提升，外观完全符合设计要求。该案例已被收录为高校特种铸造教学实践项目[[S3,S13,S36]]。其配套模具设计经由多组仿真迭代筛选，确定了最优方案的压边力、锁模力等核心工艺参数，方案已公开于压铸模具工程师手册[[S35]]。

### 压铸优化工艺后的缸体试制验证

某缸体类压铸件经 DOE 优化仿真设计进行工艺改进后，通过小批量试制验证优化方案的可行性。试制结果显示，机加工底面无缩孔类报废缺陷，确认了改进工艺的稳定性[[S44]]。
