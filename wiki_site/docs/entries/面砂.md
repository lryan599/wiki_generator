面砂在铸造标准术语体系中是一个明确的工艺材料概念，但在压铸现场口语中时常被错误地赋予缺陷含义。本章严格依据现行标准与专业文献，辨析其准确定义、作为造型材料的属性、相关缺陷的形成机理与控制措施，并说明压铸语境下的术语边界。

## 定义与标准化
面砂（facing sand）特指造型时与模样直接接触的一层特殊配制的型砂，其厚度一般仅为1~2英寸，放置在背砂之前，以获得更光洁的铸件表面 [[S27,S40,S54]]。该定义出自GB/T 5611—2017《铸造术语》等行业标准，属于工艺材料范畴，与背砂（填充砂）及单一砂并列 [[S27,S40]]。

在砂型铸造中，面砂的使用后会混入背砂，不影响铸件成型质量；而在压铸、低压铸造等需使用砂芯的工艺中，面砂可指砂芯表面层或涂料层，但后者并非标准定义 [[S14,S21,S23]]。国内铸造行业存在俗称层面的歧义：部分从业者将铸件表面附着砂粒类的视觉缺陷口头称为“面砂缺陷”，这已偏离术语本义，不属于标准术语范畴，应严格与工艺材料“面砂”加以区分 [[S14]]。

## 面砂的组成与性能要求

### 原砂
面砂所用的原砂按材质分为普通石英砂和特种砂两大类：
- **普通石英砂**：SiO₂含量≥90%，用于普通铸铁及有色合金重力铸造场景 [[S13,S41]]。
- **特种砂**：用于大型厚壁铸钢、高合金钢铸件，包括锆砂（熔点2190~2420℃）、铬铁矿砂（熔点1600~1900℃）、刚玉砂（熔点1850~2000℃）、铝矾土熟料（耐火度≥1770℃）、镁砂（熔点1840℃）、石灰石砂（耐火度>2860℃）等，可有效防止粘砂 [[S13,S17,S41]]。

原砂的颗粒特性直接影响面砂透气性：0.63~0.4 mm粒度砂透气性可达710~1000，而0.16~0.1 mm细砂透气性仅为65~100 [[S17]]。实际生产中，将硅砂目数从40~70目调整为30~50目可显著提高透气性、降低发气量 [[S52]]。

### 粘结剂
面砂粘结剂分为无机和有机两大类 [[S51]]：
- **无机粘结剂**：普通粘土、膨润土、水玻璃、水泥、磷酸盐等。
- **有机粘结剂**：桐油、亚麻油、合脂、沥青、松香、纸浆废液、糊精、糖浆、呋喃树脂、热芯盒/冷芯盒树脂等。

不同粘结剂体系对应不同的硬化工艺。例如，呋喃树脂自硬砂的固化剂加入量通常占树脂质量的30%~60% [[S51]]；改性的水玻璃面砂在600℃保温5 min后高温残留强度可低至0.01 MPa，显著改善了溃散性 [[S6,S48]]。

### 功能性添加剂
为改善防粘砂效果，面砂中常加入煤粉（添加量2%~8%），煤粉在金属液热作用下产生还原气氛，可防止金属液渗入砂粒间隙并帮助落砂时铸件表面脱壳 [[S2,S6,S24]]。其他常用添加剂包括石墨、硫磺、硼酸（铸铝合金专用）、微硅粉、促硬剂、溃散剂及各类抗吸湿组分（如氢氧化钾、六偏磷酸钠、葡萄糖、十二烷基磺酸钠、425#水泥、PVB、改性EVA等） [[S6,S24]]。

### 主要性能指标
面砂在应用中需满足多项性能要求，关键参数举例如下：

| 性能指标           | 典型值/要求                                 | 说明 |
|-------------------|---------------------------------------------|------|
| 耐火度            | 普通石英砂≥1350℃；粘土砂≥1000℃；锆砂≥2000℃ | 不同铸件材质选用不同耐火度原砂 [[S41]] |
| 透气性            | 粗粒砂710~1000；细粒砂65~100                 | 透气性与砂粒尺寸直接相关 [[S17]] |
| 高温残留强度      | 改性水玻璃砂0.01 MPa（600℃/5 min）           | 影响落砂性能 [[S6]] |
| 呋喃树脂砂热物性  | 堆积密度1500 kg/m³，粘结剂含量2 wt%，砂孔隙率40%，透气性100 cm³/min | 工艺仿真参数依据 [[S45]] |
| 涂层厚度          | 0.5~0.6 mm                                   | 防粘砂涂料推荐厚度 [[S21,S50]] |

## 面砂相关标准与牌号
面砂所用原材料有明确的国家和行业标准，主要包括：
- **GB/T 9442《铸造用硅砂》**：规定硅砂牌号命名规则为“ZGS + 二氧化硅分级代号 + 粒度分组代号 + 粒形标识 + 角形系数值”，例如ZGS96-42Q-30表示SiO₂含量≥96%、粒度集中于42号筛段、前端筛残留量大于后筛残留量的铸造硅砂 [[S3,S32]]。
- **ZBJ31009-90《铸造用膨润土和粘土》**：膨润土牌号由主要交换性阳离子、pH类型、湿压强度等级、热湿拉强度等级组合命名，如PCaS-3-5 [[S8]]。
- **ZBJ31008-90《砂型铸造用涂料》**：规定了涂料的分级、代号与技术要求 [[S33]]。

## 面砂在不同工艺中的匹配
- **重力铸造**：中大型铸铁件多采用自硬水玻璃砂作面砂；铝铸件常用热硬树脂砂；大型厚壁铸钢件选用锆砂、铬铁矿砂等特种砂配制面砂 [[S7,S10,S31]]。
- **低压铸造**：可使用宝珠砂替代普通覆膜砂，提升砂芯透气性，减少气孔缺陷 [[S31]]。
- **压铸**：绝大多数使用全金属压铸模，仅复杂内腔铸件采用砂芯。配套砂芯时，面砂需具备良好的耐冲刷性能以抵抗高压高速金属液的冲击 [[S14,S23]]。

## 面砂相关缺陷辨析
面砂本身是工艺材料，但当面砂配制不当或工艺控制不当时，会引发一系列铸造缺陷。这些缺陷常被现场人员笼统地称为“面砂问题”，但按标准术语体系，它们分属**砂眼、粘砂（机械粘砂/化学粘砂）、夹砂结疤、掉砂**等不同类别，不可混淆。

### 形成机理
面砂参与下的缺陷形成途径多样 [[S42,S30,S36,S20,S23,S47]]：
1. **砂眼（sand inclusion）**：面砂层厚度不足或砂芯/砂型强度过低，受热后从背砂上剥落、崩离，被流经的金属液裹挟，形成内部或表面充塞型砂的孔洞。
2. **粘砂（mechanical/chemical sand burn-on）**：金属液机械渗透进入粗颗粒砂型孔隙，或金属氧化物与硅砂杂质反应导致砂粒嵌合在铸件表面，形成难以清除的粘附砂层。
3. **夹砂结疤（scab）**：砂型表面层热膨胀应力超过水分凝聚区强度，表层干砂拱起开裂，金属液渗入缝隙后形成不规则金属凸起。
4. **掉砂（drop）**：合型前型腔散落砂粒未吹净、合模操作碰触砂芯使砂块脱落，掉砂被金属液包裹在铸件表面。
5. **冲砂/砂夹杂**：浇注湍流工况下，金属液往复冲刷砂型/砂芯表面，粘结剂被氧化后未粘结砂粒被剥离卷入金属液，最终形成表面砂夹杂。

所有湍流裹挟形成的表面砂夹杂，其外围均包裹一层金属氧化膜，砂粒硬度远高于铝合金基体 [[S28,S29]]。

### 宏观形貌特征
- **砂眼**：铸件表面存在充塞型砂的孔洞，清理后砂粒脱落留下不规则孔穴 [[S1,S16]]。
- **机械粘砂**：表面附着海绵状金属光泽的砂粒层，难以清除，铸件粗糙度劣化 [[S34,S46]]。
- **夹砂结疤**：厚度数毫米的不规则金属凸起，仅少量点与基体连接，下方对应凹陷区域 [[S28]]。
- **掉砂**：铸件局部区域零散分布外露砂粒嵌入层 [[S23,S26]]。

### 对铸件质量的影响
面砂相关缺陷会严重损害铸件力学性能和使用可靠性 [[S19,S43,S22]]：
- 压铸件因砂眼造成的报废量可占总废品率的25%~80%。
- 砂夹杂破坏金属基体连续性，减小有效承载面积，大幅降低塑性、疲劳强度和断裂韧性。
- 表面粘砂增加清理和机加工工时，加剧刀具磨损，并使流体通道内的介质流动稳定性下降。
- 存在面砂缺陷的铸件，气密性试压失效概率显著提高。
- 粘砂可使铸件表面粗糙度劣化2~3个等级，部分区域Ra值超出加工余量即直接报废。

### 典型工艺参数区间与控制措施
产生面砂相关缺陷的关键工艺参数范围与控制要点 [[S18,S23,S42,S50,S35,S20,S21,S22]]：

| 影响因素         | 推荐控制范围/措施                                | 缺陷风险区间                            |
|------------------|------------------------------------------------|-----------------------------------------|
| 浇注温度（铝）   | 680~730℃                                       | 超过730℃粘砂缺陷显著升高               |
| 浇注温度（铸铁） | 1380~1420℃                                     | 超过1450℃粘砂风险陡增                  |
| 砂型硬度上限     | 上箱HB65~75、下箱HB75~85                       | 低于HB75时粘砂/冲砂概率显著升高         |
| 涂料涂层厚度     | 0.5~0.6 mm                                     | 涂层低于0.4 mm时粘砂面积占比显著上升    |
| 压铸模具温度     | ≥160℃                                          | 模具温度低于160℃时砂眼发生率明显上升   |
| 真空压铸抽真空起始位置 | 580 mm                                    | 不合理设定导致缸体K面砂眼增多           |
| 砂芯强度         | 保证高温强度大于金属液冲刷动压力               | 强度不足时砂粒易被冲刷剥离形成面砂缺陷  |

工程中成熟的预防措施包括：保证砂型紧实度均匀，避免反复压抹表层；采用大截面积横浇道、多内浇口分散布置以降低局部流速；控制浇注温度在合理区间；合型前吹扫散落浮砂，合模避免碰触砂芯；优化砂芯配方，提高高温强度；铸件表面涂刷厚度均匀的防粘砂涂料。在使用冷芯盒树脂砂的铸件中，将浇注温度控制在1420℃、涂层厚度0.6 mm、冷芯树脂加入量1.2%时，粘砂面积占比可降至平均8.67% [[S50]]。

## 压铸场景中面砂术语的合理使用
在纯压铸工艺（无砂芯）中，不存在面砂作为工艺材料的实体。现场常说的“面砂缺陷”实质上是指压铸件表面的砂眼（sand inclusion）或夹杂类缺陷，其成因多与合金液清洁度、压射参数、模具温度、抽真空位置等相关，而非真正的面砂剥落 [[S14,S19,S35]]。典型压铸砂眼的成因包括合金溶液杂质或浮渣、柱塞喷射压力过低、模具温度过低、给料注入过急等，改进方向为保持溶液清洁、调整压射压力、降低紊流程度 [[S53]]。

因此，在压铸质量分析中，应按标准术语将表面砂颗粒缺陷称为“砂眼”或“砂夹杂”，避免与造型材料“面砂”相混淆，以保证诊断的准确性和技术交流的规范性。
