## 概述

缝隙式浇注系统（slot gate system）是铸造生产中一种重要的浇注系统类型，在铝、镁合金铸造领域应用广泛，但在铸钢件生产中极少使用[[S22,S27]]。该系统的核心特征是采用沿铸件全部或部分高度方向设置的单层薄片状内浇道，使金属液自下而上平稳地逐层充填型腔[[S1]]。其常用别名包括“缝隙浇道”，在英文文献中对应的术语有 slot gate system、slot gate、slot runner、slot-type gate 等[[S39,S1]]。

从分类体系上看，缝隙式浇注系统隶属于“分层注入式浇注系统”大类，是阶梯式浇注系统的一种特殊分支形式[[S43,S7]]。

## 结构与组成

缝隙式浇注系统的典型结构包含以下核心元件[[S27,S48,S4]]：

| 组成元件 | 功能说明 |
|----------|----------|
| 浇口杯 | 承接金属液，起缓冲和浮渣作用 |
| 直浇道 | 将金属液从浇口杯导入横浇道；高大铸件常采用蛇形或倾斜弯曲形直浇道以降低流速 |
| 横浇道 | 连接直浇道与集渣立筒，分配金属液 |
| 集渣立筒（集渣包/柱） | 缓冲金属液流速、挡渣、兼起侧冒口补缩作用 |
| 缝隙式内浇道（立缝） | 沿铸件高度方向布置的薄片状通道，金属液由此平稳进入型腔 |
| 冒口 | 置于铸件顶部，起集渣、排气和补缩作用 |

在低压及差压铸造场景中，缝隙式浇注系统的结构包括直浇道、横浇道、立筒（即集渣立筒）以及缝隙内浇道，铸造时金属液从底部升液管进入系统[[S27,S48]]。

### 结构类型

据《特种铸造手册（下册）》，缝隙式浇注系统依据外浇道（直浇道）的形式可分为五种结构类型[[S9]]：

| 结构类型 | 说明 | 适用铸件举例 |
|----------|------|-------------|
| 垂直式 | 直浇道为竖直布置 | 一般铸件 |
| 鹅颈式 | 直浇道呈鹅颈状弯曲，缓冲冲击 | 一般有色合金铸件 |
| 蛇形式 | 直浇道呈蛇形多次转折，大幅降速防氧化 | 铝合金活塞等 |
| 倾斜式 | 外浇道倾斜布置 | 一般铸件 |
| 刀形外浇道配缝隙内浇口 | 特殊刀形外浇道组合 | 特定场合 |

## 充型与流动特征

### 逐层填充的流体力学原理

缝隙式浇注系统实现平稳逐层充型的核心机理在于立筒与缝隙内浇道之间的流阻匹配特性[[S27,S6]]：

- **立筒**的横截面积远大于缝隙内浇道的总截面积。浇注过程中，立筒内金属液的流程随液面升高而增加，流动阻力逐渐增大；
- **缝隙内浇道**被金属液浸没的截面积随充型高度的增加而增大，流动阻力反而减小；
- 两者的阻抗匹配效应使金属液始终从当前液位对应的缝隙段以低速进入型腔，实现“分层进液”[[S27]]。

此外，缝隙内浇道采用扁平截面形式，可降低流道的水力半径，从而降低雷诺数（Re），使金属液趋向层流状态，进一步抑制湍流倾向[[S6]]。

### 充型过程表现

水模拟和数值模拟结果一致验证了缝隙式浇注系统的充型平稳特性：充型10%阶段金属液从缝隙导入型腔时液面平缓上升，30%阶段型腔内部与四周液面齐平，全程无飞溅、无明显紊流，液面波动极小[[S27,S17,S30]]。

### 型腔热分布特点

在理想充型状态下，缝隙式浇注系统可使新进入型腔的金属液温度始终高于已充入下层的金属液，形成自下而上温度逐步升高的正向温度梯度（类似顶注式），区别于普通底注式浇注系统“底部过热、顶部温度低”的反向温度分布[[S22,S4,S44]]。这种温度场分布有利于实现顺序凝固，便于顶部冒口发挥补缩作用[[S22]]。

## 工艺设计与关键参数

### 内浇道（立缝）厚度设计

缝隙式内浇道厚度δ与铸件壁厚的关系是设计的核心参数之一[[S33,S16,S41,S42]]：

| 铸件壁厚条件 | 缝隙厚度δ取值 | 备注 |
|-------------|-------------|------|
| 铸件壁厚 b ≥ 10 mm | δ = (0.7~1.0) × 铸件壁厚 | 圆筒形铸件取较小值，平板形可取较大值 |
| 铸件壁厚 b < 10 mm | δ = (1.0~1.5) × 铸件壁厚 | |
| 需将热量引向立筒与冒口时 | δ ≥ 1.5 × 铸件壁厚 | 特殊场景可突破常规取值 |

### 其他关键尺寸参数

| 参数 | 取值范围/计算式 | 来源 |
|------|----------------|------|
| 缝隙宽度 B | 15~35 mm | [[S41,S42,S5]] |
| 立筒（集渣筒）直径 d | d = (4~6)δ（约为缝隙厚度的4~6倍） | [[S5,S41,S42]] |
| 全部立缝总厚度 Σδ | Σδ = (0.008~0.012) × P（P为铸型型腔截面周边全长） | [[S16,S37]] |
| 缝隙数量 n | n = (0.016~0.028)S/δ（S为铸件外形周长） | [[S24,S40]] |
| 缝隙布置间距 | 240~300 mm | [[S40]] |

### 内浇道入口位置设计（强制要求）

缝隙的最低点**必须高于**直浇道（横浇道）进入集渣筒的入口位置。这一设计要求保证金属液在垂直过道（立筒）内的液面始终高于缝隙入口，迫使熔渣上浮留在立筒中，避免进入型腔[[S36,S18]]。

### 低压铸造液面上升速度控制

在低压铸造条件下，缝隙式浇注系统的金属液面上升速度控制参数范围一般为 1~8 cm/s（即 10~80 mm/s）[[S2]]。实际生产中，大型舱体、筒体铸件常用的控制区间为 30~50 mm/s，对应的充型压力升压速率约为 0.8~1.7 kPa/s[[S21,S40,S49]]。

### 设计计算经验公式汇总

综合多个专业教材与学术文献，缝隙式浇注系统设计常用以下五条经验公式[[S37,S24,S29,S26]]：

| 公式编号 | 计算对象 | 计算式 | 变量说明 |
|----------|---------|--------|---------|
| (1) | 全部立缝总厚度 | Σδ = (0.008~0.012)P | P——铸型型腔截面周边全长（mm） |
| (2) | 单条缝隙厚度 | δ ≤ (0.7~1.0)b | b——缝隙相连处铸件主要壁厚（mm） |
| (3) | 缝隙数量 | n = (0.016~0.028)S/δ | S——铸件外形周长（mm）；δ——单条缝隙厚度（mm） |
| (4) | 立筒直径 | d ≥ 4δ | δ——缝隙厚度（mm） |
| (5) | 型腔金属液平均上升速度 | v = (3.0~4.2)/b（cm/s） | b——铸件平均壁厚（mm） |

## 优点

缝隙式浇注系统具有以下工艺优势[[S22,S27,S32]]：

1. **充型平稳**：金属液沿垂直缝隙逐步上升充型，无飞溅、无明显紊流，仅略逊于底注式[[S15,S45]]。
2. **排气良好**：型腔中气体可自上而下的排气通道顺利排出[[S4]]。
3. **挡渣撇渣效果好**：缝隙内浇道与集渣立筒兼具撇渣功能，熔渣随液面上浮进入立筒顶部或型腔顶部冒口，大幅降低夹渣缺陷[[S27,S22]]。
4. **减少卷气与二次氧化**：平稳层流充型避免了金属液飞溅和氧化膜卷入，对铝、镁合金等易氧化合金尤为重要[[S27,S22]]。
5. **型腔热分布合理**：充型过程中始终由接近液面的缝隙口注入高温金属液，形成类似顶注式的正向温度梯度，优于底注式[[S47,S3,S22]]。
6. **利于补缩**：沿铸件全高分布的高温金属液供给通道可支撑自下而上的顺序凝固，延长缝隙浇道长度可增加补缩距离，适配高大薄壁圆筒铸件的轴向补缩需求[[S22]]。

## 缺点与局限

缝隙式浇注系统的主要局限性体现在以下几方面[[S4,S22,S7,S45]]：

1. **补缩性能不突出**：理论上的理想补缩效果在实际中存在差距。随着液面升高，集渣立筒上部金属液温度逐步下降，通过内浇道进入型腔的金属液温度也相应降低，实际补缩能力未达理论值[[S4,S22]]。
2. **冒口尺寸无明显缩减**：配套冒口的尺寸相比常规底注式系统并未显著减小[[S4,S22]]。
3. **金属液流经路线长、温降大**：浇注系统结构复杂，金属液在直浇道—横浇道—立筒—缝隙内浇道—型腔的长路径中散热面积大，充型总耗时长，温降明显高于顶注式和底注式[[S10,S22,S4]]。
4. **工艺出品率低**：传统缝隙式浇注系统的工艺出品率最低可达40%，显著低于其他浇注系统类型。实测案例数据如下：

| 铸件材质/类型 | 采用浇注系统 | 工艺出品率 | 来源 |
|-------------|------------|-----------|------|
| ZL101铝合金（转接头铸件） | 缝隙式 | 46.8% | [[S23]] |
| ZM5镁合金（后弧框铸件） | 缝隙式（蛇形浇注系统） | 43.4% | [[S23]] |
| 传统缝隙式（通用经验值） | 缝隙式 | 最低约40% | [[S4,S22]] |

5. **清理困难**：薄壁缝隙浇道与铸件连接面积大，且沿铸件全高分布，切割打磨工作量大，清理难度远高于顶注式、底注式[[S3,S45,S7]]。

### 与相关系统的边界差异

- **与顶注式**：顶注式内浇道全部设置在铸件顶部，补缩效果好但冲击力大；缝隙式以沿侧壁高度延伸的连续薄片结构取代顶部集中引入，保留了顺序凝固优势又避免了金属液直接冲击[[S47,S38]]。
- **与底注式**：底注式充型平稳但型腔下部过热、不利于顶部补缩；缝隙式金属液可沿缝隙自下而上逐层充型，充型后期型腔上部温度更高，解决了底注式补缩困难的缺陷[[S47,S32]]。
- **与阶梯式**：常规阶梯式在铸件不同高度开设多层离散内浇道，设计不当易出现“乱浇”和不同层金属液汇流夹杂问题；缝隙式的连续薄片内浇道避免了多股液流交汇的夹渣风险，更适用于高大薄壁有色合金铸件[[S3,S45]]。

## 相关缺陷与防治

当缝隙式浇注系统设计不当时，可能出现以下缺陷：

### 冷隔与浇不足

- **成因**：缝隙浇道厚度远小于铸件壁厚，金属液充型过程中温降过大导致流动性不足[[S22]]。
- **防治措施**：将缝隙浇道厚度按铸件实际壁厚进行适配（参见本文“内浇道厚度设计”表），同时适当提高浇注温度以保证金属液的流动能力[[S22,S24]]。

### 局部缩松

- **成因**：立筒与缝隙内浇道的流阻匹配失衡，导致随液位同步提升的分层进液逻辑失效，顶部金属液温度不足以建立理想的轴向顺序凝固[[S4]]。
- **防治措施**：优化立筒截面积与缝隙内浇道截面积的配比；必要时可采用分段进液的阶梯形缝隙浇注系统，改善不同高度的温度分布[[S4]]。

### 立筒对面铸件壁纵向裂纹

- **成因**：缝隙宽度过小导致截面较大的集渣立筒过于靠近铸件，引起局部过热[[S5]]。
- **防治措施**：合理控制缝隙宽度在15~35 mm范围内，避免缝隙宽度过小[[S5]]。

## 分类中的位置

缝隙式浇注系统在GB/T 5611-2017《铸造术语》中被列为独立浇注系统类型，与顶注式、底注式、中注式、阶梯式等并列[[S1]]。在国内主流铸造工艺分类体系中，缝隙式与阶梯式同属“分层注入式浇注系统”，缝隙式被视为阶梯式的特殊形式[[S43,S7]]。
