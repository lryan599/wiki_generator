## 概述

在压铸行业相关标准与专业文献中，浆状涂料是指黏度已调整至可直接使用状态、无需额外稀释的成品涂料。按标准分类，浆状涂料属于按物理状态划分的三大涂料品类之一，与膏状涂料（载体质量分数≥10%，需经载体稀释后方可使用）和粉（粒）状涂料（载体质量分数<10%）形成明确区分。该定义已被GB/T 5611《铸造术语》收录为正式术语，并在JB/T 9226—2008标准中作为分类体系的核心基础[[S14,S31,S20]]。

在压铸工艺语境下，浆状涂料扮演着在熔融金属与模具型腔之间建立隔离保护层、润滑脱模及辅助热管理的多重角色，是影响压铸件质量与模具寿命的关键辅助材料[[S22,S24]]。

## 分类与牌号

依据JB/T 9226标准，压铸用浆状涂料可采用以下成熟的分类体系[[S19,S31,S20]]：

**1. 按主要耐火粉料分类**

涂料按主要耐火粉料的不同分为九类，典型代表包括石墨类、硅酸盐类、氮化硼类等。

**2. 按载体类型分类**

| 载体类型 | 代号规则 | 典型牌号示例 |
|----------|----------|-------------|
| 水基浆状涂料 | S + J | SJ-SM（水基浆状石墨涂料） |
| 有机溶剂浆状涂料 | Y + J | YJ-SM（有机溶剂浆状石墨涂料） |

**3. 按施涂方式分类**

可分为刷涂型、喷涂型、流涂型、浸涂型。

**4. 按功能分类**

按功能可划分为润滑型、隔热型、脱模型[[S19,S31]]。

## 组成与配制

浆状压铸涂料的典型组成包括耐火填料、悬浮剂、粘结剂及功能助剂等核心组分。悬浮剂常用膨润土、羧甲基纤维素（CMC），其作用是防止固体耐火粉料分层沉淀；粘结剂可分为无机粘结剂（如硅溶胶、水玻璃）与有机粘结剂两大类[[S30,S16,S29]]。

生产验证的Lub-2型水基浆状涂料典型质量分数配比及优化乳化工艺参数如下[[S29]]：

|------|:----------:|
| 改性有机聚硅氧烷 | 14.4 |
| 蜂蜡 | 4.3 |
| 氧化聚乙烯蜡 | 2.9 |
| Span80 | 1.3 |
| OP-10 | 4.1 |
| 增稠剂 | 0.43 |
| 2,6二叔甲基对甲酚 | 0.14 |
| 杀菌防腐剂 | 0.14 |
| 消泡剂 | 0.29 |

对应的优化乳化工艺参数为：乳化温度60 ℃、乳化搅拌速度2000 r/min、乳化时间15 min[[S29]]。

制备工艺对涂料性能影响显著。球磨机、胶体磨、轮碾机等具有研磨作用的设备均适合生产浆状涂料，其中胶体磨可反复循环研磨，生产周期短、效率较高[[S16]]。

## 工艺作用

在压铸全循环中，浆状压铸涂料的核心功能包括[[S24,S13,S22]]：

- **隔离保护**：在高温合金液与模具型腔之间形成隔离保护层，避免熔融金属直接冲刷型腔表面。
- **热管理**：降低模具热导率，保持进入型腔的金属液流动性，延长充型可填充时间。
- **润滑脱模**：高温工况下保持优良润滑性能，减小铸件与型芯间的摩擦，便于推出铸件。
- **防粘模**：对铝、锌等常用压铸合金起到有效的防粘模作用。

NADCA相关规范明确指出，压铸用脱模释放材料需同时满足脱模、抗粘焊、运动部件润滑三项核心性能要求，成膜后需形成连续绝缘缓冲层阻隔熔融金属与模具钢直接接触，保证顺利脱模的同时降低模具热冲击，延长模具使用寿命[[S10,S15]]。

## 应用工艺与参数控制

### 涂覆方式与适用对象

浆状压铸涂料可采用刷涂、喷涂、浸涂三种主流涂覆方式，可施工的对象覆盖模具型腔、型芯、滑块、推出元件、压室、冲头等压铸模具全部位[[S24,S13]]。

### 稀释与浓度控制

涂料的行业通用推荐稀释比例为1:10～1:30（涂料:水），大件、薄壁、复杂铸件推荐采用10～15倍水稀释，小件、简单件推荐20～30倍水稀释。生产中可采用比色计定期校准涂料浓度以保证批次稳定性[[S4,S22]]。

### 模具温度窗口

浆状涂料喷涂时推荐模具预热温度窗口为180～280 ℃。低于150 ℃时涂料易流淌无法均匀成膜；高于398 ℃时涂料会被瞬间汽化反弹，难以附着在型腔表面[[S4]]。水基浆状涂料在模具温度200～300 ℃时可有效突破莱顿弗罗斯特效应产生的蒸汽气垫层阻碍，保证涂料在高温模具表面均匀铺展润湿[[S9,S3]]。

### 喷涂工艺参数

| 工艺参数 | 推荐范围 |
|---------|---------|
| 空气喷涂压力 | 0.2～0.5 MPa |
| 理想喷射距离 | 100～200 mm |
| 喷射时间 | 0.1～2.0 s |
| 理想干膜厚度 | 20～100 μm |

### 涂覆后的关键操作要求

喷涂或刷涂后，应待涂料中的稀释剂**完全挥发后再合模浇注**，避免铸件产生内部气孔缺陷[[S24]]。涂覆后应特别注意压铸模排气道的清理，避免因被涂料堵塞而起不到排气作用；对于转折、凹角部位应避免涂料沉积，以免造成铸件轮廓不清晰[[S24,S13]]。

## 优势与局限性

**相对粉状压铸涂料**的优势：涂层成膜均匀性更优，无需配置专用粉末喷涂系统，设备投入更低[[S27,S22]]。

**相对普通低黏度液状压铸涂料**的优势：固体有效含量更高，单次喷涂后可连续压铸8～10次，易粘模复杂零件也可维持2～3次压铸脱模周期，减少喷涂频次、提升生产效率[[S22]]。

**固有局限性**：长期静置易出现分层沉淀，使用前需充分搅拌；运输储存过程中载体占比更高，整体重量大于同有效组分的粉状涂料，运输成本相对更高[[S27,S22]]。

## 常见缺陷与对策

| 缺陷类型 | 产生机理 | 解决对策 |
|---------|---------|---------|
| 气孔 | 涂层过厚、未完全干燥即合模浇注，残留水分受热汽化卷入金属液内部 | 控制干膜厚度在推荐区间，喷涂后待稀释剂完全挥发再合模浇注 |
| 粘模、拉伤 | 涂层厚度不足，无法形成连续隔离层，熔融金属直接冲刷型腔表面 | 适当提升涂料喷涂量，保证涂层完整覆盖型腔工作表面 |
| 充型不良 | 喷涂过量导致组分在分型面、排气槽沉积形成积碳，堵塞排气通道 | 控制喷涂范围仅覆盖型腔成型区域，定期清理积碳沉积物 |
| 冷隔 | 模具预热温度过低，涂料无法快速成膜流淌堆积，多股金属流对接位置未完全熔合 | 将模具预热温度提升至180 ℃以上，保证涂料快速干燥成膜 |

## 质量检验与标准

JB/T 9226—2008对此类涂料的检验规则与试验方法有明确规定[[S19,S1,S25]]：

**取样规则**：取样前将涂料充分搅拌均匀，取样桶数不少于3桶，混合均匀后提取检测试样质量不少于1 kg。批次验收需逐批检查外观、密度、条件黏度、悬浮性等指标，任意一项指标不合格加倍取样复检，复检仍不合格判定整批产品不合格[[S19,S1]]。

**主要试验方法**涵盖：
- 外观状态检查（75 mm×25 mm玻璃片目测判断有无可见杂质）
- 密度测定（100 mL带磨口塞刻度量筒结合天平称量法）
- 条件黏度测定（按JB/T 4007规定执行）
- 悬浮性测定（静置法）
- 涂层耐磨性、发气量、高温曝热裂纹等级等共计8项标准试验方法[[S19,S25]]

浆状涂料的贮存保质期自生产日期起计为六个月（镁砂粉涂料为三个月），运输及贮存过程中应防止受潮变质；有机溶剂浆状涂料存放温度不应超过45 ℃，严禁靠近火源[[S1]]。
