## 概述

冷室压铸机指将独立保温炉中的定量熔融金属输送至压射室进行压射作业的压铸机，其压射室与坩埚/保温炉完全独立，压铸生产时通过自动给汤机械手或定量浇注系统将金属液送入压射室进行压射[[S5,S22]]。压射部件不浸入金属液中，因此区别于热室压铸机（压射室和冲头浸入保温炉熔融金属表面以下）[[S5]]。冷室压铸机的压射压力和压射速度普遍高于热室压铸机，适合生产对强度、刚度、致密性有要求的受力件、安全件和复杂结构件[[S22]]。

## 定义

根据国家标准 GB/T 20906–2007《压铸单元安全技术要求》，冷室压铸机是"独立的保温炉中的定量熔融金属被输送到压射室的压铸机"[[S5]]。该标准进一步明确了两种基础机型：卧式冷室压铸机为压射室水平安装的机型，立式冷室压铸机为压射室垂直安装的机型[[S5]]。

现行冷室压铸机产品通用国家标准为 **GB/T 21269–2023《冷室压铸机》**，于2023年9月7日发布、2024年4月1日实施，适用于卧式、立式两类冷室压铸机的设计、制造、验收、包装、运输等全流程环节[[S10,S17]]。该标准为第二次修订版，代替 GB/T 21269–2018，核心技术更新包括新增开模重复定位精度术语及检验方法、新增 50000 kN < F ≤ 60000 kN 规格卧式机型的压射性能参数、调整能效等级要求与安全要求等[[S16]]。标准以锁模力 F 作为主参数，配套定义了压铸模厚度、动模安装板行程、增压、压射压力/位移/速度-时间曲线以及开模重复定位精度等共27项相关术语[[S44,S34]]。

## 分类

冷室压铸机按压射室安装方向及合模/压射机构的相对布置，工业通用分类体系包含三大类：**卧式冷室压铸机、立式冷室压铸机、全立式冷室压铸机**[[S22,S12,S43,S32]]。

**图1 压铸机整机分类体系图**：压铸机整体分为冷室压铸机和热室压铸机两大分支，冷室压铸机下进一步细分卧式、立式、全立式三个子类[[S21]]。

### 三种机型的结构与特点对比

| 特性 | 卧式冷室压铸机 | 立式冷室压铸机 | 全立式冷室压铸机 |
|---|---|---|---|
| 压射室安装方向 | 水平安装，压室中心线平行于模具运动方向 | 垂直安装，压室中心线平行于模具分型面 | 垂直安装，压室中心线垂直于模具分型面 |
| 合模/压射机构 | 卧式布置 | 压射机构直立，合模卧式 | 全部垂直布置，压射系统在下部，开合模系统在上部 |
| 浇注口位置 | 设偏心和中心两个可调位置 | 中心浇口 | — |
| 金属液流动路径 | 转折少，压力传递损耗小 | 进入型腔时经90°转折，压力损耗较大 | 由下向上填充，流程短转折少，压力损耗小 |
| 气体卷入倾向 | 金属液与空气接触面积大，参数不当易卷入空气和氧化夹渣 | 金属液注入直立压室后气体上浮，裹卷气体较少 | 带入型腔空气少，铸件气孔率低 |
| 嵌件放置 | 不便于压铸带嵌件的铸件 | — | 模具水平放置，安放嵌件十分方便 |
| 操作维护 | 操作程序简单，易实现自动化，维修方便 | 增设反料机构，结构复杂，操作维修不便 | 操作不便，取出铸件困难 |
| 生产率 | 高 | 余料未切断前无法开模，影响生产率 | 较低 |
| 目前工业地位 | 90%商用冷室压铸机为卧式，应用最广 | 多为小型机型，用途逐渐减少 | 多为中小型机型 |
| 典型应用 | 各类有色金属铸件，也可用于黑色金属 | 必须采用中心浇口的有色金属压铸件 | 电机转子、定子等带硅钢片嵌件的零件 |

## 核心结构与组成

冷室压铸机整机由**合模机构、压射机构、液压传动系统、电气控制系统和安全防护装置**等功能模块组成[[S24]]。卧式冷室压铸机的典型整机构成如下图所示。

**图2 国标级卧式冷室压铸机典型结构示意图**：展示锁模机构、压射机构、机身液压系统的完整布局，源自国家标准 GB/T 20906–2007[[S40]]。

### 压射系统

压射系统是冷室压铸机的核心功能部件，功能是将熔融金属合金液按预设的高速、高压压入模具型腔，并在金属凝固过程中快速增压保压，直接决定铸件轮廓尺寸、力学性能、表面质量与致密性[[S24,S20]]。

其组成包含：压射液压缸组件、压射室（入料筒）、压射冲头/活塞组件、快压射蓄能器、增压蓄能器、压射油路板等[[S24,S20,S1]]。

北美压铸协会（NADCA）公开的教材将冷室压铸机压射系统的核心控制功能归纳为8项：
1. 闭合注料口
2. 慢压射速度控制
3. 快压射切换点定位
4. 快压射速度控制
5. 低压缓冲位置定位
6. 增压延时设定
7. 增压建立速率控制
8. 最终增压压力设定

该套规范是北美冷室压铸机出厂调试和日常生产管控的通用参考依据[[S15]]。

### 合模系统

合模系统（开合模+锁模机构）的核心功能是完成模具开合动作，提供足够的锁模力抵消压射过程的胀型力，保证分型面不被金属液胀开。其组成包含定型板、动型板、拉杠（哥林柱）、曲肘机构、顶出机构、调模机构[[S24,S33,S29]]。主流机型采用液压驱动曲肘连杆结构，可在行程终点实现机械锁死，刚性好、可靠性高[[S24,S33]]。

## 工作循环

### 卧式冷室压铸机工作循环

卧式冷室压铸机完整工作循环包括以下步骤[[S42,S32,S1]]：

1. **合模锁紧**
2. **给汤**：给汤机构将定量熔融金属注入压射室
3. **压射**：依次执行慢速压射→高速充填→终压增压，金属在高压下保压直至完全凝固
4. **推出余料**：压射冲头前伸推出余料
5. **开模顶出**：开模后顶出机构将铸件从模腔中顶出
6. **冲头复位**：压射冲头复位，完成一个生产循环

### 压射运动三个阶段

冷室压铸机的压射过程分为三个典型阶段[[S20,S1]]：

- **第一阶段（慢速压射）**：压射冲头以慢速推动金属液充满压室前端并堆聚在内浇口前沿，此阶段为压室内空气预留充足排气通道，同时防止金属液从浇注口溅出。为有效消除藏气问题，压射冲头在此阶段可采用匀加速运动。
- **第二阶段（高速压射）**：压射冲头快速移动，推动金属液填充进入模具型腔。
- **第三阶段（终压增压）**：压射冲头继续推进压实型腔内金属，直至金属完全凝固。该阶段需要压射系统配置专用增压机构实现。

## 与热室压铸机的比较

### 核心差异对比

| 对比维度 | 冷室压铸机 | 热室压铸机 |
|---|---|---|
| 压射室/冲头位置 | 与保温炉分离，不浸入金属液 | 压射室和冲头浸入保温炉熔融金属中 |
| 给料方式 | 需人工或自动舀取定量金属液注入压室 | 压射冲头回升时金属液自动从入料口进入压室，无需单独舀料工序 |
| 压射比压 | 高（动态30~90 MPa，增压50~300 MPa） | 低（20~50 MPa） |
| 增压机构 | 有独立的增压机构，可实现终压压实 | 无增压机构 |
| 建压时间 | 30 ms 以下，先进机型可达10 ms 以下 | 无此参数 |
| 适配合金 | 铝合金、镁合金、铜合金、黑色金属等高熔点合金 | 锌合金、铅合金、锡合金等低熔点合金；少数专用机型可压小型镁合金件 |
| 压铸件规格 | 可铸大尺寸/大质量件（最大铝合金件约60 kg） | 通常不适合大型件 |
| 最大锁模力 | 已超过36000 kN | 较小 |
| 生产率 | 需额外舀料工序，生产率相对较低（280 t冷室镁合金机约70 kg/h） | 无舀料工序，生产率更高（同规格280 t热室镁合金机可超过100 kg/h） |

**参考来源**[[S5,S14,S25,S19,S28,S8,S9]]

### 优劣势总结

冷室压铸机的优点是压射压力大，可适配大尺寸、高熔点合金铸件（铝合金、镁合金、铜合金及少量黑色金属）[[S28,S27,S8]]；缺点是热量损失大、操作流程较热室机繁琐，生产率不如热室机高[[S19,S32]]。

## 适配合金与典型应用

### 适用合金

冷室压铸机常规适配高熔点及对铁亲和性高的合金[[S27,S36,S8]]：

| 合金类别 | 典型牌号/说明 |
|---|---|
| 铝合金 | ADC10（A380）、ADC12（A383）、Al-Si系、Al-Si-Mg系、Al-Mg系等 |
| 镁合金 | AZ91D、AM60B、AM50A、AS41B等 |
| 铜合金 | 抗磁仪表配件、导电耐磨零件用 |
| 黑色金属 | 少量应用，对模具材料和工艺要求高 |

铝合金对铁有很高的化学活性且浇注温度较高，主要采用冷室压铸机，且最好选用卧式机型；镁合金主要采用冷室压铸机；铜合金因熔化温度高，通常只采用冷室压铸机[[S27]]。

### 典型产品场景

- **铝合金压铸件**：汽车结构件、3C电子外壳[[S27,S39,S45]]
- **镁合金压铸件**：轻量化汽车零部件、通讯设备壳体[[S27]]
- **铜合金压铸件**：抗磁仪表配件、导电耐磨零件[[S27,S45]]
- **典型产品实例**：汽车转向器、A380合金薄壁壳体等对强度和致密性要求高的受力件与安全件[[S22]]
