垂直分型无箱射压造型机是射压造型机的一种，造型时无需砂箱，砂型分型面与地面垂直。根据 GB/T 25370‑2020《铸造机械 术语》，其标准英文名称为 **vertical parted flaskless shoot‑squeeze molding machine** [[S19,S37]]。行业内也常称其为挤压造型机、垂直分型无箱高压造型机，习惯上将配套的自动化生产单元称为迪萨造型线 [[S30]]。该设备属于无箱造型机，压实比压大于 700 kPa，落入高压造型机范畴 [[S37]]，而同属无箱射压的还有水平分型脱箱射压造型机，两者在分型面、下芯便利性和生产率上存在明显界限 [[S15,S30]]。

## 发展简史
1963 年丹麦 DISA 公司向市场推出了第一台垂直分型无箱射压造型机；到 2006 年，该公司已向全球供应超过 1200 台此类造型线 [[S34]]。2002 年推出的 DISA 230 系列采用真空辅助填砂和双面挤压技术，将砂型错箱误差控制在 0.1 mm 以内，最高生产率可达 500 型/小时 [[S34,S31]]。

## 工作原理
垂直分型无箱射压造型机的一个完整工作循环由六个工序组成 [[S20,S2,S14,S30]]：

1. **射砂**：正压板和反压板合拢，围成密闭的造型室；压缩空气将型砂从顶部高速射入造型室，完成预紧实 [[S21,S31]]。
2. **压实**：正压板向造型室中心移动，对型砂进行高压压实（亦可采用正、反压板双向同时挤压） [[S11,S34]]。
3. **反压板起模（起模 I）**：压实后，反压板独立前移完成首次脱模，随后向上翻转 90°，让开砂型推出通道 [[S2,S10]]。
4. **推出合型**：正压板继续前移，将新成型的砂块推至与前一块砂型贴合，整串砂型在输送带上同步移动一个砂型厚度的节距 [[S31,S14]]。
5. **正压板起模（起模 II）**：正压板退回，完成二次脱模 [[S10,S30]]。
6. **造型室复位**：反压板向下翻转回到原位，与正压板重新组成造型室，进入下一循环 [[S2,S14]]。

整个循环中，压实、起模、合型均依靠主油缸前后活塞的移动完成；在不使用砂芯的情况下，生产率可超过 240 型/小时 [[S14,S20]]。

## 设备结构
### 整机组成
该设备主要由机座、造型室、正压板、反压板、射砂机构、主液压缸、液压系统、气动系统及电气控制系统构成 [[S34,S21]]。其中：
- **造型室**由侧框架、上下框架、耐磨底板、耐磨顶板、耐磨侧板和导柱等组成 [[S34,S21]]；
- 反压板上装有吹净模板的喷嘴，正、反压板均配有模板加热器 [[S34,S21]]；
- 电气控制早期使用凸轮控制盘与印刷电路板，后期普遍升级为 PLC 控制 [[S34,S21,S13]]。

### 射砂机构
标准射砂机构包含射砂筒、贮气包、射砂阀和射头等。大型机型可配置两级射砂机构：增大贮气包容积并设置两个射砂阀。先开启第一个射砂阀进行射砂，间隔 0.1～0.2 s 后开启第二个，双阀同时进气，射砂压力常用 0.4 MPa（4 kgf/cm²）。两级射砂可使砂型表面硬度平均达到 90 度以上，硬度差约 5 度，紧实度均匀性明显优于单级射砂 [[S11]]。

![垂直分型无箱射压造型机射砂机构装配图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/aea18ab6-ed87-4773-b102-4a5b4d2b4423/resource)

### 主油缸
主压实驱动部件为双向油缸，缸内设有隔板阀，将腔室分隔为前后两个独立区间，避免前后活塞运动相互干扰。前活塞杆和后活塞杆分别独立动作，即可完成压实、推型等全部主工序 [[S11]]。

垂直分型无箱高压挤压造型对型砂性能控制十分严格。下表为无箱高压挤压造型的典型型砂配比及性能指标 [[S16]]。

| 透气性 | ＞100 |
| 水分 (%) | 3.5～4.2 |
| 湿压强度 (kgf/cm²) | 1.4～1.7 |
| 湿拉强度 (gf/cm²) | ＞120 |
| 热湿拉强度 (gf/cm²) | ＞30 |

## 工装与气道设计
### 模板高度限制
不同规格机型对正、反压板侧模底板及模样总高度均有严格限制 [[S32]]：
- 500×400 mm 与 600×480 mm 机型：正压板侧一般 ≤ 230 mm，反压板侧 ≤ 80～85 mm；
- 800×600 mm 机型：正压板侧 ≤ 300 mm，反压板侧 ≤ 170 mm。

反压模板的高度还需避开其在起模翻转轨迹范围，防止与造型室干涉 [[S24]]。

### 气道设计要点
为提高深腔部位的射砂预紧实效果并防止高紧实度砂型起模时出现真空拉伤，气道设计应遵循以下要求 [[S24]]：
- 通气孔开设在型腔最高部位、死角、暗冒口顶部及芯座上；
- 排气槽开设在分型面上，从型腔通向砂型边缘；
- 模样深凹、转角等部位加装排气塞。

## 工艺特点
### 主要优势
- **无砂箱系统**：省去了砂箱采购、周转、存储和维护的全部成本，生产线布局紧凑，占地面积小 [[S36,S29,S34]]。
- **两面成型**：一块砂型两个面均可布置型腔，模板有效面积利用率高，在相同砂型尺寸下单位时间产出更多，且单位铸件型砂消耗降低 [[S36,S3]]。
- **高生产率**：无砂芯条件下可达 240 型/小时以上，新一代高端机型最高可达 550 型/小时，仅需 1 名操作人员值守 [[S4,S36,S18]]。
- **适用材料广**：可适配灰铸铁、可锻铸铁、球墨铸铁、铸钢、铝合金、黄铜等几乎所有常用砂型铸造合金 [[S29,S15]]。

### 局限性与注意点
- **下芯受限**：下芯操作只能通过专用夹具将砂芯置于垂直分型面上，单工位下芯时间不超过 7～8 s，否则会严重拉低整线节拍，导致该设备最适于无芯或少芯的中小铸件大批量生产 [[S36,S3,S24]]。
- **结构复杂度高**：整机结构精密，制造、调整和维护要求较高 [[S10,S14]]。
- **型砂要求苛刻**：必须有高紧实率、高湿强度和高透气性的型砂系统支持，否则易出现砂型损坏或铸件缺陷 [[S3]]。

## 与水平分型无箱造型机的对比
垂直分型无箱射压造型机与水平分型脱箱射压造型机同属无箱射压造型，但在以下方面差异明显 [[S15,S17,S5,S18]]：

| 比较维度 | 垂直分型无箱射压造型机 | 水平分型脱箱射压造型机 |
|----------|------------------------|--------------------------|
| 分型面方向 | 垂直 | 水平 |
| 下芯方式 | 需专用芯模工装，下芯空间小、时间短 | 可在水平分型面上直接下芯，操作方便 |
| 适用铸件类型 | 以无芯或少芯中小件为主 | 砂芯较多、形状较复杂的铸件 |
| 生产率范围 | 200～360 型/小时（最高 550） | 一般 90～120 型/小时 |
| 砂型成型 | 单腔体双侧压板挤压，两块模板均可成型 | 上下两个独立造型室分别成型 |

## 应用领域
该设备广泛应用于汽车、拖拉机、纺织机械、缝纫机、机床、管件及轻工机械等行业的中小铸件大批量生产，对无芯或少芯铸件的优势尤为突出 [[S31,S2]]。所生产的铸件材质覆盖灰铸铁、球墨铸铁、可锻铸铁、铸钢、铝合金、铜合金等 [[S29,S15]]。

## 安全与标准
- 术语定义由国家推荐性标准 GB/T 25370‑2020《铸造机械 术语》规定 [[S19,S37]]。
- 设计须满足铸造造型类设备通用安全规范，设置防止型砂射出的防护结构，负载运行时噪声声压级不得超过 85 dB(A) [[S25]]，并配置联锁安全装置以确保操作人员始终处于安全区域 [[S35]]。
