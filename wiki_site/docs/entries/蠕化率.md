## 定义

蠕化率（vermicular graphite rate / compacted graphite rate，亦称 vermicularity）是指蠕墨铸铁（compacted graphite iron, CGI）中蠕虫状石墨数量（或面积）占石墨总数量（或总石墨面积）的百分比，国家标准 GB/T 5611‑2017 中正式将其定义为“蠕墨铸铁中，蠕虫状石墨和部分团状、团絮状石墨占石墨总面积的百分比”[[S25,S44,S34]]。该参数是衡量蠕墨铸铁石墨形态是否达到蠕化要求的最核心指标，与球化率、片状石墨率共同构成石墨形态的全面描述。

石墨形态可通过石墨面积率进一步定量区分：球状石墨面积率 0.81‑0.98，团状石墨 0.61‑0.80，蠕虫状石墨 0.21‑0.40，片状石墨一般小于 0.10[[S10]]。在实际显微视场中，蠕化率 + 对应球/团状石墨的等效球化率 + 片状石墨率 ≈ 100%，评定中通常允许视场内存在不超过 5% 的片状石墨[[S27]]。

## 分类与分级

我国蠕墨铸铁金相检验标准将蠕化率分为 9 个等级，代号由蠕 95 至蠕 15，各级别对应的蠕虫状石墨数量下限如下表所示[[S4,S38,S24]]。

| 级别代号 | 蠕虫状石墨数量（面积百分比） |
|----------|-------------------------------|
| 蠕 95    | ≥95%                          |
| 蠕 90    | ≥90%                          |
| 蠕 85    | ≥85%                          |
| 蠕 80    | ≥80%                          |
| 蠕 70    | ≥70%                          |
| 蠕 60    | ≥60%                          |
| 蠕 50    | ≥50%                          |
| 蠕 40    | ≥40%                          |
| 蠕 15    | ≥15%                          |

> 注：蠕化率≥80% 为 GB/T 26655‑2011 规定的常规蠕墨铸铁基础要求，即二维抛光平面上至少 80% 的石墨应为蠕虫状，其余允许为球团状石墨，不允许出现片状石墨[[S34]]。

**国内外合格阈值的差异**  
- **国内工业实践**：考虑到多数蠕铁件用于替代高强度灰铸铁，国内标准将蠕化率下限普遍设为 ≥50%[[S9,S43]]。对于钢锭模、排气管、玻璃模具等承受热冲击的铸件，供需双方可协商采用 30%‑50% 的低蠕化率[[S34]]。  
- **国际/欧美规范**：ISO/TC 7.6 蠕铁委员会 1981 年暂行技术规范及多数欧美标准规定常规产品蠕化率 ≥80%，但同样允许部分特殊铸件的最低蠕化率放宽至 ≥50%[[S23,S43]]；ASTM A842 标准亦要求石墨以紧凑（蠕虫状）形态为主，本质上不允许存在片状石墨[[S17]]。

## 测定与评定

### 金相法基本要求
蠕化率的评定应采用金相检验法，优先从力学性能试块或指定部位切取试样。试样经抛光（不浸蚀）后，在 100 倍光学显微镜下观察，至少选取 3 个代表性视场，对照标准评级图进行比对[[S27,S13]]。石墨长径比（l/d）在 2‑10 范围内判定为蠕虫状石墨，允许存在低于 5% 的片状石墨[[S27]]。

## 主要影响因素

| 因素类别 | 影响规律 | 关键区间或备注 |
|----------|----------|----------------|
| 残余镁/稀土含量 | 活性镁含量过高会使球墨比例急剧增加，过低则蠕化不足、出现片状石墨 | 活性镁稳定区间 0.008%~0.016%[[S6,S33]]；残余稀土常规合格区间 0.045%~0.075%[[S41]] |
| 原铁液硫含量 | 硫消耗镁、稀土等蠕化元素，硫含量升高直接增大蠕化剂临界加入量，硫含量波动是蠕化率不稳的重要诱因 | 电炉熔炼要求 S≤0.03%，冲天炉熔炼要求 S≤0.06%[[S22]]；硫与镁的定量关系见[[S42,S2]] |
| 冷却速度 | 薄壁（高冷速）→ 促进球状石墨生成，蠕化率下降；厚大断面（低冷速）→ 蠕化衰退，石墨趋向片状，同样导致蠕化率降低 | 实测铸件厚度方向蠕化率分布：距金属型腔 0‑25 mm 区间蠕化率可保持 80% 以上，靠近金属型区迅速降至 30% 以下[[S26,S14]] |
| 蠕化剂加入量 | 加入量不足则蠕化不良，过量则球化比例升高，两者均偏离目标蠕化窗口 | 蠕化剂的临界加入量概算式：RE 临界 = k·[RE 残 + K' (S 原 − S 终)]，其中 k=1.3‑1.5[[S41]] |
| 孕育处理 | 消除白口倾向、延缓冲化衰退、促进石墨析出、提高蠕化率、细化晶粒 | 通常加入 0.4%~0.8% 的硅钡或 75 硅铁，炉前孕育与瞬时孕育结合[[S12,S22,S35]] |
| 浇注时间 | 处理后蠕化率呈先升后降的抛物线变化，约 22 min 后衰退加速 | 要求处理完成后 15‑20 min 内全部浇注完毕[[S11,S37]] |
| 处理温度 | 出铁温度过高加剧蠕化剂烧损，温度过低则反应速度慢、窗口短 | 建议出铁温度 1420‑1480 °C[[S12,S18]] |

## 与材料性能的关系

### 力学性能
蠕化率对蠕墨铸铁的强度、塑韧性、疲劳性能及弹性模量均有显著影响[[S40]]：
- **强度**：相同基体条件下，降低蠕化率（增加球状石墨比例）可提高屈服强度、疲劳强度及屈强比（σ₀₂/σᵦ）。珠光体基体的抗拉强度高于混合基体和铁素体基体。
- **塑韧性**：蠕化率升高时，断后伸长率与冲击吸收功随之升高，铁素体含量增加也更有利于塑性。
- **弹性模量**：取值范围 138‑165 GPa，蠕化率降低、截面减薄、珠光体量增多时模量值升高，泊松比稳定在 0.27‑0.28。

### 热物理性能
热导率随蠕化率升高而同步提升，整体介于灰铸铁与球墨铸铁之间[[S3,S16]]。当蠕化率 ≥95% 时，热导率接近灰铸铁；蠕化率低于 70% 后，热导率下降趋势尤为明显。

### 铸造与压铸工艺性能
- **流动性**：高蠕化率下，铁液碳当量接近共晶，经蠕化剂净化后流动性优异，可接近灰铸铁水平[[S30]]。
- **收缩性**：体收缩率随蠕化率升高而逐步降低，趋近灰铸铁；蠕化率越低，体收缩率越大，趋近球墨铸铁。获得无缩孔、缩松的致密铸件难度介于灰铸铁与球墨铸铁之间[[S15]]。
- **铸造应力与热裂**：残余镁、稀土含量远低于球墨铸铁，铸造应力较小，热裂倾向明显优于球铁[[S15]]。
- **压铸特殊考量**：压铸过程中的高压充型与快冷易导致蠕化窗口收窄，需将活性镁严格控制在 0.008%‑0.016% 窄区间，并通过热分析在线检测原铁液活性氧、总硫和过冷度参数，动态调整蠕化剂与孕育剂加入量，防止过量球化使蠕化率不达标[[S6,S33]]。

### 性能变化趋势汇总

| 性能指标 | 蠕化率升高 → 影响趋势 | 备注 |
|----------|----------------------|------|
| 抗拉强度、屈服强度、疲劳强度 | 降低 | 珠光体基体强度更高，但受蠕化率影响规律一致[[S40]] |
| 断后伸长率、冲击韧性 | 升高 | 铁素体基体韧性更优[[S40]] |
| 热导率 | 升高 | 蠕化率 ≥95% 时接近灰铸铁，<70% 后明显下降[[S16]] |
| 体收缩率 | 降低 | 逼近灰铸铁，致密性改善[[S15]] |
| 流动性 | 无显著恶化，高蠕化率下良好 | 接近共晶成分，净化效果好[[S30]] |
| 铸造应力与热裂倾向 | 降低 | 残余 Mg、RE 量低[[S15]] |

## 工艺控制

### 蠕化处理方法
蠕化处理的工艺选择直接影响蠕化率稳定性。常见方法包括[[S39]]：
- **包底冲入法**：适用于有自沸腾能力的蠕化剂；无自沸腾能力时需预先加入少量 FeSiMgRE 或 FeSiMgREZn 合金作为引爆剂。
- **出铁槽随流加入法**：适用于冲天炉熔炼，粒状蠕化剂均匀撒向铁流。
- **中间包法**：铁液先流入小型中间包与同时撒入的蠕化剂混合，再流入浇包。
- **炉内加入法**：适用于感应炉，出铁前加入稀土硅铁合金，借电磁搅拌促进熔吸，处理温度需高于 1480 °C，适合中大件生产。
- **喂丝法**：采用包芯线（冲天炉用：RE 19‑24%、Mg 3‑5%、Ca 2‑3%、Ba 1.5‑3.5%、Si 40‑45%；电炉用：RE 19‑24%、Mg 2‑4%、Ca 3‑4%、Ba 0.5‑2.5%、Si 40‑45%），由喂丝机精确控制给进速度与长度，结合 SinterCast 在线热分析系统可实现“欠处理”后动态补充，大幅提升蠕化率稳定性，是现代高要求蠕铁件量产的有效手段[[S39,S41]]。

### 炉前快速检测与质量保证
蠕化率控制必须依托炉前多手段联检[[S12,S18,S11,S37]]：
- **三角试片法**：浇注三角试片，冷却后断口呈银白色、两侧轻微凹缩、顶面略有凹陷并伴有均匀黑斑点及类似球铁的“铛”声，即判断为蠕化良好。
- **快速金相法**：试样蠕化率比目标要求低约 10% 左右，作为安全余量。
- **热分析法**：通过热分析仪消除人工误判，与三角试片、快速金相配合使用，可大幅降低蠕化率失控风险。

### 压铸场景下的专用控制要点
压铸蠕墨铸铁件在通用控制基础上需要特别注意[[S6,S33]]：
- 活性镁必须维持在 0.008%‑0.016% 的极窄区间；
- 铁液共晶度应保持为共晶或微亚共晶状态；
- 通过热分析实时测量原铁液活性氧、总硫和过冷度，动态计算蠕化剂加入量，避免过量球化导致蠕化率不达标。

## 压铸领域的适用性与说明

检索确认，蠕化率术语在压铸行业中并无独立特殊定义，其内涵与通用铸造领域完全一致[[S44,S21,S28]]。常规压铸以有色金属为主，很少涉及铸铁石墨形态评价，但特定铸铁压铸件（如高压充型蠕墨铸铁件）的生产中，仍需严格按上述蠕化率定义及控制方法执行[[S6]]。因此，当前词条所载定义、分级与工艺控制均直接适用于压铸相关的铸铁材料评价。

## 术语歧义提醒

英文中 “vermiculite rate” 指蛭石相关参数，与铸造领域的 “vermicular graphite rate” 是两种完全不同含义的术语，使用时应注意区分[[S19]]。
