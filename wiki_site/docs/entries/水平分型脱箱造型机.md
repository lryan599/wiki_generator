## 定义与分类

水平分型脱箱造型机（horizontal parting flaskless molding machine）是指在可脱式砂箱内制造上、下型，经下芯、合箱后脱去砂箱，且砂型的分型面为水平位置的造型机 [[S18]]。该机型适用于生产壁厚不大的中、小铸件，按紧实方法可分为压实、微振压实和射压三类 [[S18]]。其中采用射砂预紧实加压实工艺的子类在国家标准中被定名为**水平分型脱箱射压造型机**（horizontal parted flaskless shoot-squeeze molding machine），归类于脱箱造型机大类下的射压造型机子类 [[S27]]。

该设备与两类相邻机型存在明确的分类边界：

- **与垂直分型无箱射压造型机的区别**：前者的分型面为水平布置，下芯操作便捷，适配复杂砂芯铸件；后者的分型面为竖直方向，下芯困难，仅适用于无芯或简单小砂芯的中小铸件 [[S26,S19]]。
- **与有箱高压造型机的区别**：前者仅在造型紧实阶段使用临时可脱砂箱，合箱后砂箱与砂型完全分离，无需配套砂箱长期流转存储系统，占地面积更小 [[S19,S21]]。

## 结构组成

水平分型脱箱射压造型机（以下以射压类为代表）主要由以下核心部件组成 [[S10,S12]]：

| 部件名称 | 主要功能 |
|---------|---------|
| 射砂机构 | 利用压缩空气骤然膨胀将型砂高速射入上、下造型室，完成预紧实 |
| 压实机构 | 由上、下型压板及液压驱动增压组件组成，相向动作完成砂型最终紧实 |
| 上、下砂箱 | 临时容纳型砂形成砂型的可脱式箱体 |
| 模板穿梭机构 | 驱动双面模板框在造型工位与更换工位之间穿梭运动 |
| 模板框 | 承载上下相背安装的双面模板，上模板朝上组成上造型室，下模板朝下组成下造型室 |
| 脱箱机构 | 集成砂箱升降驱动组件和高精度导向柱，在合型后将砂箱从固化砂型上完全脱除 |
| 上、下型压板 | 与压实机构配合，对造型室内型砂施加压实力 |
| 横梁与底座 | 整机承载框架 |
| 控制系统 | 实现自动循环控制 |

![水平分型脱箱射压造型机结构示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/0e1b5a5b-a01d-4338-a167-8d263e8310d0/resource)

上图为水平分型脱箱射压造型机的核心工位简化结构示意图，直观展示分型压实工位的核心动作状态 [[S2]]。

**射砂系统**分为顶射式、顶射加底射式两种结构，通过压缩空气骤胀将型砂高速射入上下造型室，可适配不同深浅的模样，保证深凹部位填砂均匀 [[S10]]。采用顶射加底射式时，可根据铸件形状和大小决定分段射砂或同时射砂，提高了射砂效果 [[S11]]。

**压实机构**由上下型压板、液压驱动增压组件组成，上下压板相向动作完成砂型最终紧实 [[S20]]。

**模板更换机构**多采用穿梭式模板框结构，上下双面模板相背安装在模板框中，更换时仅需将双面模板对准定位销放置即可完成安装，操作快捷，换模耗时短 [[S12]]。

**脱箱机构**集成砂箱升降驱动组件、高精度导向柱，在合型完成后将临时砂箱从固化砂型上完全脱除，砂箱可直接返回造型工位循环使用 [[S23]]。

**砂型搬运装置**典型如APT砂型搬运系统，不使用夹板结构夹持砂型，可稳定将脱箱后的完整砂型输送至铸型输送机，输送过程中不会挤压损坏砂型边缘 [[S12]]。

## 工作原理

水平分型脱箱造型机的完整工作循环包含以下步骤 [[S12,S11,S6]]：

| 步骤 | 操作内容 |
|------|---------|
| 1. 初始工位就位 | 上下砂箱合型，将双面模板夹持住 |
| 2. 射砂预紧实 | 压缩空气骤胀将型砂射入上、下造型室，完成预紧实 |
| 3. 高压相向压实 | 上下压板相向动作，对型砂施加高比压，完成最终紧实 |
| 4. 双向平稳起模 | 上下砂箱分开，实现起模 |
| 5. 模板移出工位 | 模板框从造型工位移出 |
| 6. 下芯操作 | 在分型面上放置砂芯 |
| 7. 合箱定位 | 上下砂型合拢，导向定位 |
| 8. 脱箱回收空砂箱 | 将砂箱从固化砂型上脱除，砂箱返回循环使用 |
| 9. 砂型推送至输送机 | 将脱箱后的完整砂型移至铸型输送机，送往浇注工位 |

该全流程可实现高度自动化，部分高精度机型合箱错箱精度低至0.02 mm [[S6]]。AMF系列造型机的具体造型过程包括：模板进入后合型、翻转提升射砂压实、翻转下降开箱脱模模板退出、砂型检查下芯、合型脱箱、砂型推出六个阶段 [[S5]]。

## 核心工艺参数

### 射砂压力

射砂压力普遍为0.15～0.35 MPa（对应2～3 kgf/cm²） [[S5,S16]]。针对深模样高填砂需求或型砂流动性差的情况，可将射砂压力提升至3～5 kgf/cm² [[S16]]。

### 压实比压

重力加砂类机型（如亨特HMP系列）最大压实比压可达1.05 MPa [[S20]]；射压类机型压实比压常规取值为5～8 kgf/cm² [[S16]]；部分高压机型可按需无级调整型面挤压压力 [[S23]]。

### 砂型尺寸

主流机型适配的模板尺寸、最大砂型尺寸覆盖最高至720 × 720 mm规格区间，砂型厚度可按需灵活调整 [[S3,S7]]。

### 生产节拍

| 工况 | 典型生产率 |
|------|----------|
| 不下芯 | 最高120～200型/h |
| 下芯（单工位） | 额外增加4～6 s下箱出入时间 + 下芯时长 |

双工位或多工位机型可将下芯与造型工序并行，不会降低整体生产效率 [[S25,S23,S11,S8]]。

### 合箱精度

| 机型/类别 | 合箱错型精度 |
|----------|-------------|
| 德国AGM系列 | 0.02 mm |
| 亨特系列部分机型 | 低于0.15 mm |
| 行业一般水平 | 0.1～0.15 mm |

上述数据来源于文献 [[S6,S4,S5]]。

## 应用特点与范围

### 适配材质与铸件类型

该设备可生产铸钢、球墨铸铁、可锻铸铁、铸造非铁合金等材质的中小型铸件，典型铸件重量覆盖数公斤级至16 kg级区间 [[S8,S24,S17]]。适用于汽车、拖拉机、电力、轻纺、机械制造、五金水暖、缝纫设备、摩托车、机床、家用电器等行业的多品种中小批量生产场景 [[S5,S8,S17,S24]]。

### 工艺适配优势

水平分型面的设置使得下芯操作可在分型面上直接进行，下芯便利性远高于垂直分型脱箱造型机，对带复杂内腔、多砂芯的铸件适配性更好 [[S12,S5]]。同时，仅需对原有手工造型双面模板稍加改造即可适配新机型，非常有利于传统铸造厂老旧生产线改造升级 [[S24]]。

## 优势与局限

以下为水平分型脱箱造型机与垂直分型脱箱造型机（无箱射压造型机）、常规有箱高压造型机在六个维度上的特性对比 [[S14,S22,S21,S5,S4]]：

| 对比维度 | 垂直分型脱箱造型机 | 水平分型脱箱造型机（本条目） | 常规有箱高压造型机 |
|---------|-----------------|---------------------------|-------------------|
| 生产效率 | 最高（可达550型/h） | 中等（120～200型/h） | 较低（90～120型/h） |
| 下芯便利性 | 最低（仅适配特殊悬挂式砂芯） | **最高**（分型面上直接下芯） | 中等 |
| 占地面积 | 最小 | 中等 | 最大（需额外砂箱存储转运系统） |
| 设备投资成本 | 最低 | 中等 | 最高 |
| 砂型紧实度均匀性 | 最低 | 中等 | 最优 |
| 适用产品类型 | 少芯简单小型铸件大批量生产 | 多芯复杂中小型铸件多品种批量生产 | 大尺寸高要求铸件大批量生产 |

**优势**：水平分型下芯便利性极高，适配复杂砂芯铸件；双面模板改造成本低，便于老线改造；与有箱造型相比，无需大量砂箱流转存储，投资和占地均较小 [[S24,S12,S21]]。

**局限**：生产率显著低于垂直分型无箱射压造型机；受砂型尺寸限制，不适合大型铸件 [[S5,S22]]。
