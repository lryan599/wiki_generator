## 概述

在压铸领域，**本体试样**（specimen from casting body）指从成品压铸件的指定部位直接截取并加工得到的力学性能测试试样，用以真实反映压铸件在实际服役状态下的组织与性能表现[[S4]]。该术语与"单独铸造试样"（单铸试样）相对，属于压铸件质量检验体系中的核心概念之一，在标准文献中通常将本体取样规定为可选或仲裁性的检测方式。

本体试样的正式英文对应术语在ISO标准中多表述为**specimen from casting body**或**test specimen cut from casting**[[S5,S18]]；在ASTM标准体系中，当需要测定铸件本体真实性能时，拉伸试样须从铸件图纸标注的指定位置切取（specimens cut from the locations designated on the drawings）[[S1]]。NADCA产品标准将本体切取试样列为铸件关键服役性能核验的优先采样方式[[S34]]。在国内标准体系中，"本体试样"是通用规范术语，在铸造工艺图上通常以红实线或细实线绘制指定取样区域并标注"本体试样"字样[[S26]]。

## 定义与核心属性

本体试样的核心属性可从以下四个维度加以界定：

1. **制取方式**：从已完成的压铸件毛坯上，经线切割或机加工方式在指定部位截取所需形状和尺寸的测试小样[[S4,S10]]。
2. **性能代表性**：本体试样的测试结果反映了压铸件在真实充型与凝固条件下的实际组织、缺陷分布及性能水平，不受独立试样模具热平衡差异的影响[[S4,S11]]。
3. **破坏性**：取样过程将导致被测铸件本体报废，因此本体试样仅适用于新产品试验验证阶段或大批量稳定生产条件下的抽样检测[[S4]]。
4. **标准合规性**：本体试样的取样位置、尺寸及验收性能阈值须符合供需双方协定或相关国家标准规定。在压铸铝合金体系中，GB/T 15114-1994规定本体试样力学性能不得低于同批次单铸试样性能的75%[[S15,S27,S31]]；在锌合金压铸件体系中，本体取样检验的力学性能要求完全由供需双方协商确定[[S9]]。

## 与单独铸造试样的对比

单独铸造试样（单铸试样，separately cast test specimen）是指在不依附于实际生产压铸件的专用独立压铸模具中，按GB/T 13822-1992规定的标准化工艺直接压铸成型、用以测定合金固有性能的标准试样[[S6,S27,S4]]。两者之间的本质区别见表1。

**表1 本体试样与单独铸造试样的核心区别**
| 对比维度 | 本体试样 | 单独铸造试样 |
|---|---|---|
| **制取方式** | 从成品压铸件指定部位机加工截取 | 在专用独立模具中单独压铸成型 |
| **性能代表性** | 反映实际铸件服役条件下的真实组织与性能（含气孔、缩松、组织梯度） | 反映合金在理想标准压铸工艺下的固有性能，与实际铸件存在偏差 |
| **性能阈值（铝合金）** | 不低于同批次单铸试样性能的75%（GB/T 15114-1994） | 符合GB/T 15115规定的合金最低达标值 |
| **破坏性** | 破坏被测铸件，无法用于批量全检 | 不涉及实际生产铸件 |
| **应用场景** | 新产品试制验证、大批量稳定生产后抽样检测、性能争议仲裁 | 常规出厂验收的通用检测方式 |

参考来源：[[S4,S27]]

GB/T 13822-1992对单铸试样做出了系统的标准化规定：A型拉伸试样（直径φ6.4mm，标距50mm）用于抗拉强度和断后伸长率测定，B型拉伸试样用于抗拉强度比较试验和硬度测定，冲击韧性试样试验前截为两根，试验时摆锤冲击试样最窄面[[S24,S27,S12]]。

## 标准体系中的定位

### 国内标准

在现行国家标准体系中，本体试样的地位随着压铸合金类型的不同而有所差异：

- **铝合金压铸件**（GB/T 15114-1994）：第3.2.2条款规定，当采用压铸件本体检验时，指定部位切取试样的力学性能不得低于单铸试样的75%，有特殊要求可由供需双方协商确定[[S31,S15,S27]]。
- **锌合金压铸件**（GB/T 13821-2023）：第5.2.3条款规定，本体取样检验时的力学性能要求由供需双方商定，该标准以单铸A型拉伸试样的力学性能作为检验的参考依据[[S9,S23]]。
- **镁合金压铸件**（GB/T 25747-2022）：取样规定表中明确力学性能检验的取样方式包含本体取样，每批次力学性能测试不少于3根试样[[S8]]。检验结果的判定规则为：任一试样的力学性能不合格时，应从该不合格试样所在批中另取双倍数量的试样进行重复试验，重复试验结果全部合格则判该批产品合格[[S8]]。
- **铜合金压铸件**（GB/T 15116-2023）：取样规定表对不同检验项目明确了本体取样规则及每批次的取样数量要求[[S29]]。

### 国际标准

在ISO标准体系中，ISO 7722（铝合金铸件通用交货条件）规定，当无法制备符合要求的附铸或单铸试样时，可采用从铸件本体切取试样（specimen from casting body）的方式进行力学性能核验[[S18]]。

ASTM B85铝合金压铸件产品标准的附录X1中专门设置了《Table X1.2 Tension Test Minimums for Specimens Cut from Castings》，明确规定每件被检铸件须至少取3件本体试样分别覆盖薄、中、厚截面区域[[S2]]。ASTM B557金属材料拉伸试验方法则要求，本体拉伸试样须从铸件图纸标注的指定位置切取，若未指定取样位置，则需覆盖不同厚度区域且避开直浇口和冒口下方区域[[S1,S28]]。

NADCA发布的《Product Specification Standards for Die Casting》第7章"质量保证"章节将本体切取试样确认为铸件关键服役性能核验的优先采样方式，适用于汽车、航空等高端压铸零部件的入厂复验场景[[S34]]。

## 工艺代表性优势的物理机制

本体试样之所以比单铸试样更能真实反映压铸件的实际服役性能，其物理根源主要体现在三个方面：

### 表皮激冷效应与组织梯度

压铸件在充型凝固过程中，表层金属液接触低温模具发生激冷，形成厚度约0.5mm的等轴细晶冷硬层；从铸件表层向中心区域，压室预结晶组织（ESCs）的尺寸和面积分数逐渐增大，晶粒形貌从细小等轴晶过渡到粗大枝晶[[S11,S30]]。这种从表层到心部连续变化的微观组织梯度，直接决定了铸件整体力学响应的非均匀性，单铸试样因模具热平衡和凝固路径与生产铸件不同而无法完全复现这一特征。

### 气孔与缺陷分布的还原

压铸件内部气孔并非均匀分布。近浇口位置、厚壁区域的孔洞缺陷总体积显著高于远端区域，气孔类型可区分为圆整气孔、气缩孔、岛状缩松和网状缩松四类[[S37,S7]]。单铸试样因采用独立浇注系统和截然不同的几何结构，无法复刻实际铸件中由流道距离、壁厚差异和局部补缩条件共同决定的真实缺陷分布形态。

### 异构强化效应

压铸本体梯度异构组织在拉伸变形过程中，会从表层到心部产生应变梯度，从而诱导几何必需位错（GNDs）的堆叠，产生异构强化效应（hetero-structure strengthening）[[S13]]。该机制使得本体试样在变形时各层组织之间的力学行为不匹配得到自我补偿，测试结果与实际铸件服役性能的匹配度远高于组织相对均一的单铸试样。

## 取样位置与制备

### 取样位置选择原则

本体试样的取样位置直接影响测试结果的代表性和变异性。取样位置的选择遵循以下原则[[S4,S10]]：

- 优先选取压铸件上**受力最大的关键服役区域**，以使检测结果能够反映铸件在服役工况下的实际承载能力；
- 亦可选取工艺模拟预判的**易产生气孔缺陷位置**，开展针对性验证取样；
- 当铸件不同区域的壁厚差异显著时，应分别覆盖薄、中、厚截面[[S2]]。

![典型压铸件本体取样位置及试样尺寸示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/9a84b192-3ec6-47d6-9eb2-9ff6d3e22d92/resource)

### 制备流程

本体试样的金相及力学性能试样制备通常经过以下流程：采用精密切割机完成取样初加工，随后通过自动镶嵌机制备金相试样，依次使用不同粒度碳化硅砂纸研磨，最后以0.05μm粒度金刚石抛光膏抛光至镜面。金相试样除夹杂物检验外一般需进行浸蚀处理，夹杂物检验可直接利用明场、暗场、偏光等功能完成鉴别[[S10]]。在气孔与缩孔的微观鉴别中，气孔呈现光滑边缘和光亮内腔，缩孔则呈现不规则边缘和暗色内腔[[S4]]。

## 应用场景

### 首件检验（FAI）

在新产品试制阶段，本体试样用于验证压铸工艺方案和模具设计的合理性，准确确认铸件关键受力区域的实际性能水平，排查缩孔缺陷、力学性能不足等潜在质量风险[[S4,S3]]。

### 批次质量控制

在大批量稳定生产阶段，按批次抽样从压铸件本体切取试样进行性能核验。标准要求力学性能测试本体试样每批次不少于3根；首次检测不合格时允许加倍取样复验，若复验不合格则可判定整批产品性能不合格[[S8,S23]]。

### 产品认证与争议仲裁

当单铸试样或附铸试样的检测结果出现争议时，从压铸件本体指定位置切取的试样的检测结果可作为合规判定的仲裁依据，满足产品质量认证的溯源性要求[[S14,S19]]。ISO 7722亦明确规定，当订货合同规定必须在铸件本体切取试样上测定力学性能时，则不必再在单铸试样上测定合金力学性能[[S18]]。

## 局限性

本体试样的应用受制于以下几项关键局限[[S4,S20,S25]]：

1. **破坏性检测限制**：本体取样属于破坏性检测行为，取样后的被测铸件将直接报废，仅适合试制阶段或大批量稳定生产后的小比例抽样，无法用于产线逐件检测。
2. **性能离散性**：压铸件不同壁厚位置的冷却速率差异显著，导致不同取样位置的晶粒尺寸和致密度不一致，本体试样的测试结果离散度明显高于统一条件下铸造的标准单铸试样。
3. **薄壁件取样困难**：薄壁压铸件的可取样空间不足，难以加工出符合国家标准尺寸的拉伸或冲击试样，仅能加工微型试样开展部分性能测试。
4. **附铸试样的一致性争议**：行业内普遍存在附铸试样（integrally cast test specimen）因截面尺寸差异导致与压铸件本体凝固冷却速度不一致、性能存在偏差的争议，仅当附铸试样位置的冷却速度与目标本体区域完全匹配时，其测试结果才能近似代表本体性能[[S4]]。

|---|---|---|
| **按获取方式** | 纯本体试样（截取型） | 从成品压铸件指定位置直接截取加工 |
|  | 附铸试样（cast-on） | 压铸时随铸件一体铸出，作为铸件部位延伸 |

|  | 冲击韧性试样 | 冲击功测定，试验前截为两根 |
|  | 金相组织试样 | 微观组织、缺陷形貌及晶粒度分析 |
|  | 化学成分分析试样 | 合金成分光谱或化学分析 |

## 与其他概念的关系

- **附铸试样（cast-on test specimen / integrally cast test piece）**：附铸试样是本体试样按获取方式分类下的一个子类，指在压铸过程中将试样与铸件在同一模具内一体铸出、作为铸件某一指定部位的延续。不同于纯本体试样需从铸件切除，附铸试样单独成型于模具的试样型腔区域[[S4]]。但需注意的是，其性能代表性与纯本体试样之间可能存在差异（见局限性第4条）[[S4,S33]]。

- **单铸试样（separately cast test specimen）**：与本体试样互为对照，是压铸合金固有性能的检测基准。当实际压铸件的组织结构或尺寸限制不允许本体取样时，单铸试样的检测结果仍是最主要的验收依据[[S6,S4]]。

- **仲裁试样**：在供需双方对单铸或附铸试样的检测结果存在争议时，本体试样可上升为仲裁依据。相关标准如GB/T 15114明确规定"有特殊要求可由供需双方商定"，ISO 7722亦在控制类型1中给出了本体试样作为替代检测方案的使用条件[[S14,S19,S18]]。
