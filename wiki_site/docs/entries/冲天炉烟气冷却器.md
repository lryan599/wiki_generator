## 概述

冲天炉烟气冷却器（cooler for cupola furnace fume）是铸造烟气处理系统中的关键设备，用于将冲天炉排出的高温烟气进行冷却，以满足后续除尘与排放要求。依据 GB/T 25370‑2020《铸造机械 术语》的定义，该装置通过热交换或直接加入常温空气稀释的方式冷却冲天炉排出的高温烟气[[S12,S27]]。冲天炉正常熔炼时出口烟气温度高达 500~600 ℃，打炉操作时最高可达 900 ℃，必须进行有效冷却[[S22,S20]]。设置烟气冷却器的核心原因包括：保护下游袋式除尘器滤袋，使其处于滤料允许的长期工作温度范围；降低高温烟气的体积，缩减除尘及引风设备的规格和运行能耗；部分冷却器还能在换热过程中同步沉降粗颗粒烟尘，减少除尘器的入口粉尘负荷[[S22,S20,S4,S7]]。

## 工艺作用与流程位置

冲天炉烟气冷却器安装在冲天炉高温烟气出口的下游、袋式除尘器的上游[[S20,S29]]。在典型的二级冷却系统中，第一级粗冷却单元（如旋风水冷却器或水冷套管）紧邻冲天炉出口，对烟气进行初次降温和粗颗粒预分离；第二级精冷却器置于粗冷却单元之后、除尘器入口之前，确保进入袋式除尘器的烟气温度满足滤料耐温要求[[S20]]。

水冷式系统的典型配置如下：冲天炉 → 膨胀节 → 电动活络盖板 → 旋风水冷却器 → 多管水冷却器 → 袋式除尘器 → 引风机组 → 烟囱，同时配有独立循环水系统[[S20,S9]]。风冷式系统则无须循环水系统，流程为：冲天炉 → 膨胀节 → 强制式风冷却器 → 除尘器 → 出风管 → 风机组[[S20,S32]]。某些多除尘级别的系统中，冷却器下游还可依次串联旋风除尘器、水力除尘器、引风机和烟囱[[S13,S15]]。

## 分类与结构形式

### 冷却方式分类

冲天炉烟气冷却器可按冷却介质和换热接触形式进行分类（表1）。

| 分类依据 | 类型 | 说明 | 引用 |
|---------|------|------|------|
| 冷却介质 | 水冷式 | 适用于水资源丰富地区，利用循环水带走烟气热量 | [[S20,S6,S12]] |
|  | 风冷式（强制空冷） | 适用于北方缺水地区，通过空气冷却烟气，系统简单 | [[S20,S6,S12]] |
| 换热接触形式 | 间壁式（表面式） | 烟气与冷却介质通过金属换热壁进行热交换，不发生混合，应用最广 | [[S12,S6,S33]] |
|  | 直接混合式（稀释/喷淋） | 将常温空气或水直接喷入高温烟气中降温，结构简单但增加系统处理风量或引入水雾 | [[S12,S6,S33]] |
|  | 蒸发冷却式 | 利用水蒸发相变吸热，冷却效率高，但管外侧易结垢 | [[S20,S33]] |

常见结构形式包括旋风水冷冷却器、多管水冷却器、强制风冷冷却器、列管式冷却器等。其中旋风水冷冷却器和多管水冷却器结构简单、造价低，在冷却烟气的同时可依靠旋流作用除去粗颗粒烟尘，常作为初级冷却除尘设备使用[[S20,S4]]。

### 间壁式冷却器类型

间壁式冷却器按换热介质组合可进一步分为三类（表2），其传热特性与适用条件差异明显。

| 类型 | 工作原理 | 传热系数 / 特点 | 引用 |
|------|---------|-----------------|------|
| 炉气‑空气对流型 | 冷热流体均通过器壁对流换热，冷却空气直接排空或回收利用，无需介质回收系统 | 换热系数低，所需换热面积大，体积庞大造价高 | [[S33]] |
| 炉气‑水对流型 | 冷却水在封闭管路内流动，烟气在外部（或管内）通过风机驱动流动，两者对流换热 | 传热系数 55~170 W/m²·℃；换热面积小，结构紧凑，造价较低，冬季需防冻 | [[S33]] |
| 炉气‑水对流蒸发型 | 冷却水在管外开放空间喷淋，同时通过对流和蒸发作用带走热量 | 冷却效率高，所需换热面积小；但管外壁易结垢影响长期效果 | [[S33]] |

表3列举了四种典型的间壁式炉气冷却器结构及其功能特点，均基于文献[S21]提供的信息。

| 序号 | 冷却器形式 | 结构特点与说明 |
|------|------------|---------------|
| 1 | 炉气‑水顺流冷却器 | 炉气在管内自上而下流动，冷却水在管外流动；设有冷却水流道隔板、下部灰箱和卸灰器；通过调节冷却水流量和水温可控制炉气出口温度，回收热水可用于采暖或洗浴[[S21]] |
| 2 | 炉气下行外水冷冷却器 | 炉气先向下流动至下部灰箱再折返向上；壳体内部砌有耐火搪衬，可适应 600 ℃ 以上的高入口烟气温度；兼具粗颗粒除尘功能[[S21]] |
| 3 | 两级串联空冷器 | 两台空气冷却器串联，每台配有独立冷却风机；通过调节单台或两台冷却空气流量控制炉气温度，热侧排风可回收用于烘干[[S21]] |
| 4 | 喷淋蒸发式冷却器 | 炉气多程流经管束外部，管外布置冷却水喷嘴；传热系数高、换热面积小，但需在入口设置预除尘工序以防管束积灰[[S21]] |

### 结构组成

冲天炉烟气冷却器的主要部件包括[[S21,S2]]：

- 换热管束（冷却管），用于实现烟气与冷却介质间的热量传递；
- 承压壳体，保证气体通道的密封与结构强度；
- 导流隔板/折流板，控制流道气流分布；
- 清灰装置（吹灰器、振打装置等），用于清除受热面积灰；
- 烟气进/出风口及冷却介质（空气/水）进/出口；
- 热应力补偿金属波纹管（膨胀节），吸收高温工况下的热变形；
- 高温侧耐火搪衬，保护壳体并适应高入口温度；
- 下部灰箱及卸灰器，收集并排出沉降的粗颗粒粉尘。

在材料选择方面，常规低碳钢管束在含硫高温烟气中容易发生露点腐蚀和高温硫化腐蚀。对于高腐蚀工况，可选用 25%Cr‑20%Ni‑1%Nb 的奥氏体不锈钢（310Nb），其抗烟气高温硫化腐蚀寿命可比普通低碳钢提高至少 3 倍[[S16]]。当管内介质为液态水、壳侧为低温烟气时，ASME 规范允许采用无填充、无添加剂的 PTFE（聚四氟乙烯）或 PFA（可熔性聚四氟乙烯）管材作换热管。PTFE 管最高允许设计压力 800 kPa、最高工作温度 163 ℃；PFA 管最高允许设计压力 758 kPa、最高工作温度 163 ℃，壳侧烟气最高温度不应超过 195 ℃[[S11,S26]]。这些非金属管材可有效抵抗烟气低温露点下的酸腐蚀。

## 常见问题与预防措施

由于冲天炉烟气温度波动大、含尘量高且含有硫氧化物，冷却器在运行中面临积灰、磨损、腐蚀及热膨胀等多重挑战。

### 积灰堵塞

烟尘在 650 ℃ 以上呈半熔黏结状态，极易附着在冷却器壁面上形成坚硬的沉积物，导致换热效率下降、烟气流通阻力升高，严重时使引风不足[[S1,S14]]。防控措施包括：

- 设计足够大的冷却室，使高温烟气快速冷却至 600~650 ℃，令半熔烟尘转变为松散的固体颗粒[[S14,S3]]。
- 在 650 ℃ 以下温区，受热面排管采用垂直悬挂、与烟气流向平行的布置，管间距保持在 200~300 mm，防止烟尘搭桥[[S3]]。
- 配备吹灰器或振打装置，并在管道转弯处开设清理孔，便于日常检查和清灰[[S3,S13]]。

### 管壁磨损

管壁磨损速率与烟气流速的 3.25 次方成正比，当烟气含尘量大于 50 g/m³ 时磨损问题尤为突出[[S10]]。规范要求在此条件下将冷却器内烟气流速控制在 8 m/s 以下，同时优化气流分布，减少局部涡流和颗粒对管壁的直接冲刷[[S10,S3]]。

### 酸露点腐蚀

当冷却器管壁温度低于烟气硫酸露点时，烟气中的 SO₃ 与水蒸气生成的硫酸会凝结在管壁上引发严重化学腐蚀，且随腐蚀时间延长，管材磨损量显著上升[[S10,S17]]。防控措施包括：

1. 根据烟气中 SO₂、H₂O 的体积分数及 SO₂→SO₃ 转化率（一般 6%~10%），按公式（7‑1）和（7‑2）计算硫酸露点温度，进而确定最低允许壁温[[S10]]。
2. 提高冷却器运行压力，使管壁温度始终高于露点温度；采用气密性壳体设计，防止外界冷空气漏入造成局部降温[[S10]]。

### 热膨胀泄漏

冲天炉烟气温度在正常熔炼与打炉阶段之间大幅度变化，冷却器本体和连接管道将产生显著热位移。若未设置足够的膨胀补偿结构，累积的热应力会导致管壁、焊缝开裂，引发烟气或冷却介质泄漏[[S22]]。设计要求在冷却器与炉体连接的上下游管段安装金属波纹管膨胀节，以吸收全工况下的热变形[[S20,S22]]。

### 下游糊袋防控

若冷却器出口烟气温度低于露点，水汽和酸凝结物进入袋式除尘器会使滤袋表面的积灰形成糊状物，堵塞滤袋孔隙，导致除尘阻力剧增、滤袋失效[[S20]]。应对措施是在冷却器出口设置温度实时监测与报警装置，并与引风机联锁；当温度超过滤袋耐温限值时，自动开启紧急冷风阀混入常温空气，确保进入除尘器的烟气温度始终高于露点且在滤料允许范围内[[S20]]。

## 相关标准与规范

以下列出与冲天炉烟气冷却器设计、制造及运行直接相关的主要标准及核心要求。

### 国内标准

- **GB 8959‑2007《铸造防尘技术规程》**：明确冲天炉熔炼阶段通风除尘系统必须设置烟气冷却装置，可选水冷套、风冷冷却器等类型；冷却后的烟气需送入高效除尘器实现达标排放[[S6,S30]]。
- **GB 9078‑1996《工业炉窑大气污染物排放标准》**：规定了冲天炉炉尘的排放限值[[S30]]。
- **JB/T 6953‑1993《冲天炉烟尘排放标准》**：按不同功能区类别对冲天炉烟尘排放提出控制要求[[S30]]。
- **GB 50050‑2007《工业循环冷却水处理设计规范》**：规定了水冷式冷却器配套循环冷却水的硬度、浊度等水质控制指标，避免冷却水管内壁结垢影响换热效率[[S25]]。

### 国际规范

- **ASME BPVC Section VIII, Division 1 Case 2837**（2017/2019 版）：允许使用符合 ASTM D3307‑10 Type II 的无填充 PFA 管材作为烟气热交换器管束，并限定了管侧介质为液态水时的最高设计压力 758 kPa、最高设计温度 163 ℃ 以及壳侧烟气最高温度 195 ℃[[S11,S31]]。此前 Case 2795 对 PTFE 管材亦给出了类似许可（管侧最高压力 800 kPa、最高温度 163 ℃）[[S26]]。
- **ISO 大气腐蚀系列标准**：为冷却器材料耐酸露点腐蚀的选型提供通用评估依据[[S8]]。
- **ASTM 系列金属腐蚀测试标准**：为管材抗腐蚀性能的实验验证提供标准测试方法[[S8]]。
