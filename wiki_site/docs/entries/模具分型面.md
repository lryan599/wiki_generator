## 概述

分型面是压铸模具设计与制造的基准面，直接影响着压铸件的成型质量、脱模效率、浇注与排溢系统布局以及模具整体结构复杂度[[S42,S17]]。在国家标准GB/T 8847-2003《压铸模术语》中，分型面被明确定义为“模具上为取出铸件和浇注系统凝料可分离的接触表面”[[S23]]。多部主流压铸设计手册进一步给出扩展定义：为取出压铸件、浇注余料并安放嵌件，压铸模分割为可分离部分，合模时封闭形成型腔、开模时可分离的相互接触表面[[S42,S3,S2]]。“分模面”是“分型面”的通用公认别名，现有公开正式压铸行业文献未记载二者存在明确语义差异[[S25]]。

分型面对下列方面有直接影响：压铸模结构的繁简程度、铸件在模具内的成型位置、浇注系统的布置形式及内浇口的位置和导流方向、型腔排气条件及排溢效果、模具成型零件的组合及镶拼方法、以分型面作为加工装配基准面对铸件尺寸精度的保证程度、生产效率及成型部位的清理效果、铸件的脱模方向及脱模斜度的倾向、铸件表面美观和修整难易程度[[S30]]。

在英文术语体系中，parting surface、die parting surface、mold parting surface均对应中文术语“分型面”；parting line对应“分型线/分模线”，指动定模成型表面的交线轮廓；parting plane对应“分型面”，是压铸行业内通用公认的等价别名[[S33]]。

## 分型面的基本类型

分型面可根据结构复杂度分为单分型面、多分型面和侧分型面三类。单一分型面通过一次分型即可使铸件和浇注余料完全脱模；多分型面则因铸件结构的特殊性，需要增设一个或两个辅助分型面；侧分型面是当模具具有侧抽芯结构时，与开模方向平行的分型面[[S30]]。

### 单分型面的形状分类

| 类型 | 说明 |
|------|------|
| 平直分型面（平面分型面） | 分型面平行于压铸机动定模固定板平面，与开模方向垂直，是压铸设计中应用最广泛的分型面类型[[S11,S16]] |
| 倾斜分型面 | 分型面与压铸机动、定模固定板成一角度[[S11,S16]] |
| 阶梯分型面（折线分型面） | 分型面不在同一平面上，由多个不共面的阶梯/折线平面拼接组成，适配具有台阶轮廓的压铸件脱模需求[[S11,S2,S16]] |
| 曲面分型面 | 分型面按压铸件外轮廓的曲面形状随形设置，为非规则平面，适配带有自由曲面轮廓的异形压铸件分型[[S11,S16]] |
| 复合（综合）分型面 | 由平面、斜面、阶梯面、曲面中的两类及以上结构组合形成的分型面，适配结构复杂度较高的异形压铸件脱模需求[[S22]] |

### 多分型面的组合形式

《压铸模具设计师手册》收录的分型面类型表涵盖单分型面（直线、倾斜、折线、曲面等）和多分型面（双分型面、三分型面、组合分型面）的结构构成特点[[S7]]。由于结构的需要，当一个分型面不能满足要求时，可采用多分型面的结构形式。例如，为取出直浇道凝料而设置的分型面I-I作为辅助分型面，II-II为主分型面，开模时在顺序分型脱模机构的作用下，首先从I-I处分型，拉断并推出直浇道余料后，再从II-II处分型[[S30,S22]]。

## 分型面选择原则

选择分型面涉及铸件的形状和技术要求、浇注系统和溢流系统的布置、压铸工艺条件、压铸模的结构和制造成本、模具的热平衡等因素，这些因素往往难以兼顾，确定分型面时要予以综合考虑[[S30]]。

### 核心通用原则

1. **通过最大截面**：分型面必须通过铸件的最大截面，保证开模后铸件可随动模移动脱出定模，方便后续通过动模侧顶出机构取出铸件。这是选择分型面最基本的原则，否则开模后压铸件就无法从模具型腔中取出[[S29,S12,S30,S16]]。

2. **铸件留在动模**：由于压铸模动模部分设有推出装置，开模后必须保证压铸件脱出定模随着动模移动。设计时应使动模部分被压铸件包住的成型表面多于定模部分[[S16]]。金属液的凝固收缩会对模具中有凸出外圆形状的部分产生包紧力，而内凹形状的部分则没有，因此取件时的摩擦力在有凸出外圆形状的部分比有内凹形状的部分大[[S29]]。

3. **适配浇注与排气系统**：需适配浇注系统、溢流槽、排气槽的合理布局，避免在金属液主流束路径上提前封堵排气通道[[S29,S12]]。分型面最好布局在一个平面上，可以通过对一对模板进行简单的平磨来保证密封面的配合[[S10]]。

4. **简化模具结构与加工**：分型面形状优先选择平直贯通结构，降低模具研合加工难度，减少后续合模飞边清理工作量[[S30]]。对倾斜分型面或曲线分型面，为便于加工和研合，应采用贯通的结构形式[[S30]]。

5. **保证铸件尺寸精度**：尺寸精度要求高的部位和同轴度要求高的外形或内孔，应尽可能设置在同一半模内[[S3]]。

### 汽车轮毂类压铸件的专属设计要求

汽车轮毂类压铸件结构特殊，需采用多向分型结构，设置1个主分型面搭配多个辅助分型面，对应斜滑块侧抽芯结构时生成平行于开模方向的侧分型面，避免模具开合干涉[[S30,S25]]。

具体设计要求包括：需设置辅助曲面，且该曲面需与钢圈内壁、轮辐内壁完全相切过渡，结合适配的拔模斜度设计，从结构上完全避免模具开合过程中的运动干涉，保障上下型箱分型、外圆周多瓣侧抽芯机构的顺利动作[[S25,S21]]。

某汽车轮毂案例记载采用3个分模面搭配4个模具的布局，其中1/2圆周模具生成轮毂外圆周表面，其余2个模具制造轮辐上下型箱。创建上下箱分模面时需在轮毂参考模型内部生成辅助曲面，保证该曲面与钢圈内壁、轮辐内壁相切，同时匹配对应拔模斜度，避免模具开合时出现干涉[[S21]]。

枝杈轮辐铝合金轮毂压铸模具采用H13钢制顶模和底模，35CrMo钢制四体边模组合结构，所有分型面位置匹配顶模、底模、边模的分界面，保证四件边模同步加工时的同心度，边模之间的间隙控制在0.1mm以内[[S28]]。

## 工艺参数与控制要求

### 拔模斜度

根据GB/T 8844-2003标准：

- 成型部位未注拔模斜度时，形成铸件内侧壁（承受铸件收缩力的侧面）的拔模斜度按规定取值，构成铸件外侧壁的拔模斜度为内侧壁对应数值的1/2[[S6,S12,S40,S13]]
- 圆型芯的拔模斜度有独立规定[[S12,S40]]
- 文字、符号的拔模斜度取10°~15°[[S6,S12,S40,S13]]
- 常规铝合金轮毂轮辐侧面拔模斜度不小于8°[[S32]]

### 分型面间隙与粗糙度

| 参数项 | 要求 | 来源 |
|--------|------|------|
| 合模时分型面局部允许最大间隙（排气槽除外） | ≤0.05mm | [[S6,S8]] |
| 分型面接触工作面表面粗糙度 | Ra≤0.4μm | [[S6,S8,S40]] |
| 浇注系统表面粗糙度 | Ra≤0.4μm，转接处圆弧光滑连接 | [[S6,S8]] |
| 推杆允许凸出型面 | ≤0.1mm；复位杆允许低于型面≤0.05mm | [[S8]] |
| 镶块分型面允许高出套板分型面 | 0.05~0.10mm | [[S40]] |

## 工艺作用与影响机制

1. **对充型流动的影响**：分型面的位置选择直接改变金属液在型腔内的充型流动路径，若分型面布局不合理易在充型末端形成气阻，阻碍金属液顺利填充[[S29]]。分型面的位置和形状影响金属液流动，在不同分型面方案中，某些位置易形成气阻影响填充，而改变分型面结构可避免该问题[[S29]]。

2. **对排气的影响**：分型面设置在充型流动末端时，型腔内部气体可经由分型面位置顺利排入溢流槽与排气通道，大幅降低卷气缺陷概率[[S29,S1,S10]]。如果可能的话，可将流动平面与开有最重要排气槽的分型面部位相错开，防止排气槽在流束向侧面扩张时被过早堵塞[[S1]]。

3. **对脱模的影响**：分型面位置直接决定动定模的成型部分分布，若铸件包紧力占比在定模侧过高，开模后铸件会滞留在定模内，大幅提升取出铸件的操作难度[[S29,S16]]。

4. **对飞边的影响**：分型面封胶间隙、接合质量直接决定飞边的生成位置，分型面形状越复杂、接合间隙越大，后续铸件去除飞边的后处理工作量就越高[[S10]]。为了达到严密的密封合模，密封面的相互配合必须令人十分满意，从而使每次压射操作后就完全消除了毛刺[[S10]]。

## 缺陷形成机理

分型面设计不合理可引发以下典型缺陷[[S42,S30]]：

| 缺陷类型 | 形成机理 |
|----------|----------|
| 飞边 | 分型面接触精度不足、合模间隙超过允许阈值，高压高速充型时金属液渗入分型面间隙形成大量飞边 |
| 拉伤 | 分型面位置选择未匹配脱模斜度方向，开模时动定模成型面强行刮擦铸件表面 |
| 变形、尺寸超差 | 分型面布局不当导致型腔内气体无法有效排出，金属液充型保压阶段局部冷却收缩受力不均 |
| 无法脱模 | 分型面位置未落在铸件最大截面上，或存在倒扣区域未设置对应抽芯机构，开模后铸件被成型面完全卡滞 |

## 与抽芯机构的配合

分型面设计需优先将抽芯距离短、投影面积小的侧凹特征分配给侧向抽芯机构完成成型，以此简化抽芯驱动结构，降低金属液对侧向型芯的侧压力[[S15]]。

### 滑块侧向分型面

滑块侧部分的侧向分型面需与主分型面保持良好衔接，合模后楔紧块需对滑块施加足够预紧力，保证侧向分型面接合密封，避免金属液渗入导滑间隙形成飞边[[S24]]。

### 斜滑块侧向分型面

斜滑块侧向抽芯机构中，合模后的锁紧力压紧在斜滑块上，在套板上产生一定的预应力，使各斜滑块侧向分型面间具有良好的密封性，可防止压铸时金属液进入滑块间隙中形成飞边，影响压铸件的尺寸精度[[S24]]。斜滑块类侧向抽芯机构的侧向分型面接合处需保证足够密合性，在推出侧向分型抽芯的同时完成倒扣区域的分型分离，避免运动过程中与主分型面形成结构干涉[[S24,S43]]。

### 斜顶分型面

斜顶结构的分型面需采用工艺限位面设计，在合模充型过程中对斜顶形成侧向支撑，防止斜顶在高压金属液作用下后退错位，避免倒扣位置出现断差、飞边缺陷[[S41]]。

## 典型应用场景

### 汽车轮毂

轮毂类回转体压铸件采用多段相切曲面分型面+周向多分模抽芯的方案，适配圆周方向复杂曲面的脱模需求[[S19,S34]]。根据铸件本身基本轴对称且有中心孔的形状特点，应采用中心浇口，使金属流动顺畅，填充同步，流程短，模具结构紧凑。铸件外圆周非平滑，压铸模型腔需纵向多瓣分型[[S32]]。

### 变速箱壳体

变速箱壳体类箱体压铸件优先采用大平面主分型面，控制模具加工成本、优化排气布局[[S19,S34]]。壳体底部最大投影面作为主分型面，多数设置为平整的平面，便于浇注系统布置和加工，排气道可沿分型面顺畅开设[[S5]]。变速器后体压铸模具可设置A-A、B-B两级辅助分型面搭配主分型面的结构，对应不同侧向抽芯的需求[[S20]]。

### 离合器壳体

某离合器壳体（外形尺寸477mm×358mm×130mm）采用折线分型面，分型面沿两侧腔体最大端面分布，以侧向斜面过渡。起模斜度一致的型腔面/型芯面集中在同一侧，将包紧力更大的发动机侧大腔体设置在动模侧，保证开模时铸件随动模移动完成顺利脱模[[S39]]。

### 发动机缸体

直列四缸铝合金压铸缸体（外形尺寸480mm×380mm×270mm，质量约30kg，壁厚3.5~35mm）的分型面需适配大量侧向抽芯机构的布置，采用主分型面配合多个辅助分型面的组合形式，是目前压铸行业复杂度最高的分型面应用场景之一，对模具加工精度、合模定位精度有极高要求[[S37]]。

### 不同复杂压铸件分型面选型差异总结

| 铸件类型 | 分型面方案 | 设计考量 |
|----------|------------|----------|
| 轮毂类回转体 | 多段相切曲面分型面+周向多分模抽芯 | 适配圆周方向复杂曲面脱模需求[[S19,S34]] |
| 变速箱壳体类箱体 | 大平面主分型面 | 控制模具加工成本，优化排气布局[[S19,S34]] |
| 发动机缸体类超大尺寸多侧孔结构 | 组合多分型面搭配数十个独立抽芯机构 | 制造成本和设计难度最高[[S19,S34]] |
