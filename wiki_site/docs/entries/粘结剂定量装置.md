## 定义

粘结剂定量装置（binder dosing device）是按 GB/T 25370-2020《铸造机械 术语》明确定义的标准术语，指**具有自动调节定量功能的粘结剂吸入装置**[[S8,S15]]。该装置归类于砂处理设备大类下的液料计量子设备，与粘结剂预热器、粘结剂保温装置、催化剂定量装置共同组成完整的粘结剂供给计量链[[S15]]。

在压铸单元术语体系（GB/T 37371-2019）中，与"粘结剂"对应的计量配套设备为**脱模剂配送装置**，其功能是将脱模剂按比例与水混合并压送至喷涂机，属于压铸喷涂系统下属功能单元。二者分属完全独立的设备门类和应用场景，铸造行业内不存在将粘结剂定量装置指代为脱模剂计量装置的混淆情形[[S13]]。

## 工艺定位与作用

粘结剂定量装置在混砂工序中属于前置液料供给单元，布置在粘结剂输送装置、粘结剂预热器、粘结剂保温装置的下游，混砂机的正上游，为混制符合性能要求的芯砂提供精准的粘结剂添加量输入[[S8]]。在自动化冷芯砂制备生产线的典型布局中，该装置安装于砂温调节器出料侧下游、混砂机进料口正上游，与原砂定量给料装置直接串联[[S11,S24]]。

完整的混砂工作流程为：经温控调定的原砂先进入混砂腔体，同步计量加入固化剂预混 20~25 s，随后由粘结剂定量装置精准投放设定占砂重 0.9%~1.0% 的呋喃树脂，继续混碾 35~40 s 后卸料出合格的芯砂[[S2]]。该装置直接影响砂芯的固化时间与最终强度稳定性[[S8]]。

**加注量偏差的质量传导**：
- 定量加注量**低于**设计值：粘结膜无法完全包覆砂粒，砂芯室温抗拉强度降低，易出现制芯脱模破损、搬运掉渣缺陷[[S17]]。
- 定量加注量**超出**允许偏差范围：砂芯强度升高，但浇注过程中单位质量砂芯的总发气量同步线性升高，直接增加铸件气孔缺陷风险；过量粘结剂还会大幅提升铸后残留强度，降低砂芯溃散性，导致铸件内腔清砂难度大幅提升[[S18]]。

## 分类与原理

粘结剂定量装置按计量原理可分为以下四类，各自特点及适用场景见下表。

| 类型 | 工作原理 | 优点 | 缺点/局限 | 适用场景 |
|------|---------|------|----------|---------|
| **容积式（真空吸入量筒式）** | 粘结剂由真空阀吸入量筒，浮子随液面高低移动，由位置传感器测控定量[[S11]] | 结构简洁，定量可靠，适配冷芯盒间歇混砂工艺 | 计量精度受温度/粘度波动影响，需配合温控系统[[S5]] | 冷芯盒工艺间歇式混砂（如 Laempe LM 系列混砂机）[[S11,S16]] |
| **齿轮泵式** | 依靠啮合齿轮的齿间容积输送粘结剂，输送压力大，不易内泄漏[[S14]] | 输送稳定无脉动，维护频次低，综合运行性能优于柱塞式[[S12,S14]] | 采购成本相对偏高[[S14]] | 高粘度呋喃树脂等有机粘结剂的连续高精度输送，计量精度可达 ±2%[[S12]] |
| **活塞/柱塞式** | 依靠柱塞往复运动推送液料，通过调整行程控制输出流量；配套伺服驱动可保障配比稳定性[[S14]] | 可实现树脂与固化剂的机械联动配比，成本较低[[S14]] | 低泵速、部件磨损或含杂质时易内部泄漏；存在脉动现象[[S14]] | 树脂-固化剂双组分配比输送，建议增设管路流量监测及报警[[S14]] |
| **压力罐式** | 以罐内压力、管路通径作为定量控制参数[[S14]] | 结构简单 | 计量稳定性差，比例控制非常不稳定，工业应用占比已大幅降低[[S14]] | 已很少使用[[S14]] |

## 结构与组成

根据 GB/T 25370-2020 及相关技术文献，粘结剂定量装置及其配套子系统包括以下典型部件[[S8,S11,S14]]：

| 部件 | 功能说明 |
|------|---------|
| **储料容器**（粘结剂保温装置） | 带加热控温、液位显示功能的粘结剂保温储存罐，预先存储待输送的粘结剂，稳定液料温度以避免粘度大幅波动[[S8]] |
| **计量单元** | 设备核心模块，按计量原理配置容积腔、称重传感器、柱塞泵或齿轮泵等部件，精准控制粘结剂输出体积或质量；可联动催化剂/固化剂定量单元实现多组分自动配比[[S14]] |
| **输送管路** | 将计量后的粘结剂从储料端输送至混砂机投料口，部分场景下管路配套保温伴热结构，避免粘结剂降温时粘度上升、局部凝固堵塞管路[[S8]] |
| **控制系统** | 支持实时调整粘结剂输出用量、变频调节输送泵运行流量，保障砂型固化时间与强度的稳定性[[S8]] |

输送系统上游还包含**粘结剂预热器**（对进入混砂机前的粘结剂预热以稳定粘度）和**粘结剂输送装置**（将桶装商品粘结剂泵入保温桶的输送系统），下游经混砂机搅拌后产出合格芯砂[[S8,S15]]。

## 关键工艺参数与要求

| 参数项 | 指标/要求 | 来源 |
|--------|----------|------|
| 粘结剂流量误差（整体） | ≤ ±3% | [[S2]] |
| 齿轮泵式呋喃树脂定量误差 | ±0.4%~±0.9%（连续槽式混砂机） | [[S12]] |
| 齿轮泵式固化剂定量误差 | ±0.8%~±1.5% | [[S12]] |
| 齿轮泵计量精度 | ≤ ±2% | [[S12]] |
| 砂流量计量误差 | ≤ ±5% | [[S2]] |
| 粘结剂温度控制范围 | 25~35℃（建议区间），树脂温度 21~27℃ | [[S12,S24]] |
| 呋喃树脂工作粘度（恩氏） | 30~60 s（最优泵送特性区间） | [[S12]] |
| 注射/真空容积式单次定量响应时间 | ＜1 s | [[S5]] |
| 注射/真空容积式重复精度 | 优于 ±1% | [[S5]] |
| 胺类催化剂微量计量精度 | ≤ ±0.5 cc | [[S5]] |
| 高压压铸冷芯盒场景计量精度 | ≤ ±0.5%（高精度注射式计量结构） | [[S24]] |

**温控要求**：配套温控系统需将待计量粘结剂温度稳定控制在 25~35℃ 区间，通过粘结剂预热器与保温装置消除环境温度波动导致的粘度漂移；带压力补偿的容积式定量系统可在宽温度区间内保持计量精度，适配常规呋喃树脂、酚醛树脂等铸造有机粘结剂的工作粘度区间[[S12]]。

**校准要求**：行业规范要求每班生产前必须对砂流量、粘结剂流量、催化剂流量进行离线称重校准，消除长时间运行后的漂移误差[[S5,S12]]。

## 常见使用故障

| 故障类型 | 机理 | 影响 |
|----------|------|------|
| **粘度波动导致计量偏差** | 粘结剂预热/保温装置失效，环境温度波动改变粘结剂动态粘度，重力式定量系统瞬时流量随粘度漂移[[S5]] | 实际加注量偏离设定值，混砂批次性能不一致 |
| **管路堵塞与固化残留** | 呋喃树脂与催化剂在管路内残留接触会提前发生缩聚反应，生成固态树脂垢层逐渐沉积在泵腔、管路内壁；长期运行缩小通流截面[[S5]] | 管路堵塞，系统停机清理 |
| **粘结剂种类适配性差** | 不同类型粘结剂的粘度、固化反应特性差异大，单一泵结构若未提前标定流量-粘度曲线，直接更换不同牌号粘结剂时会出现定量失准[[S5]] | 无法保证不同工况下的加注精度 |

针对活塞/柱塞式系统在低泵速、部件磨损或介质含杂质时的内部泄漏问题，生产实践中建议增设管路流量监测及报警装置[[S14]]。

## 产线集成与自动化

粘结剂定量装置的启停信号与混砂机的搅拌周期完全绑定，可接收砂芯生产线中央控制系统（PLC）的指令动态调整粘结剂注入量，实现原砂进料信号触发→粘结剂同步定量注入→混砂完成后信号推送至下游射芯机的全自动闭环运行，满足砂芯"随用随混"的工艺要求[[S9,S20]]。

该装置作为砂处理通用型计量单元，可适配高压压铸、重力铸造、低压铸造三类工艺配套的砂芯制备线：
- **高压压铸场景**：普遍采用高精度注射式计量结构，控制误差在 ±0.5% 以内，适配冷芯盒低添加比要求[[S24]]。
- **重力铸造场景**：可兼容大流量容积式装置，降低选型成本[[S24]]。
- **低压铸造场景**：可联动砂温传感器动态微调粘结剂添加量，稳定发气性能[[S24]]。

设备本体无需进行结构性改造，仅需标定参数即可跨场景运行[[S24]]。
