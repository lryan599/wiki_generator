压射室偏离距离是压铸设备中描述压射室（shot sleeve）中心轴线相对于机器或模具参考基准偏移量的核心安装参数。根据国家标准GB/T 25370-2020《铸造机械 术语》，该术语在挤压铸造设备体系下给出了精确定义：对于卧式挤压铸造机，指“压射室中心相对于定模板安装面的距离”；对于立式挤压铸造机，则指“压射室中心相对于拉杠对称中心的距离”[[S16]]。该参数在压铸行业中亦常被称为压射室偏心距、压射室偏移量，对应英文术语为shot sleeve offset distance、shot sleeve misalignment 或 injection chamber deviation [[S2]]。

## 分类

依据拾取到的证据，压射室偏离距离可按两个维度进行分类：

**按偏移方向**
- **水平偏离**：压射室轴线在水平方向上相对于参考基准的偏移
- **垂直偏离**：压射室轴线在垂直方向上相对于参考基准的偏移 [[S9]]

**按发生状态**
- **静态偏离**：设备安装调试阶段测得的固定偏离值，由装配定位决定
- **动态偏离**：压射过程中因压室热变形、受力变形等因素引发的运行过程偏离量 [[S7,S5]]。研究表明，压室在热态工况下的总形变量处于0～0.02 mm区间，其中浇口位置的热变形最大，该热变形会叠加在冷态装配的静态偏离之上 [[S5]]

## 工艺作用与影响

压射室偏离距离直接关系到压铸工艺的稳定性与铸件质量。

**平衡胀型力偏心力矩** 
常规冷室压铸机的压射位置设有中心位置与多档偏移可调位置，通过调整该参数可适配不同偏心布置的浇注系统，从而平衡型腔胀型力的偏心力矩，降低所需的额定锁模力 [[S11,S19]]。

**锁模力需求变化**
当压射室（射流重心）偏离锁模力中心时，所需锁模力显著上升。计算表明，若偏移率达到11.8%，所需合模力将比中心对准工况高出约24%；若锁模力未同步提升，分型面金属泄漏概率增大，飞边缺陷风险明显增加 [[S20]]。

**金属液流动与卷气**
压射室偏离过大时，金属液在压室内受到不对称壁面约束，流动均匀性被破坏，容易形成不稳定卷流波面，导致金属液卷气概率大幅升高，最终使铸件内部含气量上升、孔隙率缺陷占比增加 [[S13]]。

**压射冲头运行状态与部件寿命**
压射室与压射冲头安装时须保证同轴度，若偏离量超出允许范围，二者间的摩擦阻力将显著增大、磨损加剧，严重时直接导致压射冲头与压室卡死；同时压射运动精度下降，难以达到预设的压射速度与压力，最终影响铸件成型质量 [[S10,S12,S18]]。

## 控制要求与测量方法

为确保压射室偏离距离处于允许范围内，安装调试阶段须进行精度检测。

**同轴度检测**
压射冲头与压室安装时，可采用套筒与千分表组合的方式检测二者的偏心度，经调整使同轴度符合要求 [[S10]]。

**标准规定的重合度检验**
GB/T 21269-2023《冷室压铸机》对压射室轴线与活塞杆轴线的重合度检测方法做出了明确规定：
- **卧式冷室压铸机**：在距压射室端面30 mm的A截面以及1/2压射行程位置的B截面分别设置指示器，旋转一周读取跳动值，取最大差值作为重合度误差 [[S8]]
- **立式冷室压铸机**：采用相同的测量截面与评价方法 [[S17]]

## 典型数值与机型配置

不同规格的立式冷室压铸机在标准中规定了不同的压射偏移配置选项。GB/T 21269-2023 表A.2给出的推荐值如下：

| 锁模力 (kN) | 压射位置 (0为中心) |
|:---:|:---:|
| 630 | 0 |
| 1000 | 0 |
| 1600 | 0 |
| 2500 | 0 / 80 mm |
| 4000 | 0 / 100 mm |
| 6300 | 0 / 150 mm |

[[S1]]

较小规格机型（630～1600 kN）仅设中心压射位置，2500 kN及以上机型则增设偏心压射位置，偏移量随锁模力等级递增 [[S1]]。
