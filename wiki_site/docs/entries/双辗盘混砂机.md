## 概述

双辗盘混砂机（亦称双辗盘连续混砂机、双碾盘连续混砂机、双碾盘碾轮式混砂机）是一种用于铸造砂处理工序的连续式混砂设备[[S13,S22,S20]]。该设备由两台规格完全相同的间歇辗轮式混砂机组合而成，两盘中心距约为单盘盘径的0.9倍，物料在混制过程中沿“8”字形轨迹运动，实现连续加料、混制与卸砂[[S1,S13,S9]]。

## 定义与分类

依据 GB/T 25370-2020《铸造机械 术语》，双辗盘混砂机定义为“由两套同样盘径的相互贯通的辗轮混砂机构组合成一体的连续式混砂机”，对应的标准英文术语为 **multi-mull** 或 **multimuller**[S11,S8]。在铸造设备分类体系中，该设备归属于铸造设备大类→砂处理设备子类→辗轮式混砂机分支下的连续式混砂机型[[S21,S10,S20]]。

该设备于20世纪60年代由美国辛普森（Simpson）公司在普通辗轮混砂机基础上推出，专门为高压造型、挤压造型等现代高速自动造型线配套设计，旨在满足高小时需砂量要求的同时避免砂处理工部占地面积过大的问题[[S9,S1]]。

## 结构与组成

双辗盘连续混砂机由两台规格完全相同的间歇辗轮式混砂机拼合组成，每台辗盘单元包含以下核心部件[[S13,S9,S5]]：

| 部件 | 功能描述 |
|------|----------|
| 辗轮（碾轮） | 随主轴公转并绕水平轴自转，对砂层施加辗压与搓揉作用 |
| 弹簧加压机构 | 可调整弹簧预紧力以适配不同混砂工艺要求；弹簧可采用单根或反向旋向双弹簧组合结构 |
| 十字头回转机构 | 同步驱动外刮板与内刮板完成砂层翻抛作业 |
| 外刮板 / 内刮板 | 推动物料跨盘交换并清理盘边积聚砂层；刮板可镶装硬质合金耐磨条 |
| 摆动式卸料门 | 通过控制开启角度调节物料在辗盘中的停留时间 |
| 底盘耐磨衬板 | 可加装锰钢衬板、橡胶衬板或耐磨塑料衬板，延长使用寿命 |
| 驱动装置 | 为两个主轴提供相同转速、相反转向的动力 |

## 工作原理

双辗盘连续混砂机的两台辗盘单元主轴转速相同而旋转方向相反。混砂过程中，从一个辗盘连续加入的物料受到辗轮的辗压搓揉及刮板的搅拌混合；随着刮板的转动，部分物料穿过两辗盘的交界处进入另一辗盘，整体形成“8”字形闭合运动轨迹[[S13,S9,S15]]。

物料在两盘内多次换向、反复交换位置，经充分碾压搓揉后达到均匀混合。通过调节摆动式卸料门的开启度，可控制物料在盘中的停留时间，适配不同混砂工艺的性能要求[[S13,S15]]。由于取消了间歇式混砂机的加料、卸料辅助等待时间，其生产率比同规格单盘间歇辗轮式混砂机高出50%[[S13,S18,S9]]。

## 工艺条件与应用场景

双辗盘混砂机适用于各类黏土砂和普通水玻璃砂体系的单一砂、背砂混制，同时可适配部分面砂、芯砂的混制工艺[[S9,S1,S19]]。其核心应用场景为大批量铸造车间的湿型单一砂连续混制生产，尤其适配采用连续供料系统与在线水分控制仪的高压造型、挤压造型等高速自动造型线[[S9,S1]]。

该设备通过可调加料流量控制与卸料门开度调节相结合，实现连续混砂作业下的型砂质量工艺适配[[S9,S13]]。

## 与其它混砂设备的区别

| 对比设备 | 主要区别 |
|----------|----------|
| 单盘间歇辗轮式混砂机 | 双辗盘混砂机无加料、卸料辅助时间，生产率高50%；但间歇式混砂机对型砂质量的批次控制更为精确[[S13,S18,S20]] |
| 辗轮转子混砂机 | 辗轮转子混砂机兼具辗压与转子剪切搅拌功能，结构较复杂但型砂松散性较好；双辗盘混砂机结构较简单，维修方便[[S1,S20]] |
| 转子混砂机 | 转子混砂机依靠高速旋转转子产生冲击与剪切力，混砂速度快、能耗低，但碾压搓揉效果不如辗轮式设备[[S1]] |
| 摆轮式混砂机 | 摆轮式混砂机利用离心力辗压砂流，混砂效率较高但碾压搓揉效果与辗轮式相比较差[[S5,S9]] |
| 滚筒式混砂机 | 滚筒式混砂机兼具冷却、混砂、松砂功能，结构紧凑但工作原理有本质差异[[S13]] |

## 局限性与典型缺陷

1. **砂层厚度限制**：继承传统辗轮式混砂机的特性，若砂层厚度过大，辗轮前进滚动阻力会显著升高，因此单盘允许最大装载量存在明确上限[[S9]]。

2. **吨能耗偏高**：低转速辗揉作业方式的单位产能能耗高于转子类高速混砂设备[[S9]]。

3. **连续模式下的均匀度短板**：连续作业模式下物料停留时间存在统计分布区间，部分物料可能未按“8”字形轨迹充分运动便已卸出，导致极端小批量特种砂的混制均匀度明显低于间歇式混砂机[[S20,S9]]。

4. **高损耗件维护量大**：辗轮工作面、底盘耐磨衬板、带硬质合金镶条的刮板均为日常高损耗件，更换与维护工作量较大[[S9]]。

5. **质量控制的固有难度**：部分用户为保证型砂质量，将“连续”混制改为“间歇”混制，即只有型砂性能达到工艺要求后才卸砂，但这以牺牲生产率为代价[[S20]]。
