## 概述

钢管外壁抛（喷）丸清理机是专用于清理钢管或铸管外表面的抛喷丸设备，其工件承载体通常采用V形输送辊道或托轮结构[[S16,S19]]。该设备通过高速投射的弹丸对钢管外壁进行冲击，以达到去除氧化皮、锈蚀层、旧涂层等表面附着物，并获得一定表面粗糙度的目的，广泛用于钢管防腐涂层前的预处理工序。依据弹丸加速方式的不同，设备可分为抛丸清理机和喷丸清理机两大类[[S4]]。

## 定义与分类

### 国标定义

依据GB/T 25370—2020《铸造机械 术语》，钢管外壁抛（喷）丸清理机的标准定义为：“专用于清理钢管或铸管外表面的抛喷丸设备”，其“承载体通常采用V形输送辊道或托轮结构” [[S16,S19]]。

### 抛丸与喷丸的技术区分

在钢管外壁清理场景下，两者在动力来源和工作原理上存在本质差异[[S4]]：

| 对比维度 | 抛丸清理 | 喷丸清理 |
| :--- | :--- | :--- |
| 动力来源 | 高速旋转叶轮产生的离心力 | 压缩空气 |
| 核心部件 | 抛丸器（含叶轮、叶片等） | 喷丸器（喷丸罐、软管、喷嘴总成） |
| 弹丸速度 | 60 ~ 80 m/s | 50 ~ 70 m/s |
| 生产效率 | 抛丸量大，效率高，适合大批量连续作业 | 大面积清理效率较低，整体动力消耗更高 |
| 适用场景 | 大表面积钢管外壁的常规清理 | 灵活性高，适合抛丸难以覆盖的局部死角补充清理 |

### 常见设备类型

钢管外壁抛（喷）丸清理机根据工件承载和输送方式的不同，主要有以下几种常见结构[[S16]]：

-   **辊道通过式**：以输送辊道为工件承载体，工件以连续通过的方式接受磨料冲击。
-   **V形滚轮式**：V形滚轮与输送方向成一定角度排列，可同时实现钢管的旋转和平移，确保钢管外壁360°均匀受击。
-   **托轮旋转式**：托轮作为承载体驱动钢管原地自转，适用于单批次间歇清理作业场景。

在更广泛的干法清理设备体系中，抛（喷）丸清理设备还包含履带式、吊钩式、转台式等多种机型[[S15]]。下图展示了其完整的分类体系，钢管外壁抛（喷）丸清理机可视为该体系中一种专用化的辊道输送式机型。

## 工作原理与结构组成

钢管外壁抛丸清理是一个全自动的连续作业流程，主要包括工件输送、丸料投射、丸料回收分离和除尘四个环节[[S5]]。

1.  **工件输送**：待清理钢管由V形辊道或托轮驱动，边平移边自转地送入封闭的清理室。
2.  **丸料投射**：布置在清理室内的抛丸器（或喷枪）从不同角度向钢管外壁高速投射弹丸，进行冲击清理。
3.  **丸料回收分离**：冲击工件后的弹丸与铁锈、氧化皮碎屑等杂质混合物落入底部回收机构，经斗式提升机送入丸砂分离器。分离器利用风力或磁力将杂质去除，合格的弹丸循环回储丸斗待用，杂质则被送入除尘系统[[S3]]。
4.  **除尘作业**：配套的通风除尘系统捕捉全过程产生的粉尘，经处理后达标排放，避免污染环境。

### 核心部件

任一抛喷丸设备均由弹丸循环系统、工件承载系统、控制系统、除尘系统和加卸料系统五大系统组成[[S5]]。针对钢管外壁清理机，其核心部件包括：

**抛丸器**是抛丸清理机的关键部件，由主轴、电机、连接盘、壳体护板、输丸管、分丸轮、定向套、叶轮和叶片等组成。其工作原理为：弹丸由输丸管进入，经分丸轮预加速和定向套窗口的引导，到达高速旋转的叶片上。在离心惯性力作用下，弹丸沿叶片向外运动，最终以60~80m/s的速度抛射向工件。通过调整定向套窗口的位置，可以改变弹丸的抛射方向[[S18]]。下图直观地展示了抛丸清理的投射原理。

![图：抛丸清理投射工作原理示意](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/ee124c5b-4ee2-49c3-b46f-6e024cc093a6/resource)

**喷丸器**是喷丸清理的核心，主要由喷丸罐、喷丸软管和喷嘴组成。它利用5~6 atm的压缩空气将弹丸通过喷嘴加速至50~70 m/s，射向工件表面[[S12,S18]]。

**丸料循环系统**的功能是回收、筛选并供给磨料，通常由斗式提升机、螺旋输送器、丸砂分离器、振动筛、储丸斗、弹丸流量控制阀和流丸管等构成[[S3]]。

### 丸料

可选用的丸料种类包括钢丸、钢砂、铸钢丸、铸铁丸、钢丝切丸和不锈钢丝切丸等[[S4,S13,S20]]。常用规格为0.5 ~ 2.5 mm，适用粒度范围通常在0.3 ~ 3 mm[[S9,S21,S4]]。

### 清理质量要求

-   **除锈等级**：依据GB8923-88《涂装前钢材表面锈蚀等级和除锈等级》，钢管外壁抛丸清理通常可达到Sa 2.5级（非常彻底的喷射除锈），高端防腐前处理可达到Sa 3级（使钢材表面洁净）[[S10,S2]]。Sa 2.5级处理后，钢材表面的抗滑移系数可达0.45~0.5[[S10,S2]]。
-   **表面粗糙度**：处理后表面可获得20 ~ 75 μm的粗糙度（锚纹），以增强防腐涂层的附着力[[S22]]。

## 应用领域与工艺作用

钢管外壁抛（喷）丸清理机主要应用于各类钢管在涂覆防腐层之前的表面预处理[[S27,S30]]。其典型的应用场景包括石油输送管、油井套管和大型结构钢管的防腐涂层生产线[[S22,S29,S30]]，为沥青、聚乙烯（PE）等涂层提供理想的表面状态。

其工艺作用主要体现在：

-   **彻底清理表面**：高效去除钢管外壁在轧制和储运过程中形成的氧化皮、锈蚀、油污及旧涂层[[S6,S27]]。
-   **提供锚纹粗糙度**：使钢管表面获得均匀的、具有一定粗糙度的锚纹，显著增大涂层接触面积，产生“锚固”效应，从而提升防腐涂层的附着力[[S22]]。
-   **表面强化**：通过弹丸冲击使钢管表面产生冷作硬化层和残余压应力，有助于提高其抗疲劳和抗应力腐蚀性能[[S13]]。

## 优势与局限性

### 与主要清理方式的对比

**1. 对比酸洗清理**

| 对比维度 | 抛（喷）丸清理 | 酸洗清理 |
| :--- | :--- | :--- |
| 环保与安全 | 无“氢脆”风险，无废酸排放问题 | 存在氢脆风险，需处理废酸、废液 [[S14,S17]] |
| 表面质量 | 可直接获得均匀、无光泽的表面及所需粗糙度 | 处理后表面光滑，需额外工序获取粗糙度 |
| 适用性 | 对深沟槽、异形复杂结构和管端死角清理效果受限 | 可清理复杂形状和深孔、沟槽等死角 [[S6]] |

**2. 对比手工打磨清理**

| 对比维度 | 抛（喷）丸清理 | 手工打磨清理 |
| :--- | :--- | :--- |
| 效率与一致性 | 自动化程度高，效率可达40 m²/h以上，质量均匀稳定 [[S29]] | 劳动强度大，效率低，清理质量依赖人工水平 [[S17]] |
| 灵活性与成本 | 设备一次性投入大，灵活性不如手工，无法针对性处理微小局部缺陷 | 投资成本低，灵活性极强，可点对点处理缺陷 [[S33]] |

### 自身局限性

-   **异形管材受限**：对于非圆柱形的钢管或具有复杂截面的结构管，常规的V形辊道通过式设备可能无法保证360°均匀清理。
-   **端部清理**：钢管端部的狭小区域可能存在清理死角，有时需辅以喷枪进行补充处理[[S13]]。

## 相关标准与规范

-   **GB/T 25370—2020 《铸造机械 术语》**：对钢管外壁抛（喷）丸清理机及相关的各类术语给出了明确定义[[S16,S19]]。
-   **GB 8923—88 《涂装前钢材表面锈蚀等级和除锈等级》**：规定了抛丸除锈质量的验收等级（Sa2, Sa2.5, Sa3）[[S2,S10]]。
-   **ZBJ 61009—89 《通过式抛丸清理机 基本参数》**与**ZBJ 61010—89 《通过式抛丸清理机 技术要求》**：作为辊道通过式机型的行业标准，对设备的基本参数、工作性能、安全卫生、装配精度和寿命提出了具体要求[[S25]]，包括整机噪声不大于95 dB(A)、粉尘排放浓度不大于150 mg/m³等关键指标。
