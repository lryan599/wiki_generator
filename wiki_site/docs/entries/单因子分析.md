## 定义

单因子分析（Single Factor Analysis），又称单因素试验、单因子实验或 OFAT（One-Factor-At-a-Time），是试验设计（DOE）领域中最基础的一种试验策略与方法。其核心特征为在试验过程中，仅将一个可控的输入变量（因子）设置为多个不同的水平进行试验，而系统中其他所有可能产生干扰的变量均保持固定不变。通过观测输出响应指标随该因子水平变化的规律，获得该单一因子对指标独立的影响趋势 [[S26,S22]]。

需要特别指出，单因素方差分析（Single Factor ANOVA / One-Way ANOVA）与单因子试验并非等价概念。单因素方差分析是单因子试验完成之后配套使用的一种显著性统计检验方法，归属于方差分析（ANOVA）大类，其作用是检验不同因子水平下的总体响应均值之间是否存在统计意义上的显著差异。二者之间为试验设计方案与配套分析工具的关系，不可混为一谈 [[S5,S4]]。

### 与邻近DOE方法的层次关系

单因子分析是 DOE 体系中最为基础的初代试验策略，其在整个 DOE 方法体系的层级中位于底层。相较于正交试验设计、全因子试验设计及响应曲面法（RSM）等更为高阶的多因子DOE方法，单因子分析的优势在于实施门槛低、操作简便，是工业现场初步排查因子影响、获取单一变量影响趋势时常用的手段 [[S22,S18]]。

## 分类与试验设计类型

单因素试验设计可进一步划分为四种经典类型，其适用场景和对试验单元的要求各有不同 [[S26]]：

| 设计类型 | 核心特点 | 适用条件 |
| :--- | :--- | :--- |
| 完全随机化试验 | 各配方的试验单元随机分配，允许不等重复次数 | 各试验单元基本均匀一致 |
| 随机分块试验 | 将试验单元分为若干均匀区组，消除区组间偏差 | 存在已知或可预见的干扰变异来源 |
| 拉丁方试验 | 以行列双重划分控制两类外来变异 | 需要控制两个方向上的系统性偏差 |
| 正交拉丁方试验 | 在拉丁方基础上进一步控制第三类变异 | 多来源干扰同时存在的复杂场景 |

在工程实践中，**完全随机化试验**因其安排灵活、统计分析简便，是单因子试验中最常用的实现方案。但其对试验单元的均匀性要求最高，若物质条件存在显著不均匀，试验精度将大幅下降 [[S26]]。

## 基本原理与试验步骤

### 因子水平的确定原则

单因子试验的因子水平选择需遵循三项通用原则 [[S26]]：

1.  **水平数量**：推荐设置三个及以上的水平。若仅设置两个水平，所得的响应趋势只能表征为线性关系，无法识别出曲线拐点、极值或最优区间等非线性特征。三水平及以上的试验结果往往呈现二次曲线分布，更有利于发现最佳区域。
2.  **水平间隔**：不同水平之间优先采用算术等间隔设置，在特定场景下可选用对数等间隔。各水平与理想等间隔的偏差不得超过 20%，以保证效应曲线计算的便利性。
3.  **水平的具体性**：所有水平必须对应为可直接精准控制的实际工艺参数取值或操作方式，不可设置为无法落地的抽象条件。水平的变化应能直接引起试验指标的可观测变化。

### 试验流程与数据分析

单因子分析的通用流程与统计分析步骤可归纳如下：

1.  **试验设计**：根据研究对象确定要考察的单一因子，依据上述原则选取 3 个及以上的水平，并为每个水平安排若干次重复试验，以评估试验误差 [[S26]]。
2.  **数据收集**：在控制所有其他变量恒定的条件下，逐一在每个因素水平下进行试验，记录对应的输出响应值。
3.  **单因素方差分析**：这是判断因子影响是否显著的关键步骤，其标准流程为 [[S5]]：
    *   **提出假设**：原假设 H₀ 认为所有因子水平对应的响应总体均值全部相等；备择假设 H₁ 认为至少有一个水平的总体均值与其他水平存在显著差异。
    *   **构造 F 统计量**：将总变异分解为因子水平变化引发的“组间变异”和随机误差引发的“组内变异”两部分，计算两者的比值得到 F 值。
    *   **显著性判定**：将计算所得的 F 统计量与预设显著性水平（通常 α = 0.05）下的 F 临界值进行对比。若 F > F_critical（或 P 值 < 0.05），则拒绝原假设，判定该因子对响应的影响在统计上显著。
4.  **模型建立**：
    *   若因子为 2 个水平，可采用二元线性回归建立线性影响模型。
    *   若因子为 3 个及以上的等间隔水平，则可采用一元正交多项式回归建立二次或更高阶的多项式模型，以识别响应随因子变化的非线性趋势，如极值点和拐点等 [[S26]]。

## 压铸工艺中的应用场景

在压铸领域，单因子分析广泛用于各工艺参数对铸件质量影响的初步排查与量化研究。北美压铸协会（NADCA）公开的工艺优化导则 T93-061 中，也明确推荐将单因子试验作为初步排查变量影响规律的基础方法 [[S15]]。

### 浇注温度的影响

浇注温度是压铸中通过单因子分析研究最广泛的参数之一，它对铸件的孔隙率、力学性能及缺陷数量均有显著影响。

*   **孔隙率**：在常规的铝合金压铸工艺区间内，随着浇注温度的升高，铝液中氢的溶解度同步上升，凝固过程中析出的气孔含量增加，导致铸件的整体孔隙率呈持续上升趋势。以 5A06 铝合金为例，浇注温度为 650℃ 时孔隙率为 2.35%，对应密度为 2.580 g/cm³；当温度升至 690℃ 时，孔隙率升至 8.36%，密度下降至 2.421 g/cm³ [[S14,S17,S21]]。
*   **力学性能**：力学性能随浇注温度的升高通常呈现先升后降的单峰特征。例如，5A06 铝合金在 650℃ 时抗拉强度为 216.43 MPa，延伸率为 3.18%；在 670℃ 时达到最高值，抗拉强度为 232.5 MPa，延伸率为 4.16%；温度继续升至 680℃ 时，两者分别回落到 220.61 MPa 和 3.94% [[S14]]。
*   **缩松缩孔缺陷**：对于铝合金减震塔的真空压铸，单因素变量试验显示，在 640℃ ~ 720℃ 区间内，随着浇注温度升高，铸件的缩松缩孔缺陷体积同步上升。640℃ 时缺陷体积最小，为 30.85 cc，对应铸件质量最优 [[S1]]。

### 压射速度的影响

在铝合金减震塔真空压铸案例中，压射速度在 2.5 m/s 至 4.5 m/s 的区间内变化时，其对缩松缺陷量的影响相较于浇注温度较小，但整体趋势仍是随速度提升缺陷量上升。当压射速度为 2.5 m/s 时，缺陷体积最小，为 33.73 cc [[S1]]。压射速度过大会导致金属液卷气，使铸件性能下降；速度过小则可能出现浇不足现象 [[S1]]。

### 烘干温度对气孔率的影响

在对 H3NG 气缸盖的铸造工艺优化研究中，对砂芯烘干温度进行了单因子分析。结果显示，烘干温度与铸件气孔率之间存在显著的二次非线性关系，且二次拟合良好。气孔率达到最低点的最优烘干温度区间为 **155℃ ~ 165℃**。生产验证表明，将温度控制在该区间内，可将砂芯间隙稳定控制在 ±0.1mm 以内，从而大幅降低气孔废品率 [[S23,S10]]。

## 局限性及与多因子分析的对比

单因子分析具有一项核心且固有的局限性：它完全忽略了不同输入变量之间可能存在的交互效应 [[S22,S18]]。该方法默认其余所有变量固定不变，因此只能表征单因子与响应值之间的直接关系。当实际系统中多变量交互效应显著时，单因子分析得到的结论可能出现严重偏差，无法找到全局最优的参数组合，甚至可能得出与多变量实际规律相悖的错误结论 [[S18,S12]]。因此，单因子分析通常仅适用于多因子试验的**预筛选阶段**，初步判断因子影响的大小和趋势。

下表总结了单因子分析与典型多因子分析方法的对比 [[S24,S18,S22]]：

| 特性 | 单因子分析 | 多因子分析（正交试验/全因子试验/响应面法） |
| :--- | :--- | :--- |
| **考察变量** | 一次仅一个，其余固定 | 可同时考察多个变量及其水平 |
| **交互效应** | 无法识别，完全忽略 | 可识别并量化变量间的交互作用 |
| **试验效率** | 逐一排查，对多因子问题效率低下 | 仅需少量代表性试验即可覆盖全参数空间，效率极高 |
| **结论全局性** | 局部最优，非全局最优 | 可得到高置信度的全局最优参数组合 |
| **适用阶段** | 初步筛选、单因子影响趋势摸底 | 参数优化、复杂系统建模与稳健性设计 |

## 常用软件与输出

### 软件工具

主流工业统计软件均提供单因子分析功能 [[S4,S7,S28]]：

*   **Minitab**：内置单因子方差分析模块，是六西格玛工业场景下单因子分析的主流实现工具。
*   **SPSS**：在“比较均值（Compare Means）”菜单下提供 “Oneway ANOVA” 功能，支持设定显著性水平阈值完成差异检验。
*   **Excel**：其数据分析插件直接提供了“单因素方差分析”工具，可输出完整的结果表格。
*   **Design-Expert**：可生成单因子影响图，直观展示因子与响应的关系。

### 典型分析输出

单因子分析的常规输出主要包括三类核心内容 [[S6,S8,S27,S3]]：

1.  **主效应图**：通过折线展示因子不同水平对应的响应均值，直观反映该因子对指标的影响方向和幅度。
2.  **单因子拟合线图**：建立因子水平与响应值之间的量化回归关系，可呈现线性或二次等拟合模型。
3.  **方差分析表（ANOVA表）**：一份全面的方差分析表是压铸单因子显著性检验的标准结果呈现形式，应包含以下关键统计量 [[S16,S3]]：

| 方差来源 | 偏差平方和 (SS) | 自由度 (df) | 均方 (MS) | F 值 | P 值 / 显著性 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 因子 | SS_Factor | df_Factor | MS_Factor | F = MS_Factor / MS_Error | P |
| 误差 | SS_Error | df_Error | MS_Error | - | - |
| 合计 | SS_Total | df_Total | - | - | - |

当方差分析结果中的 P 值小于 0.05 时，表明该因子的水平变化对响应指标有显著影响。例如，在针对压铸机大杠锁模力进行的单因子方差分析中，统计结果显示 P 值均 ≤ 0.05，表明不同压铸机的锁模力分布存在显著差异，这一发现成功支撑了对模具跑铝水故障原因的定位与优化决策 [[S3]]。
