## 概述
磨料循环系统（又称弹丸循环系统，英文对应术语如 abrasive recycling system）是抛喷丸设备中实现磨料定量供给、回收、分离与筛选的核心子系统，广泛应用于压铸件的抛丸清理、喷丸强化及喷砂处理等后处理工序[[S4,S6,S10]]。该系统通过对使用过的磨料进行连续分离与净化，维持稳定的磨料质量，直接影响压铸件表面清洁度、粗糙度及残余应力状态。

## 定义与术语
依据 GB/T 25370-2020《铸造机械 术语》第6.2.26条，磨料循环系统定义为：抛丸清理机中，实施按预期定量供给和回收磨料并进行分离、筛选处理的循环输送系统[[S4,S6]]。其常规组成包括提升机、螺旋输送器、丸砂分离器、振动筛、储丸斗以及弹丸闸门、流丸管等[[S4]]。该标准将“弹丸循环系统”列为其别名，并明确了丸砂分离器可利用风力和/或磁力方式去除杂质，磁力方式时亦称磁力分离器；弹丸闸门亦称磨料控制阀，用于控制磨料流动并调节流量[[S4]]。

## 分类与形式
根据磨料状态与系统规模，常见的磨料循环系统可分为：

- **干式大流量系统**：为滚筒式、履带式、吊钩式等抛丸清理机配套，采用机械提升与气流分选相结合的方式（图1），是压铸件批量后处理的典型配置[[S3,S14]]。
- **自循环式喷丸/砂系统**：将喷砂、回收、分离、过滤集成于一体，分为压送式和吸送式。压送式自循环喷丸器包含砂尘分离器、粉尘过滤器、真空泵等组件，自动完成磨料回收与粉尘过滤[[S28,S29]]；吸送式基于文丘里原理，利用压缩空气形成负压吸入磨料，无需压力容器，便携式整机质量仅 2～3 kg，适用于模具清理和小尺寸铸件修整[[S28,S30]]。
- **湿法磨料悬浮液循环系统**：用于超声加工或湿式喷砂去毛刺工序，由磨料槽、泵、控制阀、磨料回收槽等组成，实现磨料悬浮液的闭环循环供给[[S17,S21]]。亦存在振动式干法再生砂循环系统，集破碎、摩擦再生、筛分风选于一体，主要用于铸造型砂的再生回用[[S16]]。

## 结构与组成
典型磨料循环系统的主要组件及其功能如下表所示[[S4,S6,S23,S29]]。

| 组件 | 功能 |
| :--- | :--- |
| 提升机 | 将清理室底部收集的磨料提升至分离器 |
| 螺旋输送器 | 水平方向输送磨料至提升机或储料点 |
| 丸砂分离器 | 利用风力/磁力分离可用磨料与粉尘、碎丸（图2） |
| 振动筛/滚筒筛 | 初步筛除粗大杂质与砂块 |
| 储丸斗 | 储存待用的合格磨料 |
| 弹丸闸门（磨料控制阀） | 开关磨料通路并调节流量 |
| 流丸管 | 将磨料从储丸斗输送至抛丸器或喷枪 |
| 砂尘分离器/粉尘过滤器 | 在自循环系统中实现磨料与粉尘的二次分离 |
| 流幕式微粉分离器（带挡板与进风口） | 利用水平气流分离微小粉尘，调节挡板控制流量，调节风量控制除尘效果[[S23]] |

## 工作原理与流程
磨料循环系统的基本流程为：使用过的磨料自抛丸/喷丸室底部被收集，经螺旋输送器或振动输送机送至提升机，提升至丸砂分离器上部，通过振动筛或滚筒筛除去大块异物后，在分离器内借助气流和/或磁力进行分选。合格弹丸落入储丸斗，经弹丸闸门和流丸管重新供给抛丸器；粉尘与破碎细粒被吸风系统带走；废料和不可用砂粒排入废料箱[[S2,S24,S32]]。

以大型抛丸设备丸砂循环系统为例，从抛丸室回收的丸砂混合物依次通过滚筒筛、磁选滚筒及风选装置，分流为净丸料、废料和砂，净丸料返回储料斗循环使用（图1）[[S24]]。高效丸砂分离器中，丸砂经节流板呈“幕帘”状进入分离室，在不同吸风风速下沿不同轨迹降落：第一分离口抽走微粉，第二分离口排出夹带少量碎丸的型砂，第三分离口进一步净化弹丸，最终得到的可回用弹丸含砂量低于 1%[[S32]]。分离过程对风速和风压的依赖性很强，合理控制是保证分离效率的关键[[S2,S32]]。

小型自循环喷丸系统则通过文丘里喷嘴产生负压吸入磨料并喷射，使用后的磨料直接被旁侧的回收系统吸走，进入砂尘分离器和粉尘过滤器，洁净磨料重新落入喷砂罐，实现小型封闭循环[[S28,S30]]。

## 工艺参数与控制要求
磨料循环系统运行过程中的关键参数与控制要求如下所列，这些参数直接影响磨料的回收率、分离纯度及系统能耗[[S2,S5,S22,S27,S32]]。

| 参数项 | 推荐范围或指标 | 说明 |
| :--- | :--- | :--- |
| 丸砂分离段工作风速 | 4～5 m/s | 实现弹丸、砂粒与粉尘分层分离的关键[[S2,S32]] |
| 风动运砂风道风速 | 35～40 m/s | 用于风力输送砂粒至旋风分离器垂直提升[[S22]] |
| 除尘系统总阻力 | 100～150 mm水柱 | 配合自激式水力除尘器，效率可达98%～99.5%[[S5,S2]] |
| 分离后可回用弹丸含砂量 | ≤1% | 延长抛丸叶片使用寿命的硬性指标[[S32]] |
| 磨料分离效率 | ＞99% | 系统整体设计要求[[S5]] |
| 磨料流量控制 | 通过弹丸闸门调节 | 流量过大则磨料速度不足、管路压损增大；过小则有效颗粒数不足、清理效率下降[[S13]] |

可循环金属磨料的类型与粒度适配范围如下[[S8,S27,S6]]：

| 磨料类别 | 材质 | 典型丸径/粒度范围 | 适用场景 |
| :--- | :--- | :--- | :--- |
| 金属丸粒 | 锌丸、不锈钢丸 | 0.3～1 mm | 压铸件表面清理与光泽处理 |
| 铸钢丸 | S110/SS0.3～S780/SS2.5 | 覆盖低到高冲击力需求 |
| 金属砂粒 | 铸钢砂 | G120/SG0.2～G16/SG1.4 | 高硬度氧化皮去除、粗糙度调控 |
| 钢丝切丸 | 不锈钢丝切丸、碳钢丝切丸 | 根据定制 | 使用寿命长，破碎率低 |

## 在压铸件后处理中的作用
在压铸后处理中，磨料循环系统的连续稳定供料能力使得抛丸/喷砂工艺可以高效完成以下工作[[S8,S10,S14]]：
- **去除飞边与毛刺**：高速弹丸可清除分型面及浇口周边的飞边毛刺[[S15,S25]]。
- **清除氧化皮与脱模剂残留**：达到 SSPC-SP6 至 SSPC-SP10 级别的工业清洁度要求，完全去除表面粘砂、浮渣及脱模剂[[S20,S25]]。
- **调控表面粗糙度**：通过选择磨料粒度和喷射参数，为后续涂装、电镀提供适宜的锚纹深度[[S20]]。
- **引入残余压应力**：受控喷丸可在压铸件表层产生峰值达材料抗拉强度 50%～60% 的残余压应力，抵消冷却引起的表层拉应力，显著提升疲劳寿命与应力腐蚀抗力[[S1,S31,S19]]。

## 常见故障与维护
磨料循环系统中涉及气流分选与粉尘回收的旋风分离装置常见故障及其排除方法如下表[[S7]]。

| 故障现象 | 故障原因 | 排除方法 |
| :--- | :--- | :--- |
| 回收效率显著下降 | 下部不严密漏气；灰斗装满 | 采用气密性卸灰器或防底部漏风措施；定期清灰 |
| 排灰口堵塞 | 排灰口太小；粉末积存过多且受潮 | 加大排灰口直径；及时清理积存粉末 |
| 锥体或旁路磨穿 | 旋转气流速度过高，粉料磨损严重 | 修补或更换磨损件；采取减磨衬里防护 |

管路堵塞预防：在喷砂机顶部设置过滤网，阻挡大颗粒异物进入管路[[S13]]。常规运维要求包括：运转前对所有运动部件定期润滑；经常检查抛丸器叶轮、叶片、定向套及护板的磨损情况，异常时立刻更换；开机前检查抛丸器及振动输送电机的紧固螺栓有无松动[[S12]]。

## 相关设备与系统接口
在一个完整的抛喷丸清理设备中，磨料循环系统与其余四大模块形成闭环物料流和信号联锁[[S3,S6]]：
- **抛丸器/喷枪**：消耗磨料的核心执行单元，接收循环系统供给的合格磨料。
- **清理室**：容纳工件并防止磨料外逸，底部连接螺旋输送器或振动输送机，将使用后的磨料送入循环系统。
- **工件输送系统**：如吊钩、转台、辊道等，与磨料循环系统的运行节拍联动，保证清理覆盖率。
- **除尘系统**：从丸砂分离器、流幕分离器等吸风口抽取粉尘，维持系统负压和气体净化[[S3,S6]]。环保抛丸机通常配置独立除尘容器，协同循环系统实现车间低尘作业[[S26]]。

## 标准与规范
磨料循环系统的术语、组成及功能要求在 GB/T 25370-2020《铸造机械 术语》中予以明确规定[[S4,S6,S10]]。该标准定义了磨料循环系统、丸砂分离器、弹丸闸门等相关术语，是系统设计、验收及安全评估的基础依据。在选用可循环金属磨料时，可参照相关的 SAE 粒度规格（如铸钢丸 S 系列、铸钢砂 G 系列）[[S27]]。
