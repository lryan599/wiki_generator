## 概述

渗漏（Leakage），在压铸行业中也常称为漏压、耐压不良，是指压铸件在耐压试验过程中，因内部存在贯穿性缺陷，导致非预期的气体或液体向外渗出的缺陷[[S39,S31,S53]]。根据JB302-62和GB5611-85标准的压铸缺陷分类体系，渗漏不属于常规的外观类、孔洞类或裂纹类缺陷，而是被归类为后处理/性能类缺陷，其直接诱因是压铸件内部致密度不足[[S65]]。

## 缺陷特征与表现

渗漏缺陷在铸件外观上通常没有肉眼可见的异常[[S5,S31]]。其核心特征表现为：对压铸件施加超出其工作压力30%~50%的水或气体介质进行压力试验时，会出现漏水、渗油，或浸入水中时产生连续气泡的现象；在使用干式压力试验机检测时，设备会触发报警信号[[S5,S31,S14]]。

渗漏发生的根本原因是铸件内部形成了从内表面延伸至外表面的完全贯穿性通道，使承压介质能够沿此通道非预期地渗出[[S14,S45,S31]]。图1展示了一典型压铸件试压时外表面的渗漏位置分布。

## 缺陷形成机理

压铸件产生渗漏的本质，是铸件壁厚方向上存在的内部缺陷相互连通，形成了贯穿性通道[[S29,S53]]。具体而言，单独的、孤立的气孔、缩孔、裂纹或冷隔等缺陷通常不会直接导致渗漏；只有当这些分散缺陷沿壁厚方向完全连通，构成从铸件一侧通向另一侧的路径时，才会在承压时出现泄漏[[S29,S53]]。渗漏的主要形成路径包括以下几种类型：

### 缩松网络贯穿

当压铸合金的结晶温度区间较宽时，凝固过程会形成海绵状或层状的连通缩松网络。未得到补缩的松散枝晶结构直接暴露在铸件表层，形成贯穿壁厚的渗漏通道。这类渗漏区域的断口上可观察到连续不规则的缩孔通道同时连通铸件的两个表面[[S51,S29,S66]]。

### 双膜氧化夹杂路径

在轻合金压铸件中，相当比例的渗漏源自充型过程中的湍流卷裹。金属液在紊乱状态下，表面的氧化膜发生折叠，形成内部干燥-干燥面贴合的未润湿双层氧化膜缺陷（bifilm）。这些相互缠绕的薄膜层在铸件内部构成连续的薄型通道，无需依赖极高的孔隙率即可贯穿壁厚形成渗漏[[S50,S3,S24]]。此外，尺寸大于5 mm的氧化物夹杂能够直接跨接整个铸件壁面，将内外表面连通[[S3,S24]]。

### 表面致密层不足与机加工破坏

压铸过程中，最先接触低温模具型腔的表层金属液会快速凝固，形成一层完全致密的表面组织。若该致密层厚度不足，或后续机加工的切削深度超过了该层厚度，原本被致密层屏蔽的内部连通缩松和显微孔隙便会暴露于承压面，形成贯穿性渗漏通道。这类型渗漏在机加工后的压铸件失效案例中占有很高比例[[S51,S41,S27]]。

### 气孔连通与蠕虫状气孔

普通弥散分布的孤立析出性气孔受气体扩散距离和枝晶臂分隔的限制，不会自然连通形成渗漏通道。仅当铸件局部气孔体积占比达到20%~30%的极高数值时，才可能出现气孔互相连通的情形。但蠕虫状长条形气孔作为一种特殊例外，因其伴随凝固前沿同步生长形成贯穿铸件的长隧道，可直接成为高效的渗漏路径[[S50,S66,S54]]。

## 主要产生原因

渗漏的产生是压铸工艺参数、模具设计、合金特性及后续加工等因素共同作用的结果。各维度诱因及其对应的典型渗漏特征见下表[[S14,S45,S41,S36,S28,S34,S29]]。

| 维度 | 诱因 | 典型渗漏特征 |
| :--- | :--- | :--- |
| **工艺参数** | 比压（铸造压力）不足 | 铸件整体致密度偏低，渗漏呈多点随机分布，无明显集中区域。 |
| | 增压启动时机过晚 | 内浇口提前凝固，远端厚壁区域压力无法传递，集中出现缩松渗漏。 |
| | 充型速度不合理 | 引发剧烈卷气，渗漏集中于充型流头汇合位置或积气区域。 |
| | 模具温度场不当 | 内外侧温差不足，厚壁区域补缩失效，渗漏集中在厚大热节位置。 |
| | 浇注温度过高 | 扩大了凝固总收缩量，整体铸件发生缩松渗漏的概率显著上升。 |
| **模具设计** | 浇注系统布局不合理 | 流道距远端区域过远、内浇口截面积过小，增压补缩无法传递，远端集中出现渗漏。 |
| | 排气道设计不足 | 局部卷入气体无法顺利排出，在对应积气区域产生密集气孔类渗漏。 |
| | 冷却水路排布不合理 | 热节位置无针对性冷却，壁厚局部形成孤立液相区，无法补缩导致贯通缩松渗漏。 |
| **合金特性** | 流动性差 | 充型过程过早凝固，易形成冷隔、未熔合类渗漏，且渗漏位置伴随明显冷隔形貌。 |
| | 体收缩率过高 | 凝固补缩需求大，若无足够的金属液补充，易形成大面积连通缩松渗漏。 |
| | 结晶温度区间宽 | 枝晶搭接成骨架阻断补缩通道，易形成连续网络状缩松，导致大面积分布式渗漏。 |
| **后续加工** | 机加工余量过大 | 切削深度超过铸件表面致密层厚度，内部连通孔隙暴露于密封面，渗漏位置完全对应加工面。 |

### 不同合金的渗漏差异

*   **铝合金**：渗漏常发生于机加工破坏表层致密层之后，使原本被封闭的内部连通孔隙暴露[[S1,S22]]。
*   **镁合金**：因结晶温度范围宽，枝晶搭接后补缩不足，易在最后凝固的热节部位产生大量分散微孔，这些微孔连通后形成渗漏[[S1,S29]]。
*   **锌合金**：渗漏多由薄壁区域充型时卷入的分散气体连通而产生[[S1]]。

## 检测方法与评价标准

### 常用检测方法

| 检测方法 | 操作要点与特点 | 适用场景与灵敏度 |
| :--- | :--- | :--- |
| **气压鼓泡法（浸水法）** | 将铸件密封后通入超过工作压力30%~50%的压缩空气，完全浸入水中，观察是否有连续气泡溢出。成本低，但无法准确定量泄漏率，检测后工件需烘干[[S21,S3]]。 | 中低泄漏量要求的通用压铸件快速筛查。 |
| **干空气检漏法** | 包含直压法、差压法、质量流量法。利用气体压力衰减、差压传感器或质量流量计测量泄漏，无需后续烘干，适合自动化生产[[S3,S35]]。 | 各类生产线场景：直压法适用于2bar以下、100ml以内；差压法可支持10bar以上高压检测；质量流量法精度可达0.001mL/s[[S35]]。 |
| **氦质谱检漏** | 利用氦气作为示踪气体，通过质谱仪进行检测。检测灵敏度可达 10⁻¹¹ 至 10⁻¹² Pa·L/s，比普通空气检测方法上限高出10000倍，可识别极小孔隙的微小泄漏。设备和成本最高[[S35]]。 | 高精度、超低泄漏要求的高端精密压铸件[[S35]]。 |
| **真空浸渗前处理与检测** | 在浸渗后通常需进行再次气密检测以验证堵漏效果，典型设备如ULVAC真空箱自动氦检漏系统可高效完成工件的气密性终检[[S15]]。 | 属于浸渗补救工序后的必检环节。 |

### 验收标准与等级

目前，中国国家标准对压铸件的渗漏要求多由供需双方协商确定。例如，GB/T 15114-2023《铝合金压铸件》和GB/T 25747-2022《镁合金压铸件》均规定，对铸件的气压密封性或液压密封性有要求时，应按双方商定的验收标准执行[[S46,S17]]。GB/T 16253-1996《承压钢铸件》则规定，耐压致密试验可在机加工或未加工铸件上进行，且试验前不得进行氧化处理或施加保护性涂层[[S13]]。

国际标准如ASTM B系列压铸合金标准要求，当合同中规定了压力tightness要求时，需符合供需双方约定的验收准则，亦可参考ADCI Product Standard E17的相关规范[[S9,S60]]。北美压铸协会（NADCA）发布的规范核对清单，则明确规定气密性检测项目需包含试验压力、保压时长、允许泄漏阈值及不合格品的补救方案（如浸渗/修补）[[S7]]。

在通用密封性分级体系（A~G级）中，涵盖了从完全不允许渗漏到允许特定泄漏量或压降的多个等级。例如，A级完全不允许渗漏；C级允许最大泄漏率2 mL/min；E级允许压降不超过13.8×10⁻³ MPa/min[[S59,S58]]。这为汽车油路件、水冷板、气动元件等不同要求的产品提供了验收基准。

## 预防与对策

渗漏的防控措施分为生产过程中的工艺改善和缺陷产生后的补救处理。

### 工艺主动改善

1.  **提高压射比压**：提高比压可显著降低铸件缩孔率，提升基体致密度，是减少渗漏的有效预防措施。对于铝合金压铸件，以800公斤/厘米²为界，继续提高比压对质量的改善效果将显著降低[[S12,S53,S55,S62]]。
2.  **优化浇注与排气系统**：合理设计浇注系统和排气道，目的是引导金属液平稳充型，减少气体卷入，避免形成内部连通孔隙[[S53,S4,S62]]。
3.  **采用局部挤压**：针对压铸件上的厚大热节部位，在金属液处于半固态时插入挤压销进行加压补缩，可大幅降低缩孔缩松发生率。有案例显示，某壳体在该措施下，批量生产泄漏率由2.8%降至0.8%[[S6,S11,S42,S8]]。
4.  **控制模具温度**：将模具温度控制在合理区间对减少内部缺陷至关重要。有实证研究表明，铝合金压铸模具温度控制在160℃~224℃之间时，铸件内部砂眼直径可控制在0.4mm以下，显著降低渗漏风险[[S11,S42,S41]]。

### 补救处理工艺

1.  **浸渗处理**：渗漏缺陷产生后应用最广泛的补救工艺。其原理是在真空压力条件下，使液态浸渗剂渗入铸件内部微细孔隙，经固化后封堵渗漏通道[[S23,S63,S52,S61]]。
    *   **浸渗剂**：分为无机和有机两大类。**无机浸渗剂**以硅酸钠（水玻璃）为主，可耐500℃高温，成本低，但粘度高，浸润性差，有时需多次浸渗[[S23,S19,S30]]。**有机浸渗剂**以合成树脂（如PC504）为代表，浸润性好，浸渗效率高，密封性稳定，但耐温上限约200℃，价格约为无机浸渗剂的20倍[[S23,S19,S30]]。
    *   **效果**：对于微孔缺陷直径在0.5mm以下的承压压铸件，一次浸渗合格率可达95%以上[[S23,S63]]。研究表明，某汽车空调压缩机EDC前盖产品，在采取工艺优化将泄漏率降至0.8%的基础上，再经全数真空浸渗处理后，批量生产平均泄漏率可进一步降至0.2%[[S11]]。

2.  **热等静压（HIP）处理**：高端补救工艺。将铸件置于密封耐压容器中，抽真空并充入惰性介质，在接近合金固相线温度及约100MPa的高压下，将内部封闭的孔隙压实闭合并实现扩散结合。例如，Al-7Si-0.3Mg合金经HIP处理后，孔隙率可由0.5%降至0.03%，可大幅消除渗漏风险并提升力学性能[[S44,S38,S18]]。

## 相关缺陷与关联关系

渗漏并非独立于其他缺陷存在，而是多种内部缺陷在特定条件下的综合表现结果。

*   **与孔洞类缺陷（气孔、缩孔/松）的关系**：这些是构成渗漏通道的基本“单元”。单独的、孤立的气孔或缩孔不导致渗漏，只有当它们在壁厚方向上相互连通时，才会形成渗漏通道。其中，海绵状或层状分布的连通缩松网络是主要的渗漏来源之一。
*   **与裂纹类缺陷的关系**：显微裂纹、冷隔等可以直接作为渗漏通道的一部分，尤其是当它们与表面连通时。
*   **与表面缺陷及冷隔的关系**：铸件表面的霜状形貌、细小冷隔，以及氧化皮褶皱，都可能诱发局部的内部缩孔，并最终形成贯通性的渗漏路径。
*   **区别与判定**：渗漏是气密性或水密性试验的最终结果表现，属于性能类缺陷；而气孔、缩松、裂纹等是可以通过目视、X光探伤或金相检验在中间过程发现的内部组织缺陷，属孔洞类或裂纹类缺陷。判断渗漏成因时，常需将铸件从泄漏点精确位置剖开，用显微镜检查断口特征，以鉴别是氧化膜、缩松还是其他缺陷导致了渗漏通道的形成[[S64,S20,S33]]。
