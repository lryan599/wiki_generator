## 概述

碳当量（Carbon Equivalent，写作CE，部分场景也写作CEV）在铸造与焊接工程中是两类核心技术参数的统称。一类面向铸铁凝固行为评估，将Si、P等对共晶点位置有显著影响的元素折算为等效碳含量，用于表征铁液接近共晶成分的程度，关联铁液流动性、石墨析出特性与铸造工艺性能[[S8,S6]]；另一类用于碳钢与低合金铸钢的焊接性预判，将各合金元素对淬硬、冷裂及脆化的影响换算为等效碳含量，作为评价材料焊接冷裂纹倾向的参考指标[[S8,S6]]。

碳当量在非铁合金压铸的主流生产实践中未作为常规成分控制指标使用，压铸铝合金、锌合金、镁合金的相关标准与专业手册均未将碳当量列为工艺控制参数[[S24,S11]]。本词条在阐释碳当量的经典定义、公式体系与工艺判据的同时，如实反映其在压铸领域的适用范围仅限于极少数实验性铁基压铸的现状。

## 铸铁领域的碳当量

### 共晶碳当量CEV

在铸铁生产中，碳当量用于衡量铁液成分接近共晶点的程度。多数铸铁中的硅强烈缩小共晶成分中的碳含量，磷也起类似作用。为保证实际铁液能在共晶点附近凝固，铸造工作者将Si和P的含量折算为等效碳含量，与实测碳含量叠加，引出碳当量的概念[[S1]]。

普通灰铸铁、球墨铸铁常用共晶碳当量公式为[[S17,S1]]：

\[
CEV = \omega_C + \frac{\omega_{Si} + \omega_P}{3}
\]

式中各元素含量均以质量分数（%）代入。该式源于铁‑碳相图中共晶点碳含量因硅、磷而产生的偏移规律[[S1]]。

### 液相线碳当量CEL

英国铸铁研究协会（BCIRA）基于热分析法提出了液相线碳当量CEL，表达式为[[S17]]：

\[
C_{EL} = \omega_C + \frac{\omega_{Si}}{4} + \frac{\omega_P}{2}
\]

液相线温度T<sub>L</sub>（℃）与CEL的关系为：

\[
T_L = 1669 - 124 \times C_{EL}
\]

CEV通过化学分析计算获得，CEL则通过实测冷却曲线的液相线温度回归得到。T<sub>L</sub>与C<sub>EL</sub>的相关性优于T<sub>L</sub>与C<sub>EV</sub>，因此CEL在炉前快速热分析检测中得到更广泛应用[[S17]]。

### 碳当量与铸造工艺性能的关系

碳当量直接影响铁液在凝固末期的残余液相流动弥合能力。实验数据表明：在CE为4.21%～4.95%区间内，残余液相的流动弥合能力C（单位℃⁻¹）由0.00497逐步升高至0.0085，在CE≈4.67%处出现一个峰值0.00571[[S21]]。这一规律可用于评估铸铁件补缩充填能力与缩松倾向。

## 铸钢领域的碳当量

### 焊接性评估公式

国际焊接学会（IIW）推荐的碳钢与低合金钢通用碳当量公式已被全球多数标准采纳[[S6,S10,S27,S5]]：

\[
CE = \omega_C + \frac{\omega_{Mn}}{6} + \frac{\omega_{Cr} + \omega_{Mo} + \omega_{V}}{5} + \frac{\omega_{Ni} + \omega_{Cu}}{15}
\]

该公式纳入国内GB/T 7659《焊接结构用碳素钢铸件》标准，ASTM A216标准也采用相同表达式[[S5,S2]]。

不同机构依据合金体系差异定制了各自的碳当量公式，常用变体汇总如表所示[[S26]]。

| 采用机构 | 碳当量公式（质量分数%） |
|---|---|
| 国际焊接学会/ASTM A216/GB 7659 | CE = ω(C) + ω(Mn)/6 + [ω(Cr)+ω(Mo)+ω(V)]/5 + [ω(Ni)+ω(Cu)]//15 |
| 日本焊接学会（JIS G 5102） | CE = ω(C) + ω(Mn)/6 + ω(Si)/24 + ω(Ni)/40 + ω(Cr)/5 + ω(Mo)/4 + ω(V)/14 |
| 法国船级社 | CE = ω(C) + ω(Mn)/6 + [ω(Cr)+ω(Mo)+ω(V)]/5 + [ω(Ni)+ω(Cu)+ω(Si)]/15 |
| 美国焊接学会 | CE = ω(C) + ω(Mn)/6 + ω(Si)/24 + ω(Ni)/15 + ω(Cr)/5 + ω(Mo)/4 + ω(Cu)/13 + ω(P)/2 |
| ASME 锅炉及压力容器规范 | CE = ω(C) + [ω(Mn)+ω(Si)]/6 + [ω(Cr)+ω(Mo)+ω(V)]/5 + [ω(Ni)+ω(Cu)]//15 |

### 碳当量与焊接性分级判据

碳当量越大，钢材的焊接冷裂纹敏感性越高，可据此划分为三个工艺区间[[S6,S10]]：

- **CE ＜ 0.4%**：淬硬倾向不明显，焊接性良好，常规工艺下焊接接头不易产生裂纹。
- **CE = 0.4%～0.6%**：淬硬倾向增加，冷裂纹风险升高，需配套焊前预热、焊后缓冷措施。
- **CE ＞ 0.6%**：冷裂纹敏感性高，需较高预热温度及严格工艺措施，部分场景需增加焊后热处理。

### 相关标准限值

国内GB/T 7659标准仅当需方有要求时将碳当量作为附加验收指标，三个牌号的CE上限规定如下[[S2,S14]]：

| 牌号 | CE上限（%） |
|---|---|
| ZG200‑400H | ≤0.38 |
| ZG230‑450H | ≤0.42 |
| ZG275‑485H | ≤0.46 |

ASME第Ⅰ卷采纳的碳当量公式中，如Cr、Mo、V或Ni、Cu含量未能检测，允许代入简化式CE = ω(C) + [ω(Mn) + ω(Si)]/6 + 0.15，规范限定该应用场景下CE上限为0.45%，且必须保持最低120℃的预热温度[[S25]]。ASTM SA‑105锻件和SA‑106管材的补充要求也引用同一公式，CE控制上限视壁厚而定，分别为0.47、0.48或0.50[[S25]]。

面向淬硬倾向评估的扩展碳当量公式则在IIW公式基础上调整了各元素权重，表达式为CE = %C + %Mn/6 + %Mo/4 + %Cr/5 + %Ni/15 + %Cu/15 + %P/3，专门用于预测钢的淬透性和冷裂纹风险[[S15]]。

## 其他标准中的碳当量

在高温用灰铸铁领域，ASTM A319标准采用碳当量公式为[[S22]]：

\[
CE = \omega_C + 0.3(\omega_{Si} + \omega_P)
\]

该标准将耐热灰铸铁按碳当量配合不同的Cr、Cu、Ni、Mo、V合金添加等级，用于炉条、钢锭模、玻璃制品成型模等高温铸件（使用温度可达760℃）的成分控制[[S22]]。

需要指出，国内GB/T 5612《铸铁牌号表示方法》标准中，灰铸铁、球墨铸铁、可锻铸铁、蠕墨铸铁牌号均以力学性能（抗拉强度、伸长率）命名，牌号本身不直接标注碳当量数值。实际生产中碳当量需通过成分化验或热分析另行计算得出[[S9,S4]]。

## 碳当量在压铸领域的适用边界

### 主流非铁压铸合金无碳当量控制项

压铸行业的主力合金为铝、锌、镁、铜合金。相关国家标准汇编、国际高压压铸文献体系以及公开压铸专业手册一致表明，碳当量未作为这些合金的常规成分控制或工艺控制指标[[S24,S11]]。与“碳当量”概念性质相近的非铁压铸参数是“锌当量”（zinc equivalent），用于将黄铜类合金中各元素含量折算为等效锌含量，以判断合金相组成[[S24]]。

北美压铸协会（NADCA）出版的压铸工艺控制核心著作，以及大量高压压铸学术文献中，未出现同时关联“carbon equivalent”与“high pressure die casting”的针对性研究内容，碳当量不属于压铸领域通用工艺控制范畴[[S7,S12]]。

### 黑色金属压铸中的缺位现象

尽管压铸技术近年已拓展至灰铸铁、球墨铸铁、碳钢和不锈钢等黑色金属的实验性生产，所查公开专业手册（如《压铸成形工艺与模具设计》）未将碳当量作为黑色金属压铸的工艺控制指标[[S20,S13]]。手册中对铁基压铸的工艺指导停留在浇注温度、压射速度、模具预热温度等热力学参数，未涉及以碳当量调控共晶度或冷裂倾向的系统论述[[S20]]。

综上，碳当量在压铸领域的应用属于极小众属性，仅可能出现在极少数铁基压铸的特定工艺探索中，所有可查的公开压铸生产规范、行业标准与专业手册体系均未将其纳入正式生产控制要求[[S3]]。碳当量的标准术语定义出现在普通砂型铸造相关的通用铸造术语条目里，与压铸场景直接关联甚微[[S19]]。
