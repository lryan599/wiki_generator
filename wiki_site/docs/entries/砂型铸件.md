## 概述

**砂型铸件**（sand casting）是采用砂型铸造工艺生产得到的金属铸件，即将熔融金属注入以砂为主要材料制成的铸型中，凝固后获得的具备指定形状、尺寸和性能的金属零件或毛坯[[S41,S61,S23]]。世界范围内，60%～80% 的金属铸件由砂型铸造工艺产出[[S23,S61,S1]]。

在压铸技术维基体系中，砂型铸件主要作为对比参考与基础工艺背景存在。其成形原理、组织性能、适用合金及应用场景与压铸件（die casting）有本质差异，系统理解两者的区别有助于正确选择成形工艺与铸件类型。

## 分类

砂型铸造按型砂黏结剂类型可分为黏土砂型和化学黏结剂砂型两大类，其中黏土砂型又可细分为黏土湿型、黏土干型和表面干型；化学黏结剂砂型包括水玻璃砂型、树脂自硬砂型等子类[[S58,S33,S12]]。

| 砂型类别 | 主要特点 | 应用场景 |
|---|---|---|
| 黏土湿砂型 | 以黏土和水为黏结剂，湿态下直接合型浇注；资源丰富、可回收复用、生产周期短、效率高，占铸件总产量60%～70%；强度较低，易产生冲砂、夹砂、气孔等缺陷 | 中小型铸件，机械化自动化大批量生产[[S34,S58,S55]] |
| 黏土干砂型 | 经烘干后浇注，强度和透气性显著提高；生产周期长、耗能多、难以机械化 | 质量要求高的中大型铸件，单件小批生产[[S12]] |
| 表面干型 | 仅型腔表面一薄层烘干，兼有干型和湿型的优点 | 中大型铸件[[S12]] |
| 树脂自硬砂型 | 依靠有机树脂化学反应自行硬化，无需烘干，强度高，尺寸精度和表面光洁度好；成本较高，部分旧砂回用困难 | 中等批量的大型铸件[[S13,S17]] |
| 水玻璃砂型 | 以无机水玻璃为黏结剂，资源丰富、无毒无味、硬化速度快、透气性好；溃散性差、旧砂再生回用难度高，铸件易粘砂 | 单件小批量重型铸钢件[[S13,S53]] |

## 工艺特点与成形过程

砂型铸造的全成形标准流程依次包括砂料制备、造型、制芯、合箱、熔炼浇注、落砂、铸件清理及铸件检验共8个核心环节[[S35,S58]]。砂型铸造一般采用重力浇注（重力铸造），特殊需求时也可采用低压、反重力等辅助充型工艺[[S30]]。

### 与压铸工艺的核心差异

砂型铸造与压力铸造（压铸）在模具类型、充填压力和成形介质方面存在本质区别[[S59,S4,S7]]：

| 对比维度 | 砂型铸造 | 压力铸造（压铸） |
|---|---|---|
| 模具类型 | 一次性可消耗砂质铸型，每浇注一次铸型即被破坏，需重新制备 | 永久式高强度工具钢压铸模，可重复使用数千到数十万次 |
| 充填压力 | 主要依靠金属自身重力充型，充型压力接近常压 | 高压设备将熔融金属高速压入型腔，充填压力可达数十至数百兆帕 |
| 成形介质 | 型腔主体由石英砂等耐火颗粒混配黏结剂制成，铸型整体透气性好 | 型腔为致密的金属钢模，几乎无透气性，需专门设置排气溢料结构 |

## 典型材料

砂型铸件适用材料范围极为广泛，几乎所有工业铸造合金均可通过砂型工艺制备，包括灰铸铁、球墨铸铁、各类铸钢、铸造铝合金及铸造铜合金等[[S9,S24,S38,S47]]。

压铸件主流可加工合金范围为锌、铝、镁及部分铜合金等低/中熔点非铁合金；钢铁等高熔点合金常规压铸难度大，压铸模寿命低，不具备大规模商用价值[[S5,S46,S56,S2]]。

砂型铸造常用铝合金牌号包括 **208、413、213、356、319、355、514、535** 等；压铸件常用铝合金牌号则以 **380、384、386、390** 等共晶铝硅合金为主，合金设计优先保障低熔点和高流动性，以减少模损并提升生产效率[[S24,S38,S46]]。

## 组织与性能特征

### 晶粒度

砂型铸造冷却速度远低于压铸工艺。普通砂型铸造铝合金铸件晶粒尺寸通常在100～1200 μm区间，且随壁厚增大呈上升趋势，例如3 mm薄壁砂型铸铝件晶粒约为520 μm，6 mm壁厚件可达1100 μm。压铸件因金属模激冷作用，结晶速度极快，表层0.8～1.2 mm区域晶粒显著细化[[S60,S3]]。

### 力学性能

不同铸造工艺在同成分合金条件下性能差异明显：砂型铸件力学性能处于偏低水平，金属型铸件性能优于砂铸件，压铸制备的合金力学性能在三类工艺中最高[[S14,S37,S28]]。压铸件抗拉强度普遍比同成分砂型铸件高25%～40%，但延伸率稍低[[S56,S5,S50]]。

### 尺寸精度

依据 GB/T 6414 标准，砂型铸件尺寸公差覆盖 CT8～CT15 区间，普通成批生产砂型铸件公差处于 CT10～CT13 范围，重型砂铸钢件公差为 CT13～CT15[[S39,S43,S63]]。常规有色合金压铸件尺寸公差处于 IT11～IT14 区间，不受分型面及活动型芯影响的局部尺寸公差可达 CT3～CT6（IT9 级）[[S6,S63]]。

### 表面粗糙度

普通砂型铸件非加工表面粗糙度 Ra 典型区间为 12.5～100 μm：砂铸钢件常规 Ra 为 50～100 μm，砂铸铁件常规 Ra 为 12.5～25 μm，采用特殊精密造型工艺的砂铸件最优 Ra 可达 6.3 μm。常规新模压铸件表面 Ra 处于 0.4～3.2 μm 区间，最高精度可达 0.32 μm[[S25]]。

### 最小壁厚

常规砂型铝合金铸件的最小可实现壁厚约为 4～5 mm[[S45,S62]]。压铸件可实现的最小壁厚随合金不同差异显著：锌合金可至 0.3 mm，铝合金可至 0.5 mm，镁合金可至 0.6 mm[[S56,S19]]。

### 内部质量

砂型铸造的夹杂物面积占比在三类工艺（砂型铸造、金属型铸造、电渣熔铸）中最高，可达 1.0% 以上[[S16]]。

## 常见缺陷及机理

砂型铸件常见的各类缺陷及其成因如表所示[[S40,S51,S42,S48,S10,S21,S31]]：

| 缺陷类型 | 成因简述 | 与压铸件区别 |
|---|---|---|
| **气孔** | 砂型发气、浇注过程砂粒卷入或析出气体滞留；砂型本身可作为气体来源，产生较大气穴 | 压铸件气孔主要来自高压高速充型过程中型腔内气体无法完全排出，被裹挟进入金属液留存 |
| **缩孔缩松** | 金属液凝固过程中体积收缩后得不到足够的冒口补缩 | 压铸件缩孔为高压充型后外层快速凝固形成密封硬壳，内部收缩时无法获得充分补缩；压铸件的气缩孔是气孔与缩孔叠加的复合缺陷 |
| **砂眼** | 金属液冲刷作用下砂型表层脱落的砂粒被裹入铸件，为砂型铸造独有缺陷 | 压铸件因使用金属永久模具，不存在砂眼缺陷 |
| **夹渣** | 浇注过程中裹挟的熔渣、砂型表面脱落的涂料层或金属液表面氧化皮聚集 | 压铸件夹渣多为润滑剂残留、未完全排除的压铸合金氧化夹杂 |
| **裂纹（热裂）** | 高温状态下铸件凝固收缩时受到砂型机械阻碍导致开裂，冷却过程中各处冷速不均引发残余应力裂纹 | 压铸件裂纹还包含脱模不当、模具温度控制不合理引发的开裂 |
| **变形** | 铸件凝固冷却过程中收缩不均，或后续加工/热处理过程中不同部位冷却速度不一致导致几何尺寸偏离 | 压铸件变形还包含压射顶出过程受力不均引发的形变 |

## 应用领域与局限性

### 应用优势

砂型铸造适用的铸件尺寸和重量范围极大，小至100 g，大至数吨甚至数十吨，几乎适配所有金属及合金种类，是当前应用最广的铸造工艺，占全部铸件产量的60%～70%[[S1,S27,S47]]。典型应用场景包括：

- 机床床身、箱体等大型铸件
- 大型发动机缸体、气缸盖、曲轴
- 重型泵阀壳体
- 大型船舶螺旋桨[[S1,S22,S27]]

在单件、小批量生产中，砂型铸造模具（木模或铝模）投入成本低，经济性好[[S30,S26]]。

### 主要局限

- 单个砂质铸型仅可一次性浇注使用，需重复制型，生产效率远低于压铸[[S30,S57]]；
- 铸件尺寸精度较低（通常 CT7～CT11 级），表面粗糙度 ≥ 6.3 μm，近净成形程度差，后续机械加工余量较大[[S26]]；
- 大批量生产时单件成本远高于压铸工艺[[S30,S26]]；
- 材料致密度和金相组织均匀性不及压铸件[[S27]]；
- 对薄壁精密件的成形能力严重不足[[S56,S45]]。

## 与压铸件的对比关系

以下为砂型铸件与压铸件在核心参数上的结构化对比[[S52,S20,S32, S59,S7,S56,S45,S5,S25,S3,S43,S6]]：

| 对比维度 | 砂型铸件 | 压铸件 |
|---|---|---|
| 成形方法 | 重力浇注为主，砂质一次性铸型 | 高压高速充填，永久金属模具 |
| 模具/铸型 | 砂质模型，每次浇注后破坏 | 高强度工具钢压铸模，可重复使用数千～数十万次 |
| 充填压力 | 接近常压（重力充型） | 数十至数百兆帕 |
| 冷却速度 | 缓慢 | 极快（金属模激冷） |
| 晶粒度（铝合金） | 100～1200 μm（壁厚3 mm约520 μm，6 mm约1100 μm） | 显著细化，表层尤为致密 |
| 力学性能 | 抗拉强度偏低 | 抗拉强度比同成分砂铸件高25%～40% |
| 尺寸公差 | CT8～CT15（成批生产CT10～CT13） | IT11～IT14（局部CT3～CT6，IT9级） |
| 表面粗糙度 Ra | 12.5～100 μm（最优6.3 μm） | 0.4～3.2 μm（最高0.32 μm） |
| 最小壁厚 | 铝合金约4～5 mm | 锌合金0.3 mm，铝合金0.5 mm，镁合金0.6 mm |
| 可铸合金种类 | 几乎所有工业铸造合金（含钢铁） | 主要限于锌、铝、镁、铜等低/中熔点非铁合金 |
| 铸件重量范围 | 100 g～数十吨 | 以中小型为主（近年可达80 kg以上） |
| 经济批量 | 单件、小批量至大批量均可 | 仅适合千件以上的大批量生产 |
| 模具成本 | 低（木模、铝模） | 极高（高精度工具钢） |
| 内部质量 | 夹杂物面积占比最高（可达1%以上） | 存在卷气，无法常规热处理 |
| 典型应用 | 机床床身、大型缸体、泵壳、船舶螺旋桨等大件 | 汽车变速器壳、电子壳体、精密结构件等薄壁精密件 |

## 相关标准与规范

### 尺寸公差标准

GB/T 6414—1999《铸件 尺寸公差与机械加工余量》为我国砂型铸件通用尺寸公差标准，技术内容等效采用国际标准 ISO 8062:1994《铸件尺寸公差与机械加工余量体系》。标准将铸件尺寸精度从精到粗划分为 CT1～CT16 共16个等级[[S39,S44,S43]]。

- **大批量生产**砂型铸件常规可达到 CT8～CT13 级；
- **小批单件生产**砂型铸件常规可达到 CT10～CT15 级；
- 壁厚尺寸允许采用比对应基本尺寸公差粗一级的精度等级[[S39]]。

### 验收条件

砂型铸件基础验收条件在对应标准中明确包含三大维度[[S25,S8]]：

1. **尺寸公差**：符合选定的 CT 等级要求；
2. **表面粗糙度**：砂型铸钢件合格品表面粗糙度 Ra 通常不高于 50 μm，砂型铸铁件合格品表面粗糙度 Ra 通常不高于 25 μm；
3. **重量公差**：符合对应 MT 等级要求（MT 为质量公差代号，mass tolerances）。

### 重型铸钢件补充标准

重型机械用砂型铸钢件的公差等级应符合 GB/T 6414 中的 CT13～CT15 级规定，并参照 JB/T 5000.6 标准执行相关技术要求和检验规则[[S63]]。
