## 概述

“整体模”是压铸模具中一类成型零件结构的简称，其对应的标准化全称为**整体式压铸模**，亦可称作整体式模具、一体式模具。该术语在行业中常用于指代型腔与模块融合为一体的成型结构，区别于镶拼式（组合式）结构。按照压铸模构造分类，整体式压铸模属于最基本的模腔结构类型之一，在某些文献或图纸中也可特指整体型腔或整体模芯组件[[S7,S11,S8]]。对应的英文术语常见为 **monolithic die**、**one-piece die** 或 **integral die**。

## 定义与分类

在压铸模设计语境下，**整体式压铸模**的精确定义为：成形部分的型腔直接在一整块模块上加工成型，模块与型腔完全构成一个整体，不存在独立的镶件或拼接面[[S7,S10,S9]]。其结构示意通常表现为型腔或型芯直接在模板上加工完成，例如动模或定模上的型腔为一整块材料铣削而成[[S10]]。

需要明确的是，压铸模成型零件的结构形式通常分为**整体式**与**组合式（镶拼式）**两大类。其中整体式结构即本词条所指的整体模；而一些资料中所提及的“整体组合式结构”（型腔由整块材料制成后嵌入模套并固定），虽然同属成型零件为整体制造，但其并非直接以模板自身作为型腔，因此多数权威教材将其归入组合式范畴，与纯整体式相区分[[S10,S1]]。本词条聚焦于前者，即“模块即型腔”的整体式结构。

## 结构特征

整体模在结构上具有鲜明的几何与制造特征：

- **一体化成形**：型腔工作面与模板基体为同一块热作模具钢，完全消除了型腔拼接缝[[S7,S11]]。
- **紧凑的外形**：因无需额外的模套、镶块固定台肩等结构，模具整体外形尺寸较镶拼式结构显著缩小，有利于在拉杆空间有限的压铸机上使用[[S7,S11]]。
- **冷却水道布置简便**：由于没有镶拼间隙，可在模块内部连续钻削或加工出规整的冷却水道，不受拼合面干扰[[S7,S11]]。
- **无披缝痕迹**：铸件表面在对应于分型面和型腔内部均不会因镶拼而产生飞边或毛刺，特别适用于对外观要求较高的精密铸件[[S11,S7]]。

## 工艺特点

整体模的工艺性能由材料利用、制造难度和寿命表现等多个维度决定，表现出明显的优势与局限。

**主要优势**：
- **强度与刚性突出**：整体结构无接合面，抵抗高速高压金属液冲击的刚性好，变形倾向低[[S7,S6,S10]]。
- **铸件表面质量高**：无拼接缝隙，杜绝了因镶块间隙造成的飞边，零件尺寸精度更易保证[[S11,S7]]。
- **装配与维护简化**：省去了镶块的配装、研合工序，模具装配工作量小，整体尺寸紧凑[[S7,S11]]。
- **利于热管理**：冷却水道可直接在模块上加工，热传导路径连续，对模具温度控制有利，能提高压铸高熔点合金时的模具寿命[[S7,S11]]。

**显著局限**：
- **材料浪费严重**：整块模块采用昂贵的优质热作模具钢，非工作区材料消耗多，且在局部发生裂纹、烧蚀或磨损时，往往导致整个模块报废，修复极为困难[[S10,S9]]。
- **加工与热处理困难**：大型或深腔型腔的一次性加工量大，尤其对复杂曲面的切削、电火花加工要求高；同时，大尺寸整体模块在热处理时易产生变形和开裂，表面处理也受到尺寸限制[[S10,S1]]。
- **适用面较窄**：通常仅适合于形状简单、精度要求不高或小批量试制场合，随着模具加工技术的进步，整体式结构在大批量生产中已较少使用[[S9]]。

## 对比分析：整体式与镶拼式

为更清晰呈现整体模与镶拼式模具的工程差异，现将两类结构在典型维度上的对比整理如下表。数据综合自多部核心教材及设计手册[[S4,S9,S11]]。

| 对比维度 | 整体式（整体模） | 镶拼式 |
|---|---|---|
| 型腔形成方式 | 整块模板直接加工成型腔 | 由多块镶块拼合装入套板固定 |
| 强度与刚性 | 高，无形变接缝 | 良好，但拼合面可能削弱刚性 |
| 铸件表面 | 无披缝，表面光滑 | 拼合处可能产生飞边毛刺 |
| 模具尺寸 | 外形紧凑，便于适应小空间 | 需外加套板，外形较大 |
| 材料利用 | 整体采用高价热作钢，浪费大 | 仅镶块用优质钢，模体可用普通钢，成本低 |
| 加工与制造 | 简单型腔可一次加工，复杂型腔难度大 | 复杂型面可分块加工，降低难度，但装配精度要求高 |
| 维修与寿命 | 局部损伤即整模块报废，不易修复 | 易更换受损镶块，模具寿命可通过换件延长 |
| 冷却与热管理 | 冷却水道设计简单，热传导效率高 | 镶拼缝影响热扩散，冷却水道布置较困难 |
| 排气条件 | 排气仅依赖分型面与排气槽 | 镶拼间隙可辅助型腔深部排气 |
| 适用场景 | 小批量、简单件、试制、空间受限 | 大批量、复杂形状、高精度铸件 |

## 热特性与温度场

整体模因不存在镶拼气隙，其热传导路径连续，从金属液到模块本体的传热效率较高。相关数值模拟研究表明，整体式压铸模在开始连续压铸后，动模、定模及铸件位置的温度会迅速上升，并在约**10次压铸循环**后趋于稳定，达到热平衡状态[[S3]]。这一特性有利于生产过程中的工艺稳定性，并且因为传热均匀，型腔热应力集中度相对较低，对抑制热疲劳裂纹有一定益处。

## 应用场景

整体模主要应用于以下典型生产场景[[S7,S6,S11,S9]]：
- 型腔较浅的小型单型腔压铸模，例如小支架、简易壳体等。
- 形状简单、精度要求较低的压铸件，如普通连接件、垫片类零件。
- 小批量试制或新品开发阶段，要求快速出件且不进行热处理的模具。
- 受限于压铸机拉杆空间而必须压缩模具外形尺寸的情况，如某些老式设备的适配。
- 低熔点合金或小尺寸高熔点合金的压铸件，可借助整体模的冷却水道直接控温。

随着电火花加工、高速铣削技术的普及以及模具钢成本的不断优化，整体模在简单小型件领域仍有一定的性价比优势，但在现代大批量、高复杂度压铸生产中，其应用比例已大幅降低[[S9]]。
