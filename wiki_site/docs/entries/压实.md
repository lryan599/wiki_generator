## 概述

在高压压铸语境下，**压实**（亦称**局部加压**、**局部挤压**、**局部增压**，对应英文术语 *local squeeze in HPDC*）是指在压铸充型完成后，于铸件尚未完全凝固的半固态阶段，通过安装在模具上的压实销向铸件厚大热节部位施加远高于常规压射增压压力的机械力，强制实现局部补缩以消除缩孔缩松的特种压铸辅助工艺[[S12,S17,S4]]。该工艺的核心执行元件为**压实销**（*squeeze pin*），相关内容已被纳入NADCA压铸缺陷手册及工艺控制手册[[S5,S19]]。

## 定义与本质

压铸压实是在高压压铸充型流程结束之后，利用独立驱动机构（通常为液压油缸）推动压实销，直接作用于铸件内部孤立的厚大热节部位。通过短行程、高压力的强制补缩动作，将半固态区域的残留液体金属挤压到正在凝固收缩产生的空隙中，从根源上消除缩孔和缩松[[S4,S14]]。与压铸机自带的全局压射增压（intensification）不同，压实针对的是常规浇道补缩已无法到达的远端热节[[S8,S10]]。

国家标准GB/T 25370-2020《铸造机械 术语》收录了“压实、压实比压、压实行程、压实力”等标准术语条目[[S13]]。

## 分类

压铸压实可按作用范围和驱动方式进行分类[[S7]]：

| 分类维度 | 类型 | 说明 |
|---------|------|------|
| 作用范围 | 局部压实 | 工业主流形式，仅针对单个或多个孤立热节定点施压 |
| | 整体压实 | 对整个铸件全范围施压的特殊变体 |
| 驱动方式 | 液压压实 | 通过专用液压油缸驱动压实销，为工业普遍采用的形式 |
| | 机械压实 | 通过机械机构施加压力 |

需严格区分的邻近概念：

- **压实销（挤压销）**：是压铸压实**工艺的从属执行部件**，不属于独立工艺。
- **挤压铸造（squeeze casting）**：是一种采用低充型速度（<2 m/s）、无常规高速充型过程、全铸件在高压下凝固的**独立特种铸造工艺**，与作为高压压铸后处理辅助工序的压实有本质区别[[S3,S10,S16]]。

## 工艺作用

压铸压实的核心作用靶点是常规压铸工艺无法通过压射增压消除的孤立厚大热节[[S12,S17]]：

1. **消除内部缩孔缩松**：强制向正在凝固的热节区域补缩液体金属。
2. **提升致密度与密封性能**：针对壁厚差异大、局部热节难以通过浇道补缩的高气密性要求部件，显著降低泄漏风险。
3. **改善力学性能**：通过消除内部缺陷提升对应区域的静载及疲劳性能。

## 工艺参数与实施条件

压铸压实的实施效果取决于多个关键参数的精确控制：

### 主要工艺参数区间

| 参数项目 | 典型范围/数值 | 说明 |
|---------|--------------|------|
| 压实启动时间 | 高速压射结束后0.1～3 s | 对应合金固相分数约20%～40%的半固态区间[[S1,S15]] |
| 压实压力 | 型腔金属液压力3～4倍，可达270～392 MPa | 为常规型腔压力的数倍[[S1,S17]] |
| 保压时间 | 1.5～10 s | 持续至压实区域金属完全凝固[[S20,S15]] |
| 压实行程 | 6～16 mm（依热节体积计算） | 基于凝固体积收缩率计算并保留安全系数[[S1]] |
| 压实销直径 | φ5～φ25 mm | 根据热节部位情况选型[[S1]] |

### 行程计算原则

压实行程基于铸件热节部位体积与合金凝固体积收缩率计算。以铝合金为例，凝固体积收缩率β一般不超过6.5%，行程计算公式为 L = nβW/(πR²)，其中安全系数n一般取3[[S1,S20]]。

### 配合间隙

| 冷却条件 | 配合间隙 |
|---------|---------|
| 不通冷却水 | 0.04～0.06 mm[[S20]] |
| 通冷却水 | 0.02～0.04 mm[[S20]] |

## 模具结构与设备

压铸压实系统的核心机构由压实油缸、挤压销、挤压衬套及连接件组成，可根据设计需求布置在模芯或模框上[[S14,S1]]：

- **液压驱动**：既可利用压铸机自带的液压系统供油，也可采用独立的移动式小型专用液压站，以避免压铸机其他动作占用油量导致压实压力不足[[S4]]；
- **表面强化**：挤压销及衬套除氮化处理外，可采用PVD纳米镀钛工艺提升表面硬度、延长使用寿命[[S20,S1]]；
- **头部形状优化**：挤压销头部形状对压实效果有重要影响，可采用CAE软件进行模拟验证[[S1]]。

## 对铸件质量的影响

### 正面效果

- **渗漏率大幅降低**：气密性铸件试压渗漏率可从5%～12%降至0.3%～2.8%以下[[S4]]；
- **局部组织细化**：压实区域致密度显著提升，凝固组织细化[[S17]]；
- **元素再分布**：在Al-Si-Cu合金的高压压实区域，Cu元素会从2.6%偏降至1.6%，并在压实销外围表面区域富集[[S5]]。

### 工艺控制不当引发的缺陷

| 缺陷类型 | 原因 |
|---------|------|
| 飞边 | 压实压力施加过早，型腔内压力过高导致胀模[[S8]] |
| 粘模 | 压实销与衬套配合间隙不当或表面处理失效[[S20,S1]] |
| 微裂纹 | 加压启动过晚，金属已大量凝固，强行挤压引发生裂纹[[S17,S5]] |
| 补缩失效 | 启动时间过早（金属尚未开始凝固）或过晚（补缩通道已封闭）[[S15,S20]] |

## 典型应用场景

压铸压实工艺主要针对壁厚不均匀、气密性要求高的铝合金压铸件，已公开的适配合金牌号包括ADC12、Al-Si9Cu3(Fe)等，量产应用的典型铸件如下[[S12,S11,S19]]：

- 空调压缩机壳体
- ABS油泵泵体
- 铝合金汽车变速器壳体
- 电机转子端环

## 与类似工艺的区别

### 与压射增压（intensification）的区别

| 对比维度 | 压铸压实（局部压实） | 常规压射增压 |
|---------|-------------------|------------|
| 作用方式 | 压实销直接作用热节 | 压射冲头从浇口端全局传递 |
| 压力水平 | 可达270 MPa以上 | 受系统压力倍数限制 |
| 补缩限制 | 不受浇道凝固状态影响 | 浇道凝固后即无法有效传递压力 |
| 适用对象 | 孤立远端热节 | 靠近浇道的整体型腔[[S8,S5,S10]] |

### 与挤压铸造（squeeze casting）的区别

| 对比维度 | 压铸压实 | 挤压铸造 |
|---------|--------|---------|
| 工艺性质 | 高压压铸辅助工序 | 独立特种铸造工艺 |
| 充型速度 | 常规高压高速充型 | <2 m/s低速充型 |
| 加压范围 | 局部热节 | 全铸件[[S3,S10,S16]] |
| 热处理能力 | 通常不可热处理 | 可进行T6等固溶热处理[[S3]] |
