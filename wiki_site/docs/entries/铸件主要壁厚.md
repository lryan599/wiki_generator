## 概述

铸件主要壁厚是压铸工艺设计中优先参考的基准厚度参数。它指压铸件中占据整体结构占比最高、对压铸件绝大多数型壁的厚度具有代表性，且直接决定填充时间、内浇口速度、凝固时间、模具温度梯度、最终比压选择、留模时间等核心压铸工艺参数的壁厚值，不作为局部特殊位置的特征尺寸参考[[S69]]。

在压铸壁厚术语体系中，主要壁厚需与局部壁厚、公称壁厚、最小壁厚、平均壁厚等概念加以区分：

- **局部壁厚**：指压铸件中局部特殊特征位置（凸台、安装孔周边、局部热节区域）的非主体壁厚，仅作为局部特殊结构的尺寸定义，不用于整体压铸主工艺参数的基准选择[[S69]]。
- **公称壁厚**：属于工程图样标注的名义尺寸概念，是图纸上标注的平均名义壁厚。按ASME等材料标准规定，名义平均壁厚对应的最小允许实测壁厚为公称壁厚的0.875倍，仅用于图纸尺寸标注、公差校验场景[[S5,S65]]。
- **最小壁厚**：指在确定压铸工艺条件下，对应压铸合金能完整充满铸型的最小极限厚度[[S79]]。
- **平均壁厚**：通过面积加权公式计算得出。当壁厚不均匀时，可按 \(S_{n1} = (S_1A_1+S_2A_2+S_3A_3+\cdots)/(A_1+A_2+A_3+\cdots)\) 计算，或以面积占比加权的形式 \(S_m = P_1S_1 + P_2S_2 + \cdots\) 表示[[S58]]。

## 工艺作用

铸件主要壁厚是压铸工艺中具有特殊意义的参数，与整个工艺规范有着密切关系，直接影响以下核心环节[[S75]]：

- **填充时间计算**：壁厚与所需的最短填充时间存在明确的对应关系。随壁厚增大，所需填充时间相应延长[[S69,S54]]。
- **内浇口速度选择**：不同壁厚需要匹配相应的内浇口速度，以保证充型质量并避免裹气等缺陷[[S75]]。
- **凝固时间**：壁厚越大，冷却速率越低，凝固结束时铸件残留温度越高。局部的厚壁会延长整体凝固时间，进而影响脱模时间和生产节拍[[S16,S49]]。
- **模具温度梯度**：壁厚差异过大的铸件不同部位冷却速度不一致，形成不均匀热应力[[S24]]。
- **最终比压与留模时间**：壁厚直接决定持压周期和留模时间的长短[[S75]]。

充型时间与平均/主要壁厚的对应参考关系如下[[S54]]：

|:---:|:---:|
| 1.0 | 10～14 |
| 1.5 | 14～20 |
| 2.0 | 18～26 |
| 2.5 | 22～32 |
| 3.0 | 28～40 |
| 3.5 | 34～50 |
| 4.0 | 40～60 |
| 5.0 | 48～72 |
| 6.0 | 56～84 |
| 7.0 | 66～100 |
| 8.0 | 76～116 |

不同合金的充型时间经验公式如下：铝合金 \(t_f=40H\)（H为平均壁厚mm，单位ms）；高表面质量镀铬锌合金 \(t_f=6H+7\)；非镀铬普通锌合金 \(t_f=7H+13\)[[S25]]。

## 设计规范与限制

### 推荐壁厚范围

不同压铸合金的推荐壁厚范围存在差异。行业通用的中小型压铸件推荐合理主要壁厚范围如下[[S76]]：

| 合金种类 | 推荐壁厚范围（mm） | 常规最大壁厚限制 |
|:---:|:---:|:---:|
| 锌合金 | 1～3 | ≤4.5~6 |
| 铝合金 | 1.5～4 | ≤4.5~6 |
| 镁合金 | 1.5～4 | ≤4.5~6 |
| 铜合金 | 2～4 | ≤4.5~6 |

### 最小壁厚极限

在优良工艺条件下，不同压铸合金可实现的最小壁厚极限如下[[S75,S63,S79]]：

| 合金种类 | 最小极限壁厚（mm） |
|:---:|:---:|
| 锌合金 | 0.5～0.6 |
| 铝合金 | 0.7 |
| 镁合金 | 0.8 |
| 铜合金 | 1.0 |

壁厚设计值过小，超出合金流动性支持极限时会产生充型不良、欠铸、冷隔等缺陷；壁厚设计过大则会增加内部缩孔、气孔生成概率，降低力学性能和气密性[[S48]]。普通压铸场景下，壁厚超过6mm的零件不宜采用压铸工艺；有特殊使用要求的厚壁压铸件最高可放宽至12mm[[S30,S75]]。

### 等壁厚设计原则

在满足铸件强度和刚度要求前提下，压铸件应尽可能设计为均匀薄壁，避免局部肥大厚壁结构，以实现冷却过程同步，消除热节、降低内部气孔缩松缺陷率，同时缩短生产周期提升效率[[S38,S60,S78]]。

对于需要保证结构刚度的部位，不建议直接增大壁厚，可采用减薄主体壁厚并增设加强筋的结构替代。筋的厚度取相连基础壁厚的2/3～3/4；当铸件壁厚小于2mm时不宜设置加强筋，以避免憋气缺陷[[S38]]。

### 不等壁厚过渡与圆角设计

当铸件存在不等壁厚时，过渡连接需遵循以下设计规范：

**水平连接**：当壁厚比值 \(s_1/s_2\leqslant2\) 时，圆角 \(R\) 取0.2～0.25倍两壁厚之和；当 \(s_1/s_2>2\) 时，渐变过渡段长度 \(L\geqslant4\times(s_1-s_2)\)[[S76]]。

**直角拐角连接**[[S16,S29]]：

| 条件 | 内圆角设计规范 | 外圆角设计规范 |
|:---|:---|:---|
| 等壁厚 \(t\) | \(r = (0.5\sim1)\,t\) | \(R = r + t\) |
| 不等壁厚 \(t_1>t\) | \(r\geqslant\frac{1}{3}(t+t_1)\) | \(R = r + \frac{1}{2}(t+t_1)\) |

## 质量缺陷机制

### 厚壁区域缺陷

壁厚增加会导致冷却速率下降，二次枝晶臂间距增大、晶粒尺寸上升，缩松缺陷体积分数随壁厚增大显著升高，铸件承载面积缩减、力学性能和气密性出现反向下降[[S12]]。

**缩孔与缩松**：壁厚差异较大的压铸件凝固时，薄壁区域优先完成凝固，厚壁区域最后冷却形成孤立液相区，补缩通道提前中断、增压压力无法传递，最终在热节位置生成缩孔/缩松缺陷[[S9,S44]]。依据GB/T 5611-2017《铸造术语》，缩孔形状极不规则、孔壁粗糙并带有枝状晶，常出现在铸件最后凝固的厚壁区域；缩松为分散而细小的缩孔，存在缩松缺陷的铸件在气密性试验时易渗漏[[S68]]。

**气孔**：压铸充型过程中卷入型腔的气体无法在薄壁区域快速凝固阶段完全逸出，最终滞留在凝固时间长的厚壁区域内部，形成内壁光滑圆润的气孔；金属液中溶解的气体在凝固时溶解度大幅下降而析出，富集在厚壁最后凝固的区域[[S10,S53]]。

影响厚壁区域缩孔、缩松的主要因素包括[[S72,S40]]：
- **合金性质**：液态和凝固收缩量越大、凝固温度区间越宽，缺陷倾向越高。
- **铸件结构**：壁厚差越大、孤立厚壁热节越多，缺陷越易产生。
- **浇注系统设计**：补缩通道被薄壁阻断、压力传递路径不畅时缺陷显著加剧。
- **压铸工艺参数**：浇注温度过高、增压比压不足、持压时间过短时缺陷体积显著增大。

### 薄壁区域缺陷

**冷隔**：当两支或多股金属液在薄壁区域汇流时，汇流前的金属液前端表面已提前半凝固并形成致密氧化膜，两股已冷却的金属液无法完全熔合，形成带有圆滑边缘的缝隙，通常出现在远离内浇口的薄壁汇流位置[[S13,S22,S37]]。

**浇不足**：当薄壁区域的厚度低于对应合金的可充型极限时，金属液流经该区域时散热速度远高于流动速度，未完全充满型腔即提前完全凝固，在边角、薄壁远端位置出现轮廓缺损[[S34,S43]]。

冷隔和浇不足的发生概率随壁厚减小显著提升，同时随薄壁区域的流通长度增加进一步加剧[[S63]]。

## 标准与规范

### 国际标准

**ISO 8062-3:2008**规定了铸件的尺寸偏差和壁厚公差等级（DCTG），对疲劳性能要求高的铸件仅允许采用DCTG 11及以上壁厚公差等级[[S67]]。

**ASTM B85/B85M-08**配套设计规范参考美国压铸协会（NADCA）产品标准：小体积（约5 in³量级）铝合金压铸件最小壁厚可达0.09 in（约2.3mm）；金属流动条件差时壁厚推荐取0.15～0.18 in（约3.8～4.6mm）；体积超过200 in³的大型压铸件设计壁厚不应小于3/16 in（约4.8mm），同时要求压铸件壁厚尽可能保持均匀[[S47,S32,S45]]。

### 中国标准

**GB/T 6414-1999《铸件 尺寸公差与机械加工余量》**等效采用ISO 8062:1994，明确规定铸件的壁厚类尺寸公差需在普通同尺寸公差等级基础上粗一级，例如图样标注普通公差为CT6时，壁厚的允许公差等级为CT7[[S66,S28,S62]]。

### 日本标准

日本JIS压铸配套标准体系中，JIS H5302（铝合金压铸件）、JIS H5303（镁合金压铸件）的壁厚尺寸未设置单独强制限定，壁厚的未注公差需符合JIS B0409《普通压铸件尺寸公差》。公开的JIS压铸壁厚标准及加工裕度表按铸件材质、外形尺寸分类明确了最小壁厚要求和对应加工余量数值[[S7,S74]]。

## 与材料性能的关系

### 合金流动性的影响

合金流动性是决定压铸可设计最小主要壁厚的核心约束条件。流动性越好的压铸合金，可支持的主要壁厚设计下限越低：锌合金流动性优于铝合金，因此锌合金压铸件的合理主要壁厚下限可设计到比铝合金更薄的数值[[S32,S77]]。

薄壁压铸件的合金流动路径上冷却速度快，要求合金具备更长的充型流动寿命和更高的压射比压，才能保证完整充型[[S73]]。

### 收缩率与壁厚的关系

压铸行业通用规律：在相同合金、相同工艺条件下，薄壁压铸件的收缩率较小，厚壁压铸件的收缩率更大[[S21]]。壁厚越大，冷却速率越低，脱模时铸件与室温的温差越大，最终铸件的总收缩率随之增大[[S49]]。

不同压铸合金收缩率随壁厚增大的增幅排序为：铜合金 > 镁合金 > 铝合金 > 锌合金[[S3]]。

以有型芯约束条件下的锌合金压铸件为例，不同主要壁厚对应参考线收缩率如下[[S55]]：

| 铸件壁厚（mm） | 线收缩率参考区间 |
|:---:|:---:|
| 1～2 | 0.4%～0.5% |
| 2～3 | 0.5%～0.6% |
| 3～4 | 0.6%～0.7% |

### 力学性能与壁厚的关系

压铸件的力学性能随壁厚增加而下降。锌合金、铝合金两类压铸合金的强度均随壁厚增大（2.0～6.0mm区间）呈下降趋势，其中铝合金强度降幅大于锌合金[[S31]]。
