## 概述

铸造用球形陶瓷砂（Spherical ceramic sand used for foundry）是以Al₂O₃-SiO₂为主要矿物成分、粒径覆盖0.020 mm～3.350 mm的耐火颗粒物，由高铝矿石（铝矾土）经电弧炉高温熔炼、高压空气喷制冷却工艺制成的球形颗粒[[S12,S34]]。该材料在GB/T 5611—2017《铸造术语》中被给予正式定义，适用于铸造生产中的造型、制芯工序[[S12]]。其商品别名包括电熔陶粒、宝珠砂、钰珠砂、珠宝砂等[[S12,S36]]。

铸造用球形陶瓷砂的核心用途是替代天然硅砂、锆英砂作为造型制芯用砂，可解决硅砂生产中出现的表面粘砂、夹砂、烧结和落砂困难等问题，适配铸铁、碳钢、合金钢、有色金属等各类铸件生产[[S36,S8]]。由于颗粒呈球形、耐火度高、热膨胀率低，该材料能显著提升旧砂再生利用率、降低废砂排放量，属于低排放绿色铸造材料[[S36,S9]]。

## 分类与主要牌号

铸造用球形陶瓷砂按制备工艺可分为两大类[[S15]]：

- **电熔型（熔融陶瓷砂）**：以铝矾土为原料，在电弧炉中2000 ℃以上熔融后经高压气流喷吹冷却制得球形颗粒，商品名“宝珠砂”即属此类[[S12,S2]]。
- **烧结型（人工烧结陶瓷砂）**：通过喷雾造粒或滚粒造粒方法将粉料成球后送入高温窑炉烧结成型，在抗冲击性、耐磨性和热稳定性方面表现更优[[S15]]。

按矿物组成，最常见的产品子类为**铝矾土质**和**莫来石质**（主晶相为3Al₂O₃·2SiO₂）[[S14,S36]]。

主流产品粒度集中分布在50目、70目、100目三个筛段，85%以上颗粒落入该区间，对应AFS平均细度典型值在50～56范围内，可根据需求定制任意粒度规格[[S29,S36]]。产品规格涵盖10～14目、20～30目、30目、40～70目、100～140目、170目等多个粒度段[[S21,S27]]。

## 化学成分与物相组成

典型球形陶瓷砂的化学成分范围如下[[S24,S14,S8]]：

| 成分 | 含量（质量分数） |
|------|------------------|
| Al₂O₃ | 65%～75%（部分高铝款≥75%） |
| SiO₂ | 20%～28%（部分高铝款5%～20%） |
| Fe₂O₃ | ≤5% |
| TiO₂ | ≤5% |
| CaO、MgO、K₂O、Na₂O等 | 总占比<5% |

主矿物相为莫来石（3Al₂O₃·2SiO₂），伴随少量玻璃相；部分Al₂O₃含量较高的产品还含少量刚玉相（α-Al₂O₃）[[S14,S8]]。

## 物理与工艺性能

球形陶瓷砂的典型物理与工艺性能参数汇总如下[[S2,S24,S8,S19]]：

| 性能指标 | 典型值 |
|----------|--------|
| 角形系数 | ≤1.1 |
| 球度/圆度 | >95% |
| 堆积密度 | 1.9～2.0 g/cm³（部分1.95～2.05 g/cm³） |
| 耐火度 | ≥1600 ℃，部分高铝款可达1900～2000 ℃ |
| 热膨胀系数（20～1000 ℃） | 6×10⁻⁶/℃ |
| 1000 ℃热膨胀率 | 约0.13%（加热10 min下0.13%～0.7%） |
| 热导率 | 约5 W/(m·K) |
| 透气率 | >180 |
| 酸耗值 | 约1.5 mL |
| pH值 | 7.6 |
| 发气量 | 1.81 mL/g |
| 抗压强度 | >65 MPa |

球形陶瓷砂的角形系数远低于普通天然硅砂，铺粉密度比不规则硅砂提高21.3%，流动充型能力优异[[S2,S11]]。实测流动时间为14 s，优于高硅砂的15 s[[S22,S29]]。

在力学性能方面，无论烘烤前后，球形陶瓷砂制作的砂型抗拉强度均显著高于普通高硅砂和再生砂[[S7]]。

## 制备工艺

球形陶瓷砂的制备路线与天然硅砂从原矿直接破碎筛分的工艺存在本质区别，主要有三种方法[[S15,S36,S12]]：

1. **熔融吹制法**：将高铝矿石（铝矾土）送入2000 ℃以上电弧炉熔融，熔融液流出时用高压压缩空气喷吹冷却，直接得到球形颗粒。这是市面上“宝珠砂”的主要生产工艺[[S12,S2]]。
2. **喷雾造粒法**：将粉料浆体经喷雾干燥造粒后再烧结成型[[S15]]。
3. **滚粒烧结法**：将粉料通过机械滚粒成球后送入高温窑炉烧结成型[[S15]]。

不同工艺产品存在性能差异：熔融陶瓷砂密度更高，铸造过程中砂耗量和发气量相对较大；烧结陶瓷砂在抗冲击性、耐磨性和热稳定性方面表现更优。不同工艺产品不可直接等效替代混用[[S15,S36]]。

## 在铸造中的应用

### 作为型砂与芯砂

球形陶瓷砂可直接作为普通型砂用于造型工序，适配铸铁、碳钢、合金钢等各类材质铸件生产，解决普通硅砂生产中易出现的表面粘砂、夹砂、烧结、落砂困难等缺陷[[S36,S8]]。其低热膨胀特性（20～1000 ℃热膨胀系数仅为6×10⁻⁶/℃，总热膨胀率≤0.13%）可大幅降低铸件夹砂、结疤、鼠尾等膨胀类缺陷，显著提升铸件尺寸精度[[S2]]。

制备覆膜砂芯时，球形陶瓷砂表面光滑、球度高，可有效降低覆膜砂所需树脂添加量15%～20%，提升砂芯强度和透气性[[S11,S2]]。在低压铸造中，陶瓷砂覆膜砂芯的透气性优于普通硅砂覆膜砂，可有效抑制充型过程的气孔缺陷，适用于铝合金复杂腔体类铸件生产[[S18]]。在重力铸造中，陶瓷砂制备的砂型/砂芯强度高、热形变小，可大幅降低铸件裂纹缺陷[[S8]]。

### 作为涂料骨料

球形陶瓷砂的高耐火度特性使其可作为铸造涂料的耐火骨料，提升涂料抗粘砂效果[[S37]]。

### 粘结剂适应性

球形陶瓷砂化学性能稳定、耐酸碱侵蚀，pH值为7.6，酸耗值低，可兼容酸性、碱性全系列铸造粘结剂体系[[S25]]。相比普通硅砂，可减少树脂添加量15%～50%，水玻璃粘结剂添加量可降低至≤4%，适配呋喃树脂、碱性酚醛树脂、水玻璃、黏土砂等所有常规砂型铸造粘结剂工艺[[S25,S8,S2]]。同等强度下粘结剂用量可较硅砂降低15%～30%，既降低材料成本也减少浇注过程的发气量，降低气孔缺陷概率[[S8,S2]]。

### 旧砂再生与循环利用

球形陶瓷砂结构致密、耐破碎性优异，重复循环再生使用极少出现颗粒破碎，旧砂再生利用率可达90%以上甚至接近100%，废砂与粉尘排放量远低于普通硅砂[[S8,S24]]。再生工艺可采用干法、湿法、热法三类通用方案：干法成本最低，湿法更适配水玻璃粘结剂体系的残留钠杂质脱除，热法可高效烧除有机粘结剂残膜[[S1]]。相比普通破碎型人工砂，其再生后颗粒形貌更圆润，循环使用性能优于新砂[[S5]]。在不同循环次数下，补充少量新砂即可维持型砂性能稳定[[S23]]。

## 与其他铸造用砂的对比

### 与硅砂对比

普通硅砂主要成分为SiO₂，天然硅砂粒形多为近圆形或多角形，人工硅砂粒形多为尖角形[[S12]]。硅砂在573 ℃附近发生α→β石英相变，产生约1.6%的体积突变，是铸件夹砂、脉纹等膨胀类缺陷的主要诱因[[S4,S9]]。硅砂耐火度低于1700 ℃，高温下化学稳定性差，易与FeO作用产生铁橄榄石导致粘砂[[S37]]。球形陶瓷砂耐火度可达1900 ℃以上，热膨胀系数远低于硅砂，不含游离SiO₂，在环保性、铸件质量和旧砂再生方面均具明显优势[[S2,S8,S9]]。

### 与锆砂对比

锆砂主要成分为硅酸锆（ZrO₂·SiO₂），粒度细、热膨胀系数低、导热率高、耐火度可达2000 ℃，但全球储量有限、价格高昂，主要用于大型铸钢件局部面砂和精密铸造面层材料[[S12,S16]]。球形陶瓷砂在耐火度和高温性能上可与锆砂媲美，按体积计价仅为锆英砂的30%，是锆砂的经济型替代方案[[S27,S24]]。

### 与铬铁矿砂对比

铬铁矿砂主要矿物成分为FeO·Cr₂O₃，耐火度高于1790 ℃，具有优异的固相烧结特性和抗碱性熔渣性能，主要用于合金钢铸件和大型铸钢件的面砂与涂料骨料[[S12,S30]]。球形陶瓷砂按体积计价仅为铬铁矿砂的50%，且球形度高、流动性好，在多种应用场景下可作为替代材料[[S27,S25]]。

### 主要性能对比汇总

| 对比项目 | 球形陶瓷砂 | 普通硅砂 | 锆砂 | 铬铁矿砂 |
|----------|-----------|---------|------|---------|
| 主要成分 | Al₂O₃-SiO₂ | SiO₂ | ZrO₂·SiO₂ | FeO·Cr₂O₃ |
| 粒形 | 球形，角形系数≤1.1 | 多角形至近圆形 | 粒状 | 尖角形 |
| 耐火度 | 1600～2000 ℃ | <1700 ℃ | ~2000 ℃ | >1790 ℃ |
| 热膨胀系数 | 低（6×10⁻⁶/℃） | 高，相变膨胀约1.6% | 低 | — |
| 相对成本 | 中等（高于硅砂，低于锆砂/铬铁矿砂） | 低 | 高 | 较高 |

## 相关标准

截至2026年6月，国内尚未发布针对铸造用球形陶瓷砂的独立专项产品标准[[S31]]。其常规性能检验可直接沿用现有通用铸造原砂系列标准的对应方法[[S31,S20]]：

- **GB/T 2684**《铸造用砂及混合料试验方法》：用于粒度组成、含水量、含泥量、酸耗值等测试[[S3]]
- **GB/T 7143**《铸造用硅砂化学分析方法》：用于化学成分分析[[S3]]
- **GB/T 9442**《铸造用硅砂》：其中规定的角形因数测定方法可参照使用[[S3]]
- **GB/T 5611—2017**《铸造术语》：给出了铸造用球形陶瓷砂的正式定义[[S12]]

在ISO、ASTM、JIS标准体系中也未检索到针对该材料的独立专项规格或检验标准[[S31]]。

## 局限性与注意事项

1. **初始采购成本较高**：球形陶瓷砂的单价远高于普通天然硅砂，属于相对高价值材料，需通过再生循环使用平衡综合成本[[S27,S9]]。按体积计价，其成本为铬铁矿砂的约50%、锆英砂的约30%[[S27,S25]]。
2. **不同工艺产品不可混用**：熔融陶瓷砂密度较高，使用中砂耗量和发气量相对更大；烧结陶瓷砂抗冲击性、耐磨性和热稳定性更优。两种工艺产品性能存在显著差异，不可直接等效替代混用[[S15,S36]]。
3. **再生工艺条件需匹配**：熔融陶瓷砂再生后的颗粒形貌更为圆润，但不同粘结剂体系（有机树脂、水玻璃等）需配套相应的再生方案（干法、湿法或热法），以确保再生砂性能接近新砂水平[[S28,S5]]。
