## 定义与术语

**挤压销套（squeeze pin sleeve）**是配合挤压销运动的衬套，属于压铸局部加压系统的核心组成部件，国家标准 GB/T 8845‑2017 对其给出了明确定义，对应英文术语为 squeeze pin sleeve，也称 extrusion pin sleeve、local pressure pin bushing [[S7]]。挤压销套不具备独立执行挤压动作的功能，其用途专限于为挤压销提供精密的导向与密封，与仅用于常规型芯导向的普通型芯套在运动精度、配合间隙及材料要求上存在明确区分 [[S7,S6]]。

## 功能与工艺作用

挤压销套与挤压销组成运动副，在压铸充型完成、合金处于半固态阶段时，配合油缸推动挤压销对铸件厚大热节部位施加额外压力，实现强制补缩。其工艺作用包括：消除缩孔、缩松缺陷，提升铸件致密度，改善高气密性要求铸件的试压渗漏通过率 [[S1,S6,S13]]。

## 结构与组成

典型局部挤压结构由挤压油缸、挤压销、挤压销套及连接机构构成 [[S1]]。挤压销套上常设计集渣槽，以防止长期积存熔渣导致运动副卡滞 [[S11]]。

在退回状态下，挤压销配合段应高出挤压销套 1~2 mm，避免合模时金属液直接接触套内壁，可有效降低卡死风险 [[S14]]。生产中还推荐配合插芯喷涂工艺，每次开模时将挤压销顶出到位进行润滑与冷却 [[S2,S11]]。

主流采用液压油缸驱动（占比超过 90%），可通过独立伺服液压系统实现 0~25 MPa 区间内的油压连续可调；少量采用机械联动式，结构简单但参数调整灵活性较差 [[S11,S3]]。

## 材料与表面处理

挤压销套主要采用热作模具钢（常用牌号 H13）制造，出厂须进行氮化处理以提高表面硬度；在工况允许时，推荐追加 PVD 纳米镀钛工艺以进一步提升耐磨性，延长使用寿命 [[S14,S11]]。

## 关键设计参数

### 挤压销直径

铝合金压铸场景下推荐选用 8~30 mm。直径过小易断销，过大则需配置大吨位挤压油缸并占用过多的模具安装空间 [[S13,S8]]。

### 配合间隙

挤压销与挤压销套的配合间隙直接影响运动稳定性与寿命，推荐规范如下：

| 工况 / 公称尺寸 | 推荐配合间隙 | 参考公差带 |
| :--- | :--- | :--- |
| 不通冷却水 | 0.04~0.06 mm | H8/e7 |
| 通冷却水 | 0.02~0.04 mm | H8/e7 |
| 斜向布置、公称 φ15 mm | 0.05~0.11 mm | H8/e7 |

表格依据 [[S14,S8,S2]] 编制。设计与装配时需控制公差，间隙过大易窜入铝液导致卡死跑水，间隙过小则热膨胀后易拉伤、磨损 [[S9,S14]]。

### 挤压行程

挤压销的预留运动行程可按公式估算：  
$L = \dfrac{n\beta W}{\pi R^{2}}$  
其中，L 为行程，n 为安全系数（通常取 3），β 为铝合金凝固体积收缩率（一般≤6.5%），W 为待补缩热节体积，R 为挤压销半径。挤压销套的有效导向长度需覆盖该全行程 [[S14]]。

## 工艺参数与控制

局部挤压的关键工艺参数包括开始时间、保持时间、压力与速度。各参数推荐范围如下：

| 参数 | 推荐值 / 说明 |
| :--- | :--- |
| 挤压开始时间 | 高速压射充型结束后 0.1~2 s [[S14]] |
| 挤压保持时间 | 持续 1.5~2 s 后挤压销退回 [[S14]] |
| 挤压压力 | 可通过独立伺服液压系统在 0~25 MPa 范围内调节，压力越大补缩效果越好 [[S11,S8]] |
| 挤压速度 | 由液压油流量控制，需根据实际工艺调试 [[S9]] |
| 喷涂与润滑 | 每次开模时挤压销顶出接受喷涂，销上可设置油槽涂润滑脂 [[S2]] |

表内参数源自 [[S14,S8,S11,S9]]。挤压动作必须在合金半固态阶段完成；开始过早无法起效，过晚金属已凝固则补缩能力消失甚至导致挤裂 [[S14,S8]]。

## 常见缺陷与失效模式

| 失效模式 | 原因 | 解决措施 |
| :--- | :--- | :--- |
| 卡死 / 卡滞 | 间隙过大窜入铝液；集渣槽缺失导致熔渣积聚 | 严格管控配合公差；增设集渣槽；挤出销退回时高出套端 1~2 mm [[S11,S14]] |
| 早期磨损 | 表面硬度不足；间隙过小热膨胀拉伤 | 采用氮化 + PVD 镀钛；定期检测尺寸，预防性更换 [[S9,S14]] |
| 挤压不足（补缩不充分） | 挤压开始过晚；油缸压力不足 | 调试挤压延时与压力，确保在 0.1~2 s 内启动 [[S14]] |
| 配合间隙不当导致披锋或卡铝 | 间隙超差或装配偏差 | 定期检测挤压销与销套尺寸，执行预防性维护 [[S9]] |

通过上述管控，可将挤压销相关故障率降低至约 0.3%，由其引发的铸件质量缺陷占比控制在 0.12% [[S9]]。

## 应用场景

挤压销套配合挤压销主要应用于压铸件中厚大截面、热节处及对气密性有严格要求的部位，尤其适用于铝合金、镁合金、锌合金压铸件 [[S6,S13]]。铸造中通常在远离浇口、排气槽的厚壁区域设置一处到数处（一般建议不超过三处）局部挤压机构，以控制模具复杂程度 [[S13]]。

## 相关概念

- **挤压销（squeeze pin）**：在充型完成后挤入制件进行局部加压的零件，与挤压销套组成精密运动副 [[S7]]。  
- **局部挤压（local extrusion）**：通过挤压销对制件局部施加压力以增加致密度的方法 [[S7]]。  
- **挤压油缸 / 连接机构**：提供挤压驱动力并传递至挤压销，需与挤压销套的安装位置和导向精度匹配 [[S1,S9]]。  
- **真空系统**：与局部挤压协同使用可进一步降低铸件气孔率，但挤压销套的设计本身需保证密封，避免抽真空时由配合间隙吸入气体。
