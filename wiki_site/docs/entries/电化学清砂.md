## 定义

电化学清砂，亦称电化学清理、电解清理，属铸件后处理湿法清理大类下的一种化学清砂通电强化工艺 [[S1,S2,S7,S15]]。其核心定义为：利用碱性溶（熔）液通电处理以清除铸件表面粘砂的方法，其原理是碱性液电解时在铸件表面析出的初生钠能够还原氧化物 [[S15,S2]]。该工艺主要用于清理复杂铸件、铸件的复杂内腔和孔道以及需热处理的中大铸件 [[S15,S2,S1]]。

## 工艺归属与边界

电化学清砂在技术分类中具有明确边界，不属于以下两者 [[S15,S2]]：

- **电解抛光**：以阳极选择性溶解来抛亮金属表面为目标；
- **电化学去毛刺**：以去除加工边缘多余金属毛刺为核心目标。

电化学清砂与上述工艺的不同之处在于：其**核心作用是清除铸件附着的SiO₂基残砂、粘砂与附着氧化皮**，是独立的湿法化学清理衍生工艺 [[S15,S2,S25,S24]]。该工艺是在传统碱煮清砂装置基础上，通过加装直流供电系统而实现通电场强化清理效率的升级方法 [[S5,S4]]。

## 工作原理

### 电极接线与基本反应

常规电化学清砂采用**铸件接直流负极、电解液容器/坩埚接正极**的接线方式 [[S10,S28,S4]]。直流电场作用下，电解液中的 Na⁺ 定向向作为阴极的铸件表面迁移富集，在阴极俘获电子析出强还原性的初生金属钠：

> Na⁺ + e⁻ → Na  [[S10,S28,S17]]

生成的金属钠化学活性极高，直接还原铸件表面的SiO₂残砂及FeO、Fe₂O₃、Fe₃O₄等氧化皮：

> SiO₂ + 4Na → 2Na₂O + Si  [[S10,S28,S17]]
>
> FeO + 2Na → Na₂O + Fe  [[S10,S29,S17]]
>
> Fe₂O₃ + 6Na → 3Na₂O + 2Fe  [[S10,S29,S17]]

这些反应破坏了杂质与铸件基体之间的结合层，使残砂和氧化皮脱落 [[S10,S28]]。

### 阴极析气的机械剥离作用

通电过程中，阴极区域水电离出的 H⁺ 得到电子生成氢气，析出的大量氢气泡对附着在铸件表面的残砂、氧化皮起到机械剥离作用，进一步强化清砂效率 [[S10,S28,S22]]。

### 阳极反应

阳极区域 OH⁻ 因电极电位低于 Cl⁻ 而优先失去电子，生成氧气析出：

> 4OH⁻ − 4e⁻ → 2H₂O + O₂↑  [[S10,S28,S29]]

生成的氧气泡同样辅助剥离电解液侧的浮渣与杂质 [[S10,S28]]。

### 化学溶解作用

在通电前及通电过程中，熔融碱液与铸件表面粘砂发生的化学反应同样参与清理：SiO₂ 与 NaOH 反应生成硅酸钠（Na₂O·SiO₂）沉于炉底渣盘中，被定期清除 [[S28,S10]]。

## 工艺分类与电解液体系

电化学清砂按所用电解液类型可分为不同体系，各体系的处理温度区间不同 [[S28,S3]]：

| 电解液体系 | 典型成分 | 适用温度区间 |
|:---|:---|:---|
| NaOH 体系 | 100% NaOH 或 85～90% NaOH + 10～15% NaCl | 400～500 ℃ [[S28,S3,S10]] |
| Na₂CO₃ 体系 | 工业纯碱 + 工业纯盐 | 950～1000 ℃ [[S28,S3]] |
| 易熔渣体系 | 易熔渣 | 1150～1250 ℃ [[S28,S3]] |

### 高温熔融盐浴体系（主流工艺）

高温熔融盐浴型电化学清砂最常用的电解液质量配比为 **85% 工业纯碱（NaOH）搭配 15% 工业纯盐（NaCl）** [[S10,S14,S28]]。该配比下的熔融态电解液在对应温度区间可同时保障对 SiO₂ 的化学反应效率与电解过程的离子导电能力 [[S10,S14]]。

### 酸性电解液体系

以酸性电解液为工作介质的电化学清砂/除氧化皮工艺采用阳极浸蚀机制，依靠金属基体阳极溶解配合阳极析出氧气的机械剥离作用去除钢铁铸件表面厚层致密氧化皮 [[S22,S8]]。该体系清砂效率高，但需严格控制参数以避免基体过腐蚀和氢脆问题，多作为预处理工序搭配缓蚀剂使用 [[S22]]。

### 中性电解液体系

以添加少量络合剂（如葡萄糖酸钠）的氯化钠水溶液为典型工作介质的中性电解液工艺，腐蚀性低，不会对有色合金压铸件的精密尺寸精度造成明显损伤，适用于锌、铝类压铸件的温和清砂与去氧化皮场景 [[S9,S13]]。

## 典型工艺参数

### 高温熔融盐浴型

| 参数项 | 取值范围 | 来源 |
|:---|:---|:---|
| 工作电压 | 4～12 V（常见 4～10 V，部分工况 3～6 V） | [[S29,S5,S4,S28]] |
| 电流密度 | 5～12 A/dm²（按铸件表面积计算） | [[S29,S5,S4]] |
| 电流强度 | 5～10 A/cm²（铸件表面积） | [[S4,S5]] |
| 电源总输出 | ≥1000 A | [[S28,S4]] |
| 电解液配比 | 85% NaOH + 15% NaCl | [[S10,S14,S28]] |
| 处理温度 | 400～500 ℃（NaOH 体系） | [[S28,S3,S10]] |
| 铸件与坩埚壁间距 | ≥50 mm | [[S29]] |

> **来源差异说明**：电流密度的单位表述在不同来源中有差异——S29 和 S28 以 "A/dm²" 给出（5～12 A/dm²），而 S4 和 S5 以 "A/cm²" 给出（5～10 A/cm²）。在实际应用中需注意区分并按铸件有效表面积合理取值。

### 锌合金压铸件低温碱性阳极工艺

锌合金压铸件适配的低温碱性阳极电化学清砂参数不同于高温熔融盐浴工艺 [[S30,S26]]：

| 参数项 | 取值范围 |
|:---|:---|
| 电流密度 | 1.6～3.2 A/dm² |
| 处理周期 | 25～45 s |
| 电解液成分 | 混合碱性盐（磷酸三钠、偏硅酸钠为主）30～40 g/L + 表面活性剂 0.5 g/L |
| NaOH 含量 | ≤0.5 g/L |
| 电解液温度 | 60～70 ℃ |

该工艺采用阳极清洗方式优先于阴极清洗，可避免锌合金压铸件过蚀刻 [[S30,S26]]。

## 工艺流程

电化学清砂的标准全工序流程为 [[S28,S27,S3]]：

1. **铸件装框**（前处理）
2. **电化清砂**（核心工序：铸件浸入熔融电解液，接通直流电进行电解清理）
3. **冷水清洗**
4. **70～90 ℃ 热水清洗**
5. **取出铸件吹干**（干燥工序）

该工艺可同时实现铸件外表面与内腔的残砂清理，清理耗时短，铸件最终表面光洁度高 [[S27,S28]]。

对于锌合金压铸件的碱性电化学清砂，后冲洗操作要求更为严格：清砂结束后需依次经过温水清洗、空气搅拌式冷水清洗、水雾喷淋冲洗工序，以充分置换盲孔、深槽、凹陷缝隙内残留的碱性电解液；冲洗用水应优先选用低矿化度水源，避免后续电镀工序出现鼓包缺陷 [[S30,S26]]。

## 设备与工装

整套电化学清砂生产线的核心部件是在传统碱煮清砂装置基础上新增配套直流供电系统形成 [[S5,S4,S28]]，主要包括：

- 带加热自动控温功能的电解槽（坩埚） [[S5,S16,S4]]
- 专用低电压大电流直流电源（直流电机或整流器） [[S5,S4]]
- 导电铸件装夹料斗/料框 [[S16,S5]]
- 顶部密封抽风排风系统（用于收集苛性钠蒸汽） [[S6,S4,S3]]
- 沉渣收集盘 [[S16,S28]]
- 配套高碱废水预处理设施 [[S5,S4]]

## 工艺优势与局限性

### 优势

- **对复杂结构的适配性优异**：可完全清除常规机械清理无法到达的狭小区域内残留的残砂、粘砂与氧化皮，尤其适用于带砂芯的复杂内腔、盲孔、深槽类铸件 [[S4,S15,S10]]
- **清砂质量高，粉尘少**：清砂质量优于传统干法清砂工艺，作业过程粉尘污染低 [[S10,S14,S17]]
- **效率高于单纯碱煮工艺**：借助电场对 Na⁺ 的定向富集效应，清砂效率比不加电场的碱煮清理更高 [[S17,S14,S4]]

### 局限性

- **原料成本高**：每吨铸件需消耗约 40 kg 苛性钠 [[S28,S17,S3]]
- **设备腐蚀性强**：高温高浓度碱性介质对设备腐蚀作用强 [[S28,S17,S3]]
- **苛性钠蒸汽危害**：挥发蒸汽具有强刺激性，须采取严密防护和排风措施 [[S28,S3,S4]]
- **氢脆风险**：阴极析氢过程中，氢气可渗透进入高强度钢材质铸件，存在引发氢脆失效的风险 [[S28,S17,S23]]
- **可能发生局部过腐蚀**：高温高浓度碱性介质对铸件表面有腐蚀影响，需严格控制工艺参数 [[S28,S29]]

## 质量控制与检测

### 清砂效果的通用质量评价方法 [[S12,S18]]

| 检测方法 | 操作说明 |
|:---|:---|
| 目视检查法 | 以肉眼或低倍放大镜观察铸件表面，要求无残留粘砂、氧化皮，露出完整金属本色，无可见过腐蚀凹坑 [[S12]] |
| 水膜破裂法 | 铸件经最终漂洗后观察表面连续水膜状态，约 30 s 内不出现水膜破裂或局部疏水斑块则判定为清洁度合格 [[S18]] |
| 内窥镜（管道镜）检测法 | 将内窥镜探入铸件深孔、盲孔、复杂内腔内部，检查隐蔽部位残砂残留情况 [[S12]] |
| 重量损失法 | 对已知初始重量的铸件进行清砂前后称重，计算残砂去除率作为量化判定指标 [[S12]] |

### 工艺参数日常监控要求

生产过程中须定期检测电解液中苛性钠和 NaCl 的浓度，控制槽液温度在工艺设定区间，实时监控输出电流与电压的稳定性，并做好每班参数运行记录 [[S11]]。

## 安全与环保

### 作业安全

- 电解液高温挥发产生的苛性钠蒸汽具有强刺激性和腐蚀性，必须在槽体上方设置**防护罩**与**密闭抽风装置** [[S4,S3,S6]]
- 作业人员须穿戴耐碱防护服、防护手套、护目镜等个体防护用品 [[S4]]
- 作业区域须配备应急冲淋装置 [[S4]]
- 严禁将水带入坩埚，以免引起高温碱液喷溅 [[S29]]
- 苛性钠加热须在坩埚侧壁进行，不得在坩埚底部加热 [[S29]]

### 废液处理

电化学清砂产生的碱性废液的合规处理方法包括 [[S19,S20]]：

- **中和沉淀工艺**：投加酸性中和剂将 pH 值调整至排放标准区间，添加凝集剂去除悬浮硅酸钠沉淀；
- **电渗析工艺**：对废液中的碱组分进行回收，处理后上清液可部分循环回用。

沉淀污泥须经干化后按危险废物要求合规处置，符合国内金属表面处理行业废水排放相关环保规范 [[S19,S20]]。
