## 概述

密筋炉胆是热风冲天炉的环形内热式空气预热器，国家标准《铸造术语》（GB/T 5611—2017）将其定义为“热风冲天炉的环形内热式空气预热器”，炉胆内密布散热筋片以提高换热效率。装在预热带的炉胆可将鼓风预热到150 ℃~200 ℃，安装在烟囱内的炉胆则能将鼓风预热到300 ℃~400 ℃[[S6]]。该部件在中文领域常被称为密筋式热风炉胆、带肋炉胆或加强筋铸铁炉衬[[S2,S12]]。其标准英文术语为 **ribbed preheating jacket**[[S6]]。

密筋炉胆属于冲天炉熔炼设备的部件，主要安装于熔化带以上的炉身段或烟囱段，利用上升的高温炉气预热送入炉内的冷空气，以提升铁水温度、降低焦炭消耗，是国内中小型热风冲天炉中应用最广泛的内热式换热装置[[S1,S12]]。

## 工作原理

密筋炉胆由两个同心圆筒构成的夹层结构组成，冷空气从夹层通过。炉内高温炉气通过对流和辐射将热量传递给内胆壁，再由内胆外侧密布的散热筋片将热量传递给夹层中的空气，预热后的空气经热风集风箱送入冲天炉风箱参与熔炼[[S1,S12]]。实际操作中，正常风温可达200 ℃以上，加料口废气温度可降至200 ℃以下[[S9]]。

## 分类

密筋炉胆根据材料、筋片形貌和制造工艺可按以下维度分类[[S1,S12,S13]]：

| 分类维度 | 类别 | 说明 |
|----------|------|------|
| 材料类型 | 钢板焊接型 | 内胆和筋片多用锅炉钢板，外胆用普通钢板，焊接成形 |
|          | 耐热铸铁整铸型 | 采用RTCr系列耐热铸铁整体铸造，筋片与内胆一次铸出 |
| 筋片形貌 | 径向直筋型 | 筋片沿圆周径向分布，为最常见形式 |
|          | 45°倾斜筋型 | 筋片与内胆呈45°倾斜布置，目的在于进一步增加传热面积 |
| 成型工艺 | 焊接组装型 | 钢板下料后逐片焊接于内胆 |
|          | 整体铸造型 | 采用铸造工艺将筋片和炉胆一体成型 |

## 结构组成与材料

常规钢板焊接型密筋炉胆采用双同心圆筒夹层结构[[S7]]：

- **内胆**：使用12~14 mm厚的20G锅炉钢板制作，上下端常制成圆弧翻边，以减少转角处焊缝开裂风险。
- **外胆**：使用5~6 mm厚的普通钢板制作，构成夹层的外壁。
- **散热筋片**：在内胆外侧密排焊接4 mm厚的筋片，中型炉子的圆周筋片间距为9~12 mm，筋片顶端与外胆内壁之间预留2 mm的径向膨胀间隙。
- **夹层间隙**：内外胆之间的空气流通间隙根据炉径和送风量确定，中型炉子一般为27~37 mm。
- **焊缝**：筋片与内胆须焊牢焊透，常用含铬、钼、钒的低合金焊条，以保证焊缝在较高温度下的强度[[S10]]。

## 安装与膨胀补偿

密筋炉胆安装时需重点考虑热膨胀补偿与承载方式[[S7,S10,S11]]：

- **安装位置**：炉胆下缘一般设置在熔化带以上200~300 mm处，或底焦顶面以上300~500 mm处。位置越靠近熔化带，热风温度越高，但炉胆使用寿命越短。
- **膨胀节**：炉胆上端必须设置膨胀节（补偿器），以吸收内胆在高温下的轴向伸长。炉胆以上的冲天炉炉体重量应支撑在独立平台结构上，严禁压在炉胆壳体上，保证内胆自由膨胀。
- **保护圈**：为防止高温炉气直接冲刷膨胀节，通常在炉胆上口设置保护圈，保护圈与内胆间的间隙应尽量小但不得接触，以避免阻碍膨胀。
- **下端密封**：炉胆下端采用耐火泥密封，炉胆外侧与炉壳内壁之间的空隙填充绝热耐火材料，减少散热损失。
- **装入方式**：炉胆可从加料口整体吊入，加料口尺寸需满足表4-4要求；或从炉身侧部装入，炉胆上端与炉子上段之间采用2 mm薄钢板膨胀节密封。

## 运行参数与操作要求

密筋炉胆的正常运行需控制以下温度和操作要点：

- **壁温限制**：设计最高允许工作壁温不超过550 ℃；正常运行时内胆下端壁温不宜超过350 ℃，上端壁温不宜超过500 ℃[[S10,S12]]。
- **热风温度**：稳定输出温度一般为150~250 ℃[[S12]]。
- **送风阻力**：密筋炉胆投入后，送风阻力增加200~300 mm水柱，配套风机必须选用高风压硬压式风机（如罗茨鼓风机）以保证风量[[S5,S9]]。
- **停炉操作**：熔炼快结束时须及时添加压炉铁，防止高温火焰大量上窜冲击炉胆；打炉后应继续鼓风对炉胆强制冷却约半小时，避免残留铁焦继续烘烤炉胆[[S10,S5]]。
- **使用寿命**：通常为1000~2500小时，与制造质量、安装精度和操作规范密切相关[[S4,S9]]。

## 常见失效形式

密筋炉胆在反复热循环和高温炉气冲蚀下的典型失效包括[[S10,S15]]：

- **热疲劳开裂**：筋片与内胆的焊接根部为高度应力集中区，在反复启停的热循环中，交变拉应力极易超过材料疲劳极限，导致焊缝开裂，严重时破坏密封层，造成风路短路失效[[S8,S10]]。
- **内胆烧穿**：长期受高温炉气冲刷，内胆局部减薄或穿孔。
- **鼓胀变形**：超温服役或支撑结构约束不当，造成内胆径向鼓胀。
- **氧化起皮与变形**：材料在高温下严重氧化，表面起皮，伴随整体刚度下降。
- **焊缝脱落与密封失效**：焊缝在高温和热应力作用下开裂或脱落，引起夹层气体串漏。

严格的操作规程、合理的膨胀补偿设计和定期检查是延长炉胆寿命的关键。

## 与普通平壁炉胆的对比

与无筋平壁热风炉胆相比，密筋炉胆在换热性能、结构刚性和寿命方面具有明显优势，但也带来更高的风阻要求，见表2。

**表2 密筋炉胆与普通平壁炉胆性能对比**

| 对比项目 | 密筋炉胆 | 普通平壁炉胆 | 数据来源 |
|----------|----------|--------------|----------|
| 热风温度 | 200 ℃以上（可达250 ℃） | 100~120 ℃ | [[S3,S12]] |
| 换热效率 | 换热面积大，系数高 | 换热面积小，系数低 | [[S3]] |
| 结构刚性 | 筋片显著提高整体刚性 | 无筋加强，易变形 | [[S12]] |
| 使用寿命 | 1000~2500 小时 | 通常低于800 小时 | [[S12,S5]] |
| 送风阻力 | 增加200~300 mm水柱 | 较小 | [[S5]] |
| 风机要求 | 需配高风压罗茨风机等 | 常规风机即可 | [[S1,S5]] |

## 应用场景

密筋炉胆主要应用于国内中小型热风冲天炉的炉身预热段或烟囱段，原一机部定型系列覆盖φ500~φ900 mm炉径，适配相应吨位的冲天炉[[S1,S11]]。在铸铁熔炼领域，该装置因经济实用、无需外加燃料、可有效提高铁水温度和降低焦耗而被广泛采用[[S1,S12]]。
