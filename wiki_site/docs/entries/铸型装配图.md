## 概述

铸型装配图（英文：mold assembly drawing）是表示合型后铸型各组元间装配关系的工艺图[[S11,S27]]。它是铸造工艺设计的核心输出文件之一，直观反映了铸造工艺方案的整体面貌，既是设计铸造工艺装备的重要依据，也是编制铸造工艺规程的主要基础[[S18]]。该图的作用覆盖从设计评审、模具制造、装配调试、维护修理直至质量检测的全生命周期[[S24]]。

需要特别关注的是，**铸型装配图并非仅用于压铸场景**——它是面向砂型铸造、金属型铸造、消失模铸造、压铸等全部铸造工艺的通用技术文件[[S10,S18]]。在中文铸造术语体系中，其与“压铸模总装图”“模具装配图”有着本质区分：前者面向铸型（砂型、金属型等）的合型装配，而后者面向压制用钢制金属模具的整机装配[[S7,S5]]。

## 定义与范畴

根据GB/T 5611-2017《铸造术语》的正式定义，铸型装配图是“表示合型后铸型各组元间装配关系的工艺图”，其注释明确所列内容包含：铸件浇注位置，型芯数量、序号、固定方式、下芯和抽芯顺序，浇冒口系统和冷铁布置，以及砂箱结构和尺寸等[[S11,S27]]。

在压铸特定语境下，行业用语中文常直接对应“压铸模装配图”，其对应的行业英文别名可见 **die assembly drawing**，与通用铸造语境下的 **mold assembly drawing**（美式拼写，为GB/T 5611标准规定英文术语）及**mould assembly drawing**（英式拼写，常见于英联邦和欧洲文献）并行使用[[S13,S21,S23,S30]]。

## 分类形式

根据文献记录，铸型装配图常用的表达形式大致可分为如下四类，可根据铸件复杂程度独立或组合应用[[S17,S26]]：

| 类别 | 说明 |
|------|------|
| 总装图 | 表达铸型各组元整体的完整装配关系与结构 |
| 部装图 | 针对某一局部组件（如特定砂芯组合、冷铁布排等）的装配详图 |
| 全剖/半剖剖面图 | 通过剖面显示内部型腔、浇注系统、砂芯定位等不可见结构的装配关系 |
| 三维爆炸图 | 以三维爆炸视角清晰示意组件拆卸/装配顺序与空间关系 |

## 核心内容要素

### 视图选择规则

铸型装配图的主副视图应优先选用铸件的自然浇注位置；画俯视图时通常将上箱揭开（如型腔结构简单、砂芯较少，为表达冒口布置也可不揭开或仅揭开1/2）；图面应尽量保持清晰，除主要轮廓线外避免或减少使用虚线[[S18,S22]]。

### 强制标注要素

依据系统化的铸造工艺设计规范，铸型装配图上必须表达的要素包括[[S18,S22]]：

| 标注类别 | 具体内容 |
|----------|----------|
| 铸型分型面 | 所有型腔组元间的接合面 |
| 浇注系统与冒口 | 结构、全部尺寸、过滤网规格与安放位置 |
| 砂芯标注 | 砂芯形状、相互位置、装配间隙、芯头大小与定位、排气方法，并按下芯顺序编号 |
| 冷铁 | 位置、数量、大小及编号 |
| 铸型检查尺寸 | 装配时需要检验的关键部位与尺寸 |
| 附铸试样 | 附铸试验块的位置与尺寸 |
| 砂箱内框尺寸 | 砂箱内框的长宽尺寸 |
| 专用砂箱结构（若适用） | 砂箱的导向、定位、锁紧装置 |

### 尺寸标注体系

铸型装配图的尺寸标注分为**配合尺寸、安装尺寸、外形尺寸**三大类，所有标注应满足后续合型工序检验的需求[[S3]]。配合关系需明确区分过盈配合、过渡配合、间隙配合三类公差参数，保证各铸型组件的装配精度[[S9,S14]]。

### 技术要求与标题栏/明细栏

技术要求栏至少应覆盖三方面内容：所适用的铸造偏差等级、表面粗糙度要求、零件热处理规范[[S7]]。标题栏与明细栏必须标齐零件序号、零件名称、材料牌号、使用数量、热处理要求等字段，以满足机械制图标准的信息追溯需求[[S7,S29]]。

### 与其他图样的衔接关系

在压铸图纸体系中，铸型装配图处于核心位置：向上承接压铸机接口图的参数约束要求，向下拆分出独立的浇注系统图、推出机构图、冷却系统图，并最终衍生出所有对应的模具零件图，形成完整的工艺图纸体系[[S4]]。

## 工艺生命周期中的应用

铸型装配图在整个铸造生产链条中覆盖以下5个核心场景[[S24]]：

| 阶段 | 功能 |
|------|------|
| 设计评审 | 校验铸造工艺方案的技术可行性与合理性 |
| 模具制造 | 作为加工及制造的主要指导依据 |
| 装配调试 | 指导现场合型操作与装配工序实施 |
| 维护修理 | 提供组件拆装修复的参考基准 |
| 质量检测 | 作为工序检验的判定基准 |

## 常用标准与规范

铸型装配图的绘制需遵从明确的标准化要求，优先层级如下[[S15,S28]]：

| 层级 | 标准 | 涉及内容 |
|------|------|----------|
| 第一层级（国内国标）| GB/T 5611 | 铸造术语定义（含“铸型装配图”正式术语）|
| 第一层级（国内国标）| GB/T 14689 | 图纸幅面与格式 |
| 第一层级（国内国标）| GB/T 4459系列 | 机械制图细节规范 |
| 第二层级（国际标准）| ISO 128 | 工程制图通用规范 |
| 第二层级（国际标准）| ISO 5455 | 制图布局与比例规范 |

## 设计与表达常见疏漏

根据设计禁忌文献的系统性梳理，铸型装配图绘制中常见的典型疏漏及规避要点可归纳如下[[S6,S18,S1,S16]]：

| 常见疏漏 | 现象与后果 | 规避要点 |
|----------|-------------|----------|
| **遗漏配合公差** | 未标注型芯与芯头配合间隙、活动抽芯机构的配合公差 | 仅标注装配必需的公差参数，非核心尺寸不做过度约束；公差取值须符合对应铸造精度标准 |
| **干涉检查缺失** | 冷却水道与顶出元件、型芯、镶块等产生空间干涉 | 提前完成全空间干涉检查；相关结构间最小安全壁厚不小于3mm；核验开模、合型全行程内无碰撞风险 |
| **定位基准缺标** | 未设定铸型装配基准角、定位销/定位孔、合型导向装置，易导致下芯错位、合型错箱 | 在图样中醒目标识所有装配基准信息，直接指导现场装配 |
| **技术要求表述模糊** | 未明确铸件收缩率、未注铸造圆角与拔模斜度、未注明验收标准 | 在附注栏清晰列明收缩率取值、未注圆角/拔模斜度规范、引用验收标准等必要信息 |
| **明细栏信息缺失** | 未完整标注砂芯、冷铁、定制工艺部件的序号、名称、材料、数量 | 完善明细栏内容；通用标准紧固件可省略详细信息但需标注标准选型代号 |
| **视图选择不规范** | 主视图未采用自然浇注位置，图中冗余虚线过多影响图面可读性 | 主副视图采用自然浇注位置；俯视图采用上箱揭开形式；精简冗余虚线 |
| **图样更改无追溯** | 更改后未标注更改标记、更改处数、更改文件编号及更改日期 | 严格按机械制图国家标准进行划改，在更改栏内完整填写所有追溯信息 |
