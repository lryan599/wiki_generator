## 概述

V法造型（V-process、vacuum molding），标准术语为真空密封造型（vacuum sealed molding），亦称负压造型、减压造型，其命名取自英文“Vacuum”的首字母“V”[[S38,S29,S35]]。该工艺于1971年由日本公开，属于一种无需化学黏结剂的物理造型法，被誉为第三代造型法[[S38,S25]]。其核心特征是不使用任何黏结剂，依靠塑料薄膜密封砂箱并抽真空，利用铸型内外的压力差使干砂紧实成型[[S18,S30]]。

V法造型属于砂型铸造范畴的造型工艺，与真空压铸存在本质区别：前者作用对象为无黏结剂干砂构成的砂质铸型，后者通过抽除压铸金属模具型腔内的空气来减少高压充型产生的铸件气孔缺陷，二者在工艺目标、作用对象和应用场景上完全不同[[S8,S28,S18]]。经检索国内外主流铸造术语标准、高校教材与行业手册，未发现将V法造型作为真空压铸简称的权威正式指代记录[[S16,S8,S28]]。

## 工艺原理

V法造型的基本原理是：在带有抽气腔的专用砂箱内充填不含黏结剂的干砂，铸型内腔表面和砂箱顶面均覆盖密封塑料薄膜，通过真空泵抽除型内空气形成负压状态。依靠铸型内外的压力差（约40 kPa）使干砂紧实，砂粒之间的摩擦力维持铸型结构稳定。在整个造型、合箱、浇注及凝固过程中持续保持负压，待铸件完全凝固后解除真空，干砂自动溃散，即可直接取出铸件[[S18,S30,S32]]。

## 工艺流程

V法造型完整的标准化工艺流程共包含9个核心步骤[[S12,S18,S1]]：

1. **模样固定**：将开设大量密集透气孔的模样固定于带抽气腔的中空型板上。
2. **薄膜加热软化**：将高延伸率的EVA塑料薄膜加热至80~120 °C（100~140 °C亦可）使其软化[[S12,S36]]。
3. **真空覆膜**：在模样侧抽26.7~53.4 kPa真空，使软化薄膜紧密贴附于模样表面，可按需喷刷快干耐火涂料。
4. **放置砂箱**：套上自带抽气腔的专用V法砂箱。
5. **填砂震实**：向砂箱内填入无黏结剂干砂，通过微振台震实使干砂达到高紧实度。
6. **刮平封背**：刮平砂层，放置浇口杯后在砂箱顶面覆盖背膜完成砂腔密封。
7. **起模**：对砂箱侧抽真空，解除模样侧真空后完成起模，分别制备出上、下两个独立铸型。
8. **合箱浇注**：下芯后合箱锁紧砂箱，在持续维持铸型负压的状态下完成金属液浇注。
9. **脱箱落砂**：待铸件完全凝固后解除砂箱负压，干砂自动溃散即可取出铸件。冷却后的旧砂可循环回用，砂回收率可达95%以上。

## 工艺参数

### 主要操作参数

V法造型的核心工艺参数涵盖真空度、薄膜处理、型砂选用等方面，具体数值范围如下表所示。

| 参数 | 推荐值／范围 | 说明 |
|---|---|---|
| 砂型工作真空度 | −40~−66.5 kPa（约−300~−500 mmHg） | 依靠铸型内外约40 kPa压差实现干砂紧实，不同工作阶段可配置不同真空度与抽气量[[S32,S15,S26]] |
| EVA面膜加热温度 | 80~140 °C（行业最优约100 °C） | 温度过高易烧穿薄膜，过低则薄膜无法充分延展贴附复杂模样轮廓[[S36,S12]] |
| EVA面膜厚度 | 0.05~0.1 mm（常规约0.15 mm） | 在深凹模样上成型时最终厚度可薄至约0.006 mm[[S25,S30,S13]] |
| 干砂粒度 | 70/140筛号或100~200目（AFS 150~350） | 黑色金属铸件常用AFS 20~50目砂，铝合金铸件常用AFS 50~100目砂[[S17,S15,S36,S30]] |
| 干砂含水量 | ≤0.5% | 含水量过高将严重影响砂的充填性能[[S17]] |
| 砂粒集中度 | ≥85% | 宜采用近圆形砂粒，避免粒度偏析[[S17]] |
| 旧砂回用温度 | ≤50 °C | 循环回用旧砂须冷却后使用[[S17,S36]] |
| 型砂总回收率 | ≥95% | 干砂不含黏结剂，落砂后可高效回用[[S18,S37,S25]] |
| 填砂振动加速度 | 1.2~1.5 g | 水平振动比垂直振动可获得更高的干砂填充密度[[S36]] |
| 模样通气孔参数 | 孔径φ0.2~2 mm，孔距20~50 mm | 深凹转角区域需加密通气孔[[S33,S32,S13]] |

## 关键设备

V法造型的配套设备分为核心主机和辅助系统两类[[S9,S10]]：

**核心主机：**
- 水环式真空泵组（抽气量大，泵前须装滤气罐防止粉尘进入）[[S9]]
- 薄膜烘烤器（优先采用远红外加热元件，亦可用火焰或电阻元件加热）[[S9]]
- 带抽气腔的双层壁V法专用砂箱[[S9]]
- V法振实台／可调式电磁振动器[[S9,S10]]

**辅助系统：**
- 滤气罐、真空分配罐、真空管路及控制阀组[[S9]]
- EVA薄膜覆膜工装[[S9]]
- 干砂冷却循环系统[[S9]]
- 除尘装置及真空抽气尾气处理设备[[S9,S29]]

可按需选配固定式、梭动式、两工位转台式、四工位转台式等不同构型的V法造型集成机组[[S9]]。

## 铸型特性与铸件质量

### 铸型性能

V法铸型无黏结剂、无水分，整体硬度均匀，在标准真空度条件下铸型表面硬度最高可达90，铸型硬度随工作真空度提升而上升[[S32,S18]]。起模阻力极小，脱模斜度可低至1/100~1/80（甚至0°~−1°）[[S18,S25,S12]]。铸型在高温金属液作用下不易变形，浇注时的真空环境有利于金属液平稳充填型腔[[S37,S12]]。

### 铸件质量特征

- **表面质量**：铸件表面粗糙度较普通砂型铸造提升约60%，轮廓清晰，毛刺少[[S25]]。
- **尺寸精度**：可浇注壁厚低至3 mm的薄壁铸件[[S18,S25,S37]]。
- **金属利用率**：由于加工余量大幅减小且补缩效率更高，铸钢件金属利用率可提高20%，铸铁件可提高25%[[S37,S12]]。

## 典型缺陷与控制

### 塌箱

塌箱是V法造型最严重的缺陷之一。其主要成因包括[[S17,S31]]：

1. 真空度不足或浇注过程中薄膜大面积烧失导致负压丢失。
2. 薄膜覆膜后残留弹性，浇注时薄膜破口快速扩大。
3. 充型速度过慢或断流，铸型暴露时间过长失去支撑。
4. 铸型紧实度不足，砂粒摩擦力不够。
5. 涂料高温强度不足，无法支撑裸露砂层。

**控制方法**：全程维持真空度≥0.04 MPa并配置稳压罐；选用延展性好、成型后无残留弹性的EVA薄膜并严格控制加热温度；采用底注式半封闭浇注系统，在40~120 s内快速完成充型且不得断流；调整振动参数保障全砂箱均匀紧实；选用耐火度与高温强度达标的专用V法涂料并彻底烘干[[S31]]。

### 砂眼与冲砂

主要成因包括填砂密度不足导致铸型局部强度偏低、金属液高速冲蚀裸露砂层、薄膜提前破损未被发现[[S17]]。控制方法为充分振动紧实以提升填砂密度，设计浇注系统将金属液冲击高度限制在200 mm以内，覆膜后逐一检查薄膜无破损后再喷刷涂料[[S17]]。

### 机械粘砂与脉纹

主要成因是原砂粒度过粗或出现粒度偏析，在真空负压作用下金属液渗入砂粒间隙。控制方法为选用70/140目或更细的合格原砂，定期补充细砂避免砂粒粗化，在薄膜表面均匀喷刷V法专用隔离涂料[[S17,S34]]。

## 优缺点对比

### 主要优势

- **设备投资低**：无需混砂机与黏结剂供给系统，造型系统总投资比传统黏土砂造型低约30%，总动力消耗仅为湿型砂造型的60%，劳动力投入可减少35%[[S37,S18,S9]]。
- **原材料节约**：无黏结剂干砂回收率≥95%，废砂处理量大幅降低[[S37,S24,S18]]。
- **工装寿命长**：模样受薄膜保护不直接接触砂粒，不受高压冲击、化学侵蚀和高温作用，磨损量极低；还可选用木材、石膏等低成本软质材料制作模样[[S37,S4,S18]]。
- **环保效益突出**：浇注阶段释放的气体大部分被真空泵直接抽走，车间空气污染水平远低于常规化学黏结剂砂工艺[[S37,S18]]。
- **铸件质量优良**：表面光洁、尺寸精度高、金属利用率高[[S37,S12]]。

### 主要局限

- **生产率受限**：小铸件造型生产率低于湿型砂工艺，实现全自动化流水线的配套难度较高[[S37,S9,S24]]。
- **几何适应性差**：受塑料薄膜延伸性的限制，难以生产形状特别复杂、深槽深度过大的高精度铸件[[S37,S4,S24]]。
- **全流程负压维持**：从造型、合箱、浇注直到铸件完全凝固都需要稳定维持铸型负压，过程管控要求高[[S37,S9]]。
- **制芯配套复杂**：V法配套制芯工艺复杂度较高，工业实际中通常直接采用传统制芯方法适配[[S37,S24]]。

| 对比项目 | V法造型 | 湿型砂 | 树脂自硬砂 |

| 设备投资 | 较黏土砂低约30%[[S18]] | 设备成熟、投资中等 | 原材料成本较高[[S11]] |
| 动力消耗 | 为湿型砂的60%[[S18,S37]] | 基准参照 | — |
| 旧砂回收率 | ≥95%[[S18,S37]] | 中等，需补充新砂 | 有刺激性气味，废砂再生难度较高[[S11]] |
| 铸件精度 | 高，表面粗糙度提升约60%[[S25]] | 中等，受水分汽化影响易出缺陷[[S3]] | 高，比干黏土砂高2~3级[[S11]] |
| 适用场景 | 大型扁平件、艺术铸件、结构较简单精密件[[S37,S9,S5]] | 中小型铸件大批量流水生产[[S3]] | 质量要求较高的中大件单件小批生产[[S11]] |

## 应用场景

V法造型适配的材质范围较广，可生产灰铸铁、球墨铸铁、铸钢等黑色金属铸件，也可用于铜、铝、镁等有色金属铸件[[S37,S9,S24]]。其生产模式兼顾手工操作的单件小批量生产与自动化生产线的大批量生产[[S37,S24]]。

该工艺优先适配结构简单、轮廓清晰的精密铸件，尤其适合生产薄壁、面积大、形状不太复杂的扁平铸件[[S5,S4]]。亦可利用石膏、黏土甚至树叶、贝壳等天然材料制作模样，用于制造表面细节丰富的艺术铸件及精美工艺品铸件[[S4,S13]]。国内已有成熟的应用案例覆盖军品铸件、出口高精度转动体铸件等特种铸造场景[[S20,S6]]。
