## 概述

浇注排溢系统（又称浇排系统、模具浇排系统，英文对应 gating and venting system 或 feed and vent system in HPDC）是压铸模具中由**浇注系统**和**排溢系统**共同组成的整体功能单元。浇注系统指在压力作用下引导熔融金属充填型腔的通道，常规由直浇道、横浇道、内浇口、余料组成；排溢系统指容纳冷污合金、排出型腔内气体的结构单元，由溢流槽和排气槽组成。实际设计与生产中需将二者统筹作为整体考虑，以保障压铸件质量[[S43,S32,S24,S35]]。

排溢系统在压铸模具中不可缺少，其主要作用是排出和储存型腔中的气体、涂料残余物、前端污冷金属，达到转移或减少压铸件中的缩孔缩松、裹气及氧化夹杂以及冷隔等缺陷之目的[[S33]]。

## 组成与结构

压铸浇注排溢系统由浇注系统和排溢系统两大部分组成[[S13,S45]]。浇注系统承担输送、调控金属液充型流态的核心功能；排溢系统对应溢流槽和排气槽结构，两者在充型过程中协同工作[[S39,S41,S19]]。

### 浇注系统组成

浇注系统主要由**直浇道、横浇道、内浇口及余料**组成[[S24,S43]]。各类压铸机所采用的浇注系统结构有所差异，但基本组成部件一致[[S11]]。

### 排溢系统组成

排溢系统包括**溢流槽和排气道**两个部分，主要由溢流口、溢流槽和排气道组成[[S13,S38]]。当溢流槽开设在动模一侧时，为使溢流余料与压铸件一起脱模，可在溢流槽处设置推杆[[S39]]。

![压铸模具典型排溢系统结构组成示意图，标注型腔、溢口、溢流槽、排气槽、推杆等核心部件](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/6d591988-0b6e-467b-8e01-128fe82806e9/resource)

### 三个子系统的边界与协同

浇注系统、溢流系统、排气系统三者边界划分明确[[S39,S41,S19]]：

- **浇注系统**：承担输送、调控金属液充型流态的核心功能；
- **溢流系统**：对应溢流槽结构，负责收容充型前端的冷污合金、氧化物，设置在流道交汇处、充型末端等位置；
- **排气系统**：对应排气槽、排气块等结构，负责将型腔内原有气体、涂料挥发气体有序导出模具外部。

三个子系统在充型过程中协同工作：浇注系统引导金属液按预设路径推进，推动型腔内部气体向溢流槽位置聚集，最终经排气系统排出，实现平稳充型、减少卷气缺陷的目标。

## 分类

### 浇注系统类型

浇注系统按金属液进入型腔的部位和内浇口形状，大体可分为以下几种类型[[S43,S24,S11]]：

| 类型 | 特点与适用场景 |
|------|----------------|
| **侧浇口** | 开设在分型面上，内浇口布置在铸件最大轮廓的内外侧，适配板类、盘盖类、浅壳体类铸件，单型腔或多型腔模具均可适配，后续浇口去除难度低，适应性最广[[S43]] |
| **中心浇口** | 适配带有中心通孔的有底筒类、壳类铸件，内浇口开设在孔口位置，配合中心分流锥使用[[S43,S24]] |
| **顶浇口** | 由铸件顶部进料 |
| **环形浇口** | 适应环状铸件 |
| **缝隙浇口** | 缝隙式进料 |
| **点浇口** | 点状进料口 |

![五种不同类型的压铸浇注系统结构示意图，辅助理解各类浇注系统的适用场景区分要点](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/98d9e234-2c4b-42b2-8445-151ba780f47b/resource)

**多点进浇方案**：将大型复杂铸件的充型流程拆分缩短，降低不同区域金属液的温差，避免流纹、冷隔、欠铸等缺陷，同时支持采用更低的浇注温度、充填速度和增压比压，在保障铸件质量的前提下延长模具使用寿命、降低生产能耗[[S34,S9]]。

### 排气方案分类

常见排气方案及适用场景[[S21,S25,S39]]：

| 排气方案 | 特点与适用场景 |
|----------|----------------|
| **分型面排气** | 在模具分型面上对应充型末端位置开设浅槽，加工难度低，适配大部分常规结构压铸件 |
| **普通排气槽** | 设置在溢流槽后端，需控制槽深避免金属液喷出，一般配套溢流结构使用 |
| **排气块（排气塞）** | 镶嵌在模具深腔、窄缝等难以在分型面开设排气结构位置的多孔通气结构，用于局部死角导气[[S21]] |
| **真空辅助排气** | 适配高致密度要求、需后续机加工或焊接的承压类铸件，压铸前先将型腔抽为负压大幅降低气体残留 |

## 工艺作用

浇注排溢系统在压铸工艺中承担多项关键功能：

1. **排气与提高气密性**：可将型腔内的原有空气、脱模剂挥发气体有序排出，避免卷气导致的气孔缺陷，大幅提升铸件的气密性和整体致密度，保障承压类压铸件的密封性能要求[[S19,S30]]。

2. **补缩与提高致密性**：通过在铸件厚大部位对应位置设置溢流结构、采用厚内浇口从厚壁区域进料，在金属液凝固过程中持续传递压力补缩厚大区域，避免厚壁位置出现缩孔、缩松缺陷[[S12,S16]]。

3. **拓宽工艺窗口**：通过调控金属液充型流态、优化型腔整体温度分布，降低对浇注温度、压射速度等工艺参数的敏感度，减少因参数波动导致的欠铸、冷隔、流痕等缺陷出现概率[[S43,S39]]。

4. **热平衡调节**：溢流槽中高温的金属液可调节模具各部位温度，改善模具成型区域热平衡状态，减少压铸件流痕、冷隔或填充不足等现象的产生[[S38,S13]]。

5. **容纳冷污金属**：有序排出型腔中的气体和容纳冷污的金属液以及其他氧化杂物，防止在压铸件的局部产生涡流气泡、缩孔、疏松以及冷隔等压铸缺陷[[S38,S13]]。

6. **缺陷转移**：将缩孔、疏松、冷隔等压铸缺陷引导并转移到无关紧要的部位[[S38]]。

## 设计参数与准则

### 内浇口位置设计原则

内浇口位置选择需遵循以下设计原则[[S28]]：

- 避免金属液正面冲击型腔或型芯，防止过早封闭排气通道；
- 优先设置在铸件厚壁部位，保障压力有效传递；
- 尽量减少金属液流动方向改变，缩短填充流程；
- 单内浇口可避免多股金属流对冲形成涡流裹气。

### 溢流槽设计参数

溢流槽常规布置位置包括：金属液首次冲击区域、型腔最后填充区域、多股金属液流汇合位置、易产生涡流区域、型腔局部低温区域、内浇口两侧死角区域[[S45,S41,S10]]。

溢流槽总容积常规取值为对应型腔区域铸件体积的 **10%～30%**[[S33,S45]]。针对消除局部热节缩孔的溢流槽，容积需为对应热节体积的 **2～2.5 倍**[[S45,S15]]，薄壁铸件的溢流槽总容积占比需高于常规设计值[[S33]]。

溢流口设计约束要求：单个溢流槽不可设置多溢流口或过宽溢流口，溢流口总截面积需小于对应内浇口总截面积，且溢流口截面积需大于后端排气道的截面积，避免金属液倒灌回型腔[[S33,S41,S26]]。

### 排气道设计参数

排气道核心设计参数取值范围[[S33,S2]]：

| 参数 | 取值范围 |
|------|----------|
| 宽度 | 8～25 mm |
| 锌合金深度 | 0.05～0.08 mm（最大不超过 0.1 mm） |
| 铝/镁合金深度 | 0.08～0.15 mm（最大不超过 0.15 mm） |
| 铜合金深度 | 0.15～0.20 mm |
| 总截面积 | 为内浇口总截面积的 20%～50% |

排气道延伸至远离溢流槽 20～30 mm 位置后，可将深度提升至 0.3～0.4 mm 以强化排气效果[[S33]]。

### 浇道截面设计准则

浇道逐级截面缩减设计准则：主横浇道截面积需大于各分支横浇道截面积之和，从直浇道入口到内浇口的流道截面积需保持缓慢过渡，禁止流道截面积出现突然扩大或缩小的结构，避免产生低压区诱发金属液涡流裹气[[S45,S10]]。

### 排气槽面积计算公式

排气槽截面积的行业通用经验计算公式[[S27]]：

> Aq = 0.00224 × V / (λ × τ)

其中 Aq 为排气槽截面积（mm²），V 为型腔、浇注系统、溢流槽及压室注液后未填充部分的总体积（mm³），τ 为气体排出时间近似取铸件充型时间，λ 为排气槽开放系数（取值范围 0.1～1）。

实际设计时总排气槽截面积通常取内浇口截面积的 20%～50%，不得超过内浇口总面积[[S27,S33]]。

## 设计不良的后果

### 缺陷关联机制

不合理的浇排系统设计会使铝合金液充型过程中紊流加剧、气体无法有序排出，直接导致铸件出现**气洞、冷隔缺陷**；充型后凝固阶段流动路径不合理易造成补缩受阻，诱发**缩孔、缩松**类缺陷[[S31]]。

排溢系统设计不当的典型失效机制包括：不合理布局导致不同区域充型速度不一致，相邻溢流槽连通后出现金属液串流干涉，充型后期后续金属液冲击已填充的高温金属液面产生卷气，最终形成铸件内部气孔、伸长率不足等缺陷。优化独立引出排气的排溢系统可大幅降低卷气程度，某案例显示 U 型槽区域铸件伸长率可较原方案提升约 30%[[S44]]。

排溢系统的核心失效模式根源是压铸过程型腔内气体置换通道被堵塞或布局错位，无法顺利排出型腔内的气体、容纳混有气体和涂料残余的冷污金属液，直接破坏铸件质量保障逻辑[[S1]]。

## 与其它模具系统的关系

### 与冷却系统的布局协调

冷却水道布置在横浇道附近时必须距离内浇口足够远，避免浇口位置金属液过早凝固阻断补缩路径；冷却孔壁距离型腔/浇口壁的最小安全距离不小于 **10 mm**，防止冷却介质泄漏渗入型腔引发气孔缺陷[[S40]]。

### 与顶出系统的布局协调

溢流槽下方必须预留推杆布置空间，用于将浇排系统凝料从模具中顺利推出，同时需要通过三维模具动态仿真校验避免顶出杆与流道结构、冷却水道发生运动干涉，干涉校验属于压铸模 CAD 设计的必要环节[[S8,S6]]。

### 与压铸机的匹配

不同吨位、不同类型（热室/卧式冷室/立式冷室/全立式）压铸机对应专属的浇注系统结构，需结合压室充满度、锁模力参数、压射性能适配对应流道截面积比例，通过 P-Q² 图的工艺窗口匹配验证系统柔性，确保在可调工艺参数范围内都能生产合格铸件[[S37,S18]]。

## 设计验证方法

### 仿真校验

通过 CAE 模拟获取铸件充型流场、温度场分布，验证金属液充型顺序合理性、卷气区域位置、型腔排气路径畅通性，预测气孔、冷隔等缺陷的分布情况，提前优化浇排系统布局减少试模次数[[S8]]。

### NADCA 推荐设计要求

NADCA 推荐排溢系统采用恒定或收敛截面积的金属进料系统配合适配流道结构，推动空气在金属液前方从型腔有序流向排气道，溢流槽可作为额外辅助排气通道，容纳混有气体的先期冷污金属液。设计合理的溢流-排气组合可完全消除铸件表面气泡缺陷[[S22,S29]]。

### 高气密性铸件设计准则

高气密性压铸件的浇排系统设计核心准则：采用大夹角扇形侧浇口使金属液流向与铸件散热片等薄壁结构走向一致，避免出现垂直流向的紊流裹气现象，确保金属液有序填充将气体完全排出型腔，提升铸件致密度满足气密性要求[[S27]]。
