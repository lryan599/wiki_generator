## 概述

低压铸造机是适配低压浇注工艺的专用铸造设备，属于铝合金低压铸造的通用设备，广泛应用于汽车、摩托车、仪表、航空航天等领域铝合金铸件的生产[[S58]]。该设备是目前应用最广泛、完备度最高的反重力铸造设备[[S49,S11]]。

低压铸造机与高压压铸机、重力铸造机、离心铸造机分属不同的铸造设备类别。其核心差异在于驱动金属液充型的方式：低压铸造机依靠向密封保温炉内通入低压气体（0.03~0.5 MPa）驱动金属液自下而上平稳充型[[S26]]；重力铸造机依靠金属液自重充型，无密封加压系统[[S30]]；高压压铸机依靠柱塞提供数千至数万psi级高压实现高速充型[[S41]]；离心铸造机则依靠设备旋转产生的离心力驱动金属液填充型腔[[S33]]。

## 工作原理

低压铸造机的基本工作原理为：向密封保温炉内通入0.03~0.5 MPa范围的低压干燥压缩空气（或惰性气体），使金属液在气体压力作用下，沿升液管自下而上平稳上升，通过内浇口进入铸型型腔；充型完成后在设定压力下完成保压凝固；待铸件完全凝固后撤除炉内压力，升液管与浇道内未凝固的金属液依靠重力回流至保温炉内；随后开模取出铸件[[S26,S56]]。

## 结构组成

低压铸造机一般由主机、液压系统、熔池式保温炉、液面加压装置、电气控制系统、模具冷却系统六大核心模块组成，部分现代化机型还配套故障自诊断、多组工艺参数自动存储调用功能[[S58,S40]]。

### 保温炉及坩埚系统

保温炉是低压铸造机核心储热控温部件，主流分为两类：

| 类型 | 常规容量规格 | 优势 | 劣势 |
|------|-------------|------|------|
| 石墨坩埚式 | 300 kg、450 kg | 不吸潮 | 使用寿命不超过6个月 |
| 耐火炉衬式 | 300~1000 kg | 炉衬使用寿命可达5年以上 | 易吸潮，铸造停机期间也必须持续加热保温 |

正常工况下保温炉外壳（炉盖、炉门区域除外）平均温升不超过70℃[[S18]]。GB/T 28688-2012规定，采用PID控温模式时炉内金属液温度误差不超过±5%，有级功调位式控制时误差不大于±7%[[S36,S18]]。

坩埚是盛装高温熔融金属液的承压容器，主流材质分为球墨铸铁和石墨两类。受力状态下坩埚壁厚通常设置为30~35 mm，石墨材质坩埚可以避免铁元素向铜、镁合金液中迁移污染[[S29,S1]]。

### 升液管

升液管是连接坩埚内金属液面与铸型浇口的专用过渡通道，核心作用是在气压驱动下将熔融金属液平稳输送至铸型型腔底部，避免金属液紊流卷入气体。常规机型升液管上端直接与机架平台上的铸型浇口实现密封对接[[S29,S49]]。

### 模具夹紧机构

模具夹紧机构普遍采用液压油缸驱动模式，运行平稳、合模压力可连续无级调节，支持大行程开模范围适配不同高度的模具。多数机型配套侧模油缸、铸件顶出液压缸，可自动完成合模、开模、铸件顶出全流程动作[[S11]]。

### 供气系统（液面加压控制系统）

供气系统是低压铸造机的核心压力调控部件，搭载高灵敏度压力传感器和软件PID控制器，可高精度复现预先设定的压力-时间工艺曲线，自动补偿炉体泄漏、供气管路气压波动、金属液位下降带来的压力偏差。主流机型支持设置6级以上可编程压力曲线适配不同铸件的充型需求[[S49,S22]]。

### 电气控制系统

电控系统普遍采用PLC工业控制架构，可实现充型、排气、冷却、开模、顶出、取件全铸造流程的自动化运行，具备工艺参数存储调用、生产自动计数、故障自诊断及故障原因实时显示功能。主流机型操作终端配置中文触摸屏，可同步展示工艺设定加压曲线与设备实时运行的加压曲线[[S51]]。

## 分类

低压铸造机有多种分类方法[[S31]]：

**按铸型与保温炉的连接方式**，可分为顶置式（顶铸式）和侧置式（侧铸式）两大类：

| 类型 | 结构特点 | 生产效率 | 其他说明 |
|------|---------|---------|---------|
| 顶置式 | 铸型直接布置在保温炉正上方，单台保温炉仅可配套1副铸型 | 较低 | 结构简单，操作维护方便 |
| 侧置式 | 铸型布置在保温炉侧面，一台保温炉可同时为2副及以上铸型提供金属液 | 显著提升 | 装料、撇渣、金属液处理作业更便捷，但结构复杂 |

**按金属型开合方式**，可分为卧式（铸型垂直分型）和立式低压铸造机（铸型水平分型）[[S31]]。

**按机架与炉体的相对运动方式**，可分为机架活动式和炉体移动式两类[[S31]]。

此外，还存在面向特定铸件的定制化专用低压铸造机以及双工位/多工位高效机型[[S31]]。

## 核心工艺参数

GB/T 28688-2012《低压铸造机 技术条件》对核心工艺参数做出了明确定义[[S36]]：

| 参数名称 | 定义 |
|---------|------|
| 加压速率 | 升液、充型和增压阶段，保温炉内的气体压力增长量与其时间之比 |
| 升液压力 | 升液管内的金属液上升至浇口处时，保温炉内所需的压力 |
| 充型压力 | 金属液充满铸型时，保温炉内所需的压力 |
| 保压压力 | 铸型充满后，为补缩的需要，在保温炉内的金属液面上施加比充型压力高的压力 |
| 保压时间 | 金属液充型后在凝固过程中，在保压压力下所持续的时间 |
| 液位补偿压力 | 为保证充型开始以后的工艺参数重复再现，不受金属液位变动的影响，在每后一次浇注前由系统自动增加的补偿量 |

标准要求所有上述工艺参数支持按需灵活可调[[S36,S18]]。

以铝合金低压铸造为例，典型工况核心参数区间见表[[S12,S28]]：

| 参数项 | 典型范围 |
|-------|---------|
| 升液压力 | 约0.02 MPa |
| 充型压力 | 0.02~0.05 MPa |
| 最高保压压力 | 可达0.1 MPa |
| 升压速度 | 0.0014~0.05 MPa/s |
| 保压时间（15寸铝车轮） | 160 s |
| 铝液保温温度 | 720±5 ℃ |
| 充型阶段工艺参数重复性误差 | ≤2% |
| 加压速率控制误差 | ≤0.5% |

## 工艺作用：顺序凝固与补缩

低压铸造机的核心工艺目标是实现铸件自上而下的顺序凝固——远离浇口的铸件远端位置先凝固，浇口最后凝固。依靠位于铸型下方的升液管中炽热金属液，经由浇注系统为铸件提供持续的自下而上的压力补缩，避免孤立液相区形成，从根源上减少缩孔缩松缺陷，获得组织致密的铸件[[S24,S35,S34,S32]]。

数字化控制型低压铸造机通过PID数字比例阀+电液伺服系统实现分段压力闭环控制，可在15 ms内完成压力启停/换向，实时根据压力传感器反馈数据对保压阶段压力偏差进行动态补偿，避免保压阶段压力振荡或超调，保障补缩压力稳定传递到铸件所有部位[[S4,S54]]。

模具冷却方面，多通道时序冷却控制系统通过在模具各冷却回路安装温度传感器和比例流量阀，按照预设顺序依次开启/关闭冷却水路，优先冷却铸件远离浇口的远端区域，延迟开启靠近浇口位置的冷却水路，确保浇口最后凝固，维持完整的补缩通道[[S27,S13]]。

## 适用材料与生产率

低压铸造机适配的熔融金属材料种类包含铝合金、镁合金、铜合金三类[[S11]]。常规中小型铝合金铸件的典型单批次生产率为15~20件/小时[[S11]]；面向铝汽缸头的专用旋转式双腔机型生产效率可达20件/小时，单件铸件重量为7.5 kg[[S6]]。

## 设备控制不当导致的铸造缺陷

低压铸造中，因设备参数设定或控制不当可能导致多种铸造缺陷：

**缩松**：保压压力不足、压力滞后无法及时传递到热节区域，会导致热节部位液态收缩无法得到金属液补充，最终在孤立液相区形成分散的细小孔洞。经正交试验验证，缸盖类铸件缩松缺陷的影响因素权重排序为：增压压力 > 冷却时间 > 模具预热温度 > 铝液温度[[S13,S15,S10]]。

**气孔**：主要设备参数诱因包含充型速度过快导致金属液紊流卷气、模具/砂芯排气通道设计不合理或堵塞、充型后型腔内析出的气体无法顺利排出。气孔特征为表面光滑、带有轻微氧化色泽，多呈圆形或椭圆形[[S47,S2]]。

**浇不足**：主要诱因包含模具预热温度过低导致金属液流动性下降、升液压力/充型速度设置不足、局部排气通道堵塞导致型腔憋气[[S8]]。

**氧化夹杂**：主要由充型速度参数设置不合理引发——充型速度过快导致液流剧烈翻腾，液面氧化膜破裂卷入金属液内部[[S3]]。

## 局限性及与其它铸造设备的比较

### 设备成本

低压铸造机购置及维护成本仅为高压压铸机的1/3~1/5，高于普通重力铸造机[[S45,S57]]。

### 可生产铸件尺寸

常规高压压铸机受锁模力限制，最大铸件重量一般不超过40磅（约18 kg）；低压铸造机可生产最大重量达175磅（约79 kg）的大型铸件，标准机型可加工最大直径约30英寸（762 mm）的铸件[[S45,S61,S23]]。

### 模具寿命与成本

低压铸造机配套铸铁模具平均寿命高于重力铸造机的常规金属型模具；低压铸造机模具购置成本仅为高压压铸模具的2/3左右，重力铸造模具成本最低[[S57,S45,S38]]。

### 生产效率

高压压铸机生产率最高，常规机型可达50~560次/小时，生产周期通常低于60 s；低压铸造机常规工况下小铸件生产效率为150~200件/小时，大铸件为5~8件/小时，生产率约为高压压铸机的1/2；普通重力铸造机生产效率介于两者之间[[S45,S37,S56]]。

### 金属利用率

| 工艺类型 | 金属利用率 |
|---------|-----------|
| 低压铸造 | 85~95% |
| 高压压铸 | 50~60% |
| 重力铸造 | 45~55% |

低压铸造金属利用率显著高于高压压铸和重力铸造[[S17]]。

## 应用领域

低压铸造机典型应用领域覆盖[[S58,S62,S48]]：

- 汽车行业：量产铝合金轮毂、气缸盖、缸体等部件
- 航空航天工业：高性能薄壁铸件生产
- 电气行业：GIS高压壳体类耐压铝合金铸件
- 轨道交通：铝合金牵引梁铸件
- 摩托车及普通仪表行业：铝合金零件生产

## 与“低压铸造”词条的关系

“低压铸造”词条指向完整的铸造工艺方法，涵盖原理、工艺流程、参数体系、铸件特性等工艺类内容；“低压铸造机”词条指向执行该工艺的实体工业装备，属于铸造设备类目。二者分别隶属于工艺类和装备类两个完全独立的语义范畴[[S58,S26]]。

## 标准与规范

### 技术条件标准

GB/T 28688-2012《低压铸造机 技术条件》于2012年9月3日发布，2013年1月1日实施，适用于生产铝铸件的低压铸造机[[S36]]。该标准规定了11项核心工艺参数术语定义，并对保温炉性能提出明确要求：

| 项目 | 要求 |
|------|------|
| 铝液温度误差（PID控制） | 不大于±5% |
| 铝液温度误差（有级功调位式） | 不大于±7% |
| 炉体/罐体密封泄漏率 | 不大于40 L/min |
| 坩埚密封泄漏率 | 不大于25 L/min |
| 液压系统固体颗粒污染等级 | GB/T14039-2002等级代号-/18/15 |
| 连续空运转试验时长 | 不低于8 h |

### 安全标准

GB 24391-2009《低压铸造机 安全要求》为强制性国家标准，2010年7月1日实施，适用于坩埚密封和炉体密封、水平分型及垂直分型的低压铸造机[[S42,S21]]。该标准明确了8大类风险的防控要求，包括机械夹挤剪切、高压流体喷射、电击烫伤、噪声超标等，并规定：

- 铸型危险区需设置防护装置[[S42]]
- 合型机构与浇注工序强制联锁[[S43]]
- 需设置防止开型后动型板意外下滑的互锁保护装置[[S43]]
- 设备安全设计需符合GB/T15706系列标准的人机工程相关要求[[S7]]
