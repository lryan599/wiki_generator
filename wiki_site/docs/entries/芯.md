## 概述

在压铸模具语境中，“芯”泛指用于形成铸件内孔、内腔或复杂内部结构的模具零件[[S4]]。其最狭义且最核心的指代对象为**型芯**（core / mold core），即固定在模具或借助活动机构伸入型腔、在金属液凝固后抽离以形成铸件内部特征的部件。广义上，“芯”亦涵盖抽芯机构中的**芯销**（core pin）、**滑块型芯**（core slide）以及用于特殊铸造工艺的**砂芯**（sand core）、**盐芯**（salt core）等[[S4,S10]]。在多份北美压铸协会文献中，“芯”对应 **core** 或 **core pin**，强调其与型腔（cavity）的几何区别：型腔构成铸件外表面，芯构成铸件内表面[[S4,S6]]。

## 分类体系

### 按功能与运动方式分类

在压铸模具中，型芯可根据运动特征划分为固定型芯与活动型芯两大类别[[S4]]：

- **固定型芯（fixed core）**：与模具主体相对静止，沿开模方向布置。通常直接加工在型腔块上，或作为固定镶件安装于定模/动模，不需要独立的抽芯机构。
- **活动型芯（movable core / core slide）**：沿非开模方向（与分型面成一定角度）布置，通过**抽芯机构**（core-pulling mechanism）在开模时先行退出铸件，或在合模时复位。常见的驱动形式包括液压缸抽芯、斜导柱（angled pin）驱动滑块抽芯、弯销抽芯等[[S1,S4]]。

此外，按在铸件成型中的具体功能，芯可进一步区分为：

- **成型芯（forming core）**：直接参与形成铸件内孔、内腔形状，是型芯最主要的功能形态。
- **冷却芯（cooling core）**：内部布置冷却水道，兼具成型与强化局部冷却功能，多采用高热导率材料或随形冷却设计[[S12]]。
- **定位芯**：用于两半模或多滑块之间的相互定位，保证合模精度。

### 按材料分类

不同工作环境与性能要求决定型芯的材料选择[[S4,S13]]：

| 材料类别 | 典型牌号 | 主要优势 | 主要局限 |
|---|---|---|---|
| 热作模具钢 | H13 (SKD61), 8407 | 高温强度与热疲劳抗力均衡，性价比高，应用最广 | 热导率较低，急冷急热工况下表面易产生热疲劳裂纹 |
| 铍铜合金 | C17200 等 | 热导率高（约为H13的4～6倍），散热快，可缩短凝固时间、减少粘模 | 成本高，对模具冷却与维护条件敏感，需关注应力状态[[S13]] |
| 钼合金/钨基合金 | TZM 等 | 耐极高温度，热膨胀系数低 | 加工困难，成本极高，仅特殊高温合金压铸中使用 |
| 硬质合金/表面强化 | WC-Co，PVD涂层，渗氮 | 表面硬度极高，抗冲蚀与抗粘模性能优良[[S11]] | 本体韧性相对较低，多在普通钢芯表面以涂层或镶件形式应用 |

### 按工艺关联的特殊芯材

在间接挤压铸造、半固态成形或特定高压铸造场景中，部分型芯采用非金属消耗性材料，在铸件成型后通过化学或机械方式移除，以形成无法直接抽芯的复杂内腔结构[[S10]]：

- **盐芯（salt core）**：以可溶性无机盐（如NaCl、KCl基复合盐）经压制或注射成型制成型芯，浇注后以高压水溶解去除。适用于铝合金闭腔结构零部件[[S10]]。
- **砂芯（sand core）**：传统砂型铸造中应用普遍，在部分压铸派生工艺中亦有使用，通过振动或喷砂去除。

## 结构与组成参数

典型型芯的结构包含**成型段**、**定位配合段**（与模具本体或滑块配合）、**冷却水道**和**连接锁紧端**[[S7]]。

- **成型段**：直接与熔体接触，决定铸件内孔尺寸与表面质量。其几何外形需考虑拔模斜度（draft angle），以减小抽芯阻力并防止拉伤铸件。
- **配合间隙**：型芯与模具本体或滑块之间的间隙直接影响飞边和偏芯倾向。间隙过大会导致金属液渗入形成飞边，过小则因热膨胀导致卡死[[S4]]。
- **冷却水道**：在芯内部加工出冷却通道，常用方案为钻孔对插隔板或3D打印随形冷却流道，以控制芯的工作温度并延缓热疲劳[[S12]]。

抽芯机构中，表示运动学特征的关键几何参数包括[[S1,S7]]：

- **抽芯距（core-pulling distance）**：S ≥ h + K，其中h为型芯在型腔中的成型深度，K为安全裕量（通常3～5 mm）[[S7]]。
- **抽芯角度**：开模方向与芯运动方向的夹角，影响滑块行程和斜导柱长度设计。
- **锁紧块楔角**：合模时保证型芯位置精度的楔紧角度，须大于斜导柱驱动角以自锁。

## 工艺作用

型芯在高压压铸工艺流程中发挥以下核心作用[[S4]]：

1. **充型阶段**：型芯占据部分型腔空间，改变金属液流动路径。细长型芯可能阻碍充型，引起充填不良或卷气（气孔缺陷）。
2. **凝固阶段**：型芯作为内表面热交换界面，从内部对铸件施加强制冷却。型芯热导率直接影响该区域的凝固速率，进而影响缩松倾向。
3. **冷却与收缩阶段**：型芯在铸件收缩时承受收缩应力。若抽芯时机不当或冷却不均，可能导致热裂或包紧力过大。
4. **顶出与抽芯阶段**：活动型芯需先于铸件顶出动作完成抽芯，以避免铸件变形或拉伤[[S1]]。抽芯力的计算须考虑包紧力、热膨胀引起的摩擦增大以及材料粘模倾向。

## 关键工作参数与控制要求

型芯的工作环境极端苛刻，设计时需重点关注以下参数范围与技术要求：

| 参数类别 | 典型要求/范围 | 说明 |
|---|---|---|
| 工作温度 | 表面温度可达400～550 ℃（铝合金压铸）；局部接触区更高[[S3]] | 需通过内部冷却水道将温度控制在材料回火软化温度以下 |
| 表面硬度 | H13型芯通常44～48 HRC；铍铜型芯36～42 HRC；PVD涂层型芯表面可达2000 HV以上[[S11]] | 硬度不足易导致冲蚀与粘模，过高又可能引起脆性开裂 |
| 表面处理 | 渗氮（nitriding）、PVD涂层（CrN，AlCrN等）、微喷丸[[S11]] | 涂层应兼顾抗热疲劳、抗粘模与抗冲蚀能力 |
| 配合间隙 | 一般0.025～0.075 mm（依型芯尺寸与热膨胀校核） | 名义间隙需考虑热膨胀差异和加工同轴度公差 |
| 抽芯力 | 根据包紧力分析，铝压铸典型参考值0.5～2.5吨/英寸直径（依芯长度、壁厚、拔模斜度变化） | 过大抽芯力可能拉伤铸件；不足则无法顺利抽出[[S1]] |
| 抽芯距安全性 | S ≥ h + (3～5) mm[[S7]] | 避免抽芯不到位造成与铸件干涉 |

## 失效与缺陷机理

型芯在压铸生产中承受剧烈热-力耦合循环载荷，其失效形式呈现多种模式[[S4,S11]]：

| 失效模式 | 机理简述 | 关联铸件缺陷 |
|---|---|---|
| **热疲劳裂纹（heat checking）** | 周期性的急热急冷导致表面拉-压交变应力，形成网状微裂纹并逐步扩展[[S4,S5]] | 铸件孔壁出现网状凸纹，降低表面质量与型芯寿命 |
| **冲蚀（washout）** | 高速金属液流对型芯表面的机械冲刷，在浇口对侧或流速集中区域最为严重 | 型芯尺寸丧失，铸件内孔形状失真 |
| **粘模（soldering）** | 铝合金与型芯表面在高温高压下发生金属间化合物反应，局部焊合[[S3]] | 抽芯困难，铸件孔壁拉伤，型芯表面留下铝层 |
| **变形与断裂** | 细长型芯在反复热应力与机械载荷下产生塑性变形或弯曲断裂 | 芯偏（core shift），铸件壁厚不均[[S8]]；严重时型芯折断留于铸件 |
| **飞边（flash at core）** | 配合间隙过大或锁紧力不足，金属液渗入芯与模具间隙 | 铸件内孔根部出现薄片状金属飞边 |

其中，**芯偏**（core shift）特指型芯在充型压力或热载荷作用下偏离理论位置，导致铸件内孔与外表面不同心或壁厚超差[[S8]]；而**粘模**被认为是压铸高铝/高硅合金时型芯面临的最显著失效威胁之一——NADCA等机构曾专题讨论铝与模具钢的焊合机制，指出Fe-Al金属间化合物的形成是粘模失效的直接原因[[S3]]。

为提高型芯抗失效能力，常用的对策包括优化冷却使表面温度峰值低于焊合临界温度、施加PVD涂层阻隔铝接触[[S11]]、采用铍铜等高热导镶件加速散热，以及应用微喷丸或激光表面改性延缓热疲劳裂纹萌生[[S5]]。

## 应用实例与关联关系

### 与抽芯机构及滑块的关系

在实际模具中，型芯往往是**抽芯机构**的一部分——型芯安装于滑块前端，由斜导柱、液压缸或弯销驱动实现往复运动[[S1]]。美国ITI Winsert等供应商将型芯销（core pin）定义为“直接形成铸件内部形状的圆柱状或异形零件，通常安装于滑块或芯座上（mounting in a slide or core seat）”[[S6]]。这一表述与压铸模具设计资料中的“core slide”概念完全一致[[S4]]。

斜导柱抽芯机构是压铸模具中最典型的侧抽机构，其设计公式直接关联型芯成型深度：

\[
S = h + K
\]

其中 K 为安全系数（常取 3～5 mm），以确保在滑块有效行程结束时型芯已完全脱离铸件[[S7]]。

### 与型腔、浇道、排气系统的配合关系

型芯在型腔内的布局须兼顾充型流动平衡与顺序凝固。靠近内浇道的型芯首当其冲承受高温金属液直接冲击，因此该区域芯子更需考虑抗冲蚀与抗粘模强化[[S3,S11]]。排气槽的开设应避免直接朝向型芯根部，以防金属液在型芯配合面快速凝结造成活动间隙阻塞。

### 典型铸件中的芯应用案例

- **铝合金缸体（cylinder block）**：缸套孔、水套腔等大量采用活动抽芯，多个方向的油道芯、水套芯构成复杂的滑块与型芯交错结构。冷却水道芯（bore pin）通常需配以独立冷却回路。
- **自动变速器壳体/阀体**：细长油道孔多采用单侧抽芯或双侧对插型芯，精度与同轴度要求极高，芯偏缺陷在该类零件中不可接受[[S8]]。
- **闭腔结构件（如泵体、歧管）**：常借助盐芯或可溶芯实现无法抽芯的封闭腔体[[S10]]。

### 先进技术与前沿应用

**3D打印随形冷却型芯**：以选区激光熔化（SLM）等增材制造技术在H13钢芯内部加工出随形冷却水道，可显著降低芯的表观温度峰值并改善温度均匀性。研究表明，优化设计的SLM型芯在压铸过程中，其体积平均温度与表面温度分布均优于传统钻孔冷却芯，这对延缓热疲劳和抑制粘模具有显著工程意义[[S12]]。

**盐芯在高压压铸中的适用性**：专利与工艺文献表明，盐芯经表面强化后可承受压铸充型冲击，在铝合金压铸中以水溶方式去除，形成高精度闭腔。该技术已在部分汽车轻量化部件中进入小批量试产阶段[[S10]]。
