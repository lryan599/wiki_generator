## 概述

受阻收缩是铸造合金在凝固与冷却过程中，因受到外界约束或自身结构制约，无法完全实现自由收缩的物理现象。受阻收缩属于铸造固态收缩阶段的典型约束表现形式，其核心特征为实际发生的线收缩量低于同等条件下合金本征自由收缩量，且受阻程度越强，铸件内部越容易出现收缩应力、热裂、缩松、变形及残余应力等缺陷[[S53,S28,S38]]。

受阻收缩与自由收缩共同构成铸造收缩行为的两种基本类型，是铸造工艺设计与缺陷分析领域中的关键概念[[S53,S45]]。

## 定义

国家标准中对受阻收缩的正式定义为：铸件在凝固和冷却过程中，受到机械作用的阻碍，不能自由进行的收缩，受阻线收缩值一般小于自由线收缩值[[S53]]。

与之对应，自由收缩是指金属从凝固起始温度冷却至室温的整个过程中，不受阻碍能自由进行的收缩[[S53,S45]]。在工程实践中，铸件实际发生的收缩行为绝大多数属于受阻收缩，铸造模具或模样设计时采用的铸造收缩率，实质上是综合考虑各种受阻因素后的受阻收缩率，而非合金的本征自由收缩率[[S28,S30]]。

## 收缩阻力分类

铸件在收缩过程中受到的阻力，按约束来源可系统划分为以下三类[[S28,S38]]：

| 阻力类型 | 产生机制 | 主要影响因素 |
|----------|----------|--------------|
| 摩擦阻力 | 铸件表面与型腔表面间相对运动形成的阻力 | 铸件重量、型腔表面平滑程度；表面有涂料时可忽略[[S28,S38]] |
| 热阻力 | 铸件各部分温度不均、收缩不同步相互制约形成的阻力 | 铸件结构、温度分布及材料性质[[S28,S38]] |
| 机械阻力 | 铸件突出部分或内腔型芯受到型壁、型芯刚性约束产生的阻力 | 造型材料强度、退让性、铸型和型芯紧实度[[S28,S38]] |

在实际铸件凝固过程中，通常三种阻力同时存在并耦合作用，综合制约铸件的自由收缩[[S38]]。

## 发生阶段与判别

### 按凝固阶段分类

受阻收缩可依合金固液共存区的物态演化过程划分为两个阶段[[S34,S46]]：

- **准液相区**（液相线至准固相线区间）：固相率较低，液相仍具良好流动性，此阶段收缩行为主要表现为液相补缩，受阻收缩尚未引发严重力学损伤。
- **准固相区**（准固相线至固相线区间）：固相已搭建连续枝晶骨架，合金开始呈现固态力学特性并产生线收缩；该阶段的受阻收缩是诱发热裂的核心原因，受阻热节在准固相区停留时间越长，热裂趋势越大[[S33,S46]]。

### 判别准则

工程中可采用包络密度理论对受阻收缩进行定量判别[[S7,S10]]：

- 当铸件包络密度等于实体密度时，铸件处于完全自由收缩状态；
- 当包络密度小于实体密度时，铸件处于受阻收缩状态，且包络密度越小，受阻程度越严重。

更通用的充分必要条件判别准则为：铸件上任意一点与收缩中心的连线都不穿过砂型（或铸型）时，为自由收缩；当存在连接线穿过砂型且铸件存在包络密度时，则为受阻收缩[[S2,S6]]。对于L形铸件，可通过纵向厚高比ζ与横向厚高比ξ进行快速判定[[S6,S40]]。

### 受阻因子

为消除材料密度差异对受阻程度描述的影响，引入无量纲受阻因子η来广义表征几何形状对收缩的阻碍程度[[S7,S10]]：

$$ \eta = \frac{\rho'}{\rho} $$

其中ρ'为包络密度，ρ为铸件实体密度。η越大，受阻程度越小；η越小，受阻越严重。对于筒形铸件，受阻因子与外径R和壁厚δ相关，与高度H无关[[S7]]：

$$ \eta = \frac{2R\delta - \delta^2}{R^2} $$

工程研究表明，当η = 0.13（即δ:R = 1:15）时，筒形铸件极易产生热裂纹[[S10]]。

**压铸工业常用经验值：**

- A380铝合金受阻收缩率：0.45%～0.65%
- ZZnAl4Cu1锌合金受阻收缩率：0.4%～0.6%
- AZ91D镁合金受阻收缩率：0.5%～0.75%
- 灰铸铁HT200受阻收缩率：0.3%～0.5%[[S11,S31]]

所有合金的受阻收缩率均显著低于对应自由收缩率。受阻程度还与铸件结构密切相关，直杆结构收缩率为2.4%，哑铃形结构为1.64%，H形结构为0.92%，直观体现了不同包络约束程度对实际收缩的显著影响[[S19,S56]]。

压铸铝合金的受阻收缩率还受壁厚影响：壁厚≥3mm时，受阻程度越大，收缩率越小[[S31]]。

## 压铸环境中受阻收缩的特殊机制

压铸环境下，受阻收缩的热-力学耦合作用特点如下[[S43,S54]]：

1. **模具刚性约束**：压铸模具材料（如H13、8407）的热膨胀系数约为12～13×10⁻⁶/℃，而ADC12等压铸铝合金的热膨胀系数高达23～25×10⁻⁶/℃。模具不仅不具备退让性，甚至受热膨胀后会进一步拉伸正在收缩的铸件，显著放大了受阻效应[[S54]]。
2. **凝固壳强度演化**：铸件从凝固后期某一高于固相线的温度开始产生线收缩，随枝晶骨架逐步形成、高温强度不断提升，收缩从液相自由流动变形转变为带约束的固态变形[[S16]]。
3. **温度梯度分布**：铸件厚薄部位冷却速率差异导致固态收缩不同步，加大各部位的相互制约效应[[S43,S21]]。

此外，铸件在型内停留时间越长（拔芯/出型温度越低），进入弹性变形区后冷却收缩受阻越久，内部产生的收缩应力越大；应力超过材料抗拉强度时可能引发铸件开裂[[S43]]。

## 诱发的铸造缺陷

受阻收缩是多种典型铸造缺陷的核心诱因，内在关联机制如下[[S8,S15,S16,S50]]：

### 热裂

铸件在凝固末期的脆性温度区内，枝晶骨架已形成而晶间残存低熔点液相无法自由流动，受阻收缩产生的拉应力超过该温度下合金的高温强度极限时，晶间液膜被撕裂且无法得到液相补缩，即形成热裂纹[[S15,S16]]。

热裂的典型特征包括[[S15,S48]]：
- 裂纹沿晶界萌生和扩展；
- 外裂表面宽、内部窄，呈撕裂状；
- 内裂分布在最后凝固区，常与树枝晶伴生；
- 断口氧化严重，无金属光泽。

定量判据方面，当铸件应力增长速率σₜ与凝固速率R的比值大于3.8时，半固态区极易形成热裂[[S10]]。

热裂纹的SEM断口表面可清晰观察到发达的树枝晶臂结构，这是判定缺陷属于凝固过程中受阻收缩引发热裂的关键微观证据[[S36,S44]]。

### 缩松/缩孔

受阻收缩会在铸件热节位置产生额外的拉应力，降低孤立液相区的补缩驱动力，阻断补缩通道，导致原本可被完全填充的收缩空间无法得到液相补缩，最终形成分散性缩松或集中缩孔[[S50,S10]]。

![受阻收缩条件下压铸件缩孔缺陷的逐步形成过程示意](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/5f3e9dc2-3019-49f9-95f9-76adcef4f6a6/resource)

### 铸件变形

不同位置受阻收缩程度不均，破坏铸件各部位的线收缩对称性，引发非均匀翘曲和弯曲变形，显著降低铸件尺寸精度[[S5,S18]]。

### 残余应力

铸件进入弹性变形区间后，受阻收缩产生的弹性拉应力无法完全释放，最终残留在铸件内部，削弱铸件承载能力，甚至引发使用过程中的延迟开裂[[S8]]。

## 影响因素

| 影响因素 | 影响规律 |
|----------|----------|
| 合金成分 | 结晶温度范围越宽，线收缩起始温度越高，受阻收缩引发的拉应力越大；对于铝合金，Si含量越高，热膨胀系数越低，总应变量越小[[S16,S12,S54]] |
| 冷却速度 | 随凝固与冷却速度提升，内部体积收缩占比下降、外部体积收缩占比上升，受阻收缩的约束效应被部分抵消[[S12]] |
| 模具温度 | 模具温度越接近铸件温度，温度梯度越平缓，热阻力越弱，受阻程度越低[[S21,S39]] |
| 浇注温度 | 浇注温度过高会降低凝固壳在热裂危险期的高温强度；ZG35铸钢浇注温度从1580℃降至1530℃，热裂危险期结壳强度可提升31%[[S32]] |
| 铸件结构 | 封闭包络结构、壁厚突变、框形约束的受阻收缩程度远高于开放结构件[[S7,S10]] |
| 型芯退让性 | 砂芯溃散性系数越低，受阻越严重，同等几何结构下铸件热裂裂纹总长度越大[[S10,S37]] |
| 抽芯/顶出时机 | 停留过久会加大收缩应力；过早顶出则易导致铸件变形[[S43]] |

## 控制策略

### 模具设计方面

1. **分区收缩率设计**：模具设计时不统一选取单一收缩率，可根据铸件不同部位的受阻程度分别赋值收缩率，或按垂直分型面的两个轴向分别设置，适配各方向约束差异，提升最终尺寸精度[[S1,S3,S47]]。
2. **减少型芯包紧力**：在成孔型芯对应位置设计中心区减薄及孔端倒角，缓解凝固收缩对型芯的过度包紧与应力集中[[S55,S25]]。

### 工艺设计方面

1. **壁厚均匀化**：将壁厚悬殊的铸件底部改为内凹腔，使全铸件壁厚均匀，减小收缩应力，避免翘曲变形和厚大部位的缩孔缩松缺陷[[S18,S5,S49]]。
2. **加强筋设置**：箱形薄壁件合理布置对称分布的加强肋，可有效抵消收缩应力，大幅降低整体变形量[[S5,S22]]。
3. **局部冷却调控**：针对厚大热节区域，通过调整冷却时间（如电控壳体水道盖板冷却时间优化至22s），可将受阻收缩引发的泄漏废品率从0.89%降至0.47%[[S29]]。
4. **冷铁定向凝固**：在厚大热节区域匹配设计冷铁，实现定向凝固，使不同区域凝固顺序均匀化，抵消局部受阻收缩引发的缩松缺陷[[S27]]。
5. **模具温度调节**：提升模具预热温度可延缓温度梯度峰值出现时间、降低整体温度梯度最大值，从而减少因受阻收缩产生的热裂倾向[[S21]]。
6. **柔性设计**：避免框形浇注系统结构，薄壁铸件采用扁宽浇口，设置防裂工艺肋，通过自割浇口将热裂转移至内浇口处[[S32]]。

## 相关概念

受阻收缩与以下核心概念构成铸造收缩理论体系[[S53,S30,S8,S26,S38]]：

| 概念 | 关系说明 |
|------|----------|
| 自由收缩 | 受阻收缩的对偶概念，指收缩不受任何阻碍的理想状态[[S53,S45]] |
| 线收缩 | 固态收缩阶段三个方向的线尺寸缩小，受阻线收缩值是工程铸件线收缩率的实际取值基础[[S30,S53]] |
| 体收缩 | 以体积缩减表征的收缩量，主要对应液态和凝固收缩阶段[[S26]] |
| 收缩率 | 衡量收缩程度的通用量化指标，工程中采用的铸造收缩率是受阻收缩率[[S35,S28]] |
| 收缩应力 | 铸件固态收缩时因受阻而引起的铸造应力，是受阻收缩的直接力学效应[[S53]] |
| 热应力 | 铸件各部位温差造成不均衡收缩引起的应力，对应受阻收缩中热阻力分量[[S53,S8]] |
| 铸造应力 | 收缩应力、热应力和相变应力的矢量和[[S53]] |
