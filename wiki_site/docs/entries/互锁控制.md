## 概述

互锁控制（Interlock Control）在压铸语境下，是指在压铸机及其辅助设备上设置的一种控制形式，只有满足预设的特定条件时，机器才能执行后续动作。例如，当压铸安全门未完全关闭时，机器不能执行合模动作 [[S1]]。该定义是压铸场景中特有的机器安全与顺序控制手段，区别于通用电气、软件领域的常规互锁定义 [[S1]]。

在压铸单元中，互锁控制具备双重属性：它既是安全控制装置，承担防止高温金属喷射、人员误触危险运动部件的作用；同时也是工艺顺序约束装置，强制保障压铸全流程各工步按照预设工艺逻辑有序执行 [[S14,S15,S12]]。

## 分类与类型

根据实现原理和物理层次，压铸互锁控制可区分为机械互锁、电气互锁、液压互锁和软件/逻辑互锁四类 [[S11,S25,S34]]。

| 互锁类型 | 典型实现形式 | 功能特征 |
| -------- | ------------ | -------- |
| 机械互锁 | 合模到位校验插销结构：动定模未完全贴合到位时，动模侧插销无法插入定模侧对应定位孔，直接限制后续压射动作触发 [[S11]]；带机械自锁功能的急停开关：触发后即使控制系统断电也会保持触发状态，需人工手动解除锁止 [[S15]]；移动熔炉的机械锁止装置：防止位移结构失效导致熔炉倾倒 [[S15]] | 物理层面直接限制动作执行，不受控制回路失效影响 |
| 电气互锁 | 安全门回路双刀限位开关：安全门未完全关闭时直接切断合模动作控制回路，符合4类安全等级要求 [[S25]]；合模与压射回路电气联锁：只有合模完全完成对应的位置电信号触发后，压射回路才允许通电执行 [[S43]]；联锁回路动作关联表：记录各工步的互锁逻辑映射关系 [[S22]] | 通过电信号与控制回路通断实现顺序约束 |
| 液压互锁 | 安全门机械触发结构驱动先导辅助液压阀：安全门打开时直接切断合模油缸的压力供油，不受主控制回路失效影响 [[S25]]；合模压力未达到预设锁模力阈值时，液压回路压力信号无法解锁压射油路 [[S15]]；立式压射机防护门下降未封闭到位时，直接切断压射油缸供油回路 [[S30]] | 在液压动力层面强制限制危险动作执行 |
| 软件/逻辑互锁 | 固态逻辑控制电路实现全工步顺序联锁：前一个动作未完成并通过位置、压力信号双重校验后，不允许启动下一个动作 [[S34]]；集成控制系统与周边设备协同互锁：任一周边设备位置/状态未到位时，主机不能启动后续对应工步 [[S12,S39]]；“记忆校验”联锁功能：自动循环启动前自动校验液压油温、润滑油位、各工位初始位置等全部前置条件 [[S34]] | 通过程序逻辑实现复杂的条件判断与动作约束 |

## 工艺作用与典型应用

互锁控制贯穿压铸生产的全流程，具体体现在以下关键环节中：

**合模与注射互锁**
合模动作必须完全到位、锁模力达标后才允许执行压射动作，防止在未锁紧状态下触发注射，避免熔融金属从分型面喷出造成安全事故。早期压铸机通过机械杠杆互锁实现，现代机型普遍采用电气联锁完成该控制要求 [[S24,S28,S8]]。

**安全门互锁**
安全门分为前安全门和后安全门两组控制逻辑。前安全门由多组限位开关配合限位杆检测，具备防夹功能；关到不到位时，对应限位开关触发锁模动作禁止信号，液压安全门控制回路断电，无法启动合模动作。后安全门配置2个限位开关，关门不到位时直接切断控制回路，所有压铸机动作被禁止 [[S4,S44]]。

**顶出与合模互锁**
压铸机顶出机构必须完全退回至初始位置后，控制系统才允许输出合模启动指令，防止动模上的顶杆未复位就合模造成模具碰撞损坏。在斜滑块侧向分型等复杂模具场景中，该互锁为强制要求的安全防护配置 [[S21]]。

**压铸机与辅助装置互锁**
压铸机与给汤机、喷雾机、取件机器人等辅助设备之间必须设置联动互锁机制，标准明确要求各辅助设备动作状态反馈给集成控制系统，任意辅助设备动作未完成或反馈异常时，压铸机无法启动下一循环动作 [[S12,S39]]。

**真空阀控制互 lock**
在真空压铸中，真空阀关闭响应时间需不大于100 ms，即从真空关闭信号发出到阀完全关闭的最大耗时不得超过该阈值，避免型腔内真空度不符合工艺要求影响铸件质量 [[S37]]。

## 实现方式与运行规则

压铸互锁的主流实现方式以可编程逻辑控制器（PLC）为控制核心，配合分布在合模机构、安全门、顶出机构、辅助设备末端的限位开关、位置传感器采集状态信号，通过梯形图逻辑实现各动作的顺序约束，同时联动对应液压阀、气动阀的控制回路。典型应用案例为力劲DCC280卧式冷室压铸机，采用OMRON CQM1系列PLC，互锁逻辑集成在合开型运动控制梯形图中 [[S2,S34]]。

互锁控制的关键运行规则如下：
- 手动/调整操作模式下，常规自动互锁功能可以部分旁路解除，但仅限模具安装调试场景，操作时必须多名操作人员确认安全后方可执行动作 [[S40,S27]]。
- 紧急停止功能要求在多个位置设置急停按钮，任意按钮触发后立即切断控制回路，所有电机停止运转，所有液压阀复位至初始位置，急停故障必须人工复位清除后才可重启设备 [[S40,S27,S43]]。
- 所有涉及人身、设备安全的互锁动作故障，必须人工干预确认修复后才能清除报警，不能自动复位；通信链路故障时控制系统必须立即进入安全模式停止自动运行 [[S39]]。

## 互锁失效与典型缺陷

**失效原因**
互锁失效的典型原因包括：安全门行程开关损坏或信号丢失导致锁模条件被破坏；顶出机构未回原位、机械手未回到指定位置时相关位置传感器故障；操作过程中人为短接互锁线路旁路安全防护；控制系统本身故障导致互锁逻辑失效 [[S20]]。

**工艺缺陷**
互锁失效可能引发的工艺缺陷包括：模具内部存在异物未清理导致压坏模具；未完全合模就启动压射，金属液从分型面喷出产生严重飞边；模具未完全冷却开模导致铸件冷隔缺陷，甚至模具型腔受力不均产生裂纹失效 [[S20,S38,S41]]。

**安全事故**
更为严重的后果是：人员在模具区域检修时互锁失效导致模具误动作夹伤甚至死亡；未锁死的合模机构高速动作撞击周边人员；高温熔融金属从分型面喷出造成操作人员灼伤 [[S45]]。

## 关联术语与系统

**互锁控制与安全光幕**
安全光幕是互锁控制的典型输入检测部件，当光幕光路被遮挡时直接触发互锁逻辑切断危险运动输出，是压铸机模具区域周边非接触式互锁的核心组件 [[S26]]。

**互锁控制与安全PLC**
安全PLC是压铸机互锁逻辑的核心运算执行载体，相比普通PLC具备双通道冗余、实时故障诊断功能，可满足ISO 13849 Category 4等级的互锁安全要求，替代传统继电器互锁实现更高可靠性的安全逻辑 [[S39,S7]]。

**互锁控制与顺序控制**
互锁控制是压铸机顺序控制的安全约束层，用于对自动生产流程中各工步的前置条件进行强制校验，不符合条件则直接终止后续工步执行，是保障压铸机循环动作时序安全的基础 [[S16,S42]]。

**互锁控制与防呆（Poka-yoke）**
压铸机的机械位置校验插销、行程开关、未放置嵌件时自动触发停机的检测机构都属于防呆类互锁装置，从物理层面实现人为操作失误的自动拦截，属于Poka-yoke原理在压铸安全防护中的典型应用 [[S11,S18]]。

**互锁控制与压铸机控制系统**
互锁是压铸机控制系统的底层安全功能模块，优先级高于所有手动、半自动、自动模式下的常规控制逻辑，出现互锁触发时所有非安全相关的输出立即被切断，进入安全状态 [[S39]]。
