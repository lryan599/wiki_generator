## 概述

压铸设备指用于压力铸造工艺的专用生产装备，属于广义铸造设备下的高压铸造子类。与服务砂型铸造、重力铸造等工艺的通用铸造设备不同，压铸设备专门面向高压高速熔融金属充型、高压下凝固的压铸工艺要求[[S3,S15]]。依据国家标准GB/T 37371-2019，压铸机被定义为“将熔融金属通过高压、高速压射入可重复使用的模具中的动力机器”，辅助设备则为“配合压铸生产的功能设备”，二者共同组成压铸单元整体[[S15]]。

压铸设备是压铸生产的核心装备，决定着压铸成形及产品质量。压铸工艺、压铸设备、压铸模三者相互依存，密不可分[[S7]]。

## 定义与范畴

在GB/T 37371-2019的定义体系中：

- **压铸单元**：压铸机与辅助设备形成的自动化生产成套装备[[S15]]。
- **压铸机**：将熔融金属通过高压、高速压射入可重复使用的模具中的动力机器[[S15]]。
- **辅助设备**：配合压铸生产的功能设备，包括浇注装置、取件传送装置、喷涂装置、模温控制装置、清理整修装置、镶件装置、检测装置、系统安全装置、集成控制装置、铸件冷却装置、真空装置等[[S15]]。

广义的压铸生产工装设备还包括熔化炉、保温炉、抛光机、光饰机、脱模剂配混装置等工艺设备，以及抽风系统、除尘系统等通用动力安全设施[[S3]]。

## 分类体系

压力铸造设备分为**压铸机**和**自动辅机**两大核心类别[[S4,S7,S29]]。

## 压铸机结构组成

压铸机主要由合模机构、压射机构、液压系统、电气控制系统、机身等核心部分组成[[S8]]。

### 合模机构

#### 曲肘式合模机构

曲肘式合模机构是当前使用最广泛的合模型式，尤其适用于中大型压铸机。其结构由三块座板（定模座板、动模座板、尾板）和四根拉杆串联构成框架，液压合模缸驱动曲肘连杆机构完成开合模动作[[S23,S26,S30]]。

核心工作原理：合模时，压力油进入合模缸推动活塞，通过连杆使三角形铰链绕支点摆动，将力传递给动模座板；当曲肘机构达到a、b、c三点共线的“死点”位置时，机构实现机械自锁锁模[[S12,S26,S30]]。

曲肘式合模机构的主要特点[[S5,S12,S28]]：

| 特性 | 说明 |
|------|------|
| 力放大 | 曲肘连杆系统可将合模缸推力放大16～26倍[[S23,S28]] |
| 运动特性 | 开合模速度呈慢→快→慢变化，合模临近闭合时速度降低，避免模具撞击[[S12,S23]] |
| 锁模可靠性 | 利用“死点”机械自锁，刚性大、可靠性高[[S30]] |
| 控制系统 | 控制简单，操作维修方便[[S12]] |
| 局限性 | 受热膨胀不易自动补偿，不同模具厚度调整较麻烦[[S5,S30]] |

#### 全液压式合模机构

全液压式合模机构直接依靠液压缸输出推力实现开合模和锁模，无需曲肘连杆结构[[S5,S26]]。其主要特点包括：

- 结构简单，安装不同厚度模具时易于调整[[S5]]；
- 可自动补偿模具受热膨胀的尺寸变化，无需频繁调整模具厚度参数[[S5]]；
- 早期多用于小型压铸机，大型机上因合模缓慢、动力消耗大、刚性差而应用受限[[S5,S28]]；
- 后续衍生出全液压复缸增压式结构，通过增压装置使锁模力达到管路压力的二倍以上，适配中大型机型[[S28]]。

### 压射机构

#### 热室压铸机压射单元

热室压铸机的压射缸及压室整体垂直布置，压室完全浸入保温坩埚内的熔融金属液中，鹅颈通道与压室连通实现自动给料[[S8,S16]]。主要特点[[S8,S20,S34]]：

- 压室中不存在空气，压射过程中卷入气体机会少，金属液氧化夹杂少；
- 一般不配备独立增压液压缸，增压效果不如冷室压铸机明显；
- 循环时间更短，生产效率更高（镁合金最高生产节拍可达260次/小时）[[S1]]；
- 常用压射比压范围7～14 MPa[[S20]]；
- 最大常规锁模力不超过400 t[[S34]]；
- 多用于锌合金及低熔点镁合金薄壁铸件生产。

#### 卧式冷室压铸机压射单元

卧式冷室压铸机的压射室与独立保温炉分离，水平布置；人工或机械手将定量金属液舀入压室后，由压射冲头推进完成充型[[S8,S16,S34]]。主要特点[[S8,S16,S34]]：

- 配备独立液压蓄能器和增压液压缸，可在充型完成后15～35 ms内建立增压压力至额定值[[S1]]；
- 压射比压可达50～70 MPa（普通件）甚至90 MPa以上（耐压件）[[S1,S10]]；
- 适用于铝、锌、镁、铜等各类合金，可生产厚壁及大尺寸铸件；
- 当前行业最大机型锁模力已达45000 kN以上，可生产一体化超大压铸件[[S34]]。

#### 立式冷室压铸机压射单元

压室中心线垂直布置，配置独立反料冲头结构。压射前反料冲头封住喷嘴孔，可有效避免金属液内卷入杂质，适配中心浇口类铸件生产；但结构复杂、操作维修不便，生产效率低于卧式冷室压铸机，目前市场应用占比很低[[S19,S32]]。

### 电气控制系统

现代压铸机的控制系统已相当完善，可接受和处理大量工艺信息，实现人机对话和实时控制。系统构成包括可编程序控制器（PLC）、工业计算机、各种监测传感器、伺服阀、比例阀、工艺软件等[[S8]]。控制系统控制整个压铸系统按要求的动作程序顺序运行，完成压铸循环[[S8]]。

## 核心技术参数

压铸机的性能与规格通过一系列技术参数来描述，以下为重点核心参数[[S21]]。

### 主要参数定义与说明

| 参数名称 | 定义 | 意义 |
|----------|------|------|
| 锁模力 | 开合模机构合模锁紧后使两个半模保持闭合的最大锁紧力[[S21]] | 压铸机首要参数（即行业所称“吨位”），用于抵消压射时的胀型力；不足会导致分型面溢料[[S21]] |
| 压射力 | 压射机构中推动压射活塞运动的力，Fy = pg × AD（pg为液压管路工作压力，AD为压射缸活塞截面积）[[S13]] | 决定金属液充填型腔的驱动力 |
| 压射比压 | 压射冲头作用于金属液的单位面积压力 | 冷室机普通件50～70 MPa，耐压件可至90 MPa；热室机常规7～14 MPa [[S10,S20]] |
| 拉杆内距 | 四根拉杆在水平和垂直方向的净间距[[S33]] | 直接限制可安装模具的最大外形尺寸[[S33]] |
| 模具厚度范围 | 曲肘式合模机构通过调整后座板位置获得的可适配模具最小至最大厚度区间[[S21,S33]] | 小于最小厚度模具无法闭合，大于最大厚度曲肘无法到达死点锁模[[S21]] |
| 动模座板行程 | 动模板在开合模时可移动的最大距离[[S21]] | 决定开模时分型面距离，需保证铸件顺利取出[[S21]] |
| 压射速度 | 分慢压射（0.1～0.5 m/s）和快压射（0.1～5 m/s）两个阶段[[S10]] | 直接影响金属液填充形态和铸件质量 |
| 增压建压时间 | 型腔填充完成后建立增压压力至额定值的时间，要求15～35 ms之内[[S1]] | 用于压实凝固中的金属液，减少气孔缩孔缺陷 |
| 最大射料量 | 一次允许浇入压射室的最大金属液质量[[S25]] | 影响单次可压铸的最大铸件重量 |
| 空循环时间 | 压铸机按运行顺序完成一个空循环所需的时间[[S25]] | 反映设备的基本运行效率 |

## 工艺角色

压铸设备在压铸全流程中承担各工序核心角色：

1. **熔炼与保温**：熔化炉和保温炉为压铸生产提供合格的金属熔体[[S3]]；
2. **给汤**：保温浇注炉或定量浇注机将定量金属液输送至压射室[[S15]]；
3. **压射与保压**：压铸机压射机构将金属液高压高速注入模具型腔并保压凝固[[S8]]；
4. **开模取件**：合模机构打开模具，取出机械手将铸件从模具取出并传送至指定位置[[S15]]；
5. **喷涂**：喷涂系统将脱模剂雾化后定点定量喷涂在模具型腔表面[[S15]]；
6. **切边与清理**：切边压力机去除浇注系统、溢流系统及飞边毛刺[[S15,S4]]；
7. **模温控制**：模温控制装置通过介质循环调节模具温度，维持工艺窗口稳定[[S15]]。

## 镁合金冷/热室压铸机工艺对比

以镁合金为对象的冷、热室压铸工艺存在多项显著差异，对比表如下[[S8,S24]]：

| 对比维度 | 热室压铸机 | 冷室压铸机 |
|----------|-----------|-----------|
| 增压结构 | 无 | 配备液压蓄能器和增压缸 |
| 适用合金范围 | 锌、镁（低熔点） | 铝、镁、锌、铜 |
| 压射比压 | 较低，7～14 MPa | 较高，50～70 MPa |
| 循环效率 | 更高（最高260次/小时）[[S1]] | 相对较低 |
| 铸件适用性 | 薄壁中小型件 | 厚壁、大尺寸结构件 |
| 金属液质量 | 氧化夹杂少，温度波动小 | 氧化夹杂机会较大 |
