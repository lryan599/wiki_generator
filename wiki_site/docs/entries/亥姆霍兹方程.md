## 概述

亥姆霍兹方程（Helmholtz equation）是以德国物理学家赫尔曼·冯·亥姆霍兹命名的一类椭圆型偏微分方程[[S9]]。在声学与压铸超声仿真中，它是描述单频稳态声场空间分布的基本控制方程，通过从全时域声学波动方程引入时谐谐振假设、分离时空变量后得到[[S5,S21]]。其标准齐次常密度形式为 ∇²p + k²p = 0（k = ω/c），而考虑金属熔体密度空间不均时推广为 ∇·(1/ρ ∇p) + (ω²/(ρc²))p = 0 [[S2,S5,S21]]。

由于推导中消去了时间导数，亥姆霍兹方程属于频域稳态模型，在压铸与材料加工中主要用于超声净化、超声变质及超声辅助铸造工艺的声压分布计算、空化区域预测与夹杂物去除效率分析[[S11,S15]]。

## 定义与物理形式

### 标准常密度亥姆霍兹方程

在全时域线性声学波动方程中，假设声场按谐波规律变化 P(r,t) = p(r) e^(iωt)，代入后消去时间变量，得到空间部分满足的亥姆霍兹方程[[S5,S21]]。当介质密度 ρ 为常数时，方程为

\[
\nabla^{2}p + k^{2}p = 0 \qquad (k = \frac{\omega}{c})
\]

其中 k 为波数，ω 为角频率，c 为声速[[S9]]。该形式适用于密度全域均匀、声速恒定的理想条件。

### 变密度广义亥姆霍兹方程

在压铸超声仿真中，熔体内部往往存在温度梯度或溶质浓度梯度，导致密度 ρ 的空间分布不再均匀[[S2,S5]]。此时必须采用非均匀介质下的广义形式：

\[
\nabla\!\left(\frac{1}{\rho}\nabla p\right) + \frac{\omega^{2}}{\rho c^{2}}\,p = 0
\]

当还需考虑超声在熔体中的衰减时，可进一步引入复数密度 ρ_c 和复数声速 c_c，形成复数形式的亥姆霍兹方程[[S1]]：

\[
\nabla\!\left(\frac{1}{\rho_c}\nabla p\right) + \frac{\omega^{2}}{\rho_c\,c_c^{2}}\,p = 0
\]

用户提示中出现的 “∇(1ρ∇P)+ω2ρc2p=0” 实为排版省略分数线与上标的笔误，其正确的符号表达式应为上述标准广义变密度形式，即 “1ρ” 为 “1/ρ”，“ω2ρc2” 为 “ω²/(ρc²)” [[S11,S9,S24]]。

### 参数的物理意义

| 符号 | 物理意义 | 单位 |
|------|----------|------|
| p   | 声压幅值 | MPa 或 Pa |
| ρ   | 介质密度 | kg/m³ |
| c   | 声波在介质中的传播速度 | m/s |
| ω   | 角频率（ω = 2πf） | rad/s |
| k   | 波数（k = ω/c） | m⁻¹ |
| ρ_c, c_c | 复数等效密度与声速（含衰减系数） | 同相应实数单位 |

表中关键参数涵义均基于声学背景和压铸熔体的界定[[S5,S11,S15,S1]]。

### 模型的求解途径

在压铸领域，亥姆霍兹方程通常结合有限元法（如 COMSOL Multiphysics）进行求解，需设定边界条件：熔体自由液面为软声场（p=0）、工具杆端面给定声压幅值 p₀、变幅杆侧面及坩埚壁为硬声场（∂p/∂n=0）、坩埚-熔体界面为阻抗边界条件[[S22,S2,S19]]。自由场情形下也可运用亥姆霍兹‑基尔霍夫积分定理进行解析或半解析计算[[S22]]。

## 数学模型分类与相关方程

亥姆霍兹方程属于**椭圆型偏微分方程**、**稳态波动方程**。按照二阶偏微分方程的标准分类 Au_xx + B u_xy + C u_yy = f，该方程对应 B = 0、A = C = 1，判别式 B²−4AC = −4 < 0，不存在实特征线，故归入椭圆型[[S13]]。

与邻近方程的关系如下：

- **拉普拉斯方程 ∇²p = 0**：当亥姆霍兹方程中 ω = 0（即 k=0）时退化为拉普拉斯方程[[S9]]。
- **泊松方程 ∇²p = f**：有源非齐次情形下亥姆霍兹方程可化为泊松方程[[S10]]。
- **全时域波动方程（达朗贝尔方程）**：保留时间二阶导数项，属于双曲型偏微分方程，可求解瞬态多频波动问题。亥姆霍兹方程将该时间导数消去，转换为频域单频稳态形式[[S9,S7]]。

亥姆霍兹方程在声学、电磁学、量子力学中有广泛用途，但本词条重点聚焦其在压铸与材料加工领域的超声仿真应用[[S9]]。

## 方程形式对比

| 项目 | 标准常密度亥姆霍兹方程 | 变密度/复数广义亥姆霍兹方程 |
|------|----------------------|---------------------------|
| 方程形式 | ∇²p + k²p = 0 | ∇·(1/ρ ∇p) + (ω²/(ρc²))p = 0 或复数版本 ∇·(1/ρ_c ∇p) + (ω²/(ρ_c c_c²))p = 0 |
| 密度假设 | 全域恒定实数密度 ρ = const | 密度可随空间变化，必要时以复数量 ρ_c 包含衰减 |
| 声速假设 | 全域恒定实数声速 c = const | 声速可随空间变化，必要时以复数声速 c_c 包含衰减 |
| 衰减处理 | 忽略超声衰减 | 可引入衰减系数 α，通过复数波数 k_c = ω/c − iα 描述 |
| 适用熔体条件 | 低功率、弱空化、温度波动 <20 K 的均值熔体 | 中高功率、存在温度/浓度梯度的大体积熔体，衰减不可忽略 |
| 求解精度等级 | 工程级快速预测 | 高精度科研仿真级，计算复杂度较高 |
| 典型应用场景 | 均质模壁超声或小容量坩埚内部的初步声场评估 | 半连续铸造镁/铝合金熔体、梯度密度层、高功率空化区声场精确模拟 |

该对比表从控制方程、假设条件与适用场景等维度总结了二者区别，引用依据来自标准形式与各类压铸超声仿真文献[[S9,S21,S1,S2]]。

## 压铸与材料加工中的应用

### 镁合金超声净化中的声压仿真

SHAO Z W 等针对 AZ80 镁合金中 MgO 夹杂的超声净化问题，引入亥姆霍兹方程进行声压仿真[[S11,S15]]。其数值求解采用方程

\[
\nabla\!\left(\frac{1}{\rho}\nabla p\right) + \frac{\omega^{2}}{\rho c^{2}}\,p = 0
\]

并结合轴向和横向时均声辐射力公式，定量预测了杂质颗粒在驻波场中受到的声辐射力分布。在 80 W 超声功率下，计算结果形成了明显的驻波场结构，轴向声辐射力使悬浮的 MgO 粒子向声压节点平面聚集，横向辐射力与颗粒间次级辐射力则促使粒子在节点面内进一步集中[[S11,S15]]。该研究证实，变密度广义亥姆霍兹方程比常密度形式更能精确捕捉声压幅值与驻波节点位置，这对预测夹杂物迁移行为至关重要[[S11]]。

### 空化区域的预测

通过亥姆霍兹方程求解得到的声压分布，可直接划定有效空化区域。对于 AZ80 镁合金或相近的镁合金熔体，定义声压幅值超过 0.13 MPa 的区域为有效空化区[[S19]]。有限元法求解时，综合考虑坩埚壁反射等约束边界后，声压分布不再是单调衰减，而是在轴线上出现最小值后再升高的复杂结构[[S2]]。在不同超声功率（200 W、500 W、940 W、1360 W）下，工具杆端面声压幅值分别达到 1.082 MPa、1.710 MPa、2.345 MPa、2.821 MPa，有效空化深度依次为 19 mm、25 mm、30 mm、34 mm[[S19]]。下图直观展示了 1360 W 功率下镁合金熔体内的声压分布云图及声压等位线。

### 双频与多频超声的空化叠加

NING S C 等基于亥姆霍兹方程的声压模拟进一步研究了双频超声效果。15 kHz 单频超声的最大空化面积约为 20 kHz 的两倍；而双频异频超声同时施加时，空化面积大于两次单频超声的总和，归因于两个异频声场的非线性叠加诱发了新的空化区域[[S11,S12]]。这一发现为通过频率组合优化晶粒细化效率提供了理论依据。

### 铝合金与模壁超声振动

在模壁超声压铸场景中，利用亥姆霍兹方程计算 AlSi9 合金型腔内部声压可达 3–8 MPa，远高于铝合金约 1 MPa 的空化阈值[[S23]]。同时，对于 2A14 铝合金熔体，实验表明在 6 μm 的优化超声振幅下，夹杂物平均面积占比可从 2.088% 降低至 0.097%，除杂效率超过 95%[[S14,S16]]。这些工艺参数与声压仿真结果共同支撑了亥姆霍兹方程在压铸超声工艺参数匹配中的核心作用。

### 工艺参数关联

在超声辅助铸造中，亥姆霍兹方程将超声频率 f、功率（影响端面声压幅值）、熔体密度 ρ、声速 c 以及衰减特性等工艺参数直接嵌入声场计算，使声压分布、声能密度和空化区域成为可定量的工艺设计指标[[S18,S19]]。例如，在 360 W、720 W、1080 W 三档功率下，AlSi9Cu3 和 AlSi18 铝合金熔体的计算声压均超过空化阈值，为频率和功率的选型提供了数据支撑[[S6]]。

## 局限性

尽管亥姆霍兹方程在压铸超声仿真中应用广泛，但存在多方面的内在限制：

1. **时谐稳态假设**：该方程仅适用于连续单频正弦声场，无法描述瞬态脉冲超声的动态声压演化[[S5,S21]]。
2. **线性声学假设**：推导建立在微小振幅的线性声学基础上，不能处理空化泡生长、溃灭等强非线性过程[[S11,S21]]。
3. **多物理场耦合缺失**：推导时通常假设熔体密度与声速全域恒定或仅按空间分布变化，忽略温度场‑流场‑声场的实时耦合以及参数动态波动，耦合效应显著时仿真误差明显增加[[S20,S1]]。
4. **密度均匀性假设失效**：在高功率强空化区域，大量空化泡的存在导致局部熔体密度出现显著梯度，此时即便采用变密度方程，实际声压分布与仿真结果的偏差仍可能超过 20%[[S11]]。
5. **单频局限**：对于多频复合激励或宽带噪声信号，单一频率的亥姆霍兹方程无法直接叠加使用，需进行复杂的频域合成或切换到瞬态模型[[S21]]。

这些局限表明，在利用亥姆霍兹方程进行压铸超声工艺设计时，需明确其适用范围，并在必要时耦合多物理场、引入非线性修正或采用瞬态波动方程以提升预测精度。
