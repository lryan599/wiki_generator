## 概述

在压铸领域，行业内俗称的“加砂机构”属于压铸单元辅助设备下的镶件装置大类，是专门针对砂芯这类特殊铸入镶件配置的专用子装置[[S12,S14]]。在国家标准铸造机械术语中，其对应设备被正式定义为**下芯机**（core setter），属于压铸单元配套的自动化辅助设备，功能是将预制砂芯精准送入压铸模型腔完成砂芯加装作业[[S10,S26]]。

## 术语与别名

### 中文别名

自动加砂芯机构、砂芯上料机构、装芯机构、砂芯置入装置均为压铸行业内通用的工程俗称，其中“装芯机构”是压铸生产现场使用频率最高的通用叫法，所有上述别名均指向同一类砂芯自动置入功能装置[[S14,S9]]。

### 英文对应术语

- **正式表达**：core setting device、core loading unit 是对应该类装置最常用的正式英文表达[[S3]]。
- **非通用别名**：sand core loading mechanism、core insertion unit、automatic core feeder、sand core setting device、core pick-and-place unit、sand core handling system、automatic core setting machine 均为不同厂商文献衍生出的非通用别名[[S13]]。

> **注**：公开压铸行业标准中未为该砂芯专用装置单独设立独立术语，统一归类在镶件装置范畴[[S12]]。

## 工艺作用

加砂机构在完整压铸生产循环中的执行时序处于喷涂、模具清理工序完成之后，向模内置入砂芯，随后进入动模插芯、合模工序[[S22,S11]]。其动作需与压铸机主控系统设置安全互锁，完成砂芯置入后向压铸机反馈就绪信号，方可启动后续合模流程，避免砂芯放置不到位导致压射失效、模具损坏[[S23]]。

## 系统构成

压铸场景下的自动加砂芯（下芯）机构核心组成分为四大层级模块[[S7,S2]]：

| 模块层级 | 主要组成 | 关键特征 |
|---------|---------|---------|
| ① 定量供砂/射砂单元 | 砂斗、专用闸板机构、射砂筒、带防漏密封结构的非直通式射嘴、定量加砂装置 | 闸板厚度6 mm，闸板与导槽间隙控制0.5~1 mm；储砂量取所射砂芯重量的2.5~3倍可获得最优充填效果 |
| ② 移动输送单元 | 导轨/辊轮副、带缓冲功能的驱动组件（气缸/伺服电机）、导向定位杆 | 采用双160 mm直径导杆保证移动同轴度与垂直度 |
| ③ 砂芯抓取单元 | 适配不同砂芯外形的专用夹具 | 重载机型可承载50 kg及以上重量的大型复杂砂芯 |
| ④ 控制与感知单元 | PLC主控制器、位置传感器、压力检测元件 | 支持全自动、单循环、手动三种操作模式 |

## 分类

国家标准GB/T 25370-2020将压铸砂芯投放设备定义为下芯机，属于压铸辅助设备大类，其通用分类可按以下三个维度划分[[S1,S2,S21]]：

**按驱动源类型**：
- 气动驱动型
- 电动伺服驱动型
- 液压驱动型

不同驱动形式分别适配不同负载与定位精度等级。

**按安装形式**：
- 固定式加砂机构
- 工业机器人搭载式加砂机构
- 龙门式加砂机构

**按适配砂芯范围**：
- 小型通用砂芯机型
- 中大型复杂砂芯机型

适配的砂芯重量覆盖3 g~50 kg以上区间。

## 应用场景

加砂机构典型适配压铸件类型为汽车发动机缸盖、缸体等带有复杂冷却水套、气道内腔的高复杂度带砂芯铝合金铸件[[S25]]，同时支持适配镁合金、锌合金材质的带深腔复杂内腔压铸件，符合压铸行业常规主流合金品类的生产配套需求[[S8]]。

## 优势与生产效益

实际生产验证数据显示，人工放置7件总重量达35 kg的铝合金缸盖砂芯单工序耗时58秒，会导致模具温度下降约20 ℃，需将铝液浇注温度从710 ℃提升至730 ℃，易引发铸件缩孔、渣孔、漏气等缺陷，且人工搬运单重22 kg的大型砂芯存在安全隐患[[S4]]。采用压铸自动加砂机构替代人工放芯，可避免上述问题，大幅提升生产节拍，降低铸件不良率，减少人力投入与作业安全风险[[S4]]。
