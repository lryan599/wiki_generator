## 概述

Al-5Ti-1B中间合金（亦称AlTi5B1、铝钛硼合金）是一种铝基功能型晶粒细化中间合金，也是目前全球应用最广泛的铝合金晶粒细化剂。在Al-Ti-B系列晶粒细化中间合金市场中，其使用量占比超过65%，属于Ti/B>2.2的Al-Ti-B系列晶粒细化剂类别[[S7,S20,S51]]。典型的标称成分为Ti约5wt%、B约1wt%、Ti/B质量比约5:1，余量为铝，对应的国内工业牌号为AlTi5B1[[S10]]。其以线材或锭块形式供货，通过向铝及铝合金熔体中引入异质形核核心来实现晶粒细化，核心适用于工业纯铝及合金元素含量较低的铝合金凝固细化场景[[S51]]。国外经典商用牌号包括TIBOR（如"CRE Al-5Ti-1B"）[[S4,S52]]（注："CRE"前缀可能为企业特定代号，待进一步验证）。

### 核心相组成

Al-5Ti-1B中间合金的物相组成主要包括α-Al基体、TiAl₃相和TiB₂相[[S46,S47,S2]]。

- **TiAl₃相**：多以块状分布在α-Al基体中，平均粒度为30μm×50μm，95%的TiAl₃相粒度小于150μm，最大粒度不超过200μm[[S2,S26]]。其形态随熔炼温度与冷却条件变化，可分为板状、片状、块状和花瓣状等多种形貌[[S47]]。
- **TiB₂相**：为六方晶型，晶格常数a=0.3030nm、c=0.3232nm[[S38]]。90%以上的TiB₂相粒度小于2μm，弥散分布在晶界处[[S2,S46]]。TiB₂颗粒的理论晶体形态为规则六角片状[[S25]]。

中间合金中还可能含有(Al₁₋ₓ,Tiₓ)B₂三元硼化物，它是AlB₂到TiB₂的连续固溶体，在高温保温过程中可逐步转变为更稳定的TiB₂[[S38]]。

## 细化机理

Al-5Ti-1B中间合金的晶粒细化效果是多种机制协同作用的结果。目前认可度较高的机理包括复相生核理论和生长限制因子（GRF）理论。

### 复相生核理论

Al-5Ti-1B加入铝熔体后，TiAl₃相迅速溶解释放出游离Ti原子，而TiB₂因熔点高达2980℃在熔体中稳定存在[[S27,S14]]。Ti原子向TiB₂颗粒表面迁移富集，析出TiAl₃薄层[[S27,S49,S35]]。TiAl₃与α-Al之间存在良好的共格取向关系，当(001)TiAl₃ // (001)Al、[100]TiAl₃ // [110]Al时，TiAl₃可直接作为α-Al的异质形核核心，在665℃发生包晶反应L + TiAl₃ → α-Al，从而实现晶粒细化[[S14]]。

单独向铝熔体中添加TiB₂颗粒并不能实现有效晶粒细化，只有熔体中存在富余Ti的条件下才能发挥形核作用，这进一步证实了复相生核机制的主导性[[S14,S27]]。

### 生长限制因子（GRF）理论

Easton与StJohn提出的生长限制因子理论指出，Al-5Ti-1B溶解后释放的溶质Ti元素在凝固界面前沿产生强烈的成分过冷，限制α-Al晶粒的自由生长，从而协同提升细化效果[[S31]]。

### TiAl₃直接形核机制

在Ti含量超过0.15%的条件下，TiAl₃相可直接作为α-Al的异质形核核心，通过包晶反应实现晶粒细化。微量硼的存在可使包晶反应点大幅度左移，硼含量约0.001%时，Ti含量仅需0.026%即可发生包晶反应[[S14,S55]]。

## 工艺参数与操作条件

### 推荐添加量

工业上Al-5Ti-1B中间合金的常规推荐添加质量分数范围为0.1%~0.3%，对应成品铝合金中总Ti含量控制在0.005%~0.02%区间[[S45,S14,S18]]。每吨铝熔体常规添加量为0.5~2kg Al-5Ti-1B即可获得良好细化效果[[S18]]。针对A356半固态铝合金，最佳添加质量分数为0.2%~0.3%，此时晶粒当量直径可降至67~68μm，形状因子最高约0.6[[S14]]。

不同Al-5Ti-1B添加量对A356合金晶粒组织参数的影响如下表所示[[S32]]：

| Al-5Ti-1B添加量（wt%） | 晶粒当量直径（μm） | 形状因子 |
|---|---|---|
| 0 | 74.51 | 0.517 |
| 0.1 | 77.12 | 0.536 |
| 0.2 | 68.91 | 0.663 |
| 0.3 | 67.42 | 0.549 |
| 0.4 | 72.81 | 0.643 |

### 加入温度与保持时间

Al-5Ti-1B的最佳加入温度区间通常为720~760℃，此温度区间既可保证中间合金中TiAl₃相快速充分溶解，又可避免温度过高导致TiB₂颗粒团聚沉降速度过快[[S44]]。线材添加点的温度建议控制在700~730℃[[S30]]。

晶粒细化效果随保温时间的变化呈现先升后降的规律：加入后2~30min区间细化效果达到峰值，超过30min后晶粒尺寸开始逐步增大，出现明显衰退现象[[S55,S33,S44]]。部分无改性的普通Al-5Ti-1B在保温超过400min后细化作用几乎完全消失[[S55]]。不同产品形态的有效作用时长有所不同[[S53]]：

- φ9.5mm杆料：加入后10~15min达到最佳效果，有效作用时长约30min；
- 锭块状产品：加入后10~20min即可开始浇注，6h内可保持较强细化效果；
- 含稀土改性的Al-Ti-B-RE块状产品：搅拌均匀后静置15min以上即可浇注，16h保温后仍可保持显著细化效果。

### 搅拌要求

Al-5Ti-1B加入铝熔体后必须进行彻底搅拌，保证TiAl₃和TiB₂颗粒在熔体中均匀分散。若后续连续铸造总时长超过3h，需要对熔体重新搅拌，使已经团聚沉降的TiB₂颗粒重新悬浮，恢复其形核能力[[S40]]。

## 细化衰退与对策

### 衰退原因

Al-5Ti-1B细化衰退的两大核心原因[[S1,S55]]：

1. 高温保温环境下熔体中TiAl₃相逐渐溶解消失，可参与包晶反应的有效异质形核核心数量减少；
2. 密度远大于铝的TiB₂颗粒（密度约4520kg/m³）长时间静置后大量团聚、沉降至炉底，无法继续悬浮在熔体中发挥形核作用。

### 二次细化补救策略

针对细化衰退的补救措施包括[[S37,S24]]：

1. **熔体搅拌**：对长时间保温的熔体进行充分搅拌，使沉降聚集的TiB₂颗粒重新悬浮在熔体中，即可重新恢复部分细化效果；
2. **补充添加**：按工艺要求少量补充添加Al-5Ti-1B中间合金以补充损失的有效形核核心；
3. **稀土改性细化剂**：优先选用添加镧等稀土元素改性的Al-Ti-B细化剂，稀土可改善铝熔体对TiB₂的润湿性，抑制TiB₂团聚沉降，大幅延缓细化衰退进程。

## 适用合金体系与应用

### 铸造铝合金

Al-5Ti-1B可有效用于A356、ZL101、A357等亚共晶Al-Si系铸造铝合金[[S14]]。针对A356合金，最佳添加量为0.2%~0.3%，可将粗大树枝晶转变为接近球形的等轴晶[[S14]]。对A356.0合金，Al-5Ti-1B可降低α-Al等效直径与二次枝晶距，促进针片状AlSiFeMg相转变为危害更低的汉字状富铁相，合金屈服强度最高可提升7.7%[[S22]]。

ZL114A是GB/T1173-1995标准中的亚共晶Al-Si-Mg系高强度铸造铝合金，本身名义Ti元素含量为0.10%~0.20%，与Al-5Ti-1B的添加要求匹配。通过添加Al-5Ti-1B可解决ZL114A铸造过程中容易出现的组织粗大问题，辅助其稳定达到T6状态下Rm≥320MPa、Rp0.2≥280MPa、断后伸长率A≥4.5%的高性能指标要求[[S50]]。

### 变形铝合金

Al-5Ti-1B可有效作用于6082等6系变形铝合金和Al-6Mg等5系变形铝合金场景。其可在不改变合金相组成的前提下细化粗大富铁相，降低富铁相的平均尺寸，间接优化变形加工性能[[S22]]。

### 压铸铝合金

Al-5Ti-1B可应用于ADC12等压铸铝合金的晶粒细化场景。针对ADC12合金，3%质量分数的Al-5Ti-1B为最优添加量，可将粗大的一次结晶相α₁-Al从原始15μm细化至5μm，原本尺寸约10μm的α-AlFeMnSi粗大针状相可细化至1μm左右，合金布氏硬度可达110HBW、抗拉强度290MPa、屈服强度179MPa、伸长率6.2%[[S46,S54,S13]]。当添加量超过5%时细化效果不再提升，且会析出粗大针状Al₃Ti相，劣化材料力学性能[[S13]]。

## 与其它细化剂的比较

### 性能对比

相同1.5%质量分数添加条件下，针对免热处理Al-8Si-0.4Mn合金，Al-3Ti-1B可使晶粒尺寸较母材降低77.14%，Al-5Ti-1B可使晶粒尺寸较母材降低73.57%，二者细化效率略低于含TiC相的Al-Ti-C类细化剂（晶粒尺寸降低90%）[[S18]]。TiC相的单位形核效率为TiB₂相的100倍，且Al-Ti-C类细化剂不存在TiB₂颗粒团聚问题，可减少轧制、挤压加工过程中出现的表面划痕、铝箔针孔等缺陷[[S18,S43]]。

在AA1050纯铝中，AlTi5B1（即Al-5Ti-1B）与AlTi3B1的晶粒细化性能无显著差异[[S3,S23]]。

### 成本对比

不同晶粒细化剂的行业公开成本数据如下[[S36]]：

|---|---|
| Al-3Ti-1B | 5700 |
| Al-5Ti-1B | 5900 |

Al-Ti-C类细化剂整体成本高于常规Al-Ti-B产品[[S36]]。

### 选用逻辑

- **Al-5Ti-1B**：更适配低合金含量、无Zr等毒化元素的常规铝合金铸轧、压铸生产场景，综合性价比最高[[S51,S31]]；
- **Al-3Ti-1B**：更适配残留Ti含量较高的再生铝合金生产场景[[S31]]；
- **Al-Ti-C类细化剂**：专门适配含Zr等会毒化Al-Ti-B细化效果的特殊铝合金体系[[S51,S31]]；
- **Ti/B<2.2的低钛高硼类细化剂（如Al-3Ti-3B）**：适配高硅含量铝合金的细化需求[[S51]]；
- **高回炉料占比的生产场景**：优先选用低B含量的Al-Ti-B产品，避免TiB₂颗粒大量团聚沉降[[S8]]。

## 质量控制与常见缺陷

### 显微组织要求

国际和国内Al-5Ti-1B中间合金质量标准对显微组织的要求[[S25,S2,S26]]：

| 评价项目 | 控制要求 |
|---|---|
| TiB₂粒子尺寸 | ≤2μm的占比需达到90%以上 |
| TiB₂粒子分布 | 大致均匀弥散 |
| TiB₂疏松团块 | 尺寸≤25μm |
| TiB₂密集型团块 | 不允许存在 |
| TiAl₃相尺寸 | 95%的TiAl₃粒度<150μm，最大≤200μm |
| TiAl₃相平均尺寸 | 约30μm×50μm |

### 洁净度检测

中间合金洁净度检测方法包含[[S3]]：将少量细化剂样品溶解于丁醇，使用0.2μm孔径滤膜收集未溶解的金属间颗粒，结合X射线衍射识别物相，通过离心法和光散射法测定TiB₂颗粒的尺寸分布。

### 常见缺陷

**TiB₂团聚与硬质点**：TiB₂高硬度、高熔点特性以及团聚现象是硬质点夹杂缺陷的核心诱因。该缺陷会堵塞熔体过滤器、降低轧辊使用寿命，还会导致铝箔产品穿孔，大幅提升废品率[[S25,S48,S55]]。TiB₂团聚体尺寸大于5μm即判定为缺陷[[S28]]。

TiB₂团聚体形成的三种主流机理[[S28,S29]]：
1. 碰撞吸附熔体中的氧化铝膜形成大尺寸复合团聚体；
2. 氟盐（如反应残留的氟铝酸钾）在TiB₂颗粒表面形成薄层粘接相邻颗粒；
3. TiAl₃相涂层在颗粒碰撞时的粘附作用。

**细化不足**：典型诱因包括[[S48,S43,S14,S46]]：
- 铝熔体中存在Zr、Mn、Cr等元素引发细化剂中毒；
- 熔体长时间高温保温使TiAl₃和TiB₂发生沉降聚集导致细化效果衰退；
- 高硅铝合金中硅元素与游离Ti反应发生硅毒化效应，消耗有效形核核心。

**晶粒大小不均**：主要诱因是Al-5Ti-1B添加后在熔体中扩散不充分，以及随保温时间延长形核核心数量逐步衰减导致铸造过程不同时间段细化效果不一致。可通过在线铝钛硼丝逆熔体流动方向添加、控制添加温度在700~730℃、添加后熔体停留时间不超过30min的方式解决[[S21,S30,S43]]。
