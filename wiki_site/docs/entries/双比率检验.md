## 概述

双比率检验（Two-Proportion Test）亦称两样本比率检验或双比例检验，是工业统计领域用于对比两个独立二项分布样本总体比率的专项假设检验工具。在压铸质量改进场景中，该检验被广泛用于验证工艺调整、模具优化或生产方式变更前后缺陷率（如冷隔率、泄漏率、裂纹率）差异的统计显著性，为是否将改进措施固化为标准工艺提供决策依据[[S6,S8,S17]]。

双比率检验的核心输出字段包括：两组样本的缺陷事件数 X、总样本量 N、样本统计比率 p、两总体比率差 p₁ − p₂ 的点估计值、95% 置信水平下的比率差置信区间、Z 检验统计量数值、双侧 P 值，同时配套输出 Fisher 精确检验的 P 值作为小样本场景下的补充校验结果[[S6,S10]]。

## 适用条件

双比率检验的严格适用条件包括[[S12]]：

1. **独立样本**：两组抽样样本必须完全独立，两组生产过程或试验条件之间不存在交叉影响。
2. **二项分布**：每组样本的观测属性数据服从二项分布，即每个观测单元的结果仅有两类互斥状态（如合格/缺陷、泄漏/不泄漏）。
3. **大样本近似**：大样本场景下，单样本的阳性事件数 X 与阴性事件数 N−X 均不小于 5 时，可基于大样本正态近似原理计算 Z 检验统计量。

当样本量较小或大样本近似条件不满足时，应以 Fisher 精确检验的 P 值作为补充判断依据，其结论比正态近似 Z 检验更为保守稳健[[S6]]。

## 检验统计量与公式

设两个独立样本分别为样本 1 与样本 2，样本量为 n₁ 与 n₂，阳性事件数为 x₁ 与 x₂，样本比率分别为：

\[
\hat{p}_1 = \frac{x_1}{n_1}, \quad \hat{p}_2 = \frac{x_2}{n_2}
\]

原假设 H₀：p₁ = p₂（两总体比率无差异）；备择假设 H₁：p₁ ≠ p₂（两总体比率存在差异）。

在大样本条件下，合并比率估计值为：

\[
\hat{p} = \frac{x_1 + x_2}{n_1 + n_2}
\]

检验统计量 Z 的计算公式为：

\[
Z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})\left(\frac{1}{n_1} + \frac{1}{n_2}\right)}}
\]

两总体比率差的 95% 置信区间为：

\[
(\hat{p}_1 - \hat{p}_2) \pm z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}
\]

下表为双比率检验的标准化输出结构示例[[S14]]：

| 输出参数 | 示例值 |
|----------|--------|
| 两总体比率差 p(1)−p(2) 点估计 | 0.0224922 |
| 差值的 95% 置信区间 | (0.00794675, 0.0370376) |
| Z 检验统计量 | 3.03 |
| Z 检验 P 值 | 0.002 |
| Fisher 精确检验 P 值 | 0.003 |

## 与其他检验方法的边界与联系

### 明确区分

双比率检验仅适用于对比两组独立二项分布样本的总体比率差异。在压铸质量分析实践中，需注意以下区分[[S6,S7,S17]]：

- **单比率检验**：仅用于单总体比率与给定参考阈值的差异判断（如验证某模具的缺陷率是否超过目标值 2%），不涉及两组的对比，不属于双比率检验范畴。
- **两独立样本 t 检验**：适用于连续型均值数据的组间对比（如比较两种工艺下铸件抗拉强度的均值差异），不适用于比率/比例型数据。
- **卡方检验（R×C 列联表）**：当涉及 3 组及以上分类属性数据的关联性分析时（如比较 3 种模具冷却方案的缺陷率差异），应采用卡方检验，不属于双比率检验的适用范围。

### 与 2×2 列联表卡方检验的关系

双比率检验与 2×2 列联表卡方检验存在内在数学联系：在大样本场景下，双比率检验的 Z 统计量的平方值与自由度为 1 的 2×2 列联表卡方统计量完全等价，二者计算得到的双侧 P 值完全一致。但在实用性上存在差异[[S6,S17]]：

- **双比率检验**可直接输出两组比率的差值点估计与连续型置信区间，支持单侧检验，针对性地回答“两组比率差值是否存在显著差异”的问题。
- **2×2 列联表卡方检验**更适合多维度分类变量间的关联性检验，但不直接提供比率差值的量化估计结果。

在压铸工艺因子验证中，当需要同时获取“差异是否显著”与“差异的幅度估计及置信区间”时，优先采用双比率检验。

## 在压铸质量改进中的应用

双比率检验在压铸缺陷分析与工艺优化中已有广泛的工程应用。以下案例均源自实际压铸生产中的质量改进项目。

### 缺陷因子显著性验证

在因子筛选阶段，双比率检验用于逐个验证可疑因子对缺陷率的影响是否具有统计显著性。

**压铸缸体冷隔缺陷分析**[[S16]]：

- **模具冷却结构验证**：对比取消缺陷部位 4 组冷却水前后的冷隔缺陷率。样本 1（原工艺）冷隔数 78，总生产数 5069，缺陷率 1.54%；样本 2（取消 4 组冷却水）冷隔数 25，总生产数 2586，缺陷率 0.97%。Z = 2.21，P = 0.027 < 0.05，判定模具冷却结构对冷隔缺陷有显著影响，取消对应部位冷却水可显著降低冷隔率。
- **铝液温度验证**：对比 670 ℃ 与 690 ℃ 铝液温度下的冷隔缺陷率，P = 0.906 > 0.05，判定铝液温度在 670~690 ℃ 范围内对冷隔缺陷无显著影响。
- **压射速度验证**：对比不同压射速度下的冷隔缺陷率，P = 0.065 > 0.05，判定压射速度对冷隔缺陷无明显影响。

**变速器壳体右箱合箱面裂纹分析**[[S18]]：

| 验证因子 | 双比率检验 P 值 | 显著性结论 |
|----------|---------------|-----------|
| 进浇系统优化（浇口外移至边缘非厚大部位） | P = 0.038 | 显著降低裂纹缺陷率 |
| 更换滤渣舀子 | P = 0.703 | 无显著影响 |
| 压室润滑方式（内润滑改外润滑） | P = 0.774 | 无显著影响 |
| 增加保温炉铝液打渣频次 | P = 0.707 | 无显著影响 |

进浇系统的优化属于不同铸件进浇结构对比裂纹缺陷率的实证案例，验证了浇口位置对凝固裂纹的关键影响。

### 改进措施效果验证

在大批量生产中，双比率检验用于验证工艺改进措施是否稳定有效。

**低压铸造 IEM 气缸盖大批量验证**[[S8]]：充型压力-时间优化与小端结构优化（增加加强筋）两项措施在大批量生产中的双比率检验 P 值均为 0 < 0.05，判定优化后冷隔缺陷率显著下降，改进措施稳定有效。

## 结果解读与统计决策

### 显著性水平与 P 值判断

在工业统计检验中，常规显著性水平 α 通常设定为 0.05，表示拒绝真原假设（误判存在显著性差异）的允许概率[[S3,S12]]。双比率检验的统计决策规则为：

- **P < 0.05**：拒绝原假设，确认两组总体比率之间存在统计显著性差异。
- **P ≥ 0.05**：不能拒绝原假设，无充分证据表明两组总体比率之间存在差异。

### 置信区间判断

由双比率检验计算的两总体比率差的 95% 置信区间是等价的判断方式[[S12,S9]]：

- **区间不包含 0**：与 P < 0.05 结论一致，证明两组比率差异具有统计显著性。
- **区间包含 0**：与 P ≥ 0.05 结论一致，无充分统计证据确认两组总体比率不同。

### 小样本补充校验

小样本场景下，应以 Fisher 精确检验的 P 值作为补充判断依据，其结论比正态近似 Z 检验更为稳健[[S6]]。例如，低压铸造气缸盖冷隔缺陷分时段对比中，两组小样本冷隔缺陷率分别为 0.23% 和 0%，比率差为 0.0023，95% 置信区间包含 0，Z 检验 P = 0.317，Fisher 精确检验 P = 1.000，均大于 0.05，表明两组缺陷率差异不具有统计显著性[[S15]]。

### “差异显著”的工程含义

在压铸质量改进场景中，双比率检验确认的“统计显著性差异”（P < 0.05）具有明确的工程含义[[S6,S4,S17]]：

- 当 P < 0.05 时，表明观测到的缺陷率变化（如冷隔率下降、泄漏率降低）并非由随机生产波动导致，而是可归因于工艺/模具调整措施（如内浇口布局优化、模具冷却结构调整、增加真空排气等）的真实效果，改进措施可以稳定推广至大批量生产。
- 当 P ≥ 0.05 时，表明无充分统计证据证明该工艺调整对降低缺陷率具有真实效果，两组样本间的观测差异很可能属于正常生产状态下的随机波动，不需要将该调整推广至量产。
