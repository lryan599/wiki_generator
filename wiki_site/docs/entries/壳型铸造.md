## 概述
壳型铸造（英文：Shell mold casting，又称 Croning process、树脂壳铸造）是一种在树脂砂薄壳铸型中浇注铸件的精密铸造方法，属于可弃式铸造工艺[[S15,S1]]。该工艺由德国工程师 Johannes Croning 于20世纪中期发明[[S1]]。其硬化原理为：采用热塑性酚醛树脂作为粘结剂的覆膜砂，加入固化剂乌洛托品（六亚甲基四胺）；受热后乌洛托品释放亚甲基桥，使原本线型结构的热塑性酚醛树脂交联转变为不熔不溶的体型结构，从而完成薄壳的固化[[S12,S34]]。与普通砂型铸造相比，壳型铸造具有更高的尺寸精度和生产效率，人力需求更低，适于中小型高精度铸件的批量生产[[S1]]。

国家标准《GB/T 5611-2017 铸造术语》定义壳型铸造为“在树脂砂壳型中浇注铸件的精密铸造方法”，并规定壳型制造通常采用吹砂法，在180 ℃～280 ℃的模板上形成厚度6 mm～12 mm的薄壳，再经加热固化[[S15]]。

## 工艺步骤
壳型铸造的完整工艺流程如下[[S35,S1,S34]]：
1. 制备两片式金属模板，材质多为铸铁或钢，小批量时可选用铝或石墨；
2. 将模板预热至设定温度，喷涂脱模剂；
3. 模板扣在装有覆膜砂的翻斗上，翻转使覆膜砂附着于热模板表面形成薄壳；
4. 复位翻斗，倒出未结壳的松散覆膜砂；
5. 将附着半壳的模板送入烘炉，加热30～90 s使薄壳完全硬化；
6. 从模板上脱模取出半壳；
7. 如需制芯，先将壳芯置入半型内，再将两半壳型对齐夹紧或粘接组合为完整铸型；
8. 将壳型放入砂箱，用背砂或金属丸支撑，随后浇注熔融金属；
9. 金属冷却凝固后，打碎壳型取出铸件，清理浇注系统余料。

壳型铸造的核心特征在于使用人造树脂粘结剂制造无砂箱的薄壁铸型，其结构如图1所示[[S24]]。

## 造型材料
壳型用覆膜砂主要由原砂、树脂粘结剂、固化剂、润滑剂及特殊添加剂构成，典型组分配比见下表[[S4,S12,S5]]。

| 组分 | 典型含量（质量分数） | 作用与说明 |
|------|-------------------|-------------|
| 原砂 | 95% ～ 99% | 常用天然擦洗硅砂；高耐火度场合可选用锆砂、铬铁矿砂或宝珠砂[[S4,S5]] |
| 酚醛树脂粘结剂 | 1% ～ 5% | 多为热塑性酚醛树脂（novolak），提供砂粒间粘结力[[S4,S12]] |
| 固化剂（乌洛托品） | 占树脂质量的10% ～ 15% | 配成乌洛托品∶水=1∶1～1.5的水溶液加入，释放—CH₂—桥交联树脂[[S12]] |
| 润滑剂（硬脂酸钙） | 约 0.1% | 防止覆膜砂结块，改善流动性、脱模性[[S4,S12]] |
| 附加组分 | 按需 | 耐高温添加剂、防粘砂添加剂、溃散剂、增塑剂、抗老化添加剂等[[S4,S12]] |

覆膜砂的混制工艺有冷法、温法和热法三种，其中最常用的是热法覆膜，其树脂用量少、生产效率高。热法流程为：原砂加热 → 加树脂 → 加表面活性剂和部分硬脂酸钙 → 加固化剂和部分硬脂酸钙 → 冷却 → 出砂[[S4]]。冷覆膜法流程则为：原砂加煤油混砂1～2 min，加入粉状酚醛树脂、乌洛托品干混0.5～2 min，加工业酒精湿混2～3 min，吹冷空气8～15 min冷却，碾碎过筛即得成品[[S10]]。

## 工艺参数与控制
壳型（芯）制造的关键工艺参数及典型范围如下[[S15,S1,S35,S19,S20,S27,S31]]。

| 参数 | 典型范围/要求 |
|------|--------------|
| 模板预热温度 | 180～280 ℃（国标），工业常用175～370 ℃，部分取250～300 ℃ |
| 结壳时间 | 15～50 s |
| 壳厚 | 6～12 mm（国标），依有无背砂支撑及浇注温度调整 |
| 固化烘烤时间 | 30～90 s |
| 壳芯盒加热温度 | 200～300 ℃ |
| 壳芯结壳时间 | 30～180 s（对应壳厚3～10 mm） |
| 铸件最小壁厚 | 1.5～6 mm |
| 铸件表面粗糙度 | 可达 Ra 0.3～4.0 μm（文献值）；另一标准给出 Ra 6.3～12.5 μm |
| 铸件尺寸精度 | CT6～CT8 级 |
| 模具寿命 | 壳型模一般可达数千次 |

工艺控制要点：芯盒温度低于200 ℃时结壳缓慢、壳芯强度不足；超过300 ℃则表层树脂过烧，易产生气孔和脱壳[[S19]]。树脂含量应控制在原砂质量的1.0%～3.0%，含量过低导致壳芯强度不足，过高则发气量过大，引发气孔和粘砂[[S19,S9]]。

## 壳芯制造与在压铸中的应用
壳芯是“芯砂与芯盒接触后能快速硬化成一定厚度的壳层，未硬化芯砂自芯盒内倾出形成的空心薄壳芯子”[[S15]]。壳芯制造主要分顶吹法和底吹法：顶吹法以0.1～0.3 MPa压缩空气从上方进砂，充型效果好，适于复杂砂芯；底吹法从下方进砂，设备简单但仅适用于简单形状砂芯[[S2,S35]]。

在压铸领域，壳芯常用于形成复杂内腔结构。RYOBI公司采用压铸专用砂芯（RIC）结合层流充填方法（RNC），大批量生产柴油发动机V8缸体，直接成形缸体内部复杂水冷套流道，获得高刚性、高气密性的压铸件[[S17]]。保时捷V8油泵壳同样使用压铸壳芯成形内部复杂油路，该铸件单重840 g、最小壁厚2.5 mm，通过21 bar承压测试，工作温度范围‑40 ℃至150 ℃[[S17]]。

传统壳型（芯）铸造亦广泛用于汽车拖拉机阀体、船用零件、高速钢钻头、泵轮叶轮等，常以壳芯替代油砂芯，或采用双层型壳改善表面质量[[S18]]。

## 主要特点
**优点**  
- 尺寸精度可达CT6～CT8级，表面质量良好[[S31]]；  
- 生产效率高，适于自动化批量生产[[S31]]；  
- 加工余量小，相比普通砂型铸造可降低机加工成本30%以上[[S31]]。

**局限性**  
- 覆膜砂材料成本高于湿型砂[[S31]]；  
- 铸件最大外形尺寸一般限制在1 m左右，单件重量多为数十公斤[[S31]]；  
- 树脂热解产气，需保证铸型排气；  
- 模具和能源成本较高。

**适用合金**  
可铸造铝合金、镁合金、锌合金、铸铁、铸钢等绝大部分常规合金[[S1,S31]]。典型件包括齿轮箱体、气缸盖、连杆、杠杆臂等[[S1]]。

## 常见缺陷与分析
壳型/壳芯的典型缺陷及其与工艺因素的关联如下[[S13,S33,S6,S19,S9]]。

| 缺陷类型 | 机理与原因 | 主要影响因素 |
|----------|------------|--------------|
| 壳型破损 | 常温/高温强度不足，起模、搬运受力不当 | 树脂含量偏低、加热不充分 |
| 粘砂（机械/化学） | 金属液渗入砂粒间或氧化物与硅砂反应生成低熔点化合物 | 砂粒过粗、耐火度不足、型壳未上涂料 |
| 气孔 | 树脂热解生成CH₄、CO等气体卷入金属液 | 树脂含量过高、排气不畅 |
| 壳厚不均 | 芯盒温度场不均、结壳时间不足、吹砂压力不匀 | 加热均匀性、吹砂工艺设定 |
| 脱壳 | 壳层与芯盒壁粘结力过大，分型剂效果不良 | 润滑剂不足、分型剂选用不当 |

## 应用实例
除压铸领域的高端案例外，壳型铸造在传统铸件生产中亦有丰富积累。早期手册记载，曾用于铸铁汽车拖拉机阀体、船用零件，以及铸钢件和高速钢工具（如钻头）、泵轮叶轮等，常以壳芯取代油砂芯以提高内腔质量，并可通过双层型壳技术进一步改善铸件表面[[S18]]。
