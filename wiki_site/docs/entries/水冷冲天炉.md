## 概述

水冷冲天炉（water-cooled cupola）是铸铁熔炼领域的长炉龄专用熔炼设备。其核心特征是在炉体熔化带高温区设置强制水冷结构，通过循环冷却水保护炉壁和风口，抑制耐火材料侵蚀、稳定炉膛尺寸，从而实现长时间连续熔炼[[S15,S26,S20,S18]]。普通冲天炉因炉衬快速侵蚀，通常开炉十几小时后即须停炉打炉，而水冷冲天炉可连续运行数天乃至数周，已成为当前大型机械化、自动化铸造厂的主流铸铁熔炼设备[[S20,S18]]。

## 核心工作原理

水冷冲天炉的作用原理是：在炉内温度最高的熔化带区域通过强制水冷移除多余热量，使炉壁传热与冷却水吸热形成热平衡，在炉壁内侧自动生成厚度40~60 mm的保护性渣层，大幅降低炉衬侵蚀速率[[S20,S24]]。正常熔化期间，若操作参数（铁焦比、风量）保持不变，则熔化率与铁液温度可保持长期稳定[[S20]]。

## 主要分类

水冷冲天炉可按冷却部位及炉衬修砌方式划分为不同类型，几种分类维度在实际炉型中常组合使用。

**按水冷部位分类：**

| 类型 | 冷却方式 | 特点 |
|------|----------|------|
| 炉壳淋浴式（雨淋式） | 炉壳外布置环形水管喷淋冷却水，形成均匀水膜 | 结构简单、维护便捷、使用寿命长，但耗水量大，需配套循环处理系统[[S15,S26,S24]] |
| 内置水套式 | 炉壁熔化带设置封闭夹层水套，冷却水在封闭腔体内流动换热 | 耗水量远低于淋浴式，结构复杂度高、检修难度大；水套夹层净空通常取75~100 mm[[S15,S26,S24,S22]] |
| 独立水冷风口式 | 金属材质风口配独立循环冷却水系统，替代耐火材料风口 | 风口结构参数长期稳定不变，保证送风均匀性[[S2,S9]] |

**按炉衬修砌方式分类：**

| 类型 | 炉衬特征 |
|------|----------|
| 薄炉衬水冷冲天炉 | 水冷炉壁内侧预砌30~70 mm耐火材料层，熔化一段时间后完全侵蚀，后续依靠附着渣层维持工作[[S2,S10]] |
| 无炉衬水冷冲天炉 | 水冷炉壁内侧完全不砌筑耐火材料，依靠水冷壁面热平衡自动附着40~60 mm保护渣层，炉渣酸碱度不受炉衬材质限制[[S15,S2,S10,S20]] |

## 结构组成

### 水冷熔化段

水冷熔化段是水冷冲天炉的核心特征结构，由水冷炉壁、炉壁冷却装置、风箱、风口供风支管、炉缸、连续出铁槽、紧急出铁槽、插入式水冷风口、风口供水管等部件组成[[S2]]。

### 水冷炉壁及冷却装置

水冷壁受循环冷却水保护，正常操作下与高温炉内区域形成稳定热平衡，自动生成40~60 mm附着渣层，保护金属壁免受高温烧蚀，长期维持炉膛直径几乎恒定[[S20]]。

炉壁淋浴冷却装置主流有三种结构形式[[S2,S9]]：

| 结构类型 | 特点 |
|----------|------|
| 管嘴出流式 | 应用最广泛最可靠，由上百个独立管嘴将冷却水喷淋到炉壁形成均匀水膜 |
| 小孔出流式 | 结构最简单、造价最低，在环形供水管内侧钻数百个φ3~5 mm小孔出水；长期使用后水垢锈蚀会引起流量稳定性下降 |
| 环缝出流式 | 加工和安装精度要求极高，仅在少数特殊场景应用 |

水冷炉壁需设置至少两级淋浴冷却，一级位于水冷炉壁顶端，另一级设置在熔化区以上约500 mm位置，炉壁表面水膜厚度要求控制在5~15 mm之间[[S2,S9]]。

### 水冷风口

水冷风口主流有四种结构形式：整体铸造单腔风口、铸焊结构贯流风口、铜钢焊接双腔风口、铜钢焊接贯流风口。风口的服役寿命主要由风口帽决定，其主要失效形式为熔蚀、变形、焊缝渗漏[[S9]]。

### 水冷炉缸

炉缸冷却可采用三种形式：利用炉壁自上而下的淋浴冷却水自然冷却；配置独立的专用炉缸喷淋冷却装置；采用浸浴冷却，使炉缸外围水槽始终充满冷却水实现完全浸没式冷却[[S9]]。

### 风箱及供风支管

风箱分为冷风型和热风型，热风型水冷冲天炉的风箱外侧必须包覆隔热材料以减少热风温度损失。风口供风支管标配金属波纹管（热膨胀补偿器）、风口风量调节阀、三通弯管、风口窥视盖等部件[[S9,S23]]。

### 水冷套（内冷式水套）

水冷套通常采用10 mm厚度钢板焊接制作，内外壁之间焊接多道拉条抑制受热变形和开裂，水套夹层净空厚度取80~100 mm。炉径大于900 mm的大型冲天炉水套宜采用分片安装设计，以降低大尺寸水套的热应力。安装完成后须进行10克/平方厘米以上水压试验检验密封性[[S22]]。

## 关键运行参数

**冷却水参数：**

| 参数项 | 要求 |
|--------|------|
| 出水温度（硬水工况） | ≤40℃ |
| 出水温度（软水工况） | ≤80℃ |
| 冷却水沸腾 | 严禁 |
| 水质要求 | 须符合GB 50050-2007《工业循环冷却水处理设计规范》[[S23,S24]] |

**典型配套热风优化运行参数：**

| 参数项 | 典型值 |
|--------|--------|
| 热风温度 | 350℃ |
| 废气温度 | 70~80℃ |
| 焦炭固定碳 | 76% |
| 总焦铁比 | 1:8.8 |
| 平均出铁温度 | 1510℃ |
| 首包铁水温度 | ≥1450℃ |

上述参数来自国内RSLL-n系列自热供风、水冷、无炉衬冲天炉的实测数据，满足高品质铸铁件对高温铁水的需求[[S24]]。

## 优势与局限

水冷冲天炉相较于普通酸性炉衬无水冷冲天炉，具有显著的性能差异，汇总如下[[S21,S15]]：

| 方面 | 水冷冲天炉 | 普通无水冷冲天炉 |
|------|-----------|-----------------|
| 炉膛尺寸稳定性 | 水冷保护下熔化段炉膛尺寸长期稳定不变 | 随炉衬侵蚀逐步扩大 |
| 连续熔炼能力 | 可实现数天连续熔炼 | 通常仅4~10小时 |
| 送风稳定性 | 插入式水冷风口参数长期不变 | 耐火材料风口随侵蚀变形 |
| 渣系适应能力 | 炉渣酸碱性不受炉衬限制，可灵活切换 | 受炉衬材质制约 |
| 元素烧损 | 较低 | 较高 |
| 铁水成分/温度一致性 | 较好 | 较差 |
| 吨铁耐火材料成本 | 大幅降低 | 较高 |
| 水冷系统热损失 | 额外带走热量，冷风工况热效率偏低 | 无水冷热损失 |
| 系统复杂度与投资 | 结构更复杂，一次性投资较高 | 结构简单，投资低 |
| 操作与维护要求 | 对供水稳定性、水质要求严苛，需专人维护 | 要求相对较低 |

## 应用领域与实例

水冷冲天炉适配灰铸铁、球墨铸铁、可锻铸铁等多类铸铁材质的连续熔炼，可以全部以废钢为炉料进行铸铁生产[[S15,S18]]。主流应用覆盖以下三大场景[[S17,S3,S8]]：

1. **汽车铸件**：意大利FIAT TEKSID汽车铸造厂采用水冷冲天炉批量生产普通灰铸铁、合金灰铸铁及铁素体/珠光体球墨铸铁汽车零部件。
2. **机床铸件**：国内陕西机床厂配套使用淋浴式外水冷+水冷风口组合的水冷冲天炉实现稳定连续生产。
3. **管件及连续供铁**：适配连续铸铁管生产线、转炉车间铁水长周期不间断供给需求。

**典型量产案例**：英国Hayes Shell Cast Ltd采用水冷无焦冲天炉，截至1979年已累计熔炼2.5万吨高质量灰铸铁和球墨铸铁；伊朗某铸造厂部署的5 t/h水冷冲天炉每日量产球墨铸铁与灰铸铁铸件，后续已增建第二台同规格设备[[S16]]。

### 与压铸生产的关系

水冷冲天炉在压铸领域的适配关联主要体现在以下场景[[S24,S15,S14]]：

- 压铸企业大批量连续生产时，水冷冲天炉可长时间稳定供应温度、成分均匀的合格铁水，适配高压铸铁件、高牌号球墨铸铁压铸件的连续熔炼需求。
- 水冷冲天炉可灵活调整渣系酸碱度，低成本实现铁水预脱硫，为后续压铸用高品质球墨铸铁、蠕墨铸铁提供更优原料条件。
- 配套水冷冲天炉的双联熔炼工艺（冲天炉+保温炉）可实现铁水大批量连续供给，匹配自动化压铸生产线的24小时连续生产节拍。

## 安全与防爆要求

水冷冲天炉的核心安全风险在于：冷却水硬度过高会在水冷风口及炉壁内腔生成水垢，阻碍换热导致炉壁过热穿透，冷却水泄漏进入高温炉膛将直接引发爆炸[[S23]]。为此，强制安全要求包括[[S22,S24]]：

- 内冷式水套焊缝须经过10克/平方厘米以上的水压试验。
- 高处设置容积约3立方米的应急备用水箱以应对突发断水工况。
- 水套出水口完全外露，便于操作人员实时观测。
- 硬水工况出水温度不超过40℃，软水工况不超过80℃，严格禁止冷却水沸腾。
- 冷却水系统须符合GB 50050-2007规定的循环冷却水质要求。

水冷冲天炉已形成标准化故障处理规程，覆盖水冷风口渗漏、水冷炉壁渗漏、水冷炉缸穿透、意外停水四类典型故障的预兆识别、排除方案与预防措施[[S12]]。
