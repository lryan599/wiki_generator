## 概述

非烘树脂系（no-bake resin system / self-hardening resin system）是指在室温条件下，通过催化剂或固化剂引发化学反应，使砂型（芯）在不需要外部加热的条件下自行硬化的一类树脂砂粘结剂体系的总称[[S14,S32,S27]]。它是铸造用树脂粘结剂中的重要分支，与热硬树脂系（热芯盒、壳芯、覆膜砂等需要加热固化的体系）形成明确的工艺对立[[S14,S32]]。非烘树脂系的涵盖范围广泛，包括酸固化树脂、酯固化树脂和尿烷树脂等自硬类粘结剂，也涵盖在室温下通过与汽化的液态催化剂或气体固化剂反应实现硬化的气硬（冷芯盒）类树脂粘结剂[[S24,S16,S18]]。

## 定义边界：非烘树脂系与热硬树脂系的区分

根据GB/T 5611-2017《铸造术语》，非烘树脂系（亦称自硬树脂系）与热硬树脂系的根本区分标准在于固化温度条件[[S14,S32,S27]]：

| 对比维度 | 非烘树脂系 | 热硬树脂系 |
|---|---|---|
| 固化方式 | 室温化学催化反应，无需外部加热 | 必须在高于室温的环境中（通常180~260℃）发生交联反应 |
| 适用场景 | 自硬砂、冷芯盒气硬砂 | 热芯盒砂、覆膜砂、壳芯砂 |
| 核心英文术语 | no-bake resin system、self-hardening resin system | hot hardening resin system |
| 标准条目参考 | GB/T 5611-2017 4.3.30 | GB/T 5611-2017 4.3.32 |

非烘树脂系完全依赖催化剂/固化剂引发的化学自固化机制；而热硬树脂系依赖外部热源驱动交联反应，两者在工艺原理上彼此独立[[S40]]。

## 体系构成与分类

非烘树脂系按硬化方式可划分为两大独立分支[[S16,S1,S27]]：

### 1. 自硬分支（Self-hardening）

使用液态固化剂/催化剂，混砂后在室温环境中自行完成硬化，主要包括三类：

- **酸固化树脂体系**：以呋喃树脂或酚醛树脂为主体，配合酸性固化剂（磷酸、磺酸等）；
- **酯固化碱性酚醛树脂体系**：以高碱性水溶性热固性酚醛树脂为主体，配合液态有机酯；
- **酚醛脲烷树脂体系**：以苯基醚酚醛树脂（组分I）与聚异氰酸酯（组分II）双组分为主体，加胺类催化剂。

### 2. 气硬（冷芯盒）分支（Gas-cured / Cold Box）

在室温下通过吹入汽化液态催化剂或气体固化剂实现硬化，主流体系包括[[S24,S18]]：

- 酚醛尿烷-胺体系（三乙胺法等）
- 呋喃-二氧化硫体系
- 酚醛-酯体系
- 酚醛-CO₂体系

气硬冷芯盒工艺的显著优势在于硬化时不需加热，起模时砂型/芯强度已接近终强度，尺寸精度高，生产效率高[[S24]]。

## 非烘树脂系在压铸中的工艺角色

在压铸及低压铸造领域，非烘树脂系主要用于制备压铸件的内嵌成型砂芯，特别是复杂内腔、包络结构的砂芯[[S19]]。其核心工艺要求包括：

1. **常温强度**：砂芯必须具有足够的强度，以抵御高压高速金属液的冲击与冲刷[[S7]]；
2. **高温溃散性**：非烘树脂砂芯须具备可控的高溃散性，即在合金液浇注后，粘结剂可在高温下快速热解，降低铸件内部收缩阻力，减少热裂倾向，便于后续清砂[[S7,S19]]。

研究表明，呋喃树脂由于溃散性极好，金属液充型结束后，与合金液接触的砂芯粘结剂在极短时间内燃烧失效，对铸件的机械阻碍作用极小，利于控制铸件应力[[S7]]。这一特性使非烘呋喃树脂砂芯尤其适配铝合金、镁合金等有色合金的压铸/低压铸造生产场景。

## 典型非烘树脂体系与操作参数

### 酸固化自硬呋喃树脂体系

呋喃树脂自硬体系以含呋喃环的糠醇改性树脂为主体，配合酸性固化剂（磷酸或磺酸类）实现室温固化[[S4,S30]]。其主要工艺参数见表：

| 参数项 | 典型范围 | 说明 |
|---|---|---|
| 树脂加入量（占原砂质量） | 0.8%~1.4% | 根据砂粒特性调整[[S30]] |
| 固化剂用量（占树脂质量） | 15%~45% | 酸浓度影响固化速度[[S30]] |
| 可使用时间 | 6~15min | 大型件可延长至15min[[S17]] |
| 可使用时间/起模时间比 | 约50% | 通常为0.35~0.6[[S10,S30]] |
| 起模时间 | 15~45min | 取决于砂温、固化剂及砂芯复杂程度[[S10]] |
| 24h终抗拉强度 | 0.6~2.0MPa | 复杂砂芯可达1.6~2.0MPa[[S4,S17]] |
| 标准控制下发气量 | ≤10~11mL/g | 每克树脂砂发气量控制上限[[S4]] |
| 500℃残留强度 | ≈0 | 高温溃散性优良[[S4]] |

### 酚醛脲烷自硬体系（Pep-Set法）

该体系由美国Ashland公司于1968年发明，采用苯基醚酚醛树脂与聚异氰酸酯双组分，配合胺类催化剂使用[[S29]]。主要工艺参数：

| 参数项 | 典型范围 |
|---|---|
| 组分I与组分II加入比 | 1:1（质量比）[[S29]] |
| 两组分总加入量 | 占原砂质量1.4%~2.0%[[S30]] |
| 催化剂添加量 | 0.4%~8%（可宽区间调节）[[S30]] |
| 可使用时间 | 30s~200min[[S29]] |
| 可使用时间/起模时间比 | 可达75%[[S29]] |
| 树脂分解起始温度 | 约400℃[[S30]] |
| 500℃残留强度 | 完全消失[[S30]] |

### 碱性酚醛树脂体系（自硬及气硬）

碱性酚醛树脂以高碱性水溶性热固酚醛树脂为主体，可通过液态有机酯实现自硬，也可吹入甲酸甲酯或CO₂气体实现冷芯盒气硬[[S27,S28]]。该体系参数如下：

| 参数项 | 典型范围 |
|---|---|
| 树脂加入量 | 占原砂质量1.5%~2.5%[[S38]] |
| 固化剂（有机酯）用量 | 占树脂质量20%~30%[[S38]] |
| 树脂加入量2.5%时24h强度 | 可达3.15MPa[[S28]] |
| 1000℃保温2min发气量 | 五种非烘树脂中最低[[S38]] |
| 400℃残留抗压强度 | 0.23MPa[[S28]] |
| 500℃残留强度 | 几乎为零[[S28]] |
| 成分特性 | 不含氮、磷、硫，环保性好[[S28,S27]] |

## 混砂工艺与控制要求

非烘树脂砂的混制过程要求混合均匀且速度要快，主流设备为叶片式快速混砂机（含S20系列碗形间歇式混砂机及多种连续式混砂机），部分设备在外壳配有通水冷却装置以避免砂温过高损失可使用时间[[S21,S6]]。

典型混砂流程[[S6]]：

```
原砂 + 固化剂 → 混匀（约20~25s） → 加树脂 → 混匀（约35~40s） → 出砂
```

对混砂设备的定量精度要求：树脂、固化剂的流量误差不超过±3%，砂流量误差不超过±5%[[S6]]。砂温应控制在20~30℃区间，避免砂温偏高导致可使用时间大幅缩短[[S21,S30]]。

## 工艺优势与局限性

### 核心优势

非烘树脂系相对于加热类烘烤树脂体系的优势主要体现在[[S23]]：
- 无额外加热能耗，室温即可固化；
- 无需高价耐高温加热金属芯盒，普通木质或铝合金芯盒即可使用，模具采购成本低；
- 无芯盒高温变形导致的尺寸偏差问题；
- 适配大尺寸、复杂轮廓砂芯及单件小批量压铸生产；
- 工艺调试门槛低，无需配套加热控温系统。

### 现存局限性

非烘树脂系的主要局限性包括[[S29,S8,S39]]：
- 固化速度受环境温湿度影响大，低温高湿环境下固化效率显著下降（如多元醇脲烷体系在90%高相对湿度条件下，硬化强度仅为低湿工况的20%）[[S29]]；
- 单模生产周期长于热芯盒、壳芯等加热固化体系，无法完全适配高节拍连续化大批量生产；
- 部分体系（如酚尿烷）存在高温强度偏低问题，厚壁铸件浇注后容易产生毛刺缺陷[[S8]]；
- 醇酸/多元醇脲烷体系在混砂和浇注时产生烟气，对工作环境有一定影响[[S39]]。

## 与其他制芯工艺的综合对比

下表汇总了非烘树脂系与热芯盒、壳芯、温芯盒工艺的核心参数差异[[S34,S5,S23,S13,S15]]：

| 对比维度 | 非烘树脂系 | 热芯盒工艺 | 壳芯工艺 | 温芯盒工艺 |
|---|---|---|---|---|
| 固化方式 | 室温自硬/气硬化学反应 | 加热催化交联 | 覆膜砂受热熔融交联 | 低温加热辅助催化固化 |
| 芯盒温度 | 室温 | 180~260℃ | 200~280℃ | 50~175℃ |
| 生产周期 | 15~60min起模 | 几十秒即可取芯 | 结壳50~150s + 继续硬化 | 几秒至几十秒取芯 |
| 模具材质与成本 | 普通木质/铝合金，成本低 | 耐热金属芯盒，成本高 | 耐高温金属芯盒，成本高 | 带加热装置的耐高温芯盒，成本较高 |
| 适用砂芯范围 | 中大尺寸复杂砂芯，单件中小批量 | 截面厚度≤50mm小型砂芯，大批量 | 中空薄壁高精度砂芯，大批量 | ≥9kg、厚度>50mm中大砂芯，大批量 |
| 铸件尺寸精度 | 较普通粘土砂高2~3级 | 高 | 高，表面粗糙度低 | 优于热芯盒，变形少 |
| 能耗 | 极低 | 高 | 高 | 较热芯盒降低25%~30% |

## 相关标准与规范

- **GB/T 5611-2017《铸造术语》**：收录非烘树脂系（自硬树脂系）标准术语，定义涵盖酸固化树脂、酯固化树脂和尿烷树脂[[S14]]；同时明确气硬树脂系属于非加热固化范畴[[S24]]。
- **JB/T 7526-2008《铸造用自硬呋喃树脂》**：适用于以糠醇为主要原料合成、酸催化固化的非烘型砂芯粘结剂，规范了分类标记、技术要求、试验方法、检验规则及包装储运要求[[S36]]。
- **ASTM / AFS铸造行业规范**：公开规范中定义no-bake树脂粘结剂包含呋喃、酚醛、醇酸、氨基甲酸酯（异氰酸酯基）等多个品类，明确接触及制芯过程中应采取的安全操作措施[[S31]]。
