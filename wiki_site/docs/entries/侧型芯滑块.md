## 概述

侧型芯滑块（side core slide / side core slider，亦称侧抽芯滑块、滑块），是压铸模具侧向抽芯机构的核心运动元件。它可沿垂直于开合模方向或与开合模方向成一定角度滑动，核心功能是成形压铸件上无法直接脱模的侧凹、侧孔或侧凸结构[[S5,S16,S31]]。在开模阶段，侧型芯滑块先带动侧型芯从铸件中抽出，消除脱模障碍；在合模阶段，滑块再带动侧型芯复位至成形位置，从而保障铸件顺利脱模且尺寸精度符合要求。

侧型芯滑块在压铸模具中属于抽芯机构的组成部分，抽芯机构一般由成形元件、运动元件、传动元件、锁紧元件和限位元件等五大功能零件组成[[S31]]。侧型芯滑块承担的是运动元件的角色，用于连接并带动型芯或型块在导滑槽内运动。

## 分类

### 按驱动动力源分类

按驱动动力源分类，侧型芯滑块的驱动方式主要分为机动驱动、液压驱动和手动驱动三类[[S31]]。

| 驱动类别 | 典型形式 | 特点与适用场景 |
|----------|----------|----------------|
| 机动驱动 | 斜导柱驱动 | 适用于抽芯距≤40 mm的常规侧抽芯场景，结构简单，应用最广泛[[S32,S16]] |
| 机动驱动 | 弯销驱动 | 可实现更大抽芯距，支持延时抽芯，承载的侧向力比斜导柱高[[S16]] |
| 机动驱动 | 齿轮齿条驱动 | 用于弧形抽芯等特殊轨迹场景[[S1]] |
| 液压驱动 | 液压缸驱动 | 抽芯力大，抽芯行程长，适用于大型复杂侧抽芯结构[[S1,S31]] |
| 手动驱动 | 人工操作 | 结构简单、生产效率低、劳动强度大，仅在特殊场合使用[[S31]] |

### 按滑块结构形式分类

按结构形式，侧型芯滑块分为整体式和组合式两类[[S13,S15,S26]]。

- **整体式**：在滑块本体上直接加工出侧向型芯或侧向型腔，仅适用于侧成形结构简单的小规格侧抽芯场景[[S13,S15]]。
- **组合式**：将侧成形型芯与滑块本体分开加工后装配固定，可节省优质模具钢、降低加工难度，是工业中应用最广泛的结构形式[[S13,S15,S26]]。

### 按锁紧方式分类

按承载能力的不同，侧型芯滑块可配置多种锁紧结构：小侧向力场景可采用螺钉加定位销固定的分体锁紧块；中高侧向力场景采用模套板内嵌锁紧块或带辅助镶接的加强锁紧结构；超大侧向力场景采用整体式锁紧块。部分小侧抽芯场景也可利用弯销自身结构直接实现滑块锁紧[[S37]]。

## 工艺角色与工作过程

### 完整动作流程

侧型芯滑块在压铸全循环中的动作流程如下[[S24,S9]]：

1. **合模锁紧阶段**：合模时，斜导柱推动滑块完成侧向复位。定模侧的楔紧块压紧滑块并锁定，承受压铸填充时金属液对滑块产生的侧向涨型力。楔紧块的锁紧角度需比斜导柱倾角大2°～3°，以保证楔紧块完全压紧滑块，使斜导柱不直接承受金属液侧向压力，避免变形失效[[S9]]。

2. **开模抽芯阶段**：开模时，滑块随动模沿开模方向运动，斜导柱驱动滑块沿导滑槽做指定行程的侧向抽芯运动。抽芯完成后，定位装置将滑块固定在抽出极限位置，保证铸件完全脱离侧型芯轮廓不发生剐蹭[[S24]]。

3. **推出铸件**：压铸机顶出机构动作，推杆将成形铸件从动模侧推出[[S24]]。

4. **合模复位阶段**：合模动作中，斜导柱重新插入滑块斜孔，驱动滑块沿原路返回型腔侧，最终楔紧块重新压紧滑块完成锁模[[S24,S9]]。

### 延时抽芯机构（特殊工况）

针对定模侧包紧力大于动模的特殊铸件场景，可采用延时抽芯结构：滑块斜孔预留指定的空程δ，开模初期滑块保持不动，使铸件随动模运动脱离定模型腔；延时结束后才开始侧向抽芯动作[[S24]]。

## 设计参数

### 斜导柱倾角与锁紧角

| 参数 | 推荐值 / 要求 |
|------|--------------|
| 斜导柱（滑块驱动端）倾角 | 常规取15°～20°，最大不超过25°[[S1,S7,S25,S8]] |
| 楔紧块锁紧角度与斜导柱倾角关系 | 锁紧角比斜导柱倾角大2°～3°（部分资料建议3°～5°）[[S4,S9]] |

锁紧角大于驱动角的目的是保证合模时楔紧块完全压紧滑块，防止压铸过程中斜导柱直接承受金属液侧向压力[[S9]]。

### 抽拔距计算

抽拔距是将侧型芯从成形位置抽至不妨碍制品脱模位置所需的移动距离[[S23,S10,S35]]。

| 场景 | 计算公式 | 安全余量 |
|------|----------|----------|
| 侧孔/侧凹成形 | S = 侧孔成形深度 + 2～3 mm | 2～3 mm |
| 双拼合凹模抽芯 | S = √(R² − r²) + 2～3 mm | 2～3 mm |
| 多拼合瓣合模抽芯 | S = √(R² − A²) − √(r² − A²) + 2～3 mm | 2～3 mm |

其中，R为铸件最大外轮廓半径，r为侧凹处最小轮廓半径，A为拼合凹模前端弦长的1/2[[S10,S30,S35,S6]]。

### 导滑段长度与滑块尺寸约束

| 约束条件 | 要求 |
|----------|------|
| 抽芯后剩余导滑长度 | 滑块完成全部抽芯动作后，仍留存在导滑槽内部的长度不得小于滑块总长度的2/3，避免复位时滑块偏斜卡死[[S34]] |
| 滑块长宽比例 | L ≥ 0.8C（L为滑块总长度，C为滑块宽度）[[S34]] |
| 滑块长高比例 | L ≥ B（B为滑块总高度）[[S34]] |
| T形导滑结构厚度 | 10～25 mm[[S34]] |
| T形导滑肩宽度 | 6～10 mm[[S34]] |

### 配合间隙要求

侧型芯滑块的配合间隙需根据压铸合金种类和使用工况不同选择，保证在工作温度下维持合理的间隙配合性质[[S11,S38,S14]]。

| 配合部位 | 配合类型 | 适用工况 |
|----------|----------|----------|
| 圆形型芯与滑块固定配合 | H7/h6 | 通用 |
| 非圆形型芯与滑块固定配合 | H8/h6 | 通用 |
| 型芯在动模镶块内滑动配合 | H7/e8 | 压铸铝合金 |
| 型芯在动模镶块内滑动配合 | H7/f7 | 压铸锌合金 |
| 型芯在动模镶块内滑动配合 | H7/d8 | 压铸铜合金 |
| 滑块宽度与导滑槽配合 | H9/f9（常规）、H7/e8（高精度） | 锌、铝、镁合金 |
| 导滑台肩配合 | H7/f8 | 高精度滑块 |

对于铜合金压铸场景，需将滑块单独加热至100～120 ℃保温后，与室温模套装配，保证工作状态下单面配合间隙小于0.05 mm，防止金属液窜入[[S14]]。

### 表面粗糙度要求

| 表面区域 | 粗糙度要求 |
|----------|-----------|
| 滑块导滑工作面 | Ra ≤ 0.6 μm[[S34,S12]] |
| 滑块成形面区域 | Ra ≤ 0.4 μm[[S34,S12]] |
| 非密封非导滑配合面 | Ra ≤ 0.8 μm[[S34]] |
| 滑块非重要表面 | Ra ≤ 3.2 μm[[S20]] |

### 常用尺寸公差标准

滑块成形区域尺寸公差常规取IT9级，特殊高精度场景可提升至IT6～IT8级；所有运动副滑动配合区域的公差等级不低于IT7[[S11,S12,S34]]。滑块安装基准面的尺寸公差取js8，导滑槽到基准面的定位尺寸公差取f7，符合GB/T 1800系列公差标准[[S11]]。

### 滑块本体最终硬度

滑块主体热处理硬度要求为50～54 HRC。导滑槽耐磨件的热处理硬度需高于滑块，以降低两者的磨损速率，延长使用寿命[[S34,S12]]。

### 冷却与润滑

高热负荷场景下的大尺寸侧型芯滑块内部可设计点冷式或直通式冷却水道，以降低滑块整体热变形量，保证配合间隙在工作温度下仍满足设计要求。导滑运动工作面需定期喷涂耐高温脱模剂或二硫化钼润滑剂，以防干磨拉伤[[S19]]。

## 缺陷与失效机制

侧型芯滑块在热循环与交变载荷作用下，常见的失效形式包括：

- **卡死**：滑块与导滑槽配合间隙不当或润滑不足，导致运动受阻。
- **变形与磨损**：在高温金属液反复冲刷与开合模摩擦作用下，导滑面可能出现犁沟、拉毛等磨损现象[[S17]]。
- **断裂**：交变冷热载荷作用下，侧型芯滑块类热作模具件可能在裂纹源位置产生裂纹并向内部扩展，最终导致断裂[[S3]]。
- **溢料（飞边）**：锁紧不足或配合间隙过大时，金属液窜入配合间隙形成飞边。
- **铸件拉伤**：抽芯动作时型芯与铸件之间的摩擦可能导致铸件表面拉伤。
- **热疲劳裂纹**：压铸模具在数千模次生产后表面可能产生龟背状热疲劳裂纹[[S36]]。

对策包括：合理选择配合间隙与配合长度、保证导滑面硬度与表面粗糙度要求、设计有效的冷却水道控制工作温度、采用耐磨镶件或镶条导向降低磨损、合理设计锁紧角确保压铸时滑块不松动等。

## 应用场景与局限性

- **大型滑块设计难点**：滑块较宽时，按标准公差选取的配合间隙绝对值很大，需要采用条形镶件导向的方式来减小配合间隙，保证移动精度[[S38]]。
- **深腔侧抽芯局限**：带有深腔的铸件在采用斜滑块抽芯时，需认真计算开模后能取出铸件的开模行程[[S14]]。
- **高速压铸冲击**：滑块在压铸填充过程中承受高压高速金属液的侧向涨型力，须由楔紧块可靠锁紧，斜导柱不应直接承受此力[[S9]]。
- **不同合金对滑块设计的影响**：
  - 锌、铝、镁合金：配合间隙按滑块宽度常规选取即可[[S14]]。
  - 铜合金：需将滑块单独加热至100～120 ℃保温后与室温模套装配，确保单面配合间隙小于0.05 mm，防止金属液窜入[[S14]]。
