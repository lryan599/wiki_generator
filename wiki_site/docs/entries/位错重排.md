## 位错重排

### 概述与定义

**位错重排**（别称位错重构、位错排列，对应英文 dislocation rearrangement、dislocation patterning）是金属在冷/热变形形成高位错密度后，高位错组态在热激活或外力作用下，通过位错运动从无序缠结状态自发调整为低能有序组态的物理冶金过程。其实现依赖位错的滑移、攀移、交滑移三类基础运动形式[[S35,S7]]。

位错重排是回复阶段的核心微观机制，所有回复过程均以不同程度的位错重排为核心表现，通过重排可大幅降低晶体内部畸变能，实现内应力释放、位错密度部分下降，而宏观组织仍基本保留变形后的纤维特征[[S41,S35,S10]]。

---

### 微观机制与驱动力

位错重排依据发生的时机和驱动方式，可区分为静态回复场景和动态回复场景两种主要类型。

#### 静态回复中的位错重排

静态回复场景下的位错重排发生在变形停止后，依靠热激活驱动，总驱动力为变形储存的畸变能。按温度区间可划分为三个阶段[[S35,S40]]：

| 温度区间 | 主要机制 | 典型表现 |
|---------|---------|---------|
| 0.1~0.3 T_m（低温） | 点缺陷迁移、空位与间隙原子重新结合 | 电阻率下降，点缺陷密度下降 |
| 0.3~0.5 T_m（中温） | 位错滑移运动，异号位错相互抵消 | 位错密度部分降低，内应力释放 |
| >0.5 T_m（高温） | 刃型位错攀移，发生**多边化**过程 | 形成位错墙（小角度亚晶界），亚晶结构产生 |

**多边化过程**是高温回复阶段最典型的位错重排形式，其完整步骤为[[S35,S34,S2]]：

1. 塑性变形使单晶体点阵发生弯曲，滑移面上累积大量同号塞积刃型位错；
2. 热激活达到足够温度时，刃型位错获得攀移运动能力，通过滑移和攀移调整空间位置；
3. 同号刃型位错垂直于滑移面有序排列形成位错墙（小角度亚晶界），将原变形晶粒分割为若干低位错密度的亚晶块，完成多边化。

多边化后释放出来的能量代表部分应变能的降低，后一种位错组态的能量比前者的无序分布态更低[[S34]]。

#### 动态回复中的位错重排

动态回复场景下的位错重排发生在塑性变形持续进行的过程中，驱动力为外加载荷持续作用叠加高温热激活。位错的增殖速率与重排、湮灭速率最终达到动态平衡，形成内部位错密度基本稳定的等轴亚晶结构[[S19,S28]]。

层错能高的金属（如铝、镁合金）更容易发生动态回复型位错重排。动态回复的机制与静态回复相似，涉及螺型位错的交滑移和刃型位错的攀移；多边化过程同样可在动态回复中出现[[S28]]。

动态回复时的真应力-真应变曲线可分为三个阶段：①微应变区（位错密度迅速增加）；②正应变硬化阶段（位错缠结形成胞状结构，位错进一步重排，通过滑移和攀移发生多边形化）；③稳态形变阶段（加工硬化与动态回复基本达到平衡，应力趋向恒定值）[[S19]]。

### 分类

#### 按位错排布形态分类

位错重排过程中形成的位错组态可按其空间排布形态分为三类[[S34,S50,S1]]：

| 形态类别 | 特征描述 | 形成阶段 |
|---------|---------|---------|
| 胞状位错结构 | 位错缠结在晶粒边界聚集形成胞壁，胞内位错密度远低于胞壁 | 重排初期 |
| 位错墙/多边化亚晶界 | 刃型位错有序垂直排列形成低角度位错墙，分割出完整亚晶 | 重排后期 |
| 三维位错网络 | 不同滑移系的位错发生反应连接形成空间网状低能位错组态 | 多系滑移条件下 |

#### 按形成机制分类

位错重排可按形成机制分类为三类[[S18,S35]]：

| 机制类别 | 驱动方式 | 典型特征 |
|---------|---------|---------|
| 静态热激活型 | 变形结束后依靠退火热激活驱动 | 多边化亚晶形成 |
| 动态变形诱导型 | 塑性变形过程中外力持续作用驱动 | 位错增殖与重排持续动态平衡，得到等轴稳态亚晶 |
| 点缺陷辅助型 | 低温下位错借助空位迁移完成低程度重排 | 主要实现内应力部分释放 |

### 形成的组织特征

位错重排过程中，随变形量或循环次数的增加，金属内部依次形成不同层次的位错亚结构[[S51]]。

**位错缠结**的形貌为杂乱分布的高密度位错团簇，无法清晰分辨单个位错，形成条件为低循环次数、低变形量阶段，位错大量增殖后发生交互堆叠，无明显边界划分位错聚集区和低位错区[[S14,S56]]。

**位错墙**形貌为高密度位错聚集形成的墙状/带状结构，两侧为低位错密度区，属于高位错能亚结构。在高层错能金属（Fe、铝合金）中易由位错滑移和攀移后有序排列生成；后续随加载次数或变形量提升，位错墙可发展为清晰的位错胞边界[[S50,S14]]。

**位错胞**形貌为由位错墙包裹形成的封闭类胞状结构，胞内位错密度远低于胞壁。随变形量提升位错胞尺寸不断减小，在真实应变为0.15时铝合金位错胞的极限尺寸略小于1 μm[[S48,S56,S14]]。

**亚晶界**是位错重排最终生成的低能态小角度晶界，可由位错胞壁处位错的交滑移、攀移、异号位错湮灭后规则排列形成。两侧亚晶存在几到十几度的小取向差，满足位错阵列公式 θ = b/s（θ为取向差，b为柏氏矢量，s为相邻位错间距）。亚晶界类型可分为纯扭转亚晶界、部分扭转亚晶界和倾斜亚晶界三类[[S32,S38]]。

位错重排在循环载荷或变形过程中存在明确的递进序列：**位错缠结 → 位错墙 → 位错胞 → 亚晶**。该演变过程不同晶粒不同步，晶粒取向差异导致分切应力不同，位错运动先后顺序存在差异[[S51]]。

### 纳米颗粒的协同作用

#### 对位错重排的调控机制

在压铸铝合金中，纳米颗粒及析出相可通过以下机制调控位错重排过程[[S22,S9]]：

- **钉扎位错运动**：纳米颗粒钉扎小角度晶界，阻碍亚晶合并，同时促进短程剪切带形成；
- **促进位错增殖**：纳米颗粒与基体热膨胀系数差异在变形过程中产生几何必须位错，增加位错源数量，提高位错增殖速率[[S57,S43]]；
- **调控动态再结晶**：促进晶内位错重排及其向小角度晶界的转变，调控动态再结晶过程；
- **提高畸变储能**：提高位错密度和晶界取向偏差梯度，增加静态再结晶储能。

纳米颗粒对位错重排存在双重调控效果：既可以通过促进位错增殖提高重排效率加速再结晶，也可通过钉扎位错运动抑制重排延迟再结晶发生[[S12,S55]]。

#### 纳米颗粒辅助下位错缠结与位错墙的形成差异

纳米颗粒辅助下位错缠结与位错墙的形成机制存在差异：位错缠结是变形过程中位错在纳米颗粒周围快速增殖、无规则聚集的初始高密度亚结构；而位错墙是位错在纳米颗粒钉扎作用下经过持续重排、逐步沿特定方向规整排列、取向差梯度逐步升高后形成的有序亚结构[[S22,S36]]。

#### 压铸合金中的应用实例

**Al-Si压铸合金体系**：纳米Al₂O₃颗粒添加量在0.6 vol.%附近时可实现最优细化效果，此时颗粒对位错运动的钉扎作用最强。当添加量超过0.8 vol.%时纳米颗粒易发生团聚，会削弱对位错重排的正向调控效果[[S23,S33]]。

**Mg基压铸合金**（如ZK60）：尺寸小于100 nm的纳米颗粒通过钉扎位错迁移、提高基体位错密度的方式，促进位错重排过程，同时颗粒周围高密度位错区域为动态再结晶提供更高驱动力，加速位错重排向小角度晶界的转变[[S39,S11]]。

### 对压铸件性能的影响

#### 强度提升

压铸合金经位错重排形成亚晶结构后，可显著提升合金的强度。普通位错重排诱导的亚晶强化对铝合金屈服强度提升幅度通常小于50 MPa。添加纳米颗粒后通过钉扎位错、诱导更多位错塞集可进一步放大强化效果：免热处理Al-15Zn-8Si-2Cu压铸合金添加TiC/TiB₂纳米颗粒后，屈服强度从160 MPa提升至213 MPa，比仅发生普通位错重排的试样提升33.1%，抗拉强度从330 MPa提升至361 MPa[[S26]]。

#### 热稳定性与抗蠕变性能

普通位错重排生成的亚晶界热稳定性较差，当温度高于0.4 T_m时，亚晶易发生长大吞并失去对位错的钉扎作用，无法长期保持高抗蠕变性能[[S47]]。添加高熔点纳米相或稀土相的压铸合金，亚晶界可在200℃下长期稳定存在，显著延长服役温度区间[[S42,S4,S59]]：

| 合金体系 | 蠕变条件 | 蠕变性能 | 性能对比 |
|---------|---------|---------|---------|
| 压铸Mg-3Sm-0.5Zn-0.4Zr | 200℃/60 MPa | 稳态蠕变速率1.1×10⁻⁹ s⁻¹，100 h蠕变量0.06% | 显著优于未进行纳米相调控的普通压铸镁合金[[S42]] |
| 普通AZ/AM系压铸镁合金 | >120℃ | 晶界Mg₁₇Al₁₂相热稳定性差，蠕变性能恶化 | 使用温度一般不超过120℃[[S59]] |
| 含LPSO相Mg-RE压铸镁合金 | 高温 | 相界面钉扎亚晶界，晶粒热稳定性高 | 远高于普通AZ91镁合金[[S4,S59]] |

#### 疲劳性能的改善

在室温循环应力作用下，纳米颗粒通过钉扎位错、增加位错源数量来加快位错增殖速率，为动态回复提供更高驱动力，加速位错墙或位错缠结向小角度晶界的转变过程，从而促进动态应力卸载，提高疲劳寿命[[S57,S43]]。

### 与其他微观组织过程的关系

#### 回复阶段

位错重排是回复阶段的核心微观机制。中温和高温回复均以位错运动（滑移、攀移、交滑移）和重新分布为基本特征，通过位错重排实现内应力的降低和位错组态的稳定化[[S41,S35]]。

#### 再结晶形核

位错重排是再结晶形核的前置必要条件：重排过程中逐步形成的亚晶通过吞并、合并不断长大，亚晶界的位向差逐步提升最终演变为大角度晶界，可直接作为再结晶核心。部分高形变条件下重排形成的高位错密度差区域也可触发晶界弓出形核[[S31,S12,S55]]。

#### 晶粒长大

位错重排过程会逐步消耗变形储存的畸变能。当所有重排完成、再结晶全部结束后，晶体内部残留的高位错密度已被消耗至低位错密度平衡态，后续的晶粒长大过程完全依靠晶界迁移实现，不再伴随明显的位错重排行为，晶粒长大的核心驱动力为晶界总面积减少带来的体系自由能下降[[S10,S30,S53]]。

|---------|---------|-------------|

### 表征手段

#### 透射电子显微镜（TEM）

透射电子显微镜（TEM）是位错结构表征的核心手段，分辨率可达0.18~0.2 nm，可直接观察位错的原子级微观组态。通过亮场、暗场或弱束暗场成像获得位错的衬度像，结合 **g·b=0** 的不可见准则可标定单个位错的柏氏矢量[[S24]]。

| 项目 | 说明 |
|------|------|
| 适用场景 | 位错线、位错缠结、位错墙、位错胞等精细亚结构的直接观测 |
| 制样方法 | 普通位错观察采用直径3 mm薄片，在-20℃的含5 vol.%高氯酸的甲醇溶液中双喷电解减薄；亦可采用碳膜萃取复型技术获得析出相与位错的复合形貌[[S8]] |
| 典型设备 | JEM 200CX、JEOL 2010F等型号透射电镜 |
| 可获得参数 | 位错类型、柏氏矢量、位错组态特征、局部位错密度分布 |
| 局限性 | 样品制备难度高、观测视场小，统计代表性有限 |

#### 电子背散射衍射（EBSD）

电子背散射衍射（EBSD）适用于大视场下的亚晶界识别、位错胞统计、应变场定量分布分析，当前空间分辨率已达纳米级，可部分替代TEM完成位错组态的统计观测[[S44,S15]]。

| 项目 | 说明 |
|------|------|
| 适用场景 | 大视场亚晶界识别、位错胞统计、应变场定量分布分析 |
| 操作要点 | 试样抛光至无应变状态后，在场发射SEM中自动采集菊池衍射花样，逐点计算晶体取向 |
| 可获得的位错参数 | 整体位错密度分布、局部取向差（KAM值）、小角晶界/亚晶界的统计占比 |
| 亚晶界定量分类 | 0.5°~2°小角晶界对应塑性应变引入的位错型亚晶界；1°~5°亚晶界为动态回复形成的位错墙；15°以上为大角度晶界[[S58]] |
| 局限性 | 无法直接观察单个位错，无法分辨纳米级以下的精细位错组态 |

#### X射线衍射峰宽化法（XRD）

XRD适用于大统计量位错密度的批量测试，无需复杂制样，可获得宏观统计的整体位错密度[[S8,S20,S29]]。

| 项目 | 说明 |
|------|------|
| 适用场景 | 大统计量位错密度的批量测试 |
| 操作要点 | 采用常规XRD扫描获取不同晶面的衍射峰半高宽，通过Williamson-Hall方法去除晶粒细化的宽化贡献，计算平均位错密度 |
| 典型参数 | 对体心立方铁素体材料，柏氏矢量b取√3/2 a（a为晶格常数0.2863 nm），通过修正位错对比因子可提升精度[[S8]] |
| 局限性 | 无法获得位错的空间分布与组态特征，仅能得到全局平均统计值 |

| 位错滑移、攀移、交滑移 | 位错重排的实现依赖这三类位错运动形式，攀移和交滑移是实现位错有序排列形成亚晶界的关键[[S35,S7]] |
| 多边形化 | 多边形化是高温回复阶段位错重排的典型表现形式，同号刃型位错通过攀移和滑移垂直排列形成位错墙[[S34,S35,S2]] |
