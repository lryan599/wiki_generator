## 概述

芯头是铸造与压铸模具中型芯上不与金属液接触的非成型附属结构段，属于型芯的重要组成部分。其核心作用是为型芯提供定位、支撑和排气功能 [[S12,S39]]。在压铸/铸造行业内，芯头的标准通用英文译名为 **core print**，该术语同时指代模样上预留出芯头成型空间的对应凸起结构；行业通用术语“芯头间隙”对应 **core print clearance** [[S19,S33]]。

需要注意的是，core head、core tip、core nose 等术语在现有技术文献中仅出现在管材轧制、金属切削等非压铸场景，不属于压铸领域芯头的通用别名；core locating portion 仅为芯头的功能描述性短语，未形成行业统一固定术语 [[S15]]。

**芯头与相关术语的关系：**
- **型芯（core）**是压铸模具中成型铸件内表面的完整凸状零件，芯头是型芯上不参与成型的附属定位段，二者为整体与局部的从属关系 [[S35]]。
- **抽芯机构**是独立驱动活动型芯完成位移、实现抽芯复位的整套机械组件，芯头仅属于活动型芯上的局部定位段，二者无直接从属关系 [[S25]]。

## 分类

芯头可按布置形式、功能属性及特殊工况需求进行分类。

### 按布置形式分类

芯头根据在铸型/模具中的安装位置可分为两大类主流形式 [[S29,S21]]：

| 类型 | 说明 |
|------|------|
| **垂直芯头** | 合箱时垂直放置的型芯芯头。常用三种结构：上下都设置芯头保证定位支撑可靠；仅设置下芯头方便合箱；不设置上下芯头适配大尺寸高稳定性砂芯 [[S29,S39]] |
| **水平芯头** | 合箱时水平放置的型芯芯头，适配横向放置的型芯，可设置多组芯头保证型芯安放稳定 [[S29]] |

### 按功能属性分类

| 类型 | 说明 |
|------|------|
| **普通支撑芯头** | 提供基本定位与支撑功能 |
| **定位芯头** | 专门用于限制型芯沿轴线方向移动或绕轴线转动，适配形状不对称的异形型芯、多型芯共存的工况。典型定位芯头尺寸需比芯头座对应尺寸大0.2~0.3mm，保证型芯安放稳固 [[S29,S9]] |

为防止型芯绕轴线转动、轴向窜动或下芯错位，可采用异形定位结构的特殊定位芯头 [[S29,S31]]。

### 特殊定制类芯头

特殊定制类芯头可根据工况需求设计，例如仅在浇口间隙位置设置凸起的“雪花型芯头”，可大幅提升合箱时浇注系统的对接容错率，降低大尺寸铸件的冲砂缺陷概率 [[S13]]。

## 功能与工艺作用

芯头的核心功能包括以下四个方面 [[S10,S39]]：

| 功能 | 说明 |
|------|------|
| **定位固定** | 使型芯在压铸/浇注过程中保持精确位置，通过芯头与芯座的配合实现准确安放 |
| **承受载荷** | 承受型芯自身重力以及金属液充型时的高压、浮力，避免型芯偏移损坏芯座 |
| **排气** | 通过芯头预留间隙实现型芯内部废气顺畅排出模具外 |
| **密封防渗** | 通过匹配凸环结构封堵芯头与芯座的配合间隙，阻止熔融合金渗入间隙产生披缝缺陷 |

## 设计关键参数

### 配合间隙

芯头与芯座之间需预留合适间隙。湿型工况下间隙通常取0.2~1mm，干型/大件工况下间隙为2~4mm，间隙过大易导致披缝、偏芯，间隙过小易引发下芯合箱卡滞、掉砂缺陷 [[S29]]。

行业通用配合间隙范围 [[S29,S8]]：

| 间隙类型 | 范围 | 适用场景 |
|----------|------|----------|
| 侧面间隙 | 0.2~5mm | 机器造型、湿型大批量生产常用0.5~1mm；干型大件常用2~4mm |
| 顶面间隙 | 0~6mm | 比侧面间隙偏大 |
| 底面间隙 | 湿型0~1.5mm；干型0~3mm；自硬砂0~2mm | 标准未作具体规定，由工艺人员按砂型类别选用 |

### 拔模斜度

芯头推荐拔模斜度通用范围为5°~10°，其中上侧芯头的斜度取值大于下侧芯头，便于合模操作 [[S10]]。

### 承压面积

大负载芯头承压面积需满足以下公式 [[S31,S38]]：

> S ≥ k·P/σ

其中安全系数k取1.3~1.5，P为型芯受到的最大浮力，σ为对应芯座的许用压应力。

### 水平芯头长度匹配

当芯头工作段长度l₂较小时，配套支撑段长度l₁取等于l₂；当l₂长度较大时，l₁取值为0.5~0.8倍l₂ [[S29]]。

## 结构安装

芯头的标准安装固定形式 [[S18]]：

- 独立加工的水平芯头通过沉头螺钉+定位销组合装配到模样本体与模底板上
- 小尺寸垂直芯头直接用螺钉紧固在模样本体上
- 大尺寸垂直芯头通过螺钉穿模样本体与模底板锁附固定

## 常见失效与缺陷

### 热疲劳（占60%~70%）

热疲劳是压铸芯头最常见的失效形式，由压铸过程中芯头反复经受300~1000℃区间的急冷急热热交变应力作用产生微裂纹，逐步扩展为网状龟裂，严重时扩展为宏观裂纹导致芯头报废 [[S30,S27,S1]]。

![图：压铸型芯疲劳寿命分布云图，芯头过渡区为典型低寿命高应力区域](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/c14db91c-c70f-4797-a13c-c09fd38bbad9/resource)

### 冲蚀磨损

高压高速熔融金属液对芯头表面产生持续机械冲刷、高温化学腐蚀和气蚀共同作用，易发生在金属流直接冲刷的芯头部位，导致芯头尺寸超差，铸件对应位置出现毛刺、尺寸不合格缺陷 [[S27,S5]]。

### 热变形

铝合金在150~500MPa高压作用下对芯头持续冲击，在应力集中部位发生塑性变形。适当增大芯头R角或取消尖角过渡可降低变形量 [[S14]]。

### 粘模

由铝合金等活性金属与芯头材料表面亲和性过高引起，在芯头局部硬度偏低处极易发生，导致铸件同一固定位置出现完全相同的凹痕缺陷 [[S43,S16,S45]]。

### 卡死

主要诱发因素为配合间隙不当、孔径加工精度偏低，或开合模过程中异物掉入配合面造成接触面划伤咬合。室温状态下使用手施力活动相关配合件时不应出现卡滞现象 [[S41,S4,S28]]。

### 飞边

芯头与芯座长期磨损后接合缝出现间隙，或研合端面磨损后形成缝隙，导致压铸件对应通孔周边出现多余金属薄片。严重时残留在缝隙中的飞边在下次压铸时造成铸件缺肉类缺陷 [[S4,S11,S20]]。

### 焊补不良引发的断裂

压铸小型芯底部焊补不良会引发断裂失效，特征为断裂面分为含夹杂物的粗晶粒区、细晶粒区和中等晶粒区。焊补区深度约1.5mm处邻近焊缝钢淬硬至HRC52，主体部分因过度回火硬度降至HRC40 [[S44,S7]]。

## 设计与维护规范

我国现行国家标准 GB/T8844-2017《压铸模 技术条件》规定 [[S22,S42]]：

- 压铸芯头等成型零件不允许存在裂纹、机械损伤和锈蚀等影响使用的缺陷
- 成型部位未注公差尺寸的极限偏差需符合标准相关表列数值要求
- 推荐成型零件经热处理后可进行渗氮、氧化或PVD表面强化处理

压铸模总装规范要求，室温状态下所有活动芯头的相互关联活动配合部分用手施力时不应存在卡死现象，导柱导套等导向件配合公差需符合GB/T1801系列标准要求 [[S41,S37]]。
