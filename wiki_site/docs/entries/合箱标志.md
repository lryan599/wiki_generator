## 概述

“合箱标志”在铸造语境下存在两种不同指代，需加以区分：其一为压铸件表面因模具动、定模合模而自然形成的线状痕迹，本质上属于成型工艺特征；其二为模具上用于保证合模对位的定位标记，属于模具制造与装配特征[[S3,S25,S29]]。

需要特别指出，“合箱标志”术语最初收录于砂型铸造国家标准体系（中文标准编号6.2.73，英文对应 assembly mark），定义为“在没有合箱销的情况下，保证上、下箱正确配合的标志”[[S25,S29]]。在压铸行业国家标准与通用技术手册的官方术语体系中，该术语未被正式收录，属于压铸生产现场广泛使用的工厂口语化表述[[S3]]。下文按两种含义分述。

## 含义一：铸件合箱痕迹（分型线痕迹）

### 定义

压铸件表面的合箱痕迹，正式术语为**分型线痕迹**（Parting Line Mark），指压铸模具动模与定模合拢时，在铸件分型面处形成的线状表面痕迹，属于压铸成型的常规工艺特征[[S21,S38]]。其英文对应术语 Parting line mark 在国际压铸工程文档中广泛使用[[S21,S38]]；在压铸件外观检验场景中，也常使用 Witness line 指代此类可见的合模匹配痕迹[[S12,S10]]。

### 形成机制

合箱痕迹的本质是高压合金液在充填过程中渗入动、定模分型面之间的微小配合间隙，冷却凝固后残留在铸件表面形成的痕迹[[S19,S28]]。分型面一般处于铸件外形轮廓转接处，即铸件正投影外缘位置，痕迹呈现封闭的凸起线状几何形态[[S19,S28]]。

当分型面配合间隙极小、模具加工精度极高时，几乎难以观察到明显的凸起状合箱痕迹[[S19,S28]]。反之，以下因素会增大配合间隙并加剧痕迹凸起程度：

| 影响因素 | 作用机制 | 来源 |
|---------|---------|------|
| 锁模力不足、合模未锁紧 | 动定模之间产生额外间隙 | [[S34,S36]] |
| 压射速度过高 | 形成过大的压力冲击峰，推动合金液进入分型面缝隙 | [[S34,S2]] |
| 模具镶块、滑块长期磨损 | 分型面不平齐，持续产生严重飞边痕迹 | [[S36,S34]] |

### 外观形貌分类

按严重程度和成因，合箱痕迹可分为三类：

| 类型 | 形貌特征 | 成因条件 | 来源 |
|------|---------|---------|------|
| **正常线状合模痕迹** | 封闭、低凸起的线状痕迹，无多余金属薄片伸出 | 分型面配合状态良好 | [[S19,S28]] |
| **错位台阶状错模痕迹** | 高度差明显的台阶状痕迹 | 动定模合模错位、导向机构偏移或分型面存在阶梯式突变（Parting line step） | [[S38,S18]] |
| **飞边凸起状合箱痕迹** | 突出的多余金属薄片（joint flash），随模具使用次数增加不断延伸扩大 | 合金液大量溢出分型面缝隙冷却 | [[S16,S34]] |

### 对产品质量的影响

合箱痕迹对铸件质量的影响主要体现在以下方面：

- **尺寸精度**：分型面制造精度偏差与锁模偏差会直接影响位于分型面处的零件尺寸公差，导致同轴度等精度要求难以保证[[S26,S1]]。
- **气密性**：分型面处的飞边或错模台阶在装配后会形成密封间隙，破坏结合面的密封性能，产生泄漏风险[[S5]]。
- **外观与后续加工**：明显凸起或大面积飞边的合箱痕迹无法直接满足外观要求，必须额外进行打磨、去飞边作业，显著增加后续机加工工时和生产成本[[S33,S23]]。

### 控制与改善措施

合箱痕迹的控制可从模具设计、工艺参数与模具维护三个维度展开[[S34,S36,S28,S18,S4,S26]]：

| 控制维度 | 具体措施 | 适用场景与要点 | 来源 |
|---------|---------|---------------|------|
| **模具设计** | 分型面优先选择零件最大脱模截面处的光滑平整位置，避免在外观面上设置分型面 | 全线适用 | [[S4,S26]] |
| | 分型面封胶面配合间隙控制在0.03~0.05mm | 全线适用 | [[S4]] |
| | 封胶面宽度：中小型模具10~15mm，大型模具30~50mm | 按模具尺寸选取 | [[S4]] |
| **工艺参数** | 根据铸件投影面积合理计算所需合模力，确保锁模到位 | 轻度痕迹时优先检查 | [[S34,S36]] |
| | 调整压射增压机构，降低不合理压射增压峰值 | 飞边集中在浇注系统区域时 | [[S34,S36]] |
| | 匹配最优压射速度，避免过大压力冲击撑开分型面 | 全线适用 | [[S34,S36]] |
| **模具维护** | 生产前例行清理分型面残留的金属杂物 | 每班次例行 | [[S36,S34]] |
| | 定期点检分型面、镶块、滑块磨损情况 | 定期维护 | [[S36,S34]] |
| | 对磨损位置进行焊补打磨修复，保证分型面平齐 | 磨损修复 | [[S36,S34]] |

**针对性解决方案**：

- **轻度线状合箱痕迹**：检查确认锁模力、压射参数在正常范围，对分型面进行轻微抛光处理即可消除[[S28]]。
- **中度错模台阶状痕迹**：校准压铸机动定模导向机构，修正合模同轴度，调整镶块安装位置补偿错位量[[S18]]。
- **重度飞边凸起状痕迹**：以90D左前盖压铸件为例，当调整合模力和压射参数无效时，判定为镶块滑块磨损，对模框焊补打磨平整后飞边完全消除；以霸道款左前盖为例，锁模力达标但飞边集中在浇注系统区域时，调整压射增压机构降低增压峰值即可消除[[S36,S34]]。

### 与相关概念的区别

| 相关概念 | 与合箱痕迹（铸件类）的关系 | 来源 |
|---------|-------------------------|------|
| **分型线（Parting Line）** | 分型线是模具动定模接触配合的物理边界，合箱痕迹是该边界在铸件上留下的可见痕迹，前者属模具结构，后者属铸件表面特征 | [[S21,S38]] |
| **飞边（Flash）** | 飞边是分型面处溢出成型的多余金属薄片，属于可去除的成型缺陷；合箱痕迹正常状态下无明显突出量，不属于缺陷范畴 | [[S16]] |
| **披锋** | 压铸行业口语中对分型面溢出薄型多余毛刺的俗称，本质与飞边一致，与合箱痕迹无重叠含义 | [[S36]] |
| **错模/错型（Mismatch/Mold Shift）** | 错模是动定模合模错位后形成的上下偏移缺陷，属精度不合格瑕疵；正常合箱痕迹无错位偏移，是精准匹配后的正常痕迹 | [[S14]] |

## 含义二：模具合箱定位标记

### 定义

模具类合箱标志指在压铸模具上专门刻制或留存的定位标记，用于在无导柱辅助时保证动、定模的精准对位，功能与“合模对位标记”完全一致，属于压铸模具制造与装配场景下的同义表述[[S25,S29]]。

该定义来源于砂型铸造标准中的 assembly mark：在无合箱销的情况下，保证上、下箱正确配合的标志[[S25,S29]]。在压铸模具场景下，该标记用于合模前辅助动定模的快速对齐定位。

### 与铸件表面标志的关联

压铸工艺可在铸件表面铸出凸形标识[[S8]]，因此模具上预留的合箱定位标记（如刻线、编号等）也能够在铸件表面复现为可见的凸起或凹陷标识，形成一种特殊的铸件表面标志。

| GB/T 6414 / GB/T 15114 / GB/T 25747 | 跨分型面线性尺寸公差 | 在基础尺寸公差上叠加对应分型面投影面积的附加公差；错型偏差值不得超出对应CT公差等级的允许范围 | [[S9,S7,S31]] |
| GB/T 29092-2012 | 飞边缺陷术语与判定 | 分型面残留凸起飞边、错位阶梯痕迹纳入缺陷术语范畴；超出验收限值的合箱痕迹判定为不合格 | [[S16,S37]] |

| ISO 8062 | 铸件尺寸公差体系中分型面偏差适配规则 | CT公差等级数值已包含常规合箱错位带来的偏差增量 | [[S9]] |

### 不同下游行业的差异化验收要求

| 行业应用领域 | 合箱痕迹允许限值 | 来源 |
|------------|---------------|------|
| 3C电子消费类产品外观装饰面 | 凸起高度 ≤0.05mm | [[S24,S22]] |
| 汽车非外观结构件 | 凸起高度 ≤0.15mm | [[S24,S22]] |
| 汽车动力系统功能装配面 | 凸起高度 ≤0.1mm | [[S24,S22]] |
| 航空航天承力类压铸件 | 分型面合箱痕迹完全修除，无凸起残留 | [[S24,S22]] |

> 注：各行业的具体限值最终以供需双方技术协议约定为准[[S24,S22]]。

### 标准检验方法

压铸件合箱痕迹的常规检验采用目视法，凸起高度与错位偏移量使用游标卡尺、高度规等量具进行定量检测，表面粗糙度可采用对应合金的标准比较样块进行比对核验[[S6,S11]]。
