## 概述

石墨型砂（graphite molding sand）是以石墨颗粒为核心耐火骨料，添加黏结剂混制成的特种型砂，属于非石英系造型材料 [[S19,S22]]。GB/T 5611—2017《铸造术语》第4.5.16条给出了正式定义：用细颗粒石墨电极粒添加膨润土4%～5%和水4%～5%混制成的型砂，其热导率比一般湿型砂高80% [[S22,S29]]。该术语对应的英文正式名称为graphite molding sand；graphite foundry sand为通用铸造领域泛称，graphite core sand特指以石墨为骨料的砂芯制备用砂，graphite sand为行业简化口语表述 [[S3,S24]]。

石墨型砂的核心优势在于极高的导热能力和极低的热膨胀系数，使其能够作为激冷造型材料精准控制铸件冷却速率，缩小凝固区域宽度，从而显著降低缩松、渗漏缺陷概率 [[S6,S27]]。同时，石墨化学活性低，不被铝、锌、镁等常见有色合金及其氧化物浸润，抗粘砂表现优异 [[S11,S27]]。

需要明确的是，石墨型砂与石墨涂料、石墨脱模剂/离型剂、石墨乳等辅助材料存在本质区别：石墨涂料是涂刷于普通砂型型腔表面的悬浮涂覆材料，不构成整体型砂骨架；石墨脱模剂/离型剂为液态润滑介质，仅用于金属型型腔表面脱模润滑；石墨型砂要求石墨颗粒占比90%以上，普通改性石英砂中石墨添加量通常低于5%，不属于石墨型砂范畴 [[S22,S29]]。

## 分类

### 按造型用砂体系归属

石墨型砂属于非石英系特种造型用砂，与石英系砂（天然石英砂、人造石英砂）骨料体系完全不同 [[S19]]。

**图1** 造型用砂分类示意图，石墨型砂归属于非石英系特种砂 [[S19]]。

### 按石墨骨料种类分类

石墨型砂所用石墨骨料可分为三类 [[S27,S21]]：

- **鳞片石墨**：按GB/T 3518—2008分为高碳石墨（固定碳94.0%～99.0%）、中碳石墨（80.0%～93.0%）、低碳石墨（50.0%～79.0%）。铸造行业主要采用中碳和低碳品级 [[S27,S21]]。
- **微晶石墨（无定形/土状石墨）**：执行GB/T 3519—2008标准，W系列（无铁要求）牌号为铸造常用牌号 [[S27]]。
- **人造石墨颗粒**：多由废石墨电极、废石墨坩埚破碎获得，属铸造厂内部废料综合利用 [[S27,S21]]。

### 按黏结剂类型分类

石墨型砂按黏结剂体系可分为多种类型 [[S13,S28]]：

| 黏结剂类型 | 典型特征 |
|-----------|---------|
| 膨润土-水体系 | 常规湿型石墨砂，加水混制后直接造型 |
| 合脂体系 | 以合脂为黏结剂，湿强度约0.15 kg/cm²，造型适宜温度40～60℃ |
| 树脂体系 | 环氧树脂等，需稀释剂配合 |
| 水玻璃体系 | 化学硬化类 |
| 磷酸盐体系 | 化学自硬类 |
| 卤化物体系 | 特种黏结剂，可大幅降低焙烧温度 |

### 按制备工艺分类

- **机加工石墨型**：由石墨块机械加工成型，表面致密，导热性极高，模具可重复使用，但成本高 [[S10,S18]]。
- **捣实石墨型**（石墨型砂芯）：由石墨型砂捣实成型后焙烧硬化，工艺简单成本低廉，退让性好，通常为单次使用 [[S10,S18]]。

### 按粒度规格分类

根据《航空制造工程手册──特种铸造》的分类 [[S20]]：

| 粒度规格 | 主要用途 |
|---------|---------|
| 50/80目 | 常规通用砂 |
| 60/100目 | 提升铸件表面质量 |
| 粗砂 | 用作背砂，改善透气性 |

## 组成与配比

### 膨润土基石墨型砂

典型配比为：细颗粒石墨电极粒100%（基准），外加膨润土质量分数4%～5%，水质量分数4%～5%，混制后可直接造型 [[S22,S29]]。

### 合脂基石墨型砂

石墨粉100%（基准），外加合脂黏结剂质量分数15%。湿强度为0.15 kg/cm²，造型适宜温度区间为40～60℃ [[S13,S28]]。

### 捣实石墨型砂的多组分体系

《有色铸造合金及熔炼》列出了捣实石墨型砂的三类典型配比体系 [[S14,S5]]：

| 黏结剂类型 | 稀释剂 | 备注 |
|-----------|--------|------|
| 多组分黏结剂（含T99-1油、环氧树脂等） | 丙酮、乙二胺 | 丙酮、乙二胺不计入黏结剂加入量 |
| 单组分黏结剂 | 酒精 | — |
| 水溶性黏结剂 | 煤油或水 | — |

注：表中序号3配方为石墨粉100%，外加15%合脂黏结剂 [[S14]]。

## 核心物理与热学性能

### 基本物理性能

石墨骨料的密度为2.21～2.25 g/cm³，假比重1.55～1.70 g/cm³（孔隙度20%～30%时）；石墨的强度随温度升高而增加，在2500℃时达最大值 [[S11]]。

### 热学性能

| 性能参数 | 数值 | 来源 |
|---------|------|------|
| 室温导热系数 | 100～110 kcal/(m·h·℃) | [[S11]] |
| 线膨胀系数 | (0.5～4)×10⁻⁶/℃ | [[S11,S27]] |
| 天然鳞片石墨耐火度 | ＞3000℃ | [[S27]] |
| 工业级石墨耐火度 | 约2100℃ | [[S27,S11]] |
| 大气中氧化起始温度 | ＞450℃ | [[S11]] |

石墨的热容量为0.17 kcal/(kg·℃)，几乎是铸铁的一倍；室温导热系数比砂型大150倍，比不锈钢大4倍 [[S11]]。

### 高温密度变化

石墨类无机砂密度随温度升高而降低：0℃时约1460 kg/m³，500℃时降至约1410 kg/m³，2000℃时降至约1370 kg/m³ [[S15]]。

## 工艺参数与操作条件

### 混制工艺

**油类-树脂体系** [[S13,S28]]：
1. 石墨粉加热至100～150℃，干混2～3分钟；
2. 加入油类黏结剂，混制5分钟；
3. 加入稀释的环氧树脂（配套丙酮等稀释剂），混制10～15分钟后出砂。

**合脂体系** [[S13,S28]]：
- 石墨粉加热至100～150℃，合脂加热至120℃后直接加入混制，可简化工艺。
- 合脂酸值要求45～65，造型适宜温度为40～60℃；温度过低流动性差，过高易粘模。

### 固化与焙烧

石墨型砂的烘干温度区间为400～450℃ [[S16]]。对于需高温焙烧的工艺，最高可升温至950～1000℃；焙烧时应将铸型埋入石墨填砂中或在惰性气氛（N₂或H₂）下保护，以防高温氧化 [[S28,S18]]。对于航空铸件等质量要求严格的场景，焙烧后需在约1000℃下进行2 h真空除气处理 [[S18]]。

低温烘干阶段需严格控制升温速度：从室温升至220～250℃，每隔20～30℃保温2～3小时；至220～250℃后保温5～6小时使铸型固化 [[S28]]。

### 使用前预热与合金适配

- 用于镁锂合金等高活性合金时，铸型必须预先完全烘干脱气 [[S22,S29]]。
- 预热温度可根据铸件合金种类调整 [[S16]]。

### 循环使用寿命

| 铸型/砂芯类型 | 适用合金 | 可重复浇注次数 |
|-------------|---------|--------------|
| 机加工石墨型 | 铝合金 | 300～1000次 |
| 机加工石墨型 | 铜合金 | 100～200次 |
| 捣实石墨型砂芯 | 各类 | 通常单次使用 |

机加工石墨型的使用寿命数据来源于 [[S8,S10]]。

## 在压铸及铸造工艺中的角色

石墨型砂在压铸领域的适用场景有限，主要用于**半永久型压铸**和**低压压铸**场景下的砂芯制备，不能直接用作整体承压铸型 [[S22,S29]]。其核心工艺作用包括：

1. **激冷控制**：石墨型砂的高导热性使其能够作为激冷成型块，用于加快铸件热节部位的冷却速度，缩小凝固区域宽度，增强补缩，缩小缩松面积 [[S6,S22]]。

2. **晶粒细化**：快速冷却可增大结晶过冷度，减小有效晶核临界尺寸，细化晶粒组织 [[S8,S11]]。

3. **减少铸造应力**：极低的线膨胀系数可大幅降低铸件热应力，减少裂纹倾向 [[S11,S27]]。

需要特别指出：常规高压压铸工况下，因石墨型砂的承压强度不足，存在使用限制 [[S22,S29]]。

## 合金兼容性

| 合金类型 | 兼容性评价 | 说明 |
|---------|-----------|------|
| 镁合金 | 最佳适配 | 化学惰性高，不被浸润 |
| 含镁量较高的铝合金 | 最佳适配 | GB/T 5611明确推荐 |
| 钛合金 | 适配 | 广泛用于钛合金铸造砂型/砂芯 |
| 铜合金（锡青铜等） | 适配 | 激冷细化晶粒，提升致密性 |
| 锌合金 | 兼容性较好 | 浇注温度低，影响不大 |
| 高碳钢/低合金钢 | 兼容性差 | 存在渗碳污染缺陷风险 |

以上兼容性判断依据 [[S22,S27,S8]]。

## 优势与局限性

### 核心优势

1. **高导热激冷效应**：可精准控制铸件热节区域冷却速率，缩小凝固区域宽度，显著降低铸件缩松、渗漏缺陷概率 [[S6,S27]]。
2. **极低热膨胀系数**：大幅减少铸件热应力，降低裂纹倾向 [[S11,S27]]。
3. **化学惰性高**：不被金属液及其氧化物浸润，抗粘砂表现优异，清砂难度远低于常规石英砂型 [[S11,S27]]。
4. **高温强度随温度上升而增强**：2500℃时强度达峰值，是一种理想的铸型材料 [[S11]]。
5. **尺寸精度与表面光洁度高**：膨胀系数小，铸造精度好 [[S11,S8]]。

### 局限性

1. **材料成本高**：石墨型砂成本为普通石英砂的5～8倍，大规模推广存在成本门槛 [[S27]]。
2. **常温机械强度偏低**：抗剪、抗压强度低于铬铁矿砂，造型紧实度不足时易产生砂芯破损缺陷 [[S27]]。
3. **高温氧化风险**：大气环境中450℃以上即可开始氧化烧损，浇注过程中需控制型腔周边空气流通或施加惰性气氛保护 [[S11,S27]]。
4. **渗碳污染风险**：浇注高碳钢、低碳钢等碳敏感铸件时，易造成表面渗碳超标，影响铸件力学性能 [[S27]]。
5. **承压强度限制**：常规高压压铸工况下无法直接用作整体铸型 [[S29,S22]]。

## 工程应用实例

### 大型气缸套砂芯

某厂采用石墨型砂制备大型气缸套砂芯（石墨屑90%，黏土粉10%），砂芯在400～450℃烘干后使用。对比普通黏土砂芯，铸件本体强度提升15%，片状石墨长度由150～300 μm缩短至100～200 μm并呈菊花状分布，完全消除了热节处局部缩松导致的渗漏缺陷 [[S16]]。

### 半永久型低压压铸锡青铜铸件

采用石墨型铸型进行半永久型低压压铸，可大幅提升合金结晶过冷度，晶粒显著细化，铸件致密性提升，延伸率和冲击韧性成倍增长；工艺出品率相比普通砂型提升11%～23% [[S8]]。

### 航空钛合金铸件

航空领域钛合金铸件批量生产采用石墨捣实型砂芯，可适配复杂腔体结构制备，退让性优异，铸件裂纹发生率远低于陶瓷砂型铸件，生产周期短，综合生产成本远低于整体机加工石墨型 [[S18]]。

## 关联术语与概念辨析

| 术语 | 与石墨型砂的关系与区别 |
|------|----------------------|
| **石墨涂料** | 石墨粉与黏结剂、溶剂制成的悬浮涂覆材料，仅涂刷在普通砂型型腔表面，不属于整体型砂体系 [[S29]] |
| **石墨脱模剂/离型剂** | 液态或半液态润滑介质，仅用于金属型型腔表面脱模润滑，不具备砂芯骨架支撑作用 [[S29]] |
| **石墨乳** | 与脱模剂同类，为液态悬浮液，与石墨型砂在功能和组成上完全不同 |
| **石墨型（graphite mold）** | 整体机加工石墨铸型，属于永久/半永久型范畴，区别于石墨型砂（粒状松散体系）|
| **石墨型铸造（graphite mold casting）** | 采用石墨型或石墨型砂的铸造方法，GB/T 5611中列为独立词条 [[S24]] |
| **碳素砂/焦炭粉砂** | 含石墨粉的激冷砂，但石墨含量和导热能力均低于石墨型砂 [[S16]] |

## 相关标准与规范

### 国内标准

- **GB/T 5611—2017《铸造术语》**：第4.5.16条给出石墨型砂的标准定义与基础性能说明 [[S22,S29]]。
- **GB/T 3518—2008《鳞片石墨》**：规定了铸造用鳞片石墨原料的固定碳含量、粒度等质量指标 [[S27,S21]]。
- **GB/T 3519—2008《微晶石墨》**：规定了铸造用无定形石墨粉的牌号分类与技术要求 [[S27,S21]]。
- **GB/T 2684—2009《铸造用砂及混合料试验方法》**：石墨型砂各项常规性能的测试执行标准 [[S25,S9]]。

### 国外标准

- **ASTM B26**《铝合金砂型铸件》及**ASTM B108**《铝合金永久型铸件》：条款中明确允许特种碳质激冷砂作为局部砂芯/激冷块用于铝合金铸件生产 [[S23,S17]]。
- **ISO 7722**《重力铸造铝合金铸件通用交货验收规范》：明确了石墨基造型材料的使用场景要求 [[S17]]。
