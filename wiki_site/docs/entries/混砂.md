## 概述

混砂（sand mixing; sand mulling）是铸造砂处理工序的核心子环节，指将铸造用砂、粘结剂及附加物通过搅拌、挤压和揉搓作用混制成型砂或芯砂的过程[[S52,S44]]。在压铸生产（含高压、低压、重力铸造）中，混砂直接为制芯工序提供合格的混合料，其混制均匀度、粘结剂包覆效果决定了砂芯的常温强度、高温强度、透气性、发气量及溃散性等核心性能，进而影响压铸件的内腔尺寸精度、表面质量及缺陷发生概率[[S54,S5,S27]]。

混砂位于旧砂处理与配砂工序之后、制芯工序之前，是压铸砂芯制备流程的关键前置工序[[S54]]。其标准英文译名为 sand mixing 或 sand mulling，其中 core sand mixing 特指面向制芯的芯砂混制，sand preparation 则涵盖型砂制备的整体大工序[[S54,S55]]。

## 分类

混砂可从操作模式与作业连续性两个维度分类[[S52,S13,S14]]。

**按操作模式区分：**

| 类别 | 说明 | 适用场景 |
|------|------|----------|
| 手工混砂 | 人工完成配料与混制全过程 | 小批量试制、特殊配方小批量砂制备 |
| 机械混砂 | 利用混砂机完成搅拌、碾压、揉搓作业 | 工业化生产通用方式 |

**按作业连续性区分：**

| 类别 | 典型机型 | 特点与适用场景 |
|------|----------|----------------|
| 间歇式混砂 | 碗形混砂机、碾轮式混砂机 | 配料精确，配方切换灵活，适用于树脂砂、水玻璃砂等配方精度要求高的压铸砂芯混制 |
| 连续式混砂 | 搅笼式连续混砂机 | 连续进料出料，生产率高，适用于自硬砂等大批量混制作业 |

## 主要设备

### 设备分类体系

用于压铸砂芯制备的树脂砂混砂机分为间歇式与连续式两大类。间歇式包含碗形、振动式、涡流式；连续式分为固定式与移动式，连续式又可按混制方式细分为单砂单混、单砂双混、双砂三混等[[S34]]。

![压铸砂芯制备用树脂砂混砂机分类体系](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/92e9bfb9-9c27-4a29-a7e4-40f462514f33/resource)

### 常见混砂机类型

混砂机按结构类型主要分为辊轮（碾轮）式、转子式、碗形间歇式、搅笼连续式等[[S33,S23]]。

**辊轮（碾轮）式混砂机**：核心结构包含碾轮、主轴系统、刮板、碾盘及卸料机构，依靠碾轮的碾压和刮板的翻动作用完成型砂混制，常规为间歇式工作。代表国产系列为S11系列，适用于中小批量压铸砂芯制备场景，混制粘土砂性能稳定[[S23,S45,S53]]。

**转子式混砂机**：核心结构包含高速旋转混砂转子与3支仰角30°的刮板。刮板将物料高速抛起后由转子反向高速抛击混合，混砂周期短、生产率高，混制的型砂松散不结块，适配大型压铸砂芯的自动化生产线需求[[S53,S49,S43]]。

**碗形间歇式混砂机**：是压铸冷芯砂、热芯砂制备的主流间歇机型。核心结构为半球形壳体搭配牛角形搅拌叶片，物料在机内做三维循环运动，无混砂死区且自清洗性能优异。代表国产系列为S20系列，单次混砂量10 kg对应生产率1 t/h，适配小批量多品种压铸砂芯生产需求[[S6,S42,S17]]。

**搅笼式连续混砂机**：核心结构为含单/双螺旋搅拌叶片的卧式混砂槽，物料从进料口连续加入，被边搅拌边推送至出料口。代表国产系列为S24、S25、S28系列，其中滚筒内径1100 mm、长度2500 mm的滚筒固定式连续混砂机混制单一砂生产率可达40 t/h[[S38,S3,S32]]。

头部铸造设备厂商的代表产品包括：德国Eirich公司倾斜底盘转子混砂机（生产率≤60 m³/h）、丹麦DISA公司SAM系列高速转子混砂机、青岛双星S11系列碾轮混砂机、济南铸锻所捷迈S18系列混砂机、德国Laempe公司LM系列冷芯砂混砂机（可实现与射芯机联动现混现用）[[S9,S43,S28]]。

| 对比维度 | 间歇式混砂机 | 连续式混砂机 |

| 适用生产规模 | 小批量多品种 | 大批量连续生产 |

## 核心工艺参数

### 加料顺序

不同粘结剂体系对应的混砂加料顺序不同，合理的加料顺序直接影响粘结剂分散均匀性与最终性能。

**树脂自硬砂**：砂 + 固化剂 → 混匀 → 加树脂 → 混匀 → 卸砂。先投入固化剂混匀后再加入树脂，可避免树脂与固化剂直接接触发生局部提前固化[[S6,S10]]。以S206间歇式混砂机为例，原砂+固化剂混匀时间为20~25 s，加入树脂后混匀时间为35~40 s[[S6,S10]]。

**有机酯水玻璃自硬砂**：组分称量 → 加砂 → 加入有机酯混碾约15 s → 加入水玻璃混碾约30 s → 卸砂[[S15]]。

![有机酯水玻璃自硬砂标准混砂工艺流程](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/41127353-9e35-42e3-87ff-3a57be6d6872/resource)

**壳芯覆膜砂（冷覆膜法）**：原砂加煤油混砂1~2 min → 加入粉状酚醛树脂、粉状乌洛托品干混0.5~2 min → 加入工业酒精湿混2~3 min → 形成塑性砂团 → 吹冷空气冷却8~15 min → 碾碎脆性砂团0.5~1 min → 过筛后待用[[S21]]。

**粘土砂**：先加入新砂与旧砂，初次加水混匀后加入膨润土、煤粉等粉状辅料；若含重油则在初次加水混匀后加入重油；最后补充少量水调整至目标水分。该顺序可使水分在砂粒表面形成均匀水膜，加快粘土分散包覆效率[[S46]]。

### 混砂时间

混砂时间需根据混砂机形式和粘结剂体系确定。时间过短导致粘结剂分布不均匀、包覆不完整；时间过长则砂温升高、水分蒸发，反而降低性能[[S46]]。

| 混砂机类型 | 适用砂类 | 混砂时间 |
|-----------|---------|---------|
| 碾轮式 | 背砂 | 约3 min |
| 碾轮式 | 单一砂 | 3~5 min |
| 碾轮式 | 面砂 | 5~8 min |
| 摆轮式 | 背砂 | 0.5~1 min |
| 摆轮式 | 面砂 | 2~3 min |
| 碗形间歇式 | 冷芯盒砂/热芯盒砂 | 约2 min |
| S206间歇式 | 树脂自硬砂 | 原砂+固化剂20~25 s，+树脂35~40 s |

[[S46,S20,S6,S10]]

### 温度控制

在压铸覆膜砂制芯场景中，混砂温度需严格控制在≤40 ℃，以避免树脂提前固化[[S5]]。冷芯盒砂混砂过程中也应避免揉搓过度导致砂温上升，从而影响芯砂的可使用时间和流动性[[S20]]。

### 水分控制

不同粘结剂体系对水分含量要求不同。水玻璃砂水分需控制在3.0%~5.5%，过量水分会直接提升浇注过程中的发气量，大幅增加气孔缺陷生成概率[[S11]]。

### 粘结剂比例

| 粘结剂体系 | 适用铸件 | 添加量（占原砂质量） |
|-----------|---------|-------------------|
| 三乙胺冷芯盒树脂 | 铸铁/铸钢件 | 1.5%~2.0% |
| 三乙胺冷芯盒树脂 | 非铁合金铸件 | 1.0%~1.5% |
| 呋喃自硬树脂 | — | 0.8%~1.4%，固化剂为树脂质量的15%~45% |
| 水玻璃（高效混砂） | — | 1.5%~2%（比传统工艺降低50%~60%） |

[[S50,S40,S16]]

### 可使用时间

可使用时间是指水玻璃砂和树脂自硬砂混制完成后仍可制出合格砂型（芯）的最长持续时间。生产中常以型砂剩余终强度为初始终强度的80%时对应的时长作为判定基准[[S10,S48]]。可使用时间一般约为预固化总时长的1/3，常规工况下该比值通常控制在0.35~0.60区间，最高可达到0.75[[S10]]。在现场生产中，观察到混好的型砂颜色由黄变绿、开始发黏时，即认为已超过可使用时间[[S10]]。

## 在压铸流程中的作用与质量控制

### 核心质量指标限值

混砂后混合料的质量直接决定砂芯的工作性能。主要控制指标如下[[S4]]：

| 指标 | 控制范围 |
|------|---------|
| 树脂砂终强度（一般型砂） | 0.6~0.8 MPa |
| 树脂砂终强度（普通芯砂） | 0.8~1.0 MPa |
| 树脂砂终强度（复杂薄壁砂芯） | 1.6~2.0 MPa |
| 树脂砂透气性 | 约400 |
| 每克树脂砂发气量 | ≤10~11 mL |

### 对压铸件缺陷的影响

**砂眼缺陷**：混合料强度不足时，浇注过程中金属液的冲击作用会直接冲散未完全包覆的砂粒，砂粒随金属液卷入型腔内部形成砂眼[[S18,S7]]。

**气孔缺陷**：混合料透气性不足时，浇注过程中水分、粘结剂受热析出的大量气体无法及时通过砂粒间隙排出，侵入未完全凝固的金属液中形成气孔；严重时会出现浇不足、呛火等安全问题[[S18,S31]]。

### 质量控制标准

现行检测标准GB/T 2684《铸造用砂及混合料试验方法》规定了铸造用砂及混合料的含水量、含泥量、粒度、紧实率、透气性、强度、酸耗值、灼烧减量、发气量和发气速度的统一试验方法。含水量的测定支持快速红外烘干法（110 ℃~170 ℃烘干6~10 min）和105 ℃~110 ℃恒重法两种路径[[S30,S1]]。

## 相关材料

### 常用原砂

压铸砂芯制备常用原砂以硅砂为主，其二氧化硅含量需满足GB/T 9442的要求。该标准按含泥量将硅砂分为0.2、0.3、0.5、1.0、2.0五个等级，其中精选砂含泥量不大于0.2%，水洗砂含泥量不大于1.0%[[S22,S29,S26]]。非石英系特种原砂（如锆砂、铬铁矿砂、镁橄榄石砂）多用于大型铸钢件或对特种性能有要求的工况[[S22]]。

### 常用粘结剂体系

**水玻璃粘结剂**：无机体系，核心原料为硅酸钠。混制及浇注过程无有毒有害气体析出，成本低廉；但常规工艺下溃散性较差，砂芯易吸潮，旧砂回用难度较高[[S39]]。

**热芯盒树脂**：有机热固化体系，常用呋喃或酚醛树脂。芯盒加热至200 ℃~280 ℃使砂芯快速硬化，生产效率高，砂芯强度高、尺寸精度好；但生产过程会释放甲醛等刺激性气体，部分高氮配方易导致铸件产生针孔缺陷[[S36]]。

**冷芯盒树脂**：多以酚醛树脂+聚异氰酸酯双组分为粘结剂，以三乙胺为催化剂。常温下即可快速硬化砂芯，无需加热芯盒，能耗低，砂芯尺寸稳定性好，可长期存放；但催化剂易燃易爆且有毒，尾气排放污染环境，砂芯在高湿环境下易吸潮导致强度下降[[S36,S39]]。

## 与周边词条的关系

混砂词条与以下铸造领域词条存在紧密关联[[S52,S54,S19]]：

| 关联词条 | 关系说明 |
|---------|---------|
| 芯砂 | 混砂的直接产出物，即混制完成的合格混合料 |
| 配砂 | 混砂的前置工序，负责拟定砂组分种类与配比 |
| 制芯 | 混砂的下游工序，使用混制合格的芯砂成型砂芯 |

| 铸造缺陷（砂眼、气孔） | 混砂工艺异常可能直接引发砂眼、气孔等常见铸造缺陷 |
