## 概述

喷砂机器人（blasting robot）是依据国家标准GB/T 25370—2020《铸造机械 术语》正式定义的自动化表面清理设备，官方释义为“自动完成铸件表面喷砂作业的系统”[[S20]]。在国内压铸后处理行业中，该术语在实际应用中常兼指两类技术路线：以压缩空气为动力、驱动磨料从喷嘴高速射出的气动负压喷砂系统；以及利用高速旋转叶轮离心力将弹丸抛射至铸件表面的离心轮式抛丸系统。前者多用于铝合金压铸件的精细表面处理，后者则清理效率更高、冲击力更强，行业常将两类设备的集成自动化系统统称为喷砂/抛丸机器人[[S7]]。

需严格区分的是，shot blasting/sandblasting（喷砂/抛丸清理）与shot peening（喷丸强化）在工艺目的上存在本质差异：前者以去除压铸后铸件表面氧化皮、飞边与粘砂为核心功能，属于压铸后处理精整工序；后者则旨在导入表层残余压应力以提升疲劳强度[[S1]]。其通用英文别名包括Shot Blasting Robot、Sandblasting Robot、Robotic Abrasive Blasting System、Automated Blast Cleaning Cell等[[S1]]。

## 分类

依据现有技术资料，喷砂/抛丸机器人可从机械结构、磨料种类、工作方式及系统集成度四个维度进行分类。

### 1. 按机器人本体结构分类

多自由度关节型（通常为6轴）、龙门式、桁架式三类是适配不同压铸清理场景的主流结构[[S13]]。

### 2. 按磨料种类分类

压铸后处理中适配的主流磨料包括钢丸/不锈钢丸、玻璃珠、陶瓷砂/刚玉/碳化硅三类[[S22]]。针对镁合金压铸件，建议采用铝丸、玻璃丸和氧化铝丸，不宜使用含Fe、Ni或Cu的钢丸，以避免引入降低抗蚀性的杂质[[S14]]。

### 3. 按工作方式分类

分为干喷砂与湿喷砂两类，其中湿喷砂可在磨料中加入缓蚀剂，有效减少作业区粉尘污染，同时可防止铝合金压铸件处理后发生返锈[[S19]]。

### 4. 按系统集成度分类

分为单独立式喷砂清理工作站、以及与压铸生产线联动的自动清理单元两类[[S12]]。

## 核心结构与组成

一套完整的喷砂/抛丸机器人系统由以下核心部件构成[[S2]]：

- **工业机器人本体**
- **喷砂喷嘴**（气动负压式）或**离心抛丸器**（离心轮式）
- **磨料供给回收循环系统**：通常包含斗式提升机、丸砂分离器、螺旋输送机构等
- **封闭除尘系统**
- **工件旋转转台**或**专用夹紧工装**
- **电气控制系统**
- **可选配置**：自适应压力调节传感器、视觉定位识别传感器

## 关键工艺参数

针对有色合金压铸件的喷砂处理，核心工艺参数及影响关系如下表所示。

| 参数名称 | 典型设定/范围 | 影响说明 |
|:--------|:------------|:--------|
| 喷射压力 | 0.4–0.7 MPa[[S18]] | 压力过高易导致薄壁铸件局部形变及砂粒嵌入缺陷；压力过低则出现局部清理不完全、粗糙度不足[[S24]] |
| 喷嘴冲击夹角 | 不小于60°，推荐60°–90°[[S15,S18]] | 夹角过小将显著降低磨料对表面的切削效率 |
| 喷嘴工作距离 | 有色合金压铸件推荐100–200 mm[[S18]] | 薄壁件可适当增大距离以防止变形；平面件可比异形件略远[[S15]] |
| 表面粗糙度 | Ra 2.5–12.5 μm[[S5]] | 通过调整磨料粒度与压力控制；粗糙度过大（超Ra 10 μm后）涂层结合强度提升效果递减[[S5]] |
| 磨料粒度 | 依表面要求选择 | 直接影响喷砂后表面纹理与颜色[[S15]] |

## 在压铸流程中的角色

在全压铸生产链条中，喷砂机器人的工艺位置位于去浇口飞边、基础清理工序之后，涂装/喷粉/阳极氧化等前处理工序之前[[S26]]。以下流程图展示了壳体压铸件的完整生产工艺路径，其中表面清理工序为喷砂作业所处的环节[[S16]]。

![壳体压铸生产工艺流程图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/b609905c-10a3-4d5b-bf29-5ac05e8d7a26/resource)

喷砂机器人承担的核心功能包括[[S4,S14,S22,S26]]：

- 彻底清除铸件表面残留的型砂、模具附着物、氧化皮和微观毛刺；
- 将工件表面均匀粗化以提升后续涂层附着力，避免因局部覆盖不足导致的涂装炸粉缺陷；
- 在返修处理场景中对缺陷铸件进行表面修整；
- 通过表面冲击作用改善铸件抗疲劳性能，实现表面强化。

就与相邻工序的关系而言：去毛刺工序先行去除宏观飞边与浇口残留，清洗工序后续去除喷砂残留粉尘，最终进入涂装或阳极氧化环节。

## 典型应用对象

喷砂机器人在压铸行业主要应用于以下品类[[S9,S21]]：

- **汽车零部件**：铝合金气缸体、变速箱壳体、发动机支架、曲轴箱盖
- **3C电子**：镁合金笔记本外壳
- **其他**：各类铝合金/锌合金壳体、支架类薄壁件

上述铸件普遍具有复杂内腔或薄壁结构特征，对表面清理的均匀性和精细化程度要求较高，适合采用机器人自动化喷砂作业。

## 优势与局限

| 维度 | 说明 |
|:----|:----|
| **优势** ||  
| 一致性 | 喷砂覆盖均匀度不受人工熟练度差异影响[[S25]] |
| 效率 | 单位面积作业效率较人工提升约30%–100%[[S25,S27]] |
| 职业健康 | 作业人员完全隔绝于高浓度矽尘环境，彻底消除矽肺类风险[[S25]] |
| 适应性 | 适用于异形、复杂曲面铸件的均匀清理[[S14]] |
| **局限** ||  
| 投资成本 | 设备初期采购投入远高于普通人工喷砂柜[[S25]] |
| 维护门槛 | 需专业技术人员完成离线编程与日常维护[[S25]] |
| 可达性限制 | 对尺寸极小、开口受限的深层复杂微型内腔，可达性不足[[S25]] |
| 磨料消耗 | 年磨料消耗成本随处理工件总量增加而较高[[S25]] |

## 安全与环保规范

国内现行的抛喷丸设备行业安全技术规范对喷砂机器人的安全环保指标提出明确要求[[S3,S17]]：

| 指标 | 限值 |
|:----|:----|
| 单圆盘抛丸器空载噪声 | ≤88 dB(A) |
| 整机正常工作噪声 | ≤95 dB(A) |
| 除尘系统排放粉尘浓度 | ≤150 mg/m³ |
| 铝合金设备防护要求 | 配套防爆防护结构，防范铝粉爆炸风险 |
| 丸砂分离后含砂量 | ≤1% |
