## 概述
**支承柱**是压铸模动模部分的核心支撑元件。国家标准GB/T 4678.18-2024《压铸模 零件 第18部分：支承柱》将其标准中文名称定为“支承柱”，对应英文为“Support pillars”。该标准属于国家标准序列，是该零件的规范性技术依据[[S14]]。行业内“支撑柱、撑柱、支柱”等均为该零件的非标准俗称[[S14]]。

在压铸模中，支承柱安装于动模支承板与动模座板之间，位于两侧方铁的包围区域内，核心作用是承受压铸过程中的锁模力载荷，抑制动模支承板的弯曲变形，同时为推板组件提供刚性支撑[[S20,S3]]。

## 术语界定
**压铸模支承柱**不指代铸件自身的结构件。后者在行业内独立称为“铸件支撑柱/支撑杆”，属于压铸件本体的功能结构，与模具零件属性完全独立[[S11]]。

与塑胶模具的“支撑柱/撑柱”相比，二者存在明确定义差异：塑胶模支撑柱多数为非标准件，由模具制造厂商自行设计加工；压铸模支承柱已纳入国家标准序列。同时，压铸过程中金属液的胀型力和锁模力远高于注塑工况，因此压铸用支承柱的承重设计要求显著高于塑胶模同类零件[[S16]]。

## 分类与结构型式
依据GB/T 4678.18-2024，压铸模用支承柱分为三种型式[[S15,S17]]：

| 型式 | 结构特征 | 备注 |
|------|----------|------|
| A型 | 等直径无安装台阶的整体圆柱型 | 基础通用型式，直径范围25mm~120mm，长度范围80mm~450mm[[S21]] |
| B型 | 带安装定位台阶的圆柱型 | 安装定位段直径 \(D_1\) 公差等级为H7[[S17]] |
| C型 | 矩形块状支承结构 | 2024版标准新增，适配更大载荷下的支撑需求[[S14]] |

从结构装配形式看，支承柱也可分为**整体式**与**组装（分体）式**两类，两类结构均可通过调整尺寸参数提升动模支承板的抗弯强度[[S6]]。

## 材料与性能要求
GB/T 4678.18-2024对支承柱的材料与热处理提出以下推荐要求[[S17,S15]]：

| 项目 | 技术规定 |
|------|----------|
| 推荐材料 | 45钢 |
| 硬度要求 | 28 HRC ~ 32 HRC（调质处理） |
| B型安装定位段 \(D_1\) 公差 | H7（符合GB/T 1800.1—2020） |
| 形位公差 | 两端面平行度、轴线对端面垂直度，符合GB/T 1184—1996中6级精度[[S18]] |
| 其他技术要求 | 符合GB/T 4679《压铸模 零件 技术条件》的规定[[S17]] |

## 标准标记规则
GB/T 4678.18-2024规定了支承柱的标准标记格式[[S15]]：

**标记依次包括**：描述段“支承柱” + 标准段“GB/T 4678.18” + 特性段（型号 + 尺寸参数 + 材料）。

**示例**：
- A型：`支承柱 GB/T 4678.18-A-40×80-45` （直径40 mm，长度80 mm，45钢）
- C型：`支承柱 GB/T 4678.18-C-200×50×100-45` （长度200 mm，宽度50 mm，高度100 mm，45钢）[[S15]]

## 装配与安装要求
### 安装位置
支承柱安装在动模支承板与动模座板之间，穿过推板的避让孔，位于两侧方铁的包围区域内，与方铁共同构成受力支撑体系，分担压铸过程中的胀型力[[S12,S22]]。

### 布置原则
支承柱的安装位置应对称设置在压射投影的受力区域内，以有效增强动模支承板的刚性[[S3]]。

### 关键装配尺寸
支承柱与相关部件的装配间隙要求如下[[S22]]：

| 装配关系 | 尺寸要求 |
|----------|----------|
| 支承柱与推板间隙 | 单边 1.5 ~ 2.0 mm |
| 支承柱与方铁边缘距离 | ≥ 25 mm |
| 支承柱总高度（相对方铁高度） | 高出量根据模具宽度分段确定 |

### 高度高出量要求
支承柱的总高度应高于对应规格方铁的高度，高出量按模具宽度分段设置[[S22]]：

| 模具宽度 W（mm） | 支承柱高出方铁量（mm） |
|------------------|------------------------|
| W < 300 | 0.05 |
| 300 ≤ W < 400 | 0.10 |
| 400 ≤ W ≤ 700 | 0.15 |
| W > 700 | 0.20 |

## 数量选型方法
支承柱的数量配置可通过定量计算确定[[S22]]：
1. 计算两侧方铁之间的总面积：\(A = L \times W\)（\(L\) 为推杆板长度，\(W\) 为方铁间距）；
2. 根据模具规格选取系数 \(n_1\)、\(n_2\)；
3. 计算总支撑面积：\(S = A \times n_1 \times n_2\)；
4. 按选定的单根支承柱横截面积推算所需配置数量[[S22]]。

## 常见失效模式
压铸模支承柱在服役中可能出现以下失效形式[[S13,S19]]：
- **塑性弯曲**：承受载荷超过材料屈服强度，引发过载变形；
- **疲劳断裂**：数万次压铸循环产生的交变载荷作用下发生的低应力脆性断裂；
- **表面磨损**：与支承板端面长期接触摩擦导致的材料耗损和尺寸减小。

不符合安装精度要求的支承柱将导致动模支承板挠度超标，分型面产生间隙，最终使压铸件出现**飞边、厚度尺寸超差**等质量缺陷，严重时还会导致推出机构运动卡滞[[S13]]。

## 相关术语
- **支承板**：与支承柱端面接触的模板，是支承柱直接承载的对象；
- **垫块（方铁）**：与支承柱共同构成动模侧受力支撑体系的零件[[S12]]；
- **推板**：推出机构组件，支承柱需穿过推板避让孔安装[[S20]]；
- **复位杆**：模具推出机构复位元件，与支承柱同属动模侧组件[[S12]]。
