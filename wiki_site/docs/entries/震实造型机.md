## 概述

震实造型机（jolt molding machine）是砂型铸造中采用震实方法紧实型砂的造型设备，依靠工作台带动砂箱、模板及型砂共同提升至一定高度后自由下落产生震击，通过惯性力使型砂获得紧实度[[S6]]。该机型是造型机五大主要类别（震实式、压实式、震压式、射压式、抛砂机）之一，以压缩空气为动力，广泛用于中小批量、多品种的铸造生产场景[[S3,S7]]。

在震实造型机基础上附加压实机构即构成**震压造型机**（jolt-squeeze molding machine），可对震实完成后的砂型补充加压，克服单纯震实造成的上层型砂紧实不足缺陷[[S21,S8]]。

## 定义与术语

依据 GB/T 25370-2020《铸造机械 术语》：

- **震实造型机**（jolt molding machine）：采用震实方法紧实型砂的造型机[[S6]]。
- **震压造型机**（jolt-squeeze molding machine）：采用震实和压实方法（顺序进行或同时进行）紧实型砂的造型机[[S6]]。
- **震实机构**（jolt mechanism）：震实或震压造型机实现震实作用的机构，分为**震击机构**和**微震机构**两种结构型式[[S26]]。微震机构对应标准术语为微震压实造型机（shockless jolt squeeze molding machine），其微振工作频率为 8~15 Hz，振幅为数毫米至数十毫米[[S21]]。
- **震压机构**（jolt-squeeze mechanism）：由震实机构和压实机构组成的一组套迭式柱塞缸[[S26]]。
- **震实台**（bumper）：无起模机构的简易震实造型机，广泛配套 V 法铸造、消失模铸造场景用于干砂紧实作业[[S21]]。

## 分类体系

震实造型机可按三级层次进行划分：

**一级——按起模方式分类**[[S17,S8]]：

- **顶箱（杆）式**：采用顶杆或顶框或辊道起模方式，生产率较高。
- **转台式**：砂箱连同模板翻转 180° 后起模，适用于大中型铸件造型。
- **翻台式**：翻台机构翻转 180° 后接箱台垂直下降起模，适用于复杂带大吊砂块的铸件造型。

**二级——按传动方式分类**：气动驱动型、液压驱动型。

**三级——按是否配置压实功能分类**：

- 纯震实型（震实造型机）
- 震压复合型（震压造型机），进一步细分为脱箱震压型和顶箱震压型。

## 工作原理与过程

### 震实工作原理

震实造型机工作时，由工作台将砂箱、模板及型砂一同上升到一定高度后使之自由下落而产生震击，由于型砂碰撞时的惯性力作用而使型砂得到紧实。在反复震击下，型砂的紧实度随震击次数增加而增加。但超过一定限度后紧实度不再增加，继续震击反而会导致砂型产生分层、局部疏松及开裂等缺陷[[S9]]。

每造一型的常规震击次数为 10~80 次。靠近模板处因受全部砂层作用紧实度最高，上层型砂几乎没有紧实作用，因此震实法造型仅适用于砂箱高度大于 150 mm 的中高砂箱[[S9]]。

### 震实机构工作特性分类

震实机构按工作特性分为三种形式[[S10]]：

1. **空气不断进气和不膨胀**
2. **空气断进气而不膨胀**
3. **空气断进气和膨胀**

其中断进气结构空气消耗量更低、经济性更好，工业应用更广泛[[S10]]。

### 震压工作循环

震压造型机工作过程分为震实和压实两个阶段[[S8]]。

**震实阶段**：压缩空气经管路系统进入震击活塞，将震击活塞顶起；上升过程中进气口被堵死、压缩空气停止进入；活塞继续上升至排气口后靠自重落下，产生震击。多次反复震击使型砂获得初步紧实[[S8]]。

**压实阶段**：震实完成后，压缩空气进入压实气缸，压实活塞将工作台上的砂箱举起，已震实砂型上面与横臂压板接触后压实活塞继续举升，使砂型被进一步压实，从而提升砂型整体紧实度和均匀性[[S8]]。

## 主要结构与组成

震实/震压造型机的核心部件布局与功能如下[[S22,S18]]：

| 部件 | 位置与功能 |
|------|------------|
| 震击台（工作台） | 位于整机最顶部，承载模板、砂箱与型砂，直接传递震击力 |
| 震击活塞 | 嵌套在压实活塞内部、位于工作台正下方，往复运动产生震击动力 |
| 震击气缸 | 固定在机架底座、外围带弹簧减振结构，为震击活塞提供导向与气流通路 |
| 压头 | 安装在上方横梁/横臂位置，向下施压完成补充压实 |
| 压实油缸/气缸 | 安装在机座内部震击机构下方，输出举升动力 |
| 起模机构 | 分布在工作台周边，驱动砂型与模板分离 |
| 模板 | 固定在工作台台面，承载模样成型砂型内腔 |
| 机架/底座 | 承载全部机构，震实气缸通过弹簧支承在机座上以减轻对基础的冲击 |

## 典型技术参数

震实/震压造型机通用参数范围如下：

| 参数项 | 典型范围 | 说明 |
|--------|---------|------|
| 震击频率（普通砧座结构） | 约 480 次/分钟 | 约 8 次/秒 [[S16]] |
| 震击频率（微震结构） | 480~900 次/分钟 | [[S16]] |
| 震击高度 | 数毫米至数十毫米 | [[S16,S21]] |
| 压缩空气工作压力 | 0.5~0.7 MPa | 标准供气范围 [[S9,S4]] |
| 最大压实力 | 10~100 kN 级 | 覆盖主流机型 [[S16]] |
| 工作台尺寸 | 400×300 mm 至 2000×1500 mm | 适配小型至大中型砂箱 [[S16]] |
| 每型震击次数 | 10~80 次 | 超出阈值导致砂型缺陷 [[S9]] |
| 适用砂箱高度 | > 150 mm | 纯震实法适用下限 [[S9]] |

常见量产型号及特色参数覆盖：Z2520 转台震实造型机、Z2310/Z2320/Z2330 翻台震实造型机、TL-18 顶杆起模式震击造型机、Z12 系列脱箱震压造型机、Z14 系列顶箱震压造型机等，数据来源为国内铸造行业通用手册与正规产品目录[[S23,S17,S24,S14]]。

## 应用领域

震实/震压造型机在砂型铸造产线中的定位为中小批量多品种生产场景的主流适配设备[[S17,S22,S8]]：

- **材质适应**：可直接生产铸铁、铸钢材质的大中型铸件。
- **典型应用**：机床床身、柴油机壳体等结构件的铸型制造。
- **自动化兼容**：配置载型辊道的机型无需复杂改造即可直接接入机械化砂型铸造流水线。
- **国内现状**：机构比较简单、价格相对便宜，在国内大量中小铸造厂仍有广泛使用。

## 与新型造型设备的对比

相较于高压造型机、气流冲击造型机等新型砂型造型设备，震实/震压造型机的优劣势如下[[S12,S24,S20]]：

| 对比维度 | 震实/震压造型机 | 高压造型机 / 气冲造型机 |
|----------|-----------------|------------------------|
| 铸型紧实度 | 偏低且分布均匀性差（上低下高） | 更高、更均匀 |
| 噪音水平 | 震击噪声显著较高 | 工作噪音低或较低 |
| 能耗 | 单位铸型能耗更高 | 能耗效率更优 |
| 生产率 | 小时造型产能较低 | 适合大批量高速连续生产 |
| 设备成本与复杂度 | 采购与维护成本低、机械结构简单 | 设备复杂、投资成本高 |
| 砂箱适配灵活度 | 适配不同尺寸砂箱灵活度高 | 受线体节拍与工装约束 |

## 安全操作与维护

### 开机前检查

震实造型机开机前需完成全工位设备核验[[S5]]：

- 覆盖震实机构、起模机构、气吊气动推杆、铸工输送小车、回砂皮带等核心部件运行状态排查。
- 确认模板与工作台安装牢固，模板低于模板框的偏差不超过 0.5 mm。
- 核验所有安全控制开关状态正常，非授权人员不得启动设备。
- 确认设备工作范围内无无关人员停留。
- 带裂纹、开焊的砂箱、模板及吊具严禁投入生产使用。

### 设备安全标准

依据行业标准[[S25]]：

- 主进气管路必须配置总截止阀，电气系统必须设置总急停按钮。
- 设备运行噪声不得超过等效连续 A 声级 **100 dB(A)**。
- 设备首次大修累计工作时长不得低于 **12000 小时**。

### 作业过程禁令

- 禁止违规拆除安全防护设施[[S5,S13]]。
- 禁止操作人员脱离值守岗位。
- 设备运行时严禁将肢体伸入震击/压实工作区域内。
- 翻箱操作必须保持平稳，避免剧烈冲击震动。
- 压实动作完成后方可执行起模流程，防止挤箱故障。

### 润滑维护

需定期为震实造型机震击活塞、压实气缸等运动副的指定润滑孔加注润滑油，所有润滑点需设置醒目标识，确保油路通畅无堵塞[[S8,S25]]。

### 停机要求

停机后必须切断设备总气源、总电源，清理工作区域散落型砂，整理工器具定点存放，完成当班设备运行状态台账记录，为交接班提供依据[[S13]]。
