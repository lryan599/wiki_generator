根据国家标准GB/T 25370-2020《铸造机械 术语》的官方定义，振动破碎机（标准英文名称：vibrating crusher）归类于旧砂处理设备中的砂块破碎机范畴。其核心定义为：利用振动使砂块之间产生撞击和摩擦、对旧砂砂块进行破碎的机器，在破碎砂块的同时还附带实现砂再生作用[[S9,S21]]。该设备是树脂砂、粘土砂再生工艺流程中的标准化定型装备，典型配置在砂再生线前端，对落砂后产生的结块旧砂进行预处理，使废砂团块破碎为合格粒度的再生砂粒[[S9]]。

在压铸及铸造行业的实际应用中，振动破碎机除了处理结块旧砂，还常被用于对压铸工序产生的大尺寸脆性废料（如直浇道、横浇道、废铸件、飞边等）进行预破碎，使其尺寸适配后续的熔炼炉进料口，从而降低废铝合金、锌合金或镁合金的氧化烧损，提升金属回收率[[S7,S22]]。基于这一细分应用场景，该设备在行业内也常被称为 vibratory gate crusher、vibrating scrap breaker 等[[S4]]。但需注意，压铸语境下的振动破碎机与矿山、建筑领域的高功率通用破碎机在配置和目标上存在本质区别：后者以重型高耐磨破碎腔和高速冲击来破碎高硬度矿石，而前者则无重型强制破碎构件，主要依赖物料间的相互撞击与摩擦来处置低硬度的脆性物料，设备结构相对轻量化[[S10,S18]]。

### 设备结构与工作原理

依据《机械工程手册》对砂再生配套设备的结构描述，振动破碎机主要由带耐磨衬板与栅格板的破碎腔、振动机体、惯性激振器、挠性联轴器、主驱动电机、弹性减振元件（减振弹簧）及机架等部件构成[[S3,S2]]。整机支撑在弹性减振机构上，可避免剧烈振动向外部基础传递，同时机体不易被混入旧砂或废料中的硬质金属异物损伤[[S3]]。

其核心破料机制为典型的阻尼强迫振动。设备通过一台带偏心块的激振主轴，或两台交叉对称布置的振动电机提供动力。偏心块高速旋转产生的离心力作为激振源，使振动机体在减振弹簧的恢复力和激振力的共同作用下，做稳定的圆周振动[[S2,S13]]。物料进入破碎腔后，在振动作用下沿腔壁做抛掷与回旋运动，砂块或脆性金属废料之间被不断抛起、相互挤压、碰撞和搓擦，从而实现破碎[[S9,S3]]。该过程不仅使砂团解粒，还能同步对砂粒表面残留的树脂粘结膜进行轻度脱除，附带完成初级砂再生[[S9]]。

### 工艺特性与对比分析

**材料适配性**：振动破碎机通过调节振动频率、振幅等参数，可适应不同压铸合金的物理特性。例如，铝镁各占50%的合金属于脆性物料，适配压碎和冲击破碎机制；而对于其他通用压铸铝合金这类韧性物料，则可通过剪切与快速冲击实现“选择性破碎”，即优先沿物料内部缺陷、晶粒界面等脆弱面碎裂，减少破碎产物内部的微裂纹，提高回炉料的力学强度[[S15,S2]]。

**性能优势**：与传统的球磨机相比，振动破碎机的腔内介质填充率最高可达80%，介质单位时间内的冲击次数为普通球磨机的4至5倍，因而破碎效率可达其10至20倍，而能耗仅为数分之一。通过精确控制工艺参数，其出料粒度可达到2至3微米级别，针对脆性物料可实现亚微米级的超细碎[[S2]]。

**固有局限**：振动破碎机运行时噪声普遍较高，对轴承、弹簧等关键运动部件的机械强度要求也远高于常规非振动式破碎设备，在运维阶段需定期检测和校准这些易损件的工作状态[[S15,S2]]。

**与其他破碎方式的比较**：
在压铸废料及旧砂处理领域，振动破碎机与其他常见破碎机械在原理、适用场景和维护上存在显著差异，具体对比如下：

| 破碎机类型 | 核心破碎机制 | 主要适用场景 | 显著特点或参数 | 维护与易损件情况 |
| :--- | :--- | :--- | :--- | :--- |
| **振动破碎机** | 高频振动下物料间的撞击与摩擦 [[S9]] | 旧砂结块破碎（附带再生）[[S9]]、压铸脆性废料预破碎 [[S7]] | 结构轻量化，无强制破碎件 [[S3]]；破碎效率高，能耗低 [[S2]] | 噪声大，对弹簧、轴承机械强度要求高 [[S15,S2]] |
| 颚式破碎机 | 两块颚板开合运动施加挤压力 [[S9]] | 大尺寸、硬质脆性物料粗碎 [[S10]] | 破碎比通常为3~4 [[S10]]；结构简单，安装维修方便 [[S10]] | 高锰钢颚板使用寿命1.5~8个月(工况而定)[[S14]] |
| 锤式破碎机 | 高速旋转锤头的打击、冲击 [[S9]] | 中硬且磨蚀性弱的物料 (抗压强度≤100MPa, 含水率<15%) [[S8]] | 破碎比可达15~20 [[S8]] | 锤头为核心易损件，单耗约15g/t(以石灰石为例)[[S14]] |
| 剪切式破碎机 | 低转速、高扭矩下的剪切和撕裂 [[S16]] | 轻金属废料(如铝箔、易拉罐)、塑料、电缆 [[S16]] | 出料粒度更均匀 [[S16]]，粉尘和噪音更低 [[S16]] | 对比锤式破碎机，噪音与粉尘显著降低 [[S16]] |

### 工艺关联与系统集成

**工艺角色与定位**：在压铸车间或铸造厂的废料回用及旧砂再生协同处理流程中，振动破碎机承担着两级预处理的任务：其一，专用于旧砂再生工序，破碎结块砂团并完成初级再生；其二，在废金属回炉预处理工序中，对各类大尺寸脆性废料进行破碎减容[[S7]]。

**上下游设备关联**：在典型的废料回收生产线上，振动破碎机进料侧衔接带式输送机，接收经过初步分选的块状压铸废料或结块旧砂；其处理后的物料由出料侧的斗式提升机输送至下游的磁选机，以去除混入的铁质杂质。处理后的洁净炉料或再生砂随后进入后续的分选或回炉备料系统[[S1,S5]]。

**概念区分**：需要明确的是，根据GB/T 25370-2020标准，另一类专门用于破碎压铸浇道、不合格铸件以获得指定尺寸合格金属回炉料的设备，其标准术语为“回炉料破碎机”（breaker for foundry returns / iron crusher），与主要服务于旧砂再生场景的“振动破碎机”属于不同设备分支[[S20,S6]]。
