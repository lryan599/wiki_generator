## 定义

根据国家标准GB/T 5611-2017《铸造术语》的定义，调匀砂（temper sand）为：一定配比下加水混制后经松散处理达到良好综合性能的型（芯）砂[[S10,S24]]。

其核心关联工艺术语为**回性（型砂）/调匀（型砂）**，定义为：混好的粘土砂停放一段时间，使水分均匀分布，更充分地润湿粘土，更好地发挥粘土的粘结作用，对应的英文术语为homogenization of molding sand、temper of molding sand[[S31,S27,S19]]。

调匀砂也称为**回性砂**，指混碾完成后经静置工序使水分充分渗透到砂粒与粘结剂体系中，消除局部富集水团实现水分均匀分布的合格制备型砂[[S16,S2]]。

在英文铸造标准及通用文献中，调匀砂对应的术语包括：temper sand、tempering sand、mulled sand、conditioned molding sand、prepared sand，这些术语均指向经过充分碾磨混匀、水分均匀分布的合格待使用型砂，与国内调匀砂的技术内涵完全匹配[[S8,S12]]。

## 工艺作用

### 在砂处理流程中的位置

调匀砂位于传统铸造砂处理流程中混砂工序之后、松砂工序之后，正式进入造型或制芯工序之前[[S28]]。完整的调匀砂产出链路为：旧砂经破碎、过筛、冷却、磁选、定量，新砂经烘干、过筛、定量，辅料同步定量后共同进入混制工序，后续依次完成调匀、松砂工序，最终产出合格调匀砂供给造型环节[[S20]]。

### 调匀处理的本质

刚从混砂机中卸出的型砂，其水分分布不均匀，较多地集中在许多小砂团中，并且温度高于室温，立即用于造型容易黏模[[S2]]。调匀处理（回性）的本质是将混碾后的型砂停放一定时间，使砂团内的水分向周围渗透扩散，更充分地润湿粘土，更好地发挥粘土的粘结作用[[S31,S27,S19]]。

经过调匀处理的型砂湿强度和可塑性明显提升，不易粘模，流动性、透气性更优[[S2,S16]]。

## 材料构成

### 典型配方

铸铁件黏土湿型调匀砂的典型基准质量配比范围为[[S30,S4]]：

| 组分 | 占比范围（质量分数） |
|------|---------------------|
| 旧砂 | 50%～80% |
| 新砂 | 5%～20% |
| 有效膨润土 | 6%～10% |
| 有效煤粉 | 2%～7% |
| 重油/渣油（防粘砂附加物） | 约1% |

### 基础原料砂

主流工业用调匀砂采用三筛或四筛级粒度分布，85%以上的砂粒集中分布在50目、70目、100目筛段，常用原料砂粒度规格为50/100目[[S21,S14]]。

硅砂（SiO₂含量≥97%，50/100目）是调匀砂最常用的基础原料[[S14]]。在特殊应用场景下也可使用陶瓷砂（主要成分为氧化铝和二氧化硅，50/100目）作为基础原料[[S32]]。

AFS平均细度为原砂经筛析后计算得到的等效平均粒度值，常规工业生产用调匀砂的AFS值范围为40～60。铜合金（黄铜、青铜）铸造场景下使用的调匀砂典型AFS值为95～100[[S35,S5]]。

## 工艺参数

### 调匀停放时间

调匀停放时间因型砂类型而异[[S2,S16]]：

| 型砂类型 | 调匀停放时间 |
|----------|-------------|
| 湿型砂 | 4～5h |
| 干型砂 | 5～7h |
| 有色合金铸造用砂 | 2～4h |
| 普通混碾后砂 | 2～4h |

### 混砂周期

| 型砂类型 | 总混砂时间 |
|----------|-----------|
| 湿型砂 | 6～12min |
| 干型用砂 | 8～16min |
| 特种粘结剂砂 | 10～18min |

加料顺序为：先加入旧砂和粉末状原材料（粘土、煤粉等）干混，之后再加入水分和液态粘结剂进行湿混，禁止膨润土和水参与预混工序，避免生成粘土球、粘附在混料容器壁上[[S31,S11,S18]]。

### 核心控制指标

**含水量**需根据造型工艺差异化控制[[S30,S6]]：

| 造型方式 | 含水量控制范围（质量分数） |
|----------|-------------------------|
| 高密度造型 | 3.2%～4.5% |
| 普通机器造型 | 4.5%～5.5% |
| 手工造型 | 5.0%～6.0% |

**湿压强度**：标准控制范围为30～160kPa，该指标直接受粘结剂添加量、砂粒粒度、紧实度等参数的共同影响[[S25,S1]]。

**紧实率**：常规控制范围为30%～45%，该参数与含水量相关性极强，是判断调匀工艺执行是否合格的核心指标之一[[S26,S6]]。

## 分类方式

调匀砂有两种主流分类维度[[S30,S22]]：

**按粘结剂体系分类：**
- 黏土湿型调匀砂
- 干型/表干型调匀砂
- 水玻璃类调匀砂

**按适配铸件合金类型分类：**
- 铸铁专用调匀砂
- 铸钢专用调匀砂
- 非铁合金专用调匀砂（铜合金、铝合金、镁合金）

## 与预覆膜砂的对比

调匀砂与预覆膜砂在粘结机理、工艺路径和性能特征上存在本质差异[[S33,S7,S13]]：

| 对比维度 | 调匀砂 | 预覆膜砂 |
|----------|--------|---------|
| 主要粘结剂 | 膨润土 | 酚醛树脂等合成树脂 |
| 粘结方式 | 粘土吸水膨胀产生粘结力 | 树脂预先热/冷工艺均匀包覆砂粒表面 |
| 制备工序 | 混砂后停放松散 | 需经干混、湿混、覆膜等专门工序 |
| 使用状态 | 常温湿态 | 干态，需加热固化 |
| 强度 | 较低 | 较高 |
| 尺寸精度 | 一般 | 更优 |
| 适用范围 | 普通砂型铸造造型 | 壳芯、复杂薄壁铸件 |
| 制备成本 | 低 | 高 |
| 循环回用性 | 可循环回用 | 不可直接循环回用 |

预覆膜砂采用冷覆膜法制备时，包含原砂煤油预混、干混、湿混、吹冷空气等工序，与调匀砂加水后静置调匀的工艺路径完全不同[[S7]]。

## 质量检验方法

**仪器检验法：** 大批量生产车间采用型砂性能试验仪检测湿压强度、抗拉强度、透气性、紧实率等指标[[S2,S15]]。

**手捏经验法：** 单件小批量车间可采用手捏法检验。合格调匀砂的特征为：手捏时柔软易变形，松开后砂团不松散、不粘手，手印清晰，折断后断面平整均匀无碎裂[[S2,S15,S3]]。

型砂混好后，应进行调匀和松砂，使水分更趋一致，松散团块[[S15]]。松砂工序通过松砂机将砂团打碎和松开，增加空隙，提高型砂的流动性、可塑性和透气性[[S16,S19]]。

## 与压铸的关系

高压压铸、常规低压/差压金属型压铸工艺全程采用金属模具作为成型载体，不涉及砂型/砂芯成型结构，调匀砂无直接应用场景[[S34]]。

调匀砂仅在砂型重力铸造、砂型低压铸造的黏土湿型砂体系中作为合格待造型、制芯的前驱砂使用[[S34]]。

## 未调匀砂的缺陷与影响

未调匀砂（刚卸出混砂机未做停放松散处理的型砂）因水分分布不均集中在小砂团中、砂温高于室温，会导致以下问题[[S2,S17,S26,S29]]：

- 造型时易粘模
- 型砂性能波动大
- 起模困难、砂型破裂、掉砂
- 最终铸件产生夹砂、砂眼、气孔、表面粗糙等缺陷

## 工艺局限性

传统调匀斗结构设计不合理时，易出现型砂粘附斗壁问题，仅小部分砂从漏斗通道下料，无法实现有效调匀。粘附砂垮塌后反而会使型砂质量劣化[[S28]]。

在配备高性能混砂机和在线型砂质量检测仪的现代铸造线中，型砂混匀质量已经可以直接达标。过长调匀停留时间会导致砂中水分蒸发，性能反而下降，因此不再推荐在混砂机与造型机之间设置中间储砂调匀斗[[S28]]。
