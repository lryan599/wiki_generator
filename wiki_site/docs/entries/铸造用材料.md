## 概述

压铸用材料是指适用于高压压铸（HPDC）工艺生产的金属及非金属原材料。压铸用金属基合金绝大多数为有色合金，涵盖铝合金、锌合金、镁合金、铜合金、锡铅合金五大类。黑色金属（钢、铁等）因模具耐受问题极少在常规高压压铸场景中使用[[S43]]。

在工业压铸领域，各合金产量占比为：铝合金占比最高，达60%~80%；锌合金次之，占10%~20%；铜合金占1%~3%；镁合金占比随汽车及3C产品轻量化需求提升呈现逐年上涨趋势[[S10,S22]]。

压铸用合金工艺性能统一采用5级评分规则：流动性、气密性、与模具钢粘附/熔焊倾向等指标，5为优秀，4为良好，3为中等，2为较差，1为最差；热裂倾向等级定义为：5为不产生缩孔或热裂，4为热裂倾向较轻，3为存在一定程度的热裂倾向，2为较大热裂倾向，1为严重热裂倾向[[S1]]。

## 压铸铝合金

### 标准与牌号

现行中国压铸铝合金国家标准为GB/T 15115-2024《压铸铝合金》，2024年4月25日发布实施，替代旧版GB/T 15115-2009。牌号命名以YZ前缀标识压铸属性，新增YZAlSi12Fe牌号适配新能源汽车轻量化需求[[S5,S26]]。

ADC12是日本JIS H5302标准规定的Al-Si-Cu系压铸铝合金，执行JIS H 5302-2006《铝合金压铸件》标准，是国内压铸行业用量最大的铝合金牌号之一。其化学成分为Si 9.6%~12%、Fe<1.3%、Cu 1.5%~3.5%、Mg<0.3%、Mn<0.5%、Zn<1.0%、Ni<0.5%，Al为余量[[S17,S32]]。

A380是美国ASTM B85标准体系下的典型压铸铝合金，属于Al-Si-Cu系，具有优良的铸造充型性能和力学性能，适用于复杂薄壁压铸件生产，是北美工业领域用量最大的压铸铝合金牌号之一[[S25]]。

国标压铸铝合金ZL101~ZL401的化学成分（GB1173-74）：Si质量分数范围6.0%~13.0%，Cu质量分数最高4.5%，Mg质量分数0.1%~5.5%，Zn最高13.0%，铁元素含量相比普通砂型铸造同牌号标准有所放宽[[S1]]。

### 力学性能

压铸铝合金ZL101~ZL401的典型力学性能（金属型压铸试样标准测试值）：抗拉强度范围15~25公斤/毫米²（147~245MPa），断后伸长率范围0.5%~2.5%，布氏硬度HB范围50~90，其中ZL401为人工时效后测试数值[[S1]]。

高压压铸工艺对合金力学性能有显著强化作用。以Al-15Zn-8Si-2Cu合金的对比试验为例，压铸试样的抗拉强度达330MPa、屈服强度160MPa、延伸率6.1%，各项性能均显著高于重力铸造对应试样[[S39]]。

### 物理与工艺性能

压铸铝合金基础物理特性：密度约2.7g/cm³，比强度σb/γ范围9~15，导热性优良。铝硅系铝合金对模具钢粘附性较高，体收缩大，易在最后凝固区域产生集中缩孔[[S1,S4]]。

Al-Si系压铸铝合金随游离硅含量升高，材料切削加工难度显著提升，同时合金对金属坩埚的腐蚀也随之加剧。使用高铁含量的再生铝合金进行生产，可抑制粘模倾向、提升压铸模具使用寿命[[S7,S9]]。

### 工艺参数推荐

通用压铸铝合金试样的推荐工艺窗口参数为：浇注温度为液相线以上30~70℃，铸型温度200~250℃，压射比压50~70MPa，内浇口导入速度18~35m/s[[S18]]。

亚共晶铝硅合金AlSi10Mg在东芝DC1250CS压铸机上的推荐工艺：浇注温度645℃，快压射速度3.5m/s，增压压力60MPa[[S44]]。

### 应用领域

铝合金压铸件典型应用场景覆盖汽车变速器壳、离合器壳、化油器壳、转向机壳、新型轿车全铝压铸发动机气缸体，是汽车制造领域用量最大的压铸合金，同时也大量应用于电子、通信行业[[S22,S29]]。

![压铸铝合金原材料铝合金锭实物图，展示工业压铸用铝合金原料的外观形态](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d441f549-5096-4068-83a1-7b165f950482/resource)

### 材料因素引发的缺陷

铝合金压铸过程中，熔融金属保护不当、搅拌和浇注阶段湍流过度引发二次氧化是氧化物夹杂缺陷的核心诱因，对应的防控措施包括：最大程度减少熔融金属暴露于空气的时长、避免浇注过程中产生剧烈金属湍流、充分烘干模具避免水汽与高温金属反应氧化[[S28]]。

熔体中残留的卷入气体、氢元素是HPDC铝合金产生气孔缺陷的核心材料侧诱因，对应的防控措施包括采用氩气转子除气工艺降低熔体氢含量、使用K-Mould检测手段严格管控熔体洁净度[[S7]]。

## 压铸锌合金

### 标准与牌号

中国现行压铸锌合金国家标准为GB/T 13818，修改采用ASTM B86标准内容，以YZ前缀标识压铸属性[[S41]]。

ZAMAK 3（也写作Zamak 3，成分为ZnAl4）是ZAMAK锌合金系列的工业标准牌号。其基础成分为96%锌+4%铝，执行ASTM B240（铸锭）/ASTM B86（铸件）、EN1774/EN12844、JIS H2201/JIS H5301、ISO 301系列标准。具备优良压铸性能和尺寸稳定性，北美超过70%的锌压铸件采用该牌号生产[[S50]]。

ZAMAK 5（成分为ZnAl4Cu1）是ZAMAK系列另一典型牌号，对应ASTM标准Alloy 5、欧洲标准AC 41A、JIS标准ZDC1，相比ZAMAK 3具备更高的力学强度[[S37]]。

ZAMAK 4的成分中铝质量分数3.9%~4.3%，铜0.3%~0.5%，镁0.03%~0.06%。其设计目标为降低压铸过程中与模具钢的熔焊倾向，同时保留ZAMAK系列的高延展性优势[[S49]]。

### 工艺性能与参数推荐

锌合金熔点低、与模具钢亲和力小，压铸过程对模具的侵蚀程度低，可显著延长压铸模具的使用寿命，同时其电镀、焊接性能优异，适配表面电镀、喷涂处理工艺[[S9]]。

| 压射比压 | 20~50MPa |
| 填充时间（通用） | 6~40ms |
| 填充时间（需电镀件） | <20ms |
| 填充时间（需喷漆件） | <40ms |
| 保压时间（壁厚<2.5mm） | 1~2s |
| 保压时间（壁厚2.5~6mm） | 3~7s |

| 留模时间（壁厚>5mm） | 20~25s |

### 应用领域

锌合金压铸件典型应用包括建筑五金旋钮把手、装饰性结构件，其优异的表面适配性可支撑高质量电镀、喷涂处理，广泛应用于日用消费品、五金制品领域[[S9,S11]]。

## 压铸镁合金

### 标准与牌号

中国现行压铸镁合金国家标准为GB/T 25748-2010《压铸镁合金》、GB/T 25747-2010《镁合金压铸件》。牌号命名规则参考ASTM体系命名法，由“字母-数字-字母”三部分组成：第一部分为两种主要合金元素代码，第二部分为元素质量分数，第三部分表示合金发展阶段的字母[[S6]]。

AZ91D是应用最广泛的高纯压铸镁合金牌号，对应中国GB/T 25748、美国ASTM B94、日本JIS H5303、欧洲EN1753标准。Al含量约9%、Zn含量约1%，通过严格控制Fe、Cu、Ni杂质含量实现优良耐蚀性[[S46]]。

AM60（AM60B）是典型的低铝含量压铸镁合金，执行ASTM B94、JIS H5303、EN1753标准，相比AZ91D具备更高的延伸率和冲击韧性，多用于汽车安全部件、方向盘骨架等对韧性要求高的压铸件[[S16]]。

### 物理特性

压铸镁合金基础特性：密度范围1.76~1.83g/cm³，比铝轻30%，比锌铜轻2/3；比强度σb/ρ为14~16；凝固温度区间较宽，热裂倾向较高；热容量小，粘模倾向很低；在-190℃低温下仍可保持良好力学性能[[S4]]。

### 工艺参数推荐

镁合金热压室压铸推荐工艺参数[[S8,S20]]：

| 工艺参数 | 推荐范围/值 |
|----------|-------------|
| 压射比压（受力件） | 20~30MPa |
| 压射比压（耐压或大平面薄壁件） | 15~25MPa |
| 热压室高速压射速度 | 可达6m/s |
| 冷压室最高压射速度 | 可达8m/s |
| 冷压室充填速度（常规） | 40~90m/s |
| 冷压室充填速度（薄壁件） | 80~100m/s |
| 热压室浇注温度 | 约640℃ |
| 冷压室浇注温度 | 约680℃ |

### 应用领域

镁合金压铸件典型应用场景包括消费电子类A4笔记本电脑外壳、汽车座椅框架、汽车轮毂、摄像机壳体。依托其低密度、高比刚度的特性，近年在汽车轻量化、3C消费电子领域的应用规模快速扩张[[S9,S13,S23]]。

## 压铸铜合金

### 标准与牌号

中国现行压铸铜合金国家标准为GB/T 15116-1994《压铸铜合金》。牌号以YZ前缀标识压铸属性，代号以YT前缀表示。YT40-1铅黄铜即工业牌号H59压铸黄铜，是用量最大的压铸铜合金牌号之一[[S12]]。

### 特性

压铸铜合金典型特性：常用牌号锌质量分数在30%~40%区间时为最适合高压压铸的成分区间，此时熔点约900℃，结晶温度范围窄，凝固附近塑性好；密度8.2~8.5g/cm³；力学强度高，导电导热性优异[[S4]]。

铜合金熔点远高于锌、铝、镁三类常用压铸有色合金，压铸过程中模具承受的热负荷高，铜合金压铸工况下的模具寿命显著低于另外三类合金的生产工况[[S9]]。

## 压铸锡铅合金

压铸用锡铅合金是最早应用于压铸生产的合金系列。特点是比重大、熔点低：铅锑合金熔点230~240℃，锡锑合金熔点289~370℃；铅锑合金密度9.29~9.60 g/cm³，锡锑合金密度7.30~7.38 g/cm³。强度低韧性好，耐磨性耐蚀性优良，充型流动性极佳，不粘模，可铸造极薄壁复杂小铸件。铸造和加工性能好，但力学性能不高，适用于力学性能要求不高的各种复杂小零件[[S3,S48]]。

锡铅合金的年产量极低[[S48]]。多用于印刷铅字、汽车平衡重块、高精度仪表小零件等场景[[S48]]。

## 材料因素引发的典型缺陷

### 气孔

熔体中残留的卷入气体、氢元素是HPDC铝合金产生气孔缺陷的核心材料侧诱因。防控措施包括采用氩气转子除气工艺降低熔体氢含量、使用K-Mould检测手段严格管控熔体洁净度[[S7]]。

### 夹杂

熔融金属保护不当、搅拌和浇注阶段湍流过度引发二次氧化是氧化物夹杂缺陷的核心诱因。防控措施包括最大程度减少熔融金属暴露于空气的时长、避免浇注过程中产生剧烈金属湍流、充分烘干模具[[S28]]。

### 硬质点

压铸合金成分不纯、混入不明来源杂料会直接引发硬质点缺陷，硬质点会大幅提升后续机械加工过程中的刀具损耗。防控措施包括严格管控原材料纯度、避免不同牌号合金混料、执行规范的熔体熔炼与精炼工艺[[S15]]。

### 冷隔

合金冶金质量差、流动性不足会引发冷隔缺陷：充型过程中温度过低提前凝固的多个金属流股相互碰撞，无法完全熔合形成连续的铸件本体。防控措施包括严格把控合金成分处于标准范围、保证熔体温度满足流动性要求[[S35]]。

### 热裂纹

冶金质量管控不当会引发热裂纹（热撕裂）缺陷：铸件凝固末期残余液相占比不足，收缩应力超过材料高温塑性阈值，沿晶界形成开裂。防控措施包括调整合金成分优化凝固温度区间、避免合金过热温度过高[[S40]]。
