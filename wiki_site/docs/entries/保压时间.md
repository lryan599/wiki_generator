## 概述

保压时间（英文：dwell time，亦称持压时间）是高压铸造、低压铸造、挤压铸造、热冲压及注塑等制造工艺中的通用核心工艺参数。其物理本质为材料充型完成后，外界对型腔或工件持续施加预设恒定压力的时长，不同工艺场景下对应不同的核心作用目标[[S3,S18]]。

依据 GB/T 5611-2017《铸造术语》，在压铸和低压铸造场景下，保压时间明确定义为熔融金属充型完成后保持工艺压力的时长[[S24,S16]]。

## 分类与工艺归属

### 高压铸造（压铸）

高压铸造中保压时间指熔融金属充满型腔后，增压压力建立完成到内浇口完全凝固期间，压射冲头持续向金属传递压力的总时长。核心作用是使正在凝固的金属在压力下完成结晶，消除缩孔缩松缺陷，获得高致密度铸件[[S14,S10,S25]]。若时长不足，厚壁部位无法完成补缩而产生内部缺陷；时长过长则无额外增益，还可能使铸件内部残余应力过高[[S4]]。

### 低压铸造

低压铸造中保压时间指型腔压力提升至结晶压力后，在该压力下维持的时长，直到铸件完全凝固为止。核心作用是通过持续压力传递保证金属液向凝固区域补缩[[S29,S28]]。若时长不足，铸件未完全凝固即卸压，将导致"放空"报废及缩松缺陷；时长过长则浇口金属完全冻结，铸件脱模困难，同时降低工艺收得率[[S22]]。

### 挤压铸造

挤压铸造中保压时间指充型完成后，压头持续向型腔施加高压的时长。核心作用是完全消除铸件内部孔隙，实现组织高度致密，获得接近锻件力学性能的铸件。该工艺的保压时长普遍高于普通低压铸造[[S17]]。

### 热冲压

热冲压工艺中保压时间指热成形后工件在闭合模具的恒定合模压力下持续贴紧模腔冷却的时长。核心作用是完成奥氏体向马氏体的相变淬火，保证22MnB5等硼钢工件获得所需高强力学性能，典型优化值约为60s[[S3]]。

### 注塑

注塑工艺中保压时间对应英文术语 packing time，指熔体完全充满型腔后，螺杆在预设保压压力下持续前移补料直到浇口完全固化封口的时长。核心作用是补偿塑料熔体冷却过程中的体积收缩，控制制品最终尺寸精度，消除凹陷、缩孔缺陷，稳定成型质量[[S3,S11,S1,S5]]。

## 工艺作用

各工艺场景下保压阶段均为充型完成后到泄压/开模前的核心工艺环节，其共性目标在于利用持续压力抵消材料冷却或凝固过程中的体积收缩，避免内部缺陷，提升致密度与成型精度。保压时间须保证在进料浇口完全固化前完成压力传递，否则压力无法作用到型腔内部而失效[[S3,S12]]。

在压铸中，增压保压过程是补缩过程，补充因冷却而出现的空间。在一定的保压时间内，金属液在最终压力下补缩和固化同时进行，将可能产生的压铸缺陷降至最低[[S25]]。在注塑中，保压压力和保压时间的组合直接影响制品收缩率和最终尺寸精度；若保压不足，浇口未冻结就撤除压力，熔体会发生倒流，型腔压力迅速下降，导致制品密度不足、表面凹陷及内部真空泡等缺陷[[S1,S5]]。

## 关键影响因素

保压时间的选择需综合考虑以下因素：

**合金/材料特性**：合金结晶温度区间越大、熔点越高，所需保压时间越长。共晶合金的保压时间短于宽结晶温度区间的固溶型合金[[S14,S6]]。

**铸件/工件壁厚**：保压时间随壁厚增大而延长。浇口开设在薄壁处对应所需保压时间更短，浇口开设在厚壁热节部位所需保压时间更长[[S27]]。

**模具/铸型条件**：模具或铸型温度越高、冷却能力越弱（如干砂型），对应铸件整体凝固速度越慢，所需保压时间越长；金属型或水冷铸型冷却速度快，相同壁厚下所需保压时间可适当缩短[[S27]]。

**浇注/进料温度**：浇注或进料温度越高，材料完全凝固所需时长越长，保压时间需适度延长[[S20]]。

上述因素的综合影响可参考下表：

| 影响因素 | 影响趋势 |
|----------|----------|
| 合金结晶温度区间 | 越大，保压时间越长[[S14,S6]] |
| 铸件壁厚 | 越大，保压时间越长[[S27]] |
| 浇口位置 | 薄壁处短，厚壁处长[[S27]] |
| 模具温度 | 越高，保压时间越长[[S27]] |
| 铸型冷却能力 | 越弱，保压时间越长[[S27]] |
| 浇注温度 | 越高，保压时间越长[[S20]] |

## 典型数值与推荐范围

### 低压铸造保压时间推算

根据铸件最大壁厚划分的不同金属材料单位厚度保压时间参考值如下[[S23]]：

| 材料 | 壁厚条件 | 单位厚度保压时间 (s/mm) |
|------|----------|--------------------------|
| 铝合金 | ≤50mm | 0.5 |
| 铜合金 | ≤50mm | 1.5 |
| 钢和铸铁 | ≤50mm | 0.5 |

### 锌合金压铸保压时间推荐值

生产中锌合金压铸件基于壁厚的保压时间推荐值如下[[S8]]：

| 壁厚 | 保压时间 |
|------|----------|
| < 2.5mm | 1～2s |
| 2.5～6mm | 3～7s |

保压时间整体应大于0.5s[[S8]]。

### 各工艺典型取值实例

不同工艺场景的典型保压时间取值差异显著[[S3]]：

| 工艺 | 零件/材料 | 典型保压时间 |
|------|-----------|--------------|
| 低压铸造 | 铝合金壳体铸件 | 1min |
| 低压铸造 | 水龙头铸件 | 5～12s |
| 低压铸造 | 280滤清器座 | 6～10min |
| 挤压铸造 | 试验件 | 约75s |
| 半固态压铸 | A356铝合金 | 0.05s |
| 热冲压 | 22MnB5硼钢板 | 60s |
| 注塑 | 常规塑件 | 8s |

低压铸造工业实例：某结构类铸件的低压铸造完整工艺参数记录中，浇注温度720℃、模具温度300～450℃、保压时间250s[[S15]]。

## 缺陷关联

**保压时间过短**：铸件未完全凝固时压力提前释放，未凝固金属液回流进坩埚或料筒，直接产生欠铸、缩松、内部气孔等缺陷，无法获得完整合格铸件[[S9]]。在低压铸造中，过早卸压还会造成铸件"放空"报废[[S29,S28]]。

**保压时间过长**：浇口及升液管内金属液完全冻结，造成铸件脱模困难，大幅降低金属收得率，延长生产周期、降低生产效率[[S9]]。在低压铸造中，浇口残留长度增加，严重时会产生"冻结"现象，使铸件、浇口、升液管凝结在一起[[S22]]。

不同保压时间对HPDC铝合金铸件孔隙率有显著影响：保压时间过短时铸件存在大量大尺寸缩孔、气孔；随着保压时间优化至约10min，铸件孔隙密度降低，细小孔隙占比升高，力学性能波动幅度降至最低[[S13]]。

## 计算模型与规律

保压时间与铸件当量厚度（模数）的平方呈线性关系。针对气动元件阀体系列铝合金铸件的低压铸造工艺研究，通过FT-1、FT-2、FT-3三类阀体铸件的数值模拟与生产数据回归拟合，获得了以下拟合方程[[S26]]：

$$t = 0.614 \times R^2 + 11.406$$

其中铸件当量厚度 $R = V/S$（$V$为铸件体积，$S$为铸件散热面积），保压时间$t$单位为秒。

该公式基于平方根定律（丘里诺夫公式）推导而来。根据平方根定律，铸件的凝固层厚度与凝固时间的平方根成正比，铸件凝固时间与当量厚度的平方成正比。上述拟合方程仅适用于低压铸造铝合金阀体类铸件的保压时间估算[[S26]]。

需注意，铸件凝固时间受众多因素影响，难以精确计算，目前生产上通常凭经验确定保压时间[[S26,S9]]。

## 工艺对比

不同工艺在保压时间的量级、控制逻辑上存在显著差异[[S3,S16]]：

| 工艺 | 典型保压时间量级 | 控制逻辑 | 终止判定依据 |
|------|------------------|----------|--------------|
| 高压铸造 | 0.1～几十s | 压射充型结束后维持冲头压力 | 内浇口完全冻结 |
| 低压铸造 | 几十s～十几min | 维持炉内气压 | 铸件整体凝固完成 |
| 挤压铸造 | 几十s～上百s | 充型完成后维持压头机械压力 | 铸件厚大部位完全结晶 |
| 热冲压 | 几十s级 | 闭合模具后维持合模力淬火冷却 | 钢板冷却至马氏体相变完成 |
| 注塑 | 几十s级 | 充型完成后维持螺杆背压持续补缩 | 浇口完全固化 |
