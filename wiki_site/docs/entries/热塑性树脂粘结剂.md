## 概述
热塑性树脂粘结剂（thermoplastic resin binder）是有机树脂粘结剂的重要子类，在压铸及铸造工艺中用作覆膜砂、涂料、型芯胶粘剂等场景的粘结组元。其核心特征为可在特定温度区间内反复加热软化/熔化、冷却回复固态，分子间不发生交联反应，化学结构无不可逆改变[[S5,S25,S9]]。

## 定义与基本属性
### 标准定义
根据GB/T 5611-2017《铸造术语》，热塑性树脂粘结剂定义为：在特定温度范围内反复加热、冷却而分子间不发生交联反应的线型树脂粘结剂，加热时软化或熔化，冷却时回复到固体状态[[S5]]。该定义在《中国机械工业标准汇编 铸造卷》和《机械基础制造工艺标准汇编 铸造》中保持一致[[S9,S12]]。

> **注意**：该定义与通用化工领域中的热塑性胶粘剂存在明确差异——压铸与铸造语境下的热塑性树脂粘结剂系指具备上述温致可逆软化特性的线型树脂材料，而非室温固化的普通热塑性胶粘剂[[S9]]。

### 基本特性
- **可逆相变**：加工过程仅发生物理相变，无不可逆交联，理论上可实现100%回收复用[[S20,S27,S38]]。
- **热挥发特性**：有机粘结剂在400～500℃焙烧条件下可完全挥发分解，残碳率低于1%[[S26]]。

## 分类与常见品类
压铸领域已确认的主流热塑性树脂粘结剂品类如下[[S11,S39,S31]]：

| 品类 | 特征说明 |
|------|----------|
| 聚乙烯（PE） | 常见商用聚合物，热塑性粘结剂基础组元 |
| 聚丙烯（PP） | 商用热塑性聚合物，用于粘结剂复配 |
| 聚苯乙烯（PS） | 用于注射成形型芯粘结体系 |
| 聚乙二醇（PEG） | 低残碳，可水溶，用于可溶型芯 |
| 聚乙烯醇（PVA） | 常用有机粘结剂，受热挥发迅速 |
| 丙烯酸类树脂 | 低残碳热塑性粘结剂系列 |
| 热塑性酚醛树脂 | 苯酚与甲醛摩尔比>1条件下酸性缩聚，可溶于乙醇、不溶于水，100～200℃软化[[S25,S31]] |
| 石蜡 | 分子含18～32个碳原子，低粘度、高挥发性[[S39]] |
| 松香 | 熔点75～135℃，醇基涂料常用粘结剂[[S31]] |

### 热塑性酚醛树脂
线型热塑性酚醛树脂由苯酚和甲醛在酸性介质中、苯酚与甲醛摩尔比大于1的条件下缩聚而成，为块状固体或粉状，可溶于乙醇，不溶于水。常用作醇基涂料粘结剂，配合乌洛托品使用可在涂层加热时固化获得较高强度[[S25,S31]]。

## 热物理性能

### 熔化温度区间
不同品类热塑性粘结剂熔化温度差异显著[[S39,S31]]：

| 粘结剂品类 | 熔化温度区间 |
|-----------|------------|
| 石蜡基粘结剂 | 60～80℃ |
| 蜂蜡基粘结剂 | 60～80℃ |
| 巴西棕榈蜡基粘结剂 | 87～176℃ |
| 硬脂酸（添加剂） | 65～75℃ |
| 通用热塑性酚醛树脂 | 100～200℃ |
| 松香 | 75～135℃ |

### 热分解特性
- 聚乙二醇（PEG）热塑性粘结剂在氮气惰性气氛下热分解起始温度约为300℃，分解机理为分子链断链与氧化降解，分解完成后残余碳量极低，属低残碳品类[[S40]]。
- 石蜡基粘结剂因低分子量特性，挥发性高，利于脱脂工序，热分解时引起的坯件变形量小于其他热塑性粘结剂体系[[S39]]。

### 流变特性
- 压铸与金属注射成形工艺中，热塑性粘结剂配制喂料的典型粘度需控制在20～200 Pa·s区间，以保证良好流动性和无注射缺陷[[S32]]。
- 低分子量热塑性粘结剂可有效降低喂料粘度，消除注射残余应力，保证粉末堆积和收缩过程的各向同性[[S32]]。

## 工艺作用与核心场景
热塑性树脂粘结剂在压铸全流程中可用于三个核心场景[[S1]]：

1. **熔模精密压铸快速原型**：替代传统蜡模，提升原型蠕变强度和耐机械损伤性，降低搬运损坏风险。
2. **可溶型芯基体组元**：作为可溶型芯的核心组分之一，实现复杂内腔铸件的型芯水溶脱出。
3. **低压铸造型壳粘结组元**：降低焙烧后残留杂质。

### 焙烧工艺与预制型
在压铸用预制型制备中，不同粘结剂体系需经焙烧工序使粘结剂分解排出。含热塑性有机粘结剂的预制型在400～500℃条件下焙烧，可基本完全挥发[[S26]]。

## 优缺点与局限性

### 优势

| 对比维度 | 热塑性树脂粘结剂 | 热固性树脂粘结剂 | 无机粘结剂 |
|---------|---------------|---------------|----------|
| 可回收性 | 100%可回收，仅物理相变[[S20,S27]] | 固化后不可逆，不可回收[[S27]] | 不可回收 |
| 焙烧残留 | 400～500℃完全挥发，残碳<1%[[S26]] | 较高残留 | 残留率>90%，形成难清理杂质[[S7,S26]] |
| 溃散性 | 接触熔融金属后快速熔化流失，清砂难度低[[S13,S21]] | 溃散性一般 | 清砂困难 |
| 低分子挥发性 | 有利于脱脂工序[[S39]] | 不明显 | 不适用 |

### 局限性
- **常温强度偏低**：热塑性粘结剂制备的砂芯常温强度偏低，搬运和装配中需谨慎操作[[S13]]。
- **明确熔点增加工艺控制难度**：具有明确熔点的粘结剂在充型过程中若温度梯度分布不均，不同区域先后发生熔化-凝固相变，产生不均匀热应力，可能引发型芯开裂[[S11,S18,S34]]。

## 缺陷机理与预防

### 气孔缺陷
热塑性树脂粘结剂在高温金属液冲刷下受热分解产生大量含氢、烃类的挥发性气体（粘结剂分子含C、H化合物），若型芯排气路径不通畅，气体被高压金属液包裹无法溢出，最终在铸件内部形成气孔缺陷，严重时表面鼓泡变形报废[[S8,S29,S2]]。

**预防措施**：控制粘结剂添加量在合理区间；设置型芯专用排气通道；浇注前对型芯充分预热排气；采用分级升温脱除型芯内部残留低分子挥发分[[S15]]。

### 粘砂与焊缝缺陷
不当使用可引发两类粘砂相关问题[[S6,S2]]：
- **亮碳缺陷**：粘结剂残留碳质富集在铸件表面形成类似大象皮的亮碳褶皱形貌。
- **机械粘砂**：粘结剂失效后砂粒被高压金属液渗入型腔表面形成机械粘砂，严重时出现合金与模具粘结的焊合缺陷。

**预防措施**：控制粘结剂高温裂解后的溃散性能；型芯表面均匀涂刷匹配压铸温度的耐火涂料；合理控制压铸充型速度避免金属液对型芯的高速冲刷[[S16]]。

### 残留杂质
压铸完成后若热塑性树脂粘结剂未完全热解溃散，残留在铸件内腔缝隙中形成难以清除的残留杂质，降低铸件后续密封性能与装配精度[[S14]]。

**预防措施**：根据压铸合金温度匹配粘结剂完全分解温度；铸件出模后采用高压水溶或热风吹扫工艺彻底脱除残留粘结剂组分；选择浇注后溃散性符合合金工艺要求的粘结剂牌号[[S31,S15]]。

### 型芯开裂
热塑性树脂粘结剂存在明确熔点，充型过程中温度梯度分布不均时不同区域先后发生熔化-凝固相变，产生不均匀热应力，当热应力超过型芯抗拉强度即引发型芯开裂，导致铸件内腔出现缺肉、断芯类缺陷[[S11,S18,S34]]。

**预防措施**：选择低熔点区间分布均匀的多组分复配热塑性粘结剂，避免单一树脂相在窄温度区间内发生突变相变；通过预热模具降低充型瞬间的温差幅度[[S34]]。

## 典型应用案例

### 可溶型芯
本田工程采用含热塑性树脂粘结剂的氯化钾可溶型芯量产铝合金进气歧管复杂弯曲通道构件，在高压压铸工艺下成型后将型芯溶解脱除，无需多组拼合金属抽芯结构，大幅简化复杂内腔构件压铸工艺[[S35]]。

另有公开应用采用添加热塑性粘结剂的碱金属盐芯用于高压压铸封闭水冷通道型铝合金缸体，型芯可耐受700℃级铝合金充型压力，脱芯介质为60～80℃热水，脱芯时间小于30min，铸件内部通道清洁度满足汽车发动机使用要求[[S30]]。
