## 概述

**铁液浇包**（中文别名“铁水包”、“浇注包”）是铸造车间用于容纳、转运、保温及浇注铁液（铁合金熔体）的核心设备，其主体为内衬耐火材料的钢制容器[[S6,S12]]。该术语与相应功能定义被正式收录于国家标准 GB/T 25370-2020《铸造机械 术语》，对应英文术语包括 *molten iron ladle*、*hot metal ladle*、*iron pouring ladle*、*casting ladle for ferrous metals* 等[[S6,S15,S4]]。

需特别说明的是，在常规低熔点有色合金（如铝合金、锌合金）压铸生产中，几乎不涉及铁液浇包的使用；其典型应用集中在**砂型铸造**、**铁基合金特殊压铸/压力铸造成形**工艺中，早期铁合金压铸实验即采用人工铁液浇包向压射室加料的操作模式[[S7,S19]]。因此，本词条立足于广义铸造与特殊压铸工艺阐述该设备。

## 分类与术语

### 标准分类体系

依照 GB/T 25370-2020，铁液浇包按功能和结构可划分为多种品类[[S6]]：

| 分类 | 名称 | 核心功能或结构特征 |
|------|------|---------------------|
| 通用类 | 吊包（suspension ladle） | 采用机械吊运的浇包 |
| 通用类 | 倾转式浇包（tilting ladle） | 通过绕水平轴转动包体倒出金属液 |
| 功能类 | 底注包（bottom pouring ladle） | 浇注口设在底部，通过塞杆及其启闭机构控制浇注 |
| 功能类 | 茶壶包（teapot spout ladle） | 浇注时金属液从包底经倾斜流槽流出，具备良好的自动挡渣能力 |
| 功能类 | 扇形包（sector pouring ladle） | 扇形截面设计，浇注量与转动角度成正比，可实现定量浇注 |
| 功能类 | 鼓形包（drum ladle） | 绕水平轴转动的圆柱形浇包，辐射面积小、散热较慢 |
| 处理类 | 球化包（spheroidization ladle） | 专用于加入球化剂对铁液进行球化处理 |
| 处理类 | 压力浇包（pressurized ladle） | 在包内液面上建立并控制气压，实现按一定流量浇出铁液 |
| 处理类 | 压力球化包（pressurized spheroidization ladle） | 能承受球化剂蒸气压力的密封盖式球化包 |
| 处理类 | 脱硫包（desulfurization ladle） | 通过底部吹气搅拌进行铁液脱硫处理 |
| 辅助类 | 转运包（transfer ladle） | 在熔炼炉/保温炉与浇注工位之间转运金属液 |
| 辅助类 | 升降浇包（lifting ladle） | 造型线上能够上下升降的小型浇包 |

行业标准 JB/T 5771-91 进一步将浇包产品型式归纳为**普通倾注式**和**茶壶式**两类[[S14]]。

### 按操作方式分类

浇包按人工/自动浇注模式可划分为[[S18,S6]]：
- **手动浇包**：包含手端包（容量 ≤20 kg，单人操作）与抬包（容量 50~100 kg，由 2~4 人协同操作），适用于中、小件浇注。
- **自动浇包**：包含铁液自动转运系统配套机型、可按预设程序运行的自动倾转浇注机型，适配流水线连续生产场景。GB/T 25370-2020 定义的“铁液自动转运系统”即包含炉前接铁液车、铁液转运车、转运包、转运轨道、铁液处理站、扒渣平台及安全报警设施等[[S6]]。

## 核心结构与材料

### 壳体构造

浇包的外层承载壳体（包体，ladle body/pot）由厚度 **2~16 mm** 的钢板焊接制造，需具备足够的结构强度以承受铁液满装载荷和吊运操作工况[[S18,S6,S1]]。

### 耐火内衬

浇包内壁砌筑或捣实的耐火材料层称为包衬（refractory lining/ladle lining）[[S6,S12]]。**高铝质、莫来石材质**是铁液浇包的主流内衬选型。选型依据要求耐火材料同时具备以下性能[[S3,S9]]：
- 高耐火度，能够承受高温铁液长时间接触；
- 良好的抗热震性能（resistance to thermal shock/spalling），防止因急热急冷导致衬体剥落；
- 不与高温铁液发生有害化学反应；
- 低导热系数，有效降低铁液转运过程的温降幅度。

部分特殊应用场合也可采用石墨质内衬替代常规耐火材料[[S3]]。

### 保温与覆盖措施

为减少铁液在转运和等待过程中的辐射散热损失，可在壳体与耐火内衬之间增设隔热层[[S3]]；铁液进入浇包并完成扒渣后，采用**稻草灰或专用集渣保温剂**覆盖铁液表面，实现被动保温效果[[S8]]。

## 工艺角色与定位

### 在生产线中的定位

铁液浇包处于铸造生产流程中**熔炼炉/保温炉与浇注工位之间**。其承担的核心功能包括[[S6,S10]]：
1. **铁液转运**：将熔炼合格的铁液从熔炼工位运送至浇注工位；
2. **炉前处理**：在浇包内完成球化处理、脱硫处理、孕育处理等对铁液化学成分和石墨形态进行调整的关键工序；
3. **浇注执行**：通过倾转机构或底注机构将铁液按工艺要求注入铸型（含砂型或铁基合金压铸设备的压射室）。

### 定量浇注功能

部分特殊设计的浇包类型可精准控制出液量，实现定量浇注，无需额外计量装置即可匹配不同铸件所需铁液量。例如[[S6,S10]]：
- **扇形包**：浇注量与包体转动角度成正比；
- **压力浇包**：通过控制包内气压调节浇注流量。

## 关键操作参数

### 铁液温度范围

铁液浇包中适用的铁液温度因合金种类和工艺阶段不同而异[[S10,S11]]：

| 合金/工艺类型 | 适用温度范围 | 备注 |
|---------------|-------------|------|
| 普通灰铸铁 | 1220~1400 ℃ | 薄壁/重要灰铸铁件可取 1350~1400 ℃ |
| 球墨铸铁（球化处理后出包温度） | ~1400 ℃ | 经球化处理和孕育后进入浇注工序的温度 |

小容量浇包的铁液温降速率远高于大容量浇包（原因是热容量较小），同一生产调度中，小容量浇包内的铁液转运与等待时间需严格控制，应优先完成浇注作业[[S11,S13]]。

### 烘干预热要求

浇包在正式接触铁液前，**必须完成充分烘干预热**。未干燥的冷包接触高温铁液会引发包衬内残余水分急速汽化，导致铁液剧烈沸腾、飞溅乃至爆炸等严重安全事故[[S8,S1,S6]]。烘包可通过专门的烘包器或在线烘包方式完成[[S6]]。

### 铁液面高度与防溢

吊包中铁液面应**低于包口 30 mm 以上**，以防止铁液在吊运过程中溢出；吊运过程中必须卡好安全卡扣[[S8]]。

## 操作规范与安全要求

### 浇注操作规范

铁液浇包的规范化操作流程包括如下核心要点[[S8]]：
1. **扒渣与覆盖**：出铁完成后需彻底扒除铁液表面浮渣，之后立即使用保温覆盖剂覆盖。
2. **挡渣浇注**：浇注时应采取挡渣措施，浇包嘴尽量贴近浇口杯，避免铁液飞溅或产生旋涡卷入空气。
3. **浇口杯充满维持**：浇口杯应保持充满状态，不得引起铁液飞溅或产生涡流。
4. **引气操作**：浇注开始后应立即进行引气操作，大型铸件应点燃出气孔旁的引火材料。
5. **异常处理**：浇注时如发生铁液猛烈沸腾并从冒口连续喷出铁液，应立即停止浇注。
6. **回炉处理**：浇注剩余或经判定不合格的铁液应倒入专用铁模中，按等级分类存放。

### 安全操作规程

浇包作为承载高温铁液的移动式设备，其设计与操作须严格遵守安全规范[[S1,S14,S8]]：

| 安全项目 | 具体要求 |
|----------|----------|
| 重心安全 | 装满铁液后浇包重心须比旋转轴心至少低 **200 mm**，防止意外倾覆 |
| 载荷校验 | 浇包载荷应按浇注时最大负荷的 **125%** 校验 |
| 吊运安全 | 吊钩、链条、钢丝绳等承力部件不得有裂纹或损伤，定期进行探伤检查 |
| 铁液装载 | 浇包内液态金属不得超过内壁高的 7/8 |
| 工具干燥 | 扒渣、堵铁等接触铁液的金属工具必须保持干燥 |

行业标准 JB/T 5550-91 规定[[S14]]：
- 浇包手轮空程角度须**小于 20°**；
- 包轴、吊攀、吊杆等受力零件设计安全系数取值为 **10**；
- 浇包须能承受 **2 倍额定容量**的负载下、按工作状态操作 **2 小时**，且所有表面及焊缝不得出现任何裂纹。

## 缺陷机理与质量问题

因浇包操作或工艺控制不当而引发的铸件缺陷主要涉及以下类型：

1. **冷隔与浇不足**：铁液在浇包内停留时间过长，温降超标，导致进入铸型时温度不足，铸件出现冷隔或浇不足缺陷[[S16]]。
2. **气孔**：浇包浇注操作不稳，铁液流束落差大、紊流严重，大量卷入空气，最终在铸件内部形成气孔缺陷[[S1]]。
3. **夹渣**：耐火内衬材料因热震剥落或侵蚀脱落的颗粒随铁液进入铸型，形成非金属夹杂缺陷；浇注前未有效扒渣或茶壶包挡渣结构失效亦会造成浮渣卷入[[S3]]。
4. **球化/孕育衰退**：铁液在浇包内停留过久，球化或孕育处理效果随时间衰减，导致铸件石墨形态不良、力学性能不合格[[S10]]。

## 技术局限性

铁液浇包在当前技术条件下存在若干局限性[[S7]]：

- **温降速率快**：无主动保温装置（如感应加热或燃气伴热）的常规浇包，铁液温降速率较高，不适用于长距离或长时间转运场景。
- **定量精度有限**：传统人工操作浇包的出液量控制依赖操作工经验，定量精度较难匹配铁合金高精度压铸工艺对加料量的严格要求。早期铁合金压铸实验已表明，常规铁液浇包向压射室加料方式存在人工消耗大、稳定性差的问题。
- **环保与职业健康**：铁液镁处理球化包作业时会产生大量烟气和强烈光辐射，需配套高能力排风系统，浇注区域须保证良好通风[[S8]]。

## 相关术语关联

- **浇钢包（molten steel ladle）**：GB/T 25370-2020 中与铁液浇包并列的另一类浇包，专门用于钢液的转运和浇注，其耐火材料选型及保温要求与铁液浇包有所区别[[S15]]。
- **保温炉（holding furnace）**：熔炼炉生产的熔融金属的储存和温度保持设备，与浇包在功能上形成互补，同属熔体转运与温度管理系统的组成部分[[S12]]。
- **浇注机（pouring machine）**：按照工艺要求将金属液定量注入铸型的机器或装置。浇包可作为浇注机的供液源或集成部件，二者在生产线上为上下游关系[[S6]]。
