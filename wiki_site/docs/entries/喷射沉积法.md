## 概述

喷射沉积法，又称喷射成形或喷射铸造，是一种将液态金属雾化与沉积成形合为一体的快速凝固近终形材料制备技术[[S16]]。该技术于1968年由英国斯旺西大学的A.R.E.辛格首次提出[[S11,S16]]，其广义范畴覆盖了仅通过雾化金属直接沉积制备纯金属或合金坯料的全系列工艺，包括喷射轧制、喷射锻造、离心喷射沉积、喷射涂层等多种衍生形式[[S11,S35]]。

在颗粒增强铝基复合材料的制备领域，喷射沉积法存在狭义的学术共识定义：将基体金属在坩埚中熔炼后，通过压力送入雾化器，在高速惰性气流作用下分散为细小液滴形成雾化锥，同时通过一个或多个喷嘴向雾化锥内喷射入预先准备的增强颗粒，使增强颗粒与金属雾化液滴共同在基板上沉积并快速凝固，直接得到颗粒增强金属基复合材料的工艺。该工艺的常规冷却速度可达10³~10⁶ K/s，沉积坯初始致密度可达理论密度的95%~98%[[S36,S16,S27]]。

## 定义与术语辨析

### 狭义定义与工艺归属

用于制备颗粒增强铝合金的喷射沉积法，其狭义定义为：将基体金属熔炼后送入雾化器，在高速惰性气流作用下雾化为细小液滴形成雾化锥，同时通过喷嘴向雾化锥内喷入增强颗粒，使二者共同在基板上沉积并快速凝固，直接获得颗粒增强金属基复合材料[[S36]]。该工艺属性上属于快速凝固近终形材料制备技术[[S16]]。

### 与通用“喷射沉积/喷射成形”的关系

广义的喷射沉积（喷射成形）为上位概念，不特指制备复合材料的工艺，覆盖仅通过雾化金属直接沉积、无额外添加增强相的纯金属/合金坯料的全系列工艺，包含喷射轧制、喷射锻造、离心喷射沉积、喷射涂层等多种衍生形式[[S11,S35]]。颗粒增强铝合金喷射沉积法是其下位特化工艺，在通用喷射沉积设备上增设独立的增强颗粒输送与喷入单元。

### 与Osprey工艺的区别

Osprey工艺是1974年英国Osprey Metals公司取得专利的商业化通用喷射沉积工艺，属于广义喷射成形的典型工业化实现，其原生设计中未配置额外的增强颗粒喷入装置，最初用于直接制备普通合金锻造坯料，不属于专门的颗粒增强铝合金制备工艺[[S10,S1,S40]]。后续基于Osprey工艺衍生的Alcan Cospray技术属于向其添加颗粒喷射模块后的变体，是喷射共沉积类工艺的工业化代表[[S1,S40]]。

### 与等离子喷涂/热喷涂的区别

等离子喷涂、常规热喷涂类工艺属于表面涂层制备领域，不具备制备整体大尺寸颗粒增强铝合金致密坯料的能力，和本研究的喷射沉积法明确无关[[S14,S5]]。

## 分类与子类

制备颗粒增强铝合金的喷射沉积法主要分为以下子类：

### 1. 喷射共沉积

由Lavernia等学者开发，特征是在常规喷射成形装置上额外设置独立的颗粒雾化喷吹器，将预先制备好的外加陶瓷增强颗粒（如SiC、Al₂O₃等）在指定位置吹入金属雾化锥，使二者共同沉积得到快速凝固的复合坯。该工艺可灵活调控增强相的体积分数，增强相在基体中分布均匀[[S41,S25]]。

### 2. 反应喷射沉积

由Lawley等学者开发，特征是将喷射成形工艺与原位化学反应合成相结合。增强相不在外部预先添加，而是在金属液滴的飞行过程或沉积阶段，通过以下三类途径直接在基体中原位生成[[S4,S12,S21,S41]]：

- **气-液反应**：熔体直接与雾化气体反应形成增强相；
- **液-液反应**：通过双金属喷射反应形成增强相；
- **固-液反应**：通过外加粒子与金属雾滴的反应形成增强相。

原位生成的增强相界面无污染物、与铝基体结合强度更高，可彻底解决外加增强相的润湿性差、界面反应难以控制的问题[[S4,S12,S41]]。

## 工艺流程

颗粒增强铝合金喷射沉积完整工艺流程分为五大环节[[S36,S40,S38,S26,S10]]：

| 环节 | 内容描述 |
|------|----------|
| **1. 合金熔炼** | 将铝合金锭在真空/惰性气体保护下通过感应加热熔化，控制熔体成分均匀并达到指定过热度[[S36,S10]] |
| **2. 气体雾化** | 熔化后的铝合金熔体经预热导流管流出，被高压惰性气流破碎为尺寸范围的弥散熔滴射流[[S36,S40]] |
| **3. 增强颗粒注入** | 将预先制备的陶瓷/金属增强颗粒通过专门的送粉系统喷射注入雾化锥区域，与高速飞行的铝合金熔滴充分混合[[S36,S26]] |
| **4. 沉积** | 不同凝固状态的熔滴（5~50μm熔滴完全凝固、中等尺寸熔滴半固态、>150μm熔滴保持液态）共同撞击在预热的基板表面，附着、铺展、熔合形成连续半液态沉积层[[S40,S13]] |
| **5. 后续凝固及致密化** | 沉积层在基板强制冷却作用下逐步完成快速凝固，获得的沉积坯后续可通过热挤压、热等静压等工艺完成全致密化处理[[S36,S10]] |

## 核心工艺参数

喷射沉积制备颗粒增强铝合金的关键工艺参数常规取值范围及对沉积坯质量的影响规律如下[[S2,S13,S7,S29,S18,S33]]：

| 参数类别 | 常规取值/设置 | 对质量的影响 |
|----------|--------------|-------------|
| **雾化气体种类** | 氮气、氩气等惰性气体；铝合金场景优先选用氩气以避免氮化反应[[S7,S33]] | 影响熔体纯净度，氩气可避免与铝熔体的有害反应 |
| **雾化气体压力** | 0.5~1.0 MPa[[S2]] | 0.8 MPa时致密度可达96%，成型性最佳；压力过低成型性差，过高（1.0 MPa）致密度降至90%，熔滴过度冷却导致液相不足孔隙率上升[[S2,S18]] |
| **熔体过热度** | 液相线以上150~200 K[[S13]] | 保证熔体流动性同时避免过度氧化[[S13]] |
| **沉积距离** | 300~450 mm[[S13,S7]] | 过小：熔滴温度过高，沉积层裹入气体形成大尺寸气孔；过大：熔滴完全凝固，沉积层铺展性差孔隙率升高[[S7]] |
| **基板运动方式** | 棒材：旋转+轴向连续下拉模式；管材/板材：水平旋转轴支撑/水平往复扫描基板模式[[S40]] | 维持圆周方向沉积均匀性，保证沉积距离恒定[[S40]] |
| **合金熔体流量** | 单雾化器1~20 kg/min，双雾化器系统可达80 kg/min[[S29]] | 气液比（GMR）升高使熔滴平均尺寸降低、固相占比增加，宏观偏析降低但微偏析程度上升[[S18,S29]] |
| **增强颗粒注入位置** | 设置于雾化锥中上部区域[[S33]] | 保证颗粒与半固态熔滴充分混合，接触时间短可抑制界面有害反应发生[[S33]] |

气体压力与沉积坯成型性及密度的具体对应关系如下[[S2]]：

| 气压/MPa | 气体流量/g·s⁻¹ | 成型性 | 铸造密度/% |
|-----------|----------------|--------|------------|
| 0.5 | 32.3 | 差 | — |
| 0.7 | 45.2 | 一般 | 80 |
| 0.8 | 51.7 | 好 | 96 |
| 1.0 | 64.6 | 好 | 90 |

## 设备构成

喷射沉积制备颗粒增强铝合金的核心设备构成及对应功能如下[[S9,S40,S36,S7,S11]]：

| 设备单元 | 功能 |
|----------|------|
| **熔炼单元** | 包含感应加热坩埚、保护气氛控制系统，用于在真空/惰性气氛下完成铝合金的均匀熔炼，防止熔体氧化[[S7,S36]] |
| **气体雾化单元** | 包含气体储罐、调压系统、雾化喷嘴。喷嘴分为环孔/环缝两类，Laval超音速结构可使气流出口速度超过1Ma，高效破碎合金熔体生成细尺寸雾化熔滴[[S9]] |
| **增强颗粒输送注入单元** | 包含粉末储料仓、载气输送管路、注入喷嘴，用于将增强颗粒精准送入雾化锥区域实现均匀共混[[S36,S7]] |
| **密闭沉积室单元** | 内置惰性气氛保护、排气与粉尘回收系统，防止喷射过程中熔滴氧化[[S36,S7]] |
| **沉积基板及运动执行单元** | 包含高蓄热系数的水冷基板、运动机构，可实现旋转、轴向连续下拉、水平往复扫描等多种运动模式，适配棒材、管材、板材等不同形状沉积坯的制备[[S40,S7,S9]] |

设备按结构形式可分为立式设备（适配锭坯类产品）和卧式设备（适配管、环、板类产品）[[S9]]。

## 微观组织特征

喷射沉积制备颗粒增强铝基复合材料属于典型快速凝固工艺，冷却速度可达10³~10⁶ K/s[[S36,S37,S24]]，其微观组织具有以下特征：

- **基体晶粒细化**：常规铝基体系下基体晶粒尺寸处于10~100 μm范围[[S37]]，部分高固溶铝合金的基体晶粒可低至1.5~5 μm[[S24]]，沉积态呈均匀等轴状[[S19]]；
- **增强相细小弥散**：以Al-70%Si电子封装材料为例，铝基体晶粒尺寸约10 μm，初生Si相尺寸仅为5 μm，细小的团块状Si相均匀弥散分布在铝基体中，无常规铸造工艺下的粗大针状/五瓣状初生Si相[[S22,S10]]；
- **界面结合良好**：喷射沉积过程中增强相与高温金属液滴的接触时间极短（单颗雾化液滴的高温飞行时间约0.3 s），界面反应程度极低，可基本避免有害界面相生成[[S36]]；
- **无宏观偏析**：快速凝固特征使合金成分分布均匀，消除了常规铸锭冶金中常见的宏观偏析缺陷[[S36,S31]]。

## 优点与局限性

### 核心优势

- 兼具快速凝固特征：晶粒细化、无宏观偏析、合金成分分布均匀[[S36,S31]]；
- 增强相分散均匀，界面反应程度低，材料综合力学性能优于同成分下铸造法制备的复合材料[[S36,S31,S7]]；
- 工艺流程短，将金属雾化与沉积成形合并，可直接制备接近最终形状的盘、棒、管、板类坯料，大幅减少后续加工余量[[S36,S7]]；
- 支持多喷嘴同步喷入多种增强相，可低成本制备梯度或多相复合的铝基材料[[S36,S7]]。

### 现存技术局限性

- 直接沉积的坯料存在2%~20%的孔隙，无法直接制备成品零件，必须经过热挤压、热等静压等二次致密化处理[[S28,S36,S10,S37]]；
- 雾化过程惰性气体消耗量大，雾化成本相较于普通铸造工艺更高，材料过喷损失占比不可忽略，整体沉积收得率有待进一步提升[[S28,S36]]；
- 工艺为多参数耦合控制过程，需精准调控金属液流率、雾化气压、喷射距离、基板运动参数等多个变量，工艺调试难度大[[S36,S37]]。

## 与其它制备方法的对比

喷射沉积、搅拌铸造法、粉末冶金法制备颗粒增强铝基复合材料的多维度对比如下[[S28,S6,S8,S37,S31]]：

| 对比维度 | 喷射沉积法 | 搅拌铸造法 | 粉末冶金法 |
|----------|-----------|-----------|-----------|
| **制备效率** | 单喷嘴25~200 kg/min，单坯最大质量可达600 kg[[S37]] | 适合大规模量产[[S28]] | 低于喷射沉积[[S37]] |
| **生产成本** | 因雾化单元设备投入与气体消耗成本高于普通铸造，低于粉末冶金[[S28,S37]] | 工艺最简单，成本最低[[S28]] | 设备成本较高，工艺繁琐[[S6]] |
| **材料致密度** | 直接沉积态95%~98%，后续致密化可达100%[[S36,S37]] | 较低，易有气孔和夹杂[[S28,S8]] | 坯料空洞率较大，须二次塑性加工[[S6]] |
| **颗粒分散均匀性** | 增强相随雾化流同步注入，避免静置分层偏聚，显著优于搅拌铸造[[S31,S8]] | 增强体易偏聚或上浮下沉[[S28,S8]] | 与混合工序充分时相当[[S6]] |
| **界面反应程度** | 高温停留时间极短，几乎无界面反应[[S31,S36]] | 较高，搅拌过程易引入气体[[S8]] | 常规高温长时烧结时反应较明显[[S31]] |

## 力学性能

喷射沉积制备的颗粒增强铝基复合材料可实现强度、模量与塑性的优良匹配。以10vol.%SiC增强8090铝基复合材料为例：相较于未增强的8090合金，弹性模量从79.5 GPa提升至95.9 GPa，强度同步提升仅出现小幅延展性下降，其综合力学性能优于同成分下搅拌铸造法制备的复合材料[[S20]]。

## 应用领域与应用案例

### 海外已公开的工业级应用

- **英国Osprey金属公司**：1974年实现喷射沉积技术工业化，2000年推出用于电子封装的Al-(22%~70%)Si合金产品，产品直径250 mm、质量20 kg，Si含量偏差可控制在±2%以内，可使用普通刀具加工，表面可镀覆Ni、Cu、Ag及Au[[S10,S11]]；
- **加拿大Alcan公司**：采用COSPRAY工艺将SiC或B₄C引入铝锂基8090合金和铸锭冶金2618合金制备金属基复合材料，10体积%SiC增强8090合金的弹性模量可从79.5 GPa提升至95.9 GPa[[S20,S1,S25]]；
- **美国Howmet公司**：取得Osprey技术授权，开发专属喷射沉积工艺开展高温合金涡轮盘和环用材料的工业化研发[[S25]]。

### 国内已公开的工业级应用

- **上海钢铁研究所**：在喷射沉积制备复合轧辊领域取得重大工业化进展[[S11]]；
- **中南大学**（陈振华、黄培云）：发明的多层喷射沉积专利技术，通过往复沉积强化冷却效果，解决了传统工艺制备厚大坯件时组织易恶化为铸态的问题。已成功应用于制备Al-Fe-V-Si合金、6013/6061/6066铝合金、SiC颗粒增强复合材料，大型多层喷射沉积装置可制备单件质量达100 kg的锭坯和管坯[[S11]]。

## 适用材料体系

喷射沉积法除颗粒增强铝合金之外，可应用于制备铁基材料、铝基材料、铜基材料、镁基材料、高温合金等多类金属基复合材料体系[[S31,S34,S13,S15]]。已完成工艺验证的成熟材料包括Cu-Zr、Cu-Ti铜合金、M2高速钢、Stellite高温合金、Al-Fe-V-Si耐热铝合金[[S13,S31]]，其中颗粒增强镁基复合材料也是该工艺的典型适用对象[[S15]]。
