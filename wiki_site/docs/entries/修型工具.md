## 定义与概述

“修型工具”在压铸全流程中并非指向单一作业场景，其内涵覆盖以下几类典型应用 [[S21,S16,S7,S17]]：

1.  **砂型/砂芯造型修整工具** —— 用于修整砂型（芯）的型面、开设浇冒口、挖沟槽、修圆角与曲面等，是造型工序中的标准工具类别。中国铸造行业权威标准汇编中将其列为造型工具项下的独立术语（编码6.2.58） [[S21]]。
2.  **压铸模保养与修复工具** —— 用于模具日常周检、型腔表面拉伤/龟裂的打磨抛光、分型面研配、补焊后精修等。压铸车间通常配备专职的“修型钳工”，负责使用该类工具进行模具的维修保养工作 [[S7,S18]]。
3.  **压铸件后处理修整工具** —— 用于去除铸件的内浇口残留、分型面飞边、毛刺及各类多余金属。该类别涵盖从手持工具到机械化设备的广泛范围 [[S17,S23,S4]]。

在英文语境中，对应的术语因应用场景不同而各异，包括 trimming tool（修边工具）、fettling tool（铸件清整工具）以及 die repair/polishing tool（模具修复/抛光工具）等。其中，GB/T 8845-2017《模具术语》已正式收录 trimming die（修边模）和 trimming punch（修边冲头）作为标准化中英文术语 [[S2]]。

## 分类与工具特征

### 1. 砂型造型手工修型工具

砂型铸造修型工具多为手持式，常用工具钢、青铜等材料制造，既可由车间自制，也有标准化厂商供货 [[S11,S16]]。主要种类及功能如下：

| 工具名称 | 功能说明 | 
| :--- | :--- |
| 镐刀/刮刀 | 修整砂型（芯）较大平面，开挖浇冒口，切割大沟槽，敲入砂型固定钉 [[S16,S11]] | 
| 成型镶刀 | 修整内外圆角、方角与弧形面，可按需定制异形形态 [[S16,S11]] | 
| 提钩/砂钩 | 修理砂型（芯）深而窄的底面和侧壁，清理内部散砂 [[S16,S11]] | 
| 双头铜勺（秋叶） | 铜制双头匙形结构，用于修整曲面或窄小凹面 [[S16,S11]] | 
| 半圆（竹片梗/平先杆） | 修整垂直弧形内壁与底面 [[S16,S11]] | 
| 圆头修型刀 | 修整圆形及弧形凹槽 [[S16,S11]] | 
| 法兰梗（光槽锯刀） | 修整深窄底面及管子两端法兰窄边 [[S16,S11]] | 
| 压勺 | 一端弧面另一端平面，修整较小平面，开设小尺寸浇口 [[S16,S11]] |

### 2. 压铸件后处理修整工具

针对压铸件的飞边、毛刺及余料，其修整工艺分为手工与机械两大体系 [[S4,S23]]：

| 工艺体系 | 常用工具/设备 | 适用对象 | 
| :--- | :--- | :--- |
| **手工修磨** | 锥刀、风钻、油石、锉刀、刮刀、砂带、木槌、钢丝刷等 | 铝、镁、铜合金压铸件；产量不大或不便于机械清理的场景 [[S4,S23]] | 
| **机械修磨** | 抛光机、砂带磨床、金属切削机床、专用压力切边机（配切边模） | 全品类压铸件及钢铸件；产量大、质量要求严格的场景 [[S4,S23]] |

在机械清理中，trimming die（修边模）是一种关键工装。它由淬硬钢制成，轮廓与铸件形状精确匹配，利用剪切作用切除内浇口、溢流槽端口余料、分型面飞边及芯孔毛刺等 [[S17,S23]]。

### 3. 压铸模保养用修型工具

模具保养场景下的修型工具主要指用于型腔、型芯、分型面修复和抛光的手持动力工具及钳工工具 [[S7,S15,S18,S5]]。要求操作者能灵活使用电动工具、超声波工具等手持动力工具，以提高修型效率 [[S15]]。

## 工艺参数与操作要求

### 修型动力工具参数

常用的手持动力修型工具依赖于气动或电力驱动，其关键工艺参数包含：

| 参数项 | 典型范围/规格 | 
| :--- | :--- |
| 手持电动工具主轴转速 | 1000 ~ 30000 r/min [[S15,S3]] | 
| 气动修型笔工作气压 | 0.4 ~ 0.6 MPa [[S15,S3]] | 

### 相关耗材与修磨路径

修型作业所用的耗材需按修磨精度要求递进使用，通常的耗材替换路径为 [[S15,S3]]：
1.  **粗磨砂轮**
2.  **不同目数的砂布**
3.  **油石**
4.  **抛光膏**
5.  **旋转锉磨头**
操作原则：磨料粒度应由粗到细顺序更换，每次更换磨料种类时必须将前一种磨料彻底清除，并建议与前次磨削纹路呈交叉方向进行抛光，以确保均匀去除刀纹 [[S15]]。

## 安全操作与常见缺陷

### 核心安全要求

- 作业人员应佩戴防尘面罩进行粉尘防护 [[S20,S15]]。
- 修磨方向需避开人体，防止碎屑飞溅伤人 [[S20,S15]]。
- **严禁使用淬硬钢制铲刀/工具直接接触模具成型表面**，以防损伤型面 [[S20,S14]]。清理分型面或缝隙内的嵌合飞边时，需使用铜、木槌等软质工具或以不伤害型面为前提进行铲除 [[S14]]。

### 操作不当引发的常见质量问题

手工作业或参数控制不当时，易产生以下缺陷 [[S15,S14,S7]]：
- **模具型腔棱边塌角** —— 打磨抛光时操作过度导致。
- **修磨局部过切形成凹坑** —— 集中针对局部深伤痕打磨，而未顺延周围均匀抛光。
- **型腔表面残留工具划痕** —— 修磨工艺不规范或未按粒度顺序递进抛光。
- **铸件表面“缺肉”** —— 后续清理飞边毛刺时修除余量过大。
- **飞边增厚或加大的恶性循环** —— 铲除飞边时破坏了型腔表面平整度和光洁度，导致后续压铸产生更严重的飞边 [[S14]]。

## 在压铸全流程中的工艺位置

修型工具的应用贯穿压铸工艺的模具准备阶段与铸件后处理阶段：

1.  **模具准备/维护阶段** [[S7,S9,S18,S5]]：
    *   淬硬后的型腔手工修磨、抛光。
    *   修正热处理变形及分型面研配。
    *   模具补焊后的精修加工。
    *   日常/定期保养中对型腔表面拉伤痕迹、轻微龟裂进行打磨抛光。

2.  **铸件后处理阶段** [[S17,S23,S14]]：
    *   铲除浇注系统、排溢系统、分型面及型芯/顶杆位置产生的飞边和毛刺。
    *   修除铸件表面的多肉、凸起及残留金属。
    *   利用修边模等专用工具，自动化地切除内浇口等余料。

## 相关概念

- **飞边**：在分型面、顶杆或滑块间隙处渗入的金属薄片，是修型工具在后处理阶段的主要清除对象。修型工具清除飞边的作业质量直接影响模具后续使用状态，若不当铲除会加剧飞边产生 [[S14,S12]]。
- **模具保养**：修型工具是执行模具周检、型腔修复与损伤处理的核心工具。压铸模每压射约 10000 模次后需进行一次拆模检查与修型保养 [[S7,S18]]。
- **铸件清理**：压铸件脱模后，需通过手工或机械化的修整工具去除附带的多余物，修型工具（如锉刀、砂轮机、切边模）是铸件清理工序的重要组成部分 [[S23]]。
- **修边模 (Trimming Die)**：由 GB/T 8845-2017 标准定义的压铸修型关键装备，利用淬硬钢制冲模与匹配形状对铸件进行精准剪切，以去除飞边和浇口余料 [[S2,S17]]。
