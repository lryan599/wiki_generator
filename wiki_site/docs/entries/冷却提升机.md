## 概述与歧义说明

“冷却提升机”在铸造工程语境下为兼具冷却与提升功能的工艺设备。依据现行国家标准 GB/T 25370-2020，英文对应术语为 **cooling elevator** [[S1]]。

**重要歧义提示**：尽管本词条初始研究假设将“冷却提升机”定位为压铸生产中用于冷却并提升铸件的辅助设备，但当前所有检索到的工程图纸、标准条目及专业书籍证据均指向 **旧砂冷却提升机**（用于砂型铸造、消失模铸造等工艺中的热旧砂冷却与输送），未发现直接描述压铸铸件冷却提升设备的工程证据 [[S2,S3,S5]]。因此，下文所述设备结构、工作原理及应用场景主要基于旧砂冷却提升机的既有知识，其在压铸后工序中冷却铸件的适用性仍需工程验证。

## 定义与分类

- **标准定义**：cooling elevator / 冷却提升机，在铸造机械术语中被列为独立条目 [[S1]]。
- **功能定位**：同时实现物料的垂直或倾斜提升与冷却降温，常用于热砂处理系统，需配合除尘装置使用 [[S2,S5]]。
- **类别划分**：主流冷却提升机可划分为 **风冷式、喷淋式、浸水式** 三类，配套的提升输送机构可选配 **链板式、斗式、网带式** 结构，以适配不同物料的耐高温、耐磨损输送需求 [[S1]]。

## 结构与组成

冷却提升机的工作原理基于物料在提升过程中的逆流换热。典型的风冷式结构包含以下核心部件 [[S3,S5]]：

| 部件 | 功能说明 |
|---|---|
| 受料口 | 承接上游来料，引导物料进入提升通道 [[S5]] |
| 提升输送单元（提升胶带/提升带） | 带动物料向上输送，表面可设置料斗或挡料板 [[S3,S5]] |
| 可调挡料板 / 调节板 | 控制料层厚度与抛洒状态 [[S5]] |
| 进风通道 / 侧向导风结构 | 引入冷空气，与下落热砂形成逆流换热 [[S3,S5]] |
| 排风通道 / 砂尘分离器 | 排出含尘热风，分离夹带的粉尘 [[S5]] |
| 卸料口 | 将冷却后的物料排出至下游设备 [[S5]] |
| 驱动电机 | 提供提升动力 |
| 保温罩壳（视工艺需求） | 控制散热或防止外部污染 |

**工作原理**：物料由受料口进入，经提升带向上输送；在提升过程中，物料被抛洒或形成瀑布流下落，与由进风通道引入的逆流冷空气充分接触，实现高效的热交换降温。含尘热风经砂尘分离器净化后从排风口排出 [[S3,S5]]。

## 工艺作用

1.  **降温冷却**：通过强制风冷或水冷介质，将高温物料的温度降低至后续工艺允许的范围，防止热量积聚对下游设备或产品质量造成影响。
2.  **温度均匀性控制**：虽然当前证据来自压铸模具温度场研究，但其揭示的冷却原理——即配置冷却结构后可显著降低温度峰值、稳定温度场、抑制热量堆积——为理解冷却提升机减少物料热应力、降低后续变形风险的工艺角色提供了可移植的逻辑参考 [[S4]]。
3.  **工序衔接**：旧砂冷却提升机通常串联在落砂机之后、旧砂再生或储存设备之前，作为砂处理系统中的关键温控节点 [[S2]]。

## 系统布局与相关关系

旧砂冷却提升机并非独立运行，而是铸造车间砂处理系统中的核心设备。其典型系统布局包括 [[S2]]：

-   **上游接口**：落砂机、振动输送机等热砂来源设备。
-   **下游接口**：旧砂再生设备、砂库或混砂机。
-   **配套系统**：必须连接除尘系统以处理换热过程产生的大量含尘热风；风冷式需配置进风管路和风机 [[S2]]。

## 证据缺口与待研究项

鉴于研究假设与现有证据之间存在显著偏倚，以下条目在当前研究中缺乏直接压铸证据支持，特此注明以防误用：

-   **工艺参数**：提升速度（m/min）、冷却时间（s）、铸件温降（℃）、冷却介质流量等具体数值均未在压铸语境下获得证据。
-   **应用场景**：不明。可确认的应用领域为消失模铸造及砂型铸造的旧砂冷却 [[S2,S3,S5]]，压铸铸件冷却输送的应用未见实例。
-   **关联设备**：未找到与压铸取件机器人、切边机、输送带的具体接口证据。
-   **常见问题与维护**：冷却不均、网带卡滞、腐蚀结垢、水雾飞溅等常见问题在当前证据库中无专门讨论。

本词条因此定稿为以现有工程证据呈现的旧砂冷却提升机内容，并明确记录存疑之处，待后续补充压铸方向的专项资料。
