## 概述

热芯盒法（hot box process/热芯盒）是一种加热硬化类树脂砂制芯工艺。其原理是将添加了潜伏性催化剂的液态热固性树脂粘结剂与原砂混成芯砂，经射芯机射入预热至特定温度的金属芯盒中，在热作用和催化剂共同作用下，树脂快速缩聚固化成型。当砂芯表层形成数毫米硬壳后即可从芯盒中取出，依靠砂芯自身余热及固化反应放热完成内部进一步硬化[[S30,S32,S44]]。

该方法在20世纪60年代发展成熟[[S21,S32]]，适用于中、小尺寸（砂芯截面厚度≤50 mm）的高精度砂芯批量生产，在汽车、拖拉机、内燃机及液压件等行业的大批量流水线中得到广泛应用[[S2,S11]]。

## 定义与术语

- **中文标准名称**：热芯盒法，别名热芯盒[[S2,S55]]
- **英文名称**：hot box process / hot-box core making / hot-box method[[S55]]
- **工艺分类归属**：属于树脂砂制芯工艺大类下的加热硬化分支，与温芯盒法同属热芯范畴[[S49]]

## 工艺原理

**树脂热硬化反应机制**：呋喃热芯盒树脂由尿素、甲醛和糠醇在一定摩尔比下缩聚而成（呋喃I型），其分子结构中含游离羟甲基、胺基、亚胺基等活性基团。常温下潜伏催化剂（如氯化铵-尿素水溶液）呈中性或弱酸性，维持非活化状态，保证芯砂具有足够的可使用时间。芯盒受热后，催化剂分解释出酸性离子，促使树脂发生失水缩聚及双键间聚合反应，最终形成不溶不熔的体型网状三维结构，实现砂粒粘结固化[[S46,S45,S40]]。

**热固化基本过程**：将混好的液态热固性树脂砂射入预热至200~250 ℃的金属芯盒内，表层砂受热后粘结剂在几十秒内缩聚硬化。砂芯表层形成5~10 mm硬壳后即可脱模，出盒后利用砂芯余热及固化反应放热完成中心部分自行硬化[[S21,S22]]。

## 材料体系

### 热芯盒树脂分类

热芯盒树脂按化学结构可分为三类，其性能主要取决于尿素、苯酚、糠醇含量及甲醛摩尔比[[S2,S30,S44]]。

**1. 脲醛呋喃（呋喃I型）树脂**  
属含氮树脂，是使用最广泛的热芯盒树脂。按含氮量分三级[[S40]]：
- 高氮型（氮质量分数 11%~14%）：适配非铁合金铸件
- 中氮型（氮质量分数 3%~7%）：适配铸铁件
- 低氮型（氮质量分数 0.3%~3%）：适配球墨铸铁件

特点：粘度低、粘结强度高、保存期长；吸湿性强，发气量大[[S40,S1]]。

**2. 酚醛呋喃（呋喃II型）树脂**  
无氮或极低含氮量，高温性能好、抗粘砂能力强，是铸钢件生产的主流用树脂。其缺点是常温强度低、硬透性差、型砂脆性大[[S30,S40,S44]]。

**3. 脲-酚醛呋喃共聚物（UF/PF/FA）**  
同时引入脲醛和酚醛结构，兼顾呋喃I型和II型特性，弥补单一类型树脂的性能短板[[S40,S44]]。

各类树脂对工艺性能的影响趋势如下表所示[[S2]]。

| 性能趋向 | 增加尿素含量 | 增加苯酚含量 | 增加糠醇含量 |
|:---|:---|:---|:---|
| 强度 | 上升 | 下降 | 上升 |
| 发气量 | 上升 | 下降 | 下降 |
| 含氮量 | 上升 | 下降 | 下降 |
| 吸湿性 | 上升 | 下降 | 下降 |
| 脆性 | 下降 | 上升 | 下降 |
| 硬化速度 | 上升 | 下降 | 上升 |
| 存放性 | 上升 | 下降 | 上升 |

### 原砂要求

原砂选用50/100筛号的中粗砂，泥分质量分数应低于0.5%；粒形优先选用圆形或椭圆形以降低树脂添加量。原砂pH宜为中性至弱酸性（有利于酸固化树脂体系），需完全干燥（水分破坏树脂粘结膜），入混砂机前砂温应严格控制在15~30 ℃[[S2,S18]]。

### 催化剂体系

热芯盒催化剂全部为潜伏型，常温呈中性或弱酸性保证芯砂存放期，受热后释放强酸催化树脂固化[[S45,S32,S40]]：
- **高氮树脂**：一般使用铵盐类（如氯化铵水溶液）
- **中氮树脂**：使用铜盐水溶液（硝酸铜、硫酸铜等）
- **低氮/无氮树脂**：使用铜盐与磺酸盐复配体系

国内工厂最常用的高/中氮树脂催化剂配方为氯化铵:尿素:水 = 1:3:3（质量比）。其中尿素常温下可中和部分酸性，延长存放期，同时降低游离甲醛气味[[S40,S23,S16]]。

### 常用添加剂

- **硅烷偶联剂**：提升树脂膜与砂粒附着力，含氮树脂适配氨基硅烷，无氮树脂适配苯氧硅烷[[S30,S18]]
- **氧化铁粉**（Fe₂O₃）：防止铸件针孔、皮下气孔、渗碳缺陷，加入量为树脂量的8%~15%[[S1,S30]]
- **三氯化铁**（FeCl₃）：提升低温环境下树脂硬化速度[[S30,S1]]
- **硼酸/硼砂**：按需加入，调节砂芯高温性能[[S18]]

## 工艺装备

### 热芯盒模具结构

热芯盒核心组成部分包括：芯盒本体、定位装置、排气装置、顶出机构、测温装置、加热装置[[S31]]。分盒形式主要有垂直分盒与水平分盒（水平对开式）两类[[S37]]。芯盒本体壁厚常用范围为20~30 mm，以保证足够的蓄热能力，防止连续生产过程中温度大幅波动[[S47,S42]]。

### 加热方式

热芯盒加热方式主要分三类[[S37,S54]]：
- **电加热**：采用插入式管状电热元件或通用电热板，温控精度高，温度均匀性好，适配大批量精密砂芯生产
- **煤气/燃油加热**：升温速度快，采用带燃烧腔的框型结构，适合大型厚壁砂芯
- **复合加热**：集成两种加热方式优势

加热功率经验公式：N = 0.075G（kW），其中G为每小时消耗砂量（kg/h）[[S37]]。

### 排气设计

芯盒总排气道面积应为射砂孔总面积的30%~50%。若低于20%将导致砂芯表面疏松、充型残缺[[S42,S19]]。排气方式按优先级排序为：
1. **排气槽排气**：开设在分盒面、射砂面及镶块结合面，深度取0.2~0.4 mm，出口端可扩展至1 mm以上；一盒多芯布局下相邻芯腔排气槽不得连通[[S15,S34,S12]]
2. **间隙排气**：顶芯杆/活块与芯盒配合间隙取0.15~0.3 mm，兼顾排气能力与防卡滞要求[[S12,S19,S34]]
3. **排气塞排气**：仅用于前两种方式无法覆盖的砂芯紧实困难死角[[S42]]

### 标准化工艺流程

**1. 混砂工序**[[S36,S50,S27]]：
干砂 + 氧化铁粉 → 干混 20~30 s → 加催化剂（液体）混碾 50~60 s → 加树脂混碾 50~60 s → 卸砂。热芯盒砂卸砂后有效使用期不超过3~4 h。

**2. 制芯及后处理工序**[[S5,S3,S4]]：
芯盒预热至工作温度 → 喷涂甲基硅油类脱模剂（见湿为宜，喷1次可连续射芯10~15次）→ 射砂 0.5~1 s → 盒内硬化达到规定时间 → 开盒顶出砂芯 → 人工修整飞边毛刺 → 按需涂刷水基/醇基石墨耐火涂料并烘干（残余水分 < 0.5%）→ 尺寸精度、表面强度、通气道通畅性检验 → 合格后入库。

## 优势与局限

### 优势

- 硬化速度快，制芯生产率高，硬化周期可短至几十秒[[S33,S22]]
- 砂芯强度高，常规无需额外芯骨补强[[S33]]
- 砂芯尺寸精度高、表面光洁度好[[S33,S22]]
- 浇注后芯砂溃散性好[[S33]]
- 树脂加入量低（2%~3%），综合材料成本较低[[S21,S18]]
- 工艺流程简单，易于实现自动化流水线生产[[S33]]

### 局限性

- **截面壁厚受限**：仅适配截面厚度≤50 mm的中小型砂芯，截面尺寸突变时易出现局部过烧、局部固化不足，导致制芯破损率偏高[[S2,S33,S11]]
- **模具成本与能耗**：需采用耐高温金属芯盒及加热系统，工装成本较高；整体能耗高于冷芯盒工艺[[S2,S33,S45]]
- **环境问题**：生产过程中产生刺激性烟气及游离甲醛排放，不符合绿色铸造要求[[S39,S45,S33]]
- **吸湿性与存放期**：呋喃I型树脂砂芯吸湿性强，存放过程中强度易下降30%~40%，存放期一般不宜超过24 h[[S5,S22]]
- **气孔倾向**：呋喃树脂含氮量若控制不当，铸件易出现针孔、皮下气孔缺陷[[S22,S1]]
- **应用趋势**：随着冷芯盒工艺普及，热芯盒法整体应用占比呈下降趋势[[S11,S22]]

## 常见缺陷及控制

| 缺陷类型 | 原因分析 | 解决方案 |
|:---|:---|:---|
| 粘砂 | 砂芯局部射砂紧实度不足 | 优化射砂参数提升紧实均匀性；高要求铸件砂芯涂刷防粘砂涂料[[S33,S41]] |
| 气孔/针孔 | 树脂含氮量过高、发气量过大 | 控制树脂含氮量；加入树脂量8%~15%的氧化铁红粉[[S33,S1]] |
| 砂芯酥脆/过烧 | 树脂/固化剂配比不当、固化过度或原砂杂质超标 | 原砂含泥量 ≤1%且完全烘干；严控粘结剂与催化剂加入量；避免芯盒温度过高[[S33,S5]] |
| 内腔毛刺 | 热芯盒覆膜砂受热软化缩沉或冷却收缩产生毛细裂纹，铁液渗入 | 优化芯盒加热温度和硬化时间，避免砂芯过烧开裂[[S41]] |
| 砂芯变形 | 芯盒加热温度不均、顶出机构顶出力不均衡 | 保证芯盒加热系统温度场均匀，合理布置顶杆位置[[S33]] |

## 质量控制与检测

**强度检测**[[S7,S8]]：
- 起芯（热拉）强度：在210 ℃±5 ℃条件下加热20 s测得；常规砂芯合适的常温抗弯强度为4~5 MPa
- 检测用8字形标准试样，取样需搓碎至松散粒状后成型测试

**发气量检测**[[S9,S7]]：
发气量测定仪升温至1050 ℃±5 ℃恒温，准确称取1 g±0.01 g砂样放入瓷舟，送入测定仪红热部分封闭后读取最大发气量数值（cm³/g），检测时长15 s；发气量随树脂加入量升高而升高。

**尺寸精度检测**[[S38,S53]]：
- 依据铸件工艺卡指定检验工具检测
- 芯盒加工精度要求：工作表面无明显缺陷，分盒面平行度 ≤0.05:100，垂直度 ≤0.02:100，合模间隙 ≤0.1 mm，射砂孔三孔同轴度 ≤0.2 mm
- 铸件整体尺寸精度可达CT9~CT10级

## 应用领域

热芯盒法在以下领域得到典型应用：

- **叶轮类铸件**：采用呋喃树脂砂制芯可显著提升叶轮铸件尺寸精度，已成为该类高精度流体零部件的主流制芯工艺之一[[S21]]
- **气缸盖铸件**：广泛用于批量生产复杂水腔芯和螺栓孔芯。在16V190柴油气缸盖及IEM集成排气铝合金气缸盖等产品中实现成熟量产，曲面水腔砂芯成型精度可满足铸件壁厚公差6~8 mm的设计要求[[S41,S43,S53]]
- **液压件**：热芯盒覆膜砂砂芯强度和溃散性好，适用于液压阀体复杂薄壁内腔砂芯生产，可提升内部油道表面质量并减少清砂难度[[S11]]
- **汽车、拖拉机行业**：各类大批量流水生产的中小型高精度砂芯[[S2,S11]]
