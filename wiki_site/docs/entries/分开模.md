## 概述

“分开模”并非现行压铸领域国家标准或行业规范中的正式术语。在GB/T 8847-2003《压铸模术语》、GB/T 8845-2017《模具术语》等官方文件中均未收录该词条[[S17,S4]]。该表述仅出现在少数行业工艺文档与一线生产口语场景中，属于压铸行业的小众俗称，未形成全行业统一的标准化定义。

根据现有文献资料检索，"分开模"主要存在两类非标准口语化指代：其一指向特定模具结构——分体式压铸模具（英文术语 **split die**）；其二用于通俗描述压铸循环中动模与定模沿分型面完全分离的动作片段。

## 第一类指代：分体式模具结构

### 定义

在压铸模具设计语境下，"分开模"对应的核心概念是 **split die**（分体式模具），指将模腔主体沿轮廓拆分为多个独立块体，再将各块体嵌入模架中固定的分块式特殊模具结构。该结构设计的核心目的是降低复杂异形型腔的机械加工难度[[S1,S7,S25]]。

### 结构组成

分体式压铸分开模由 **模体** 与 **模架** 两大模块组成，其通用基础组成单元可归纳为以下8大类[[S21,S8,S10,S19,S2]]：

| 组成大类 | 包含零件/组件 |
|---------|--------------|
| 定模镶块、动模镶块 | 型腔、型芯、活动型芯 |
| 导向零件 | 导柱、导套 |
| 推出复位机构 | 推杆、推管、卸料板、推板、推杆固定板、复位杆、导向柱、导套、限位钉 |
| 浇注系统组件 | 浇口套、分流锥、内浇道、横浇道、直浇道 |
| 排溢系统组件 | 溢流槽、排气槽 |
| 抽芯机构组件 | 活动型芯、滑块、斜滑块、斜销、弯销、齿轮齿条、楔紧块、限位钉、限位块 |
| 模架支承固定零件 | 套板、座板、支承板、垫块 |
| 加热冷却系统 | 冷却通道、加热装置 |

模体包含定模和动模，细分有型腔、浇注系统、溢流排气系统、抽芯机构、导向部分、模体部分及加热冷却系统；模架包含推出机构、预复位机构（摆轮、摆轮架、预复位推杆）、模脚垫块与座板[[S16,S5,S24]]。

### 不同压铸合金的适配与限制

分体式分开模在不同压铸合金应用中存在显著的差异化要求[[S15,S28,S22,S14]]：

| 合金类型 | 适配特点 | 分体结构注意事项 |
|---------|---------|----------------|
| 锌合金 | 不易粘模、熔点低不腐蚀模具；适配热室压铸；可压铸极薄复杂铸件 | 分体镶拼结构可大幅延长模具寿命 |
| 铝合金 | 对铁亲和力强易粘模；比热容大、凝固收缩率高；必须采用冷室压铸工艺 | 分体结构需额外预留足够的冷却通道空间，避免局部热应力不均导致开裂 |
| 镁合金 | 冲蚀模具程度低于铝合金；不易粘模；熔体粘度低、流动性好；可适配冷室与部分改进型热室压铸 | 需严格控制模具温度场均匀性，动定模温差不得高于50℃，避免铸件裂纹缺陷 |

## 第二类指代：开模动作片段

### 定义

"分开模"的第二类口语化指代用于通俗描述压铸全循环工序中，铸件完全凝固后动模与定模沿分型面执行完全分离的动作片段。该表述属于标准开模工序的部分场景化描述，并非独立的压铸工序节点[[S12]]。

### 工艺参数要求

开模动作阶段的核心工艺参数包含以下几项[[S20,S6,S11,S9,S13]]：

| 参数项 | 要求/校核公式 | 说明 |
|-------|-------------|------|
| 开模时间 | ≥2 s | 厚壁铸件、结构复杂模具需适当延长 |
| 最大开模距离校核 | H<sub>max</sub> > h₁ + h₂ + h₃ + h₄ + 10 mm | H<sub>max</sub>为压铸机最大开模距离；h₁为定模厚度；h₂为动模厚度；h₃为铸件推出距离；h₄为铸件及其浇道总高度 |
| 动模座板行程校核 | L ≥ h₃ + h₄ + 10 mm | L为动模座板行程，保证铸件顺利取出 |
| 开模速度 | 分慢速段与常速（快速）段两级调节 | 慢速段防止分型面磕碰研伤；常速段根据铸件大小、脱模阻力优化 |
| 开模力 | 需低于压铸机额定最大开模力 | 开模力为克服铸件包紧力与运动摩擦力的总和 |

### 常见缺陷及控制措施

开模动作过程中可能出现的质量缺陷及其控制措施[[S23,S27,S18]]：

| 缺陷类型 | 诱因 | 控制措施 |
|---------|------|---------|
| 粘模拉伤 | 模温过高、金属液高速冲蚀型腔表面导致金属与模具表面生成化合物粘连 | 保证脱模剂喷涂均匀到位；合理匹配开模延时时间，避免过早开模 |
| 分型面研伤 | 开模初始速度过快、动定模导向机构磨损导致动作错位刮擦分型面 | 开启开模慢速段设置；定期维护导柱导套导向精度 |
| 动定模不同步 | 压铸机液压回路压力偏差、合模曲肘机构动作不同步 | 定期校准压铸机开合模液压参数；对曲肘机构做定期润滑维护 |
| 复位不良 | 斜销延时抽芯动作行程设置不足、合模过程中推杆与活动型芯发生干涉 | 优化滑块斜孔后空当量尺寸，按公式 δ = M sinα 核算延时抽芯行程，避免干涉 |

## 与相关术语的关系

"分开模"作为非标准俗称，与压铸领域若干标准术语存在明确的语义差异[[S4,S17]]：

- **开模**：压铸标准工序术语，指代从合模锁紧状态到动模后退至设定完全打开位置的完整动作过程。"分开模"仅描述其中模具两半完全分离的状态片段，不对应"开模"的完整定义边界。
- **分模**：模具设计环节的专属术语，指代模具开发阶段分型面规划、模具分腔的设计环节。"分开模"作为生产现场用语，与设计阶段的"分模"概念不同。
- **分型面**：标准定义为模具上用于取出铸件和浇注系统凝料的可分离接触表面（parting line）。"分开模"描述的是沿分型面分离的动作或分体结构本身，与"分型面"这一几何概念有所区别。

## 术语待考证说明

由于"分开模"未在GB/T 8847-2003《压铸模术语》及GB/T 8845-2017《模具术语》等国家标准中获得收录，其标准化定义暂时缺失[[S17,S4]]。本文档基于现有行业文献中出现的两类主要指代进行梳理：分体式模具结构（split die）与开模动作片段。后续若出现更新的标准或权威文献给出明确定义，本词条应予以修订。
