## 定义与概述

**铁素体铸钢**（ferritic cast steel）是基体组织以铁素体为主的铸钢，在中国铸造国家标准体系中属于明确的铸钢分类术语[[S25]]。其定义为：基体组织为铁素体的铸钢，涵盖铁素体耐蚀铸钢、铁素体耐热铸钢和磁性钢三大子类[[S25,S34]]。

室温下铁素体铸钢的显微组织特征显著：碳溶解于α-Fe中形成间隙固溶体，保留体心立方晶格结构，常温下溶碳量极低，仅为约0.0008%，显微组织呈多面体等轴晶粒形态[[S4]]。该类材料在室温下呈铁磁性[[S30]]。

## 分类体系

铁素体铸钢可按性能用途和合金化程度划分，形成四大基本类别[[S21,S34]]：

| 分类维度 | 类别 | 说明 |
|----------|------|------|
| 按性能用途 | 铁素体耐蚀铸钢 | 在特定腐蚀介质中具抗腐蚀能力 |
| 按性能用途 | 铁素体耐热铸钢 | 在500℃以上具抗氧化性 |
| 按性能用途 | 磁性钢 | 具有特定磁性能的铸钢 |
| 按合金化程度 | 铁素体碳钢 | 以碳为主要合金元素 |
| 按合金化程度 | 铁素体合金铸钢 | 含合金元素总量较低 |
| 按合金化程度 | 铁素体不锈钢铸钢 | 以铬为主要合金元素，Cr≥12% |
| 按合金化程度 | 铁素体耐热铸钢 | 含有铬、硅或铝等抗氧化元素 |

在中国国家标准分类中，铁素体型被明确列为耐热铸钢的四种基础金相类别之一（其余为珠光体、马氏体、奥氏体），同时也被列为耐蚀铸钢的四种基础基体类型之一（铁素体、马氏体、奥氏体及双相）[[S21,S34]]。

## 组织特征与热处理特性

铁素体铸钢的关键组织特性在于其相变行为：**铁素体不锈钢铸钢加热至1000℃以上几乎不发生向奥氏体的固态相变**，因此无法通过常规热处理淬硬，也不能依靠热处理实现晶粒细化以改善力学性能[[S19]]。这一特性从根本上决定了铁素体铸钢与马氏体铸钢、奥氏体铸钢在使用性能和工艺路径上的差异。

实际生产中，铁素体不锈钢铸件通常仅进行退火处理以消除铸造应力，退火温度为800～850℃。为避免脆性，退火保温后应在空气中冷却，也可水冷，但不宜随炉缓慢冷却[[S19]]。

## 典型牌号与命名规则

### 中国牌号

国内典型的铁素体不锈钢铸钢牌号包括 ZG1Cr17、ZG00Cr18Mo2 等，铁素体耐热铸钢如 ZG1Cr19Al3 等[[S26]]。牌号命名采用铸钢代号“ZG”后跟碳的名义万分含量，后续标注各合金元素的符号及百分含量[[S37]]。

工程用铁素体碳钢铸钢采用不同的命名规则：以铸钢代号“ZG”后跟两组数字，第一组为屈服强度（MPa）数值，第二组为抗拉强度（MPa）数值。例如 ZG200-400 表示屈服强度为200 MPa、抗拉强度为400 MPa的工程用铸钢[[S9]]。

### 国际对应标准

ASTM A743/A743M 标准收录了铁素体型耐蚀铸钢牌号，其完整化学成分表可直接用于中外牌号成分匹配[[S6]]。ASTM A217/A217M 为高温承压用铁素体/马氏体铸钢专用标准，收录了全系列低合金铁素体耐热承压铸钢牌号，并提供完整的UNS编号及各元素质量分数范围[[S3]]。

## 化学成分与力学性能

### 典型成分范围

铁素体不锈钢铸钢以铬为主要合金元素，铬的质量分数一般在13%～30%之间[[S26]]。高铬含量使钢表面可形成致密的Cr₂O₃钝化膜，赋予其优良的常温耐腐蚀性能和高温抗氧化性能[[S14]]。

GB/T 2100-2002标准中的铁素体铸钢牌号及化学成分表明确标注：除给出范围外，所有未标注范围的元素含量均为上限值[[S5]]。

### 典型力学性能

退火态铁素体铸钢的常规力学性能范围如下[[S30]]：

| 力学性能指标 | 数值范围 |
|------------|---------|
| 屈服强度 σ_s | 98～166 MPa |
| 抗拉强度 σ_b | 180～280 MPa |
| 伸长率 δ | 30%～50% |
| 断面收缩率 ψ | 70%～80% |
| 冲击韧性 α_K | 156～196 J/cm² |
| 布氏硬度 | 50～80 HBS |

铁素体铸钢在退火态下强度与硬度较低，但具有良好的塑性和韧性[[S4,S30]]。值得注意的是，铁素体不锈钢铸钢的晶粒在高温加热时易粗大，且在350～500℃温度区间长期服役易发生**475℃脆性**，导致冲击韧性显著下降[[S1,S26]]。

## 工艺特性

### 铸造性能

铁素体铸钢作为直接铸造成形的铸态产品，其成分和组织设计适配铸造工艺需求[[S4]]。铬含量超过16%的铁素体不锈钢铸钢，铸态组织粗大倾向明显，这是其铸造生产中的重要关注点[[S26]]。

### 焊接性能

铁素体铸钢整体焊接性能优良，可通过焊补修复铸造缺陷[[S40]]。其焊前预热、施焊温度、焊后热处理的参数要求可参照承压类铁素体-马氏体铸钢的施焊规范执行[[S28]]。焊接时的主要问题之一是近缝区晶粒急剧长大引起的脆化，且这种晶粒粗化无法通过后续热处理改善[[S1]]。

### 机加工性能

铁素体组织本身硬度较低（退火态50～80 HBS），有利于切削加工[[S30]]。但应注意高铬含量的铁素体不锈钢铸钢晶粒粗大倾向可能影响加工表面质量。

## 缺陷与应用限制

铁素体铸钢存在以下几类典型缺陷和应用限制[[S7,S26,S31]]：

| 缺陷/限制类型 | 发生条件 | 表现与影响 |
|-------------|---------|-----------|
| 475℃脆性 | 400～500℃长期服役 | 与含铬铁素体的有序化相关，韧性急剧下降 |
| σ相脆性 | 550～700℃长期保温 | σ相析出导致脆性显著上升 |
| 室温脆性 | 常温条件 | 高铬铁素体钢室温冲击韧性低 |
| 晶粒粗大 | 高温加热/铸态 | 无法通过热处理细化，恶化力学性能 |
| 低温脆性 | 低温服役 | 冲击韧性极差，不适用于低温服役场景 |
| 热强性不足 | 高温服役 | 高温热强性较低 |

475℃脆性和σ相脆性均可通过加热至475℃以上然后快速冷却来改善[[S26]]。为改善焊缝区与热影响区的力学性能，通常还添加少量的Ti和Nb，以阻止热影响区晶粒长大[[S26]]。

## 在压铸产业链中的角色

### 压铸模具应用：未证实

**现有公开的压铸行业量产实践中，未检索到铁素体铸钢被用作铝合金/锌合金压铸主型腔成型模具材料的记录**[[S38,S17,S20]]。

原因在于其材料性能与压铸模具服役要求存在根本性差距：铁素体铸钢的高温硬度、热疲劳抗性远低于马氏体铸钢及H13（1.2344）热作模具钢，无法承受压铸过程中金属液的高速冲刷和反复急冷急热的工况[[S20,S17]]。压铸行业主流型腔模具材料为H13热作模具钢，其工作温度可达650℃时仍保持硬度，并具有良好的耐热疲劳性和抗熔蚀性[[S17]]。

### 压铸零件应用：未证实

黑色金属压铸的量产件主流采用302/304、316牌号的奥氏体不锈钢，仅少量试验性采用410、431牌号马氏体钢，**不存在铁素体铸钢压铸零件的工业落地信息**[[S38]]。

### 辅助装备：理论适配

铁素体铸钢在压铸产业链中**仅理论适配铝合金熔炼环节的辅助装备部件**，如坩埚构件、搅拌工装等长期接触熔融铝液的零部件[[S33]]。研究数据表明，X17牌号铁素体类钢在800℃条件下抗熔融铝液溶解的性能优于奥氏体12X18H9T钢[[S33]]。但此结论为实验室条件下的熔蚀对比，尚缺乏压铸生产现场的长期验证数据。

## 与易混淆材料的区分

铁素体铸钢需与以下材料明确区分[[S4,S25,S34]]：

| 对比材料 | 关键区分特征 |
|---------|------------|
| 铁素体球墨铸铁 | 铁素体铸钢含碳量远低于球墨铸铁，碳当量<2.11%，无游离石墨相析出；球墨铸铁基体为铁素体但分布球状石墨 |
| 变形铁素体钢 | 铁素体铸钢为直接铸造成形的铸态产品，晶粒形态与锻轧加工获得的变形材存在显著差异 |
| 马氏体铸钢 | 铁素体铸钢加热时不发生向奥氏体的固态相变，无法淬硬；马氏体铸钢可通过淬火获得马氏体组织 |
| 奥氏体铸钢 | 铁素体铸钢室温组织为体心立方铁素体，呈铁磁性；奥氏体铸钢为面心立方奥氏体，无铁磁性 |

## 相关标准

铁素体铸钢涉及的主要中国标准与国际标准如下：

- **GB/T 2100-1980**《不锈耐酸钢铸件技术条件》：共收录17个牌号，按金相组织分为马氏体钢、铁素体钢、奥氏体钢、奥氏体-铁素体钢、沉淀硬化钢五类，其中包含完整的铁素体铸钢系列牌号，并配套规定了化学成分、热处理工艺和晶间腐蚀试验方法[[S24]]。

- **GB/T 2100-2002**《一般用途耐蚀钢铸件》：等效采用ISO 11972:1998《通用耐蚀铸钢》，共收录19个耐蚀铸钢牌号，包含全部铁素体型牌号，设有明确的化学成分要求、热处理规范和常温力学性能指标[[S27,S5]]。

- **ASTM A743/A743M**：收录铁素体型耐蚀铸钢牌号，提供完整的化学成分要求和常温力学性能数据[[S6,S15]]。

- **ASTM A217/A217M**：高温承压用铁素体/马氏体铸钢专用标准，收录全系列低合金铁素体耐热承压铸钢牌号，提供化学成分范围和力学性能要求[[S3,S12]]。

> **注意**：GB/T 6967《工程结构用中、高强度不锈钢铸件》收录的牌号全部为马氏体不锈钢，**未收录任何铁素体铸钢牌号及相关技术条款**[[S11]]。该标准与GB/T 2100系列互为补充，各自覆盖不同的不锈钢铸钢类型。
