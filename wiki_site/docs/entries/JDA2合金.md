## 概述
JDA2合金是上海交通大学轻合金中心丁文江院士、彭立明教授团队开发的免热处理压铸铝合金体系，属于JDA系列中的Al‑Mg‑X分支[[S21,S8,S13]]。中文常称为JDA2合金、JDA2体系合金，英文对应JDA2 alloy或JDA2 die‑cast aluminum alloy[[S13,S12]]。该体系目前已公开JDA2a和JDA2b两个性能定位不同的亚型[[S12]]。两类合金均在压铸态下即可达到目标力学性能，完全省去传统高性能压铸铝合金所需的T6/T7热处理工序及其后续矫形工序[[S11]]。依托高强高韧特性与独特的薄壁效应，JDA2体系主要面向汽车底盘类高冲击、高疲劳耐久要求的受力结构件，并在大型一体化薄壁车身结构件领域具有广泛的应用前景[[S21]]。

## 化学成分
JDA2体系合金属于上海交通大学公开的SJTU‑Al‑Mg‑Si‑Mn类免热处理合金范畴，其代表性的化学成分范围列于下表[[S14,S19]]。

| 元素 | 含量 / wt.% |
|------|------------|
| Si   | 2.0~13.0   |
| Fe   | ≤0.1       |
| Mn   | 0.4~2.0    |
| Mg   | 0.5~2.0    |
| Zr   | 0.1~0.5    |
| Sb   | 0.1~0.3    |
| V    | 0.01~0.1   |
| Ti   | 0.15~0.3   |
| Mo   | 0.2~1.0    |
| Y    | 0.1~0.3    |

在JDA2b合金中，通过添加Ti、Zr、V等微量元素实现晶粒细化，同时采用Ca/稀土复合变质方法获得细小的共晶Mg₂Si相，从而在保证铸造流动性的同时提升力学性能[[S12]]。通过调节Mg与Si的比例并联合上述微量元素的调控，合金可在铸态下实现屈服强度＞180 MPa、延伸率＞10%的强塑配合[[S19]]。

## 力学性能
JDA2a与JDA2b在性能上具有明确的分工。JDA2a的密度低于纯铝，耐腐蚀性能优异，铸态延伸率超过15%，但强度水平低于JDA2b[[S12]]。JDA2b的铸态力学性能则更为突出，其显微组织中含有40%~50%的共晶组织，为压铸成形提供了良好的工艺基础[[S12]]。文献报道的JDA2b室温拉伸性能如下：

| 性能指标 | 典型范围或值 |
|----------|------------|
| 屈服强度 | 180~220 MPa |
| 抗拉强度 | 360~400 MPa |
| 断后伸长率 | 8%~12% |

JDA2b在铸态下已可达到甚至超越传统AlSi10MnMg合金经T6热处理后的力学性能[[S11,S12]]。与商业标杆合金相比，其铸态性能显著高于经SQ‑T6处理后的德国Silafont‑36合金[[S12,S9]]。完全免热处理的特性使应用JDA2b的压铸件省去了常规高性能压铸件所需的热处理费用和后续矫形成本[[S11]]。

## 工艺特性与薄壁效应
JDA2系列为完全免热处理合金，压铸件在铸态下即可直接使用，无需固溶‑时效处理[[S11]]。在压铸工艺性方面，JDA2b通过共晶组织设计以及Ti、Zr、V晶粒细化，有效克服了部分低硅铝镁系合金流动性差的痛点，降低了薄壁铸件生产过程中的报废率[[S8,S12]]。

JDA2体系存在典型的薄壁效应：在减薄壁厚的情况下，铸件的力学性能可进一步提升，而非衰减[[S21,S8]]。这一特征使其特别匹配大型薄壁车身结构件一体化压铸对材料的要求。19 mbar级双阶段超真空辅助高压压铸工艺已被证实可显著降低Al‑Si‑Mg‑Mn类合金的孔隙率，使铸态抗拉强度提升5.6%、伸长率提升43%，且伸长率的性能波动可降低71%[[S2]]；对于同属铝镁硅锰系的JDA2合金，此类高真空压铸工艺同样是实现高性能薄壁铸件的重要技术手段。

## 应用
JDA2体系合金的核心应用目标为对冲击韧性和疲劳耐久要求高的汽车底盘类受力结构件[[S21]]。与此同时，凭借薄壁效应和优异的压铸充型能力，该体系在大型薄壁车身结构件（如减震塔、纵梁等）领域具备广泛的应用前景，是新能源汽车一体化压铸车身的关键候选材料之一[[S21,S8]]。
