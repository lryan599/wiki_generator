## 压板压实造型机

### 概述
压板压实造型机是压实造型机大类下的一个设备子类，其官方标准定义为“单纯借助压力紧实砂型的造型机”[[S18,S29]]。在国家标准《铸造机械 术语》（GB/T 25370-2020）中，该设备术语编号为4.1.12，其上位术语“压实造型机”编号为4.1.11 [[S20,S2]]。该设备完全属于通过压板直接对型砂施压实现砂型紧实的铸造造型设备。

该设备中文别名包括“压板压实式造型机”和“压实造型机”，标准英文对应术语为 **squeezing molding machine** [[S23]]。需要注意，jolt squeeze machine 对应的是“震压造型机”，high pressure moulding machine 是压实造型机的下属子类“高压造型机” [[S23,S25,S29]]。

### 工作原理与压实方式
压板压实造型机通过液压、机械或气压作用于压板、柔性膜或组合压头，使砂箱内型砂获得紧实 [[S18,S23,S28]]。其压实方法主要分为单向压实和差动压实两大类 [[S16]]。

**单向压实**包含以下两种核心构型：
- **上压式**：压头从上向下，将辅助框内的型砂压入砂箱，至压头底平面与砂箱上边缘相平为止。余砂存在于砂箱上方 [[S14,S16]]。
- **下压式**：型砂加到与砂箱上沿齐平后，工作台带动型板向上升起，将辅助框内型砂压入砂箱，最终模板与砂箱齐平。余砂存在于下方 [[S14,S16]]。

**差动压实**则同时采用上压和下压两种动作，先由压头预压（上压），然后模样下压，最后压头再终压，其砂型紧实度和均匀度均优于单向压实 [[S16]]。

两类构型下砂型的紧实度分布特征存在明显差异。型砂在垂直压力作用下向下流动时，与砂箱壁、模型壁产生外摩擦，砂粒间产生内摩擦，导致压力沿砂箱高度递减，越远离压头的位置紧实度越低 [[S14,S10]]。上压式与下压式压实造型机的工作原理简图直观展示了两种造型方法下型砂紧实度分布的差异 [[S22]]。

为改善普通平压板压实带来的砂型不同位置紧实度分布不均的缺陷，可采用非平面轮廓的 **成形压头** 对型砂进行压实，以适配模样的高低差特征 [[S9,S17]]。

### 设备分类与等级
根据压实比压（砂型表面单位面积所受的压力）的大小，压板压实造型机可分为三个等级 [[S3,S7]]：

| 等级 | 压实比压范围（MPa） |
|------|-------------------|
| 低压压实 | 0.13～0.4 |
| 中压压实 | 0.4～0.7 |
| 高压压实 | 0.7～1.5 |

高压造型机（high pressure molding machine）是压实造型机的下属子类，在国家标准中定义为“采用压实比压大于700kPa的压实造型机” [[S28,S29]]。采用高比压（一般可达1.3MPa左右）并配合浮动的多触头压头，可进一步提高砂型的紧实度和均匀性 [[S17]]。

### 核心工艺参数与性能
普通压板压实造型机的主要技术参数范围如下 [[S3,S7,S4,S17,S24]]：

| 参数 | 典型范围/数值 |
|------|-------------|
| 压实比压 | 低压0.13～0.4MPa，中压0.4～0.7MPa，高压0.7～1.5MPa |
| 适配砂箱高度 | 200～250 mm |
| 适配砂箱尺寸 | 350×250 mm 至 1500×1200 mm |
| 起模方式 | 顶杆起模、回程起模 |
| 全机械化生产率 | 100～120 箱/小时 |
| 砂型表面硬度 | A型75～90 |
| 铸件尺寸公差等级 | CT8～CT9 |

### 铸造工艺中的角色
在砂型铸造全流程中，压板压实造型机位于混砂工序之后、制芯与合箱浇注工序之前，其直接承接混砂工序输出的合格型砂，完成砂型紧实工序，输出高刚性合格砂型供给后续造型工位 [[S16,S10]]。

以震压式压板压实造型机（即先震击后压实的组合机型）为例，其完整工作流程依次为：填砂 → 震动紧砂 → 压板压实顶部松散型砂 → 起模 [[S15]]。经过压实后，砂型整体紧实度较单纯震击得到明显提升。

### 应用领域与适用性
压板压实造型机主要面向低高度、结构简单的扁平中小铸件生产，典型应用场景包括 [[S17,S4,S24]]：
- 普通机电零部件
- 小型汽车铸件
- 普通机床小件的批量生产

当模样较高、特别是模样顶面具有很大平面时，采用压实式造型机难以获得均匀的砂型紧实度。此外，由于压实所需总压力取决于砂箱内框面积，当砂箱很大时机器结构将变得笨重，因此压实法只适用于较小砂箱且模样高度不高的造型 [[S17]]。

### 与其他造型设备的比较
基于铸造设备手册的对比，压板压实造型机与其他主流造型机的性能差异如下 [[S16,S6,S8]]：

| 对比设备 | 压板压实造型机的相对表现 |
|---------|----------------------|
| 震击式造型机 | 砂型上部紧实度更高，噪声更低，生产效率高30%以上；但对深模样的下部紧实效果弱于震击机型 |
| 抛砂式造型机 | 型砂紧实速度更快，产出砂型一致性更好，对操作工人技术要求更低；但无法适配超大砂箱的大铸件生产 |
| 气流冲击式造型机 | 无需配套高压储气系统，维护成本更低，砂型紧实度分布受模样轮廓影响更小；但砂型整体平均密度较低，透气性表现略差 |

### 优缺点
**主要优势** [[S16,S10,S4,S5,S12]]：
- 设备结构简单，制造成本低，一般中小型工厂均可自行制造
- 运行噪声远低于震击类机型
- 无需特殊复杂设备基础，搬移方便
- 操作维护门槛低

**主要局限性** [[S16,S10,S5,S12]]：
- 普通单向压板压实工况下，砂型沿砂箱高度方向紧实度分布不均，砂箱上部紧实度远高于底部
- 仅适合低高度扁平铸件，不适用于深腔复杂模样的造型生产
- 高规格大砂箱机型的总压力需求显著提升，设备体积和占地空间随之大幅增加
