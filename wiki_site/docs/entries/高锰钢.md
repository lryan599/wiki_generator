## 分类与基本定义

高锰钢属于奥氏体锰钢大类，是已工程化和产业化的五大类铸造耐磨材料中历史最悠久、应用最广泛的一类铸造抗磨钢，我国高锰钢铸件产量在各类合金钢铸件中居首位 [[S16,S13,S8]]。

标准型Mn13高锰钢又称Hadfield钢，1882年由英国冶金学家R.A.Hadfield研制发明，其初始成分为w(C)=1.35%、w(Si)=0.69%、w(Mn)=12.76%，属于含锰量10%以上的合金钢 [[S16,S2,S13,S29]]。

## 化学成分

### 基础高锰钢（ZGMn13系列）

普通ZGMn13系列高锰钢基础化学成分范围：C 1.0%~1.4%、Mn 11%~15%、Si 0.3%~0.8%，S ≤ 0.05%、P ≤ 0.10%。Mn/C比值推荐控制在8~10，以保证水韧处理后获得单相奥氏体组织 [[S49,S46]]。

### 铬改型高锰钢（ZGMn13Cr2/ZG120Mn13Cr2）

铬改型高锰钢是在传统ZGMn13高锰钢中加入铬元素（Cr 1.50%~2.50%）得到的改型材料 [[S37,S7]]：

| 元素 | 质量分数范围（%） |
|------|-------------------|
| C | 0.91~1.30 |
| Mn | 11.0~14.0 |
| Cr | 1.50~2.50 |
| Si | 0.30~0.80 |
| S | ≤ 0.040 |
| P | ≤ 0.070 |

### 含铜高锰钢

含铜高锰钢以普通ZGMn13成分为基础添加Cu元素，现有公开标准未对其Cu质量分数作强制限定，属于行业内改进型高锰钢合金化研究范畴，其牌号与普通ZGMn13、ZGMn13Cr2共同归属高锰钢大类 [[S29]]。

## 铸态与热处理后的显微组织

### 铸态组织

高锰钢凝固时液态金属中首先析出γ相奥氏体核心，随奥氏体枝晶长大，在非平衡铸造冷却条件下大部分奥氏体被保留至室温，奥氏体以枝晶形态生长，晶界和晶内伴随析出碳化物 [[S30,S36]]。

**M₃C型碳化物**（(Fe,Mn)₃C）：碳在奥氏体中的溶解度随温度下降而降低，当冷却过程中越过Fe-Mn-C相图ES线时，碳从过饱和奥氏体中析出，同时Mn、Fe原子替代部分渗碳体中的Fe原子形成该型复合碳化物，沿奥氏体晶界呈网状分布，部分也在奥氏体晶内析出 [[S46,S30,S52]]。

**M₇C₃型碳化物**：当钢中Cr质量分数达到1.5%~2.5%时，部分Cr原子取代M₃C碳化物中的Fe、Mn原子，随Cr/C比值升高，碳化物晶格结构转变为M₇C₃型。该型碳化物高温稳定性优于M₃C，多沿奥氏体晶界断续分布，硬度可达1800~2800HV [[S29,S39,S4]]。

ZGMn13Cr2铸态下的组织为奥氏体和M₃C、M₇C₃碳化物，此描述已由公开材料条目验证 [[S29]]。

### 水韧处理

水韧处理是高锰钢的关键热处理工艺，国家强制要求最低固溶温度不低于1040℃，实际生产常用区间为1050~1100℃，保温时间通常1~1.5h，随后在≥950℃的入水温度下快速水冷，水池水温需控制在40℃以下并保持流动 [[S47,S7,S23]]。

完整工艺步骤 [[S12,S7]]：

| 步骤 | 工艺参数 | 目的 |
|------|---------|------|
| 分段升温（常温~600℃） | 薄壁件70℃/h，中等壁厚件50℃/h，厚壁/复杂件30~50℃/h | 避免铸件开裂 |
| 保温及继续升温 | 600℃保温1~2h后以100~150℃/h升温至1050~1100℃ | 使碳化物充分溶入奥氏体 |
| 水淬 | 快速移入流动冷水池（≥950℃入水，水温<40℃） | 避免碳化物重新析出 |

水韧处理后铸态奥氏体+大量网状碳化物组织转变为单一过饱和单相奥氏体组织，硬度为180~220HBW，韧性与塑性良好。工业实际生产中可能残留极少量分散未溶碳化物，需符合GB/T 13925—1992金相检验标准 [[S49,S36,S44]]。

## 力学与耐磨性能

### 常规力学性能

经水韧处理后高锰钢的典型力学性能 [[S32,S17,S15]]：

| 性能指标 | 数值 |
|---------|------|
| 抗拉强度 σ_b | 637~735 MPa |
| 屈服强度 σ_s | ≥ 396 MPa |
| 延伸率 δ | ≥ 50% |
| 冲击韧度（常温） | ＞ 100 J/cm² |
| 冲击韧度（-80℃） | 39~49 J/cm² |
| 硬度 | 180~225 HBS |

高锰钢在低温条件下仍保持较好的冲击韧度，无明显冷脆转变特征 [[S32,S17]]。

### 加工硬化特性

水韧处理后的高锰钢基体原始硬度仅为170~230HBS，在强烈冲击或挤压载荷作用下，表层发生显著的塑性变形诱发加工硬化，同时伴随奥氏体向马氏体转变，硬化后表层硬度可提升至500~800HBW，硬化层深度可达10~20mm，表层下未变形基体仍保持高韧性，实现高耐磨与高抗冲击性能的协同 [[S17,S14,S28,S24]]。

## 液态模锻（挤压铸造）工艺应用

### 工艺参数

液态模锻高锰钢的常规工艺参数范围为：比压100~150MPa，浇注温度1400~1450℃，保压时间120~150s，该参数适配高锰钢类黑色金属的压力凝固需求，可有效抑制常规铸造易产生的缩孔缺陷 [[S45,S9]]。

### 组织致密化效果

液态模锻工艺下，压力作用使金属液凝固形核率显著提升，晶粒大幅细化，可完全消除常规铸造工艺易产生的缩孔、缩松缺陷。水韧处理后材料密度较金属型铸造提升约1.55%，组织均匀性远高于砂型及常规金属型铸造高锰钢 [[S27,S33]]。

## 局限性

传统高锰钢的核心使用局限性包括 [[S43,S14,S10,S22]]：

1. **低载荷工况不适用**：仅在强烈冲击、大挤压载荷条件下才能通过表面塑性变形实现充分的加工硬化；低冲击/低载荷工况下表层硬度无法提升至500HB以上，原始硬度仅约200HB，耐磨性甚至低于普通淬火高碳钢或铸铁 [[S18]]。

2. **高温使用受限**：使用温度超过300℃时会从奥氏体中析出碳化物，材料脆化且原有加工硬化效应消失，无法在高温工况下服役 [[S50,S24]]。

3. **铸造热裂倾向高**：材料自由线收缩值为2.4%~3.0%，远高于普通碳素钢，同时导热系数仅为碳素钢的1/4~1/6，铸造过程热应力大，裂纹类废品占总废品的50%左右 [[S43]]。

4. **超高冲击工况下磨损失效**：当冲击载荷过高时，加工硬化层发生在亚表面而非最表面，最表面硬度和耐磨性都较差 [[S10]]。

5. **塑性变形倾向**：低载荷长期服役过程中受外力作用易发生塑性变形，高锰钢斗齿等零件与铲斗连接部位易出现开叉失效 [[S10,S22]]。

## 与其他耐磨材料的简要对比

| 材料类型 | 特征 | 适用工况 |
|---------|------|---------|
| 高锰钢 | 高韧性、强加工硬化特性 | 大冲击载荷磨损工况 [[S26,S18,S31]] |
| 高铬铸铁 | 硬度高、常温即高耐磨，脆性较高 | 低冲击磨料磨损工况 [[S26,S18]] |
| 低合金耐磨钢 | 经适配热处理后获中等硬度与良好强韧性配合 | 中等冲击、中低载荷磨损场景 [[S18]] |

高锰钢采用液态模锻工艺制造后可进一步提升组织致密性和耐磨性，可直接适配挖掘机斗齿、球磨机衬板、破碎锤头类零件，无需后续复杂水韧处理即可获得更优的耐磨服役性能 [[S29,S20]]。
