## 概述

吃砂量是铸造工艺设计中一个核心尺寸参数，通常指模样（或砂芯）与砂箱壁、箱顶（底）及箱带之间预留的最小砂层厚度[[S1,S10,S11]]。在带砂芯的压铸、低压铸造等特种工艺中，该术语也延伸至砂芯内部芯骨表面到工作表面之间的砂层厚度，或单位重量铸件所对应的芯砂消耗量[[S7,S12]]。吃砂量的合理选取直接关系到砂型（芯）的紧实度、强度、发气量以及铸件的凝固行为和最终质量。

> **适用范围说明**：吃砂量的设计原理主要源于砂型铸造，其取值体系在树脂自硬砂、高压造型、静压造型等工艺中已有大量经验数据。对于采用砂芯形成复杂内腔的压铸与低压铸件，因金属型（模具）自身提供高强度刚性支撑，砂芯吃砂量可显著小于普通砂型铸造，但仍须保证芯骨不暴露并兼顾溃散性[[S7]]。本条目以压铸及带砂芯的特种铸造为应用背景，同时援引砂型铸造中的通用设计准则。

## 术语辨析

### 中文同义术语

在工程现场与工艺文献中，吃砂量常与下列术语混用或关联：

- **用砂量 / 砂耗量**：侧重整体型（芯）砂的消耗总量，与吃砂量正相关。[[S9,S17]]
- **砂层厚度**：对应于吃砂量作为空间距离的定义，尤指最小砂层厚度。[[S9,S10]]
- **砂铁比**：指每生产单位重量合格铸件所消耗的型砂总质量（造型+制芯），是吃砂量宏观控制效果的衍生指标。吃砂量整体取值越大，砂铁比越高。[[S17,S9]]

### 英文对应术语

依据 GB/T 5611—2017《铸造术语》[[S15]]及国际常用表述[[S20]]，对应关系如下：

| 中文术语      | 英文对应                                        |
|---------------|-------------------------------------------------|
| 吃砂量（厚度）| sand thickness, minimum sand thickness         |
| 用砂量/砂耗量 | sand consumption, sand usage                    |
| 芯砂吃砂量    | core sand consumption                           |
| 砂铁比        | sand metal ratio（GB/T 5611—2017 官方对应词）   |

## 工艺作用与缺陷影响

### 对型（芯）强度与充型质量的作用

吃砂量不足将导致砂型局部紧实困难，强度不足，浇注时易发生胀砂、粘砂、掉砂甚至跑火等缺陷[[S1,S10]]；对于砂芯，芯骨外部预留砂层过薄则金属液可能渗透至芯骨，造成严重粘砂，内腔清砂困难[[S12]]。反之，吃砂量过大虽然提高了安全性，却显著增加型砂用量、增大砂芯发气总量，超出临界值后将大幅提高气孔、砂眼缺陷概率[[S8]]。

### 对铸件凝固与热模数的影响

采用ProCAST数值模拟对几何模数分别为5 mm、10 mm的ZL205A铝合金铸件的研究表明[[S2,S18]]：

- 当吃砂量 **< 1.4 倍** 铸件几何模数时，砂型在金属液高温作用下长时间处于保温状态，导致铸件实际热模数大于几何模数，凝固时间延长；
- 当吃砂量 **> 1.4 倍** 几何模数后，砂型开始对铸件产生冷却效果；
- 当吃砂量 **> 4 倍** 几何模数时，铸件热模数趋于稳定，不再随吃砂量增大而明显变化。

该规律对带砂芯的压铸件凝固控制有直接指导意义：过薄的砂层可能使砂芯在充型和凝固初期成为“保温层”，改变顺序凝固条件，诱发缩松等缺陷。

## 测量与控制方法

- **称重法**：通过统计造型/制芯环节的总砂消耗量与同期合格铸件总质量，计算平均砂铁比，间接评估吃砂量的整体水平[[S2]]。
- **数值模拟法**：利用ProCAST等铸造工艺仿真软件，建立铸件‑砂型（芯）热模型，求解铸件热模数分布，反推满足凝固工艺要求的最小吃砂量[[S2]]。
- **工艺参数调节**：吃砂量在设计阶段即被确定，主要通过调整砂箱尺寸、芯骨直径、模样在砂箱中的布置方式以及砂芯壁厚来实现。实际生产中还需配合型砂配方、结合剂种类及涂层工艺，以保证在给定吃砂量下砂芯强度与溃散性平衡[[S7,S12]]。

## 应用场景

### 砂型铸造

吃砂量是砂箱工艺设计的必算参数，直接决定一箱中铸件的排列数量和砂箱尺寸选用，适用于手工造型、震击造型、静压造型及树脂自硬砂工艺[[S10,S11]]。

### 带砂芯的特种铸造

在高压铸造铝合金缸体、低压铸造转子发动机缸体等复杂内腔零件中，砂芯被高温金属液完全包覆。此时砂芯吃砂量极小，砂芯设计需选用合脂等粘结剂保证高温强度，同时兼顾浇注后良好的溃散性以利于清砂[[S7]]。压铸缸体砂芯常采用覆膜砂、热芯盒或冷芯盒制芯工艺[[S16]]。
