加工余量，又称铸件机械加工余量，是铸造工艺中一项核心设计参数。它指为保证铸件加工面尺寸和零件精度，在铸件工艺设计阶段预先增加、并在后续机械加工过程中被切去的金属层厚度[[S19,S38,S29,S55]]。加工余量直接影响铸件的最终质量与生产成本，与铸件尺寸公差、收缩余量、起模斜度等参数共同构成铸件尺寸精度控制体系。

### 加工余量的分类

根据不同的划分维度，加工余量可分为多种类型：

**1. 按加工阶段划分**

-   **总加工余量（毛坯余量）**：指毛坯铸件某一表面尺寸与最终零件设计尺寸的差值，等于该表面从毛坯到成品整个加工过程中各工序切除金属层厚度的总和[[S47,S50,S52]]。
-   **工序余量**：指单道机械加工工序中切除的金属层厚度。总加工余量等于各工序余量之和[[S47,S50]]。

工序余量又可进一步分为：

-   **单边余量**：用于非对称加工表面，如单一平面，表示单侧切除的厚度[[S47,S16]]。
-   **双边余量**：用于对称加工表面，如回转体内外圆，表示直径方向上总切除量的一半[[S47,S16,S72]]。

**2. 按加工部位划分**

-   **外表面余量**：指铸件外表面预留的加工余量。
-   **内腔余量**：指铸件内孔、内腔等表面预留的加工余量。通常，受砂芯变形、浇注时内表面更易粘附夹渣等因素影响，铸件内表面（铸孔）的加工余量需大于外表面[[S62,S48]]。

### 与相关工艺参数的概念辨析

加工余量需与以下几个易混淆的工艺参数明确区分：

-   **与收缩余量**：收缩余量是为补偿铸件在凝固冷却过程中的线收缩，在设计模样、芯盒时放大的尺寸。它属于模样尺寸的一部分，并非后续需要切削去除的金属层，二者功能与设置目的完全不同[[S56,S76,S64]]。
-   **与起模斜度**：起模斜度是为方便模样从铸型或砂芯从芯盒中取出，在垂直于分型面的壁面上预留的工艺斜度。起模斜度不纳入加工余量范畴，但在有加工要求的表面上，其设置范围不能超出加工余量[[S56,S39]]。
-   **与铸件尺寸公差**：铸件尺寸公差（代号CT，分CT1至CT16共16个等级）指铸件实际尺寸允许的变动范围。它与加工余量配套使用，共同定义铸件的极限尺寸。具体而言，铸件的某一表面在铸态下的最大尺寸不应超过其最终零件尺寸、要求的加工余量（RMA）及铸造总公差（CT）之和[[S76,S38,S12]]。

### 主要影响因素

加工余量的大小选取并非一成不变，主要受以下几方面因素共同影响[[S11,S62,S61,S60,S71]]：

-   **铸造合金种类**：不同合金的收缩特性、表面质量差异显著。例如，铸钢件表面粗糙、变形量大，其加工余量通常大于灰铸铁件；有色金属铸件（如铜、铝合金）的加工余量则相对更小[[S60,S76]]。
-   **铸造工艺方法**：铸造工艺的精度直接决定所需余量的大小。例如，熔模铸造可达0.3~1.5mm的单面余量，而砂型铸造通常需要3~10mm[[S21,S75]]。
-   **生产批量与工装精度**：大批量生产采用高精度机械化工装，铸件尺寸一致性好，可选用较小的加工余量；而单件小批量手工造型则因误差较大，需预留更多余量[[S62,S60]]。
-   **浇注位置**：铸件在浇注时的空间位置对余量有显著影响。位于浇注顶面的区域更易积聚气孔、夹渣等缺陷，其加工余量等级需比底面、侧面降低一级选用，通常为侧、底面余量的两倍[[S11,S62,S71]]。
-   **铸件基本尺寸**：铸件的轮廓尺寸和结构复杂度越大，形位误差控制难度越高，所需的加工余量也相应增大[[S11,S62]]。

### 相关标准规范

加工余量的选取有严格的国家标准和国际标准可依。

**1. 国家标准**

-   **GB/T 11350-89《铸件机械加工余量》**：该标准适用于砂型铸造、金属型铸造、压力铸造、低压铸造和熔模铸造等工艺，明确规定加工余量代号为MA，等级由精到粗共分A至J九个等级[[S32,S31]]。余量值的选取需根据加工表面的最大基本尺寸或其距加工基准的较大尺寸，在标准提供的表格中查取[[S22]]。标准规定了标注格式，例如 `加工余量按GB/T11350 CT10-MA H/G级`，其中斜线上方H为顶面余量等级，下方G为底、侧面余量等级[[S37,S22]]。
-   **GB/T 6414-1999《铸件尺寸公差与机械加工余量》**：该标准等效采用ISO 8062体系，用于替代GB/T 11350的相应内容。其将加工余量代号更新为RMA，等级扩展为A至K共10级[[S63,S61]]。余量值依据最终加工后铸件的最大轮廓尺寸范围选取，其标注示例为 ` GB/T 6414 - CT12 - RMA6(H)`[[S63,S38]]。
-   **标准体系的核心差异**：GB/T 11350-89采用“包容原则”，即加工余量（MA）数值中已包含铸件尺寸公差（CT）的负偏差，贴合国内直接绘制铸造工艺图的习惯。而基于ISO体系的系列标准中，加工余量（RMA）与尺寸公差（CT）在数值上相互独立[[S31]]。

**2. 国际标准**

-   **ISO 8062**：是铸件尺寸公差与机械加工余量的基础国际标准，RMA代号即源于此体系[[S63,S61]]。
-   **ASTM/ADCI标准**：对于压铸件，美国材料与试验协会（ASTM）及北美压铸协会（NADCA）标准建议，为避免刀具过度磨损，最小机加工余量通常设为0.010英寸（约0.254mm）。总加工余量为该最小值、铸件公差与加工公差之和[[S23,S30,S28]]。

### 不同铸造工艺的加工余量对比

不同铸造工艺因其成型原理和精度等级不同，所需的加工余量存在显著差异。下表对比了几种常见工艺的典型加工余量范围：

| 铸造方法 | 典型单面加工余量范围 | 备注 |
| :--- | :--- | :--- |
| **砂型铸造** | 3 ~ 10 mm | 顶面余量通常为侧面的2倍，粗加工等级下余量甚至可达20mm[[S62,S57,S41]] |
| **金属型铸造** | 0.76 ~ 2.3 mm | 铸件尺寸越大，余量要求越高[[S57,S69]] |
| **熔模铸造（水玻璃）** | < 2 mm | 精度较高[[S21]] |
| **熔模铸造（硅溶胶）** | 0.3 ~ 1.5 mm | 高精度下最小可达0.25mm左右[[S21,S75]] |
| **压力铸造** | 0.2 ~ 0.8 mm | 通常直接选用0.5mm，余量极小，以保护致密表面[[S24,S35,S26]] |

### 压铸件的加工余量

压铸件的加工余量选取有其显著的特殊性。

**1. 选取原则**
压铸件因在高压、高速下充型，表层因激冷作用形成一层致密的细晶组织，厚度约为0.5~1.2mm，这使得压铸件具有优异的力学性能。因此，其加工余量的核心原则是**越小越好**[[S26,S35,S46,S20]]。

-   **优先非切削加工**：当压铸件精度不满足要求时，应优先考虑校正、拉光、挤压、整形等精整加工方法，以保留强度较高的致密表层[[S18,S44,S49]]。
-   **避免大余量切削**：过大的加工余量不仅破坏性能优异的表层，还可能暴露铸件内部的气孔、缩松等缺陷，反而降低零件的力学性能和气密性[[S8,S18,S44]]。
-   **基准与方向性**：应尽量选取不受分型面及活动成型件影响的表面作为毛坯加工基准面。待加工内表面尺寸以大端为基准，外表面尺寸以小端为基准计算余量[[S18,S44,S35,S26]]。

**2. 典型余量值**
压铸件的机械加工余量值很小，常规小尺寸压铸件推荐单边余量为0.3~0.5mm，整个合理的余量区间为0.2~0.8mm[[S35,S26]]。对于铰孔加工，余量选取另有规定：直径小于18mm的铸孔，余量为孔径的1%；直径大于18mm的铸孔，余量为孔径的0.4%~0.6%，且绝对值不超0.3mm[[S35,S26]]。

**3. 压铸件成型尺寸基准与余量的关联**
当压铸件带有脱模斜度且同时留有加工余量时，成型尺寸的基准选择需遵循特定规则，如下表所示[[S14,S7]]：

| 加工余量情况 | 型腔尺寸基准 | 型芯尺寸基准 |
| :--- | :--- | :--- |
| **无加工余量** | 以大端为基准 | 以小端为基准 |
| **两面留加工余量** | 以小端为基准 | 以大端为基准 |
| **单面留加工余量** | 以非加工面的大端为基准，加上加工余量 | 以非加工面的小端为基准，减去加工余量 |

### 余量选取不当的典型缺陷

加工余量的设定直接影响铸件质量与成本，选取不当会导致以下问题：

| 余量情况 | 带来的主要问题 |
| :--- | :--- |
| **余量不足** | 1. **造成铸件报废**：无法完全切除铸件表层的夹砂、气孔、缩松等缺陷，加工后表面残留“黑皮”，导致尺寸和表面粗糙度不达标[[S62,S65,S70]]。<br>2. **增加刀具损耗**：刀具直接切削坚硬的铸态表皮，加剧磨损[[S62]]。<br>3. **典型案例**：某铝合金发动机缸体主油道因加工余量仅0.695mm，在铸件出现热缩凹陷时，加工后便暴露黑皮缺陷[[S6]]。 |
| **余量过大** | 1. **增加生产成本**：浪费金属材料，显著增加机械加工工时[[S62,S65]]。<br>2. **降低零件性能**：过度切除性能优异的致密表层，导致加工后表面的力学性能下降[[S62,S11,S40]]。 |

### 应用实例

**1. 阀体类铸件**
在对内腔清洁度和承压安全性要求极高的高压临氢或氧气工况阀门（特别是大口径DN≥200）中，为完全清除砂芯下表面易粘附的渣孔、浮砂、夹渣等缺陷，阀体内腔的加工余量应设置为阀体外部平均加工余量的**2.0倍**[[S59,S13,S27]]。

**2. 缸体类铸件**
对于增压缸体等铸件，可根据各部位的重要性和后续加工要求，实施**差异化分区余量设计**。例如，上表面和需要珩磨的内孔段，因有较高的精度和表面质量要求，可设置较大的加工余量（如8mm），而下表面和侧面设置相对较小的余量（如5mm）[[S27]]。
