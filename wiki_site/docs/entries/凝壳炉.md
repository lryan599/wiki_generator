## 定义与概述

凝壳炉（英文标准术语：**skull furnace**，常用别名 **skull melting furnace**、**vacuum skull melting**、**Induction Skull Melting，简称 ISM**；中文亦称作凝壳熔炼炉、真空凝壳炉）是一种采用"凝壳"原理的特种真空熔炼设备[[S12,S16,S33]]。根据中国铸造术语国家标准中的明确定义，凝壳炉是"采用真空自耗电弧重熔法熔炼和铸造活性金属的真空电弧炉，先熔化的金属液在水冷铜坩埚内凝结成壳，可防止坩埚对后熔化金属液的污染，坩埚内金属液在真空中完成浇铸"[[S26]]。

该技术起源于20世纪50年代初的美国矿业局阿巴尼试验站：研究者在普通自耗电极电弧炉熔炼过程中意外发现，若将熔池中熔融金属倾注出来，坩埚内会残留一个凝固的金属壳，由此开发出凝壳熔炼技术[[S16,S11]]。

凝壳炉主要用于钛、锆等活性金属及难熔金属的精密铸造熔炼工序，是整个精密铸造工艺链中的核心上游熔炼设备[[S19,S35]]。

## 工作原理

凝壳熔炼的基本原理在于：在真空环境下，利用保护气氛或高真空条件，通过电弧、感应或电子束等热源将金属炉料熔化为高温熔体；水冷铜坩埚的强制冷却作用使与坩埚内壁接触的金属液迅速凝固，形成一层由同材质金属构成的固体壳层（即**凝壳**），这层凝壳将高温熔体与水冷铜坩埚完全隔离[[S16,S32]]。

凝壳在熔炼过程中承担两个核心作用[[S16]]：
1. 防止坩埚材料对活性金属熔体的污染，使坩埚内衬与熔体成分相同；
2. 保护水冷坩埚工作表面不被高温液态金属破坏。

凝壳的形成受坩埚冷却强度和输入功率的共同控制。合理的冷却水温度和流量与炉子输入功率相匹配，可使凝壳厚度维持在工艺允许范围内，保证75%以上的内部金属液处于设定过热度的完全液态，供后续浇注使用[[S13,S16]]。

## 工艺作用与产线定位

凝壳炉在钛、锆等活性金属铸造工艺链中承担**熔炼与浇注**工序：将预先制备的自耗电极或散装炉料在真空下熔化为具备一定过热度的洁净金属液，随后通过坩埚倾转在3～6秒内完成浇注，金属液在真空中浇入铸型[[S13,S35]]。

熔炼完成后，可倾倒的熔融金属占坩埚内总金属量（含凝壳）的**70%～80%**（即金属熔化浇注率），残留于坩埚内的同成分凝壳可重复用于下一炉次[[S18,S20]]。凝壳炉可直接为离心铸造、真空压铸、熔模精密铸造等后续成型工序提供完全无耐火材料污染的高温金属液[[S13]]。

## 结构与组成

### 总体结构

凝壳炉的基本组成与普通真空自耗电弧熔炼炉相似，但因用途不同存在显著差异[[S13]]：
- 配备大尺寸熔铸室及高抽速真空系统；
- 采用短粗型可翻转水冷铜坩埚（结晶器）；
- 配备更大功率的供电系统以获得大的熔池；
- 坩埚下方增设离心铸造装置，用于致密薄壁铸件的离心浇注；
- 配置气动快速升降电极机构，保证在数秒内完成金属液倾倒。

### 水冷铜坩埚

坩埚材质以紫铜或无氧铜为常规选择，经锻造和机加工成型，兼顾高导热性和导电性[[S30]]。其典型设计参数如下[[S9]]：
- 坩埚内径D与自耗电极直径d之比 **d/D = 0.45～0.75**；
- 坩埚深度H与内径D之比 **H/D = 1.2～1.5**；
- 坩埚壁厚设计范围 **10～50 mm**，增加壁厚可显著降低烧穿风险。

坩埚外侧冷却系统有两种成熟结构[[S15]]：
1. **管状冷却式**：将冷却水管焊接在坩埚外壁，冷却水路方向稳定，坩埚意外烧穿时冷却水不易直接涌入熔体，安全性更高；
2. **整体水套式**：采用一体式密闭不锈钢水套，冷却效率更优，水流采用自下而上的流向以避免气塞和"蒸汽袋"形成。

### 真空系统

凝壳炉标配由机械泵、罗茨泵、油增压泵（或油扩散泵）组成的高真空抽气机组，可实现10⁻⁵ mmHg级以上的工作真空度，满足钛合金等高活性金属真空熔炼的防氧化要求[[S13]]。

### 冷却水系统

凝壳炉配套**双水源配置**[[S15]]：
- **保安水源系统**：储存足量软化纯水，可在停电停水等紧急工况下持续向坩埚等核心冷却部位供水，防止坩埚烧穿；
- **循环水源系统**：配置大容量蓄水池，保证正常工作时坩埚冷却水进出水温差稳定在**10℃**左右。

### 离心铸造装置

工业生产用凝壳炉的离心铸造装置采用立式离心铸造机，配置于熔炼坩埚正下方。离心盘直径一般为**600～1600 mm**，转速范围**0～600 r/min**，采用皮带传动和变频调速[[S3]]。

## 主要类型

### 按热源和熔炼方式分类

| 类型 | 主要特点 | 行业地位 |
|------|----------|----------|
| **真空自耗电弧凝壳炉** | 直流供电，自耗电极作负极、水冷坩埚作正极，起弧熔化电极；结构简单，维护成本低，易于大型化；需预先制备自耗电极，废料直接回收难度高 | 当前钛合金精密铸件生产中最广泛使用的主流设备[[S19,S25]] |
| **感应凝壳炉（ISM）** | 分块绝缘式水冷铜坩埚与中频感应线圈耦合，电磁力实现熔体搅拌与悬浮；无需自耗电极，可直接使用废钛料、海绵钛等；熔体过热度更高，成分均匀，可快速切换合金品种 | 适合小批量多品种特种高温合金精密铸造[[S10]] |
| **真空非自耗凝壳炉** | 采用非自耗电极持续引弧熔化炉料 | 存在未完全解决的技术瓶颈，尚未成为行业主流设备[[S19]] |
| **电子束凝壳炉** | 高真空电子束轰击加热炉料，可大量使用低成本回炉料 | 同样存在未完全解决的技术瓶颈，尚未成为行业主流设备[[S19]] |

### 按炉体结构分类

凝壳炉按炉体结构可分为五大类[[S2,S13]]：
- **卧式炉**：炉体为卧式圆筒形，制造容易、操作方便，是铸钛工业最早使用的形式；
- **立式炉**：空间利用率高，坩埚和离心盘的相互位置更加合理，分为直立圆筒形和直立方形；
- **双室炉/多室炉**：熔炼室与铸型冷却室由闸板阀隔离，浇注完成后铸型可单独在真空或惰性气氛下冷却，熔炼室可提前开启清理，大幅提升生产效率；
- **连续式炉**：配置多工位铸型预热室和铸件冷却室，按工艺全流程实现高度机械化和自动化的连续生产。

## 关键工艺参数

采用真空自耗电弧凝壳炉进行钛合金凝壳熔炼的主要工艺参数参见下表[[S18]]。

| 项目 | 参数范围 | 说明 |
|------|----------|------|
| 熔炼真空度 | 1.33×10⁻¹～10⁻³ Pa | 真空度过高易导致高蒸气压合金元素挥发，过低则引起氧化和电弧辉光放电 |
| 熔炼电流密度 | 0.4～0.6 A/mm² | 用电极截面积确定熔炼电流 |
| 熔炼电压 | 25～35 V | 较低电压可防止侧弧，加速熔炼速度 |
| 电极直径与坩埚直径比 d/D | 0.45～0.9 | 间隙不得小于30～40 mm避免侧弧击穿坩埚 |
| 熔池深度与坩埚直径比 H/D | (1.2～1.3)∶1 | 熔池深度H可按公式 H = (4.5×I)/100 由熔炼电流I(A)计算 |
| 金属熔化浇注率 | 70%～80% | 可倾倒出的熔融金属占坩埚内金属总量的百分比 |
| 金属过热度 | 60～80 ℃ | 加大熔化电流、减小坩埚-电极间隙可提高过热度 |
| 总浇注时间 | 3～6 s | 断弧后超过18 s时熔池表面金属会凝固 |
| 电极预热时间 | 15～20 s | 起弧后以20%额定电流预热，可提高后续熔化速率 |
| 离心转速 | 200～500 r/min（视铸件可调整至400～800 r/min） | 用于薄壁复杂钛合金铸件的离心加压成型 |

| 项目 | 控制要求 | 说明 |
|------|----------|------|
| 坩埚进水温度 | ≤20 ℃ | 冷水源温度上限 [[S18]] |
| 坩埚出水温度 | ≤35 ℃ | 保障熔炼安全的核心要求 [[S18]] |

## 优缺点分析

### 主要优点

- 依靠自生成的同材质金属凝壳作为内衬，从本质上杜绝坩埚材料对高活性钛、锆等合金的污染[[S5,S25]]；
- 结构相对简单，日常维护成本低，设备大型化难度小[[S25]]；
- 可获得70%以上具有合格过热度的熔融金属直接浇注，适配离心浇注工艺制备高致密度薄壁钛合金铸件[[S25]]；
- 真空环境下熔炼可有效脱除熔体中气体杂质，提升合金纯净度和力学性能[[S5]]。

### 主要局限性

- 自耗电弧凝壳炉需提前制备成形自耗电极作为炉料，对原料初始形状有要求，边角废料直接回收难度高[[S25]]；
- 熔炼过程中无法中途取样调整成分，需提前保证电极成分均匀合格[[S25]]；
- 整体熔化速率受电极进给速度和电弧热输入限制，生产效率低于部分大容量常压熔炼炉[[S25]]；
- 熔体无电磁搅拌作用（仅ISM型具备），成分均匀性低于常规真空感应炉[[S25]]。

### 与普通真空自耗电弧炉的差异

普通真空自耗电弧炉仅用于制备实心铸锭，无可倾转坩埚和配套离心浇注系统，产出为全部凝固的大尺寸铸锭。凝壳炉配备短粗型可倾转水冷坩埚，保留70%以上熔融金属可直接浇注铸件，在铸型适配性和铸件成型效率上远优于普通自耗电弧炉[[S13]]。

## 应用场景

凝壳炉可适配熔炼的金属材料包括：钛及钛合金、锆及锆合金、镍基高温合金、钽铌等难熔金属[[S19,S27]]。

典型铸件产品涵盖：
- 航空发动机压气机机匣、整流叶片、附件液泵叶轮、承力框和支承架[[S19]]；
- 钛合金医用植入件、钛精密运动部件、钛高尔夫球头[[S31]]；
- 双吸锆叶轮等化工用耐腐蚀复杂结构件[[S14]]。

浇注方式可依据铸件要求选择重力浇注或离心浇注：离心转速通常控制在**200～500 r/min**，针对薄壁复杂铸件可适当调整至**400～800 r/min**，铸件出炉温度需控制在**400 ℃**以下，以避免高温出炉引发的表面氧化[[S18]]。

## 常见缺陷与对策

| 缺陷类型 | 成因 | 解决措施 |
|----------|------|----------|
| **凝壳过厚（熔化浇注率不足）** | 坩埚底部冷却强度过高，熔化功率不足，导致底部凝壳厚度超标，可倾倒熔融金属占比低于70% | 优化水冷系统分区冷却参数，适当降低坩埚底部冷却水流量，提升电弧输入功率 [[S3]] |
| **坩埚碳污染** | 石墨材质水冷坩埚在高温下与钛合金发生渗碳反应，每炉次碳增量可达0.02%，降低合金塑性和薄壁铸件流动性 | 全面更换为全水冷铜坩埚；仅在安全冗余要求极高的特殊场景下使用石墨坩埚并严格控制熔炼时长 [[S1]] |
| **金属液氧化** | 熔炼真空度低于1.3×10⁻¹ Pa阈值，炉膛残留氧分压过高，钛合金熔体与氧反应生成硬脆富氧α相层 | 将熔炼阶段真空度稳定控制在1.33×10⁻¹～10⁻³ Pa区间，必要时充入高纯氩气保护抑制氧化 [[S18]] |
| **凝壳熔穿/水蒸气爆炸** | 冷却强度不足或熔化速率过高，凝壳厚度低于设计阈值，液态金属直接接触铜坩埚甚至击穿，引发高温液态钛与冷却水反应发生水蒸气爆炸 | 严格控制冷却水压力0.25～0.3 MPa、水流速≥3 m/s，匹配对应熔化功率，定期检查凝壳厚度 [[S25,S3]] |

## 标准与安全规范

截至现有检索结果，尚未发现独立的中国国家标准（GB）或机械行业标准（JB/T）针对凝壳炉的设计、试验或安全制定专项标准。已确认的是：中国铸造术语类国家标准中收录了"凝壳炉（skull furnace）"的正式定义条目[[S26]]。

在操作安全层面，凝壳炉除需满足通用真空自耗电弧炉安全规范外，还需注意以下特殊要求[[S28]]：
- 铸型直浇道口需与离心盘同心，避免金属液飞溅；
- 铸型需对称紧固，防止离心工况下松动或损坏；
- 熔炼结束断电后，电极提升与坩埚翻转动作时序须准确配合，防止电弧击穿坩埚或电极残头撞击坩埚。
