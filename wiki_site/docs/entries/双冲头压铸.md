双冲头压铸，又称精速密压铸、套筒双冲头压铸法，是1966年由通用动力公司发明的一种特殊压铸工艺[[S4,S2,S25]]。其核心特征在于压射冲头由同轴嵌套的筒形外压射冲头和中心柱状内压射冲头组成，通过先协同运动完成低速充型、后独立补压的独特动作逻辑，实现低湍流充填与后续大行程补缩压实[[S4,S2,S25,S24]]。在工业界，该技术的NADCA收录官方注册名称为Acurad Process，相关英文描述包括dual plunger die casting、two-stage plunger injection等[[S4]]。

需要强调的是，双冲头压铸是专用压射硬件改造与专属压铸工艺深度绑定的复合技术，无法脱离专用双冲头压射机构而单独实现其工艺效果[[S30,S3]]。

## 工作原理

双冲头压铸的完整工作过程分为三个明确的速度阶段，其逻辑兼顾了平稳充型和强力补缩两个目标[[S8,S24,S28,S5,S27]]。

| 阶段 | 动作描述 | 速度特征与控制要求 |
|---|---|---|
| 第一阶段（第1速） | 内外冲头同步低速推进，将压室内金属液平稳推送至内浇口附近。 | 速度为普通压铸的1/10或更低[[S2,S20]]。 |
| 第二阶段（第2速） | 内外冲头同步小幅提速，推动金属液以层流状态平稳充填型腔，避免涡流卷气。 | 充型速度低，以层流充填为目标[[S5,S3]]。 |
| 第三阶段（第3速） | 充型完成后，外冲头受凝固壳层阻挡而停止，内冲头独立向前推进，穿透余料表面凝固壳，将未凝固金属液压入铸件进行补充压实。 | 补压启动需经延时控制，内冲头单独动作；开模前内冲头先退回，随后外冲头随开模将余料饼推出[[S8,S24,S28,S5]]。 |

在上述过程中，从充型结束到内冲头补压启动之间存在一个关键的时间延迟，该延迟由时间继电器控制。其目的是等待型腔壁面、浇口系统及余料表面形成密封凝固壳层，以确保后续补压压力能有效传递至铸件内部。延迟时长取决于合金总容积，若操作不当将导致补压效果完全失效[[S28,S31]]。

## 设备结构与组成

### 压射单元核心结构

双冲头压铸的压射单元采用内外嵌套式冲头布置：外层为筒形外压射冲头，与普通单冲头结构相似；内层为中心柱状内压射冲头，专门用于充填结束后穿透余料中心进行补压。两个冲头均可独立运动[[S15,S20,S25]]。

驱动方式上，外冲头由独立外液压缸驱动，内冲头由独立内液压缸驱动。二者能协同执行初始压射动作，充型完成后内冲头可单独按设定延时启动完成补压动作，完全支持独立控制[[S20,S15]]。

此外，内外冲头均配备独立的水冷通道和润滑路径：外冲头冷却水通过外压射杆细长孔进入冲头内部螺旋槽完成循环；内冲头冷却水通过内压射杆内置独立水管实现冷却；外冲头通过压室四周输送润滑介质，内冲头则经外压射杆内部细长润滑孔完成润滑。两个冲头的运动位置由对应行程开关精准反馈[[S20]]。

### 配合精度与适配机台

压射机构的装配精度直接影响工艺稳定性。外压射冲头与压室的直径配合间隙范围为0.08~0.13mm（每边间隙为总直径间隙的1/2），冲头偏摆度不得大于最大间隙值的1/2，以避免金属液渗入间隙造成卡滞故障[[S33]]。

双冲头压铸通常仅适配锁模力4000~6300kN（400~630t）的大型压铸机，不适用于小型压铸机。在普通压铸机上使用时需对原有压射机构进行改造[[S3,S2,S38,S22]]。

## 工艺参数与控制要求

### 核心量化参数

双冲头压铸拥有与普通单冲头压铸显著不同的工艺参数体系，其核心特征是大内浇口、低充型速度、高补压比压以及严格的冷却控制。

| 参数项 | 典型值或范围 | 说明 |
|---|---|---|
| 慢压射冲头速度 | 76~300 mm/s | 约为普通压铸的1/10或更低[[S20,S2,S3]]。 |
| 内浇口金属液流速 | 2~6 m/s | 仅为普通压铸充填速度的20%左右，以实现平稳层流充填[[S3,S31,S13]]。 |
| 充型总时长 | 0.3~1.0 s | - [[S20,S31]] |
| 内冲头补压比压 | 35~100 MPa（350~1000 kgf/cm²） | - [[S3,S31,S23,S20]] |
| 内冲头补进行程 | 50~150 mm | - [[S3,S31,S23,S20]] |
| 内浇口厚度 | 3~15 mm（普通压铸内浇口的约10倍） | 通常与铸件壁厚相当，以保证压力有效传递[[S3,S31,S23,S13]]。 |

### 定向顺序冷却控制

双冲头压铸要求配套采用定向顺序冷却策略：冷却必须从金属液最后充填的型腔远端开始，越靠近内浇口冷却越慢，保证铸件由远及近向内浇口方向顺序凝固。只有实现这种定向顺序凝固，厚内浇口、低速充型和内冲头补压的综合作用才能充分发挥[[S3,S22,S5,S1]]。

### 不同材料的参数差异

对于不同合金，工艺参数需作相应调整。铝合金的内冲头比压以800 kgf/cm²为有效上限，超过该阈值后力学性能不再随压力显著提升[[S10]]。镁合金因热容量更小，为保证补压阶段余料与铸件内仍存在可流动的金属液，压射充型速度需提升至5~8 m/s，压射总驱动力需比铝合金压铸高出约15%[[S5,S39]]。

## 与单冲头压铸的对比

双冲头压铸在充型原理、工艺参数和铸件质量等方面与普通单冲头压铸存在系统性的差异。

| 对比维度 | 双冲头压铸 | 普通单冲头压铸 |
|---|---|---|
| 压射冲头结构 | 内外嵌套双冲头，可独立动作 | 单一冲头 |
| 充型速度 | 极低（2~6 m/s，约为普通1/100） | 较高（10~30 m/s）[[S13]] |
| 内浇口特征 | 厚大（3~15mm），截面积约10倍 | 薄小 |
| 补缩机制 | 充型后内冲头独立大行程补压 | 依赖增压阶段 |
| 铸件强度 | 较普通压铸提升20%以上[[S3,S38,S2]] | - |
| 铸件致密度 | 提升3%~5%[[S3,S38,S2]] | - |
| 内部缺陷 | 气孔、疏松大幅减少，可热处理与焊接[[S3,S38,S2,S21]] | 气孔、缩松较常见 |
| 适用壁厚 | ≥4~5mm的厚壁件[[S2]] | 薄壁及常规壁厚 |
| 设备与成本 | 需专用双冲头机构，成本高，调试复杂 | 设备简单，成本较低 |

该技术还可采用一种衍生形式——分布式局部补压型：除主压射冲头外，在压铸模具厚大部位额外设置独立补充压射冲头，专门针对局部厚壁区域进行压实，以获得更高致密度[[S3,S35]]。

## 主要优势与局限性

### 技术优势

双冲头压铸在消除缺陷和提升性能方面具有显著优势。它可减少甚至完全消除铸件内部皮下气孔，大幅降低废品率；铸件致密性、力学强度和内部质量均匀性显著提升，能够进行后续热处理与焊接加工。同时，低速充型与更低的喷射倾向有利于延长压铸模具和压铸机的整体使用寿命，并可扩展压铸可适配的材料种类[[S31,S25,S23,S29]]。

### 局限性

该工艺的局限性同样明显。增设二次压射机构导致设备采购或改造成本显著高于常规压铸机；双冲头压射机构的动作时序调试难度大，调试周期长；厚大内浇口必须使用专用切除设备去除浇口和余料，工序复杂度提升，且材料利用率低于常规压铸[[S31,S25,S2,S23,S29]]。此外，该工艺仅适用于壁厚4~5 mm以上的厚壁铸件，无法用于薄壁件生产[[S2,S13]]。

## 常见缺陷及分析

双冲头压铸的核心风险集中在两个冲头的动作不同步以及补压时机控制失当。

- **同步速度偏差**：双冲头同步动作阶段的速度偏差超过允许范围，会造成压室内金属液提前发生剧烈喷溅和涡流卷气，导致铸件表层及内部异常气孔率显著升高[[S18,S29]]。
- **补压启动时机错误**：若外冲头停止后内冲头补压启动过早，铸件外壳尚未完全凝固，会将型腔中尚未完全成型的金属液挤穿产生穿透性毛刺，甚至从分型面喷出；若补压启动过晚，型腔中部分金属液已完全凝固，补压压力无法有效传递至热节部位，铸件厚壁处会产生难以消除的局部集中缩孔，导致气密性不达标[[S1,S34]]。
- **补压速度不匹配**：补压阶段内冲头移动速度若与型腔金属凝固收缩速率不匹配，会造成冲头卡滞在压室内，出现回锤失败故障[[S7]]。

针对上述缺陷的防控方案主要包括：提前精准匹配双冲头各阶段动作时序；调试液压系统控制阀确保两个冲头驱动液压回路的压力、流量参数一致；设置基于铸件外壳凝固的延时触发联动控制机制；定期维护双冲头滑配部位的润滑、冷却与同轴度精度，避免金属液渗入间隙造成卡滞粘咬[[S1,S7]]。

## 分类方式

根据不同的维度，双冲头压铸可按如下方式分类。

**按驱动形式分类**：全液压双缸独立驱动型——内外冲头分别配置独立液压油缸，可独立控制行程和输出压力；机械联动触发型——由同一主液压缸提供动力，通过机械限位或离合机构触发内冲头独立补压动作[[S35]]。

**按联动动作方式分类**：同步同进后独立补压型——充型阶段内外冲头保持同步运动，充型完成后外冲头停止，仅内冲头继续推进完成整体铸件补缩；分布式局部补压型——除主压射冲头外，在模具厚大部位另设独立补充压射冲头，针对局部厚壁区域单独压实[[S35,S3]]。

## 典型应用场景

双冲头压铸适合生产壁厚4~5 mm以上、对致密性和气密性有严格要求的高性能厚壁压铸件及承载结构件，典型代表如气缸体[[S5]]。不适用于壁厚小于3.5 mm的薄壁铸件，薄壁件在此工艺下易出现欠铸缺陷[[S13,S22]]。

从行业分布看，国内压铸件最主要的应用领域为汽车及摩托车行业，占总应用市场的48%。其余主要领域包括日用品、机械产品及零部件、家电及电子产品等[[S16]]。这些领域中的大型、厚壁、高强度要求的结构件是双冲头压铸的潜在应用对象。
