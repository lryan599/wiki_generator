## 概述

在压铸工艺语境中，“送风压力”指为实现型腔清洁、残留介质吹干、冷却通道清理及气动元件驱动等目的所使用的压缩空气工作压力，属于辅助系统参数[[S13,S14,S12]]。该参数在中文压铸行业培训资料、技术手册中常与“压铸吹气压力”混用，对应国际通用英文术语包括 air blow pressure、blowing pressure 及 air supply pressure，三者均广泛见于行业文献，尚无统一强制性标准指定唯一译名[[S13,S14]]。需特别注意，在通用铸造标准体系中，“送风压力”一词传统上指代冲天炉熔炼时克服炉料阻力向炉膛鼓风的工作压力，该含义不属压铸领域范畴，两者应予区分[[S6,S17,S16]]。

## 工艺作用

送风压力在压铸完整生产循环中承担多项功能[[S8,S12,S1,S11,S13]]：

1. **型腔清洁**：开模取件后、喷涂脱模剂前，以压缩空气吹扫型腔表面附着的铝屑、残渣、飞边及分型面异物，保证型腔清洁度，避免合模时压伤模具或堵塞排气通道。
2. **脱模剂吹干**：脱模剂喷涂完成后，对型腔表面通入压缩空气，吹干残留的脱模剂混合液及水分，防止液滴残留接触高温金属液瞬间汽化引发铸件气孔缺陷。
3. **冷却通道吹扫**：模具冷却通道通水冷却结束后，通入压缩空气清除通道内残余冷却水，防止水路残留水影响冷却效率或渗入型腔。
4. **气流负压压铸**：在气流负压（低真空）压铸工艺中，压室吹气口与排气系统吹气口同时通入压缩空气，使型腔排气侧形成低负压环境，减少充型过程卷入的气体量。
5. **气动执行驱动**：驱动压铸机周边喷涂装置、取件机械手等气动执行机构完成指定动作。

## 典型参数范围

常规压铸生产场景下，送风压力的通用操作范围为 **0.4～0.6 MPa（4～6 bar）**，该数值被国内主流压铸技术培训资料及行业现场操作规范广泛采用[[S14,S13]]。对应的气源压力一般设定为 **5～6 kg/cm²**（约0.5～0.6 MPa），可覆盖绝大多数中小型压铸机的常规生产需求[[S14]]。

具体至不同工序场景，推荐值可进一步细分[[S12,S3,S11]]：

| 工序场景 | 推荐送风压力范围 |
|---|---|
| 型腔表面铝屑残渣清洁 | 0.5～0.6 MPa（5～6 bar） |
| 脱模剂喷涂后吹干 | 0.4～0.5 MPa（4～5 bar） |
| 模具深冷却通道吹扫残余水 | 0.4～0.6 MPa（4～6 bar） |
| 压室内部清理吹气 | 0.4～0.5 MPa（4～5 bar） |
| 气流负压压铸工艺用气 | 0.4～0.6 MPa（4～6 bar）[[S13]] |

按压铸机类型区分，热室压铸机因生产的铸件普遍体积较小、型腔结构相对简单，送风压力推荐整体取值区间可略低于冷室压铸机，常规取 **0.4～0.5 MPa（4～5 bar）** 即可满足绝大多数应用场景[[S12,S11]]。

## 参数关联与影响

### 与相关工艺参数的关联

送风压力的取值与多项压铸工艺参数存在联动关系[[S12,S5,S8]]：

- **吹气持续时间**：呈负相关趋势。压力偏低时需延长吹气时长方能实现同等吹干、清洁效果；压力充足时可在较短时间内达到吹扫目的。
- **喷嘴配置**：与吹气喷嘴的排布密度及喷嘴孔径直接相关。对于复杂型腔深腔、窄缝等喷嘴覆盖不足的位置，需适当提高送风压力以保证气流穿透力和清扫效果。
- **模具温度**：模温过低时（低于150℃），残留在型腔表面的水分挥发难度显著提升，需匹配更高送风压力以加快介质带出速度[[S5]]。

### 压力不当导致的缺陷

送风压力偏离合理区间会直接诱发各类铸件质量缺陷[[S15,S1,S2]]：

- **压力偏低（低于 0.3 MPa）**：型腔表面残留的脱模剂混合液、抽芯滑块槽内残留水分无法被完全吹净。合模后残留介质接触高温金属液瞬间汽化，体积膨胀逾千倍，直接导致铸件内部形成气孔缺陷；脱模剂局部聚集还会造成铸件对应位置表面粘模、积碳[[S15]]。
- **压力偏高（高于 0.7 MPa）**：高压高速气流会将型腔表面刚成型的薄脱模剂涂层吹破，破坏正常的润滑隔离层，同时高速气流裹挟杂质撞击型腔表面也会加速模具表面局部积碳与磨损[[S2]]（另见通用压缩空气管路管理要求中对压缩空气含水含油的控制主张 [[S2]]）。

## 设备与控制系统

压铸送风压力的供给与控制依赖以下核心硬件单元[[S12,S3,S4]]：

| 硬件组件 | 功能说明 |
|---|---|
| 空气压缩机 | 气源动力单元，提供高压压缩空气，常用工作压力为0.7 MPa级 |
| 压缩空气管路系统 | 从气源向各用气工位输送压缩空气 |
| 油水分离器/空气过滤器 | 去除压缩空气中的油分、水分及固体杂质，保证气源清洁度，防止水分随气流进入型腔引发铸件气孔 |
| 精密调压阀 | 精确调节并稳定输出送风压力至工艺所需设定值 |
| 吹气压力表 | 实时显示当前供气压力数值，供操作与巡检人员监控 |
| 吹气喷嘴 | 按指定排布与角度向型腔目标位置喷射气流，实现清洁与吹干作业；铜管喷嘴可根据型腔表面形状弯成不同喷吹方向，并随模具更换而整体拆换 |

在自动喷涂系统中，吹气压力表、空气过滤器及喷嘴等组件集成于脱模剂自动喷涂装置中，经压缩空气管路统一供气，由压铸机计算机控制吹气与喷涂的动作时序和持续时间[[S12,S3]]。

## 应用注意要点

1. **压缩空气质量**：压缩空气中含油含水是导致薄壁压铸件质量劣化的重要外部因素，必须对压缩空气管路中的油水捕集器进行经常性检查和排放（尤其在雨天），保持压缩空气干燥[[S2]]。
2. **吹气位置与时间**：喷涂完毕后若吹气不彻底，型腔或抽芯滑块槽内有脱模剂混合液残留，插芯时会被带入型腔，需查明原因并调整吹气位置、吹气时间，以及改善模具局部排水[[S15]]。
3. **压力与模具温度匹配**：低温模具条件下需关注残留水分挥发难度，必要时适度提高送风压力以保证吹干效果[[S5,S8]]。
