## 概述

移动式压注模（移动式传递模）是压注成型工艺中应用最为广泛的一类模具，属于**机外装卸式压注模**。其核心特征是模具的上下模部分均不与液压机滑块或工作台面固定连接，除加热和加压工序外，加料、清理型腔、安放嵌件等所有辅助操作均在压机工作空间之外完成 [[S1,S28,S9]]。在国内压注成型场景中，移动式压注模占绝大多数，可将辅助操作转移到压机外，显著提高了操作灵活性 [[S1]]。

移动式压注模依靠普通液压机即可成型，对设备无特殊要求，适配于小批量试制、中小型塑件生产以及多嵌件复杂制品的成型 [[S25,S29]]。其加料室与模具本体完全可分离，是该类模具区别于固定式压注模最显著的结构特征之一 [[S5,S14]]。

## 分类与定位

压注模的分类体系包含多个维度，移动式压注模在其中占据重要位置 [[S1,S2,S5,S18,S25]]：

| 分类维度 | 主要类型 | 移动式压注模所属类别 |
|----------|---------|---------------------|
| **按压机适配类型** | 普通液压机用压注模、专用液压机用压注模 | 普通液压机用压注模 |
| **按固定方式** | 移动式、固定式、半固定式压注模 | **移动式** |
| **按型腔数量** | 单型腔、多型腔压注模 | 两者均有 |
| **按分型面特征** | 单水平分型面、双水平分型面、带垂直分型面压注模 | 三者均有 |
| **按加料腔结构** | 罐式压注模、柱塞式压注模、活板式、楔形式 | 主要为**移动式罐式压注模** |

其中，移动式罐式压注模是目前国内最为广泛使用的传递模类型，对设备无特殊要求，可在普通压机上进行模塑成形，模内设有主流道、分流道和浇口，加料室与模具本体可完全分离 [[S25,S4]]。

## 与固定式压注模的核心差异

移动式与固定式压注模在结构设计、操作方式和适用场景上存在显著区别 [[S1,S14,S28,S9,S29]]：

| 对比维度 | 移动式压注模 | 固定式压注模 |
|----------|-------------|-------------|
| **安装固定方式** | 无任何部件与压机固定连接 | 上下模分别与压机滑块、工作台固连 |
| **辅助操作位置** | 所有非热压工序在机外完成 | 全流程在压机工作空间内完成 |
| **脱模方式** | 手工或通用卸模架完成 | 内置推出机构自动完成 |
| **适配生产规模** | 小批量试制、小尺寸塑件生产 | 大批量、大中型塑件生产 |
| **劳动强度** | 较高 | 较低 |
| **模具结构** | 简单，制造周期短，造价低 | 复杂，造价高 |
| **使用寿命** | 工作部位易磨损，寿命较短 | 寿命较长 |

## 结构组成

移动式压注模的完整结构包含以下核心组成部分 [[S14,S5,S16,S26,S31,S2]]：

- **加料腔**：与模具本体完全可分离，用于盛装成型物料并对其预加热，是完成压注动作的承载部件。加料室截面通常为圆形，底部可设计40°~45°斜角台阶以确保施压时与模板贴紧、防止溢料 [[S24]]。
- **压料柱（柱塞）**：独立活动部件，不与压机连接，将压机压力传递给加料腔内的熔融物料，推动物料经浇注系统注入型腔。
- **型腔**：由凹模、凸模、型芯组成，直接接触塑件，完成塑件外形和内部结构的成型。
- **浇注系统**：包含主流道、分流道、浇口，作为熔融物料从加料腔进入型腔的导流通道。多型腔模具需保证各型腔进料均衡。
- **导向机构**：由导柱、导套（或导向孔）组成，完成上下模开合模时的精准定位导向，同时保证压料柱与加料腔的同轴度。
- **脱模机构**：适配通用卸模架的非内置结构，可通过手动或外置机构完成塑件推出，包含推杆、复位杆等部件。
- **加热系统**：无内置独立加热元件，完全依靠普通液压机上下工作台的加热板完成模具、加料腔、压料柱的预热，适配整体取放机外操作的需求。

## 工作过程

移动式压注模的完整操作流程如下，核心特点是所有工序除热压环节外均在压机外完成 [[S5,S8,S31,S28]]：

1. **合模**：在压机外手动将上下模沿导向机构闭合，安放好嵌件。
2. **加料**：将独立的可分离加料腔放置在模具上方，称量后的成型物料加入加料腔内，装入活动压料柱。
3. **预热**：将装配完成的整体模具推入普通液压机的上下加热板之间预热，使物料升温至熔融态。
4. **压注**：压机上工作台下行推动压料柱，将熔融物料经浇注系统高速挤入完全闭合的型腔内，同时由压料柱施加的压力完成锁模动作。
5. **固化**：物料在闭合型腔内保压、受热完成交联固化，得到成型塑件。
6. **出模**：压机上行完成开模动作，将整套模具从压机工作空间移出，手工取下加料腔和压料柱，使用配套卸模架完成分型，取出塑件并清理型腔及浇注系统残料。

## 设计参数与工艺条件

### 加料腔尺寸设计

加料腔尺寸是移动式压注模设计的核心参数 [[S12,S22,S18,S24]]：

| 设计参数 | 计算公式/准则 | 说明 |
|----------|-------------|------|
| **加料腔横截面积 A** | A = (1.1~1.25) × S | S 为所有型腔与浇注系统在分型面上的投影面积之和 |
| **加料腔高度 H** | H = V/F + 1.2 (cm) | V 为加料腔容积，F 为加料腔截面积 |
| **底部斜角台阶** | 40°~45° | 施压时让加料室与模板贴紧，防止溢料 |
| **定位方式** | 平面贴合 / 定位销 / 挡销 / 3~5mm凸台配合 | 四种可选定位方式 |

### 压注单位压力

不同适配材料对应的压注单位压力区间如下 [[S16,S22,S30,S10]]：

| 材料类型 | 压注单位压力 (MPa) |
|----------|-------------------|
| 酚醛塑料 | 50~80 |
| 纤维增强塑料 | 80~120 |
| 环氧树脂等低压封装塑料 | 2~10 |
| 橡胶压注成型 | 约 6.9 (1000 psi) |

### 锁模力要求

普通液压机配套的移动式压注模依靠加料室底部的压注力直接传递实现锁模，总锁模力必须大于型腔内物料压力产生的分型面胀开力 [[S21,S18,S4]]：

**F锁 = (p₁ × A) / 1000**  (单位：kN)

其中 p₁ 为实际单位挤压力（MPa），A 为加料腔横截面积。需保证成型过程分型面不溢料。

### 工作温度与成型周期

| 工艺条件 | 参数范围 | 说明 |
|----------|---------|------|
| 常规工作温度 | 130~190 °C | 热固性塑料压注成型 [[S22]] |
| 橡胶模具加热温度 | 约 350 ℉（约177 °C） | 橡胶压注成型 [[S30]] |
| 充模时间 | 10~30 s | 物料需在该时段内快速充模完成 [[S22,S7]] |
| 橡胶固化周期 | 10~15 min | 常规尺寸胶料 [[S30]] |

## 优缺点

### 优点 [[S28,S29,S25]]

- 整体结构设计简单，制造周期短，造价远低于固定式压注模。
- 操作灵活，可快速在同一台普通压机上更换不同模具适配不同产品生产，设备利用率灵活度高。
- 机外操作空间充足，可高效安放多数量、结构复杂的嵌件，适配多嵌件异形塑件、新产品试制场景。
- 对设备无特殊要求，可直接适配任意型号普通液压机，无需采购带双液压缸的专用压注成型设备，设备投入成本低。

### 缺点 [[S28,S29,S1,S9]]

- 人工操作占比高，劳动强度大，限制模具整体重量，仅可用于小尺寸塑件生产，无法适配大中型塑件成型。
- 频繁机外转移取放过程中模具容易自然冷却，成型过程温度波动大，对塑件固化一致性存在负面影响。
- 工作部件频繁人工拆卸操作易出现磨损，整体使用寿命短。
- 压机停机等待辅助操作的时间占比高，生产效率低于固定式压注模，不适合大批量规模化生产。

## 适用材料与应用场景

移动式压注模适配的典型成型材料包括酚醛、三聚氰胺甲醛、环氧树脂等热固性塑料，也可用于各类机械橡胶制品的成型 [[S7,S19,S13,S30]]。不饱和聚酯和脲醛则由于低温下硬化速度过快，通常不适用于成型较大塑件 [[S7]]。

该类模具适用于中小批量热固性塑件、带多嵌件的复杂精密制品生产。代表性工业制品包括 [[S16,S23,S1]]：

- 绝缘酚醛塑料骨架
- 电器插座盖板
- 线圈骨架
- 电子元件低压封装件
- 精密机械橡胶密封件

广泛应用于电子、电器领域 [[S16,S22]]。
