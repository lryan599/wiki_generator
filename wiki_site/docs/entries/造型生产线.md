## 在压铸知识库中的定位与边界

根据对中英文压铸技术文献的系统检索，**造型生产线（molding line / moulding line）本质上属于砂型铸造（sand casting）工艺范畴的专用自动化生产系统**，在压力铸造（die casting，特别是高压压铸 HPDC）领域并无对等概念；压铸文献中不存在将“造型生产线”作为压铸主体工艺单元进行描述的证据[[S1]]。

因此，在本知识库中，“造型生产线”应被理解为一个**对比参考概念**，用于帮助技术人员区分砂型铸造与压铸在生产流程、装备体系上的根本差异。若检索压铸相关的“die casting production line”或“die casting molding line”，文献实际指向的是以压铸机（die casting machine）为核心的自动化压铸单元（die casting cell）及其周边辅助设备，其工艺原理与砂型造型线的“造型-合箱-浇注-落砂”循环截然不同[[S1]]。

### 规范英译说明

- **Molding line / Moulding line**：指砂型铸造中，从型砂制备、造型、合箱、浇注、冷却到落砂的完整自动化生产线[[S1]]。
- **Die casting line / Die casting cell**：指以高压压铸机为主体，集成给汤、喷涂、取件、切边等周边设备的自动化压铸生产单元[[S1]]。
- 两者在装备结构、控制对象、模具/铸型材料上存在根本差异，不可混用。

## 造型生产线的定义与工艺作用（砂型铸造范畴）

该部分内容仅作为对比参考，以明确造型生产线在铸造技术体系中的一般含义，所述特征均不适用于压铸工艺。

造型生产线是指将造型机与各种辅机（如翻箱机、合箱机、落箱机、压铁机、分箱机等）按既定工艺流程用输送装置（如铸型输送机、辊道等）连接起来，组成一套机械化或自动化的砂型铸造生产系统。其基本功能是完成从型砂填入砂箱、紧实成型、起模、下芯、合箱、浇注金属液、冷却凝固、落砂直至旧砂回用处理的连续循环作业[[S1]]。

其核心工艺特征包括：
- 使用可循环再生的型砂（多为湿型黏土砂）作为造型材料。
- 必须包含砂处理系统，以完成旧砂冷却、磁选、破碎、混砂等再生工序。
- 铸型为一次性或半永久性砂型，浇注后需经落砂工序使铸件与型砂分离。
- 整个生产线是物流与工艺高度耦合的连续型流水线。

以上特征与压铸工艺中使用的**永久金属模具、高速高压充型、无砂处理系统**等属性形成本质区别[[S1]]。

## 分类与常见形式

造型生产线的分类主要依据造型机类型、分型方式及砂箱使用情况，以下分类均基于砂型铸造文献[[S2]]，在压铸技术中无对应类别。

### 按分型方向分类

| 类型 | 特征 | 典型生产率（型/小时） |
|------|------|-----------------------|
| 垂直分型造型线 | 采用垂直分型无箱造型工艺，铸型在垂直方向分型，造型与合箱在同一工位完成，无需砂箱。代表设备为迪砂（DISA）造型线。 | 最高可达 500 型/小时以上（无箱） |
| 水平分型造型线 | 上下砂箱水平分型，造型后需独立完成合箱、压铁、浇注等工序。应用最广泛的传统布局形式。 | 视砂箱尺寸及自动化程度，通常 60~240 型/小时 |

### 按紧实方式分类

- **静压造型线**：通过压缩空气瞬间释放或气流预紧实加液压压实，铸型硬度均匀，适于复杂铸件[[S2]]。
- **射压造型线**：利用压缩空气将型砂射入砂箱并辅以机械压实，生产率高，常见于无箱造型线[[S2]]。
- **震压造型线**：传统紧实方式，噪声及磨损较大，现多被静压或射压取代。

上述所有分类均围绕砂型铸造展开，未见与压铸模具制造或压铸生产直接相关的“配套造型线”定义[[S1]]。

## 核心构成与设备组成

一条完整的造型生产线通常由以下核心子系统构成[[S2]]：

1. **砂处理系统**——包括旧砂筛分、磁选、冷却、混砂、松砂及型砂输送装置，确保型砂性能稳定。
2. **造型机**——实现型砂紧实与起模的核心设备，决定铸型质量与生产率。
3. **翻箱机与合箱机**——用于水平分型线的砂箱翻转、下芯后的上下箱合箱。
4. **浇注机**——自动或半自动浇注金属液。
5. **铸型输送装置**——如铸型输送小车、辊道或链条输送机，完成各工位间的转运。
6. **冷却段**——保证铸件在落砂前充分凝固冷却。
7. **落砂机**——使铸件与型砂分离，型砂进入回用系统。
8. **除尘与环保设备**——控制粉尘及烟气排放。

需要再次强调，以上设备清单完全针对砂型铸造工艺链。压铸生产线则以**压铸机、保温炉、给汤机/自动浇注装置、喷涂机器人、取件机器人、切边机、模温机、真空系统**等为核心构成，两者在设备清单上无重叠项[[S1]]。

## 关键参数与典型范围

以下参数源自砂型铸造造型生产线技术规范，仅列出以建立参考基准[[S3]]。**这些参数在压铸模具制造或压铸生产参数体系中均无对应项**。

| 参数项 | 说明与典型范围 |
|--------|----------------|
| 生产率 | 通常以“完整型/小时”或“半型/小时”表示；中小型无箱线 200~555 型/小时；大型有箱线 30~120 型/小时 |
| 砂箱内尺寸 | 根据铸件大小，从 400mm×500mm 至 3000mm×2000mm 不等 |
| 压实比压 | 静压及射压造型线 0.7~1.2 MPa（高压造型可达 1.5~2.5 MPa） |

| 单循环时间 | 包含造型、合箱、浇注等，由节拍最慢工位决定 |
| 自动化程度 | 从人工下芯的半自动线到全线无人干预的全自动线 |
| 型砂紧实率 | 控制目标 35%~42%（湿态） |
| 旧砂温度 | 要求冷却后 ≤ 50°C（高于室温 10~15°C） |

## 与压铸相关工艺的差异与边界

造型生产线与压力铸造（高压压铸、低压铸造）、重力铸造（金属型重力铸造）在工艺原理、模具/铸型材料、生产组织方式上的根本差异，应列为该条目的核心关系说明[[S1]]。

| 对比维度 | 造型生产线（砂型铸造） | 压铸生产线（以 HPDC 为主） |
|----------|------------------------|----------------------------|
| 铸型/模具 | 一次性砂型，可循环再生砂 | 永久金属模具（H13 类热作模具钢） |
| 充型驱动力 | 重力    | 高速、高压机械推动（数十至上百 MPa） |
| 砂处理系统 | 必备且规模庞大 | 无需砂处理 |
| 模具制造 | 仅需制作模样与芯盒 | 完整的模具设计、加工、热处理、表面处理流程 |
| 典型节拍 | 数十秒至数分钟/型 | 压铸循环时间：20~120 秒/模次 |
| 合金适应性 | 铁基、铜基、铝基等通用性强 | 以铝、锌、镁等低熔点合金为主 |
| 铸件精度 | CT8~CT12（取决造型方法） | CT4~CT6（高尺寸精度与薄壁能力） |
| 适用批量 | 单件小批至大规模连续生产均可 | 主要用于大批量、高节拍生产 |

可见，**造型生产线与压铸生产线属于铸造技术家族下的两个不同的工艺分支，不存在“压铸造型混合线”或“为压铸服务的砂型造型线”等标准概念**[[S1]]。在部分生产实践中，压铸件的原型或小批量试制可能采用砂型铸造（此时会用到造型生产线），但这是产品开发路径问题，而非工艺单元的集成或混合。

## 相关标准与安全要求

以下标准均应被理解为适用于砂型铸造造型生产线的标准，**不直接适用于压铸机或压铸单元**，此处列出仅作关联参考[[S4]]。

- **GB/T 28688-2012** 《铸造机械 术语 第1部分：基本术语》——明确了造型机、造型生产线等术语定义。
- **GB/T 31561-2015** 《铸造机械 安全要求》——涉及铸造生产线（含造型生产线）的一般安全设计要求。
- **GB 20905-2007** 《铸造机械 安全要求》——涉及铸造车间造型、制芯、清理等设备的安全防护。
- **ISO 23062:2022** Foundry machinery — Safety requirements for molding and coremaking machinery and associated equipment —— 国际范围内适用的造型及制芯设备安全标准。
- 另有各主要造型线制造商（如 DISA、KW、新东工业等）提供的设备技术规范与企业标准。

安全设计要点通常涵盖：运动部件联锁防护、自动线急停系统分区控制、粉尘防爆、高温金属液喷溅防护、噪声控制等[[S4]]。

## 结论

在压铸技术知识库中，“造型生产线”条目必须以**差异比较**而非**压铸组成部分**的方式进行定义。它代表了砂型铸造中成熟的自动化生产模式，但在工艺原理、设备构成、模具体系上均与压铸技术无直接从属或集成关系。研究检索确认，中英文文献均未支持“压铸造型生产线”或“die casting molding line”作为规范的工艺装备术语[[S1,S2,S3]]。因此，建议该条目在压铸知识库中保持为比较参考词条，不作为压铸本体知识子项展开。
