## 概述
电渣熔铸炉是实施电渣熔铸（electroslag casting，ESC）工艺的专用设备，在国家标准GB/T 25370-2020《铸造机械 术语》中被正式归类为特种铸造设备 [[S7]]。该设备利用电流通过液态渣池产生的电阻热熔化自耗电极，金属熔滴经熔渣精炼后直接在水冷结晶器内凝固成形，实现金属精炼与铸造成形一体化 [[S40,S10]]。其冶金过程渣池温度可达1900℃，在强制冷却条件下可获得定向结晶、组织致密的高性能铸件 [[S10,S26]]。

电渣熔铸炉的典型产品覆盖从几十克（如假牙、牙套）到几十吨（如船用柴油内燃机曲轴）的碳钢、合金钢、非铁合金、球墨铸铁及高强度铸铁铸件，可生产供压铸工序使用的高品质纯净合金锭坯（铝合金、镁合金、锌合金、工模具钢、高温合金等），也可直接制备近终形异形压铸受力件，如水轮机导叶、涡轮盘等关键部件 [[S40,S12]]。

## 定义与工作原理
根据GB/T 25370-2020，电渣熔铸炉是以自耗电极为原料，在熔渣电阻热作用下熔化金属，并使之在水冷异型结晶器内逐步凝固，直接获得最终成形铸件的特种铸造设备 [[S7]]。其核心工作回路如下：变压器输出的低电压大电流经自耗电极→渣池→金属熔池→铸锭→底水箱返回电网，熔渣的高电阻（一般控制在0.25~0.35 Ω·cm）是产生焦耳热的直接热源 [[S20,S8]]。

电渣熔铸炉与电渣重熔炉（electroslag remelting furnace，ESR）同源，均属于无电弧自耗电渣冶炼设备，核心发热原理相同，金属熔滴穿过渣层过程中被精炼，去除硫、磷及非金属夹杂物，后续在水冷结晶器强制冷却下实现定向结晶凝固 [[S8,S32,S4]]。两者的核心差异如表1所示。

| 对比维度 | 电渣熔铸炉（ESC） | 电渣重熔炉（ESR） |
|----------|-------------------|-------------------|
| 设计目标 | 直接一步成形最终异形铸件，省去后续大量切削加工 | 生产简单圆/方规整锭坯，为锻造、轧制提供原材料 |
| 结晶器特征 | 定制目标铸件的异形轮廓，紫铜水冷结构 | 通用标准圆柱形、正锥度结构 |
| 典型产品 | 水轮机导叶、整体轧辊、异形曲轴等复杂异型件 | 最大100t级的大型规整钢锭 |
| 工艺控制精度 | 依目标铸件体积适配，控制精度要求更高以匹配异形连续凝固成形 | 常规标准化参数控制，5t级典型供电为交流20kA、开路电压75V [[S31]] |

英文术语“electroslag melting and casting furnace”已有零星文献提及，但尚未完成术语规范化验证，正式名称为electroslag casting furnace（ESC） [[S10,S26,S4,S24]]。

## 设备结构与组成

电渣熔铸炉的完整结构分为供电系统、机械传动系统、熔铸系统三大组成部分，核心部件包括降压变压器/短网供电单元、自耗电极进给机构、水冷结晶器、可调控渣池、抽锭/引锭机构、底水箱冷却单元 [[S5,S36,S27]]。

### 2.1 传动机构系统
由伺服电机、减速机、滚珠丝杠、立柱导轨、电极夹头等组成 [[S18,S10]]。自耗电极进给机构有四种典型传动形式：丝杠螺母式、链条链轮式、钢丝绳滚筒式、液压缸驱动式。正常熔炼时电极送给速度为50~60 mm/min，非工作快升速度为2~4 m/min [[S36,S27]]。

### 2.2 控制系统
配备可编程自动化控制系统，搭配伺服控制器、PAC模块、断电抱闸结构，实现电极升降的稳定控制，确保在断电情况下滑轨不会因失电导致脱落 [[S18]]。

### 2.3 电源调控系统
输出低电压大电流维持渣池稳定发热，常规工作电压范围为30~80 V（中小型设备为25~60 V），工作电流范围为3000~10000 A，特定合金钢工况可选用6000~6500 A的匹配区间 [[S17,S14,S9]]。

### 2.4 熔铸系统

**结晶器**：电渣熔铸炉的核心成形部件，采用紫铜材质定制的异形水冷结晶器，工作内腔设计为上小下大的正锥度结构以避免铸锭包渣并方便脱模，结晶器总高度为其直径的2.5~4倍。冷却水从结晶器底部通入、上部流出，强化底部冷却以实现定向凝固，无结晶器二次直接水冷结构 [[S5]]。

**渣池**：深度通常取配套结晶器直径的1/3~1/2，熔渣工作电阻控制在0.25~0.35 Ω·cm [[S20]]。

**底水箱**：采用紫铜面板+铸铁外壳的通水导电结构，既承担底部强制冷却功能保障铸锭自下而上的定向结晶，又作为电流回路的导通部件 [[S5]]。

**抽锭/脱模机构**：分为两种工作模式——抽锭式可在熔铸过程中匀速从结晶器下方引出凝固后的铸锭，实现长尺寸铸锭连续生产；充填式需待整炉金属完全凝固后吊离结晶器脱模，更适配短尺寸异型铸件的生产 [[S5]]。

## 分类体系

电渣熔铸炉可按三个维度划分类型，如表2所示 [[S12,S36]]：

| 分类维度 | 类型 | 特征 |
|----------|------|------|
| 供电相数与电极配置 | 单相单极 | 最常用，结构简单，可用较大的填充比 |
| | 单相双极 | 电流从一电极经渣池流回另一电极，感抗小、电耗较低，适于生产扁锭 |
| | 三相三极 | 电网平衡，功率因素高，可用3个结晶器同时熔铸3个锭坯，适于异型坯件 |
| 工作气氛 | 敞开式非保护气氛 | 常规大气环境下工作 |
| | 保护气氛型 | 带密封充气结构，适配含铝、钛等易氧化元素的活性合金熔铸 |
| 工位配置 | 单工位 | 单一工位间歇生产 |
| | 多工位 | 双臂交替供电连续生产，适于较大批量场景 |

此外，有衬电渣炉结构以耐火材料坩埚替代常规水冷结晶器，可灵活调控整炉金属液的成分与温度，适配精密小型铸件如复合冷轧辊、曲轴等的小批量生产 [[S12]]。

## 工艺参数

### 4.1 核心电参数与适用范围
电渣熔铸炉核心电参数见表3。

| 参数项 | 典型范围 | 说明 |
|--------|----------|------|
| 工作电压 | 30~80 V（中小型25~60 V） | 电压随配套结晶器直径增大而提升；升高电压可提升熔渣温度、优化金属熔池形态，强化轴向定向结晶 [[S17,S14]] |
| 工作电流 | 3000~10000 A | 特定合金钢工况可选用6000~6500 A；电流过大将加深金属熔池破坏轴向结晶并引发漏液事故，电流过小易致铸件表面冷隔、内部夹杂 [[S17,S9]] |
| 冷却水压 | 0.15~0.2 MPa | 出水温度需低于45℃ [[S17]] |
| 熔渣电阻 | 0.25~0.35 Ω·cm | 依靠电流通过熔渣的电阻热实现自耗电极熔化与精炼 [[S20]] |

### 4.2 熔铸速度与电极进给
电渣熔铸配套自耗电极进给速度通常为0.2~0.6 mm/s，对应适配铝合金的最优熔铸速度为15 g/s，小批量异形件工况下熔铸速度适配区间为10~30 g/s [[S10,S11,S30]]。熔铸速度匹配不当会引发电极与金属熔池短路断电，或电极脱离渣池停止熔化，导致熔铸过程中断 [[S11]]。

### 4.3 渣系选择
电渣熔铸常用渣系如表4所示。

| 适用合金 | 代表性渣系成分 | 特性与要求 |
|----------|----------------|-------------|
| 大多数钢种 | 70%CaF₂ + 30%Al₂O₃（二元基础渣系） | CaF₂降低熔渣熔点、粘度及表面张力，提高电导率，利于吸附夹杂；Al₂O₃增加渣系电阻，提高渣温和熔化速度 [[S9,S20]] |
| 铜合金 | 80%CaF₂ + 20%NaF | 熔点1160~1180℃ [[S14]] |
| 铝合金 | NaCl-MgCl₂-冰晶石渣系 | NaCl对铝熔体进行变质处理，冰晶石吸附铝熔体内杂质；渣系熔点需低于目标金属约100℃；渣层高度控制20~30 mm [[S30]] |

所有渣系使用前需在500~600℃温度下烘烤4~6 h去除水分，避免铸件气孔缺陷 [[S3]]。

## 在压铸领域的角色与熔炼设备适用性

电渣熔铸炉在压铸行业有两类应用定位：一是生产供压铸工序使用的高品质纯净合金锭坯（铝合金、镁合金、锌合金、工模具钢、高温合金等）；二是直接制备近终形异形压铸受力件，如大型曲轴、涡轮盘、水轮机导叶等关键部件 [[S40,S26,S12]]。

表5给出了电渣熔铸炉与常规熔炼设备的适用性对比。

| 对比炉型 | 电渣熔铸炉优势 | 电渣熔铸炉劣势 | 适用场景判断 |
|----------|---------------|---------------|-------------|
| 感应熔炼炉 | 精炼除杂效果更优（可有效去除S、O、N等有害元素） | 熔炼效率更低、电耗更高、工序长、生产费用较高 | 适合对合金纯净度要求极高的小众批次生产；感应炉热效率高、熔化速度快，更适合压铸量产场景下的常规铝合金、锌合金大批量连续熔炼 [[S29,S5]] |
| 反射炉 | 提纯等级远高于反射炉，可深度脱硫、去除氧化物夹杂 | 不适合大排量普通再生合金生产 | 反射炉热效率仅15%~30%，对废杂铜等低品质原料适配性好但无深度精炼能力，电渣熔铸适用于要求严格的高附加值合金品种 [[S29,S2]] |
| 真空电弧炉 | 无需真空系统即可通过渣层实现有效除杂，设备更简单，对含铝、钛等活性金属的成型适配性优于常规真空电弧炉 | 去气效果较差，含铝、钛高活性合金成分管控难度大（保护气氛型可在一定程度上弥补） | 电渣熔铸对活性金属成型有优势，但真空电弧炉去气（尤其是氢）效果更好 [[S12,S13]] |

## 质量控制与典型缺陷

电渣熔铸炉生产的铸件气体含量可低至0.002%，夹杂物总量为0.004%~0.006%，铸件密度相比普通铸造提升0.33%~1.37%，铸态性能可达锻件水平，金属利用率提升40% [[S10]]。

表6列出了设备及工艺因素引发的典型缺陷与改进对策。

| 缺陷类型 | 设备/工艺因素 | 对策措施 |
|----------|--------------|----------|
| 局部缺陷集中 | 熔铸速率参数不匹配 | 精准控制熔铸速度在15 g/s区间，将缺陷引导至铸件非核心加工部位，避免影响构件服役性能 [[S26,S22]] |
| 成分偏析 | 熔池冷却不均匀 | 优化水冷结晶器的换热参数，保持定向凝固稳定，抑制宏观偏析 [[S26]] |
| 夹渣 | 渣料纯度不达标 | 严格管控熔渣成分，确保渣中SiO₂≤2.0%、FeO≤0.5%、S≤0.05%、P≤0.02% [[S40,S21]] |
| 晶粒粗大（超声探伤杂波） | 大尺寸电渣熔铸件结晶特征 | 采用低频率探伤、双斜探头检测等适配性检测方案降低检测误差 [[S21]] |

在合理熔速（15 g/s）控制下，电渣熔铸异形铸件的缺陷集中分布在异形上部非受力区域，可通过后续机加工去除，不影响服役性能 [[S22]]。

## 安全操作规范

电渣熔铸炉面临高温熔融渣泄漏、高压触电、水冷故障引发的渣爆三类典型风险，对应安全操作规范如表7所示 [[S1,S39]]。

| 风险类型 | 安全规范 |
|----------|----------|
| 高温熔融渣泄漏 | 作业前确认渣坑完全干燥无积水，倾倒热渣前铺设干沙，禁止渣道区域堆积潮湿物料 |
| 高压触电 | 高压电气区域独立设置封闭操作室，作业人员穿戴绝缘防护用具，设备周边敷设绝缘橡胶垫，停电检修时执行挂牌上锁制度 |
| 水冷故障引发渣爆 | 作业全过程保持冷却水水压不低于0.2 MPa，冷却水出口温度不得超过80℃，配备备用供水系统，突发停电时立即启动应急人工供水避免结晶器冷却水断流 |

### 日常维护
每日巡检水压表、水温表及输水胶管老化程度；每3~5天试运行应急备用供水泵；每月紧固各导电铜排连接螺栓；每年度对设备绝缘性能开展全量检测；定期清理结晶器内壁附着的渣层残留避免换热效率下降 [[S6]]。

## 技术发展动态

近年电渣熔铸领域的主要技术进展如下 [[S12,S26]]：

1. **加压电渣熔铸**：在熔铸舱内施加可控压力，进一步降低铸件内部微孔隙率，提升高负荷压铸模具钢坯的致密度。
2. **保护气氛电渣熔铸**：在熔铸舱内通入惰性保护气体，抑制铝合金、镁合金等活性压铸合金的氧化烧损与吸气，解决常规电渣熔铸对含铝、钛高活性合金成分管控难度大的痛点。
3. **智能化控制**：通过多物理场实时传感+AI算法动态调控熔速、渣池温度与冷却参数，配套CAFE数值模拟提前预判晶粒生长与缺陷分布，大幅降低人工操作偏差，提升铸件批次稳定性。

国内工业应用方面，电渣熔铸技术已实现Φ1000mm×2500mm大型钢锭的稳定量产，应用于三峡、白鹤滩等大型水电项目的关键构件制造，相比传统工艺金属利用率提升40%，制造周期缩短65% [[S10]]。
