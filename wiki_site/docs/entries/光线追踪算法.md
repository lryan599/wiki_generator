光线追踪算法（Ray Tracing Algorithm）在压铸领域的激光深熔焊模拟中，特指一种基于几何光学原理，通过追踪大量离散光线在匙孔（Keyhole）壁面的多重反射路径，以精确计算激光能量空间沉积分布的数值计算方法[[S1]]。与传统的预设体积热源模型（如Goldak双椭球热源）直接假设能量分布不同，该算法通过模拟光线与动态匙孔界面的实际物理交互，实现了能量分布与瞬态匙孔形貌的动态耦合[[S1]]。

### 算法流程与实现

光线追踪算法在压铸件激光焊接仿真中的核心作用，是替代或改进传统的体积热源模型，以更逼真地描述激光能量在匙孔内的沉积机制[[S1,S2]]。传统的Goldak双椭球热源等模型依赖于预设的形状参数，这些参数需要针对特定工艺通过大量试错进行校准，且无法反映匙孔内部的光学物理过程。光线追踪算法则直接从物理层面入手：入射激光束被离散化为大量光线，每条光线进入匙孔后，在由计算流体力学（CFD）和流体体积法（VOF）实时追踪的气/液界面（匙孔壁面）上发生多次反射和吸收[[S1]]。每次反射依据菲涅尔（Fresnel）吸收定律在壁面沉积一部分能量，剩余能量继续追踪，直至光线能量耗尽或逃出匙孔[[S1]]。

这一过程实现了能量沉积与匙孔形貌的双向动态耦合：通过光线追踪计算出的能量分布作为热源项输入CFD求解器，驱动熔池流动与匙孔界面的演变，而新的界面形态又作为下次光线追踪的几何边界条件[[S1]]。

在压铸铝合金（如ADC12, A380）的场景下，光线追踪对于准确预测匙孔形貌的改进至关重要。仅采用传统的体积热源模型往往会因忽略匙孔底部的多重反射效应而低估熔深，或因能量分布假设不当而高估熔宽。光线追踪算法通过考虑匙孔内壁的多次反射，使得能量能够被“引导”至匙孔深处，这解释了实验中观察到的深宽比极高的焊缝形貌[[S1,S2]]。特别是，它能捕捉到前壁（激光直接照射侧）强烈加热和后壁通过反射获得能量的物理机制，从而更精准地预测匙孔的深度、宽度、波动行为及稳定性[[S1]]。这种预测能力的提升直接减少了工艺研发中的试错成本，提高了构件修复和焊接的工艺—形貌映射准确性[[S1]]。

### 关键参数与影响因素

光线追踪模型的预测精度受多个计算和物理参数的影响[[S1]]：

| 参数类别 | 关键参数 | 对预测精度的主要影响 |
| :--- | :--- | :--- |
| **计算参数** | 光线数量、最大反射次数 | 光线数量越多，能量分布统计噪声越低，但计算成本线性增加；反射次数限制过小会低估能量吸收，导致熔深预测偏浅。 |
| **物理参数** | 激光波长、材料吸收率、布儒斯特角效应 | 波长与菲涅尔吸收系数直接相关，决定了每次反射的能量沉积份额；材料在不同温度、相态下的光学属性变化对耦合计算至关重要。 |
| **环境干扰** | 蒸气羽流、等离子体屏蔽 | 匙孔上方的高温金属蒸气和等离子体云会对入射激光产生散射、折射和吸收，削弱到达匙孔底部的能量，是导致算法对真实深熔焊预测产生偏差的关键因素，通常需要附加的子模型进行补偿。 |

### 耦合求解与软件实现

在主流的过程模拟框架中，光线追踪通常作为一个独立模块与CFD求解器显式耦合。每个时间步内的典型耦合循环如下：首先，VOF界面追踪算法根据当前流场更新匙孔界面的几何形状；接着，光线追踪模块基于此界面形状执行光线追踪计算，得到能量在界面上的空间分布；最后，该能量分布作为边界热源或体积热源项被传递回CFD求解器的能量方程，求解下一时间步的温度场和流场[[S1]]。这一架构在商业和开源软件中均有应用，例如 FLOW-3D 和 OpenFOAM 的某些定制求解器中已集成了此类多重反射光线追踪模块[[S1]]。

### 局限性与权衡

尽管物理保真度更高，光线追踪算法在实际应用中仍面临若干局限性[[S1]]：
- **物理层面**：算法基于几何光学假设，完全忽略了激光的衍射和干涉等波动光学效应，这在匙孔开口尺寸接近激光波长时会引入原理性误差。
- **计算成本**：模拟包含大量光线在动态变化界面上的多次反射，计算开销远大于预定义的体积热源模型。在进行大规模或长时程的工业级焊件仿真时，计算资源需求是重要的瓶颈。
- **实时应用**：高昂的计算成本使其目前主要限于离线的高精度仿真和工艺机理研究，难以直接应用于需要毫秒级响应的实时在线控制和自适应焊接系统中。
- **场景局限**：对于无法形成稳定匙孔的薄板热导焊或激光钎焊等场景，多重反射增强吸收的机理不复存在，光线追踪模型的必要性显著降低。
