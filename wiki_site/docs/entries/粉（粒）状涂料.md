## 概述

粉（粒）状涂料是压铸领域内近二十年发展起来的新型干式脱模润滑材料，属于无液态载体（无溶剂、无大量水分）的全固体分压铸辅助材料体系[[S30]]。它随着高真空压铸技术和高强韧压铸合金在汽车保安零件上的成功应用而得到开发，以满足传统水基和油基涂料无法满足的高真空度型腔环境要求[[S12,S14]]。

> **术语辨析**：本词条所论“粉（粒）状涂料”指压铸生产过程中直接用于模具型腔、压室与冲头表面的干式脱模/润滑涂料。它与铸件成型后用于表面防护与装饰的**粉末涂装材料**在作用对象、施加阶段与功能目标上完全不同，分属两个独立的工业材料应用体系[[S1,S33,S34]]。

## 定义与分类

粉（粒）状涂料是以空气为载体进行分散喷涂的全固体分新型压铸涂料，主要依靠固体基材自身的润滑特性以及在模具表面形成的气隙热阻效应协同发挥脱模作用[[S21]]。根据形态与使用方式，可分为两大子类[[S20,S24]]：

1. **粉状涂料**：以压缩空气送入压室及型腔，呈雾状均匀沉积，涂层覆盖整个型腔工作面，适配全型腔润滑需求。
2. **粒状涂料**：以离散颗粒形式送入压室，多数颗粒因其密度较小，在金属液注入时上浮并附着在压室壁形成润滑膜，多用于压室与压射冲头的专项润滑。

按树脂体系分类，粉末涂料可分为热固型和热塑型两大类别[[S8,S35]]。

## 工艺作用

在压铸过程中，粉（粒）状涂料承担以下核心功能[[S16,S20,S30]]：

| 功能维度 | 具体作用 |
|---------|---------|
| 脱模 | 在模具表面形成1~5μm厚均匀极薄涂层，降低铸件推出力，实测对AlSi9Cu3铝合金及AZ91HP镁合金的推出力仅为传统水基涂料的1/2 |
| 润滑 | 固体润滑组分（石墨、氮化硼等）提供高温滑移润滑，减少金属液对模面的冲刷 |
| 模具保护 | 隔绝高温金属液直接冲刷模面，减少粘模与焊合倾向；涂层内微气隙形成热阻，降低对模面的激冷作用 |
| 真空维持 | 无水分挥发，保障高真空压铸工况下型腔内的高真空度要求 |
| 热管理 | 涂料熔化热远低于水基涂料的气化热，避免模面骤冷骤热产生的交变张应力，减少热疲劳龟裂 |

## 组成与基材体系

当前工业应用中主流的压铸粉状/粒状涂料基材体系主要分为四类[[S35,S21,S10,S33]]：

| 基材类别 | 核心成分 | 主要特点 |
|---------|---------|---------|
| 石墨类 | 微细石墨粉，搭配少量蜡类助剂 | 耐高温性能优异，高温下可实现稳定滑移润滑 |
| 氮化硼类 | 氮化硼（白石墨）为主体 | 外观白色，不污染铸件表面，适配高外观要求场景；可根据合金与工艺条件分型号选配 |
| 金属皂类 | 硬脂酸锌、硬脂酸钙等 | 熔点适中，受热可在模具表面均匀铺展成润滑膜 |
| 氟化物类 | 聚四氟乙烯（PTFE）等低摩擦系数氟树脂粉末 | 隔离性能优异，模具表面残留极低 |

国内公开研制的压铸粉状涂料配方以石墨、滑石粉、聚乙烯蜡、氮化硼作为核心基础原料，搭配其他功能性助剂制备而成[[S21]]。

### 流动性参数

现有公开的压铸用润滑类粉状材料的典型流动性参数可达26.2~26.5 s/50g，该流动性数值可保障粉末在喷涂气流中均匀分散，实现大范围均匀涂覆[[S23]]。

## 应用方法与工艺参数

### 主流涂覆工艺

粉状压铸涂料的工业主流涂覆工艺分为三大类[[S12,S14,S28,S30]]：

1. **静电喷涂**：粉末颗粒带电后吸附在模面，压缩空气消耗量极低，无外泄粉尘。
2. **负压/真空吸送**：在封闭型腔环境内将粉末吸入实现均匀覆膜，完全无粉尘溢出。
3. **气动撒粉**：依托压缩空气将粉末输送至压室和型腔，形成雾状均匀沉积覆盖整个工作面。部分粒状涂料送入压室后可在金属液注入时上浮，在压室内壁形成润滑膜，专门适配压射冲头与压室的润滑场景。

### 工艺配套要求

采用粉状脱模剂需具备以下配套条件[[S9]]：
- 配备真空装置；
- 压室结构需改造以便于粉状涂料导入；
- 模具应附加冷却通道以保证温度恒定；
- 压铸节拍需与喷粉运作相配合。

## 与传统液体涂料的对比

### 核心差异

粉状/粒状涂料与传统水基/油基液态涂料的核心差异如下[[S16,S30,S8,S9]]：

| 对比维度 | 粉状/粒状涂料 | 传统水基涂料 |
|---------|-------------|------------|
| 成膜条件 | 无需特定模温窗口触发莱顿效应，宽温区间可均匀成膜 | 需在特定温度窗口喷涂，受莱顿弗罗斯特效应影响 |
| 涂层厚度 | 1~5 μm | 较厚，受稀释比影响大 |
| 热冲击 | 熔化热远低于水基涂料气化热，热冲击小 | 汽化吸热强烈，模面骤冷骤热，易引发龟裂 |
| 推出力（AlSi9Cu3） | 约为水基涂料的1/2 | 基准值 |
| 铸件气孔率 | 显著降低 | 受水性载液挥发产气影响 |
| 喷涂时间 | 极短，缩短开模间隔 | 需等待水分挥发 |
| 环境排放 | 无废气、无废水 | 产生大量水蒸气与油雾 |
| 储存稳定性 | 长期暴露在空气中不变质 | 需防冻、防沉降分层、防细菌滋生 |
| 涂料利用率 | >90% | 受喷涂反弹和过喷影响 |

### 环保与综合效益

粉状涂料喷涂过程几乎无废气、无废水产生，无噪声无泄漏，综合能耗远低于传统水基涂料喷涂工艺[[S16,S30]]。同时，运输、储存环节无需考虑防冻、沉降分层问题，操作安全性更高[[S8]]。

## 性能影响

### 对铸件品质的影响

使用粉状压铸涂料可大幅降低铸件内部气孔率——无过量水性载液的挥发产气风险，涂层无大量固体沉积物残留，不会污染铸件表面，铸件表面光洁度显著提升，力学性能得到改善[[S16,S30]]。

### 对模具寿命的影响

粉状涂料在模具表面形成1~5μm厚的均匀极薄涂层，一方面隔绝高温金属液直接冲刷模面，减少粘模焊合倾向；另一方面依托涂层内的微气隙形成热阻，其熔化热远低于水基涂料的汽化热，从根源上避免水基喷涂过程中模面骤冷骤热产生的交变张应力，减少模具表面因热冲击引发的龟裂现象，延长模具服役寿命[[S16,S30]]。

### 潜在缺陷与防控

粉状涂料使用不当可能引发以下质量问题[[S17,S5,S3,S13,S26,S27]]：

| 缺陷类型 | 成因 | 防控措施 |
|---------|------|---------|
| 涂料堆积 | 过量喷涂或工艺参数不当，粉末在型腔转角、凹入区域、排气槽堆积 | 精准控制单位面积用粉量；定期清理易积粉区域 |
| 涂层附着不均 | 喷涂流场/电场分布不均匀，局部模面无有效粉末沉积 | 优化喷涂工位气流/电场分布，保证覆盖均匀 |
| 铸件夹杂 | 未完全熔融附着的松散粉状涂层受高速金属液冲刷脱落，形成非金属夹杂硬质点 | 控制喷涂量，保证粉末在模面充分熔融附着 |

## 商品化与发展现状

### 国外发展

国外工业领域2000年前后逐步实现粉状压铸涂料商业化落地，代表产品包括[[S21,S35]]：
- 德国厂商研发的氮化硼基系列粉状涂料（分型号适配不同合金与工艺条件）；
- 日本花野商社与日本制钢所2001年联合开发的触变成形专用粉状涂料；
- 肯天（Chem-Trend）、埃奇森（Acheson）、欧区爱（HA）、德润宝（Petrofer）等主流压铸化工企业均已完成自身粉状涂料产品线的布局。

### 国内发展

国内已自主研发出组分为石墨、滑石粉、聚乙烯蜡、氮化硼复配的压铸粉末涂料，经性能验证可满足常规铝合金压铸脱模要求[[S21]]。目前国内压铸厂家少有粉末涂料应用实例，发展空间较大[[S35]]。

当前压铸粉末涂料市场需求年增速达10%，适配高真空压铸场景，完美匹配高强韧汽车保安件高真空压铸的工艺需求[[S35,S28,S14]]。

## 标准与规范

目前针对压铸粉状涂料的专门标准尚不完善，以下相关标准可为粉末表征与涂装安全提供支撑：

- **DIN体系**：DIN-Taschenbuch 133 颗粒测量通用规范、DIN-Taschenbuch 247 粉末冶金通用测试规范[[S18]]；
- **GB体系**：GB 6514 涂装作业安全规范（控制作业区粉末爆炸浓度限值≤10 g/m³）、GB 6515 防尘作业规范（控制作业区粉尘浓度≤10 mg/m³）[[S19]]；
- **JIS H系列**：金属表面处理与热喷涂通用规范，包含术语定义、涂层附着力测试等相关方法[[S7]]。

> **说明**：DIN-Taschenbuch 247原为粉末冶金领域通用测试规范，其颗粒表征方法可借鉴用于压铸粉末涂料的粒度分布与流动性测试。GB 6514/GB 6515为粉末涂装作业安全与防尘通用标准，并非压铸专用规范，但其粉尘浓度限值（爆炸极限≤10 g/m³，健康限值≤10 mg/m³）适用于压铸粉末喷涂作业环境。JIS H系列为金属表面处理与热喷涂通用规范，可间接支持压铸粉末涂层附着力等性能测试。
