## 概述

“劈箱造型”在铸造领域存在两种主要语境下的不同含义。在**砂型铸造**中，它是一个国家标准定义的规范化造型方法；在**压铸（压力铸造）**中，它则演变指代一类特殊的整体分型侧向抽芯机构。本词条以压铸技术为核心进行阐述，同时明确其与砂型铸造术语的关联与区别。

## 压铸领域的劈箱造型：劈杆抽芯机构

在压铸模具设计中，“劈箱造型”指一种**利用斜面驱动力同步带动多块大型组合型腔滑块侧向分离的整体分型抽芯机构**，亦称为“劈杆抽芯机构”或“劈箱机构” [[S1,S9]]。其核心功能是解决普通单向侧抽芯机构无法处理的复杂脱模难题——特别是针对**四周带有连续、封闭U形侧凹的框体类压铸件** [[S1]]。

与常规的斜导柱或液压侧抽芯仅用于局部、孤立倒扣的脱模不同，劈箱机构属于**组合式同步大行程分型技术**。它驱动的是构成型腔主体的整块滑块，覆盖铸件多个侧面甚至全周的连续侧凹区域，是一种整体型腔分体式运动技术 [[S1,S9]]。

### 别名与英文术语

- **中文同义/近义术语**：劈杆抽芯机构、劈箱机构、对开箱、哈夫模（哈夫块分型结构）、分块型腔机构、劈箱滑块侧向分型机构 [[S9]]。
- **英文对应术语**：split cavity die, parting box mold, split die design, sectional die construction 等 [[S9]]。

### 核心组成与动作原理

劈箱机构由一套复杂的联动部件构成，以实现“先抽芯、后顶件”的顺序动作，避免滑块与顶出元件发生干涉 [[S1,S5]]。

| 部件分类 | 典型零部件 | 功能说明 |
| :--- | :--- | :--- |
| **驱动与分型组** | 30°斜面劈杆、双侧对合组合型腔滑块、复位楔紧块 | 劈杆利用斜面将直线运动转化为滑块的侧向分型运动，复位楔紧块在合模时将滑块锁紧 [[S5]]。 |
| **二级推件组** | 劈杆推板、主推件板（前推板）、联动锁钩机构、导杆导向副 | 实现两级顶出：第一级仅驱动劈杆抽芯，第二级才推出铸件，两者通过锁钩实现顺序联动 [[S5]]。 |
| **导向与复位组** | 导柱、导套、复位杆 | 确保动定模及各推板的精确导向与顺序复位 [[S5]]。 |

其完整工作循环时序如下 [[S5]]：
1.  **开模**：压铸机开模，动定模分型，铸件留在动模侧。
2.  **一级顶出（抽芯）**：压铸机顶杆推动劈杆推板单独前移。劈杆的30°斜面驱动两侧滑块同步向外平移，完成侧向抽芯，使侧凹完全脱离铸件。此阶段，主推件板保持静止。
3.  **二级顶出（推件）**：当劈杆推板前行一段距离（例如某案例中为55mm [[S5]]）后，锁钩将劈杆推板与主推件板锁为一体。随后，两块推板同步前进，带动所有顶杆将已完成侧抽芯的铸件完整推出型腔。
4.  **合模复位**：合模时，复位杆带动已锁紧的两块推板同步后退，先使所有推杆复位。到位后，锁钩脱开，劈杆推板继续单独复位，最终复位楔紧块迫使滑块向内对合锁紧，准备下一次循环 [[S5]]。

### 设计要点与驱动方式

- **分型面与精度**：劈箱分型面应选择在铸件侧凹的最大轮廓处，并力求平整。合模结合面需进行机械精加工，表面粗糙度不低于∇4，全长不平行度≤0.5mm，对基准面的不垂直度允差≤0.3mm [[S11]]。
- **驱动配合**：劈杆斜面与滑块配合间隙控制在0.01~0.03mm，滑块滑槽导向配合精度选用H7/h6级，以兼顾运动流畅性与密封性 [[S9]]。
- **驱动方式选择** [[S9]]：
    - **机械劈杆驱动**：利用压铸机自身的顶出动作，无需额外动力源，成本低、可靠性高，适用于中小抽芯力场景。
    - **外置油缸驱动**：可独立控制抽芯时序和力量，适用于大吨位、大抽芯距的重型机构，常用于大型深腔框架类铸件。
- **设备适配性**：该机构仅需压铸机提供足够的顶出行程，无需对压铸机本体结构进行额外改造，即可在常规卧式冷室压铸机上使用 [[S5]]。

### 应用场景与局限性

- **典型应用**：ZL7铝合金全周U形槽框体铸件、深腔箱形结构件、带完整环绕侧凹的新能源电机壳基座等中大型复杂结构件 [[S1,S10]]。
- **主要局限性**：模具整体尺寸大，制造与加工成本高；运动配合副多，维护工作量大。因此仅适合批量定型的中大型复杂铸件，不适合小尺寸、结构简单的低成本量产场景 [[S10]]。

### 缺陷关联

- **可避免的缺陷**：该机构能有效防止因脱模不畅导致的侧凹区域拉伤、铸件卡滞、顶出干涉引起的顶面变形，以及多独立抽芯分型面错位引发的批量侧部飞边 [[S7]]。
- **可产生的衍生缺陷**：长期生产后，驱动斜面磨损或定位维护不到位，可能导致劈箱分型面错位（铸件侧部错边）或合模锁模力下降（侧部披锋/飞边） [[S7]]。

## 砂型铸造领域的劈箱造型

根据国家标准GB/T 5611-2017（铸造术语）的定义，劈箱造型（split box molding）是砂型铸造中的一种手工造型方法，指 **“为了便利造型与下芯，将三开箱造型中的中箱沿垂直方向劈开成两半或若干部分进行造型的方法”** [[S2]]。该技术主要用于解决大型、复杂、多砂芯的箱体类铸件（如机床床身）的造型和合箱难题，属于成熟的标准工艺 [[S2, S11]]。关于其详细的分型原则、砂箱设计参数已超出本压铸技术知识库的核心范围，此处不再展开。
