## 概述

切边余量（英文标准术语：trimming allowance [[S4]]）是压铸工艺设计与模具设计中的一个重要交叉概念。在压铸生产中，铸件从模具中取出后，必然带有内浇口、横浇道、溢流槽以及沿分型面生成的飞边等非产品本体结构。切边余量是指在待切除的上述结构与产品本体之间预先预留的可控金属层厚度，其作用是保证在后续切边工序（通过切边模进行冲切）完成后，产品本体边缘无残留毛刺、不会发生崩缺或切肉缺陷 [[S26,S27,S23]]。

切边余量与**机械加工余量**存在本质区别：机械加工余量是面向车、铣、磨等机加工序预留的金属层，用于修正铸件尺寸偏差和消除表面缺陷，属于产品本体待切削去除的部分；而切边余量属于非产品本体结构与产品本体的连接过渡层，仅需通过冲切方式直接分离即可完成移除 [[S11,S22]]。同样，切边余量也不同于**毛坯余量**（加工总余量）——毛坯余量是全流程所有待去除金属层的总和，切边余量仅是压铸工序后专门用于冲切移除飞边、浇口、溢流槽的局部预留过渡层 [[S31,S21]]。

## 英文术语辨析

压铸领域涉及切边余量的英文术语存在使用场景的差异，需注意区分：

- **trimming allowance**：压铸行业通用标准术语，覆盖所有待冲切除去的飞边、浇口、溢流槽的预留余量场景 [[S32,S4]]。
- **flash allowance**：在压铸领域特指对应分型面自然生成飞边的预留冲切余量；在锻造领域则指填充型腔用的飞边槽预留金属量 [[S32,S25]]。
- **trim stock**：指待切除的实体坯料总量，多用于大尺寸浇口、厚截面边缘需要整体移除的场景 [[S1]]。
- **gate removal allowance**：特指内浇口周边专门用于浇口移除操作的预留余量 [[S1]]。

## 工艺作用与必要性

在压铸充型过程中，沿模具分型面不可避免会生成极薄的飞边。受模具吹胀（die blow）影响，分型面处通常存在约 0.25 mm 的吹胀间隙，但实际生成的飞边厚度仅约 0.075~0.127 mm（0.003~0.005 in）[[S23]]。如此薄的飞边在切边工序中无法被干净利落地切断，通常只会被压折翻边，后续仍需昂贵的额外清理工序去除。

为保障切边质量，必须在分型面周边预先设计一个厚度约为 0.5 mm（0.02 in）、宽度约 12 mm 的可控切边凸缘（trim bead 或 safety edge），使切边过程能够整齐切断、无残留 [[S23]]。该凸缘即为切边余量的核心体现。

在压铸全流程中，切边余量的工艺角色定位如下：充型与凝固阶段生成飞边、浇口、溢流槽等附带结构 → 开模取件后冷却 → 切边工序通过切边模以冲切方式将上述附带结构沿预留余量处分离 → 获得净产品本体 [[S26,S19,S34]]。

## 分类

按所处位置，切边余量可分为以下三类 [[S27,S26,S17]]：

| 类型 | 说明 |
|---|---|
| 分型面处切边余量 | 沿分型面周边预留，用于切除分型面自然飞边，是最主要的切边余量类型 |
| 浇口周边切边余量 | 位于内浇口与铸件本体的连接处，用于冲切或折断移除浇口 |
| 渣包/溢流槽周边切边余量 | 位于溢流槽、排气槽等排溢系统结构与铸件本体的连接处 |

不同位置的余量取值和冲切受力特征存在显著差异，其中浇口周边和溢流槽周边的余量设计还需考虑浇道末端形状（应设计为圆角而非尖角，以避免切边后尖角残留）[[S1]]。

## 各合金切边余量特点

不同压铸合金因材料切断强度、切边方式及后处理工艺的不同，切边余量的取值策略存在明显差异 [[S1,S19]]：

| 合金种类 | 切边余量特点 |
|---|---|
| **锌合金** | 切边余量可设置极小。可利用冷冻滚桶清理工艺，在液氮低温下使锌变脆，直接将飞边、浇口脆断移除 |
| **铝合金** | 内浇口宜设计为宽薄结构，对应切边余量适配该结构以降低冲切难度、避免浇口残留线缩松；浇口较厚时须使用带锯切除 |
| **镁合金** | 材料切断强度较低，可采用更小的切边余量取值 |
| **黄铜合金** | 常规内浇口厚度大，需设置更大的切边余量，通常采用带锯切除方式 |

## 切边余量推荐值

### 典型范围

压铸件表层存在厚度约为 0.5~0.8 mm 的致密均匀受力层，具有较高的力学性能，切边余量取值应尽量不破坏该层 [[S37,S28]]。常规后续机加工的总切边余量控制在 0.3~1.5 mm 范围内，小尺寸铸件可取最低 0.3 mm 余量 [[S37,S28]]。

主流压铸合金切边余量基础推荐范围 [[S6,S5,S24,S2]]：

| 合金 | 基础推荐范围（mm） |
|---|---|
| 铝合金 | 0.4~1.8 |
| 锌合金 | 0.3~1.2 |
| 镁合金 | 0.4~1.5 |

余量随铸件最大外形尺寸增大而递增，且与压铸件加工余量分段规则匹配：待加工的内表面尺寸以大端为基准，外表面尺寸以小端为基准 [[S6,S5,S24,S2]]。

### 按铸件尺寸分段推荐值

《金属压铸模设计技巧与实例》表 2-19 给出了按基本尺寸分段的单边切边余量推荐值（切边余量参照该表中待加工余量规范）[[S35]]：

| 基本尺寸（mm） | 单边切边余量推荐（mm） |
|---|---|
| ≤ 50 | 0.3~0.5 |
| >50~100 | 0.5~0.8 |
| >100~200 | 0.8~1.2 |
| >200~400 | 1.2~1.5 |
| >400~630 | 1.2~1.8 |

### 北美标准基准

北美压铸协会（NADCA）配套的 ASTM 标准中规定，压铸件切边留料常规最小取值为 0.010 英寸（约 0.25 mm），最大取值为该最小基础值加上加工补偿量和铸件公差的总和。该数值属于北美行业通用切边余量基准规范 [[S10]]。

## 切边余量与切边模设计的关联

切边余量是切边力计算的核心输入参数。切边力计算公式为 [[S19]]：

**Fₜ = L × T × S**

其中：
- Fₜ —— 切边力（N）
- L —— 切边总长度（mm）
- T —— 需切断的飞边或内浇口厚度（mm），即切边余量厚度
- S —— 铸件材料的切断强度（KN/mm²）

对于大多数压铸合金，S 值可取 0.214 KN/mm²（对应 31 000 lb/in²）；黄铜、白铜、镁合金及 43 号铝合金的切断强度存在差异，直接影响切边余量的基础取值 [[S19]]。

切边模冲头与凹模的常规单边设计间隙推荐取 0.03 英寸（约 0.075 mm）；当需要冲剪整个铸件壁厚的孔时，凹模孔径应比冲头直径大 0.06 英寸（约 0.15 mm）[[S19,S30]]。卸料板和压垫上弹簧的总作用力通常取为理论切边力的 0.035 倍 [[S19]]。

## 切边余量取值的核心影响因素

影响切边余量取值的因素涉及铸件设计、模具设计、合金特性及后处理方式等多个维度 [[S1,S19,S36,S7]]：

| 影响因素 | 说明 |
|---|---|
| 铸件材料 | 切断强度、脆性、韧性决定切边方式和余量基础取值 |
| 铸件尺寸与壁厚 | 最大外形尺寸越大，余量取值越大 |
| 分型面设计 | 台阶状分型面需适当加大余量以降低切边模复杂度；分型面设置在铸件最大轮廓中部时，预留飞边距凸模端面 10~15 mm 的切边余量为行业常规最优配置 [[S36,S12]] |
| 飞边厚度 | 飞边尺寸直接决定切边总行程与切边力大小，飞边厚度增大时切边余量需同步调整 [[S36]] |
| 切边方式 | 冲切、锯切、人工折断、冷冻滚桶清理等不同方式对应不同余量要求 [[S1,S19]] |
| 后续加工要求 | 需配合拔模斜度、铸件收缩率等尺寸公差范围，避免切边后轮廓尺寸超差 [[S7]] |

## 与其他设计要素的协同关系

### 与浇注系统的协同设计

压铸模内浇口与溢流槽的连接位置预留切边余量时，需将浇道末端设计为圆角而非尖角，避免切边后尖角部分残留在铸件表面难以清除。小尺寸的内浇口与溢流槽接口厚度对应小切边余量，可实现手工折断作业，降低后续加工成本 [[S1]]。铝铸件的浇口宜设计为宽而薄的结构，使浇口断面薄弱而易于打掉，同时保持原有浇口面积和充型时间 [[S26]]。

### 与排气槽、溢流槽的协同设计

溢流槽外侧配套的飞边与排气槽残留结构，预留的切边余量需适配溢流槽的布置位置。可在溢流槽本体上预制定位凸块，用于切边时铸件快速定位，消除铸件温度波动导致的尺寸偏差引发的切边错位问题 [[S1,S17]]。

### 与收缩率及拔模斜度的协同匹配

切边余量取值需适配压铸过程的铸件收缩率以及对应拔模斜度的尺寸公差范围，避免切边后铸件轮廓出现尺寸超差问题 [[S7]]。双面预留切边余量时，型腔尺寸取 D' = D + 2δ，型芯尺寸取 d' = d - 2δ，以保证脱模斜度与余量的协调 [[S9]]。

## 缺陷与设计意义

### 余量过小的不利影响

切边余量取值过小会直接引发两类典型缺陷 [[S20,S14]]：

1. **飞边清除不彻底**：残留毛边超出允许公差，需额外手工打磨或砂轮清理。
2. **切边损伤铸件本体**：出现切肉、表面压痕等质量问题。圆形截面压铸类零件径向切边时，余量不足极易在切边凸模下部与铸件接触部分产生压痕，需通过优化切边模凸模结构匹配合理余量方可消除 [[S14]]。

![切边缺陷示意图：切肉与残余毛边](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/b09877e9-c624-40fb-aa58-5c6ce652967d/resource)

### 余量过大的负面效应

余量过大将造成多重负面影响 [[S3,S34]]：

1. **材料浪费**：多余金属直接提升材料损耗率。
2. **切边力升高**：切边总截面积变大导致切边力显著增大。
3. **模具寿命缩短**：切边刃口磨损加剧，崩刃风险上升，切边模服役寿命降低。
4. **设备负载加重**：切边设备负载上升，自动化压铸产线易出现过载停机故障 [[S34]]。
