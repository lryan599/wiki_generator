## 概述

预测维护（又称预知维修，对应英文术语 predictive maintenance / PdM）在压铸语境下，是指基于压铸机、压铸模具及压铸工艺的实时运行状态监测数据，对设备、工装、系统的潜在劣化趋势进行预判，在故障实际发生前针对性安排维护作业的方法。它将以时间为基准的定期计划式维修，升级为以技术状态为依据的非固定周期的精准维护策略[[S22,S2]]。

预测维护与压铸现场常见的另外两类维护策略存在根本性区别[[S16,S32]]：

| 维护策略 | 触发条件 | 核心特征 |
|---------|---------|---------|
| 反应式/纠正性维护 (Reactive / Corrective Maintenance) | 设备或工装故障实际发生后 | 无预先状态监测和维护规划，事后抢修 |
| 预防性维护 (Preventive Maintenance) | 基于历史经验按固定时间周期 | 故障发生前执行，定期更换保养，但存在过度维护导致资源浪费的风险 |
| 预测性维护 (Predictive Maintenance) | 实时采集运行状态数据，预判到故障即将发生的精准时机 | 仅在必要时安排维护，最大化设备有效运行时长，大幅减少不必要维护投入 |

## 基本原理

预测维护的核心逻辑是根据状态监测数据预测设备何时可能发生故障[[S19,S30]]。在压铸场景下，其闭环流程通常涵盖：数据采集→特征提取→状态评估→故障预测→维护计划生成与执行。

信号处理是实现有效特征提取的关键环节。通用的工业级状态监测数据处理标准框架涵盖五大类分析方法，完全适配压铸场景下振动、温度、压力等多类异构状态数据的处理需求[[S6,S36]]：

| 分析类别 | 典型方法 |
|---------|---------|
| 时域分析 | 能量分析、统计特征提取 |
| 频域分析 | 快速傅里叶变换 (FFT) |
| 时频分析 | 小波变换 |
| 统计分析 | 控制图、多元统计 |
| 智能分析 | 神经网络、模式识别 |

在故障预测模型层面，实际韩国压铸工厂的应用案例表明，在覆盖75项压铸工艺参数的采集系统上，使用反向传播神经网络（BPNN）、支持向量机（SVM）、决策树等机器学习算法，可实现短射、裂纹、气泡、变形等8类典型压铸缺陷的提前预判，打通了设备劣化趋势与铸件产品质量之间的关联逻辑[[S4,S37]]。

## 传感与监测体系

现代压铸机本身已具备相当程度的数据采集基础能力：可采集压射曲线、顶出/合模过程速度曲线，支持设定值与实际运行参数的实时对比并输出偏差范围，记录存储机器运行总时长、动作次数、各类运行参数、故障类型及发生时间等全生命周期数据，并支持远程故障诊断与系统升级[[S21,S15]]。

面向压铸预测维护的通用监测传感器及其适用场景包括[[S19,S30]]：

| 传感器类型 | 主要监测对象与故障类型 |
|-----------|---------------------|
| 振动传感器 | 泵、电机的错位、不平衡、机械松动与磨损 |
| 电流/电压传感器 | 驱动电机的供电运行参数 |
| 超声波检测装置 | 管道、储罐的泄漏，移动部件的机械隐性故障 |
| 红外热成像装置 | 全机各测点的异常温度波动 |
| 液体属性检测传感器 | 液压油等工质的关键指标变化 |

在压铸机常规点检方面，行业实践已形成分层保养制度：涵盖液压系统油温油位检查、蓄能器氮气压力校验、过滤器定期清洁更换、大杠受力状态校准等项目。对大于1000 t的压铸机每3年校准一次大杠受力，小于1000 t的压铸机每5年校准一次。这些周期性点检数据可与实时传感数据融合，提升预测模型准确率[[S35]]。

## 关键部件应用实践

### 压射系统

压射系统是预测维护监测的重点对象，传感器安装需遵循特定规范[[S8]]：

- **位移传感器**：固定在与压射缸外壳相连的板上，金属导杆连接固定于活动冲头的短刚性杆，借助冲头运动和传感器内置复位弹簧实现导杆自由伸缩。安装位置需避开200 °C以上高温区域。热压室压铸机建议在压射缸顶部加装小型辅助架固定传感器；若无法满足该条件，需在压射杆上加装延伸杆连接导杆，但该方案易引发导杆刚性不足、振动过大，导致位移读数偏差。
- **压力传感器**：气动压铸机需尽可能靠近压射缸布置以降低信号传输误差；使用快换接头连接至液压回路以适配快速拆装需求。冷室压铸机可直接将压力传感器布置在压射端预留孔位。

布勒（Bühler）H-630SC半固态专用压铸机的实践案例中，配备了金属流头监测传感器用于确定金属充填位置和评估压射过程，同时配备安装在型腔或浇道内的压电类直接读数传感器，实时监测半固态金属充填时的压力变化，用于判断坯料表观黏度、铸件质量分类、识别铸件凝固节点[[S23]]。

### 模具系统

模具内嵌传感器的预测维护方案应在模具设计阶段提前规划。建议借助CAD工具将常用传感器类型数字化存入元件库，完成传感器孔位和线缆通道的预设计，避免后期在淬火硬化的模具本体上通过EDM开孔加工带来的高昂改造成本和改模损伤风险[[S7]]。

以horse shoe形参考测试模具为例，型腔传感器网络共布署8个内嵌传感器，包括分布在不同型腔位置的5处温度传感器和3处不同浇道/型腔位置的压力传感器，配套金属流头接触传感器，可同步采集5路型腔温度曲线、3路型腔压力曲线以及柱塞位移与速度全周期曲线，用于监测不同厚度区域模具的热疲劳趋势，预判型腔磨损与浇道堵塞故障[[S14,S25]]。

针对浇口凝固行为监测，采用可耐受最高700 °C工况的Kistler压电式型腔压力传感器，分别安装在浇口前流道、浇口后型腔侧、铸件中心、型腔顶端共4个位置，同步配套K型热电偶温度传感器（直径2 mm，距型腔表面10 mm处安装）。该方案最早在Bühler 400吨级高压压铸机上得到工业验证，可精准监测浇口凝固时间变化，提前预判模具热变形和浇道磨损劣化趋势[[S26]]。

### 模具冷却系统

采用铠装K型热电偶（线径0.1 mm，响应时间7～10 ms），在步阶形不同厚度铸件对应的模具区域内，分别在距型腔表面1 mm、3 mm、6 mm深度位置布署多组温度传感器，可实时监测不同深度位置的热传导变化，预判冷却水道结垢、堵塞和泄漏故障[[S18]]。

## 收益与挑战

### 核心收益

预测维护在压铸领域的效益已在实践中得到验证[[S28,S19,S27]]：
- 设备维护所需时间最小化；
- 非计划停机导致的生产时间浪费最小化；
- 备件库存及购置成本最小化；
- 设备利用率可从传统50%～55%提升至70%～75%，显著降低突发故障造成的长时间生产中断。

### 实施挑战

压铸行业落地预测维护面临多重现实障碍：

1. **数据基础薄弱**：多数压铸企业仅不到30%的车间实施了固定周期的防护性维护，仅10%的企业系统性地记录与归档故障全流程数据，缺少用于训练预测模型的高质量标注数据集[[S9]]。

2. **技术路线局限性**：基于设备状态直接评估的方法易受压铸现场高温、高湿、高振动工况干扰；基于物理模型推演的方法算力需求极高；纯数据驱动的统计/机器学习方法可解释性不足，多数故障预警结论难以被一线运维人员理解与采信[[S33]]。

3. **初期投入高**：需批量部署振动、红外热成像、压力、电流等多类型感知终端，搭建边缘计算与云边协同的数据分析架构，中小压铸企业短期内投入产出比难以达标[[S19]]。

4. **复合型人才缺口**：运维人员需同时掌握压铸工艺原理、设备机械/液压/电气架构知识以及数据分析与故障预警模型使用能力，行业内相关人才缺口较大[[S12]]。
