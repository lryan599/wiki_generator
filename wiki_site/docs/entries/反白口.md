## 定义与分类

**反白口（inverse chill）** 是中国机械工业国家标准定义的铸件内部显微组织类缺陷，典型特征为灰铸铁件断面的中心部位出现白口组织或麻口组织，而外层为正常的灰口组织[[S4]]。该缺陷本质上是铸铁件在特定凝固条件下发生局部按介稳定系（Fe-Fe₃C）结晶而形成的渗碳体或莱氏体组织异常，与铸铁件常见的表层白口缺陷（冷却较快处先凝固形成白口）在空间分布上正好相反，故称“反白口”。

反白口缺陷并非压铸（高压/低压铸造有色金属）领域的专属术语。在现有公开的NADCA标准、ISO国际标准、JIS压铸标准体系中，铝合金、锌合金压铸件不存在以“反白口”命名的独立缺陷条目[[S3,S5]]。有色铸造生产现场俗称的“反白口”，实质上是借用了铸铁领域“外层正常组织、内部白色硬点”的形貌特征进行类比，指代有色金属铸件内部的反向偏析导致的白色硬质点或白斑缺陷子类[[S3,S5]]。因此，本词条以铸铁（灰铸铁、球墨铸铁、蠕墨铸铁等）领域的反白口为主体进行阐述。

## 金相特征

### 宏观断口

反白口在铸件断面上表现为内部密集的块状或点状白色硬块，周围及外层为正常的灰色断口组织[[S11]]。在活塞环类薄壁铸件中，反白口区域从径向观察呈现出三个特征区：白口区、过渡区和明灰区[[S8]]。

### 显微组织

反白口的显微组织通常为莱氏体（共晶渗碳体与奥氏体转变产物的机械混合物）或多相混合组织。对球墨铸铁反白口处的金相观察表明，从铸件外层到内部，基体组织依次为珠光体、针状体和渗碳体，最后凝固的中心区域在较大过冷条件下按介稳定系结晶形成白口组织[[S8,S9]]。

反白口组织中还存在分布特征：块状反白口边缘及细小反白口部分均分布在共晶团边界处，表明该部位凝固点最低、最后凝固[[S11]]。

## 形成机理

反白口的形成机理核心在于化学成分偏析与凝固条件异常导致石墨化受阻。

### 元素偏析机制

在缸套等铸铁件凝固过程中，反白口部位聚集了低熔点的富铁硫化物。这些硫化物在铸铁结晶时以液态形式存在于共晶体边界上，吸附于共晶体界面，阻碍碳的扩散，阻止石墨晶核的生长，并与奥氏体及Fe₃C共同形成低熔点三元共晶体[[S11]]。

硫元素与锰元素的比值（Mn/S）是影响反白口严重程度的关键参数：当Mn/S < 6时，出现大块反白口；当Mn/S ≥ 10时可有效抑制反白口的产生[[S11]]。

### 凝固过冷机制

在活塞环等薄壁铸件中，由于内浇口与铸件连接处长时间受到流经铁水的热作用，形成局部热节。该处铁水凝固时，基体中的石墨化元素（C、Si）含量由表及里逐渐减少，凝固中心部位的化学成分由过共晶成分转变为共晶甚至亚共晶成分。同时铁水存在较大的过冷度，加上劣质孕育剂使石墨形核困难，导致最后凝固区域的铁水无法按稳定系析出石墨，而按介稳定系结晶形成白口组织[[S8,S9]]。

## 工艺影响与危害

反白口缺陷对铸件性能有显著负面影响：

- **性能不均**：反白口区域与正常基体之间硬度差异明显，导致铸件整体力学性能不均匀[[S11]]。
- **强度与韧性降低**：白口组织呈硬脆特性，致使铸件强度下降、韧性恶化[[S11]]。
- **切削加工性恶化**：白口组织硬度极高，刀具磨损加剧甚至无法切削[[S11]]。
- **消除困难**：反白口试块需加热至960～980℃保温15h以上才能转变为灰口组织，而常规白口试样仅需900～920℃保温1.5h即可完全转变，两者处理难度差异显著[[S11]]。

## 影响因素与工艺参数控制

反白口的产生受化学成分、孕育处理、浇注工艺等多方面因素的影响。以下为主要影响因素及控制要点：

| 影响因素 | 影响规律 | 推荐控制范围 |
|---------|---------|-------------|
| Mn/S比值 | 比值越低，反白口越严重 | Mn/S ≥ 10[[S11]] |
| 硫含量(w(S)) | 高硫促进富铁硫化物生成，阻碍石墨化 | w(S) < 0.1%[[S11]] |
| 硅含量(w(Si)) | 硅促进石墨化，有助于抑制反白口 | w(Si) ≥ 1.7%[[S11]] |
| 残余锰量(w(Mn))—铁素体基体球铁 | 锰促进碳化物稳定，高残留增大反白口风险 | w(Mn) < 0.4%[[S7]] |
| 残余锰量(w(Mn))—珠光体基体球铁 | 同上，但范围略宽 | w(Mn)=0.3%~0.5%[[S7]] |
| 球化剂Mg/RE比 | 残余稀土含量高加重偏析倾向 | Mg/RE > 1[[S7]] |
| 孕育处理 | 高效孕育剂提供石墨形核衬底，减少过冷 | 含钡硅铁孕育剂，二次孕育[[S9]] |
| 浇注温度（中小件） | 适当提高浇注温度有助于防止反白口 | 视具体条件调整[[S7]] |

## 工程对策与预防措施

### 成分控制

在保证球化合格的条件下，尽量减少碳化物稳定元素含量，特别是残留锰量。使用低稀土含量的球化剂，避免稀土元素加重偏析[[S7]]。需注意，单纯提高碳硅含量不能有效防止反白口，反而可能因硅含量增大恶化其他元素的扩散条件，加剧偏析趋势[[S7]]。

### 孕育处理优化

采用含钡硅铁孕育剂替代传统石墨电极粉孕育剂。钡、钙等元素可与铁水中的碳、硫形成碳化物、硫化物质点，作为石墨形核的衬底，促进石墨形核，减少过冷，有利于消除反白口[[S9]]。在实际操作中，可在大包内加入0.1%孕育剂进行初次孕育，再根据试片断口情况，在小浇包内加入0.15%~0.30%钡硅孕育剂进行二次孕育[[S9]]。

### 浇注工艺优化

- 浇注温度控制：提高中小件浇注温度，使铁液始终处于良好的孕育状态[[S7]]。
- 浇注时间与停留：球化处理后尽量缩短浇注停留时间，避免孕育衰退[[S7]]。

## 与相近术语的鉴别

| 缺陷/术语 | 本质差异 | 适用领域 |
|----------|---------|---------|
| 反白口（inverse chill） | 内部组织缺陷：断面中心白口组织，外层正常[[S4]] | 灰铸铁、球墨铸铁 |
| 白口（chill/white iron） | 表层组织异常：铸件快冷表层出现白口[[S11]] | 铸铁件 |
| 反偏析（inverse segregation） | 成分不均匀：溶质集中在先凝固区域，与正偏析方向相反[[S4]] | 通用铸造 |
| 白斑（铝合金） | 宏观低倍试片上形状不定的白色纯铝块状物，显微硬度极低，由纯铝熔体溅入合金熔体结晶形成[[S1,S10]] | 铝合金铸锭 |
| 变色、斑点（压铸） | 表面缺陷：由脱模剂不当、石墨润滑剂落入、模温过低引起表面颜色异常[[S6]] | 铝合金/锌合金压铸件 |
| 硬质点（hard spot） | 内部硬夹杂缺陷：由氧化物（刚玉）、碳化物、卤化物盐等引入[[S5]] | 压铸件 |

特别说明：压铸件的表面变色斑点属于脱模剂选型不当、喷涂过量、石墨润滑剂落入铸件表层或模温过低导致的不规则凝固造成的表面缺陷，其位置为铸件表面，形成机理与内部组织转变无关，与反白口本质不同[[S6]]。铝合金铸造领域的白斑缺陷属纯铝夹杂，在宏观试片上与基体边界清晰、颜色发白，显微组织为纯铝组织、无正常合金枝晶网络、显微硬度远低于基体[[S1,S10]]，同样与反白口有别。
