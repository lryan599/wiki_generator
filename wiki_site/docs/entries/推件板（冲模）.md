## 概述

推件板（冲模，英文：stamping die ejector plate）是冷冲压模具中将滞留在凹模型孔内的冲件（复合模中紧包在凸模上的制件）或废料反向推出的板状零件[[S2,S12]]。根据 GB/T 8845-2017《模具 术语》，冲模语境下推件板（slide feed plate）定义为“将制件推入下一工位的板状零件”[[S19]]，在实际应用中推件板主要担负推出凹模内工件或废料的功能，须与推件装置的其他零件配合动作[[S2]]。英文术语中，“stamping die ejector plate”是冲模专用称谓，无跨场景歧义[[S11]]。

## 分类与结构形式

### 按安装位置和动力来源分类

推件板按安装位置可分为**上模推件板**（用于凹模在上模的倒装式模具）和**下模推件板**（用于凹模在下模的顺装模具）[[S2,S12]]。按动力来源可分为**刚性推件板**（依靠压力机打杆机械驱动）和**弹性推件板**（依靠弹簧、橡胶等弹性元件驱动）[[S29,S31]]。 

- 刚性推件装置动作逻辑：模具回程时，压力机上横梁的打料横杆撞击贯穿模柄的打杆，推力经推板、连接推杆依次传递至推件板，将工件刚性推出；推件力大，工作可靠，适合厚板冲裁[[S29,S52]]。
- 弹性推件装置动作逻辑：依靠预先压缩的弹簧或橡胶弹力完成推件，推件过程中对冲件有同步压料作用，工件平直度更高，适用于薄板、高平直度要求的冲裁；刚性打料结构常作为弹力不足时的保险冗余[[S31,S48]]。

弹性推件装置进而又分**固定式弹压**和**可调式弹压**两种结构。固定式弹压的弹簧/橡胶预紧力不可调节，装入模具后无法在不拆解状态下调整弹力，不利于装配阶段通过试冲检查凸、凹模间隙[[S23]]。可调式弹压通过外部螺塞调节弹力，更换弹簧或调整弹力时无需拆模；在装配过程中可松开螺塞让凸模进入凹模型孔进行试冲校核间隙，部分结构可待间隙调整完毕后再装入弹压组件，避免干涉[[S23,S35]]。

### 典型结构形式

常见推件板结构分为**整体式**和**组合式**。

- **整体式推件板**：板体下端设有限位凸缘以限制推出行程；装入模具后上端面需与凹模大面齐平，不能低于凹模工作面；圆形推件板的凸缘根部常设退刀槽，便于加工和装配贴合；推件板与凹模型孔优先采用全长度配合，导向性好，可防止受力不均时发生偏斜卡滞[[S61]]。板面上还可开设让位槽，为冲件已成形凸起部分预留空间[[S2]]。
- **组合式推件板**：由推板主体和带限位作用的盖板通过开槽沉头螺钉连接固定。推板主体可做成直通异形结构，便于线切割加工异形轮廓和异形孔，修配难度低；盖板可简化为规则矩形，降低加工难度，还可额外布置弹性元件实现辅助压料，防止推件偏斜[[S4,S43,S61]]。

## 装配关系与配合要求

推件板在冲模中的典型空间位置与装配关系如下：

- 倒装式复合模中，推件板布置于上模的凹模型腔内部，其上的孔位需与穿过推件板的冲孔凸模留有让位空间，避免干涉[[S21,S32]]。
- 推件板与凹模型孔为滑动配合，配合段长度应导向充分；与卸料板分处对侧工作区域，动作路径互不干涉[[S2]]。

装配技术要求：

- 推件板装配完成后，工作面应高出对应凹模刃口面 **0.2～0.5 mm**，不得低于凹模工作面[[S10,S15]]。
- 所有连接推杆（顶杆）长度误差应 **不大于 0.1 mm**，以保证推件板受力均匀、不偏斜[[S10,S15]]。
- 推件板在整个运动行程内动作应顺畅无阻滞[[S15]]。

配合间隙推荐值如下表所示。

**推件板与凹模型孔的双面间隙**（精密配合，适合薄软材料）[[S16,S22,S57]]：  

| 轮廓尺寸 / mm | 双面间隙 / mm |
|---------------|----------------|
| ≤30           | <0.025         |
| >30～80       | <0.035         |
| >80～180      | <0.040         |
| >180～315     | <0.060         |

**推件板与冲孔凸模的双面间隙**（较松配合，含大凸模与小凸模情况）[[S24,S30]]：  

| 轮廓尺寸 / mm | 双面间隙 / mm |
|---------------|----------------|
| ≤30           | <0.10          |
| >30～80       | <0.15          |
| >80～180      | <0.20          |
| >180～315     | <0.25          |
| 小凸模 (d<2 mm) | 稍大于推件板与凹模的配合间隙，以实现导向和保护 |

## 关键设计参数

### 推件行程

模具开启状态下，推件板工作面需高出凹模平面 **0.2～0.5 mm**，以保证推件可靠性[[S6,S54]]。弹性元件总压缩量通用公式为：

\[
S_{\text{总}} = S_{\text{预}} + S_{\text{工作}} + S_{\text{修磨}}
\]

其中，推件板工作行程推荐取 \(S_{\text{工作}} = t + 1\text{ mm}\)（\(t\)为板料厚度），模具修磨/调整量 \(S_{\text{修磨}}\) 一般取 **4～10 mm**[[S6,S16,S49]]。

### 推件力计算

- **英制经验公式**（适用于冲压卸料/推件力估算）[[S1,S8,S25]]：
  \[
  F = 855 \times P \times T
  \]
  式中 \(F\) 为推件力（lb），\(P\) 为所有冲孔凸模刃口总周长（in），\(T\) 为板厚（in）。常数 855 是 0.00117 的倒数，用于简化计算[[S8,S25]]。

- **公制公式**（单位 kN）[[S60]]：
  \[
  P_s = K \times L \times t
  \]
  式中 \(L\) 为冲切总周长（mm），\(t\) 为板厚（mm），\(K\) 为推件系数：
  - 低碳钢，厚 \(<1.5\text{ mm}\)，边缘或临近已有切口冲裁：\(K = 0.0103\)
  - 同材料其余位置冲裁：\(K = 0.0145\)
  - 低碳钢，厚 \(>1.5\text{ mm}\)：\(K = 0.0207\)
  - 硬材料：\(K = 0.0241\)

### 弹簧/气弹簧选型原则

1. **预压力校验**：弹簧总预紧力 \(F_{\text{预}} \ge F_{\text{推件}} / n\)，保证足够初始推件力[[S6,S16,S49]]。
2. **最大许可压缩量校验**：弹簧允许的最大压缩量必须大于推件所需总压缩量，避免超载失效[[S6,S16,S49]]。
3. **结构空间适配**：所选弹簧的安装尺寸须完全匹配模具内部预留空间[[S6,S16,S49]]。

## 失效形式与维护

冲模推件板的常见失效模式及原因包括：

- **变形**：推板厚度不足、刚性不够时，在推出力作用下产生弹性或塑性弯曲，导致推杆中心线与运动方向不平行，进而引发推杆拉伤、啃损甚至折断[[S27]]。
- **卡滞**：推板与配合零件过紧、凹模导向段过短、冲切深度过大、推板失去导向或材料硬度不足被镦粗挤死，均会造成动作不畅[[S20]]。
- **歪斜**：推件力分布不均、施力点偏离推板重心，导致推板倾斜并与导向面干涉[[S20]]。

日常维护要点：定期检查推板动作是否顺畅无卡滞，确认推件力足够、推板端面与凹模（凸模）端面齐平；及时清理配合面污物并添加润滑；导向结构需定期校准防止偏斜[[S20]]。对弯曲、拉深类模具的弹压推件板，可适当进行淬硬处理以提升耐磨性，减少板材流动摩擦带来的磨损[[S12]]。

## 与压铸模推出机构的区别

冲模推件板与压铸模中的推板（ejector plate）在功能、结构和工况上有本质区别，不可混淆：

- **术语与定义**：GB/T 8845-2017 明确区分了冲模推件板（slide feed plate）与塑料模、压铸模推件板（ejector plate），后者定义为“推出制品、制件的板状零件”[[S19]]。压铸模标准 GB/T 4678.8-2017 规定的“推板”是推出系统的安装基体，用于固定推杆、复位杆等[[S5,S51]]。
- **材料与硬度**：冲模推件板常用工具钢，硬度通常在 43～62 HRC 以上；压铸模推板推荐材料为 45、50、3Cr2Mo，硬度仅 28～32 HRC[[S51]]。
- **驱动方式**：冲模推件板常用压力机打杆刚性驱动、弹簧/橡胶弹顶驱动或可调弹压装置驱动[[S31,S45]]；压铸模推板依靠压铸机顶杆机动、液压缸或手动驱动，合模时由复位杆复位[[S5]]。
- **工作环境**：冲模推件板在 **常温** 下工作，承受冲压过程的中等摩擦载荷[[S18]]；压铸模推板承受 **100～300℃** 左右的模具环境温度及热交变载荷、大质量铸件的脱模反力[[S33]]。

英文术语方面，“ejector plate”在塑料模、压铸模场景中即指推件板，与冲模中的推件板（stamping die ejector plate）需加以区分[[S11]]。
