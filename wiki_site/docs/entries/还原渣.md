## 概述

还原渣（reducing slag）在国内铸造行业标准体系中具有明确界定。标准定义指出：在熔炼还原阶段加入较多还原剂后生成、可使金属液中金属氧化物还原为金属单质的炉渣，典型类别包括白渣与电石渣 [[S27,S31]]。在压铸领域，该概念主要源自铸钢熔炼，但铝合金、锌合金压铸熔炼净化体系中同样存在类比场景——部分生产场合将铝合金精炼工序中添加还原剂后生成的低氧势反应渣也归类为还原渣 [[S34]]。

还原渣具备直接还原金属氧化物的能力，这一特征是其区别于常规精炼渣的核心判定依据 [[S13,S34]]。

## 定义

依据《机械基础制造工艺标准汇编 铸造 上》中的明确条文：

> 还原渣（reducing slag）：在炼钢还原期中加入较多还原剂后造成的、能使钢液中金属氧化物还原成金属的炉渣，如白渣和电石渣。 [[S3,S27,S31]]

白渣是还原期中加入石灰、萤石、焦炭粉、硅铁粉后形成的还原渣，冷却后呈白色且能自行粉碎；电石渣则为含有 CaC₂ 的还原渣类型 [[S3,S40]]。二者的共同特征是渣中 FeO 含量维持在极低水平，从而阻断炉渣向金属液传递氧原子的路径 [[S26]]。

在铝合金、锌合金压铸场景中，标准定义下的还原渣不属于专门用于粗金属还原冶炼的还原性熔炼渣，其生成过程伴随金属液脱氧除杂作业，本质归类为合金液净化渣 [[S23,S27]]。

英文对应术语方面，中国压铸铸造行业公认的标准对应词为 **reducing slag**；reduction slag 泛指所有还原冶炼工序产出的渣料，不属于压铸铸造领域的通用规范术语；deoxidizing slag 特指仅承担脱氧单一功能的脱氧渣，属于还原渣的窄范围子集 [[S3,S27,S26]]。

## 与邻近概念的关系

还原渣与氧化渣、浮渣、铝灰等术语之间存在明确界定边界：

- **氧化渣（oxidizing slag）**：在熔炼氧化期生成，含大量高活性氧化铁（FeO），可向金属液传递氧原子，推动氧化脱碳等反应 [[S13,S34]]。
- **浮渣（dross/cinder）**：熔融金属与大气或耐火材料反应生成的多相固态聚集物，通常浮于熔池表面 [[S13]]。
- **铝灰**：铝合金压铸熔炼所有产出渣料的统称。还原渣属于铝灰中特定的低氧势还原性子类 [[S29]]。

还原渣区别于氧化渣的关键在于：氧化渣向金属液传递氧，而还原渣从金属液中夺取氧，两者化学作用方向截然相反 [[S26]]。

## 分类

### 按渣系类型

电弧炉炼钢中两类典型还原渣（电石渣与白渣）的化学成分组成范围如表所示 [[S40]]：

| 成分 | 电石渣 (wt%) | 白渣 (wt%) |
|------|-------------|-----------|
| CaO | 55~65 | 50~60 |
| SiO₂ | 10~15 | 15~25 |
| MgO | 6~10 | 5~10 |
| Al₂O₃ | 2~4 | 2~4 |
| FeO | ≤0.5 | ≤0.5 |
| MnO | ≤0.3 | ≤0.3 |
| CaF₂ | 5~10 | — |
| CaS | ≤1.0 | ≤1.0 |
| CaC₂ | 1~4 | 无 |

电石渣含 CaC₂ 而白渣不含，这一特征是区分两种还原渣类型的关键指标 [[S40]]。

### 按熔炼碳含量选型

还原渣类型与合金碳含量之间存在严格的对应关系 [[S16]]：

| 钢中碳质量分数 (%) | 推荐还原渣类型 |
|-------------------|---------------|
| <0.35 | 白渣 |
| 0.35~0.60 | 弱电石渣 |
| >0.60 | 电石渣 |

### 铝合金体系下的类别归属

对于铝合金压铸而言，还原渣不属于“粗金属还原冶炼渣”范畴，而是合金液净化过程中伴随脱氧除杂所生成的渣料，可视为合金液净化渣的子类 [[S23,S27]]。部分大量使用再生铝锌废料的压铸企业，其熔炼生成的还原渣会混入废料表面氧化层及杂质带出的次生渣组分，某些研究据此将其划入再生回收渣类别，不同来源资料在分类上存在差异化判定 [[S29]]。

## 形成机理

1. **铸钢体系中的形成路径**：在碱性电弧炉炼钢的还原期，于氧化渣扒除后加入焦炭粉、硅铁粉等还原剂，同时配加石灰、萤石造渣。还原剂将渣中 FeO 还原为 Fe，使渣中 FeO 含量降至 0.5% 以下，同时生成的 CaO–SiO₂–CaF₂ 基渣系与钢液持续接触，完成脱氧、脱硫反应 [[S33,S38]]。

2. **铝合金压铸中的净化生成路径**：铝合金熔炼时，采用 NaCl-KCl 基熔剂（常见配比 NaCl 45%、KCl 50%、冰晶石 5%）覆盖或处理熔体，熔剂润湿并吸附 Al₂O₃ 等氧化物夹杂，生成密度显著低于铝液的复合渣相上浮至液面，扒除即完成除渣净化 [[S8,S15]]。

3. **铝合金熔炼渣的物相组成**：结晶光学检测表明铝合金熔炼渣相组成一致性高，其中 60%~75% 为金属铝及其各类氧化铝，其余组分为石英、尖晶石、硫化物等夹杂物 [[S29]]。XRD 图谱可明确识别出 Al、NaCl、KCl、α-Al₂O₃、γ-Al₂O₃ 等核心物相 [[S35]]。

4. **不同工位的渣组分差异** [[S37]]：

| 生成工位 | 渣组分特征 |
|---------|-----------|
| 熔炼炉内 | 金属氧化物 + 金属铝 + 电解质 + 熔剂 |
| 静置炉内 | 金属氧化物 + 金属铝 + 熔剂 |
| 出铝抬包 | 金属氧化物 + 金属铝 + 电解质 |
| 铸造工位 | 金属氧化物 + 金属铝（无电解质和熔剂） |

## 工艺参数与控制要求

### 成分控制指标

- 普通品质铸件熔炼：还原渣中 FeO 质量分数 ≤1.0% [[S38]]。
- 高质量铸件熔炼：还原渣中 FeO 质量分数需控制在 0.5% 以下 [[S33]]。
- 当 FeO 质量分数处于 0.5%~0.7% 区间时，普通质量铸钢件表面缺陷会明显增多；FeO 高于 0.7% 的炉渣不具备出钢条件 [[S33]]。
- 还原期总渣量占金属液总质量比例约为 3%~4% [[S38]]。
- 控制合适碱度（CaO/SiO₂ ≈ 2.5）且流动性良好的还原渣可实现 50%~70% 的钢液脱硫率 [[S38]]。

### 铝合金除渣熔剂参数

- 推荐熔剂体系：NaCl-KCl 系专用熔剂，最优成分比例为 NaCl 45%、KCl 50%、冰晶石 5% [[S8]]。
- 熔剂加入量推荐：每吨铝 1 kg [[S1]]。
- 铝渣与非金属成分质量比控制在 1:1~1:3 区间 [[S8]]。
- 常用铝熔炼熔剂类别及功能见下表 [[S14]]：

| 熔剂类别 | 主要成分 | 核心功能 |
|---------|---------|---------|
| 精炼剂 | NaCl、KCl、氟化物等 | 除气除渣 |
| 覆盖剂 | NaCl、KCl | 防氧化吸气 |
| 打渣剂 | NaCl、KCl、氟化物 | 清炉除渣 |

### 温度与分离参数

- 铝热态还原渣无盐回收工艺：转炉工作温度 649℃~816℃，转速 10~30 r/min，渣料停留滚动时间 1~10 min，铝金属回收率可达 80% [[S8]]。
- 铝合金熔炼适当提高过热温度可降低熔体黏度，利用斯托克斯定律提升还原渣夹杂物上浮速度；静置澄清可先将半径大于 0.01 cm 的大颗粒还原渣从合金液中分离 [[S9,S21]]。

### 镁合金压铸炉渣清理规范

- 清理前通入混合保护气（N₂ 流量 100、SF₆ 流量 20）保持 5 min [[S10]]。
- 单次渣清理操作时长需控制在 3 min 以内；若超时需盖炉等待 10~15 min 待炉内白烟完全散尽后再继续 [[S10]]。
- 现行 GB 26488-2025《镁合金压铸安全生产规范》已将熔炉打渣作业安全要求纳入强制管理条目 [[S28]]。

## 与铸件缺陷的关联

还原渣混入合金液后直接关联的典型压铸铸件缺陷包括以下三类 [[S39,S6]]：

1. **夹渣**：还原渣裹入金属液形成的夹杂类缺陷，多分布在铸件上部、型芯下表面等位置。断口无光泽呈暗灰色。
2. **硬质点**：分散的硬质还原渣夹杂物，多在机械加工工序被发现，会造成刀具损坏。
3. **渣气孔**：还原渣与气孔复合生成的缺陷，多成群集结在铸件上表面。

缺陷成因主要追溯至熔炼扒渣不净、浇注系统设计不当以及浇注工艺参数不合理 [[S39]]。

## 控制与除渣技术

### 人工扒渣

- 使用预热的扒渣工具，扒渣过程应平稳操作防止渣子卷入熔体内 [[S1]]。
- 理想扒取渣块直径为 2.5~3.8 cm，单次扒渣可去除 95% 以上表面浮渣 [[S32]]。
- 浮游法除渣通过向合金液中通入惰性气体（氮气、氩气）或使用氯化盐熔剂反应生成气泡，气泡上浮吸附悬浮的细小还原渣颗粒并带出至液面去除 [[S9,S41]]。

### 在线联合除渣工艺

铝合金除渣作业推荐采用在线除气 + 泡沫陶瓷过滤联合工艺：先通过在线除气单元鼓入惰性载气带出溶解氢和悬浮还原渣，再流经泡沫陶瓷过滤网吸附剩余细小还原渣颗粒，可大幅降低合金液非金属夹杂物含量 [[S30,S20]]。

### 熔剂法除渣

铝液中还原除渣存在上熔剂法和下熔剂法两种工艺结构，利用密度差实现对不同粒度与密度夹杂物的捕捉。上熔剂法适用于密度小于金属液的氧化物夹渣，下熔剂法针对密度大于金属液的沉渣 [[S17]]。

## 铝熔体中高熔点氧化物夹杂的成分示例

还原渣中的高熔点氧化物夹杂通常以 Al₂O₃ 为主体，辅以 MgO、SiO₂ 等组分。典型成分示例如下 [[S25]]：

| 组分 | 质量分数 (wt%) |
|------|---------------|
| Al₂O₃ | 54.19 |
| SiO₂ | 7.10 |
| MgO | 32.85 |
| MnO | 5.86 |
