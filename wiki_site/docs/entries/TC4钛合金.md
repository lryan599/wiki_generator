## 概述

TC4钛合金是一种典型的(α+β)型两相钛合金，名义成分为含6%的α稳定元素Al和4%的β稳定元素V，是国内外应用最广泛的钛合金牌号之一[[S12,S11,S35]]。该合金具有良好的综合力学性能、焊接性能与铸造性能，可在350℃下长期服役，被广泛用于制备航空航天构件[[S24,S18]]。

## 定义与分类

TC4属于Ti-Al-V系α+β型双相钛合金，室温下α相与β相共存。在常规状态下，β相体积占比为5%~25%，平衡态下约为7%~10%[[S11,S16,S27]]。

**国际对应牌号**[[S12,S16,S27]]：

| 国别/体系 | 对应牌号 |
|-----------|---------|
| 美国 | Ti-6Al-4V（Grade 5）、UNS R56400 |
| 苏联/俄罗斯 | BT6 |
| 英国 | IMI-318 |

在压铸及铸造材料体系中，TC4属于钛合金铸造类别，对应的铸造态牌号常记为ZTC4。美国ASTM标准中，铸造钛合金的牌号形式为Grade C+顺序号，Ti-6Al-4V铸造态对应Grade 5[[S37]]。

## 化学成分与组织结构

### 化学成分

TC4钛合金以Ti为余量，主要合金元素及典型范围见下表。

| 元素 | 含量范围（质量分数/%） |
|------|----------------------|
| Al | 5.5~6.8 |
| V | 3.5~4.5 |
| Fe | ≤限值（受标准管制） |
| C | ≤限值 |
| N | ≤限值 |
| H | ≤限值 |
| O | ≤限值 |
| Ti | 余量 |

主元素成分为Al 5.5~6.8%、V 3.5~4.5%，杂质元素Fe、C、N、H、O的最高含量受到严格限定[[S11,S34,S4]]。

### 相组成与相变温度

TC4合金在高温下为β相组织，在约1000℃以下为(α+β)两相组织。由于实际成分波动，其(α+β)/β相转变温度实测值为1000±14℃[[S34]]。

### 显微组织特征

铸态TC4合金从高温熔化状态冷凝后得到粗晶魏氏组织：α相呈白色片状，β相分布于α相的片层之间[[S34]]。

![铸态TC4钛合金粗大魏氏组织，×150](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d7a82f72-8a15-4930-8578-60767856afea/resource)

经850℃退火处理后，微观组织转变为α相为基体、β相呈短棒状和不规则点状分布在α相周围的结构[[S34]]。

## 物理与力学性能

### 物理性能

TC4钛合金的密度为4.5 g/cm³，属于典型的(α+β)双相钛合金[[S11,S28]]。

### 铸造工艺性能参数

| 参数 | 数值 |
|------|------|
| 液相线温度 | 1650℃ |
| 固相线温度 | 1570℃ |
| 凝固温度区间 | 80℃ |
| 充型流动性 | 与工业纯钛相近 |
| 集中缩孔体积 | 1.2~1.3% |
| 缩松体积 | 约0.75%（倾向较低） |
| 线收缩率 | 1.5~1.6% |
| 热裂纹倾向 | 低，不易产生热裂 |

以上数据均来源于权威材料手册[[S9]]。

### 力学性能

铸态TC4钛合金经退火后的典型室温力学性能如下[[S9]]：

| 性能指标 | 典型值（退火态） |
|----------|-----------------|
| 抗拉强度 σ_b | 约931 MPa（95 kgf/mm²） |
| 屈服强度 σ_0.2 | 约841 MPa（85.8 kgf/mm²） |
| 断后延伸率 δ | 10% |

经固溶处理（水淬）加时效后，抗拉强度可提升至约114.6 kgf/mm²（约1124 MPa），屈服强度约107.6 kgf/mm²（约1056 MPa），延伸率降至约5%[[S9]]。退火态与铸态的强度对比数据显示，800℃保温4h退火后UTS和0.2%屈服强度均显著提升[[S10,S36]]。

400℃高温条件下，抗拉强度约为63 kgf/mm²（约618 MPa），400℃/100h持久强度约为58 kgf/mm²（约569 MPa）[[S9]]。

![TC4合金铸件铸态与800℃保温4h退火态UTS及0.2%屈服强度对比](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/f3e4cabb-3c0d-4357-8efa-4571921f786e/resource)

### 钛合金刷盒的特殊力学要求

针对钛合金刷盒这一航空特定铸件，合同规定的力学验收要求为：抗拉强度≥890 MPa、屈服强度≥820 MPa、断后伸长率≥5%、断面收缩率≥10%[[S17]]。该指标不属于民用通用铸造钛合金国标GB/T 6614-2014的最低要求，属于针对该特定航空铸件的定制化合同级非标要求，但成品需符合GJB 2896A-2017航空钛铸件通用标准。该性能要求与铸态退火TC4的典型性能区间匹配，具备可实现性[[S17,S11]]。

## 铸造工艺角色与参数

### 熔炼与浇注工艺

TC4钛合金铸造常用的熔炼工艺包括真空感应凝壳熔炼（ISM）和真空电弧熔炼（VAR）[[S18]]。以真空感应凝壳熔炼重力浇注为例，公开的工艺参数为[[S22]]：

| 工艺参数 | 控制值 |
|----------|--------|
| 炉内真空度 | 始终低于 10⁻¹ Pa |
| 浇注温度 | 1700±10℃ |
| 石墨铸型预热温度 | 300℃ |

钛合金最理想的熔炼与铸造条件是在真空下熔炼、在高压下浇注和凝固，但真空凝壳熔炼浇注工艺不利于消除铸件气体缺陷，主要原因是金属熔炼除气能力较小，且铸件在真空下凝固会促进气泡长大[[S5]]。消除气体缺陷最有效的方法包括[[S5]]：
- 铸型真空除气；
- 保证铸型有良好的透气性并设置合理排气道；
- 采用离心浇注或压力铸造工艺。

### 工艺技术路线

TC4复杂航空构件（如非对称翼型受油管）的熔模精密铸造全流程包括：前置型壳材料优化→射蜡工艺优化→型芯制备→浇注→铸件表面质量检测→缩松缩孔检测→界面反应分析→显微组织观测→热等静压处理→高真空焊接→低成本离心铸造工艺开发[[S13]]。

## 缺陷机理

### 气孔缺陷

气孔的产生程度与浇注温度密切相关。浇注温度过低时，铸件凝固加快，气泡难以从金属液中上浮排除，气孔多集中在铸件上部；过高的浇注温度则加剧液钛与铸型反应，促进气泡生成[[S5]]。在真空凝壳熔炼+真空凝固条件下，金属液除气能力有限，且真空中气泡容易长大，因此该工艺本身不利于消除气孔[[S5]]。

防控措施包括[[S5]]：
- 铸型真空除气；
- 提高铸型透气性、设置合理排气道；
- 采用离心浇注或压力铸造。

### 缩孔与缩松

TC4合金易形成集中缩孔（缩孔体积约1.2~1.3%），缩松倾向较低（约0.75%）[[S9]]。由于钛液比重较小且凝壳炉过热度较低，补缩能力差，冒口补缩作用有限。提高铸件致密度的最有效办法为采用斜度补贴法和离心浇注[[S1]]。

### α污染层（α-case）

α-case是熔融钛与陶瓷铸模材料反应生成的氧稳定脆性层。其深度随铸型预热温度升高而增大，生产中需额外预留加工余量[[S32,S19]]。去除α-case的常用工艺为含氢氟酸的化学铣削浴或喷砂酸洗组合工艺[[S19,S1]]。钛铸件出厂前一般采用吹砂酸洗处理，彻底去除α脆性层[[S5]]。

### 氢脆

当合金中氢含量达到数十ppm级别时，钛中氢以钛氢化物微晶形式析出，依据服役条件不同分为两类[[S21]]：
- **冲击氢脆**：随氢含量升高、温度降低、应变速率升高而加剧；
- **低应变速率氢脆**：在中等温度区间随应变速率降低而更显著。

通过真空除氢退火可消除氢脆风险[[S21,S15]]。

### 热裂与冷裂

TC4作为含β同晶元素的α+β合金，具备优良的抗热裂性能，即使使用无退让性的加工石墨型芯也可获得无热裂的铸件[[S5]]。实际生产中，表面冷裂纹易在α脆性层处萌生并向铸件内部扩展。防控措施包括[[S5]]：
- 采用低导热低热容陶瓷熔模型壳；
- 提高铸型预热温度、降低金属浇注温度；
- 喷砂酸洗彻底去除α脆性层。

### 缺陷汇总

| 缺陷类型 | 主要特征/产生原因 | 防控措施 |
|----------|------------------|----------|
| 气孔 | 浇注温度过低或过高、真空下气泡长大 | 铸型真空除气、提高透气性、离心浇注/压力铸造 |
| 缩孔/缩松 | 集中缩孔明显、补缩能力差 | 斜度补贴法、离心浇注 |
| α-case | 熔融合金与铸型反应生成氧稳定脆性层 | 预留加工余量、化学铣切/喷砂酸洗去除 |
| 氢脆 | 氢含量超标引发钛氢化物析出 | 真空除氢退火 |
| 表面冷裂 | α脆性层在热应力下开裂并向内部扩展 | 低导热型壳、提高铸型温度、去除α层 |

以上汇总基于钛合金铸件常见缺陷的规范性描述[[S2]]及各单项机理的独立证据。

## 后处理与焊接

### 退火处理

TC4铸态合金的标准退火处理制度包括[[S10,S15]]：
- **800℃保温4h、炉冷**：可细化晶界α相及α片层宽度，同步提升铸件室温强度与塑性；
- **843℃保温2h、空冷**：通用航空工业退火规范。

### 真空除氢退火

当铸件在电加工、化学铣切、酸洗、焊接等过程中吸氢导致氢含量超标时，需进行真空除氢退火处理，以消除氢脆隐患[[S15,S21]]。

### 热等静压（HIP）

TC4铸件（ZTC4）的热等静压规范为[[S15]]：

| 参数 | 数值 |
|------|------|
| 介质与压力 | 氩气，100~140 MPa |
| 保温温度 | 920±10℃（低于合金相变点15~20℃） |
| 保温时间 | 2~2.5 h |
| 冷却方式 | 随炉冷却至300℃以下出炉 |

HIP处理后，铸件内部孔洞与疏松被挤压焊合，致密度与力学性能显著改善。研究同时指出，HIP处理可能使铸件强度略有降低但塑性显著改善，这与高温诱使晶粒粗化有关[[S10]]。

### 焊接性能

TC4合金自身可焊性优良，minor铸造缺陷如圆角处气孔可通过TIG焊配合匹配填充金属进行焊补[[S19]]。异种钛合金焊接方面，TC4/Ti17线性摩擦焊接头的抗拉强度高于TC4基金属但低于Ti17，延伸率相较两种母材均有所降低[[S30]]。

## 应用领域

TC4钛合金的应用高度集中于航空航天领域。据统计，我国航空航天用关键部件超过90%由TC4合金铸造而成[[S24,S18]]。典型应用构件包括[[S24,S18,S14]]：
- **航空发动机**：压气机盘与叶片、发动机前支承座、附件泵叶轮、压气机机匣（如F100等型号）；
- **飞机结构**：机身/机翼/尾翼承力件、梁、接头、隔框、机轮轮毂、刹车壳体、襟翼轨道、直升机旋翼桨毂；
- **航天器**：火箭与导弹前体等关键部位；
- **特定铸件**：钛合金刷盒（航空定制铸件，有专项力学性能要求）[[S17]]。

在石化及船舶工业中，TC4也用于制备耐腐蚀泵阀类构件[[S14]]。

## 相关变体与关联

### 同系合金

TC4（Ti-6Al-4V）属于Ti-Al-V系中典型的中等强度α+β型两相钛合金[[S16]]。同系关联合金包括：
- **TC3**：Ti-Al-V系板材用牌号，含较少的铝和钒，具有更好的工艺塑性[[S27]]；
- **BT6（苏联/俄罗斯）**和**IMI-318（英国）** 为TC4的国际对应牌号[[S12]]；
- **Ti-6Al-4V ELI**（超低间隙元素版）：为低温服役条件专门开发，通过严格控制间隙元素含量（尤其是氧）来保证低温韧性与塑性[[S27]]。

### 其他相近钛合金体系

- **TC4-DT**（Ti-6Al-4V损伤容限型变体）：为待补充关系（当前证据未覆盖该牌号的具体对比数据）。
- **Ti-6242**（Ti-6Al-2Sn-4Zr-2Mo）：属于近α型高温钛合金，服役温度高于TC4，但当前证据未覆盖二者在压铸工艺中的直接对比。
- 含β同晶元素的α+β合金（如TC4）总体上比含β共析元素的合金（如Ti-4Al-4Mn、Ti-8Mn）具有更优良的抗热裂性能[[S5]]。

### 改善压铸性能的合金调整

当前证据中未发现针对TC4通过添加稀土或调整成分以专门改善压铸性能的系统性研究结论，该方向标记为**待补充**。
