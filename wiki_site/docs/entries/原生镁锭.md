**原生镁锭**是一种通过硅热法或卤水电解法生产的高纯度重熔用纯金属原料 [[S26]]，其核心用途是作为镁元素的高纯度载体，向铝合金及镁合金熔体中进行元素添加 [[S33,S17]]。该产品明确区别于面向直接铸造使用的镁合金锭（如ASTM B93标准所涵盖的砂型铸造、永久型铸造、熔模铸造及压铸用镁合金锭）[[S28]]，属于有色金属冶炼领域的上游基础原材料。

现行国家标准为 **GB/T 3499-2023**，该标准于2023年12月28日发布，2024年7月1日正式实施，归口于全国有色金属标准化技术委员会 [[S26]]。该版本修改采用了国际标准 **ISO 8287:2021**，实现了国内标准与国际主流规范的对接，并替代了此前的GB/T 3499-2011版本 [[S26]]。在国际上，美国材料与试验协会的 **ASTM B92/B92M** 标准则涵盖了重熔用纯镁锭及镁棒的相应规范 [[S8]]。

## 牌号与化学成分

原生镁锭的等级根据其纯度的不同进行划分，纯度越高，杂质元素含量越低，对最终合金的性能控制越有利 [[S7,S39]]。

现行GB/T 3499标准下，原生镁锭的主要牌号及化学成分要求如下表所示 [[S5,S36]]。镁的含量为100%减去所有列示杂质元素实测总和后的余量 [[S36]]。

| 牌号 | Mg含量 (≥, %) | 杂质元素 (≤, %) | | | | | 杂质总和 (≤, %) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| | | **Fe** | **Si** | **Ni** | **Cu** | **Al** | |
| **Mg99.95** | 99.95 | 0.02 | 0.01 | - | 0.005 | 0.01 | 0.05 [[S5,S16]] |
| **Mg99.90** | 99.92 | 0.04 | 0.01 | 0.001 | 0.01 | 0.02 | 0.08 [[S5,S16]] |
| **Mg99.80** | 99.85 | 0.05 | 0.03 | 0.002 | 0.02 | 0.05 | 0.15 [[S5,S16]] |

此外，标准规定所有牌号的镁锭中，钠（Na）元素的质量分数不得大于0.01%，钾（K）元素的质量分数不得大于0.005% [[S36,S34]]。对于表中未单独规定的其他杂质元素，当其含量大于0.01%时，应计入杂质总和 [[S34,S36]]。

## 物理属性

原生镁锭的核心物理属性源于纯镁，其关键参数如下表所示。

| 属性 | 参数 | 来源 |
| :--- | :--- | :--- |
| 20℃ 密度 | 1.73 - 1.74 g/cm³ | [[S27,S3,S18]] |
| 熔点 (标准大气压) | 648.8 - 650 ℃ | [[S3,S18,S2]] |
| 沸点 (标准大气压) | 1090 - 1107 ℃ | [[S3,S18]] |
| 熔化温度下固态密度 | 约 1.65 g/cm³ | [[S18]] |
| 凝固体积收缩率 | 4.2% | [[S2,S18]] |
| 650℃ 冷却至 20℃ 冷却体积收缩率 | 约 5% | [[S2,S18]] |

纯度对镁锭的性能有明确影响。杂质含量（特别是铁、镍等）较高的低纯度镁锭在与水汽接触时更易发生表面腐蚀，因此储存要求更为严格；而高纯度镁锭表面可形成更稳定的钝化氧化膜，耐蚀性更好 [[S24]]。在合金化应用中，高纯度牌号（如Mg99.95）能够有效避免杂质元素超标导致合金耐腐蚀性能下降，适用于高性能合金的制备；低纯度牌号（如Mg99.80）则可用于对杂质容忍度较高的普通合金，以降低原料成本 [[S7,S39]]。

## 在铝合金制造中的工艺角色

原生镁锭是 **Al-Cu-Mn-Mg** 系高强度铝合金的关键原料之一。镁元素加入铝熔体后，在后续的时效过程中，可与铝、铜形成 Al₂CuMg 强化相，实现析出强化，同时在铝基体中发挥固溶强化作用，显著提升合金的综合力学性能 [[S11,S17]]。

在普通熔炼工艺中，镁锭的加入操作有严格要求：
1.  **加入时机**：应在铝基体熔化，且Al-Cu、Al-Mn等中间合金完全溶解后进行。加入前需清除熔体表面熔渣 [[S15,S1]]。
2.  **加入温度**：推荐在 660℃ ~ 720℃ 的温度区间内加入 [[S15]]。
3.  **加入方式**：采用带孔钟罩或钢笼，将镁锭压入熔体液面以下，以避免镁液漂浮于表面发生剧烈氧化燃烧 [[S9,S15]]。
4.  **烧损控制**：在此常规非浸没式工艺下，镁元素的烧损率约为 5% ~ 15%，即实际回收率为 85% ~ 95%。工业生产配料时，通常将镁元素的实收率计为 95%，以补偿烧损 [[S9,S1]]。

采用配备 **EMP（电磁泵）** 的现代铝合金熔炼系统，可将镁锭直接投入熔体液面下的高速涡旋中，实现浸没式快速熔化。此工艺可将镁元素回收率提升至 96.4% 以上，烧损率低于 3.6%，在理想条件下回收率接近 100% [[S38]]。

## 入厂检验与控制要求

原生镁锭作为原料入厂时，需依据GB/T 3499标准进行验收，以保障后续合金熔炼的质量 [[S26,S29]]。

- **组批规则**：同一熔炼号的镁锭划为一个检验批 [[S34,S20]]。
- **取样方法**：从每批中随机抽取至少3块镁锭，去除表面0.5~1.5mm厚的氧化层后，在指定点位钻孔获取镁屑，并用磁铁彻底清除混入的铁屑，通过四分法缩分得到分析试样 [[S34,S20]]。
- **检验项目**：
    - **化学成分检验**：检测主元素镁及铁、硅、镍、铜、铝等规定杂质元素含量 [[S34,S20]]。仲裁分析按照GB 13748系列标准执行 [[S36,S20]]。
    - **外观检验**：镁锭表面应平整清洁，不得有残留熔剂、夹渣、飞边、氧化黑孔等缺陷 [[S34,S20]]。

## 包装、储运与安全

由于镁的化学性质极为活泼，原生镁锭在包装、储运和安全管理方面有严格规定。

- **包装**：镁锭出厂前需经酸洗、浸油等表面处理，并使用蜡纸包裹。外包装可选择铁桶、尼龙编织袋或纸箱，单包装重量根据包装形式不同控制在 24~108 kg 范围内 [[S34,S31,S23]]。
- **标识**：锭体表面标明熔炼号。外包装上需按品级进行分色标识（特级品为蓝色、一级品为红色、二级品为绿色、三级品为黄色），并注明批号、净重、生产厂等信息 [[S34]]。
- **储存**：必须存放于干燥、清洁、通风的专用库房内，环境相对湿度不得高于60%。垛底应设不低于 100 mm 的垫木进行隔离，严禁与酸、碱等腐蚀性化学品同库储存 [[S10,S34]]。
- **运输**：应使用具备防雨设施的清洁车辆装运，敞车运输时必须覆盖防水苫布，严禁与潮湿或腐蚀性物料混装 [[S31,S10]]。
- **防火安全**：块状原生镁锭在室温下无明显燃烧风险，但温度超过其熔点（约650℃）后会在空气中自燃。镁屑、镁粉的起燃温度仅约400℃，且与水反应会生成氢气易引发爆炸 [[S41,S13]]。因其属于第4.1类易燃固体 [[S6]]，扑救镁火必须采用干沙或专用D级灭火剂覆盖，严禁使用水或二氧化碳灭火器 [[S6,S13]]。原生镁锭的反应活性高于镁合金锭 [[S4]]。

## 与其他标准的关联

GB/T 3499《原生镁锭》作为基础原料标准，被多项下游压铸及铸造工艺标准所直接引用，作为限定生产所用镁原料品质的依据。例如，**GB/T 25748《压铸镁合金》** 标准即以原生镁锭作为指定原料，而 **GB/T 40809《铸造铝合金半固态流变压铸成形工艺规范》** 的参考文献中也直接列出了该标准 [[S29]]。
