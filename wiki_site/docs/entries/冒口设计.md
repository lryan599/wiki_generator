## 概述

冒口（英文 riser / feeder / feeder head）是铸型内用于储存金属液的辅助储腔，其核心功能是在铸件凝固过程中补充金属液，抵消体积收缩，避免缩孔、缩松缺陷，同时兼具排气、集渣的作用，并非最终铸件的组成部分[[S61,S75,S68]]。冒口设计（riser design / feeder design）即为根据铸件结构、合金收缩特性及浇注系统等因素，确定冒口的位置、类型、尺寸和数量，并校核其补缩能力的工艺设计活动[[S61]]。该词条主要适用于需要顺序凝固和重力补缩的砂型铸造、金属型铸造、低压/差压铸造等工艺；在高压铸造中，传统冒口极少使用，而是以增压和局部挤压等方式实现补缩[[S68,S1]]。

## 冒口补缩的基本条件

为获得致密铸件，冒口设计须同时满足三个基本条件[[S55,S76,S68,S71]]：

1. **凝固时间条件** – 冒口的凝固时间应大于或等于铸件被补缩部位的凝固时间，即冒口模数不低于铸件模数；
2. **补缩液量条件** – 冒口内应储备足够金属液，以补偿铸件的液态收缩、凝固收缩以及浇注后型腔扩大的体积；
3. **补缩通道条件** – 在整个凝固期间，冒口与热节之间必须保持通畅的液态补缩通道，扩张角应朝向冒口。

## 冒口分类

冒口可按两大类层级进行划分[[S15,S23,S61]]。

### 普通冒口

| 分类依据 | 类型 | 特征 | 典型应用场景 |
|----------|------|------|--------------|
| 顶部是否敞开 | 明冒口 | 贯通上型，与大气相通；排气、集渣效果好，便于观察液面；模数通常取1.2倍铸件模数[[S33,S3]] | 单件小批中大型铸钢件、上部可开设敞口的铸件 |
| 顶部是否敞开 | 暗冒口 | 整体被型砂包覆，不伸出铸型顶面；布置灵活，可贴近热节设置[[S64,S3]] | 机器造型、可锻铸铁、球墨铸铁及中小批量灰铸铁件 |
| 布置位置 | 顶冒口 | 设置在铸件最高位置，压头大，排气、排渣优异[[S3,S37]] | 绝大多数有上部热节的铸件 |
| 布置位置 | 侧冒口 | 设置在铸件侧面；用于顶部无法布置冒口的侧部热节[[S3,S37]] | 孤立侧部热节、低位热节配合冷铁使用 |
| 布置位置 | 贴边冒口 | 介于顶冒口与侧冒口之间，适用于铸件中下部热节[[S3,S23]] | 下部肥大的热节 |

### 特种冒口

| 类型 | 工作方式 | 主要特点 |
|------|----------|----------|
| 大气压力冒口 [[S61,S91]] | 通过透气砂芯或吊砂延长大气压力作用 | 可显著提高暗冒口的有效补缩距离 |
| 压缩空气冒口 [[S15,S23]] | 通入压缩气体直接加压 | 提高补缩压力，适用于大型铸件 |
| 发气压力冒口 [[S61,S15]] | 在冒口内设置气弹或发气物质产生压力 | 建立压力增加补缩效果 |
| 保温冒口 [[S7,S61]] | 采用低热导率的保温材料制成冒口套 | 可缩小冒口体积约60%，延长凝固时间 |
| 发热冒口 [[S7,S61]] | 冒口套中含铝热剂等发热材料，接触金属液后放热 | 补缩效率可超过30%，工艺出品率提升10%~25% |
| 电热冒口 [[S61]] | 利用电极放电电弧加热 | 多用于大型铸钢件和钢锭 |

## 冒口设计理论基础

### 模数（热模数）

模数 $M$ 定义为铸件（或冒口）体积 $V$ 与散热表面积 $A$ 的比值，即 $M = V/A$ [[S18,S83,S10]]。其物理意义是等效凝固厚度，模数越大，凝固时间越长。依据 Chvorinov 平方根定律，凝固时间 $t = (M/k)^2$，其中 $k$ 为凝固系数（与材质、铸型、浇注温度相关）[[S13,S83]]。该规则由捷克工程师 Nicolas Chvorinov 于1940年提出，奠定了现代冒口定量设计的基础[[S13,S18]]。

### 顺序凝固与补缩距离

顺序凝固要求铸件由远离冒口的薄壁区域向靠近冒口的热节逐步推进，最终使全部未凝固液相集中于冒口中，从而将收缩缺陷转移至冒口内部[[S43,S71]]。冒口的有效补缩距离定义为冒口区与末端区之和，超出该距离则产生轴线缩松[[S43,S22,S27]]。设置冷铁、补贴以及使用发热/保温冒口均可延长有效补缩距离，减少所需冒口数量[[S6,S43]]。

不同合金的冒口有效补缩距离经验值如下（$T$ 为截面厚度）[[S5]]：

| 合金类型 | 板件补缩距离 | 杆件/备注 |
|----------|--------------|-----------|
| 铸钢 | 约 4~6$T$ | 杆件 2~3$T$ |
| 灰铸铁 | 4~8$T$ | — |
| 球墨铸铁（壁厚10~50 mm） | 3~4$T$ | — |
| 窄结晶区间铝合金（如 ZL102） | 约 4.5$T$ | — |
| 宽结晶区间铝合金 | 约 2$T$ | — |
| 镁合金 | 几乎无稳定自由补缩距离 | 必须配合冷铁 |

## 关键设计参数

### 模数匹配

模数法是应用最广的冒口设计方法，核心要求为[[S18,S8,S74]]：
$$M_r > M_n > M_c$$
其中 $M_r$、$M_n$、$M_c$ 分别为冒口、冒口颈、铸件被补缩处的模数。工程上通用：
- 冒口模数 $M_r = 1.2 M_c$（对顶明冒口）；
- 侧暗冒口时 $M_c:M_n:M_r = 1:1.1:1.2$；
- 冒口颈长度经验值 $L = 2.4 M_c$ [[S8,S36]]。

采用 Chvorinov 时间法则校核时，通常取 $t_{riser} \ge 1.25\,t_{casting}$ [[S51]]。

### 冒口位置选择

基本原则[[S43,S22,S76]]：
1. 就近设置在热节的上方（顶冒口）或旁侧（侧冒口）；
2. 尽量放在铸件最高、壁厚最大的部位，争取一个冒口覆盖多个热节；
3. 布置在需要后续加工的表面上，减少清整工作量；
4. 避开应力集中和重要受力部位，防止组织粗大降低力学性能；
5. 低位热节应采用冷铁隔离，避免高位冒口跨区补缩导致的缩松；
6. 结合补贴设计保证补缩通道通畅。

### 冒口形状

- 球形冒口体积/散热面积比最大，凝固时间最长，补缩效率最优[[S74]]；
- 圆柱形冒口加工方便，应用最广[[S74]]；
- 腰形（椭圆形）冒口适用于长条状热节，同体积下补缩覆盖范围更大[[S74]]。

### 冒口尺寸确定

#### 热节圆法（适用于铝、镁合金金属型铸造）

- 明冒口根部直径 $D = (1.2\sim1.5)d$（$d$ 为热节圆直径）[[S2]]；
- 暗冒口 $D = (1.2\sim2.0)d$[[S2]]；
- 需要横向补缩时 $D = (2\sim4)d$[[S92,S2]]；
- 冒口高度：顶冒口 $H = (0.8\sim1.5)D$，侧冒口 $H = (2\sim3)D$，暗冒口 $H = (1.2\sim2)D$[[S2,S92]]；
- 铝合金高要求铸件（如铝硅合金）冒口直径原则上不超过 100 mm，超限时需采取辅助措施[[S92]]。

#### 铸铁件分段比例法[[S11]]

| 铸件规模 | 冒口直径 $D_r$ | 冒口高度 $H_r$ |
|----------|----------------|----------------|
| 小件 | $D_r = (1.2\sim2.0)\delta_c$（$\delta_c$ 为壁厚或热节圆直径） | $H_r = (1.0\sim1.5)D_r$ |
| 中件 | $D_r = (1.0\sim1.2)\delta_c$ | $H_r = (1.2\sim1.8)D_r$ |
| 大件 | $D_r = (0.6\sim1.0)\delta_c$ | $H_r = (1.5\sim2.0)D_r$ |

#### 冒口颈设计

冒口颈（riser neck）是连接冒口与铸件的关键结构，其模数需介于铸件模数与冒口模数之间[[S36,S8]]。典型要求：
- $M_n = 1.1 M_c$；
- 冒口颈长度 $L = 2.4 M_c$；
- 截面形状应保证在铸件完全凝固后迅速凝固断开，以降低冒口切割去除工作量[[S8,S36]]。

### 铸造工艺出品率

冒口金属消耗量通常占浇注总重的 25%~50%[[S42]]，工艺出品率一般控制在 60% 左右[[S74]]。特定承压阀门铸件标准规定更加严格：碳钢及铬钼钢铸件出品率 ≤48%，奥氏体不锈钢 ≤45%，镍基合金砂模承压铸件 ≤42%[[S62]]。

## 冒口套与覆盖剂

### 冒口套分类与材料

按功能冒口套分为三类[[S38]]：
- **纯保温冒口套**：以漂珠、珍珠岩、硅酸铝纤维等低导热材料为主，适用于大尺寸冒口和低熔点非铁合金；
- **发热保温冒口套**：同时含保温组分和铝热发热组分（铝粉+氧化铁系等），应用最广，适合绝大多数普通铸件；
- **纯发热冒口套**：无保温组分，发热功率高，适合高压造型线中的小尺寸孤立热节“点补缩”。

发热冒口套的发热剂一般为铝热剂，氧化剂采用钾盐（KMnO₄、KClO₃、KNO₃），助熔剂可用冰晶石等氟化物；保温组分可采用漂珠、膨胀蛭石、岩棉等；耐火骨料为高铝矾土、硅砂；粘结剂可选树脂、水玻璃、糊精等，要求无毒无污染、不影响铸件表面质量[[S38,S24,S31]]。

冒口套厚度 $δ$ 一般取 $δ = (1\sim1.5)M_e$（$M_e$ 为有效模数），当 $δ/M_e > 1$ 后模数扩展系数 $E$ 提升极小，市售普通保温冒口的 $E$ 典型值为 1.3~1.4，补缩效率可达 25%~45%[[S54]]。采用模数扩展系数 $f$（MEF）后，带保温/发热套的冒口模数修正公式为：$M_r' = 1.2 M_c / f$，小尺寸保温套 $f=1.8$，大尺寸 $f=1.3$[[S9,S80]]。

### 发热保温覆盖剂

明冒口顶部热损失以辐射为主。覆盖剂在金属液表面形成熔融层、烧结层、过渡层和膨胀保温层四层结构，兼具保温、缓慢放热和造渣排渣功能[[S79]]。常用配比示例（质量分数）：稻壳灰 40%、漂珠 20%、低碳石墨 25%、粉煤灰 5%、木炭 5%、CaF₂+Na₂CO₃ 5%；要求能在液面快速铺展、随液面同步下降并不结硬渣壳[[S79,S31,S45]]。

## 设计流程与 CAE 优化

通用冒口设计的一般步骤如下[[S22,S55]]：
1. 确定冒口安放位置；
2. 计算铸件被补缩处的模数 $M_c$；
3. 根据模数法或比例法确定冒口和冒口颈模数；
4. 确定冒口形状和几何尺寸，优选标准系列；
5. 基于有效补缩距离校核冒口数目；
6. 校核补缩能力（体积收缩补偿）；
7. 通过凝固过程数值模拟（CAE）验证和迭代优化。

CAD/CAE 集成优化流程[[S12,S70,S34]]：导入三维铸件模型 → 预模拟定位热节和收缩量 → 参数化冒口设计 → 全流程凝固仿真 → 提取缩孔缩松分布、孤立液相区验证 → 迭代调整尺寸/位置 → 输出工艺文件。智能化工具可自动完成模数批量计算和标准冒口选型，大幅提升设计效率和工艺出品率[[S34]]。

## 缺陷与冒口设计不当的关联

| 缺陷类型 | 与冒口设计相关的成因 | 预防措施 |
|----------|----------------------|----------|
| 缩孔、缩松 | 冒口模数不足、补缩液量不够、补缩通道提前封闭或冒口数量不足[[S49,S84]] | 模数法保证 $M_r=1.2M_c$、$M_n=1.1M_c$；配合冷铁和补贴建立顺序凝固[[S35,S14,S68]] |
| 冒口根部裂纹（热裂） | 冒口与铸件连接处应力集中，凝固末期固相体积分数 85~95% 时枝晶间液膜破裂[[S29,S85]] | 避开应力集中位置，控制温度梯度 |
| 气孔（气缩复合缺陷） | 冒口顶部排气设计缺失，型腔内气体无法上浮排出，封闭在冒口与铸件结合区域[[S69]] | 设置明冒口或出气通道，使用透气芯 |

冒口颈尺寸失当是常见失效模式：冒口颈偏大时，冒口内缺陷可能连通至铸件本体；冒口颈偏小时，补缩通道提前封闭，铸件热节得不到补缩[[S49]]。通过优化冒口尺寸、确保冒口凝固时间晚于被补缩热节 25% 以上，可将缺陷最终转移至冒口内部[[S51]]。

## 特殊工艺中的冒口设计

### 低压铸造

常规低压铸造几乎不设独立冒口，补缩主要通过底部上升的浇注系统直接完成；仅当铸件顶部厚大区域有特殊需求时才设置暗冒口[[S48,S58]]。

### 差压铸造

利用全流程压力场强化补缩，一般仅配置暗冒口，配合冷铁和底部缝隙式浇注系统实现顺序凝固，无需大尺寸明冒口即可抑制缩松[[S50]]。

### 金属型铸造

冒口需完成补缩、排气、集渣多功能；冒口顶部需设计独立排气孔，保证型腔气体顺利排出，防止气孔与缩松共生[[S40]]。铝、镁合金金属型铸造中，优先选用明冒口以利用大气压力提升补缩效率；冒口尺寸可比砂型冒口适当缩小[[S87]]。

### 高压铸造（压铸）

高压压铸几乎不使用传统冒口，原因包括[[S68]]：
- 充型速度极高（可达数 m/s），整体凝固周期极短，重力补缩无法及时供给；
- 设置独立冒口会显著降低工艺出品率，不符合高效生产要求。

替代补缩方式主要有两种：
1. **整体增压补缩**：充型结束后短时间内将压射压力提升，通过已充满的内浇道强制补偿全局体积收缩[[S1,S39]]。
2. **局部挤压（挤压销）**：在铸件厚大热节对应的模具位置设置独立挤压销，在金属液处于半固态的窗口期启动挤压，将额外金属液压入热节区域[[S67,S44]]。

挤压销核心设计参数[[S1,S89]]：
- 挤压直径：典型 $\varphi5\sim\varphi25$ mm；
- 挤压行程：$L = n\beta W/(\pi R^2)$，其中铝合金凝固收缩率 $\beta\le6.5\%$，安全系数 $n$ 一般取 3；
- 动作窗口期：高速压射结束后 0.1~2 s 启动挤压，保压 1.5~2 s 后退回；启动过早无法留下有效补缩进量，过晚则热节已凝固导致失效。

## 标准与规范

国内主要标准：
- **GB/T 5611-2017《铸造术语》**：明确了冒口、明冒口、暗冒口、保温冒口、发热冒口、大气压力冒口等术语定义与功能[[S30,S61]]；
- 明冒口定义亦纳入《机械工程名词 第二分册》[[S33]]；
- **JB/T 11484-2013** 及 **GB/T 35740-2017**：规定了阀门铸件的极限工艺出品率要求，间接约束冒口设计[[S62]]。

国际标准方面：
- 冒口常用术语：riser（美式）、feeder（英式）[[S47,S60]]；暗冒口别名 blind riser / closed riser，明冒口 open riser[[S64,S47]]。
- **ASTM A781/A781M** 补充要求 S52 明确规定了去除浇冒口后残留凸起/凹陷的允许公差[[S86,S53]]。
