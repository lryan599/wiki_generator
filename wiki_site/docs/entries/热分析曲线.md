## 概述

热分析曲线（Thermal analysis curve），在铸造与压铸领域亦称冷却曲线（Cooling curve）或凝固热分析曲线，是指利用热分析仪器记录到的合金在凝固过程中温度随时间变化的关系曲线[[S21,S54]]。该曲线通过分析温度-时间（T-t）关系及其一阶、二阶导数曲线，可确定合金的液相线温度、固相线温度及各相变节点温度，判断不同结晶反应阶段，并预测石墨组织形态，是压铸/铸造生产炉前快速检测金属液质量的核心技术手段[[S21,S54,S17]]。

## 基本原理

液态金属从完全液态冷却、逐步凝固至全固态的过程中，将同时释放自然冷却热和相变潜热。当周围传热介质的热传导特性保持稳定时，实测得到的温度随时间变化过程可直接映射金属液的热量释放规律[[S54,S17]]。因此，基于冷却过程中温度与时间的关系曲线，便可间接推导合金的化学成分、评估球化/孕育处理效果，并预判铸件的金相组织与最终力学性能[[S54]]。

微分热分析法在普通冷却曲线的基础上进行微分处理，可测量合金结晶过程中的温度变化速率，相比简单热分析能更精准地判定相变起止点以及微小的热变化[[S16,S17]]。

## 曲线类型与解读

### 冷却曲线（T-t）

典型T-t冷却曲线在合金凝固过程中整体呈温度随时间逐步下降的趋势。在经过各相变反应阶段时，因相变潜热释放会出现斜率变缓、温度平台甚至局部温度回升的特征。以亚共晶铝合金为例：过热金属液快速降温至液相线后初晶相开始形核，潜热大量释放导致降温速度显著放缓；随后进入缓慢降温阶段；共晶反应阶段出现明显温度平台；相变完全结束后冷却速度再次加快[[S42,S39]]。

### 一阶导数曲线（dT/dt）

一阶导数曲线由原始温度采样序列中相邻温度差值除以对应时间差值计算得到，反映瞬时冷却速率[[S44,S40,S4]]。在合金无明显相变的均匀冷却阶段，dT/dt通常为接近恒定的负值；当发生相变释放潜热时，dT/dt向大于当前冷却速率的方向偏移，出现局部峰值甚至正向过零的特征。其曲线与零基线围成的积分面积与该相变释放的总结晶潜热成正比，可用于定量计算各析出相的体积分数[[S39,S53]]。T-t曲线的拐点对应的一阶导数值为零，dT/dt曲线上的过零点与T-t曲线上的拐点一一对应[[S45]]。

### 二阶导数曲线（d²T/dt²）

二阶导数曲线是对dT/dt数值序列再做一阶差分运算得到的温度对时间的二阶导数。该类曲线可进一步放大原始冷却曲线中信号极其微弱的相变特征，实现形核起始温度的精准判定，识别低占比复杂金属间化合物相的析出时刻[[S44,S37,S41]]。直接基于未降噪原始温度数据计算得到的二阶及以上阶次导数信号噪声水平极高，行业通用方案是采用16位工业级模数转换器采集原始数据，并配合专用数字平滑降噪算法，才能获得有效的用于判定凝固特征的高阶导数曲线[[S41]]。

### 再辉现象

再辉现象的物理机制为：合金熔体冷却到低于某一相的平衡形核温度发生形核后，结晶释放的潜热速率超过了合金向环境的散热速率，导致冷却过程中温度停止下降甚至出现小幅回升。亚共晶合金初晶α-Al形核生长阶段、共晶反应阶段均可能出现再辉，再辉的幅度可直接反映熔体形核能力的强弱[[S42,S51,S13]]。

## 关键特征参数

热分析曲线上的核心特征参数及其物理含义如下表所示：

| 特征参数 | 符号 | 含义 |
|----------|------|------|
| 液相线温度（初晶温度） | T_L | 合金中初生固相开始析出的温度 |
| 共晶形核温度 | T_EN | 共晶相开始形核的温度 |
| 共晶过冷最低温度 | T_FU | 共晶过冷达到的最低温度 |
| 共晶开始温度 | T_ES | 奥氏体生长结束、共晶凝固正式启动的温度点 |
| 共晶最高生长温度 | T_ER | 共晶再辉达到的最高平台温度 |
| 共晶结束温度 | T_EE | 共晶反应完全完成的温度点 |
| 固相线温度 | T_S | 全部液相完全转变为固相的温度 |
| 共晶过冷度 | ΔT_EU = T_EN - T_FU | 反映形核物质充足程度的指标 |
| 再辉幅度 | ΔT_R = T_ER - T_FU | 可评估熔体细化、变质处理效果 |
| 共晶平台长度 | t_EE - t_ES | 共晶相变过程的总持续时长 |

## 铸造领域的应用

### 球墨铸铁球化率判定

热分析法测定球墨铸铁球化率共有三类成熟工程方法[[S28]]：
1. 直接读取普通冷却曲线的特征值
2. 基于不同凝固组织热导率差异的相对热导率法
3. 微分热分析法

直接利用冷却曲线判定的原理为：经球化处理后的铁液初生奥氏体开始析出温度T_AL、共晶温度T_E均显著低于普通灰铸铁；当共晶回升温度ΔT2一定时，球化率随共晶过冷度ΔT1增大而升高[[S28]]。

采用相对热导率法判定球化状态有明确工业阈值[[S55]]：
- ΔT < 3.5K：球化合格概率极高（绿区）
- ΔT > 11K：判定球化失败（红区）
- 3.5K < ΔT < 11K：需补充金相检验判定（黄区）

### 石墨形态预测

使用含稀土球化剂的铸铁，热分析曲线特征温度可直接判定石墨形态[[S55,S23]]：

| 条件 | 预测石墨形态 |
|------|-------------|
| ΔT > 6K 且 T_ER > 1423K | 片状石墨 |
| ΔT > 6K 且 T_ER = 1413~1418K | 蠕墨 |
| ΔT极小且T_ER、T_EU均 < 1413K | 球状石墨 |

大型蠕墨铸铁件2t级生产实测验证[[S9,S23]]：
- 共晶再辉率 > 35K/min：蠕虫状石墨 > 85%
- 再辉率 < 10K/min：完全片状石墨
- 再辉率20~35K/min：石墨形态不确定，可能为蠕虫状或球状
- 球墨铸铁孕育后需将再辉率阈值调整至 < 18K/min 方可保证球化率 > 90%

### 灰铸铁孕育效果评估

热分析法是灰铸铁生产中定量评估孕育效果的成熟手段。相同冷却条件下GG25铸铁共晶过冷度ΔT（K）与共晶团数Z（个/cm²）经1036根标准试棒生产验证满足线性关系Z=716-22ΔT，即过冷度越小共晶团数目越多，形核效果越好[[S20]]。

### 铸铁化学成分快速评估

经大量生产数据回归得到成熟工程回归公式[[S18,S43]]：

- 碳当量：CE = 8.14 - 7.970×10⁻³·T_L + 4.75×10⁻³·T_ER - 1.10×10⁻³·DTE
- 共晶度：Sc = 2.06 - 2.13×10⁻³·T_L + 1.28×10⁻³·T_ER - 9.4×10⁻⁴·β
- 含碳量：C = 11.57 - 6.82×10⁻²·T_L

不同车间需基于自身稳定生产工况校准回归系数以保证检测精度[[S43]]。

### 缩松倾向预测

热分析冷却曲线的形状特征可直接预测球墨铸铁缩松倾向：再辉阶段石墨生成量越少，再辉后降温阶段的石墨化膨胀填充能力越强，越有利于减少缩松缺陷；亚共晶度过大的球墨铸铁石墨球分布不均、石墨化膨胀不足，缩孔缩松发生概率显著升高[[S6,S18]]。

### 铝合金质量控制应用

在Al-Si压铸合金领域，热分析是评估孕育细化效果的核心炉前手段：初晶过冷度ΔT（初晶析出前最低温度T_NI与再辉后最高温度T_NR的差值）随细化效果增强而显著减小。工业实践中Ti质量分数达到0.03%时可保证获得优异的细化效果[[S46,S36]]。

热分析法在铝合金压铸质量控制场景中已开展多方向工程应用验证[[S33,S36]]：
- 基于Al-Si-Mg合金550℃三元共晶转变的结晶潜热特征快速评估Mg元素烧损导致的成分偏差
- 通过凝固过程热特征计算固相分数评估铝硅合金的含氢量
- 基于夹杂物的高热容特征定量检测铝合金液中的夹杂物含量
- 在线预测Al-Si压铸合金的晶粒细化程度与共晶变质效果

## 测试设备与操作规程

### 系统组成

铸造热分析系统核心组成包含微型计算机系统、取样杯、样杯支座/取样器、夹持器、测温热电偶及其补偿导线、A/D转换器等整套部件，常规炉前检测可在2~3min内完成一次完整测试[[S8]]。

### 热电偶要求

测温热电偶是铸造热分析系统的核心精度影响部件[[S8,S5]]：
- **材质选择**：铸钢生产多选用铂铑10-铂、铂铑30-铂铑6热电偶；铸铁和非铁合金生产常采用镍铬-镍硅热电偶
- **电极要求**：热电极直径0.5~0.7mm，热端采用氩弧对焊且焊点不得存在氧化夹杂
- **精度要求**：1373K下测温误差不得大于±1.2K；部分工业应用要求超精密级镍铬-镍硅电偶在1100℃时电动势偏差不超过±0.05mV，1000℃时测温精度可达±1℃

### 样杯要求

样杯直接影响试样与环境间的传热，其尺寸、形状、壁厚、热电偶安装位置、造型材料等均能影响冷却曲线的形状和最终的测量结果[[S8,S52,S25]]。样杯技术要求包括：

- 常温抗拉强度大于150MPa，金属液浇注后壳体不得出现裂纹
- 液态金属在样杯中凝固过程需近似满足牛顿冷却条件
- 造型材料在热分析工作温度范围内不与合金元素发生反应
- 常用铸铁样杯以酚醛树脂或呋喃树脂为黏结剂，分为圆形、方形两类
- 热电偶安装位置需尽可能靠近试样热节中心
- 铸铁试样重量通常控制在300~450g区间
- 铸铁检测时常用激冷方案：内壁激冷涂料或底部碲基激冷块，底部激冷块适用温度范围可达1230~1350℃，可保证大孕育量的过共晶铸铁也能实现白口凝固

### 数据采集规范

标准化采集流程[[S37,S53,S50]]：将经过校准的K型热电偶嵌入预热后的采样坩埚中心位置，连接高速数据采集设备，设置20Hz至100Hz区间内的采样频率连续采集温度-时间点；单次测试完成后至少进行3次平行样验证；全程控制浇铸温度、浇铸高度、冷却工装状态完全一致，保障测试结果的可复现性。

### 炉前检测精度

炉前现场热分析法的常规工业测试精度为[[S5,S32]]：
- 碳当量 ±0.05%
- 碳含量 ±0.05%
- 硅含量 ±0.1%
- TA100型熔铝热分析系统测温相对误差可控制在0.1%以内

## 局限性与误差来源

### 主要误差来源

炉前现场热分析方法的主要误差来源包括[[S47,S5]]：
- 热电偶自身固有精度偏差
- 热电偶测温迟滞效应
- A/D转换器采样精度误差
- 样杯传热条件波动引入的偏差
- 测试所用数学模型随炉料配比和熔炼工艺变化未及时修正带来的拟合偏差

### 适用范围局限

炉前冷却曲线热分析法存在明确适用范围局限性[[S15,S24]]：
- 仅适用于具有共晶转变一次结晶特征的铸造合金，无法对无明显共晶平台的复杂多相合金进行高精度成分预测
- 传统牛顿冷却模型在计算相变潜热和凝固分数时存在较大系统偏差

## 与实验室热分析方法的区别与联系

炉前冷却曲线热分析法与实验室DTA（差热分析）、DSC（差示扫描量热法）的核心差异如下表所示[[S31,S22,S57,S11]]：

| 对比维度 | 冷却曲线法（CCA） | 差热分析（DTA） | 差示扫描量热法（DSC） |
|----------|-------------------|-----------------|----------------------|
| 测量原理 | 仅采集单一样品温度-时间数据，无需参比物 | 记录试样与惰性参比物的温差随温度/时间的变化 | 测量进出试样的热流量/功率差 |
| 定量能力 | 仅通过特征点拟合得到工业近似成分参数，无法直接测定焓变 | 可得到相变温度特征值，仅能定性半定量相变焓 | 曲线下积分面积直接与相变总热量成正比，可高精度定量相变焓和比热 |
| 操作场景 | 无需精密程序控温，炉前3min即可完成检测，设备成本低 | 需精密程序控温，测试周期长，设备造价高 | 需精密程序控温，测试周期长，最高工作温度受限于铝合金熔化温度（DSC低于DTA） |
| 应用定位 | 生产现场炉前快速质量检测 | 实验室研发 | 实验室定量热物性测定首选 |
