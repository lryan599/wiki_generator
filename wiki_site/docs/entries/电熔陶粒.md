## 概述

电熔陶粒，国家标准名称为**铸造用球形陶瓷砂**（Spherical ceramic sand used for foundry），是以Al₂O₃-SiO₂为主要矿物成分，粒径覆盖0.020mm～3.350mm的人工合成球形耐火颗粒物。[[S10,S26]] 商品名称包括电熔陶粒、电熔陶瓷砂、电熔陶粒砂、宝珠砂、钰珠砂、珠宝砂、熔融陶瓷砂等，在铸造材料分类体系中归属**非石英系人工合成铸造原砂**。[[S10,S1,S8]]

该材料以铝矾土为原料，经电弧炉2000℃以上高温完全熔融后，通过高压空气喷射冷却直接形成球形颗粒，即**电熔-风淬一步成球工艺**，再经筛分分级得到不同粒度规格的成品。[[S10,S1]] 其核心矿物相为人工合成莫来石，属于绿色环保型铸造材料，可适配碳钢、合金钢、铝合金、镁合金等多类铸件的生产。[[S1,S8]]

## 化学成分与制备原理

### 典型化学成分

国内主流商用牌号电熔陶粒的化学成分限制要求如下[[S21,S4]]：

| 组分 | 含量（质量分数） |
|---|---|
| Al₂O₃ | ≥75% |
| SiO₂ | 5%～20% |
| Fe₂O₃ | ≤5% |
| TiO₂ | ≤5% |
| 其他 | 微量 |

### 制备工艺

电熔陶粒以铝矾土为原料，完整的制备工艺流程为[[S10,S1]]：铝矾土经预煅烧处理后送入电弧炉，在2000℃以上高温下完全熔融，随后通过高压空气喷射处理使熔融液滴在冷却过程中直接形成球形颗粒物，再经筛分分级得到不同粒度规格的成品。该工艺属于电熔-风淬一步成球技术，与烧结法莫来石陶粒相比，产品的致密度、耐火度和抗破碎性能更为优异。[[S12,S24]]

## 核心物理属性

电熔陶粒的关键量化物理指标如下[[S21,S4]]：

| 技术指标 | 参数值 |
|---|---|
| 粒形 | 球形，角形系数≤1.1 |
| 堆集密度 | 1.95～2.05 g/cm³ |
| 耐火度 | ≥1790℃（实际区间1800～2000℃） |
| 热膨胀率（1000℃, 10min） | 0.13% |
| pH值 | 7.6（中性） |
| 显微结构 | 致密无缺陷的人工合成莫来石相球形颗粒 |

电熔陶粒的结构致密、强度高，循环使用过程中极少发生破碎，年砂耗损耗低于5%，可大幅减少生产环节的粉尘排放与废砂产出量。[[S21,S4]]

## 在压铸及相关工艺中的应用

### 铸造造型与制芯

电熔陶粒的基本应用场景是作为砂型铸造的造型、制芯用砂，其球形结构具有良好的流动性和填充性，能够有效填充铸造模具，提高铸件的成形质量，减少粘结剂使用量。低热膨胀率特性使生产的铸件尺寸精度高，破裂及表面缺陷少。[[S8,S21,S4]]

### 压铸涂料耐火骨料

电熔陶粒可作为压铸涂料的耐火骨料使用。其高耐火度（≥1790℃）、低膨胀率（0.13%）的特性可有效减少铸件粘砂和表面缺陷，适配铝、镁等多种压铸合金的生产要求。[[S8,S10]]

### 保温冒口材料

电熔陶粒可用于制备压铸保温冒口，作为高耐火度骨料提升冒口在高温下的结构稳定性，延长金属液凝固补缩时长，从而提升铸件工艺出品率。保温冒口套一般由耐火材料、保温材料、发热材料和粘结剂组成，其中耐火骨料用于提高冒口套在高温时的强度和耐火度。[[S6,S15]]

### 压铸件喷砂清理

电熔陶粒可作为压铸铝合金、镁合金铸件的喷砂清理介质，替代传统含硅磨料实现铸件表面氧化皮和毛刺的去除，可有效避免游离硅尘危害。[[S28,S1]] 在干喷砂装置或具备磨料自循环回收功能的双室式喷砂机中均可使用。[[S3,S22]]

## 性能优势与介质对比

### 与传统喷砂介质的对比

《航空制造工程手册──特种铸造》中收录的弹丸材质对比表提供了不同介质性能的横向参考[[S20]]。相较于钢丸、棕刚玉、玻璃珠等传统压铸处理介质，电熔陶粒的核心优势体现在以下方面[[S2,S21]]：

- **无嵌入缺陷**：球形结构无尖锐棱角，冲击软质压铸铝/镁合金表面时不易发生磨料嵌入缺陷；
- **循环寿命长**：致密结构强度高，年砂耗损耗低于5%，循环回用寿命远高于棱角形棕刚玉；
- **绿色环保**：无游离二氧化硅成分，可减少矽尘职业健康风险，属于环保型绿色磨料。

| 项目 | 电熔陶粒 | 棕刚玉 | 钢丸/玻璃珠 |
|---|---|---|---|
| 颗粒形状 | 球形，角形系数≤1.1 | 尖角形（经破碎获得） | 球形/近球形 |
| 是否含游离SiO₂ | 无 | 无（Al₂O₃基） | 玻璃珠含SiO₂ |
| 年砂耗 | <5% | 较高（棱角易破碎） | 依材质不等 |
| 嵌入铸件风险 | 低 | 较高 | 钢丸有铁污染风险 |

（注：对比信息综合自[[S2,S4,S21]]，弹丸材质对比表来源[[S20]]）

### 使用局限性

电熔陶粒的Al₂O₃含量约为75%，低于电熔刚玉的≥90%，在极端高温苛求工况下耐火性能存在差距；同时其堆集密度（1.95～2.05 g/cm³）虽较锆英砂、铬矿砂低，可降低砂铁比，但喷砂冲击动能也因此偏低。[[S12,S4]]

## 相关概念辨析

### 电熔陶粒与电熔刚玉

电熔刚玉（含白刚玉和棕刚玉）是Al₂O₃质量分数≥90%的α-Al₂O₃基原砂，经2000～2400℃电熔后破碎筛分得到尖角形颗粒，无球形化步骤，多用于大型铸钢件型面涂层和熔模铸造耐火填料。电熔陶粒Al₂O₃含量约为75%，含莫来石和SiO₂相，经电熔-风淬一步制成球形结构，用于整体造型制芯。[[S10,S12,S18]]

### 电熔陶粒与莫来石陶粒

常规莫来石陶粒多采用预合成莫来石粉经造粒、1600℃左右烧结工艺制备，部分产品存在孔隙率较高的缺陷。电熔陶粒是电熔完全熔融后风淬得到的致密莫来石相球形颗粒，致密度、耐火度和抗破碎性能均优于烧结莫来石陶粒。[[S12,S24,S1]]

### 电熔陶粒与陶瓷丸

陶瓷丸多为高Al₂O₃陶瓷粉体经干压成型后烧结制备，多用于表面喷丸清理和研磨介质场景，颗粒规格较小且硬度更高，不适合作为大粒度铸造造型砂使用。两者虽同属陶瓷球形颗粒范畴，但制备工艺和应用场景完全不同。[[S8,S14]]

## 微观形貌特征

电熔陶粒颗粒全部呈规则球形、表面光滑、无凹凸脉纹。扫描电镜下可观察到其致密无裂纹的表面形貌，这与其优异的结构强度和低破碎率直接相关。[[S21,S4]]
