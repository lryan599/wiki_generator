> 本条目解释铸造领域术语“造型”的两个独立技术含义。

## 概述

在铸造/压铸领域中，“造型”一词存在明显的工艺歧义，指向两类铸造工艺，二者在铸型制备方式、工装寿命和工艺特性上存在根本差异，本词条采用消歧分节方式分别阐述[[S31,S42]]：

1.  **砂箱造型**：使用一次性砂质铸型的传统造型工艺，属于砂型铸造。
2.  **铁型覆砂造型**：使用可重复利用的铸铁金属型（铁型）且内腔覆以砂层的特种造型工艺，属于半永久型铸造。

## 1. 砂箱造型

### 1.1. 定义

砂箱造型属于砂型铸造范畴，是将型砂装填到配套砂箱内围绕模样紧实，从而形成铸型的主流砂型制造工艺[[S42]]。该工艺在全球范围内承担着约70%~80%的铸件生产任务[[S7,S18,S42]]。

其基本原理是：以原砂、粘结剂（如粘土、树脂、水玻璃等）及辅助材料配制成型砂，填入带有模样（即铸件复制品）的砂箱内紧实，起模后留下型腔，合箱待浇注。浇注、冷却后，砂型被破坏以获得铸件[[S3,S8]]。

### 1.2. 工艺作用与流程

砂箱造型的核心作用是为金属液提供一次性成型的型腔。其完整工艺流程包含以下关键步骤[[S3,S8,S41]]：

1.  **型砂制备与混配**：将原砂、粘结剂、水及添加剂按比例混匀。
2.  **填砂与紧实**：将型砂填入砂箱，通过手动或造型机紧实，使型砂获得足够强度。
3.  **起模**：将模样从紧实的砂型中取出，形成铸件型腔。
4.  **下芯与合箱**：在型腔内安放砂芯（形成铸件内腔），将上下砂箱准确合拢锁紧。
5.  **浇注与落砂**：注入金属液，冷却后破坏砂型，取出铸件。

### 1.3. 分类

砂箱造型按所用粘结剂的主流细分类型，可分为粘土砂造型、树脂砂造型和水玻璃砂造型三大类[[S31,S29,S44]]：

| 造型类别 | 粘结剂 | 硬化/使用方式 | 特性与应用 |
| :--- | :--- | :--- | :--- |
| **粘土砂造型** | 粘土/膨润土 | 分湿型（不烘干）和干型（需烘干） | 成本低、适应性强，适用于绝大多数普通铸件[[S30,S31]] |
| **树脂砂造型** | 合成树脂（如呋喃树脂） | 造型后自行硬化 | 铸型强度高、尺寸精度好，适用于要求较高的铸件[[S29,S31]] |
| **水玻璃砂造型** | 硅酸钠水溶液（水玻璃） | 通过吹入CO₂气体或加入有机酯等方式快速硬化 | 硬化速度快，多用于铸钢件生产[[S29,S31]] |

### 1.4. 核心设备与工装

砂箱造型的核心工装与设备包括砂箱、模板和造型机。

-   **砂箱**：用于容纳和支撑砂型的刚性框架。
    -   **材料与分类**：按材料可分为木质、铝合金、铸铁；按用途可分为通用砂箱和专用砂箱[[S2,S33]]。
    -   **结构设计**：普通手工/半机械化造型用砂箱多为单层壁带排气孔结构；高压/气冲造型线用砂箱则采用双层壁高强度结构，以保证刚度和砂型附着力[[S2,S33]]。其规格设计要求标准化、系列化和通用化[[S2,S33]]。
-   **模板**：用于在砂型中形成铸件外形和浇注系统印痕的模具，是砂箱成型的基准[[S8]]。
-   **造型机**：实现机械化和自动化造型。主要类型及生产率[[S22]]：

    | 造型机类型 | 适用场景 | 典型生产率 |
    | :--- | :--- | :--- |
    | 震压造型机 | 一般机械化造型 | 50 ～ 80型/小时 |
    | 多触头高压造型机 | 高精度大批量造型 | 140 ～ 240型/小时 |
    | 射压造型机 | 高效率自动化造型（如迪萨线） | 最高可达 200 ～ 360型/小时 |
    | 抛砂造型机 | 中大型铸件造型 | - |

### 1.5. 关键工艺参数与材料

#### 型砂性能参数

以应用最广的粘土湿型砂为例，其行业通用参数控制范围如下[[S36,S40]]：

| 参数 | 常规机器造型 | 高密度造型 |
| :--- | :--- | :--- |
| 紧实率 | 38% ～ 45% | 38% ～ 45% |
| 湿压强度 | 60 ～ 150kPa | 120 ～ 180kPa |
| 透气性 | 100 ～ 200 | 40 ～ 200 |
| 含水量 | 4.5% ～ 5.5% | 3.2% ～ 4.5% |

#### 原材料

-   **原砂**：主要为硅砂，其牌号按“砂类型-粒度组别-颗粒形状”规则命名[[S10]]。
-   **粘结剂**：常用膨润土（粘土砂）、水玻璃（模数通常选2.3~2.4，波美度51~54）和呋喃树脂等[[S10,S29,S37]]。
-   **辅助材料**：包括用于防粘砂的煤粉、改善溃散性的木屑、水玻璃有机酯固化剂（加入量可低至2.8%~4.5%）等[[S10,S29,S31]]。

### 1.6. 适用范围

砂箱造型适用性极强，几乎可适配所有可铸造合金（铸钢、铸铁、铝合金等），并能生产从几公斤到数十吨级的各类铸件，同时适用于单件小批量生产和自动化流水线大批量生产[[S17,S42]]。自硬砂造型生产线中的标准砂箱，可见于工程结构图[[S21]]。

### 1.7. 常见缺陷与局限性

#### 常见缺陷及机理[[S17,S43]]

-   **粘砂**：金属液渗入砂粒间隙，形成表面附着物。
-   **夹砂**：砂型表层受热膨胀起皱翘起，金属液渗入形成结疤。
-   **气孔**：砂型内水分或发气物产生的气体侵入金属液并滞留。
-   **砂眼**：型砂脱落进入型腔，被金属液包裹形成孔洞。

#### 优点与局限性

-   **优点**：造型材料廉价易得，生产灵活性极强，可生产超大尺寸铸件，工艺技术门槛低[[S3,S42]]。
-   **局限性**：铸件尺寸精度偏低，表面粗糙度较高；部分型砂（如水玻璃砂）溃散性差，旧砂回用困难；浇注过程产生粉尘和废气[[S42,S29]]。

### 1.8. 关联标准

-   **GB/T 5611-2017《铸造术语》**：定义了包括两箱造型、多箱造型、脱箱造型等十余种砂箱类造型方法的术语与规范[[S19,S30]]。
-   **GB/T 2684系列标准**：规定了铸造用砂及混合料的性能测试方法，涵盖含水量、含泥量、紧实率、透气性等关键检测维度[[S19]]。

## 2. 铁型覆砂造型

### 2.1. 定义

铁型覆砂造型，又称铁模覆砂工艺，正式定义为在铸铁材质的金属型（即铁型）内腔预置一薄层型砂而形成铸型的特种铸造工艺[[S25]]。它由金属型铸造和壳型铸造演变而来，属于半永久型（金属型）铸造大类，兼具两者的优势[[S4,S9,S25,S38]]。在国际文献中，其概念与“permanent mold with sand lining”相吻合（通用度高），而国内常使用的“iron mold coated sand”在国际上通用度较低[[S5,S11,S14]]。

该工艺的核心原理是利用铁型的高刚性和可调节的覆砂层厚度，控制铸件冷却速度，实现顺序凝固或同时凝固，甚至可利用球墨铸铁的石墨化膨胀实现自补缩无冒口铸造[[S25]]。

### 2.2. 工艺作用与流程

铁型覆砂造型的作用是制备一种刚度高、可重复使用的半永久铸型。其全流程包含11个核心工序，为闭环循环生产[[S4,S16,S25]]。

1.  **铁型与模板准备**
2.  **合模预热**：将铁型和模板加热至工作温度（180~320℃）[[S9]]。
3.  **射砂充填**：通过射砂孔将覆膜砂射入铁型与模板间的缝隙。
4.  **保温固化**：利用预热的热量使覆膜砂固化，形成均匀的薄砂层。
5.  **脱模起模**：将模板与带有固化砂层的铁型分离。
6.  **下芯与合型锁模**：安放砂芯后，将铁型合拢并锁紧。
7.  **浇注**
8.  **型内冷却**
9.  **开型取件**
10. **铁型残砂清理**
11. **循环准备**：清理后的铁型返回步骤1，实现循环生产。

### 2.3. 核心设备与工装

-   **铁型**：工艺的核心工装，材质为铸铁，要求耐热冲击、抗变形、尺寸稳定性高。其内腔是保证覆砂层厚度的基础[[S4,S9]]。
-   **射砂装置**：多采用热芯盒/壳芯机类结构，配置分布均匀的小规格射砂孔，确保将覆膜砂完整充填至薄层缝隙中[[S4]]。
-   **加热固化系统**：用于铁型和模板的预热。可采用燃气加热、电加热，或利用浇注后铸件余热维持工作温度[[S4,S9]]。

铁型覆砂射砂工位的核心装配状态：吹砂头、覆砂层、铁型、模板的配合[[S15]]。

### 2.4. 关键工艺参数与材料

#### 工艺参数

铁型覆砂造型的关键工艺参数行业通用范围如下[[S1,S4,S9]]：

| 参数 | 典型/通用范围 | 备注 |
| :--- | :--- | :--- |
| 铁型预热/工作温度 | 180 ～ 320℃ | 部分场景最低可至150~190℃ |
| 射砂压力 | 0.15 ～ 0.6 MPa | - |
| 射砂时间 | 2 ～ 6s | 小部件生产多取3~5s |
| 覆砂层厚度 | 3 ～ 15 mm | 常规工况多取5~8mm |
| 固化时间 | 30 ～ 150s | 小部件生产可短至15~20s |

配套工艺控制要求：铁型温度需保持稳定，若连续生产导致温度过高，需采用风冷或间接水冷强制降温，以防覆砂层焦化[[S26,S34]]。电炉铁液的出炉温度推荐控制在1500~1520℃[[S26]]。

#### 材料与介质

-   **覆膜砂**：工艺主流采用酚醛树脂覆膜砂，其热固化特性适配铁型预热温度场，固化后可形成均匀高强度砂层[[S4,S9]]。
-   **涂料**：针对特殊工况，可在覆砂层表面涂刷耐火保护涂料，以防铁液侵蚀砂层导致粘砂[[S4]]。

### 2.5. 适用范围与应用

该工艺适配合金类型广泛，包括铸钢、灰铸铁、球墨铸铁、可锻铸铁、铜合金等[[S9,S25]]。典型量产铸件包括球墨铸铁曲轴、凸轮轴、制动毂、缸套、耐磨齿盘、压力阀体等30余种[[S25]]。该工艺特别适合中等以上批量的自动化生产线，国内已有近百家铸造企业应用[[S25]]。

### 2.6. 优势、缺陷与局限性

#### 工艺优势

-   **铸件质量高**：平均尺寸精度可达CT7级，表面粗糙度低至12.5μm，铸件致密度比普通砂型铸造平均提升0.86%[[S1,S9]]。
-   **工艺出品率高**：利用铁型高刚性，可实现球墨铸铁件的无冒口铸造，工艺出品率最高可达90%，加工余量小[[S9,S25]]。

#### 常见缺陷及机理[[S16]]

-   **覆砂层脱落**：主要由铁型表面清洁度不足、预热温度不均、覆膜砂固化不充分导致。
-   **铸件气孔**：由覆砂层残留过量水分、射砂排气不充分或浇注过程裹气引发。
-   **夹渣**：在球铁曲轴生产中，缺陷率常为1.2%～2%，极端工况下可达4.9%。主要由铁液纯净度不足、扒渣不彻底、浇注系统不合理造成。
-   **局部缩松**：多发生于内浇口附近过热区域，因铁液局部长时间加热、凝固顺序失控导致。
-   **表面粘砂**：当铁型温度过高时，覆砂层会焦化导致强度丧失，从而引发铸件粘砂[[S26]]。

#### 工艺局限性

-   **工装成本高、修改难**：铁型工装整体造价偏高，且一旦定型，更改困难[[S25]]。
-   **通用性差、技术要求高**：铁型通用化适配难度高，特殊结构铸件的铁型设计难度较大[[S25]]。
-   **自动化普及不足**：现有行业生产线整体机械化、自动化普及度仍有待提高[[S25]]。

## 3. 与压铸的关系

在高压/半固态压铸工艺体系中，不存在传统意义上利用型砂制备铸型的“造型”工序[[S20,S23]]。

压铸体系下的类似“造型”环节，本质是模具准备或浆料制备工序，与砂箱/铁型覆砂造型有本质区别[[S20,S41,S12]]：
-   **模具预热**：将可反复使用的钢制压铸模预热至工作温度，以确保金属液充型完整。
-   **半固态浆料/坯料制备**：在流变或触变压铸前，通过机械搅拌、电磁搅拌等方式制备具有非枝晶球状组织的浆料或锭料。

上述所有压铸工序均采用可反复使用的整体合金钢模具作为成型载体，无需制备消耗性的砂质铸型，这与砂箱造型的一次性砂型及铁型覆砂造型中的可消耗砂层有根本性不同[[S42]]。
