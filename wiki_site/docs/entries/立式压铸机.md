## 概述

立式压铸机（vertical die casting machine）是冷室压铸机的一个重要分支。其核心特征是压射方向与合模方向相互垂直：压射室（压室）中心线呈垂直布置（压射冲头沿竖直方向运动），而合模机构带动模具沿水平方向开合，即压室中心线与模具分型面平行 [[S43,S51,S47,S57]]。这一特征使立式压铸机在结构、工艺和应用场景上与卧式压铸机（压射与合模均为水平）及全立式压铸机（压射与合模均为垂直）形成明确区分 [[S9,S38]]。立式压铸机行业主流采用冷室结构，也存在少量特殊热室立式机型 [[S39,S4]]。

## 分类与概念辨析

### 在压铸机大类中的位置

压铸机通常按压室受热条件分为冷室压铸机和热室压铸机两大类。冷室压铸机进一步按压室放置方向分为卧式、立式和全立式三种 [[S4,S10,S33]]。

### 与全立式压铸机的区别

全立式压铸机的合模机构和压射机构均为垂直布置，压室中心线与模具分型面垂直 [[S43,S51,S38]]。全立式压铸机可进一步分为冲头上压式和冲头下压式两种子机型 [[S38,S70,S10]]。此分类方式不适用于常规立式压铸机，是两者之间的重要区分依据 [[S38,S24]]。

### 冷室与热室分类

立式压铸机行业以冷室结构为主流，即压室与保温熔炉分离，需要从熔炉取液浇入压室。少量热室立式机型则将压室与熔炉集成一体，适用于低熔点合金压铸 [[S39,S4]]。

## 结构与组成

立式压铸机主要由以下系统构成：

| 系统 | 核心部件 | 功能 |
|------|---------|------|
| 压射机构 | 压射缸、压射冲头、料筒（压室）、反料冲头 | 按预设速度推动压室内金属液完成充型，保压阶段持续传递压力直至铸件完全凝固；反料冲头在压射前封堵喷嘴，压射结束后上升切断并推出余料 [[S39,S58,S21]] |
| 合模机构 | 定模板、动模板、曲肘扩力机构或全液压合模缸 | 带动压铸模完成开合动作，合模后提供足够锁模力防止压射时分型面胀开 [[S39,S58,S21]] |
| 液压系统 | 液压泵、蓄能器、控制阀组 | 为全机各机构动作提供动力与能量 [[S39,S37,S21]] |
| 电气控制系统 | PLC控制柜、传感器 | 按预设程序控制所有执行部件的时序运行 [[S39,S37,S21]] |
| 安全防护装置 | 防护门、急停装置、安全警示标识 | 覆盖合模、压射等危险动作区间，实现操作防护 [[S39,S37,S21]] |

立式压铸机结构上的一个重要特征是增加了反料（下冲头）机构，配备切断、顶出余料的下油缸，使整机结构复杂度高于卧式压铸机 [[S5,S64,S12]]。典型国产机型如J1513型立式冷压室压铸机采用三级压射系统和曲肘扩力合模机构，工作压力为120公斤/厘米² [[S26]]。

## 工作原理

### 工作循环

立式冷室压铸机的标准工作循环依次为 [[S43,S51,S67,S30]]：

1. **合模**：压铸模水平合拢并锁紧。
2. **浇注**：将定量合金液浇入垂直压室，金属液被已封住喷嘴孔的反料冲头托住，不会提前流入型腔。
3. **压射**：压射冲头下行，当接触金属液面后反料冲头开始下降打开喷嘴，金属液在高压下被压入模具型腔完成充型。
4. **保压**：冲头持续加压至铸件完全凝固。
5. **切断余料**：压射冲头回程，反料冲头上升切断并顶出压室内的余料。
6. **开模与取件**：余料取走后，反料冲头降至原位，模具开模，推出机构取出压铸件。

### 充型特点与杂质管控

充型时金属液需经过90°垂直转角进入型腔，这一流动特征带来两个核心工艺效应 [[S43,S47,S18]]：

- **压力损耗增大**：相比卧式压铸机，金属液转折带来的流动阻力更大，部分压射压力被消耗。
- **杂质管控效果更佳**：漂浮在金属液表面的氧化渣和杂质在压射时最后进入余料侧，不会进入型腔，有利于防止杂质进入铸件。

这一工艺特点使立式压铸机天然适配中心浇口类压铸件的生产 [[S43,S47,S18]]。

### 国标规定

GB/T 21269-2023《冷室压铸机》对立式冷室压铸机设有独立的基本参数表和压射性能表，对锁模力、压射力、压射速度等性能指标作出了规范性要求。立式冷室压铸机最大压射速度测试条件规定为2 m/s ± 0.2 m/s，压射室充填系数取0.95 [[S31,S55,S25,S7,S17]]。

## 与卧式压铸机的对比

下表总结立式冷室压铸机与卧式冷室压铸机在各维度的主要差异：

| 对比维度 | 立式冷室压铸机 | 卧式冷室压铸机 |
|----------|--------------|--------------|
| 压室布置 | 压室中心线平行于模具分型面，压射方向垂直 | 压室中心线垂直于模具分型面，压射方向水平 [[S11,S62]] |
| 余料处理 | 配备反料冲头切断并顶出余料，结构复杂度高 | 无额外反料机构，压射冲头直接推出余料，结构简单 [[S5,S64,S12,S11]] |
| 占地面积 | 压射机构直立，占地面积显著小于同吨位卧式机 | 水平布置，占地面积较大 [[S11,S22,S61]] |
| 金属流动路径 | 需转折90°进入型腔，压力损耗大 | 进入型腔转折少，压力损耗小，有利于增压作用 [[S11,S49,S12]] |
| 气体/夹渣卷入 | 压室空气不易进入型腔，氧化渣留于余料侧 | 金属液与空气接触面积大，压射时易卷入空气和氧化夹渣 [[S18,S11,S62]] |
| 中心浇口适应性 | 天然适配中心浇口，无需额外机构 | 需设置专门的中心浇口切断机构，模具设计复杂度高 [[S22,S32,S40]] |
| 生产效率 | 余料未切断前不能开模，循环步骤更多，效率较低 | 浇注后任何时间均可开模，易实现全自动化，效率更高 [[S49,S11,S12]] |
| 操作与维修 | 结构复杂，压射零部件调整更换不便，维修难度高 | 结构简单，调整更换方便，维修便利 [[S49,S12,S54]] |
| 适用合金 | 铝合金、镁合金、铜合金等有色金属 | 除有色金属外，还可适配黑色金属压铸 [[S46,S11,S12]] |

## 应用领域

立式压铸机的典型应用场景为需要设置中心浇口的压铸件生产，尤其适配中小批量的中心浇口小型铝合金压铸件 [[S5,S64,S40]]。

主要应用方向包括：

- **电机转子与定子**：大规格铝制电机转子、定子类带硅钢片嵌件的压铸件常采用立式压铸机生产。专用50吨级立式转子压铸机可生产最大直径200 mm的电机转子，配合自动化上下料系统可实现批量生产 [[S22,S47,S6]]。
- **带嵌件零件**：配备垂直开合模具的立式压铸机便于放置嵌件，特别适合生产带有预制嵌件的锌合金、铝合金装饰件和结构件 [[S48]]。
- **下游行业**：覆盖汽车零部件、电机制造、电子元器件、五金装饰件等领域。压铸行业中家电电子产品占比约11%，汽车摩托车零部件占比约48%，立式压铸机参与相关细分生产场景 [[S36]]。
