## 概述

在压铸领域语境下，“陶瓷型制造”并非指广义的陶瓷型铸造全流程工艺，其术语边界特指采用硅酸乙酯基陶瓷浆料制备铸型、用以直接浇注成型钢制压铸模本体的制模工序[[S1,S11,S15,S34]]。该工艺需与以下三类概念明确区分：单独用于铸件内腔成型的熔模铸造/压铸用陶瓷型芯制造工艺；仅在常规金属压铸模型腔表面制备薄层陶瓷防护涂层的改性工艺；以陶瓷为铸型直接浇注普通金属铸件的通用陶瓷型铸造工艺[[S34]]。

陶瓷型制造采用能耐高温的陶瓷质耐火材料为骨料、水解硅酸乙酯为粘结剂，在催化剂作用下经灌浆、结胶、起模及焙烧等工序制成铸型[[S28]]。用陶瓷型浇注的铸件具有较高的尺寸精度和较小的表面粗糙度值，被广泛用于冲模、锻模、玻璃模、压铸模等模具的快速精密制造[[S1,S15]]。

## 分类

中文体系下，陶瓷型按结构可分为两大类别[[S1,S15]]：

- **整体陶瓷型**：全部由陶瓷浆料灌注成型，仅用于小型铸件或陶瓷镶块制备[[S1,S3]]。
- **复合陶瓷型**：仅型腔工作面使用薄层陶瓷浆料，背衬采用底套替代。复合陶瓷型可进一步细分为砂底套陶瓷型和金属底套陶瓷型[[S15]]。其中CO₂水玻璃砂底套因制作方便、透气性好、成本低，是目前工业应用最广泛的类型[[S1]]。

![陶瓷型制造分类结构图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/3a720c39-338e-4791-874b-7a56f595aa01/resource)

英文语境下，陶瓷型铸造（Ceramic mold casting）分为两个核心子类：肖氏工艺（Shaw process）和Unicast工艺，二者均可用于生产压铸模等工装零件[[S44,S46]]。Shaw工艺采用耐火骨料、水解硅酸乙酯和胶凝剂配制浆料，固化后点火烧除挥发分形成可控微裂纹以提高透气性，最终经1000℃左右高温焙烧固化；Unicast工艺无需点火预烧，浆料半固化起模后直接在约1038℃下完全焙烧，流程更简化、成本更低，已成为主流工艺[[S44,S20]]。

## 工艺流程

陶瓷型制造全流程工序完整覆盖：零件图/铸件工艺设计→母模制造→底套制备→硅酸乙酯水解→陶瓷浆料配制→灌浆→结胶硬化→起模→喷烧焙烧→合型浇注→铸件清理→热处理→检验入库[[S2,S18,S63]]。

![陶瓷型铸造工艺示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/4dbf90d5-8587-4caf-a836-b889d99a37da/resource)

### 灌浆

浆料中加有催化剂后，黏结剂从溶胶状态转变为凝胶状态，黏度随之增大。从开始黏度增大至迅速增大之前的阶段称为"结胶苗子"，此时需立即进行灌浆。灌浆过早，易冲走母模表面的分型剂引起黏模，且凝固时间长导致耐火材料沉淀、分层和铸型开裂；灌浆太迟，浆料流动性太差，型腔轮廓不清晰[[S5]]。

### 结胶与起模

结胶时间可通过催化剂加入量调整：小件控制在4~6 min，中大件控制在8~15 min[[S5]]。起模的最佳操作窗口为浆料固化后尚处于弹性状态（砂型硬度计测试硬度80~90），一般起模时间约为2倍结胶时间[[S5]]。过早起模因固化不完全导致裂纹，过迟则浆料失去弹性易损坏铸型。真空条件下灌浆可进一步提升浆料填充质量，减少缺陷[[S28]]。

### 焙烧

陶瓷型壳的焙烧需严格按温度-时间规范控制[[S52]]：以4~5℃/min速率升温至250~300℃，保温1~1.5 h；再以3~4℃/min升温至450~550℃，保温1.5~2 h；之后以4~5℃/min升温至700~750℃，保温1.5~2 h；再以3~4℃/min升温至950~1350℃，保温2~3 h；最后随炉冷却。为保证铸型精度，焙烧应在氧化气氛中进行[[S52]]。

## 材料组成

### 耐火骨料

陶瓷型用耐火骨料要求杂质少、熔点高、高温热膨胀系数小[[S36,S56]]。针对高要求合金钢锻模、压铸型、玻璃器皿模具，优先选用刚玉粉、锆英砂粉、碳化硅；铸铁件、铝铸件可选用价格较低的铝矾土、硅石粉[[S36,S56]]。耐火骨料需由多种不同粒度级配组合，小件或高表面质量产品细粉占比更高。细粉总占比超过60%会导致铸型透气性接近零、易产生裂纹[[S36]]。

### 粘结剂——硅酸乙酯水解液

硅酸乙酯水解液的标准配方（质量分数%）如下[[S4]]：

| 组分 | 含量（质量分数%） |
|------|----------------|
| 硅酸乙酯 | 52~60 |
| 加水量 | 10~12 |
| 乙醇加入量 | 28~38 |
| 盐酸 | 0.3~0.6 |
| 甘油（附加物） | 0.05 |
| 硼酸（附加物） | 0.03 |

水解液中添加0.5%左右的醋酸或甘油可抑制焙烧阶段的大裂纹产生[[S28,S56]]。

### 陶瓷浆料典型配方

石英粉陶瓷浆料的标准配方[[S41]]：

| 组分 | 含量 |
|------|------|
| 270目石英粉 | 55 g |
| 50~100目石英砂 | 30 g |
| 30~50目石英砂 | 15 g |
| 水解液（对粉料重） | 40 mL/100g |
| 氢氧化钙（对水解液） | 0.2~0.4 g/100mL |
| 10%硼酸甘油溶液（对水解液） | 3~5 mL/100mL |

### 添加剂

- **催化剂**：固体催化剂包括氢氧化钙、氧化镁，液体催化剂包括苛性钠、碳酸铵水溶液，用于精准控制浆料结胶时间[[S28]]。
- **透气剂**：分为两类，一类为喷烧过程中烧掉或升华的淀粉、糊精等，另一类为分解产气的双氧水、碳酸钡等。双氧水添加量为耐火粉料总重量的0.2%~0.3%[[S28,S56]]。
- **脱模剂**：常用上光蜡、机油、有机硅油、凡士林等，上光蜡与机油配合使用效果最佳[[S28]]。

## 性能指标

陶瓷型铸造成型的铸件性能指标[[S1,S3,S15]]：

| 性能指标 | 典型范围 |
|---------|---------|
| 表面粗糙度 | Ra1.2~12.5 μm |
| 尺寸精度 | CT5~CT8 |
| 铸件单重 | 数十克至十几吨 |

铸造成型的模具使用寿命常接近或超过机械加工的模具，制造成本更低[[S1,S3]]。陶瓷型耐火度高、高温性能稳定，可适应高温合金、合金钢、碳钢、铸铁、铜合金、铝合金等多种合金的浇注[[S15]]。

## 与陶瓷型芯的区别

需特别注意陶瓷型制造与陶瓷型芯制造的区别。陶瓷型芯是作为可移除的内置构件，用于成型常规抽芯结构无法加工的铸件复杂内腔、微细冷却通道，浇注完成后通过化学腐蚀或高压溶解等方式脱除[[S7,S43,S8]]。而在压铸工艺场景下，陶瓷型芯可配套制造传统刚性金属抽芯难以实现的复杂异形内流道类压铸件[[S65,S30,S60]]。

## 缺陷与控制

陶瓷型制造中常见的缺陷及控制措施：

| 缺陷类型 | 产生原因 | 控制措施 |
|---------|---------|---------|
| 裂纹 | 起模时间不当、细粉占比过高、水解液配比不当 | 控制结胶时间与起模窗口，优化骨料粒度级配，添加醋酸或甘油抑制焙烧裂纹 |
| 分层 | 灌浆过早、粉料沉淀 | 控制灌浆最佳时机（结胶苗子阶段），边灌浆边振动 |
| 表面气孔 | 灌浆卷入空气 | 控制灌浆速度，边灌浆边振动使气泡上浮，真空灌浆 |
| 变形 | 喷烧不均匀、型壁过厚 | 控制焙烧升温速率，复合型代替整体型降低成本与变形风险 |

*数据来源：[[S5,S28,S36]]*

## 应用场景

陶瓷型制造已成为大型厚壁精密铸件生产的重要方法，主要应用于[[S1,S3,S15]]：

- **模具制造**：冲模、锻模、塑料模、玻璃模、橡胶模、压铸模、金属型、热芯盒等表面形状不易加工的铸件。除能大量节约加工工时外，其使用寿命往往接近或超过机械加工的模具[[S1,S3]]。
- **精密铸件**：如叶轮等中型精密铸钢件[[S15]]。
- **黑色金属压铸**：陶瓷材料如硼化锆碳化硅（ZRBSC）在1982.2℃高温下可抗氧化、耐热冲击，具有高高温强度、耐烧蚀、耐腐蚀性能，可显著提升黑色金属压铸过程中铸型的抗热疲劳能力[[S17]]。

## 先进工艺——3D打印陶瓷型

3D打印DSPC（直接壳型制造）工艺采用陶瓷粉末等材料作为主要成形材料，喷头喷射粘结剂将粉末粘结成形，无需单独制作和装配型芯即可制备出含复杂内腔的陶瓷型[[S45]]。DSPC工艺层厚控制在0.125~0.175 mm，铸件尺寸精度达±2 mm/100 mm，角度精度达±0.5°。如舰船用水冷排气管（400 mm×200 mm×150 mm铝合金铸件，含7个独立型芯），从陶瓷型制造到完成铸件仅需3周[[S45]]。
