压铸岛是将压铸主机与浇注、取件、喷涂、冷却、检测、切边、打码、输送等一系列辅助设备集中布置、协同运行的闭环生产成套系统。行业内常将其视为全配置、高自动化压铸单元的特指名称，以实现从熔融金属到合格铸件的全流程自动化生产。

国家标准 GB/T 37371-2019《压铸单元 术语》将“压铸单元”定义为压铸机与辅助设备形成的自动化生产成套装备[[S16]]。在此基础上，压铸岛更强调以单台压铸机为核心、将所有配套设备按产品就近集中布局，尽量减少跨岛搬运，构成一条产品导向的全工序集成产线[[S15,S8]]。

“产品原则布置”对应的英文术语为 Product Layout，它是压铸岛设计所遵循的布局原则——所有相关设备围绕单一产品的生产流程就近布置，使物料路径最短[[S31]]。因此“产品原则布置”并非压铸岛的别名，而是其布局理论来源。“9000t 压铸岛”则特指采用 90000 kN 级超大型压铸机的一体化压铸岛，仅在特定案例中使用[[S11,S20]]。

## 分类

压铸岛可按多维度分类，常见方案如下[[S15,S14,S24,S7]]。

**按压铸机吨位**
| 类别 | 吨位范围 | 典型应用 |
|---|---|---|
| 传统压铸岛 | 2000 t 级 | 中小型压铸件 |
| 大型一体化压铸岛 | 4000～6000 t 级 | 新能源汽车大型结构件 |
| 超大型一体化压铸岛 | 9000 t 级及以上 | 超大尺寸一体化车身结构件 |

**按自动化程度**
| 类别 | 特征 |
|---|---|
| 简易半自动化压铸岛 | 仅喷涂、取件自动化，渣包、料饼等需人工处理 |
| 全自动化压铸岛 | 自动浇注、自动切边、自动去毛刺、在线检测、自动换模等全工序无人化 |
| 压铸柔性单元 | 多品种小批量快速换型，工艺参数自适应调整，可混线生产 |

**按产品对象**
| 类别 | 示例产品 |
|---|---|
| 一体化压铸单元 | 后地板、前机舱、电池托盘等大型车身结构件 |
| 中型零部件专用压铸岛 | 汽车变速箱壳体、副车架等 |
| 小型精密件压铸岛 | 消费电子壳体、通讯器件等 |

## 设备构成

典型的大型全自动化压铸岛由以下核心设备与辅助系统组成（以一体化后车身压铸岛为参考）[[S2,S14,S15,S21,S10]]。

| 系统/设备 | 功能 |
|---|---|
| 集中熔化炉 | 将铝合金锭连续熔化，常见的熔化能力为 4 t/h |
| 机边保温炉 | 维持铝液温度，容量可达 10 t，内置透气砖通入氩气进行除气处理，可将铝液密度指数控制在 2% 以下 |
| 给汤机（自动浇注装置） | 定量浇注铝液至压室 |
| 压铸机主机 | 提供锁模力与高速压射，将铝液充填至模具型腔；锁模力可达 6000～90000 kN |
| 模具及热平衡系统 | 包含多回路仿形喷涂装置、模温机（大型岛标配 30 台以上）、真空系统（型腔真空度＜0.005 MPa）等 |
| 取件机器人/机械手 | 开模后将铸件取出并转移到后续工位 |
| 冷却水槽/快冷站 | 将取出的铸件快速冷却 |
| 去渣包/等离子切割单元 | 去除溢流渣包、料饼和浇道 |
| 打码工位 | 激光刻印明码与追溯二维码 |
| 在线 X-Ray 探伤设备 | 对铸件内部缺陷（缩孔、裂纹等）进行抽检或全检，数据与压铸工艺参数关联追溯 |
| 去毛刺单元 | 去除铸件表面飞边 |
| 自动装箱站及 AGV | 按品质等级将合格品装箱入库，AGV 完成铝锭上料和成品转运 |
| 中央集成控制系统 | 对整个岛的工艺参数、设备状态进行集中监控与数据交互 |

相较于简化描述，实际标准配置还必须包括成套真空系统、多单元模温机、等离子去料渣系统、油雾及粉尘除尘系统等[[S2]]。部分压铸岛采用集成式 CTM 温控站替代分散模温机，可将温控单元占地面积减少 70% 以上，装机功率节约 70% 以上[[S10]]。

## 工艺过程

以大型一体化压铸岛典型流程为例，全工序在岛内完成，无多余中转环节[[S2]]：

1. 铝锭在集中熔化炉中被连续熔化；
2. 铝液泵将铝液输送至机边保温炉，进行保温及除气处理；
3. 给汤机将定量铝液注入压室；
4. 压铸机高压压射，铝液在真空型腔中充填并保压凝固，形成带浇注系统的毛坯；
5. 开模后取件机器人将毛坯取出，送入水冷工位快速冷却；
6. 搬运机器人将冷却后的毛坯送至去渣包工位，去除溢流渣包；
7. 等离子切割单元去除料饼和浇道；
8. 打码工位刻印明码及追溯二维码；
9. 去毛刺工位清除表面飞边；
10. 按品质检测需求进入在线 X-Ray 探伤工位抽检；
11. 人工工位完成 100% 外观全检；
12. AGV 将合格品运送至自动装箱站，机器人按品质等级分类装箱入库。

## 优势与局限性

**优势**

- **质量一致性**：全自动化设备匹配使生产周期高度统一，铸件尺寸精度和质量稳定性显著优于传统“单机+人工”模式[[S15,S14]]。
- **效率与节拍**：集成化设计消除了物料跨岛搬运，单岛节拍可优化至 90 s 左右[[S2]]；传统压铸岛经优化后节拍也可压缩 10%～15%[[S4]]。
- **减少占地与节能**：采用 CTM 集成温控站可减少温控单元占地面积 70% 以上，节约设备成本 30% 以上，装机功率节约 70% 以上[[S37,S10]]。
- **降低人力依赖**：AGV 自动投料、全流程机器人操作，替代叉车转运高温铝液和人工搬运大尺寸铸件等高强度作业[[S2,S15]]。

**局限性**

- **生产柔性受限**：大型一体化压铸岛为特定大尺寸集成铸件定向开发，难以灵活切换多品种、小批量小尺寸铸件生产[[S30]]。
- **调试与维护复杂**：一体化模具需布设 10 个以上模内温度传感器，设置数百组油温、水温、冷却时间参数，全新产线适配调试周期长达半年以上[[S30]]。

## 术语关联与布局原则

“压铸岛”在工业工程设施规划中的层级关系如下[[S31]]：

- **三大经典布局原则**：产品原则布置（Product Layout）、工艺原则布置、固定式布置；
- **单元生产**（Cell Manufacturing）：工序集中的生产组织模式；
- **压铸岛**：单元生产模式下产品原则布置在高压铸场景中的具体实现，属面向产品对象的高度集成化制造单元。

国家标准《压铸单元 术语》（GB/T 37371-2019）定义的“压铸单元”与英文 die casting cell 对应[[S16]]。在行业实践中，“压铸岛”这一术语更强调全集成、单岛全流程覆盖，而“压铸单元”则是一个覆盖范围更广的通用概念[[S15,S16]]。
