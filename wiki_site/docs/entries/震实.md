## 概述

震实（jolt compaction / jolt ramming）是铸造生产中用于紧实型（芯）砂的工艺方法。其核心原理为通过震动撞击使松散型砂在惯性作用下获得紧实 [[S20,S41]]。在砂型铸造领域，震实是机器造型的主要紧砂手段之一，对应的造型设备称为震实造型机（jolt molding machine）[[S22,S23]]；当震实与压实工序复合应用时，则称震压造型，设备为震压造型机（jolt-squeeze molding machine）[[S20,S21]]。此外，在消失模铸造、V法铸造以及落砂工序中，“震实”指利用激振力完成干砂充填紧实或铸件与砂型分离的过程 [[S13,S20,S7]]。

## 定义

根据国家标准 GB/T 5611-2017《铸造术语》的定义：

> **震实** jolt compacting：在低频率和高振幅运动中，下落冲程撞击使型砂因惯性获得紧实的过程。[[S41]]

GB/T 25370-2020《铸造机械 术语》给出等价定义：通过震动撞击使型（芯）砂因惯性获得紧实的过程，通常采用冲击气缸对造型工作台产生震动撞击。[[S20]] 英文术语对应 jolt compaction、jolt ramming、jolt molding、vibration compaction 等表述，在不同场景下可交互使用 [[S2,S32]]。

## 工艺作用

震实工艺在铸造生产中的主要作用场景包括：

1. **砂型紧实（震实造型）**：工作台将砂箱、模板及型砂一同抬升至指定高度后自由下落产生震击，通过连续反复的惯性撞击使型砂逐步紧实。常规单型循环震击次数范围为 10～80 次。[[S18,S22,S5]]
2. **砂芯制造（震实制芯）**：震实制芯机用于制造大、中型砂芯，是主流机器制芯方法之一。[[S1,S3]]
3. **干砂充填紧实（消失模铸造）**：振实台通过振动消除干砂颗粒间静摩擦力，使砂料获得流动性，充填至泡沫模样复杂内腔的微小缝隙中完成紧实。[[S40,S7]]
4. **落砂清理**：依靠激振力将冷却后铸件与砂型、砂芯分离，属于铸造落砂工序的主流作业方式。[[S20]]
5. **含胶结剂松散料压实**：振动分两阶段——先以低频大振幅克服颗粒间摩擦力完成大颗粒排布，再以高频振动激发胶凝材料触变效应完成微观结构致密化。[[S31,S37]]

## 震实造型机分类与工作原理

### 工作原理

砂型铸造震实造型机以压缩空气为动力。工作台承载砂箱、模板及型砂，由震击活塞抬升至一定高度后自由下落，与机座产生撞击，使型砂在惯性冲击力作用下获得紧实。震击高度一般为 30～60 mm，震击次数每分钟 30～50 次，一般不超过 80 次。[[S18,S22,S5]]

按进排气运动特性，震击工作机构可分为三类 [[S23]]：

| 类型 | 特点 |
|------|------|
| 空气不断进气和不膨胀型 | 压缩空气持续进入，经济性较差 |
| 空气断进气而不膨胀型 | 进气截止后立即排气，空气消耗量低 |
| 空气断进气和膨胀型 | 进气截止后气缸内空气膨胀后再排气 |

其中断进气结构消耗量更少、经济性更好，应用最为广泛。[[S23]]

### 按起模方式分类

震实造型机按起模结构可分为顶箱（杆）式、转台式和翻台式三类，均由震实机构、相应起模机构及控制系统组成，以压缩空气为动力。[[S22,S23,S28]]

| 类型 | 代表型号 | 起模方式 | 适用场景 |
|------|---------|----------|---------|
| 顶杆式 | TL-18 | 顶杆顶起砂箱起模，工作台四周完全开放 | 适合简单上型快速造型，无需翻转，效率高于翻/转台式 [[S28,S17]] |
| 转台式 | Z2520 | 砂箱连带模板整体翻转 180°，接箱台下降起模 | 带大吊砂块的大中型铸件，可接入机械化造型线 [[S22,S28]] |
| 翻台式 | Z2320、Z2310、Z2330 | 砂箱连带模板经翻台机构整体翻转 180° | 复杂铸件造型，可调节震击力获得所需砂型紧实度 [[S28]] |

## 紧实度分布特征与震压复合工艺

纯震实造型的紧实度沿砂箱高度方向呈上松下紧的梯度分布特征：靠近模板位置的型砂受全部上层砂料惯性作用，紧实度最高；砂箱上层型砂几乎不受震击作用，紧实度偏低。因此震实造型仅适用于砂箱高度大于 150 mm 的较高砂箱。[[S18,S22,S24]]

为改善上层紧实度不足的固有缺陷，衍生出震压造型机——先震实底部型砂，再通过压实机构补充压实上层型砂，使整体紧实度分布趋于均匀。[[S18,S39,S5]]

震实造型生产线实操工艺要求包括 [[S9]]：
- 翻台式、顶杆式机身作业前需检查震实机构运行状态
- 下砂型震实后供补压用的砂层厚度一般控制在 20～40 mm
- 震实过程中需同步塞紧直浇道周边砂料
- 震实后砂型表面需刮平与砂箱箱口齐平

## 特种铸造中的震实（消失模/V法）

在消失模及 V 法等特种铸造工艺中，震实的核心作用是通过振动力完成干砂向复杂型腔的充填紧实。干砂充填过程可分为三个阶段，每个阶段颗粒受力状态不同，振动参数直接影响充填效果。[[S10,S40]]

### 振实台分类

消失模铸造用振实台按多种维度分类 [[S29]]：
- **按振动维数**：一维振实台、二维振实台、三维振实台
- **按控制方式**：普通式振实台、调频式振实台、定位式振实台
- **按振动方向**：垂直振实台、水平振实台、圆周振实台

不同类型振实台适应不同铸件复杂度需求。对复杂铸件推荐采用三维振动或垂直面圆周运动振动，以有效防止泡沫模样变形和充填不实缺陷。[[S7,S13]]

### 关键工艺参数

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| 振动频率 | 10～80 Hz（消失模），30～80 Hz（推荐） | 高频低幅振动可减少泡沫模样变形 [[S40,S13]] |
| 振幅 | 0.2～2 mm（消失模），0.5～1.5 mm（推荐） | 振幅过大会损伤泡沫模样 [[S40,S7,S13]] |
| 振动加速度 | 1～2g | 保证干砂充填效果 [[S13,S40]] |
| 振实时间 | 20～60 s | 三维振实确保充填充分 [[S7]] |
| 气囊充气压力 | 0.3～0.7 MPa | 调频气垫振实台的气垫悬浮支撑 [[S40]] |

## 振动源类型对比

铸造领域震实设备的激振源主要分机械类和气动类两种 [[S35,S18,S21]]：

| 属性 | 机械类（电动激振器） | 气动类（冲击气缸） |
|------|---------------------|---------------------|
| 动力原理 | 电机驱动偏心块旋转产生离心惯性力 | 压缩空气驱动活塞抬升后自由下落撞击砧座 |
| 典型频率 | 10～50 Hz | 8～15 Hz（微振） |
| 振幅 | 可调，通过偏心距调整 | 几毫米至数十毫米 |
| 能量供给 | 电力驱动 | 复用铸造车间压缩空气管网 |
| 运行噪声 | 约 90 dB(A) | 可达 100 dB 以上 |
| 维护特征 | 需定期更换轴承、补充润滑油，年维护成本中等偏低 | 核心密封件易损耗，年维护成本中等 |

## 与气冲造型、静压造型的对比

在砂型铸造高压紧实工艺中，震实/震压与气冲造型、静压造型形成互补技术路线。[[S12,S18,S33,S38]]

| 对比维度 | 震实/震压造型 | 气冲造型 | 静压造型 |
|----------|---------------|---------|---------|
| 紧实原理 | 震击惯性力 + 可能辅以压实 | 0.35～0.5 MPa 压缩空气瞬间释放压力波（>30 MPa/s） | 气流预紧实（5 MPa/s） + 压头压实 |
| 砂型高度适配 | 150～400 mm | 200～350 mm | 适配范围最广 |
| 紧实度均匀性 | 纯震实上松下紧，震压可改善 | 整体尚可，高模样间隔、狭窄区易“架桥” | 紧实度高且均匀，避免架桥缺陷 |
| 铸件尺寸精度 | CT10～CT12 级 | CT8～CT9 级 | CT7～CT9 级 |
| 单型紧实时间 | 中等 | 约 0.3 s | 冲击阀关闭后 3.5 s 再压实 |
| 生产率 | 中等 | 最高可达 170 整型/h 以上 | 一般 |
| 运行成本 | 中等偏低 | 最低 | 约为气冲的 3 倍 |
| 噪声 | 震击噪声大 | 气流冲击噪声大 | 噪声低（70～80 dB） |

- **GB/T 5611-2017**《铸造术语》：在“6.2 造型制芯”部分给出震实、震实造型机、震实台、震压造型机等术语的官方定义。[[S41]]
- **GB/T 25370-2020**《铸造机械 术语》：明确震实（jolt compaction / jolt ramming）定义，统一震实制芯机、jolt table 等设备的中英文术语，并将震实归属于紧实工艺类别。[[S20,S6]]
- **ZBJ 61003-88**《惯性振实台》：规范惯性振实台的支架式、辊道式两种基本型式；安装倾斜度要求支架上平面 ≤0.3/1000，工作台升起后 ≤0.8/1000；空运转连续测试 ≥1 h；首次大修工作时长 ≥5000 h；配套自硬砂在合适工作周期内紧实率应大于 10%。[[S26]]
- **JB/T 5772-91**《电动激振器》：规范电动激振器额定频率 50 Hz，额定电压 660/380/220 V，外壳防护等级 IP55、接线盒 IP56，B 级绝缘，额定工况温升符合标准限值。[[S26]]
