## 概述

离心铸造是一种将液态金属浇入旋转铸型，在离心力持续作用下完成充型与凝固的铸造工艺方法[[S34,S37,S49]]。其铸件轴线与旋转铸型轴线重合，多数工业场景用于生产圆筒形、管状、套筒类等旋转对称构件[[S34,S37]]。工艺所适用的材料范围极为广泛，覆盖绝大多数铸造金属与合金，并可拓展至玻璃、混凝土等非金属材料的成形[[S37]]。在工程应用中，离心铸造以铸件组织致密、金属利用率高、无需型芯即可生产中空回转体为主要技术特征[[S10,S18,S44]]。

## 工作原理

离心铸造的核心工作机制基于液态金属在旋转铸型中受到的离心力作用[[S34,S22,S38]]：

*   **充型与成形**：金属液在离心力作用下被甩向型腔外侧，紧贴铸型内壁完成充型，同时克服重力作用避免金属液产生"淋落"现象。对于真离心铸造，可无需型芯自动形成规整的圆筒内孔[[S34,S22,S67]]。
*   **顺序凝固与补缩**：金属在离心力场下实现由模壁指向旋转中心的顺序凝固过程。离心力所提供的持续压力显著改善了凝固过程中的晶间补缩效果，使铸件致密度远高于常规重力铸造[[S18,S34]]。
*   **渣气分离**：密度较低的熔渣、气体和夹杂物因受到的离心力较小，会向铸件内表面（自由表面）偏聚，并可通过后续机加工去除，从而获得组织纯净、力学性能优异的铸件[[S34,S22,S18]]。
*   **离心力计算**：离心力的大小可通过公式 $P = m R \omega^2 = m R (\pi n/30)^2$ 进行计算，其中 $m$ 为合金质量， $R$ 为旋转半径， $\omega$ 为角速度， $n$ 为转速。在合金和铸件尺寸确定后，离心力的控制主要通过调整铸型转速实现。常规工业离心铸造转速区间为250至1500转/分，重力系数（G）范围通常在30至100 G之间[[S34,S22]]。

## 相关术语辨析

在离心铸造的研究与工程应用中，存在若干含义相近或易混淆的英文术语，其具体定义及相互区别如下[[S36,S23,S11,S51,S4,S25,S61]]：

*   **真离心铸造 (True centrifugal casting)**：最主流的工业离心铸造工艺。不使用型芯，金属液被直接甩向旋转模具外壁成形，铸件基本为管状中空体。旋转轴可采用水平、竖直或任意倾斜角度。
*   **半离心铸造 (Semi-centrifugal casting)**：用于制造形状比真离心铸造更复杂的轴对称铸件（如轮坯），可设置型芯。铸型始终沿竖直轴旋转，转速低于真离心铸造。
*   **离心浇注/非真离心铸造 (Centrifuging)**：铸件形状不受回转体限制，铸件轴线无需与旋转轴线重合。多个型腔通过中央直浇道与径向流道呈放射状排列，仅利用离心力提供充型压力以降低孔隙率。
*   **Spin casting (离心橡胶模铸造，CRMC)**：一种小众成型工艺，采用半永久硅橡胶模具，仅适用于低熔点锡铅合金、锌合金或热固性塑料小零件制造。其与常规工业离心铸造在模具、合金种类和产品规模上完全不同，二者互不隶属。

## 工艺分类

离心铸造技术体系可从以下四个核心维度进行分类[[S3,S14,S59,S48,S22]]：

*   **按铸型旋转轴空间位置**：分为**卧式离心铸造**（旋转轴水平或与水平线夹角极小）、**立式离心铸造**（旋转轴垂直）和**倾斜轴离心铸造**（旋转轴与水平线存在较大夹角）。
*   **按离心力作用方式**：分为**真离心铸造**、**半离心铸造**和**非真离心铸造**（离心加压铸造）。
*   **按铸型材质**：分为金属型、衬耐火材料金属型、砂型及其他材料铸型离心铸造。
*   **按铸型工作温度**：分为**冷模离心铸造**和**热模离心铸造**。

### 按旋转轴分类

![图2 离心铸造的两种基本类型：a) 立式离心铸造，b) 卧式离心铸造](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/fc048e36-2eed-448b-a483-08268e899399/resource)

**立式离心铸造**：铸型绕垂直轴旋转，金属液在离心力作用下形成抛物面自由表面，导致铸件沿高度方向呈现上薄下厚的壁厚差异。该工艺适用于生产高度小于直径的圆环类、轮类、轧辊及刹车鼓等铸件[[S59,S27]]。

**卧式离心铸造**：铸型绕水平轴或与之夹角极小的轴旋转，铸件各部分冷却条件均匀，沿长度和圆周方向壁厚均匀。此工艺主要用于生产长度大于直径的各类套筒、管道、缸套及辊类等长回转体铸件[[S59,S54]]。

**倾斜轴离心铸造**：铸型旋转轴与水平线存在明确夹角，可有效改善立式离心铸造带来的铸件内表面抛物面变形问题，典型产品为双金属轧辊、锥形套等特种回转铸件[[S14,S59]]。

### 按离心力作用方式分类

*   **真离心铸造**：完全依靠离心力使金属液贴紧铸型内壁而成型出中空内腔，铸件轴线与旋转轴线重合，不使用型芯。典型产品包括长管道、缸套、空心辊等[[S25,S61,S48]]。
*   **半离心铸造**：铸型整体为轴对称结构，通常设置专用型芯以成型非圆形内孔。离心力起辅助充型与强化凝固补缩的作用，所需铸型转速显著低于真离心铸造。适用于叶轮、辐条轮、圆盘类等铸件[[S14,S59,S25]]。
*   **非真离心铸造**：铸件形状不受回转体限制，离心力仅用于提升凝固时的压力以改善铸件致密度。铸型转速最低，适用于多型腔小尺寸异形铸件的批量生产[[S48,S59,S61]]。

### 按铸型工作温度分类

*   **冷模离心铸造**：金属铸型密封在水套中，通入循环冷却水强制冷却，使铸型本体温度在工作过程中长期维持在约40至250℃区间。此方式适用于大批量快速生产中小型铸件[[S48,S12]]。
*   **热模离心铸造**：铸型仅依靠空冷或少量喷水方式进行冷却，工作温度维持在350至450℃区间。铸型内壁通常预置砂层或耐火涂料，有利于降低铸件表面因激冷产生的白口缺陷[[S48,S12]]。

## 工艺过程

典型的离心铸造工艺可划分为四个核心环节[[S16,S8,S66,S17]]：

1.  **熔炼**：将预处理的炉料装入熔炼设备进行熔化，通过造渣、扒渣净化熔体并进行成分检测。完成终脱氧、球化或变质等精炼处理后，将金属液出炉温度控制在合金熔点以上100至230℃区间。
2.  **模具准备**：完成铸型清理后，将铸型预热至工艺要求温度并均匀涂覆耐火或防粘涂料。待涂料充分烘干后，安装型芯、端盖等附属部件，确保铸型动平衡合格。
3.  **浇注**：提前启动铸型旋转至设定转速。将定量金属液沿铸型旋转方向平稳注入，避免产生飞溅和裹气。对于长管件，浇注过程中需同步移动浇注槽以保证金属液沿轴向均匀分布。
4.  **旋转凝固、拔件与后处理**：浇注完成后，维持铸型转速直至铸件完全凝固，期间可通过覆盖保护渣控制内表面氧化，并调控冷却速度以保证由外向内顺序凝固。待内表面温度降至安全区间后减速停机。随后通过顶杆、拔管机等专用工装取出铸件，依次完成去除浇冒口、表面清理、内孔精加工、缺陷检测和必要的热处理。

## 关键工艺参数

离心铸造的最终质量受多个关键工艺参数的综合影响，主要参数及其典型数据范围如下：

**重力系数（G值）**是衡量离心力大小的核心无量纲参数，其定义为有效重度与单位体积金属熔体重力的比值[[S29,S62]]。不同合金类型的推荐G值范围如下表所示：

| 合金类型 | 推荐重力系数 (G) 范围 | 来源 |
| :--- | :--- | :--- |
| 重合金 | 35 - 100 | [[S53]] |
| 铝合金、镁合金等轻合金 | 80 - 120 | [[S53,S29,S62]] |
| 铜合金 | 50 - 100 | [[S25,S56]] |
| 中小铸铁缸套 | 40 - 90 | [[S9]] |

**铸型转速**的确定可采用Konstantinov公式 $n = \beta 55200 / \sqrt{\gamma r}$，其中 $\beta$ 为修正调整系数，用以补偿理论计算与实际生产的差异[[S53,S62,S46]]。部分材料的调整系数 $\beta$ 值如下：

| 铸件种类 | $\beta$ 值范围 | 来源 |
| :--- | :--- | :--- |
| 铜合金卧式套筒件 | 1.2 - 1.4 | [[S46]] |
| 铜合金立式圆环件 | 1.0 - 1.6 | [[S46]] |
| 铸铁套筒件 | 1.2 - 1.5 | [[S46]] |
| 铸钢套筒件 | 1.0 - 1.3 | [[S46]] |
| 铝合金套筒件 | 0.9 - 1.1 | [[S46]] |

**浇注温度**因合金而异，且由于离心力增强了充型能力，可比同种合金的重力铸造浇注温度略低5至10 K[[S15,S62]]。已知的部分合金浇注温度推荐值如下：
*   6082铝合金：液相线温度以上50至100 K[[S15]]
*   高稀土镁合金：750℃[[S58]]
*   C95700铝青铜：975至1740℃[[S58]]

**铸型预热温度**需平衡防止金属液激冷与避免凝固时间过长导致偏析、晶粒粗大的关系[[S62]]。例如，6082铝合金的H13钢模具推荐预热温度为573 K[[S15]]，灰铸铁铸型用于镁合金的预热温度为150℃[[S58]]。

**涂料与脱模剂**配方因合金类型而异，例如：铸铁件使用以石英粉、石墨粉为基础的配方；铸钢件使用水玻璃基配方；锡青铜采用滑石粉加松香水玻璃的组合；铝合金则采用氧化锌加水玻璃基的涂料[[S2]]。

## 设备与工装

离心铸造机是实现该工艺的核心设备，按结构可分为立式、卧式和倾斜轴离心铸造机三大类[[S1]]。其驱动方式主要有两种：一种是通过电机经传动带或联轴器直接驱动主轴的**直接驱动**，适用于中小型悬臂式离心铸造机；另一种是通过电机驱动托轮旋转，依靠托轮与铸型外壁间的摩擦力带动大质量铸型转动的**摩擦驱动**，适用于大型滚筒式离心铸造机[[S1]]。为保证传动稳定，托轮与铸型中心的支撑角 $\alpha$ 宜控制在 $95^{\circ}$ 至 $120^{\circ}$ [[S1]]。

为实现工艺柔性，主流离心铸造机多采用交流整流子电机、直流电机或变频电机实现无级调速，以便在浇注和凝固过程中动态调整转速[[S35,S47]]。典型的专用设备，如水冷金属型离心铸管机，集成了移动离心主机、浇注系统、拔管机和水冷循环系统，专门用于大批量生产直径1000mm以下的离心球墨铸铁管[[S66]]。

## 优点与局限性

### 优点

*   **铸件质量高**：在离心力场下顺序凝固，补缩效果优异，铸件外层组织致密，气孔、缩松、夹渣等缺陷率显著低于重力铸造。同时，低密度夹杂物向铸件内表面偏聚，可通过机加工去除，保证了有效工作层的高纯净度和力学性能[[S18,S36,S44]]。
*   **工艺出品率高**：生产中空轴对称铸件时无需型芯，并且多数情况下可完全省去浇注系统和冒口，金属利用率可达90%至95%，典型球墨铸铁离心铸管的工艺出品率可超过96%[[S10,S44,S41]]。
*   **双金属复合能力**：可便捷地制备双金属复合铸件，例如钢背铜套、双金属轧辊等，从而获得外层耐磨耐蚀、内层强韧的梯度性能结构，显著降低了综合生产成本[[S18,S36,S19]]。
*   **特定形状的高效生产**：对于管、筒、套、环等旋转对称铸件，离心铸造是一种节省材料、能耗和生产环节的高效工艺[[S10]]。

### 局限性

*   **形状适应性有限**：工艺经济性最高的领域集中于中空轴对称铸件，对于异形铸件的适配性较差，应用范围受限[[S10,S33,S18]]。
*   **内表面质量与精度不足**：完全依靠离心力形成的自由表面尺寸精度差、表面较为粗糙，往往需要预留更大的加工余量并进行后续机加工[[S10,S33,S18]]。
*   **存在成分偏析风险**：对于凝固过程中析出相或第二相与金属基体密度差异较大的合金（如碳当量过高的球墨铸铁），极易在离心力作用下产生密度偏析[[S10,S33,S18]]。
*   **设备投资高昂**：专用离心铸造机结构复杂，造价昂贵，导致车间前期投资高于常规铸造方法，在单件或小批量生产场景下经济性不佳[[S10,S33]]。

## 常见缺陷与防治

离心铸件在生产过程中可能产生一些特有的铸造缺陷，其成因与防治措施如下表所示[[S55,S60,S30,S21,S63,S67]]：

| 缺陷类型 | 主要成因 | 防治措施 |
| :--- | :--- | :--- |
| **成分偏析** | 离心力场下不同密度的固液相或析出相发生相对移动 | 提高冷却速度，合理降低重力系数（G值），选用凝固区间窄的合金 |
| **缩孔** | 未建立由型壁指向中心的定向凝固顺序 | 控制凝固方向（由外向内顺序进行），厚壁件采用薄涂层与水冷铸型，预留加工余量 |
| **裂纹** | 纵向裂纹多因设备动平衡不佳导致振动；横向裂纹多由铸件收缩受阻或铸型表面缺陷引起 | 优化设备动平衡，降低不合理转速，避免形成粗大柱状晶，改善铸型表面质量 |
| **气孔** | 熔体中未排出的气体、涂料或脱模剂分解产生的气体被滞留 | 严格进行熔体精炼除气，控制涂料喷涂厚度并充分烘干 |
| **坍流** | 转速过低、停机过早或浇注温度过高，导致金属液淋落 | 待铸件整体温度低于固相线100至300℃后再停机，避免砂型局部过热 |

## 应用领域

离心铸造技术已在多个工业领域获得大规模、高普及度的应用[[S18,S40,S10,S45]]：

*   **铸铁管**：全球范围内每年生产的球墨铸铁件近半数由离心铸造工艺制造。离心球墨铸铁管是市政给排水管网系统的主流管材。
*   **发动机缸套**：内燃机汽缸套几乎全部采用离心铸造工艺进行批量生产，是该工艺最具代表性的应用之一。
*   **铜合金轴套与滑动轴承**：离心铸造生产的铜合金轴承材料，其力学性能普遍优于砂型铸造和连续铸造产品，广泛应用于各类机械装备。
*   **双金属轧辊**：离心铸造是制备兼具外层耐磨、内层强韧特性的复合轧辊的主流工艺之一，在冶金行业中应用广泛。
*   **环形与盘状轴对称工件**：普遍用于飞轮、造纸机滚筒、高速钢套圈等零部件的批量生产。

## 离心铸造与其他铸造工艺的对比

离心铸造与砂型铸造、压力铸造、连续铸造等常见工艺在技术经济指标上存在显著差异。综合多项资料，核心指标的定性或定量对比汇总如下[[S20,S64,S13]]：

| 对比指标 | 离心铸造 | 砂型铸造 | 压力铸造 | 连续铸造 |
| :--- | :--- | :--- | :--- | :--- |
| **适用合金** | 几乎所有铸造合金 | 不限 | 多用于非铁合金（铝、锌、镁等） | 多用于铸铁、铸钢、铜合金 |
| **铸件重量范围** | 从数克到数十吨 | 不限 | 多为中小件 | 截面形状限制，可生产长材 |
| **壁厚 (mm)** | > 8 | > 3 | < 1（薄壁件优势） | 与截面相关 |
| **尺寸公差** | 中等 | 较低 | 高 | 较低 |
| **表面粗糙度** | 外表面较好，内表面粗糙 | 较粗糙 | 光滑 | 较粗糙 |

## 工艺标准

涉及离心铸造的部分国际标准与行业规范包括[[S50]]：

*   **ASTM B271-08**：铜合金离心铸件标准。该标准规范了铜合金离心铸造产品的化学成分与力学性能要求。
*   **ASTM B208-06**：铜合金拉伸试样制备标准。统一了离心铸件力学性能测试的试样制备方法与标准流程。
*   **ISO 4382/1:1991**：滑动轴承用铸造铜合金国际标准。其中专门针对离心铸件特性，对锡含量的允许波动范围给出了特殊规定。
