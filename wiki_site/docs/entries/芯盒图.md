## 定义与术语定位

芯盒图是指导芯盒制造、检验与装配的专用工程图样。在国家标准 GB/T 5611-2017《铸造术语》中，“芯盒图”被列为独立的规范术语，对应英文为 core box drawing [[S13,S19]]。芯盒图本质上属于铸造工艺装备图纸体系，既区别于表达压铸模的模具总装图和零件图，也区别于仅承载几何形状信息的三维数模，是一种承载完整制造要求（尺寸公差、形位公差、材料热处理、表面处理等非几何技术条件）的二维法定工艺文件 [[S18,S32]]。

在压铸模具设计制造场景下，压铸模专属术语标准 GB/T 8847—2003 未单独收录“芯盒图”一词，其实际应用中常作为“芯盒”词条下的配套附属工艺图样出现 [[S5]]。芯盒图包含装配图和分图两类图样，需明确标注芯盒的材料、结构、紧固和定位方式等关键信息 [[S8,S18]]。

## 芯盒图的图样类型与简化规范

芯盒图的绘制方法在工业实践中可分为两类：标准机械制图方法（先绘总图，后逐一拆件绘制零件图）和简化设计法。简化设计法已在铸造工装领域形成一套行之有效的规范，适用于普通金属芯盒、热芯盒及壳芯盒的设计 [[S32]]。

**简化画法的核心规则：**

1. 简单芯盒可直接在总图上标注全部加工尺寸，无需为各组成零件单独拆分出图 [[S32]]。
2. 定位销、锁紧螺钉、螺母、蝶形螺母等标准紧固件无需绘制全形或剖视图，仅从安装孔中心线引出件号并在零件明细表中注明标准件规格即可 [[S18,S32]]。
3. 螺钉紧固方式、夹紧装置、模板定位销/导向销及配套螺母垫圈、工厂标准直浇道模/阻流片等，均可通过同一条引出线标注多个件号，引出点可从孔或中心线上选取 [[S8]]。
4. 视图布置应以芯盒在实际工作位置摆放时的主视图为准 [[S32]]。

这也意味着芯盒图属于一种专用的特殊工程图样类型，其出图方式显著区别于常规压铸模具零件图——后者需针对每一个独立加工零件分别绘制完整标注视图 [[S32]]。

## 图面要素与技术要求

芯盒图作为制造依据，需覆盖以下核心信息类别：

### 1. 芯腔尺寸计算与标注

芯盒图图面上芯腔尺寸必须严格按以下公式计算后标注 [[S7,S22]]：

\[
A = (A_C + A_Y) (1 + \varepsilon)
\]

其中 \(A\) 为最终芯腔尺寸，\(A_C\) 为产品图标准尺寸，\(A_Y\) 为铸造工艺尺寸（含加工余量、起模斜度、工艺补正量），\(\varepsilon\) 为对应铸件的线收缩率。标注基准应尽可能与对应铸件图基准统一，并便于机械加工划线及直接测量 [[S3,S32]]。

### 2. 强制技术要求

芯盒图上通常以“技术要求”文字条款形式标注以下内容 [[S8,S18]]：

| 项目 | 典型要求值 | 说明 |
|:---|:---|:---|
| 毛坯材料牌号 | ZL101、HT200 等 | 芯盒本体毛坯加工前须进行人工时效处理 [[S18,S23]] |
| 工作表面尺寸公差 | ≤0.2 mm（常规负向公差 -0.2 mm） | 工作面按 IT11 级精度或等效光洁度等级制造 [[S21,S8]] |
| 工作表面粗糙度 | Ra ≤ 1.6 μm（对应原机加工 ∇6 等级） | [[S18,S23]] |
| 分盒面平行度 | ≤0.05:100 | 分盒面光洁度不低于 Ra1.6 μm [[S23]] |
| 分盒面垂直度 | ≤0.02:100 | [[S23]] |
| 未注铸造圆角 | R4 mm 等 | 按铸件尺寸大小取值 [[S8]] |
| 未注铸造斜度 | 2° | 同一套芯盒应尽可能统一采用单一起模斜度 [[S3,S18]] |
| 缩尺标注 | 手工金属芯盒实例中 1% 缩尺已纳入标准加工尺 | [[S8]] |

### 3. 配合面公差

各个关键配合面的允许间隙/偏差要求需在芯盒图上明确或引用通用技术条件 [[S21,S29]]：

| 配合关系 | 允许间隙/偏差 | 备注 |
|:---|:---|:---|
| 普通对开式芯盒分合面 | ≤0.1 mm | |
| 热芯盒分合面（公称尺寸 <500 mm） | ≤0.05 mm | |
| 热芯盒分合面（公称尺寸 >500 mm） | ≤0.1 mm | |
| 活块与芯盒活块窝座 | ≤0.2~0.3 mm | |
| 凸台中心距 ≤500 mm | ±0.15 mm | |
| 凸台中心距 >500 mm | ±0.20 mm | |
| 圆棒芯盒最远两盒腔中心距 | ≤±0.1 mm | |
| 芯盒定位销位置 | ≤±0.25 mm | |

### 4. 图纸标题栏

符合国内压铸工装制图规范的芯盒图标题栏常规必填字段包括：模具编号、产品名称/产品号、型芯材料牌号、对应铸件收缩率、绘图比例、单位（默认毫米）、图幅、投影视角、设计/审核/批准签字栏及图纸张数序号 [[S26,S24]]。

## 分类

按表现形式和结构对应关系，芯盒图通常与芯盒实体的分类一一对应。金属芯盒按结构形式可分为 **敞口整体式、对开组合式、套框脱落式** 三大核心类别，按分盒面特征可细分为单面敞开式、单面脱落式、垂直对开式、水平对开式、曲折分盒面等子类 [[S9,S1,S22]]。对应地，常见芯盒图亦相应表达这些不同结构类型的制造信息。

|:---|:---|
| **芯盒**（实体工装） | 芯盒是将芯砂成型为型芯的实体工艺装备，芯盒图是用于指导该实体工装加工制造的二维工程图样。二者为设计输出文档与实物产品的对应关系 [[S1,S9]]。 |
| **模具总装图** | 模具总装图表达整套压铸模全部零部件的配合装配关系；芯盒图仅针对制芯用芯盒这一单一工装，不属于压铸模本体图样序列，两者覆盖对象独立 [[S17,S15]]。 |

## 在模具设计与制造流程中的角色

芯盒图是制芯工装加工、检验与标准化传递的法定工艺文件，在以下环节中起基准依据作用 [[S32,S18,S6]]：

- **数控加工**：作为编程工序单的基准依据，指导芯盒本体、镶块的铣削与车削加工；
- **装配检验**：作为三坐标检测、钳工修配的基准文件，用于核验分盒面配合间隙和芯腔尺寸精度；
- **标准化设计**：作为同型号芯盒工装的统一设计模板，保证批量生产砂芯的尺寸一致性。

## 常见局限性与易出错场景

在工程实践中，芯盒图纸环节存在以下典型问题 [[S2,S27]]：

1. 二维投影表达复杂内部砂芯型腔特征困难——若仅采用三视图而未设置足够的局部剖视图，容易遗漏异形内腔、局部凸台、通气槽等关键成型特征。
2. 简化画法使用不当时，可能漏标标准件安装位置、夹紧装置安装基准等工艺信息。
3. 设计疏漏常见形式：拔模斜度标注错误、铸件收缩率未计入加工尺寸、定位销位置尺寸超差，最终导致后续制芯过程错边和砂芯尺寸超差。
4. 工艺装备图纸本身的错误（投影错误、尺寸漏标、收缩量/拔模斜度/加工余量数值不正确）是导致铸件尺寸与几何形状不合格的首要因素。
