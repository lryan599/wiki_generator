## 概述

低压铸造工艺参数（low pressure die casting process parameters）是指在低压铸造过程中，用于控制金属液充型与凝固行为的一系列可调节物理量。其核心是保温炉内金属液面上的气体压力-时间曲线，各工艺参数的设定直接决定充型平稳性、补缩效果及铸件最终致密度与力学性能[[S13,S5]]。

国家标准GB/T 28688‑2012《低压铸造机 技术条件》将工艺参数的控制精度定义为“每次浇注时保温炉内金属液面上的气体压力-时间曲线与给定曲线的接近程度”，并要求加压速率、升液压力/时间、充型压力/时间、结壳时间、增压时间、保压压力/时间、液位补偿压力等9项参数应方便可调[[S5,S2]]。

低压铸造全流程通常划分为升液、充型、增压、保压、卸压5个阶段，各阶段分别由相应的压力、速度、时间和温度参数协同控制[[S13,S11]]。

## 工艺参数分类

低压铸造的工艺参数可划分为压力相关、速度相关、时间相关、温度相关、传热相关五大类[[S13,S11]]：

| 参数类别 | 包含参数 | 对应工艺阶段 |
|---------|---------|-------------|
| 压力相关 | 升液压力、充型压力、增压压力、保压压力、液位补偿压力 | 升液、充型、增压、保压 |
| 速度相关 | 升液速度（升液增压速度）、充型速度（充型增压速度）、升压增压速度 | 升液、充型、增压 |
| 时间相关 | 升液时间、充型时间、结壳时间、增压时间、保压时间 | 全程 |
| 温度相关 | 浇注温度（浇铸温度）、模具预热温度 | 浇注前及充型阶段 |
| 传热相关 | 金属液-铸型传热系数、铸型间传热系数、铸型-大气传热系数、冷却方式 | 凝固冷却阶段 |

## 主要参数定义与计算

### 升液压力与升液速度

升液压力（p₁）是指金属液在升液管内上升至浇口处时，保温炉内所需的气体压力[[S5,S11]]。其理论计算公式为[[S11,S10]]：

```
p₁ = μρgh₁
```

式中，μ为充型阻力系数（1.0～1.5，阻力小取下限，阻力大取上限）；ρ为金属液密度（kg·m⁻³）；g为重力加速度（m·s⁻²）；h₁为金属液面至浇道的高度（m）[[S11,S10,S7]]。

升液速度（v₁）是升液阶段金属液上升至浇口的速度。为防止金属液自浇口进入型腔时产生喷溅飞溅，并使型腔中气体易于排出，升液速度一般控制在5～15 cm/s（铝合金）[[S11,S10]]。在压力控制模式下，升液增压速度（即压力的增长速率）一般控制在2.5～4 kPa/s[[S13]]。升液阶段所需时间由 t₁ = h₁/v₁ 确定[[S1]]。

### 充型压力与充型速度

充型压力（p₂）是指金属液充满铸型时保温炉内所需的压力[[S5,S11]]。其计算公式为[[S11,S10]]：

```
p₂ = μρgh₂
```

式中，h₂为型腔顶部与坩埚中金属液面的距离（m）；μ为充型阻力系数（1.0～1.5），充型阻力小的厚壁铸件取下限，充型阻力大的薄壁铸件取上限[[S11,S1]]。

充型速度对铸件成形质量影响极大：充型速度过慢会使铝液在流动过程中温度大幅下降，导致浇不足；充型速度过快则易引发飞溅、紊流、裹气及氧化物夹渣[[S10]]。铝合金充型速度一般控制在2～4 cm/s，对应的充型增压速度为0.4～1 kPa/s[[S13]]。对于带砂芯的铸件，充型增压速度应取下限（0.4 kPa/s），以避免充型过快冲击砂芯造成砂芯断裂[[S13]]。

### 增压压力与增压时间

增压阶段是充型结束后继续升高压力，使铸件在较高压力下凝固。增压压力（结壳增压值）一般比充型压力高0.5～1倍[[S13]]。增压时间（从充型压力升至保压压力所需的时间）内的升压增压速度一般控制在0.5～1.5 kPa/s[[S13,S5]]。

值得注意，文献中所见的增压速度0.4 kPa/s与1.5 kPa/s并不存在直接冲突：0.4 kPa/s是充型阶段的增压速度推荐值（适用于有砂芯的铸件），而1.5 kPa/s是充型完成后增压阶段的升压增压速度上限值，两者分别对应不同的工艺阶段[[S13]]。

### 保压压力与保压时间

保压压力（holding pressure）是铸型充满后，为补缩需要而在保温炉内金属液面上施加的、比充型压力更高的压力[[S5]]。保压时间（holding time）是金属液充型后在保压压力下所持续的时间[[S5]]。

实验研究表明，A356‑T6铝合金在保压压力从85 kPa提升至300 kPa的过程中，铸件的抗拉强度、屈服强度和伸长率均随之升高，高保压压力可细化微观组织、消除孔隙缺陷[[S12]]。

保压时间的选择应以浇口部位达到约90%凝固为判断依据。保压时间过短会导致浇口补缩不足，产生缩孔缺陷；保压时间过长则会使浇口过早凝固堵塞，脱模时拉断浇口或导致铸件变形[[S13]]。

### 结壳时间

结壳时间（shelling time）是在充型结束后、充型压力不变的条件下，铸型内金属液凝固成金属薄壳所需的时间[[S2,S5]]。该参数主要用于控制铸件表面凝固壳的形成时机，为后续增压补缩提供合适的凝固前沿条件。

### 液位补偿压力

液位补偿压力是为了保证充型开始以后各工艺参数的重复再现而不受金属液位变动的影响，在每一次后续浇注前由系统自动增加的、对应于液位下降所对应的成型压力增量[[S5]]。

## 典型参数范围与工程案例

### 低压慢速充型策略

铝合金升液速度推荐控制在10～15 cm/s（对应升液增压速度2.5～4 kPa/s），充型速度推荐控制在2～4 cm/s（对应充型增压速度0.4～1 kPa/s）。该参数配置可使金属液处于层流状态，避免湍流卷气与二次氧化，保证型腔中气体平稳排出，尤其适用于带砂芯的复杂薄壁铸件及内部带有大量加强筋的中空结构铸件[[S13,S7]]。

## 参数对铸件质量的影响

### 保压压力的影响

在85 kPa到300 kPa范围内提高保压压力，A356‑T6铝合金铸件的抗拉强度、屈服强度及伸长率均随之升高。高保压压力可细化α‑Al枝晶并改善共晶Si粒子形态，同时显著消除缩松等孔隙缺陷[[S12]]。

### 保压时间的影响

保压时间过短会导致浇口部位补缩不足，产生缩孔缺陷；保压时间过长则会使浇口杯内铝液凝固引起浇口堵塞，脱模时拉断浇口或将铸件拉变形[[S13]]。

### 充型速度的影响

充型速度过慢会使金属液在流动过程中温度大幅下降，导致流动性降低，易产生浇不足等缺陷；充型速度过快则引发飞溅、紊流、裹气，加剧铝液氧化程度，若型腔中气体来不及逸出，还会导致气孔缺陷或表面轮廓不清晰[[S10]]。

### 浇注温度的影响

低压铸造的浇注温度通常比重力铸造低15～20 ℃（用户提示中存在该说法，当前证据中未直接提供与其他铸造方法的系统比较数据，但A356铝合金后副车架的优化浇注温度为707.89 ℃，ZL201副车架的重力砂型铸造浇注温度为700 ℃而金属型低压铸造为720 ℃）[[S8,S3]]。

## 相关关系

- **升液压力与充型压力**：两者均基于同一理论公式 p=μρgH 计算，区别仅在于金属液上升高度H的不同（升液高度h₁对应升液压力，型腔总高度h₂对应充型压力）[[S11,S10]]。
- **保压压力与保压时间**：二者共同决定铸件凝固补缩效果，属于凝固阶段的关键耦合参数[[S13,S12]]。
- **充型阻力系数μ**：取值范围1.0～1.5，针对简单厚壁件取下限，复杂薄壁件取上限，直接影响各阶段压力的理论计算值[[S11,S10,S1]]。
