## 概述
粘结剂预热器（Binder pre-heater）是铸造机械中砂处理工段的一种专用辅助装备，其核心功能是对进入混砂机前的粘结剂进行预热，使其达到规定的温度区间并保持粘度稳定。依据国家标准定义，该装置归入“造型制芯设备”的配套子系统，而非压铸机主机本体的直接组成部分[[S9,S16]]。

在压铸工厂中，粘结剂预热器主要用于压铸件所需砂芯（如树脂砂、壳芯、冷芯盒或热芯盒芯）的制备工段，不属于压铸机主机配套系统，不存在“模涂料预热器”“脱模剂预热器”等被标准认可的同义工业别称[[S9,S16]]。

## 定义
根据GB/T 25370-2020《铸造机械 术语》，粘结剂预热器定义为：对进入混砂机前的粘结剂进行预热，将其控制在指定温度范围内，使粘结剂粘度保持稳定的专用装置[[S9]]。

## 分类
粘结剂预热器可从加热方式、安装位置及适用场景三个维度进行分类[[S9]]：

| 分类维度 | 类别 | 说明 |
|---|---|---|
| 按加热方式 | 电加热式 | 以电阻加热元件直接提供热源 |
| | 导热油恒温式 | 以导热油为热传导介质，实现间接均匀加热 |
| | 水浴加热式 | 以水为热传导介质进行加热 |
| 按安装位置 | 管路在线式 | 安装于输送管路中，物料流经时被加热 |
| | 储料罐集成式 | 加热单元集成于储料罐体上 |
| 按适用场景 | 自硬树脂砂生产线 | 配套树脂砂混砂系统 |
| | 覆膜砂生产线 | 配套覆膜砂制备工艺 |
| | 冷芯盒/热芯盒制芯线 | 配套各类制芯工艺 |

## 工艺作用
粘结剂预热器在砂芯制备工段具有以下工艺作用：

1. **降低流变粘度**：预热可显著降低粘结剂的自身流变粘度，提升混砂环节粘结剂与原砂的混合均匀性[[S1]]。

2. **保证定量输送精度**：低温高粘度状态下粘结剂易出现定量偏差，导致砂芯强度波动；预热后将粘度稳定在适宜范围，可保证粘结剂定量输送精度[[S1]]。

3. **调控溶剂挥发效率**：通过调节预热温度，可在可控工艺范围内调整溶剂与水分的挥发效率，提升砂芯涂层与基体的结合力[[S1]]。

粘度与型砂流动性的关联性已被实验研究证实：随着粘结剂粘度降低（对应预热升温效果），型砂流动性先上升后趋于平稳[[S1]]。

## 结构与组成
粘结剂预热器可采用电阻式发热管作为核心加热元件，配套导热油或循环水作为热传导介质，通过热对流实现物料均匀加热[[S11]]。典型结构包含以下组成部分：

- **加热元件**：电阻式发热管，为系统提供由电能转化的热能[[S11]]。
- **导热介质**：导热油或循环水，将热量传递至物料[[S11]]。
- **加热腔体**：容纳物料与导热介质进行换热的容器[[S2]]。
- **搅拌机构**：加装于加热腔体内，通过强制对流使物料温度均匀化，防止壁面物料局部过热[[S2]]。
- **保温层**：在加热腔体外壁包裹保温棉层，降低热辐射散失，提升热利用效率[[S7]]。
- **温控系统**：包含上限温度联锁报警机构，将加热温度严格限定在粘结剂允许的安全温度区间内[[S4]]。

设计时需校核发热管换热面积，避免发热管表面温度过高导致导热介质结焦碳化[[S11]]。

## 工艺参数
不同压铸合金配套砂芯制备场景下，粘结剂预热参数存在明显差异[[S13,S10]]：

| 压铸合金类型 | 预热控制要求 | 原因 |
|---|---|---|
| 镁合金 | 预热目标温度需控制在较低区间 | 避免粘结剂活性过高导致型砂发气量超标，引发镁液充型安全隐患 |
| 铝合金 | 预热温度可适当提升 | 保证砂芯强度以适配铝合金金属液的高冲刷性能 |
| 锌合金 | 预热控制精度要求较低 | 低熔点合金对砂芯性能要求门槛较低 |

## 常见问题与解决措施
粘结剂预热器在长期运行中可能出现以下问题：

| 常见问题 | 原因 | 解决措施 |
|---|---|---|
| 过热导致粘结剂失效 | 加热温度过高引发高分子组分热降解 | 设置上限温度联锁报警机构，将加热温度限定在安全温度区间内[[S4]] |
| 加热温度分布不均 | 腔体内物料受热不均 | 在加热腔体内加装搅拌混合结构，通过强制对流使温度均匀化[[S2]] |
| 腔体与管路结垢堵塞 | 残留物料在腔体内壁和输送管路内壁结垢 | 定期停机拆解清理加热腔和输送管路；可增设在线反冲洗清洁机构[[S15]] |
| 热损失大、能耗过高 | 非优化结构导致热辐射散失大 | 在加热腔体外壁整体包裹保温棉层，降低热辐射散失[[S7]] |

## 应用场景
粘结剂预热器主要用于压铸工厂的配套砂芯制备工段，服务于压铸生产所需的预制型/砂芯制造环节。使用预热后粘结剂制备的压铸砂芯为压铸件生产提供必要的型芯支撑，不同合金类型对粘结剂预热的温控精度和温度区间要求各有侧重[[S13,S10]]。
