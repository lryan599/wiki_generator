## 概述

金属熔体吸气是铸造与压铸领域的核心熔炼特性之一，属于过程现象类术语。该术语定义为金属在加热和熔炼过程中，其固态及液态从周围环境（炉气、空气、水分等）吸收氢、氧、氮等气体的行为。溶解进入金属的气体会在后续凝固过程中引发铸锭气孔、缩松、板带材起泡及分层，甚至导致氢脆，显著劣化材料的力学性能与气密性[[S17,S5]]。

需要特别注意的是，**吸气（gas absorption/gas pickup）与卷气（gas entrainment）属于两类性质完全不同的气体引入过程**。吸气本质上是化学过程：气体分子需在熔体表面发生解离，以原子形式固溶进入熔体内部，是由界面化学反应与体相扩散共同作用的结果。卷气则是纯粹的物理过程，指熔体在流动或充型阶段机械裹挟气泡，气体以独立气泡形态悬浮于熔体中，未发生真正的溶解[[S32,S38,S54]]。这一区分对理解压铸缺陷根源及其控制策略具有关键意义。

工业铸造领域中，所谓金属吸气主要指向**吸氢**。氢原子半径仅为0.037 nm，几乎可溶解于所有工业金属与合金中，因此熔体总含气量通常可近似等效为含氢量[[S17]]。在铝合金熔体中，氢占溶解气体总体积的70%～85%以上，这一现象尤为突出[[S46,S37]]。

## 吸气机理

金属熔体的完整吸气过程由**吸附**与**扩散**两个核心阶段构成。从微观上看，该过程可拆解为四步连续机制[[S49,S55]]：

1. 气体分子碰撞至金属液表面；
2. 吸附态气体分子在熔体表面解离为原子；
3. 单原子气体通过化学吸附固定于熔体表层；
4. 气体原子由熔体表层向内部扩散，完成溶解。

在上述四步中，**扩散过程为控速环节**，决定了整体的吸气速率[[S1,S48]]。

### 吸附阶段

吸附分为两类机制[[S49,S55]]：

- **物理吸附**：由金属表层原子表面力场对气体分子的弱黏附作用引起。仅能形成单分子层吸附，吸附态气体始终保持分子形态，无法直接被吸收。随温度升高、气体分压降低，物理吸附作用减弱。
- **化学吸附**：属强选择性吸附，需金属与气体之间存在亲和力。气体分子在此阶段完成解离为原子的过程，这是金属能够吸收气体的必经过渡步骤。惰性气体（如氦、氩）因与金属无亲和力，仅能发生物理吸附，无法溶解进入熔体[[S49]]。

### 扩散阶段与溶解度定律

被吸附在金属表面的气体原子，需由浓度较高的表面向浓度较低的内部扩散，方可完成溶解。扩散动力学遵循费克第一定律与平方根定律。对于双原子气体，其在金属中的溶解度与对应气体分压的平方根成正比，即**Sievert定律（平方根定律）**[[S48,S11,S24]]：

$$C = K\sqrt{p}$$

式中，\(C\) 为气体溶解度（常用单位：cm³/100g或mL/100g），\(K\) 为溶解度常数，\(p\) 为气体分压。

气体通过金属单位表面积的扩散速度可进一步表示为[[S48]]：

$$V_{\text{扩}} = -\left(D_0K/x\right)\sqrt{p}\exp\left[-E_D/(2RT)\right]$$

由上式可见，**气体分压越大、温度越高、扩散系数越大，金属吸气速度就越快**[[S48]]。对于铝合金，溶解氢的扩散系数与温度、氢分压的关系同样满足上述规律[[S47]]。

## 常见吸气种类与来源

### 可被熔体吸收的主要气体

不同气体在金属熔体中的溶解特性存在显著差异[[S17,S43]]：

- **氢气（H₂）**：氢原子半径极小（0.037 nm），几乎可溶解于所有金属及合金中，是铸件气孔形成的最主要气体。氢占铝合金熔体溶解气体的绝对主导地位。
- **氧气（O₂）**：性质极活泼。在铁、镍、铜等金属熔体中可直接溶解形成氧化物固溶体；而在铝、镁等活泼金属中，氧仅能生成不溶性氧化物夹杂（如Al₂O₃、MgO），并不直接固溶于熔体。
- **氮气（N₂）**：在铁中有一定溶解度，在多数有色金属（如铜、锡、锌、镉、铅等）中几乎不溶解。
- **水蒸气（H₂O）**：是熔炼过程中诱发金属吸气的**最主要气体化合物**。水蒸气在高温下与金属反应，生成的氢和氧极易被熔体吸收[[S5]]。
- **二氧化碳（CO₂）**、**一氧化碳（CO）**、**二氧化硫（SO₂）**：SO₂能与铜、镍、铁等反应，增加金属含硫量及含氧量；CO和CO₂在某些条件下也可能参与气孔反应[[S5,S2]]。

### 气体来源的五大类别

熔体中溶解气体的全部常见来源可归为以下五类[[S5,S22,S13,S2,S11]]：

| 来源类别 | 具体内容与机制 |
|---------|-------------|
| 炉料来源 | 金属炉料自身固溶的气体；表面吸附的水分、锈蚀层、残留油污/乳液、电解质残留（如电解铜板上CuSO₄·5H₂O）等。潮湿季节露天堆放时吸附水分尤为严重。 |
| 炉气来源 | 燃料燃烧生成的水蒸气、碳氢化合物；炉内高湿度环境。环境大气水汽分压过高时（如南方梅雨季节），即使电阻反射炉内也会显著提升铝合金等活泼金属的吸氢量。 |
| 耐火材料来源 | 新炉或长期停炉后，耐火材料及坩埚表面吸附的水分未彻底烘除，是初始熔炼阶段大量吸气的重要诱因。炉渣及残留熔剂亦可吸附水分。 |
| 辅助材料来源 | 含结晶水的熔剂（如潮解的氯化镁等盐类）、未充分干燥的精炼气体。精炼气体中水分含量过高会显著降低除气效果。 |
| 操作来源 | 与熔体接触的浇包、搅拌工具表面吸附水分未彻底烘烤去除；铸型油基涂料高温挥发生成的氢源。浇注过程中熔体转注落差过大、液流溅射亦会引发二次吸气。 |

### 铝合金熔体中氢的核心来源

对于铝及铝合金，熔炼和保温阶段吸气的核心氢源为环境中的**水蒸气**。水蒸气与铝液发生反应生成原子态氢后溶解进入熔体[[S13,S37]]：

$$2\text{Al} + 3\text{H}_2\text{O} \rightarrow \text{Al}_2\text{O}_3 + 3\text{H}_2$$

炉气中碳氢化合物在高温下分解也可提供原子态氢。而直接的分子态氢无法大量溶解进入铝液[[S37]]。研究指出，在30%相对湿度的普通天气条件下，750°C时纯铝熔体中的平衡氢含量可达约1 mL/kg（0.1 mL/100g）；若熔体接触潮湿耐火材料或湿气体，环境水蒸气分压接近1 atm时，熔体中氢浓度可跃升至约10 mL/kg[[S16]]。

## 影响吸气量的因素

影响金属熔体实际吸气量的因素涵盖合金特性、工艺参数与操作环境等多方面。

### 合金成分的影响

合金元素对氢的溶解度有显著调制作用。以铝合金为例[[S46,S7]]：

- **降低氢溶解度的元素**：Cu、Si、Mn、Zn；
- **提升氢溶解度的元素**：Ti、Zr、Mg（在镁含量2%～3%时氢溶解度达最大值）；
- **促进吸氢的元素**：Na及Mg会破坏铝表面氧化膜的完整性，使氧化膜变得疏松，进一步促进吸氢过程。

### 温度的影响

温度是影响气体溶解度和扩散速率的核心参数。氢在铝中的溶解为吸热反应，温度升高，溶解度和扩散速率同步增大[[S47,S44]]。纯铝液相线温度（约660°C）时氢溶解度为0.69 cm³/100g，850°C时升至约1.7 cm³/100g；温度升至1000°C时氢溶解度可超过40 mL/kg[[S16,S33,S18]]。**在高温下长时间保温会显著增加熔体总吸气量**。

### 气体分压的影响

遵循Sievert定律（\(C = K\sqrt{p}\)），气体分压越大，平衡溶解度越高[[S48,S11]]。环境中水蒸气分压升高（如高湿度季节）会直接驱动熔体吸氢量增加[[S16]]。

### 熔体表面氧化膜的作用

完整致密的氧化膜可有效阻滞氢向熔体内部扩散。以铝合金为例，表面形成的氧化铝（Al₂O₃）膜具有保护作用；一旦氧化膜被外力搅动、流动冲击等因素破坏，熔体的吸气速率将大幅提升[[S33,S11]]。

### 其他影响因素汇总

| 影响因素 | 作用机制 |
|---------|---------|
| 熔炼时间 | 扩散过程需要时间，接触时间越长，吸气量越多[[S48,S1]] |
| 搅拌与液面扰动 | 破坏氧化膜、增大气液接触面积，加速吸气[[S33]] |
| 接触面积 | 气液接触面积越大，吸气速率越高[[S29]] |
| 熔体停留状态 | 在炉内静置时间过长，持续暴露于炉气，增加含气量[[S1]] |

## 吸气导致的后果

溶解于熔体中的气体在凝固阶段因溶解度骤降而析出，是铸件气孔缺陷的根本驱动力。以铝为例，在熔点温度下液态铝与固态铝的氢溶解度之比接近**19:1**（液相约0.69 cm³/100g，固相约0.036 cm³/100g）[[S33,S18]]。镁合金在700°C时氢溶解度可达50 mL/100g，凝固过程中当含氢量达到约14.5 mL/100g时即大量析出形成显微气孔[[S10]]。

### 力学性能劣化

铝液含氢量从0.15 cm³/100g降至0.10 cm³/100g时，对应铸件的抗拉强度可提升约10%[[S37]]。气孔缺陷大幅降低铸件的有效承载面积，引发应力集中，显著削弱抗拉强度、延伸率和疲劳寿命[[S51,S37]]。

### 致密性与气密性

弥散分布的氢致显微孔隙大幅降低铸件基体致密性，直接导致气密性不合格，出现渗漏问题[[S51]]。

### 氢脆

若溶解氢含量过高，部分沿晶界析出的氢会诱发氢脆，降低材料的韧性及塑性，严重时可引发氢致延迟开裂[[S17,S51]]。

### 压铸件中溶解氢析出气孔与卷气气孔的区分

在压铸场景下，两类气孔的形貌与成因差异显著[[S27,S40,S46]]：

| 对比维度 | 溶解氢析出气孔（吸气后果） | 机械卷气气孔（卷气后果） |
|---------|------------------------|---------------------|
| 形成机理 | 凝固时过饱和氢析出 | 充型时湍流裹挟空气 |
| 形貌 | 细小、弥散均匀分布的球形针孔 | 孔径较大（1～5 mm），孔壁光滑发亮 |
| 尺寸 | 通常小于0.5 mm | 1～5 mm |
| 分布 | 多分布在皮下1～2 mm处，均匀弥散 | 分布无规律，多集中于充型末端、热节区域 |
| 气体种类 | 以氢为主 | 以空气（N₂、O₂）、脱模剂/润滑油挥发气体为主 |

在压铸件总气孔来源中，溶解氢析出的弥散针孔占比可达70%以上，其余为裹挟空气及辅料挥发气体[[S46]]。

## 与压铸工艺的特殊关联

压铸生产中存在区别于常规铸造的二次吸气和卷气风险。

### 熔体输送过程中的二次吸气

熔体从保温炉舀取、通过开放式流道输送至压射室的过程中，若存在过大的液流落差或溅射现象，将破坏熔体表面氧化膜的完整性，引发二次吸气，甚至裹挟大量空气进入熔体[[S11]]。

### 慢压射阶段与卷气

压铸慢压射阶段中，压射冲头的加速度控制不当（dv/ds数值超过2.0～2.5时）会引发液面紊流、飞溅，大幅提升卷气概率。常规压射室内的熔体填充度推荐控制在40%～80%区间，避免过低填充度引发液面紊乱卷气[[S9]]。

### 快压射阶段的气液界面更新

快压射阶段的高流速会快速更新熔体自由表面，**大幅提升气液接触面积，显著加速氢的溶解速率**，同时也会提升裹挟大气泡的概率[[S29]]。压铸充型过程中的填充速度、浇注温度、模具温度、排气设计等参数均与充型阶段卷气、界面更新等有直接关联[[S14]]。

## 预防与控制措施

### 源头预防

源头防控是减少金属熔体吸气的最有效策略[[S23,S52,S25]]：

- 对炉料、熔炼工具提前进行烘干脱水处理；
- 选用符合对应合金成分要求的低含水量合格熔剂；
- 控制熔炼车间环境湿度，避免高湿天气下长时间敞开放置金属原料与熔化工具；
- 回炉料投入前清理干净并进行烘干，新料与回炉料配比中回炉料不超过50%[[S52]]；
- 熔炼温度推荐控制在670°C～760°C范围内，避免过高温度加剧吸气和氧化[[S52]]。

### 熔炼过程保护

- 采用高纯度惰性气体（氩气/氮气）在熔体表面形成连续保护气层[[S41,S16]]；
- 在液面铺展合格熔剂形成隔离覆盖层；
- 采用密闭炉膛结构减少炉内空间与外界高湿空气的交换[[S41]]。

### 除气（Degassing）工艺

#### 浮游除气（气体吹洗法）——基本工艺

向熔体内部通入不与熔体反应的惰性气体（如氮气、氩气），生成大量分散气泡。利用熔体侧氢分压远高于气泡内初始氢分压（\(p_{\text{H}_2} \approx 0\)）的分压差，驱动氢原子向气泡内扩散，随气泡上浮带出熔体实现除氢。**气泡尺寸越小、分布越均匀、上浮路径越长，除气效率越高**[[S41,S53,S36]]。

直接吹洗法操作参数要求[[S35,S6]]：

| 参数 | 要求 |
|------|------|
| 惰性气体氧体积分数 | 低于0.03% |
| 惰性气体水分含量 | 低于3 g/L |
| 氮气压力（铜合金参考） | 0.02～0.05 MPa |
| 常规除气效率 | 30%～50% |

该方法无需复杂动设备，成本较低，但因直接通入的气泡尺寸偏大、分布均匀性差，除气不够彻底，通常作为预处理工艺适配中小容量常规非高要求铸件生产场景[[S6]]。

#### 旋转除气（旋转喷吹除气）——高效工艺

通过高速旋转的石墨转子将通入熔体的惰性气体剪切破碎为微米级微小气泡，实现气泡在整个熔池内均匀弥散分布，大幅增大气液接触面积。对ADC12铝合金的试验表明[[S28]]：

| 参数项目 | 推荐值/效果 |
|---------|-----------|
| 转子转速 | 300～400 r/min（超过400r/min效果反而下降） |
| 氮气流量 | 20～30 mL/min |
| 除气时间 | 6 min以上 |
| 终氢含量 | 可降至0.12 mL/100g Al以下 |
| 除气效率 | 60%～80% |

该方法适配对气密性、力学性能要求较高的压铸铝合金批量在线处理场景[[S26,S28]]。

#### 真空除气——高要求工艺

将密闭炉膛内气压抽至低真空环境，大幅降低熔体表面氢分压，驱动溶解氢自发析出上浮排出。可将高要求铝合金熔体氢含量降至0.1 mL/100g Al以下。为减少低沸点合金元素（如镁、锌）的挥发损耗，需严格控制真空度阈值。该工艺适配小批量高精密特种合金及航空级铸件生产场景[[S35,S57]]。

### 压铸件工艺参数控制

- **慢压射速度**：控制加速度dv/ds不超过2.0～2.5，避免液面紊流飞溅[[S9]]；
- **熔体填充度**：压射室内填充度保持在40%～80%[[S9]]；
- **熔体转注**：保持平稳流动，避免敞露落差和液流冲击，防止氧化膜破裂[[S11]]；
- **涂料使用**：脱模剂和冲头润滑剂使用量应尽可能少，减少挥发气体来源[[S27]]。

## 含气量标准限值

压铸铝合金熔体氢含量的控制限值根据制品等级而不同[[S23,S41,S39]]：

| 制品等级 | 氢含量限值 |
|---------|-----------|
| 常规普通压铸制品 | ≤ 0.15～0.2 mL/100g Al |
| 航空航天类高要求压铸件 | ≤ 0.1 mL/100g Al |
| 航空7075合金（锻件） | ≤ 0.12 mL/100g Al（超过则探伤易出现亮片缺陷） |

国内现行压铸铝合金核心标准包括[[S3,S50,S30]]：GB/T 15115—2024《压铸铝合金》、GB/T 15114—2023《铝合金压铸件》、GB/T 40809—2021《铸造铝合金半固态流变压铸成形工艺规范》。其中GB/T 40809—2021明确规定铝合金液含氢量检测可采用闭路循环法（按YS/T 600执行）或密度当量快速检测法。

国际现行主流标准包括ISO 3522《铸造铝合金 化学成分和力学性能》、EN 1676:2020《铝和铝合金 重熔用合金锭规范》、EN 1706:2020《铝和铝合金铸件 化学成分和物理性能》[[S34,S3]]。

## 氢溶解度数据

### 纯铝中氢的溶解度（0.1 MPa 大气压下）[[S33,S18]]

| 温度条件 | 氢溶解度（cm³/100g） |
|---------|---------------------|
| 850°C（液态） | 约1.7 |
| 658°C（液态，接近熔点） | 0.69 |
| 658°C（固态，接近熔点） | 0.036 |
| 300°C（固态） | 0.001 |

铝液从液态凝固为固态的过程中，氢溶解度骤降约19倍（液固溶解度比约18.8:1），是铸件凝固析出氢气孔的核心诱因[[S47,S18]]。

### 镁中氢的溶解度[[S10,S7]]

| 温度条件 | 氢溶解度（cm³/100g） |
|---------|---------------------|
| 700°C（液态） | 约50 |
| 接近熔点（液态） | 约26 |
| 接近熔点（固态） | 约6～8 |

镁合金固液相氢溶解度之比远小于铝合金（约1.4倍 vs 约19倍），因此溶解氢析出气孔在镁合金中通常不如铝合金严重[[S16]]。
