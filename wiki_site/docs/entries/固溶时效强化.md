## 固溶强化与固溶-时效强化的概念区分

固溶强化（solid solution strengthening）与固溶-时效强化（precipitation hardening / age hardening）是两类不同的强化机制，在压铸铝合金领域需加以区分：

- **固溶强化**：合金元素固溶于基体金属中造成一定程度的晶格畸变，晶格畸变增大位错运动阻力，使滑移难以进行，从而提升合金强度、硬度。其强化效果由尺寸错配度、化学错配度及溶质原子浓度共同决定，强化幅度有限，通常作为合金强化的辅助手段 [[S17,S27,S47]]。

- **固溶-时效强化（沉淀硬化/析出强化）**：包含两个核心热处理步骤的组合强化工艺。第一阶段为**固溶处理**——将合金加热到略低于固相线的温度，保温使可溶组元充分溶入基体后快速冷却，得到亚稳定过饱和固溶体；第二阶段为**时效处理**——将过饱和固溶体在室温或高于室温的温度下保温，促使纳米级共格GP区、亚稳过渡相逐步析出，通过细小析出相阻碍位错运动实现远高于单纯固溶强化的强度提升效果 [[S63,S20,S23,S12]]。

两者核心差异在于：固溶强化仅依靠溶质原子直接溶入基体造成晶格畸变，无额外析出相参与，单一处理步骤的强化上限较低；固溶-时效强化以固溶处理获得过饱和固溶体作为组织准备，后续依靠时效阶段析出的第二相粒子产生更强的位错阻碍效果，可实现更高的强化幅度。铝合金工业应用中的高强度级别几乎都依靠该组合工艺实现 [[S51,S79,S1]]。

典型可时效强化铝合金二元相图中，合金元素在高温下具有较大固溶度且固溶度随温度降低显著减小，这是实现固溶-时效强化的基本条件 [[S50]]。

## 工艺类别与标准代号

固溶-时效强化属于铝合金、镁合金等有色合金的热处理工艺范畴。压铸领域常用的国内外热处理状态代号对照如下：

### 变形铝合金（GB/T 16475-2008 / ANSI H35.1 体系）

| 状态代号 | 定义 | 说明 |
|:---:|------|------|
| T4 | 固溶热处理后自然时效至基本稳定的状态 | 适用于固溶热处理后不再进行冷加工、予以自然时效的产品 [[S24,S43]] |
| T5 | 由高温成型过程冷却后进行人工时效的状态 | 适用于高温成型冷却后不经冷加工、予以人工时效的产品 [[S24]] |
| T6 | 固溶热处理后进行人工时效的状态 | 固溶热处理后不再冷加工，经沉淀热处理显著提高力学性能 [[S24,S43,S67]] |
| T7 | 固溶热处理后过时效/稳定化处理的状态 | 主要用于提升合金抗应力腐蚀、抗剥落腐蚀性能 [[S43,S67]] |

### 铸造铝合金（GB/T 1173 体系）

| 状态代号 | 定义 | 说明 |
|:---:|------|------|
| F | 铸态 | 铸造后自然冷却至室温的状态 [[S5,S76]] |
| T4 | 固溶热处理加自然时效处理 | [[S5,S76]] |
| T5 | 固溶热处理加不完全人工时效处理 | 获得良好综合力学性能 [[S5,S76]] |

### 国际标准等效关系

美国铝业协会AA体系及ASTM标准中的T状态代号与国内GB/T 16475体系基本等效。T7状态典型工艺为分级过时效——100~120°C保温数小时后升至160~180°C保温24h以上，析出的η相优先沿晶界长大，可显著降低7xxx系铝合金应力腐蚀开裂敏感性；T6状态典型工艺为450°C左右固溶数小时后淬火，120°C时效24h获得峰值强度 [[S33,S6]]。ASTM标准与ISO标准的铝合金及热处理状态存在等效对应关系，如ASTM中2014(T4)、2014(T6)、6061(T6)等状态均在ISO Recommendation/R 828 1968(E)中存在对等的代号表述，两者要求相近但不完全一致 [[S49,S41,S56]]。

国内配套测试标准GB/T 13822规定了压铸有色合金力学性能测试用A型、B型拉力试样的形状尺寸要求，适用于T4/T6/T7状态下压铸铝合金抗拉强度、伸长率、硬度的测定 [[S76]]。

### 淬火介质选择

压铸铝合金常用淬火介质按冷却能力从高到低排序依次为：干冰-丙酮混合物（-68°C）、冰水、室温清水、40~90°C热水、沸水、雾化水、200~220°C热油、空气。形状简单的铸件可选用室温清水获得最高冷却速度以保障力学性能；形状复杂易变形开裂的铸件可选择80°C左右热水、热油或PAG聚合物淬火剂以降低冷却速率控制变形开裂风险 [[S75,S21,S16]]。

### 固溶温度与时效温度的影响规律

- **固溶温度**：低于最优值时，Mg₂Si、Al₂Cu等强化相未完全溶解，合金抗拉强度、硬度、延伸率均处于较低水平；高于最优值时，出现共晶相过烧、共晶硅粗化、晶粒异常长大，强度与延伸率显著下降。ADC12合金在固溶温度从480°C提升至510°C时，抗拉强度逐步上升至233 MPa、硬度上升至120 HBW，超过510°C后强度与延伸率同步下降 [[S39,S18,S14]]。

- **时效温度与时间**：时效时间不足或温度过低时，强化相析出不充分，强度、硬度低于峰值；时效时间过长或温度过高时，发生析出相粗化的过时效现象，强度与硬度先达到峰值后逐步下降。ADC12合金在200°C时效过程中，硬度随时间的增加先升高至峰值128 HBW后逐步降低 [[S18,S4,S54,S52]]。

## 强化机理（微观层面）

### 时效析出序列

铝合金时效过程的典型析出顺序为：过饱和固溶体α₃ → 过饱和度降低的固溶体 + GP区 → 固溶体 + 亚稳过渡相θ' → 平衡固溶体α + 平衡析出相θ。根据析出阶段不同可划分为**欠时效**（GP区、亚稳过渡相析出阶段，强度随保温时间上升）、**峰时效**（细小平衡相均匀完全析出，强度达到最大值）、**过时效**（平衡相粗化长大，强度随时效时间延长逐步下降）三个典型阶段 [[S63,S7]]。

#### Al-Cu系合金（以Al-4%Cu为例）

析出序列为：过饱和α固溶体 → GP区 → θ''相 → θ'相 → 平衡θ(CuAl₂)相。各阶段析出相与铝基体的共格关系依次为：完全共格 → 完全共格 → 半共格 → 完全非共格。合金的强化效果随共格应变增大先升高后降低，在θ''相占比最高时达到峰值 [[S32,S59,S69,S30]]。

#### Al-Mg-Si系合金

析出序列为：GP区 → 有序区 → β''(Mg₂Si)亚稳相 → β'(Mg₂Si)亚稳相 → 平衡β(Mg₂Si)相。所有析出相均沿铝基体的{100}取向呈针状分布。优先析出的纳米级盘状Mg₂Si强化相（尺寸约3~6 nm）可在几乎不牺牲塑性的前提下大幅提升合金的强度，是新能源汽车轻量化结构件的主流强化路径 [[S65,S74,S55,S26,S8]]。

#### Al-Si-Cu-Mg系压铸合金

该系合金的时效过程呈现**双峰强化**特征：第一个硬度峰值对应GP区/θ''相和Q'相大量析出，位错切过细小亚稳相占主导；第二个硬度峰值对应亚稳相向θ'相和Q'相转变，位错无法切过大尺寸析出相转而以绕过机制（Orowan机制）运动。两峰之间的硬度谷对应θ''相部分溶解、θ'相尚未完全形核长大的阶段 [[S53,S45]]。

### 位错与析出相交互机制

位错与纳米析出相存在两类典型交互强化机制：

- **切过机制（Friedel效应）**：当析出相尺寸较小且与基体共格时，位错可以直接切过析出相实现滑移。
- **绕过机制（Orowan机制）**：当析出相尺寸较大且为半共格/非共格状态时，位错无法切过，只能绕过析出相并在其周围留下位错环，通过增大位错运动阻力实现强化 [[S53,S68,S57]]。

### 过时效与组织粗化

过时效过程中，随保温时间进一步延长或加热温度进一步升高，析出相发生奥斯瓦尔德熟化——小尺寸析出相溶解，溶质原子向大尺寸析出相扩散聚集，最终得到尺寸大、间距宽的非共格平衡析出相，共格应变消失，对位错的阻碍作用显著下降，合金出现明显软化 [[S78,S59]]。

## 在压铸工艺中的角色与流程位置

固溶时效处理属于压铸成形完成后的后端热处理工序。

### 常规压铸件的固溶处理限制

常规高压压铸件因高速高压充型过程中不可避地存在卷气缺陷，铸件内部含有被压缩的空气及残留润滑剂分解产生的封闭气穴。在固溶加热温度升高时，气穴内部压力同步上升，当压力超过加热后软化的表层金属的屈服强度时，表层金属发生塑性隆起形成鼓包（起泡现象），严重时内部气体突破表层喷出形成破口 [[S38,S71,S2,S77]]。

- 普通压铸A380合金在490°C下固溶20 min、500°C下固溶15 min后即可出现肉眼可见的表面起泡 [[S29,S48]]。
- 采用真空压铸工艺的Al-Si-Cu合金在500°C下固溶2 h时表面起泡尺寸小于100 μm，可满足基本无可见缺陷的要求；固溶超过4 h后粗磨表面出现尺寸约416 μm的可见起泡 [[S48]]。

### 高完整性压铸件的固溶时效强化

采用高真空压铸、间接挤压成形（双控成形）等工艺可大幅减少卷气缺陷，使压铸件具备承受完整固溶-时效热处理的能力 [[S37,S73]]。

"双控成形"（间接挤压铸造）通过控制低速层流充型完全消除卷气缺陷。以ZL104合金为例，经**535°C固溶3 h + 175°C时效5 h**的完整T6热处理后，无明显起泡、变形、开裂缺陷，布氏硬度从铸态的59.95 HBW提升至96.46 HBW，抗拉强度从140 MPa提升至257 MPa，性能提升幅度远高于普通压铸件 [[S73]]。

不同成形工艺制备的铝合金经相同固溶时效处理后的性能差异：压铸合金因冷却速度极快获得更细小的晶粒、更多固溶入基体的合金元素，析出相分布更均匀，最终的抗拉强度和硬度显著高于重力铸造合金；挤压铸造合金因致密度高、缺陷少，在保留高强度的同时还可获得高于压铸合金的塑性 [[S13]]。
