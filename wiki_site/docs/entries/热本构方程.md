## 概述

热本构方程（thermal constitutive equation），又称热变形本构模型或高温流变应力模型，是材料加工与模拟领域中描述高温热变形条件下材料流变应力与应变、应变速率、热力学温度之间定量映射关系的核心力学补充方程，其核心隐式关系为 σ = f(ε, ε̇, T)，用于表征材料在热加工过程中随热力历史变化的力学响应[[S29,S58,S33,S12]]。

热本构方程与常温力学本构方程的核心差异在于：常温本构通常仅考虑应变、应变率对应力的影响，不引入温度作为核心耦合变量；而热本构方程需将温度作为核心输入变量，考虑高温下动态回复、动态再结晶等微观组织演化对应力的影响[[S18]]。与比热、导热系数、热膨胀系数等热物性参数的区别在于：热物性参数是仅与温度相关的材料固有热物理属性，无力学应变/应变速率耦合关系，而热本构方程是热力耦合的力学响应描述[[S37,S19]]。在压铸、选区激光熔化（SLM）等高温制造过程的有限元模拟中，热本构方程是预测材料变形抗力、残余应力分布的核心输入[[S37,S19]]。

## 分类与典型形式

热本构方程的主流分类体系分为两大类[[S58,S51,S45,S62]]：

- **唯象本构模型**：不直接关联微观变形机理，通过宏观实验数据拟合获得映射关系。形式简单、易于嵌入商用有限元软件，广泛用于工程模拟。典型形式包括幂律模型、Johnson-Cook模型、Arrhenius型本构模型。
- **基于物理的本构模型**：引入描述位错密度、动态再结晶分数、晶粒尺寸等组织演化的内变量，耦合微观机制与宏观流变行为。预测精度更高，可适用更宽的热力参数范围。典型形式包括统一内变量本构模型、动态再结晶耦合模型。

### Johnson-Cook (J-C) 本构模型

J-C模型是应用最广泛的唯象本构模型之一，其完整数学表达式为[[S57,S36,S17]]：

σ = (A + B εⁿ) [1 + C ln(ε̇/ε̇₀)] [1 - T\*^p]

其中 T\* = (T - T_r) / (T_m - T_r)。方程由三项乘积构成，分别描述应变硬化效应、应变速率硬化效应和热软化效应。

**主要参数及其物理意义**：

| 参数 | 物理意义 | 单位 |
|------|---------|------|
| A | 参考温度与参考应变速率下的屈服应力 | MPa |
| B | 应变硬化系数 | MPa |
| n | 应变硬化指数 | — |
| C | 应变速率硬化系数 | — |
| p | 热软化指数 | — |
| ε̇₀ | 参考应变速率 | s⁻¹ |
| T_r | 参考温度 | K |
| T_m | 材料熔化温度 | K |

该模型优点是形式简单，少量实验数据即可完成参数标定；不足是未考虑应变、应变速率、温度三者之间的交叉耦合作用，无法准确描述应变软化行为[[S57,S36]]。

### Arrhenius 型本构模型

Arrhenius型本构模型是描述高温流变应力的另一经典唯象模型，由Sellars和Tegart提出，其完整数学表达式分为三部分[[S2,S55]]：

① **应力分区函数**：F(σ) 根据应力水平采用三种表达式[[S2,S57]]：
   - 低应力区（ασ < 0.8）：F(σ) = σ^n₁（幂律形式）
   - 高应力区（ασ > 1.2）：F(σ) = exp(βσ)（指数形式）
   - 全应力区：F(σ) = [sinh(ασ)]ⁿ（双曲正弦统一形式）

② **速率-温度耦合表达式**：ε̇ = A·F(σ)·exp(-Q/(RT))

③ **引入Zener-Hollomon参数后的显式应力**：σ = (1/α) ln{ (Z/A)^(1/n) + [(Z/A)^(2/n) + 1]^(1/2) }[[S2,S56]]

其中A为指数前常数，α = β/n₁ 为应力修正系数，n为应力指数，Q为热变形激活能（单位J/mol），R为通用气体常数（8.314 J/(mol·K)），Z = ε̇·exp(Q/(RT)) 为Zener-Hollomon参数[[S2,S56]]。引入应变补偿的Arrhenius方程可将α、n、Q、lnA全部表示为关于真应变的高阶多项式（通常5～7阶），进一步提升宽应变范围内的预测精度[[S2,S56,S43,S64]]。

## 工艺角色与工程应用

### 压铸充型与凝固模拟中的热本构

在压铸凝固过程的数值模拟中，热本构方程的引入采用分步耦合路径：首先通过充型流场求解得到瞬态温度场结果，将温度场作为输入导入应力求解模块，代入基于热弹塑性理论构建的广义胡克定律热本构方程，通过位移迭代法依次求解应变、应力场的分布[[S6,S20,S30]]。后续可升级为弹塑性-蠕变耦合本构，适配凝固过程中固液两相区的非均匀力学行为，支撑完整热力耦合仿真流程[[S34,S23]]。

基于热本构方程的工程应用标准流程[[S44,S23,S5]]：
1. 输入实测随温度变化的弹性模量、泊松比、热膨胀系数参数
2. 区分铸件凝固过程中的液态区、固液两相区、固态区三个阶段，分别赋予适配的本构模型
3. 采用单向/双向热力耦合算法计算热应力分布，输出残余应力云图与铸件变形量
4. 结合固液两相区的临界拉应变阈值判定热裂危险区域

以ADC12（JIS A383.0）压铸铝合金为例，其专用弹塑性-蠕变热本构方程的特性为：在573K（300℃）及以上高温区间，总应变中几乎全部为蠕变应变，仅存在极少量塑性应变。采用此模型可避免因使用传统室温自然时效试样测得的应力-应变数据导致的残余应力预测值被高估50%～64%的误差问题[[S37,S35,S10]]。

### SLM温度场模拟中的热本构

在选区激光熔化（SLM）温度场-应力场耦合模拟中，热本构方程基于连续介质力学框架，将熔池周边区域的材料分为液态、半固态、固态三个阶段分别赋予适配的热粘塑性本构参数，可准确描述激光扫描瞬间熔池周边高温度梯度区域内材料的高温塑性流动行为，同时追踪逐层扫描、快速冷却过程中的材料收缩约束关系，复现逐层累积的残余应力演化全流程[[S59,S21,S31]]。

常用的实现方案为在通用有限元软件（ABAQUS/ANSYS）中通过自定义用户材料子程序（UMAT/VUMAT），将随温度动态更新的Johnson-Cook热粘塑性模型与热弹塑性分段本构耦合，适配SLM过程中快热快冷（加热/冷却速率可达10³～10⁶ K/s）的极端非平衡变形条件[[S1,S59]]。

## 参数获取方法

### 热压缩实验规范

热本构方程的基础数据通常通过单道次热压缩实验获得[[S50]]：
- 试样制备：将铸锭机加工为直径8～10 mm、高度12 mm的圆柱试样
- 试样两端垫石墨纸以降低摩擦
- 纵向中点焊接热电偶测温
- 以5～10℃/s的升温速率升温至目标温度后保温2～5 min以保证试样温度均匀
- 覆盖预设的全温度区间、全应变速率区间，最大真应变可达0.7～0.916
- 压缩完成后立即水淬保留变形后的微观组织

### Arrhenius型参数标定步骤

针对Arrhenius类型热本构核心参数（Q、α、n、A）的基础求解步骤[[S48,S52]]：

1. 固定应变取值，对热压缩得到的流变应力数据取对数
2. 分别绘制不同温度下的拟合曲线：
   - ln(ε̇) - σ 线性拟合，得斜率β
   - ln(ε̇) - lnσ 线性拟合，得斜率N
   - 计算 α = β/N
   - ln(ε̇) - ln[sinh(ασ)] 线性拟合，得应力指数n
   - ln[sinh(ασ)] - 1/T 线性拟合，取多组斜率平均值得s
   - 热变形激活能 Q = n·R·s

应变补偿型优化方法：在应变0.05～0.8区间内以0.05为步长，逐点求解不同应变下的Q、α、n、lnA，使用5～7阶多项式对各参数关于应变的函数关系进行拟合，完成全应变区间的参数补全[[S43,S64]]。

### 拟合优度评判

拟合优度量化评判采用相关系数R（或决定系数R²）反映预测值与实验值的整体线性吻合程度，同时采用平均相对误差AARE量化全局平均偏差，两类指标结合使用可同时规避单一指标的评判偏差[[S8]]。

## 影响因素与典型数据

### 热变形变量对流变应力的影响规律

热变形三大变量对流变应力的影响规律[[S64]]：
- **应变速率效应**：相同变形温度下，随应变速率升高，连续加工硬化占比提升，动态软化被抑制，流变应力增大
- **温度效应**：相同应变速率下，随变形温度升高，原子热激活作用增强，动态回复、动态再结晶等软化行为占比提升，流变应力降低
- **应变效应**：流变应力随真应变的变化由加工硬化与动态软化的竞争关系决定，曲线存在应力峰值或稳态流变特征

## 局限性、假设与适用范围

### 模型适用边界

常规Arrhenius热本构模型的适用边界[[S64]]：常规变体仅适用于单向热压缩加载模式，当温度高于1200℃（接近固相线）时预测误差大幅升高，需采用分段描述的修正本构。

### 唯象模型的通用简化缺陷

传统Arrhenius、Johnson-Cook等唯象模型大多不耦合实时微观组织演化过程，仅通过宏观应变量等效映射组织变化，无明确物理意义的参数占比高，仅能适配单向加载下的低复杂度变形工况，多轴复杂应力状态下的外推适用性极差[[S45,S32,S28]]。

### 液固两相区的本构不足与改进方向

近固相线及液固两相区常规热本构的主要偏差来源[[S22]]：常规模型默认基体为完全致密的单相连续体，未考虑固液两相共存下液相流动、固相骨架变形、液固界面作用、液相偏析等特殊行为，无法适配不同固相体积分数下的流变响应突变特征。

主流改进方向[[S46,S16]]：采用基于连续介质混合理论的两相耦合本构框架，将总应力分解为固相应力分量和液相应力分量。固相部分采用适配固相体积分数的黏塑性本构，液相部分采用达西定律描述渗流行为，通过引入固相体积分数 f_S 作为核心变量关联不同相的力学响应，适配全区间的半固态流变特征。
