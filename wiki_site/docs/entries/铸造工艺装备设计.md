## 概述

铸造工艺装备设计（foundry tools design）[[S5,S35,S37]]是指对铸件生产所使用的工艺装备进行规划、计算并绘制技术图样的工程活动。它属于铸造领域内独立于铸造工艺设计的一项技术设计环节[[S33]]，核心产出文件包括模样图、芯盒图、砂箱图和模板装配图[[S33]]。铸造工艺装备设计的英文规范名称在国家标准体系中定为 **foundry tools design**[[S5,S37]]。

## 定义

根据国家标准 GB/T 5611-2017《铸造术语》，铸造工艺装备设计的正式定义为：“铸件生产所使用工艺装备的设计，工艺装备包括模样、模板、芯盒、砂箱、量具、夹具、样板等。”[[S35,S37]] 这一权威定义划定了该工程活动的对象边界。

从国内高校铸造专业教材的规范表述来看，铸造工艺装备设计**不属于铸造工艺设计的范畴**，而是以正式的铸造工艺图和会签完成的铸件图为依据，同时需适配对应造型、制芯设备技术要求的一项独立设计工作。[[S33]]

## 工艺作用

在全铸造生产流程中，铸造工艺装备设计位于铸造工艺设计完成之后的落地执行环节。标准流程顺序为：零件图纸审查 → 铸造工艺性分析 → 确定铸造工艺方案 → 绘制铸造工艺图 → 铸件图 → 填写工艺卡 → 铸型装配图 → **开展铸造工艺装备设计**。[[S33]] 它因此是连接工艺策划与实际生产制造的核心中间环节，直接决定后续铸件生产的可行性、成型精度与生产稳定性。[[S33]]

## 设计范围与核心对象

### 设计对象

铸造工艺装备是造型、制芯及合箱过程中所使用的模具和装置的总称，包括模样、模底板、模板框、砂箱、砂箱托板、压铁、套箱、芯盒、烘干板（器）、砂芯修整磨具、组芯及下芯夹具、挤压机下芯框、量具及检具等。[[S13,S36]] 其中，通常所称铸造模具仅指模板（含模样、模底板）和芯盒，是铸造工艺装备的核心部分。[[S13,S36]]

### 按功能分类

铸造工艺装备按功能可分为四大类：[[S59,S23]]

| 类别 | 主要装备 | 功用 |
|------|----------|------|
| 造型用装备 | 模板、模底板、砂箱、托板、压铁、套箱 | 形成铸型型腔，支撑砂型结构 |
| 制芯用装备 | 芯盒、加热装置 | 形成砂芯 |
| 辅助用具 | 浇注系统模样、下芯夹具、组芯夹具 | 辅助造型、制芯与合箱 |
| 工艺用具 | 量具、检验样板、砂芯修整磨具 | 检测与控制工艺质量 |

### 造型工艺装备的设计范围

造型工艺装备的设计范畴覆盖：模样、模底板、浇注系统（含冒口）、排气系统、安全柱（起型导柱）、模板框、加热装置、砂箱及其锁紧装置、托板、压铁、套箱、下芯框（芯罩）等。[[S38]]

## 设计输入与输出

### 核心输入依据

铸造工艺装备设计需以以下信息作为前置输入依据：[[S33,S39]]

- 有效产品图及铸件技术条件
- 正式版铸造工艺图
- 经各部门共同会签完成的铸件图
- 生产批量与试制计划进度表
- 现场在用铸造生产设备参数（造型机与制芯机规格、砂箱尺寸、工装固定方式等）
- 工装制造部门的加工能力与生产经验

### 设计产出文件

常规设计输出交付物包括：[[S33,S18]]

- 模样图（通常仅绘制金属模样图）
- 芯盒图
- 砂箱图（标注材料、结构、紧固定位方式与全部关联尺寸）
- 模板装配图（仅针对采用模板造型工艺的铸件）
- 配套的型板布置图、工装材料规格说明、工装尺寸精度与表面粗糙度技术要求

大批量生产场景下，还需额外设计砂芯成型烘干器（随形烘干板）、下芯及组芯夹具、砂芯与合箱专用检验样板、专用尺寸量具、成型压头、特殊压铁等专用工装。[[S33]]

## 设计程序与通用原则

### 设计程序

依据行业标准 JB/T 9167.2—1998，铸造工装设计的选择应遵循结构化程序：[[S1,S43]]

1. **调研分析**：分析产品结构特点与精度要求、生产计划与工艺条件、工艺工序分类、对工装的基本要求、典型工装结构可行性
2. **确定最佳工装系统**：按优先级依次选用标准工装 → 通用工装 → 组合工装 → 可调工装 → 成组工装 → 专用工装
3. **核算工装总工作量**
4. **确定结构原则**：综合考虑毛坯类型、材料特点、结构特点与精度、定位基准、设备型号、生产批量及生产条件
5. **编制工装设计任务书**（按 JB/T 9167.3 规定执行）[[S10,S57]]

设计的核心依据文件还包括 JB/T 9167.4—1998《工艺装备设计程序》，该标准明确要求设计环节优先采用已有国标与行业标准以提升工装的标准化和通用化率。[[S10]]

### 经济性原则

在保证铸件质量的前提下，应以工装全生命周期（设计、制造、使用、维护）的综合成本为基础进行多方案对比，控制工装复杂程度，核算工装利用率、负荷率等技术经济指标，实现综合效益最优。[[S1,S43]]

工装复杂系数 K 的计算公式为：[[S1,S43]]

$$K = \frac{C}{T_b C_n} + \frac{N_j}{N_{jb}} + \frac{G_b}{G} + \frac{N_c}{N_{cb}} + \frac{L}{L_b}$$

式中，C 为工装设计、制造、维护费用；Nj 为工装专用件件数；G 为工装最高精度等级；Nc 为保证产品尺寸要求的计算尺寸数目；L 为工装最大尺寸；下标带 b 者均为企业相应项目的平均值。

## 与上下游工艺环节的关系

### 与铸造工艺设计的交互

正式的铸造工艺图与会签后的铸件图是工装设计的核心依据。复杂工装设计完成后，需经上游工艺部门会签确认方可投入制造。大批量生产场景下，通常先使用木模、木芯盒完成多轮工艺参数调试并迭代更新正式铸造工艺图后，再启动金属材质工装的正式设计，双方可通过签署专项“技术协议”明确设计边界与责任划分。[[S33,S39]]

### 与压铸工艺调试的交互

压铸工装设计阶段需提前校核适配现有压铸机的锁模力、压室容量、安装尺寸、开合模行程等参数。工装制造完成交付后，须在压铸生产现场进行试生产验证，通过生产指定数量的铸件检验起模顺畅性与金属液充型质量，统计关键尺寸的过程能力指数 CPK。调试过程中反馈的成型缺陷问题可反向迭代优化工装结构设计。[[S32,S18]]

### 与机加工工装设计的交互

经多方会签完成的铸件（毛坯）图是下游机械加工工序设计机加工工装的核心输入依据。工装试生产阶段产出的合格铸件需批量送至机加工环节开展适配性验证，机加工部门出具可正常装夹、加工的验收结论后方算完成铸造工装的最终验收。这一过程确保铸件的预留加工余量、定位夹紧基准完全匹配后续机加工工装的设计要求。[[S67,S18]]

## 材料选型与强度设计

### 材料选型原则

铸造工装材料选型需同时满足三类要求：力学性能要求、工艺性能要求和经济性要求。在对应服役工况下需兼顾强度、耐磨性、耐腐蚀性、耐热性，同时保障加工性能和热处理性能的适配性，以降低全生命周期成本。[[S60]]

### 不同场景下的材料适配

砂型铸造工装的材料选择灵活性较高，不同生产规模匹配不同材料方案：[[S56,S23]]

| 生产类型 | 常用材质 | 特点 |
|----------|----------|------|
| 单件小批量 | 木材（木模） | 含水率控制在8%~12%，同层木板纹理正反交叉、相邻层木纹相互垂直以减少变形 |
| 成批生产 | 环氧树脂玻璃钢（塑料模） | 价格仅为金属模的20%~50%，不吸潮，变形小 |
| 大批量生产 | 铝合金、铸铁、钢材（金属模） | 强度高、耐磨、使用寿命长 |

《简明实用铸造手册》的表9-4对比了木材、铝合金、铜合金、铸铁、石膏、水泥、塑料、菱苦土共8种材料的使用场景与适用生产规模，为选型提供了量化依据。[[S75]]

### 压铸及挤压铸造模具选材

挤压铸造模具零件选材可依据《模具材料应用手册 第二版》表1-26的推荐，针对锌合金、铝合金、铜合金、黑色金属铸件，分别给出凹型、镶块、冲头等零件的推荐热作模具钢牌号（如 5CrNiMo、4Cr5MoSiV1），特殊工况下还可选用难熔金属。[[S14]]

### 超大型压铸模材质要求

面向新能源汽车一体化压铸的超大型压铸模，成型零件需选用抗热裂、抗高温氧化、高耐磨的优质热作模具钢，并优化型腔内部流道布置以均衡应力分布。[[S31]]

## 热平衡设计

### 定义与核心地位

铸造金属模型热平衡定义为：在连续生产过程中，模具周期性吸收铸件释放的液态冷却热与凝固潜热，同时向环境散发热量，当两部分热量数值相等时，模具所处的稳定温度状态即为热平衡状态。该状态是保障铸件质量、稳定生产节拍、延缓模具热疲劳失效的核心前提。[[S46]]

### 热平衡的定量核算

经典金属型热平衡核算公式为：[[S76,S8]]

$$\Sigma Q_{\text{得}} = \Sigma Q_{\text{失}}$$

其中：
- $Q_{\text{得}} = Q_{\text{液}} + Q_{\text{凝}} + Q_{\text{固}}$
- $Q_{\text{液}}$：合金液从浇注温度降至液相线温度所放出的热量
- $Q_{\text{凝}}$：合金在凝固区间内放出的热量（含结晶潜热）
- $Q_{\text{固}}$：铸件冷却时放出的热量

各项可通过比热、温度差和合金质量等参数完成定量计算。[[S76,S8]]

### 影响因素

在恒定铸造节拍下，热平衡状态受以下因素影响：[[S46]]
- **铸件壁厚**：壁厚越大，铸件释放热量越多，模具热平衡温度越高
- **模具壁厚**：壁厚越大，蓄热能力越强，热平衡温度越低
- **模具表面涂层保温性能**：保温性能越强，模具热平衡温度越低
- 可通过强制风冷、水冷、油冷等手段调控热平衡状态

### 热平衡设计方法

压铸模具热平衡设计可根据精度要求采用四类递进方法：[[S21]]

| 设计方法 | 特点 |
|----------|------|
| 经验型设计（Casual） | 依赖模具制造商经验布置水道，优化程度较低 |
| 基于经验的工程设计（Informed Experience） | 设计师对热流有一定理解，水道总体布设合理 |
| 基础热平衡定量核算（Basic Heat Balance） | 计算各区域热量输入量，按计算值布置冷却水道，可控性较高 |
| 有限元分析（FEA）精准优化 | 可精确定位冷却水道并确定控温设定值，推荐与热平衡计算结合应用 |

### 非时间依赖式热平衡调控

可直接集成于模具本体的热平衡调控结构包括：去除型腔附近多余金属基体形成空气保温层，在特定区域填充保温材料实现局部保温，或在热集中区域布置强制冷却结构。这类方法无需生产过程动态干预，稳定性好。[[S30]]

## 模具寿命设计

### 寿命定义

压铸模具寿命定义为在保障合格铸件产出质量的前提下，可完成的总有效生产循环次数，包含模具首次服役寿命与各次维修修复之后的累计服役寿命。[[S49]] 模具热疲劳失效占总失效案例约70%。[[S73]]

### 不同材质/工艺的寿命数据

| 模具类型 | 常规服役寿命 |
|----------|------------|
| 国产H13铝合金压铸模 | 3~5万件（常规）；工况优化后可达40~100万模次 [[S49,S65]] |
| 锌合金热作模具 | 可达约100万模次 [[S49]] |
| 常规钢材铜合金压铸模 | 几千至6.5万次 [[S15]] |
| 难熔金属铜合金压铸模 | 8~10万次 [[S15]] |
| 3Cr2W8V钢黑色金属压铸模 | 不超过350次 [[S15]] |
| TZM钼基合金黑色金属压铸模 | 可达15000次 [[S15]] |

### 寿命预测与优化

针对铝合金、铜合金压铸等高温低周疲劳工况，可采用 Coffin-Manson 公式预测模具热疲劳寿命：[[S73]]

$$N_F = \left[ \frac{C \cdot \varepsilon_f}{\alpha(T_2 - T_1) - \frac{(1-\mu_1)}{E_1}\sigma_1 - \frac{(1-\mu_2)}{E_2}\sigma_2} \right]^{\frac{1}{n}}$$

式中，NF 为失效循环次数，C、n 为经验常数（常规取值 0<C<1，n=0.5），εf 为材料断裂时的实际应变。

延长模具使用寿命的核心设计措施包括：控制模壁厚度保障结构刚度；将尖角全部过渡为圆弧消除应力集中；冷却水道孔距型腔表面距离不小于25mm实现均匀温控；降低单次热循环的温度差以减小热交变应力。[[S20,S15]]

## 常见分类

### 按铸造方法分类

依据行业标准 JB/2180—82，铸造工装可按铸造方法划分为以下大类：[[S29]]

- 砂型铸造工装
- 金属型铸造工装
- 压力铸造工装
- 离心铸造工装
- 熔模铸造工装

同时涵盖清理用工装与专用量具检具等配套类别，各类工装依据统一的编号规则进行管理。[[S29]]

### 金属型工装的分类体系

金属型铸造工艺装备首先分为整体抖落式金属型和装合金属型两大类。装合金属型进一步按分型面特征划分为：单分型面、两个互相垂直分型面和多分型面三个子类，每个子类下再按开合结构细分为移动式、转合式等。[[S54]]

## 压铸模具设计要点

### 设计前置输入依据

压铸模具设计的前置输入依据必须包含7类核心信息：压铸件零件图及技术要求、压铸合金材料的收缩特性、压铸机的模板尺寸/压室尺寸/整机性能参数、模具加工设备的加工能力、压铸现场作业条件以及压铸件生产批量。[[S45,S63]]

### 核心设计原则

1. 选用结构简单、动作可靠的机构设计
2. 具备足够强度与刚度以承受锁模力与胀型力
3. 易损件拆装维护便捷
4. 浇注系统预留试模修改余量
5. 选取适配的模具材料与热处理工艺保障服役寿命
6. 合理选取公差余量保证模具配合精度
7. 尽可能使用标准化压铸模具零件提升互换性、降低制造成本 [[S45]]

### 设计流程

压铸模具设计的完整流程分为6个阶段：压铸件工艺性分析 → 压铸方案规划（分型方案、型腔布置、浇注系统位置、压铸机选型计算）→ 浇注及排溢系统设计 → 压铸模具整体结构与各功能机构设计 → 总装图绘制 → 各压铸模具零件的详细设计（形状、尺寸公差、材料与全部技术要求）。[[S45]]

### 详细设计覆盖范围

压铸模具详细设计涵盖：分型面设计、型腔与成型镶块设计、型芯设计、浇注系统（直浇道、横浇道、内浇口）设计、溢流槽设计、排气槽设计、定/动模座板与套板结构设计、导向定位装置设计、侧抽芯机构设计、推出机构设计、冷却/加热热平衡系统设计、模具材料选型与全流程精度要求设定。[[S71]]

### 浇注系统组成与设计原则

标准压铸浇注系统由直浇道、横浇道、内浇口三部分核心单元组成，冷室压铸机与热室压铸机的浇注系统结构差异集中在分流锥、直浇道两个组成部分。[[S69,S50]]

浇注系统的设计原则包括：充型路径尽可能短以减少热量传输损失与压力损耗；保障充型平稳且排气效果优异；避免浇口位置破坏铸件外观质量；适配型腔数量分布与压铸件成型特性；防止活动型芯发生变形与位移。[[S74]]

多浇口浇注系统设计需先通过计算确定型腔填充时间、浇口速度、浇口截面参数、流道尺寸与溢流槽参数，再完成多浇口整体布局与参数化CAD建模。[[S3,S68]]

### 压铸装备选型校核

压铸装备选型时需完成两项核心参数校核：[[S4]]
- **压射力校核**：P₀ ≥ p × A₀（P₀为压射力，p为压射比压，A₀为压射冲头投影面积）
- **合模力校核**：P合 ≥ p × A合（P合为合模力，A合为铸件在分型面上的投影总面积）

两项参数均须低于所选设备的量程上限。

## 局限性与常见设计失误

### 经验设计的固有局限

传统铸造工装经验设计方法依赖设计者个人经验，难以全面覆盖充型、凝固过程中所有多因素耦合的复杂影响，容易在生产阶段出现气孔、夹杂、缩孔、缩松等难以预判的缺陷，工艺执行稳定性不足。[[S77]]

### 计算机辅助优化的局限

基于计算机CAE仿真与机器学习的铸造工装辅助优化方法当前存在明显局限性：可纳入模拟考量的工艺参数覆盖范围有限，难以将模具长期磨损、环境温度随季节波动等实际工况变量完全纳入模型；训练数据样本量不足容易导致预测结果精度不足、泛化能力差。[[S9]]

### 几何设计对工装成本的刚性约束

铸件早期设计阶段的几何细节选择若不恰当，会直接排除低成本、短周期的工装制备工艺选项，大幅不必要地抬高工装整体制造成本与交付周期。带有额外倒扣、复杂内凹结构的铸件，必须额外设置活动抽芯机构或溃散砂芯等特殊工装配套结构，将大幅提升工装设计难度、制造成本和生产运行复杂度。[[S61,S48]]

### 典型设计失误导致的质量问题

| 设计失误 | 导致缺陷 |
|----------|----------|
| 浇冒口系统设计不合理 | 冷隔、浇不足、卷气、缩松缩孔 [[S27,S11]] |
| 排气槽设计不足 | 气孔 [[S27,S11]] |
| 顶出杆布局不合理、型芯刚度不足 | 铸件顶出变形、鼓面凹陷 [[S27,S11]] |
| 合模定位精度不足 | 错型、飞边过大 [[S27,S11]] |
| 收缩率参数选择不合理或未考虑温升引起的型腔膨胀 | 铸件尺寸超差、批量不合格 [[S70]] |
| 孤立热节未设补缩通道或外冷铁 | 内部集中缩松，影响力学性能与气密性 [[S62]] |
| 斜肋平行排布、金属壁不对称分布 | 铸件热裂、变形 [[S48]] |
