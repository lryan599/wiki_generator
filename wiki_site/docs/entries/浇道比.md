## 概述

浇道比（又称浇口比、流道比，英文 **gating ratio**）是浇注系统设计的核心结构参数，指浇注系统中直浇道A₁、横浇道A₂、内浇道A₃的横截面积之比 [[S13,S29]]。在压铸工艺中，浇道比直接决定了金属液在浇注系统内的流动行为，是控制充型状态和铸件质量的关键要素 [[S13]]。

浇道比的计算要素包括三部分：直浇道为压室/浇口套到横浇道入口的全部截面积总和，横浇道为所有主横浇道、分支横浇道的总截面积之和，内浇道为全部通向型腔的内浇口总截面积之和 [[S3,S27]]。

## 分类

根据截面积比值关系，浇注系统可分为以下三类 [[S29]]：

| 类型 | 截面积关系 | 特征 |
|---|---|---|
| 开放式浇注系统 | A₁ < A₂ < A₃ | 浇道截面积逐级增大 |
| 半封闭式浇注系统 | A₁ < A₂，A₂ > A₃ | 横浇道截面积最大 |
| 封闭式浇注系统 | A₁ > A₂ > A₃ | 浇道截面积逐级减小 |

节流参数 C（内浇口截面积与浇道最小截面积之比）是判定封闭/扩张程度的关键指标：当 C < 0.9 时为节流式系统，C > 1.1 时为扩张式系统，0.9 ≤ C ≤ 1.1 为临界节流区间 [[S30]]。

## 工艺作用

浇道比直接决定金属液在浇注系统内的流速变化趋势。当截面积从直浇道到横浇道再到内浇口逐步收窄时，金属液流速沿程递增，可精准调控进入型腔的内浇口速度；若截面积设置不合理出现扩张段，会形成涡流和涡蚀现象，加剧卷气与模具冲蚀风险 [[S30,S6]]。

合理的浇道比可实现充型过程平稳流动，大幅降低紊流程度与金属液卷气概率，进而减少铸件内部气孔缺陷的生成 [[S21]]。不当的大比例扩张式浇道比设计会使型腔充型过程出现大量未排气的封闭区域，导致憋气、冷隔等缺陷 [[S33]]。

浇道比还直接影响充型总时长：在固定内浇口截面积的前提下，浇道比取值越大，横浇道与直浇道的流通截面积越高，压射阶段的流量阻力越小，整体充型时间可缩短，同时可减少浇道系统内的热损失，避免金属液提前凝固 [[S14,S34]]。

## 典型范围

### 不同铸造场景的浇道比参考值

| 铸造类型 | 推荐浇道比（A₁∶A₂∶A₃） | 来源 |
|---|---|---|
| 铝合金高压冷室压铸 | 1∶(1.1~1.5)∶(0.6~1.0) | [[S22]] |
| 镁合金高压压铸 | 1∶(1.3~1.8)∶(0.4~0.8) | [[S22]] |
| 锌合金热室压铸 | 1∶(1.25~1.6)∶1 | [[S22]] |
| 灰铸铁砂型铸造（常规） | 1.2∶1.5∶1 | [[S2]] |
| 小型灰铸铁件 | 1∶1.5∶(0.5~0.8) | [[S2]] |
| 铜合金金属型铸造 | 1∶(1.5~2.0)∶(0.8~1.0) | [[S22]] |
| 铝合金低压铸造 | 1∶(2.0~2.5)∶(1.2~1.5) | [[S22]] |

### 压铸机类型与面积比经验值

冷室压铸机推荐内浇口截面积∶横浇道截面积 = 1∶3~1∶4，热室压铸机推荐比值为 1∶2~1∶3 [[S20,S34]]。

### 标准推荐

日本 JIS H5302 配套设计规范中，热室锌合金压铸的横浇道与内浇口面积比推荐控制在 1.3~1.6 [[S10]]。ISO 1338 通用铸造工艺导则推荐非铁合金封闭式浇注系统采用 1.2∶1.1∶1 的浇道比以实现高效补缩 [[S38]]。

## 影响因素与选取依据

| 影响因素 | 选取原则 | 来源 |
|---|---|---|
| 合金种类 | 镁合金、铝合金推荐偏大浇道比以降低流动阻力，弥补其冷却速度快、流动性较差的特点；锌合金可取偏小浇道比以减少浇注系统金属损耗 | [[S26,S19]] |
| 铸件壁厚与结构 | 薄壁复杂铸件选偏大浇道比，降低沿程阻力，保证高速充填避免浇不足冷隔；厚壁简单铸件选偏小浇道比，实现低速平稳充型减少卷气 | [[S12,S18,S37]] |
| 压铸机压射能力 | 压铸机压射比压大、流量高时选偏小浇道比即可满足内浇口速度要求；小吨位低压射能力设备需偏大浇道比以降低沿程压力损失 | [[S5,S16]] |

## 相关关系

### 与内浇口速度的匹配

在固定压射压力和冲头速度的条件下，浇道比越大，横浇道流通截面积越高，内浇口处的金属液流速越高。若浇道比取值固定，可通过调整压射速度改变流量，间接调控内浇口速度达到合金对应的推荐范围：铝合金 30~40 m/s，镁合金 15~70 m/s，锌合金 30~50 m/s [[S36,S19,S1]]。

### 与填充时间的反向匹配

在铸件总容积确定的前提下，浇道比越大，压射阶段的总流量越高，充型完成所需的填充时间越短，二者可通过充型时间公式 t = 铸件总容积/流量 建立定量匹配关系 [[S14,S36]]。

### 多型腔分配原则

多型腔压铸模的各分支横浇道总截面积之和应等于主横浇道截面积，距离主横浇道更远的远端内浇口对应的分支浇道截面积需要适当放大，以保证各型腔的流量分配均匀 [[S32,S24]]。

### 与 P-Q² 图的工程验证

浇道比与压射压力的匹配关系可通过 P-Q² 图定量验证：不同浇道比对应的模具特性线（DL 线）与压铸机性能线（ML 线）的交点必须落在推荐的工艺窗口内，保证匹配后的充型参数符合工艺设计要求。浇道比过大或过小都会导致 DL 线偏移出工艺窗口：过小则压力损失过大无法达到指定内浇口速度，过大则流量失控导致内浇口速度过高加剧模具冲蚀 [[S5,S11,S25]]。

## 设计原则与缺陷规避

浇道比设计要求尽可能避免截面积突变的扩张式设计，规避扩张式系统的临界节流区间 0.9 ≤ C ≤ 1.1（C = 内浇口截面积/浇道最小截面积），防止涡蚀现象损坏压铸模具 [[S30]]。

常规压铸浇道比采用收敛式增速形设计原则，要求从直浇道到横浇道再到内浇口的截面积逐步减小 [[S20,S34]]。

![铸造工艺中浇道与横浇道的平面设计及流态分析图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/76e18140-bad2-4837-a145-eb4152ddb2ef/resource)

不合理浇道比引发的典型缺陷包括：喷射和卷气导致内部气孔，扩张式设计引发涡流对冲导致冷隔，阻力过大导致浇不足等；改善方向为调整浇道比使充型过程趋向平稳收敛流动 [[S21,S33,S23]]。
