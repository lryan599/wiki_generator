## 概述

在铸造技术语境下，“陶瓷模”是一个多义术语，主要指两种差异显著的工艺实体：其一为**陶瓷型铸造（Ceramic Mold Casting）**，即以陶瓷浆料灌注成型、经焙烧制成的一次性铸型，用于浇注金属铸件[[S11,S24]]；其二为**陶瓷材料模具（Ceramic Die/Mold）**，即采用氧化铝、氮化硅、碳化硅、氧化锆等工程陶瓷经烧结制成的整体模具或模具镶块，用于拉深、挤压、压铸等成形工序[[S48,S33]]。此外，在陶瓷制品成型领域，“陶瓷模”亦可指代将陶瓷粉料压制成型的金属模具，该含义与铸造场景无直接关联[[S47]]。

本条目聚焦于铸造及相关热加工场景下的陶瓷模具。

## 定义与工艺原理

**陶瓷型铸造**是以陶瓷作为铸型材料的金属铸造工艺，融合了石膏型铸造与熔模铸造的技术特点。其工艺路线为：阳模型 → 阴模型 → 陶瓷铸型 → 金属铸件。主流工艺包括 Shaw 工艺和 Unicast 工艺两大类[[S11,S24]]。

- **Shaw 工艺**：以耐火骨料、水解硅酸乙酯、酒精和胶凝剂配制成浆料，浇灌入带轻微锥度的砂箱中，使用可重复使用的模样成型。浆料迅速硬化至橡胶态后脱模，随即点火喷烧使挥发性物质逸出并形成陶瓷微裂纹，最后在约 980℃（1800℉）下焙烧固化。这些微裂纹有助于排气及缓解凝固收缩时的热胀冷缩[[S24]]。
- **Unicast 工艺**：与 Shaw 工艺类似，但无需点火喷烧，而是通过部分固化后脱模，再在约 1038℃（1900℉）下完全焙烧。对于低熔点金属浇注，可跳过焙烧工序直接使用“生坯”状态铸型[[S24,S7]]。

两种工艺均可制备两种类型的陶瓷铸型：**整体陶瓷铸型**和**复合陶瓷铸型**——后者以廉价的耐火粘土为背衬，表面覆以 6.4~13 mm 厚的陶瓷浆料面层，显著节省陶瓷材料成本[[S7]]。

**陶瓷型精铸模具**则是陶瓷型铸造的重要衍生应用类别，以水解硅酸乙酯为粘结剂，以 Al₂O₃ 刚玉粉、锆英粉或硅铝系材料为耐火骨料，以有机/无机铵盐、氧化镁为固化剂，用于直接铸造锻模、注塑模、压铸模、玻璃模等工装[[S48,S18]]。

## 材料成分与配比

陶瓷型铸造的浆料配比因工艺而异：

| 工艺 | 耐火骨料 | 粘结剂 | 固化剂/催化剂 |
|------|----------|--------|--------------|
| Unicast 工艺 | 细粒锆英砂、煅烧高铝莫来石 | — | — |
| Shaw 工艺 | 粗/细混合硅线石 | 水解硅酸乙酯 | 液态催化剂 |
| 陶瓷型精铸模具 | 刚玉粉（Al₂O₃）、锆英粉、硅铝系材料 | 水解硅酸乙酯 | 有机/无机铵盐、MgO |

高铝基陶瓷型铸造两种主流工艺均能承受极高的金属浇注温度，不会发生金属渗透或粘砂缺陷[[S7]]。对于钛合金等活性金属，陶瓷型壳面层优先采用 Y₂O₃、ZrO₂、CaO 等惰性氧化物陶瓷材料，以降低熔体与铸型的界面化学反应活性[[S45]]。

## 工艺特性与关键参数

### 精度指标

陶瓷型铸造可实现的典型精度参数如下[[S24]]：

| 指标 | 数值 |
|------|------|
| 尺寸公差（首段 25mm） | 0.1 mm |
| 每增加 1mm 增量公差 | 0.003 mm |
| 表面粗糙度 RMS | 2~4 μm |
| 最薄截面厚度 | 1.3 mm |
| 拔模斜度 | 1° |
| 铸件重量范围 | 100 g ~ 数吨 |

陶瓷型精铸模具的尺寸精度可达 5~7 级，表面粗糙度 Ra 为 1.25~6.3 μm。与同规格机加工锻钢模具相比，抗热疲劳强度更高，使用寿命可提升 0.5~3 倍[[S48,S18]]。

### 浇注工艺

陶瓷型常采用热型浇注，浇注时陶瓷型的温度应保持在 200℃ 左右。浇注后铸件宜在型内冷却至室温再开箱，以防裂纹或变形[[S28]]。

### 适配合金范围

陶瓷型浇注可适配的合金范围广泛：涵盖钢铁、高温非铁合金，以及铝合金、铜合金（特别是铍铜）、镁合金、钛合金、锌合金[[S24,S34]]。

## 优点与局限

### 优点

- **耐高温性能优异**：可承受钢、高温镍基合金等高熔点合金的浇注温度，有效阻止金属液渗透和粘砂缺陷[[S7,S34]]。
- **抗粘焊性能优越**：陶瓷材料与熔融铝合金润湿性差、几乎不发生元素互扩散。相比之下，H13 等常规压铸工具钢在铝合金冲刷下，铁元素向铝液中扩散生成 Fe₂Al₅、Fe₃Al 等金属间化合物层，导致粘焊缺陷[[S46,S35]]。
- **表面质量与精度高**：熔模与陶瓷型铸造均可实现精细细节、光滑表面和高尺寸精度[[S34]]。
- **可重复使用模样**：不同于熔模铸造的一次性蜡模，陶瓷型铸造使用可重复使用的模样[[S24]]。

### 局限

- **不可重复使用**：陶瓷铸型为一次性的消耗品，仅适合中小批量铸件生产[[S24]]。
- **脆性高、抗热冲击差**：陶瓷材料热膨胀系数远低于热作模具钢，在循环温度交变工况下易因热应力集中发生脆性断裂失效[[S37,S5]]。
- **仅适用于中小批量**：常规热作工具钢模具韧性好、可承受反复急冷急热工况、使用寿命长，适合大批量压铸生产[[S24]]。

## 典型失效模式

陶瓷铸型的常见失效形式包括[[S37,S12]]：

- 热疲劳裂纹萌生与扩展
- 高温冲蚀剥落
- 高温化学侵蚀
- 焙烧不当导致的整体开裂

## 应用场景与实例

### 模具工装制造

陶瓷型铸造广泛用于制造锻模、注塑模、压铸模、玻璃模、冲裁模、挤压模等工装[[S11,S24]]。陶瓷型铸件和应用举例涵盖塑料模、玻璃模、冲模等多种类型，对应材料覆盖钢、铸铁、铜合金等，应用领域涉及汽车、玩具、灯具制造等[[S2]]。

### 快速模具制造

采用 RP（快速原型）-陶瓷型工艺可大幅缩短试制周期。以子午线轮胎模为例，制造周期仅为传统机加工工艺的 1/3，成本仅为传统工艺的 1/4[[S28]]。

### 高温涡轮壳铸造

以硅溶胶为黏结剂、莫来石为耐火原材料制备陶瓷型壳，经真空感应炉浇注镍基高温合金，配合 ProCAST 仿真优化可消除缩孔、缩松和热裂缺陷[[S3]]。

### 黑色金属压铸陶瓷型芯

在黑色金属压铸中，采用氧化铝等陶瓷材料制成的一次性可耗型芯，不需拔模斜度，浇注完成后通过氢氧化钠或弱酸溶液溶解除去[[S36]]。

### 陶瓷镶块

嵌入钢制压铸模具型腔高负荷受热区域的独立陶瓷构件，可提升模具局部耐冲刷、抗热疲劳性能，实现易损部位的替换[[S25]]。

## 关联概念对比

| 对比对象 | 差异 |
|----------|------|
| **半永久金属模** | 陶瓷模耐热性更高但脆性大、不可重复使用 |
| **石膏模** | 陶瓷模耐高温性能大幅提升，可浇注钢、高温合金等高熔点合金；石膏模仅适合低熔点有色合金 |
| **砂型铸造** | 陶瓷模尺寸精度和表面质量更优异，铸件表面粗糙度更低 |
| **快速陶瓷模** | 结合 3D 打印/快速原型技术，无模化制造，适合复杂结构铸件的快速试制 |

## 工程陶瓷材料在模具中的应用

除陶瓷型铸造外，采用氮化硅、碳化硅、氧化锆等工程陶瓷直接烧结制成整体模具或模具组件是另一重要方向。以下为典型陶瓷材料的关键性能参数。

### 氮化硅陶瓷（Si₃N₄）

烧结工艺分为反应烧结和热压烧结两类[[S41,S6]]：

| 工艺 | 工艺要点 | 成品特征 |
|------|----------|----------|
| 反应烧结 | 硅粉 + Si₃N₄ 粉混合成型，1200℃预氮化后机加工，1400℃保温 20~25h 完成全氮化 | 尺寸变化仅 0.1%~0.3%，精度极高；壁厚不超过 20~30mm |
| 热压烧结 | 添加 MgO、Al₂O₃ 等烧结助剂，石墨模具中 1600~1700℃、20~30MPa 高压烧结 | 几乎无残留气孔，致密度接近理论值 |

关键实测性能参数[[S44,S41]]：

| 参数 | 反应烧结 | 热压烧结 |
|------|----------|----------|
| 室温抗弯强度 | 200 MPa | 700~1200 MPa（加添加剂可达 1500 MPa） |
| 高温强度保持 | 1200~1350℃ 无衰减 | — |
| 热膨胀系数 | 2.53×10⁻⁶/℃ | — |
| 室温电阻率 | 1.1×10¹⁴ Ω·cm | — |
| 抗氧化温度上限 | 1000℃ | — |
| 耐腐蚀性 | 除熔融 NaOH 和 HF 外，耐所有无机酸及部分碱 | — |

在压铸冶金领域，氮化硅的主流应用部件包括热电偶保护套管、铝合金金属液输送支座、高温密封栓塞等，与熔融有色金属的浸润性极差，几乎不发生界面反应[[S38,S27]]。

在拉深成形工况下，氮化硅陶瓷模具对比常规 Cr12 合金模具、铜基合金模具，工件破损率极低，修模后累计拉深寿命超过 2000 件，工件表面无明显划伤[[S48,S16,S31]]。

### 氧化锆增韧陶瓷（PSZ）

氧化镁部分稳定的氧化锆增韧陶瓷关键性能参数[[S23,S8]]：

| 参数 | 数值 |
|------|------|
| 密度 | 5.9~6.05 g/cm³ |
| 硬度 | ≥88 HRA |
| 抗弯强度 | 900~1300 MPa |
| 断裂韧度 | 7~14 MPa·m^(1/2) |
| 线膨胀系数 | 10×10⁻⁶/℃ |
| 熔点 | 2715℃ |

氧化锆增韧陶瓷适配热挤压模具，耐高温、耐腐蚀、耐磨损性能优异，使用寿命可比普通硬质合金模具高出数十倍[[S39,S33]]。在拉深厚度 0.4mm 的 SUS304L 不锈钢工件时，可稳定完成 150 万次以上的拉深作业且表面粗糙度不恶化[[S33]]。国产氧化锆陶瓷制备的电池电芯成形粉模累计使用寿命可达 660 万次，性价比为普通合金模具的 4 倍[[S23,S26]]。

### 碳化硅陶瓷（SiC）

碳化硅陶瓷关键性能参数[[S21,S22,S30,S44]]：

| 参数 | 数值 |
|------|------|
| 密度 | 3.2×10³ kg/m³ |
| 室温弯曲强度 | 200~250 MPa |
| 抗压强度 | 1000~1500 MPa |
| 莫氏硬度 | 9.2~9.5（第三高硬度结构陶瓷） |
| 1400℃ 抗弯强度 | 500~600 MPa |
| 氧化温度区间 | 900~1300℃ 缓慢氧化 |

碳化硅陶瓷以共价键结合，烧结工艺分为反应烧结、热压烧结和常压烧结三类。反应烧结以 α-SiC 和碳为原料，在 1600~1700℃ 硅蒸气氛围中使硅与碳反应生成 β-SiC；热压烧结需添加 B₄C、Al₂O₃ 等烧结助剂；常压烧结需添加 0.36% 硼和 0.25% 以上的碳，烧结温度高达 2100℃[[S42]]。
