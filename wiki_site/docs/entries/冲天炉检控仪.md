## 概述

冲天炉检控仪（tester and controller for cupola melting）是用于检测和控制冲天炉熔炼过程的仪器的总称。在GB/T 5611-2017《铸造术语》中，其定义为包含风量和风压测定仪、料位仪、测温仪、热分析仪、熔炼过程综合测控仪、铸铁成分和性能速测仪等在内的设备类别`[[S23,S10,S2]]`。GB/T 25370-2020《铸造机械 术语》进一步将其规范为“冲天炉熔炼过程监测控制装置”，列入铸造领域专用监测类设备，典型示例包括冲天炉风量风压检测仪、冲天炉微机控制装置`[[S30]]`。

在冲天炉熔炼系统及铸造车间中，冲天炉检控仪是核心配套检测控制装备，负责对熔炼全流程的关键工艺参数进行实时采集、反馈调节，是保障铁液质量稳定、降低冶炼能耗、避免炉况故障的关键装备`[[S17,S18,S29,S28]]`。

## 分类与类型

冲天炉检控仪按监测对象可分为以下典型类型`[[S22,S15,S32,S26,S1]]`：

| 类型 | 监测对象与功能 | 典型检测器/仪表 |
|---|---|---|
| 温度检测仪 | 铁液温度、炉气温度、热风温度 | 热电偶测温装置、非接触式辐射/红外高温计 |
| 炉气成分分析仪 | 炉气中CO₂、CO、O₂含量 | 红外线气体分析仪、气相色谱仪 |
| 压力/风速检测仪 | 送风压力、送风量 | 毕托管、孔板差压变送器、U形/弹簧式压力计 |
| 料位检测仪 | 炉料顶面高度 | 炉气压差式料位计、光电式料位计、激光料位计 |
| 集成式综合检控台 | 多参数同步监测与闭环控制 | CDZ型多功能综合检测仪、C399型微机优化控制仪 |

## 工作原理

### 料位检测

**炉气压差式**：在加料口下沿约一批炉料位置的炉膛壁上安装测压管，一端连接差压计，另一端连通大气。满料时测压管被埋没，差压计产生压差，回路断开，加料机停机；炉料下降至测压管以下时，差压计两端压力相等，电极接通回路，发出电信号控制加料机完成加料。该类装置结构简单、灵敏度高，无需外加动力，电子线路采用三级放大、二极管/稳压管耦合设计，抗干扰能力强`[[S18,S26]]`。

**光电式**：在加料口下沿450~500mm处两侧炉壁开设对开光控孔，一侧安装光源（22V/250W全反射灯泡，经两次凸透镜聚焦后光束光斑面积仅3mm²），另一侧安装硅光电池。炉料未下降时光线被遮挡，加料机不工作；炉料下降至光控孔以下时，硅光电池接收光信号并转为电信号，触发加料机自动加料`[[S35,S18]]`。

### 风量风压检测

使用毕托管或标准孔板采集风管气流的差压信号，经差压变送器转换为直流电信号，送入微机按风量公式计算实时风量值。风量计算公式为：

$$W = 58.6kD^{2}P^{1/2}$$

其中，W为风量（m³/min），D为风管内径（m），k为流量系数（通常取0.83~0.84），P为测得压力（Pa）`[[S15,S29]]`。风压则通过压力变送器直接测量风箱内送风压力，用于判断风口结渣、炉料搭棚等炉况异常`[[S29]]`。

### 炉气成分分析

利用红外线气体分析仪，通过不同气体组分对特定波长红外辐射的选择性吸收特性，检测料位线以下400~500mm处采集的炉气中CO₂、CO、O₂含量`[[S33,S1]]`。

### 温度测量

温度测量分为接触式（各类热电偶配显示仪表）与非接触式（光学高温计、辐射高温计、比色高温计、红外高温计），以接触式热电偶高温计应用最为广泛，具有测量准确、可靠、简便的优点`[[S15]]`。

## 关键技术参数

### 集成式检控仪性能规格

CDZ型冲天炉多功能综合检测仪可自动连续检测、显示、记录风量、风压、熔化速度、铁液温度（点测与连测），同时自动显示并控制料位高度，所有检测参数可集中在同一张记录纸上输出连续彩色变化曲线`[[S17]]`。其主要技术规格如下`[[S20]]`：

| 参数 | 规格 |
|---|---|
| 记录仪精度 | ±1℃ |
| 测量反应速度 | <1s |
| 供电电源 | AC 220V |
| 使用环境温度 | 0~45℃ |
| 消耗功率 | 约55W |
| 外形尺寸 | 750×400×450mm |

C399型冲天炉熔炼微机优化控制仪可通过微机系统自动检测、数显并打印记录风量、风压、铁液温度、炉气CO₂含量、熔化速度和料位等参数，支持风量定值控制和基于过桥铁液温度与炉气CO₂含量的自动优化控制，通过PID运算输出调节信号，控制电动执行器调整阀门开度`[[S17]]`。其主要技术规格如下`[[S13]]`：

| 检测参数 | 量程 | 精度 |
|---|---|---|
| 风量 | 0~300 m³/min | — |
| 风压 | 0~40 kPa | — |
| 铁液温度 | — | ≤±10℃ |
| 炉气CO₂含量 | — | — |
| 料位 | — | — |
| 熔化速度 | 0~30 t/h | — |

### 熔炼工艺关键被控参数

| 参数 | 控制范围/指标 | 意义 |
|---|---|---|
| 送风强度 | 100~150 m³/(m²·h)`[[S29]]` | 送风过量导致热量浪费，不足则燃烧不完全 |
| 送风温度（内置炉胆预热） | 150~200℃`[[S23,S10]]` | 提高炉温、还原性气氛，利于铁液过热 |
| 送风温度（烟囱内预热） | 300~400℃`[[S23,S10]]` | 进一步降低焦耗、加快熔化速度 |
| 送风温度（外热式预热） | 最高约500℃`[[S23,S10]]` | 高温送风强化熔炼 |
| 铁水出炉温度（铸造焦） | 不低于1500℃`[[S7]]` | 保障铁液充分过热，避免冷隔、浇不足缺陷 |
| 铁水出炉温度（冶金焦） | 不低于1450℃`[[S7]]` | 同上 |
| 炉气CO₂（正常工况） | 13%~16%`[[S33,S1]]` | 反映焦炭燃烧完全程度 |
| 炉气CO（正常工况） | 5%~8%`[[S33,S1]]` | 判断炉内还原性气氛 |
| 炉气O₂（正常工况） | 约1%`[[S33,S1]]` | 衡量供风合理性 |
| 燃烧比（理想工作区） | 55%~70%`[[S33]]` | 炉气CO₂含量对应16.5%~11%为理想状态 |
| 底焦高度 | 需通过层铁焦比保持稳定`[[S27]]` | 底焦高度异常将导致铁液温度波动大、焦耗异常升高、熔化速率不稳定 |

## 标准与规范

- **GB/T 5611-2017《铸造术语》**：对冲天炉检控仪作出正式定义，明确其涵盖的具体仪器类别`[[S23,S10,S2]]`。
- **GB/T 25370-2020《铸造机械 术语》**：将“冲天炉熔炼过程监测控制装置”列入铸造专用监测类设备`[[S30]]`。
- **GB 4208《外壳防护等级(IP代码)》**（等效采用IEC 529）：规定工业现场高粉尘高湿环境使用的铸造检测设备最低防护等级不得低于IP2X，露天使用设备防水特征值不得低于3，适配冲天炉周边多尘、高温蒸汽的工况`[[S24]]`。
- **ZBJ 61012-88《冲天炉技术条件》**：规定冲天炉设备出厂前须通过不少于4小时的连续试炉测试，实测铁水温度、熔化速率、能耗参数需达设计额定值的95%以上`[[S14,S7]]`。
- **ASTM A197/A197M**：针对冲天炉熔炼生产的可锻铸铁铸件，规定最低抗拉强度、屈服强度、断后伸长率的强制技术指标`[[S9,S8]]`。

## 与压铸工艺的关联

常规灰铸铁高压压铸在当前工业场景中几乎未实现大规模量产应用，铸铁熔点高、半熔融特性差，难以适配常规压铸设备工况`[[S19]]`。普通冲天炉直接熔炼的铁水不被主流铸铁压铸生产线采用。

当前主流铸铁类压铸生产线几乎全部采用感应熔炼炉作为主熔炼设备。仅在少数特殊铸铁压铸试验生产线中采用“冲天炉+感应炉”双联工艺——冲天炉先完成粗炼出低成本铁水，后续送入感应炉精准调整成分与温度，再供给压铸机使用，此类场景占压铸行业总生产线比例极低`[[S21,S12]]`。

两种熔炼工艺对比`[[S21,S12]]`：

| 对比维度 | 冲天炉直接熔炼 | 感应炉直接熔炼 |
|---|---|---|
| 熔炼成本 | 极低 | 单位能耗远高于冲天炉 |
| 连续大批量生产能力 | 强 | 相对较弱 |
| 铁液成分稳定性 | 差 | 高 |
| 铁液温度控制精度 | 差 | 高 |
| 适配压铸质量要求 | 难以满足 | 可满足 |

用于压铸场景的铸铁铁水需满足的基础要求：含碳量2.4%~4.0%，出炉温度不低于1300℃，含硫量控制在0.03%~0.07%区间，硅、锰烧损率波动幅度不超过10%，以避免铁液氧化夹渣、气孔缺陷率过高`[[S6]]`。

在双联工艺场景下，冲天炉检控仪承担的职责是保障粗炼阶段铁水的温度与成分波动处于可控范围，为后续感应炉精准调控提供一致的基础铁液，降低双联体系的总体调控压力与能耗。

## 优势与局限

**传统经验操作**依赖操作人员通过观察风口、渣口、加料口炉气、炉渣、铁液等现象判断炉况，完全凭经验调整工艺参数`[[S15]]`，参数控制精度差，铁液质量一致性难以保证。

**检控仪自动控制**可实时采集多参数并实现闭环自动调节。CDZ型仪器能将所有检测参数集中于同一记录纸输出连续彩色曲线，便于数据分析积累与工艺优化`[[S17]]`；C399型可基于铁液温度目标参数与炉气CO₂限制参数自动优化控制风量，使冲天炉稳定运行在最优状态`[[S17]]`。

冲天炉检控仪的局限在于：当前检测与控制主要针对冲天炉典型工艺参数（风量、风压、温度、料位、炉气成分），但铁液最终化学成分的在线精确检测仍依赖于炉前热分析、快速光谱分析等辅助检测手段，存在检测响应滞后的问题。
