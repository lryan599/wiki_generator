## 概述

热固性树脂粘结剂（thermosetting resin binder）是铸造生产中的一种重要有机粘结材料，属于树脂粘结剂下按热行为分类的一个核心类别。其核心特性在于加热时分子间发生不可逆的交联反应，形成三维体型网络结构，从而将分散的砂粒牢固粘结为整体，冷却后无法恢复原有状态和性能[[S5,S29,S10]]。在通用砂型铸造领域，它广泛用于制芯、造型及涂料组分，是保证砂芯/砂型强度、尺寸精度和表面质量的关键功能材料。

**适用性说明**：当前所有检索到的热固性树脂粘结剂相关技术内容均属于通用铸造领域（砂型铸造）的成熟应用，未检索到针对压铸场景的专属改性规则、工艺适配规定或针对性应用描述[[S5,S17]]。因此，本条目所述内容均基于通用铸造范畴的应用证据，若需应用于压铸特定场合，应结合具体工艺条件进行独立验证。

## 分类与对应工艺体系

根据GB/T 5611-2017，铸造用树脂粘结剂按受热时状态变化特性分为热固性树脂粘结剂和热塑性树脂粘结剂两大类。热固性树脂粘结剂依据其固化机制和适用工艺，可进一步细分为热硬树脂系、自硬树脂系和气硬树脂系三大体系，各类对应不同的商品化树脂品种和制芯/造型工艺[[S5,S31]]。

**热固性树脂粘结剂的核心原料类型与工艺体系对应关系**如下表所示：

| 核心原料类型 | 主要商品化工艺体系 | 典型固化模式 |
| :--- | :--- | :--- |
| 酚醛树脂（PF） | 壳法（壳型/壳芯）树脂、热芯盒树脂、自硬树脂、冷芯盒树脂 | 热固化、酸/酯固化、气固化 |
| 呋喃树脂（FA） | 自硬树脂（酸固化）、热芯盒树脂、冷芯盒树脂（SO₂法） | 催化剂固化、热固化、气固化 |

## 固化机理

热固性树脂粘结剂的固化本质是低分子预聚体在特定条件下发生化学交联反应，从初始的线型或支链型结构转化为不溶不熔的三维体型网络结构，从而将砂粒粘结成为具有设定强度的整体。根据触发固化的能量或化学物质形式，固化机理主要分为以下三类[[S5,S29,S9,S23,S24,S31,S43]]。

### 1. 热固化（热硬树脂系）

加热是触发交联反应的直接能量来源。预聚体分子上带有的活性官能团（如羟甲基、酚羟基、呋喃环上的双键等）在高于室温的条件下（通常为200–260 °C）发生缩聚或加聚反应。例如，在经典的壳法（Croning工艺）中，热塑性酚醛树脂（Novolak）与固化剂乌洛托品（六次甲基四胺）的混合物被加热时，乌洛托品分解释放甲醛，提供亚甲基桥（-CH₂-），将酚醛树脂的线型分子交联为体型网络，实现固化并产生强度[[S1,S23]]。热固化过程的原理可通过下图直观理解。

### 2. 催化剂固化（自硬树脂系）

在室温条件下，通过预先混入砂中的固化剂/催化剂触发交联反应，无需外部加热即可使砂型（芯）硬化。酸固化呋喃树脂、酯固化碱性酚醛树脂和酚醛尿烷树脂均属此列。其反应从催化剂与树脂接触开始，体系粘度逐渐上升，同时强度连续增长，直至达到工艺要求的起模强度[[S5,S29,S24]]。20 °C常规工况下，自硬树脂砂的可使用时间（釜寿命）可通过调整固化剂类型与添加量进行调控，砂温通常控制在20–30 °C区间[[S38]]。

### 3. 气体固化（气硬树脂系）

室温下，将预先与树脂混合好的型砂射入芯盒紧实后，通入气态固化剂或汽化的液态催化剂，气体在砂粒孔隙间快速扩散，并与树脂活性基团发生交联反应，短时间内即能使砂芯强度接近终值。此工艺不需加热，起模时型、芯尺寸精度高，生产效率高。典型体系包括酚醛尿烷-三乙胺体系、呋喃-二氧化硫体系、酚醛-甲酸甲酯/CO₂体系等[[S5,S29,S31,S43]]。

## 关键性能参数与控制目标

热固性树脂粘结剂及其配制型砂的性能直接影响铸件质量，以下为关键控制参数与典型目标值。

| 性能参数 | 典型控制目标 / 范围 | 说明 |
| :--- | :--- | :--- |
| **终强度（24 h抗拉/抗压）** | 型砂：0.6–0.8 MPa<br>普通芯砂：0.8–1.0 MPa<br>复杂砂芯：1.6–2.0 MPa<br>非铁合金用砂可适当降低 [[S3]] | 保证搬运、合型、浇注过程中的完整性的关键指标，过高会增加成本与缺陷倾向。 |
| **发气量** | 一般要求 ≤ 10–11 mL/g [[S3,S35]] | 发气量随粘结剂加入量增加而增加，含量过高时发气量易超过12 mL/g，显著增加铸件气孔缺陷风险。 |
| **可使用时间（釜寿命）** | 通过固化剂类型与添加量调控，砂温控制在20–30 °C [[S38]] | 指混合料保持可造型性能的时间窗口。超过可用时间会导致流动性下降、紧实度不足，引发强度降低和粘砂等缺陷。 |
| **表面安定性** | ≥ 85%–90% [[S3,S35]] | 表征型（芯）表面抵抗搬运磨损及金属液冲刷的能力，不足时易导致冲砂、砂眼及机械粘砂。 |
| **透气性** | 约 400 [[S3,S35]] | 树脂砂透气性较粘土砂高，但因其发气量大，须通过集中排气措施配合，防止气孔缺陷。 |
| **高温强度与溃散性** | 500°C残留抗压强度接近0为佳 [[S38]]<br>糠醇含量越高，高温强度越高 [[S35,S3]] | 影响铸件收缩受阻产生的热裂倾向及落砂难度。酯固化碱性酚醛树脂砂在500°C时残留强度几乎为零，溃散性优异。 |
| **热膨胀率** | 相对较低为佳 [[S38]] | 热膨胀率低有助于减少毛刺、脉纹等缺陷倾向。 |
| **游离甲醛/苯酚含量** | 游离甲醛 < 5%，游离苯酚应尽可能低 [[S11]] | 影响工人健康与环保合规，是评判树脂品质和安全性的重要指标。 |

## 主要缺陷与关联关系

使用热固性树脂粘结剂的型砂在铸造生产中可能引发多种特定缺陷，其形成机理与控制方案如下。

### 1. 机械粘砂

- **形成机理**：树脂在500 °C左右热解，树脂膜烧蚀，砂粒间形成间隙，使液态金属易于渗入。同时，树脂砂流动性差、超过可使用时间导致紧实度不足、表面安定性差，也会诱发此缺陷[[S35,S34]]。
- **控制方案**：选用耐热性更高（如高糠醇含量）的树脂；保证砂粒充填紧实度和表面安定性；添加0.5%–2.0%氧化铁粉以提高热强度；厚大铸钢件可搭配使用铬铁矿砂和耐火涂层，并适当降低浇注温度；选用粒度分布覆盖4–5个筛号的细原砂[[S35,S4,S34]]。

### 2. 脉纹

- **形成机理**：浇注后，砂型表层树脂烧蚀严重，砂粒间隙变大，同时石英砂发生相变膨胀，表层强度不足而产生裂纹，液态金属渗入其中形成毛刺状凸起。此缺陷是呋喃树脂砂的特有表面缺陷，多见于型芯拐角、热节位置，严重时呈网状分布[[S35,S34]]。
- **控制方案**：选用低热膨胀原砂（如锆砂、铬铁矿砂）；提升回用旧砂占比以降低砂体膨胀率；添加1%–2%氧化铁粉增加体系塑性；适当降低浇注温度并提升涂层质量[[S35,S34]]。

### 3. 气渣坑 / 气渣隔

- **形成机理**：这是树脂自硬砂特有的铸造缺陷。树脂发气量大，在铸件大平面朝上浇注时，易形成被涂料层封闭的“气袋”，气袋压力足以将金属液推开形成凹坑。随后气体泄出，低温熔渣流入凹坑形成黄色渣层。补浇的金属液被渣层分隔，即形成气渣隔缺陷[[S8]]。
- **控制方案**：降低树脂加入量（再生砂中≤1.5%）；控制旧砂灼减量≤3.5%；严格控制粉尘含量，厚大截面件选用较粗原砂；加强人工通气（扎出φ3–φ5 mm通气孔，孔距60–80 mm）；在分型面设置出气溢流冒口，其截面积应大于内浇口截面积[[S8]]。

### 4. 裂纹（热裂/冷裂）

- **形成机理**：铸件壁厚差大，冷却过程收缩受阻、冷却速率过快；或树脂砂残留强度偏高、退让性不足，共同诱发热裂或冷裂[[S2,S34]]。
- **控制方案**：优化铸件结构，力求壁厚均匀；合理设置工艺肋；协调布置浇冒口以保证冷却速度均匀；提升树脂砂溃散性；控制浇注温度并精炼金属液减少夹杂[[S2,S34]]。

## 与无机粘结剂的对比

热固性树脂粘结剂与以水玻璃为代表的无机粘结剂相比，各有显著优缺点[[S6,S25]]。

| 对比维度 | 热固性树脂粘结剂 | 无机粘结剂 (如水玻璃) |
| :--- | :--- | :--- |
| **型/芯强度** | 高，强度发展好 | 相对较低，取决于改性技术 |
| **尺寸精度** | 高 | 较低 |
| **溃散性** | 优，落砂清理容易 | 差，常需复杂的后处理设备 |
| **生产效率** | 高，工艺选择性多（热、冷、自硬） | 中等，部分工艺（如酯硬化）适用于单件小批量 |
| **环保性** | **差**，制芯、浇注过程排放含酚、含氮有毒气体，环保合规成本高 | **优**，无毒无害，无有毒气体排放，绿色铸造 |
| **能源消耗** | 热法则需配套加热装置与能耗 | 有机酯、CO₂等常温硬化法可降低能耗 |

## 标准与规范

热固性树脂粘结剂的生产、检验与应用有明确的国内外标准体系支撑。

### 国内行业标准

- **JB/T 3828-2013《铸造用热芯盒树脂》**：规定了热芯盒树脂的含氮量分级、牌号、技术要求（外观为棕黄色至深棕色半透明黏性液体）及测试方法，由一汽、圣泉、东风汽车等单位联合编制[[S33,S41]]。
- **JB/T 7526-2008《铸造用自硬呋喃树脂》**：规定了自硬呋喃树脂的分类标记、技术要求、试验方法等[[S33,S37]]。
- **JB/T 8834-2013《铸造覆膜砂用酚醛树脂》**：规定了壳法用酚醛树脂的软化点、聚合速度等核心指标[[S1,S37]]。
- **JB/T 11738-2013《铸造用三乙胺冷芯盒法树脂》** 和 **JB/T 11739-2013《铸造用自硬碱性酚醛树脂》**：分别针对气硬冷芯盒和自硬碱性酚醛树脂的规范[[S37]]。
- **GB/T 41972-2022《铸铁件铸造缺陷分类及命名》**：修改采用ISO/TR 16078:2013，对气孔、粘砂、裂纹等缺陷的术语和定义给出了四级分类与编码规则，为缺陷判定提供依据[[S40,S5]]。

### 国际规范与安全限值

- **美国铸造协会（AFS）**：将铸造用有机粘结剂按固化机制分为热固型（thermosetting）、自硬型（self-setting）和气硬型（vapor-cured）三类[[S17]]。
- **接触限值**：美国OSHA标准29CFR1910.1048对作业场景的甲醛暴露有明确限值[[S28]]。作业时需配备通风除尘设施，防护人员免受酚类、甲醛、呋喃挥发物及热解产物（如一氧化碳）的危害[[S28]]。

## 应用案例（通用砂型铸造）

### 1. 汽车发动机缸体/缸盖水套砂芯 - 热盒/温盒工艺

采用液态热固性树脂作为粘结剂，与砂料混合后吹入加热的钢制芯盒中，固化10–30秒后取出。该工艺生产的砂芯整体强度高，浇注时挥发分低，可显著降低铸件气孔缺陷发生率，是汽车铸件制造的主流成熟工艺[[S21]]。

### 2. 高精度传动及液压铸件 - Croning壳型/壳芯工艺

采用热固性酚醛树脂覆膜砂，树脂加入量占硅砂质量的2%–4%。覆膜砂被送入预热至200–260 °C的金属成型模中，仅在接触热表面的区域形成预定厚度的固化壳层。该工艺生产的铸件尺寸精度高、表面光洁度好，可有效避免粘砂缺陷，已广泛应用于液压件、汽车精密结构件[[S17,S21]]。
