## 概述

拉制模（又称拉拔模、拉伸模、拉丝模，英文对应 drawing die）是金属塑性成形模具中的一大独立类别，专门用于金属线材、棒材、管材等在冷状态下受拉力通过模孔、实现减径延伸成形的工艺装备，属于通用金属塑性加工模具范畴[[S16,S4]]。国家标准 GB/T 8845-2017《模具术语》整合了多项旧模具标准，新增拉制模专属完整术语章节，对其类型、零部件、设计要素及模孔分区等进行了系统规范[[S16]]。

## 定义与工艺作用

拉制模的官方定义为：使金属坯料在拉力作用下通过模孔延伸变形为棒材、线材、丝材、型材等制品的模具[[S26]]。其核心工艺原理为：金属坯料在拉力作用下通过截面逐渐缩小的模孔，产生塑性变形，实现截面积减小、长度增加并获得与模孔截面相应的形状与尺寸精度[[S11]]。拉拔属于冷变形加工，所获产品具有尺寸精度高、表面质量好，并因加工硬化而提高强度的特点[[S2]]。

## 术语边界澄清

在压铸语境下，现有公开标准及专业文献中不存在“拉制模”的独立特殊术语定义。个别压铸文献中出现的“以反拉制件为目的的侧抽芯机构”表述中，“反拉”仅为动作指代，不属于独立模具类型，与通用金属拉拔成形的拉制模术语边界清晰，不存在歧义[[S17,S25,S19]]。因此，本词条所述“拉制模”专指金属冷拉拔成形用模具，而非压铸模的抽芯部件。

## 分类

### 按模芯材料分类

按照模芯材质的不同，拉制模可分为以下子类[[S1,S24,S15]]：

| 类别 | 模芯材料 | 适用范围与特征 |
|------|----------|----------------|
| 钢质拉制模 | 钢质材料（如T8A、T10A） | 适用直径 φ16~100 mm 棒材、φ16~300 mm 管材及异型断面制品 |
| 硬质合金拉制模 | 硬质合金（如YG6、YG8） | 寿命比钢模高上百倍，广泛用于常规线材、棒材、管材拉拔 |
| 天然金刚石拉制模 | 天然金刚石 | 高精度细丝拉拔 |
| 人造金刚石拉制模 | 聚晶金刚石（PCD）/单晶金刚石（MCD） | 高精度细丝拉拔，工业应用广泛 |
| 金刚石涂层拉制模 | 钨钢模芯表面涂敷金刚石层 | 兼具钢模韧性与金刚石表面耐磨性 |
| 陶瓷拉制模 | 碳化钛基、三氧化二铝基、四方氧化锆等陶瓷 | 高硬度高耐磨性，适合不锈钢等难拉拔材料的冷拉成形 |

### 按用途分类

按拉制成形制品的类型划分[[S1]]：

- **线材拉制模**：用于成形金属线材
- **棒材拉制模**：用于成形金属棒材
- **管材拉制模**：用于成形金属管材
- **圆形拉制模**：用于成形截面形状为圆形的拉制件
- **异形拉制模**：用于成形截面形状为非圆形的异型金属制件

### 按结构分类

按结构形式可分为两类[[S1,S26]]：

- **整体式拉制模**：由整体材料制成，模孔各区段参数清晰标注润滑角、压缩角、各区域长度等设计要素
- **组合式拉制模**：由模芯和模套组成，部分结构增设模盖及粉体填充层；结构刚性更高，可承受更大拉制载荷，硬质合金与金刚石模芯均需通过过盈配合镶入钢制模套以提供预压应力[[S23,S10]]

## 模孔结构

拉制模模孔标准分区由 6 个功能区组成[[S26,S20]]：

| 区域名称 | 功能说明 |
|----------|----------|
| 入口区 | 导入拉制材料和润滑剂，通常采用 60° 锥角 |
| 润滑区 | 将润滑剂导入压缩区的过渡区域，一般采用 40° 锥角，长度不少于工作区长度 |
| 压缩区 | 实现坯料截面减缩，是发生塑性变形的主要区域 |
| 定径区 | 保证制品最终截面尺寸精度，最适宜形状为圆柱形 |
| 减压区 | 降低模孔出口处工作压力 |
| 出口区 | 导出拉制完成的制品，形状为倒锥形 |

各区交界应圆滑过渡，不允许有明显的过渡棱边[[S20]]。

## 核心工艺参数

GB/T 8845-2017 定义了拉制过程相关的核心设计要素[[S26]]：

| 参数名称 | 定义 |
|----------|------|
| 减面率 | 材料拉制后截面积与拉制前截面积之比 |
| 延伸率 | 材料拉制后长度与拉制前长度之比 |
| 缩径率 | 材料拉制后直径与拉制前直径之比 |
| 拉制力 | 材料拉制过程所需的作用力 |
| 拉制道次 | 满足拉制工艺所需的拉制次数 |

## 关键设计参数推荐值

### 压缩区半角（半模角）

工业拉制常用半模角范围为 2°~10°，随被拉制材料不同而有所差异[[S5,S21]]：

| 被拉制材料 | 推荐压缩区半角范围 |
|------------|-------------------|
| 低碳钢 | 2°~10° |
| 铜及铜合金 | 约 8°（多数铜线材工业拉制的常规最优值） |
| 铝及铝合金 | 约 7° |
| 不锈钢 | 稍大角度以适配更高变形抗力 |

8°半角是多数铜线材工业拉制的常规最优取值，对应拉制力接近最低值[[S9,S21,S3]]。

### 定径区长度

定径区长度推荐为拉制后产品直径的 0.6~1.2 倍。软质合金拉制模的定径区长度相对短于硬质合金拉制模。过长的定径区会增大拉制阻力、加剧模孔磨损[[S20,S26]]。

### 单道次最大减面率

实际工业生产中，受材料加工硬化与润滑条件限制，单道次减面率一般不大于理想极限值[[S21,S22,S13,S3]]：

| 材料 | 常规单道次最大减面率 |
|------|---------------------|
| 低碳钢 | 约 30% |
| 铜及铜合金 | 25%~30% |
| 铝及铝合金 | 约 30% |
| 不锈钢 | 不超过 40% |
| 理想无摩擦理论极限 | 63% |

### 拉拔速度适配区间

不同规格产品的拉拔速度区间差异显著[[S18]]：

| 产品类型 | 拉拔速度范围 |
|----------|--------------|
| 大规格棒材 | 约 0.5~2 m/s |
| 有色金属中等规格管材 | 3~100 m/min |
| 细规格线材高速拉制 | 可达数千米/分钟 |

## 制造工艺与质量要求

### 制造工艺流程

拉制模的完整制造工艺流程包括[[S24]]：

1. **模坯制备**：钢模选用 T8A/T10A 等工具钢；硬质合金模选用 YG6/YG8 等牌号模芯，并外镶钢制模套通过过盈配合提供预压应力
2. **粗加工**：圆形断面钢模在通用金属切削机床上加工；异形断面模孔采用电火花或线切割加工
3. **精加工**：通过电火花成型、振动研磨、多工序机械抛光完成模孔精加工，保证各区平滑过渡无棱边痕迹
4. **表面处理**：可根据服役需求在模孔工作面镀 0.02~0.05 mm 厚的铬层或沉积金刚石类硬质涂层以提升耐磨性
5. **出厂检测**

### 质量控制指标

拉制模出厂质量控制指标要求[[S24]]：

| 质量指标 | 要求 |
|----------|------|
| 钢模工作区域硬度 | 58~65 HRC |
| 硬质合金模表面质量 | 无裂纹、麻点缺陷 |
| 模孔成型尺寸公差 | 可控制在 ±0.005 mm 以内 |
| 工作区域表面粗糙度 | Ra ≤ 0.02 μm |

## 常见缺陷与失效形式

### 模孔开裂

失效机理：模芯未获得足够预压应力时，拉制过程中模孔受持续循环交变拉应力作用，局部应力集中区域萌生裂纹并逐步扩展贯穿模芯[[S23,S10]]。

失效判定标准：无损探伤检测发现模芯内部存在可见扩展裂纹、拉制产品出现非连续异常划伤[[S23]]。

预防优化措施：硬质合金/金刚石模芯采用过盈配合钢制模套提供预紧力，合理优化模孔过渡圆角以降低应力集中系数[[S23,S10]]。

## 与压铸工艺的应用关系

### 压铸件冷拉校直

拉制模可作为压铸件冷拉校直工装中的精准整形成型部件，对压铸得到的异形杆、薄壁管状铸件进行微量过拉校形，消除铸件内部残余应力并提升形位精度[[S2]]。

### 压铸-拉拔复合工艺

压铸-拉拔复合工艺可先通过压铸高效得到近净形粗坯，再通过后续多道次冷拉减径加工，得到力学性能显著提升、尺寸精度远超普通压铸工艺的高性能棒线材产品，大幅减少后续切削加工余量，提升材料利用率[[S11]]。
