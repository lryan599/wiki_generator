## 定义
在压铸生产与质量管理语境下，“一批”（生产批/检验批）是指**在工艺条件稳定可控的前提下，由同一台压铸机、同一套模具、同一合金牌号、连续生产的同一班次内产出的全部压铸件** [[S6,S13,S2,S20]]。该定义是开展抽样检验、质量追溯与责任判定的最小生产单元，区别于通用制造业中宽泛的同规格同订单批量概念，也不同于仅以金属液熔炼批次为边界的炉次概念 [[S20,S1]]。

现行国内压铸件国家标准（GB/T 15114、GB/T 25747、GB/T 13821、GB/T 15116）均采用统一的组批原则：同一牌号、同一台压铸机、同一套模具、连续生产的同一班次压铸件为一批次；特殊情况下的组批由供需双方商定 [[S20,S6,S13,S2]]。

## 批次划分方式
根据实际管控精度的不同，压铸行业采用多种批次划分方式：

| 划分方式 | 说明 | 典型应用场景 |
|---------|------|-------------|
| 按熔炼炉次 | 由同一铝水包/熔化炉产出的同牌号合金浇注得到的全部铸件 | 原材料质量控制，合金化学成分追溯 |
| 按模具+压铸机 | 同一台压铸机搭配同一套模具产出的连续生产件 | 压铸成型质量管控 |
| 按生产班次 | 连续运行同一班次内产出的所有同牌号同规格压铸件 | 在线产量统计与批次管理 |
| 按连续生产时段 | 工艺稳定状态下24小时或48小时内的同规格产出 | 过程控制与阶段性质量评估 |
| 按客户订单要求 | 根据交付需求定制组批 | 特定客户的交付批次管理 |

其中，ASTM压铸标准明确规定：正常生产首周的检验批为单台压铸机单套模具24小时产出；正常运行满一周后，检验批可扩大至对应生产设备48小时产出；若设备状态、合金成分或模具状态发生显著变化，需重新划定新批次边界 [[S1,S15]]。

## 批次规模的约束因素
压铸批次的典型规模并非固定不变，受多因素综合影响。核心影响因素包括：压铸机运行稳定性、模具服役状态、合金成分稳定性、铸件尺寸/复杂度/质量等级要求以及客户交付节拍 [[S23]]。

一项关键的行业禁忌是：**禁止将两台不同压铸机产出的同规格铸件合并为同一批次**。原因在于若一台压铸机的废品率高于另一台，合并后将导致整批废品率异常升高，使抽样检验的统计有效性丧失 [[S23]]。

## 批次标识与标记方式
压铸件要实现有效的批次管理，必须在铸件本体打刻易于读取的批次标记。行业主流的5种实现方式如下：

### 1. 手工刻印
生产时用印章或钢字头在铸件非加工面打印批次编号。操作简单，无需改模；缺点是标识易模糊，增加操作工时 [[S5]]。

### 2. 太阳花标识
在模具分型面加工凹形太阳花结构，铸件上铸出相应凸起标识（线条宽0.5 mm，凸出0.3 mm）。中心数字代表年份，外圈12个扇区对应1~12月，扇区内打点数量代表当月批次。优点是模具结构简单；缺点是仅能追溯到批次，无法精确到日及班次 [[S11,S5]]。

![压铸件太阳花批次标记示意图，圆形分型面标识中心数字代表年份，外圈12扇区代表月份，扇区内打点表示当月批次序号](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d78b29dc-c132-4ade-8eb0-4879640f4455/resource)

### 3. 型芯组合标识
用两组可旋转指针型芯分别标注生产日期和班次：日期型芯含31条短线代表天数，班次型芯以字母A/B/C对应不同班次。优点是追溯信息详细清晰；缺点是模具制作要求高 [[S11,S5]]。

### 4. 批次标记螺钉
模具上更换带有日期、班次标记的螺钉区分批次。螺钉端面槽内打刻年份、月份、日期及班次。压铸件上可保留清晰追溯信息，但在装配干涉部位无法实施 [[S11]]。

### 5. 推杆与柱钉组合标识
在模具柱钉安装孔内反向安装推杆，通过更换不同标记的柱钉实现批次区分。标识面与铸件基本平齐，制作成本低廉 [[S9,S11]]。

## 批次追溯体系
压铸件批次追溯的核心是从原材料进厂开始，通过产品唯一标识匹配全流程生产记录，可回溯至对应批次的操作者、生产机台、当时的压铸工艺参数及原材料炉批次等关键信息，以便快速筛选可疑件、明确质量责任 [[S5,S21]]。

![压铸产品控制程序流程图，展示从产品标识到检验、异常处理的全流程批次追溯管控逻辑](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/42cca54c-3497-4c84-bdac-70b31c31a8c5/resource)

追溯体系从原材料进厂开始，延伸至铸件交付乃至终端使用环节，使任一阶段发现的质量问题均可定位到具体工序、环节及责任人 [[S21]]。

## 抽样检验标准
国内现行压铸有色合金国家标准（GB/T 15114/15115/25747/13821/15116）均直接引用 **GB/T 2828.1**（等同采用 **ISO 2859**）作为逐批抽样检验的基础依据，同时引用 **GB/T 2829** 作为周期检验的补充标准 [[S4,S12,S7,S16,S24]]。

压铸件的尺寸、重量、表面质量及内部质量项目需按该标准进行抽样，抽样数量随批次规模匹配，通常每批抽样数量不低于3件 [[S3]]。

### 抽样检验典型示例
在选定的典型条件下，应用标准抽样方案的实例：

| 参数项 | 取值 |
|--------|------|
| 批次总数量 | 600件 |
| AQL（可接收质量限） | 1.5% 废品率 |
| 检验等级 | I级 |
| 抽样方案 | 一次抽样 |
| 抽样检验数量 | 80件 |
| 接收判定（Ac） | 废品数 ≤ 3时，判定该批合格 |
| 拒收判定（Re） | 废品数 ≥ 4时，判定该批不合格，须全检筛选 |

该方案来源于BS 6001（与ISO 2859对应）中相应的抽样表，压铸行业广泛采用这一统计检验逻辑 [[S23]]。

## 镁合金压铸件取样规定
以GB/T 25747—2022《镁合金压铸件》为例，不同检验项目的取样要求明确区分：

| 检验项目 | 取样规则 |
|---------|---------|
| 化学成分 | 炉内取样，每批1个试样 |
| 力学性能 | 单铸或本体取样，每批不少于3根（按GB/T 13822） |
| 尺寸 | 按GB/T 2828.1、GB/T 2829抽检，每批不少于3件 |
| 重量 | 按GB/T 2828.1、GB/T 2829抽检，每批不少于3件 |
| 表面质量 | 逐件检验 |
| 内部质量 | 按GB/T 2828.1、GB/T 2829抽检，每批不少于3件 |

上述取样规则适用于出厂检验，型式检验项目由供需双方商定 [[S3]]。

## 检验结果的批次判定原则
根据现行国标体系，压铸件各检验项目的批次判定遵循统一逻辑 [[S20,S6,S13,S2]]：

- **化学成分**：初次检验不合格，允许取双倍数量试样复验；复验全部合格则判该批合格，仍有试样不合格则判该批不合格。
- **力学性能**：试样每组3根，如其中2根不合格则判该批不合格；允许以加倍试样复验，若复验中仍有2根不合格但总平均值合格，可判该批合格；不合格试样多于2根则判该批不合格。
- **尺寸**：任一抽检产品尺寸不合格，判该批次不合格。
- **重量**：任一抽检产品重量不合格，判该批次不合格。
- **表面质量**：任一抽检产品表面质量不合格，判该批次不合格。
- **内部质量**：任一抽检产品内部质量不合格，判该批次不合格。

## 关联术语辨析
压铸场景下，“一批”相关的核心术语具有明确的边界区分：

| 术语 | 定义 | 边界特征 |
|------|------|---------|
| 炉次（heat number） | 单一熔炼炉次内产出的全部同牌号熔融金属及后续浇注得到的铸件 | 仅以熔炼环节为边界 |
| 生产批次（production batch） | 同设备、同模具、同班次产出的全部铸件 | 覆盖压铸成型全环节，是责任判定的基本单位 |
| 检验批次（inspection lot） | 抽样检验的统计单元 | 可由多个连续稳定生产的炉次/生产批次合并构成 |
| 子批次（sub-lot） | 大批次内按更小生产时段拆分的管控单元 | 用于精细化质量管控 |

对于压铸原材料的组批，标准规定由同一铝水包或熔化炉的产品组成一个批次 [[S8]]；压铸镁合金锭也可由同一牌号的多个熔化炉产品组成一批 [[S25]]。ASTM标准进一步明确：检验批为24小时或48小时生产时段内的产出 [[S1,S15]]。

## 常见异常问题
压铸行业批次管理中的典型异常包括 [[S18]]：

- **混批**：不同工艺条件（不同设备、模具或参数状态）产出的铸件混合流入后工序，导致质量问题难以追溯；
- **批次标识遗漏**：铸件本体无标记或标记缺失，无法进行有效追溯；
- **追溯链路失效**：生产记录缺失或不完整，无法定位问题根因；
- **批次一致性不足**：不同批次铸件的工艺波动导致产品性能一致性差。

上述问题可能直接导致批量报废、客户拒收或召回等严重质量损失 [[S18]]。
