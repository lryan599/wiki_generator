## 定义与范畴

**砂芯铸造模下模（下芯盒）** 是砂芯铸造模（芯盒）的下半成型部分，依据GB/T 8845-2017《模具术语》定义为“安装在成型设备下工作台面的砂芯铸造模部分”[[S31,S2]]。其核心功能是在制芯流程中承载芯砂，为顶出机构、排气腔、密封结构提供安装基体，与上芯盒合模后共同围出完整砂芯成型型腔，支撑冷芯盒、热芯盒等各类制芯工艺完成砂芯固化、顶出的全流程操作[[S6,S1]]。

## 术语从属与并列关系

砂芯铸造模（芯盒）是上位类总术语，下芯盒与上芯盒为水平对开式芯盒的两个平行分体组成部件[[S31]]。下芯盒/上芯盒与定芯盒/动芯盒属于不同分型维度的术语体系：前者以安装在制芯设备上下工作台面的位置划分，后者以制芯设备固定/移动工作台的安装属性划分。实际生产中，部分工况下下芯盒可直接作为定芯盒使用[[S17]]。

对应英文检索词包括：lower core box、bottom core box、core box lower half、drag half of core box。

## 核心结构组成

典型下芯盒的核心组成结构包括[[S24,S20,S15]]：

| 组成部分 | 说明 |
|---------|------|
| 下芯盒本体 | 成型砂芯下半型腔的主体结构，通常为带有凸缘的薄壁箱形结构 |
| 芯腔 | 直接成型砂芯下半部分的工作型腔 |
| 底板 | 下芯盒底部基础板，连接制芯设备工作台 |
| 分盒面凸缘 | 与上芯盒接触的配合面，加宽加厚以提高刚度和便于操作 |
| 耐磨护板 | 安装在铝质芯盒凸缘面的钢板，防止磨损 |
| 定位销 | 预装在下芯盒上，与上芯盒销套配合实现合模定位 |
| 排气装置 | 包括排气槽、排气塞等，排出制芯过程中产生的气体 |
| 顶出机构安装位 | 容纳顶芯杆、推板等出芯组件的空间 |
| 加强筋 | 中大型下芯盒外壁配置，提升强度与刚度 |

## 分类形式

### 按结构形式分类

**整体式下芯盒** 为单块整体铸造成型结构，结构刚性好、定位精度高，适合中小批量或形状简单的砂芯生产[[S1,S30]]。敞口整体式芯盒填砂与砂芯支撑面为同一平面，填砂紧实后翻转180°起芯[[S22]]。

**分体式下芯盒** 采用盒体与局部镶块拼装结构，便于复杂型腔加工，易磨损部件可单独更换，大幅降低制造成本和维护难度[[S1,S30]]。

### 按制芯工艺分类

**热芯盒工艺用下芯盒** 需配套内置电加热管的加热板结构，长期在200℃以上高温环境下工作[[S25,S11]]：

| 特性 | 参数 |
|-----|------|
| 壁厚 | 20~30 mm（不含加热板） |
| 主体材质 | 灰铸铁（HT150/HT200）为主 |
| 附加装置 | 加热板、电热管、水冷射砂板 |
| 壁厚最低要求 | 最薄处不小于20 mm（电加热法）[[S29]] |

**冷芯盒工艺用下芯盒** 无需配置加热装置，但需增设密封构件[[S6,S4]]：

| 特性 | 说明 |
|-----|------|
| 密封要求 | 分盒面和顶出机构位置需设O形密封圈、密封胶条 |
| 排气方式 | 配套排气腔盖板实现固化废气定向收集 |
| 可选材质 | 铝合金、木材、塑料等多种轻量化材质 |
| 制造成本 | 较低，无需加热系统 |

## 材料选型

依据不同制芯方法、砂芯尺寸及生产批量，下芯盒主体材料选择如下[[S20,S17]]：

| 材料类别 | 常用牌号 | 适用场景 | 特点 |
|---------|---------|---------|------|
| 铝合金 | ZL101、ZL102、ZL104、ZL201 | 中小规格芯盒 | 轻巧、易加工、表面光洁、不生锈；但强度硬度低、不耐磨，需在摩擦面加装30号钢耐磨片 |
| 灰铸铁 | HT150、HT200 | 大型芯盒、大批量制芯 | 强度硬度高、耐磨性好、价格低廉；但较重易锈 |
| 铜合金/碳素钢 | — | 活动镶块、活块构件 | 满足高耐磨性要求 |

铸铁材质（HT150/HT200）的强度及抗热疲劳性能优于铝合金，适合大批量长期制芯工况[[S20]]。

## 关键设计参数与技术条件

### 尺寸与精度

金属芯盒内腔工作表面尺寸公差及表面粗糙度要求[[S1,S12]]：

| 参数项 | 要求值 |
|-------|--------|
| 型腔尺寸公差 | ≤0.2 mm |
| 表面粗糙度 Ra | ≤1.6 μm |
| 分盒面间隙 | <0.1 mm[[S3,S5]] |
| 未注起模斜度 | 常规取值2°[[S16]] |

### 壁厚与加强筋

下芯盒壁厚依据平均轮廓尺寸和材质选型，常规中小铝合金芯盒壁厚约10 mm[[S20,S12]]。加强筋设计参数[[S24,S9]]：

| 参数项 | 要求值 |
|-------|--------|
| 加强筋厚度 | 芯盒壁厚的0.8~1.0倍（δ₁ = 0.8δ） |
| 加强筋高度 | h ≥ 3δ（最低不少于3倍壁厚） |
| 加强筋斜度 | 0.5°~2°（根据高度选取） |
| 热芯盒壁厚 | 不小于20 mm（不含加热板），配合满足 h≥3δ 的加强筋可有效抑制变形[[S29]] |

### 定位配合

定位销通常安装在下芯盒上，上芯盒对应安装定位销套[[S20,S15]]：

| 参数项 | 要求值 |
|-------|--------|
| 销/套材质 | T8/T10工具钢 |
| 工作部分淬火硬度 | 40~45 HRC |
| 常用定位销直径 | 8 mm、10 mm、12 mm（适配不同尺寸芯盒） |
| 首批基准芯盒定位销中心距偏差 | ±0.05 mm |
| 不使用成型烘干器时定位销孔中心距偏差 | ±0.25 mm[[S3,S5]] |
| 合模后重合性尺寸偏差 | 见对应精度等级表[[S3,S5]] |

### 耐磨防护

铝质下芯盒需在凸缘面加装耐磨护板：Q235-A钢制，厚度3 mm，用沉头螺钉紧固[[S9,S20]]。防磨片上紧固螺钉头应低于工作表面，并在沉头螺钉头开口两端“冲眼”防松[[S5]]。活块与窝座间隙不超过0.2~0.3 mm[[S3,S5]]。

## 工艺作用与设备接口

### 制芯流程中的作用

在制芯过程中，下芯盒承载芯砂并与上芯盒闭合形成完整型腔。射砂填砂时，下芯盒提供型腔下边界和支撑面；压实和固化阶段，下芯盒维持型腔形状稳定；开模取芯时，顶出机构通过安装在下芯盒下方的顶柱、推板、顶芯杆组件将砂芯推出[[S2,S3]]。

### 设备接口适配

机用下芯盒底部必须设置与射芯机工作台连接的安装耳和螺栓孔，孔位尺寸需完全匹配制芯机工作台的T形槽参数[[S2,S3]]。下芯盒配套的下模座板需预留足够空间安装顶柱、下推板、下推杆组件，保证设备推出力可无损耗传递到顶芯杆完成砂芯脱模[[S3,S2]]。

### 砂芯与金属型/半固态压铸的接口

| 工艺类型 | 接口要求 |
|---------|---------|
| 金属型铸造 | 砂芯芯头与金属型对应安装位的配合间隙控制在0.2~0.3 mm，避免合模时芯头挤压断裂[[S18,S23]] |
| 半固态压铸 | 砂芯尺寸精度偏差需≤0.2 mm，同时预留砂芯受热膨胀的补偿间隙，防止压铸过程中砂芯挤压破损出现跑铝缺陷[[S18,S23]] |

## 常见缺陷与失效模式

### 定位失效与错位

下芯盒与上芯盒的定位销、销套长期磨损会导致配合间隙超标。研究表明，典型的销-套间隙为0.25 mm，加上0.35 mm的附加磨损量后，上下芯盒横向错位偏移最大可达1.5 mm，直接引发砂芯错边、尺寸超差[[S19,S5]]。应对措施：首批基准芯盒定位销中心距偏差控制在±0.05 mm以内，定期检测定位副磨损量，及时更换磨损超差的销套组件[[S27]]。

### 磨损

下芯盒分盒面、刮砂面频繁经受开合摩擦和砂流冲刷，是磨损高发区域[[S9]]。铝质芯盒耐磨性不足，需加装耐磨护板。活块燕尾槽配合面磨损会导致活块定位精度下降，日常需定期检测配合间隙，超差后及时更换钢质耐磨镶件[[S20]]。芯头处磨损量不得超过芯头间隙的40%[[S26]]。

### 变形

铝合金下芯盒若壁厚不足、未按要求设置加强筋，长期经受制芯、加热循环载荷易出现分盒面平面度超差变形[[S24,S26]]。热芯盒下芯盒壁厚不小于20 mm（不含加热板），配合满足 h≥3δ 的加强筋设计，可有效抑制长期工况下的变形失效[[S29,S24]]。

### 热疲劳裂纹

热芯盒下芯盒长期承受冷热交替循环应力，若壁厚不足、材料热疲劳性能不达标，易在边缘尖角、应力集中区域萌生裂纹。铸铁材质（HT150/HT200）的抗热疲劳性能优于铝合金，可减少裂纹萌生概率[[S29,S20]]。

### 粘砂

冷芯盒下芯盒内腔表面粗糙度高于Ra 1.6 μm、脱模剂喷涂不到位时，芯砂易粘附在芯腔表面形成粘砂缺陷，直接影响砂芯表面质量[[S7,S1]]。预防措施：保持内腔干燥无油污，每班生产前均匀喷涂脱模剂（如硅胶、石墨或聚四氟乙烯喷涂）[[S7]]。

## 维护要点

日常每班停机后，操作工需清除下芯盒分盒面、芯腔内表面、销套上的积砂和污垢，清理通气塞、排气槽内残留垫砂，检查紧固件牢固性及活块、镶块是否松动位移，清理完成后均匀喷涂分型剂[[S8]]。

维修工定期点检内容包括[[S8]]：
- 下芯盒与上芯盒的外形错边偏差
- 芯盒工作面的磨损量
- 定位销/套的磨损程度
- 下芯盒本体的变形量
- 检查不合格组件及时更换
