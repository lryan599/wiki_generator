## 概述

“中间合金法”在铝合金领域需区分两种不同应用场景。传统熔铸领域通用的“中间合金”泛指为添加难熔合金元素而预制的二元或三元中间媒介，用于常规合金的成分调控[[S16,S20]]。在颗粒增强铝基复合材料领域，“中间合金法”是一类细分液态制备技术，其核心为：预先制备含有高体积分数陶瓷增强相的专用中间合金块体（如冷压致密预制块或燃烧合成铸锭），再将此中间合金加入常规铝熔体中，通过后续分散工序使增强颗粒均匀分布于铝基体中[[S10,S24]]。

## 定义

中间合金法是一种颗粒增强铝基复合材料的液态制备技术。该方法先将金属粉与高体积分数的陶瓷增强相制成中间合金，再利用机械搅拌或超声波辅助将中间合金分散于铝熔体，经浇铸或压铸成型，获得颗粒增强铝基复合材料[[S10,S24]]。该法属于搅拌铸造大类下的细分液态复合工艺[[S18]]。

## 工艺分类定位

在非连续增强铝基复合材料的制备体系中，中间合金法属于搅拌铸造大类的液态复合分支，与浸渗铸造法、粉末冶金法、喷射法和反应合成法并列为五大主流路径[[S18]]。

## 工艺原理与流程

### 中间合金制备阶段

以TiC/Al、TiB₂/Al、(TiC-TiB₂)/Al中间合金为例，典型制备流程为[[S10]]：
1. **球磨混合**：将配制好的金属粉与陶瓷增强相混合粉同ZrO₂球按球料比8:1混合，在50rpm转速下球磨24h；
2. **冷压成型**：将混合粉末冷压为φ40mm×30mm的致密预制块；
3. **燃烧合成**：将冷压块置于100Pa真空炉中，以30℃/min速率升温至约900℃引燃燃烧合成反应，保温10min后对块体施加50MPa轴向压力并保压90s，获得成品中间合金。

### 铝熔体复合阶段

中间合金加入铝熔体后的复合工艺步骤为[[S10]]：
1. **熔铝与加入中间合金**：将铝锭在800–850℃下完全熔化，加入定量预制中间合金；
2. **机械搅拌分散**：对熔体进行不少于3min的充分机械搅拌；
3. **超声波辅助分散**：采用频率20.35kHz的超声波对熔体进行3min辅助分散；
4. **浇铸成型**：将熔体降温至750℃，倒入预热至100℃的钢模中浇铸成型，获得颗粒增强铝基复合材料铸锭。

## 中间合金类型与形态

颗粒增强铝基复合材料领域专用中间合金的主要类型与制备方式如下[[S10,S3,S16]]：

| 中间合金类型 | 常见形态 | 主流制备工艺 |
| ------------ | -------- | ------------ |
| TiC/Al | 冷压致密预制块、合金铸锭 | 燃烧合成法 |
| TiB₂/Al | 冷压致密预制块、合金铸锭 | 燃烧合成法 |
| (TiC-TiB₂)/Al | 冷压致密预制块、合金铸锭 | 燃烧合成法 |
| Al-Cr、Al-Cu、Al-Fe等（通用型） | 铸锭 | 熔配法、热还原法 |

## 工艺参数与控制要点

### 压铸适配的中间合金法制备参数

在颗粒增强铝合金的压铸应用中，典型的中间合金法工艺参数如下[[S7,S10]]：

| 参数项 | 范围/条件 | 备注 |
| ------ | --------- | ---- |
| 铝锭熔化温度 | 800–850℃ | 完全熔化后加入中间合金 |
| 机械搅拌时间 | ≥3min | 充分机械搅拌 |
| 超声波频率 | 20.35kHz | 辅助分散 |
| 超声波持续时间 | 3min | 伸入熔体内处理 |
| 浇铸温度 | 750℃ | 倒入预热至100℃的钢模 |

### 中间合金通用技术要求

用于压铸的中间合金应满足[[S9]]：
- 熔点与目标压铸合金的熔炼温度接近；
- 化学成分均匀稳定；
- 目标添加元素占比尽可能高，以减少中间合金用量降低杂质带入风险；
- 材料脆度适宜，便于破碎配料；
- 长期储存不发生成分劣化或腐蚀变质。

## 常见增强颗粒与基体合金

颗粒增强铝合金体系常用的陶瓷增强颗粒包括：SiC、Al₂O₃、B₄C、TiB₂、TiC等，颗粒尺度覆盖微米尺度、纳米尺度及微纳混杂尺度三大类[[S27]]。

商用Duralcan Al/SiCₚ颗粒增强铝基复合材料适配AA360高压压铸基体，SiC颗粒的体积分数一般控制在10%~20%区间；高压压铸伴随的高冷却速率可显著降低增强颗粒的宏观偏析倾向，提升铸件力学性能[[S33]]。

压铸领域广泛使用的Al-Ti-B中间合金中，TiB₂颗粒常规控制尺寸≤2μm，极少量最大不超过5μm；工业量产商用牌号涵盖Al-5Ti-0.2B、Al-5Ti-0.6B、Al-5Ti-1B三个主流系列[[S28]]。

## 优势与特点

中间合金法相对于直接纯金属/颗粒漩涡加入法的优势[[S22,S32]]：
- 避免高熔点元素直接加入导致的铝合金熔体过热；
- 显著提升难熔、易氧化合金元素的实收率；
- 合金化学成分控制精度和均匀性更高；
- 可有效抑制密度差引发的宏观成分偏析；
- 适配多元复杂高力学性能压铸铝合金的规模化生产；
- 熔炼过程整体能耗更低。

## 局限与挑战

该方法的局限性包括[[S22,S32]]：
- 额外的中间合金预制工序提升了整体制备成本；
- 中间合金预制环节可能引入额外杂质造成熔体污染；
- 长期高温保温状态下部分高密度增强颗粒存在沉降风险。

## 应用场景

中间合金法可适配高压压铸、低压压铸、重力铸造三类主流铸造场景，可直接嵌入现有压铸产线，无需大规模设备改造[[S6,S17]]。目前已量产应用的颗粒增强铝合金部件包括汽车制动盘、发动机支架、传动系统壳体、新能源汽车车身结构件等高耐磨、高强度轻量化构件[[S6,S17]]。

低压铸造制备的SiC颗粒增强铝基复合材料可获得力学性能优于常规重力铸造的铸件；SiC颗粒添加后复合材料的硬度、耐磨性随颗粒体积分数提升而上升，抗拉强度先升高后降低，延伸率随颗粒含量上升而下降[[S6]]。

## 与相关方法的关系对比

### 搅拌铸造法

直接向熔融金属基体中添加外部增强颗粒，通过机械搅拌混合均匀后浇注成型。优势是工艺简单、设备成本低、适合大规模生产、增强体种类与含量可调；缺点是增强颗粒与铝熔体润湿性差、易团聚、搅拌过程易卷入气孔、颗粒分布易因密度差上浮或下沉[[S17,S6,S26]]。

### 原位反应法

在金属基体内通过元素/化合物间的化学反应原位生成热力学稳定的陶瓷增强相。优势是增强颗粒尺寸细小、界面清洁无杂质污染、与基体润湿性好、界面结合强度高；缺点是反应体系对增强体种类约束大，反应产物尺寸和形貌控制难度较高，一般需配套搅拌工序，整体工艺路线复杂、能耗较高[[S6,S17,S25]]。

### 粉末冶金+挤压法

将金属粉末与增强颗粒混合，经冷压、低温烧结后通过热挤压致密化成型。优势是增强相分布均匀、基体与增强体配比灵活可调、可制备高体积分数增强复合材料，避免全熔高温工序带来的氧化与界面过量反应问题；缺点是设备复杂度高、生产成本高、难以生产形状复杂的近净形铸件，量产规模受限[[S17,S25]]。

### 中间合金法与其他工艺的组合

中间合金法可与上述三类工艺组合应用[[S15,S21]]：
- 将预分散有纳米增强陶瓷颗粒的中间合金加入常规搅拌铸造熔体，可改善外加增强颗粒的润湿性与分散均匀性；
- 作为中间添加工序配套原位反应法使用，可进一步细化原位生成的增强颗粒尺寸；
- 将中间合金预制块作为粉末冶金工序的原料，可提高混合粉末的分散度，提升最终挤压态复合材料的综合力学性能。
