## 定义

压射杆是压铸机压射机构的核心传力部件。根据国家标准 GB/T 25370-2020《铸造机械 术语》的正式定义，压射杆（英文对应术语为 **plunger rod**，别名**冲头杆**）是“冲头与压射驱动装置之间的连接杆” [[S5,S12]]。在冷室压铸机中，压射杆的前端通过螺纹与压射冲头（俗称锤头）刚性固连，后端通过联轴节或连接法兰与压射缸内部的压射活塞连接，动力由压射活塞传递至压射杆，再驱动压射冲头完成压射动作 [[S16,S49]]。

### 与相关部件的边界区分

- **压射冲头（压射头/锤头）**：直接接触熔融金属、在压室内往复运动的柱塞，作用是直接推送金属液进入型腔并承受金属凝固阶段的压力 [[S12]]。
- **压射活塞杆**：安装在压射活塞上、位于压射缸内部的传动杆件，属于液压缸的组成部分 [[S22]]。
- **压射杆**：连接压射活塞杆末端和压射冲头的中间传动部件，既不直接接触高温金属液，也不处于压射缸内部的液压油工作环境中 [[S12,S22]]。

## 工艺作用

在压铸的三个核心压射阶段中，压射杆分别承担不同的传力任务 [[S45,S49]]：

1. **慢速压射阶段**：传递压射活塞的动力，带动冲头低速运动，推动金属液缓慢越过压室浇料口，避免金属液溅出并充分排出压室内多余气体。
2. **快速充型阶段**：传递蓄能器释放的高压动力，带动冲头高速推进，将金属液高速注入模具型腔完成充型。
3. **增压阶段**：传递增压器输出的高压载荷，将高压作用于充型完成的金属液，实现压实补缩，保证铸件内部致密性。

## 分类与结构

### 冷室与热室压铸机的压射杆差异

| 特征 | 冷室压铸机 | 热室压铸机 |
|---|---|---|
| 布置方式 | 水平布置 | 垂直布置 |
| 压射比压 | 40~60 MPa | 7~14 MPa |
| 工作行程 | 较长 | 较短 |
| 工作环境 | 不长期接触高温熔液 | 长期处于熔炉上方高温环境 |
| 冷却结构 | 通常带内部强制冷却通道 | 部分小型机型无强制冷却结构 |

冷室压铸机的压室与保温炉分开放置，压室水平放置；热室压铸机的压室直接置于熔炉坩埚内，压室竖直放置 [[S6,S50]]。

### 典型结构形式

- **实心压射杆**：结构简单、整体刚性高，适用于小型低吨位热室压铸机、低压压铸设备。无内部冷却水路，制造成本低 [[S7,S10]]。
- **空心带内部冷却通道的压射杆**：内部设置进出水回路，采用内插铜管强制冷却结构，可将低温冷却水优先输送至靠近冲头的高温端，适用于中大型冷室压铸机，可有效降低杆体热变形，延长整体使用寿命 [[S7]]。

### 连接方式

- **冲头侧**：现代设计可采用多组螺纹紧固件连接，取代传统单一内外螺纹连接方式，避免螺纹变形导致冲头难以拆卸，提升维护便捷性 [[S7]]。
- **活塞侧**：通过压射杆末端的凸肩与压射缸侧的螺帽/联轴节固定，保证传动时载荷均匀传递 [[S10,S25]]。

### 材料性能特征

**42CrMo** 属超高强度合金结构钢，调质处理后屈服强度≥930 MPa，弹性模量 212 GPa，具备优良的淬透性、无明显回火脆性，拥有较高的疲劳极限和抗多次冲击能力，低温冲击韧性良好 [[S11,S13]]。

**H13（SKD61/1.2344）** 热作模具钢化学成分为 C 0.39%、Si 1.00%、Cr 5.40%、Mo 1.35%、V 1.00%，高温耐磨性好、抗热裂能力强，氮化处理后表面硬度可进一步提升 [[S38,S23]]。SDCE 标准推荐冷室压铸机压射杆使用 H13 热作模具钢，要求热处理后硬度达到 46~48 HRC [[S25]]。

**40Cr** 经调质处理后屈服强度可达 785 MPa 以上，伸长率≥9%，拥有良好的低温冲击韧性和低缺口敏感性，适用于中低载荷的压铸场景 [[S4]]。

### 材料适用工况对比

| 材料系列 | 适用场景 | 特点 |
|---|---|---|
| 42CrMo (4140) | 常规中小吨位冷室压铸机，铝/镁/锌合金非铜合金压铸 | 成本适中，常温力学性能优异 |
| H13 (SKD61/1.2344) | 大吨位高比压压铸机，高温铜合金压铸或长期连续高强度作业 | 耐热疲劳和高温耐磨性更优 |
| 40Cr（低成本结构钢）| 小吨位热室压铸机，低载荷低产能经济型场景 | 成本低，仅适用于低载荷工况 |

## 工艺参数关联

### 压射力与比压

压射力的大小由压射缸截面积和工作液压力决定。最大增压压射力一般为压铸机合模力的 1/10 左右，压射比压的常规适用范围为 30~500 MPa，需根据铸件材质、致密性要求进行针对性选择 [[S19,S28,S32]]。

UH1250t 规格超高速压铸设备的实测压射力范围为 570~1041 kN，搭配 φ140 mm 冲头时可获得的铸造压力（压射比压）范围为 37~68 MPa [[S8]]。

### 压射行程

压射行程由压射缸结构直接确定，决定冷室压铸机的冲头推出距离 [[S18,S35]]。压射行程计算公式为 [[S27]]：

$$L_p = \frac{4G}{\pi d_k^2 \gamma K_k}$$

式中：\\(G\\) — 单次浇注金属量；\\(\\gamma\\) — 合金比重；\\(K_k\\) — 充填系数（取值 0.75）；\\(d_k\\) — 压室直径。

### 压射杆直径选取

压射活塞杆直径选取需同时考虑刚性要求与回程力需求，推荐压射缸横截面积与压射杆横截面积的比值约为 0.4，对应压射杆直径约等于压射缸直径的 0.63 倍 [[S27]]。

### 动态工作应力

实测 4.0 m/s 注射速度工况下，压铸机压射杆最大工作应力可达 43 MPa，稳定运行工况下常规应力水平约 30 MPa，存在明显的冲击压和补偿压特征区间 [[S24,S39]]。

### 长径比限制

压射杆长径比过大时自身刚性不足，易在高速压射的惯性冲击下发生挠曲变形，限制设备最高允许压射速度，同时会增大增压阶段的杆身弹性变形量，直接缩短有效增压行程，因此长径比参数必须和最大允许压射速度阈值、增压行程设计值进行匹配校核 [[S51,S8]]。

## 常见缺陷与失效模式

| 失效类型 | 产生原因 | 影响后果 |
|---|---|---|
| **杆身弯曲** | 长径比设计不合理、材料硬度偏低、受偏心载荷或非轴向外力冲击，发生塑性弯曲失稳 [[S51,S17]] | 压射运动不同心，运动卡滞，压射力传递偏心，铸件局部产生飞边、金属填充不均 |
| **表面磨损** | 与密封导向部件反复滑动摩擦，金属磨屑、铝渣等硬质颗粒进入摩擦界面加剧磨损 [[S30,S46]] | 杆身尺寸超差，导向配合间隙增大，运行晃动量增大，压射速度稳定性降低 |
| **表面拉伤** | 表面硬度过低、润滑条件不良，配合界面发生黏着咬合 [[S31,S44]] | 金属碎屑进入配合间隙加剧磨损，划伤后表面应力集中风险升高，提升疲劳断裂概率 |
| **疲劳断裂** | 长期承受交变拉压载荷、热交变应力作用，在表面划痕、截面过渡尖角等应力集中位置萌生微裂纹并逐步扩展 [[S20,S40]] | 压射传力路径突然中断，型腔金属填充不足产生批量废品，严重时引发设备撞击事故 |
| **冲头连接松动** | 螺纹连接部位长期受交变冲击载荷作用、多次装拆磨损导致螺纹配合间隙变大 [[S30,S42]] | 压射杆与冲头不同轴，产生附加偏心弯矩，加速杆身弯曲和密封磨损，增压建压时间变长，铸件出现内部缩松缺陷 |

## 设计标准与尺寸规范

依据 SDCE（北美压铸学会）冷室压铸机推荐标准，与 NADCA 规范同源的压射杆设计要求如下 [[S10,S25,S36]]：

| 要求项目 | 规范指标 |
|---|---|
| 材料最低抗拉强度 | ≥793 N/mm² (115,000 Psi) |
| 前端螺纹标准 | 二级统一螺纹配合标准 |
| 螺纹导向段直径公差 | +0.030 / +0.000 / -0.001 英寸，精磨后尺寸大于螺纹外径 |
| 两端凸肩平面与杆轴线垂直度偏摆 | ≤0.001 英寸 |
| 螺纹及导向直径与杆轴线同轴度偏摆 | ≤0.002 英寸 |
| 冷却水回流截面积 | 不得小于进水管截面积 |
| 油缸末端凸肩 | 适配配套螺帽或联轴节尺寸 |

## 装配与维护

### 与相关部件的装配关系

压射杆前端通过连接螺纹与压射冲头刚性固连，后端通过联轴节或连接法兰与压射缸内部的压射活塞刚性连接 [[S16,S22,S49]]。压射杆内部设置贯通的冷却水路，与压射冲头内置冷却回路连通，实现强制连续冷却 [[S7]]。压射杆穿过压射缸导向套做直线往复运动，压射活塞外缘配套的活塞密封组件与压射缸内壁形成动密封，隔绝高低压油腔 [[S22,S49]]。

### 分级维护要点

压射杆纳入压铸机分级定期维护体系 [[S1,S3,S48]]：

| 维护等级 | 工作内容 |
|---|---|
| **日常维护** | 检查杆身表面有无拉伤痕、油液泄漏点，同步核验压射冲头与压室的同轴度 |
| **定期检查** | 拆解检查压射杆与活塞的连接紧固状态、配套密封件磨损情况，及时更换老化失效的密封组件 |
| **小修** | 清洁杆身导向面，重新润滑导向副 |
| **中修** | 检测杆身直线度，修复局部轻微拉伤 |
| **大修** | 杆身拉伤深度超过许用值、直线度超差无法修复时直接报废更换 |
