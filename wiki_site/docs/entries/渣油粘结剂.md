## 1. 概述

**渣油粘结剂**（residual oil binder）是一种以石油炼制过程中减压蒸馏塔底部残余物为基础原料，经稀释、改性制成的铸造用有机油类粘结剂。依据GB/T 5611-2017《铸造术语》的正式定义，渣油粘结剂是“由炼油厂蒸馏塔底部剩留的残渣制成的油类粘结剂”，通常由减压渣油加适量催干剂，用煤油或轻柴油按2∶1左右的比例稀释制备而成[[S8,S11,S6,S13]]。

在铸造粘结剂体系中，渣油粘结剂归属于**有机粘结剂**大类下的**油类粘结剂**子类，按材料来源属于**人造材料**中的**工业副产品**类粘结材料[[S24,S17]]。该粘结剂主要用于替代传统植物油（如桐油、亚麻仁油）制作砂型铸造的Ⅱ级和Ⅲ级砂芯，经环烷酸铁等催干剂改性的产品可扩展至汽车、机械、液压件等各类合金铸件的Ⅰ~Ⅱ级砂芯[[S21,S18]]。

## 2. 分类与定位

### 2.1 在粘结剂体系中的位置

铸造用粘结剂按化学属性分为无机和有机两大类别，渣油粘结剂属于有机类中的油类分支[[S24]]。按材料来源划分，砂芯粘结剂可分为天然材料和人造材料两大类，渣油明确归入人造材料下的工业副产品类别，与合脂粘结剂、纸浆废液等同属一个子类，区别于酚醛树脂、呋喃树脂等合成树脂类粘结剂[[S17]]。

### 2.2 与相似材料的区别

**与石油沥青粘结剂的区别**：渣油粘结剂由减压渣油直接添加催干剂和稀释剂制成，无需经过氧化工序；而石油沥青类铸造粘结剂是减压渣油再经深度氧化得到的产物，铸造常用软化点为45~60℃，且需配制成乳化液方可使用[[S12,S19,S7]]。二者在制备工艺和产品形态上存在本质差异。

**与重油的定位区别**：渣油是经过专门调配的独立芯砂粘结剂；而普通重油在砂型铸造中通常作为湿型砂的附加剂与煤粉配合使用，作用是增强防粘砂效果和改善起模性能，并非独立的砂芯粘结剂[[S3]]。

## 3. 原料与化学组成

### 3.1 基础原料

渣油粘结剂的基础原料为原油减压蒸馏过程塔底产出的减压渣油。不同产地原油加工的减压渣油性能存在明显差异，例如大庆减压渣油S76基体密度为0.9345 g/cm³，软化点42℃，黏度2.69 Pa·s，中性胶质占35%，油和蜡占55%，沥青质占10%；大庆S61基体密度0.95 g/cm³，软化点38~40℃，黏度10~15 Pa·s，中性胶质占20%~23%，油和蜡占70%~75%，沥青质占2.8%[[S1]]。

### 3.2 化学组成

渣油的化学基础为各类碳氢化合物及氧、硫、氮等衍生物的大分子混合物。主要组分为：

- **油质**：占主体；
- **胶质**：质量分数为20%~25%[[S21]]；
- **沥青质**：少量；
- **碳氢质和半焦油质**：质量分数均低于1%[[S21]]。

### 3.3 硬化机理

渣油粘结剂的硬化过程属于复杂的物理化学反应，分阶段进行：油质在加热过程中蒸发胶凝并向胶质转化；胶质进一步发生氧化、聚合反应生成沥青质；进一步升高温度或延长烘烤时间，沥青质又向碳氢质和半焦油质转化。胶质和沥青质均为多环结构，通过环状结构的侧链进行反应，使高分子胶质和沥青质交织，溶胶转变为凝胶乃至固态，最终在砂粒表面形成韧性的亮黑色固体粘结膜，使砂芯获得强度[[S21,S20]]。

## 5. 工艺使用参数

### 5.1 稀释与配比

渣油使用前必须加稀释剂降低粘度。推荐采用裂化柴油作稀释剂，渣油与裂化柴油的质量比控制在2∶1~4∶1时稀释效果最优；煤油亦可作为稀释剂，但效果弱于裂化柴油[[S21,S16]]。

芯砂中渣油粘结剂的常规添加量为**原砂重量的3%~4%**；非铁合金铸件（铝合金等）用砂芯的渣油加入量可适当降低[[S21]]。

### 5.2 附加物

渣油砂中常加入**占原砂重量约2%的膨润土**，糊精可少加或不加。膨润土与糊精的推荐质量比为（2∶1）~（3∶1）。在相同附加物条件下，渣油砂的湿压强度高于同配方的桐油砂和合脂砂[[S16,S18]]。水分随粉状附加物的加入而加入，适宜水分加入量为膨润土量的1/2或膨润土加糊精量的1/3[[S18]]。

### 5.3 催干剂

为降低烘干温度和扩大烘烤温度范围，可加入环烷酸铁等催干剂。环烷酸铁效果最优，**添加量为渣油重量的9%~12%**，可将烘干温度降低约20℃[[S21,S16,S20]]。也可选用FeCl₃、AlCl₃，但效果不及环烷酸铁[[S20]]。

### 5.4 混制工艺

渣油砂的标准混制工艺流程为[[S16,S18]]：

```
原砂 + 粉状附加物 →[混拌2 min] 加水及液态附加物 →[混拌10~15 min] 加入渣油 →[混拌2~5 min] 出砂
```

### 5.5 烘干硬化规范

渣油砂采用热烘干硬化工艺，不适用于室温自硬或气硬类制芯工艺[[S16]]：

| 条件 | 参数 |
|------|------|
| 推荐烘烤温度 | 230~250℃ |
| 最佳烘干温度（工艺试样，未添加催干剂） | 250~260℃ |
| 添加催干剂后烘烤温度 | 降低约20℃，即约230~240℃ |
| 保温时间 | 1~2 h |
| 薄壁小型砂芯 | 取温度区间下限 |

[[S21,S16,S20]]

烘干过程需加强炉内通风，保证供氧以促进脱氢聚合反应，同时将挥发物及时排出[[S16]]。

### 5.6 烘干质量判定

烘干合格的渣油砂芯外观应呈**亮黑色或褐色**，表面坚硬、棱角清晰。呈暗黑色说明烘烤过度，呈黄褐色说明烘烤不足，出现严重花斑则说明烘烤过度已发生局部燃烧[[S23,S16]]。

## 6. 性能特征

### 6.1 强度等级

按砂芯比强度（单位粘结剂添加量对应的干拉强度）分类，渣油粘结剂属于**第2组**，比强度区间为**0.3~0.5 MPa/1%**，介于低比强度的沥青类（第3组，<0.3 MPa/1%）和高比强度的树脂及干性油类（第1组，>0.5 MPa/1%）之间[[S9]]。

S61渣油粘结剂工艺试样干拉强度≥1.6 MPa，S76渣油粘结剂工艺试样干拉强度≥2.0 MPa[[S22]]。

### 6.2 优势特性

- **存储稳定性**：混制后的芯砂可存放2~3周仍不失效；预制砂芯在入炉烘烤前可长期放置不变形、不松散[[S23,S18]]。
- **抗吸湿性**：成品砂芯长期放置后无需“回烧”即可直接投入使用[[S18]]。
- **脱模性能**：不粘附芯盒，整班连续操作无需擦拭芯盒[[S18]]。
- **发气特性**：发气量较普通植物油脂砂更低，浇注过程中无特殊刺激性气味[[S21]]。
- **铸件表面质量**：浇注后渣油析出的光亮碳可有效防止粘砂，铸件表面光洁度较好[[S23]]。
- **成本优势**：粘结剂成本仅为呋喃树脂砂的1/5~1/6，远低于合成树脂类粘结剂[[S27,S14]]。

### 6.3 局限性

- **砂芯脆性**：烘干后砂芯质地偏脆，受外力磕碰易损坏断裂，运输破损率高于树脂砂芯[[S21,S23]]。
- **烘透性差**：未添加催干剂的渣油砂烘透性较差，适用烘烤温度窗口较窄[[S21]]。
- **工艺效率**：依赖高温烘烤，烘干能耗远高于无需烘烤的树脂自硬砂工艺，生产效率偏低[[S23]]。
- **精度等级限制**：无法适配对尺寸精度要求较高的Ⅰ级复杂砂芯生产，不适用于自动化大批量流水线铸型场景[[S23]]。
- **烟气排放**：用作湿型砂附加物时，浇注过程释放大量烟气，污染作业环境[[S3]]。
- **砂回用率**：渣油砂目前无规模化高回用率的成熟应用方案[[S14]]。

## 8. 应用场景

渣油粘结剂可替代传统植物油用于制作复杂程度Ⅱ级和Ⅲ级的砂芯，改性产品可扩展至汽车、机械、液压件等各类合金铸件的Ⅰ~Ⅱ级砂芯生产[[S21,S18]]。适用的合金种类涵盖铸铁、铸钢及非铁合金（铝合金、铜合金等），非铁合金铸件用砂芯中渣油加入量可较铁合金场景适当减少[[S16]]。渣油砂适配手工制芯与机器制芯两种方式[[S16]]。

典型应用实例涵盖柴油机缸盖、纺织机械零件、浮筒锚链架等砂芯的制芯场景[[S26]]。

当作为湿型砂附加物使用时，渣油与煤粉配合可增强煤粉的防粘砂效果，提升型砂的流动性、起模性能和型砂韧性，但浇注时产生的大量烟气限制了其在该场景的独立大量使用[[S3]]。
