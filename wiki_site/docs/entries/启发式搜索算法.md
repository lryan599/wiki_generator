## 定义

在压铸模具冷却系统设计语境下，启发式搜索算法是一种面向冷却水道布局自动生成的专用搜索策略。其基本原理是在状态空间搜索中引入估价函数，利用与压铸制造约束绑定的启发信息来指导搜索方向，从而快速从庞大的可能加工方案集中筛选出符合加工性要求的布局方案，避免传统遍历搜索的指数级计算开销[[S20,S3]]。

该场景下的启发式搜索与通用AI领域的启发式搜索存在本质差异：所有启发规则完全绑定压铸制造硬约束，包括水道不得与型腔表面接触、避开模架导柱/顶出机构/镶块等现有零部件、限制钻孔最大深度以降低加工成本，优化目标直接指向冷却均匀性与缩短压铸成型周期，而非通用搜索面向的抽象路径规划问题[[S18,S11]]。

## 基本原理与工作流程

冷却系统启发式搜索采用标准估价函数形式 `f(n) = g(n) + h(n)`，其中 `g(n)` 表示从冷却拓扑起点到当前节点的实际已产生延伸加工代价，`h(n)` 为从当前节点到冷却回路终点的预估剩余加工代价。通过调整 `h(n)` 的权重可动态控制搜索方向，在大规模状态空间下大幅减少无效搜索分支，提升运算效率[[S20,S7]]。

算法部署在冷却回路拓扑连通和最优回路路径搜索完成之后，负责将抽象的冷却中心线自动延伸至模具可加工外边界，生成符合钻削加工约束的实际冷却水道布局[[S3,S16,S18]]。该算法在冷却系统自动生成全流程中的位置如下图所示。

![冷却系统快速编辑与自动完成整体流程框图，图中标注了启发式搜索算法所在的下游环节](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/58e377ea-8ddf-4c61-bb41-552d790dad25/resource)

标准执行流程分为两步[[S4,S16]]：

**Step 1：单向延伸判断与优先选择面标记**

依次检查每段冷却水道，按以下三条约束判定是否仅存在单一有效延伸方向：

1. 水道沿当前方向延伸时与其他非相邻非共线水道发生干涉 → 仅能选择反向延伸；
2. 水道延伸终点为直接接触熔融金属的型面 → 禁止从该方向延伸；
3. 水道延伸时与导柱、推杆等其他模具部件发生干涉 → 该方向不被优先选择。

当水道仅存在单一合法延伸方向时，沿该方向延伸至模具边界面，同时将该边界面标记为“优先选择面”，作为后续其他水道延伸选择的启发指引[[S4,S16]]。

**Step 2：双向延伸启发式选择**

对剩余存在两个可选延伸方向的水道，再次重复 Step 1 检查后，依据以下启发规则确定延伸方向[[S4,S16]]：

- 若两个方向终点均为优先选择面或均不是优先选择面 → 优先选择延伸距离更短的方向以降低钻孔深度、减少加工成本；若两方向延伸距离完全相等，可任选其一，并将新的终点边界面标记为优先选择面；
- 若仅单个方向终点是优先选择面 → 直接选择该方向完成水道延伸。

## 启发信息构造

启发信息的质量直接决定搜索效率和结果的加工性。本研究场景中的核心启发构造方法如下[[S4,S16]]：

| 启发信息类型 | 构造规则 | 工程目标 |
|---|---|---|
| 优先选择面 | 单向延伸水道到达的边界面自动标记；后续水道优先向已标记面延伸 | 减少模具外表面加工开口数量，统一进出水方向 |
| 延伸距离权重 | 双向可选时选择延伸距离较短的方向 | 降低钻削深度，减少刀具损耗，提升水道可加工性 |
| 连通性代价 | 采用总加权公式 `W = Σ wᵢ`，其中 `wᵢ` 为冷却路径中第 `i` 条边的延伸距离权值 | 量化冷却回路整体加工成本，辅助布局方案比较 |

## 状态空间与参数

### 状态空间表示

基于图论将冷却系统表达为连通图结构，每个图节点对应一段可双向延伸的冷却水道，所有节点的延伸方向组合共同构成该冷却回路的完整状态空间[[S6,S16]]。

### 加工方案组合爆炸

对于任意一个完整的冷却回路，其可能的人工枚举加工方案总数计算公式为[[S3,S4]]：

```
N = 2ᵏ · ∏ᵢ₌₁ᵐ (nᵢ + 1)
```

其中：
- `k` — 冷却回路中独立冷却通道数目；
- `m` — 回路中共线冷却通道组的数目；
- `nᵢ` — 第 `i` 组共线通道的通道数目。

该计算结果表明传统穷举式逐一验证所有方案的计算成本会指数级上升，完全不满足工程落地的效率要求，这是引入启发式搜索的根本动因[[S3]]。

## 典型算法变体

除面向水道延伸的专用启发式规则外，冷却系统布局优化领域适配的通用启发式子类包括[[S14,S1,S5,S10]]：

| 算法类型 | 应用方式 | 核心特点 |
|---|---|---|
| 贪心搜索 | 优先选择水道延伸距离最短的方向 | 实现钻孔深度最小化，直接导向加工成本最优 |
| 遗传算法 | 以模具温度分布均匀度为核心优化目标，将水道与零部件干涉约束集成到适应度函数中 | 可将模具工作温差从 60℃ 降低至 30℃ 以内 |
| 模拟退火算法 | 通过可控概率接纳部分劣解避免陷入局部最优 | 适配复杂型腔多约束下的冷却布局全局寻优 |
| Beam Search | 在搜索每一层保留指定数目（beam width 可调）的 Top 候选方案 | 在搜索效率和方案多样性之间取得平衡 |
| 禁忌搜索 | 通过记忆结构避免重复访问已验证的无效布局方案 | 大幅降低无效搜索开销 |

## 优势与局限性

### 核心优势

- 通过预先定义的领域启发信息将搜索范围大幅缩小，无需遍历全部加工方案[[S3,S4]]；
- 自动规避水道干涉问题，直接面向机加工可行性优化钻孔路径[[S3,S4]]；
- 生成的布局钻孔深度更小、加工成本更低[[S4]]；
- 相比传统人工设计的开发效率有显著提升[[S3,S4]]。

### 固有局限性

- 搜索得到的最终结果高度依赖预先设计的启发函数质量[[S3,S16]]；
- 无法保证得到全局最优解，仅能得到局部最优或次优解[[S4,S16]]；
- 极端场景下可能完全无法生成满足全部约束的有效布局方案[[S4,S16]]。

### 冲突解决 Fallback 机制

当启发式搜索无法输出满足要求的冷却回路布局时，系统允许设计人员手动选择水道延伸方向进行干预；若人工干预后仍无法生成合格方案，则返回上一级冷却拓扑生成步骤更换备选拓扑路径后重新发起启发式搜索，直至得到加工性合格的布局结果[[S16]]。
