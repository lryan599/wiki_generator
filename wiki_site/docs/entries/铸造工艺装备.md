## 概述

在压铸语境中，“铸造工艺装备”存在**广义**与**狭义**两种理解。广义上，它指高压压铸全生产流程中涉及的各类**装备与工装的集合体**，包含压铸机、模具及全套自动化周边设备等[[S37,S44]]。狭义上，它沿袭传统铸造专业的界定，专指**铸造生产中工艺相关的模具与装置**，即通常所说的“工艺装备”与“工装”，其核心是压铸模，还涵盖取件夹具、专用量具检具等外围辅具，**不包含压铸机等动力主机设备**[[S10,S33]]。

为防止混淆，需要厘清“铸造设备”、“工装”、“模具”三者之间的关系：通常“铸造设备”指压铸机等动力主机及大型辅机；“工装”是“工艺装备”的行业简称，覆盖范围大于“模具”；“模具”则是“工艺装备”（工装）的核心子集。三者属于**包含关系而非对等关系**[[S10,S32,S80]]。

本文以广义理解为主线，介绍压铸工艺装备的全貌，同时对狭义理解及其包含范围作出说明。

## 定义

**广义定义**（面向高压压铸单元）
  - 根据GB/T 37371-2019《压铸单元 术语》：压铸机是将熔融金属通过高压、高速压射入可重复使用的模具中的**动力机器**；辅助设备是配合压铸生产的**功能设备**，包括浇注装置、取件传送装置、喷涂装置、模温控制装置、清理整修装置、镶件装置、检测装置、系统安全装置、集成控制装置、铸件冷却装置、真空装置等；压铸单元是压铸机与辅助设备形成的**自动化生产成套装备**[[S37,S44]]。

**狭义定义**（面向传统铸造工装设计）
  - 根据《中国模具设计大典 第5卷 铸造工艺装备与压铸模设计》：铸造工艺装备是**造型、制芯及合箱过程中所使用的模具和装置总称**；其中铸造模具（含压铸模）是工艺装备的**核心组成部分**。该定义源于砂型铸造，压铸属于金属型铸造大类，其工装定义在此框架下自然延伸[[S10,S33]]。狭义压铸工艺装备特指**压铸模具、各类专用工装夹具（取件夹具、镶件夹持夹具、下芯夹具）、专用量具与检具、压射冲头/浇口套等配套易耗工装、专用喷涂工装、铸件后处理整形工装**等，**不包含压铸机**[[S10,S88]]。

两种理解下的涵盖范围对比见下表：

| **理解维度** | **是否包含压铸机** | **核心涵盖范围** | **代表文献依据** |
| :--- | :--- | :--- | :--- |
| **广义（压铸单元）** | 是 | 压铸机 + 模具 + 全套周边自动化辅机 + 集成控制系统 | GB/T 37371-2019[[S37,S44]] |
| **狭义（专用工装）** | 否 | 压铸模具 + 取件/镶件夹具 + 量具检具 + 压射耗材工装 + 专用喷涂/整形工装 | 中国模具设计大典 第5卷[[S10,S88]] |

| **后处理装备** | 切边压力机、铸件冷却装置、打磨抛光设备、检测装置（尺寸/内部缺陷检测）、除尘排放装置 | 去除铸件多余浇注系统、溢流槽和飞边毛刺；完成铸件冷却、尺寸与完整性检测；处理生产过程中产生的油雾、粉尘和废弃物 | [[S44,S68,S95]] |

## 核心装备举例

### 注射成形装备——压铸机

压铸机是压铸生产的核心动力设备，其性能直接决定压铸产品的质量、生产效率和运行成本。按压室受热条件不同，分为**冷室压铸机**与**热室压铸机**两大类[[S80]]。

#### 冷室压铸机

**基本结构**：由锁模单元、注射单元、动力系统、控制系统四大核心部分组成，分为卧式、立式和全立式三种类型。其中**卧式机型**的压射室与开模方向平行，为行业主流机型，可支持最大45000 kN的锁模力，适配铝、镁、铜等熔点较高的有色合金铸件生产。冷室压铸机的压射锤头采用中空水冷结构以延长使用寿命[[S42,S50]]。

**工作原理**：压射室与保温炉相互独立。生产时，将定量金属液注入冷态压射室（shot chamber），由液压驱动的柱塞推动金属液高速高压充入模腔，保压至铸件完全凝固后开模顶出。此结构可有效减少高熔点合金对压射系统的接触腐蚀时间[[S54,S14,S24]]。

#### 热室压铸机

**基本结构**：由机架、开合模机构、带鹅颈结构的压射组件、锁嘴机构、内置合金液的保温坩埚、动力与控制系统组成。锁模部分相对水平面约有15°倾角，压射组件整体浸没在坩埚内的合金液中[[S38,S20,S96,S11,S64]]。

**工作原理**：工作时，压射冲头上行，合金液自动进入压室；随后冲头下行，将合金液沿鹅颈通道、喷嘴压入模具型腔；保压至铸件凝固后冲头上行，多余合金液回流入压室，完成一次压铸循环。常规压射压力为7~14 MPa，生产节拍最快可达15次/分钟，主要适配锌、铅、锡等低熔点合金及小型镁合金铸件生产[[S54,S14,S11,S64]]。

### 模具及温控装备——压铸模

压铸模具是高压压铸生产的**核心工艺装备**，属于工艺装备体系中最关键的组成部分，一般由**动模**和**定模**两大主体组成。定模固定在压铸机定模板上，动模随动模板完成开合动作。完整的压铸模由**成形工作零件、浇注系统、溢流与排气系统、抽芯机构、推出复位机构、导向机构、模架支撑零件、温度调节系统**共8类功能结构组成[[S23,S72,S59,S32]]。

**工作原理**：压铸模与压铸机构成联动整体。合模时动、定模闭合构成完整型腔、浇注流道和溢流排气通道；高压金属液在压铸机压射作用下充填满型腔，在压力场下快速冷却凝固成形；开模时动、定模分离，压铸机上的顶出机构联动模具推出系统将铸件从动模侧推出，完成生产循环[[S55,S93]]。

### 熔炼与保温装备——定量保温炉

**基本结构**：由炉体、熔池、密封盖、升液管四大核心部分组成，主流采用电阻加热结构。其工艺功能是将待浇注的合金液稳定控制在工艺要求的温度区间内。部分机型内置定量输送机构，可实现无需人工干预的精准给液，适配不同吨位的冷室压铸机[[S48,S29,S43]]。

**工艺作用**：作为冷室压铸机的配套周边设备，定量保温炉承接前端熔炼炉输送来的合格合金液，在压铸生产全周期内维持合金液温度稳定。带密封结构的炉型可通入保护性气体，避免镁、铝等合金液氧化燃烧，保障后续给汤工序中金属液质量的一致性[[S28]]。

---

### 熔体输送装备——自动给汤机

**核心结构**：由耐高温舀料机构、蜗轮蜗杆传动系统、行走定位组件、电气控制模块组成。常规机型的舀料轴采用无油润滑结构以适配高温工况[[S15,S82]]。

**工艺作用**：自动给汤机是冷室压铸自动化生产单元的核心配套设备，替代人工完成合金液的定量转运和浇注工序。它能保障每次浇注的金属液重量、浇注轨迹和浇注时间一致性，消除人工操作精度偏差，大幅提升生产节拍稳定性，同时降低高温金属液操作的安全风险[[S27,S77,S35]]。

---

### 喷涂脱模装备——压铸自动喷涂系统

**基本结构**：分为固定式和往复式两大类型，由独立排布的多组吹气铜管、多组喷涂铜管、升降/行走驱动机构、自动稀释供液单元、电气控制模块组成。铜管可根据型腔轮廓任意弯折调整喷射角度，适配不同复杂程度的压铸模[[S4]]。

**工艺作用**：压铸自动喷涂系统在每次开模取件后同步完成多道工序：首先吹出压缩空气清理型腔内残留的铝屑和废渣，随后向动模和定模型腔表面均匀喷涂脱模剂，最后再次吹气吹干多余残留液。同时可按不同型腔位置单独调整喷涂量，保障脱模效果的一致性，辅助精准控制模具工作温度区间，提升铸件表面质量[[S63]]。

---

### 取件自动化装备——压铸取件机器人

**基本结构**：通常采用多轴（典型为6轴）工业机器人结构，末端配备三爪气动夹持器。部分机型可集成吹气、喷涂辅助功能，由电控柜实现全流程示教编程控制，可完成点到点或连续轨迹定制运动[[S94,S47,S76]]。

---

### 后处理装备——压铸切边机

**基本结构**：分为机械型和液压型两大类型，核心配套定制化切边模。模腔轮廓匹配铸件外形，利用上下模对合的剪切力将铸件的浇注系统余料、分型面飞边、溢流槽余料一次性切除，适配不同吨位的压铸产品批量切边需求[[S84,S66]]。

**工艺作用**：压铸切边机作为压铸后清理工序的核心设备，替代手工切割或锯切，完成大批次铸件的飞边和浇注余料清理。机械型切边机动作快、生产效率高，适配中小批量常规铸件；液压型切边机工作台、行程、压力参数可自由调整，适配大尺寸复杂铸件。切边一致性远高于手工操作，可大幅降低后道精加工工序的工作量[[S12,S66,S73,S2]]。

## 关键参数与选型依据

### 压铸机锁模力

锁模力是选用压铸机时**首要确定的参数**，其作用是为了克服压铸时金属液在分型面上的反压力（胀型力），以锁紧模具分型面，防止金属液飞溅并保障铸件尺寸精度。

常规核算公式（安全系数K取值为1.25）：
$$F_{\text{锁}} \ge K \times (F_{\text{主}} + F_{\text{分}})$$
式中，$F_{\text{主}}$ 为分型面总投影面积产生的主胀型力，$F_{\text{分}}$ 为侧向抽芯机构产生的分胀型力[[S92,S3]]。

国家标准（原GB/T 10925 / JB/T 8083）规定的主要序列为：热室压铸机 630 kN~6300 kN；立式冷室压铸机 630 kN~6300 kN；卧式冷室压铸机 630 kN~25000 kN，并可按需定制31500 kN、40000 kN超大锁模力机型[[S49,S81,S5]]。

### 射料量选型

常规射料量选型要求：铸件质量与溢流排溢系统总质量之和，需控制在设备最大射料量的 **30%–70%** 范围内，以保证充型稳定性和铸件成品率[[S8]]。

### 选型导则

- **合金类型优先匹配**：锌合金等低熔点合金推荐选用热室压铸机；铝合金、铜合金及大型镁合金铸件推荐选用冷室压铸机；带镶嵌件、要求组织致密的回转体零件推荐选用全立式冷室压铸机[[S40]]。
- **传统选型法**：以锁模力核算为基础，被广泛采用[[S40]]。
- **压射能量选型法（$P-Q^2$ 图法）**：将压铸机等效为液态金属泵，对比设备可提供的压射能量与铸件成形所需的压射能量，提升匹配精准度[[S40]]。

## 装备状态与铸件质量的关联

压铸工艺装备的运行状态直接影响铸件质量，装备磨损、温控漂移或运动异常会对应引发典型铸造缺陷。

### 模具磨损与缺陷

- **导柱、导套长期磨损** → 动定模合模对位偏移 → **错边（错缝）** 缺陷，需更换导柱、导套进行修复[[S52,S87]]。
- **分型面、滑动配合滑块磨损** → 增大合模间隙，分型面处金属液溢出 → **飞边**缺陷；模具镶块磨损、闭锁元件失效也会直接催生飞边[[S52]]。
- **型腔表面热疲劳龟裂、局部掉块、冲蚀磨损** → 在对应铸件位置形成不规则凸起的 **多肉（带肉）缺陷**，需对损伤部位焊补修模后抛光处理[[S52,S60,S71]]。
- **型腔表面光洁度持续下降** → 粘模倾向增加 → 铸件表面 **拉伤、网状毛刺**；磨损累积还会逐步改变型腔原始尺寸，导致铸件**批量尺寸超差**[[S75,S71]]。模具磨损（压铸次数增加）对锌合金、铝镁合金及铜合金压铸件表面光洁度等级的影响具有规律性的下降趋势[[S78]]。

### 保温炉温控漂移与缺陷

- **浇注温度大幅低于设定值** → 金属液充型流动性下降 → 不同位置的金属流汇交时无法完全熔合 → **冷隔、欠铸**类缺陷[[S91,S74]]。
- **浇注温度过高** → 增大金属液溶解气体含量，充型后析出气体无法完全排出 → **气孔**，严重时与收缩孔洞合并形成**气缩孔**[[S1]]。
- **浇注温度偏差破坏顺序凝固条件** → 缩短补缩通道存续时间 → 厚大热节部位补缩不足 → **缩孔、缩松**缺陷[[S39,S86]]。模具温度变化对气缩孔发生比例存在直接影响[[S21]]。

### 喷涂不均与缺陷

- **局部喷涂量过大** → 残留未挥发脱模剂与水汽无法在合模前排出，充型时卷入金属液 → 对应位置出现**气孔、鼓泡**类缺陷[[S62,S70]]。
- **局部漏喷** → 该区域金属液直接接触高温裸露模壁 → 极易发生粘模 → 取出时形成**表面拉伤**；同时该部位模温骤降，金属液充型流动前沿快速凝固 → 易形成**冷隔**[[S69,S70]]。

### 取件系统故障与缺陷

- **取件机器人运动轨迹偏移** → 对尚未完全冷却定型的铸件施加偏斜方向的额外外力 → 脱模时受力不均 → **整体扭曲翘曲变形**[[S52,S87]]。
- **抓取用力过猛、位置偏差** → 铸件局部受外力撕裂 → **裂纹、机械损伤**缺陷[[S52]]。

### 切边机故障与缺陷

- **切边机合模错位** → 切边刃口与铸件本体对位偏移，无法沿轮廓边缘整齐切除浇道与飞边 → 铸件边缘**残留多余不规则飞边**，同时造成铸件本体局部**受挤压错位 → 错边、变形**缺陷[[S52,S25]]。
