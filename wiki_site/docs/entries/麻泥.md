## 概述

“麻泥”在铸造领域具有明确的标准化定义，但需严格区分其在不同铸造分支中的涵义与使用语境。

在**砂型铸造**标准体系中，“麻泥”为正式收录的术语。在**压铸**领域，现行所有国家标准（包括GB/T 29092-2012《镁及镁合金压铸缺陷术语》）及NADCA公开技术资料中，均未将“麻泥”收录为压铸缺陷的正式条目[[S1,S5,S6,S10]]。然而，压铸一线生产经验中存在将一类表面颗粒状粗糙缺陷俗称为“麻泥”的非标准表述[[S8]]。本词条同时涵盖两个领域的内涵，并侧重辨析两者差异。

## 砂型铸造中的“麻泥”

### 定义与材质

依据GB/T 5611-2017《铸造术语》，“麻泥”亦称“烂泥砂”，对应英文术语 **loam**[[S3,S5,S10]]。其定义为：由天然粘土砂、砂粉与粘土加水混合调制而成的泥状造型面料[[S3]]。该材料具有良好可塑性，用于手工砌砖刮板造型工艺。

### 工艺用途

麻泥主要用于下列砂型铸造场景[[S3]]：
- **麻泥型（loam mold）/烂泥砂型**：以刮板与砌砖方式逐层构筑的大型铸型。
- **麻泥铸造（loam casting）/砌砖铸造**：采用麻泥造型的铸造方法。
- **麻泥车板**：刮制麻泥型时使用的旋转刮板工具。
- **麻泥捏和机**：用于调制麻泥的专用搅拌设备。

## 压铸行业俗称“麻泥”的表面缺陷

### 术语说明

该叫法属于**非标准**的生产经验用语，并非GB/T 5611或GB/T 29092等标准收录的压铸缺陷术语。建议在正式技术文件或质量报告中优先使用标准缺陷名称，但有必要在技术交流中对该俗称予以理解与辨析。

### 外观特征

此类缺陷表现为铸件表面出现颗粒状粗糙分布，形似**泥点状附着物**，呈不规则麻点形态[[S8]]。缺陷可能散布于铸件局部表面，严重时可大面积存在，使铸件表面质量显著下降。

### 形成机理

俗称“麻泥”的表面缺陷主要有以下成因[[S8,S11,S9]]：

1. **脱模剂/涂料过量**：型腔工作面上多余的涂料（石墨类）被高速金属液冲刷脱落，混入金属表层形成石墨夹杂物，呈现颗粒状粗糙分布。此为该类缺陷最典型的成因[[S8]]。
2. **涂料燃烧残留**：过量脱模剂局部堆积后，涂料燃烧分解产生的残留物质附着在铸件表层，与金属基体结合形成不规则麻点状粗糙层[[S11]]。
3. **氧化夹渣卷入**：合金液在充型过程中卷裹表层氧化夹渣，混入铸件表面区域，形成泥状分散的颗粒缺陷[[S9]]。

### 与其他标准缺陷的鉴别

“麻泥”类缺陷需与以下标准术语收录的缺陷加以区分（表1）：

| 对比缺陷 | 标准术语 | 关键特征 | 与“麻泥”类缺陷的差异 |
|---------|---------|---------|-------------------|
| 麻点 | 麻点（熔模铸造） | 分散内凹圆点状凹坑，源于型壳钠盐高温挥发氧化[[S2]] | 成因属熔模铸造型壳反应，非压铸涂料残留；呈独立凹坑而非大面积颗粒附着 |
| 麻面/皱皮 | 皱皮（elephant skin） | GB/T 5611标准术语，不规则粗粒状或褶皱表皮，带深网状沟槽[[S4,S10]] | 属表皮形貌缺陷，无涂料残留夹杂物的泥状属性 |
| 针孔 | 气孔（gas porosity） | 表层及内部分散显微孔洞，属气相析出类缺陷[[S6]] | 无表层夹渣颗粒特征 |
| 疏松 | 疏松（microshrinkage） | 晶界枝晶间微孔群，多位于厚壁区域内部[[S6]] | 不单独以表面层泥状颗粒形式存在 |
| 砂眼 | — | 铸件内部或表面嵌入砂粒的孔洞类缺陷 | 仅存在于砂型铸造场景，压铸场景无砂型砂粒来源 |

### 影响因素

该类压铸表面粗糙（俗称“麻泥”）缺陷的产生概率受多类工艺条件影响（表2）[[S7,S8]]：

| 影响因素类别 | 具体条件 | 影响趋势 |
|------------|---------|---------|
| 工艺参数 | 浇注温度过高或过低 | 增加缺陷风险 |
| 工艺参数 | 高速充型速度不合理 | 增加缺陷风险 |
| 工艺参数 | 压射增压压力不足 | 增加缺陷风险 |
| 模具条件 | 模具温度未控制在150~240 ℃区间 | 增加缺陷风险 |
| 模具条件 | 型腔表面粗糙度过高 | 增加缺陷风险 |
| 模具条件 | 热喷涂/离型涂层残留积碳 | 增加缺陷风险 |
| 材料条件 | 合金液除气不充分 | 增加缺陷风险 |
| 材料条件 | 杂质含量/氧化夹渣占比过高 | 增加缺陷风险 |

### 预防与控制措施

针对该类俗称“麻泥”的压铸表面缺陷，可落地防控措施如下表所列（表3）[[S8,S11,S7]]：

| 措施类别 | 具体方法 |
|---------|---------|
| 工艺调整 | 严格控制脱模剂喷涂用量，每次涂刷后用压缩空气吹散，确保型腔涂料分布均匀[[S8]] |
| 工艺调整 | 优化压射参数：浇注温度、充型速度、压射比压调整至对应合金的合理区间[[S7,S8]] |
| 模具维护 | 定期清理型腔表面积碳残留[[S7]] |
| 模具维护 | 通过油磨处理降低型腔表面粗糙度[[S7]] |
| 模具维护 | 维持模具温度稳定在150~240 ℃的热平衡状态[[S7]] |
| 材料优化 | 合金液充分除气精炼处理，控制杂质和氧化夹渣含量处于较低水平[[S11,S7]] |

## 注意事项

- 在砂型铸造领域，“麻泥”为标准术语，指造型材料而非缺陷。
- 在压铸技术文件或检测报告中，不应使用“麻泥”作为缺陷判定名称，建议参照GB/T 13820或企业内部缺陷图谱使用对应的标准缺陷术语进行判定。
