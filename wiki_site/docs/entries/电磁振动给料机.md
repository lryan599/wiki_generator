## 概述

电磁振动给料机（electromagnetic vibrating feeder，EVF）是一种采用电磁振动器作为激振源，驱动给料机槽体产生振动，使物料在槽体上以抛掷运动方式运行的给料设备[[S29,S21,S17]]。该设备在国家标准GB/T 25370-2020《铸造机械 术语》中作为铸造机械专用给料设备正式收录，条目编号为10.1.4[[S29,S17]]。

电磁振动给料机利用接近共振频率的电磁铁吸力实现激磁，具备几乎无惯性启停的特性，可通过调压方式无级调节输送量，并支持隔振悬挂安装[[S5]]。设备无转动部件，无需日常润滑，结构简单，工作可靠，维护成本低[[S32,S20]]。

## 分类

电磁振动给料机可按多个维度进行分类[[S5,S14,S19,S23]]：

| 分类维度 | 类型 | 说明 |
|---|---|---|
| 安装形式 | 吊挂式、座式 | 吊挂式为主要安装形式 |
| 料槽结构 | 直槽直线型、螺旋圆盘振动料斗型 | 直槽型用于直线输送，螺旋料斗型用于定向整理上料 |
| 激振电源类型 | 不经整流（100Hz）、半波整流（50Hz） | 不经整流型振幅小、噪声大，适配小型轻量工件；半波整流型振幅大，适配大吨位料斗 |
| 卸料方向 | 直卸式、侧卸式 | — |
| 国内主流系列 | DZL系列、GZ系列、Y48系列等 | DZL₁配A/B两种料槽，DZL₂配A/B/C/D四种料槽，A型为标准无盖槽，B/C/D型备有槽盖 |

## 工作原理

### 电磁激振原理

电磁振动给料机由接近共振频率的电磁铁吸力激磁。220V或380V交流电经对应整流方式输出脉冲交变电流，驱动电磁铁周期性产生交变磁场，吸引衔铁往复运动[[S5,S19]]。激振频率根据应用场景适配：中小型零件给料场景可选50Hz～100Hz；直径≥500mm的大规格料斗可选16Hz～25Hz[[S30]]。

**电磁铁气隙调整要求**[[S1]]：
- 电磁铁垂直安放时，气隙 δ₀ = 0.8S（S为衔铁行程，cm）
- 电磁铁切向安放时，气隙 δ₀ = 0.7S
- 气隙过小易引发撞击和运行噪声；气隙过大会导致激振力下降、功耗升高

### 近共振工作状态

电磁振动给料机运行在系统工作频率接近固有频率的近共振工作区间，调谐质Z（工作频率与参振体自振频率之比）通常处于以下区间[[S4]]：
- 低临界亚共振态：0.75 < Z < 0.95
- 近共振态：1.05 < Z < 1.2

设计要求设备自振频率为电磁铁激振频率的1.05～1.1倍，以保障运行稳定性[[S30]]。近共振工作状态使设备具备几乎无惯性启停的特性，可无惯性连续调整生产率，适配压铸车间、铸造车间的定量分配作业场景[[S5]]。

### 弹性系统与物料运动

电磁振动给料机的弹性元件为倾斜布置的弹簧片或吊簧，带动料斗同时产生上下振动和扭转振动，合成运动为螺旋向上的物料输送轨迹[[S19,S20]]。物料在槽体上的运动分为滑动、跳跃两类，可通过振动方向角、物料与轨道间静摩擦因数、振幅、振动频率的组合条件判定运行状态[[S19,S26]]。典型直径300mm的振动料斗上料速度可达4～10m/min[[S19]]。

## 结构与组成

电磁振动给料机典型结构无转动部件，核心部件包含以下五大组成部分[[S32,S20]]：

| 部件 | 功能说明 |
|---|---|
| 电磁振动器（激振器） | 提供激振动力源，由电磁铁与衔铁组成 |
| 振动给料槽/料斗 | 承载并输送物料，可为直槽或螺旋料斗 |
| 弹性元件（吊簧/弹簧片） | 倾斜布置，支撑料槽并产生复合振动 |
| 减震元件（橡胶弹簧/减震器） | 隔离振动，减少对支撑结构的影响 |
| 控制单元 | 通过自耦变压器、电位计、硅整流器或可控硅调压实现无级调节 |

## 关键工艺参数与计算

### 生产率计算

电磁振动给料机的生产率可用下式计算[[S32,S13]]：

> Q = 60 · B · h · S · v · K （t/h）

式中：
- B —— 槽体宽度（m）
- h —— 槽中物料厚度（m）
- S —— 输送速度（m/s）
- v —— 物料容积比重（t/m³）
- K —— 料槽断面物料充满系数，一般取 0.6～0.7

### 运行参数

设备的振幅和振角可通过粘贴振幅指示牌和振角指示牌在被测设备侧壁进行现场快速测试，无需专用仪器[[S4,S33]]。

## 在压铸与铸造工艺中的角色与应用

电磁振动给料机在压铸及有色合金铸造生产线中有以下典型应用场景[[S6,S10,S3,S24,S15,S22]]：

1. **熔炼定量配料**：为熔炼定量炉系统定量输送焦炭、石灰石等熔炼辅料，配合定量斗完成配料
2. **废料回收输送**：为LOTUSS铝薄料回收系统提供连续可控的废铝屑、薄料输送，稳态作业条件下配套铝回收效率最高可达98%
3. **浇冒口与回炉料输送**：压铸后处理环节输送破碎后的浇冒口、回收废铸件完成返炉配料
4. **合金粒料定量添加**：输送干合金粒料、颗粒状孕育剂/变质剂完成定量精准添加
5. **压铸零件定向送料**：对小型精密压铸成品零件实现定向有序送料，输送过程无剧烈搅动、撞击，不会损伤已精加工的薄壁脆性压铸零件表面[[S16,S19]]

## 选型与安装要点

### 选型约束

电磁振动给料机具有明确的适用边界[[S5,S19]]：
- **适配物料**：干砂、颗粒状松散物料，可输送物料粒度覆盖从粉状到400～500mm块料[[S32]]
- **不适用物料**：粘滞性湿粉状物料、带油污水渍的极轻薄片类零件
- **输送距离**：不适宜长距离输送
- **电源敏感性**：电源电压波动会直接影响输送生产率

### 安装要求

| 安装要点 | 具体要求 |
|---|---|
| 进料端连接 | 需配置软性连接，禁止料仓内物料直接重压在给料机料槽上[[S5,S1]] |
| 隔振基础 | 静止基座质量需达设备可动部分总质量的5～10倍以上[[S1,S5]] |
| 减震效果 | 配套橡胶弹簧/弹簧隔振脚可将传递至支撑结构的动载荷降低至原值的1/20以下[[S30]] |
| 输送轨道倾角 | 倾斜角设置值不应大于5°，保障物料稳定爬升[[S16]] |
| 防尘防潮 | DZL系列B/C/D型备有封闭槽盖，适配压铸车间高防尘密封需求[[S5]] |

## 常见缺陷与故障

### 典型故障及排查

| 故障现象 | 可能原因与排查方向 |
|---|---|
| 振动不稳 | 参振体固定联接元件松动、焊接部位开焊[[S4]] |
| 振幅衰减 | 出厂调试完成的匹配振动参数被破坏（擅自旋动刚度调节螺母、增减代用弹簧、增减附设装置、加长缩短槽体）[[S11]] |
| 弹性件断裂 | 金属弹簧表面裂纹、中心线弯曲超限、残余变形超标；橡胶弹簧接合面开胶、撕裂、老化开裂[[S11]] |
| 物料堵塞 | 输送粘滞性湿粉状物料、料仓物料过重压迫料槽[[S5]] |
| 线圈过热/噪声过大 | 电磁铁气隙调整不当（过小引发撞击噪声，过大降低激振力增大功耗）[[S1]] |
| 输送生产率波动 | 电源电压波动[[S5]] |

### 弹性件报废判定标准[[S11]]

- 金属弹簧：出现表面裂纹、中心线弯曲大于长度的1%、残余变形大于最大变形量的5%
- 橡胶弹簧：出现接合面开胶、橡胶本体撕裂、老化开裂

### 寿命与噪声要求

在正常使用条件下，首次大修期工作时间不少于6000小时；空运转等效连续A声级噪声不超过85dB(A)[[S31,S9]]。激振器连接板与活化圈上平面垂直度需符合GB1184公差等级10级精度，激振器接触面用0.04mm塞尺塞入深度不能超过剖分面外缘周长的10%[[S31]]。
