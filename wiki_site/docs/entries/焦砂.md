## 概述

焦砂（别名枯砂）是一种典型的铸造表面缺陷。该缺陷表现为黑褐色烧结砂粒牢固粘附于铸件表面，难以通过普通喷砂清理去除，常需借助打磨等机械手段才能清除[[S3,S14,S10]]。在工程实践中，如叶轮等内部流道类铸件的技术文件中，明确要求交付前必须完全去除残留焦砂及毛刺，否则视为不合格[[S3,S14]]。

焦砂对应的标准英文术语为 **sand burn-on**，在国际通用铸造缺陷分类体系中被编码为 **D221 Burn-on** 类缺陷[[S10]]。

## 缺陷分类与辨析

焦砂是铸件表面粘砂类缺陷的一种，其形成机制与外观特征均有别于其他相关缺陷。

| 缺陷类型 | 核心定义 | 形成机理关键点 | 与焦砂的核心区别 |
| :--- | :--- | :--- | :--- |
| **焦砂 (Sand burn-on)** | 砂粒自身烧结固结后牢固附着于铸件表面 | 砂粒在持续高温下发生固相/液相烧结[[S10]] | （本词条） |
| **砂孔/砂眼 (Sand inclusion)** | 铸件内部或表面充塞型砂的孔洞 | 型砂脱落并被金属液包裹[[S4]] | 砂粒松散，未发生烧结 |
| **机械粘砂 (Metal penetration)** | 金属与砂的机械混合物渗入铸件表面 | 金属液单纯渗入砂粒缝隙[[S10,S12]] | 金属渗透为主，无砂粒自主烧结 |
| **化学粘砂 (Burn-on)** | 低熔点化合物牢固粘附在铸件表面 | 金属氧化物与砂中杂质反应生成低熔点化合物[[S12]] | 化学反应产物粘附，非砂粒烧结 |

## 形成机理

焦砂的形成是一个涉及高温物理与化学变化的复杂过程，其机理可分为以下几个阶段[[S1,S11]]：

1.  **粘结剂烧蚀：** 高温金属液充型接触砂芯后，砂芯表层的有机粘结剂（如树脂）首先受热分解、烧蚀，导致砂粒间原始的粘结结构被破坏，间隙增大。
2.  **砂粒烧结：** 砂芯表层长时间处于超过其材料烧结阈值的高温环境中。砂粒之间通过固相扩散，甚至因低熔点相的出现而发生液相流动，导致砂粒互相粘结、固结，形成致密化烧结层。烧结的程度和速度直接影响缺陷的形成[[S11]]。
3.  **氧化物渗透强化：** 金属液在高温下氧化生成的低熔点氧化物会渗入已初步烧结的砂粒缝隙中，进一步强化烧结层与铸件基体的结合，最终形成难以清除的深色焦砂层。

## 主要影响因素与预防措施

焦砂的形成受多种工艺因素综合影响，下表汇总了核心影响因素及其对应的预防措施[[S10,S9,S2]]。

| 影响因素 | 具体说明 | 预防与解决措施 |
| :--- | :--- | :--- |
| **金属液温度** | 浇注温度过高，易超出砂粒烧结温度区间，直接诱发烧结。 | 在满足充型要求的前提下，控制浇注温度在材料允许的最低过热度区间。 |
| **砂芯材料** | 普通硅砂热膨胀量大、耐火度低，抗烧结性能差。 | 选用耐火度高、热膨胀量小的型砂，如铬铁矿砂、锆砂、烧结陶瓷砂、宝珠砂等。 |
| **砂芯紧实度** | 紧实度不足会扩大砂粒间隙，加速金属液渗透和后续烧结。 | 保证制芯过程中砂芯整体具有均匀且足够高的紧实度。 |
| **烘烤工艺** | 烘干不足或不当，可能导致砂芯局部过烧，使其强度降低，抗渗透能力下降。 | 严格管控烘芯温度与保温时间，避免局部热失效。对于重要砂芯，可采取二次烘干工艺。 |
| **涂料防护** | 砂芯表面涂料厚度不足或耐火度不够，无法有效隔离金属液与砂芯。 | 采用高耐火度涂料，封闭砂芯表面孔隙。在厚大热节等关键位置，可进行局部二次浸涂以增强防护。 |
| **金属液特性** | 高流动性合金（如高磷合金）更易渗入砂粒间隙，加剧缺陷。 | 将残留磷含量维持在最低必要水平，避免反应性致密度问题。 |

## 典型发生位置

焦砂缺陷并非均匀分布，而是高发于热负荷集中且冷却缓慢的区域，主要包括[[S10,S13]]：
*   铸件厚大热节处
*   砂芯尖角部位
*   内浇口附近
*   薄壁且受热集中的砂芯区域

在上述位置，砂芯长时间处于远超其他区域的超高温服役状态，砂粒发生烧结的概率大幅提升。

## 对铸件质量的影响

焦砂的存在对铸件质量有多方面的负面影响[[S3,S10]]：
*   **表面质量与精度：** 大幅提升铸件表面粗糙度，破坏名义尺寸精度。
*   **机械加工：** 烧结砂层硬度高，加工时加剧刀具磨损。若加工余量不足，易导致“黑皮”残留，造成废品。
*   **功能性：** 严重时会堵塞铸件内部流道，直接影响液压件、水冷腔等承压结构的气密性，导致泄漏。

## 相关标准

在国内外铸造相关标准中，焦砂被明确列为一种应予以控制和清除的缺陷[[S15,S12,S8]]：
*   **ASTM 标准：** ASTM相关铸件质量标准明确规定，铸件表面不得有焦砂（burnt-on sand），且表面应相当光滑。
*   **国内标准：** 《中国机械工业标准汇编 铸造卷》正式收录了“枯砂[焦砂]”这一规范术语。相关铸造缺陷分类标准中，将其描述为牢固粘附在铸件部分或整个表面上的烧结砂层。
