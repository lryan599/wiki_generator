## 概述

脱蜡釜（标准术语 dewaxing autoclave，亦称作 dewaxing kettle、wax autoclave、wax melting-out furnace、wax removal boiler、steam dewaxing autoclave、dewaxing oven）是熔模精密铸造工序中用于快速脱除模料的高压特种铸造设备。在型壳制备完成后、浇注前完成蜡模的脱除任务，为后续型壳焙烧和金属浇注做准备[[S9]]。设备结构复杂，包含承压釜体、蒸汽供给系统、蜡液收集回收系统和控制模块等，属熔模铸造专用核心装备[[S9]]。

脱蜡釜属于熔模铸造的特定工序装备，不属于常规压铸工艺链的标配设备。仅当压铸行业采用可熔蜡芯制备复杂内腔压铸件的特种辅助工艺时，脱蜡釜才作为蜡芯脱除设备被使用 [[S16,S19]] 。因此，本词条定位为跨熔模铸造/特种带蜡芯压铸的共享辅助工艺设备[[S19]]。

## 分类

脱蜡釜按加热方式与工作介质可分为以下类型[[S17]]：

- **燃油加热蒸汽脱蜡釜**：利用燃油锅炉产生高压蒸汽，工作压力通常可达 0.7~1.0 MPa，工作温度 168~180°C[[S2]]。
- **电加热蒸汽脱蜡釜**：利用电热元件产生蒸汽，工作压力与温度范围同上[[S2]]。
- **真空脱蜡釜**：在负压条件下进行脱蜡作业，适用于特定型壳体系。
- **热风脱蜡炉**：以热空气作为加热介质。

其中蒸汽脱蜡釜是当前工业生产中应用最广的类型[[S17]]。燃油型与电加热型蒸汽脱蜡釜是当前工业使用主流，在国内有多家制造厂商提供相应设备[[S2]]。

## 结构与组成

脱蜡釜由多个功能模块构成，核心部分包括 [[S3,S11,S1]]：

| 部件 | 功能 |
|---|---|
| 承压釜体 | 提供密闭高压工作空间，承受高温高压蒸汽循环 |
| 快速升压蒸汽供给系统 | 在数秒内将釜内压力升至设定值，保证型壳不胀裂 |
| 密封舱门 | 带联锁安全装置的门体闭锁机构，防止带压误开 |
| 安全泄压装置 | 防止超压事故，含自动排气孔与调节阀 |
| 蜡液过滤收集回收系统 | 配置细缝滤板以过滤固体杂质并减少喷溅，回收蜡液 |
| 自动控制系统 | 压力/温度监测与程序控制模块 |
| 装载车与转台 | 转运型壳进入釜体，部分配置旋转式盛蜡装置 |

釜门闭锁机构与蒸汽阀采用安全联锁设计，操作不当将被程序拒绝执行，避免发生安全事故[[S3,S11]]。

## 工作原理

脱蜡作业开始后，高温高压蒸汽被快速通入密封釜体内，在极短时间内（通常为若干秒内）达到设定压力。蒸汽的热量通过型壳传递至蜡模表层，使蜡模的表层快速熔化并流出，在蜡模整体热膨胀对型壳产生胀裂力之前形成蜡模与型壳之间的间隙[[S9,S18,S10,S13]]。

这一快速升压的过程是脱蜡工艺的技术核心。蜡模整体的热膨胀系数远大于陶瓷型壳的膨胀系数，若升压过慢，蜡模整体受热膨胀而仍未熔化，将产生足以胀裂型壳的巨大应力，最终导致型壳开裂[[S9,S13,S18]]。

脱蜡完成后，釜体缓慢释压以避免型壳内外压差过大，再次保证型壳完整性[[S10,S1]]。

## 工艺参数

硅溶胶型壳是高温合金精密铸件的主流型壳体系，其蒸汽脱蜡的主要工艺参数如下：

| 工艺参数 | 范围或要求 | 来源 |
|---|---|---|
| 蒸汽压力 | 0.6~0.75 MPa（硅溶胶型壳） | [[S9]] |
| 升压至0.6 MPa 时间 | < 14 s | [[S9]] |
| 蒸汽温度 | 约 150~160°C | [[S18]] |
| 总脱蜡作业时长 | 6~10 min | [[S9]] |
| 蜡料回收率 | 可达 95% | [[S7]] |

石膏型脱蜡的参数有所不同：

| 工艺参数 | 范围或要求 | 来源 |

| 脱蜡时长 | 1~2 h | [[S5]] |
| 脱蜡后空气中静置时间 | ≥ 24 h | [[S5]] |

燃油/电蒸汽脱蜡釜的核心工作参数可在以下范围运行[[S2]]：

| 参数 | 范围 |
|---|---|
| 工作压力 | 0.7~1.0 MPa |

## 常见问题与缺陷

脱蜡过程中的典型问题包括[[S10]]：

1.  **升压速度过慢**：蜡模膨胀导致型壳胀裂，浇注后产生毛刺缺陷。
2.  **释压速度过快**：型壳内外压差过大，导致型壳破损。
3.  **蒸汽湿度过高/釜内积水过多**：开盖时出现蜡盘喷溅、冒泡、冒沫现象[[S1]]。
4.  **设备安全风险**：脱蜡釜属高压特种设备，门体联锁失效或违规操作等可引发爆炸或人身伤害事故[[S1]]。

蜡液回收中的问题亦受重视。过滤系统的改进（如安装细缝滤板）可大幅减轻喷溅及固体杂质对回收蜡品质的污染[[S1]]。

## 应用场景

脱蜡釜的主流应用是在熔模铸造领域中，为硅溶胶、硅酸乙酯等陶瓷型壳的精密铸件生产完成脱蜡工序。它替代了传统热水脱蜡法，显著缩短了脱蜡时间，降低型壳开裂率，并实现蜡料的高效回收利用[[S14,S18]]。

在压铸领域内，常规流程中不使用脱蜡釜；仅在少数需要可熔蜡芯形成复杂内腔的特种压铸工艺中被作为辅助蜡芯脱除设备，该用法较为特殊并非常规[[S19]]。

## 相关标准与规范

目前尚无脱蜡釜独立专属的国家标准或国际标准。现行 **GB/T 25370-2020《铸造机械 术语》**中收录了脱蜡、熔模铸造等相关设备术语，但未给出脱蜡釜详细规格[[S6]]。脱蜡釜的压力容器本体需满足 **GB 150 系列**（压力容器安全规范）的要求[[S6]]。
