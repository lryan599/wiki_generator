## 概述

气冲造型（air impact molding）是利用燃气或压缩空气瞬间膨胀所产生的压力波紧实型砂的造型方法，GB/T 5611-2017《铸造术语》与国际标准同步收录了该术语定义[[S16,S38]]。气冲造型属于砂型铸造领域的独立砂型紧实工序，是大批量砂型流水生产线的核心造型工艺之一，不属于模具制造、制芯、脱模或表面处理环节[[S8,S36]]。

## 核心原理

气冲造型通过高速气体压力波从砂箱背面直接冲击松散型砂，使砂粒获得向下的高速动能。当砂粒运动受到下部模板与模样阻碍后，动能转化为紧实能，完成砂型紧实[[S34,S22]]。紧实过程分为两个阶段[[S22]]：

| 阶段 | 描述 |
|------|------|
| 第一阶段：自上而下加速与初步紧实 | 砂箱顶部气压骤升，型砂表层形成高紧实度“初实层”；初实层如同一块高速压板，在气压推动下向下移动，推动下层砂粒初步紧实 |
| 第二阶段：自下而上冲击紧实 | 初实层抵达模板后发生剧烈冲击，靠近模板处紧实度大幅提升，随后冲击自下而上逐层传递，直至砂层顶部完成紧实 |

## 技术演进

气冲造型存在两条气体介质技术路线[[S9,S32]]：

- **燃气爆炸型**：可燃气与空气预混后点火爆炸产生冲击波，1962年获得美国专利，1978年由瑞士GF公司实现突破，1980年商业化。此后被压缩空气路线完全淘汰。
- **压缩空气型**：1953年取得专利，早期为高压气冲（工作压力5.0~9.0 MPa，最高15 MPa）。20世纪80年代后发展为低压气冲（0.35~0.45 MPa），可直接复用铸造车间常规压缩空气气源，成为主流[[S9,S34]]。

## 系统与操作流程

气冲造型机标准操作流程[[S9,S22]]：

1. 向砂箱及辅助框内填充型砂；
2. 工作台带动模板、砂箱、辅助框同步抬升，与上部气冲头储气室形成封闭造型腔；
3. 冲击阀在数毫秒内快速开启，储气室压缩空气以垂直压力波冲击型砂完成紧实；
4. 紧实结束后关闭冲击阀，排气阀排出残余气体，完成单轮造型工序。

## 工艺参数与紧实度要求

### 核心工作参数

| 参数类型 | 数值范围 | 说明 |
|----------|----------|------|
| 气冲工作压力 | 0.4~0.6 MPa（JB/T 5773-91） | 主流低压气冲路线，早期高压气冲为5.0~9.0 MPa |
| 快开阀开启速度 | <0.01 s | 对升压速率起决定作用 |
| 气冲紧实总时长 | 10~20 ms | 典型规格780mm×620mm×300mm砂型仅需0.01 s |
| 气包容积 | 砂箱体积+辅助框体积的3~6倍 | 容积越大越有利于提高升压速度 |
| 余隙内平均升压速率 | ≥23 MPa/s（JB/T 5547-91），通常>30~50 MPa/s，最高80~100 MPa/s | 影响紧实效果的核心因素 |
| 耗气量 | 5~8 m³/m³型砂 | 低于振击造型及射压造型的用气量 |
| 型砂紧实率 | 38%~42% | 推荐使用低紧实率型砂 |

### 紧实度分布规律

无模紧实时，靠近模底板处紧实度最高，沿砂箱高度上升紧实度逐步降低，砂型顶部紧实度最低。该分布兼顾了分型面区域的高紧实度与砂型整体良好透气性，优于振击紧实、普通压实及常规高压紧实等同类型造型方法[[S4,S22]]。

有模紧实时，当深凹比（型腔内深凹处高度与最小宽度之比）≤2.5时，匀实性优于慢压紧实工艺；当深凹比>2.5时，会产生“拱效应（架桥现象）”，深凹区入口上方形成高紧实区，入口下方出现紧实度极低区域[[S4]]。

### 硬度与尺寸要求

依据JB/T 5547-91标准[[S15]]：

- 砂型分型面平均硬度不低于85（B型硬度计）；
- 砂型垂直面硬度不低于80；
- 砂型最顶部约30 mm区域无法达到合格紧实度；
- 气冲造型不适用于砂箱高度低于150 mm的矮砂箱[[S20,S33]]。

## 设备组成与规格参数

气冲造型机核心组成单元包括：加砂系统、气冲头单元（含储气室与快开冲击阀）、模板更换系统、机身机架、液压系统、气动控制系统、电气控制系统[[S9]]。砂箱规格与生产率的适应范围如下：

| 参数 | 范围 |
|------|------|
| 砂箱长 | 500~1600 mm |
| 砂箱宽 | 400~1250 mm |
| 砂箱高 | 150~500 mm |
| 生产率 | 10~200 型/h |

（数据来源：行业标准JB/T 5773-91及《砂型铸造设备》[[S5,S9,S26]]）

## 工艺优势

气冲造型相较于同期高压造型具有多项优势[[S34,S9,S20]]：

- 同等砂型强度下工作压力更低，压缩空气消耗更少，成本低于静压造型；
- 砂型垂直硬度高，整体硬度均匀，分型面向上的梯度硬度分布使透气性优异，可显著减少铸件气孔缺陷；
- 仅单次气冲即可完成紧实，无需震击或压实辅助工序，造型周期极短，660mm×480mm规格气冲造型线实际生产率可达170整型/h以上[[S34]]；
- 模样与模板无需额外设置排气塞，工装制造成本低、维护量小；
- 紧实应力在冲击后快速松弛平衡，起模回弹量小[[S4]]。

## 固有缺陷与质量问题

单一气冲造型存在以下已由长期工业实践证实的缺陷[[S33,S25,S26]]：

1. **噪声**：大型气冲造型机峰值噪声可达95 dB；
2. **密封问题**：砂箱与模板、辅助框之间密封难度高，失效后导致型砂外喷、砂型四周硬度偏低，浇注时引发铁液跑火，造成铸件黏砂、胀砂、尺寸变形等缺陷；
3. **架桥现象（拱效应）**：模板布置不合理时，在高模样深凹区上方产生型砂阻滞，导致下方紧实度严重不足，铸件缺陷率显著上升；
4. **顶部紧实不足**：砂型最上方约30 mm区域紧实度不合格，不适用于矮砂箱或高模样铸型；
5. **设备疲劳开裂**：强烈循环压力波长期冲击导致机架焊缝与板材疲劳开裂，已成为制约气冲造型发展的普遍问题[[S31]]。

20世纪90年代末，单一气冲造型机已基本停产，逐步被改进型气冲设备及静压造型机替代[[S26]]。

## 技术改进路线

为解决单一气冲造型的缺陷，行业开发了四种已商业化的改进型气冲工艺[[S25]]：

| 改型方案 | 原理 | 适用场景 |
|----------|------|----------|
| 两次气冲造型机 | 第一次弱气冲预紧实，第二次强气冲整体紧实 | 高模样、大吊砂砂型 |
| 增压气冲造型机 | 配置大小两个冲击阀，先开小阀预紧实再开大阀强冲击 | 高模样造型，减少间距布置 |
| 流态化气冲造型机 | 辅助框侧壁引入横向气流使型砂局部流态化，提升深狭区填充能力 | 模样之间或模样与砂箱之间的狭窄空间 |
| 差动气冲造型机 | 栅格冲击阀下方加装带孔挡板，辅助框增设导向环，使压力波优先冲击四周 | 方形砂箱四角紧实度提升 |

此外，气冲+压实复合工艺（含弹性橡胶压头压实或多触头压实）同样旨在弥补砂型顶部紧实度不足的缺陷[[S25]]。

## 与其他高密度造型工艺的对比

气冲（单一）造型与高压造型、静压造型在紧实原理、透气性、起模性能、噪声、设备投资、对机架冲击等25项指标上存在显著差异：气冲造型对设备机架的冲击破坏程度最高，静压造型起模性能最优[[S12,S8]]。

## 与其他造型/制芯方法的关系

| 方法 | 硬化方式 | 应用特点 |
|------|----------|----------|
| 气冲造型 | 物理冲击紧实，无需加热或化学反应 | 大批量流水线砂型制备 |
| 热芯盒制芯 | 180~280℃热芯盒内树脂硬化 | 砂芯强度高，无需芯骨，精度高，存在甲醛释放[[S24]] |
| 冷芯盒制芯 | 常温下吹入三乙胺等气态催化剂硬化 | 能耗低，尺寸稳定性好，芯盒可用低成本材料，催化剂有毒[[S24,S10]] |
| 自硬砂造型 | 常温下树脂或水玻璃自行硬化 | 适用于中大铸件，退让性好，但高温变形量较大[[S24,S30]] |

气冲造型本身属于砂型紧实工序，与上述制芯方法为平行的造型/制芯工艺路径[[S16,S38]]。

## 应用场景与限域

气冲造型大规模应用于汽车、拖拉机、通用机械等行业中等尺寸灰铸铁及球铁铸件的批量生产，实际运行气冲造型线班产可达1000多箱[[S27,S34]]。

气冲造型本身不属于压铸生产核心工序[[S37]]。其与压铸的间接关联体现在压铸工厂附属砂型铸造环节中制造压铸钢质母模砂型坯料，以及为需内嵌砂芯的特殊压铸件制备附带砂芯的砂型。常规量产冷室/热室压铸的永久金属型生产不直接使用气冲造型工艺[[S14]]。
