## 概述

卧式离心铸造是离心铸造的核心子类之一，其铸型旋转轴线处于水平状态，或与水平线夹角极小（不超过4°）[[S31,S7,S52]]。将液态金属浇入绕水平轴旋转的铸型内，金属液在离心力作用下充型并凝固成形，冷却后可得到组织致密的中空圆柱形铸件[[S60,S20]]。

该方法本质上是一种真正的离心铸造工艺，无需砂芯即可形成中空内孔。由于离心力的作用，密度较小的气体和非金属杂质向铸件内侧聚集，金属液由外向内顺序凝固，铸件的气孔和缩松缺陷少，力学性能优良[[S8,S44]]。

## 分类体系

卧式离心铸造在离心铸造的分类体系中占有重要地位。按铸型旋转轴线的空间位置，离心铸造主要分为**立式离心铸造**和**卧式离心铸造**两大类[[S7,S11,S51]]：

| 分类维度 | 类别 | 说明 |
|---------|------|------|
| 按旋转轴空间位置 | 卧式离心铸造 | 铸型绕水平轴或与水平线夹角＜4°旋转 [[S31,S44]] |
| 按旋转轴空间位置 | 立式离心铸造 | 铸型绕垂直轴旋转 [[S23,S8]] |
| 按旋转轴空间位置 | 倾斜轴离心铸造 | 旋转轴与水平线和垂直线均有较大夹角，应用很少 [[S41,S52]] |
| 按离心力应用 | 真正离心铸造 | 无型芯，仅靠离心力成形铸件内腔 [[S7,S40]] |
| 按铸型材料 | 金属型离心铸造 | 采用金属模具，使用寿命长 [[S7,S51]] |
| 按铸型材料 | 砂型离心铸造 | 采用砂型模具 [[S7,S4]] |
| 按铸型工作温度 | 冷模离心铸造 | 水冷金属型，本体温度40~250℃ [[S7,S51]] |
| 按铸型工作温度 | 热模离心铸造 | 空冷或少喷水冷却，铸型温度350~450℃ [[S7,S51]] |

| 按设备结构 | 滚轮式离心铸造机 | 中大型铸件生产 [[S11,S41]] |
| 按设备结构 | 多工位离心铸造机 | 大批量生产，如缸套铸造 [[S4,S65]] |

## 核心原理与工艺特点

### 工作原理

卧式离心铸造运行时，铸型高速旋转产生50G至100G的离心力（G为重力系数，即离心力与重力的比值）[[S60,S20]]。金属液在离心力作用下均匀紧贴铸型内壁向外分布，依次完成充型和凝固[[S60,S8]]。

离心力计算的基本公式为[[S35,S32]]：

\[
F = \rho \omega^2 R = \frac{\gamma}{g}R\omega^2
\]

式中：F为单位体积金属所受离心力（N/cm³），ρ为金属液密度，ω为旋转角速度，R为旋转半径，γ为金属重度，g为重力加速度。

### 与立式离心铸造的根本区别

卧式离心铸造与立式离心铸造的根本区别在于铸型旋转轴的方向不同，这导致了两种工艺在铸件壁厚分布、适用产品类型上的本质差异[[S35,S37,S8]]：

| 对比维度 | 卧式离心铸造 | 立式离心铸造 |
|---------|------------|------------|
| 旋转轴方向 | 水平轴 | 垂直轴 |
| 铸件自由表面形状 | 圆柱面（均匀） | 抛物面（上薄下厚） |
| 沿铸件长度方向壁厚 | 均匀 | 存在壁厚差，铸件越高差异越大 |
| 冷却条件 | 沿轴向各部分冷却条件相近 | 各高度冷却条件不同 |
| 适用长径比 | 长径比＞1的长筒形、管形铸件 | 高度小于直径的盘状、环状铸件 |
| 典型产品 | 铸铁管、气缸套、长轴套 | 轮盘、环、轴承 |
| 生产模式 | 大规模流水线生产 | 小批量定制化生产 |
| 设备占地 | 水平方向占地较大 | 纵向占地小，空间利用率高 |

卧式离心铸造的核心优势在于：铸件沿长度方向和圆周方向的冷却条件均相近，离心力在全圆周和全长度方向分布均匀，因此铸出的圆筒形铸件壁厚高度均匀，无立式离心铸造因重力与离心力合力沿高度变化带来的厚度偏差[[S35,S8,S44]]。

立式离心铸造受竖直方向重力与水平方向离心力的合力共同作用，金属液内表面呈抛物面形状，铸件高度方向呈现上薄下厚的壁厚差异。铸件高度越大、转速越低，壁厚差越明显[[S35,S37,S23]]。

### 关键工艺优势

1. **无需型芯生产空心铸件**：金属液在离心力作用下紧贴铸型内壁，自动形成中空内孔，简化了套筒、管类铸件的生产过程[[S44,S40]]。
2. **铸件组织致密**：金属液由外向内顺序凝固，补缩效果好，气体和夹渣因密度较小移向内表面，气孔缩松缺陷少[[S44,S8]]。
3. **金属利用率高**：几乎无需浇注系统和冒口，以离心球墨铸管为例，1t铸管仅消耗1.04t铁液，成品率可超过96%[[S40]]。
4. **充型能力强**：离心力通常为重力数十倍，可浇注流动性较差的合金和薄壁铸件[[S44,S40]]。

## 工艺流程与关键参数

### 完整工艺流程

卧式离心铸造的完整工艺分为四大阶段[[S59,S36,S63,S60]]：

![卧式离心铸造全工艺步骤流程图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/e7081da0-f3e2-4347-8187-6e7415ec4121/resource)

**图注**：离心铸造完整工艺流程图，涵盖铸型准备、合金准备、浇注冷却及脱模清理全过程，适用于卧式离心铸造的通用工艺描述[[S69]]。

**① 模具准备阶段**：确定毛坯尺寸，选取对应材质铸型，预热铸型至规定温度，喷涂保护涂料，装配端盖与浇口，调整离心机转速至额定工况[[S59,S36,S63]]。

**② 浇注阶段**：将熔炼完成的定量合金液注入高速旋转的水平铸型内。浇注时金属液进入方向应与铸型旋转方向一致，避免飞溅紊流[[S63,S39]]。对于长铸件（如铸管），浇注槽还需沿轴向匀速运动，保证金属液沿铸件长度方向均匀铺展[[S39]]。

**③ 凝固阶段**：金属液在离心力作用下紧贴铸型内壁，逐层由外向内凝固。可根据工艺需求配套水冷或空冷控温，促进顺序补缩[[S59,S60]]。

**④ 脱模阶段**：待铸件完全冷却至对应合金的安全脱模温度后，停机取出铸件，清理铸型完成循环[[S59,S36]]。

### 转速与离心力参数

铸型转速是卧式离心铸造最核心的工艺参数。转速过低，铸件会发生淋落、不致密、夹杂、气孔等缺陷；转速过高，易产生偏析和表面裂纹[[S39,S64,S71]]。

**常用转速计算公式**：

① 基于重力系数G的转速公式[[S56,S58]]：

\[
n = 299 \sqrt{\frac{G}{R}}
\]

式中：n为铸型转速（r/min），G为重力系数，R为铸件内径（cm）。

② 康斯坦丁诺夫（Konstantinov）公式[[S58,S71]]：

\[
n = \beta \cdot \frac{55200}{\sqrt{\gamma r}}
\]

式中：γ为合金有效重度，r为铸件内半径，β为修正系数（铜合金取1.2~1.4，铝合金取0.9~1.1）。

**重力系数G的常规取值范围**[[S56,S58,S20]]：

| 合金类型 | 重力系数范围 |
|---------|------------|
| 重合金（铸铁、铸钢等） | 35~100 |
| 轻合金（铝合金等） | 80~120 |
| 通用常规范围 | 40~90 |

### 浇注温度

浇注温度需结合合金类型严格控制，通常比同材质重力铸造低5~10K，以避免偏析和晶粒粗大[[S70,S66]]。

| 合金类型 | 常规浇注温度范围 | 数据来源 |
|---------|----------------|---------|
| 铸铁类 | 1327~1370℃ | [[S30]] |
| 铜合金（C95700铝青铜等） | 975~1100℃ | [[S66]] |
| 铝合金 | 液相线以上50~100K | [[S15]] |
| 巴氏合金（ZSnSb11Cu6） | 430℃（优化值） | [[S1,S50]] |

### 模具预热温度

模具预热的核心作用是防止高温金属液接触冷模具发生激冷产生冷隔，同时避免模具热冲击损伤[[S15,S64]]。

常规模具预热温度范围：150~350℃[[S12,S64,S1]]。实际取值因合金不同而异：铝合金离心铸造模具预热温度可取300℃（573K）[[S12]]，巴氏合金轴瓦生产时钢质坯体预热温度为320℃[[S50,S1]]。

### 浇注速度与充型时间

针对铝合金、铜合金等常用材质，常规充型时长可按浇注速度分为三个档位[[S66,S57]]：

| 浇注速度 | 充型时长范围 |
|---------|------------|
| 快浇注 | 6.54~14.73 s |
| 中速浇注 | 10.07~19.39 s |
| 慢浇注 | 14.73~30.02 s |

浇注速度需要避免过快导致的紊流裹气和过慢导致的金属液前端凝固充型不足[[S66,S64]]。

### 浇口位置与浇注系统设计

浇口安装位置需保证金属液进入铸型时不发生溅流、不碰撞高速旋转的模具内壁[[S36,S39]]。对于大直径巴氏合金轴瓦类卧式离心铸造，浇口设置在圆心正下方60mm位置可实现平稳充型[[S1,S5]]。

长铸件配套的移动浇注槽需沿轴向匀速运动，保证金属液沿铸件长度均匀铺展[[S39]]。

## 适用产品与典型应用

卧式离心铸造主要生产长度大于直径的套筒类、管类中空铸件[[S60,S11,S32]]，其铸型绕水平轴旋转，铸件各部分冷却条件和成形条件相同，壁厚沿径向和轴向都均匀，因此可稳定生产长径比很大的中空回转体[[S9,S44,S16]]。

### 主要产品类型

**（1）铸铁管（输水管、煤气管等）**
该产品是卧式离心铸造最具代表性的大批量应用。无需砂芯即可成型中空管壁，工艺出品率高（1t铸管仅消耗1.04t铁液，成品率可超过96%）[[S40,S11]]。管径范围从最小75mm到最大3000mm[[S41]]。

**（2）气缸套**
广泛应用于内燃发动机，属重要的消耗性零件，与活塞环组成摩擦副，需求量极大[[S4]]。汽车和拖拉机缸套毛坯内径一般在90~200mm，属小型缸套；船舶和机车用缸套尺寸更大，有的内径可超过400mm[[S4]]。卧式离心铸造可获得组织致密、耐磨性能优异的缸套毛坯[[S11,S52]]。

**（3）轴套与套筒类零件**
包含铜套、钢背衬套等各类中空回转耐磨零件。生产过程无需额外型芯，工艺简化、生产效率高[[S11,S40]]。

**（4）双金属复合管与复合空心轧辊**
可实现不同性能金属的冶金结合：外层耐磨耐蚀（如白口铸铁）、内层强韧（如灰铸铁），达到优异的综合性能匹配[[S40,S65,S11]]。

### 长径比较大的工件为何更适合卧式离心铸造

立式离心铸造的铸型绕垂直轴旋转，在重力与离心力共同作用下，金属液自由表面呈抛物面形态；铸件高度越高，上下端壁厚差异越大，无法保证长轴向工件的壁厚均匀度。

卧式离心铸造的铸型绕水平轴旋转，铸件沿轴向和径向各部分的冷却条件、成形条件完全一致，最终铸件壁厚沿径向和轴向都均匀，无明显厚度偏差。因此，卧式离心铸造可稳定生产长径比很大的中空回转体[[S9,S44,S42,S23]]。

### 双金属离心铸造的冶金结合机理

双金属卧式离心铸造的关键是先后将两种不同成分的金属液依次注入水平旋转的铸型中[[S3,S65,S54]]：

1. 首先浇注外层金属液（如耐磨合金），在离心力作用下均匀分布于铸型内壁；
2. 待外层金属冷却至接近固相线的半凝固状态时；
3. 再浇注内层高温金属液（如高强度合金）；
4. 内层金属液在外层金属内表面的冲刷和热作用下，两种金属界面发生充分的元素互扩散与局部熔合；
5. 最终形成均匀牢固的冶金结合层，结合强度远高于机械复合方式。

双金属结合质量的关键控制要素包括：两层金属的浇铸间隔时间、铸型预热温度、内层金属浇铸温度、离心转速等[[S54,S53]]。

## 缺陷与质量控制

### 常见缺陷类型

卧式离心铸造的典型工艺缺陷包括以下类型[[S67,S61,S64,S19,S29]]：

| 缺陷类型 | 特征与成因 | 防治措施 |
|---------|----------|---------|
| 缩孔 | 形成于铸件内表面，可敞露也可隐避；厚壁铸件更易发生，由凝固顺序不当引起 | 水冷铸型增加温度梯度，内表面添加绝热覆盖剂，凝固由型壁向中心推进 |
| 纵向裂纹 | 由转子偏心不平衡离心力引发 | 改善动平衡，适当降低转速 |
| 横向裂纹 | 由轴向收缩受阻、铸件端部缺陷、表面缺陷导致 | 控制浇注温度和浇注速度，改善铸型表面质量 |
| 气孔 | 金属液中气体析出、涂料或润滑剂分解产气滞留 | 控制金属液精炼质量，降低涂料发气量 |
| 偏析 | 不同密度相在离心力场下沉浮运动导致成分不均匀 | 合理控制转速和浇注温度 |
| 冷隔 | 金属液温度过低或浇注速度过慢，先后注入的金属液未完全熔合 | 提高浇注温度和模具预热温度，保证浇注速度 |
| 分层 | 金属液与模具表面高速接触先凝固形成异质薄层 | 控制模具预热温度和浇注速度 |
| 坍流 | 转速过低、停车过早、浇注温度过高导致合金液淋落，在内表面形成凹陷、凸起或金属瘤 | 铸件冷却至低于固相线100~300℃后再停止铸型旋转，防止砂型局部过热 |

### 与立式离心铸造缺陷分布的对比

基于2026年同工况下巴氏合金（浇注温度430℃）离心铸造的ProCAST数值模拟结果[[S62,S31,S23,S5]]：

| 对比指标 | 卧式离心铸造 | 立式离心铸造 |
|---------|------------|------------|
| 充型阶段用时 | 7.15 s | 6.55 s |
| 初始温度至完全凝固总时长 | 149.4 s | 116.6 s |
| 缩孔、缩松总面积 | 0.5629 cm³ | 0.8263 cm³ |
| 缺陷分布特征 | 主要分布于圆环中部最后凝固区 | 多处出现严重缺陷，分布无规律 |

立式离心铸造受不同方向金属液汇流扰动的影响，温度场分布较无规律且多处出现缺陷[[S62,S10]]；卧式离心铸造受水平旋转轴约束，铸件各方向冷却条件均匀，缺陷总量明显更低，但卧式工艺特有的周向螺旋流场可能引发螺旋状冷隔缺陷，纵向偏心失衡则易引发沿轴向分布的纵向裂纹[[S62,S44,S67]]。

### 模具材料

双层模具的外模材料最好用铸钢或球墨铸铁，灰铸铁可用于小型机器的外模（应设加强肋或增加壁厚，并进行消除应力热处理），内模材料可用灰铸铁[[S36,S4]]。

### 冷却方式

大批量生产的卧式悬臂离心铸造机采用铸型内水冷（轴也内水冷）或水管多孔外水冷和轴承壳水冷；中、大型立式离心铸造机以轴中心水冷为主[[S36,S4]]。

## 标准与规范

卧式离心铸造产品检测与验收涉及以下国内外标准[[S13,S43,S21,S46,S33]]：

**中国国家标准（GB）：**

- **GB/T 1176《铸造铜合金技术条件》**：明确了离心铸造工艺代号为Li，允许离心铸造场景下特定铜合金的磷元素含量最高提升至1.5%[[S13]]。
- **GB/T 5611《铸造术语》**：对包括离心铸造在内的全铸造领域1254项基础术语作出明确定义[[S33]]。
- **GB/T 1174《铸造轴承合金》**：规定了锡基、铅基等轴承合金的硬度测试要求，适配卧式离心铸造轴瓦类铸件的验收[[S21]]。

**国际标准（ISO）：**

- **ISO 1338:1997《铸造铜合金——化学成分和力学性能》**[[S43]]
- **ISO 4382/1:1991(E)《滑动轴承用铸造铜合金》**：对离心铸件的锡含量允许范围放宽至10.5%~13%[[S21,S33]]

**美国ASTM标准：**

- **ASTM B271-08《铜合金离心铸件》**：明确铜合金离心铸件的成分与性能验收要求[[S43,S46]]
- **ASTM B208-06**：《砂型、金属型、离心铸造和连续铸造铜合金拉伸试样的制备》[[S43,S46]]

## 最新研究与前沿动态

### ProCAST数值模拟

近年来，ProCAST数值模拟技术在卧式离心铸造工艺优化中得到了广泛应用[[S27,S45,S22]]：

- **2024年高稀土镁合金研究**：通过数值模拟预测卧式离心铸造中熔体充型流场、温度场分布，计算不同工艺参数下的晶粒尺寸与力学性能[[S27,S25]]。

![卧式离心铸造与重力铸造不同时间点固相分数分布对比图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/77a2f06f-4bb1-4436-8e68-e6be0919db02/resource)

- **2026年大直径推力轴瓦研究**：通过ProCAST结合4因素3水平正交试验，获得卧式离心铸造最优工艺参数组合：巴氏合金铸造温度430℃、坯体初始温度320℃、铸造位置偏置距离40mm、离心转速380r/min；优化后铸件缩孔缩松总体积占比低于0.2%，产品检测完全符合质量要求[[S50,S1,S5]]。

### AI驱动的工艺优化

近5年来，将BP神经网络、遗传算法等智能优化算法与数值模拟结果相结合的AI方法，已在卧式离心铸造工艺优化中得到应用[[S17,S73]]。此类方法无需建立对象精确机理模型，即可以数据驱动方式快速完成多工艺参数全局寻优，有效降低铸件缩孔、气孔、偏析等缺陷的发生率，提升铸件力学性能的一致性[[S17,S73]]。
