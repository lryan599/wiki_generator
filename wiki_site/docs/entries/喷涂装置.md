## 概述

喷涂装置，在压铸领域中标准名称为喷涂系统（spray system），是压铸单元的核心辅助设备，其核心功能是将压铸脱模剂进行配比、雾化，并定点定量地喷涂在模具型腔表面[[S16,S13]]。行业中也常称之为压铸脱模剂喷涂系统、压铸模具喷涂机，英文通用术语为die spray system或mold release spray unit[[S13]]。

该装置在压铸生产循环中占据独立的工艺窗口，位于铸件开模顶出与取件完成之后，以及下一循环合模浇注之前。其标准动作顺序为：喷涂机构下降→吹气清理型腔→定向喷涂定模型腔→定向喷涂动模型腔→二次吹气吹扫→喷涂机构归位待机[[S34,S4]]。

喷涂装置的核心作用体现在四个方面[[S23,S36]]：
1. **隔离保护**：在高温合金熔体与模具表面之间形成隔离保护层，避免直接冲蚀，延长模具寿命。
2. **润滑脱模**：形成均匀润滑层，降低铸件脱模阻力，防止粘模及拉伤。
3. **热平衡控制**：利用雾化脱模剂的蒸发吸热效应，控制模具热平衡，稳定型腔工作温度场。
4. **清理型腔**：清除开模后残留的金属碎屑与氧化皮，避免下一模产生表面缺陷。

## 分类

喷涂装置可按移动特征和自动化程度两个维度进行分类。

### 按移动特征分类

喷涂装置可划分为固定式与移动式两大类别。固定式喷涂装置将喷头安装在动、静型或定模板上，仅适用于80~150吨小型压铸机、型腔无深槽的简单形状铸件生产场景，其喷涂效率高、周期短[[S26,S1]]。

移动式喷涂装置则可细分为多个子类，以适应不同吨位及复杂程度的模具。详细的分类及应用范围如下表所示。

| 型式 | 说明 | 应用范围 |
| :--- | :--- | :--- |
| 固定喷嘴式 | 将4~6个喷嘴固定在压铸机动静型上 | 80~150吨小型压铸机 |
| 上下移动式 | 喷头带6~18个喷嘴，从压铸机上方伸入喷涂 | 250~800吨中型压铸机 |
| 水平（横向）移动式 | 喷头带6~18个喷嘴，从压铸机侧方伸入喷涂 | 1000~2000吨大型压铸机 |
| 与取件装置一体式 | 喷射头与取件机构为一体，横向伸入喷涂 | 250~1200吨压铸机 |

表格数据来源：[[S29]]

此外，为满足新能源汽车一体化压铸对大尺寸、复杂型腔铸件的需求，多轴可编程式（如应用工业机器人）喷涂装置日益重要，其支持自定义喷涂轨迹，可柔性适配深腔及复杂结构[[S26,S28,S36]]。

### 按自动化程度分类

| 类别 | 说明 |
| :--- | :--- |
| 手动喷涂 | 依靠人工操作喷枪，适用于小批量生产或换型调试阶段[[S28]]。 |
| 半自动喷涂 | 部分动作可预设程序，剩余步骤由人工辅助完成[[S28]]。 |
| 全自动喷涂 | 与压铸机控制系统完全联动，无需人工介入完成全流程动作[[S28]]。 |

## 工艺角色

在压铸循环中，喷涂装置位于开模取件之后、合模之前[[S4,S34]]。除涂覆脱模剂外，喷涂装置还承担着关键的工艺属性调控职责：

- 形成防护与润滑涂层，其效果直接关联铸件粘模、拉伤等缺陷的产生与否[[S8]]。
- 调控模具热平衡：通过雾化液滴的蒸发吸热，稳定模具表面温度。理想的模具表面温度区间为150~250℃，在该区间内，脱模剂可在1秒内形成有效皮膜[[S8,S9]]。
- 清除残留物：利用压缩空气对开模后的型腔进行吹扫，清除铝屑、残渣，防止铸件表面缺陷[[S34,S4]]。

## 结构与组成

一套典型的全自动喷涂装置由以下七大核心模块构成[[S16,S4,S30,S8,S34]]：

1. **喷头/喷嘴单元**：包含单或多喷嘴组件及雾化装置。喷嘴类型如扇形喷嘴（适合大面积均匀覆盖）和雾化喷嘴（适合复杂深腔）[[S8]]。部分全自动喷涂机配备活动微型雾化喷嘴，可作90°自动旋转[[S30]]。
2. **管路单元**：分为独立的脱模剂输送管路与压缩空气管路，配置空气过滤器、压力表等组件[[S4]]。
3. **泵站单元**：即脱模剂自动稀释压送机，可将原液按设定比例与水混合后加压输送。常用稀释倍率为40~300倍，最大压送量可达32 L/s，工作压力为0.2 MPa[[S30]]。
4. **压力控制单元**：包含独立的压缩空气减压阀、喷涂压力表和吹气压力表，可分别调节喷涂介质与吹扫气源的压力[[S4]]。
5. **压缩空气系统**：为雾化和型腔吹扫提供洁净气源，通常配置多级过滤以去除杂质和水分[[S16]]。
6. **移动机构单元**：包含气缸或伺服电机驱动的垂直/水平往复运动模组，或多轴工业机器人，带动喷头完成预设路径动作[[S16,S34]]。
7. **控制系统（PLC）单元**：支持自定义喷涂动作时序、各区喷涂时长及运动速度，并可读取模具温度传感器数据以动态调整工艺参数[[S16,S34]]。

## 关键参数

### 性能参数定义

根据国家标准GB/T 37371-2019《压铸单元 术语》，喷涂装置的性能参数定义如下[[S32]]：

- **喷涂量**：单位时间喷出的脱模剂体积。
- **喷涂定位精度**：喷涂设定位置与实际位置的偏差。
- **喷涂周期**：从启动喷涂至喷涂结束并回到原位的时间。

### 工艺参数推荐范围

行业公认的主流喷涂工艺参数及其推荐范围如下表所示。

| 参数项 | 推荐值与范围 |
| :--- | :--- |
| 压缩空气喷涂压力 | 0.3~0.6 MPa（最优0.6 MPa） |
| 雾化压力 | 0.5~0.6 MPa |
| 脱模剂流体输送压力 | 最高0.2 MPa |

## 应用场景

喷涂装置的选型需根据压铸机类型、合金材料及模具特征进行匹配。

- **按压铸机吨位**：固定式喷涂用于80~150吨小型机；上下往复式用于250~800吨中型机；水平移动式用于1000~2000吨大型机[[S29]]。已研发出适用于10000~16000kN超大型冷室压铸机的平移式喷涂装置，面向一体化压铸结构件的大范围喷涂需求[[S4]]。
- **按合金与工艺**：低流量雾状喷涂系统适用于热室压铸机生产小型锌合金薄壁件；高压力大流量系统则用于冷室压铸机生产铝合金、镁合金铸件，以穿透高温模具表面的蒸汽膜，确保脱模剂有效附着[[S19,S12,S20]]。

## 相关标准与规范

与喷涂装置直接相关的标准与规范主要包括：

- **GB/T 37371-2019 《压铸单元 术语》**：定义了喷涂系统、喷涂量、喷涂定位精度等核心术语[[S13,S32]]。
- **GB/T 39957-2021 《压铸单元 技术条件》**：在4.5节明确规定了喷涂装置的技术要求，包括需符合JB/T 12284.2、配置防滑落装置、雾化均匀无滴漏、压缩空气压力≥0.5 MPa等[[S35,S6]]。
- **GB 20906-2007 《压铸单元安全技术要求》**：等效采用欧洲标准EN 869:1998，对包括喷涂设备在内的压铸辅助设备提出强制性的通用安全要求[[S31]]。
- **JB/T 12284.2 《压铸用喷涂机 第2部分：技术条件》**：压铸喷涂机的行业技术标准[[S35,S6]]。
- **NADCA技术报告**：北美压铸协会（NADCA）的研究文献中包含关于水基脱模剂喷涂热流特性的研究，已纳入其技术报告序列（如T91-115）[[S27]]。

## 潜在缺陷与维护要点

### 装置异常与铸件缺陷的关联

喷涂装置运行异常将直接导致压铸缺陷，具体关联关系如下[[S8]]：

- **喷嘴堵塞/走位偏差** → 喷涂覆盖不全 → 局部缺润滑 → 粘模、拉伤。
- **雾化压力不足** → 脱模剂局部堆积 → 铸件表面形成流痕、冷隔。
- **喷涂后滴漏/吹扫不净** → 型腔残留脱模剂液滴 → 铸件内部产生气孔。
- **长期喷涂不均** → 模具表面局部温度梯度超过50℃ → 加速模具热龟裂。

### 维护要点

| 维护频率 | 核心操作要点 |
| :--- | :--- |
| 每班 | 检查压缩空气压力≥0.5MPa；检查喷嘴堵塞与滴漏；确认喷涂空循环动作顺畅[[S24,S21]]。 |
| 每周 | 疏通所有喷嘴管路；清理喷嘴表面积碳[[S7,S3]]。 |
| 每月/每季 | 给升降移动副加注润滑脂；校验脱模剂计量泵流量精度至±5%以内；检查防滑落安全机构有效性[[S7,S3]]。 |
