## 概述

外冷铁是铸造工艺中的一类激冷构件。造型或制芯时，将其放置于模样（芯盒）表面，作为铸型的组成部分，用以加快铸件特定部位的局部冷却速度，实现可控的凝固顺序[[S8,S35,S26]]。外冷铁不与铸件本体发生熔合，使用后可以回收并重复使用，其材质选用范围通常不受铸件材质的严格限制[[S8,S30]]。

## 定义与核心原理

### 基本定义

外冷铁是一种放置在铸型型腔边界上的高导热、高蓄热物体，其核心职能是快速吸收铸件热节区域的热量，改变该区域的等效散热能力，从而调节局部凝固时间[[S4,S7]]。

### 与内冷铁的区别

外冷铁与内冷铁在位置、熔合状态、可复用性及材质要求上存在根本差异：外冷铁布置在铸型型腔的外侧，不参与铸件本体熔合，可反复使用，材质可选用铸铁、铜、石墨等多种材料；内冷铁则需插入型腔内部，最终必须与铸件完全熔合成为铸件的组成部分，其材质须与铸件本体成分相近，通常仅在外冷铁激冷效果不足时才考虑使用，多用于铸件上可通过后续机械加工去除该部分的区域[[S6,S30,S27]]。

### 激冷机理

外冷铁的激冷作用建立在凝固模数理论之上，即凝固模数 *M=V/A* （体积除以有效散热表面积）。由于外冷铁的热导率和热容量远高于砂型，它能显著增大铸件热节处的等效散热面积，进而降低该区域的等效模数。这使得原本滞后凝固的厚大部位能够与薄壁区域实现同步（同时凝固）或按设计方向加速冷却（顺序凝固），从而消除因补缩通道中断而产生的收缩类缺陷[[S4,S7,S35]]。在顺序凝固工艺中，应用外冷铁可将铸件等效模数 *M₁* 控制在补缩壁模数 *Mₚ* 的 0.83~0.91 倍范围内，以保证向冒口方向的温度梯度[[S11,S15]]。

## 分类与结构

外冷铁按与铸件的接触方式分为两大类[[S26,S34]]：

*   **直接外冷铁（成型外冷铁/明冷铁）**：与铸件表面直接接触，激冷作用强。可进一步细分为：
    *   **无气隙外冷铁**：布置于铸件底部或内侧，在重力和铸件收缩力作用下，与铸件贴合更紧密，激冷效果更强。
    *   **有气隙外冷铁**：布置于铸件顶部或外侧，因存在接触间隙，激冷效果相对无气隙冷铁稍弱[[S6]]。
*   **间接外冷铁（隔砂外冷铁/暗冷铁）**：冷铁基体与铸件之间留存一层特定厚度的型砂（挂砂层），激冷作用平缓且可调控。其主要优点为：可避免灰铸铁表面生成白口层或过冷石墨层；可降低强激冷引发的铸件开裂风险；铸件外观平整度好，且不存在冷铁与铸件熔接的问题[[S26,S34]]。

## 材质与特性

不同材质的外冷铁，其激冷效率主要由材料的热扩散系数和热容量共同决定。正确选择材质必须结合冷铁尺寸与被激冷热节的体积[[S13]]。

*   **铸铁**：热容量较大，成本经济，是铸钢件生产中通用的冷铁材质[[S30]]。
*   **铜合金**：导热性能最优。在大尺寸工况下（冷铁不因自身热容量先饱和），铜质冷铁的激冷效率可达到铸铁的4倍，适用于大体积热节的强激冷需求。但小尺寸冷铁受限于热容量，其优势不明显[[S13]]。
*   **石墨**：激冷效率约为铜的1/2，同时是铸铁的2倍。激冷作用均匀，常用于锡青铜等有色合金铸件[[S28,S10]]。
*   **铸钢**：强度与耐用性好，适合制作需反复使用的通用型冷铁。对于铝镁合金铸件，大型铸钢冷铁的激冷效果也优于石墨[[S13]]。

| 材质 | 关键特性 | 适用场景 |
| :--- | :--- | :--- |
| **铸铁** | 热容量大，成本低[[S30]] | 铸钢件通用选择[[S30]] |
| **铜合金** | 导热性最优；大尺寸时效率约为铸铁4倍[[S13]] | 大体积热节、需强激冷部位[[S10]] |
| **石墨** | 激冷均匀；效率约为铸铁2倍[[S13]] | 锡青铜铸件、要求避免白口的铸铁件[[S28,S26]] |
| **铸钢** | 强度高，耐用[[S30]] | 通用、反复使用的场景[[S30]] |

## 工艺作用与应用场景

外冷铁在铸造工艺中具有多重作用[[S26,S35,S5]]：

1.  **防止收缩缺陷**：在冒口难以补缩的位置消除缩孔、缩松[[S26]]。
2.  **实现可控凝固**：与冒口配合，利用末端效应增大冒口有效补缩距离，减少冒口数量；或使铸件各部位趋向同时凝固，防止变形[[S6,S33]]。
3.  **防止热裂**：消除壁厚交叉或急剧变化部位的热应力，预防裂纹[[S26]]。
4.  **提高工艺出品率**：通过缩小冒口体积，可将铸钢件工艺出品率提升至70%以上[[S5,S8]]。
5.  **改善组织与性能**：细化局部晶粒，提高铸件表面硬度与耐磨性，减轻或防止厚壁部位的元素偏析[[S26]]。

**典型应用案例如下**：

*   **砂型铸造热节处理**：用于轴头根部、法兰根部、厚大截面凸台及加工孔处的局部激冷，避免产生缩松[[S35,S18]]。
*   **大型及高合金铸钢件**：在阀体、高压蒸汽室、轮类铸件、风机叶轮等产品上普遍应用[[S5,S34,S35]]。例如，在高压临氢阀体铸造中使用隔砂外冷铁，可避免气体缺陷并保证外表质量[[S34]]。
*   **有色合金铸造**：虽以内冷铁为辅，但外冷铁仍是调控铸件热节的主流激冷手段[[S10]]。在铝合金弹体铸造中，针对厚大凸台部位使用铜合金外冷铁，是配合暗冒口消除缩松缺陷的关键工艺[[S38]]。

## 设计与计算要点

外冷铁的设计计算主要基于模数法，核心是确定冷铁的工作表面积和厚度。

### 厚度选取

冷铁厚度的确定需依据合金种类和热节圆直径 **T**。增大厚度能提升激冷效率，但超过约30mm后，对降低铸件热模数的贡献将趋于平缓，因此无需追求过厚的冷铁[[S15,S14]]。

| 适用合金类型 | 经验厚度比值 (δ/T) |
| :--- | :--- |
| 铸钢件 | 0.3 ~ 0.8 [[S22,S21]] |
| 灰铸铁件 | 0.25 ~ 0.5 [[S17,S21]] |
| 铝/镁合金件 | 0.8 ~ 1.9 [[S17]] |

### 结构参数与工作面积

**直接外冷铁**的接触面宽度 **B** 与热节圆直径 **T** 的推荐关系为 **B = (1~1.4)T**，通常厚度δ取20~30mm[[S28,S37]]。对于有、无气隙两种类型，其工作表面积的计算公式核心是建立铸件原始模数 *M₀* 与目标等效模数 *M₁* 的关系[[S7,S15,S11]]。可利用单位体积冷铁工作表面积计算列线图快速求解[[S11]]。

**隔砂外冷铁**设计时，冷铁厚度推荐为 **δ = (1~1.4)T**。铸件与冷铁之间的**隔砂层厚度**一般为20~30mm；当此厚度超过40mm时，冷铁将基本丧失激冷作用[[S24,S25]]。

## 操作规范与质量控制

外冷铁的使用效果高度依赖于严格的表面处理和安装工艺[[S12,S28]]：

*   **表面处理**：工作表面须经抛丸或喷砂彻底除锈。可选用松香酒精、炭灰水、机油等涂料进行刷涂，并在使用前充分烘干，严禁表面残留水分、油污或存在锈蚀，以防引起铸件气孔[[S12,S23,S28]]。
*   **使用维护**：齿轮类铸件用外冷铁允许重复使用10~15次。当表面氧化层厚度达到0.2~0.4mm时，必须停用或重新修磨。在冷铁上适当钻排气孔可帮助提升透气性[[S5,S20,S28]]。
*   **安放设计**：冷铁侧面应设计为45°斜角，使砂型与冷铁交接处形成平缓过渡，以降低收缩应力引发裂纹的风险。当激冷面积大时，应采用多块小型冷铁交错布置并留有间隙，避免大面积连续冷铁造成中心应力过大而开裂[[S15,S25]]。

## 缺陷、风险与局限性

**常见缺陷**：外冷铁表面处理不当（残留水、油污）会直接导致铸件产生内部气穴或皮下气孔[[S23,S31]]。过强的激冷作用则可能引发铸件表面产生白口组织、局部过硬（影响加工）、熔合不良（冷隔）等缺陷[[S18,S26,S31]]。

**局限性**：

*   **难以独立解决极端热节**：当铸件热节厚度大于其连接壁厚度的4倍时，仅靠外冷铁无法消除厚大部位的缩松缺陷，必须与冒口配合使用[[S17,S38]]。
*   **应用位置受限**：对深藏于型腔内部的复杂厚大热节，外冷铁通常无法触及并发挥作用[[S31]]。
*   **对铸件材质的风险**：在灰铸铁件上激冷过强极易造成表面白口。工艺控制不当时，由剧烈温度梯度引起的收缩应力也可能致使铸件整体开裂[[S26,S6]]。

## 与其他工艺措施的协同

外冷铁作为一种局部温度场调控手段，很少孤立使用，通常需与冒口、补贴等形成互补系统：

*   **与冒口配合**：在两冒口之间安放外冷铁，可在铸件中部形成“激冷端”，使冷铁两侧朝向对应冒口的温度梯度都得到扩大，从而将原冒口的末端区拓展为两个独立的补缩通道，显著增大冒口的有效补缩距离[[S33,S9]]。通过此配合，可减少冒口数量、缩小冒口体积，显著提升工艺出品率[[S8,S16]]。
*   **与补贴配合**：在铸件立壁上设置外冷铁的同时增设梯度补贴，可共同构建从铸件远端向冒口方向逐步升高的温度梯度，确保严格顺序凝固，防止立壁补缩通道过早断开而产生的缩孔[[S33,S29]]。
*   **与内冷铁配合**：仅在外部激冷方案确实无法满足厚大热节的致密要求时，才允许在非承压、非关键的厚壁区域配合使用内冷铁。内冷铁的用量需严格控制，其重量通常占该激冷部位铸件重量的5%~10%[[S27,S15]]。对于压力容器等关键承压铸件，工艺规程通常禁止使用内冷铁[[S15]]。
