## 概述
解理断裂是金属材料在正应力作用下，原子结合键被直接破坏，沿表面能最小的特定结晶学平面（即解理面）发生的低能穿晶脆性断裂[[S28,S54,S14]]。该断裂属于脆性断裂的子类，断裂前几乎无宏观塑性变形（工程上通常判定为光滑拉伸试样的断面收缩率小于5%），裂纹扩展速率极高，断裂面基本垂直于最大拉应力方向[[S44,S18]]。

解理断裂常见于体心立方（BCC）和密排六方（HCP）晶格的金属及合金中，面心立方（FCC）晶系的金属在常规工况下几乎不发生解理断裂[[S28,S54,S21]]。典型的解理面因晶系而异：体心立方金属的解理面为{100}晶面，密排六方金属为{0001}晶面，三角晶系为{111}晶面[[S50,S25]]。

## 发生条件

解理断裂的触发与多个因素密切相关，通常需要以下条件的共同作用：

| 影响因素 | 促发解理断裂的倾向性条件 | 依据来源 |
|---------|------------------------|---------|
| 应力状态 | 三向不等拉伸高约束状态（应力状态软性系数α小），拉应力占比高 | [[S13,S54,S45]] |
| 环境温度 | 低温环境，低于韧脆转变温度T<sub>c</sub>时解理断裂应力低于屈服强度 | [[S45,S54,S56]] |
| 应变速与载荷 | 高应变速率、冲击载荷作用 | [[S54,S45,S56]] |
| 晶体结构 | 体心立方、密排六方晶格金属及合金易发生 | [[S28,S54,S21]] |
| 晶粒尺寸 | 晶粒粗大时解理断裂倾向显著提升 | [[S48,S63]] |
| 第二相与夹杂 | 解理面上分布的第二相粒子、夹杂物优先引发微裂纹 | [[S56,S48]] |
| 材料纯净度 | 杂质元素在晶界偏聚或存在冶金缺陷时正断抗力降低 | [[S13,S35,S54]] |
| 缺口效应 | 构件表面缺口、结构突变产生应力集中，增大脆性倾向 | [[S35,S13]] |

解理断裂与材料冷脆性直接相关。当温度降低时，材料的屈服强度随温度下降快速升高，而解理断裂应力基本保持稳定。当温度低于韧脆转变温度T<sub>c</sub>时，应力先达到解理断裂应力，材料在几乎无宏观塑性变形的情况下优先发生解理断裂[[S56,S45,S6]]。

## 微观机制

### 裂纹萌生与扩展

解理裂纹的萌生位置大多处于晶界、亚晶界、孪晶界、杂质及第二相界面等位错易于塞积的区域[[S28,S14]]。主流的裂纹萌生模型包括：

- **位错塞积模型**：单向塞积、双向塞积机制，位错在障碍处塞积导致应力集中，引发裂纹萌生[[S28,S36]]。
- **位错反应机制（Cottrell机制）**：柏氏矢量较小的位错相互反应生成柏氏矢量较大的位错，大位错如同楔子塞入解理面将其劈开[[S28,S14]]。
- **位错交叉滑移与刃型位错合并**：建立于解理生核前存在变形的条件之上[[S28]]。

### 解理台阶与河流花样

解理台阶是解理断口最基本的微观构成单元，其形成途径主要有两种[[S28,S16,S14]]：

1. **螺位错交割形成**：解理裂纹在扩展过程中与螺位错相交割，直接生成高度等于柏氏矢量的台阶。
2. **次生解理或撕裂连接**：两个相互平行、存在高度差的解理裂纹，通过次生解理面或撕裂的方式互连形成台阶。同方向的解理台阶汇合后高度增加，反方向的台阶汇合后消失。

河流花样是解理断口最典型的微观形貌特征，由大量小解理台阶在裂纹扩展过程中逐步汇合形成——支流不断并入主流，河流的顺流方向与解理裂纹的扩展方向完全一致[[S16,S14,S50]]。当解理裂纹穿过小角度倾斜晶界时仅改变流向即可通过；而当裂纹通过扭转晶界或大角度晶界时，会激增大量新的河流支流。遇到粗大第二相粒子阻挡时同样会触发河流激增[[S20,S50]]。

### 其他微观花样

- **扇形花样**：解理裂纹穿过晶界后，在新晶粒内某一点重新形核，随后以该点为裂纹源呈扇形向外辐射扩展，形成沿扩展方向被拉长的狭长解理面形貌[[S16,S53]]。
- **舌状花样**：与解理裂纹高速扩展时在裂纹前沿生成的变形孪晶直接相关，裂纹沿变形孪晶与基体的界面扩展生成舌状微观形貌[[S16,S14]]。

## 宏观断口特征

解理断裂的宏观断口具有以下典型特征[[S28,S44,S52,S54]]：

- 断口平齐且垂直于拉伸方向，无明显宏观塑性变形
- 断面分布大量无规则取向的解理小刻面，对强光转动时可见明显闪光反光现象
- 整体呈现结晶状（颗粒状）外观

**放射区特征**：从断裂起源位置向外发散的放射状撕裂棱，放射源直接对应断裂起始点。加载速率越高、温度越低时，放射区在整个断口中的占比越大[[S45,S18,S11]]。

**人字纹特征**：板材类构件发生解理断裂时，宏观断口常呈现人字纹花样。人字纹的交点位置直接对应断裂起源处，人字纹开口的反向即为裂纹扩展方向；若构件表面存在预先加工的缺口，则人字纹的尖顶方向与裂纹扩展方向一致[[S45,S18,S41]]。

## 断裂分类与区别

解理断裂在断裂分类体系中属于脆性断裂大类下的穿晶脆性断裂[[S44,S18,S21]]。下面将解理断裂与几种易混淆的断裂模式进行系统对比：

| 对比维度 | 解理断裂 | 准解理断裂 | 沿晶断裂（晶间断裂） | 韧窝断裂（微孔聚合型） |
|---------|---------|-----------|-------------------|---------------------|
| 断裂大类 | 脆性断裂 | 过渡混合断裂 | 脆性断裂（室温）或韧性断裂（高温） | 韧性断裂 |
| 裂纹路径 | 穿晶，沿特定解理面 | 穿晶 | 沿晶界 | 穿晶或沿晶 |
| 塑性变形 | 极小，断面收缩率<5% | 存在局部塑性变形 | 室温下极小 | 显著宏观塑性变形 |
| 裂纹起源 | 晶界、相界面上位错塞积处 | 晶粒内部空洞、夹杂物和第二相粒子处 | 晶界脆性沉淀相或杂质偏聚处 | 夹杂物或第二相粒子为核心形成微孔洞 |
| 微观形貌 | 解理台阶、河流花样、舌状花样、扇形花样 | 类解理小平面、向四周发散的短河流花样、小韧窝、短而弯曲的撕裂棱 | 冰糖状晶粒外观 | 等轴或拉长型韧窝 |
| 宏观断口 | 平齐光亮结晶状，可见放射区和人字纹 | 较平整，呈脆性特征 | 结晶状或冰糖状 | 暗灰色纤维状，可见纤维区、放射区、剪切唇 |
| 常见场景 | 低温、冲击、三向拉伸应力，BCC/HCP金属 | 淬火+低温回火组织（马氏体、贝氏体），脆韧转变温度区间 | 氢脆、应力腐蚀、高温蠕变、回火脆 | 常规室温拉伸，塑性良好的材料 |

- **解理断裂与准解理断裂的异同**：准解理断裂起源于晶粒内部的空洞、夹杂物和第二相粒子，裂纹向四周不连续局部扩展；而解理断裂起源于晶界或相界面，裂纹由晶界向晶内呈河流走向连续扩展。准解理小平面的位向与基体解理面{100}无严格对应关系，且通常伴随着更多局部塑性变形[[S61,S19,S55]]。
- **解理断裂与沿晶断裂的区分**：解理断裂为穿晶断裂，裂纹穿过晶粒内部沿解理面分离；沿晶断裂裂纹沿晶界扩展，断口呈现冰糖状晶粒外观。两者的判据关键在于裂纹扩展路径是穿晶还是沿晶，须借助高倍电镜观察确认[[S22,S61]]。

## 检测与表征方法

### 扫描电镜断口分析

扫描电镜（SEM）是解理断裂断口分析最核心的手段[[S24,S34]]。分析流程如下：

1. **宏观分析**：首先用肉眼、放大镜或低倍体视显微镜观察断口，确定断裂性质、裂纹源位置与扩展方向[[S51,S12]]。
2. **微观SEM观测**：将新鲜、未污染、未手触的断口样品送入电镜，优先使用二次电子模式观测断口形貌，识别河流花样、解理台阶等特征[[S24,S38]]。判定的核心SEM识别特征为：解理面上的解理台阶汇流形成典型河流花样，支流汇合方向与裂纹扩展方向一致[[S50,S13,S35]]。
3. **能谱辅助分析**：配备能谱仪（EDS）的扫描电镜可对解理面的第二相、夹杂物进行微区成分分析，明确致脆杂质相[[S24,S51]]。
4. **动态原位观察**：在配有动态拉伸台的扫描电镜上，可实时观测解理裂纹萌生、扩展直至断裂的全过程[[S51]]。

### 样品准备要求

断口样品需保持新鲜无手触痕迹。若表面存在油污、氧化膜，需依次用汽油除油、有机溶剂（乙醇、丙酮等）配合超声波清洗。腐蚀产物需先完成观测分析后再做去除处理。对于完全不导电的断口，需提前喷镀5~10nm厚度的碳膜或金属导电膜[[S24,S2]]。

### X射线衍射测定解理面

X射线衍射（XRD）可测定解理面的晶体学特征。基于布拉格定律nλ=2dsinθ，通过测量入射X射线的衍射角θ计算解理面对应的晶面间距d，将测得的d值与标准PDF卡片比对，即可标定解理面对应的米勒指数{hkl}，确定其所属的晶体学晶面类型[[S62]]。

此外，同步辐射X射线形貌术（SXRT）可用于解理裂纹尖端区域的晶格曲率、局部应变分布的定量观测，获取裂纹扩展路径上的微观晶格变形信息[[S42]]。

## 在压铸中的应用与关联

### 常见出现场景

压铸合金中解理断裂及准解理断裂的出现较为普遍，以下为各压铸合金体系的典型断裂行为：

| 压铸合金 | 断裂特征 | 依据来源 |
|---------|---------|---------|
| AlSi10Mg（亚共晶铝硅合金） | 高浇注温度（645~660°C）下断口以脆性解理面、撕裂岭为主；高快压射速度（5.0m/s）下出现大量解理面、解理台阶，脆性解理断裂占比显著提升 | [[S9,S46]] |
| ADC12铝合金 | 薄壁部位拉伸断口普遍呈现准解理混合断裂特征，可见大量解理平面、撕裂棱与少量韧窝；真空压铸工艺下解理面占比随真空度提升而降低，韧窝深度变大 | [[S49,S58,S8]] |
| A380铝合金 | 未添加改性剂时断口分布大量解理平台和极少量韧窝，以脆性解理断裂为主；添加La+Y后解理平台占比大幅下降，断裂向韧性转变 | [[S30]] |
| 高Fe含量铝合金 | 长杆状富铁金属间相成为裂纹萌生点，断口出现更多脆性解理表面，伸长率随Fe含量升高明显下降 | [[S1]] |
| 压铸镁合金（Mg-Dy-Sr-Nd-Zr） | Dy含量低时以解理台阶和层状分布为主，呈脆性解理断裂；Dy含量增加后解理台阶占比降低，断裂模式从纯解理向准解理过渡 | [[S43]] |
| 半固态压铸ZGK521镁合金 | 相比金属型铸态，断口解理面数量显著减少，撕裂棱增多，伸长率明显改善 | [[S31]] |
| T6热处理压铸镁合金 | 断口呈现短而不连续的河流花样、较多撕裂棱、解理面和台阶，韧窝数量明显减少，为准解理断裂 | [[S26]] |

### 压铸件解理断裂的判定特征

**宏观判定**：断口平齐光亮、无明显塑性变形和颈缩，边缘无剪切唇，裂纹源常位于断面内部，裂纹扩展路径从中心呈径向辐射状分布[[S23]]。

**微观判定**：扫描电镜下可观测到清晰解理台阶、连续或分支的河流花样特征、撕裂棱，部分高占比解理的试样还会出现二次裂纹[[S54,S9]]。

### 影响因素

压铸件中以下因素容易促发解理断裂[[S9,S46,S1,S30]]：

- 粗大针状共晶硅、硬脆非金属夹杂物成为应力集中点
- 气孔、缩松等铸造缺陷引发局部应力集中
- 截面突变引起的应力集中与铸造残余应力
- 不合理的工艺参数（过高的浇注温度、过大的快压射速度）导致晶粒粗化、卷气缺陷增多，断口中解理面占比随之升高

### 失效分析标准流程

压铸件断口分析的标准流程为[[S12,S11,S27]]：宏观形貌观测判断断裂性质与裂纹源位置 → 微观SEM观测解理等特征确定断裂机制 → 必要时配合能谱微区成分分析明确致脆杂质相。

## 影响解理断裂倾向性的因素

综合上述各节信息，影响解理断裂倾向性的核心因素可归纳如下：

| 因素类别 | 影响规律 | 依据来源 |
|---------|---------|---------|
| 环境温度 | 温度降低，解理断裂倾向增大；温度低于韧脆转变温度时优先发生解理 | [[S56,S45,S6]] |
| 应变速率 | 高应变速率/冲击载荷降低解理裂纹扩展应力阈值，增大倾向 | [[S56,S54]] |
| 晶粒尺寸 | 晶粒粗大降低抗解理断裂能力，细化晶粒符合Hall-Petch关系抑制解理 | [[S48,S63]] |
| 第二相与夹杂 | 解理面上第二相粒子作为裂纹萌生核心，大幅提升解理概率 | [[S56,S48]] |
| 杂质与纯净度 | 杂质元素在晶界偏聚降低正断抗力；降低杂质含量可抑制解理 | [[S13,S35,S54]] |
| 缺口与结构突变 | 产生应力集中和三向不等拉伸硬应力状态，显著增大解理倾向 | [[S35,S13]] |
| 应力状态 | 三向拉应力状态下应力状态软性系数α降低，倾向增大 | [[S13,S54,S45]] |
| 材料晶体结构 | BCC、HCP金属易发生；FCC金属在常规条件下极少发生 | [[S28,S54,S21]] |
