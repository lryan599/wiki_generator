## 概述

挤压模芯轴（extrusion die core / extrusion mandrel）是挤压模具中用于形成空心制品内腔轮廓的核心零件，广泛应用于铝及铝合金、铜合金、镁合金等有色金属的热挤压生产。芯轴通过与外部凹模（模孔）配合形成环形挤出间隙，使金属在高温高压下流经该间隙后获得所需的内孔形状与尺寸，是生产无缝或焊合空心型材、管材的关键成形部件。

依据国家标准 GB/T 8845-2017《模具术语》，挤压模芯轴被定义为“用于空心件内部定形的芯模部分”[[S10]]。

## 分类

根据芯轴在挤压过程中的运动状态和装配方式，挤压模芯轴可划分为以下三类[[S4,S10,S30]]：

| 类别 | 特征 | 适用工况 |
|---|---|---|
| **固定芯轴** | 整体安装在芯轴支座上，挤压全程位置保持不变 | 一般空心型材挤压 |
| **浮动芯轴** | 可随金属流动同步发生轴向移动，降低芯轴所受拉载荷，避免断轴 | 无缝管材挤压、高精度内孔要求场景 |
| **穿孔芯轴（穿孔针）** | 用于坯料穿孔工序，从实心铸锭中预先穿透形成内孔通路的轴类零件 | 无缝管材穿孔挤压，配置独立穿孔系统的挤压机 |

## 工艺作用与装配关系

挤压模芯轴的核心工艺作用是直接成形空心零件的内部轮廓，与外部凹模共同挤出符合尺寸要求的管材或空心型材[[S47,S65]]。

在典型的挤压装备系统中，芯轴的装配协同关系为：芯轴后端与芯轴支座连接固定，芯轴支座安装在穿孔压杆上，穿孔压杆从空心挤压轴的内孔中穿过；工作时芯轴头部伸入挤压筒内的坯料中，与模套固定的凹模形成环形挤出缝隙，完成空心制品成形[[S22,S64,S28]]。

## 结构与组成

### 几何特征

**工作带（定径带）**  
工作带是保证挤压制品内腔形状、尺寸精度和表面质量的核心区段，垂直于挤出方向设置。定径带高度依据制品规格调整：小规格挤压机可将芯轴工作带伸出模子定径带 1~3mm，大规格挤压机对应伸出量为 10~12mm[[S55,S10]]。

**脱模斜度**  
热挤压工况下，芯轴工作端通常设计有 10'~3° 的小斜度，部分场景可提升至 5°，以便于芯轴从挤压完成的空心工件中顺利脱出，避免卡滞[[S14]]。

**过渡圆角**  
芯轴各台阶过渡位置须加工成大圆弧过渡结构，避免尖角位置应力集中导致过早断裂；所有工作段与非工作段衔接位置需设置光滑过渡圆角[[S34,S36]]。

**冷却通道**  
大负载长时程挤压工况下，可在芯轴内部加工内置循环冷却通道，通入冷却水或压缩空气带走芯轴接触高温坯料产生的热量，稳定工作温度，延长模具寿命。部分高压铝合金挤压场景可配置液氮冷却系统，将挤压速度提升一倍以上[[S59,S56]]。

### 安装与定位方式

| 定位方式 | 特点 | 适用场景 |
|---|---|---|
| **锥面定位** | 定位精度高，可同时实现密封和自动对中，同轴度控制精度好 | 对管材壁厚均匀性要求高的高精度挤压场景 [[S66,S13]] |
| **螺纹连接** | 拆装便利性高，连接承载力强；带弹性锁紧结构的螺纹连接适配直径大于10mm的中大型芯轴 | 通用挤压场景 [[S49,S17]] |

### 结构选型

根据模芯宽度或直径 b 的取值，芯轴结构选型遵循以下准则[[S54]]：

| 模芯尺寸 | 推荐结构 | 特点 |
|---|---|---|
| b ≤ 10mm | 锥式结构 | 强度刚度好，挤压过程稳定性高，制品壁厚偏差小，但加工难度较高 |
| 10mm < b < 20mm | 锥台式结构 | 强度刚度稍低于锥式，但加工难度降低 |
| b ≥ 20mm | 凸台式结构 | 加工便利性最佳，出口端需对应做一空刀设计 |

芯轴定径带在入口端需伸出 1~2mm（伸出量等于分流桥的弹性下挠量），以避免分流桥弹性下塌低于下模定径带入口；同时在模孔出口端，芯轴定径带应比下模定径带突出 1~2mm[[S54]]。

## 关键材料与性能

### 常用材料

挤压模芯轴主选铬系中合金热作模具钢。材料选择依据挤压产品的形状复杂程度、挤压比和生产批量而定[[S40,S18,S62]]：

| 应用工况 | 推荐芯轴材料 |
|---|---|
| 圆棒、圆管挤压（低载荷场景） | 45钢、5CrNiMo、5CrMnMo、5CrNiW [[S40,S18]] |
| 形状复杂的空心型材、薄壁管材 | 3Cr2W8V 等热作模具钢 [[S18]] |
| 常规有色金属挤压（主选方案） | 4Cr5MoSiV(H11)、4Cr5MoSiV1(H13)、4Cr5MoWSiV；钼系 H10 [[S62,S5]] |
| 高工作温度、大挤压比工况（芯轴头部镶块） | W6Mo5Cr4V2、6W6 或 GH761 高温合金 [[S62,S5]] |
| 批量大、精度要求高的民用建筑铝型材 | 4Cr5MoSiV1(H13) 类 [[S18]] |
| 宽厚比大于50的扁宽薄壁型材、带筋壁板型材 | 更高等级的高热强、高耐磨、高抗蠕变且氮化性能良好的专用模具钢 [[S18,S40]] |
| 超高温、大变形量难变形材料（冷挤压重载场景，批量≥10⁵/5×10⁵件） | YG15、YG20C 硬质合金（抗压强度为淬硬模具钢 1.5 倍以上，弹性模量为模具钢 2 倍以上）[[S26]] |

### 硬度要求

热作挤压模芯轴的常规使用硬度范围为 40~55HRC[[S63]]，精准硬度匹配如下[[S20,S67]]：

| 被挤压材料 | 推荐硬度（HRC） |
|---|---|
| 钢挤压 | 46~50 |
| 铜/钛/镍合金挤压 | 42~50 |
| 铝镁合金挤压 | 48~52 |

复杂形状的芯轴，实际选用硬度需比常规推荐值低 4~5HRC 以避免脆断[[S20,S62]]。

### 核心性能

1. **红硬性**：铬钼系热作模具钢红硬性可达 550~600℃，可在该温度区间长期保持足够高硬度；低合金工具钢红硬性仅为 180~250℃，无法满足热挤压芯轴工作要求[[S63,S35,S1]]。

2. **抗回火性与冲击韧性**：3Cr2W8V 在 580℃左右硬度仍高于 H13，抗回火软化能力更优；但温度超过 600℃后，3Cr2W8V 硬度下降速度显著快于 H13[[S52]]。在冲击韧性方面，H13 冲击韧度从 300℃时的约 10J/cm² 随回火温度升高快速上升至近 70J/cm²，而 3Cr2W8V 始终维持在 9~15J/cm² 的低区间，H13 的冷热疲劳抗力和抗冲击性能远优于 3Cr2W8V[[S15]]。

3. **导热性**：H13 和 4CrMo4 的热导率随温度变化特性存在差异——4CrMo4 热导率整体高于 H13 且随温度升高略有下降，H13 热导率随温度升高缓慢上升。热导率差异直接影响芯轴热疲劳产生机理[[S38]]。

4. **高温强度**：热挤压模具工作环境要求芯轴材料常温下抗拉强度不低于 1300MPa，500~600℃ 高温环境下抗拉强度不低于 1000MPa[[S8]]。

## 工艺参数与操作条件

### 预热与工作温度

| 参数 | 推荐范围 | 说明 |
|---|---|---|
| 预热温度 | 200~300℃ | 保证芯轴初始服役韧性，避免脆性开裂；寒冷工况需进一步提升预热充分度 [[S29,S12]] |
| 中途停工保温温度 | 250~300℃ | 减小热冲击引发的开裂风险 [[S12]] |
| 连续工作温度下限 | ≥250℃ | 保持材料良好使用性能 [[S29]] |
| 最高工作表面温度 | 低于回火温度 | 避免芯轴发生高温回火软化 [[S12,S29]] |

### 冷却与润滑

冷却方式推荐采用压缩空气冷却或循环水冷却，使散热效果与工件加热升温效果达到平衡；禁止采用过强冷却方式或过低的冷却下限温度，防止诱发冷热疲劳裂纹[[S29,S12]]。

润滑方式依据挤压温度与材料选型[[S21,S37,S12]]：

| 工艺类型 | 推荐润滑方案 |
|---|---|
| 常温/低温温热挤压钢件（≤350℃） | 草酸盐处理 + 二硫化钼 + 氯化石蜡 |
| 中温温热挤压（680~800℃） | 906 系列低温玻璃润滑剂 |
| 高温热挤压黑色金属 | 黏度 10~100Pa·s 专用热挤压玻璃润滑剂 |
| 铝、铜合金挤压 | 水基或油基石墨润滑剂 |

需特别注意：过量易燃油基润滑剂会燃烧气化形成高压气流，对芯轴表面产生冲蚀，造成气蚀沟痕，大幅降低芯轴服役寿命[[S12,S42]]。

### 挤压比与坯料温度对寿命的影响

挤压比提升会显著降低芯轴使用寿命。以 3Cr2W8V 钢制芯轴挤压不锈钢管为例：挤压比为 10 时平均寿命为 70 件；挤压比提升至 18 时降至 50 件；挤压比达 35 时仅为 20 件[[S5,S39]]。在相同挤压比下，被挤压坯料温度越高、变形抗力越大，芯轴寿命明显下降。例如 3Cr2W8V 钢制硬度 51~53HRC 的模具，挤压纯铜管平均寿命超 300 件，挤压黄铜管为 200 件，挤压含镍 30% 的白铜管不足 100 件[[S5,S39]]。

## 失效机制

挤压模芯轴在高温、高压和剧烈摩擦耦合条件下服役，主要失效形式包括以下五类：

| 失效类型 | 产生机制 | 典型表现 |
|---|---|---|
| **热疲劳龟裂** | 表面反复接触高温坯料与冷却介质交替循环，表层升温膨胀受内层约束产生压应力，降温收缩产生拉应力；循环交变热应力超过材料屈服强度后积累塑性应变，在刀痕、磨痕等应力集中处萌生网状或放射状微裂纹 [[S33,S57]] | 表面网状/放射状裂纹 |
| **磨损** | 芯轴与高温变形坯料之间的剧烈相对滑动摩擦伴随高温氧化，氧化皮硬颗粒引起磨粒磨损，产生黏着磨损、氧化磨损和磨粒磨损多重复合作用 [[S33,S57]] | 工作带尺寸超差、表面粗糙度急剧上升 |
| **塑性变形** | 长时间工作温度过高导致材料表面回火软化，屈服强度大幅下降，在低于额定工作载荷的应力下发生局部镦粗、缩颈、工作带塌陷 [[S23,S51]] | 管材尺寸精度超差、形状畸变 |
| **断裂** | 早期脆断多由偏载过大、预热不充分、硬度过高、材料冶金缺陷或表面应力集中导致（寿命几十至数百次）；热机械疲劳断裂由循环交变载荷与热载荷共同作用，疲劳裂纹逐步扩展后失稳断裂 [[S42,S24]] | 整体断裂 |
| **冲蚀** | 过量油基润滑剂燃烧产生高压高速气流，持续冲刷芯轴工作表面形成宽而深的气蚀沟痕 [[S42,S12]] | 表面气蚀沟痕 |

## 强度校核与设计准则

### 压应力与稳定性校核

挤压模芯轴的压应力强度校核公式为[[S48]]：

$$d_{min} \geq \sqrt{\frac{4P_{0}}{\pi [\sigma_{压}]}}$$

式中 $d_{min}$——最小断面直径（mm）；$P_0$——挤铸力（N）；$[\sigma_{压}]$——材料抗压许用压力。以 H13 为例，$[\sigma_{压}]$ 取 590MPa。

纵向弯曲应力校核公式为[[S48]]：

$$L_{max} \leq 91.8 \times \frac{d_{min}^2}{\sqrt{P_0}}$$

式中 $L_{max}$——芯轴最大自由长度（mm），用于校核细长芯轴挤压过程中失稳弯曲的风险。

### 分流桥-模芯组合结构强度校核

分流组合模的分流桥-模芯组合结构需进行抗弯和抗剪强度校核[[S9,S54]]：

抗弯强度校核公式：
$$h = L \sqrt{\frac{P}{2 [\sigma]}}$$

式中 $h$——分流桥危险断面临界高度（mm）；$L$——分流桥两桥墩间距（mm）；$P$——作用在挤压垫上的单位比压（MPa）；$[\sigma]$——模具材料许用弯曲应力。3Cr2W8V或4Cr5MoV1Si钢在400℃环境下取 1000MPa。

抗剪强度校核公式：
$$\tau = \frac{Q}{A} \leq [\tau]$$

式中 $\tau$——剪应力（MPa）；$Q$——分流桥端面承受的总压力（N）；$A$——受剪总面积（mm²）。4Cr5MoV1Si钢在400℃环境下许用剪切应力取 500MPa。

### 尺寸公差设计准则

中国 GB 体系中，挤压模芯轴的尺寸公差、形位公差设计需遵循 GB/T 1800.1~1800.4 极限与配合系列标准、GB/T 1804 未注公差标准、GB/T 1182 形位公差通则、GB/T 1184 未注形位公差值等基础标准；模具轴类通用零件的标准尺寸系列参考 GB/T 2822 优先数和模具行业指导文件《模具用标准尺寸》中轴类零件规格[[S58,S7,S46]]。

ISO 标准体系由 ISO/TC29/SC8 分委会组织制定，ISO 12165《压塑、注塑、压铸模具零件 术语和符号》对轴类芯件进行了统一术语定义与标准化要求[[S32,S58]]。DIN 标准体系由 NA121“模具和合模装置”委员会负责，涉及热挤压芯轴类构件的设计要求可参考 DIN ISO 24233 压铸成型模具说明书标准[[S32]]。

## 应用场景

### 铝合金挤压

挤压模芯轴是铝型材热挤压工艺的核心零件，广泛应用于分流组合模、舌形模结构中控制空心型材/管材的内孔形状与尺寸。适配铝合金坯料 300~595℃ 的挤压工艺温度，可生产挤压铝圆管规格覆盖 φ20mm×1.5mm~φ625mm×15mm，最薄壁厚可达 0.38mm，壁厚公差精度可达 ±0.07mm 级别[[S43,S16]]。

### 铜合金挤压

铜及铜合金挤压坯料温度区间为 595~995℃，挤压压力普遍高于 690MPa。芯轴需选用镍基或铁基耐热合金等高耐热、高耐磨材料，适配管材、线材、电气接插件等铜合金挤压制品的生产需求[[S11]]。

### 其他有色金属挤压

芯轴可适配镁、锌、铅等低熔点有色金属的热挤压生产。镁合金挤压坯料温度与铝合金接近，锌合金挤压坯料温度区间为 205~345℃，芯轴服役压力低于铝和铜合金[[S11]]。

### 挤压铸造/广义压铸场景

挤压模芯轴在调整强度校核参数、更换对应耐高温材料后可直接用于挤压铸造场景。典型的铜合金阀体挤铸模已集成对应结构芯轴作为内孔成型零件，但需按挤压铸造的二次加压力参数重新校核压应力和纵向弯曲强度，调整许用应力取值[[S48,S61]]。

## 与相关术语的区别

挤压模芯轴与普通压铸模芯销（die casting core pin）、穿孔针（piercer/piercing mandrel）在结构、功能和使用场景上存在本质差异[[S25,S41,S44,S43]]：

| 对比维度 | 挤压模芯轴 | 压铸模芯销 | 穿孔针 |
|---|---|---|---|
