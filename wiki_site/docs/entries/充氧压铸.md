## 概述

充氧压铸是2011年公布的材料科学技术正式名词[[S16]]。其别称包括无气孔压铸、加氧压铸、氧气压铸，对应的英文名称为pore-free die casting、oxygenated die casting，商业上称为POCAST法或Pore2Free Die Casting Process（P.F法）[[S16, S20, S19]]。该工艺的核心定义为：在金属液充填型腔前，往型腔和压室内充入干燥的氧气以置换原有空气的压铸工艺。充型过程中未排出的氧气与高温活性金属液反应生成粒径1 μm以下的弥散金属氧化物微粒，该类微粒总质量占铸件总质量的0.1%~0.2%，不会对铸件力学性能产生负面影响，可从根源消除或减少气孔缺陷，提升铸件致密性，使铸件支持热处理强化与焊接加工[[S27,S6,S21,S2]]。

充氧压铸技术发展关键时间节点：1969年美国爱列克斯公司首先正式提出充氧压铸工艺方法[[S8,S2,S41]]；20世纪70年代早期，加拿大率先通过分析锌合金压铸件气孔内气体成分提出了无气孔压铸的核心概念，同期日、美、苏技术人员同步开展试验研究，国际铅锌研究组织（ILZRO）获得相关美国专利3382910号[[S8,S2,S41]]；1970年Fuso轻合金有限公司与日本轻金属株式会社启动联合研究工作，1972年首次实现可热处理摩托车轮毂批量量产；到1980年，充氧压铸件的年产能已达到1500吨[[S8,S2,S41]]。

POCAST工艺及Pore2Free（P.F法）的研发主导主体为日本轻金属株式会社，属于该企业主导开发的专利技术。日本轻金属后续与国际铅锌研究组织（ILZRO）签订了交叉许可协议，共享无松孔压铸技术的研发成果。截至20世纪80年代，日本境内已有11家该工艺的商业授权使用方，美国境内有1家商业授权方[[S7,S2,S16]]。

## 工艺原理与化学机制

### 消除气孔的机制对比

普通压铸未对模具型腔及压室内的原有空气做特殊干预，金属液高速充型过程中极易卷入空气形成气孔缺陷，铸件无法进行热处理，服役温度上限低，气密性表现差[[S32,S13]]。

真空压铸消气孔的底层逻辑为在金属液充填型腔前通过抽真空装置直接将型腔内大部分空气向外排出，降低充型时的卷气概率以减少内部气孔[[S23,S21,S13]]。

充氧压铸消气孔的底层逻辑区别于上述两种工艺：通过用氧气完全置换型腔内空气，再利用氧气与高温金属液的化学反应直接消耗掉腔内剩余全部气体，反应生成的弥散氧化物不会形成气体空腔，从根源上消除气孔缺陷[[S42,S22,S32]]。

| 对比维度 | 普通压铸 | 真空压铸 | 充氧压铸 |
|----------|----------|----------|----------|
| 消气孔机制 | 无特殊干预，气体被卷入金属液形成气孔 | 通过外置真空系统抽出型腔内空气 | 用氧气置换空气，经化学反应消耗剩余氧气生成固体氧化物 |
| 热处理兼容性 | 不可热处理（加热致气孔鼓泡） | 可热处理 | 可热处理 |
| 设备复杂性 | 简单 | 需模具密封结构及真空系统，结构复杂 | 仅需充氧控制装置，结构简单 |
| 投资成本 | 低 | 高 | 远低于真空压铸 |
| 生产效率 | 基准 | 较普通压铸提升10%~20%（可提前开模） | 需额外3~6 s充氧工序，略低于真空压铸 |

### 氧气置换与氧化物弥散填充原理

压铸前将干燥氧气充入压室和压铸模型腔，驱赶出原有的空气（其中氮气约占80%，氢气来自涂料挥发），最大程度用氧气完全置换原有混合气体。充氧完成后在金属液填充型腔过程中，部分氧气从排气系统直接排出，残留氧气与高温活性金属液快速发生氧化反应，消耗氧气后型腔内部形成局部瞬时真空，进一步消除气孔生成的气体条件[[S33,S40,S2]]。

以铝合金充氧压铸为例，化学反应方程式为：

\[
4\mathrm{Al} + 3\mathrm{O}_2 = 2\mathrm{Al}_2\mathrm{O}_3
\]

生成的Al₂O₃微粒尺寸小于1 μm，弥散分布在铸件基体中，总质量仅占铸件总质量的0.1%~0.2%，不会对铸件力学性能和机加工性能造成负面影响。原本空气占据的气孔空间完全被这些细小固体氧化物填充，从而消除气孔缺陷，铸件可实现热处理不鼓泡[[S15,S43,S2]]。

## 工艺过程与装置结构

### 标准操作时序

充氧压铸标准操作时序如下[[S22,S42,S4,S24]]：

- 合模至动定模分型面剩余3~5 mm间隙时启动充氧；
- 短暂停留1~2 s后继续合模；
- 合模完成后持续充氧一段时间；
- 充氧结束后立即进行浇注与压射作业；
- 总充氧时长根据铸件大小、复杂程度和充氧孔位置确定，常规取值范围为3~6 s，部分工艺区间为3~7 s。

### 装置构成

充氧压铸装置主要由氧气瓶、压力表、干燥器、电磁阀、节流阀、连接管路、动定模及压射冲头等核心部件构成[[S18,S38,S21]]。氧气经由干燥器去除水分后，通过电磁阀与节流阀控制进入型腔的流量和压力。

充氧方法主要有两种[[S11,S42,S3]]：
- **压室加氧**：立式冷压室压铸机多从反料冲头处充氧，结构简单、密封可靠，适合中心浇口铸件；卧式冷压室压铸机在压室中金属液与氧气接触面积大，容易氧化，优先适配立式压铸机[[S42,S22]]。
- **模具上加氧**：在压铸模上开设专用充氧孔将氧气注入型腔，对排除铸件气孔缺陷具有较大的针对性。

## 关键工艺参数与控制要求

| 参数类别 | 推荐范围/要求 | 说明 |
|----------|--------------|------|
| 充氧总时长 | 3~6 s（部分工艺3~7 s） | 依据铸件大小、复杂程度及充氧孔位置确定 |
| 充氧起始时机 | 动定模分型面间隙3~5 mm时 | 停留1~2 s后再合模，合模后继续充氧 |
| 充氧压力 | 0.3~0.7 MPa（常用0.4~0.7 MPa） | 确保充足氧气流量；部分手册推荐0.3~0.5 MPa[[S12,S42,S24]] |
| 压射速度与压射比压 | 与普通压铸相同 | 无需额外调整[[S42,S4]] |
| 模具预热温度 | 约250 ℃ | 略高于普通压铸，以加速涂料残留气体挥发排出，避免残余氧气聚集形成氧气孔[[S22,S44,S10]] |
| 涂料 | 仅可使用无挥发气体的非油脂类涂料（如胶体石墨水剂） | 禁止使用含油脂类涂料，否则析出氢气在高温下引发爆炸[[S17]] |

## 模具设计要点

充氧压铸模具设计需满足以下特殊要求[[S14,S3,S42]]：

- **内浇口截面积**：应适当大于普通压铸设计值，保障氧气在充氧阶段可顺畅通过内浇口完成压室和全型腔的气体置换。
- **充氧口位置**：应设置在距离浇注系统最远、型腔最深处，条件允许可在多个死角位置增设充氧口，消除氧气无法到达的“死点”。
- **排气槽布置**：需对应布置在金属液最后充填的位置，与充氧口位置形成气流对冲路径，保证原有空气可被完全排出。
- **充氧通道封堵结构**：必须设计可自动封闭的液压或机械封堵结构，压铸作业前需完全封堵通道，防止合金液灌入充氧管路。

## 优缺点与局限性

### 优势

- **消除气孔缺陷，提升力学性能**：充氧压铸所得铝合金压铸件铸态强度较普通压铸提升约10%，延伸率提升50%~100%；经热处理后强度可再提升30%以上，屈服强度可提升100%；可在200~320 ℃环境中正常服役[[S42,S22,S32]]。同参数条件下（40 cm/s压射速度），充氧压铸铝合金件抗拉强度可达277 MPa、延伸率1.5%，而同参数普通压铸仅为242 MPa、延伸率1.1%[[S11]]。
- **可热处理与可焊接**：铸件无皮下气孔，支持T6等热处理强化工艺不鼓泡[[S31,S35,S33]]。
- **设备简单，投资低**：与真空压铸相比，无需复杂模具密封结构与专用真空系统，整体结构简单、操作便捷，初期设备投入仅为真空压铸的数分之一[[S40,S42]]。
- **降低模具载荷**：充氧过程型内反压力大幅降低，金属液对模具的冲击载荷减小，对模具维护要求更低，可间接延长模具使用寿命[[S11]]。

### 局限性

- **生产成本增加**：需额外配置充氧控制装置，生产过程消耗氧气，会延长压铸生产循环时间，相较普通压铸单件生产成本提升10%~15%[[S42]]。
- **卧式压铸机适配性差**：卧式压铸机采用充氧压铸时，压室中金属液与氧气接触面积大，易出现额外氧化缺陷，优先适配立式压铸机[[S42,S22]]。
- **型腔结构敏感性**：充氧效果受型腔结构影响大，若充氧孔位置设计不合理，型腔会存在氧气无法完全置换空气的"死点"，导致局部残留气孔缺陷[[S25]]。
- **涂料选择受限**：无法使用常规含油脂类涂料，仅可使用胶体石墨水剂等无挥发气体涂料，易引入石墨夹杂缺陷影响铸件性能[[S17]]。
- **国内应用基础薄弱**：国内现有充氧压铸的研究和工艺积累不足，暂未形成大规模量产的成熟体系，与国外主流应用水平存在明显差距[[S30]]。

### 与真空压铸的综合对比

| 对比维度 | 充氧压铸 | 真空压铸 |
|----------|----------|----------|
| 消气孔机制 | 氧化反应原位消耗型内气体，生成瞬时真空 | 外置抽气系统直接排出型内气体 |
| 热处理兼容性 | 可热处理 | 可热处理 |
| 消气孔效果 | 铸件无气孔，致密度高 | 铸件低孔隙率，致密度高 |
| 设备投资 | 仅为真空压铸的数分之一 | 需要复杂密封结构与专用真空系统 |
| 模具维护 | 要求较低 | 要求较高 |
| 生产效率 | 需额外3~6 s充氧工序，效率略低 | 可提前开模，效率较普通压铸提升10%~20% |
| 单件成本 | 较普通压铸高10%~15% | 设备改造成本高 |

## 适用材料与典型产品

### 适用合金

充氧压铸的基础适配合金为常见压铸铝合金，对铝合金成分烧损程度极低，不会影响合金原有性能[[S42,S15]]。此外，该工艺可适配AZ91镁合金、AM60镁合金。日本轻金属株式会社已实现该两类镁合金的充氧压铸量产，铸件可正常完成热处理强化，力学性能优于同参数普通镁合金压铸件[[S16]]。

### 典型产品

日本轻金属株式会社于1983年起采用充氧压铸大批量生产轿车铝合金轮毂，1986年该工艺生产的铝轮毂年产量达70万个，占日本当年全部铝轮毂产量的10%。与传统工艺生产的铝轮毂相比，质量降低15%，机加工余量从2~3 mm缩减至0.75 mm，轮毂售价降低10%[[S30]]。美国铸锻公司于1982年起采用充氧压铸生产汽车铝合金轮毂及相关零件，替代原低压铸造工艺，铝轮毂质量进一步降低18%[[S30]]。

日本轻金属株式会社采用充氧压铸量产AZ91镁合金计算机整体磁头支架，替代原有多层叠合支架结构，大幅降低零件总重；同时批量生产AM60镁合金汽车和摩托车轮毂，相较同规格铝合金轮毂减重15%[[S16]]。

已实现商业化量产的充氧压铸零件还包括[[S11,S30,S22]]：汽车用制动器缸体、连杆、齿轮齿条转向机构外壳、进气集合管、液压变速器壳体、提升门铰链、耐高温液压阀体、计算机托架、热交换器等。
