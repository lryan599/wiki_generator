## 概述

砂芯涂料表干炉是砂芯制备流程中的关键设备，用于在砂芯完成耐火涂料涂覆后，通过控温、控风实现仅快速干燥涂料表层，同时保留砂芯内部适当湿度的专用炉。它在制芯工序之后、完全烘干/组芯工序之前运行，属于砂型（芯）烘干专用铸造设备[[S1,S14,S24]]。

国家标准 GB/T 25370-2020《铸造机械 术语》中明确定义：**砂芯涂料表干炉**为专门用于砂芯表面涂料干燥的烘干炉[[S14]]。其核心工艺目的是使涂层获得足够的表面强度，以支撑砂芯在后续搬运、下芯及组芯过程中不发生流淌、脱落，同时避免将砂芯本体完全烘干，从而保留砂芯原有的强度与退让性[[S5,S15]]。

## 分类体系

砂芯涂料表干炉按结构形式、加热方式及操作方式可形成以下分类体系[[S3,S21,S32,S42,S43,S47]]：

| 分类维度 | 类型 | 特征 | 适用场景 |
|----------|------|------|----------|
| **结构形式** | 箱式（室式） | 间歇作业封闭炉体，配台车装载工件，结构简单，装载量大 | 单件小批量、大尺寸砂芯 |
| | 隧道式（水平连续） | 砂芯随输送链/台车依次通过预热、表干、冷却工位，运行速度可调 | 汽车零部件等大批量连续生产 |
| | 垂直连续式 | 砂芯随垂直循环吊篮上下运动，占地面积小 | 中、小型砂芯大规模量产 |
| **加热方式** | 电加热（含远红外） | 电阻丝/远红外元件为热源，温度均匀易控，节能20%-40% | 各类中小型砂芯 |
| | 燃气/燃油 | 燃烧热气流经换热后送入炉内，热效率高，运行成本低 | 大尺寸砂芯和高产能生产线 |
| | 热风循环（强制对流） | 循环风机驱动热风经静压风箱、高速风嘴射向砂芯表面，是目前主流类型 | 表层干燥均匀性要求高的工况 |
| | 红外辐射 | 红外辐射直接作用于涂层水分子，干燥速度快 | 涂层薄、体积小的精密砂芯 |
| **操作方式** | 间歇式 | 单批次装料后密闭运行 | 小批量、多品种砂芯生产 |
| | 连续式 | 砂芯连续进出炉，工位分区独立控温 | 大批量流水线生产 |

## 工作原理与工艺机制

### 热风循环干燥原理

砂芯涂料表干炉的主流技术路径为强制对流热风循环干燥。其循环实现原理为：助燃风与燃气在烧嘴内混合后进入燃烧/换热室完成热交换，经循环风机送入送风室、静压风箱，由高速风嘴阵列喷射至砂芯表面进行对流换热；换热后的气体一部分外排，剩余部分经回风通道返回燃烧/换热室重新参与循环[[S19,S43]]。

![砂芯涂料表干炉强制对流热风循环工作原理图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/38a6cb46-ade1-4e9f-b93e-1f3070757dcc/resource)

### 表干选择性脱水机制

"表干"的工艺本质是通过三段式温控配合精准热风速度和短干燥时间，实现**仅使涂料表层快速脱水，而不全面烘干砂芯本体**。其控制机理如下[[S4,S6,S18,S19]]：

1. **预热段**（约100℃）：保证砂芯内外层均匀受热，防止内外温差过大导致涂料表层急骤失水开裂，同时炉内保持较高湿度，抑制水分向砂芯内部迁移。
2. **表干段（高温段，120-180℃）**：采用定向高速热风（平均风速约15 m/s）直接作用于砂芯表面，快速蒸发涂层水分，同时限定干燥时间，避免热量深度传入砂芯内部。
3. **冷却段**：引入冷风快速降低砂芯出炉温度，砂芯表层最终残余含水率控制在约0.5%左右，内部保留适量水分以维持砂芯原有强度。

水分的迁移与蒸发过程分为三个子过程：表面涂料水分汽化→蒸汽在紊流热风中向四周扩散→内部一定深度水分向表层迁移[[S43]]。

## 主要结构与组成

砂芯涂料表干炉的典型结构分为**温度控制系统**和**热风循环系统**两大核心模块[[S1,S28]]。主要组成部件包括：

- **助燃风机与烧嘴**：提供燃气/助燃风混合并燃烧热源
- **燃烧/换热室**：完成热交换的核心热区
- **循环风机**：热风循环的动力部件
- **送风室与静压风箱**：均压送风结构，配置导流板以提升流场均匀性
- **高速风嘴**：阵列式金属风嘴，安装于静压风箱出风口侧，定向喷射热风至砂芯表面
- **排湿与通风装置**：烟道系统，配合闸门调节排湿量
- **输送机构**：辊道式（连续通过式炉）、链式/垂直吊篮式（立式炉）、台车型（室式炉）[[S47]]
- **温度传感器**：送风/回风温度监测
- **耐热内衬**：高孔隙率全纤维耐火保温内衬，炉壁隔热层[[S10]]

### 热风送风子系统

热风循环系统中配置多组阵列式金属高速风嘴，安装于静压风箱出风口侧，可将循环热风定向喷射至砂芯表面，强化对流传热效果[[S43,S45]]。

### 工艺布局

典型砂芯涂料表干炉沿砂芯输送方向划分为三个功能区[[S4,S11,S19]]：

- **预热段**（1段）：温度约100℃，避免涂料层急骤失水开裂
- **高温段**（1~3段）：独立控温，完成涂料层的彻底烘干
- **冷却段**（1段）：冷风降温至安全出炉温度

### 输送机构与炉型匹配

| 炉型 | 输送机构类型 | 适用对象 |
|------|-------------|----------|
| 连续通过式（直通式） | 辊道式 | 大批量中小砂芯流水线生产 |
| 立式烘炉 | 链式/垂直吊篮式 | 中、小砂芯大规模连续生产 |
| 室式（箱式） | 台车型 | 大尺寸、重型异形砂芯 |

## 关键工艺参数

### 核心指标

根据对砂芯涂料表干炉运行状态的实测研究，优化后的核心工艺参数应满足以下指标[[S6,S7,S30]]：

| 参数 | 改进前工况 | 运行要求 | 优化后达到 |
|------|-----------|---------|-----------|
| 热风温度均匀性 | ±3℃ | ≥±1.5℃ | ±1.3℃ |
| 出风口热风速度 | 平均15 m/s | ≥17 m/s | 17.7 m/s |
| 高温段送风与回风温差 | 平均19.7℃ | ≥25℃ | 25.5℃ |
| 砂芯表面平均含水率 | 0.45% | ≤0.3% | 0.18% |

### 不同涂料体系的工艺窗口

水基涂料砂芯的典型表干工艺窗口如下[[S47]]：

| 砂芯类型 | 工艺条件 | 总干燥时长 |
|----------|---------|-----------|
| 小体积油类粘结剂砂芯 | 梯度升温→保温→降温 | 约2.5 h |
| 发动机缸体类复杂砂芯 | 170℃±10℃保温 | 3.5~4 h |

## 工艺缺陷及其对铸件质量的影响

### 表干不充分

砂芯表干不足时，涂层残留未蒸发水分，导致涂层发黏、附着力不足脱落[[S24,S25,S36]]。浇注时，涂层含水率过高会瞬间汽化产生大量侵入性气体，引发铸件产生**气孔**及**粘砂**缺陷[[S2,S29]]。

> 干型或表面干燥型的干燥层中残余水分含量应小于0.5%[[S29]]。

### 表干过度

表干过度——热风温度超标或烘干时间过长——将导致砂芯表层过度失水脆化，砂芯整体溃散性下降，使后续落砂清理难度大幅提升[[S1,S23,S39]]。此外，涂层因干裂产生破损，浇注时局部集气超出砂型排气能力，同样会在铸件截面上造成大面积气孔缺陷[[S2]]。

## 选用与设计要点

### 炉型匹配原则

选用砂芯涂料表干炉时需综合考虑[[S36,S13]]：

- **砂芯尺寸与形状**：决定输送机构类型（辊道/链式/台车）及送风布局
- **涂料类型**：水基涂料需配置更充足的排湿风量；醇基涂料需强化防爆设计与VOC废气收集
- **生产线节拍**：炉型线速度和总产能须与上游制芯、下游浇注工位严格匹配

### 能效优化设计

行业共识的能效优化方案包括[[S30,S10]]：

| 优化措施 | 原理 |
|----------|------|
| 连续式比值燃烧控温 | 替代传统大小火脉冲燃烧，提高控温精度 |
| 静压风箱增加导流结构 | 提升流场均匀度，降低无效热风功耗 |
| 炉体出入口配置空气幕（10-20 m/s） | 减少热泄漏 |
| 高孔隙率全纤维耐火保温内衬 | 降低炉壁散热损失 |

### 废气处理设计

砂芯表干过程中排出的含溶剂废气浓度需严格控制在爆炸下限以下，可采用活性炭吸附、冷凝回收或RTO焚烧工艺进行达标处理。同时需配套局部补风系统，保证炉内微负压，防止无组织逸散[[S36,S41]]。

## 常见问题与局限性

### 设备常见故障

日常维护需重点检查以下项目[[S42,S47]]：

- 炉体炉衬无裂纹、无倒塌等损坏
- 炉门启闭灵活、无变形
- 烟道通畅无堵塞
- 加热元件及电气装置通电安全可靠
- 运行风机无异响
- 测温模块定期校准

### 水基涂料体系的应用限制

水基砂芯涂料配套表干炉存在固有的应用局限性：能源消耗高、占用车间场地面积大；表干工序耗时过长会制约砂芯生产线整体节拍；涂层内部残留的隐性水分仍可能引发铸件气孔缺陷[[S36]]。

## 上下游配套与产线衔接

砂芯涂料表干炉在铸造自动化产线中处于中间衔接位置[[S14,S26]]：

- **上游输入**：砂芯涂料混合搅拌设备 → 浸涂/喷涂/淋涂施涂设备 → 砂芯转运机器人
- **下游输出**：表干合格砂芯 → 组芯工序（部分可直通下芯），或进入砂芯完全烘干炉 → 下芯工序

表干工序作为中间预处理环节，可大幅降低后续全烘工序的能耗和时长[[S5,S15]]。
