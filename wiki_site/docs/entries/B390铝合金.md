## 概述
B390铝合金（ASTM标准牌号B390.0，UNS编号A03900）是一种过共晶Al-Si-Cu-Mg系压铸铝合金，硅含量16.0~18.0%，铜含量4.0~5.0%，属于可热处理强化的铸造铝合金[[S21,S14]]。该合金在ASTM B85标准中正式收录，常见供货状态包括铸态（F）、T5和T6，状态代号符合ANSI H35.1规范[[S10,S7]]。B390.0专为压铸工艺开发，而主合金元素区间相近的A390.0（UNS A13900）则主要面向砂型铸造和金属型铸造[[S3]]。  

图1为铝及铝合金分类图，明确将铸造铝合金分为非热处理型和热处理型，B390属于热处理型铸造铝合金，支撑其基本分类属性。

![图1 铝及铝合金分类图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/0696a0b3-2a7b-4899-9af2-8c6ec249e89f/resource)

## 化学成分
B390.0的标准化学成分范围主要依据ASTM B85‑96（牌号SC174A，UNS A03900）[[S15]]，并参考《铝及铝合金材料手册》[[S9]]，见表2。

**表2 B390.0 铝合金化学成分（质量分数/%）** [[S15,S9]]

| 元素 | 含量范围 |
|------|----------|
| Si | 16.0 ~ 18.0 |
| Fe | ≤ 1.3 |
| Cu | 4.0 ~ 5.0 |
| Mn | ≤ 0.10 |
| Mg | 0.45 ~ 0.65 |
| Zn | ≤ 0.10 |
| Ti | ≤ 0.20 |
| 其他每种杂质 | ≤ 0.10 |
| 杂质总和 | ≤ 0.20 |
| Al | 余量 |

B390.1为同一合金族变体，主合金元素区间一致，但其在凝固过程中额外析出Al₁₅相，相组成与基础390.0存在差异[[S13]]。

## 物理性能
B390铝合金在20°C下的密度为2.73 g/cm³[[S9]]。关键热物理参数与弹性参数如下[[S9]]：

| 性能项目 | 数值 |
|----------|------|
| 液相线温度 | 650 °C |
| 固相线温度 | 505 °C |
| 20–100 °C线热膨胀系数 | 18.0 μm/(m·K) |
| 拉伸弹性模量 | 81.2 GPa |
| 压缩弹性模量 | 82.8 GPa |

低热膨胀系数主要归因于组织中高达16~18%的初晶硅相，使其在温升工况下尺寸稳定性明显优于常规低硅压铸铝合金[[S9,S24]]。

## 力学性能
B390.0属脆性断裂控制材料，多数工况下断裂前无法达到0.2%永久变形，其0.2%屈服强度通常与抗拉强度相等[[S9]]。标准测试条件如下：  
- 砂型/永久型铸件及高密度精密压铸件采用直径12.7 mm标准试样；  
- 普通压铸件采用直径6.4 mm试样，试验时不加工表面；  
- 布氏硬度测试载荷500 kg，钢球直径10 mm；  
- 疲劳试验采用R.R.Moore旋转梁法，循环次数5×10⁸[[S9]]。

不同铸造工艺与热处理状态下的典型延伸率（标距50 mm）见表3。

**表3 B390.0合金典型延伸率** [[S9]]

| 工艺/状态 | 延伸率（%） |
|-----------|------------|
| 普通压铸（F、T5） | 1.0 |
| 高速高密度精密压铸（F、T5） | 1.0 |
| 高速高密度精密压铸（T6、T7） | < 1.0 |
| 砂型铸造（全部状态） | < 1.0 |
| 永久型铸造（F、T5） | 1.0 |
| 永久型铸造（T6、T7） | < 1.0 |

> T5 — 自热加工冷却后人工时效；T6 — 固溶处理+人工时效；T7 — 固溶处理+稳定化处理，均遵循ANSI H35.1[[S10]]。室温拉伸强度、屈服强度及硬度具体数值可查阅《铝及铝合金材料手册》表6‑44[[S9]]。

## 工艺特性

### 压铸工艺要点
B390.0高硅合金对H13压铸模具存在明显磨蚀效应，且浇注温度升高会显著加剧熔体对模具钢的侵蚀[[S25]]。图2展示了不同合金浇注温度与模具侵蚀度的关系曲线，为选择安全浇注温度窗口提供直接依据。

![浇注温度与模具侵蚀度关系图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/efe7f0a6-e817-40af-94b3-2a08363702d2/resource)

压铸模浇道系统的参考参数列于表4，可用于浇口初步设计[[S23]]。

**表4 B390压铸模浇道设计参考值** [[S23]]

| 参数 | 数值 |
|------|------|
| 模具温度 | 250 °C |
| 浇口厚度 | 0.70 mm |

此外，模具温度与压铸件线收缩率呈正相关，随模具温度升高，铸件最终冷却收缩率逐步增大并趋近峰值，在精密尺寸控制时需予以补偿[[S5]]。

### 半固态与挤压铸造
对于要求更高致密度的B390零件，可采用挤压铸造或半固态成形工艺。多阶段压力控制（升液、低压保压、高压升压、保压、泄压）能有效改善补缩、减少缩孔缺陷[[S17]]。挤压铸造已被引入铝合金活塞、空调压缩机缸体等高耐磨件的生产中，可进一步提升部件服役寿命[[S16]]。

## 组织与相组成
B390合金铸态组织包含初晶硅、共晶硅、大量Al₂Cu相、Mg₂Si相以及Al-Fe-Si金属间化合物（如Al₅FeSi等）[[S1,S4]]。由于硅含量更高（16~18%），B390中初晶硅形核密度较390.0更高，在0.5~4 K/s的常规压铸冷却速率范围内初晶硅尺寸未发生明显粗化；其与390.0的另一个显著差异是共晶反应中会同步形成Al₁₅相[[S13]]。弥散分布的初晶硅硬质点赋予合金优异的耐磨性能，但同时导致机加工刀具磨损严重，必须采用PCD/SCD等超硬刀具并合理选择切削速度以保证表面质量[[S18]]。

## 应用领域
B390铝合金最具代表性的应用是无缸套高硅铝合金发动机缸体。通用汽车、保时捷等企业已在量产机型中采用390系列合金通过高压压铸直接成型缸体，从而取消缸套装配工序，实现轻量化[[S19,S22]]。图3为标注“390 ALLOY”的高压压铸缸体实物，直观展示了该合金在动力总成中的使用形态。

## 与相关合金的关系

### B390.0与A390.0
B390.0与A390.0的主合金元素区间接近（均约17%Si、4.5%Cu、0.6%Mg），但二者工艺定位完全不同[[S3,S9]]：
- B390.0（UNS A03900）为压铸牌号，成分和杂质上限依据ASTM B85压铸标准控制；
- A390.0（UNS A13900）专用于砂型铸造和金属型（永久型）铸造，铁含量等杂质管控适配重力浇注需求。

### B390/A390与ADC14
ADC14是日本JIS标准下与B390.0对应的牌号（见表1），但其Si含量仅为9.6~12%（近共晶范围），远低于B390的16~18%[[S11,S20]]。表5对比了这些合金的关键特性差异[[S20,S11]]。

**表5 B390/A390与ADC14特性对比** [[S20,S11]]

| 特性 | B390 / A390（过共晶，约17%Si） | ADC14（近共晶，约11%Si） |
|------|-------------------------------|--------------------------|
| 耐磨性 | 优（初晶硅硬质点） | 中 |
| 高温强度 | 高 | 中 |
| 热膨胀系数 | 低 | 高 |
| 耐热裂性 | 优 | 较差 |
| 铸造流动性 | 一般 | 好 |
| 典型应用 | 缸体、活塞、涡旋盘 | 一般结构件、盖类 |
