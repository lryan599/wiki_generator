## 概述

缩裂（shinkage crack），亦称收缩裂纹，是压铸件在凝固阶段因补缩不足、在收缩应力作用下于热节或最后凝固部位由缩孔/缩松衍生出的线性断续裂纹缺陷，断口常伴随树枝状结晶特征，部分断口存在轻微氧化痕迹[[S25,S20]]。

在铸造缺陷体系中，缩裂归属于**收缩缺陷**大类下的连续性裂纹子类，与归属于冶金缺陷大类的热裂在分类层级上不同[[S19,S41]]。该术语在国内外标准与权威手册中通用，shrinkage crack、shrinkage cracking、contraction crack、solidification crack、shrinkage fissure 均为其规范英文别名，其中 shrinkage crack 在国内机械制造标准汇编中作为正式术语条目收录[[S3,S7]]。

## 核心特征

### 形貌与分布

压铸缩裂常呈分支状、不规则裂纹，典型分布位置为铸件热节区域、厚壁与薄壁过渡区、浇口附近。裂纹可为断续线性延伸，断面存在清晰的树枝晶组织特征[[S34,S7,S37]]。

![典型压铸件表面缩裂宏观形貌](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/4a48ac51-b558-4826-8091-bbecac216fb9/resource)

### 与热裂、冷裂的区别

GB/T 29092—2012《镁及镁合金压铸缺陷术语》对热裂纹和冷裂纹给出了明确的鉴定特征：
- **热裂纹**：呈不规则曲折长缝隙，多分布于尖角内侧、厚薄交接处、热节区，断面严重氧化呈深灰/黑色无金属光泽，沿晶开裂[[S22]]；
- **冷裂纹**：呈平直折线状裂纹，断面洁净有金属光泽，仅轻微氧化甚至无氧化，穿晶开裂[[S22]]；
- 缩裂不属于上述两类，而是由收缩补缩不足引发的独立缺陷形态。

ASTM 标准进一步给出了缩裂（线性收缩裂纹）与热裂的六项核心鉴别准则[[S21]]：

| 鉴别维度 | 缩裂（线性收缩裂纹） | 热裂 |
|---------|---------------------|------|
| 出现数量 | 一般多发 | 通常单根出现 |
| 萌生位置 | 铸件截面中部 | 近表面位置 |
| 分布区域 | 均匀截面也可出现 | 热节或截面突变处 |
| 温度梯度 | 低温度梯度区域 | 高温度梯度区域 |
| 裂纹走向 | 无明确应力指向性 | 垂直于最大主应力方向 |
| 消除方式 | 可通过充足补完全部消除 | 仅可通过调整应力/热场分布消除 |

### 与缩孔、缩松的边界

缩孔/缩松仅为不规则孔洞类缺陷，无连续性裂隙延伸特征；而缩裂是由缩孔/缩松进一步受收缩应力撕裂形成的断续线性裂纹，二者存在衍生关联但属不同缺陷形态[[S22,S11]]。

## 形成机理

缩裂形成于合金凝固末期的准固相区，此时枝晶骨架已完全搭接，残余液相流动性极低。当铸件不同区域因冷却速率差异导致凝固收缩受到已凝固部分的约束时，局部集中的拉应力若超过该温度下合金的极限延伸率与高温强度，且晶间残留液相无法及时补缩开裂的微小缝隙，裂纹即萌生并沿晶界逐步扩展[[S18,S23,S30,S37,S12]]。

核心驱动力在于三项因素的错配：凝固收缩量、收缩受阻碍程度、残余液相补缩能力[[S30]]。

现代热裂判据研究表明，相较于传统强度理论，采用“等效粘塑性应变大于临界粘塑性应变”的判据更贴合压铸准固相区的材料流变特性；受阻热节在准固相区的停留时间越长，缩裂发生概率越高[[S38,S6,S1]]。

## 产生原因

### 合金成分因素

合金存在明确的脆性温度区间（热脆性区间），该区间范围直接决定缩裂倾向性：
- 凝固温度区间越宽，热脆性区间越大，线收缩量越高，收缩受阻时产生的拉应力更大，缩裂风险更高；
- 当合金成分接近共晶点时，凝固温度区间大幅收窄，缩裂倾向性显著降低[[S15,S18,S29]]。

压铸铝合金中特定元素对缩裂倾向影响如下[[S29,S40,S31]]：

| 元素/相 | 影响 |
|---------|------|
| Cu 含量升高 | 增大热裂倾向 |
| Mg 过量 | 促进晶界低熔点 Mg₂Si 相析出，提升缩裂风险 |
| 粗大 β-Al₉Fe₂Si₂ 脆性相 | 阻塞枝晶间补缩通道，显著提升缩裂概率 |
| 细化元素（Y、Sr、Al-Ti-B） | 细化晶粒、优化补缩通道，降低热裂倾向性 |

此外，合金中存在的孔洞会造成局部应力集中、降低有效承载面积；氧化夹杂与基体结合力弱，二者均会成为裂纹萌生源，加速收缩裂纹扩展[[S31,S16,S37]]。

### 压铸工艺参数

| 工艺参数 | 影响规律 |
|---------|---------|
| 模具温度 | 控制在 200–250 ℃ 可有效降低温度梯度与热应力；模温过低会大幅提升冷却不均度，缩裂风险显著上升[[S30,S35,S8]] |
| 浇注温度 | 过高会粗化晶粒、增大总收缩量、削弱残余液相补缩能力，提升缩裂倾向[[S30,S35,S8]] |
| 压射比压 | 过高会增大薄壁与热节区域凝固速率差异；在不影响致密度前提下适当降低比压可改善抗裂表现[[S30,S35]] |
| 留模时间 | 推荐按 3 s/mm 壁厚设置，避免铸件在高温低强度状态下受机械拉应力撕裂[[S30,S35]] |
| 保压时间 | 不足则补缩通道内金属液无法填充收缩缝隙，直接诱发缩裂；过长则浇口过早完全凝固，料柄处被撕裂产生裂纹[[S17,S33]] |

### 模具设计因素

- 铸件结构壁厚突变、内圆角尺寸过小，导致局部收缩应力集中[[S8,S35,S36,S43]]；
- 拔模斜度过小，铸件凝固后包紧型芯阻力过大，脱模过程被机械撕裂[[S43]]；
- 浇口尺寸过小、布置不合理，导致补缩通道提前闭合，压力无法传递至热节区域[[S8,S35]]；
- 溢流槽容量不足，无法将收缩缺陷转移至非功能区[[S8]]。

## 对铸件性能的影响

缩裂直接削弱压铸件有效承载截面积，大幅降低其抗拉强度、伸长率及疲劳寿命。带穿透性缩裂的压铸件完全丧失气密/水密性能，服役过程中裂纹会沿受力方向持续扩展，引发突发断裂失效，严重降低长期使用可靠性[[S4,S44,S13]]。

## 检测方法

| 检测方法 | 适用范围 | 局限性 |
|---------|---------|--------|
| 目视检测（含 ≥5× 放大镜辅助） | 暴露于外表面的宏观开口缩裂 | 无法检出内部及微裂纹[[S32,S9]] |
| 着色渗透/荧光渗透检测 | 所有表面开口的缩裂微小裂纹 | 仅限表面连接性缺陷，无法检出内部缺陷[[S27,S32]] |
| X 射线检测 | 铸件内部大尺寸缩裂 | 对贴合紧密的闭合型微缩裂检出灵敏度有限[[S27,S32]] |
| 超声波检测 | 铸件内部隐藏型缩裂定位定量 | — [[S32,S9]] |
| 金相检测结合 SEM 观察 | 判定缩裂枝晶间微观特征，区分缩裂与其他类型裂纹 | — [[S32,S9]] |

## 预防与对策

### 铸件结构优化
保证壁厚均匀，将尖角过渡圆角加大至 R2 以上，避免出现孤立热节区域；必要时增设工艺筋[[S39,S40]]。

### 工艺控制
- 模具预热温度提升至 200 ℃ 左右[[S40]]；
- 适当降低浇注温度，调整合适的压射比压以改善补缩效果[[S30]]；
- 通过数值模拟提前预判热节分布，调整冷却水道排布使铸件凝固温度场均匀，避免局部应力集中[[S10]]。

### 合金成分优化
- 添加 ≤0.1% 的 Al-Ti-B 细化剂细化晶粒[[S40,S45]]；
- 控制有害杂质含量，避免形成脆性相堵塞补缩通道[[S29,S40]]。

### 模具工艺强化
针对厚大热节区域可增设局部挤压销机构，在铸件凝固过程中强制补缩消除缩裂[[S2]]；优化浇注/充型系统降低收缩阻力[[S39]]。
