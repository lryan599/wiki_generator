## 概述

在铸造技术语境下，“补贴”一词存在两种截然不同的工艺属性与用途：其一为补缩类补贴，指为实现铸件顺序凝固而附加的楔形补缩通道；其二为加工补贴，对应压铸行业中为后续机加工预留的金属层（machining allowance）。JB/T 2435标准将“补贴”列为独立铸造工艺术语，规定采用红色实线标注并注明各部尺寸 [[S34]]。压铸语境下的加工补贴需在压铸毛坯图的工艺附加物栏明确标注，用于后续切削加工，不属于补缩用途 [[S13,S30,S35]]。

## 补缩类补贴

### 定义与工艺作用

在砂型铸造等重力铸造工艺中，当冒口附近存在热节或铸件尺寸超出冒口补缩距离时，利用补贴可建立向冒口延伸的补缩通道，实现顺序凝固 [[S13]]。补贴工艺的实施可实现消除铸件下部热节处的缩孔、延长补缩距离、减少冒口数目等作用 [[S4]]。

### 补贴分类

补缩类补贴按材质和实现形式可分为四类 [[S13,S41]]：

| 补贴类型 | 特点 | 局限性 |
|---------|------|--------|
| **金属补贴** | 与铸件一体成型，直接建立补缩通道；操作方便，可依靠模具成型 | 需后续人工切割清理，非规则区域清理难度大、成本高；材料成本增加 |
| **加热补贴** | 通过耐火隔片将发热/蓄热材质的补贴区域与铸件本体分隔，延长补缩通道凝固时间 | 耐火隔片需加热到1480℃以上才有效，主要用于铸钢件 |
| **发热（保温）块补贴** | 采用发热材料实现保温补缩效果，应用具有良好的经济效益 | 材料需与发热冒口相似 |
| **暗补贴** | 不与铸件直接接触，通过隔砂层传热对铸件局部区域进行保温加热 | 工艺控制要求较高 |

金属补贴最常见的缺点在于清理工序中切割困难，增加清理及机械加工成本；加热补贴、发热补贴和暗补贴则可克服金属补贴清理难度大的缺陷 [[S4,S13,S41]]。

### 补贴尺寸确定方法

对于齿轮铸件轮缘和轮毂的补贴尺寸，多采用滚圆法（A.Heuvers氏滚圆法）来确定：对于重要部位的热节用扩大滚圆法，次要部位用不扩大滚圆法 [[S13]]。

垂直补贴的尺寸可通过板形碳钢铸件试验总结的补贴厚度与铸件壁高、壁厚关系曲线查找，并采用补偿系数进行修正 [[S4]]。水平补贴最大长度为冒口模数的4.7倍 [[S4]]。

### 补缩补贴效果

优化补贴与冒口设计后，铸件本体的缩松含量可显著降低，缩松缺陷主要集中在补贴和冒口可去除区域，使铸件本体满足致密度要求 [[S33]]。

### 补缩补贴结构图示

金属补贴结构为铸件与冒口之间的倾斜金属过渡结构，与铸件一体成型，用于建立通畅的定向补缩通道以消除铸件缩孔缺陷 [[S9]]。

## 加工补贴（压铸机加工余量）

### 定义

在压铸工艺语境下，加工补贴是指为后续机械切削去除而预留的金属层，旨在将压铸毛坯的精度提升至零件图纸要求的最终尺寸精度和表面粗糙度等级 [[S15,S20,S22]]。该补贴属于“压铸工艺附加物”，必须在压铸毛坯图上明确标注 [[S28,S30,S35]]。

### 设置加工补贴的必要性

设置加工补贴的核心原因包括 [[S31,S43]]：
- 去除铸造斜度，使表面达到装配要求的尺寸精度
- 达到所要求的高精度尺寸
- 因压铸工艺和模具技术限制，铸件未能压铸出的某些形状需要后续加工
- 补偿压铸生产过程中模具成型零件磨损、掉块带来的铸件尺寸偏差
- 消除压铸成型自然产生的少量尺寸公差
- 去除浇冒口或工艺筋、凸台等产品不需要的多余部分

### 加工补贴与相关余量的核心差异

| 对比项 | 加工补贴（machining allowance） | 拔模斜度余量（draft allowance） | 收缩余量（shrinkage allowance） |
|--------|-------------------------------|------------------------------|--------------------------------|
| **工艺目的** | 预留切削层以获得最终尺寸精度和表面粗糙度 | 使凝固后的铸件能顺利从钢制压铸型中顶出 | 抵消金属液凝固冷却阶段的线性收缩 |
| **设置位置** | 待加工表面 | 垂直于分型面的成型表面 | 模具型腔尺寸（预先放大） |
| **是否切削去除** | 要求全部通过后续切削去除 | 不要求全部去除，仅保证脱模 | 已包含在毛坯轮廓中，不单独切削 |
| **属性** | 毛坯上预留待切削金属层 | 锥度附加量 | 模具设计阶段预先的尺寸补偿量 |

引用来源：[[S1,S22,S40]]

### 加工补贴设计准则

压铸件加工补贴设计应遵循以下核心准则 [[S15,S16,S25,S44]]：

1. **优先采用精整加工**：当铸件尺寸精度或形位公差不满足要求时，应首先考虑校正、拉光、挤压、整形等精整加工方法，保留压铸件表层强度较高的致密层。
2. **选用较小加工余量**：必须采用机械加工时应选用较小的加工余量。
3. **基准面选择**：尽量以不受分型面及活动成型部件影响的表面作为毛坯基准面 [[S16,S18,S20]]。
4. **避免过量余量**：过大的加工余量会暴露压铸件内部不够致密的组织，降低工件服役性能。

### 压铸件机械加工余量推荐值

国内主流压铸工艺手册收录的推荐值（按基本尺寸区间划分） [[S8]]：

| 基本尺寸（mm） | 整形前余量（mm） | 整形后余量（mm） |
|:------------:|:------------:|:------------:|
| 0～25 | 0.20 | 0.10 |
| 25～63 | 0.30 | 0.15 |
| 63～100 | 0.45 | 0.20 |
| 100～160 | 0.70 | 0.25 |
| 160～250 | 1.00 | 0.30 |
| 250～400 | 1.5 | 0.40 |
| >400 | 2.2 | 0.50 |

余量取值需取铸件最大外轮廓尺寸与待加工表面公称尺寸两个对应余量的平均值。待加工的内表面尺寸以大端为基准，外表面尺寸以小端为基准 [[S25]]。

### 铰孔加工余量

压铸件铰孔加工余量推荐值 [[S25,S27]]：

| 孔径 D（mm） | 加工余量 δ（mm） |
|:----------:|:------------:|
| ≤6 | 0.05 |
| 6～10 | 0.10 |
| 10～18 | 0.15 |
| 18～30 | 0.20 |
| 30～50 | 0.25 |
| 50～80 | 0.30 |

直径小于18mm的孔，铰孔余量为孔径的1%；直径大于18mm的孔，铰孔余量为孔径的0.6%～0.4%，且不超过0.3mm [[S25]]。

### 国际标准取值参照

- **ASTM/NADCA标准**：常规最小加工补贴为0.25mm（0.010英寸），以避免刀具过度磨损。总加工补贴为该最小基础值、铸件尺寸公差、机加工工艺公差三者的总和。最大补贴为基础最小余量叠加成型公差与铸造公差之和 [[S3,S18,S20]]。
- **ASM铸造手册**：铝合金压铸件推荐的机械加工余量范围为0.5～1.0mm（0.020～0.040英寸），明确禁止超过1.0mm，以防止过度切削去除表层致密层后暴露内部孔隙，导致力学性能、外观及耐蚀性下降 [[S6,S32]]。
- **GB/T 6414**：定义压铸类铸件的要求机械加工余量（RMA）分级标注体系，格式示例为“GB/T6414-CT12-RMA6(H)” [[S45]]。

### 加工补贴的粗/精加工分层

压铸件总加工补贴可拆分为两层 [[S15,S46]]：
- **粗加工补贴**：对应粗切削工步的切除量，用于去除铸件表面的不规则凸起、轻微氧化层和尺寸偏差。
- **精加工补贴**：对应精切削/铰孔工步的切除量，用于保证最终零件的尺寸精度和表面粗糙度。铰孔加工余量即为典型的精加工补贴取值。

### 加工补贴与分型面的配合

加工补贴区域的边界设置直接影响分型面选择：
- 优先将机加工补贴区域布置在易于后续去除飞边的分型面侧 [[S19,S5]]。
- 避免高公差要求的补贴面跨不同半模成型，以保证机加工基准的一致性 [[S18,S20]]。
- 若待加工成型表面与基准点分别处于不同的两个半模中，需在基础最小余量之外额外增加跨分型面尺寸公差对应的补偿量 [[S18,S20]]。

### 加工补贴与表面精度的关系

压铸模正常使用寿命内，不同合金的自然表面粗糙度可维持在 [[S15,S17]]：
- 锌合金压铸件：Ra 1.6～3.2 μm
- 铝、镁合金压铸件：Ra 3.2～6.3 μm
- 铜合金压铸件：受模具龟裂影响表面粗糙度最差

若设计表面精度要求优于上述自然成型水平，则需要对应预留额外的加工补贴以满足表面加工要求 [[S15]]。

### 过量补贴的不利影响

加工补贴设置过大是压铸件缺陷的典型诱因：

| 不利影响 | 机理 | 来源 |
|---------|------|------|
| **气孔暴露** | 过量补贴突破表层0.5～0.8mm的致密冷硬层，暴露内部弥散分布的多孔组织，导致铸件气密性和力学性能大幅下降 | [[S6,S21,S31,S50]] |
| **热节与缩孔** | 局部壁厚异常增厚会形成额外工艺热节，大幅提升凹陷、缩孔、缩松缺陷风险，同时导致模具局部过热 | [[S26,S29]] |
| **工艺匹配偏失** | 铸件整体质量上升偏离压铸预设的PQ²工艺匹配区间，压射过程参数稳定性下降 | [[S12]] |
| **经济性下降** | 金属原材料浪费，违背压铸工艺少无切削的设计优势，大幅提升后续机加工的生产节拍和刀具损耗成本 | [[S43,S49]] |

### 合理补贴取值依据

压铸件表层受激冷作用形成的致密均匀组织厚度通常为0.5～0.8mm。标准推荐的压铸件加工补贴取值一般为0.3～0.5mm，合理的小体量补贴可保证机加工完成后仍保留部分表层致密组织，维持铸件的力学性能与气密性 [[S14,S31,S50]]。合理范围内的小体量加工补贴可支撑压铸件实现0.3～1.2mm的机加工余量，相比砂型铸造等工艺可大幅减少材料消耗 [[S10,S43]]。
