## 概述

电渣熔炼，亦称电渣重熔（英文术语 Electroslag Remelting，缩写 ESR），是一种将化学精炼与控制凝固相结合的无电弧自耗二次熔炼方法，于20世纪50年代在电渣焊基础上发展而来 [[S21,S3,S24]]。其核心原理是利用电流通过高电阻熔渣时产生的焦耳热熔化自耗电极，金属熔滴穿过渣池完成精炼，随后在水冷结晶器中定向凝固，获得高纯净度、组织致密的铸锭 [[S21,S24]]。

## 定义与工作原理

电渣重熔的原料为由其他冶炼方法（如电弧炉、感应炉等）预先制备的自耗电极 [[S1]]。重熔过程中，电流经由变压器、自耗电极、熔渣、金属熔池、已凝固铸锭及底水箱形成回路 [[S24]]。由于熔渣在回路中电阻最大，承担了变压器二次电压的大部分压降，从而在渣池中产生大量焦耳热，使渣池温度远高于金属熔点（通常达 1750 °C 以上，电极下端至金属熔池中心区域可达约 1900 °C）[[S21,S1]]。

自耗电极端部在过热渣池中逐渐熔化形成液态金属膜，金属液在重力、电磁力及渣池运动的共同作用下汇聚成熔滴并滴落，穿过渣池进入金属熔池 [[S21,S24]]。在此过程中，金属熔滴与高温熔渣充分接触（俗称“渣洗”），完成脱硫、脱磷及去除非金属夹杂物等精炼反应 [[S24,S2]]。随后，金属液在水冷结晶器的强制冷却下自下而上定向凝固，顶部始终有渣池和金属熔池实现补缩，最终形成组织致密的铸锭 [[S1,S2]]。

## 工艺流程

电渣熔炼的典型工艺流程可分为两种基本形态 [[S2,S3]]：

1. **充填式**：金属熔体充满结晶器后该熔次即告完成，待铸锭全部凝固后脱模取出。
2. **抽锭式**：重熔过程在结晶器内连续进行，凝固的铸锭由引锭机构从结晶器下部匀速抽出，可获得更长尺寸的铸锭，实现连续生产。

![充填式与抽锭式两种电渣熔炼工作形态对比示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/a5a6a2a0-b1d1-4eba-b518-51a2571b26db/resource)

整个工艺过程的本质是将初炼电极通过熔渣精炼后在水冷结晶器中重新凝固，集熔化、精炼、浇注和凝固于同一装置内完成，有效减少了钢液的二次污染 [[S1]]。

## 在压铸产业链中的定位

电渣熔炼在压铸产业链中属于**间接应用环节**，不直接用于压铸零件的充型生产，而是作为前置工序用于提升压铸相关原材料的纯净度 [[S19]]。

其核心应用领域为**压铸模具用热作模具钢的精炼**。广泛用于 H13（4Cr5MoSiV1）、8407、DAC 系列、HM1、HM3 等牌号的高端压铸模具钢生产。使用经验表明，采用电渣重熔的 4Cr5MoSiV1 钢制造的铝合金压铸模具，其使用寿命可比常规冶炼的同牌号钢模具提高 2～3 倍 [[S19]]。

公开可检索的资料中，未发现将电渣熔炼直接用于大批量压铸铝合金、镁合金、锌合金母材量产的通用工艺记录。该技术当前在压铸领域的应用集中于高端模具钢的质量提升 [[S19]]。

## 工艺制度与关键参数

### 熔渣体系

电渣熔炼的渣系以 CaF₂-Al₂O₃ 为基础，主流配方为 **70% CaF₂ + 30% Al₂O₃** [[S4,S8]]。可按需添加 CaO、MgO 等组分以调整碱度，常规碱度控制在 0.7～1.5 区间 [[S8]]。渣料分为起熔导电渣和正常工作渣两类，工作渣使用前需充分烘烤以去除水分 [[S28]]。

渣系组分对脱硫、脱氧等精炼效果具有决定性影响。通过合理选择 CaF₂-CaO-Al₂O₃ 体系的组分，可实现对硫化物、氧化物夹杂的高效吸附和去除 [[S22,S25]]。

### 电气参数与冷却条件

| 参数类别 | 典型范围或要求 |
|---------|--------------|
| 供电方式 | 工频交流 |
| 熔炼电压 | 50～60 V |
| 熔炼电流 | 6000～6500 A（依装备吨位而定） |
| 熔化速率 | ≤ 1200 kg/h（控制上限可有效抑制偏析） |
| 冷却水进水温度 | 不高于 25 °C |
| 冷却水出水温度 | 45～50 °C |

结晶器采用反方向水冷却结构——冷却水从下部进入、上部流出，以加大下部冷却强度，形成自下而上的强定向冷却条件 [[S2]]。此外，可采用惰性气体（氩气）气幕保护工艺，以降低钢液从大气中吸氢的概率 [[S28]]。

## 工艺效果与质量影响规律

1. **偏析控制**：适当提高冷却强度、降低熔化速率，可减小合金偏析程度，获得更细的二次枝晶臂间距；将熔速控制在合理区间可有效抑制偏析产生 [[S17,S28]]。
2. **夹杂物去除**：熔渣的高吸附作用可将硫化物、氧化物等非金属夹杂物去除率提升至 **90% 以上**，显著降低夹杂物总含量 [[S27,S10]]。这一效果远优于真空电弧重熔（VAR）工艺。
3. **致密度**：定向凝固条件下铸锭致密度远高于常规砂型铸造，内部无明显缩孔疏松缺陷 [[S1,S17]]。
4. **氢含量风险**：常规敞开式电渣熔炼容易从大气和含水渣料中吸入氢，导致钢中氢含量高于真空类熔炼工艺（如 VAR），易引发氢脆风险 [[S27,S28]]。

## 与相关二次精炼工艺的对比

| 对比维度 | 电渣重熔（ESR） | 真空电弧重熔（VAR） | 真空感应熔炼（VIM） |
|---------|---------------|------------------|-------------------|
| 设备与运行成本 | 低（无需高真空系统） | 高 | 高 |
| 脱硫能力 | **强**（可大幅降低 S 含量） | 弱（不能去除硫） | 弱 |
| 去除非金属夹杂物能力 | **强**（去除率可达 90% 以上） | 一般 | 弱 |
| 脱气（氢/氧/氮）能力 | 弱（常规工艺易吸氢） | 强（真空下去除氢） | 强（高真空下高效脱除） |
| 易挥发元素（如 Mg）烧损 | 较小 | 较大 | 视真空度而定 |
| 铸锭表面质量 | **优**（无需额外打磨） | 较差 | 一般 |
| 铸锭截面形状 | 可为矩形或圆形 | 一般为圆形 | 视坩埚而定 |
| 最大可制锭型 | 大（可达 180 t 级） | 较小（约 55 t 级） | 较小 |
| 适用场景 | 高端压铸模具钢、大截面矩形铸锭 | 航空航天级合金、要求极低气体含量 | 高端合金初炼，常搭配 ESR 或 VAR |

<center><small>表：ESR、VAR 与 VIM 工艺核心特性对比 [[S6,S26,S11,S22]]</small></center>

电渣重熔因其优异的脱硫除杂能力、较低的设备运行成本以及良好的铸锭表面质量，成为高端压铸模具钢生产的主流精炼工序。真空电弧重熔则更适合对气体含量要求极低的航空航天级合金制备。三类工艺在实际生产中常根据合金要求搭配使用，形成双联（VIM+ESR 或 VIM+VAR）甚至三联精炼路线。

## 典型缺陷及其对压铸应用的影响

### 1. 氢致白点

敞开式电渣熔炼若渣料未充分烘干或未采用惰性气幕保护，钢中吸氢量过高。在后续冷却过程中，过饱和的原子氢在钢内微孔隙中析出并合成为分子氢，产生巨大内压，与热应力、组织应力共同作用，在钢材内部形成细小裂纹（在纵向断口上呈银白色斑点，故称白点） [[S23,S27]]。

**对压铸的影响**：带有白点缺陷的压铸模具钢在冷热循环载荷作用下会快速扩展开裂，大幅降低模具使用寿命，甚至出现早期脆性断裂 [[S27]]。

### 2. 成分偏析

熔化速率控制过高时，凝固过程中溶质元素富集形成局部成分不均匀的斑块状偏析，偏析区域的硬度、韧性远低于基体 [[S18,S27]]。

**对压铸的影响**：偏析区域在压铸模具服役过程中会成为热疲劳裂纹的核心起源，导致模具提前出现龟裂纹，降低压铸件的合格率 [[S27]]。

### 3. 表面质量缺陷

如果渣系组分不合理或渣壳厚度不均匀，铸锭表面可能产生局部结疤、凹坑等缺陷，后续锻造加工需额外增加打磨工序。

**对压铸的影响**：表面缺陷残留部位在压铸工况下容易出现铝液熔蚀坑 [[S27]]。

## 标准与规范

在国内现行标准体系中，《GB/T 1299-2000 合金工具钢》等压铸模具钢通用材料标准中，明确将电渣重熔列为高端热作模具钢（如 H13、HM1、HM3 等牌号）的推荐制备工艺，其高端产品均要求采用电渣重熔精炼 [[S5]]。

国际标准方面，ASTM A681 系列热作模具钢标准及 ISO 4957 等工具钢通用标准中，将电渣重熔作为高纯净级模具钢的可选验证工艺 [[S5]]。

值得注意的是，针对压铸铝合金产品的 GB/T 1196（重熔用铝锭）、ISO 17615、ASTM B179 等主流压铸原材料标准中，暂未将电渣熔炼列为量产铝合金锭的通用强制制备要求，该技术仅适用于小批量高端特种压铸合金的定制化生产 [[S5]]。
