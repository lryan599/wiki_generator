## 1 术语溯源与概述

“金属模（塑料模）”这一混合表述，其源头可追溯至日文术语**金型（かながた/kanagata）**。在1975年出版的《工程技术日语手册》中，“金型”被明确标注对应中文“金属模、金属型”的含义[[S5]]。该术语传入中文工业体系后，广泛覆盖了铸造、注塑、冲压等所有采用金属材料加工制造的模具全品类，是中文跨行业语境下“金属模（塑料模）”混合表述的直接来源；同时，日本JIS金型标准体系（如JIS B 5101压铸模用型腔模体、JIS B 5102压铸模用导柱、JIS B 5106塑料模用型腔模体等）中该术语本身就同时覆盖压铸模与塑料模等多个细分品类，进一步促成了这类混合表述的行业普及[[S10,S3]]。

在压铸及金属铸造领域，“金属模”泛指由金属材料制成、可重复使用的铸型。需要指出的是，在国内铸造标准术语体系中，“金属型”被规定为优先使用的正式表述，并明确“金属型禁止称作‘金属模’”[[S7]]；但在通用工业语境下，“金属模”仍被广泛使用，其同义表述包括“硬模”、“永久型”等[[S7,S11]]。

> **提示：** 在正式标准文件与技术规范中应优先使用“金属型”；在工程实践与跨行业通用语境下，“金属模”仍为可接受的通俗表述。

## 2 定义与基本概念

**金属模（Metal Mold / Permanent Mold）**是用金属材料制成的铸型，在重力作用下将熔融金属浇入型腔获得铸件[[S7,S11]]。该工艺俗称**硬模铸造**，亦可称为**永久型铸造**（Permanent Mold Casting）。其本质特征在于模具可反复循环使用，与砂型铸造（铸型单次使用）、消失模铸造（每产出一件铸件消耗一个泡沫塑料模样）存在根本属性差异[[S8,S11,S44]]。

与砂型铸造相比，金属模铸造的铸件冷却速度更快、组织更致密，所得铸件力学性能平均高出约15%；同时铸件尺寸精度、表面质量与生产稳定性亦明显优于砂型铸造[[S11]]。

> **术语边界说明：** “永久型”（Permanent Mold）的范畴比“金属型/金属模”更广，除金属铸型外还包含石墨材质等可长期重复使用的非金属铸型[[S7]]。

### 2.1 铸造金属模与注塑模的核心差异

铸造用金属模与塑料成型用**注塑模**存在本质差异：铸造金属模直接接触千摄氏度级别的熔融金属，需承受极强热冲击，温控系统设计侧重调控高温熔体的降温速率，配套钢材优先选型耐热、抗热疲劳的热作模具钢；注塑模接触的是粘流态塑料熔体，工作温度普遍低于400℃，温控系统侧重加速塑料制品冷却固化，配套钢材优先选型镜面性能优异、耐塑料化学腐蚀的专用塑料模具钢[[S42,S40,S1,S12]]。

## 3 分类体系

### 3.1 按铸造工艺分类

在铝合金铸造工艺体系中，金属模属于**重力铸造法**的下属分支，与砂模、壳模、熔模铸造法并列；压力铸造法则涵盖加压铸造法与加压凝固法两大分支[[S34]]。

![铝合金铸造法分类体系图，展示重力铸造法下属包含金属模、砂模、壳模、熔模铸造法，压力铸造法下属包含加压铸造法、加压凝固法](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/71768651-3676-48d0-94d5-74b77710bc48/resource)

压铸用金属模（即压铸模）按适配工艺可进一步分为三大类[[S30,S48]]：

| 类别 | 说明 |
|------|------|
| 金属型重力铸造模 | 依靠重力充型，无外加压力 |
| 低压铸造模 | 在较低压力下充型凝固 |
| 高压压铸模 | 高压高速充型，可细分为热室压铸机用、立式冷室压铸机用、卧式冷室压铸机用、全立式压铸机用等 |

### 3.2 压铸模按结构形式分类

根据压铸机类型的适配差异，压铸模可分为[[S30]]：
- 热室压铸机用压铸模
- 冷室压铸机用压铸模（卧式、立式、全立式）

## 4 结构与组成

压铸模（高压压铸用金属模）由定模和动模两个主要部分组成。定模固定在压铸机定模安装板上，与压射机构的压室连接；动模安装在动模安装板上，随动模安装板移动实现合模或开模[[S26]]。

![压铸模具全结构组成分类框图，展示模体（定模/动模）与模架（推出机构/预复位机构）两大一级模块及其下属各核心部件的完整从属关系](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/8bd2ff3e-5ccf-404d-8c8b-206929fbd606/resource)

### 4.1 核心组成模块

压铸模由以下八大功能模块构成[[S31,S26,S30]]：

| 模块 | 构成 | 功能 |
|------|------|------|
| **模架** | 套板、座板、垫板等支承固定零件，以及导向零件、推出复位机构 | 组合固定所有模具部件并实现压铸机上的可靠安装，保证开合模运动精度[[S31,S26]] |
| **成型部分** | 型腔（成型外表面）与型芯（成型内表面） | 决定压铸件几何形状和尺寸精度[[S30]] |
| **浇注系统** | 直浇道、横浇道、内浇口、浇口套、分流锥等 | 沟通型腔与压室，是熔融金属填充型腔的必经路径，直接决定金属填充流动状态[[S30,S31]] |
| **溢流排气系统** | 溢流槽与排气槽 | 溢流槽储存冷金属和涂料余烬；排气槽快速排出压室、浇道和型腔内气体，减少气孔缺陷[[S26,S30]] |
| **侧抽芯机构** | 滑块、斜销、楔紧块、限位挡块等 | 抽动与开合模方向运动不一致的活动型芯，合模前完成插芯、推出前完成抽芯[[S30,S26]] |
| **导向机构** | 导柱与导套 | 引导动模与定模的开合模运动，保证两个半模精准对中，避免模具错位[[S30]] |
| **推出复位机构** | 推板、推杆固定板、推杆、复位杆等 | 将压铸件从模具中顶出，并完成推出元件复位[[S30,S26]] |
| **温控系统** | 分布于模体关键区域的加热和冷却通道（水冷、风冷） | 水冷与风冷组合实现对模具热平衡的精准控制，避免金属液过早凝固或模具过热，保证铸件质量一致性[[S46,S26]] |

## 5 材料与制造要求

### 5.1 常用热作模具钢

压铸模成型部件主流选用的热作模具钢牌号如下[[S50,S20]]：

| 牌号 | 对应标准/来源 | 说明 |
|------|---------------|------|
| H13 | 国标4Cr5MoSiV1（GB/T1299），美标AISI H13 | 含Cr约5%的中合金热作模具钢，应用最广泛 |
| SKD61 | 日本JIS标准 | H13的日本近似牌号 |
| 8407 | 瑞典一胜百（ASSAB） | 电渣重熔（ESR）精炼钢，纯净度高 |
| 8417 / 1.2344ESR | 进口优质钢 | 适用于一体化大型压铸模 |

这类材料要求具备良好的淬透性、热稳定性、高热导率和抗热疲劳性能[[S50]]。H13钢经标准热处理后硬度范围通常为42~52HRC；大型一体化压铸模优先保证抗开裂韧性，推荐硬度控制在45~48HRC；普通小型压铸模可提升至50~52HRC[[S50,S49]]。其热导率可达28.4W/(m·K)，可在型腔工作温度低于600℃的工况下长期稳定服役[[S55]]。

### 5.2 表面强化处理

压铸热作模具钢常用的表面强化工艺包含气体渗氮、氮碳共渗（软氮化）、离子喷镀等，处理后模具表面硬度可达55~60HRC，氮化层厚度不小于0.4mm，可显著提升模具的耐磨性、抗金属液腐蚀性能和脱模效果[[S55,S2,S49]]。

## 6 工艺角色与操作条件

### 6.1 压铸循环中的任务

以冷室压铸机为例，压铸循环各阶段功能划分如下[[S29,S17]]：

| 阶段 | 细分动作 | 功能说明 |
|------|----------|----------|
| **锁模** | 低速启动 → 快速锁模 → 低压保护合模 → 高压锁模 | 防止合模冲击；异物检测保护模具；建立足够锁模力 |
| **压射** | 锤前低速排气 → 一快压射 → 二快高速充型 → 增压补缩 | 保障填充速度与后期致密度 |
| **冷却** | — | 铸件在型腔内凝固冷却 |
| **开模** | — | 动模后撤，分型面打开 |
| **顶出** | — | 推出机构将铸件顶出 |
| **取件** | — | 取出铸件，完成一个循环 |

### 6.2 关键工艺参数

#### 6.2.1 压力与速度

压铸过程中作用于金属液的压力范围为**20~200MPa**，充型初始速度为**0.5~70m/s**，充型时间仅为**0.01~0.03s**[S33,S43]。金属液通过内浇口的线速度普遍为**10~100m/s**，常规铝合金铸件充型速度为**15~25m/s**[S28,S13]。

| 参数 | 范围/推荐值 | 备注 |
|------|-------------|------|
| 压射比压范围 | 2~250 MPa | 涵盖填充比压与增压比压[[S28]] |
| 非受力件/外观件推荐增压比压 | 60 MPa | 经验推荐值[[S21]] |
| 受力件/有密封要求件推荐增压比压 | 80 MPa | 经验推荐值[[S21]] |
| 内浇口速度 | 10~100 m/s | 高速充型的核心参数[[S28]] |

#### 6.2.2 锁模力

压铸过程中，型腔内带压金属液对型腔壁面产生**胀型力**（亦称反压力）；锁模力须大于胀型力，以防止金属液从分型面溢出[[S41,S19]]。

常规计算公式[[S41,S21]]：

> **锁模力 ≥ K × 增压比压 × 总投影面积**

其中K为安全系数，一般取1.2~1.3。胀型力分为分型面胀型力与侧向胀型力[[S19]]。

![压铸过程胀型力与锁模力作用原理示意，展示压射力、胀型力、锁模力作用方向及金属液、压铸模具、压射冲头相对位置](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/0c0e9283-af7d-4acf-aade-0fcd67a09691/resource)

#### 6.2.3 温度参数

不同合金对应的压铸模具常规工作温度区间[[S51,S19]]：

| 合金类型 | 模具工作温度区间 |
|----------|-------------------|
| 锌合金 | 150~200℃ |
| 铝合金 | 200~300℃ |
| 镁合金 | 220~300℃ |
| 铜合金 | 300~380℃ |

模具预热阶段需将模温升至**210~300℃**，以保障涂料涂覆效果、减少熔体降温、延长充型流程，同时降低热冲击提升模具寿命[[S36,S19]]。

## 7 失效与缺陷

### 7.1 常见失效模式

国内铝合金压铸模的失效模式统计占比如下[[S38]]：

| 失效模式 | 占比 | 说明 |
|----------|------|------|
| 热疲劳（龟裂） | 63%~67% | 最常见的失效形式，由反复急冷急热循环引起 |
| 开裂 | 15%~18% | 机械应力或热应力超过材料强度极限 |
| 磨损 | 6% | 摩擦引起的材料损耗 |
| 冲蚀 | 4% | 高温高速金属液冲刷造成表面破损 |
| 粘铝失效 | 2% | 铝液与模具表面反应形成粘铝层 |

**热疲劳龟裂**是压铸模具最典型的正常消耗失效形式。模具型腔表面在反复急冷急热循环中交替承受压应力和拉应力，循环累积超过材料屈服强度后形成网状裂纹。常规裂纹深度不超过1mm，裂纹内部残留有氧化物与液态金属残留物[[S9,S35]]。

**冲蚀失效**多发生在近浇口位置，由高速高压高温金属液持续冲刷模具表面，同时伴随金属液与模具材料的化学反应生成脆性相，以及气泡爆破产生的气蚀共同作用，造成模具表面形貌破损，出现凹凸不平的受损区域[[S35]]。

### 7.2 模具缺陷导致的铸件问题

由压铸模具缺陷直接导致的铸件典型质量问题[[S32]]：

| 模具缺陷 | 对应的铸件问题 |
|----------|----------------|
| 模具表面龟裂 | 铸件表面出现网状花纹 |
| 冲蚀区域 | 铸件表面粘铝、拉伤、轮廓畸变 |
| 模具变形 | 铸件尺寸超差 |
| 模具排气不畅 | 铸件内部卷气、缩松等缺陷 |

## 8 压铸模与注塑模对比

### 8.1 核心差异

压铸模与注塑模在六个核心维度上的差异如下[[S14,S38]]：

| 对比维度 | 压铸模 | 注塑模 |
|----------|--------|--------|
| 工作温度 | 200~380℃（锌/铝/镁/铜合金） | 几十至一百余摄氏度 |
| 注射压力 | 20~200 MPa，远大于注塑 | 相对较低 |
| 材料要求 | 高红硬性、抗热疲劳性能优异的热作模具钢 | 镜面性能优异、耐腐蚀的塑料模具钢 |
| 排气系统 | 侧重溢流排气系统，适配金属液充型特征 | 排气要求相对简单 |
| 温控重点 | 调控高温熔体降温速率，保持热平衡 | 加速塑料制品冷却固化 |
| 典型失效 | 以热疲劳龟裂为主（63%~67%） | 以常规磨损为主，几乎不存在热疲劳龟裂 |

### 8.2 设计共性

两类模具在设计层面的共性特征包括：分型面选择逻辑、拔模方向设计、收缩率设置、顶杆与复位杆设计原理高度相似。工程实践中可直接复用UG Moldwizard等注塑模专业设计模块的部分功能提升压铸模设计效率，仅需适配压铸模更高的强度要求与特殊浇注系统设计规则[[S53]]。
