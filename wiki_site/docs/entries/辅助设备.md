## 概述

压铸辅助设备（ancillary equipment）是指在压铸单元内完成附加工艺功能、配合压铸生产的功能设备，与压铸主机（压铸机）共同构成自动化压铸生产成套装备[[S17,S29]]。压铸机本体核心仅包含合模单元、注射单元、动力系统及基础原生控制单元，凡不直接承担锁模、高压压射及原生顶出核心动作的配套功能设备均归属于辅助设备范畴[[S28,S6]]。

国标对辅助设备的定义明确了两层含义：一方面，辅助设备是"配合压铸生产的功能设备"[[S17]]；另一方面，它是"在压铸单元内完成附加工艺功能的装置，不属于压铸机本体的核心组成部分"[[S29]]。在设备分类体系中，全部压力铸造设备划分为压铸机和自动辅机两大一级类别，辅助设备即为自动辅机类别的具体构成[[S3,S30]]。

## 分类体系

按照国标GB/T 37371-2019的分类框架，压铸辅助设备的典型品类涵盖浇注装置、取件传送装置、喷涂装置、模温控制装置、清理整修装置、镶件装置、检测装置、系统安全装置、集成控制装置、铸件冷却装置、真空装置等核心配套设备[[S17]]。结合工艺阶段，可将辅助设备划分为以下功能大类[[S17,S32,S29]]：

| 工艺阶段 | 典型辅助设备 | 主要功能 |
|---------|-------------|---------|
| 预压铸阶段 | 熔炼炉、保温炉、定量浇注炉（给料机） | 金属熔炼、保温储存、定量给料 |
| 在机运行阶段 | 喷涂机（喷雾机）、模温机（模温控制装置）、真空装置、镶件预装装置 | 模具喷涂与冷却、模具温度调控、型腔真空排气、镶件预置 |
| 脱模后阶段 | 取件机器人（取件机械手）、铸件冷却装置、输送装置 | 铸件自动取出、快速冷却、工位间流转 |
| 精整阶段 | 切边机、清理整修装置、铸件检测装置、打标除尘装置 | 浇道/飞边切除、铸件整形清理、外观检测与标识 |

## 各工艺阶段设备详解

### 预压铸阶段设备

预压铸阶段的核心辅助设备为熔炼与供料系统。**熔炼炉**的功能是将固态合金熔化为符合成分要求的熔融金属；**保温炉**用于存放金属熔体并维持在预设温度范围内；**定量浇注炉（保温浇注炉/给料机）**同时具备熔体保温、定量计量及向压射室输送金属液的功能，可大幅降低人工浇注误差，提升金属液供给一致性[[S17,S32]]。

保温炉的选型依据为：容量需满足盛装压铸机0.5～1小时内的金属液消耗量，避免金属液在炉内停留时间过长导致过多氧化损失，同时要求坩埚的金属液表面积尽可能小，以减少舀取金属液时的氧化[[S7,S9]]。对于镁合金专用保温炉，其结构应保证金属液免受氧化，需配置防氧化密封装置[[S7]]。

常规压铸合金的参考熔炼温度区间如下[[S23]]：

| 合金类型 | 参考熔炼温度 |
|---------|-------------|
| 锌合金 | 约427 ℃（800 ℉） |
| 镁合金 | 约649 ℃（1200 ℉） |
| 铝合金 | 约677 ℃（1250 ℉） |
| 铜合金 | 约927 ℃（1700 ℉） |

### 在机运行阶段设备

在机运行阶段，多类辅助设备协同工作以保障压铸工艺的稳定性：

- **压铸模喷涂机（喷雾装置）**：将配比好的脱模剂雾化后，定点定量地喷涂到模具型腔表面，形成高功能脱模外层，降低铸件粘模风险，同时为模具型腔提供降温作用[[S17,S32,S10]]。供给喷涂机的压缩空气压力需达到0.5 MPa以上，喷涂量和喷涂时序可针对不同模具型腔点位独立设置[[S32]]。

- **模温机（模温控制装置）**：通过介质循环，在±1 ℃精度范围内自动调控模具工作温度，抑制铸件热裂、冷隔等缺陷[[S32,S17]]。模温机分为水式和油式两类：**水式模温机**换热系数高、冷却速度快，最高工作温度约180 ℃，适配铝合金压铸的常规温控需求；**油式模温机**可实现350 ℃以内的高温控制，适配镁合金等高模温要求的压铸生产[[S18,S11]]。

- **真空装置**：在特定瞬间快速抽出压铸模型腔内的气体，降低铸件内部孔隙率，提升铸件致密度[[S32,S17]]。

上述几类设备协同运行时，可直接将压铸生产节拍缩短30%以上[[S32]]。

### 脱模后阶段设备

- **取件机器人（取件机械手/自动取件机）**：在压铸机顶出动作完成后直接抓取铸件并移载至指定工位，避免人工取件的高温烫伤风险。其夹持负载需与对应压铸件重量匹配，正常工作范围需覆盖压铸机开模行程内所有工位，失电工况下需具备铸件防掉落功能[[S32,S17]]。

- **铸件冷却装置**：将刚脱模的高温铸件快速降温至工艺要求温度[[S17]]。

- **输送装置**：实现各工位之间铸件的自动流转，消除人工转运的节拍断点[[S17,S29]]。

### 精整阶段设备

精整阶段辅助设备的循环周期要求小于压铸主循环周期，避免拖累整条产线效率[[S17,S32]]：

- **切边机**：一次性切除铸件的浇注系统、溢流槽与周边飞边毛刺[[S17]]。
- **后续清理整修装置**：完成铸件整形作业[[S17]]。
- **铸件检测装置**：识别铸件外观缺损并实时反馈[[S17]]。

## 典型参数与运行条件

依据国标GB/T 39957-2021的规定，各类核心辅助设备的参数要求如下[[S32]]：

| 设备类别 | 参数项目 | 规定值/要求 |
|---------|---------|------------|
| 模温机（水循环） | 最高工作温度 | ≤180 ℃ |
| 模温机（油循环） | 最高工作温度 | ≤320 ℃ |
| 模温机 | 系统工作压力 | ≤1.0 MPa |
| 模温机 | 温度控制精度 | ±1 ℃ |
| 喷涂装置 | 压缩空气压力 | ≥0.5 MPa |
| 喷涂装置 | 关闭状态 | 喷涂结束无滴漏 |
| 喷涂装置 | 动作安全性 | 上升行程末端具备减速功能 |
| 取件装置 | 夹持负载 | 与对应铸件重量匹配 |
| 取件装置 | 安全要求 | 失电工况下具备防掉落功能 |
| 清理整修装置 | 循环周期 | 小于压铸循环周期 |

## 选型与匹配原则

辅助设备的选型需严格遵循与压铸主机、模具及生产节拍的匹配原则。前文已述，保温炉容量需与压铸机金属液消耗速率匹配（0.5～1小时消耗量）[[S7,S9]]。更广泛的选型准则包括[[S12]]：

1. **控制信号接口匹配**：辅助设备须与对应型号压铸机的自动循环控制协议适配，确保信号交互正常。
2. **模具空间与工艺适配**：配套的模温参数、喷涂动作路径、取件行程需与目标压铸模具的尺寸参数匹配，才可接入整线自动化系统。
3. **合金专配要求**：不同压铸合金对辅助设备提出特殊配置需求。例如，镁合金专用保温炉需配置防氧化密封结构以避免金属液氧化，镁合金压铸生产需配套油式模温机以维持更高模具工作温度，适配镁合金薄件充型要求[[S7,S18]]。

《压铸技术手册》收录的周边设备选型表9-16给出了不同锁模力压铸机（DCC160至DCC2000等主流机型）对应配套的熔炉、喷涂机、取件机、切边机选型参考，可供实际工程选型查阅[[S34]]。

## 常见故障与维护

### 典型故障模式

1. **真空阀堵塞**：真空辅助压铸系统的典型故障形式，可通过采集射出开始、真空开始、AVV SOL、高速SOL等关键控制信号的周波数据完成故障溯源定位[[S16]]。
2. **喷涂覆盖不到位**：喷嘴布置不合理、喷涂路径不到位直接导致模具型腔粘铝故障，针对性调整喷嘴排布优化喷涂覆盖范围可有效降低该类故障的发生概率[[S10]]。
3. **冲头卡滞**：浇注辅助系统的典型故障之一，发生场景与压铸生产中的冲头工况直接相关[[S24]]。
4. **浇口垫渣故障**：不同压铸设备间浇口垫渣故障发生率存在显著差异，可利用无故障生产次数、故障次数及C1000值等量化指标进行故障相关性分析[[S25]]。

### 维护体系

压铸辅助设备的日常维护要点包括[[S1]]：
- 检查设备油路接头泄漏情况；
- 确保运行油温保持在10～50 ℃范围内；
- 确认系统压力状态正常；
- 验证安全装置动作有效性及各辅助设备信号交互正常。

在分级维护体系方面[[S21]]：
- **周度维护**：清理导轨、大杠、曲肘部位积尘，检查所有电磁阀固定状态。
- **月度维护**：检查各位置行程开关安装紧固度和动作灵敏度，确认设备互锁逻辑有效性。
- **年度维护**：运行满1年需完成所有行程开关统一更换，完成液压系统滤芯更换、管路磨损排查、辅助传动机构精度校准。

NADCA《压铸过程控制》配套教程亦建议跨部门（运维、工艺、质量）定期审核每台辅助设备的运行性能，严格遵循设备厂商指定的维护周期执行备件更换和精度校准[[S8,S19]]。

## 安全规范与标准

### 国内标准体系

- **GB 20906-2007《压铸单元安全技术要求》**：明确规定压铸辅助设备（给料、抽芯、喷涂类）必须与压铸主机实现联动安全互锁。配套的环绕式封闭防护门需配备双冗余独立限位开关，安全回路独立于主控制电路，防护门未完全闭合时设备无法启动运行[[S13]]。

- **GB/T 37371-2019《压铸单元 术语》**：建立了辅助设备的官方定义与品类划分框架[[S17]]。

- **GB/T 39957-2021《压铸单元 技术条件》**：规定了各辅助设备的参数限值与功能要求[[S32]]。

- **GB 26488-2025《镁合金压铸安全生产规范》**：针对镁合金压铸配套辅助设备提出额外安全要求，包含水冷却压射杆的流量差监测报警功能、油温过热保护装置，以及切边机、热整形机等周边辅助设备的安全防护联锁配置要求[[S33,S22]]。

### 国际标准参照

德国VDG关联压铸安全体系遵循DIN 1688系列压铸相关标准，对压铸周边辅助设备的结构精度、部件安全冗余、电气防护等级提出明确分级要求，与NADCA 207系列H-13压铸模具钢验收标准共同构成欧美压铸行业辅助设备配套模具的通用质量控制体系[[S4,S2]]。
