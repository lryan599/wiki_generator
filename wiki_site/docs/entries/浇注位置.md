## 概述

“浇注位置”在铸造工艺设计中指金属液进入型腔时铸件在铸型中所处的空间位置及充型路径的总体布局 [[S7,S31]]。在砂型铸造等重力充型工艺中，该术语侧重于铸件在重力场下的静置方位，以保证重要工作面向下、厚大部分朝上便于补缩等 [[S31,S12,S33]]。在压铸领域，“浇注位置”的含义发生根本性转变，它是一个面向高压高速充型特性的顶层全局布局概念，涵盖从压室到型腔末端的完整金属液流动通道规划，而非仅指代单一进浇点位置 [[S22,S26]]。该词条重点阐述压铸场景下的定义与应用，并在相关章节说明其与砂型铸造概念的关键区别。

**压铸“浇注位置”与“内浇口位置/进浇点”的核心边界差异：**
- **浇注位置**：属于浇注系统设计阶段的全局布局工作，包含浇口套、直浇道、横浇道、内浇口、料饼及溢流排气系统的整体空间规划 [[S26,S54]]。
- **内浇口位置（gate location / ingate position）**：仅指横浇道末端与型腔相连的进浇通道的空间点位及其导流方向，是浇注位置设计的核心环节，但外延远小于前者 [[S22,S53]]。

**英文术语对照：**
- 压铸“浇注位置”在英文文献中没有完全对等的单一术语。业界通常以“gating system design”或“gate position selection”的语境来表达其涵盖范围。
- “gate location”与“ingate position”仅对应中文“内浇口位置/进浇点位置”，无法承载全局流道布局的完整含义 [[S53,S9]]。
- “sprue”对应直浇道/浇口套，属于浇注位置布局的上游组成部分 [[S43]]。

## 范畴边界

压铸“浇注位置”的设计范围包括以下组成部分 [[S26,S54]]：

| 组成部分 | 说明 |
|:---|:---|
| 料饼/余料 | 压室末端凝固形成的金属块，是浇注系统的起点 |
| 直浇道（sprue） | 连接压室与横浇道的通道，在冷室卧式压铸机中为水平通道 |
| 横浇道（runner） | 将金属液分配至各内浇口的过渡通道 |
| 内浇口（gate/ingate） | 金属液进入型腔的最终节流通道 |
| 型腔（cavity）充型路径 | 金属液在型腔内的流动方向与充填顺序规划 |
| 溢流/排气系统 | 与浇注位置协同设计，实现有序排气与排渣 |

浇注位置设计需要同步关联模具分型面，确定铸件在压铸模型腔中的整体安放空间方位，以适配高压充型下的排气与流态控制需求 [[S7,S22]]。图1直观展示了立式冷压室压铸机浇注系统的完整结构，涵盖直浇道、横浇道、内浇道与余料的上下游分布关系：

## 分类与常见形式

压铸浇注位置可按金属液流入方向、浇口位置及浇口结构形式进行多维度分类 [[S41,S51,S36,S52]]。

### 按浇口位置与金属液流入方向分类

| 类型 | 布置特点与充型特性 |
|:---|:---|
| **顶注（中心浇口/顶浇口）** | 金属液从铸件顶部或中心进入型腔，流程短且均匀，内部结晶致密；模具结构较复杂，适用于轮类、盖类铸件 [[S41,S53]] |
| **侧注（侧浇口）** | 从铸件侧面进浇，是最常用的形式；可灵活布置在铸件不同厚薄区域，适配多数几何形状 [[S41]] |
| **底注** | 金属液从型腔底部向上填充，利于排气，但浇注系统压头损失较大 |
| **阶梯式进浇** | 在铸件不同高度上设置多个进浇口，分段填充以改善充填顺序与热场分布 |

### 按浇口结构形式分类

| 类型 | 特点与适用场景 |
|:---|:---|
| **点浇口** | 进浇截面极小，适用于外观要求高的薄壁件，去浇口方便 |
| **缝隙浇口** | 沿铸件一侧开设长缝隙进浇，形成片状金属流，利于薄壁平板件充填 |
| **环形浇口** | 浇口环绕型腔四周布置，适用于筒状、管状铸件，实现均匀充型 [[S41,S54]] |
| **切向浇口** | 金属液沿型腔切向导入，产生旋转流态，适合圆形件 |
| **扇形浇口** | 横浇道过渡区呈扇形展开，使金属液展开放射进入型腔 |

## 核心设计原则

### 浇口位置选取核心原则

行业公认的浇注位置设计原则归纳如下 [[S45,S2,S38,S55,S27,S9]]：

| 原则 | 说明 |
|:---|:---|
| **厚壁处进浇** | 内浇口优先设置在铸件厚壁处，实现金属液由厚壁向薄壁顺序填充，有利于压力传递与补缩 [[S45,S38]] |
| **最短流动路径** | 浇口位置使金属液充型流动路径最短，避免过多迂回曲折，减少涡流卷气与热量损失 [[S45]] |
| **优先充填深腔/难排气区** | 进入型腔的金属液应先流向深腔难以排气的部位，不宜立即封闭分型面，保障排气顺畅 [[S45,S38]] |
| **避免冲击型芯/型壁** | 浇口进流方向避免金属液直接冲击型芯或型壁，降低能量耗散与模具冲蚀风险 [[S45,S38]] |
| **减少流股对冲与分流** | 多股浇口时应使金属流相互促进、顺序填充，避免对面相撞造成能量耗散与卷气 [[S45]] |

### ISO通用设计指引

ISO及相关文献明确的浇口位置设计要求 [[S53,S21,S42]]：

1. 浇口位置需将金属液流动方向导向最短型腔维度，尽可能减小充型流动距离。
2. 利用铸件自身形状引导金属液流动，匹配充型路径与铸件轮廓以降低湍流。
3. 圆形、椭圆形类型腔优先将浇口布置在型腔中心区域，实现中心充型。
4. 高表面质量、低孔隙度要求的关键区域需靠近浇口，获得直接的金属流供给。
5. 浇口尽可能布置在铸件最厚的截面区域，远离型腔内凸起的凸台、孔特征。

### NADCA设计要点

NADCA《Gating Die Casting Die》官方指南中明确的设计要求 [[S42]]：

| 设计参数 | 要求 |
|:---|:---|
| 浇口高度 | 不得超过铸件对应位置壁厚的75%，以保证后续冲切无变形或断裂 |
| 浇口宽高比 | 浇口宽度与浇口高度的比值需大于10，保证金属液进入型腔时呈均匀的展开放射流状态 |
| 多路分支横浇道 | 每经过一个90°折弯或“Y”型分岔位置，横浇道截面积需比上游增大3%–10%，保证各型腔进液流量均匀 |

### 中国通用设计规范

中国压铸浇注位置通用设计规范涵盖以下核心原则 [[S31,S12,S33,S1]]：

1. 铸件重要加工面、主要工作面应朝下或位于侧面，避免上部气孔、夹渣、组织疏松缺陷。
2. 铸件宽大平面应朝下。
3. 大面积薄壁部位应朝下或侧立，利用下部更高的金属液压力保证充型，避免浇不足、冷隔缺陷。
4. 厚大部位布置在分型面上部/侧部，便于实现顺序凝固补缩，减少缩孔缩松。
5. 尽可能减少型芯数量，便于型芯的固定、排气和检验。

## 工艺作用

### 对充型模式的影响

合理的浇注位置可实现金属液分层、层流充型，避免多股流对冲、分散液滴裹气等不良现象。长窄类铸件若浇注位置选取不当，易产生漩涡卷入大量气体导致冷隔缺陷 [[S45,S38]]。

### 对排气效果的决定作用

浇注位置直接决定型腔排气效果：若布置不当，易形成充型死角导致型腔局部气体无法排出，产生气孔、铸件表面烧焦等缺陷。适配排气槽/溢流槽布局的浇注位置可实现型腔气体随金属液推移有序排出，大幅降低裹气风险 [[S45,S14]]。

### 对压力传递与补缩的影响

将浇口设置在铸件厚壁热节区域，可保证压射增压压力有效传递到待补缩部位，配合厚壁向薄壁的顺序凝固特性，显著降低缩孔、缩松缺陷产生概率 [[S38,S37]]。

### 对模具热平衡的影响

分布均匀的多浇口浇注位置可以使金属液热量分散传递至模腔不同区域，避免局部区域热量集中，使模具型腔温度场分布更均匀，满足不同部位同步充型、同步凝固的工艺要求 [[S45,S9]]。

### 对模具服役寿命的影响

浇注位置设置不当会导致金属液高速直接冲击模具型芯或型壁，加剧模具冲蚀、粘模和热疲劳裂纹生成速率。符合NADCA标准的合理浇口位置可分散金属液流向、避免直接冲刷模具工作面，有效降低模具损耗，延长服役寿命 [[S45,S5]]。

## 缺陷关联机制

### 气孔

浇注位置设计不合理导致的气孔形成机理：金属液导入方向不当、流动速度过高产生喷射，或者过早堵住排气道、正面冲击型壁形成漩涡包裹空气，最终形成气孔缺陷。该类气孔多分布于排气不良或深腔区域。锌合金压铸中不当的内浇道位置是型腔充填过程中卷气的核心诱因，后续会导致铸件内部出现游离气孔，或在烘烤、电镀后表面鼓泡显露 [[S30,S44]]。

### 冷隔

不当的浇口位置使得金属液浇注流路过长，不同金属流股对接未完全熔合、结合力薄弱，过早凝固的金属流汇合后无法形成完全冶金结合，最终形成边缘光滑的线性冷隔缺陷[[S49]]。在镁合金压铸中，不当的浇口位置会使排溢系统设计无法覆盖远端充填区域，产生“憋气”反压，加剧多股液流汇合区域的冷隔形成[[S13]]。

### 浇不足（欠铸）

不当的浇注位置使得金属液充型远端距离浇口过远，充填过程中温降过大，同时远端区域排气不畅，型腔背压升高，最终导致远端轮廓无法完整充型。浇不足缺陷多出现于远离内浇口的薄壁、低排气区域[[S40,S20]]。

### 缩松与缩陷

浇注位置布置在远离铸件厚壁、壁厚突变区域时，保压压力无法通过未完全凝固的金属液有效传递到厚壁部位，厚壁处金属凝固收缩得不到后续金属液的补缩，最终形成内部缩松、表面缩陷[[S30,S32]]。

### 飞边

浇注位置设计不当导致内浇口流速过高，形成过高的压力冲击峰，超出锁模力对应的型腔承载极限，金属液从分型面间隙溢出形成薄片状飞边[[S30,S32]]。

### 缺陷排查逻辑

当固定所有压铸工艺参数后，缺陷位置始终集中在充填路径远端、无溢流槽布置区域时，优先判定为浇注位置选型不当导致的流场异常。排查步骤如下 [[S11,S53]]：

① 确认缺陷分布是否随流路从浇口向末端呈现温度梯度正相关。
② 排查远端区域排气槽、溢流槽是否被主流路完全覆盖。
③ 通过数值模拟比对现有浇口位置下的充填顺序、温度场分布，确认是否存在提前凝固区域。

## 应用场景与典型案例

### 薄壁壳体类——搅拌器外壳

ADC12 铝合金搅拌器外壳，外形尺寸260 mm × 260 mm × 280 mm，壁厚1–2 mm，属于深腔薄壁壳体。浇注位置选型为侧方切入式进浇，严格控制内浇口方向和尺寸以保证深腔充填完整，搭配大排气块优化远端排气，避免冷隔、浇不足缺陷。高速压射速度设定为5–6 m/s，高速切换点在内浇口前约30 mm处 [[S56,S28]]。

### 高强度结构支架类——变速器后体

铝合金变速器后体铸件，平均壁厚6 mm、局部4 mm，力学性能要求高，无冷隔、裂纹缺陷。浇注位置选型为从铸件厚壁侧进浇，保证压力可以有效传递到受力厚壁部位，实现厚壁处的补缩，避免缩松降低力学性能。比压设定为63 MPa，浇注温度控制在640–650 ℃ [[S4,S16]]。

### 外观装饰件——锌合金外观件

高表面质量要求的锌合金外观件，采用分区流量均衡的浇注位置设计，多个内浇口的通流面积按对应充填区域的金属体积比例分配，避免流股对冲、汇合不良形成冷流缺陷，保证所有外观面的金属流交汇位置不在外观装饰区域。最终铸件表面完整性显著提升，冷流缺陷几乎完全消除 [[S18,S48]]。

### 铝合金汽车发电机上盖优化案例

原单浇口浇注位置设计导致不对称充填，远端低温区域产生冷隔、卷气缺陷。优化改为双Y型分流式的双浇注位置设计，充填顺序高度对称，温度分布均匀，冷隔和卷气缺陷完全消除。同时浇口处金属流速从128 m/s降低至62 m/s，减少金属液对模具的冲蚀，延长模具寿命 [[S47,S39]]。

### 反面设计案例——圆锥齿轮

将齿轮工作面布置在远离浇口的充型末端属于典型错误选型，该错误会导致工作面出现冷隔、气孔、轮廓不完整等缺陷。重要受力/功能工作面需避免安排在充型远端 [[S3,S33]]：

## 相关标准

### GB/T 13822-92《压铸有色合金试样》

该标准明确规定压铸试样浇注位置的布置要求：拉力试样布置在横浇道上方，圆试样在一侧，扁试样和冲击试样在另一侧。同时溢流槽体积需与试样体积相近，保证金属液平推流向溢流槽，实现型腔热平衡、排出夹渣和气体 [[S17]]。
