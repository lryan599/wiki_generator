## 概述

离散相模型（Discrete Phase Model，DPM）是计算流体动力学（CFD）中基于欧拉-拉格朗日框架的弥散多相流仿真模型[[S18,S6]]。在该框架下，流体相作为连续介质，通过直接求解均化纳维-斯托克斯（N-S）方程获得流场；离散相（如固体颗粒、气泡、液滴）则作为离散介质，通过跟踪大量粒子的运动轨迹进行求解。两相之间可实现动量、质量和能量的双向交换[[S6,S17]]。

DPM 在 ANSYS Fluent 等主流 CFD 软件中被广泛实现，主要适用于离散相体积分数较低的稀疏两相流场景[[S18]]。在压铸工艺数值模拟中，DPM 常用于追踪熔融金属中的夹杂物、氧化膜、气泡或喷射液滴的运动行为，为缺陷预测与工艺优化提供依据[[S17]]。

## 定义与基本原理

离散相模型将流体视为连续介质，离散相视为离散介质，二者组成弥散多相流体体系[[S6]]。模型基本假设包括：离散相颗粒简化为球形，密度均一、表面光滑；稳态气动阻力为颗粒运动的主导作用力，常规计算中可忽略其余次要作用力（包括视质量力、热泳力、布朗力、Saffman升力等）[[S6]]。

颗粒在湍流场中的扩散通常采用随机轨道模型（DRW），通过颗粒作用力平衡方程叠加流体脉动引发的瞬时速度来计算颗粒运动轨迹[[S6]]。

DPM 的数值求解思路为：首先完成连续相流场的独立求解与收敛，随后定义离散相喷射源（包括初始位置、速度、粒径分布、材料属性等），在指定区域内逐颗粒计算运动轨迹。常规稳态仿真采用非相间耦合方案；需考虑双向作用的场景则启用瞬态求解模式，实现每轮迭代的相间数据交换[[S6,S17]]。

## 核心假设与适用范围

### 体积分数阈值

DPM 的核心适用条件是离散相体积分数通常低于 10%，部分工程应用场景允许阈值上限延伸至 10%~12%[[S6,S28,S17]]。仅在该阈值范围内，可忽略离散相之间的相互作用以及离散相对连续相流场的反向干扰[[S6,S28]]。当压铸仿真中夹杂物、气泡等离散相体积占比超过该阈值时，DPM 将完全失效，无法准确模拟颗粒间相互作用[[S18,S11]]。

### 耦合方式

DPM 框架下默认采用非相间耦合（单向耦合）规则。在离散相体积分数远低于 10% 时，颗粒间的碰撞、聚合等相互作用可忽略[[S6]]。仅当离散相质量载荷较高、密相颗粒聚集行为不可忽略时，才需显式启用对应的颗粒碰撞子模型[[S6,S9]]。典型工业案例中，钢液夹杂物追踪仿真可采用单向耦合 DPM，明确定义夹杂物受力平衡方程包含重力浮力合力、曳力、升力、压力梯度力、虚拟质量力共 5 类受力来定量表征夹杂物运动[[S16]]。

## 在压铸工艺中的应用场景

### 流态示踪与充型分析

DPM 可用于追踪压铸充型过程中不同时刻金属液粒子的运动路径与流向，识别易发生紊流冲击、卷气、合流不足的区域，辅助优化浇注系统设计，降低粘模、冷隔等充型缺陷发生率[[S27]]。

### 夹杂物追踪

压铸过程中常见的夹杂物类型包括尖晶石类铝镁氧化物 Al₂O₃-MgO、富 Zr/Ti 粗大初生相、冲头/模具润滑残留形成的硬质金属间化合物等[[S19,S25]]。DPM 可直接追踪不同粒径夹杂物在充型流场中的迁移路径与最终捕获位置，用于预测铸件夹渣缺陷分布[[S13]]。

### 半固态浆料颗粒模拟

在铝合金离心凝固等场景中，可基于 DPM 低体积分数颗粒追踪逻辑，对半固态压铸过程中固相析出颗粒的沉降与分布规律进行求解。典型工业参考案例中对直径 0.01 mm 的 Mg₂Si 相、0.02 mm 的 AlFeMnSi 相进行运动轨迹追踪与最终分布预测[[S17]]。

### 气泡/气孔追踪

压铸充型过程中卷入的空气、润滑剂分解气、水蒸气等可设置为 DPM 离散相进行轨迹追踪，结合排气系统仿真结果优化排气孔位置与尺寸，降低卷气缺陷风险[[S3,S29]]。

### 模具润滑剂喷雾仿真

高压压铸喷涂工序中，可将润滑剂液滴设置为 DPM 液滴相，追踪喷雾在模具表面的沉积分布与热交换过程，优化喷雾路径与时间参数，避免润滑剂局部过量分解产气导致气孔缺陷[[S5,S22,S24]]。

## 模型关键参数与设置

### 颗粒类型选型

在压铸 DPM 仿真中，应根据追踪对象的物理本质选择对应颗粒类型[[S17]]：

| 应用场景 | 颗粒类型 |
|---------|---------|
| 非金属夹杂物追踪 | 惰性颗粒 |
| 半固态固相颗粒追踪 | 惰性颗粒 |
| 充型气泡追踪 | 空气/气相惰性颗粒 |
| 润滑剂喷雾液滴追踪 | 可蒸发/可热解的液滴颗粒 |

### 作用力模型

DPM 仿真中需根据压铸工况选择适用的作用力模型。常用曳力模型为 Schiller-Naumann 公式[[S1]]：

- Re < 1000 时：C_D = 24/Re × (1 + 0.15 × Re⁰·⁶⁸⁷)
- Re ≥ 1000 时：C_D = 0.44

升力可采用修正的 Saffman 升力模型，适配压铸高速紊流流场中的颗粒受剪切力场景。虚拟质量力系数文献典型取值为 0.5，适用于密度远低于熔融金属的气相、低密度夹杂物的受力计算[[S1]]。

研究中也可采用更完整的多力耦合方案。Chen 等人耦合 LES 湍流模型与 DPM 时，完整计入曳力、升力、压力梯度力、虚拟质量力和壁面润滑力对颗粒运动的影响，准确获得了卷渣渣滴的尺寸、空间分布与速度特征[[S15]]。

### 边界条件类型

压铸 DPM 仿真中模具壁面的边界条件设置规则[[S8,S26]]：

| 边界类型 | 适用场景 | 物理含义 |
|---------|---------|---------|
| 反射（Reflect） | 高速充型阶段未接触凝固前沿的型腔内壁面 | 颗粒撞击壁面后反弹 |
| 捕获（Trap） | 铸件与模具接触的已凝固区域壁面 | 夹杂物/气泡被凝固前沿包裹，无法继续运动 |
| 壁膜（Wall-film） | 喷涂仿真时的模具热壁面 | 润滑剂液滴撞击壁面后附着形成油膜 |
| 逃逸（Escape） | 模具分型面的排气孔、溢流槽出口 | 气泡/轻质夹杂物通过排气系统排出型腔 |

### 粒径分布

压铸铝合金中氧化物夹杂物的尺寸集中分布于 2~50 μm 区间，数密度随粒径增大呈指数递减趋势，可采用 Rosin-Rammler 分布进行拟合[[S4,S12]]。该参数取值为铸造领域已公开的工程统计参考值，需根据实际压铸合金牌号调整。连铸钢液中夹杂物平均三维数密度的实测数据显示，15 μm 以下夹杂物数密度可达 1.27 × 10¹² 个/m³[[S4]]。

### 数值离散格式

压铸 DPM 仿真的推荐数值离散格式配置[[S7]]：

| 离散类型 | 推荐方法 |
|---------|---------|
| 时间离散 | 欧拉离散 |
| 梯度离散 | 高斯线性插值离散 |
| 拉普拉斯项离散 | 带正交修正的高斯理论线性插值 |
| 对流项离散 | 二阶精度的高斯线性迎风插值 |

## 限制与注意事项

### 高颗粒浓度不适用

当离散相体积分数超过 10%~12% 时，DPM 不再适用，需改用欧拉-欧拉多相流模型（如 Mixture 模型或 Eulerian 模型）[[S18,S28]]。

### 计算成本限制

DPM 采用逐颗粒独立计算运动轨迹的模式。当压铸充型过程中待追踪的夹杂物数量达到 10¹⁴ 个/m³ 量级时，计算量将呈指数级膨胀，无法实现全尺度全颗粒的并行求解[[S9]]。实践中常采用“包裹”粒子方法，使每一个计算粒子代表若干个实际颗粒，以降低计算量[[S9]]。

### 湍流弥散处理精度

DPM 内置的 DRW 随机游走模型仅适配低浓度稀疏相、球形简化颗粒的假设场景。在压铸高湍流、多形态多尺寸夹杂物密集分布的工况下，无法高效准确模拟颗粒的碰撞、聚合或破碎行为[[S9]]。

## 与其他多相流模型的关系

在 ANSYS Fluent 提供的主流多相流模型中，VOF、Mixture 和 Eulerian 模型均采用欧拉-欧拉框架，DPM 是唯一基于欧拉-拉格朗日框架的离散相追踪模块[[S18]]。

**两类框架对比**[[S18,S28,S11]]：

| 对比维度 | DPM（欧拉-拉格朗日） | VOF / Mixture / Eulerian（欧拉-欧拉） |
|---------|---------------------|--------------------------------------|
| 离散相处理方式 | 逐颗粒追踪运动轨迹 | 处理为互相贯穿的连续介质 |
| 适用体积分数 | < 10%~12% | 无严格限制，Eulerian 可处理高浓度 |
| 相间相互作用 | 默认忽略，可按需启用碰撞模型 | 自动计入相间耦合 |
| 计算成本 | 随颗粒数指数增长 | 随相数增加而增加 |

**压铸充型仿真中模型选择逻辑**：

- **VOF 模型**：通过追踪网格内各流体相的体积分数实现互不相溶流体界面的精准捕捉，适合充型自由表面演化、卷气行为预测[[S20,S21,S2]]。
- **Mixture 模型**：求解单一组混合流体的动量、连续性与能量方程，适合低体积分数的气泡上浮、小尺寸夹杂物沉降类需求，计算效率高于全 Eulerian 模型[[S18,S28]]。
- **Eulerian 模型**：为每个相独立求解控制方程，适合高浓度颗粒悬浮流、大量气泡聚合破碎类复杂多相流，计算精度高但资源开销远大于 DPM[[S18,S28]]。

在高端铸造仿真研究中，DPM 常与 LES 湍流模型、VOF 模型等耦合使用。Cho 等人将 LES、DPM 与 VOF 耦合应用于钢液-氩气两相瞬态流动研究，模拟结果与插钉板工业试验数据吻合[[S15]]。Chen 等人进一步将 LES、DPM 与传热、凝固、偏析模型耦合，通过改进夹杂物捕获判定条件，成功预测了连铸坯整个断面上夹杂物的数量与空间分布[[S15]]。

## 在软件中的实现

### ANSYS Fluent

ANSYS Fluent 中 DPM 的标准实现流程为[[S6,S17]]：

1. 完成连续相流场的独立求解与收敛
2. 定义离散相喷射源（初始位置、速度、粒径分布、材料属性等）
3. 设定边界条件（反射、捕获、逃逸、壁膜等）
4. 对指定区域内所有颗粒逐段计算运动轨迹
5. 稳态仿真采用非相间耦合，双向耦合场景启用瞬态求解模式

ANSYS Fluent 中 DPM 为四种多相流模型之一，与 VOF、Mixture、Eulerian 模型并列[[S18]]。

### OpenFOAM

OpenFOAM 平台的离散相相关功能集成在其原生开源求解器体系中，适配 DPM 类拉格朗日颗粒追踪逻辑，支持自定义求解器的二次开发，可基于现有基础模块组合实现压铸过程中气固、液固稀疏两相流的离散相轨迹仿真[[S23]]。

## 总结

离散相模型（DPM）是压铸数值模拟中追踪稀疏离散相行为的重要工具，在夹杂物分布预测、气泡追踪、喷雾仿真等场景中具有不可替代的优势。其核心适用条件为离散相体积分数低于 10%~12%，在该范围内可高效求解颗粒运动轨迹。当离散相浓度超出阈值时，应选用欧拉-欧拉框架下的 Mixture 或 Eulerian 模型[[S18,S28,S11]]。实际工程应用中，需根据追踪对象的物理本质合理选择颗粒类型、作用力模型和边界条件，并注意计算成本与湍流弥散精度的限制[[S9,S17]]。
