## 概述

球化包是球墨铸铁生产过程中用于对铁液进行球化处理的专用铸造装备，通用中文同义词包括球化处理包、球化铁水包，对应英文术语为 nodularizing ladle、spheroidizing ladle、treatment ladle for spheroidization 或 ductile iron ladle[[S9]]。球化包属于广义通用铸造装备，其工业应用几乎全部集中在砂型铸造球墨铸铁生产领域，在高压压铸领域公开应用案例极少，整体与压铸工艺关联度极低[[S9]]。国家标准 GB/T 5611—2017 中定义了相关术语"密容加镁包"（sealed spheroidizing treatment ladle），指在带有密封包盖的铁水包中用钟罩压入纯镁对铁液球化处理的装置[[S9]]。

## 分类与类型

主流球化包按工艺方式可分为四大类，不同类型在镁吸收率、工艺适用性和操作复杂度方面存在明确差异[[S7,S10]]。

| 类型 | 工艺特征 | 镁吸收率 | 适用场景 |
|------|----------|----------|----------|
| 冲入法球化包 | 包底设堤坝或凹坑，球化剂置入后覆盖，铁液冲入未放球化剂一侧 | 40%~45% | 常规球墨铸铁件生产，应用最广 |
| 盖包法球化包 | 冲入法的延伸，增加密封包盖 | 高于冲入法，可节约球化剂30% | 对镁吸收率和环保要求较高的场合 |
| 喷吹法球化包 | 以载流气体携带球化剂喷入铁液 | — | 铁液炉外处理 |
| GF转包 | 瑞士乔治·费歇尔公司1971年公布的专用转包，可用纯镁处理 | 约40% | 浇注量大或大批量生产，对原铁液含硫量≤0.3% |

## 结构与组成

冲入法球化包的核心结构包括包体、包底堤坝或凹坑、包盖（盖包法）、浇嘴及耐火材料内衬。

关键设计参数[[S7,S10]]：
- **高径比**：球化处理包内铁液的 H/D ≥ 1.5~2，用球化包处理半包铁液则违背后初衷
- **凹坑尺寸**：凹坑宽度以包底直径的 1/4~1/3 为宜，投影面积小的凹坑增加深度有利于延缓爆发
- **凹坑余深**：装入球化剂和覆盖剂后尚余 20~25 mm，铁液进入包坑与覆盖剂熔融成半固态物质，可延缓球化合金过早爆发，提高镁收得率

盖包法在冲入法基础上增加一个密封包盖，可稳定球化效果、减少铁液温降、使氧化渣量减少50%，同时降低车间烟尘和光污染[[S10]]。

## 工作原理与操作条件

### 球化处理基本冶金过程

球化处理是向铁水中加入球化剂促使石墨成球状。镁是强球化元素，但密度小、沸点低（1107 ℃），直接加入铁水会剧烈沸腾并严重烧损[[S8]]。稀土镁合金球化剂综合了稀土和镁的优点，加入量一般为铁水量的 1.3%~1.8%，是我国应用最广的球化剂[[S8]]。

冲入法操作规范[[S1,S6]]：

| 操作步骤 | 工艺要求 |
|----------|----------|
| 包体预热 | 预先预热至 600~800 ℃（暗红或红色） |
| 原铁液温度 | 处理 0.5~3 t 铁液时控制在 1400~1430 ℃ |
| 球化剂粒度 | 15~30 mm，粉状物占比 ≤10% |
| 球化剂放置 | 置入包底堤坝或凹坑区域 |
| 覆盖顺序 | 球化剂上方依次覆盖硅铁粉（粒度等于或小于球化剂），再覆盖无锈铁屑或草灰、苏打、珍珠岩等集渣剂 |
| 首次冲入量 | 铁液冲向未放球化剂一侧，冲入总量的 1/2~2/3，深度 ≥200~260 mm |
| 反应控制 | 沸腾持续 ≥1 min，反应过迟可用钢钎捅破覆盖层 |
| 补充冲入 | 沸腾将结束时冲入剩余铁液 |
| 扒渣 | 加集渣剂搅拌后彻底扒渣，再覆盖保温剂 |
| 处理温降 | 0.5~3 t 包铁液降温 50~100 ℃ |
| 球化反应总时长 | 通常 60~120 s |

### 球化情况判断

铁液冲入一定高度（>250~260 mm）后开始起爆，均匀持续沸腾 ≥1 min 表示反应正常；反应过猛喷出铁液表示烧损过大；补加铁液时液面逸出镁光及白黄火焰表示正常；处理后铁液表面易形成氧化膜表示正常[[S6]]。

## 缺陷与影响

### 球化不良与球化衰退

GB/T 41972—2022 标准明确定义[[S2]]：
- **球化不良**：因原铁液含硫量高、球化剂加入量不足或球化处理后停留时间过长，导致石墨未球化或球化率过低
- **球化衰退**：铁液含硫量过高或球化处理后停留时间过长引发的球化不良衍生缺陷

### 球化包操作层面的防控措施

从球化包操作角度防控球化不良缺陷的措施[[S11,S5]]：
- 避免铁液直接冲刷球化剂，防止覆盖物被冲跑、合金受高温冲击导致镁烧损
- 保证铁液快速冲到包容量的 2/3 以上高度，增加镁在铁液中上浮路程以提高吸收率
- 浇注后及时清理包底残留熔渣，防止残留铁液高温烘烤后续球化剂
- 采用双包轮换使用，避免包体过热提前烘烤球化剂造成镁损耗
- 球化剂装填后须摊平、舂实，表面覆盖物应填满合金块缝隙并有约 10 mm 覆盖层

## 工艺衔接关系

球化包的常规作业衔接逻辑为：承接自熔炼炉出铁槽的原铁液，完成球化处理和一次炉前孕育后，转运至浇注工位，可在随流孕育装置配合下对接浇注系统完成后续铸件浇注工序[[S13]]。以离心球墨铸铁管生产为例，冷模法采用二次孕育（球化包内 0.4%~0.6% + 随流 0.1%~0.25%），热模法采用三次孕育（球化包内 0.4%~0.6% + 扇形包内 0.3%~0.4% + 随流 0.2%~0.3%）[[S13]]。
