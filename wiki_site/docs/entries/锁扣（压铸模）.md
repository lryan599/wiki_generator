## 概述

锁扣（压铸模）是压铸模具中的一类核心锁紧与定位功能组件。在国家标准 GB/T 8845-2017《模具术语》中，压铸模锁扣对应的英文术语为 latch lock 或 locking device [[S25]]。在压铸模领域，锁扣至少存在三类主流应用场景：① 与侧向抽芯滑块配套的锁紧楔块（latch lock/locking block），用于在压射充型阶段锁紧滑块防止其在熔融金属高压作用下发生退让位移；② 大型压铸模自带的可在模内直接吸收分模胀型力的整体式锁紧装置（interlock）；③ 分体式金属型压铸模具的手动合模锁紧构件[[S34,S14,S5]]。本词条所述的锁扣需与锻模平衡错移用锁扣、冲压模具的定位止动 latch 以及压铸机整机的合模锁机构严格区分，锁扣属于模具本体自带的组件[[S34,S14,S5]]。

## 工艺作用

滑块配套锁扣的核心作用是抵消高压金属液作用在侧型芯上的反压力，防止滑块在压射充填过程中发生退让位移，避免斜导柱在承压状态下产生弯曲变形，保障压铸件侧向孔、倒扣等特征结构的尺寸精度[[S34]]。

大型压铸模所采用的整体式锁扣（V1型/V2型）可在模具内部直接吸收大部分分型面胀型力，从而降低对压铸机额定锁模力的需求，在锁模力不足的压铸机上生产大平面薄壁压铸件[[S14,S36]]。

分体式金属型所用手动锁扣的功用在于保证两半型合模状态下贴合紧密，消除分型面溢料及上下模错位缺陷，保障小批量生产铸件的尺寸一致性[[S5]]。

## 分类与结构

### 滑块配套锁扣（侧向锁紧块）

滑块配套锁扣依据结构形式可分为六种典型类型[[S34]]：

| 类型 | 结构特征 | 优缺点 |
|-------|----------|--------|
| 普通螺钉锁紧式 | 楔紧块以螺钉紧固于模套板 | 结构简单，调整方便；刚性较差，螺钉易松动 |
| 加长型辅助楔紧块 | 在原有楔紧块外侧附加辅助块以增加整体刚性 | 刚性较普通式有所提升 |
| 模套内置式锁紧块 | 楔紧块固定于动模套板内 | 强度高、刚性好，适用于反压力较大的工况 |
| 整体式锁紧块 | 锁紧块与模套一体成型 | 锁紧力大但材料消耗多且不可调整 |
| 改进型可换楔紧块 | 楔紧块为独立制作的可更换件 | 可进行热处理，耐磨性好，便于调整、维修方便 |

### 大型压铸模内置整体锁扣

大型压铸模自带的整体或分体式锁扣主要用于在模具闭合后将分型面胀型力直接内耗，具体分为两类[[S14]]：

- **V1型（大平面活动楔块式）**：在模具两侧对称布置活动楔块，受力均匀、稳定性较好。楔角 α 一般选大于 6°，可实现力学自锁，较小的推力 P₁ 即可输出较大的合模力 Q₁，在不计摩擦时 Q₁ = P₁/(2 tan α)。
- **V2型（锁销式）**：以锁紧销作为核心承压件，结构紧凑，适用于模具四个角均无干涉、无侧向滑块的布局场景。锁紧销的直径决定了可达的合模力，锁紧销应与型腔投影面积对称布置以便均匀接受分模力。

### 分体金属型用手动锁扣

手动金属型用锁扣主要用于小批量压铸或重力浇注工况下的模具锁紧，常见四类结构[[S5,S22]]：

| 类型 | 组成 | 适用场景 |
|------|------|----------|
| 偏心锁 | 手把、锁扣主体、偏心转件、轴销等 | 操作便捷，适用于中小型金属型 |
| 摩擦锁（摩擦卡） | 仅含摩擦紧固手把与模具凸耳两个主要零件 | 结构极简，拆装快速，适用于铰链式中小型金属型 |
| 套钳锁（螺旋锁） | 夹板、螺杆、固定平板、转轴、凸耳等 | 可承受极高锁紧力，适用于大中型金属型；螺杆采用方形或梯形螺纹，需加大间隙防止受热咬死 |
| 楔销锁 | 带斜度的楔销、两组带锥孔的凸耳 | 适用于铰链式金属型，锥孔斜度 4°～5° |

## 材料与热处理

### 推荐材料

压铸模锁扣属于承受侧向作用力的模具耐磨结构件，常用推荐材料牌号为 T10A（优质碳素工具钢）、Cr12MoV（合金模具钢）、H13（对应国标 4Cr5MoV1Si，属热作模具钢），选型需满足压铸过程中抗冲击、抗磨损、抗热疲劳的性能要求[[S10,S23]]。NADCA#207-2015 及 NADCA#207-90 北美压铸协会标准可作为 H13 类锁扣材料的质量验收依据[[S2]]。

### 硬度规范

锁扣淬火后硬度需根据生产的压铸合金类型区别控制[[S28,S23,S30]]：

| 压铸合金 | 推荐硬度（HRC） |
|----------|----------------|
| 铝合金、镁合金 | 43～48 |
| 锌合金 | 48～50 |
| 铜合金 | ≤42（不超过 44） |

一种常见的实践方案为：铝/镁合金压铸用锁扣热处理硬度控制于 HRC 43～47，锌合金压铸用锁扣取 HRC 48～50，铜合金压铸用锁扣不超过 HRC 42[[S28]]。

### 表面处理

锁扣工作面推荐进行软氮化或氮化处理，氮化层深度 0.08～0.15 mm，表面硬度 HV ≥ 600，可显著提升配合面耐磨性，防止粘模失效[[S10,S7,S23]]。

## 设计要点与参数

### 锁紧角度

滑块配套锁扣的锁紧块斜面角度（α′）应比对应斜导柱的倾斜角（α）大 3°～5°。此差值可保证开模初始阶段锁紧块先快速脱离滑块承压面，避免不必要的摩擦磨损；合模临近终点时锁紧块才接触滑块实现锁紧，使斜导柱与滑块侧孔完全脱空，避免斜导柱在压射过程中直接承受侧向胀型载荷[[S34,S4,S16]]。

大型 V1 型锁扣的楔角 α 需大于 6°以实现力学自锁[[S14]]。

### 典型结构参数

压铸模锁扣的结构设计可参照以下参数体系（基于通用锁扣设计原则，与锻模锁扣设计参数可相互借鉴）[[S35,S15,S11]]：

| 锁扣类型 | 参数要求 |
|----------|----------|
| 平衡锁扣（形状锁扣） | H = h + (10～30) mm，b = (1.5～2) H，α = 3°～5° |
| 圆形锁扣 | H ≥ 30 mm，宽度 B ≥ 1.5H，α = 5°，间隙 S ≤ 0.3 mm |
| 纵向锁扣 | H ≥ 25 mm，B ≥ 1.5H，α = 5°，Δ = 3～5 mm |
| 侧面锁扣 | H ≥ 25 mm，B ≥ 2H，L ≥ 2B，α = 5°，δ ≤ 0.3 mm |
| 四角锁扣 | 作用与侧面锁扣相似，用于防止各方向错移及相对转动 |

配合面间隙最大不超过 0.3 mm；导面采用零间隙设计时 δ = 0[[S35]]。

### 圆角设计

过渡圆角推荐值：R₁ = R₄ = 2～3 mm，R₂ = R₃ = 3～5 mm。凸锁扣突出部分优先布置于动模侧，便于生产中清理氧化皮及防止卡模[[S35,S11]]。

### 连接方式

锁扣与压铸模模板、滑块的常规连接方式为内六角螺钉加定位销组合紧固，以保证配合面定位精度及抗侧向冲击强度[[S19]]。

### 公差与配合

锁扣与安装座板的固定配合优先采用 H7/m6 或 H7/k6 精度等级；锁扣与滑块的滑动导向配合按其对应标准选取间隙精度；所有未注形位公差应符合 GB/T 1184-1996 规定的 C 级精度要求[[S17,S20]]。

## 与其他模具部件的关系

### 与导柱导套的配合

锁扣布置位置需避开模具四角导柱导套的安装区域，不得与动、定模合模运动路径发生干涉。合模时导柱导套优先进入导向配合完成粗定位，随后锁扣接触并逐步完成精锁止动作[[S26,S1,S13]]。导柱和导套的中心位置至模具边缘距离一般取 1.5d（d 为导柱直径）[[S26]]。

### 与滑块（行位）的配合

锁扣的受力承压面应与滑块的对应锁止面 100% 完全贴合，不得存在局部间隙或翘曲等缺陷，确保压射充型过程中滑块不发生退让位移。滑块导向槽的尺寸公差需满足行业通用 f7 精度要求[[S17,S34]]。

### 与斜顶及顶出机构的配合

锁扣的布局须与斜顶的全行程运动轨迹及顶出机构的动作范围保持充足的安全距离，完全避免在开模、顶出执行过程中锁止块与运动部件发生碰撞干涉[[S17]]。

## 常见缺陷与失效

锁扣常见的四类失效模式及对应处置方案[[S6,S27,S16]]：

| 失效模式 | 根因 | 对策 |
|----------|------|------|
| 工作面磨损 | 锁止面硬度不足、频繁开合模引起的摩擦损耗 | 选用 T10A 材料，热处理硬度控制在 HRC 40～50 区间；锁止面附加耐磨涂层或实施表面氮化处理 |
| 咬死（热咬合） | 配合间隙过小、模具长期高温运行且润滑不足 | 预留适配压铸热膨胀量的合理配合间隙，定期加注耐高温润滑剂 |
| 断裂 | 锁扣受力截面强度不足、侧向锁止载荷过载 | 增大锁扣受力截面尺寸，按实际侧向胀型力完成强度校核 |
| 定位偏移 | 锁扣安装定位销松脱、锁止面贴合度不足 | 定期巡检定位销紧固状态，修研锁止承压面保证全幅贴合 |

锁止零部件磨损或松动将直接导致滑块退让、配合间隙异常增大，高温铝液侵入滑块锁止面缝隙，最终在压铸件侧孔位置产生毛刺及轮廓错型等质量缺陷。日常生产中需定期巡检锁扣的磨损状态，提前排查隐患[[S29]]。

## 应用场景

1. **大侧向力工况**：生产大型平面类压铸件时，采用外部附加的独立锁扣结构，将侧向分模胀型力在模具本体内部直接吸收，大幅降低对压铸机额定锁模力的需求，使小吨位设备具备生产大投影面积铸件的能力[[S36,S14]]。
2. **多向分型/多侧抽芯模具**：带多个不同方向侧抽芯的复杂压铸模，沿各抽芯的侧向胀型力方向布置对应独立锁扣，可分别抵消不同方向上的金属液侧向压力，避免多个滑块同步退让产生铸件轮廓超差[[S19]]。
3. **超高定位精度要求的压铸件**：对合模精度及侧孔位置公差有极高要求的精密汽车结构件、高端通信器件生产场景中，采用模面整体式对锁锁扣可抵消普通导柱导套无法抵消的热变形错移，将合模位置精度控制在较高水平[[S26]]。

## 相关标准

中国现行压铸模领域国家标准中，与锁扣直接或间接相关的标准主要包括：

- **GB/T 8845-2017《模具术语》**：明确了压铸模锁扣对应的英文术语为 latch lock 及 locking device，属于压铸模核心锁紧类功能零件[[S25]]。
- **GB/T 8844-2003《压铸模技术条件》**：规定了压铸模零件的总体技术要求，模具零件不允许有裂纹，未注形位公差按 GB/T 1184-1996 C 级执行[[S3,S24]]。
- **GB/T 4679-2003《压铸模零件技术条件》**：规定了压铸模零件的通用技术条件[[S31]]。
- **GB/T 4678.1～2003 系列《压铸模零件》**：为压铸模标准零部件（包括锁紧配合的相关结构件）提供了尺寸要素[[S31]]。

国际方面，**NADCA #207-2015** 及 **NADCA #207-90** 可作为 H13 类压铸模具锁扣材料的质量验收标准[[S2]]。
