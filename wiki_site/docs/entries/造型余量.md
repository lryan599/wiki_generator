## 概述

“造型余量”（pattern allowance，也称模样余量）是传统砂型铸造领域长期使用的正式术语，指制造砂型所用模样上额外增设的各类预留量总和，用以补偿铸造过程中产生的尺寸变化与表面缺陷[[S12,S40]]。该术语不属于压铸领域现行标准体系中的正式规范表述；在压铸语境下，同类功能要求被拆分为若干独立的、有明确定义的余量条目分别管理[[S1,S11]]。

压铸件的余量设计逻辑与砂型铸造存在根本差异：压铸件表面为快速凝固形成的细晶粒致密层，力学性能与致密度最优，因此所有余量数值普遍远小于砂型铸造，且不推荐做过大切削加工，应优先采用校正、拉光、压光、整形等精整加工方式，保留高强度致密表层[[S32,S6]]。

## 压铸场景下的余量构成

依据国内铸造标准体系，压铸造型相关余量由三类核心子余量组成[[S38]]：

- **收缩余量（shrinkage allowance）**：为补偿铸件从液态至固态冷却过程中的尺寸收缩，模具型腔尺寸相对于铸件图纸尺寸额外增大的数值[[S38]]。该余量是模具成型尺寸计算的核心输入参数。

- **机械加工余量（machining allowance）**：为保证机加工后表面精度，在铸件待加工部位额外预留、后续切削去除的金属层厚度[[S38]]。在GB/T 6414-1999标准中明确定义为“要求的机械加工余量（RMA）”——在毛坯铸件上为后续机械加工去除铸造表面影响、达到要求表面特征和尺寸精度预留的金属余量[[S37,S15]]。

- **工艺补正量（molding allowance）**：由于工艺上的原因，在铸件相应部位非加工面上增加的金属层厚度，用以抵消铸造过程中因工艺条件波动产生的尺寸误差[[S38]]。

值得注意的是，传统砂型铸造造型余量体系中的“震型余量”（shake allowance，即敲模余量）和“变形补偿余量”（distortion allowance）在压铸模具设计中并不作为独立余量参数出现，其功能分别由模具结构设计、合理的顶出布局及工艺参数优化承担。

## 机械加工余量（RMA）的确定

### 标准体系

GB/T 6414-1999在技术内容上等效采用ISO 8062:1994《铸件尺寸公差与机械加工余量体系》，取代了旧版GB/T 6414-1986和GB/T 11350-1989[[S22,S15]]。该标准体系的关键特点为：

- 用“要求的机械加工余量（RMA）”取代了原先将铸件尺寸公差下偏差计入余量的旧定义[[S15]]；
- 铸件要求的机械加工余量数值由最终机械加工后成品铸件的最大轮廓尺寸决定，同一铸件不同加工面的RMA值通常相同[[S15,S37]]；
- 在附录A中补充了ISO 8062:1994未规定的压铸件大批量生产通常可达到的公差等级[[S15]]。

### 公差与余量等级

GB/T 6414-1999中铸件尺寸公差共设16级，代号CT1至CT16。压铸件常规选用尺寸公差等级范围为CT4～CT7，无特殊要求时推荐CT7级[[S1,S30]]。壁厚公差在CT1～CT15范围内比其他尺寸的一般公差粗一级，例如标注一般公差CT10时，壁厚公差为CT11[[S30]]。

机械加工余量（RMA）共设A、B、C、D、E、F、G、H、J、K共10个等级。压铸件常规选用RMA等级范围为B～D级，无特殊要求时推荐选用D级[[S1,S30]]。A、B级仅在供需双方就夹持面、基准面等提前商定模样装备、铸造工艺和机加工工艺的成批生产特殊场合使用[[S1,S30]]。锌合金、铝合金、镁合金、铜合金四类压铸合金对应的RMA等级范围均为B～D级[[S9]]。

### 余量与公差的尺寸换算

双侧加工的铸件毛坯尺寸与加工余量的核心换算公式为[[S36]]：

```
R = F + 2RMA + CT/2
```

式中：
- R —— 铸件毛坯尺寸
- F —— 最终机械加工完成后的成品铸件尺寸
- RMA —— 要求的机械加工余量
- CT —— 对应尺寸的铸件公差

单侧加工场景下，余量换算需结合脱模斜度对应尺寸差进行调整，避免加工余量不足[[S19,S45]]。

### 压铸件加工余量推荐值

压铸加工余量取值原则为尽量最小化，以保留表层致密组织、避免暴露内部气孔和缩松缺陷[[S14,S32]]。不同压铸合金在不同公称尺寸区间的推荐加工余量如下表所示[[S7,S41]]：

| 公称尺寸（mm） | 锌合金 | 铝合金 | 镁合金 | 铜合金 |
|:---:|:---:|:---:|:---:|:---:|
| <100 | 0.1～0.4 | 0.1～0.4 | 0.1～0.4 | 0.3～0.5 |
| 100～200 | 0.2～0.5 | 0.2～0.6 | 0.2～0.6 | 0.3～0.7 |
| >200 | 0.5～0.7 | 0.5～0.8 | 0.5～0.8 | 0.7～1.0 |

> 所有压铸件最大加工余量不超过1.0mm。此表不包括深孔或大侧面因脱模斜度额外产生的余量需求[[S41]]。

压铸件机械加工余量一般取0.3～0.5 mm；壁厚超过5 mm的大型铸件，加工余量不应超过0.8～1.0 mm[[S6,S32]]。铰孔加工余量有单独推荐值[[S5,S8]]。

国际方面，NADCA标准给出的机加工余量推荐最小值为0.010英寸（约0.25 mm），并指出总机加工余量为该最小值叠加铸造公差与机加工公差之和，强调应尽可能控制最小机加工余量以保留表层性能[[S17]]。

### 加工余量的基准选取规则

当压铸件留有不同加工余量配置时，成型尺寸基准选取规则如下[[S45]]：

| 余量配置 | 型腔基准 | 型芯基准 |
|:---|:---|:---|
| 无加工余量 | 以大端为基准 | 以小端为基准 |
| 双面留加工余量 | 以小端为基准 | 以大端为基准 |
| 单面留加工余量 | 以非加工面大端为基准 | 以非加工面小端为基准 |

当加工余量受脱模斜度影响时，应尽可能控制大端和小端的余量值均落入推荐范围[[S14,S20]]。

余量在毛坯图上的标注，压铸件待加工表面用符号“√”标明，尖角指向被加工表面，应用示例：√0.5表示该平面留有加工余量0.5 mm[[S6]]。

## 收缩余量的确定

### 计算收缩率

压铸模具设计时采用的收缩余量以“计算收缩率 φ”为参数，其特殊性在于需同时包含两个分量[[S33,S43]]：

1. **压铸件的冷却收缩量**——合金从脱模温度降至室温的尺寸缩小量；
2. **钢制成型零件的受热工作膨胀量**——模具型腔从室温升至工作温度后的尺寸增大量。

计算公式为[[S33]]：

```
φ = (A' - A) / A × 100%
```

式中：A'为计算后的模具成型尺寸，A为压铸件公称尺寸。

### 影响收缩率取值的因素

计算收缩率的最终取值受以下工况条件综合影响[[S33,S43]]：

| 影响因素 | 对收缩率的影响趋势 |
|:---|:---|
| 铸件结构复杂度与型芯数量 | 结构复杂、型芯多、收缩受阻大 → 收缩率偏小 |
| 铸件壁厚 | 薄壁件收缩率小，厚壁件收缩率大 |
| 脱模温度 | 脱模温度越高 → 与室温温差越大 → 收缩率越大 |
| 模具型腔温度分布 | 靠近浇口处型腔温度高 → 收缩率较大；远离浇口处型腔温度低 → 收缩率较小 |

### 模具成型尺寸的综合计算

压铸模具成型尺寸的综合计算公式为[[S25,S19]]：

```
A' = (A + A·φ + n·Δ) ± Δ'
```

式中：
- A' —— 计算后的模具成型尺寸
- A —— 压铸件基本尺寸
- φ —— 压铸件综合计算收缩率（%）
- n —— 补偿和磨损系数（铸件精度6～7级时n=0.5，8～10级时n=0.45）
- Δ —— 铸件公称尺寸偏差
- Δ' —— 压铸模成型部分的制造偏差

该公式将收缩余量、加工余量和制造偏差同时纳入计算，实现成型尺寸的精确设计。

## 模具设计中的余量协同

在压铸模具3D设计流程中，余量设置需按特定顺序进行协同操作：调入制件3D图后，应优先对所有再加工表面加放对应等级的加工余量，之后再设置脱模斜度、加放合金收缩量[[S23]]。这一顺序保证了余量分配与脱模斜度、收缩补偿的逻辑一致，避免成型尺寸设计冲突。

模芯周边加工余量放量存在经验调整公式，可适配不同企业的设备工艺能力[[S1]]：

```
a_j = a₁ + 0.75(a₂ - a₁)·p_max / p₀
```

式中：a₁为不可随意修改的最小尺寸下限极限值，a₂为可调整的上限基数，p_max为最大比压，p₀为基准比压。

当压铸件的尺寸精度、形位精度达不到设计要求而必须进行机械加工时，应优先考虑采用精整加工方法（校正、拉光、压光、整形等），尽量以不受分型面及活动成型影响的表面为毛坯基准面[[S14,S20]]。

## 余量设置不当的缺陷

余量取值过大或过小均会引发压铸件缺陷：

**1. 余量过小——加工黑皮与尺寸超差**

造型余量取值过小，叠加铸件铸造公差后部分区域机加工余量为负值，导致机加工完成后仍残留未去除的铸态黑皮、氧化层，无法满足表面质量和尺寸精度要求，最终造成成品尺寸超差[[S13]]。

**2. 余量过大——暴露内部缺陷**

压铸件表层为快速凝固形成的细晶粒致密层，具有良好的力学性能和物理性能。余量取值过大，机加工时会直接切穿此致密表层，暴露原本位于铸件内部的缩松、孔隙、夹杂类缺陷，直接降低成品的力学性能和气密性[[S14,S39]]。

**3. 余量与脱模斜度匹配不当——脱模卡滞与拉伤**

模具设计阶段未合理预留机加工余量、对应成型面脱模斜度匹配错误，会导致铸件成型过程中无法顺利从型腔脱出，出现脱模卡滞和铸件表面拉伤的典型缺陷[[S16]]。

行业实践经验表明，合理的小余量设计完全可以覆盖并去除内部缺陷。以铝合金缸盖铸件为例，实测缩孔底部距铸件表面仅1.742 mm，设置2.5 mm的加工余量即可通过机械加工完全去除缩松缺陷，无需大幅增大切削余量破坏表层致密组织[[S31]]。

## 与相关标准参数的关系

“造型余量”在压铸模具设计中不是独立参数，而是与以下标准体系中的参数协同工作：

| 关联参数/标准 | 关系说明 |
|:---|:---|
| GB/T 6414-1999 铸件尺寸公差（CT） | 公差等级决定余量设计裕度；CT等级与RMA等级需协同选取 |
| GB/T 6414-1999 要求机械加工余量（RMA） | 造型余量中加工余量部分的标准体系体现 |
| ISO 8062:1994 | 国标等效采标对象，国际通用参考[[S37]] |
| 脱模斜度 | 影响余量大小端分配，需按余量配置确定基准[[S45]] |
| 计算收缩率 φ | 模具成型尺寸的基础参数，将收缩余量量化纳入计算公式[[S33]] |

压铸模具参数分析设计流程强调在设计过程中反复核对、调整、修改与余量相关的各项参数，当计算参数涉及客户协商要求时，需经反复调整和校对才能得到合理且满足质量要求的模具设计方案[[S23]]。
