## 概述

“提钩”在金属成形制造领域并非单一语义术语，其在不同工艺语境下指代两类完全不同的对象：在砂型铸造中，提钩是一种手工修型工具，有明确的国家标准定义和英文规范名称；在压铸模具与注塑模具设计中，提钩则是顺序定距分型机构的核心联动构件，行业内亦称摆钩、压钩或拉钩，属于分型控制辅助功能模块，尚无完全统一的全球通用英文对应术语[[S21,S23,S19,S24,S26,S1]]。

本词条重点围绕压铸模具领域的提钩展开，同时对砂型铸造提钩予以区分说明。

## 定义与术语辨析

### 压铸模具领域的提钩

压铸提钩是顺序定距分型机构（亦称多次分型机构）的核心功能部件，通过铰接式摆钩结构配合弹簧实现自动复位，用于控制模具多个分型面按预设顺序、预设距离依次打开[[S3,S19,S2]]。其典型应用场景包括定模侧抽芯压铸模、点浇口压铸模、卧式压铸机中心浇口压铸模以及二次推出机构[[S3,S5,S8]]。在压铸循环中，提钩的初始功能是钩住对应模板以限制开模顺序，完成预设分型行程后由限位滚轮或拨块触发脱钩，从而解锁后续分型面[[S10,S19,S2]]。

### 砂型铸造领域的提钩

在砂型铸造手工造型修型场景下，提钩（亦称砂钩）是一种修型工具，国家标准GB/T 5611—1998《铸造术语》明确将其定义为“用以提出砂型中的散砂和修光砂型的侧壁及底角的工具”，标准英文术语为 **cleaner**[[S22,S23]]。其材料多为工具钢或青铜，形态为带弯钩端的长杆结构[[S12,S21]]。

### 术语辨析：提钩、拉钩、钩块、钩销

**提钩与拉钩**：压铸模具领域，“提钩”与“拉钩”属于工业日常语境中的近义表述。提钩更侧重初始啮合后提拉锁止、控制顺序分型动作，拉钩则是对所有钩形锁紧/牵引构件的泛称，二者并非严格的同物异名关系[[S24]]。

**提钩与钩块**：钩块指“钩块齿扇斜抽芯结构”中的块状带钩抽芯零件，结构形态为块状而非长条钩杆，功能属于侧抽芯驱动部件，与提钩在功能和形态上均不同[[S33,S15]]。

**提钩与钩销**：钩销是带钩头的圆柱销类定位/牵引零件，属于小型销类零件，与提钩是完全不同的零件类别[[S15]]。

### 英文术语说明

压铸模具提钩尚无完全标准化的全球通用英文对应术语。国内压铸设计手册及相关文献中对应“提钩/拉钩”的通用英文表述通常为 **hook latch** 或 **pull hook**；注塑模具同类构件在英文文献中多称为 **side hook**[[S1,S28]]。"lifting hook"在压铸领域通常指压铸模整体吊装用的吊钩，并非分型机构零件[[S13]]。

## 类别归属

提钩属于压铸模具 **顺序定距分型机构（多次分型机构）**的核心构件，部分场景下也应用于 **二次推出机构** 中。在模具功能分类中，提钩不属于标准抽芯机构或常规顶出机构的大类，而属于辅助功能模块中的分型控制联动部件[[S19,S24,S26,S17]]。

## 核心功能与工作过程

### 工作时序

提钩在压铸循环中的工作时序分为四个阶段[[S5,S19,S2,S26,S25]]：

1.  **合模锁止**：合模状态下，配套压缩弹簧预紧使提钩钩头锁紧对应动/定模板，将待锁模板锁为一体。
2.  **第一级分型**：开模初始阶段保持钩合状态，强制第一级分型面按预设距离优先打开，此时可完成定模侧抽芯动作或将浇注系统余料从浇口套中拉出。
3.  **脱钩解锁**：到达设定分型行程后，安装在模具上的压钩、滚轮或压块接触提钩尾部斜面，迫使其绕铰接轴摆动脱钩，解锁后续分型面。
4.  **顺序分型与复位**：后续分型面依次打开完成压铸件脱模；合模时，提钩在压钩作用下保持脱钩位置，待模板移动至合模位置后脱离压钩束缚，在弹簧作用下重新钩紧对应模板，完成复位。

### 驱动方式

主驱动力来自压铸机开模动力，配套压缩弹簧实现合模后自动复位锁合。特殊结构可采用液压辅助驱动[[S2,S19]]。

### 典型应用功能

- 保证多分型面按预设顺序、预设距离依次打开，避免压铸件包紧力异常导致留模错位[[S3,S19]]。
- 实现自动拉断浇注系统余料（可拉断直径小于10mm的直浇道余料）[[S29,S2]]。
- 适配定模侧复杂侧抽芯结构的动作时序要求[[S5,S4]]。
- 用于二次推出机构中控制两级推板的先后动作[[S8]]。

## 结构特征与组成

### 典型结构

压铸提钩的典型结构为铰接式摆钩：通过销轴/转轴铰接在对应模板上形成旋转运动副，钩头部分与配套啮合挡块或定位销形成卡扣配合，尾部设置斜面与滚轮或压块配合实现脱钩动作[[S11,S18,S19,S32]]。

### 关联部件

提钩协同工作的关联部件包括[[S5,S11,S18]]：

| 部件类别 | 关联部件举例 |
|---------|------------|
| 驱动/复位件 | 压簧（压缩弹簧）、压钩、滚轮、拨块 |
| 限位件 | 限位杆、限位螺钉、定距螺钉 |
| 模板类 | 定模座板、定模套板、动模套板、卸料板 |
| 侧抽芯部件 | 斜销、侧滑块、弯销 |
| 推出部件 | 推杆、推板导柱、复位杆 |
| 浇注系统件 | 浇口套、压射冲头 |

> **说明**：目前尚无针对压铸提钩的专项独立材料标准，上述数据综合自相关模具零件常用材料规范。起重类吊钩可参考GB/T 4678.20中45钢调质处理硬度28～32 HRC的规定[[S34]]。

## 设计参数与操作条件

### 关键设计参数

| 设计项 | 推荐值/要求 | 来源与说明 |
|-------|------------|-----------|
| 钩头啮合角 | 比配套挡块斜面角度大1°～3° | 来自注塑模具设计经验，便于沿啮合面转动脱钩[[S28]]，可作为压铸提钩设计参考 |
| 钩头配合面斜度 | 5° | 摆钩头部与动模套板接触面应设5°斜度，有利于复位[[S19]] |
| 弹簧弹力 | 需大于摆钩受力点的力矩 | 避免非正常脱钩[[S4]] |
| 限位杆长度 | 需同时满足侧抽芯抽芯距要求，且保证斜销不脱离侧滑块斜孔 | 防止动作干涉[[S4]] |

### 设计约束

1.  摆钩支点与力点之间应避免产生可能脱钩的反向力矩，确保合模状态下锁紧可靠[[S3,S4]]。
2.  提钩的设置应为成双对称布置，以保证分型时受力平衡[[S17]]。
3.  不适用于抽芯力过大的重型侧抽芯结构[[S3]]。
4.  不适用于无顺序分型需求的普通单分型面压铸模[[S3]]。

### 标准与规范

目前尚无GB、ISO、NADCA、JIS发布针对压铸提钩的独立专项尺寸、材料或性能标准，其设计制造可参考通用压铸模零件标准GB/T 4679《压铸模零件 技术条件》[[S34]]。

## 常见分类

提钩可按以下三个维度进行分类[[S3,S5]]：

| 分类维度 | 类别 | 说明 |
|---------|------|------|
| 按驱动方式 | 机动提钩 | 依靠开模力直接驱动脱钩 |
|  | 弹簧复位提钩 | 依靠配套压缩弹簧实现锁紧与复位 |
| 按结构形状 | 摆钩 | 铰接式摆动钩体，最常见形式 |
|  | 压钩 | 配合摆钩尾部的固定斜面构件 |
|  | 槽钩 | 滑槽式钩合结构 |
|  | 斜钩 | 斜面驱动钩合结构 |
| 按安装位置 | 动模侧安装提钩 | 铰接于型芯固定板或动模套板 |
|  | 定模侧安装提钩 | 铰接于定模板或定模套板 |

## 失效模式与应对措施

压铸提钩在长期循环载荷下常见以下失效模式[[S31]]：

| 失效模式 | 主要原因 | 应对方案 |
|---------|---------|---------|
| 断裂 | 过载冲击、材料强度不足 | 选用调质淬火处理的高强度热作模具钢 |
| 塑性变形 | 长期反复冲击载荷 | 优化接触斜面角度，降低接触应力 |
| 配合面过度磨损 | 长期摩擦接触 | 配合面做渗氮等耐磨处理 |
| 动作卡滞 | 弹簧失效、配合间隙不当 | 定期检查弹簧预紧力，合理设置运动配合间隙 |

## 与其他模具部件的关联

提钩与典型周边部件的装配配合关系如下[[S11,S18,S5]]：

-   **铰接关系**：通过销轴/转轴与对应模板铰接，形成旋转运动副。
-   **卡扣配合**：钩头与配套啮合挡块/定位销形成钩锁卡扣。
-   **脱钩触发**：尾部斜面与滚轮、压钩或压块接触，实现脱钩动作。
-   **弹簧复位**：压缩弹簧或拉簧提供钩紧复位力。
-   **限位约束**：限位杆/定距螺钉限定各分型面的最大分型距离。

## 应用场景与局限

### 典型应用场景

1.  **定模侧抽芯压铸模**：必须设置顺序分型机构锁住主分型面，提钩是完成此功能的关键构件[[S4,S5]]。
2.  **点浇口模具与卧式压铸机中心浇口模具**：需要顺序分型以取出浇注系统凝料[[S3]]。
3.  **二次推出机构**：通过提钩控制一次推板与二次推板的先后动作时序[[S8]]。
4.  **多次分型复杂结构压铸模**：如三次分型薄壁深筒压铸模[[S25]]。

### 不适用场景

-   抽芯力过大的重型侧抽芯结构[[S3]]。
-   无顺序分型需求的普通单分型面压铸模[[S3]]。

### 跨领域参考

摆钩式顺序分型机构在注塑模具中亦有广泛应用，结构动作逻辑与压铸模具领域的提钩基本一致，设计参数可相互参考[[S27,S28]]。
