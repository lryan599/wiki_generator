## 概述

流动性（砂）（flowability of sand）是型（芯）砂在外力或自重作用下，沿模样表面和砂粒间相对移动的能力[[S21,S42]]，属于型砂/芯砂制备的关键工艺性质之一。该指标直接影响砂芯吹制、射砂紧实过程中砂粒对芯盒复杂型腔的填充效果，对于高压造型、射压造型以及压铸生产中砂芯的制备尤为关键[[S20,S37]]。

流动性（砂）与金属液流动性（fluidity of metal）是完全独立的两类铸造性能指标，标准术语体系中二者分别设有独立条目：金属液流动性表征熔融金属本身的充型流动能力，而砂流动性描述的是砂粒群在外力或自重下的相对移动行为，二者在适用场景、测试方法和调控逻辑上截然不同[[S21,S39,S4]]。

## 分类

根据砂粒表面是否存在湿粘结膜，砂流动性可分为以下两类：

- **干态流动性**：以覆膜砂为典型代表，属于松散无明显湿粘结膜的干砂体系，流动性好。壳芯机用覆膜砂需配套特殊防漏射嘴结构设计，以避免射砂结束后砂粒在自重下从射嘴持续漏出[[S26,S5]]。
- **湿态流动性**：以粘土湿型砂为典型代表，砂粒表面覆盖有水合粘结剂膜，通常采用紧实率、舂击后试样高度差、侧孔挤出砂质量等指标表征[[S20,S9]]。

## 不同粘结剂体系的流动性差异

| 粘结剂体系 | 流动性特点 | 参考来源 |
|---|---|---|
| 粘土砂（钙基膨润土） | 流动性最优 | [[S20]] |
| 粘土砂（普通粘土） | 流动性次之 | [[S20]] |
| 粘土砂（钠基膨润土） | 流动性最小 | [[S20]] |
| 树脂砂 | 整体流动阻力较低，流动性较好，可轻松完成复杂结构砂芯的射砂充型，无需额外增大射砂口尺寸 | [[S9,S35]] |
| 水玻璃砂 | 流动性相对较差，冷芯盒射砂口设计面积需大于常规树脂砂芯盒 | [[S30]] |

在水玻璃砂体系中，硅灰掺量对流动性有显著不利影响：随硅灰加入量从1g增至5g，改性水玻璃砂型的流动性从2.57g逐步降至2.15g[[S34]]。微硅粉加入量超过4g后，流动性下降速率加快[[S27]]。

## 影响因素

砂流动性主要受以下因素影响：

| 影响因素 | 作用规律 | 参考来源 |
|---|---|---|
| 原砂粒形 | 圆形砂（角形系数约1.1）流动性远优于多角形砂（角形系数约1.3）；颗粒间摩擦力更小，流动阻力更低 | [[S19,S20]] |
| 原砂粒度与级配 | 选用粒度大且均匀的原砂流动性好；通过粗砂中混入适量细砂（如50/100目、70/140目级配），提升铺粉密度至1.68g/cm³，可显著提升流动充型能力 | [[S20,S15]] |
| 粘结剂种类 | 钙基膨润土砂流动性最好，普通粘土次之，钠基膨润土砂流动性最小 | [[S20]] |
| 粘结剂加入量 | 粘结剂加入量增加会提升砂芯强度，但流动性同步下降；碱性酚醛树脂添加量从1.2%降至1.0%时，流动性可明显提升 | [[S7,S43]] |
| 含水量 | 水分过高会导致砂粒黏附团聚，流动性大幅下降 | [[S19]] |
| 砂温 | 砂温过高会引发粘结剂提前预固化，流动性大幅衰减；旧砂温度应≤35℃；合理控制砂温在25~35℃区间 | [[S19,S24]] |
| 附加物（润滑剂） | 硬脂酸钙可有效防止覆膜砂结块，增加流动性，改善脱模性 | [[S44]] |
| 附加物（油类） | 在粘土砂中掺入约0.5%重油或煤油可显著提升流动性 | [[S20]] |

正交试验结果表明，氢氧化钾、六偏磷酸钠、葡萄糖、2%十二烷基磺酸钠水溶液四个变量对型砂流动性的影响极差R分别为0.056、0.020、0.037、0.053，各因素影响程度存在差异[[S40]]。

## 测试方法

砂流动性的试验方法目前有多种，尚未完全统一，以下为常用方法：

### 阶梯硬度差法

在圆柱形湿压强度标准试样筒中放置半圆形金属凸台，加入110~120g型砂，用锤击式制样机舂打压实后，翻转试样筒测得A处硬度H<sub>A</sub>，顶出试样取下凸台后测得B处硬度H<sub>B</sub>，流动性按 η = (H<sub>A</sub> / H<sub>B</sub>) × 100% 计算。两处硬度差值越小，型砂流动性越好[[S17]]。该方法适用于高压造型工艺场景[[S17]]。

![阶梯硬度法型砂流动性试验装置示意图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d4a50d2c-f357-4b57-9a4d-4e18cc87bce9/resource)

阶梯硬度法测定型（芯）砂流动性装置，标注有凸台、压头、冲头，通过试样A、B两处硬度差评定流动性[[S36,S41]]。

### 漏孔质量法

称取170g型砂放入标准试样筒，在锤击式制样机上舂打一次后，放置于带φ25mm漏孔的漏孔模上，再冲击两次，通过漏孔的砂质量占原试样总质量的百分比即为流动性 η = (m<sub>A</sub> / m<sub>B</sub>) × 100%[[S17]]。

### 侧孔质量法

在圆柱形标准试样筒侧面开设φ12mm小孔，预先用塞柱塞紧，称取185g试样倒入试样筒，拔出塞柱后在锤击式制样机上舂击10次，收集从小孔中被挤出的砂子并称量质量，以挤出砂质量占试样总质量的百分比作为流动性指标。该方法可有效反映型砂充填铸型轮廓、凹槽的能力及吹射性，适用于湿态强度较低的树脂砂[[S2,S9]]。

### 环形空腔法

先舂制一个和待测砂样同质量的圆柱形标准试样，将同质量型砂填入下端带环形空腔的特制试样筒中，舂打3次后测量试样高度h，流动性按 η = ((h<sub>0</sub> − h) / (h<sub>0</sub> − h<sub>1</sub>)) × 100% 计算（其中h<sub>0</sub> = 50mm，h<sub>1</sub> = 35mm）。该方法可反映型砂受力后向各个方向移动的能力，适用于湿态强度较低的型砂如树脂砂[[S2,S16]]。

![环形空腔法型砂流动性测定装置简图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/d8ebba10-1d28-49f5-8d69-dcf37c0fb368/resource)

环形空腔法测定型（芯）砂流动性装置，标注有φ50、φ78等关键尺寸及试样高度h[[S38]]。

### 覆膜砂计时法

使用底孔直径为φ6mm的黏度杯，用手堵住底孔后将覆膜砂添满刮平，移开手指同时用秒表计时，全部砂粒从黏度杯流出的总时长即为该覆膜砂的流动性指标，单位为秒（s）[[S26]]。

### AFS标准测试方法

- **AFS千分表舂击法**：将千分表安装在AFS标准砂器上，调整初始读数为100，对砂样先后舂击5次后，记录最终读数即为以百分数表示的AFS流动性指数[[S10]]。
- **AFS罗威尔（Rowell）法**：在AFS标准试样筒中放置60度角缩径模块，砂样过4号筛后倒入试样筒并舂击3次，得到楔形试样后在楔形面两端测量紧实硬度，两端硬度比值乘以100得到罗威尔流动性指数，可表征型砂在异形模表面均匀舂实及流入受限模型部位的能力[[S31]]。
- **Hall标准漏斗法**（对应ISO 4490标准）：测量50g测试砂样全部通过标准漏斗流出的总时间即流动时间，仅适用于非粘结性的自由流动型砂对比测试[[S25]]。

## 工艺参数

砂流动性与射砂工艺参数密切相关。流动性较好的砂可选用更低的射砂压力，流动性差的砂需选用更高的射砂压力以保证砂芯紧实度[[S35]]。

### 射砂压力

| 工艺类型 | 射砂压力范围 | 参考来源 |
|---|---|---|
| 热芯盒制芯（大型复杂砂芯） | 0.5~0.7 MPa | [[S7]] |
| 热芯盒制芯（简单小砂芯） | 0.4~0.5 MPa | [[S7]] |
| 覆膜砂壳芯制芯 | 0.15~0.4 MPa | [[S33]] |
| 工业热芯盒制芯（通用） | 0.3~0.6 MPa | [[S35]] |

射砂压力过高会导致卷气、砂粒充型不稳，压力过低则导致砂芯局部疏松[[S12]]。

### 射砂时间

- 热芯盒制芯通用射砂时间：0.5~1s[[S7]]
- 覆膜砂壳芯制芯射砂时间：3~10s[[S33]]

射砂时间过长会导致已结壳的型壳滑移，造成结壳厚度不足；射砂时间过短则砂无法完全充型[[S33]]。

### 芯盒排气设计

芯盒排气结构可通过形成压力差大幅提升砂的流动填充效果。冷芯盒工艺中排气面积取进气面积的50%~70%为最优；排气塞/排气槽需设置在砂芯深凹、厚实的区域，垂直高深芯盒下部多设置排气结构，避免砂流动填充出现死角[[S30,S14]]。

## 对质量的影响

### 流动性不足

砂流动性不足会导致射砂填充过程中砂芯不同部位密实度不均，局部疏松区域密度偏低，砂芯整体强度下降。金属液充型过程中易发生冲砂，最终在铸件内部形成砂眼、夹砂缺陷，同时砂芯裹气量增加，铸件气孔缺陷概率上升[[S12,S23]]。流动性过差也是导致射不满、射不紧等工艺废品的重要原因[[S37]]。

### 流动性过高

砂流动性过高时，砂粒在射砂压力作用下易从芯盒配合间隙溢出，导致砂芯边缘出现多余飞边毛刺，砂芯整体孔隙率上升；在同等粘结剂添加量下砂芯表面强度下降，浇注过程中易出现粘模、砂芯局部表面脱落的问题[[S6]]。

## 改进措施

行业通用砂流动性优化措施包括：

1. **选用优质原砂**：选用低角形系数的球形原砂（如宝珠砂，角形系数≤1.1）替代普通多角硅砂，减少砂粒间摩擦阻力；宝珠砂的铺粉密度比不规则硅砂提高21.3%，流动充型能力卓越[[S15,S19]]。
2. **降低粘结剂添加量**：在保证砂芯强度的前提下尽可能降低粘结剂添加量，避免粘结剂过量导致砂粒粘连团聚；降低树脂添加量15%~20%的同时可维持砂芯性能[[S7,S15]]。
3. **控制砂温**：合理控制砂温在25~35℃区间，避免砂温过高引发粘结剂提前预固化，导致流动性大幅衰减[[S24]]。
4. **添加润滑剂**：添加适量硬脂酸钙等流动性改善润滑剂，降低砂粒之间的内摩擦力，同时改善砂芯脱模性[[S44]]。
5. **优化原砂级配**：通过粗砂中混入适量细砂提升铺粉密度，降低孔隙率，改善流动充型能力[[S15]]。
