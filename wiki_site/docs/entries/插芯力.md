## 术语边界与概览

在压铸领域正式设计资料中，「抽芯力」与「插芯力」指代两种方向相反、工况不同的独立参数，边界清晰，不存在同一术语混用两种含义的规范写法[[S11,S6]]。

- **抽芯力（core pulling force / core extraction force）**：开模后，将已被铸件包紧的活动型芯从铸件中脱出所需施加的驱动力[[S17,S13,S36]]。
- **插芯力（core insertion force）**：合模前或合模过程中，液压抽芯器驱动活动型芯向模具型腔内部移动并抵达成型工位的输出驱动力；其核心作用是抵消金属液充型阶段作用在侧向型芯成型面上的分胀型力，防止型芯在高压充型中后退[[S34,S6,S27]]。

仅部分非正式文档将“插芯动作的驱动力”泛化为“抽芯力”的别名，正文设计计算中两者不应混用[[S11]]。本文以侧抽芯机构的作用力为核心，同时涵盖抽芯力与插芯力两个方向。

## 抽芯力

### 定义与物理含义

压铸时，金属液充填型腔并冷却凝固收缩，对包围的活动型芯产生包紧力。抽芯机构启动瞬间需要克服的合力即为**起始抽芯力**，通常所指的抽芯力均指起始抽芯力；后续继续抽拔仅需克服小得多的机构运动阻力，称为**相继抽芯力**[[S17,S13,S36,S39]]。

抽芯力的组成包括[[S13,S36,S25]]：
1. 铸件凝固收缩对型芯产生的**包紧力**（核心组分）
2. 型芯与铸件接触面之间的**摩擦阻力**
3. 抽芯机构运动副的**滑动摩擦阻力**
4. 型芯运动过程的**惯性力**

### 计算公式

压铸领域通用标准抽芯力理论公式为[[S39,S23,S14,S31,S2,S1,S36]]：

$$F_{\text{抽}} = A l p (\mu \cos\alpha - \sin\alpha)$$

式中：
- \(F_{\text{抽}}\) — 起始抽芯力（N）
- \(A\) — 被铸件包紧的型芯成型部分断面周长（cm）
- \(l\) — 被铸件包紧的型芯成型部分长度（cm）
- \(p\) — 挤压应力（单位面积包紧力）
- \(\mu\) — 压铸合金对型芯的摩擦系数
- \(\alpha\) — 型芯成型部分的出模斜度（°）

### 各合金参数取值范围

不同压铸合金的单位面积包紧力 \(p\) 及摩擦系数 \(\mu\) 取值[[S39,S14,S29,S2,S26,S17,S22]]：

| 合金种类 | 单位面积包紧力 \(p\)（MPa） | 摩擦系数 \(\mu\) |
|---|---|---|
| 锌合金 | 6–8 | 0.2–0.25 |
| 铝合金 | 10–12 | 0.2–0.25 |
| 镁合金 | 约7–8.4（约为铝合金的0.7倍） | 0.2–0.25 |
| 铜合金 | 12–16 | 0.2–0.25 |

> 早期文献中亦有合金类别摩擦系数取值的细分：锌/铝/镁合金 \(\mu_m = 0.25\)，铜合金 \(\mu_m = 0.35\)[[S16,S17,S22]]。此数据在部分手册中以包紧力经验公式形式出现。

### 经验简化公式

圆形型芯抽芯力简化估算式[[S29]]：

$$P = 16 d L \quad \text{(N)}$$

式中 \(d\) 为型芯直径（mm），\(L\) 为被包紧长度（mm）。

包紧力经验公式[[S16]]：

$$P_b = \mu_m F_b p + L k$$

式中 \(\mu_m\) 为摩擦阻力无因次系数（锌/镁合金取0.25，铜合金取0.35），\(F_b\) 为被包紧表面积，\(k\) 为与型芯结构相关的常数。

### 影响抽芯力的主要因素

抽芯力的核心影响因素及其作用规律[[S16,S43,S42]]：

| 因素 | 影响方向 |
|---|---|
| 型芯被包紧表面积增大 | 抽芯力 ↑ |
| 型芯断面非圆形/结构复杂 | 抽芯力高于同周长圆形 |
| 铸件平均壁厚增大 | 抽芯力 ↑ |
| 同一机构抽拔多型芯 | 收缩叠加效应使抽芯力 ↑ |
| 合金凝固收缩率/弹性模量大 | 抽芯力 ↑ |
| 浇注温度高、铸模温差大 | 抽芯力 ↑ |
| 压射比压高 | 抽芯力 ↑ |
| 留模时间长 | 抽芯力 ↑ |
| 出模斜度小 | 抽芯力 ↑ |
| 型芯表面粗糙度高 | 抽芯力 ↑ |
| 润滑条件差 | 抽芯力 ↑ |

## 插芯力

### 定义与物理含义

插芯力是液压抽芯器特有的工况参数，指向型芯插入型腔方向。其作用是提供足够推力将侧向活动型芯送入工作位置，并在金属液高压充型阶段抵消侧向型芯成型面上所受的金属液反作用力（分胀型力），防止型芯后退窜料[[S34,S6,S27]]。

### 计算公式

当液压抽芯器未明确标注插芯力参数时，可采用经验公式计算[[S6,S27]]：

$$F_{\text{插}} = 0.785 D_{\text{插}}^2 p_{\text{管}}$$

式中：
- \(F_{\text{插}}\) — 液压抽芯器的插芯力（N）
- \(D_{\text{插}}\) — 液压抽芯器的液压缸直径（m）
- \(p_{\text{管}}\) — 压铸机液压系统管道压力（Pa）

### 插芯力在分胀型力计算中的角色

液压抽芯机构分胀型力计算中，插芯力可抵消部分侧向型芯所受的金属液推力[[S6,S27,S9,S15]]：

$$F_{\text{分}} = \sum (A_{\text{芯}} p \tan\alpha - F_{\text{插}})$$

式中 \(A_{\text{芯}}\) 为侧向活动型芯成型端面投影面积，\(p\) 为压射压力，\(\alpha\) 为楔紧块楔紧角。

插芯力越大，对楔紧块的受力载荷降低越明显，特殊情况下（如插芯力足够大时）可不设楔紧装置[[S11]]。

## 与抽芯机构设计的关系

### 常用抽芯机构分类

按动力源分为三大类[[S43,S32,S33]]：

| 类别 | 动力来源 | 特点 | 适用场景 |
|---|---|---|---|
| **机动抽芯**（斜销、弯销、斜滑块、齿轮齿条） | 压铸机开模力 | 结构紧凑，效率高，应用最广 | 中小型抽芯力/抽芯距 |
| **液压抽芯** | 液压缸压力液 | 抽芯力大，抽芯距长，时序独立可控 | 大中型模具/特殊角度 |
| **手动抽芯** | 人工操作 | 结构简单，成本低，效率低 | 小批量/试制 |

### 斜销抽芯机构选型规则

适合抽芯距短、抽芯力中等、靠近分型面的场合[[S10,S19,S24]]：
- 斜销推荐工作角度：**15°–22.5°**，最大不得超过 **25°**
- 依托压铸机开模动力完成抽芯，结构简单成本低
- 不适合大抽芯力、长距离抽芯

### 液压抽芯机构选型规则

适合抽芯力大、抽芯距长、抽芯时序需独立控制的中大型模具[[S42,S9,S20,S15]]：
- 计算抽芯力需乘以 **1.2–1.3** 的安全系数后选型额定推力
- 抽芯器缸径选型公式：

$$D = \sqrt{\frac{4F}{\pi p_{\text{油}}}}$$

\(p_{\text{油}}\) 为压铸机液压系统供油压力，常规取值 **10–16 MPa**[[S20,S42]]
- 额定插芯力可降低楔紧块载荷，特定条件下可不设楔紧装置[[S11]]

### 抽芯力对机构刚度的约束

抽芯力计算结果对机构设计的约束规则[[S44,S9,S30]]：

| 约束项 | 设计要求 |
|---|---|
| 导滑长度 | 合模后滑块留在导滑槽内长度 ≥ 滑块总长的 **2/3**，防止偏载卡滞 |
| 斜销/连接销轴直径 | 随抽芯力增大同步加大，保证抗剪切刚度，防止变形断裂 |
| 楔紧块刚性 | 随抽芯力增大增大设计余量，防止金属液反力推动滑块后退窜料 |
| 锁模力校核 | 抽芯力计算完成后需同步校核分胀型力，匹配压铸机锁模力参数 |

## 缺陷关联

### 抽芯力不足引发的缺陷

抽芯力不足导致型芯无法顺利脱出铸件，引发铸件表面沿抽芯方向的机械拉伤，严重时出现局部撕破缺肉[[S5,S12,S45]]。典型案例：摩托车右大盖斜抽芯位置因抽芯力不足引发型芯弯曲变形，最终导致铸件表面出现明显拉伤掉肉现象[[S5,S45]]。

### 抽芯力过大引发的缺陷

抽芯力超出型芯结构强度承受范围时，直接引发**断芯**缺陷，型芯断裂残留在铸件内部，严重影响生产连续性[[S40]]。

### 抽芯动作卡滞与粘模拉伤

抽芯力不稳定或抽芯动作卡滞，导致铸件侧抽芯位置出现**粘模拉伤**缺陷。典型案例：CG125铃木刀左后盖压铸件侧抽芯位置出现表面粗糙的粘模拉伤痕迹，原因为模具长时间生产导致表面温度升高、涂料堆积，抽芯过程中铸件与型芯发生粘连[[S7,S41,S38]]。

### 抽芯力不合理引发的变形

抽芯时受力不均或抽芯动作速度过快，导致铸件在未完全冷却硬化的状态下被强行拉动，出现局部或整体尺寸超差的变形问题[[S28,S3]]。

### 插芯力过小引发的缺陷

插芯力过小导致侧向型芯在高压充型过程中被金属液推动发生后退移位，最终形成铸件侧孔尺寸超差、孔位偏移（**孔偏**）缺陷，严重时飞边夹在型芯与滑块配合间隙中，影响后续循环生产[[S40,S3]]。

## 测量与验证方法

工程化抽芯力/插芯力测量验证的三种主要方法[[S18,S35]]：

| 方法 | 操作方式 | 特点 |
|---|---|---|
| **液压系统压力传感器法** | 通过压铸机液压系统自带压力传感器实时采集抽芯器工作油压，结合液压缸活塞面积换算得到实际输出力 | 可实现在线监测 |
| **动态测力传感器法** | 在抽芯器与活动型芯的连接环节加装动态测力传感器，直接采集抽芯全过程力变化曲线 | 可获得起始抽芯力、运动阶段抽芯力实测值 |
| **标准测力计静态标定法** | 在抽芯机构的模板之间安装标准测力计，对核心力值进行静态校准标定 | 用于定量校准和验证[[S35]] |
