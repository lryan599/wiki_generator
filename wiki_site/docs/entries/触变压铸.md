## 工艺作用

半固态金属触变压铸成形工艺主要包含三个工艺流程：半固态合金原始坯料的制备、原始合金坯料的半固态重熔加热和半固态坯料的触变压铸成形[[S55]]。

- **非枝晶坯料制备**：首先通过电磁搅拌、机械搅拌等方法获得具有细小球状初生固相的半固态金属浆料，并将该浆料完全凝固成坯料。该流程通常独立于成形车间进行，以便于控制工艺和降低成本[[S10]]。
- **二次重熔加热**：根据铸件重量，将凝固后的坯料切分成一定尺寸，放入加热装置进行快速半固态重熔加热，精确控制其固相分数或液相分数[[S10]]。
- **压射成形**：当坯料各处温度达到预定值且基本均匀后，将其送入压铸机的压射室，进行压射成形、保压、开模取件及清理型腔[[S10]]。

该工艺的核心原理是利用半固态金属的触变性（Thixotropy），即浆料在静止时粘度较高，可作为固体被搬运，而在受到剪切力时粘度急剧下降，像液体一样具有良好的流变填充能力[[S10]]。以下示意图展示了包含微观组织演变的完整工艺链路。

## 关键工艺参数

触变压铸的工艺参数对压铸过程的稳定性和铸件质量至关重要。关键参数主要包括坯料的温度（即固相分数）、压射速度、压射比压以及模具预热温度等[[S28,S35]]。

### 1. 二次加热温度与固相分数

二次加热阶段需确保坯料整体温度均匀，使其各处固相分数均匀一致，防止因局部粘度差异导致充型流动不稳定。为防止立式加热时坯料坍塌，液相分数通常控制得较低；卧式加热时则可适当提高液相分数以降低充型阻力[[S28,S35]]。固相分数的增加会显著提升铸件致密性，研究显示当固相分数达到0.5时，铸件内部基本探测不到显微孔洞[[S48,S53]]。常用触变压铸合金的二次加热温度汇总如下：

| 合金牌号 | 加热温度（℃） | 备注 |
| :--- | :--- | :--- |
| A356（ZL101A） | 580–587（≥575） | 对应液相体积分数 35%–65% [[S48,S53,S28]] |
| A357 | 约 580 | [[S48,S53]] |
| A319 | 约 574 | [[S48,S53]] |
| A6082 | 635–640 | [[S48,S53]] |
| A7075（7A04） | 620–625 | [[S48,S53]] |
| AZ91D（镁合金） | 575–580 | [[S48,S53]] |

### 2. 压射比压与压射速度

由于半固态浆料的表观粘度比同种液态金属高约三个数量级，流动阻力大，触变压铸需要匹配较高的压射压力与适当的压射速度[[S56]]。实际生产中，低速段取0.2–0.3 m/s，高速段取3–5 m/s[[S24,S1,S22]]。适当提高动态压射力和增压压力可进一步减少铸件内部显微孔洞，提升力学性能[[S49]]。

| 参数项 | 常规范围 / 商用机型参数 | 备注 |
| :--- | :--- | :--- |
| 压射压力 | 4.1–13.8 MPa | 商用机型常规参数 [[S43]] |
| 最高压射冲头速度 | 可达 7.6 m/s | 商用机型极限参数 [[S43]] |
| 动态压射力（A356） | 8.27–20.67 MPa | 生产研究数据 [[S49]] |
| 增压压力（A356） | 20.67–41.34 MPa | 生产研究数据 [[S49]] |
| 最大增压压力 | 41.3 MPa | 商用机型极限参数 [[S43,S11]] |

从A356铝合金的应用数据来看，当动态压射力由8.27 MPa提升至20.67 MPa，增压压力由20.67 MPa提升至41.34 MPa时，铸态抗拉强度可从115.76 MPa大幅提升至192.93 MPa，伸长率从2.8%提升至9.3%[[S49]]。

### 3. 模具预热温度

模具预热的主要目的是减轻其对半固态坯料的激冷作用，防止坯料提前凝固导致充型不足或冷隔，同时减轻高温浆料对模具的热冲击以延长模具寿命[[S3,S49]]。

- **触变压铸常规模具温度**：250–275°C [[S47]]。
- **传统液态青铜压铸模具温度**：350–500°C [[S47]]。

相比传统压铸，触变压铸的模具工作温度显著降低，有效减小了热负荷，从而显著延长模具使用寿命[[S47]]。

## 微观组织与流变学基础

触变压铸所用坯料，必须预先制备为具有非枝晶初生相的半固态组织。经过电磁搅拌等工艺处理并二次重熔后，其初生相形貌呈球状、近球状或蔷薇状，完全区别于传统铸造的连续树枝晶网络结构[[S58]]。通过高固相制浆方法获得的浆料，其初生α-Al晶粒球形度可达0.89–0.92，平均晶粒直径约58–61 μm，晶粒分布均匀[[S42]]。

**与常规压铸的微观对比：**
常规液态压铸成形的铝合金，其初生α-Al为典型的连续枝晶网络。在高速紊流充型过程中，枝晶骨架容易包裹大量气体，凝固后形成微气孔，导致铸件致密度低且无法进行完整的热处理强化[[S18,S56,S39]]。而触变压铸的非枝晶球状组织在充型时流动平稳，呈层流状态，卷气倾向极低[[S56,S18]]。当固相分数达到0.5时，可完全消除显微孔洞，获得近无气孔的致密铸件，因此能够直接进行T4/T5/T6等固溶时效热处理以获得高强度[[S53,S39]]。

## 适用材料与牌号

触变压铸技术主要适用于铝合金和镁合金，对于高熔点的钢铁类合金目前仍处于实验室探索阶段，受限于模具的耐高温寿命[[S36]]。

- **铝合金**：A356（ZL101A）因具有宽泛的半固态工艺窗口和优良的充型流动性，是应用最广泛的触变压铸铝合金[[S32,S4]]。7075（7A04）作为高强变形铝合金，通过触变压铸可获得接近锻件水平的力学性能，解决了传统液态压铸无法成形该类高熔点高强合金的问题[[S54]]。
- **镁合金**：AZ91D是镁合金触变成形的主流牌号，其触变压铸可大幅降低镁合金氧化燃烧的风险[[S48,S53]]。

## 常见缺陷及对策

### 液析（固液偏析）

表现为铸件厚壁处α-Al初生固相颗粒与共晶液相分布不均，形成局部液相偏聚。

- **形成机理**：当半固态浆料固相分数不均，充型时液相优先于固相高速流动，造成局部液相富集；过低的固相分数也容易引发此缺陷[[S38,S14,S13]]。
- **不利影响**：一方面改变了局部表观黏度，增大卷气倾向；另一方面导致铸件微观组织和力学性能不均匀[[S38]]。
- **控制对策**：设计合理的控流式浇注系统防止液相高速偏流、将固相分数控制在合理区间、均衡模具温度以减小铸件内外的温度梯度、优化充型挤压速度[[S13]]。

### 缩松

在铸件最后凝固的热节位置形成细小且分散的孔洞。

- **形成机理**：铸件厚壁热节部位的补缩通道提前被堵塞，保压压力无法有效传递至该区域，导致凝固收缩得不到金属液补足[[S2]]。
- **控制对策**：优化产品结构减少壁厚差异以消除热节；对厚壁热节部位采用定点喷涂、局部冷却等方式均衡模温；设置合理的补缩冒口系统或局部加压结构；利用数值模拟预判缺陷分布以优化工艺参数[[S2,S30]]。

### 充型不良

表现为金属流对接后未完全熔合形成不规则线状缝隙（冷隔），或铸件薄壁位置未完全充满（欠铸）。

- **形成机理**：半固态浆料充型前沿温度过低，表观黏度急剧升高；一次快压射速度过低导致浆料在压室中热量损失过大而提前凝固[[S41]]；模具温度过低或流道过长。
- **控制对策**：适当提高一次快压射速度以避免浆料热量损失[[S41]]；将模具预热温度控制在250–275°C[[S47]]；优化流道设计以缩短充型流程；合理设置排气系统避免型腔困气[[S17]]。

### 氧化物夹杂

半固态坯料或充型过程中产生的氧化皮被卷入铸件内部，形成高危夹杂缺陷[[S8]]。

- **形成机理**：（1）坯料二次加热时表面生成的氧化皮被挤压入型腔；（2）充型时浆料前沿的氧化膜在两股金属流对接时被裹入铸件内部；（3）压射速度过高引发紊流卷气，产生二次氧化物[[S8]]。
- **控制对策**：确保坯料冶金质量以减少原始氧化来源；优化充型路径并保持层流充填以避免流股碰撞；在金属流对接位置设置大容积溢流槽以承接氧化冷料；压射速度控制在2 m/s以下并优化排气系统，可选用95 kPa级真空排气进一步降低夹杂概率[[S8]]。

## 工业应用与案例

半固态触变压铸技术自诞生以来，在汽车工业等高要求领域取得了显著的商业化应用，主要集中于铝合金结构件和安全件的生产。

- **Alumax公司（美国）**：作为全球首家实现半固态成形技术商业化的公司，其于1994年建成的生产线可年产2400万件铝合金零件。1995年即为美国车企批量交付了25万件汽车空调箱体和200万件火箭支架底座，并持续为刹车系统、燃料输送系统及悬挂系统提供底盘和结构件[[S7,S60]]。
- **AEMP公司（美国）**：采用触变压铸工艺为福特汽车大规模批量生产铝合金压缩机活塞，成品率接近100%。该生产线基于电磁搅拌连续铸造的非枝晶坯料，实现了大批量稳定量产[[S5]]。
- **Magneti Marelli公司（意大利）**：利用触变压铸生产汽车喷油系统中的射油道零件。该零件对质量、耐腐蚀性及复杂形状要求苛刻，采用触变压铸后不仅保证了产品高合格率，且生产费用比传统机加工降低50%，实现了日产数千件的稳定供货[[S60]]。

## 相关标准与规范

涉及半固态触变压铸件及所用合金的相关标准，涵盖了材料、铸件质量与尺寸公差等多个层面。

- **中国国家标准**：GB/T 15115—2024《压铸铝合金》规定了压铸用铝合金的化学成分；GB/T 15114—2023《铝合金压铸件》详细规定了铝合金压铸件的化学成分、力学性能、尺寸精度及内外表面质量等技术要求[[S52]]。此外，还有航空行业标准HB 5012—2011《铝合金压铸件规范》用于高端受力结构件的特殊质量管控要求，以及GB/T 8733—2016《铸造铝合金锭》规范半固态原料坯料的冶金质量基准[[S52]]。
- **国际标准与规范**：北美压铸协会发布的NADCA产品规范标准中包含了压铸件质量保证和缺陷排查指南；ASTM系列标准（如ASTM B86、ASTM B252）覆盖了铝合金、锌合金压铸件的性能及表面处理规范[[S57]]。国际标准化组织（ISO）的铸件公差工作组（如ISO/TC3/WG7）制定的半固态铸件尺寸公差体系也覆盖了触变压铸件的精度管控要求[[S50]]。
