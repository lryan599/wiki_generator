## 概述

清砂是压铸后处理中的关键工序。依据GB/T 25370-2020《铸造机械 术语》，清砂的定义为**清除铸件表面粘砂及内腔残留砂芯的过程** [[S26]] 。在压铸语境下，该工序覆盖可溶砂芯、树脂砂芯、盐芯等多种压铸用芯材的残留清理操作，并非普通铸件表面除砂作业 [[S26]]。

清砂位于压铸生产全流程的前期阶段：开模取出铸件并完成二次冷却后，随即进入清理工部执行清砂作业，常规安排在切边/去除浇冒口工序之前。对清洁度要求极高的场景，可在切边完成后补充二次清砂，整体早于检验、修补、热处理等工序的执行节点 [[S9,S13]]。

## 概念边界

压铸清砂与以下相关概念存在明确范畴边界：

**清砂与落砂**：落砂是砂型铸造特有的工序，指用手工或机械方法使铸件与整铸型砂、砂箱分离的操作，不适用于无砂箱的高压压铸生产场景 [[S17]]。

**清砂与除芯**：除芯仅特指采用锤击或其他方式清除铸件内腔砂芯的单一操作，属于完整清砂工序的一个子环节，无法覆盖清砂同时清理表面粘砂的全流程作业内容 [[S9]]。

**清砂与抛丸清理**：抛丸清理是利用高速旋转叶轮带动磨料冲击铸件表面的表面处理工艺，仅能去除铸件外表面的残砂、氧化皮及飞边毛刺，无法触及压铸件复杂内腔深处的残留砂芯，无法满足清砂全范围清理的要求 [[S9]]。

## 工艺目的与适用场景

压铸清砂的核心工艺目的：彻底去除嵌埋在压铸件复杂内型腔中的残留砂芯与表面附着的残余砂粒，避免后续机加工或服役过程中出现**砂眼**、**内部水道/油路堵塞**等失效缺陷 [[S31]]。

在高压压铸（HPDC）中，以下典型场景必须使用砂芯/盐芯，后续必须配套清砂工序：

- 带复杂内部冷却水道、油路的铝合金压铸件
- 新能源汽车大型中空轻量化结构件
- 内燃机闭式缸体
- 铝合金活塞油冷通道

这类复杂内型腔构件无法通过常规可抽拔金属芯直接成形 [[S31,S20]]。

## 清砂方法分类

依据GB/T 25370-2020，行业内主流清砂技术分为三大类 [[S9,S5,S30]]：

| 大类 | 细分方法 | 工作原理 |
|------|---------|---------|
| **干法清砂** | 机械振动清砂 | 依靠激振结构产生振动实现砂芯与铸件分离 |
| | 喷丸清砂 | 以压缩空气为动力将磨料高速喷射至铸件表面 |
| | 抛丸清砂 | 依靠高速旋转叶轮产生离心力将铁丸/钢丸抛向铸件 |
| **湿法清砂** | 水力清砂 | 高压水流喷射，利用剪切、渗透、冲刷作用清除残砂 |
| | 水爆清砂 | 高温铸件浸水，水渗入砂芯后急剧汽化膨胀产生爆炸力剥离砂芯 |
| | 电液压清砂 | 特制电极产生高压脉冲放电形成水基冲击波击碎砂芯 |
| **化学清砂** | 普通化学清砂 | 在碱性或酸性溶液中清除粘砂层 |
| | 电化学清砂 | 利用化学溶液通电处理清除残砂和氧化层 |

喷丸与抛丸在工作原理上存在差异：喷丸清理以压缩空气为动力，适合处理复杂内腔和细节区域；抛丸清理依靠叶轮离心力，生产效率更高，适合大批次铸件的外表面清砂 [[S9,S5]]。

## 压铸场景清砂工艺选型

针对不同材质与结构的压铸件，清砂工艺选型遵循以下通用原则 [[S30,S12]]：

| 铸件类型 | 推荐清砂方法 | 注意事项 |
|---------|------------|---------|
| 铝合金等低熔点薄壁压铸件 | 优先选择高压水力清砂 | 避免冲击式清砂造成表面损伤 |
| 铸钢/铸铁类结构复杂压铸件 | “先抛丸后局部补喷丸”联合工艺 | 兼顾效率与清理全面性 |
| 高精度薄壁精密压铸件 | 禁止大冲击负荷方案 | 防止工件不可控变形 |

**盐芯的特殊清砂方式**：高压压铸用盐芯（主要成分为氯化物、氟化物）的清砂无需复杂的物理破碎操作。将铸件浸入冷水即可溶解去除内部盐芯，使用热水或加压水射流可进一步提升清砂效率。盐芯与熔融金属反应率极低，不会在铸件内部留下难以清除的杂质残留 [[S20,S8]]。

## 主要工艺参数

### 喷丸清砂参数

喷丸作业时需严格控制以下工艺参数 [[S12]]：

| 参数项 | 推荐范围/条件 |
|-------|------------|
| 普通喷砂清理气压 | 2~5 atm |
| 喷铁丸清理气压 | 5~6 atm |
| 铸件表面温度 | ≤150°C（避免高速气流激冷产生裂纹） |
| 普通砂类磨料粒度 | 40~70号（单颗粒径<1mm） |
| 有色金属铸件磨料 | 粒度需进一步减小，防止表面砸伤变形 |

### 水力清砂参数

水力清砂依靠高压泵输出的高压水，通过喷嘴转化为高速射流，以剪切、渗透、冲刷作用击碎残留砂芯。在高压水射流中添加磨料后形成水砂清砂工艺，可进一步提升高硬度残砂的清理效率 [[S7]]。

不同材质铸件的水力清砂推荐参数 [[S12]]：

| 铸件材质 | 推荐水压 | 喷嘴距离 |
|---------|---------|---------|
| 铝合金压铸件 | 8~10 MPa | 1000~1200 mm |
| 铸钢件 | 20~40 MPa及以上 | 1000~1200 mm |

### 电化学清砂参数

电化学清砂采用熔融电解液，典型工艺参数 [[S12]]：

- 电解液配比：85% 工业纯碱 + 15% 工业纯盐（质量占比）
- 电流密度：5~10 A/cm²（每单位铸件表面积）
- 工作电压：4~10 V
- 铸件接直流电源负极，电解液槽接正极

适合普通清砂工艺无法处理的复杂铸件内腔残留顽固粘砂的清理。

### 水爆清砂注意事项

水爆清砂仅推荐用于**铸钢件**。脆性铝合金、薄壁铸铁件采用该工艺极易产生冷裂缺陷 [[S27]]。

## 清砂效果检测与质量要求

### 常用检测方法

清砂效果常用检测方法 [[S15]]：

1. 视觉目视检查铸件表面状态
2. 压缩空气吹扫探测内部残砂残留
3. 水压试验确认内部通道通畅性
4. 铸件表面粗糙度对比样块判定表面光洁度

### 验收标准

- GB/T铸造标准体系中，清砂验收质量要求为：铸件外观无可见粘砂、内部通道无残留砂粒、表面粗糙度Ra参数符合产品图纸规定 [[S15]]。
- NADCA E系列产品标准明确了不同等级清砂作业对应的尺寸公差控制要求 [[S36]]。
- 美国ASTM标准体系中涉及压铸清砂后铸件质量要求的规范包括：ASTM B94/B94M《镁合金压铸件标准规范》、ASTM B791《锌铝合金铸造及压铸件标准规范》 [[S19,S2]]。

## 缺陷与质量影响

### 清砂不彻底导致的缺陷

清砂不彻底会导致以下质量问题 [[S23,S11]]：

- 铸件内部通道发生残砂堵塞
- 出现砂眼夹杂缺陷
- 后续机加工过程中残留砂粒造成刀具异常磨损

### 不当清砂操作引起的缺陷

不当清砂操作包括过度击打、过高压力喷丸等行为，会引发 [[S25]]：

- 铸件变形
- 表面粗糙度恶化
- 薄壁件开裂

## 与前序/后序工序的衔接

清砂工序与前序工序的衔接匹配要求 [[S4,S10]]：

- 制芯工序需确保砂芯溃散性达标，避免高强度残砂难以清理
- 压射工序需控制铸件留模温度，以获得适宜开展清砂作业的铸件基体强度

清砂工序与后序工序的衔接匹配要求：

- **切边工序**：需在铸件完全去除残留砂粒之后开展，避免冲床模具被砂粒磨损
- **热处理工序**：热处理前必须完成全流程清砂，防止砂粒在高温环境下与铸件基体发生反应
- **表面处理工序**：清砂质量直接影响后续涂层的附着力表现

## 安全、环保与局限

### 安全与环保要求

清砂作业安全环保相关要求 [[S32,S22]]：

| 防控方面 | 具体要求 |
|---------|---------|
| 粉尘防控 | 配套高效吸尘设备，保障车间作业区域粉尘浓度达到国家标准限值；镁合金清砂作业区域严禁明火作业 |
| 噪声防控 | 高噪声清砂设备加装消声设施，采用密闭隔离方式降低噪声对外传播影响 |
| 废砂处理 | 建立废砂再生闭环处理系统，完成废砂中杂质去除后实现砂粒循环复用，符合相关环保排放规定 |

### 工艺局限性

常规振动清砂在处理复杂内腔、细长盲孔类压铸件时，清砂介质难以抵达腔体深处的死角位置 [[S34]]。对于此类结构，需结合产品结构设计优化或采用电液压清砂、电化学清砂等特种工艺加以解决。
