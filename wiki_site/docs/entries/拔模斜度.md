## 定义

拔模斜度，亦称起模斜度、拔模角度、脱模斜度，是为使铸件或塑件顺利脱离模具型腔和型芯，在与模具开合方向平行的壁面上设置的倾斜角度[[S25,S52,S26]]。其核心作用是减少脱模摩擦阻力、降低脱模包紧力，保证制品顺利脱出模具，避免制品拉伤、卡模，同时延长模具使用寿命[[S25,S52,S34]]。

拔模斜度的力学作用机制在于：设置斜度后，可将铸件冷却收缩产生的垂直于型芯壁面的正压力部分分解为沿开模方向的分力，大幅降低脱模时的摩擦力。金属铸件冷却收缩时，外表面会因带斜度的结构自然脱离型腔壁；内表面虽会向型芯方向收缩贴紧型芯，但斜度可有效降低包紧力的作用效果[[S20,S24,S30]]。

## 工艺作用

拔模斜度在铸造和注塑工艺中发挥以下关键作用：

1. **减小脱模力**：脱模力与拔模斜度的定量关系可表达为 \(F_{脱} = F_{正}(f \cdot \cos\alpha - \sin\alpha)\)，其中 \(\alpha\) 为拔模斜度，摩擦系数 \(f\) 确定时，斜度越大，所需脱模力越小[[S50,S17]]。
2. **改善表面质量**：合理的拔模斜度可使脱模过程顺畅，避免铸件表面出现沿开模方向的擦痕、起毛等表面质量缺陷[[S52,S48,S49]]。
3. **减少模具磨损**：拔模斜度不足会显著增大脱模时的滑动摩擦阻力，加剧模具型壁磨损，提升型腔表面拉伤、粘模的概率[[S52,S48,S49]]。
4. **保证尺寸精度**：拔模斜度在图纸标注中不与尺寸公差叠加计算，未指定端点的尺寸按斜度值独立核算[[S52,S51,S26]]。

## 分类

### 按拔模表面属性分类

| 类别 | 定义 | 关键特性 |
|------|------|----------|
| **外拔模斜度** | 成型件垂直于分型面的外侧壁上设置的斜度[[S20,S52]] | 冷却收缩时外侧面金属/塑料会自然与型腔壁分离，所需斜度较小[[S20,S24]] |
| **内拔模斜度** | 成型件垂直于分型面的内侧壁（含内孔壁、凹挡壁）上设置的斜度[[S20,S52]] | 冷却收缩时内侧壁材料会向凸出型芯方向收缩贴紧型芯，通常需为外拔模斜度的2倍；深窄内孔的斜度可达外拔模斜度的3倍以上[[S20,S52,S24]] |

### 按表面状态分类

拔模斜度的设置形式有三种：增加铸件壁厚、减少铸件壁厚、加减铸件壁厚相结合[[S26,S11]]。选用规则如下：
- 加工面：采用增加铸件厚度的方式生成，保证加工余量覆盖斜度带来的尺寸偏差[[S26,S43]]；
- 非加工面（不配合）：可采用增加、加减或减少铸件厚度三种形式[[S26,S43]]；
- 非加工面（与其他零件配合）：拔模斜度应适当减半，以保证装配尺寸精度[[S43,S26]]。

### 按适用材质分类

不同合金在压铸中的最小拔模斜度存在差异[[S1,S24,S2]]：

| 合金 | 最小拔模斜度（比例） | 对应角度 |
|------|---------------------|----------|
| 锌合金 | 1:200 | 0.3° |
| 铝合金 | 1:100 | 0.6° |
| 镁合金 | 1:100 | 0.6° |
| 黄铜及青铜 | 1:80 | 0.7° |

### 按工艺类型分类

- **砂型铸造**：粘土砂造型与自硬砂造型的拔模斜度按不同标准选取；起模困难的模样允许将斜度提升至标准值的1倍；高压造型（工作比压≥700kPa）可将斜度值增加最多50%[[S3,S43]]。
- **金属型铸造**：金属铸型无退让性，冷却收缩后抱紧铸型阻力大于砂型铸造，需设置比砂型铸造更大的拔模斜度[[S23,S4]]。
- **熔模铸造**：深度不大的通孔、蜡模可自由收缩的特征可不设拔模斜度；其他垂直于分型面的熔模表面需设置斜度以降低取模阻力[[S11]]。
- **注塑成型**：常规塑料制品脱模斜度范围为0.5°~5°，复杂深腔结构需提升至4°~6°[[S52,S7]]。

## 标准参数与推荐值

### 国家标准依据

- GB/T 8844—2003《压铸模技术条件》规定：未注拔模斜度时，成型内侧壁的斜度不得大于标准规定值，外侧壁斜度不超过内侧壁规定值的1/2；圆型芯的拔模斜度需符合对应标准规定；文字符号的拔模斜度推荐取10°~15°[[S12,S31,S34]]。未注斜度方向时，默认按减少铸件壁厚方向加工[[S12,S31,S35]]。
- GB/T 6414《铸件尺寸公差》明确规定，拔模斜度不属于该标准定义的铸件尺寸公差覆盖范围，需在标准要求外单独核算对应的尺寸增量/减量[[S51,S26,S18]]。
- JB/T 5105-91《铸件模样 起模斜度》规定砂型铸造用木模样、金属模样、塑料模样的起模斜度不纳入GB 6414-86的尺寸公差范围内，斜度数值独立核算[[S26,S55]]。

### 通用压铸拔模斜度范围

通用压铸/铸造场景下，常规拔模斜度的比例范围为5/1000~25/1000（即1:40~1:200），平均取值为10/1000（即1:100）；芯头与活块结构的推荐拔模斜度范围为10/1000~20/1000（即1:50~1:100）[[S57]]。

### NAPCA设计计算

压铸拔模斜度可通过行业通用公式量化计算：斜度增量 \(D = L/C\)，其中 \(L\) 为脱模方向上的特征长度，\(C\) 为对应不同压铸合金、不同内外壁属性的经验常数。铝合金外表面 \(C=100\) 时斜度比例为1:100，内表面 \(C=30\sim35\) 时斜度比例为1:30~1:35[[S2,S54]]。

### 增压缸体案例

针对增压缸体类液压承压铸件的结构特征，行业推荐外拔模斜度取1:100，内拔模斜度取1:35，该组参数适配缸体类铸件的脱模需求，匹配铸件收缩后内外壁的脱模阻力差异[[S25]]。

## 设计准则与限制

### 通用设计规则

- 所有平行于模具开模方向的成型面均需设置拔模斜度[[S52,S34]]。
- 内表面（型芯成型面）拔模斜度通常为外表面（型腔成型面）的2倍[[S52,S34,S2,S5,S14]]。
- 常规压铸件分模面两侧的最小拔模斜度推荐取1°~1.5°[[S14]]。
- 拔模斜度引起的尺寸偏差不计入尺寸公差值内[[S27]]。

### 特殊结构要求

| 结构类型 | 最小拔模斜度要求 |
|----------|-------------------|
| 箱形深腔零件（深度50~100mm） | 凹模部分≥1.5°[[S46]] |
| 箱形深腔零件（深度>100mm） | ≥1°[[S46]] |
| 容器类深腔成型品内侧 | 至少比外侧大1°[[S46]] |
| 薄壁结构底部加强肋 | ≥0.5°[[S46]] |
| 薄壁结构壁部纵向加强肋 | ≥0.25°[[S46]] |
| 压铸件表面文字、图案类凸起 | ≥10°[[S34,S40,S27]] |

### 表面质量要求

| 表面状态 | 拔模斜度要求 |
|----------|-------------|
| 光面小制件（注塑） | 1°[[S7]] |
| 光面大制件（注塑） | 3°[[S7]] |
| 蚀纹面 Ra<6.3 | 3°[[S7,S48]] |
| 蚀纹面 Ra≥6.3 | 4°[[S7,S48]] |
| 细皮纹 | ≥1.5°[[S52,S48,S29]] |
| 中皮纹 | ≥2°[[S48,S29]] |
| 粗皮纹 | 3°~5°[[S48,S29]] |
| 橡皮纹、条纹等超大纹面 | 5°~10°[[S48,S29]] |

### 拔模斜度过小与过大的影响

- **斜度不足**：导致脱模摩擦阻力过大，铸件凝固收缩后包紧模具型壁，脱模过程中发生硬性刮擦，在铸件表面形成沿开模方向的线状拉伤痕迹，严重时出现整片面状伤痕、掉肉缺肉现象[[S8,S10,S16,S58]]；抽芯斜度不足时还会直接拉裂铸件浇口等薄弱部位[[S13]]；斜度过小还会导致顶出变形，铸件在顶出力作用下发生形状畸变[[S58]]。
- **斜度过大**：会导致铸件脱模方向两端尺寸差过大，额外增加铸件非功能侧加工余量，提升后续机械加工成本；同时过大的斜度会占用更多尺寸公差，无法满足高精度尺寸面的公差要求，还可能导致深腔类零件的局部壁厚不符合设计规范[[S29,S38]]。

## 缺陷关联

### 机械拉伤

国标GB/T 29092-2012将机械拉伤定义为开模时由于模具型芯或拔模斜度不够导致的铸件损伤，表现为铸件表面沿开模方向出现连续擦伤痕迹，痕迹具有一定深度，严重时形成面状伤痕甚至掉肉[[S33,S16,S21]]。预防措施包括修正模具斜度、降低型腔表面粗糙度、调整顶出机构保证顶出力平衡[[S33,S16,S21,S58]]。

### 粘模拉伤

铸件与型壁发生焊合粘连，脱出时铸件被撕破拉伤，严重时表面出现多肉或缺肉缺陷[[S10]]。拔模斜度不足是主要诱因之一[[S8,S10]]。

### 顶出变形

拔模斜度不足导致脱模阻力远大于铸件顶出部位的结构刚性，铸件在顶出力作用下发生形状畸变，局部出现凹陷、翘曲等不可逆变形[[S58]]。

## 各工艺应用差异

| 工艺类型 | 斜度选取特点 | 与其他工艺的差异 |
|----------|-------------|-----------------|
| 压铸 | 锌合金最小拔模斜度1:200（0.3°），铝合金/镁合金1:100（0.6°），铜合金1:80（0.7°）；内表面斜度为外表面的2倍[[S1,S24,S2]] | 精度高，拔模斜度影响尺寸精度更为敏感 |
| 金属型铸造 | 需设比砂型铸造更大的拔模斜度，斜度取值随铸件特征高度升高而减小[[S23,S4]] | 金属型无退让性，收缩抱紧力更大 |
| 熔模铸造 | 深度不大的通孔、可自由收缩的特征可不设拔模斜度[[S11]] | 蜡模可自由收缩，要求相对宽松 |
| 注塑成型 | 常规范围0.5°~5°，复杂结构4°~6°；ABS模芯35'~1°，模腔1°20'[[S52,S46,S48]] | 收缩率与金属不同，温度梯度影响更大 |

四类主流工艺的共性规律：①所有垂直于分型面的成型面均需设置拔模斜度；②内表面/孔特征的拔模斜度需大于外表面斜度；③拔模斜度取值随成型特征高度/深度增加而适当减小；④表面粗糙度更高、带有皮纹/滚花的成型面需要更大的拔模斜度[[S52,S24,S2]]。
