## 概述

石膏型铸造模（plaster mold/gypsum casting mold）是将石膏浆料灌注至预定模样所在的型箱内，待浆料胶凝硬化后经脱模、烘干、焙烧成型，可直接用于承接熔融金属液凝固成型的铸型构件，属于**可消失一次性铸型**类别 [[S2]]。

石膏型本身是石膏型精密铸造工艺流程中的中间产物，二者不能等同；石膏型可独立作为功能部件存在，而全流程石膏型铸造还包含制模、配浆、灌浆、脱蜡、浇注、清砂等完整全链条工序 [[S26,S27]]。

石膏型铸造主要面向非铁合金铸件生产，铸件重量覆盖0.03 kg至908 kg，可制备最小壁厚0.5~1 mm的薄壁复杂构件 [[S32,S2]]。现行国家标准 **GB/T 39314-2020** 明确规定了铝合金石膏型铸造的完整工艺流程和质量验收要求；中国机械工程学会铸造学会编纂的《铸造手册 第6卷 特种铸造》明确收录了石膏铸型的材料配方、制备工艺和性能测试全流程要求 [[S27,S1]]。

## 分类

石膏铸型按结构形态可分为**整体实心石膏模**和**薄壁壳层石膏模**两类；按制备工艺可进一步细分为五个子类 [[S21,S23,S32]]：

| 分类名称 | 工艺特点 |
|---------|---------|
| 普通非发泡石膏型 | 常规灌注成型，透气性极低 |
| 压蒸石膏型（Antioch型） | 经蒸汽加压处理，形成连通孔隙，透气性大幅提升 |
| 发泡石膏型 | 添加发泡剂形成大量内部气泡孔隙，透气性改善显著 |
| 熔模脱蜡石膏型 | 用蜡模作为母模，加热熔失蜡模获得型腔 |
| 起模（拔模）石膏型 | 通过机械方式拔出模样形成型腔 |

按脱模方式的大类划分，石膏型铸造可分为**熔模石膏型铸造**与**起模石膏型铸造**两条并行技术路线。起模石膏型可进一步细分为普通石膏型、发泡石膏型、压蒸石膏型三类 [[S32,S21,S23]]。

## 结构与组成

### 材料组成

常规铸造用石膏型的质量组成为 **70%~80%的半水石膏基体**与 **20%~30%的功能添加剂** [[S2,S32]]：

- **基体材料**：α型或β型半水石膏
- **耐火填料**：SiO₂-Al₂O₃系材料（石英粉、滑石、硅砂等）
- **增强相**：玻璃纤维、无机纤维等
- **调节剂**：促凝剂/缓凝剂（如Na₂SiF₆、NaCl、MgCl₂等）
- **特殊添加剂**：发泡剂或相变调节剂（特定配方）

常规石膏浆料的水料比范围为 **120~180**（100份干混合料对应120~180份重量的水） [[S9,S33,S32]]。

### 特定工程配方示例

针对ZL203合金叶轮的专用配方 [[S7]]：

| 组分 | 比例 |
|-----|------|
| 石膏 | 65% |
| 石英粉 | 35% |
| 氧化钙（外加） | 1% |
| 氯化钡（外加） | 0.06% |
| 水（外加） | 85%~90% |

### 基体材料对比：α型与β型半水石膏

α型半水石膏是在1.3大气压力下湿态水蒸气环境中对二水石膏做脱水处理得到，晶粒完整致密，拌和用水量少，硬化后铸型强度高、线收缩率低，是高端精密铸造石膏模的首选原料；β型半水石膏是普通干加热环境下煅烧得到，晶粒疏松多孔，拌和用水量高，硬化后铸型强度低，成本更低，适配普通艺术铸件、低精度小件的生产场景 [[S10,S32]]：

| 性能指标 | α型半水石膏 | β型半水石膏 |
|---------|------------|------------|
| 显微组织 | 致密 | 疏松多孔 |
| 完全干燥后抗压强度 | 5~15 MPa | 2~5 MPa |
| 拌和用水量 | 低 | 高 |
| 线收缩率 | 小 | 较大 |
| 适用场景 | 大尺寸、高精度铸件 | 艺术铸件、低精度小件 |
| 成本 | 较高 | 较低 |

## 工艺作用

### 制型工艺流程

石膏型铸造分为**起模法**与**熔模法**两条并行工艺路线 [[S30,S32]]：

- **起模法**：模样设计制造 → 铸型灌浆 → 浆料硬化 → 起模（机械拔出） → 铸型蒸汽处理/干燥 → 焙烧
- **熔模法**：压型设计制造 → 熔模压制 → 组装蜡模组 → 灌浆 → 硬化 → 脱蜡（加热熔失） → 焙烧

后续统一衔接：合金熔炼浇注 → 凝固冷却 → 铸件清理 → 热处理 → 精整 → 检验入库。

### 胶凝与硬化

常规石膏浆料的初凝时间为 **7~20 min**，终凝时间不超过 **20 min**，石膏完全固化后即可开展后续起模或脱蜡工序 [[S7,S33]]。

### 焙烧与脱水工艺

石膏型需分步加热完成脱水 [[S15,S9,S4,S16]]：

| 焙烧阶段 | 温度范围 | 作用 |
|---------|---------|------|
| 低温段 | 120~260 °C | 脱除自由水 |
| 高温段 | 200~760 °C | 脱除结晶水 |

总保温时间随铸型尺寸变化：小尺寸铸型短至 **2 h**，大尺寸铸型最长可达 **24 h**。脱水后石膏型线收缩率通常为 **1%~2%**，强度较焙烧前下降约 **50%** [[S9,S4]]。

![石膏型焙烧工艺温度变化曲线图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/75b1e65e-b337-4125-97e0-7a0b423949a3/resource)

图1展示了石膏型焙烧工艺温度变化曲线，随焙烧时间推进温度呈阶梯式上升，对应分阶段脱除自由水和结晶水的工艺要求，直观说明了梯度升温的必要性 [[S18]]。

### 浇注参数

| 参数 | 典型范围 | 备注 |
|-----|---------|------|
| 铝合金浇注温度 | ≈700 °C（一般），≤720 °C（大型薄壁件） | 石膏型导热性差，浇注温度可低于其他铸造方法 |
| 石膏型预热温度 | 150~300 °C | 大型复杂薄壁件取上限，中小型壁厚件取下限 |

石膏型工作温度过低时受金属液激热易产生裂纹，温度过高则铸件凝固速度慢、易出现粗大组织 [[S28]]。

### 浇注方式适配

石膏型可适配**重力浇注、低压浇注、差压浇注、真空辅助浇注、低压加压浇注**等多种浇注方式，其中低压石膏型已实现ZL203合金叶轮这类复杂大尺寸铸件的稳定量产 [[S29,S7]]。

## 关键性能

### 导热性

石膏铸型的导热系数仅为普通砂型的 **1/3~1/15**（约砂型的10%），热传导速率极低 [[S13,S11,S25]]。非发泡石膏型的金属液冷却速度比砂型低 **3倍以上**，发泡石膏型低 **6倍以上** [[S11,S25]]。

低导热性的双重效应：
- **优点**：大幅延长金属液充型后的保持流动时间，适配薄壁复杂铸件的充型需求，可实现厚度低至0.5 mm的超薄壁铸件稳定成型 [[S13]]
- **缺点**：铸件凝固缓慢，晶粒易出现粗大问题[[S11,S25]]

### 透气性

常规非发泡石膏型透气率极低，仅为同规格普通砂型的 **1/10以下**；经过高温蒸汽加压处理的压蒸石膏型（Antioch型）和掺加发泡剂的发泡石膏模内部形成连通孔隙，透气性能大幅提升，可避免浇注过程中的卷气缺陷 [[S15,S17]]。

### 强度

| 石膏类型 | 完全干燥后抗压强度 | 适用场景 |
|---------|------------------|---------|
| α型半水石膏 | 5~15 MPa | 大尺寸铸件 |
| β型半水石膏 | 2~5 MPa | 小尺寸、低载荷铸件 |

### 溃散性与清理

石膏模的溃散性能优异，浇注完成后仅需低压水流冲洗即可完全破碎清除，不存在砂型铸造残留砂芯难以清理的问题，尤其适合复杂内流道、异形深腔铸件的后处理 [[S32]]。

此外，添加约20%硫酸镁调节剂可得到**水溶石膏芯**，在常温水中的复溶溶解率可达 **80%~95%**，无需人工凿除即可从铸件复杂内空腔中自行溶出 [[S5]]。

### 最高工作温度

石膏型最高适用工作温度约为 **1200 °C**，仅可用于铝合金、镁合金、锌合金及部分低熔点铜合金等非铁金属的铸造，无法适配钢铁等高熔点铁基合金的生产。此外，石膏中的硫元素会与铁发生反应生成缺陷 [[S15,S13]]。

## 常用合金

石膏型适配下列非铁合金的铸造 [[S32,S13]]：

- **铝合金**（最常用）：以A356为代表的铝硅系铸造铝合金
- **锌合金**
- **镁合金**
- **铜合金**（部分低熔点牌号）

石膏型**无法适配铁系合金**，因为石膏中的硫会与铁反应导致铸件缺陷 [[S13]]。

## 与压铸的关系

常规高压压铸要求铸型承受 **30~100 MPa** 级高压、**0.5~50 m/s** 高速充型工况，石膏型常温焙烧后强度不足以承受该类载荷，且石膏最高工作温度仅约 **1200 °C**，无法兼容常规高压压铸的工艺要求 [[S28,S14,S12,S13]]。

行业中石膏型的主流应用场景是为高压压铸提供**低成本快速试制样件**，验证产品设计，大幅降低试模成本。压铸模具的成本比石膏型模具成本高约 **10倍**，石膏型模具不仅便宜，而且生产周期短，可变性大，实现设计更改又快又便宜。石膏型铸造并非为了与压力铸造竞争，而是在为压力铸造生产原型模中发挥关键作用 [[S12,S14]]。

## 与其他铸型的对比

### 与砂型铸造的对比

| 对比维度 | 石膏型铸造 | 砂型铸造 |
|---------|-----------|---------|
| 模具成本 | 较高 | 较低 |
| 导热系数 | 仅为砂型的1/3~1/15 | — |
| 表面光洁度 | Ra 0.8~3.2 μm，远优于砂型 | 较粗糙 |
| 尺寸精度 | CT3级 | 较低 |
| 可铸最小壁厚 | 0.5 mm | 受限 |
| 溃散性 | 优异，水流冲洗即可清除 | 残留砂芯难清理 |
| 透气性 | 极低（非发泡型） | 较好 |
| 生产效率 | 较低，周期长 | 较高 |
| 适用合金 | 仅非铁金属 | 几乎所有合金 |

石膏型铸造可实现表面粗糙度达 **Ra 0.8~3.2 μm**、尺寸精度达 **CT3级** 的铸件 [[S32]]。

## 缺陷与防止

石膏型铸造常见缺陷及成因收录于《中国材料工程大典》等专业手册中 [[S8,S31]]：

| 缺陷类型 | 产生原因 | 防止措施 |
|---------|---------|---------|
| 气孔 | 石膏型透气性差、浇注时水分未完全脱除 | 优化浇注系统、抽真空辅助、确保充分焙烧脱水 |
| 表面针孔 | 石膏型吸潮、浇注温度不当 | 控制浇注温度、铸型充分干燥 |
| 缩孔/缩松 | 凝固补缩困难 | 优化浇冒口设计、采用差压/低压浇注 |
| 石膏夹杂 | 铸型强度不足、浇注冲刷 | 改善石膏配方、控制浇注速度 |
| 晶粒粗大 | 导热性差导致凝固缓慢 | 控制铸型预热温度、优化合金成分 |
| 尺寸精度不合格 | 焙烧收缩控制不当 | 严格控制焙烧工艺、预留收缩余量 |

## 应用领域

石膏型铸造的应用特点与选型建议 [[S12,S14]]：
- 具备以下条件时可优先考虑石膏型铸造：
  1. 铸件尺寸在4~36英寸范围内
  2. 毛坯或砂型铸件需大量机加工，而石膏型铸件可减少机械加工
  3. 由多个简单件焊接成的复杂组合件可用石膏型铸造整体成型
  4. 铸件壁厚极薄（≤0.5 mm）
- 主要应用行业：航空、航天、兵器、电子、艺术品铸造等 [[S32]]
- 典型零件：叶轮、锁具零件、齿轮、阀门、管件、工装、装饰品等 [[S2]]
