## 概述

球磨机衬板液态模锻模具是针对高锰钢、中铬合金等耐磨球磨机衬板开发的特种挤压铸造工装，其功能是将定量熔融耐磨金属注入型腔后施加高压，使金属在压力下完成结晶凝固同时产生少量塑性变形，由此消除衬板内部缩孔缩松缺陷，显著提升致密度与耐磨性能[[S14,S8]]。球磨机衬板安装于球磨机筒体内壁，直接承受研磨体与物料的反复冲击和磨损，对材质致密性和耐磨性要求极高。格子型球磨机的结构如图1所示，筒体内侧装有格子栅板与各类衬板，构成复杂的磨损环境[[S21]]。

![格子型球磨机剖面示意图，显示筒体、给矿端、排矿端、格子栅板及衬板位置](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/93c23bd6-5487-49b9-b15c-3c8fbcbdce86/resource)

采用液态模锻工艺生产衬板，可免去或减少后续机械加工，获得接近最终形状的致密耐磨件，而模具设计是保证该工艺成功的关键。

## 模具总体结构

球磨机衬板液态模锻模具由凸模（压头）、凹模、模架、浇注系统、排气与溢流系统、推出机构、导向系统和温度调节系统等核心部分组成[[S30,S19,S23]]。一种典型液态模锻模具的三维结构如图2所示，直观展示了上下模座、压头（凸模）和型腔的相对位置[[S16]]。

![液态模锻模具三维结构示意图，标注上下模座、压头与型腔](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/5c95dc14-65fe-4511-aa27-e83b5e497f29/resource)

凹模多采用**可分组合凹模**，由2～3块拼接而成，以适应衬板波浪形复杂轮廓，便于加工和脱模[[S30,S23]]。凸模与凹模之间的**单边配合间隙**取0.12 mm，配合等级采用H9/f9，既能防止金属液喷出形成毛刺，又可避免卡滞咬死[[S27,S19]]。

模架由动模座板、定模座板、套板、支承板、垫块等标准支承固定零件组成，将各功能部件定位并固定在液压机上；相关零件设计可优先选用GB/T 4678系列压铸模标准件以降低制造成本[[S33,S29,S15]]。

浇注系统由直浇道、横浇道、内浇口和分流锥构成，起到平稳充型、减少卷气以及在保压阶段传递压力进行补缩的作用[[S11,S33]]。排气与溢流系统包括开设在型腔角落、厚大部位的排气槽和位于充型末端的溢流槽，用以排出型腔气体、涂料挥发物及冷污金属液，防止衬板棱角塌陷和气孔缺陷[[S27,S11,S28]]。

推出机构采用顶杆、顶板、推杆固定板与推板导柱导套的组合结构，在衬板凝固完成后将其平稳顶出，避免变形；复位杆可使推出零件自动复位[[S30,S7,S19]]。导向系统对于大尺寸、高精度的球磨机衬板模具须设置独立的导柱和导套，引导动、定模精准对合；中小件可仅依靠冲头与凹模配合实现自导向[[S30,S3,S11,S42]]。

模具的温度调节系统嵌有电热管加热通道和循环水冷却通道：生产前将模具预热至150～250 ℃（常用200 ℃），生产中通冷却水将模温稳定在设定区间，防止因频繁冷热交变产生热疲劳开裂[[S11,S26,S39]]。推荐工作零件材质选用H13热作模具钢，以适应高锰钢液的高温工作环境[[S39]]。

## 关键参数设计计算

### 分型面与收缩率
分型面优先选择通过衬板最大横截面的平面，避免倾斜分型面产生过大侧向力，保证成形后易脱模并减少加工余量[[S32,S11]]。钢质液锻件在高压下的凝固收缩率显著降低：大气压下钢的收缩率约1.5%，70 MPa压力下约0.55%，100 MPa下约0.35%[[S13]]。模具型腔尺寸需按缩尺加算，例如型芯尺寸计算公式为：

```
D' = (D - Δ/2 + Dε₁ + T)⁰_(-T)
```

式中D为液锻件最大极限尺寸，ε₁为综合收缩率，T为制造公差[[S4]]。

### 浇注系统尺寸
内浇口截面积可按

```
F内 = Q / (ρ·v·T)
```

计算，其中Q为铸件质量，ρ为液态金属密度，v为内浇口处流速，T为填充时间[[S31]]。内浇口**厚度**推荐1.5～3 mm，**长度**2～3 mm，且内浇口厚度与铸件壁厚比应小于1/4；对薄壁小件可取0.5～2.0 mm[[S12,S44]]。

横浇道推荐厚度＞6.2 mm，长度≥32 mm，宽度≤84.4 mm；转弯角度宜取80°～85°，避免90°直角转折；横浇道截面积应大于直浇道出流截面积的1.2倍[[S12,S31]]。直浇道料饼厚度取直浇道直径的1/2～1/3，脱模斜度1°30′～2°[[S1]]。立式冷压室机型中，分流锥直径按

```
d₃ ≥ √(d₂² - (1.1～1.3) d₁²)
```

确定，且须满足(d₂ − d₃)/2 ≥ 3 mm[[S22]]。

### 排气溢流系统
排气槽深度取0.05～0.1 mm，宽度5～20 mm，开设在型腔转角、分型面结合处等气体易聚集的位置；溢流槽设置在充型末端，容纳温度偏低、含浮渣的多余金属液[[S42,S11]]。

### 液锻比压与锁模力
液锻比压是保证衬板内部质量的核心参数，通用经验公式为

```
p = K₁·K₂ [1 + 0.001 (H/a)³]
```

式中K₁为合金种类系数，K₂为液锻方式系数；对于间接液锻可简化为p = K₁·K₂[[S20]]。不同材料的推荐比压参见表1。

**表1 不同材料液态模锻推荐比压**
| 材料类别 | 推荐比压 (MPa) | 来源 |
|----------|----------------|------|
| 钢质耐磨件（球磨机衬板、斗齿等） | 100～150 | [[S24]] |
| 铝合金 | 40～100 | [[S40]] |
| 铸铁类 | 50～120 | [[S9]] |

液锻力 P₀ = p·A，其中A为与压力垂直的受压投影面积。锁模力必须大于胀型力（胀型力为比压与型腔总投影面积之积），为安全起见锁模力取胀型力的**1.1～1.3**倍[[S9,S40]]。

### 模具强度与刚度校核
对于整体式矩形凹模，侧壁与底部厚度可按刚度条件校核：

```
t₁ = t₂ = ∛(K p a⁴ / (E δ))
```

式中K为与型腔侧壁间长度和型腔深度比值相关的常数，p为液锻压力，a为型腔深度，E为模具材料弹性模量（钢取2.1×10⁵ MPa），δ为允许变形量，推荐控制在**0.05～0.1 mm**[[S4]]。强度校核可用弯矩公式进行验算[[S4]]。

### 温度控制参数
黑色金属（如高锰钢）衬板液锻模具的预热温度范围为150～400 ℃，常用200 ℃[[S5]]。冷却水流量可按

```
L = Q / (y·K·(T₃ - T₄))
```

计算，其中Q为单位时间散热量，y为冷却介质密度，K为热导率（取0.64 W/(m·K)），T₃与T₄分别为出、进水温度[[S9]]。合理布置冷却水道，结合仿真分析（如冲头正下端区域温度较高[[S43]]），可控制模温均匀，延长模具寿命。

## 关键结构及零件设计

### 凹模与凸模
凹模采用可分组合结构，由2～3块拼合，垂直分型，便于加工衬板的波浪形型腔，并在脱模时使模具打开[[S30,S23]]。凸模（压头）的作用是封闭型腔并传递压力，需要与凹模保持精密配合：单边间隙0.12 mm，H9/f9配合等级[[S27,S19]]。

### 导向机构
大尺寸球磨机衬板模具合模压力大且尺寸精度要求高，必须配置独立的导柱、导套导向系统，引导动、定模精准对合，防止错位[[S11,S42]]。

### 推出机构
优先采用顶杆式推出，顶出位置选择在衬板非工作面或后续待加工区域，使推出力均匀分布，避免衬板变形。脱模力可按下式估算：

```
F = A·l·p·(μ cosα - sinα)
```

高锰钢材料可取单位面积包紧力p = 5 MPa，摩擦系数μ = 0.5，并结合合理的脱模斜度设计[[S41,S11]]。推出机构应配备导向及复位装置[[S30,S7]]。

### 镶块结构
在浇口等受高温钢液持续冲刷的部位，单独设置H13热作模具钢材质的可更换镶块，通过沉头螺钉固定在模架的沉槽中。损坏时只需更换镶块而无需拆卸整体模具，降低维护成本[[S41,S38]]。

## 模具材料与热处理

球磨机衬板液态模锻模具的工作零件（型腔镶块、冲头等）推荐选用**4Cr5MoSiV1（H13）热作模具钢**，其在高温下的冷热疲劳抗力显著优于3Cr2W8V钢，可大幅提升服役寿命[[S39,S2]]。H13钢的推荐热处理规范为：淬火加热温度1050～1100 ℃，随后进行550～600 ℃两次高温回火[[S17]]。

3Cr2W8V钢虽在600～700 ℃下仍保持高硬度和强度，但其冷热疲劳性能较差，不适用于频繁冷热交变的衬板液锻生产工况[[S17,S10]]。5CrNiMo属于低耐热高韧性钢，热稳定性不足，工作温度超过550 ℃后表层硬度快速下降，因此仅推荐用作模套、固定板等非直接接触高温钢液的支撑零件[[S17,S10,S37]]。

在实际生产中，黑色金属液态模锻模具的寿命可达3000～6000件，若结构合理、材料与热处理工艺得当，可进一步延长[[S25]]。

## 典型缺陷与防止措施

### 冲蚀（腐蚀）
高温高锰钢液对型腔表面持续冲刷，导致浇口等部位产生冲蚀，使表面凹凸不平、棱角变钝[[S6,S36]]。**防止措施**：在金属液流入的浇口位置设置可更换的H13耐磨镶块；对型腔表面进行渗氮或氧化处理，形成致密防护层[[S6,S36]]。

### 热疲劳裂纹
模具在反复接触1300 ℃以上高温钢液和冷却介质的过程中，型腔表面承受交变热应力，易产生龟裂状短裂纹，裂纹内常残留氧化物和金属残渣[[S6,S34,S35]]。**防止措施**：严格控制模具预热温度与钢液浇注温度的温差，避免过快的冷却速度；优先选用高热疲劳抗力的H13钢替代3Cr2W8V；生产间歇期间对模具进行保温缓冷处理[[S6,S34,S35]]。
