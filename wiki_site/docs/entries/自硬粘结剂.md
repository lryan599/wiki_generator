## 概述

自硬粘结剂（no-bake binder / self-hardening binder），又称冷硬粘结剂（cold setting binder），是一类在室温条件下通过催化剂或固化剂引发化学反应而自行固化的粘结剂体系，制得的型（芯）砂无需烘干工序即可在室温下完成硬化[[S5,S3,S16]]。该粘结剂属于化学粘结剂大类，依靠混砂时预先加入的液体固化剂（而非后续通入的气态硬化剂或外部加热）实现交联固化，因而区别于热芯盒、冷芯盒（气硬）及覆膜砂等工艺用粘结剂[[S6,S41,S39]]。

自硬粘结剂是铸造行业中用于制造砂型与砂芯的关键化学粘结材料，其成型精度高、无需大型烘干设备，在中大型铸件单件小批量生产以及复杂内腔成型方面具有广泛的应用优势[[S42,S52]]。

## 固化机理

自硬粘结剂实现室温固化的核心过程是粘结剂组分与预先混入砂中的固化剂之间发生化学反应，使低分子量的液态树脂或水玻璃逐步交联形成三维网络结构，从而在砂粒之间建立牢固的粘结桥。

**酸催化呋喃树脂自硬体系**：以含呋喃环的糠醇树脂为主要组分，在室温下与酸类固化剂（磷酸、对甲苯磺酸、苯磺酸、硫酸乙酯等）接触后，树脂分子中的羟甲基、活性氢原子及呋喃环上的不饱和双键之间发生缩聚与双键聚合反应，由线型结构转化为体型交联网络完成固化[[S24,S35,S13]]。

**酚醛聚氨酯（PEP SET）自硬体系**：由双组分苯基醚酚醛树脂（含羟基—OH）与聚异氰酸酯（含异氰酸根—NCO）预先混合到砂中，在叔胺类催化剂作用下，羟基与异氰酸根发生逐步加成反应生成聚氨酯结构，快速形成交联网络。该体系硬化特性几乎不受环境温度影响，可使用时间长、起模时间短[[S26,S9,S15,S1]]。

**酯固化碱性酚醛树脂自硬体系**：采用强碱催化合成的碱性酚醛树脂（pH 12～14），在室温下与液态有机酯反应，逐步完成酚醛分子的交联，形成不溶不熔的三维网络结构[[S43,S31,S9]]。

**水玻璃有机酯自硬体系**：粘结剂主体为硅酸钠水溶液（水玻璃），有机酯作为固化剂触发水玻璃水解反应，逐步生成硅酸凝胶，通过脱水缩合形成连续的—Si—O—Si—三维聚合网络，实现砂粒间的粘结硬化[[S3,S53,S22]]。

### 有机系自硬粘结剂

- **呋喃树脂自硬粘结剂**：以糠醇为主要原料，配以甲醛、尿素和/或苯酚等合成，含呋喃环的热固性树脂，以酸类为固化剂[[S24,S17]]。
- **酸固化酚醛树脂自硬粘结剂**：甲阶段酚醛树脂，含水量8%～25%，以有机磺酸混合物为固化剂[[S2,S13]]。
- **酯固化碱性酚醛树脂自硬粘结剂**：强碱催化合成的水溶性热固酚醛树脂，pH 12～14，以液态有机酯为固化剂[[S43,S31]]。
- **酚醛聚氨酯（PEP SET）自硬粘结剂**：双组分苯基醚酚醛树脂与聚异氰酸酯组合，以液体叔胺为催化剂[[S26,S9,S1]]。
- **脲醛树脂自硬粘结剂**：以尿素与甲醛缩合而成的氨基树脂，可用酸催化在室温下固化[[S5,S9]]。

### 无机系自硬粘结剂

- **有机酯固化水玻璃自硬粘结剂**：以硅酸钠水溶液为粘结剂主体，液态有机酯为固化剂，反应生成硅酸凝胶固化[[S3,S53,S9]]。
- **磷酸盐自硬粘结剂**：以磷酸盐水溶液或磷酸为粘结剂，以金属氧化物（如MgO、FeO、Cr₂O₃、Fe₂O₃及其混合物）为硬化剂[[S9,S15]]。

## 主要类型及典型参数

### 呋喃树脂自硬体系

呋喃树脂自硬砂是应用最广泛的自硬砂体系。JB/T 7526—2008 标准对铸造用自硬呋喃树脂的技术要求作出了规定[[S17,S40]]。

**呋喃自硬树脂技术要求**[[S29,S40,S17]]：

| 指标 | 技术要求 |
|---|---|
| 外观 | 淡黄色至棕色透明或半透明均匀液体 |
| 黏度 | ≤100 mPa·s |
| pH值 | 6.5～7.5 |
| 游离甲醛（质量分数） | ≤0.5% |
| 密度 | 1.15～1.25 g/cm³ |
| 水分（质量分数） | 2%～10% |
| 常温储存期 | ≥180 天 |

**按氮含量分类**[[S38,S29]]：

| 分类代号 | 氮含量（质量分数） |
|---|---|
| W（无氮） | ≤0.3% |
| D（低氮） | >0.3%～2.0% |
| Z（中氮） | >2.0%～5.0% |
| G（高氮） | >5.0% |

树脂中氮含量主要来源于尿素改性组分，氮含量越高，树脂成本越低，但用于铁合金铸件时产生针孔类气孔缺陷的风险越大；对铜基合金铸件影响较小[[S37]]。

**配比与工艺参数**[[S37,S2,S29]]：

| 参数 | 典型值/范围 |
|---|---|
| 树脂加入量（占砂质量） | 0.8%～1.4% |
| 固化剂加入量（占树脂质量） | 15%～45% |
| 砂温控制区间 | 21～27°C |
| 固化剂总酸度（夏季） | 18%～20% |
| 固化剂总酸度（春秋季） | 23%～26% |
| 固化剂总酸度（冬季） | 27%～31% |
| 可使用时间占比（相对起模时间） | ≈50% |

### 酚醛聚氨酯自硬体系（PEP SET）

该体系由聚苄醚酚醛树脂（组分I）与聚异氰酸酯（组分II）构成粘结剂主体，液体叔胺作为催化剂（组分III）。高沸点芳香烃类混合溶剂用于调节粘度[[S25,S1]]。

| 参数 | 典型值 |
|---|---|
| 组分II加入量（占组分I质量） | 18%～20% |
| 催化剂加入量（占组分I质量） | 2%～10% |
| 可使用时间与起模时间比 | ≈1:2 |
| 成本特点 | 较高 |
| 关键限制 | 含氮量较高，铸钢件易产生气孔 |

该体系硬化速度快、可使用时间长、起模时间短，硬化特性几乎不受室温变化影响，但成本较高[[S9,S25,S1]]。

### 酯固化水玻璃自硬体系

改性水玻璃自硬砂在环保和溃散性方面具有显著优势[[S4,S21,S33]]。

| 性能指标 | 典型值 |
|---|---|
| 粘结剂加入量（占砂质量） | ≤2.0% |
| 有机酯固化剂加入量（占粘结剂质量） | ≤50% |
| 800°C下发气量 | 低至 5.8 ml/g |
| 高温残留抗拉强度 | ≤0.05 MPa |
| 旧砂再生回用率 | ≥90% |
| 残留Na₂O含量 | ≤0.3% |

## 工艺参数与操作控制

### 关键时间参数

自硬砂工艺中有三个核心时间参数控制生产过程[[S7,S50]]：

| 参数 | 定义 | 典型范围 |
|---|---|---|
| 可使用时间（工作/操作时间） | 混砂后至型砂抗压强度增长到0.07MPa（或终强度保留80%）所允许的最长操作窗口 | 约固化时间的1/3 |
| 起模时间（脱模时间） | 混砂后至砂型/砂芯抗拉强度达到0.14MPa（或抗压强度0.4MPa）可无变形取出模样/芯盒的时间 | 15～45 min |
| 可使用时间/起模时间比值 | 表征硬化特性优劣的参数 | 0.35～0.6，最高可至0.8 |

超过可使用时间后对砂体进行扰动操作，将直接破坏已形成的粘结桥，导致最终强度大幅下降[[S2,S37]]。

### 混砂操作要求

混砂工序需严格执行安全规范：先将酸催化剂（固化剂）与原砂充分分散混合，再添加树脂组分，严禁树脂与催化剂在无砂状态下直接接触，否则将发生剧烈放热反应引发危险[[S10,S37]]。对大型砂型/砂芯生产，推荐将零残留连续混砂机设置在造型工位旁，确保混制完成的砂料可在可使用时间内完成装填紧实，避免提前固化失效[[S46]]。

## 常见缺陷与限制

### 氮致针孔缺陷

尿素改性的呋喃自硬树脂体系中，氮元素含量随尿素添加比例同步上升，过高的氮含量在铁合金铸件中可导致表面出现针孔类缺陷。该类缺陷对铜基合金铸件影响极小[[S37]]。

### 硫致表面缺陷

磺酸类催化剂残留的硫元素超标时，一方面导致铸钢件表面质量劣化，另一方面造成球墨铸铁件表层球状石墨衰退，破坏铸件表层力学性能[[S20,S52]]。

### 发气导致皮下气孔

有机自硬粘结剂在高温浇注过程中发生热解，释放CO、烃类等大量挥发性气体。若砂型排气通道设计不合理、涂层透气性不足，气体无法及时从铸型内逸出，将滞留在金属液与砂型界面形成皮下气孔缺陷[[S11]]。图3展示了呋喃自硬树脂铸造过程中因热解析出的亮碳缺陷在铸件基体中的典型显微形貌。

### 强度不足导致的砂眼

自硬砂未达到完全固化时间就提前脱模，或超过工作时间后进行扰动操作，均会导致砂芯强度不足。搬运、合型过程中砂芯受力断裂掉砂，进入金属液后在铸件内部形成砂眼缺陷[[S8]]。

### 环境与职业健康风险

传统呋喃树脂自硬粘结剂含有游离甲醛成分，受OSHA标准29CFR1910.1048法规管控。混砂、制芯、浇注、落砂工序释放的甲醛、酚类、呋喃甲醇蒸汽会刺激人体眼、呼吸道黏膜，长期接触可引发皮炎及中枢神经抑制。同时，呋喃树脂被证实存在致癌性，浇注高温分解还会释放SO₂、NOₓ、CO等有毒气体，污染生产环境[[S10,S20]]。

## 应用场景

自硬粘结剂成型的砂型/砂芯无需整体烘干，仅可配套低温烘烤工序，具备强度高、产尘少的优势，适配绝大多数铸件生产需求[[S42]]。

**适用生产模式**：尤其适配小批量、多品种的中大型铸件制造场景，适合复杂内腔铸件的成型[[S42,S46]]。

**适用铸造工艺**：自硬砂流动性优良，无需高紧实度设备即可完成大型砂型/砂芯的装填，适配重力铸造、低压铸造等所有需要制备砂型砂芯的工艺路线；所得砂型/砂芯强度高，可承受低压铸造的充型压力冲击[[S46]]。

**复砂降本工艺**：可采用面砂用自硬砂、背砂用普通砂的复砂工艺以降低生产成本[[S42]]。

**与热芯盒/冷芯盒工艺对比**：相比热芯盒、壳型铸造工艺，自硬粘结剂无需配置芯盒加热设备，设备投入更低，适合小批量柔性生产；相比冷芯盒工艺，自硬粘结剂无需配置气体发生、气体净化及尾气处理系统，操作门槛更低，但生产节拍低于热芯盒、冷芯盒工艺，不适合极高批量的小型砂芯流水线生产[[S51]]。

### 旧砂再生特性

呋喃自硬砂正常工况下旧砂再生率可达75%～80%，控制再生砂总硫含量低于0.1%即可直接复用配套酚醛聚氨酯自硬粘结剂[[S52]]。酯固化改性水玻璃自硬砂再生回用率可达到90%以上[[S4]]。

| **冷硬粘结剂** | 与自硬粘结剂为等效术语，系同一对象的两种不同称呼[[S5,S16]] |
| **自硬砂** | 由原砂 + 自硬粘结剂 + 固化剂等混合而成的造型材料，自硬粘结剂为其核心组成部分[[S9,S19,S49]] |
| **化学硬化砂** | 所有依靠化学反应完成粘结固化的型/芯砂的总称，按硬化方式分为自硬砂、气硬砂、热硬砂三个子类[[S9,S19]] |
| **有机酯水玻璃自硬砂** | 自硬砂的细分品类，特指以水玻璃为粘结剂、有机酯为固化剂的自硬砂类型[[S9,S15]] |
| **热固化粘结剂** | 需在高于室温的温度下（如热芯盒140～250°C、覆膜砂200～280°C）触发交联固化的粘结剂体系，代表类型为热芯盒树脂、覆膜砂用树脂[[S41,S6,S39]] |
| **冷芯盒（气硬）粘结剂** | 先将粘结剂与砂混合后射入常温芯盒，再通过后续通入气态/雾化固化剂（三乙胺、SO₂、甲酸甲酯等）触发快速硬化的体系，属于气硬范畴[[S6,S30,S41]] |
