## 概述

脱落式芯盒是国家标准GB/T 5611铸造术语体系下的正式芯盒子类，标准英文术语为troughed core box。其标准定义为：形成芯子轮廓的组块，在有倾斜侧壁的斗形外框内组合而成的芯盒，造好的芯子与组块一起自框内倒出，分离后获得芯子[[S31,S1]]。脱落式芯盒属于铸造及压铸领域制芯辅助工装类别，主要用于制备带有侧面凹陷、倒扣、深腔等特征、难以直接垂直起芯的复杂砂芯或可溶芯[[S16,S8]]。

与普通整体敞开式芯盒（不可拆卸，仅适配简单砂芯）、分体式对开芯盒（两半开合，适配平直分盒面砂芯）、普通组合多向拆卸式芯盒（多块独立板件紧固，操作复杂度高）不同，脱落式芯盒依靠带斜度的斗形外框实现组块整体同步脱出，无需复杂定位锁紧结构即可高效取出复杂砂芯[[S16,S8,S17]]。

## 结构与组成

套框脱落式芯盒的标准结构由底板、外套框和芯腔内盒块三部分组成[[S22,S6]]：

- **底板与外套框**：通常采用方形结构，以螺钉固定。外套框的侧板、端板、底板需使用木档或框架加固[[S17,S22]]。
- **芯腔内盒块**：形成砂芯轮廓的组块，根据实际使用尺寸确定是否需要额外加固[[S17]]。

外套框内壁与芯腔内盒块外壁的配合面设置1:10~1:12的导向斜面，对应角度约4.8°~5.7°，以保证制芯完成后内盒块随砂芯一同从外框中无阻碍滑脱，随后分离内盒块获得完整砂芯[[S22,S17]]。

小型脱落式芯盒的典型结构由套框、左右垂直可分的半圆芯壳、底座三个零件组成，适配小尺寸异形砂芯的手工制芯场景[[S6,S20]]。

木质芯盒框的垂直导向斜度通常取5°~6°[[S10]]；芯盒外壁及加强肋斜度取值范围为0.5°~2°[[S20]]。

## 分类与同类别芯盒对比

脱落式芯盒按适用尺寸可分为三个规格等级[[S17,S22,S11]]：

| 规格 | 适用条件 | 说明 |
|------|----------|------|
| 小型 | 常规小尺寸砂芯 | 简约结构，见图示 |
| 中型 | 芯高 H≥100mm，砂芯轮廓平均尺寸 (a+b)/2 ≥200mm | 带标准外框加固 |
| 大型 | 芯高 H≥150mm，砂芯轮廓平均尺寸 (a+b)/2 ≥300mm | 重型框架结构 |

脱落式芯盒与其他同类别芯盒的核心差异如下[[S16,S8,S17]]：

| 芯盒类型 | 结构特征 | 脱盒方式 |
|----------|----------|----------|
| 整体敞开式芯盒 | 不可拆卸整体结构 | 直接垂直起芯 |
| 分体式对开芯盒 | 两个半体，定位销/锁紧机构开合 | 沿分盒面开合取芯 |
| 普通组合多向拆卸式芯盒 | 侧板、端板、定位板等多块独立件，紧固件固定 | 多向分步拆卸，操作复杂 |
| 脱落式（套框）芯盒 | 斗形外框、组块式内盒，配合斜面 | 整体倒出后分离组块，高效 |

## 关键设计参数

### 配合斜度
- 外套框内壁与内盒块外壁配合斜度：1:10~1:12（约4.8°~5.7°）[[S22]]
- 芯盒外壁及加强肋斜度：0.5°~2°[[S20]]
- 木质可卸式芯盒框垂直导向斜度：5°~6°[[S10]]

### 配合间隙控制要求
- 合模状态下分盒面平面间隙：≤0.1mm[[S14]]
- 活块与芯盒窝座配合间隙：0.2~0.3mm[[S3]]
- 定位销与销套同轴度误差：≤0.01mm[[S29]]

### 材料与耐磨处理

盒体材料依尺寸和载荷选用[[S16,S6,S34]]：

| 应用场景 | 推荐材料 |
|----------|----------|
| 中小型芯盒 | 铝合金（ZL101、ZL102、ZL103、ZL104、ZL201等） |
| 大型重载芯盒 | HT150、HT200灰铸铁 |
| 小体积活块、镶块 | 青铜或低碳钢 |

铝质芯盒的刮砂面、分盒面需加装厚度3mm的30钢或Q235-A钢制防磨护板，采用沉头螺钉加定位铆钉双重紧固[[S20,S7,S34]]。

### 定位与锁紧
- **定位元件**：45钢制造定位销，根据芯盒轮廓尺寸选型：≤300mm选φ8mm，300~500mm选φ10mm，≥500mm选φ12mm，工作段淬火硬度45~50HRC；止口定位适配小尺寸立式填砂芯盒[[S29]]。
- **锁紧/防松结构**：蝶形螺母-活节螺栓、快速铰链螺杆、滑键式防松挡块，防止制芯紧实过程中活块或分盒面窜动[[S6,S3]]。

## 工艺作用与工作原理

脱落式芯盒的核心工艺价值在于：制芯完成后，砂芯与芯腔内盒块一同从斗形外框中整体倒出，随后将各内盒块依次分离，获得外形完整的砂芯。这一过程避免了砂芯与芯盒内壁之间因侧面凹陷、倒扣等特征导致的干涉取芯难题[[S31,S22]]。

在不同制芯工艺场景下，脱落式芯盒的配套脱芯方式有差异：

- **热芯盒制芯**：配套垂直/水平双向顶出机构，开盒后先将砂芯稳定留在下半芯盒，再由下顶芯机构平缓顶出，避免薄壁复杂砂芯磕碰断裂[[S25,S21]]。
- **冷芯盒制芯**：增设密封环、排气腔配套结构，三乙胺催化硬化完成后，多方向分步拆卸芯盒组件取出复杂内腔砂芯[[S1]]。
- **压铸用可溶盐芯制芯**：通过多块内盒的分步脱离，直接取出带深侧凹的复杂可溶芯，无需强制脱模，避免盐芯开裂[[S9,S27]]。

## 定位与术语边界

脱落式芯盒的标准英文术语为troughed core box[[S31]]。detachable core box、separable core box、disassemblable core box、removable core box均可作为等效英文别名[[S19]]。需注意以下术语区分：

- loose piece core box对应带活块的普通芯盒，不属于脱落式芯盒范畴；
- drop-out core box易与单独活块取出工艺混淆，并非脱落式芯盒的标准译名[[S19,S13]]；
- 压铸活块和可溶型芯属压铸模组件或可除型芯范畴，与脱落式芯盒（制芯工装）存在工序和概念上的本质区别[[S19,S5]]。

## 与压铸模活块/滑块的关系

脱落式芯盒与压铸模活块/滑块机构分别服务于不同工序[[S5,S15]]：

- **脱落式芯盒**：制芯工序的前置工装，用于预制带复杂特征的砂芯或可溶芯，属于砂制备阶段的分体脱盒工具；
- **压铸模活块/滑块**：压铸工序的模具核心组件，在压铸充型阶段直接成型零件侧凹特征，属于金属液成型阶段的侧向抽芯部件。

二者的导向斜面设计原理相通，耐磨选材与间隙控制的工程规范可互相参考[[S5,S15]]。

## 应用场景

脱落式芯盒典型应用于带倒扣、深腔、复杂内腔特征的压铸件砂芯预制。代表性工业应用案例包括270汽油机曲轴箱、传动箱体等结构件的砂芯制作，可适配铝合金、铸铁、铜合金等绝大多数压铸合金的制芯需求[[S6,S20,S8]]。

## 优势与局限

**优势**[[S16,S1,S34]]：
- 深腔、倒扣特征的砂芯取芯过程中碎裂率大幅降低，复杂砂芯完整性显著提升；
- 无需内置复杂侧抽芯结构即可完成多方向分步脱盒，适配手工/半自动化的中小批量制芯场景；
- 易磨损的刮砂面、分盒面采用独立可拆卸耐磨护板设计，局部磨损后可单独更换，大幅延长工装使用寿命。

**局限**[[S34,S1]]：
- 开合盒的人工分步操作工序多于集成侧抽的整体芯盒，纯机械化连续制芯场景下生产节拍比带自动侧抽的整体芯盒低15%~30%；
- 不适用于数万件以上的超高生产批量，人工频繁拆卸会导致导向面快速磨损，芯盒定位精度偏移。
