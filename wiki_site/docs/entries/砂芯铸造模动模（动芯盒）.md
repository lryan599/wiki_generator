## 定义

**砂芯铸造模动模（动芯盒）**是安装在制芯设备移动工作台面的砂芯成型部件，与安装在设备固定工作台的定芯盒（静模）共同构成对开式砂芯成型模具的两个半模组成部分。二者边界划分的核心判定依据是所属安装侧在制芯过程中是否具备主动开合移动及顶出砂芯的动作能力[[S7]]。对应英文术语为 movable core box、core box moving half、core box ejector half[[S7]]。

需特别指出，此“动模”属砂芯铸造（制芯）领域工装术语，与压铸模具中的“动模”为完全不同的工艺场景——砂芯动芯盒用于砂芯成型，压铸动模用于液态金属充型成型，二者不可混淆[[S7]]。

动芯盒为通用结构，可适配热芯盒、冷芯盒、壳芯等所有主流制芯工艺，并非某单一工艺的专属部件[[S3,S5]]。

## 结构与组成

动芯盒的典型组成部分包括砂芯成型面、定位机构、顶芯出件装置以及适配不同工艺的加热/冷却元件和导向配合元件[[S15,S4,S16]]。以可燃气体加热壳芯盒为例，其剖面结构中完整包含了动芯盒本体、定位销套、顶芯杆、加热板、排气塞等核心构件，各部件位置关系见图示[[S11,S15]]。

![可燃气体加热式动芯盒（壳芯侧）整体剖面结构](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/704c49f6-839d-44ca-95c7-54847131dcfc/resource)

**定位机构**方面，动芯盒侧安装定位销套以适配定芯盒侧的定位销。为防止两半芯盒因温度差异导致定位销抱死，销套分为圆孔和椭圆孔两种结构型式，椭圆孔用于补偿热膨胀方向上的尺寸变化[[S15]]。定位销材料为T10，硬度50～55 HRC；定位销套材料为T8，硬度48～53 HRC；定位销与销套之间采用间隙配合[[S15]]。

**顶芯机构**（出芯机构）是动芯盒功能的核心构件。合模、射砂、固化完成后开模时，砂芯由预设的结构措施保留在动芯盒型腔内，随后由顶出机构驱动顶芯杆将砂芯从动芯盒中顶出。部分机型可在开模后将动芯盒整体旋转90°，使分型面朝下再完成顶出，以降低砂芯坠落破损率[[S6]]。

**加热元件**方面，热芯盒和壳芯盒工况下动芯盒需配备加热装置——加热板或直接嵌入电热管，工作温度长期保持200℃以上；冷芯盒工况下无额外加热要求，为室温工作状态[[S1,S2]]。

## 分类体系

动芯盒可按以下三个维度进行分类[[S12,S8]]：

| 分类维度 | 类别 |
|---------|------|
| 按驱动方式 | 手动式、气动式、液压式 |
| 按适配制芯工艺 | 热芯盒动侧、冷芯盒动侧、壳芯盒动侧 |
| 按分型形式 | 水平分型动芯盒、垂直分型动芯盒 |

**驱动形式**与工况适配关系：手动驱动适配小批量简单砂芯的小型制芯设备；气动驱动适配中小机型、常规节拍自动化生产；液压驱动适配大吨位锁模要求、大尺寸复杂砂芯的高速量产工况，输出动力稳定[[S6]]。

## 工艺作用与工作原理

动芯盒在制芯全流程中的动作逻辑为[[S6]]：

1. **合模阶段**：动芯盒随设备移动侧向定芯盒侧位移，完成合模锁紧；
2. **射砂与固化**：射砂头对准射砂口，将芯砂射入型腔，在相应的温度/气氛条件下固化；
3. **开模取芯**：固化完成后，动芯盒随移动侧同步位移开模，依靠预设结构使砂芯完全保留在动芯盒型腔内；
4. **顶出砂芯**：顶出机构驱动顶芯杆将砂芯从动芯盒中顶出。

**动芯盒与制芯机的安装接口**为标准T型槽结构，芯盒上的紧固耳和螺栓孔位与制芯机工作台T型槽适配，可实现快速定位锁紧[[S3,S4]]。

### 砂芯保留在动芯盒的措施

为保证开模后砂芯可靠留在动芯盒侧以便顺利顶出，常用的设计措施包括[[S3,S17]]：

- 将砂芯形状较复杂的半边或大半边型腔置于动芯盒内，增大动侧包容面积；
- 对称砂芯将分盒面向静模侧偏移0.3～1 mm，使两半芯盒对砂芯的包络面积不等；
- 动芯盒内设置更小的拔模斜度；
- 在动芯盒芯头处设置抓卡结构，强制砂芯留在动侧；
- 在静芯盒侧设置辅助推杆（弹簧顶杆）。

## 材料选择

动芯盒各部位根据工况选用不同材料[[S1,S10,S5]]：

| 材料 | 适用部位/场景 | 特性与寿命 |
|------|-------------|-----------|
| HT20-40灰铸铁 | 芯盒本体（常规量产） | 热容量大、耐热性好，适配200℃以上长期工作的热芯盒工艺，使用寿命可达4×10⁴～1×10⁵次 |
| 铝合金 | 芯盒本体（小批量、轻量化需求） | 质量轻、加工性能好 |
| T8、T10或45钢，淬火硬度50～55 HRC | 高磨损型腔镶块、定位销、定位销套 | 耐磨、硬度高 |
| 铜合金 | 高导热需求局部部位 | 导热性能优异 |

## 精度与配合要求

| 精度项目 | 要求 |
|---------|------|
| 定位孔中心距偏差 | ≤±0.05 mm（含成形烘干器修整导具的首套芯盒，一般对开芯盒为±0.25 mm） |
| 分盒面间隙 | < 0.1 mm |
| 平均轮廓尺寸<300 mm的芯盒合模错位偏差 | ≤0.05 mm |
| 300～500 mm的芯盒合模错位偏差 | <0.1 mm |
| >500 mm的芯盒合模错位偏差 | <0.12 mm |

以上精度要求直接关系砂芯成型质量，定位销与销套之间采用间隙配合，销套分圆孔、椭圆孔两种形式补偿热膨胀[[S3,S4]]。

## 排气设计

动芯盒排气系统的设计直接影响砂芯填充紧实度。总排气道面积应为射口面积的0.1～0.5倍。优先采用排气槽排气和顶芯杆间隙排气方式，在内腔死角位置加装排气塞，以改善砂芯紧实度[[S3,S15]]。

## 缺陷与机理

因动芯盒设计、制造、使用不当可能引发以下典型砂芯缺陷[[S9]]：

| 缺陷类型 | 产生机理 |
|---------|---------|
| 错边（砂芯错位） | 动定芯盒定位配合精度不足、动侧定位套磨损导致两侧合模偏移 |
| 分型面多肉飞边 | 动芯盒合模面间隙过大，射砂时型砂从缝隙溢出，固化后形成多余飞边 |
| 砂芯脱模变形 | 动芯盒拔模斜度取值过小，顶出时顶杆作用力分布不均导致砂芯受力弯曲变形 |
| 射砂不实（充型不良） | 动芯盒排气通道设计不合理，型腔内排气不畅，射砂时砂流填充受阻导致局部紧实度不足 |
| 动芯盒工作面磨损 | 动芯盒型腔壁长期受高速射砂冲刷侵蚀，导致砂芯局部轮廓尺寸超差；尤其正对射砂口位置的型面更易吹损 |

## 应用场景

动芯盒是当前批量砂芯生产的主流配置结构，典型工业应用场景包括[[S2,S13]]：

- 汽车发动机缸体水套芯自动化制芯；
- 发动机缸盖油道芯等结构复杂的异形砂芯生产；
- 摩托车气缸盖凸轮室、链盒室等砂芯的覆膜砂制芯。

在汽车动力铸件砂芯生产线上，动芯盒与自动化制芯设备配合，实现射砂、固化、开模、顶出、取芯的全流程自动化[[S6]]。

## 周边关系

动芯盒通过定位销套与定芯盒的定位销实现合模定位，通过安装螺栓固定在芯盒移动底板上。若适配侧向抽芯结构，则动侧本体需预留侧向滑块运动导向槽。开合模动力由制芯机移动侧的油缸或气缸直接提供[[S15]]。
