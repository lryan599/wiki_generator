## 概述

转台式抛（喷）丸清理机是以转台为工件承载体的抛喷丸设备，在作业过程中，被放置在转台上的工件随转台做连续或分批次转动，接受高速弹丸的冲击，以实现表面清理或强化。依据GB/T 25370-2020《铸造机械 术语》，其标准英文术语为**rotary table blasting machine** [[S2,S14]]。该设备在压铸后处理工序中承担清除氧化皮、去除毛刺、表面强化等关键任务 [[S3,S19]]。

常见的名称及别名包括：转台式抛丸机、转台式喷丸机、行星转台式抛丸机、子母转台式抛喷丸清理机；英文别名主要有 **turntable shot blasting machine**、**rotary table shot peening machine**、**tumble blast machine with turntable**、**satellite table blasting machine** 等 [[S2,S16,S20]]。

## 工艺作用

转台式抛（喷）丸清理机在压铸件后处理中具有多重作用 [[S19,S3]]：

- **表面清理**：去除铸件表面的氧化皮、粘砂、残留涂层及杂质。
- **去毛刺**：清除压铸件的飞边、毛刺。
- **表面毛化与精整**：获取具有装潢意义的表面质量，优化表面粗糙度，大幅提升后续涂层附着力。
- **表面强化**：通过弹丸冲击使工件表层产生塑性变形，引入残余压应力，提高疲劳强度和抗应力腐蚀能力。
- **缺陷暴露**：提前暴露压铸件皮下气孔、渣孔等表面缺陷，是质量检测前的重要预处理工序。

## 抛丸与喷丸的区别

转台式设备包含离心抛丸（叶轮驱动）与气动喷丸（压缩空气驱动）两种工作方式，二者存在核心技术差异，对比见下表 [[S4,S11,S12,S27]]。

| 对比项 | 转台式抛丸机 | 转台式喷丸机 |
|---|---|---|
| 动力源 | 高速旋转叶轮（离心力） | 5～6atm 压缩空气 |
| 弹丸速度 | 60～80 m/s | 50～70 m/s |
| 生产效率 | 高，功率消耗小 | 较低，功耗大 |
| 适用场景 | 喷丸强度要求高、品种少、批量大、形状简单的工件 | 喷丸强度要求低、品种多、批量小、形状复杂的工件 |
| 灵活性 | 较低，抛射方向相对固定 | 高，喷嘴方位可灵活调整，气压可控 |

## 结构与组成

转台式抛（喷）丸清理机通常由以下七大核心部分组成 [[S2,S21,S23,S30]]。

**1. 转台机构（工件承载体）**
转台是设备的核心，常见形式包括固定单转台式、回转单转台式、回转双转台式、行星转台式四类 [[S9,S30]]。普通转台多采用摩擦轮传动，台面为带圆孔的网状圆盘，表面铺设带孔橡胶板，掉落的弹丸和杂物可通过圆孔漏入下方回收结构，可实现工件连续或步进式转动 [[S30]]。行星转台可配置2～24个独立自转的子转台，工件在清理工位实现全面覆盖，同时在外部工位同步完成装卸料，形成连续加工单元 [[S16]]。

**2. 抛丸器/喷枪单元**
离心式抛丸机配置抛丸器，核心部件包括带叶片的叶轮、分丸轮、定向套，是产生高速弹丸束的关键部件 [[S4,S21]]。气动式喷丸机配置喷枪单元，包括喷嘴、喷丸罐、流量调节阀，通过调整压缩空气压力、喷射距离、喷射角度控制冲击效果，适合局部精准强化或复杂内腔清理 [[S4,S22]]。

**3. 清理室体（抛喷丸室）**
容纳工件完成清理作业的封闭腔室，具备防止弹丸逸出的密封防护结构，可配置耐磨橡胶门帘、密封隔尘结构，部分机型将室体分隔为不同扇形密封工位，分别用于装卸料和抛喷丸清理 [[S4,S9]]。

**4. 弹丸循环回收系统**
由斗式提升机、螺旋输送器、丸砂分离器、振动筛、储丸斗、弹丸闸门、流丸管等组成，完成散落弹丸的收集、输送，通过风力/磁力组合方式分离可重复使用弹丸与碎丸、杂质，实现弹丸的定量供给与循环利用 [[S2,S30,S21]]。

**5. 除尘净化系统**
收集抛喷丸过程中产生的含尘气流，经分离粉尘后排放，降低作业环境粉尘浓度，满足环保与职业安全要求 [[S28,S23]]。

**6. 电控系统**
控制整机运行逻辑，可实现转台转速、抛丸器/喷枪启停、弹丸流量、清理周期的自动化设置，部分机型可与上下游压铸生产线联动，实现全流程无人化作业 [[S30,S21]]。

**7. 加卸料装置**
配合各类转台结构实现工件的装载与卸出，行星转台式机型可在清理的同时完成装卸作业 [[S21,S16]]。

## 转台分类与形式

根据工件承载转台的配置形式，转台式抛丸清理机可划分为四种基本类型 [[S9,S30]]。

| 类型 | 结构特点 | 适用场景 |
|---|---|---|
| 固定单转台式 | 橡胶幕帘将空间分隔为室内清理区和室外装卸区，结构简单，成本低 | 中小批量扁平薄壁件 |
| 回转单转台式 | 转台可随门整体转出室外，工件装卸翻转操作空间更大 | 需较大翻转空间的工件 |
| 回转双转台式 | 两个转台绕同轴线回转，一台室内清理时另一台室外同步上下料，作业效率高 | 追求高生产率的批量作业 |
| 行星（子母多工位）转台式 | 工件随母转台公转、子转台独立自转，大转台转过半圈后工件自动翻面180°，实现全表面无死角清理 | 批量工件连续作业，均匀清理要求高 |

## 典型工艺参数

**设备规格参数**
转台直径通用范围为1500～3500mm，额定承载能力覆盖500～2000kg。国产Q3525A型转台直径为2500mm，最大可清理工件尺寸为1000mm×700mm×400mm，额定最大负载约1000kg [[S5,S30]]。

**射弹速度**
抛丸模式下叶轮驱动的弹丸抛射速度为60～80m/s；喷丸模式下5～6atm压缩空气驱动的弹丸喷射速度为50～70m/s [[S12,S25]]。

**处理时长**
常规铸件清理作业时长为5～15min [[S8]]；低氧化层轻薄压铸件可缩短至3～5min；指定抛丸强化场景可延长至10～18min [[S16,S1]]。

**弹丸选型**
针对压铸件的弹丸选型指南如下 [[S29,S26]]：

| 工件材质与场景 | 推荐弹丸材质与规格 |
|---|---|
| 普通铝合金压铸件（光亮细致表面、涂装附着） | φ0.3mm（60%）+ φ0.14～0.29mm（40%）不锈钢丸 |
| 需后序涂装的大尺寸铝合金压铸件 | φ0.8mm（50%）+ φ0.6mm（20%）+ φ0.4mm（20%）不锈钢丸 |
| 锌合金压铸件 | 锌丸，抛丸后配合钝化处理 |
| 有色金属通用清理 | 直径0.3～0.5mm弹丸 |
| 有色金属表面抛丸强化 | 直径0.5～1.0mm弹丸 |

## 在压铸行业中的应用

转台式抛丸机专门适配铝合金、锌合金等轻薄、怕碰撞压铸件的表面处理，可避免传统翻滚式清理工艺造成的铸件棱角磕碰损伤 [[S3,S16]]。具体应用包括：

- 铝合金压铸件氧化皮去除、飞边毛刺清理、表面抛丸强化。
- 锌合金压铸件的表面清理与钝化前处理。
- 汽车及摩托车铝合金轮毂、发动机连杆、齿轮等部件的批量表面处理 [[S16]]。

## 与其他清理设备的对比

各类常用抛喷丸清理设备的特点与适用范围对比如下 [[S13,S12,S25]]。

| 设备类型 | 工件承载体 | 适用铸件特征 | 局限性 |
|---|---|---|---|
| 滚筒式抛丸机 | 滚筒 | 15kg以下、不怕碰撞的中小型铸件 | 工件强制翻转，薄壁压铸件易变形磕碰 |
| 履带式抛丸机 | 环形橡胶履带 | 150kg以下、不怕碰撞的铸件 | 不适用于外观精度要求高的铝合金压铸件 |
| 吊钩式抛丸机 | 吊钩 | 中大型、形状复杂铸件 | 单批次装载效率低于转台式 |
| **转台式抛丸机** | **转台** | **高度小于50mm的扁平、薄壁压铸件** | **不适用于深内腔、高度超过50mm的异形铸件** |

## 优缺点与局限性

**核心优势**
- 结构紧凑，密封性能优异，装卸料操作便捷 [[S16,S12]]。
- 专为扁平、薄壁压铸件设计，无强制翻滚动作，工件外观完好率高 [[S12,S13]]。
- 双转台/行星转台机型可实现边清理边装卸的并行作业模式，生产率高 [[S16]]。

**固有局限**
- 工件仅靠转台自转改变朝向，对下表面、凹槽遮蔽区域存在清理死角 [[S26,S12]]。
- 不适用于带有复杂深内腔、高度超过50mm的异形压铸件的全表面清理 [[S12]]。

**环保与维护要求**
- 必须配套风量充足的袋式除尘系统，控制粉尘外溢；整机设置密闭隔音罩并安装减振基座，将车间噪声控制在职业安全限值以内 [[S26,S23]]。
- 抛丸器主轴轴承每周加注2号钙基润滑脂，其余传动轴承每3～6个月加注同型号润滑脂，链条、销轴等活动部件每周加注N46机械油 [[S5,S15]]。
- 定期检查抛丸器叶片、分丸轮、定向套、室体护板等耐磨件的磨损状态，超限后及时更换；定期筛分去除破碎弹丸并补全新弹丸，防止降低清理效率 [[S5,S15]]。

- **上游关联**：压铸件落砂/除芯工序、弹丸材料（不锈钢丸、锌丸、铝切丝丸等）、抛丸器动力系统。

- **下游关联**：除尘系统、压铸件精整、表面涂装前处理工序。
