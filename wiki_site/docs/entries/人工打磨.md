## 概述

人工打磨是压铸件去毛刺精整体系中的一类手工清理工艺，指操作人员以手持工具为主要作业载体，手动去除压铸件浇注系统余料、飞边、毛刺、残留痕迹的后处理工序[[S22,S38,S57]]。该工艺位于压铸件清理（cleaning/fettling）与精整（finishing）工序的范畴之下，属于手工去毛刺大类，与机械去毛刺、自动化去毛刺同属压铸后清理阶段的第一层级并列工艺[[S13,S9,S22]]。

人工打磨工艺适用于铝合金、镁合金等常见压铸材料，可适配从数百毫米小型减震塔到1.4 m以上大型车身结构件的一体化压铸件尺寸范围[[S1,S14,S53]]。

## 定义与术语

### 中文名称

该工艺通用中文名称为**人工打磨**，常见的同义别名包括：**传统人工打磨**、**手工打磨**、**手工作业去毛刺**、**手动锉削打磨**[[S22,S9]]。

### 英文对应术语

根据GB/T 25370-2020《铸造机械 术语》国家标准，人工参与的铸件精细清理、去毛刺工序对应到 finishing、fettling、deburring 术语体系之下。行业通用的英文对应术语包括[[S13,S30,S4]]：

| 英文术语 | 说明 |
|---|---|
| manual grinding | 人工打磨（通用） |
| manual deburring | 人工去毛刺 |
| hand grinding | 手工打磨 |
| traditional manual grinding | 传统人工打磨 |
| manual fettling | 人工铸件精整（英式用法） |

## 工艺分类与体系位置

在压铸件去毛刺和清理工艺体系中，人工打磨属于**手工去毛刺**大类，是铸件清理（cleaning/fettling）和精整（finishing）工序的组成部分，不属于机械去毛刺子类[[S13,S9]]。

人工打磨与以下工艺路线为第一层级的并列关系[[S22,S11,S33,S8]]：

| 分类层级 | 并列工艺 |
|---|---|
| 手工去毛刺 | **人工打磨** （木锤、锉刀、角磨机等） |
| 机械去毛刺 | 冲床、锯床、砂轮机、切边压力机 |
| 自动化去毛刺 | 机器人打磨、自动化打磨单元 |
| 表面清理类 | 滚筒清理、振动研磨、抛丸清理、喷砂清理 |

### 流程位置

在一体化压铸全流程中，人工打磨位于压铸工序之后、机械加工和表面处理之前的清理工部内。具体排布顺序为[[S48,S51,S10,S46]]：

1. 压铸件从模具中取出
2. 锯切、冲边去除大体积浇冒口（残留1~2 mm余根）
3. **人工打磨**——去除浇口余根、分型面飞边、顶杆痕迹等
4. 抛丸清理或振动清理
5. 机械加工
6. 表面处理

## 工艺属性

### 投入成本

人工打磨的前期设备投入极低，仅需简易手持工具。然而在批量生产场景下，人工劳务费用在整体打磨成本中占有极高比例。压铸制造成本的构成中，作业费和劳务费在总成本中的占比显著。铸件清理工序整体工作量约占压铸机总生产工作量的1/5~1/3，人工打磨在传统中小型压铸车间的劳动力占比极高[[S19,S7]]。

### 作业效率

人工打磨的作业效率明显低于自动化方式。对比数据显示[[S34,S54,S5,S22]]：

| 场景 | 人工打磨 | 自动化方案 |
|---|---|---|
| 变速箱壳体铸件打磨 | 64 件/小时 | 90 件/小时（效率提升约40%） |
| 大批量小型件 | 逐件人工打磨 | 振动去毛刺可同时处理数十件 |

人工打磨的效率还受人员熟练度、疲劳状态、情绪波动等因素显著影响，存在作业速度和合格率的随机波动[[S39]]。

### 环境条件

人工打磨的作业环境条件较差。打磨过程产生高游离二氧化硅的金属粉尘和磨屑粉尘，依据GBZ2.2-2007标准，铝合金/氧化铝粉尘8小时时间加权平均容许总尘浓度限值≤3 mg/m³。未配备合格吸尘设施的工位粉尘浓度极易超标[[S47,S15,S36]]。

打磨配套的振动清理工序噪声范围为89~96 dB，8小时工作制等效连续A声级职业接触限值为85~90 dB[[S15]]。

### 质量结果

人工打磨的作业效果完全依赖操作人员经验，不同工人的作业结果偏差大，无法保证打磨质量一致性。在大批量高一致性要求的生产场景下缺陷率会显著上升[[S39,S22]]。

## 操作条件与参数

### 典型工具

一体化压铸人工打磨阶段的典型工具如下[[S39,S46,S3,S42,S59]]：

| 工具类型 | 适用范围 |
|---|---|
| 便携式角磨机 | 大面积余料去除、浇口余根打磨 |
| 悬挂式砂轮机 | 大中型铸件局部打磨 |
| 手持磨头（Al₂O₃、SiC等） | 精细修磨 |
| 粗细规格锉刀（须配有牢固手柄） | 小型件、异形面打磨 |
| 砂带磨具 | 曲面及分型面磨光 |
| 小型风动打磨工具 | 狭窄沟槽、内圆角等死角位置 |
| 异形旋转锉（工具钢，60~65 HRC） | 铝、镁、锌等低硬度轻合金铸件 |

**安全操作要求**：锉刀严禁作为榔头、撬棒、冲头使用以防折断伤人[[S46]]。

### 磨料粒度

适用于铝合金、镁合金等一体化压铸常用材料的推荐磨料粒度[[S50,S27]]：

| 粒度规格 | 适用场景 |
|---|---|
| 30目 | 粗打磨——去除大余量残余飞边 |
| 46目 | 通用工况打磨 |
| 60目 | 精细表面精修打磨 |

### 作业环境要求

**通用除尘通风要求**[[S21,S6,S46]]：
- 打磨砂轮、砂带必须配套湿法除尘系统，作业前必须先开启除尘系统再启动打磨设备
- 打磨工位配备局部吸尘罩
- 车间设置自然通风或机械负压排风系统，保证粉尘浓度低于国家标准限值

**镁合金打磨额外强制要求**[[S44,S43]]：
- 打磨场所应单独设置，使用防爆电气设备及防爆开关、插座
- 必须使用湿式除尘器，整套除尘系统在负压状态下运行
- 不同防火分区的除尘系统不得互联
- 配备粉尘浓度监测报警仪
- 设置氢气监测报警仪

### 个人防护要求

操作人员必须佩戴防护眼镜、防尘口罩，穿戴光滑防静电工作服、防滑工作鞋[[S35,S46]]。镁合金打磨作业人员额外要求：穿戴防静电工作服，进入作业区域前先经过人体静电消除装置释放静电[[S44]]。

## 缺陷处理类型

人工打磨可处理的典型一体化压铸件缺陷类型包括[[S51,S46,S29]]：

- 分型面飞边
- 浇口余根（大浇冒口去除后残留1~2 mm厚度）
- 毛刺
- 顶杆痕迹凸起
- 铸件表面错边/龟裂纹

打磨后通常可获得R0.05~0.075 mm的圆角过渡边缘[[S24]]。

### 毛刺厚度控制要求

毛刺厚度直接影响去除工艺的可行性[[S29,S25]]：

| 毛刺隔层厚度 | 去除难度 |
|---|---|
| ≤0.1 mm | 正常合格状态，轻微用力即可去除 |
| ≥0.3 mm | 异常状态，外力去除易导致铸件基体缺肉 |

## 局限性、缺陷与隐患

### 质量缺陷

人工打磨易产生的典型质量缺陷包括[[S56,S39,S22]]：

| 缺陷类型 | 具体表现 |
|---|---|
| 切口不整齐 | 打磨边缘规整度差 |
| 铸件局部损伤变形 | 过大的打磨力度导致基体变形 |
| 漏打毛刺残留 | 人工视觉检测疏漏 |
| 过切 | 打磨量超过要求，尺寸超差 |
| 非打磨面意外划伤 | 工具误触非作业区域 |
| 打磨区域平整度不一致 | 手工操作无法保证均匀去除量 |

### 作业质量波动

人工打磨的工艺参数无法实现精确复用，打磨轨迹与打磨量完全由操作人员主观判断。作业效果受人员熟练度、当日体力疲劳状态、情绪波动等因素影响极大。相同批次铸件的打磨质量离散度远高于自动化打磨工艺，作业效率与合格率均会随人员状态出现明显的随机波动[[S39]]。

### 职业健康与安全风险

人工打磨作业的核心职业健康安全危害分为三类[[S47,S36,S15,S49]]：

| 危害类型 | 具体因素 | 参考限值 | 职业病风险 |
|---|---|---|---|
| 化学性危害 | 金属粉尘、磨屑粉尘 | 铝合金/氧化铝总尘浓度≤3 mg/m³（GBZ2.2-2007） | 铸工尘肺病 |
| 物理性噪声危害 | 打磨与振动清理噪声 | 89~96 dB，8h等效连续A声级限值85~90 dB | 噪声聋 |
| 手传振动危害 | 手持振动打磨工具 | 4h等能量频率计权振动加速度≤5 m/s²（GBZ2.2-2007） | 职业性手臂振动病 |

## 应用场景

### 适用场景

人工打磨适用于以下场景[[S22,S55,S59]]：

- 小批量试制铸件
- 结构复杂异形压铸件
- 打磨余量分布无规则的定制化铸件
- 冲床、砂带机无法触及的曲面、内圆角、沟槽等狭小位置
- 需要灵活调整打磨位置的特殊批量件

在这些场景下，人工打磨无需前期编程和专用工装投入，适配性强，试生产调试周期短，整体成本远低于自动化方案。

### 不适用场景

人工打磨不推荐用于以下场景[[S39,S16]]：

- 大批量连续化铸件生产
- 高精度尺寸一致性要求的压铸件
- 对表面打磨均匀度要求高的外观类压铸件

此类场景下人工打磨作业效率低、质量波动大，长期综合运营成本远高于自动化方案，无法满足大规模生产节拍要求。

## 与其他去毛刺工艺的对比

人工打磨、机器人自动化打磨、振动去毛刺三类工艺的多维度对比[[S34,S54,S5,S22,S40]]：

| 维度 | 人工打磨 | 机器人自动化打磨 | 振动去毛刺 |
|---|---|---|---|
| 初期设备投入 | 极低 | 高 | 中等 |
| 大批量单件成本 | 极高（人工成本占比大） | 低（批量摊薄） | 最低 |
| 作业效率 | 低（典型64件/h） | 高（可达90件/h，提升约40%） | 最高（批量同时处理） |
| 质量一致性 | 差（随人员波动大） | 优良 | 稳定（仅限低要求毛刺） |
| 工艺灵活性 | 最高（可应对无规律异形件） | 较高（可重新编程） | 最差 |
| 职业健康安全 | 风险较高（直接暴露于粉尘、噪声、振动） | 风险低 | 中等 |

## 相关标准与规范

### 安全标准

- **GB 26488-2025《镁合金压铸安全生产规范》**：明确规定镁合金压铸件人工打磨清理时，必须使用气动工具，湿式除尘设备未开启时禁止打磨作业；须采用无扬尘清扫方式和非火花清扫工具收集金属粉尘；动火作业前必须彻底清理残留镁合金粉尘并配备符合GB55036要求的消防设施后经审批方可作业[[S43]]。
- **GB 20906-2007《压铸单元安全技术要求》**（修改采用EN 869:1998）：针对打磨作业常见的振动、高温热表面、飞边飞溅等残余危险，要求向操作人员明确个体防护装备的使用指引[[S41]]。

### 质量验收标准

- **GB/T 13821《锌合金压铸件》**：规定了3个不同等级压铸件的网状毛刺、顶杆痕迹允许尺寸限值，以及各类表面缺陷总占比的限制要求，是人工打磨后锌合金压铸件外观验收的直接法定依据[[S28]]。
- **ASTM B 85《铝合金压铸件标准规范》、ASTM B 86《锌合金压铸件标准规范》**：明确压铸件分型面毛刺修整后的外观、表面粗糙度验收要求[[S52]]。
- **ISO 7722:1985《重力铸造或相关工艺生产的铝合金铸件交货及验收通用技术条件》**：规定铝合金压铸件打磨后表面质量、边角锐边处理的通用验收准则[[S31]]。
- **美国压铸协会 ADCI E系列标准**：针对量产工况下的分型面飞边打磨余量、加工公差给出通用指导值[[S52]]。
