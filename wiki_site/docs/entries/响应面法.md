## 概述

响应面法（Response Surface Methodology, RSM）是由英国统计学家G.E.P.Box与K.G.Wilson于1951年首次正式提出的一种融合数学建模与统计分析的试验优化方法[[S12,S17,S4,S41,S2]]。该方法的核心理念在于：通过科学合理的试验设计获取有限样本数据，采用低阶多项式（通常为二次多项式）作为代理模型逼近真实的因素-响应函数关系，并据此在连续设计空间内预测、分析与确定最优参数组合[[S7,S9,S25]]。值得注意的是，早期提出的RSM未将噪声因素纳入分析框架，直至20世纪80年代R.H.Myers将噪声因素引入该体系后，该方法才在工业领域得到大规模推广应用[[S12]]。

响应面法相对于传统单因子遍历法和正交试验法具有显著优势：所需试验次数更少，数据采集效率更高；能够在连续全设计空间内进行参数寻优，而非仅在离散指定点寻优；可量化评估因素主效应、二次效应以及跨因素的交互作用；具备成熟的方差分析、显著性检验与残差诊断工具链；可输出显式多项式数学映射关系，直接反演得到指定目标下的最优参数组合[[S17,S25,S9,S4,S28]]。

## 数学基础

### 函数关系与近似模型

响应面法的基本出发点是将响应量\( Y \)与因素集\( x_1, x_2, \dots, x_n \)之间的关系表示为如下一般函数形式[[S12,S2]]：

\[ Y = f(x_1, x_2, \dots, x_n) + \varepsilon \]

式中，\( f \)为待求的响应面函数，\( \varepsilon \)为拟合误差。当因素与响应之间的关系远离最优区域时，常采用一阶线性模型逼近[[S12,S15,S4]]：

\[ y = \beta_0 + \sum_{i=1}^{k}\beta_i x_i + \varepsilon \]

当一阶模型逼近发现拟合误差显著，或因素与响应之间确认为非线性关系时，需转而采用二阶模型进行精确逼近。二阶响应面法的标准多项式回归形式为[[S12,S22,S2,S35]]：

\[ y = \beta_0 + \sum_{i=1}^{k}\beta_i x_i + \sum_{i=1}^{k}\beta_{ii}x_i^2 + \sum_{i<j}\sum\beta_{ij}x_i x_j + \varepsilon \]

式中，\( \beta_0 \)为常数项，\( \beta_i \)为因素\( x_i \)的一次效应系数，\( \beta_{ii} \)为二次效应系数，\( \beta_{ij} \)为两因素交互效应系数，\( \varepsilon \)服从正态分布\( N(0, \sigma_y^2) \)。

### 最小二乘拟合

响应面多项式回归系数的估计采用最小二乘准则：通过最小化残差平方和求得最优回归系数，其矩阵表述形式为[[S12,S26,S7]]：

\[ \mathbf{B} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y} \]

### 响应曲面与等高线

响应曲面的数学定义为期望响应值\( \eta = f(x_1, x_2, \dots, x_n) \)构成的超曲面[[S12,S40]]。等高线是使用平行于自变量坐标平面的平面截取响应曲面后得到的等响应值投影线，每条等高线上的所有点对应完全相同的响应输出数值[[S12,S40]]。

## 常用试验设计类型

响应面法常用的二阶试验设计主要有两种：中心复合设计（Central Composite Design, CCD）和Box-Behnken设计（BBD）。

### 中心复合设计（CCD）

中心复合设计的试验点由三部分构成：\( k \)个因素对应的\( 2^k \)个两水平全因子（或部分因子）点、\( 2k \)个轴向（星型）点以及\( N_0 \)个重复中心点[[S2,S1,S13]]。CCD共有三种亚型：普通CCD的轴向点距离设计中心\( \alpha > 1 \)；面心CCD的轴向点直接落在立方体面上；内接CCD的轴向点距离设计中心为\( 1/\alpha \)[[S1,S13]]。为有效估计输出响应的试验方差，中心点至少需重复3至5次[[S1,S13]]。总试验点数计算公式为\( n = 2^k + 2k + N_0 \)[[S2]]。

### Box-Behnken设计（BBD）

Box-Behnken设计的试验点由两两扰动两个变量的组合点构成，通用总点数计算公式为\( 2n_{dv}(n_{dv}-1) + n_c + 1 \)（\( n_{dv} \)为因素数量，\( n_c \)为中心点重复次数）[[S1,S13,S6]]。BBD不包含超出原始因素水平范围的轴向点，所有设计点全部落在原始设定的三水平参数区间内部，分布在立方体各条棱边的中点位置[[S6,S15,S42]]。

### CCD与BBD综合对比

两类设计在试验点结构、试验次数、适用范围等方面存在显著差异，下表进行系统对比。

| 对比维度 | 中心复合设计（CCD） | Box-Behnken设计（BBD） |
|---|---|---|
| **试验点构成** | \( 2^k \)个全因子点 \( +2k \)个轴向点 \( +N_0 \)个中心点 | 两两变量扰动组合点 \( +n_c \)个中心点 \( +1 \) |
| **轴向点** | 包含\( 2k \)个轴向点，部分类型轴向点可超出原始因素设定的立方体边界 | 不存在轴向点，所有试验点均位于预先划定的因素安全操作区间内[[S6,S15]] |
| **极端水平组合** | 包含完全的角点（所有因素同时取最高/最低值），支持角部插值预测 | 不生成角点极端组合，角部预测需依赖外推，角点区域预测精度较差[[S6,S1,S43]] |
| **序贯扩展能力** | 具备序贯性，可在一阶两水平试验基础上仅追加轴向点和少量中心点直接升级为完整二阶设计 | 不支持序贯扩展，单次设计直接生成全部三水平试验点[[S15]] |
| **适用因素数量** | 2~6个因素，总试验次数约14~90次[[S6]] | 2~5个因素，3因素标准BBD仅需15次，4因素仅需29次[[S6,S42]] |
| **试验次数（同因素数）** | \( 2^k+2k+N_0 \)；7因素无中心点重复时143次 | \( 2n_{dv}(n_{dv}-1)+n_c+1 \)；7因素无中心点重复时仅需85次，约为CCD的47%[[S2,S13,S1]] |
| **拟合精度特点** | 全域对响应曲面的整体拟合效果更优，全域预测效率更高，更适合大范围探究非线性关系 | 在设计空间中心区域的预测精度更高，试验量更精简，试验执行成本更低[[S15,S30,S6]] |

## 一般实施步骤

响应面法的完整通用实施流程可规划为以下五个核心阶段[[S39,S15,S6,S32,S4]]。

1. **因子筛选阶段**：当候选因素数量较多时，采用Plackett-Burman等两水平筛选设计，从大量候选因素中快速识别出对响应值有显著影响的关键因子[[S39]]。

2. **一阶模型寻优阶段**：若当前试验条件远离最优响应区域，采用一阶线性模型逼近响应曲面，通过最速上升法或最速下降法快速定位最优响应区域[[S15,S4,S39]]。

3. **高阶试验设计阶段**：在已定位的最优邻域范围内，选用中心复合设计（CCD）或Box-Behnken设计（BBD）生成试验样本点，构建二阶模型的实验矩阵[[S6,S39,S4]]。

4. **模型拟合与诊断阶段**：采用最小二乘法拟合二次多项式回归模型；通过方差分析（ANOVA）检验模型整体显著性（模型P值通常需<0.0001）、失拟项显著性（失拟P值需>0.05）以及各因素项的显著性（F值和P值）；通过残差正态概率图检验残差分布的正态性；通过决定系数\( R^2 \)（通常需>0.94）、调整\( R^2 \)、预测\( R^2 \)等指标评估拟合精度[[S10,S38,S45,S32]]。

5. **模型验证与优化阶段**：采用实测样本或独立数值模拟检验模型预测精度（预测值与验证值偏差通常应<6%），最终基于拟合完成的响应面方程求解最优参数组合[[S37,S11,S19]]。

## 压铸领域典型应用

响应面法在压铸工艺优化中已得到广泛研究与应用，覆盖铝合金和镁合金两大主流材料体系。

### 典型优化对象

**铝合金方面**：涵盖A360、A356、AlSi10MnMg、A380、ADC12等常用压铸牌号，典型铸件包括减震塔、冷却盖板、水泵座、新能源汽车逆变器壳体、通信机箱、变速箱壳体等[[S33,S36,S29]]。

**镁合金方面**：涵盖AZ91D、LA42等牌号，典型铸件包括轮毂、滤波器壳体等[[S29,S33]]。

### 典型输入因素与输出响应

压铸RSM优化的典型可控工艺因素包括：浇注温度、模具温度（含预热温度）、慢压射速度、快压射/注射速度、增压压力、保压时间及真空度等[[S46,S20,S37]]。对应的典型输出响应变量包括：孔隙率/缩松体积、抗拉强度、延伸率、表面粗糙度、硬度、局部热节温差、凝固时间及二次枝晶臂间距（SDAS）等[[S20,S29,S37]]。

### 建模方法与统计特征

压铸领域的响应面优化普遍采用包含常数项、各因素线性项、交互项及二次项四类项的二阶多项式回归模型拟合工艺参数与响应变量的映射关系[[S46,S22]]。

对回归方程经方差分析（ANOVA）后，典型统计特征如下表所示。

| 统计指标 | 典型压铸RSM研究中的取值或判定 | 含义 |
|---|---|---|
| 模型F值 | 可达61.29 | 数值越大，模型显著性越好 |
| 模型P值 | <0.0001 | 模型统计极显著 |
| 失拟项P值 | >0.05（典型值0.2102） | 失拟不显著，回归方程拟合质量可靠 |
| 决定系数\( R^2 \) | 0.94~0.99（典型案例0.9762） | 模型对实验数据的拟合度高 |
| 调整\( R^2 \)与预测\( R^2 \)差值 | <0.2 | 泛化预测能力良好 |
| 变异系数CV | <10%（典型案例1.30%） | 实验数据可信度和精确度高 |
| 信噪比（Adeq Precision） | >4（典型值20.032） | 信号充足，回归模型可用于预测 |

以上典型统计特征表明，正确构建的压铸响应面回归模型具备高精度的拟合能力和良好的预测可靠性[[S10,S38,S45]]。

### 单目标与多目标优化

**单目标优化场景**：以最小化铸件总缩松率或局部热节温差为单一目标进行工艺参数寻优。在典型A356铝合金减震塔优化案例中，以模具温度、浇注温度、压射速度为因素，铸件总缩松率为响应，经RSM-CCD设计与寻优后，得出最优工艺参数为模具温度约205.6°C、浇注温度约666.0°C、压射速度约2.625 m/s，预测缩松率降至3.47%[[S5]]。

**多目标优化场景**：同时对孔隙率、二次枝晶臂间距（SDAS）等多个相互约束的质量目标进行协同寻优。在典型AZ91D镁合金轮毂优化案例中，以浇注温度和充型压力为设计变量，以缩松体积与SDAS为双目标，经双层RSM模型与多目标优化后，缩松率从4.1%降至2.1%，二次枝晶臂间距从88.5μm降至81.2μm[[S29]]。

## 优势与局限性

### 核心优势

1. **试验效率高**：相对于传统单变量遍历优化和正交试验法，所需试验次数更少，试验数据采集效率更高[[S17,S25,S4,S9]]。

2. **连续空间寻优**：支持在连续全设计空间内进行参数寻优，突破正交试验仅能在离散指定点寻优的局限[[S25,S9,S28]]。

3. **交互效应量化**：可量化评估因素主效应、二次效应以及跨因素的交互作用，更贴近真实复杂系统的作用机制[[S25,S9]]。

4. **模型诊断体系完善**：配套成熟的方差分析、显著性检验、残差诊断工具链，系统性评估模型拟合质量[[S25,S4]]。

5. **显式反演能力**：可输出显式多项式数学映射关系，直接反演得到指定目标下的最优参数组合[[S9,S28]]。

### 应用局限性

1. **非线性拟合能力边界**：标准二阶多项式响应面的拟合能力仅能适配低阶非线性关系，无法精确逼近高度非线性、非光滑的复杂隐式函数[[S7,S25,S34]]。

2. **"维度灾难"**：当设计变量维度很高时，试验点样本量呈指数级增长，试验/仿真成本急剧上升[[S7,S34]]。

3. **外推精度下降**：标准响应面多项式在远离试验点的设计空间外推区域预测精度大幅下降，仅在试验样本覆盖的局部区域内可保证可靠性[[S25,S34]]。

4. **多目标寻优需额外组合算法**：常规二阶响应面方法在多目标优化场景下需额外搭配多目标优化算法（如遗传算法NSGA-II）使用，无法独立直接输出帕累托最优解集[[S25,S3]]。
