## 定义与基本概念

**模具清理装置**是压铸模具日常维护与定期保养的核心配套设备，其设计目的是去除模具分型面、型腔表面残留的脱模剂、铝屑、积碳等污染物，避免因模具合模不严、金属或排气通道堵塞而引发的压铸件质量缺陷[[S19]]。

在压铸工艺中，模具清理装置的功能定位涵盖即时在线喷吹与深度离线清洗两个层面。早期集成式压铸模清理润滑装置已采用压缩空气喷吹方式，安装在压铸机主机侧部，开模时移动至动定模之间清除压铸模表面飞边等松散附着物[[S6]]。

## 在线清理与离线清理

根据是否需将模具从压铸机拆下，模具清理装置分为两大工作模式：

- **在线清理**：无需拆卸模具，在正常生产节拍间隙即可完成快速除垢，不中断连续生产流程，清理效率高[[S18,S2]]。
- **离线清理**：需将模具从压铸机整机拆下，待模具完全降温后移入专用清洗工位完成全面深度除垢，通常用于模具定期大保养场景，耗时更长[[S18,S2]]。

## 主要清理方法分类

按清理原理与介质划分，压铸模具主流清理类别涵盖机械清理、热清理、化学清洗、干冰清洗、激光清洗、超声波清洗六大类[[S18,S2,S12,S21]]。

各方法的工艺特性与应用差异如下：

| 清理方法 | 典型作业模式 | 是否原位作业 | 对模具基体的磨损 | 二次污染/清理要求 |
|---|---|---|---|---|
| 喷砂/喷丸清理 | 离线 | 否 | 存在磨损 | 需二次清理去除表面浮尘 |
| 化学清洗（酸碱法/盐浴法） | 离线 | 否 | 存在腐蚀风险 | 需二次清理去除残留化学试剂 |
| 超声波清洗 | 离线 | 否 | 极低 | 清洗介质需处理 |
| 干冰清洗 | 在线 | 是 | 无磨损 | 干冰完全升华，无二次清理 |
| 激光清洗 | 在线 | 是 | 无磨损 | 几乎无后处理废液 |

数据来源：[[S18,S2,S12,S3,S20]]。

## 干冰清洗装置

### 工作原理

干冰清洗的核心作用机制包含三重效应[[S18,S2,S13,S4,S22,S16]]：

1. **热冲击效应**：-78℃的干冰颗粒冲击模具表面，使表面附着的涂料残留、污垢等脆化开裂。
2. **动能机械剥离效应**：高速干冰颗粒将脆化的污垢层直接击碎带走。
3. **升华膨胀“微型爆炸”效应**：干冰冲击瞬间在千分之几秒内体积膨胀近800倍，形成微型爆炸将污垢层剥离。

由于模具基体与表面涂层、油污、残渣及结垢具有不同的热膨胀系数，在干冰冲击下，污染物即被剥离。热冲击仅作用于污染物与基体相交的表面，即使模具表面温度短时间内降至干冰温度（-78℃），也不会对模具基体金属产生热应力，不影响基体组织[[S1,S13]]。

### 系统组成

完整的压铸用干冰清理系统核心组成包括[[S2,S5,S1]]：

- 液态CO₂储罐
- 干冰造粒机（制备直径3mm、长度2.5～10mm的柱状干冰颗粒）
- 压缩空气系统
- 干冰喷射机（分为单喉管高冲击力型和双喉管低冲击力型）
- 干冰保温存储箱
- 专用喷嘴及喷射软管
- 大规模连续生产场景下还需配套CO₂气体监测装置

### 核心工艺参数

干冰清洗的核心工艺参数如下[[S2,S13,S18]]：

| 参数项 | 典型值/范围 |
|---|---|
| 干冰颗粒直径 | 2～3mm |
| 喷射速度 | 约300m/s |
| 干冰消耗量 | 0.5～1kg/min |
| 1m²型腔表面清理耗时 | 1～5min |
| 可通透清理的排气孔直径 | ≥φ0.7mm |
| 可通透清理的排气孔深度 | ≥50mm |

### 核心优势

干冰清洗在压铸模具清理中具备多项显著优势[[S18]]：

- **可在线作业**：无需拆卸模具，省去模具降温、拆卸、安装、重新加热的全流程耗时，大幅提升生产效率。
- **无二次污染**：干冰完全升华气化，无残留溶剂或液体残渣，清理后可直接投入生产。
- **无模具表面磨损**：有效延长压铸模具使用寿命，综合经济效益优于喷砂、化学清理。
- **缝隙与隐蔽处清理**：对模具排气孔及局部复杂拐角等处具有优异清理能力，φ0.7mm以上排气单孔（孔深50mm以上）可直接通透清洗。

### 安全操作要求

由于CO₂比空气密度大，在封闭空间或地势低洼区域进行干冰清理作业时，必须保证足够通风，防止操作人员发生CO₂窒息风险[[S18]]。

## 喷砂清理装置

### 工艺方法及参数

压铸模具喷砂清理可采用干式或湿式模式[[S3,S12]]：

- **干式介质**：铝质粘土粉、氧化铝粉。
- **湿式介质**：10～15μm粒度玻璃珠与水的混合喷磨剂。

关键工艺参数[[S3]]：

| 参数项 | 数值 |
|---|---|
| 压缩空气工作压力 | 0.5～0.7MPa |
| 喷射距离 | 10～120mm |
| 常规保养介质 | 粒度90μm玻璃丸 |
| 结垢严重时介质 | 120目白刚玉砂 |
| 常规保养清理耗时 | ≤1～2min |
| 结垢严重模具清理耗时 | 5～20min |

喷砂清理作业对模具型腔的尺寸精度与表面粗糙度影响极低[[S3]]。然而，喷砂清洗在精密模具（如橡胶轮胎模具）场景下除垢率仅约85%，且可能在模具表面形成密集凹坑[[S17]]。

## 化学清洗装置

### 工艺方法及参数

压铸模具化学清洗包含酸碱法和盐浴法两种主流工艺[[S12,S21]]：

| 工艺类型 | 主要操作条件 |
|---|---|
| 酸碱法-除油 | 2.5%～5%苛性钠溶液煮沸30min |
| 酸碱法-除积碳 | 3.5%～8%稀硫酸，55～65℃，浸泡5～10min |
| 酸碱法-中和 | 1%浓度碱液中和残留酸 |
| 盐浴法 | 425～480℃苛性钠、硝酸钠、高锰酸钾混合熔融盐，浸泡20～50min |

此外，碱煮清理可采用质量分数20%～30%的苛性钠溶液或40%～50%的苛性钾溶液加热煮沸；碱爆清理采用500～520℃、质量分数90%～95%的苛性钠熔融液，处理后快速入水产生高压蒸汽完成脱垢[[S24]]。

化学清洗需后续二次清理，且对铝合金压铸模具基体存在腐蚀风险[[S12]]。

## 激光清洗装置

### 作用机制

激光清洗主要存在两类作用机制[[S7]]：

1. **吸收系数差异型**：利用基片与附着物对指定波长激光的吸收系数差异，大部分激光能量被附着物吸收使其气化蒸发或瞬间膨胀脱离模具表面，基片因吸能极低不会受损。
2. **声波反射型**：采用高功率高重复率脉冲激光作用于表面，通过声波反射产生的微小爆炸将附着物粉碎为粉末后由吸尘装置收集，基片也不会产生损伤。

### 性能特征

激光清洗适配压铸模具的高精度无磨损清理场景，可实现原位无介质作业[[S20]]。与其他清理方法的多维度性能对比如下：

| 比较项目 | 喷丸 | 干冰清洗 | 湿法化学清洗 | 激光清洗 |
|---|---|---|---|---|
| 原位清洗 | 不能 | 是 | 不能 | 是 |
| 劳动强度 | 高 | 中间值 | 高 | 低 |
| 自动化程度 | 低 | 低 | 低 | 高 |
| 噪声水平 | 中间值 | 高 | 低 | 低 |
| 基体的磨损 | 是 | 不磨损 | 中间值 | 不能（不磨损） |
| 环境的危害 | 中间值 | 中间值 | 高 | 低 |
| 后处理的废液 | 高 | 低 | 高 | 低 |

数据来源：[[S20]]。

## 超声波清洗装置

超声波清洗属于压铸模具的典型离线清理工艺。将完整模具浸入清洗介质后通过超声振动作用5～10min即可完成整体型腔、缝隙的全面除垢[[S2]]。该设备已被GB/T 25370-2020《铸造机械 术语》收录为标准术语[[S9]]。

## 应用场景与维护周期

### 日常清理

每次压铸件取出后，利用喷涂装置在喷涂脱模剂之前，先用压缩空气喷吹型腔及分型面，吹掉脱模剂残留、铝屑等附着物。发生粘模现象时需使用专门工具小心清除[[S19]]。

### 定期维护

按生产批量划分的清理维护周期如下[[S19,S11]]：

| 维护级别 | 执行时机 | 主要内容 |
|---|---|---|
| 小规模维护 | 每个压铸批次完成后 | 拆卸模板和顶出板，清洗顶杆、复位杆，清除分型面和型腔附着的金属及脱模剂污物 |
| 中维修 | 数个批次或预定压射量完成后 | 在小维护基础上，增加浇道、浇口、溢流槽、排气道的深度清理 |
| 大维修 | 模具达到预期寿命约1/3时 | 修整模框平面度和直角度，恢复模具功能精度，同步清理冷却通道内水垢 |

### 去应力处理周期

压铸模具全生命周期去应力维护周期为[[S2]]：

- 新模首次去应力回火：5000～10000次压射后执行
- 后续去应力处理：每完成10000～20000次压射执行一次
- 回火温度需低于模具原回火温度30～50℃

## 装置选型依据

选择模具清理装置时需综合考虑以下因素[[S18,S20,S17]]：

- **模具材料与污染物类型**：铝合金压铸模具应避免使用存在腐蚀风险的化学清洗方法。
- **生产节拍要求**：需要不中断连续生产的场景优先选用在线清理装置（如干冰清洗、激光清洗）。
- **模具精度要求**：高精度模具应优先选择无磨损的清理方式（干冰清洗、激光清洗）。
- **经济性与环保要求**：干冰清洗全程无二次污染，综合经济效益优于喷砂、化学清理；激光清洗自动化程度最高，但设备成本较高。
