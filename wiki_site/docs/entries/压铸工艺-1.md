## 概述

压铸（die casting），在国际标准《GB/T 20906—2007 压铸单元安全技术要求》中被定义为“熔融金属被射入模具中，并在高压状态下被保持到完全凝固为止的过程”[[S32]]。其中文全称为“压力铸造”，简称“压铸”，又称高压铸造（High Pressure Die Casting, HPDC），英文术语“die casting”是铸造领域广义压铸大类的统称，而“high pressure die casting”（HPDC）专门指代具备高压力、高充型速度特征的主流高压压铸工艺，可明确区别于低压铸造、重力铸造等低充型压力的工艺类别[[S35,S60,S39]]。

该工艺属于近净成形（near-net-shape）技术，可实现尺寸公差达 CT3~CT6 级、表面粗糙度 Ra 0.8~3.2 μm 的复杂薄壁件直接成形，材料利用率可达 60%~80%[[S35,S7]]。

## 工艺原理

压铸核心工艺过程分为两大关键阶段[[S7,S63,S34]]：

1. **高压高速充型**：金属液在高压力驱动下，以 0.5~120 m/s 的充型速度快速填充模具型腔。
2. **持压凝固**：金属完全填充型腔后，高压力持续作用于正在凝固的金属，通过压力补缩消除部分缩孔缺陷。

高压、高速是压铸法与其他铸造方法的根本区别。高压压铸在全铸造工艺体系中处于最高压力与最高速度的区域，明显区别于重力砂型铸造、永久模铸造、挤压铸造、半固态铸造的参数区间[[S27]]。

![不同铸造工艺的压力与速度关系对比图，突出高压压铸处于最高压力与最高速度区域](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/46b54e80-1691-4197-9c2d-d29166472651/resource)

压铸全流程压力演化分为四个阶段[[S5]]：第一阶段为压室排气（压射冲头慢速推进金属液）；第二阶段为金属液填充型腔，产生压力峰；第三阶段为压力继续上升，完成充型；第四阶段为增压阶段，增压压力作用于正在凝固的金属液上实现压力下凝固。高压压铸过程中，活塞压力的变化曲线清晰展示了这四个阶段的压力特征[[S78]]。

## 核心工艺参数

压铸工艺的核心参数包括压力、速度、时间与温度四大类[[S5]]。

### 压射比压

压射比压是压室内金属液单位面积上所受的压力。典型取值范围如下[[S5]]：

| 铸件类型 | 压射比压范围 |
|---|---|
| 普通结构件 | 40~50 MPa |
| 承载件 | 50~80 MPa |
| 薄壁件/气密性要求件 | 80~100 MPa |
| 冷室压铸机动态压射压力 | 30~90 MPa |
| 冷室压铸机增压压射压力 | 50~300 MPa |

### 压射速度

- 慢压射速度：0.1~0.8 m/s[[S22]]
- 快压射速度常规范围：0.1~7 m/s，现代高性能压铸机可覆盖 0.05~10 m/s 的全区间[[S22]]
- 内浇口速度可达 10~50 m/s，部分超过 80 m/s[[S63]]

### 慢/快压射切换点

慢压射速度一般在 0.1~0.8 m/s 范围内选择；快压射速度与内浇口注入速度成正比，从低向高调节，在不影响铸件质量的前提下以较低的快压射速度为宜[[S22]]。

## 设备与分类

压铸机按压室的受热条件和结构特征，主要分为热室压铸机和冷室压铸机两大类。

### 热室压铸机

热室压铸机的压射室与保温熔炉为一体化设计，压室直接浸没在熔融金属液面以下，压射部件位于坩埚上方[[S46,S53]]。其典型特征为：

| 项目 | 参数/说明 |
|---|---|
| 适用合金 | 锌、铅、锡等低熔点合金为主；部分小规格机型可用于镁合金 |
| 主流锁模力上限 | 约 400 吨 |
| 压射压力 | 7~14 MPa |
| 生产效率 | 小型热室压铸机可达 400~1000 次/h |
| 优点 | 工序简单、效率高、金属消耗少、工艺参数稳定 |

### 冷室压铸机

冷室压铸机的压射室与保温炉完全分离，压铸时需将定量金属液从外部保温炉舀入压室[[S46,S18]]。其典型特征为[[S81,S46]]：

| 项目 | 参数/说明 |
|---|---|
| 结构布局 | 主流为卧式布局 |
| 压射压力 | 可达 40~60 MPa |
| 最大锁模力 | 现有大规格机型可超过 45,000 kN |
| 适用合金 | 铝合金、镁合金、铜合金甚至黑色金属 |
| 生产效率 | 75~85 次/h（冷压室）；80~100 次/h（典型范围） |
| 最大铝合金铸件质量 | 约 40~50 kg |

### 改进工艺

#### 真空压铸

真空压铸是通过真空泵在金属充型前将压铸模型腔内的空气抽出建立真空环境，再压入金属液的方法[[S65,S74]]。高真空压铸工艺可将型腔内真空度控制到 5~10 kPa，铸件含气量低于 4 mL/100g Al，可支持铸件热处理和焊接，显著提升力学性能[[S10]]。

#### 充氧压铸（无气孔压铸）

充氧压铸主要在铝合金压铸中应用，充型前用氧气置换型腔和压室内的空气，氧气与铝液反应生成占总质量 0.1%~0.2%、尺寸小于 1 μm 的弥散氧化铝微粒[[S57,S47]]。充氧后铸件铸态强度可提升 10%，延伸率提升 0.5~1 倍，可进行热处理，可在 200~300 ℃ 温度环境下工作[[S57,S47,S67]]。与真空压铸相比，充氧压铸装置结构简单，操作方便，投资少[[S47]]。

## 典型材料与合金牌号

### 压铸铝合金

常用压铸铝合金以 Al-Si-Cu 系列应用最为广泛[[S66,S69]]：

| 牌号 | 标准体系 | 主要特征 |
|---|---|---|
| ADC12（A383） | JIS | 铸造性能优异，充型流动性好，切削加工性能优良 |
| ADC10（A380） | JIS/ASTM | A380 系列合金占美国铝合金压铸件产量的 85% 左右[[S48,S51]] |
| YL102 | GB（中国） | 相当于原苏联标准中的 JI2，约占铝合金压铸件的 80% 以上[[S66]] |
| YL104/Y112/Y113 | GB（中国） | 综合性能好，常用于承载件 |

日本摩托车行业 85% 以上压铸件采用 Al-Si-Cu 系列合金（ADC10、ADC12 等）[[S66]]。

### 压铸镁合金

常用压铸镁合金主要牌号如下[[S73,S4,S14,S9]]：

| 牌号 | 合金系 | 典型成分 | 主要特征与应用 |
|---|---|---|---|
| AZ91D | Mg-Al-Zn | Mg-9Al-0.7Zn-0.2Mn | 全球应用最广泛的压铸镁合金，优良铸造性能与结构强度，用于壳盖类、方向盘、支架等[[S73,S4]] |
| AM60(B) | Mg-Al-Mn | 铝含量约 6% | 优异的韧性、吸震性能与强度平衡，用于座椅支架、仪表盘等保安件[[S73,S4]] |
| AM50A | Mg-Al-Mn | — | 良好韧性与吸震性，用于汽车结构件[[S73]] |
| AS41B | Mg-Al-Si | — | 含硅增强抗蠕变性能，用于高温服役部件[[S73]] |

AZ91D 中，A 代表铝元素、Z 代表锌元素，数字 9 对应铝的质量分数约 9%，数字 1 对应锌的质量分数约 1%，后缀 D 代表高纯度级别[[S16,S37,S28]]。

压铸 AZ91-Ce 稀土改性镁合金已开展完整的工程化压铸工艺研究，通过 ProCAST 数值模拟优化了浇注温度、压射速度、模具预热温度等关键工艺参数，验证了其微观组织优化与力学性能提升效果[[S17]]。

### 压铸锌合金

压铸锌合金熔点约 365~385 ℃，熔融状态流动性优异，可制造形状复杂的薄壁精密压铸件[[S69,S3]]。行业应用最广泛的是 3 号锌合金；此外 ZA-8、ZA-12、ZA-27 等高铝锌基合金可采用冷室压铸机进行生产[[S69,S3,S73]]。

## 优点

压铸工艺的主要优点如下[[S13,S33,S31,S56,S72,S20]]：

| 优点 | 说明 |
|---|---|
| 尺寸精度高 | 可达 IT11~IT13 级，工艺控制优化后可达到 IT9 级 |
| 表面粗糙度低 | 典型范围 Ra 0.8~3.2 μm，最优可达 Ra 0.4 μm |
| 可成形复杂薄壁件 | 锌合金最小壁厚 0.3 mm，铝合金 0.5 mm，最小铸出孔径 0.7 mm |
| 材料利用率高 | 60%~80%，毛坯利用率可达 90% |
| 生产效率极高 | 冷室压铸机 50~100 次/h，热室压铸机 400~1000 次/h |
| 组织致密 | 表层晶粒细化，抗拉强度可比砂型铸件提高 25%~30% |
| 可实现嵌铸 | 可直接嵌铸其他材料零件，简化制造工艺 |

## 缺点与局限性

| 缺点 | 说明 |
|---|---|
| 内部气孔缺陷 | 高速充型时型腔内气体难以完全排出，气孔残留导致普通压铸件不能进行固溶热处理[[S20,S72,S42]] |
| 设备与模具成本高 | 压铸模具制造周期长，小批量生产经济性差[[S72,S20]] |
| 压铸件尺寸受限 | 受锁模力与装模空间限制 | 
| 合金种类受限 | 目前主要适用铝、锌、镁、铜等低熔点有色金属，黑色金属压铸模寿命极短[[S72,S20]] |

## 主要缺陷类型及成因

依据标准 STN 42 1240，压铸缺陷可分为五大类：形状尺寸类缺陷、表面类缺陷、连续性中断类缺陷、夹杂物类缺陷、冶金类缺陷[[S64]]。

### 气孔

金属液高速充型过程中紊流裹挟的气体、熔炼过程溶解的气体、铸型表面有机物分解产生的气体无法完全排出，残留在铸件内部形成内壁光滑的球形或近球形孔洞[[S64,S23,S52,S12]]。气孔会显著降低铸件疲劳强度，且普通压铸件内部残留气孔是无法进行固溶热处理的核心原因。

### 缩孔缩松

金属液凝固过程中液态收缩与固态收缩产生的体积变化得不到足够的金属液补缩，最后凝固的热节区域形成内壁粗糙、带有明显树枝晶特征的集中孔洞（缩孔）；大量分散微小缩孔聚集则形成缩松[[S23,S52,S12,S40]]。缩孔缩松会破坏铸件结构连续性，大幅降低气密性与力学性能。

### 冷隔

两股或多股温度较低的金属液流在充型过程中相遇，前沿金属液已经提前部分凝固，未能完全熔合形成带有圆角边缘的表面接缝[[S64,S52,S59,S62]]。成因通常与浇注温度不足、压射压力偏低、充型流程过长相关。

### 裂纹

- **热裂**：形成于铸件准固相凝固阶段，凝固收缩产生的应力超过固相骨架强度极限，沿晶界撕裂，断口带有明显氧化色[[S64,S12,S52,S62]]。
- **冷裂**：形成于铸件完全冷却进入弹性状态后，铸造残余应力超过合金强度极限，通常为穿晶断裂，断口呈金属光泽[[S12]]。

## 应用领域

国内压铸件行业分布为：汽车及摩托车行业占 48%，日用品占 28%，机械产品及零部件占 13%，家电及电子产品占 11%[[S41,S33,S43]]。

### 汽车工业

汽车工业是压铸件最大的应用领域。压铸铝合金在传统汽车各部位的占比为[[S19,S49]]：

| 部位 | 占比 |
|---|---|
| 发动机部件（缸套、缸盖、活塞、进气歧管等） | 61.9% |
| 车轮 | 15.7% |
| 散热器 | 12.0% |
| 底盘及悬架系统 | 4.6% |
| 车身覆盖件 | 1.2% |
| 车身受力件及保险杠 | 1.1% |

### 新能源汽车

新能源汽车三电核心结构件典型压铸应用场景包括[[S71,S25,S11,S77]]：
- **动力电池包壳体**：高真空压铸一体化下壳体（如特斯拉 Model Y 采用 AlSi10MnMg 合金一体压铸下壳体，替代钢制结构减重 40%）
- **电机壳体**：复杂水冷通道电机壳体
- **电控单元壳体**：高电磁屏蔽性能壳体

![高压压铸生产的新能源汽车三电系统典型精密压铸件实物图](http://192.168.150.150:8000/api/v1/workspaces/die_casting_wiki_v2/document-elements/e70bec20-858a-4fb9-8809-9e763e4ec5b3/resource)

### 航空航天

压铸件在航空航天非主承力构件中获得应用。典型案例如 A340~500(600) 大型客机采用的 AlSiCu3 合金化学供氧装置支座，单重 0.46 kg，最小壁厚 1 mm，满足轻量化和高刚性要求，2002 年获德国压铸协会一等奖[[S49]]。

### 消费电子

3C 产品（计算机、通信、消费电子）是镁合金压铸件的重要应用领域，如笔记本电脑外壳、手机外壳、数码相机壳体等[[S21]]。
